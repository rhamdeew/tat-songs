# Оригинал

### Айдар Галимов - Алмачуар

```
Жил уйнатып атлар узды олы юлдан,
Тояклардан кутэрелгэн тузан тынган.
Томырылып чапкан ярсу алмачуар
Эллэ инде, минем яшьлек узе булган.

Пар канатым, тулпар атым алмачуар
Купме биек урлэр мендек, кичтек сулар.
Текэ ярдан канатланып оча иден,
Зур упкыннар очраганда юлда синен.

Сискэндереп атлар кешни ак томанда
Тулпар атым — алмачуар котэ анда.
Томаннарга эзлэп керэм, адашам да,
Ялгыз калам тагын томан таралганда.
```

# Перевод

### Айдар Галимов - Иноходец

```
Ветер играя, кони промчались по большой дороге,
Поднятая копытами пыль улеглась.
Быстро скачущий красавец-иноходец,
Видно, уже моя молодость прошла.

Пара крыльев, конь-тулпар — иноходец,
Сколько высоких гор преодолел, рек переплыл.
С крутого берега взлетая, летел бы,
Когда большие обиды встречались на пути твоём.

Ржут кони в белом тумане,
Мой конь-тулпар — иноходец ждёт там.
В туман углубляясь ищу, и теряюсь,
Остаюсь один, когда туман рассеется снова.
```
