# Оригинал

### Билгесез / Илгиз Закиров - Асылъяр (өченче вариант)

```
Таң туган чагында
Ашыгам яныңа,
Хыялымда шулай ашыгам.
Киртәләр юк миңа
Баралам мин сиңа,
Иссәләр дә җилләр каршыдан.

Уянсын яшь күңел,
Йоклар вакыт түгел -
Саф мәхәббәт өчен бу таңнар.
Күңел иркен кырдай,
Хисләр язгы җырдай,
Әйт, нишләттең мине, Асылъяр?
```

# Перевод

### Неизвестен / Ильгиз Закиров - Асылъяр (третий вариант)

```
На рассвете дня
Спешу к тебе я,
В мечтах своих так к тебе спешу.
Нет преград мне,
Иду к тебе я,
Пусть даже ветры дуют навстречу.

Пусть проснётся молодое сердце,
Не время для сна -
Эти зори для чистой любви.
Душа вольна, как степь,
Чувства, словно весенняя песня,
Скажи, что сделала со мной ты, Асылъяр?
```
