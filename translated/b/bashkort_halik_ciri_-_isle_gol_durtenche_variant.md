# Оригинал

### Башкорт халык җыры - Исле гөл (дүртенче вариант)

```
Исле гөл исле була,*
Яран гөл төсле була.
Сагынганда айга багам,
Гел синең төсле була.

Тәрәзәм төбе исле гөл,
Исерәмен исенә.
Кайларда гына булсам да,
Син бит минем исемдә.

Исле гөлем матур була
Чәчәкләр атҡан чакта.
Шул чәчәкләргә карыймын
Сине сагынган чакта.
```

# Перевод

### Башкирская народная песня - Ароматный цветок (четвертый вариант)

```
Ароматный цветок ароматен,*
Любимый цветок цветной.
Когда тоскую по луне, смотрю,
Всегда твоего цвета.

На дне моего окна ароматный цветок,
Помню его аромат.
Где бы я ни был,
Ты в моих мыслях.

Мой ароматный цветок красив
Когда цветы расцветают.
На эти цветы смотрю
Когда скучаю по тебе.
```
