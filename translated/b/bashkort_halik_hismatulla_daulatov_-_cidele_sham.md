# Оригинал

### Башкорт халык / Хисмәтулла Дәүләтов - Җиделе шәм

```
Җиделе лә шәм якты яна
Эчендә мае булса.
Җиде уянып, җиде торып,
Җиде төн уртасында.

Җиделе шәм, мамык милтә,
Ватылды, җиргә төште.
Уф, йөрәгем, янам-көям,
Иркәм исемә төште.

Җиделе шәм, мамык милтә,
Яна да күмерләнә.
Алсу алмам бәгырекәем (Оныта алмаем бәгырькәем)
Онытмам гомергә дә (Оныта алмам гомергә дә).
```

# Перевод

### Башкирский народ / Хисматулла Даулетов - Семисвечье

```
Семисвечье ярко горит,
Если внутри есть воск.
Семь просыпается, семь встает,
В полночь семи ночей.

Семисвечье, свеча моя,
Разбилось, упало на землю.
Ох, сердце мое, я горю-томлюсь,
Пал на имя твое.

Семисвечье, свеча моя,
Горит, но тлеет.
Не отдам тебе сердца (Забыть не могу сердца)
Не забуду в жизни (Не отдам в жизни).
```
