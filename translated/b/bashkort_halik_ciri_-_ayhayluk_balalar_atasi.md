# Оригинал

### Башкорт халык җыры - Әйһәйлүк (Балалар атасы)

```
Иртәнчәккәй торып, тышка чыксам,
Әйһәйлүк,
Бер кыз килә сыер да, ай һәүкәләп,
һәпем, һәпәүләкәем.
Бер аз гына торып сөйләшкәнчә,
Әйһәйлүк,
Карт анасы йөри шул, ай һәүкәләп,
һәпем, һәпәүләкәем.
Сызылып кына чыккан, мыегымны,
Әйһәйлүк,
Кызлар сорый чәчмәүдә, әй, чугына,
һәпем, һәпәүләкәем.
Сызылып кына чыккан, мыегымны,
Әйһәйлүк,
Вәгъдә иттем сөйгән дә кызыма,
һәпем, һәпәүләкәйем.
```

# Перевод

### Башкирская народная песня - Айхайлюк (Отец детей)

```
Встав утром пораньше, выйду наружу,
Айхайлюк,
Девушка идет вдоль реки, месяц сопровождает,
хэпем, хэпэвлякэем.
Поговорив немного,
Айхайлюк,
Старая мать идет за ней, месяц сопровождает,
хэпем, хэпэвлякэем.
Вышла она, изогнувшись, моя тонкая,
Айхайлюк,
Девушки просят в косах, ай, сокол,
хэпем, хэпэвлякэем.
Вышла она, изогнувшись, моя тонкая,
Айхайлюк,
Обещал я любимой своей девушке,
хэпем, хэпэвлякэйем.
```
