# Оригинал

### Билгесез - Эзләдем табалмадым

```
Күзләреңне, күзләреңне,
Күзләреңне күзләдең.
Синең зәңгәр күзләреңнән
Мин өметләр эзләдем.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.

Күзләремнән, күзләремне,
Бер минут алмас идем.
Язмыш кая ташласа да
Мин синнән калмас идем.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.

Сөя микән, сөйми микән,
Белсәң икән серләрен.
Белки аның үзенеңдә
Өзеләдер үзәге.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.
```

# Перевод

### Бильгесез - Я поисках, не нашёл

```
Твои глаза, твои глаза,
Твои глаза искал.
Из твоих небесных глаз
Я искал надежды.

Я искал издалека,
Ты искала издалека.
Где ты, моя любовь, ты,
Не нашёл я.

Из моих глаз, моих глаз,
Я не отдавал ни на минуту.
Пусть даже судьба бросит,
Я не отстану от тебя.

Я искал издалека,
Ты искала издалека.
Где ты, моя любовь, ты,
Не нашёл я.

Любит или не любит,
Знай же, тайны его.
Может быть, у него самого
Сердце разорвётся.

Я искал издалека,
Ты искала издалека.
Где ты, моя любовь, ты,
Не нашёл я.
```
