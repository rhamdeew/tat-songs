# Оригинал

### Билгесез - Яңа ел (бишенче вариант)

```
Якынлаша Яңа ел,
Дулкынлана йөрәкләр.
Озатып кала иске ел,
Теләп изге теләкләр.
Килгән дуслар, туганнар,
Күмәкләшеп тулганнар.

Гомерләр барда, атлыйбыз алга.
Утырма син бер дә моңаеп.
Сөенеч-шатлык, мәхәббәт, саулык
Алып килсен безгә Яңа ел!

Барлык дөнья шатлана,
Балкып яналар утлар.
Яңа елдан башлана
Тагын яңа шатлыклар.
Кыш бабайдан - бүләкләр,
Кушып изге теләкләр.

Гомерләр барда, атлыйбыз алга.
Утырма син бер дә моңаеп.
Сөенеч-шатлык, мәхәббәт, саулык
Алып килсен безгә Яңа ел!
```

# Перевод

### Неизвестен - Новый год (пятый вариант)

```
Приближается Новый год,
Волнуются сердца.
Провожая старый год,
Загадывая светлые желания.
Пришли друзья, родные,
Собрались помогая друг другу.

Пока есть жизнь, шагаем вперёд.
Не сиди ты, печалясь.
Радость-веселье, любовь, здоровье
Пусть принесёт нам Новый год!

Весь мир ликует,
Сияя, пылают огни.
С Нового года начнутся
Вновь новые радости.
От Деда Мороза - подарки,
Добавляя светлые пожелания.

Пока есть жизнь, шагаем вперёд.
Не сиди ты, печалясь.
Радость-веселье, любовь, здоровье
Пусть принесёт нам Новый год!
```
