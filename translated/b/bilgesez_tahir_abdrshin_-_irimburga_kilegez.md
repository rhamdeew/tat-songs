# Оригинал

### Билгесез / Таһир Абдршин - Ырымбурга килегез!

```
Сезнең белән танышырга
Килдем мин ераклардан.
Җырлар белән, моңнар белән
Кайнар сәламнәр белән.

Килегез, килегез
Ырымбурны күрегез!
Килегез, килегез
Безгә кунакка!

Урал буйларыннан мин килдем.
Сакмар суларын кичтем.
Шул якларда туып үстем,
Шул якларның суын эчтем.

Килегез, килегез
Ырымбурны күрегез!
Килегез, килегез
Безгә кунакка!

Хушлашырга вакыт җитте,
Дусларым, кардәшләрем.
Бергә булган минутларны
Онытмам мин һәр көн саен.

Килегез, йөрегез
Җәлил йөрегән әзләрдән.
Килегез, дусларым
Безгә кунакка!
```

# Перевод

### Неизвестен / Тахир Абдршин - Приезжайте в Оренбург!

```
Чтобы с вами познакомиться,
Приехал я издалека.
С песнями, с напевами,
С горячими приветами.

Приезжайте, приезжайте,
Оренбург посмотрите!
Приезжайте, приезжайте
К нам в гости!

С уральских берегов я приехал,
Сакмарские воды переплыл.
В этих краях родился и вырос,
Воду этих мест я пил.

Приезжайте, приезжайте,
Оренбург посмотрите!
Приезжайте, приезжайте
К нам в гости!

Прощаться время пришло,
Друзья мои, братья мои.
Минуты, проведённые вместе,
Не забуду я каждый день.

Приезжайте, приходите
По следам, где Джалиль ходил.
Приезжайте, друзья мои,
К нам в гости!
```
