# Оригинал

### Билгесез - Полюбил күңел сине

```
Полюбил күңел сине,
Пожалей әзрәк мине.
Почему хат язмыйсың,
Наверно, яратмыйсың.
Сөймәсәң дә напиши,
Күңелемне утеши.

Бирдем сердцаны сиңа,
Булсын вечно для тебя.
Кил син вместе булырбыз,
Счастьелы без торырбыз.
Сөймәсәң дә напиши,
Күңелемне утеши.

Баш авырта, яна йөрәк,
Невозможно тузәргә.
Если ты меня не любишь,
То придется үләргә.
Сөймәсәң дә напиши,
Күңелемне утеши.

Полюбил күңел сине,
Пожалей әзрәк мине.
Почему хат язмыйсың,
Наверно, яратмыйсың.
Сөймәсәң дә напиши,
Күңелемне утеши.
```

# Перевод

### Билгесез - Полюбил күңел сине

```
Полюбил күңел сине,
Пожалей немного меня.
Почему же письмо не шлешь,
Наверно, не любишь ты.
Если молчишь - так напиши,
Утешь моё сердце.

Отдал тебе частичку души,
Пусть будет вечно для тебя.
Приди, будем вместе,
Счастливыми будем.
Если молчишь - так напиши,
Утешь моё сердце.

Голова болит, сердце горит,
Невозможно терпеть.
Если ты меня не любишь,
То придется умереть.
Если молчишь - так напиши,
Утешь моё сердце.

Полюбил күңел сине,
Пожалей немного меня.
Почему же письмо не шлешь,
Наверно, не любишь ты.
Если молчишь - так напиши,
Утешь моё сердце.
```
