# Оригинал

### Билгесез - Бәхет сатылмый ул

```
Һәркемгә бирелә бәхет,
Һәркемнең үз өлеше.
Булмый беркемне гаепләп,
Өлешеңә - көмешең.

Бәхет сатылмый бер җирдә,
Бәхетне Ходай бирә.
Тугач та аны фәрештәләр
Җилкәңә кит(е)реп элә.

Бик тә авыр яшәүләре,
Бәхетнең күрке үтсә.
Бәхет гөлең урынына
Шаулап кычыткан үссә.

Бәхет сатылмый бер җирдә,
Бәхетне Ходай бирә.
Тугач та аны фәрештәләр
Җилкәңә кит(е)реп элә.

Булмый ул бер кире килсә,
Булмый, бәхет булмаса.
Карыйм дигән агачыңа
Бәхет кошы кунмаса.

Бәхет сатылмый бер җирдә,
Бәхетне Ходай бирә.
Тугач та аны фәрештәләр
Җилкәңә кит(е)реп элә.
```

# Перевод

### Билгесез - Счастье не продается

```
Счастье дается каждому,
У каждого свой путь.
Не стоит никого винить,
Судьба своя придет.

Счастье не продается нигде,
Счастье Господь дает.
Даже после смерти тебя
Ангелы на плечи возьмут.

Если жизнь тяжелой была,
Когда счастья не видно.
Вместо сада цветов твоих
Радостно птички поют.

Счастье не продается нигде,
Счастье Господь дает.
Даже после смерти тебя
Ангелы на плечи возьмут.

Если не вернется оно,
Не будет счастья, увы.
Если на дерево твое
Птица счастья не сядет.

Счастье не продается нигде,
Счастье Господь дает.
Даже после смерти тебя
Ангелы на плечи возьмут.
```
