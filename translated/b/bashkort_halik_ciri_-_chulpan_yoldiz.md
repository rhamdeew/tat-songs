# Оригинал

### Башкорт халык җыры - Чулпан йолдыз

```
Сызылып аткан таңнар чакта,
Су өсте дулкынлана.
Сине сагынып яшь йөрәгем шул
Су кебек дулкынлана.

Ал таң ата, чулпан калка,
Чулпан - таңның көзгече.
Чулпан - ал таңның көзгече
Минем сагыну билгесе.

Ал таң ата, чулпан калка,
Чулпан таңны уята.
Чулпан ал таңны уята,
Миңа, сине уйлата.
```

# Перевод

### Башкир народная песня - Звезда Венера

```
Во времена восходящих утр,
Волны рябью на воде.
Мое юное сердце плачет о тебе,
Так же как волна на воде.

Прошлое утро уходит, Венера остается,
Венера - зеркало утра.
Венера - зеркало прошлого утра,
Она моя тоски о тебе.

Прошлое утро уходит, Венера остается,
Венера будит утро.
Венера будит прошлое утро,
Навевает мысли о тебе.
```
