# Оригинал

### Башкорт халык җыры - Ямьле Агыйдел буйлары (өченче вариант)

```
Агыйделкәй алкын, суы салкын
Ак үрдәкләр каршыда йөзә алмый.

Ямьле Агыйдел буйлары шул
Бетерәдер хәсрәтле уйларны.

Әчем генә, тышым, әй ут яна,
Бер җырламый җаным да түзә алмый.

Ямьле Агыйдел буйлары шул
Бетерәдер хәсрәтле уйларны.
```

# Перевод

### Башкирская народная песня - Берега Ямель-Агидель (третий вариант)

```
Агиделька берег, вода прохладна,
Белые утята не могут плыть напротив.

Таковы берега Ямель-Агидель,
Они избавят от тоскливых мыслей.

Сердце только ноет, тело горит,
Душа не выносит, не может петь.

Таковы берега Ямель-Агидель,
Они избавят от тоскливых мыслей.
```
