# Оригинал

### Билгесез - Бәхет гөлем

```
Бәхет гөлем синең өчен генә
Үсә минем тәрәз төбендә.
Син янымда булган чакта гына
Йөрәгемнән моңнар түгелә.

Бәхет гөлем күңелем түрләрендә
Чәчәк атты язлар килгәндә.
Гомерем буе көттем сине генә,
Хисләремә сагыш тидермә.

Бәхет гөлем үрли күңелемдә,
Син булганда минем янымда.
Үпкәләрем калсын үткәннәрдә,
Син генә тик кирәк, тик җаныма.

Бәхет гөлем бул янымда гына,
Назлы карашыңны яшермә.
Сөюләрем бары сина гына,
Бәхетебез булсын гомергә.
```

# Перевод

### Билгесез - Цветок счастья

```
Цветок счастья твой
Растёт у меня под окном.
Только когда ты со мной
Сердце моё не страдает.

Цветок счастья в глубине души моей
Расцветает, когда приходят весны.
Всю жизнь свою ждал только тебя,
Не позволяй тоске затронуть чувства.

Цветок счастья растёт в моём сердце,
Когда ты рядом со мной.
Пусть воспоминания остаются в прошлом,
Ты нужна лишь мне, лишь душе моей.

Цветок счастья будь рядом со мной,
Не скрывай нежный взгляд свой.
Все поцелуи только для тебя,
Пусть будет счастье наше на всю жизнь.
```
