# Оригинал

### Билгесез - Без ике фәрештә

```
Хыяллар илендә,
Мәхаббәт җирендә,
Без биербез икәү
Бәхетнең түрендә.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.

Тотышып куллардан,
Утлардан, сулардан,
Бергәләп үтәргә
Язсын бу юллардан.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.

Катырак очарга,
Күкләргә очарга,
Телим синең белән
Дөньядан качарга.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.
```

# Перевод

### Бильгесез - Мы двое ангелов

```
В стране грез,
На земле любви,
Вдвоём мы будем
На склоне счастья.

Только ты, только я,
Только я, только ты.
В этом светлом мире
Мы двое ангелов.

Держась за руки,
Сквозь огонь, сквозь воду,
Пройти бы вместе
Этим путём.

Только ты, только я,
Только я, только ты.
В этом светлом мире
Мы двое ангелов.

Лететь выше,
В небеса лететь,
Хочу с тобой
Скрыться от мира.

Только ты, только я,
Только я, только ты.
В этом светлом мире
Мы двое ангелов.
```
