# Оригинал

### Билгесез / Юлай Моратов - Күгәрчен җыры

```
Гөлдер, гөлдер,
Хәзер язгы көндер.
Бирешмәдек сыуыкка,
Кыш үткәрдек кыекта.

Гөлдер, гөлдер,
Яз, безне сөендер.
Тышта гөрләп йөрербез,
Рәхәт үтәр көнебез.

Гөлдер, гөлдер,
Гөлләр үсәр көндер.
Һәр чак күңелле булсын,
Иртәләр тыныч тусын.
```

# Перевод

### Неизвестен / Юлай Моратов - Голубиная песня

```
Цветы, цветы,
Весенние теперь деньки.
Не поддались мы холодам,
Зиму провели в краю родном.

Цветы, цветы,
Весна, ты нас обрадуй вновь.
Мы будем петь под небесами,
И счастьем наполнятся наши дни.

Цветы, цветы,
Цветы растут повсюду.
Пусть радость не покинет нас,
И утро встретит тишиной.
```
