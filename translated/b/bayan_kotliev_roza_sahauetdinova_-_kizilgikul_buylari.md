# Оригинал

### Баян Котлыев / Роза Сәхәүетдинова - Кызылгыкүл буйлары

```
Кызылгыкүл сулары,
Яшел тугай буйлары.
Кошлар моңы бөдрә таллар да.

Ташкын сулар аккач,
Балан чәчәк аткач.
Күлгә киләм йөрәк талган да.

Кызылгыкүл язлары,
Табигатьнең назлары.
Күлдә казлар язга куана.

Төшләремдә күрдем,
Син аккошы күлнең.
Сабыр йөрәк көтә, юана.

Кызылгыкүл буйларын,
Күп тирәкле юлларын.
Бергә үтәр идек тагын да.

Бик сагындым иркәм,
Ерак юлга китәм.
Юксынырмын ялгыз чагымда.
```

# Перевод

### Баян Котлиев / Роза Сахаутдинова - Берега Кызылгика

```
Воды Кызылгика,
Зелёные берега.
Песня птиц и ивы над рекой.

Когда весенние воды текут,
Когда цветы балы расцветают.
Я приду к озеру, уставшее сердце на плече.

Лета Кызылгика,
Ласки природы.
Гуси на озере радуются лету.

Во сне я видел,
Ты белая птица озера.
Терпеливое сердце ждёт и скучает.

По берегам Кызылгика,
По тернистым дорогам.
Вместе пройдём ещё раз.

Очень соскучился, родной,
Ухожу в далёкий путь.
Не засну в одиночестве.
```
