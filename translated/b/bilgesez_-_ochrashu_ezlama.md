# Оригинал

### Билгесез - Очрашу эзләмә

```
Еллар аркылы
Килдең кичегеп,
Үткән сөюне
Искә төшереп.
Кабат очрашу
Кирәкме икән?
Олы хисләрне,
Уятып, иркәм?!

Булып кал бары
Яшьлек ялгышым,
Күңел түремдә -
Назым, сагышым
Эзләмә мине,
Хәзер без үткән.
Гомерләр үткән,
Соң инде, иркәм...

Әйләнеп кайтмас,
Яшьлек тә - үткән.
Тәүге сөюләр...
Беренче үпкән...
Тормыш йомгагын
Инде без сүткән.
Язмышта безне
Башкалар көткән...

Булып кал бары
Яшьлек ялгышым,
Күңел түремдә -
Назым, сагышым
Эзләмә мине,
Хәзер без үткән.
Гомерләр үткән,
Соң инде, иркәм...
```

# Перевод

### Бильгесез - Не искәтеп очрашу

```
Через года
Ты пришла в полночь,
Вспоминая
Прошлые поцелуи.
Нужна ли встреча
Снова?
Разве пробуждать
Сильные чувства?!

Останься просто
Моя молодая ошибка,
В моём сердце -
Моё сожаление, тоска
Не ищи меня,
Сейчас мы прошли.
Жизни прошли,
Теперь уже, ой, как больно...

Не возвращайся,
Молодость тоже прошла.
Первые поцелуи...
Первая вина...
Итоги жизни
Мы уже разрушили.
Другие ждут нас
В судьбе...

Останься просто
Моя молодая ошибка,
В моём сердце -
Моё сожаление, тоска
Не ищи меня,
Сейчас мы прошли.
Жизни прошли,
Теперь уже, ой, как больно...
```
