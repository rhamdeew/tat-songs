# Оригинал

### Башкорт халык җыры - Ялчыгол (икенче вариант)

```
Ялчы, ялчы, Ялчыгол,
Йәлсемәгән Ялчыгол.
Байда елкы көткәнмен,
Тайга корык тотканмын.

Ялсы, ялсы, Ялсыгол,
Йәлсәмәгән Ялсыгол.
Байда көтү көттем миндә,
Кара тирен түктем мин.

Аран байда елкы көтеп
Күпме юллар тападым.
Күпме юллар тападым да,
Әммә бәхет тапмадым.

Байлар сары бал эчкәндә,
Идел суын эчтем мин.
Идел суын эчтем миндә,
Батыр булып үстем мин.
```

# Перевод

### Башкирская народная песня - Ялчыгол (второй вариант)

```
Ялчы, ялчы, Ялчыгол,
Неугомонный Ялчыгол.
В доме бая ждал ялкость,
В тайге искал покой.

Ялсы, ялсы, Ялсыгол,
Неугомонный Ялсыгол.
В доме бая ждал встречи,
Скинул чёрную кожу.

Где-то в доме бая, жаждой
Много дорог искал я.
Много дорог искал я,
Но счастья не нашёл.

Пока баяры мёд пили,
Я пил воду Идели.
Пил воду Идели я,
Стал богатырём сильным.
```
