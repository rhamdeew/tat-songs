# Оригинал

### Билгесез - Бөктермә

```
Исләрем китеп йөримен,
Нишләде бу сөйгәнем?
Кыскартам да бетерәм, -
Ди яңа матур күлмәген.

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр - бөктерсеннәр,
Ә син менә бөктермә.

Нигә соң бозырга, җаным?
Килешә бит күлмәгең.
Миңа ошар дисеңме әллә
Бөктереп йөрүләрең?

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр - бөктерсеннәр,
Ә син менә бөктермә.

Әллә нишләп куя күңел,
Күз алдыма китереп.
Синнән башка да җитәрлек
Йөрүчеләр бөктереп.

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр - бөктерсеннәр,
Ә син менә бөктермә.
```

# Перевод

### Билгесез - Бёктермя

```
Я ислями уйгып йөрем,
Ничек бу сёгнём? 
Кыскартам да бетерәм, -
Бу яңа матур кылым.

Сене дә, мине дә син
Адам бозып ташлама.
Бёктергәннәр - бёктерсеннәр,
Ә сен бёктереп ташлама.

Ничек бозырга, җаным?
Йөрегә килешә торган.
Миңа ошар дисеңме әллә
Бёктереп йөрүләрең?

Сене дә, мине дә син
Адам бозып ташлама.
Бёктергәннәр - бёктерсеннәр,
Ә сен бёктереп ташлама.

Әллә ничек куя күңел,
Күз алдыма китереп.
Сеннән башка да җитәрлек
Йөрүчеләр бёктерелеп.

Сене дә, мине дә син
Адам бозып ташлама.
Бёктергәннәр - бёктерсеннәр,
Ә сен бёктереп ташлама.
```
