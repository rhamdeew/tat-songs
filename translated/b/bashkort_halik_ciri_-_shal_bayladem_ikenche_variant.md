# Оригинал

### Башкорт халык җыры - Шәл бәйләдем (икенче вариант)

```
Шәл бәйләдем, шәл бәйләдем,
Шәл бәйләдем, шәл бәйләдем,
Шәлем түгәрәк түгел дә
Шәлем түгәрәк түгел.

Әннә-гидер, әннә-гидер генәем лә
Әннә-гидер, әннә-гидер генәем.

Очып барып сөйләшергә,
Очып барып сөйләшергә,
Җаным күбәләк түгел дә
Җаным күбәләк түгел.

Кара бәрән тиресе дә
Кара бәрән тиресе дә,
Кама булыр тисеңме дә
Кама булыр тисеңме?

Сикереп төшеп биегәннән,
Сикереп төшеп биегәннән
Язык булыр тисеңме лә
Язык булыр тисеңме?

Әннә-гидер, әннә-гидер генәем лә
Әннә-гидер, әннә-гидер генәем.
```
