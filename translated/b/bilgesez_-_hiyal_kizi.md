# Оригинал

### Билгесез - Хыял кызы

```
Төн, төшләремдә бер кыз килә.
Кыз кай яктан син? Исемең кем?
Бер кердең төшемә...
Бутадың, күңелне туздырдың,
Сулышым - син генә...
Хыял кызы...

Телисеңме килер өчен яныңа,
Күңел кылы белән бәйлим һәр йолдызны.
Ай нурына күпер салам сиңа дип.
Хыял кызы...
Синең өчен диеп йөрәк хыялга,
Язган хатларымны саклар күңел тузы.
Тик минеке булып кал син хыялда...
Хыял кызы...

Бер син бердәнбер,
Төшемә кер,
Хәй, хыял булып мине җылыт...
Бер кердең төшемә...
Бутадың, күңелне туздырдың,
Сулышым - син генә...
Хыял кызы...

Телисеңме килер өчен яныңа,
Күңел кылы белән бәйлим һәр йолдызны.
Ай нурына күпер салам сиңа дип.
Хыял кызы...
Синең өчен диеп йөрәк хыялга,
Язган хатларымны саклар күңел тузы.
Тик минеке булып кал син хыялда...
Хыял кызы...
```
