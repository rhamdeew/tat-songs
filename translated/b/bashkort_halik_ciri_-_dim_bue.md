# Оригинал

### Башкорт халык җыры - Дим буе

```
Димдә тудым, Димдә үстем,
Җиләк җыйдым тугайда,

Әй, матур Дим буе,
Бик сагындым бу юлы.

Күп хәсрәтләр күрдем мин,
Сер бирмәдем шулай да.

Әй, матур Дим буе,
Бик сагындым бу юлы.

Дим суларын иңдәп-буйлап
Йөзәрсен, үрдәккәем.

Ой, матур Дим буе,
Бик сагындым бу юлы.

Бик күп кайгыларга түздең,
Түзәрсең, йөрәккәем.

Әй, матур Дим буе,
Бик сагындым бу юлы.

Былбыл да бар, кондыз да бар
Димнең таллы ягында.

Әй, матур Дим буе,
Бик сагындым бу юлы.

Хәсрәт тә бар, кайгы да бар
Читтә йөргән чагында.

Әй, матур Дим буе,
Бик сагындым бу юлы.
```

# Перевод

### Башкирская народная песня - Дим буе

```
На Диме родился, на Диме вырос,
Собирал землянику в лесу,

Ах, красивый берег Дима,
Так соскучился по этой дороге.

Много горя видел я,
Но секретов не выдал.

Ах, красивый берег Дима,
Так соскучился по этой дороге.

По водам Дима, вдоль берегов
Плыл ты, мой утёнок.

Ой, красивый берег Дима,
Так соскучился по этой дороге.

Ты выносишь много печалей,
Выносишь, сердце моё.

Ах, красивый берег Дима,
Так соскучился по этой дороге.

Соловей поёт, кукушка кукует
На берегах Дима.

Ах, красивый берег Дима,
Так соскучился по этой дороге.

Горе есть, печаль есть
О тех, кто вдали.

Ах, красивый берег Дима,
Так соскучился по этой дороге.
```
