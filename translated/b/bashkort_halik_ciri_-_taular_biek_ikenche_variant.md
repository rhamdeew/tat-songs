# Оригинал

### Башкорт халык җыры - Таулар биек (икенче вариант)

```
Таулар биек, таулар биек,
Таулар биек менәргә.
Бачкычлар юк, бачкычлар юк,
Шул тауларга менәргә.

Болындагы чәчәкләрне
Тапамам дигән идем.
Туып үскән авылымны
Ташламам дигән идем.

Болындагы чәчәкләрне
Тапарга туры килде.
Туып үскән авылымны
Ташларга туры килде.
```

# Перевод

### Башкирская народная песня - Высокие горы (второй вариант)

```
Горы высоки, горы высоки,
Горы высоки, взойти.
Лестниц нет, лестниц нет,
На те горы взойти.

Цветы в небесах
Якобы не нашёл.
Село, где вырос
Якобы не оставил.

Цветы в небесах
Пришлось искать.
Село, где вырос
Пришлось оставить.
```
