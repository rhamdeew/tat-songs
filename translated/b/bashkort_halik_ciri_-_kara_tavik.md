# Оригинал

### Башкорт халык җыры - Кара тавык

```
Дустым чакырды кунакка,
Тавык бирде бүләккә,

Сайлап алдым охшаганын
Үзем менеп төнәккә.

Кара тавык бүләк диеп,
Кешеләргә мактандым.

Әтәч булып кычкырган иде -
Керер урын тапмадым.
```

# Перевод

### Башкирская народная песня - Черный петух

```
Друг мой позвал в гости,
Подарил курицу,

Выбрал я подходящую
И поджег самолично.

Черной курицей-подарком
Похвалялся перед людьми.

Петухом стал кричать -
Так и не нашел покоя.
```
