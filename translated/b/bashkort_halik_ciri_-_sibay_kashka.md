# Оригинал

### Башкорт халык җыры - Сыбай кашка

```
Сыбай да гына кашка, басма ташка,
Юлсыз гына җирдән дә үткәндә.
Бу дөнъяда авыр ялгыз башка,
Дус-иш кирәк дөнъя да көткәндә.

Сыбай гына кашка кырач ялын
Таудан гына өргән дә җил тарай.
Ерак җиргә китсәк, сыбай кашкам
Башкынаен борып та бер карый.

Ерактан ук агарып, ай, күренгән
Урал таукаеның да ак ташы.
Кайда гына булмый, нидәр күрми
Ир башкае белән дә ат башы.
```

# Перевод

### Башкирская народная песня - Сыбай кашка

```
Сыбай скала, неприступный камень,
Сквозь бездорожье, пустынные земли.
В этом мире тяжело одному,
Во всём мире нужен друг и труд.

Сыбай скала, крутой берег,
С горы веют ветра.
В чужие края уйдём - Сыбай скала
Повернётся, взглянет вслед.

Со стороны, ещё светлая луна
Белый камень Уральских гор.
Где бы ни было, что-то да видно
С мужской головой и лошадиным.
```
