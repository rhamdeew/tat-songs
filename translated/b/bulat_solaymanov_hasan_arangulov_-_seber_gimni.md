# Оригинал

### Булат Сөләйманов / Хәсән Арангулов - Себер гимны

```
Син тугансың
Минем йөрәгемдә,
Мин туганчы
Газиз анамнан.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.

Җаным кебек
Бик кадерле миңа
Туган җирнең
Изге туфрагы.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.

Бу дөньяда
Себер агачының
Мин серле
Моңлы яфрагы.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.
```

# Перевод

### Булат Сөләйманов / Хасан Арангулов - Гимн Сибири

```
Ты родилась
В моём сердце,
Прежде чем я родился
От любимой матери.

Сибирь, Сибирь,
Только в тебе одной на земле
Моя любовь
Обрела себя.

Как душа моя
Дорога мне
Родной земли
Священная почва.

Сибирь, Сибирь,
Только в тебе одной на земле
Моя любовь
Обрела себя.

В этом мире
Сибирского дерева
Я таинственный
Печальный листок.

Сибирь, Сибирь,
Только в тебе одной на земле
Моя любовь
Обрела себя.
```
