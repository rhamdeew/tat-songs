# Оригинал

### Билгесез - Авылым юлы

```
Авылымны бик сагындым,
Атларга чана җиктем.
Бер көн дә каласы килми,
Кайтырга юлга чыктым.

Арма инде, талма инде,
Әйдә инде, малкаем.
Көтә кебек мине анда
Яшьли сөйгән җанкаем.

Җилле буран чыга тора,
Армас микән атларым?
Юлны карлар күмеп китсә,
Адашмасмы атларым?

Ап-ак кар өстендә кала
Атларның тояк эзе.
Авыл юлында калды бит
Яшьлекнең бетмәс эзе.
```

# Перевод

### Билгесез - Дорога в мою деревню

```
Сильно соскучился по своей деревне,
Погрузил сани на лошадей.
Даже один день не живу в городе,
Выхожу на дорогу, чтобы вернуться.

Уже не то, уже не се,
Уже не то, моя родная.
Ждёт меня там,
Моя любимая душа.

Сильная метель поднимается,
Не остановятся ли лошади?
Если дорогу засыплет снегом,
Не заблудятся ли лошади?

На белом снегу остаются
Следы копыт лошадей.
На деревенской дороге остались ведь
Следы молодости, что не исчезнут.
```
