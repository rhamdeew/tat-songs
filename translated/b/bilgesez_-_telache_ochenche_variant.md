# Оригинал

### Билгесез - Теләче (өченче вариант)

```
Минем ничек яратканны син үзең дә беләсең,
Яратышып яшик әле, яшик әле, Теләчем.

Теләкләрнең изгеләрен телисеңдер Теләче,
Безгә һәрчак бәхет кенә телә-телә, Теләчем.

Килә сине шатлыкларга, шатлыкларга күмәсе,
Шатланышып яшик әле, яшик әле, Теләчем.

Теләкләрнең изгеләрен телисеңдер Теләче,
Безгә һәрчак бәхет кенә телә-телә, Теләчем.

Кайчагында сагындырып бәгырьләрне теләсең,
Сагынышып яшик әле, яшик әле, Теләчем.

Теләкләрнең изгеләрен телисеңдер Теләче,
Безгә һәрчак бәхет кенә телә-телә, Теләчем.
```

# Перевод

### Неизвестен - Желающий (третий вариант)

```
Как сильно я любил, ты и сама ведь знаешь,
Давай любя проживем еще, проживем, Желающий мой.

Ты желаешь самых светлых пожеланий, Желающий,
Нам всегда лишь счастья желай-желай, Желающий мой.

Хочется окунуться в радости, в радости,
Давай радуясь проживем еще, проживем, Желающий мой.

Ты желаешь самых светлых пожеланий, Желающий,
Нам всегда лишь счастья желай-желай, Желающий мой.

Иногда заставляешь сердца тосковать,
Давай тоскуя проживем еще, проживем, Желающий мой.

Ты желаешь самых светлых пожеланий, Желающий,
Нам всегда лишь счастья желай-желай, Желающий мой.
```
