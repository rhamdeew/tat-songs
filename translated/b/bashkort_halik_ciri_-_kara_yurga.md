# Оригинал

### Башкорт халык җыры - Кара юрга

```
Йомры матур тоягыңны
Дагаларбыз, юргам.
Кырач матур ялларыңны
Укаларбыз, юргам.
Өстеңдәге сылыуыңны
Төшерәле, юргам
Һайт, кара юргам,
Тайт, кара юргам.

Башыңдагы йөгәнеңне
Көмешләрбез, юргам,
Кара-кырпу үңдегеңне
Нагышларбыз, юргам.
Өстеңдәге сылыуыңны
Төшерәле, юргам.
Һайт, кара юргам,
Тайт, кара юргам.
```

# Перевод

### Башкирская народная песня - Черный журавль

```
Твой красивый, пушистый хвост
Мы украсим, журавль.
Твои красивые крылья
Мы застегнем, журавль.
Твою росу на спине
Сбросим вниз, журавль.
Пой, черный журавль,
Пляши, черный журавль.

Твой груз на голове
Серебрим, журавль,
Твой черно-серый цвет
Вышиваем, журавль.
Твою росу на спине
Сбросим вниз, журавль.
Пой, черный журавль,
Пляши, черный журавль.
```
