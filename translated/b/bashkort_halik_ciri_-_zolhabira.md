# Оригинал

### Башкорт халык җыры - Зөлхәбирә

```
Зөлхәбирә китап укый,
Кыя шәлен ябып башына*.
Зөлхәбирә үксеп елый,
Хур булдым дип кеше каршында

Мустафа мулла бай икән,
Ун ике дә кәртә тоттырган.
Бай булса да, акылы сай икән,
Кәрт уйнатып, кызын оттырган.
```

# Перевод

### Башкирская народная песня - Зульхабира

```
Зульхабира читает книгу,
Надевает чепчик на голову.
Зульхабира взрослеет и плачет,
Позор, говорит перед людьми

Мустафа-мулла был богат,
Двенадцати лет держал за руку.
Богат, но разум был здрав,
Разогнал гадалку и отдал дочь замуж.
```
