# Оригинал

### Билгесез / Зөфәр Хәйретдинов - Алтын алма

```
Алманың алтын чагы,
Көзләрнең салкын чагы.
Алтын алмам, кил яныма,
Ачык кайнар кочагым.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.

Ачык кайнар кочагыма
Кер әйдә, җылытармын -
Көзнең ачы салкыннарын
Гомергә онытырсың.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.

Аңладым нигә алманың
Алтын ул дигәннәрен,
Алтын алмам, кайнар алмам
Нәкъ миңа тигәнгәдер.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.
```

# Перевод

### Неизвестен / Зуфар Хайретдинов - Золотое яблоко

```
Золотая пора яблок,
Холодная пора осенних дней.
Золотое яблоко моё, иди ко мне,
Раскрыты жаркие объятья мои.

Только в золотых яблоках
Жаркие месяцы лета.
Или только мне досталось
Самое горячее из яблок.

В мои раскрытые объятья
Войди же, я согрею -
Горкую осеннюю стужу
Навсегда ты позабудешь.

Только в золотых яблоках
Жаркие месяцы лета.
Или только мне досталось
Самое горячее из яблок.

Теперь я понял, почему яблоко
Золотым назвали люди,
Золотое яблоко, горячее яблоко
Словно мне одному предназначено судьбой.

Только в золотых яблоках
Жаркие месяцы лета.
Или только мне досталось
Самое горячее из яблок.
```
