# Оригинал

### Башкорт халык җыры - Аслаев

```
Ачучан да арыслан, ай, баласы,
Юаш түгел юлбарыс баласы.
Барыннан да түзем авырлыкка
Күпне күргән ата баласы.

Нарат та үсә, ай, ямь өчен,
Бал кортлары оча бал өчен.
Комсыз гына үләр, ай, мал өчен,
Ир-егеткәй үләр дан өчен.
```

# Перевод

### Башкирская народная песня - Аслаев

```
Даже лев свиреп, о, сын,
Не юнец леопард, сын.
Всех тяжелее терпеть испытания
Старый отец, видевший многое, сын.

Растет бамбук, о, ради блага,
Пчелы летят за медом.
Без похорон умрет, о, ради стада,
Мужчина умрет за честь.
```
