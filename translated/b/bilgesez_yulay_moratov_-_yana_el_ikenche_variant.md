# Оригинал

### Билгесез / Юлай Моратов - Яңа ел (икенче вариант)

```
Бар җиһанга бәхет теләп
Җирнең шары әйләнә.
Әткәй - әнкәй туганнарың,
Җан дусларың эргәдә.

Яңа ел, яңа җыр,
Барсыда шат булыр.
Тәннәргә, җаннарга,
Яңа ел тулыр.

Яңа елның, яңа җыры,
Мәңге истә калачак.
Сихри моңнарга күмелгән
Безнең алтын балачак.

Яңа ел, яңа җыр,
Хыяллар чында булыр
Бәхеткә, шатлыкка,
Яңа ел булыр.

Киләчәккә өметләр зур
Сүнмәс яшьлек дәртебез.
Томыш юлын дуслар белән
Бергә - бергә үтәрбез.

Яңа ел, яңа җыр,
Хыяллар чында булыр
Бәхеткә, шатлыкка,
Яңа ел булыр.
```

# Перевод

### Неизвестен / Юлай Моратов - Новый год (второй вариант)

```
Желая счастья всей Земле,
Вращается земной наш шар.
Отец и мать, твои родные,
Друзья душевные с тобой.

Новый год, новая песня,
Все будут счастливы сейчас.
Для тел и душ наших
Новый год наполнит нас.

Новогодняя песня
Навсегда в памяти останется.
Волшебными мелодиями окутано
Наше золотое детство.

Новый год, новая песня,
Мечты все сбудутся.
К счастью, к радости
Новый год приведёт нас.

Надежды на будущее велики,
Неугасима юности пора.
Дорогой жизни вместе с друзьями
Мы пройдём, рука об руку.

Новый год, новая песня,
Мечты все сбудутся.
К счастью, к радости
Новый год приведёт нас.
```
