# Оригинал

### Билгесез - Йөрәкләр мәңгегә бәйләнгән

```
Син генә, син генә, бары тик син генә.
Бәхетем бул син, бул гына.
Хыяллар чынлыкка әйләнгән вакытта
Янәшәмдә минем бул гына.

Сөенәм мин шуңа: кадерле син миңа,
Йөрәкләр мәңгегә бәйләнгән.
Сөюләр нур булып, күңелләргә тулып,
Мәхаббәт күкләрен әйләнгән.

Булыйкчы гел бергә, юллар юк кирегә,
Тормышларны җырлар иттик без.
Карамыйк үткәнгә, үтепләр киткәнгә,
Бик бәхетле гомер кичер(ер)без.

Сөюләр нур булып, күңелләргә тулып,
Мәхаббәт күкләрен әйләнгән.
Карамыйк үткәнгә, үтепләр киткәнгә,
Бик бәхетле гомер кичер(ер)без.
```

### Неизвестен - Сердца навеки связаны

```
Лишь ты, лишь ты, только ты одна.
Будь моим счастьем, только им.
Когда мечты становятся явью,
Будь рядом со мной, только рядом.

Я радуюсь тому, что ты мне дорога,
Сердца навеки связаны.
Любовь светом стала, души наполняя,
По небесам любви кружится.

Давай будем всегда вместе, не нужно других дорог,
Мы превратили наши жизни в песни.
Не будем смотреть в прошлое, на то, что ушло,
Мы проживём счастливую жизнь.
```
