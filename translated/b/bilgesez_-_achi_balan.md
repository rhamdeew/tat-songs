# Оригинал

### Билгесез - Ачы балан

```
Өй түрендәге баланны
Кисмә - ярамас, балам.
Язмышыңа кагылмасын
Балан ачысы, балам.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.

Озакламас карлар явар,
Җыя торчы баланын.
Язмыш юлы якты булсын,
Әй, Ходаем, баламның.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.

Балан ачы булса да ул,
Чәчәге матур аның.
Авыр чакта ярдәм сузган,
Онытма, Ходай барын.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.
```

# Перевод

### Билгесез - Горькое дитя

```
Не трогай, не секи
Дитя, что под кровом дома.
Пусть не коснётся тебя
Горькость дитя.

Дитя моё, горькое дитя,
Не ешь, не трать, не рви.
Вкус горького дитя
Останется в жизни как эхо.

Скоро пойдут снега,
Собирай, дитя, свои листья.
Пусть будет путь твой светлым,
О, Господи, дитя моё.

Дитя моё, горькое дитя,
Не ешь, не трать, не рви.
Вкус горького дитя
Останется в жизни как эхо.

Пусть и горькое дитя,
Цветок его красив.
В трудный час просит помощи,
Не забывай, есть Господь.

Дитя моё, горькое дитя,
Не ешь, не трать, не рви.
Вкус горького дитя
Останется в жизни как эхо.
```
