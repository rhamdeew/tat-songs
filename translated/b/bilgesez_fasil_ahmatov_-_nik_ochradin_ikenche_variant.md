# Оригинал

### Билгесез / Фасил Әхмәтов - Ник очрадың? (икенче вариант)

```
Ник очрадың минем юлымда син,
Нигә йөрәгемә ук аттың?
Югалырлык булгач, ник табылдың?
Мин югалттым сине, югалттым.

Матур җырың, татлы сүзең белән
Боек күңелемне юаттың.
Тагын киләм, дидең, ник килмәдең?
Мин югалттым сине, югалттым.
```

# Перевод

### Неизвестен / Фасиль Ахметов - Зачем встретилась? (второй вариант)

```
Зачем встретилась ты на пути моём,
Почему стрелой пронзила сердце?
Когда должна была пропасть, зачем нашлась?
Я потерял тебя, потерял.

Красивой песней, сладким словом
Ты успокоила мою печаль.
Сказала: вернусь, - так почему не пришла?
Я потерял тебя, потерял.
```
