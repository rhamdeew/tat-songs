# Оригинал

### Әнгәм Атнабаев / Флүрә Талипова - Әнкәемнең йомшак куллары

```
Синең куллар, мине сөя-сөя,
Бишекләргә салган куллар ул.
Соңгы телемнәрен миңа биреп,
Үзе коры калган куллар ул.

Әнкәй, синең йомшак кулларың.
Әнкәй, синең йомшак кулларың.

Синең куллар, керләремне юып,
Топасланып беткән куллар ул.
Утта пешеп, салкыннарда өшеп,
Сөялләнеп беткән куллар ул.

Әнкәй, синең йомшак кулларың.
Әнкәй, синең йомшак кулларың.

Унбиш яшьлек кызлар кулына да
Алыштырмам синең кулыңны.
Шул кулларың белән тагын бер кат
Иркәләче, әнкәй, улыңны (кызыңны).

Әнкәй, синең йомшак кулларың.
Әнкәй, синең йомшак кулларың.
```

# Перевод

### Ангам Атнабаев / Флюра Талипова - Нежные руки моей мамы

```
Твои руки, с любовью меня
В колыбель укладывали.
Последние крохи мне отдавая,
Сами огрубели.

Мама, твои нежные руки.
Мама, твои нежные руки.

Твои руки, мою одежду стирая,
До мозолей истёрлись.
В огне обжигались, на холоде стыли,
До костей износились.

Мама, твои нежные руки.
Мама, твои нежные руки.

Даже на руки пятнадцатилетних девушек
Не променяю твои руки.
Этими руками ещё раз
Приласкай, мама, своего сына (свою дочь).

Мама, твои нежные руки.
Мама, твои нежные руки.
```
