# Оригинал

### Алмаз Латыйпов - Су буенда учак яна

```
Тып-тын калган тонге авыл,
Йоклый авыл, сунгэн уты.
Кайттым сина еллар узгач,
Хэтер уйный, керми йокы.

Су буенда учак яна.
Кем кабызган ул учакны?
Тошсэм эгэр су буена,
Табармын кук балачакны.

Тыныч сулап йоклый авыл,
Йорим тонге урамында.
Таныш сукмак, таныш кое —
Утсыз да мин курэм монда.

Су буенда учак яна.
Кем кабызган ул учакны?
Тошсэм эгэр су буена,
Табармын кук балачакны.

Хэтерлэсэн, онытмасан,
Мин башканы омет итмим.
Ялантэпи урамында
Уйнап ускен балан бит мин.

Су буенда учак яна.
Кем кабызган ул учакны?
Тошсэм эгэр су буена,
Табармын кук балачакны.
```

# Перевод

### Алмаз Латыйпов - У воды костер горит

```
Притихла в ночи деревня,
Спит деревня, погасив огни.
Вернулся к тебе спустя годы,
Память играет, не идет сон.

У воды костер горит.
Кто зажег тот костер?
Если спущусь к воде,
Найду синее детство.

Тихо дыша спит деревня,
Хожу по ночной улице.
Знакомая тропинка, знакомый колодец —
Без света и здесь я вижу.

У воды костер горит.
Кто зажег тот костер?
Если спущусь к воде,
Найду синее детство.

Если помнишь, не забываешь,
Я другого не надеюсь.
На улице Ялантепе
Играя рос ребенком я ведь.

У воды костер горит.
Кто зажег тот костер?
Если спущусь к воде,
Найду синее детство.
```
