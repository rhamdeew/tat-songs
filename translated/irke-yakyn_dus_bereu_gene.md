# Оригинал

### Иркэ - Якын дус берэу генэ

```
Дус-ишлэр бик куп була ул,
Барысы да яхшы барса.
Янэшэдэ кемнэр калыр,
Авырлык килэ калса?

Авып барган агачларга
Булышмас торэу генэ.
Иптэшлэр бик куп булса да,
Якын дус берэу генэ.

Якын дус янэшэ булса,
Кайгы якын килэ алмас.
Араларга дошман тугел,
Жил-давыл да керэ алмас.

Авып барган агачларга
Булышмас торэу генэ.
Иптэшлэр бик куп булса да,
Якын дус берэу генэ.

Авыр мондый заманада
Дуслыкны саклап калу.
Хэркемнен кулыннан килми
Яхшы дус була алу.

Авып барган агачларга
Булышмас торэу генэ.
Иптэшлэр бик куп булса да,
Якын дус берэу генэ.
```

# Перевод

### Иркэ - Близкий друг только один

```
Приятелей-товарищей очень много бывает,
Все хороши, когда всё хорошо идёт.
Кто рядом останется,
Когда беда приходит?

Гнущимся деревьям
Не поможет буря.
Товарищей хоть очень много,
Близкий друг только один.

Если близкий друг рядом,
Печаль близко подойти не сможет.
Для них враг не страшен,
Ветер-буря тоже войти не сможет.

Гнущимся деревьям
Не поможет буря.
Товарищей хоть очень много,
Близкий друг только один.

Трудно в такое время
Дружбу сохранить.
Не каждому под силу
Хорошим другом стать.

Гнущимся деревьям
Не поможет буря.
Товарищей хоть очень много,
Близкий друг только один.
```
