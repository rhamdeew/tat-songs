# Оригинал

### Фирзар Муртазин - Хуш инде

```
Хуш инде, мэнгегэ
Хуш инде.
Китэ яз, китэ наз —
Кыш инде.
Тагын бер текэлеп
Карыймчы:
Рэсеменне кунелемэ
Алыймчы.

Ул китэ, мэнгегэ
Дип китэ.
Сон килде мэхэббэт, —
Ник китэ?
Чаналар кузгалып
Киттелэр,
Йорэкне оздереп
Уттелэр.

Эзлэрен кар-буран
Сылады.
Юлларга жил ятып
Елады.
Хуш инде, мэнгегэ
Хуш инде.
Утте яз, буранлы
Кыш инде.
```

# Перевод

### Фирзар Муртазин - Прощай же

```
Прощай же, навеки
Прощай же.
Уходит весна, уходит нега -
Зима же.
Ещё раз украдкой
Взглянем:
Твой образ в сердце моем
Оставим.

Он уходит, навеки,
Говорит, уходит.
Поздно пришла любовь -
Почему уходит?
Сани зашевелились,
Поехали,
Сердце разрывая,
Проехали.

Следы их пурга-метель
Заметет.
На дороги ветер лег,
Плачет.
Прощай же, навеки
Прощай же.
Прошла весна, вьюжная
Зима же.
```
