# Оригинал

### Хат - Синсез синсез

```
Син киттен, хуш диден
Мин калдым... ялгызым
Атылды, ватылды куктэге йолдызым
Лэкин мин сине онытмыйм,
Хэтирэлэр белэн тулы тон.
Исеменнне синен кабатлыйм
Саташам, кайда тош, кайда он?

Синсез, синсез, ай да кайгылы синсез
Иртэнге жил сине юксына иркэм
Йолдызларнын нурлары кунелсез
Кайт-кайт, мин сине котэм

Бакчада голлэрен, тик сине килгэнне
Котэлэр чэчэклэр иреннэренне
Ышанам...танда уятып,
Тик синен назлы кочуын
Йорэгемне минем юатыр
Югалыр эссез борчулар

Синсез, синсез, ай да кайгылы синсез
Иртэнге жил сине юксына иркэм
Йолдызларнын нурлары кунелсез
Кайт-кайт, мин сине котэм
```

# Перевод

### Хат - Без тебя, без тебя

```
Ты ушла, сказала прощай
Я остался... один
Упала, разбилась моя звезда в небе
Но я тебя не забываю,
Ночь полна воспоминаний.
Твое имя повторяю
Запутался, где сон, где явь?

Без тебя, без тебя, как печально без тебя
Утренний ветер тоскует по тебе, дорогая
Лучи звезд безрадостны
Вернись, вернись, я тебя жду

В саду цветы только твоего прихода
Ждут цветы на рассвете
Верю... проснувшись на заре,
Только твои нежные объятия
Утешат мое сердце
Исчезнут бесконечные печали

Без тебя, без тебя, как печально без тебя
Утренний ветер тоскует по тебе, дорогая
Лучи звезд безрадостны
Вернись, вернись, я тебя жду
```
