# Оригинал

### Иркэ - Жэй

```
Елнын ин матур чагы жэйнен ал танында,
Чэчэклэр исе килэ яшел кин болыннарда,
Балкып яна йолдызлар тонге hаваларда,
Аларда бит шаhитлэр hэр соешкэн парларга.

Жэй килде безгэ буген,
Ямьле жэй, матур жэй!
Жэй жырлый безгэ жырын,
Ямьле жэй, матур жэй!

Балкып яна йолдызлар тонге hаваларда,
Аларда бит шаhитлэр hэр соешкэн парларга.
Коенабыз суларда, шатланабыз без бергэ,
Кояш нурларын сибэ, hавалардан безгэ.

Жэй килде безгэ буген,
Ямьле жэй, матур жэй!
Жэй жырлый безгэ жырын,
Ямьле жэй, матур жэй!
```

# Перевод

### Иркэ - Лето

```
Самая прекрасная пора года - летняя заря,
Цветов аромат витает в зеленых широких лугах,
Сияют звезды в ночных небесах,
Они ведь свидетели каждой влюбленной пары.

Лето пришло к нам сегодня,
Прекрасное лето, красивое лето!
Лето поет нам свою песню,
Прекрасное лето, красивое лето!

Сияют звезды в ночных небесах,
Они ведь свидетели каждой влюбленной пары.
Купаемся в водах, радуемся вместе,
Солнце лучи свои дарит нам с небес.

Лето пришло к нам сегодня,
Прекрасное лето, красивое лето!
Лето поет нам свою песню,
Прекрасное лето, красивое лето!
```
