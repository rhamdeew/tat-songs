# Оригинал

### Әшрәф Талипов / Сара Садыйкова - Сагынам Урал буйкаен

```
Сагынамын Урал буен,
Әллә шунда үскәнгә.
Моңланып җырлап җибәрәм,
Исләремә төшкәндә.

Урал буйкаена чыксам,
Күзем карап туялмый.
Ямьле Урал буйкайлары
Күз алдымнан югалмый.

Кайда гына мин булсам да,
Сагынам Урал буйкаен,
Сагынам Урал буйкайларын,
Шунда татлы уйларым.

Ямьле Урал буйларында
Аккош бала уйната.
Урал буе исемә төшсә,
Уйлата да җырлата.

Җырлата да елата...
```

# Перевод

### Ашраф Талипов / Сара Садыйкова - Скучаю по берегам Урала

```
Скучаю по берегам Урала,
Ведь именно там я вырос.
С грустью пою песни,
Когда вспоминаю те края.

Когда выхожу на берега Урала,
Глаза не могут насмотреться.
Прекрасные берега Урала
Не исчезают из моих глаз.

Где бы я ни был,
Скучаю по берегам Урала,
Скучаю по уральским берегам,
Там мои сладкие мечты.

На прекрасных берегах Урала
Белый лебедь деток растит.
Когда вспомню берега Урала,
Заставляет и думать, и петь.

И петь, и плакать...
```
