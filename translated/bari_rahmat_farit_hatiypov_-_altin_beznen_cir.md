# Оригинал

### Бари Рәхмәт / Фәрит Хатыйпов - Алтын безнең җир

```
Бакча безнең ил,
Алтын безнең җир,
Җиргә тезләнеп,
Чәчәк өздем мин.

Менә шул чакта,
Җирне кочаклап,
Үбәсем килде, -
Сыймый кочакка.

Йөрәккә сыйды,
Туйганчы сөйдем:
«Йөрәгемдә кал
Мәңгегә!» - дидем.
```

# Перевод

### Бари Рәхмәт / Фарит Хатыпов - Золотая наша земля

```
Страна наша сад,
Золотая наша земля,
На землю встану на колени,
Цветок я нарвал.

Вот в тот момент,
Землю обхватив за плечи,
Пришло мое чувство, -
Не сдержалось в объятиях.

В сердце уместилось,
Сказал, как родной:
"В сердце моем останься
Навечно!" - сказал.
```
