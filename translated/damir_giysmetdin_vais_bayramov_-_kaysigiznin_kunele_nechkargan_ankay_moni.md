# Оригинал

### Дамир Гыйсметдин / Вәис Бәйрәмов - Кайсыгызның күңеле нечкәргән?.. (Әнкәй моңы)

```
Башым салам, юкәгә сөяләм,
Моңга сусап җаннар тилмерә...
Әкрен генә «Карурман»ны сузам,
Шул җыр миңа кинәт көч бирә.

Байтак йөрдем фани дөнья буйлап,
Тоялмадым тәмен тормышның.
Сагынып кайттым изге чишмәләрне,
Салкыннарын бездәге кышның...

Бишек җыры, әнкәй моңы белән
Сеңгән җанга изге мәхәббәт.
Аның белән яшәүләре - җәннәт,
Ансыз яшәүләре - кыямәт.

Байтак йөрдем фани дөнья буйлап,
Тоялмадым тәмен тормышның.
Сагынып кайттым изге чишмәләрне,
Салкыннарын бездәге кышның...

Бар язганым ята ак кәгазьдә...
Кайсыгызның күңеле нечкәргән?
Бармы арагызда елатырлык көйче -
Кайсыгызга Ходай моң биргән?

Байтак йөрдем фани дөнья буйлап,
Тоялмадым тәмен тормышның.
Сагынып кайттым изге чишмәләрне,
Салкыннарын бездәге кышның...
```

# Перевод

### Дамир Гыйсметдин / Ваис Байрамов - Чья душа утончённа?.. (Материнская мелодия)

```
Склоню главу, прислонюсь к подушке,
Души, томясь, тоскуют по мелодии...
Тихонько «Карурман» я запеваю,
И песня эта вдруг даёт мне силы.

Далёко я ходил по бренному миру,
Не насытился вкусом бытия.
Вернулся я, скучая, к родникам святым,
К прохладе нашей зимней стороны...

Колыбельная, материнский напев -
Впитала душа святую любовь.
Жить с ней - это райское блаженство,
А жизнь без матери - страшный суд.

Далёко я ходил по бренному миру,
Не насытился вкусом бытия.
Вернулся я, скучая, к родникам святым,
К прохладе нашей зимней стороны...

Всё написанное лежит на белой бумаге...
Чья душа утончённа средь вас?
Есть ли среди вас певец до слёз,
Кому Господь мелодию даровал?

Далёко я ходил по бренному миру,
Не насытился вкусом бытия.
Вернулся я, скучая, к родникам святым,
К прохладе нашей зимней стороны...
```
