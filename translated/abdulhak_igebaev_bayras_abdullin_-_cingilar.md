# Оригинал

### Абдулхак Игебаев / Байрас Абдуллин - Җиңгиләр

```
Сөйкемле дә, гаҗәп шәфкатьле дә,
Безнең яклардагы җиңгиләр.
Юкка мени алар инсафлыгын
Алан гөлләренә тиңлиләр.

Ихласлыгы белән ямьне арбый
Безнең яклардагы җиңгиләр.
Киңәш сорап килсәң һәрчак алар
Яхшылыкка гына тиңлиләр.

Хәстәрле дә алар, җигәрле дә,
Һич кенә дә тыңгы бирмиләр.
Абыйларым дәртле булгангамы?
Яшь йөрәкле безнең җиңгиләр.
```

# Перевод

### Абдулхак Игебаев / Байрас Абдуллин - Невестки

```
Милые и удивительно ласковые
Невестки в наших краях.
Не зря их честность сравнивают
С красными цветами.

Искренностью своей украшают мир
Невестки в наших краях.
Когда за советом приходишь, они всегда
Лишь к добру направляют.

И заботливы они, и сердечны,
Ни капли покоя не знают.
Братья мои, разве не дороги нам?
Молодые сердцем наши невестки.
```
