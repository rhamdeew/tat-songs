# Оригинал

### Казан егетлəре - Авылым урамнары

```
Таллары шаулый жиллэрдэ,
Койли курешу коен.
И минем туган авылым,
И минем туган оем.

Оныта алмыйм, онытмыйм
Авылым урамнарын —
Хэтер булып еракларга
Сузыла торганнарын.

Уз итеп яшэсэм дэ мин
Илемнен башка ягын,
Авылымда котеп тора
Куптэнге балачагым.

Оныта алмыйм, онытмыйм
Авылым урамнарын —
Хэтер булып еракларга
Сузыла торганнарын.

Рэхмэтемне белдерергэ
Сузлэрем житми минем.
Кырларын, урамнарын кук
Кунелен иркен синен.

Оныта алмыйм, онытмыйм
Авылым урамнарын —
Хэтер булып еракларга
Сузыла торганнарын.
```

# Перевод

### Казан егетлəре - Улицы моего села

```
Ивы шелестят на ветру,
Стадо пасётся весело.
О, мое родное село,
О, мой родимый дом.

Не могу забыть, не забуду
Улицы моего села —
Памятью в дали далёкой
Они протягиваются.

Хоть и живу я теперь
В другой стороне земли,
В моём селе ждёт меня
Давняя моя юность.

Не могу забыть, не забуду
Улицы моего села —
Памятью в дали далёкой
Они протягиваются.

Благодарность выразить
Слов у меня не хватит.
Поля твои, улицы твои пусть
Душе будут милы.

Не могу забыть, не забуду
Улицы моего села —
Памятью в дали далёкой
Они протягиваются.
```
