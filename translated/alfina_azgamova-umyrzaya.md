# Оригинал

### Альфина Азгамова - Умырзая

```
Карашларын кара урман,
Кузлэрен умырзая.
Зэнгэрлегендэ адаштым,
Чэчэклэр жыя, жыя.

Синен кунел кара урман,
Союен умырзая.
Адашудан куркып торам,
Эй язлар утэ зая.

Ялгышларын кара урман,
Назларын умырзая.
Сою голлэрен сулганчы
Сина илтер юл кая?

Барыр юлларымда урман,
Сулмый тор, умырзая.
Кичексэм дэ кичерэсен,
Котэсен кунел тыя.

Язмышларын кара урман,
Син узен умырзая.
Адаштырма язларымны,
Лэйсэннэр булып явам.

Мизгелгэ тин гомерегез
Утмэсен эле зая.
Кайтып килэм язларыма
Жыярга умырзая.
```

# Перевод

### Альфина Азгамова - Незабудка

```
Взгляды твои - темный лес,
Глаза - незабудки.
В их синеве заблудилась,
Цветы собираю, собираю.

Твоя душа - темный лес,
Любовь твоя - незабудка.
Заблудиться боюсь,
Ах, весны проходят напрасно.

Ошибки твои - темный лес,
Ласки твои - незабудки.
Пока цветы любви не завяли
К тебе ведущий путь где?

На моих дорогах лес,
Не вянь, незабудка.
Хоть опоздаю, простишь,
Будешь ждать, душу терзая.

Судьбы твои - темный лес,
Ты сама - незабудка.
Не заблудить мои весны,
Соловьями стану и спою.

На мгновение равной жизни вашей
Пусть не пройдет напрасно.
Возвращаюсь к своим веснам
Собирать незабудки.
```
