# Оригинал

### Дәрдмәнд / Татар халык көе - Кораб

```
Шаулый диңгез...
Җил өрәдер...
Җилкәнен киргән кораб!
Төн вә көндез
Ул йөрәдер:
Юл бара ят ил карап...

Чыкты җилләр,
Купты дулкын -
Ил корабын җил сөрә!..
Кайсы юллар,
Нинди упкын
Тарта безне җан сорап?!
```

# Перевод

### Дардманд / Татарская народная песня - Корабль

```
Шумит море...
Ветер гонит...
Корабль с поднятым парусом!
День и ночь
Он плывёт:
Путь держит к чужим краям...

Поднялись ветра,
Взбунтовались волны -
Ветер гонит корабль народа!..
Какие дороги,
Какая стихия
Влечёт нас, души испытывая?!
```
