# Оригинал

### Салават Фатхетдинов - Энкэмнен догалары

```
Догаларнын кунелемнэн
Юк эле югалганы —
Саклый мине, яклий мине
Энкэмнен догалары.

Ташлама, энкэй, ташлама
Мине изге доганнан.
Ташласан изге доганнан
Мин бэхетле булалмам.

Тылсымлы да, шифалы да
Энкэмнен догалары —
Догаларнын шатлык кына
Китерэ торганнары.

Ташлама, энкэй, ташлама
Мине изге доганнан.
Ташласан изге доганнан
Мин бэхетле булалмам.

Кырыс ла тормыш юллары,
Авыр ла доньялары!
Хэллэремне жинелэйтэ
Энкэмнен догалары.

Ташлама, энкэй, ташлама
Мине изге доганнан.
Ташласан изге доганнан
Мин бэхетле булалмам.

Кабатлыйм мин дэ энкэйдэн
Ойрэнгэн догаларны.
Житкерермен тукми-чэчми
Мин сезгэ дэ аларны.

Ташлама, энкэй, ташлама
Мине изге доганнан.
Ташласан изге доганнан
Мин бэхетле булалмам.
```

# Перевод

### Салават Фатхетдинов - Мамины молитвы

```
Из моего сердца молитвы
Еще не исчезли —
Хранят меня, защищают меня
Мамины молитвы.

Не оставляй, мама, не оставляй
Меня без святой молитвы.
Если оставишь без святой молитвы,
Я не смогу быть счастливым.

Волшебные и целебные
Мамины молитвы —
Только радость приносят
Эти молитвы.

Не оставляй, мама, не оставляй
Меня без святой молитвы.
Если оставишь без святой молитвы,
Я не смогу быть счастливым.

Крутые жизненные дороги,
Тяжелы житейские дела!
Облегчают мои заботы
Мамины молитвы.

Не оставляй, мама, не оставляй
Меня без святой молитвы.
Если оставишь без святой молитвы,
Я не смогу быть счастливым.

Повторяю и я от мамы
Выученные молитвы.
Донесу обязательно
Их и до вас.

Не оставляй, мама, не оставляй
Меня без святой молитвы.
Если оставишь без святой молитвы,
Я не смогу быть счастливым.
```
