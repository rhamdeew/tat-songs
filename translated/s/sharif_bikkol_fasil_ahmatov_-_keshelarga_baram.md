# Оригинал

### Шәриф Биккол / Фасил Әхмәтов - Кешеләргә барам

```
Куанычтан җан җылынса,
Якты сукмак ярам;
Шатлыгымны өләшергә
Кешеләргә барам!

Күңелемне юатырга
Бетсә соңгы чарам,
Хәсрәтләрдән арынырга
Кешеләргә барам.

Кемлегемне сынар булсам
Кешеләргә барам!
И, кешеләр, сезнең белән
Өзелмәсен арам!
```

# Перевод

### Шәриф Биккол / Фасил Әхмәтов - К людям иду

```
Когда от радости душа теплеет,
Я путь свой светлый выбираю сам;
Чтоб счастье с вами разделить скорее,
К людям иду!

Когда иссякнут все мои опоры,
Чтоб сердце успокоить, без обмана,
Чтоб от печалей отряхнуть узоры,
К людям иду.

Когда хочу я испытать, кто есть я,
К людям иду!
О люди, с вами пусть не рвётся нить,
Пусть не растает наша связь вовек!
```
