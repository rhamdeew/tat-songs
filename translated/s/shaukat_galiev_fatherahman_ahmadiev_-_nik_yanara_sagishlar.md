# Оригинал

### Шәүкәт Галиев / Фәтхерахман Әхмәдиев - Ник яңара сагышлар?

```
Шомырт тагын чәчәк койды,
Сиңа дип өзалмадым,
Ак чәчәкләр очкан чакта
Сагынуга түзалмадым...

Шул чәчәкләргә кушылып
Очар идем жилләрдә,
Табышмыйча калмас идек,
Йөрсәк дә кай илләрдә...

Калды күңел әрнетергә
Киткән юлларың гына,
Калды охшатып карарга
Кызың, улларың гына.

Инде ничә язлар үтте,
Күптән тынды сугышлар,
Яз килгән саен яңара,
Ник яңара сагышлар?

...Шомырт тагын чәчәк койды,
Сиңа дип өзалмадым,
Ак чәчәкләр очкан чакта
Сагынуга түзалмадым...
```

# Перевод

### Шәүкәт Галиев / Фәтхерахман Әхмәдиев - Почему вновь оживает тоска?

```
Черемуха снова роняет цветы,
Для тебя их сорвать не смогла я,
Когда белые лепестки ввысь уплывают,
Я тоску не смогла удержать...

Я бы с теми цветами взлетела
И помчалась бы в вольных ветрах,
Мы бы не разошлись, не потерялись,
Даже если бы шли по чужим краям...

Лишь твои уходящие тропы
Сердце ранят, как прежде, до дна,
Лишь смотреть, узнавая, остались
Твои дочери, сыновья.

Сколько вёсен уже миновало,
Давным-давно умолкли бои,
Каждой весной вновь оживает -
Почему оживает тоска?

...Черемуха снова роняет цветы,
Для тебя их сорвать не смогла я,
Когда белые лепестки ввысь уплывают,
Я тоску не смогла удержать...
```
