# Оригинал

### Саҗидә Сөләйманова / Мәсгудә Шәмсетдинова - Буген әни бәйрәме (Кем уяткан Ләйләне?)

```
Иртүк торып битен юган,
Кем уяткан Ләйләне?
Уятмаган, үзе торган -
Буген әни бәйрәме.

Пәрдәләрне ачып куйды,
Көн кояшлы, әй, ямьле!
Якты булсын, балкып торсын -
Бүгән әни бәйрәме.

Өстәл өстен кат-кат сөртте,
Җәйде ап-ак җәймәне.
Чиста булсын, пөхтә булсын -
Буген әни бәйрәме.

Күмәкләшеп без дә, дуслар,
Җырлап алыйк, әйдә әле: -
Котлы булсын! Гөрләп торсын
Әниләрнең Бәйрәме!
```

# Перевод

### Сажида Сулейманова / Мәсгудә Шәмсетдинова - Сегодня мамин праздник

```
Встав пораньше, пол помыла,
Кто разбудил Лейлу?
Не будил, сама встала -
Сегодня мамин праздник.

Шторы отворила распахнуты,
День солнечный, о, какой прекрасный!
Пусть будет светло, пусть сияет -
Сегодня мамин праздник.

Стол оттерла раз за разом,
Расстелила скатерть белоснежную.
Пусть будет чисто, опрятно -
Сегодня мамин праздник.

Давайте вместе мы, друзья,
Споём, ну-ка же: -
С праздником! Пусть звучит
Мамин праздник!
```
