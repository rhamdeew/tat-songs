# Оригинал

### Шаһинур Мостафин / Фатих Мөхәммәтшин - Килермен

```
Килермен мин тагын бер сезнең якларга,
Арада күп еллар, күп юллар ятса да.

Мәхәббәт - мәңгелек,
Яши ул гомергә;
Йолдыздай балкый ул
Күңелнең түрендә...

Килермен мин тагын бер сезнең якларга,
Эзләрең бит синең һәр җирдә саклана.

Яшьлек чак җырга тиң,
Тынуны һич белми.
Йөрәктә - мәхәббәт,
Сүнми ул, сүрелми...

Килермен мин тагын бер сезнең якларга,
Йөрәктә урын юк, бәгырем, башкага.

Гомерем буена
Гел сине эзләрмен.
Нур сибәр юлымда
Һаман да күзләрең...
```

# Перевод

### Шаһинур Мостафин / Фатих Мөхәммәтшин - Вернусь

```
Я ещё вернусь к твоим краям,
Хоть пройдут годы, пролегут пути.

Любовь - вечна,
Живёт она всю жизнь;
Как звездой сияет она
В душе глубокой...

Я ещё вернусь к твоим краям,
Следы твои везде хранятся.

Молодость подобна песне,
Застывать не умеет.
В сердце - любовь,
Не гаснет она, не тускнеет...

Я ещё вернусь к твоим краям,
В сердце нет места, любовь моя, для другого.

Всю жизнь буду
Искать тебя.
На мой путь свет scattering
Вечно глаза твои...
```
