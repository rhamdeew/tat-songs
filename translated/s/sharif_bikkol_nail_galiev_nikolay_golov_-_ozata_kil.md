# Оригинал

### Шәриф Биккол / Наил Галиев (Николай Голов) - Озата кил

```
Сайра, сайра, сандугачым,
чиртмә-чиртмә талларында;
хыялыйлар иткән идең
мине яшьлек таңнарымда.

Әле булса,
тыңлый-тыңлый,
моңнарыңа хәйран калам:
чут-чут сайрар минутыңны
мин зарыгып көтеп алам.

Чиртә-чиртә күңелемнең
иң-иң нечкә кылларына,
озата кил
гомеремнең
яшәлмәгән елларына!
```

# Перевод

### Шариф Биккол / Наил Галиев (Николай Голов) - Проводи меня

```
Пой, пой, соловей мой,
на ивовых тонких ветвях;
ты в мечты погружал
меня в юности ранних зорях.

И сейчас ещё,
слушая, слушая,
песнями твоими восхищён:
каждый миг твоего едва слышного пения
я с волнением жду и ловлю.

Тронь, тронь души моей
самые-самые тонкие струны,
проводи меня
в годы жизни моей,
которые не прожиты!
```
