# Оригинал

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Керпе

```
Ай, һай керпе генәсе,
Уктай үткен энәсе.
Хәтта фил дә җиңелә,
Күрде исә - чигенә.
```

# Перевод

### Шаукат Галиев / Мәсгудә Шәмсетдинова - Ёжик

```
Ой, малютка-ёжик,
Иголок - острейших игла.
Даже слон становится легче,
Ешь лишь - он убегает.
```
