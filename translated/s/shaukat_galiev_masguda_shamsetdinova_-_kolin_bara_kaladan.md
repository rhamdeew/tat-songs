# Оригинал

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Колын бара каладан

```
Колын бара каладан,
Ияргәннәр балалар.
Артыннан һич калмыйлар,
Бик яратып карыйлар.

Колын кала күрмәгән,
Кала колын күрмәгән.
Колын бара каладан,
Кала аңа караган.
```

# Перевод

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Жеребёнок едет из города

```
Жеребёнок едет из города,
Дети следом пристали.
Ни на шаг не отстают,
С любовью на него глядят.

Жеребёнок город не видел,
Город жеребёнка не знал.
Жеребёнок едет из города,
А город глядит ему вслед.
```
