# Оригинал

### Сафуан Әлибай / Альбина Имаева - Гөлназ

```
Язлар инде минем артта калды,
Сагынырга калды гөлләрем.
Болотларның җиле моңсуланды,
Гел кояшлап тормый көннәре.

Гөлназым, Гөлназым,
Син барыбыр кил син,
Кил язым.

Китте инде кошлар еракларга,
Яфраклар да калды талпынып.
Ялкыннары сүнгәч яфракларның,
Үзәкләргә үтте салкыны.

Гөлназым, Гөлназым,
Син барыбыр кил син,
Кил язым.
```

# Перевод

### Сафуан Алибәй / Альбина Имаева - Гюльназ

```
Весна уже осталась позади,
Мне лишь грустить о моих цветах.
Ветер в облаках печально стынет,
Не всегда сияют дни в лучах.

Гюльназ моя, Гюльназ моя,
Ты все равно приди,
Приди, весна моя.

Птицы улетели далеко,
Листья лишь остались, дрогнув в тишине.
Когда погасли их огни-отблески,
Холод в самое сердце прошел ко мне.

Гюльназ моя, Гюльназ моя,
Ты все равно приди,
Приди, весна моя.
```
