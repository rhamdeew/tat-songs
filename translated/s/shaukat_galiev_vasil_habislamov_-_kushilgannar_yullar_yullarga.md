# Оригинал

### Шәүкәт Галиев / Вәсил Хәбисламов - Кушылганнар юллар юлларга

```
Кушылганнар юллар юлларга,
Кушылганнар куллар кулларга,
Йөрәкләрнең якын арасы,
Йөрәкләрнең якын арасы.

Офыклардан офыкларны үтеп,
Гел барасы иде, барасы!

Табыштылар безнең йөрәкләр,
Кавыштылар безнең теләкләр,
Шатлансын да дуслар, соклансын,
Шатлансын да дуслар, соклансын!

Мәхәббәтнең шушы кайнарлыгы
Гомер буйларына саклансын!

Аерылмасын күзләр күзләрдән,
Борылмасын йөзләр йөзләрдән,
Үтеләсе юллар үткәнче,
Үтеләсе юллар үткәнче!

Үтеләсе юлларыбыз үтеп,
Безгә дигән гомер беткәнче!
```

# Перевод

### Шәүкәт Галиев / Вәсил Хәбисламов - Соединились дороги с дорогами

```
Соединились дороги с дорогами,
Соединились ладони с ладонями,
Сердцам - совсем короткая даль,
Сердцам - совсем короткая даль.

Через горизонты, минуя миры,
Все дальше идти, все дальше идти бы!

Наши сердца друг друга нашли,
Наши желания в вечность сошлись,
Пусть друзья порадуются, восхитятся,
Пусть друзья порадуются, восхитятся!

Пусть этот жар любви
Сохранится на все годы жизни!

Пусть взгляды не расходятся со взглядами,
Пусть лица не отворачиваются от лиц,
Пока не пройдены пути,
Пока не пройдены пути!

Пока не пройдены наши дороги,
Пока не иссякнет нам отмеренный век!
```
