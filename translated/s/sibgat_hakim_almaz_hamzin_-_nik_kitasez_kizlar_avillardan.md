# Оригинал

### Сибгат Хәким / Алмаз Хәмзин - Ник китәсез, кызлар, авыллардан?

```
Ник китәсез, кызлар, авыллардан
Егетләрне ялгыз калдырып?
Нинди бәхет чишмә буйларыннан
Бер әйләнсәң кичен пар булып.

Ник китәсез, кызлар, авыллардан
Авылларның алып бар ямьен?
Җәйге алсу таңнар, моңсу таңнар,
Кемнәр өчен сузар тальянын?

Ник китәсез, кызлар, авыллардан
Егетләргә төшереп зур хәсрәт?
Сездән башка, алар ярты гына,
Була мени ярты мәхәббәт?
```

# Перевод

### Сибгат Хәким / Алмаз Хәмзин - Почему уезжаете, девушки, из деревень?

```
Почему уезжаете, девушки, из деревень,
Оставляя парней в одиночестве?
Что за счастье у ручьев-истоков -
Вечером пройтись вдвоем, парой.

Почему уезжаете, девушки, из деревень,
Унося у деревень всю прелесть?
Летние розовые зори, тоскливые зори,
Для кого тальянку растянет?

Почему уезжаете, девушки, из деревень,
Парням в сердце принося большую печаль?
Без вас они лишь наполовину,
Разве бывает любовь наполовину?
```
