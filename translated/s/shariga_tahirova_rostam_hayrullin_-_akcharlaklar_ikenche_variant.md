# Оригинал

### Шәрига Таһирова / Рөстәм Хәйруллин - Акчарлаклар (икенче вариант)

```
Кама өсләрендә очкан
Акчарлак кебек,
Безнең еллар юллар яра
Үрләргә менеп.

Хыял канатларын кагып
Мәхәббәттән илһам алып
Яңрат, яшьлек, туган як моңын!

Акчарлаклар, акчарлаклар,
Чал дулкыннарда.
Яшь йөрәкнең ялкыннары
Кабына җырларда.

Хыял канатларын кагып
Мәхәббәттән илһам алып
Яңрат, яшьлек, туган як моңын!

Акчарлаклар, акчарлаклар,
Атыла уклардай.
Дәртле уйлар күкрәкләрдә
Яна утлардай.

Хыял канатларын кагып
Мәхәббәттән илһам алып
Яңрат, яшьлек, туган як моңын!
```

# Перевод

### Шарига Тахирова / Рустам Хайруллин - Белые мотыльки (альтернативная версия)

```
Как белый мотылёк
Над вершинами гор,
Наши годы, как тропинка,
Взбираются на холмы.

С крыльями мечты,
Вдохновлённые любовью,
Обновляй, молодость, тоску родного края!

Белые мотыльки, белые мотыльки,
Среди шумных волн.
Пламя молодого сердца
Вспыхивает в песнях.

С крыльями мечты,
Вдохновлённые любовью,
Обновляй, молодость, тоску родного края!

Белые мотыльки, белые мотыльки,
Как стрелы, пущённые.
Пылкие мысли в грудях
Горят, как огонь.

С крыльями мечты,
Вдохновлённые любовью,
Обновляй, молодость, тоску родного края!
```
