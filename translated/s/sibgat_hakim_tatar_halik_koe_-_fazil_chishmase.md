# Оригинал

### Сибгат Хәким / Татар халык көе - Фазыл чишмәсе

```
Чылтырап аккан чишмәгә
Суга дип килгән идем;
Тиз генә әйләнеп өйгә
Кайтырмын дигән идем.

Өздереп нигә карадың,
Ник тоттың чиләгемнән?
Суларым түгел, ялкыным
Түгелде йөрәгемнән.

Кызарып янган шәфәкътан
Күзләрем алалмадым.
Тибрәнде телем, әйтергә
Сүзләрем табалмадым.

Чишмәгә төшкән сукмакның
Читләре үлән генә.
Вәгъдәм бар сиңа, белмәсен
Тик күрше-күлән генә.
```

# Перевод

### Сибгат Хаким / Татарская народная мелодия - Родник Фазила

```
К сверкающему ручью,
Чтобы набрать воды, я шёл;
Думал быстро вернуться домой,
Так хотел.

Почему ты испуганно смотрела,
Почему не взяла моё ведёрко?
Не воды мне не хватало, пламя
Потухло в моём сердце.

От пылающей зари
Не мог отвести глаза.
Дрожал мой язык, чтобы сказать,
Слов не мог найти.

У тропинки к ручью
Обочины только в траве.
Есть у тебя обещание моё, не знает
Только соседка.
```
