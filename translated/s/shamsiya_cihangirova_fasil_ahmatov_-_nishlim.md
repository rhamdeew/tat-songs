# Оригинал

### Шәмсия Җиһангирова / Фасил Әхмәтов - Нишлим

```
Сине өзелеп сөюемне
Ничек аңлатыйм икән?
Йокыларыңнан уятып,
Таң булып атыйм микән?
Әллә шатлык яше булып
Күзләреңнән тамыйммы?
Сандугач булып сайрыйммы,
Гөлләр булып карыйммы?
Җырлар булып, моңнар булып
Күңеленә тулыйммы?
Әллә үзем генә булып,
Гел яныңда торыйммы?
```

# Перевод

### Шамсия Джихангирова / Фасиль Ахметов - Что мне делать

```
Как мне объяснить тебе,
Что люблю тебя до боли?
Разбудив тебя ото сна,
Рассветом ли взойти мне?
Иль слезой радости
Из глаз твоих пролиться?
Соловьём ли мне запеть,
Цветами ли явиться?
Песнями ли, напевами ли
Наполнить твоё сердце?
Иль просто самой собой
Быть рядом с тобой всегда?
```
