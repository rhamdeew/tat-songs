# Оригинал

### Солтан Гыйлемханов / Мөдәрис Газетдинов - Яңа таңнар көтә

```
Яшь гомерне ике килми диләр,
Бер килгәне китми калгандыр.
Без күрәсе шаулы язлар әле,
Ямьле җәйләр әле алдадыр.

Безне әле яңа таңнар көтә,
Барыр юлга киртә куймаган.
Яшик әле, якты дөнья бездән,
Без дөньядан әле туймаган.

Гомер җәйләрен дә узмадык без,
Салкын көзе әле килмәгән.
Дус-ишләргә бүләк итәр өчен
Күңел чәчкәләре шиңмәгән.

Безне әле яңа таңнар көтә,
Барыр юлга киртә куймаган.
Яшик әле, якты дөнья бездән,
Без дөньядан әле туймаган.

Куркытмыйлар безне җил-давыллар,
Куркытмыйлар ачы салкыннар.
Йөрәкләрдә туңган күңелләрне
Эретерлек кайнар ялкын бар!

Безне әле яңа таңнар көтә,
Барыр юлга киртә куймаган.
Яшик әле, якты дөнья бездән,
Без дөньядан әле туймаган.
Без дөньядан әле туймаган...
```

Прекрасное лето ещё обманчиво маячит.

Мы ещё не прошли лета жизни своей,
Холодная осень ещё не наступила.

Горит пламя жаркое!
