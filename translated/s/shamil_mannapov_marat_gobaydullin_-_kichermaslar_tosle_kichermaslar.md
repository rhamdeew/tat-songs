# Оригинал

### Шамил Маннапов / Марат Гобәйдуллин - Кичермәсләр төсле, кичермәсләр

```
Синең белән аерылганнан бирле
Телим гөлгә бүтән үрелмәскә, -
Безне таныштырган алсу гөлчәчәкләр
Кичермәсләр төсле, кичермәсләр.

Синең белән аерылганнан бирле
Йоклый алмыйм айлы кичләрдә, -
Безне кавыштырган айлы матур кичләр
Кичермәсләр төсле, кичермәсләр.

Синең белән аерылганнан бирле
Телим хәзер искә төшермәскә, -
Рәнҗетелгән яшьлек, уртак сөю-хисләр
Кичермәсләр безне, кичермәсләр.
```

# Перевод

### Шамиль Маннапов / Марат Габайдуллин - Воспоминаний цветные, воспоминания

```
С тех пор, как расстались с тобой,
Хочу я, чтобы не цвести другим, -
Те алые цветочки, что нас познакомили,
Воспоминаний цветные, воспоминания.

С тех пор, как расстались с тобой,
Не сплю я в лунные ночи, -
Те лунные красивые вечера, что нас соединили,
Воспоминаний цветные, воспоминания.

С тех пор, как расстались с тобой,
Хочу теперь не вспоминать, -
Раненое детство, общие любовь-чувства
Воспоминаний нас, воспоминания.
```
