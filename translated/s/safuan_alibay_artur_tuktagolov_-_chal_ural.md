# Оригинал

### Сафуан Әлибай / Артур Туктаголов - Чал Урал

```
Чал Уралның итәгендә
Гүзәл туган илкәем,
Каялардан — далалардан,
Җилә иркә җилкәем.

Һай Урал, чал Урал —
Син минем бердән-берем,
Чал Урал, чал далам —
Әй, газиз туган җирем!.

Тиз каладан — калаларны
Җырлый — җырлый урадым.
Сәфәр гизеп-йөреп кайтып,
Һәй Ауропа таныдым.

Һай Урал, чал Урал —
Син минем бердән-берем,
Чал Урал, чал далам —
Әй, газиз туган җирем!.

Америка оҗмах диләр
Шунда тормыш корганнар.
Тамыр җәйгән аталарым
Бөтен җирдә булганнар.

Һай Урал, чал Урал —
Син минем бердән-берем,
Чал Урал, чал далам —
Әй, газиз туган җирем!.
```

# Перевод

### Сафуан Алибай / Артур Туктаголов - Седой Урал

```
У подножия седого Урала
Моя прекрасная родина,
Из скал - из степей,
Ветер ласково плечи гладит.

Ой Урал, седой Урал -
Ты моя единственная,
Седой Урал, седая степь моя -
Ой, дорогая родная земля!

Быстро от города к городам
Пел - пел проезжал.
Странствуя и странствуя вернувшись,
О Европу узнал.

Ой Урал, седой Урал -
Ты моя единственная,
Седой Урал, седая степь моя -
Ой, дорогая родная земля!

В Америке рай говорят
Там жизнь построили.
Корни пустившие отцы мои
Повсюду были.

Ой Урал, седой Урал -
Ты моя единственная,
Седой Урал, седая степь моя -
Ой, дорогая родная земля!
```
