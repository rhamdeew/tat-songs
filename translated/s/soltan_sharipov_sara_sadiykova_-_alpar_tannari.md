# Оригинал

### Солтан Шәрипов / Сара Садыкова - Алпар таңнары

```
Алпар кырларында таңнар ата,
Кошлар сайрый назлап күңелне.
Яшьлек тойгылары синдә туды,
Синдә яшәү миңа күңелле.

Әй, Алпарым, Алпарым,
Синдәй ямьле кайда бар тагын?

Моңнар суза тальян, җырлар суза,
Чирмешән буйларын яңгыратып.
Туган авылым, үстердең син мине,
Яшел чирәмнәрдә уйнатып.

Әй, Алпарым, Алпарым,
Синдәй ямьле кайда бар тагын?

Яратам мин туган авылымны,
Шат күңелле эшчән яшьләрен.
Бөек хезмәт белән даннар алган
Батыр халкын, гүзәл кешеләрен.

Әй, Алпарым, Алпарым,
Синдәй ямьле кайда бар тагын?

Алпар күлләрендә йөзә үрдәк,
Оча казлар, канат кагынып.
Читтә торган авылдашлар кайта,
Кайта үз Алпарын сагынып.

Әй, Алпарым, Алпарым,
Синдәй ямьле кайда бар тагын?
```

# Перевод

### Солтан Шарипов / Сара Садыкова - Зори Алпара

```
Над полями Алпара встают зори,
Птицы нежно поют для души.
Чувства юности во мне родились здесь,
Жизнь с тобою мне так хороша.

Ах, мой Алпар, мой Алпар,
Где найти ещё красоты такой?

Льются песни и печали льются,
Вдоль черёмухи звенят.
Родная деревня, ты вырастила меня,
Среди зелёных лугов играть.

Ах, мой Алпар, мой Алпар,
Где найти ещё красоты такой?

Я люблю свою родную деревню,
Весёлую молодёжь трудовую.
Великим трудомославу снискавший
Отважный народ, прекрасные люди.

Ах, мой Алпар, мой Алпар,
Где найти ещё красоты такой?

На озёрах Алпара плывут утки,
Гуси летят, взмахнув крылом.
В дальних краях живущие земляки вернутся,
Вернутся, скучая по своему Алпару.

Ах, мой Алпар, мой Алпар,
Где найти ещё красоты такой?
```
