# Оригинал

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Бозау белән чикерткә

```
Атласа да тын гына,
Тавыш бирә кыңгырау:
Чең, чең, чең, чең...
Ишеткән, ди, чикерткә.

Мин дә монда ялгызау,
Кил бергә сайрыйк, бозау.
Чрик, чрик, чрик, чрик...
Чең, чең, чең...
```

# Перевод

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Телёнок и кузнечик

```
Ходит тихо-тишиной,
Колокольчик подаёт:
Динь, динь, динь, динь...
Слышит, говорят, кузнечик.

Я и тут один, телёнок,
Приходи, давай споём.
Чрик, чрик, чрик, чрик...
Динь, динь, динь...
```
