# Оригинал

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Тәгәри тормыш арбасы

```
Тәгәри тормыш арбасы,
Ай үтәсен көн үтә;
Язны кичеп, җәйне гизеп,
Гомер көзенә җитә.

Тәгәри тормыш арбасы
Язмышның юлларыннан.
Урап тотам дилбегәмне,
Төшермим кулларымнан.

Олавыма шатлыклар да,
Борчулар да төялгән,
Табышлар һәм югалтулар
Катлап-катлап өелгән.

Тәгәри тормыш арбасы
Язмышның юлларыннан.
Урап тотам дилбегәмне,
Төшермим кулларымнан.

Тормыш йөгем арта бара,
Чакрымнар узган саен.
Текә үргә менүләрнең
Табам бит әле җаен!

Тәгәри тормыш арбасы
Язмышның юлларыннан.
Урап тотам дилбегәмне,
Төшермим кулларымнан.
```

# Перевод

### Сания Ахметжанова / Айдар Файзрахманов - Колесо жизни

```
Катится колесо жизни,
Проходят месяцы, проходят дни;
Минуя весну, пройдя сквозь лето,
Достигает осени судьбы.

Катится колесо жизни
По дорогам предназначения.
Крепко держу я вожжи,
Не выпущу их из рук своих.

В мою телегу сложены
И радости, и печали,
Находки и потери
Наслоились друг на друга.

Катится колесо жизни
По дорогам предназначения.
Крепко держу я вожжи,
Не выпущу их из рук своих.

Бремя жизни всё тяжелей,
Чем дальше проходят годы.
Но восхождений на вершины
Я ещё найду дорогу!

Катится колесо жизни
По дорогам предназначения.
Крепко держу я вожжи,
Не выпущу их из рук своих.
```
