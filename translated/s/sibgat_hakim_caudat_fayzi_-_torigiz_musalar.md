# Оригинал

### Сибгат Хәким / Җәүдәт Фәйзи - Торыгыз, Мусалар

```
Кайвакыт күп төрле
Уйлардан йөдисең...
Килә бер җир буйлап
Кычкырып йөрисем:
Басуда - арышлар,
Бакчада әнисләр
Өлгерде, торыгыз,
Торыгыз, Алишлар...

Кайвакыт бүлмәдән
Качасың юри син...
Килә бер җир буйлап
Кычкырып йөрисем:
Дошманнар тынмаслар,
Коялар яңадан
Кургашлар, торыгыз,
Торыгыз, Курмашлар...

Кайвакыт исним дә
Чәчәкле җил исен,
Килә бер җир буйлап
Кычкырып йөрисем:
Чәчәкләр үсәләр,
Көн саен тормышта
Сез кирәк, торыгыз,
Торыгыз, Мусалар...
```

# Перевод

### Сибгат Хәким / Җәүдәт Фәйзи - Вставайте, Муса

```
Порой от разных мыслей
Ты вымотаешься...
И хочется по просёлку
Кричать во весь голос:
В полях - рожь золотая,
В саду - анисы
Созрели, вставайте,
Вставайте, Алишы...

Порой из комнаты
Ты нарочно бежишь...
И хочется по просёлку
Кричать во весь голос:
Враги не умолкнут,
Снова льют
Свинец - вставайте,
Вставайте, Курмаши...

Порой вдыхаю я
Ароматный цветочный ветер,
И хочется по просёлку
Кричать во весь голос:
Цветы поднимаются,
День за днём в жизни
Вы нужны - вставайте,
Вставайте, Муса...
```
