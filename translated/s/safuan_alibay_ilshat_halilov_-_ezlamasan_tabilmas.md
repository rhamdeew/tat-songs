# Оригинал

### Сафуан Әлибай / Илшат Хәлилов - Эзләмәсәң - табылмас

```
Ашкынам айлар, еллар,
Сине эзлимен мин,
Кайда гына югалдың?
Һаман белмимен мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.

Эзләрмен җирдә, күктә,
Янармын утта мин.
Мең урап гәләмнәрне,
Барыбер табармын мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.

Бергә әйләнәбез
Иксез-чиксез дөнъяда.
Җиргә бит кешеләр,
Серле генә күзләп,
Сагышлардан йөзәп,
Бер-бересен эзләп
Табар өчен ярала.

Ашкынам айлар, еллар,
Сине эзлимен мин,
Кайда гына югалдың?
Һаман белмимен мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.
```

# Перевод

### Сафуан Алибаев / Илшат Халилов - Не ищи - не найдешь

```
Исчезают месяцы, годы,
Я ищу тебя,
Куда же ты пропал?
Всё не знаю я,
Ведь любовь
Без поиска не найдётся вовек,
В твоём сердце не вспыхнут огни.

Буду искать на земле, на небесах,
Гореть в огне я буду.
Обойду тысячи миров,
Всё равно найду,
Ведь любовь
Без поиска не найдётся вовек,
В твоём сердце не вспыхнут огни.

Вместе блуждаем
В безграничном мире.
Ведь людям на земле
Тайком смотреть в глаза,
Плыть сквозь тоску,
Ища друг друга,
Ради встречи любить.

Исчезают месяцы, годы,
Я ищу тебя,
Куда же ты пропал?
Всё не знаю я,
Ведь любовь
Без поиска не найдётся вовек,
В твоём сердце не вспыхнут огни.
```
