# Оригинал

### Шәүкәт Галиев / Шакир Мәҗитов - Юл җыры

```
Паровоз чакыра тагын
Ерак җирләргә,
Өйдә яту эшмени ул
Бездәй ирләргә!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Алда без шәһәр салачак
Далалар көтә.

Сөйгән ярлар да көтсен бер,
Сагынсын әле.
Тын яткан далада утлар
Кабынсын әле!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Алда без юллар сузачак
Урманнар көтә.

Җилкендереп күңелләрне
Ерак җир тарта,
Йөргән саен, күргән саен
Барыр җир арта!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Безне якты гомер юлы,
Зур тормыш көтә!
```

# Перевод

### Шаукат Галиев / Шакир Мажитов - Дорожная песня

```
Паровоз опять зовёт
В края далёкие,
Домой сидеть - не то дело
Для таких парней, как мы!

Колёса, колёса
Мчатся вперёд,
Впереди нам город строить
Ждут просторы.

Пусть любимые ждут,
Пусть скучают по нам.
Пусть в тихой степи огни
Загорятся ярко!

Колёса, колёса
Мчатся вперёд,
Впереди нам дороги прокладывать
Ждут леса.

Манят к себе наши души
Край далёкий,
Чем больше ходим, чем больше видим -
Тем больше мест, куда стремимся!

Колёса, колёса
Мчатся вперёд,
Нас ждёт светлый жизненный путь,
Великая жизнь!
```
