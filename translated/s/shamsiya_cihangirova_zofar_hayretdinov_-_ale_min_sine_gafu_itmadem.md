# Оригинал

### Шәмсия Җиһангирова / Зөфәр Хәйретдинов - Әле мин сине гафу итмәдем

```
Әле мин сине гафу итмәдем,
Иңемә килеп сарыла күрмә.
Күз яшьләреңне күрсәтмә миңа,
Ялына күрмә, ялына күрмә.

Әле мин сине гафу итмәдем,
Өметләр баглап чәчәк китермә.
Йомшак ла аның күңеле диеп,
Гафу үтенмә, гафу үтенмә.

Әле мин сине гафу итмәдем,
Каршыма килеп башыңны имә.
Яратуымны белеп торасың,
Кире кайт, димә, кире кайт, димә.
```

# Перевод

### Шамиля Джихангирова / Зуфар Хайретдинов - Я всё ещё не простил тебя

```
Я всё ещё не простил тебя,
Не приходи ко мне с упрёком.
Не показывай мне свои слёзы,
Не моли, не моли.

Я всё ещё не простил тебя,
Не срывай цветы с надежд.
Не говори о мягком сердце,
Не проси прощения, не проси прощения.

Я всё ещё не простил тебя,
Не кланяйся передо мной.
Ты ведь знаешь мою любовь,
Не возвращайся, не возвращайся.
```
