# Оригинал

### Сафуан Әлибай / Мөфтәдин Гыйләҗев - Җырлар белән...

```
Җырлар белән көткән көннәр үтте,
Җырлар белән якты таң атты.
Сагышларга бары җырлар юлдаш,
Җырлар белән күңел канатлы.

Җырлар белән чыктык без юлларга,
Җырлар белән очтык күкләргә.
Тик сайрашып кына, кошлар кебек,
Кавышырга язсын күпләргә.

Җырлар белән, сагынып, очрашабыз,
Чишмә җырын бергә тыңларга.
Очраштырсын безне яңа җырлар,
Яңа юллар, - яңа елларда!
```

# Перевод

### Сафуан Әлибай / Мөфтәдин Гыйләҗев - С песнями...

```
С песнями прошли мы дни ожиданий,
С песнями взошёл рассвет ясный.
В горестях лишь песни спутники,
С песнями душа крылата.

С песнями мы вышли в путь-дорогу,
С песнями взмывали к небесам.
Лишь бы, щебеча, как птицы,
Многим суждено было встретиться.

С песнями, тоскуя, мы встречаемся,
Чтоб вместе слушать песнь родника.
Пусть нас сведут новые песни,
Новые дороги - в новых годах!
```
