# Оригинал

### Шәмсия Җиһангирова / Фасил Әхмәтов - Тик син диеп

```
Тик син диеп,
Янып-көеп,
Үтә лә гомерләрем;
Үткән гомеремне уйлап,
Ярсына күңелләрем.

Тик син диеп
Өзгәләнәм,
Уйланам иртә-кичен;
Ике гомер яшәсәм дә
Янармын синең өчен.

Тик син диеп,
Янып-көеп,
Үтмәс лә гомерләрем -
Очрашырбыз, кавышырбыз,
Ышана күңелләрем.
```

# Перевод

### Шамися Джихангирова / Фасиль Ахматов - Только ты

```
Только ты,
Горя и плача,
Проходит моя жизнь;
Вспоминая прожитые дни,
Рвется сердце надвое.

Только ты
Тревожит меня,
Думаю утром и вечером;
Даже если проживу две жизни,
Все равно буду страдать ради тебя.

Только ты,
Горя и плача,
Не проходит моя жизнь -
Встретимся, сойдемся воедино,
Верит сердце мое.
```
