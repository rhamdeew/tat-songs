# Оригинал

### Сәгыйть Рәмиев / Татар халык көе - Сызла, йөрәк

```
Сызла, сызла, сызла, күңелем!
Сызла бер туктамыйча,
Сызла айлар, сызла еллар,
Сызла төн йокламыйча!

Сызла, сызлап, син мәхәббәт
Ялкынын гөлләр яса,
Мәңге хәсрәт һәм кара төн
Елларын көннәр яса!...
```

# Перевод

### Сәгыйть Рәмиев / Татар халык көе - Сызла, йөрәк - Ной, сердце

```
Ной, ной, ной, душа моя!
Ной, не умолкая,
Ной месяцами, ной годами,
Ной, ночь не зная сна!

Ной, ной - и жар любви
В цветах пускай расцветает,
Вечную скорбь и черную ночь
В дни светлые обращай!...
```
