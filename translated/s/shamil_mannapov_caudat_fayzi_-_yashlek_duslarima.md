# Оригинал

### Шамил Маннапов / Җәүдәт Фәйзи - Яшьлек дусларыма

```
Сөенечләрем, көенечләрем дә бар,
Тормышым да тулы, мул сыман,
Тик шулай да
Яшьлек дуслар белән
Уртак көенечләрне,
Уртак сөенечләрне юксынам.

Торган җирдә җырлы юлларым бар,
Алар да бик якын, уз сыман,
Тик шулай да
Яшьлек дуслар белән
Бергә узган юлны,
Бергә сузган җырны юксынам.

Янәшәмдә яңа танышларым,
Яңа дусларым да күп сыман,
Тик шулай да
Яшьлек дуслыгыннан
Матур дуслык бер дә,
Якын дуслык Җирдә юк сыман...
```

# Перевод

### Шамил Маннапов / Джавдат Файзи - Моим друзьям юности

```
Есть и радости, есть и печали,
Жизнь моя полна, богата,
Но всё равно
С друзьями юности
Общей грусти,
Общей радости я ищу.

На родной земле есть песни моих дорог,
Они тоже близки, словно родные,
Но всё равно
С друзьями юности
Пройденный вместе путь,
Спетую вместе песню я ищу.
Рядом новые знакомые,
Новых друзей много, словно,
Но всё равно
Красивой дружбы, чем дружба юности,
Близкой дружбы на Земле нет...
```
