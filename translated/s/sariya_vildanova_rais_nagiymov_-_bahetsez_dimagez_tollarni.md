# Оригинал

### Сәрия Вилданова / Рәис Нәгыймов - Бәхетсез димәгез толларны

```
Карларда - йолдызлар,
Күңелдә - ак нурлар,
Җаннарда алланып таң ата.
Бәхетсез димәгез, димәгез толларны,
Алар да берәүне ярата.

Карларда - салкынлык,
Йөзләрдә - яктылык,
Төн килә эңгерле кич аша.
Кыланчык димәгез, димәгез толларны,
Шикле уй хисләрне каралта.

Сөюдә - серлелек,
Язмышта - төрлелек,
Адымнар шыгырдап аңлаша.
Хиссез ул димәгез, димәгез толларны,
Аларда сагышлы моң ята.
```

# Перевод

### Сәрия Вилданова / Раис Нагимов - Не называйте вдов несчастными

```
В снегах - звёзды,
В душах - белые лучи,
В сердцах светло рассветает рассвет.
Не называйте вдов несчастными, не называйте их вдовами,
Они тоже кого-то любят.

В снегах - холод,
На лицах - свет,
Ночь приходит сквозь сумеречный вечер.
Не называйте их несчастными, не называйте их вдовами,
Сомнительные мысли темнят чувства.

В любви - тайна,
В судьбе - разнообразие,
Шаги понимают друг друга без слов.
Не говорите, что они бесчувственны, не называйте их вдовами,
В них живёт грустная мелодия.
```
