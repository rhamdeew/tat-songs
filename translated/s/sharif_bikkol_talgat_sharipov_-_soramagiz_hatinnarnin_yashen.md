# Оригинал

### Шәриф Биккол / Тәлгать Шәрипов - Сорамагыз хатыннарның яшен

```
Сорамагыз хатыннарның яшен -
Кузгатмагыз алар күңелен,
Аккан сулар, үткән еллар чиге
Ак томандаторсын күмелеп.

Сорамагыз хатыннарның яшен -
Ирләр булып кала күрегез;
Алар өчен һәр чак җылы сүзгә
Иркен булсын йөрәк түрегез.

Сорамагыз хатыннарның яшен -
Алар безгә серле калсыннар,
Җиткәндә дә гомер төшлегенә,
Ун сигездә була хатыннар.
Сорамагыз хатыннарның яшен -
Ун сигездә килеш калсыннар ...
```

# Перевод

### Шәриф Биккол / Тәлгать Шәрипов - Не спрашивайте о возрасте женщин

```
Не спрашивайте о возрасте женщин -
Не тревожьте их нежную душу,
Грань ушедших лет и вод бегущих
Пусть укроет белый туман.

Не спрашивайте о возрасте женщин -
Оставайтесь мужчинами всегда;
Для них к теплому слову пусть будет
Открыта ваша сердечная дверь.

Не спрашивайте о возрасте женщин -
Пусть они останутся тайной для нас,
И в разгаре земной их дороги
Женщины - в восемнадцать лет.
Не спрашивайте о возрасте женщин -
Пусть красивы в восемнадцать остаются...
```
