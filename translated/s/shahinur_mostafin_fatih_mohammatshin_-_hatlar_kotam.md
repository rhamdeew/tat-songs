# Оригинал

### Шаһинур Мостафин / Фатих Мөхәммәтшин - Хатлар көтәм

```
Ләйсән яңгыр көткән җиргә охшап, иркәм,
Түземсезлек белән синнән хатлар көтәм.

Хатлар белән бергә
Бәхет киләсыман,
Күңел киңлегендә
Кояш көләсыман...

Тургай җырын көтеп, татлы уйга талган
Иген кыры сыман, хатлар көтәм һаман.

Хатлар килеп торса,
Җырлап яши күңел.
Гомер матур уза,
Тормыш итү җиңел.

Чәчәк атар өчен яз килүен көткән
Бөре кебек, синнән хатлар көтәм күптән.

Очкын - күзләремдә,
Ялкын - йөрәгемдә.
Янып китәрмен күк,
Кинәт дөрләрмен дө...
```

# Перевод

### Шаһинур Мостафин / Фатих Мухамметшин - Жду писем

```
Как земля, ждущая дождя в марте, любимая,
С нетерпением жду от тебя писем.

Вместе с письмами
Будто счастье приходит,
В просторах души
Солнце смеется...

Ждущей жаворонка песню, в сладкой мысли растаяв
Как пашня, жду писем всегда.

Если письма приходят,
Душа живет, поет.
Жизнь проходит красиво,
Жить легко.

Как цветок, ждущий прихода весны для расцвета
Как волк, давно жду от тебя писем.

Искры - в глазах моих,
Пламя - в сердце моем.
Сгорю до неба,
Вдруг дрожью пробьет...
```
