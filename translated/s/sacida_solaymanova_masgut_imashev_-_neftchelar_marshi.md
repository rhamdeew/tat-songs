# Оригинал

### sacida_solaymanova_masgut_imashev_ - _neftchelar_marshi

```
Җырлап чыгабыз иртән кырларга,
Кырлар, урманнар уяна.
Көч бар беләктә,
Кодрәт йөрәктә -
Кеше бәхеткә юл яра.

Суда йөзәргә,
Күктә очарга,
Галәм серләрен
Ачарга!

Корыч манара аумас җилләрдә,
Күңел омтыла тирәнгә.
Таулар селкенә,
Йөрәк җилкенә,
Максат бер генә - җиңәргә!

Суда йөзәргә,
Күктә очарга,
Галәм серләрен
Ачарга!

Җырлап чыгабыз иртән кырларга,
Кырлар, урманнар уяна.
Утлар кабына,
Нурлар агыла,
Җыр кала туар елларга.
```

# Перевод

### Саҗидә Сөләйманова / Мәсгуть Имашев - Марш нефтяников

```
С песней выходим мы утром в поля,
Поля и леса просыпаются.
Сила в руках,
Мощь в сердцах -
К счастью людскому путь прокладывается.

По воде плыть,
В небо взлетать,
Тайны вселенной
Открывать!

Стальные вышки не гнутся ветрам,
Душа устремляется в глубь.
Горы дрожат,
Сердца горят -
Цель лишь одна: победить!

По воде плыть,
В небо взлетать,
Тайны вселенной
Открывать!

С песней выходим мы утром в поля,
Поля и леса просыпаются.
Огни загораются,
Свет разливается,
Песня останется грядущим годам.
```
