# Оригинал

### Сөмбел Билалова - Бэхетле тош

```
Тан ата горур тауларны
Ефэк нурларга кумеп
Сайрамачы сандугачым
Серле тошемне булеп?

Имеш язгы кубэлэктэй
Биим жанымны телеп
Син гашыйк булгансын икэн
Минем биюне куреп.

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугачым син булсан икэн

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугач син тугел микэн.

Томан таралгач кырларга
Донья куздэн югала
Кургэн тошлэрем генэ тик
Жырда янгырап кала

Никтер шулай китулэре
Килудэн авыр була
Чегэн кызыдай сихерлэп
Синен янында кала.

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугачым син булсан икэн

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугач син тугел микэн.

Башын эйлэнде тонем хэм
Конем буталды тэмам.
Тошемдэ кургэннэр белэн.
Уянгач горлим, янам.

Таннар атсын сандугачлы
Хыяллар сурелмэсен
Тоште кургэннэрнен
Онгэ бары яхшысы кучсен.

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугачым син булсан икэн

Сандугачлы таннар
Синсез генэ атам иркэм
Тошемне булгэн
Сандугач син тугел микэн.
```

# Перевод

### Сюмбель Билалова - Счастливый сон

```
Рассвет встречает горы гордые,
Шелковым светом одевая,
Мой соловей поющий нежно,
Тайны сна моего делая.

Словно весенняя бабочка,
Душу мою жадно целуя,
Ты влюбился, видно, в танец мой,
Моё движенье созерцая.

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую
Птицу — если б ты был рядом!

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую —
Не соловей ли это ты?

Туман рассеется над полем,
И мир исчезнет из виденья,
Лишь сны привидевшиеся
В песне отзовутся эхом.

Почему так тяжело расставанье,
Чем встреча трудней бывает?
Как цыганская девчонка,
Околдовав, с тобой остаюсь.

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую
Птицу — если б ты был рядом!

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую —
Не соловей ли это ты?

Вскружилась голова моя,
И день смешался совершенно
С приснившимися образами,
Проснувшись, плачу и страдаю.

Пусть наступают рассветы с пеньем,
Пусть грёзы не стираются,
Из всех приснившихся видений
Пусть только лучшие сбываются.

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую
Птицу — если б ты был рядом!

Рассветы с соловьём,
Без тебя встречаю, милый,
Сон мой разделившую —
Не соловей ли это ты?
```
