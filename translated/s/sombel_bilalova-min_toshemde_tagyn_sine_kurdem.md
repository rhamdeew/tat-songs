# Оригинал

### Сөмбел Билалова - Мин тошемдэ тагын сине курдем

```
Буген тоштэ тагын сине курдем,
Килеп утырдын да яныма,
Кайтыр идем, кайтыр идем, диден,
Сине кургэн тэуге языма.

Кутэрелден, китим эле,  дисен,
Сине кургэн тэуге языма.
Гонахларны жуеп, хаталарны куеп,
Кайтыйм эле, дисен, янына.

Кузлэремэ тутырып карадын да,
Буталдым, мин диден, буталдым.
Курми калдым, сизми калдым,
Югалттым, мин сине югалттым.

Эй ерактан ишетелде тагын,
Бэгыркэем, дигэн тавышын,
Бар да бетеп, бар да юкка чыкты,
Калды бары йорэк сагышы.

Кузлэремэ тутырып карадын да,
Буталдым, мин диден, буталдым.
Курми калдым, сизми калдым,
Югалттым, мин сине югалттым.
```

# Перевод

### Сюмбель Билалова - Во сне я снова тебя увидела

```
Сегодня во сне снова тебя увидела,
Пришёл и сел рядом со мною,
"Вернулся бы, вернулся бы", — сказал
К той весне, когда тебя увидела.

"Поднимусь и уйду", — говоришь,
К той весне, когда тебя увидела.
"Смыв грехи, покаявшись в ошибках,
Вернусь к тебе", — говоришь.

В глаза мне пристально смотрел ты,
Запуталась я, говорю, запуталась.
Перестала видеть, чувствовать,
Потеряла, потеряла тебя.

Издалека снова послышался
"Дорогая моя" — голос твой,
Всё кончилось, всё оказалось напрасным,
Осталась лишь сердечная тоска.

В глаза мне пристально смотрел ты,
Запуталась я, говорю, запуталась.
Перестала видеть, чувствовать,
Потеряла, потеряла тебя.
```
