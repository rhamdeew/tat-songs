# Оригинал

### Сания Әхмәтҗанова / Булат Шәрипов - Дөнья гизә сабантуй

```
Яңгырый милли җыр-моңнар,
Чакырып әллә кайдан.
Һәр милләтне якын итә
Безнең кунакчыл мәйдан.

Сабантуй! Сабантуй!
Чын татарча милли туй!
Бердәмлек үрнәге булып,
Дөнья гизә сабантуй!

Сабантуйда түрдә булсын
Ата-баба йоласы;
Гореф-гадәтләрне күреп
Үссен заман баласы.

Сабантуй! Сабантуй!
Чын татарча милли туй!
Бердәмлек үрнәге булып,
Дөнья гизә сабантуй!

Кыр батырлары көч бирә
Сабантуй батырына.
Очырыйк ак күгәрченнәр
Зәңгәр күк чатырына.

Сабантуй! Сабантуй!
Чын татарча милли туй!
Бердәмлек үрнәге булып,
Дөнья гизә сабантуй!
```

# Перевод

### Сания Әхмәтҗанова / Булат Шарипов - Сабантуй для всего мира

```
Звучат народные песни,
Созывая отовсюду.
Каждый народ делает ближе
Наш гостеприимный двор.

Сабантуй! Сабантуй!
Настоящий татарский народный праздник!
Образ единства,
Сабантуй для всего мира!

Пусть в Сабантуй будет в очаге
Обычай предков;
Пусть современные дети,
Увидев традиции, растут.

Сабантуй! Сабантуй!
Настоящий татарский народный праздник!
Образ единства,
Сабантуй для всего мира!

Силы дают селянин-богатырь
Сабантуевскому богатырю.
Пусть белые голуби летают
К синему небу.

Сабантуй! Сабантуй!
Настоящий татарский народный праздник!
Образ единства,
Сабантуй для всего мира!
```
