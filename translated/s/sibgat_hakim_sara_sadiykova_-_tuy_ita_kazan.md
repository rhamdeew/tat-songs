# Оригинал

### Сибгат Хәким / Сара Садыкова - Туй итә Казан

```
Батыр һәйкәле алдына,
Өзеп алып хисләрдән,
Салалар туй чәчәкләре
Яңа өйләнешкәннәр.

Киләчәк бөек эшләргә
Үзен багышлый яшьлек.
Уйлана ерак чорларга
Узып барышлый яшьлек.

Гөр килеп туй итә яшьлек,
Үзе башкода Казан,
Ленталар белән бизәлеп,
Кая ашкына Казан?

Киләчәк якты елларның
Каршына чаба Казан.
```

# Перевод

### Сибгат Хаким / Сара Садыкова - Гуляет Казань

```
К памятнику Батыра,
Пронзая чувства насквозь,
Вручают свадебные цветы
Тем, что связали судьбы.

Великим делам грядущим
Посвящает себя молодость.
Задумывается о временах далёких,
Пронощаясь, молодость.

Громко празднует молодость,
В самой главной Казани,
Украсившись лентами,
Куда стремится Казань?

Навстречу ярким годам грядущим
Мчится Казань.
```
