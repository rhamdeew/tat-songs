# Оригинал

### Шәехзадә Бабич / Айрат Кобагышев - Халкым өчен (икенче вариант)

```
Ап-ак алтын җырларымны
Җырламыймын данлык өчен.
Җырлыймын тик илем өчен,
Үз туган(дай) халкым өчен.

Саф көмештәй җырларымны
Җырламыймын алтын өчен.
Җырлыймын тик саф көмештәй
Саф йөрәкле халкым өчен.

Чәчәк төсле җырларымны
Җырламыймын зәвкем өчен.
Җырлыймын тик чәчәк кебек
Кызларга бай халкым өчен.

Дәртле кайнар җырларымны
Җырламыймын дәртем өчен.
Җырлыймын тик мәңге шат,
Дәртле, көләч халкым өчен.
```

# Перевод

### Шаехзадэ Бабич / Айрат Кобагишев - За мой народ (второй вариант)

```
Чисто-золотые песни мои
Не пою я для славы.
Пою я только для красоты,
Для своего родного народа.

Чистые, как серебро, песни мои
Не пою я ради золота.
Пою я только чистые, как серебро,
Для народа с чистым сердцем.

Цветные, как цветы, песни мои
Не пою я для удовольствия.
Пою я только, как цветок,
Для народа, богатого девушками.

Страстные, горячие песни мои
Не пою я ради страсти.
Пою я только для вечно радостного,
Страстного, смеющегося народа.
```
