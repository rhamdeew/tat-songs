# Оригинал

### Светлана Шәрипова / Роберт Тимербаев - Синең өчен (өченче вариант)

```
Яңгыр ява иде көзге кичтә,
Мин очраттым сине, кадерлем.
Сулы чәчең, моңсу карашыңны
Бүгенгедәй ачык хәтерлим.
Шул мизгелдән минем йөрәк яусыз
Бирелде тик сиңа әсиргә.
Синең мәхәббәтсез сулырмын дип
Секунд саен күңел ачына.

Синең өчен туганмын мин дөнъяга
Синең өчен күктә йолдыз атыла
Синең өчен.
Син булганга кояш яктырта көн дә,
Син булганга ай-йолдызлар бар төндә,
Син булганга.

Сине яулар өчен җыеп бирәм
Җир шарының мең төр гөлләрен.
Җил булырмын сине назлар өчен,
Синең өчен яшәп үләрмен.
Таңнарның кабул ит алсулыгын,
Зәңгәрлеген чиксез даръяның.
Бар галәмнең йолдызларын алып,
Чибәркәем, сиңа барамын.

Синең өчен туганмын мин дөнъяга
Синең өчен күктә йолдыз атыла
Синең өчен.
Син булганга кояш яктырта көн дә,
Син булганга ай-йолдызлар бар төндә,
Син булганга.
```

# Перевод

### Светлана Шәрипова / Роберт Тимербаев - Для тебя (третий вариант)

```
Лил осенний дождь в вечерней тиши,
И тебя я встретил, дорогая.
Твои мокрые волосы, твой печальный взгляд
Я сегодня помню, будто было вчера.
С той минуты сердце, без боя, покорно
Я отдал тебе, в плен твой один.
Без твоей любви я увяну -
Каждую секунду тоска сжимает грудь.

Для тебя я в этот мир рожден,
Для тебя в небесах вспыхнет звезда,
Для тебя.
Потому что ты - и солнце светит днем,
Потому что ты - и ночь хранит звезды,
Потому что ты.

Чтобы тебя завоевать, соберу
Тысячи цветов земного шара.
Стану ветром, чтоб тебя ласкать,
Для тебя я жить и умирать готов.
Прими алость рассветов,
Синеву бескрайнего моря.
Взяв звезды всей вселенной,
Красавица моя, к тебе иду.

Для тебя я в этот мир рожден,
Для тебя в небесах вспыхнет звезда,
Для тебя.
Потому что ты - и солнце светит днем,
Потому что ты - и ночь хранит звезды,
Потому что ты.
```
