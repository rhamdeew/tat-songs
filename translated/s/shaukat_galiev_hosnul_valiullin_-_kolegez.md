# Оригинал

### Шәүкәт Галиев / Хөснул Вәлиуллин - Көлегез

```
Кояш бөтен кешегә дә
Бертигез көлеп карый,
Көлү бит ул, гөнаһ түгел,
Начальникка да ярый.

Көлегез,
Көлегез,
Файдасы күп, белегез!

Сау-сәламәт, таза була,
Шаярган, көлгән кеше,
Көлеп эче авыртса да,
Авыртмый бүтән төше!

Көлегез,
Көлегез,
Файдасы күп, белегез!
```

# Перевод

### Шәүкәт Галиев / Хөснул Вәлиуллин - Смейтесь

```
Солнце всем глядит в лицо,
С одинаковой улыбкой -
Смех не грех, он и начальству
Подойдет вполне, привыкнет.

Смейтесь,
Смейтесь,
Знайте: в этом польза есть!

Кто смеется и шутит,
Будет крепким, будет здравым;
Даже если бок заболел -
Все остальное не накажет.

Смейтесь,
Смейтесь,
Знайте: в этом польза есть!
```
