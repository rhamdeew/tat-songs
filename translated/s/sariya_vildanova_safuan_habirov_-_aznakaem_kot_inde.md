# Оригинал

### Сәрия Вилданова / Сафуан Хәбиров - Азнакаем, көт инде

```
Салкын җилләр каурыемны каерган иде,
Каты язмыш туган яктан аерган иде.

Туган ягым - Азнакаем,
Көт инде.
Мин барыбер гел синеке
Бит инде.

Язмышларым җиңеләйде, ярам төзәлде.
Туган якка алып кайта гомер җилләре.

Туган ягым - Азнакаем,
Көт инде.
Мин барыбер гел синеке
Бит инде.

Яз тургаең булып кайтам, кабул ит инде,
Чит җирләрнең бер үксезе булмам бит инде.

Туган ягым - Азнакаем,
Көт инде.
Мин барыбер гел синеке
Бит инде.
```

# Перевод

### Сария Вилданова / Сафуан Хабиров - Азнакаево, жди же

```
Холодные ветры обжигали мою грудь,
Суровая судьба оторвала от родного края.

Родной мой край - Азнакаево,
Жди же.
Я всё равно вечно твоя,
Ведь правда.

Судьбы мои стали легче, жизнь наладилась.
Годы жизни несут обратно к родному краю.

Родной мой край - Азнакаево,
Жди же.
Я всё равно вечно твоя,
Ведь правда.

Вернусь твоей весной, прими же меня,
Чужих земель я не сирота ведь, правда.

Родной мой край - Азнакаево,
Жди же.
Я всё равно вечно твоя,
Ведь правда.
```
