# Оригинал

### Шәмсия Җиһангирова / Илдар Мәгъдәнов - Гөлләр алдым (беренче вариант)

```
Гөлләр алдым, суга салдым,
Агып китте гөлләрем.
Аккан сулар кебек яшьлек
Үтәр диеп белмәдем...

Гөлләр алдым, суга салдым,
Тотып адылар микән?
Гөлләр кебек яшьлегемне
Күреп калдылар микән?

Кире барсам, гөлләр алсам,
Суларга салмас идем.
Яшьлегемнең кадерләрен
Белмичә калмас идем.
```

# Перевод

### Шамсия Джихангирова / Ильдар Мегданов - Цветы сорвала (первый вариант)

```
Цветы сорвала, в воду опустила,
Уплыли мои цветы прочь.
Как текучая вода, молодость
Уйдёт - не знала я...

Цветы сорвала, в воду опустила -
Не зацепятся ли за что?
Не увидели ли, как мою молодость
Как цветы утратил я?

Вернись обратно, цветы бы собрала,
В воду не бросила б их.
Цену моей молодости
Не знала б - не потеряла б.
```
