# Оригинал

### Шәриф Биккол / Нур Даутов - Онытырга сине көчем юк ... (икенче вариант)

```
Кайдан килгән сиңа,
Кайдан килгән
Бу чаклы да яман саранлык?
Җанкаемны фида кылганда да,
Йөрәгеңне яулар, йөрәгеңне яулар,
Йөрәгеңне яулар чарам юк.

Күңелемә кырау була алмас,
Карашларың өргән салкынлык.
Син сөймәскә мине иреклесең,
Сөйдермәскә исә, сөйдермәскә исә,
Сөйдермәскә исә хакым юк.

Яздырсаңда соңғы өметемнән,
Ачуым юк сиңа, үсем юк.
Гомер буе көтер түземем бар,
Онытырга сине, онытырга сине,
Онытырга сине көчем юк...
```

# Перевод

### Шариф Биккол / Нур Даутов - Забыть тебя нет сил... (вторая версия)

```
Откуда в тебе,
Откуда в тебе
Такая злая скупость взялась?
Даже когда душу отдаю,
Твое сердце завоевать, твое сердце завоевать,
Твое сердце завоевать - нет у меня сил.

Иней в душе не станет,
Холод, что веет из твоих глаз.
Ты вправе меня не любить,
А заставить полюбить, заставить полюбить,
Заставить полюбить - нет у меня прав.

Хоть лишишь последней надежды,
Нет на тебя обиды, нет злобы.
Терпения на целую жизнь хватит,
А забыть тебя, забыть тебя,
Забыть тебя - нет у меня сил...
```
