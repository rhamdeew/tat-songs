# Оригинал

### Шәриф Биккол / Ризван Хәкимов - Бозма вәгъдәләреңне

```
Сөюләрем көчле,
Һәтәр яшен төсле,
Сак бул, син, ут белән уйнама!
Вәгъдәләрең бозсаң,
Ятларга кул сузсаң,
«Гафү итәр бу!» - дип уйлама.

«Тиң, түгел бит, ул, - дип, -
Миңа гына булл!» - дип,
Хатлар язып сине юл чыкмам,
Төнең тәрәз чиртеп,
«Күрим әле бер!» - дип,
Чакырырсың,
Ә мин, мин чыкмам.
Төнең тәрәз чиртеп,
«Күрим әле бер!» - дип,
Чакырырсың,
Ә мин, мин чыкмам.

Мен газапка түзеп,
Мине дөнья гизеп,
Эзләрсең син,
Ә мин табылмам,
Мин табылмам көтмә,
Синең өчен күктә,
Балкыр йолдыз булып кабынмам.
Мин табылмам көтмә,
Синең өчен күктә,
Балкыр йолдыз булып кабынмам.
```

# Перевод

### Шариф Биккол / Ризван Хәкимов - Не нарушай своих обещаний

```
Моя любовь так сильна,
Как грозная молния,
Осторожнее, не играй с огнём!
Если ты нарушишь обещания,
Протянушь руку другим,
Не думай: «Всё будет прощено!»

«Тебе не подходит, - напишу, -
Только для меня!» -
Письма пишу, но к тебе не выйду,
В окно твоё постучишь,
«Увидимся хоть раз!» - позовёшь,
А я, я не выйду.
В окно твоё постучишь,
«Увидимся хоть раз!» - позовёшь,
А я, я не выйду.

Сквозь муки терпя,
Меня мир обходя,
Ищешь ты, но меня не найдёшь,
Меня не найдёшь, не жди,
Для тебя на небе
Сияющей звездой не зажгусь.
Меня не найдёшь, не жди,
Для тебя на небе
Сияющей звездой не зажгусь.
```
