# Оригинал

### Саҗидә Сөләйманова / Сәүбән Чуганаев - Ах, җаныем, Бибисара!

```
Сүс дилбегә, юкә чана,
Алып булмый акчага.
Сырхауларга дәвалар бар,
Сагнуларга юк чара.

Ах, җаныем, Бибисара,
Сагнуларга юк чара.

Яна күмер, уза гомер,
Узган гомер бичара.
Узган гомер берни түгел,
Бәхтең җуйсаң ни чара.

Ах, җаныем, Бибисара,
Бәхтең җуйсаң ни чара.

Карлыгачым муеның ала,
Канат очларың кара.
Без йөрибез кавышалмыйча,
Яшь гомер үтеп бара.

Ах, җаныем, Бибисара,
Яшь гомер үтеп бара.
```

# Перевод

### Саҗидә Сөләйманова / Сәүбән Чуганаев - Ах, душа моя, Бибисара!

```
Шлея плетёная, липовые сани -
Не купишь это за деньги.
Больным найдутся лекарства,
Тоскующим нет спасенья.

Ах, душа моя, Бибисара,
Тоскующим нет спасенья.

Уголь горит, проходит жизнь,
Жизнь прошедшая - сиротина.
Прошедшие годы - ничто,
Когда потеряно счастье.

Ах, душа моя, Бибисара,
Когда потеряно счастье.

Ласточка моя, шея бела,
Концы крыльев у тебя черны.
Мы ходим, не встретившись,
Юная жизнь уходит мимо.

Ах, душа моя, Бибисара,
Юная жизнь уходит мимо.
```
