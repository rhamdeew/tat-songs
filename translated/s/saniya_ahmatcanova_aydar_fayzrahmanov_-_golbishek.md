# Оригинал

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Гөлбишек

```
Сайлап алып матур төсләр
Күңелем түрләреннән,
Үрим әле гүзәл бишек
Сөюем гөлләреннән.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.

Бәйләм-бәйләм гөлләр китер,
Үрелсен хисләребез.
Аллы-гөлле хыялларга
Күмелсен кичләребез.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.

Берәүгә дә бирмим сине,
Тапмадым югалтырга.
Ирек куймам икәү үргән
Хисләрне чуалтырга.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.
```

```

Из оттенков моей души,
Я плету ещё прекрасную колыбель
Из цветов моей любви.

Припев:
Будем качаться мы вдвоём
В колыбели любви.
Любить меня - только любить
Оставляет молодой меня.

Свяжу-свяжу цветов принесёт,
Пусть вьются наши чувства.
В цветочно-яркую мечту
Пусть превратятся наши ночи.

Не нашла потерять.

Чувства мутить.
```
