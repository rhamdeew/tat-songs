# Оригинал

### Сәет Шәкүр / Әнвәр Бакиров - Таң (беренче вариант)

```
Ерак тау артыннан күтәрелеп
Таң билгесе сызыла офыктан.
Куе караңгылык сыегая,
Йомшак җилләр исә котыптан.

Уяна да татлы йокысыннан
Таңны мактап сайрый сандугач.
Сәхрәләрдә, яшел болыннарда
Эш иртәсе туа, таң тугач.

Таң көч бирә эшчән беләкләргә,
Таң күтәрә рухны, күңелне.
Таң, җыр суза, биек зәңгәр күккә
Таң бизәкли безнең гомерне.
```

# Перевод

### Сәит Шәкүр / Әнвәр Бакиров - Рассвет (первый вариант)

```
Восходя из-за далёких гор,
Заря рисуется на горизонте.
Густая тьма редеет,
Мягкие ветры дуют с полюса.

Просыпается от сладкого сна,
Зарю воспевая поёт соловей.
На рассветах, на зелёных лугах,
Ранняя работа начинается, когда заря родится.

Заря даёт силу трудящимся рукам,
Заря поднимает дух, душу.
Заря, песню тянуть, в высокое синее небо,
Заря украшает нашу жизнь.
```
