# Оригинал

### Сибгат Хәким / Хөснул Вәлиуллин - Күңелем Ленин белән сөйләшә

```
Казан әле, Казан уянмаган,
Яфрак белән яфрак серләшә.
Йөрим таңда Ленин урамында,
Күңелем Ленин белән сөйләшә.

Бу чәчәкләр җирдә бик күп булыр,
Салкын җилләр әгәр тимәсә...
Уйларымны төбәп киләчәккә,
Күңелем Ленин белән сөйләшә.

Без узарга тиеш, әй, юллар бар,
Зур эшләргә чакырып ил дәшә.
Шулай юлга чыгар алларыннан
Күңелем Ленин белән сөйләшә.

Янар өчен каян килсен хисләр,
Бүгенгене йөрәк сөймәсә.
Казан әле, Казан уянмаган,
Күңелем Ленин белән сөйләшә.
```

# Перевод

### Сибгат Хәким / Хөснул Вәлиуллин - Моя душа говорит с Лениным

```
Казань еще спит, Казань не проснулась,
Листок с листком ведет свой тихий разговор.
По утру иду по улице Ленина,
Моя душа с Лениным говорит.

Цветов на этой земле будет много,
Если холодный ветер их не тронет...
Устремляя мысли в грядущее,
Моя душа с Лениным говорит.

Нам надлежит пройти - есть дороги,
Страна зовет на большие дела.
И так, перед выходом в путь,
Моя душа с Лениным говорит.

Откуда взяться пламени чувствам,
Если сердцу не мило сегодня?
Казань еще спит, Казань не проснулась,
Моя душа с Лениным говорит.
```
