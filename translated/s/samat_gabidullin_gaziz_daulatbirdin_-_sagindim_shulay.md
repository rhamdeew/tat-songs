# Оригинал

### Самат Габидуллин / Газиз Дәүләтбирдин - Сагындым Шулай

```
Еракта, офыкта
Яшен утлары, яшен утлары.
Еракта, шул якта
Минем дусларым, минем дусларым.

Еракта, офыкта
Кошлар тавышы, кошлар тавышы.
Күңелдә шул чакта
Сагыну (дуслар) сагышы, сагыну (дуслар) сагышы.

Кошларны сагынып,
Шомыртлар шаулый, шомыртлар шаулый.
Мин дә бит дусларны
Сагындым шулай, сагындым шулай.
```

# Перевод

### Самат Габидуллин / Газиз Дәүләтбирдин - Так соскучился

```
Вдали, на горизонте
Огни молний, огни молний.
Вдали, на той стороне
Мои друзья, мои друзья.

Вдали, на горизонте
Голоса птиц, голоса птиц.
И в сердце в эту пору
Тоска по (друзьям), тоска по (друзьям).

Скучая по птицам,
Шумят черёмухи, шумят черёмухи.
И я по друзьям своим
Так соскучился, так соскучился.
```
