# Оригинал

### Сибгат Хәким / Сара Садыкова - Бишек җыры (өченче вариант)

```
Шаулар елгалар, елгада сулар,
Тиздән чәчәкләр, чәчәкләр чыгар.
Безнең бакчага сандугач килер,
Килер дә синең исмеңне сорар,
Әлли-бәлли, әлли-бәлли,
Килер дә синең исмеңне сорар.

Таң ату белән, өйрәтер беркөн,
Өйрәтер моңлы матур яз көен.
Сандугач инде нәкъ ярты юлда,
Йокла, чибәрем, йокла син минем,
Әлли-бәлли, әлли-бәлли,
Йокла, чибәрем, йокла син минем.
```

# Перевод

### Сибгат Хаким / Сара Садыкова - Колыбельная (третий вариант)

```
Шумят ручьи, в ручьях вода,
Скоро расцветут цветы, цветы.
К нам в садик соловей прилетит,
Прилетит и спросит твоё имя,
Баю-бай, баю-бай,
Прилетит и спросит твоё имя.

С рассветом однажды научит он,
Научит нежной весенней песне.
Соловей уже на полпути,
Спи, моя красавица, спи, родная,
Баю-бай, баю-бай,
Спи, моя красавица, спи, родная.
```
