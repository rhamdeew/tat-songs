# Оригинал

### Себер татарлары халык җыры - Себер татарлары җыры

```
Себер ягы мал ягы ла,
Казан ягы тал ягы.
Казаннарга кайтмас идем,
Дусларның туган ягы.

Бакчадагы алмагачка
Тайлар бәйләсәм икән.
Егет үзе картаймаса,
Дөнья картайта микән?

Тау башында таш тегермән,
Яшь бодай тарта микән?
Сез дусларым белән бергә
Мәңге яшәсәң икән.
```

# Перевод

### Народная песня сибирских татар - Песня сибирских татар

```
Сибирский край - край табунов,
Казанский - ивный край.
В Казань бы не вернулся я -
Здесь отчий край друзей.

В саду к яблоням я бы привязал
Резвых жеребят.
Если сам не старишься, джигит,
Разве мир не старит нас?

На горе стоит жернов каменный,
Молодую пшеницу молет ли?
О, если б с вами, друзья мои,
Вместе вечно жить нам.
```
