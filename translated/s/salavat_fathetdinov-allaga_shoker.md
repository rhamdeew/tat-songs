# Оригинал

### Салават Фатхетдинов - Аллага шокер

```
Бер максатсыз, гел далага
Узмасын гомер.
Исэнмесез, яшибез без,
Аллага шокер!

Кайгы белэн, шатлык белэн
Уза бу гомер.
Без бит бергэ, шунсы шатлык,
Аллага шокер!

Алла бирсэ, жырларбыз без,
Узганчы гомер.
Исэнмесез, жырлыйбыз без,
Аллага шокер!
```

# Перевод

### Салават Фатхетдинов - Благодарность Аллаху

```
Не без цели, не впустую
Пусть проходит наша жизнь.
Будем здравы, будем живы,
Аллаху хвала!

С горем вместе, с радостью
Проходит эта жизнь.
Мы ведь вместе, какая радость,
Аллаху хвала!

Если Аллах даст, будем петь мы,
Пока длится жизнь.
Будем здравы, будем петь мы,
Аллаху хвала!
```
