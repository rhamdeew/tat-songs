# Оригинал

### Шәмсия Җиһангирова / Илфат Дәүләтшин - Каз өмәсе (икенче вариант)

```
Авылларда каз өмәсе гөрли,
Юл буена канат сипкәннәр.
Канат өсләренә баса-баса
Кызлар каз юарга киткәннәр.

Әллә нинди генә сәер чак бу,
Үзе моңсу, үзе күңелле.
Каз гомере кире кайтмый торган
Яшьлек кебек кыска гомерле.

Җырлый-җырлый кызлар каз юалар,
Юл буена канат сипкәннәр.
Туй-табыннарына кызлар чыгар,
Кызлар каз юарга киткәннәр.

Авылларда каз өмәсе гөрли,
Юл буена канат сипкәннәр.
Канат өсләренә баса-баса
Кызлар каз юарга киткәннәр.
Авылларда каз өмәсе гөрли.
```

# Перевод

### Шәмсия Җиһангирова / Илфат Дәүләтшин - Казовая толока (второй вариант)

```
По деревням гудит казовая толока,
К обочине крыльями устлали дорогу.
По крыльям, ступая, тихонько шурша,
Девчонки ушли гусей отмывать.

Странная, редкая это пора -
И грустна, и веселая сразу.
Жизнь гуся, как юность, не возвращаясь,
Так коротка и так невозвратима.

Песни поют - гусей девчонки моют,
К обочине крыльями устлали дорогу.
Скоро к свадебным столам девчонки выйдут,
Девчонки ушли гусей отмывать.

По деревням гудит казовая толока,
К обочине крыльями устлали дорогу.
По крыльям, ступая, тихонько шурша,
Девчонки ушли гусей отмывать.
По деревням гудит казовая толока.
```
