# Оригинал

### Сафуан Әлибай / Роберт Газизов - Коен

```
Ярсып-ярсып, өерелеп,
Коен уйный, коен.
Калсаң коен уртасында,
Кыен икән, кыен.

Юлларыма гөлләр үскән,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Моңаямын шулай?

Шомнар салып, кемдер ярсып
Үтте сымак җырлап.
Уйларымны өерелтеп
Алып китте ерак.

Юлларыма гөлләр шыткан,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Ямь тапмыймын шулай?

Күңелемдә өерелеп
Коен уйный, коен.
Тыныр иде, син янда юк / Тындырырга, син янда юк, /
Кыен миңа, кыен.

Юлларыма гөлләр үскән,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Ямь тапмыймын шулай?
```

# Перевод

### Сафуан Әлибай / Роберт Газизов - Буря

```
Вихрем, яростно, кружась,
Буря пляшет, буря.
Если в самый центр попадёшь -
Тяжко, правда, тяжко.

На моих путях цветы,
Птичий звон разлёгся.
Только я зачем-то всё
Так тоскую снова?

Смуту сея, кто-то, в запале,
Прошёл, будто песней.
Закружил мои раздумья,
Унёс далеко.

На моих путях цветы,
Птичий звон разлёгся.
Только я зачем-то всё
Не найду ямь снова?

В сердце, вихрем, кружась,
Буря пляшет, буря.
Утихла б - тебя со мной нет / Унять её - тебя со мной нет, /
Тяжко мне, о тяжко.

На моих путях цветы,
Птичий звон разлёгся.
Только я зачем-то всё
Не найду ямь снова?
```
