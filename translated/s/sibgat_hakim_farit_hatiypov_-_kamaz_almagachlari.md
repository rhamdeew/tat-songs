# Оригинал

### Сибгат Хәким / Фәрит Хатыйпов - КамАЗ алмагачлары

```
Сез безнең узган яшьлеккә,
Бу җирда алмаш бары,
Сездә дәвамы яшьлекнең,
КамАЗ алмагачлары.

Киләсез урамнар буйлап,
Таңнарны каршыларга,
Сезнең юл КамАЗлар аша
Гасырдан гасырларга.

Сез безнең узган яшьлеккә
Бу җирда алмаш бары,
Якты хәерле юл сезгә,
КамАЗ алмагачлары.
```

# Перевод

### Сибгат Хаким / Фарит Хатипов - Яблони КамАЗа

```
Вы - наша прошедшая молодость,
На этой земле замена всему,
В вас продолжение молодости,
Яблони КамАЗа.

Идёте вы по улицам,
Встречать рассветы,
Ваш путь через КамАЗы
Из века в век.

Вы - наша прошедшая молодость,
На этой земле замена всему,
Светлого пути вам,
Яблони КамАЗа.
```
