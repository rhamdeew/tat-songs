# Оригинал

### Сафуан Әлибай / Рим Хәсәнов - Аталар

```
Аталар, аталар, аталар...
Илемнең, җиремнең терәге;
Без тыныч, без исән, ил исән,
Типкәндә аталар йөрәге.

Аталар, аталар, аталар...
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар...
Көтәбез сагынып кайтканны,
Аталар китерә түгелме
Учында күтәреп ал таңны?!

Аталар, аталар, аталар...
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар...
Күпләре әйләнеп кайтмыйлар...
Кайтканнар яралы көенчә
Сафлардан калмыйлар, атлыйлар.

Аталар, аталар, аталар...
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар...
Саклыйлар, данлыйлар Ватанны.
Сабыйлар көтсеннәр атаны,
Көтсеннәр тик эштән кайтканны!

Аталар, аталар, аталар...
Гел алда, гел алда атлыйлар.
```

# Перевод

### Сафуан Әлибай / Рим Хәсәнов - Отцы

```
Отцы, отцы, отцы...
Опора страны, опора земли;
Мы в мире живём, мы живы, жива страна,
Пока бьются отцовские сердца.

Отцы, отцы, отцы...
Всегда впереди, всегда впереди идут.

Отцы, отцы, отцы...
Мы ждём их, тоскуя по возвращению,
Не отцы ли приносят
На ладонях поднимая рассвет?!

Отцы, отцы, отцы...
Всегда впереди, всегда впереди идут.

Отцы, отцы, отцы...
Многие не вернутся назад...
Вернувшиеся - раненые,
Не выпадают из строя, идут.

Отцы, отцы, отцы...
Всегда впереди, всегда впереди идут.

Отцы, отцы, отцы...
Берегут и славят Родину.
Пусть малыши ждут отца,
Ждут - но лишь с работы вернувшегося!

Отцы, отцы, отцы...
Всегда впереди, всегда впереди идут.
```
