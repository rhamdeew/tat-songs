# Оригинал

### Сания Әхмәтҗанова / Фәрит Әхмәтов - Туган авылым Түнтәр

```
Борма-борма Түнтәр юлы
Еракка алып китә.
Читләргә чыгып киткәннәр
Юксынып гомер итә.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.

Хәтерем сукмакларыннан
Үткәннәремә кайттым.
Сагышларыма дәваны,
Түнтәрем, синдә таптым.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.

Баш урманның аланнарын
Каплаган куе үлән.
Бер җырлап утырыйк әле
Яшьлек дусларым белән.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.
Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.
```

# Перевод

### Сания Әхмәтҗанова / Фарит Ахматов - Родное село Тюнтэр

```
Вьётся-вьётся дорога в Тюнтэр,
В даль уносит меня.
Кто ушёл в чужие края,
С тоской живёт день ото дня.

Святых родников твоих
Воды я не напился.
Слушая трели соловья,
Улицы вспоминались.

По тропам моей памяти
В прошлое я вернулся.
Лекарство моей тоске,
Тюнтэр, в тебе нашёлся.

Святых родников твоих
Воды я не напился.
Слушая трели соловья,
Улицы вспоминались.

Поляны большого леса
Покрыла густая трава.
Давай посидим, распоёмся
С друзьями юности сперва.

Святых родников твоих
Воды я не напился.
Слушая трели соловья,
Улицы вспоминались.
Святых родников твоих
Воды я не напился.
Слушая трели соловья,
Улицы вспоминались.
```
