# Оригинал

### Самат Шакир / Әнвәр Бакиров - Төнбоеклы күл буенда

```
Төнбоеклы күл буенда
Саклана безнең эзләр.
Текәп карыйлар шикелле
Миңа бик таныш күзләр.

Күзләрен йомган чәчәкләр,
Чык төшкән өсләренә.
Бергә йөргән вакытларым
Төшә шул исләремә...

Төнбоеклы күл буенда
Икәү серләшкән идек.
Мәңге бергә булырбыз, дип,
Өзеп сөйләшкән идек.

Төнбоеклы күл өстендә
Йолдызлар җем-җем итә...
Яшьли сөйгән яр онтылмый.
Үкенеп гомер үтә.
```

# Перевод

### Самат Шакир / Энвер Бакиров - У озера с кувшинками

```
У озера с кувшинками
Хранятся наши следы.
Смотрят пристально, как будто,
На меня родные глаза.

Цветы сомкнули веки,
На них легла роса.
Вспоминаются мгновенья,
Когда мы рядом были...

У озера с кувшинками
Мы вдвоем делились тайной.
Говорили, что навеки,
Верили словам прерывистым.

Над озером с кувшинками
Звезды тихо мерцают...
Юную любовь не забыть,
С сожаленьем жизнь проходит.
```
