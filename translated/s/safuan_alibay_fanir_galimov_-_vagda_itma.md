# Оригинал

### Сафуан Әлибай / Фәнир Галимов - Вәгъдә итмә

```
Миңа йолдыз вәгъдә итмә,
Омтылсаң да зәңгәр күккә.
Атылыр да сүнәр йолдыз,
Эзсез калыр мәңгелеккә.

Миңа ташкын вәгъдә итмә,
Дулкыннарың чайкый-чайкый.
Күбекләнеп боз актаргач,
Ул да тыныр, кабат чыкмый.

Миңа таулар вәгъдә итмә,
Янар таулар җирдә сирәк.
Юлдаш булсын тау күтәрер,
Синең генә тугры йөрәк.
```

# Перевод

### Сафуан Әлибай / Фәнир Галимов - Не обещай

```
Не обещай мне звезду,
Хоть тянешься к лазурному небу.
Вспыхнет и погаснет звезда,
Бесследно останется в вечности.

Не обещай мне поток,
Волны его колышут-колышат.
Вспенившись и разбив лёд,
Он тоже утихнет, не появится вновь.

Не обещай мне горы,
Пылающие горы на земле редки.
Пусть спутником будет гора, что поднимет,
Твоё только верное сердце.
```
