# Оригинал

### Шәйхи Маннур / Мансур Мозаффаров - Казан каласы

```
Казан каласы таш кала,
Безнең сөйгән башкала.
Күңелләрне дәртләндергән
Җырлар шунда башлана.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.

Агыйделдән, Ык, Шөгердән,
Зәй, Зөядән Казанга
Килә яшьләр, канатланып,
Кошлар кебек язларда.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.

Илләр гизеп, сулар кичеп
Йөрсә дә егет һаман,
Кайтып килә ул яңадан -
Сагындыра шул Казан.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.
```

# Перевод

### Шяхи Маннур / Мансур Мозаффаров - Город Казань

```
Город Казань - каменный город,
Наша любимая столица.
Песни, что воспламеняют сердца,
Начинаются там.

И-их, город Казань,
Среди цветочных садов!
Для молодого сердца,
Для пламенной страсти
В тебе исцеление находится.

С Агидели, с Ика, с Шешмы,
С Зая, с Зои - в Казань
Приезжают молодые люди, расправив крылья,
Как птицы весной.

И-их, город Казань,
Среди цветочных садов!
Для молодого сердца,
Для пламенной страсти
В тебе исцеление находится.

Страны обойдя, воды переплыв,
Сколько бы ни странствовал юноша,
Возвращается он снова -
Казань томит его по себе.

И-их, город Казань,
Среди цветочных садов!
Для молодого сердца,
Для пламенной страсти
В тебе исцеление находится.
```
