# Оригинал

### Сәкинә Хәйруллина / Марат Галимов - Син булмасаң (беренче вариант)

```
Син булмасаң кемне сөяр идем,
Кем юлына гөлләр сибәр идем.
Мин кыюсыз яннарыңа килдем,
Бүгенгедәй син бик чибәр идең.

Әле ярый сөю безне күргән,
Сиңа мине, миңа сине биргән.
Язмышларны уртак итеп үргән,
Пар юлыгыз шушы булыр дигән.

Сөю безгә кунак кына түгел,
Яулап алды өебезнең түрен.
Сабырлыкка, җылы назга төреп,
Бүләк итте безгә бәхет серен.

Әле ярый сөю безне күргән,
Сиңа мине, миңа сине биргән.
Язмышларны уртак итеп үргән,
Пар юлыгыз шушы булыр дигән.
Әле ярый сөю безне күргән,
Сиңа мине, миңа сине биргән.
Язмышларны уртак итеп үргән,
Пар юлыгыз шушы булыр дигән...
```
