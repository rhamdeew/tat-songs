# Оригинал

### Сөмбел Билалова - Кайчан гына

```
Кайчан гына килерсен син
Кузлэренэ генэ карар идем
Бер суз эйтмэсэн кузлэреннэн
Анлар идем сине анлар идем.

Кунелемдэ мон дингезе
Дулкын гына талгын гына
Килер эле диеп котэ-котэ
Гомерем утэ ялгыз гына

Кайчан гына килерсен син
Газап утларында янмас идем
Якыннрын жил давылга биргэн
Учак коннэренэ калмас идем

Кунелемдэ мон дингезе
Дулкын гына талгын гына
Килер эле диеп котэ-котэ
Гомерем утэ ялгыз гына
```

# Перевод

### Сёмбель Билалова - Когда же

```
Когда же ты придёшь,
Я бы только в глаза смотрела.
Не говоря ни слова, по твоим глазам
Поняла бы тебя, поняла бы.

В моём сердце печальное море,
Только волны, только буря.
Всё жду, что придёшь,
А жизнь проходит в одиночестве.

Когда же ты придёшь,
Я бы не горела в огне мучений.
Не отдала бы близких ветру и буре,
Не осталась бы в горестные дни.

В моём сердце печальное море,
Только волны, только буря.
Всё жду, что придёшь,
А жизнь проходит в одиночестве.
```
