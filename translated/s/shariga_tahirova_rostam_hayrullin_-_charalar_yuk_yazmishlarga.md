# Оригинал

### Шәрига Таһирова / Рөстәм Хәйруллин - Чаралар юк язмышларга

```
Исеңдәме, ямьле язда
Сокланыштык йолдызларга.
Тиңдәшләр идек пар былбылларга,
Гашыйклар идек дөньяга.

Күктә киек казлар җыры,
Күзләреңдә юллар моңы.
Янар йөрәктә дөрләп кабынды
Югалтуның авыр моңы.

Җан газизем, син моңайма,
Җиңел түгел бит болай да.
Өмет нур яксын сагынуларга
Чаралар юк язмышларга.
```

# Перевод

### Шәрига Таһирова / Рөстәм Хәйруллин - Нет управы на судьбу

```
Помнишь, в дивную пору весны
Мы смотрели на звезды с восторгом.
Соловьями были мы парными,
Влюблены были в мир без разлуки.

В небе - песня диких гусей,
В твоих глазах - мелодии дорог.
В пылающем сердце вспыхнула
Тяжкая песнь утраты.

Душа моя, не грусти,
Ведь и так нелегко.
Пусть луч надежды светит тоске -
Нет управы на судьбу.
```
