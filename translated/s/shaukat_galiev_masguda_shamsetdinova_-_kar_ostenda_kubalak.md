# Оригинал

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Кар өстендә күбәләк

```
Кар өстендә күбәләк,
Өшемиме икәнни?
Әле язга иртәрәк,
Төшенмиме икәнни?

Кар өстендә сары нур,
Каян килгән көлгәләк?
Салкын карны эретеп
Яз ясый бер күбәләк.
```

# Перевод

### Шәүкәт Галиев / Мәсгудә Шәмсетдинова - Бабочка на снегу

```
Бабочка на снегу -
Неужели мерзнет?
Для весны еще рано -
Разве не поймет ли?

На снегу - желтый свет,
Откуда взялась ты, цветочек?
Тая холодный снег,
Весну творит одна бабочка.
```
