# Оригинал

### Сылу - Алтын яфрак

```
Алтын козлэр кавыштырды,
Аерды салкын карлар,
Йорэгемне ялгыштырды
Синен назлы карашлар.

Бергэ булган хэр мизгелне
Хэтерлимен мин хаман,
Кара тондэ айга карап
Уйга калам, монланам.

Истэлеккэ алтын яфрак
Коздэн сорый алмадым,
Эзлэренэ карлар яуган,
Сине курэ алмадым.

Бергэ булган хэр мизгелне
Хэтерлимен мин хаман,
Кара тондэ айга карап
Уйга калам, монланам.
```

# Перевод

### Сылу - Золотой листок

```
Золотая осень встретила,
Разделила холодом снегов,
Сердце мое обманула
Твоими нежными взглядами.

Каждое мгновение вместе
Вспоминаю я всё ещё,
В чёрной ночи глядя на луну
Остаюсь дома, грущу.

В память золотой листок
Из осени взять не смог,
По следам твоим снег выпал,
Тебя увидеть не смог.

Каждое мгновение вместе
Вспоминаю я всё ещё,
В чёрной ночи глядя на луну
Остаюсь дома, грущу.
```
