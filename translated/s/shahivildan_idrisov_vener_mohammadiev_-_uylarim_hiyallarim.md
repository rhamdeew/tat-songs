# Оригинал

### Шаһивилдан Идрисов / Венер Мөхәммәдиев - Уйларым - хыялларым

```
Еллар аша кире кайтам,
Туган туксаннан узам.
Яңа пешкән икмәк исен
Тагы тойгандай булам.

Әнкәемнең елмаюын
Тагы күргәндәй булам.
Тупылдатып яратуын
Аркамда тоям һаман.

Балачакка кайтарсаң да,
Күкләргә күтәрсәң дә.
Туган җирдән аермагыз
Уйларым - хыялларым.
```

# Перевод

### Шахвилдан Идрисов / Венер Мухаммадиев - Мои мысли - мои мечты

```
Сквозь годы возвращаюсь,
Прохожу мимо родного порога.нЗапах свежеиспечённого хлеба
Вновь ощущаю, словно пробую на вкус.

Улыбку моей мамы
Вновь вижу, словно впервые.
Её горячую любовь,
Всё ещё ношу за спиной.

Даже если вернёшь в детство,
Даже если поднимешь к небесам.
От родной земли не отделяйте
Мои мысли - мои мечты.
```
