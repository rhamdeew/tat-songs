# Оригинал

### Сабит Моканов / Латыйф Хәмиди - Казах вальсы

```
Йомшак нурлар сибә-сибә килә май,
Яшел киемнәрен кия киң тугай.
"Сагынып кайттым туып үскән илне" - дип,
Киң тугайда канат кага бер тургай.

Ай-һай, чибәр тургай,
Сайрап җибәр, тургай!
Безнең туган илдә
Шат яшәргә уңай.
Сайра, сайра киң тугайда,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!

Тургай сөя туып үскән якларын,
Туган илнең кырларын, куакларын.
Тургай сайрый мактый-мактый җырында
Игенченең сабан сөргән чакларын

Ай-һай, чибәр тургай,
Сайрап җибәр, тургай!
Безнең туган илдә
Шат яшәргә уңай.
Сайра, сайра киң тугайда,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!
```

# Перевод

### Сабит Моканов / Латыйф Хәмиди - Казахский вальс

```
Май приходит, мягким светом осыпая,
Широкий луг зеленый свой наряд надел.
«Я по родной земле скучал» - заливая,
В широком лугу крылом взмахнул один тургай.

Ай-вай, прекрасный тургай,
Запой, тургай, пой!
В нашей родной стране
Жить и радоваться впору.
Пой, пой в широком лугу,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!

Тургай любит места, где он вырос,
Поля родной земли, кустарники ее.
Тургай в песне славит, славит,
Времена, когда пахарь вел свой плуг.

Ай-вай, прекрасный тургай,
Запой, тургай, пой!
В нашей родной стране
Жить и радоваться впору.
Пой, пой в широком лугу,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!
```
