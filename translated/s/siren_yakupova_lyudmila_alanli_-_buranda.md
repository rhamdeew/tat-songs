# Оригинал

### Сирень Якупова / Людмила Аланлы - Буранда

```
Ак кышларның гүя аккошлары -
Ак каеннар йөзә урамда.
Чуклы-чуклы канат очларына
Кар мамыгы зырлап урала.

Әйлән-бәйлән уйный тартма-йортлар,
Уйларымның уртасына басып.
Тәрәзләрдә җемелдәшкән утлар
Бүген миңа йолдызлардай тансык.

Көртләр ярып барам синең арттан,
Эзләреңне күмә көнчел буран.
Ни була соң, борылып кулың бирсәң,
Адаштырмам диеп, адаштырмам...
```

# Перевод

### Сирень Якупова / Людмила Аланлы - В пурге

```
Белой зимы, как лебеди, -
Белые березы плывут по улице.
На бахромчатые кончики крыльев
Снежная пыльца вихрем наматывается.

Кружатся в пляске домики-ящики,
Вставая в самую середину моих мыслей.
Огни, мерцающие в окнах,
Сегодня мне желанны, как звезды.

Прорезая сугробы, иду за тобой,
Твои следы заносит ревнивая метель.
Что будет, если ты обернешься, дашь руку,
Сказав: «Не собьюсь», - не собьюсь...
```
