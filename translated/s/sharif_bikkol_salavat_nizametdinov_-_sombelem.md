# Оригинал

### Шәриф Биккол / Салават Низаметдинов - Сөмбелем

```
Тугайдагы гөлләр сымак
Табигать ямьнәребез,
Сөмбелем, җан гөлем.

Бер-беребез өчен генә
Яралган җаннарыбыз.
Сөмбелкәем, Сөмбелкәем гөлкәем
Яралган җаннарыбыз.

Әйтешмәдек авыр сүзләр,
Итмәдек хыянатлар,
Сөмбелем, җан гөлем.

Кавышырбыз дип торганда,
Араларга керде ятлар.
Сөмбелкәем, Сөмбелкәем гөлкәем
Араларга керде ятлар.

Әлегечә аерымбыз
Рәнҗемә, кичер мине,
Сөмбелем, җан гөлем.

Кавышыр чаралар булса
Дәръялар кичер идем.
Сөмбелкәем, Сөмбелкәем гөлкәем
Дәръялар кичер идем.
```

# Перевод

### Шариф Биккол / Салават Низаметдинов - Сөмбелем

```
Как цветы на лугу -
Красота нашей природы,
Сөмбелем, душа моя.

Лишь друг для друга
Созданы наши души.
Сөмбелкәем, Сөмбелкәем, цветочек мой -
Созданы наши души.

Не бросали тяжких слов,
Не знали измены,
Сөмбелем, душа моя.

Когда ждали нашей встречи,
Встали между нами чужие.
Сөмбелкәем, Сөмбелкәем, цветочек мой -
Встали между нами чужие.

Пока мы врозь,
Не обижайся, прости меня,
Сөмбелем, душа моя.

Будь только шанс нам сойтись,
Я бы реки переправил.
Сөмбелкәем, Сөмбелкәем, цветочек мой -
Я бы реки переправил.
```
