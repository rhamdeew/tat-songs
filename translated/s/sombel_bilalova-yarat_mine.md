# Оригинал

### Сомбель Билялова - Ярат мине

```
Тормыш юлын бергэ узабыз
Гомер йомгакларын сутэбез
Кончелэрнен кузе тимэсен
Араларга ятлар кермэсен.

Дорлэп янсын сою учагы
Ярат мине ярат син тагын
Кадеремне белеп кадерлэ
Союемне зинхар сундермэ.

Ой турендэ голлэр усэлэр
Бер берлэрен назлап убэлэр
Сою голе безнен шинмэсен
Кознен кыраулары тимэсен.

Дорлэп янсын сою учагы
Ярат мине ярат син тагын
Кадеремне белеп кадерлэ
Союемне зинхар сундермэ.

Яннарымда жанын булганда
Алсуланып ата таннарда
Бэхетемэ тигэн ярым син
Сина атап жырлар язам мин.

Дорлэп янсын сою учагы
Ярат мине ярат син тагын
Кадеремне белеп кадерлэ
Союемне зинхар сундермэ.
```

# Перевод

### Сомбель Билялова - Люби меня

```
Жизненный путь пройдём вместе
Века закаты встретим с тобой
Пусть завистников глаза не коснутся
Пусть чужие меж нами не встанут.

Пусть пылает любви очаг
Люби меня, люби ещё
Цени меня, познав мою цену
Мою любовь прошу не сломи.

Вокруг дома растут цветы
Друг друга нежно целуют
Пусть наш цветок любви не увянет
Пусть морозы его не коснутся.

Пусть пылает любви очаг
Люби меня, люби ещё
Цени меня, познав мою цену
Мою любовь прошу не сломи.

Когда душою будешь рядом
В рассветы алые и ясные
Ты - половина моего счастья
Для тебя песни сочиняю я.

Пусть пылает любви очаг
Люби меня, люби ещё
Цени меня, познав мою цену
Мою любовь прошу не сломи.
```
