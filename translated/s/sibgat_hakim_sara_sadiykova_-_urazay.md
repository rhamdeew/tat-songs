# Оригинал

### Сибгат Хәким / Сара Садыкова - Уразай

```
Үзәннәрнең яме синдә,
Бөтен зәңгәрлек синдә.
Нәрсә турында сөйлисең
Агыйдел белән Сөнгә?

Уразай, Уразай,
Ятсам төшләремә керә
Синдә булып узган җәй.

Карыйсың ярлардан ерак,
Борынгы агым суга.
Сокланам синең бүгенге
Бәхетле язмышыңа.

Уразай, Уразай,
Ятсам төшләремә керә
Синдә булып узган жәй.
```

# Перевод

### Сибгат Хаким / Сара Садыкова - Уразай

```
Души рек тобой,
Вся прелесть во тебе.
О чём рассказываешь
Агидель тебе, Сюнь?

Уразай, Уразай,
Во снах моих живёт
Лето, бывшее с тобой.

Смотришь издалека,
В древнее речное русло.
Тревожит сегодняшнее
Счастливое твоё предназначение.

Уразай, Уразай,
Во снах моих живёт
Лето, бывшее с тобой.
```
