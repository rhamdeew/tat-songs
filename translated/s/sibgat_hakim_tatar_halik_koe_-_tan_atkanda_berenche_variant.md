# Оригинал

### Сибгат Хәким / Татар халык көе - Таң атканда (беренче вариант)

```
Уяна иркен болыннар,
Ачыла гөл керфекләре;
Тибрәнеп яфрак өстендә
Уйный чык бөртекләре.

Уяна кошлар тирәктә,
Җәйге җил куя исеп.
Кузгала җырлар йөрәктә.
Шунда син искә төшеп.

Уяна таңы илемнең,
Кырларга сибелә нуры.
Тиң булып безнең елларга
Балкысын гомер юлы.
```

# Перевод

### Сибгат Хәким / Татар народная мелодия - Рассвет (первая версия)

```
Пробуждаются вольно луга,
Раскрывают ресницы цветов;
На листе, колыхаясь нетуго,
Играют росинок шнурок.

Пробуждаются птицы на ветках,
Летний ветер дохнул горячо.
Поднимаются песни в сердцах -
И ты вспоминаешься мне вдруг ещё.

Пробуждается утро отчизны,
Свет струится по полям в тишине.
Пусть, ровесно годам нашим чистым,
Путь жизни сияет во мне.
```
