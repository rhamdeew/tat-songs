# Оригинал

### Салават Фәтхетдинов - Ак яр

```
Ак яр дигэн монлы койне
Кемнэр уйлап чыгарган?
Жырласам еларсын кебек,
Керфек очын чыланган.

Сал бишмэтен, эл чойлэргэ,
Килгэч кунарсын инде.
Сойгэненне ятлар алган
Дисэм еларсын инде.

Агыйделнен пароходы
Каршы бара агымга.
Бэгыркэем саргаерсын
Бу кадэрле сагынма.

Доньяда ин матурлардан
Ике нэрсэ бар дилэр.
Берсе анын яшьлек, гомер,
Берсе сойгэн яр дилэр.
```

# Перевод

### Салават Фатхетдинов - Белый берег

```
Эту печальную песню "Белый берег"
Кто сочинил и создал?
Когда пою, словно плачу,
Расплетается кружевной край.

Сними бешмет, люди танцуют,
Придя, повеселись теперь.
Любимую чужие забрали,
Скажу - заплачу теперь.

По Агидели пароход
Идёт против течения.
Моя печень желтеет,
Не тоскуй так сильно.

На свете из самых прекрасных
Две вещи есть, говорят.
Одна из них - молодость, жизнь,
Другая - любимый друг, говорят.
```
