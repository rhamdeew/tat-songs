# Оригинал

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Әнкәем-сердәшем

```
1. Сабый чактагыдай, иркәләнеп,
Алларыңа башым салыймчы?
Үзем күптән әни булсам да мин,
Нәни кызың булып калыймчы?

2. Чәчләремнән сыйпа әле, әни,
Тойыйм кулларыңның җылысын.
Изге йөрәгеңә сыйдыргансың
Мәхәббәтнең иң-иң олысын.

2. Өзми-куймый бәхетем сайраганда,
Син һәрвакыт әзер тыңларга;
Җанымдагы телсез сагышларны,
Ярый син бар, сүзсез аңларга.
```

# Перевод

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Мамочка, душевный друг

```
1. Как в детстве, приласканной, нежной,
Можно ли голову к вам приклонить?
Хоть я давно уже стала мамой,
Можно ли вашей малышкой побыть?

2. Погладь по волосам меня, мама,
Дай ощутить тепло твоих рук.
В сердце святом ты вместила
Самую главную силу любви.

2. Когда без умолку счастье поёт мне,
Ты всегда готова меня слушать;
Немые тоски моей души -
Хорошо, что ты есть, понять без слов.
```
