# Оригинал

### Сафуан Әлибай / Рим Хәсәнов - Тамчы тама (беренче вариант)

```
Яз турында көй чыгарып,
Тагы тамды тамчылар.
Әллә шатлык яше булып,
Күзләремнән тамдылар.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.

Җырлый - җырлый тамчы тамса,
Җирдә гөлләр уяныр,
Сөймәсләр дә сөяр язда,
Уйланмаслар - уйланыр.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.

Күңелемә тамчы белән
Язым нурлар яудырсын,
Гашыйк итеп, язлар тагы,
Акыллардан яздырсын.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.
```

# Перевод

### Сафуан Алибай / Рим Хасанов - Тамчы Тама (Первый вариант)

```
Выпуская мелодию о лете,
Вновь капают капли.
Вроде бы слёзы радости,
Из моих глаз капают.

Сливаясь, капля по капле,
Перебирая серебряные нити,
Лишь о лете тоскую
Мои годы кружатся.

Поёт - поёт, капля поёт,
На земле проснутся цветы,
Даже безмолвные полюбят летом,
И не думая - подумают.

Сливаясь, капля по капле,
Перебирая серебряные нити,
Лишь о лете тоскую
Мои годы кружатся.

Пусть в моё сердце каплей
Лето обрушит свет,
Сделав влюблённым, снова лето
Напишет с ума.

Сливаясь, капля по капле,
Перебирая серебряные нити,
Лишь о лете тоскую
Мои годы кружатся.
```
