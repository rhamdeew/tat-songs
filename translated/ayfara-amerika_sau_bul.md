# Оригинал

### Айфара - Америка сау бул

```
Америка дигэн серле сузне
Гел ишетеп белэ идек без
Дошман диеп хэтта саный идек
Начар итеп колэ идек без

Кем уйлаган шул суз кунеллэрнен
Ин туренэ утеп керер дип
Тирэ юндэ Америка булыр
Безнен арттан калмый йорер дип

Нигэ инде безгэ иялэнден
Син куэтле, син гажэеп бай
Яшэр идем синсез генэ
Син дэ яшэ минсез генэ
Америка гудбай

Кибетлэрдэ Америка хэзер
Тик ул гына хэзер урамда
Ике туеп ойгэ керэм дисэн
Синнэн алда кергэн ул анда

Ничек кенэ котылырга синнэн
Эйлэнэ бит инде башларым
Киену дэ инде синенчэ бит
Синдэ чыккан чэйлэр, ашлары

Нигэ инде безгэ иялэнден
Син куэтле, син гажэеп бай
Яшэр идем синсез генэ
Син дэ яшэ минсез генэ
Америка гудбай

Яратмыйм да инде узенне мин
Тошлэремэ нигэ керэсен
Телэмэгэн килеш яраттырып
Белмим нинди кызык курэсен

Кичэ бер кыз белэн таныштык бит
Тиз яулады минем кунелне
Америка анын исеме дэ
Менэ бэла, менэ кунелле

Нигэ инде безгэ иялэнден
Син куэтле, син гажэеп бай
Яшэр идем синсез генэ
Син дэ яшэ минсез генэ
Америка гудбай
```
