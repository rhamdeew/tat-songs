# Оригинал

### Гүзел Әхметова - Сагындым

```
Кайда иркэм югалдын син,
Кайда йоргэн юлларын?
Кайдан гына белим икэн-
Кайда оча уйларын?

Сагындырды,сагындым шул,
Сагындырды кузлэрен.
Сагындым шул,сагындырды
"Соям"дигэн сузлэрен.

Ай эндэшэ туз,дип,бераз-
Ана жинел эйтергэ!
Ай да белми кунелемэ
Нинди кинэш бирергэ.

Сагындырды,сагындым шул,
Сагындырды кузлэрен.
Сагындым шул,сагындырды
"Соям"дигэн сузлэрен.

Кайт син,бэгърем,тик югалма,
Кайт син ялгыз жаныма.
Кайт син бутэн китмим,диеп,
Гомерлеккэ калырга.

Сагындырды,сагындым шул,
Сагындырды кузлэрен.
Сагындым шул,сагындырды
"Соям"дигэн сузлэрен.
```

# Перевод

### Гузель Ахметова - Скучала

```
Куда, милый мой, пропал ты,
Куда дороги ведут?
Откуда же мне узнать -
Куда мысли твои летят?

Тосковала, скучала я так,
Тосковала по глазам твоим.
Скучала я так, тосковала
По словам твоим "люблю".

Луна хоть бы терпенье дала -
Легче было бы сказать!
Луна не знает, какой совет
Моей душе подать.

Тосковала, скучала я так,
Тосковала по глазам твоим.
Скучала я так, тосковала
По словам твоим "люблю".

Вернись ты, дорогой, только не исчезай,
Вернись к одинокой душе моей.
Вернись, говоря, что больше не уйдёшь,
Чтобы навеки остаться.

Тосковала, скучала я так,
Тосковала по глазам твоим.
Скучала я так, тосковала
По словам твоим "люблю".
```
