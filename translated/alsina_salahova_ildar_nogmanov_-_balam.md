# Оригинал

### Алсинә Сәлахова / Илдар Ногманов - Балам

```
Кочагыма алам назлап,
Бирәмен җан — җылымны,
Ана өчен бала бәгыръ
Улыммы ул, кызыммы.

Балам,балам, син татлы җимешем,
Син бер җаным минем йөрәкнең,
Балам, балам, алтыным-көмешем,
Син, сабыем, йөрәк тибешем.

Җырлар җырлыйм сиңа атап,
Кояшым-аем, диеп,
Иң-иң матур сүзләр әйтәм,
Сине иркәләп-сөеп.

Балам,балам, син татлы җимешем,
Син бер җаным минем йөрәкнең,
Балам, балам, алтыным-көмешем,
Син, сабыем, йөрәк тибешем.

Язмышымның бүләге син,
Иң олы сөенечем,
Тик шатлыгым була күр.
Булма син көенечем.

Балам,балам, син татлы җимешем,
Син бер җаным минем йөрәкнең,
Балам, балам, алтыным-көмешем,
Син, сабыем, йөрәк тибешем.
```

# Перевод

### Алсина Салахова / Ильдар Ногманов - Моё дитя

```
Беру в объятья с нежностью,
Отдаю душу — своё тепло,
Для матери дитя дороже всего,
Будь то сын мой или дочь моя.

Моё дитя, дитя моё, ты сладкий плод мой,
Ты единственная душа моего сердца,
Моё дитя, дитя моё, золото-серебро моё,
Ты, малыш мой, биение сердца.

Песни пою, тебе посвящая,
Называя солнцем-луной моей,
Самые-самые нежные слова говорю,
Лаская и любя тебя.

Моё дитя, дитя моё, ты сладкий плод мой,
Ты единственная душа моего сердца,
Моё дитя, дитя моё, золото-серебро моё,
Ты, малыш мой, биение сердца.

Ты дар моей судьбы,
Самая большая радость моя,
Будь только счастьем моим,
Не будь моей печалью.

Моё дитя, дитя моё, ты сладкий плод мой,
Ты единственная душа моего сердца,
Моё дитя, дитя моё, золото-серебро моё,
Ты, малыш мой, биение сердца.
```
