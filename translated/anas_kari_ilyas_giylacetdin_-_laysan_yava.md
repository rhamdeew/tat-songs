# Оригинал

### Әнәс Кари / Ильяс Гыйләҗетдин - Ләйсән ява

```
Яз иртәсен котлагандай,
Матур булып
Зәңгәр күкне каплап алды
Аксыл болыт.
Бизәкләнгән төсле булды
Бөтен һава.
Кар өстенә шыбыр-шыбыр
Яңгыр ява.

Ләйсән, ләйсән,
Иң беренче язгы яңгыр.
Ләйсән, ләйсән,
Син куандыр…
Ләйсән, ләйсән,
Иң беренче язгы яңгыр,
Көчлерәк яу,
Күңелләрне син куандыр.

Җир өстендә яткан карлар
Эреп бетсен.
Басуларда эш кайныйсы
Көннәр җитсен.
Без чыгарбыз бакчаларга
Ул көннәрдә.
Яшелчәләр утыртырбыз
Түтәлләргә.
```

# Перевод

### Анас Кари / Ильяс Гыйләҗетдин - Идёт дождь

```
Словно весть о приходе весны,
Прекрасным видом
Лазурное небо укрыло
Белое облако.
Словно украшенным казался
Весь небосвод.
На снег тихо-тихо
Дождь идёт.

Лейсан, лейсан,
Самый первый весенний дождь.
Лейсан, лейсан,
Ты обрадуй…
Лейсан, лейсан,
Самый первый весенний дождь,
Посильнее лей,
Сердца ты обрадуй.

На земле лежащий снег
Пусть растает весь.
Пусть настанут дни,
Когда в полях кипит работа.
Мы выйдем в огороды
В те дни.
Овощи посадим
На грядки.
```
