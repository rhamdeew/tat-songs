# Оригинал

### Билгесез - Кошлар юлы

```
Тагын килде көзнең айлары,
Талда инде кошлар сайрамый,
Юллар еракка (а)лар җыеныр,
Чит җирләргә китү кыендыр.

Сагынам, өзелеп сагынам,
Аерылдым туган ягымнан.
Тасма-тасма тормыш юлларым,
Юлдаш миңа минем елларым.

Җилдән җитез булыр уйларым,
Яз җырлары булыр канатлар,
Туган якның борма юллары
Күңелем язда әле кабатлар.

Сагынам, өзелеп сагынам,
Аерылдым туган ягымнан.
Тасма-тасма тормыш юлларым,
Юлдаш миңа минем елларым.

Тагын килде көзнең айлары,
Талда инде кошлар сайрамый,
Юллар еракка (а)лар җыеныр,
Чит җирләргә китү кыендыр.

Сагынам, өзелеп сагынам,
Аерылдым туган ягымнан.
Тасма-тасма тормыш юлларым,
Юлдаш миңа минем елларым.
```
