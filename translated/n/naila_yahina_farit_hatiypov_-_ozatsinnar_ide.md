# Оригинал

### Наилә Яхина / Фәрит Хатыйпов - Озатсыннар иде

```
Кояшлар да баер, атар таңнары да,
Ерагаер безнең аралар.
Әмма юллар кая илтсәләр дә,
Синең күзләр озата баралар.

Озатсыннар иде алар мине
Бәхет юлларыннан офыкка.
Син юллаган якты юллардан мин
Армас идем күпме барып та.
```

# Перевод

### Наилә Яхина / Фәрит Хатыйпов - Пусть бы провожали

```
И солнца зайдут, и рассветы взойдут -
Удаляются наши пути.
Но куда бы дороги ни повели,
Твои глаза меня будут вести.

Пусть бы они провожали меня
К горизонту дорогой счастья.
По светлым путям, что мне ты послал,
Я бы шёл без устали, сколько ни странствуй.
```
