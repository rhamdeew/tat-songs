# Оригинал

### Нәфисә Сабирҗанова / Рәис Нәгыймов - Сине көттем

```
Бу язларда сине көттем,
Уйладым мин сине генә.
Беркем белмәгән серемне
Тыңлады тик сирень генә.
Шомырт чәчәкләре ул чак
Хыялыма тәңгәл иде.
Умырзаяларым гүя
Күзләреңнең зәңгәрлеге...
Ләйсән яңгырлар яуганда
Көттем мин сине өзелеп.
Яратуың бураннарда
Калгандыр, ахры, күмелеп.
```

# Перевод

### Нәфисә Сабирҗанова / Рәис Нәгыймов - Я тебя ждала

```
Этой весной я ждала тебя,
Думала только о тебе.
Тайну мою, никому неведомую,
Слушала лишь сирень.
Цветы черёмухи тогда
С мечтой моей совпали.
А подснежники - будто
Синева твоих глаз...
Когда лейсанские дожди шли,
Я ждала тебя, не в силах забыть.
Твоя любовь в метелях,
Видно, осталась, засыпанной навек.
```
