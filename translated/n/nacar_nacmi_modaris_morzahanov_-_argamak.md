# Оригинал

### Наҗар Нәҗми / Мөдәрис Морзаханов - Аргамак

```
Мында Аргамак авылы,
Аргамак ураманнары.
Мында бар Аргымак тауы,
Аргымак аланнары.

Аргамаклар җилгән алан -
Дүртөйлө сабан туе.
Исемнәрендә җисем бар,
Шулай ул гомер буе.

Заман аргмагында син,
Җиләсең, туган ягым.
Гомергә шулай җилсен, дим,
Ал бирми аргымагым.
```

# Перевод

### Нәҗар Нәҗми / Мөдәрис Морзаханов - Аргамак

```
Здесь село Аргамак стоит,
И улицы его - Аргамак.
Здесь есть гора Аргымак,
И луга - Аргымак.

Луг, где мчатся аргамаки, -
Сабантуй Дюртюли.
В их именах - сама суть,
Так было испокон веков.

На аргамаке времени ты,
Несёшься, край родной.
Пусть вечно мчит он так, - прошу,
Мой аргамак, не стой.
```
