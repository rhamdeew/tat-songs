# Оригинал

### Нәби Дәүли / Сара Садыйкова - Татарстан - минем республикам

```
Татарстан - минем республикам -
Россиянең матур җирендә.
Ленин аңа үзе исем кушкан,
Бәхет теләп гомер-гомергә.

Карлары бар аның, язлары бар,
Ташкыннары аның ни тора!
Еракларга киткән тургайлар да
Сагынып кайта туган йортына.

Башка илләр миңа кирәк түгел,
Хыялым юк минем ул хакта.
Туган җирләремнең гүзәллеге
Кергән күңелләргә яшь чакта.

Батырларны күрсәң, гаҗәпләнмә, -
Батырлар күп татар илендә.
Татарстан - минем республикам -
Россиянең матур җирендә.
```

# Перевод

### Нәби Дәүли / Сара Садыйкова - Татарстан - моя республика

```
Татарстан - моя республика -
В краю прекрасном, в сердце России.
Ленин имя сам ей даровал,
Счастья пожелав на все века.

Есть у ней снега, весна цветет,
Полноводье - диво, красота!
И жаворонки, в даль улетевшие,
С тоскою возвращаются домой.

Мне иные страны не нужны,
Не мечтаю вовсе о другом.
Красота родной моей земли
В юные сердца вошла теплом.

Если встретишь храбрых - не дивись, -
Храбрецов немало в земле татарской.
Татарстан - моя республика -
В краю прекрасном, в сердце России.
```
