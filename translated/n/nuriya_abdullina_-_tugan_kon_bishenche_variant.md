# Оригинал

### Нурия Абдуллина - Туган көн (бишенче вариант)

```
Туган көнеңне котларга килдек бергәләп,
Ак бәхетләр тели сиңа, тели һәр йөрәк.

Кояш та нурларын сибә, сиңа иртәдән.
Кошлар котлау җырын җырлый, сине иркәләп.

Бәхет балкый бүген синең, көләч йөзеңдә,
Җыелганга җан дусларым табын түренә.

Кояш та нурларын сибә, сиңа иртәдән.
Кошлар котлау җырын җырлый, сине иркәләп.

Гөл чәчәкләргә күмелсен барыр юлларың,
Кайгы - хәсрәтсез үтсеннәр гомер юлларың.

Кояш та нурларын сибә, сиңа иртәдән.
Кошлар котлау җырын җырлый, сине иркәләп.
```

# Перевод

### Нурия Абдуллина - День рождения (пятый вариант)

```
Мы пришли поздравить тебя в этот день,
Белых счастий тебе желает каждое сердце.

Солнце с утра рассыпает тебе свои лучи,
Птицы поют поздравительные песни, лаская тебя.

Счастье сияет сегодня в твоём лучистом лице,
Что собрались мы, друзья родные, за почётным столом.

Солнце с утра рассыпает тебе свои лучи,
Птицы поют поздравительные песни, лаская тебя.

Пусть путь твой утопает в цветах,
Пусть без горя и бед пройдут дороги жизни твоей.

Солнце с утра рассыпает тебе свои лучи,
Птицы поют поздравительные песни, лаская тебя.
```
