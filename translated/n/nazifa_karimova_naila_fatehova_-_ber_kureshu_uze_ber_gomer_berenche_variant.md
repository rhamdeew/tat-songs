# Оригинал

### Назифа Каримова / Наилә Фатехова - Бер күрешү - үзе бер гомер (беренче вариант)

```
Безнең очрашулар,
Бик еш булмаганга
Күрешүне гел көтә күңел.
Сагынуларга түзмик,
Бер күрешү үзе,
Бер күрешү үзе - бер гомер.

Бер күрешү үзе, бер күрешү үзе,
Бер күрешү үзе бер гомер.

Безнең әдәбият, сәнгәть, мәдәният,
Күңелләргә якты нур бирер.
Мәнгегә хәтердә бер күрешү үзе,
Бер күрешү үзе - бер гомер.

Бер күрешү үзе, бер күрешү үзе,
Бер күрешү үзе бер гомер.

Мәскәүләрдән Казан Уфаларга кадәр,
Безнең җырлар булды бер күпер.
Нинди акыллы сүз - бер күрешү үзе,
Бер күрешү үзе - бер гомер.

Бер күрешү үзе, бер күрешү үзе,
Бер күрешү үзе бер гомер.
```

# Перевод

### Назифа Каримова / Наиля Фатехова - Одно лишь сражение - вся жизнь (первый вариант)

```
Наши редкие встречи
Из-за того, что редки были,
Душа всегда ждёт борьбы.
Не вытерпев тоски разлуки,
Одно лишь сражение,
Одно сраженье - вся жизнь.

Одно лишь сражение, одно лишь сражение,
Одно сраженье - вся жизнь.

Наша литература, искусство, культура
Дарят душам яркий свет.
Вечной памятью одно лишь сражение,
Одно сраженье - вся жизнь.

Одно лишь сражение, одно лишь сражение,
Одно сраженье - вся жизнь.

От Москвы до Казани, до Уфы
Наши песни стали мостом.
Как мудры эти слова - одно лишь сражение,
Одно сраженье - вся жизнь.

Одно лишь сражение, одно лишь сражение,
Одно сраженье - вся жизнь.
```
