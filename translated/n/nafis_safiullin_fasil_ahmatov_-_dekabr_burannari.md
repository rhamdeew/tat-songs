# Оригинал

### Нәфис Сафиуллин / Фасил Әхмәтов - Декабрь бураннары

```
Килде инде көтеп алган
Декабрь бураннары,
Килеп җитәр-җитмәс безне
Туңдыра торганнары.

Декабрь бураннары -
Туңдыра торганнары,
Елый-елый морҗалардан
Сызгыра торганнары.

Урап-урап бураннары
Томалый урамнарны.
Ап-ак калын юрган белән
Яба карурманнарны.

Декабрь бураннары -
Көрт өя торганнары,
Улый-улый кар туздырып
Тезә кар диварлары.
```

# Перевод

### Нафис Сафиуллин / Фасил Ахметов - Декабрьские метели

```
Пришли, как долго ждали мы,
Декабрьские метели,
Едва коснувшись, до костей
Пронзают нас и стелют.

Декабрьские метели -
Нас студят, как хотели,
Рыдая-плача, по трубам
Свистят и завывают.

Бушуя, кружат метели,
Улицы заметают,
Белоснежным пуховым одеялом
Все дороги укрывают.

Декабрьские метели -
Сугробы громоздят они,
Воя-воя, снег рассыпая,
Возводят снежные стены.
```
