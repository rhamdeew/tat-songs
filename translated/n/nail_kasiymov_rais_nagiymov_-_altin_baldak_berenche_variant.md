# Оригинал

### Наил Касыймов / Рәис Нәгыймов - Алтын балдак (беренче вариант)

```
Бәхет белән туды туй бокалын
Хыянәтең ватты телгәләп.
Җиз кыңгырау чыңын хәтерләтеп
Алтын балдак, төштең тәгәрәп.

Алтын балдак, алтын балдак,
Язмышларны булмый ялгап,
Язмышларны булмый ялгап.
Алтын балдак, алтын балдак,
Язмышларны булмый ялгап,
Хыянәтне булмый аңлап.

Алтын балдак, хыялымны алдап
Кояш булып яндың офыкта.
Хыянәткә шаһит булган балдак,
Мәхәббәткә булдың зур нокта.

Алтын балдак, алтын балдак,
Язмышларны булмый ялгап,
Язмышларны булмый ялгап.
Алтын балдак, алтын балдак,
Язмышларны булмый ялгап,
Хыянәтне булмый аңлап.
```

# Перевод

### Наил Касыймов / Рәис Нәгыймов - Золотое кольцо (первый вариант)

```
Счастьем родился бокал на свадьбе,
Но измена твоя его вдребезги бьёт.
Как звон медных колокольцев в памяти,
Золотое кольцо, ты упало, скатившись.

Золотое кольцо, золотое кольцо,
Не связать нам судьбы воедино,
Не связать нам судьбы воедино.
Золотое кольцо, золотое кольцо,
Не связать нам судьбы воедино,
Не понять нам измены глубины.

Золотое кольцо, обманув мои грёзы,
Ты на дальнем горизонте солнцем горел.
Кольцо, что свидетелем стало измены,
Для любви ты поставило точку большую.

Золотое кольцо, золотое кольцо,
Не связать нам судьбы воедино,
Не связать нам судьбы воедино.
Золотое кольцо, золотое кольцо,
Не связать нам судьбы воедино,
Не понять нам измены глубины.
```
