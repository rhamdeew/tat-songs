# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Яшьлекләр күңелдә! (Без бергә)

```
Шул яшькә җиткәнбез инде без,
Яшьлекне сагынып яшибез.
Тормышның ямьнәрен табарга
Сөяргә бүген дә тиеш без!
Кызганыч булса да узганы,
Гомерләр заяга узмады.
Үкенсәк тә кайчак ялгыштан,
Хыянәт итүләр булмады!

Уздырдык язларны, җәйләрне без бергә,
Без бергә, без бергә!
Куркытмый, кышлар да куркытмый,
Яшьлек бар, яшьлек бар күңелдә...
Уздырдык язларны, җәйләрне без бергә,
Без бергә, без бергә!
Куркытмый, кышлар да куркытмый,
Яшьлек бар, яшьлек бар күңелдә...

Сабырлык - йөрәкләр учларда,
Сер бирми яшибез дошманга.
Авырлык килсә дә уза ул,
Таяныч, өмет бар дусларда!
Тормышны төзибез үзебез,
Һәрвакыт ак булды йөзебез,
Үкенмә син гомер уза дип,
Кадерен яшәүнең беләбез!

Уздырдык язларны, җәйләрне без бергә,
Без бергә, без бергә!
Куркытмый, кышлар да куркытмый,
Яшьлек бар, яшьлек бар күңелдә...
Уздырдык язларны, җәйләрне без бергә,
Без бергә, без бергә!
Куркытмый, кышлар да куркытмый,
Яшьлек бар, яшьлек бар күңелдә...
```
