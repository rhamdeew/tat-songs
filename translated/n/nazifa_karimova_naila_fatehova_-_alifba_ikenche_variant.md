# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Әлифба (икенче вариант)

```
Әлиф, бә, тә, сә,
Күңел күзе күрсә,
Җим, ха, хә, дәл,
Маллар булыр хәләл.
Әлиф, бә, тә, сә, җим, ха, хә, дәл.

Зәл, ра, шин, син,
Коръән нуры - Йәсин.
Шин, сад, дад, та -
Изгелекләр кайта.
Зәл, ра, шин, син, шин, сад, дад, та.

За, гайн, гәйн, фә.
Кыл Аллага сәҗдә.
Каф, кәф, ләм, мим,
Сабырлыклар телим.
За, гайн, гәйн, фә, каф, кәф, ләм, мим.

Нун, һә, уау, йә.
Көн узмасын зая.
Йә Раббым, син бар,
Йә Аллаһу әкбәр,
Йә Аллаһу әкбәр.
```

# Перевод

### Назифа Каримова / Наилә Фатехова - Алифба (второй вариант)

```
Алиф, ба, та, са,
Если видит глаз души,
Джим, ха, хе, дал,
Станет чистым ваш удел.
Алиф, ба, та, са, джим, ха, хе, дал.

Зал, ра, шин, син,
Свет Корана - Ясин.
Шин, сад, дад, та -
Возвращаются доброта.
Зал, ра, шин, син, шин, сад, дад, та.

За, гайн, гайн, фа.
Поклонись Аллаху ты.
Каф, кеф, лам, мим,
Терпенья нам прошу.
За, гайн, гайн, фа, каф, кеф, лам, мим.

Нун, ха, уа, йа.
Пусть не гаснет день в пустую.
О Господь, Ты есть,
Йа Аллаху акбар,
Йа Аллаху акбар.
```
