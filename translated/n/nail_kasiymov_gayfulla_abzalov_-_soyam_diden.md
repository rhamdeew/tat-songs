# Оригинал

### Наил Касыймов / Гайфулла Абзалов - Сөям, дидең

```
Җирдә яшәү серләрен мин
Үзгәрүдән эзләдем.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.

Тын күлләр дә гел тын булмый
Ярда дулкын эзләре.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.

Үзгәрер шул күзләрең дә
Үзгәрер шул йөзләрең.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.
```

# Перевод

### Наил Касыймов / Гайфулла Абзалов - Ты сказала: люблю

```
Тайну жизни на земле я
В переменах находил.
Пусть неизменными останутся
Твои слова «люблю».

Даже тихие озёра не всегда
Безмятежны - у берега следы волн.
Пусть неизменными останутся
Твои слова «люблю».

И глаза твои изменятся,
И изменится твой лик.
Пусть неизменными останутся
Твои слова «люблю».
```
