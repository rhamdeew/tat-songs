# Оригинал

### Нәзирә Ишколова / Радик Мөхәмәдиев - Тәүге мәхәббәт

```
Үткәннәрне эзләп кайттым тагы
Сагышларга җаным чорналды.
Кичке тальян моңы, яшьлегемнең чыңы
Йөрәгемне сыгып уралды.

Тавыш бирде тәүге сөю хисе
Яшьлегемнең ерак таңыннан.
Хаталарга юмарт, гаделлеккә саран
Язмыш юлын үттем яңыдан.

Ялгыз аккош күлдә кыйгак - кыйгак
Татлы төшләремнән уята.
Таңнар кыелганда, уйлар уелганда
Тәүге сөю күңел юата.

Үрсәләнгән чакта бөтен дөнъям
Уйлар дәръясында югалсам.
Өмет сүрелгәндә, яшьләр ябырылганда
Тәүгә мәхәббәттән көч алам.
```

# Перевод

### Назира Ишколова / Радик Мухамадиев - Первая любовь

```
Снова вернулся, прошлое ища,
Душа моя окутана тоскою.
Вечерний тальян, колокол юных лет
Сжимает сердце кольцом неземною.

Подала голос первая любовь
С далекой утренней поры мечтаний.
К ошибкам щедр, к правде скуп судьбы путём -
Я заново прошёл его страданий.

Одинокий лебедь в озере - кигак, кигак -
Из сладких снов меня зовёт обратно.
Когда рассвет ломается, мысли режут след,
Первая любовь мне сердце лечит кротко.

Когда весь мир мой в трепете дрожит,
Тону в потоке дум без оговорок.
Когда погаснет надежда, слёзы хлынут вдруг -
Я силу черпаю в первой любви, как в море.
```
