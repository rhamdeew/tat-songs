# Оригинал

### Наҗар Нәҗми / Заһир Исмәгыйлев - Кичке Агыйдел

```
Җилкенсез көймә шикелле,
Ай йөҙә талгын (ялгыз) гына.
Ай янында җем-җем иткән
Йолдыз да йолдыз гына
Ай йөзә ялгыз гына,
Ай йөзә ялгыз гына,
Ай йөзә.

Кала утлары җемелдәп
Идел өстенә төшә.
Әйтерсең алар сузылып
Агыйдел суын эчә,
Идел өстенә төшә,
Идел өстенә төшә,
Агыйдел.

Серләшә кебек бер җиһан,
Бар агачлар тын гына.
Коча кебек Димне Идел
Әкерен генә, тын гына,
Әкерен генә, тын гына,
Әкерен генә, тын гына,
Тын гына.
```

# Перевод

### Наҗар Нәҗми / Заһир Исмәгыйлев - Вечерняя Агыйдель

```
Как лодка без весел,
Луна плывет тихо, одна.
Рядом с луной мерцают
Звезды - лишь звезды одни.
Луна плывет одна,
Луна плывет одна,
Луна плывет.

Огни города мерцая
Падают на Идель.
Словно они, протянувшись,
Пьют воды Агыйдели,
Падают на Идель,
Падают на Идель,
Агыйдель.

Будто весь мир шепчется,
Все деревья молчат.
Словно обнимает Дим Идель
Тихо, медленно, молча,
Тихо, медленно, молча,
Тихо, медленно, молча,
Молча.
```
