# Оригинал

### Наҗар Нәҗми / Сәйяр Хәбибуллин - Таң алдыннан

```
Таң атарга калды өч сәгать,
Өч сәгатьтән инде китәсең.
Өч сәгать - ул миңа өч минут,
Өч минуттан ялгыз итәсең.

Кушымта:
Син китәсең...Кала якты көн,
Якты көндә калам көн күреп,
Кала кояш, ләкин китәсең,
Йолдызларның барын сүндереп.

Кояш баер, янар йолдызлар,
Кабатланыр таңнар яңадан.
Кабатланыр көннәр, сәгатьләр,
Булыр ай, ел шулар саныннан;

Кабатлана барыр мизгелләр.
Кабатланыр гореф-гадәтләр -
Кабат килеп, кабат китәрсең,
Кабатланмас шушы сәгатьләр -
Гомернең бер алтын кисәге,
Иртәгә ул инде кичәге...
```

# Перевод

### Наҗар Нәҗми / Сәйяр Хәбибуллин - Перед рассветом

```
До рассвета осталось три часа,
Через три часа ты уже уйдёшь.
Три часа - для меня как три минуты,
Через три минуты оставишь в одиночестве.

Припев:
Ты уходишь... Остаётся светлый день,
В светлом дне мне жить и продолжать,
Остаётся солнце, но ты уходишь,
Погасив все звёзды до одной.

Солнце сядет, вспыхнут снова звёзды,
Снова повторятся зори по утрам.
Повторятся дни и эти часы,
Будут месяц, год из их числа;

Повторяться станут все мгновенья.
Повторятся и обряды, и обычаи -
Снова ты придёшь, потом уйдёшь опять,
Но не повторятся эти часы -
Золотая часть моей жизни,
Завтра они будут уже вчерашним...
```
