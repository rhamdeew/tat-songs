# Оригинал

### Нәфкыя Сафина / Урал Рәшитов - Тәкыялар үрдем гөлләрдән

```
Болыннарда йөрим, җыям,
Тәкыялар үрәм гөлләрдән.
Буддым гашыйк ахры, ала алмыйм
Күзләремне синең күзләрдән.

Чәчәкләрнең, ай, төсләре ачык,
Аллы-гөлле икән үзләре.
Кыңгыраудан үрдем мин тәкыя,
Булсын диеп күзең төсләре.

Чәчәк бәйләмнәре кочак тулы,
«Тәкыялар үрдем» дисең син.
Ярсып тибә йөрәк дигәннәре,
Чәчләремә кагылып үтсәң син.

Чәчәкләрнең, ай, төсләре ачык,
Аллы-гөлле икән үзләре.
Кыңгыраудан үрдем мин тәкыя,
Булсын диеп күзең төсләре.

Кыңгыраудай зәңгәр күзләреңә
Ай, килешә, чәчәк тәкыя.
Мәңгелеккә бергә булып, икәү,
«Иркәм» диеп дәшү - бер хыял.

Чәчәкләрнең, ай, төсләре ачык,
Аллы-гөлле икән үзләре.
Кыңгыраудан үрдем мин тәкыя,
Булсын диеп күзең төсләре.
```

# Перевод

### Нәфкыя Сафина / Урал Рәшитов - Венки плела из цветов

```
По лугам брожу, собираю,
Из цветов венки я плету.
Видно, я влюбилась - не в силах
Оторвать от глаз твоих взгляд.

Ах, у цветов расцветы ясны,
Пестры сами, разноголосье.
Из колокольчиков венок я сплела,
Чтобы цветом - как твои глаза.

Охапки цветов - в объятьях полны,
«Венки я сплела», - говоришь ты.
Сердце, говорят, бьется яростно,
Если ты коснешься моих волос.

Ах, у цветов расцветы ясны,
Пестры сами, разноголосье.
Из колокольчиков венок я сплела,
Чтобы цветом - как твои глаза.

К синеве, как колокол, глаз твоих
Ах, к лицу венок из цветов.
Вместе навсегда, вдвоем -
«Иркәм» сказать - одна лишь мечта.

Ах, у цветов расцветы ясны,
Пестры сами, разноголосье.
Из колокольчиков венок я сплела,
Чтобы цветом - как твои глаза.
```
