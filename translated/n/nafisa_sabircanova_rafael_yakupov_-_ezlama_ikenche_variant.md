# Оригинал

### Нәфисә Сабирҗанова / Рафаэль Якупов - Эзләмә (икенче вариант)

```
Эзләмә карлы урамда
Юлдан минем эзләрне.
Эзләмә, күрә алмассың
Сөю тулы күзләрне.
Эзләмә, ишетә алмассың
Назлы таныш тавышны.
Эзләмә, күптән югалттык
Ул иң якты сагышны.
Эзләмә мине болыннан -
Кыңгыраулы, җиләкле.
Эзләмә мине бураннан, -
Аңламадың кирәкне.
Эзләмә акланыр сүзләр,
Туплап бар хәтереңне.
Эзләмә, таба алмассың
«Сөеклең», «кадерлең»не.
```

# Перевод

### Нафиса Сабирҗанова / Рафаэль Якупов - Не ищи (второй вариант)

```
Не ищи в заснеженной улице
Следы с моей дороги.
Не ищи, не увидишь
Глаза полные любви.
Не ищи, не услышишь
Нежный знакомый голос.
Не ищи, давно потеряли
Ту самую светлую тоску.
Не ищи меня на лугу -
С колокольчиками, с ягодами.
Не ищи меня в метели, -
Не поняла нужное.
Не ищи оправдывающие слова,
Собрав все воспоминания.
Не ищи, не найдёшь
«Любимую», «дорогую».
```
