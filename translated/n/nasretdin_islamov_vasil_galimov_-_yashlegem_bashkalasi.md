# Оригинал

### Нәсретдин Исламов / Вәсил Галимов - Яшьлегем башкаласы

```
Колач җәеп каршы алдың
Син яшьлек елларымда.
Бәхетле язмыш теләдең
Нур сибеп юлларыма.

Нефтекама - гүзәл кала
Яшьлегем башкаласы.
Данлы хезмәт сөючеләр,
Нефтьчеләр каласы.

Сокланамын алтын куллы
Асыл кешеләреңә.
Кушылып ага синең даның
Арлан нефтьләренә.

Нефтекама - гүзәл кала
Яшьлегем башкаласы.
Данлы хезмәт сөючеләр,
Нефтьчеләр каласы.

Горур калам, күкрәгемдә
Кабыздың нурлы ялкын.
Җыр, хезмәт юлдаш булганга.
Бәхетле синең халкың.

Нефтекама - гүзәл кала
Яшьлегем башкаласы.
Данлы хезмәт сөючеләр,
Нефтьчеләр каласы.
```

# Перевод

### Насретдин Исламов / Васил Галимов - Столица моей молодости

```
Руки распахнул навстречу
Ты в года моей младости.
Счастья светлую тропинку
Пожелал ты мне искристо.

Нефтекама - город чудный
Столица моей младости.
Славных трудников любители,
Город нефтяников славный.

Восхищаюсь я золотыми
Руками людьми твоими.
Сливаются в поток славы
Твои ароматные нефти.

Нефтекама - город чудный
Столица моей младости.
Славных трудников любители,
Город нефтяников славный.

Горжусь тобой, ты в груди моей
Сияешь как огонь пламенный.
Песня, труд - пути схожие.
Счастлив ты, народ твой драгоценный.

Нефтекама - город чудный
Столица моей младости.
Славных трудников любители,
Город нефтяников славный.
```
