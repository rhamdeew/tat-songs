# Оригинал

### Наил Касыймов / Зөфәр Хәйретдинов - Оныттыр (беренче вариант)

```
Үткәннәр турында сорама,
Кагылма авырткан ярама.
Көнләмә, ул диеп каршымда,
Яратсаң оныттыр барсын да.

Онытырга телимен күптәннән,
Күтәрмә пәрдәсен үткәннең,
Калсын ул булып бер ялгышым,
Була алсаң, бул минем язмышым.

Үткәннәр турында сорама,
Ярамнан әле дө кан тама.
Ялгышлар күтәрмәм кабаттан,
Ышандыр, оныттыр, яратсаң.
```

# Перевод

### Наил Касыймов / Зуфар Хайретдинов - Заставь забыть (первый вариант)

```
О прошедшем не спрашивай,
К больной ране не прикасайся.
Не грусти, не говори напротив,
Если любишь - заставь забыть всё.

Давно хочу я забыть,
Не поднимать завесу прошлого,
Пусть останется она одной ошибкой,
Если можешь - стань моей судьбой.

О прошедшем не спрашивай,
Из раны всё ещё течёт кровь.
Ошибки не подниму вновь,
Убеди, заставь забыть, если любишь.
```
