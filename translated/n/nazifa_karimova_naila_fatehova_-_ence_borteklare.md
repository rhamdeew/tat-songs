# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Энҗе бөртекләре

```
Әй, балачаклар, энҗе табар чаклар,
Якты дөньяга без дә чыгыйк әле,
Халык күңеленнән барлап табарбыз,
Еллар җуймаган энҗе бөртекләрен!

Халкым җырлары, сандугач моңы
Йөрәкләрне бер җилкетер әле
"Сарман", Карурман", "Җидегән чишмәләрдә"
Энҗе бөртекләре, энҗе бөртекләре!

Тукайда кебек, безнең күңелдә дә,
"Туган тел" диеп җырлар туар әле,
Саклыйк кадерләп, ана телебезне
Энҗе бөртекләрен, энҗе бөртекләрен!
```

# Перевод

### Нәзифә Кәримова / Наилә Фатехова - Жемчужные зерна

```
Эй, детства дни - пора искать жемчужины,
В светлый мир давай выйдем и мы,
В сердце народа по крупицам соберём
Жемчужные зёрна, не стёртые годами!

Песни моего народа, соловьиный звон
Ещё раз сердца встревожит и взовьёт:
«Сарман», «Карурман», «У родников Джидеган» -
Жемчужные зёрна, жемчужные зёрна!

Как у Тукая, в наших сердцах
Ещё родятся песни - «Туган тел»,
Беречь давай святую речь родную,
Жемчужные зёрна, жемчужные зёрна!
```
