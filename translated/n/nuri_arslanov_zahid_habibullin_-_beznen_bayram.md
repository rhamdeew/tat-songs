# Оригинал

### Нури Арсланов / Заһид Хәбибуллин - Безнең бәйрәм

```
Ерак калды давыл,
Күкрәп үтте яулар...
Шаулар безнең җиңү,
Мәңгелеккә шаулар!

Бөтен тирә-якта
Бүген әйлән-бәйлән:
Кара көннәр артта,
Якты безнең бәйрәм,
Якты безнең бәйрәм!

Бәйрәм итә бүген
Дуслык теләгәннәр.
Сугыш күрмәсеннәр,
Сугыш күрмәгәннәр.

Әйлән әйдә, дөнья,
Тынычлыкка әйлән!
Дөнья халкы, тыңла:
Бүген бездә бәйрәм,
Бүген бергә бәйрәм!

Җиңү хөрмәтенә
Күтәр бәйрәм котлап!
Безнең бокаллардан
Дөнья торсын чыңлап!

Ләйсән булып төште
Безгә бүген бар ямь, -
Язның үзе төсле
Матур безнең бәйрәм,
Матур безнең бәйрәм!
```

# Перевод

### Нури Арсланов / Заһид Хәбибуллин - Наш праздник

```
Далеко унеслась буря,
Громом прошли бои...
Пусть славой звенит Победа,
Пусть вечно звенит в груди!

Повсюду вокруг сегодня
Кружится хоровод:
Чёрные дни позади,
Светел наш праздник,
Светел наш праздник!

Празднуют нынче те,
Кто дружбы желал другим.
Пусть не узнают войны
Те, кто её не видел.

Кружись же, мир, к миру,
К покою кружись!
Народы, услышьте:
Сегодня у нас праздник,
Сегодня наш общий праздник!

В честь Победы
Поднимем тост праздничный!
Пусть от наших бокалов
Мир звенит, как звонок!

Ләйсәнем пал сегодня,
Нам вся благодать, -
Как сама весна,
Прекрасен наш праздник,
Прекрасен наш праздник!
```
