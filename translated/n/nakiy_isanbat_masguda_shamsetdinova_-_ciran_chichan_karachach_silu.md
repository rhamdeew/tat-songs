# Оригинал

### Нәкый Исәнбәт / Мәсгудә Шәмсетдинова - Җирән Чичән, Карачәч Сылу

```
Күп гиздем сулар тапмый җир йөзендә,
Тоз күрдем Әстерханның диңгезендә,
Баскынчак күлендә дә татлы су юк,
Канмады бу сусыным һичбер җирдә.

Күп гиздең сулар тапмый җир йөзендә,
Тоз күрдең Әстерханның диңгеәендә,
Иделнең тамагыннан татып баксаң,
Татлы су бар диңгезнең төп үзендә.
```

# Перевод

### Нәки Исәнбәт / Мәсгудә Шәмсетдинова - Жирэн Чичэн, Карачач Сылу

```
Много бродил я, не нашёл вод на земле,
Соль tasted я в Астраханском море,
И в озере Баскинчак нет сладкой воды,
Нигде не утолилась эта жажда.

Много бродила ты, не нашла вод на земле,
Соль tasted ты в Астраханском море,
Но если попробуешь у самого устья Волги,
Сладкая вода есть в самой глубине моря.
```
