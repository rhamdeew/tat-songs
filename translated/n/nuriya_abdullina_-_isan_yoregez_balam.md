# Оригинал

### Нурия Абдуллина - Исән йөрегез, балам!

```
Ыргый-сикерә ага сулар,
Алга җилә замана.
Җилеп барган аргамактай
Ашыгалар балалар.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.

Аңлый ана, алар да бит
Балалар үстерәләр.
Балалары бәхетенә
Алар да йөгерәләр.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.

Сезнең дә кочыкларыгыз
Канат ныгытыр әле.
Утырып уйлар уйлый-уйлый,
Сез дә көтерсез әле.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.
```

# Перевод

### Нурия Абдуллина - Будьте живы-здоровы, дитя!

```
Скачет, пенится река,
Время мчится всё вперёд.
Словно жеребцы в разбеге,
Дети спешат, не ждут.

Кто-то уходит, кто-то вернётся -
Не удержать поток.
Сердце матери рвётся:
Будьте живы-здоровы, дитя!

Мать понимает - и они ведь
Тоже растят детей.
К счастью своих малышей
Они бегут скорей.

Кто-то уходит, кто-то вернётся -
Не удержать поток.
Сердце матери рвётся:
Будьте живы-здоровы, дитя!

И ваши объятья тоже
Крыльями станут сильней.
Сядете, мыслью к мысли,
Вы тоже повзрослеете.

Кто-то уходит, кто-то вернётся -
Не удержать поток.
Сердце матери рвётся:
Будьте живы-здоровы, дитя!
```
