# Оригинал

### Нәҗибә Сафина / Фәрит Хатыйпов - Сагынам (өченче вариант)

```
«Ни хәлләрең бар синең?» - дип,
Соравыңны сагынам.
Күңелемә сеңеп калган
Моңнарыңа табынам.
Сагынам, сагынам,
Моңнарыңа табынам.
Сагынам.

Гомеремнең юлларында,
Минут саен абынам.
Ә шулай да «очамын» дип,
Һаман күккә талпынам.
Сагынам, сагынам,
Һаман күккә талпынам.
Сагынам.

Галәм аша кулларымны
Сиңа сузам - зарыгам.
Чәчләреңә, күзләреңә,
Күңелеңә кагылам...
Сагынам, сагынам,
Күңелеңә кагылам...
Сагынам.
```

# Перевод

### Нәҗибә Сафина / Фәрит Хатыйпов - Скучаю (третья версия)

```
«Как ты там, скажи, родная?» -
Твой вопрос я вспоминаю.
Твой напев, в душе застывший,
Обожаю, им дышаю.
Скучаю, скучаю,
Твой напев, им дышаю.
Скучаю.

На дорогах долголетья
Каждый миг спотыкаюсь,
Но всё так же, молвя «лечу»,
К небу вновь устремляюсь.
Скучаю, скучаю,
К небу вновь устремляюсь.
Скучаю.

Через бездну мирозданья
Руки к тебе протягую,
До волос твоих, до глаз твоих,
До сердца твоего тянусь...
Скучаю, скучаю,
До сердца твоего тянусь...
Скучаю.
```
