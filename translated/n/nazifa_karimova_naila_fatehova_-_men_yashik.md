# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Мең яшик

```
Кайда татар яши - шунда татар иле.
Мәскәүләр дә күптән безнең туган ил.
Кайда гына бакма - татар урамнары
Моңлы Тукай килде, данлы Муса Җәлил.

Кушымта:
Бер булыйк,
Бергәләп мең яшик.

Мәскәүләрдә яшәү - үзе бер зур сынау.
Күз яшьләренә һич Мәскәү ышанмый.
Ничә гасыр инде алышына буын.
Татар турында төрле-төрле сүз тынмый.

Без бит шундый милләт: без бетмибез әле,
Мең үлеп түгелме мең тапкыр терелгән.
Чит, диеп, читләтмәгез - Татарстан чиге
Сызып үтә татарның йөрәк түреннән.
```

# Перевод

### Назифа Каримова / Наиля Фатехова - Тысяча лет живём

```
Где живёт татарин - там и татарская земля.
Москва тоже давно стала нашей родной страной.
Куда ни посмотри - татарские улицы,
Пришёл мелодичный Тукай, славный Муса Джалиль.

Припев:
Будем вместе,
Вместе тысячу лет живём.

Жить в Москве - само по себе великое испытание.
Слезам Москва никогда не верит.
Уже сколько веков сменяются поколения.
О татарах не умолкают самые разные слова.

Ведь мы такой народ: мы ещё не исчезнем,
Не умерли же, тысячи раз возродившись.
Не называйте чужим - граница Татарстана
Проходит по самому сердцу татарина.
```
