# Оригинал

### Нәсимә Абзалова / Илһам Хисмәтуллин - Чәчкәләрнең нигә кирәге бар?

```
Чәчкәләрнең нигә кирәге бар,
Өзеп бирер кешең булмаса?!
Нәрсә соң ул күктә янган кояш,
Үз кояшың балкып тормаса.

Син янымда булган чакта гына
Матур җырлар туа күңелемдә.
Эчкерсез саф мәхәббәтем минем
Юлдаш булсаң иде гомергә.

Тормыш юлын бергә башлаганбыз,
Ахыргача бергә булыйк без.
Авырлыклар килсә, бергә уртаклашыйк,
Шатлыкларны бергә бүлик без.
Бик бәхетле, иркәм, булыйк без,
Тормыш юлын бергә үтик без.
```

# Перевод

### Нәсимә Абзалова / Илһам Хисмәтуллин - Зачем нужны цветы?

```
Зачем нужны эти цветы,
Коль некому их подарить?
Что значит солнце в небесах,
Когда твоё не светит мне?

Лишь рядом ты - и в сердце моём
Рождаются красивые песни.
Моя любовь чиста, искрення,
Будь спутником мне навсегда.

Мы вместе путь свой начали,
Дойдём до конца вдвоём.
Придут трудности - разделим их,
И радости вдвоём поделим.
Будем счастливы, любимый мой,
Пойдём по жизни вместе.
```
