# Оригинал

### Никифор Тукмачев / Рамил Әсхәдулла - Җырым сиңа!

```
Күз карашым юлларда,
Каләмнәрем кулларда,
Сиңа атап бер җыр яздым
Сагынганда җырларга.

Дөнья буйлап җыр таралыр
Җырчыдан - җырчыларга,
Өзлеп яратканнар гына
Сагынып җыр чыгара.

Бу җырымда тик син генә
Аңлый алыр юллар бар,
Сәламнәрем җиткерергә
Ярый әле җырлар бар.

Дөнья буйлап җыр таралыр
Җырчыдан - җырчыларга,
Өзлеп яратканнар гына
Сагынып җыр чыгара.

Җырлыйм сиңа багышлап
Син җырлар яратканга,
Сүзләрем чыга җыр булып
Язгы таңнар атканда.

Дөнья буйлап җыр таралыр
Җырчыдан - җырчыларга,
Өзлеп яратканнар гына
Сагынып җыр чыгара.
```

# Перевод

### Никифор Тукмачев / Рамиль Ашадулла - Песня тебе!

```
Взгляд мой в строках,
Перо в руках,
Для тебя песню я написал,
Когда скучал, в песнях.

По всему миру песня разойдётся
От певца - к певцам,
Только те, кто любят сами,
Тоскуя, песни слагают.

В этой песне моей лишь ты
Поймёшь все строки,
Привет мой донести -
Есть же песни на свете.

По всему миру песня разойдётся
От певца - к певцам,
Только те, кто любят сами,
Тоскуя, песни слагают.

Пою для тебя с посвящением,
Раз ты песни любишь,
Слова мои льются песней,
Когда весенние рассветы встают.

По всему миру песня разойдётся
От певца - к певцам,
Только те, кто любят сами,
Тоскуя, песни слагают.
```
