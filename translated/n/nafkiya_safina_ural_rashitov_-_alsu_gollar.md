# Оригинал

### Нәфкыя Сафина / Урал Рәшитов - Алсу гөлләр

```
Үстерәсең хәзер гөлләр син,
Таралашып беткәч балалар.
Алсу гөлләр язын, язын чәчәк атып,
Йөрәгеңдә урын алалар.

Алсу гөлләр чәчәк бәйләме
Шаулап үсә тәрәз төбендә.
Әнкәй, бәгърем, сине онытмый
Саклый балаң күңел түрендә.

Әнкәй безне берчак онытмый,
Каршы чыгып ала таңнарда.
«Газизләрем нигә, нигә кайтмый?» - диеп,
Елап ала күңеле тулганда.

Алсу гөлләр чәчәк бәйләме
Шаулап үсә тәрәз төбендә.
Әнкәй, бәгърем, сине онытмый
Саклый балаң күңел түрендә.
```

# Перевод

### Нәфкыя Сафина / Урал Рәшитов - Алые цветы

```
Теперь растишь ты цветы одна,
Разлетелись дети по домам.
Алые цветы весной, весной распустятся
И в сердце место отведут мечтам.

Алые цветы - букет цветочный
Пышно разрастается в окне.
Мама, дорогая, не забуду -
Бережет дитя тебя в душе.

Мама нас ни разу не забудет,
На рассвете выйдет у ворот.
«Милые, зачем вы, зачем не едете?» -
Плачет, если сердце разольет.

Алые цветы - букет цветочный
Пышно разрастается в окне.
Мама, дорогая, не забуду -
Бережет дитя тебя в душе.
```
