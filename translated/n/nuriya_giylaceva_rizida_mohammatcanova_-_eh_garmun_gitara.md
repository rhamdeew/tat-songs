# Оригинал

### Нурия Гыйләҗева / Ризида Мөхәммәтҗанова - Эх, гармун, гитара!

```
Гармун алдым кулларыма
Телләренә басарга.
Гитара алдым кулыма,
Якын дусым ясарга.

Кушымта:
Эх, гармун, гитара,
Эх, гармун, гитара.
Икесе дә кызыктыра,
Син анда уйнап кара.

Кил әле син, кил әле син,
Гармун уйный бер егет.
Бер елмаеп көл әле син,
Җырлап, үзеңнеке ит!

Кушымта.

Гармунчының иң шәбе син,
Гармуныңны бир әле.
Телләрендә тылсым тулы,
Мин дә уйный(ы)м әле.

Кушымта.
```

# Перевод

### Нурия Гильяжева / Ризида Мухаммаджанова - Ах, гармонь, гитара!

```
Гармонь взял в свои руки,
Прикоснулся к струнам.
Гитару взял в руки,
Сделал близким другом.

Припев:
Ах, гармонь, гитара,
Ах, гармонь, гитара.
Оба инструмента интересны,
Ты поиграй немного.

Иди же ты, иди же ты,
Парень играет на гармони.
Ты хоть немного улыбнись,
Спой и стань своим!

Припев.

Ты - самый лучший гармонист,
Дай же гармонь сюда.
В струнах тайна содержится,
Я тоже поиграю.

Припев.
```
