# Оригинал

### Наҗар Нәҗми / Шамил Колбарисов - Үтте бу матур көн

```
Үтте бу матур көн.
Ачты күзләрен төн -
Уяндылар күктә йолдызлар.

Үтте бу матур көн
Җир йөзендә көн-төн
Керфек какмый торган нурлар бар.

Үтте бу матур көн
Йомды күзләреңне.
Кояш нуры айнын йөзендә.

Үтте бу матур көн
Калды кояшлы көн.
Кояшлы көн калды күңелдә.
```

# Перевод

### Наҗар Нәҗми / Шамил Колбарисов - Прошел этот прекрасный день

```
Прошел этот прекрасный день.
Ночь распахнула веки -
В небе пробудились звезды.

Прошел этот прекрасный день.
На земле и днем и ночью
Есть лучи, что не смыкают ресниц.

Прошел этот прекрасный день,
Сомкнулись твои глаза.
Луч солнца - на лице луны.

Прошел этот прекрасный день,
Остался солнечный день.
Солнечный день остался в сердце.
```
