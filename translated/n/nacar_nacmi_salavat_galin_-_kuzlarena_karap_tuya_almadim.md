# Оригинал

### Наҗар Нәҗми / Салават Галин - Күзләреңә карап туя алмадым

```
Күзләреңә карап туя алмадым,
Әйтә алмадым әйтер сүземне.
Һәр минутым йылдар кеүек оҙон,
Ярай әле йөрәк түҙемле...

Нурлы гөлләр чәчәк атар өчен
Җылы язны өзелеп сагына.
Гөлләр булсаң, җылы язлар булып,
Мин кайтырмын синең яныңа.
```

# Перевод

### Наҗар Нәҗми / Салават Галин - Не насытиться взглядом твоих глаз

```
Не насытился, глядя в твои глаза,
Не смог сказать слова, что хотел.
Каждая минута тянулась, как годы,
Хорошо, что сердце терпеливо...

Светлые цветы, чтобы расцвести,
С тоской до боли ждут тёплой весны.
Если ты - цветы, я стану тёплой весной,
И к тебе вернусь, к тебе одной.
```
