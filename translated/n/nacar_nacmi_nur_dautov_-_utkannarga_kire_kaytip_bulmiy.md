# Оригинал

### Наҗар Нәҗми / Нур Даутов - Үткәннәргә кире кайтып булмый

```
Үткәннәргә кире кайтып булмый,
Үткәннәргә, борылып карама.
Сүнгән утның син көлләрен сипмә
Үз яраңа, минем ярама...

Үткәннәргә кире кайтып булмый,
Таныш җиргә кайткан шикелле.
Булса да ул үзе тик безнеке,
Безнеке дә түгел бит инде.
```

# Перевод

### Нәҗар Нәҗми / Нур Даутов - К прошлому назад не воротиться

```
К прошлому назад не воротиться,
К прошлому не оборачивайся, не смотри.
Пепел угасшего огня не сыпь ты
Ни на свои, ни на мои раны...

К прошлому назад не воротиться,
Словно в знакомые края вернуться.
Даже если это лишь наше,
Теперь уже оно не наше вовсе.
```
