# Оригинал

### Нәҗибә Сафина / Людмила Аланлы - Син көткән яңа ел

```
Син көткән яңа ел - сизәсеңме?
Шатлыклары мөлдерәмә.
Күк йөзендә болыт түгел,
Безгә дигән бәхет йөзә.

Йолдыз булып, карлар булып,
Безгә дигән бәхет килә.
Шатлык, сөенечләр белән
Яңа еллар гөрләп килә!

Син көткән яңа ел! Йә, каршы ал!
Ак хис булып оча карлар.
Ниләр сөйли ябалак кар:
Бәхетләр күп, бетсен моң-зар!

Йолдыз булып, карлар булып,
Безгә дигән бәхет килә.
Шатлык, сөенечләр белән
Яңа еллар гөрләп килә!
```

# Перевод

```
Скучала ли ты по Новому году - чувствуешь ли?
Радости его не уловимы.
На небесах нет облаков,
Счастье плывёт к нам.

В виде звёзд, в виде снега
Счастье приходит к нам.
С радостью, утешением
Новые года стучатся!

Скучала ли ты по Новому году! Да, встречай!
Снега летают белыми чувствами.
Что говорит пушистый снег:
Пусть будет много счастья, пусть закончится горе!

В виде звёзд, в виде снега
Счастье приходит к нам.
С радостью, утешением
```
