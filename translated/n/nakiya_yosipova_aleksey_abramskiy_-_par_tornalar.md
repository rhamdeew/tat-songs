# Оригинал

### Нәкыя Йосыпова / Алексей Абрамский - Пар торналар

```
Cанап калдым, парлап-парлап,
Парсызлыкның хәлен аңлап.
Берсе калды, кыйгак-кыйгак,
Йөрәк әрни аны уйлап.

Зәңгәр күктә минем күзем,
Кайтырсыз дип сезне күзлим,
Йөрәгемдә шомнар туа,
Китәрсез дип тагын көзен.

Канат-канат, кабат-кабат
Кайтырсыз сез парлап-парлап.
Хыялымны каршы алам,
Пар торналар сезгә карап.
```

# Перевод

### Нәкыя Йосыпова / Алексей Абрамский - Парные журавли

```
Считал я вас, по парам, по парам,
Без пары - знаю, какова та кара.
Один остался: крик да крик,
Сердце ноет, думая о нем.

В синем небе - мой взгляд,
Жду вас: вернетесь ли назад,
В сердце тени, тревоги встают -
Вновь осенью вы улетите, уйдете.

Крыло к крылу, раз за разом
Вы вернетесь - парами, рядом.
Я встречаю свою мечту,
Парным журавлям в глаза гляжу.
```
