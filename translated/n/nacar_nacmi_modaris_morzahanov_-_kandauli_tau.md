# Оригинал

### Наҗар Нәҗми / Мөдәрис Морзаханов - Кандаулы тау

```
Кандаулы тау текә-текә,
Менгәч, текәсе бетә.
Текә күңелле сөйгән яр
Тауга менгәнне көтә,
Менгәч, текәсе бетә.

Кандаулы тау башы төтен -
Учаклар сүнмәгән чак,
Учаклар сүнгән чакта да
Күңел сүрелмәгән чак,
Учаклар сүнмәгән чак.

Кандаулы тау асты болын,
Агыйдел алкын ага.
Зәңгәр киңлекләр үзенә
Җаныңа тартып ала,
Агыйдел алкын ага.

Кандаулы тау башында җил,
Бүген җил тынган икән.
Туған җирнең (якның) гүзәллеген
Кем күреп туйган икән?
Кем җырлап туйган икән?
```

# Перевод

### Назар Наджми / Мударис Морзаханов - Кандаулы тау

```
Гора Кандаулы крута-крута,
Поднимешься - крутизны нет.
Возлюбленную с сердцем крутым
Ждёт тот, кто взойдёт на гору,
Поднимешься - крутизны нет.

Вершина Кандаулы в дыму -
То время, когда очаги не гасли,
И даже когда очаги гасли,
Душа не остывала,
То время, когда очаги не гасли.

Под Кандаулы - луг широкий,
Агыйдел шумливо течёт.
Синие просторы к себе
Твою душу притягивают,
Агыйдел шумливо течёт.

На вершине Кандаулы - ветер,
Сегодня, видно, стих.
Красотою родной земли (стороны)
Кто когда насытился?
Кто песней насытился?
```
