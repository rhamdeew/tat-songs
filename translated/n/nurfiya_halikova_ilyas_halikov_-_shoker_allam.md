# Оригинал

### Нурфия Халикова / Ильяс Халиков - Шөкер, Аллам

```
Шөкер, Аллам, без дә кабат
Хак иманга кайтабыз.
Корьән аятьләре укып
Үссен балаларыбыз.
Ашык син дә икендегә,
Аллаһ әҗерен бирер.
Ахшам намазына кадәр -
Теләк әйтергә өлгер -
Искәртеп барча жиhанга,
Азан намазга дәшә.
```

# Перевод

### Нурфия Халикова / Ильяс Халиков - Благодарю, Аллах

```
Благодарю, Аллах, мы вновь
К истинной вере возвращаемся.
Читая аяты Корана,
Пусть подрастают наши дети.
Спеши и ты к икенде,
Аллах награду дарует.
До вечерней молитвы -
Успей желание сказать -
Напоминая всему миру,
Азан к намазу зовет.
```
