# Оригинал

### Наҗар Нәҗми / Рим Хәсәнов - Миндә син бар!..

```
Ике йөрәк - ике йолдыз...
Йолдыз яна һаваларда.
Миндә син бар!..
Миндә син бар? -
Аралар юк араларда
Аралар юк араларда.

Ике йөрәк - ике бөркет,
Бөркет оча биекләрдә.
Миндә син бар!..
Миндә син бар! -
Нурлы канат өметләрдә,
Канатлылар биекләрдә.

Ике йөрәк - ике аккош,
Томан аккош күлләрендә.
Миндә син бар!..
Миндә син бар! -
Томаннар юк йөрәгемдә,
Язгы таңнар йөрәгемдә.
```

# Перевод

### Наҗар Нәҗми / Рим Хәсәнов - Во мне ты есть!..

```
Два сердца - две звезды...
Звёзды горят в высотах неба.
Во мне ты есть!..
Во мне ты есть! -
Нет расстояний меж нами,
Нет расстояний меж нами.

Два сердца - два орла,
Орлы парят в небесной выси.
Во мне ты есть!..
Во мне ты есть! -
Крылья света в надеждах,
Крылатые - в высотах.

Два сердца - два лебедя,
В тумане лебеди на озёрах.
Во мне ты есть!..
Во мне ты есть! -
Нет туманов в моём сердце,
Весенние рассветы в моём сердце.
```
