# Оригинал

### Наҗар Нәҗми / Нариман Сабитов - Агыйделдә ак томан (икенче вариант)

```
Агыйделдә ак томан,
Параход тавышлары.
Дымлы керфекләр шикелле,
Ярында камышлары.
Ярында камышлары,
Параход тавышлары.
Ярында камышлары,
Параход тавышлары.

Агыйделдә ак томан,
Ат кешни болыннарда.
Ат эзләп йөргән эзләрем
Калган шул урыннарда.
Калган шул урыннарда,
Чыкланган болыннарда.
Калган шул урыннарда,
Чыкланган болыннарда.

Агыйделдә ак томан,
Борылган Идел белән.
Нигә шул кадәр якынсың,
Әй, туган Иделгенәм?!
Әй, туган Иделгенәм!
Йөрәгем синең белән!
Әй, туган Иделгенәм!
Йөрәгем синең белән!
```

# Перевод

### Наҗар Нәҗми / Нариман Сабитов - Белый туман на Агидели (второй вариант)

```
На Агидели белый туман,
Голоса пароходов звенят.
Словно влажные ресницы, дрожат
Камыши у ее берегов.
Камыши у ее берегов,
Голоса пароходов звенят.
Камыши у ее берегов,
Голоса пароходов звенят.

На Агидели белый туман,
Конь ржет по лугах на просторе.
Следы, что искали коня, навсегда
Остались на тех же просторах.
Остались на тех же просторах,
На лугах, омытых росой.
Остались на тех же просторах,
На лугах, омытых росой.

На Агидели белый туман,
На изгибе великой Идели.
Почему ты так близка для меня,
О, родная Иделем моя?!
О, родная Иделем моя!
Сердце мое с тобой!
О, родная Иделем моя!
Сердце мое с тобой!
```
