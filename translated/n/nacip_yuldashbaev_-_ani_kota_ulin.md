# Оригинал

### Нәҗип Юлдашбаев - Әни көтә улын

```
Тагы тора кояшлы май,
Күктә кошлар кайталар.
Тирә-якка асыл гөлләр
Гөрләп чәчәк аталар.

Әни көтә улын, әле кайтыр, диеп,
Әни көтә улын, исән, диеп.

Көтә иде әни улын,
Нигә улы кайтмады?
Чәчәк атты ун тугыз ел,
Нигә башка атмады?

Әнкәй елый, улым кайда калды? - диеп,
Әнкәй елый, җирдә бәхет юк, диеп.

Әни тора, башын игән,
Сары чәчәк - кулында.
Бу әнине тагын ниләр
Көтә тормыш юлында?

Әнкәй әйтми башка, улым, саумы, диеп,
Әнкәй әйтми башка, улым, саумы, диеп.
Әни көтә улын, әле кайтыр, диеп,
Әни көтә улын, исән, диеп.
```

# Перевод

### Нәҗип Юлдашбаев - Мама ждет сына

```
Снова встает солнечный май,
В небо птицы возвращаются.
Вокруг благородные цветы
Буйно вновь распускаются.

Мама ждет сына: вернется же, думает,
Мама ждет сына: жив он, думает.

Мама ждала сына,
Почему же сын не пришел?
Цвел девятнадцать лет цветок -
Почему потом не расцвел?

Мать рыдает: где же сын остался? - думает,
Мать рыдает: нет счастья на земле.

Мать стоит, склонясь устало,
Желтый цветок - в ладони.
Что еще ее, бедняжку,
Ждет на жизненной дороге?

Мать не скажет больше: «Сынок, здравствуй», думает,
Мать не скажет больше: «Сынок, здравствуй», думает.
Мама ждет сына: вернется же, думает,
Мама ждет сына: жив он, думает.
```
