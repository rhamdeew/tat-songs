# Оригинал

### Наҗар Нәҗми / Фәрит Хатыйпов - Язгы урман

```
Агачлар яфрак ярганда,
Мең - мең кошлар сайраганда,
Мин тәү күрдем сине.
Әйтерсең, яз шул чагында,
Бүләк итеп кочагында,
Сине алып килде.

Юлларыма юлдаш итсәм,
Синең белән китсәм, китсәм,
Мин адашмам микән?
Язгы яшел, яшь урманда,
Синең белән адашсам да,
Миңа рәхәт, иркәм.
```

# Перевод

### Нажар Нажми / Фарит Хатыпов - Летний лес

```
Когда деревья листья сбросили,
Тысячи птиц запели в небе,
Я впервые увидел тебя.
Скажи, лето в тот миг пришло,
Притащив тебя в объятьях,
Принесло тебя как дар.

Если сделаю тебя спутницей,
Если пойду с тобой, пойду,
Не заблужусь ли я?
В зеленом, молодом лесу,
Даже если заблужусь с тобой,
Мне легко и просторно.
```
