# Оригинал

### Наил Касыймов / Татар халык көе - Ай, Дунай

```
Ай, Дунайда айлы кичтә
Су эчерәм атларга.
Ай кояшка әйләнде бит
Син килгәч бу якларга.

Ай кояшка әйләнде дә
Йокыларым качырды.
Синең килүеңне (елмаюын) күреп
Күк капусы (кабагы) ачылды.

Ай, Дунайда күрдем әле
Йолдыздан яшь тамганын.
Тик бер үзем генә беләм:
Синсез яши алмамын.
```

# Перевод

### Наил Касымов / Татарская народная мелодия - Ай, Дунай

```
Ай, на Дунае в лунную ночь
Воду пою коням.
Ай, на солнце превратилось, ведь,
Когда ты пришла в эти края.

Ай, на солнце превратилось оно
И прогнало мои сны.
Видя твоё пришествие (улыбку)
Отворились небесные врата.

Ай, на Дунае видел я
Как звезда роняет слёзы.
Только сам я только знаю:
Без тебя не смогу жить.
```
