# Оригинал

### Нияз Акмал / Фәрит Хатыйпов - Туган тел турыңда

```
Туган телдә һәрбер хисне
Җөмләгә төреп була.
Әйтергә теләгәнеңне
Әйтеп бетереп була.

Туган телдә тик бер хисне
Җөмләгә төреп булмый:
Аны ничек яратканны
Әйтеп бетереп булмый...
```

# Перевод

### Нияз Акмал / Фәрит Хатыйпов - О родном языке

```
На родном языке любое чувство
В фразу можно облечь.
Все, что сердце сказать желает,
Можно до конца изречь.

Но одно лишь чувство родным языком
В слова не облечь:
Как сильно его полюбил -
Нельзя до конца перечь...
```
