# Оригинал

```
Дуслык җыры тынмый яңгырасын,
Җиребезне ай-хак мең урасын
Иң матур җыр бездә - дуслык җырлары, җырлары.
Күңелләрдән күчкән күңелләргә
Җәйгор булып төшкән киң җирләргә
Иң матур җыр бездә - дуслык җырлары, җырлары.

Сабуллашып ерак китсәктә,
Без кайларда гына җитсәктә,
Дуслык җыр калыр йөрәктә, йөрәктә.

Кайгы түгел, шатлык-нур өләшеп,
Авырлыклар җиңдек бергәләшеп,
Торган җирләр, безнең башка уйлар бер, уйлар бер.
Бер хыяллып, бер теләкле итеп,
Мәңгелеккә хуш йөрәкле итеп,
Тоташтырган безне олы туган җир, туган җир.

Сабуллашып ерак китсәктә,
Без кайларда гына җитсәктә,
Дуслык җыр калыр йөрәктә, йөрәктә.

Бергә инде ярсып ут-суларга,
Бер дәртеңне алып йолдызларга,
Еллар агышында дуслык чыныкты, дуслык чыныкты.
Сагындырып кына очраштырып,
Очраштыргач тагы хушлаштырып,
Сыный икән вакыт безнең дуслыкны, дуслыкны.

Сабуллашып ерак китсәктә,
Без кайларда гына җитсәктә,
Дуслык җыр калыр йөрәктә, йөрәктә.
```
