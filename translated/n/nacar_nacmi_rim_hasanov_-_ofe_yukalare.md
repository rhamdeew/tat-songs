# Оригинал

### Наҗар Нәҗми / Рим Хәсәнов - Өфе юкәләре

```
Яшьлек урамнары,
Тулы хыяллары,
Без - шатлыкка туймас исәрләр.
Өфе юкәләре
Безнең хакта да әле
Яфрак яра-яра сөйләрләр.

Безнең менгән атлар,
Еллар, аргамаклар,
Көмеш тезген, алтын йөгәннәр.
Өфе юкәләре
Безнең хакта да әле
Чәчәк ата-ата сөйләрләр.

Гомер ике килми,
Утлар ике сүнми,
Без - мең янып тик бер сүнәрләр.
Өфе юкәләре
Онытмаслар әле,
Яфрак коя-коя сөйләрләр.
```

# Перевод

### Наҗар Нәҗми / Рим Хәсәнов - Уфимские липы

```
Улицы юности,
Полные грёз,
Мы - безумцы, не знавшие меры в счастье.
Уфимские липы
О нас ещё
Будут шептать, распуская листву.

Наши кони резвые,
Годы - скакуны,
Серебряные поводья, золотые узды.
Уфимские липы
О нас ещё
Будут петь, расцветая цветами.

Жизнь не приходит дважды,
Огонь не гаснет дважды,
Мы - сгорать тысячью, угаснуть лишь раз.
Уфимские липы
Не позабудут нас,
Будут шептать, роняя листву.
```
