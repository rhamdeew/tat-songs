# Оригинал

### Наилә Сарманаева / Илдус Низамов - Туйга бүләк

```
Язлар җитеп килә, бозлар агып китә
Ярлар белән гүя серләшеп.
Үткән артта кала, гомер алга чаба
Үткән юллар кала тезелеп.

Туйлар килә-китә
Гомер шулай итә,
Ак бәхетләр телим мин сезгә!

Җырлап үтсен гомер, тыныч булсын күңел,
Швтлык төеп типсен йөрәкләр.
Олы мәхәббәтнең кайнар хисен тойсын
Чын күңелдән сөя белгәннәр.

Туйлар килә-китә
Гомер шулай итә,
Ак бәхетләр телим мин сезгә!

Яшьлек бирешмәгән, хисләр сүрелмәгән,
Кабынмаган алар сүнәргә.
Бергә яккан учак янсын һаман дөрләп,
Ике йөрәк бергә типкәндә.

Туйлар килә-китә
Гомер шулай итә,
Ак бәхетләр телим мин сезгә
```

# Перевод

### Наиля Сармаева / Ильдус Низамов - Подарок на свадьбу

```
Весна подходит, льды уходят вдаль
И будто шепчутся с берегами.
Прошлое остаётся позади, жизнь мчится вперёд
Оставляя пройденные дороги позади.

Свадьбы приходят и уходят
Такова жизнь,
Желаю вам белого счастья!

Пусть жизнь пройдёт в песнях, пусть будет покой в душе,
Пусть сердца стучат в унисон.
Пусть почувствуют горячее чувство великой любви
Те, кто умеет любить от всей души.

Свадьбы приходят и уходят
Такова жизнь,
Желаю вам белого счастья!

Молодость не сдалась, чувства не угасли,
Они не готовы погаснуть.
Пусть разгорается костёр, что зажжён вместе,
Когда два сердца бьются в унисон.

Свадьбы приходят и уходят
Такова жизнь,
Желаю вам белого счастья!
```
