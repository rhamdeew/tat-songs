# Оригинал

### Нәфисә Хәбибдиярова / Виталий Агапов - Миләүшәле күл

```
Күл буенда йөрдем мин,
Балачагымны эзләп:
Су төбендә - яшел үлән,
Яр читендә - су тездән.

Әнә зәңгәр күлмәкле кыз
Күл буеннан йөгерә,
Баскан эзләренә аның
Миләүшәләр тезелә.

Авылымның зәңгәр күле
Мине хәтерли микән?
Күл тирәли камыш түгел,
Зәңгәр миләүшә үскән.

Әнә зәңгәр күлмәкле кыз
Күл буеннан йөгерә,
Баскан эзләренә аның
Миләүшәләр тезелә.

Сары чәчәк арасында
Шәмәхә миләүшәләр.
Балачагым булып, миңа
Ерактан эндәшәләр.

Әнә зәңгәр күлмәкле кыз
Күл буеннан йөгерә,
Баскан эзләренә аның
Миләүшәләр тезелә.
```

# Перевод

### Нафиса Хабибдиярова / Виталий Агапов - Озеро, усыпанное лилиями

```
У озера я бродил,
Детство свое ища:
На дне - зелень травы,
На краю - у воды цветы.

Вот в голубом платье девочка
Вдоль озера бежит,
На следы её ног
Лилии ложатся.

Голубое озеро моего села
Помнит ли меня?
Вокруг озера не камыш растет,
Голубые лилии цветут.

Вот в голубом платье девочка
Вдоль озера бежит,
На следы её ног
Лилии ложатся.

Среди желтых цветов
Белые лилии.
Станет моим детством,
Издалека шепчутся мне.

Вот в голубом платье девочка
Вдоль озера бежит,
На следы её ног
Лилии ложатся.
```
