# Оригинал

### Нәҗип Әсәнбаев / Рим Хәсәнов - Зәйтүнгөл (беренче вариант)

```
Бер үк сукмаклардан йөрегәнбез,
Бер чишмәнең суын эчкәнбез.

Ничек яшәр идек бу дөньяда,
Әгәр икәү бергә булмасак?
Бәхет кошы читләп үтәр иде,
Бер-беребез өчен тумасак!

Бәхет тә бит сайрар кошка охшаш,
Аның да бар ике канаты;
Бер канаты аның - мәхәббәте,
Икенчесе - сайрар бакчасы.
```

# Перевод

### Нажип Асанбаев / Рим Хасанов - Зайтунгуль (первый вариант)

```
Мы шагали по одним тропинкам,
Из одного родника воду пили.

Как бы жили мы на этом свете,
Если б вместе нам не суждено?
Птица счастья стороной бы пролетела,
Коль не встретились бы мы давно!

Счастье тоже словно птица поющая,
Есть у него два крыла;
Одно крыло его - любовь большая,
Второе - сад, где песня расцвела.
```
