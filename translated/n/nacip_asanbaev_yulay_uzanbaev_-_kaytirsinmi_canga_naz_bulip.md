# Оригинал

### Нәҗип Әсәнбаев / Юлай Үзәнбаев - Кайтырсыңмы җанга наз булып?..

```
Киек казлар оча каңгылдашып,
Түшләренә җилләр урала.
Сагынулары, каурый-каурый булып,
Ятып калды безнең урамга.

Аста яшел сизәм озата бара,
Яулык болгый күктән ак болыт.
Еракларга елап киткән казлар (парлар)
Җырлап кайтыр илгә яз булып.

„Сагынырбыз" дигән җырларыбыз
Калды зәңгәр күктә язылып.
Минем дә бит язлар көткән мәлем,
Кайтырсыңмы җанга наз булып?..
```

# Перевод

### Нәҗип Әсәнбаев / Юлай Үзәнбаев - Вернёшься ли ты, лаской для души?..

```
Дикие гуси летят, перекликаясь,
Ветры обвивают их грудь.
Их тоска, перышко к перышку,
Легла на нашу улицу.

Внизу зелёный луг провожает,
С неба машет белое облако.
Гуси (пары), плача, ушли в дали,
Песней вернутся в родной край весной.

Наши песни «будем скучать»
Остались в синем небе строками.
И я ведь жду весенних дней -
Вернёшься ли ты, лаской для души?..
```
