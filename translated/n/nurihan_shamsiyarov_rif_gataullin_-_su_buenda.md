# Оригинал

### Нурихан Шәмсияров / Риф Гатауллин - Су буенда

```
Су буенда, уйга талып,
Талларны тибрәтәсең.
Көйгә салып уйларыңны,
Әйт, кемне уятасың?!

Агым сулар, дулкынланып,
Синең моңнан чайкала.
Сулар ага, йөрәк яна,
Яшьлектә әрнү кала.

Су буенда ялгыз калсаң,
Су зиһеннәрне ала.
Идел дә бит ике ярда,
Сизәмсең, ярың кала?!
```

# Перевод

### Нурихан Шәмсияров / Риф Гатауллин - У реки

```
У реки, погрузившись в думы,
Ты дрожишь, ветви тревожа.
Положив мысли на мотив,
Скажи, кого будишь ты?!

Быстрая вода, волнуясь,
От твоей мелодии плещется.
Воды текут, сердце горит,
В молодости печаль остаётся.

Если у реки останешься один,
Вода забирает сознание.
Ведь Идел (Волга) с двух берегов,
Чувствуешь - твой любимый остаётся?!
```
