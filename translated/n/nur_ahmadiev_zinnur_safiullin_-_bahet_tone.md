# Оригинал

### Нур Әхмәдиев / Зиннур Сафиуллин - Бәхет төне

```
Каеннары ай-яй ак икән,
Сөю белән тулы чак икән,
Каеннардан якты моң тама.

Бу бит бәхет төне,
Эзлә бәхетеңне,
Юк, йоклама, егет, йоклама!

Егет яна кызга күз атып,
Сандугачлар йөри озатып,
Йолдызлардан энҗе сибелә.

Табышалар парлар,
Кавышалар ярлар,
Юк, бәгърем, юк бит син генә.

Бәхет төне узып китәрме,
Безгә бәхет биреп китәрме?
Каеннардан сора, каеннан.

Күңелемнең күркен
Яулый алмас беркем -
Исемнәрең каеп язылган!
```

# Перевод

### Нур Әхмәдиев / Зиннур Сафиуллин - Ночь счастья

```
Березы - ах, белы до боли,
Пора любви полна до дна,
С берез струится светлый напев.

Это ночь счастья,
Ищи свое счастье,
Нет, не спи, джигит, не спи!

Парень горит, на девичий взор,
Соловьи его провожают,
С звезд осыпаются жемчужины.

Находят пары,
Сходятся берега,
Нет, родной мой, нет, не ты один.

Уйдет ли ночь счастья,
Подарив нам счастье?
Спроси у берез, у березы.

Красоту моей души
Никто не сможет покорить -
Твои имена на ней вырезаны!
```
