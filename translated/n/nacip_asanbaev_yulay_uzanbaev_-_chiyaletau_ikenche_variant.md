# Оригинал

### Нәҗип Әсәнбаев / Юлай Үзәнбаев - Чиялетау (икенче вариант)

```
Чиялетау буйларында
Калды яшьлек эзләрем.
Диңгез кичеп, чит җирләрдән
Мин бәхетләр эзләдем!

Зәңгәр офыклар артында
Чиялетауым калды.
Ак тирмәле зәңгәр күктә
Ата-бабалар калды.

Кояш чыкса офыклардан,
Чиялетауны күрәм.
Туган илем, сине уйлап,
Күз яшьләремне түгәм!...
```

# Перевод

### Нәҗип Әсәнбаев / Юлай Үзәнбаев - Чиялетау (второй вариант)

```
У берегов Чиялетау
Остались юности следы.
Через моря, в чужих краях
Я счастья искал, я к нему шел!

За синью дальних горизонтов
Мой Чиялетау остался.
В голубом небе с белой юртой
Остались предки, род и память.

Солнце взойдет из-за окраин -
Чиялетау вновь увижу.
Родная земля, о тебе думая,
Я слезы роняю...
```
