# Оригинал

### Айдар Галимов - Синен кузлэр

```
Синен кузлэр куз тугелдер,
Алар тирэн коедыр,
Кузлэреннен коесыннан
Шифа суга коендыр.
Синен куллар кул тугелдер,
Алар аккош канаты,
Пар аккоштай йорсэк иде
Бер-беренне яратып.

Кояш балкыган йозенэ
Тутырып карыйм эле,
Кузлэрендэ бер нур булып,
Мэнгегэ калыйм эле.

Синен йозен йоз тугелдер,
Ул бит кояш нурыдыр,
Кояш кебек елмаюын
Нурлар белэн тулыдыр.
Синен кашлар каш тугелдер,
Айнын кыйгач дугасы,
Айга карап котеп алыйк
Уртак таннар туасын.

Кояш балкыган йозенэ
Тутырып карыйм эле,
Кузлэрендэ бер нур булып,
Мэнгегэ калыйм эле.
```

# Перевод

### Айдар Галимов - Твои глаза

```
Твои глаза не глаза,
Они глубокий колодец,
Из колодца глаз твоих
Целебной водой окунаюсь.
Твои руки не руки,
Они лебединые крылья,
Парой лебедей ходили бы,
Друг друга любя.

В твоё солнцем озарённое лицо
Не насмотрюсь,
В глазах твоих светом став,
Навеки останусь.

Твоё лицо не лицо,
Оно ведь солнечный луч,
Как солнце улыбка твоя
Лучами полна.
Твои брови не брови,
Луны косая дуга,
На луну глядя, подождём,
Пока общие рассветы встанут.

В твоё солнцем озарённое лицо
Не насмотрюсь,
В глазах твоих светом став,
Навеки останусь.
```
