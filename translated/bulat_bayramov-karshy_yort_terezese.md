# Оригинал

### Булат Байрамов - Каршы йорт тэрэзэсе

```
Каршы йорт тэрэзэсендэ
Курдем кызнын гузэлен.
Йорэгемдэ ут кабынды,
Озгэлэнэ узэгем.
Озгэлэнэ узэгем...

Каршы йорт тэрэзэсе,
Каршы йорт тэрэзэсе,
Кон дэ ометлэнеп котэм
Ачылганын пэрдэсен.

Кыз мине уртэгэн сыман,
Тэрэзэдэн куренми.
Лэкин ул куренмэгэннэн
Мэхэббэтем суренми.
Мэхэббэтем суренми...

Каршы йорт тэрэзэсе,
Каршы йорт тэрэзэсе,
Кон дэ ометлэнеп котэм
Ачылганын пэрдэсен.

Хэзер курэсем дэ килми
Каршы йорт тэрэзэсен.
Тэрэз дорес курсэтми ул
Хатын-кызнын шэулэсен.
Озгэлэнмэ узэгем...

Каршы йорт тэрэзэсе,
Каршы йорт тэрэзэсе,
Кон дэ ометлэнеп котэм
Ачылганын пэрдэсен.
```

# Перевод

### Булат Байрамов - Окно дома напротив

```
В окне дома напротив
Увидел красоту девушки.
В сердце моем огонь загорелся,
Душа моя встревожилась.
Душа моя встревожилась...

Окно дома напротив,
Окно дома напротив,
Каждый день с надеждой жду
Открытия занавески.

Девушка словно избегает меня,
В окне не показывается.
Но от того, что её не видно,
Любовь моя не угасает.
Любовь моя не угасает...

Окно дома напротив,
Окно дома напротив,
Каждый день с надеждой жду
Открытия занавески.

Теперь и смотреть не хочется
На окно дома напротив.
Окно правдиво не показывает
Сияние красавицы.
Не тревожься, душа моя...

Окно дома напротив,
Окно дома напротив,
Каждый день с надеждой жду
Открытия занавески.
```
