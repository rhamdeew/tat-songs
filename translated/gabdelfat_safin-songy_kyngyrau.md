# Оригинал

### Габдельфат Сафин - Сонгы кынгырау

```
Кинэт кенэ гашыйк булдым
Гонахсыз кузлэренэ
Серле нурлар чагылды кук
Самими йозлэрендэ.
Нигэ диеп мин карадым
Син йозгэндэ ак жилдэ
Укучым иден лэ минем
Нигэ керден кунелгэ.

Сонгы кынгырау чынлама
Йорэк бар тыя алмаган
Уз укучысын яратып,
Яратып туя алмаган.

Котеп алам дэресемне
Сэгате тэулеккэ тин
Бер карасам мэхэббэтем
Бер карсам сенлем син.
Такта янына чыкканда
Кагылып киттен ялгыш
Язмышларда ялгышлар бар,
Бу ялгыш татлы ялгыш.

Сонгы кынгырау чынлама
Йорэк бар тыя алмаган
Уз укучысын яратып,
Яратып туя алмаган.

Син дэ тойдын ,мин дэ тойдым
Жылылык араларда
Нинди кузлэр белэн карыйм
Дэрестэ балаларга
Ничэмэ куз арасыннан
Туя алмыйм сина карап
Назлы каршына урап
Дэшэсен ярат,ярат.

Сонгы кынгырау чынлама
Йорэк бар тыя алмаган
Уз укучысын яратып,
Яратып туя алмаган.
```

# Перевод

### Габдельфат Сафин - Последний звонок

```
Внезапно влюбился
В невинные глаза её,
Таинственные лучи отразились
На искреннем лице.
Зачем я посмотрел,
Когда ты шла в белом платье,
Ты была моей ученицей,
Зачем вошла в сердце.

Последний звонок, не звени,
Сердце не может сдержаться,
Свою ученицу полюбив,
Полюбив, не может насытиться.

Готовлю свой урок
Сутками напролёт,
Один взгляд - моя любовь,
Один взгляд - и ты рядом.
Когда вышла к доске,
Случилась ошибка,
В судьбах бывают ошибки,
Эта ошибка - сладкая ошибка.

Последний звонок, не звени,
Сердце не может сдержаться,
Свою ученицу полюбив,
Полюбив, не может насытиться.

И ты почувствовала, и я почувствовал
Тепло между нами,
Какими глазами смотреть
На детей на уроке?
Среди стольких глаз
Не могу насмотреться на тебя,
Нежному взгляду поддавшись,
Сердце шепчет: люби, люби.

Последний звонок, не звени,
Сердце не может сдержаться,
Свою ученицу полюбив,
Полюбив, не может насытиться.
```
