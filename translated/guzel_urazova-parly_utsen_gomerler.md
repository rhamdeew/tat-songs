# Оригинал

### Гүзәл Уразова - Парлы утсен гомерлэр

```
"Гузэл донья, серле донья,
Донья тугэрэк!"- дилэр.
Доньянын бар ямен тоя
Парлашып яши кемнэр.

Аклансын изге хыяллар,
Уртак булсын бэхетлэр!
Уз ярын тапсын ялгызлар,
Парлы утсен гомерлэр!

Кин кунелгэ бар да сыя,
Яхшы булгач ниятлэр.
Парлы тормыш матурлана,
Булса куркэм гадэтлэр.

Аклансын изге хыяллар,
Уртак булсын бэхетлэр!
Уз ярын тапсын ялгызлар,
Парлы утсен гомерлэр!

Жырлар тынлап кем монлана?..
Ярсыз, парсыз кунеллэр...
Уз илендэ ялгызлар да,
Юк!.. Бэхетсез тугеллэр!

Аклансын изге хыяллар,
Уртак булсын бэхетлэр!
Уз ярын тапсын ялгызлар,
Парлы утсен гомерлэр!
```

# Перевод

```
"Прекрасный мир, таинственный мир,
Мир круглый!" - говорят.
Кто живет парами,
Наслаждаясь всей красотой мира.

Пусть светлеют святые мечты,
Пусть будет общим счастье!
Пусть найдут свою половину одинокие,
Парами пусть пройдут века!

В широкую душу все вмещается,
Когда намерения добры.
Парная жизнь прекрасна,
Если есть красивые традиции.

Пусть светлеют святые мечты,
Пусть будет общим счастье!
Пусть найдут свою половину одинокие,
Парами пусть пройдут века!

Кто грустит, слушая песни?..
Сердца без любви, без пары...
Даже одинокие в своей стране
Нет!.. Не бывают несчастными!

Пусть светлеют святые мечты,
Пусть будет общим счастье!
Пусть найдут свою половину одинокие,
```
