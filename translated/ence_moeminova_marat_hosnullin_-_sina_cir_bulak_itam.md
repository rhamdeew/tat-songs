# Оригинал

### Энҗе Мөэминова / Марат Хөснуллин - Сиңа җыр бүләк итәм

```
Исемеңне әйтми генә
Сиңа җыр бүләк итәм.
Минем ничек сөюемне
Соңлап аңларсың, иркәм.

Исемеңне әйтсәм әгәр,
Җыр таралыр дөньяга.
Исемеңне ишетерләр,
Ятлар керер арага.

Исемеңне әйтми генә
Сиңа җыр бүләк итәм.
Җырым яныңа килер дә
Серемне сөйләп китәр.

Исемеңне әйтсәм әгәр,
Җыр таралыр дөньяга.
Исемеңне ишетерләр,
Ятлар керер арага.
```

# Перевод

### Энже Мөэминова / Марат Хөснуллин - Я дарю тебе песню

```
Не называю имя
Я дарю тебе песню.
Как я люблю тебя нежно,
Ты поймешь потом, моя радость.

Если скажу я имя,
Песня разлетится по миру.
Имя услышат все,
Песни войдут в сердца.

Не называю имя
Я дарю тебе песню.
Песня подойдёт к тебе
И расскажет мой секрет.

Если скажу я имя,
Песня разлетится по миру.
Имя услышат все,
Песни войдут в сердца.
```
