# Оригинал

### Альбина Апанаева - Сою голе сулмасын

```
Кубэлэктэй ботерелэ
Ак кулмэкле яшь кэлэш
Кумэк шатлыклар берлэшэ
Кавышкач ике сердэш

Чэчэк аткач чэчэк голе
Гомерлеккэ сулмасын
Мэхэббэт чэчэклэренэ
Яманнар кул сузмасын

Тормыш сукмагына чыгып
Табышкач ике йорэк
Язмыш юлыннан узганда
Мэхэббэт булсын терэк

Бэхетле гомерлэр килсен
Кунелдэ изге телэк
Берсен берсе соеп, назлап
Яшэсен ике йорэк

Язмыш сынавын утсеннэр
Уйнап, шаярып, колеп
Гашыйклар бергэ яшэсен
Тормышнын кадрен белеп

Берсен берсе соеп, назлап
Яшэсен ике йорэк
Берсен берсе соеп, назлап
Яшэсен ике йорэк
```

# Перевод

### Альбина Апанаева - Цветок любви пусть не завянет

```
Как бабочка порхает
В белом платье молодая невеста
Песочные радости соединяются
Встретившись, два сердца

Цветок распустившись
Навеки пусть не завянет
К цветам любви
Злые руки пусть не тянутся

На жизненную тропу выйдя
Найдя друг друга, два сердца
Проходя путь судьбы
Любовь пусть будет опорой

Счастливые жизни пусть придут
В душе святое желание
Друг друга любя, лаская
Пусть живут два сердца

Испытание судьбы пусть пройдут
Играя, шутя, смеясь
Влюбленные вместе пусть живут
Жизни цену зная

Друг друга любя, лаская
Пусть живут два сердца
Друг друга любя, лаская
Пусть живут два сердца
```
