# Оригинал

### Билгесез - Ирекле күбәләкләр

```
Йолдызлар да икебезгә яна,
Җилләр исә икебез өчен дә.
Саф һавалар, сулар синең өчен,
Мин яшимен синең өчен дә.

Синең өчен тудым, сине көттем,
Икебезгә безнең бер гомер.
Миннән китмә, җаным миннән китмә,
Ничек яшәрмен мин берүзем?

Күктә кояш икебезгә яна,
Йөрэк тибә икебез өчен дә.
Җиләк пешә, чәчкә ата гөлләр,
Синен өчен, минем өчен дэ.

Синең өчен тудым, сине көттем,
Икебезгә безнең бер гомер.
Миннән китмә, җаным миннән китмә,
Ничек яшәрмен мин берүзем?
```

# Перевод

### Билгесез - Свободные бабочки

```
Звезды горят для нас двоих,
Ветры дуют для нас обоих.
Чистый воздух, воды для тебя,
Я живу и для тебя.

Для тебя родился, тебя ждал,
Одна жизнь у нас на двоих.
Не уходи от меня, душа моя, не уходи,
Как я проживу один?

Солнце на небе горит для нас двоих,
Сердце бьется для нас обоих.
Земляника созревает, цветут полевые цветы,
Для тебя, для меня.

Для тебя родился, тебя ждал,
Одна жизнь у нас на двоих.
Не уходи от меня, душа моя, не уходи,
Как я проживу один?
```
