# Оригинал

### Билгесез - Казан утлары (икенче вариант)

```
Казан утлары сибә нурларын
Суларына моңсу Иделнең.
Гүзәл кала, нурлы Казан,
Сине күрми йөрәк түзми,
Янулары бетми күңелнең.

Кая барсам да чыкмый исемнән,
Зифа буйлы Идел гүзәле.
Сиңа кайтмый сөйгән калам,
Сине күрми ямьле калам,
Өзелә бит минем үзәгем.

Җырлыйм мин тагын сиңа багышлап,
Матурларын сайлап җырларның.
Сөйгән калам, нурлы калам,
Синдә атты сөю таңым,
Синдә үтте яшьлек елларым.
```

# Перевод

### Бильгесез - Огни Казани (второй вариант)

```
Казанские огни рассыпают свет
По водам грустной Волги.
Прекрасный город, светлый Казань,
Сердце не выдержит, не увидев тебя,
Тоска не кончается в душе.

Куда бы я ни шёл, не выходит из мыслей,
Прекрасная Волга с тонким голосом.
Родной город не возвращается к тебе,
Без тебя тоскующий город,
Ведь рвётся моё сердце.

Пою я снова в твою честь,
Отбирая самые красивые песни.
Любимый город, светлый город,
Тобой прошла моя любовь,
Тобой прошли мои юные годы.
```
