# Оригинал

### Айрат Ильясов - Умырзая

```
Мэхэббэт - дулкын.
Жаным - ут, ялкын.
Уз язым якын,
Булса да салкын.

Гомер аз гына,
Сою, наз гына.
Мин - умырзая,
Бул син яз гына.

Яз кояшыннан
Кунел нур жыя.
Ямьлэнэ донья.
Сагышым дэрья.

Гомер аз гына,
Сою, наз гына.
Мин - умырзая,
Бул син яз гына.

Тормыш юлында
Бул син яз гына.
Безнен бэхеткэ
Бер адым гына.

Гомер аз гына,
Сою, наз гына.
Мин - умырзая,
Бул син яз гына.
```

# Перевод

### Айрат Ильясов - Умырзая

```
Любовь - волна.
Душа моя - огонь, пламя.
Моя собственная весна близка,
Хоть и прохладна.

Жизнь коротка,
Любовь нежна лишь.
Я - надежда,
Будь ты только весной.

От весеннего солнца
Душа собирает свет.
Прекрасен мир.
Тоска моя - море.

Жизнь коротка,
Любовь нежна лишь.
Я - надежда,
Будь ты только весной.

На пути жизни
Будь ты только весной.
К нашему счастью
Лишь один шаг.

Жизнь коротка,
Любовь нежна лишь.
Я - надежда,
Будь ты только весной.
```
