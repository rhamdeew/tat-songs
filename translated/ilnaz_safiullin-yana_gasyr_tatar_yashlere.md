# Оригинал

### Ильназ Сафиуллин - Яна гасыр татар яшьлэре

```
Тормыш агышында мэнге шау-шу,
Гел бер яктан таннар яктыра.
Шул тэулекнен гади бер мизгелен
Туктатырга безгэ ни тора?

Эйе, эйе, эйдэ бергэ жырлыйк,
Йорэк тагын шулай яшьнэрме?
Килэчэкнен бугенгесен сайлый
Яна гасыр татар яшьлэре.

Авырлыклар аша биеклеккэ,
Эйдэ, татар, кушыл агымга!
Миллэтебез килэчэге очен
Нигез ташы монда салына.

Эйе, эйе, эйдэ бергэ жырлыйк,
Йорэк тагын шулай яшьнэрме?
Килэчэкнен бугенгесен сайлый
Яна гасыр татар яшьлэре.
```

# Перевод

### Ильназ Сафиуллин - Татарская молодёжь нового века

```
В потоке жизни вечный шум-гам,
Всё с одной стороны утра светят.
Простой миг тех суток
Остановить что нам мешает?

Да, да, давайте вместе споём,
Сердце снова так молодеет?
Будущего сегодняшний день выбирает
Татарская молодёжь нового века.

Через трудности к высотам,
Давай, татарин, присоединяйся к потоку!
Для будущего нашего народа
Краеугольный камень здесь закладывается.

Да, да, давайте вместе споём,
Сердце снова так молодеет?
Будущего сегодняшний день выбирает
Татарская молодёжь нового века.
```
