# Оригинал

### Илдар Ногманов - Ярат мине (икенче вариант)

```
Йөрәгемнең читенә
Килә сине биклисем.
Яратам дигән сүземне
Нигә син ишетмисең?

Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Көнләшүләремне берүк,
Шикләремне син тарат.
Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Яратуларың йөрәккә
Булмасын иде газап.

Елмаеп син үт әле,
Яннарыма кил әле.
Йөрәгемне үзең белән,
Җаным, алып кит әле.

Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Көнләшүләремне берүк,
Шикләремне син тарат.
Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Яратуларың йөрәккә
Булмасын иде газап.

Мәхәббәттән исереп,
Яшим мин синең өчен.
Яннарымда син булганда,
Тоям мин сөю көчен.

Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Көнләшүләремне берүк,
Шикләремне син тарат.
Ярат мине, ярат мине,
Ярат син, чынлап ярат.
Яратуларың йөрәккә
Булмасын иде газап.
```

# Перевод

### Илдар Ногманов - Люби меня (второй вариант)

```
К краю сердца моему
Так хочу тебя запереть.
Слова «люблю» мои
Почему ты не слышишь, ответь?

Люби меня, люби меня,
Люби по-настоящему, люби.
Ревность мою не оставь,
Сомнения все развей.
Люби меня, люби меня,
Люби по-настоящему, люби.
Пусть твоя любовь для сердца
Не будет мукой, не будет бедой.

Улыбнись и пройди,
Подойди ко мне, подойди.
Сердце мое с собой,
Душа моя, унеси.

Люби меня, люби меня,
Люби по-настоящему, люби.
Ревность мою не оставь,
Сомнения все развей.
Люби меня, люби меня,
Люби по-настоящему, люби.
Пусть твоя любовь для сердца
Не будет мукой, не будет бедой.

Опьянев от любви,
Я живу ради тебя.
Когда ты рядом со мной,
Чую силу любви.

Люби меня, люби меня,
Люби по-настоящему, люби.
Ревность мою не оставь,
Сомнения все развей.
Люби меня, люби меня,
Люби по-настоящему, люби.
Пусть твоя любовь для сердца
Не будет мукой, не будет бедой.
```
