# Оригинал

### Илсур Хөснетдинов / Зөфәр Хәйретдинов - Шомырт бәйләме

```
Сагынып көтәм язлар кайтканын,
Шомырт чәчәкләре атканын.
Күңелемдә шомырт исләре -
Яшьлегемнең кайнар хисләре.

Шомырт чәчәгенең исләре
Хәтерләтә язгы кичләрне.
Су буйларын ничә әйләндем,
Бирергә дип шомырт бәйләмен.

Шомырт чәчәкләре гел истә,
Очрашасы иде ничек тә.
Кайтсын иде шул чак әйләнеп,
Шомырт чәчәгенә бәйләнеп.

Шомырт чәчәгенең исләре
Хәтерләтә язгы кичләрне.
Су буйларын ничә әйләндем,
Бирергә дип шомырт бәйләмен.

Ак чәчәкләр кабат кулымда,
Очрасаң син бүген юлымда -
Бүләк итеп шомырт бәйләмен,
Язмышымны сиңа бәйләрмен.

Шомырт чәчәгенең исләре
Хәтерләтә язгы кичләрне.
Су буйларын ничә әйләндем,
Бирергә дип шомырт бәйләмен.
```

# Перевод

### Ильсур Хуснутдинов / Зуфар Хайретдинов - Букет черемухи

```
Скучая, жду возвращения весны,
Цветения черемухи.
В душе моей ароматы черемухи -
Горячие чувств моей юности.

Ароматы цветков черемухи
Напоминают весенние вечера.
Сколько раз я ходил вдоль рек,
Чтобы подарить букет черемухи.

Цветы черемухи всегда в памяти,
Встретиться как-нибудь надо бы.
Вернулось бы то время,
С букетом черемухи в руках.

Ароматы цветков черемухи
Напоминают весенние вечера.
Сколько раз я ходил вдоль рек,
Чтобы подарить букет черемухи.

Белые цветы снова в моих руках,
Если встретлю тебя сегодня на пути -
В подарок букет черемухи,
Судьбу свою с тобою свяжу.

Ароматы цветков черемухи
Напоминают весенние вечера.
Сколько раз я ходил вдоль рек,
Чтобы подарить букет черемухи.
```
