# Оригинал

### Илсөяр Ганиева / Сара Садыкова - Кайтыгыз, торналар

```
Күл карап озатты
Торналар китүен;
Сагынып көтә ул
Кабат яз җитүен.

Чылбырдай тезелеп,
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?

Йөрмәгез боегып,
Каңгырып чит илдә;
Оялар корырсыз
Яңадан шул күлдә.

Чылбырдай тезелеп,
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?

Туган як күле киң,
Күңелләр күлдәй киң;
Туган җир тормышы
Шул гүзәл күлгә тиң.

Чылбырдай тезелеп.
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?
```

# Перевод

### Илсөяр Ганиева / Сара Садыкова - Возвращайтесь, журавли

```
Озеро взглядом
Провожало их путь;
С тоской ожидает
Возвращенья весны.

Цепочкой сплетаясь,
Вернитесь, журавли;
Где есть озеро такое,
В камышах красоты?

Не скитайтесь в печали,
Не тужите вдали;
Снова гнёзда вы свяжете
На том же озере.

Цепочкой сплетаясь,
Вернитесь, журавли;
Где есть озеро такое,
В камышах красоты?

Широко озеро края,
И души широки;
Жизнь родной стороны
Тому озеру сродни.

Цепочкой сплетаясь,
Вернитесь, журавли;
Где есть озеро такое,
В камышах красоты?
```
