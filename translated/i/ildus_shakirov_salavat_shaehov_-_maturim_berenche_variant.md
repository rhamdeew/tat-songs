# Оригинал

### Илдус Шакиров / Салават Шәехов - Матурым (беренче вариант)

```
Матурымның якты йөзен
Килеп тора күрәсем.
Чәчәкләрдән килә аңа
Тәкыялар үрәсем.
Матурым, килә сине күрәсем,
Иреннәрең гел дә килә үбәсем.

Яшәр идем матурыма
Багышлап тормышымны.
Рәхмәт әйтеп Ходаема,
Тапшырып язмышымны.

Матурым белән диңгездә
Килә минем йөзәсем.
Хыяллардан ятьмә үреп,
Килә энҗе сөзәсем.
Матурым, килә сине күрәсем,
Иреннәрең гел дә килә үбәсем.

Яшәр идем матурыма
Багышлап тормышымны.
Рәхмәт әйтеп Ходаема,
Тапшырып язмышымны.
```

# Перевод

### Илдус Шакиров / Салават Шәехов - Красавица моя (первый вариант)

```
Светлое лицо моей красавицы
Всё хочется мне видеть вновь и вновь.
Из цветов хочу я для неё
Сплести подушки с нежностью, любовь.
Красавица моя, хочу тебя я видеть,
Твои губы всё хочется целовать.

Я бы жил для моей красавицы,
Посвятив ей жизнь мою.
Благодаря Всевышнего,
Судьбу свою вручив ему.

С красавицей моей по морю
Хочется мне плыть вдвоём.
Из мечтаний соткав постель,
Хочу нанизывать я жемчуга.
Красавица моя, хочу тебя я видеть,
Твои губы всё хочется целовать.

Я бы жил для моей красавицы,
Посвятив ей жизнь мою.
Благодаря Всевышнего,
Судьбу свою вручив ему.
```
