# Оригинал

### Илдар Юзеев / Сара Садыйкова - Кайтам инде

```
Ямансулап болыннарны буйлама,
Оныткандыр инде диеп уйлама;
Ничек итеп онытыйм мин ярымны
Кара кашлы, назлы җанымны?!

Ничек итеп онытыйм туган ягымны,
Анда узган алтын балачагымны?!
Сагындырды Агыйделнең камышы,
Челтерәп аккан чишмә тавышы.

Сагындырды сызылып таңнар атуы,
Әнкәемнең иркәләп уятуы;
Кайтам инде туып-үскән ягыма -
Кырларыма, тугайларыма...
```

# Перевод

### Илдар Юзеев / Сара Садыйкова - Возвращаюсь уже

```
Не броди по лугам в тоске,
Не думай: мол, забыла - нет;
Как мне забыть свою половинку,
Чернобровую, нежную душу?!

Как мне забыть родную землю,
Золотое детство тех времен?!
Соскучилась по камышам Агидели,
По шепоту звонких родников.

Соскучилась по светлым зорям,
По маминому ласковому пробужденью;
Возвращаюсь уже в край, где родилась и выросла -
К моим полям, к моим лугам...
```
