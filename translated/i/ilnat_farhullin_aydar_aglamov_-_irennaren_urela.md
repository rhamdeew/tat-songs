# Оригинал

### Илнат Фәрхуллин / Айдар Әгъләмов - Иреннәрең үрелә

```
Татлы, баллы иреннәрең,
Елмаялар иреннәр.
Кил яныма, әй, чибәрем,
Гашыйк булдым үзеңә.

Иреннәрең, иреннәрең үрелә,
Үрелә, күренә.
Иреннәрең, иреннәрең үрелә,
Теләгәнең күренә.

Гашыйк итте матур иренннәрең,
Гел-гел татлы иреннәр.
Алма, чия, кура-кура җиләк,
Бөрлегәндәй иреннәр.

Иреннәрең, иреннәрең үрелә,
Үрелә, күренә.
Иреннәрең, иреннәрең үрелә,
Теләгәнең күренә.

Иреннәрең, иреннәрең,
Сүзнең сөйли тәмлеләрен.
Иреннәрең, иреннәрең,
Син минем, тик минем иреннәрем.

Иреннәрең, иреннәрең үрелә,
Үрелә, күренә.
Иреннәрең, иреннәрең үрелә,
Теләгәнең күренә.
```

# Перевод

### Илнат Фәрхуллин / Айдар Агламов - Твои губы манят

```
Сладкие, мёдом полные губы,
Губы улыбаются.
Подойди ко мне, о, красавица,
Влюбился я в тебя.

Твои губы, твои губы манят,
Манят, видны.
Твои губы, твои губы манят,
Желания твои видны.

Влюбили красивые твои губы,
Всегда сладкие губы.
Яблоко, вишня, корзина ягод,
Как земляника губы.

Твои губы, твои губы манят,
Манят, видны.
Твои губы, твои губы манят,
Желания твои видны.

Твои губы, твои губы,
Вкусные слова твои говорят.
Твои губы, твои губы,
Ты моя, только мои губы.

Твои губы, твои губы манят,
Манят, видны.
Твои губы, твои губы манят,
Желания твои видны.
```
