# Оригинал

### Ибраһим Биектаулы / Раиф Бәһаветдинов - Хәтер исән!

```
Казан өчен башын салган
Шаһидләргә гүя дога,
Агыладыр айлар-еллар
Моңлы бер җыр: «Татар-Болгар».

Ничәнче йөз язлар килә -
Җирдө матур гөлләр үсә...
Һәр көз моңлы җилләр исә -
Хәтер имин! Хәтер исән!!!

Беркемгә дә, беркайчан да
Якты дөнья килми ике...
Сызланды ра күңел читен:
Мең дә биш йөз илле ике.

Ничәнче йөз язлар килә -
Җирдө матур гөлләр үсә...
Һәр көз моңлы җилләр исә -
Хәтер имин! Хәтер исән!!!

Җәрәхәттән арынырга -
Вакыт-табиб, вакыт-пәрдә.
«Татар» дигән иман барда
Әрнү китми йөрәкләрдән...

Ничәнче йөз язлар килә -
Җирдө матур гөлләр үсә...
Һәр көз моңлы җилләр исә -
Хәтер имин! Хәтер исән!!!
```

# Перевод

### Ибрагим Биектаули / Рауф Багаутдинов - Пусть жива память!

```
Положив головы за Казань,
Шахидам - словно мольба.
Льют месяцы, лиют годы
Песню грустную: «Татаро-Булгар».

Какое ни придет весен столетие -
Цветут прекрасные цветы на земле...
Каждой осенью ветры поют -
Пусть хранится память! Пусть жива память!!!

Никому и никогда
Светлый мир не приходит дважды...
Душу терзает тоска тяжкая:
Тысяча пятьсот пятьдесят два.

Какое ни придет весен столетие -
Цветут прекрасные цветы на земле...
Каждой осенью ветры поют -
Пусть хранится память! Пусть жива память!!!

Чтобы зажили раны
Время - врач, время - завеса.
Пока есть вера «Татар» живая -
Не уходит боль из сердец...

Какое ни придет весен столетие -
Цветут прекрасные цветы на земле...
Каждой осенью ветры поют -
Пусть хранится память! Пусть жива память!!!
```
