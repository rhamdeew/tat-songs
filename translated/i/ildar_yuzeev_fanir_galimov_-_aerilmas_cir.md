# Оригинал

### Илдар Юзеев / Фәнир Галимов - Аерылмас җыр

```
Сүзен, көен әйбәт белми торып,
Матур җыр башлавы бик кыен.
Җанга якын җырны бик тиз таптык:
Син сүзләрен белдең, мин - көен.

Бер-беребезне ихлас сөйгәнгәдер,
Күңел җыры бик тиз табылган.
Ул җыр хәзер моңсу сәлам булып,
Ишетелә яшьлек ягыннан.

Бергә булалмадык, ялгыз җырлап,
Гомерләрне күпме яндырдык.
Өзелеп сөйгәннәргә ядкар итеп,
Аерылмас бер җыр калдырдык.
```

# Перевод

### Илдар Юзеев / Фанир Галимов - Неразлучная песня

```
Не зная толком ни слов, ни мелодии,
Трудно начать красивую песню.
Песню, близкую душе, нашли мы очень быстро:
Ты знал слова, я - мелодию.

Наверное, искренне полюбили друг друга,
Песня души нашлась быстро.
Та песня теперь - печальный привет,
Доносящийся из молодости.

Не суждено было быть вместе, мы пели поодиночке,
Сколь много жизней сожгли.
В память о любви, что была прервана,
Оставили одну неразлучную песню.
```
