# Оригинал

### Илфак Шиһапов - Авыл эте

```
Саташып өреп куя ул,
Авылда этләр өрсә.
Оясыннан чыкмый ята,
Хуҗага кунак килсә.
Хуҗа кушмый тавышланмый -
Ялгыш өрсә, эш харап.
Ай тулган кышкы төннәрдә
Ул улый айга карап.

Авыл эте,
Чылбыр озын, савыт тулы.
Булсын хуҗа каты куллы,
Оя җылы.
Авыл эте,
Саташканда гына нигә
Кышкы урман төшкә керә,
Этләр өрә.

Көн аралаш ит бирәләр,
Ипи ташлый килгәннәр.
Белмим: ник мондый тормышны
Эт тормышы дигәннәр.
"Өс!" диеп, хужа өстерсә,
Юл күрсәтеп бер адым,
Теләсә кайсы дошманның
Өзәчәк ул бугазын.

Авыл эте,
Чылбыр озын, савыт тулы.
Булсын хуҗа каты куллы,
Оя җылы.
Авыл эте,
Саташканда гына нигә
Кышкы урман төшкә керә,
Этләр өрә.

Чылбырдан ялгыш ычкынсаң,
Җай белеп кайту кирәк.
Хуҗасыз катлаулы тормыш,
Эт күңеле бер сөяк.
Ышанма әкиятләргә
"Эт бүре булган" дигән.
Кайчандыр бүре бабайлар
Теләп муенчак кигән.

Авыл эте,
Чылбыр озын, савыт тулы.
Булсын хуҗа каты куллы,
Оя җылы.
Авыл эте,
Саташканда гына нигә
Кышкы урман төшкә керә,
Этләр өрә.
```

# Перевод

### Илфак Шиһапов - Деревенский пес

```
Залает он, путая думы,
Когда в деревне лай встает.
Из будки нос не высунет он,
К хозяину гость придет.
Без воли не подаст он голос -
Ошибся - дело пропадет.
В ночи зимы, под полным лунным,
Он воет, в лунный свет глядит.

Деревенский пес,
Цепь длинна, миска полна.
Пусть хозяин - рука тверда,
Будка тиха, тепла.
Деревенский пес,
Лишь в горячем бреду - отчего
Зимний лес приходит во сне,
И псы в ответ поют-воют.

Через день дают ему мясо,
Хлеб швырнут проходя.
Не знаю: зачем эту долю
Зовут собачьей судьбой.
«Фас!» - хозяин потянет повод -
И он шагнет, как научён,
Горло любому врагу он
Перегрызет без слов.

Деревенский пес,
Цепь длинна, миска полна.
Пусть хозяин - рука тверда,
Будка тиха, тепла.
Деревенский пес,
Лишь в горячем бреду - отчего
Зимний лес приходит во сне,
И псы в ответ поют-воют.

Сорвешься с цепи ненароком -
Надо дорогу домой узнать.
Без хозяина жизнь непроста,
Для пса - одна кость мечта.
Не верь ты сказкам про то,
Что «пес был когда-то волком».
Когда-то волчьи прадеды
Сами желали надеть ошейник.

Деревенский пес,
Цепь длинна, миска полна.
Пусть хозяин - рука тверда,
Будка тиха, тепла.
Деревенский пес,
Лишь в горячем бреду - отчего
Зимний лес приходит во сне,
И псы в ответ поют-воют.
```
