# Оригинал

### Илдус Гыйләҗев / Резидә Гыйләҗева - Гел исемдә Буралы

```
Сагындым ла чишмәләрнең
Челтерәгән тавышын.
Төштә күрәм бәбкә саклап
Уйнап үскән тау башын.

Әй, язмышым, юлларымны
Шул якларга бор әле.
Исләремнән гел чыкмый бит
Туган авылым Буралы.

Күз алдымда абый белән
Бергә көтү көткәннәр.
Сагынуның ни икәнен
Белә читкә киткәннәр.

Әй, язмышым, юлларымны
Шул якларга бор әле.
Исләремнән гел чыкмый бит
Туган авылым Буралы.

Мең кат урар кебек булдым
Су буйларын, инешен.
Шунда калган эзләремне
Эзләп кайтып килешем.

Әй, язмышым, юлларымны
Шул якларга бор әле.
Исләремнән гел чыкмый бит
Туган авылым Буралы.
```

# Перевод

### Илдус Гыйләҗев / Резидә Гыйләҗева - Всегда в памяти Буралы

```
Как соскучился я по журчанью
Родных ключей звенящему.
Во сне держу утят, играя,
На той вершине детскому.

О, доля, поверни дороги
Туда, где сердце вешнее.
Не выветрится из души
Родное село Буралы.

Вижу брата - с ним в одиннадцать
Стадо вместе пасли мы.
Тот, кто уехал в даль, узнает,
Как жжёт тоска по малой родине.

О, доля, поверни дороги
Туда, где сердце вешнее.
Не выветрится из души
Родное село Буралы.

Словно тысячи кругов я
Обошёл потоки, плёсы.
Возвращаюсь, следы свои
Отпечатав - снова кроссы.

О, доля, поверни дороги
Туда, где сердце вешнее.
Не выветрится из души
Родное село Буралы.
```
