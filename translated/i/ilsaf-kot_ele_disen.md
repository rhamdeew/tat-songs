# Оригинал

### Ильсаф - Кот эле дисен

```
Анлый алмыйм мин, анлый алмыйм мин —
Эллэ соясен, эллэ соймисен?
Уза гомерлэр, уза гомерлэр,
Кот эле, дисен, кот эле, дисен.

Яшьлекнен танын бергэ каршылап,
Купме сон эле ялгыз йорисем?
Чэчэкле язлар, кунелдэ назлар,
Кот эле, дисен, кот эле, дисен.

Коттереп килгэн сою мэнгелек —
Янымда минем, иркэм, бары син.
Мэнге котэргэ риза булсам да,
Кот эле, дисен, кот эле, дисен.

Чэчлэр агарган, чэчлэр чаларган,
Эллэ шаяртып, уйнап йорисен.
Сою утында яндырасын да,
Кот эле, дисен, кот эле, дисен.
```

# Перевод

### Ильсаф - Ожидай, говоришь

```
Не могу понять я, не могу понять я —
Любишь ли меня, иль не любишь?
Проходят годы, проходят годы,
Ожидай, говоришь, ожидай, говоришь.

Рассвет юности вместе встречая,
Сколько ночей я один хожу?
Цветочная весна, нежность в сердце,
Ожидай, говоришь, ожидай, говоришь.

Принёсшая ожидание любовь вечная —
Только ты рядом, душа моя.
Хоть вечно ожидать согласен,
Ожидай, говоришь, ожидай, говоришь.

Волосы седые, волосы поседели,
А ты шутишь, играя, ходишь.
В огне любви сжигаешь же,
Ожидай, говоришь, ожидай, говоришь.
```
