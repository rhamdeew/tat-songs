# Оригинал

### Илдар Юзеев / Мирсәет Яруллин - Янар чәчәк (беренче вариант)

```
Күкрәгеңә алып кадар чәчәк, янар чәчәк,
Эзләп тугайларга барырсың.
Син эзләмә аны, уз йөрәгең янар чәчәк,
Янар чәчәк булып кабынсын, янар чәчәк,
Яшьлегебез үзе янар чәчәк, янар чәчәк,
Тугайлардан аны эзләмә, янар чәчәк,
Тургайлардан биек күтәрелә, янар чәчәк,
Сайрый, җырлый, сөя, эзләнә, янар чәчәк.
```

# Перевод

### Илдар Юзеев / Мирсает Яруллин - Горящий цветок (первый вариант)

```
К груди прижмёшь цветок - горящий цветок,
И в лугах его ты станешь искать.
Не ищи его: в твоём сердце - горящий цветок,
Пусть вспыхнет он горящим цветком.
Наша юность сама - горящий цветок,
Не ищи его в лугах - горящий цветок,
Выше жаворонков он поднимается - горящий цветок,
Поёт, звучит, любит, ищет - горящий цветок.
```
