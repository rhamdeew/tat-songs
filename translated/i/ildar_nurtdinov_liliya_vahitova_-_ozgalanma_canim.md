# Перевод

### Илдар Нуртдинов / Лилия Вахитова - Не изменяйся, душа моя!

```
Не изменяйся, душа моя, не изменяйся
Ведь в глазах осеннее время года
Лета мои тоже прошли в цветах, ведь
Не навечно, известно, они.

Будешь скучать, золотистые листья,
По тоской наполненной осени,
Злые ветра, коснувшись твоей души,
Сорвавшие жёлтые листья.

Пойдём же, я поведу тебя
К щедрым осенним деньгам.
Вдохнув сладкий нектар
Срывать яблоки.

Будешь скучать, золотистые листья,
По тоской наполненной осени,
Злые ветра, коснувшись твоей души,
Сорвавшие жёлтые листья.

Иначе в мыслях года
На холодное утро с белым туманом
Вдыхая память прошедшего
Остаться только в мечтах.

Будешь скучать, золотистые листья,
По тоской наполненной осени,
Злые ветра, коснувшись твоей души,
Сорвавшие жёлтые листья.
```
