# Оригинал

### Илдар Хөсни / Зиннур Сафиуллин - Юксынам

```
Ничә көн, ничә ай, ничә ел
Зар елый, тилмерә йөрәгем.
Еламый, тилмерми нишләсен -
Югалтты сөюдә терәген.
Яшәргә омтылыш-төләген.

Көн түгел, уйларым чуала,
Тормышның бер гаме юк сыман.
Юк, беркем кирәкми, чибәрем,
Тик сине, тик сине юксынам.

Тынгысыз җаныма иш булган
Син минем гомерлек сагышым.
Ничек соң үзгәртим, юк бер юл,
Тынгысыз гомерем агышын,
Әллә бу тәкъдирем, язмышым?

Көн түгел, уйларым чуала,
Тормышның бер яме юк сыман.
Миңа тик син кирәк, чибәрем,
Тик сине, тик сине юксынам.
```

# Перевод

### Илдар Хөсни / Зиннур Сафиуллин - Скучаю

```
Сколько дней, сколько месяцев, сколько лет
Плача, стонет моё сердце.
Не плачет, не стонет - что делать -
Потеряла любовь свою опору.
Стремление и мука жить.

Не дни, мысли мои путаются,
В жизни словно нет ни одной заботы.
Нет, никто не нужен, моё солнышко,
Только тебя, только тебя скучаю.

К моему беспокойному духу приобщённый,
Ты - моя вечная тоска.
Как же я изменился, нет пути,
Беспокойной жизни моей течение,
Иль это приговор мой, судьба?

Не дни, мысли мои путаются,
В жизни словно нет ни одной радости.
Мне только ты нужен, моё солнышко,
Только тебя, только тебя скучаю.
```
