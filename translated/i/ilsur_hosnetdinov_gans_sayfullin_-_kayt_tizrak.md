# Оригинал

### Илсур Хөснетдинов / Ганс Сәйфуллин - Кайт тизрәк

```
Кошлар таңда сайрарга
Куна нечкә талларга.
Кызыкмагыз, дусларым,

Артык чибәр ярларга.
Су буендагы талым
Сөйлә миңа син барын.
Сөйгән ярым кайтыр микән,
Әйтче син, талым, талым.

Таң алдыннан кошлар сайрый
Тал чыбыгына кунып.
Чибәр ярларга кызыкма

Сөйгән ярларың булып.
Талларымның бөресен
Яз көнендә күрерсең.
Бик сагындым, кайт тизрәк,
Читтә нишләп йөрисең?
```

# Перевод

### Илсур Хөснетдинов / Ганс Сәйфуллин - Вернись скорей

```
Птицы на рассвете запоют,
Сядут на тонкие талы.
Не прельщайтесь, мои друзья,

Слишком красивыми берегами.
Ива у воды, скажи мне всё,
Скажи, поведай обо всём мне ты.
Вернётся ли любимый мой,
Скажи мне, ива, ива ты.

Перед рассветом птицы поют,
На ивовый прут садятся.
Не тянитесь к красивым берегам,

С любимыми берегами не сравнятся.
Почки моих ив
Увидишь в весенние дни.
Я так соскучилась - вернись скорей,
Что ты бродишь вдали одни?
```
