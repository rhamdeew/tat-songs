# Оригинал

### Илдар Юзеев / Ислам Сафин - Онытма мине!

```
Сакла мәңгегә истәлегемне,
Исә юл җиле, аерылу җиле.
Сау бул, саф гөлем,
Зәңгәр яшьлегем,
Онытма мине!
Онытмам сине!

Калыр еракта Гашыйклар иле,
Кагылыр килеп яшьлегем җиле.
Кайтыр моң булып,
Әйтер мин булып:
Онытма мине!
Онытмам сине!

Тормыш юлында, гомерем соңында
Яшьлек дәфтәрен актарсаң иде...
Күңелең үсәр,
Исеңә төшәр:
Онытма мине!
Онытмам сине!
```

# Перевод

### Илдар Юзеев / Ислам Сафин - Не забудь меня!

```
Храни навеки память обо мне,
Дует путь‑ветер, ветер расставаний.
Прощай, мой чистый цветок,
Моя синяя юность,
Не забудь меня!
Не забуду тебя!

Далеко останется страна Влюблённых,
Коснётся, придёт ветер моей юности.
Вернётся напевом,
Скажет, став мной:
Не забудь меня!
Не забуду тебя!

На жизненном пути, в конце моей жизни
Листай бы ты тетрадь юности...
Душа твоя взрастёт,
Вспомнишь меня:
Не забудь меня!
Не забуду тебя!
```
