# Оригинал

### Иркэ - Бул инде дустым гына

```
Синен кузлэр мина карый,
Хэзинэ кургэн сыман.
Асыл сузлэр каян табыйм,
Сине кимсетми торган?

Нур сибуче кузлэренне
"Гаепле сагына" димэ.
Йорэгемне озгэлисен,
"Яратам сине" димэ.

Тылсымлы коч ия сина
Сойкемле сояген бар.
Йорэккэ эмер бирэ алмыйм,
Анын бит сойгэне бар.

Нур сибуче кузлэренне
"Гаепле сагына" димэ.
Йорэгемне озгэлисен,
"Яратам сине" димэ.

Серлэремне ачып салдым,
Сина авыр булмасмы?
Дуслар булып калыйк дисэм,
Кунеллэрен тулмасмы?

Нур сибуче кузлэренне
"Гаепле сагына" димэ.
Йорэгемне озгэлисен,
"Яратам сине" димэ.
```

# Перевод

### Иркэ - Будь теперь только другом

```
Твои глаза на меня смотрят,
Словно сокровище увидев.
Драгоценные слова где найти,
Чтобы тебя не обидеть?

Светозарным твоим глазам
Не говори "виновато скучаю".
Ты душу мою терзаешь,
Не говори "люблю тебя".

Волшебная сила владеет тобой,
У тебя есть возлюбленная.
Сердцу приказать не могу,
У него ведь есть любимая.

Светозарным твоим глазам
Не говори "виновато скучаю".
Ты душу мою терзаешь,
Не говори "люблю тебя".

Тайны свои раскрыла,
Тебе не тяжело ли?
Если скажу "останемся друзьями",
Не переполнятся ли сердца?

Светозарным твоим глазам
Не говори "виновато скучаю".
Ты душу мою терзаешь,
Не говори "люблю тебя".
```
