# Оригинал

### Илсөяр Ганиева / Г.Хәйдәр - Агыйделдә берәү керләр чайкый

```
Агыйделдә берәү керләр чайкый,
Озын толымнарын чылатып.
Сөйгән яры киткән еракларга,
Агыйделкәй сылуын елатып.

Ай нурында гына, әй аппагым,
Нинди серле көйләр көйлисең?
Агыйделнең уйчан камышыдай,
Агымсуга ниләр сөйлисең?

Сөйгән ярың еракларга китсә,
Көтүләре кыен зарыгып.
Ерак киткәннәр дә бер кайталар,
Үткән гомер кайтмый сагынып.
```

# Перевод

### Илсөяр Ганиева / Г.Хайдар - На Агидели одиночка качает весла

```
На Агидели одиночка качает весла,
Вытягивая длинные волны.
Любимый ушёл в далёкие дали,
Плачет о любви к реке.

Только в лунном свете, ай апагым,
Какие таинственные песни поёшь?
Как мечтательный камыш Агидели,
Что ты говоришь течению?

Если любимый ушёл вдалеке,
Ждать его тяжело, со стоном.
Даже ушедшие вдали возвращаются,
Прошедшая жизнь не возвращается с тоской.
```
