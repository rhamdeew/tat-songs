# Оригинал

### Иркэ - Мэхэббэт булганда

```
Мэхэббэт булганда,
Зарыгып котэсен
Кавышыр минутлар житуен.
Кавышкач, тоймыйсын,
Шул мизгел гомернен
Куз ачып йомганчы утуен.

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.

Мэхэббэт булганда,
Кошлардай сайрыйбыз,
Кунелен кыллары монлана.
Сойгэнен булмаса,
Сагынган минутта
Картайган йорэк тэ сызлана.

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.

Мэхэббэт мэнгелек
Кабынган учактай
Хэрвакыт союне остэсэ,
Нигэ сон яшэргэ,
Озелеп соймэскэ,
Нишлэрмен ялкынлап соймэсэм?

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.
```

# Перевод

### Иркэ - Когда есть любовь

```
Когда есть любовь,
Страстно ждёшь
Минуты встречи.
Встретившись, не насытишься,
Тот миг из жизни
Пролетает в мгновение ока.

Когда есть любовь,
Мы творим жизнь.
Ждать, говоря "впереди", долгий путь.
Любимые полюбят,
Надежды сбудутся,
Будь с любимым возлюбленным вместе.

Когда есть любовь,
Поём как птицы,
Душевные струны печалятся.
Если любимого нет рядом,
В минуты тоски
И стареющее сердце болит.

Когда есть любовь,
Мы творим жизнь.
Ждать, говоря "впереди", долгий путь.
Любимые полюбят,
Надежды сбудутся,
Будь с любимым возлюбленным вместе.

Любовь вечная,
Как пылающий огонь,
Всегда любовь превозмогает.
Зачем потом жить,
Разлучившись, не любя,
Что мне делать, если не буду пылко любить?

Когда есть любовь,
Мы творим жизнь.
Ждать, говоря "впереди", долгий путь.
Любимые полюбят,
Надежды сбудутся,
Будь с любимым возлюбленным вместе.
```
