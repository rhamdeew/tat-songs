# Оригинал

### Илдар Юзеев / Мирсәет Яруллин - Фәүзия җыры

```
Нинди матур озын юлга чыктык,
Янар чәчәк иде юл башы.
Чәчәк сулган, синең йөрәгеңә
Ниләр булган, яшьлек юлдашым?

Яшьлегемне искә төшергәндәй,
Дәфтәр битен кичке җил ача.
Еракларда калган тавышыңны
Җил дә алып килми, ичмаса.

Мин югалттым янар чәчәгемне,
Көзләр җиткән, эзлим, табалмыйм;
Ут сал минем сүнгән йөрәгемә,
Синнән башка балкып яналмыйм.
```

# Перевод

### Ильдар Юзеев / Мирсаит Яруллин - Песня Фаузии

```
Какой красивой длинной дорогой мы вышли,
Пылающий цветок был в начале пути.
Цветок завял, что же с тобой случилось,
В твоём сердце, спутница юности?

Будто вспоминаю мою молодость,
Вечерний ветер листает страницы тетради.
Твой голос, оставшийся вдали,
Даже ветер не приносит, увы.

Я потерял мой пылающий цветок,
Осени наступили, ищу - не нахожу;
Огонь зажги в моём погасшем сердце,
Кроме тебя не сияет, не горит.
```
