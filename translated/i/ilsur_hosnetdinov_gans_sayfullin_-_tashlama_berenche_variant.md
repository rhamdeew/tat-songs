# Оригинал

### Илсур Хөснетдинов / Ганс Сәйфуллин - Ташлама (беренче вариант)

```
Ташлама син мине ташлама,
Тормыш әле яңа башлана.
Юллар озын, еллар кыска бит,
Алмаштырма мине башкага.

Ташлама син мине ташлама,
Яңа тормыш инде башлама,
Ялгыз итмә, бәгърем гафу ит,
Ялгыз тормыш бигрәк авыр бит.

Юллар буйлап бергә барырбыз,
Кышкы юлны бергә ярырбыз,
Адашмик та шундый юллар да,
Читен булыр авыр еллар да.

Ташлама син мине ташлама,
Яңа тормыш инде башлама,
Ялгыз итмә, бәгърем гафу ит,
Ялгыз тормыш бигрәк авыр бит.

Вәгъдә бирдек без бит мәңгегә,
Янган утка бергә керергә,
Утлар сүнмәс, сүнсә кабыныр,
Йөрәк һаман сине сагыныр.

Ташлама син мине ташлама,
Яңа тормыш инде башлама,
Ялгыз итмә, бәгърем гафу ит,
Ялгыз тормыш бигрәк авыр бит.
```

# Перевод

### Ильсур Хуснутдинов / Ганс Сайфуллин - Не бросай меня (первый вариант)

```
Не бросай меня, не бросай,
Жизнь лишь впереди начинается.
Дороги длинны, года коротки,
Не меняй на другого меня.

Не бросай меня, не бросай,
Новую жизнь не начинай,
Не оставляй одного, любимый прости,
Одиночество вдвойне тяжело.

Дорогами вместе пройдём,
Зимний путь вместе пробьём,
Даже заблудиться можем на таких путях,
Тяжелыми станут и года.

Не бросай меня, не бросай,
Новую жизнь не начинай,
Не оставляй одного, любимый прости,
Одиночество вдвойне тяжело.

Клятву дали мы на век,
В пламя вместе войти,
Огни не погаснут, если погаснут - вспыхнут вновь,
Сердце вечно будет скучать по тебе.

Не бросай меня, не бросай,
Новую жизнь не начинай,
Не оставляй одного, любимый прости,
Одиночество вдвойне тяжело.
```
