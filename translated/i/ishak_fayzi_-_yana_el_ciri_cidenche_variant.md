# Оригинал

### Исхак Фәйзи - Яңа ел җыры (җиденче вариант)

```
Бизәкле чыршылар
Ерактан балкыйлар,
Бәйрәмгә чакырып торалар.
Тотышыйк кулларны,
Туачак елларны
Бергәләп каршылыйк, туганнар.

Биегез, җырлагыз,
Оялып тормагыз,
Беркем дә күңелсез калмасын.
Яңа ел кичендә
Һәр кеше өчен дә
Өр-яңа матур җыр яңгырасын.

Яңа ел килгәндә
Синдә дә, миндә дә
Иң якты хыяллар уяна.
Яңа ел, Яңа ел,
Бәхетләр алып кил,
Шатлыклар алып кил дөньяга.
```

# Перевод

### Исхак Фәйзи - Песня Нового года (седьмой вариант)

```
Нарядные ёлки
Сияют издалека,
Зовут нас на праздник скорей.
Возьмёмся за руки,
Грядущие годы
Встретим мы вместе, родня.

Танцуйте, пойте,
Не стойте в стесненьи,
Пусть не грустит ни один.
В вечер под Новый
Для каждого снова
Пусть зазвучит песня вновь.

Когда Новый год приходит,
И в тебе, и во мне
Просыпаются светлые сны.
Новый год, Новый год,
Принеси нам счастье,
Принеси радость миру.
```
