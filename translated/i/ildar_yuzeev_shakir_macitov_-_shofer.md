# Оригинал

### Илдар Юзеев / Шакир Мәҗитов - Шофер

```
Шофер, сии машинаң белән
Ил байлыгын ташып йөртәсең
Караңгы төндә дә,
Давыллы көндә дә
Барлык авырлыкларны
Җиңеп үтәсең.
Шофер, иркен зур юллардан
Кешеләргә бәхет илтәсең.

Шофер, тыңла мотор тавышын,
Карама җилгә, давыл, яңгырга.
Күпер юк, дип торма,
Яңа юллар эзлә,
Ятып калма кырда.
Шофер, ятып калма кырда,
Эчең пошса, матур җыр җырла.

Шофер, очасың җил кебек,
Ак каеннар калалар җилпенеп.
Кырлар тыңлый синең
Дәртле җырларыңны.
Кыскарт, шофер, тизрәк кайтыр юлларыңны.
Шофер, иркәң кочагына
Очып кайт син, кайт син җилкенеп.
```

# Перевод

### Илдар Юзеев / Шакир Мәҗитов - Шофер

```
Шофер, своей машиной ты
Богатство края возишь по стране,
И в темной ночи,
И в бурной грозе
Все тяжкие преграды
Побеждаешь в пути.
Шофер, по просторным большим дорогам
Ты людям счастье везешь.

Шофер, слушай гул мотора,
Не смотри на ветер, бурю, дождь.
Нет моста - не стой,
Новые пути ищи,
Не ложись в поле.
Шофер, не ложись в поле,
Если скучно - пой красивую песню.

Шофер, ты летишь, как ветер,
Белые березы машут вслед.
Поля слушают твои
Пылкие песни.
Сократи, шофер, путь - скорей домой.
Шофер, в объятья милой
Прилети, вернись, вернись на крыльях.
```
