# Оригинал

### Илнат Фәрхуллин - Талларым елап кала

```
Ак күлмәгең хәтеремдә,
Сулышымда исләрең.
Кичен ятып, иртән торсам,
Телемдә гел исемең.

Талларым елап кала
Сиңа барган юлларда.
Яратуым чын күңелдән,
Зинһар, ялгыз калдырма.

Зәңгәр төймәң төшеп калган,
Кызыл тута билеңдә.
Миңа мәңге он(ы)тылмаслык
Эз калдырдың күңелдә.

Талларым елап кала
Сиңа барган юлларда.
Яратуым чын күңелдән,
Зинһар, ялгыз калдырма.
```
