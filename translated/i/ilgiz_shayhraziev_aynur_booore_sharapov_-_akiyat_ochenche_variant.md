# Оригинал

### Илгиз Шәйхразиев / Айнур BoooRe Шарапов - Әкият (өченче вариант)

```
Бу төннең йолдызлары сирәк бүген,
Озакка син яуладың йөрәк түрен,
Син генә, башкалары кирәк түгел.
Чакырсам - яннарыма килер идең,
Сине генә яратам, дияр идең,
Җанымдагы борчулар китәр иде,
Яратам...

Син генә миңа кирәк,
Көтелгән бүләк,
Ашкынып тибә йөрәк,
Бу безнең мәхәббәт.
Син генә миңа кирәк,
Бу сөю, димәк,
Бетмәсен иде бу әкият.

Бакчамда үсәр идем, чәчкә булып,
Назлы карашың белән мине җылыт,
Сине яраткан күңелем хискә тулып.
Чын мәхәббәтне күрәм күзләреңнән,
Язмыш юлларын бүләм синең белән,
Якты дөньяда бергә без мәңгегә.

Син генә миңа кирәк,
Көтелгән бүләк,
Ашкынып тибә йөрәк,
Бу безнең мәхәббәт.
Син генә миңа кирәк,
Бу сөю, димәк,
Бетмәсен иде бу әкият.

Бул янымда...
Гел янымда...
Син генә миңа кирәк...
Ашкынып тибә йөрәк...
Син генә миңа кирәк...
Бетмәсен иде бу әкият...

Син генә миңа кирәк,
Көтелгән бүләк,
Ашкынып тибә йөрәк,
Бу безнең мәхәббәт.
Син генә миңа кирәк,
Бу сөю, димәк,
Бетмәсен иде бу әкият.
```
