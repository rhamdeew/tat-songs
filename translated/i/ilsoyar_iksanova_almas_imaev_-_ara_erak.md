# Оригинал

### Илсөяр Иксанова / Алмас Имаев - Ара ерак ...

```
Ара ерак ...
Очрашулар көтмим,
Берни өмет итмим,
Бары сагынам.
Язларымда, сусап назларыңа,
Каеннарга барып сарылам.

Арак ерак ...
Беләм, аңлыйм барын.
Тик оныта алмыйм,
Бары яратам.
Җәйләремдә, үзәгемне өзеп,
Йөрәгемне телеп ата таң.

Ара ерак ...
Синсез аткан таңда
Ераклыкны аңлап,
Сыза йөрәгем.
Бәлки, янда булсаң, шушы чаклы (тикле)
Сөймәс идем сине... сөйгәнем.

Сөйгәнем, сөйгәнем,
Сөйгәнем.
```

# Перевод

### Илсөяр Иксанова / Алмас Имаев - Далеки ты...

```
Далеки ты...
Не жду встреч больше,
Ничего не надеюсь,
Только скучаю.
Весной моей, томясь нежностью твоей,
К берёзам иду - лечусь.

Далеки ты...
Знаю, понимаю всё.
Только не могу забыть,
Только люблю.
Летом моим, рвясь на части,
Зовом сердце зову рассвет.

Далеки ты...
Без тебя вставшим утром,
Поняв даль расстояния,
Ноет сердце моё.
Возможно, будь ты рядом, до этих пор (столько)
Не любила бы тебя... любимый.

Любимый мой, любимый мой,
Любимый.
```
