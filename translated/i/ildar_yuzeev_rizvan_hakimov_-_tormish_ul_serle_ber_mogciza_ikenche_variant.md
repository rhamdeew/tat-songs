# Оригинал

### Илдар Юзеев / Ризван Хәкимов - Тормыш ул серле бер могҗиза (икенче вариант)

```
Кайчакта үзеңне бәхетсез тоясың:
Өметең гөлләрен җил өзә...
Нигә соң, нигә соң күренми Кояшым?
Тормыш ул - серле бер могҗиза.

Кайчакта үзеңне бәхетле тоясың:
Күңелне иркәләп, нур йөзә.
Бер җанга Кояшым ничекләр сыясың?
Тормыш ул - серле бер могҗиза.

Бәхетле минутлар, бәхетсез сәгатьләр
Барсы да мизгелдәй бер уза...
Нурыңны коеп кал, гомереңне тоеп кал:
Тормыш ул - серле бер могҗиза...
```

# Перевод

### Илдар Юзеев / Ризван Хәкимов - Жизнь - таинственное чудо (второй вариант)

```
Порой ты себя ощущаешь несчастным:
Цветы надежды срывает ветры...
Почему же, почему не видится Солнце моё?
Жизнь - таинственное чудо.

Порой ты себя ощущаешь счастливым:
Лаская душу, плывёт сияние.
Как ты, Солнце моё, вмещаешься в одну душу?
Жизнь - таинственное чудо.

Счастливые минуты, несчастливые часы -
Всё, как миг, проходит разом...
Пролей свой свет, почувствуй свой век:
Жизнь - таинственное чудо...
```
