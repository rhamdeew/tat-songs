# Оригинал

### Илдар Юзеев / Луиза Хәйретдинова - Каршы ал, туган калам!

```
Синнән дә ягымлы
Як җирдә юк икән,
Казаным, Казаным!
Әниләр җылысы
Ят җирдә юк икән,
Казаным, Казаным!

Диңгезләрдән, тайгалардай
Ерак-ераклардан,
Һәй!
Изге моңнар тартты сиңа табан,
Каршы ал, туган калам!

Син туган ягымның йөрәге икәнсең,
Казаным, Казаным!
Сөйгәнем шикелле,
Син мине көткәнсең,
Казаным, Казаным!

Диңгезләрдән, тайгалардан
Ерак-ераклардан,
Һәй!
Изге моңнар тартты сиңа табан,
Каршы ал, туган калам!

Алдыңда баш иеп.
Кабатлыйм күңелдән,
Казаным. Казаным!
Аерылмам Иделдән,
Аерылмам илемнән,
Казаным, Казаным!

Диңгезләрдән, тайгалардан
Ерак-ераклардан.
Һәй!
Изге моңнар тартты сиңа табан,
Каршы ал, туган калам!
```

# Перевод

```
Нет земли нежнее тебя,
Казань моя, Казань моя!
Материнского тепла
На чужбине не найти,
Казань моя, Казань моя!

С морей, из таежной глуши,
Издалека-далека,
Эй!
Священные мелодии влекли меня к тебе,
Встреть меня, родной мой город!

Ты сердце родимого края,
Казань моя, Казань моя!
Как возлюбленная,
Ты меня ждала,
Казань моя, Казань моя!

С морей, из таежной глуши,
Издалека-далека,
Эй!
Священные мелодии влекли меня к тебе,
Встреть меня, родной мой город!

Перед тобой склоняюсь,
Прошепчу душой опять,
Казань моя, Казань моя!
Не расстанусь с Иделем,
Не расстанусь с родиной,
Казань моя, Казань моя!

С морей, из таежной глуши,
Издалека-далека,
Эй!
Священные мелодии влекли меня к тебе,
Встреть меня, родной мой город!
```
