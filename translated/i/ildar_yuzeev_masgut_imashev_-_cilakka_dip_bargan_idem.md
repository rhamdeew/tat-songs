# Оригинал

### ildar_yuzeev_masgut_imashev_ - Җиләккә дип барган идем...

```
Җиләккә дип барган идем,
Юри генә адаштым,
Сине күргәч, сүзсез калдым,
Адаштым, дип алдаштым.
Синең йөргән юлларыңда
Җиләкләрем түгелде,
Кырын каравыңны белгәч,
Күңелләрем сүрелде.
Җиләклеккә бармам инде,
Бармам инде, адашмам.
Карурманда адаштым, дип,
Инде бүтән алдашмам...
```

# Перевод

### Илдар Юзеев / Мәсгуть Имашев - Шёл я за ягодой...

```
Шёл я за ягодой,
Нарочно заблудился,
Увидев тебя - онемел,
«Заблудился», - солгал, смутился.
По тропам, где ты проходила,
Мои ягоды рассыпались,
Узнав твой косой, холодный взгляд,
Душа моя опечалилась.
Не пойду я больше в ягодник,
Не пойду и не собьюсь.
Что в глухом лесу заблудился,
Больше я не солгу, не клянусь...
```
