# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Ак томаннар

```
Ак томаннар, күк томаннар
Уралган юлларыма.
Югалма барыр юлымнан,
Бәхетем кулларыңда.

Кушымта:
Адашма ак томаннарда,
Томаннар бер таралыр.
Сиңа булган мәхәббәтем
Гомер буе сакланыр.

Ак томаннар таралсыннар
Без үтәсе юлларда.
Таратыйк, җаным, сагышны
Иртәнге ак томанда.

Кушымта:

Сөю серләрен сакласын,
Югалмасын елларда.
Ак томаннар, күк томаннар
Төшкән барыр юлларга.

Кушымта:
```

# Перевод

### Инзилә Галиуллина / Илгиз Субаев - Белые туманы

```
Белые туманы, синие туманы
Обняли мои пути.
Не дай мне сбиться с дороги,
Мое счастье - в твоих руках.

Припев:
Не заблудись в белых туманах,
Туманы однажды растают.
Любовь моя к тебе
На всю жизнь сохранится.

Пусть белые туманы рассеются
На дорогах, по которым нам идти.
Развеем, родная, печаль
В утреннем белом тумане.

Припев:

Пусть хранит тайны любви,
Не исчезнет в годах.
Белые туманы, синие туманы
Легли на путь впереди.

Припев:
```
