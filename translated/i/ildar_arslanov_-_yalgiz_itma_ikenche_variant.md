# Оригинал

### Илдар Арсланов - Ялгыз итмә (икенче вариант)

```
Юк, соң түгел әле, соң түгел,
Күрештек бүген кабат.
Без бәхетле бергә генә,
Син ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.

Сине күрдем, ничә еллар узгач,
Хыялларны кабат уятып.
Яши идем һаман сөеп,
Бәлки, күрешербез диеп.

Юк, соң түгел әле, соң түгел,
Күрештек бүген кабат.
Без бәхетле бергә генә,
Син ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.

Онытмаган әле син үзең дә,
Күзләреңдә күрәм сагышлар.
Синең сүзне генә көтәм,
Аерылмыйк инде бүтән.

Юк, соң түгел әле, соң түгел,
Күрештек бүген кабат.
Без бәхетле бергә генә,
Син ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.
Ялгыз итмә син, ярат, мине ярат.
```
