# Оригинал

### Илфак Шиһапов - Дала

```
Кылганлы Бөек дала,
Ярсыган атлар чаба.
Баш очында эленгән
Бер ялгыз Тилгән.

Артта калыптай тузан,
Атлар ерак юл узган.
Горур басып калалар
Дәшмәс балбаллар.
Артта калыптай тузан,
Атлар ерак юл узган.
Горур басып калалар
Дәшмәс балбаллар.

Төнге далада утлар,
Ял итә арган атлар.
Иртәгә тагын юлга
Чакыра дала.

Очкыннар чыга таштан,
Көн белән төн тоташкан.
Алда бетмәс юл көтә
Тоташып күккә.
Очкыннар чыга таштан,
Көн белән төн тоташкан.
Алда бетмәс юл көтә
Тоташып күккә.
```

# Перевод

### Илфак Шиһапов - Степь

```
Ширин Великая степь,
Раненые лошади скачут.
В переднем краю повис
Одинокий Тилгән.

Позади остаётся пыль,
Лошади прошли далёкий путь.
Гордо ступают
Непобедимые балбалы.
Позади остаётся пыль,
Лошади прошли далёкий путь.
Гордо ступают
Непобедимые балбалы.

На ночной степи костры,
Отдыхают уставшие лошади.
Завтра снова в путь
Зовёт степь.

Искры вылетают из камня,
День с ночью соединяются.
Впереди ждёт бесконечный путь,
Соединяющийся с небом.
Искры вылетают из камня,
День с ночью соединяются.
Впереди ждёт бесконечный путь,
Соединяющийся с небом.
```
