# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Туган ягым (алтынчы вариант)

```
Туган ягым, җанга якын
Тузанлы урамнарың.
Колач җитмәс киңлекләргә
Жәелгән урманнарың.

Кушымта:
Киң болыннар, шау чәчәкле
Җиләкле аланнарың.
Сандугач моңнары тулган
Сызлып аткан таннарың.

Туган ягым - мәгърур таулар,
Челтери чишмәләре.
Хисләрдән ташкан күңелем
Җырларын көйли әле.

Кушымта:

Балачагым сукмаклары,
Яшьлегем иртәләре.
Сиңа булган хисләр якты -
Йөрәктә йөртәм әле.

Кушымта:
```

# Перевод

### Инзилә Галиуллина / Илгиз Субаев - Родной край (шестой вариант)

```
Родной мой край, душе ты близок,
Твоих пыльных улиц звон.
В необъятные просторы
Растянулись леса вон.

Припев:
Ширь лугов, цветы шумящие,
Ягодные полянки.
Песни соловьев наполнены
Розовеющие зори.

Край родной - горы горделивые,
Журчат родники живые.
Сердце, чувствами переполнено,
Песни снова напевает.

Припев:

Тропки детства моего,
Юности рассветы ранние.
Чувства светлые к тебе
В сердце бережно храню.

Припев:
```
