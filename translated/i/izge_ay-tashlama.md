# Оригинал

### Изге Ай - Ташлама

```
Ташлама син мине, ташлама,
Тормыш эле яна башлана,
Юллар озын, еллар кыска бит,
Алмаштырма мине башкага.

Ташлама син мине, ташлама,
Яна тормыш инде башлама,
Ялгыз итмэ бэгърем, гафу ит,
Ялгыз тормыш бигрэк авыр бит.

Юллар буйлап бергэ барырбыз,
Кышкы юлны бергэ ярырбыз,
Адашмыйк дэ шундый юлларда
Читен булыр авыр елларда.

Ташлама син мине, ташлама,
Яна тормыш инде башлама,
Ялгыз итмэ бэгърем, гафу ит,
Ялгыз тормыш бигрэк авыр бит.

Вэгъдэ бирдек бит без мэнгегэ,
Янган утка бергэ керергэ,
Утлар сунмэс, сунсэ - кабыныр,
Йорэк хаман сине сагыныр.

Ташлама син мине, ташлама,
Яна тормыш инде башлама,
Ялгыз итмэ бэгърем, гафу ит,
Ялгыз тормыш бигрэк авыр бит.
```

# Перевод

### Изге Ай - Не оставляй

```
Не оставляй ты меня, не оставляй,
Жизнь ещё только начинается,
Дороги длинны, годы коротки ведь,
Не меняй меня на другого.

Не оставляй ты меня, не оставляй,
Новую жизнь уже не начинай,
Одинокой не делай душу мою, прости,
Одинокая жизнь особенно тяжела ведь.

По дорогам вместе пойдём,
Зимний путь вместе проложим,
Не собьёмся на таких дорогах
Трудно будет в тяжёлые годы.

Не оставляй ты меня, не оставляй,
Новую жизнь уже не начинай,
Одинокой не делай душу мою, прости,
Одинокая жизнь особенно тяжела ведь.

Обещание дали мы навеки,
В пылающий огонь вместе войти,
Огни не погаснут, погаснут - вспыхнут,
Сердце всё ещё тебя ждёт.

Не оставляй ты меня, не оставляй,
Новую жизнь уже не начинай,
Одинокой не делай душу мою, прости,
Одинокая жизнь особенно тяжела ведь.
```
