# Оригинал

### Ибраһим Биектаулы / Раиф Балаһутдинов - Чиксез рәхмәтем

```
Минем күңелгә
Нур булып кердең,
Бөтен гомергә
Киң сулыш бирдең.

Яшим хисләрнең
Олысын тоеп,
Синең йөрәкнең
Җылысын тоеп.

Җирнең тылсымы
Синдәгенәдер,
Таудай зур бәхет
Миндәгенәдер.

Синең елмаю -
Дөнья рәхәте.
Сиңа, бәгырем,
ЧИКСЕЗ РӘХМӘТЕМ!
```

# Перевод

### Ибраһим Биектаулы / Раиф Балаһутдинов - Бесконечная благодарность

```
В мою душу
Ты вошёл, как свет,
На всю жизнь
Подарил дыханье.

Живу, ощущая
Великую силу чувств,
Ощущая тепло
Твоего сердца.

Земная тайна -
Таится в тебе,
Горой великое счастье -
Живёт во мне.

Твоя улыбка -
Радость мира.
Тебе, родной,
БЕСКОНЕЧНАЯ БЛАГОДАРНОСТЬ!
```
