# Оригинал

### Илфат Мусин / Җәүдәт Гыйльманов - Такыя

```
Болыннарда йөрдем,
Күп чәчәк күрдем.
Матурларын өзеп,
Такыя үрдем.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.

Нинди генә төсләр
Юк такыямда:
Ак та,кызыл,зәңгәр
Барсы бар анда.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.

Бәләкәй мин үзем
Җәй кызы кебек.
Куйдым такыяны
Башыма киеп.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.
```

# Перевод

### Илфат Мусин / Джаудат Гильманов - Венок

```
По лугам бродила,
Много цветов видала.
Лучшие срывала -
Венок сплетала.

В мой венок
Солнце приютилось.
В этом круге
Лето светлое вместилось.

Сколько же красок
В моем венке:
Белый, алый, синий -
Все они в нем.

В мой венок
Солнце приютилось.
В этом круге
Лето светлое вместилось.

Я сама малая,
Словно дочка лета.
Надела венок,
На голову надела.

В мой венок
Солнце приютилось.
В этом круге
Лето светлое вместилось.
```
