# Оригинал

### Илдар Шәрипов - Бәхетлемен (икенче вариант)

```
Тормышымның учаклары сүр(е)леп
Барган мәлдә сине очраттым.
Серле күзле ачык йөзләреңен
Бер күрүдән үк яраттым.

Бәхетлемен синең белән генә,
Бәхетлемен синең яныңда.
Язмышыма чиксез рәхмәтлемен
Очратканым өчен юлымда.

Гомеремнең якты маягы син,
Ышанычлы кешем, терәгем.
Синең өчен генә тибә минем
Хисләр тулы булган йөрәгем.

Бәхетлемен синең белән генә,
Бәхетлемен синең яныңда.
Язмышыма чиксез рәхмәтлемен
Очратканым өчен юлымда.

Тормыш биргән озын баскычларда
Янәшәдән үтик гел бергә.
Ходай биргән безгә мәхәббәтне
Саклыйк әле, җаным, гомергә.

Бәхетлемен синең белән генә,
Бәхетлемен синең яныңда.
Язмышыма чиксез рәхмәтлемен
Очратканым өчен юлымда.
Бәхетлемен...
Бәхетлемен...
```

С загадочными глазами, открытым лицом
Полюбил тебя с первого взгляда.

Счастлив я только лишь с тобой,
Счастлив я, когда ты рядом.
Бесконечно благодарен своей судьбе
За встречу с тобой на пути моём.

Ты - яркий свет в моей жизни,
Мой надёжный человек, моя опора.
Лишь для тебя бьётся
Полное чувств моё сердце.
