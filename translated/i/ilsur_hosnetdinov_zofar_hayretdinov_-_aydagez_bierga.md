# Оригинал

### Илсур Хөснетдинов / Зөфәр Хәйретдинов - Әйдәгез биергә

```
Кичәләрдә гармун уйный,
Гармунчы ниләр уйлый?
Чибәр кызлар аңа карап, аңа карап,
Елмая биеп туймый.

Чабата, чабата, чабата тула оек,
Син, җаный, нигә боек?
Әйдә биергә, әйдә бергә биергә,
Нигә ялгыз йөрергә?

Охшатып нечкә билләргә, нечкә билләргә,
Сыгыла идәннәре.
Күз кысышып куйды инде, куйды инде,
Бер-берсен сөйгәннәре. Эх...

Чабата, чабата, чабата тула оек,
Син, җаный, нигә боек?
Әйдә биергә, әйдә бергә биергә,
Нигә ялгыз йөрергә?

Чакыра бишле биюгә, бишле биюгә,
Безнең гармун моңнары.
Биер идем таңга чаклы, таңга чаклы,
Егетләре, кызлары. Эх...

Чабата, чабата, чабата тула оек,
Син, җаный, нигә боек?
Әйдә биергә, әйдә бергә биергә,
Нигә ялгыз йөрергә?
Әйдә киттек биергә.
```

# Перевод

### Ильсур Хоснетдинов / Зуфар Хайретдинов - Пойдём танцевать

```
На вечерах гармонь играет,
О чём думает гармонист?
Красивые девушки на него глядя, на него глядя,
Улыбаясь танцуют, не насытясь.

Лапти, лапти, лапти весь обутый чулок,
Ты, душа моя, отчего так?
Пойдём танцевать, пойдём вместе танцевать,
Зачем одному ходить?

Подравнивая тонкие талии, тонкие талии,
Прогибаются половицы.
Глаза прищурились уже, уже,
У тех, кто друг друга любит. Эх...

Лапти, лапти, лапти весь обутый чулок,
Ты, душа моя, отчего так?
Пойдём танцевать, пойдём вместе танцевать,
Зачем одному ходить?

Зовут на танец парами, парами,
Наши гармонные мелодии.
Танцевала бы до рассвета, до рассвета,
Парни, девушки. Эх...

Лапти, лапти, лапти весь обутый чулок,
Ты, душа моя, отчего так?
Пойдём танцевать, пойдём вместе танцевать,
Зачем одному ходить?
Пошли танцевать.
```
