# Оригинал

### Илдар Юзеев / Мәсгут Имашев - Җиләккә бармам инде

```
Җиләккә дип барган идем,
Юри генә адаштым.
Сине күргәч, сүзсез калдым,
«Адаштым» дип алдаштым.

Синең йөргән юлларыңда
Җиләкләрем түгелде.
Салкын каравыңны белгәч,
Күңелләрем сүрелде.

Җиләкләргә бармам инде.
Бармам инде, адашмам.
«Җиләклектә адаштым», - дип
Инде бүтән алдашмам.
```

# Перевод

### Илдар Юзеев / Мәсгут Имашев - За ягодой больше не пойду

```
Я шел за ягодой когда-то,
Нарочно сбился я с пути.
Тебя увидев, онемел я,
«Заблудился», - соврал, прости.

По тропам, где ходила ты,
Мои ягоды осыпались.
Узнав твой холодный взгляд,
Душа и чувства притуплялись.

За ягодой я не пойду.
Не пойду, не заблужусь.
«В ягоднике заблудился», -
Так больше лгать я не решусь.
```
