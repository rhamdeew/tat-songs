# Оригинал

### Исхак Фәйзи / Фәрид Низамиев - Ялгыз ай

```
Йолдызлар җемелди,
Йолдызлар нур чәчә,
Йолдызлар нәрсәдер серләшә.
Ай ялгыз бүген дә,
Ай бездән көнләшә,
Чөнки без бүген дә янәшә.

Ялгыз Ай, әйт әле, нигә син
Күкләрдә ялгызың йөзәсең?
Ялгызың йөзмә син,
Галәмне күзлә син,
Үзеңә бер йолдыз эзлә син.

Төн бүген бик матур!
Әллә ай көлгәнгә,
Әллә син "Яратам!" дигәнгә?
Бар дөнья сөенә,
Нурларга күмелә,
Мин сине өзелеп сөйгәнгә.

Ялгыз Ай, әйт әле, нигә син
Күкләрдә ялгызың йөзәсең?
Ялгызың йөзмә син,
Галәмне күзлә син,
Үзеңә бер йолдыз эзлә син.

Йолдызлар җемелди,
Йолдызлар нур чәчә,
Йолдызлар нәрсәдер серләшә.
Әйтерсең лә күктән
Ай безгә эндәшә:
- Гомергә булыгыз янәшә.

Ялгыз Ай, әйт әле, нигә син
Күкләрдә ялгызың йөзәсең?
Ялгызың йөзмә син,
Галәмне күзлә син,
Үзеңә бер йолдыз эзлә син.
```

Потому что мы сегодня тоже вместе.

Одинокая Луна, скажи, почему ты
В небесах плаваешь в одиночестве?

Себе одну звезду ищешь ты.

В лучах утопает,
Потому что я тебя люблю всей душой.
