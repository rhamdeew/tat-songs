# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Сагындыра бергә чак

```
Һаман яратаммы, аңлый алмыйм,
Синең утларыңда мин янам.
Тавышыңны гына ишетсәм дә,
Кош тоткандай, шуңа куанам.

Төшләремдә күрсәм, очрашуга
Юраганым килә һәрчак тап.
Карашыңнан укыйм сөюеңне,
Сагындыра, ахры, бергә чак.

Үткәнемдә калган керсез хисләр
Канатланып оча киңлеккә.
Тик йөрәкләр генә тынгы тапмый,
Сагынулар авыр бигрәк тә.
```

# Перевод

### Инзилә Галиуллина / Илгиз Субаев - Скучает время быть вместе

```
Все так же люблю ли - понять не могу,
В твоих огнях я сгораю дотла.
Лишь голос услышу - и радость в груди,
Как птицу поймав, ликует душа.

Увижу во сне - и встречу гадаю,
Предвестье ее ловлю на лету.
Взглядом читаю: любовь ты скрываешь,
Скучает, видать, наше время вдвоем.

Чистые чувства из прошлого дня
Крылья расправив, летят в высоту.
Только сердцам не найти тишины -
Так тяжела эта горькая тоска.
```
