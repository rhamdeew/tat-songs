# Оригинал

### Илдус Гыйләҗев / Илгиз Закиров - Уразай авылы яшьләре җыры

```
Янәшәдә шаулап ага
Стәрле боргаланып.
Чатыр тау да сиңа карый,
Сокланып, горурланып.

Уразаем - туган авылым,
Гөрләп үсәсең.
Дәртләндереп күңелләрне
Җырга күчәсең.

Армый эшләп арттырабыз
Кырларның уңдырышын.
Хыялларга канат итеп
Яшьлегем омтылышын.
Киләчәгем юлы булып
Күренә урамнарың.
Чатыр таудан югарырак
Күтәрелсен даннарың.

Уразаем - туган авылым,
Гөрләп үсәсең.
Дәртләндереп күңелләрне
Җырга күчәсең.
```

# Перевод

### Ильдус Гильяжев / Ильгиз Закиров - Песня молодёжи деревни Уразай

```
Рядом весело течёт
Звёздная излучина.
На тебя глядит Чатыр-тау,
Взволнованно, гордо.

Уразай моя - родная деревня,
Ты возвышенно растёшь.
Воспламеняя сердца,
В песню переливаешься.

Не покладая рук приумножаем
Урожай полей своих.
Вдохновляя мечтами
Порывы молодости моей.
Будущее твоим путём
Становятся твои улицы.
Пусть высятся над Чатыр-тау
Твои славные заслуги.

Уразай моя - родная деревня,
Ты возвышенно растёшь.
Воспламеняя сердца,
В песню переливаешься.
```
