# Оригинал

### Исхак Фәйзи - Әлли-бәлли бәү

```
Иртән иртүк торабыз,
Кырда урак урабыз.
Көрмәле бәби арбасын
Кыр читенә куябыз.

Куш:
Әлли-бәлли бәү,
Әлли-бәлли бәү,
Уракчылар җырын тыңлап,
Сабыебыз үссен дәү.

Чайкалалар игеннәр,
Урак ура киленнәр.
Көрмәле арба янында
Үсә бара кибәннәр.

Йокла, йокла сабыем,
Эш эшләсен әниең.
Саф һавада йокың туйгач,
Тәмле булыр мәмиең.
```

# Перевод

### Исхак Файзи - Әлли-бәлли бәү

```
Ранним утром мы встаём,
В поле жнём мы серпами.
Телегу с младенцем любимым
На край поля ставим.

Припев:
Әлли-бәлли бәү,
Әлли-бәлли бәү,
Песню жнецов слушая,
Пусть ребёнок растёт, мама.

Колосья шелестят,
Невестки жнут серпами.
Рядом с телегой любимой
Детки подрастают.

Спи, спи, дитя моё,
Пусть мама работает.
Когда в чистом воздухе
Выспишься - вкусной будет каша.
```
