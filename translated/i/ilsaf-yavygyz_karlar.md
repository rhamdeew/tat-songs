# Оригинал

### Ильсаф - Явыгыз карлар

```
Явыгыз, карлар, явыгыз,
Урамнарга тулыгыз.
Сандугачлар киткэн инде,
Тирэклэргэ куныгыз.

Яумагыз, карлар, яумагыз,
Мэхэббэт голлэренэ.
Булсын эле бу тормышта
Бэхетле коннэр генэ.

Явыгыз, карлар, явыгыз,
Басу-кырларга тулып.
Кунеллэрдэ тирбэлегез,
Чиксез шатлыклар булып.

Яумагыз, карлар, яумагыз,
Мэхэббэт голлэренэ.
Булсын эле бу тормышта
Бэхетле коннэр генэ.

Явыгыз, карлар, явыгыз,
Коттем сезне сагынып.
Жемелдэгез чэчлэремэ,
Иннэремэ сарылып.

Яумагыз, карлар, яумагыз,
Мэхэббэт голлэренэ.
Булсын эле бу тормышта
Бэхетле коннэр генэ.
```

# Перевод

### Ильсаф - Падайте, снега

```
Падайте, снега, падайте,
На улицы наполняйте.
Соловьи улетели уже,
На тополя поселитесь.

Не лийте, снега, не лийте
На цветы любви.
Пусть будет в этой жизни
Только счастливые дни.

Падайте, снега, падайте,
На поля и холмы, наполняя.
В сердцах колыхайтесь,
Бесконечными радостями ставая.

Не лийте, снега, не лийте
На цветы любви.
Пусть будет в этой жизни
Только счастливые дни.

Падайте, снега, падайте,
Ожидаю вас, скучая.
Ласкайте мои волосы,
К моим плечам прижимаясь.

Не лийте, снега, не лийте
На цветы любви.
Пусть будет в этой жизни
Только счастливые дни.
```
