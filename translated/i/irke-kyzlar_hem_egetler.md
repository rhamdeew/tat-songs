# Оригинал

### Иркэ - Кызлар хэм егетлэр

```
Кайда хэзер чын ир-атлар
Э юньле кызлар кайда
Хэрбер кеше мэхэббэттэн
Узенэ эзли файда

Ирлэр бигрэк ялкауланды
Хатын-кызлар бик азды
Сэер заманалар килде
Ялкынлы хислэр узды

Э сез, э сез юкка укенэсез
Э сез, э сез гел бездэн колэсез
Э сез, э сез юкка укенэсез
Э сез, э сез гел бездэн колэсез

Хатыннарны берничек тэ
Булмый ойдэ утыртып
Берсузне дэ тынламыйлар
Булмый берни анлатып

Э сез, э сез безне ишитмисез
Э сез, э сез колакка элмисез

Бер генэ кыска итэк тэ яныгыздан уза алмас
Хыянэт итмэгэн ир-ат бу доньяда табылмас

Э сез, э сез берни дэ белмисез
Э сез, э сез безне тикшерэсез
Э сез, э сез берни дэ белмисез
Э сез, э сез безне тикшерэсез
Э сез, э сез берни дэ белмисез
Э сез, э сез безне тикшерэсез
```
