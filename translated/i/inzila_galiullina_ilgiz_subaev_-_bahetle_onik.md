# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Бәхетле онык

```
Олы бәхет бүләк итеп,
Арттырып гаилә ямен.
Сабый бүген аваз салды,
Оныгым туды бүген.

Кушымта:
Дәвамчысы гаиләбезнең,
Бәхетле бул сабыем.
Әти-әни җылысында
Узсын килер гомерең.

Җил-давылдан саклап сине
Күз нурым, дип дәшәбез.
Аллаһ биргән олы бүләк,
Без син диеп яшәрбез.

Кушымта:

Әниеңнең өмете бул,
Әтиеңнең кызчыгы.
Әби-бабайларың өчен
Яраткан оныкчыгы.

Кушымта:

Тәүфыйклы бул, сәламәт бул,
Тигезлектә үс җимеш.
Баламның бар өмете син,
Син - безнең алтын, көмеш.

Кушымта:
```

# Перевод

### Инзилә Галиуллина / Илгиз Субаев - Счастливый внук

```
Большим счастьем одарил,
Украсил наш семейный дом.
Сегодня крикнул наш малыш,
Сегодня внук родился в нем.

Припев:
Продолженье нашей семьи,
Будь счастливым, мой сыночек.
В тепле отца и матери
Пусть пройдет твой путь грядущий.

От ветров и бурь храним,
Говорим: ты свет в очах.
Дар великий от Аллаха,
Мы живем ради тебя.

Припев:

Стань надеждой для мамы,
Доченькой для отца.
Для бабушки и дедушки
Самой милой внучкой будь.

Припев:

Будь с удачей, будь здоров,
Ровным плодом ты расти.
Вся надежда в сыне - ты,
Ты - наш золото и серебро.

Припев:
```
