# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Көткәнемне белгәндер

```
Тәрәзәмә, тәрәзәмә, тәрәзәмә кемдер кага.
Яңгыр микән, яңгыр микән, яңгыр микән, җил микән?
Әллә сөйгәнем килгәнме, кем икән ул, кем икән?
Кем икән ул, кем икән?

Әйе-әйе, әйе-әйе, әйе-әйе, ул килгәндер.
Көткәнемне, көткәнемне, көткәнемне белгәндер.
Таң йокысыннан уятып, хәлен белим, дигәндер,
Көткәнемне белгәндер.

Тәрәзәмә, тәрәзәмә, тәрәзәмә шык-шык кага,
Чакырадыр, чакырадыр, чакырадыр янына.
Чыгыйм әле, күреп калсын, сызылып аткан таң гына,
Күреп калсын таң гына.
```

# Перевод

### Энже Мөэминова / А. Шихапов - Тот, кто знал мою тоску

```
Окно, окно, окно кто-то стучит.
Дождь ли это, дождь ли это, дождь ли это, ветер ли?
Может, мой любимый пришёл, кто он, кто он?
Кто он, кто он?

Да-да, да-да, да-да, он пришёл.
Знающий, знающий, знающий мою тоску.
Разбудив от утреннего сна, пусть узнает, говорит,
Знающий мою тоску.

Окно, окно, окно тихо-тихо стучит,
Позовёт, позовёт, позовёт к себе.
Только выйду, пусть увидит, только вставшее утро,
Пусть увидит утро.
```
