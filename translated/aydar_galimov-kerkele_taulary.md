# Оригинал

### Айдар Галимов - Кэркэле таулары

```
Кэркэле авылы кайда-
Биек тау итэгендэ.
Тауларын да сагынып кайтам-
Читлэргэ китэмен дэ.

Кэркэле таулары биек,
Биек булса да меник.
Туган якнын, сойгэн ярнын,
Дусларнын кадерен белик.

Кэркэле авылын чыккач
Унга киткэн юллар бар.
Сезнен кунелдэ нилэр бар,
Бездэ сагыш- моннар бар.

Кэркэле таулары биек,
Биек булса да меник.
Туган якнын, сойгэн ярнын,
Дусларнын кадерен белик.

Кэркэлэнен чишмэлэре
Ашыагып ага челтерэп
Челтерэп аксын чишмэлэр
Илгэ иминлек телэп

Кэркэле таулары биек,
Биек булса да меник.
Туган якнын, сойгэн ярнын,
Дусларнын кадерен белик.
```

# Перевод

### Айдар Галимов - Горы Киркялы

```
Село Киркялы где-то —
У подножья высокой горы.
По горам его тоскуя возвращаюсь —
В дальние края уезжаю.

Горы Киркялы высоки,
Высоки, но достижимы.
Родной стороны, любимой,
Друзей цену знаем.

Из села Киркялы выйдя,
Есть дороги вправо.
В ваших душах что есть,
У нас есть тоска-мелодии.

Горы Киркялы высоки,
Высоки, но достижимы.
Родной стороны, любимой,
Друзей цену знаем.

Киркялы родники
Журча текут вниз,
Журча пусть текут родники,
Стране безопасность желая.

Горы Киркялы высоки,
Высоки, но достижимы.
Родной стороны, любимой,
Друзей цену знаем.
```
