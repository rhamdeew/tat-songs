# Оригинал

### Асылъяр - Талы-талы

```
Тал эйлэнэ, тал эйлэнэ,
Талга яфрак бэйлэнэ.
Кургэнче сузем куп була,
Кургэч теллэр бэйлэнэ.

Талы, талы, Талы богелеп тора.
Тагын бер курэсе килэ,
Узэк озелеп тора.

Тал тирэклэр утыртканнар
Агыйдел читлэренэ.
Алсу алма суы тамган,
Бэгырем, битлэренэ.

Талы, талы, Талы богелеп тора.
Тагын бер курэсе килэ,
Узэк озелеп тора.

Былбыл булсам, бакчаларда
Бер тирэк сайлар идем.
Шул тирэктэ сине мактап
Иртэ-кич сайрар идем.

Талы, талы, Талы богелеп тора.
Тагын бер курэсе килэ,
Узэк озелеп тора.
```

# Перевод

### Асылъяр - Ива-ива

```
Ива вращается, ива вращается,
К иве листок привязывается.
Пока не увижу, слов много будет,
Увидев, языки связываются.

Ива, ива, ива склонившись стоит,
Еще раз увидеть хочется,
Сердце разрывается.

Ивовые тополя посадили
По берегам Агидели.
Розовый яблочный сок капнул,
Дитя мое, на щеки.

Ива, ива, ива склонившись стоит,
Еще раз увидеть хочется,
Сердце разрывается.

Будь я соловьем, в садах
Один тополь выбрал бы.
На том тополе тебя восхваляя,
Утром-вечером пел бы.

Ива, ива, ива склонившись стоит,
Еще раз увидеть хочется,
Сердце разрывается.
```
