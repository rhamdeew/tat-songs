# Оригинал

### Айфара - Тонге кубэлэк

```
Кон яктысын алыштырып
Тонге утлар кабына
Кичке шэфак ялганда
Тон юрганын ябына

Тонге кубэлэк булу
Нигэ кирэк кыз бала
Болай яшэу топсез упкын
Булганлыгын син анла

Тонге тормыш кара урман
Адашуы тиз анда
Яшьлегенне сату нигэ
Кабат уйла хэм анла

Тонге кубэлэк булу
Нигэ кирэк кыз бала
Болай яшэу топсез упкын
Булганлыгын син анла

Каен сызлып учагыннан
Чыга икэн бу серен
Нигэ шундый конгэ калдын
Оздерден бэхет голен

Тонге кубэлэк булу
Нигэ кирэк кыз бала
Болай яшэу топсез упкын
Булганлыгын син анла
```

# Перевод

### Айфара - Ночная бабочка

```
Сменив дневной свет
На ночные огни,
Когда вечерняя заря догорает,
Укрываешься ночным покрывалом.

Быть ночною бабочкой
Зачем нужно, девочка,
Такая жизнь - безумная обида,
Пойми это.

Ночная жизнь - темный лес,
Легко заблудиться в нем,
Зачем продавать молодость,
Снова подумай и пойми.

Быть ночною бабочкой
Зачем нужно, девочка,
Такая жизнь - безумная обида,
Пойми это.

Береза плача из очага
Выходит эта тайна,
Почему в такой день осталась,
Разорвав цветок счастья.

Быть ночною бабочкой
Зачем нужно, девочка,
Такая жизнь - безумная обида,
Пойми это.
```
