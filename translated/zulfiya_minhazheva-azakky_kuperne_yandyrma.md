# Оригинал

### Зульфия Минхажева - Азаккы куперне яндырма

```
Хуш диеп кулынны бирэсен,
Вэгъдэнне ашыгып озэсен.
Кузлэрен яшерми соясен
Еламыйча ничек тузэсен.

Азаккы куперне яндырма
Син мине ометсез калдырма,
Кузлэремэ тагын бер кара
Курерсен йорэгем гел яра.

Доньяга килеп булмый кабат
Тормышта без икэу пар канат
Канатсыз оча алмый кошлар
Аерылмасын безнен язмышлар

Азаккы куперне яндырма
Син мине ометсез калдырма,
Кузлэремэ тагын бер кара
Курерсен йорэгем гел яра.
```

# Перевод

### Зульфия Минхажева - Не сжигай последний мост

```
Прощаясь, руку протянешь,
Обещанье торопливо дашь.
Глаза свои не скрывая любишь,
Как без слёз теперь переживёшь.

Не сжигай последний мост,
Не оставляй меня без надежды,
В глаза мои взгляни ещё раз,
Увидишь - сердце всё в ранах.

В мир второй раз не прийти,
В жизни мы с тобой - пара крыльев.
Без крыльев не летают птицы,
Пусть наши судьбы не разлучатся.

Не сжигай последний мост,
Не оставляй меня без надежды,
В глаза мои взгляни ещё раз,
Увидишь - сердце всё в ранах.
```
