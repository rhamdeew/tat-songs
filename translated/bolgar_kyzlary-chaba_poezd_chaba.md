# Оригинал

### Болгар кызлары - Чаба поезд чаба

```
Чаба поезд, чаба
Туган якка таба, —
Каршы ала горур имэннэр.
Бозып яшьлек антын,
Туган якка кайта
"Хуш мэнгегэ, сау бул!" дигэннэр.

Чаба поезд, чаба
Туган якка таба, —
Туземнэрем узды чамадан.
Авылыма кайтсам,
Дусларымны тапсам,
Яши башлармын кук янадан.

Чаба поезд, чаба
Туган якка таба, —
Йорэк ярсый, кунел ашкына.
Эйтерсен лэ буген,
Ак каеннар тугел,
Балачагым чыккан каршыма.
```

# Перевод

### Болгар кызлары - Мчится поезд, мчится

```
Мчится поезд, мчится
К родной стороне, —
Навстречу встают гордые дубы.
Нарушив клятву юности,
К родной стороне возвращается
Тот, кому говорили: "Прощай навеки, будь здоров!"

Мчится поезд, мчится
К родной стороне, —
Чемодан с пожитками везу.
Если в село вернусь,
Если друзей найду,
Заживу заново.

Мчится поезд, мчится
К родной стороне, —
Сердце рвется, душа торопится.
Кажется сегодня
Не белые березы,
А детство мое вышло навстречу.
```
