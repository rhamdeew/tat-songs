# Оригинал

### Ленар Сайфиев - Кошчыгым

```
Нигэ диеп мин очырттым
Кошчыгымны кулларымнан?
Онытылыр дип уйласам да,
Чыкмыйсын уйларымнан.

Кошчыгым, кошчыгым,
Сайравына сусадым.
Синсез яшэу - яшэу тугел,
Сонлап кына анладым.

Мине калдырган якларга
Кошчыгым карый микэн?
Башкаларны якын итеп
Ятларга сайрый микэн?

Кошчыгым, кошчыгым,
Сайравына сусадым.
Синсез яшэу - яшэу тугел,
Сонлап кына анладым.

Баш очымда кугэрченнэр,
Пар аккошлар кулемдэ.
кабат кайчан тоярмын мин
Кошчыгымны инемдэ?

Кошчыгым, кошчыгым,
Сайравына сусадым.
Синсез яшэу - яшэу тугел,
Сонлап кына анладым.
```

# Перевод

### Ленар Сайфиев - Птичка моя

```
Зачем, скажи, выпустил я
Птичку мою из рук своих?
Хоть и думал, что забуду,
Не выходишь из мыслей моих.

Птичка моя, птичка моя,
По песне твоей я скучаю.
Жизнь без тебя - не жизнь вовсе,
Теперь только понимаю.

Туда, где меня покинула,
Птичка моя смотрит ли?
Других приблизив к сердцу,
Чужим поет ли?

Птичка моя, птичка моя,
По песне твоей я скучаю.
Жизнь без тебя - не жизнь вовсе,
Теперь только понимаю.

Над головой голуби,
Белые лебеди в озере.
Когда снова почувствую я
Птичку мою на груди?

Птичка моя, птичка моя,
По песне твоей я скучаю.
Жизнь без тебя - не жизнь вовсе,
Теперь только понимаю.
```
