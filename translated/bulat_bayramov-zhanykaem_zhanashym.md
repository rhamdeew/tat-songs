# Оригинал

### Булат Байрамов - Жаныкаем жанашым

```
Ин кадерле кеше буген
Минем очен бер генэ.
Ярты суздэн дэ анлаучы
Мине бит, иркэм, син генэ.

Жаныкаем, жанашым,
Сина тобэп карашым.
Гомер буе янэшэндэ
Килэ минем барасым.

Син кунлемнен акчарлыгы,
Яшэу коче бируче.
Сыйпап-сыйпап чэчлэремннэн
Бердэн берем диюче.

Жаныкаем, жанашым,
Сина тобэп карашым.
Гомер буе янэшэндэ
Килэ минем барасым.

Ямьле донья, матур донья
Бары тик безнен очен.
Уртак бэхетне сакларга
Бирермен ботен кочем.

Жаныкаем, жанашым,
Сина тобэп карашым.
Гомер буе янэшэндэ
Килэ минем барасым.
```

# Перевод

### Булат Байрамов - Душа моя, спутница моя

```
Самый дорогой человек сегодня
Для меня только один.
С полуслова понимающий
Меня ведь, милая, только ты.

Душа моя, спутница моя,
На тебя устремлен мой взгляд.
Всю жизнь рядом с тобой
Хочется мне идти.

Ты белизна моей души,
Дающая силы жить.
Лаская-лаская мои волосы
Единственной называешь.

Душа моя, спутница моя,
На тебя устремлен мой взгляд.
Всю жизнь рядом с тобой
Хочется мне идти.

Прекрасный мир, красивый мир
Весь только для нас.
Чтобы сохранить общее счастье
Отдам все свои силы.

Душа моя, спутница моя,
На тебя устремлен мой взгляд.
Всю жизнь рядом с тобой
Хочется мне идти.
```
