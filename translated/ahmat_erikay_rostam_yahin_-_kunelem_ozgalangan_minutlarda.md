# Оригинал

### Әхмәт Ерикәй / Рөстәм Яхин - Күңелем өзгәләнгән минутларда

```
Күңелем өзгәләнгән минутларда,
Эштән бушаганда кичләрен,
Күзләремә тагын очрамасаң,
Син булмасаң, бәгърем, нишләрмен?

Истән чыкмый синең һәрбер сүзең,
Күз алдымнан китми төсләрең.
Салкын җилләр керсә арабызга,
Син ташласаң мине нишләрмен?

Сүрелерлек түгел мәхәббәтем,
Суынырлык түгел хисләрем,
Гомерем буйларына газаплансам,
Оныталмасам сине нишләрмен?
```

# Перевод

### Ахмат Ерикей / Рустам Яхин - В минуты, когда тоскует душа

```
В минуты, когда тоскует душа,
Когда вечера без дела проходят,
Если больше не встречу твоих глаз,
Если не будет тебя, сердце моё, что мне делать?

Не забыть мне каждое твоё слово,
Не уходит образ твой из глаз.
Если холодный ветер ворвётся меж нами,
Если ты оставишь меня, что мне делать?

Не угаснет моя любовь,
Не остынут мои чувства,
Если буду мучиться всю жизнь,
Если не смогу тебя забыть, что мне делать?
```
