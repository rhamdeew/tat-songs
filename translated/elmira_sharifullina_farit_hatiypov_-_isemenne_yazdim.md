# Оригинал

### Эльмира Шәрифуллина / Фәрит Хатыйпов - Исемеңне яздым...

```
Исемеңне яздым, исемеңне
Бакчабызда, тәрәз каршында.
Исемеңне яздым, гөлләр чәчтем,
Сулар сиптем язлар башында.

Чәчәк булып шытты синең исем,
Нинди тылсым, сихри көч синдә.
Тургай җыр башлады, җир яшәрде,
Ил ямьләнде бер көн эшендә.

Җаным кебек җанлы чәчәкләрне
Күз нурларым түгеп иркәлим.
Исемең булып үскән ал гөлләрнең
Куенына, иркәм, керәм мин.

Җаным кебек җанлы чәчәкләрнең
Хуш исләре, никтер, исертә.
Исемең булып шыткан ал гөлләрне
Саклармын мин мәңге өзмичә!..
```

# Перевод

### Эльмира Шарифуллина / Фарит Хатыпов - Написал твое имя...

```
Написал твое имя, твое имя
В нашем саду, у самого окна.
Написал твое имя, цветы сыпал,
Лиловыми красками весну встречал.

Цветком стало твое имя,
Какая чарующая, волшебная сила в тебе.
Жаворонок запел, земля ожила,
Страна зацвела за один день.

Живые цветы, подобно душе моей,
Мои глаза ласкают, как лучи.
В кусты алых цветов, ставших твоим именем,
Войду я, ласкаясь, в саду.

Сладкий аромат живых цветов,
Подобных душе моей, почему-то томит.
Алые цветы, рассыпавшиеся твоим именем,
Буду хранить я вечно, не отпуская!..
```
