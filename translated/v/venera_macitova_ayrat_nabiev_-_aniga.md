# Оригинал

### Венера Мәҗитова / Айрат Нәбиев - Әнигә

```
Авырлыклар сындырмады,
Була белдең түзем.
Син лаеклы каршылыйсың
Гомереңнең көзен.

Сөеклем дип дәшүчең бар,
Әнием дип әйтүчең бар.
Нәнәй диеп, дүрт күз белән
Килүеңне көтүчең бар.

Бәхет касәң түп-түгәрәк,
Күзләр генә тимәсен.
Кайгы-хәсрәтнең җилләре
Бәгыреңне телмәсен.

Сөеклем дип дәшүчең бар,
Әнием дип әйтүчең бар.
Нәнәй диеп, дүрт күз белән
Килүеңне көтүчең бар.

Исән-сау һәм сәламәт бул,
Рәхәттә генә яшә.
Рәхмәтләрем чиксез диеп,
Йөрәк сиңа эндәшә.

Сөеклем дип дәшүчең бар,
Әнием дип әйтүчең бар.
Нәнәй диеп, дүрт күз белән
Килүеңне көтүчең бар.
```

# Перевод

### Венера Мачитова / Айрат Набиев - Маме

```
Не сломили тебя невзгоды,
Ты умела быть терпеливой.
Ты достойно встречаешь осень
Своей жизни - мудрой, красивой.

Есть, кто шепчет тебе «любимая»,
Есть, кто скажет: «мама моя».
Есть, кто «нэнэй» зовёт, с нетерпеньем
Твоего возвращенья ждёт дня.

Пусть кружка счастья будет полной,
Пусть дурной не коснётся глаз.
Пусть ветра печали и горя
Не порвут твою душу враз.

Есть, кто шепчет тебе «любимая»,
Есть, кто скажет: «мама моя».
Есть, кто «нэнэй» зовёт, с нетерпеньем
Твоего возвращенья ждёт дня.

Будь здорова, жива и сильна,
В мире, радости проживай.
«Мои благодарности - безмерны»,
Сердце тихо тебе говорит, знай.

Есть, кто шепчет тебе «любимая»,
Есть, кто скажет: «мама моя».
Есть, кто «нэнэй» зовёт, с нетерпеньем
Твоего возвращенья ждёт дня.
```
