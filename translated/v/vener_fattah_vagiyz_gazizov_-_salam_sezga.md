# Оригинал

### Венер Фәттах / Вәгыйзь Газизов - Сәлам сезгә

```
Сәлам сезгә, исәнмесез,
Иң кайнар җылы сәлам.
Бергә кушылып җырласын
Җыр сөйгән моңлы галәм.

Без җырларбыз, сез тыңларсыз,
Бүген бездә бәйрәм бит.
Мондый матур тантананы
Ходай үзе биргән бит.
Сәлам сезгә, сез дә безгә
Кушылып җырлашыгыз.
Моң илендә безнең белән
Бергә күңел ачыгыз.

Сәлам сезгә, саулармысыз,
Йөрәкләр бергә типсен.
Җырлар иле, моңлар иле
Халыкны бердәм итсен.

Без җырларбыз, сез тыңларсыз,
```
