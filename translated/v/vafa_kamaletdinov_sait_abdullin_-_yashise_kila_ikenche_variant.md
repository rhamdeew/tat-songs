# Оригинал

### Вафа Камалетдинов / Сәит Абдуллин - Яшисе килә (икенче вариант)

```
Тирә - яктан, яуган кардан
Яз исе килә.
Кайгыларны якка куеп,
Яшисе килә.
Кайгыларны якка куеп,
Сырхауларны ерак куып
Яшисе килә.

Һавалардан, борын ярып,
Яз исе килә.
Күңелләрдә шатлык тавы,
Яшисе килә.
Күңелләрдә шатлык тавы,
Ташып тора хисләр явы,
Яшисе килә.

Кош - кортларның тавышыннан
Яз исе килә.
Бурычларны бирер өчен,
Яшисе килә.
Бурычларны бирер өчен,
Дус -ишләрне күрер өчен
Яшисе килә.
```

# Перевод

### Вафа Камалетдинов / Саит Абдуллин - Хочется жить (второй вариант)

```
Отовсюду, от падающего снега
Весенний запах веет.
Отбросив печали в сторону,
Хочется жить.
Отбросив печали в сторону,
Болезни прогнав вдаль,
Хочется жить.

С ветров, разрывающих ноздри,
Весенний запах веет.
В душах радостный звон,
Хочется жить.
В душах радостный звон,
Чувства через край переполняют,
Хочется жить.

От голосов птиц и зверей
Весенний запах веет.
Чтобы долги отдать,
Хочется жить.
Чтобы долги отдать,
Чтобы друзей увидеть,
Хочется жить.
```
