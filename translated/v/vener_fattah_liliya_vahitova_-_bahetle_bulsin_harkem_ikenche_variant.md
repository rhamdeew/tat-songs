# Оригинал

### Венер Фәттах / Лилия Вахитова - Бәхетле булсын һәркем (икенче вариант)

```
Бәхетле булсын һәркем
Күл артына төшеп кояш байый
Көннең болганышын туктатып
Сәгат кенә кеше гомерләрен
Бер туктамый саный тук та тук

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында

Ә гомерләр арыш башагыдай
Ачы җилләр исә бөгелә
Җилләрдән ачы кайгыдан да
Кайнарланнан яшләр түгелә

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында

Сынауларын биреп язмыш шулай
Туземлеккә ахры өйрәтә
Фәрештәләр колакка пышылдап
Сабыр бул диеп берүк җәлләтмә

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында
```

# Перевод

### Венер Фәттах / Лилия Вахитова - Пусть будет счастлив каждый (второй вариант)

```
Пусть будет счастлив каждый
За озером солнце клонится, тая,
Дневную сумятицу останавливая,
Часы человеческие жизни
Без устали считают: тик-так, тик-так

С полей тянет запахом ржи
Скворцы распевают в лугу
Пусть каждый бы был счастлив
На выбранном жизненном пути

А жизни, как колосья ржи,
Гнутся, когда дуют горькие ветры,
От ветров, от горькой печали
Горячие слезы льются

С полей тянет запахом ржи
Скворцы распевают в лугу
Пусть каждый бы был счастлив
На выбранном жизненном пути

Так судьба, давая испытания,
Похоже, учит терпению,
Ангелы шепчут на ухо:
Будь терпелив, себя не терзай

С полей тянет запахом ржи
Скворцы распевают в лугу
Пусть каждый бы был счастлив
На выбранном жизненном пути
```
