# Оригинал

### Вилдан Вәлиев / Венер Мөхәммәдиев - Ачытау (икенче вариант)

```
Ачытаудан көтеп алдым
Сызылып таңнар атканын.
Күңел җылый, өзелә үзәк,
Талмасын канатлары.
Салкын чишмә чылтырыйдыр
Ачытау асларында.
Бер җылы сүз җитәр иде
Сагынган чакларымда.
Йөгерәм дә менеп китәм
Ачытау башларына.
Исән чакта язасым бар
Шул тауның ташларына,
Истәлеккә язасым бар
Ул тауның ташларына.
Ачытау башларына...
```

# Перевод

### Вилдан Валиев / Венер Мухаммадиев - Ачытау (второй вариант)

```
С Ачытау встречал рассветы,
Как заря струится ввысь.
Плачет сердце, рвётся душа -
Лишь бы крыльям не сломиться.
Холодок ручья звенит он
Под склоном Ачытау.
Одного тёплого слова
Хватило бы в час тоски.
Я бегу и вмиг взлетаю
На вершины Ачытау.
Пока жив, хочу начертить
На камнях той горы строки,
На память хочу оставить
На камнях той горы строки.
На вершины Ачытау...
```
