# Оригинал

### Венера Асаева / Роберт Тимербаев - Исләремә төшәр кебек

```
Яздан киттем,
Наздан киттем,
Синән киттем
Ятим иттем
Йөрәгемне.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.

Сөйгән күзләр,
Назлы сүзләр,
Җылы куллар,
Кайтыр юллар
Сагындырды.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.

Синсез сулып,
Сагыш тулып
Йөрәккәем
Коштай булып
Талпындырды.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.
```

# Перевод

### Венера Асаева / Роберт Тимербаев - Будто в память падет

```
Из весны ушла,
Из ласки ушла,
От тебя ушла -
Сиротой оставила
Сердце своё.

Будто в память падет,
Будто в память падет,
Словно сон.

Любящие глаза,
Нежные слова,
Тёплые руки,
Дороги назад
Соскучили.

Будто в память падет,
Будто в память падет,
Словно сон.

Без тебя увядая,
Печалью полняясь,
Сердечко моё
Птицей
Рвётся в полёт.

Будто в память падет,
Будто в память падет,
Словно сон.
```
