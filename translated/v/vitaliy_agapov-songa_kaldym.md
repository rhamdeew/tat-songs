# Оригинал

### Виталий Агапов - Сонга калдым

```
Эзлэдем, тапмадым синен кебек мин,
Гузэл чэчкэлэрне жир йозеннэн.
Сина кайттым, иркэм, талпынып, очып,
Житешмэдем — сонга калдым мин.

Житешмэдем, иркэм, бэйлэгэнсен син
Тормыш юлларынны башка белэн.
Кайтып, каеннарга соялеп еладым,
Серлэремне ана сойлэдем.

Куз яшьлэрем сортеп, кулга чэчкэ тотып,
Сезне котларга дип туйга килдем.
Кузлэрендэ синен монсу утлар,
Кая китте шаян чакларын?
Кузлэрендэ синен монсу утлар,
Кая китте шаян чакларын?
```

# Перевод

### Виталий Агапов - Остался последним

```
Искал, не нашёл подобной тебе
Прекрасных цветов на лике земли.
К тебе возвратился, милая, стремясь, летя,
Не успел — остался последним я.

Не успел, дорогая, связала ты
Жизненные дороги с другим.
Вернувшись, к берёзам прижавшись, рыдал,
Тайны свои им поведал.

Слёзы утирая, цветы в руках держа,
Пришёл на свадьбу поздравить вас.
В глазах твоих печальные огни,
Куда ушли счастливые мгновенья?
В глазах твоих печальные огни,
Куда ушли счастливые мгновенья?
```
