# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Тормышка җыр

```
Тәү башлап чәңгелдәк эченнән
Дөньяга яңгыраган бер тавыш.
Шушы көннән башлап яшәргә
Кабул итеп алган бу тормыш.

Шул чакта әнкәем әйткәндер:
Шушыннан башлана ул тормыш,
Катлаулы яшәүнең юллары,
Тайпылмый атларга син тырыш.

Бу тормыш өйрәтте көләргә,
Әнкәем, телендә сөйләргә,
Халкым һәм туган илемне,
Ярымны өзелеп сөяргә.

Тормыш ул кадерле һәр кемгә,
Хаклыбыз без аны зурларга.
Тормыш үзе яшәү һәм көрәш, -
Шуңа ул мактала җырларда.
```

# Перевод

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Песня о жизни

```
С колыбели, в самый первый миг,
В мир ворвался голос одинокий.
С этого дня жизнь меня приняла,
Протянув свои дороги строгие.

Тогда, наверное, мать сказала:
Отсюда начинается жизнь,
Пути её трудны и кручены,
Ты без колебаний шагай, держись.

Эта жизнь научила смеяться,
Говорить, как мать научила,
Народ и родную землю
И любимую - горячо любить.

Жизнь дорога для каждого сердца,
Мы вправе её возвеличить.
Жизнь - это жить и бороться, -
Вот почему её славят в песнях.
```
