# Оригинал

### Венера Шәрәфисламова / Зарема Әюпова - Тамчы тама (икенче вариант)

```
Моңсу бер жыр ишетелә
Тып-тып тамган тамчыдан.
Туеп булмый яшәүләрдән,
Гомер бигерәк аз сыман.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.

Шыбыр-шыбыр ява яңгыр,
Түбәдә бии тамчы.
Яшнәп торган яшьлек үтә
Күзләр ачып йомганчы.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.

Тамчы тама, тамчы тама,
Ага яңгыр сулары.
Аккан судай гомер үтә,
Уза яшьлек еллары.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.
```

# Перевод

### Венера Шәрәфисламова / Зарема Әюпова - Капля капает (второй вариант)

```
Слышна печальная песня
Из капли, что тикает в ночь.
Не насытиться жизнью неволей,
Так короток век, словно точь.

Капля капает, капля стекает
Дождевой прозрачной водой.
Как те воды, прошедшая жизнь
Не вернется уже облаком вновь.

Шурша, моросит дождик мягко,
На крыше танцует капля.
Яркая юность проходит -
Лишь моргнёшь, и нет ее следа.

Капля капает, капля стекает
Дождевой прозрачной водой.
Как те воды, прошедшая жизнь
Не вернется уже облаком вновь.

Капля капает, капля капает,
Текут дождевые воды.
Как вода, утекает жизнь,
Уходят юности годы.

Капля капает, капля стекает
Дождевой прозрачной водой.
Как те воды, прошедшая жизнь
Не вернется уже облаком вновь.
```
