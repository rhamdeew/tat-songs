# Оригинал

### Владимир Мордань / Әнвәр Бакиров - Миләш ялкыны

```
Данлы Идел дулкыны шаулый,
Идел өстендә акчарлак.

Миләш ялкыны,
Балан ялкыны
Күнлемнән чыкмас
Дусларым, гомергә.

Славута дулкыннары шаулый,
Язмышлар бергә бәйләнгән.

Миләш ялкыны,
Балан ялкыны
Күнлемнән чыкмас
Дусларым, гомергә.

Идел-Днепр - алар бертуганнар,
Нур сибә дуслык йолдызы.

Үргә атлыйбыз
Ил белән бергә.
Дуслык йолдызы
Нур сибәр гомергә.
```

# Перевод

### Владимир Мордань / Анвар Бакиров - Пламя рябины

```
Славная Волга волнами шумит,
Над Волгой чайка белая парит.

Пламя рябины,
Пламя калины
Из сердца не выйдет
Навеки, друзья мои.

Славута волнами шумит,
Судьбы наши связаны навек.

Пламя рябины,
Пламя калины
Из сердца не выйдет
Навеки, друзья мои.

Волга-Днепр - они словно братья,
Звезда дружбы свет проливает.

Мы шагаем ввысь
Вместе со страной.
Звезда дружбы
Будет светить вечно.
```
