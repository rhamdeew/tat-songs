# Оригинал

### Венера Шәрәфисламова / Илшат Җамалов - Әнкәй чәе

```
Эзләмәгез тәмле чәйләр,
Бер кайда юк мин беләм.
Иң тәмле чәй - әнкәй чәе,
Ясалганы сөт белән.

Эзләмәгез тәмле чәйләр,
Бер кайда юк, мин беләм.
Иң тәмле чәй, ясап биргән,
Әнкәй үз кулы белән.

Мичтән чыккан җылы икмәк,
Җиләк - җимеш каклары.
... Күз алдымда һаман тора,
Әнкәемнең ашлары.

Эзләмәгез тәмле чәйләр,
Бер кайда юк, мин беләм.
Иң тәмле чәй, ясап биргән,
Әнкәй үз кулы белән.
```

# Перевод

### Венера Шарафисламова / Илшат Джамалов - Мамина чай

```
Не ищите чая слаще,
Нигде его нет, я знаю.
Самый вкусный чай - мамин,
С молоком он закипает.

Не ищите чая слаще,
Нигде его нет, я знаю.
Самый вкусный - тот, что сделан
Маминою доброй рукою.

Тёплый хлеб из русской печи,
Сушёные ягоды, плоды -
... Перед глазами всё стоит
Еда моей мамы.

Не ищите чая слаще,
Нигде его нет, я знаю.
Самый вкусный - тот, что сделан
Маминою доброй рукою.
```
