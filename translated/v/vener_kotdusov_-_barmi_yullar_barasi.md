# Оригинал

### Венер Котдусов - Бармы юллар барасы

```
Бармы юллар барасы,
Бармы җырлар язасы?
Бармы юлым, җырлар җырым,
Әйт син, кәккүк баласы.

Кара урман арасы,
Кычк(ы)ра кәккүк баласы,
Кылларының арасында
Йөрәгемнең ярасы.

Урал тауда җәй башы,
Уч төбендә тау ташы.
Син сөясең, мин янамын,
Барыбер ялгыз каласы.

Кара урман арасы,
Кычк(ы)ра кәккүк баласы,
Кылларының арасында
Йөрәгемнең ярасы.

Иксез-чиксез аралар,
Майланмаган арбалар.
Ярым аек көннәр-төннәр
Агымсудай калалар.

Кара урман арасы,
Кычк(ы)ра кәккүк баласы,
Кылларының арасында
Йөрәгемнең ярасы.
Кара урман арасы,
Кычк(ы)ра кәккүк баласы,
Кылларының арасында
Йөрәгемнең ярасы...
```

# Перевод

### Венер Котдусов - Есть ли дороги, куда идти

```
Есть ли дороги - идти?
Есть ли песни - писать их?
Есть ли мой путь, мой напев,
Скажи ты, кукушка-детёныш.

Среди тёмного леса
Кричит кукушкино чадо,
А меж её струнами
Рана моего сердца.

На Урале - начало лета,
В пригоршне - горный камень.
Ты любишь, я сгораю,
А всё равно - в одиночестве.

Среди тёмного леса
Кричит кукушкино чадо,
А меж её струнами
Рана моего сердца.

Бескрайние дали,
Немазаные телеги.
Полуясные дни и ночи
Останутся, как бег воды.

Среди тёмного леса
Кричит кукушкино чадо,
А меж её струнами
Рана моего сердца.
Среди тёмного леса
Кричит кукушкино чадо,
А меж её струнами
Рана моего сердца...
```
