# Оригинал

### Венер Котдусов - Әй, алтын көз

```
Урамнарда йөри алтын көз,
Сайрар кошлар, кайларда сайрыйсыз?
Көзге күктә ай, ата йолдызлар,
Күктәге йолдыз, җирдә гашыйклар.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.

Сайрар кошлар тезелеп очтылар,
Юксынма син, язын кайтырлар.
Язын кошлар, сине сагынып,
Тәрәзәңә килеп кагарлар.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.

Урамнарда инде җәй түгел,
Сагындыр, көз, тик һаман яшь күңел.
Урамнарда йолдызлар кебек
Чибәр кызлар йөри нур сибеп.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.
```

# Перевод

### Венер Котдусов - Эй, золотая осень

```
По улицам ходит золотая осень,
Певчие птицы, где вы поете?
В осеннем небе луна, падают звезды,
В небе звезды, на земле влюбленные.

Эй, золотая осень, мои пылающие звезды,
Падают страстно, пророчат мечты мои:
Эй, влюбленные, скажите свои желания,
Чтобы сердца не остались одинокими.

Певчие птицы улетели вереницей,
Не тосковай - весной вернутся.
Весной птицы, по тебе соскучившись,
Прилетят и постучат в твое окно.

Эй, золотая осень, мои пылающие звезды,
Падают страстно, пророчат мечты мои:
Эй, влюбленные, скажите свои желания,
Чтобы сердца не остались одинокими.

На улицах уже не лето,
Тоску навеяла осень, а сердце все молодо.
По улицам, словно звезды,
Ходят красавицы, свет разливая.

Эй, золотая осень, мои пылающие звезды,
Падают страстно, пророчат мечты мои:
Эй, влюбленные, скажите свои желания,
Чтобы сердца не остались одинокими.
```
