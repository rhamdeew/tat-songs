# Оригинал

### Эльвира Хайруллина - Очрашырга иде

```
Су остендэ ай чайкала,
Тибрэтмэ, жил, аемны.
Ай янында курсэн иде
Озелеп сойгэн ярынны.

Очрашырга иде тизрэк,
Ай яктысы беткэнче,
Таннар атып, сандугачлар
Тирэклэрдэн киткэнче.

Исмэ, жилем, ашыкма эле,
Селкетмэ син аемны.
Котэм эле су буенда —
Курсэтмэсме ярымны.

Очрашырга иде тизрэк,
Ай яктысы беткэнче,
Таннар атып, сандугачлар
Тирэклэрдэн киткэнче.

Капламасын болытларым
Минем матур аемны.
Дулкыннарга эйтэм эле —
Алып китсен кайгымны.

Очрашырга иде тизрэк,
Ай яктысы беткэнче,
Таннар атып, сандугачлар
Тирэклэрдэ нкиткэнче.
```

# Перевод

### Эльвира Хайруллина - Встретиться бы скорее

```
Над водой луна качается,
Не колебли, ветер, месяц мой.
У луны увидеть бы
С тоской любимого дорогого.

Встретиться бы скорее,
Пока лунный свет не погас,
Пока зори не взошли,
Пока соловьи не улетели с тополей.

Дуй, ветер мой, не торопись,
Не качай ты месяц мой.
Жду я здесь у берега —
Не покажется ли любимый мой.

Встретиться бы скорее,
Пока лунный свет не погас,
Пока зори не взошли,
Пока соловьи не улетели с тополей.

Пусть не закроют тучи
Мой прекрасный месяц.
Волнам скажу я —
Унесите мою печаль.

Встретиться бы скорее,
Пока лунный свет не погас,
Пока зори не взошли,
Пока соловьи не улетели с тополей.
```
