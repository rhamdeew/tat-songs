# Оригинал

### Булат Зиганшин - Жиханда

```
Аз гына яз гына тугел,
Хэр фасыл ярату оченгэ.
Хэрвакыт ярата йорэк,
Янадыр мэхэббэт хисендэ.

Аз гына яз гына тугел,
Йорэклэр ярата гомергэ.

Жиханда ярату гына
Гузэллек, матурлык калдыра.
Бу гомер тикмэгэ тугел
Бирелгэн сина да, мина да.

Яхшыга ышанам узем,
Хэр йорэк ярату оченгэ.
Доньяны ярату кирэк,
Иминлек мэхэббэт кочендэ.

Бу жихан саклансын очен,
Шундый коч бирелгэн кешегэ.

Жиханда ярату гына
Гузэллек, матурлык калдыра.
Бу гомер тикмэгэ тугел
Бирелгэн сина да, мина да.

Ярату очен хэр замана!
```

# Перевод

### Булат Зиганшин - В мире

```
Не только весной мало,
Каждый сезон для любви.
Всегда любит сердце,
Пламенеет в чувстве любви.

Не только весной мало,
Сердца любят всю жизнь.

В мире только любовь
Красоту, прелесть оставляет.
Эта жизнь не для попрания
Дана и тебе, и мне.

В хорошее верю сам,
Каждое сердце для любви.
Мир любить нужно,
Покой в объятиях любви.

Чтобы этот мир сохранился,
Такая сила дана человеку.

В мире только любовь
Красоту, прелесть оставляет.
Эта жизнь не для попрания
Дана и тебе, и мне.

Любить в любые времена!
```
