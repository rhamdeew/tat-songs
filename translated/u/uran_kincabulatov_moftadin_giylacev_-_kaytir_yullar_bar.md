# Оригинал

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Кайтыр юллар бар

```
Әйди матур тормыш,
Ялгышмаса язмыш,
Безгә дә бер кайтыр юл ятыр...
Безнең яшьлек җыры -
Язгы кояш нуры
Бөреләрне назлап уятыр.

Үтәр айлар, еллар,
Яшел ак каеннар
Саргаерлар безне сагынгач.
Якты сөю хисен
Сайрар иртә, кичен
Безне гашыйк иткән сандугач.

Туып үскән җирдә
Дуслар белән бергә
Гөрләшербез әле бер заман.
Яшьлек тугаена,
Сабан туйларына
Кайта алмасак булыр бик яман.
```

# Перевод

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Есть пути возвращенья

```
Манит светлая жизнь,
Если не ошибется судьба,
И нам ляжет одна дорога домой...
Песнь нашей юности -
Солнечный луч весны
Нежно разбудит почки.

Пройдут месяцы, годы,
Зелёные белые берёзы
Пожелтеют, тоскуя по нам.
Светлое чувство любви
Будет петь утром и вечером
Соловей, что нас сделал влюблёнными.

На земле, где мы родились,
С друзьями вместе
Ещё когда‑нибудь будем шуметь.
К родным лугам юности,
К сабантуям
Не суметь вернуться - будет очень худо.
```
