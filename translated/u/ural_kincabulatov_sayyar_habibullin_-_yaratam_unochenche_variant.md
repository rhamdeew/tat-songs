# Оригинал

### Ural Kinjabulatov / Sayyar Khabibullin - Яратам (унөченче вариант)

```
Кыш уртасын сайлап алган
Ялган язым булып калма.
Бул юлымда, язмышымның
Бәхет язы булсаң гына.

Йөрәгемдә син - алсу таң,
Гөлләр булып мин нур йотам,
Бәхет язы кебек янам.
Сине күрсәм, хискә батам,
Сине күрсәм, чәчкә атам.
Мин яратам, яратам!

Ак каеннар кебек, язын
Тамыр җәям, яфрак ярам.
Син булмасаң, мин боегам,
Мин моңаям, мин саргаям.
```

# Перевод

### Урал Кинҗәбулатов / Сәйяр Хәбибуллин - Люблю (тринадцатый вариант)

```
Не стань моей обманной весной,
Что середину зимы избрала.
Будь на моем пути - лишь тогда,
Если ты - счастья весна для судьбы.

В сердце моем ты - розовый рассвет,
Я, как цветы, глотаю твой свет,
Пылаю, как счастливая весна.
Увижу тебя - в чувствах тону,
Увижу тебя - в цвету расцвету.
Я люблю, я люблю!

Как белые березы весной,
Пускаю корни, расправляю листья.
Без тебя я гасну,
Я грущу, я желтею.
```
