# Оригинал

### Уран Кинҗәбулатов / Рәшит Җиһанов - Мин гашыйк булдым

```
Мин гашыйк булдым,
Әнкәй, тиргәмә.
Гашыйк күңелдә
Дөнья тирбәлә.

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге -
Мәхәббәт икән!

Безнең тормышны
Мәхәббәт нурлый.
Гашыйк күңелдә
Сандугач сайрый!

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге -
Мәхәббәт икән!

Яшьлекнең яме -
Мәхәббәт икән.
Безнең бәхеткә
Мәхәббәт - җилкән.

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге -
Мәхәббәт икән!

Атылган йолдыз
Бер балкып сүнә,
Сүнмәскә балкый
Мәхәббәт кенә!

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге -
Мәхәббәт икән!
```

# Перевод

### Уран Кинҗәбулатов / Рәшит Җиһанов - Я влюбился

```
Я влюбился,
Мама, не ругай.
Влюбленным сердцем
Весь мир раскачай.

Сердце стучало,
Ждало любви вновь.
Колыбель счастья -
Только любовь!

Нашу жизнь щедро
Любовь озарит.
В сердце влюбленном
Соловей звучит!

Сердце стучало,
Ждало любви вновь.
Колыбель счастья -
Только любовь!

Вся прелесть юности -
В той самой любви.
Для нашего счастья
Любовь - наш парус в пути.

Сердце стучало,
Ждало любви вновь.
Колыбель счастья -
Только любовь!

Падая, звезда
Раз вспыхнет - и тьма,
Неугасимо
Лишь любовь сама!

Сердце стучало,
Ждало любви вновь.
Колыбель счастья -
Только любовь!
```
