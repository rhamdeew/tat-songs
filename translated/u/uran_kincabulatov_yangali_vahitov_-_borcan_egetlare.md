# Оригинал

### Уран Кинҗәбулатов / Янгали Вахитов - Бөрҗән егетләре

```
Гашыйкларның иле бит ул
Аның Бөрҗәне, диләр шул, эх кенә!
Кызлар безгә гыйшык тотса,
Минем бер җаным, диләр шул, эх кенә!

Зәңгәр күкне байкап оча
Урал тау бөркетләре.
Бер карауда гашыйк итә
Бөрҗәннең егетләре!

Безнең Бөрҗән - куе урман,
Агач саен сандугач шул, эх кенә!
Бер җырламый күңел булмый, / Кош түгел, үзең сайрарсың, /
Күңел тулы җыр булгач шул, эх кенә!

Зәңгәр күкне байкап оча
Урал тау бөркетләре (лә).
Кара кашлы, кара күзле
Бөрҗәннең егетләре!

Бөрҗән тавы гел асыл таш,
Адым саен хәзинә шул, эх кенә!
Салават күпере аша
Килегез, дуслар, безгә шул, эх кенә!

Зәңгәр күкне байкап оча
Урал тау бөркетләре (лә).
Ак теләкле, ут йөрәкле / Ут йөрәкле, үтә дәртле /
Бөрҗәннең егетләре!
```

# Перевод

### Уран Кинзябulatов / Янгали Вахитов - Парни из Бурзяна

```
Край влюбленных, говорят,
Вот он, Бурзян, говорят, ах!
Если девушки нас любят,
Душа моя, говорят, ах!

Голубое небо пронзая в полёте,
Орлы с Уральских гор.
С одного взгляда влюбляют
Парни из Бурзяна!

Наш Бурзян - густой лес,
На каждом дереве соловей, ах!
Не проживёшь и дня без песни,
Когда душа полна песни, ах!

Голубое небо пронзая в полёте,
Орлы с Уральских гор.
Чернобровые, черноглазые
Парни из Бурзяна!

Гора Бурзян - сама драгоценность,
Каждый шаг - сокровище, ах!
Через Салаватов мост
Приезжайте к нам, друзья, ах!

Голубое небо пронзая в полёте,
Орлы с Уральских гор.
С светлыми мечтами, с огненными сердцами,
С сердцами пламенными, full of страсти,
Парни из Бурзяна!
```
