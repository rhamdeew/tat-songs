# Оригинал

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Мәхәббәт йәйгоры

```
Күзләрем гел синдә, гүзәлем,
Мәхәббәт утында йөрәгем.
Мин көтәм кояшлы яңгырны,
Балкыр дип мәхәббәт йәйгоры.

Зәңгәр су гөмбәзле күңелем,
Мин нигә бәхетле түгелмен.
Яу әйдә кояшлы яңгырым,
Балкытып мәхәббәт йәйгорын.

Ишет син мәхәббәт тавышын,
Карашлар мәңгегә кавышсын.
Яшьлекнең кояшлы яңгыры
Балкытсын мәхәббәт йәйгорын.
```

# Перевод

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Радуга любви

```
Мои глаза всегда на тебе, красавица,
Сердце мое в огне любви.
Я жду солнечный дождь -
Чтоб засияла радуга любви.

Душа моя под куполом лазури,
Почему же я не счастлива?
Пролейся, солнечный дождь,
Озари сияньем радугу любви.

Услышь ты голос любви,
Пусть взгляды навеки сойдутся.
Солнечный дождь юности
Пусть зажжет радугу любви.
```
