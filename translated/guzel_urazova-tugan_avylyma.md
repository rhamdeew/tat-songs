# Оригинал

### Гөзел Уразова - Туган авылыма

```
Чынлы суы жэелеп ага,
Язгы ташкыннар килсэ.
Ташкыннар кебек шаулашып,
Авылымда яшьлэр усэ.

Туган авылым минем каршымда,
Кунелем хаман омтыла шунда.

Ераклардан карап килэм
Чынлы басуларына.
Шунда гына тынгы табам
Кунелем ярсуларына.

Туган авылым минем каршымда,
Кунелем хаман омтыла шунда.

Шушы кырлар буйлап минем
Яшьлек юлларым уткэн.
Шул юллардан яшьлек дустым
Елмаеп чыкса икэн!

Туган авылым минем каршымда,
Кунелем хаман омтыла шунда.
```

# Перевод

### Гөзел Уразова - Моя родная деревня

```
Вода в Чынлы весело течёт,
Когда весенние разливы приходят.
Словно половодье шумя,
В деревне моей молодёжь растёт.

Родная деревня моя передо мной,
Душа моя всё стремится туда.

Издалека смотрю
На берега Чынлы.
Только там покой найдёт
Душа моя от печалей.

Родная деревня моя передо мной,
Душа моя всё стремится туда.

По этим полям
Прошли дороги моей юности.
С тех дорог друг юности
Улыбаясь вышел бы!

Родная деревня моя передо мной,
Душа моя всё стремится туда.
```
