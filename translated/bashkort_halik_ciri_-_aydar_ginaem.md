# Оригинал

### Башкорт халык җыры - Айдар гынаем

```
Айдар гынаем да
Вайдар гынаем.
Сары сандугачкаем да
Кара карлыгачкаем.

Бас, бас, әйдә бас,
Их, их, кара каш!
Йөрәк җилкенеп торсын да
Гомер дәртлерәк узсын.

Ах, ах, кызлары,
Ах, ах, үзләре!
Сары сандугач сымак та
Кара карлыгач сымак.
```

# Перевод

### Башкирская народная песня - Мой милый Айдар

```
Мой милый Айдар,
Мой ненаглядный.
Ты соловушка мой золотой,
Ты ласточка моя чернокрылая.

Пляши, пляши же, в пляс!
Ах, эти чёрные брови!
Пусть сердце трепетно забьётся,
Пусть жизнь проходит веселее.

Ах, эти девушки,
Ах, какие красавицы!
Словно золотые соловьи,
Словно ласточки чернокрылые.
```
