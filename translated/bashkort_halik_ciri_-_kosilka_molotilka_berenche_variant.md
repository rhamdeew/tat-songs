# Оригинал

### Башкорт халык җыры - Косилка, молотилка (беренче вариант)

```
Уйнап алабыз әле /лә/,
Җырлап алабыз әле,
Уйнасак та, җырласак та,
Биеп алабыз әле.

Косилка, молотилка,
Сортировка, веялка.

Уйныйбыз бергә-бергә /дә/,
Җырлыйбыз бергә-бергә.
Уйнап-көлөп, җырлап-биеп,
Эшлибез бергә-бергә.

Косилка, молотилка,
Сортировка, веялка.

Улай итәргә кирәк /лә/,
Былай итәргә кирәк.
Җитәр инде бу уенны
Тәмам итәргә кирәк.

Косилка, молотилка,
Сортировка, веялка.
```

# Перевод

### Башкирская народная песня - Косилка, молотилка (первый вариант)

```
Поиграем ещё /ля/,
По поём ещё,
Если будем играть и петь,
Ещё потанцуем.

Косилка, молотилка,
Сортировка, веялка.

Будем играть вместе /де/,
Будем петь вместе.
Играя-смеясь, поёт-танцуя,
Будем работать вместе.

Косилка, молотилка,
Сортировка, веялка.

Надо так делать /ля/,
Надо так делать.
Хватит этой игры,
Нужно заканчивать.

Косилка, молотилка,
Сортировка, веялка.
```
