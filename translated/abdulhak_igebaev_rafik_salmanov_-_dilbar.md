# Оригинал

### Абдулхак Игебаев / Рафик Сәлмәнов - Дилбәр

```
Син яшәгән якка табан
Иссәләр назлы җилләр,
Җилләр белән сәламемне
Сиңа җибәрәм, Дилбәр.

Нәкъ үзең кебек елмаеп,
Таң балкып көлсә әгәр,
Таңгы нурлар белән сиңа
Сәлам җибәрәм, Дилбәр.

Кабул ит, кайчан килсә дә.
Минем сәламне, Дилбәр,
Сәламем сиңа йөрәкнен
Кайнар сөюен сөйләр.
```

# Перевод

### Абдулхак Игебаев / Рафик Салманов - Дильбар

```
Туда, где ты живёшь сейчас,
Повеют нежные ветра,
И с ветром шлю тебе привет,
Моя Дильбар, моя мечта.

Когда рассвет, как ты, смеясь,
Сияньем озарит простор,
С лучами утренней зари
Тебе привет пошлю в укор.

Прими, когда б ни прилетел,
Мой нежный шёпот, Дильбар,
Мой шёпот — сердца моего
Горячей страсти жаркий дар.
```
