# Оригинал

### Ильназ Сафиуллин - Изге Казаным

```
Нинди сэер тон… Уйлар, фикерлэр йокларга ирек бирми. Карыйм: тулган ай яктысында… Шэhэр!.. Матур, шул ук вакытта монлы, шул ук вакытта боек…

Якты телэк белэн,
Якты омет белэн
Бу вакытны каршы алдым…
Назлы жиле белэн,
Сыгылмас канаты белэн
Килеп керде Казан рухы.

Тонге урамда, кузлэрне чагылдырып,
Йокыдан уяна Казан утлары.
Йорэккэем, эйт, ничек тузэргэ?..
Йорэккэем, ничек сокланмаска?!.
Экияттэ генэ мондый донья!..
Юк шул, иркэм, чынбарлыкта да бар
Экияттэй матур доньялар…
Шуларнын берсе—башкалабыз Казан!..

Табигате белэн,
Горур халкы белэн,
Мэдэнияте белэн бай илем.
Заман хатасы белэн
Канлы сугышлар уткэн,
Куз яшьлэре тугелгэн  бай жирем!..
Жаныма наз остэп,
Оча Казан рухы.
Оча биеккэ, артта кала нуры.
Ана кушылып биилэр йолдызлар,
Йолдыз тугел—Казан утлары…
Ана кушылып жырлыйлар йолдызлар,
Йолдыз тугел—Казан моннары,
Йолдыз тугел—Казан утлары…

Кайтырсын, вакыт тауларын урап,
Жинэрсен кайгы, хэсрэт карашын,
Хэм булэк итеп hэрбер йорт туренэ
Китерерсен гомерле бэхетлэр…

Килэчэген, тарихын кебек, боек,
Бугенгесе калыр оныкларга…
Ата-бабалар карый зэнгэр куктэн…
Мэнге урнэк булыр мирасым,
Мэнге балкы нурлы башкалам,
Мэнге балкы изге Казаным!.
```

# Перевод

### Ильназ Сафиуллин - Священная моя Казань

```
Какая чудесная ночь… Мысли, думы не дают покоя. Смотрю: в свете полной луны… Город!.. Прекрасный, в то же время печальный, в то же время величественный…

Со светлым желанием,
Со светлой надеждой
Это время встретил…
С нежным ветром,
С несгибаемым крылом
Вошел дух Казани.

На ночной улице, отражаясь в глазах,
Пробуждаются от сна огни Казани.
Сердечко мое, скажи, как вынести?..
Сердечко мое, как не восхищаться?!
Только в сказке такой мир!..
Нет же, дорогая, в действительности тоже есть
Сказочно прекрасные миры…
Один из них — наша столица Казань!..

Природой своей,
Гордым народом своим,
Культурой богатый мой край.
Ошибкой времен
Кровавые войны прошли,
Слезы пролились на богатой земле моей!..
К душе моей ласку источая,
Летит дух Казани.
Летит ввысь, сзади остается свет.
К нему присоединяясь, танцуют звезды,
Не звезды — огни Казани…
К нему присоединяясь, поют звезды,
Не звезды — мелодии Казани,
Не звезды — огни Казани…

Вернешься, времени горы обогнув,
Победишь печали, горя взгляд,
И в подарок каждому дому
Принесешь вечное счастье…

Будущее твое, как история, величественно,
Сегодняшнее останется потомкам…
Предки смотрят с синего неба…
Навеки примером станет наследие мое,
Навеки сияет светлая столица моя,
Навеки сияет священная моя Казань!
```
