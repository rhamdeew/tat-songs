# Оригинал

### Башкорт халык җыры - Ямьле Агыйдел буйлары (беренче вариант)

```
Агыйделкәйләрнең аръягында
Кына гөлләр үскән бер төптә.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.

Сагынасыңмы диеп ник сорыйсың?
Исләремнән чыкмый бөртек тә.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.

Агыйделкәйләрнең аръягында
Керәшкәдән* кибән кем куйган?

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.

Сагынасыңмы диеп ник сорыйсың?
Туган-үскән илдән кем туйган.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.

--
* Керәшкә - үлән төре.
```

# Перевод

### Башкирская народная песня - Берега Агидели (первый вариант)

```
Позади Агидельских гор
Цветы растут на одном корне.

Берега Ямьле Агидели
Развеют тоскливые мысли.

Зачем ты спрашиваешь, скучаешь ли?
Из памяти не выходит образ.

Берега Ямьле Агидели
Развеют тоскливые мысли.

Позади Агидельских гор
Кто поставил похожий на керяшку?

Берега Ямьле Агидели
Развеют тоскливые мысли.

Зачем ты спрашиваешь, скучаешь ли?
Кто утолял в родной стране.

Берега Ямьле Агидели
Развеют тоскливые мысли.

--
* Керяшка - вид травы.
```
