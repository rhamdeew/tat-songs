# Оригинал

### Билгесез - Дим буйлары сары чәчәк

```
Дим буйлары сары чәчәк,
Өзәсен дә аласын.
Сары чәчәкләрне өзеп,
Үзең ялгыз каласын.

Дим буйларына төшмәсәм,
Өзмәгән булыр идем.
Сары чәчәкләрне өзеп,
Йөрмәгән булыр идем.

Эх дусларым, сезгә әйтәм,
Сары чәчәк өзмәгез.
Сары чәчәкләрне өзеп,
Ялгыз кала күрмәгез.
```

# Перевод

### Билгесез - Желтые цветы вдоль дороги

```
Желтые цветы вдоль дороги,
Пусть их даже сорвут.
Выбив желтые цветы,
Ты один останешься.

Если б я не зашла на обочину,
Не сорвала бы их.
Выбив желтые цветы,
Не пошла бы дальше.

Ох, друзья мои, говорю вам,
Не рвите желтый цвет.
Выбив желтые цветы,
Не оставайтесь в одиночестве.
```
