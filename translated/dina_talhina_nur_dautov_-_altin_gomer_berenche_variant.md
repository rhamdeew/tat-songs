# Оригинал

### Динә Талхина / Нур Даутов - Алтын гомер (беренче вариант)

```
Алтын да гына алтын балдак,
Алтын гына еллар, ай уза.
Алтын тузганнары сизелми,
Безнең алтын гомерләр туза.

Тузыр да гына гомерләрнең
Кадерен белми тотыла.
Шатланып тә яшәр урынга,
Үкенечтән утлар йотыла.

Алтын да гына алтын балдак,
Вакытларны булмый шул алдап.
Алтын балдак биргән язларны
Көзләргә булмый шул ялгап.
```

# Перевод

### Дина Талхина / Нур Даутов - Золотая жизнь (первый вариант)

```
Золотое, словно золото, дитя,
Золотые годы, месяцы летят.
Золотая пыль незаметно тает,
Наша золотая жизнь проходит вся.

Проходит жизнь, а мы не ценим,
Не понимаем, что дано.
Вместо радости и счастья
Огнём сжигает нас одно - печаль.

Золотое, словно золото, дитя,
Время невозможно обмануть.
Вёсны, что дарило золотое детство,
Не приклеить к осени уж никогда.
```
