# Оригинал

### Фан Валиахметов - Сарманай

```
Кил эле син Сарманайга,
Котэрмен юл чатында.
Сина эйтер сузлэрем бар
Тик икебез хакында.

Сарманай, Сарманай,
Юлларында очраштыр,
Саргайтма безне болай.

Сарманай буйлары таллык,
Ярлары яшел чирэм.
Чэчэклэр жыйган тугайга
Сине сагынып килэм.

Сарманай, Сарманай,
Юлларында очраштыр,
Саргайтма безне болай.

Сарманай сулары салкын,
Салкын булса да йозэм.
Сагынуга чик-чама юк —
Тик сине котэм, тузэм.

Сарманай, Сарманай,
Юлларында очраштыр,
Саргайтма безне болай.
```

# Перевод

### Фан Валиахметов - Сарманай

```
Приезжай ты в Сарманай,
Встречу на дороге.
Есть слова, что скажу тебе
Только о нас двоих.

Сарманай, Сарманай,
На дорогах встреть нас,
Не изводи нас так.

Сарманай — по берегам ивы,
Склоны — зелёные луга.
В заливные луга, где цветы собирал,
По тебе тоскуя иду.

Сарманай, Сарманай,
На дорогах встреть нас,
Не изводи нас так.

Сарманай — воды прохладны,
Хоть прохладны, но плыву.
Тоске нет предела —
Только тебя жду, терплю.

Сарманай, Сарманай,
На дорогах встреть нас,
Не изводи нас так.
```
