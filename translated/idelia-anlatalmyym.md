# Оригинал

### Idelia - Анлаталмыйм

```
Карангы булмэдэ мин яшим сыман
Янгырлы тэрэзэгэ карыйм хаман
Машина тавышында синен шэулэн
Хэм минем хыялым ул юлда озелгэн.

Анлата алмыйм,оныта алмыйм
Шашам сыман
Мин адашам,мин югалам
Юкка чыгам.
Анлата алмыйм.

Кара хислэр дингезе кунелемдэ
Елмаюынны курэм тошлэремдэ
Куз яшьлэремне саный тыныч булмэ
Синсез донья мине ялгызлыкка кумэ.

Анлата алмыйм,оныта алмыйм
Шашам сыман
Мин адашам,мин югалам
Юкка чыгам.
Анлата алмыйм.
```

# Перевод

### Idelia - Не могу объяснить

```
Словно в темноте живу,
На дождливое окно смотрю всё время.
В шуме машин твой силуэт,
И моя мечта на той дороге разбилась.

Не могу объяснить, не могу забыть,
Словно схожу с ума.
Я блуждаю, я теряюсь,
Выхожу впустую.
Не могу объяснить.

Море тёмных чувств в душе моей,
Твою улыбку вижу в снах.
Слёзы мои считает тишина,
Без тебя мир меня к одиночеству толкает.

Не могу объяснить, не могу забыть,
Словно схожу с ума.
Я блуждаю, я теряюсь,
Выхожу впустую.
Не могу объяснить.
```
