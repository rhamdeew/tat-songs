# Оригинал

### Дилья Нигматуллина - Парлылар кинэше

```
Аерыламын, диеп соныннан
Кушылмый берэу дэ никахта.
Улгэнче бергэ дип тормышта
Вэгъдэлэр бирешэ ин алдан.

Кэккуклэр зарланып саныйлар
Ялгыз уткэн монсу кичлэрен.
Бэхетлелэр бергэ саклыйлар
Мэхэббэтнен керсез хислэрен.

Бэхетен тапмассын елап та,
Ни генэ булса да тормышта,
Рэнжетмэ ярынны син юкка
Янадан йорэген яуларга
Булмас бит иркэлэп, назлап та.
```

# Перевод

### Дилья Нигматуллина - Вина пары

```
Расстанемся, сказав потом,
Никто не соединяется в браке.
До смерти вместе, говоря в жизни,
Обещания дают заранее.

Кукушки, скорбя, считают
Одиноко прошедшие грустные вечера.
Счастливые вместе хранят
Любви искренние чувства.

Счастья не найти даже плача,
Что бы ни случилось в жизни,
Не мучай возлюбленную зря,
Чтобы снова сердце ранить —
Не получится лаской и нежностью.
```
