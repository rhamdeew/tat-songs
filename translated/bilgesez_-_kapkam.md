# Оригинал

### Билгесез - Капкам

```
Җилфер-җилфер искән җилләр,
Җил капканы җимергән.
Теләп алган каза түгел,
Җил хәлләрен кем белгән?
Ник аларсың һаман артса
Җил җимергән капкалар.
Җил җимергән капкаларны
Кемнәр кайтып каккалар?

Китәләр яшьләр авылдан,
Калалар картлар торып.
Искегә терәү куясың,
Яңаны булмый корып.

Күңелдә бетмәс сагышлар,
Юксыну сагышыдыр.
Тышта кемдер йөри кебек,
Малайлар тавышыдыр.
Бәла-казалар китереп,
Аптыратты җиле дә.
Тышта кемдер йөри дисәм,
Йөри икән җил генә.

Китәләр яшьләр авылдан,
Калалар картлар торып.
Искегә терәү куясың,
Яңаны булмый корып.
```

# Перевод

### Безымянный - Капкан

```
Ветер дует тихо-тихо,
Ветер разорвал ворота.
Не по ненависти льются слезы,
Кто способен в ветре разобраться?
Зачем ты всё время уходишь,
Ветром разрушенные ворота.
Кто же вернёт обратно
Ветром разрушенные ворота?

Уходят молодые из села,
Остаются старики стоять.
Опираешься на старое,
Новое без смысла выливается.

В сердце неиссякаемая тоска,
Тоска по отсутствию.
Снаружи кто-то ходит, как будто,
Голоса ребят слышны.
Принеси несчастья-беды,
Даже ветер запутался.
Если снаружи кто-то ходит,
То ходит только ветер.

Уходят молодые из села,
Остаются старики стоять.
Опираешься на старое,
Новое без смысла выливается.
```
