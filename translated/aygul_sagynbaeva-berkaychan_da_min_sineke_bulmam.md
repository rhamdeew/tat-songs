# Оригинал

### Айгуль Сагынбаева - Беркайчан да мин синеке булмам

```
Эзлэмэ син кабат очрашулар,
Юлларыбыз куптэн аерылган.
Тормыш сукмаклары чэбэлэнгэн,
Омет канатлары каерылган,
Омет канатлары каерылган.

Беркайчан да мин синеке булмам,
Кояш инде тошкэ авышкан.
Кулларымнан голлэр сибелсэ дэ,
Узып булмый икэн язмыштан,
Узып булмый икэн язмыштан.

Эзлэмэ син кабат очрашулар,
Уз тинен бар, сина ни кирэк?
Яшьлек хыялларын озелеп сагына,
Эллэ жулэрме икэн бу йорэк,
Эллэ жулэрме икэн бу йорэк.

Беркайчан да мин синеке булмам,
Кояш инде тошкэ авышкан.
Кулларымнан голлэр сибелсэ дэ,
Узып булмый икэн язмыштан,
Узып булмый икэн язмыштан.
```
