# Оригинал

### Гөзел Уразова - Тошэрмен ислэренэ

```
Керермен тошлэренэ,
Тошэрмен ислэренэ.
Минем генэ исем китмэс
Синен ул хислэренэ.

Телэрсен кочакларга,
Янарсын учакларда.
Эмма синен яннарында
Мин булмам ул чакларда.

Тарарсын чэчлэренне,
Котерсен дэшкэнемне.
Минем кулъяулыгым белэн
Сортерсен яшьлэренне.

Телэрсен кочакларга,
Янарсын учакларда.
Эмма синен яннарында
Мин булмам ул чакларда.

Килерсен яннарыма,
Басарсын алларыма.
Барыбер тузэ алмассын
Йорэген янганына.

Телэрсен кочакларга,
Янарсын учакларда.
Эмма синен яннарында
Мин булмам ул чакларда.
```

# Перевод

### Гөзел Уразова - В снах твоих буду

```
Войду в сны твои,
В снах буду дышать.
Моё имя не сходит
С тех чувств твоих.

Потянешься к объятьям,
Будешь гореть в огне.
Но рядом с тобою
Меня не будет в те моменты.

Расчёсываешь волосы,
Ждёшь друга.
Моим платочком
Вытираешь слёзы.

Потянешься к объятьям,
Будешь гореть в огне.
Но рядом с тобою
Меня не будет в те моменты.

Придёшь ко мне,
Прижмёшься к ладоням.
Всё равно не сможешь стерпеть
Что сердце пылает.

Потянешься к объятьям,
Будешь гореть в огне.
Но рядом с тобою
Меня не будет в те моменты.
```
