# Оригинал

### Ильсия Бадретдинова - Очрашу

```
Кайвакыт була шулай,
Кондез - кояш, тонлэ - ай.
Икенче конгэ исэ
Болытлы жиллэр исэ.
Куктэн тама куз яшьлэре
Янгыр тамчысы булып,
Булса булсын безнен остэ
Кыска гомерле болыт.

Бик куп еллар утэсен
Бэхетенне котэсен.
Кычкырып дэшмэсэ дэ,
Ул котэ янэшэдэ.
Жир шары бер эйлэнэ дэ,
Табыша ике йорэк.
Очрашкан ике йорэккэ
Бары бер бэхет кирэк.

Корылган донья шулай,
Кояшсыз яктырмый ай,
Ялгыз йорэк котэ пар,
Кара жир котэ ак кар.
Янгыр тамчысыдай оча
Жихан буйлап жир шары,
Бу жирдэ очрашу котэ
Хэр кемнен насыйп яры.
```

# Перевод

### Ильсия Бадретдинова - Встреча

```
Иногда бывает так,
Днём - солнце, ночью - луна.
На второй день же
Облачные ветры веют.
С небес капают слёзы
Дождевыми каплями,
Пусть будет над нами
Короткожизненное облако.

Очень много лет пройдёт
Счастье своё ты будешь ждать.
Хотя и не кричит,
Оно ждёт рядом.
Земной шар повернётся раз,
И встретятся два сердца.
Встретившимся двум сердцам
Нужно только одно счастье.

Устроен мир так,
Без солнца не светит луна,
Одинокое сердце ждёт пару,
Чёрная земля ждёт белый снег.
Как дождевая капля летит
По всей вселенной земной шар,
На этой земле встречи ждёт
Каждого суженый.
```
