# Оригинал

### Гузель Уразова - Сою нуры

```
Сизэсенме икэн узен шуны?..
Мин эйтмичэ генэ яратам...
Житкэндэ дэ гомерлэрнен соны,
Син булырсын минем йорэктэ...

Сою нуры безне кавыштырды,
Гомерлеккэ ярын булдым синен.
Йорэгемнен кочле яратуы;
Сина тиннэр булмас минем очен.

Белэсенме икэн, сою нуры
Мина сине генэ уйлата...
Безнен яшэешнен якты таны:
" Сез бэхетле!"- диеп уята.

Сою нуры безне кавыштырды,
Гомерлеккэ ярын булдым синен.
Йорэгемнен кочле яратуы;
Сина тиннэр булмас минем очен.

Утеп кенэ тора еллар саны...
" Кайтмас инде!"- диеп санама?..
Гомер купме, белеп булмый аны...
Сою нуры безне ташлама?!.

Сою нуры безне кавыштырды,
Гомерлеккэ ярын булдым синен.
Йорэгемнен кочле яратуы;
Сина тиннэр булмас минем очен.
```

# Перевод

### Гузель Уразова - Свет любви

```
Чувствуешь ли ты это?..
Я молча тебя люблю...
Даже когда жизни придет конец,
Ты будешь в моем сердце...

Свет любви нас соединил,
На всю жизнь твоей стала.
Сильная любовь моего сердца;
Тебе равного не будет для меня.

Знаешь ли, свет любви
Заставляет думать только о тебе...
Светлый рассвет нашей жизни:
"Вы счастливы!" - будит.

Свет любви нас соединил,
На всю жизнь твоей стала.
Сильная любовь моего сердца;
Тебе равного не будет для меня.

Проходит лишь годы считая...
"Не вернется!" - не думай так...
Сколько жизни, не узнать этого...
Свет любви нас не оставляй!

Свет любви нас соединил,
На всю жизнь твоей стала.
Сильная любовь моего сердца;
Тебе равного не будет для меня.
```
