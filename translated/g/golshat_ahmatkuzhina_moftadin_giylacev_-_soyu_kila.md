# Оригинал

### Гөлшат Әхмәткужина / Мөфтәдин Гыйләҗев - Сөю килә

```
Ашкына поезд, ашкына,
Каршы килә яшел кыр.
Әй сөюем, олы юлдай,
Мине үзеңә чакыр.

Бәхетем син әле килгән
Гөлчәчәкле кыр сыман.
Йөгерче, йөгер каршыма,
Сабыр итүе озак.

Көтүе ай-һай авыр бит,
Поезд килә дә килә.
Тилмертеп җаным, тилертеп,
Каршыма сөю килә.
```

# Перевод

### Гөлшат Әхмәткужина / Мөфтәдин Гыйләҗев - Приходит любовь

```
Мчит поезд, мчит он,
Навстречу - зелёный простор.
О, любовь моя, как путь широкий,
Позови меня в свой простор.

Счастье моё - ты, только пришедшая,
Словно поле в цвету весной.
Беги же, беги навстречу мне,
Терпенье так долго длится.

Ожидание - ой как тяжко,
Поезд всё идёт и идёт.
Томит душу, кружит голову -
Навстречу любовь идёт.
```
