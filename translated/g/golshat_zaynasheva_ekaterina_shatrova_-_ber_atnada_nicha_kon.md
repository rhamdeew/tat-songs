# Оригинал

### Гөлшат Зәйнәшева / Екатерина Шатрова - Бер атнада ничә көн?..

```
Бер атнада ничә көн?
Бер атнада җиде көн.
Җиде төрле исеме -
Җидесен дә яттан бел:

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе -
Онытма мәңге!..

Шимбә белән якшәмбе -
Икесе ял көннәре.
Ял көннәре атнаның -
Чәчәкләре, гөлләре.

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе -
Онытма мәңге!..

Ял көннәре атнаның
Чәчәкләре, гөлләре.
Атнаның яфраклары,
Димәк, калган көннәре.

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе -
Онытма мәңге!..
```

# Перевод

### Гөлшат Зәйнәшева / Екатерина Шатрова - Сколько дней в неделе?..

```
Сколько в неделе дней?
В неделе - семь дней.
Семь разных имён -
Выучи все наизусть:

Понедельник, вторник, среда,
Четверг, пятница,
Суббота, воскресенье -
Не забывай никогда!..

Суббота и воскресенье -
Оба выходные дни.
Выходные у недели -
Её цветы, её сады.

Понедельник, вторник, среда,
Четверг, пятница,
Суббота, воскресенье -
Не забывай никогда!..

Выходные у недели -
Её цветы, её сады.
А листва у этой недели -
Значит, прочие дни.

Понедельник, вторник, среда,
Четверг, пятница,
Суббота, воскресенье -
Не забывай никогда!..
```
