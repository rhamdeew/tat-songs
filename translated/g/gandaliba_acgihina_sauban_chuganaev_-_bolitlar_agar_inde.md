# Оригинал

### Гандәлибә Аҗгихина / Сәүбән Чуганаев - Болытлар агар инде

```
Кышларымда узып бара,
Тагы бер язым җитә.
Көтүләрем бушка булгач,
Гомерем синсез үтә.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.

Салкын томаннар сарыла
Иртәләрем талларга.
Ходаем, түземлекләр бир
Ялгыз калган ярларга.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.

Сөенәм дә, көенәм дә,
Язларым узып бара.
Булмадың бәхет йолдзым,
Булдың гомерлек яра.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.
```

# Перевод

### Гандәлибә Аҗгихина / Сәүбән Чуганаев - Облака светлеют уже

```
В моих зимах всё уходит,
Вот и новая весна.
Раз надежды были тщетны -
Жизнь проходит без тебя.

Облака светлеют уже,
Капли скоро потекут.
Сердце, что осталось жечься,
Будет жечься, будет жечься вновь.

Холодные туманы
Обвивают утренние ивы.
Боже, дай терпение
Берегам, что в одиночестве остались.

Облака светлеют уже,
Капли скоро потекут.
Сердце, что осталось жечься,
Будет жечься, будет жечься вновь.

То радуюсь, то скорблю я,
Мои весны уходят вдаль.
Не была ты моей звездой счастья,
Стала вечной раной ты.

Облака светлеют уже,
Капли скоро потекут.
Сердце, что осталось жечься,
Будет жечься, будет жечься вновь.
```
