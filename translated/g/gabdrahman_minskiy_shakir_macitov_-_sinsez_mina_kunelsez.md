# Оригинал

### Габдрахман Минский / Шакир Мәҗитов - Синсез миңа куңелсез

```
Сагынып көтәм сине, ярым,
Заргып көтәм, тиз кайт, жаным,
Синсез миңа бик куңелсез,
Ах, бик куңелсез!

Синсез миңа шатлык та юк,
Синсез миңа бер ямь дә юк,
Синсез миңа яшәү дә юк,
Бәгырем, кайт инде!

Сәгать сукты, көн авышты,
Нигә соңга калдың инде?
Сагынып көтүләре авыр,
Аңла син мине.

Синсез миңа шатлык та юк,
Синсез миңа бер ямь дә юк,
Синсез миңа яшәү дә юк,
Бәгырем, кайт инде!
```

# Перевод

### Габдрахман Минский / Шакир Мәҗитов - Без тебя мне тоскливо

```
Скучая жду тебя, мой милый,
Изныв, я жду: вернись скорей, любимый,
Без тебя мне так тоскливо,
Ах, так тоскливо!

Без тебя мне радости нет,
Без тебя мне и красы нет,
Без тебя мне жизни нет,
Сердце, возвращайся!

Час пробил, день к закату клонится,
Почему ты задержался?
Тяжело томиться в ожидании,
Пойми меня.

Без тебя мне радости нет,
Без тебя мне и красы нет,
Без тебя мне жизни нет,
Сердце, возвращайся!
```
