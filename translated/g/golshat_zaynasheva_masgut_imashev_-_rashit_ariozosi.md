# Оригинал

### Гөлшат Зәйнәшева / Мәсгуть Имашев - Рәшит ариозосы

```
Миңа насыйп Казан кызы,
Кайда соң син, кайда соң син?
Әллә инде җирдә түгел,
Йолдыздамы, айдамы син?

Нинди ялкын, нинди утлар
Йөрәккәемне әрнетә?
Алда мине ниләр көтә,
Язмышымны кем хәл итә?

Кая барсам, ник мин ялгыз,
Язмыш күгем ник сагышлы?
Кем соң аңлар йөрәгемне,
Кем җылытыр бу язмышны?
```

# Перевод

### Гульшат Зайнашева / Масгут Имашев - Ариозо Рашита

```
Мне суждена казанская дева,
Где же ты, где же ты?
Иль уже не на земле ты,
Звездою ли стала, луною ли ты?

Какое пламя, какой огонь
Моё сердце терзает?
Что впереди меня ждёт,
Кто решит мою судьбу?

Куда ни пойду, почему я один,
Почему небо судьбы печально?
Кто же поймёт моё сердце,
Кто согреет эту судьбу?
```
