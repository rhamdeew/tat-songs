# Оригинал

### galiahmat_shahi_vitaliy_agapov_ - Галиәхмәт Шаһи / Виталий Агапов - Курай (беренче вариант)

```
Курай, курай, ышанычлы дус та,
Сердәш тә син татар халкына.
Мәңге яшәр халык мәхәббәте
Җырга-моңга, чабыш атына.

Курай, курай, ничә гасыр инде
Халкым сине кулдан төшерми.
Татар бит ул шундый, атына да
Сызгырмыйча су да эчерми.

Курай, курай, бар сагышын сөйли,
Бар моңнарын көйли татарның.
Оча күңел хыялына таба
Иярендә тулпар атларның.

Курай, курай, ничә гасыр инде
Юлдашы син татар җырының.
Синең кебек әнә сайрый тургай
Күкрәгендә иген кырының.
```

# Перевод

### Галиәхмәт Шаһи / Виталий Агапов - Курай (первый вариант)

```
Курай, курай, ты друг надежный,
Ты доверенный татарскому народу.
Вечно будет жить любовь народа
К песне-моңу, к скачущему коню.

Курай, курай, сколько уж веков
Народ тебя из рук не выпускает.
Татар таков: и коня
Без свиста водой не напоит.

Курай, курай, всю печаль расскажешь,
Все напевы татарские выводишь.
Летит душа к своей мечте
На седле крылатых скакунов.

Курай, курай, сколько уж веков
Ты спутник татарской песни.
Так же поет жаворонок
В груди хлебного поля.
```
