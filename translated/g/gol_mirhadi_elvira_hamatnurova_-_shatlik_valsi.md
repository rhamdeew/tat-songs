# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Шатлык вальсы

```
Челтерәп көләләр җирдә чишмәләрем,
Зәңгәр күкне нурлап кояш көлә.
Елмая каршыма очраган кешеләр, -
Күңелемдә шатлык: алар белә.

Җемелдәшеп күктә бииләр йолдызлар
Урманда куышып җилләр уйный.
Җир шары әйләнә; сөенәм, сөеләм,
Күңелем шатланып һич тә туймый.

Сабантургай сайрап талпына һавада,
Узыша кошларым моңнар түгеп.
Шатлыкка, җырларга, моңга сусаганнар
Килегез, күңелемне бирәм бүлеп.
```

# Перевод

### Гюль Мирхади / Эльвира Хаматнурова - Вальс радости

```
Журчат мои родники в земле смеясь,
Лазурное небо светит солнце, улыбаясь.
Встречные люди мне улыбаются навстречу, -
В душе моей радость: они знают это.

Шелестя в небе, танцуют звёзды,
В лесу, встречаясь, ветра играют.
Вращается земной шар; люблю, любима,
Душа моя ликует, не насытясь никогда.

Жаворонок, распевая, рвётся в небеса,
Птицы мои парят, рассыпая мелодии.
Кто жаждет радости, песен и напевов -
Приходите, душу свою дарю, деля.
```
