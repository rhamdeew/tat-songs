# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Ышанам!

```
Нигә башын түбән идең,
Аерлабыз дисеңме?
Бәлки әле аерылмабыз,
Аерылмабыз ышанам!

Ышанам, ышанам,
Аерылмабыз ышанам!

Соңгы тапкыр күрәм диеп
Башын салма иңемә.
Бәлки әле кавышырбыз,
Кавышырбыз ышанам!

Ышанам, ышанам,
Кавышырбыз ышанам!

Без барасы озын юллар
Бер булырга охшаган.
Бәлки әле очрашырбыз,
Очрашырбыз ышанам!

Ышанам, ышанам,
Очрашырбыз ышанам!
```

# Перевод

### Гульшат Зайнашева / Илһам Шакиров - Верю!

```
Почему голову опустил,
Хочешь расстаться, что ли?
Может, ещё не расстанемся,
Не расстанемся, верю!

Верю, верю,
Не расстанемся, верю!

В последний раз увижу, думая,
На плечо не клади.
Может, ещё встретимся,
Встретимся, верю!

Верю, верю,
Встретимся, верю!
Мы идём долгими дорогами
Будто стать единым целым.
Может, ещё свидимся,
Свидимся, верю!

Верю, верю,
Свидимся, верю!
```
