# Оригинал

### Гөлшат Зәйнәшева / Рәүф Адилов - Чишмә башы

```
Кайда безнең чишмә башы,
Туктап калган агышы.
Югалган чишмәләр кебек
Безнең милләт язмышы.

Таңнарга чыгып эзләсәм
Табармын чишмә башын.
Кайда безнең чишмә диеп,
Сайрыйдыр минем сандугачым.

Балтырганнар өзә идем
Су буенда таллыкта.
Шушы җирләрдә адаштым
Сукмак эзләп бардыкта.

Таңнарга чыгып эзләсәм
Табармын чишмә башын.
Кайда безнең чишмә диеп,
Сайрыйдыр минем сандугачым.

Табар идем чишмә башын
Улаклар куяр өчен.
Шул чишмә буена килеп
Сабырлык җыяр өчен.

Таңнарга чыгып эзләсәм
Табармын чишмә башын.
Кайда безнең чишмә диеп,
Сайрыйдыр минем сандугачым.
```

# Перевод

### Гөлшат Зәйнәшева / Рауф Адилов - Источник

```
Где наш ключевой источник,
Остановившийся поток.
Как потерянные родники,
Такова судьба нашего народа.

Если выйду на рассвете искать,
Найду ли ключевой источник.
Где наш родник, спрашивает
Мой песенный жаворонок.

Стояла я, как балтика,
На берегу воды, в тоске.
На этих землях заблудился,
Идя по следу источника.

Если выйду на рассвете искать,
Найду ли ключевой источник.
Где наш родник, спрашивает
Мой песенный жаворонок.

Нашел бы я ключевой источник
Для того, чтобы птички пели.
Подойдя к той воде источника,
Чтобы набрать терпения.

Если выйду на рассвете искать,
Найду ли ключевой источник.
Где наш родник, спрашивает
Мой песенный жаворонок.
```
