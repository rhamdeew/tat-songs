# Оригинал

### Гөлнур Куатова / Юлай Моратов - Җыр җырладым

```
Җыр җырладым, тик бергенә,
Җырларым бит керде йөрәккә.
Кич ятуга, шул моң чыңлый,
Тагы җырла әле сирекләп.

Яратам дип, бик эз әйтәсең,
Назланасым килә ешырак.
Күрешәсем килә минем көндә,
Тик аралар гына бик ерак.

Саран булма хискә, җырга,
Күңел ачып бүген сөергә.
Ике түгел тик бер яшибез
Калсын әле сагынып сөйләргә.
```

# Перевод

### Гөлнур Куатова / Юлай Моратов - Песню спела

```
Песню спела, но лишь одну,
Ведь песни вошли в сердце.
Вечером, на сон грядущий,
Эта мелодия звучит,
Спой ещё, прошу, немного.

Люблю, говоришь - скудно на слова,
Влюблённой хочу быть почаще.
Хочется видеть тебя каждый день,
Только расстояния так далеки.

Не будь скуп на чувства, на песни,
Душу открой сегодня для любви.
Не двое, мы только одни живём,
Останемся тоскующими влюблёнными.
```
