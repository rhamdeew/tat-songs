# Оригинал

### Гәрәй Рәхим / Рифкать Сәйфетдинов - Бәйрәм (беренче вариант)

```
Татар җыелган җирдә
Бәйрәмсез булмый бер дә,
Бәйрәм, бәйрәм.
Ике татар - бер базар,
Базарда телләр язар,
Бәйрәм, бәйрәм, бәйрәм.

Җыелышса бергә -
Татар бергә гомергә.

Җәйның матур көнендә,
Илнең бөтен җирендә
Бәйрәм, бәйрәм.
Җыелышып татарлар
Сабан туе ясарлар.
Бәйрәм, бәйрәм, бәйрәм.

Җыелышса бергә -
Татар бергә гомергә.
```

# Перевод

### Гарай Рахим / Рифкать Сайфетдинов - Праздник (первый вариант)

```
Где татары соберутся,
Без праздника не обойтись,
Праздник, праздник.
Двое татар - уже базар,
На базаре языки развяжут,
Праздник, праздник, праздник.

Соберутся вместе -
Татарин с татарином навек.

В летний день, такой красивый,
По всей стране, на всех просторах
Праздник, праздник.
Соберутся татары -
Сабантуй устроят.
Праздник, праздник, праздник.

Соберутся вместе -
Татарин с татарином навек.
```
