# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Изге эшләр эшлә

```
Изгелектән, игелектән,
Торсын синең гомер юлың.
Синнән үрнәк алып үссен,
Горурланып кызың, улың.

Изге булган эшләр эшлә!
Башкаларга юлны күрсәт!
Ярты юлда тукталганга,
Ялгышларын әйтеп төзәт!

Яшә гөрләп, яшә җырлап,
Җырың калсын буыннарга.
Балаларга, оныкларга,
Рәхәтләнеп бер тыңларга.

Изге булган эшләр эшлә!
Башкаларга юлны күрсәт!
Ярты юлда тукталганга,
Ялгышларын әйтеп төзәт!

Кеше дигән, бу исемне,
Югары йөрт, югары тот!
Эшлә, тырыш, уйла,булдыр,
Бәхет дигән, кошны син от!

Изге булган эшләр эшлә!
Башкаларга юлны күрсәт!
Ярты юлда тукталганга,
Ялгышларын әйтеп төзәт!
```

# Перевод

### Гөлфия Шакирова / Лилия Вахитова - Делай добрые дела

```
Пусть дорогой твоей жизни
Станут доброта и свет.
Пусть, беря с тебя пример,
Дочь и сын тобой гордятся.

Делай только добрые дела!
Покажи другим дорогу!
Тем, кто на полпути застыл,
Ошибаясь, помоги исправить.

Живи в силе, пой, живи,
Пусть твоя песня детям останется.
Чтобы детям и внукам
С радостью её слушать.

Делай только добрые дела!
Покажи другим дорогу!
Тем, кто на полпути застыл,
Ошибаясь, помоги исправить.

Имя «человек» держи
Высоко, с честью неси.
Действуй, трудись, мечтай, свершай,
Птицу счастья поймай ты сам.

Делай только добрые дела!
Покажи другим дорогу!
Тем, кто на полпути застыл,
Ошибаясь, помоги исправить!
```
