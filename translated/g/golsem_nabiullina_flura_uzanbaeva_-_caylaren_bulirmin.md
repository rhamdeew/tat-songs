# Оригинал

### Гөлсем Нәбиуллина / Флүрә Үзәнбаева - Җәйләрең булырмын

```
Җәйләр булып мин килермен,
Салкын кышларда.
Күңелеңне боексытма,
Иркәм мин барда!

Минем янда син бәхетле,
Көчле булырсың.
Мин дә синең белән иркәм,
Горур булырмын!

Салкын көздә карайталмас,
Синең күңлеңне.
Яздай балкып яктыртырмын,
Синең йөзеңне!

Минем янда син бәхетле,
Көчле булырсың.
Мин дә синең белән иркәм,
Горур булырмын!

Туар таңың, килер көнең,
Гел аяз булыр.
Арып-талган йөрәгеңә,
Сөю-наз тулыр!

Минем янда син бәхетле,
Көчле булырсың.
Мин дә синең белән иркәм,
Горур булырмын!
```

# Перевод

### Гульсем Набиуллина / Флюра Узанбаева - Стану твоим летом

```
Я приду к тебе как лето,
В стужу зимних дней.
Не печалься понапрасну,
Милый, я с тобой!

Рядом со мной ты счастливым,
Сильным будешь ты.
Я с тобою буду нежной,
Гордой буду я!

Холод осени не сможет
Твоё сердце тронуть.
Я, как весна, сиять заставлю
Твоё лицо!

Рядом со мной ты счастливым,
Сильным будешь ты.
Я с тобою буду нежной,
Гордой буду я!

Каждый рассвет твой, каждый день твой
Будет ясным впредь.
Уставшее сердце твоё
Любовью наполню!

Рядом со мной ты счастливым,
Сильным будешь ты.
Я с тобою буду нежной,
Гордой буду я!
```
