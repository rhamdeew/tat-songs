# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Зурәткәем, зурәнкәем

```
Якты дөньяның яме алар,
Дип өйрәтә әткәй-әнкәй.
Безне сөеп назлап яшәсен
Гел зурәткәй һәм зурәнкәй.

Узең туеп күзең туймаслык,
Тел йотарлык сезнең сыйлар.
Көлеп каршы алыр зурәткәй,
Зурәнкәем сөйеп сыйлар.

Җылы куен тик сездә генә,
Канат бирә сезнең сүзләр.
Горурланып шуклыкка хәттә
Нурлар сибә сезнең күзләр.

Гомерлеккә күңелгә салдык
Сезнең сөю, теләкләрне.
Бүген сезгә гомерләр теләп
Тибә онык йөрәкләре.
```

# Перевод

### Гөл Мирһади / Эльвира Хаматнурова - Бабушка милая, дедушка родной

```
Вы - отрада этого света,
Так твердили мама с папой.
Чтоб любя нас, нежно жили
Милая бабушка и дед.

Угоститесь досыта - всё мало,
Ваши яства - язык проглотишь.
Улыбаясь встретит бабушка,
Дед родной с любовью угостит.

Только у вас так тепло в объятьях,
Слова ваши крылья дарят.
Даже шалостями гордясь,
Взоры ваши светом брызжут.

Навсегда в душе мы спрятали
Вашу нежность, ваши просьбы.
Пусть же, долгих лет желая,
Бьются внуков сердца сегодня.
```
