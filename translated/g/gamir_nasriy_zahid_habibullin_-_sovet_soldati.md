# Оригинал

### Гамир Насрый / Заһид Хәбибуллин - Совет солдаты

```
Кидең өскә соры шинельне,
Яклар өчен туган илеңне,
Ил өчен утка кердең син,
Ярты дөньяны гиздең син, -
Дан сиңа, дан, совет солдаты!

Эзең калды Карпат тавында,
Каның тамды Эльба ярында;
Авыр походлар үттең син,
Берлинга барып җиттең син, -
Дан сиңа, дан, совет солдаты!

Даның яңгырый синең еракта -
Тын океанда, биек Карпатта;
Мәгърур басып ил чигендә
Сакта торасың бүген дә, -
Дан сиңа, дан, совет солдаты!
```

# Перевод

### Гамир Насрый / Заһид Хәбибуллин - Советский солдат

```
Надел ты серую шинель,
Ради краёв своей родной земли,
Ради страны вошёл ты в огонь,
Половину света исходил ты, -
Слава тебе, слава, советский солдат!

След твой остался в Карпатских горах,
Кровь твоя текла по берегу Эльбы;
Тяжёлые походы прошёл ты,
До Берлина дошёл ты, -
Слава тебе, слава, советский солдат!

Слава твоя разносится далеко -
В Тихом океане, в высоких Карпатах;
Гордо ступая на границе родной
Охраняешь ты и по сей день, -
Слава тебе, слава, советский солдат!
```
