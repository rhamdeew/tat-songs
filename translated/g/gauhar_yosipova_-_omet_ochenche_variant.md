# Оригинал

### Гәүһәр Йосыпова - Өмет (өченче вариант)

```
Таралдылыр томан кебек,
Яшьлекнең хыяллары.
Үкенечсез узмый бит ул
Гомерләрнең юллары.

Дөнья булгач - бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!

Гомер юлларында күпме
Хата, ялгышлары да,
Өмет җитәклидер мине
Адашыр чакларымда.

Дөнья булгач - бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!

Айкалдыра, чайкалдыра
Тормышның киртәләре,
Бик өмет белән нурлансын
Һәркемнең иртәләре.

Дөнья булгач - бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!
```

# Перевод

### Гаухар Юсупова - Надежда (третий вариант)

```
Рассеются, как туман,
Мечты юности.
Без сожаленья проходят ведь
Дороги жизни.

Если есть мир - всё бывает:
Бывают трудные времена.
Смотри вперед: там горят
Маяки надежды!

На дорогах жизни сколько
Ошибок, заблуждений,
Надежда поведет меня
В минуты сомнений.

Если есть мир - всё бывает:
Бывают трудные времена.
Смотри вперед: там горят
Маяки надежды!

Качает, колышет
Жизненная буря,
Пусть надеждой озарится
Каждое утро.

Если есть мир - всё бывает:
Бывают трудные времена.
Смотри вперед: там горят
Маяки надежды!
```
