# Оригинал

### Гыйләҗ Минаев / Фәрит Хатыйпов - Болгар - чишмә яныңда

```
Чишмәләр дә шаһит үткәннәргә,
Күпме гасыр узып китсә дә,
Ташларында калган кан таплары,
Күз яшьләре күптән кипсә дө.

Янәшәдә үскән төз каеннар,
Нык тамырлы, горур имәннәр.
Искә алып данлы үткәннәрне
Чал башларын түбән игәннәр.
```

# Перевод

### Гиляз Минаев / Фарит Хатыпов - Булгар - у родника

```
Родники - свидетели прошлого,
Сколько веков ни минуло бы,
На камнях их остались пятна крови,
Хоть слёзы давно высохли.

Рядом выросли стройные берёзы,
Крепкокорневые, гордые дубы.
Вспоминая славное прошлое,
Они склонили седые головы.
```
