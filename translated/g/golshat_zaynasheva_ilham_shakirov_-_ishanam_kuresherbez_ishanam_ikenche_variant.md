# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Ышанам! (Күрешербез, ышанам!..) (икенче вариант)

```
Ник моңаеп башын идең,
Аерлабыз дисеңме?
Бәлки әле аерылмабыз,
Аерылмабыз шикелле.

Соңгы тапкыр күрәм диеп
Башын салма иңемә.
Бәлки әле күрешербез,
Күрешербез иртәгә?..

Без барасы озын юллар
Бер булырга охшаган.
Озакламый очрашырбыз,
Очрашырбыз ышанам!

Ышанам, ышанам,
Очрашырбыз ышанам!
```

# Перевод

### Гөлшат Зәйнәшева / Илһам Шакиров - Верю! (Встретимся, верю!..) (второй вариант)

```
Что ты голову склонила,
Говоришь: нам разойтись?
Может, всё же не расстанем,
Не расстанем мы, кажись.

Не склоняй ко мне головку,
Не прощайся навсегда.
Может, всё же мы увидимся,
Встретимся уже завтра?..

Долгий путь, что нам идти,
Словно создан для двоих.
Очень скоро мы увидимся,
Встретимся - я верю в них!

Верю я, верю я,
Встретимся - я верю я!
```
