# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Алдашмагыз!

```
Алдадым диеп шатланмагыз.
ул бит куаныч түгел
кемнең нинди сүз сөйләвен
Барыбер тоя күңел.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.

Дөреслекне юк итмәгез
гөнаһларга кермәгез.
Изгелек эшләр урынга
Ялган, хәйлә үрмәгез.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.

Ходай биргән бер гомергә
бик матур яшәү кирәк.
Яшәү бәхет шушы җирдә.
Диеп типсен безнең йөрәк.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.
```

# Перевод

### Гульфия Шакирова / Лилия Вахитова - Не обманывайте!

```
Не радуйтесь, что обманули.
Ведь это не радость,
Какие бы слова
Ни чувствовала душа.

Не обманывайте, не обманывайте,
Не говорите лжи.
Ради Аллаха людям
Не желайте зла.

Не уничтожайте правду,
Не впадайте в грехи.
Добрые дела пусть
Не вырастят ложь и коварство.

Не обманывайте, не обманывайте,
Не говорите лжи.
Ради Аллаха людям
Не желайте зла.

Во всей жизни, что дал Аллах,
Нужно жить очень красиво.
Счастье жизни - на этой земле.
Так стучит наше сердце.

Не обманывайте, не обманывайте,
Не говорите лжи.
Ради Аллаха людям
Не желайте зла.
```
