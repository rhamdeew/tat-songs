# Оригинал

### Гөлшат Зәйнәшева / Виталий Агапов - Газиз туган як

```
Иделе киң, диңгезгә тиң,
Ямьле безнең туган ягыбыз.
Башка бер ил кирәк түгел,
Шунда үтсен тормыш юлыбыз.

И туган як, газиз туган як,
Бишегемә былбыл кунган як.
Сандугачлар белән сайрашып,
Сабый чагым узган як.

Туган як дип сагынып кайта,
Язлар җиткәч, сайрар кошлары.
Язлар матур, җәйләр ямьле,
Чыныктыра салкын кышлары.

И туган як, газиз туган як,
Бишегемә былбыл кунган як.
Сандугачлар белән сайрашып,
Сабый чагым узган як.

Ярсулы безнең йөрәкләр,
Без үсәбез катлаулы чорда.
Яшь йөрәгең белән ярсып,
Илне сөйсәң, кушыл бу җырга.

И туган як, газиз туган як,
Бишегемә былбыл кунган як.
Сандугачлар белән сайрашып,
Сабый чагым узган як.
```

# Перевод

### Гөлшат Зәйнәшева / Виталий Агапов - Дорогой родной край

```
Широка Идель, как море без края,
Как прекрасна родная сторона.
Другая страна нам вовсе не надо,
Пусть здесь пройдёт вся жизненная тропа.

Эй, край родной, мой дорогой,
Колыбель, где вился соловей.
С соловьями в перекличке
Детство прожил я здесь.

С тоской по краю родному вернутся
С весной певчие птицы сюда.
Весны прекрасны, лета цветисты,
Закаляет морозная зима.

Эй, край родной, мой дорогой,
Колыбель, где вился соловей.
С соловьями в перекличке
Детство прожил я здесь.

Смелы наши сердца и горячие,
Мы растём в непростое время.
Если любишь страну ты сердцем юным,
Подпевай этой песне вместе с нами.

Эй, край родной, мой дорогой,
Колыбель, где вился соловей.
С соловьями в перекличке
Детство прожил я здесь.
```
