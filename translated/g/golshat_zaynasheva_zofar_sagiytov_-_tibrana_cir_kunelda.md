# Оригинал

### Гөлшат Зәйнәшева / Зөфәр Сәгыйтов - Тибрәнә җыр күңелдә

```
Язын сыерчыклар кайтып,
Оясын тапса,
Бакчаларда алмагачлар
Шау чәчәк атса,
Кояш көлсә, ай елмайса
Илем күгендә, -
Тибрәнә җыр дулкыннары
Минем күңелдә!

Бәхетле елмаеп, ана
Баласын кочса,
Тыныч күктә ак юл сызып,
Самолет очса,
Дуслар гөрләшеп утырса
Табын түрендә, -
Тибрәнә җыр дулкыннары
Минем күңелдә!

Шауласа бодай кырлары,
Бөртеккә тулып.
Чакырса ерак офыклар,
Яңа юл булып.
Үсеп чыкса зур бер кала
Туган җиремдә, -
Тибрәнә җыр дулкыннары
Минем күңелдә!..
```

# Перевод

### Гульшат Зайнашева / Зуфар Сагитов - В душе колышется песня

```
Весной вернутся птицы,
Найдя своё гнездо,
В садах яблони
Шумно расцветут,
Солнце засмеется,月亮 подмигнет
В небе моем родном, -
Колышутся волны песни
В моей душе!

Счастливо улыбаясь, мама
Обнимет ребенка,
В мирном небе оставляя белый след,
Полетит самолет,
Друзья, смеясь, сидят
За праздничным столом, -
Колышутся волны песни
В моей душе!

Шумят поля пшеничные,
Наполненные колосом.
Зовут далекие горизонты,
Становясь новой дорогой.
Если вырастет большой город
На моей родной земле, -
Колышутся волны песни
В моей душе!..
```
