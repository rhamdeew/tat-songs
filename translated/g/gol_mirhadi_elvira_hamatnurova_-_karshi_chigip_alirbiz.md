# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Каршы чыгып алырбыз

```
Кырлар тынып йокога талганнар
Кар-буранны көтәләр.
Ак туннарын киеп куйганнар,
Кыш бабайны көтәләр.

Бик зарыгып сагынсалар да
Алар сабыр итәләр
Йөгерешеп безнең бәйрәмгә
Чабып килеп җитәләр

Кыш бабайны каршы алырга
Бар җәнлекләр килгәннәр
Безнең җырны ишеткән алар,
Утлар яна күргәннәр.

Кыш бабабыз,синең килгәнне,
Көтәбез без барыбыз.
Тавыш бир, бабай арысаң,
Каршы чыгып алырбыз.
```

# Перевод

### Гуль Мирхади / Эльвира Хаматнурова - Встретим с радостью

```
Поля затихли, спать легли они,
Метелей ждут, вьюги.
Белые шубы надели свои,
Деда Мороза ждут они.

Хоть очень сильно они заждались,
Терпеньем запаслись.
К нашему празднику вприпрыжку, вскачь
Примчатся, прибегут.

Деда Мороза встречать сюда
Все звери собрались.
Нашу песню услышав тогда,
Огни увидев, зажглись.

Дедушка Мороз, мы ждём тебя,
Мы все, все здесь стоим.
Подай нам знак, коль устал, любя,
Мы встретим, подхватим.
```
