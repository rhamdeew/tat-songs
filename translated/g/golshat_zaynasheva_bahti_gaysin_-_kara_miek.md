# Оригинал

### Гөлшат Зәйнәшева / Бәхти Гайсин - Кара мыек

```
Сызылып киткән кара мыек, -
Әллә каралта микән?
Мыек астыннан елмая,
Әллә ярата микән?

Сызылып киткән кара мыек
Йөрәгемә ут салды.
Бер көн озата килде дә
Әллә кая югалды.

Сызылып киткән кара мыек
Кайларда йөри микән?
Кайнар мәхәббәт сүзләрен
Кемнәргә сөйли икән?

Сызылып киткән кара мыек -
Буяган мыек икән;
Буялган мыеклыларга
Ышану кыен икән!..
```

# Перевод

### Гөлшат Зайнашева / Бахти Гайсин - Черные усы

```
Тонко вычерчены черные усы -
То ли чернятся, как тень?
Из-под усов улыбнётся,
Неужто любит меня?

Тонко вычерчены черные усы
Сердцу моему пламя дали.
Как-то раз проводил меня -
И исчез неизвестно куда.

Тонко вычерчены черные усы,
Где же теперь ты бродишь?
Горячие слова любви
Кому ты нынче скажешь?

Тонко вычерчены черные усы -
Оказалось, крашеные.
Тем, кто усы красят,
Верить так трудно!..
```
