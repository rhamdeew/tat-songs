# Оригинал

```
Без яраткан айлы кичләр

Еракта калган икән.

Үләннәр үскән инде.

Башкалар сөйгән инде.

Без яраткан чәчәкләрне
Кыраулар төшеп алган.

Ятларга насыйп булган,
Ятларга насыйп булган.

Без йөргән сукмакларга
Үләннәр үскән инде.
Яшьлектә сөйгән ярларны
Башкалар сөйгән инде.

Кабат-кабат уйлап куям,
Исләргә төшкән саен.
Сөюләр сүрелгән инде
Оныту гына кыен,
Оныту гына кыен.

Без йөргән сукмакларга
```
