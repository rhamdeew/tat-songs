# Оригинал

### Гөлшат Зәйнәшева / Гөлнара Беляева - Без - табигать дуслары

```
Җәйге ямьле иртәдә
Чыгабыз болыннарга.
Әйлән-бәйлән уйныйбыз
Иң матур урыннарда.

Без - табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.

Керәбез урманнарга,
Йөрибез аланнарда.
Куркытмыйбыз кошларны,
Тимибез боланнарга.

Без - табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.

Җәйге ямьле иртәдә,
Көн уянган сәгатьтә,
Күңелле дуслар белән
Ял итү табигатьтә.

Без - табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.
```

# Перевод

### Гульшат Зайнашева / Гульнара Беляева - Мы - друзья природы

```
В летнее прекрасное утро
Выходим мы на луга.
В круг свой дружный собираемся
В самых красивых местах.

Мы - дети природы,
Мы друзья природы.
Нас приветственно встречают
Птицы родного края.

Входим в леса мы,
Гуляем по полянам.
Не пугаем птиц,
Не трогаем оленей.

Мы - дети природы,
Мы друзья природы.
Нас приветственно встречают
Птицы родного края.

В летнее прекрасное утро,
В час, когда солнце встаёт,
С весёлыми друзьями
Отдыхаем на природе.

Мы - дети природы,
Мы друзья природы.
Нас приветственно встречают
Птицы родного края.
```
