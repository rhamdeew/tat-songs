# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Дуслар (өченче вариант)

```
Дусым белән ерак түгел
Күрше торабыз.
Уйныйбыз курчаклар белән
Өйләр корабыз
Дус булып гел дә
Йөрибез бергә.
Бармы дуслар безнең шикелле?!

Үпкәләшкән чаклар була,
Озакка түгел.
Ялгызлыкка һичкенә дә
Чыдамый күңел.
Ачулансак та,
Бик елашсак та,
Яңадан дуслашу күңелле!

Дәресләрдә булышабыз
Бер-беребезгә.
Чыкмый миннән, чыкмый синнән
Бер серебез дә.
Үрнәк алыгыз,
Дуслар табыгыз,
Дус булыгыз безнең шикелле!

Шатланабыз, кайгырабыз
Икегә бүлеп.
Ашыйбыз хәттә кәнфитне
Урталай телеп.
Бәләкәй диеп
Карама көлеп.
Дуслык безнең өчен кадерле!
```

# Перевод

### Гөл Мирһади / Эльвира Хаматнурова - Друзья (третья версия)

```
С подружкой живем совсем недалеко,
Рядом, словно соседи.
В куклы играем, домики строим,
Дружбой живем и верим.
Дружно всегда,
Ходим бок о бок.
Есть ли такие друзья, как мы, еще кто-нибудь?

Случится обида - она недолга,
Скоро тоска растает.
Сердце ни капли одиночества
Вынести не желает.
Хоть рассердимся,
Хоть наплачемся -
Мириться снова так радостно!

На уроках помогаем
Друг другу без просьбы.
Не выйдет ни из меня, ни из тебя
Ни одна наша тайна.
Берите пример,
Ищите друзей,
Дружите крепко, как мы, всегда!

Радость и горе делим
Только на двоих.
Даже конфету делим ровно,
На две половины.
Не смейся, глядя,
Что мы малы еще.
Дружба для нас - сокровище!
```
