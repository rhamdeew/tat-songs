# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Кил яныма

```
Елмайганда, чәчәкләр дә
Көнләшә торган чагың.
Күмелгәнсең сагышларга,
Ни булды сиңа тагын?

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.

Күңелеңнең болытларын
Кул белән таратырмын.
Күзләремә каратырмын,
Өзелеп яратырмын.

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.

Су буйлары куе томан,
Чакырам томаннарга.
Алып барам чишмәләргә,
Чылтырап торганнарга.

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.
```

# Перевод

### Галяхмат Шахи / Виталий Агапов - Подойди ко мне

```
Даже цветы, когда не улыбнёшься,
Завидуют твоему времени.
 Погрузился ты в печали,
Что с тобой ещё случилось?

Куда ушли твои радости,
Почему взгляд такой мрачный?
Подойди ко мне, исправлю
Раны твоего сердца.

Облака на душе твоей
Рассеку своими руками.
На глаза тебя взгляну,
Буду любить всей душой.

Куда ушли твои радости,
Почему взгляд такой мрачный?
Подойди ко мне, исправлю
Раны твоего сердца.

Вдоль рек густой туман,
Зову в туманы.
Отведу к родникам,
К сверкающим водам.

Куда ушли твои радости,
Почему взгляд такой мрачный?
Подойди ко мне, исправлю
Раны твоего сердца.
```
