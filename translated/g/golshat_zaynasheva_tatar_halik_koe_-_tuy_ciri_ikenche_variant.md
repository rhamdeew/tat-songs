# Оригинал

### Гөлшат Зәйнәшева / Татар халык көе - Туй җыры (икенче вариант)

```
Икәү бергә утыргансыз,
Бергә булсын уегыз;
Икегез дә пар килгәнсез,
Котлы булсын туегыз!

Пар былбыллар кавышкандай,
Ничек кавыштыгыз сез?!
Бу кадәр матур пар булып
Ничек табыштыгыз сез?!

Сокланып карап туймассың
Зифа буегызга да.
Онытмагыз чакырырга
Бәбәй туегызга да.
```
