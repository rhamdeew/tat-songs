# Оригинал

### Гөлшат Зәйнәшева / Сара Садыйкова - Илдарым

```
Еллар үтә, еллар үтә,
Күңелкәем сине көтә.
Көтмә, күңел, көтмә, күңел,
Кавышулар мөмкин түгел.
Илдарым, Илдарым,
Булалмадың сөйгән ярым.

Гомер уза, гомер уза,
Сине уйлап йөрәк сыза.
Никтер һаман, никтер һаман
Юллар бергә килер сыман.
Илдарым, Илдарым,
Булалмадың сөйгән ярым.
```

# Перевод

### Гөлшат Зәйнәшева / Сара Садыйкова - Ильдарым

```
Годы идут, годы идут,
Сердце мое тебя все ждет.
Не жди, сердце, не жди, сердце,
Нам сойтись не суждено.
Ильдарым, Ильдарым,
Не стал ты любимой долей.

Жизнь идет, жизнь идет,
От мыслей о тебе сердце ноет.
Почему-то все кажется, все кажется,
Будто наши дороги сойдутся.
Ильдарым, Ильдарым,
Не стал ты любимой долей.
```
