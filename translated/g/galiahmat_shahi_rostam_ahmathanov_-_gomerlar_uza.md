# Оригинал

### Галиәхмәт Шаһи / Рөстәм Әхмәтханов - Гомерләр уза ...

```
Агачтан өзелеп
Яфраклар коела.
Яшь чакта гомерләр
Үтмәс күк тоела.

Сизми дә каласың
Гомернең үткәнен.
Күңелдә сайраган
Былбылның киткәнен.

Хәтерләр чиртәләр
Күңелнең кылларын.
Җыр булып очалар
Канатлы елларым.

Вакытның үткәнен
Үзгәртеп булмый шул!
Гомернең узганын
Кайтарып булмый шул!
```

# Перевод

### Галияхмет Шахи / Рустам Ахметханов - Годы уходят ...

```
Обрываясь с деревьев,
Листья осыпаются.
В молодости годы
Кажутся - не пройдут.

Не почувствуешь,
Как жизнь проходит.
Что соловей, певший в сердце,
Улетел - не поймёшь.

Память дёргает
Струны души.
Песней улетают
Мои крылатые годы.

Прошедшее время
Не переделать!
Прожитую жизнь
Не воротить!
```
