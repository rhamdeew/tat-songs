# Оригинал

### Габделхак Шәмсетдинов / Зөфәр Хәйретдинов - Бер елмайсаң, Әминә

```
Ничек әйтеп бирим икән
Сиңа булган хисемне?..
Җырларыма кушып җырлыйм,
Иркәм, синең исемне.

Күзләреңнән нур сирпелә
Бер елмайсаң, Әминә.
Син бит минем бердәнберем,
Җан сөйгәнем син генә.

Сагышлардан газапланам
Күрми торсам аз гына.
Гөлләр сула, кошлар тына
Син булмасаң янымда.

Күзләреңнән нур сирпелә
Бер елмайсаң, Әминә.
Син бит минем бердәнберем,
Җан сөйгәнем син генә.

Синнән башка пар юк миңа,
Син - күземнең карасы.
Йөрәгемә шифа була
Синең назлы карашың.

Күзләреңнән нур сирпелә
Бер елмайсаң, Әминә.
Син бит минем бердәнберем,
Җан сөйгәнем син генә.
```
