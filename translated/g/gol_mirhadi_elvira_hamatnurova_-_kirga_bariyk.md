# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Кырга барыйк

```
Яңгыр ява,
Кояш көлә,
Күпер сузып,
Күкне бүлә, -
Зәңгәр күккә
Күпер буйлап
Их, менәсе,
Уйнап-уйнап!

Көмеш тамчы
Җиргә төшә,
Болоннарда
Җиләк пешә, -
Сулап, сулап
Җиләк исе
Болыннарда
Их,йөрисе!

Кояш күктә
Күпер төзеп,
Җиләк тезеп.
Безне көтә, -
Әткәй, Әнкәй,
Кырга барып,
Йөреп кайтыйк
Яллар алып!
```

# Перевод

### Гөл Мирһади / Эльвира Хаматнурова - Поедем в поле

```
Дождик идёт,
Солнце смеётся,
Радугу тянет,
Небо делит, -
По синеве
Радугой ввысь
Ах, бы подняться,
Прыгать, кружить!

Серебром капли
На землю летят,
В лугах поспевает
Душистая ягода, -
Дыша и дыша
Её ароматом,
По лугам бы
Ах, побродить!

Солнце на небе
Радугу строит,
Ягоды нанижет -
Ждёт нас оно, -
Әткәй, Әнкәй,
В поле поедем,
Погуляем
И вернёмся с отдыхом!
```
