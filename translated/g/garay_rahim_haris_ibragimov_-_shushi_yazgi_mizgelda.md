# Оригинал

### Гәрәй Рәхим / Харис Ибрагимов - Шушы язгы мизгелдә

```
Тып - тып итеп
Тыпылдап,
Чылтыр - чылтыр
Чылтырап,
Биек йортлар түбәсеннән
Боз сөңгеләр коела.

Ялт - йолт итеп
Ялтырап,
Дерелдәп һәм
Калтырап
Урамдагы гөрләвектә
Кояш нуры коена.

Чырык - чырык
Чыркылдап,
Очып йөри
Пырхылдап
Өй турендә чыпчыклар,
Шушы язгы мизгелдә.

Карамыйча бернигә
Өйдән чыгып чабар идем -
Әни кушкан эшем бар.
```

# Перевод

### Гәрәй Рәхим / Харис Ибрагимов - В это весеннее мгновенье

```
Кап-кап, мерно
Капает,
Звонко-звонко
Звенит,
С крыш высоких домов
Сосульки срываются вниз.

Блеск да всполох,
Сиянье,
Дрожью тонкой
Дрожа,
В уличном ручейке
Купается лучик солнца.

Чирик-чирик,
Щебеток,
Вспархивая,
Порхая,
Под крышей кружат воробьи
В это весеннее мгновенье.

Не глядя ни на что,
Выбежал бы из дома -
Да есть у меня мамино дело.
```
