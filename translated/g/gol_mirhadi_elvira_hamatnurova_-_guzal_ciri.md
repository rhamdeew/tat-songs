# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Гүзәл җыры

```
Әнкәйнең назлы кызы мин
Сайрар кошчык - илемдә.
Якты моңнар, матур җырлар
Яши гүзәл күңлемдә

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Сөя җирем, назлый җилем,
Тоям кояш җылысын,
Үсәм шатлык өстәп җиргә,
Көтеп таңның җырлысын.

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Таңда-кошчык, төндә-йолдыз
Булып яшисем килә.
Күңелләргә якты моңнар,
Җыр өләшәсем килә.

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Моңнар булып кунганмын.
```

# Перевод

### Гуль Мирхади / Эльвира Хаматнурова - Прекрасная песня

```
Нежная дочка мамы я,
Поющая птичка - в краю моём.
Светлые мелодии, прекрасные песни
Живут в моей прекрасной душе.

Ради песен для вас, друзья,
Я родилась на свет.
На землю мелодиями
Снизошла я.

Люблю землю мою, ласкаю край мой,
Впиваю тепло солнца,
Расту, добавляя радость земле,
В ожидании певней зари.

Ради песен для вас, друзья,
Я родилась на свет.
На землю мелодиями
Снизошла я.

Птичкой на заре, звездой в ночу
Хочу жить мне.
К сердцам светлые мелодии,
Песни хочу дарить я.

Ради песен для вас, друзья,
Я родилась на свет.
На землю мелодиями
Снизошла я.

Мелодиями снизошла я.
```
