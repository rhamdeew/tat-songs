# Оригинал

### Гөлшат Зәйнәшева / Бату Мулюков - Яшь кызларга

```
Чәчәккә күмелсен,
Кояшка үрелсен, -
Үстер син яшьлеген гөлләрен
Гомергә бер килә,
Кадерен бел генә -
Унсигез яшь тулган көннәрең.

Җырла син өздереп,
Уйна син сыздырып,
Жәлләмә гармуның телләрен.
Гомергә бер килә,
Кадерен бел генә -
Унсигез яшь тулган көннәрең.

Хезмәттә үсәрсең,
Алмадай пешәрсең,
Эшләсәң сызганып җиңнәрең.
Гомергә бер килә,
Кадерен бел генә -
Унсигез яшь тулган көннәрең.
```

# Перевод

### Гөлшат Зайнашева / Бату Мулюков - Юным девушкам

```
Пусть в цветах утонет,
К солнцу потянется, -
Ты взрасти цветы своей юности.
Раз в жизни приходит,
Береги лишь, -
Дни, когда тебе восемнадцать.

Пой ты без удержу,
Играй ты с разливом,
Не жалей язычков гармошки.
Раз в жизни приходит,
Береги лишь, -
Дни, когда тебе восемнадцать.

В труде ты поднимешься,
Как яблоко, созреешь,
Если закатаешь рукава.
Раз в жизни приходит,
Береги лишь, -
Дни, когда тебе восемнадцать.
```
