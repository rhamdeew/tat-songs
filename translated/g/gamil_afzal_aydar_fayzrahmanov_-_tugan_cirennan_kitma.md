# Оригинал

### Гамил Афзал / Айдар Фәйзрахманов - Туган җиреңнән китмә

```
Ак болытлар, күк болытлар
Күлләргә карап үтә.
Әйтә кебек төнбоеклар:
- Туган җиреңнән китмә!

Беркайда да оҗмах түгел,
Туган илеңнән китмә син
Изге җиреңнән китмә.

Башын чайкап уйга талган
Туган як каеннары;
Җаннарыңа кереп калган
Каенның кайгылары.

Каеннарны моңлы итмә,
Туган җиреңнән китмә син,
Изге җиреңнән китмә.

Су буенда баш ияләр
Сыгылма таллар сиңа.
Алларыма там, диярләр
Таң кебек ярлар сиңа.

Ярларыңны ятим итмә,
Талларыңны ятим итмә,
Туган җиреңнән китмә.
```

# Перевод

### Гамил Афзал / Айдар Файзрахманов - Не уходи с родной земли

```
Белые тучи, небесные тучи
Смотрят на озера в пути.
Словно кувшинки шепчут, звуча:
- Не уходи с родной земли!

Нигде на свете нет рая,
Не покидай родную страну,
Не покидай святую землю.

Качают головы, в думе березы
Родного края молчат;
В душу твою навсегда вросли
Печали берез, их взгляд.

Не делай березы печальными,
Не уходи с родной земли,
Не покидай святую землю.

У воды склоняются гибкие ивы
Перед тобой в тишине.
«У нас останься», - шепчут берега,
Светлые, как заря в окне.

Не делай берега сиротами,
Не делай ивы сиротами,
Не уходи с родной земли.
```
