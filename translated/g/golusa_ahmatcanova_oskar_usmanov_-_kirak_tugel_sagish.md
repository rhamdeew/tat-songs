# Оригинал

### Гөлүсә Әхмәтҗәнова / Оскар Усманов - Кирәк түгел сагыш

```
Күз яшьләре түгеп,
Еллар үтте диеп,
Сагышланып утта
Яшәгәнмен көеп.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.

Бәхет килер диеп
Көткән чакларымны,
Искә алам хәзер
Яшьлек елларымны.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.

Бәхет килә икән
Көтә белсәң генә.
Уйлар синең белән,
Бәхетем син генә.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.
```

# Перевод

### Гулюса Ахметжанова / Оскар Усманов - Не нужна печаль

```
Слёзы лились рекой,
Годы прошли стороной,
В тоске я сгорала,
Словно в пламени жила.

Не нужна печаль.
Не нужна ошибок даль.
В печалях, в ошибках
Не запутай меня, судьба.

Когда счастье ждала,
Верой жила и мечтой,
Вспоминаю сейчас
Те годы молодости.

Не нужна печаль.
Не нужна ошибок даль.
В печалях, в ошибках
Не запутай меня, судьба.

Счастье приходит лишь
К тем, кто умеет ждать.
Мысли с тобою,
Ты моё счастье одно.

Не нужна печаль.
Не нужна ошибок даль.
В печалях, в ошибках
Не запутай меня, судьба.
```
