# Оригинал

### Гөлнара Рәшитова / Урал Рәшитов - Әлшәем

```
Борманалып ага Дим сулары,
Ярларында шаулый таллары.
Талларыңда сайрый кошлары -
Шуңа ямьле Әлшәй яклары,

Әлшәем минем, яшәгән җирем,
Тик сина гына бәйле язмышым минем.

Яшел-хәтфә урман-кырлары,
Җилләреңдә курай моңнары.
Кошларның да моңлы җырлары -
Шуңа матур Әлшәй яклары.

Әлшәем минем, яшәгән җирем,
Тик сина гына бәйле язмышым минем.

Алсуланып ата таңнары,
Таңкызыңдай сылу кызлары,
Бахадирдай батыр уллары -
Шуңа данлы Әлшәй яклары.

Әлшәем минем, яшәгән җирем,
Тик сина гына бәйле язмышым минем.
```

# Перевод

### Гульнара Рашитова / Урал Рашитов - Мой Альшей

```
Извиваясь, течёт река Дим,
На берегах шумят ивы.
В ивах твоих поют птицы -
Потому прекрасны края Альшея.

Альшей мой, земля, где живу,
Только с тобой связана судьба моя.

Зелёным бархатом леса и поля,
В ветрах твоих курая напевы.
И птиц печальные песни -
Потому красивы края Альшея.

Альшей мой, земля, где живу,
Только с тобой связана судьба моя.

Алым разгорается заря,
Красавицы, как зарница, дочери,
Как Бахадир, отважные сыны -
Потому славны края Альшея.

Альшей мой, земля, где живу,
Только с тобой связана судьба моя.
```
