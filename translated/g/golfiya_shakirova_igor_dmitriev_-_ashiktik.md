# Оригинал

### Гөлфия Шакирова / Игорь Дмитриев - Ашыктык

```
Әллә син җибәрдең хата,
﻿﻿Әллә мин ясадым.
Белмим, нигә кабатладык
Ялгыш булган бу адымны.

Ашыктык бугай, ашыктык
﻿﻿Хаталарны ясарга.
Инде хәзер үткәннәргә
Юл юк кире кайтырга.

Үткәннәрдән бары калды
Томанлы уйлар гына.
Төн йокысыз калдырган
Сагышлы моңнар гына.

Ашыктык бугай, ашыктык
﻿﻿Хаталарны ясарга.
Инде хәзер үткәннәргә
Юл юк кире кайтырга.

Ни җитмәде, аңламадык,
Үкенүдән ни файда.
Исләремә төшмә әле,
Ямансу бит болай да.

Ашыктык бугай, ашыктык
﻿﻿Хаталарны ясарга.
Инде хәзер үткәннәргә
Юл юк кире кайтырга.
Ашыктык бугай, ашыктык
﻿﻿Хаталарны ясарга.
Инде хәзер үткәннәргә
Юл юк кире кайтырга.
```
