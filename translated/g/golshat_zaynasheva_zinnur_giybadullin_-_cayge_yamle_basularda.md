# Оригинал

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Җәйге ямьле басуларда

```
Җәйге ямьле басуларда
Бик ямансу кичләрен.
Җәйге кичтә яратам мин
Кырда арыш исләрен.

Җәйге ямьле басуларда
Дулкынлана арышлар.
Йөрәктә уянып куя
Онытылган сагышлар.

Җәйге ямьле басуларда
Карыйм сезнең якларга.
Чык каршыма, күрешербез
Эштән кайткан чакларда.
```

# Перевод

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - В летних милых полях

```
В летних милых полях
Так тоскливы вечера.
В летний вечер люблю я
Запах ржи на просторах.

В летних милых полях
Волнуется спелая рожь.
В сердце вдруг пробуждаются
Позабытые грусти.

В летних милых полях
Я гляжу в вашу сторону.
Выйди мне навстречу, свидимся
Когда с работы вернусь.
```
