# Оригинал

### Гөлфия Шакирова / Гүзәлия - Көтмә мине

```
Көтмә мине, көтмә мине,
Кайтмыйм ул ярга кабат.
Башка ярга күчте тормыш,
Тормый үткәнгә карап.

Көтмә, көтмә, көтмә мине,
Моңсуланма, моңланма.
Сагыш күлләренә төшеп,
Юк, кирәкми, уйланма.

Тулган айга серең сөйләп,
Үткәрмәче көнеңне.
Берни эшләтеп тә булмый
Төшкә кергән өнеңне.

Көтмә, көтмә, көтмә мине,
Моңсуланма, моңланма.
Сагыш күлләренә төшеп,
Юк, кирәкми, уйланма.
Көтмә, көтмә, көтмә мине,
Моңсуланма, моңланма.
Сагыш күлләренә төшеп,
Юк, кирәкми, уйланма.
Сагыш күлләренә төшеп
Юк, кирәкми, уйланма...
```
