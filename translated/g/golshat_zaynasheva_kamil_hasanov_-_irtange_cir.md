# Оригинал

### Гөлшат Зәйнәшева / Камил Хәсәнов - Иртәнге җыр

```
Таңны мактап талларда
Кошлар канат кактылар.
Тамчы гөлләр тибрәнеп,
Алкаларын тактылар.

Күбәләкләр уянып,
Канатларын җилпеде.
Кондызлар җемелдәвек
Ефәк күлмәген киде.

Чәчәкләр чык суында
Коенып юындылар.
Бал кортлары безелдәп,
Чәчәкләргә кундылар.

Син дә, матур сеңелем,
Татлы йокыңнан уян.
Җәйге таңның яменә,
Гүзәллегенә куан.

Уян, сеңелем, күзең ач,
Җәйге иртә нурында.
Кояш чыкты алланып
Синең тәрәз турыңда.
```

# Перевод

### Гөлшат Зәйнәшева / Камил Хәсәнов - Утренняя песня

```
Тальники славят рассвет,
Птицы взмахнули крылами.
Капли-цветы, чуть дрожа,
Надели серьги сияния.

Бабочки, проснувшись, легко
Крыльями мягко взмахнули.
Бобры, мерцая, надели
Шелковый свой наряд.

В росе цветы купались,
Умылись чистой водой.
Пчёлы, жужжа, опускались
На лепестки за медом.

Ты тоже, сестрица милая,
Проснись от сладкого сна.
Летнего утра красою,
Его красотой восхищайся.

Проснись, сестрица, глаза открой
В сиянии летней зари.
Солнце взошло, зарумянилось
У твоего окна.
```
