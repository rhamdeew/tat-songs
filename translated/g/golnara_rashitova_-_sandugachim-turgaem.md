# Оригинал

### Гөлнара Рәшитова - Сандугачым-тургаем

```
Сандугач басып сайраган
Тал бер генә була ул.
Җан яраткан, күңел тарткан
Яр бер генә була ул.
Сандугачым, сайра, сайра
Үзәкләрне өздереп,
Гашыйк булган бу йөрәкне
Булмас хәзер түздереп.

Сандугачым-тургаем,
Күрсәм куллар болгаем.
Мин болгагач син дә болга,
Мин дә мәхрүм калмаем.
Сандугачым-тургаем...

Сандугачым икәнсең лә,
Җитәр көзләр - китәрсең.
Җаның сөймәгән ярларга
Булсаң - ни хәл итәрсең?
Сандугачым, бас талыңа,
Тагын да кагынырсың.
Читкә киткәч шул талыңны,
Илеңне сагынырсың.

Сандугачым-тургаем,
Күрсәм куллар болгаем.
Мин болгагач син дә болга,
Мин дә мәхрүм калмаем.
Сандугачым-тургаем...

Сандугач басып сайраган
Тал бер генә була ул.
Җан яраткан, күңел тарткан
Яр бер генә була ул.
Сандугачым, сайра, сайра
Үзәкләрне өздереп,
Гашыйк булган бу йөрәкне
Булмас хәзер түздереп.

Сандугачым-тургаем,
Күрсәм куллар болгаем.
Мин болгагач син дә болга,
Мин дә мәхрүм калмаем.
Сандугачым-тургаем...
```
