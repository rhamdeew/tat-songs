# Оригинал

### Гыйльман Идрисов / Рим Хәсәнов - Галия җыры

```
Айның якты нуры җиргә атылганда,
Чәчләремнән сыйпап җилләр искәндә.
Хисләр ташый микән
Яшьли сөйгән ярым
Тәүге умырзая бүләк иткәндә?

Сәлам белән бергә, хатлар китерсәләр
Үтеп киткән язның кипкән гөлләрен.
һәр тибеше саен йөрәк көтә микән
Тиздән килеп җитәр кавышу көннәре.
```

# Перевод

### Гыйльман Идрисов / Рим Хәсәнов - Песня Галии

```
Когда лунный свет на землю падает,
Когда ветер нежно волосы ласкает.
Неужели чувства через край?
Юный мой любимый,
Первый подснежник мне даря?

Если б с приветом письма приносили
Высохшие цветы ушедшей весны.
Каждым ударом сердце не ждёт ли
Дней скорой, близкой нашей встречи.
```
