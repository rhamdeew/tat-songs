# Оригинал

### golfiya_yunisova_ildar_salimov_ - Өфем

```
Синдә белем, илһам алам,
Яшәү өчен дуслар табам,
Тау башында гоур калам,
Яшьлек төсем -
Гүзәл Өфем.

Сагышларны куа-куа,
Дүрт тарафтан елга юа,
Синдә аяз, якты дөнья
Һәр кем өчен,
Нурлы Өфем.

Урамнарың тора гөрләп,
Йортлар калка күккә үрләп,
Илне данлап, яна дөрләп
Куәт - көчең,
Уңган Өфем.

Сагышларны куа-куа,
Дүрт тарафтан елга юа,
Синдә аяз, якты дөнья
Һәр кем өчен,
Нурлы Өфем.

Хисле калам, көчле калам,
Газиз сиңа барлык балаң.
Гомер буе хәйран калам,
Хыял төсем -
Өфем, Өфем!

Сагышларны куа-куа,
Дүрт тарафтан елга юа,
Синдә аяз, якты дөнья
Һәр кем өчен,
Нурлы Өфем.
```

# Перевод

### Гөлфия Юнысова / Илдар Салимов - Моя Уфа

```
В тебе беру я знанья, вдохновенье,
Для жизни нахожу друзей в пути,
На вершинах гор живу я в восхищенье,
Цвет юности моей -
Прекрасная Уфа.

Прочь печали, гоня их непрестанно,
Со всех сторон тебя реки омывают,
У тебя - мир ясный, светозарный
Для каждого,
Светлая Уфа.

Улицы твои шумят, кипя простором,
Дома взлетают к небу, как мечта,
Прославляя край, горит могучим взором
Сила - мощь твоя,
Умелая Уфа.

Прочь печали, гоня их непрестанно,
Со всех сторон тебя реки омывают,
У тебя - мир ясный, светозарный
Для каждого,
Светлая Уфа.

Остаюсь я чутким, остаюсь я сильным,
Все твои дети сердцем с тобой,
Целый век дивлюсь я чуду светлокрылому,
Цвет мечты моей -
Уфа, Уфа!

Прочь печали, гоня их непрестанно,
Со всех сторон тебя реки омывают,
У тебя - мир ясный, светозарный
Для каждого,
Светлая Уфа.
```
