# Оригинал

### Гөлия / 16quyan - Себер тракты

```
Киртәләргә карап
Үтә елларым.
Авыр һава сулап
Китә җаннарым.

Юллардагы утлар
Яктырталар мине.
Елап беткән еллар
Чагылалар инде.

Күктә яна ай.
Янында йолдыз.
Нигә соң болай
Миңа күңелсез?

Көчле бураннар
Томалый юлны.
Башымда шул бер
Борчый торган тынгы.

Буран тик үземә
Юлларда тыныч,
Тик күңелемдә
Бар бер аяныч.

Күктә яна ай.
Янында йолдыз.
Нигә соң болай
Миңа күңелсез?
```

# Перевод

### Гулия / 16quyan - Сибирский тракт

```
Взгляд на заборы,
Прошедшие годы.
Тяжело дыша,
Душа моя рвётся.

Огни на дорогах
Озаряют меня.
Проплаканные годы
Отражаются вдали.

На небе горит луна.
Рядом с ней звезда.
Почему же так
Печально мне на душе?

Сильные метели
Заметают дорогу.
В голове та одна
Мучающая меня мысль.

Метель лишь вокруг меня,
На дорогах спокойно,
Только в душе моей
Одно лишь стенанье.

На небе горит луна.
Рядом с ней звезда.
Почему же так
Печально мне на душе?
```
