# Оригинал

### Гөл Чәчәк / Владимир Федоров - Кадерлеләрем

```
Бәхетлемен диеп уянам,
Һәр яңа көн җиргә туган саен.
Якыннарым булсын янәшәмдә,
Мәхрүм итмә шуннан, Ходаем!

Кызым, улым - бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.

Сагынып кайтамын ераклардан,
Кочагыма алып шатланам.
Бәхетемнең күреп җимешләрен,
Куңелем оча канатланып.

Кызым, улым - бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.

Сезнең белән бергә елмаям,
Сагышлардан табам дәваны.
Килгәнсез шушы якты дөньяга,
Яшьлегемнең булып дәвамы.

Кызым, улым - бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.
```

# Перевод

### Гөл Чәчәк / Владимир Федоров - Дорогие мои

```
Просыпаюсь с мыслью: я счастлива,
Каждый новый день, что на землю рожден.
Пусть родные будут рядом со мною,
Не лишай меня этого, Боже!

Дочка, сын - единственные,
Мои дорогие.
В саду любви взошедшие
Бутоны цветов.

С тоской возвращаюсь издалека,
Обнимаю - и радость в груди.
Вижу плоды моего счастья -
И сердце летит, расправив крыла.

Дочка, сын - единственные,
Мои дорогие.
В саду любви взошедшие
Бутоны цветов.

С вами вместе улыбаюсь,
От печалей нахожу исцеленье.
Вы пришли в этот светлый мир -
Продолженье моей молодости.

Дочка, сын - единственные,
Мои дорогие.
В саду любви взошедшие
Бутоны цветов.
```
