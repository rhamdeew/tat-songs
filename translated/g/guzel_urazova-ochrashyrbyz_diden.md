# Оригинал

### Гүзәл Уразова - Очрашырбыз диден

```
"Очрашырбыз," – диден... Очрашмадык...
Югалдын син инде мэнгегэ...
Алларында кар-буранлы юллар..
Ах адашты ялгыз жан гына.

"Очрашырбыз," – диден... Очрашмадык...
Нигэ генэ жаным жилкенде?
Мине юатырдай тоштер инде
Урсэлэнгэн буран шикелле...

Тошлэремэ кереп иркэлэден,
Юксынуларыма кушылып.
Сепертмэле карлар оерелэ,
Сэгать тагын кучми башына...

"Очрашырбыз," – диден... Очрашмадык...
Нигэ генэ жаным жилкенде?
Мине юатырдай тоштер инде
Урсэлэнгэн буран шикелле...
```

# Перевод

### Гүзәл Уразова - Сказал, что встретимся

```
"Встретимся", - сказал... Не встретились...
Потерялся ты уже навсегда...
Впереди снежно-буранные дороги...
Ах, заблудилась одинокая душа.

"Встретимся", - сказал... Не встретились...
Почему только душа моя дрожит?
Меня, словно утешая, снова снится
Разбушевавшаяся метель...

В мои сны входя, ласкает,
К тоске моей присоединяясь.
Метельные снега кружатся,
Время опять не переносится в начало...

"Встретимся", - сказал... Не встретились...
Почему только душа моя дрожит?
Меня, словно утешая, снова снится
Разбушевавшаяся метель...
```
