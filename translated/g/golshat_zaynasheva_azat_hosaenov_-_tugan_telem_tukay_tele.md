# Оригинал

### Туган телем - Тукай теле

```
Туган телем - Тукай теле,
Җырлы-моңлы шигъри тел.
Туган илне яраттырып
Яшәтүче сихри тел.

Кушымта:
Тукай җыры йөрәгемдә,
Шигырьләре күңелдә.
Тукай дигән йолдыз балкый,
Балачагым күгендә.

Рәсемнән елмая Тукай,
Һәр тарафка нур сибеп.
Илтә кебек туган җиргә,
Илгә лаек бул, диеп.

Кушымта.

Туган телем-Тукай теле,
Җырлы-моңлы шигъри тел.
Бишегемдә әткәм-әнкәм,
Бүләк иткән сихри тел.

Кушымта.
```

# Перевод

### Гөлшат Зәйнәшева / Азат Хөсәенов - Мой родной язык - язык Тукая

```
Мой родной язык - язык Тукая,
Песенный, напевный, поэтичный.
Он родную землю полюбить велит,
Язык волшебный, жизнь дарящий.

Припев:
Песня Тукая - в моем сердце,
Стихи его - в душе моей.
Звезда по имени Тукай сияет
В небе моего детства.

С портрета улыбается Тукай,
Сеет свет по всем сторонам.
Будто к родной земле ведет и шепчет:
«Будь достоин своей страны».

Припев.

Мой родной язык - язык Тукая,
Песенный, напевный, поэтичный.
В колыбели отец и мать
Подарили мне язык волшебный.

Припев.
```
