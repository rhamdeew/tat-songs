# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Яз килә, яз көлә!

```
Гөрләвек, гөрләвек
Җырлап ага урамнан.
Арынган, арынган
Җирләр карлы юрганнан.

Яз килә,
Яз килә, -
Ак карлар эреп беткән.
Яз көлә,
Яз көлә, -
Кояш елмая күктән.

Көймәләр, көймәләр
Узышабыз йөздереп.
Сыерчык, сыерчык
Сайрап тора өздереп.

Яз килә,
Яз килә, -
Ак карлар эреп беткән.
Яз көлә,
Яз көлә, -
Кояш елмая күктән.

Йокыдан, йөкыдан
Җир уяна яз килгәч.
Урамга ашыгам
Тәрәзәдән яз килгәч.

Яз килә,
Яз килә, -
Ак карлар эреп беткән.
Яз көлә,
Яз көлә, -
Кояш елмая күктән.
```
