# Оригинал

### Гөлнара Рәшитова / Урал Рәшитов - Умырзая (икенче вариант)

```
Күңелемә килеп кердең
Язгы ташкын, күкрәү булып,
Тәүге ләйсән булып явып,
Язгы умырзаялар алып.

Умырзая, умырзая,
Узмасын гомер зая.
Әйт син миңа, умырзая,
Яшьлек ярым минем кая?

Умырзая чәчәкләрен
Өзсәм йөрәк тибә ярсып.
Үткәннәрне искә алып,
Кил син умырзаялар алып.

Умырзая, умырзая,
Узмасын гомер зая.
Әйт син миңа, умырзая,
Яшьлек ярым минем кая?
```

# Перевод

### Гөлнара Рәшитова / Урал Рәшитов - Подснежник (второй вариант)

```
Ты вошла ко мне в сердце
Как весенний поток и гром,
Первые теплые капли пролив,
Принеся весенний подснежник в дом.

Подснежник мой, подснежник мой,
Пусть жизнь не прожита впустую.
Скажи мне, нежный подснежник,
Где половина юности, родная моя?

Стоит сорвать твои цветы -
Сердце пульсом бьется, вспыхнув вдруг.
Вспомнив прошлое, приходи же
Со снегом ранним, с подснежником, друг.

Подснежник мой, подснежник мой,
Пусть жизнь не прожита впустую.
Скажи мне, нежный подснежник,
Где половина юности, родная моя?
```
