# Оригинал

### Гөлнур Якупова / Рамил Гыймрани - Кем булдың син миңа?

```
Кем булдың син миңа,
Ярым гына микән?
Бала кебек иркәләнгән миңа
Син үзең түгел микән?

Ярым, җаным микән,
Тогры ярым микән?
Җаным, ярым, серле ярым
Син үзең түгел микән?

Нур булдың син миңа,
Җыр булдың син миңа.
Сөеп туймас, онытылмас
Яр булдың син миңа.
```

# Перевод

### Гульнур Якупова / Рамил Гыймрани - Кем ты стала для меня?

```
Кем ты стала для меня,
Только половинка ль моя?
Как ребенка ласкала меня ты
Разве не ты ли, не ты ли, моя?

Половинка, душа моя,
Настоящая ль половинка моя?
Душа моя, половинка, тайная половинка
Разве не ты ли, не ты ли, моя?

Светом стала ты для меня,
Песней стала ты для меня.
Ненасытной любви, незабвенной
Другом стала ты для меня.
```
