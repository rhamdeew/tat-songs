# Оригинал

### Гөлшат Зәйнәшева / Рәис Нәгыймов - Янам, көям

```
Янам, көям, янам, көям,
Ут капты күлмәгемә.
Ут капмады күлмәгемә,
Капты яшь йөрәгемә.

Янмамын дигән идем,
Көймәмен дигән идем.
Сөю - газап, беркемне дә
Сөймәмен дигән идем.

Уф, дип, йолдызларга карыйм,
Карыйм зәңгәр төннәрдә.
Гашыйк булдым буй җитмәслек
Йолдызга тиң берәүгә.

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю - газап, беркемне дә
Сөймәмен дигән идем.

Килә икән ул мәхәббәт
Юрап тормыйча гына.
Тиңме сиңа, тиң түгелме,
Сорап тормыйча гына.

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю - газап, беркемне дә
Сөймәмен дигән идем.

Яшерен генә сөеп йөрсәм,
Мәңге түзәрмен кебек.
Үзе белсә, сөймим дисә,
Егылып үләрмен кебек...

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю - газап, беркемне дә
Сөймәмен дигән идем.
```

# Перевод

### Гульшат Зейняшева / Раис Нагимов - Горю, пылаю

```
Горю, пылаю, горю, пылаю,
Огонь охватил мое платье.
Огонь не охватил мое платье,
Охватил молодое сердце.

Не буду гореть, говорила,
Не запылает, говорила.
Любить - пытка, никого
Не любить, говорила.

Ах, вздыхая, смотрю на звезды,
Смотрю в темные ночи.
Стала жертвой любви недостижимой
Равной единственной звезде.

Не буду гореть, говорила,
Не запылает, говорила;
Любить - пытка, никого
Не любить, говорила.

Приходит же эта любовь
Не задерживаясь.
Равен ли тебе, неравен ли,
Не спрашивая.

Не буду гореть, говорила,
Не запылает, говорила;
Любить - пытка, никого
Не любить, говорила.

Если тайно люблю,
Будто вечно терпеливая.
Если сама узнает, не люблю, говорит,
Будто изнываю до смерти...

Не буду гореть, говорила,
Не запылает, говорила;
Любить - пытка, никого
Не любить, говорила.
```
