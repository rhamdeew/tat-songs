# Оригинал

### Габдулла Тукай / Билгесез - Яңгыр (беренче вариант)

```
Былчыракта яңгыр артыннан балалар йөгрешә,
Кычкыралар: «Җиргә, - дип, - алтын төшә! алтын төшә!»

- Шауламагыз, и балалар! Без җыярбыз барчасын,
Тиз ул алтыннарны амбарга төярбез барчасын.

Тик җыярбыз юл буе шыгрым тулы йөкләр көе,
Бар келәтләр туп-тулы хуш исле орлыклар көе!
```

# Перевод

### Габдулла Тукай / Билгесез - Дождь (первый вариант)

```
По грязным лужам вслед за дождём ребята носятся,
Кричат: «На землю, мол, сыплется золото! золото!»

- Тише, дети! Мы соберём его всё без остатка,
В амбар то золото скоро навалим всё без остатка.

Лишь собирать - и весь путь грохочет, нагруженный доверху,
Все кладовые битком, благоухающие спелыми зёрнами.
```
