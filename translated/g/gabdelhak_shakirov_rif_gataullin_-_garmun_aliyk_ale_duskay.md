# Оригинал

### Габделхак Шакиров / Риф Гатауллин - Гармун алыйк әле, дускай

```
Гармун алыйк әле, дускай,
Гармун уйнатыйк әле.
Яшь чактагы көй-моңнарны
Кабат уятыйк әле.

Йөрәкләрне җилкендерсен
Тальянның көмеш чыңы.
Бар яшьләрнең күңеленә
Керсен халкыбыз моңы.

Бабайлар моңыннан башка
Туган җир ятим бит ул.
Көйсез-моңсыз үткән көннәр -
Яшәлмәгән көн бит ул.
```

# Перевод

### Габделхак Шакиров / Риф Гатауллин - Возьмём-ка гармонь, дружок

```
Возьмём-ка гармонь, дружок,
Давай сыграем снова.
Мелодии юных лет
Разбудим вновь и снова.

Пусть сердце взволнует
Тальяночки серебро,
Пусть в души всей молодёжи
Вольётся народа добро.

Без песни наших дедов
Родная земля - сирота.
Дни без напева и чувства -
Нежитые дни, пустота.
```
