# Оригинал

### Габделҗаббар Кандалый / Мәсгудә Шәмсетдинова - Ай-һай, Сәхипҗамалым

```
Карлардин, ләй, ап-ак,
Ай, чәчәкләрне ал да
Кулларыңа бәйләм ит.

Ай-һай, Сәхипҗамалым,
Син матурга гашыйк булып
Быел җәйне саргайдым.

Әгәр миндә күңлең була икән,
Берәр, берәр елга сабыр ит.

Ай-һай, Сәхипҗамалым,
Син матурга гашыйк булып
Быел җәйне саргайдым.
```
