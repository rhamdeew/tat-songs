# Оригинал

### Гарифҗан Мөхәммәтшин / Рөстәм Яхин - Ага сулар (беренче вариант)

```
Ага сулар, ага сулар,
Бер болгана, бер тына.
Агымсулар кебек күңелем
Кая шулай талпына?

Ашкыналар хыялларым
Аккан сулар артыннан.
Мәхәббәтем чыгар кебек
Барыр юлым чатыннан.

Ага сулар, ага сулар,
Дулкынлана-болгана.
Соңлаганнар ялгыз кала -
Ашык, күңелем, соңлама!

Язгы сулар аккан чакта
Бер моңсу, бер ямансу.
Аккан сулар - ярсу күңел,
Ярсу күңел - агымсу...
```

# Перевод

### Гарифжан Мухамметшин / Ростам Яхин - Текут воды (первый вариант)

```
Текут воды, текут воды,
То мутнеют, то стихают.
Как поток, моя душа -
Куда так рвется, не знает?

Рвутся к ним мои мечты,
Следом за бегущей влагой.
Будто встретится любовь
На развилке моей дороги.

Текут воды, текут воды,
Волны вздыбятся, смятенье.
Кто замедлил - в одиночестве.
Спеши, душа, не опоздай!

Вешние воды когда бегут -
То печаль, то боль немая.
Буйна вода - горячий дух,
Горячий дух - поток реки...
```
