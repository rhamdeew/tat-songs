# Оригинал

### Гөлшат Зәйнәшева / Рәүф Мортазин - Үзең аңла

```
Аңлатырга йөрәк хисләремне
Җитмәс кебек минем сүзләрем,
Үзең аңла инде, үзең аңла,
Нигә уйчан карый күзләрем.

Җырласам да, назлап, тибрәталмам
Күңелеңнең нечкә кылларын.
Үзең аңла инде, үзең аңла,
Нигә моңлы минем җырларым.

Сабыр итәм, өзелеп сагынсам да,
Очрашканда уттай янам мин.
Үзең аңла инде, үзең аңла, -
Нигә кайнар сулыш алам мин.
```

# Перевод

### Гөлшат Зәйнәшева / Рәүф Мортазин - Пойми сама

```
Объяснить сердечные чувства
Слов моих как будто не хватает,
Пойми сама уже, пойми сама,
Почему задумчивы глаза мои.

Пусть пою, лаская, - не сумею
Тронуть тонких струн твоей души.
Пойми сама уже, пойми сама,
Почему так грустны песни мои.

Я терплю, пусть рвусь от тоски по тебе,
При встрече я пылаю, словно огонь.
Пойми сама уже, пойми сама, -
Почему так жарко дышу я.
```
