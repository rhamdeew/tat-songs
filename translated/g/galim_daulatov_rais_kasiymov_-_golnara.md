# Оригинал

### Галим Дәүләтов / Рәис Касыймов - Гөлнара

```
Туган якка очкан кошлар
Күкрәк белән юл яра.
Яз моңнарын тыңласам да.
Уемда син, Гөлнара.

Туган якта, маяк төсле,
Яшь каеннар агара.
Сиңа кушылып сайрарга
Кошлар кайта, Гөлнара

Туган якка нур өстисең.
Яз гөлеңдәй, Гөлнара.
Гөлләр арасында үсеп,
Үзең гөл булмый кара?
```

# Перевод

### Галим Давлетов / Раис Касымов - Гульнара

```
Птицы, улетевшие на родину,
Грудью пробивают себе путь.
Если слушаю весенние мелодии -
В мыслях ты, Гульнара.

На родной стороне, словно маяк,
Белеют молодые берёзы.
Чтобы петь вместе с тобой,
Возвращаются птицы, Гульнара.

На родную сторону лучом воссияешь,
Подобно весеннему цвету, Гульнара.
Среди цветов росла ты -
Неужто сама цветком не стала?
```
