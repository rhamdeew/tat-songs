# Оригинал

### Габдулла Тукай / Заһид Хәбибуллин - Карга

```
Барча кошның да үзенчә сайравы бар, шигъре бар,
Карга, мескен, җырлый белми, кычкыра тик: "карр!" да "карр!"
```

# Перевод

### Габдулла Тукай / Заһид Хәбибуллин - Ворона

```
У каждой птицы своя песнь, свой стих и дар,
А бедной вороне не дано петь - лишь крик: «карр!» да «карр!»
```
