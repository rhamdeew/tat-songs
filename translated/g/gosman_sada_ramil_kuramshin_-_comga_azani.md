# Оригинал

### Госман Садә / Рамил Курамшин - Җомга азаны

```
Булды инде замананың
Юлдан язганы.
Казан иңләп оча таңда
Җомга азаны.

Канат итеп җәя Казан
Яшел әләмен.
Иман гына нурландыра
Күңел галәмен.

Замананың язмышларны
Кабат язганы;
Илгә-көнгә бәхет юрый
Җомга азаны.
```

# Перевод

### Госман Садә / Рамил Курамшин - Пятничный азан

```
Уж время оступилось, сбилось,
С пути сошло давно.
Над Казанью на рассвете
Пятничный азан вино.

Казань расправит, словно крылья,
Свой зеленый стяг святой.
Лишь иман озаряет
Мир души живой.

Вновь судьбы переписало
Время без конца;
Счастье людям, днем и миру
Вещает азан у чца.
```
