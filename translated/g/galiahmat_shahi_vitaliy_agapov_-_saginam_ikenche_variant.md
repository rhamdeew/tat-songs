# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Сагынам (икенче вариант)

```
Таш калада җилгә ялынам,
Алып кил, дип, чәчәк исләрен.
Мин аларны шундый сагынам,
Исләреннән таныйм төсләрен.

Янган йөрәгемә ялынам,
Янма, әйләнәсең күмергә.
Авылымны шундый сагынам,
Ничек кенә кайтып килергә.

Яңгыр болытына ялынам,
Сафландырып үт, дип, һаваны.
Чишмә челтерәвен сагынам,
Анда безнең яшьлек дәвамы.

Янган йөрәгемә ялынам,
Янма, әйләнәсең күмергә.
Авылымны шундый сагынам,
Ничек кенә кайтып килергә.

Һавадагы кошка ялынам,
Әйтче, кайда сөйгән ярым, дип,
Мин аны бит шундый сагынам,
Кай тарафка эзләп барыйм, дип.

Янган йөрәгемә ялынам,
Янма, әйләнәсең күмергә.
Авылымны шундый сагынам,
Ничек кенә кайтып килергә.
```

# Перевод

### Галиәхмәт Шаһи / Виталий Агапов - Скучаю (второй вариант)

```
В каменном городе ветру молюсь:
Принеси мне, прошу, аромат цветов.
Так по ним я тоскую - клянусь,
По их запаху узнаю их цвет.

Сердцу пылающему я молюсь:
Не гори - станешь пеплом, углем.
Так по родной деревне тоскую,
Как же вернуться туда с трудом.

Туче дождевой я молюсь:
Пройди, очищая, небесный простор.
Журчанье ключа я тоскую,
Там наша юность живет до сих пор.

Сердцу пылающему я молюсь:
Не гори - станешь пеплом, углем.
Так по родной деревне тоскую,
Как же вернуться туда с трудом.

Птице в небе я молюсь:
Скажи, где любимая моя?
Я по ней так тоскую - клянусь,
В какую мне сторону путь держать?

Сердцу пылающему я молюсь:
Не гори - станешь пеплом, углем.
Так по родной деревне тоскую,
Как же вернуться туда с трудом.
```
