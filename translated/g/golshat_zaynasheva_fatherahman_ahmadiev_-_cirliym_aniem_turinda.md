# Оригинал

### Гөлшат Зәйнәшева / Фәтхерахман Әхмәдиев - Җырлыйм әнием турында

```
Бер кыз бала җыр чыгарган,
Багышлап әнисенә.
Икәү бергә җырлыйк, диеп,
Өйрәткән энесенә:

«Кояш, кояш мәңге балкый,
Мәңге нур сибә;
Кояш гомере телим мин,
Әнием, сиңа!»

Җырны чишмәләр ишеткән,
Кошлар отып алганнар.
Сандугачлар һәм тургайлар
Матур көйгә салганнар:

«Кояш, кояш мәңге балкый,
Мәңге нур сибә;
Кояш гомере телим мин,
Әнием, сиңа!»

Кушыл син дә, җырлыйк бергә,
Яңа матур җыр тусын.
Җирдә әниләр, кояштай,
Озын гомерле булсын:

«Кояш, кояш мәңге балкый,
Мәңге нур сибә;
Кояш гомере телим мин,
Әнием, сиңа!»
```

# Перевод

### Гульшат Зайнашева / Фетерахман Ахмадиев - Пою я о матери

```
Девочка песню создала,
Посвятив её матери.
Давайте споём вдвоём, -
Братишку научила она:

«Солнце, солнце вечно сияет,
Вечно светит лучами;
Жизнь солнечная мне нужна,
Мама, только для тебя!»

Песню ручейки услыхали,
Птицы подхватили её.
Соловьи и жаворонки
Взели красивую мелодию:

«Солнце, солнце вечно сияет,
Вечно светит лучами;
Жизнь солнечная мне нужна,
Мама, только для тебя!»

Присоединяйся и ты, споём вместе,
Пусть новая рождается песня.
Пусть матери на земле, как солнце,
Долгожительными будут:

«Солнце, солнце вечно сияет,
Вечно светит лучами;
Жизнь солнечная мне нужна,
Мама, только для тебя!»
```
