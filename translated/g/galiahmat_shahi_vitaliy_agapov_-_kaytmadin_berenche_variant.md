# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Кайтмадың (беренче вариант)

```
Акты ярсып кар сулары,
Карап тордым акканын.
Кабат атты ак чәчәкләр,
Тик син генә кайтмадың.

Яңа яуган кар өстендә
Эзләреңне тапмадым.
Кояш булып баедың да
Таңнар булып атмадың.

Курасында җиләк пеште,
Өзми сиңа сакладым.
Бәгыремне сагыш кисте,
Юатырга кайтмадың.

Яңа яуган кар өстендә
Эзләреңне тапмадым.
Кояш булып баедың да
Таңнар булып атмадың.

Көзге ачы җилләр исте,
Җилгә каршы атладым.
Күңелдәге болытларны
Таратырга кайтмадың.

Яңа яуган кар өстендә
Эзләреңне тапмадым.
Кояш булып баедың да
Таңнар булып атмадың.
```

# Перевод

### Галиахмат Шахи / Виталий Агапов - Не вернулся (первый вариант)

```
Растаял бердән кар сулар,
Агышын күздә тутырдым.
Яңәртеп ачтыл ак чәчәкләр,
Син генә кайтмадың әле.

Төк алган кар өстендә
Изләреңне тапмадым.
Кояш булып янып чыктың,
Таңнар булып атмадың.

Көрәсендә җиләк пеште,
Сиңа өземи сакладым.
Бәхетемне сагынч җыртты,
Юатырга кайтмадың.

Төк алган кар өстендә
Изләреңне тапмадым.
Кояш булып янып чыктың,
Таңнар булып атмадың.

Көзге ачы җилләр ысты,
Каршы җилгә атладым.
Йөрәкдәге болытларны
Таратырга кайтмадың.

Төк алган кар өстендә
Изләреңне тапмадым.
Кояш булып янып чыктың,
Таңнар булып атмадың.
```
