# Оригинал

### Гөлшат Зәйнәшева / Заһир Исмәгыйлев - Сагынырмын Урал тау буйларын (Урал тавы)

```
Сагынырмын Урал тау буйларын,
Тын кичләрен, курай моңнарын.
Сагынырмын алсу таң атканда
Яңгыраган башкорт җырларын.

Буйларында Урал тауларының
Чәчәк атты яшьлек елларым.
Кая гына китеп яшәсәм дә,
Урал, сине мактап (сагынып) җырлармын.
```

# Перевод

### Гөлшат Зәйнәшева / Заһир Исмәгыйлев - Буду тосковать по приуральским склонам (Урал-тау)

```
Буду тосковать по склонам Урала,
По тихим вечерам, по звону курая.
Буду тосковать, когда заря алеет,
По башкирским песням, что звенят тогда.

На просторах Уральских гор
Расцвели мои годы юные.
Куда б ни ушёл и где б ни жил,
Урал, о тебе, с тоской, я буду петь.
```
