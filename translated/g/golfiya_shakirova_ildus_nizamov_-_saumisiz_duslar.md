# Оригинал

### Гөлфия Шакирова / Илдус Низамов - Саумысыз, дуслар!

```
Менә инде килеп җиттек,
Сәлам якын дусларга.
Чакырамын барыгызны
Матур җырлар тыңларга.

Саумысыз, дуслар, саумысыз,
Ничек соң хәлләрегез.
Күтәрелсен безнең белән
Сезнең күңелләрегез.

Рәхәтләнеп ял итегез,
Уйнашыйк та көлешик.
Ямьле итеп тормышларны,
Бергәләп җырлап яшик.

Саумысыз, дуслар, саумысыз,
Ничек соң хәлләрегез.
Күтәрелсен безнең белән
Сезнең күңелләрегез.

Җыр-бию, уен-көлкеләр
Менә сезнең хөрмәткә.
Бүгенге булган тамаша
Яллар булсын йөрәккә.

Саумысыз, дуслар, саумысыз,
Ничек соң хәлләрегез.
Күтәрелсен безнең белән
Сезнең күңелләрегез.
```

# Перевод

### Гөлфия Шакирова / Илдус Низамов - Здравствуйте, друзья!

```
Вот мы и пришли уже,
Всем родным друзьям - привет.
Приглашаю вас послушать
Красоту любимых песен.

Здравствуйте, друзья, здравствуйте,
Как сегодня ваши дела.
Пусть с нами поднимутся
Ваши светлые сердца.

Отдохните с наслаждением,
Поиграем и посмеёмся.
Пусть жизнь будет светлой - вместе
Песней дружно жить начнёмся.

Здравствуйте, друзья, здравствуйте,
Как сегодня ваши дела.
Пусть с нами поднимутся
Ваши светлые сердца.

Песни, танцы, игры, смех -
Всё сегодня для вас, друзья.
Пусть сегодняшний наш праздник
Станет отдыхом для сердца.

Здравствуйте, друзья, здравствуйте,
Как сегодня ваши дела.
Пусть с нами поднимутся
Ваши светлые сердца.
```
