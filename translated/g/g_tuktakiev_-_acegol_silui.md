# Оригинал

### Г. Туктакиев - Әҗегол сылуы

```
Әҗегол үстерә
Кызларның сылуын.
Тиңнәре бар микән
Сезгә чит кызларның?

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!

Матурлый йөзеңне
Әҗегол таңнары.
Күзләреңдә балкый
Йолдызлар нурлары.

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!

Җидегән чишмәдәй
Өфенең сулары.
Биг(е)рәк сөйкемле
Әҗегол кызлары.

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!
```

# Перевод

### Г. Туктакиев - Красавица Аджегол

```
Аджегол растит
Красу девичью.
Есть ли равные вам
Среди чужих девушек?

Красавица Аджегола,
Красавица Аджегола,
Разрываюсь от тоски,
Скучаю по твоему лицу!

Укрась твое лицо
Рассветы Аджегола.
В глазах твоих сияют
Звезд лучи.

Красавица Аджегола,
Красавица Аджегола,
Разрываюсь от тоски,
Скучаю по твоему лицу!

Как семьдесят источников
Воды Уфы.
Особенно милы
Девушки Аджегола.

Красавица Аджегола,
Красавица Аджегола,
Разрываюсь от тоски,
Скучаю по твоему лицу!
```
