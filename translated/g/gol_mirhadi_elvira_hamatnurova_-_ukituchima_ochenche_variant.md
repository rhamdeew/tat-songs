# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Укытучыма (өченче вариант)

```
Киләчәге зур илемнең
Синең кулыңда, -
Укытучым! Бүген җырлыйм
Синең турыда.

Иң ягымлы,сабыр, гадел
Укытучым - сез!
Күңелемне, уйларымны
Яктыртучы - сез!

Олы йөрәгеңне тиңлим
Чиксез далага, -
Акыллы сүзләр табасың
Һәрбер балага.

Иң ягымлы,сабыр, гадел
Укытучым - сез!
Күңелемне, уйларымны
Яктыртучы - сез!

Җитәкләп алып менәсең
Белем үренә,
Шуңа урының да синең
Йөрәк түрендә.

Иң ягымлы,сабыр, гадел
Укытучым - сез!
Күңелемне, уйларымны
Яктыртучы - сез!
```

# Перевод

### Гөл Мирһади / Эльвира Хаматнурова - Моему учителю (третий вариант)

```
Будущее моей великой страны
В твоих руках, -
Учитель мой! Сегодня я пою
О тебе.

Самый добрый, терпеливый, справедливый -
Учитель мой - вы!
Сердце, мысли, душу
Озаряете - вы!

Твое большое сердце я сравню
С бескрайней степью, -
Мудрые слова находишь
Для каждого ребенка.

Самый добрый, терпеливый, справедливый -
Учитель мой - вы!
Сердце, мысли, душу
Озаряете - вы!

За руку ведешь нас вверх
К вершинам знаний,
Потому и место ваше -
В самой глубине сердца.

Самый добрый, терпеливый, справедливый -
Учитель мой - вы!
Сердце, мысли, душу
Озаряете - вы!
```
