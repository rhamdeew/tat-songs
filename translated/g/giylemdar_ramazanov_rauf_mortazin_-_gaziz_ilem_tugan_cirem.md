# Оригинал

### giylemdar_ramazanov_rauf_mortazin_ - Газиз илем - Туган җирем

```
Бала чагым узган җирем,
Газиз илем - Туган җирем,
Туган җирем,
Син тормышым, җырым минем,
Мәхәббәтем, моңым минем,
моңым минем,
Сөям сине, нурлы җирем.

Синнән башка миңа ямь юк,
Ят суларда мындый тәм юк,
Мындый тәм юк.
Сине мактап туеп булмый,
Синнән читтә торып булмый,
торып булмый,
Синнән башка җырым тумый.
```

# Перевод

### Гыйлемдар Рамазанов / Рәүф Мортазин - Дорогая моя страна - Родная земля

```
Где прошло мое детство,
Дорогая страна - родная земля,
Родная земля,
Ты - жизнь моя, песня моя,
Любовь моя, моя мелодия,
моя мелодия,
Люблю тебя, сияющая земля.

Без тебя мне нет отрады,
В чужих водах нет такого вкуса,
Такого вкуса нет.
Тебя хвалить - не насытиться,
Вдали от тебя не устоять,
не устоять,
Без тебя моя песня не родится.
```
