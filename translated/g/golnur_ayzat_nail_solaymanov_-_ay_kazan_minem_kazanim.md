# Оригинал

### Гөлнур Айзәт / Наил Сөләйманов - Әй, Казан, минем Казаным

```
Әй, Казан, минем Казаным,
Татар иле йөрәге!
Синдә яшәүче халкымның
Аяз булсын көннәре.

Әй, Казан, минем Казаным,
Күңелгә якын җирем!
Синдә хисләрем канатлы,
Яңгырый дәртле җырым.

Казаным - бәхет каласы,
Бәхет нуры шәhәрдә!
Һәр балаңа насыйп булсын
Иман белән яшәргә.

Әй, Казан, минем Казаным,
Күңелгә якын җирем!
Синдә хисләрем канатлы,
Яңгырый дәртле җырым.

Тынычлык шәhәре - Казан,
Монда hәркем дус, туган.
Чал тарихтан кайтаваз син -
Мәшhүр Идел йортыннан.

Әй, Казан, минем Казаным,
Күңелгә якын җирем!
Синдә хисләрем канатлы,
Яңгырый дәртле җырым.
Яңгырый дәртле җырым...
```

# Перевод

### Гульнур Айзат / Наил Сулейманов - О, Казань, моя Казань

```
О, Казань, моя Казань,
Сердце татарского края!
Пусть будут ясными дни
Живущего в тебе народа.

О, Казань, моя Казань,
Близкая сердцу земля!
В тебе чувства мои крылаты,
Звучит страстная песня моя.

Казань моя - город счастья,
Свет счастья в городе!
Пусть каждому ребёнку дано будет
Жить с верой.

О, Казань, моя Казань,
Близкая сердцу земля!
В тебе чувства мои крылаты,
Звучит страстная песня моя.

Город мира - Казань,
Здесь каждый друг, родной.
Отголосок седой истории ты -
Из знаменитого Идель-дома.

О, Казань, моя Казань,
Близкая сердцу земля!
В тебе чувства мои крылаты,
Звучит страстная песня моя.
Звучит страстная песня моя...
```
