# Оригинал

### Гөлшат Зәйнәшева / Айрат Кобагышев - Кышкы моңнар (икенче вариант)

```
Ап-ак карлар ява, карлар ява,
Җирдә кышны сагынып көтә микән?
Сагынып та көтәр кешең булса,
Бу гомерләр матур үтә икән.

Ап-ак карлар ява, карлар ява,
Уйландыра кышкы зәңгәр кичләр,
Бер йомырлам бозга әйләнерме,
Син булмасаң, инде йөрәк нишләр?

Ап-ак карлар ява, карлар ява,
Карлар түгел, күктән моң коела
Яннарыңда сөйгән кешең булса,
Салкын кыш та җылы, җылы тоела.
```

# Перевод

### Гөлшат Зәйнәшева / Айрат Кобагышев - Зимние мелодии (второй вариант)

```
Белым-белым снегом сыплет, сыплет,
Будто землю зима в ожиданьи держит.
Если есть, кто ждёт тебя, скучая,
Эта жизнь и вправду хорошеет.

Белым-белым снегом сыплет, сыплет,
Задумчивы зимних вечеров лазури,
Не превратится ль в комочек льда
Сердце, если ты не рядом будешь?

Белым-белым снегом сыплет, сыплет,
Не снег - а с неба льётся нежный зов.
Если рядом с тобой любимый человек,
Даже стужа зимняя теплом согрета.
```
