# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Аллага шөкер! (икенче вариант)

```
Биссмиладан көнне башлап
Рәхмат укыйм ходайга
Рәхмат җирләр күкләргә
нурлы кояшым айга

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк

Әнә безне шатландырып
үсеп килә балалар.
Изге теләкләрне теләп.
җырлансын матур җырлар

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк

Гөнаһлардан бәлаләрдан
Фәрештәләр сакласын
Җитәкләшеп атлый торган
юлларыбыз уң булсын

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк
```

# Перевод

### Гөлфия Шакирова / Лилия Вахитова - Хвала Аллаху! (второй вариант)

```
С бисмилляхи день начинаю,
Благодарю я Бога.
Спасибо земле и небесам,
Светлому солнцу, луне.

С хвалою Аллаху живём,
Скажем: хвала Аллаху.
ИншаАллах, будем радоваться,
ИншаАллах, утешимся.

Вот, нас радуя, растут дети.
С добрыми пожеланиями
Пусть звучат красивые песни.

С хвалою Аллаху живём,
Скажем: хвала Аллаху.
ИншаАллах, будем радоваться,
ИншаАллах, утешимся.

От грехов и от бед
Пусть ангелы нас хранят.
Пусть благим будет наш путь,
По которому идём рука об руку.

С хвалою Аллаху живём,
Скажем: хвала Аллаху.
ИншаАллах, будем радоваться,
ИншаАллах, утешимся.
```
