# Оригинал

### Гамир Насрый / Заһид Хәбибуллин - Сайра син, тургай

```
Гөрләп акты кар сулары,
Ямьләнде тугай.
Ак болытлардан югары
Сайрады тургай.

Сайра син, тургай,
Шауласын тугай.
Шаулап үсәр
Кырларда бодай.

Шаулады иртәнге җилдә
Назлы ак каен;
Җырладылар игенчеләр
Мактап яз аен.

Сайра син, тургай,
Шауласын тугай.
Шаулап үсәр
Кырларда бодай.

Җырладылар яшь егетләр,
Кушылды кызлар.
Күкрәкләрендә балкыды
Алтын йолдызлар.

Сайра син, тургай,
Шауласын тугай.
Шаулап үсәр
Кырларда бодай.

Балкыды язгы кояшның
Якты нурлары;
Алтын диңгезгә әйләнер
Колхоз кырлары.

Сайра син, тургай,
Шауласын тугай.
Шаулап үсәр
Кырларда бодай.
```

# Перевод

### Гамир Насрый / Заһид Хәбибуллин - Пой, жаворонок

```
Бурно побежали снега,
Расцвёл луг степной.
Выше белых облаков
Пел жаворонок звонкий.

Пой, жаворонок,
Пусть шумит луг.
Шумно взойдёт
Пшеница в полях.

В утреннем ветре шумела
Ласковая белая берёза;
Пели хлеборобы,
Прославляя весенний месяц.

Пой, жаворонок,
Пусть шумит луг.
Шумно взойдёт
Пшеница в полях.

Пели молодые парни,
Девушки им вторили.
На их груди сияли
Золотые звёзды.

Пой, жаворонок,
Пусть шумит луг.
Шумно взойдёт
Пшеница в полях.

Засияли яркие лучи
Весеннего солнца;
В золотое море обратятся
Колхозные поля.

Пой, жаворонок,
Пусть шумит луг.
Шумно взойдёт
Пшеница в полях.
```
