# Оригинал

### Гөлшат Зәйнәшева / Ренат Еникеев - Ардуан батыр

```
Батырларның гомере мәңге сүнмәс,
Ил күгендә йолдыз булып балкыр.
Шундый якты йолдызларның берсе -
Ардуан батыр, Ардуан батыр.

Урал итәгендә завод төзеп,
Ил шаулаткан даны һаман артыр.
Нигезләргә бәхет, ташын тезгән
Ардуан батыр, Ардуан батыр.

Халык бәхете өчен көрәшкәннең
Узган юлларында эзе матур.
Гади эшче исемен бөек йөрткән
Ардуан батыр, Ардуан батыр.

Лачыннарга тиңдәш үз улларын
Онытмый халкым, югалтмый хәтер.
Безнен белән ил төзешә бергә
Ардуан батыр, Ардуан батыр!
```

# Перевод

### Гөлшат Зайнашева / Ренат Еникеев - Ардуан батыр

```
Жизнь богатырей не гаснет вечно,
На небе Родины звездой сияет.
И среди таких звезд светлых -
Ардуан батыр, Ардуан батыр.

У подножья Урала завод подняв,
Громкой славой страна озарена.
Счастье к основаниям, камень к камню
Клал Ардуан батыр, Ардуан батыр.

Кто за счастье народа сражался,
В пройденных тропах след прекрасен.
Имя простого рабочего возвысил
Ардуан батыр, Ардуан батыр.

Соколиных своих сыновей
Народ не забудет, не теряет память.
Вместе с нами страну он строит -
Ардуан батыр, Ардуан батыр!
```
