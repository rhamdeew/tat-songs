# Оригинал

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Туган җиремә (беренче вариант)

```
Тугай ярып аккан чишмәләр кебек,
Саф хисләрем җырыма талпына.
Җырым кайнар чишмә булса,
Татлы суын үзем эчми,
Бирер идем хезмәт сөйгән халкыма!

Һәрбер шатлык кояш нурлары булып,
Үрли минем күңелем түренә.
Йөрәк җырларым да шулай
Кояш кебек якты нурлар
Сипсен иде туып-үскән җиремә!
```

# Перевод

### Гульшат Зайнашева / Зиннур Гыйбадуллин - Родной земле моей (первый вариант)

```
Словно родники, сквозь рощи пробиваясь,
Чистые чувства в песню мою льются.
Если б песня стала родником горячим,
Сладкую воду сам бы не испил я,
Отдал бы её народу, любящему труд!

Каждая радость, солнечным лучом сверкая,
В глубине души моей взлетает.
Пусть и песни сердца моего
Светлым солнцем ясных дней
Озаряют край, где вырос я и где рождён!
```
