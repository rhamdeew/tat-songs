# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Көзге җыр (беренче вариант)

```
Мин көзге сукмактан
Мәктәпкә барам
Иелеп кулыма
Яфраклар алам.
Кайчандыр ямь - яшел
Үскән яфраклар
Кабызды учыма
Кызыл учаклар

Моңсу көз кояшы
Нурларын суза.
Тезелеп көньякка
Кошларым оча
Җырларын сагынып
Сайрар кошларның
Узарбыз сукмагын
Салкын кышларның

Көздән соң, кыштан соң
Килер яз ае.
Мин үсәм мизгелләр
Алышкан саен.
Ашыгып мәктәпкә
Атлыйм юлымнан
Яфраклар бәйләмен
Тотоп кулыма.
```

# Перевод

### Гөл Мирһади / Эльвира Хаматнурова - Осенняя песня (первый вариант)

```
Я по осенней тропке
Иду в свою школу,
Наклонюсь - в ладони
Листья соберу.
Те листья, что когда-то
Зеленью сияли,
Разожгли в моих ладонях
Алые костры.

Грустное осеннее солнце
Тянет свои лучи.
В ряд к югу улетают
Мои птицы.
С тоской по их напевам -
Певчих птиц -
Мы пройдем тропою
Холодных зим.

После осени, после зимы
Придет весенний месяц.
Я расту, когда мгновения
Друг друга сменят.
Поспешно в школу
Шагаю по пути,
Держа в руках
Связку листьев.
```
