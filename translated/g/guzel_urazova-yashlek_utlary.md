# Оригинал

### Гүзәл Уразова - Яшьлек утлары

```
Гомерлэрдэ эле язмы, козме?
Йорэклэрдэ -- яшьлек утлары!..
Эйткэн сузем эллэ хак тугелме?
Яшэр кочне бирэ ул бары!

Яшьлектэге учакларнын
Сундермэек утларын!
Онытылмас ул чакларнын
Кадерле минутлары...

Исбат итэм диеп уз- уземне
Яхшылыкка кунел омтыла.
Тоя белеп гузэл мизгеллэрне,
Йорэклэрдэ яна жыр туа.

Яшьлектэге учакларнын
Сундермэек утларын!
Онытылмас ул чакларнын
Кадерле минутлары...

Тормыштагы кыен мизгеллэрне
Житкермэскэ кирэк упкынга...
Йорэк саклий хаман жинэр кочне,
Кыенлыклар тагын он(ы)тыла.

Яшьлектэге учакларнын
Сундермэек утларын!
Онытылмас ул чакларнын
Кадерле минутлары...
```

# Перевод

### Гүзәл Уразова - Огни молодости

```
В душах ведь есть весна и осень?
В сердцах - огни молодости!..
Разве сказанное мною неправда?
Живительную силу даёт она!

Молодости очагов
Не погасим огни!
Незабываемых тех времён
Дорогие минуты...

Доказать себя стремясь,
К добру душа тянется.
Вкушая прекрасные мгновения,
В сердцах рождается новая песня.

Молодости очагов
Не погасим огни!
Незабываемых тех времён
Дорогие минуты...

Жизни трудные моменты
Не должны довести до обиды...
Сердце хранит всё побеждающую силу,
Трудности снова забываются.

Молодости очагов
Не погасим огни!
Незабываемых тех времён
Дорогие минуты...
```
