# Оригинал

### Гөлия Шәйдуллина / Фәнис Сибгатуллин - Сагынамын каеннарны

```
Каеннарга мин сөялдем,
Кайгыларны сөйләдем.
Синнән башка бу дөньяда
Беркемне дә сөймәдем.

Авылымның каеннарын
Кайтып күрәсем килә.
Синең белән тормыш юлын
Бергә үтәсем килә.

Каеннарга мин сөялдем,
Акрын җилләр искәндә.
Төне буе йоклый алмыйм
Син исемә төшкәндә.

Авылымның каеннарын
Кайтып күрәсем килә.
Синең белән тормыш юлын
Бергә үтәсем килә.

Каеннарга мин сөялдем,
Каен аңлар хисемне.
Йолдызлардан эзләп табам
Синең матур исмеңне.

Авылымның каеннарын
Кайтып күрәсем килә.
Синең белән тормыш юлын
Бергә үтәсем килә.
```

# Перевод

### Гөлия Шәйдуллина / Фәнис Сибгатуллин - Тоскую по березам

```
Березам я приклонилась,
Им печали излила.
В этом мире, кроме милый,
Никого не полюбила.

Березы моей деревни
Вновь увидеть я мечтаю.
С тобой рядом по дороге
Жизнь пройти вдвоём желаю.

Березам я приклонилась,
Ветер тихий шелестит.
Ночью сна уже не вижу -
Вспомню имя - сердце стынет.

Березы моей деревни
Вновь увидеть я мечтаю.
С тобой рядом по дороге
Жизнь пройти вдвоём желаю.

Березам я приклонилась,
Поймут чувства белоствольные.
Среди звёзд ищу, как искру,
Твоё имя дорогое.

Березы моей деревни
Вновь увидеть я мечтаю.
С тобой рядом по дороге
Жизнь пройти вдвоём желаю.
```
