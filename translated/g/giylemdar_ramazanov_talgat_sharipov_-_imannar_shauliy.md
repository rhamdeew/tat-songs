# Оригинал

### Гыйлемдар Рамазанов / Тәлгать Шәрипов - Имәннәр шаулый

```
Тормыш кочагында,
Икәү яшь чагында,
Бер-беребезне өзелеп сөйгәндә.

Алтын көз көнендә,
Басу киңлегендә
Утырттык без япь-яшь имәннәр.

Безнең гомер белән
Ярышып үсә имән,
Ямь-яшеллек каплый җирләрне.

Без утырткан өчен
Биреп яшьлек көчен
Җиңәр алар хәтәр җилләрне.

Ярсу еллар үтәр,
Якты язлар җитәр,
Шул яланга икәү килербез.

Гомер матур үтә,
Урман шаулап үсә
Яшьлек дәвам итә диярбез.
```

# Перевод

### Гыйлемдар Рамазанов / Тәлгать Шәрипов - Дубы шумят

```
В объятьях жизни,
Вдвоём в пору юности,
Когда без памяти любили друг друга.

В золотую осень,
На просторе поля
Мы посадили совсем юные дубы.

С нашей судьбой
Дуб состязаясь растёт,
Зелёный убор ложится на землю.

Мы посадили -
Отдав силы юности,
Им устоять против грозных ветров.

Годы бурные пройдут,
Светлые весны придут,
На ту поляну вдвоём мы вернёмся.

Жизнь красиво идёт,
Лес шумливо растёт -
Мы скажем: юность продолжается.
```
