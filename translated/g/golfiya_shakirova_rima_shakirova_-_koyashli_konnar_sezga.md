# Оригинал

### Гөлфия Шакирова / Рима Шакирова - Кояшлы көннәр сезгә

```
Сезнең хакта уйларым,
Сәлам сезгә, дусларым.
Күтәренке кәеф теләп
Яңгыратам җырларым.

Кояшлы көннәр сезгә,
Рәхәтлек - күңелегезгә.
Сәламәтлек, муллык, байлык,
Бәрәкәт өегезгә.

Күңелләр торсын җырлап,
Күңелле көйләр сорап.
Телим сезгә теләкләрне
Бәхет-шатлыкка юрап.

Кояшлы көннәр сезгә,
Рәхәтлек - күңелегезгә.
Сәламәтлек, муллык, байлык,
Бәрәкәт өегезгә.

Ачык булсын юлыгыз,
Якты булсын уегыз.
Тулып торсын табыннарда
Ризык-нигъмәт, сыегыз.

Кояшлы көннәр сезгә,
Рәхәтлек - күңелегезгә.
Сәламәтлек, муллык, байлык,
Бәрәкәт өегезгә.
```

# Перевод

### Гульфия Шакирова / Рима Шакирова - Солнечные дни для вас

```
Думаю о вас,
Привет вам, друзья мои.
Со светлой радостью
Возвожу я песни.

Солнечные дни для вас,
Покой - в души ваши.
Здоровье, изобилие, богатство,
Благословение - в дом ваш.

Пусть сердца поют,
Ищут сердечные напевы.
Желаю вам пожеланий
На пути к счастью и радости.

Солнечные дни для вас,
Покой - в души ваши.
Здоровье, изобилие, богатство,
Благословение - в дом ваш.

Пусть будет путь ваш открытым,
Светлым - ваша жизнь.
Пусть полноводною рекою
Течёт щедрость и благость.

Солнечные дни для вас,
Покой - в души ваши.
Здоровье, изобилие, богатство,
Благословение - в дом ваш.
```
