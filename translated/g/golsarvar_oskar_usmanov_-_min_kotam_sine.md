# Оригинал

### Гөлсәрвәр / Оскар Усманов - Мин көтәм сине

```
Син мине белмисең әлегә,
Мин инде күптән беләм.
Гашыйк итеп кердең күңелгә
Беренче күргән көннән,
Чыкмыйсың йөрәгемнән.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.

Хыялым сиздермим һич кемгә,
Күңелдә өмет кенә.
Төштә дә,өндә дә син генә
Юл күрсәт бәхетемә,
Хыялым йөрәгемдә.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.

Әгәр дә карасаң күземә,
Танырсың мине,беләм.
Китәрмен җирнең дә читенә,
Китәрмен синең белән,
Чакырсаң үзең белән.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.
```

# Перевод

### Гюльсарвар / Оскар Усманов - Я тебя жду

```
Ты меня ещё не знаешь,
А я уже давно знаю.
Влюблённым вошла в сердце
С первого дня, как увидела,
Не выходишь из моего сердца.

Через мои мечты,
Через мои мысли
Познала бы тебя, говорю,
Ты сам меня.
Через мои песни
Сердце моё стучит
Пришла бы ты, говорю,
Я тебя жду.

Мои мечты никому не выдают,
В душе только надежда.
И во сне, и наяву только ты
Указываешь путь к моему счастью,
Моя мечта в моём сердце.

Через мои мечты,
Через мои мысли
Познала бы тебя, говорю,
Ты сам меня.
Через мои песни
Сердце моё стучит
Пришла бы ты, говорю,
Я тебя жду.

Если посмотришь в мои глаза,
Узнаешь меня, я знаю.
Уйду и на край земли,
Уйду с тобой,
Если позовёшь меня с собой.

Через мои мечты,
Через мои мысли
Познала бы тебя, говорю,
Ты сам меня.
Через мои песни
Сердце моё стучит
Пришла бы ты, говорю,
Я тебя жду.
```
