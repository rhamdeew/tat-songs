# Оригинал

### Гөлфия Шакирова / Оскар Усманов - Язгы җырым син

```
Сиңа әйтер
Матур сүзләремне
Энҗеләрдән тездем, кара әле.
Күңел назын
Салып шул сүзләргә,
Сөйлиселәр минем бар әле....

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң....

Ашыгыплар
Миңа килгән чакта,
Адымнарың килә тыңлыйсым.
Синең эзне
Меңнәр арасыннан
Килә иркәм минем таныйсым...

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң....

Сагыш күлен
Бигрәк тирән диләр,
Төшә күрмә мине калдырып.
Карап синең
Килер юлларыңа,
Көтәм, бәгърем, күзем талдырып...

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң....
```

# Перевод

### Гөлфия Шакирова / Оскар Усманов - Ты моя весенняя песня

```
Для тебя
Красивые слова
Нанизала, как жемчуг, посмотри.
Вложив нежность
В эти слова,
Мне еще так хочется говорить....

Как у соловья
Есть своя песня,
Так ты мой напев, моя весенняя песня.
Когда зовешь
Меня "душа" искренне,
Это само по себе равносильно счастью....

Когда спешишь
Ко мне на встречу,
Мне хочется слушать твои шаги.
Твои следы
Среди тысяч троп
Хочу узнать, родной мой...

Как у соловья
Есть своя песня,
Так ты мой напев, моя весенняя песня.
Когда зовешь
Меня "душа" искренне,
Это само по себе равносильно счастью....

Говорят, озера
Печали глубоки,
Не дай мне упасть туда, оставив меня.
Смотря в стороны
Твоих дорог грядущих,
Жду тебя, милый, глаза устают...

Как у соловья
Есть своя песня,
Так ты мой напев, моя весенняя песня.
Когда зовешь
Меня "душа" искренне,
Это само по себе равносильно счастью....
```
