# Оригинал

### Гәрәй Рәхим / Фасил Әхмәтов - Комсомол (беренче вариант)

```
Хаклык өчен көрәш барган далаларда
Томырылып чапкан атларда
Син идең ул, комсомол,
Без идек ул, комсомол,
Баручы Ватанны сакларга.

Син бүген дә якты илңең горурлыгы;
Сине һәркем батыр ул дияр.
Син бүген дә, комсомол,
Без бүген дә, комсомол,
Иң алгы сафтагы эшсөяр.

Ленин биргән данлы, якты исемеңне
Һәрвакыт син горур йөртәсең.
Син бүген дә, комсомол,
Без бүген дә, комсомол,
Саклаучы тыныч ил иртәсен!
```

# Перевод

### Гәрәй Рәхим / Фасил Әхмәтов - Комсомол (первый вариант)

```
В степях, где шла борьба за правду,
На мчавших во весь дух конях,
Ты был тогда, комсомол,
Мы были тогда, комсомол,
Шедшие Родину стеречь.

И ныне ты - гордость светлой страны;
Тебя каждый назовет героем.
Ты ныне, комсомол,
Мы ныне, комсомол,
В переднем ряду трудолюбивый.

Данное Лениным славное, светлое имя
Ты всегда несешь с гордостью.
Ты ныне, комсомол,
Мы ныне, комсомол,
Хранители мирного утра страны!
```
