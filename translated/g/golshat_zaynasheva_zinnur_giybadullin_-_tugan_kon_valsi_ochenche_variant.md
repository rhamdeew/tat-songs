# Оригинал

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Туган көн вальсы (өченче вариант)

```
Алтын кояш сибә нурларын.
Кояш балкыган язгы иртәдәй,
Якты булсын, якты булсын
Тормыш юлларың.

Яшьлегебез матур булганга,
Матур җырлар килә күңелгә.
Мактап яшьлекнең матур таңнарын
Җырла, дускай, җырла, дускай,
Туган көнеңдә.

Шаулап-гөрләп сүнмәс дәрт белән
Шат җыр булып яшә син җирдә.
Туган көнеңдә тагын бер теләк:
Мәңге яшь бул, мәңге яшь бул,
Картайма бер дә.
```

# Перевод

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Вальс ко дню рождения (третий вариант)

```
Золотое солнце сеет свет.
Как весенним утром солнце сияет,
Пусть светлыми, пусть светлыми
Будут пути твоей жизни.

Молодость наша была прекрасной -
Оттого и песни в сердце прекрасные.
Восхваляя молодости светлые зори,
Пой, дружок, пой, дружок,
В день рождения.

Шумно, звонко, с неугасимой страстью
Живи на земле, словно радостная песня.
И в твой день рождения ещё одно пожелание:
Вечно будь молод, вечно будь молод,
Не старей никогда.
```
