# Оригинал

### Галиәхмәт Шаһи / Вәкил Мурзин - Матур үтсен кичегез

```
Ага сулар, исә җилләр,
Үтә көннәр тезелеп.
Сезнең белән очрашуны
Көтеп алдым өзелеп.

Яңа (матур) җырлар тыңлый-тыңлый
Матур үтсен кичегез.
Яныгыз да булсын һәр чак
Иң кадерле кешегез.

Без җырлаган җырлар сезнең
Күңелләргә хуш килсен.
Алда бәхет бар әле дип
Юраганнар юш килсен.

Яңа җырлар тыңлый-тыңлый
Матур үтсен кичегез.
Яныгыз да булсын һәр чак
Иң кадерле кешегез.

Дулкын-дулкын агыламын
Чиртә күңел кылларны.
Җырлый-җырлый үтсәк иде (үтик)
Гомернең ак юлларын.

Яңа җырлар тыңлый-тыңлый
Матур үтсен кичегез.
Яныгыз да булсын һәр чак
Иң кадерле кешегез.
```

# Перевод

### Галиәхмәт Шаһи / Вәкил Мурзин - Пусть ваш вечер пройдет красиво

```
Текут воды, веют ветры,
Дни проходят чередой.
Встречи с вами я дождался
Сердцем, с горькою бедой.

Новые (красивые) песни слушая,
Пусть красивым будет вечер ваш.
Пусть всегда, в любой минуте
Рядом будет самый дорогой человек.

Наши песни, что мы пели,
Пусть душе придутся вам.
Пусть сбудутся предсказанья:
Впереди еще есть счастье там.

Новые песни слушая,
Пусть красивым будет вечер ваш.
Пусть всегда, в любой минуте
Рядом будет самый дорогой человек.

Волной за волной я льюсь,
Трону струны сердца вновь.
Петь бы, петь бы да пройти нам
Белые дороги жизни вдаль.

Новые песни слушая,
Пусть красивым будет вечер ваш.
Пусть всегда, в любой минуте
Рядом будет самый дорогой человек.
```
