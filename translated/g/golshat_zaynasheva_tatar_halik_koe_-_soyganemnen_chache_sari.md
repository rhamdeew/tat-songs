# Оригинал

### Гөлшат Зәйнәшева / Татар халык көе - Сөйгәнемнең чәче сары

```
Сөйгәнемнең чәче сары,
Сары майлап тарыйдыр.
Сандугач та сары, бәлки
Шуңа моңлы сайрыйдыр.

Сөйгәнемнең чәче сары,
Сары тасмалап үрә.
Сары чәчәгем, дип кенә
Яратам шуңа күрә.

Сөйгәнемнең чәче сары,
Сары - буямаганга.
Аңар атап җыр чыгардым
Күреп туялмаганга.
```

# Перевод

### Гөлшат Зәйнәшева / Татар халык көе - У любимого волосы русые

```
У любимого волосы русые,
Маслом топлёным прядь пригладит.
И соловей, быть может, желтый -
Оттого так жалобно ладит.

У любимого волосы русые,
Жёлтой лентой заплетает.
«Цветочек мой золотой», -
Оттого-то сердце пылает.

У любимого волосы русые,
Не окрашены - так сияют.
Для него я песню сложила -
Насмотреться не перестаю.
```
