# Оригинал

### Гөлфия Юнысова / Нурия Абдуллина - Дим буйлары

```
Туган ягым Дим буйлары,
Үзәнеңдә әрәмә.
Әрәмәгә барсам, әнкәй,
Озакласам, әрләмә.

Әрәмәдә пешкән диләр
Шомырт, балан, бөрлегән.
Яшь кояшлы булгангадыр,
Җимеш иртә өлгергән.

Чакыра төсле үзенә
Куак артыннан куак.
Әйтерсең дә, бала чагым
Яшенеп тора сымак.
```

# Перевод

### Гульфия Юнусова / Нурия Абдуллина - Берега Дёмы

```
Родной мой край - берега Дёмы,
В твоём овраге - сады.
В сад схожу, матушка,
Задержусь - не reproach me.

В саду созывали сердца
Черёмуха, рябина, малина.
Видно, солнцем молодым согрета -
Ягоды рано поспели.

Каждый кусточек манит к себе
Один за другим в глубину.
Будто детство моё
Спряталось там, живёт.
```
