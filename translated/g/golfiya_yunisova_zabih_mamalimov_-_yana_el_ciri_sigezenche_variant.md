# Оригинал

### Гөлфия Юнысова / Зәбих Мәмәлимов - Яңа ел җыры (сигезенче вариант)

```
Үтеп китте бер ел тагы,
Тарих булып кына калды.
Һәр сәгате күңел түрендә.
Кояш кебек балкып-көлеп,
Өметләргә яшәү биреп,
Өр-яңа ел туа җиремдә.

Өр-яңа ел туа җиргә,
Ниләр бүләк итәр безгә? -
Бөтәсе дә әле сер генә.
Сиздерми гомер үткәнен,
Кабатламый ул бүтәнен, -
Һәр Яңа ел була бер генә.

Тирә-яҡта сафлык, назлар,
Өерелеп очкан карлар
Ак теләктән генә торалар:
Килсен кышлар, килсен җәйләр,
Күкләр булсын һәрчак зәңгәр,
Имин генә булсын дөньялар.

Шат йолдызлар нурын сипсен,
Алда бары бәхет көтсен,
Дулкынлансын иген кырлары.
Бар Җир шарын урап үтсен,
Һәр йөрәккә барып җитсен
Без җырлаган дуслык җырлары.
```

# Перевод

### Гөлфия Юнысова / Зәбих Мәмәлимов - Песня Нового года (восьмой вариант)

```
Прошёл ещё один год,
Став историей уже.
Каждый час в душе моей.
Сияя, как солнце в зной,
Подарив надежде жизнь,
Новый год стучится в дом.

Новый год стучится в дом,
Что же подарит он нам? -
Всё ещё тайна одна.
Не замечая жизни бег,
Не повтоит следом в след, -
Новый год приходит раз.

Кругом чистота, мороз,
Извиваются в полёте снежинки,
Создавая белый свет пожеланий:
Пусть приходят зимы, приходят лета,
Небеса пусть будут ясны,
Миры пусть будут безопасны.

Сияние счастливых звёзд да льётся,
Будущее счастьем встречай,
Пусть колышутся поля в зернах.
Пусть обойдёт по всему миру,
Пусть дойдёт до каждого сердца
Спетая нами песня дружбы.
```
