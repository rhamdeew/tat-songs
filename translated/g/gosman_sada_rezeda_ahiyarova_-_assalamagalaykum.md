# Оригинал

### Госман Садә / Резеда Ахиярова - Әссәламәгаләйкүм!

```
Таң ата да кояш чыга,
Кояшны сәламли көн.
- Саумы, Кояш, сүнмәс учак!
Әссәламәгаләйкүм!

Тел ачкычы, иман биргән
Сөекле туган телем!
Үз телемдә сиңа сәлам,
Әссәламәгаләйкүм!

Әй игенче, күкрәп үссен
Кырларда синең иген.
Икмәк тотып сәлам юллыйм,
Әссәламәгаләйкүм!

Ачыйк бәхет ишекләрен,
Бәхетле булсын һәркем.
Әй кардәшләр, милләттәшләр,
Әссәламәгаләйкүм!

Таң ата да кояш чыга,
Кояшлы бездә һәр көн.
Кояш белән сәлам юллыйм,
Әссәламәгаләйкүм!
```

# Перевод

### Госман Садә / Резеда Ахиярова - Әссәламәгаләйкүм!

```
Көн бетә, һәм кояш чыга,
Кояшны сәламли көн үзләре.
- Саумы, Кояш, сүнмәс өчен!
Әссәламәгаләйкүм!

Тел ачкычы, иман керткән
Сөюк туган телебез!
Үз телебездә сиңа сәлам,
Әссәламәгаләйкүм!

Әй, ипик, үсәргә булсын
Кырларда синең бөк.
Ипек тотып сәлам юллыйм,
Әссәламәгаләйкүм!

Ачыгыз бәхет ишекләрен,
Бәхетле булсын һәркем.
Әй, кардәшләр, милләттәшләр,
Әссәламәгаләйкүм!

Көн бетә, һәм кояш чыга,
Кояшлы бездә һәр көн.
Кояш белән сәлам юллыйм,
Әссәламәгаләйкүм!
```
