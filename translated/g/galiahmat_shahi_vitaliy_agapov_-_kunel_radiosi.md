# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Күңел радиосы

```
Сез сораган җыр яңгырый
«Күңел» дулкыннарында.
Ут өстәгез, янсын йөрәк
Сөю ялкыннарында.

«Күңел» радиосы сезгә
Күңелле яллар тели.
Эфир аша сәлам юллый,
Я җырлый ул, я сөйли.

«Күңел»нең хезмәткәрләре -
Егетләре, кызлары.
Сүзгә оста, тәмле телле
Һәм ягымлы үзләре.

«Күңел» радиосы сезгә
Күңелле яллар тели.
Эфир аша сәлам юллый,
Я җырлый ул, я сөйли.

«Күңел» дулкыннары сезнең
Чиртә күңел кылларын.
Җырдан-моңнан аерылмый
Үтик гомер юлларын.

«Күңел» радиосы сезгә
Күңелле яллар тели.
Эфир аша сәлам юллый,
Я җырлый ул, я сөйли.
```

# Перевод

### Галиәхмәт Шаһи / Виталий Агапов - Радио «Күңел»

```
Звучит желанная вам песня
На волнах «Күңел» сейчас.
Прибавьте огня - пусть сердце
Горит в пламени любви для нас.

Радио «Күңел» вам желает
Весёлых, светлых выходных.
По эфиру шлёт приветы -
То поёт оно, то говорит.

Служители «Күңел» - ребята,
Парни да девушки все.
Мастера слова, сладкоречивы
И сами по нраву вам все.

Радио «Күңел» вам желает
Весёлых, светлых выходных.
По эфиру шлёт приветы -
То поёт оно, то говорит.

Волны «Күңел» ваши души
Тронут, струны зазвенят.
Не разлучаясь с песней, с музыкой
Пройдём дороги жизни в лад.

Радио «Күңел» вам желает
Весёлых, светлых выходных.
По эфиру шлёт приветы -
То поёт оно, то говорит.
```
