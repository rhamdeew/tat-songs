# Оригинал

### Гөлүзә Арсланова - Шылтырат, миӊа, син, шылтырат

```
Шылтырат, миӊа, син, шылтырат,
Минем хәлкәйләремне сорап,
Ярсу күӊелкәемне юат,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.

Шылтырат, миӊа, син, шылтырат,
Тулган күӊелкәеӊне бушат,
Хәлкәйләреӊне генә сөйләп,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.

Озын-озын юлларны урап,
Син килерсеӊ хәлемне сорап,
Ышанам очрашуга юрап,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.
```

# Перевод

### Гөлүзә Арсланова - Позвони, мне, ты, позвони

```
Позвони, мне, ты, позвони,
Про здоровье мое расспроси,
Утоли мою бурную душу,
Позвони, мне, ты, позвони.

Алло, алло, я слушаю,
По твоему голосу тоскую.

Позвони, мне, ты, позвони,
С сердца переполненного отпусти,
Лишь о своих делах расскажи,
Позвони, мне, ты, позвони.

Алло, алло, я слушаю,
По твоему голосу тоскую.

Обогнув дальние-дальние дороги,
Ты придешь, о здоровье спросишь,
Верю встрече, судьбу предвещая,
Позвони, мне, ты, позвони.

Алло, алло, я слушаю,
По твоему голосу тоскую.
```
