# Оригинал

### Гөлфия Шакирова / Эльвира Хаматнурова - Мин яратам җырларга

```
Баш очымда - калфагым,
Аягымда - башмагым.
Ничек инде җырламыйсың,
Минем бит шундый чагым.

Хыялларым - ак җилкән,
Кузгала һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.

Күлмәгем аллы - гөлле,
Аллы - гөлле чәчәкле.
Чәчәкле генә түгел ул,
Үзенә бер бизәкле.

Хыялларым - ак җилкән,
Кузгела һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.

Шатлана бит күңелем,
Ямьле иртә - бүгенем.
Мин бит аңлыйм: барсы шуның,
Бер дә юкка түгелен.

Хыялларым - ак җилкән,
Кузгела һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.
```

# Перевод

### Гульфия Шакирова / Эльвира Хаматнурова - Я люблю петь

```
На голове мой платок,
На ногах - башмаки.
Как же мне не петь,
Ведь у меня такая пора.

Мечты мои - белый парус,
Поднимается каждое утро.
Мелодии слагая с напевами,
Я люблю петь.

Платок мой - с цветами,
С цветами, с цветами.
Не только с цветами он,
Но и украшениями убран.

Мечты мои - белый парус,
Поднимается каждое утро.
Мелодии слагая с напевами,
Я люблю петь.

Радуется моё сердце,
Прекрасное утро - сегодняшний день.
Ведь я понимаю: всё в этом мире
Не напрасно, не зря.

Мечты мои - белый парус,
Поднимается каждое утро.
Мелодии слагая с напевами,
Я люблю петь.
```
