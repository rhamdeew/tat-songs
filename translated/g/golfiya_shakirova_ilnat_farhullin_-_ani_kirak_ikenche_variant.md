# Оригинал

### Гөлфия Шакирова / Илнат Фәрхуллин - Әни кирәк (икенче вариант)

```
Инде үсеп, буйга җитеп,
Дөнья куган чагыңда да
Әни кирәк, әни кирәк.
Улым, диеп хәл белергә,
Балам диеп, кайгыртучы
Әни кирәк, әни кирәк.

Эч сереңне сөйләр өчен,
Алыр өчен яшәү көчен
Әни кирәк, әни кирәк.
Чын бәхетне тояр өчен,
Иңгә канат куяр өчен
Әни кирәк, әни кирәк.

Догасыннан калдырмаган,
Уйлап ятып йокламаган
Әни кирәк, әни кирәк.
Изгеләрдән изге кеше,
Баштан сыйпар өчен дә бит
Әни кирәк, әни кирәк.

Эч сереңне сөйләр өчен,
Алыр өчен яшәү көчен
Әни кирәк, әни кирәк.
Чын бәхетне тояр өчен,
Иңгә канат куяр өчен
Әни кирәк, әни кирәк.
[ речитатив:
Догасыннан калдырмаган,
Уйлап ятып йокламаган...
Изгеләрдән изге кеше..
Баштан сыйпар өчен дә бит...
Уңышларны күрергә дә,
Матур сүзләр әйтергә дә,
Шатлык уртаклашырга да...
Һәр җирдәге фәрештә ул... ]
Эч сереңне сөйләр өчен,
Алыр өчен яшәү көчен
Әни кирәк, әни кирәк.
Чын бәхетне тояр өчен,
Иңгә канат куяр өчен
Әни кирәк, әни кирәк.
```

Нужна мама, нужна мама.

Сказать: «Дитя», тревожно любя -
Нужна мама, нужна мама.

Чтобы погладить по голове -
