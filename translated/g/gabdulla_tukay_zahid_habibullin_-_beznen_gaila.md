# Оригинал

### Габдулла Тукай / Заһид Хәбибуллин - Безнең гаилә

```
Әткәй, әнкәй, мин, апай, әби, бабай һәм бер песи -
Безнең өйдә без җидәү: безнең песи - җиденчесе.

Бергә ашый, чәй эчә, безнең лә
*
бергә йоклый ул,
Хезмәте дә бар: өйне тычкан явыздан саклый ул.
```

# Перевод

### Габдулла Тукай / Заһид Хәбибуллин - Наша семья

```
Отец, мать, я, сестра, бабушка, дедушка и одна кошка -
В нашем доме нас семеро: наша кошка - седьмая.

Вместе ест, с нами чай пьет, с нами же
*
вместе спит она,
И служба есть: дом от злых мышей хранит она.
```
