# Оригинал

### Гөлкәй Вәлиәхмәт / Раил Өметбаев - Сиңа гына (өченче вариант)

```
Бүлек итимме йолдызлар,
Син әйт кенә кайсысын.
Бары синең өчен генә
Сайлыйм иң-иң яктысын.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.

Сиңа гына чәчәк ата
Бакчадагы гөлләрем.
Бәхет-шатлыкка күмелә
Синең янда көннәрем.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.

Канатланып, канатланып
Яныңа кайтып киләм.
Минем өчен бик кадерле
Икенлегеңне беләм.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.
```

# Перевод

### Гюлькей Валиахмат / Раиль Уметбаев - Только тебе (третий вариант)

```
Разделила ли звёзды,
Ты скажи только какие.
Лишь для тебя одной
Выбираю самую-самую яркую.

Только тебе, только тебе
Лучи луны и солнца.
Только тебе посвящаются
Самые красивые песни.

Только тебе цветут
Цветы в моём саду.
В счастье и радости погружены
Дни рядом с тобой.

Только тебе, только тебе
Лучи луны и солнца.
Только тебе посвящаются
Самые красивые песни.

Расправив крылья, расправив крылья
Возвращаюсь к тебе.
Для меня очень дорога
Твоя близость, я знаю.

Только тебе, только тебе
Лучи луны и солнца.
Только тебе посвящаются
Самые красивые песни.
```
