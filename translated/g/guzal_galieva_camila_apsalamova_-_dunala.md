# Оригинал

### Гүзәл Галиева / Җәмилә Әпсәләмова - Дунала

```
Яфрак ярды дунала
Язгы уыз дөньяда.
Шундый матур урманы,
Җәннәт инде торганы.

Көзге урман уйлана...
Томрап пешә дунала...

... Чәчкә атты дунала
Зөбәрҗәттәй дөньяда.
Муел дип җан аталар
Сындырып "яраталар".

Көзге урман уйлана...
Томрап пешә дунала...

Эй юләркәй, дунала,
Юкка җаның сызлана,
Бәхетеңне көт көн дә,
Байлыгың үз өстендә.

Көзге урман уйлана...
Томрап пешә дунала...

Юк тик (шул) аны күрүче,
Яннарына килүче.
Башын ия дунала,
Ямьсезмен, дип, моңлана.

Көзге урман уйлана...
Томрап пешә дунала...
```

# Перевод

### Гүзәл Галиева / Җәмилә Әпсәләмова - Дунала

```
Дунала распустила лист
В свежей весенней тиши.
Такой красивый лес,
Словно рай, где стоишь.

Осенний лес задумался...
Томно наливается дунала...

... Дунала расцвела
В мире, словно изумруд.
Зовут её «муел», жаждут,
Ломая, «любят» грубо.

Осенний лес задумался...
Томно наливается дунала...

Эх, дурёха, дунала,
Напрасно душа болит,
Жди своё счастье каждый день,
Твоё богатство при тебе.

Осенний лес задумался...
Томно наливается дунала...

Нет ведь того, кто увидит её,
Кто подойдёт поближе.
Склоняет голову дунала,
«Я некрасива», - стонет.

Осенний лес задумался...
Томно наливается дунала...
```
