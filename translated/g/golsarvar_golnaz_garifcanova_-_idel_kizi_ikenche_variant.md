# Оригинал

### Гөлсәрвәр / Гөлназ Гарифҗанова - Идел кызы (икенче вариант)

```
Сөйкемле кыз Идел буйларында
Матур итеп җырлый моңланып,
Дәрья дулкыннары, таллары да,
Барысы да тыңлый тын калып.

Идел кызы диеп үз иткәндәй,
Ошыйсың дип безнең күңелгә,
Акчарлаклар дулкын өсләрендә
Ак канатын кага Иделгә...

Моң агыла Идел дулкынында
Чишмә суларына кушылып.
Иң саф хисләр йөрәк ялкынында,
Күңелгә керә хуш булып...

Идел кызы диеп үз иткәндәй,
Ошыйсың дип безнең күңелгә,
Акчарлаклар дулкын өсләрендә
Ак канатын кага Иделгә...
```

# Перевод

### Гөлсәрвәр / Гөлназ Гарифжанова - Дочь Волги (второй вариант)

```
Милая девушка на берегах Волги
Красиво поёт, с тоской в напеве,
И речные волны, и ивы -
Все слушают, замерев в тишине.

Словно приняли тебя за дочь Волги,
Ты пришлась по сердцу нам,
Чайки белые над волной
Бьют белым крылом по Волге...

Песня льётся в волжской волне,
Сливаясь с водой родников.
Самые чистые чувства в огне сердца
Входят в душу, ласково, вновь...

Словно приняли тебя за дочь Волги,
Ты пришлась по сердцу нам,
Чайки белые над волной
Бьют белым крылом по Волге...
```
