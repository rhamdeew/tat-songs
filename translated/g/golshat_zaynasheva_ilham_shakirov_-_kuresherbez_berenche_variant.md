# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Күрешербез (беренче вариант)

```
Ник моңаеп уйга калдың?
Айрылабыз дисеңме?
Бәлки әле айрылмабыз,
Айрылмабыз шикелле.

Соң мәртәбә күрәм диеп,
Башың салма җилкәмә.
Бәлки әле очрашырбыз,
Очрашырбыз иртәгә?..

Без барасы ерак юллар
Бер булырга ошаган.
Озакламый күрешербез,
Күрешербез, ышанам!..
```

# Перевод

### Гульшат Зайнашева / Ильгам Шакиров - Встретимся (первый вариант)

```
О чем задумчиво затихла?
Хочешь расстаться, говоришь?
Может, пока не расстанемся,
Будто и не расстанемся вовсе.

В последний раз увижу, мол,
Не клади голову на плечо мне.
Может, ещё встретимся,
Встретимся ли завтра?..

Мы идём далёкими дорогами,
Стремясь стать единым целым.
Не за горами встреча наша,
Встретимся, верю я!..
```
