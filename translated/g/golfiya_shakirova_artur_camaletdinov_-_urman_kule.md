# Оригинал

### Гөлфия Шакирова / Артур Җамалетдинов - Урман күле

```
Урман күле түгәрәк,
Кил, җаныем, бергәләп.
Сөюне сүз белән генә
Бетерә алмам сөйләп.

Урманнары, күлләре,
Алан тулы гөлләре.
Кемнең генә шуны күреп
Хушланмас күңелләре.
Хисләр ул гүя урман,
Керсәң - адаша торган.
Син ул мине гашыйк иттең,
Ихластан яраттырган,
Үзеңә караттырган...

Урманнарда йөрербез,
Күлендә су керербез.
Гомергә без бергә дигән
Вәгъдәләрне бирербез.

Урманнары, күлләре,
Алан тулы гөлләре.
Кемнең генә шуны күреп
Хушланмас күңелләре.
Хисләр ул гүя урман,
Керсәң - адаша торган.
Син ул мине гашыйк иттең,
Ихластан яраттырган,
Үзеңә караттырган...

Аккошлар кайта монда
Күлдә оя корырга.
Кешегә күп кирәк мени
Бик бәхетле булырга.

Урманнары, күлләре,
Алан тулы гөлләре.
Кемнең генә шуны күреп
Хушланмас күңелләре.
Хисләр ул гүя урман,
Керсәң - адаша торган.
Син ул мине гашыйк иттең,
Ихластан яраттырган,
Үзеңә караттырган...
```
