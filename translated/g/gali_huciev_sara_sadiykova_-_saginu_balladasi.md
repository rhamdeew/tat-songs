# Оригинал

### Гали Хуҗиев / Сара Садыкова - Сагыну балладасы

```
Сугышка сез чыгып киткәндә,
Әле без калганбыз бишектә.
Инде һич оныта алмыйбыз
Без сезне ничек тә, ничек тә.

Тимерче учагың сүнмәгән,
Ул гел көтә сине, әткәем.
Ил бәхетен сакларга алмашка
Кадерләп үстерде Илкәем.

Каберең кайдадыр билгесез,
Туфрагын күмгәндер үләннәр.
Китмиләр күңелдән беркайчан
Ил өчен көрәшеп үлгәннәр.
```

# Перевод

### Гали Хуҗиев / Сара Садыкова - Баллада о тоске

```
Когда на войну вы ушли,
Мы в люльках еще оставались.
И ныне забыть не смогли
Мы вас никак, никак не смогли.

Не гаснет твой кузни огонь,
Он ждет тебя, отец мой родной.
На смену, хранить счастье страны,
С любовью взрастил тебя Илкәем.

Могила твоя где-то без вестей,
Травы, должно, скрыли тот прах.
И не уходят из сердца вовек
Павшие в битве за родной край.
```
