# Оригинал

### Гөлшат Зәйнәшева / Илгиз Мәҗитов - Урман юлында

```
Аерылдык урман юлында,
Җир җиләкле аланда.
Бик ямансу булды миңа
Бер ялгызым калганда.

Борылдым урман юлыннан,
Атладым синең эздән.
Җир җиләгенә чык түгел,
Яшь тамды минем күздән.

Җир җиләкләре коелыр,
Урман юлы тапталыр.
Көтәрмен сине гомергә,
Мәхәббәтем саф калыр.
```

# Перевод

### Гөлшат Зәйнәшева / Илгиз Мәҗитов - На лесной тропе

```
Расстались мы на лесной тропе,
На поляне земляничной.
Как же тоскливо стало мне,
Когда осталась я одна.

Свернула я с лесной дороги,
Пошла по твоему следу.
Не роса на землянике -
Слеза скатилась с моих глаз.

Земляника осыплется,
Лесная тропа утопчется.
Я буду ждать тебя навек,
Любовь моя чистой останется.
```
