# Оригинал

### Газиз Кашапов / Арслан Батыршин - Аксубай талларына

```
Инеш буе яшел чирәм,
Яши күз алларымда.
Кайткан саен сәлам бирәм
Аксубай талларына.

Сокланып һич тә туялмыйм
Туган як таңнарына.
Нигә кош булып куналмыйм
Аксубай талларына.

Кайтып киләм туган якка,
Озын юл алларымда.
Баш иям чыгып юллардан
Аксубай талларына.
```

# Перевод

### Газиз Кашапов / Арслан Батыршин - К ивам Аксубая

```
У ручья зеленый ковер
Живет перед глазами.
Всякий раз, вернувшись, шлю поклон
Аксубайским ивам.

Любуюсь - и не наполняюсь
Рассветами родными.
Почему не птицей мне спуститься
На ивы Аксубая.

Возвращаюсь в край родимый,
Дорога вдаль ложится.
С дорожной пыли кланяюсь
Аксубайским ивам.
```
