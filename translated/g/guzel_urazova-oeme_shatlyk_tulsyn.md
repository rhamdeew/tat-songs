# Оригинал

### Гүзәл Уразова - Оемэ шатлык тулсын

```
Матур голлэр устерэмен,
Оемэ бэхет телэп,
Ул голлэрнен гомерлэре
Миннэн икэнен белеп.
Оебездэ хэммэбезгэ
Шатлык бирсен голлэрем.
Ямьле булсын оебез дэ,
Шуны тели йорэгем.

Туган коннэр генэ тугел,
Хэр коне ямьле булсын.
Бэхет тойсын безнен кунел,
Оемэ шатлык тулсын.

Тэмле ашлар пешерэмен,
Чиксез ярату белэн,
Оебездэ хэммэбезгэ
Исэнлек- саулык телэп.
Нилэр генэ эшлэсэм дэ,
Якыннарым - кунелдэ.
Аларны да, голлэрне дэ
Яратырмын гомергэ.

Туган коннэр генэ тугел,
Хэр коне ямьле булсын.
Бэхет тойсын безнен кунел,
Оемэ шатлык тулсын.
```

# Перевод

### Гүзәл Уразова - Пусть дом наполнится радостью

```
Красивые цветы выращиваю,
Для дома счастье желая,
Зная, что жизнь тех цветов
Короче, чем моя.
В нашем доме всем нам
Радость дайте, цветы мои.
Красивым пусть будет наш дом,
Этого желает моё сердце.

Не только дни рождения,
Каждый день пусть будет прекрасным.
Счастье пусть чувствует наша душа,
Пусть дом наполнится радостью.

Вкусные блюда готовлю
С безграничной любовью,
В нашем доме всем нам
Здоровья желая.
Что бы ни делала,
Близкие мои - в душе.
И их, и цветы
Буду любить всю жизнь.

Не только дни рождения,
Каждый день пусть будет прекрасным.
Счастье пусть чувствует наша душа,
Пусть дом наполнится радостью.
```
