# Оригинал

### Габдулла Тукай / Рөстәм Яхин - Кем белер кадереңне?

```
Кем белер кадереңне, җаным,
Дәртле күңел булмаса?
Наз итәр кемнәрге гөл -
Каршыңда был-был булмаса?

Сүрәтеңнең иң чыны, бел;
Шагыйрең күңлендәдер,
Көзгеләрдән чын төсеңне
Күрмисең - син ул булмаса.
```

# Перевод

### Габдулла Тукай / Рөстәм Яхин - Кто узнает твою цену?

```
Кто цену узнает твою, родная,
Коль сердце жаром не горит?
Кто будет ласково, как роза,
Когда соловий перед тобой не трепещет?

Знай: самый верный образ твой
В душе поэта лишь живет;
В зеркалах не видишь истинных красок,
Если тебя в них нет.
```
