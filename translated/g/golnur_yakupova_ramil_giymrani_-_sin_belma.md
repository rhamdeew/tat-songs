# Оригинал

### Гөлнур Якупова / Рамил Гыймрани - Син белмә

```
Сиңа гына дигән бу хисләрем
Энҗе булып җилгә сибелә.
Яшерен генә өзелеп сөюемне
Җилләр генә белсен, син белмә.

Сиңа гына дигән бу назларым
Чәчәк булып җилгә сибелә.
Яшерен генә өзелеп сөюемне
Чәчәкләргә әйттем, син белмә.

Сиңа гына дигән бу сүзләрем
Җырым булып җилгә сибелә.
Яшерен генә өзелеп сөюемне
Мин җырларга салдым, син белмә.
```

# Перевод

### Гөлнур Якупова / Рамил Гыймрани - Ты не знай

```
Лишь тебе мои чувства - как жемчуг,
По ветру рассыплются в пыль.
Тайную, рвущую душу любовь мою
Пусть знают ветра, а ты - не знай.

Лишь тебе моя нежность - как цветы,
По ветру разлетаются вновь.
Тайную, рвущую душу любовь мою
Я шепнул цветам - ты не знай.

Лишь тебе мои слова - как песня,
По ветру рассыплется вновь.
Тайную, рвущую душу любовь мою
Я в песни вложил - ты не знай.
```
