# Оригинал

### Галимулла Нәбиев / Илдус Низамов - Яшьлек кыңгыраулары

```
Кыңгыраулы юргалар кебек
Чыңлап узды яшьлек елларым.
Ят таңнарга омтылды күңел,
Үз бәхетем читтән юлладым.

Нигә яшьлекнең юллары урау?
Ерак-еракта чыңлый кыңгырау...

Басып торам юллар чатында,
Уйлата шул синең хакыңда.
Кыңгыраулар, җиз кыңгыраулар
Моң уйната һаман якында.

Синнән хәбәрме моңлы яңгырау?
Сәламнәр әйтеп чыңлый кыңгырау.

Кыңгыраулар чыңлый якында,
Борма-борма юллар чатында.
Чәчәк булып кая атылма,
Тик онытма сөю хакында.

Күңелдә һаман чыңлый яңгырау,
Кавыштырырмы җырлы кыңгырау?..
```

# Перевод

### Галимулла Нәбиев / Илдус Низамов - Колокола молодости

```
Как тройки с колокольчиками
Звеня пролетели годы юности.
К чужим зарям стремилось сердце,
Своё счастье искал вдали.

Почему дороги молодости извиваются?
Вдали-вдали звенит колокольчик...

Стою на распутье дорог,
О тебе думаю вспоминаю.
Колокола, бронзовые колокола
Песню играют всё рядом.

От тебя ли весть мелодичный звон?
Приветы шлёт, звенит колокольчик.

Колокола звенят рядом,
На извилистом перекрёстке дорог.
Цветком расцвети куда хочешь,
Только не забывай о любви.

В душе всё звенит перезвон,
Сольётся ли в песню колокольный звон?..
```
