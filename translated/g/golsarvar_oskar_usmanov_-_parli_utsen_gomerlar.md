# Оригинал

### Гөлсәрвәр / Оскар Усманов - Парлы үтсен гомерләр

```
«Гүзәл дөнья, серле дөнья,
Дөнья түгәрәк!» - диләр.
Дөньяның бар ямен тоя
Парлашып яши кемнәр.

Аклансын изге хыяллар,
Уртак булсын бәхетләр!
Үз парын тапсын ялгызлар,
Парлы үтсен гомерләр!

Киң күңелгә бар да сыя,
Яхшы булгач ниятләр.
Парлы тормыш матурлана,
Булса күркәм гадәтләр.

Аклансын изге хыяллар,
Уртак булсын бәхетләр!
Үз парын тапсын ялгызлар,
Парлы үтсен гомерләр!

Җырлар тыңлап кем моңлана?..
Ярсыз, парсыз күңелләр...
Үз илендә ялгызлар да,
Юк!.. Бәхетсез түгелләр!

Аклансын изге хыяллар,
Уртак булсын бәхетләр!
Үз парын тапсын ялгызлар,
Парлы үтсен гомерләр!
```

# Перевод

### Гөлсәрвәр / Оскар Усманов - Пусть годы идут в паре

```
«Прекрасен мир, загадочен мир,
Мир круглый!» - так говорят.
Всю прелесть мира чувствуют
Те, кто живут в паре.

Пусть сбудутся святые мечты,
Пусть будет счастье общим!
Пусть одинокие найдут свою пару,
Пусть годы идут в паре!

В широкое сердце все вмещается,
Когда добры намерения.
Парная жизнь расцветает,
Если привычки красивы.

Пусть сбудутся святые мечты,
Пусть будет счастье общим!
Пусть одинокие найдут свою пару,
Пусть годы идут в паре!

Кто, слушая песни, тоскует?..-
Без милого, без пары сердца...
И в своем краю одинокие -
Нет!.. Они не несчастны!

Пусть сбудутся святые мечты,
Пусть будет счастье общим!
Пусть одинокие найдут свою пару,
Пусть годы идут в паре!
```
