# Оригинал

### Гөлшат Зәйнәшева / Рәшит Абдуллин - Мандолина

```
Уйна, дустым, мандолина,
Кыллары чыңлап торсын.
Чишмә кебек челтери дип,
Егетләр тыңлап торсын.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары.

Уйна, дустым, мандолина,
Кыллары нечкө-нечкө.
Сөйсәң, сөй ярның матурын,
Калмасын үкенечкә.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары.

Уйна, дустым, мандолина,
Кылларын чиртеп кенә.
Яшик җырлап, бу илләрдә,
Күңелле итеп кенә.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары!..
```

# Перевод

### Гөлшат Зәйнәшева / Рәшит Абдуллин - Мандолина

```
Играй, мой друг, на мандолине,
Пусть струны звонко прозвенят.
Как родниковое журчанье -
Пусть парни слушают, молчат.

Терлин, терлин, терлин, терлин -
Струны мандолины.
Тростниковый палец, серебряный ноготь -
Это руки друга моего.

Играй, мой друг, на мандолине,
Пусть струны тонко-нежно звенят.
Любя - люби красавицу-невесту,
Чтоб не осталось сожаленья в ряд.

Терлин, терлин, терлин, терлин -
Струны мандолины.
Тростниковый палец, серебряный ноготь -
Это руки друга моего.

Играй, мой друг, на мандолине,
Лишь чуть касаясь струн рукой.
Будем жить, поя в этих землях,
Лишь веселою судьбой.

Терлин, терлин, терлин, терлин -
Струны мандолины.
Тростниковый палец, серебряный ноготь -
Это руки друга моего!..
```
