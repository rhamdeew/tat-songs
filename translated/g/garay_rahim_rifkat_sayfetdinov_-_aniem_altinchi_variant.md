# Оригинал

### Гәрәй Рәхим / Рифкать Сәйфетдинов - Әнием (алтынчы вариант)

```
1.
Йөрәккә дәрт керә,
Тәнемә көч бирә,
Балам диеп, өзелеп,
Һәрвакыт әйткәнең.
Син минем яклаучым,
Беләм, саклаучым.
Бердәнбер киңәшчем,
Әнием минем.
2.
Тормышлар авыр дип,
Талчыктым ардым, дип.
Зарланып, бер сүз дә
Беркайчан әйтмәдең.
Күп газап кичердең,
Язмышка баш идең.
Син безне үстердең,
Әнием минем.
3.
Гомерем үзды, дип,
Картайдым, туздым, дип.
Сыкранып, сызланып,
Беркайчан әйтмәдең.
Йөрәгең сафлыгы,
Куңелең пакълыгы.
Син һаман яшь кебек,
Әнием минем.
```

# Перевод

### Гарей Рахим / Рифкат Сайфетдинов - Мама (шестой вариант)

```
1.
В сердце вдохновенье даришь,
Силу в теле моём пробуждаешь,
«Ребёнок мой» - с любовью такой
Вечно говоришь.
Ты - защитница моя,
Знаю, хранительница моя.нЕдинственный советчик,
Мама моя.
2.
«Жизнь трудна» - не жаловалась,
«Изнемогла, устала» - не говорила.
Никогда, стеная, ни слова
Не произнесла.
Муки многие перенесла,
С судьбой споря, голову подняла.
Нас взрастила ты,
Мама моя.
3.
«Жизнь прошла» - не говорила,
«Постарела, износилась» - не жаловалась.
Вздыхая, страдая,
Никогда не сетовала.
Чистота сердца твоего,
Невинность души твоей.
Ты всё молодая,
Мама моя.
```
