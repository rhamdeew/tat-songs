# Оригинал

### Гөлнара Сабитова / Оскар Усманов - Өзгәләнмә

```
Язмышлардан язмышларга күчеп
Кемне, бәгърем, бүген эзлисең?
Үткәннәрне хаталарга бәйләп
Юлларыңда кемне күзлисең?

Очрашканда карама син
Өзгәләнеп күзләремә.
Үткәннәргә юллар ябык,
Чирәм үскән эзләренә.

Ташка тиңлисеңдер йөрәгемне,
Әллә миңа җиңел дисеңме?
Йөрәгемне учлап басып торам,
Алалмыйча күктән күземне.

Очрашканда карама син
Өзгәләнеп йөзләремә.
Кирәк түгел инде хәзер
Яз кояшы көзләремә.
```

# Перевод

### Гөлнара Сабитова / Оскар Усманов - Не терзайся

```
От судьбы к судьбе переходя,
Кого, родной, сегодня ищешь ты?
Прошлое к ошибкам привязав,
На дорогах чьих следов ждешь ты?

При встрече не смотри ты
С болью в мои глаза.
В прошлое дороги заперты,
Травой заросли следы.

Камню ты, быть может, сердце уподобишь,
Иль считаешь, мне легко?
Я сжимаю сердце на ладони,
Не отрывая взгляд от небес высоко.

При встрече не смотри ты
С тоской на мое лицо.
Мне уже не нужно больше
Весеннее солнце осенью моей.
```
