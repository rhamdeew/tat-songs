# Оригинал

### Гөлназ Гарипова / Андрей Литвинов - Язмышка ышан

```
Күкләрдән әзли hәр кызый
Җирдән көтәсен.
Яшьлек шундый җүләр инде
Нихәл итәсен.
Күкләрдән күзли hәр кызый
Җирдән табасын, тик табалсын!
Очып кына үтә күңел
Җир-күк арасын!

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара -
Юллар тоташып бара...
Юллар тоташып бара...

Беркайчан да ашыктырма
Килер язларны.
Таш яуса да сиңа булыр
Сиңа язганы.
Язларыңны ашыктырма,
Язмышка ышан, ышан гына!
Көтеп алган бәхет кенә
Кадерле була.

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара -
Юллар тоташып бара...
Юллар тоташып бара...

Күпме йөкләр күтәрсәң дә
Арымас иңең.
Мин - җиңеллек китерүче
Бәхетең синең.
Күпме юллар үтелсә дә
Язмышка ышан, ышан гына!
Гомер дигән тирән савыт
Тамчылап тула.

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара -
Юллар тоташып бара...
Юллар тоташып бара...
```

# Перевод

### Гульназ Зарипова / Андрей Литвинов - Верь в судьбу

```
С небес ждёт каждую девчонку,
С земли ждёт ответа.
Молодость ведь такая лёгкая -
Что с ней поделаешь?
С небес видит каждую девчонку,
Пусть на земле найдёт, лишь бы нашла!
Душа порхает между небом и землёй!

Уведи меня в свои мечты,
Подними в свои сильные руки.
Не спрашивай ничего, посмотри на меня -
Дороги соединяются...
Дороги соединяются...

Никогда не торопи
Наступающие весны.
Даже если камни упадут тебе -
Весна останется с тобой.
Не торопи свои весны,
Верь в судьбу, только верь!
Жданная лишь радость
Станет дорогой.

Уведи меня в свои мечты,
Подними в свои сильные руки.
Не спрашивай ничего, посмотри на меня -
Дороги соединяются...
Дороги соединяются...

Сколько бы грузов ни нёс ты -
Спина не согнётся.
Я - создатель лёгкости,
Твоего счастья.
Сколько бы дорог ни прошло -
Верь в судьбу, только верь!
Глубокий сосуд, называемый жизнью,
Наполняется по капле.

Уведи меня в свои мечты,
Подними в свои сильные руки.
Не спрашивай ничего, посмотри на меня -
Дороги соединяются...
Дороги соединяются...
```
