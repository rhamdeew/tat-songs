# Оригинал

### Габдулла Тукай / Татар халык - Җил

```
Исəр җил! көчлесең, син бик батырсың,
Ачулансаң, җиһанны кузгатырсың.

Бик иркенлəп исəсең җир йөзендə,
Котыртып болгатасың диңгезен дə.

Көтү төсле болытларны куасың,
Телəрсəң кайсы якларга борасың.

Иреклесең: исəсең дə исəсең,
Тузан, кар туздырып юллар кисəсең.

Ирекле син, исəр җил, əйдə ис, ис!
Сиңа баш юк Ходайдан башка һич, һич!
```

# Перевод

### Габдулла Тукай / Татар халык - Ветер

```
Пьяный ветер! Ты силен, ты храбрец,
Рассердившись - взворочаешь весь свет.

Ты свободно гуляешь по земле,
Буйством взбалтываешь море во мгле.

Гонишь тучи, как стадо, по степям,
Куда хочешь - туда их повернешь сам.

Ты свободен: всё дуешь да дуешь,
Пыль и снег взметаешь, пути крушишь.

Ты свободен, пьяный ветер, дуй, дуй!
Над тобой нет владыки, кроме Бога, ничуть!
```
