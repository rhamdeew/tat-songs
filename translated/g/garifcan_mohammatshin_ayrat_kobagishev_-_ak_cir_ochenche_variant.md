# Оригинал

### Гарифҗан Мөхәммәтшин / Айрат Кобагышев - Ак җыр (өченче вариант)

```
Бар дөнья ак кына,
Күңелләр шат кына,
Коела, коела ап-ак кар;
Ак уйнак буранда
Без икәү урамда -
Бу ак төн гомерлек бер ядкарь.

Яңа ел кичендә
Ак карлар эчендә
Адаштык, ахры, без адаштык;
Сихри ай нурында
Мәхәббәт турында
Иң изге вәгъдәләр алмаштык.

Яңа ел кичләре
Керсез саф хисләрне
Гомергә бер җепкә бәйләде;
Коела ап-ак кар,
Бу ак төн - ак ядкарь,
Каеннар шаһите вәгъдәнең.
```

# Перевод

### Гарифжан Мухамметшин / Айрат Кобагышев - Белая песня (третий вариант)

```
Весь мир бело-белый,
На сердце радостно,
Падает, падает белый снег;
В белой игривой метели
Мы вдвоём на улице -
Эта белая ночь - память на всю жизнь.

В новогоднюю ночь
Среди белых снегов
Мы заблудились, право, заблудились;
Под волшебным сиянием луны
О любви
Самые святые обещания мы дали друг другу.

Новогодние вечера
Чистые чувства без изъяна
На одну нить связали навсегда;
Падает белый снег,
Эта белая ночь - белая память,
Берёзы - свидетели обещаний.
```
