# Оригинал

### Гөлфия Юнысова / Эльвира Хаматнурова - Әниләр бит яз кебек!

```
Язгы кояш балкый күктә
Үзе кайнар, үзе якты.
Әниләрнең карашы да
Нурлы итә тирә-якны.

Бөреләрне тирбәлдерер
Җылы, йомшак җиләр исә.
Әниләрнең назлы сүзе
Бәхетле һәм көчле итә.

Карны тишеп, чәчәк чыга,
Гөрләтеп тора алсу таңнар.
Әниләр бәйрәме белән
Башланалар зәнгәр язлар.

Яз бәйрәмен куанышып
Каршылайбыз, җырлап-биеп.
Әниләр бит үзләре дә
Мәңге көләч язлар кебек.
```

# Перевод

### Гульфия Юнусова / Эльвира Хаматнурова - Мамы ведь как весна!

```
Весеннее солнце сияет в небе,
Сами горячее, само светлое.
Взгляд мамы тоже
Окружает всё сиянием.

Почки колышет
Тёплый, мягкий ветерок.
Ласковое слово мамы
Делает счастливыми и сильными.

Пробивая снег, расцветает цветок,
Шумят розовые рассветы.
С праздником мам
Начинаются лазоревые весны.
Весенний праздник встречая
Мы поём и танцуем.
Мамы ведь сами тоже
Вечно цветущие, как весны.
```
