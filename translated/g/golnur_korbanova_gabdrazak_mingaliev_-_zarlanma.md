# Оригинал

### Гөлнур Корбанова / Габдразак Миңгалиев - Зарланма

```
Һәр көн саен бер ялгызың
Чишмәләргә киләсең,
Сөйгән ярым ташлады, дип,
Күз яшеңне түгәсең.

Зарланм(а) әле дөньядан,
Зарланмачы, әй, сеңел!
Күрче, дөнья нинди матур,
Зарланыр чагың түгел.

Парлыларны күрсәң әгәр,
Үзәкләрең өзелә.
Һәрнәрсәнең үз чиге бар,
Син хәзергә түз генә.

Зарланм(а) әле дөньядан,
Зарланмачы, әй, сеңел!
Уйнап-көлеп, җырлап-биеп,
Шаулап үтсен яшь гомер.
```

# Перевод

### Гөлнур Корбанова / Габдразак Мингалиев - Не жалуйся

```
Каждый день одна ты
К родникам приходишь,
Мол, любимый бросил, -
И слезами льёшься.

Не жалуйся ты на мир,
Не жалуйся, сестрёнка!
Посмотри, как мир красив,
Не время для сетований.

Если пары видишь -
Сердце рвётся в боли.
Всё имеет свой предел,
Ты пока терпи лишь.

Не жалуйся ты на мир,
Не жалуйся, сестрёнка!
В играх, в смехе, в песнях, в танце
Пусть шумит твоя юность.
```
