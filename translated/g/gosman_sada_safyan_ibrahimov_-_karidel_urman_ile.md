# Оригинал

### Госман Садә / Сафьян Ибраһимов - Каридел - урман иле

```
Карлыганнар җыйыйм дисәң,
Каридел - урман иле.
Карлыгандай кара күзле
Кызларның туган иле.
Кызларның туган иле,
Каридел - урман иле.

Карлыганга барган чакта,
Икәүләп барып кара.
Кайсы юлдан барсагыз да,
Яшьлеккә алып бара.
Яшьлеккә алып бара,
Икәүләп барып кара.

Кабып кара, татып кара
Кап-кара карлыганнар.
Гүя кызлар юри генә
Өзмичә калдырганнар.
Өзмичә калдырганнар,
Кап-кара карлыганнар.

Карлыганнар җыйыйм дисәң,
Каридел - урман иле.
Карлыгандай кара күзле
Кызларның туган иле.
Кызларның туган иле,
Каридел - урман иле!
```

# Перевод

### Госман Садә / Сафьян Ибраһимов - Каридел - лесной край!

```
Коль карлыган хочешь собрать,
Каридел - край лесной.
Там, как ягоды, черноглазые
Девушки - край родной.
Девушки - край родной,
Каридел - край лесной.

Если к карлыгану идешь,
Вдвоем ступай, взгляни.
С какой бы тропы ни пошел,
К юности путь ведёт.
К юности путь ведёт,
Вдвоем ступай, взгляни.

Попробуй, сорви, распробуй
Чёрный-чёрный карлыган.
Будто девичьи руки нарочно
Не срывали до конца.
Не срывали до конца,
Чёрный-чёрный карлыган.

Коль карлыган хочешь собрать,
Каридел - край лесной.
Там, как ягоды, черноглазые
Девушки - край родной.
Девушки - край родной,
Каридел - край лесной!
```
