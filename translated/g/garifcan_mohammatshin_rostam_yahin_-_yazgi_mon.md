# Оригинал

### Гарифҗан Мөхәммәтшин / Рөстәм Яхин - Язгы моң

```
Чит җирләрдән, илен сагынып,
Тургайлар кайтты инде;
Тургай күк талпына күңелем -
Моңнарым артты инде.

Ярсуыннан ярга сыймый,
Ташкыннар акты инде.
Күңеллемдә язгы ташкын -
Моңнарым артты инде.

Яшьлек хыялларым кебек,
Таңнарым якты инде.
Күңелемдә яз яктысы -
Моңнарым артты инде.

Яшьлегем - гомерем язы,
Язлар бик татлы инде.
Мәхәббәтен көтә күңелем,
Моңнарым артты инде...
```

# Перевод

### Гарифҗан Мөхәммәтшин / Рөстәм Яхин - Весенняя мелодия

```
Из чужих краев, тоскуя по родине,
Жаворонки уже вернулись;
Как жаворонок, рвется мое сердце -
Мои песни умножились.

В разливе, в ярости, не вмещаясь в берега,
Паводки уже пошли.
В душе моей весенний разлив -
Мои песни умножились.

Как мечты моей юности,
Мои рассветы уже светлы.
В душе моей весенний свет -
Мои песни умножились.

Юность моя - весна моей жизни,
Весны уже так сладки.
Сердце ждет своей любви,
Мои песни умножились...
```
