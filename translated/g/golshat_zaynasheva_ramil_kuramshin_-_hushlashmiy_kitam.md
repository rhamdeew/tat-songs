# Оригинал

### Гөлшат Зәйнәшева / Рамил Курамшин - Хушлашмый китәм

```
Тагын күрешергә булсын,
Хушлашмый китәм.
Искә алып бер җырларсыз,
Җыр бүләк итәм.

Җырларым бар, моңнарым бар
Йөрәк түремдә.
Җыр белән чын дуслар таптым
Туган җиремдә.

Чын дусларым икәнсез сез,
Шатландым күреп.
Йөзләрегез шундый нурлы,
Тулган ай кебек.

Тагын күрешергә булсын,
Хушлашмый китәм.
Сезнең белән бер күрешү
Бер гомер икән!..
```

# Перевод

### Гөлшат Зайнашева / Рамил Курамшин - Ухожу, не прощаясь

```
Пусть нам снова встретиться суждено,
Ухожу, не прощаясь.
Вы, вспомнив, спойте мне песню -
Песней вас одарю.

Есть у меня песни, есть у меня напевы
В тайниках сердца.
Песней нашёл я верных друзей
На родной земле.

Вы и есть мои верные друзья -
Я рад вас видеть.
Лица ваши так сияют,
Как полная луна.

Пусть нам снова встретиться суждено,
Ухожу, не прощаясь.
Одна встреча с вами -
Целая жизнь!..
```
