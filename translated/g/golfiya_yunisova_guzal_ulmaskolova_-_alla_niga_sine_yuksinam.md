# Оригинал

### Гөлфия Юнысова / Гүзәл Үлмәсколова - Әллә нигә сине юксынам?

```
Нигә өметләнеп көтә микән
Чыгарсың дип үзең каршыма.
Ник эзли соң сине күңелкәем,
Борчу уйлар салып башыма.

Әллә нигә сине юксынам,
Әллә нигә сиңа тартылам,
Күрергә кош кебек талпынам,
Очрасаң - кояштай яктырам.

Табыштырды гомер юлларыбыз,
Мин ышанам бары (һәр чак) язмышка.
Бүләк итәм ярсу йөрәгемне,
Күрмәшкә генә салышма.

Әллә нигә сине юксынам,
Әллә нигә сиңа тартылам,
Күрергә кош кебек талпынам,
Очрасаң - кояштай яктырам.
```

# Перевод

### Гульфия Юнусова / Гузель Ульмаскулова - Отчего я скучаю по тебе?

```
Почему с надеждой ждёшь ты,
Что сама выйдешь ко мне навстречу.
Почему ищет тебя душа моя,
Кладя на голову тревожные мысли.

Отчего я скучаю по тебе,
Отчего к тебе тянетесь,
Стремлюсь увидеть, как птица,
Встретимся - светлю, как солнце.

Разбежались наши жизненные пути,
Я верю во всякую судьбу.
Дарю тебе своё сердце,
Не сравнивай с несуществующим.

Отчего я скучаю по тебе,
Отчего к тебе тянетесь,
Стремлюсь увидеть, как птица,
Встретимся - светлю, как солнце.
```
