# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Сабан туе гүзәле

```
Бар кешенең күзе синдә,
Сабан туе гүзәле.
Бүген сине күзәтерләр,
Мең карашка түз әле.

Өйләнмәгән егетләрнең
Үзәкләрен өз әле,
Кырыктагы ирләрнең дә
Үзәкләрен өз әле.
Җитмештәге бабайларның
Үзәкләрен өз әле.
Минем генә үзәгемне,
Гүзәлкәем, өзмәле.

Яндыр әле йөрәкләрне,
Сабан туе гүзәле.
Бүген сине күзәтерләр,
Тимәс микән күзләре?

Өйләнмәгән егетләрнең
Үзәкләрен өз әле,
Кырыктагы ирләрнең дә
Үзәкләрен өз әле.
Җитмештәге бабайларның
Үзәкләрен өз әле.
Минем генә үзәгемне,
Гүзәлкәем, өзмәле.

Җилкет әле күңелләрне,
Сабан туе гүзәле.
Яшел чирәмнәргә басып,
Күзең сирпеп уз әле.

Өйләнмәгән егетләрнең
Үзәкләрен өз әле,
Кырыктагы ирләрнең дә
Үзәкләрен өз әле.
Җитмештәге бабайларның
Үзәкләрен өз әле.
Минем генә үзәгемне,
Гүзәлкәем, өзмәле.
```

# Перевод

### Галиәхмәт Шаһи / Виталий Агапов - Красотка сабантуйская

```
Все глаза на тебе,
Красотка сабантуйская.
Сегодня на тебя смотрят,
Выдержи тысячу взоров.

Сердца холостых юношей
Разбивай, не щади,
Сердца мужей в расцвете лет
Тоже рви, не щади,
Сердца стариков семидесяти
Тоже рви, не щади.
Только мое сердце,
Красавица, не тревожь.

Разжигай же сердца,
Красотка сабантуйская.
Сегодня на тебя смотрят,
Не ослепнут ли глаза?

Сердца холостых юношей
Разбивай, не щади,
Сердца мужей в расцвете лет
Тоже рви, не щади,
Сердца стариков семидесяти
Тоже рви, не щади.
Только мое сердце,
Красавица, не тревожь.

Оковывай же души,
Красотка сабантуйская.
Наступая на зеленые лужи,
Пусть твой взгляд скользит легко.

Сердца холостых юношей
Разбивай, не щади,
Сердца мужей в расцвете лет
Тоже рви, не щади,
Сердца стариков семидесяти
Тоже рви, не щади.
Только мое сердце,
Красавица, не тревожь.
```
