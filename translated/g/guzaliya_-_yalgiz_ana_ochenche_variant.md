# Оригинал

### guzaliya_ - Ялгыз ана (өченче вариант)

```
Зар елады ана өзелеп-өзелеп,
Югалтты ул бердәнбер баласын.
Язмыш тартып алды, жәлләмәде,
Кызганмады улын, анасын.

Каеннарны кочты ана,
Каенлыкта ул адашты.
Кайрыларын сыйпый, назлый,
Балам диеп саташты.

Өзгәләнде ана, күз алдыннан
Китми сабый чагы баланың.
Алып китте язмыш, йотты сугыш,
Утлы җиле йотты баланы.

Каеннарны кочты ана,
Каенлыкта ул адашты.
Кайрыларын сыйпый, назлый,
Балам диеп саташты.
```

# Перевод

### Гүзәлия - Одинокая мать (третий вариант)

```
Навзрыд плакала мать, обрываясь на слезах,
Потеряла она единственного сына.
Судьба вырвала, не пожалела,
Не пощадила ни сына, ни мать.

Обняла берёзы мать,
В берёзовой роще заблудилась.
Гладит их, ласкает,
Бормочет: «Сынок», - мечется.

Мать изнемогала, и пред глазами
Не уходит младенческая пора ребёнка.
Судьба унесла, война проглотила,
Огненный её ветер сына поглотил.

Обняла берёзы мать,
В берёзовой роще заблудилась.
Гладит их, ласкает,
Бормочет: «Сынок», - мечется.
```
