# Оригинал

### Галия Сафиуллина / Айнур Сәйфиев - Күз тимәсен бәхетемә

```
Күз тимәсен бәхетемә,
Сокланмагыз артыгын.
Язмыш җиле котыртмасын
Яшәвемнең шатлыгын.

Бәхеткә күз тия диләр,
Сакларга кирәк аны.
Минем бит гүзәл тормышның
Шау чәчкә аткан чагы.

Бәхетемә күз тимәсен,
Көнләшмәгез сокланып.
Бәхетемне бүлешәмен
Моңнарыма көй салып.

Бәхеткә күз тия диләр,
Сакларга кирәк аны.
Минем бит гүзәл тормышның
Шау чәчкә аткан чагы.

Бәхеткә күз тия икән,
"Сөбханалла" диегез.
Аллаһы биргән бүләккә
Шөкер итә белегез.

Бәхеткә күз тия диләр,
Сакларга кирәк аны.
Минем бит гүзәл тормышның
Шау чәчкә аткан чагы.
```

# Перевод

### Галия Сафиуллина / Айнур Сәйфиев - Пусть счастье не сглазят

```
Пусть счастье мое не сглазят,
Не восхищайтесь чрезмерно.
Пусть ветер судьбы не взбудоражит
Радость моего бытия.

Говорят: счастье сглазить легко,
Его беречь необходимо.
Ведь сейчас - пора моей прекрасной жизни,
Когда она пышно цветет.

Пусть счастье мое не сглазят,
Не завидуйте, любуясь.
```
