# Оригинал

### Гөлнур Вәлиева / Виталий Агапов - Ак төнбоек

```
Су буйларында йөримен,
Камышлар шавын тыңлап.
Төшләремә ник кермисең, иркәм,
Сөю җырларын җырлап.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.

Су буйларында йөримен,
Сагышларым таратып.
Яннарымда булмасаң да,
Яшим сине яратып.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.

Су буйларында йөримен,
Синең эзләрең буйлап.
Үтәр айлар, үтәр еллар,
Туймамын сине уйлап.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.
```

# Перевод

### Гөлнур Вәлиева / Виталий Агапов - Белая кувшинка

```
У берегов воды брожу,
Шелест камышей ловлю.
Отчего не входишь в сны мне, милый,
Песни любви напою.

У воды белая кувшинка
Плывет тихо, не спеша.
Отчего душа печальна,
Знают лишь ивы одна.

У берегов воды брожу,
Разгоняю свою тоску.
Хоть тебя со мной не будет рядом,
Я живу, любя тебя.

У воды белая кувшинка
Плывет тихо, не спеша.
Отчего душа печальна,
Знают лишь ивы одна.

У берегов воды брожу,
По твоим следам иду.
Пройдут месяцы, пройдут годы,
Не устану я мечту.

У воды белая кувшинка
Плывет тихо, не спеша.
Отчего душа печальна,
Знают лишь ивы одна.
```
