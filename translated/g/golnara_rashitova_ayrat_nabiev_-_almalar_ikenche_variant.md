# Оригинал

### Гөлнара Рәшитова / Айрат Нәбиев - Алмалар (икенче вариант)

```
Алмалар үсә икән,
Өзелеп төшә икән.
Алма кебек дуслар барда
Күңелләр үсә икән.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.

Алмалар уңган икән,
Бәхеткә булган икән.
Серләремне бүлешердәй
Дуслар табылган икән.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.

Алмаларны өзәрбез,
Өстәлләргә тезәрбез.
Дуслар белән бу дөньяларда
Дәрьяларны гизәрбез.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.
```

# Перевод

### Гульнара Рашитова / Айрат Набиев - Яблоки (второй вариант)

```
Яблоки растут - как диво,
С веток падают красиво.
Пока есть друзья, как яблоки,
Сердца растут счастливее.

Яблоки алые, нежные,
Попробуй - сладость безбрежная.
Разговоры с друзьями - покой,
Отдых душевный, дорогой.

Яблоки уродились щедро,
К счастью пришлись они, несметно.
Нашлись друзья, с кем тайну делить,
С кем можно сердце открыть.

Яблоки алые, нежные,
Попробуй - сладость безбрежная.
Разговоры с друзьями - покой,
Отдых душевный, дорогой.

Яблоки мы сорвем,
На столы их рядком кладем.
С друзьями в этом мире большом
Мы океаны обойдем.

Яблоки алые, нежные,
Попробуй - сладость безбрежная.
Разговоры с друзьями - покой,
Отдых душевный, дорогой.
```
