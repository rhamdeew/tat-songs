# Оригинал

### Гөлшат Зәйнәшева / Сара Садыкова - Идел суы ага торыр

```
Идел суы ага торыр,
Алысларга бара торыр.
Кылыч тотып килсә дошман,
Идел-суга бата торыр.
Кылыч тотып килсә дошман,
Идел-суга бата торыр.

Идел суы ага торыр,
Дулкыннарын кага торыр.
Өлкән җиңги угълан тапты,
Безнең нәсел арта торыр.
Өлкән җиңги угълан тапты,
Безнең нәсел арта торыр.

Идел суы ага торыр,
Җилкән көймә бара торыр,
Идел кичкән ир-егетләр
Чын дусларын таба торыр!..
Идел кичкән ир-егетләр
Чын дусларын таба торыр!..
```

# Перевод

```
Волга будет течь водою,
Уходя к дали земной.
Если враг придет с мечом -
В Волге канет он волной.
Если враг придет с мечом -
В Волге канет он волной.

Волга будет течь водою,
Волны бить о берег свой.
Старшая золовка сына
Породила - род живой.
Старшая золовка сына
Породила - род живой.

Волга будет течь водою,
Ладья под парусом плывет,
Кто Волгу смело перейдет -
Тех верная дружба найдет!..
Кто Волгу смело перейдет -
Тех верная дружба найдет!..
```
