# Оригинал

### Гөлшат Зәйнәшева / Татар халык көе - Кердем карурманнарга

```
Кердем карурманнарга,
Калдым караңгыларга.
Газиз башлар сау булганда,
Нигә ямансуларга?..

Кердем карурманнарга,
Бер кычкырып җырларга.
Бәгърем, бергә булсак сүз бир,
Ни күрсәк тә чыдарга.

Кердем карурманнарга,
Кермәдем аланнарга.
Көтегез, кайтырбыз, дидек,
Илләрдә калганнарга!..
```

# Перевод

### Гульшат Зайнашева / Татарская народная мелодия - К недобрым людям

```
Постучала я к злодеям,
Осталась во тьме.
Когда любимые целы,
Куда ж я попала?..

Постучала я к злодеям,
Чтоб крикнуть и запеть.
Милый, будь мы вместе - дай слово,
Что всё переживем вдвоем.

Постучала я к злодеям,
Не на широкие луга.
«Пождите, вернёмся», - молвили,
Тем, кто остался на века!..
```
