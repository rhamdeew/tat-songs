# Оригинал

### Гөлшат Зәйнәшева / Мәсгут Имашев - Кукмара (беренче вариант)

```
Каршы ала, озатып кала
Поездларны Кукмара.
Поездлар ерак китсә дә,
Ерагаймасын ара.

Их, Кукмара, Кукмара,
Ерагаймасын ара.
Сагынырсың, саргаерсың,
Читләргә китеп кара.

Ярсып поездлар узганда
Тетрәп кала таулары.
Җәйге таңда үзәкне өзә
Сандугач сайраулары.

Их, Кукмара, Кукмара,
Ерагаймасын ара.
Сагынырсың, саргаерсың,
Читләргә китеп кара.

Туган җирнең бик күңелле
Бер төбәге Кукмара.
Кукмара кешеләренең
Эшләре гөрләп бара.

Их, Кукмара, Кукмара,
Ерагаймасын ара.
Сагынырсың, саргаерсың,
Читләргә китеп кара.
```

# Перевод

### Гульшат Зайнашева / Мәсгут Имашев - Кукмара (первый вариант)

```
Встречает и провожает
Поезда Кукмара.
Хоть поезда далеко уходят,
Не отдаляется душа.

Ах, Кукмара, Кукмара,
Не отдаляется душа.
Скучаешь, тоскуешь,
Когда в чужие края уезжаешь.

Когда со свистом поезда проходят,
Вздрагивают горы.
Летним утром душу трогают
Соловьиные трели.

Ах, Кукмара, Кукмара,
Не отдаляется душа.
Скучаешь, тоскуешь,
Когда в чужие края уезжаешь.

Родной земли отрадный
Уголок - Кукмара.
Дела людей Кукмары
Громко гремят, цветут.

Ах, Кукмара, Кукмара,
Не отдаляется душа.
Скучаешь, тоскуешь,
Когда в чужие края уезжаешь.
```
