# Оригинал

### Гамил Афзал / Илһам Шакиров - Гөлмәрьям

```
Җәйләүләрдә көтү куды Гөлмәрьям,
Гөлләр өзде бер бәйләм Гөлмәрьям,
Күзләрендә чаялыгы чаткысы,
Йөзләрендә таң нурының яктысы,
Күзен алмый кулындагы гөлләрдән.

Болыннарда көтү йөртте Гөлмәрьям,
Сулар эчте көмештәй күлләрдән.
Җилбер җилбер исә сахра җилләре,
Җәйрәп яткан туган-үскән җирләрнең
Матурлыгын үзенә җыйган Гөлмәрьям.

Болыннардан эштән кайта Гөлмәрьям,
Гармунлы урамнарда нинди ямь!
Сылу үскән авыл кызы, ил кызы,
Сандугачлы таңнарның йолдызы -
Шундый күркәм минем иркәм Гөлмәрьям.
```

# Перевод

### Гамил Афзал / Илһам Шакиров - Гульмарьям

```
На летних пастбищах гурты гнала Гульмарьям,
Цветы рвала в один букет Гульмарьям,
В глазах - искра задора, смелый свет,
В лице - сиянье утренней зари,
Не отводя взора от цветов в ладонях.

На лугах пасла гурты Гульмарьям,
Пила воды из озер, как серебро.
Легко-легко веют степные ветры,
И красоту родной, распластанной земли
Всю в себе собрала Гульмарьям.

С лугов с работы возвращается Гульмарьям,
В гармошечных улицах - какая благодать!
Статная деревенская дочь, дочь земли,
Звезда рассветов с соловьиной песней -
Такой прекрасной мне милой Гульмарьям.
```
