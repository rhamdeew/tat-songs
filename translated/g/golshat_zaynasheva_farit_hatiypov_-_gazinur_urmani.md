# Оригинал

### Гөлшат Зәйнәшева / Фәрит Хатыйпов - Газинур урманы

```
Урманчы булган Газинур
Урманнарга җырлап кергән
Ак каеннар арасында,
Кошлар моңын тыңлап йөргән.

Газинур урманын кичтем
Җәйнең матур бер таңында.
Узәкләрне өзеп - өзеп,
Кәккүк кычкырган чагында.

Ак каеннар белә микән,
Кәккүк тәвышы нигә серле?
Кәккүк моңына күчкәндер
Газинурның газиз гомре...
```
