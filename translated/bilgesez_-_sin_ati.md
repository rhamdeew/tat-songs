# Оригинал

### Билгесез - Син, әти

```
Бармы икән җир йөзендә бизмән
Яхшылыкны синең үлчәрлек?
Синнән күчкән яшәү көче, әти,
Мең дарьяны җырлап кичәрлек.

Өебезнең үзәге син, әти,
Гаиләбезнең йөрәге син, әти.
Барыбызның терәге син, әти.
Язмышыбыз бүләге син, әти.

Гомер үзәнендә синең эзләр
Дәшә безне олы юлларга.
"Чәчләремә кырау төшкән", диеп,
Үпкәләмә әле елларга.

Өебезнең үзәге син, әти,
Гаиләбезнең йөрәге син, әти.
Барыбызның терәге син, әти.
Язмышыбыз бүләге син, әти.

Шаян бабайлраның иң шаяны,
Әйбәт бабайларның әйбәте.
Бүләк булсын әле бу җырыбыз,
Оныкларның сиңа рәхмәте.

Өебезнең үзәге син, әти,
Гаиләбезнең йөрәге син, әти.
Барыбызның терәге син, әти.
Язмышыбыз бүләге син, әти.
```

# Перевод

### Билгесез - Сын, оте

```
Есть ли на свете, отец,
Мера доброты твоя?
Сила жизни, что от тебя, отец,
Способна пропеть тысячу рек.

Ты - центр нашего дома, отец,
Ты - сердце нашей семьи, отец.
Ты - опора для всех нас, отец.
Ты - подарок нашей судьбы, отец.

Твои следы в русле жизни
Ведут нас по великим путям.
Сказав "иней на волосах",
Не буду жаловаться на года.

Ты - центр нашего дома, отец,
Ты - сердце нашей семьи, отец.
Ты - опора для всех нас, отец.
Ты - подарок нашей судьбы, отец.

Среди отважных отцов ты отважнейший,
Среди замечательных отцов ты замечательнейший.
Пусть эта песня будет подарком,
Благодарность твоих детей тебе.

Ты - центр нашего дома, отец,
Ты - сердце нашей семьи, отец.
Ты - опора для всех нас, отец.
Ты - подарок нашей судьбы, отец.
```
