# Оригинал

### Малика - Оныт эле аны

```
Минем яныма дип килгэндэ ул
Ник чыгасын анын юлына.
Яратмый ул сине, соя бары мине
Минем йорэктэ дэ ул гына.

Оныт эле, оныт эле аны
Карама син анын кузенэ,
Ул бит инде мине, бары мине соя
Белэсен бит инде узен дэ.

Элек без бит якын дуслар идек
Хэзер инде дэшми йорисен.
Соймэгэн ул сине, сойсэ китмэс иде
Нигэ инде шуны белмисен.

Аны котеп вакыт уздырма син
Эрэм  уза инде яшьлеген.
Ул бит мине генэ, бары мине соя
Мин дэ анын очен яшимен.
```

# Перевод

### Малика - Забудь её

```
Когда он идёт ко мне,
Зачем ты встаёшь на его пути.
Он не любит тебя, любит только меня,
В моём сердце тоже только он.

Забудь её, забудь её,
Не смотри в её глаза,
Он ведь теперь меня, только меня любит,
Ты ведь знаешь это и сама.

Раньше мы были близкими друзьями,
Теперь уж не говоришь - проходи.
Не полюбил он тебя, если б полюбил - не ушёл бы,
Почему же ты этого не понимаешь.

Не трать время, ожидая его,
Уходит уже молодость твоя.
Он ведь меня одну, только меня любит,
И я для него живу.
```
