# Оригинал

### Казан егетлəре - Алматанын алмасы

```
Алматанын алмасы
Тэмле хуш ислэр сибэ.
Сине генэ соям,
Сине генэ котэм,
Сине генэ курэсем килэ.

Алматанын кар суы
Биек таулардан килэ.
Сине генэ котэм,
Сине генэ сагынам,
Сине генэ курэсем килэ.

Алматанын кар суы
Минем йорэк ярсуы.
Сине генэ котэм,
Сине генэ соям,
Сине генэ килэ курэсем.
```

# Перевод

### Казан егетлəре - Алматинские яблоки

```
Алматинские яблоки
Разливают сладкий аромат.
Лишь тебя я люблю,
Лишь тебя я жду,
Лишь тебя видеть хочу.

Алматинские снежные воды
С высоких гор стекают.
Лишь тебя я жду,
Лишь о тебе тоскую,
Лишь тебя видеть хочу.

Алматинские снежные воды —
Моего сердца целебная влага.
Лишь тебя я жду,
Лишь тебя я люблю,
Лишь тебя видеть хочу.
```
