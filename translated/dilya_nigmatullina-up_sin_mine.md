# Оригинал

### Диля Нигматуллина - Уп син мине

```
Уп син мине иренемнэн
Сусауларым беткэнче.
Иннэремнэн кочып алып,
Кузлэремэ генэ багып,
Уп син мине эйдэ, уп.

Уп син мине, коч син мине, уп.
Иренендэ иреннэрем бит.
Бер убуен мен газапка торсын,
Иреннэрен тэме юксындырсын.
Бер убуен мен рэхэткэ торсын,
Иреннэрен тэме юксындырсын.

Алсу чия иреннэрен
Кайнар убуемне котэ бит.
Сонга калмыйк очашырга,
Балдакларны алышырга
Биршэн вэгъдэ безне котэ бит.

Уп син мине, коч син мине, уп.
Кузлэрендэ минем кузем бит.
Бер убуен мен газапка торсын,
Иреннэрен тэме юксындырсын.
Бер убуен мен рэхэткэ торсын,
Иреннэрен тэме юксындырсын.

Бер убешуен дэ багу
Азапларга мин сине.
Убулэрен сусыз чишмэ,
Тик син генэ йорэгемдэ,
Татлы булсын мэхэббэт.

Уп син мине, назла мине, уп.
Иренендэ иреннэрем бит.
Бер убуен мен газапка торсын,
Иреннэрен тэме юксындырсын.
Бер убуен мен рэхэткэ торсын,
Иреннэрен тэме юксындырсын.
```

# Перевод

### Диля Нигматуллина - Целуй меня

```
Целуй меня до губ моих,
Пока не иссякнет жажда.
Из плена освободив,
Только в глаза мне глядя,
Целуй меня, давай, целуй.

Целуй меня, обнимай меня, целуй.
На губах моих — губы твои.
Пусть один поцелуй мучителен будет,
Вкус губ пусть тоскою томит.
Пусть один поцелуй блаженством станет,
Вкус губ пусть тоскою томит.

Розовые щёки твои
Горячий поцелуй мой ждёт ведь.
Не будем медлить встречаться,
Детей воспитывать
Обещание одно нас ждёт ведь.

Целуй меня, обнимай меня, целуй.
В глазах твоих — мой взгляд ведь.
Пусть один поцелуй мучителен будет,
Вкус губ пусть тоскою томит.
Пусть один поцелуй блаженством станет,
Вкус губ пусть тоскою томит.

Ни одним поцелуем
В муки не ввергну тебя.
Поцелуи — источник без жажды,
Только ты в сердце моём,
Сладкой пусть будет любовь.

Целуй меня, ласкай меня, целуй.
На губах моих — губы твои.
Пусть один поцелуй мучителен будет,
Вкус губ пусть тоскою томит.
Пусть один поцелуй блаженством станет,
Вкус губ пусть тоскою томит.
```
