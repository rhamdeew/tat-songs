# Оригинал

### Айдар Шакиров - Иркэмэ

```
Кузлэренэ тутырып карап
"Соям" диясем килэ,
Сагындым сине озелеп,
Улеп курэсем килэ.

Чэчлэреннэн сыйпап, иркэм,
"Яратам" дияр идем.
Сагынуларымны бетереп,
Суырып убэр идем.

Сине уйлап йорим кон дэ,
Куз алдымда син генэ.
Минем ничек союемне
Узен белмисен генэ.

Чэчлэреннэн сыйпап, иркэм,
"Яратам" дияр идем.
Сагынуларымны бетереп,
Суырып убэр идем.
```

# Перевод

### Айдар Шакиров - Моей милой

```
Пристально глядя в глаза твои,
"Люблю" хочется сказать,
Соскучился по тебе истосковавшись,
Умереть хочется.

По волосам твоим поглаживая, милая,
"Люблю" сказал бы.
Завершив мою тоску,
Поцеловал бы крепко.

О тебе думаю и днём,
Перед глазами только ты.
Как я тебя люблю,
Ты просто не знаешь.

По волосам твоим поглаживая, милая,
"Люблю" сказал бы.
Завершив мою тоску,
Поцеловал бы крепко.
```
