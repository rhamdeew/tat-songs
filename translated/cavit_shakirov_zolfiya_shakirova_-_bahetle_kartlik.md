# Оригинал

### Җәвит Шакиров / Зөлфия Шакирова - Бәхетле картлык

```
Яшь чагында, көч барында
Йөзләрдә шатлык.
Яшьлек үтәр, алда көтсен
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

Көч барында гармуннарны
Уйныйсың тартып.
Җырлы-моңлы булсын иде
Без көткән картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

Матур тормыш сукмагыннан
Бергәләп атлыйк,
Алда көтсен һәммәбезне
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

"Байлык - бәхет" дигәннәргә
Ышанма артык,
Бу дөньяда иң зур байлык -
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.
```

# Перевод

### Җәвит Шакиров / Зөлфия Шакирова - Счастливая старость

```
В молодости, пока есть силы,
На лицах - радость.
Пройдёт молодость, пусть впереди ждёт
Счастливая старость.

Не печалюсь, что старею,
Не горюю слишком.
Ещё не настигла нас
Счастливая старость.

Пока есть силы, гармони
Играешь, напевая.
Пусть будет песенной, мелодичной
Старость, что нас ждёт.

Не печалюсь, что старею,
Не горюю слишком.
Ещё не настигла нас
Счастливая старость.

По тропе прекрасной жизни
Давайте шагать вместе,
Пусть впереди ждёт всех нас
Счастливая старость.

Не печалюсь, что старею,
Не горюю слишком.
Ещё не настигла нас
Счастливая старость.

Тем, кто говорит "богатство - счастье",
Не верь слишком,
В этом мире самое большое богатство -
Счастливая старость.

Не печалюсь, что старею,
Не горюю слишком.
Ещё не настигла нас
Счастливая старость.
```
