# Оригинал

### Ильдар Ахметов - Энилэр кузе

```
Ин йомшак куллар, ин кочле иннэр,
Тик энилэрдэ югала тиннэр.
Энилэр кузе ин матур кузлэр,
Энилэр сузе ин назлы сузлэр.

Алар бирэлэр ин дорес кинэш,
Алар булалар ин яхшы сердэш.
Ин нечкэ кунел, ин саф зур йорэк,
Ин кайнар сою, ин изге телэк.

Ин жылы кунел тик энилэрдэ,
Бэхет бар алар булган ойлэрдэ.
Яшэу яме сез биргэн бит безгэ,
Энилэр, рэхмэт, мен рэхмэт сезгэ.
```

# Перевод

### Ильдар Ахметов - Материнские глаза

```
Самые мягкие руки, самые сильные плечи,
Только у матерей можно их встретить.
Материнские глаза - самые красивые глаза,
Материнские слова - самые нежные слова.

Они дают самый правильный совет,
Они становятся лучшими спутниками.
Самая тонкая душа, самое чистое большое сердце,
Самая горячая любовь, самое святое желание.

Самая тёплая душа только у матерей,
Счастье есть в домах, где они живут.
Радость жизни вы подарили нам,
Матери, спасибо, тысяча раз спасибо вам.
```
