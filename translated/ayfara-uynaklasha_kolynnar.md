# Оригинал

### Айфара - Уйнаклаша колыннар

```
Экиятен ерак торсын
Бу могжиза болында
Иркенлектэ кинэнешеп
Уйнаклаша колыннар

Уйнаклаша, уйнаклаша
Уйнаклаша колыннар
Колыннарга болыннар

Тук-тук килэ куш бу яклар
Желберди ял толымнар
Балачакны хэтерлэтэ
Болындагы колыннар

Уйнаклаша, уйнаклаша
Уйнаклаша колыннар
Колыннарга болыннар

Булдермэсен алар юлын
Оермэлэр, коеннар
Толпар булып буйга житсен
Уенчак шул колыннар

Уйнаклаша, уйнаклаша
Уйнаклаша колыннар
Колыннарга болыннар
```

# Перевод

### Айфара - Резвятся жеребята

```
Пусть сказка будет далеко
В этом чуде луговом,
Весело заржав,
Резвятся жеребята.

Резвятся, резвятся,
Резвятся жеребята
На лугах жеребята.

Стук-стук приходят птицы в эти края,
Развевается грива их,
Детство напоминают
Луговые жеребята.

Резвятся, резвятся,
Резвятся жеребята
На лугах жеребята.

Не заблудятся на своем пути
Холмы и долины,
Тулпаром став, до степи достигнут
Игривые те жеребята.

Резвятся, резвятся,
Резвятся жеребята
На лугах жеребята.
```
