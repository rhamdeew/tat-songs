# Оригинал

### Айфара - Сау бул

```
Син – йолдызга,
Ник урелим?
Яратсам да, дэшмэдем.
Тоштэ курэм сине генэ,
Э син никтер мине генэ
Курмэгэндэй яшэден.

Сонгы сузем "сау бул" инде,
Мина кирэктер шул гына,
Сау бул инде, купме туздем,
Буген ометемне оздем,
Сау бул,
Сау бул,
Сау бул, тик бэхетле бул!

Бергэ булу язмагандыр,
Бэхет сина телэдем.
Булэк итэм сина голлэр,
Узды омет тулы коннэр,
Оч иреккэ йорэгем.

Сонгы сузем "сау бул" инде,
Мина кирэктер шул гына,
Сау бул инде, купме туздем,
Буген ометемне оздем,
Сау бул,
Сау бул,
Сау бул, hэм бэхетле бул!

Укенэсе булыр микэн,
Нигэ элек дэшмэдем?
Синен белэн hэрчак уем?
Сине куру мина кыен,
Тик куренмэс яшьлэрем.

Сонгы сузем "сау бул" инде,
Мина кирэктер шул гына,
Сау бул инде, купме туздем,
Буген ометемне оздем,
Сау бул,
Сау бул,
Сау бул, тик бэхетле бул!
```

# Перевод

### Айфара - Прощай

```
Ты - к звёздам,
Зачем стремлюсь?
Хоть и любил, не сказал.
Во сне вижу только тебя,
А ты словно меня
Не замечая живёшь.

Последнее слово "прощай" уже,
Мне нужно только это,
Прощай уже, сколько терпел,
Сегодня надежду разорвал,
Прощай,
Прощай,
Прощай, только будь счастлива!

Быть вместе не суждено,
Счастья тебе желаю.
Дарю тебе цветы,
Прошли дни, полные надежд,
Сердце к свободе летит.

Последнее слово "прощай" уже,
Мне нужно только это,
Прощай уже, сколько терпел,
Сегодня надежду разорвал,
Прощай,
Прощай,
Прощай, и будь счастлива!

Будет ли сожаление,
Почему раньше не сказал?
С тобой всегда мечтаю?
Видеть тебя мне трудно,
Только незаметны мои слёзы.

Последнее слово "прощай" уже,
Мне нужно только это,
Прощай уже, сколько терпел,
Сегодня надежду разорвал,
Прощай,
Прощай,
Прощай, только будь счастлива!
```
