# Оригинал

### Айдар Шакиров - Экълимэ

```
Син минем ягыма карамыйсын да,
Син минем турыда уйламыйсын да.
Карама, уйлама, эндэшмэ, Экълимэ, —
Барыбер син бит уемда!

Йорэгем жылынып китэ,
Елмаеп карасан уземэ.
Карама, уйлама, эндэшмэ, Экълимэ,
Син генэ минем уемда!

Барыбер кар эрер, су булып китэр,
Яз коне йорэкнен кылларын чиртер.
Мин котэм сагынып, кил минем яныма,
Гузэлем минем, Экълимэ!

Йорэгем жылынып китэ,
Елмаеп карасан уземэ.
Карама, уйлама, эндэшмэ, Экълимэ,
Син генэ минем уемда!
```

# Перевод

### Айдар Шакиров - Эклиме

```
Ты на меня не смотришь,
И обо мне не думаешь.
Не смотри, не думай, не говори, Эклиме, —
Всё равно ты в моих мыслях!

Сердце моё согревается,
Когда улыбаясь смотришь на меня.
Не смотри, не думай, не говори, Эклиме,
Только ты в моих мыслях!

Всё равно снег растает, водой станет,
Весенний день сердечные струны затронет.
Я жду, скучая, приди ко мне,
Красавица моя, Эклиме!

Сердце моё согревается,
Когда улыбаясь смотришь на меня.
Не смотри, не думай, не говори, Эклиме,
Только ты в моих мыслях!
```
