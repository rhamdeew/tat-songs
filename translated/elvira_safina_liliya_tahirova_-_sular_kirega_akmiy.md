# Оригинал

### Эльвира Сафина / Лилия Таһирова - Сулар кирегә акмый

```
Ялганмый юллар, ялганмый уйлар,
Юлларда - күзем. Сүнгән бар утлар.
Кайдан эзләргә? Кайда эзләриң?
Язмыш ник сукты, ник сукты тезләргә?

Сулар кирегә акмый,
Сулар кирегә акмый, беләм мин,
Язмышымда язганны,
Язмышымда язганны күрәм мин.
Назлы сүзләрең синең йөрәгемнең түрендә генә,
Уйларымда сөйләшәм мин синең белән әле дә.

Салдың суларга, салдың утларга,
Аерылды юллар. Ник көтәм инде?
Сиңа елмайган кебек елмаймам,
Үттеләр уртак язлар. Мин бүтән инде...

Сулар кирегә акмый,
Сулар кирегә акмый, беләм мин,
Язмышымда язганны,
Язмышымда язганны күрәм мин.
Назлы сүзләрең синең йөрәгемнең түрендә генә,
Уйларымда сөйләшәм мин синең белән әле дә.

Эзлим җәйләрдән, эзлим кышлардан,
Сине күрмичә ничек түзәрмен?
Таба алмасам сагынуга дәва,
Безнең сукмаклардан кат-кат үтәрмен.

Сулар кирегә акмый,
Сулар кирегә акмый, беләм мин,
Язмышымда язганны,
Язмышымда язганны күрәм мин.
Назлы сүзләрең синең йөрәгемнең түрендә генә,
Уйларымда сөйләшәм мин синең белән әле дә.
```
