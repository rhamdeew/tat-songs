# Оригинал

### Дилә Булгакова / Артур Туктаголов - Әткәй йорты (беренче вариант)

```
Әткәйле йорт - атлы йорт,
Төзек ипле, маллы йорт.
Әткәйле йорт - кунакчыл йорт,
Бишекле йорт, баллы йорт.
Әткәйле йорт - җылы йорт,
Учагыңда сүнмәс ут.
Арбалы йорт, чалгылы йорт,
Бу йорттан һич китмәс кот.

Гореф-гадәтен саклаган,
Нигезле йорт, нурлы йорт.
Балалы йорт, курайлы йорт,
Тальян моңлы, җырлы йорт.

Изге тупсаң каршы алган
Табынлы йорт, туганчыл йорт.
Фатыйхасын биреп калган,
Бисмиллалы газиз йорт.

Гореф-гадәтен саклаган,
Нигезле йорт, нурлы йорт.
Балалы йорт, курайлы йорт,
Тальян моңлы, җырлы йорт.
Әткәйле йорт, нурлы йорт.
```

# Перевод

### Диля Булгакова / Артур Туктаголов - Отцовский дом (первый вариант)

```
Отцовский дом - дом с конями,
С крепкой нитью, со скотиной.
Отцовский дом - гостеприимный дом,
С колыбелью дом, медовый дом.
Отцовский дом - тёплый дом,
В очаге твоём не гаснет пламя.
Дом с повозкой, дом с санями,
Из этого дома не уйдёт счастье.

Обычаи свои хранящий,
Основа-дом, светлый дом.
Дом с детьми, дом с кураем,
С гармонной песней, с музыкой дом.

Святых гостей встречавший,
Дом молитвенный, родной дом.
Благословение давший,
С Бисмиллой священный дом.

Обычаи свои хранящий,
Основа-дом, светлый дом.
Дом с детьми, дом с кураем,
С гармонной песней, с музыкой дом.
Отцовский дом, светлый дом.
```
