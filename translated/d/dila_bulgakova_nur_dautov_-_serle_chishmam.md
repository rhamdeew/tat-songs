# Оригинал

### Дилә Булгакова / Нур Даутов - Серле чишмәм

```
Чишмәнең татлы суларын
Ятып, йотлыгып эчәм.
Ни хәлдә яшәдең әле
Серле сердәшем - чишмәм?

Сагындым, чишмәм, сагындым,
Сагынганда киләмен.
Бала чагым синдә калды,
Сизмәдем шул ул үткәнен.

Чишмәм минем, серле чишмәм,
Гөл бакча сукмаклары.
Әниләр дә хәтерлиләр
Килен булган чакларын.

Күчтәнәчкә суын алдым,
Чиләкләрне тутырып.
Тагы бер, бер елыйм әле,
Яннарыңа утырып.
```

# Перевод

### Дилә Булгакова / Нур Даутов - Мой сокровенный родник

```
Сладкую воду родника
Пью, припадая, жадно глотая.
Как ты живёшь сейчас,
Мой сокровенный друг - родник?

Скучала я, родник мой, скучала,
Когда скучаю - прихожу.
Моё детство с тобой осталось,
Не заметила я, как оно прошло.

Родник мой, сокровенный родник,
Тропинки цветущего сада.
И матери помнят
Времена, когда невестами были.

Набрала воды в вёдра,
До краёв наполнив.
Ещё разок, разок поплачу,
Сидя рядом с тобой.
```
