# Оригинал

### Динә Талхина / Роза Сәхәүетдинова - Сагынырсың әле, сагынырсың

```
Сагынырсың әле мине, сагынырсың
Ярсу язның ташкынында.
Ташканында синең хисләрең,
Мин булмамын яннарыңда.
Нишләрсең һин, нишләрсең?

Сагынырсың әле мине, сагынырсың
Ямьле җәйнең таңнарында.
Таңда кошлар өзәр үзәкне.
Мин булмамын яннарыңда,
Үткәннәрне булмас төзәтеп.

Сагынырсың әле мине, сагынырсың
Күркәм көзнең кырларында.
Көзнең яме җанны кыер.
Мин булмамын яннарыңда,
Бер йөрәккә сагыш ничек сыер?

Сагынырсың әле мине, сагынырсың
Салкын кышның буранында.
Моңсуланыр синең карашың.
Мәңгелеккә безнең сөюебез
Бураннарда йөрер адашып.
```

# Перевод

### Дина Талхина / Роза Сахаутдинова - Скучаешь ли, скучаешь

```
Скучаешь ли ты по мне, скучаешь
В бурном разливе весны.
Переполнены твои чувства,
Но меня нет рядом с тобой.
Что же ты делаешь, что же?

Скучаешь ли ты по мне, скучаешь
В прекрасных зорях лета.
На заре птицы тревожат душу.
Меня нет рядом с тобой,
Прошлого не исправить никак.

Скучаешь ли ты по мне, скучаешь
На красивых просторах осени.
Осенняя грусть терзает душу.
Меня нет рядом с тобой,
Как же тоску из сердца вырвать?

Скучаешь ли ты по мне, скучаешь
В холодной зимней метели.
Грустью наполнен твой взгляд.
Навечно наша любовь
Блуждает, заплутав в метелях.
```
