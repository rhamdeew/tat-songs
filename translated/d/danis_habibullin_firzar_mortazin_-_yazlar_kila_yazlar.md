# Оригинал

### Данис Хәбибуллин / Фирзәр Мортазин - Язлар килә, язлар

```
Язлар килә, язлар,
Кайта киек казлар.
Су буйлары ямьгә күмелә.

Э Син,иркәм,нигә
Кайтмыйсың бирегә,
Язлар китермисең күңелгә?

Майлар җитә,майлар,
Көтеп алган айлар,
Ак чәчәккә чума бакчалар.

Бик шат булыр идем
Хисләрем дә минем
Шушы язда чәчәк атсалар.

Еллар уза, еллар
Гуя аккан сулар,
Аккан сулар, алар тин безгэ

Айлап аккан сулар,
утеп озын юллар
Кушылырлар эле дингезгэ
Дингездэге судай
утеп озын юллар
Кушылырбыз эле без бергэ
```

# Перевод

### Данис Хәбибуллин / Фирзар Мортазин - Вёсны приходят, вёсны

```
Вёсны приходят, вёсны,
Возвращаются вновь гуси.
Берега рек в зелень одеты.

А ты, мой милый, почему
Не возвращаешься ко мне,
Не приносишь весну в душу?

Май наступает, май,
Месяцы ожидания,
Сады белым цветом укрыты.

Так счастлива была бы я,
Если б чувства мои
Этой весной расцвели.

Годы проходят, годы
Словно бегущие воды,
Бегущие воды, они подобны нам

Месяцами текущие воды,
Преодолев долгий путь
Впадают всё же в море
Как воды в море,
Пройдя долгий путь
Соединимся всё же мы вместе
```
