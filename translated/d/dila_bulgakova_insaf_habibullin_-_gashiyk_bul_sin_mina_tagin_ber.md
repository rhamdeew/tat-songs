# Оригинал

### Дилә Булгакова / Инсаф Хәбибуллин - Гашыйк бул син миңа тагын бер

```
Офыкларны нурлап балкып киткән
Салават күпере күк бу гомер -
Көннәрне серле итәм дисәң,
Гашыйк бул син миңа тагын бер.

Ак көннәрне төннәр алыштыра,
Ай яктысы уртак, таңы бер -
Елларыңны җырлы итәм дисәң,
Гашыйк бул син миңа тагын бер.

Син сулаган һава - минем һава,
Без яшәгән гомер, бары бер -
Үткәннәрең үкенмәслек булсын дисәң,
Гашыйк бул син миңа тагын бер.
```

# Перевод

### Дилә Булгакова / Инсаф Хәбибуллин - Влюбись в меня ещё раз

```
Горизонты озаряя, засиял
Мост Салавата - небо этой жизни.
Если хочешь сделать дни волшебными,
Влюбись в меня ещё раз.

Светлые дни сменяют ночи,
Лунный свет общий, рассвет один.
Если хочешь сделать годы песенными,
Влюбись в меня ещё раз.

Воздух, которым ты дышишь - мой воздух,
Жизнь, что мы прожили - всё едино.
Чтобы о прошлом не жалеть,
Влюбись в меня ещё раз.
```
