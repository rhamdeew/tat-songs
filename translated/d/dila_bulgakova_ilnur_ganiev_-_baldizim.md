# Оригинал

### Дилә Булгакова / Илнур Ганиев - Балдызым

```
1. Кырлай урманнары серле әкият,
Эчләренә кермә ялгызың.
Зәңгәр күзләреңнән нур бөркелә,
Кем бәхетенә үстең балдызым.

Чәчләреңә кунган ал калфагың,
Чиккән читек минем бүләгем.
Килешәдер ай-һай зифа буйларыңа,
Балитәкле батист күлмәгең.
Газиз балдыз Гөлйөзем,
Батыр Кырлай егетенә
Димләрмен әле үзем!

2. Толымыңда талир тәңкә чыңлый,
Чишмәләрдән сулар алганда.
Кемгә насыйп яр булырсың дим,
Юл буеннан карап калганда.
```

# Перевод

### Диля Булгакова / Илнур Ганиев - Свояченица

```
1. Кырлайские леса - волшебная сказка,
Не входи в них одна.
Из синих глаз твоих льётся свет,
Чьим счастьем станешь ты, свояченица моя.

Красный калфак на твоих волосах,
Вышитые читек - мой подарок тебе.
Как же идёт твоей стройной фигуре
Батистовая рубашка с кружевами.
Дорогая свояченица Гульюзум,
Храбрецу из Кырлая
Я сам тебя отдам!

2. В подоле твоём звенят монеты,
Когда воду из родников берёшь.
Кому же судьбою станешь ты суженой,
Думаю я, глядя с дороги.
```
