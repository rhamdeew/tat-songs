# Оригинал

### Дина Камалетдинова / Фирзәр Мортазин - Танырсыңмы

```
Сагынуларым ярсу диңгез кебек,
Ярларыннан ташый ургылып.
Танырсыңмы таннар аткан чакта,
Яңгырасам моңлы җыр булып.

Сусаганда җыеп эчэрсеңме,
Чык тамчысы булсам чирәмдә.
Кумассыңмы ак күбәләк булып,
Уйнап очсам синең тирәңдә.

Танырсыңмы йөргән юлларыңда,
Чәчәк атсам язгы гөл булып.
Авырлыклар килсә чакыр мине,
Мин барырмын назлы җил булып.
```

# Перевод

### Дина Камалетдинова / Фирзәр Мортазин - Узнаешь ли

```
Моя тоска подобна бурному морю,
Бьётся волнами о берега.
Узнаешь ли меня на рассвете,
Когда зазвучу печальной песней.

Соберёшь ли и выпьешь меня,
Если стану росинкой на лице земли.
Не отмахнёшься ли, когда белой бабочкой
Буду играя кружить возле тебя.

Узнаешь ли меня на твоих дорогах,
Когда расцвету весенним цветком.
Если придут невзгоды - позови меня,
Я приду к тебе нежным ветром.
```
