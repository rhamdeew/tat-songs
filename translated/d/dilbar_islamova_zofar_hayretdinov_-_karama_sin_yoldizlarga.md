# Оригинал

### Дилбәр Исламова / Зөфәр Хәйретдинов - Карама син йолдызларга

```
Карама син йолдызларга,
Алар бит берни белми.
Сөюен үзе белмәгән
Йолдыздан беләмени?!

Ә йолдызлар, йолдызлар,
Сүнәләр дә балкыйлар.
Җирдәге гашыйклар серен
Алар бары саклыйлар.

Карама син йолдызларга,
Нәрсәдер өмет итеп.
Күз сал, сылу, янәшәңдә
Бер егет тора көтеп.

Ә йолдызлар, йолдызлар,
Сүнәләр дә балкыйлар.
Җирдәге гашыйклар серен
Алар бары саклыйлар.

Карама син йолдызларга,
Сорау бирмә аларга.
Тик йөрәгең юл күрсәтер
Мәхәббәтең табарга.

Ә йолдызлар, йолдызлар,
Сүнәләр дә балкыйлар.
Җирдәге гашыйклар серен
Алар бары саклыйлар.
```

# Перевод

### Дильбар Исламова / Зуфар Хайретдинов - Не смотри на звёзды

```
Не смотри ты на звёзды,
Они ведь ничего не знают.
Разве узнает звезда,
Что сама не познала любви?!

Ах, звёзды, звёзды,
То гаснут, то сияют.
Тайны земных влюблённых
Они лишь хранят.

Не смотри ты на звёзды,
На что-то надеясь.
Посмотри, красавица, рядом с тобой
Стоит и ждёт один юноша.

Ах, звёзды, звёзды,
То гаснут, то сияют.
Тайны земных влюблённых
Они лишь хранят.

Не смотри ты на звёзды,
Не задавай им вопросов.
Только сердце укажет путь,
Чтобы найти твою любовь.

Ах, звёзды, звёзды,
То гаснут, то сияют.
Тайны земных влюблённых
Они лишь хранят.
```
