# Оригинал

### Дамир Гарифуллин / Фәрит Хатыйпов - Арча таулары

```
Арча, диләр, Арча, диләр,
Арча биек тауларда.
Язмыш адәм балаларын
Йөртә әллә кайларда.

Их... Арча тауларыннан,
Менәлмим алларыннан,
Тормыш җиле аермасын,
Ярларын парларыннан.

Арча тавы матур булып
Күренә алсу таңнарда.
Сөю гөле бер кайчан да,
Сулмый гашыйк җаннарда.
```

# Перевод

### Дамир Гарифуллин / Фарит Хатыпов - Горы Арчи

```
Арча, говорят, Арча, говорят,
Арча в высоких горах.
Судьба детей человеческих
Ведёт неведомо куда.

Ох... С гор Арчи,
Не взойду я на вершины,
Пусть ветер жизни не разлучит
Влюблённых с их половинками.

Гора Арча прекрасная
Видна в алых зорях.
Цветок любви никогда
Не увянет в сердцах влюблённых.
```
