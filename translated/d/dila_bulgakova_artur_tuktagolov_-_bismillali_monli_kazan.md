# Оригинал

### Дилә Булгакова / Артур Туктаголов - Бисмиллалы моңлы Казан

```
Күлмәгең ак, күңелең пакъ,
Түбәтәең чигүле.
Чит итмисең, үз итәсең,
Дус итәсең син һәр кемне.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.

Саклап калдың, аклап калдың,
Гасырларның чал тарихын.
Бармагыңда, көмеш балда,
Хат догалы Кол Шәрифең.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.

Нурлы булсын туган көнең,
Мәчетләрдән килә азан.
Намазыңны калдырмадым
Бисмиллалы моңлы Казан.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.
```

# Перевод

### Диля Булгакова / Артур Туктаголов - Казань с Бисмиллой певучая

```
Рубаха твоя бела, душа чиста,
Тюбетейка твоя вышита.
Не чужим считаешь, своим считаешь,
Другом делаешь ты каждого.

Благословен день твоего рожденья,
Вечно живи, великая Казань.
На тысячелетних небесах твоих
Начертано имя Всевышнего.

Ты сохранила, защитила
Вековую седую историю.
На пальце твоём, на серебряном перстне,
Молитва начертана Кул-Шарифа.

Благословен день твоего рожденья,
Вечно живи, великая Казань.
На тысячелетних небесах твоих
Начертано имя Всевышнего.

Пусть светел будет день твоего рожденья,
Из мечетей звучит азан.
Не оставила ты намаза,
Казань певучая с Бисмиллой.

Благословен день твоего рожденья,
Вечно живи, великая Казань.
На тысячелетних небесах твоих
Начертано имя Всевышнего.
```
