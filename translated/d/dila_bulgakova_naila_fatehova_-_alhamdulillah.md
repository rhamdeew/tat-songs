# Оригинал

### Дилә Булгакова / Наилә Фатехова - Әлхәмдүлилләһ

```
Бисмилласыз уянган юк,
Әйтәмен көн дә.
Таңымны башлыйм намаздан,
Әлхәмдүлилләһ.

Тәһарәтләр алу фарыз -
Пакьләнәм көн дә.
Ак намазлыгым җәелгән -
Әлхәмдүлилләһ.

Чал чәчемдә ак яулыгым
Төшкән иңемә.
Хаҗдан кайткан бүләк күлмәк,
Әлхәмдүлилләһ.

Кулымда нәнәм дисбесе,
Дога телемдә.
Өстәлемдә ризык тулы,
Әлхәмдүлилләһ.

Биш вакыт намаз укыгач
Бәхетле кем дә.
Таңымны башлыйм намаздан,
Әлхәмдүлилләһ.
```

# Перевод

### Дилә Булгакова / Наилә Фатехова - Альхамдулиллях

```
Не просыпаюсь без Бисмиллы,
Говорю я каждый день.
Начинаю утро с молитвы,
Альхамдулиллях.

Совершать омовение - долг,
Очищаюсь каждый день.
Белый молитвенный коврик расстелен,
Альхамдулиллях.

На седых волосах белый платок
Ниспадает на плечи.
Платье - подарок из хаджа,
Альхамдулиллях.

В руках моих бабушкины чётки,
Мольба на устах.
На столе полна благодати,
Альхамдулиллях.

Пять раз в день прочитав намаз,
Счастлив каждый.
Начинаю утро с молитвы,
Альхамдулиллях.
```
