# Оригинал

### Дилә Булгакова / Нур Даутов - Баллы Башкортстаным (икенче вариант)

```
Башкортстаным кунакчыл як,
Ул безнең изге кала.
Баллап кына чәйләр эчсәң,
Тәнеңә шифа була.

Яшәртә дә, давалый да
Гәрәбәләй балабыз.
Юкә балы, чәчәк балы
Күчтәнәчкә алыгыз.

Кәрәзле бал - уңган төяк,
Данлы төяк - даныбыз.
Бүләкләргә лаек булды
Башкортстан балыбыз.

Медальларга лаек булды
Башкортстан балыбыз.
Кәрәзле бал - уңган төяк,
Таралган бит даныбыз.
```

# Перевод

### Диля Булгакова / Нур Даутов - Медовый мой Башкортостан (второй вариант)

```
Башкортостан мой - гостеприимный край,
Он наша священная крепость.
Если чай с медом пить,
Телу твоему исцеление будет.

Омолаживает и лечит
Чудесный наш мед.
Липовый мед, цветочный мед -
Для здоровья возьмите.

Мед с малиной - удачный сорт,
Славный сорт - наша слава.
Наград достоин был
Башкортостана мед.

Медалей достоин был
Башкортостана мед.
Мед с малиной - удачный сорт,
Разнеслась ведь слава наша.
```
