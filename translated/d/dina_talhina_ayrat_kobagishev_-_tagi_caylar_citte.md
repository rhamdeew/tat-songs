# Оригинал

### Динә Талхина / Айрат Кобагышев - Тагы җәйләр җитте

```
Җылы җәйнең яме -
Җәйгор балкып яна.
Җиргә яңгыр түгел,
Җиргә гәүһәр тама.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Ак чәчәкләр атты аланда.
Чалгы тотып, күз алдында
Син торасың һаман да.

Җылы җәйнең җаны -
Сандугачкай өзә үзәкне.
Җәй тугайларга чигә
Асылларын бизәкнең.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Бер туктаусыз кәккүк чакыра,
Сиңа еллар санаганда
Сакауланган, бахырың.

Җылы җәйнең тәме -
Җир җиләге бал гына.
Сибелеп кенә үскән,
Чиләгеңне ал гына.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Акланкайлар шау гөл генә.
Кайтмаска киттең син,
Җәйләр килер гел генә.
```

# Перевод

### Дина Талхина / Айрат Кобагышев - Снова лето пришло

```
Теплого лета краса -
Радуга светом сияет.
Не дождь на землю идёт,
Жемчугом с неба стекает.

Ах, ах,
Снова лето пришло,
Белых цветов расцвела поляна.
Седину в волосах затая,
Ты стоишь предо мной, как и прежде.

Теплого лета душа -
Соловей щебечет сердцу.
Летние рощи дарят
Драгоценных камней ожерелье.

Ах, ах,
Снова лето пришло,
Без устали кукушка кукует,
Когда считаю годы твои,
Замираю, грущу я.

Теплого лета вкус -
Земляника сладка, как мёд.
Рассыпалась средь травы,
Бери ягоды - вот.

Ах, ах,
Снова лето пришло,
Акации шумят в цвету.
Ты ушёл безвозвратно,
А лето приходит всё так же.
```
