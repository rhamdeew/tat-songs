# Оригинал

### Даһи Бабичев / Рафик Сәлмәнов - Яз җыры (икенче вариант)

```
Яфрак яра чуклы каен,
Татлы исен сибеп,
Бөреләрен ачты гөлләр,
Язгы тынны сизеп.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.

Чәчәк ата туган ягым,
Яшәрделәр кырлар.
Якты кояш чәчә нурын,
Шаулый (яңгырый) дәртле җырлар.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.

Гөрләп илем каршы ала
Хезмәт белән көнне.
Алга, данлы яшьлек, алга,
Иҗат көтә безне.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.
```

# Перевод

### Дахи Бабичев / Рафик Салманов - Весенняя песня (второй вариант)

```
Листья распускает пышная берёза,
Сладкий аромат разлив,
Раскрыли бутоны цветы,
Весеннее дыханье ощутив.

Весна, весна пришла
В наш край, друзья.
Восхваляя весну,
Поздравляя нас,
Поют птицы.

Цветами покрывается родная сторона,
Зазеленели поля.
Ясное солнце льёт свой свет,
Звучат вдохновенные песни.

Весна, весна пришла
В наш край, друзья.
Восхваляя весну,
Поздравляя нас,
Поют птицы.

Громко край мой встречает
День трудом своим.
Вперёд, славная молодость, вперёд,
Творчество нас ждёт.

Весна, весна пришла
В наш край, друзья.
Восхваляя весну,
Поздравляя нас,
Поют птицы.
```
