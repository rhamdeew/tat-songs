# Оригинал

### Дилә Булгакова / Дамир Әделбаев - Дуслар белән

```
Шатлыкларны тыныч (шашып) каршыладым,
Кайгыларны яшьсез күтәрдем.
Яннарымда дуслар булгангамы,
Тоймаганмын еллар үткәнен.

Сизмәгәнмен, көзге җилләр килеп,
Чәчләремә бәсләр (кырау) сипкәнен.
Аңладым, тик (мин) дуслар белән генә
Йөрәгемнең япь-яшь икәнен.

Еллар дулкынында чайкалганда
Сез бар чакта дуслар, югалмам!
Барысына түзеп булыр сымак,
Ялгызлыкка гына сыдамам (түзә алмам).
```

# Перевод

### Диля Булгакова / Дамир Аделбаев - С друзьями

```
Радости встречала я спокойно,
Горе переносила без слёз.
Пока рядом были друзья,
Не замечала, как годы прошли.

Не почувствовала, как осенние ветры
Иней на волосы мои рассыпали.
Поняла я, что только с друзьями
Моё сердце молодо и свежо.

Когда качает меня волна лет,
Пока вы есть, друзья, не пропаду!
Ко всему, кажется, можно привыкнуть,
Только одиночества не вынесу.
```
