# Оригинал

### Дамир Гыйсметдин / Зөфәр Хәйретдинов - Ярты

```
Авылларда киң таралган
Исеме аның «Ярты».
Ансыз яши алмый бүген,
Ни яше дә, ни карты.

Печәнен дә әзерләтә,
Онын кайтарта тартып.
Утынын кистереп куя -
Эшне тиз тота ярты.

Ифрат зур ярдәме тия
Әби һәм бабайларга.
Яхшысын да, яманын да
Ярты чыга майларга.

Күпме көрәшсә дә влач,
Җиңәлмәде яртыны.
Инде хәзер хөрмәт итә
Ихтыяҗын халкының.

Ярты-йорты дип көлмәгез,
Әйләнеп бара бергә,
Имән кебек егетләрне
Күгәреп сылый җиргә (Җиңеп аудара җиргә).

Ярдәме аның бер көнлек,
Зыянын кем санаган.
Соңгы юлга шуның үзен
Озатыйк безнең арадан.
```

# Перевод

### Дамир Гыйсметдин / Зуфар Хайретдинов - Ярты

```
В деревнях широко распространён,
Имя его - «Ярты».
Без него не живут сегодня,
Ни стар, ни млад.

И печь он готовит,
Муку просеивая, веет.
Дрова нарубит впрок -
Дело быстро делает ярты.

Огромна помощь его
Бабушкам и дедушкам.
И хорошее, и плохое
Ярты на масло пустит.

Сколько ни боролся врач,
Не победил ярты.
Теперь уже уважает
Нужды народа своего.

Не смейтесь над ярты,
Он кружась идёт,
Дубовых молодцев
Кувырком валит на землю.

Помощь его - на день,
А вред кто считал.
В последний путь его самого
Проводим из наших рядов.
```
