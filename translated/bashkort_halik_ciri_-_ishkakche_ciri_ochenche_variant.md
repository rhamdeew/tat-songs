# Оригинал

### Башкорт халык җыры - Ишкәкче җыры (өченче вариант)

```
Агыйделкәйләрне, ай кичкәндә,
Бер җырладым ишкәк ишкәндә,
Бер җырладым, дуслар, бер еладым
Туган илем искә төшкәндә лә.

Агыйделдән актым, ай, сал белән
Сал бәйләдем сыры тал белән (лә).
Ахирәтләр күрке, ай, мал белән
Дөнья күрке сөйгән яр белән (лә).
```

# Перевод

### Башкирская народная песня - Песня рыбака (третий вариант)

```
Спустился к речке Агидель в лунную ночь,
Запел я песню рыбака в тишине,
Запел я песню, друзья, и заплакал,
Когда вспомнил родную землю.

Спустился по Агидели на лодке,
Привязал лодку к ивовой ветке.
Ахирет мне видится домом с хозяйством,
Мир мне видится с любимой возлюбленной.
```
