# Оригинал

### Салават Фатхетдинов - Шэжэрэ

```
Жиде буын бабамнарнын исеме югалмасын,
Зур бабамнар тырышыплар тозегэн шэжэрэсен.
Кемнэр кайчан туган, улгэн, кайсы гасыр яшэгэн,
Барысын да тарих итеп ак кэгазьгэ теркэгэн.

Шэжэремне алга салып, Коръэнемне кулга алып,
Ихластан кылам дога.
Улгэннэр тыныч йокласын, исэннэр жырлап яшэсен,
Нэсел ыругыма берук бэрэкэт бир син Ходай.

Иллэр туар болыннарга ядъкарь булсын шэжэрэ,
Якынайсын бар туганнар, ерагаймасын ара.
Ходай безгэ узе биргэн газиз якын туганны,
Кеше бердэ ятим тугел, янда булса туганы.

Шэжэремне алга салып, Коръэнемне кулга алып,
Ихластан кылам дога.
Улгэннэр тыныч йокласын, исэннэр жырлап яшэсен,
Нэсел ыругыма берук бэрэкэт бир син Ходай.
```

# Перевод

### Салават Фатхетдинов - Родословная

```
Пусть не исчезнут имена семи поколений предков,
Прадеды старались составить родословную.
Кто когда родился, умер, в каком веке жил,
Всё как историю записали на белой бумаге.

Выложив родословную, взяв Коран в руки,
Искренне творю молитву.
Пусть умершие спокойно покоятся, живые пусть живут, воспевая,
Моему роду-племени дай благословение, Всевышний.

Пусть родословная станет памятью для будущих поколений,
Пусть все родные сблизятся, не отдалятся друг от друга.
Бог нам самим дал дорогих близких родных,
Человек не одинок, если рядом есть родня.

Выложив родословную, взяв Коран в руки,
Искренне творю молитву.
Пусть умершие спокойно покоятся, живые пусть живут, воспевая,
Моему роду-племени дай благословение, Всевышний.
```
