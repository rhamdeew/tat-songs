# Оригинал

### Габдельфат Сафин - Шимберем

```
Шимберемдэ тан сызылган чакта
Тоннен пэрдэлэрен каралап,
Жэйрэп яткан болын ослэреннэн
Ак мамыктай томан тарала.

Болыннарны инлэп Кашык ага (?),
Суга иелеп усэ таллары,
Кагылдашып кайткан кыр казларын
Каршы ала анын ярлары.

Шимбер имэннэре шаулавында
Узэк озгеч монлы тавыш бар,
Сандугачлар коен чыгарганнар,
Э сузлэрен язган камышлар.

Сина булган кайнар сою жырын
Кунелемдэ йортэм, Шимберем.
Сагынганда кайтып сыенырлык
Изге газиз жирем син минем.
```

# Перевод

### Габдельфат Сафин - Мой Шимбер

```
Когда рассвет проглядывал в Шимбере,
Раздвигая ночные завесы,
С лугов, где мирно паслись стада,
Белый туман расстилался, как вата.

Луга стонали от Кашык-реки,
К воде склонялись ивы гибкие,
Диких гусей, что с шумом летели,
Встречали радостно берега.

В шуме дубов шимберских
Есть особенный печальный голос,
Соловьи мелодии выводят,
А камыши слова записывают.

Пламенную песню любви к тебе
В душе храню я, мой Шимбер.
Когда скучаю - возвращаюсь к радости,
Священная дорогая земля моя.
```
