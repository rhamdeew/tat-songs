# Оригинал

### Әхмәт Фәйзи / Салих Сәйдәшев - Уянсыннар кырлар

```
Дәртле җырлар агыла киң күкрәктән,
Ташып чыга дулкын шикелле.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Таш мае иде дәртләр ташкын сыман,
Туган кырлар җанга сөйкемле.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Эшкә бара колхоз батырлары,
Җәелеп ята иркен басулар.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Эшкә бара колхоз матурлары,
Дәртлеләр һәм эштә ярсулар.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!
```

# Перевод

### Ахмат Файзи / Салих Сайдашев - Пробудитесь, поля

```
Душевные песни льются из широкой груди,
Волной переливаясь через край.

Пробудитесь, поля,
Пусть поют богатыри,
Пусть звучит по земле
Голос труда!

Камнем были печали, словно бурный поток,
Родные поля душе так милы.

Пробудитесь, поля,
Пусть поют богатыри,
Пусть звучит по земле
Голос труда!

На работу идут колхозные герои,
Широко раскинулись вольные поля.

Пробудитесь, поля,
Пусть поют богатыри,
Пусть звучит по земле
Голос труда!

На работу идут колхозные красавицы,
С душой трудятся, в работе молодцы.

Пробудитесь, поля,
Пусть поют богатыри,
Пусть звучит по земле
Голос труда!
```
