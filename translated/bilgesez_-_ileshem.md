# Оригинал

### Билгесез - Илешем

```
Илеш киле, Илеш җире,
Чишмә дә, инеш кенә.
Чишмәләрнең, инешләрнең
Сулары көмеш кенә.

Дөньядагы гүзәл төбәк
Син генә, Илеш кенә.
Кал син, Илеш, шушы килеш,
Бәхетле килеш кенә.

Илеш җире икмәк кенә,
Җиләк тә җимеш кенә.
Халкының да яратканы
Хезмәт кенә, эш кенә.

Дөньядагы гүзәл төбәк
Син генә, Илеш кенә.
Кал син, Илеш, шушы килеш,
Бәхетле килеш кенә.

Илеш киле бик кечкенә,
Таныш та белеш кенә.
Дуслары да гашыйк итәр
Бер генә күреш кенә.

Дөньядагы гүзәл төбәк
Син генә, Илеш кенә.
Кал син, Илеш, шушы килеш,
Бәхетле килеш кенә.
```

# Перевод

### Билгесез - Илеш

```
Илеш приди, Илеш земля,
Источники и ручьи.
Источников и ручьев
Воды серебряные.

Краше в мире нету края
Только ты, Илеш.
Оставайся, Илеш, такой,
Счастливой, только такой.

Земля Илеш хлеб родной,
Клубника и плоды.
Любимое народом
Труд и работа.

Краше в мире нету края
Только ты, Илеш.
Оставайся, Илеш, такой,
Счастливой, только такой.

Илеш краше мала,
Знаю я с детства.
Даже друзья влюбляются
В одну лишь встречу.

Краше в мире нету края
Только ты, Илеш.
Оставайся, Илеш, такой,
Счастливой, только такой.
```
