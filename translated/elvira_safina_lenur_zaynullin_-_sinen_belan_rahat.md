# Оригинал

### Эльвира Сафина / Ленур Зәйнуллин - Синең белән рәхәт

```
Ай-йолдызлар бүләк итмә әле,
Нурын сипсен алар күпләргә.
Мәхәббәтең, назлы карашларың
Алып менсен мине күкләргә.

Синең белән шундый рәхәт миңа.
Карашыңнан назлар түгелә.
Сөюеңне тоеп яшәгәнгә,
Бер елмая күңел, бер көлә.

Аңларсыңмы мине, җаныкаем,
Шатлык яшен тоя алмасам.
Сүзләр беткәч сөю аңлатырга,
Сандугачлар булып сайрасам.

Синең белән шундый рәхәт миңа.
Карашыңнан назлар түгелә.
Сөюеңне тоеп яшәгәнгә,
Бер елмая күңел, бер көлә.

Булсаң иде һәрчак яннарымда,
Сулмасын дип синсез чәчәгем.
Беләсең бит, йөрәк син дип тибә.
Яшәү түгел синсез яшәвем.

Синең белән шундый рәхәт миңа.
Карашыңнан назлар түгелә.
Сөюеңне тоеп яшәгәнгә,
Бер елмая күңел, бер көлә.
```

# Перевод

```
Пусть луна и звёзды не делятся светом,
Пусть его рассеивают для многих.
Твоя любовь, твой нежный взор
Поднимают меня к небесам.

С тобой мне так легко.
Из твоего взгляда льются нежности.
Живу, ощущая твой поцелуй,
Сердце смеётся, а душа плачет.

Ты понимаешь меня, моя любовь,
Если не чувствую счастья.
Когда слова кончатся, чтобы объяснить поцелуй,
Я буду щебетать птичкой.

С тобой мне так легко.
Из твоего взгляда льются нежности.
Живу, ощущая твой поцелуй,
Сердце смеётся, а душа плачет.

Будь же всегда рядом со мной,
Чтобы мой цветок не увял без тебя.
Ведь ты знаешь, сердце бьётся только для тебя.
Жизнь без тебя не жизнь.

С тобой мне так легко.
```
