# Оригинал

### Чәчкә / Наил Табанаков - Сөйлә сереңне

```
Күзләреңнән карап беләм:
Нидер әйтәсең килә.
Тотма эчеңдә сереңне,
Тыңлармын ипләп кенә.

Сөйлә сереңне, сөйлә,
Сөйлә, сөйлә, яшермә.
Шул чагыңда җиңел була
"Сөйгәнем" дип дәшергә.

Серле карашларың белән
Өзелеп син карама.
"Сөям" диеп сереңне чиш,
Йөрәгемне кадама.

Сөйлә сереңне, сөйлә,
Сөйлә, сөйлә, яшермә.
Шул чагыңда җиңел була
"Сөйгәнем" дип дәшергә.

Һәр сүзеңне көтеп алам
Сереңне чишәр өчен.
Хисләреңне аңлатырга
Җитәрме икән көчең?

Сөйлә сереңне, сөйлә,
Сөйлә, сөйлә, яшермә.
Шул чагыңда җиңелула
"Сөйгәнем" дип дәшергә.
```

# Перевод

### Чачка / Наил Табанаков - Расскажи свой секрет

```
По твоим глазам я вижу:
Ты хочешь что-то сказать.
Не держи в себе секрет,
Я буду слушать внимательно.

Расскажи свой секрет, расскажи,
Расскажи, расскажи, не скрывай.
В этот миг легче будет
Сказать: «Любимый мой».

Не смотри на меня так
Таинственным взглядом.
Раскрой свой секрет словом «люблю»,
Не пронзай мое сердце.

Расскажи свой секрет, расскажи,
Расскажи, расскажи, не скрывай.
В этот миг легче будет
Сказать: «Любимый мой».

Я ловлю каждое слово,
Чтобы раскрыть твой секрет.
Хватит ли у тебя сил
Объяснить свои чувства?

Расскажи свой секрет, расскажи,
Расскажи, расскажи, не скрывай.
В этот миг легче будет
Сказать: «Любимый мой».
```
