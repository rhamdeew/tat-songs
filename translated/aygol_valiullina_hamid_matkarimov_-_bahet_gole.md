# Оригинал

### Айгөл Вәлиуллина / Хәмид Мәткәримов - Бәхет гөле

```
Өебезнең түрләренә
Утырттым бәхет гөле
Сөенечемнән куанып,
Ашкынып йөргән көнен.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.

Бәхетлеләрнең бәхете
Гөлгә дә күчә икән.
Чәчкәләргә бәхет өстәп
Яшәргә язсын, иркәм.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.

Гөлебезнең хуш исеннән
Айнымый йөргән чагым.
Ялкынына сөю кушып,
Саклыйк гаилә учагын.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.
```

# Перевод

### Айгуль Валиуллина / Хамид Маткаримов - Цветок счастья

```
У порога нашего дома
Посадила цветок счастья
В те дни, когда от радости
Переполняло ликование.

Цветок счастья подарила
Любимому возлюбленному.
Цветок счастья цветёт
Только в счастливом доме.

Счастье счастливых
Передаётся и цветам.
Добавляя цветам счастья,
Пусть живём, мой милый.

Цветок счастья подарила
Любимому возлюбленному.
Цветок счастья цветёт
Только в счастливом доме.

От благоухания нашего цветка
Кружится голова.
Добавляя в пламя любовь,
Сохраним очаг семьи.

Цветок счастья подарила
Любимому возлюбленному.
Цветок счастья цветёт
Только в счастливом доме.
```
