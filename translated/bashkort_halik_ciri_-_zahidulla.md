# Оригинал

### Башкорт халык җыры - Заһидулла

```
Чабып кына барып, ат бәйләдем
Әйлекүлкәй буена, камышка.
Татлы йокылардан уянып киттем
Заһидулла дигән тавышка.

Эчкенәең янса, салкын су эч,
Янган йөрәгеңә файда ул.
Төштә генә күреп сөйләшмәсәк,
Былай күрешүләр кайда ул.
```

# Перевод

### Башкирская народная песня - Захидулла

```
Прибежав, привязал коня
К устью реки Айлекуль, к камышам.
Из сладких снов проснулся я
На голос Захидуллы.

Если сердце твое горит, пей холодную воду,
Это пойдёт на пользу твоему пылающему сердцу.
Если не поговорим во сне,
То где такие разговоры?
```
