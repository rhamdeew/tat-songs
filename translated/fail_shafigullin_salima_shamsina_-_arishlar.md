# Оригинал

### Фаил Шәфигуллин / Сәлимә Шәмсина - Арышлар

```
Оныта алмыйм һаман,
Газапланам, янам,
Инде кайда гына йөрмәдем.
Күңелемдә арыш -
Сары сагыш.
Башак шавын ишетәм төннәрен,
Узса җилләр исеп,
Дулкынлана ничек,
Офыкларга җитеп офыктан.
Күпме җырлар оттым
Һәм оныттым,
Тик арышлар җырын онытмам,
Дулкынлана арыш,
Басуларда сагыш,
Көннәр инде көзгә авышкан.
Авылым кырларына
Мин һаман да
Сезне сагынып кайтам, арышлар.
```

# Перевод

### Фаиль Шафигуллин / Салима Шамсина - Волнения

```
Не могу забыть,
Раздражен, страдаю,
Куда только не ходил я.
В сердце моем волнение -
Тоска желтая.
По ночам слышу шуршание хлебов,
Веют ветры прошлого,
Так и бушует,
Достигая берегов в тоске.
Сколько песен спел я
И забыл,
Но песню волнений не забуду,
Бушует волнение,
Тоска на ступенях,
Дни уже склонились к осени.
В поля родного села
Я не перестаю
Возвращаться, тоскуя по вам, волнения.
```
