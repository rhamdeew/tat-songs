# Оригинал

### Әхмәт Ерикәй / Мансур Мозаффаров - Хәерле иртә

```
Әкрен генә таң килә,
Ак челтәрләрен үреп.
Җилкәннәрен киереп,
Илгә яңа ямь биреп…

Таң белән бергә
Ирекле илгә
Яңа җыр килә гөрләп,
Хәерле иртә теләп.

Яңа көн күркәм булсын,
Дәрт тулсын күңелләргә.
Таң шикелле елмаеп,
Кул бир яңа көннәргә!..

Таң белән бергә
Ирекле илгә
Яңа җыр килә гөрләп,
Хәерле иртә теләп.

Алсу йөзле кыз кебек,
Шат кояш күтәрелә.
Һәр яңага сокланып,
Ашкынып йөрәк тибә…

Таң белән бергә
Ирекле илгә
Яңа җыр килә гөрләп,
Хәерле иртә теләп.
```

# Перевод

### Ахмат Ерикей / Мансур Мозаффаров - Доброе утро

```
Тихонько рассвет приходит,
Белые кружева сплетая.
Паруса свои расправив,
Краю новую красу даря…

Вместе с рассветом
В свободный край
Новая песня приходит звеня,
Доброе утро желая.

Пусть новый день прекрасным будет,
Пусть вдохновенье наполнит сердца.
Словно заря улыбаясь,
Протяни руку новым дням!..

Вместе с рассветом
В свободный край
Новая песня приходит звеня,
Доброе утро желая.

Как девушка с нежным лицом,
Радостно солнце встаёт.
Восторгаясь всему новому,
С трепетом сердце бьёт…

Вместе с рассветом
В свободный край
Новая песня приходит звеня,
Доброе утро желая.
```
