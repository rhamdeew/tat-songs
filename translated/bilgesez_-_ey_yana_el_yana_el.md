# Оригинал

### Билгесез - Эй, Яңа ел, Яңа ел

```
Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Килеп җитәр буранлап
Онытмыйча без барын.
Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Шатландырып халыкны,
Килеп торсын ел саен!

Яңа үрләр яулап килгән кебек
Яңа еллар бик тиз килеп җитә.
Яңа сулыш биреп һәр кемгә дә
Яңа еллар бәхетләргә илтә.
Онытмыйча, сине дә, мине дә,
Өләшә ул бәхет шатлыгын да
Әкияттәй серле ул Яңа ел
Әйләндерә хәтта җир шарын да.

Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Килеп җитәр буранлап
Онытмыйча без барын.
Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Шатландырып халыкны,
Килеп торсын ел саен!

Яңа елда гүя яшәрәбез,
Әйләнсәк тә еллар кичәсеңдә.
Шушы бәйрәм генә киләчәккә
Өмет булып яна күңелләрдә.

Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Килеп җитәр буранлап
Онытмыйча без барын.
Эй, Яңа ел, Яңа ел,
Әйләндерә җир шарын,
Шатландырып халыкны,
Килеп торсын ел саен!
```
