# Оригинал

### Альфис Киямов - Гафу ит

```
Тэуге кабат сине
Мин кургэн идем.
Сина гашыйк булып,
Мин йоргэн идем.

Без кавышмадык,
Гафу ит, иркэм,
Бергэ булырга
Язмаган икэн.

Яшьлек артта калды,
Сынмады кунел.
Ачы газаплар да
Булмады тугел.

Без кавышмадык,
Гафу ит, иркэм,
Бергэ булырга
Язмаган икэн.

Синсез яшэу хэзер
Авыр бигрэк тэ.
Бергэ булмасак та,
Син гел йорэктэ

Без кавышмадык,
Гафу ит, иркэм,
Бергэ булырга
Язмаган икэн.
```

# Перевод

### Альфис Киямов - Прости

```
Впервые снова тебя
Я увидел.
В тебя влюбившись,
Я ходил.

Мы не соединились,
Прости, милая,
Быть вместе
Не было суждено.

Молодость позади осталась,
Не сломалась душа.
Горькие муки тоже
Не прошли зря.

Мы не соединились,
Прости, милая,
Быть вместе
Не было суждено.

Без тебя жить теперь
Особенно тяжело.
Хоть и не вместе,
Ты все в сердце

Мы не соединились,
Прости, милая,
Быть вместе
Не было суждено.
```
