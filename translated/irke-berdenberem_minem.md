# Оригинал

### Иркэ - Бердэнберем минем

```
Эллэ инде кунелемэ кучте
Галэмдэге ботен гузэллек
Кон дэ синен килуенне
Соям иркэм диюенне
Котэ кунел шуна бизэлеп
Лэззэт бишегендэ тирбэлеп.

Гомергэ сайлаган ярым
Бердэнберем син минем
Сине тапкач кына курдем
Гузэллеген бу жирнен.

Хыялларым синен белэн бергэ
Хэр мизгелем минем яз кебек
Бэхет сыймый йорэгемэ
Кучэ минем кузлэремэ.
Кузлэремнэн тора боркелеп
Хислэр китэ мэйдан сутелеп.

Гомергэ сайлаган ярым
Бердэнберем син минем
Сине тапкач кына курдем
Гузэллеген бу жирнен.

Сабый кебек тоям уземне мин
Юлым жэннэт булып урелэ.
Кояшка тин карашларын
Жылы язлар синен назын
Синен кочак кебек жирдэ
Минем белэн жыр да соенэ.

Гомергэ сайлаган ярым
Бердэнберем син минем
Сине тапкач кына курдем
Гузэллеген бу жирнен.
```

# Перевод

### Иркэ - Единственный мой

```
Наверное, уже в душу мою переселилась
Вся красота мира,
Каждый день твоего прихода
Жду, любимый, говоря.
Ждёт душа, украшаясь этим,
В колыбели наслаждений покачиваясь.

Избранный на всю жизнь любимый,
Единственный ты мой,
Только найдя тебя, увидела
Красоту этой земли.

Мечты мои с тобою вместе,
Каждый миг мой словно весна.
Счастье не помещается в сердце,
Переливается через мои глаза.
Из глаз моих струясь,
Чувства уходят, поле орошая.

Избранный на всю жизнь любимый,
Единственный ты мой,
Только найдя тебя, увидела
Красоту этой земли.

Как ребёнок чувствую себя,
Путь мой раем стелется.
Подобные солнцу твои взгляды,
Тёплые вёсны, твоя нежность.
На земле, словно в твоих объятиях,
Со мною и песня радуется.

Избранный на всю жизнь любимый,
Единственный ты мой,
Только найдя тебя, увидела
Красоту этой земли.
```
