# Оригинал

### Йолдыз Миңнуллина / Эльмир Низамов - Тукай сүзе

```
Салкын җилләр өметләрне өзеп,
Үзәкләргә үтә язганда,
Казаныма кайта шагыйрь сүзе,
Тукай сүзе кайта Казанга.

Тукай сүзе! Тукай сүзе!
Тукай сүзе кайта Казанга!

Моңнар иңри, сак тынлыкны ерып,
Күңел кылы меңгә теленә.
Туган илгә кайта халык җыры,
Тукай җыры кайта илемә.

Тукай җыры! Тукай җыры!
Тукай җыры кайта илемә!

Мәйданнардан уза дөрли язып -
Кем дә дәртле, кем дә шат анда -
Ватаныма кайта Тукай язы,
Мәңгелек яз кайта Ватанга!

Тукай язы! Тукай язы!
Тукай язы кайта Ватанга!
Тукай язы! Тукай язы!
Тукай язы кайта Ватанга!
Тукай язы! Тукай язы!
Тукай язы кайта Ватанга!..
```

# Перевод

### Йолдыз Миңнуллина / Эльмир Низамов - Слово Тукая

```
Холодные ветры надежды срывают,
В самое сердце весна входит вновь,
В мой Казан возвращается слово поэта,
Слово Тукая вернулось в Казань.

Слово Тукая! Слово Тукая!
Слово Тукая вернулось в Казань!

Плачут напевы, пронзив тишину,
Струна души на тысячу языков.
В родную страну возвращается песня народа,
Песня Тукая вернулась в мой край.

Песня Тукая! Песня Тукая!
Песня Тукая вернулась в мой край!

Через майданы идет, багряно разливаясь -
Кто пылок, кто радостен там -
В мою Отчизну возвращается весна Тукая,
Вечная весна вернулась в Отчизну!

Весна Тукая! Весна Тукая!
Весна Тукая вернулась в Отчизну!
Весна Тукая! Весна Тукая!
Весна Тукая вернулась в Отчизну!
Весна Тукая! Весна Тукая!
Весна Тукая вернулась в Отчизну!..
```
