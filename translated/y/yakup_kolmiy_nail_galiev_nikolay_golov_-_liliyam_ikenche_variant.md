# Оригинал

### Якуп Колмый / Наил Галиев (Николай Голов) - Лилиям (икенче вариант)

```
Язгы күктә, зәңгәр кыңгырау
Нурлар уйный, мин көләм.
Җиремнең бер күрке булып
Чәчәк атты Лилиям.

Лилиям, Лилиям!
Чәчәк атты Лилиям!

Җәйге күктә кара болыт
Яңгырлы яшен көйлә.
Кайда сугылырга белми
Иелә шул Лилиям.

Лилиям, Лилиям!
Иелә шул Лилиям!

Җилләр исәр, болыт күчәр,
Гөлләр җиргә иелә.
Ниди язмыш көтә сине
Җирнең күрке Лилиям?

Лилиям, Лилиям!
Җирнең күрке Лилиям!

Язлар килер, күктәй булыр
Зәңгәр кыңгырау мин беләм.
Йөрәгемдә чәчәк булып
Аткан идем Лилиям.

Лилиям, Лилиям!
Чәчәк ат син, Лилиям!
```

# Перевод

### Якуп Колмый / Наил Галиев (Николай Голов) - Лилиям (второй вариант)

```
В небе вешнем, колокольчик голубой,
Свет играет - я смеюсь.
Краем милой этой земли
Расцвела моя Лилиям.

Лилиям, Лилиям!
Расцвела моя Лилиям!

В небе летнем туча чёрная,
Дождь с грозою запоёт.
Не умея, где удариться,
Клонится моя Лилиям.

Лилиям, Лилиям!
Клонится моя Лилиям!

Ветер свищет, тучи движутся,
Цветы к земле склоняются.
Что за доля ждёт тебя,
Красота земли, Лилиям?

Лилиям, Лилиям!
Красота земли, Лилиям!

Придут вёсны, станет небо
Синим колокольчиком - я знаю.
В сердце моём цветком когда-то
Расцвела ты, Лилиям.

Лилиям, Лилиям!
Расцветай же, Лилиям!
```
