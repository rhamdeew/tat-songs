# Оригинал

### Якуп Колмый / Юлай Үзәнбаев - Син елмайсаң

```
Син елмайсаң көн яктыра,
Яшә шулай көлеп кенә,
Зифа каен кебек кенә,
Тормышка ямь биреп кенә.

Карашыңнаң көн яктыра,
Йөзең кояш кебек миңа.
Шул кояштай көлеп кенә
Яшә җылы биреп кенә.

Канатланып, җырлап яшәп,
Күңел кылын чиртеп кенә.
Ул җыр безне җилкендерсен
Йөрәккә дәрт биреп кенә.

Гөлгә сине тиңләр идем,
Язгы гөлләр иртә шиңә.
Нур өләшеп яшә җирдә
Тормышка ямь биреп кенә.
```

# Перевод

### Якуп Колмий / Юлай Узянбаев - Когда ты улыбаешься

```
Когда ты улыбаешься - день светлеет,
Живи так, только смеясь,
Как стройная берёза,
Давая жизнь красоту.

От твоего взгляда день светлеет,
Лицо твоё - как солнце для меня.
Так живи, словно то солнце,
Даря только тепло.

Расправив крылья, живя и поя,
Касаясь струн души.
Пусть та песня вдохновит нас,
Даря сердцу страсть.

Я бы приравнял тебя к розе,
Весенние цветы рано увядают.
Делясь светом, живи на этой земле,
Даруя жизни красу.
```
