# Оригинал

### Якуп Колмый / Билгесез - Зәйтүн гөл

```
Яз да яшел,
Көз дә яшел
Зәйтүн гөл.
Яфрак коймый,
Һич коромый
Үсә гөл.

Өйне ямьли,
Күзне ялмый
гөлгенәм.
Тиңдәше юк,
Көндәше юк,
бергенәм.

Минем дә бит,
Бер кызым бар,
Бер дигән,
Өйне ямьли
Зәйтүн гөлдәй
Зәйтүнәм.
```

# Перевод

### Якуп Колмый / Неизвестен - Оливковый цветок

```
Весной зелёный,
Осенью зелёный
Оливковый цвет.
Не опадает,
Не увядает
Растёт цветок.

Дом украшает,
Глаза пленяет
Цветочек мой.
Нет ей равных,
Нет подобных,
Одна такая.

У меня ведь
Есть дочь одна,
Единственная,
Дом украшает,
Как оливковый цвет,
Оливка моя.
```
