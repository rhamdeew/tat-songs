# Оригинал

### Йомабикә Солтанова / Халык - Замана такмаклары (дүртенче вариант)

```
Кариделгә барам әле
Азык-төлек алырга.
Бизнесмен миңа калдырган
Сәндер белән кабырга.

Универмаг шыр ялангач,
Тик тора сатучылар.
Урам тулы алыпсатар,
Бездәйне "атучылар".

Караидел урамы киңме?
Киң түгел ул, тар әле.
Кирәк-яракны алырга
Ярый чегән бар әле.

Ельцин абзый чит илләрдә
Иң почетлы урында.
Инде ул күптән онытты
Иле, халкы турынада.

Берчә язып, берчә сызып,
Указларын укытты.
Горбачевның әшнәләре
Төп башына утыртты.

Гади халыкларга акча
Тамчылап кына тама.
Бизнесменнар һәр ел саен
Җидешәр миллион ала.

Бәяләрне күтәрмим, дип,
Ельцин бирде бит антлар.
Илне упкынга этәрде
Булдыксыз демократлар.

Хәзер бит демократия,
Елцинны тетәмәле.
Якты киләчәк килер дип,
Мин өмет итәм әле.
```

# Перевод

### Йомабикә Солтанова / Халык - Частушки о наших временах (четвертый вариант)

```
К Кариделю схожу сейчас -
Продовольствие купить.
Бизнесмен мне на прощанье
Лавку да ребро сулил.

Универмаг стоит раздетый,
Продавцы лишь ждут-пустуют,
Улица полна перекупов -
Нас таких в упор «прострелят».

Широка ли улица Караидель?
Нет, она пока что узка.
Чтобы нужное достать,
Хорошо, что есть цыганка.

Дядя Ельцин за границей
Сидит в самом почетном кресле,
Давно забыл он про страну,
Про народ свой тоже, если честно.

То напишет, то зачеркивает -
Указы нам читает вслух.
А дружки Горбачева ловко
Обвели его вокруг.

Обычным людям деньги капают
Лишь по капле, кое-как.
Каждый год бизнесмены
По семь миллионов хап.

«Цены не поднимем», - клялся
Ельцин. Клятвы все пропали:
В бездну Родину столкнули
Неловкие демократы.

Теперь, мол, демократия -
Ельцина ругать пора.
Верю: светлое придет,
Есть надежда все же в нас.
```
