# Оригинал

### Якуп Колмый / Рим Хәсәнов - Лилиям (өченче вариант)

```
Җәйге күктә кара болыт
Яңгырлы яшен көйлә.
Кайда сугылырга белми
Иелә, бөгелә Лилиям.

Лилиям, Лилиям!
Иелә, бөгелә Лилиям!

Җилләр исәр, болыт күчәр,
Гөлләр җиргә иелә.
Ниди язмыш көтә сине
Җирнең күрке Лилиям.

Лилиям, Лилиям!
Җирнең күрке Лилиям!

Язлар килер, күктәй булыр,
Зәңгәр кыңгырау мин беләм.
Күңелем чәчәк булып
Чәчәк ат син, Лилиям!

Лилиям, Лилиям!
Чәчәк ат син, Лилиям!
```

# Перевод

### Якуп Колмый / Рим Хәсәнов - Лилиям (третий вариант)

```
В летнем небе туча черная,
Дождь и молния заводят напев.
Не зная, где удариться,
Гнется, сгибается Лилиям.

Лилиям, Лилиям!
Гнется, сгибается Лилиям!

Ветры веют, тучи плывут,
Цветы к земле склоняются.
Какая судьба тебя ждет,
Украса земли, Лилиям.

Лилиям, Лилиям!
Украса земли, Лилиям!

Придут весны - и небо станет,
Синий колокольчик, знаю я.
Душа моя - как цветок,
Распустись, Лилиям!

Лилиям, Лилиям!
Распустись, Лилиям!
```
