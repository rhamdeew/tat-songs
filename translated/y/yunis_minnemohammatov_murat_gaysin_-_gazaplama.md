# Оригинал

### Юныс Миңнемөхәммәтов / Мурат Гайсин - Газаплама

```
Газаплама, сөймим сине диеп,
Өзгәләмә инде бәгырьне.
Минем кебек башка булмас, җаным,
Белүчеләр синең кадерне.

Газаплама, йөрәк таштан түгел,
Туктап калыр, җитәр, уйнама!
Бозлы карашларың белән, иркәм,
Җаннарымны башка кыйнама.

Газаплама!Пыяладай күңел
Челпәрәмә килеп ватыла.
Башка берәү миңа кирәк түгел,
Җаным-тәнем сиңа тартыла!

Газаплама, ничек бармын шулай,
Кабул итеп берүк ал мине.
Мәхәббәтем сүрелерлек түгел.
Яратамын бары бер сине.
```

# Перевод

### Юныс Миңнемөхәммәтов / Мурат Гайсин - Не мучай

```
Не мучай, не говори: «не люблю»,
Не рви мне душу на части.
Другой, как я, не найдёшь, родной,
Кто знает тебе цену и счастье.

Не мучай - сердце не из камня,
Оно устанет, хватит, не играй!
Своим холодным взглядом, милая,
Мне души больше не терзай.

Не мучай! Хрупкая, как стекло, душа
Осколками рассыплется, ломаясь.
Мне больше никого не надо,
Душа и тело к тебе тянутся, срываясь.

Не мучай - я такая, как есть,
Прими меня, прошу, такой.
Любовь моя не угасает.
Люблю я только тебя одну.
```
