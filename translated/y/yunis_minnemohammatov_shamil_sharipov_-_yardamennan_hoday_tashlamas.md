# Оригинал

### Юныс Миңнемөхәммәтов / Шамил Шәрипов - Ярдәменнән Ходай ташламас

```
Ике күзе тулы яшь кенә,
Сызлану үтеп үзәгенә!
Һәр яңа көн - көрәш мәйданы,
Тешне кысып кирәк түзәргә!

Түзем бул син, берүк бирешмә!
"Ник яшим соң җирдә мин", - димә!
Язмыш сынаулары алдында
Башларыңны һич тә син имә!

Иман булса күңел турендә
Ярдәменнән Ходай ташламас!
Гомерең юлында абынып
Егылсаңда, тизрәк торып бас!

Түзем бул син, берүк бирешмә!
"Ник яшим соң җирдә мин", - димә!
Язмыш сынаулары алдында
Башларыңны һич тә син имә!

Авыр уйлар алма башыңа,
Иссә дә җилләре каршыңа!
Һәр туасы якты иртәне
Яңа өмет белән каршыла!

Түзем бул син, берүк бирешмә!
"Ник яшим соң җирдә мин", - димә!
Язмыш сынаулары алдында
Башларыңны һич тә син имә!
```

# Перевод

### Юныс Миңнемөхәммәтов / Шамил Шәрипов - Не оставит Господь своей помощи

```
В глазах слезой наполнен взгляд,
Боль пронзает самую суть!
Каждый новый день - поле битвы,
Стиснув зубы, нужно терпеть!

Будь терпелив, не сдавайся!
Не говори: «Зачем мне жить?»
Перед испытаньями судьбы
Не склоняй головы своей!

Если в сердце живет вера,
Господь своей помощью не оставит!
Если на жизненной дороге споткнешься,
Упадешь - скорее встань!

Будь терпелив, не сдавайся!
Не говори: «Зачем мне жить?»
Перед испытаньями судьбы
Не склоняй головы своей!

Тяжких мыслей не бери на себя,
Даже если ветры против!
Каждое светлое утро, что рождается,
Встречай с новой надеждой!

Будь терпелив, не сдавайся!
Не говори: «Зачем мне жить?»
Перед испытаньями судьбы
Не склоняй головы своей!
```
