# Оригинал

### Ильназ Гарипов - Кызлар

```
Безнен газиз туган якнын
Кызлар эле, кызлары,
Эйтерсен лэ, иртэлэрен
Балкый тан йолдызларын.

Эх кызлар безнен, кызлар,
Гузэллэрнен гузэле.
Бержирдэ юк андый кызлар,
Балкып тора йозлэре.

Йозлэре нурлы аларнын,
Сойкемле сойлэмнэре.
Безнен кызлар бигрэк унган,
Ашкына йорэклэре.

Эх кызлар безнен, кызлар,
Гузэллэрнен гузэле.
Бержирдэ юк андый кызлар,
Балкып тора йозлэре.

Булэк итэр идем сезгэ
Доньянын бар голлэрен.
Бэхет-шатлык китерсеннэр
Шушы бэйрэм коннэре.

Эх кызлар безнен, кызлар,
Гузэллэрнен гузэле.
Бержирдэ юк андый кызлар,
Балкып тора йозлэре.
```

# Перевод

### Ильназ Гарипов - Девушки

```
Нашего дорогого родного края
Девушки, девушки наши,
Скажешь ли — утром они
Сияют как утренние звезды.

Ах девушки наши, девушки,
Красавица из красавиц.
Нигде нет таких девушек,
Светятся их лица.

Лица их светлые,
Милая их речь.
Наши девушки особенно прекрасны,
Пылкие их сердца.

Ах девушки наши, девушки,
Красавица из красавиц.
Нигде нет таких девушек,
Светятся их лица.

Подарил бы я вам
Все цветы мира.
Счастье-радость принесите
В эти праздничные дни.

Ах девушки наши, девушки,
Красавица из красавиц.
Нигде нет таких девушек,
Светятся их лица.
```
