# Оригинал

### Әнгам Атнабаев / Билгесез - Сагынганда (өченче вариант)

```
Сагынганда ак болытка кара,
Ак болытлар булып мин агам.
Шыбыр-шыбыр яңгыр ява, димә,
Күз яшьләре булып мин тамам.

Сагынганда ак буранга карап,
Ак бураннар булып мин явам.
Ак бураннар адаштыра, димә,
Сине адаштырам мин һаман.

Сагынганда ак диңгезгә карап,
Ак диңгезләр булып мин агам.
Ярларына дулкын кага, димә,
Йөрәгеңә килеп мин кагам.
```

# Перевод

### Ангам Атнабаев / Неизвестен - Когда скучаешь (третий вариант)

```
Когда скучаешь — на белые облака смотри,
Белыми облаками я плыву вдали.
Шелестящий дождик льётся, говоришь ты,
Это слёзы мои каплями текут.

Когда скучаешь — на белые метели гляди,
Белыми метелями я лечу в пути.
Белые метели путают, говоришь ты,
Это я тебя сбиваю с пути.

Когда скучаешь — на белое море взгляни,
Белыми морями я теку в дали.
Волны бьются о берега, говоришь ты,
Это я стучусь в сердце твоё.
```
