# Оригинал

### Әнәс Кари / Ильяс Гыйләҗетдин - Ләйсән яңгыр (өченче вариант)

```
Яз иртәсен котлагандай,
Матур булып
Зәңгәр күкне каплап алды
Аксыл болыт.
Бизәкләнгән төсле булды
Бөтен һава.
Кар өстенә шыбыр-шыбыр
Яңгыр ява.

Кушымта:
Ләйсән, ләйсән,
Иң беренче яңгыр.
Ләйсән, күңелләрне
Син куандыр…
Ләйсән, ләйсән,
Язгы җылы яңгыр,
Ләйсән, ләйсән,
Ләйсән, ләйсән!

Җир өстендә яткан карлар
Эреп бетсен.
Басуларда эш кайныйсы
Көннәр җитсен.
Без чыгарбыз бакчаларга
Ул көннәрдә.
Яшелчәләр утыртырбыз
Түтәлләргә.
```

# Перевод

### Анас Кари / Ильяс Гыйләҗетдин - Весенний дождь (третий вариант)

```
Словно весть о приходе весны,
Прекрасным видом
Лазурное небо укрыло
Белое облако.
Словно украшенным казался
Весь небосвод.
На снег тихо-тихо
Дождь идёт.

Припев:
Лейсан, лейсан,
Самый первый дождь.
Лейсан, сердца
Ты обрадуй…
Лейсан, лейсан,
Весенний тёплый дождь,
Лейсан, лейсан,
Лейсан, лейсан!

На земле лежащий снег
Пусть растает весь.
Пусть настанут дни,
Когда в полях кипит работа.
Мы выйдем в огороды
В те дни.
Овощи посадим
На грядки.
```
