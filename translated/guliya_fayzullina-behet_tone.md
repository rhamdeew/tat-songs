# Оригинал

### Гулия Файзуллина - Бэхет тоне

```
Каеннары ай-хай ак икэн,
Cою белэн тулы чак икэн,
Каеннардан якты нур тама.

Бу бит бэхет тоне,
Эзла бэхетенне,
Юк йоклама егет, йоклама.

Егет яна кызга куз атып,
Сандугачлар йори озатып,
Йолдызлардан энже сибелэ.

Табышалар парлар,
Кавышалар ярлар,
Юк бэгърем,
Юк бит син генэ.

Бэхет тоне утеп китэрме,
Безгэ бэхет булэк итэрме?
Каеннардан сора каеннан.

Кунелемнен куркен
Яулый алмас беркем,
Исемнэрен каеп язылган. 

Бу бит бэхет тоне,
Эзла бэхетенне,
Юк йоклама егет, йоклама.
```

# Перевод

### Гулия Файзуллина - Ночь счастья

```
Березы ах какие белые,
Любовью полна эта пора,
С берез светлый свет капает.

Это ведь ночь счастья,
Ищи свое счастье,
Не спи, джигит, не спи.

Джигит влюбленно смотрит на девушку,
Соловьи провожают,
Из звезд жемчуг рассыпает.

Встретятся пары,
Соединятся возлюбленные,
Нет, дорогой,
Нет, только ты один.

Пройдет ли ночь счастья,
Подарит ли нам счастье?
Спроси у березы березу.

Красоту моего сердца
Не сможет похитить никто,
Имена на березе высечены.

Это ведь ночь счастья,
Ищи свое счастье,
Не спи, джигит, не спи.
```
