# Оригинал

### Башкорт халык җыры - Сандугач (икенче вариант)

```
Җырла, егет, киреп киң күкрәгең,
Яңгыратып Уралның буйларын.

Кайда да гына сайрый сандугач,
Су буйкайларында тугайда.

Яшь йөрәккәеңнән чыккан җырлар
Дәртләндерсен Уралның улларын.

Агыйделгә төшеп кармак салдым
Чуртан балык белән сазанга.

Егет кенә чакта даның китсен
Өфе каласына, Казанга.
```

# Перевод

### Башкирская народная песня - Сандугач (второй вариант)

```
Пой, юноша, возвратись к широкой груди своей,
Разбуди берега Урала.

Пусть везде поет сандугач,
На берегах реки в лесу.

Пусть песни, исходящие из твоего юного сердца,
Возбудят уральские горы.

Спал в реку удочку, приманку
На угря и сазана.

Пусть юноша в свои дни прославится
В городе Уфа, в Казани.
```
