# Оригинал

### Дилья Нигматуллина - Хэтер тузмаган

```
Ничэ йорт акты елгада,
Гасырлар булып ташып.
Каршымда басып торасын,
Кочарга мине куркып.

Хислэрен ага кузеннэн,
Хэтернен бер тамчысы.
Куз яше сузсез анлата
Узган еллар ачысын.

Хаа-уа-уау
Ничэ ел узган,
Хаа-уа-уау
Купме су аккан,
Хаа-уа-уау
Кулмэклэр тузган,
Хаа-уа-уау
Хэтер тузмаган.

Кулыма минем урелден,
Калтырады бармаклар.
Бер берсен бездэн бигрэк тэ
Сагынган безнен куллар.

Хаа-уа-уау
Ничэ ел узган,
Хаа-уа-уау
Купме су аккан,
Хаа-уа-уау
Кулмэклэр тузган,
Хаа-уа-уау
Хэтер тузмаган.

Бер мизгел килеп туктатты
Кызу вакыт сорешен.
Жир шарын икенче якка
Борып куйды йорешен.

Тынлыкта калды бар донья,
Сузсез калды бар жихан.
Суз генэ ярдэм итэ алмас
Ике йорэк кузгалган.

Хаа-уа-уау
Ничэ ел узган,
Хаа-уа-уау
Купме су аккан,
Хаа-уа-уау
Кулмэклэр тузган,
Хаа-уа-уау
Хэтер тузмаган.
```

# Перевод

### Дилья Нигматуллина - Память не стёрлась

```
Сколько домов по реке плыло,
Века пронеслись.
Стоишь напротив меня,
Боишься меня обнять.

Чувства текут из глаз,
Памяти капля одна.
Слёзы без слов рассказывают
Прошедших лет горечь.

Хаа-уа-уау
Сколько лет прошло,
Хаа-уа-уау
Сколько воды утекло,
Хаа-уа-уау
Одежды истёрлись,
Хаа-уа-уау
Память не стёрлась.

В мою руку из твоей
Дрожат пальцы.
Больше нас самих
Соскучились наши руки.

Хаа-уа-уау
Сколько лет прошло,
Хаа-уа-уау
Сколько воды утекло,
Хаа-уа-уау
Одежды истёрлись,
Хаа-уа-уау
Память не стёрлась.

Один миг пришёл и остановил
Горячего времени бег.
Земной шар в другую сторону
Повернул свой ход.

В тишине остался весь мир,
Безмолвным стал весь свет.
Только словом не поможешь
Двум взволнованным сердцам.

Хаа-уа-уау
Сколько лет прошло,
Хаа-уа-уау
Сколько воды утекло,
Хаа-уа-уау
Одежды истёрлись,
Хаа-уа-уау
Память не стёрлась.
```
