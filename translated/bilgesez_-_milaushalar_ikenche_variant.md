# Оригинал

### Билгесез - Миләүшәләр (икенче вариант)

```
Мөлдерәмә тулган күңелем,
Чайкала, гел түгелә.
Үсеп чыккан миләүшәләр,
Текәлгәннәр күземә.
Үсеп чыккан миләүшәләр,
Текәлгәннәр күземә.

Кочагыма алып кыстым
Миләүшә чәчәкләрен.
Түзми йөрәк, гел кабатлый
Син сөенеп киләчәген.
Түзми йөрәк, гел кабатлый
Син сөенеп киләчәген.

Сиңа гына багышладым
Бар җырларым, Миләүшә.
Җырларымда, хыялымда
Синең белән серләшәм.
Җырларымда, хыялымда
Синең белән серләшәм.
Синең белән серләшәм...
Серләшәм...
```

# Перевод

### Билгесез - Миләүшәләр (второй вариант)

```
Сердце полно тумана,
Качается, льётся слезами.
Выросли цветы миләүшә,
Скатились в глаза мои.
Выросли цветы миләүшә,
Скатились в глаза мои.

Взял в объятия свои
Цветы миләүшә.
Не терпит сердце, всё повторяет:
Ты приходишь во сне.
Не терпит сердце, всё повторяет:
Ты приходишь во сне.

Тебе я отдал все
Песни свои, Миләүшә.
В песнях моих, в мечтах моих
С тобой тайны деля.
В песнях моих, в мечтах моих
С тобой тайны деля.
С тобой тайны деля...
Деля...
```
