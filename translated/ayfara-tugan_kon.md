# Оригинал

### Айфара - Туган кон

```
Кук йозе чалт аяз буген
Кояш та серле елмая
Жир тала татлы йокыга
Жир йозе тулган нурларга

Э буген синен туган кон
Унсигез яшь тула сина
Йозлэрен шатлыктан балкый
Кузлэрен бэхеткэ тула

Туган кон белэн
Котлыйбыз сине
Телэклэрдэн
Ин саф изгесен

Дусларын бергэ жыелган
Котлаулар синен хормэткэ
Зур бэхет, шатлык телилэр
Кумэлэр сине булэклэр

Туган кон белэн
Котлыйбыз сине
Телэклэрдэн
Ин саф изгесен

Зур юллар аяк басасы
Куп булыр алда киртэсе
Бирешмэ язмышларга син
Заяга гомрен утмэсен

Туган кон белэн
Котлыйбыз сине
Телэклэрдэн
Ин саф изгесен
```

# Перевод

### Айфара - День рождения

```
Небесная высь светло-ясная сегодня,
Солнце тайно улыбается,
Земля-поле в сладком сне,
Земная поверхность полна лучами.

Ах, сегодня твой день рождения,
Восемнадцать лет исполняется тебе,
Лицо сияет от радости,
Глаза полны счастья.

С днем рождения
Поздравляем тебя,
Из пожеланий
Самое чистое и святое.

Друзья твои собрались вместе,
Поздравления в твою честь,
Большого счастья, радости желают,
Осыпают тебя подарками.

С днем рождения
Поздравляем тебя,
Из пожеланий
Самое чистое и святое.

Большие дороги будешь ты проходить,
Много будет впереди преград,
Не сдавайся судьбе,
Пусть жизнь не пройдет зря.

С днем рождения
Поздравляем тебя,
Из пожеланий
Самое чистое и святое.
```
