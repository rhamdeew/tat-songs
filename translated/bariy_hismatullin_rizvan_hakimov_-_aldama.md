# Оригинал

### Барый Хисмәтуллин / Ризван Хәкимов - Алдама

```
Алдама мине, алдама,
Алдап юуаныч тапма.
Йөрәгемә утлар яккач,
Башкаларны яратма.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт - уенчык тугел,
Аның белән шаярма!

Күңелләрнең хисләрен
Сөйлимен мин кичләрен.
Эзләсәң дә таба алмассың
Минем кебек ишләрне.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт - уенчык тугел,
Аның белән шаярма!

Өзелгән җеп ялганса да,
Анын төене кала.
Уеннан уймак чыгарсан,
Күзләрдән яшьләр тама.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт - уенчык тугел,
Аның белән шаярма!
```

# Перевод

### Барый Хисматуллин / Ризван Хакимов - Не обманывай

```
Не обманывай меня, не обманывай,
Обмануть - найдешь вину.
Когда в сердце огонь вспыхнет,
Других любить не буду.

Не обманывай же, нет, не обманывай,
В сердце моем горит огонь!
Любовь - не игрушка,
С ней не шути!

Чувства сердец
Я поведаю ночью.
Ищи - не найдешь
Таких, как я, дел.

Не обманывай же, нет, не обманывай,
В сердце моем горит огонь!
Любовь - не игрушка,
С ней не шути!

Оборванная нить, хоть и лжет,
Но узел остается.
Если выйдешь из игры,
Слезы из глаз капнут.

Не обманывай же, нет, не обманывай,
В сердце моем горит огонь!
Любовь - не игрушка,
С ней не шути!
```
