# Оригинал

### Айрат Шәйхетдинов / Айдар Тимербаев - Парлы ялгызлар

```
Култыклашып йөри алар,
Елмаялар кешегә.
Тик нигәдер өйгә кайткач
Күңелләре күшегә.
Ызгышмыйлар, талаш та юк,
Кычкырмыйлар үзара,
Тышта нурлы карашларның
Өйдә нуры югала.

Бар парлылар, бар ялгызлар,
Пары бар, үзе ялгыз,
Пар түгелле парлашканнар,
Бик усал уйный язмыш,
Язмыш...

Берсен берсе күрмәгәндәй
Йөриләр эндәшмичә.
Кайчакларда айлар уза
Үзара сөйләшмичә.
Ә урамда, карап торсаң,
Бик бәхетле пар сыман.
Гаиләсе, мал-мөлкәте,
Бөтенесе бар сыман.

Бар парлылар, бар ялгызлар,
Пары бар, үзе ялгыз,
Пар түгелле парлашканнар,
Бик усал уйный язмыш,
Язмыш...

Кемдер китә, түз(ә) алмыйча,
Күңел караңгысына.
Якты эзләп үз җанына,
Ышанып табасына.
Кемдер кала, көч таб(а) алмый,
Тормыш кага бер көйгә.
Мең сәбәпләр эзләп таба
Кайтып кермәскә өйгә.

Бар парлылар, бар ялгызлар,
Пары бар, үзе ялгыз,
Пар түгелле парлашканнар,
Бик усал уйний язмыш,
Язмыш...
Язмыш...
```

# Перевод

### Айрат Шайхетдинов / Айдар Тимербаев - Парные одиночки

```
Под руку ходят они,
Улыбаются людям вокруг.
Но почему-то домой придя,
Их души уходят в испуг.
Не ругаются, ссор нет совсем,
Не кричат друг на друга они,
Светлый взгляд на улице их
Дома гаснет, как свет в тиши.

Есть парные, есть одинокие,
Есть пара, но сам одинок,
Не пара те, кто в паре,
Жестоко играет рок,
Рок...

Словно друг друга не видя,
Они молча живут день за днём.
Иногда месяцы тянутся,
И друг с другом они ни о чём.
А на улице, посмотри-ка,
Как счастливая пара они.
Есть семья у них, есть богатство,
И как будто полны их дни.

Есть парные, есть одинокие,
Есть пара, но сам одинок,
Не пара те, кто в паре,
Жестоко играет рок,
Рок...

Кто-то уходит, не вынеся
Темноты души своей.
Ищет свет для себя самого,
Веря в лучших грядущих дней.
Кто-то остаётся, сил не найдя,
Жизнь вгоняет в один мотив.
Тысячу причин находя,
Чтобы в дом не вернуться, уйдя.

Есть парные, есть одинокие,
Есть пара, но сам одинок,
Не пара те, кто в паре,
Жестоко играет рок,
Рок...
Рок...
```
