# Оригинал

### Айдар Тимербаев - Гафу итмә

```
Бүтән сиңа берни әйтә алмам,
Чөнки йөрәгем кушмый миңа.
Хәзер инде соңдыр ни әйтсәм дә,
Мин бит хыянәт иттем сиңа.

Гафу итмә мине,гафу итмә,
Алыштырдым мин сине башкага.
Синең күзләрдән аккан ачы яшьләр,
Хәзер үземнең күзләрдән тама.

Менә тагын икәү күл буенда,
Ә синең кулда сары чәчәк.
Шул сары чәчәкләр шикелле,
Моңсу булыр безнең киләчәк.

Гафу итмә мине,гафу итмә,
Алыштырдым мин сине башкага.
Синең күзләрдән аккан ачы яшьләр,
Хәзер үземнең күзләрдән тама.

Бүтән сиңа берни әйтә алмам,
Чөнки насыйп түгелме сиңа.
Хәзер инде соңдыр ни әйтсәм дә,
Мин бит хыянәт иттем сиңа.

Гафу итмә мине,гафу итмә,
Алыштырдым мин сине башкага.
Синең күзләрдән аккан ачы яшьләр,
Хәзер үземнең күзләрдән тама.
```

# Перевод

### Айдар Тимербаев - Не прощай меня

```
Больше ничего не могу сказать тебе,
Потому что сердце мне не велит.
Теперь уж поздно, что бы ни говорил,
Ведь я тебе изменил.

Не прощай меня, не прощай,
Я заменил тебя другой.
Горькие слёзы из твоих глаз,
Теперь текут из моих.

Вот снова двое у берега озера,
А в твоих руках жёлтые цветы.
Как эти жёлтые цветы,
Печальным будет наше будущее.

Не прощай меня, не прощай,
Я заменил тебя другой.
Горькие слёзы из твоих глаз,
Теперь текут из моих.

Больше ничего не могу сказать тебе,
Потому что не судьба нам быть с тобой.
Теперь уж поздно, что бы ни говорил,
Ведь я тебе изменил.

Не прощай меня, не прощай,
Я заменил тебя другой.
Горькие слёзы из твоих глаз,
Теперь текут из моих.
```
