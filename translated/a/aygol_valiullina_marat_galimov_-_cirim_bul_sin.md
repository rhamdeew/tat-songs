# Оригинал

### Айгөл Вәлиуллина / Марат Галимов - Җырым бул син

```
Энҗе сибеп бизәр идем, юлга гөлләр тезәр идем,
Хисләремне минем әгәр аңлый алсаң.
Бәхет тәмен белер идем, сөю-назга төрер идем,
Яннарымда минем азга гына калсаң.

Җырым бул син, моңым бул син
Бары миңа, миңа дигән.
Ярты җаным, алсу таңым,
Сине ятка, ятка бирмәм.

Минем дә бит күңел синдә, йөрәк чыгар булып тибә,
Канатланып йөрим күрсәм сине читтән.
Хыялымны чын итүче, тормышымны җыр итүче,
Язым бул син, китмә, кал син иртән.

Җырым бул син, моңым бул син
Бары миңа, миңа дигән.
Ярты җаным, алсу таңым,
Сине ятка, ятка бирмәм.
Җырым бул син, моңым бул син
Бары миңа, миңа дигән.
Ярты җаным, алсу таңым,
Сине ятка, ятка бирмәм...
```

# Перевод

```
Жемчугом рассыпала бы, цветами путь устилала бы,
Если бы ты только понял мои чувства.
Вкус счастья познала бы, в любви и неге жила бы,
Если бы ты лишь немного остался рядом.

Будь моей песней, будь моей тоской,
Всё для меня, только для меня.
Половина души, рассвет мой алый,
Никому тебя, никому не отдам.

Ведь моё сердце с тобой, готово выпрыгнуть из груди,
Лечу окрылённой, когда вижу тебя издалека.
Ты воплощаешь мои мечты, превращаешь жизнь в песню,
Ты моя весна, не уходи, останься до утра.

Будь моей песней, будь моей тоской,
Всё для меня, только для меня.
Половина души, рассвет мой алый,
Никому тебя, никому не отдам.
```
