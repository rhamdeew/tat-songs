# Оригинал

### Әхмәт Рәшитов / Сара Садыкова - Сине генә сагынам

```
Йөрәгемә сагыш уты өстәп,
Алмагачлар ак шәл ябынган.
Ә мин сине, солдат сөйгәнемне,
Бары сине генә сагынам.

Матур сүзләр сөйләп, бар егетләр
Каш сикертеп уза янымнан.
Башкаларга күзем төшәр димә,
Бары сине генә сагынам.

Тәүге үбүләрем вәгъдә сиңа,
Гыйшкым уты синнән кабынган.
Көннәр буе, айлар, еллар буе
Бары сине генә сагынам.

Сөйгәнемә җырым сәлам илтсен
Туып үскән Идел ягыннан.
Онытма син, бәгърем, онытма син,
Бары сине генә сагынам.
```

# Перевод

### Ахмат Рашитов / Сара Садыкова - Только тебя скучаю

```
Тоской огонь в душе моей пылает,
Яблони в белый шаль укутались.
А я тебя, солдата дорогого,
Лишь только тебя одного скучаю.

Красивые слова мне говорят все парни,
Бровями манят, проходя со мной.
Не думай, что на них смотрю с надеждой,
Лишь только тебя одного скучаю.

Первые поцелуи — обещанье для тебя,
Огонь любви моей от тебя зажёгся.
День за днём, и месяц за месяцем, годами
Лишь только тебя одного скучаю.

Пусть песня моя привет передаст любимому
С родных берегов великой Волги.
Не забывай, мой милый, не забывай,
Лишь только тебя одного скучаю.
```
