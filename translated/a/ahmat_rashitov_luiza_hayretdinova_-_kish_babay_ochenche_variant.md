# Оригинал

### Әхмәт Рәшитов / Луиза Хәйретдинова - Кыш бабай (өченче вариант)

```
Кыш бабай, Кыш бабай,
Безгә бик таныш бабай;
Сакал-мыек ап-актан,
Килгән безгә ерактан.

Бүләкләр алып килгән,
Бүреген кыңгыр кигән.
Әйлән-бәйлән уйната,
Биетә һәм җырлата.

Безгә конфетлар бирә,
Үзе елмая-көлә.
Кыш бабай безгә тагын
Яңа ел алып килә.
```

# Перевод

### Ахмат Рашитов / Луиза Хәйретдинова - Дед Мороз (третий вариант)

```
Дед Мороз, Дед Мороз,
Всем знакомый добрый дед;
Борода белым-бела,
К нам пришёл издалека.

Подарки нам принёс он,
В шубу звонкую одет.
Хороводы заведёт,
Пляшет с нами и поёт.

Раздаёт нам он конфеты,
Сам смеётся и шутит.
Дед Мороз нам снова
Новый год приносит.
```
