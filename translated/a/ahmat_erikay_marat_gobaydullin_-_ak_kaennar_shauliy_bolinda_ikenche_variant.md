# Оригинал

### Әхмәт Ерикәй / Марат Гобәйдуллин - Ак каеннар шаулый болында (икенче вариант)

```
Ак каеннар шаулый болында,
Әкрен генә җилләр искәндә;
Нурлы йөзең искә төшкәндә
Сагыну хисе таша җырымда.

Тау буенда, чишмә буенда
Ал чәчәкләр йөгергән төсле.
Синнән хәбәр китергән төсле,
Ак каеннар шаулый болында.

Ямьле күлнең зәңгәр суында
Ялгыз аккош моңаеп йөзә.
Үзәкләремне өзә-өзә,
Ак каеннар шаулый болында.
```

# Перевод

### Ахмат Ерикей / Марат Гобәйдуллин - Белые берёзы шумят на лугу (второй вариант)

```
Белые берёзы шумят на лугу,
Когда ветер нежно подует;
Когда светлый лик твой я вспоминаю,
Тоска переполняет песню мою.

У подножья гор, у родника
Алые цветы словно бегут.
Словно весть о тебе приносят,
Белые берёзы шумят на лугу.

В синих водах прекрасного озера
Одинокий лебедь печально плывёт.
Разрывает сердце моё на части,
Белые берёзы шумят на лугу.
```
