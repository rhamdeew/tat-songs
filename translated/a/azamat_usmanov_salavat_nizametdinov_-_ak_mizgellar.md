# Оригинал

### Азамат Усманов / Салават Низаметдинов - Ак мизгелләр

```
Шомырт күпереп чəчәк атсынмы,
Җирне каплап тəүге кар япсынмы -
Хыял илтə бала чакларыма,
Яшьлегемə, туган якларыма ...

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!

Кырпак карда тəүге (яңа) эз калдырып,
Көннəр буе аякны талдырып,
Барлык хəсрəтлəрне оныткандай,
Канатланам яңа җыр откандай.

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!

Тəүге кар, шау чəчəк - ап‑ак мизгел,
Яралы күңелне дәвалый гел.
Актан‑ак, пакътан‑пакъ хыял төсле,
Ак мизгелсез яши алмам төсле ...

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!
```

# Перевод

### Азамат Усманов / Салават Низаметдинов - Светлые мгновения

```
Пусть черёмуха вспенится белым цветом,
Пусть первый снег укроет землю пледом -
Мечта несёт в мои детские годы,
В мою юность, к родным просторам...

Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят!
Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят, приходят!

Оставив первый след на снежном покрывале,
Днями напролёт усталости не зная,
Словно все печали я забыла,
Воспаряю, будто песня зазвучала.

Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят!
Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят, приходят!

Первый снег, цвет черёмухи - светлое мгновение,
Раненую душу исцеляет всегда.
Белые-белые, чистые-чистые, словно грёзы,
Без светлых мгновений жить не могу, наверно...

Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят!
Белые-белые, чистые-чистые мгновения
Пусть приходят, приходят, приходят!
```
