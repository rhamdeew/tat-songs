# Оригинал

### Әхмәт Ерикәй / Мансур Мозаффаров - Сипкелле диләр мине…

```
Туткыллы димиләр мине,
Сипкелле диләр мине.
Әзрәк сипкелле булсам да,
Сөйкемле диләр мине.

Буйларың зифа димиләр,
Аз гына кыска диләр.
Ләкин шуңа карамастан.
Эшкә бик оста диләр.

Кашларың кара димиләр,
Күзләрең серле диләр,
Шушы күзләр егетләрнең
Күңелен сихерли диләр.

Чәчләрең бөдрә димиләр,
Билләрең бөглә диләр.
Яраттыра белмәсә дә,
Ярата белә диләр.

Туткыллы димиләр мине,
Сипкелле диләр мине.
Әзрәк сипкелле булсам да,
Сөйкемле диләр мине.
```

# Перевод

### Ахмат Ерикей / Мансур Мозаффаров - Веснушчатой меня зовут…

```
Упрямой называют меня,
Веснушчатой зовут меня.
Хоть немного веснушчата я,
Милой все же зовут меня.

Рост твой хрупким называют,
Чуть коротковатым зовут.
Но несмотря на это,
К работе очень ловкой зовут.

Брови твои черными называют,
Глаза твои таинственными зовут,
Эти глаза у парней
Сердца околдовывают, говорят.

Волосы твои кудрявыми называют,
Талию тонкой зовут.
Хоть не умеет заставлять любить,
Умеет любить, говорят.

Упрямой называют меня,
Веснушчатой зовут меня.
Хоть немного веснушчата я,
Милой все же зовут меня.
```
