# Оригинал

### Әкрәм Даутов - Тәвәкәллә

```
Син бер генә караш салдың миңа,
Йөзең күркәм, тик карашың сүлпән.
Күкрәгендә ялкын саклаганчы,
Тәвәккәллә, тәвәккәллә, тәвәккәллә, иркәм!

Бакчагызда сайрый бер сандугач,
Мин дә ярсып турагыздан үтәм,
Тәрәзәдән яшерен караганчы,
Тәвәккәллә, тәвәккәллә, тәвәккәллә, иркәм!

Кавышырга вәгъдә бирдек икәү,
Атаң-әнәң риза булырмы икән?
Ике йөрәк бергә тибә икән,
Тәвәккәллә, тәвәккәллә, тәвәккәллә, иркәм!
```

# Перевод

### Акрам Даутов - Решайся

```
Ты только раз взглянула на меня,
Лицо прекрасно, но взгляд застенчив твой.
Пока в груди огонь хранишь ты,
Решайся, решайся, решайся, милая!

В саду твоём поёт один соловей,
И я, любя, мимо дома пройду,
Пока ты тайно смотришь из окна,
Решайся, решайся, решайся, милая!

Мы оба обещали встретиться вдвоём,
Согласятся ли твои отец и мать?
Два сердца вместе бьются ведь,
Решайся, решайся, решайся, милая!
```
