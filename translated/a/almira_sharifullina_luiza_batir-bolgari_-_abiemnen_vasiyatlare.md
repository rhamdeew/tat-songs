# Оригинал

### Эльмира Шәрифуллина / Луиза Батыр-Болгари - Әбиемнең васыятьләре

```
Иңдәреңдә фәрештәләр,
Ди миңа ак әбием.
Фәрештәм син дип сөяләр
Әтием һәм әнием.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
Үзе тарих, тере тарих
Безнең әби-бабабыз.
Зурласак кына аларны
Олы булып калабыз.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
Әбиемнең васыятьләре
Офыгымны киңәйтә.
Иңнәремдә фәрештәләр
Изгелеккә өйрәтә.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
```

# Перевод

### Эльмира Шәрифуллина / Луиза Батыр-Болгари - Завещание бабушки

```
На плечах твоих ангелы,
Говорит мне моя бабушка.
Ты мой ангел, так говорят
Мои отец и мать.
Каждое утро, каждый светлый день
С Фатихи я начинаю.
Каждое дело, каждый свой поступок
Добру я посвящаю.
Сама история, живая история —
Наши дедушка и бабушка.
Если мы их почитаем,
Великими сами станем.
Каждое утро, каждый светлый день
С Фатихи я начинаю.
Каждое дело, каждый свой поступок
Добру я посвящаю.
Завещание бабушки моей
Расширяет мой горизонт.
Ангелы на моих плечах
Добру меня учат.
Каждое утро, каждый светлый день
С Фатихи я начинаю.
Каждое дело, каждый свой поступок
Добру я посвящаю.
```
