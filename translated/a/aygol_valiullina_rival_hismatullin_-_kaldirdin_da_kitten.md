# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Калдырдың да киттең

```
Калдырдың да киттең,
Мине ятим иттең.
Карамадың кире борылып.
Сине искә алам,
Шуннан ләззәт табам,
Эзләреңә атлыйм орылып.

Калдырдың да киттең,
Син яндырып үттең
Сөюләрнең соңгы күперен.
Ашыгулар яман,
Ашыгулар яман,
Еллар узгач үзең үкендең.

Калдырдың да киттең,
Ә мин сине көттем,
Хәлең сорап искән җилләрдән.
Синең тарафыңнан,
Җилләр канатыннан,
Җил бәрелгән сәлам җилләгән.

Калдырдың да киттең,
Син яндырып үттең
Сөюләрнең соңгы күперен.
Ашыгулар яман,
Ашыгулар яман,
Еллар узгач үзең үкендең.
Ашыгулар яман,
Ашыгулар яман,
Еллар узгач үзең үкендең...
```

Иду по следу твоему, склонясь.

Последний мост любви без сожаленья.

Лишь переданный привет прохладный.
