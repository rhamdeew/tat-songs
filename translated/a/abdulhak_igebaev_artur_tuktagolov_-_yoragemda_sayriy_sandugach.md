# Оригинал

### Абдулхак Игебаев / Артур Туктаголов - Йөрәгемдә сайрый сандугач

```
Купшылланып гөрләп чәчәк ата
Син утырткан зифа алмагач.
Алмагачны кочаклардай булып
Йөрәгемдә сайрый сандугач.

Аерылышу сагышларын йотып,
Сине һуштан язып сагынгач,
Уфтанма, дим назлап юаткандай
Йөрәгемдә сайрый сандугач.

Хозурланып таңгы офыкларда
Син яраткан йолдыз кабынгач,
Йолдыз нурларында талпынгандай,
Йөрәгемдә сайрый сандугач.

Аерылышу сагышларын йотып,
Сине һуштан язып сагынгач,
Уфтанма, дим назлап юаткандай
Йөрәгемдә сайрый сандугач.
```

# Перевод

### Абдулхак Игебаев / Артур Туктаголов - В моём сердце поёт соловей

```
Благоухая, пышно цветёт
Твоя изящная яблонька.
Словно обнимая яблоню,
В моём сердце поёт соловей.

Поглощая печаль разлуки,
Когда тоскую, вспоминая тебя,
Словно нежно утешает: "Не грусти",
В моём сердце поёт соловей.

Когда в утренних небесах
Зажигается твоя любимая звезда,
Словно трепеща в звёздном сиянье,
В моём сердце поёт соловей.

Поглощая печаль разлуки,
Когда тоскую, вспоминая тебя,
Словно нежно утешает: "Не грусти",
В моём сердце поёт соловей.
```
