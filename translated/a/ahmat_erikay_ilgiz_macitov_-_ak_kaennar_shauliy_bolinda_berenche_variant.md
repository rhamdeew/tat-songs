# Оригинал

### Әхмәт Ерикәй / Илгиз Мәҗитов - Ак каеннар шаулый болында (беренче вариант)

```
Ак каеннар шаулый болында,
Әкрен генә җилләр искәндә;
Нурлы йөзең искә төшкәндә
Сагыну хисе таша җырымда.

Тау буенда, чишмә буенда
Ал чәчәкләр йөгергән төсле.
Синнән хәбәр китергән төсле,
Ак каеннар шаулый болында.

Ямьле күлнең зәңгәр суында
Ялгыз аккош моңаеп йөзә.
Үзәкләремне өзә-өзә,
Ак каеннар шаулий болында.
```

# Перевод

### Ахмат Ерикей / Илгиз Маджитов - Белые берёзы шумят на лугу (первый вариант)

```
Белые берёзы шумят на лугу,
Когда тихо веют ветра;
Когда вспоминаю светлый твой лик,
Чувство тоски переполняет мою песню.

У подножия горы, у родника
Словно бегут алые цветы.
Словно весть от тебя приносят,
Белые берёзы шумят на лугу.

В синей воде прекрасного озера
Одинокий лебедь печально плывёт.
Разрывая моё сердце,
Белые берёзы шумят на лугу.
```
