# Оригинал

### Абдулхак Игебаев / Тәлгать Шәрипов - Сиңа (алтынчы вариант)

```
Ал булаеммы сиңа,
Гөл булаеммы сиңа?
Ефәк чәчеңне тузгыткан
Җил булаеммы сиңа,
Җил булаеммы сиңа?

Төн булаеммы сиңа,
Көн булаеммы сиңа,
Татлы төшеңнән дә татлы
Өн булаеммы сиңа,
Өн булаеммы сиңа?

Сут булаеммы сиңа,
Кот булаеммы сиңа?
Күңелкәеңне җылыткан
Ут булаеммы сиңа,
Ут булаеммы сиңа?

Яз булаеммы сиңа,
Наз булаеммы сиңа,
Былбыл моңыннан да моңлы
Саз булаеммы сиңа,
Саз булаеммы сиңа,
Саз булаеммы сиңа?
```

# Перевод

### Абдулхак Игебаев / Талгат Шарипов - Тебе (шестой вариант)

```
Стану ли алой зарёй для тебя,
Стану ли цветком для тебя?
Ветром, что развеял твои
Шелковые волосы,
Стану ли ветром для тебя?

Стану ли ночью для тебя,
Стану ли днём для тебя,
Голосом слаще твоих
Сладких снов,
Стану ли голосом для тебя?

Стану ли счастьем для тебя,
Стану ли радостью для тебя?
Огнём, что согрел твоё
Сердце,
Стану ли огнём для тебя?

Стану ли весной для тебя,
Стану ли нежностью для тебя,
Мелодией задушевнее напева
Соловья,
Стану ли мелодией для тебя,
Стану ли мелодией для тебя?
```
