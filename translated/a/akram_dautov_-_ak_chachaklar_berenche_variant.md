# Оригинал

### Әкрәм Даутов - Ак чәчәкләр (беренче вариант)

```
Таллыкта алан,
Аланны япкан
Ап-ак чәчәкләр.
Без бергә идек,
Калфаклар үрдек
Шул ак чәчкәдән.

Алсу йөзеңә
Бигрәк килешә
Чәчкә калфагың!
Синнән дә сылу
Чәчәкләр булу
Мөмкинме тагын?!

Дөрләсен хисләр!
Тәмле, хуш исләр
Җилдә уйнасын!
Безнен күңелләр
Сулган чәчәкләр
Кебек булмасын!
```

# Перевод

### Акрам Даутов - Белые цветы (первый вариант)

```
Сорванные в ивняке,
Покрывшие поляну
Белоснежные цветы.
Мы были вместе тогда,
Плели венки из них,
Из тех белых цветов.

Так идёт тебе
К нежному лицу
Твой цветочный венок!
Может ли быть
Ещё прекраснее
Цветов, чем ты?!

Пусть волнуются чувства!
Сладкие, приятные ароматы
Пусть играют на ветру!
Пусть наши сердца
Не будут как цветы
Увядшие!
```
