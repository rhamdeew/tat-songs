# Оригинал

### Айгөл Вәлиуллина / Риназ Фазлыйәхмәтов - Җилләр булып

```
Онытмыйсың әллә һаман, миңа
Җилләр булып сөю юллыйсың.
Чәчләреңнән назлап үбәм дисең,
Артларымнан куып уйныйсың.

Җилләр булып, җилләр булып кына
Син киләсең хәзер яныма.
Җилләр тынса - шаян сүзләреңне,
Сине күң(е)лем өзелеп сагына.

Җилләр булып кереп куеныма,
Шушы бит ул сөю, күрәсең.
Чәчкә сибеп үтәр юлларыңа,
Иренеңнән килә үбәсем.

Җилләр булып, җилләр булып кына
Син киләсең хәзер яныма.
Җилләр тынса - шаян сүзләреңне,
Сине күң(е)лем өзелеп сагына.

Җилләр булып, җилләр булып кына
Чакырасың үзең артыннан.
Шул җилләрең әсир итте җаным,
Мин җилләргә кабат сатылам.

Җилләр булып, җилләр булып кына
Син киләсең хәзер яныма.
Җилләр тынса - шаян сүзләреңне,
Сине күң(е)лем өзелеп сагына.
```

# Перевод

### Айгуль Валиуллина / Риназ Фазлыяхматов - Став ветрами

```
Ты не забываешь, видно, обо мне,
Став ветрами, любовь свою даришь.
Говоришь, что нежно поцелую в волосы,
А сам за мною следом веселишься.

Став ветрами, только лишь ветрами
Ты приходишь сейчас ко мне.
Когда ветра стихнут - твои нежные слова,
Душа моя тоскует о тебе.

Став ветрами, ворвался в объятия,
Ведь это и есть любовь, пойми.
Цветами осыплю твои дороги,
Так хочется поцеловать твои губы.

Став vetрами, только лишь ветрами
Ты приходишь сейчас ко мне.
Когда ветра стихнут - твои нежные слова,
Душа моя тоскует о тебе.

Став ветрами, только лишь ветрами
Ты зовёшь меня за собой.
Эти ветра пленили душу мою,
Я снова ветрам себя отдаю.

Став ветрами, только лишь ветрами
Ты приходишь сейчас ко мне.
Когда ветра стихнут - твои нежные слова,
Душа моя тоскует о тебе.
```
