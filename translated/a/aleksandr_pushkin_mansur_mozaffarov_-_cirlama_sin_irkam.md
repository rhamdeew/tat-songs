# Оригинал

### Александр Пушкин / Мансур Мозаффаров - Җырлама син, иркәм

```
Җырлама син, иркәм, минем алда
Грузиянең моңлы җырларын.
Минем искә төшә үткән тормыш,
Ерак илнең зәңгәр кырлары.

Хәтерләтә миңа синең җырың
Теге айны, ерак даланы;
Күз алдыма китерә анда калган,
Ялгыз калган бер каз баланы.
```

# Перевод

### Александр Пушкин / Мансур Мозаффаров - Не пой, милая

```
Не пой, милая, при мне
Грузии печальных песен.
Вспоминается мне прошлая жизнь,
Далекой стороны лазурные степи.

Напоминает мне твоя песня
Ту луну, далекую долину;
Встает пред взором оставшийся там,
Оставшийся один гусенок одинокий.
```
