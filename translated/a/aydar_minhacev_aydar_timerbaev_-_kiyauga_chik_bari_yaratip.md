# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Кияүгә чык бары яратып

```
Уянасың күреп төшләреңдә,
Күзең төшкән диеп берәүгә.
Юк, ашыкма, «кирәк син» дип кенә,
Иртәрәктер сөям дияргә.

Ярат үзең, яратсыннар сине,
Яшәсеннәр син дип җан атып.
Кирәк булган өчен генә түгел,
Кияүгә чык бары яратып.

Юк, эзләмә, сайлый күрмә ярны,
Алтын-көмешләргә кызыгып.
Яналмассың сөю утларында,
Байлык белән булмый җылынып.

Ярат үзең, яратсыннар сине,
Яшәсеннәр син дип җан атып.
Кирәк булган өчен генә түгел,
Кияүгә чык бары яратып.

Юк, кайгырма яшьлек үтә дип тә,
Ялгызлыкта син дә калмассың.
Ашкынулар ялгышлыкка илтә -
Үкенергә Ходай язмасын.

Ярат үзең, яратсыннар сине,
Яшәсеннәр син дип җан атып.
Кирәк булган өчен генә түгел,
Кияүгә чык бары яратып.
```

# Перевод

### Айдар Минхаджев / Айдар Тимербаев - Выходи замуж только по любви

```
Просыпаешься, увидев во сне,
Что влюбилась, мол, в кого-то ты.
Нет, не спеши говорить «нужен ты»,
Рано ещё говорить «люблю».

Люби сама, пусть любят тебя,
Пусть живут, душой к тебе стремясь.
Не только потому, что нужна ты,
Выходи замуж только по любви.

Нет, не ищи, не выбирай милого,
Прельстившись золотом и серебром.
Не разгоришься ты огнём любви,
Богатством нельзя душу согреть.

Люби сама, пусть любят тебя,
Пусть живут, душой к тебе стремясь.
Не только потому, что нужна ты,
Выходи замуж только по любви.

Нет, не печалься, что молодость уходит,
В одиночестве ты не останешься.
Поспешность к ошибкам приводит -
Да не напишет Бог сожалений.

Люби сама, пусть любят тебя,
Пусть живут, душой к тебе стремясь.
Не только потому, что нужна ты,
Выходи замуж только по любви.
```
