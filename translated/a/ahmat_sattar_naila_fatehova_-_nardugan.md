# Оригинал

### Әхмәт Саттар / Наилә Фатехова - Нардуган

```
Лә-лә-лә-лә-лә
Нардуган, Нардуган,
Нардуганда ел туган,
Нардуган, Нардуган,
Нардуганда көн туган.

Кушымта:
Нардуган, әй Нардуган,
Нардуганда ел туган,
Нардуган, әй Нардуган,
Нардуганда көн туган.

Нардуган, Нардуган,
Урамнарда кар, буран.
Нардуган, Нардуган,
Уйный, җырлый Нардуган.

Нардуган, әй Нардуган,
Нардуганда ел туган,
Нардуган, әй Нардуган,
Нардуганда мин туган.
```

# Перевод

### Ахмат Саттар / Наилә Фатехова - Нардуган

```
Ла-ла-ла-ла-ла
Нардуган, Нардуган,
В Нардуган родился год,
Нардуган, Нардуган,
В Нардуган родился день.

Припев:
Нардуган, ой Нардуган,
В Нардуган родился год,
Нардуган, ой Нардуган,
В Нардуган родился день.

Нардуган, Нардуган,
На улицах снег, буран.
Нардуган, Нардуган,
Пляшет, песни поёт Нардуган.

Нардуган, ой Нардуган,
В Нардуган родился год,
Нардуган, ой Нардуган,
В Нардуган родилась я.
```
