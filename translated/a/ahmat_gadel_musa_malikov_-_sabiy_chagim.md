# Оригинал

### Әхмәт Гадел / Муса Маликов - Сабый чагым

```
Ниләргә икән сине сагынам,
Бәгърем өзелә, җилләр иссәләр.
Әремле урам, туган авылым,
Хуш исле бөтнек, талгын чишмәләр.

Әллә инде мин үсеп җиттем,
Үзем сизмәстән, тоймастан үзем.
Хатирәләрем яшьлеккә илтте,
Йомып ачканчы сабый чак күзен.

Сабый чагыма юк инде юллар,
Офыктан ерак бала чак иле.
Әремле үрләр, газиз авыллар
Һаман дәшәләр, дәшәләр мине.
```

# Перевод

### Ахмат Гадел / Муса Маликов - Моё детство

```
Почему же я скучаю по тебе,
Сердце рвётся, когда ветры налетают.
Улица родная, деревня моя,
Душистая смородина, чистые родники.

Кажется, я уже вырос и созрел,
Сам того не замечая, ненасытный сам.
Воспоминания унесли в молодость,
Закрывая и открывая глаза детства.

К моему детству нет больше дорог,
Страна детства далеко за горизонтом.
Родные холмы, любимые деревни
Всё ещё манят, манят меня.
```
