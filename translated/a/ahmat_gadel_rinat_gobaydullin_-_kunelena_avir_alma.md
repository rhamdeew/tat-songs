# Оригинал

### Әхмәт Гадел / Ринат Гобәйдуллин - Күңелеңә авыр алма

```
Яр дип, яр эзләүчеләр бар.
Яр төрле булганын беләм.
Ярга җиттем дип, упкынга
Очканнар барын беләм.

Күңелеңә авыр алма —
Тормыш түгел, алсу алма.
Күңелеңә авыр алма —
Тормыш түгел, алсу алма.

Наз дип, наз эзләүчеләр бар.
Наз татлы булганын беләм.
Наз таптым дип, борчуларга
Чумганнар барын беләм.

Күңелеңә авыр алма —
Тормыш түгел, алсу алма.
Күңелеңә авыр алма —
Тормыш түгел, алсу алма.

Яр дип яр эзләүчеләр бар
Ярның бер булганын беләм.
Ярга җиттем дип, упкынга
Очканнар барын беләм.

Күңелеңә авыр алма —
Тормыш түгел, алсу алма.
Күңелеңә авыр алма —
Тормыш түгел, алсу алма.
```

# Перевод

### Ахмат Гадел / Ринат Гобәйдуллин - Не принимай близко к сердцу

```
Есть те, кто ищут себе пару.
Я знаю — все они разные.
Я знаю тех, кто думал: «Нашёл!» —
А сгинул в пропасти.

Не принимай близко к сердцу —
Жизнь не яблоко кислое.
Не принимай близко к сердцу —
Жизнь не яблоко кислое.

Есть те, кто ищут нежность.
Я знаю — нежность сладка.
Но знаю тех, кто думал: «Нашёл!» —
А утонул в печали.

Не принимай близко к сердцу —
Жизнь не яблоко кислое.
Не принимай близко к сердцу —
Жизнь не яблоко кислое.

Есть те, кто ищут пару,
Я знаю — пара одна.
Я знаю тех, кто думал: «Нашёл!» —
А сгинул в пропасти.

Не принимай близко к сердцу —
Жизнь не яблоко кислое.
Не принимай близко к сердцу —
Жизнь не яблоко кислое.
```
