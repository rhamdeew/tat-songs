# Оригинал

### Айгөл Вәлиуллина / Илдус Дәүләнов - Югалма

```
Ай, кояш, йолдызлы тормышың
Карага күмелсә кинәттән,
Үзең дә аңламас әрнүең
Чайпалып түгелсә йөрәктән...

Югалма, югалтма үзеңне!
Кан саркып торса (да) ярада,
Син җиргә яшәргә килгәнсең,
Алга бар, артыңа карама.

Исемең, җаныңны тапларлык
Нахакка төрсәләр бервакыт.
Язмышның бу керле уенын
Булмаса һич кенә туктатып...

Югалма, югалтма үзеңне!
Кан саркып торса (да) ярада,
Син җиргә яшәргә килгәнсең,
Алга бар, артыңа карама.

Юлыңны дәвамлы итәргә
Иң соңгы сулышың алганда,
Көч ал син өметтән үрелеп,
Бары тик югалма, югалма!

Югалма, югалтма үзеңне!
Кан саркып торса (да) ярада,
Син җиргә яшәргә килгәнсең,
Алга бар, артыңа карама.
Алга бар, артыңа карама...
```

# Перевод

### Айгуль Валиуллина / Ильдус Дауланов - Не теряйся

```
Луна, солнце, звёздная жизнь твоя
Вдруг погрузится во тьму,
И песня, что сам не поймёшь,
Прольётся волною из сердца...

Не теряйся, не теряй себя!
Пусть кровь сочится из раны,
Ты на землю пришёл, чтобы жить,
Иди вперёд, не оглядывайся назад.

Твоё имя, твою душу
Несправедливо посадят когда-нибудь.
Эту грязную игру судьбы
Никак не остановив...

Не теряйся, не теряй себя!
Пусть кровь сочится из раны,
Ты на землю пришёл, чтобы жить,
Иди вперёд, не оглядывайся назад.

Чтобы продолжить свой путь,
Когда возьмёшь последний вздох,
Черпай силы из надежды,
Только не теряйся, не теряйся!

Не теряйся, не теряй себя!
Пусть кровь сочится из раны,
Ты на землю пришёл, чтобы жить,
Иди вперёд, не оглядывайся назад.
Иди вперёд, не оглядывайся назад...
```
