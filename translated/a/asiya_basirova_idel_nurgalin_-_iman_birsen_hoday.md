# Оригинал

### Асия Басырова / Идел Нургалин - Иман бирсен Ходай

```
Бу дөнъяда ни бар үзең белән,
Фаныйлыктан алып китәрлек.
Иман бирсен Ходай зиһенемә,
Мәңгелеккә генә җитәрлек.

Сабырлыкны бирсә Ходай бирә,
Җанның тынычлыгын юатып,
Ничек туса кеше, шулай китә,
Елап туа, китә елатып.
```

# Перевод

### Асия Басырова / Идель Нургалин - Дай мне веру, Господь

```
Что есть в этом мире с тобою,
Что увезти можно из бренной земли.
Дай мне веру, Господь, в мою душу,
Чтобы хватило лишь до вечности.

Терпение дарует лишь Господь,
Успокаивая душевный покой,
Каким родился человек, таким и уйдет,
Плача родился, уходит в слезах.
```
