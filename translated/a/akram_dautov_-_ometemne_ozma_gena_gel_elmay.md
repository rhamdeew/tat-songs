# Оригинал

### Әкрәм Даутов - Өметемне өзмә генә (Гел елмай)

```
Бер генә елмаеп карадың,
Гомергә калдырдың газабын.

Гел елмай, гел кара күзләремә,
Ялкын сал, ук каза йөрәгемә,
Җил генә, үтмәсен үзәгемә,
Тик өметемне өзмә генә.

Елмаеп биек тау башында,
Торасың кебек син каршымда.

Гел елмай, гел кара күзләремә,
Ялкын сал, ук каза йөрәгемә,
Җил генә, үтмәсен үзәгемә,
Тик өметемне өзмә генә.
```

# Перевод

### Акрам Даутов - Не обрывай надежды моей (Всё улыбайся)

```
Всего лишь раз улыбкой одарила,
На всю мне жизнь страданье подарила.

Всё улыбайся, всё смотри в мои глаза,
Зажги огонь, пусть сердце жаром обожжёт,
Пусть будет ветром только, не пройдёт сквозь меня,
Только надежды моей не обрывай.

Улыбаясь на вершине той горы высокой,
Ты стоишь передо мной.

Всё улыбайся, всё смотри в мои глаза,
Зажги огонь, пусть сердце жаром обожжёт,
Пусть будет ветром только, не пройдёт сквозь меня,
Только надежды моей не обрывай.
```
