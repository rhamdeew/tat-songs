# Оригинал

### Айдар Минһаҗев / Фәнис Мортазин - Сау бул

```
Син - йолдызга,
Ник үрелим?
Яратсам да, дәшмәдем.
Төштә күрәм сине генә,
Ә син никтер мине генә
Күрмәгәндәй яшәдең.

Соңгы сүзем "сау бул" инде,
Миңа кирәктер шул гына,
Сау бул инде, күпме түздем,
Бүген өметемне өздем,
Сау бул, сау бул,
Сау бул, тик бәхетле бул!

Бергә булу язмагандыр,
Бәхет сиңа теләдем.
Бүләк итәм сиңа гөлләр,
Узды өмет тулы көннәр,
Оч иреккә йөрәгем.

Үкенәсе булыр микән,
Нигә элек дәшмәдем?
Синең белән hәрчак уем?
Сине күрү миңа кыен,
Тик күренмәс яшьләрем.
```

# Перевод

### Айдар Минһаҗев / Фанис Мортазин - Прощай

```
Ты - словно звезда,
Как мне дотянуться?
Хоть и любил, не сказал.
Во сне вижу только тебя,
А ты почему-то меня
Словно не замечая жила.

Последнее слово "прощай" теперь,
Мне нужно только это,
Прощай же, сколько я терпел,
Сегодня надежду свою оборвал,
Прощай, прощай,
Прощай, но будь счастлива!

Не суждено нам быть вместе,
Счастья тебе желаю.
Дарю тебе цветы,
Прошли дни, полные надежд,
Пусть моё сердце летит на волю.

Будет ли мне жаль,
Почему раньше не сказал?
Всегда ли буду о тебе думать?
Трудно мне видеть тебя,
Но не видны мои слёзы.
```
