# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Син назлар эзләмә

```
Син назлар эзләмә үземнән,
Ышаныч җепләре өзелгән.
Тарсынып җаныма кагылам,
Син идең яратып табылган.

Елларым үттеләр, киттеләр,
Күземнән яшьләрем киптеләр.
Җыелган ачуым таралды,
Каралган болытлар агарды.

Син идең бәхетем чаткысы,
Сөюнең яктысы, татлысы.
Ник салып таптадың хисләрне,
Сагышлы иттең син кичләрне.

Елларым үттеләр, киттеләр,
Күземнән яшьләрем киптеләр.
Җыелган ачуым таралды,
Каралган болытлар агарды.

Син назлар эзләмә үземнән,
Ышаныч җепләре өзелгән.
Тарсынып җаныңа кагылам,
Син идең яратып табылган.

Елларым үттеләр, киттеләр,
Күземнән яшьләрем киптеләр.
Җыелган ачуым таралды,
Каралган болытлар агарды.
```
