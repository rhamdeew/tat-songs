# Оригинал

### Анна Попова / Оскар Усманов - Бәхетле булу серләре

```
Һәр кеше бәхетен эзли,
Чын мәхәббәтен көтә.
Саф йөрәкләр таба ала
Тиң ярын меңнәрдән дә.

Бәхетле булу серләре
Күп түгел, гади генә.
Яратып һәм яратылып
Яшәргә кирәк җирдә.

Һәр туган таңны елмаеп
Каршы алыйк сөенеп.
Күңелдә ачу сакламый
Яшик кичерә белеп.

Бәхетле булу серләре
Күп түгел, гади генә.
Яратып һәм яратылып
Яшәргә кирәк җирдә.

Бәхетле булсаң бүлеп бир
Бер өлешен дустыңа.
Буләк иткән саен бәхет
Сиңа да арта бара.
```

# Перевод

### Анна Попова / Оскар Усманов - Секреты счастья

```
Каждый ищет своё счастье,
Ждёт настоящей любви.
Чистые сердца находят
Свою половинку из тысяч.

Секреты счастья
Не так уж сложны, просты они.
Любить и быть любимым -
Вот как нужно жить на земле.

Каждый рассвет с улыбкой
Встречаем мы, радуясь.
Не храня обид в душе,
Живём, умея прощать.

Секреты счастья
Не так уж сложны, просты они.
Любить и быть любимым -
Вот как нужно жить на земле.

Если счастлив ты, подели
Частицу его с другом.
Чем больше даришь счастья,
Тем больше его к тебе возвращается.
```
