# Оригинал

### Айдар Минһаҗев / Марат Мухин - Оныт әле аны

```
Минем яныма дип килгәндә ул
Ник чыгасың аның юлына.
Яратмый ул сине, сөя бары мине
Минем йөрәктә дә ул гына.

Оныт әле, оныт әле аны
Карама син аның күзенә,
Ул бит инде мине, бары мине сөя
Беләсең бит инде үзең дә.

Элек без бит якын дуслар идек
Хәзер инде дәшми йөрисең.
Сөймәгән ул сине, сөйсә китмәс иде
Нигә инде шуны белмисең.

Оныт әле, оныт әле аны
Карама син аның күзенә,
Ул бит инде мине, бары мине сөя
Беләсең бит инде үзең дә.

Аны көтеп вакыт уздырма син
Әрәм уза инде яшьлегең.
Ул бит мине генә, бары мине сөя
Мин дә аның өчен яшимен.
```

# Перевод

### Айдар Минһаҗев / Марат Мухин - Забудь уже её

```
Когда она идёт ко мне,
Зачем ты выходишь ей навстречу.
Не любит она тебя, любит только меня,
И в моём сердце лишь она одна.

Забудь уже, забудь уже её,
Не смотри ты ей в глаза,
Она ведь теперь меня, только меня любит,
Ты ведь и сам уже это знаешь.

Раньше мы были близкими друзьями,
Теперь уже и не разговариваешь.
Не любила она тебя, если б любила - не ушла бы,
Почему же ты этого не понимаешь.

Забудь уже, забудь уже её,
Не смотри ты ей в глаза,
Она ведь теперь меня, только меня любит,
Ты ведь и сам уже это знаешь.

Не трать время, ожидая её,
Твоя молодость уже проходит напрасно.
Она ведь только меня, только меня любит,
И я для неё живу.
```
