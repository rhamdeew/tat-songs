# Оригинал

### Әнисә Таһирова / Салават Низаметдинов - Могҗизага килә ышанасым

```
Күкне каплап алган кара болыт,
Җирдә салкын томан, җил-явым.
Баш очында кояш балкыр әле,
Могҗизага килә ышанасым.

Күңел күкләрендә сагыш йөзә,
Сүнәр - сүнмәс яна учагым.
Учакларым дөрләп янар әле,
Могҗизага килә ышанасым.

Югалдылар якты хыялларым,
Күпме сүнеп, күпме токанды.
Югалганым бер табылыр әле,
Могҗизага килә ышанасым.
```

# Перевод

### Аниса Тахирова / Салават Низаметдинов - Хочу верить в чудеса

```
Небо затянуло тучей черной,
На земле холодный туман, ветер и дожди.
Над головою солнце светит всё же,
Хочу верить в чудеса.

В небесах души печаль плывёт,
То гаснет, то горит мой очаг.
Мои огни пылают ещё жарко,
Хочу верить в чудеса.

Потерялись светлые мечты мои,
Сколько их погасло, сколько утихло.
Но потерянное ещё найдётся,
Хочу верить в чудеса.
```
