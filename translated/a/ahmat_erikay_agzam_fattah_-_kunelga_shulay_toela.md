# Оригинал

### Әхмәт Ерикәй / Әгъзам Фәттах - Күңелгә шулай тоела

```
Яз көне яфрак яра тал,
Көз килгәч яме коела,
Һәрвакыт язгы чәчәк син,
Күнелгә шулай тоела.

Йолдызлар, күктән сибеп нур,
Балкыйлар төннәр буена.
Тик синнән якты йолдыз юк,
Күнелгә шулай тоела.

Болында яңгырый матур җыр,
Шат яшьләр бергә җыела.
Тик синнән оста җырчы юк,
Күңелгә шулай тоела.

Син генә җәйге алсу таң,
Син генә алтын кыйбла.
Ашкынып көткән кешем син,
Күңелгә шулай тоела.
```

# Перевод

### Ахмат Ерикей / Агзам Фаттах - Так душе кажется

```
Весной листвою зеленеет ива,
Осенью с неё листва спадает,
Но ты всегда весенний цветок,
Так душе моей кажется.

Звёзды, рассыпав с неба свет,
Сияют в ночи напролёт.
Но ярче звёзд тебя нет,
Так душе моей кажется.

На лугу звучит прекрасная песня,
Весёлая молодёжь собирается вместе.
Но певца искуснее тебя нет,
Так душе моей кажется.

Ты одна — летняя алая заря,
Ты одна — золотая кибла.
Ты — мой желанный человек,
Так душе моей кажется.
```
