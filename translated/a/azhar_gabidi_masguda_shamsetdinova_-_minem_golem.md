# Оригинал

### Әзһәр Габиди / Мәсгудә Шәмсетдинова - Минем гөлем!

```
Тәңкә карларны себереп,
Тышта буран котыра,
Тәрәзә төбендә гөлем
Ал чәчәктә утыра.

Янып торган чәчәкләре
Булмәбезне балкыта,
Тышта январь, ә чәчәкләр
Җәй исләре аңкыта.
```

# Перевод

### Азһар Габиди / Масгуда Шамсетдинова - Мой цветок!

```
Рассыпая снег монетами,
Вьюга на дворе бушует,
А на подоконнике цветок мой
Алым цветом красуется.

Горящие его лепестки
Озаряют нашу комнату,
За окном январь, а цветы
Напоминают запахи лета.
```
