# Оригинал

### Айгөл Гимадиева / Эльвира Хаматнурова - Кыш бабай (икенче вариант)

```
Серле әкиәт иленән
Юлға сыға бер бабай.
Тәҙ(е)рәнән күҙен алмай
Уны көтә һәр малай.

Аҡ һаҡал, тылсым таяҡ
Һәр кем уға бик ныҡ шат.
Исеме уның ябай
Кем була ул?
- Кыш бабай!!!

Ҡыңғыраулы пар аттарҙа
Оса, елә, ашҡына.
Өр-яңы йыл алып килә
Матур теләктәр телә!

Аҡ һаҡал, тылсым таяҡ
Һәр кем уға бик ныҡ шат.
Исеме уның ябай
Кем була ул?
- Кыш бабай!!!

Мин теләйем һәр-бер өйҙә
Өҫтәл булһын түңәрәк.
Һәр баланы көтһөн, әйҙә,
Шыршы төбөндә бүләк.

Аҡ һаҡал, тылсым таяҡ
Һәр кем уға бик ныҡ шат.
Исеме уның ябай
Кем була ул?
- Кыш бабай!!!
```

# Перевод

### Айгуль Гимадиева / Эльвира Хаматнурова - Дед Мороз (второй вариант)

```
Из страны волшебных сказок
Выходит в путь один дедок.
Не сводя глаз с окна
Его ждёт каждый мальчишка.

Белая борода, волшебный посох
Ему все очень рады.
Имя его простое
Кто это?
- Дед Мороз!!!

На звенящей паре коней
Мчится, летит, спешит.
Новый год несёт с собой
Прекрасные пожелания дарит!

Белая борода, волшебный посох
Ему все очень рады.
Имя его простое
Кто это?
- Дед Мороз!!!

Я желаю в каждом доме
Стол чтоб был накрыт большой.
Каждого ребёнка пусть ждёт
Под ёлкой подарок.

Белая борода, волшебный посох
Ему все очень рады.
Имя его простое
Кто это?
- Дед Мороз!!!
```
