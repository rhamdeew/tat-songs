# Оригинал

### Әхмәт Ерикәй / Заһид Хәбибуллин - Айлы төн

```
Җәйге төн, якты төн,
Күктә ай елмая.
Шул айның шәүләсе
Тирбәлә елгада.

Ай күрә чәчәкле,
Бизәкле кырларны;
Көнләшеп тыңлый ул
Җирдәге җырларны,

Сандугач үзенең
Күңелле җырында
Белдерә сөмбелне
Сөюе турында.

Ә сөмбел оялчан,
Җырламый, көйләми,
Хисләре турында
Кешегә сөйләми.

Гармунчы төн буе.
Сыздыра, түзми ул.
Кемнедер юксына,
Нәрсәдер эзли ул.

Юксынма, моңайма,
Ташламас чибәрен.
Уйна син, уйна син,
Кавышу көйләрен!..
```

# Перевод

### Ахмат Ерикей / Захид Хабибуллин - Лунная ночь

```
Летняя ночь, светлая ночь,
В небе луна улыбается.
Её сияние
Колышется в реке.

Луна видит цветущие,
Украшенные поля;
С удовольствием слушает она
Земные песни.

Соловей в своей
Радостной песне
Признаётся василисе
В своей любви.

А василиса стеснительна,
Не поёт, не напевает,
О своих чувствах
Никому не рассказывает.

Гармонист всю ночь
Выводит, не может остановиться.
Кого-то он скучает,
Что-то ищет.

Не скучай, не грусти,
Не оставит красавицу.
Играй же, играй же
Мелодии встречи!..
```
