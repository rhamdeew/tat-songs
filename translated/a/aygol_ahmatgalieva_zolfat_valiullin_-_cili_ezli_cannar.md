# Оригинал

### Айгөл Әхмәтгалиева / Зөлфәт Вәлиуллин - Җылы эзли җаннар

```
Кулларымнан багып юрамагыз,
Сорамагыз ничек диеп хәлең.
Мәхәббәтне эзләп, җылы эзләп,
Учларымда учак яккан мәлем.

Кушымта:
Мәхәббәткә табынганнар
Минем хәлне бәлки аңлар.
Ак бураннар кочагында
Җылы эзләп йөри җаннар.

Күбәләктәй очам ут-ялкынга,
Ай күрер дә якты йолдыз алыр...
Тик мәхәббәт, көмеш бер чың булып,
Безнең җанның җылысында калыр...
```

# Перевод

### Айгуль Ахметгалиева / Зульфат Валиуллин - Души ищут тепла

```
Не держите меня за руки,
Не спрашивайте, как мои дела.
Любовь ищу, тепло ищу,
В моих краях костёр зажгла.

Припев:
Кто любви поклоняется,
Меня, быть может, поймёт.
В объятиях белой метели
Тепло душа ищет и ждёт.

Как мотылёк лечу к огню-пламени,
Луна увидит, звезда заберёт...
Но любовь, словно серебряный звон,
В тепле наших душ навсегдаживёт...
```
