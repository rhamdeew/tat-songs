# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Ике болыт

```
Һаваларда ике болыт
Ике якка агалар.
Алар да безнең шикелле
Ерагая баралар.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.

Кавыша (а)лмый ике болыт,
Эх, белергә уйларын.
Юатыйм диеп үреләм,
Җитмәсә дә буйларым.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.

Ике болыт мөлдерәмә,
Туладыр күңелләре.
Бәлки, күрештерми калмас,
Кояш бит сүрелмәде.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.
```

# Перевод

### Айдар Минхаджев / Айдар Тимербаев - Два облака

```
В небесах два облака
В разные стороны плывут.
Они, как мы с тобою,
Вдаль уходят, разлучаясь.

Между двух облаков
Мост построить я хочу.
По тому мосту с тобою
Вместе остаться я хочу.
По тому мосту с тобою
Вместе остаться я хочу.

Не встречаются два облака,
Эх, узнать бы их мысли.
Утешить их стремлюсь,
Хоть не дотянуться мне.

Между двух облаков
Мост построить я хочу.
По тому мосту с тобою
Вместе остаться я хочу.
По тому мосту с тобою
Вместе остаться я хочу.

Два облака прозрачны,
Полны их сердца тоской.
Может, встреча не оставит,
Ведь солнце не погасло.

Между двух облаков
Мост построить я хочу.
По тому мосту с тобою
Вместе остаться я хочу.
По тому мосту с тобою
Вместе остаться я хочу.
```
