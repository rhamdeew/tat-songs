# Оригинал

### Әхсән Баян / Фуат Әбүбәкеров - Зәңгәр кыңгыраулар

```
Кырда йөргән чакта кыңгырау гөл
Бүләк иткәннәрең истә гел.
Шуннан бирле, сине сагынганда
Кыңгыраулар чыңын ишетәмен.

Җирдә, күктә, кая барсам — шунда
Уем синең белән очраша.
Бөтен хыялларым чынга аша
Зәңгәр кыңгыраулар чыңласа.

Син биеккә очып, күздән киткәч,
Чыңлаганын күкнең ишеттем.
Күңелемне дәшкән күк гөмбәзен
Мин, кыңгырау, диеп хис иттем.

Җирдә, күктә, кая барсам — шунда
Уем синең бешн очраша.
Бөтен хыялларым чынга аша
Зәңгәр кыңгыраулар чыңласа.
```

# Перевод

### Ахсан Баян / Фуат Абубакеров - Синие колокольчики

```
Когда гуляли в поле, колокольчик-цветок
Ты мне дарила — помню всё.
С тех самых пор, скучая по тебе,
Я слышу звон колокольчиков.

На земле, в небесах, куда ни пойду — там
Мои мысли с тобою встречаются.
Все мои мечты становятся явью,
Когда звенят синие колокольчики.

Когда ты улетела ввысь, скрылась из глаз,
Я услышал, как зазвенело небо.
Небесный купол, тронувший мою душу,
Я ощутил как колокольчик.

На земле, в небесах, куда ни пойду — там
Мои мысли с тобою встречаются.
Все мои мечты становятся явью,
Когда звенят синие колокольчики.
```
