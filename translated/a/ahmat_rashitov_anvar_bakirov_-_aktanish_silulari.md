# Оригинал

### Әхмәт Рәшитов / Әнвәр Бакиров - Актаныш сылулары

```
Актаныш кызлары белән,
Ах, таныш булулары!
Күпме егетне саргайта
Актаныш сылулары.

Ярлар эзләп илләр гиздем,
Ятларга күз атмадым.
Актаныш кызлары кебек
Гүзәлләрне тапмадым.

Агыйделдә үскән мәллә,
Суларын эчкән мәллә.
Таң алсуы йөзләренә
Нур булып күчкән мәллә?

Актаныш кызлары белән,
Ах, таныш булулары!
Күпме егетне саргайта
Актаныш сылулары.
```

# Перевод

### Ахмат Рашитов / Анвар Бакиров - Актанышские красавицы

```
С девушками Актаныша,
Ах, познакомиться бы!
Скольких парней томят тоской
Актанышские красавицы.

Искал любимых, страны объездил,
На чужих даже не смотрел.
Как актанышские девушки,
Красавиц таких не встречал.

На Агидели выросли, может,
Воды её испили, может.
Зарево зари на их лица
Светом перешло, может?

С девушками Актаныша,
Ах, познакомиться бы!
Скольких парней томят тоской
Актанышские красавицы.
```
