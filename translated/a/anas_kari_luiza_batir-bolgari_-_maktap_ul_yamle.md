# Оригинал

### Әнәс Кари / Луиза Батыр-Болгари - Мәктәп — ул ямьле

```
Мәктәп — ул ямьле,
Бик күңелле, ди,
һичкемнең аннан
Кайтасы килми.
Анда әти күк
Олы абый бар,
Аны "Укытучы" дип
Бар(ы)сы атыйлар.

Балаларны ул
Сөеп укыта,
Аларга китап,
Дәфтәр тарата.
Иң кирәклесе
"Әлифба" икән,
Белем ачкычы бит
Шун(ы)сында икән.

Мәктәп — ул ямьле,
Бик күңелле, ди,
һич кемнең аннан
Кайтасы килми.
Белемле итәр
Мине ул мәктәп,
"Әлифба" дигән
Кадерле китап.
```

# Перевод

### Анас Кари / Луиза Батыр-Болгари - Школа прекрасна

```
Школа прекрасна,
Так радостна, говорят,
Никому не хочется
Из неё уходить.
Там, словно добрый дедушка,
Старший брат есть,
Его «Учителем»
Все называют.

Детей он
С любовью учит,
Им книги раздаёт
И тетради.
Самая нужная —
«Алифба», оказывается,
Ключ к знаниям ведь
Именно в ней.

Школа прекрасна,
Так радостна, говорят,
Никому не хочется
Из неё уходить.
Образованным сделает
Меня эта школа,
«Алифба» — вот
Драгоценная книга.
```
