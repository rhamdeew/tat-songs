# Оригинал

### Абдулла Сәләхетдинов / Җәүдәт Фәйзи - Җәлил йолдызлары

```
Озак килә озын юллардан
Җиргә ерак йолдыз нурлары…
Ерак йолдыз булып кабынган —
Әсирлектә илен сагынган
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.

Узган чакта озын юллардан
Сүрелмиләр йолдыз нурлары…
Йолдыз нуры кебек саф калган —
Тап төшерми антын саклаган
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.

Озак килә озын юллардан
Мәңге сүнми йолдыз нурлары…
Сүнмәс йолдыз нуры шикелле,
Җир йөзенә балкып сирпелде
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.
```

# Перевод

### Абдулла Салахетдинов / Джаудат Файзи - Звёзды Джалиля

```
Издалека, с дальних дорог
До земли свет звёздный льётся…
Звездой далёкой воспылав —
В плену Родину вспоминав,
Джалиля бессмертные…
Джалиля бессмертные песни.

Сквозь времена, с дальних дорог
Не меркнет звёздное сиянье…
Чист, как звёздный свет, остался —
Клятве данной не изменялся,
Джалиля бессмертные…
Джалиля бессмертные песни.

Издалека, с дальних дорог
Навеки светит звёздный луч…
Как вечный свет звезды нетленной,
Над всей землёй сияньем пенным
Джалиля бессмертные…
Джалиля бессмертные песни.
```
