# Оригинал

### Әлфирә Бикмәтова / Булат Махиянов - Туймазы

```
Туймазының җир астыннан
Ургыла кара алтын.
Шул алтынга җан бирүче
Егетләр уттай ялкын.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.

Туймазы уңганнарына
Буйсына Урал ташы.
Алар ясаган вазалар,
Әйтерсең, йөзек кашы.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.

Туймазыны данга күмә
Алдынгы маяклары.
Эшләргә дәрт, илһам бирә
Җырларга бай яклары.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.
```

# Перевод

### Альфира Бикматова / Булат Махиянов - Туймазы

```
Из недр Туймазов земных
Бьёт фонтаном чёрное золото.
Джигиты, что дарят ему жизнь,
Словно пламя, горячи и молоды.

Ах, Туймазы, Туймазы,
Прекрасные берега Усеня.
Любя тебя, в песню вплетаю
Сокровенные думы свои.

Перед мастерами Туймазов
Склоняется уральский камень.
Вазы, что они создают,
Словно оправы драгоценных колец.

Ах, Туймазы, Туймазы,
Прекрасные берега Усеня.
Любя тебя, в песню вплетаю
Сокровенные думы свои.

Туймазы славой увенчаны,
Маяки их в авангарде идут.
Вдохновение и силу для труда дарят
Края, богатые песнями.

Ах, Туймазы, Туймазы,
Прекрасные берега Усеня.
Любя тебя, в песню вплетаю
Сокровенные думы свои.
```
