# Оригинал

### Әминә Бикчәнтәева / Әнвәр Бакиров - Беренче яңгыр

```
Чыгыгыз урамга, нәниләр,
Беренче яңгырны күрегез.
Мин үсәм, яңгырда үсәм, дип,
Кинәнеп, яланбаш йөрегез!

Яңгыр, яу, яу, әйдә, гөрелдәп,
Чиләкләр тулсыннар мөлдерәп.
Бөтен җир тузаннан арынсын!
Яшәреп бөреләр ачылсын!
```

# Перевод

### Амина Бикчантаева / Анвар Бакиров - Первый дождь

```
Выходите на улицу, малыши,
Посмотрите на первый дождь.
Я расту, под дождём расту, говоря,
Распустив волосы, босиком бегите!

Дождь, лей, лей же, давай, громыхая,
Пусть вёдра наполнятся, прозрачные.
Пусть вся земля очистится от пыли!
Пусть зеленея, почки распустятся!
```
