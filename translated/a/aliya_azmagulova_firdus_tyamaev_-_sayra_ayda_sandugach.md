# Оригинал

### Алия Әзмәгулова / Фирдүс Тямаев - Сайра, әйдә, сандугач

```
Бакча артындагы талда
Өздереп сайрый бер кош,
Сайрый ул җәйләрдә генә,
Ишетелми көзен-кыш.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Сайрый минем сандугачым,
Күңелдә эз калдырып,
Сайра, иркәм, син дә шулай,
Йөрәгемне яндырып.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Иркәм килми, хәбәр бирми,
Сандугач та сайрамый,
Йөрәккәем синнәң башка
Башкаларны сайламый.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Бакча артындагы талда
Өздереп сайрый бер кош,
Сайрый ул җәйләрдә генә,
Ишетелми көзен-кыш.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.
```

# Перевод

```
На иве за садом дальним
Заливается одна птица,
Поёт она лишь летом,
Не слышна осенью и зимой.

Давай, давай, соловей,
Исполни моё желание,
Только ты понимаешь
Моё молодое сердце.

Поёт мой соловушка,
Оставляя след в душе,
Пой, милый мой, и ты так же,
Разжигая моё сердце.

Давай, давай, соловей,
Исполни моё желание,
Только ты понимаешь
Моё молодое сердце.

Милый не приходит, весточки не шлёт,
И соловей не поёт,
Моё сердечко кроме тебя
Других не выбирает.

Давай, давай, соловей,
Исполни моё желание,
Только ты понимаешь
Моё молодое сердце.
```
