# Оригинал

### Алсу Шәйдуллина / Марсель Вәгыйзов - Яшисе килә (бишенче вариант)

```
Яшисе килә шулкадәр,
Алда бик зур бурычлар.
Тыныч булсын илем күге,
Гүзәл булсын тормышлар.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.

Яшисе килә шулкадәр,
Насыйп булсын гомерләр,
Тулы, тигез, чиксез бәхет,
Якты кояшлы көннәр.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.

Яшисе килә шулкадәр,
Кеше булып, көч биреп.
Күңелемдәге барлык моңны
Җырларым аша биреп.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.
```

# Перевод

### Алсу Шайдуллина / Марсель Вагизов - Так хочется жить (пятый вариант)

```
Так хочется жить,
Впереди столько важных дел.
Пусть небо родины будет мирным,
Пусть жизнь будет прекрасной.

Так хочется жить,
Делая мир красивее.
Каждое утро просыпаться
С радостью и счастьем.

Так хочется жить,
Пусть судьба подарит годы,
Полные, ровные, безграничное счастье,
Светлые солнечные дни.

Так хочется жить,
Делая мир красивее.
Каждое утро просыпаться
С радостью и счастьем.

Так хочется жить,
Быть человеком, дарить силы.
Всю мелодию в моей душе
Передать через мои песни.

Так хочется жить,
Делая мир красивее.
Каждое утро просыпаться
С радостью и счастьем.
```
