# Оригинал

### Абдулхак Игебаев / Заһир Исмәгыйлев - Тәүге сөю

```
Мөмкинме соң дәртле йөрәк хисен
Гади сүзләр белән аңлату?
Иң кайнары яшьлек җырларымның —
Тәүге сөю, тәүге ярату.

Шатлык кына түгел, кайгыны да
Мөмкин әле, дуслар, оныту.
Иң җуелмас эзләр калдырганы —
Тәүге сөю, тәүге ярату.

Нигә шулай матур була икән
Яз аенда балкып таң ату?
Иң аязы яшьлек таңнармның —
Тәүге сөю, тәүге ярату.
```

# Перевод

### Абдулхак Игебаев / Захир Исмагиев - Первая любовь

```
Возможно ли трепетное чувство сердца
Простыми словами выразить?
Самое горячее в песнях юности моей —
Первая любовь, первое влечение.

Не только радость, но и печаль
Можно, друзья, ещё забыть.
Но неизгладимый след оставила —
Первая любовь, первое влечение.

Почему так прекрасно бывает,
Когда весной сияет рассвет?
Самый светлый из юношеских зорь —
Первая любовь, первое влечение.
```
