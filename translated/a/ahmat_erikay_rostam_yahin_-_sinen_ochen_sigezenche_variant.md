# Оригинал

### Әхмәт Ерикәй / Рөстәм Яхин - Синең өчен (сигезенче вариант)

```
Синең өчен кояш балкып торсын,
Синең өчен янсын йолдызлар.
Көндезләрен, кичен синең өчен
Бер куаныр йөрәк, бер сызлар.

Синең өчен гөлләр үстерермен,
Юлларыңда юлдаш булырмын.
Синең бәх(е)тең өчен, синең өчен
Тормыш утларына керермен.

Синең өчен урын түрдән булыр
Йөрәктә дә, матур җырда да.
Синең өчен тудым, синең өчен,
Синең өчен яшим дөньяда.
```

# Перевод

### Ахмат Ерикей / Рустам Яхин - Для тебя (восьмой вариант)

```
Для тебя пусть солнце ярко светит,
Для тебя пусть звёзды горят.
Днём и вечером для тебя
То ликует сердце, то болит.

Для тебя я буду розы выращивать,
На путях твоих быть спутником.
Ради счастья твоего, для тебя
В огонь я жизни брошусь.

Для тебя почётное место будет
И в сердце, и в прекрасной песне.
Для тебя родился, для тебя,
Для тебя живу я в этом мире.
```
