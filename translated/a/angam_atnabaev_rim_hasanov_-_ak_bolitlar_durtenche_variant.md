# Оригинал

### Ангам Атнабаев / Рим Хәсәнов - Ак болытлар (дүртенче вариант)

```
Баш очыннан ак болытлар ага,
Ак болытлар — безнең балачак.
Балачакка кеше кулы җитми,
Ул мәңгегә ап-ак калачак,
Ак болытлар — безнең балачак.

Баш очыннан ак болытлар ага,
Ак болытлар — безнең ярату.
Яратуны ташка бәйләп булмый,
Яратуның була канаты,
Ак болытлар — безнең ярату.
```

# Перевод

### Ангам Атнабаев / Рим Хасанов - Белые облака (четвёртый вариант)

```
Над головой белые облака плывут,
Белые облака — наше детство.
До детства не дотянется рука,
Оно навеки белым останется,
Белые облака — наше детство.

Над головой белые облака плывут,
Белые облака — наша любовь.
Любовь к камню не привяжешь,
У любви есть крылья,
Белые облака — наша любовь.
```
