# Оригинал

### Башкорт халык җыры - Карабай* (өченче вариант)

```
Там, там, тамчы, там,
Тамганыңны яратам.
Синең тамганыңны карап,
Күңелемне юуатам.

Иртә дә кил, кич тә кил,
Тәрәзәмне чиртә кил.
Чиртүеңнән танырмын да
Каршы чыгып алырмын.

Ике килен - килендәш,
Берсе ялкау, берсе яшь.
Тарак юк дип җавап бирә
Чәч тарарга иренгәч.

* - шул көйне иҗат итүче курайчының исеме
```

# Перевод

### Башкирская народная песня - Карабай* (третий вариант)

```
Там-там, капля, там,
Тоскую по твоим следам.
Твой след смотрю, не отходя,
Сердце таю, как вода.

Рано приди, поздно приди,
К утру ко мне постучи.
Твой стук услышу я у двери,
Встречу тебя, как в первый раз.

Две невестки - подруги,
Одна ленива, другая молода.
Говорит, что гребня нет,
И волосы расчесать боится.

* - имя композитора этой мелодии
```
