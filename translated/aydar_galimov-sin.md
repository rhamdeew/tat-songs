# Оригинал

### Айдар Галимов - Син

```
Син, син, син,
Син - минем тормышым.
Аткан ал таным син.
Син, син, син,
Сулаган сулышым,
Яшэгэн жаным син.

Сина мин инде -
Язмышын, олешен.
Кайларга китсэн дэ
Мин - синен йорэк тибешен.

Син, син, син,
Бер юлдан бергэлэп
Атларга тиеш без.
Син, син, син,
Адымнар ялгышмыйк.
Абынмыйк, юл тигез.

Сина мин инде -
Язмышын, олешен.
Кайларга китсэн дэ
Мин - синен йорэк тибешен.

Син, син, син,
Яши алмабыз бит
Мин синсез, син минсез.
Син, син, син,
Сузыйк тормыш жебен
Тоерсез, озексез.

Сина мин инде -
Язмышын, олешен.
Кайларга китсэн дэ
Мин - синен йорэк тибешен.
```

# Перевод

### Айдар Галимов - Ты

```
Ты, ты, ты,
Ты — моя жизнь.
Занявшийся алый рассвет — ты.
Ты, ты, ты,
Взятое дыхание,
Жившая душа — ты.

Тебе я уже —
Судьба твоя, доля.
Куда бы ни ушла,
Я — твоего сердца биение.

Ты, ты, ты,
Одной дорогой вместе
Идти должны мы.
Ты, ты, ты,
Шагов не ошибёмся.
Не споткнёмся, дорога ровна.

Тебе я уже —
Судьба твоя, доля.
Куда бы ни ушла,
Я — твоего сердца биение.

Ты, ты, ты,
Жить не можем ведь
Я без тебя, ты без меня.
Ты, ты, ты,
Протянем жизни нить
Без узлов, без разрывов.

Тебе я уже —
Судьба твоя, доля.
Куда бы ни ушла,
Я — твоего сердца биение.
```
