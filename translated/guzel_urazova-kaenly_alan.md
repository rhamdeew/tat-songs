# Оригинал

### Гузел Уразова - Каенлы алан

```
Каенлы алан чакыра безне,
Сагына сезне алан.
Анда минем дусларым бар,
Ашкына кунел хаман.

Кайда гына булсам да
Сагынам каеннарны.
Шул аланда сагынам
Ин изге тойгыларны.

Тынларга иде шунда барып
Ак каеннын шаулавын.
Кайтмыйм эле тынламыйча
Анда кошлар сайравын.

Кайда гына булсам да
Сагынам каеннарны.
Шул аланда сагынам
Ин изге тойгыларны.

Кунеллэрне жилкендерэ
Кичке каен шэулэсе.
Шул аланда сокланырлык
Ак каеннын гэудэсе.

Кайда гына булсам да
Сагынам каеннарны.
Шул аланда сагынам
Ин изге тойгыларны.
```

# Перевод

### Гузел Уразова - Берёзовая поляна

```
Берёзовая поляна зовёт нас,
По нам скучает поляна.
Там мои друзья,
Сердце к ним стремится.

Где бы я ни была,
Скучаю по берёзам.
В той поляне скучаю
По самым святым чувствам.

Слушать бы там, придя,
Шёпот белых берёз.
Не вернусь не послушав
Птиц пение среди берёз.

Где бы я ни была,
Скучаю по берёзам.
В той поляне скучаю
По самым святым чувствам.

Души полонят
Вечерние лучи берёз.
В той поляне восхитительны
Белые стволы берёз.

Где бы я ни была,
Скучаю по берёзам.
В той поляне скучаю
По самым святым чувствам.
```
