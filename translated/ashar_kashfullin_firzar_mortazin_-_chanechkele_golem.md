# Оригинал

### Әсхәр Кәшфуллин / Фирзәр Мортазин - Чәнечкеле гөлем

```
Чәнечкеле гөлем
Чәчәк ата төнен,
Тәүлек кенә аның гомере.
Күзен күккә төбәп,
Шатлык моңын көйләп,
Сүнә чәчәк, сүнә тилмереп.

Чәнечкеле гөлем
Чәчәк ата төнен,
Зәңгәр йолдызларга үрелеп.
Бал кортлары күрми,
Күбәләкләр белми
Сөю көткән чагын бирелеп.
Зәңгәр йолдызларга үрелеп.

Чәнечкеле гөлем
Чәчәк атты төнен,
Тик бер тәүлек кенә күренеп.
Чәчәкләрең сулды,
Күз яшьләргә тулдың,
Гашыйк чакларыңа бирелеп.
```

# Перевод

### Асхар Кашфуллин / Фирзар Мортазин - Мой цветок в колокольчиках

```
Мой цветок в колокольчиках
Цветет ночью,
Лишь сутки его жизнь.
Глаза к небу подняв,
Напевая радостный мотив,
Угасает цветок, угасает безустанно.

Мой цветок в колокольчиках
Цветет ночью,
К синим звездам устремляясь.
Пчелы не увидят,
Бабочки не узнают,
Как он отдался в час ожидания любви.
К синим звездам устремляясь.

Мой цветок в колокольчиках
Отцвел ночью,
Лишь сутки явив себя.
Твои цветы увяли,
Ты наполнился слезами,
Отдавшись влюбленным мгновениям.
```
