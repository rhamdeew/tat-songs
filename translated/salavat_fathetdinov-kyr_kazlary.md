# Оригинал

### Салават Фатхетдинов - Кыр казлары

```
Кыр казлары китте, санамадым,
Жилпер-жилпер канат каганып.
Монсуланып карап озатып калдым,
Кабат кайтуларын сагынып.

Кыр казлары китте, санамадым,
Белеп торам барыда кайтмасын.
Алып китте кочле жил-оермэ,
Ераклашкан казлар тавышын.

Кыр казлары китте, санамадым,
Яз кайтырлар эле санармын.
Йорэгемдэ калган монсу сагышлары,
Янарылар кабат язларын.
```

# Перевод

### Салават Фатхетдинов - Полевые гуси

```
Полевые гуси улетели, не считал я,
Трепеща крылами в небесах.
Грустно провожая их взглядом,
Жду возвращения, скучая.

Полевые гуси улетели, не считал я,
Знаю, что не все вернутся вновь.
Унёс могучий ветер-ураган
Далёких гусей голоса.

Полевые гуси улетели, не считал я,
Весной вернутся, я надеюсь.
В сердце остались печальные думы,
Оживут они снова весною.
```
