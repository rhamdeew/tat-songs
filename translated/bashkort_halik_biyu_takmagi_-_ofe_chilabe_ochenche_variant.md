# Оригинал

### Башкорт халык бию такмагы - Өфе - Чиләбе (өченче вариант)

```
Ардагы дуслык юлын,
Без корычтан салабыз.

Өфе, Чиләбе,
Пешә кура җиләге.

Дәртле җырлар җырлый-җырлый
Без шул юлдан барабыз.

Өфе, Чиләбе,
Пешә кура җиләге.

Бергәләшеп эшләгәндә,
Җиңел юллар табыла.

Өфе, Чиләбе,
Пешә кура җиләге.

Тормыш юлын үзгәртәек,
Яңыртаек тагын да.

Өфе, Чиләбе,
Пешә кура җиләге.
```

# Перевод

### Народный танец башкир - Уфа - Челабинск (третий вариант)

```
Дорогу дружбы позади,
Мы сталью сложим.

Уфа, Челябинск,
Ветерок пшеничный.

Песни грустные поём-поём,
Мы по той дороге идём.

Уфа, Челябинск,
Ветерок пшеничный.

Вместе работая,
Лёгкие тропинки находим.

Уфа, Челябинск,
Ветерок пшеничный.

Путь жизни изменяем,
Ещё больше обновляем.

Уфа, Челябинск,
Ветерок пшеничный.
```
