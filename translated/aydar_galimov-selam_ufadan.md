# Оригинал

### Айдар Галимов - Сэлам Уфадан

```
Без Агыйделдэн жыр алып килдек,
Урал таулары сэлам юллары.
Ничэ кат инде жир шарын урый
Татар жырынын данлы юллары.

Агыйдел яры
Монга уралган,
Сэлам Уфадан,
Сэлам Уралдан.
Йоретэ безне
Жыр канатлары.
Монлы голбишек –
Уфа яклары.

Кая барсак та, кайда йорсэк тэ,
Сез, туганнарым, каршы аласыз.
Йорэк тоташа монлы жыр аша,
Без туган жирнен газиз баласы.

Агыйдел яры
Монга уралган,
Сэлам Уфадан,
Сэлам Уралдан.
Йоретэ безне
Жыр канатлары.
Монлы голбишек –
Уфа яклары.

Сагынган чакта, жирсегэн чакта
Безнен моннардан шифа алыгыз.
Без бит жир шарын жырга чорныйбыз,
Хэерле сэфэр телэп калыгыз.

Агыйдел яры
Монга уралган,
Сэлам Уфадан,
Сэлам Уралдан.
Йоретэ безне
Жыр канатлары.
Монлы голбишек –
Уфа яклары.
```

# Перевод

### Айдар Галимов - Привет из Уфы

```
Мы от Агидели песню принесли,
Уральские горы — приветные дороги.
Сколько раз уже землю обходят
Татарской песни славные пути.

Агидели берег
Мелодией окутан,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодичный соловей —
Уфимские края.

Куда ни пойдём, где ни пройдём,
Вы, родные мои, встречаете.
Сердце соединяется через мелодичную песню,
Мы родной земли дорогое дитя.

Агидели берег
Мелодией окутан,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодичный соловей —
Уфимские края.

Когда тоскуете, когда грустите,
От наших мелодий исцеление берите.
Мы ведь землю песнями обходим,
Доброго пути желая, оставайтесь.

Агидели берег
Мелодией окутан,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодичный соловей —
Уфимские края.
```
