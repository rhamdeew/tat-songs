# Оригинал

### Әхмәт Ерикәй / Әгъзам Фәттах - Кил, иркәм, гармунда уйнарга

```
Кичләрен күңелле Иделдә,
Мин телим көймәдә йөрергә;
Мин телим яңа җыр җырларга,
Кил, иркәм, гармунда уйнарга.

Мин синен чәчеңне сыйпармын,
Иң матур көеңне отармын.
Таралсын көйләрең кырларга,
Кил, иркәм, гармунда уйнарга.

Түшеңә чәчәкләр кадармын,
Күзләрең күленә карармын.
Яратам моңлы көй тыңларга,
Кил, иркәм, гармунда уйнарга.

Күз нурым, акыллым, чибәрем,
Мин сине кадерләп үбәрмен.
Күңелле булсын бар дусларга,
Кил, иркәм, гармунда уйнарга.
```

# Перевод

### Ахмат Ерикей / Агзам Фаттах - Приди, любимая, сыграй на гармони

```
Вечера веселые на Волге,
Хочу я в лодке плыть по глади;
Хочу я новые песни петь,
Приди, любимая, сыграй на гармони.

Я твои волосы поглажу нежно,
Самую красивую мелодию сыграю.
Пусть разольются твои напевы по полям,
Приди, любимая, сыграй на гармони.

На грудь твою цветы рассыплю,
В озера глаз твоих взгляну.
Люблю я грустные напевы слушать,
Приди, любимая, сыграй на гармони.

Свет очей моих, разумная, прекрасная,
Я тебя с любовью поцелую.
Пусть будет весело всем нашим друзьям,
Приди, любимая, сыграй на гармони.
```
