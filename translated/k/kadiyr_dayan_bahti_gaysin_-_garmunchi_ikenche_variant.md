# Оригинал

### Кадыйр Даян / Бәхти Гайсин - Гармунчы (икенче вариант)

```
Эх, гармуның, гармуныңда
Гармуныңның тавышы!
Уйна дәртле, шат-көйләрне,
Бетсең йөрәк сагышы.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.

Эх, гармуның, гармуныңда
Гармуныңның телләре!
Син уйнасаң, эшчәннәрнең
Рәхәт үтә көннәре.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.

Гармуныңның телләренә
Дәрт ялкыны кабынсын.
Син уйнаган шат-көйләрне
Яшь йөрәкләр сагынсын.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.
```

# Перевод

### Кадыйр Даян / Бәхти Гайсин - Гармонист (второй вариант)

```
Ах, в гармони, в гармони
Звук её, гармони!
Играй тревожные, светлые напевы,
Утоляй сердечную тоску.

Ах, звук гармони,
Утоляй сердечную тоску.

Ах, в гармони, в гармони
Языки её!
Если ты играешь, трудящихся
Дни проходят спокойно.

Ах, звук гармони,
Утоляй сердечную тоску.

На язычки гармони
Пусть вспыхнет пламя страсти.
Напевы, что ты сыграл,
Пусть томятся молодые сердца.

Ах, звук гармони,
Утоляй сердечную тоску.
```
