# Оригинал

### Кәрим Тинчурин / Салих Сәйдәшев - Сандугач (беренче вариант)

```
Җир җиләге ачы була, ачы була
Үз вакытында, үз вакытында пешмәсә, шул.
Җәйге җилләр яшь йөрәккә шифа була,
Сөйгән ярың каш уйнатып эндәшсә, шул.
Сандугач, кошчыгым, күңелемне ач син бүген.

Киң яланның хәбәрче ак күбәләге
Сиңа бәгырем, сәламемне ирештерсен.
Ак каенның җиләс, салкын күләгәсе
Бүген безне күрештерсен, сөендерсен.
Сандугач, кошчыгым, күңелемне ач син бүген.

Кузгалаклар, аксыргаклар айкый-айкый,
Тырыс тутырып җиләк җыйдым ярлар өчен.
Җиткәннәрен, пешкәннәрен сайлый-сайлый,
Тырыс тутырып булыр микән ярлар өчен.
Сандугач, кошчыгым, күңелемне ач син бүген.
```

# Перевод

### Карим Тинчурин / Салих Сайдашев - Соловей (первый вариант)

```
Земляничная ягода бывает горькой, горькой,
Если не созреет в своё время, в своё время.
Летние ветры целебны для молодого сердца,
Если любимая друга, играя бровями, подмигнёт - то.
Соловей, моя птичка, открой мою душу сегодня.

Белый вестник широкого поля,
Пусть донесёт тебе, мой соловей, мой привет.
Ягодка белой берёзы, её прохладная тень,
Пусть在今天 встретит нас, порадует.
Соловей, моя птичка, открой мою душу сегодня.

Кузгалаки, белые стебельки колышутся-колышутся,
Я старательно набирала землянику для друзей.
Выбирая-выбирая спелые, созревшие,
Старательно набирала - хватит ли для друзей.
Соловей, моя птичка, открой мою душу сегодня.
```
