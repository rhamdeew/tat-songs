# Оригинал

### Кәүсәрия Шәфикова / Гөлнара Зиязова - Ак каеннар (беренче вариант)

```
Җәй буена каеннарга
Серләремне сөйләдем.
Талгын җилгә кушылдым да,
Сагынуымны сөйләдем.

Каеннарны сагышсыз, дип
Уйлап ялгышканмын ла.
Язмышлары тиң икән бит
Кеше язмышларына.

Ак каеннар тын тордылар,
Сабыр иткәннәр икән.
Үз хәлләрен сөйләр өчен
Көзне көткәннәр икән.

Каеннарны сагышсыз, дип
Уйлап ялгышканмын ла.
Язмышлары тиң икән бит
Кеше язмышларына.
```

# Перевод

### Каўсария Шафикова / Гульнара Зиязова - Белые березы (первый вариант)

```
Все лето березам
Тайны свои открывала.
С ласковым ветром сливалась
И о тоске рассказала.

Я думала: березы без печали,
Как ошибалась я в том.
Их судьбы, видно, равны
С людской судьбой во всем.

Белые березы молчали,
Терпели, видно, давно.
Чтобы поведать о доле своей,
Они ждали осень давно.

Я думала: березы без печали,
Как ошибалась я в том.
Их судьбы, видно, равны
С людской судьбой во всем.
```
