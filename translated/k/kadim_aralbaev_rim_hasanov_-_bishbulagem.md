# Оригинал

### Кәдим Аралбаев / Рим Хәсәнов - Бишбүләгем

```
Бишбүләгем - моң төягем,
Бишегем, газиз җирем,
Синдә минем тамырларым,
Кырыңа тамган тирем.

Бишбүләгем - хис төягем,
Мәхәббәт ҡаеннарым.
Чишмәләрең суын эчтем,
Сөюдән айнымадым.

Бишбүләгем - Дим буйлары,
Моңло сандугач иле.
Тальяныңның көмеш чыңы
Мәңге тынмасын иде.

Бишбүләгем - бердән-берем,
Син дип тибә йөрәгем.
Биш яшәүдәй бер гомерем
Булсын сиңа бүләгем ...
```

# Перевод

### Кәдим Аралбаев / Рим Хасанов - Бишбулягем

```
Бишбулягем - мой груз напевный,
Колыбель моя, земля родная,
В тебе мои живые корни,
Мой пот, что пал на твоё поле.

Бишбулягем - мой груз чувств нежных,
Мои берёзы любви священной.
Я пил из ваших родников,
От этой любви не очнулся вновь.

Бишбулягем - берега Дёма,
Край соловья с печальной песней.
Пусть серебристый звон тальянки
Не умолкает вечно здесь.

Бишбулягем - ты единственный,
Ты - сердцу вечный мой припев.
Пять жизней - вот одна судьба мне,
Пусть станет для тебя мой дар ...
```
