# Оригинал

### Касыйм Тәхау / Рөстәм Яхин - Шушма суы

```
Шушма суы-бормалы су,
Ярларында - тал-тирәк.
Бер кыз җырлый әрәмәдә -
Ничек түзсен яшь йөрәк.

Былбыл сайрый, тал тирбәлә
Шушма суы буенда.
Алсу йөзем, зәңгәр күзем,
Тик син генә уемда.

Шушма суы дулкынлана
Таң җилләре искәндә.
Бәгыремне өзә җанкай
Судан кайтып килгәндә.

Шушма суы тыныч була
Кояш чыкканда, иртән.
Шушма буенда үскәнгә
Сылу булгансың, иркәм!
```

# Перевод

### Касыйм Тахау / Рөстәм Яхин - Шушма суы

```
Шушма - вьётся, излучиной льётся,
По берегам - тальник, тополя.
В роще девушка песней поётся -
Как стерпеть молодому сердцу - нельзя.

Соловей разливается, ивы качает
У реки Шушмы волна.
Румяна щека, синева глаз сверкает -
Только ты у меня в мыслях одна.

Шушма воды волнует
Ветерок рассветных снов.
Сердце рвёт, любимая, жгут меня,
Когда ты возвращаешься из вод.

Шушма вода бывает тиха
На заре, когда солнце встаёт.
Ты у Шушмы росла - потому и красива,
Моя нежная, мой свет, мой полёт!
```
