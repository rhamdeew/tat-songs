# Оригинал

### Клара Булатова / Мәсгуть Имашев - Сәлим җыры

```
Таң җырым, уят син,
Иркәләп юат син
Мин эзләп тилмергән
Сагышлы ярымны.
Тирбәлеп таңнарда,
Җиткер син аңарга
Күңелдә төйнәлгән
Бөтен аһ-зарымны.
Таң җырым, уят син,
Иркәләп юат син
Мин эзләп тилмергән
Сагышлы ярымны.

Белсен ул мин барын,
Эзләсен үз ярын...
Сакласын, әйт аңа,
Тик күңел сафлыгын!
Ышансын, белсен ул
Мин килер көн барын,
Чыксын ул каршыма,
Чыксын ул атлыгып.
Белсен ул мин барын,
Эзләсен үз ярын...
Сакласын, әйт аңа,
Тик күңел сафлыгын!
```

# Перевод

### Клара Булатова / Мәсгуть Имашев - Песня Салима

```
Песнь зари, разбуди,
Приласкай, утоли
Того, кого ищу,
Мою тоску-любовь.
Качаясь на заре,
Ты донеси ему
Все вздохи, что в душе
Свернулись в горький зов.
Песнь зари, разбуди,
Приласкай, утоли
Того, кого ищу,
Мою тоску-любовь.

Пусть знает - я жива,
Пусть ищет свой приют...
Скажи ему: храни
Лишь чистоту души!
Пусть верит, знает он -
Настанет день, приду,
Пусть выйдет мне навстречу,
Пусть выйдет поспешив.
Пусть знает - я жива,
Пусть ищет свой приют...
Скажи ему: храни
Лишь чистоту души!
```
