# Оригинал

### Камил Кәримов / Зиннур Гыйбадуллин - Шәле тәрәзәләре

```
Кояш нурларыннан эрләп
Шәл бәйли киленнәре.
Ил җылысын тояр идең -
Шәлегә бер кил әле.

Олы юлдан каршы ала
Пардан киенгән нарат.
Тәрәзәләр сине көтә,
Шул наратларга карап.

Тәрәзәләр - яз өлгесе,
Таллары - яшел бәйләм.
Чәчәкләр иркен җәйләүгә
Чигүле яулык җәйгән.

Олы юлдан каршы ала
Пардан киенгән нарат.
Тәрәзәләр сине көтә,
Шул наратларга карап.

Җирдә-күктә шат авазлар,
Кызлар җыры кырларда.
Йөрәкләр ашкына кырга
Шул җырларны тыңларга.

Олы юлдан каршы ала
Пардан киенгән нарат.
Тәрәзәләр сине көтә,
Шул наратларга карап.
```

# Перевод

### Камил Кәримов / Зиннур Гыйбадуллин - Окна с шалью

```
Из солнечных лучей прядя,
Невестки вяжут шаль.
Ты ощутишь тепло страны -
К ее шали заглянь.

С большого тракта встречу даст
Сосна в паре, при наряд.
Окна ждут тебя, глядят
На те сосны подряд.

Окна - образец весны,
Ивы - зеленый сверток.
Цветы на вольный летний стан
Расстлали вышитый платок.

С большого тракта встречу даст
Сосна в паре, при наряд.
Окна ждут тебя, глядят
На те сосны подряд.

В небе и на земле - напев,
Песни девушек в полях.
Сердца спешат на луг послушать
Эти песни на полях.

С большого тракта встречу даст
Сосна в паре, при наряд.
Окна ждут тебя, глядят
На те сосны подряд.
```
