# Оригинал

### Камил Гәрәев / Фәрит Хатыйпов - Янаул

```
Мәңге яши күңелләрдә
Гөл бакчалы Янаул.
Янаулга бер килгәннәр,
Мәңгелеккә кала ул.
Янаул, Янаул!
Мәңгелеккә капа ул.

Яңалыкны һәр вакыт та
Алга куя Янаул.
Шуңа да бит чәчкә ата,
Шуңа күрә яңа ул.
Янаул, Янаул!
Тормышында - яңа юл.

Яңаулның киләчәге
Яктылыкка яңа юл.
Янаул, - дип әйтсәләр дә
Йөрәк ярсып яна ул.
Янаул, Янаул!
Яктылыкка - яңа юл.
```

# Перевод

### Камил Гараев / Фарит Хатыпов - Янаул

```
В сердцах живет навеки
Янаул - садов цветник.
Кто в Янаул однажды
Пришел - навек там сник.
Янаул, Янаул!
Навек к нему приник.

Всегда вперёд новинку
Выдвигает Янаул.
Потому он расцветает,
Потому он вечно юн.
Янаул, Янаул!
В его судьбе - новый путь.

Будущее Янаула -
К свету новый путь ведет.
Даже имя прозвучало -
Сердце вспыхнет, запоет.
Янаул, Янаул!
К свету - новый путь.
```
