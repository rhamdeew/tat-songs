# Оригинал

### Кави Нәҗми / Заһид Хәбибуллин - Җиңеп кайт!

```
Чулпан калка, таң ата,
Кыз сөйгәнен озата,
Исән барып, исән кайт,
Дошманнарны җиңеп кайт!

Миннән сиңа теләк шул -
Ил мактарлык үрнәк бул.
Мин тагын телим сиңа -
Үткенлек кылычыңа.

Боргы уйный еракта,
Туплар сөйләшкән якта.
Егет китте шул якка,
Атланып аргамакка.

Теләр идем тагын да,
Бик сагынган чагында, -
Язып җибәр сәлам-хат!
Сау бул, җиңеп кайт!
```

# Перевод

### Кави Нәҗми / Заһид Хәбибуллин - Вернись с победой!

```
Чолпан встает, заря встает,
Девушка милого провожает.
Живым уйди, живым вернись,
Врагов одолев - вернись!

От меня тебе пожеланье -
Будь примером для похвалы народа.
И еще желаю тебе -
Остроты твоему клинку.

Горн звучит вдали,
Там, где пушки говорят.
Парень ушел в ту сторону,
На аргамаке верхом.

И еще бы я хотела,
В час, когда особенно тоскливо, -
Напиши, пришли привет-письмо!
Прощай, вернись с победой!
```
