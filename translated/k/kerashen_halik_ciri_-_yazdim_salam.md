# Оригинал

### Керәшен халык җыры - Яздым сәлам

```
Ишек алкайлары алма агач,
Кисеп алдым алма булмагач.
Алдым да гына каләм, яздым сәлам,
Йөрәккәемә чыдый алмагач.

Ишек алкайлары яшел чирәм,
Иртә генә торып ат җибәр.
Ихласларым белән хат язамын,
Каршысына, җаный, хат җибәр.

Ике дә генәй аккош әй кычкыра,
Агыйделкәйләрнең буенда.
Хатларыма каршы хат салмыйсың,
Әллә юкмы, җаныем, уеңда?
```

# Перевод

### Керәшен халык җыры - Написал привет

```
У порога ветви яблони растут,
Срезал их, когда плодов не ждут.
Взял перо и написал привет,
Сердце не смогло стерпеть разлук.

У порога стелется трава,
Рано встань - коня скорей седлай.
С искренностью я письмо пишу,
В ответ мне, милый, письмецо пришли.

Два белых лебедя кричат вдали
У берегов родной Агидели.
На письма ты ответа не даешь -
Иль нет, любимый, обо мне в душе?
```
