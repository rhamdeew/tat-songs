# Оригинал

### Клара Булатова / Рәис Нәгыймов - Тамчы җыры

```
Тып-тып тама тамчы колак салчы,
Дөп-дөп тибә йөрәк, тыңлачы!
Тып-тып төшә - карны, бозны тишә,
Үзенекен итә бу тамчы.

Тып-тып төшә тамчы - бозны тишә,
Дөп-дөп тибә йөрәк, ни кирәк?
Бу тамчылар үзенекен итә,
Синең сүзне көтә бу йөрәк.

Күз алдыңа юри китер әле:
Шул тамчыдай, минем серләр, дип,
Юлсыз гына, тавыш-тынсыз гына
Йөрәгеңә синең үрләр, дип...

Тып-тып төшө тамчы - бозны тишә,
Дөп-дөп тибә йөрәк, ни кирәк?
Бу тамчылар үзенекен итә,
Синең сүзне көтә бу йөрәк.
```

# Перевод

### Клара Булатова / Раис Нагимов - Песенка о капельке

```
Кап-кап каплет дождик - прислушайся,
Тук-тук стучит сердце, послушай!
Кап-кап падает - пробивает снег и лёд,
Прорубает себе путь эта капля.

Кап-кап падает капля - пробивает лёд,
Тук-тук стучит сердце, зачем это нужно?
Эти капли прокладывают себе дорогу,
Это сердце ждёт твоего слова.

Представь сейчас перед собой:
Как эта капля, мои тайны, говорят,
Без дороги, беззвучно лишь
Токут к твоему сердцу...

Кап-кап падает капля - пробивает лёд,
Тук-тук стучит сердце, зачем это нужно?
Эти капли прокладывают себе дорогу,
Это сердце ждёт твоего слова.
```
