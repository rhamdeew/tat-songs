# Оригинал

### Кадыйр Даян / Мәсгуть Бәширов - Сабан тургае

```
Сайра, сайра, канат кагып,
Күккә үрлә син, тургай.
Синең көмеш тавышыңнан
Чыңлап торсын киң тургай.

Күккә менеп, җиргә карап,
Мактап җырла кырларны,
Мактап сайра аланнарны,
Безнең эшчән кызларны.

Язгы тугай бигерәк матур,
Сокланып сайрый тургай.
Сайра, сайра, җырчы тургай,
Чыңласын безнең тугай.
```

# Перевод

### Кадыйр Даян / Мәсгуть Бәширов - Жаворонок сабана

```
Пой, пой, взмахивая крыльями,
К небу взмывай, жаворонок.
Пусть от твоего серебряного голоса
Звенят просторные луга.

Поднимись к небу, глянь на землю,
Похвально спой о полях,
Воспой и запой о полянах,
О наших трудолюбивых девушках.

Весенний луг особенно прекрасен,
С восторгом поет жаворонок.
Пой, пой, певчий жаворонок,
Пусть звенит наш луг.
```
