# Оригинал

### Клара Булатова / Роберт Андреев - Язлар кайтты

```
Синең белән язлар кайтты миңа,
Җырлар килде язлар артыннан.

Гел башкача ачылып китте дөнья,
Гел башкача көнем балкыды.

Киң кырларга чыксам ялгыз йөрмим,
Кыр юлларым җырга тиң минем.

Юлларымда юлдаш җыр һәм кояш,
Уйларымда юлдаш син минем.
```

# Перевод

### Клара Булатова / Роберт Андреев - Вернулись весны

```
С тобой ко мне вернулись весны,
И песни вслед за ними пришли.

Иначе распахнулся мир огромный,
Иначе засияли мои дни.

Выйду в широкие поля - не один,
Мои полевые тропы песне сродни.

В дорогах моих спутник - песня и солнце,
В мыслях моих спутница ты - одна.
```
