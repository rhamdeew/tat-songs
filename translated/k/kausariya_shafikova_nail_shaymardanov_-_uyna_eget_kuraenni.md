# Оригинал

### Кәүсәрия Шәфикова / Наил Шәймарданов - Уйна, егет, кураеңны

```
Күкрәген киң - әйдә уйна,
Курай моңы яңграсын.
Кышның сары бураннары,
Көткәндә дә, күңелләр дә,
Тал тирәкләр яфрак ярсын.

Уйна, уйна, уйна кураеңны
Уйна, уйна, тынын җитәрлек.
Булсын җырлап җибәрерлек,
Булсын биеп китәрлек.

Еракларга даны киткән,
Курай моңы белән яшә.
Башкортстанның егете син,
Чал каялар бөркете син,
Алда сине бәхет көтә.

Уйна, уйна, уйна кураеңы
Уйна, уйна, тынын җитәрлек.
Булсын җырлап җибәрерлек,
Булсын биеп китәрлек.

Башкортстанның куенында,
Чәчәк ата, дуслык иле.
Шушы дулук җипләренә,
Кагылмасын, сугылмасын,
Дошманлыкның салкын җиле.

Уйна, уйна, уйна кураеңны
Уйна, уйна, тынын җитәрлек.
Булсын җырлап җибәрерлек,
Булсын биеп китәрлек.
```

# Перевод

### Кәусария Шафикова / Наил Шәймарданов - Играй, джигит, на курае

```
С широкой грудью - давай играй,
Пусть курая мелодия звенит.
Зимние жёлтые бураны,
Когда ждёшь, и сердца тоже,
Пусть ивы листвой зазеленеют.

Играй, играй, играй на курае,
Играй, играй, пока хватит дыханья.
Пусть будет так, чтоб петь хотелось,
Пусть будет так, чтоб в пляс пуститься.

Далеко слава разнеслась
О курая мелодии - живи с ней.
Ты джигит Башкортостана,
Ты орёл седых утёсов,
Впереди тебя счастье ждёт.

Играй, играй, играй на курае,
Играй, играй, пока хватит дыханья.
Пусть будет так, чтоб петь хотелось,
Пусть будет так, чтоб в пляс пуститься.

В объятьях Башкортостана
Цветёт дружбы благодатный край.
На эти курая струны
Пусть не коснётся, не подует
Враждебности холодный ветер.

Играй, играй, играй на курае,
Играй, играй, пока хватит дыханья.
Пусть будет так, чтоб петь хотелось,
Пусть будет так, чтоб в пляс пуститься.
```
