# Оригинал

### Кон-тон - Алдама мине алдама

```
Алдама мине, алдама,
Алдап куаныч тапма.
Йорэгемэ утлар яктын,
Башкаларны яратма.

Алдама мине, алдама,
Йорэгемдэ ут яна.
Мэхэббэт уенчык тугел,
Анын белэн шаярма.

Озелгэн жеп ялганса да,
Анын тоене кала.
Уеннан уймак чыгарсан,
Кузлэрдэн яшьлэр тама.

Алдама мине, алдама,
Йорэгемдэ ут яна.
Мэхэббэт уенчык тугел,
Анын белэн шаярма.

Алдама зинхар, алдама,
Гомерлеккэ яраттым.
Тормыш юлында югалсан,
Каерылыр бит канатын.

Алдама мине, алдама,
Йорэгемдэ ут яна.
Мэхэббэт уенчык тугел,
Анын белэн шаярма.
```

# Перевод

### Кон-тон - Не обманывай меня, не обманывай

```
Не обманывай меня, не обманывай,
Не ищи радости в обмане.
Огни зажег ты в сердце моем,
Не люби других.

Не обманывай меня, не обманывай,
В сердце моем пламя горит.
Любовь - не игрушка,
Не играй с ней.

Пусть и оборвалась нить случайно,
Узел ее остался.
Коль из игры печаль родишь,
Слезы польются из глаз.

Не обманывай меня, не обманывай,
В сердце моем пламя горит.
Любовь - не игрушка,
Не играй с ней.

Не обманывай, молю, не обманывай,
На всю жизнь полюбила.
Если потеряешься на жизненном пути,
Крылья сломаются ведь.

Не обманывай меня, не обманывай,
В сердце моем пламя горит.
Любовь - не игрушка,
Не играй с ней.
```
