# Оригинал

### Клара Булатова / Рәис Нәгыймов - Шушы яктан, шушы туфрактан без

```
Шушы яктан, шушы туфрактан без,
Китә калсак үзгә бер якка,
Таллар җырлап, чишмәләре чыңлап,
Өянкеләр елап озата.

Шушы яктан, шушы туфрактан без,
Читтә йөрсәк - шуңа күрәдер,
Чишмә чыңлап, өянкеләр елап,
Таллар җырлап төшкә керәдер.

Шушы яктан, шушы туфрактан без,
Шушы җиргә дәшеп кайтырга,
Таллар җырлап, чишмәләре чыңлап,
Өянкеләр зурлап чакыра.
```

# Перевод

### Клара Булатова / Рәис Нәгыймов - Мы из этой стороны, из этой земли

```
Мы из этой стороны, из этой земли,
Если вдруг уйдём в другую даль,
Ивы запоют, и звонкие ключи,
Освещая путь, оплачет тополь.

Мы из этой стороны, из этой земли,
Если мы вдали - оттого, наверно,
Звон ключей и плач тополей
В снах приходит - ивы в песне верной.

Мы из этой стороны, из этой земли,
Эта земля нас к себе зовёт,
Ивы запоют, ключи зазвенят,
Тополя торжественно позовут.
```
