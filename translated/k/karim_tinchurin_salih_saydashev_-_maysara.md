# Оригинал

### Кәрим Тинчурин / Салих Сәйдәшев - Мәйсәрә

```
Чыбылдыгың чуп-чуар,
Эчләрендә кем кунар?
Сызылып моңсу, якты таңнар аткан чакта
Мамык мендәр тешләп кем елар?

Үкси-үкси, Мәйсәрә җан,
Яшь урынына түкмә кан,
Яшь урынына түкмә кан.

Чыбылдыгың бизәген
Ука белән бизәдең,
Бәхилләшеп аерылганда, елый-елый,
Ал яулыгың болгап изәдең,

Үкси-үкси, Мәйсәрә җан,
Яшь урынына түкмә кан,
Яшь урынына түкмә кан.

Чыбылдыгың ак чугын
Бәйрәмнәрдә тагарсың.
Сөйгән ярың, Булат бәгырең исеңә төшсә
Төнлә ялгыз айга карарсың.

Үкси-үкси, Мәйсәрә җан,
Яшь урынына түкмә кан,
Яшь урынына түкмә кан.
```

# Перевод

### Карим Тинчурин / Салих Сәйдәшев - Мэйсәрә

```
Тысячелеге шат, безгәк,
Кем ялар энә кебек?
Пыт-пыт, күк чыбык, якты иртәннәр утырган вакытта
Тышның ялыкка барып кем ялар?

Окси-окси, Мэйсәрә җан,
Яңа урынына түкмә сөн,
Яңа урынына түкмә сөн.

Тысячелеге бизәк
Иңгә белән бизәлгән,
Бүленеп, гизләп аерылганда, ялар-ялар,
Ал күзегез чыбыкка әйләнгән,

Окси-окси, Мэйсәрә җан,
Яңа урынына түкмә сөн,
Яңа урынына түкмә сөн.

Тысячелеге ак чыгын
Бәйрәмнәрдә ташарсың.
Сөйгән ярың, Булат бәгырең исегә төшсә
Төнлә ялгыз айга җырларсың.

Окси-окси, Мэйсәрә җан,
Яңа урынына түкмә сөн,
Яңа урынына түкмә сөн.
```
