# Оригинал

### Кәүсәрия Нуруллина / Шамил Шәрипов - Ап-ак карлар (беренче вариант)

```
Ап-ак карлар ява,
Ап-ак карлар ява,
Юлларыма салып ак келәм.
Ап-ак чәчәкләрдәй (ап-ак чәчәкләр кебек)
Яуган ап-ак карлар
Безгә бәхет килтерсин диям.

Ап-ак карлар ява,
Ап-ак карлар ява,
Каз мамыгы гүя тарала.
Синсез көннәремнән,
Кайгы-сагышымнан,
Ялгыз көннәремнән арала.

Ап-ак карлар ява,
Әнҗе карлар ява,
Күбәләктәй яусын ак карлар.
Килер юлларыма,
Минем яннарыма
Тик чыкмасын салкын бураннар.
```

# Перевод

### Кәүсәрия Нуруллина / Шамиль Шәрипов - Белый снег (первый вариант)

```
Белый снег яуга,
Белый снег яуга,
Белый чыршыларча юлларга төшәм.
Белый чәчәкләрчә (белый чәчәкләр кебек)
Яуган белый снег
Бизгә бәхет китерсин диям.

Белый снег яуга,
Белый снег яуга,
Каз мамыгы гына тарала.
Синсез көннәрдән,
Кайгы-сагыштан,
Ялгыз көннәрдән арала.

Белый снег яуга,
Әнҗе снег яуга,
Күбәләкчә яуга белый снег.
Килер юлларга,
Минем яныма
Тик чыкмасын салкын бураннар.
```
