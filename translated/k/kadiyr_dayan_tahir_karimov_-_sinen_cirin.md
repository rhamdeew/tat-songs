# Оригинал

### Кадыр Даян / Таһир Кәримов - Синең җырың

```
Син җырладың кич һәм мин тыңладым,
Күңелемдә хәзер син генә.
Мин онытмам синең җырларыңны
Күңелемдә чыңлар гел генә.

Синең җырың минем йөрәгемә
Язгы чәчәк булып үрелде.
Матур дөнья миңа шул вакытта,
Тагы да матур булып күрендең.
```

# Перевод

### Кадыр Даян / Таһир Кәримов - Твоя песня

```
Ты пела вечером - я слушал молча,
В душе моей теперь одна лишь ты.
Я не забуду песни твои - их звонко
В моей душе звучат, как сны.

Твоя песня в сердце мне легла
Весенним цветом, тихо расцвела.
И мир тогда казался мне красивым,
А ты - еще прекрасней, чем была.
```
