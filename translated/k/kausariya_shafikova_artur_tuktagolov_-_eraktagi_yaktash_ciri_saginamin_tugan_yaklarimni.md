# Оригинал

### Кәүсәрия Шәфикова / Артур Туктаголов - Ерактагы якташ җыры (Сагынамын туган якларымны)

```
Сандугачлар моңлы сайрадымы?
Әллә курай моңы агылды?
Күңелемнең нечкә кылларына
Үзәк өзгеч сагыш сарылды.

Төшләремдә күреп уянамын
Акчарлаклы иделкәемне.
Сагынамын туган туфрагымны,
Сагынамын газиз илкәйне.
Төшләремдә күреп уянамын
Акчарлаклы иделкәемне.
Сагынамын Татарстаным сине,
Сагынамын газиз илкәйне.

Чит җирләрдә булган эчәр суын
Ниләр күреп ерак киткәнмен?
Туган җирдән, яшәү бәхетеннән,
Үз үземне мәхрүм иткәнмен.

Төшләремдә күреп уянамын
Акчарлаклы иделкәемне.
Сагынамын туган якларымны,
Сагынамын газиз илкәйне.

Үкенүләр авыр газап икән,
Туган туфрак гафү итәрме?
Кайта алсам, җилләр назлап кына,
Иңнәремнән сөеп хисерме?

Төшләремдә күреп уянамын
Акчарлаклы иделкәемне.
Сагынамын туган туфрагымны,
Сагынамын газиз илкәйне.
Төшләремдә күреп уянамын
Акчарлаклы иделкәемне.
Сагынамын Башкортстанымны мин,
Сагынамын газиз илкәйне.
```

# Перевод

### Кәүсәрия Шәфикова / Артур Туктаголов - Песня далекого земляка (Скучаю по родным краям)

```
То соловьи запели грустно?
Иль курая разлился стон?
По тонким струнам сердца нежным
Разлив печаль разорвал сон.

Во сне увидев - просыпаюсь,
Мой Идел с чайками вдали.
Скучаю по земле родимой,
Скучаю по святой земле.
Во сне увидев - просыпаюсь,
Мой Идел с чайками вдали.
Скучаю по тебе, Татарстан,
Скучаю по святой земле.

На чужбине воду пил я,
Что видел, что увлекло меня?
От родины, от счастья жизни
Себя отрёк и лишил себя.

Во сне увидев - просыпаюсь,
Мой Идел с чайками вдали.
Скучаю по родным просторам,
Скучаю по святой земле.

Горька расплата, боль разлуки -
Простит ли почва родная мне?
Коль возвращусь, лаская ветром,
Плечи поцелует он во сне?

Во сне увидев - просыпаюсь,
Мой Идел с чайками вдали.
Скучаю по земле родимой,
Скучаю по святой земле.
Во сне увидев - просыпаюсь,
Мой Идел с чайками вдали.
Скучаю по Башкортостану,
Скучаю по святой земле.
```
