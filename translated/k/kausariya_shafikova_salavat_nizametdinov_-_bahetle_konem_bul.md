# Оригинал

### Кәүсәрия Шәфикова / Салават Низаметдинов - Бәхетле көнем бул

```
Кош булып барырмын сиңа,
Җаныңда терәк булса.
Өзелексез сайрап торырмын -
Җаныңа кирәк булсам.

Яңгырлар булып явырмын,
Яшнәрмен яшен булып.
Түгелермен күзләремнән
Бәхетле (шатлыклы) яшең булып.

Калырмын синең гомерлек
Алдыңда гөлең булып.
Син бер генә кара минең
Бәхетле көнем булып.
```

# Перевод

### Кәусария Шафикова / Салават Низаметдинов - Будь моим счастливым днем

```
Птицей к тебе прилечу,
Если стану опорой душе.
Буду без устали петь -
Если нужен твоей душе.

Дождями прольюсь над тобой,
Вспыхну молнией яркой грозой.
С глаз моих я прольюсь
Твоей счастливой (радостной) слезой.

Навек у твоих я останусь
Цветком, что у ног распустился.
Ты лишь взгляни на меня -
Счастливым днем твоим быть мне приснилось.
```
