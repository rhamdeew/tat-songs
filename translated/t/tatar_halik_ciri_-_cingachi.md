# Оригинал

### Татар халык җыры - Җиңгәчи

```
Сикреп төштем тирәккә лә.
Гөл яфрагы кирәккә.

Ай җиңгәчи,
Сөйгәнемне димләче.

Гөл яфрагы дару диләр
Яна торган йөрәккә.

Ай җиңгәчи,
Сөйгәнемне димләче.

Сандугачлар сайраса да,
Йөрәккә ял буладыр;

Ай җиңгәчи,
Сөйгәнемне димләче.

Биш минутның өч минуты
Сезне уйлап узадыр.

Ай, җиңгәчи,
Сөйгәнемне димләче.
```

# Перевод

### Татарская народная песня - Цинга́чи

```
Прыгнула на тополь я,
Лепесток цветка нужен мне.

Ах, цингачи,
Не выдавай мою любимую.

Лепесток цветка - лекарство, говорят,
Для горящего сердца.

Ах, цингачи,
Не выдавай мою любимую.

Хоть и поют соловьи,
Сердцу покой настаёт;

Ах, цингачи,
Не выдавай мою любимую.

Из пяти минут три минуты
Проходят в мыслях о вас.

Ах, цингачи,
Не выдавай мою любимую.
```
