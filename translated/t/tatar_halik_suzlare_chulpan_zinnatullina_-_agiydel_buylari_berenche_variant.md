# Оригинал

### Татар халык сүзләре / Чулпан Зиннәтуллина - Агыйдел буйлары (беренче вариант)

```
Агыйделнең пароходы,
Кычкырта сузып-сузып.
Ургылып аккан сулардай,
Гомерләр бара узып.

Агыйделнең ар ягында,
Каенлыкта элмәлек.
Гомернең узасын белдем,
Тиз узасын белмәдем.

Агыйделнең ар ягында,
Казлар канат төзәтә.
Булма матур, бул бәхетле,
Бәхет матур күрсәтә.

Агыйделдә ялгыз үрдәк,
Югалткандыр баласын.
Баш имәгез кайгыларга,
Табыгыз бер чарасын.
```

# Перевод

### Чулпан Зиннәтуллина - Берега Агыйдела (первый вариант)

```
Пароход на Агыйделе
Долго тянет свой гудок.
Как воды, что мчатся бурно,
Ускользает жизни срок.

За Агыйделом, по ту сторону,
В березняке - виселица.
Знал: пройдет когда-то жизнь,
Не знал - как скоро минется.

За Агыйделом, по ту сторону,
Гуси крылья поправляют.
Не будь красивой - будь счастливой,
Счастье красоту являет.

На Агыйделе один утенок
Потерял, должно быть, дитя.
Не склоняйтесь перед горем,
Отыщите выход, путь найдя.
```
