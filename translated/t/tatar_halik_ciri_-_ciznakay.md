# Оригинал

### Татар халык җыры - Җизнәкәй

```
Кыз:
Тәрәзәдән нигә бактың, җизнәкәй?
Егет:
Яшь йөрәккә ут яктың, балдызым-ялгызым.

Кыз:
Бу бармагың ник кәкере, җизнәкәй?
Егет:
Алтын балдак кигәнгә, балдызым-ялгызым.

Кыз:
Бу йөзләрең ник саргайган, җизнәкәй?
Егет:
Сине өзелеп сөйгәнгә, балдызым-ялгызым.

Кыз:
Мич башына ник мендең җизнәкәй?
Егет:
Караватың дип белдем, балдызым-ялгызым.

Кыз:
Балдызыңны нигә үптең, җизнәкәй?
Егет:
Җәмәгатем дип белдем, балдызым-ялгызым,

Кыз:
Балдыз сөю язык, диләр, җизнәкәй,
Егет:
Язык казык башында, балдызым-ялгызым.
```

# Перевод

### Татарская народная песня - Зятюшка

```
Девица:
Зачем в окно заглядывал, зятюшка?
Парень:
Ты вспыхнула огнём в сердце, сестрёнка-одиночка.

Девица:
Отчего твой палец крив, зятюшка?
Парень:
Потому что кольцо ношу, сестрёнка-одиночка.

Девица:
Почему лицо побледнело, зятюшка?
Парень:
Оттого что люблю тебя без памяти, сестрёнка-одиночка.

Девица:
Зачем на печь взобрался, зятюшка?
Парень:
Принял за твою кровать, сестрёнка-одиночка.

Девица:
Почему целовал свою балдыз, зятюшка?
Парень:
Решил, что это жена, сестрёнка-одиночка.

Девица:
Любить балдыз - грех, говорят, зятюшка.
Парень:
Грех - только в словах да на колу, сестрёнка-одиночка.
```
