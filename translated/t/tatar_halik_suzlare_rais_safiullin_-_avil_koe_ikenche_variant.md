# Оригинал

### Татар халык сүзләре / Рәис Сафиуллин - Авыл көе (икенче вариант)

```
Биек тауның башларында,
Ах, мин каеннар булсам.
Ах, үзем каеннар булып,
Син былбыл булып кунсаң.

Аклы ситсы күлмәк алдым,
Бәйрәмдә киярмен дип.
Яшьләй күңелемне салдым,
Гомергә сөярмен дип.

Агыйдел сулары кебек,
Тар сулар булмас инде.
Минем йөрәккәем кебек
Ярсыган булмас инде.
```

# Перевод

### Татарские народные слова / Анис Сафиуллин - Деревенская мелодия (второй вариант)

```
На вершинах высоких гор,
Ах, стать бы мне берёзой белой.
Ах, сама бы берёзой стала,
А ты соловьём бы сел на ветви.

Платье из светлого ситца купила,
Чтобы в праздник его надеть.
С юных лет своё сердце отдала,
Чтобы любить на всю жизнь.

Как воды Агидели светлой,
Не будет теснин в пути.
Как моё сердце верное,
Не будет больше страдать.
```
