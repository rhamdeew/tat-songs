# Оригинал

### Татар халык җыры - Наласа авылы көе (беренче вариант)

```
Өздереп гармун уйныйлар
Наласа авылында.
Йөрәгемә утлар яктың -
Янам мин ялкынында.

Төнбоеклы тирән күлдә
Тезелеп йөзә аккошлар.
Үсеп җиткәч, чит җирләрдә
Йөртә безне язмышлар.

Бала чагым гөрләп үтте
Наласа кырларында.
Сагынуларым, сагышларым
Бу моңлы җырларымда.
```

# Перевод

### Татарская народная песня - Мелодия села Наласа (первый вариант)

```
Играют на гармони, наигрывая
В селе Наласа.
Огоньки зажигаешь в сердце моём -
В пламени я сгораю.

В ночном, глубоком озере
Встань белые лебеди плавать.
Вырастя, в чужих краях
Нас судьбы уносят.

Моё детство шумно прошло
На полях Наласы.
Тоски мои, печали мои
В этих мелодичных песнях моих.
```
