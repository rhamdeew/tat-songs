# Оригинал

### Татар халык җыры - Көмеш телле гармун сөйли

```
Гармун алдым мин кулыма,
Чыктым урман юлына.
Кызлар белән таныштыр, дип,
Куштым мин гармуныма

Яңгыратып бистәләрне,
Гармун уйный кичләрен.
Көмеш телле гармун сөйли
Яшь егетнең хисләрен.

Кызлар күңеле гармунчыда,
Гармунчыга яр булчы.
Гармунчы белән гөрләшеп,
Гомрең рәхәт узсачы.
```

# Перевод

### Татар халык җыры - Серебряноязыкая гармонь говорит

```
Гармонь в ладони взял я,
На лесную тропу шагнул.
«С девушками сведи меня», -
Прошептал я своей гармони.

Раззвоняя все селенья,
Вечерами гармонь играет.
Серебряноязыкая гармонь говорит
О чувствах юного парня.

Девичье сердце - у гармониста,
Стань ему милой парой.
С гармонистом в любви живя,
Пусть жизнь твоя пройдет привольно.
```
