# Оригинал

### Татар халык җыры - Өй артында шомыртым

```
Өй артында шомыртым бар,
Яннарында бөрлегән шул,
Яннарында бөрлегән;
Бөрлегәндәй матур булып
Чәчәк ата ил генәм

Өй артында шомыртым бар,
Чәчәкләр - яннарында шул,
Чәчәкләр - яннарында;
Без туганбыз илебезнең
Иң матур таңнарында.

Өй артында шомыртым бар,
Яннарында чияләр шул,
Яннарында чияләр;
Чияләр дә безгә карап:
«Сез бәхетле!» - дияләр.
```

# Перевод

### Татар халык җыры - Черёмуха за домом

```
За домом черёмуха стоит,
По бокам - земляника там,
По бокам - земляника;
Как земляника, дивно цветёт
Моя родная страна.

За домом черёмуха стоит,
Цветы - по бокам её,
Цветы - по бокам её;
Мы родились в стране своей
На самых красивых рассветах.

За домом черёмуха стоит,
По бокам - вишни там,
По бокам - вишни;
И вишни, глядя на нас:
«Вы счастливы!» - говорят.
```
