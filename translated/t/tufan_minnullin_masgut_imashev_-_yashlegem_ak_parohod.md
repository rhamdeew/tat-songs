# Оригинал

### Туфан Миңнуллин / Мәсгут Имашев - Яшьлегем - ак пароход

```
Ак пароход йөзеп килә Иделләрдә,
Кул болгадым,
борылышлардан күренгәндә.
Каршылады аны акчарлаклар,
Куанышып уйнадылар алар,
Шатлыклы уйлар уятып күңелләрдә.

Ак пароход узып китте Иделләрдә,
Кул болгадым,
борылышларга күмелгәндә.
Озаттылар аны акчарлаклар,
Моңсу гына кычкырдылар алар,
Ямансу уйлар калдырып күңелләрдә.
```

# Перевод

### Туфан Миңнуллин / Мәсгут Имашев - Юность моя - белый пароход

```
Белый пароход плывёт по Волге,
Я махнул рукой,
когда из-за изгибов он показался.
Встретили его белые чайки,
радостно кружились они,
пробуждая в сердце светлые думы.

Белый пароход уплыл по Волге,
Я махнул рукой,
когда скрылся за изгибами.
Провожали его белые чайки,
только грустно крикнули они,
оставив в сердце печальные думы.
```
