# Оригинал

### Тәслимә Низами / Урал Рәшитов - Казан - Кырлай - Симетбаш

```
Ник бу кадәр сагындыра икән
Газиз авылым Симетбаш?
Туган нигез моңаймасын,
Кайтыйк әле, кайтыйк, авылдаш.

Казан - Кырлай - Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.

Олыгая барган саен тарта,
Тарта икән туган як.
Симетбашым, әл дә син бар,
Минем бик бәхетле, иң бәхетле чак.

Казан - Кырлай - Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.

Нурислам чишмәсе суын эчә
Өзелеп сайрар сандугач.
Жырларымда синең моңың
Мәңге яшәр, мәңге, Симетбаш.

Казан - Кырлай - Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.
```

# Перевод

### Таслима Низами / Урал Рашитов - Казань - Кырлай - Симетбаш

```
Отчего так тянет, так тоскуется
По родной деревне - Симетбаш?
Пусть очаг не знает грусти вовсе,
Возвратимся, возвратимся, односельчанин.

Казань - Кырлай - Симетбаш дорога
Близкой стала, если песней пройти.
Не могу найти свою я юность -
Похоронена, в тумане скрыта.

Чем старее, тем сильнее тянет,
Тянет к родине, к родным местам.
Симетбаш мой, слава Богу, ты есть,
Мой счастливый, самый счастливый час.

Казань - Кырлай - Симетбаш дорога
Близкой стала, если песней пройти.
Не могу найти свою я юность -
Похоронена, в тумане скрыта.

Пьет из Нурислама родника
Соловей, и сердце в нем поет.
В песнях моих твоя мелодия
Вечно будет жить, навек, Симетбаш.

Казань - Кырлай - Симетбаш дорога
Близкой стала, если песней пройти.
Не могу найти свою я юность -
Похоронена, в тумане скрыта.
```
