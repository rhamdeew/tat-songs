# Оригинал

### Татар халык җыры - Дим-дим

```
Җилфер-җилфер, дим-дим,
Йөргән чакта, дим-дим,
Җилләр ача куенымны.
Янып-көеп, дим-дим,
Йөргән чакта, дим-дим,
Дуслар ача күңелемне.

Җилфер-җилфер, дим-дим,
Итүе лә, дим-дим,
Җилнең алып китүе.
Күңелле дус, дим-дим,
Ишләр белән, дим-дим,
Бергә гомер итүе.

Җилфер-җилфер, дим-дим,
Ука чуклар, дим-дим,
Ник җилферди, җил дә юк?
Синең кебек, дим-дим,
Матур ярлар, дим-дим,
Авылда түгел, илдә юк.
```

# Перевод

### Татарская народная песня - Дим-дим

```
Шелест-шелест, дим-дим,
Когда гуляю, дим-дим,
Ветры открывают душу мою.
Горя-покалывая, дим-дим,
Когда гуляю, дим-дим,
Друзья открывают сердце моё.

Шелест-шелест, дим-дим,
И даже дуновение, дим-дим,
Как уносит ветер прочь.
Весёлый друг, дим-дим,
С соседями вместе, дим-дим,
Вместе жизнь проживать.

Шелест-шелест, дим-дим,
Иглы колют, дим-дим,
Зачем шелестит, ветра ведь нет?
Таких как ты, дим-дим,
Красивых красавиц, дим-дим,
Нет ни в селе, ни в целом свете.
```
