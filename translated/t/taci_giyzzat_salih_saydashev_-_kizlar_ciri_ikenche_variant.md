# Оригинал

### Таҗи Гыйззәт / Салих Сәйдәшев - Кызлар җыры (икенче вариант)

```
Җырлый-җырлый, гөрли-гөрли
Өмәләрдә күңел ачабыз.

Ай-һай, дусларым,
Дусларым, дус-ишләрем.
Аңламыйсыз йөрәгемнең
Нигә ярсып типкәнен.
Аңламыйсыз күңелем тәрәзәсен
Кемнең килеп чирткәнен.

Ак тулалар тиз керешә
Аркау буен тигез эрләгәч.
```

# Перевод

### Таҗи Гыйззәт / Салих Сәйдәшев - Песня девушек (второй вариант)

```
Поючи, звеня, смеясь,
Мы на омэ веселимся.

Ай-ха, друзья мои,
Друзья, товарищи мои.
Вы не знаете, зачем
Сердце так пылко стучит.
Вы не знаете, кто к окну
Моей души подошёл и постучал.

Белые нити быстро идут,
Когда ровно выведешь уток.
```
