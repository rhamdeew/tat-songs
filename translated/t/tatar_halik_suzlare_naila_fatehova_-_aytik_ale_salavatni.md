# Оригинал

### Татар халык сүзләре / Наилә Фатехова - Әйтик әле салаватны

```
Әйтик әле салаватны
Пәйгамбәрләр рухына,
Әйтик әле салаватны
Пәйгамбәрләрнең үзенә.
Әйтик әле салаватны
Рәсүлулла рухына,
Әйтик әле салаватны
Рәсүлулланың үзенә.

Әйтик әле салаватны
Әминәнең рухына,
Әйтик әле салаватны
Әминәнең үзенә.
Әйтик әле салаватны
Хәдичәнең рухына,
Әйтик әле салаватны
Хәдичәнең үзенә.

Әйтик әле салаватны
Фатыйманың рухына,
Әйтик әле салаватны
Фатыйманың үзенә.
Әйтик әле салаватны
Пәйгамбәрләр рухына,
Әйтик әле салаватны
Пәйгамбәрләрнең үзенә.
```
