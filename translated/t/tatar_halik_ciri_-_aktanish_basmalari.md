# Оригинал

### Татар халык җыры - Актаныш басмалары

```
Агыйделдән күренеп тора
Актаныш басмалары шул;
Кызлар чәчен үреп салса,
Җилферди тасмалары.

Апрель җитми, агып бетми
Агыйделнең бозлары шул;
Мул уңышлар яулап ала
Егетләре, кызлары.

Каян барсаң, шуннан туры
Актанышның юллары шул;
Хезмәтләрдә җиңү яулый
Безнең район уллары.
```

# Перевод

### Татарская народная песня - Актанышские башмаки

```
С Агидели виднеются вдали
Актанышские башмачки;
Когда девушки косы заплетают,
Словно шёлковые ленточки.

Апрель не наступит, не растают
Льдины Агидели той;
Великие успехи одерживают
Юноши и девушки с душой.

Куда ни пойдёшь, прямо оттуда
Актанышские дороги идут;
В трудах победы добиваются
Нашего района сыновья.
```
