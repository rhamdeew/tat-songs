# Оригинал

### Татар халык җыры - Мөслим (икенче вариант)

```
Мөслимнең киң болыннары,
Матур әрәмәләре шул,
Матур әрәмәләре;
Шунда чыгып күңел ача
Эшчән, батыр яшьләре.

Уралып-уралып ага
Мөслимнең Ык сулары шул,
Мөслимнең Ык сулары;
Шул суларда йөзеп үсә
Алсу йөзле кызлары.

Ал, зәңгәр чәчәкләр җыйдым
Мөслим су буйларыннан шул,
Мөслим су буйларыннан;
Туган илем, үскән җирем
Чыкмыйсың уйларымнан.
```

# Перевод

### Татарская народная песня - Муслим (второй вариант)

```
Широкие луга Муслима,
Красивые рощи те,
Красивые рощи;
Там выходя, душу раскрывают
Трудолюбивые, смелые молодёжь.

Извиваясь-изгибаясь течёт
Муслима Ик реки вода та,
Муслима Ик воды;
В тех водах, плавая, растут
Румяными щёками девы.

О, голубые цветы собирал я
Муслима воды берегов от тех,
Муслима воды берегов;
Родной мой край, выросшая моя земля
Не выходишь ты из моих мыслей.
```
