# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Авыл көе (дүртенче вариант)

```
Биек тауларның башында,
Ах, мин каеннар булсам;
Ах, үзем каеннар булып,
Син былбыл булып кунсаң.

Безнең янда салкын чишмә -
Гәрәбә эри торган;
Бер сандугач булсын икән
Арада йөри торган.

Ал дими күңелләрем,
Гөл дими күңелләрем;
Синнән башка бүтәннәрне
Теләми күңелләрем.
```

# Перевод

### Татар халык җыры - Татар халык җыры - Деревенский напев (четвертый вариант)

```
На вершинах гор высоких,
Ах, березами бы стать;
Ах, мне быть листвой березовой,
Тебе - соловьем слетать.

У нас рядом - ключ студеный,
Где янтарь, как будто, тает;
Пусть бы соловей один лишь
Между нами пролетает.

Не манит ни алый цвет,
Не манит цветок иной;
Без тебя других не хочет,
Не желает сердце мое.
```
