# Оригинал

### Татар - Башкорт халык җыры - Гөлҗамал

```
Шәмдәлләрдә генә утлар яна, Гөлҗамал,
Җиткән кызлар киндер җеп эрлиләр.

Энҗе дә мәрҗән кызларның кул бавы,
Авыр җан сөйгәннәрнең булмавы.
```

# Перевод

### Татар - Башкирская народная песня - Гульжамал

```
В лампадах только горят огни, Гульжамал,
Подросшие девушки прядут льняную нить.

Жемчуг и коралл - браслеты девушек,
Отсутствие тех, кого любила душа.
```
