# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Хатлар язам

```
Хатлар язам, ай, ярыма,
Мәскәү калаларына;
Әгәр тизрәк син кайтмасаң,
Үзем барам яныңа.

Өстәлләрдә кара торсын,
Кулларым яза торсын;
Сине күреп сөйләшкәнче,
Хатларым бара торсын.

Тәрәзә төбем исле гөл,
Сал да балдагыңны эл.
Сагынмагач, хат язмыйлар,
Сагынганны шуннан бел.
```

# Перевод

### Татар халык җыры - Пишу письма

```
Письма пишу, ах, милому,
В города московские;
Если скоро ты не вернёшься -
Я сама приду к тебе.

Пусть на стол легли листочки,
Пусть рука не устаёт;
Пока с тобой не свидимся -
Письма к тебе пусть идут.

У окна у меня цветок,
Брось и кольцо своё на нём.
Не тоскуя, писем не пишут -
По письмам и узнают тоску.
```
