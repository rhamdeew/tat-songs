# Оригинал

### Татар халык җыры - Казансу

```
Әбәү, Алла, уф, Алла,
Торган җирләрем кала.
Торган җирләрем жәл түгел,
Туып үскән ил кала.

Казансуны чыккан чакта,
Җуйдым беләзегемне.
Китәрсең дә кайтмассың дип,
Өзмәгез үзәгемне.

Казансуга басма салдым,
Тар булмаска чыгарга.
Бергә булган вакытларны
Исләреңнән чыгарма.

Кашың кара, кашың кара,
Каралткангадыр инде.
Карыйм, карыйм, карап туймыйм,
Яраткангадыр инде.

Казаннарга барам әле,
Казан суы тәмлегә.
Бир кулыңны, бергә булыйк,
Аерылмаска мәңгегә.

Аерылмаска, аерылмаска,
Аерылмаска мәңгегә.
```

# Перевод

### Татарская народная песня - Казанка

```
Ох, Аллах, уф, Аллах,
Остаются родные края.
Не жаль родных краёв,
Остаётся страна, где вырос.

Когда выходила к Казанке,
Омыла я свои запястья.
Что уйдёшь и не вернёшься -
Не перерезайте мне arteria.

Положила ловушку в Казанку,
Чтоб не было тесно плыть.
Времена, когда были вместе -
Не изгоняй из своей памяти.

Брови твои черны, брови твои черны,
Вероятно, почернели уже.
Смотрю, смотрю, насмотреться не могу,
Вероятно, уже полюбил.

В Казань еду сейчас,
Вода Казанки сладка становится.
Дай руку твою, будем вместе,
Чтоб не расставаться никогда.

Не расставаться, не расставаться,
Не расставаться никогда.
```
