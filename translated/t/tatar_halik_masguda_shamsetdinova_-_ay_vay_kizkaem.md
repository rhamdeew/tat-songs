# Оригинал

### Татар халык / Мәсгудә Шәмсетдинова - Ай, вай, кызкаем

```
Ай-вай, кызкаем, печәнчегә бирәем,
Ай-вай, кызкаем, печәнчегә бирәем,
Ай-вай, димәем,
Печән чабу бигрәк авыр,
Ай-вай, сөймәем.

Ай-вай, кызкаем, терлекчегә бирәем,
Ай-вай, кызкаем, терлекчегә бирәем,
Ай-вай, димәем,
Терлекченең эше авыр,
Ай-вай, сөймәем.

Ай-вай, кызкаем, балыкчыга бирәем,
Ай-вай, кызкаем, балыкчыга бирәем,
Ай-вай, димәем,
Балыкчының тәне суык,
Ай-вай, сөймәем.
```

# Перевод

### Татар халык / Мәсгудә Шәмсетдинова - Ай, вай, кызкаем

```
Ай-вай, девчонка моя, отдам тебя косарю,
Ай-вай, девчонка моя, отдам тебя косарю,
Ай-вай, не скажу,
Сенокос - работа тяжка,
Ай-вай, не люблю.

Ай-вай, девчонка моя, отдам тебя скотнику,
Ай-вай, девчонка моя, отдам тебя скотнику,
Ай-вай, не скажу,
У скотника труд нелёгок,
Ай-вай, не люблю.

Ай-вай, девчонка моя, отдам тебя рыбаку,
Ай-вай, девчонка моя, отдам тебя рыбаку,
Ай-вай, не скажу,
У рыбака тело стынет,
Ай-вай, не люблю.
```
