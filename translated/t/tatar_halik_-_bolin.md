# Оригинал

### tatar_halik_ - Болын

```
Бигрәк матур болын,
Бигрәк матур...
Ник микән мин бер кош булмадым?
Нигә икән чүт-чүт сайрап
Болын талларына кунмадым?

Сагынам син, болын,
Сагынам сине...
Сагынып җырлыйм синең турында.
Сагынып җырлаганга йөрәгемнән
Моңлы чыга минем җырым да.
```

# Перевод

### tatar_halik_ - Луг

```
Так прекрасен луг,
Так прекрасен...
Отчего я птицей не стал?
Почему, едва пропев,
На ивы луга не слетал?

Я тоскую, луг,
Я тоскую по тебе...
С тоской пою я о тебе.
И от этой тоски из сердца
Печальной выходит моя песня.
```
