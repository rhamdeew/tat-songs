# Оригинал

### Татар - Башкорт халык җыры - Борай

```
Җырлыйк Борай көйләренә,
Без бит Борай яклары.
Башкаемның ни уйларга
Белми торган чаклары.

Кулъяулыгым төшеп калды
Борай болыннарында.
Үземнең дә исем китә
Болай булуларыма.

Бара торгач бер утырдым
Биек тау билләренә.
Без түгел кошлар талпына
Кайтырга илләренә.

Урал тавы кайда диләр,
Урал тавы күренми.
Яшьлектәге саф мәхәббәт
Нишләсәң дә сүрелми.
```

# Перевод

### Башкирская народная песня - Бурай

```
Споём напевы Бурая,
Мы ведь с бураевских краёв.
Не знают, о чём мечтать,
Порой наши сородичи.

Платок выпал из рук
На луга Бурая.
И сама память fadeет
От того, что всё так получается.

На ходу присела однажды
На гребни высоких гор.
Не мы, а птицы стремятся
Вернуться в свои края.

Где Урал-гора, говорят,
Урал-гора не видна.
Чистая любовь молодости
Чем ни занимайся, не fadeет.
```
