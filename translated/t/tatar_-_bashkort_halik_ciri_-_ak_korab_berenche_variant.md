# Оригинал

### tatar_ - Татар - Башкорт халык җыры - Ак кораб (беренче вариант)

```
Агыйделдә ак кораб ла
Утырдым алга карап.
Бер генә сүз әйтер идем
Нурлы йөзеңә карап.

Агыйделне иңләп, буйлап
Йөзәләсе йөрәккәем.
Яшь башыңа кайгы төшсә
Түз инде йөрәккәем.

Ал булмый, булмый, булмый ла
Гел булмый, булмый, булмый.
Бер булганмый сулар тыңмый
Яратмый булмый, булмый.
```

# Перевод

### tatar_ - Татар - Башкирская народная песня - Белый корабль (первый вариант)

```
По Агидели белый корабль,
Сел я, глядя лишь вперёд.
Я бы слово одно сказал,
В светлый лик твой глядя вновь.

Вдоль и поперёк Агидели
Плыть желает сердце моё.
Если горе на юность ляжет,
Потерпи, сердце моё.

Нет - не выйдет, не выйдет, не выйдет,
Никогда не выйдет, не выйдет.
Не удержать и воды - не слушают,
Не любить не выйдет, не выйдет.
```
