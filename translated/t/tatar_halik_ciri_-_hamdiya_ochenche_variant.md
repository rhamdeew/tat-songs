# Оригинал

### Татар халык җыры - Хәмдия (өченче вариант)

```
Агиделкәй аккын, суы салкын,
Барма якын, дулкын какмасын.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиңне таралган чагы.

Бер битең айдай, берсе кояш,
Яман күздән аллам сакласын.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиңне таралган чагы.

Ярлыгынай булып яраралмаган,
Җаннар көйдерер өчен яралган.
```

# Перевод

### Татарская народная песня - Хамдия (третий вариант)

```
Агидель быстрая, вода холодна,
Не подходи близко, волна не ударит.

Белый платок Хамдии
Когда она недавно стала невестой;
Хамдии слов не говорите,
Время когда её коса расплетена.

Одно лицо твоё как месяц, другое как солнце,
От дурного глаза пусть Аллах сохранит.

Белый платок Хамдии
Когда она недавно стала невестой;
Хамдии слов не говорите,
Время когда её коса расплетена.

Не суждено быть суженой,
Рождена чтобы жечь души.
```
