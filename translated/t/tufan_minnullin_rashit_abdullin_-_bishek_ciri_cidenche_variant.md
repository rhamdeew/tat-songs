# Оригинал

### Туфан Миңнуллин / Рәшит Абдуллин - Бишек җыры (җиденче вариант)

```
Куанычларым, шатлыгым булып,
Балам - багалмам, үсәсеңме син?
Сагынып җырлар җырыма минем,
Күземнең нуры, күчәрсеңме син?

Бәхет гөлемнең чәчәге булып,
Былбыл кошкаем, яшәрсеңме син?
Әнием диеп, кадерлем диеп,
Сандугачкаем, дәшәрсеңме син?

Гомерең сукмагын зур юллар буйлап,
Өмет йолдызым, утәрсеңме син?
Илеңнен данын, намус, вөҗданын,
Балам, әй балам, итәрсеңме син?
```

# Перевод

### Туфан Миңнуллин / Рәшит Абдуллин - Колыбельная песня (седьмой вариант)

```
Радостью моей, счастьем моим,
Ребёнок мой - родная кровь, растёшь ли ты?
С тоской по тебе в песнях моих,
Свет очей моих, промелькнёшь ли ты?

Став цветком цветника моего счастья,
Мой соловушка, поживёшь ли ты?
Маму зовя, родную маня,
Мой ласточка, отзовёшься ли ты?

По большим дорогам по жизненному пути,
Звезда надежды моя, пройдёшь ли ты?
Честь страны своей, совесть, честь и достоинство,
Ребёнок мой, о дитя моё, сохранишь ли ты?
```
