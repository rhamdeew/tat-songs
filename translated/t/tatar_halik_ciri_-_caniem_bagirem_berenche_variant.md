# Оригинал

### Татар халык җыры - Җаныем, бәгырем (беренче вариант)

```
Иртә буран, кич тә буран,
Мамык шәлеңне уран шул,
Мамык шәлеңне уран.
Кигән киемең күрсәм дә,
Җаныем, бәгырем,
Үзең күргәндәй булам.

Ишек алдында йөримен,
Карлар яуса көримен шул,
Карлар яуса көримен.
Синнән башка ямь табалмый,
Җаныем, бәгырем,
Ямансулап йөримен.

Тимер чана карлар яра,
Бу йөрәгем ник яна шул,
Бу йөрәгем ник яна?
Бу йөрәгем белеп яна,
Җаныем, бәгырем,
Сөйгәнем ятка кала.
```

# Перевод

### Татарская народная песня - Душа моя, родная (первый вариант)

```
Утром вьюга, вечером метель,
Ватный твой платок в клочья, о,
Ватный твой платок в клочья.
Видеть твою одежду -
Душа моя, родная,
Будто самой тебя вижу.

У двери хожу я,
Снег пусть валит - жгу, о,
Снег пусть валит - жгу.
Кроме тебя краски не найду,
Душа моя, родная,
Хожу, тоскуя.

Железные сани снег режут,
Это сердце зачем болит, о,
Это сердце зачем болит?
Это сердце зная болит,
Душа моя, родная,
Любимый мой остаётся в стороне.
```
