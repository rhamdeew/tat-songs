# Оригинал

### Татар халык җыры - Күк күгәрчен (икенче вариант)

```
Күгәрченнәр ояда,
Оялары пыяла.
Күгәрчен кебек гөрләшеп,
Күк күгәрчен, сандугачым,
Яшик бергә дөньяда.

Сандугачым, син сайрама,
Күгәрченем, син гөрлә.
Күк күгәрчен, сандугачым,
Үзем аерым, күңелләрем
Гел синең белән бергә,

Күгәрченнәр басып кына
Талларны сыгылдыра.
Бергә чакта беленми ул,
Күк күгәрчен, сандугачым,
Аерылгач сагындыра.
```

# Перевод

### Татар халык җыры - Небесный голубь (второй вариант)

```
Голуби в гнезде,
Гнезда у них стеклянные.
Голубиной воркотнёй,
Голубь небесный, соловей мой,
Давай жить вместе на земле.

Соловей мой, не пой,
Голубь мой, воркуй.
Голубь небесный, соловей мой,
Я отдельно, а душой
Я всегда с тобой вместе,

Голуби, присев лишь,
Ивы пригибают.
Пока мы вместе - не заметишь,
Голубь небесный, соловей мой,
Разлучишься - тоской томит.
```
