# Оригинал

### Татар халык җыры - Сагынам, дуслар (дүртенче вариант)

```
Сагынам, дуслар, сагынам, дуслар,
Сагынам, дуслар, илкәйне, лә,
Сагынам, дуслар, илкәйне.
Илкәй белән бергә сагынам
Яшьли сөйгән иркәмне.

Агасың Иделкәем лә,
Биек ярларың белән, шул,
Биек ярларың белән.
Калдың ла туган илкәем
Сөйгән ярларым белән.

Йөзем бакчасына да кереп
Өзеп алдым бер йөзем, шул,
Өзеп алдым бер йөзем.
Өзеп алган йөзем кебек,
Чит җирләрдәй берүзем.
```

# Перевод

### Татар халык җыры - Скучаю, друзья (четвертый вариант)

```
Скучаю, друзья, скучаю, друзья,
Скучаю, друзья, по родной стороне,
Скучаю, друзья, по родной стороне.
И с краем родным вместе скучаю
По милой, что в юности я полюбил.

Течешь ты, Волга-матушка,
С высокими берегами,
С высокими берегами.
Осталась, родная страна,
С любимыми берегами.

В виноградник я входил,
Сорвал одну гроздь винограда,
Сорвал одну гроздь винограда.
Как гроздь сорванная, одинок я,
В чужих землях, как один.
```
