# Оригинал

### Татар халык җыры - Әтнә

```
Җырлыйк Әтнә көйләренә,
Җырчыларга таралсын;
Мул уңышлар үстерүдә
Безнең Әтнә дан алсын.

Матур итеп кошлар сайрый
Әтнә су буйларында;
Ятсам - төштә, торсам истә -
Син генә уйларымда.

Кызарып кояш чыкмыйча,
Уйсу җирдә су кипми;
Көлемсерәп карауларың
Күз алларымнан китми.

Шушы көйне матур итеп
Җырлыйлар Әтнәләрдә;
Үзәкләремне өзәләр,
Исмеңне әйтәләр дә.
```

# Перевод

### Татарская народная песня - Этна

```
Давайте споём песни Этны,
Пусть разнесутся средь певцов;
В умножении богатых успехов
Пусть наша Этна славу обретёт.

Красиво птицы заливаются
На берегах реки Этны;
Ложусь ли спать - во сне, проснусь ли -
Лишь ты одна в моих мыслях.

Пока заря не разгорится,
На низине роса не высохнет;
Твой улыбающийся взгляд
Не уходит из моих глаз.

Эту мелодию прекрасно
Поют в краях Этны;
Сердца наши растревожат,
И имя твоё произнесут.
```
