# Оригинал

### Татар халык җыры - Әлфия (беренче вариант)

```
Кызыл бәйли, кызыл кия
Колхоз кызы Әлфия.
Йөрәгемә ут кабынды,
Әлфия, дия, дия.

Тырышып эшли, хезмәт сөя
Колхоз кызы Әлфия;
Ахры, акылым җуям инде,
Әлфия, дия, дия.

Бите алсу, ирене чия
Колхоз кызы Әлфия;
Йөрәгемә ут кабынды,
Әлфия, дия, дия.
```

# Перевод

### Татар халык җыры - Татарская народная песня - Альфия (первый вариант)

```
Красным повяжет, в красное одета
Колхозница Альфия.
В сердце моем огонь разгорелся,
Альфия, шепчу, шепчу.

Трудом живет, работу любит
Колхозница Альфия;
Видно, рассудок теряю уже,
Альфия, шепчу, шепчу.

Щеки румяны, губы - как вишни
Колхозница Альфия;
В сердце моем огонь разгорелся,
Альфия, шепчу, шепчу.
```
