# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Аппагым-җанашым

```
Сандугачым, былбылым,
Нигә тындың, кошкаем, аппагым,
Талды мәллә телләрең, җанашым?

Минем дә бит, кошкаем,
Синең төсле, былбылым, аппагым,
Җырлап үтә көннәрем, җанашым.
```

# Перевод

### tatar_halik_ciri_ - Татарская народная песня - Аппагым, жанашым

```
Соловей мой, былбылым,
Отчего умолк ты, пташка, аппагым,
Неужели утомился ты, жанашым?

И у меня ведь, пташка,
Как и у тебя, былбылым, аппагым,
Дни мои проходят в песнях, жанашым.
```
