# Оригинал

### tatar_halik_ciri_ - Татар  халык җыры - Борынгы көй (икенче вариант)

```
Шамшәрифләр дигән, ай, шәһәрдә
Кич ахшамсыз капкалар ябылмый;

Узган гомер белән калган хәтер
Сатып алыйм дисәң дә табылмый.
```

------

# Перевод

### Татар халык җыры - Борынгы көй (второй вариант)

```
В Шамшәрифах, ай, в городе том
Вечером ворота не закрывают;

С прошедшей жизнью оставшая память
Не сыщется, даже купить пожелав.
```
