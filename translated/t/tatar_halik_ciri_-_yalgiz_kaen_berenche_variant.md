# Оригинал

### Татар халык җыры - Ялгыз каен (беренче вариант)

```
Ялгыз каен булмый, диләр,
Ялгыз каен күлләрдә ләй;
Ялгыз каеннардан ялгыз
Мин бүгенге көннәрдә.

Ялгыз каен булмый, диләр,
Менә бер ялгыз каен ла;
Ялгыз каеннардан ялгыз
Иттең бит, җаныкаем.

Ялгыз каен булмый, диләр,
Була торгандыр инде лә;
Һәркемнең дә бер хәсрәте
Була торгандыр инде.
```

# Перевод

### Татарская народная песня - Одинокий дуб (первый вариант)

```
Говорят, не бывает одиноких дубов,
Одинокие дубы только над озёрами;
Из всех одиноких дубов
Я самый одинокий в эти дни.

Говорят, не бывает одиноких дубов,
Вот есть один одинокий дуб;
Из всех одиноких дубов
Ты сделал одиноким, мое сердце.

Говорят, не бывает одиноких дубов,
Должно быть, уже стало;
У каждого ведь есть своя печаль,
Должно быть, уже стало.
```
