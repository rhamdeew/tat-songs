# Оригинал

### Татар халык җыры - Җаным жәл түгел сиңа

```
Алым жәл түгел сиңа,
Гөлем жәл түгел сиңа;
Алым түгел, гөлем түгел,
Җаным жәл түгел сиңа

Алларым диям сине,
Гөлләрем диям сине;
Алым түгел, гөлем түгел,
Җаннарым диям сине.

Әрәмәдә карама,
Карамада кондыз бар;
Караңгы дип килми калма,
Ай булмаса, йолдыз бар.

Түгәрәк күлдән салкын су
Эчтем җаным көйгәнгә;
Ерак җирне якын итеп
Килдем сине сөйгәнгә.
```

# Перевод

### Татар халык җыры - Тебя мне не жаль, душа

```
Рук не жаль мне для тебя,
И цветов не жаль, любя;
Не лишь рук и не цветов -
Жаль не душу для тебя.

Сердцем зову я тебя,
Цветом зову я тебя;
Не лишь рук и не цветов -
Всей душою зову тебя.

В роще - вяз да черный лес,
У корней бобер живет;
Не останься, молвя: темно, -
Нет луны - звезда взойдет.

Из кругляшки-озера
Пил я холод, душу жгло;
Даль я сделал близкой мне,
К тебе с любовью я пришел.
```
