# Оригинал

### Татар халык җыры - Исле гөл (икенче вариант)

```
Исле гөлем, исле була
Яран гөл төсле була шул.
Сагынганда гөлгә карыйм
Ул синең төсле була.

Тәрәзә төбем исле гөл,
Исләнәм исләренә лә.
Сагынганда гөлгә карап
Төшерәм исләремә.

Исле гөлем, исле була
Чәчәкләр аткан чакта да.
Шул чәчәкләргә карыйм мин
Сине сагынган чакта.
```

# Перевод

### Татарская народная песня - Ароматный цветок (второй вариант)

```
Ароматный мой цветок, ароматный,
Любимый на цветок похож.
Когда скучаю, на цветок гляжу -
Он на тебя похож.

Под окном моим ароматный цветок,
Вдыхаю аромат его.
Когда скучаю, на цветок смотрю,
Погружаюсь в воспоминания.

Ароматный мой цветок, ароматный,
Когда цветы расцветают.
На эти цветы я смотрю,
Когда по тебе скучаю.
```
