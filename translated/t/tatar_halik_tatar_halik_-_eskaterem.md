# Оригинал

### Татар халык / Татар халык - Эскәтерем

```
Эскәтерем киң генә,
Өстәл белән тиң генә.
Минем сөйгән җаныкаем
Җир йөзендә бер генә.

Сандугачның балалары
Бар да кунган тирәккә.
Мәхаббәтең бигрәк каты,
Кан төшерде йөрәккә.

Ике аккош очып бара,
Агыйделдә кышлавы.
Җан бирүдән авыр икән
Сөйгән ярның ташлавы.
```

# Перевод

### Татар халык / Татар халык - Эскәтерем

```
Эскәтерем широка,
Словно стол, ровна она.
Мой любимый, мой родной -
Он на свете лишь один.

У птенцов у соловья
Все на тополь сели вдруг.
Твоя любовь - такая жгучая,
Кровь в сердце мне пустила.

Два лебедя летят вдали,
Зимуют на Агидели.
Тяжелей, чем жизнь отдать,
Когда любимый бросит вдруг.
```
