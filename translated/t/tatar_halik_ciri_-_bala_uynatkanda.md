# Оригинал

### Татар халык җыры - Бала уйнатканда

```
Әлли-бәли итәргә тал бишеге бар аның,
Иркәләргә сөяргә, үз әнкәсе бар аның,
Әй-әй-йәй.
Әтисенә ай кебек, әнисенә көн кебек,
Апасына гөл кебек лә, әбисенә туп кебек,
Килеп киткән кешеләргә чүп кадәр дә юк кебек,
Килеп киткән кешеләргә чүп кадәр дә юк кебек.

Тыпыр-тыпыр биидер, аягы җиргә тимидер,
Очкан коштай талпынып, канатларын кагынып,
Һап-һап-һап...
Чәбәк-чәбәк әткәсе, биетеп тора әнкәсе,
Такмак әйтә апасы ла: әттә-түдел-түдәсе...
Авызын-күзен мимылдатып карап тора әбкәсе,
Авызын-күзен мимылдатып карап тора әбкәсе.

Үсеп буйга җитәр бу, галим булып китәр бу
Туган-үскән иленә тырышып хезмәт итәр бу,
Әй-әй-йәй...
Минем кызым бик иркә, башларында кәсинкә,
Аягында бәтинкә дә, күлмәге дә сәрпинкә,
Алтын чәчле, зәңгәр күзле - коеп куйган кәртинкә,
Алтын чәчле, зәңгәр күзле - коеп куйган кәртинкә.
Әй!!!
```

# Перевод

### Татарская народная песня - Когда ребёнок играется

```
Иволговая колыбель качать умеет,
Свою маму ласкать и любить умеет,
Ай-ай-яй.
Для отца - как луна, для матери - как солнце,
Для тёти - как цветок, для бабушки - как кукла,
Для гостей зашедших - как пылинка нет,
Для гостей зашедших - как пылинка нет.

Топ-топ танцует, ножкой не ступает,
Как птичка летит, крыльями машет,
Хап-хап-хап...
Папа-папа рядом, мама танцует,
Сестрёнка приговаривает: ать-тюдель-тюдес...
Бабушка глядит, улыбается,
Бабушка глядит, улыбается.

Вырастет большим, учёным станет,
Для родной страны трудиться будет,
Ай-ай-яй...
Моя дочь очень мила, косынка на голове,
Башмачки на ножках, платьице красивое,
Золотоволосая, синеглазая - картина нарисованная,
Золотоволосая, синеглазая - картина нарисованная.
Ай!!!
```
