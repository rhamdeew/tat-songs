# Оригинал

### Татар халык бию такмагы - Биисе килә

```
Биючеләр, басыгыз,
Әйләнмәсә башыгыз.
Басыгыз, басыгыз,
Әйләнмәсә башыгыз.

Биючеләр, биегез,
Матур безнең көебез.
Биегез, биегез,
Матур безнең көебез.

Биючеләр биегәндә
Идәннәре бөгелә.
Ярышларда җиңеп чыктык,
Безнең биисе килә.

Тыпыр-тыпыр, тыпыр-тыпыр,
Тыпыр-тыпыр биергә,
Эшлибез дә, җырлыйбыз да.
Вакыт та бар биергә.

Тыпыр-тыпыр тыпырдашып.
Бергә басыек әле.
Уйнап-көлеп, җырлап-биеп,
Күңел ачыек әле.
```

# Перевод

### Татар халык бию такмагы - Хочется танцевать

```
Плясуны, вставайте,
Коль не кружится голова.
Вставайте, вставайте,
Коль не кружится голова.

Плясуны, пляшите,
Наш напев - он так красив.
Пляшите, пляшите,
Наш напев - он так красив.

Когда плясуны пляшут,
Половицы гнутся враз.
В состязаньях победили -
Нам плясать охота, сейчас.

Топ-топ, топ-топ,
Топ-топ плясать нам всласть,
И работаем, и поем мы,
Есть и время поплясать.

Топ-топ, притопами,
Вместе встанем-ка сейчас.
Играя, смеясь, распевая-танцуя,
Откроем сердце в этот час.
```
