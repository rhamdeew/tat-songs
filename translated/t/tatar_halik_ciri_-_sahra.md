# Оригинал

### Татар халык җыры - Сәхрә

```
Сәхрә, сәхрә, сәхрәләрдә кошлар
Әй, ник моңлана?
Ашый торган җимеше юк, юк, юк.
Ашый торган җимеше юк, юк, юк.

Моңсыз гына егет
Әй, ник моңлана?
Сөйгән яры янында юк, юк, юк.
Сөйгән яры янында юк, юк, юк.
```
