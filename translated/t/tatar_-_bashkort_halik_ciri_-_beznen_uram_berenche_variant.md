# Оригинал

### tatar_ - Татар - Башкорт халык җыры - Безнең урам (беренче вариант)

```
Чәчем сүттем тарарга да,
Тәрәзә ачтым карарга.
Киткән юлларыңа чыгам,
Сагынганда карарга.

Аргы якка китеп бара
Чөгендер утаучылар.
Белсәң иде, кемнәр икән
Араны бутаучылар?!

Шәл яулыгым бәйләмиләр,
Читләре тузар диеп.
Кем уйлаган, үз иптәшем,
Араны бозар диеп?

Аргы яктан бирге якка
Кемнәр күпер салганнар?
Без аерылдык та сайргайдык,
Саргаймасын калганнар!

Безнең урам - матур урам,
Тезелеп үсә ак каен.
Шул каеннарга карыймын
Сине сагынган саен.
```

# Перевод

### tatar_ - Башкирская народная песня - Наша улица (первый вариант)

```
Косу распустила - расчесать бы,
Окно распахнула - глядеть.
На тропы, где ты уходил, выхожу,
Когда скучаю - смотреть.

На тот берег идут свёклу полоть.
Если б ты знал, кто меж нами смуту мутит?!

Шаль мою не завязывают - мол, края изотрутся.
Кто бы подумал: свой товарищ
Меж нами всё испортит, мол?

Кто мост построил с того берега на этот?
Мы разошлись - исхудали, иссохли,
Пусть не желтеют оставшиеся!

Наша улица - красивая улица,
Белые берёзы в ряд растут.
На эти берёзы смотрю
Каждый раз, как по тебе тужу.
```
