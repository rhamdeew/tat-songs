# Оригинал

### Тимер Йосыпов / Юлай Моратов - Тыныч йоклый микән үлгәннәр?

```
Тыныч йоклый микән үлгәннәр,
Аларга бит җилләр кагылмый?
Баш очында болыт агылмый,
Калкан чакта язгы үләннәр, үләннәр.

Тыныч ята микән үлгәннәр,
Колак салмый сайрар кошына?
Кояш кунмый алар учына,
Шарлап аккан чакта үзәннәр, үзәннәр.

Калкан чакта язгы үләннәр,
Ургыламы болыт үрләргә?
Тыныс ята микән гүрләрдә
Илем диеп яуда үлгәннәр, үлгәннәр?
```

# Перевод

### Тимер Йосыпов / Юлай Моратов - Спят ли мирно павшие?

```
Спят ли мирно павшие,
Ветер их не тревожит ли?
Не плывут над ними тучи,
Когда поднимаются весенние травы, травы.

Тихо ли лежат павшие,
Не внимая песне птиц?
Не садится солнце на их ладони,
Когда в долинах шумит поток, поток.

Когда поднимаются весенние травы,
Рвутся ли тучи к гребням?
Тихо ли в могилах покоятся
Те, кто пал в бою за Родину, за Родину?
```
