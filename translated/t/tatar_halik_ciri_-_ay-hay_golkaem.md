# Оригинал

### Татар халык җыры - Ай-һай, гөлкәем

```
Җирән атым, ай, ялларын
Алтын тарак белән лә
Тарыймын, шул,
Ай-һай, гөлкәем.

Сез дусларым килә дигәч,
Кайдан калку - шуннан ла
Карыймын, шул,
Ай-һай, гөлкәем,.

Алмагачның, алмаларын
Был-был комлар кунып ла
Ашасын, шул,
Ай-һай, гөлкәем,

Уйнап җибәр, тартып җибәр -
Был-был кошлар сайравына
Охшасын, шул
Ай-һай, гөлкәем.
```

# Перевод

### Татарская народная песня - Ай-хай, моё цветочек

```
Рыжего коня моего, ай,
Гребешком золотым, ай
Расчешу я того,
Ай-хай, моё цветочек.

Когда вы, друзья мои, придёте,
Откуда появитесь - оттуда, ай,
Смотрю я на того,
Ай-хай, моё цветочек.

Яблоки на яблоне,
Проглотив комары, ай,
Пусть съест их тот,
Ай-хай, моё цветочек.

Пусть играет, потанцует -
На пение соловьёв, ай,
Пусть будет похож тот,
Ай-хай, моё цветочек.
```
