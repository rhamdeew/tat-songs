# Оригинал

### Тимер Йосыпов / Рамил Медъяр-Камал - Биеп кал (икенче вариант)

```
Аланнарга чыгып кына
Бер сызгыраек әле.
Тын кичләрдә яшьлек җырын
Бер сыздыраек әле.

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.

Бозмаек без араларны,
Форсат юк телләшергә.
Берәү сиңа туйга килә,
Берәү бәхилләшергә...

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.

Сылап-сыйпап бетерерсең
Битеңдәге сырыңны.
Туй кызганда, сузма әле
Бәхилләшеү җырыңны.

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.
```

# Перевод

### Тимер Йосыпов / Рамил Медъяр-Камал - Спляши напоследок (второй вариант)

```
Выйдем в поле, на простор,
Свистнем звонко разок.
В тихий вечер песню юных
Протянем ещё разок.

Эй, добавь веселья свадьбе -
Землю встряхни, спляши.
Если молодость уходит,
Напоследок от души.

Не разрушим мы родство,
Нет нам времени болтать.
Кто-то к свадьбе к тебе едет,
Кто-то - чтобы попрощать...

Эй, добавь веселья свадьбе -
Землю встряхни, спляши.
Если молодость уходит,
Напоследок от души.

Пригладить успеешь после
На лице свои следы.
Пока свадьба разгорается,
Не тяни прощальные песны.

Эй, добавь веселья свадьбе -
Землю встряхни, спляши.
Если молодость уходит,
Напоследок от души.
```
