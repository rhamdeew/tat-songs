# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Баламишкин

```
Ват, җимер коймасын,
Урамга атын куймасын;
Кызлы кеше кызын тыйсын,
Егетләрне тыймасын.

Әнкәй миңа хат язган,
Утын бетте, кайт, диеп;
Хатка каршы хатлар яздым,
Әй почмагын як, диеп.

Зимагур булды башларым,
Бер нужа да күрмәдем;
Караңгы төн, ява яңгыр,
Өстемә бер күлмәгем.
```

# Перевод

### Татар халык җыры - Баламишкин

```
Пусть не ломает, не рушит забор,
Коня на улицу не выпускает;
У кого есть дочь - пусть дочь свою держит,
А парней - не удерживает.

Мать мне письмо написала:
Дров не осталось, вернись, говорит;
Я ей в ответ письма писал:
Эй, свой угол береги, говорит.

Стал я зимагуром с головой,
Ни одной нужды не знал;
Ночь темна, и дождь льет,
На мне лишь одна рубашка.
```
