# Оригинал

### Татар халык җыры - Исле гөл (өченче вариант)

```
Исле гөлем, исле була
Яран гөл төсле була ла.
Сагынганда айга карыйм
Ай синең төсле була.

Исле гөлем, исле гөлем,
Исле гөлләрем сиңа ла.
Исле гөлем саф күңелем
Бүләк итәм мин сиңа.

Исле гөлем, исле була
Чәчәкләр аткан чакта ла.
Шул чәчәкләргә карыйм мин
Сезне сагынган чакта.
```

# Перевод

### Татарская народная песня - Ароматный цветок (третий вариант)

```
Ароматный мой цветок, ароматным будет,
Любимая подобна цветку прекрасному.
Когда скучаю, на луну смотрю я,
Луна похожа на тебя становится.

Ароматный мой цветок, ароматный мой цветок,
Ароматные цветы мои тебе дарю.
Ароматный мой цветок, чистая душа моя,
В подарок отдаю тебе.

Ароматный мой цветок, ароматным будет,
Когда цветы распускаются весной.
На эти самые цветы смотрю я,
Когда по вам скучаю я.
```
