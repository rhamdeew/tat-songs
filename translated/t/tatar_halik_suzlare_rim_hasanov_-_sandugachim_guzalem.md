# Оригинал

### Татар халык сүзләре / Рим Хәсәнов - Сандугачым - гүзәлем

```
Сандугачым, бик сайрыйсың,
Ни моңың бар бу айда?
Мин дә синең кебек моңлы,
Сабыр итәм шулай да.

Сандугачым - гүзәлем,
Тибрәтмә тал үзәген;
Түзәргә калгач, түзәрмен,
Өзелсә дә үзәгем.

Сандугачның балалары
Талдан имәнгә очты;
Сандугачым телгә килеп,
Сабыр итәргә кушты.

Сандугачым - гүзәлем,
Тибрәтмә тал үзәген;
Түзәргә калгач, түзәрмен,
Өзелсә дә үзәгем.
```

# Перевод

### Татарские народные слова / Рим Хасанов - Моя соловушка - красавица

```
Соловушка, так звонко ты поешь,
Что за грусть в этой луне?
Я, как ты, печалью полон,
Терплю, хоть тяжко мне.

Соловушка моя - красавица,
Не качай сердцевину ивы;
Если суждено терпеть - стерплю,
Даже если сердце надорвется.

Дети соловья
С ивы к дубу перелетели;
Соловушка моя заговорила,
Терпеть велела мне.

Соловушка моя - красавица,
Не качай сердцевину ивы;
Если суждено терпеть - стерплю,
Даже если сердце надорвется.
```
