# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Гармун, гармун

```
Гармун, гармун, гармуннарның
Алтын-көмеш телләре.
Урамнарда берәү йөри,
Абау ла, гармуннары.
Ник моңая гармуннары,
Әллә бармы моңнары?
Гармун, гармун,
Гармун, гармун, шул.

Моңаюның шул сәбәбе:
Төшкән гармун сәдәбе.
Кызлар белән танышырга
Шул булмасмы сәбәбе.
Гармун, гармун, гармун,
Гармун, гармун.

Гармун, гармун,
Гармун, гармун,
Сайрашырга тел булсын.
Егет әйтә, тиң булсын, дип,
Кызлар әйтә, үз булсын, дип,
Биргән сүзләр - сүз булсын, дип.
Гармун, гармун, гармун,
Гармун, гармун.
```

# Перевод

### Татар халык җыры - Гармонь, гармонь

```
Гармонь, гармонь, у гармоней
Золотые-серебряные язычки.
По улицам кто-то ходит,
Ай да, с гармонью.
Отчего грустит его гармонь,
Неужто в ней тоска?
Гармонь, гармонь,
Гармонь, гармонь, так.

Причина той грусти такая:
Померкла у гармони слава.
С девчатами знакомиться -
Не в этом ли причина.
Гармонь, гармонь, гармонь,
Гармонь, гармонь.

Гармонь, гармонь,
Гармонь, гармонь,
Пусть язычок для трелей будет.
Парень говорит: пусть будет вровень,
Девушки говорят: пусть будет своя,
Чтобы данное слово было словом.
Гармонь, гармонь, гармонь,
Гармонь, гармонь.
```
