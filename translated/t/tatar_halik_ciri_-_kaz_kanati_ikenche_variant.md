# Оригинал

### Татар халык җыры - Каз канаты (икенче вариант)

```
Каз канаты кат-кат була,
Ир канаты ат була;
Чит җирләрдә бик күп йөрсәң,
Сөйгән ярың ят була.

Каз канаты каурый-каурый,
Хатлар язарга ярый.
Уйнамагач та көлмәгәч,
Бу дөнья нигә ярый.

Каз канаты ап-ак була.
Ир канаты ат була;
Чит илләрдә йөргән чакта
Бик сагынган чак була.
```

# Перевод

### Татарская народная песня - Гусиное крыло (второй вариант)

```
Гусиное крыло многослойно,
Мужское крыло - как конское;
Если много странствуешь по чужим краям,
Любимая становится чужой.

Гусиное крыло перо к перу,
Годится для писем.
Когда не играешь и не смеёшься,
Зачем нужен этот мир?

Гусиное крыло белоснежно.
Мужское крыло - как конское;
Когда бродишь по чужим странам,
Наступает время тоски.
```
