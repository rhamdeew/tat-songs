# Оригинал

### Татар халык җыры - Зөләйха

```
Ай югары, ай югары,
Айга менәсем килә;
Айга менеп, түбән карап,
Сине күрәсем килә.

Кызарып кояш чыкканда
Нурлар төшә түтәлгә.
Күзем төшсә, күңелем төшми
Синнән башка бүтәнгә.

Озын кара чәчләреңне
Тасма кирәк үрергә.
Өзелеп сөйгән ярың килгән,
Чык, Зөләйха, күрергә.

Аклы-чуклы ал тасмаңны
Тагып җибәр тәңкәңә.
Шул егетне яратам, дип,
Әйт, Зөләйха, әнкәңә.
```

# Перевод

### Татарская народная песня - Зулейха

```
Ой высоко, ой высоко,
Хочется подняться на луну;
Взобравшись на луну, глядя вниз,
Хочется увидеть тебя.

Когда красное солнце всходит,
Лучи падают на крышу.
Если глаза мои спускаются, душа не спускается
Ни на кого, кроме тебя.

Твои длинные черные волосы
Нужно собрать в ленту.
Твой возлюбленный, любивший тебя всей душой, пришел,
Выйди, Зулейха, увидеть его.

Подарков красную ленту
Прикрепи к своей груди.
Скажи, Зулейха, своей матери:
«Я люблю того парня».
```
