# Оригинал

### Тәлгать Сабиров / Феликс Мортазин - Кайтып килешем

```
Бала чакта көтеп ала идек
Олы юлдан кунак кайтканны,
Иң беренче хәбәр сала идек
«Ачып куегыз!», - дип капканы
Менә бүген үзем кунак инде
Олы юлдан кайтып килешем!
Саклаганнар минем эзләрне дә
Бөдрә таллар, җитез инешем!
Менә инде бүген үзем кунак!
Олы юлдан кайтып килешем!

Юлларың да башка икән хәзер
Йортларың да инде яңарган!
Безне назлап баштан сыйпап сөйгән
Карт-карчыклар гына калмаган!
Тургайларның җырын тыңлый, тыңлый
Олы юлдан кайтып килешем!
Салкын сулар эчәр өчен кайтам
Олы чишмәм тауларын төшеп
Тургайларның җырын тыңлый, тыңлый
Олы юлдан кайтып килешем!

Бүген кемнәр көтеп алыр икән
Олы юлдан минем кайтканны
Бала-чага хәбәр салыр микән
«Ачып куегыз!», - дип капканы
Менә бүген үзем кунак инде
Олы юлдан кайтып килешем!
Саклаганнар минем эзләрне дә
Бөдрә таллар, җитез инешем!
Тургайларның җырын тыңлый, тыңлый
Олы юлдан кайтып килешем!
```

# Перевод

### Тәлгать Сабиров / Феликс Мортазин - Возвращаюсь домой

```
В детстве мы встречали с нетерпеньем
Гостя, что вернулся по большому тракту,
Первыми мы весть его несём:
«Отворяйте!», - к воротам кричали.
Вот сегодня сам я - гость дороги,
По большому тракту возвращаюсь!
Сберегли мои следы, как память,
Кудри ив да быстрый мой ручей.
Вот сегодня сам я - гость дороги!
По большому тракту возвращаюсь!

И дороги стали уж другими,
И дома теперь обновлены.
Те, кто нас ласкал, по голове гладя,
Старики да старушки - не остались.
Жаворонков песню слушая, слушая,
По большому тракту возвращаюсь!
Возвращаюсь пить холодные воды,
Сходя с гор к родной большой кринице.
Жаворонков песню слушая, слушая,
По большому тракту возвращаюсь!

Кто сегодня выйдет мне навстречу
С большого тракта - моего прихода?
Детвора ли весть разнесёт:
«Отворяйте!» - к воротам крикнет?
Вот сегодня сам я - гость дороги,
По большому тракту возвращаюсь!
Сберегли мои следы, как память,
Кудри ив да быстрый мой ручей!
Жаворонков песню слушая, слушая,
По большому тракту возвращаюсь!
```
