# Оригинал

### Татар халык җыры - Сагынам, дуслар (беренче вариант)

```
Сагынам, дуслар, сагынам, дуслар,
Сагынам туган илкәйне, шул,
Сагынам туган илкәйне.
Илкәй белән бергә сагынам
Яшьли сөйгән иркәмне.

Агыйделнең аръягында
Калды туган ил генәм, шул,
Калды туган ил генәм.
Калмады туган ил генәм,
Калды нечкә бил генәм. (Калды сөйгән яр гына.)

Өй артындагы бакчадан
Өзеп алдым бер йөзем, шул,
Өзеп алдым бер йөзем.
Өзеп алган йөзем кебек,
Чит илләрдә берүзем.
```

# Перевод

### Татарская народная песня - Скучаю, друзья (первый вариант)

```
Скучаю, друзья, скучаю, друзья,
Скучаю по родному краю, да,
Скучаю по родному краю.
Вместе с краем тем скучаю
По любимому с юных лет.

За рекою Агиделью
Остался родной мой край, да,
Остался родной мой край.
Не остался родимый край,
Осталась лишь тонкая талия. (Остался лишь милый друг.)

Из садика за домом
Сорвала я гроздь винограда, да,
Сорвала я гроздь винограда.
Словно сорванная гроздь,
Я одна в чужих краях.
```
