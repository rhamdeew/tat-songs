# Оригинал

### Татар - Башкорт халык җыры - Карагат кара була, бөрлегән кызыл була

```
Карагат кара була,
Бөрлегән кызыл була.

Сөйгәнеңне ятлар алса,
Күңелгә кыен була.

Карагат та түтәлдә,
Бөрлегән дә түтәлдә.

Җаным сөйми, күңелем ятмый,
Синнән башка бүтәнгә.

Карагат ашамим мин,
Бөрлегән ашамим мин.

Өзелеп сөйгән яркаемны
Мәңгегә ташламим мин.
Карагаттай, бөрлегәндәй,
Мәңге бергә яшәрбез.
```

# Перевод

### Татар - Башкирская народная песня - Голубика черная, костяника красная

```
Голубика черная бывает,
Костяника красная бывает.

Если у любимого заберут чужие,
На душе тяжело становится.

Голубика и вянет,
Костяника тоже вянет.

Душа не любит, сердце не приемлет
Никого, кроме тебя.

Голубики есть не буду,
Костяники есть не буду.

Разлученную любовь свою
Навсегда не брошу я.
Как голубика, как костяника,
Вместе жить будем вечно.
```
