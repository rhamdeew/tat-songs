# Оригинал

### Татар - Башкорт бию такмагы - Әпипә (икенче вариант)

```
Бас, кызым, Әпипә,
Син басмасаң, мин басам.
Синең баскан эзләреңә
Мин дә китереп басам.

Бас, кызым, Әпипә,
Син басмасаң, мин басам.
Әпипәнең туйларында
Биеп калам, ичмасам.

Бас, кызым, Әпипә,
Идәннәре селкенсен;
Идәннәре селкенсен дә,
Кияү күңеле җилкенсен.

Бас, кызым, Әпипә,
Сындыр идән тактасын;
Сындырсаң идән тактасын,
Кияү түләр акчасын.
```

# Перевод

### Татарская - Башкирская плясовая - Апипа (второй вариант)

```
Танцуй, дочка, Апипа,
Если ты не станцуешь, я станцую.
По следам твоим танцующим
Я пойду вослед за тобою.

Танцуй, дочка, Апипа,
Если ты не станцуешь, я станцую.
На свадьбе у Апипы
Буду танцевать, хоть не пил бы.

Танцуй, дочка, Апипа,
Пусть полы заходят ходуном;
Пусть полы заходят ходуном,
И жених воспрянет духом.

Танцуй, дочка, Апипа,
Сломай половицы в пол;
Коль сломаешь половицы,
То жених заплатит золотом.
```
