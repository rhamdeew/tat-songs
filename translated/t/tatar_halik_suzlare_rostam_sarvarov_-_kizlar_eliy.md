# Оригинал

### Татар халык сүзләре / Рөстәм Сәрвәров - Кызлар елый

```
Мендәренә чумып бер кыз елый,
Күз яшьләрен күрә ай гына.
Ерактагы ярын сагына ул -
Аны уйлый, көтә, кайгыра.

Төн тынлыгын кочып бер кыз елый,
Тик ай гына белә хәлләрен.
Шатлыгыннан елый - китергәннәр
Сөйгәненең кайтыр хәбәрен.

Ак каеннар янына бер кыз килгән,
Сорамагыз, нигә килгән ул.
Каеннарга сарылып кызлар елый
Ятлар белән күрсә сөйгәнен.
```

# Перевод

### Татарские народные слова / Рөстәм Сәрвәров - Девушки плачут

```
Уткнувшись в подушку, девушка плачет,
Лишь луна видит ее слезы.
Тоскует по далекому любимому -
Думает о нем, ждет, горюет.

Обняв тишину ночи, девушка плачет,
Лишь луна знает ее беду.
Плачет от радости - принесли
Весть о возвращении любимого.

К белым березам пришла девушка,
Не спрашивайте, зачем пришла.
К березам прижавшись, девушки плачут,
Если любимого видят с чужими.
```
