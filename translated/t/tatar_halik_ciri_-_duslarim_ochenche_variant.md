# Оригинал

### tatar_halik_ciri_ - Дусларым (өченче вариант)

```
Әй дусларым, сезгә ниләр куйыйм,
Казаннардин кайткан маллар юк.
Якын дуслар җыелганнар,
Бер җырламый хәлем юк.
Дусларым!
Сез дусларым хакы өчен
Бер җырламый хәлем юк.
Дусларым!
Сез дусларым хакы өчен
Бер җырламый хәлем юк.

Кистем дә генә агач, китте кыйгач,
Эчләреннән очты сандугач.
Кычкырып җырлап җибәрәм
Сез дусларны сагынгач.
Дусларым!
Сәхраларга чыгып җырлыйм
Сез дусларны сагынгач.
Дусларым!
Сәхраларга чыгып җырлыйм
Сез дусларны сагынгач.
```

# Перевод

### Татар халык җыры - Друзья мои (третий вариант)

```
Эй, друзья мои, чем же вас угощу,
Казанских товаров не привёз с собой.
Близкие друзья собрались,
Мне без песни не сидеть в тиши.
Дусларым!
Ради вашей дружбы, вашикук,
Мне без песни не сидеть в тиши.
Дусларым!
Ради вашей дружбы, вашикук,
Мне без песни не сидеть в тиши.

Срубил лишь дерево - покосился ствол,
Из его сердцевины соловей вспорхнул.
Я во весь голос песню отпущу,
Как только по вам, друзья, взгрустнусь.
Дусларым!
Выхожу в степь и песню завожу,
Как только по вам, друзья, взгрустнусь.
Дусларым!
Выхожу в степь и песню завожу,
Как только по вам, друзья, взгрустнусь.
```
