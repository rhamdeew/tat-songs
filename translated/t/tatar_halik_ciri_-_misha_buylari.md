# Оригинал

### Татар халык җыры - Мишә буйлары

```
Уйна, дустым, гармуныңны,
Якташларым тыңласын;
Син уйнасаң, мин җырлармын
Мактап Мишә елгасын.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.

Туган авылым Агайбаш ла,
Мишә су буйларында;
Мишә буе бөдрә таллар
Гел минем уйларымда.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.

Мишә буен яңгыратып
Уйна матур көйләргә;
Син уйнаган, мин җырлаган
Калсын сагнып сөйләргә.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.
```

# Перевод

### Татар халык җыры - Берега Миши

```
Играй, друг мой, на гармошке,
Пусть земляки послушают;
Ты играй - а я спою,
Славя речку Мишу.

Ясны берега Миши,
Волнами играет вода;
На откосах птицы поют,
Тоску наводят они.

Родное село Агайбаш -
У вод реки Миши;
Кудрявые ивы Миши
Всегда в моих мыслях.

Ясны берега Миши,
Волнами играет вода;
На откосах птицы поют,
Тоску наводят они.

По берегам Миши звени,
Играй красивые напевы;
Твоя игра и мой напев
Пусть остались для памяти с тоской.

Ясны берега Миши,
Волнами играет вода;
На откосах птицы поют,
Тоску наводят они.
```
