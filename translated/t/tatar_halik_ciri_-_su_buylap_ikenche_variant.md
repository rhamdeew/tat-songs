# Оригинал

### tatar_halik_ciri_ - Татар  халык җыры - Су буйлап (икенче вариант)

```
Идел бит ул, тирән бит ул,
Тирән бит ул, киң бит ул (ла).
Караңгы төн, ямансу көн -
Без аерылган көн бит ул.

Идел буйларында йөрдем
Тугайларын буйладым (ла).
Аерылгада кул бирмәдем
Күрешербез дип уйладым.

Идел буйларына барсам,
Сөйләр идем Иделгә (лә).
Минем сагышларымны да
Илтеп сал дип, диңгезгә.
```

# Перевод

### Татар  халык җыры - Су буйлап (икенче вариант)

```
Идель - он глубок, он бездонен,
Глубок он, широк он (ла).
Ночь мрачна, и день печален -
День разлуки это был.

По Идели я бродил,
По лугам его ходил (ла).
В час прощанья рук не сжал -
Верю, вновь увидимся.

Если к Идели приду,
Я бы ей слова сказал (лә).
И печали мои тоже
Унеси, отдай в море.
```
