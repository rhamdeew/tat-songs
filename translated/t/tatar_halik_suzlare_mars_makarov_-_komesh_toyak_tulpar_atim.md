# Оригинал

### Татар халык сүзләре / Марс Макаров - Көмеш тояк тулпар атым

```
Туп-туп кына басып килә
Көмеш тояк тулпар атым, һай!
Канатланып үзем киләм,
Җитмәс, диеп, язган хатыма

Атым килә туп-туп атлап,
Ташлы юлларны таптап, һай!
Сагынганда карар идем,
Карурман тора каплап.

Бергә чаклар искә төшә,
Казан (Өфе), диеп бер җырласам, һай!
Кемне сагынып җырлар идем,
Казаннарда (Өфеләрдә) син булмасаң.

Туп-туп кына басып килә
Көмеш тояк тулпар атым, һай!
Канатланып үзем киләм,
Җитмәс, диеп, язган хатым.
```

# Перевод

### Татарские народные слова / Марс Макаров - Мой серебрянокопытный тулпар

```
Цок-цок звонко поступая,
Мой серебрянокопытный конь-тулпар, эй!
Словно с крыльями лечу я,
К той, что "не успею" написала мне.

Мчится конь мой, цокот звонкий,
Каменистый путь топча, эй!
В час, когда тоска настигнет,
Взгляд застилет снежная метель.

Вспоминаю те мгновенья,
Лишь спою "Казань" ("Уфа"), эй!
Для кого же пел я песни,
Если в Казани (Уфе) тебя нет?

Цок-цок звонко поступая,
Мой серебрянокопытный конь-тулпар, эй!
Словно с крыльями лечу я,
К той, что "не успею" написала мне.
```
