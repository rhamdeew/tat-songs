# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Сагынам, дуслар (өченче вариант)

```
Сагынам, дуслар, сагынам, дуслар,
Сагынам туган илкәйне, шул,
Сагынам туган илкәйне.
Илкәй белән бергә сагынам
Яшьли сөйгән иркәмне.

Баш куям тезләреңә лә,
Яшь коям эзләреңә, шул,
Яшь коям эзләреңә.
Әллә ниләр бирер идем
"Аһ" дигән сүзләреңә.

Агасың Иделкәем лә,
Биек ярларың белән, шул,
Биек ярларың белән.
Каласың, туган илкәем
Сөйгән ярларым белән.
```

# Перевод

### Татар халык җыры - Сагынам, дуслар (өченче вариант)

```
Скучаю, друзья, скучаю, друзья,
Скучаю по родной сторонке, да,
Скучаю по родной сторонке.
С ней вместе тоскую я сердцем
По милой, любимой в юности.

К коленям твоим припаду,
Слезами следы омою, да,
Слезами следы омою.
Чего только не отдала бы
За слово твоё «ах».

Течёшь ты, Волга-река моя,
С высокими берегами, да,
С высокими берегами.
Останешься, родная сторонка,
С возлюбленными берегами.
```
