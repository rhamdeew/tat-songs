# Оригинал

### Татар халык җыры - Уйна тальян гармуныңны

```
Уйна тальян гармуныңны
Безнең авыл көенә.
Тәрәз саен моңнар тулып
Керсен авыл өенә.

Уйна тальян гармуныңны,
Кулларың да талмасын.
Моңсызлар да бер моңлансын
Йөрәкләре сызлансын.

Уйна тальян гармуныңны,
Егетләр җырлап үтсен.
Туган авылыннан киткәннәр
Ишетсә яшен түксен.
```

# Перевод

### Татар халык җыры - Сыграй, тальян-гармонь

```
Сыграй, тальян-гармонь,
На напев нашей деревни.
Пусть в каждое окно, наполнив, входят песни
В сельские дома.

Сыграй, тальян-гармонь,
Пусть руки твои не устанут.
Пусть и безголосые запоют,
Пусть заноет их сердце.

Сыграй, тальян-гармонь,
Пусть парни проходят с песней.
Кто ушел из родной деревни,
Услышав, пусть слезу прольет.
```
