# Оригинал

### Тәбрис Фәхретдинов / Марат Гобәйдуллин - Каракүл

```
Каракүлгә мин гел кайтам
Ерак булса да ара.
Мондый матур, гүзәл җиргә
Кайтмыйча түзеп кара.

Кушымта:
Кайткан чакта Каракүлгә
Сәлам бирә танышлар.
Җырым аша һәммәгезгә
Телим, дуслар, уңышлар.

Кайтсам, төшәм күл буена,
Куркытмый җил дә давыл.
Каракүл син минем өчен
Кадерле туган авыл.

Яр буйлары куе камыш,
Каракүл бик тирән ул.
Өзелеп, сагынып кайтыр өчен
Әти-әни кирәк ул.
```

# Перевод

### Табрис Фахретдинов / Марат Гобайдуллин - Каракуль

```
В Каракуль я всегда возвращаюсь
Даже если далека дорога.
В такую красивую, прекрасную землю
Не могу не вернуться - не получится.

Припев:
Когда еду в Каракуль
Приветствуют меня знакомые.
Через мою песню всем вам
Желаю, друзья, успехов.

Если вернусь - лягу к озеру,
Не пугает ни ветер, ни буря.
Каракуль, ты для меня
Дорогой родной аул.

Берегам густой камыш,
Каракуль очень глубокий он.
Чтобы, тоскуя, возвращаться вновь
Отец и мать нужны.
```
