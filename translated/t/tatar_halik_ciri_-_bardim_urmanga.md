# Оригинал

### Татар  халык җыры - Бардым урманга

```
Бардым урманга - соңга калдым,
Алма агачын кистем лә тал диеп.
Безгә кемнәр теләде икән
Яшьли сөйгән ярларыңнан кал, диеп.

Бардым урманга - соңга калдым,
Куакларын уздым ла өч яклап.
Ходай кушып кавыша калсак
Утырып елар идем лә кочаклап.
```

# Перевод

### Татар халык җыры - Бардым урманга

```
Пошла я в лес - да опоздала,
Яблоню срубила, приняв за иву.
Кто нам, скажите, пожелал
От юной любви оторвать, разлучить нас.

Пошла я в лес - да опоздала,
Кусты обошла с трёх сторон.
Коль Бог сведёт нас по своей воле,
Сяду бы и плакала, обняв тебя.
```
