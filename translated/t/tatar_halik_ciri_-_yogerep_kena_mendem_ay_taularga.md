# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Йөгереп кенә мендем, ай тауларга

```
Йөгереп кенә мендем, ай тауларга, ай тауларга,
Җилфер-җилфер торган ай хат өчен, ай хат өчен

Җырламас та идем бу җырымны, бу җырымны
Җырлыйм сез балалар ай хакы өчен

Җилфер генә иде җилфер йөрмәс идем, йөрмәс идем
Җил тавышкайларын ай сизмәсә, ай сизмәсә

Җырламас та идем бу җырымны
Сез балаларымны ай сөймәсә, ай сөймәсә.
```

# Перевод

### татар_halik_ciri_ - Татарская народная песня - Бегом взошла я, ай, на горы

```
Бегом взошла я, ай, на горы, на горы,
Ради письма, что трепетало, ай, трепетало.

Не пела б я эту песню, эту песню,
Но по вашему праву, дети, ай, по праву пою.

Если б трепетало - не шла бы, не шла бы,
Если б ветра голосов я, ай, не слышала.

Не пела б я эту песню,
Если б вас, детей моих, ай, не любила.
```
