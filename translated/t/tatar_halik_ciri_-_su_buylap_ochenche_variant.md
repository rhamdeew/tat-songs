# Оригинал

### Татар халык җыры - Су буйлап (өченче вариант)

```
Идел бит ул, тирән бит ул,
Тирән бит ул, киң бит ул (ла).
Караңгы төн, томанлы көн -
Без аерылган көн бит ул.

Идел бит ул, агым бит ул,
Идел диңгезгә китә (лә).
Аккан сулар, искән җилләр
Йөрәгемне җилкетә.

Идел буйларына барсам,
Әйтер идем Иделгә (лә).
Минем кайгыларымны да
Илтеп сал дип, диңгезгә.
```

# Перевод

### Татарская народная песня - Вдоль реки (третий вариант)

```
Это Волга, глубока она,
Глубока она, широка она.
Темной ночью, в день туманный
Мы расстались - тот был день она.

Это Волга, течет она,
Волга к морю уплывает.
Текут воды, ветры веют,
Сердце мое волнуют.

Коль приду к волжским берегам,
Я сказала б Волге той:
Ты печали мои все
Отнеси в море с собой.
```
