# Оригинал

### Татар халык җыры - Замана такмаклары (икенче вариант)

```
Әнә килә автомобиль,
Төягәннәр пыяла.
Үзе кызлар белән йөри,
Сөйләшергә ояла.

Егетләр чибәр булалар,
Эчмәгән чакларында.
Мәхәббәтсез күренәләр,
Магазин чатларында.

Бездә хәзер шундый мода:
Кызлар кия фуражка.
Иреннәренә сылыйлар,
Кило ярым краска.

Агыйделдә, Агыйделдә,
Агыйделдә боз биек.
Эчкәч, барып кочаклаган,
Багананы, кыз, диеп.

Исемнәре Габдулла,
Исемнәре Габдулла.
Ярны сөйсән, икене сөй,
Берсе кандидат була.

Безнең авыл егетләре
Җидешәр кат өйләнә.
Теше коелып бетсә дә,
«Мин егет», - дип сөйләнә.
```

# Перевод

### Татарская народная песня - Такмаки о современности (второй вариант)

```
Вот едет автомобиль,
Стекло нагружено.
С девчонками сама ходит,
Разговаривать стесняется.

Парни красивые бывают,
В трезвые времена.
Без любви выглядят,
В магазинах у входов.

У нас теперь такая мода:
Девочки носят фуражку.
Губы красят помадой,
Кило с половиной.

На Агидели, на Агидели,
На Агидели лёд высокий.
Выпьет, пойдёт обнимет,
Столб, приняв за девушку.

Имена у них Габдулла,
Имена у них Габдулла.
Любишь девушку - люби и вторую,
Один из кандидатов станет.

 Наши деревенские парни
По семь раз женятся.
Даже если все зубы выпадут,
«Я парень», - говорят они.
```
