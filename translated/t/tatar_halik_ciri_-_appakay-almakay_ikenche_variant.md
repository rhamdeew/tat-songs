# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Аппакай-алмакай (икенче вариант)

```
Җырлый-җырлый чәчәк җыям, чәчәк җыям,
Урал суы буеннан, буеннан;
Аппакай-алмакай,
Урал суы буеннан, буеннан,

Җырлап чәчәк җыйганда, җыйганда,
Син китмисең уемннан, уемннан,
Аппаккай-алмакай,
Син китмисең уемнан, уемнан.

Ап-ак урманкаем бөдрә-бөдрә,
Яфракларын төрләндереп күренә.
Аппакай-алмакай,
Каеннарны тибрәндереп җил өрә.

Каенлыкта шаулый-шаулый, сөенә-сөенә
Җиләк җыя авыл кызы Мөнирә.
Аппакай-алмакай,
Маңгай чәчен артка сирпеп бөгелә.
```

# Перевод

### tatar_halik_ciri_ - Татарская народная песня - Аппакай-алмакай (второй вариант)

```
Пою, да пою - цветы срываю, срываю
У реки Урал, у воды, у воды;
Аппакай-алмакай,
У реки Урал, у воды, у воды.

Пою и цветы собираю, собираю,
Ты не уходишь из дум моих, из дум моих,
Аппаккай-алмакай,
Ты не уходишь из дум моих, из дум моих.

Белая рощица кудрява-кудрява,
Пестрят на ней переливы листвы.
Аппакай-алмакай,
Ветер качает березы, звеня.

В березняке шумит и радуется,
Ягоды рвет деревенская Мөнирә.
Аппакай-алмакай,
Челку назад откинув, склонилась.
```
