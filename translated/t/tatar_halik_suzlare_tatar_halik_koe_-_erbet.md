# Оригинал

### Татар халык сүзләре / Татар халык көе - Эрбет

```
Эрбет юлларын мактыйлар,
Ак юллар тыпараек.
Эрбет кызларын мактыйлар,
Бер сөйләшеп караек.

Самавыр куйдым кайнарга
Эрбетнеке байларга.
Карама, кызлар, байларга,
Кара зифа буйларга.

Иртән торып тышка чыксам -
Кар яуган көрәгемә.
Кар яумаган көрәгемә,
Кан салган йөрәгемә.

Су буйлары, су буйлары,
Су буйларында нарат.
Суга барсам - елап кайтам,
Аккан суларга карап.
```
