# Оригинал

### tatar_halik_ciri_ - Татар  халык җыры - Сагынам, дуслар (икенче вариант)

```
Сагынам, дуслар, сагынам, дуслар,
Сагынам, дуслар, илкәйне, шул,
Сагынам, дуслар, илкәйне.
Илкәй белән бергә сагынам
Яшьли сөйгән иркәмне.

Яшь коям эзләреңә, дә,
Баш куям тезләреңә шул,
Баш коям тезләреңә.
Әллә ниләр бирер идем
"Аһ" дигән сүзләреңә.

Ишек алды яшел чирәм
Былбыл баласы кача, шул,
Былбыл баласы кача.
Бик сагынсам айга карыйм
Айны да болыт баса.
```

# Перевод

### Татар халык җыры - Скучаю, друзья (второй вариант)

```
Скучаю, друзья, скучаю, друзья,
Скучаю, друзья, по родному краю,
Скучаю, друзья, по родному краю.
С краем родным вместе скучаю
По милому, что любила в юности.

Слёзы лью по твоим следам,
Голову склоняю к твоим коленям,
Голову кладу на колени твои.
Ах, что бы ни отдала
За твои слова: «Ах».

У ворот зелёный луг,
Птенец соловья улетает,
Птенец соловья улетает.
Если сильно затоскую - гляжу на луну,
И луну закрывают облака.
```
