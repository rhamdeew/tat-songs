# Оригинал

### Туфан Миңнуллин / Резеда Ахиярова - Кайда алар?

```
Кайда алар? Кайда икән алар -
Сөю утларында янган чаклар,
Мөлдерәмә тулып ташкан күңелләргә
Йолдызлардан сихри моңнар тамган чаклар?

Кайда алар? Кайда икән алар -
Таңны таңга ялгап йөргән чаклар.
Хыялыбыз бишегендә тирбәлгәндә
Ай нурыннан тәкыялар үргән чаклар?

Кайда алар? Кайда икән алар -
Өмет ярларына җиткән чаклар.
Мәхаббәтне маяк итеп киләчәккә
Ай юлыннан икәү атлап киткән чаклар?
```

# Перевод

### Туфан Миңнуллин / Резеда Ахиярова - Где они?

```
Где они? Где же они -
Те мгновения, что горели в огне любви,
Когда в сердца, переполненные хрусталём,
С звёзд волшебные мелодии лились?

Где они? Где же они -
Те мгновения, когда рассвет к рассвету вели.
Когда в колыбели наших грёз качались
И из лунного света подушки ткали?

Где они? Где же они -
Те мгновения, когда к берегам надежды приплыли.
Когда с любовью как маяком в будущее
По лунной дорожке вдвоём шагнули?
```
