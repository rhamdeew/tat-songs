# Оригинал

### Татар - Башкорт халык җыры - Авыл көе (сигезенче вариант)

```
Ак келәтнең ишегендә
Калды каеш дилбегән.
Калмады каеш дилбегән,
Калды туган илгенәм.

Ай-һай аккош, матур кош ла,
Күп әйләнмә җиргә төш.
Канат очына хат куям,
Илем кайда шунда төш.

Аклы ситса күлмәгемнең
Якаларын кем уйган?
Китәсем килеп китмимен,
Туган илдән кем туйган?
```

# Перевод

### Татар - Башкирская народная песня - Деревенский напев (восьмой вариант)

```
У ворот белого амбара
Остался ременной повод.
Не повод там остался -
Осталась Родина одна.

Ай-һай, лебедь, птица ясная,
Не кружись - на землю сядь.
К краю крыла письмо привяжу:
Где мой край - туда и падь.

У белого ситца платья
Кто вырезал воротник?
Хочу уйти - да не ухожу,
Кто Родиной мог насытиться?
```
