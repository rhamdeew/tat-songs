# Оригинал

### Татар халык җыры - Әзифә

```
Калдырып кит, иркәм, кәләпүшең.
Читләренә укалар йотармын. Әзифә.

Калдырып китсәң жәлләмичә,
Хәсрәтемнән ялкын ут йотармын. Әзифә.

Төштем дәрьяларга, кер юарга.
Кер юарга түгел, еларга. Әзифә.

Еладым, еладым, юк беркайда.
Өзелеп сөйгән яр гынам, син кайда? Әзифә.
```

# Перевод

### Татар халык җыры - Әзифә

```
Оставь, любовь моя, свой калфак.
По краям его кружева проглочу. Әзифә.

Если уйдешь, не жалея,
С горя пламя огня проглочу. Әзифә.

Спустилась к водам - белье стирать.
Не белье стирать - плакать. Әзифә.

Плакала, плакала - нигде тебя нет.
Любимый мой, которого рвала душа, ты где? Әзифә.
```
