# Оригинал

### Татар халык җыры - Салкын чишмә (беренче вариант)

```
Биек тә биек тауның астында
Чылтырап ага суның салкыны.

Тау астында салкын чишмә
Чылтыр-чылтыр агадыр.
Иҗтиһат кыл кавышырга,
Гомер үтеп барадыр.

Эчләрем тулы сагыш уты,
Су сипсәм дә, бетми ялкыны.

Тау астында салкын чишмә
Чылтыр-чылтыр агадыр.
Иҗтиһат кыл кавышырга,
Гомер үтеп барадыр.
```

# Перевод

### татар халык җыры - Салкын чишмә (беренче вариант)

```
Под высокой-превысокой горой
Звенит прохлада бегущей воды.

Под горой холодный родник
Звеня-звеня струится.
Старайся, чтоб нам сойтись,
Жизнь уходит, уносится.

В груди моей огонь тоски,
Хоть лей водой - не гаснет пламя.

Под горой холодный родник
Звеня-звеня струится.
Старайся, чтоб нам сойтись,
Жизнь уходит, уносится.
```
