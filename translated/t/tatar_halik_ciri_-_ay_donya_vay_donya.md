# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Ай дөнья, вай дөнья

```
Дәрья суларын буйладым,
ай дөнья,
«Уф» дип әйтеп сулыймын шул,
вай дөнья.
Акылларымнан язам шул,
ай дөнья,
Узган көнемне уйласам,
вай дөнья.
Авыр гомер кичердем шул,
ай дөнья,
Хак язганга чара юк шул,
вай дөнья.
```

# Перевод

### Татар халык җыры - Ай дөнья, вай дөнья

```
Я моря обошёл своими тропами,
ай, дунё,
и, вздохнув, шепчу «уф» усталый,
вай, дунё.
Разум мой теряю в сумерках,
ай, дунё,
как прошлое вспоминаю -
вай, дунё.
Тяжёлую жизнь я вынес,
ай, дунё,
что Богом написано - не избежать,
вай, дунё.
```
