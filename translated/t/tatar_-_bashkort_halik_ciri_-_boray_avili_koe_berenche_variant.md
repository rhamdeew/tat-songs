# Оригинал

### Татар - Башкорт халык җыры - Борай авылы көе (беренче вариант)

```
Бара торгач бер утырдым
Биек тау битләренә.
Без түгел, кошлар талпына
Кайтырга илләренә.

Су буенда ике тирәк,
Берсен кисәргә кирәк.
Берсен киссәң ялгыз кала,
Ялгыз нәрсәгә кирәк.

Урал тавы биек диләр
Урал тавы күренми.
Яшьлектәге саф мәхәббәт
Нишләсәң дә сүрелми.

Карагайлы* карурманны
Үтәр көннәр булырмы.
Каршы килеп, кул бирешеп
Күршер көннәр булырмы.
```

# Перевод

### Татар - Башкирская народная песня - Мелодия села Борай (первый вариант)

```
Шел я дальше - сел немного
На откосы гор крутых.
Не мы, а птицы рвутся
Возвратиться в край родных.

У воды стоят две ивы,
Одну срубить бы надо.
Срубишь - другая одинока,
К чему ей одинокой быть?

Говорят: Урал высок, да
Сам Урал не виден мне.
Чистая любовь из юности
Не гаснет ни при чем.

Сквозь хвойный темный бор
Будут ли дни пройти?
Встретиться, сойтись за руки -
Будут ли такие дни?
```
