# Оригинал

### Таһир Асылгәрәй - Язгы яңгыр

```
Үтәр әле кышлар да,
Килер сагынган язлар.
Кояш нурыннан алда,
Болыннарымны назлар.

Язгы яңгыр, ямь уятыр,
Кышның әзләрен юар.
Тәүге тапкыр, нур уйнатып,
Яшь үләннәре туар.

Көннәр чумар чәчкәгә,
Төсләрен яңгыр буяр.
Төннәр йолдыз чәчсә дә,
Уңыш яңгыры кояр.

Язгы яңгыр, ямь уятыр,
Кышның әзләрен юар.
Тәүге тапкыр, нур уйнатып,
Яшь үләннәре туар.
```

# Перевод

### Тахир Асылгарей - Летний дождь

```
Пройдут и зимы эти,
Придут желанные лета.
Перед лучами солнца
Мои зеленые луга расцветут.

Летний дождь, пробуждает нежность,
Смоет зимнюю скуку.
Впервые, проснувшись светом,
Зеленые травы оживут.

Дни опустятся на цветы,
Дождь раскрасит их красками.
Ночи, хоть и звезды сыплются,
Будут плодотворно светиться.

Летний дождь, пробуждает нежность,
Смоет зимнюю скуку.
Впервые, проснувшись светом,
Зеленые травы оживут.
```
