# Оригинал

### Татар халык җыры - Аллы-гөлле гөлчәчәк (беренче вариант)

```
Аллы-гөлле гөлчәчәкне
Иснә, бәгырем, исле ул;
Үзе алсу, үзе нәфис,
Бигрәк синең төсле ул.

Гөл чәчәгенә бәйләнеп,
Чия пешә бал кебек.
Буең зифа яшь тал кебек,
Йөзең тулган ай кебек.

Алларым, диям сине,
Гөлләрем, диям сине;
Аллы-гөлле чәчәгем,
Дип, өзелеп сөям сине.
```

# Перевод

### Татарская народная песня - Цветочек с лепестками (первый вариант)

```
Цветочек с лепестками
Помни, мое сердце, он ароматный;
Сам по себе светлый, сам по себе чудесный,
Особенно твой цветной.

Привязавшись к цветку,
Ягоды спелые как мед.
Твоя спина гибкая, как молодая осина,
Твое лицо полная луна.

Мои лепестки, я скажу тебе,
Мои цветы, я скажу тебе;
Цветочек с лепестками,
Так, прерываясь, целую тебя.
```
