# Оригинал

### Татар халык җыры - Матур булсын

```
Син дә җырла, батыр егет,
Син дә җырла, матур кыз!
Матур булсын, матур булсын,
Матур булсын бу тормыш!

Безнең ил кояшы балкый
Якты булыр көн саен;
Якты булсын, якты булсьн,
Якты булсын көн саең!

Илебезнең халыклары
Матур тормыш коралар;
Тыныч булсын, тыныч булсын,
Тыныч булсын дөньялар!
```

# Перевод

### Татарская народная песня - Пусть будет прекрасно

```
И ты пой, смелый парень,
И ты пой, красивая девушка!
Пусть прекрасно, пусть прекрасно,
Пусть прекрасно будет эта жизнь!

Солнце нашей страны сияет,
Каждый день всё ярче;
Пусть светлым, пусть светлым,
Пусть светлым будет твой день!

Народы нашей земли
Прекрасную жизнь построят;
Пусть мирно, пусть мирно,
Пусть мирно будет миры!
```
