# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Казан көе

```
Казаннарга барсаң, дустым,
Казан суларын буйла;
Җилсез көнне тал селкенсә,
Мин сагына дип уйла.

Без урамнардан узабыз
Җырлый-җырлый сыздырып;
Йокың килсә, йокла, җаный,
Уятырмын сызгырып.

Аккошлар оча һавада
Ерактагы далада;
Җаным сөйгән сандугачым
Көтә булыр калада.
```

# Перевод

### Татар халык җыры - Казан көе

```
Если в Казань ты поедешь, мой друг,
По казанским водам пройди;
Если в безветрии ива дрогнет,
Подумай: я скучаю, знай.

Мы по улицам будем идти,
Напевая, свистя в пути;
Если сон навалится - спи, родная,
Я разбужу тебя свистом своим.

Лебеди в небе летят
Над далёкой степью вдали;
Мой соловей, моей души любимый,
В городе ждёт меня.
```
