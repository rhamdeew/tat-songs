# Оригинал

### Татар халык җыры - Асылъяр (беренче вариант)

```
Әйдә барыйк сахраларга, сахраларга,
Гөлләр араларына, Асылъяр.

Матур яр, гүзәлем,
Өзелә лә үзәгем.

Безнең хәзер тиң чагыбыз, тиң чагыбыз
Былбыл балаларына, Асылъяр.

Асылъяр, күгәрчен,
Күрми ничек түзәрмен.

Яшьлек безнең багыбыз, багыбыз,
Гөлләр үпкән чагыбыз, Асылъяр.

Матур яр, кошкаем,
Аерылабыз, дускаем.

Түтәлдәги гөлләр кебек, гөлләр кебек,
Гөрләп үскән чагыбыз, матур яр.

Асылъяр, кошкаем,
Аерылмыйк, дускаем.
```

# Перевод

### Татарская народная песня - Асылъяр (первый вариант)

```
Пойдём на степи, на степи,
Среди цветов, Асылъяр.

Красивая подруга, моя красавица,
Разрываюсь же, разрываюсь моё сердце.

Нынче наш черёд, наш черёд,
Соловьи-дети, Асылъяр.

Асылъяр, голубка,
Как перенесу, если не увижу.

Молодость - наше счастье, наше счастье,
Время, когда цветы целовали, Асылъяр.

Красивая подруга, моя птичка,
Расстаёмся, подруга.

Как цветы на лугу, как цветы,
Время, когда росли цветущие, красивая подруга.

Асылъяр, моя птичка,
Не расстаёмся, подруга.
```
