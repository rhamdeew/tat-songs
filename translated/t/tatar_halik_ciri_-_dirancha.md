# Оригинал

### Татар  халык җыры - Дыранча

```
Көянтәсе үзәкле лә...
Көянтәсе үзәкле лә,
Чиләкләре бизәкле.
Үтеп киткәч борылып карый,
Шунысы өзә үзәкне.

А-а-а-а-й...
Өй түбәсе дыранча ла,
Давыл чыкса җил ача.

Өстәлләрдә матур көзге...
Өстәлләрдә матур көзге
Күзләрне чагылдыра.
Киткән чакта "сау бул" дигән
Сүзләре сагындыра.

А-а-а-а-й...
Өй түбәсе дыранча ла,
Давыл чыкса җил ача.
```

# Перевод

### Татар халык җыры - Дыранча

```
Коромысло узорное...
Коромысло узорное,
Ведра с тонкой резьбой.
Уходя, она оглянет,
И от этого больней.

А-а-а-а-й...
Крыша дома дыранча,
Выйдет буря - ветер рвет.

На столах красивое зеркало...
На столах красивое зеркало
Отражает свет очей.
А слова: «Прощай, не забывай»
Сердце тянет все сильней.

А-а-а-а-й...
Крыша дома дыранча,
Выйдет буря - ветер рвет.
```
