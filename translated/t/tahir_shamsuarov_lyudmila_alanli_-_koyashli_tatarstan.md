# Оригинал

### Таһир Шәмсуаров / Людмила Аланлы - Кояшлы Татарстан

```
Күбәләк кар явар төбәк,
Бураннар бурар урын;
Кышлары елап китсә дә,
Аклыгы калыр урын.

Күкләре гел зәңгәр аның,
Узганы - моңлы дастан.
Гүзәл як минем туган җир -
Кояшлы Татарстан!

Гөрләвекләрне җырлатып
Кояшым мартта көлә.
Яз үзе безнең якларны
Сагынып, эзләп килә.

Күкләре гел зәңгәр аның,
Узганы - моңлы дастан.
Гүзәл як минем туган җир -
Кояшлы Татарстан!
```

# Перевод

### Таһир Шәмсуаров / Людмила Аланлы - Солнечный Татарстан

```
Край, где снег кружится бабочкой,
Где вьюги свой затеют круг;
Пусть зимы плачут, уходя, -
Белизна здесь остаётся вдруг.

Небеса его всегда лазурны,
Прошлое - певучий дастан.
Край прекрасный, мой родимый дом -
Солнечный Татарстан!

Ручьям даря живые песни,
Солнце в марте мне смеётся вновь.
Сама весна к нашим краям
С тоской, в поисках приходит вновь.

Небеса его всегда лазурны,
Прошлое - певучий дастан.
Край прекрасный, мой родимый дом -
Солнечный Татарстан!
```
