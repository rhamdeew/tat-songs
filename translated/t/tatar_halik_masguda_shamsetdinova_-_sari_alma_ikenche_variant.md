# Оригинал

### Tatar halik / Masguda Shamsetdinova - Sary alma (ikenche variant)

```
Сандык кынам тулы сары алма,
Эх, нурым, җанашым, җанашым,
Түбәләмә, тулама.

Ялтыр да гнай ни ялтырый,
Эх, нурым, җанашым, җанашым,
Сөйгәнемнең тәңкәсе.

Алыр да гнай идем шул матурны,
Җанашым, бирми әнкәсе.
А-а...
```

# Перевод

### Татарский народ / Мәсгудә Шәмсетдинова - Жёлтое яблоко (второй вариант)

```
Сундук мой полон жёлтых яблок,
Эх, свет мой, душа моя, душа моя,
Не осыпайся, не вянь.

Сияет, ой как сияет,
Эх, свет мой, душа моя, душа моя,
Монетка моего любимого.

Взял бы я того красавца,
Душа моя, не отдаёт его мать.
А-а...
```
