# Оригинал

### Татар халык җыры - Татар халык җыры - Өфе (беренче вариант)

```
Өфенең ак суларында,
Вак балыклар уемда шул,
Вак балыклар уемда.

Алма кебек матур кызлар,
Үсә Өфе буенда.

Өфе кайда, Өфе кайда,
Өфе Урал ягында шул,
Өфе Урал ягында.

Кошлар булып очар идем,
Бик сагынган чагымда.

Өфеләргә барган чакта,
Кара урманнар үтәм шул,
Кара урманнар үтәм.

Сине сагынам, сине (дә) көтәм
Кирәкми миңа бүтән.
```

# Перевод

### Татар халык җыры - Уфа (первый вариант)

```
В белых водах Уфы,
Мелкой рыбкой мои думы,
Мелкой рыбкой мои думы.

Девы, словно яблоки румяны,
На берегах Уфы растут.

Где Уфа, где Уфа,
Уфа уральской стороны,
Уфа уральской стороны.

Птицей бы взлетел я
В час сильной тоски.

Когда в Уфу иду-еду,
Черные леса прохожу,
Черные леса прохожу.

По тебе тоскую, тебя же жду -
Мне другого не нужно.
```
