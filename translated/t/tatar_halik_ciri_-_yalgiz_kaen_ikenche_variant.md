# Оригинал

### Татар халык җыры - Ялгыз каен (икенче вариант)

```
Авылымның урамыннан
Кемнәрдер гармун суза
Шуның көен көйли-көйли
Су буйларына узам

Яр өстендә ялгыз каен
Җилләр исә чайкала
Су буйларына төшәм мин
Сине сагынган чакларда

Суларда басып тын гына
Суларга басып кына
Парлылар уза янымнан
Юк бәгърем син генә.

Яр өстендә ялгыз каен
Җилләр исә чайкала
Су буйларына төшәм мин
Сине сагынган чакларда
```

# Перевод

### Татар народная песня - Одинокая берёза (второй вариант)

```
По улице деревни моей
Чей-то гармони звонко плывёт,
Под эту мелодию, напевая,
Я к речным берегам иду.

Над обрывом берёза одна,
Ветер листьям качаться велит,
Я спускаюсь к воде к тебе,
Когда тоска по тебе щемит.

По воде стою, замерев,
Только воду ступнёй берегу,
Парочки мимо идут, а рядом
Тебя одного не вижу.

Над обрывом берёза одна,
Ветер листьям качаться велит,
Я спускаюсь к воде к тебе,
Когда тоска по тебе щемит.
```
