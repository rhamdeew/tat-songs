# Оригинал

### Татар халык - Таң алды

```
Таң әтәчләре кычкыра,
Әллә таң ата микән?
Тәрәзә ачып карап калды,
Әллә ярата микән?

Таң алдыннан тал тибрәтеп
Сайрый сары сандугач;
Җырлау түгел, еларсың да
Сөйгәнеңне сагынгач.

Әйдә, җаныем, озат мине
Уйсу җиргә җиткәнче;
Таңнар атар, Чулпан калкыр,
Без аерылып киткәнче.
```

# Перевод

### Татарская народная - Перед рассветом

```
Петухи поют на заре,
Не восходит ли заря?
В окно поглядела -
Не любит ли меня?

Перед рассветом иву трясет,
Поёт соловей жёлтый;
Не песня поётся - ты тоскуешь,
Когда любимого скучаешь.

Пойдём, душа моя, проводи меня,
Пока не дойдём до могилы;
Солнца взойдут, Чулпан всходит,
Пока мы не расстались.
```
