# Оригинал

### Татар халык җыры - Отаз авылы көе

```
Алсу күбәләкләр оча
Зәңгәрсу болытларда;
Әрнемәче, йөрәккәем,
Бер генә минутларга.

Агыйделнең аръягында
Калды бөрлегәннәрем;
Калмады бөрлегәннәрем;
Калды бер дигәннәрем.

Сандугачым, гүзәлем,
Күрми ничек түзәрмен?
Бәлки, күрми дә түзәрмен,
Өзелер инде үзәгем.
```

# Перевод

### Татарская народная песня - Песня родного села

```
Синие бабочки летают
В синеве облаков;
Тоскую, сердечко,
Всего лишь несколько минут.

На груди Агида
Остались поломанные;
Не осталось поломанных;
Остались лишь одни.

Скворечник мой, соколица,
Как я без тебя выдержу?
Может быть, и без тебя выдержу,
Но сердце уже разорвется.
```
