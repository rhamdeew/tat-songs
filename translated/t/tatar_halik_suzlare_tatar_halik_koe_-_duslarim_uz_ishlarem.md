# Оригинал

### Татар халык сүзләре / Татар халык көе - Дусларым - үз ишләрем

```
Идел суы салкын, килә дулкын,
Ак үрдәкләр каршы йөзәлмый.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.

Сез, дусларым, миңа бигрәк якын,
Бер җырламый күңелем түзалмый.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.

Мин җырламыйм, дуслар, җырлый белеп,
Җырлыйм сезне якын күргәнгә.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.
```

# Перевод

### Татарские народные слова / Татарская народная мелодия - Друзья мои - дела мои

```
Волга-река студёна, волны идут,
Белые утки против течения не плывут.
Эх, друзья мои - дела мои,
Без вас, друзей, что мне делать.

Вы, друзья мои, мне особенно близки,
Без песни душа моя не выдержит.
Эх, друзья мои - дела мои,
Без вас, друзей, что мне делать.

Я пою не потому, друзья, что петь умею,
Пою оттого, что вас вижу рядом.
Эх, друзья мои - дела мои,
Без вас, друзей, что мне делать.
```
