# Оригинал

### Татар халык җыры - Өзелә бит үзәгем

```
Сандугачым, әйт үземә,
Кайсы талга кунасың?
Күзем йомсам, күз алдымда
Торган төсе буласың.

Сандугачым, гүзәлем лә,
Тибрәтмә тал үзәген.
Исләремә төшермәгез
Өзелә минем үзәгем.
```

# Перевод

### Татарская народная песня - Разрывается сердце моё

```
Соловей мой, скажи мне скорее,
На какую же иву ты сядешь?
Лишь закрою я очи - и снова
Твой прекрасный мне образ приснится.

Соловей мой, красавица милая,
Не качай ты ивовых веток.
Не давайте мне вспоминать, люди,
Разрывается сердце от боли.
```
