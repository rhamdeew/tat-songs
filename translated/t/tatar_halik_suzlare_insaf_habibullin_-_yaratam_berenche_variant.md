# Оригинал

### Татар халык сүзләре / Инсаф Хәбибуллин - Яратам (беренче вариант)

```
Синең кара кашларың,
Синең кара күзләрең,
Синең ягымлы сүзләрең...
Яна синең йөзләрең.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.

Сандугачның күрсәң иде
Юлга тезелгәннәрен.
Һичкем белми йөрәгемнең
Нигә өзелгәннәрен.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.

Алма бакчасына керәм,
Үргән чәчемне сүтәм.
Сабыр итәм, сине көтәм,
Кирәкми миңа бүтән.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.
```

# Перевод

### Татар халык сүзләре / Инсаф Хәбибуллин - Яратам (первый вариант)

```
Твои черные брови,
Твои черные глаза,
Твои ласковые слова...
Пылают твои щеки.

Я люблю тебя,
Люблю, люблю.
Я тебя, все равно,
К себе поверну.

Если б ты увидела соловьев,
Вдоль дороги выстроенных.
Никто не знает у сердца моего,
Отчего надорвано оно.

Я люблю тебя,
Люблю, люблю.
Я тебя, все равно,
К себе поверну.

Вхожу я в яблоневый сад,
Расплетаю заплетенные волосы.
Терплю, тебя жду,
Мне больше ничего не нужно.

Я люблю тебя,
Люблю, люблю.
Я тебя, все равно,
К себе поверну.
```
