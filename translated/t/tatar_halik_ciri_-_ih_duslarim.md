# Оригинал

### tatar_halik_ciri_ - Татар халык җыры - Их, дусларым

```
Их, дусларым, дусларым,
Дусларым, дус-ишләрем;
Сез дуслардан аерылгач,
Ялгыз башым нишләрмен?

Уйсу җирдә дым була,
Ком бар җирдә су була;
Яшь чакларда уйна да көл
Картайгачтын соң була.

Их, дусларым, дусларым,
Дусларым, бәгырьләрем;
Яшь вакытлар ике килми
Белегез кадерләрен.

Их, дусларым, дусларым,
Дусларым, дус-ишләрем;
Сез дуслардан аерылгач,
Ялгыз башым нишләрмен?
```

# Перевод

### Татар халык җыры - Эх, друзья мои

```
Эх, друзья мои, друзья,
Друзья мои, товарищи;
Как мне жить, когда разлука
Разведет меня с друзьями?

В низине - сырость стелется,
Где песок - там воду встретишь;
В молодые годы смейся,
На старость уж не успеешь.

Эх, друзья мои, друзья,
Дорогие мне сердца;
Молодость не повторится -
Знайте цену ей всегда.

Эх, друзья мои, друзья,
Друзья мои, товарищи;
Как мне жить, когда разлука
Разведет меня с друзьями?
```
