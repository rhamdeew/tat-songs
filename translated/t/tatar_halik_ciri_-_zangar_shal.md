# Оригинал

### Татар халык җыры - Зәңгәр шәл

```
Карлыгачкай кара, муены ала,
Очып кына киткәч, абау, бәгырем, югала.

Башың яланбаш икән,
Бөркән зәңгәр шәлеңне;
Чит җирләрдә бик күп йөрдем,
Белә алмадым хәлеңне.
Ал яулык, зәңгәр шәл,
Үткән гомер бигрәк жәл.

Атадан бала яшьләй калса,
Күкрәк көчләре белән, абау, бәгырем, мал таба

Башың яланбаш икән,
Бөркән зәңгәр шәлеңне.
Чит җирләрдә бик күп йөрдем,
Белә алмадым хәлеңне.
Ал яулык, зәңгәр шәл,
Үткән гомер бигрәк жәл.
```

# Перевод

### Татарская народная песня - Голубой платок

```
Ласточка-малышка, шейка чёрная,
Улетев, исчезла, ох, голубка моя, пропала.

Если голова у тебя непокрыта,
Накинь свой голубой платок;
По чужим краям много ходил я,
Не мог узнать твою судьбу.
Тот платок, голубой шаль,
Прошлая жизнь - досаднее жал.

Если отцом сын останется молод,
Силой груди, ох, голубка моя, добудет скот

Если голова у тебя непокрыта,
Накинь свой голубой платок.
По чужим краям много ходил я,
Не мог узнать твою судьбу.
Тот платок, голубой шаль,
Прошлая жизнь - досаднее жал.
```
