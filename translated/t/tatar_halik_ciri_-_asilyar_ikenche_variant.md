# Оригинал

### Татар  халык җыры - Асылъяр (икенче вариант)

```
Әйдә киттек (барыйкчы) сахраларга, сахраларга,
Гөлләр араларына, Асылъяр.
Матур яр, гүзәлем,
Өзелә лә үзәгем.

Безнең хәзер тиң чагыбыз, тиң чагыбыз
Былбыл балаларына, Асылъяр.
Матур яр, гүзәлем,
Күрми ничек түзәрмен.

Аклы ситса күлмәгеңне, күлмәгеңне
Агыйделдәй озынмы Асылъяр?
Матур яр, аппагым
Ни теләгең бар тагын?

Кашың кара, йөзләрең ак, йөзләрең ак,
Айлы кичтә тудыңмы, Асылъяр?
Матур яр, кошкаем
Сагынам сине көн саен.
Асылъяр, кошкаем
Сагынам сине көн саен.
```

# Перевод

### Татарская народная песня - Асылъяр (второй вариант)

```
Пойдём-ка мы (пойдёмте) в степи, в степи,
Среди цветов, Асылъяр.
Красавица, моё сокровище,
Рвётся и разрывается сердце моё.

Нынче наше время, наш час,
Как у соловьих, Асылъяр.
Красавица, моё сокровище,
Как выдержу, не видя тебя.

Если рубашку белую, рубашку
Длинную, как Агидель, Асылъяр?
Красавица, мой светло-белый,
Чего ещё ты желаешь?

Брови твои черны, лицо твоё бело, лицо твоё бело,
Ты в лунную ночь родилась, Асылъяр?
Красавица, моя пташка,
Скучаю по тебе каждый день.
Асылъяр, моя пташка,
Скучаю по тебе каждый день.
```
