# Оригинал

### Алмаз Хәмзин / Оскар Усманов - Әбием коймаклары

```
Сагынабыз балачакны,
Җен-пәрине һәр төннең.
Урамнарда ялан аяк
Тәпиләгән җәй көнен.
Үсеп җиткәч хәтирәләр
Күз алдыннан үтәләр,
Балачакның һәр мизгеле
Туктап безне көтәләр.

Өзелеп-өзелеп сагындыра
Әбием коймаклары,
Мичтә кызган табасының
Кулында уйнаклавы.

Сагындыра әниемнең
Ачуланган чаклары,
Эшкә-тәртипкә өйрәткән
Әтием чыбыклары.
Хәзер рәхмәт кенә әйтәм
Әтием-әниемә.
Аларны миңа тудырган
Кадерле әбиләргә.

Өзелеп-өзелеп сагындыра
Әбием коймаклары,
Мичтә кызган табасының
Кулында уйнаклавы.
```

# Перевод

### Алмаз Хамзин / Оскар Усманов - Бабушкины коймаки

```
Скучаем мы по детству,
По душе каждой ночи той.
По улицам босыми
Бегали в денёк летной.
Когда взрослеем, память
Проходит пред глазами,
Каждый миг из детства
Останавливается с нами.

До слёз, до слёз тоскую я
По бабушкиным коймакам,
Как сковородку раскалённую
В руках она держала.

Скучаю я по маминой
Сердитости порой,
По отцовским прутикам,
Что к делу приучали.
Теперь лишь благодарность
Отцу-матери своей.
И дорогим бабулям,
Что их мне подарили.

До слёз, до слёз тоскую я
По бабушкиным коймакам,
Как сковородку раскалённую
В руках она держала.
```
