# Оригинал

### Билгесез - Бик сагындым туган якларымны

```
Бик сагындым газиз әнкәемне,
"Улым" диеп әйткән сүзләрен.
Сагындым мин якын дусларымның,
Якташларның көләч йөзләрен.
Сагындым мин якын дусларымның,
Якташларның көләч йөзләрен.

Әй кадерле ямьле Агыйделем,
Әй сөекле туып-үскән җир.
Истән чыкмый миңа бүләк итеп
Әткәм-әнкәм моңнар биргән җир.
Истән чыкмый миңа бүләк итеп
Әткәм-әнкәм моңнар биргән җир.

Айлар, еллар, көннәр бик тиз үтә,
Язлар җитә, җәйләр яшәрә.
Ниләр җитә туган җирләреңдә
Якын дуслар белән яшәүгә?
Ниләр җитә туган җирләреңдә
Туганнарың белән яшәүгә?
Әйтегезче, дуслар, ни җитә?
```

# Перевод

### Билгесез - Очень скучаю по родным местам

```
Очень скучаю по дорогой маме,
По словам, которые она говорила: "Сынок мой".
Скучаю я по близким друзьям,
По улыбающимся лицам земляков.
Скучаю я по близким друзьям,
По улыбающимся лицам земляков.

Ах, драгоценная, милая Агыйдель,
Ах, любимая земля, где вырос и окреп.
Не выходит из памяти как подарок
Та земля, где отец-мать песни мне дарил.
Не выходит из памяти как подарок
Та земля, где отец-мать песни мне дарил.

Месяцы, годы, дни быстро проходят,
Приходят зимы, весны цветут.
Чего не хватает на родной земле,
Жить с близкими друзьями?
Чего не хватает на родной земле,
Жить со своими родными?
Скажите, друзья, чего не хватает?
```
