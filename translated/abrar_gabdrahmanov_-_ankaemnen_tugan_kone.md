# Оригинал

### Абрар Габдрахманов - Әнкәемнең туган көне

```
Нинди матур, нинди күркәм,
Бу нинди көн икән?
Аяз көнне, чуак көнне
Әнкәем туган икән.

Котлыймын сине,
Котлыймын сине, әнкәй!
Туган көнең белән
Котлыймын сине, әнкәй!

Бакчадагы сандугачлар
Кемне мактый икән?
Сандугачлар, чәчәкләр дә
Әнкәйне котлий икән.

Котлыймын сине,
Котлыймын сине, әнкәй!
Туган көнең белән
Котлыймын сине, әнкәй!

Кояш көлә, ләйсән сибә (балкып көлә) —
Бу нинди көн икән?
Аяз көндә ләйсән (кояш) булып
Әнкәем туган икән.

Котлыймын сине,
Котлыймын сине, әнкәй!
Туган көнең белән
Котлыймын сине, әнкәй!
```

# Перевод

### Абрар Габдрахманов - День рождения мамы

```
Как прекрасен, как чудесен,
Что за день сегодня?
В ясный день, в светлый день
Моя мама родилась.

Поздравляю тебя,
Поздравляю тебя, мама!
С днём рождения
Поздравляю тебя, мама!

Соловьи в саду
Кого восхваляют?
Соловьи и цветы
Маму поздравляют.

Поздравляю тебя,
Поздравляю тебя, мама!
С днём рождения
Поздравляю тебя, мама!

Солнце улыбается, дождик льётся (светло улыбается) —
Что за день сегодня?
В ясный день дождём (солнцем)
Моя мама родилась.

Поздравляю тебя,
Поздравляю тебя, мама!
С днём рождения
Поздравляю тебя, мама!
```
