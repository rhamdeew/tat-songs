# Оригинал

### Алия Исрафилова - Юанычым

```
Чишмэм сукмагыннан атлыйм,
Белэсенме, ул мина хаман якын.
Бергэлэп йоргэннэр хэтергэ килэлэр,
Исеменне кабатлыйм.

Чишмэм сулары гел тонык,
Улагы каршында утырам тынып.
Челтерэп су ага, ишетелэ ул мина
Синен тавышын булып.

Чишмэм куанычым минем,
Якын сердэшем, юанычым минем.
Сагышым тарала, хислэрем янара,
Узен хэзер кайларда?
```

# Перевод

### Алия Исрафилова - Мое утешение

```
По тропинке к роднику иду,
Знаешь ли, он мне все еще близок.
Вместе ходившие в память приходят,
Имя твое повторяю.

Родника воды всегда чисты,
У него сижу, отдыхая.
Журча течет вода, слышится мне
Твоим голосом.

Родник - радость моя,
Близкий друг сердца, утешение мое.
Тоска рассеивается, чувства вспыхивают,
Где же ты теперь?
```
