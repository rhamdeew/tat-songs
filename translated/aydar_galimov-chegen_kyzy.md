# Оригинал

### Айдар Галимов - Чегэн кызы

```
Кулларымны кулларына салып
Чегэн кызы язмыш юрыйсын.
"Котмэ", — дисен, — омет итмэ, — дисен,
"Йорэгеннэн аны жуй!" — дисен.
"Котмэ", — дисен, — омет итмэ, — дисен,
"Йорэгеннэн аны жуй!" — дисен.

Чегэн кызы,
Тоз салма ярама,
Утэли карама,
Яшэу юк янадан
Аннан башка.
Башны бэреп булмый ташка.
Ташлар булсан — эйдэ ташла.
Тик бэхетле булырсынмы
Миннэн башка?

Чегэн кызы,
Хэтта кэккуклэр дэ
Мэхэббэткэ омет жуйганнар.
Минем союемэ
Минсез ясин чыгып
Миннэн башка жирлэп куйганнар.

Чегэн кызы,
Тоз салма ярама,
Утэли карама,
Яшэу юк янадан
Аннан башка.
Башны бэреп булмый ташка.
Ташлар булсан — эйдэ ташла.
Тик бэхетле булырсынмы
Миннэн башка?

Ачуланма, кызый, сузен ялган.
Кэккуклэр дэ санап бетмэгэн.
Союлэр дэ безнен эле сурелмэгэн,
Анлашырга вакыт утмэгэн.

Чегэн кызы,
Тоз салма ярама,
Утэли карама,
Яшэу юк янадан
Аннан башка.
Башны бэреп булмый ташка.
Ташлар булсан — эйдэ ташла.
Тик бэхетле булырсынмы
Миннэн башка?
```

# Перевод

### Айдар Галимов - Цыганочка

```
Руки мои в её руки положив,
Пусть цыганочка судьбу погадает.
"Не жди", — скажешь, — "не надейся", — скажешь,
"Из сердца её сотри!" — скажешь.
"Не жди", — скажешь, — "не надейся", — скажешь,
"Из сердца её сотри!" — скажешь.

Цыганочка,
Не сыпь соль на рану,
Не смотри с укором,
Нет жизни заново
Без неё.
Нельзя головой о камень биться.
Если камни — давай бросай.
Только будешь ли счастлива
Без меня?

Цыганочка,
Даже кукушки
К любви надежду потеряли.
Мою любимую
Без меня ясно выйдя,
Кроме меня другому отдали.

Цыганочка,
Не сыпь соль на рану,
Не смотри с укором,
Нет жизни заново
Без неё.
Нельзя головой о камень биться.
Если камни — давай бросай.
Только будешь ли счастлива
Без меня?

Не сердись, дорогая, слово ложно.
Кукушки тоже не досчитались.
Любовь наша ещё не отзвучала,
Время понять друг друга не прошло.

Цыганочка,
Не сыпь соль на рану,
Не смотри с укором,
Нет жизни заново
Без неё.
Нельзя головой о камень биться.
Если камни — давай бросай.
Только будешь ли счастлива
Без меня?
```
