# Оригинал

### Билгесез / Зөфәр Хәйретдинов - Торналар (өченче вариант)

```
Торналар оча көньякка,
Тасма булып төзелеп.
Күк йөзеннән сибелә кебек,
Сагышлы моң төзелеп.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.

Сокланам да, көнләшәм дә
Сезгә карап торналар.
Бәхетлеләр сезнең кебек
Һәрчак бергәдер алар.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.

Торналар биек очалар,
Кайталмаслар тиз генә.
Түзәр алар күрешмичә
Түзә алмам шул мин генә.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.
```

# Перевод

### Неизвестен / Зуфар Хайретдинов - Журавли (третий вариант)

```
Журавли летят на юг,
Выстроившись лентой.
Словно с неба льётся
Грустная мелодия.

Не грустите, птицы мои,
Вы ведь вместе, одной стаей.
Осиротевшую душу, одинокую душу
Никто не утешит.

Любуюсь и завидую
Вам, глядя, журавли.
Счастливы, как вы,
Те, кто всегда вместе.

Не грустите, птицы мои,
Вы ведь вместе, одной стаей.
Осиротевшую душу, одинокую душу
Никто не утешит.

Журавли высоко летят,
Не вернутся скоро.
Они вынесут разлуку,
А я один не выдержу.

Не грустите, птицы мои,
Вы ведь вместе, одной стаей.
Осиротевшую душу, одинокую душу
Никто не утешит.
```
