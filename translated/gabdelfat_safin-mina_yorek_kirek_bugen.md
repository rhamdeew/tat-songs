# Оригинал

### Габделфат Сафин - Мина йорэк кирэк буген

```
Йорэгемне учым белэн
Кысып тоткан чакларда,
Кинэш бирэсен — йорэкне
Килэчэккэ сакларга.

Сузлэрен хактыр, кадерлем,
Саклану кирэктер ул,
Сакланган йорэк, элбэттэ,
Ин эйбэт йорэктер ул.

Тореп кенэ йортер идем —
Саклап кына булмый ла!
Саклаган да булыр идем,
Буген кирэк ул мина!

Сузлэрен хактыр, кадерлем,
Саклану кирэктер ул,
Сакланган йорэк, элбэттэ,
Ин эйбэт йорэктер ул.

Янсын йорэк! Янсын дорлэп!
Тамсын, эйдэ, каннары!
Мина йорэк буген кирэк —
Килэчэккэ аннары...

Сузлэрен хактыр, кадерлем,
Саклану кирэктер ул,
Сакланган йорэк, элбэттэ,
Ин эйбэт йорэктер ул.
```

# Перевод

### Габделфат Сафин - Мне нужно сердце сегодня

```
Когда сердце мое в кулак
Сжимаю я,
Советуешь - сердце
Для будущего беречь.

Слова твои правдивы, дорогая,
Беречь его нужно,
Сбереженное сердце, конечно,
Самое лучшее сердце.

Жил бы себе спокойно -
Только беречь нельзя!
Сберег бы и его,
Сегодня нужно оно мне!

Слова твои правдивы, дорогая,
Беречь его нужно,
Сбереженное сердце, конечно,
Самое лучшее сердце.

Пусть горит сердце! Пылает ярко!
Пусть капает кровь его!
Мне сердце сегодня нужно -
А для будущего потом...

Слова твои правдивы, дорогая,
Беречь его нужно,
Сбереженное сердце, конечно,
Самое лучшее сердце.
```
