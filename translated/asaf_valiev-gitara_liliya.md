# Оригинал

### Асаф Валиев - Гитара + Лилия

```
Бодрэ чэчен тараладыр
Таратып караганым юк
Нинди була ул мэхэббэт
Яратып караганым юк

Аклы кулмэк кигэн идем
Итэклэре кин икэн
Сандугач сандугачка тин
Э мин кемгэ тин икэн

Эй гитара, эй гитара
Чынлап тора кыллары
Гитара уйный бер егет
Бар микэн сойгэн яры

Ярым юк, парымюк дидем
Сине таптым, яр таптым
Бер курудэ гашыйк булдым
Бер курудэ яраттым

Аклы кулмэк кигэнсен дэ
Итэклэре кин икэн
Сандугач сандугачка тин
Э син мина тин микэн.

Эй гитара, эй гитара
Чынлап тора кыллары
Гитара уйный бер егет
Бар микэн сойгэн яры
```

# Перевод

### Асаф Валиев - Гитара + Лилия

```
Кудрявые волосы расчесываются,
Расчесывать не пробовал,
Какова будет та любовь,
Любить не пробовал.

Белое платье надела,
Подол его широк,
Соловей соловью подобен,
А я кому подобен?

Эй гитара, эй гитара,
Звенят натянуто струны,
На гитаре играет парень,
Есть ли у него любимая?

Милого нет, пары нет - говорил,
Тебя нашел, милую нашел,
В один миг влюбился,
В один миг полюбил.

Белое платье надела,
Подол его широк,
Соловей соловью подобен,
А ты мне подобна ли?

Эй гитара, эй гитара,
Звенят натянуто струны,
На гитаре играет парень,
Есть ли у него любимая?
```
