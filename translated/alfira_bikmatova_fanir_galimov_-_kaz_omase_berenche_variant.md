# Оригинал

### Альфира Бикмәтова / Фәнир Галимов - Каз өмәсе (беренче вариант)

```
Каз өмәсе су буена
Озаталар казларны.
Ак йолдызлар бии бии
Иркәлиләр кызларны.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.

Чишмә суы көмеш кебек,
Кызлар казлар юалар.
Мамык кебек карны яра,
Болыт яллы юргалар.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.

Шәмдәй шома ак каеннар
Чишмә юл буен бизи.
Кояш кызы уянган да,
Елмая, кулын изи.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.
```

# Перевод

### Альфира Бикматова / Фанир Галимов - Гусиная помочь (первый вариант)

```
У воды гусиная помочь,
Провожают гусей в дальний путь.
Белые звёзды в танце кружат,
Девушек ласково балуют.

В этот час родные края
Светлы даже в ночи глухой.
Гусиная помочь — большой праздник:
Гости приходят толпой.

Вода родника, как серебро,
Девушки гусей омывают.
Живот белый, как хлопок нежный,
Облака по небу плывают.

В этот час родные края
Светлы даже в ночи глухой.
Гусиная помочь — большой праздник:
Гости приходят толпой.

Словно свечи, белые берёзы
У дороги к роднику стоят.
Дочь солнца проснулась рано,
Улыбается, руку протянув назад.

В этот час родные края
Светлы даже в ночи глухой.
Гусиная помочь — большой праздник:
Гости приходят толпой.
```
