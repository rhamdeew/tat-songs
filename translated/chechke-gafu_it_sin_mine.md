# Оригинал

### Чэчкэ - Гафу ит син мине

```
Гафу ит син мине,яшьлегемнен голе,
Ул чагында кылган ялгышка.
Юкка конлэшуне,синнэн читлэшуне
Анладым шул бик сон тормышта.

Гомерлэр тиз узар,кичер мине зинхар
Гонахлы мин синен каршында.
Тугры сузлэрен дэ,монсу кузлэрен дэ
Тошлэремэ кереп алжытма.

Гафу ит син мине,сунгэн ометенне
Хыялынны торгэн томанга.
Назлап кунеленне,тэуге союенне
Салган очен бердэн моннарга.

Гомерлэр тиз узар,кичер мине зинхар
Гонахлы мин синен каршында.
Тугры сузлэрен дэ,монсу кузлэрен дэ
Тошлэремэ кереп алжытма.

Гафу ит син мине,инде ятлар голе
Бэхетле бул алда тормышта.
Уйларында кайчак яшьлегенэ кайтсам
Безнен союлэрне онытма

Гомерлэр тиз узар,кичер мине зинхар
Гонахлы мин синен каршында.
Тугры сузлэрен дэ,монсу кузлэрен дэ
Тошлэремэ кереп алжытма.
```
