# Оригинал

### Билгесез / Риф Арслан - Ай утыра елмаеп

```
Күктә йолдызлар нурлардан
Чигү чигәләр каеп.
Көйәнтәле кызны кочып
Ай утыра елмаеп.

Сандугачлар моң сибүдән
Бар дөнъя калган баеп.
Моңдан гашыйклар исергән
Мин генә йөримен аек.

Сандугачларның моңнары
Чайпылалар түгелә.
Минем дә гашыйк чакларым
Һаман яши күңелдә.

Сандугачлар моң сибүдән
Бар дөнъя калган баеп.
Моңдан гашыйклар исергән
Мин генә йөримен аек.
```

# Перевод

### Неизвестен / Риф Арслан - Луна сидит, улыбаясь

```
В небе звёзды светом льются,
Цветут цветы, качаясь.
Обнимая девушку в тени,
Луна сидит, улыбаясь.

Соловьи рассыпали печаль -
Весь мир замер, восхищаясь.
От этих песен влюблённые пьяны,
Лишь я брожу, шатаясь.

Соловьиные напевы
Разливаются повсюду.
И мои влюблённые мгновенья
Всё живут в сердце, не уйду.

Соловьи рассыпали печаль -
Весь мир замер, восхищаясь.
От этих песен влюблённые пьяны,
Лишь я брожу, шатаясь.
```
