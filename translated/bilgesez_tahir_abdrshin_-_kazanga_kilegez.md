# Оригинал

### Билгесез / Таһир Абдршин - Казанга килегез!

```
Сезнең белән күрешергә
Килдем мин ераклардан.
Җырлар белән, моңнар белән
Кайнар сәламнәр белән.

Килегез, килегез
Казаннарны күрегез!
Килегез, дусларым
Безгә кунакка!

Идел буйларыннан мин килдем.
Кама суларын кичтем.
Шул якларда туып үстем,
Шул якларның суын эчтем.

Килегез, килегез
Казаннарны күрегез!
Килегез, килегез
Безгә кунакка!

Хушлашырга вакыт җитте,
Дусларым, кардәшләрем.
Бергә булган минутларны
Онытмам мин һәркөн саен.

Килегез, йөрегез
Тукай йөрегән әзләрдән!
Килегез, дусларым
Безгә кунакка!
Килегез, дусларым
Безгә Казанга!
```

# Перевод

### Автор неизвестен / Таһир Абдршин - Приезжайте в Казань!

```
Чтобы с вами повстречаться
Я приехал издалека.
С песнями, с мелодиями,
С горячим сердечным приветом.

Приезжайте, приезжайте,
Посмотрите на Казань!
Приезжайте, друзья мои,
К нам в гости!

С берегов Волги я приехал,
Воды Камы я пересёк.
В тех краях я родился и рос,
Воду тех краёв я пил.

Приезжайте, приезжайте,
Посмотрите на Казань!
Приезжайте, приезжайте,
К нам в гости!

Пришла пора прощаться,
Друзья мои, братья мои.
Минуты, проведённые вместе,
Не забуду я каждый день.

Приезжайте, приходите
По следам, где ходил Тукай!
Приезжайте, друзья мои,
К нам в гости!
Приезжайте, друзья мои,
К нам в Казань!
```
