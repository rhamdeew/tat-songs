# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Булмас инде, булмас

```
Зарыгыплар көтәм, ни диярсең икән,
Әйтәм, дисең, җавабыңны иртәгә.
Ышанасы килә, бул үземә генә,
Карый күрмә, җаныкаем, бүтәнгә.

Булмас инде, булмас, гомергә дә булмас
Күңелемнең сине сөеп туясы.
Яшиселәр килә синең өчен генә,
Синнән башка нигә соң ул дөньясы.

Узмый бер дә вакыт, йөрәк яна ярсып,
Мөлдерәмә тулып ташый хисләрем.
Ышанасым килә, юк иде лә бер дә
Иртәгә без шулай өзелеп көткәнем.

Булмас инде, булмас, гомергә дә булмас
Күңелемнең сине сөеп туясы.
Яшиселәр килә синең өчен генә,
Синнән башка нигә соң ул дөньясы.

Яшәүләргә җирдә, икебезгә бергә,
Ни җитә соң, бәгырькәем, ни җитә.
Эх, ялгыша күрмә, ышанасы килә,
Синең җавабың илтер дип бәхеткә.

Булмас инде, булмас, гомергә дә булмас
Күңелемнең сине сөеп туясы.
Яшиселәр килә синең өчен генә,
Синнән башка нигә соң ул дөньясы.
```

# Перевод

### Айдар Минхаджев / Айдар Тимербаев - Не будет уже, не будет

```
Изнывая жду, что же ты скажешь,
Говоришь, дашь свой ответ завтра.
Хочется верить, будь только моей,
Не смотри, душенька моя, на других.

Не будет уже, не будет, никогда не будет,
Чтобы сердце моё насытилось любовью к тебе.
Жить хочется только ради тебя,
Кроме тебя зачем же мне этот мир.

Не проходит время, сердце горит пылая,
До краёв наполнены бьются мои чувства.
Хочется верить, не было же вовсе
Завтра, которого мы так истово ждали.

Не будет уже, не будет, никогда не будет,
Чтобы сердце моё насытилось любовью к тебе.
Жить хочется только ради тебя,
Кроме тебя зачем же мне этот мир.

Жить на земле, нам двоим вместе,
Чего же хватит, сердечко моё, чего хватит.
Эх, не ошибись только, хочется верить,
Что твой ответ приведёт к счастью.

Не будет уже, не будет, никогда не будет,
Чтобы сердце моё насытилось любовью к тебе.
Жить хочется только ради тебя,
Кроме тебя зачем же мне этот мир.
```
