# Оригинал

### Малика - Яна ел

```
Кунелемдэ  -  яна уйлар,
Биеп йори хыял.
Алла бирсэ, барысы да
Чынга ашар быел...

Яна елны, яна конне,
Котеп алдык, котеп.
Яна конгэ керик дуслар,
Бисмилланы эйтеп...

Алла бирсэ, алла бирсэ,
Чынга ашар хыял.
Уткэн елдан бэхетлерэк
Булырбыз без быел...

Кунелемдэ - якты уйлар,
Кузлэремдэ - нурлар.
Бер – берсеннэн аермасын
Иркэм, безнен куллар.
```

# Перевод

### Малика - Новый год

```
В душе моей - новые мысли,
Танцует в сердце мечта.
Даст Аллах - всё сбудется,
Воплотится в этом году...

Новый год, новый день
Мы встречаем с надеждой.
Для нового дня нужны друзья,
Произнося "Бисмиллах"...

Даст Аллах, даст Аллах -
Воплотится мечта.
Счастливее прошлого года
Будем мы в этом году...

В душе моей - светлые мысли,
В глазах моих - свет.
Пусть не разлучает друг с другом
Бог наши руки.
```
