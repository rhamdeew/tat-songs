# Оригинал

### Харис Нигъмәтҗан / Рәсимә Денисламова - Казансу дулкыннары

```
Казансуның йөгерек дулкыннары
Хәтерлиләр безне бүген дә
Яр буенда вәгьдәләшкән идек
Аерылмабыз диеп гомергә

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар,
Очрашуга тагын килдем бүген,
Сине көтеп таңнар аттылар.

Казансуның йөгерек дулкыннары
Сагыш өстәп ярга кагалар
Сине генә күрү теләге
Янган утка мине салалар.

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар,
Очрашуга тагын килдем бүген
Сине көтеп таңнар аттылар

Казансуга карап уйга баттым
Сулар белми минем хәлемне
Күпме еллар инде үтеп киткән
Тик нәрсәдер өти йөрәкне.

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар
Очрашуга тагын килдем менә
Сине көтеп таңнар аттылар.
```

# Перевод

### Харис Нигматжан / Рәсимә Денисламова - Волны Казанки

```
Быстрые волны Казанки
Вспоминают нас и сегодня,
На берегу клятву давали
Что на всю жизнь не расстанемся.

Обеты ветра унесли,
Мечты мои в воду утекли,
Сегодня снова пришёл на встречу,
Тебя ожидая, рассветы встречал.

Быстрые волны Казанки
Добавляя тоску ударяются о берег,
Желание видеть только тебя
Бросает меня в пламя огня.

Обеты ветра унесли,
Мечты мои в воду утекли,
Сегодня снова пришёл на встречу,
Тебя ожидая, рассветы встречал.

Смотрю на Казанку, погрузился в думы,
Воды не знают моего состояния,
Сколько лет уже пролетело,
Но что-то всё равно мучает сердце.

Обеты ветра унесли,
Мечты мои в воду утекли,
Вот снова пришёл на встречу,
Тебя ожидая, рассветы встречал.
```
