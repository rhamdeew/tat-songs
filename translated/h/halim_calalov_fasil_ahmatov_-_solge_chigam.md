# Оригинал

### Хәлим Җәләлов / Фасил Әхмәтов - Сөлге чигәм

```
Сөлге чигәм асыл җепләр белән.
Йөрәк хисен салып бу җырга.
Насыйп булса иде бу бүләгем
Мәйданнарда җиңгән батырга.

Ефәк җепләр белән сөлге чиктем,
Уртасында чәчәк-гөл генә.
Җиңеп алсаң, егет, бу бүләкне,
Вәгъдә бирәм сиңа мәңгегә.
```

# Перевод

### Халим Джелялов / Фасил Ахметов - Тонкую ниточками

```
Тонкую ниточками ткал.
Сердце чувства вложу в эту песню.
Если бы достался этот дар
Победителю в битвах на поле.

Тонкую материяю ткала,
В центре - цветочек.
Если выиграешь, юноша, этот дар,
Обещаю тебе вечную любовь.
```
