# Оригинал

### Хәмидулла Гыйбадуллин / Виталий Агапов - Мәндәемә кайтам

```
Әнкәемнең чулпы тавышлары,
Әткәемнең чалгы янавы
Колакларда, йөрәкләрдә чыңлый,
Онытып булмый, булмый аларны.

Мәндәемә кайтам, авылдашлар,
Сезне сагынып, сезне яратып.
Ак теләкләр генә теләп кайтам,
Шикләремне җилгә таратып.

Адашканда, юлдан тайпылганда,
Барыр юнәлешем югалса,
Түмгәкләргә килеп сөртенгәндә,
Дуслар дошман булып тоелса.

Мәндәемә кайтам, авылдашлар,
Сезне сагынып, сезне яратып.
Ак теләкләр генә теләп кайтам,
Шикләремне җилгә таратып.

Кайтам монда иман яңартырга,
Чишмәләрдә битем юарга.
Шыткан тамырымны эзләп кайтам,
Кайтам монда кабат туарга.

Мәндәемә кайтам, авылдашлар,
Сезне сагынып, сезне яратып.
Ак теләкләр генә теләп кайтам,
Шикләремне җилгә таратып.
```

# Перевод

### Хәмидулла Гыйбадуллин / Виталий Агапов - Возвращаюсь в Мәндәй

```
Звон материнских чулпы-украшений,
Звяк отцовской косы
В ушах, в сердцах звенит и не смолкает,
Не забыть, не забыть их.

В Мәндай возвращаюсь, земляки,
Вас тоскуя, вас любя.
С добрыми желаньями лишь возвращаюсь,
Сомненья по ветру развеяв.

Когда блуждаю, с пути сворачивая,
Если теряю направление,
Если о кочки спотыкаюсь,
Если друзья врагами кажутся.

В Мәндай возвращаюсь, земляки,
Вас тоскуя, вас любя.
С добрыми желаньями лишь возвращаюсь,
Сомненья по ветру развеяв.

Возвращаюсь веру обновить,
В родниках умыть лицо.
Возвращаюсь корни прорастающие отыскать,
Возвращаюсь вновь родиться.

В Мәндай возвращаюсь, земляки,
Вас тоскуя, вас любя.
С добрыми желаньями лишь возвращаюсь,
Сомненья по ветру развеяв.
```
