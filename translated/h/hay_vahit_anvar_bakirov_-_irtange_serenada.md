# Оригинал

### Хәй Вахит / Әнвәр Бакиров - Иртәнге серенада

```
Күмелгәнсең ак мамык мендәргә,
Толымнарың җәеп син, иркәм.
Сине өзелеп сагынып,
Сөю белән янып,
Тәрәзәңне чиртәм мин иртән.

Вакыт инде... Уян, иркәм, уян,
Керфегеңне күтәр, күзең ач!
Нинди гүзәл ал таң,
Нурлар сибә чулпан,
Тал-тирәктә сайрый сандугач.

Шатлык көен сузып, бар табигать
Көтә сине - гөлләр иркәсен.
Уян, иркәм, уян,
Тан нуры белән ян,
Каршы алыйк без эш иртән!

Вакыт инде... Уян, иркәм, уян,
Керфегеңне күтәр, күзең ач!
Нинди гүзәл ал таң,
Нурлар сибә чулпан,
Тал-тирәктә сайрый сандугач.
```

# Перевод

### Хай Вахит / Анвар Бакиров - Утренняя серенада

```
Если ты уснешь среди белых облак,
Ты устал, отдыхай, моя радость.
Я тоскую по тебе, изнывая,
Горю любовью,
Завтра постучусь в твое окно.

Время пришло... Проснись, моя радость, проснись,
Подними одеяло, открой глаза!
Как прекрасен алый рассвет,
Звезды рассыпались по небу,
Иволга поет в осине у ручья.

Вся природа, радуясь,
Ждет тебя - цветущая весна.
Проснись, моя радость, проснись,
Гори в лучах зари,
Давай встретим вместе утро!

Время пришло... Проснись, моя радость, проснись,
Подними одеяло, открой глаза!
Как прекрасен алый рассвет,
Звезды рассыпались по небу,
Иволга поет в осине у ручья.
```
