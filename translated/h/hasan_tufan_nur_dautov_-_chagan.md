# Оригинал

### Хәсән Туфан / Нур Даутов - Чаган

```
Кояш чыкты,
Көзге яфракларга
Алтын төсле нурлар сибелде,
Өй алдында (кырында) чаган... сары, уйчан:
Кемнедер ул сагына шикелле.

Яфрак коя чаган көзге җилдә,
Яңгыр яуган кара төннәрдә.
Яфрак коя чаган моңсу гына
Истәлеккә калган эзләргә.

...Яфраклары коела моңсу гына,
Яфраклары җәйге өметнең...
Көрсенмә, син, чаган, -
Без бит икәү:
Мин дә бит нәкъ синең кебекмен!.
```

# Перевод

### Хәсән Туфан / Нур Даутов - Чаган

```
Солнце вышло,
На осенние листья
Золотые лучи рассыпало,
Перед домом (у поля) чаган... желтый, задумчивый:
Словно по ком-то он тоскует.

Роняет листья чаган в осеннем ветре,
В черных ночах под дождем.
Роняет листья чаган печально
На следы, что остались в памяти.

...Листья его опадают печально,
Листья летней надежды...
Не грусти ты, чаган, -
Ведь нас двое:
Я ведь тоже точь-в-точь как ты!.
```
