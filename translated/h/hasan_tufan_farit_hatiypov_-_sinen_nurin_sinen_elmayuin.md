# Оригинал

### Хәсән Туфан / Фәрит Хатыйпов - Синең нурың (Синең елмаюың)

```
Кояшта тудың микән,
Йолдызда тордың микән?
Өйгә ай кергәнме дисәм,
Ул синең нурың икән.

Кигәннәрең ал икән,
Гөләп белән пар икән.
Алны уңмый диләр иде,
Халык сүзе хак икән.

Керфекләрең кагасың,
Моңсу гына багасың.
Ай кояштан нурлар ала,
Ә син каян аласың?
```

# Перевод

### Хәсән Туфан / Фәрит Хатыйпов - Твой свет (Твоя улыбка)

```
Ты родилась ли в солнце,
Жила ль среди звезды?
«Луна вошла в мой дом?» -
Да это ты, твой свет.

Одежды твои - алые,
С румяной розой в лад.
«Алый не будет счастлив»,
Так люди говорят - и правды ряд.

Ты ресницами взмахнёшь,
И смотришь чуть грустно, тихо.
Луна берёт лучи у солнца,
А ты - откуда их берёшь?
```
