# Оригинал

### Халык / Асаф Вәлиев - Сине генә көттем мин

```
Көянтәмне ал әле,
Йөгереп суга бар әле.

Су буенда камышлык,
Сандугачлар сайраша.
Син елмайсаң нур сибелә
Чишмәнең ак ярына.

Челтерәп аккан сулар кебек
Син елмаеп көләсең.

Сине генә көттем мин,
Сине генә сагынам.
Синең зәңгәр күзләреңне,
Чәчәкләрдән әзләдем.
```

# Перевод

### Народная / Асаф Вәлиев - Я ждал только тебя

```
Сорвал бы дождик, что ли,
Скачком к воде бы побежал.

На берегу тростник шумит,
Соловьи поют вовсю.
Когда ты не улыбаешься - свет гаснет
На белом берегу ключа.

Как журчащие струи воды,
Ты улыбаешься и смеешься.

Я ждал только тебя,
Я скучаю только по тебе.
Твои голубые глаза
Я ценил больше цветов.
```
