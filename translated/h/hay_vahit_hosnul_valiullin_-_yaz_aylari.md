# Оригинал

### Хәй Вахит / Хөснул Вәлиуллин - Яз айлары

```
Яз айлары, яз айлары
Яшелләтә кырларны;
Яз айлары хәтерләтә
Онытылган җырларны.

Яз сулары, яз сулары
Актара яр, комнарны;
Яз айлары кабындыра
Сүрелгән ялкыннарны.

Яз нурында, яз нурында
Чәчәк ата үрмә-гөл.
Яз-айлары сагынуымны.
Тирәнәйтә бермә-бер.

Яз аенда кошлар оча
Тирәктән тирәкләргә,
Хезмәт җыры тулып ташый
Йөрәктән йөрәкләргә.
```

# Перевод

### Хай Вахит / Хуснул Валиуллин - Месяцы весны

```
Месяцы весны, месяцы весны,
Зеленеют ими поля;
Месяцы весны напоминают
Песни, что забыл я.

Воды весны, воды весны
Размывают берег, пески;
Месяцы весны разжигают
Погасшие огоньки.

В свете весны, в свете весны
Зацветает вьюн-цветок.
Месяцы весны мою тоску
Углубляют во сто крат.

В месяцы весны летят
Птицы с тополя на тополь,
Песня труда переливается
От сердца к сердцу.
```
