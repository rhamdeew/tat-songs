# Оригинал

### Халык җыры - Ырымбур-кала

```
Аһ, Ырымбур-кала, сәхрә дала,
Күккә тиеп тора манара.
Сау булыгыз дуслар, без китәбез,
Бәлки күршә алмабыз, тиз ара.

Аһ, Ырымбур-кала, сәхрә дала,
Күккә тиеп тора ара тау.
Сау булыгыз дуслар, без китәбез,
Бәлки күршә алмабыз, исәнсау.

Аһ, Ырымбур-кала, сәхрә дала,
Елкылары чыккан бәйгегә.
Сау булыгыз дуслар, без китәбез,
Бәлки күршә алмабыз, мәңгегә.
```

# Перевод

### Халык җыры - Оренбург-город

```
Ах, Оренбург-город, степь бескрайняя,
К небу тянется твой минарет.
Будьте здравы, друзья, мы уходим,
Может, скоро не свидимся - нет.

Ах, Оренбург-город, степь бескрайняя,
К небу тянется дальний хребет.
Будьте здравы, друзья, мы уходим,
Может, больше не свидимся - нет.

Ах, Оренбург-город, степь бескрайняя,
Скакуны твои вышли в забег.
Будьте здравы, друзья, мы уходим,
Может, вовсе не свидимся - век.
```
