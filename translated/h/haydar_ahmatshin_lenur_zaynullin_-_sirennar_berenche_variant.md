# Оригинал

### Хәйдәр Әхмәтшин / Ленур Зәйнуллин - Сиреньнәр (беренче вариант)

```
Сиреньнәр чәчәге шикелле
Син һаман яшәреп торасың.
Хуш исләр таратып хәзер дә
Гомерем юлыннан барасың.

Сиреньнәр, сиреньнәр
Ул синең иң татлы, хуш исле иреннәр.
Сиреньнәр чәчәген үпкәндә
Күргәндәй булам мин сине дә.

Сиреньнәр чәчәге шикелле
Сулышың шундый да хуш исле.
Чәчләрең син тарап үргәндә
Шул сирень чәчәген куш әле.

Сиреньнәр, сиреньнәр
Ул синең иң татлы, хуш исле иреннәр.
Сиреньнәр чәчәген үпкәндә
Күргәндәй булам мин сине дә.

Мәхәббәт билгесе шикелле
Сиреньнәр шул хисләр көзгесе.
Шулай да сиңа дип бүләккә
Ничек соң аларны өзәсең?

Сиреньнәр, сиреньнәр
Ул синең иң татлы, хуш исле иреннәр.
Сиреньнәр чәчәген үпкәндә
Күргәндәй булам мин сине дә.
```
