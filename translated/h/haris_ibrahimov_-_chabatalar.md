# Оригинал

### Харис Ибраһимов - Чабаталар

```
Чабаталар - чабата
Сезне халык ярата.
Чабатамны кигәндә
Аякларым рәхәттә.

Чабаталар корылган
Матур үреп ясалган.
Табигатьнең исләре,
Очсыз, җайлы, уңайлы.

Җәен һава бизәрли
Көзен сулар җибәрми.
Кышын аяк җылыта
Бигрәк әйбәт чабата.

Кунакларга барганда
Аягымда чабата,
Басмагыз сез аякка!
Кидем яңа чабата.
```

# Перевод

### Харис Ибраһимов - Лапти

```
Лапти, лапти - вот они,
Их народ у нас любит.
Надеваю я лапти -
И ногам моим уютно.

Лапти сплетены умело,
Красотой своей светлеют.
Запахи родной природы,
Дёшевы, удобны, ладны.

Летом воздух приукрасят,
Осенью воды не впустят.
Зимой ноги согревают -
Ах, какие добрые лапти.

В гости иду - на ногах лапти,
Не наступайте мне на ногу!
Я надел их, да новые лапти.
```
