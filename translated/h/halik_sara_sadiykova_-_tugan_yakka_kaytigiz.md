# Оригинал

### Халык / Сара Садыкова - Туган якка кайтыгыз

```
Акланмагыз намус алларында,
Вакытлар аз, тизерәк кайтыгыз.
Кешеләргә гомер ике килми,
Туган якка ешырак кайтыгыз!

Кайтыгыз сез сабан туйларына,
Кайтыгыз сез кышын тагын да.
Сагынганда җан җылысын бирә
Туган якнын салкын кары да.

Читтә йөргәннәрнең күңелендә
Тирбәлдерә торган сагыш бар.
Очар кошлар булып кайтыгыз сез
Тирбәлгәндә күлләр, камышлар.
```

# Перевод

### Народ / Сара Садыкова - Возвращайтесь на родину

```
Не скрывайте совесть в глубине,
Времени мало, возвращайтесь скорее.
Две жизни людям не даётся,
На родину возвращайтесь пораньше!

Возвращайтесь вы на сабантуи,
Возвращайтесь вы и зимой тоже.
Когда тоскует - даёт тепло души
Даже холодный снег родной стороны.

В сердцах тех, кто вдали странствует,
Тоска вечная трепещет всегда.
Вернитесь птицами, возвращайтесь,
Когда трепещут озёра да камыши.
```
