# Оригинал

### Харрас Әюп / Луиза Батыр-Болгари - Сиңа гына язган хатлар

```
Әгәр еракларга китеп барсам,
Сагынудан синдә янса утлар -
Бәхет тамчылары булып тамар
Сиңа гына язган назлы хатлар,
Сиңа гына язган назлы хатлар.

Чәчләреңә инде чал керсә дә,
Искә төшәр бергә булган чаклар.
Яшьлек тургайлары булып килер
Сиңа гына язган назлы хатлар,
Сиңа гына язган назлы хатлар.

Йөрәкләрдә янган сөю утын
Сүрелдерми мәңге шулай саклар -
Син көт кенә, килер сагынганда
Сиңа гына язган назлы хатлар.
Сиңа гына язган назлы хатлар...
```

# Перевод

### Харрас Әюп / Луиза Батыр-Болгари - Лишь тебе написанные нежные письма

```
Если я в дальние края уйду,
В тебе от тоски заполыхают огни -
Каплями счастья прольются
Лишь тебе написанные нежные письма,
Лишь тебе написанные нежные письма.

Пусть даже в волосах уже седина,
Вспомнятся дни, когда мы были вместе.
Словно жаворонки юности, прилетят
Лишь тебе написанные нежные письма,
Лишь тебе написанные нежные письма.

Огонь любви, горящий в сердцах,
Не угаснет - так навек его сохранят;
Ты только жди - в часы тоски придут
Лишь тебе написанные нежные письма,
Лишь тебе написанные нежные письма...
```
