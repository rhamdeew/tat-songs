# Оригинал

### Халисә Хәкимуллина / Мәсгудә Шәмсетдинова - Язгы тамчы

```
Тып, тып, тып,
Ты-пы-тып, тып, тып, тып,
Тамчы тама талларга,
Талларымның бөреләре
Куаныша шуңарга,
Куаныша шуңарга.

Тып, тып, тып,
Ты-пы-тып, тып, тып, тып,
Яз тама тамчы булып.
Кояш нурлары биешә
Шул тамчыга куанып,
Шул тамчыга куанып.
```

# Перевод

### Халиса Хакимуллина / Масгуда Шамсетдинова - Весенняя капель

```
Кап, кап, кап,
Ка-па-кап, кап, кап, кап,
Капли падают на ивы,
Ивовые почки
Радуются этому,
Радуются этому.

Кап, кап, кап,
Ка-па-кап, кап, кап, кап,
Весна капает каплями.
Солнечные лучи танцуют,
Радуясь той капле,
Радуясь той капле.
```
