# Оригинал

### Хөснулла Түләкәев / Фәүзә Гасакирова - Сагышлы күңел

```
Сайрыйлар былбыллар, шаулыйлар елгалар,
Күгемдә чалт аяз, кояшлы.
Әйтегез, нигә соң, нигә соң күңелем
Гел генә, гел генә сагышлы?

Ялгыз да түгелмен, бар җандай дусларым,
Адымым ырамлы, алышлы.
Әйтегез, нигә соң, нигә соң күңелем
Гел генә, гел генә сагышлы?

Яшәүе ямь генә, урамнар гөр килә,
Кешеләр җор сүзле, шатлыклы.
Сагындым мин сине, өзелеп сагындым,
Тик шуңа күңелем сагышлы.
```

# Перевод

### Хуснулла Тулакаев / Фауза Гасакирова - Тосковливая душа

```
Поют соловьи, шумят потоки,
В лазури неба - ясный свет, солнце в выси.
Скажите, почему же, почему душа моя
Вечно, вечно тоскует?

Не одинока я - есть друзья, словно весь свет,
Шаги мои легки, в радости движение.
Скажите, почему же, почему душа моя
Вечно, вечно тоскует?

Прекрасна жизнь, улицы полны жизни,
Люди с речью живой, в счастье и радости.
Скучаю по тебе, разрываясь от тоски,
Лишь поэтому душа моя грустит.
```
