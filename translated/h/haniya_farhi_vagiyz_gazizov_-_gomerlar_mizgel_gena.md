# Оригинал

### Хәния Фәрхи / Вәгыйзь Газизов - Гомерләр мизгел генә

```
Сөеп-сөелепләр яшик,
Яшәү бит мизгел генә.
Бәхет кошың сине күрер,
Әйбәт бул үзең генә.

Сабырлыклар, түземлекләр
Кешене бизи генә.
Яшик әле җырлый-җырлый,
Гомерләр мизгел генә.

Мәхаббәтле яшәр өчен
Бул фәкать түзем генә.
Бер-беребезнең кадерен белик,
Яшәү бит мизгел генә.

Сабырлыклар, түземлекләр
Кешене бизи генә.
Яшик әле җырлый-җырлый,
Гомерләр мизгел генә.
Гомерләр мизгел генә...
Сабырлыклар, түземлекләр
Кешене бизи генә.
Яшик әле җырлый-җырлый,
Гомерләр мизгел генә.
Гомерләр мизгел генә...
```
