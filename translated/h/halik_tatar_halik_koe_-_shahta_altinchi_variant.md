# Оригинал

### Халык / Татар халык көе - Шахта (алтынчы вариант)

```
Китә казлар, китә казлар,
Китә казлар Донбасска шул;
Миндә уйлап торам әле
Шул казлардан калмаска.

Шахталарга мин төшәрмен, / Шахталарга син төшәрсең, /
Җырлап күмер кисәрмен шул. /Җырлап күмер кисәрсең; /
Әгәр сине бик сагынсам, / Әммә мине бик сагынсаң, /
Бик саргайсам, нишләрмен? / Бик саргайсаң (ла) нишләрсең? /

Әгәр сине бик сагынсам, / Бик сагынсаң, бик саргайсаң, /
Йөрәккәем тынмаса,
Сагынуларым(ң) чиктән (лә) ашса,
Чакыртырмын Донбасска. / Чакыртырсың Донбасска. /
```

# Перевод

### Народ / Татарская народная мелодия - Шахта (шестой вариант)

```
Уезжают рабочие, уезжают рабочие,
Уезжают рабочие в тот Донбасс;
Я всё думаю и думаю,
Как бы не остаться от этих рабочих.

В шахты я спущусь, / В шахты ты спустишься, /
Рубя уголь, буду петь я. / Рубя уголь, будешь петь ты; /
Если я очень буду скучать по тебе, / А если меня очень будешь скучать, /
Очень завяну, что делать мне? / Очень завянешь - что тебе делать? /

Если я очень буду скучать по тебе, / Очень будешь скучать, очень завянешь,
Если моё сердечко не успокоится,
Если тоска по мне (тебе) через край перельётся,
Позову тебя в Донбасс. / Позовёшь меня в Донбасс. /
```
