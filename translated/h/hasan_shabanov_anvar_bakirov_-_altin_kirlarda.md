# Оригинал

### Хәсән Шабанов / Әнвәр Бакиров - Алтын кырларда

```
Канат кагып
Кошлар сайраган чагында
Бөдрә талларда,
Җәй кояшы җиргә
Якты нурын чәчеп
Балкып торганда,

Хезмәт батырлары
Җырлап иген ура
Алтын кырларда.

Су буенда таллар,
Куе камышкайлар
Суга иелгән...
Таллар кебек сыгылып,
Камыш кебек куерып
Уңган игеннәр.

Хезмәт батырлары
Җырлап иген ура
Алтын кырларда.
```

# Перевод

### Хасан Шабанов / Анвар Бакиров - На золотых полях

```
Когда, крылья взмахивая,
Птицы распевали
На кудрявых ивах,
И летнее солнце на землю
Свет свои лучи рассыпая,
Сияло,

Герои труда
Песни поют, хлеб жнут
На золотых полях.

У воды ивы,
Густые камыши
К воде наклонились...
Как ивы согнувшись,
Как камыш иссохнув
Умелые хлеборобы.

Герои труда
Песни поют, хлеб жнут
На золотых полях.
```
