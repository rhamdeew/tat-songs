# Оригинал

### Халисә Мөдәрисова / Мөдәрис Газетдинов - Танымадың

```
Танымадың мине, танымадың,
Бердәнберем, яшьлек хыялым.
Мин бит сине сөеп туялмадым,
Аңлатырга гына, аңлатырга гына,
Аңлатырга гына соңладым.
Аңлатырга гына, аңлатырга гына,
Аңлатырга гына соңладым.

Танымадың мине, танымадың.
Танытырга нигә тоймадың?
Гомер буе сиңа ашкынсам да
Яратырга сине, яратырга сине,
Яратырга сине соңладым.
Яратырга сине, яратырга сине,
Яратырга сине соңладым.
Яратырга сине соңладым...
```

# Перевод

### Халисә Мөдәрисова / Мөдәрис Газетдинов - Не узнала

```
Не узнала меня, не узнала,
Единственный мой, юности сон.
Я ведь любить тебя не уставала,
Объяснить лишь, объяснить лишь,
Объяснить лишь опоздала.
Объяснить лишь, объяснить лишь,
Объяснить лишь опоздала.

Не узнала меня, не узнала.
Почему не почувствовала - узнать?
Хоть всю жизнь к тебе я устремлялась,
Полюбить тебя, полюбить тебя,
Полюбить тебя опоздала.
Полюбить тебя, полюбить тебя,
Полюбить тебя опоздала.
Полюбить тебя опоздала...
```
