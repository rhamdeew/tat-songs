# Оригинал

### Халидә Шәяхмәтова / Флүрә Үзәнбаева - Терәк булыр кешең кирәк

```
Терәк булыр кешең кирәк
Тормыш юлын үтәргә.
Теләкләрең кабул булып
Рәхәт гомер итәргә.

Сайрап яшәр өчен кирәк
Сандугачка тал-тирәк.
Бергә булсаң ярың белән
Дөнья була түгәрәк.

Аккошларга кайтыр өчен
Таза елгалар кирәк.
Гомер матур үтсен өчен
Таһир-Зөһрәләр кирәк.

Сайрап яшәр өчен кирәк
Сандугачка тал-тирәк.
Бергә булсаң ярың белән
Дөнья була түгәрәк.

Гомер шаулап үтә икән
Әйтерсең аккан елга.
Ярыңа ярәшеп яшәү
Тиң бит ул матур җырга.

Сайрап яшәр өчен кирәк
Сандугачка тал-тирәк.
Бергә булсаң ярың белән
Дөнья була түгәрәк.
Бергәләшеп парың белән
Яшәве күңеллерәк.
```

# Перевод

### Халида Шахматова / Флюра Узенбаева - Нужен человек, который станет опорой

```
Нужен человек, который станет опорой
Для пути жизни.
Чтобы желания исполнялись
И жизнь была спокойной.

Для счастливой жизни нужно
Дерево сандугач.
Если ты вместе со своей половинкой
Мир становится круглее.

Чтобы вернуться к лебедям
Нужны чистые реки.
Чтобы жизнь прошла красиво
Нужны Тахир и Зухра.

Для счастливой жизни нужно
Дерево сандугач.
Если ты вместе со своей половинкой
Мир становится круглее.

Жизнь проходит веселее
Скажи реке, текущей.
Жить, подобающей тебе
Равна той красивой песне.

Для счастливой жизни нужно
Дерево сандугач.
Если ты вместе со своей половинкой
Мир становится круглее.
Жить вместе с партнёром
Гораздо радостнее.
```
