# Оригинал

### Хәйдәр Әхмәтшин / Рәсим Фәсыйхов - Бер кочак шатлыгым

```
Кочаклап алам да чәчәгең,
Шатланып нурларым чәчәмен.
Янымда бары тик син генә
Булганга йөрәгем сөенә.

Бер кочак шатлыгым булып син
Җаныма кереп тә тулдың син.
Шулкадәр бәхетле минутлар
Китмәсен, туктасын вакытлар.
Каршыма язмышым китергән
Син икән ул, җаным, син икән.
Син икән бер кочак шатлыгым,
Бәхеткә уралган байлыгым.

Шатлыгым бер кочак булса да,
Бәхетем җаныма тулса да,
Мин сине кадерләп сакларга,
Әзер бит читләрдән якларга.

Бер кочак шатлыгым булып син,
Җаныма кереп тә тулдың син.
Шулкадәр бәхетле минутлар
Китмәсен, туктасын вакытлар.
Каршыма язмышым китергән
Син икән ул, җаным, син икән.
Син икән бер кочак шатлыгым,
Бәхеткә уралган байлыгым.
Каршыма язмышым китергән
Син икән ул, җаным, син икән.
Син икән бер кочак шатлыгым,
Бәхеткә уралган байлыгым...
```

Ты стала охапкой моей радости,
Заполнила душу до краёв.

Не уходят, пусть время застынет.
Судьба ко мне навстречу привела
Именно тебя, родная, тебя.
Это ты - моя охапка радости,
Моё богатство, в счастье укутанное.

Пусть радость всего лишь охапка,
Пусть счастье душу переполняет,
