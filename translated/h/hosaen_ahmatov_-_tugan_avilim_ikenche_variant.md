# Оригинал

### Хөсәен Әхмәтов - Туган авылым (икенче вариант)

```
Туган авылым Сакмар буйларында,
Шунда үтте минем балачак.
Сакмар дулкыннары, курай моңы
Йөрәгемдә мәңге калачак.

Атам - анам, биләп, теләк иткән,
Ирәндекем, ямьле Сакмарым.
Мин онытмам сезне һич бервакыт,
Туган авылым, туган якларым.
```

# Перевод

### Хөсәен Әхмәтов - Родная деревня (второй вариант)

```
Родная деревня у Сакмара стоит,
Там прошли мои детские дни.
Волны Сакмара, звон курая звучит -
В сердце навечно они.

Отец мой и мать - мне желали пути,
Ирендек мой, светлый Сакмар.
Никогда не забыть мне вас, как ни идти,
Родной мой аул, родной край мой дар.
```
