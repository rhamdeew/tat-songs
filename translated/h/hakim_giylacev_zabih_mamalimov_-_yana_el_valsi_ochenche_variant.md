# Оригинал

### Хәким Гыйләҗев / Зәбих Мәмәлимов - Яңа ел вальсы (өченче вариант)

```
Бар тирә-як сөттәй ап-ак,
Җем-җем итеп кар көлә:
Көмеш кыңгыраулар тагып,
Шатлык бөркеп, нурдай балкып,
Җиргә Яңа ел килә.

Көрткә чумып коена чыршы,
Энәсе яшел генә:
Яшел кыңгыраулар тагып,
Шатлык бөркеп, нурдай балкып,
Җиргә Яңа ел килә.

Дөнья бәйрәм, дөнья хәйран,
Йөрәктән (йөрәктәге) җыр бөркелә:
Моңлы кыңгыраулар тагып,
Шатлык бөркеп, нурдай балкып,
Җиргә Яңа ел килә.

Кабул булсын бар теләкләр,
Бәхет керсен һәр өйгә,
Мең-мең кыңгыраулар тагып,
Шатлык бөркеп, нурдай балкып,
Җиргә Яңа ел килә.
```

# Перевод

### Хәким Гыйләҗев / Зәбих Мәмәлимов - Новогодний вальс (третий вариант)

```
Всё вокруг, как молоко, белым-бело,
Снег, искрясь, смеётся:
Серебром звенят колокольцы,
Радость брызжет, светом льётся -
К нам на землю Новый год идёт.

Ёлка, в сугробах купаясь,
Зелена её хвоя:
В зелень звонкие колокольцы,
Радость брызжет, светом льётся -
К нам на землю Новый год идёт.

Мир - как праздник, мир - как диво,
Песня из сердца рвётся:
Звон тоскливых колокольцев,
Радость брызжет, светом льётся -
К нам на землю Новый год идёт.

Пусть сбываются желания,
Счастье входит в каждый дом,
Тысячи звенят колокольцев,
Радость брызжет, светом льётся -
К нам на землю Новый год идёт.
```
