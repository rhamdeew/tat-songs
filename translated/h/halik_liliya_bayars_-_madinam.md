# Оригинал

### Халык / Лилия Баярс - Мәдинәм

```
Ай янында якты йолдыз,
Минем йолдызым гына
Мәдинәм гөлкәем
Минем йолдызым гына бәгыркәем

Кычкырып җырлап җибәрәм,
Моңайган чакларымда
Мәдинәм гөлкәем
Моңайган чакларымда бәгыркәем.

Ай янында якты йолдыз,
Айның урталарында
Мәдинәм гөлкәем
Айның урталарында бәгыркәем.

Таң атуга ул да сүнә
Калам ялгызым гына,
Мәдинәм гөлкәем
Калам ялгызым гына бәгыркәем.
```

# Перевод

### Народ / Лилия Баярс - Мадинам

```
Рядом с луной яркая звезда,
Только моя звезда,
Мадинам, моё сияние,
Только моя звезда, моя радость.

Кричу и запеваю,
В минуты грусти моей,
Мадинам, моё сияние,
В минуты груди моей - моя радость.

Рядом с луной яркая звезда,
В середине месяца,
Мадинам, моё сияние,
В середине месяца - моя радость.

На рассвете она гаснет,
Остаётся я одинокий,
Мадинам, моё сияние,
Остаётся я одинокий - моя радость.
```
