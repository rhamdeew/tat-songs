# Оригинал

### Халисә Мөхәммәдиева / Наил Шәймарданов - Кичерегез, якын затлар

```
Бу юлы да кайта алмадым,
Кичерегез, якын затлар,
Көтәсездер ял көннәрен,
Күптән юктыр бездән хатлар.

Кайтулардан китү кыен,
Әнкәй-әткәй елап кала.
Тормыш шулай яратылган:
Туган йорттан оча бала.

Очрашулар сезнең белән
Йөрәкләргә тик май яга.
Күрешмичә торган гомер
Сулар кебек бик җай ага.

Кайтулардан китү кыен,
Әнкәй-әткәй елап кала.
Тормыш шулай яратылган:
Туган йорттан оча бала.

Эшләребез уңган төсле
Бу дөньяда сез булганга.
Балам, диеп иртә-кичен
Изге догалар кылганга.

Кайтулардан китү кыен,
Әнкәй-әткәй елап кала.
Тормыш шулай яратылган:
Туган йорттан оча бала.
```

# Перевод

### Халиса Мухаммадиева / Наил Шаймарданов - Простите, родные

```
И в этот раз не смог вернуться,
Простите, родные, простите,
Вы ждёте выходных и праздников,
Давно от нас не слышно вести.

Так трудно уезжать из дома,
Родители плачут, провожая.
Такой уж создан этот мир:
Из отчего гнезда птенец взлетает.

Встречи с вами - это счастье,
Бальзам целительный для сердца.
Жизнь без борьбы и без усилий
Как летний ручеёк несётся.

Так трудно уезжать из дома,
Родители плачут, провожая.
Такой уж создан этот мир:
Из отчего гнезда птенец взлетает.

Наши дела идут удачно,
Ведь вы есть с нами на земле.
«Дитя моё», - твердя с любовью,
Вы шлёте нам святые молитвы.

Так трудно уезжать из дома,
Родители плачут, провожая.
Такой уж создан этот мир:
Из отчего гнезда птенец взлетает.
```
