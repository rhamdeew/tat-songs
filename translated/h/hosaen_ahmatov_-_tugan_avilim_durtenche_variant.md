# Оригинал

### Хөсәен Әхмәтов - Туган авылым (дүртенче вариант)

```
Туган авылым Сакмар буйларында,
Шунда үтте минем балачак.
Сакмар дулкыннары, курай моңы,
Йөрәгемдә мәңге калачак.

Атам - анам синдә туып үскән,
Синдә үтте яшьлек елларым.
Мин онытмам сине бер вакытта,
Туган авылым, туган якларым.
```

# Перевод

### Хусаин Ахметов - Родная деревня (четвёртый вариант)

```
Родная деревня моя на берегах Сакмара,
Там прошло моё детство.
Волны Сакмара, напевы курая,
Всегда останутся в моём сердце.

Отец и мать у тебя родились и выросли,
У тебя прошли мои юные годы.
Я никогда не забуду тебя,
Родная деревня моя, родные края.
```
