# Оригинал

### Хәмид Мәткәримов - Сагынам сине (беренче вариант)

```
Ялгыз чакларыңда уйланасың микән,
Минем төсле сагыш утында янасың микән?

Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.
Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.

Син бит минем йөзләр-меңнәр арасыннан
Сайлап алган бердәнбер гөл сөю назларыннан.

Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.
Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.

Хыял-уйларымда, назлап кулларымда
Сөеп кочар идем, булсаң әгәр яннарымда.

Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.
Сагынам сине, юксынам сине,
Йөрәгем минем ярата сине, сөйгәнем.
```

```
В свои одинокие моменты думаешь ли,
Горит ли твоё сердце в тоске как моё?

Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.
Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.

Ты ведь единственный цветок любви,
Который я выбрал из тысяч и тысяч людей.

Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.
Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.

В моих мечтах-грёзах, ласково в руках
Обнимал бы тебя, будь ты рядом со мной.

Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.
Скучаю по тебе, тоскую по тебе,
Сердце моё любит тебя, любимая.
```
