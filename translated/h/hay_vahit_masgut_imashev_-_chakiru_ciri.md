# Оригинал

### Хәй Вахит / Мәсгут Имашев - Чакыру җыры

```
Очраттым мин аны яр өстендә,
Күзләреннән күзем алмадым.
Күңелемнең аны бик күптәннән бирле
Эзләгәнен шундук аңладым.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.

Дөньяның бу кадәр матурлыгын
Беренче кат күрдем шул чакта.
Оныта алмам мәңге, мәңге оныта алмам,
Гомер буе аерым торсак та.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.

Күңелемнең аны бик күптәннән
Эзләгәнен шундук аңладым.
Утлы йөрәгемнән хисләр ташып чыкты,
Тик үзенә әйтә алмадым.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.
```

# Перевод

### Хай Вахит / Мәсгут Имашев - Песня зова

```
Яр янында очралдык әле,
Күзләреннән күз дә алып алмадым.
Күңелем олып эзләгән кеше
Шул ук сән исәнен дә аңладым.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.

Бу дөньның матурлыгын озынвасын
Беренче тапкыр күргән идем шул вакытта.
Онытмыйм һәм онытмыйм да,
Гомер буе аерым торсак та.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.

Күңелем олып эзләгән кеше
Шул ук сән исәнен дә аңладым.
Йөрәгемнән ут чыгып ташлады,
Ләкин үзенә әйтә алмадым.

Сәрия, күз нурым,
Мәңге онытылмас җырым.
Син кайда, син кайда?
Сине эзлим һаман да.
```
