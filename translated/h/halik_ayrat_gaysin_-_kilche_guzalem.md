# Оригинал

### Халык / Айрат Гайсин - Килче, гүзәлем!

```
Күл буенда камышлар
Шашып шаулыйлар.
Мәхәббәтсез кешеләр
Яши алмыйлар.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.

Күл буенда бөдрә таллар
Сер чишә дулкыннарга.
Чорнап алды мәхәббәтең
Әйләнеп ялкыннарга.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.

Серләрем күп, сердәшем юк,
Сөйләргә ме талларга?
Гөлләр булып үсәр идем
Син киләсе юлларга.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.
```

# Перевод

### Народная песня / Айрат Гайсин - Приходи, красавица!

```
У озера камыши
Раскинувшись шумят.
Без любви люди
Жить не могут.

Приходи, приходи, красавица,
Приходи ко мне!
Разрывается, разрывается сердце
В пору сильной тоски по тебе.

У озера кудрявые ивы
Открывают тайны волнам.
Твоя любовь охватила
Превратившись в пламя.

Приходи, приходи, красавица,
Приходи ко мне!
Разрывается, разрывается сердце
В пору сильной тоски по тебе.

Тайн у меня много, подруги нет,
Разве скажешь ивам?
Цветами бы я росла
На тропах, где ты пройдёшь.

Приходи, приходи, красавица,
Приходи ко мне!
Разрывается, разрывается сердце
В пору сильной тоски по тебе.
```
