# Оригинал

### Хәй Вахит / Сара Садыкова - Заһид җыры

```
Истә һаман авыл урамнары
Күз алдымда болын юллары.
Гармун тартып, зәңгәр кичләреңдә
Шунда үтте яшьлек елларым.

Шунда калды шаян чак җырларым,
Каеннарның тулып ялына.
Эзләп кайттым яшьлек эзләремне,
Тынгы тапмам микән җаныма.
```

# Перевод

### Хай Вахит / Сара Садыкова - Песня Захида

```
Помню всё ещё деревни улицы,
Перед взором луговые дали.
Под гармонь в те синие вечера
Там прошли мои весенние годы.

Там остались те счастливые песни,
У берёз, что волнами шумели.
Я вернулся, след юности ищу,
Не найти ли мне покой для души.
```
