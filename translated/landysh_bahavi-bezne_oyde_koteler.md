# Оригинал

### Ландыш Бахави - Безне ойдэ котэлэр

```
Эштэме, ерак юлдамы,
Каршыласан, иркэлэр.
Кайтырга тиешлэр диеп,
Безне ойдэ котэлэр.

Читкэ китсэк, минутлары
Елдан озак утэлэр.
Ят яр безгэ кирэк тугел,
Безне ойдэ котэлэр.

Язмышлар кайчак кайтасы,
Куперлэрне сутэлэр.
Сонгы омет якты учак,
Безне ойдэ котэлэр.

Бэхет кояшы елмайса,
Жаннар эреп китэлэр.
Шатлыкларны булешергэ
Безне ойдэ котэлэр.
```

# Перевод

### Ландыш Бахави - Нас дома ждут

```
Любимый, далека ль дорога,
Встретишь - ласкают.
Что вернуться нам должны,
Нас дома ждут.

Если уйдем далеко, минуты
Дольше года проходят.
Чужая любовь нам не нужна,
Нас дома ждут.

Судьбы порою повернутся,
Мосты разрушат.
Последняя надежда - яркий огонек,
Нас дома ждут.

Если солнце счастья не улыбнется,
Души измучаются.
Чтобы радостью делиться
Нас дома ждут.
```
