# Оригинал

### Рустам Закиров - Кунелен сынмасын

```
Тэрэзэнэ бер кон
Бэрде карлы янгыр
Кемдер эйтте: "Туз син,
Язмышын шулайдыр".

Китте ул ашыгып,
Китте ул хушлашмый.
Кемгэдер упкэлэп
Доньяны анламый.

Авыр коннэрендэ
Тошенне кыс, корэш.
Бирешмэ, елама,
Туз инде, сенелкэш.

Син ялгызын тугел
Сынмасын кунелен.
Балаларын очен
Яшэ, яшэ бэгърем.

Карангы тоннэрдэ
Ялгызын калганда,
Каргама доньяны
Кунелен тулганда.

Тоннэн сон кон туа
Ач тэрэзэлэрен.
Кояш узе алыр
Кара пэрдэлэрен.
```

# Перевод

### Рустам Закиров - Не разбивай сердце

```
В окно однажды
Ударил снежный дождь,
Кто-то сказал: "Терпи,
Такова твоя судьба".

Он ушёл поспешно,
Он ушёл, не простившись.
На кого-то обидевшись,
Не понимая мира.

В трудные дни
Сожми зубы, борись.
Не сдавайся, не плачь,
Терпи же, сестра.

Ты не одинока,
Пусть не разобьётся сердце.
Ради своих детей
Живи, живи, родная.

В тёмные ночи,
Когда остаёшься одна,
Не кляни этот мир,
Когда сердце полно боли.

После ночи рождается день -
Открой свои окна.
Солнце само снимет
Чёрные занавесы.
```
