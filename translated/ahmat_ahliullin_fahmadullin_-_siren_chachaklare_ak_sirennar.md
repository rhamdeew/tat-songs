# Оригинал

### Әхмәт Әһлиуллин / Ф.Әхмәдуллин - Сирень чәчәкләре (Ак сиреннәр)

```
Мине сагынып килер юлларыма
Сирень чәчәкләрен сибәрсең.
Мин яратам, бәгърем, сирень чәчәкләрен,
Берсен миңа өзеп бирерсең.

Язлар җитте, ямьле җәйләр килде,
Сирень чәчәкләре сибелде.
Мин яратам, бәгърем, сирень чәчәкләрен,
Берсен миңа өзеп бирерсең.
```

# Перевод

### Ахмат Ахлиуллин / Ф.Ахмадуллин - Цветы сирени (Белая сирень)

```
Когда я возвращаюсь по дороге домой,
Ты мне сирень рассыпаешь.
Я люблю, родная, цветы сирени,
Ты мне одну веточку срываешь.

Вёсны прошли, прекрасное лето пришло,
Цветы сирени рассыпались.
Я люблю, родная, цветы сирени,
Ты мне одну веточку срываешь.
```
