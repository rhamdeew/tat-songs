# Оригинал

### Зуфар Хайретдинов - Агыйдел сылуына

```
Агыйдел агышлары,
Зифадыр камышлары.
Агыйделнен бер сылуы –
Йорэгем сагышлары.

Агыйдел яры текэ,
Сулары ерак китэ.
Язны коткэн назлы голдэй
Кунелем сине котэ.

Агыйдел суларында
Аккош канат кагына.
Аккош кебек сылукаем
Син генэ уйларымда.

Агыйдел яры текэ,
Сулары ерак китэ.
Язны коткэн назлы голдэй
Кунелем сине котэ.

Таннар аткан чагында
Агыйдел якларында,
Яннарына барыр идем
Дулкын канатларында.

Агыйдел яры текэ,
Сулары ерак китэ.
Язны коткэн назлы голдэй
Кунелем сине котэ.
```

# Перевод

### Зуфар Хайретдинов - Красавице Агидели

```
Агидели течения,
Стройны её камыши.
У Агидели есть красавица –
Моего сердца печали.

Агидели берег крут,
Воды её вдаль текут.
Словно нежный цветок, ждущий весну,
Душа моя тебя ждёт.

В водах Агидели
Лебедь машет крылом.
Словно лебедь, красавица моя,
Только ты в моих мыслях.

Агидели берег крут,
Воды её вдаль текут.
Словно нежный цветок, ждущий весну,
Душа моя тебя ждёт.

Когда рассвет занимается
На берегах Агидели,
К твоим щекам я б отправился
На крыльях волны.

Агидели берег крут,
Воды её вдаль текут.
Словно нежный цветок, ждущий весну,
Душа моя тебя ждёт.
```
