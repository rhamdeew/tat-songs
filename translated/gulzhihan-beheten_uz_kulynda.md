# Оригинал

### Гөлжиһан - Бэхетен уз кулында

```
Сорау бирдем агым суга
Бэхет ул нинди була?
Белсэм - каршы чыгар идем
Чэчэк тотып кулыма
Минем дэ бит бардыр хакым
Бит бэхетле булырга
Бит бэхетле булырга

Сорау бирдем горур тауга
Бэхетем йори кайда?
Юлсыз жирдэн юллар салып
Эзлэр идем тапканчы
Бит бэхетле булып кына
Яшэр идем мон-зарсыз
Яшэр идем мон-зарсыз

Сихри донья эйтер мина
Бэхетен уз кулымда
Гел алга бар, туктап калма
Табарсын бэхетенне
Анларсын, доньяда яшэу
Ул узе бэхет инде
Шул узе бэхет инде
Анла син, доньяда яшэу
Шул узе бэхет инде
Ин олы бэхет инде

Бэхетен уз кулында
```

# Перевод

### Гульжихан - Счастье в твоих руках

```
Вопрос задала текущей воде:
Каким бывает счастье?
Если б знала — встретила бы
С цветком в руке.
Есть ведь и у меня право
Быть счастливой,
Быть счастливой.

Вопрос задала гордой горе:
Куда ушло мое счастье?
По бездорожью дороги проложив,
Искала бы, пока не найду.
Ведь счастливой будучи,
Жила бы без печали,
Жила бы без печали.

Волшебный мир скажет мне:
Счастье в твоих руках.
Все время иди вперед, не останавливайся,
Найдешь свое счастье.
Поймешь, что жизнь на свете —
Она сама и есть счастье,
Она сама и есть счастье.
Пойми ты, жизнь на свете —
Она сама и есть счастье,
Самое великое счастье.

Счастье в твоих руках.
```
