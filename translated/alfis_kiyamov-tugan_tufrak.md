# Оригинал

### Альфис Киямов - Туган туфрак

```
Чылтыр-чылтыр аккан чишмэлэрем,
Сабый чактан кергэн моннарым.
Шушы изге жирдэн атлап киттем
Матурлыгын тоеп доньянын.

Арышларым, бодайларым,
Басуларым, кырларым —
Минем якты ометлэрем,
Жырланачак жырларым.

Чит жирлэргэ китсэм,
Сагынып кайтам
Буразналар, башак ислэрен.
Оннэремдэ курэм, тошлэремдэ —
Ал шэфэкъле авыл кичлэрен.

Арышларым, бодайларым,
Басуларым, кырларым —
Минем якты ометлэрем,
Жырланачак жырларым.
```

# Перевод

### Альфис Киямов - Родная земля

```
Журчащие-журчащие текущие родники мои,
С детских лет знакомые печали мои.
С этой святой земли шагнул и ушел
Красоту почувствовав мира.

Ржи мои, пшеницы мои,
Поля мои, степи мои —
Мои светлые надежды,
Воспеваемые песни мои.

В чужие края уезжая,
Скучая возвращаюсь
К запахам полыни, колосьев.
В снах вижу, в грезах —
Алые зари деревенских вечеров.

Ржи мои, пшеницы мои,
Поля мои, степи мои —
Мои светлые надежды,
Воспеваемые песни мои.
```
