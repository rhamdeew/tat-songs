# Оригинал

### Puma - Жыр доньясы

```
Хислэр ташып кунелемдэ
Урын эзли узенэ.
Сэхнэдэ, бэхет илендэ
Яшимен кук сизелэ.

Чикли алмыйм хислэремне,
Жырларымда чагыла.
Эсир итеп йорэклэрне,
Жыр доньясы чакыра.

Жырлыйсым килде минем дэ,
Сине куреп сэхнэдэ.
Синен кебек биимен дэ,
Ирек бирэм хислэргэ.

Чикли алмыйм хислэремне,
Жырларымда чагыла.
Эсир итеп йорэклэрне,
Жыр доньясы чакыра.

Жырлар юлдаш гомерлэргэ
Олкэннэргэ, яшьлэргэ.
Жыр доньясы кунеллэргэ
Кочлэр бирэ яшэргэ.

Чикли алмыйм хислэремне,
Жырларымда чагыла.
Эсир итеп йорэклэрне,
Жыр доньясы чакыра.
```

# Перевод

### Puma - Мир песни

```
Чувства переполняют душу мою
Ищут место себе.
На сцене, в стране счастья
Живу, словно небо чувствую.

Не могу сдержать чувства свои,
В песнях моих отражаются.
Пленяя сердца,
Мир песни зовёт.

Захотелось и мне петь,
Видя тебя на сцене.
Как ты, танцую тоже,
Волю даю чувствам.

Не могу сдержать чувства свои,
В песнях моих отражаются.
Пленяя сердца,
Мир песни зовёт.

Песни - спутник на всю жизнь
Взрослым и молодым.
Мир песни душам
Силы даёт для жизни.

Не могу сдержать чувства свои,
В песнях моих отражаются.
Пленяя сердца,
Мир песни зовёт.
```
