# Оригинал

### Алмаз Хәмзин / Марат Хөснуллин - Сыерчыклы, сандугачлы яз

```
Ал кояшта безнең белән
Уянып көтәм.
Гел елмая, гел үзенчә,
Гел-гел үзенчә.
Уйный-көлә, назлый-сыйпый
Иркәли генә.
Андый кояш башка җирдә
Юк ул минемчә.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.

Һәр кемнең үз ягы җырлы,
Моңлы, гүзәлле.
Кызлар ул сылулар,
Чиксез чибәрләр.
Язлары да, көзләре дә
Бигрәк күңелле.
Ай йолдызы кояшына,
Күңел үрелде.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.

Сезнең якта ай сагышлы,
Нигә сагышлы?
Ул бит ялгыз, бер ялгызы,
Япа-ялгызы.
Сары сагыш, ачы язмыш
Безгә язмасын.
Мин елмаеп айга карыйм,
Ул да елмайсын.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.
```

# Перевод

### Алмаз Хамзин / Марат Хоснуллин - Весна со скворцами и соловьями

```
С алым солнцем вместе с нами
Просыпаюсь я.
Всё улыбкой озарённым,
По-особому всегда.
Играет, нежит, ластится,
Лаская нас теплом.
Такого солнца больше нет
Нигде, мне кажется, кругом.

Со скворцами, соловьями,
С жаворонками, с гармонью.
Скучаю, скучаю по алому солнцу,
По родным моим краям.

У каждого свой край певучий,
Печальный и прекрасный.
Девушки там — красавицы,
Бесконечно прелестны.
И вёсны их, и осени
Особенно душевны.
К луне и солнцу тянется
Моя душа мятежная.

Со скворцами, соловьями,
С жаворонками, с гармонью.
Скучаю, скучаю по алому солнцу,
По родным моим краям.

В ваших краях луна печальна,
Отчего же так грустна?
Ведь она одна, совсем одна,
Совсем-совсем одна.
Пусть жёлтая тоска, горькая судьба
Минует стороной нас.
Я улыбаюсь ей, смотря на луну,
Пусть и она улыбнётся в ответ.

Со скворцами, соловьями,
С жаворонками, с гармонью.
Скучаю, скучаю по алому солнцу,
По родным моим краям.
```
