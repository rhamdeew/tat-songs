# Оригинал

### Фәнис Яруллин / Фәрит Хатыйпов - Сөбханалла

```
Өйда бүген зур тантана,
Һәрбер йөздә шатлык яна.
Аяклары җиргә тими -
Биеп йөри ата-ана.

Әбисе әйтә: «Сөбханалла!
Зур үскән бит безнең бала.
Күз тимәсен, күз тимәсен!
Кызыбыз таккан җиң төймәсен».

Зур куаныч әнкәсенә,
Сөеп карый бәбкәсенә:
"Үзе таккан! Үзе таккан!
Энә-җепне үзе тапкан!

Бармагына кадамаган,
Һичбер төше канамаган».
Гаҗәпләнгән ата кеше:
«Ай-яй, шәп-ди, - кызның эше,

Нык утырткан төймәсен-ди, -
Моңар һичкем тимәсен-ди, -
Пыяла белән капларбыз-ди, -
Истәлеккә сакларбыз», - ди.

Куанышкан һәркем шулай.
Инде эшләр китәр уңай:
Сөбханалла! Сөбханалла!
Күлмәгенә төймә таккан,
Унҗидедә бер кыз бала!
```

# Перевод

### Фанис Яруллин / Фарит Хатыйпов - Субханалла

```
В доме сегодня большое торжество,
На каждом лице радость сияет.
Ноги не касаются земли -
Танцуют от счастья отец и мать.

Бабушка говорит: «Субханалла!
Как выросла наша девочка.
Пусть сглаз не коснётся, не коснётся!
Дочка пришила пуговицу на рукав».

Великая радость для матери,
С любовью глядит на малышку:
"Сама пришила! Сама пришила!
Иголку с ниткой сама нашла!

Палец не уколола,
Ни одна дырочка не закровила".
Изумлён отец:
«Ай-яй, молодец, - дочкино дело,

Крепко пришила пуговицу, -
Пусть никто не тронет её, -
Под стекло положим, -
На память сохраним», - говорит.

Так радуется каждый.
Теперь дела пойдут хорошо:
Субханалла! Субханалла!
К рубашке пуговицу пришила,
В семнадцать лет девочка одна!
```
