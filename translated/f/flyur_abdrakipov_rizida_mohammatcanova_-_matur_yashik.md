# Оригинал

### Флюр Абдракипов / Ризида Мөхәммәтҗанова - Матур яшик

```
Дуслар белән бергә яшик
Бергә яшәү күңелле.
Бу дөньяда безнең гомер
Мизгел генә түгелме?

Чын дуслыкның кадерен белик,
Чит - ятлар дус түгелләр.
Дус - туганны хөрмәт итик,
Нурлы булсын күңелләр.

Җырлап яшик, гөрләп яшик,
Эшләребез уң булсын.
Кунакларга ишек ачык
Өстәлебез мул булсын.

Тормыш матур, дөньялар киң,
Ачык күңелләребез.
Чын барлыкка әйләнсеннәр
Якты өметләребез.
```

# Перевод

### Флюр Абдракипов / Ризида Мөхәммәтҗанова - Живём красиво

```
С друзьями вместе жить давай,
Вместе жить - отрада.
В этом мире наша жизнь -
Разве не миг, как награда?

Ценим силу верной дружбы,
Чужаки - не друзья.
Друга, словно родного, чтим,
Пусть светлеют сердца.

Жить бы с песней, жить в размахе,
Пусть дела идут в лад.
Двери гостям пусть будут открыты,
И столы - богатый ряд.

Жизнь прекрасна, мир широк,
Сердца наши - распахнуты.
Пусть в явь обращаются
Наши светлые мечты.
```
