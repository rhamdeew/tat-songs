# Оригинал

### Феликс Мортазин / Ленур Зәйнуллин - Серле сүзләр

```
Сөю сукмагыннан
Икәү атлап киткән
Алар гади, серле,
Сүзләр белә икән:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
«Сөям сине, бәгърем»
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Нинди көчле сүзләр!
«Сөям сине, бәгърем»
Нинди хисле сүзләр!

Галәм киңлегеннән
Сөю нуры иңгән
Серле сүзләр белән
Сөю үзе килгән:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
Сөям сине, бәгърем"
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Мәңге яшәр сүзләр!
«Сөям сине, бәгърем»
Янып яшнәр сүзләр,

Серле сүзләр белсәң
Сөю мәңге яшәр
Гади, серле сүзләр
Назлап сиңа дәшәр:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
«Сөям сине, бәгърем»
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Назлы күркәм сүзләр,
«Сөям сине, бәгърем»
Утка керткән сүзләр,

«Сөям сине, бәгърем»
Бигрәк алтын сүзләр,
«Сөям сине, бәгърем»
Җанга якын сүзләр!
```

# Перевод

### Феликс Мортазин / Ленур Зайнуллин - Таинственные слова

```
По тропе любви
Вдвоём они шагали,
Они простые, тайные
Слова лишь повторяли:

«Люблю тебя, душа моя»
Какие простые слова,
«Люблю тебя, душа моя»
Какие тайные слова!
«Люблю тебя, душа моя»
Какие сильные слова!
«Люблю тебя, душа моя»
Какие чувственные слова!

Из просторов мироздания
Свет любви снизошёл,
С тайными словами
Сама любовь пришла:

«Люблю тебя, душа моя»
Какие простые слова,
«Люблю тебя, душа моя»
Какие тайные слова!
«Люблю тебя, душа моя»
Вечно живущие слова!
«Люблю тебя, душа моя»
Горящие, сияющие слова!

Если знаешь тайные слова,
Любовь будет жить вечно,
Простые, тайные слова
Нежно тебе скажут:

«Люблю тебя, душа моя»
Какие простые слова,
«Люблю тебя, душа моя»
Какие тайные слова!
«Люблю тебя, душа моя»
Нежные прекрасные слова,
«Люблю тебя, душа моя»
В огонь ввергающие слова!

«Люблю тебя, душа моя»
Поистине золотые слова,
«Люблю тебя, душа моя»
Душе близкие слова!
```
