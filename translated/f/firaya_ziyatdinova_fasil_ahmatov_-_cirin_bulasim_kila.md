# Оригинал

### Фирая Зыятдинова / Фасил Әхмәтов - Җырың буласым килә

```
Ай кызыдай ачык йөзле,
Матур буласым килә.
Җил-давылга бирешмәслек
Батыр буласым килә.

Җилдән дә җитез буласым,
Уңган буласым килә.
Дуслар сөенерлек матур
Тормыш корасым килә.

Синең өчен тулган айның
Нуры буласым килә.
Күңелеңнән мәңге чыкмас
Җырың буласым килә.
```

# Перевод

### Фирая Зыятдинова / Фасил Әхмәтов - Твоей песней хочу быть

```
Светлоликая, как дочь луны,
Хочу я быть красивой.
Не сломиться под ветром и бурей -
Хочу я быть отважной.

Быстрей ветра хочу я быть,
Умелой хочу я быть.
Чтоб друзья радовались - красивую
Жизнь воздвигнуть хочу я.

Для тебя - быть лучом
Полной луны хочу я.
Не уйти из твоей души -
Твоей песней хочу я быть.
```
