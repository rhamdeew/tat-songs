# Оригинал

### Фирая Зыятдинова / Рөстәм Вәлиев - Бәхет өчен күп кирәкми

```
Бәхет өчен күп кирәкми,
Гел янәшә барсын юллар.
Бәхет өчен күп кирәкми,
Минем куллар - синең кулда,
Синең куллар - минем кулда.
Бәхет өчен күп кирәкми,
Артта калсын ап-ак эзләр.
Бәхет өчен күп кирәкми,
Минем күзләр - синең күздә,
Синең күзләр - минем күздә.
Тормыш - мизгел, бары мизгел,
Әллә ниләр өмет итмә.
Ахыргача бергә булсак,
Минем өчен шул да җитә,
Бәхет өчен шул да җитә.
```

# Перевод

### Фирая Зыятдинова / Рөстәм Вәлиев - Для счастья многого не надо

```
Для счастья многого не надо,
Пусть рядом тянутся пути.
Для счастья многого не надо,
Мои ладони - в твоих ладонях,
Твои ладони - в моих ладонях.
Для счастья многого не надо,
Пусть белый след останется позади.
Для счастья многого не надо,
Мои глаза - в твоих глазах,
Твои глаза - в моих глазах.
Жизнь - миг, всего лишь миг,
Не жди от нее несметных чудес.
Если будем вместе до конца,
Мне и этого довольно,
И для счастья этого довольно.
```
