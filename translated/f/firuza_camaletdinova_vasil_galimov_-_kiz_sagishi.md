# Оригинал

### Фирүзә Җамалетдинова / Вәсил Галимов - Кыз сагышы

```
Ак каеннар янына бер кыз килгән,
Сорамагыз нигә килгәнен.
Каеннарга сарылып,
Каеннарга сарылып
Елый кызлар,
Ятлар белән күрсә сөйгәнен.

Мендәренә чумып бер кыз елый,
Күз яшьләрен күрә ай гына.
Сөйгәне бит аны,
Сөйгәне бит аны
Калдырган да,
Китеп барган бүтән янына.

Ак каеннар янына бер кыз килгән
Йөрәккәен сагыш телгәндә.
Җанга сердәш эзләп,
Җанга сердәш эзләп
Йөри кызлар
Айсыз калган кара төннәрдә.
```

# Перевод

### Фирүза Джамалетдинова / Василь Галимов - Тоска девушки

```
К берёзкам белым пришла одна дева,
Не спрашивайте, зачем пришла она.
Обнимая берёзки,
Обнимая берёзки,
Плачет дева,
Говорит о любимом в тоске ночной.

Прижалась к груди одна дева, плачет,
Лишь месяц видит её слёзы.
Ведь он его,
Ведь он его
Оставил и
Ушёл в чужие края.

К берёзкам белым пришла одна дева
С тоской в сердце.
Ища в душе собеседника,
Ища в душе собеседника,
Ходит дева
В тёмные ночи без луны.
```
