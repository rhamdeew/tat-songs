# Оригинал

### Фәрит Аббясов / Эльвира Хаматнурова - Туган көнең белән! (өченче вариант)

```
Туган көнең белән сине тәбрик итәм.
Сизә күрмә,төннең салкынын!
Гомерең буе югалтма син.
Мәхәббәтнең кайнар ялкынын.

Мәхәббәтле кеше бик тиз чәчәк ата.
Бирә аңа саулык һәм шатлык.
Яратулар тәнгә шифа бирә.
Парлы яшәү үзе зур байлык.

Яннарыңда булса сөйгән ярың.
Фәрештәләр саклар күк йөзен.
Ике йөрәк бергә тибәр әле.
Гөлләр шиңмәс өчен һәр көзен.

Кояш нурын,сөеп сипсен сиңа!
Йөзеңнән һич нурлар сүнмәсен.
Матур тормышыңда,татлы сулыш.
Бер кайчанда синнән китмәсен.
```

# Перевод

### Фарит Аббясов / Эльвира Хаматнурова - С днём рождения! (третий вариант)

```
С днём рождения поздравляю тебя.
Не видеть горя, холода ночи!
Всю жизнь свою не теряй,
Тепло любви, горячий свет.

Человек, полный любви, быстро цветёт.
Даёт ему здоровье и радость.
Любовь исцеляет душу.
Счастливая любовь - великое богатство.

Если рядом любимый человек,
Ангелы хранят небеса.
Пусть два сердца бьются в унисон.
Цветы не вянут каждый день.

Солнечный свет льётся на тебя!
Пусть свет не гаснет на твоём лице.
В прекрасной жизни, сладком дыхании.
Пусть никогда не покинет тебя.
```
