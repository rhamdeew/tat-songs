# Оригинал

### Фирая Зыятдинова / Луиза Батыр - Фирая Зыятдинова / Луиза Батыр-Болгари - Бер син генә

```
Әллә нинди моңнар ишетәм
Сандугачлар тавышында.
Шатлыгым да,
Сагышым да
Тик син генә,
Бер син генә.

Әллә нинди чыңнар ишетәм
Салкын чишмә агышында.
Шатлыгым да,
Сагышым да
Тик син генә,
Бер син генә.

Еллар, моңнар... Гомерләрнең,
Күрәм, үтеп барышы ла...
Шатлыгым да,
Сагышым да
Тик син генә,
Бер син генә.
```

# Перевод

### Фирая Зыятдинова / Луиза Батыр-Болгари - Лишь ты одна

```
Слышу я неведомые напевы
В голосах лесных соловьёв.
Радость моя,
Печаль моя -
Только ты одна,
Лишь ты одна.

Слышу я неведомые звуки
В струях ледяного ключа.
Радость моя,
Печаль моя -
Только ты одна,
Лишь ты одна.

Годы, мелодии... Жизней
Вижу я бег непрестанный...
Радость моя,
Печаль моя -
Только ты одна,
Лишь ты одна.
```
