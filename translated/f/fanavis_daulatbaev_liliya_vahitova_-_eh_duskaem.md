# Оригинал

### Фәнәвис Дәүләтбаев / Лилия Вахитова - Эх, дускаем

```
Әй, дускаем, белче шуны -
Зарлы күңел - ул борчулы.
Авыр чакта сабыр булып
Көйләү кирәк тормышыңны.

Әй, дускаем, шуны аңла -
Вакытлыча монда бар да.
Булмый калмас кайгылар да,
Күңелеңә авыр алма.

Теләр идем һәрчак синең
Шат күңелле булганыңны.
Матур итеп яшим дисәң
Тәртипкә сал уйларыңны.

Алга куйган максат барда
Ярты юлда туктап калма.
Өмет белән яшә җирдә -
Яхшы булыр диеп бар да.

Көне җитми белеп булмый
Ни күрсәтер тормыш алда.
Җиңү яула көчең барда,
Уңыш юлдаш тырышларга.
```

# Перевод

### Фанавис Давлетбаев / Лилия Вахитова - Эх, друг мой

```
Да, друг мой, знай же ты это -
Сердце скорбное - оно скрупулезное.
В тяжелое время будь терпением
И настрой свою жизнь.

Да, друг мой, пойми же это -
Временное же всё здесь.
Не оставайся грустить,
Не тяготь сердце свое.

Желаю я всегда тебе
Радостного настроения.
Если хочешь красиво жить
Приведи мысли в порядок.

Если цель поставлена впереди
Не останавливайся наполовине.
Живи на земле с надеждой -
Всегда будет лучше.

Не угадать, не знает
Что принесет жизнь впереди.
Победа ждет, если силы есть
Успех сопровождает трудолюбивых.
```
