# Оригинал

### Фәнис Яруллин / Илнур Ганиев - Сагышларым (өченче вариант)

```
1. Сагышларым бирдем җилләргә,
Алып кит дип ерак илләргә.
Канатлары җилнең сыгылды.
Сагышларым җиргә,
Таралышып җилдә,
Сары чәчәк булып коелды.

2. Сагышларым салдым суларга,
Адашсын дип ерак юлларда,
Елга туктап калды ярсудан.
Сагышларым бар да,
Утырып калды ярда,
Ташулардан калган боз сыман.

3. Сагышларым аттым яшенгә,
Яндыр дидем, җиргә төшермә.
Яшен сүнде күкләр йөзендә.
Синнән башка беркем,
Күтәрә алмый диеп,
Сагышларым кайтты үземә.
```

# Перевод

### Фанис Яруллин / Илнур Ганиев - Мои печали (третий вариант)

```
1. Печали свои отдал я ветрам,
Унеси их, молил, в дальние края.
Но крылья ветра сложились,
И печали мои на землю,
Развеявшись по ветру,
Жёлтыми цветами упали.

2. Печали свои пустил я по водам,
Пусть заблудятся в дальних путях,
Но река остановилась у берега.
Все печали мои
Остались сидеть на берегу,
Словно лёд после разлива.

3. Печали свои метнул я в молнию,
Сожги их, молил, не дай упасть на землю.
Но молния погасла в небесах.
Кроме тебя никто,
Не может поднять их, говоря,
Печали мои вернулись ко мне.
```
