# Оригинал

### Фәрдия Сәфәргалиева / Ренат Фаттахов - Казансудан томан күтәрелә

```
Казансудан томан күтәрелә,
Күр әле, күр, җирдә ак болыт.
Күтәрелә күккә ак томаннар,
Җирнең ап-ак сәламе булып.

Казансуның ап-ак томаннары
Күтәрелә зәңгәр күкләргә.
«Ак томандай хыял таралды», - дип,
Кемнәр сөйләр икән кемнәргә?

Казансудан томан күтәрелә,
Тын иртәгә гүзәллек биреп.
Ак томаннар ак болытлар булып
Кайтыр әле илләрне гизеп...
```

# Перевод

### Фәрдия Сәфәргалиева / Ренат Фаттахов - Над Казансу поднимается туман

```
Над Казансу поднимается туман,
Посмотри, посмотри, на земле белый облак.
Белый туман поднимается к небу,
Как белоснежное приветствие земли.

Белые туманы Казансу
Поднимаются в синие небеса.
«Мечта, как белый туман, рассеялась», -
Кто и кому об этом расскажет?

Над Казансу поднимается туман,
Даря красоту тихому утру.
Белые туманы, став белыми облаками,
Ещё вернутся, странствуя по землям...
```
