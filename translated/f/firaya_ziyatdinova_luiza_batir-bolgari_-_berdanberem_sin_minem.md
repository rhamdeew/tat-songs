# Оригинал

### Фирая Зыятдинова / Луиза Батыр-Болгари - Бердәнберем син минем

```
Алтыным ла син, диеп,
Ялкыным ла син, диеп
Үстерәм бәбкәемне
Иркәләп, сөеп.
Улыкаем син минем,
Юлыма нур сибүче
Бәхетем минем.
Бәлли-бәү, бәлли-бәү.

Син минем ышанычым,
Бердәнберем дигәнем;
Син - бөркетем, лачыным,
Илгә бүләгем.
Матур булып үсәрсең,
Батыр булып үсәрсең,
Ак өметем син минем,
Бәхетем минем.
Бәлли-бәү, бәлли-бәү,
Бәлли-бәү, бәлли-бәү.
```

# Перевод

### Фирая Зыятдинова / Луиза Батыр-Болгари - Ты моя единственная

```
Ты - моё золото, жар мой,
С этой мыслью растю тебя,
Нежно глажу, обнимаю,
С любовью укачивая.
Ты - мой сынок, мой светлый луч,
Что путь мой озаряет,
Ты - счастье моё.
Баю-бай, баю-бай.

Ты - моя опора, вера,
Единственный, единственный;
Ты - мой орёл, мой сокол,
Подарок моей стране.
Красивым будешь вырастать,
И смелым будешь вырастать,
Ты - белая надежда,
Ты - счастье моё.
Баю-бай, баю-бай,
Баю-бай, баю-бай.
```
