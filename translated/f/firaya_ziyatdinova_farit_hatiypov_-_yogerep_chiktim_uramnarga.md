# Оригинал

### Фирая Зыятдинова / Фәрит Хатыйпов - Йөгереп чыктым урамнарга

```
Йөгереп чыктым урамнарга, -
Таңнар аткан чак иде:
Дөнья әле алсу иде,
Үзәннәргә зәңгәр өфрәк
Томан яткан чак иде.

Йөгереп чыктым урамнарга, -
Кояш батар чак җиткән:
Җиргә моңсу эңгер төшкән,
Бер дә үтмәс төсле иде,
Бу гомер кайчан үткән?!
```

# Перевод

### Фирая Зыятдинова / Фарит Хатыпов - Я выбежала на улицы

```
Я выбежала на улицы,
В миг, когда вставала заря:
Мир ещё розовел повсюду,
А в долах голубая дымка
Тумана лежала, храня.

Я выбежала на улицы,
Когда уж солнце к закату шло:
На землю села грустная сумерка,
Казалось, ей не кончиться вовсе,
Когда ж эта жизнь пронеслась так давно?.
```
