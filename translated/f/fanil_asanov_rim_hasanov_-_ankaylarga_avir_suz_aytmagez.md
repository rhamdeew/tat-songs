# Оригинал

### Фәнил Әсәнов / Рим Хәсәнов - Әнкәйләргә авыр сүз әйтмәгез

```
Әнкәйләргә авыр сүз әйтмәгез,
Караймасын матур йөзләре.
Йөрәкләрдә мәңге иркәләсен
Әнкәйләрнең сөю сүзләре.

Әнкәйләргә авыр сү әйтмәгез,
Хәтерләрдә берүк калмасын.
Назлы, нечкә ана күңелләре,
Кайгы-хәсрәт күреп янмасын.

Әнкәйләргә авыр сүз әйтмәгез,
Күзләреннән яшләр акмасын.
Әнкәй безгә тормыш бүләк итә,
Әнкәй ача бәхет капкасын.
```

# Перевод

### Фаниль Асанов / Рим Хасанов - Не говорите тяжёлых слов матерям

```
Не говорите тяжёлых слов матерям,
Пусть не видят они красивых лиц.
Пусть в сердцах всегда звучат
Слова любви матерей.

Не говорите тяжёлых слов матерям,
Пусть не останется в памяти.
Нежные, тонкие материнские сердца
Пусть не плачут от горя и утрат.

Не говорите тяжёлых слов матерям,
Пусть из глаз не льются слёзы.
Мать дарит нам жизнь,
Мать открывает ворота счастья.
```
