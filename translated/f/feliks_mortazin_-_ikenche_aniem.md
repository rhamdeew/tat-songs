# Оригинал

### Феликс Мортазин - Икенче әнием

```
Әле кайчан гына
Сине белми идем,
Ә бүген син минем
Икенче әнием!

Кайнана белән бер ел
Ике елга тиң диләр
Безгә ике йөз ел бергәләп
Яшәргә язган, димәк!

Йөзең кояш синең
Коймакларың тәмле,
Бәби карыйсың бар
Озак яшә яме!

Әнием икәү хәзер
Бәхетем тулы инде
Бу бәхетне мине яратып
Ходаем үзе бирде

Әнием диярмен
Улым диярсеңме?
Тудырган әнием
Үпкәләмәс әле!

Кайнана белән бер ел
Ике елга тиң диләр
Безгә ике йөз ел бергәләп
Яшәргә язган димәк!

Бу бәхетне мине яратып
Ходаем үзе бирде
```

# Перевод

### Феликс Мортазин - Вторая мама

```
Ведь совсем недавно
Я тебя не знала,
А сегодня ты моя
Вторая мама!

С невесткой вместе один год
Равны двум годам говорят,
Значит, нам вместе двести лет
Прожить написано!

Лицо твоё - солнце,
Блинчики твои вкусные,
На внуков смотришь
Долго живи же!

Мамы двое теперь
Счастье моё полно уже,
Это счастье, меня полюбив,
Сам Господь дал

Мамой называть буду,
Сыном назовёшь ли?
Родная мама
Не обидится ведь!

С невесткой вместе один год
Равны двум годам говорят,
Значит, нам вместе двести лет
Прожить написано!

Это счастье, меня полюбив,
Сам Господь дал
```
