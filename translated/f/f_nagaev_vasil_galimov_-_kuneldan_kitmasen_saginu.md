# Оригинал

### Ф. Нагаев / Вәсил Галимов - Күңелдән китмәсен сагыну

```
Өзелеп сагындым, шүңамы
Бар дөнья сарыдан ябынган.
Тузгытып сап-сары яфраклар.
Син киттең күңелем ярыннан.

Мәхәббәт диңгезен ялгызым
Җилкәнсез ничекләр кирчәрмен.
Айрылу, сагыну сагышын
Ничек соң бер үзем эчәрмен.

Бәгырьне телсә дә газаплар,
Юк инде, юк инде ялыну.
Күңелдән китмәсен бары тик,
Бары тик калсын шул сагыну.
```

# Перевод

### Ф. Нагаев / Василь Галимов - Пусть тоска по сердцу не уходит

```
Разорвалась, тоска по мне
Весь мир с тоски по мне замкнулся.
Закрутились сухие листья.
Ты ушёл из моего сердца.

Море любви, я одинок
Как без ветра корабли в беде.
Разлука, тоска и боль
Как же мне в одиночестве страдать.

Пусть бы сердце и мучилось,
Нет, нет, не устану.
Пусть не уходит из сердца,
Пусть остается тоска по тебе.
```
