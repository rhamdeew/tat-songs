# Оригинал

### Фәтхерахман Әхмәдиев - Чистай вальсы

```
Гүзәл Кама буйларында
Урман, күлләр, болыннар.
Кичке нурлар яктысында
Уйный серле дулкыннар.

Чистай вальсына
Ашкына хисләр.
Җыр белән уза
Чистайда кичләр.

Гөрли Кама, җырлый Кама, -
Ярда Чистай каласы.
Чистай үсә, җырга күчә
Шат, бәхетле авазы.

Яшәү шатлыгы
Чистай җырында,
Чистай сәгате
Дуслар кулында.

Агыйделнең суын алып,
Кама коя Иделгә;
Шат күңелле халкы белән
Чистай яши күңелдә.

Омтыла хисләр
Чистай ягына -
Якын дусларны
Күңел сагына.
```

# Перевод

### Фәтхерахман Әхмәдиев - Чистайский вальс

```
Вдоль прекрасной Камы
Лес, озёра, луга.
В свете вечерних лучей
Играют тайные волны.

К Чистайскому вальсу
Спешат наши чувства.
С песней проходят
Чистайские вечера.

Гудит Кама, поёт Кама, -
На берегу город Чистай.
Чистай растёт, в песню входит
Его радостный, счастливый голос.

Радость жизни
В песне Чистая,
Чистайский час
В руках друзей.

Взяв воды Агидели,
Кама льёт в Волгу;
С народом весёлым, душевным
Чистай живёт в сердце.

Тянутся чувства
К Чистаю -
Душа тоскует
По близким друзьям.
```
