# Оригинал

### Фәннур Сафин / Расил Әхәтов - Шәфкать туташына

```
Кар кызыдай килдең,
Төшләремә кердең,
Коткаручым булып нигәдер.
Елмай әле тагын,
Дөньяның шат чагын
Минем дә күрәсем килә бер.

Елмаерга ләкин
Иртәрәктер бәлки,
Иртәрәктер миңа... кем белә.
Көннәр китсә җирдән,
Төннәр килә икән,
Сердәш булып кала төн генә.

Шәфкать туташыдай,
Кояш сыйпап уза,
Бөтен хәят нурга күмелә.
Елмаерга безгә,
Соң түгелдер бәлки,
Соң түгелдер, туташ... Кем белә?!
```

# Перевод

### Фаннур Сафин / Расил Ахатов - Сестре Шафкать

```
Словно снежинка пришла ты,
В мои сны вошла,
Спасительницей моей став почему-то.
Улыбнуться ещё раз,
Мира светлую пору
Мне бы тоже увидеть хоть однажды.

Но улыбаться, быть может,
Рано ещё пока,
Рано мне ещё... кто знает.
Когда дни уходят с земли,
Ночи приходят, говорят,
Лишь ночь остаётся поверенной.

Как сестра Шафкать,
Солнце ласково скользит,
Вся жизнь в свет погружена.
Улыбаться нам,
Может, не поздно ещё,
Не поздно, сестра... Кто знает?!
```
