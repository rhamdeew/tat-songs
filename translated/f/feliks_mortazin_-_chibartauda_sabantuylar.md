# Оригинал

### Феликс Мортазин - Чибәртау сабантуйлар

```
Чибәртау сукмакларында
Ялан аяк эзләрем
Үрдәкләр белән узышып
Инешеңдә йөзгәнем!

Чибәртауда сабан туйлар
Яңгырый тирә яклар
Тояк белән җирне тырмап
Тояк белән җирне тырмап
Чабыштан кайта атлар.

Чибәртауның болыннары
Печән чапкан җирләрем
Тәүге мәхәббәт язларым
Чәчәк аткан җирләрем

Чибәртауда сабан туйлар
Яңгырый тирә яклар
Тояк белән җирне тырмап
Тояк белән җирне тырмап
Чабыштан кайта атлар

Чибәртауда каенлыкта
Җиләк җыйдым җәйләрен
Сез дусларым белән генә
Язмышымны бәйләдем

Чибәртауда сабан туйлар
Яңгырый тирә яклар
Тояк белән җирне тырмап
Тояк белән җирне тырмап
Чабыштан кайта атлар
```

# Перевод

### Феликс Мортазин - Сабантуй в Чибәртау

```
На тропинках Чибәртау
След босой стопы моей,
С утками наперегонки
В твоём омуте я плыл.

В Чибәртау - сабантуи,
Эхом полнится окрест,
Копыта землю роют,
Копыта землю роют -
С бегов несутся кони.

Луга Чибәртау - мои,
Где косил я сено в срок,
Весны первой любви моей,
Где цвели мои цветы.

В Чибәртау - сабантуи,
Эхом полнится окрест,
Копыта землю роют,
Копыта землю роют -
С бегов несутся кони.

В берёзовой роще Чибәртау
Летом ягоды я рвал,
Лишь с вами, друзья мои,
Я судьбу свою связал.

В Чибәртау - сабантуи,
Эхом полнится окрест,
Копыта землю роют,
Копыта землю роют -
С бегов несутся кони.
```
