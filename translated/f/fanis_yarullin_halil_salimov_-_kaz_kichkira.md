# Оригинал

### Фәнис Яруллин / Хәлил Сәлимов - Каз кычкыра

```
Кыйгак-кыйгак каз кычкыра,
Адашып калган микән?
Һаваларга оча алмый,
Канаты талган микән?

Кыйгак-кыйгак каз кычкыра,
Тавышы ерак китә.
Очкан кошлар өстән генә
Хәлләрен сорап китә...
Кыйгак-кыйгак-кыйгак...
```

# Перевод

### Фанис Яруллин / Халил Салимов - Гусь кричит

```
Га-га-га гусь кричит,
Неужели он сбился с пути?
Не может взлететь в небеса,
Неужели сломал крыло?

Га-га-га гусь кричит,
Его голос уходит вдаль.
Пролетающие птицы
Лишь о здоровье спрашивают...
Га-га-га-га...
```
