# Оригинал

### Фәтхелкадыйр Сөләймәнов / Галимҗан Таган (Гәлимйән Таған) - Зәки Вәлиди көе (җыры) /Эскадрон/

```
Батыр егет килә ат өстендә,
Аты уйный авызын тартканда.
Булат савыт кигән, сөңге тоткан,
Ташны яра (ташлар оча) уклар атканда.

Ак очкыннар оча ат авызыннан,
Ташлар сына аяк астында.
Алга чаба батыр, алга бара,
Чаң тузаны кала артында.

Ник ашыга батыр, ник ашкына,
Ни ишеткән батыр, ни күргән?!
Намусы бар егет нигә ятсын,
Ил читенә бүген яу килгән.
```

# Перевод

### Фатхелкадыйр Соляманов / Галимджан Таган (Галямьян Таган) - Песня Заки Валиди /Эскадрон/

```
Богатырь едет верхом на коне,
Конь играет, когда натянет поводья.
В стальной броне, с копьём в руке,
Разит камни (камни летят), пускает стрелы.

Белые искры летят из конской пасти,
Камни ломаются под ногами.
Вперёд мчится богатырь, вперёд устремляется,
Пыль от копыт остаётся позади.


Куда спешит богатырь, куда стремится,
Что слышал богатырь, что видел?!
Какой гордый юноша станет лежать,
Когда сегодня враг пришёл на нашу землю.
```
