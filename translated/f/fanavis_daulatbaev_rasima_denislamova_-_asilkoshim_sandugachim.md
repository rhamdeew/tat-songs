# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Асылкошым - сандугачым

```
Асылкошым минем - сандугачым,
Ничәнче кат сиңа талпынам.
Йөрәгемдә булган сөю хисен
Әйтергә соң никләр тартынам?

Асылкошым минем - сандугачым,
Уйларымда сине чакырам.
Күңелемә рәхәт булып китә
Күрмәсәм дә сине якыннан.

Асылкошым минем - сандугачым,
Күз алдымда һәрчак буй-сының.
Хыялымда синең белән яшим,
Тоясыңмы икән син шуны?

Асылкошым минем - сандугачым,
Күңел кат-кат сиңа тартыла.
Мин күптәннән үлеп гашыйк сиңа,
Тик әйтергә генә тартынам.
```

# Перевод

### Фанавис Давлетбаев / Рассимя Денисламова - Милый моя - соловей мой

```
Милый моя - соловей мой,
Саннарсыз тапкырлар сиңа өмит күтәрәм.
Йөрәгемдәге мәхәббәт хиссен
Әйтәргә соңга ни өчен тайпынам?

Милый моя - соловей мой,
Уйларымда сине чакырам.
Күңелем рәхим булып китә
Күрмәсәм дә сине яныннан.

Милый моя - соловей мой,
Күз алдымда һәрвакыт бул-сың.
Хыялымда синең белән яшәм,
Тоймасыңмы соң син шуны?

Милый моя - соловей мой,
Күңел чылбырлап сиңа тартыла.
Мин озак вакыттан үлгән гъашыйк сиңа,
Тик әйтәргә генә тайпынам.
```
