# Оригинал

### Феликс Мортазин - Төнге күктә якты йолдыз бар

```
Адашканда, юллар тапмаганда
Көнне караңгылык каплаганда
Адашканда төнге күккә кара
Төнге күктә якты юллар бар
Адашканда йолдызларга кара
Адашканда йолдызларга кара
Юл күрсәтеп яналар йолдызлар

Авыр чакта атлар тартмаганда
Җирне караңгылык каплаганда
Авыр чакта төнге күккә кара
Төнге күктә якты юллар бар
Авыр чакта йолдызларга кара
Авыр чакта йолдызларга кара
Синең өчен яналар йолдызлар

Онта калсаң әгәр тамырларың
Уйлый калсаң: «Мин биектә!» - диеп
Күтәрелеп төнге күккә кара
Төнге күктә сиңа җавап бар
Онытканда йолдызларга кара
Онытканда йолдызларга кара
Бик биектә яналар йолдызлар
```

# Перевод

### Феликс Мортазин - В ночном небе горит яркая звезда

```
Когда заблудишься, путей не находя,
Когда день накроет темнота,
Заблудившись, взгляни на ночное небо -
В ночном небе есть светлые пути.
Заблудишься - взгляни на звезды,
Заблудишься - взгляни на звезды,
Путь укажут, разгораясь, звезды.

Когда тяжко, кони не потянут,
Когда землю накроет темнота,
Когда тяжко - взгляни на ночное небо,
В ночном небе есть светлые пути.
Когда тяжко - взгляни на звезды,
Когда тяжко - взгляни на звезды,
Для тебя горят звезды.

Если вдруг забудешь свои корни,
Если подумаешь: «Я высоко!»
Поднимись и взгляни на ночное небо -
В ночном небе для тебя есть ответ.
Забыв - взгляни на звезды,
Забыв - взгляни на звезды,
Очень высоко горят звезды.
```
