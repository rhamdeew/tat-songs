# Оригинал

### Фәтхерахман Әхмәдиев / Тәлгать Вәлиев - Сине сөйдем, гүзәлем

```
Яшьлегем чәчәк атканда
Сине сөйдем, гүзәлем.
Тик бер үзем генә беләм
Үзәгем өзелгәнен.

Бакчада кошлар сайрады,
Кошларны күзәтмәдем;
Шул чагында сине уйлап
Өзелде үзәкләрем.

Аяз күктә ай-елмая,
Күңелләргә дәрт биреп.
Син дә ялгыз, мин дә ялгыз,
Ай йөзә безне күреп.
```

# Перевод

### Фәтхерахман Әхмәдиев / Тәлгать Вәлиев - Я любил тебя, красавица

```
Когда юность моя цвела,
Я любил тебя, красавица.
Лишь один я знаю это -
Сердце мое разрывалось.

В саду пели птицы звонко,
Я не слушал их тогда;
В ту пору, думая о тебе,
Разрывалась моя душа.

В ясном небе месяц улыбчив,
Сердцам он дает задор.
Ты одна, и я одинокий,
Месяц плывет, видя нас.
```
