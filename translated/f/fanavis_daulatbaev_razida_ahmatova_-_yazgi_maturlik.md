# Оригинал

### Фәнәвис Дәүләтбаев / Рәзидә Әхмәтова - Язгы матурлык

```
Чәчәккә күмелде, күмелде
Шомыртлар, сиреньнәр.
Җаннарны сихерли, сихерли
Туган җир, синең ямь!

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.

Тыңлап шул туймаслык, туймаслык
Кошларның сайравын.
Тылсымы язларның - язларның -
Агыла җанда моң.

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.

Үләннәр өстенә, өстенә
Чәчәкләр сибелгән.
Яратып туймаслык, туймаслык
Туган җир, синең ямь!

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.
```

```
Цветами покрылись, покрылись
Сирень, ландыши.
Души околдовает, околдовает
Родная земля, твой аромат!

Поле, луг, равнины, равнины
Зеленью наряжаются.
Природа оживает, оживает,
Сердца тоскуют.

Слушая ту неиссякаемую, неиссякаемую
Птичью песню.
Удивленье весны - весны -
По душе стелется тоска.

Поле, луг, равнины, равнины
Зеленью наряжаются.
Природа оживает, оживает,
Сердца тоскуют.

Над травами, над травами
Цветы рассыпались.
Любовью неиссякаемой, неиссякаемой
Родная земля, твой аромат!

Поле, луг, равнины, равнины
Зеленью наряжаются.
Природа оживает, оживает,
Сердца тоскуют.
```
