# Оригинал

### Фәридә Тимербаева / Роберт Тимербаев - Кантюковка юллары

```
Олы күлгә сикердем,
Рәхәтләнеп су кердем.
Яшь чакларны искә алып
Болыннарда йөгердем.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.

Бабай бакчасында үсә
Баланның иң татлысы.
Бу якларда гына үсә
Егет-кызның затлысы.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.

Юл төшкәндә керегез,
Бу авылны күрегез.
Кантюковка суын эчкәч
Шунда тартыр күңелегез.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.
```

# Перевод

### Фарида Тимербаева / Роберт Тимербаев - Дороги Кантюковки

```
В большое озеро бросилась,
С наслаждением в воду вошла.
Вспоминая юные годы,
По лугам я бежала.

Дороги Кантюковки,
Пролетают годы жизни.
Только в сердце берегу я
Песни молодости своей.

В саду у деда растёт
Самая сладкая рябина.
Только в этих краях растёт
Самый благородный парень и девушка.

Дороги Кантюковки,
Пролетают годы жизни.
Только в сердце берегу я
Песни молодости своей.

Заглянув на дорогу, заходите,
Эту деревню увидите.
Попив воды кантюковской,
Там сердце вас и потянет.

Дороги Кантюковки,
Пролетают годы жизни.
Только в сердце берегу я
Песни молодости своей.
```
