# Оригинал

### fanuza_zolkarnaeva - Фәнүзә Зөлкарнаева-Нәбиуллина / Рәзидә Әхмәтова - Үтте яшьлек юләрлеге

```
Үтте яшьлек юләрлеге,
Булмаганда йөгәнлеге.
Назларымнан торган язым,
Моңнарымны калдым язып.

Үтте яшьлек юләрлеге,
Бар дөньяның күңеллеге.
Сандугачлар җырын тыңлап,
Таңнар атты, йөрәк сызлап.

Үтте яшьлек юләрлеге,
Җитте алтын мизгелеге.
Каршы алды шулай көзем,
Ышанмадым, хәтта үзем.
Килеп җиткән икән көзем,
Ышанмадым хәтта үзем.
```

# Перевод

### Фәнүзә Зөлкарнаева-Нәбиуллина / Рәзидә Әхмәтова - Прошла шалость юности

```
Прошла шалость юности,
Когда не было узды.
Весна моя из нежных ласк,
Я песни грусти записал.

Прошла шалость юности,
Веселье целого мира.
Слушал песню соловьёв,
Рассветы - сердце ныло вновь.

Прошла шалость юности,
Золотой миг отзвенел.
Так осень встретила меня,
Не верил я, даже сам.
Выходит, осень уж пришла,
Не верил я, даже сам.
```
