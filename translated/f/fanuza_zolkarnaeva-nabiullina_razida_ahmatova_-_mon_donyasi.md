# Оригинал

### Фәнүзә Зөлкарнаева-Нәбиуллина / Рәзидә Әхмәтова - Моң дөньясы

```
Җырлары тын чишмә сыңнарыннан,
Сандугач моңнар бүләк итте.
Чәчәкләр дә назлы елмаешып,
Моң тулы карашларын сирпте.

Минем күңел моңнар дөньясы ул,
Көе хистән туҡылган дөньям.
Нурлы, чагу, матур шул дөньяда
Мин үземде ирекле тоям.

Әнкәй - бергенәмнең бишек җыры -
Моң дөньясының тәхетендә.
Күңелемнең җыр-моң дөньясыннан -
Ходай, аерма бәхетемнән!
```

# Перевод

### Фануза Зулькарнаева-Набиуллина / Разида Ахматова - Мир мелодий

```
Песни её - словно звон чистого родника,
Соловей подарил нам напевы свои.
Цветы, нежно улыбаясь,
Рассыпали взгляды, полные мелодий.

Моя душа - это мир мелодий,
Мой мир, сотканный из чувств.
В этом светлом, прекрасном, дивном мире
Я ощущаю себя свободной.

Матери колыбельная - единственная моя -
На престоле мира мелодий.
От мира песен и напевов моей души -
Боже, не отлучай меня от счастья!
```
