# Оригинал

### Флера Шәрипова / Зарема Әюпова - Бүгенгенең белик кадерен!

```
Бөреләргә тулган хисләремне
Бүген килеп, чәчәк аттырам.
Теләсәң, ян дөрләп минем белән,
Чаткылардан ялкын чакырам.

Борма-борма урау юллар белән
Сиңа килдем бүген, кадерлем.
Үкенүләр үткәннәрдә калсын,
Бүгенгенең белик кадерен!

Хәзер инде бер көч җиңә алмас,
Сүндерә алмас минем хисләрне.
Яннарыңда гел булырмын, иркәм,
Иртәләрен, көнен, кичләрен.

Борма-борма урау юллар белән
Сиңа килдем бүген, кадерлем.
Үкенүләр үткәннәрдә калсын,
Бүгенгенең белик кадерен!
```

# Перевод

### Флера Шәрипова / Зарема Әюпова - Ценить сегодняшний день!

```
Чувства, полные бутонов, я
Сегодня превращу в цветы.
Если хочешь - ярко пламеней со мной,
Из искры зову огонь в мечты.

По извилистым, витым дорогам
Я пришла к тебе, мой дорогой.
Пусть останутся в прошлом сожаленья,
Ценим день сегодняшний с тобой!

Ныне силу чувства не одолеть,
Не погасит их никто во мне.
Рядом быть с тобой я буду вечно -
Утром, днем и в сумеречной тьме.

По извилистым, витым дорогам
Я пришла к тебе, мой дорогой.
Пусть останутся в прошлом сожаленья,
Ценим день сегодняшний с тобой!
```
