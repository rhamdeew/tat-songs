# Оригинал

### Физәлия Дәүләтгәрәева / Фәнир Фәизов - Тагын кайтыр өчен китәмен

```
Туган ягым мине каршы ала,
Киңләп ачып яшел кочагын.
Тәрәзәдән күзен алмый әнкәй,
Сүндермичә кайнар учагын.

Авыл очындагы каен кызы
Чуклы шәлен изи ерактан.
Алып китә мине аякларым
Сукмаклардан, үзем яраткан.

Елга буйларында бәпкә үлән.
Ялан тәпи үстек йөгереп.
Чабып үткән наян балачакның
Артларыннан булмый өлгереп.

Чишмәләргә төшеп рәхмәт әйттем,
Без сыйланып үскән суына.
Чишмә сулы гүзәл туган яклар
Сагындырды гомер буена.

Таныш сукмаклардан тагын да бер
Яшьлек эзләреннән үтәмен.
Сау бул инде, газиз туган авыл.
Тагын кайтыр өчен китәмен.
```

# Перевод

### Физалия Даулятгараева / Фанир Фаизов - Уезжаю, чтобы вернуться вновь

```
Родная сторона встречает меня,
Раскрывая широко зелёные объятья.
От окна не отводит глаз мать,
Не гася пылающий очаг.

Берёзовая роща на краю села
Шелестит листвой издалека.
Уносят меня мои ноги
По тропинкам, что сам я любил.

Вдоль ручья растёт нежная трава.
Бегу босиком по лугу.
За промчавшимся счастливым детством
Не успеть мне угнаться.

Преклонившись к родникам, благодарил,
Водою которых мы росли.
Края родные с родниковой водой
Тосковать заставляли всю жизнь.

По знакомым тропинкам ещё раз
По следам юности пройду.
Будь здоров же, дорогое родное село.
Уезжаю, чтобы вернуться вновь.
```
