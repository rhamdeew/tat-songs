# Оригинал

### Фәнир Галимов - Кайтыйк дускай

```
Кайтыйк дускай, кайтыйк туган якка,
Ап - ак карлы, салкын кышларда.

Карлар ярып, урманарга барып,
Сагынгандыр безне алар да.

Кайтыйк дускай, кайтыйк туган якка,
Шаулап, гөрләп язлар килгәндә.

Тау башында яшьлек учаклары,
Кабызаек бозлар киткәндә.

Кайтыйк дускай, кайтыйк туган якка,
Җәй көненең җылы таңында.

Бергә үтик авылым урамыннан,
Тартып, уйнап тальян гармунын.

Кайтыйк дускай, кайтыйк туган якка,
Көзге җилләр каршы иссә дә.

Безнең моңны көтә каенкайлар,
Болытларга кадәр үссә дә.
```

# Перевод

### Фанир Галимов - Вернемся, друг

```
Вернемся, друг, вернемся на родину,
В белоснежные, прохладные зимы.

Пробивая снега, идя в леса,
Наверное, скучают и они по нас.

Вернемся, друг, вернемся на родину,
Когда придут гремящие, шумящие лета.

На вершине горы молодые чайки,
Затанцуем, когда уйдут льды.

Вернемся, друг, вернемся на родину,
В тепло утра летнего дня.

Вместе пройдем по улице нашей деревни,
Тянем, играем на тальян-гармони.

Вернемся, друг, вернемся на родину,
Даже если против веют осенние ветры.

Наши ели ждут этого,
Пока они не вырастут до облаков.
```
