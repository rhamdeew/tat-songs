# Оригинал

### Фәннур Сафин / Ринат Гобәйдуллин - Туган җиремә (өченче вариант)

```
Кай тарафка гына күз атсам да,
Чишмә сыман ургып моң чыгар.
Имәннәрең гүя юл югалтып
Туктап калган моңсу юлчылар.

Бу юллардан күпме сагышлылар,
Бәхетлеләр җырлап узгандыр.
И Туган җир, мине шатлыкларың,
Сагышларың белән сызландыр.

Кырларыңнан җыям моңнарымны,
Бердәнберем - туган төбәгем.
Китсәм дә мин үзем еракларга,
Синдә кала минем йөрәгем.
Синдә кала минем йөрәгем...
```

# Перевод

### Фаннур Сафин / Ринат Гобәйдуллин - Родной земле (третий вариант)

```
Куда ни брошу взгляд,
Мелодия бьет ключом родника.
Твои дубы, будто путь потеряв,
Стоят печальными странниками.

По этим дорогам сколько грустных,
Сколько счастливых с песней прошло.
О Родина, твоими радостями,
Твоей печалью дай мне прозвучать.

С твоих полей собираю мелодии,
Мой единственный - родной мой край.
Даже если уеду я в дальние страны,
В тебе останется моё сердце.
В тебе останется моё сердце...
```
