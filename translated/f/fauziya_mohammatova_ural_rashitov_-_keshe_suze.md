# Оригинал

### Фәүзия Мөхәммәтова / Урал Рәшитов - Кеше сүзе

```
Белсәк тә без янәшәдә
Җиде ятлар йөргәнне
Сизми калдык арабызга
Кеше сүзе кергәнне.

Шиәде мәхәббәт гөлләре
Тиде җиде ят күзе.
Җиде ятларга калдырып
Аерды кеше сүзе.

Кеше сүзе, кеше сүзе.
Җиде ят итте безне.
Җиде ятлар да кавыша,
Өзмик өметебезне.

Адашып калды мәхәббәт
Җиде юлның чатында.
Җиде ятларга әйләндек
Кеше сүзе хакында.
```

# Перевод

### Фаузия Мухамматова / Урал Рашитов - Человеческое слово

```
Хотя знали мы, что рядом
Семь чужаков бродили,
Не почувствовали меж нами
Человеческое слово проникло.

Завяли цветы любви
От глаз семи чужих.
Оставив семи чужакам
Разлучило нас человеческое слово.

Человеческое слово, человеческое слово.
Сделало нас семью чужих.
Даже семь чужих соединятся,
А нашу надежду не разорвать.

Потерялась любовь
На развилке семи дорог.
В семь чужаков превратили нас
Из-за человеческого слова.
```
