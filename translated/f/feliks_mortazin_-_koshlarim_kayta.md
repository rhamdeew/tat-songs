# Оригинал

### Феликс Мортазин - Кошларым кайта

```
Кайталар кошлар
Сайрарга таңда,
Оя корырга,
Чәчәкле язда!

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!

Кайталар кошлар
Канат кагынып,
Шашып, ашкынып
Язны сагынып,

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!

Кайткан кошкайлар
Кире китмәгез
Кошчыккайларны
Ятим итмәгез!

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!
```

# Перевод

### Феликс Мортазин - Возвращаются мои птицы

```
Возвращаются птицы
Петь на рассвете,
Гнёзда свивать
В цветущую весну!

Будь милосерден,
Не стреляй в птиц!
Не лишай свободы,
В клетку не запирай!

Возвращаются птицы,
Взмахом крыла,
Сумасбродно, спешно,
Скучая по весне,

Будь милосерден,
Не стреляй в птиц!
Не лишай свободы,
В клетку не запирай!

Вернувшиеся пташки,
Не улетайте прочь,
Птенчиков малых
Сиротами не оставляйте!

Будь милосерден,
Не стреляй в птиц!
Не лишай свободы,
В клетку не запирай!
```
