# Оригинал

### Фәүзия Рәхимгулова / Мөфтәдин Гыйләҗев - Тәүге нур

```
Йөрәкнең иң тәүге нуры,
Йөрәкнең иң тәүге нуры,
Кадерлем минем.
Өзелеп сөйгәнен күңелемнең,
Өзелеп сөйгәнен күңелемнең,
Син белсәң иде, син белсәң иде.

Әнә бер ялгыз күбәләк,
Әнә бер ялгыз күбәләк
Гизә болыннан.
Ул бит минем мәхәббәтем,
Ул бит минем мәхәббәтем
Очты кулымнан, очты кулымнан.

Былбыл сайрый талга кунып,
Былбыл сайрый талга кунып,
Белче кадерен.
Сине өзелеп сагынучы,
Сине өзелеп сагынучы
Мин ул, кадерлем, мин ул, кадерлем.
```

# Перевод

### Фаузия Рахимгулова / Муфтадин Гилязев - Первый луч

```
Самый первый луч сердца,
Самый первый луч сердца,
Моя родная.
Тот, кого душа любит всем сердцем,
Тот, кого душа любит всем сердцем,
Если бы ты знала, если бы ты знала.

Вот одинокая бабочка,
Вот одинокая бабочка
Летит над лугом.
Ведь это моя любовь,
Ведь это моя любовь
Улетела из рук, улетела из рук.

Соловей на иве распевается,
Соловей на иве распевается,
Знаешь, как дорог он.
Тот, кто скучает по тебе всей душой,
Тот, кто скучает по тебе всей душой
Это я, родная, это я, родная.
```
