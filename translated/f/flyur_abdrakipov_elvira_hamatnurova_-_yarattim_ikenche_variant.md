# Оригинал

### Флюр Абдракипов / Эльвира Хаматнурова - Яраттым (икенче вариант)

```
Мин сине яраттым,
Карагач күзләргә.
Мәхәббәткә чумдым,
Чумгандай күлләргә.

Кушымта:
Бар дөнья әйләнә
Тик сине уратып.
Хисләрем бәйләнә
Тик сине яратып.

Мәхәббәт очкыны
Читләргә тимәсен.
Ярату хисләрем
Сүнмәсен, сүнмәсен.

Йөрәгеңә синең
Ут-ялкын өрермен.
Сөюемнең назын
Тик сиңа бирермен.
```

# Перевод

### Флюр Абдракипов / Эльвира Хаматнурова - Любимая (второй вариант)

```
Я тебя так люблю,
В твои карие глаза.
В любовь с головой окунулась,
Как в волны ныряла б.

Припев:
Весь мир вращается,
Лишь тебя обнимая.
Чувства мои связываются,
Лишь тебя любя.

Пламя любви
Пусть не коснется чужих.
Чувства мои любви
Пусть не гаснут, не гаснут.

В твоё сердце
Огонь пламени раздую я.
Нежность моей любви
Тебе лишь подарю.
```
