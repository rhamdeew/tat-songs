# Оригинал

### Фирая Зыятдинова / Луиза Батыр-Болгари - Китмим әле яшьлегемнән

```
Бер-бер артлы еллар уза,
Көмеш өстәп чәчләремә.
Ә шулай да мин ышанам
Яшен булып яшьнәремә.

Без бит җиргә якты, җылы,
Ямь өстәргә яратылган.
Китмим әле яшьлегемнән -
Яз гөлләре арасыннан.

Уйлар әле шактый зурдан -
Язлар алып килү көздә.
Алда әле ак өметләр,
Йөрәк һаман унсигездә.

Без бит җиргә якты, җылы,
Ямь өстәргә яратылган.
Китмим әле яшьлегемнән -
Яз гөлләре арасыннан.

Якты кояш, ал гөлләргә
Килә һаман сөенәсе;
Елый-көлә, яна-көя
Яратасы, сөеләсе!

Без бит җиргә якты, җылы,
Ямь өстәргә яратылган.
Китмим әле яшьлегемнән -
Яз гөлләре арасыннан.
```

# Перевод

### Фирая Зиятдинова / Луиза Батыр-Болгари - Ещё не уходила я из молодости

```
Год за годом проходят,
Серебром украшая мои волосы.
Но всё же я верю
В молнию, сверкающую в моих мечтах.

Ведь мы рождены нести свет и тепло
На эту землю, украшать её красотой.
Ещё не уходила я из молодости -
Из среди весенних цветов.

Мысли пока ещё велики -
Весны, приносящие осенью.
Впереди ещё светлые надежды,
Сердце всё в том же ожидании.

Ведь мы рождены нести свет и тепло
На эту землю, украшать её красотой.
Ещё не уходила я из молодости -
Из среди весенних цветов.

Яркое солнце, к алым цветам
Всё приходит с радостью;
Плачет-смеётся, горит-печалится
Для любимых, для любимых!

Ведь мы рождены нести свет и тепло
На эту землю, украшать её красотой.
Ещё не уходила я из молодости -
Из среди весенних цветов.
```
