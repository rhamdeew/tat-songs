# Оригинал

### Фәннур Сафин / Зиннур Гыйбадуллин - Кояшлы Ватаным

```
Без атлаган юлдан кояшлырак,
Яктырак юл бармы дөньяда!
Яшьлек өчен якты юллар ачык,
Дәртләндерә безне замана.

Галәмнәргә юллар ачык,
Юллар дәшә, юллар чакыра.
Яңа үрләр яулый яшьлек
Тыныч еллар хакына.

Яңа серләр ача алсаң гына,
Бу Җир шары, дөнья матур ул.
Киң офыклар дәшә безне алга -
Юлга чыксаң, яшьлек, батыр бул!

Галәмнәргә юллар ачык,
Юллар дәшә, юллар чакыра.
Яңа үрләр яулый яшьлек
Тыныч еллар хакына.
```

# Перевод

### Фаннур Сафин / Зиннур Гыйбадуллин - Солнечная Родина моя

```
Нет пути светлей на свете,
Чем дорога, что прошли!
Для молодости светлые пути открыты,
Вдохновляет нас эпоха наших дней.

К мирозданью путь открыт,
Дороги манят, дороги зовут.
Новых вершин достигнет молодость
Во имя мирных лет.

Лишь тайны новые открой,
И мир прекрасен, Земля прекрасна.
Широкие просторы манят нас вперёд -
В путь выходи, о юность, смелой будь!

К мирозданью путь открыт,
Дороги манят, дороги зовут.
Новых вершин достигнет молодость
Во имя мирных лет.
```
