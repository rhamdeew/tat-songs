# Оригинал

### Фәннур Сафин / Зөфәр Хәйретдинов - Хуш, Сәрия!

```
Кайда бармас, ниләр күрмәс
Инде минем бу башым?
Сине эзләп юлга чыктым,
Синең белән хушлашып.

Сау бул инде, хуш, Сәрия!
Мәңгелек мәхәббәтем.
Сагышымның иң татлысы
Төзелмәс җәрәхәтем.

Минем язмыш мәхәббәтем
Тик синең кулларыңда.
Сине эзләп йөрим ялгыз
Киек каз юлларында.

Сау бул инде, хуш, Сәрия!
Мәңгелек мәхәббәтем.
Сагышымның иң татлысы
Төзелмәс җәрәхәтем.

Эзли торгач калдым инде
Диңгезләргә җилкәнсез.
Сине эзләп йөремдем мин -
Син йөрәктә икәнсең.

Сау бул инде, хуш, Сәрия!
Мәңгелек мәхәббәтем.
Сагышымның иң татлысы
Төзелмәс җәрәхәтем.
```

# Перевод

### Фаннур Сафин / Зуфар Хайретдинов - Прощай, Сария!

```
Куда не пойдёт, чего не увидит
Теперь эта голова моя?
Вышел в путь, тебя разыскивая,
С тобой прощаясь навсегда.

Будь здорова же, прощай, Сария!
Моя вечная любовь.
Слаще всех моих печалей,
Незаживающая рана.

Моя судьба и моя любовь
Лишь в твоих руках одних.
Ищу тебя, брожу я в одиночку
По диким гусиным путям.

Будь здорова же, прощай, Сария!
Моя вечная любовь.
Слаще всех моих печалей,
Незаживающая рана.

Искать продолжив, я остался
Без паруса среди морей.
Искал тебя я повсюду -
А ты в сердце у меня.

Будь здорова же, прощай, Сария!
Моя вечная любовь.
Слаще всех моих печалей,
Незаживающая рана.
```
