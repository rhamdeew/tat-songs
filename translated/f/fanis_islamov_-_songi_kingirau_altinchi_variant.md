# Оригинал

### Фәнис Исламов - Соңгы кыңгырау (алтынчы вариант)

```
Соңгы кыңгырау яңгырый бүген,
Яңа юл яра гомер агышы.
Соңгы кыңгырау - әнкәй догасы -
Изге юл тели аның тавышы.
Очар кошлар без: ямьле бу язда,
Юл җиңел булмас, сынаулар алда.
Киңәш бирергә, ярдәм итәргә
Абый-апалар булмаслар янда.
Мәктәп елларын онытмыйк, дуслар,
Туган оясын онытмый кошлар.
Соңгы кыңгырау еракта калыр,
Еллар узгач та колакта чыңлар,
Моңлы җыр булып колакта чыңлар.
```

# Перевод

### Фәнис Исламов - Последний звонок (альтернативный вариант)

```
Последний звонок звенит сегодня,
Новый путь откроет жизнь.
Последний звонок - мольба учителя -
Желает ему доброго пути.
Улетаем мы, как птицы, в тёплом лете,
Путь не легок, испытания впереди.
Советом поделиться, помочь
Старших братьев и сестёр не будет рядом.
Годы в школе не забудем, друзья,
Гнездо родное птицы не забудут.
Последний звонок останется позади,
Спустя годы в ушах продолжит звенеть,
Тихой песней в ушах будет звенеть.
```
