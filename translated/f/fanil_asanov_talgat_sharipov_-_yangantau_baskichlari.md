# Оригинал

### Фәнил Әсәнов / Тәлгать Шәрипов - Янгантау баскычлары

```
Яллар итә-итә менәм
Янгантау баскычларын.
Йөрәк сереңне ачырга,
Йөрәк сереңне ачырга,
Булмады ачкычларым.

Янгантау баскычларыннан
Эзләреңне эзләдем.
Син киткән юлларга карап,
Син киткән юлларга карап,
Талды инде күзләрем.

Янгантау баскычларының
Кемнәр койган корычын?
Әллә ниләр бирер идем,
Әллә ниләр бирер идем,
Сине бер күрер өчен,
Сине бер күрер өчен.
```

# Перевод

### Фаниль Асатов / Талгат Шарипов - Ступени Янгантая

```
Иду-иду по ступеням
Поднимаюсь по Янгантай.
Сердце твоё открыть,
Сердце твоё открыть,
Нет у меня ключей.

По ступеням Янгантая
Шаг за шагом искала.
Глядя на пути, куда ушёл,
Глядя на пути, куда ушёл,
Устали мои глаза.

Кто ковал железо
Ступеней Янгантая?
Что бы я ни отдала,
Что бы я ни отдала,
Чтобы увидеть тебя,
Чтобы увидеть тебя.
```
