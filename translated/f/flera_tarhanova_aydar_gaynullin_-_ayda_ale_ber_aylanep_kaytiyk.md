# Оригинал

### Флера Тарханова / Айдар Гайнуллин - Әйдә әле, бер әйләнеп кайтыйк...

```
Әйдә әле, бер әйләнеп кайтыйк
Кичке урманнарны, икәүләп.
Өебез тулы чиксез якты хисләр
Ятмасыннар әле биләүдә.

Безнең йөзләрдәге бәхет нуры
Балкыш булып төнгә таралсын, -
Аерым йөреп күпме тилмергәннәр
Үз ярарын эзләп табалсын.
```

# Перевод

### Флера Тарханова / Айдар Гайнуллин - Давай вернемся, сделав круг...

```
Давай вернемся, сделав круг,
По вечерним лесам - вдвоем.
Пусть в доме светлые чувства наши
Не будут больше в пелене.

Пусть счастья свет в наших лицах
Разливается в ночь сияньем, -
Чтоб те, кто порознь долго маялся,
Нашли свою рану, свое признанье.
```
