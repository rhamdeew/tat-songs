# Оригинал

### Феликс Мортазин - Сабыйларым-нарасыйларым

```
Илләребез берүк тыныч булсын
Кояш тигез сипсен җиргә нурын
Нарасыйлар белми үсмәсеннәр
Әти-әни дигән сүзнең барын

Сабыйларым, нарасыйларым
Белмәгез җирдә сагыш барын
Шат көлүгә берни җиталмас
Шат хисләрне хәсрәт җиңалмас

Киләчәктә яшик үкенмичә
Туры карап яшикче күзләргә
Нарасыйлар еласыннар бары
Хисләнгәндә җылы сүзләргә!

Сабыйларым, нарасыйларым
Белмәгез җирдә сагыш барын
Шат көлүгә берни җиталмас
Шат хисләрне хәсрәт җиңалмас

Әти-әни дигән олы исем
Һәркемгә дә әйтү насыйп булсын
Куеныңда, олы йөрәгеңдә
Сабыеңа һәрчак урын булсын

Сабыйларым, нарасыйларым
Белмәгез җирдә сагыш барын
Шат көлүгә берни җиталмас
Шат хисләрне хәсрәт җиңалмас
```

```

Солнце пусть равномерно льёт свой свет на землю.
Пусть малыши не знают, что такое расти,
Всё значение слов "мама" и "папа".

Мои малыши, мои детки,
Не знайте печалей земных.
Ничто не сможет помешать смеху радостному,
Грусть не победит чувства счастливые.

В будущем живите без сожалений,
Прямо в глаза смотрите, пока живёте.
Пусть малыши плачут только
На теплые слова, когда тронуты чувствами!

Мои малыши, мои детки,
Не знайте печалей земных.
Ничто не сможет помешать смеху радостному,
Грусть не победит чувства счастливые.

Великое имя "мать и отец"
Пусть каждому будет сказано.
В твоей душе, в твоём большом сердце,
Пусть всегда найдётся место для твоего ребёнка.

Мои малыши, мои детки,
Не знайте печалей земных.
```
