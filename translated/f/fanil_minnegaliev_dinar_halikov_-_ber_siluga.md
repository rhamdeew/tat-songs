# Оригинал

### Фәнил Миңнегалиев / Динар Халиков - Бер сылуга

```
Көтмәгәндә сылукаем
Очрадың юлларымда.
Җитәкләшеп күп йөредек
Якты күл буйларында.

Безне күреп серләштеләр
Яр буенда камышлар.
Әйтерсеңдә алар өчен
Без күптәнге танышлар.

Якты күлнең дулкыннары
Тирбәләләр тын гына.
Мәхәббәтнең утларында
Янмый икән кем генә?!

Безне күреп серләштеләр
Яр буенда камышлар.
Әйтерсеңдә алар өчен
Без күптәнге танышлар.

Бер ялгызым утырамын
Якты күл буйларында.
Сылукаем сихерләде
Чыкмассың уйларымнан.

Безне күреп серләштеләр
Яр буенда камышлар.
Әйтерсеңдә алар өчен
Без күптәнге танышлар.
```

# Перевод

### Фаниль Миннегалиев / Динар Халиков - Одной слезе

```
Неожиданно скользнула
Ты в мою тропинку.
Долго мы бродили вдвоём
По берегам светлого озера.

Увидев нас, зашептались
Камышовые заросли на берегу.
Но ведь для них
Мы были знакомы издавна.

Волны светлого озера
Нежно колышутся.
Разве кто-нибудь
Не горит в огне любви?!

Увидев нас, зашептались
Камышовые заросли на берегу.
Но ведь для них
Мы были знакомы издавна.

Сижу я одиноко
По берегам светлого озера.
Слеза моя околдовала,
Не выходит из мыслей.

Увидев нас, зашептались
Камышовые заросли на берегу.
Но ведь для них
Мы были знакомы издавна.
```
