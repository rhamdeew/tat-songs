# Оригинал

### Феликс Мортазин - Кулъяулык чигә кызлар

```
Алтыннандыр җепләре,
Көмештәндер җепләре
Кулъяулык чигә кызлар
Ил сагында егетләре
Исән-имин кайтсыннар дип
Кулъяулык чигә кызлар!

Бераз моңсу күзләре
Әмма көләч йөзләре
Кулъяулык чигә кызлар
Тиздән кайтыр егетләр дип
Җырлый, җырлый кич утырып
Кулъяулык чигә кызлар!

Изгедер теләкләре
Сабырдыр йәрәкләре
Кулъяулык чигә кызлар
Армиядә егетләре
Илләр имин булыр диеп
Кулъяулык чигә кызлар!
```

# Перевод

### Феликс Мортазин - Девушки вышивают платочки

```
Нити - будто из злата,
Нити - будто из серебра,
Девушки платочки вышивают -
Парни стражей у родной земли.
Чтобы живыми, целыми вернулись,
Девушки платочки вышивают!

Чуть печальны их глаза,
Но лица светлые, смеются,
Девушки платочки вышивают.
«Скоро парни возвратятся», -
Песню, песню напевая у огня,
Девушки платочки вышивают!

Святы их пожеланья,
Терпеливы их сердца,
Девушки платочки вышивают.
Парни в армии сейчас -
«Пусть страны будут в мире», -
Девушки платочки вышивают!
```
