# Оригинал

### Фатыйх Кәрим / Әнвәр Бакиров - Сагындым (беренче вариант)

```
Идел буйларына тагын
Әйләнеп кайтсам иде,
Ал чәчәкләр арасында
Бер җырлап ятсам иде.

Җир җиләге белән мине
Иркәм сыйласа иде,
Элеккечә бер иркәләп,
Башым сыйпаса иде.
```

# Перевод

### Фатыйх Карим / Анвар Бакиров - Соскучился (первый вариант)

```
Вернуться бы к берегам Идели
И снова пройти теми тропами,
Среди алых цветов, распластавшись,
Напев свой пропеть под ветрами.

Чтоб земляникой меня угостила
Нежно любимая, милая,
Как прежде прижала с улыбкой
И голову тихо погладила.
```
