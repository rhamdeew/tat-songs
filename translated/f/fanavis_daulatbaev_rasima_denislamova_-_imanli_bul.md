# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Иманлы бул

```
Байлык диеп хәрам җыйма,
Бул гөнаһтан куркулы.
Күрмәгәндәй читләп үтмә-
Егылганга суз кулны.

Игелекләр кылып яшәп
Син тормышта бул данлы.
Туры юлдан алга атла-
Бул һәрвакыт иманлы!

Синнән ярдәм өмет иткән
Мохтаҗ кешеләр бардыр.
Явызлык белән алдырма,
Яхшылык белән алдыр!

Игелекләр кылып яшәп
Син тормышта бул данлы.
Туры юлдан алга атла-
Бул һәрвакыт иманлы!

Кыямәт көн бәндәләргә
Тотасы булыр җавап.
Һәр көнеңдә рәхмәт ишет,
Аз булса да җый савап.

Игелекләр кылып яшәп
Син тормышта бул данлы.
Туры юлдан алга атла-
Бул һәрвакыт иманлы!
```

# Перевод

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Верь

```
Не собирай незаконно богатство,
Берегись от греха.
Не проходи мимо, как незнакомый,
Протяни руку согбенному.

Совершая добрые дела,
Ты будешь уважаем в жизни.
Шагай вперед по праведному пути -
Будь всегда верующим!

Люди, нуждающиеся в помощи,
Будут надеяться на тебя.
Не отплачивай злом,
Отплачивай добром!

Совершая добрые дела,
Ты будешь уважаем в жизни.
Шагай вперед по праведному пути -
Будь всегда верующим!

В день Суда рабам
Придется держать ответ.
Каждый день слушай милость,
Собирай заслуги, пусть даже немного.

Совершая добрые дела,
Ты будешь уважаем в жизни.
Шагай вперед по праведному пути -
Будь всегда верующим!
```
