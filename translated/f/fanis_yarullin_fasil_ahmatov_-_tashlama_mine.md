# Оригинал

### Фәнис Яруллин / Фасил Әхмәтов - Ташлама мине

```
Ташлама мине, ташлама,
Рәхәт син барны тою,
Актык көннәремә чаклы
Ташлама мине, Сөю!

Ташлама мине, ташлама,
Гел минем җанда уян,
Соңгы сулышыма кадәр
Ташлама мине, Хыял!

Ташлама мине, ташлама,
Гел тор сафлыгың чәчеп,
Йөрәгем соң кат типкәнче
Ташлама мине, Яшьлек!
```

# Перевод

### Фанис Яруллин / Фасил Ахметов - Не оставляй меня

```
Не оставляй меня, не оставляй,
Радость всего познавать с тобой,
До самых светлых моих дней
Не оставляй меня, Любовь!

Не оставляй меня, не оставляй,
Всегда в душе моей живи,
До последнего моего дыханья
Не оставляй меня, Мечта!

Не оставляй меня, не оставляй,
Всё чистоту свою храня,
Пока сердце стучит в последний раз
Не оставляй меня, Юность!
```
