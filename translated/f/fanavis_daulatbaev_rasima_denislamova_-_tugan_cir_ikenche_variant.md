# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Туган җир (икенче вариант)

```
Туган җирдән тәпи басып
Атлап киткәнбез.
Шушы җирдә авыр дими
Гомер итәбез.

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!

Җирдә кылган изгелектән
Безне танырлар.
Туган җирдә тармакланган
Безнең тамырлар.

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!

Туган җирем - сызылыплар
Таңнар аткан җир.
Кадерле син гомерлеккә
Бәхет тапкан җир!

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!
```

# Перевод

### Фанавис Даулатбаев / Рашима Денисламова - Родная земля (второй вариант)

```
С родной земли мы сорвались
И поскакали прочь.
На этой земле не легко
Жизнь свою пройти.

Земля, где черпали силы, живя,
Земля, где чувствовали радость.
Для сердец всегда близка,
Драгоценная родная земля!

По благим делам, что на земле совершили,
Нас узнают.
В родной земле разрослись
Наши корни.

Земля, где черпали силы, живя,
Земля, где чувствовали радость.
Для сердец всегда близка,
Драгоценная родная земля!

Родная моя земля - со следами
Рассветов, что встали.
Ценна ты, на всю жизнь
Счастье обретшее земля!

Земля, где черпали силы, живя,
Земля, где чувствовали радость.
Для сердец всегда близка,
Драгоценная родная земля!
```
