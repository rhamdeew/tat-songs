# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Тальян гармун (беренче вариант)

```
Урам буйлап егет узды
Тальян гармун сыздырып.
Моңына күмеп авылны,
Кызларны кызыктырып.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.

Нигә соң мин былбыл түгел
Көмеш айлы кичләрдә.
Яшьлектә калган сөйгән яр
Ник кермисең төшләргә.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.

Тыңла, дустым, яңгырап чыга
Тальянның тавышлары.
Шул тальянның тавышында
Гомернең агышлары.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.
```

# Перевод

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Тальян гармун (первый вариант)

```
По улице прошёл юноша
С играющим тальян-гармонью.
Во всю округу звучало,
Девушек прельщало.

Тальян-гармонь, тальян-гармонь,
Откуда такая грустная ты?
Хочу вернуться в юность,
Чтобы снова тебя послушать.

Почему же я не возвращаюсь
В серебряные лунные ночи.
Любимая, оставленная в юности,
Почему не войдешь ты в мои сны.

Тальян-гармонь, тальян-гармонь,
Откуда такая грустная ты?
Хочу вернуться в юность,
Чтобы снова тебя послушать.

Слушай, мой друг, звучит
Мелодия тальяна.
В этой мелодии тальяна
Все потоки жизни.

Тальян-гармонь, тальян-гармонь,
Откуда такая грустная ты?
Хочу вернуться в юность,
Чтобы снова тебя послушать.
```
