# Оригинал

### Фәнис Яруллин / Риф Гатауллин - Сөялдем сөяннәргә (беренче вариант)

```
Сөялдем сөяннәргә
Тирбәлдем имәннәрдә;
Бар шатлыгымны бирәмен
Кадерен белгәннәргә.

Тукталдым аланнарда,
Сокландым баланнарга;
Бар ялкынымны бирәмен
Кабына алганнарга.

Йөгердем урамнарда,
Уралдым бураннарга;
Йөрәк давылын бирәмен
Тып-тыныч булганнарга.

Ашкындым офыкларга,
Үрелдем болытларга;
Бу җырларым канат булсын
Күңеле боекларга.
```

# Перевод

### Фанис Яруллин / Риф Гатауллин - Прильнула к любящим

```
Прильнула я к любящим,
Качалась на дубах;
Всю радость мою отдам
Тем, кто ценит её.

Остановилась на полянах,
Восхищалась соколами;
Весь свой огонь отдам
Тем, кто принял меня.

Бежала я по улицам,
Обернулась к бураным;
Бурю сердца отдам
Тем, кто был спокоен.

Взлетела к горизонтам,
Взмыла к облакам;
Пусть эти песни станут крыльями
Для великих душой.
```
