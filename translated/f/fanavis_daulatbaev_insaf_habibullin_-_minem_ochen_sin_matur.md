# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Минем өчен син матур

```
Тавышкаең кыңгыраудай
Колагымны чыңлатыр.
Кемдер өчен кемдер матур,
Минем өчен син матур!
Минем өчен, минем өчен,
Минем өчен син матур!

Кош булам да яннарыңа
Очып барам,тик чакыр.
Кемдер өчен кемдер матур,
Минем өчен син матур!
Минем өчен, минем өчен,
Минем өчен син матур!

Ялганлый бу дия күрмә,
Яратуым чынлаптыр.
Кемдер өчен кемдер матур,
Минем өчен син матур!
Минем өчен, минем өчен,
Минем өчен син матур!

Юктыр ахры дөньясында
Сиңа,иркәм,тиң матур.
Кемдер өчен кемдер матур,
Минем өчен син матур!
Минем өчен, минем өчен,
Минем өчен син матур!
```

Для меня, для меня,
Для меня ты красив!

Для меня ты красив!

Не думай, что это ложь,

Для меня, для меня,
Для меня ты красив!

Нет на свете никого
