# Оригинал

### Фәния Чанышева / Резедә Бүләкова - Бала өчен

```
Бар хәсрәтем, бар хәстәрем - бала өчен,
Хөрмәтем дә, хезмәтем дә - бала өчен.
Үкенүләр, үтенүләр - бала өчен,
Йокысыз төн, тынгысыз көн - бала өчен.

Утка - суга укталуым - бала өчен,
Чыгар юлдан тукталуым - бала өчен.
Таңнан әүел кайнар аш та - бала өчен,
Тәгәргән кайнар яшь тә - бала өчен.

Иртә - кичем, йөрәк хисем - бала өчен,
Барлык эшем, барлык көчем - бала өчен.
Янам, түзәм, көям, күнәм - бала өчен,
Горур башны түбән иям...бала өчен.

Утка - суга укталуым - бала өчен,
Чыгар юлдан тукталуым - бала өчен.
Таңнан әүел кайнар аш та - бала өчен,
Тәгәргән кайнар яшь тә - бала өчен.
```
