# Оригинал

### Фәһим Шәехов / Әсхәр Кәшфуллин - Ромашкалар (бишенче вариант)

```
Яратамы, яратмыймы?
Ярата микән?

Ромашкаларның керфеген
Өзмәгезче берәмләп.
Күңелләрне сөендереп,
Кырларны тора ямьләп.

Ромашкаларның керфеген
Өзгәч булмый шул ялгап.
Яратамы, яратмыймы?
Язмышны булмый алдап.

Гөлләрнең дә җаннары бар,
Тик сөйләшә белмиләр.
Ак керфекләре өзелгәч,
Еламыйлар, көлмиләр.

Ромашкаларның керфеген
Өзгәч булмый шул ялгап.
Яратамы, яратмыймы?
Язмышны булмый алдап.

Яратамы, яратмыймы?
Ярата микән?
Яратамы?
```

# Перевод

### Фахим Шаехов / Азар Кашфуллин - Ромашки (пятый вариант)

```
Любит ли? Не любит ли?
Любит ли она?

Не рвите ромашек
Одну за одной.
Они сердца согревают,
Поля своим светом зовут.

Ромашек нельзя рвать,
Иначе не будет игры.
Любит ли? Не любит ли?
Судьбу не обманешь.

У цветов тоже есть души,
Но говорить не умеют.
Когда белые лепестки обрываются,
Не плачут, не смеются.

Ромашек нельзя рвать,
Иначе не будет игры.
Любит ли? Не любит ли?
Судьбу не обманешь.

Любит ли? Не любит ли?
Любит ли она?
Любит ли?
```
