# Оригинал

### Феликс Мортазин / Альфред Якшимбетов - Яшимен сөеп

```
Төпсез күзләреңә карыйм
Яратам диеп
Сине генә, сине генә
Яшимен сөеп!
Йөрәгемнән чыккан хисләр
Тәүге хисләрем
Күкрәгемә кысып-кысып
Сөеп иркәлим!

Төпсез күзләреңә карап
Очып әйләнәм
Юлларыбыз гомерлеккә
Бергә бәйләнгән
Йөрәгемнән чыккан хисләр
Тәүге хисләрем
Күкрәгемә кысып-кысып
Сөеп иркәлим!

Йөрәгемнән өзлеп төшкән
Сөю сүзләре
Сине күзли, сине генә
Минем күзләрем!
Йөрәгемнән чыккан хисләр
Тәүге хисләрем
Күкрәгемә кысып-кысып
Сөеп иркәлим!
```

# Перевод

### Феликс Мортазин / Альфред Якшимбетов - Живу, любя тебя

```
В бездонные глаза гляжу,
Шепчу: люблю,
Тебя одну, тебя одну
Любя, живу!
Из сердца вышедшие чувства,
Мои первые чувства -
К груди прижму, прижму сильней,
Лаская, люблю!

В бездонные глаза глядя,
Я в полете кружусь,
Наши пути навек,
Навек переплелись.
Из сердца вышедшие чувства,
Мои первые чувства -
К груди прижму, прижму сильней,
Лаская, люблю!

Из сердца сорванные
Слова любви моей
Тебя лишь ждут, тебя одну
Мои глаза, поверь!
Из сердца вышедшие чувства,
Мои первые чувства -
К груди прижму, прижму сильней,
Лаская, люблю!
```
