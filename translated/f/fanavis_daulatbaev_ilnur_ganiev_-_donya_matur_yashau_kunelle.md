# Оригинал

### Фәнәвис Дәүләтбаев / Илнур Ганиев - Дөнья матур - яшәү күңелле

```
1.Сызылыплар алсу таң атканда,
Шатлыклы хис били күңелне.
Чут-чут итеп кошлар сайраганда,
Дөнья матур, яшәү күңелле.

Дуслар булып җирдә, тату яшәп,
Матур уздырыйкчы гомерне.
Тормыш ямен тоя белгәннәргә,
Дөнья матур, яшәү күңелле.

2.Илаһыдан тиңсез бүләк итеп,
Кабул ит син һәрбер көнеңне.
Кадерләрен белеп яшәгәндә,
Дөнья матур, яшәү күңелле.

3.Без бу җирдә бары кунак кына,
Әллә дустым шулай түгелме?
Җаның тыныч, күңелең шат булса,
Дөнья матур, яшәү күңелле.
```

# Перевод

### Фәнәвис Дәүләтбаев / Илнур Ганиев - Мир прекрасен - жить весело

```
1. Когда на востоке встает рассвет,
Радостное чувство наполняет сердце.
Когда птицы поют чирикая,
Мир прекрасен, жить весело.

Будучи друзьями на земле, мирно живя,
Давайте красиво проживем жизнь.
Для тех, кто умеет чувствовать вкус жизни,
Мир прекрасен, жить весело.

2. Принимай каждый день как бесценный дар
От Всевышнего.
Когда живешь, зная их ценность,
Мир прекрасен, жить весело.

3. Мы на этой земле лишь гости,
Разве не так, мой друг?
Если твоя душа спокойна, сердце радостно,
Мир прекрасен, жить весело.
```
