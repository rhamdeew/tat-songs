# Оригинал

### Феликс Мортазин / Альфред Якшимбетов - Ал хисләрнең олысын (Кояшым)

```
Кояшым булып кабындың
Балкыттың нурларыңны!
Икәү бергә атлыйк әйдә
Бир әле кулларыңны!

Наз тулы күзеңә карыйм да
Тоям күңелең җылысын
Ташкынлы мәхәббәт, бүләгем!
Ал хисләрнең олысын!

Назлы сөю утларында
Яныйк әйдә парлашып!
Сокландырып бар җиһанны
Бар дөньяны шаулатып!

Наз тулы күзеңә карыйм да
Тоям күңелең җылысын
Ташкынлы мәхәббәт, бүләгем!
Ал хисләрнең олысын!

Бакчамдагы чәчәгем син
Мәңге сүнмәс нурым син
Гомер юлын узган чакта
Ходай биргән ярым син!

Наз тулы күзеңә карыйм да
Тоям күңелең җылысын
Ташкынлы мәхәббәт, бүләгем!
Ал хисләрнең олысын!
```

Пойдём вдвоём вместе,

Гляжу в твои глаза, полные нежности,

Пылкая любовь, мой дар!
Великое из всех чувств!

В огнях нежной любви
Будем гореть, сливаясь вместе!
Поражая всю вселенную,

Великое из всех чувств!

Ты цветочек в моём саду,

Ты - половинка, подаренная Богом!
