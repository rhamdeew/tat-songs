# Оригинал

### Фәрдия Сәфәргалиева / Рәшит Абдуллин - Мин юк булмам

```
Мин юк булмам, туган авылымның
Болынында чәчәк булырмын.
Йә булмаса, сихри кичләрендә
Йолдыз булып балкып торырмын.

Мин юк булмам, язлар килгән саен
Әйләнермен җылы җилләргә.
Тауларда үсәрмен чирәм булып,
Тургайдай сайрармын күкләрдә.

Шуннан мине юксынганнар эзләр,
Сагынганнар аңлар сүземне.
Туган ягымның бер тамчысыдай
Яшим мин хис итеп үземне.
```

# Перевод

### Фәрдия Сәфәргалиева / Рәшит Абдуллин - Я не исчезну

```
Я не исчезну - в лугу моей деревни
Стану цветком среди травы.
А если нет - в волшебных вечерах
Звездою буду светить в ночи.

Я не исчезну: с каждым приходом весны
Обернусь тёплыми ветрами.
В горах взойду я нежной зеленью,
Как жаворонок, запою в небесах.

Тогда меня искать начнут тоскующие,
И те, кто скучал, поймут мои слова.
Как капля родины моей -
Так я живу, собой её чувствуя.
```
