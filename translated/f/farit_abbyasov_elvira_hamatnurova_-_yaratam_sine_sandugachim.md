# Оригинал

### Фәрит Аббясов / Эльвира Хаматнурова - Яратам сине, Сандугачым!

```
Язлар килә сандугачым,язлар килә.
Яратасым килә,мин сине.
Кочагыма алып,күкрәгемә салып.
Гомерлеккә сөйдем мин сине.

Хәр бер минутларым,синең белән.
Тәннәремә кертә саф хава.
Беләсеңме иркәм,син булмагач.
Йөрәккәем минем бик яна.

Авыр чакларымда,сине сагынам.
Искә алып бераз,уйланам.
Кулларыма тотып,гармунымны.
Кабат,кабат бәгърем,моңланам.

Түземлекләр булмый,кай чакларда.
Күзләремнән яшләр,агыла.
Миңа бер сүз әйтсәң,җаным диеп дәшсәң.
Күк йөзендә,йолдыз атыла.

Күк йөзендә,йолдыз атыла.
Күк йөзендә,йолдыз атыла..
```

# Перевод

### Фарит Аббясов / Эльвира Хаматнурова - Люблю тебя, Соловей мой!

```
Весны приходят, соловей мой, весны приходят.
Хочется любить мне тебя.
Обняв в объятиях, прижав к груди своей,
Навеки полюбил я тебя.

Каждая минута моя с тобой
Вливает в тело чистый воздух.
Знаешь ли, милый, когда тебя нет,
Сердце моё очень болит.

В трудные минуты скучаю по тебе,
Вспоминая, немного задумываюсь.
Взяв в руки гармонь свою,
Снова, снова душа моя тоскует.

Не хватает терпения порой,
Из глаз моих слёзы текут.
Если скажешь мне хоть слово, назовёшь душой,
На небе звезда падает.

На небе звезда падает.
На небе звезда падает..
```
