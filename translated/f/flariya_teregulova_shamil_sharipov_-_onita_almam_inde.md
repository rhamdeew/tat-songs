# Оригинал

### Флария Терегулова / Шамил Шәрипов - Оныта алмам инде

```
Оныта дигән идем,
Уйламыйм дигән идем,
Күзләреңнең карашларын
Хәтердэн җуйган идем

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.

Шомыртлар ап-ак чәчәктә,
Сагышларым яңара.
Йөрәгемнең яралары
Язлар җиткәч яңара.

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.

Тимим, калсын сагышым,
Язылган ул язмышым.
Еллар үтә, арта моңнар,
Гомер үтеп барышы.

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.
```

# Перевод

### Флария Терегулова / Шамил Шарипов - Уже не смогу забыть

```
Я говорил: забуду,
Я говорил: не думаю,
Взгляд твоих глаз
Из памяти я вычеркнул.

Уже не изменить,
Уже не позабыть.
От полных до краёв мелодий
Мне не уйти, не отрешить.

Черёмуха в белом цвете -
И снова грусть жива.
Раны сердца оживают,
Когда приходит весна.

Уже не изменить,
Уже не позабыть.
От полных до краёв мелодий
Мне не уйти, не отрешить.

Не трону - пусть будет грусть,
Такой мне записан путь.
Годы идут, и множится песня,
Так жизнь свою ведёт бег.

Уже не изменить,
Уже не позабыть.
От полных до краёв мелодий
Мне не уйти, не отрешить.
```
