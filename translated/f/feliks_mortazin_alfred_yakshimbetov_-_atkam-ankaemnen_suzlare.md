# Оригинал

### Феликс Мортазин / Альфред Якшимбетов - Әткәм-әнкәемнең сүзләре

```
Дөнья мәшәкате ерагайта
Кеше белән кеше арасын!
Кеше белән кеше арасына
Ничек итеп юллар саласы?

Күңелемә уеп язар идем
Әткәм-әнкәемнең сүзләрен
Тагын бер кат башым ияр идем
Сезнең каршыгызга тезләнеп!

Кемдер китә, кемдер килә җиргә
Дөнья көтә адәм баласы!
Бер бербезгә мәрхәмәтле булыйк
Бездән дә бит дөнья каласы!

Күңелемә уеп язар идем
Әткәм-әнкәемнең сүзләрен
Тагын бер кат башым ияр идем
Сезнең каршыгызга тезләнеп!

Барыр юллар нурлы булсын диеп
Якты булсын диеп киләчәк
Туган нигез, әткәм-әнкәм рухы
Яшәр өчен өмет бирәчәк

Күңелемә уеп язар идем
Әткәм-әнкәемнең сүзләрен
Тагын бер кат башым ияр идем
Сезнең каршыгызга тезләнеп!
```
