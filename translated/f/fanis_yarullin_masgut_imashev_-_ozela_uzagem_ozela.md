# Оригинал

### Фәнис Яруллин / Мәсгут Имашев - Өзелә үзәгем, өзелә...

```
Өзелә үзәгем, өзелә,
Кайлардан табыйм соң дәвалар?
Сулышым кагылса гөлләргә,
Гөлләр дә ялкынлап яналар.

Өзелә үзәгем, өзелә,
Йолдызлары сүнде күгемнең.
Кайда соң яп-якты көннәрем,
Ник гамьсез бер сабый түгелмен?

Өзелә үзәгем, өзелә,
Хыялның каерылды канаты...
Нигә соң язмышым күз-яшьнең
Ачысын, тозлысын яңартты?
```

# Перевод

### Фанис Яруллин / Масгут Имашев - Разрывается сердце, разрывается...

```
Разрывается сердце, разрывается,
Где же мне найти теперь лекарство?
Коснётся дыханье моё до цветов -
И цветы запылают, как пламя.

Разрывается сердце, разрывается,
Звёзды погасли на небе моём.
Где же светлые дни мои прежние,
Почему беззаботным ребёнком не стал?

Разрывается сердце, разрывается,
У мечты переломаны крылья...
Отчего же судьба моя горечь и соль
Слёз моих всё сильнее вкушает?
```
