# Оригинал

### Фирая Зыятдинова / Рөстәм Сәрвәров - Әрләмәгез!

```
Орышмагыз, әрләмәгез,
Булмый инде, туганнар.
Оялчанлык дигән нәрсә
Җанымда бар, канда бар.

Пешмәгән, дип, әрләмәгез,
Яши белми, димәгез.
Туры гына барыйм әле, -
Намусыма тимәгез.

Үзем өчен бернәрсә дә
Сорый алмыйм кешедән.
Әрсез була алу-алмау
Ул бит намус эше лә.

Үзем дә белеп бетермим
Күңелдә ниләр барын.
Беләмен тик: тугры юлдаш -
Саф намусым, вөҗданым.
```

# Перевод

### Фирая Зиятдинова / Рустам Сарваров - Не осуждайте!

```
Не ссорьтесь, не осуждайте,
Не надо, родные мои.
Стыдливость - вот эта черта
Во мне есть, где только найти.

Не спелая, мол, не судите,
Что жить не умею, не лейте.
Я прямо иду по пути -
Мою честь не троньте, простите.

Для себя я ничего
Не прошу у людей никогда.
Без просьб обходиться умею -
Ведь это же честь навсегда.

Я сама до конца не пойму,
Что таится в душе у меня.
Знаю лишь: верный спутник -
Чистая честь моя и совесть моя.
```
