# Оригинал

### Феликс Мортазин - Дулый буран (беренче вариант)

```
Җил сызгыра морҗа башларында
Тәрәзәләр каткан бозланып
Хуҗа булып җирдә буран йөри
Эшләп йөри җиңен сызганып!

Дулый буран, йолкый буран
Юлаучының якасын
Буран йолкыса йолкысын
Тик язмыш йолкымасын

Елганы да бозга төреп куйган
Ап-ак көртләр өйгән кырларга
Шыпырт кына карап тора сыман
Якындагы куе урман да

Дулый буран, йолкый буран
Юлаучының якасын
Буран йолкыса йолкысын
Тик язмыш йолкымасын

Бирешмичә сары яфрак тора
Селкенмичә тынып калган ул
Курыкмыйча көчле бураннардан
Агачында торып калган ул

Дулый буран, йолкый буран
Юлаучының якасын
Буран йолкыса йолкысын
Тик язмыш йолкымасын
```

# Перевод

### Феликс Мортазин - Бушует буран (первый вариант)

```
Ветер свищет на верхушках сосен,
Окна покрылись ледяной корой.
Хозяином по земле буран ходит,
Рукава закатав, работает с душой!

Бушует буран, хлещет буран
По воротнику путника.
Пусть буран хлещет, как хочет,
Только б судьба не била!

Речку он заковал во льды,
Белоснежные сугробы на полях взметнул.
Только шёпотом смотреть, кажется, может
Густой лес, что рядом примолк.

Бушует буран, хлещет буран
По воротнику путника.
Пусть буран хлещет, как хочет,
Только б судьба не била!

Не сдаваясь, желтый лист держится,
Не дрогнув, застыл он в тишине.
Не боясь могучих бурь и вьюг,
На дереве своём остался он.

Бушует буран, хлещет буран
По воротнику путника.
Пусть буран хлещет, как хочет,
Только б судьба не bila!
```
