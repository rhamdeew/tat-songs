# Оригинал

### Фәнәвис Дәүләтбаев / Вәгыйзь Газизов - Җанда мәхаббәт барда

```
Матур итеп үткәрик
Тик бер генә гомерне.
Бер-береңне яратып
Яшәүләре күңелле!

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!

Күңелдә сөю барда
Уртак теләк-уйлар да.
Иңгә иң куеп, иркәм,
Атлыйк әле без алга!

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!

Булсын куллар кулларда,
Чыгыйк изге юлларга.
Бәхет ул җирдә яшәү
Җанда мәхәббәт барда.

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!
```

# Перевод

### Фәнәвис Дәүләтбаев / Вәгыйзь Газизов - В сердце любовь жива

```
Прекрасно проживем
Лишь одну жизнь.
Любя друг друга,
Счастливо будем жить!

О, счастливые дни мои,
С тобой душа поёт.
Тебя одну люблю,
Искал, нашёл!

В сердце любовь жива
И общие мечты.
Плечом к плечу, легко и смело
Шагаем вперёд!

О, счастливые дни мои,
С тобой душа поёт.
Тебя одну люблю,
Искал, нашёл!

Пусть будут руки в руках,
Пойдём по светлым путям.
Счастье - это жить на земле,
Где в сердце любовь жива.

О, счастливые дни мои,
С тобой душа поёт.
Тебя одну люблю,
Искал, нашёл!
```
