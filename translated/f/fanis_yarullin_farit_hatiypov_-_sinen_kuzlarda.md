# Оригинал

### Фәнис Яруллин / Фәрит Хатыйпов - Синең күзләрдә

```
Уткән язларның,
Уткан назларның,
Матурлыкларын
Чыкмыйм эзләргә.
Язлар шатлыгы,
Назлар сафлыгы,
Синең күзләрдә,
Синең күзләрдә.

Теләгән чакта,
Колачым җәеп,
Сөю диңгезен
Иңләп йөзәргә.
Һич уйлап тормый,
Кереп югалам,
Синең күзләргә,
Синең күзләргә.

Яңа җәйләрнең
Җыеп ямьнәрен,
Баеп хисләргә,
Баеп төсләргә -
Яшьлегем минем
Адашып йөри,
Синең күзләрдә,
Синең күзләрдә.
```

# Перевод

### Фанис Яруллин / Фарит Хатыпов - В твоих глазах

```
Минувших вёсен,
Минувших нежностей
Красоту
Не выхожу искать.
Вёсен веселье,
Нежности чистота -
В твоих глазах,
В твоих глазах.

Когда захочется,
Раскрыв объятья,
В море любви
Нырнуть, плывя.
Не раздумывая,
Вхожу, теряясь,
В твои глаза,
В твои глаза.

Новых summers
Собирая дожди,
Богатея чувствами,
Богатея красками -
Моя юность
Блуждает, заплутав,
В твоих глазах,
В твоих глазах.
```
