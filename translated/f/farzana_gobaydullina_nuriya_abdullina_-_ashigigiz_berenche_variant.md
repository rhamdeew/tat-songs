# Оригинал

### Фәрзәнә Гөбәйдуллина / Нурия Абдуллина - Ашыгыгыз (беренче вариант)

```
Ашыгыгыз, ашыгыгыз
Шәфкатьлек күрсәтергә,
Рәнҗетмәгез әнкәйләрне,
Хатаны төзәтергә.

Тормыш бит ул, ишек сымак,
Кердең, чыктың, гомерләр юк,
Ашыгып үтә сукмаклары,
Май шәмендәй тиз сүнә.

Үкенүләр ялмамасын
Бусагага җиткәндә,
Риза булсын әткәй-әнкәй
Дөнъялардан киткәндә.
```

# Перевод

### Фәрзәнә Гөбәйдуллина / Нурия Абдуллина - Спешите (первый вариант)

```
Спешите, спешите
Милосердие явить,
Не обижайте матерей,
Ошибки исправить.

Жизнь ведь, словно двери:
Вошел - вышел, а лет уж нет,
Спешит её тропинка,
Как свеча - быстро гаснет.

Пусть не гложет сожаленье
У порога, на краю,
Пусть довольны будут мать с отцом,
Уходя из этого мира.
```
