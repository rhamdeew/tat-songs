# Оригинал

### Флария Терегулова / Азат Имаев - Бер моң булып яши

```
Ак чәчкәләр миңа бүләк иттең,
Ак сөюләр белән сөйдердең.
Ник соң, иркәм, хыянәтең белән
Гомерлеккә кара көйдердең?

Чәчкә гомередәй кыска булды
Биргән вәгъдәң, әйткән сүзләрең.
Әрнү, сагыш, сызланулар белән
Каршы алам гомер көзләрен.

Тәүге сөюемнән калган хисләр
Бер моң булып яши күңелдә.
Саксыз кагылсагыз, алар хәзер
Күз яшьләре булып түгелә...
```

# Перевод

### Флария Терегулова / Азат Имаев - Живёт одной мелодией

```
Белых цветов ты мне подарил,
Белой любовью меня окрылил.
Почему же, родной, изменой своей
Навек мне сердце вчерна опалил?

Срок цветка был краток - таков же был срок
Обещаний твоих, произнесённых слов.
С болью, тоской, с ноющей тревогой
Я встречаю осени жизни свой зов.

Чувства, что с первой любовью пришли,
В сердце живут одной мелодией.
Чуть прикоснитесь небрежно - и они
Станут слезами, прольются рекой...
```
