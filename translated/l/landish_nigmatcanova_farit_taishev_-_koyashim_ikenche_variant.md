# Оригинал

### Ландыш Нигъмәтҗанова / Фәрит Таишев - Кояшым (икенче вариант)

```
Юллар кабат кушылды,
Булмаган да кебек ялгышлар.
Сүзсез генә аңлаштык,
Эзләнүдән арган карашлар.

Кояшым, минем кояшым,
Күкләремә серлелек өстәп торасың.
Кояшым, алтын кояшым,
Барлык җиһан безгә бәхетләр юрасын.

Безнең күңел ул чакта
Ашкына иде шул иреккә.
Тылсым тулы дөньяны
Эзләгән идек без биектән.

Кояшым, минем кояшым,
Күкләремә серлелек өстәп торасың.
Кояшым, алтын кояшым,
Барлык җиһан безгә бәхетләр юрасын.

Җанда сөю яшәсен,
Дөрләп янсын безнең йөрәкләр.
Шикле уйлар таралсын
Күңелләрне тырнап йөдәткән.

Кояшым, минем кояшым,
Күкләремә серлелек өстәп торасың.
Кояшым, алтын кояшым,
Барлык җиһан безгә бәхетләр юрасын.
```

# Перевод

### Ландыш Нигъмәтҗанова / Фарит Таишев - Моё солнышко (второй вариант)

```
Дороги вновь пересеклись,
Как будто не было ошибок.
Поняли друг друга без слов,
Уставшие от поисков взгляды.

Моё солнышко, моё солнышко,
Ты добавляешь тайну в моё небо.
Моё солнышко, золотое солнышко,
Пусть весь Вселенная шлёт нам счастье.

В ту пору наша душа
Стремилась к той свободе.
Мир, полный чудес,
Мы искали на вершинах.

Моё солнышко, моё солнышко,
Ты добавляешь тайну в моё небо.
Моё солнышко, золотое солнышко,
Пусть весь Вселенная шлёт нам счастье.

Пусть живёт любовь в душе,
Пусть горят наши сердца.
Пусть развеются сомнения,
Которые царапали души, лишая сна.

Моё солнышко, моё солнышко,
Ты добавляешь тайну в моё небо.
Моё солнышко, золотое солнышко,
Пусть весь Вселенная шлёт нам счастье.
```
