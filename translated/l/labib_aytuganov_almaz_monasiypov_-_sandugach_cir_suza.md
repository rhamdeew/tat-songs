# Оригинал

### Ләбиб Айтуганов / Алмаз Монасыйпов - Сандугач җыр суза

```
Сандугач җыр суза
Тын гына, моң тына;
Өзелеп-өзелеп,
Сандугач җыр суза
Тын гына, моң гына -
Кемне ул юксына?

Ул көтә назлысын,
Килми шул сөйгәне;
Сагына-сагына,
Ул көтә назлысын,
Килми шул сөйгәне -
Бер генәм дигәне.

Сандугач җыр суза
Тын гына, моң гына;
Өзелеп-өзелеп,
Сандугач җыр суза
Тын гына, моң гына -
Иркәсен юксына.
```

# Перевод

### Ләбиб Айтуганов / Алмаз Монасыйпов - Соловей выводит песню

```
Соловей выводит песню -
Тихо, нежно, тонко;
Рвясь на вздохи,
Соловей выводит песню -
Тихо, лишь напев -
Кого он так тоскует?

Он ждёт свою ласковую,
Не идёт та любимая;
Тоскуя-тоскуя,
Он ждёт свою ласковую,
Не идёт та любимая -
Единственная моя.

Соловей выводит песню -
Тихо, лишь напев;
Рвясь на вздохи,
Соловей выводит песню -
Тихо, лишь напев -
Свою милую он ищет.
```
