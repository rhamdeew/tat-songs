# Оригинал

### Ләбиб Айтуганов / Фасил Әхмәтов - Әй, Кабаным!

```
Әкрен генә шәфәкъ сүнә,
Әй, Кабаным, сиңа ямьле кич килә!
Көмешләнә бөдрә дулкыннарың,
Карап туя алмыйм һич кенә!

Кемнәр генә сине белми,
Әй, Кабаным, сине Тукай җырлаган!
Сиңа килсәм, ишетәм кебек һаман
Аның җырын ерак еллардан.

Яратам мин якты кичең,
Әй, Кабаным, күңелем түзми килмәсәм!
Әйтерсең лә, ярларыңа басып,
Чал тарихың белән сөйләшәм.
```

# Перевод

### Лябиб Айтуганов / Фасиль Ахметов - Эй, Кабаным!

```
Постепенно гаснет заря,
Эй, Кабаным, к тебе приходит нежная ночь!
Серебрятся волны Бодра,
Не могу наглядеться никак!

Кто же тебя не знает,
Эй, Кабаным, ведь Тукая пел о тебе!
Если иду к тебе, слышу словно
Его песню издалёка лет.

Обожаю я твою светлую ночь,
Эй, Кабаным, не терпит душа не прийти!
Говори, ступая по берегам твоим,
Беседую с историей Тукая.
```
