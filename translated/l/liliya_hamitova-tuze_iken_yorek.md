# Оригинал

### Лилия Хамитова - Тузэ икэн йорэк

```
Ялгыз керэм быел язларыма
Ялгызлыкнын ачы язмышы.
Адэм баласынын кузлэренэ
Алдан язылган шул ялгышы.

Эрнулэргэ тузэ икэн йорэк,
Тузэ икэн йорэк барсына.
Тик шулайда учларыма кысам,
Ярсыма дим ана, ярсыма

Сою тулы безнен сукмаларга
Сары сагыш кына ятармы?
Мин яраткан ап-ак ромашкалар
Жэен сары чэчэк атармы

Эрнулэргэ тузэ икэн йорэк,
Тузэ икэн йорэк барсына.
Тик шулайда учларыма кысам,
Ярсыма дим ана, ярсыма

Сагыш куле бигрэк тирэн икэн,
Суы тозлы, димэк, куз яше.
Тик шулайда омет сунми икэн,
Сине котеп, бэхетем, мин яшим!

Эрнулэргэ тузэ икэн йорэк,
Тузэ икэн йорэк барсына.
Тик шулайда учларыма кысам,
Ярсыма дим ана, ярсыма
```

# Перевод

### Лилия Хамитова - Сердце умеет терпеть

```
Одинокой вхожу я в весну этой жизни,
Горькая участь моего одиночества.
В глазах человеческих
Заранее написана та ошибка.

К страданиям сердце привыкло терпеть,
Сердце умеет терпеть всё на свете.
Только когда к краю себя подведу,
"Довольно!" - скажу тогда, довольно!

На тропинки наши, полные холода,
Только жёлтая печаль ли ляжет?
Мои любимые белоснежные ромашки
Летним жёлтым цветом зацветут ли?

К страданиям сердце привыкло терпеть,
Сердце умеет терпеть всё на свете.
Только когда к краю себя подведу,
"Довольно!" - скажу тогда, довольно!

Озеро печали оказалось глубоким,
Вода солёная, видно, это слёзы.
Только вот надежда не угасает,
Тебя дожидаясь, счастье моё, я живу!

К страданиям сердце привыкло терпеть,
Сердце умеет терпеть всё на свете.
Только когда к краю себя подведу,
"Довольно!" - скажу тогда, довольно!
```
