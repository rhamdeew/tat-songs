# Оригинал

### Ләбиб Лерон / Татар  халык - Чума үрдәк, чума каз

```
Чума үрдәк, чума каз,
Агым суны ярата шул, ярата.
Чупырдашып алар суда
Дулкыннарны тарата шул, тарата.

Чума үрдәк, чума каз,
Алар шулай коена шул, коена.
Күк йөзендә алсу кояш -
Аннан нурлар коела шул, коела.

Чума үрдәк, чума каз,
Канатларын кагына шул, кагына.
Йөзеп китте каз-үрдәкләр
Суның аргы ягына шул, ягына.

Чума үрдәк, чума каз,
Йөзә-чума уйныйлар шул, уйныйлар.
Су чәчрәтә каз-үрдәкләр,
Уйнап һич тә туймыйлар шул, туймыйлар.
```

# Перевод

### Ләбиб Лерон / Татар халык - Ныряет утка, ныряет гусь

```
Ныряет утка, ныряет гусь,
Любят воду быстрых струй, любят.
Плещутся они в воде,
Волны разгоняют, разгоняют.

Ныряет утка, ныряет гусь,
Так они купаются, купаются.
В небе - розовый закат,
Сыплет светом он, сыплет.

Ныряет утка, ныряет гусь,
Крыльями взмахнут, взмахнут.
Поплыли гуси-утки
К дальнему берегу, к берегу.

Ныряет утка, ныряет гусь,
Плавают-ныряют, играют.
Брызги сыплют гуси-утки,
И игрой не насытятся, не насытятся.
```
