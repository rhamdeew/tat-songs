# Оригинал

### Лилия Гарифуллина / Фирзәр Мортазин - Барыбер яратам

```
Мин сине барыбер яратам,
Нинди генә булсаң да.
Йөрәгемнең ярасына
Кайчак тозлар салсаң да.

Мин сине барыбер яратам,
Ярату - татлы газап.
Эзләмә читтән сөюләр,
Син дә бик мине ярат.

Мин сине барыбер яратам
Һәр күзәнәгем белән.
Ни әйтергә теләвеңне
Йөрәгем белән сизәм.

Мин сине барыбер яратам,
Ярату - татлы газап.
Эзләмә читтән сөюләр,
Син дә бик мине ярат.

Мин сине барыбер яратам,
Нинди генә булсаң да.
Йөрәгемнең ярасына
Кайчак тозлар салсаң да.
```
