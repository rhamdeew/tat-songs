# Оригинал

### Ләйлә Дәүләтова / Альфред Якшимбетов - Иң зур ялгышым

```
Үз кулларым белән бирдем бәхетемне.
Җир йотмаган мине нигә шул вакыт?
Бөтен гомеремне бирер идем хәзер
Әгәр ялгышларны булса туктатып.

Кичер мине, балам, гомер бүләк иттем,
Ләкин сиңа бәхет бирә алмадым.
Синнән башка яшәү - яшәү түгеллеген
Ачы газапларга төшкәч аңладым.

Син бит, балам минем, йөрәк түрләремнең
Өзелепләр төшкән иң зур ялгышым.
Беркем белми торган ачы язмышым син,
Мәңге бетми торган гомер сагышым.
```

# Перевод

### Ляйля Даулатова / Альфред Якшимбетов - Моя главная ошибка

```
Своими руками отдала я счастье.
Почему земля меня не поглотила тогда?
Всю жизнь свою отдала бы сейчас,
Если б можно было ошибки остановить.

Прости меня, родная, жизнь разделила,
Но счастья дать тебе я не смогла.
Что жизнь без тебя - не жизнь вовсе,
Я поняла, когда в муки попала.

Ты ведь, родная моя, всех сердечных страданий
Моя главная ошибка роковая.
Ты - горькая судьба моя неведомая,
Ты - вечная тоска моя живая.
```
