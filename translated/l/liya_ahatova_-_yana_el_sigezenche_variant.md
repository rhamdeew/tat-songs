# Оригинал

### Лия Әхәтова - Яңа ел (сигезенче вариант)

```
Эх, Яңа ел, Яңа ел,
Эх, Яңа ел, Яңа ел.

Ап-ак карлар явып үтте
Безнен юлларыбызга.
Яңа ел да килеп житте,
Ашыгып безнең янга.

Эх, Яңа ел, Яңа ел
Җилкетә күңелләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Җилкетә күңелләрне,
Эх, Яңа ел, Яңа ел.

Яңа елда чынга аша
Бөтен безнең хыяллар.
Күктәге бик күп йолдызлар
Ак юл теләп калалар.

Эх, Яңа ел, Яңа ел
Җилкетә күңелләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Җилкетә күңелләрне,
Эх, Яңа ел, Яңа ел.
```

```
Эх, Новый год, Новый год,
Эх, Новый год, Новый год.

Белый снег прошёл и выпал
На дорожки нам с тобой.
Новый год уже подходит,
Поспешив к нам за чертой.

Эх, Новый год, Новый год,
Сердца он качает.
Поздравим мы с праздником,
Пожеланья загадаем.
Поздравим мы с праздником,
Пожеланья загадаем.
Сердца он качает,
Эх, Новый год, Новый год.

В Новый год сбываются
Все наши мечты.
Звёзды в небе многочисленные
Шепчут: путь тебе светлый.

Эх, Новый год, Новый год,
Сердца он качает.
Поздравим мы с праздником,
Пожеланья загадаем.
Поздравим мы с праздником,
Пожеланья загадаем.
Сердца он качает,
Эх, Новый год, Новый год.
```
