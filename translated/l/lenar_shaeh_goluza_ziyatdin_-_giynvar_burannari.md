# Оригинал

### Ленар Шәех / Гөлүзә Зиятдин - Гыйнвар бураннары

```
Ялгыз гына йөрим бүген
Казан урамнарында.
Кайчагында узып барам
Гыйнвар бураннарын да.

Әйдә, гыйнвар бураннары
Күңелне урасыннар.
Сагышлы дип тормасыннар,
Ак бәхет юрасыннар.

Син җанны иркәләмәгәч,
Басмагач ярсуымны,
Бураннарга назлануым
Үземнең качуымнан.

Әйдә, гыйнвар бураннары
Күңелне урасыннар.
Сагышлы дип тормасыннар,
Ак бәхет юрасыннар.

Казан урамында йөрим,
Буран мине очыртсын.
Җылы кочагыма кереп,
Язгы җил булып чыксын.

Әйдә, гыйнвар бураннары
Күңелне урасыннар.
Сагышлы дип тормасыннар,
Ак бәхет юрасыннар.
```

# Перевод

### Ленар Шаех / Гулюза Зиатдин - Январские метели

```
Одинока хожу сегодня
По улицам Казани.
Иногда прохожу сквозь
Январские метели.

Давай, январские метели,
Пусть сметают настроение.
Пусть не стоят в тоске,
Пусть гонят белое счастье.

Если ты не освободишь душу,
Не сожмешь мою любовь,
Мое упрямство перед метелями
Происходит от моего бегства.

Давай, январские метели,
Пусть сметают настроение.
Пусть не стоят в тоске,
Пусть гонят белое счастье.

Хожу по улице Казани,
Пусть метель сбивает меня с пути.
Войдя в мои теплые объятия,
Пусть выйдет весенним ветром.

Давай, январские метели,
Пусть сметают настроение.
Пусть не стоят в тоске,
Пусть гонят белое счастье.
```
