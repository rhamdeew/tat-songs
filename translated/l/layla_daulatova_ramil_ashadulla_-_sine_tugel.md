# Оригинал

### Ләйлә Дәүләтова / Рамил Әсхәдулла - Сине түгел

```
Әллә үзебез башкамы,
Әллә сөю үзгәргән.
Ят карашлар гына калган
Сөю тулы күзләрдән.

Яралардан арына алмый,
Ярсып янса да күңел.
Без әле дә яратабыз,
Ләкин син мине түгел,
Ләкин мин сине түгел.

Бергә салган күперләрне
Яндырсаң да, түзиммә?
Җанны бәйләп тота алмассың,
Калдырсаң да үземне.

Яралардан арына алмый,
Ярсып янса да күңел.
Без әле дә яратабыз,
Ләкин син мине түгел,
Ләкин мин сине түгел.

Мәңге җуелмаслык булып
Калса да яраларың.
Бер сүнгән йолдыз кабынмый,
Кабына яңалары.

Яралардан арына алмый,
Ярсып янса да күңел.
Без әле дә яратабыз,
Ләкин син мине түгел,
Ләкин мин сине түгел.
```
