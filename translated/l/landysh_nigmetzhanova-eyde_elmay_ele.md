# Оригинал

### Ландыш Нигъмәтҗанова - Эйдэ елмай эле

```
Елмайсан галэмнэр яктырыр
Кон болытлы булганда
Йорэктэ шат хислэр уяныр
Бары бер елмай гына
Синен шаян кузлэрен
Вулкан сыман ут чэчэ
Ах харап булдым кебек
Гыйшыктан йорэк чэнчэ

Эйдэ эйдэ бер елмай эле
Сихри нурлар чэчеп бар жиханга
Сина карап мин елмаермын
Кинэт килеп чыгып алларына

Тик сина карап елмаялар
Кояш, ай хэм йолдызлыр
Курэсен тик сина сокланып
Синнэн урнэк алалар
Мизгелгэ елмайсан да
Йорэгем дулкынлана
Уйларым мине ташлап
Бары сина ашкына

Эйдэ эйдэ бер елмай эле
Сихри нурлар чэчеп бар жиханга
Сина карап мин елмаермын
Кинэт килеп чыгып алларына
```

# Перевод

### Ландыш Нигъмәтҗанова - Давай, улыбнись

```
Улыбнешься - миры озарятся,
Когда день облачный бывает,
В сердце радостные чувства пробуждаются,
Только улыбнись.
Твои озорные глаза
Как вулкан огонь рассыпают,
Ах, погибла я словно
От любви сердце трепещет.

Давай, давай, улыбнись же,
Волшебные лучи рассыпая по всему миру,
Глядя на тебя, я улыбаюсь,
Вдруг появившись перед ними.

Только глядя на тебя улыбаются
Солнце, луна и звезды,
Видишь, только тобой восхищаясь,
С тебя пример берут.
На мгновение улыбнешься -
Сердце мое волнуется,
Мысли меня покидают,
Все к тебе стремятся.

Давай, давай, улыбнись же,
Волшебные лучи рассыпая по всему миру,
Глядя на тебя, я улыбаюсь,
Вдруг появившись перед ними.
```
