# Оригинал

### Ләйсән Кәшфи / Гөлнара Хәсәнова - Сәлам Өфедән

```
Без Агыйделдән җыр алып килдек,
Урал таулары сәлам юллады.
Ничә кат инде җир шарын урый
Татар җырының данлы юллары.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек -
Өфе яклары.

Кая барсак та, кайда йөрсәк тә,
Сез, туганнарым, каршы аласыз.
Йөрәк тоташа моңлы җыр аша,
Без туган җирнең газиз баласы.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек -
Өфе яклары.

Сагынган чакта, җирсегән чакта
Безнең моңнардан шифа алыгыз.
Без бит җир шарын җырга чорныйбыз,
Хәерли сәфәр теләп калыгыз.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек -
Өфе яклары.
```

```
Мы принесли песню с Агидели,
Уральские горы шлют привет.
Сколько раз уже облетает земной шар
Славные строки татарской песни.

Берег Агидели
В melody оплёл,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодийный райский сад -
Краёв Уфа.

Куда бы ни шли, где бы ни бродили,
Вы, земляки, встретите нас.
Сердца соединяются через мелодичную песню,
Мы - любимые дети родной земли.

Берег Агидели
В melody оплёл,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодийный райский сад -
Краёв Уфа.

В минуты тоски, когда устали от земли,
Принимайте исцеление от наших мелодий.
Ведь мы облетаем земной шар с песней,
Желая вам счастливого пути.

Берег Агидели
В melody оплёл,
Привет из Уфы,
Привет с Урала.
Несут нас
Песни крылья.
Мелодийный райский сад -
Краёв Уфа.
```
