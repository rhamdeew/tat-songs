# Оригинал

### Ландыш Нигъмәтҗанова - Тылсымлы суз

```
Матур синен кузлэрен,
Елмаюлы йозлэрен.
Мина бер суз эйтуеннэн
Дулкынлана йорэгем.
Тылсымнары бармы эллэ
Яратам дигэн сузнен?
Жавап итеп чын кунелдэн
Эйтэм мин аны узем.

Тылсымлы суз эйтуенне
Суздэ шул диде кунел.
Яратам дигэн сузенне
Гел ишетергэ телим.
Ничэ мэртэбэ эйтсэн дэ,
Барыбер артык тугел.
Тылсымлы суз ишетуне
Ярата минем кунел.

Киям дэ матур кулмэгем,
Чишмэлэргэ килэмен.
Чишмэлэргэ сер итеп кенэ
Гашыйк булдым диямен.
Тылсымнары бармы эллэ
Яратам дигэн сузнен?
Жавап итеп чын кунелдэн
Эйтэм мин аны узем.

Тылсымлы суз эйтуенне
Суздэ шул диде кунел.
Яратам дигэн сузенне
Гел ишетергэ телим.
Ничэ мэртэбэ эйтсэн дэ,
Барыбер артык тугел.
Тылсымлы суз ишетуне
Ярата минем кунел.
```

# Перевод

### Ландыш Нигъмәтҗанова - Волшебное слово

```
Красивы твои глаза,
Улыбчивое лицо твоё.
От одного слова твоего
Волнуется сердце моё.
Есть ли волшебство
У слова "люблю"?
Отвечая от чистого сердца,
Говорю его сама.

Волшебное слово твоё —
В слове том душа живёт.
Твоё слово "люблю"
Всё время хочу слышать.
Сколько раз ни говори,
Всё равно мало.
Волшебное слово слышать
Любит душа моя.

Надеваю красивое платье,
К родникам иду.
Родникам по секрету только
Скажу, что влюбилась.
Есть ли волшебство
У слова "люблю"?
Отвечая от чистого сердца,
Говорю его сама.

Волшебное слово твоё —
В слове том душа живёт.
Твоё слово "люблю"
Всё время хочу слышать.
Сколько раз ни говори,
Всё равно мало.
Волшебное слово слышать
Любит душа моя.
```
