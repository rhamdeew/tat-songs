# Оригинал

### Л. Ахметова / Гөлсәрвәр - Син булганда (өченче вариант)

```
Йөрәгем сиңа тартыла,
Яратам, янам, көям.
«Син җаным сайлаган ярым,
Үземә тиңем», - диям.

Эй, ярым син, ярым син,
Бәхетемнең нуры син.
Мәхәббәтем, жаным син.

Безнең өчен чәчәк ата
Далаларның гөлләре.
Җырлый ярату турында
Тауларның чишмәләре.

Эй, ярым син, ярым син,
Бәхетемнең нуры син.
Мәхәббәтем, жаным син.

Кояш нурлары башкача,
Яктыра яңа кебек.
Бар дөнья шатлана
Безнең бәхетне күреп.

Эй, ярым син, ярым син,
Бәхетемнең нуры син.
Мәхәббәтем, жаным син.
```

# Перевод

### Л. Ахметова / Гульсарвар - Когда ты рядом (третий вариант)

```
Сердце к тебе стремится,
Люблю, пылаю, сгораю.
«Ты половинка, что душа избрала,
Ты равен мне», - говорю.

Эй, милый мой, милый мой,
Ты свет моего счастья.
Ты любовь моя, душа моя.

Для нас цветут
Полевые цветы.
Поют о любви
Горные родники.

Эй, милый мой, милый мой,
Ты свет моего счастья.
Ты любовь моя, душа моя.

Солнечные лучи по-другому
Светят, словно впервые.
Весь мир ликует,
Видя наше счастье.

Эй, милый мой, милый мой,
Ты свет моего счастья.
Ты любовь моя, душа моя.
```
