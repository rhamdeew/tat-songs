# Оригинал

### Ләйсән Кәшфи / Людмила Аланлы - Язны алып кил

```
Язларны син алып кил инде,
Сагындым, сагындым, сагындым.
Ничә ел язларсыз үтелде,
Әрнүле көзләргә табындым.

Язларым, көзләрең нурыдай
Офыклар артында адашкан.
Боз булган күңелем дәрьясы
Шашыныр синең бер караштан.

Язларга соңлама ялгыш та,
Боз астын ташкыннар тарсынган.
Син генә таратыр сагышта
Оҗмахлар җылысы бар сыман.
```

# Перевод

### Ляйсан Кашфи / Людмила Аланлы - Принеси весну

```
Весны принеси же ты,
Скучаю, скучаю, скучаю.
Сколько лет прошло без весны,
Покорялся грустным осеням.

Мои весны, как лучи твоих осеней,
Потерянные за горизонтом.
Река души моей, покрытая льдом,
Вспыхнет от одного твоего взгляда.

Не опаздывай к веснам, не ошибись,
Потоки подо льдом сдерживаемы.
Только ты развеешь тоску,
Как будто есть тепло рая.
```
