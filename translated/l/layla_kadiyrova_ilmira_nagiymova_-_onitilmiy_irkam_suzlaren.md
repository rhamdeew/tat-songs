# Оригинал

### Ләйлә Кадыйрова / Ильмира Нәгыймова - Онытылмый, иркәм, сүзләрең

```
Өзелгэн чәчәкнең моңсу киләчәген
Юк, кирәкми, иркәм, сөйләвең.
Сине генә бары, сине генә
Офыклардан эзли күзләрем.

Төшләремдә бүген сине күрдем,
Ә таңымда кабат эзләдем.
Синсез тормыш кояшсыз күк сыман,
Онытылмый, иркәм, сүзләрең.

Кирәк түгел миңа башка берәү,
Син диеп тибә ярсып йөрәгем.
Минем җаным син дип кенә сулый,
Мин бар назым сиңа бирәмен.

Төшләремдә бүген сине күрдем,
Ә таңымда кабат эзләдем.
Синсез тормыш кояшсыз күк сыман,
Онытылмый, иркәм, сүзләрең.

Бармы бер көн сине уйламаган,
Сине генә һәрчак көтәмен.
Мин яратам сине, ишетәсеңме,
Шулай диеп тибә йөрәгем.

Төшләремдә бүген сине күрдем,
Ә таңымда кабат эзләдем.
Синсез тормыш кояшсыз күк сыман,
Онытылмый, иркәм, сүзләрең.
Онытылмый, иркәм, сүзләрең....
```
