# Оригинал

### Лариса Абдуллина / Ильяс Аллаяров - Наратлар урманында

```
Исеңдәме яңгыр яуды
Наратлар урманында.
Юк, яумады ул, җырлады ул,
Без икеү турасында.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.

Күңелләргә сеңеп калды
Моңнары шул мизгелнең.
Әллә ярсу чагы иде
Иңнәремә түгелдең.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.

Шул урманда яктыртың син,
Сүрелгән хисләремне.
Синең белән үтсәм иде
Калган гомерләремне.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.
```

# Перевод

### Лариса Абдуллина / Ильяс Аллаяров - В сосновом лесу

```
Помнишь, дождь прошёл
В сосновом лесу.
Нет, не дождь - он пел,
О нас двоих одну.

Пел дождь, пел, звучал,
Сердца озарял.
Слёзно песню слушал
Юный сосновый лес.

В душах навсегда
Остался тот напев.
Может, в пору яростную
Ты прильнул к моим плечам.

Пел дождь, пел, звучал,
Сердца озарял.
Слёзно песню слушал
Юный сосновый лес.

В том лесу зажги
Мои угасшие чувства.
С тобой бы мне пройти
Остаток жизни всю.

Пел дождь, пел, звучал,
Сердца озарял.
Слёзно песню слушал
Юный сосновый лес.
```
