# Оригинал

### Ләйсән Кәшфи / Зиннур Сафиуллин - Тамчылар (икенче вариант)

```
Тагын да язлар килә, кояш елмаеп көлә,
Мәхаббәтнең ялгышлары йөрәкне телә, телә!

Тамчылар тамган чакта, кар-сулар аккан чакта,
Ни өчен соң тилмерәбез икебез ике якта?

Язгы өмет яңара, карыйм алсу таңнарга,
Бер карашың җитәр иде күңелемне аңларга!

Тамчылар тамган чакта, кар-сулар аккан чакта,
Ни өчен соң тилмерәбез икебез ике якта?

Бәллүр тамчы җырына талда бөре уяна,
Безне кавыштырыр өчен язлар килә дөньяга!

Тамчылар тамган чакта, кар-сулар аккан чакта,
Ни өчен соң тилмерәбез икебез ике якта?
```

# Перевод

### Ләйсән Кәшфи / Зиннур Сафиуллин - Капли (второй вариант)

```
Снова весны идут, солнце смеясь золотится,
Ошибки любви резали сердце - режут, как нити.

Капают капли с крыш, бегут по снегам ручьи,
Почему мы томимся, по разные стороны дни?

Весенняя надежда оживает в рассветный час,
Одного твоего взгляда хватило б понять мой рассказ.

Капают капли с крыш, бегут по снегам ручьи,
Почему мы томимся, по разные стороны дни?

На хрустальный звон капели ива почку раскрыла едва,
Чтобы нас соединить, в мир приходят весны, любя.

Капают капли с крыш, бегут по снегам ручьи,
Почему мы томимся, по разные стороны дни?
```
