# Оригинал

### Лилия Хамитова - Тылсымлы телэк

```
Ничэ мэртэбэ укыдым,
Истэлекле дэфтэрне...
Бэхет телэп килэчэккэ,
Дуслар язган сузлэрне....

Бэхетле бул, бэхетле бул,
Кабатлый минем йорэк.
Бэхетле бул, бэхетле бул,
Ин зур тылсымлы телэк...

Тормышында барысыда,
Яхшы булсын дигэнсез.
Ин зур телэк бу доньяда,
Бэхетле бул дигэнсез.

Бэхетле бул, бэхетле бул,
Кабатлий минем йорэк.
Бэхетле бул, бэхетле бул,
Ин зур тылсымлы телэк...

Яшэмен кунел турендэ,
Дусларнын шул телэген.
Доньяда хэрбер кешегэ,
Бэхет тели йорэгем!

Бэхетле бул, бэхетле бул,
Кабатлый минем йорэк.
Бэхетле бул, бэхетле бул,
Ин зур тылсымлы телэк...
```

# Перевод

### Лилия Хамитова - Волшебное пожелание

```
Сколько раз перечитывала
Тетрадь воспоминаний...
Счастья желая в будущем,
Слова, что писали друзья...

Будь счастлива, будь счастлива,
Повторяет моё сердце.
Будь счастлива, будь счастлива,
Самое великое волшебное пожелание...

В твоей жизни пусть всё
Будет хорошо, говорят вам.
Самое главное желание в этом мире -
"Будь счастлива!" - говорят вам.

Будь счастлива, будь счастлива,
Повторяет моё сердце.
Будь счастлива, будь счастлива,
Самое великое волшебное пожелание...

Ношу в душе своей
То пожелание друзей.
Каждому человеку в мире
Счастья желает сердце моё!

Будь счастлива, будь счастлива,
Повторяет моё сердце.
Будь счастлива, будь счастлива,
Самое великое волшебное пожелание...
```
