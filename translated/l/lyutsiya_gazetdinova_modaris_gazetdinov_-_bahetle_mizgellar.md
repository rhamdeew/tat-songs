# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Бәхетле мизгелләр

```
Өз(е)леп сөйгән ярлар гына
Мәңгелеккә кавыша,
Язмышны кушып язмышка
Пар балдаклар алыша!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!

Юлыгызга, бүгенгедәй,
Сибелсеннәр ак гөлләр!
Йөрәкләрдә гел саклансын
Бу бәхетле мизгелләр!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!

Югалмасын, тапланмасын,
Суынмасын саф хисләр!
Арагыздан үтмәсеннәр
Хәтта юхалы җилләр!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!
```

# Перевод

### Люция Газетдинова / Мөдәрис Газетдинов - Счастливые мгновения

```
Лишь те, кто всем сердцем любил,
Навек соединяются,
Судьбу соединяя с судьбой,
Кольцами обмениваются!

Пусть страна счастья навсегда
Откроет тёплые объятья.
Живите, не гасите вы
Очаг семейного огня!

Пусть на ваш путь, как нынче, вновь
Сыпятся белые цветы!
И в сердцах всегда хранится
Эти счастливые мгновения!

Пусть страна счастья навсегда
Откроет тёплые объятья.
Живите, не гасите вы
Очаг семейного огня!

Не исчезнут, не будут смяты,
Не остынут чистые чувства!
Меж вами пусть не пройдут
Даже лёгкие ветра!

Пусть страна счастья навсегда
Откроет тёплые объятья.
Живите, не гасите вы
Очаг семейного огня!
```
