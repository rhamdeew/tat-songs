# Оригинал

### Лилия Хамитова - Куз тимэсен балама

```
Ачык тосле тубы белэн
Уйный, балам, йогерэ.
Карыйм мона шатлык белэн,
Бэхет телэп гомергэ.

Туп, туп, туп тэгэри,
Балам шатлана, колэ.
Нэни тэпилэре белэн
Туп артыннан йогерэ.

Уяна да hэр кон саен,
Гел урелэ тубына.
Тик тормас дип, колеп куям,
Куз тимэсен балама.

Туп, туп, туп тэгэри,
Балам шатлана, колэ.
Нэни тэпилэре белэн
Туп артыннан йогерэ.

Хэрэкэтсез тора белми,
Коне буе йогерэ.
Карыйм мона шатлык белэн,
Бэхет телэп гомергэ.

Туп, туп, туп тэгэри,
Балам шатлана, колэ.
Нэни тэпилэре белэн
Туп артыннан йогерэ.
```

# Перевод

### Лилия Хамитова - Пусть не коснётся дурной глаз моего ребёнка

```
С ярким цветным мячиком
Играет, бежит мой малыш.
Смотрю на него с радостью,
Счастья желая на всю жизнь.

Мяч, мяч, мяч катится,
Мой ребёнок радуется, смеётся.
На маленьких ножках
За мячом бежит.

Просыпается каждый день,
Всё тянется к мячику.
Чтоб не устал, смеясь говорю:
Пусть не коснётся дурной глаз моего ребёнка.

Мяч, мяч, мяч катится,
Мой ребёнок радуется, смеётся.
На маленьких ножках
За мячом бежит.

Не может стоять без движения,
Весь день напролёт бегает.
Смотрю на него с радостью,
Счастья желая на всю жизнь.

Мяч, мяч, мяч катится,
Мой ребёнок радуется, смеётся.
На маленьких ножках
За мячом бежит.
```
