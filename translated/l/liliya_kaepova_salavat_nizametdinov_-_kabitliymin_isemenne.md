# Оригинал

### Лилия Кәепова / Салават Низаметдинов - Кабытлыймын исемеңне

```
Күктән кояш бүләк иттеләрме -
Шатлыгыма тамам исердем.
"Мин бәхетле!"- диеп, ҡайта-ҡайта
"Мин бәхетле!"- диеп,
Кабатлыймын синең исемеңне!

Кирәк чакта дуслар таралышты-
Ялгызлыкны ничек кичердем?
Яшь аралаш тешләремне кысып,
Тешләремне кысып,
Кабатладым синең исемеңне.

Йөгәнләргә әгәр кирәк булса,
Үрәпчегән ярсу хисемне
Һәр чак шулай, чиксез өмет белән,
Чиксез өмет белән
Кабатлыймын синең исемеңне!
```

# Перевод

### Лилия Кәепова / Салават Низаметдинов - Повторяю твое имя

```
Будто солнце мне с небес дарили -
Я от радости до дна пьянел.
«Я счастлива!» - твердя раз за разом,
«Я счастлива!» - твердя,
Повторяю твое имя.

В час беды друзья разбрелись -
Как я вынесла ту пустоту?
Слёзы смешаны, зубы стиснув,
Зубы стиснув,
Повторяла я твое имя.

Если надо укротить поводьями
Разбушевавшийся гневный пыл,
Я всегда, с беспредельной надеждой,
С беспредельной надеждой
Повторяю твое имя!
```
