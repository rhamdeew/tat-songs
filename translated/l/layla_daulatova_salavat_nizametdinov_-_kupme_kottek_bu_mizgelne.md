# Оригинал

### Ләйлә Дәүләтова / Салават Низаметдинов - Күпме көттек бу мизгелне

```
Көзге салкыннарда адашырлык
Язгы назлар барын сиздеңме?
Ятлар түгел, үзебездә белми
Күпме көттек, көттек бу мизгелне.

Ятлар түгел, үзебездә белми
Күпме көттек, без бу мизгелне.

Давыллардан күпме саклап килдем
Түгелергә торган хисемне,
Сүнеп барган өметләрне ялгап,
Күпме көттек, көттек бу мизгелне.

Сүнеп барган өметләрне ялгап,
Күпме көттек, без бу мизгелне.

Җирдә вакыт туктап калса калсын
Тик калдырсын янда үзеңне.
Яшәү көче беткән чакларда да
Күпме көттек, көттек бу мизгелне.

Яшәү көче беткән чакларда да
Күпме көттек, без бу мизгелне.
```

# Перевод

### Ләйлә Дәүләтова / Салават Низаметдинов - Сколько ждали мы этот миг

```
В осенний холод можно заблудиться,
А ты весеннюю ласку ощутил?
Не чужие - мы сами, не заметив,
Сколько ждали, ждали этот миг.

Не чужие - мы сами, не заметив,
Сколько ждали мы этот миг.

Сколько бурь оберегал я чувство,
Что готово было разлиться,
Сшивая гаснущие надежды,
Сколько ждали, ждали этот миг.

Сшивая гаснущие надежды,
Сколько ждали мы этот миг.

Пусть на земле застынет время -
Только бы ты остался рядом.
Даже когда угасает сила жизни,
Сколько ждали, ждали этот миг.

Даже когда угасает сила жизни,
Сколько ждали мы этот миг.
```
