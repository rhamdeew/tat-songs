# Оригинал

### Ландыш Нигметжанова - Очрашканда

```
Очрашканда кузлэр убештелэр
Ирененэ-ирнем кагылды
Эх кайтасы иде шул мизгелгэ
Белсэн иде ничек сагындым

Син бер генэ гомер минем очен
Мин бит сина гомер табыны
Югалтасым килми мэхэббэтем
Килче мина килче ягымлым

Вэгъдэ белэн вэгъдэ кавышканда
Уртак йолдызыбыз кабынды
Тоннэрендэ эзлим шул йолдызны
Белсэн иде ничек сагындым

Син бер генэ гомер минем очен
Мин бит сина гомер табыны
Югалтасым килми мэхэббэтем
Килче мина килче ягымлым

Йорэклэрдэн чыккан сою жыры
Гомер жырларыбыз дип язылды
Кунел елый буген жырсыз гына
Мин бит сине шундый сагындым

Син бер генэ гомер минем очен
Мин бит сина гомер табыны
Югалтасым килми мэхэббэтем
Килче мина килче ягымлым
```

# Перевод

### Ландыш Нигметжанова - При встрече

```
При встрече глаза поцеловались,
К губам моим коснулись губы твои,
Ах, хотелось бы вернуться к тому мгновению,
Знал бы ты, как я скучала.

Ты один на всю жизнь для меня,
Я ведь тебе на всю жизнь поклоняюсь,
Не хочу терять любовь мою,
Приходи ко мне, приходи, милый мой.

Когда обещание с обещанием встретились,
Наша общая звезда засияла,
В ночах ищу ту звезду,
Знал бы ты, как я скучала.

Ты один на всю жизнь для меня,
Я ведь тебе на всю жизнь поклоняюсь,
Не хочу терять любовь мою,
Приходи ко мне, приходи, милый мой.

Песня любви, вышедшая из сердец,
Как песни нашей жизни записалась,
Душа плачет сегодня без песни лишь,
Я ведь тебя так сильно скучала.

Ты один на всю жизнь для меня,
Я ведь тебе на всю жизнь поклоняюсь,
Не хочу терять любовь мою,
Приходи ко мне, приходи, милый мой.
```
