# Оригинал

### Лилия Гыйбадуллина / Илнур Ганиев - Син очар канатым

```
1. Хыялым каршына,
Мин сиңа ашкынам.
Юлларга ак томан сарылган.
Таңнарым атса да,
Кояшым батса да,
Мин сине, бер сине сагынам.

2. Син - минем сагышым,
Гомерлек табышым.
Нигә соң соңарып табылдың?
Язмышлар яралы,
Һай текә (текәдер) ярлары.
Мин сине өзелеп сагындым.

3. Гомерләр узганчы,
Йөрәкләр тузганчы,
Мәңгегә, гомергә яраттым.
Хыяллар күгендә
Без бергә, без бергә.
Син - минем очар пар канатым.
```

# Перевод

### Лилия Гибадуллина / Ильнур Ганиев - Ты мое крыло свободы

```
1. Перед моим видением,
Я падаю к тебе.
По дорогам белый туман.
Даже если взойдет мое утро,
Даже если зайдет мое солнце,
Я скучаю только по тебе.

2. Ты - моя тоска,
Моя находка на всю жизнь.
Почему же ты пришел так поздно?
Раны судьбы,
О, как болят (болят же) раны.
Я в отчаянии скучал по тебе.

3. Пока жизни проходят,
Пока сердца насыщаются,
Я люблю вечно, всю жизнь.
В небе мечты
Мы вместе, мы вместе.
Ты - мое парное крыло свободы.
```
