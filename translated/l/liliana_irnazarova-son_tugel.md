# Оригинал

### Лилиана Ирназарова - Сон тугел

```
Упкэлэмэ бэгърем, упкэлэмэ,
Сине шулай сонлап тапканга.
Жэйлэр, козлэр, кышлар утеп китеп,
Язын горлэвеклэр акканда.

Сон тугел эле, жаным, яратырга,
Сон тугел эле соеп янырга.
Калган гомерлэрне узебез очен, жаным,
Ярыйдыр бит яшэп каларга.

Сузлэреннэн тугел, кузлэреннэн
Анлыйм синен тормыш юлынны.
Мэхэббэт дип белеп арыганнан
Тирэн яраларын уелды.

Сон тугел эле, жаным, яратырга,
Сон тугел эле соеп янырга.
Калган гомерлэрне узебез очен, жаным,
Ярыйдыр бит яшэп каларга.

Алмагачлар ялгыз чэчэк ата,
Матур козлэр килсэ янадан.
Безнен хислэр хич тэ ялгышмасын,
Синнэн матур чэчэк таба алмам.

Сон тугел эле, жаным, яратырга,
Сон тугел эле соеп янырга.
Калган гомерлэрне узебез очен, жаным,
Ярыйдыр бит яшэп каларга.
```

# Перевод

### Лилиана Ирназарова - Не конец

```
Не сердись, сердце моё, не сердись,
Что тебя так поздно нашла.
Лето, осень, зима пролетели,
Когда весной ручьи зажурчали.

Не конец ещё, душа моя, любить,
Не конец ещё гореть в любви.
Оставшуюся жизнь для нас двоих, родной,
Стоит ведь прожить до конца.

Не по словам, а по глазам твоим
Понимаю твой жизненный путь.
От боли, что любовью зовётся,
Глубокие раны зажили.

Не конец ещё, душа моя, любить,
Не конец ещё гореть в любви.
Оставшуюся жизнь для нас двоих, родной,
Стоит ведь прожить до конца.

Яблони в одиночестве цветут,
Когда красивая осень вновь придёт.
Наши чувства пусть не ошибаются,
Прекраснее тебя цветка не найду.

Не конец ещё, душа моя, любить,
Не конец ещё гореть в любви.
Оставшуюся жизнь для нас двоих, родной,
Стоит ведь прожить до конца.
```
