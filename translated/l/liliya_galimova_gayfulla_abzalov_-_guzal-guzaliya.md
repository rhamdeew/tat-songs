# Оригинал

### Лилия Галимова / Гайфулла Абзалов - Гүзәл-Гүзәлия

```
Күзем төште сылу-чибөргө
Менә кайгы - инде нишләргә?!
Айлы кичләрен дә,
Ямьле көннәрен дө
Сине уйлыйм, Гүзөл-Гүзөлем.

Әй! Гүзәл-Гүзәлия,
Аңламыйсың син нигә?
Әй! Гүзәл-Гүзәлия,
Яратмыйсың син нигә?

Сөю утында мин, Гүзәлем,
Сине генә көткән йөрәгем,
Бик ялкынлы икән,
Янып китмәс микән,
Сине уйлыйм, Гүзөл-Гүзөлем.

Әй! Гүзәл-Гүзәлия,
Аңламыйсың син нигә?
Әй! Гүзәл-Гүзәлия,
Яратмыйсың син нигә?
```

# Перевод

### Лилия Галимова / Гайфулла Абзалов - Гузель-Гузелия

```
Угас мой взор, красавица-роза,
Вот горе - ну что теперь делать?!
В лунные ночи,
В прекрасные дни
Думаю о тебе, Гузель-Гузель моя.

Ай! Гузель-Гузелия,
Не понимаешь ты почему?
Ай! Гузель-Гузелия,
Не любишь ты почему?

В пламени любви, моя Гузель,
Моё сердце, ждущее только тебя,
Очень пламенне оказалось,
Не сгорело бы ли оно,
Думаю о тебе, Гузель-Гузель моя.

Ай! Гузель-Гузелия,
Не понимаешь ты почему?
Ай! Гузель-Гузелия,
Не любишь ты почему?
```
