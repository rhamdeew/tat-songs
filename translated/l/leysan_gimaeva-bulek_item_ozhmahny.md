# Оригинал

### Лейсан Гимаева - Булэк итэм ожмахны

```
Кара кузем минем,
Нурлы йозем, кайда син?
Мэхэббэттэн мин тилерэм,
Гел сине генэ эзлим.

Менэ кошлар, урманнар,
Кубэлэклэр, йолдызлар.
Шушы ожмах матурлыгын
Булэк итэм мин сина.

Яндырдын, койдерден,
Утка эйлэндерден бит.
Кояш булып жылыт мине,
Сагындырдын бит инде.

Менэ кошлар, урманнар,
Кубэлэклэр, йолдызлар.
Шушы ожмах матурлыгын
Булэк итэм мин сина.

Сине генэ котэм,
Синдэ былбыл тавышы.
Хаман йорим сине эзлэп,
Котеп таннар атканын.

Менэ кошлар, урманнар,
Кубэлэклэр, йолдызлар.
Шушы ожмах матурлыгын
Булэк итэм мин сина.
```

# Перевод

### Лейсан Гимаева - Дарю тебе красоту рая

```
Черные глаза мои,
Лучистое лицо, где ты?
От любви я схожу с ума,
Всё тебя одного ищу.

Вот птицы, леса,
Бабочки, звёзды.
Эту райскую красоту
Дарю тебе я.

Ты сжёг меня, испепелил,
В огонь превратил ведь.
Солнцем стань, согрей меня,
Ты заставил тосковать.

Вот птицы, леса,
Бабочки, звёзды.
Эту райскую красоту
Дарю тебе я.

Только тебя жду,
В тебе соловья голос.
Всё хожу, тебя ищу,
Ждёт рассвета восход.

Вот птицы, леса,
Бабочки, звёзды.
Эту райскую красоту
Дарю тебе я.
```
