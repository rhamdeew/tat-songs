# Оригинал

### Люция Әблиева / Эльвира Нигъмәтуллина - Иртән

```
Уяна иртән иртүк,
Сөрәнли: «Кик-ри-күк!»
Әтәчкә башка эш - чүп,
Тавык-чебешләре күп.

«Кик-ри-күк!», «Кыт-кыт-кыт!»
Ничек тиз уза вакыт?!
«Чип-чип-чип-чип-чип-чип-чип»
Чык суын куйдык эчеп!

«Кыт-кыт» кытлый тавыклар,
Тулымы,- дип,- савытлар?
«Чип-чип» чили чебиләр,
Бернәрсә дә белмиләр.

«Кик-ри-күк!», «Кыт-кыт-кыт!»
Ничек тиз уза вакыт?!
«Чип-чип-чип-чип-чип-чип-чип»
Чык суын куйдык эчеп!
```

# Перевод

### Люция Абляева / Эльвира Нигъмәтуллина - Рано утром

```
Рано утром просыпается петух,
Кричит: «Ки-ри-ку!»
Для петуха другая работа - соломинка,
Куры и цыплята много.

«Ки-ри-ку!», «Кыт-кыт-кыт!»
Как быстро проходит время?!
«Чип-чип-чип-чип-чип-чип-чип»
Вышли, водички испили!

«Кыт-кыт» квохчут куры,
Наполним, - спрашивают, - посуду?
«Чип-чип» щебечут птенцы,
Ничего не знают.

«Ки-ри-ку!», «Кыт-кыт-кыт!»
Как быстро проходит время?!
«Чип-чип-чип-чип-чип-чип-чип»
Вышли, водички испили!
```
