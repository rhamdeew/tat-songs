# Оригинал

### Илсия Бадретдинова - Елый икэн ирлэр дэ

```
Ометсездэн чара таба алмый,
Эрну сонгы чиккэ житкэндэ.
Куз яшлэрен курсэтмичэ качып,
Елый икэн, елый ирлэр дэ.

Язмыш белэн кочен сынашканда,
Уземнеке итэм,- дигэндэ
Куз яшлэрен курсэтмичэ качып,
Елый икэн, елый ирлэрдэ.

Мэнге бергэ диеп вэгъдэ биргэн
Ярлар ятлар белэн утэндэ.
"Мин конлэшмим!- диеп кукрэк кагып,
Елый икэн, елый ирлэр дэ.
```

# Перевод

### Илсия Бадретдинова - Плачут оказывается мужчины тоже

```
От безнадёжности не найдя выход,
Когда терпение дошло до предела,
Не показывая слёз, убегая,
Плачут оказывается, плачут мужчины тоже.

Когда с судьбой силой меряясь,
Сказав "Сделаю это своим",
Не показывая слёз, убегая,
Плачут оказывается, плачут мужчины тоже.

Обещавшие быть вместе навеки
Любимые с другими уходят.
"Я не смирюсь!" - грудью бия,
Плачут оказывается, плачут мужчины тоже.
```
