# Оригинал

### Асаф Валиев - Челтер чишмэ буенда

```
Жанга шифалы сулары
Бормалы су буйлары
Сойлэшеп сузем бетмэде
Курэп кузем туймады

Челтер-челтер чишмэ ага
Кызлар жырлап су ала
Су алганда суырып упсэм
Упкэн эзлэре кала

Жырлый-жырлый су алганда
Чилэклэре тиз тула
Чишмэ суы эчкэн кызнын
Кунеле чибэр була

Челтер-челтер чишмэ ага
Кызлар жырлап су ала
Су алганда суырып упсэм
Упкэн эзлэре кала

Чишмэ буена кил але
Чишмэ белэн серлэшик
Бабайлардан калган гадэт
Гомерлеккэ берлэшик

Челтер-челтер чишмэ ага
Кызлар жырлап су ала
Су алганда суырып упсэм
Упкэн эзлэре кала
```

# Перевод

### Асаф Валиев - У журчащего родника

```
Для души целебные воды,
Извилистые берега реки,
Разговаривая, слово не кончалось,
Глядя, глаза не насыщались.

Журча-журча родник течет,
Девушки поют, воду берут,
Когда воду брала, дразня поцеловал,
Обиженные следы остались.

Напевая-напевая, когда воду брали,
Ведра быстро наполнялись,
У девушки, что родниковой воды попила,
Душа красивой становится.

Журча-журча родник течет,
Девушки поют, воду берут,
Когда воду брала, дразня поцеловал,
Обиженные следы остались.

К роднику приходи-ка,
С родником поделимся тайнами,
Обычай, оставшийся от дедов,
На всю жизнь объединимся.

Журча-журча родник течет,
Девушки поют, воду берут,
Когда воду брала, дразня поцеловал,
Обиженные следы остались.
```
