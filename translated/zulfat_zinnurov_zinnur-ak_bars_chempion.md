# Оригинал

### Зульфат Зиннуров Зиннур - Ак Барс чемпион

```
Ооо.. Ак Барс чемпион...

Тынды кебек донья кичкэ кадэр,
Сабыр гына кочне саклагыз.
Ут шикелле бозлар тагын кызар,
Уйный буген, уйный Ак Барс.

Ышанам мин ышанам,
Ышанудан туктамам.
Ышанучы - миллион,
Тик Ак Барс чемпион.

Ооо.. Ак Барс чемпион...

Боз сарае шаулый дингез кебек,
Бездэн дэ коч керсен уенга.
Тавышын бир, якла Ак Барсны,
Тамчыдан бит куллэр жыела.

Ышанам мин ышанам,
Ышанудан туктамам.
Ышанучы - миллион,
Тик Ак Барс чемпион.

Ооо.. Ак Барс чемпион...

Уен кыза э, ялкын гуя
Йорэклэрдэн чыга ургылып.
Ботен Казан, ботен Татарстан
Кабатлыйбыз эйдэ бер булып.

Ышанам мин ышанам,
Ышанудан туктамам.
Ышанучы - миллион,
Тик Ак Барс чемпион.

Ооо.. Ак Барс чемпион...
```

# Перевод

### Зульфат Зиннуров Зиннур - Ак Барс чемпион

```
Ооо.. Ак Барс чемпион...

Тихо словно мир до самого вечера,
Терпеливо силы сохраняйте.
Как огонь на льду снова разгорится,
Играет сегодня, играет Ак Барс.

Верю я, верю,
Верить не перестаю.
Верящих - миллион,
Только Ак Барс чемпион.

Ооо.. Ак Барс чемпион...

Ледовый дворец шумит как море,
От нас тоже сила входит в игру.
Голос подай, поддержи Ак Барс,
Из капель ведь реки собираются.

Верю я, верю,
Верить не перестаю.
Верящих - миллион,
Только Ак Барс чемпион.

Ооо.. Ак Барс чемпион...

Игра накаляется, пламя льётся
Из сердец выходя, бурля.
Вся Казань, весь Татарстан
Повторяем давайте воедино.

Верю я, верю,
Верить не перестаю.
Верящих - миллион,
Только Ак Барс чемпион.

Ооо.. Ак Барс чемпион...
```
