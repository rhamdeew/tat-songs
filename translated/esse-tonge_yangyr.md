# Оригинал

### Эссе - Тонге янгыр

```
Тоне буе янгыр яуды,
Чыландырды урамнарны.
Тагын юды бездэн калган
Юлдагы тузаннарны.

Янгыр тамчылары
Шакый тубэлэрне, калайларны.
Онытмагыз дия кебек
Булган шатлык-кайгыларны.

Уткэн гомер искэ тошэ
Ерактагы утка ошап.
Яннарымда син булганга,
Шатлыгымнан убэм сыйпап.

Янгыр тамчылары
Шакый тубэлэрне, калайларны.
Онытмагыз дия кебек
Булган шатлык-кайгыларны.

Ява тирэ тонге янгыр,
Иртэн кояш нур сибэ.
Курэсен шул авырлыкка
Мэхэббэте булган тузэр.

Янгыр тамчылары
Шакый тубэлэрне, калайларны.
Онытмагыз дия кебек
Булган шатлык-кайгыларны.
```

# Перевод

### Эссе - Ночной дождь

```
Всю ночь шёл дождь,
Промочил улицы.
Смыл оставшуюся
Дорожную пыль.

Капли дождя
Стучат по крышам, по жести.
Словно просят не забывать
Бывшие радости и печали.

Прошедшая жизнь вспоминается,
Как далёкий огонёк.
Что ты была рядом со мной —
От радости себя обнимаю.

Капли дождя
Стучат по крышам, по жести.
Словно просят не забывать
Бывшие радости и печали.

Льёт вокруг ночной дождь,
Утром солнце лучи рассыплет.
Увидишь — той печали
Любовь стала исцелением.

Капли дождя
Стучат по крышам, по жести.
Словно просят не забывать
Бывшие радости и печали.
```
