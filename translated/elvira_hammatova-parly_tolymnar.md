# Оригинал

### Эльвира Хамматова - Парлы толымнар

```
Озын кара чэчлэренне
Салдын икегэ булеп.
Парлы бэхетлэр телэден
Парлы толымнар уреп.

Парлы, парлы толымнарын
Синен белэн мин идем.
Бэхетлэрен пар булсын дип
Парлы толымнар урден.

Толымнар кук озын булсын
Без утэсе гомер бергэ.
Гел янэшэ барса иде,
Аерылмыйча гомергэ.

Парлы, парлы толымнарын
Синен белэн мин идем.
Бэхетлэрен пар булсын дип
Парлы толымнар урден.

Иртэн бер тон назлы куллар
Толымнарын таратып.
Гомергэ бергэ булырга
Йорэк белэн яратып.

Парлы, парлы толымнарын
Синен белэн мин идем.
Бэхетлэрен пар булсын дип
Парлы толымнар урден.
```

# Перевод

### Эльвира Хамматова - Жемчужные волны

```
Длинные черные волосы твои
Разделила надвое.
Жемчужное счастье желая,
Жемчужные волны плетя.

Жемчужные, жемчужные волны
С тобой я создавала.
Чтобы счастье парным было,
Жемчужные волны заплела.

Пусть волны долго длятся,
Мы проведем жизнь вместе.
Всё рядом бы шли,
Не расставаясь навеки.

Жемчужные, жемчужные волны
С тобой я создавала.
Чтобы счастье парным было,
Жемчужные волны заплела.

Утром одним движеньем нежные руки
Волны расплетают.
Чтобы навек быть вместе,
Сердцем любя.

Жемчужные, жемчужные волны
С тобой я создавала.
Чтобы счастье парным было,
Жемчужные волны заплела.
```
