# Оригинал

### Эльнара Нурмухаметова - Тик син диеп

```
Тик син диеп, син диеп,
Янып-коеп, утэлэр гомерлэрем.
Уткэн гомеремне, гомеремне уйлап,
Ярсына кунеллэрем.

Тик син диеп, син диеп,
Хислэр оеп, утлар ягам наз остэп,
Ни хэл итим, уз-уземне,
Уз-уземне яраттыра алмыйм кочлэп.

Тик син диеп, син диеп озгэлэнэм,
Ут йотам, ут йотам иртэ-кичен.
Ике гомер, ике гомер яшэсэм дэ,
Янармын синен очен.

Тик син диеп, син диеп,
Жаным коеп, нэумезлэнеп, кимлэнеп.
Йоргэн булам берэр, булам берэр заман
Анлар диеп сойлэнеп.

Тик син диеп, син диеп,
Озелэм дэ, син дип тагын жан ата.
Беркем дэ кузгэ, кузгэ куренми,
Сине генэ яратам.

Тик син диеп, син диеп озгэлэнэм,
Ут йотам, ут йотам иртэ-кичен.
Ике гомер, ике гомер яшэсэм дэ,
Янармын синен очен.

Тик син диеп, син диеп,
Янып-коеп, утэлэр гомерлэрем.
Уткэн гомеремне, гомеремне уйлап,
Ярсына кунеллэрем.

Тик син диеп, син диеп,
Хислэр оеп, утлар ягам наз остэп,
Ни хэл итим, уз-уземне,
Уз-уземне яраттыра алмыйм кочлэп.

Тик син диеп, син диеп озгэлэнэм,
Ут йотам, ут йотам иртэ-кичен.
Ике гомер, ике гомер яшэсэм дэ,
Янармын синен очен.
```

# Перевод

### Эльнара Нурмухаметова - Только тебя говоря

```

Прошедшую жизнь, жизнь вспоминая,
Разрывается душа моя.

Только тебя говоря, тебя говоря,
Чувства копя, огни зажигаю нежные,
Что делать мне с самой собой,
Себя заставить полюбить не могу.

Только тебя говоря, тебя говоря, томлюсь,
Огонь глотаю, огонь глотаю утром-вечером.
Две жизни, две жизни если проживу,
Буду гореть ради тебя.

Только тебя говоря, тебя говоря,
Душа сгорает, изнемогая, тоскуя.
Беспокойной становлюсь когда-то,
Говоря о них.

Только тебя говоря, тебя говоря,
Страдаю и снова душа взлетает к тебе.
Никого глазам, глазам не видно,
Только тебя люблю.

Только тебя говоря, тебя говоря, томлюсь,
```
