# Оригинал

### Айдар Галимов - Бодрэ таллар

```
Ашыкма син! Ерагайма!
Уткэннэрнен монын кем тынлар?
Балачагын горлэп уткэн инеш буйларында
Зарыгып котэ сине, сагынып, бодрэ таллар.

Ашыкма син! Тайгак юллар.
Хэл алырга ирек бирми алар.
Мен шифалы булыш белэн сине дэваларга
Офыкларга карап котэ бодрэ таллар.

Ашыкма син ваз кичэргэ
Омет булып сызылып ата таннар.
Заманынын давылында хакын юк улэргэ
Сине сагынып котэ монда бодрэ таллар.

Онытма син — гомер кыска
Кошлар булып куктэ оча еллар.
Гонахларын куптер диеп, синен очен hаман
Су буенда намаз укый бодрэ таллар. 

Ашыкма син! Ерагайма!
Уткэннэрнен монын кем жырлар!
Бала чагын горлэп уткэн инеш буйларында
Зарыгып котэ сине, сагынып, бодрэ таллар.
```

# Перевод

### Айдар Галимов - Кудрявые ивы

```
Не спеши! Не удаляйся!
Кто услышит мелодию прошедших дней?
На берегах, где детство прожил гордо,
Печалясь ждут тебя, тоскуя, кудрявые ивы.

Не спеши! Дороги скользкие.
Взять передышку не дают они.
Тысячами целебных трав тебя лечить
На горизонты глядя, ждут кудрявые ивы.

Не спеши отчаиваться —
Надеждой тянутся рассветы.
В буре времени нет права умирать,
Тебя скучая ждут здесь кудрявые ивы.

Не забывай — жизнь коротка,
Птицами в небе летят года.
Грехов много, говорят, но за тебя всё ещё
У воды молитву читают кудрявые ивы.

Не спеши! Не удаляйся!
Кто споёт мелодию прошедших дней!
На берегах, где детство прожил гордо,
Печалясь ждут тебя, тоскуя, кудрявые ивы.
```
