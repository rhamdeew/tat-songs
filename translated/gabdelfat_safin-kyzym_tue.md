# Оригинал

### Габделфат Сафин - Кызым туе

```
Агым сулар кебек гомер узган..
Тоялмыйда уткэн ни ара...
Буген кызым яна тормыш башлый,
Буген бездэ горлэп туй бара

Кызым туе, кызым туе
Дус-туганнар жыелган
Кабатланмас бу табынга
Яшьлек юлы сузылган...

Сою-назлар булып яшьлек тама..
Чыда кунел, чыда туз генэ
Кычкыралар эче-эче диеп
Тик бу бары татлы суз генэ..

Кызым туе, кызым туе
Дус-туганнар жыелган
Кабатланмас бу табынга
Яшьлек юлы сузылган...

Бу язмышнын шашып яшнэве дер
Мэххэббэтнен кабат балкуы
Яшьлеклэрдэ яккан учакларнын
килэчэккэ кучкэн ялкыны...

Кызым туе, кызым туе
Дус-туганнар жыелган
Кабатланмас бу табынга
Яшьлек юлы сузылган...
```

# Перевод

### Габделфат Сафин - Свадьба дочери

```
Как текущие воды жизнь прошла...
Не насытишься прошедшим...
Сегодня дочь моя новую жизнь начинает,
Сегодня у нас торжественно свадьба идет.

Свадьба дочери, свадьба дочери,
Друзья-родные собрались,
Не повторится это торжество -
Юности путь протянулся...

Любовью-лаской каплет молодость...
Терпит душа, терпит лишь,
Кричат "горько-горько",
Но это все сладкие слова лишь...

Свадьба дочери, свадьба дочери,
Друзья-родные собрались,
Не повторится это торжество -
Юности путь протянулся...

Этой судьбы спешное сияние -
Любви новое свечение,
В юности зажженных костров
В будущее перешедшее пламя...

Свадьба дочери, свадьба дочери,
Друзья-родные собрались,
Не повторится это торжество -
Юности путь протянулся...
```
