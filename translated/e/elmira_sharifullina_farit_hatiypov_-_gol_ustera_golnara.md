# Оригинал

### Эльмира Шәрифуллина / Фәрит Хатыйпов - Гөл үстерә Гөлнара

```
Гөлнара күзен ачканчы,
Якты уянган икән.
Якты иртә уянганга,
Гөлләр куанган икән,

Гөл чәчәге елтыр-елтыр
Елмая Гөлнарага.
Чиләгенә җыйган яңгыр
Суын сибә ул аңа.

Гел елмаеп торсын гөлләр,
Гөлнара торган төшкә.
Гөлләргә су сибә-сибә,
Өйрәнә кызлар эшкә...
```

# Перевод

### Эльмира Шәрифуллина / Фарит Хатыпов - Цветы выращивает Гульнара

```
Когда Гульнара открыла глаза,
Светло проснулось утро.
Светло проснувшемуся утру
Рады были цветы.

Цветок цветка светло-светло
Улыбается Гульнаре.
Собравшаяся роса в кувшине
Она льёт на неё.

Пусть всегда улыбаются цветы,
Гульнара стоит в сне.
На цветы льётся вода-вода,
Девушки учатся работе...
```
