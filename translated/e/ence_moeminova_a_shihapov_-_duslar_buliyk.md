# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Дуслар булыйк

```
Сукмаклар төрле булса да,
Бер безнең юллар.
Дуслар булыйк гомергә дип
Бирешик куллар.

Кушымта:
Дуслар булыйк,
Дуслар булыйк,
Булыйк гел бергә.
Дуслар булыйк,
Дуслар булыйк,
Гомер-гомергә.

Бердәм көч таулар күчерә,
Күчерә диңгез.
Тау күчерер дуслар булыйк,
Дуслар булыйк без.

Берни булмас безнең өчен
Давыллар, җилләр.
Без аларга каршы барып
Яуларбыз үрләр.
```

# Перевод

### Энҗе Мөэминова / А. Шихапов - Будем друзьями

```
Пути, пусть и разные,
Но один у нас путь.
Давайте будем друзьями
На всю жизнь, подадим руки.

Припев:
Будем друзьями,
Будем друзьями,
Всегда будем вместе.
Будем друзьями,
Будем друзьями,
Из жизни в жизнь.

Единая сила горы переносит,
Переносит моря.
Горы перенесем, будем друзьями,
Мы будем друзьями.

Ничего не будет для нас
Бурь, ветров.
Мы против них пойдем
И завоюем высоты.
```
