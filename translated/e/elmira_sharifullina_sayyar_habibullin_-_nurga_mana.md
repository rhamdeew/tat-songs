# Оригинал

### Эльмира Шәрифуллина / Сәйяр Хәбибуллин - Нурга мана

```
Күз карашың таң нурыдай
Мәрхәмәтле, сөйкемле.
Бер кеше дә елмаялмый
Миңа синең шикелле.

Кушымта:
Көләч йөзең бар нәрсәдән
Ягымлырак күренә.
Шул килүдән кереп калдың
Йөрәгемнең түренә.

Җырларың белән зурладың,
Моңнарыңа исердем.
Кемдер татый да алмаган
Сихри тойгы кичердем.

Күз карашың уйга салды,
Наз биләде җанымны.
Бер елмаеп карауларың
Нурга мана таңымны.
```

# Перевод

### Эльмира Шарифуллина / Саяр Хабибуллин - Нурга мана

```
Твой взгляд - как утренний свет,
Милосердный, ласковый.
Никто больше не улыбается мне
Таким образом, как ты.

Припев:
Твое плаксивое лицо
Выглядит еще симпатичнее всего.
Ты вошла в это приближение
В сердце моем.

Ты возвысила меня песнями,
Я утонула в твоих звуках.
Ощущение волшебства
Кто-то пробовал, но не взял.

Твой взгляд заставил задуматься,
Ласка захватила душу.
Одним улыбкой ты смотришь
Ты озарила мою зорю.
```
