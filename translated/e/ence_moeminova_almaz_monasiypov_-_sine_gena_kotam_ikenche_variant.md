# Оригинал

### Энҗе Мөэминова / Алмаз Монасыйпов - Сине генә көтәм (икенче вариант)

```
Кайвакытта телим
Уйламаска сине.
Онытырга телим
Үткәннәрне.
Ләкин мөмкин түгел -
Йөрәк каршы килә:
Никтер фәкать сине
Юксынам.

Сине генә көтәм,
Сине генә сагынам,
Синнән башка тормыш юк сыман.

Күз алдымда тора
Гәүдәләрең синең,
Уйчан карый торган
Күзләрең.
Ярсу йөрәк белән
Күрешү көнен көтәм.
Никтер фәкать сине
Юксынам.
```

# Перевод

### Энҗе Мөэминова / Алмаз Монасыйпов - Жду только тебя (второй вариант)

```
Иногда желаю
Не думать о тебе.
Хочу забыть
Прошлые дни.
Но это невозможно -
Сердце противится:
Как будто специально
Ты мне снишься.

Жду только тебя,
Тоскую только по тебе,
Без тебя жизнь - будто нет.

Перед глазами стоит
Твоя фигура,
Твои задумчивые глаза,
Смотрящие на меня.
С любящим сердцем
Жду день встречи.
Как будто специально
Ты мне снишься.
```
