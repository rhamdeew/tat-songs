# Оригинал

### Эльмира Шәрифуллина / Фирзәр Мортазин - Тик сиңа

```
Давылларны узып,
Йолаларны бозып,
Мин сиңа, мин сиңа киләмен.
Җанымның бар назың,
Кайгымның бик азын
Тик сиңа, тик сиңа бирәмен.

Кояштан да көчле,
Моңнардан да хисле
Йөрәгем утына төрәмен.
Җанымның бар назын,
Кайгымның бик азын
Тик сиңа, тик сиңа бирәмен.

Төшеңә керәмен,
Мин сиңа киләмен,
Бәхетне бер синдә күрәмен.
Кайгымның бик азын,
Җанымның бар назын
Тик сиңа, тик сиңа бирәмен.
```

# Перевод

### Эльмира Шәрифуллина / Фирзәр Мортазин - Только тебе

```
Пройдя сквозь бури,
Сломав льдины,
Я иду к тебе, я иду к тебе.
Всю нежность души моей,
Самую малую боль
Только тебе, только тебе дарю.

Сильнее солнца,
Чувственнее стонов,
Я разожгу сердечный огонь.
Всю нежность души моей,
Самую малую боль
Только тебе, только тебе дарю.

В твой сон вхожу,
Я иду к тебе,
Счастье вижу только в тебе.
Самую малую боль,
Всю нежность души моей
Только тебе, только тебе дарю.
```
