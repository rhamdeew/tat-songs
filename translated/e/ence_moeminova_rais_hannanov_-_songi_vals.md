# Оригинал

### Энҗе Мөэминова / Рәис Ханнанов - Соңгы вальс

```
Парталар артында калдылар,
Калдылар балачак еллары.
Аттестат кулларда,
Иртәгә чыгарга
Безгә зур, җаваплы юлларга.

Кушымта:
Тагын бер уйнасын,
Оркестр тагын бер уйнасын.
Гомергә онтылмас
Кадерле минутлар,
Озайсын беразга, озайсын!

Канатлы хыяллар күңелне
Еракка, еракка тарталар.
Мәктәпне сагынып
Без дә бер кайтырбыз,
Кошлар да әйләнеп кайталар.

Бакчада агачлар тирбәлә,
Гөлләрне уятып җил исә.
Инде таң сызыла,
Ал нурлар сузыла
Синең дә, минем дә юлыма.
```

# Перевод

### Энҗе Мөэминова / Раис Ханнанов - Последний вальс

```
Остались за партами,
Остались детские года.
Аттестат в руках,
Завтра выйти
На большие, ответственные пути.

Припев:
Еще бы прозвучал,
Оркестр еще бы сыграл.
Драгоценные минуты
Не исчезнут из жизни,
Пусть продлятся немного, продлятся!

Крылатые мечты сердце
Увлекают вдаль, вдаль.
Тоскуя по школе
Мы тоже когда-то вернемся,
Птицы тоже возвращаются обратно.

В саду деревья колышатся,
Ветер будит цветы.
Уже рассвет рисуется,
Алые лучи протягиваются
На твой путь и на мой путь.
```
