# Оригинал

### Эльмира Шәрифуллина / Рәис Нәгыймов - Актанышым - ак илем

```
Актанышым - ак илем,
Ак илемдә - ак көнем.
Илдө-көндә, тирә-юньдә
Шөһрәт казанган җирем.

Актанышым - ак илем.
Канат чыгарган җирем.
Еллар буйлап күп йөредем,
Тапмадым сиңа тиңен.

Басуларың, кырларың -
Бәрәкәт иңгән җирем.
Болындагы чәчәкләрең
Күңелдә таба тиңен.

Актанышым - ак илем,
Ак илемдә - ак көнем.
Мәрхәмәтле кочагыңда
Оятлы булмас йөзем.
```

# Перевод

### Эльмира Шәрифуллина / Райс Нагимов - Актаныш - белое мое поле

```
Актаныш мое - белое поле мое,
В белом поле - белый день мой.
Во всех краях, во всех сторонах
Прославленный я талантливый певец.

Актаныш мое - белое поле мое,
Крылатый я певец.
Много лет ходил я повсюду,
Тебе равного не нашёл я.

Горы твои, холмы твои -
Благословенные земли мои.
Цветы твои в глубине
На душе ищут покой.

Актаныш мое - белое поле мое,
В белом поле - белый день мой.
В твоём милосердном гнезде
Не стыдно будет лицу моему.
```
