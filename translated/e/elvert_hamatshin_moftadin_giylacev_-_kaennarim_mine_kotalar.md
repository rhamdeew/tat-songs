# Оригинал

### Эльверт Хәмәтшин / Мөфтәдин Гыйләҗев - Каеннарым мине көтәләр

```
Үстем туган якта,
Тыңлап балачакта
Каеннарның моңлы җырларын.
Онытмамын һич тә,
Булыр мәңге истә
Ак каенлы туган җирләрем.

Колагымда чыңлый,
Бер дә истән чыкмый
Таң атканда кошлар сайравы.
Сагышларым арта,
Үзләренә тарта
Каеннарның моңлы шаулавы.

Сагынып туган якны,
Гүзәл балачакны.
Сагышларда көннәр үтәләр.
Ә бит туган якта,
Гүзәл балачакта,
Каеннарым мине көтәләр.
```

# Перевод

### Эльверт Хаматшин / Мөфтидин Гилячев - Ждут меня мои дубы

```
Вырос я на родине,
Слушал в детстве я
Печальные песни дубов.
Никогда не забуду,
Останется в памяти навеки
Родная земля с белыми дубами.

В ушах звенит,
Не выходит из памяти
Пение птиц на рассвете.
Тоска моя растет,
Тянет к себе
Печальный шум дубов.

Тоскую по родине,
По прекрасному детству.
Дни проходят в тоске.
Ведь на родине-то,
В прекрасном детстве,
Ждут меня мои дубы.
```
