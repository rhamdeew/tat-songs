# Оригинал

### Энҗе Мөэминова / Роберт Андреев - Бишек җыры (сигезенче вариант)

```
Мендәреңә, юрганыңа
Ак тыш кидерәм.
Җырлый-җырлый күңелеңә
Аклык иңдерәм.

Кушымта:
Йокла инде, йокла, сабый,
Ак төшләр күреп,
Ак болытлар арасында
Уйнап йөр син, уйнап йөр син,
Ак колын кебек.
Әлли, бәлли, бәү...

Ак бишеккә салам сине,
Ак чаршау корам.
Аклык белән, сафлык белән,
Тулсын бар дөньяң!
```

# Перевод

### Энҗе Мөэминова / Роберт Андреев - Песня колыбельная (восьмая версия)

```
На грудь твою, на ходьбу твою
Я белый плащ накину.
Пою-пою в твое сердце
Я белоснежный свет волью.

Припев:
Спи уже, спи, младенец,
Видя белые сны,
Между белых облаков
Играй, играй ты, играй, играй,
Как белый снег.
Алли, балли, бау...

В белую колыбель тебя положу,
Белый свет пусть окружит.
Белизной и чистотой
Пусть весь мир твой будет полон!
```
