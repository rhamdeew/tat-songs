# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Килер ул

```
Без икәү утырган урында
Тын гына чәчәкләр тирбәлә.
Көмештәй коелган нурларда
Бизәкләр уйната күләгә.

Без икәү утырган урынга
Көмештәй ай нуры коела.
Мин синең хәлеңне аңлыйм дип
Әйтәдер шикелле тоела.

Без икәү утырган урында
Тын гына яфраклар серләшә.
Килер ул, килер ул, көт диеп
Йолдызлар җемелдәп эндәшә.
```

# Перевод

### Энже Мөменинова / А. Шихапов - Придёт он

```
Там, где мы с тобой сидели,
Тихо трепетали цветы.
В лучах, как серебро, горели
Узоры тени на воде.

Там, где мы с тобой сидели,
Серебристый свет луны струил.
Скажет странный ветерок, что, мол,
Я понимаю твою боль.

Там, где мы с тобой сидели,
Тихо листья шепчутся.
Придёт он, придёт, надеюсь, жду,
Звёзды тихо падают вниз.
```
