# Оригинал

### Эльвира Сафина / Риваль Хисмәтуллин - Тик кил генә!

```
Бар шатлыгым - сине яратуым,
Хыял белән яшим мин хәзер.
Йөрәккәем шашып яратудан
Яннарыңа очарга әзер.

Сөю уртак, тик юллар аерым,
Уйларымда тик син, тик син генә.
Сине күреп күңел бер сөенсен,
Сылтау табып кил, тик кил генә!

Бернәрсәне килми үзгәртәсем,
Мин сөюем белән бәхетле.
Син янымда булсаң, бәлки, әле,
Тәмен белмәс идем бәхетнең.

Сөю уртак, тик юллар аерым,
Уйларымда тик син, тик син генә.
Сине күреп күңел бер сөенсен,
Сылтау табып кил, тик кил генә!

Күңел "кил" ди, акыл аннан көлә,
Белә бит ул синең килмәсең.
Якты сөю, татлы сөю белән
Гомеремнең үтеп китәсен.

Сөю уртак, тик юллар аерым,
Уйларымда тик син, тик син генә.
Сине күреп күңел бер сөенсен,
Сылтау табып кил, тик кил генә!
```
