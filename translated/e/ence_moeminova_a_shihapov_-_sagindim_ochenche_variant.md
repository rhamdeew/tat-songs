# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Сагындым (өченче вариант)

```
Мин сагындым тавышыңны,
Сагындым сүзләреңне,
Сагындым гөлдәй керфекле
Сөйкемле күзләреңне.

Әллә синең тавышыңны
Сандугач алды микән?
Сүзләрең гөл арасында
Адашып калды микән?

Кушымта:
Гөлдәй керфекле күзләрең,
Сагышка күмелдеме?
Миңа киләсе сукмагың
Томанга төрелдеме?

Мин сагындым тавышыңны,
Сагындым сүзләреңне,
Сагындым гөлдәй керфекле
Сөйкемле күзләреңне.

Сандугачлар тавышыңны
Кайтарып бирер микән?
Гөлләр синең тавышыңны
Миңа җиткерер микән?

Кушымта:
Гөлдәй керфекле күзләрең
Сөеп карарлар микән?
Миңа килер сукмаклардан
Томан таралыр микән?
```

# Перевод

### Энҗе Мөэминова / А. Шихапов - Скучал (третий вариант)

```
Я скучал по твоему голосу,
Скучал по твоим словам,
Скучал по глазам твоим,
Что цветами ласкали взор.

Может, голос твой соловей
Забрал с собой в полет?
Может, слова твои в цветах
Затерялись средь волны?

Припев:
Глаза твои цветами ласковы,
В тоске утонул я?
Желанье к тебе прийти
Сквозь туман укрылся?

Я скучал по твоему голосу,
Скучал по твоим словам,
Скучал по глазам твоим,
Что цветами ласкали взор.

Может, соловьи тебе
Вернут голос мой в ответ?
Может, цветы донесут
Голос твой до сердца?

Припев:
Глаза твои цветами ласковы,
Взглянут ли на меня?
Придут ли туманы с тоской
И рассеет их земля?
```
