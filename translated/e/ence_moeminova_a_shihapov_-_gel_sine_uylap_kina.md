# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Гел сине уйлап кына

```
Зәй буйларында күк томан,
Күк томан болыт сыман.
Шул томаннар арасыннан
Килеп чыгарсың сыман.

Кушымта :
Зәй аръягында әрәмә,
Барам су буйлап кына.
Зәй буйларында йөрим мин
Гел сине уйлап кына!

Таң җилләре синең якка
Юл ярып исә сыман.
Алар да синең килүне
Сагынып көтә сыман.

Дулкыннар белән кушылып
Мин дә агармын сыман.
Шул дулкыннар арасында
Сине табармын сыман.
```

# Перевод

### Энже Мөэминова / А. Шихапов - Только думая о тебе

```
Вдоль берегов Зәя туман,
Небесный туман, как облако.
Из этих туманов словно
Выходишь ты.

Припев:
Позади Зәя - аромат,
Иду вдоль по воде.
Вдоль берегов Зәя хожу я
Только думая о тебе!

Утренние ветры к тебе
Открывают путь.
Они тоже ждут твоего прихода
С тоской.

Соединяясь с волнами
Я тоже светлею.
Среди этих волн
Нахожу тебя.
```
