# Оригинал

### Энҗе Мөэминова / Әхмәт Хәйретдинов - Сине генә көтәм (беренче вариант)

```
Уйламаска телим
Кайвакытта сине,
Онытырга телим
Үткәннәрне мин.
Ләкин мөмкин түгел,
Йөрәк каршы килә,
Никтер фәкать сине юксынам.

Сине генә көтәм,
Сине генә сагынам,
Синнән башка тормыш юк сыман.

Күз алдымда тора
Бөдрәләрең синең,
Уйчан карый торган
Коңгырт күзләрең.
Ярсу йөрәк белән
Күрешү көнен көтәм,
Никтер фәкать сине юксынам.

Сине генә көтәм,
Сине генә сагынам,
Синнән башка тормыш юк сыман.
```

# Перевод

### Энже Мөэминова / Ахмет Хайретдинов - Только тебя жду (первый вариант)

```
Хочу не думать
Иногда о тебе,
Хочу забыть
Прошлые дни.
Но это невозможно,
Сердце противится,
Ведь я тебя скучаю.

Только тебя жду,
Только тебя вспоминаю,
Без тебя будто жизни нет.

Передо мной стоит
Твой облик,
Размышляющий взгляд
Твоих темных глаз.
С тоскующим сердцем
Жду дня встречи,
Ведь я тебя скучаю.

Только тебя жду,
Только тебя вспоминаю,
Без тебя будто жизни нет.
```
