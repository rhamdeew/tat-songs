# Оригинал

### Элина Балагутдинова / Шахзат Размәтов - Бәхет сере

```
Бәхет ачкычларын табу -
Тормышның зур мәгънәсе.
Ачкыч кемдә, шулдыр инде -
Мәхаббәтнең иясе.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары...

Ике ягыңда фәрештә -
Әти-әниең булса.
Дус-туганнар янда булып,
Ярдәм кулларын сузса.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары...

Бәхеттер ул хәтта, дустым,
Бу дөньяда яшәүләр.
"Мин яратам", "Мин бәхетле"
Диеп көн дә дәшүләр.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары...
```

# Перевод

### Элина Балагутдинова / Шахзат Размәтов - Секрет счастья

```
Найти ключи от счастья -
Великий смысл жизни.
У кого ключи, тот и есть -
Владыка любви.

Ломаясь, любить,
Душой страдать,
Вот это в этом мире -
Быть счастливым...

Если по сторонам ангелы -
Твои родители.
Если друзья и родные рядом,
Протягивают руки помощи.

Ломаясь, любить,
Душой страдать,
Вот это в этом мире -
Быть счастливым...

Даже это счастья, дружок,
Жить на этом свете.
Говорить "Я люблю", "Я счастлив"
Каждый день жизни.

Ломаясь, любить,
Душой страдать,
Вот это в этом мире -
Быть счастливым...
```
