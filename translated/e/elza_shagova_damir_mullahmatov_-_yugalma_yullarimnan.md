# Оригинал

### Эльза Шагова / Дамир Муллахмәтов - Югалма юлларымнан

```
Югалма син юлларымнан,
Кояшым бул янымда.
Мәңге балкып, җылытып тор
Сөюем ялкынында.

Югалма син уйларымнан,
Яшә гел хыялымда.
Яктыртучы маягым бул
Йөрисе юлларымда.

Югалма син елларымнан,
Мең яшә күңелемдә.
Назларымны, сөюемне
Сакла син күңелеңдә.

Югалма син уйларымнан,
Яшә гел хыялымда.
Яктыртучы маягым бул
Йөрисе юлларымда.

Югалма син моңнарымнан,
Наз тулы йөрәгемнән.
Җырланасы җырларымның
Югалма юлларыннан.

Югалма син уйларымнан,
Яшә гел хыялымда.
Яктыртучы маягым бул
Йөрисе юлларымда.
```

# Перевод

### Эльза Шагова / Дамир Муллахмәтов - Не исчезай с моих дорог

```
Не исчезай с моих дорог,
Будь со мной, мое солнце.
Вечно сияй и согревай
Мою любовь трепетную.

Не исчезай из мыслей моих,
Живи в моих мечтах.
Будь маяком, что освещает
Путь мой дорогами.

Не исчезай из лет моих,
Живи в моем сердце.
Храни в душе мои ласки,
Любовь мою нежную.

Не исчезай из мыслей моих,
Живи в моих мечтах.
Будь маяком, что освещает
Путь мой дорогами.

Не исчезай из тоски моей,
Из сердца, полного ласки.
Не уходи из песен моих,
Песен, что звучат.

Не исчезай из мыслей моих,
Живи в моих мечтах.
Будь маяком, что освещает
Путь мой дорогами.
```
