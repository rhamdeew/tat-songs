# Оригинал

### Эльвира Тимергалиева / Адис Маликов - Килеп китсәң ни була?

```
Бик озаклап күзләреңә
Карап торасым килә.
Йөзләремдә сулышыңны
Кабат тоясым килә.

Ник ул кадәр тилмертәсең,
Килеп китсәң ни була?
Кайнар кочагыма алып
Кабат үпсәм ни була?

Шуннан соң, бәлки, өметем
Бераз сүрелер иде.
Сөя икән ләбаса диеп
Күңелем юаныр иде.

Ник ул кадәр тилмертәсең,
Килеп китсәң ни була?
Кайнар кочагыма алып
Кабат үпсәм ни була?
```
