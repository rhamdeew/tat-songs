# Оригинал

### Айдар Галимов - Ай алкалы татар кызы

```
Яшел бэбкэ яшел улэн,
Табигать тэ бэбкэ узе.
Су буенда казлар карый
Ай алкалы татар кызы.

Татар кызы, татар кызы,
Ай алкалы татар кызы.
Су буенда казлар карый
Ай алкалы татар кызы.

Талир тэнкэ, ефэк тасма
Толымына ургэн узе.
Карашыннан мон тугелэ
Ай алкалы татар кызы.

Татар кызы, татар кызы,
Ай алкалы татар кызы.
Карашыннан мон тугелэ
Ай алкалы татар кызы. 


Зэнгэр кузле, ап-ак казлар
Кангылдашып суда йозде.
Хыялымда бэбкэ саный,
Ай алкалы татар кызы.

Татар кызы, татар кызы,
Ай алкалы татар кызы.
Хыялымда бэбкэ саный,
Ай алкалы татар кызы.
```

# Перевод

### Айдар Галимов - Луноликая татарочка

```
Зелёные луга, зелёная трава,
И природа тоже сама зелёная.
У воды гусей пасёт
Луноликая татарочка.

Татарочка, татарочка,
Луноликая татарочка.
У воды гусей пасёт
Луноликая татарочка.

Ивовые монетки, шёлковые ленты
На рукава себе повязала.
От её взгляда мелодия льётся —
Луноликая татарочка.

Татарочка, татарочка,
Луноликая татарочка.
От её взгляда мелодия льётся —
Луноликая татарочка.

Синеглазые, белоснежные гуси
Плещутся в воде весело.
В моих мечтах красавицей кажется
Луноликая татарочка.

Татарочка, татарочка,
Луноликая татарочка.
В моих мечтах красавицей кажется
Луноликая татарочка.
```
