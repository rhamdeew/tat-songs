# Оригинал

### Илсия Бадретдинова - Хикмэт язмыш

```
Хикмэт язмыш сыный ботен яклап,
Ялгыштыра саклап хэр адымда.
Эгэр качсан, тота куып,
Язмышлардан булмый узып.
Шундый хикмэт, хикмэт язмыш.

Берозлексез уйнаса да,
Биткэ сугып кыйнаса да,
Нишлэп була? Безнен язмыш.
Берэу елый,берэу колэ,
Берэу соя, берэу коя.
Шундый тормыш... Хикмэт язмыш.

Хикмэт язмыш дингез кебек серле,
Син хэркемнен торле уз язмышы.
Кайчакларда тоташ аяз,
Кунеллэрдэ дэ хэрчак яз
Шундый хикмэт, хикмэт язмыш.

Берозлексез уйнаса да,
Биткэ сугып кыйнаса да,
Нишлэп була? Безнен язмыш.
Берэу елый,берэу колэ,
Берэу соя, берэу коя.
Шундый тормыш... Хикмэт язмыш.

Хикмэт язмыш ташлый ярдан ярга,
Сала яна яра хэр яна кон.
Булсын рэхэт, булсын газап.
Барысына тик бер жавап:
Шундый хикмэт, хикмэт язмыш.

Берозлексез уйнаса да,
Биткэ сугып кыйнаса да,
Нишлэп була? Безнен язмыш.
Берэу елый,берэу колэ,
Берэу соя, берэу коя.
Шундый тормыш... Хикмэт язмыш.
```

# Перевод

### Илсия Бадретдинова - Мудрая судьба

```
Мудрая судьба испытывает всех подряд,
Заблуждает, оберегая на каждом шагу.
Если убежишь, догонит преследуя,
От судьбы не уйти.
Такова мудрость, мудрая судьба.

Хоть без перерыва играет,
Хоть до смерти мучает,
Что поделать? Наша судьба.
Один плачет, другой смеётся,
Один любит, другой ненавидит.
Такова жизнь... Мудрая судьба.

Мудрая судьба тайна как море,
У каждого своя особая судьба.
Иногда сплошная ясность,
В сердцах же всегда весна
Такова мудрость, мудрая судьба.

Хоть без перерыва играет,
Хоть до смерти мучает,
Что поделать? Наша судьба.
Один плачет, другой смеётся,
Один любит, другой ненавидит.
Такова жизнь... Мудрая судьба.

Мудрая судьба бросает с берега на берег,
Наносит новую рану каждый новый день.
Будет ли радость, будет ли мука,
На всё лишь один ответ:
Такова мудрость, мудрая судьба.

Хоть без перерыва играет,
Хоть до смерти мучает,
Что поделать? Наша судьба.
Один плачет, другой смеётся,
Один любит, другой ненавидит.
Такова жизнь... Мудрая судьба.
```
