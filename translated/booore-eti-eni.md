# Оригинал

### Бооре - Эти-эни

```
Бу доньянын ин авыры -
Канатны каеруы,
Балаларны зур юлга
Энилэр озатуы.

Эти-эни, эни-эти -
Сез хэрчак минем кочем.
Беренче урында
Мин яшим сезнен очен.

Сез мине яратып устердегез,
Минем очен бик куп коч,тир туктегез.
Минем энинен йомшак куллары хэм уйлары
Мине шушы доньяда зур йорэкле ясады.
Этинен каты рухы, тирэн уйлы сузлэре
Миннэн шушы доньяда чын егет ясады.

Мин сезгэ гомер буе терэк булып торырмын,
Хэм сезгэ мин суз бирэм:
Йозегезгэ кызыллык китермэм.

Бу доньянын ин авыры -
Канатны каеруы,
Балаларны зур юлга
Энилэр озатуы.

Эти-эни, эни-эти -
Сез хэрчак минем кочем.
Беренче урында
Мин яшим сезнен очен.

Вакыт уза, лэкин оныта алмыйм мин синен кузлэренне,
Ин ягымлы энием,
Син минем ин ягымлы энием.
Гомер уза, лэкин оныта алмыйм мин синен сузлэренне,
Ин яраткан этием,
Син минем ин яраткан этием.

Кайда булсамда, мин сезнен белэн,
Мин сезгэ хэрчак терэк булам,
Мин хэрчак сезгэ терэк булам.
Мин сезнен белэн.

Кайда булсамда, мин сезнен белэн,
Мин сезгэ хэрчак терэк булам,
Мин хэрчак сезгэ терэк булам.
Мин сезнен белэн.

Бу доньянын ин авыры -
Канатны каеруы,
Балаларны зур юлга
Энилэр озатуы.

Эти-эни, эни-эти -
Сез хэрчак минем кочем.
Беренче урында
Мин яшим сезнен очен.
Сез хэрчак минем кочем
Беренче урында
Мин яшим сезнен очен
```
