# Оригинал

### Заһир Хәсәнов / Рамил Курамшин - Йомралы

```
Топольләр, башларын иеп,
Тирбәләләр җилләрдә.
Яшьлек гөлем чәчәк атты
Шушы гүзәл илләрдә.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.

Аккошлар килеп төшәләр
Акчарлак күлләренә.
Туган ягын сагынып кайта
Синең гүзәлләрең дә.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.

Шатлыкка, нурга төренеп
Синдә таңнар уяна.
Таңнар шулай мәңге балкыр,
Тыныч булсын бу дөнья.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.
```

# Перевод

### Заһир Хәсәнов / Рамил Курамшин - Йомралы

```
Тополя, склоняясь низко,
Ветром тихим колышась,
Мой цветок весенней юности
В этих краях распустился.

Йомралы, Йомралы,
Улицы у тебя извилистые.
По извилистым улицам твоим
Я ходил - и сердце не насытилось.

Лебеди садятся плавно
На озёра с белой чайкой.
По родному краю скучая,
И твои красавицы возвращаются.

Йомралы, Йомралы,
Улицы у тебя извилистые.
По извилистым улицам твоим
Я ходил - и сердце не насытилось.

В радости, в сиянье светлом
У тебя рассветы встают.
Пусть рассветы так вечно будут,
Пусть будет мирным этот светлый мир.

Йомралы, Йомралы,
Улицы у тебя извилистые.
По извилистым улицам твоим
Я ходил - и сердце не насытилось.
```
