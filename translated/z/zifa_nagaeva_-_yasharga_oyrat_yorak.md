```

Йөрәккә кадалган очып үткән угы.

Йотып зар-утны, җыеп бар укны.

Мин сиңа кирәкмен бүген, син миңа кирәк,

Җил килеп бәрә - чайкалмыйм, чайкалмыйм,
Сүз килеп бәрә - акланмыйм, акланмыйм.
Җанга авыр көннәремне хәтердән җуям,
Кисәкләп җыям йөрәкне, кисәкләп җыям.

Кисәкләп җыйдым шатлыкларны да,
Бәхетләрне дә җыйдым кисәкләп.
Сукмаклардан зур юлларга чыгар көч кирәк,
Алып барды мине йөрәк, барды җитәкләп.

Мин сиңа кирәкмен бүген, син миңа кирәк,

Кыялар аша да упкыннар үтә,
Мин кичәдә түгел инде, яңа көн көтә.
```

# Перевод

### Зифа Нагаева - Научи жить, сердце

```
С одной стороны - скала, с другой - обрыв,
Стрела пролетная вонзилась в сердце.
Устало сердце, устало, от меня устало,
Глотая жар скорби, собирая стрелы.

Я нужна тебе сегодня, ты нужен мне,
Научи снова жить, снова, сердце!
Не прошу, не жду помощи с чужих сторон.
Не отдала, не отдам сердце - чтоб топтали.
Ветер бьет - не качаюсь, не качаюсь,
Слово бьет - не оправдываюсь, не оправдываюсь.
Тяжелые дни души из памяти стираю,
По кусочкам собираю сердце, по кусочкам собираю.

По кусочкам я собрала и радости,
И счастье собирала по кусочкам.
С троп на большие дороги нужна сила,
Вел меня сердце, вело за руку.

Я нужна тебе сегодня, ты нужен мне,
Научи снова жить, снова, сердце!

Не отдала, не отдам сердце - чтоб топтали.

Не раните людскую душу - остается в сердце,
Сегодня счастье гостит у тебя, завтра - у меня.
И через скалы проходят обрывы,
Я уже не во вчерашнем - новый день ждет.

Я нужна тебе сегодня, ты нужен мне,
Научи снова жить, снова, сердце!

Не отдала, не отдам сердце - чтоб топтали.

По кусочкам собираю сердце, по кусочкам собираю.
```
