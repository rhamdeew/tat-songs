# Оригинал

### Земфира Садыйкова / Рәзидә Әхмәтова - Бәхет ачкычлары сөюдә!

```
Тик үзеңнән тора, үзеңнән
Көннәреңнең ничек үтүе ...
Яратырга кирәк кешеләрне
Яратырга кирәк үзеңне.

Гомереңнең ничек үтеүе
Тик үзеңнән тора, үзеңнән.
Оныт инде йөрәк әрнүен
Бәхет ачкычлары үзеңдә!

Тик үзеңнән тора, үзеңнән
Тормыш елгасының агышы ...
Күңелең саф, җаның шат икән
Белер йөрәк бәхет килерен.

Гомереңнең ничек үтеүе
Тик үзеңнән тора, үзеңнән.
Оныт инде йөрәк әрнүен
Бәхет ачкычлары үзеңдә!

Уйларыңнан килә яктылык
Яктылыктан тыуа яхшылык!
Үз-үзеңнән гаеп эзләмә -
Бәхет ачкычлары сөюдә ...

Гомереңнең ничек үтеүе
Тик үзеңнән тора, үзеңнән.
Оныт инде йөрәк әрнүен
Бәхет ачкычлары үзеңдә!
```
