# Оригинал

### Зәйни Шаһиморат / Луиза Батыр-Болгари - Кайт, иркәм Сәрвиназ!

```
Җырларың күңелгә моң сала, Сәрвиназ,
Сагышың йөрәккә ут сала.
Сагышың йөрәккә ут сала, Сәрвиназ,
Җырларың күңелгә моң сала.

Кайт, иркәм, мин көтәм, Сагындым, Сәрвиназ.
Син диеп, ут йотам, тилмерәм.
Син диеп, тилмерәм, ут йотам, Сәрвиназ,
Кайт, иркәм, мин көтәм, мин көтәм.

Җыр җырлап яшәргә, уйнарга, көләргә
Син лаек бит, минем бер генәм.
Бакчадагы кошлар, һавада йолдызлар -
Алар бар да сиңа гашыйклар.

Кайт, иркәм, мин көтәм, Сагындым, Сәрвиназ.
Син диеп, ут йотам, тилмерәм.
Син диеп, тилмерәм, ут йотам, Сәрвиназ,
Кайт, иркәм, мин көтәм, мин көтәм.
```
