# Оригинал

### Зөлфә Һадиева / Филарит Гасыймов - Татарстаным

```
Акчарлак канат җәя
Идел күкрәгендә.
Дуслар сәламе яңгырый
Зәңгәр күкләрендә.

Татарстан - туган ягым!
Көрәштә ирек алдың.
Данлы хезмәт җырына күмеп
Алкышлый сине халкым.

Җирләрең юмарт синең,
Күкрәп иген уңа.
Мәһабәт фонтан булып
Кара алтын ургыла.

Татарстан - туган ягым,
Тарихта ис мең - алтын!
Данлы хезмәт җырына күмеп
Алкышлый сине халкым.
```

# Перевод

### Зөлфә Һадиева / Филарит Гасыймов - Татарстан мой

```
Чайка расправляет крылья
Над грудью Идели-реки.
Привет друзей звучит эхом
В лазури небесной выси.

Татарстан - край родимый!
В борьбе ты свободу взял.
В песню славного труданНарод тебя воспевал.

Земли твои щедры,
Колос наливается в полях.
Могучим фонтаном
Бьет черное золото в недрах.

Татарстан - край родимый,
В истории имя - золото!
В песню славного труда
Народ тебя воспевал.
```
