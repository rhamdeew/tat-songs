# Оригинал

### Замирә Гәрәева / Айрат Нәбиев - Синең өчен (икенче вариант)

```
Йөрәкләрдән хисләр генә таша,
Ә күзләрдән назлар түгелә.
Синең өчен йолдыз кебек янам,
Кояш булам тормыш күгеңдә.

Синең өчен бу дөньяга килдем,
Cинен өчен көнем һәм төнем
Һәр сулышымны «син» дип кенә сулыйм,
Сиңа багышлыймын гомерем.

Ходай биргән олы мәхәббәте
Югалмыйк без, бәгърем, тормышта.
Синең өчен утка, суга керәм,
Аралармын сине сагыштан.

Синең өчен бу дөньяга килдем,
Cинен өчен көнем һәм төнем
Һәр сулышымны «син» дип кенә сулыйм,
Сиңа багышлыймын гомерем.

Синең өчен күңел борчуларым,
Синдә минем өмет, хыялым.
Синең белән бергә чакта гына
Яшәвемнең ямен тоямын.

Синең өчен бу дөньяга килдем,
Cинен өчен көнем һәм төнем
Һәр сулышымны «син» дип кенә сулыйм,
Сиңа багышлыймын гомерем.
```

# Перевод

### Замира Гараева / Айрат Набиев - Для тебя (второй вариант)

```
Из сердец лишь чувства переполняют,
А из глаз нежность льётся рекой.
Для тебя горю я, словно звезда,
Стану солнцем в небе жизни твоей.

Для тебя я на этот свет родился,
Для тебя мои день и ночь,
Каждым вздохом лишь «ты» произношу,
Тебе посвящаю жизнь свою.

Богом данную великую любовь
Не потеряем мы, родная, в жизни.
Для тебя в огонь и в воду войду,
От печали уберегу тебя.

Для тебя я на этот свет родился,
Для тебя мои день и ночь,
Каждым вздохом лишь «ты» произношу,
Тебе посвящаю жизнь свою.

Для тебя душевные тревоги,
В тебе моя надежда и мечта.
Только рядом с тобой
Я познаю радость бытия.

Для тебя я на этот свет родился,
Для тебя мои день и ночь,
Каждым вздохом лишь «ты» произношу,
Тебе посвящаю жизнь свою.
```
