# Оригинал

### Зульфат Зиннуров Зиннур - Ашыктырма

```
Югалсам да азга гына
Син узен килеп яныма
Упкэлэгэн буласын
Буласын..

Эле алай гына тугел
Хэзер индэ менэ син гел
Туй дип ашыктырасын
Ашыктырасын..

Юк димэле, мин ойлэнмим
Нигэ кияргэ камыт
Бер генэ килгэн яшьлекнен
Кадерен бел ди, халык

Менэ узен санап кара
Куп тэ тугел танышканга
Нигэ хаман туй дисен
Туй дисен..

Мин янында бит кон саен
Нилэр житми индэ тагын
Нигэ тузэ белмисен
Тузэ белмисен..

Юк димэле, мин ойлэнмим
Нигэ кияргэ камыт
Бер генэ килгэн яшьлекнен
Кадерен бел ди, халык

Эйтсэм гел ачуын килэ
Туй итэргэ инде эллэ
Олгермэбез дисенме
Дисенме..

Эгэр яратасын икэн
Ашыктырма мине иркэм
Тыя бел нэфесенне
Нэфесенне..

Юк димэле, мин ойлэнмим
Нигэ кияргэ камыт
Бер генэ килгэн яшьлекнен
Кадерен бел ди, халык
```

# Перевод

### Зульфат Зиннуров Зиннур - Не торопи

```
Хоть и немного потерялся я,
Ты сама пришла ко мне близко,
Обижаться будешь на меня,
Будешь же...

Не только так теперь уже,
Вот теперь ты всё время
О свадьбе торопишь меня,
Торопишь...

Нет же, я не женюсь пока,
Зачем мне надевать ярмо,
Одну лишь пришедшую молодость
Цени же, народ мой

Вот сама посчитай теперь,
Немного же мы знакомы,
Почему всё свадьбу твердишь,
Свадьбу твердишь...

Я рядом с тобой каждый день,
Чего же ещё не хватает,
Почему терпеть не можешь,
Терпеть не можешь...

Нет же, я не женюсь пока,
Зачем мне надевать ярмо,
Одну лишь пришедшую молодость
Цени же, народ мой

Если говорю - сердишься,
Свадьбу играть уже хочешь,
Неужели постарели мы,
Постарели...

Если любишь меня правда,
Не торопи меня, милая,
Сдерживай своё желание,
Желание...

Нет же, я не женюсь пока,
Зачем мне надевать ярмо,
Одну лишь пришедшую молодость
Цени же, народ мой
```
