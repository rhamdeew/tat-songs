# Оригинал

### Зәйтүнә Галиева / Раил Едиханов - Язны сагыну (икенче вариант)

```
Дим сулары шашып,
Ярларыннан ташып,
Шаулап ага язгы ташкында.
Күкрәгемдә ярсып,
Йөрәк тибә шашып,
Туган якларыма талпына.

Кайталмадым тагын,
Сагынсам да барын:
Дим буеның бөдрә талларын,
Бала чакның язын
Күрәлмәдем тагын

Авылымның язгы таңнарын.
Күпме гомер үткән,
Тагын язлар җиткән.
Дим өстендә бозлар киткәндер.
Су буенда таллар,
Моңсуланып ярда,
Өметләнеп мине көткәндер.
```

# Перевод

### Зәйтүнә Галиева / Раил Едиханов - Соскучаясь по весне (второй вариант)

```
Дим, взбесившись, мчится,
Через берега разлится,
Шумно вешний в разливе поток.
В груди буря бьётся,
Сердце вихрем рвётся,
К родным сторонам летит как мотылёк.

Не вернулся снова,
Хоть тоска готова:
По кудрявым ивам у Дима,
По весне из детства
Не увидеть снова

Мне рассветов весенних родного села.
Сколько лет промчалось,
Снова весна встречалась.
Лёд на Диме уж, наверно, сошёл.
Ивы у воды там,
На печальном берегу,
С надеждой меня, наверно, ждут.
```
