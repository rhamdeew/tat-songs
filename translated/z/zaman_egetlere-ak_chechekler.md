# Оригинал

### Zaman Egetlere - Ак чэчэклэр

```
Таллыкта алан,
Аланны япкан
Ап-ак чэчэклэр.
Без бергэ идек,
Калфаклар урдек
Шул ак чэчкэдэн.

Алсу йозенэ
Бигрэк килешэ
Чэчкэ калфагын!
Синнэн дэ сылу
Чэчэклэр булу
Момкинме тагын?!

Дорлэсен хислэр!
Тэмле, хуш ислэр
Жилдэ уйнасын!
Безнен кунеллэр
Сулган чэчэклэр
Кебек булмасын!
```

# Перевод

### Заман егетләре - Белые цветы

```
В ивняке цветут,
Поляну укрыв,
Белоснежные цветы.
Мы были вместе,
Венки плели
Из тех белых цветов.

Твоему светлому лицу
Так идёт
Цветочный венок!
Может ли быть
Ещё красивее
Цветов, чем ты?!

Пусть дрожат чувства!
Сладкие, нежные ароматы
Пусть играют на ветру!
Пусть наши сердца
Не будут словно
Увядшие цветы!
```
