# Оригинал

### Зөләйха Минһаҗева / Урал Рәшитов - Син (беренче вариант)

```
Син, син, син, син минем тормышым,
Аткан ал таңым син,
Син, син, син
Сулаган сулышым,
Яшәгән җаным син.

Сиңа мин инде
Язмышың, өлешең.
Кайларга китсәң дә,
Мин синең - йөрәк тибешең.

Син, син, син
Бер юлдан бергәләп
Атларга тиеш без.
Син, син, син
Адымнар ялгышмыйк.
Абынмыйк, юл тигез.

Сиңа мин инде
Язмышың, өлешең.
Кайларга китсәң дә,
Мин синең - йөрәк тибешең.

Син, син, син,
Яши алмабыз бит
Мин синсез, син минсез.
Син, син, син
Сузыйк тормыш җебен
Төерсез, өзексез.

Сиңа мин инде
Язмышың, өлешең.
Кайларга китсәң дә,
Мин синең - йөрәк тибешең.
```

# Перевод

### Зөләйха Минһаҗева / Урал Рәшитов - Ты (первый вариант)

```
Ты, ты, ты - моя жизнь,
Ты мой рассвет алый,
Ты, ты, ты -
Мой вдох и выдох,
Ты моя живая душа.

Для тебя я уже -
Твоя судьба, твоя доля.
Куда бы ты ни ушел,
Я твое биенье сердца.

Ты, ты, ты -
По одной дороге вместе
Нам идти суждено.
Ты, ты, ты -
Пусть не собьются шаги,
Пусть ровной будет дорога.

Для тебя я уже -
Твоя судьба, твоя доля.
Куда бы ты ни ушел,
Я твое биенье сердца.

Ты, ты, ты -
Не сможем жить,
Я без тебя, ты без меня.
Ты, ты, ты -
Протянем нить жизни
Без узлов, без разрывов.

Для тебя я уже -
Твоя судьба, твоя доля.
Куда бы ты ни ушел,
Я твое биенье сердца.
```
