# Оригинал

### Зөһрә Ганиева / Мөфтәдин Гыйләҗев - Ак күбәләк (икенче вариант)

```
Күп йөредем гөлләр арасында,
Чәчкәләрне үзең өзмәдең.
Ак күбәләк булып тугайларда
Кемне шулай сагынып эзләдең.

Сине генә көттем, сагынып көттем.
Каршыларга таңны килмәдең.
Ник соң үзең минем тормышыма
Ак күбәләк булып кермәдең?

Йөрәккәем ялкын белән тулы,
Хыялларым синдә чагында.
Искә ал син мине, иркәләнеп
Ак күбәләк очса яныңда.
```

# Перевод

### Зөһрә Ганиева / Мөфтәдин Гыйләҗев - Белая бабочка (второй вариант)

```
Долго шла я меж цветов весенних,
Ты бутоны сам не срывал.
Белой бабочкой по лугам ранним
Кого так тоскуя искал?

Я тебя ждала, тосковала,
Ты встречать рассвет не пришел.
Почему же в жизнь мою ты, милый,
Белой бабочкой не вошел?

Сердце пламя бережно хранит,
Все мечты мои в тебе одной.
Вспомни обо мне, когда ласково
Белая бабочка рядом со мной.
```
