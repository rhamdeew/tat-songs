# Оригинал

### Зарема Әюпова / Ринат Гыйззәтуллин - Әнкәй гөле (икенче вариант)

```
Иркәләп һәм сөеп кенә
Үстерә әнкәй гөлләр.
Әнкәй догасыннан тора
Алдагы безнең көннәр.

Туган йорттан киткәч ерак,
Булсын дип истәлеге,
Бер сабагын алдым өзеп,
Әнкәй синең гөлеңнең.

Араларны якынайтып
Гөлең гел чәчәк ата.
Күзләремә килә яшьләр
Уйласам синең хакта.

Туган йорттан киткәч ерак,
Булсын дип истәлеги,
Бер сабагын алдым өзеп,
Әнкәй синең гөлеңнең.

Әнкәй гөлләрдәй саф,
Баладай киң күңелең.
Беләсең син балакаем
Бик якын да түгелем.

Әнкәемнең үсә гөле
Ямь биреп ерак җирдә.
Сагынганда көч аламын,
Карыйм да синең гөлгә.
```

# Перевод

### Зарема Әюпова / Ринат Гиззатуллин - Материнский цветок (второй вариант)

```
С любовью и лаской нежной
Растит мама свои цветы.
Благословеньем материнским
Наполнены наши дни.

Уезжая из родного дома,
Чтоб память со мной была,
Я стебелёк один сорвала -
Твой цветок, родная мама.

Сближая расстоянья,
Твой цветок всё так же цветёт.
Слёзы застилают глаза,
Когда о тебе я вспоминаю.

Уезжая из родного дома,
Чтоб память со мной была,
Я стебелёк один сорвала -
Твой цветок, родная мама.

Мамочка, как цветы чиста,
Как дитя широка душой.
Ты же знаешь, доченька моя,
Я совсем не близко от тебя.

Маминого цветка красота
Дарит радость в дальней стороне.
Когда скучаю, силы черпаю,
Глядя на твой цветок родной.
```
