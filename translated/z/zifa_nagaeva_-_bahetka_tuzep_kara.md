# Оригинал

### Зифа Нагаева - Бәхеткә түзеп кара

```
Агы бар, каралары,
Язмышның яралары.
Ах иткән  төннәрнең, дуслар,
Бар икән дәвалары.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.

Җиле бар, томаннары,
Кары бар, бураннары.
Икегә бүлеп җиңәbez
Язмышның сынауларын.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.

Гомер таулары бик текә,
Үрләргә менәсе бар.
Үргә менеп безгә насыйп
Бәхетне күрәсе бар.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.
```

# Перевод

### Зифа Нагаева - Попробуй счастье найти

```
Есть боль, есть раны,
Раны судьбы на нас.
Есть и средства от боли,
Ох, эти ночи, друзья.

Идём мы вместе вдвоём,
Рядом со мной равный идет.
Кто выдержит тоску-печаль,
Попробуй счастье найти.

Есть ветер, есть туманы,
Есть снег, есть метели.
На двоих делим мы,
Испытания судьбы.

Идём мы вместе вдвоём,
Рядом со мной равный идет.
Кто выдержит тоску-печаль,
Попробуй счастье найти.

Горы жизни очень круты,
Хочется на них взобраться.
Взобраться вверх и увидеть нам
Счастье, что нам суждено.

Идём мы вместе вдвоём,
Рядом со мной равный идет.
Кто выдержит тоску-печаль,
Попробуй счастье найти.
```
