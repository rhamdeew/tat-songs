# Оригинал

### Зифа Нагаева - Йөрәкләрне тамгаладык

```
Су буйлары каз үләне,
Анда казлар йөзәме?
Анда казлар йөзәме?

Суларга, ярлар тар иде,
Безгә дә, ярлар бар иде,
Иң үзәкне өзгәне.

Күл - казларсыз, яр - дулкынсыз,
Тынып тормасын инде,
Тынып тормасын инде.

Яшьлектән үк, табыласы,
Язмышыма языласы
Ярым булгансың инде.

Парлап йөзгән ак казларның
Юлын бормагыз инде.
Юлын бормагыз инде.

Ярсыз яшәү бигърәк читен,
Ялгыз гомер бит ул китек,
Парсыз булмагыз инде.

Суда талларсыз инде лә,
Казлар тамгасыз инде,
Казлар тамгасыз инде.

Йөрәкләрне тамгаладык,
Адашмас юллар сайладык,
Бергә каласы инде.
```

# Перевод

### Зифа Нагаева - Мы сердца отметили

```
По воде - гусиная трава,
Там ли гуси плывут?
Там ли гуси плывут?

Берега были узки для вод,
И для нас берега нашлись,
Но боль в самом сердце - рубец.

Озеро - без гусей, берег - без волн,
Пусть не смолкнет оно,
Пусть не смолкнет оно.

С юных лет - чтоб найтись,
В мою судьбу вписаться
Половинкой ты стала уже.

Пути белых гусей, что плывут вдвоём,
Не сворачивайте их,
Не сворачивайте их.

Без берега жить - так тяжко,
Одинокая жизнь - с изъяном,
Не будьте без пары, прошу.

В воде - без ив, уж видно,
Гуси - без меток, видно,
Гуси - без меток, видно.

Мы сердца отметили,
Выбрали верные тропы,
Нам вместе остаться теперь.
```
