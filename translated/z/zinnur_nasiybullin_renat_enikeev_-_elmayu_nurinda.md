# Оригинал

### Зиннур Насыйбуллин / Ренат Еникеев - Елмаю нурында

```
Бу җырым елмаю,
Елмаю турында.
Күңелләр җылына
Елмаю нурында.
Елмаю нурында
Күңелләр җылына.

Балкысын күңелләр
Елмаю нурыңда.
Күңелләр нурында
Дөньялар җылына.
Дөньялар җылына
Күңелләр нурында.
```

# Перевод

### Зиннур Насыйбуллин / Ренат Еникеев - В сиянии улыбки

```
Мой этот напев - про улыбку,
Про свет её тёплый.
Согреты сердца
Сияньем улыбки.
В сиянии улыбки
Согреты сердца.

Пусть сердца сияют
В луче твоей улыбки.
В сиянии сердец
Согреются миры.
Согреются миры
В сиянии сердец.
```
