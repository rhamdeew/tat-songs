# Оригинал

### Зифа Нагаева - Бөртекләп җыелган бәхет

```
Син яшь чакта, мин яшь чакта, ул чакта
Ярый әле калмаганбыз без ятка.
Бары да булды язмышта, яшибез житеш тормышта,
Син минем хакка түзгәнгә, мин синен хакка.

Күлләвектән күлләр җыела,
Тамчылардан диңгез коела.
Бөртекләп җыелган дөньям,
Бөртекләп өелгән шатлык,
Бөртекләп җыелган бәхеткәем син!

Яшьлекләр бар, бар үткәннәр аралар да,
Үзәкләрне өзеп тора балалар да.
Саклап калдык сөюне дә, тапламадык,
Юл куймадык күңелләрне караларга.

Күлләвектән күлләр җыела,
Тамчылардан диңгез коела.
Бөртекләп җыелган дөньям,
Бөртекләп өелгән шатлык,
Бөртекләп җыелган бәхеткәем син!

Миңа бүген бик кадерле гомеремнең
Һәрбер көне, һәрбер төне, һәрбер таңы.
Кайтып булса үткәннәргә үтәр идем мин кабаттан
Шул гомерне бары синең белән тагын.

Күлләвектән күлләр җыела,
Тамчылардан диңгез коела.
Бөртекләп җыелган дөньям,
Бөртекләп өелгән шатлык,
Бөртекләп җыелган бәхеткәем син!
```

# Перевод

### Зифа Нагаева - Счастье, собранное по крупицам

```
Когда ты был юн, когда я была юна, тогда
Хорошо, что мы не стали чужими.
Всё было в судьбе, живем в достатке,
Ты терпел ради меня, я - ради тебя.

Из луж рождаются озёра,
Из капель разливаются моря.
Мой мир по крупицам собран,
Радость по крупицам сложена,
Счастье, собранное по крупицам, - это ты!

Есть молодость, есть расстояния прошлых лет,
И дети, что сердце порой рвут.
Мы сберегли любовь, не растоптали,
Не дали душе потемнеть.

Из луж рождаются озёра,
Из капель разливаются моря.
Мой мир по крупицам собран,
Радость по крупицам сложена,
Счастье, собранное по крупицам, - это ты!

Мне нынче очень дороги в моей жизни
Каждый день, каждая ночь, каждый рассвет.
Если б можно вернуться к прошлому, я бы снова прошла
Эту жизнь, но лишь с тобой опять.

Из луж рождаются озёра,
Из капель разливаются моря.
Мой мир по крупицам собран,
Радость по крупицам сложена,
Счастье, собранное по крупицам, - это ты!
```
