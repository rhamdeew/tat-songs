# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Тавыш бирче

```
Синең хакта яшьлек хәтирәсе
Гомер җибен сүтә киредән.
Еллар аша сиңа эндәшәм мин,
Тавыш бирче, тавыш бирче,
Тавыш бирче яшьлек иленнән.

Сөйгән җаннар генә куш канатлы,
Алышларда көчле җил белән.
Бәхет өчен пар канатым кирәк,
Тавыш бирче, тавыш бирче,
Тавыш бирче бәхет иленнән.

Сөюсез һәм синсез елларымны
Алып ташлар идең иңемнән.
Мәхәббәтем, бик сагындым сине,
Тавыш бирче, тавыш бирче,
Тавыш бирче сөю иленнән.
```

# Перевод

### Зухра Котлыгильдина / Салават Низаметдинов - Голос подай

```
Воспоминанья о юности в твою честь
Нить жизни прячет, прячет снова.
Сквозь годы я ищу тебя,
Голос подай, голос подай,
Голос подай из страны юности.

Только любящие души с двумя крылами,
В борьбе с сильным ветром вместе.
Для счастья нужны мне пара крыльев,
Голос подай, голос подай,
Голос подай из страны счастья.

Годы без любви и без тебя
Ты бы смела с меня, как с плеч.
Любовь моя, так скучал я по тебе,
Голос подай, голос подай,
Голос подай из страны любви.
```
