# Оригинал

### Зухра Шарифуллина - Эх тала тала

```
Сандугачлар иялэшкэн
Су эчэргэ милэштэн.
Кайгыларым бетэр иде
Сине куреп сойлэшсэм.

Эх, тала, тала, тала,
Тала ике белэгем.
Нигэ тала, нигэ яна
Икэн минем йорэгем?

Агыйлделгэ басма салдым
Каенын сайлап кына.
Яннарына барыр идем
Кош булып сайрап кына.

Эх, тала, тала, тала,
Тала ике белэгем.
Нигэ тала, нигэ яна
Икэн минем йорэгем?

Агыйделкэйне кичкэндэ
Жырлыйм ишкэк ишкэндэ.
Озелэ узэк, яна йорэк
Жаныем искэ тошкэндэ.

Эх, тала, тала, тала,
Тала ике белэгем.
Нигэ тала, нигэ яна
Икэн минем йорэгем?

Агыйдел суы тирбэлэ
Уртасында сал булса.
Суга салсан, калкып чыгар,
Безгэ насыйп яр булса

Эх, тала, тала, тала,
Тала ике белэгем.
Нигэ тала, нигэ яна
Икэн минем йорэгем?
```

# Перевод

### Зухра Шарифуллина - Ах степь, степь

```
Соловьи слетелись стаей
Воду пить к ручью лесному.
Горе моё утихло бы,
Если б с тобой поговорить.

Ах, степь, степь, степь,
Степь по обе стороны.
Почему степь, почему горит
Моё сердце почему?

На Агидель мостик кинул
Из берёзы выбирая.
К милой хотел бы я пойти
Птицей песни распевая.

Ах, степь, степь, степь,
Степь по обе стороны.
Почему степь, почему горит
Моё сердце почему?

Переправляясь через Агидель
Пою, вёсла погружая.
Рвётся струна, пылает сердце,
Когда душа о прошлом вспоминает.

Ах, степь, степь, степь,
Степь по обе стороны.
Почему степь, почему горит
Моё сердце почему?

Вода Агидели колышет
Плот посредине реки.
В воду бросишь — всплывёт наверх,
Если суждена нам любовь.

Ах, степь, степь, степь,
Степь по обе стороны.
Почему степь, почему горит
Моё сердце почему?
```
