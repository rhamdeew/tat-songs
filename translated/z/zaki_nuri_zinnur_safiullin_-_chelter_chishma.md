# Оригинал

### Зәки Нури / Зиннур Сафиуллин - Челтер чишмә

```
Җанга шифалы сулары,
Бормалы су буйлары.
Сөйләшеп сүзем бетмәде,
Күреп күзем туймады.

Челтер-челтер чишмә ага,
Кызлар җырлап су ала.
Су алганда суырып үпсәм,
Үпкән эзләре кала.

Җырлый-җырлый су алганда,
Чиләкләре тиз тула.
Чишмә суы эчкән кызның
Күңеле чибәр була.

Челтер-челтер чишмә ага,
Кызлар җырлап су ала.
Су алганда суырып үпсәм,
Үпкән эзләре кала.

Чишмә буена кил але,
Чишмә белән серләшик.
Бабайлардан калган гадәт,
Гомерлеккә берләшик.

Челтер-челтер чишмә ага,
Кызлар җырлап су ала.
Су алганда суырып үпсәм,
Үпкән эзләре кала.
```

# Перевод

### Зәки Нури / Зиннур Сафиуллин - Звонкий родник

```
Воды целебные для души,
Берега у извилистой воды.
Разговор - словам конца не было,
И глазами насытиться не смог.

Журчит-струится родник,
Девушки, напевая, воду берут.
Если, черпая воду, украдкой поцелую -
Следы поцелуя остаются.

Песню поют, воду набирая -
Ведра их быстро наполняются.
Девушка, что пьет родниковую воду,
Сердцем и лицом красивой бывает.

Журчит-струится родник,
Девушки, напевая, воду берут.
Если, черпая воду, украдкой поцелую -
Следы поцелуя остаются.

К роднику приди, дорогая,
С родником поделимся тайной.
Обычай от дедов остался -
Соединимся навек.

Журчит-струится родник,
Девушки, напевая, воду берут.
Если, черпая воду, украдкой поцелую -
Следы поцелуя остаются.
```
