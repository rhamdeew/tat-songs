# Оригинал

### Земфира Садыйкова / Артур Туктаголов - Дөнья матур (беренче вариант)

```
Кешеләргә нәрсә җирдә җитми,
Дәрман юкмы гѳрләп яшәргә?
Мал артыннан куып гомер үтә...
Ә яшәргә? Кайчан яшәргә?

Дөнья матур яши белгәннәргә
Рәхәтләнеп кенә, яшә дә яшә!
Шатланып - кѳлеп кал, гѳрләп гел яшә,
Дуслар белән бергәләп яшә!

Нигә шулай җаның ѳзгәләнә
Кунак булыр гына дөньяда?
Мәңге яшәр кебек була кеше
Ике кѳнлек шушы дөньяда...

Дөнья матур яши белгәннәргә
Рәхәтләнеп кенә (гелән), яшә дә яшә!
Шатланып - кѳлеп кал, гѳрләп гел яшә,
Дуслар белән бергәләп яшә!
```

# Перевод

### Земфира Садыйкова / Артур Туктаголов - Мир прекрасен (первый вариант)

```
Чего людям на земле не хватает,
Нет ли сил, чтобы жить весело?
Жизнь проходит в погоне за богатством...
А жить? Когда же жить?

Мир прекрасен для тех, кто умеет жить,
Наслаждайся, живи, только живи!
Радуйся, смейся, живи весело,
Живи вместе с друзьями!

Почему так меняется душа твоя,
Ведь ты лишь гость в этом мире?
Человек живёт, как будто вечно,
В этом двухдневном мире...

Мир прекрасен для тех, кто умеет жить,
Наслаждайся, живи, только живи!
Радуйся, смейся, живи весело,
Живи вместе с друзьями!
```
