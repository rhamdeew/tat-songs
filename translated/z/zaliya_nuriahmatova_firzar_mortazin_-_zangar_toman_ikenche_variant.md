# Оригинал

### Залия Нуриәхмәтова / Фирзәр Мортазин - Зәңгәр томан (икенче вариант)

```
Күңел һаман көтә әле...
Нидер үзгәрер сыман...
Хыялларыма юл ачып,
Таралыр кебек томан.
Зәңгәр томан, капламачы
Барасы юлларымны...
Киләчәккә өмет итеп,
Җиктем хыялларымны.
Язмыш төрле, дөнья серле,
Һәркөн көтә яңалык.
Эх, диеп яшәмик җирдә,
Шатландырсын чынбарлык.
Күңел һаман көтә әле..
Үзгәрер дөньялар да..
Гөлбакчалар чәчәк атар,
Мин эз салган юлларда.
```

# Перевод

### Залия Нуриахматова / Фирзәр Мортазин - Зыгурушка, туман (второй вариант)

```
Сердце всё ждёт...
Что-то изменится, наверное...
Открывая путь мечтам,
Рассеивается, как туман.
Сизый туман, окутывающий
Все дороги, что впереди...
Надеясь на будущее,
Я оставляю свои мечты.
Судьба разнообразна, мир загадочен,
Каждый день ждёт новизны.
Ах, не зря мы живём на земле,
Пусть счастье наступит.
Сердце всё ждёт...
Изменятся и миры...
Цветы расцветут,
На путях, где я оставил след.
```
