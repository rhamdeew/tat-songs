# Оригинал

### Зинаида Захарова / Людмила Аланлы - Әнигә (алтынчы вариант)

```
Табыныңнан ризык өзелмәсен,
Тәмле булсын эчкән чәйләрең.
Тыныч булсын, әни, күңелләрең,
Җылы гына булсын җәйләрең.

Үтәсе юлларың булсын озын,
Алтын гына булсын гына гомер көзең.
"Гел бәхетле яшә, әни!", - диеп,
Балаларың теләк тели бүген.

Авырлыклар сине урап узсын,
Яшькә чыланмасын күзләрең.
Беркайчан да онытмабыз, әни,
Безгә әйткән назлы сүзләрең.

Үтәсе юлларың булсын озын,
Алтын гына булсын гына гомер көзең.
"Гел бәхетле яшә, әни!", - диеп,
Балаларың теләк тели бүген.
```

# Перевод

### Зинаида Захарова / Людмила Аланлы - Маме (шестой вариант)

```
Пусть хлеб твой никогда не иссякнет,
Вкусным пусть будет выпитый чай.
Пусть будет спокойно на душе, мама,
Тёплыми пусть будут лишь летние дни.

Пусть долгими будут дороги твои,
Золотой пусть будет лишь осень жизни твоей.
«Живи счастливо всегда, мама!» - так
Дети твои желают сегодня.

Пусть тяжесть стороной пройдёт,
Слезами пусть не наполнятся глаза.
Никогда не забудем, мама,
Твои ласковые слова, сказанные нам.

Пусть долгими будут дороги твои,
Золотой пусть будет лишь осень жизни твоей.
«Живи счастливо всегда, мама!» - так
Дети твои желают сегодня.
```
