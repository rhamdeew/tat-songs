# Оригинал

### Зиннәт Вәли / Сара Садыкова - Иртәнге нур

```
Туган илкәем, сөйгән җиркәем
Иртәнге саф нурга күмелә.
Урман буйлары, басу юллары
Бигрәк инде матур күренә.

Уяна сабый, елмаеп карый
Иртәнге саф кояш нурына.
Матур уйларым, татлы хыялым
Кушылсын шат хезмәт җырына.

Иртәнге нурлар, иң якты уйлар,
Кая барсам, минем күңелемдә,
Һәрвакыт шулай, сайрасын тургай
Туган илнең тыныч күгендә.
```

# Перевод

### Зиннат Вэли / Сара Садыкова - Утренний луч

```
Родной край, любимая земля
В утреннем чистом свете купается.
Лесные опушки, полевые тропы
Особенно прекрасно выглядят.

Просыпается младенец, улыбаясь смотрит
На утренний чистый солнечный луч.
Прекрасные мысли, сладкие мечты
Пусть сольются в радостную трудовую песню.

Утренние лучи, самые светлые мысли,
Куда бы я ни шёл, всегда в моей душе,
Пусть всегда так поёт соловей
Под мирным небом родной страны.
```
