# Оригинал

### Зәйнәп Биишева / Нур Даутов - Нигә? (беренче вариант)

```
Нигә күрдем, нигә яраттым мин,
Нигә күңелемне яуладың?
Нигә синең моңсу күзләреңнән
Күзләремне ала алмадым?

Тик кайда соң!..
Сөйгән йөрәкләрне
Аерырлык киртә бармы ни?!
Сөю хисләрем дулкын-дулкын булып
Ярсып килә, иртә язмы ни...
```

# Перевод

### Зәйнәп Биишева / Нур Даутов - Почему? (первый вариант)

```
Зачем увидела, зачем полюбила я,
Зачем ты сердце мое покорил?
Почему от глаз твоих печальных
Я глаз своих оторвать не смогла?

Но где же он!..
Есть ли преграда,
Что разлучит влюбленные сердца?!
Чувства любви волной за волною
Мчатся ко мне, будто ранняя весна...
```
