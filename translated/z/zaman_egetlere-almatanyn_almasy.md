# Оригинал

### Zaman Egetlere - Алматанын алмасы

```
Алматанын алмасы
Тэмле хуш ислэр сибэ.
Сине генэ котэм,
Сине генэ сагынам,
Сине генэ курэсем килэ.

Алматанын кар суы
Биек таулардан килэ.
Сине генэ котэм,
Сине генэ сагынам,
Сине генэ курэсем килэ.

Алматанын кар суы
Минем йорэк ярсуы.
Сине генэ котэм,
Сине генэ сагынам,
Сине генэ килэ курэсем.

Алматанын таулары
Озелэдер каеш баулары.
Сине генэ котэм,
Сине генэ сагынам,
Сине генэ килэ курэсем.
```

# Перевод

### Заман егетләре - Алматинские яблоки

```
Алматинские яблоки
Источают сладкие ароматы.
Только тебя жду,
Только по тебе скучаю,
Только тебя хочу увидеть.

Алматинская талая вода
С высоких гор течёт.
Только тебя жду,
Только по тебе скучаю,
Только тебя хочу увидеть.

Алматинская талая вода —
Моего сердца целебная влага.
Только тебя жду,
Только по тебе скучаю,
Только тебя хочу увидеть.

Алматинские горы
Разрывают берёзовые узы.
Только тебя жду,
Только по тебе скучаю,
Только тебя хочу увидеть.
```
