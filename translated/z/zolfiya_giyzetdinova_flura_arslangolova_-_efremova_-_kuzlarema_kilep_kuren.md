# Оригинал

### Зөлфия Гыйзетдинова / Флүрә Арсланголова - Ефремова - Күзләремә килеп күрен

```
Синсез үткән һәрбер көнем
Өзгеләнгән күңел гөлем.
Соңлый күрмә. гомерләр кыска,
Күзләремә килеп күрен.

Җанымның юк хис бер тере
Саф хисләрем әле тере.
Сүнмәс боры яшәү нурын
Күзләремә килеп күрен.

Сыер сүзең, сыер серең,
Синсез буш бит йөрәк түрем.
Өметләрем өзелгәнче
Күзләремә килеп күрен.
```

# Перевод

### Зульфия Гизетдинова / Флюра Арсланголова - Ефремова - Являясь пред моими глазами

```
Без тебя протекает каждый мой день
Изменилась душевная радость моя.
Не откладывай, жизни коротки,
Являясь пред моими глазами.

Нет в душе моей ни одного живого чувства
Чистые чувства мои всё ещё живы.
Негасимый свет жизни твоей
Являясь пред моими глазами.

Ласковое слово твоё, тайная твоё тайна,
Без тебя пусто сердце моё.
Пока не прервутся надежды мои
Являясь пред моими глазами.
```
