# Оригинал

### Зарема Әхмәтҗанова / Мөдәрис Газетдинов - Язмагандыр инде

```
Синең белән очрашулар сорап
Язмышка дип күпме ялварма.
Сагышымны синең сагышларга
Язмагандыр инде ялгарга.

Күпме табынсам да могҗизага,
Тылсымнарга, сихри көчләргә,
Шатлыгымны синең шатлыкларга
Язмагандыр инде өстәргә.

Язмагандыр инде, язмагандыр,
Гаебе юк һич беркемнәрнең.
Йөрәгемә басам ялкыннарын
Синсез генә үткән көннәрнең.
Синсез генә үткән көннәрнең...
```

# Перевод

### Зарема Ахматҗанова / Мөдәрис Газетдинов - Не суждено

```
Сколь ни проси у судьбы свиданий,
Сколько ни моли - не умоляй.
Моей тоске к твоим печалям
Не суждено, увы, пристать.

Сколько б ни верил я в чудеса,
В чары, в силы колдовские,
Моей радости к твоим радостям
Не суждено уже прибавить.

Не суждено, не суждено -
Вины нет ни на ком из нас.
Я к сердцу прижимаю пламя
Дней, что без тебя прошли.
Дней, что без тебя прошли...
```
