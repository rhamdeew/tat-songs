# Оригинал

### Зөлфәт / Татар халык көе - Чатырлар

```
Далаларда чатыр корыр идем,
Яннарында учак ягар идем.
Далаларда чатыр корды ятлар.
Чатыр яннарында кешни атлар.
Күн иярләр түзми шәп атларга.
Йөрәк ничек түзсен еракларда?!
```

# Перевод

### Зөлфәт / Татарская народная песня - Чатырлар

```
На лугах бы сухим как чатыр,
Вдоль них стремительно скакал.
На лугах чатыр уже засох.
Вдоль чатыра скачут кони.
Небеса не выносят быстрых коней.
Как вынести сердцу расстояния?!
```
