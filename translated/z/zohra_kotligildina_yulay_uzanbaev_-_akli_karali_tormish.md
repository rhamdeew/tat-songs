# Оригинал

### Зөһрә Котлыгилдина / Юлай Үзәнбаев - Аклы - каралы тормыш

```
Бер көн - табыш,
Бер көн - сагыш,
Аклы-каралы тормыш.
Табышларга - сөендереп,
Сагышларга - көендереп
Эй, сыный безне язмыш.

Бер көн - кояш,
Бер көн - буран,
Әссе-суык көннәре
Кояш нурлары көйдереп,
Бураннары куш иттереп,
Сыный тәкъдир җилләре.

Бер бөлдерә,
Бер көлдерә,
Өлешкә төшкән гомер.
Хәсрәтләргә-югалтуга,
Шатлкларга-елатуга
Сабырлар гына түзә.
```

# Перевод

### Зухра Котлыгильдина / Юлай Үзәнбаев - Белая и чёрная жизнь

```
Один день - радость,
Один день - печаль,
Жизнь в чёрно-белых тонах.
Радостями - развеселяя,
Печалями - огорчая
Эх, испытует нас судьба.

Один день - солнце,
Один день - вьюга,
Жаркие и холодные дни
Солнечными лучами жаря,
Буранами сбивая с толку,
Испытует ветра судьбы.

То заставит плакать,
То рассмешит,
Жизнь, вечно в делах.
Горем-потерей,
Радостью-слезами
Только терпение вынесет.
```
