# Оригинал

### Зифа Нагаева - Буран (беренче вариант)

```
Минеке бу буран, минеке...
Еласын-дуласын, мин аны
Ходайдан сорадым, түзәрлек
Хәлләре калмагач җанымның...

Минеке бу буран, минеке!
Кермә, син, чыгалмас башкаем,
Кермә, син, сөюем кабынган -
Елларны буранга ташладым.

Юлларга чәчләрем көярлек,
Чыгам да, ялан баш йөгерәм...
Минеке бары тик бу сөю,
Минеке, минеке бу буран!

Минеке бу буран, минеке!
Кермә, син, чыгалмас башкаем,
Кермә, син, сөюем кабынган -
Елларны буранга ташладым.

Алып китәр әле буран минем сагышларымны,
Сиңа илтеп кушар төсле минем язмышларымны...
```

# Перевод

### Зифа Нагаева - Буран (первый вариант)

```
Мой этот буран, только мой...
Пусть плачет-ревет - я его
У Бога просила, чтоб вынести,
Когда уж у сердца нет сил...

Мой этот буран, только мой!
Не входи - обратно не выйдешь, родной,
Не входи: любовь моя вспыхнула -
Годы вьюге я бросила в прах.

Чтобы волосам на дорогах гореть,
Выхожу и босой бегу...
Мое - лишь одно это чувство,
Мое, мое - этот буран!

Мой этот буран, только мой!
Не входи - обратно не выйдешь, родной,
Не входи: любовь моя вспыхнула -
Годы вьюге я бросила в прах.

Унесет еще буран мои тоски,
Словно к тебе пришьет мои судьбы...
```
