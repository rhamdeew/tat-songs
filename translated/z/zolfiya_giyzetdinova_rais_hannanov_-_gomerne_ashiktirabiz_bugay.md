# Оригинал

### Зөлфия Гыйзетдинова / Рәис Ханнанов - Гомерне ашыктырабыз, бугай

```
Гомерне ашыктырабыз, бугай,
Атларны куабыз шикелле.
Талларда сандугач, инде бер сайрагач,
Әйләнеп кайтуы икеле.

Кырык эшең кырык якка ташлап,
Башла баштан узган гомерне.
Талларда сандугач, инде бер сайрагач,
Әйләнеп кайтуы икеле.

Гомерне ашыктырабыз, бугай,
Кәккүктән еллар санаттырып.
Яшьлегемә кайтып, тагын бер яратып,
Әйләнеп кайт таңнар аттырып.

Кырык эшең кырык якка ташлап,
Башла баштан узган гомерне.
Талларда сандугач, инде бер сайрагач,
Әйләнеп кайтуы икеле.
```

# Перевод

### Зөлфия Гыйзетдинова / Рәис Ханнанов - Похоже, мы торопим жизнь

```
Похоже, жизнь мы торопим, увы,
Как будто гоним коней в разгон.
На ивах соловей - раз споёт,
Уже не вернётся, сомнён.

Оставь сорок дел по сорока путям,
Начни прожитую жизнь сперва.
На ивах соловей - раз споёт,
Уже не вернётся, молва.

Похоже, жизнь мы торопим, увы,
Кукушка нам годы считает.
Вернусь бы к юности, вновь бы любить,
Чтоб рассветы опять возвращать.

Оставь сорок дел по сорока путям,
Начни прожитую жизнь сперва.
На ивах соловей - раз споёт,
Уже не вернётся, молва.
```
