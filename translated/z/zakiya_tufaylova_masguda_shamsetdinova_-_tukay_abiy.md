# Оригинал

### Зәкия Туфайлова / Мәсгудә Шәмсетдинова - Тукай абый

```
Әле укый белмәсәк тә,
"Туган тел"не җырлыйбыз.
Бу - Тукай абый бүләге,
Иң гүзәлбер җыр, дибез.

Әкиятләр сөйләп, тыңлап,
Телләребез ачыла.
"Су анасы", "Шүрәле"ләр -
Безнең дуслар барсы да.
```

# Перевод

### Закия Туфайлова / Мэсгудэ Шамсетдинова - Дядя Тукай

```
Хоть ещё читать не умеем,
"Родной язык" поём мы.
Это - дяди Тукая подарок,
Самая прекрасная песня, говорим мы.

Сказки рассказывая, слушая,
Открываются наши языки.
"Водяная", "Лешие" -
Все они наши друзья, в самом деле.
```
