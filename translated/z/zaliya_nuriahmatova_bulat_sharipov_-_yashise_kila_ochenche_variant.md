# Оригинал

### Залия Нуриәхмәтова / Булат Шәрипов - Яшисе килә (өченче вариант)

```
Яшәгән саен яшисе килә,
Барсын яңадан башлыйсы килә..
Бөтен гөнаһлар югалсын иде,
Юллар турыдан тик барсын иде..
Матур теләкләр тормышка ашсын,
Насыйп ризыклар йөрәккә ятсын.
Балалар гөрләп яшәсен җирдә..
Оныклар шау-гөр килсеннәр өйдә.
Яшәгән саен яшисе килә..
Алда ни барын беләсе килә.
Авырлыкларны җиңәсе килә,
Батыр йөрәкле буласы килә.
Дөньяда һәрчак тынычлык булсын.
Бакчаларыбыз чәчәккә тулсын.
Кояш елмаеп йокларга ятсын..
Җирдә көн саен алсу таң атсын.
```

# Перевод

### Залия Нуриахматова / Булат Шарипов - Жизнь приходит (третий вариант)

```
С каждым днём всё ближе жизнь,
Хочется начать всё сначала...
Пусть исчезнут все грехи,
Пусть дороги будут прямыми...
Пусть сбываются добрые мечты,
Пусть в сердце будет благословение.
Пусть дети счастливо живут на земле..
Пусть дома будет радость и веселье.
С каждым днём всё ближе жизнь..
Хочется всё знать наперёд.
Хочется побеждать трудности,
Хочется быть храбрым и отважным.
Пусть всегда будет мир на земле.
Пусть сады наши будут полны цветов.
Пусть солнце улыбаясь засыпает..
Пусть на земле каждый день встаёт ясное утро.
```
