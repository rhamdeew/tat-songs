# Оригинал

### Заман егетләре - Зэнгэр тоймэ

```
Иреннэрен кызыл чия,
Кузлэрен зэнгэр тоймэ.
Гомерлеккэ сойдем сине,
Ташлап китэр бу, димэ.

Зэнгэр тоймэм, дип иркэлэп,
Назлап кына дэшэрмен.
Сундермэ мэхэббэтенне,
Мин коч алып яшэрмен.

Гел сина гына багышлыйм
Жырларымнын монлысын.
Котэм синнэн, зэнгэр тоймэм,
Мэхэббэтнен олысын!..
```

# Перевод

### Заман егетләре - Голубые пуговки

```
Губки — красная смородина,
Глазки — голубые пуговки.
На всю жизнь полюблю тебя,
Не говори, что бросишь ты.

Моя голубоглазка, нежно
С лаской я тебя зову.
Не разрушай любовь мою,
Я с силой буду жить.

Только тебе я посвящаю
Всех моих песен грусть.
Жду от тебя, голубоглазка,
Великой любви!..
```
