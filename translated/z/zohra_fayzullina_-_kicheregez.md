# Оригинал

### Зөһрә Фәйзуллина - Кичерегез

```
Төнне-көнгә ялгый-ялгый
Айлар үтә, еллар үтә.
Яфракларын коя-коя
Гомеркәйләр көзе җитә.

Әйтелмәгән хис-тойгылар,
Үтәлмәгән сүз-вәгъдәләр.
Үкендереп, көрсендереп ,
Бәгерләрне телеп үтә.

Кичер мине, җан сөйгәнем,
Чайпылдырдың хисләремне.
Бирә алмадым тик үзеңә
Мөлдерәмә тулы көе.

Кичерегез урман-кырлар,
Уйнап үскән газиз туфрак.
Мин йөримен чит якларда
Чит яклардан яклау сорап.

Көзге җилләр ачы икән,
Гомер үкенечле икән.
Соңлап кына кадерләрен
Белүләре кыен икән.
```

# Перевод

### Зухра Файзуллина - Кичерегез

```
Иртәнге якта уңа-уңа
Айлар китә, еллар китә.
Яфракларын як-якка
Күңелләргә көн җитә.

Әйтеп бетерелмәгән хисләр,
Үтәп җыелмаган сүз-вәгъдәләр.
Күк-күк булеп, иңегеп,
Бәгерләрне җеп-җеп үтә.

Кичер мине, күңел сөйгәнем,
Күк-күк булдың хисләремне.
Бирә алмадым синең өчен
Күк-күк булган күңелемне.

Кичерегез урман-кырлар,
Ойнап үскән күк-күк табанак.
Мин йөримин чит якларда
Күк-күк булган яктауын сорап.

Күңел йөркү ялгыткыч,
Күк-күк булган күңелләр.
Сондакы тик кадерләрен
Белүләре кыек икән.
```
