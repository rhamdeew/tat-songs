# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Бер гомер аз

```
Бер гомер аз, сине яратырга,
Тик карашлар белән озатырга.
Мәңгелеккә йөрәгемә кердең,
Мәхәббәтең белән елатырга.

Бер гомер аз гына,
Бер гомер аз сине яратырга.

Бер гомер аз, сине сагынырга,
Уйларымда гына табынырга.
Төннәремдә күзләремне йомам,
Карашларың белән ябынырга.

Бер гомер аз,
Бер гомер аз, сине сагынырга.

Бер гомер аз, безгә кавышырга,
Күпме еллар үткән кавышырга.
Очрашканбыз ахры бу дөньяда,
Сарыларга гына сабышырга.

Бер гомер аз гына,
Бер гомер аз, безгә кавышырга.
Бер гомер аз,
Бер гомер аз, сине сагынырга.
Бер гомер аз,
Бер гомер аз, сине яратырга.
```

# Перевод

### Зөһрә Котлыгилдина / Салават Низаметдинов - Одной жизни мало

```
Одной жизни мало - чтоб тебя любить,
Лишь взглядами вдогонку проводить.
В сердце навсегда ты вошла, как свет,
Любовью своей заставляя плакать.

Одной жизни мало,
Одной жизни мало - чтоб тебя любить.

Одной жизни мало - по тебе скучать,
Только в мыслях тихо поклоняться.
По ночам я закрываю глаза,
Чтобы укрыться твоим взглядом.

Одной жизни мало,
Одной жизни мало - по тебе скучать.

Одной жизни мало - чтобы нам сойтись,
Сколько лет прошло, чтоб нам сойтись.
Мы, видно, встретились в этом мире,
Лишь чтобы обняться, прижаться.

Одной жизни мало,
Одной жизни мало - чтобы нам сойтись.
Одной жизни мало,
Одной жизни мало - по тебе скучать.
Одной жизни мало,
Одной жизни мало - чтоб тебя любить.
```
