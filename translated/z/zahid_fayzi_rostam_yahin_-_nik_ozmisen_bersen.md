# Оригинал

### Заһид Фәйзи / Рөстәм Яхин - Ник өзмисең берсен?

```
Ник өзмисең берсен чәчәкләрнең,
Өзәр көнең җиткән бит инде?
Ник өзмисең? -
Чәчәкләрнең берсе
Синең өчен,
Синең өчен үскән бит инде.

Ник өзмисең берсен чәчәкләрнең,
Сайлыйсыңмы һаман төсләрен?
Ник өзмисең?
Белмисеңме әллә
Бер чәчәккә
Сарыласын күңел хисләрең?

Өз син берсен - тормыш чәчәгеңне,
Ялгызлыкта гомрең узмасын!
Өз син берсен!
Ләкин назлап сакла,
Сакла үзен -
Вакытыннан алда сулмасын!
```

# Перевод

### Заһид Фәйзи / Рөстәм Яхин - Почему не сорвёшь один?

```
Почему не сорвёшь один из цветов,
Время сорвать ведь уже пришло?
Почему не сорвёшь?
Один из цветов
Для тебя,
Для тебя ведь вырос он.

Почему не сорвёшь один из цветов,
Ты всё выбираешь их краску?
Почему не сорвёшь?
Разве не знаешь,
Что к одному цветку
Потянется сердце, его чувства?

Сорви один - цветок своей жизни,
Чтобы в одиночестве век не прошёл!
Сорви один!
Но бережно храни,
Храни его -
Пусть не увянет раньше срока!
```
