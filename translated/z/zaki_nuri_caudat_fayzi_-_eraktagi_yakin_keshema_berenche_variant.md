# Оригинал

### Зәки Нури / Җәүдәт Фәйзи - Ерактагы якын кешемә (беренче вариант)

```
Таулар сере - синең ачкан серләр,
Таулар керә минем төшемә.
Күңел күзем һаман сиңа карый -
Ерактагы якын,
Ерактагы якын кешемә.

Мәхәббәтем кайнар сулыш өрсен
Көзен-кышын юлда өшемә.
Сиңа оча минем йөрәк хисем
Ерактагы якын,
Ерактагы якын кешемә.

Сукмакларын үрдән-үргә узсын,
Уңыш телим авыр эшеңдә.
Җыр җибәрәм яңа көйгә салып
Ерактагы якын,
Ерактагы якын кешемә.
```

# Перевод

### Заки Нури / Джаудат Файзи - Далекому близкому человеку (первый вариант)

```
Тайны гор - твои раскрытые тайны,
Горы входят в сны мои ночные.
Взгляд души всё к тебе обращён -
Далёкому близкому,
Далёкому близкому человеку.

Пусть любовь моя горячим дыханьем
Греет в осень и в зиму в пути.
К тебе улетает сердечное чувство -
Далёкому близкому,
Далёкому близкому человеку.

Пусть тропинки ведут ввысь и выше,
Удачи желаю в тяжёлой работе.
Шлю песню, вплетая в новый напев -
Далёкому близкому,
Далёкому близкому человеку.
```
