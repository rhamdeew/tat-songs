# Оригинал

### Билгесез / Ирек Галимов - Минем илемә

```
Башкортстаным гүзәл җирем,
Синдә үтә минем гомерем,
Яшьлек елларым, язмыш юлларым.

Әткәй, әнкәй, туганнарым,
Миңа якын булганнарым,
Рәхмәтле сиңа, газиз илемә.

Чит илләр дә бәхет тапмам,
Сездән мәңге баш тартмам,
Туган җиркәем, әткәй - әнкәем.

Язмыш безне аермасын,
Ходай ятим калдырмасын,
Яшик гел бергә, бергә гомергә.

Башкортстаным гүзәл җирем,
Синдә үтә минем гомерем,
Яшьлек елларым, язмыш юлларым.

Алда әле бар күп эшләр,
Күп ачышлар, төзелешләр,
Барсы да сиңа, минем илемә.
```

# Перевод

### Неизвестен / Ирек Галимов - Моей земле

```
Башкортостан, прекрасный край,
В тебе проходит жизнь моя,
Годы юности, пути судьбы.

Отец, мать, родные мои,
Все, кто близок мне и дорог,
Благодарен тебе, родная земля.

В чужих краях не найду счастья,
Никогда не откажусь от вас,
Родная земля, отец мой, мать моя.

Пусть судьба нас не разлучит,
Пусть Господь не оставит сиротами,
Будем жить вместе, вместе навсегда.

Башкортостан, прекрасный край,
В тебе проходит жизнь моя,
Годы юности, пути судьбы.

Впереди ещё много дел,
Много свершений, много строек,
Всё это для тебя, моя земля.
```
