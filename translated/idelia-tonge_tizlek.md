# Оригинал

### Idelia - Тонге тизлек

```
Кояш куптэн баткан, тонге утлар уянган.
Тон патшалык итэ
Мин тон кочен тоям, кошка эурелэм
Мин очам.

Тонге койлэр, тонге тизлек
Тоннен сулышын ишет
Тонге серлэр, тонге хислэр.

Тонге якты утлар юлны яктырталар
Очам жилгэ каршы.
Билгесезлек яна борылышта
Яна узгэрешлэр котэлэр.

Тонге койлэр, тонге тизлек
Тоннен сулышын ишет
Тонге серлэр, тонге хислэр.
```

# Перевод

### Idelia - Ночная скорость

```
Солнце давно село, ночные огни проснулись.
Ночь царствует,
Я чувствую силу ночи, превращаюсь в птицу,
Я лечу.

Ночные мелодии, ночная скорость,
Слышу дыхание ночи,
Ночные тайны, ночные чувства.

Ночные яркие огни дорогу освещают,
Лечу навстречу ветру.
Неизвестность на новом повороте,
Новые перемены ждут.

Ночные мелодии, ночная скорость,
Слышу дыхание ночи,
Ночные тайны, ночные чувства.
```
