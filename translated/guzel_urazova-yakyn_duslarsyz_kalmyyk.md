# Оригинал

### Гөзел Уразова - Якын дусларсыз калмыйк

```
Тормышнын элегэ башы,
Мэхэббэте мэнгелек.
Матур жырлар белэн бизик,
Булэк итик изгелек.

Ялгыз утмэсен гомерлэр,
Зур эшлэр белэн кайныйк,
Илне-жирне яр яратыйк,
Якын дусларсыз калмыйк.

Яшьлекне бергэ каршылап
Бергэ озатыр очен
Ешрак курешик, дусларым,
Укенеч калмас очен.

Ялгыз утмэсен гомерлэр,
Зур эшлэр белэн кайныйк,
Илне-жирне яр яратыйк,
Якын дусларсыз калмыйк.

Аралар ерак булса да
Сез дусларны сагынам,
Бергэ йоргэн вакытларны
Уйлап сезгэ талпынам.

Ялгыз утмэсен гомерлэр,
Зур эшлэр белэн кайныйк,
Илне-жирне яр яратыйк,
Якын дусларсыз калмыйк.
```

# Перевод

### Гөзел Уразова - Не останемся без близких друзей

```
Жизни первые шаги,
Любовь навеки.
Красивыми песнями украсим,
Подарим доброту.

Не пусть проходят годы одиноко,
Великими делами закипим,
Родину-землю ещё больше полюбим,
Без близких друзей не останемся.

Юность вместе встречая,
Вместе провожая,
Чаще встречаемся, друзья мои,
Чтобы сожалений не осталось.

Не пусть проходят годы одиноко,
Великими делами закипим,
Родину-землю ещё больше полюбим,
Без близких друзей не останемся.

Хоть расстояния далеки,
Вас, друзей, скучаю,
О временах, что вместе прошли,
Думая, к вам стремлюсь.

Не пусть проходят годы одиноко,
Великими делами закипим,
Родину-землю ещё больше полюбим,
Без близких друзей не останемся.
```
