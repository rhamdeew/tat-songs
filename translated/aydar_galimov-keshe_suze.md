# Оригинал

### Айдар Галимов - Кеше сузе

```
Белсэк тэ без янэшэдэ
Жиде ятлар йоргэнне
Сизми калдык арабызга
Кеше сузе кергэнне.

Шиэде мэхэббэт голлэре
Тиде жиде ят кузе.
Жиде ятларга калдырып
Аерды кеше сузе.

Кеше сузе, кеше сузе.
Жиде ят итте безне.
Жиде ятлар да кавыша,
Озмик ометебезне.

Адашып калды мэхэббэт
Жиде юлнын чатында.
Жиде ятларга эйлэндек
Кеше сузе хакында.

Кеше сузе, кеше сузе.
Жиде ят итте безне.
Жиде ятлар да кавыша,
Озмик ометебезне.
```

# Перевод

### Айдар Галимов - Людская молва

```
Знай мы, что рядом
Семь чужих идут,
Не заметили бы между нами
Людской молвы проникновения.

Увяли цветы любви,
Коснувшись семи чужих глаз.
Семи чужим оставив,
Разлучила людская молва.

Людская молва, людская молва
Семью чужими сделала нас.
Семь чужих соединяются,
Разрывают надежду нашу.

Заблудилась любовь
На развилке семи дорог.
Семью чужими стали
Из-за людской молвы.

Людская молва, людская молва
Семью чужими сделала нас.
Семь чужих соединяются,
Разрывают надежду нашу.
```
