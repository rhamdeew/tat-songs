# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Гомерлеккә булсын

```
Чын мәхәббәт юк димә син.
Ничек инде булмасын.
Сүндерергә ашыкма син
Күңелеңнең кояшын.

Әй дускаем, мәхәббәтне
Күңелең белән зурла.
Ярты юлда туктап калма,
Тапканчы аны юлла!

Сине көтеп зарыккандыр
Сулмас сөю чәчәгең.
Күңелеңә дәрт өстәсен
Өмет белән яшәвең.

Таптың исә тик югалтма
Җан яраткан ярыңны.
Гомерлеккә сайла берүк
Үзеңә тиң парыңны.
```

# Перевод

### Фанавис Даулатбаев / Инсаф Хабибуллин - Пусть будет на всю жизнь

```
Не говори, что нет любви.
Как может быть, чтобы не было.
Не спеши гасить ты свет
Своего сердца.

Да, друг мой, любовь
Величи в своем сердце.
Не останавливайся на полпути,
Ищи, пока не найдешь!

Тебя дожидаясь, изнемог
Вечнозеленый цветок твой.
Пусть сердце твое воспылает
И живёт надеждою.

Раз нашел, не теряй
Свою любимую душой.
Выбери себе под стать
Пару на всю жизнь.
```
