# Оригинал

### Эдуард Успенский / Владимир Шаинский - Зәңгәрсу вагон

```
Беркайчан да кабат кайтмаслык булып,
Минутлар да китә бик ерак.
Узган гомерләр безгә кызганыч та,
Алдагысы ләкин яхшырак.

Җәймәдәй җәелеп,озын юл сузыла,
Чиген күрергә - офыкка кара.
Һәркем дә,һәркем дә бәхеткә омтыла,
Зәңгәрсу вагон җилдереп бара.

Рәнҗетсәгез кемнедер сез гел юкка,
Календарьлар ябар бу битне.
Яңа маҗаралар алда,дусларым,
Әй,машинист арттыр тизлекне!

Җәймәдәй җәелеп,озын юл сузыла,
Чиген күрергә - офыкка кара.
Һәркем дә,һәркем дә бәхеткә омтыла,
Зәңгәрсу вагон җилдереп бара.

Зәңгәрсу вагон чаба,тирбәлә,һай,
Поезд өсти һаман тизлеген.
Ел буена сузылсын иде лә ул,
Ник тиз үтә икән шушы көн?

Җәймәдәй җәелеп,озын юл сузыла,
Чиген күрергә - офыкка кара.
Һәркем дә,һәркем дә бәхеткә омтыла,
Зәңгәрсу вагон җилдереп бара.
```
