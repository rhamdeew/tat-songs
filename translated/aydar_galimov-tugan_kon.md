# Оригинал

### Айдар Галимов - Туган кон

```
Доньяда син ин кадерлем,
Бердэн- берем син минем.
Нилэр генэ булэк итим
Туган коненэ синен

Жырларымнын ин монлысын,
Таннарымнын ин нурлысын,
Кунелемнен бар жылысын
Булэк итэрмен барысын.

Йолдызлы ин зур галэмне ,
Чэчэк аткан бар голне,
Эллэ булэк итэргэме
Бу тантаналы конне.

Жырларымнын ин монлысын,
Таннарымнын ин нурлысын,
Кунелемнен бар жылысын
Булэк итэрмен барысын.  

Котлыймын мин чын кунелдэн
Сине туган конен белэн.
Синен бэйрэм- минем бэйрэм
Гел шулай булса икэн.

Жырларымнын ин монлысын,
Таннарымнын ин нурлысын,
Кунелемнен бар жылысын
Булэк итэрмен барысын.
```

# Перевод

### Айдар Галимов - День рождения

```
В мире ты самая дорогая,
Единственная ты моя.
Что же только подарю
В твой день рождения

Из песен моих самую мелодичную,
Из рассветов моих самый светлый,
Из души моей всё тепло
В подарок дарю всё.

Самую звёздную вселенную,
Все расцветшие цветы,
Может, подарить
В этот торжественный день.

Из песен моих самую мелодичную,
Из рассветов моих самый светлый,
Из души моей всё тепло
В подарок дарю всё.

Поздравляю я от чистого сердца
Тебя с днём рождения.
Твой праздник — мой праздник
Пусть всегда так будет.

Из песен моих самую мелодичную,
Из рассветов моих самый светлый,
Из души моей всё тепло
В подарок дарю всё.
```
