# Оригинал

### Ильсаф - Бэхетле бул гына

```
Мин буген тош курдем
Хэм елап уяндым.
Ул тоштэ мин сине югалттым.
Ялгызым нишлэрмен,
Кэдерлем, эх, жаным?
Гомерлек бит сине яраттым!

Юрадым ул тошне
Гомерен озынга.
Мэнгегэ югалма,
Бэхетле бул гына!

Изгелек телэдем,
Якшыга юрадым.
Шатлыгы син мина доньянын.
Ягымлы кузлэрен
Яратып туймамын!
Тормышнын бар ямен тыямын.

Юрадым ул тошне
Гомерен озынга.
Мэнгегэ югалма,
Бэхетле бул гына!

Мин буген тош курдем,
Кэдерем, монладым.
Тагында кочлерэк яраттым!
Юатты сузлэрен,,
Янымда булуын.
Курдемдэ- кайгымны тараттым.

Юрадым ул тошне
Гомерен озынга.
Мэнгегэ югалма,
Бэхетле бул гына!
```

# Перевод

### Ильсаф - Будь только счастлива

```
Сегодня я видел сон
И проснулся в слезах.
В том сне я потерял тебя.
Что делать мне одному,
Дорогая моя, душа?
Ведь всю жизнь тебя любил!

Благословил тот сон
На долгие годы.
Навеки не пропадай,
Будь только счастлива!

Добра желал,
К хорошему стремился.
Ты радость мне всего мира.
Твоих прекрасных глаз
Не могу налюбоваться!
Всю красоту жизни впитываю.

Благословил тот сон
На долгие годы.
Навеки не пропадай,
Будь только счастлива!

Сегодня я видел сон,
Дорогая, грустил.
Ещё сильнее полюбил!
Твоих нежных слов,
Чтобы рядом ты была.
Во сне развеял печаль свою.

Благословил тот сон
На долгие годы.
Навеки не пропадай,
Будь только счастлива!
```
