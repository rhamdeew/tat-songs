# Оригинал

### Башкорт халык җыры - Гөләллә

```
Килегез дә көлегез,
Килегез дә көлегез,
Уйнагыз да көлегез.

Гөләллә, гөләллә,
Гөләиллә-ләем.

Матур баса бит әле,
Матур баскан биюләре
Күңел ача бит әле.

Гөләллә, гөләллә,
Гөләиллә-ләем.

Изәннәре бөгелсен,
Әнә шулай биегәндә
Бар дөньялар гөр килсен.

Гөләллә, гөләллә,
Гөләиллә-ләйем.

Йөзләрегез ал икән,
Сикереп-төшеп биюләре
Биг(е)рәк күңелле икән. (Үзе бер гомер икән)

Гөләллә, гөләллә,
Гөләиллә-ләем.
```

# Перевод

### Башкирская народная песня - Гюль-Гюль

```
Приходите и поёмте,
Приходите и поёмте,
Поиграйте и поёмте.

Гюль-гюль, гюль-гюль,
Гюль-гюль-ляйем.

Прекрасно танцует,
Прекрасны её танцы,
Сердце открывает.

Гюль-гюль, гюль-гюль,
Гюль-гюль-ляйем.

Пусть сгибаются колени,
Вот так танцуя,
Пусть приходит весь мир.

Гюль-гюль, гюль-гюль,
Гюль-гюль-ляйем.

Краснеют ваши лица,
Прыгая в танцах,
Сердца ещё более радостны. (Ведь это одна жизнь)

Гюль-гюль, гюль-гюль,
Гюль-гюль-ляйем.
```
