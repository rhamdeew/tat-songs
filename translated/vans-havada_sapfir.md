# Оригинал

### Ванс - Хавада Сапфир

```
Жылаган кузлэр, матур сузлэр
Ахыр очырашуда, мин оны онытмыйм.
Синен жылын мине жылыта,
Э хэзер катып улергэ кала.
Хыяллар суынды,
Мина башка бернэрсэ дэ кирэкми.
Анла, мине анла,
Минем кузлэрем сине генэ курэ иде.

Хавада Сапфир, синен очен
Хавада Сапфир оча.

Боз йорэгемне син генэ,
Син генэ эретэлэсен.
Язмыш сина, бер дэ
Якты йозен курсэтмэде.
Сизми калдым, ничек бер узем
Утырам, салкында, суыкта.
Котэм нэрсэне, мин узем белмим,
Инде нэрсэ минем белэн булган.

Хавада Сапфир, синен очен
Хавада Сапфир оча.
```

# Перевод

### Ванс - Сапфир в воздухе

```
Глаза в слезах, прекрасные слова
В последней встрече, я её не забываю.
Твоё тепло меня согревает,
А сейчас застыв умирать остаётся.
Мечты остыли,
Мне больше ничего не нужно.
Пойми, меня пойми,
Мои глаза видели бы только тебя.

В воздухе Сапфир, для тебя
В воздухе Сапфир летит.

Ледяное сердце моё только ты,
Только ты растапливаешь.
Судьба тебе ни разу
Светлый лик не показала.
Онемел остался, как-то сам
Сижу в прохладе, в холоде.
Жду чего-то, сам не знаю,
Что теперь со мною стало.

В воздухе Сапфир, для тебя
В воздухе Сапфир летит.
```
