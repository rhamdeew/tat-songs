# Оригинал

### Эльмира Җәлилова / Фирзәр Мортазин - Үз ярың табу кирәк

```
Мәңге килмәгәнбез җиргә,
Без килгәнбез китәргә.
Ходай биргән гомерләрне
Кирәк тагын үтәргә.

Каенга имән пар түгел,
Тирәккә тирәк кирәк.
Бәхетле гомер кичәргә
Үз ярың табу кирәк.

Яратмасаң - башлама син,
Башлагансың - ташлама.
Язмышларда пар сагышлар
Ялгышлардан башлана.

Каенга имән пар түгел,
Тирәккә тирәк кирәк.
Бәхетле гомер кичәргә
Үз ярың табу кирәк.

Яшәргә язсын яратып,
Тирәккә булыр тирәк.
Соңгы сулышкача бергә
Типсен гел парлы йөрәк.

Каенга имән пар түгел,
Тирәккә тирәк кирәк.
Бәхетле гомер кичәргә
Үз ярың табу кирәк.
Бәхетле гомер кичәргә
Үз ярың табу кирәк...
Үз ярың табу кирәк...
```
