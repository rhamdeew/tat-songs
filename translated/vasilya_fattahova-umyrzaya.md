# Оригинал

### Василя Фаттахова - Умырзая

```
Кунелемэ килеп керде
Язгы ташкын кукрэу булып.
Тэуге Лэйсэн булып яуды
Язгы умырзаялар алып.

Умырзая, умырзая,
Узмасын гомер зая.
Эйт син мина, умырзая,
Яшьлек ярым минем кая?

Умырзая чэчэклэре
Курсэм, йорэк тибэ ярсып.
Уткэннэрне искэ алып,
Тип син умырзаялар алып.

Умырзая, умырзая,
Узмасын гомер зая.
Эйт син мина, умырзая,
Яшьлек ярым минем кая?
```

# Перевод

### Василя Фаттахова - Надежда

```
В мою душу ворвалась
Весенним половодьем рокочущим.
Первым дождём пролилась
Весенние надежды неся.

Надежда, надежда,
Пусть жизнь не пройдёт зря.
Скажи мне, надежда,
Где юность моя?

Надежды цветы
Когда вижу, сердце бьётся учащённо.
Вспоминая прошедшие дни,
Говорю: "Неси надежды".

Надежда, надежда,
Пусть жизнь не пройдёт зря.
Скажи мне, надежда,
Где юность моя?
```
