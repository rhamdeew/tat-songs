# Оригинал

### Билгесез - Бәхет сулышын тыңла

```
Тәүге бәхет килә аны көткәннәргә,
Саф йөрәкләр тик көтеп алыр.
Минем бәхет синдә, булыйк мәңге бергә,
Синсез, җаным, күңелем сулыр.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.

Күңел күзе күргән, ярын аңлый белгән,
Бәхеткәен шул саклап кала.
Тыйма хисләреңне, тыңла йөрәгемне,
Бәхетебез сулышын тыңла.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.

Бәхет үзебездә саклый белик кенә,
Оҗмах була икән җирдә дә.
Зәңгәр күккә очып, бар җиһанны кочып,
"Мин - бәхетле!" диясем килә.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.
```

# Перевод

### Бильгесез - Слушай дыхание счастья

```
Первое счастье приходит к тем, кто ждет,
Чистые сердца только ждут.
Мое счастье в тебе, будем вечно вдвоем,
Без тебя, душа моя, сердце плачет.

Мысли ведут к тебе, к тебе,
Ты для меня дорог, для меня.
Сердце полно чувств, полно,
Слушай дыхание счастья, слушай.

Сердце видит глазами, понимает завтра,
Хранит счастье в себе.
Не сдерживай чувств, слушай мое сердце,
Слушай дыхание нашего счастья.

Мысли ведут к тебе, к тебе,
Ты для меня дорог, для меня.
Сердце полно чувств, полно,
Слушай дыхание счастья, слушай.

Счастье хранится в нас, как в хрупкой вазе,
Даже в этой земной тьме.
Взлети под синее небо, обними весь мир,
Захочется сказать: "Я счастлив!"

Мысли ведут к тебе, к тебе,
Ты для меня дорог, для меня.
Сердце полно чувств, полно,
Слушай дыхание счастья, слушай.
```
