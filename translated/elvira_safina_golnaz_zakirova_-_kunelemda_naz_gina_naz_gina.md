# Оригинал

### Эльвира Сафина / Гөлназ Закирова - Күңелемдә наз гына (Наз гына)

```
Бергә үткән юллар гел күңелле,
Бергә чакта якты көннәрем.
Син булганга гына чәчәк ата
Күңелемдә бәхет гөлләре.

Сөеп багам синең күзләреңә,
Күзләреңдә бары наз гына.
Син янымда булган чакларымда
Күңелемдә бары яз гына.

Кулларымны алмам кулларыңнан,
Үткән чакта тормыш юлларын.
Синең исем генә телләремдә,
Синең хакта гына уйларым.

Сөеп багам синең күзләреңә,
Күзләреңдә бары наз гына.
Син янымда булган чакларымда
Күңелемдә бары яз гына.

Торган саен күбрәк гашыйк булып,
Синең өчен ясим җан атып.
Бәхетебез гел елмаеп торыр
Яшәсәк гел шулай яратып.

Сөеп багам синең күзләреңә,
Күзләреңдә бары наз гына.
Син янымда булган чакларымда
Күңелемдә бары яз гына.
```

# Перевод

### Эльвира Сафина / Гульнара Закирова - В моём сердце только нежность (Только нежность)

```
Вместе пройденные пути всегда счастливы,
Вместе проведённые светлые дни.
Только ты заставляешь цвести
Счастливые цветы в моём сердце.

С любовью смотрю в твои глаза,
В твоих глазах одна нежность.
Когда ты рядом со мной,
В моём сердце только весна.

Не отнимай свои руки от моих,
Пройдённые дороги жизни.
Только твоё имя на моих устах,
Только о тебе мои мысли.

С любовью смотрю в твои глаза,
В твоих глазах одна нежность.
Когда ты рядом со мной,
В моём сердце только весна.

Чем больше становлюсь, тем больше влюблён,
Ради тебя душу отдаю.
Наше счастье будет всегда смеяться,
Если будем жить, так любя.

С любовью смотрю в твои глаза,
В твоих глазах одна нежность.
Когда ты рядом со мной,
В моём сердце только весна.
```
