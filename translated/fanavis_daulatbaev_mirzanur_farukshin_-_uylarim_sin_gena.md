# Оригинал

### Фәнәвис Дәүләтбаев / Мирзанур Фәрукшин - Уйларым син генә

```
Кайчан килер икән диеп
Сине уйлап йөримен.
Язны көткән гөлләр сыман
Сине көтә күңелем.

Чакырам сине төшемдә,
Чакырам хыялымда.
Әй сөеклем- асылкошым,
Син генә уйларымда!

Килер булсаң телим сиңа
Юлларның хәерлесен.
Тик син генә йөрәгемә
Якынсың, кадерлесең!

Янәшәмдә итеп тоям
Булмасаң да янымда.
Башка беркем кирәк түгел
Дөньяда син барында!
```

# Перевод

### Фәнәвис Дәүләтбаев / Мирзанур Фәрукшин - Мой единственный в мыслях

```
Когда придет ты, думаю,
Без конца и края.
Как ждут цветы о лете,
Так ждёт сердце, моя.

Зову тебя во сне я,
Зову в мечтах порой.
Ах, возлюбленный мой - душа моя,
Ты лишь в мыслях со мной!

Если придешь, пожелаю
Дорог счастливых тебе.
Ты лишь для сердца моего
Близкий, ценный, в судьбе!

Пусть даже не будет рядом,
Ты всегда в сердце мне.
Никого не надо больше
В этом мире, кроме те!
```
