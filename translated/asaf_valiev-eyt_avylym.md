# Оригинал

### Асаф Валиев - Эйт авылым

```
Авыл киче тасмаларда
Эй агыла ят моннар
Нинди монсу эллэ нинди
Телсез калган гармуннар

Эйт авылым, кайда сон мон
Мон кайда сон эйт урам
Аулак ойле хэм гармунлы
Яшьлегемне сагынам

Йорэклэргэ очкын салып
Гармунымда уйныймын
Авылыма моннар биреп
Урамнарны буйлыймын

Эйт авылым, кайда сон мон
Мон кайда сон эйт урам
Аулак ойле хэм гармунлы
Яшьлегемне сагынам

Авыл киче, яшь йорэклэр
Гармун монга сусаган
Сою булып мэннар кайтсын
Авылыма янадан

Эйт авылым, кайда сон мон
Мон кайда сон эйт урам
Аулак ойле хэм гармунлы
Яшьлегемне сагынам
```

# Перевод

### Асаф Валиев - Скажи, деревня моя

```
В деревенских вечерних улочках
Эх, текут чужие печали,
Какая грустная или какая
Безмолвными остались гармони.

Скажи, деревня моя, где же печаль,
Печаль где же, скажи, улица,
С задушевными мелодиями и гармонью
Молодость свою вспоминаю.

В сердца искру бросая,
На гармони своей играю,
Деревне своей печали даря,
По улицам хожу.

Скажи, деревня моя, где же печаль,
Печаль где же, скажи, улица,
С задушевными мелодиями и гармонью
Молодость свою вспоминаю.

Деревенский вечер, молодые сердца
По гармонной печали тоскуют,
Любовью став, пусть вернутся мелодии
В деревню мою снова.

Скажи, деревня моя, где же печаль,
Печаль где же, скажи, улица,
С задушевными мелодиями и гармонью
Молодость свою вспоминаю.
```
