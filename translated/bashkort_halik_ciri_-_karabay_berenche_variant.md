# Оригинал

### Башкорт халык җыры - Карабай* (беренче вариант)

```
Әллиләү генәйем,
Бәллиләү генәйем,
Әсмәбикә тыпыр-тыпыр
Басып килә генәйем.
Күз тимәсен буйларына,
Үсеп килә генәйем.

* - шул көйне иҗат итүче курайчының исеме
```

# Перевод

### Башкирская народная песня - Карабай* (первый вариант)

```
Аллилау, только Аллилау,
Баллилау, только Баллилау,
По чуть-чуть, шаг за шагом
Иду по снегу.
Пусть не замечает глаз,
Пусть растет, как есть.
```
