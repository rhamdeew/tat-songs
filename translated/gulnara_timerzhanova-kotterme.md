# Оригинал

### Гульнара Тимержанова - Коттермэ

```
Син бит мина барыбер килэчэксен,
Килми йореп газиз иттермэ.
Коткэннэргэ минуты да озак,
Коттермэче, шатлык, коттермэ.

Коттермэ, коттермэ, коттермэ,
Яшибез бит жирдэ бер генэ.
Кутэрсен биеккэ-биеккэ
Нурлардан яралган оермэ.

Адресымны, бэлки, белмисендер,
Йорисендер шуна гел читтэ.
Гомер – мизгел, гомер утэ китэ,
Коттермэчэ, сою, коттермэ.

Коттермэ, коттермэ, коттермэ,
Яшибез бит жирдэ бер генэ.
Кутэрсен биеккэ-биеккэ
Нурлардан яралган оермэ.

Бернэрсэ дэ жирдэ ялгыз тугел,
Шатлык-кайгы йори гел бергэ.
Колач жэеп килэчэккэ чабам,
Каршыма чык, бэхетем, коттермэ.

Коттермэ, коттермэ, коттермэ,
Яшибез бит жирдэ бер генэ.
Кутэрсен биеккэ-биеккэ
Нурлардан яралган оермэ.
```

# Перевод

### Гульнара Тимержанова - Жди

```
Ты ведь ко мне всё равно придёшь,
Не приходя, дорогая, не мучай.
Ждущим и минуты долги,
Ждущее счастье, жди.

Жди, жди, жди,
Живём ведь на земле одной.
Поднимешь высоко-высоко
Из лучей сотканную грусть.

Адреса моего, может, не знаете,
Ходите поэтому всё вдали.
Жизнь — миг, жизнь проходит,
Ждущая любовь, жди.

Жди, жди, жди,
Живём ведь на земле одной.
Поднимешь высоко-высоко
Из лучей сотканную грусть.

Ничто на земле не одиноко,
Радость-печаль ходят всегда вместе.
Расправив крылья, в будущее спешу,
Навстречу выйди, счастье моё, жди.

Жди, жди, жди,
Живём ведь на земле одной.
Поднимешь высоко-высоко
Из лучей сотканную грусть.
```
