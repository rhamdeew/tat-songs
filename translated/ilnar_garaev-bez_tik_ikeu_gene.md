# Оригинал

```
Без тик икэу генэ белгэн юллар буйлап
Барабыз мэхэббэт тавына
Барыр юллар ерак бергэ барып житу
Бэхет алга гашыйклар жанына.

Бэхет очен генэ яшьлекнен язында
Син мине мин сине сайладым
Икэу бергэ генэ мэхэббэт тавында
Тынларбыз пар кошлар сайравын.

Икэу бергэ генэ мэхэббэт тавына
Житэрбез яратып доньяны
Без бит икэу генэ бер жандай булганда
Бик матур яшьлекнен хыялы.

Бэхет очен генэ яшьлекнен язында
Син мине мин сине сайладым
Икэу бергэ генэ мэхэббэт тавында
Тынларбыз пар кошлар сайравын.

Шул таунын урлэрен яулаган чагында
Донья ямьлэнер тагын да
Бергэ-бергэ генэ мэхэббэт сагында
Калырбыз мэхэббэт тавында.

Бэхет очен генэ яшьлекнен язында
Син мине мин сине сайладым
```
