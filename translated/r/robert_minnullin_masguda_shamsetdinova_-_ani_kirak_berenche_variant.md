# Оригинал

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Әни кирәк (беренче вариант)

```
Көннәр якты булсын өчен,
Йокы татлы булсын өчен
Әни кирәк, әни кирәк.
Йокы татлы булсын өчен
Әни кирәк.

Җил-яңгырдан саклар өчен,
Усаллардан яклар өчен
Әни кирәк, әни кирәк.
Усаллардан яклар өчен
Әни кирәк.

Ашлар тәмле булсын өчен,
Дөнья ямьле булсын өчен
Әни кирәк, әни кирәк.
Дөнья ямьле булсын өчен
Әни кирәк.

Гөлләр чәчәк атсын өчен,
Бәхет-шатлык артсын өчен
Әни кирәк, әни кирәк.
Иң кадерле кеше җирдә
Әни, димәк.
```

# Перевод

### Роберт Миңнуллин / Масгуда Шамсетдинова - Нужна мама (первый вариант)

```
Чтобы дни были светлы,
Чтобы сон был сладок,
Нужна мама, нужна мама.
Чтобы сон был сладок,
Нужна мама.

Чтоб от ветра и дождя укрыла,
Чтоб от зла защитила,
Нужна мама, нужна мама.
Чтоб от зла защитила,
Нужна мама.

Чтобы пища была вкусной,
Чтобы мир был прекрасен,
Нужна мама, нужна мама.
Чтобы мир был прекрасен,
Нужна мама.

Чтобы цвели цветы,
Чтобы счастье прибавлялось,
Нужна мама, нужна мама.
Самый дорогой человек на земле -
Это мама, значит.
```
