# Оригинал

### Разил Вәлиев / Резеда Ахиярова - Оныт инде

```
Оныт мине, зинһар, оныт инде -
Утка яктым синең хатларны,
Утка яктым ялган вәгъдәләрне,
Утка яктым бергә чакларны.

Оныт мине, зинһар, оныт инде...
Җил тарата учак көлләрен.
Җилгә очты бөтен хыялларым,
Утта янды гашыйк көннәрем.

Оныт мине, зинһар, оныт инде,
Салма мине утлы күмергә.
Кайтып булмый узган мәхәббәткә,
Кайтып булмый узган гомергә.
```

# Перевод

### Разил Валиев / Резеда Ахиярова - Забудь же

```
Забудь меня, прошу, забудь же -
Пусть унесут твои письма,
Пусть унесут ложные обещания,
Пусть унесут встречи вместе.

Забудь меня, прошу, забудь же...
Ветер развеет костра пепел.
Ветру унеслись все мои мечты,
В огне сгорели влюбленные дни.

Забудь меня, прошу, забудь же,
Не бросай в пылающую золу.
Не вернуться к прошедшей любви,
Не вернуться к прошедшей жизни.
```
