# Оригинал

### Расих Шагаев / Таңсылу Габидуллина - Тәтешем баскычлары

```
Яр өстендә оя корган
Безнең Тәтеш каласы.
Идел суын эчеп үскән
Шул каланың баласы.

Иделнең аккошлары ла
Тәтешнең баскычлары.
Баскан саен ялтырыйлар
Бәхетнең ачкычлары.

Алмагачлар кочып ала
Килеп төшсә кунаклар.
Чия пешкәч кызара тау,
Эх, матур да шул чаклар.

Иделнең аккошлары ла
Тәтешнең баскычлары.
Баскан саен ялтырыйлар
Бәхетнең ачкычлары.

Адым саен тарих аның,
Ташы саен истәлек.
Тәтеш халкы хезмәт сөя,
Бирсен Ходай исәнлек.

Иделнең аккошлары ла
Тәтешнең баскычлары.
Баскан саен ялтырыйлар
Бәхетнең ачкычлары.

Йолдызлар күп төнге күктә,
Җирдә Тәтеш бер генә.
Сый-хөрмәтләр күрсәтербез,
Иртә-кичтә кил генә.

Иделнең аккошлары ла
Тәтешнең баскычлары.
Баскан саен ялтырыйлар
Бәхетнең ачкычлары.
```

# Перевод

### Расих Шагаев / Тансылу Габидуллина - Ступени моего Тетюша

```
На крутом берегу возведён
Наш родной город Тетюш.
Водой Волги взращён
Каждый сын этих мест.

Белых лебедей Волги стая
И ступени Тетюша.
С каждым шагом сияют,
Ключи счастья даря.

Яблони обнимают
Встречая гостей дорогих.
Когда спеет черёмуха, красны горы,
Ах, прекрасны те дни.

Белых лебедей Волги стая
И ступени Тетюша.
С каждым шагом сияют,
Ключи счастья даря.

Каждый шаг его - история,
Каждый камень - память жива.
Тетюшский народ труд любит,
Дай им, Боже, здоровья сполна.

Белых лебедей Волги стая
И ступени Тетюша.
С каждым шагом сияют,
Ключи счастья даря.

Звёзд немало в ночном небосводе,
Но Тетюш на земле лишь один.
Почёт и уважение воздадим,
Приходи утром, вечером - приходи.

Белых лебедей Волги стая
И ступени Тетюша.
С каждым шагом сияют,
Ключи счастья даря.
```
