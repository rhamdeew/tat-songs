# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Рәхмәтеңнән, Ходаем, ташлама!

```
Дөнъя авыр. Кайчак танып булмый:
Кайда аклык, кайда каралык.
Бер гөнаһсыз ап-пак чәчкәләр дә
Җил-давылда бетә каралып.

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!

Гөнаһларың булса, һич яшермә
Кил тәүбәгә һәр чак, эй кеше.
Дөнъяга син, өстә тик яктылык,
Бисмиллалы булсың һәр эшең!

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!

Кыл якшылык, чәч син иман нурын,
Бүген чәч, иртәгә соң булыр;
Синең чәчкән шушы нурларыңнан
Бу дөнъя барыбер яктырыр!

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!
```

# Перевод

### Рамиль Чурагулов / Ризван Хәкимов - Не оставь, Господь, без милости Твоей!

```
Мир тяжёл. Порой не различить:
Где белым светом, где чернотой.
Даже белоснежные цветы без вины
В буре ветров темнеют, как тьмой.

Счастья, наставленья прошу у Господа,
С молитвой мой день начинается.
И споткнусь когда, и упаду -
Не оставь, Господь, без милости Твоей!
Не оставь, Господь, без милости Твоей!

Если грехи - не прячь их никогда,
К покаянью приходи, о человек.
В мир вноси ты лишь свет и чистоту,
С именем Всевышнего пусть будет каждый век!

Счастья, наставленья прошу у Господа,
С молитвой мой день начинается.
И споткнусь когда, и упаду -
Не оставь, Господь, без милости Твоей!
Не оставь, Господь, без милости Твоей!

Делай добро, разбрасывай свет веры,
Сеять сегодня - завтра будет поздно;
От света, что ты посеешь,
Этот мир всё равно озарится.

Счастья, наставленья прошу у Господа,
С молитвой мой день начинается.
И споткнусь когда, и упаду -
Не оставь, Господь, без милости Твоей!
Не оставь, Господь, без милости Твоей!
```
