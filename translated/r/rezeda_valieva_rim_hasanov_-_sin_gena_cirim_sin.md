# Оригинал

### Резеда Вәлиева / Рим Хәсәнов - Син генә (Җырым син)

```
Юлыма чыгасың,
Кулларын сузасың,
Оныттым(ң), кадерлем, дисеңме (дисең дә)?
Онытылмый назларың,
Яшьлегем язлары
Исемдә, исемдә, исемдә.

Шелтәле карама,
Ят керде арага.
Сагынып яшимен мин генә. (Сагынып яшимен дип кенә)
Катлаулы (катмарлы) язмышым,
Иң татлы сагышым -
Син генә, син генә, син генә...

Гомерләр үтсә дә,
Көзләрем җитсә дә,
Көзләрем яз иткән нурым син.
Сүнсәм дә сүрелмәс,
Үлсәмдә өзелмәс
Җырым син, җырым син, җырым син!
```

# Перевод

### Резеда Валиева / Рим Хасанов - Только ты (Ты - моя песня)

```
Выходишь тебе навстречу,
Простираешь свои руки,
«Я забыл(а), любимая(ый)», - хочешь сказать (или скажешь)?
Не забываются ласки твои,
Молодости моей весны
В имени, в имени, в имени.

Не смотри с упрёком,
Чужое вкралось расстояние.
Тоскую, только я живу (Тоскую, будто только я живу).
Сложная судьба моя,
Самая сладкая тоска моя -
Только ты, только ты, только ты...

Пусть пройдут века,
Пусть сгорят глаза мои,
Ты - свет, что глаза мои зажёг.
Если погасну - не погаснет,
Если умру - не оборвётся
Ты моя песня, ты моя песня, ты моя песня!
```
