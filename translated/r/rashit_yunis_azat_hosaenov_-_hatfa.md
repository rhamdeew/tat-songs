# Оригинал

### Рәшит Юныс / Азат Хөсәенов - Хәтфә

```
Хәтфә урман буйларыннан
Йөрми микән кем генә?
Сиңа җыйган чәчәкләрнең
Ник алдың берсен генә?

Әх, күзеңнең карасы,
Йөрәгемнең парәсе,
Әйт, җаныем, авылыгызга
Кайсы юлдан барасы?

Хәтфә урман буйларыннан
Чишмә ага чылтырап.
Мине күргәч ник качасың
Елмаеп, җырлар җырлап?

Әх, күзеңнең карасы,
Йөрәгемнең парәсе,
Әйт, җаныем, авылыгызга
Кайсы юлдан барасы?
```

# Перевод

### Рәшит Юныс / Азат Хөсәенов - Хәтфә

```
По бархатным лесным тропинкам
Кто же там не ходил?
Из цветов, что для тебя собрал,
Почему ты взяла лишь один?

Ах, чернота твоих глаз,
Отрада сердца моего,
Скажи, душа моя, к селу вашему
Какою тропой идти?

По бархатным лесным тропинкам
Родник звенит, струясь.
Увидев меня, зачем ты убегаешь,
Улыбаясь, песню напевая?

Ах, чернота твоих глаз,
Отрада сердца моего,
Скажи, душа моя, к селу вашему
Какою тропой идти?
```
