# Оригинал

### Рәми Гарипов / Рафик Сәлмәнов - Минем йолдызым

```
Гомерем буе мин бит ялгызым,
Эзлимен сине, якты йолдызым.
Югалдың кайларда?
Юл алдың кайдарга?
Синсез, йолдызым, юлым бик озын...

Нигәдер күпне көтә күңелем,
Көткәнгә микән күпне киң илем!
Дауыллы елларда,
Бормалы юлларда
Мин җиңел йөкле булмам, сөеклем!

Чакыра тагы ерак юлларым,
Юлдашым минем - йөрәк моңнарым,
Тынгысыз көннәрдә,
Йолдызсыз төннәрдә
Күрче мин барын, яктырт юлларым!
```

# Перевод

### Рәми Гарипов / Рафик Сәлмәнов - Моя звезда

```
Всю жизнь я, кажется, одинок,
Ищу тебя, моя яркая звезда.
Где ты потерялся?
Куда путь держишь?
Без тебя, звезда моя, путь так долог...

Чего-то многого ждёт моё сердце,
Много ли ждала моя широкая грудь!
В бурные ветра,
В извилистые дороги
Я с лёгкой ношей не был, любимая!

Зовут меня снова далёкие дороги,
Мой попутчик - сердечная песня,
В неспокойные дни,
В беззвёздные ночи
Узри всё, озари мои пути!
```
