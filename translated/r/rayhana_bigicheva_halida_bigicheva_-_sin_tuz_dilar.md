# Оригинал

### Рәйхана Бигичева / Халидә Бигичева - Син түз, диләр

```
Син түз, диләр, тик әйтмиләр.
Ничек түзеп бетәргә?
Сүнгән мәхәббәт хисләрен
Ничек яңа итәргә?

Әллә инде шул яшәүме,
Әллә инде шул язмыш?!
Әллә инде мин ялгыштым,
Әллә син шундый, тормыш...

Ышанмыйм мин сүзләреңә,
Ышанмыйм күзләреңә.
Зая үткән гомерләргә
Нинди бәя бирергә?

Әллә инде шул яшәүме,
Әллә инде шул язмыш?!
Әллә инде мин ялгыштым,
Әллә син шундый, тормыш...

Арыдым алданулардан.
Ышандым, тик алдандым.
Гомерем буе яхшылыкка,
Җылы сүзгә мохтандым.

Әллә инде шул яшәүме,
Әллә инде шул язмыш?!
Әллә инде мин ялгыштым,
Әллә син шундый, тормыш...
```
