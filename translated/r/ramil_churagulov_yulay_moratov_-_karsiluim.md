# Оригинал

### Рамил Чурагулов / Юлай Моратов - Карсылуым

```
Язларымда көткән идем,
Син, җаным, соңлап килдең.
Гомеремә карлар белән
Карсылу булып кердең.

Карсылуым - мәхәббәтем,
Зинһар өшетмә.
Сөюләрнең иң кайнарын
Бүләк ит миңа!

Язларымда көткән идем
Назлы мәхәббәт гөлем.
Ак кышымны яз иттең дә
Карсылу булып килдең.

Карсылуым - мәхәббәтем,
Зинһар өшетмә.
Сөюләрнең иң кайнарын
Бүләк ит миңа!

Язларымда көткән идем,
Карсылуым - кыш кызы.
Умырзая булып килдең,
Саумы, мәхәббәт язым!

Карсылуым - мәхәббәтем,
Зинһар өшетмә.
Сөюләрнең иң кайнарын
Бүләк ит миңа!
```

# Перевод

### Рамил Чурагулов / Юлай Моратов - Моя встречная метель

```
Веснами своими ждал я,
Ты, душа моя, неспешно пришла.
В мою жизнь вместе с метелями
Встречной вьюгой вошла.

Моя встречная - любовь моя,
Прошу, не пророжай.
Самую горячую из всех любвей
Подари мне в дар!

Веснами своими ждал я
Нежный цветок любви моей.
Белую зиму в весну превратил
Встречной вьюгой пришёл.

Моя встречная - любовь моя,
Прошу, не пророжай.
Самую горячую из всех любвей
Подари мне в дар!

Веснами своими ждал я,
Моя встречная - зимы девушка.
Примулой весенней пришла ты,
Здравствуй, весна любви моей!

Моя встречная - любовь моя,
Прошу, не пророжай.
Самую горячую из всех любвей
Подари мне в дар!
```
