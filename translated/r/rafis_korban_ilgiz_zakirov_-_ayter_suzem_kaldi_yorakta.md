# Оригинал

### Рафис Корбан / Илгиз Закиров - Әйтер сүзем калды йөрәктә

```
Төнге урам буйлап
Төрле уйлар уйлап,
Җитәкләшеп күпме йөрсәк тә,
Әллә онытылып,
Бер үкенеч булып
Әйтер сүзем калды йөрәктә.

Аерылды юллар,
Язгы ташкын сулар
Югалттылар синең эзеңне.
Йөрәгемдә йөртәм
Онытмыйча, иркәм,
Әйтелмичә калган сүземне.

Бер очрарсың диеп,
Күзләреңнән сөеп,
Бик сагындым, диеп дәшәргә
Йөрәгемдә йөртәм
Онытмыйча, иркәм...
Шул сүзем дәрт бирә яшәргә.
```

# Перевод

### Рафис Корбан / Ильгиз Закиров - Слова остались в сердце

```
По ночной улице
Разные думы думая,
Сколько б мы ни шли, обнявшись,
Словно забытые,
Одним сожалением став,
Слова мои остались в сердце.

Разошлись дороги,
Весенние бурные воды
Смыли твой след.
В сердце своём храню
Не забывая, милый мой,
Невысказанное слово.

Надеясь встретить вновь,
В глаза твои смотря,
Сказать, что так скучала,
В сердце своём храню
Не забывая, милый...
Это слово даёт мне силы жить.
```
