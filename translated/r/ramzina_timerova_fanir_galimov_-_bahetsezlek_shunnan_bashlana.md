# Оригинал

### Рәмзинә Тимерова / Фәнир Галимов - Бәхетсезлек шуннан башлана

```
Дуслар гына булып очрашулар
Уяттылар сөю хисләрен.
Кавышырга һич тә мөмкин түгел
Сагышланып үтә кичләрем.

«Сөям» диеп килмә яннарыма,
Язмышым бит минем башкада.
Кеше бәхетен тартып ала күрмә
Бәхетсезлек шуннан башлана.

«Сөю - шатлык», диләр, мин көенәм,
Карашларың өзә бәгырьне.
Көтмәгәндә уйламаган идем
Булырсың, дип шундый кадерле.

«Сөям» диеп килмә яннарыма,
Язмышым бит минем башкада.
Кеше бәхетен тартып ала күрмә
Бәхетсезлек шуннан башлана.
```

# Перевод

### Рамзина Тимерова / Фанир Галимов - Безначалие начинается с этого

```
Встречи только с друзьями
Пробудили чувство любви.
Не представляется возможным
Ночами скучать без устали.

Не приходи ко мне, говоря «люблю», 
Моя судьба совсем другая.
Человеческое счастье забирается незаметно,
Безначалие начинается с этого.

Любовь - радость, - говорят, я плачу, 
Твой взгляд рвет мое сердце.
Неожиданно, не думая,
Я подумал о тебе с такой заботой.

Не приходи ко мне, говоря «люблю», 
Моя судьба совсем другая.
Человеческое счастье забирается незаметно,
Безначалие начинается с этого.
```
