# Оригинал

### Рәшит Фазлыев / Мөдәрис Морзаханов - Мәхәббәтем (өченче вариант)

```
Үзеңә каратып бер сөй,
Кочаклап ярат инде!
Сине сагынган чакларым,
Сагыштан сакла мине!

Үзеңә каратып бер сөй,
Кочаклап ярат инде!
Үтә кайгылы чакларым,
Кайгымнан сакла инде!

Мәхәббәтем, сиңа чиксез,
Юк җирдә аның тиңе.
Синнән башка яши алмам,
Бәхетсез итмә инде!
```

# Перевод

### Рашит Фазлиев / Модарис Морзаханов - Моя любовь (третий вариант)

```
Посмотри на меня и скажи,
Обними и люби же!
В минуты, когда тоскую по тебе,
Спаси меня от этой тоски!

Посмотри на меня и скажи,
Обними и люби же!
В мои печальные минуты,
Спаси от горя же!

Моя любовь, к тебе безгранична,
Нет ей равной на земле.
Без тебя не могу я жить,
Не делай же несчастным!
```
