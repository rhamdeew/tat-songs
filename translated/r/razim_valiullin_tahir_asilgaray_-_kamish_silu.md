# Оригинал

### Разим Вәлиуллин / Таһир Асылгәрәй - Камыш сылу

```
Һай, камыш сылукай
Җырлале, моң булсын.

Сагышлар ярасын
Басарлык җыр тусын.

Камышым, сылуым
Кубызлар кушылсын.

Яшь кызлар ушыннан
Язарлык җыр тусын.

Кураем, уйнале
Егетләр ут алсын.

Очкыннар чәчрәтеп
Биерлек көй калсын.

Камышым сылуым
Сыз да сыз төрлесен.

Әбиләр - апайлар,
Агайлар - бабайлар,
Малайлар - шалайлар
Барсыда биесен.
```

# Перевод

### Разим Валиуллин / Тахир Асылгараев - Камыш красавица

```
Ой, камыш красавица
Пойте, пусть звучит мелодия.

Пусть скорби утихнут
Звучать способная песня пусть родится.

Мой камыш, моя красавица
Курай的声音 пусть присоединится.

Из-за молодых девушек
Слагаемая песня пусть народится.

Мой кураи, играй
Парни пусть загорятся.

Искры разбрасывая
Танцевальную мелодию пусть оставит.

Мой камыш, моя красавица
И скрип и скрежет пусть разной будет.

Бабушки - тётушки,
Дяди - дедушки,
Парни - ребята
Все пусть танцуют.
```
