# Оригинал

### Рифат Сәлах / Илназ Гарипов - Күрше кызы

```
Күршедә яшисең,
Эндәшәм дәшмисең,
Күреп кенә калам синең көлүеңне.
Син генә уемда,
Сер генә син миңа,
Бер күз сирпеп җилкетәсең күңелемне.

Күрәсеңме, күргәнеңне күрәм,
Минем уйлар синең уйда әле.
Яратмаган булып яраттырып
Әйдә, кылан, әйдә, уйна әле!

Йөрәктә йөрисең,
Үзең дә белмисең -
Койма аша гына хәзер күрәм сине.
Җанымны яндырып,
Гел битараф булып,
Сыныйсыңмы, күрше кызы, хисләремне?

Күрәсеңме, күргәнеңне күрәм,
Минем уйлар синең уйда әле.
Яратмаган булып яраттырып
Әйдә, кылан, әйдә, уйна әле!

Торасың күршедә,
Сизмисең берни дә,
Уйнап кына йөрисең мин, күрше кызы.
Мин булсам якында,
Карашың янымда -
Уйларымда йөрисең син, күрше кызы.

Күрәсеңме, күргәнеңне күрәм,
Минем уйлар синең уйда әле.
Яратмаган булып яраттырып
Әйдә, кылан, әйдә, уйна әле!
```

# Перевод

### Рифат Сәлах / Илназ Гарипов - Соседская дочь

```
Рядом живёшь ты,
Не зови и не вздыхаешь,
Вижу только твою улыбку издали.
Только ты в мыслях,
Ты лишь тайна для меня,
Взглядом одним трепещешь сердце моё.

Видишь ли, что вижу тебя я вижу,
Мои мысли о тебе всё ещё.
Не любящей притворяясь, заставляя любить,
Давай, хитри, давай, играй же!

В сердце ходишь ты,
Сама того не зная -
Через забор лишь вижу тебя сейчас.
Душу мою зажигая,
Вечно равнодушною seeming,
Испытываешь ли, соседка, чувства мои?

Видишь ли, что вижу тебя я вижу,
Мои мысли о тебе всё ещё.
Не любящей притворяясь, заставляя любить,
Давай, хитри, давай, играй же!

Живёшь по соседству,
Ничего не чувствуешь,
Играя лишь ходишь ты, соседская дочь.
Если я рядом,
Взгляд твой рядом -
В мыслях моих ходишь ты, соседская дочь.

Видишь ли, что вижу тебя я вижу,
Мои мысли о тебе всё ещё.
Не любящей притворяясь, заставляя любить,
Давай, хитри, давай, играй же!
```
