# Оригинал

### Рәфик Юныс / Сара Садыкова - Якты өмет (Төнге җыр)

```
Тамчы-тамчы ак йолдызлар
Күңелемә тамалар.
Ак уйларым ераклардан
Сине эзләп табарлар.

Якты-якты өмет күрәм
Айның уйчан йөзендә.
Син дә хәзер шушы зәңгәр
Төн чатыры эчендә!

Татлы-татлы хыяллардан
Мөмкин түгел арыну.
Сагынасым килеп тора
Арткан саен сагыну!
```

# Перевод

### Рафик Юныс / Сара Садыкова - Светлая надежда (Ночная песня)

```
Каплями белых звезд
На сердце падает свет.
Светлые мысли издалека
Тебя найдут, тебя приведут.

Я вижу светлую надежду
В задумчивом лике луны.
И ты сейчас под этим синим
Ночным шатром тишины!

От сладких, сладких грез
Нельзя никак отрешиться.
Хочется снова скучать,
Чем больше - тем сильней тоска!
```
