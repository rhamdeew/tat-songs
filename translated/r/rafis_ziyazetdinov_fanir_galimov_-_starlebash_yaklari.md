# Оригинал

### Рәфис Зиязетдинов / Фәнир Галимов - Стәрлебаш яклары

```
Җирнең асыл ноктасы
Стәрлебаш исемле.
Шул якларга ашкындыра
Әллә соң син тылсымлы?!

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!

Кайтыйм әле, күрим әле
Стәрлебаш якларын.
Күңелемнән җуйыйм әле
Сагыну газапларын.

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!

Туган як ул-әни кебек:
Изге дә, кадерле дә.
Мәхәббәтем шуңа көчле,
Сүнми дә, сүрелми дә.

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!
```

# Перевод

### Ряфис Зиязетдинов / Фанир Галимов - Стороны Стерлебаша

```
Драгоценная точка земли
Стерлебашем зовётся.
К тем краям стремится душа -
Неужель ты волшебна?!

Вишни в рощах там
«Иди-иди» нашепчут.
Ты зажжённый огонь любви
Неужель уже знают?!

Вернусь ещё, увижу ещё
Стороны Стерлебаша.
Из сердца очищу ещё
Муки тоски по дому.

Вишни в рощах там
«Иди-иди» нашепчут.
Ты зажжённый огонь любви
Неужель уже знают?!

Родной край словно мать:
И свят, и дорог.
Любовь моя к тебе сильна,
Не гаснет, не меркнет никогда.

Вишни в рощах там
«Иди-иди» нашепчут.
Ты зажжённый огонь любви
Неужель уже знают?!
```
