# Оригинал

### Рәми Гарипов / Салават Низаметдинов - Әкият (беренче вариант)

```
Әкиятсез бала бала булмый,
Мәхәббәтсез - шагыйр шигырсыз.
Әкиятемне тартып алмасаңчы,
Калдырмачы мине син җырсыз.

Тик әкият кенә булсын toyгым,
Мин барыбер аңа табынам.
Әкиятем әгәр юкка чыкса,
Мин чыгарам аны яңадан.

Гел ышанасым килә могҗизага,
Ышанычсыз күңел - ул җырсыз.
Әкиятсез бала бала булмый,
Мәхәббәтсез - шагыйр шигырсыз.
```

# Перевод

### Рәми Гарипов / Салават Низаметдинов - Сказка (первый вариант)

```
Ребёнок без сказки - не настоящий ребёнок,
Поэт без любви - без стихов.
Не отнимай у меня мою сказку,
Не оставляй меня без песен.

Пусть сказка будет моей единственной мыслью,
Я всё равно ей поклоняюсь.
Если моя сказка исчезнет,
Я создам её заново.

Всегда хочется верить в чудо,
Душа без веры - она без песен.
Ребёнок без сказки - не настоящий ребёнок,
Поэт без любви - без стихов.
```
