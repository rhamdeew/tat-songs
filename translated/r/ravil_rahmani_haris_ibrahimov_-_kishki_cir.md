# Оригинал

### Равил Рахмани / Харис Ибраһимов - Кышкы җыр

```
Чәчәкләнеп, ак кар булып,
Ява җиргә җиһан нуры.
Кояшның бу - галәм тулып
Сагынып килгән кышкы җыры.

Иңә назлы йомшак аһәң,
Бөтен назлы шау чәчәктә, -
Чәчәк сибеп керә сөю
Яңа елга, киләчәккә!

Арта аклык, кайта сафлык,
Очкынланып оча өмет...
Ялкыннарын карга төреп,
Язлар тора алда көлеп.
```

# Перевод

### Равил Рахмани / Харис Ибраһимов - Зимняя песня

```
Расцветая, белым снегом
Льется свет вселенной на землю.
Это солнца - полный космос -
Зимний гимн, пришедший с тоскою.

Льется нежный, мягкий говор,
В каждом шорохе цветущем, -
Сыплет лепестками любовь
В Новый год, в грядущее время!

Прибывает белизна, 
Возвращается чистота,
И, искрясь, летит надежда...
Пламя в снег завернув, 
Весна впереди улыбается.
```
