# Оригинал

### Рәис Гыймадиев / Фирзәр Мортазин - Моңлы язмышым

```
Киләчәккә уйларым бар, якты уйларым.
Сезнең белән үтсә иде гомер юлларым.

Сезнең белән бәйләдем моңлы язмышны,
Яннарымда сез булганга сөям тормышны.

Күңелемдә сагышым бар, моңлы сагышым.
Мизгеленә кәдәр тоям вакыт агышын.

Сезнең белән бәйләдем моңлы язмышны,
Күңелемдә җыр булганга сөям тормышны.

Йөрәгемдә җырларым бар, туган җырларым.
Насыйп булса, сезгә тагын, тагын җырлармын.

Сезнең белән бәйләдем моңлы язмышны,
Йорәгемдә җыр булганга сөям тормышны.
```

# Перевод

### Раис Гыймадиев / Фирзәр Мортазин - Моя печальная судьба

```
Есть у меня мысли о будущем, светлые мысли.
Если бы с вами проходили мои жизненные пути.

С вами я связал свою печальную судьбу,
Потому люблю жизнь, что вы рядом со мной.

В душе моей есть тоска, печальная тоска.
До мгновения чувствую течение времени.

С вами я связал свою печальную судьбу,
Потому люблю жизнь, что в душе моей живет песня.

В сердце моем есть песни, родные песни.
Если будет суждено, вам снова и снова спою.

С вами я связал свою печальную судьбу,
Потому люблю жизнь, что в сердце моем живет песня.
```
