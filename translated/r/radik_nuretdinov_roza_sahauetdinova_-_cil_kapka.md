# Оригинал

### Радик Нуретдинов / Роза Сәхәүетдинова - Җил капка

```
Җил капканы җил ачтымы,
Кемдер керә, кемдер бар.
Кайтыр кешем булмаса да,
Килер кешем кемдер бар.

Җил капканы җил ачтымы,
Чылтырады келәсе.
Кемнән хәбәр алып килгән,
Бик тә килә беләсем.

Җил капканы җил ачтымы,
Кемдер керә, кемдер бар.
Көткән кешем килмәсә дә,
Килер кешем кемдер бар.
```

# Перевод

### Радик Нуретдинов / Роза Сәхәүетдинова - Ветер открыл калитку

```
Ветер, это ветер калитку открыл,
Кто-то заходит, кто-то ушёл.
Хоть нет того, кого жду домой,
Есть кто-то, кто всё же придёт.

Ветер, это ветер калитку открыл,
Звенит её лёгкий засов.
Чью же весть он сюда донёс,
Очень хочу я узнать всё.

Ветер, это ветер калитку открыл,
Кто-то заходит, кто-то ушёл.
Даже если тот, кого жду, не придёт,
Есть кто-то, кто всё же придёт.
```
