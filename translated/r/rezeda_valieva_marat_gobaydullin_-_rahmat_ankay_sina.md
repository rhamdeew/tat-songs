# Оригинал

### Резеда Вәлиева / Марат Гобәйдуллин - Рәхмәт, әнкәй, сиңа

```
Шомырт чәчәк атса,
Илгә кошлар кайтса,
Җаным тула минем назларга.
Рәхмәт, әнкәй, сиңа,
Якты тормыш миңа
Бүләк иткән өчен язларда.

Йөрәк җылың белән,
Бишек җырың, белән
Күңел кылларымны чирткәнсең.
Рәхмәт, әнкәй, сиңа,
Сабый чакларымнан
Бу дөньяга гашыйк иткәнсең.

Язгы кояш төсле,
Сулмас яшәү көчен
Биргәнсең син миңа гомергә.
Рәхмәт, әнкәй, сиңа,
Кояш булып миңа
Мәңге-мәңге яшә бу җирдә.
```

# Перевод

### Резеда Вәлиева / Марат Гобәйдуллин - Спасибо, мама, тебе

```
Когда цветёт черёмуха,
Когда к нам птицы возвращаются,
Душа моя полна нежности.
Спасибо, мама, тебе
За светлую мне жизнь,
Подаренную вешними днями.

Твоим теплом сердечным,
Твоей колыбельной
Ты струны моей души тронула.
Спасибо, мама, тебе,
С младенческих лет
Ты мир этот мне полюбить помогла.

Как солнце по весне,
Несгибаемую силу жить
Ты мне дала на целый век.
Спасибо, мама, тебе,
Будь мне солнцем,
Вечно-вечно живи на земле.
```
