# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Айгөлем

```
Синсез, нурсыз көнем,
Ай - йолдызсыз көнем,
Сөю - назлар көтеп үтә гомерем.

Бәхет, язым килер,
Җанга сөю керер,
Килсәң яныма Айгөлем.

Көннәр аяз булсын,
Күңелдә яз булсын,
Мәңге балкып торсын таң йолдызы.

Йөрәгемнән китмә,
Мине ялгыз итмә,
Матур җырла тиң оҗмах кызы.

Айлар, еллар үтәр,
Күпме гөлләр шиңәр,
Син генә сулсмассың сөю гөлем.

Сөю таҗын үрим,
Гомер бергә булыйк,
Килче яныма Айгөлем.
```

# Перевод

### Рамил Чурагулов / Ризван Хакимов - Айгөлем

```
Без тебя, моё солнце без света,
День без луны и звёзд,
Любовь и нежность ждут, проходит моя жизнь.

Счастье, моя весна придёт,
Любовь войдёт в душу,
Приходи ко мне, Айгөлем.

Пусть дни будут ясными,
Пусть в душе будет весна,
Пусть вечно сияет утренняя звезда.

Не уходи из моего сердца,
Не оставляй меня одинокой,
Прекрасно пой, девочка рая.

Месяцы, годы пройдут,
Сколько цветов завянет,
Только ты не завянешь, мой цветок любви.

Венец любви украшу,
Давай проживём жизнь вместе,
Приходи ко мне, Айгөлем.
```
