# Оригинал

### Ренат Харис / Эльмир Низамов - Атлар чаба (икенче вариант)

```
Атлар кая алып бара -
Чамалыйбыз, беләбез.
Ә хәзергә кара яллы
Туры атта җиләбез -
Тояклары чакма чага,
Чакма булмый чаткысыз.
Без бүген якты дөньяга
Җилеп йөрер чагыбыз.

Көмеш яллы атлар алда,
Тояклары ак карда.
Безнең дә атланасы бар
Буран яллы атларга.

Көзге алтын яфраклардан
Ясарбыз әле бизәк.
Без әлегә җилдерәбез,
Алтыннарга кул изәп -
Тояклардан очкын оча,
Әйләнә йолдызларга.
Йолдыз сибеп калдырабыз
Безгә гашыйк кызларга.

Көмеш яллы атлар алда,
Тояклары ак карда.
Безнең дә атланасы бар
Буран яллы атларга.

Алтын яллы атлар чаба
Алтын яфраклы юллардан.
Безгә таба бәхет килә,
Без барасы юлларда.

Көмеш яллы атлар алда,
Тояклары ак карда.
Безнең дә атланасы бар
Буран яллы атларга.
```

# Перевод

### Ренат Харис / Эльмир Низамов - Кони скачут (второй вариант)

```
Куда несут нас кони -
Мы чувствуем, мы знаем.
А нынче мчимся мы
На вороных прямых -
Подковы искры высекли,
Без искр не будет огня.
Сегодня нам по свету
Лететь - настал наш час.

Впереди кони с серебряной гривой,
Подковы - по белому снегу.
И нам ещё предстоит
Оседлать коней с вьюжной гривой.

Из осенней золотой листвы
Мы узор ещё сложим.
Пока же мы несёмся,
Золоту маша рукой -
Из-под копыт искры летят,
Превращаясь в звёзды.
Мы звёздной россыпью
Одарим влюблённых в нас дев.

Впереди кони с серебряной гривой,
Подковы - по белому снегу.
И нам ещё предстоит
Оседлать коней с вьюжной гривой.

Кони с золотой гривой мчатся
По дорогам, усыпанным золотой листвой.
К нам навстречу счастье идёт,
По тем дорогам, где мы идём.

Впереди кони с серебряной гривой,
Подковы - по белому снегу.
И нам ещё предстоит
Оседлать коней с вьюжной гривой.
```
