# Оригинал

### Разил Вәлиев / Фуат Әбүбәкеров - Чаптарым

```
Нинди язмыш безне көтә икән?
Кая чаба икән чаптарым?
Аргамагым эзләп чаба төсле
Бер кайгысыз колын чакларын.
Аргамагым чаба, һаман чаба, -
Тирә-юньдә бары ут кына...
Берәүләргә алтын, алтын яңгыр ява,
Алтын ирләр яуда, яуда ятып кала...
Юллар илтә безне упкынга.

Аргамагым, күпме чапсаң да син,
Әйләнмәссең кабат колынга.
Икебезнең язмыш бер үк төсле -
Икебез дә җирдә кол гына.
Безнең өйгә бәхет килер микән?
Булыр микән бәйсез чакларым?
Кан коюлар безгә, безгә кирәк түгел,
Ирек даулап ярсый, ярсый безнең күңел, -
Ярсып чаба минем чаптарым.

Аргамагым илләр буйлап чаба,
Ирек даулап керә яуларга.
Кара көннәр, янып, артта кала,
Таңнар ата, офык аллана.
Ирләр белән бергә илләр чаба,
Эзләп чаба бәйсез чакларын...
Батыр ирләр, илләр яуда бәхет таба...
Бәхет эзләп һаман, һаман алга чаба
Ялкын яллы кызыл чаптарым!
```

# Перевод

### Разил Вәлиев / Фуат Әбүбәкеров - Мой скакун

```
Какая судьба нас ждёт, скажи?
Куда летит мой скакун?
Мой аргамак будто ищет
Дни беззаботные жеребячьи.
Мой аргамак мчится, всё мчится, -
Вокруг лишь пламя одно...
Одним льётся золото, золотой дождь идёт,
Золотые мужи в бою, в бою ложатся...
Дороги ведут нас в пропасть.

Аргамак мой, сколько б ты ни мчал,
Жеребёнком не станешь вновь.
Наша доля одного цвета -
Мы оба на земле лишь рабы.
Придёт ли счастье в наш дом?
Будут ли часы свободы?
Кровопролитие нам, нам не нужно,
За свободу ярится, ярится сердце наше, -
В порыве мчится мой скакун.

Аргамак мой по странам несётся,
За свободой вступает в сражения.
Чёрные дни, сгорая, остаются позади,
Рассветы встают, алеет горизонт.
С мужами вместе страны мчатся,
Ищут часы независимости...
Храбрые мужи, страны в бою находят счастье...
В поисках счастья всё вперёд, всё вперёд несётся
Рыжий с огненной гривой мой скакун!
```
