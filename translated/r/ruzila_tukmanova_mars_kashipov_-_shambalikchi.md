# Оригинал

### Рузилә Тукманова / Марс Кәшипов - Шәмбалыкчы

```
Тау ягында Шәмбалыкчы,
Болынлыклар, урманнар.
Җырга гашыйк, сүзгә тапкыр
Яши дуслар, туганнар.

Шәмбалыкчы, балыкчы,
Балыкчы, Шәмбалыкчы.
Беркайда да табалмассың
Андагы матурлыкны.

Монда туган җырчы Фәхри,
Эчкән чишмә суларын.
Сандугачкай моңы белән
Яңгыраткан кырларын.

Шәмбалыкчы, балыкчы,
Балыкчы, Шәмбалыкчы.
Беркайда да табалмассың
Андагы матурлыкны.

Яшьләр чишмә буйларында
Уенлыкта җырлыйлар.
Кайнар мәхәббәт сүзләрен,
Җырчы моңын саклыйлар.

Шәмбалыкчы, балыкчы,
Балыкчы, Шәмбалыкчы.
Беркайда да табалмассың
Андагы матурлыкны.
```

# Перевод

### Рузилә Тукманова / Марс Кашипов - Шамбалыкчи

```
У подножья гор Шамбалыкчи,
Луга, леса кругом.
Влюблённые в песни, речисты
Живут друзья, родня.

Шамбалыкчи, рыбацкий,
Рыбацкий, Шамбалыкчи.
Нигде больше не найдёшь
Такой красоты, как там.

Здесь родился певец Фахри,
Пивший родниковую воду.
Соловьиным напевом
Оглашавший поля.

Шамбалыкчи, рыбацкий,
Рыбацкий, Шамбалыкчи.
Нигде больше не найдёшь
Такой красоты, как там.

Молодёжь у родника
С весельем песни поёт.
Горячие слова любви,
Певческую мелодию хранят.

Шамбалыкчи, рыбацкий,
Рыбацкий, Шамбалыкчи.
Нигде больше не найдёшь
Такой красоты, как там.
```
