# Оригинал

### Радик Муслухов / Илназ Баһ - Бер күрешү - үзе бер гомер (дүртенче вариант)

```
Якты дөньялардан туймасак та,
Кабат туып булмый яңадан.
Кеше бер кат туа, бер кат яши,
Тик бер кабат китә дөньядан!

Киләчәкне алдан белеп булмый,
Кем гомере кайчан өзелер.
Исәнлектә күбрәк күрешәек,
Бер күрешү үзе бер гомер!

Оҗмах булулары дөрестер ул,
Бер киткәннәр кире кайтмыйлар.
"Ни хәл?" диеп барып сорап булмый,
Үзләре дә кайтып әйтмиләр!

Киләчәкне алдан белеп булмый,
Кем гомере кайчан өзелер.
Исәнлектә күбрәк күрешәек,
Бер күрешү үзе бер гомер!

Күпме яшәсәк тә гомерләрне,
Никтер язып булмый яңадан.
Язмышлар бит безнең маңгайларга
Әнкәй сөте белән язылган!

Киләчәкне алдан белеп булмый,
Кем гомере кайчан өзелер.
Исәнлектә күбрәк күрешәек,
Бер күрешү үзе бер гомер!
```

# Перевод

### Радик Муслухов / Ильназ Багиев - Одно усилие - целая жизнь

```
Даже если не ощутим все эти светлые миры,
Второй раз родиться уже невозможно.
Человек рождается один раз, живёт один раз,
Только один раз уходит из этого мира!

Заранее будущее узнать нельзя,
Чья жизнь когда прервётся.
Больше будем бороться пока мы живы,
Одно усилие - целая жизнь!

Правда то, что они становятся ангелами,
Ушедшие назад не возвращаются.
"Как дела?" - пойти и спросить невозможно,
Сами они не приходят рассказать!

Заранее будущее узнать нельзя,
Чья жизнь когда прервётся.
Больше будем бороться пока мы живы,
Одно усилие - целая жизнь!

Сколько бы мы ни жили эти жизни,
Почему-то прожить их заново нельзя.
Судьбы ведь на наши лбы
Маминим молоком записаны!

Заранее будущее узнать нельзя,
Чья жизнь когда прервётся.
Больше будем бороться пока мы живы,
```
