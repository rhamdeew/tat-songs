# Оригинал

### Рәмил Хаҗиев - Кил каршыма туган авылым

```
Эх, сагындым авылымны,
Сызылып аткан таңнарын,
Тугайларын, урманнарын,
Күңелле бала чагын.

Кил каршыма туган авылым
Өзелеп сине сагындым.

Араларда озын юллар,
Көннәр үтә сизелми.
Эх, сагындым әткәй-әнкәй,
Күрәсем килә сезне.

Кил каршыма туган авылым,
Өзелеп сине сагындым.

Кайтыр идем туган авылым
Яшьлегемне сагынып.
Күңелемдә тулы сагыш,
Очар идем кош булып.

Кил каршыма, туган авылым,
Өзелеп сине сагындым.
```

# Перевод

### Рамиль Хаҗиев - Встреть меня, родная деревня

```
Ах, скучаю по деревне,
По рассветам, что встают,
По лугам и по лесам,
По веселому детству.

Встреть меня, родная деревня,
Я до боли по тебе тоскую.

Между нами длинные дороги,
Дни проходят незаметно.
Ах, скучаю по отцу и матери,
Так хочу увидеть вас.

Встреть меня, родная деревня,
Я до боли по тебе тоскую.

Я бы вернулся в деревню,
Скучая по своей юности.
В сердце - полная тоска,
Я бы птицей к тебе прилетел.

Встреть меня, родная деревня,
Я до боли по тебе тоскую.
```
