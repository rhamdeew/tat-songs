# Оригинал

### Расим Низамов - Каеннар арасында

```
Ак каеннар арасына,
Ак каеннар янына,
Кереп киттен син йогереп,
Эй, кагына- кагына.

Сылу сынын, ак кулмэге,
Син аклыкка кумелден.
Кузлэремэ син узен дэ
Каен булып куренден.

Кайсы каенда, кайсы син,
Аералмый тик торам.
Ялгыз каенны кочканмын
Мин аны син дип торам.

Каеннар белэн шаярып,
Озаклап уйнадыгыз.
Ахрысы, мине ботенлэй
Курми дип уйладыгыз.

Сылу сынын, ак кулмэге,
Син аклыкка кумелден.
Кузлэремэ син узен дэ
Каен булып куренден.

Кайсы каенда, кайсы син,
Аералмый тик торам.
Ялгыз каенны кочканмын
Мин аны син дип торам.
```

# Перевод

### Расим Низамов - Среди берёз

```
Среди белых берёз,
Рядом с белыми берёзами,
Вбежала ты, торопясь,
Ах, шурша листвой.

Красивый стан, белое платье,
Ты в белизну облачилась.
В моих глазах ты сама
Берёзкой показалась.

Где берёза, где ты,
Не могу различить.
Одинокую берёзку обнимаю,
Принимая её за тебя.

С берёзками играя,
Долго забавлялась.
Наверное, совсем меня
Не видеть думала.

Красивый стан, белое платье,
Ты в белизну облачилась.
В моих глазах ты сама
Берёзкой показалась.

Где берёза, где ты,
Не могу различить.
Одинокую берёзку обнимаю,
Принимая её за тебя.
```
