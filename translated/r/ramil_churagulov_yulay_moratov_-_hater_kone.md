# Оригинал

### Рамил Чурагулов / Юлай Моратов - Хәтер көне

```
Бүген авыл-шәһәрләрдә
Ыгы-зыгы хөкем сөрә.
Барлык халык якыннарның
Зиратларын барып күрә.

Хәтер көне, матәм көне
Бүген шәһәр-авылларда.

Мең бәрәкәт исәннәргә.

Дога кылыйк, Корьән укыйк!
Кунак кына гомерләрдә,
Адәм булыйк, адәм булыйк!

Каберстан яннарыннан
Берүк, бәндәм, урап узма.
Каберләрнең хуҗалары
Өмет итә бездән дога.
Речитатив: Зират яны, хәтер йорты...
Ята монда чал гасырлар.
Истәлекләр, шәҗәрәләр,
Хәтер яңартып торалар.

Хәтер көне, матәм көне
Бүген шәһәр-авылларда.
Тыныч йокы үлгәннәргә,
Мең бәрәкәт исәннәргә.
Теләп изге дога кылыйк,
Дога кылыйк, Корьән укыйк!
Кунак кына гомерләрдә,
Адәм булыйк, адәм булыйк!

Һәркем бер көн монда килә,
Гомеркәйләр бик тиз үтә!

Теләп изге дога кылыйк,
Дога кылыйк, Корьән укыйк!
Кунак кына гомерләрдә,
Адәм булыйк, адәм булыйк!

Исәннәрнең кадерен белик,
Үлгәннәрнең каберен белик!
Хак сүзләрне - хакыйкатьне
Буыннарга ирештерик.

Хәтер көне, матәм көне
Бүген шәһәр-авылларда.
Тыныч йокы үлгәннәргә,
Мең бәрәкәт исәннәргә.
Теләп изге дога кылыйк,
Дога кылыйк, Корьән укыйк!
Кунак кына гомерләрдә,
Адәм булыйк, адәм булыйк!

Хәтер йорты һәйкәл кебек
Тора барсын белеп, күреп.
Дөнья фани икәнлеген
Һәр көн хәтергә төшереп.

Хәтер көне, матәм көне
Бүген шәһәр-авылларда.

Мең бәрәкәт исәннәргә.
Теләп изге дога кылыйк,
Дога кылыйк, Корьән укыйк!
Кунак кына гомерләрдә,
Адәм булыйк, адәм булыйк!

Зират яны, хәтер йорты,
Алларында үсә гөлләр.
Күзләр ачып йомган кебек,
Эх! кыска шул бу гомерләр!
```
