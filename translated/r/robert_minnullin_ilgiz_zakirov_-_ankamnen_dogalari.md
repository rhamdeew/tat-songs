# Оригинал

### Роберт Миңнуллин / Илгиз Закиров - Әнкәмнең догалары

```
Догаларның күңелемнән
Юк әле югалганы -
Саклый мине, яклый мине
Әнкәмнең догалары.

Ташлама, әнкәй, ташлама
Мине изге догаңнан.
Ташласаң изге догаңнан
Мин бәхетле булалмам.

Тылсымлы да, шифалы да
Әнкәмнең догалары -
Догаларның шатлык кына
Китерә торганнары.

Ташлама, әнкәй, ташлама
Мине изге догаңнан.
Ташласаң изге догаңнан
Мин бәхетле булалмам.

Кырыс ла тормыш юллары,
Авыр ла дөньялары!
Хәлләремне җиңеләйтә
Әнкәмнең догалары.

Ташлама, әнкәй, ташлама
Мине изге догаңнан.
Ташласаң изге догаңнан
Мин бәхетле булалмам.

Кабатлыйм мин дә әнкәйдән
Өйрәнгән догаларны.
Җиткерермен түкми-чәчми
Мин сезгә дә аларны.

Ташлама, әнкәй, ташлама
Мине изге догаңнан.
Ташласаң изге догаңнан
Мин бәхетле булалмам.
```

# Перевод

### Роберт Миннуллин / Ильгиз Закиров - Молитвы моей матери

```
Молитвы из души моей
Ещё не исчезали -
Хранят меня, оберегают
Моей матушки молитвы.

Не оставляй, мамочка, не оставляй
Меня без святых молитв своих.
Коль оставишь без святых молитв,
Я не смогу быть счастливым.

Чудесны и целебны
Моей матушки молитвы -
Молитвы те, что только радость
Приносят в этот мир.

Не оставляй, мамочка, не оставляй
Меня без святых молитв своих.
Коль оставишь без святых молитв,
Я не смогу быть счастливым.

Как жестоки жизни тропы,
Как тяжёл бывает мир!
Облегчают мне страданья
Моей матушки молитвы.

Не оставляй, мамочка, не оставляй
Меня без святых молитв своих.
Коль оставишь без святых молитв,
Я не смогу быть счастливым.

Повторяю я от мамы
Выученные молитвы.
Донесу я непременно
Их до вас в своих молитвах.

Не оставляй, мамочка, не оставляй
Меня без святых молитв своих.
Коль оставишь без святых молитв,
Я не смогу быть счастливым.
```
