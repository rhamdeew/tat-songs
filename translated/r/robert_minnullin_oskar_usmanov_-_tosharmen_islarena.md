# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Төшәрмен исләреңә

```
Керермен төшләреңә,
Төшәрмен исләреңә.
Минем генә исем китмәс
Синең ул хисләреңә.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.

Тарарсың чәчләреңне,
Көтерсең дәшкәнемне.
Минем кулъяулыгым белән
Сөртерсең яшьләреңне.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.

Килерсең яннарыма,
Басарсың алларыма.
Барыбер түзә алмассың
Йөрәгең янганына.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.
```

# Перевод

### Роберт Миңнуллин / Оскар Усманов - Паду в твои воспоминанья

```
Войду в твои сны,
Паду в твои воспоминанья.
Лишь мое имя не угаснет
В твоих тех чувствах.

Захочешь обнять,
Сгоришь ты в кострах.
Но рядом с тобою
Меня не будет тогда.

Распустишь ты волосы,
Будешь ждать моего зова.
Моим платком
Сотрешь ты слезы.

Захочешь обнять,
Сгоришь ты в кострах.
Но рядом с тобою
Меня не будет тогда.

Придешь ты ко мне,
Встанешь передо мной.
Все равно не сможешь вынести,
Что сердце в огне.

Захочешь обнять,
Сгоришь ты в кострах.
Но рядом с тобою
Меня не будет тогда.
```
