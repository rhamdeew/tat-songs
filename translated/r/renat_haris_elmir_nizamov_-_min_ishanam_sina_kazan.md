# Оригинал

### Ренат Харис / Эльмир Низамов - Мин ышанам сиңа, Казан!

```
Туган җирем елгасыннан алып,
Тутырырмын суның салкынын.
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.
Чишмәләрдә, кизләү, инешләрдә
Җырлап ага суның алкыны.
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.

Мин ышанам сиңа, серле Казан!
Мин ышанам сиңа, алтыным!
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.

Күзләремә алтын нурлар кайтты,
Күрәсезме, ничек балкыймын?!
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.

Мин ышанам сиңа, серле Казан!
Мин ышанам сиңа, алтыным!
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.
Мин ышанам сиңа, серле Казан!
Мин ышанам сиңа, алтыным!
Мин Казанга бирә алсам иде
Йөрәгемнең ярсу ялкынын.
Мин ышанам сиңа, Казан!..
```

Подари Казани мог бы я
Сиянье пламенного сердца.

Поет течение вод.

Сиянье пламенного сердца.
