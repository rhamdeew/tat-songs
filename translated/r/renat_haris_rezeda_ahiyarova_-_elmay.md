# Оригинал

### Ренат Харис / Резеда Ахиярова - Елмай

```
Уяндыңмы, елмай -
Көнне башла шулай.
Елмай үз-үзеңә,
Әйтәсе сүзеңә,
Тәрәзәдән кергән,
Өйне нурга төргән,
Һәрчак изге көнгә,
Сәлам биргән гөлгә.

Елмай, елмай, елмай,
Елмай, елмай, елмай.

Уяндыңмы, елмай -
Көнне башла шулай.
Елмай сәбәпсезгә -
Яхшы, әйбәт сүзгә.
Табарсың һәркемгә:
Бәхетлегә, кимгә,
Олыга, кечегә -
Җиде ят кешегә.

Елмай, елмай, елмай,
Елмай, елмай, елмай.

Уяндыңмы, елмай -
Көнне башла шулай.
Зилзиләләр тынар,
Чыксаң юлың уңар.
Авырлыклар арыр,
Башлаган эш барыр.
Бар дөньямны бизә
Карап минем күзгә.

Елмай, елмай, елмай,
Елмай, елмай, елмай.
```

# Перевод

### Ренат Харис / Резеда Ахиярова - Улыбайся

```
Проснулся - улыбнись,
Так день свой и начни.
Улыбнись себе,
Словам, что на душе,
Свету, что в окно вошел,
Дом сияньем обволок,
Каждому святому дню,
Цветку, что шлет привет.

Улыбайся, улыбайся, улыбайся,
Улыбайся, улыбайся, улыбайся.

Проснулся - улыбнись,
Так день свой и начни.
Улыбнись без повода -
Добрым, светлым словом.
Ты найдешь его для всех:
Для счастливых и для тех,
Для больших и для малых -
Для семи чужих людей.

Улыбайся, улыбайся, улыбайся,
Улыбайся, улыбайся, улыбайся.

Проснулся - улыбнись,
Так день свой и начни.
Утихнут лихолетья,
Выйдешь - путь твой светел.
Отойдут все тяжести,
Дело споро сложится.
Укрась мне весь мой мир,
Взгляни в мои глаза.

Улыбайся, улыбайся, улыбайся,
Улыбайся, улыбайся, улыбайся.
```
