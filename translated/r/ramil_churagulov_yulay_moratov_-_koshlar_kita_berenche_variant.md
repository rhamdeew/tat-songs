# Оригинал

### Рамил Чурагулов / Юлай Моратов - Кошлар китә (беренче вариант)

```
Туган җирен ташлап кошлар китә,
Һаваларда кала сагыш-эз.
Сагышларны җиңгән изге җаннар
Туган җирнең бетмәс яме сез.

Мин дә китәм туган җирдән,
Кайтырмын, дип кошлар белән.
Туган туфрак тарткан чакта
Канатларым талмас, беләм...
Мин дә китәм туган җирдән.

Туган җирен ташлап кошлар китә,
Кошлар китә ташлап оясын.
Мин дә китәм туган нигеземнән,
Соңгы китү генә булмасын.

Мин дә китәм туган җирдән,
Кайтырмын, дип кошлар белән.
Туган туфрак тарткан чакта
Канатларым талмас, беләм...
Мин дә китәм туган җирдән.

Туган җирен ташлап кошлар китә,
Кошлар белән җирдән китә моң.
Туган туфраклардан аерылу
Нигә шундый авыр, моңсу соң?

Мин дә китәм туган җирдән,
Кайтырмын, дип кошлар белән.
Туган туфрак тарткан чакта
Канатларым талмас, беләм...
Мин дә китәм туган җирдән.

Туган җирен ташлап кошлар китә,
Кыйгак-кыйгак, җирем, хуш, илем
Күпкә китмим, кайтыр өчен китәм
Туган җирем, сагынып көт мине.

Мин дә китәм туган җирдән,
Кайтырмын, дип кошлар белән.
Туган туфрак тарткан чакта
Канатларым талмас, беләм...
Мин дә китәм туган җирдән.
```

# Перевод

### Рамил Чурагулов / Юлай Моратов - Улетают птицы (первый вариант)

```
Покидая родной край, улетают птицы,
В небесах остаются тоска и следы.
Победив тоску, благородные души -
Вы - неиссякаемая прелесть родной земли.

Я тоже ухожу с родной земли,
С мыслью вернуться, вместе с птицами.
Когда родная земля меня притягивает -
Знаю, крылья мои не устанут...
Я тоже ухожу с родной земли.

Покидая родной край, улетают птицы,
Птицы улетают, покидая гнезда.
Я тоже покидаю родной очаг,
Пусть это не будет последним уходом.

Я тоже ухожу с родной земли,
С мыслью вернуться, вместе с птицами.
Когда родная земля меня притягивает -
Знаю, крылья мои не устанут...
Я тоже ухожу с родной земли.

Покидая родной край, улетают птицы,
Вместе с птицами улетает печаль.
Расставание с родной землёй
Почему如此 тяжело, грустно ведь?

Я тоже ухожу с родной земли,
С мыслью вернуться, вместе с птицами.
Когда родная земля меня притягивает -
Знаю, крылья мои не устанут...
Я тоже ухожу с родной земли.

Покидая родной край, улетают птицы,
Прощай, прощай, земля моя, край родимый,
Не навсегда ухожу - чтобы вернуться.
Родная земля, тоскуй и жди меня.

Я тоже ухожу с родной земли,
С мыслью вернуться, вместе с птицами.
Когда родная земля меня притягивает -
Знаю, крылья мои не устанут...
Я тоже ухожу с родной земли.
```
