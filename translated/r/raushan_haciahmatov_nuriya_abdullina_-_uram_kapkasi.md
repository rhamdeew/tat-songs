# Оригинал

### Раушан Хаҗиәхмәтов / Нурия Абдуллина - Урам капкасы

```
Салкын җилләр ачып яба
Урам капкасын.
Ачып яба, эленмәгән аның келәсе.
Балам кайтыр бүген диеп,
Бәлки кайтыр төнен, диеп,
Яна сизгер ана йөрәге.

Салкын җилләр шакымагыз,
Урам капкасын,
Борчымагыз әнкәемне, тыныч йокласын.
Балам хәлең ничек, диеп,
Кайталмадың кичен, тиеп,
Тынгы белми ана йөрәге.

Бүген нигә тынып калган
Урам капкасы?
Җил исмәгәч, чылтырламый аның келәсе.
Имин-аман бул син, балам,
Бәләләрдән сакла, Аллам,
Бирә күрмә бала кайгысын.
```

# Перевод

### Раушан Хаҗиәхмәтов / Нурия Абдуллина - Уличная калитка

```
Холодные ветры отворяют-затворяют
Уличную калитку.
Отворяют-затворяют, да не висит на ней замка.
Всё жду: сегодня сын вернётся,
А может, ночью он придёт,
Горит тревогой материнское сердечко.

Холодные ветры, не стучите
В уличную калитку,
Не тревожьте мою мать, пусть спит спокойно.
«Как ты, сынок?» - спросить,
Ты не смог прийти к вечеру,
И не найдёт покоя сердце матери.

Почему сегодня стихла
Уличная калитка?
Не дует ветер - не звенит её задвижка.
Будь жив-здоров, сыночек,
От бед храни, о Боже,
Не дай мне горя по ребёнку.
```
