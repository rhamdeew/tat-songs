# Оригинал

### Рагидә Янбулатова / Таһир Кәримов - Истәлек (беренче вариант)

```
Таң атканда, саубуллашкан чакта,
Кулларымны куям кулыңа.
Хәтерлә син татлы минутларны,
Шул хисләрне ал син юлыңа.

Моңсу истәлекләр булса әгәр,
Үзең белән юлга алма син.
Якты шатлык булсын йөрәгеңдә,
Сагышлы моң юлдаш булмасын.

Очрашмабыз, дигән шомлы уйлар
Килмәсеннәр синең башыңа.
Син кайтканда алсу чәчәк тотып,
Мин чыгармын синең каршыңа.
```

# Перевод

### Рагидә Янбулатова / Таһир Кәримов - Воспоминание (первый вариант)

```
На заре, в минуту расставанья,
Руки кладу в руки твои.
Сбереги ты сладкие мгновенья,
Эти чувства в путь свой забери.

Если грусть-воспоминанье встретишь,
Не бери с собой его в дорогу.
Пусть в душе сияет светлой радость,
А печальный напев - не спутник.

Пусть тревожной мысли о разлуке
Не явиться в голову твою.
Ты вернёшься - с алым цветком в руках,
Я выйду навстречу тебе.
```
