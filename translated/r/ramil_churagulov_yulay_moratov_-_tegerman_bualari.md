# Оригинал

### Рамил Чурагулов / Юлай Моратов - Тегермән буалары

```
Алтын яллы толпар атларыбыз
Алып китте безне бала чактан.
Төшләремдә моңсу гына кешни
Тәүге тапкыр меңгән җитез атлар.

Сагындыра бала чакның
Юалы урманнары.
Җәйрәп яткан ак дулкынлы,
Тегермән буалары.

Яшьлегемнең тегермән буалары,
Дулкын булып, йөрәкләргә кага.
Ярларында яшьлек ярларыбыз
Яулык болгап, безгә һаман бага.

Сагындыра бала чакның
Юалы урманнары.
Җәйрәп яткан ак дулкынлы,
Тегермән буалары.

Сагындыра яшьлегемнең
Юалы урманнары.
Җәйрәп яткан, ак дулкынлы,
Тегермән буалары.
```

# Перевод

### Рамил Чурагулов / Юлай Моратов - Плотины мельниц

```
Златогривые кони-скакуны
Увели нас прочь из детских лет.
В снах моих печально тихо ржут
Те резвые, что в первый раз оседланы.

Скучаю по детства
Тихим, дремным лесам.
Ширью белых волн разлитым -
Плотинам мельниц.

Плотины мельниц юности моей
Бьются волной и стучат в сердца.
На их берегах - берега нашей юности,
Махая платком, всё смотрят на нас.

Скучаю по детства
Тихим, дремным лесам.
Ширью белых волн разлитым -
Плотинам мельниц.

Скучаю по юности
Тихим, дремным лесам.
Ширью белых волн разлитым -
Плотинам мельниц.
```
