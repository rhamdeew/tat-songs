# Оригинал

### Рәис Гыймадиев / Виталий Агапов - Көтәм!

```
Сине көтә күңелкәем,
Күптәннән юк килгәнең.
Елына бер чәчәк ата
Минем кура җиләгем.

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоек чәчәгендә.

Минем әле һәрнәрсәгә
Куанып яшәр чагым.
Бар куанычым синдә бит,
Картайма инде, җаным!

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоок чәчәгендә.

Бәлки, сөю бакчабызда
Мин гади бер чәчәктер,
Синдәй мәхәббәт кошлары
Бу дөньяда сирәктер.

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоек чәчәгендә.
```

# Перевод

### Рәис Гыймадиев / Виталий Агапов - Жду!

```
Ждет тебя душа родная,
Давно ты не приходил.
Раз в году цветет отрадой
Мой малинник у края сил.

С первыми рассветами
Чувства в сердце рождаются.
Соловей мой, ты вчера пел
На цветке кувшинки.

Мне еще пора такая -
Радость жить и брать взахлеб.
Вся моя отрада в тебе ведь,
Не старей, любимый мой!

С первыми рассветами
Чувства в сердце рождаются.
Соловей мой, ты вчера пел
На цветке кувшинки.

Может, в нашем саде любви
Я всего лишь простой цветок,
Но такие птицы любви, как ты,
В этом мире редкий итог.

С первыми рассветами
Чувства в сердце рождаются.
Соловей мой, ты вчера пел
На цветке кувшинки.
```
