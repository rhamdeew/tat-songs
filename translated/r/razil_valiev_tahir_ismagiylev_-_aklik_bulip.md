# Оригинал

### Разил Вәлиев / Заһир Исмәгыйлев - Аклык булып

```
Тагын бер ел иңләп узды -
Язмышым урамнарын.
Кайгыларны адаштыра
Яңа ел бураннары.

Тагын бер ел шаулап узды,
Туздырып ап-ак карны.
Ак кар булып уйнап оча
Яңа ел шатлыклары.

Тагын бер ел узды җирдән,
Тик хәтер булып калды.
Аклык булып җиргә ята
Яңа ел болытлары.

Тагын бер ел гөрләп узды -
Сиздем гомер узганын.
Бәхет кебек балкый күктә
Яңа ел йолдызлары.
```

# Перевод

### Разил Валиев / Тагир Исмагилов - Белым снегом

```
Прошёл ещё год, неслышно ступая,
По улицам моей судьбы.
Печали разбрасывает, теряет
Новогодние бури.

Прошёл ещё год, шумя и метя,
Вьюга пороша запорошила.
Белым снегом играя летит
Новогоднее радость.

Прошёл ещё год по земле,
Остался лишь памятью в сердцах.
Белой бахромой лежит на земле
Новогодние тучи.

Прошёл ещё год, грохоча,-
Почувствовал, что жизнь пролетела.
Как счастье сияют в небе
Новогодние звёзды.
```
