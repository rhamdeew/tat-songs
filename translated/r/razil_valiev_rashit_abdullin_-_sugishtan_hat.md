# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Сугыштан хат

```
Әнкәем, хат язам:
Ничек соң хәлләрең?
Юрама яманга
Килмәсә хәбәрем.

Әткәем, хат язам:
Юк башым игәнем,
Һәрвакыт хәтердә
«Сынатма!» дигәнең.

Сөйгәнем, хат язам
Сугышның үзеннән.
Тик шәүләм сүнмәсен,
Китмәсен күзеңнән.
```

# Перевод

### Разиль Валеев / Рәшит Абдуллин - Письмо с войны

```
Матушка, пишу письмо:
Как твои дела?
Не думай о дурном,
Если нет вестя.

Батюшка, пишу письмо:
Голову не гну,
В памяти всегда живут
Слова твои: «Не слабею».

Милая, пишу письмо
Прямо из огня войны.
Лишь бы мой след не гас,
Не исчезал из твоей груди.
```
