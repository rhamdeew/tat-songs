# Оригинал

### Ризида Мөхәммәтҗанова - Яралы күбәләк

```
Китәр идем ерак-ерак,
Ал язмыш таңнарына.
Яралы күбәләк булып
Очам гел яннарыңда.

Кушымта:
Яшәвемнең яме микән
Хәсрәт-сагышларым да?
Яралы күбәләк булдым
Синең язмышларыңда.

Энҗе бөртекләре тама
Гөл-чәчкә таҗларына.
Язмышым илтә хисләрне
Җәһәннәм сазларына.

Кушымта.

Иртә таңда кайнар энҗе
Чыкның тамчыларында.
Җәрәхәткә тозлар сибеп,
Җанны камчылый гына.

Кушымта.
```

# Перевод

### Ризида Мөхәммәтҗанова - Раненая бабочка

```
Ушла бы я далеко-далеко,
К рассветам алой судьбы.
Раненой бабочкой снова
Лечу всё рядом с тобой.

Припев:
Неужто радость жизни -
Мои печали и тоска?
Раненой бабочкой стала
В твоей судьбе навсегда.

Капли жемчужные падают
На лепестки у цветов.
Судьба уносит чувства
В болота адских миров.

Припев.

На ранней заре горячий жемчуг -
В каждой росинке слеза.
Солью посыплет раны
И только душу хлестнёт.

Припев.
```
