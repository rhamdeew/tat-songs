# Оригинал

### Рахат Лукум - Рэхэт Лукум

```
Уйларымда йортэм,
Котэм сине, котэм.
Эшлэремдэ курэм,
Котэм сине, котэм.
Гашыйк булдым куптэн,
Котэм сине, котэм.
Кирэк тугел бутэн,
Котэм сине, котэм.

Остэлемдэ рэхэт лукум
Синен очен, синен очен.
Йорэгемнен ин турендэ
Синен исем, синен исем.
Минем кебек тоеп кара
Сою хисен, сою хисен.
Рэхэт лукум бит мин сина...

Кузлэремэ карап,
Ярат мине, ярат.
Чэчлэремне тарап,
Ярат мине, ярат.
Хэркон кабат, кабат
Ярат мине, ярат.
Инемдэ бул канат,
Ярат мине, ярат.

Остэлемдэ рэхэт лукум
Синен очен, синен очен.
Йорэгемнен ин турендэ
Синен исем, синен исем.
Минем кебек тоеп кара
Сою хисен, сою хисен.
Рэхэт лукум бит мин сина...
```

# Перевод

### Рахат Лукум - Рахат-лукум

```
В мыслях дома живу,
Жду тебя, я жду.
В делах вижу тебя,
Жду тебя, я жду.
Влюблён давно уже,
Жду тебя, я жду.
Никого не нужно,
Жду тебя, я жду.

На столе рахат-лукум
Для тебя, для тебя.
В глубине моего сердца
Твоё имя, твоё имя.
Почувствуй, как и я,
Чувство любви, чувство любви.
Рахат-лукум ведь я для тебя...

В глаза мои смотря,
Люби меня, люби.
По волосам гладя,
Люби меня, люби.
Каждый день снова, снова
Люби меня, люби.
Будь крыльями души,
Люби меня, люби.

На столе рахат-лукум
Для тебя, для тебя.
В глубине моего сердца
Твоё имя, твоё имя.
Почувствуй, как и я,
Чувство любви, чувство любви.
Рахат-лукум ведь я для тебя...
```
