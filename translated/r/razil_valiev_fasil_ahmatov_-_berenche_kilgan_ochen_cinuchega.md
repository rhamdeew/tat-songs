# Оригинал

### Разил Вәлиев / Фасил Әхмәтов - Беренче килгән өчен (Җиңүчегә)

```
Җилне узып чаба егет,
Дулкынлана кыр өсте.
Үзе үстергән игеннәр
Аңа булыша төсле.

Сабантуйларда сынала
Батыр егетләр көче.
Колга башындагы сөлге -
Беренче килгән егет өчен!

Тоягың җиргә тимәсен,
Җилдер тизрәк, әй юрга!
Авыл башында егетне
Бар халык көтеп тора.

Сабантуйларда сынала
Батыр егетләр көче.
Былбыл телле тальян гармун -
Беренче килгән егет өчен!

Ат тупырдавын уздырып,
Егет йөрәге кага.
Кайсы кыз гына сокланып
Карамый бүген аңа!

Сабантуйларда сынала
Батыр егетләр көче.
Иң сылу кызның йөрәге -
Беренче килгән егет өчен!
```

# Перевод

### Разил Валиев / Фасиль Ахметов - Для первого пришедшего (Победителю)

```
Мчится парень, опережая ветер,
Волнами идёт по полю.
Хлеб, что он растил сам,
Будто помогает ему.

На Сабантуе проверяется
Сила смельчаков-парней.
Полотенце в руках -
Тому, кто пришёл первым!

Пусть копыто не касается земли,
Гони быстрее, мой конь!
У начала деревни парня
Весь народ с нетерпением ждёт.

На Сабантуе проверяется
Сила смельчаков-парней.
Соловьём поющая тальянка -
Тому, кто пришёл первым!

Громко цокая копытами,
Сердце парня стучит.
Какая же девушка сегодня
Не смотрит на него с восхищением!

На Сабантуе проверяется
Сила смельчаков-парней.
Сердце самой красивой девушки -
Тому, кто пришёл первым!
```
