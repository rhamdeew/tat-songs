# Оригинал

### Рамил Чурагулов / Азат Имаев - Сабантуй (икенче вариант)

```
Исән-аман кышлап чыкты
Мал-туар йорт-курада.
Ашлык, икмәк күпереп үсәр
Беркөн киң басуларда.

Батырга да бер ял кирәк -
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм - сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!

Мәйданнарда көч сынаша
Сабантуй батырлары.
Чын егет сайлый кәләшкә
Кызларның асылларын.

Батырга да бер ял кирәк -
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм - сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!

Сабантуйларын гөрләтеп
Яңгырасын матур җыр.
Имин торсыннар дөньялар!
Икмәк булса, җыр булыр!

Батырга да бер ял кирәк -
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм - сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!
```

# Перевод

### Рамил Чурагулов / Азат Имаев - Сабантуй (второй вариант)

```
Пережили зиму в мире, в лад,
Скот и птица - в каждом дворе.
Зерна, хлеб развернутся в рост
На широких полях в поре.

И герою нужен отдых -
Тысячу забот отложи.
С песней, с танцем, шумно, звонко
Пусть пройдет наш праздник - Сабантуй!
Эх! Сабантуй, Сабантуй,
Самый светлый, самый шустрый той!

На майданах сила мерится,
Там - батыры Сабантуя.
Истинный джигит в невесты
Выбирает лучших из девчат.

И герою нужен отдых -
Тысячу забот отложи.
С песней, с танцем, шумно, звонко
Пусть пройдет наш праздник - Сабантуй!
Эх! Сабантуй, Сабантуй,
Самый светлый, самый шустрый той!

Пусть Сабантуи гремят повсюду,
Пусть звучит красивая песня.
Пусть хранятся миры и страны!
Есть хлеб - будет песня!

И герою нужен отдых -
Тысячу забот отложи.
С песней, с танцем, шумно, звонко
Пусть пройдет наш праздник - Сабантуй!
Эх! Сабантуй, Сабантуй,
Самый светлый, самый шустрый той!
```
