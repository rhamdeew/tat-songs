# Оригинал

### Рәми Гарипов / Нур Даутов - Мин яратам гашыйкларны

```
Мин яратам гашикларны,
Гашыйклар матур була.
Назлап сөйгән кыз алдында
Егет тә батыр була.

Тик кайчандыр таныш хисләр
Йөрәгемә ят кебек:
Хәзер инде яшь йөрәгем
Авызлыклы ат кебек!..
Мин яратам гашыйкларны!
```

# Перевод

### Рями Гарипов / Нур Даутов - Я люблю влюблённых

```
Я люблю влюблённых,
Влюблённые прекрасны.
Перед любимой девушкой, ласково любимой,
И парень смелым становится.

Но когда-то знакомые чувства
В моём сердце словно чужие:
Теперь моё молодое сердце
Как осёдланный конь!..
Я люблю влюблённых!
```
