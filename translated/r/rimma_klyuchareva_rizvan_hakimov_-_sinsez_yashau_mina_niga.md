# Оригинал

### Римма Ключарева / Ризван Хәкимов - Синсез яшәү миңа нигә?

```
Ялгызыма чәчкә язлар
Калдырдың да киттең нигә?
Алданыр да чәчкәләргә,
Сагынмас дидең мәллә?

Синсез язлар миңа нигә?
Синсез язлар миңа нигә?
Сагынуларым сагышларга
Әйләнә дә кабат килә.

Кәккүк саный әкрен генә,
Көннәремне бәйләп төнгә.
Йөрәк елый өнсез генә,
Җәйләр килгәч синсез генә.

Синсез җәйләр миңа нигә?
Синсез җәйләр миңа нигә?
Сагынуларым сагышларга
Әйләнә дә кабат килә.

Дөньяларым ялгын алган,
Сагышларда янсам, янам.
Синсез генә сине көтеп,
Мин көзләрдә инде калам.

Синсез яшәү миңа нигә?
Синсез яшәү миңа нигә?
Сагынуларым сагышларга
Әйләнә дә кабат килә.
```

# Перевод

```
Весны с цветами мне одинокой
Оставил и ушёл, зачем?
Обманут будут, словно цветы,
Сказал, что скучать не буду ведь?

Зачем мне весны без тебя?
Зачем мне весны без тебя?
Тоска по тебе превращается
В печаль и приходит вновь.

Кукушка считает едва слышно,
Дни мои связывая в ночь.
Сердце плачет беззвучно,
Когда лето придёт без тебя.

Зачем мне лето без тебя?
Зачем мне лето без тебя?
Тоска по тебе превращается
В печаль и приходит вновь.

Миры мои огнем охвачены,
В тоске горю, всё в огне.
Лишь ожидая тебя, без тебя,
Я в осенях теперь останусь.

Зачем мне жизнь без тебя?
Зачем мне жизнь без тебя?
Тоска по тебе превращается
В печаль и приходит вновь.
```
