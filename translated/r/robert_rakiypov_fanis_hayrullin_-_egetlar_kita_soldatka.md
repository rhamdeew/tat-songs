# Оригинал

### Роберт Рәкыйпов / Фәнис Хәйруллин - Егетләр китә солдатка

```
Тальян чыңына кушылып
Сибелә дәртле җырлар,
Егетләр китә солдатка-
Озата бара кызлар.

Озата бара зур юлга
Дуслары, танышлары.
Күкләрдә киек казларның
Хушлашу тавышлары...

Ак кулъяулыклар җилферди
Йөгәннәр, аелларда.
Әниләр моңсулыгы бар
Юлдагы каеннарда.

Дугаларда кыңгыраулар -
Бииләр генә атлар,
Ике атлап, бер йөгереп
Малайлар калмый арттан.

Егетләр китә солдатка -
Кемдә юк кызыгулар!
Тынычлык белән кайтыгыз -
Ясарбыз кызыл туйлар...
```

# Перевод

### Роберт Рәкыйпов / Фәнис Хәйруллин - Парни уходят в солдаты

```
Под звон тальяна, в лад
Сыплются песни резвые,
Парни уходят в солдаты -
Девушки провожают.

На дальний путь провожают
Друзья и знакомые.
В небе прощально кричат
Дикие гуси сизые...

Белые платки трепещут
На уздечках, в повозках.
Материнская грусть -
В березах вдоль дороги.

На дугах звенят колокольцы -
Лишь пляшут кони лихие,
Вдвое шагают, вперегон
Мальчишки не отстают.

Парни уходят в солдаты -
Кого не тронет это!
Возвращайтесь с миром -
Сыграем красные свадьбы...
```
