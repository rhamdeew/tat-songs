# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Берләшик!

```
Ызгыштан-талаштан
Арыган Җир-ана.
Канкоеш, сугыштан
Ул авыр тын ала.

Без - Адәм уллары,
Без - Һава кызлары,
Серләрне бүлешик,
Тату һәм дус яшик!
Кулларны бирешик,
Сөйләшик, күрешик!
Чын кеше исемен
Без горур йөртик.

Илләр күп бит алар,
Җир-ана бер генә.
Ул бик зур булса да,
Ул бит бик кечкенә.

Без - Адәм уллары,
Без - Һава кызлары,
Серләрне бүлешик,
Тату һәм дус яшик!
Кулларны бирешик,
Сөйләшик, күрешик!
Чын кеше исемен
Без горур йөртик.

Берләшик, берләшик!
Аерылмыйк берьюкка.
Ни җитә татулык,
Ни җитә дуслыкка!

Без - Адәм уллары,
Без - Һава кызлары,
Серләрне бүлешик,
Тату һәм дус яшик!
Кулларны бирешик,
Сөйләшик, күрешик!
Чын кеше исемен
Без горур йөртик.
```

Ей тяжело дышать.

Мы - сыны Адама,
Мы - дочери Евы,
Секреты поделим,

Поговорим, встретимся!
Имя Человека
Мы гордо несём.

Стран на свете много,
Мать-земля одна.
Пусть она огромна -
Всё же так мала.

Объединимся, объединимся!

Что сравнится с согласием,
Что сравнится с дружбой!
