# Оригинал

### Резеда Вәлиева / Мәсгудә Шәмсетдинова - Кышкы ямь

```
Тәрәзәләр көмеш сүрәт,
Көмеш кенә урамнар,
Бар агачлар, алмагачлар
Көмеш кебек булганнар.

Зəңгəр күктəн җем-җем итеп
Көмеш йолдызлар ява,
Көмеш йолдызлар чыңыннан
Зыңлап торгандай һава.

Əйтерсең лə җир йөзенə
Көмеш нурлар таратып
Гаҗəеп бер аккош йөзə
Җилпеп ап-ак канатын.
```

# Перевод

### Резеда Вәлиева / Масгуда Шамсетдинова - Зимняя краса

```
Окна в серебряных рамках,
Серебряные улицы,
Все деревья, яблони
Стали словно серебром.

С голубого неба, звонко падая,
Сыплются серебряные звёзды,
От звёздного звона
Воздух звенит, кажется.

Будто по земле,
Разбрасывая серебряные лучи,
Пролетает удивительный белый лебедь,
Оставляя белоснежное крыло.
```
