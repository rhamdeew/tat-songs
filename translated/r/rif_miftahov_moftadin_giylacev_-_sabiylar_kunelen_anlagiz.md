# Оригинал

### Риф Мифтахов / Мөфтәдин Гыйләҗев - Сабыйлар күңелен аңлагыз

```
Кояшка омтылган гөлләрдәй,
Сабыйлар кочакка үрелә,
Әтисе, әнисе янында
Калырга гомергә гел бергә.

Әтисе, әнисе булганда
Бер ни дә кирәкми сабыйга.
Алданып вакытлы язмышка
Салмагыз баланы кайгыга.

Баланы ятларга калдырып,
Ятимлек утына салмагыз.
Өзелеп яраткан балага
Чит кешеләр булып калмагыз.
```

# Перевод

### Риф Мифтахов / Мөфтәдин Гыйләҗев - Поймите душу малышей

```
Как цветы, что к солнцу тянутся,
Малыши к объятьям простираются,
Им рядом с папой, с мамой
Быть всегда, всю жизнь, едиными.

Если рядом папа и мама -
Больше ничего не нужно малышу.
Не обманом мимолётной доли
Не ввергайте детство в грусть.

Не оставляйте ребёнка чужим,
Не бросайте в пламя сиротства.
Для ребёнка, что любит беззаветно,
Не становитесь посторонними.
```
