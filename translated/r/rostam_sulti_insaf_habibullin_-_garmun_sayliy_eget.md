# Оригинал

### Рөстәм Сүлти / Инсаф Хәбибуллин - Гармун сайлый егет

```
Бармаклары белән гүя,
Гармун телләрен саный.
Берсен ала, берсен куя,
Бер егет гармун сайлый.

Гармун сайрый сандугачтай,
Гармун сайрый да сайрый.
Көйгә көйне ялгый-ялгый,
Бер егет гармун сайлый.

Уйнамый түзеп буламы,
- Кулда гармун булганда?!
Тын гына торып кара син
Җаның моңга тулганда.

Гармун сайрый сандугачтай,
Гармун сайрый да сайрый.
Көйгә көйне ялгый-ялгый,
Бер егет гармун сайлый.

Кызлар күңелен җилкетер
Гармуннар бармы монда?
Тукта-тукта, нинди көй бу?
Йөрәкне урый моңга!

Гармун сайрый сандугачтай,
Гармун сайрый да сайрый.
Көйгә көйне ялгый-ялгый,
Бер егет гармун сайлый.
```

# Перевод

### Ростэм Сюлти / Инсаф Хабибуллин - Парень играет на гармошке

```
Пальцами своими, будто,
Перебирает голоса гармошки.
Тот берёт, тот кладёт,
Парень на гармошке играет.

Гармошка поёт, как соловей,
Гармошка поёт и поёт.
Мелодию к мелодии сплетает-сплетает,
Парень на гармошке играет.

Удержаться можно ли,
- Когда гармошка в руках?!
Постой-ка, посмотри ты,
Когда душа песней полнится.

Гармошка поёт, как соловей,
Гармошка поёт и поёт.
Мелодию к мелодии сплетает-сплетает,
Парень на гармошке играет.

Сердца девичьи тронуть
Есть ли тут гармошки?
То-то замирает, что за мелодия?
В сердце ударяет песня!

Гармошка поёт, как соловей,
Гармошка поёт и поёт.
Мелодию к мелодии сплетает-сплетает,
Парень на гармошке играет.
```
