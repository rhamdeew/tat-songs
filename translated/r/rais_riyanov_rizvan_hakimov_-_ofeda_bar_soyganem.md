# Оригинал

### Рәис Риянов / Ризван Хәкимов - Өфедә бар сөйгәнем

```
Өфе юллары карурман,
Өфе юллары озын.
Шунда сөйдем ил гүзәлен -
Урал таулары кызын.

Төшеп Йөрүзән буена,
Җиләген кабалмадым.
Өфе юлына чыкмыйча,
Һич тынгы табалмадым.

Зәңгәрле күлмәк кигәнем -
Өфедә бар сөйгәнем.
«Уф!..» дигәннәрне бетерә
Бары Өфе дигәнем.

Өфе кичләрен сихерли
Айның балкып чыгышы...
Җаныңны назлап эретә
Саф мәхәббәт сулышы.
```

# Перевод

### Раис Риянов / Ризван Хәкимов - Любимая в Уфе

```
Дороги Уфы необычны,
Дороги Уфы длинны.
Там любил я красивую -
Дочь Уральских гор.

Падаю от любви к тебе,
Не могу сдержать чувства.
Не выходя на уфимскую дорогу,
Ни одного отдыха не нашел.

Надел синий колпак -
Любимая в Уфе.
Говорю "Уф!.." вместо слов
И только "Уфа" говорю.

Волшебные вечера Уфы
Выход луны, блестящей...
Твоя душа, нежно греет
Чистое дыхание любви.
```
