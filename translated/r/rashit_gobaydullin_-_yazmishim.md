# Оригинал

### Рәшит Гобәйдуллин - Язмышым

```
Язмышым, кайтар мине урамыма.
Анда калды сабый чакларым.
Анда - гамьсез шатлык белән узган
Әй, бала чагым, хәсрәтсез чагым.

Язмышым, кайтар мине урамыма.
Төшер исемә гүзәл таңнарын.
Буйдан-буйга узган бакчаларын,
Кабан күлен, Бакалтайларын.

Язмышым, кайтар мине урамыма,
Мин бит аның кочагында үстем.
Аермады дусларымнан, рәхмәт сиңа,
Гомер юлымны алар белен кичтем.
```

# Перевод

### Рашит Гобайдуллин - Судьба моя

```
Судьба моя, верни меня на мою улицу.
Там остались мои детские годы.
Там - беззаботной радостью прошедшие
Ах, детство моё, время без горестей.

Судьба моя, верни меня на мою улицу.
Опусти на меня свои прекрасные рассветы.
Сады, протянувшиеся от края до края,
Озеро Кабан, Бакалтаи.

Судьба моя, верни меня на мою улицу,
Ведь я вырос в её объятиях.
Не разлучался с друзьями, спасибо тебе,
По жизненному пути прошёл вместе с ними.
```
