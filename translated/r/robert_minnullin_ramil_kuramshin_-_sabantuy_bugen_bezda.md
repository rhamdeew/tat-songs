# Оригинал

### Роберт Миңнуллин / Рамил Курамшин - Сабантуй бүген бездә!

```
Шатлыклар күңелебездә,
Ал кояш күгебездә!
Килегез, дуслар, кунакка -
Сабантуй бүген бездә!

Иң уртага утыртканнар
Иң озын бер колганы -
Колганың да бик тиз генә
Менеп булмый торганы.

Җиңгиләр бире җибәргән
Бизәкле бүләкләрен,
Ә малайлар көрәшергә
Сызганган беләкләрен.

Ә кызлар киеп чыкканнар
Иң матур күлмәкләрен.
Латыйп бабай төяп килгән
Иң ныклы чүлмәкләрен.

Абыйлар алып килгәннәр
Иң җитез юртакларын.
Гармунчылар алып чыккан
Җырларның уртакларын.

Җилферди чиккән сөлгеләр!
Бүләкләр - күбебезгә!
Көрәшик тә, узышыйк та -
Сабантуй бүген бездә!
```

# Перевод

### Роберт Миннуллин / Рамил Курамшин - Сабантуй сегодня у нас!

```
Радость в наших сердцах,
Яркое солнце в небе!
Приходите, друзья, в гости -
Сабантуй сегодня у нас!

Посредине установили
Самый длинный шест -
Но на шест-то очень быстро
Нельзя взобраться, увы.

Молодые жены поднесли
Свои украшенные дары,
А парни для борьбы
Натянули свои бицепсы.

А девушки надели
Самые красивые платья.
Дед Латип привёз
Самые крепкие горшки.

Братья привели
Самых резвых коней.
Гармонисты вывели
Общие песни для всех.

Развеваются шарфики!
Подарки - многим из нас!
Будем бороться, соревноваться -
Сабантуй сегодня у нас!
```
