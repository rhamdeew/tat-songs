# Оригинал

### Рәүф Сәлтәшев - Яшьләр туе

```
Яз исенә тулган авыл.
Яшь йөрәкләр ярәшә.
Кәләшнең ак күлмәгеннән
Умырзая көнләшә.

Кыңгырау чыңлавы белән
Авыл урамы тулы.
Яшь йөрәкләр ярәшәләр -
Авылда яшьләр туе.

Тулпар атлар җилеп уза
Авылым урамыннан.
Яшьләр туе - табигать туе
Зәңгәр яз моңнарыннан.

Кыңгырау чыңлавы белән
Авыл урамы тулы.
Яшь йөрәкләр ярәшәләр -
Бәхетләр булсын тулы.

Яшьләр туе. Барсы бии,
Кияве, кәләшләре...
Гомер булсын, булсын туйлар!
Алтыны, көмешләре.
```

# Перевод

### Рәүф Сәлтәшев - Свадьба ребят

```
Весенним духом дышит деревня.
Сердца молодых обручаются.
Белое платье невесты
Ревниво влечет подснежник.

Звоном колокольчиков полна
Вся деревенская улица.
Сердца молодых обручаются -
В деревне свадьба ребят.

Крылатые кони несутся
По улицам моего села.
Свадьба ребят - природный праздник,
Напетый голубой весной.

Звоном колокольчиков полна
Вся деревенская улица.
Сердца молодых обручаются -
Пусть счастья будет сполна.

Свадьба ребят. Все кружатся,
Женихи и невесты...
Пусть будет жизнь, пусть будут свадьбы,
С золотом и серебром.
```
