# Оригинал

### rahimcan_sharifcanov_rostam_hayrullin_ - Чаллыда туган диләр

```
Күзләреңдә зәңгәр очкын,
Кулларың уңган диләр.
Әллә белеп, әллә белми -
Чаллыда туган диләр.

Чаллы диләр, Чаллы диләр,
Гигант төзелеш диләр.
Туган җирнең матурлыгы,
Җирнең бизәге диләр.

Күкрәп үскән урманнарын
Гөлбакчага тиңлиләр.
Күңелдәге серләремне
Дуслар әле белмиләр.

Чаллы диләр, Чаллы диләр,
Гигант төзелеш диләр.
Туган җирнең матурлыгы,
Җирнең бизәге диләр.

Кояш Кама суларына
Төшкән дә кунган диләр.
Мактансам да хакым бар шул -
Чаллыда туган диләр.

Чаллы диләр, Чаллы диләр,
Гигант төзелеш диләр.
Туган җирнең матурлыгы,
Җирнең бизәге диләр.
```

### Рәхимҗан Шәрифҗанов / Рөстәм Хәйруллин - Говорят, родился в Челнах

```
В глазах твоих - лазурный искр,
А руки - мастера, твердят.
То ли с умыслом, то ли без -
Говорят: в Челнах роднят.

Челны твердят, Челны твердят -
Гигант-стройкой их зовут.
Красота земли родной -
Украшенье всей земли вокруг.

Шумят, подросшие, леса -
Им сад цветущий по плечу.
Тайны сердца моего
Друзья пока что не прочтут.

Челны твердят, Челны твердят -
Гигант-стройкой их зовут.
Красота земли родной -
Украшенье всей земли вокруг.

Солнце в Камы воды, мол,
Упало - и осталось тут.
И мне хвалиться есть за что -
Говорят: в Челнах роднут.

Челны твердят, Челны твердят -
Гигант-стройкой их зовут.
Красота земли родной -
Украшенье всей земли вокруг.
```
