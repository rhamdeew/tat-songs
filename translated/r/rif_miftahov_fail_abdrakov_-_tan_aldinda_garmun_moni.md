# Оригинал

### Риф Мифтахов / Фаил Абдраков - Таң алдында гармун моңы

```
Таң алдында гармун моңы...
Хыялый итәр кебек.
Үтеп барган яшьлегемә
Кабаттан илтәр кебек.

Таң алдында гармун моңы...
Йөрәгем эрер кебек.
Миңа язмаган ярымны
Алыр да килер кебек.

Таң алдында гармун моңы...
Йолдызлар тыңлый кебек.
Ә ай, бахыр, мине аңлап,
Сыгылып елый кебек...
```

# Перевод

### Риф Мифтахов / Фаил Абдраков - Мелодия гармони перед рассветом

```
Перед рассветом стонет гармонь...
Словно в грёзы манит вновь.
К юности моей прошедшей
Увести готова вновь.

Перед рассветом стонет гармонь...
Словно сердце тает вдруг.
Будто суженый мне, не писан,
Постучится в дом мой вдруг.

Перед рассветом стонет гармонь...
Словно звёзды внемлют ей.
А луна, бедняжка, понимая,
Сгибаясь, плачет по мне...
```
