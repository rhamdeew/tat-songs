# Оригинал

### Радик Хәкимҗанов / Флүрә Талипова - Аккошлар кайтты, аккошлар!

```
Аккошлар кайтты, аккошлар,
Илгә гүзәллек кайтты.
Куанычлы хәбәрләрдән,
Таннар да балкып атты.

Аккошлар кайтты, аккошлар,
Назга күмелде җаннар.
Язлар безне очраштырды,
Булыйк мәңгелек парлар.

Аккошлар кайтты, аккошлар,
Илгә ак бәхет кайтты.
Күнелләрдә сафлык артты,
Дөньялар нинди якты.

Аккошлар кайтты, аккошлар,
Назга күмелде җаннар.
Язлар безне очраштырды,
Булыйк мәңгелек парлар.

Аккошлар кайтты, аккошлар,
Ямь өстәлде күлләргә.
Бәхет-шатлыкның олысы,
Яшәү туган җирләрдә.

Аккошлар кайтты, аккошлар,
Назга күмелде җаннар.
Язлар безне очраштырды,
Булыйк мәңгелек парлар.
```

# Перевод

### Радик Хәкимҗанов / Флүрә Талипова - Вернулись лебеди, лебеди!

```
Вернулись лебеди, лебеди -
В край наш вернулась краса.
От радостных вестей сияньем
Рассветы вспыхнули в глазах.

Вернулись лебеди, лебеди -
Души укрыла нежность.
Весна нас свела в этой встрече,
Пусть будем парой навек.

Вернулись лебеди, лебеди -
Белое счастье в краю.
В сердцах прибавилось чистоты,
Как светел теперь этот мир.

Вернулись лебеди, лебеди -
Души укрыла нежность.
Весна нас свела в этой встрече,
Пусть будем парой навек.

Вернулись лебеди, лебеди -
К озёрам добавилась прелесть.
Самая большая радость -
Жить на земле родной.

Вернулись лебеди, лебеди -
Души укрыла нежность.
Весна нас свела в этой встрече,
Пусть будем парой навек.
```
