# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Гүзәл Өфем

```
Көмеш сулы Агыйделем
Каридел кочагында.
Каршы ала гүзәл Өфем
Биек тау башларында.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!

Гөл-чәчәкле бакчалары,
Бульварлары, урамнары.
Мең йөрәкләрдә уята
Мең-мең мутур җырларны.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!

Салават батыр тулпар атында
Каршы ала, озата.
Өфедә тик бер булганнар
Кире әйләнеп кайта.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!
```

# Перевод

### Рамил Чурагулов / Айрат Нәбиев - Прекрасная моя Уфа

```
Моя Агидель с серебряной водой
В объятиях Караидели.
Встречает прекрасная моя Уфа
На вершинах высоких гор.

Уфа моя, Уфа моя, прекрасная Уфа!
Моё прошлое, моё будущее!
Прекрасный цветок России,
Цветущий Урала цвет-цветок!!!

Сады её в цветах,
Бульвары и улицы.
В тысячах сердец будит
Тысячи прекрасных песен.

Уфа моя, Уфа моя, прекрасная Уфа!
Моё прошлое, моё будущее!
Прекрасный цветок России,
Цветущий Урала цвет-цветок!!!

Батыр Салават на тулпаре
Встречает и провожает.
Те, кто побывал в Уфе лишь раз,
Возвращаются вновь.

Уфа моя, Уфа моя, прекрасная Уфа!
Моё прошлое, моё будущее!
Прекрасный цветок России,
Цветущий Урала цвет-цветок!!!
```
