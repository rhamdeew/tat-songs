# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Торналар киткән чакта

```
Безне ташлап еракка,
Торналар киткән чакта.
Саубуллашу тавышлары,
Әллә ниләр уйлата.

Торналар киткән чакта,
Әллә ниләр уйлата, уйлата.
Торналар киткән чакта,
Әллә ниләр уйлата, уйлата.

Торналар киткән чакта,
Җиргә сары моң ята.
Бездән башка алар ничек,
Яшәр икән чит якта.

Торналар киткән чакта,
Җиргә сары моң ята, моң ята.
Торналар киткән чакта,
Әллә ниләр уйлата, уйлата.
```

# Перевод

### Расих Ханнанов / Рим Хасанов - Когда журавли улетели

```
Оставив нас навсегда,
Когда журавли улетели.
Звуки прощания,
Какие-то мысли навевают.

Когда журавли улетели,
Какие-то мысли навевают, навевают.
Когда журавли улетели,
Какие-то мысли навевают, навевают.

Когда журавли улетели,
На землю желтая тоска легла.
Как же им без нас
Живётся в чужих краях.

Когда журавли улетели,
На землю желтая тоска легла, легла.
Когда журавли улетели,
Какие-то мысли навевают, навевают.
```
