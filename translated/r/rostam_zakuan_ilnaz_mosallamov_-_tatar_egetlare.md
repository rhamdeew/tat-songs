# Оригинал

### Рөстәм Зәкуан / Илназ Мөсәлләмов - Татар егетләре

```
Ил чигендә сакта торып
Без чыныккан егетләр.
Иминлеген саклый илнең
Менә безнең кебекләр.

Эх, татар егетләре,
Ялтырый итекләре.
Чибәр кызлар өзелеп сөя,
Чибәр кызлар өзелеп сөя
Нәкъ безнең кебекләрне.

Әгәр сөйсә - өзелеп сөя,
Гомерлеккә яратып.
Аңлата белә башкача,
Йөрми гайбәт таратып.

Эх, татар егетләре,
Ялтырай итекләре.
Чибәр кызлар өзелеп сөя,
Чибәр кызлар өзелеп сөя
Нәкъ безнең кебекләрне.

Гармун алса кулларына -
Кушылып сайрый кошлар.
Егетләр җырлап җибәрсә -
Көнләшә сандугачлар.

Эх, татар егетләре,
Ялтырый итекләре.
Чибәр кызлар өзелеп сөя,
Чибәр кызлар өзелеп сөя
Нәкъ безнең кебекләрне.

Кайда булса да сынатмый
Чын татар егетләре.
Мәйданнарда билен бирми
Нәкъ безнең кебекләре.

Эх, татар егетләре,
Ялтырый итекләре.
Чибәр кызлар өзелеп сөя,
Чибәр кызлар өзелеп сөя
Нәкъ безнең кебекләрне.
Эх, татар егетләре...
```

# Перевод

### Рустем Закуан / Ильназ Мусаллямов - Татарские джигиты

```
На рубежах страны в дозоре
Мы, закаленные джигиты.
Безопасность страны хранят
Вот такие, как мы.

Эх, татарские джигиты,
Блестят их сапоги.
Красавицы без памяти любят,
Красавицы без памяти любят
Именно таких, как мы.

Если любит - то без памяти любит,
На всю жизнь обожая.
Умеет иначе объясниться,
Не ходит сплетни распространять.

Эх, татарские джигиты,
Блестят их сапоги.
Красавицы без памяти любят,
Красавицы без памяти любят
Именно таких, как мы.

Гармонь возьмет в руки -
Подпевают, присоединяясь, птицы.
Когда джигиты запоют -
Завидуют соловьи.

Эх, татарские джигиты,
Блестят их сапоги.
Красавицы без памяти любят,
Красавицы без памяти любят
Именно таких, как мы.

Где бы ни были - не подведут
Настоящие татарские джигиты.
На майданах не сдаются
Именно такие, как мы.

Эх, татарские джигиты,
Блестят их сапоги.
Красавицы без памяти любят,
Красавицы без памяти любят
Именно таких, как мы.
Эх, татарские джигиты...
```
