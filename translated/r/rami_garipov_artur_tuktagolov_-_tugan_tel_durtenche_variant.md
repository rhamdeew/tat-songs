# Оригинал

### Рәми Гарипов / Артур Туктаголов - Туган тел (дүртенче вариант)

```
Мин халкымның чәчәк күңеленнән,
Бал кортындай, энҗе җыямын,
Җыямын да - җанлы энҗеләрдән
Хуш исле бер кәрәз коямын.
Шуңа да мин беләм тел кадерен:
Бер телдән дә телем ким түгел -
Көчле дә ул, бай да, ягымлы да,
Ким күрер тик аны ким күңел!

Анам теле миңа - чәчән теле, -
Аннан башка минем халкым юк:
Йөрәгендә халкы булмаганның
Кеше булырга да хакы юк!

Халкым теле миңа - хаклык теле, -
Аннан башка минем илем юк:
Илен сөймәс кенә телен сөймәс,
Иле юкның гына теле юк!
Анам теле миңа - чәчән теле, -
Аннан башка минем халкым юк:
Йөрәгендә халкы булмаганның
Кеше булырга да хакы юк!

Анам теле миңа - чәчән теле, -
Аннан башка минем халкым юк:
Йөрәгендә халкы булмаганның
Кеше булырга да хакы юк!
```

# Перевод

### Рами Гарипов / Артур Туктаголов - Родной язык (четвертый вариант)

```
Я из цветущей души своего народа,
Как пчела, собираю жемчуг,
Собираю - из живых жемчужин
Душистый соты складываю.
Потому-то я цену языка понимаю:
Мой язык ни перед чем не отстает -
Он и могуч, и богат, и приятен,
Лишь ничтожная душа не ценит его!

Материнский язык для меня - язык поэта, -
Без него моего народа нет:
У кого в сердце нет народа,
Тот не имеет права быть человеком!

Язык моего народа для меня - язык правды, -
Без него моей страны нет:
Кто страну не любит, тот язык не любит,
Тот, у кого нет страны, того нет языка!
Материнский язык для меня - язык поэта, -
Без него моего народа нет:
У кого в сердце нет народа,
Тот не имеет права быть человеком!

Материнский язык для меня - язык поэта, -
Без него моего народа нет:
У кого в сердце нет народа,
Тот не имеет права быть человеком!
```
