# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Алга, дибез

```
Алга дибез, алга, дибез,
Артка таба барабыз,
Алга барырга кушкан бит
Безгә әби-бабабыз.
Их, их, безгә әби-бабабыз.

«СССР» диеп язылган
Самолет канатына,
Алдынгылар сайланырлар
Совет аппаратына.
Ух, ух, совет аппаратына.

Җирдә оҗмах төзер идем,
Эш килми шул кулымнан.
Алга карап артка бармыйк
Коммунизм юлыннан!
Уф, уф, коммунизм юлыннан!
```

# Перевод

### Разил Вәлиев / Луиза Батыр-Болгари - Вперёд, говорим

```
Вперёд зовём, вперёд зовём,
А сами пятимся назад,
А ведь идти вперёд велели
Нам прадеды и деды в лад.
Эх, эх, нам прадеды и деды в лад.

На крыле самолёта надпись:
«СССР» летит в дыму,
Передовых назначат служить
В советском верном аппарату.
Ух, ух, в советском верном аппарату.

Я б рай на землю выстроил,
Да руки не умеют так.
Глядя вперёд, не пятимся
Дорогой коммунизма в шаг!
Уф, уф, дорогой коммунизма в шаг!
```
