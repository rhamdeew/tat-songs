# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Ашкынып кайттым тагын

```
Күңелемне телгәләмә,
Әй ачы сары сагыш!
Туган ягым һәрчак истә
Аерды бары язмыш.

Авыл урамнары буйлап
Йөгерә балачагым.
Сагышларга түзалмыйча
Ашкынып кайттым тагын.

Туган яксыз үткән көннәр
Йөрәктә бетмәс ярам.
Күңелләрем сусаганда
Җыр-моңны аннан алам.

Авыл урамнары буйлап
Йөгерә балачагым.
Бергә үскән дус-ишләрне
Сагынып кайттым тагын.

Иң беренче тәпи баскан
Моңлы як, җырлы ягым.
Рәхмәтемне әйтергә дип,
Ашкынып кайткан чагым.

Авыл урамнары буйлап
Йөгерә балачагым.
Әнкәйнең кайнар ашларын
Сагынып кайттым тагын.

Сагышларга түзалмыйча
Ашкынып кайттым тагын.
```

# Перевод

### Рамил Чурагулов / Ризван Хакимов - Вернулся я, переполненный чувствами

```
Не тревожь мою душу,
Горечь горькой тоски!
Родной край мой всегда в памяти,
Лишь судьба разлучила.

По деревенским улицам
Бежит моё детство.
Не выдержав тоски,
Вернулся я снова, переполненный чувствами.

Дни без родного края -
Незаживающая рана в сердце.
Когда душа изнывает от жажды,
Я черпаю из него песни и мелодии.

По деревенским улицам
Бежит моё детство.
Скучая по друзьям и товарищам,
С кем рос вместе, вернулся я снова.

Где первый шаг я сделал,
Мой край напевный, мой край певучий.
Чтобы сказать спасибо,
Вернулся я, переполненный чувствами.

По деревенским улицам
Бежит моё детство.
Скучая по горячей материнской стряпне,
Вернулся я снова.

Не выдержав тоски,
Вернулся я снова, переполненный чувствами.
```
