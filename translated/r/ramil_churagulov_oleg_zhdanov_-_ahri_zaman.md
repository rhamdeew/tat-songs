# Оригинал

### Рамил Чурагулов / Олег Жданов - Ахры заман

```
Дөрләп яна дөнъя,
Бакчалар әйләнде, ком сүлгә.
Ахры заман килде,
Каннарсымнарлар җитми кешегә.
Заман ахры килде,
Ата - улны, ана - кызны белми.
Кешегә ни булды?
Талаш - ызгышлар, утлы - сугышлар кабынды.

Бу ахры заман?
Кем яклар соң  толларны?
Үлем чалгы кайрый
Кем саклар йортларны?
Күктән кергән изге көръән күрсәтер юлны
Гөнаһлыны юлга салыр.

Заман ахры килде,
Кешеләр әйләнде мәхлүккә.
Хурладылар динне
Имансын әхлак яткан чүплекта.
Заман ахры килде,
Җәнлек булып кеше кеше ашый.
Югалтык без юлны
Гөнаһлы җаннар куркмый Ходайдан.
Ни булды?

Бу ахры заман?
Кем яклар соң  толларны?
Үлем чалгы кайрый
Кем саклар йортларны?
Күктән кергән изге көръән күрсәтер юлны
Гөнаһлыны юлга салыр.
```

# Перевод

### Рамил Чурагулов / Олег Жданов - Конец времён

```
Горит мир ярким пламенем,
Сады обратились в пески.
Конец времён наступил,
Людям не хватает сострадания.
Конец времён пришёл,
Не знают отец - сына, мать - дочь.
Что случилось с людьми?
Ссоры, распри, огненные войны вспыхнули.

Это ли конец времён?
Кто защитит вдов?
Коса смерти косит,
Кто сохранит дома?
Священный Коран, ниспосланный с небес, укажет путь,
Приведёт грешника на праведный путь.

Конец времён наступил,
Люди превратились в чудовищ.
Покинули веру,
Имам и нравственность лежат в мусоре.
Конец времён пришёл,
Человек, как зверь, поедает человека.
Потеряли мы путь,
Грешные души не боятся Бога.
Что стало?

Это ли конец времён?
Кто защитит вдов?
Коса смерти косит,
Кто сохранит дома?
Священный Коран, ниспосланный с небес, укажет путь,
Приведёт грешника на праведный путь.
```
