# Оригинал

### Расих Ханнанов / Әмир Әбделханов - Йәнчишмәм минең

```
Яшлеккә тиң кайнар хисләр,
Йолдызлы зәңгәр кичләр.
Кошлар кебек моңнар түгеп
Уенга чыга яшьләр.

Әй, йәнчишмәм, йәнчишмәм
Ятларга серем чишмәм.
Әй, йәнчишмәм, минең йәнчишмәм
Ятларга серем чишмәм.

Нур чәчә гөл, көлә гөл
Кем туйган гөлне сөеп?
Яшьләнә җир, яшәрә җир
Кышкы салкынны жиңеп.

Әй, йәнчишмәм, йәнчишмәм
Ятларга серем чишмәм.
Әй, йәнчишмәм, минең йәнчишмәм
Ятларга серем чишмәм.

Уйлар да ак, йөзләр дә шат,
Күңелгә илһам тулган.
Без, яз кебек, якты сибеп
Яшәргә җиргә туган.

Әй, йәнчишмәм, йәнчишмәм
Ятларга серем чишмәм.
Әй, йәнчишмәм, минең йәнчишмәм
Ятларга серем чишмәм.
```

# Перевод

### Расих Ханнанов / Әмир Әбделханов - Мой Янчишмэ

```
Горячие чувства, ровня юности,
Звёздные синие вечера.
Как птицы, льют напевы нежные,
На игры выходят ребята.

Эй, Янчишмэ мой, Янчишмэ,
Чужим не выдаю секреты.
Эй, Янчишмэ, мой Янчишмэ,
Чужим не выдаю секреты.

Цветок сияет, смеётся цветок -
Кто пресытится, нежно любя?
Земля молодеет, земля зеленеет,
Зимний мороз одолев.

Эй, Янчишмэ мой, Янчишмэ,
Чужим не выдаю секреты.
Эй, Янчишмэ, мой Янчишмэ,
Чужим не выдаю секреты.

Мысли чисты, лица светлы,
В сердце струится вдохновение.
Мы, словно весна, дарим сиянье -
Рождённые жить на земле.

Эй, Янчишмэ мой, Янчишмэ,
Чужим не выдаю секреты.
Эй, Янчишмэ, мой Янчишмэ,
Чужим не выдаю секреты.
```
