# Оригинал

### Роберт Миңнуллин / Билгесез - Багышлау җыры

```
Синең белән узган минутларны
Айдар-елларга да алышмим.
Уем белән мин гел синең янда,
Өзгәләнсен әле синең җан да -

Бу җырымны сиңа, бу җырымны сиңа,
Бу җырымны сиңа багышлыйм!

Сагынулар, саргаюлар белән,
Югалтулар белән танышкач.
Мине сөюеңне төшенерсең,
Син дә мине искә төшерерсең -

Бу җырымны сиңа, бу җырымны сиңа,
Бу җырымны сиңа багышлыйм!

Күңелләрем шатлык тулы иде,
Шатлык идем, идем сагыш мин.
Сиңа бары шатлык-шатлык теләп,
Сиңа бары аклык-аклык теләп,

Бу җырымны сиңа, бу җырымны сиңа,
Бу җырымны сиңа багышлыйм!
```

# Перевод

### Роберт Миңнуллин / Билгесез - Песня посвящения

```
Минуты, проведённые с тобой,
Не променял бы и на годы.
Душой я всегда рядом с тобой,
Пусть изменится даже твоя душа -

Эту песню тебе, эту песню тебе,
Эту песню тебе посвящаю!

Скитаясь по тоске и печали,
Познав, что значит терять,
Ты вспомнишь свою любовь ко мне,
И ты меня тоже вспомнишь -

Эту песню тебе, эту песню тебе,
Эту песню тебе посвящаю!

Наши сердца были полны радости,
Я был радостью, я был тоской.
Желаю тебе только радость-радость,
Желаю тебе только счастье-счастье,

Эту песню тебе, эту песню тебе,
Эту песню тебе посвящаю!
```
