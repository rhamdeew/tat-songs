# Оригинал

### Рәзил Хәкимов / Мөфтәдин Гыйләҗев - Мизгелләр агышы

```
Югалту, табышлар,
Үкенү ялгышлар,
Газаплы юксыну, сагышлар,
Үтмәгән киртәләр,
Тумаган иртәләр,
Ул безнең киләчәк язмышлар.

Мизгелләр, минутлар
Туктаусыз агалар,
Вакытның аяусыз агышы.
Яшьлегем киченнән,
Ак томан эченнән
Ишетәм кебек мин тавышың.
```

# Перевод

### Разиль Хакимов / Муфтедин Гыляжев - Течение мгновений

```
Потери, находки,
Ошибки, упрёки,
Мучительная тоска и печаль,
Непройденные преграды,
Несбывшиеся рассветы -
Это наша грядущая судьба.

Мгновения, минуты
Без устали текут,
Беспощадный поток времени.
Из вечера моей юности,
Сквозь белый туман
Слышу, будто бы, твой голос.
```
