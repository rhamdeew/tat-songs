# Оригинал

### Роберт Миңнуллин / Илнур Ганиев - Сабантуй бүген бездә

```
1.Шатлыклар -күңелебездә,
Ал кояш - күгебездә.
Килегез, дуслар, кунакка,
Сабантуй бүген бездә.

Җилферди чиккән сөлгеләр,
Бүләкләр - күбебезгә!
Көрәшик тә, узышыйк та -
Сабантуй бүген бездә!
Җилферди чиккән сөлгеләр,
Бүләкләр - күбебезгә!
Сөйләшик тә җырлашыйк та
Үз туган телебездә.

Иң уртага утыртканнар
Иң озын бер колганы -
Колганың да бик тиз генә
Менеп булмый торганы.

Абыйлар алып килгәннәр
Иң җитез юртакларын.
Гармунчылар алып чыккан
Җырларның уртакларын.
```

# Перевод

### Роберт Миннуллин / Ильнур Ганиев - Сабантуй сегодня у нас

```
Радость в наших сердцах,
Алое солнце в небесах.
Приходите, друзья, в гости,
Сабантуй сегодня у нас.

Полотенца с вышитой каймой,
Подарки - многим из нас!
И борьбу, и скачки проведём -
Сабантуй сегодня у нас!
Полотенца с вышитой каймой,
Подарки - многим из нас!
Поговорим и песни споём
На родном своём языке.

В самый центр поставили
Самый длинный шест -
На такой шест и взобраться
Очень быстро не получится.

Братья привели с собой
Самых резвых скакунов.
Гармонисты подхватили
Припевы песен.
```
