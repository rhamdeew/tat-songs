# Оригинал

### Рафаэль Сафин / Шамил Колбарисов - Янбикә

```
Базарларга барсаң, ак тула ал,
Хакксынасын үзем түләрмен.
Әҗәлләрең җитсә, әйт үземә,
Янбикәм, Янбикәм,
Синең өчен үзем үләрмен.

Урманнарга керсәм, адашмамын,
Менгән атым акбуз ат булса.
Авыр юллар үтеп, мин кайтырмын.
Янбикәм, Янбикәм,
Көтәм дигән сүзен хак булса.
```

# Перевод

### Рафаэль Сафин / Шамил Колбарисов - Янбикя

```
Если на базар пойдешь - купи белый тюль,
Я за него сама заплачу.
Если смерть подступит - мне скажи,
Янбикя, Янбикя,
За тебя я сама умру.

Если в лес войду - не заблужусь,
Если конь мой белый - Акбузат.
Через трудный путь я вернусь.
Янбикя, Янбикя,
Если верно слово: «Я буду ждать».
```
