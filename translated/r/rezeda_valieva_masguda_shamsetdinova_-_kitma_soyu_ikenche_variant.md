# Оригинал

### Резеда Вәлиева / Мәсгудә Шәмсетдинова - Китмә, сөю (икенче вариант)

```
Китмә, сөю, китмә күңелемнән,
Ялгызлыкка мине ташлама.
Күрәсеңме, язның назлавына
Пар тургайлар ничек шатлана.

Китмә, сөю, китмә күңелемнән,
Көзләр хөкеменә күнмә син,
Күңелемә нурлар српеп янган
Зәңгәр йолдызларым сүнмәсен.

Үрелмәсен күңелем гөлләренә
Көзге кырауларның салкыны,
Язлар булсын һаман йөрәгемдә,
Сүрелмәсен яшьлек ялкыны.
```

# Перевод

### Резеда Вәлиева / Мәсгудә Шәмсетдинова - Не уходи, любовь (второй вариант)

```
Не уходи, любовь, не уходи из сердца,
Не отдавай меня в объятья одиночества.
Видишь, как весны ласке
Радуются в паре жаворонки.

Не уходи, любовь, не уходи из сердца,
Не покоряйся приговору осени,
Пусть не гаснут мои синие звезды,
Что светом опалили душу.

Пусть холод осенних изморозей
Не тронет цветов моей души,
Пусть весны всегда живут в моем сердце,
Пусть не угаснет пламя юности.
```
