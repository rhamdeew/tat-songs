# Оригинал

### Рәшит Шәкүр / Салават Низаметдинов - Хисләремне әйтеп бирер өчен

```
Хисләремне әйтеп бирер өчен
Нинди сүзләр эзләп табырга?
Туган якның алтын тауларынмы
Сиңа, җаным алып барырга!

Шул тауларның көмеш чишмәләрең
Уеп куергамы алдыңа,
Күктән йолдыз тотып, шунымы әллә,
Учларыма илтеп салырга!

Бөтен нәрсә бары синең өчен,
Синең өчен бу җир, бу һава.
Син боексаң, болытларны ярып,
Сиңа багып, кояш елмая.

Сиңа, җаным, бөтен якты дөнья,
Иркә җилләр, җылы яңгырлар ...
Яңгыр үткәч, әлеп куяемчы
Юлларыңа гәлсәр җәйгорлар!
```

# Перевод

### Рәшит Шәкүр / Салават Низаметдинов - Чтобы чувства мои сказать

```
Чтобы чувства мои сказать,
Какие слова мне отыскать?
Может, горы золотые края родного
Тебе, любовь моя, преподнести?

Серебро родников тех горных
Перед тобой бы высечь, положить,
Или, звезду с небес сорвав,
В ладони тебе донести?

Всё на свете - лишь для тебя,
Для тебя эта земля, этот воздух.
Ты взглянёшь - раздвинув облака,
Улыбается солнце, глядя на тебя.

Для тебя, любовь, весь светлый мир,
Нежный ветер, тёплые дожди...
После дождя бы выложить
Твоим путям жемчужные дождинки!
```
