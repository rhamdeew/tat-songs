# Оригинал

### Рәми Гарипов / Рәүф Сәлтәшев - Зарлы кош

```
Дулкынлана, шаулый ак каен
Толымнарын җилгә туздыра.
Ботагына аның кәкүк кунып,
Ялгызлыкта гомер уздыра.
Тик мин генә кәкүк моңын тыңлап
Моңсулана кәкүк каенда.
Син шул каен булсаң, кәкүк булып
Калыр идең шушы каенда.

Чакыра кәкүк, эй,чакыра
Чакырыуын бер кем ишетми.
Зарлы кошны парлы кошлар ахыры
Үз янында һич тә ишитми.

# Перевод

### Рәми Гарипов / Рәүф Сәлтәшев - Плачущая птица

```
Волной шумит, колышется белая берёза,
Косы свои по ветру распускает.
На её ветвь садится кукушка,
И в одиночестве век коротает.
Лишь я одна, слушая зов кукушки,
Печалюсь вместе с нею на берёзе.
Будь ты той берёзой - кукушкой бы
Осталась ты на этой же берёзе.

Кукушка кличет, эй, кукушка кличет -
Её зова никто не слышит.
Плачущую птицу птицы парные, увы,
Рядом с собой совсем не слышат.
