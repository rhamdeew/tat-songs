# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Егетләр җыры (беренче вариант)

```
Әйдә барыйк, кызлар карыйк, -
Орчык эрләгәннәрен;
Аяз көнне яшен суксын
Безне сөймәгәннәрен.

Әйдә барыйк, кызлар карыйк,
Керик куеннарына.
Асылынсак, асылыныйк
Кызлар муеннарына.

Әйдә барыйк, кызлар карыйк
Казан ярминкәсендә.
Сынатмасын егет кеше
Мәхәббәт өлкәсендә.
```

# Перевод

### Разил Валиев / Луиза Батыр-Болгари - Песня парней (первый вариант)

```
Пойдем, посмотрим на девушек, -
Как прядут они веретено;
Пусть в ясный день ударит гром
По тем, кто нас не любит.

Пойдем, посмотрим на девушек,
Зайдем в их теплые объятья.
Если вешаться - то повиснем
На девичьих шеях.

Пойдем, посмотрим на девушек
На казанской ярмарке.
Пусть парень не ударит в грязь лицом
В делах любви.
```
