# Оригинал

### Разил Вәлиев / Мирсәет Яруллин - Табигать балалары

```
Җилкәдә - поход биштәре,
Күңелдә - серле уйлар.
Безнең хыял, шатлык эзләп,
Дөньяны мең кат буйлар.

Без - табигать балалары,
Безнең бәхетле чаклар...
Җир өстеннән төнне куып,
Дөрләп янар учаклар.

Каршылар безне урманнар,
Озатып калыр кырлар.
Юлдаш булыр мәктәпләрдә
Өйрәнгән дәртле җырлар.
```

# Перевод

### Разил Вәлиев / Мирсәет Яруллин - Дети природы

```
За плечами - походный ранец,
В сердце - тайные думы.
Мы, мечту и радость ища,
Мир обходим тысячи раз.

Мы - дети природы,
Наши счастливые часы...
Гоня ночь с лица земли,
Пышно разгорятся костры.

Нас встречают леса,
Провожают поля.
С нами в пути - задорные песни,
Выученные в школе.
```
