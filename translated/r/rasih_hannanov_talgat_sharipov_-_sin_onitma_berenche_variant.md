# Оригинал

### Расих Ханнанов / Тәлгать Шәрипов - Син онытма (беренче вариант)

```
Ник очрадык миңа тормышымның
Зур юлында сердәш булмагач,
Күзләремә карап «сөям!» дидең,
Үзен башкаларны уйлагач?

Иркәләдең чәчләремнән сыйпап,
Гөлләр өздең миңа аралап.
Нигә шулай керсез йөрәгемне
Гомерлеккә киттең яралап?

Һич онытма, дуслар кирәк булса,
Тәүге мәхәббәтнең ялкынын
Тап төшерми саклап йөретүче
Саф күңелле берәү барлыгын.
```

# Перевод

### Расих Ханнанов / Тәлгать Шәрипов - Не забудь (первый вариант)

```
Зачем мы встретились на главном пути моей жизни,
Если не быть нам друг другу доверенным,
Ты, глядя мне в глаза, сказала: «Люблю»,
Когда думала о других?

Ты ласкала, проводя ладонью по моим волосам,
Срывала мне цветы по дороге.
Зачем же ты, так чистое сердце,
Ушла, навеки его ранив?

Никогда не забудь: если понадобятся друзья,
Есть тот, кто бережет пламя первой любви,
Кто не дает ему угаснуть,
С чистым сердцем - живет он.
```
