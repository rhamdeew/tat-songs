# Оригинал

### Роза Вәлиева / Мөфтәдин Гыйләҗев - Искә төшер

```
Хискә тулы йөрәгемне
Тагын бер яндыр әле.
Керфекләрем очларыннан
Тамчылар тамдыр әле.

Бер исемә төшер әле.
Күңелле чакларымны.
Күз алдыма китер әле
Үз туган якларымны.

Искә төшер авылымның
Тыныч айлы төннәрен.
Кышларында чәчәк аткан
Әнкәемнең гөлләрен.
```

# Перевод

### Роза Вәлиева / Мөфтәдин Гыйләҗев - Вспомни

```
Сердце, полное чувств,
Снова опали мне.
С ресниц моих
Пусть капли прольются.

Вспомни ещё раз
Мои светлые дни.
Представь мне снова
Мои родные края.

Вспомни мой аул,
Тихие лунные ночи.
И мамины цветы,
Что в зимы расцветали.
```
