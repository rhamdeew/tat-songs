# Оригинал

### Рәмзинә Тимерова / Риф Арслан - Яраттым (беренче вариант)

```
Мин ярттым сине, яраттым,
Син бит минем ике канатым.
Мәхәббәтем җиккән пар атым,
Гомерлеккә диеп яраттым.

Яратмадым түгел, яраттым,
Әллә ничек үзеңә караттым.
Алтын кулың әсир иттеме?
Тәмле телен тәъсир иттеме?

Әллә инде назлы карашың,
Күңелем кылын сиртеп үттеме?
Сине кушып җырласын диеп,
Үзенең көен салып киттеме.

Яратмадым түгел, яраттым,
Әллә ничек үзеңә караттым.
Алтын кулың әсир иттеме?
Тәмле телен тәъсир иттеме?
```

# Перевод

### Рәмзинә Тимерова / Риф Арслан - Яраттым (первый вариант)

```
Я полюбил тебя, я полюбил -
Ты для меня два белых крыла.
Упряжь любви - мой парный конь,
Я полюбил навек, навсегда.

Не разлюбил - я полюбил,
Как-то тебя к себе приворожил.
Золотые руки в плен взяли?
Сладкие речи очаровали?

Или нежный твой взгляд
По струнам сердца прошёл, прозвенел?
Чтобы вместе с тобой он запел,
Свой напев оставил и улетел.

Не разлюбил - я полюбил,
Как-то тебя к себе приворожил.
Золотые руки в плен взяли?
Сладкие речи очаровали?
```
