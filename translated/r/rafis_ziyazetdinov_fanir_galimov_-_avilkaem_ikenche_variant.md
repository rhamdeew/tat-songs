# Оригинал

### Рәфис Зиязетдинов / Фәнир Галимов - Авылкаем (икенче вариант)

```
Яшәсәм дә еракларда
Тора алмыйм мин синсез.
Читкә типмә, авылкаем,
Буласым килми сансыз.

Йөзәрләгән балаларың
Дөньяларны гизәләр.
Авырлыклар килгән чакта
Сиңа табынып түзәләр.

Гомерлеккә уелгандыр,
Мәхәббәт авылыма.
Агарса да чигә чәчем,
Бетмәс сөюем сиңа.
```

# Перевод

### Рафис Зиязетдинов / Фанир Галимов - Моя деревня (второй вариант)

```
Хоть живу я вдали,
Без тебя не могу.
Не гони меня, деревенька,
Не хочу быть забытым.

Сотни твоих детей
Мир широкий обходят.
Когда тяжко бывает,
К тебе, как к святыне, идут.

Навек вырезано
В сердце - любовь к моему селу.
Пусть виски поседеют,
Не иссякнет моя любовь к тебе.
```
