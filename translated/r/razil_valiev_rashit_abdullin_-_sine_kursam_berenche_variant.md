# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Сине күрсәм (беренче вариант)

```
Сине күрсәм, сине күрсәм,
Сагышларым уяна.
Сине күрсәм, сине күрсәм,
Сагышларым уяна да,
Йөрәгем утта яна.

Сине күрсәм, сине күрсәм,
Күңелне моңнар сара.
Сине күрсәм, сине күрсәм,
Моңнар сара, әрни башлый
Йөрәгемдәге яра.

Сине күрсәм, сине күрсәм,
Җир әйләнә кирегә.
Сине күрсәм, сине күрсәм,
Җирләр әйләнә кирегә,
Син киләсең бирегә.
```

# Перевод

### Разиль Вәлиев / Рәшит Абдуллин - Увидев тебя (первый вариант)

```
Увидев тебя, увидев тебя,
Тоска моя просыпается.
Увидев тебя, увидев тебя,
Тоска моя просыпается,
Сердце моё горит огнём.

Увидев тебя, увидев тебя,
Душу мою терзают печали.
Увидев тебя, увидев тебя,
Печали терзают, начинается страдание
Раны в моем сердце.

Увидев тебя, увидев тебя,
Земля вертится вспять.
Увидев тебя, увидев тебя,
Земли вертятся вспять,
Ты идешь ко мне.
```
