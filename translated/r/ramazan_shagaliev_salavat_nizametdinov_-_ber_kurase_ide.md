# Оригинал

### Рамазан Шагалиев / Салават Низаметдинов - Бер күрәсе иде

```
Бер күрәсе иде,
Бер үтәсе иде
Синең белән яшьлек эзеннән.
Таң аттырып бергә,
Куркып тормый бер дә,
Олыларның әйткән сүзеннән.
Бер күрәсе иде,
Бер үтәсе иде!

Бер күрәсе иде,
Бер үрәсе иде
Такыялар сиңа икәүләп.
Карап туймас идем,
Ару тоймас идем,
Көне буе сине иркәләп.
Бер күрәсе иде,
Бер үрәсе иде!

Бер күрәсе иде,
Бер үбәсе иде,
Миңнәреңнән назлап, күзеңнән.
Сөю җырын көйләп,
Эч серемне сөйләп,
Төшермичә сине теземнән.
Бер күрәсе иде,
Бер үбәсе иде!
```

# Перевод

### Рамазан Шагалиев / Салават Низаметдинов - Хоть раз бы увидеть

```
Хоть раз бы увидеть,
Хоть раз бы пройти
По следам нашей юности с тобой.
Вместе рассвет встречая,
Ни капли не робея,
Не слушая старших наказов порой.
Хоть раз бы увидеть,
Хоть раз бы пройти!

Хоть раз бы увидеть,
Хоть раз бы сплести
Тебе венки вдвоём на лету.
Не насмотрелся б,
Не утомился б,
Целый день бы ласкал красоту.
Хоть раз бы увидеть,
Хоть раз бы сплести!

Хоть раз бы увидеть,
Хоть раз бы поцеловать,
Лаская твои родинки, взгляд.
Песню любви напевая,
Тайну души открывая,
Не отпуская тебя с колен наугад.
Хоть раз бы увидеть,
Хоть раз бы поцеловать!
```
