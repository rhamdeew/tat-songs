# Оригинал

### Рөстәм Мингалим / Фасил Әхмәтов - Уйныйк әле дулкыннардай

```
Боргаланып, боргаланып
Ага бит ул Иделкәй.
Ярга түгел, күңелләргә
Кага бит ул Иделкәй.

Матур эшләребез бар,
Якты теләгебез бар.
Идел төсле талу белмәс
Ярсу йөрәгебез бар.

Юлларыбыз, юлларыбыз
Иделкәйгә тиң икән.
Күзләреңдә минем күзләр,
Уйларыңда кем икән?

Матур эшләребез бар,
Якты теләгебез бар.
Идел төсле талу белмәс
Ярсу йөрәгебез бар.

Дулкыннары, дулкыннары
Уйный бит ул Иделнең.
Уйныйк әле дулкыннардай
Кочагында илемнең.

Матур эшләребез бар,
Якты теләгебез бар.
Идел төсле талу белмәс
Ярсу йөрәгебез бар.
```

# Перевод

### Рөстәм Мингалим / Фасил Әхмәтов - Сыграем, словно волны

```
Извиваясь, извиваясь,
Течет наш батюшка Идель.
Не к берегам - к сердцам стучится,
Стучится Идель, светлый день.

Есть у нас дела прекрасные,
Есть у нас мечта ясна.
Как Идель - не знает устали
Наша пылкая душа.

Пути наши, пути наши
Иделю сродни, видать.
В твоих глазах - мои глаза,
В твоих думах - кто же там?

Есть у нас дела прекрасные,
Есть у нас мечта ясна.
Как Идель - не знает устали
Наша пылкая душа.

Волны, волны Иделя -
Играют, плещут вдалеке.
Сыграем, словно эти волны,
В объятьях родины-земли.

Есть у нас дела прекрасные,
Есть у нас мечта ясна.
Как Идель - не знает устали
Наша пылкая душа.
```
