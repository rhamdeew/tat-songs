# Оригинал

### Рәзинә Насыйбуллина / Рубис Зарипов - Киек казлар очты, күрдегезме?

```
Киек казлар очты, күрдегезме,
Киек казлар очты каңгылдап.
Мамыклары гына яман сулап,
Күл өстендә калды зар елап.
Киек казлар очты, күрдегезме,
Киек казлар очты тезелеп.
Бәгырьләргә төшеп, җанны телеп,
Сагыш калды җилдә өзелеп.
Киек казлар очты, күл калды,
Сыны катып елый камышы.
Һаваларда киек каз сагышы,
Гомерләрнең көзгә агышы.
Гомерләрнең көзгә агышы...
```

# Перевод

### Рәзинә Насыйбуллина / Рубис Зарипов - Дикие гуси улетели, видели ли вы?

```
Дикие гуси летели - видели ль вы,
Дикие гуси летели, гогоча.
Лишь их пухи, тяжко вздохнув,
На озере плачем остались, дрожа.
Дикие гуси летели - видели ль вы,
Дикие гуси летели строем.
В сердце упали, душу рассекли,
Печаль в ветрах оборвалась стоном.
Дикие гуси летели - озеро осталось,
Камыш окаменел, рыдает, хрипя.
В небесах - тоска диких гусей,
В осень стекают года бытия.
В осень стекают года...
```
