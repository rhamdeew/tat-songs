# Оригинал

### Роберт Миңнуллин / Мөфтәдин Гыйләҗев - Якты кичләр, назлы кичләр

```
Икәү эчкән чишмә шул ук,
Икәү тапкан тугай шул.
Тик үзебез  менә,
Тик үзебез генә
Үзгәргәнбез бугай шул.

Без үзебез үзгәрсәк тә
Кыр да шул ук, тугай шул.
Әле дә мин сине,
Әле дә син мине
Яратабыз бугай шул.

Һавада икәү сокланган
Бар йолдызлар шул, ай шул.
Андый якты кичләр,
Андый назлы хисләр
Онытылмый бугай шул.
```

# Перевод

### Роберт Миннуллин / Мофтадий Гилязев - Светлые вечера, нежные вечера

```
Всё тот же ручей, из которого вдвоём пили,
Всё та же лужайка, что нашли вдвоём.
Только мы с тобой,
Только мы с тобой,
Должно быть, изменились.

Даже если мы изменимся,
Поле всё то же, лужайка та же.
Всё ещё я тебя,
Всё ещё ты меня,
Должно быть, любим.

Всё те же звёзды и луна,
Что восхищали нас обоих в небе.
Такие светлые вечера,
Такие нежные чувства
Никогда не забудутся.
```
