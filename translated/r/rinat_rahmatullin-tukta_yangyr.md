# Оригинал

### Ринат Рахматуллин - Тукта янгыр

```
Тэрэзэгэ козге янгыр сибэли
Узэклэрне сагыну озгэли
Миндэ дэ бит кайгы хислэре
Янгыр юды синен эзлэрне

Тукта янгыр,тукта янгыр яума эле
Сагыш басты кунелемне анла эле
Син кайда сон сойгэн ярым кайда син
Эзлэп сине тэрэзэдэн котэм мин.

Ачуланып мина китеп бардын
Янгыр арасында югалдын
Упкэлэткэн булсам гафу ит
Синсез мина бу доньяда кыен бит

Тукта янгыр,тукта янгыр яума эле
Сагыш басты кунелемне анла эле
Син кайда сон сойгэн ярым кайда син
Эзлэп сине тэрэзэдэн котэм мин.

Козлэрдэн сон нигэ жэйлэр килми
Кабат очрашырбызмы белмим
Ометемне озми котэм сине
Кунелемне юат минем кил эле.

Тукта янгыр,тукта янгыр яума эле
Сагыш басты кунелемне анла эле
Син кайда сон сойгэн ярым кайда син
```
