# Оригинал

### Рамил Гыймрани - Синең өчен (бишенче вариант)

```
Синең өчен - туганмын җиргә,
Янәш барырга дип гомергә.
Пар аккоштай очыйк ла җаным,
Киң офокларга гел бергә.

Сулыр һауам, йөрәк дәвам,
Синең өчен - янам җаным, янам! (Синең өчен генә - мин янам!)

Җил-давыллар, авыр сынаулар,
Бергә чакта, үтә, сизелми.
Син булганда, бары син барында,
Авырлыклар килсә, бирешмәм!

Сулыр һауам, йөрәк дәвам,
Синең өчен - янам җаным, янам!
```

# Перевод

### Рамиль Гимрани - Для тебя (пятая версия)

```
Для тебя родился я на свет,
Чтобы рядом шёл с тобой сквозь годы.
Как певец белоснежный, в небесах летит мой дух,
Вместе к просторам мы плывём.

Слёзы лью и сердце ноет,
Для тебя горю я, мой любимый, горю! (Только для тебя один я и горю!)

Бури, шторма, тяжкие испытания -
Вместе проходят, не чувствуя тревоги.
Когда ты со мной, только о тебе и думаю,
Если трудности придут - не сдамся!

Слёзы лью и сердце ноет,
Для тебя горю я, мой любимый, горю!
```
