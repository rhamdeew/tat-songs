# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Ай батканда, таң атканда

```
Ай батканда, таң атканда
Уздым җырлап урамнардан.
Ай батканда, таң атканда
Кемнәр язмыш юрамаган?!

Бара идек таңда урманнарга,
Кошлар моңын бергә тыңларга.
Синең өчен былбыл сайрый анда,
Минем өчен кәккүк моңлана.

Ай батканда, таң атканда
Кайтып киләм туйларыңнан.
Төн карасы, җан ярасы
Китми минем уйларымнан.

Бара идек таңда урманнарга,
Кошлар моңын бергә тыңларга.
Синең өчен былбыл сайрый анда,
Минем өчен кәккүк моңлана.

Ай батканда, таң атканда
Җырлыйм да күк, җылыйм да күк.
Мең шатлыклар телим сиңа...
Туктамыйча сана, кәккүк!

Бара идек таңда урманнарга,
Кошлар моңын бергә тыңларга.
Синең өчен былбыл сайрый анда,
Минем өчен кәккүк моңлана.
```

# Перевод

### Разил Валиев / Рашит Абдуллин - Когда луна заходит, когда рассвет встаёт

```
Когда луна заходит, рассвет встаёт,
Я с песней по улицам прохожу.
Когда луна заходит, рассвет встаёт,
Кто судьбою своей не дорожит?!

Мы уходили на рассвете в леса,
Вместе слушать птичьи голоса.
Для тебя там соловей поёт,
Для меня кукушка слёзы льёт.

Когда луна заходит, рассвет встаёт,
Возвращаюсь я с твоих свадеб.
Тьма ночная, рана душевная
Не уходят из моих дум.

Мы уходили на рассвете в леса,
Вместе слушать птичьи голоса.
Для тебя там соловей поёт,
Для меня кукушка слёзы льёт.

Когда луна заходит, рассвет встаёт,
И пою я много, и плачу много.
Тысячу радостей желаю тебе...
Считай без остановки, кукушка!

Мы уходили на рассвете в леса,
Вместе слушать птичьи голоса.
Для тебя там соловей поёт,
Для меня кукушка слёзы льёт.
```
