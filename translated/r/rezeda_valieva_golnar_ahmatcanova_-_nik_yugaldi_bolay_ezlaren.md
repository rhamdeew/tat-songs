# Оригинал

### Резеда Вәлиева / Гөлнар Әхмәтҗанова - Ник югалды болай эзләрең?

```
Кайтырмын дип киткән идең,
Бәхет вәгъдә иткән идең,
Оныттыңмы биргән сүзләрең?
Кар-бураннар куды микән,
Язгы сулар юды микән,
Ник югалды болай эзләрең?

Күңелемнән ничек куйыйм,
Җаннарымнан ничек җуйыйм,
Күңелем һаман сине көтә бит.
Вәгъдәсез яр нигә кирәк,
Оныт аны, оныт йөрәк!
Яшь гомерем зая үтә бит.

Кошлар кайта, кошлар китә,
Гомеркәйләр һаман үтә
Аккан сулар кебек тын гына.
Язлар бәхет алып килми,
Якты таңнар шатлык бирми,
Күңелемдә сагыш-моң гына.
```

# Перевод

### Резеда Валиева / Гульнар Ахметжанова - Куда твои следы?

```
Обещал вернуться,
Счастье пообещал,
Забыл ли ты свои слова?
Снежные бураны охватили,
Весенние воды ушли,
Куда ты исчез, оставив следы?

Как вычеркнуть из души,
Как смыть с души,
Сердце всё равно ждет тебя.
Безобещающая любовь зачем нужна,
Забудь её, забудь, сердце!
Молодая жизнь проходит впустую.

Птицы возвращаются, птицы уходят,
Судьбы проходят мимо,
Как течение реки, тихо.
Лета не приносят счастья,
Светлые рассветы не дарят радости,
В сердце лишь тоска да печаль.
```
