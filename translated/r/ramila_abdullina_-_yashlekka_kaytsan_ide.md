# Оригинал

### Рамилә Абдуллина - Яшьлеккә кайтсаң иде

```
Яшьлеккә, яшьлеккә
Юллар ачылмый хәзер.
Мәктәп елларын сагынып,
Өзгәләнәдер бәгырь.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Ак алъяпкыч, кара форма
Тагын бер кисәң иде.

Яшьлекнең, яшьлекнең
Хатирәләре якты.
Шуңа күрә сагынабыз
Бу гүзәл, матур чакны.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Шул бәхетле мизгелләрне
Тагын кичерсәң иде.

Яшьлек бит, яшьлек бит -
Иң көчле, җегәр вакыт.
Таулар-ташлар актарырдай,
Зур көчләр тора ташып.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Анда калган көч-дәртләрне
Монда күчерсәң иде.
```

# Перевод

### Рамилә Абдуллина - Если бы вернуться в юность

```
В юность, в юность
Пути не открыты теперь.
По школьным годам тоскуя,
Сжимается сердце, поверь.

В юность, в юность
Вернуться бы хоть на миг.
Белый фартук, форма чёрная -
Снова надеть бы их.

У юности, у юности
Воспоминанья светлы.
Потому и скучаем
По этой прекрасной поре.

В юность, в юность
Вернуться бы хоть на миг.
Те счастливые мгновенья
Ещё бы раз прожить.

Юность ведь, юность ведь -
Самая сильная пора.
Горы и камни сдвинет -
Бурная сила полна.

В юность, в юность
Вернуться бы хоть на миг.
Там оставшиеся силы и пыл
Сюда бы перенести.
```
