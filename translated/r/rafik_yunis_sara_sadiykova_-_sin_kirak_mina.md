# Оригинал

### Рәфик Юныс / Сара Садыкова - Син кирәк миңа

```
Син кирәк миңа - кайгы телгәндә,
Син кирәк миңа - шатлык килгәндә.
Көләч кояшка карап көлгәндә,
Төн йолдызларга сагыш килгәндә.

Сагыныр өчен еракларда,
Сокланыр өчен бергә чакларда.
Барча йөзләрдә, синең төсмерне
Сагынып эзләргә - син генә кирәк миңа!

Син кирәк миңа, кирәк хатларда
Матур исмеңне кабатларга.
Хыял дөньясын тынмый гизәргә,
Йөз өмет өзеп мең кат түзәргә!

Сагыныр өчен еракларда,
Сокланыр өчен бергә чакларда.
Барча йөзләрдә, синең төсмерне
Сагынып эзләргә - син генә кирәк миңа!
```

# Перевод

### Рафик Юныс / Сара Садыкова - Ты нужна мне

```
Ты нужна мне - когда печаль обрушится,
Ты нужна мне - когда радость придёт.
Когда смеюсь, на южное солнце глядя,
Когда тоска по ночным звёздам охватит.

Чтобы скучать вдали,
Чтобы любоваться, когда вместе.
Во всех лицах твой образ
Искать с тоской - лишь ты нужна мне!

Ты нужна мне, нужна в письмах
Повторять твоё прекрасное имя.
Мир мечты непрестанно украшать,
Сто надежд лелея, тысячу раз терпеть!

Чтобы скучать вдали,
Чтобы любоваться, когда вместе.
Во всех лицах твой образ
Искать с тоской - лишь ты нужна мне!
```
