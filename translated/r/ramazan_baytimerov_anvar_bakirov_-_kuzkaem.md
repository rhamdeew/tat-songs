# Оригинал

### Рамазан Байтимеров / Әнвәр Бакиров - Күзкәем

```
Бәрәкәтле синең кырларыңда
Маңгай тирем тамды, Күзкәем.
Туып үскән авылым Күзкәемдә
Җуелмасын иде эзкәем.

Күзкәем, Күзкәем,
Җуелмасын иде эзкәем.

Тырыш хезмәт белән һәр көн саен
Матурлана авылым Күзкәем.
Күзкәемне данлап җырлар өчен,
Җитмәс кебек җылы сүзкәем.

Күзкәем, Күзкәем,
Җитмәс кебек җылы сүзкәем.

Минзәләгә, бөтен тирә-якка
Йолдыз булып балкый Күзкәем.
Бәхет биргән җирем, әй Күзкәем,
Ил алдында якты йөзкәең.

Күзкәем, Күзкәем,
Ил алдында якты йөзкәең.
```

# Перевод

### Рамазан Байтимеров / Әнвәр Бакиров - Кузкаем

```
На твоих благодатных полях
Пал мой пот, Кузкаем.
В родной деревне Кузкаем
Пусть не сотрётся мой след.

Кузкаем, Кузкаем,
Пусть не сотрётся мой след.

Трудом упорным день за днём
Хорошеет моя Кузкаем.
Чтобы славить Кузкаем песней,
Тёплых слов, как будто, мало мне.

Кузкаем, Кузкаем,
Тёплых слов, как будто, мало мне.

Минзәләгә, всем окрестным сёлам
Звёздным светом сияет Кузкаем.
Земля, что счастье мне дарила, ах, Кузкаем,
Перед страной лицо твоё светло.

Кузкаем, Кузкаем,
Перед страной лицо твоё светло.
```
