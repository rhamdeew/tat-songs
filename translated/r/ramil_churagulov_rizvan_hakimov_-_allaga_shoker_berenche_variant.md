# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Аллага шөкер! (беренче вариант)

```
Бер максатсыз, гел заяга
Узмасын гомер.
Исәнмесез, яшибез без,
Аллага шөкер!

Кайгы белән, шатлык белән
Уза бу гомер.
Без бит бергә - шунсы шатлык,
Аллага шөкер!

Алла бирсә, җырларбыз без,
Узганчы гомер.
Исәнмесез, җырлыйбыз без,
Аллага шөкер!
```

# Перевод

### Рамил Чурагулов / Ризван Хәкимов - Слава Аллаху! (первый вариант)

```
Без цели, всё зря
Пусть не пролетает жизнь.
Здоровья вам, живём мы,
Слава Аллаху!

С печалью, с радостью
Пролетает эта жизнь.
Мы ведь вместе - в том радость,
Слава Аллаху!

Алла даст, будем петь мы,
Пока живы.
Здоровья вам, поём мы,
Слава Аллаху!
```
