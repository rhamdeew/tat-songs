# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Насретдин

```
Авыр юлда бер ялгызың барганда,
Уйларыңны кара кайгы сарганда,
Ашыкма син бар дөньяны каргарга,
Ул таратыр бөтен кайгы-хәсрәтең...

Насретдин, Насретдин, Насретдин,
Хуҗа Насретдин.

Еллар буе еламасаң, көлмәсәң,
Мәхәббәтнең ни икәнен белмәсәң,
Төннәр буе матур сүзләр эзләсәң,
Төшләреңдә тылсымлы сүз өйрәтер...

Насретдин, Насретдин, Насретдин,
Хуҗа Насретдин.

Әгәр дөнья йөртсә сине «өф» итеп,
Уйлый күрмә, килде тоташ бәхет, дип.
Уйла аны тик өч көнлек тәхет, дип,
Бу турыда сиңа һәрчак искәртер...

Насретдин, Насретдин, Насретдин,
Хуҗа Насретдин.
```

# Перевод

### Разил Валиев / Рәшит Абдуллин - Насретдин

```
Когда один идешь по тяжкому пути,
Когда черная тоска сжимает мысли,
Не спеши ты весь белый свет клясти -
Он рассеет твою печаль и горечь...

Насретдин, Насретдин, Насретдин,
Ходжа Насретдин.

Если годы ты не плакал, не смеялся,
Если не узнал, что значит быть любимым,
Если ночи напролет слова искал ты -
Во сне научит слову волшебному...

Насретдин, Насретдин, Насретдин,
Ходжа Насретдин.

Если мир тебя посадит на «уф» престол,
Не подумай, будто счастье навсегда пришло.
Думай: это трон всего на три дня,
И об этом он всегда тебе напомнит...

Насретдин, Насретдин, Насретдин,
Ходжа Насретдин.
```
