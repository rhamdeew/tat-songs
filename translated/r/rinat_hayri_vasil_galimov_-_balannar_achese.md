# Оригинал

### Ринат Хәйри / Вәсил Галимов - Баланнар әчесе

```
Яз иде без барып урманга
Тапканда мәхәббәт акланын.
Тик икәү күрдек без аланда
Баланның чәчкәләр атканын.

Жәй тагын килдек без урманга
Күрергә мәхәббәт аланын,
Кызарып пешсен дип, өзмәдек
Ямь-яшел тәлгәшен баланның.

Мин хәзер ялгызым. Югалттым.
Табалмыйм мәхәббәт аланын.
Шулай да әчесе ник миңа.
Тик миңа калган соң баланның?
```

# Перевод

### Ринат Хайри / Васил Галимов - Горечь калины

```
Весною пошли мы в лес,
Отыскав любви поляну.
Лишь вдвоём увидели тогда,
Как цветёт калина рьяно.

Летом вновь пришли мы в лес,
Чтоб увидеть ту поляну,
Чтоб краснели ягоды скорей,
Не срывали гроздь калины алой.

Я теперь один. Я потерял.
Не найти любви поляны.
Почему ж так горько мне сейчас,
Коли мне досталась лишь калина?
```
