# Оригинал

### Равил Исаев / Мөфтәдин Гыйләҗев - Киек казлар (өченче вариант)

```
Каңгылдашып күктән
Киек казлар үтә.
Казлар күңеле сизә көз аен.
Казлар сагыныр фәкать
Җылы язны гына,
Мин бит сагынам сине көн саен.

Киек казлар күктән
Шуңа күнеп беткән,
Яшәү өчен күчү бер гадәт.
Мин сөюем белән,
Янып яшим гелән,
Януыма бит иркәм син сәбәп.

Яз кояшы балкыр,
Казлар кабат кайтыр,
Туган күлләр өчен талпынып.
Шул казларга карап,
Синсез еллар санап,
Сүнәр микән сөю ялкыным.
```

# Перевод

### Равил Исаев / Муфтадин Гилазев - Журавли (третий вариант)

```
Клином курлыш в небе
Журавли пролетают.
Сердца журавлей чувствуют осень.
Журавли скучают лишь
По тёплой весне,
А я скучаю по тебе каждый день.

Журавли в небе
К этому привыкли,
Перелёт - для жизни их привычка.
А я со своей любовью,
Всегда живу, горя,
Ты ведь причина, мой милый, моему горению.

Весеннее солнце сияет,
Журавли вернутся опять,
Стремясь к родным озёрам.
Глядя на тех журавлей,
Считая годы без тебя,
Погаснет ли пламя моей любви.
```
