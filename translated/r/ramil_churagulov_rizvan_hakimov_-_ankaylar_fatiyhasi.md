# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Әнкәйләр фатыйхасы

```
Чәчкәләрдәй назлап безне
Газиз әнкәй үстергән.
Чал төшмәгән, еллар аңа
Ап-ак калфак кидергән.

Әнкәйләрнең уй-теләге -
Һәрчак ярдәмче.
Юлга чыкма әнкәйләрдән
Фатыйхалар алмыйча.

Әнкәемнең ак чәчләре -
Ап-ак чәчкәләр.
Гомер биргән, иман биргән
Безгә әнкәйләр.

Әнкәйләрнең уй-теләге -
Һәрчак ярдәмче.
Юлга чыкма әнкәйләрдән
Фатыйхалар алмыйча.

Мәтрүшкәле чәйләр белән
Һәрчак көтә аналар.
Суынмасын самавырлар,
Кайнап торсыннар алар.

Әнкәйләрнең уй-теләге -
Һәрчак ярдәмче.
Юлга чыкма әнкәйләрдән
Фатыйхалар алмыйча.
```

# Перевод

### Рамил Чурагулов / Ризван Хәкимов - Материнское благословение

```
Нас, как цветы, лелея нежно,
Дорога мать взрастила.
Седина ей не легла,
Годы белый калфак надели.

Мысли, желания матерей -
Вечно нам поддержка.
В путь не выходи, не взяв
Материнского благословения.

Белые волосы мамы моей -
Словно белые цветы.
Жизнь дала и веру дала
Нам материнская любовь.

Мысли, желания матерей -
Вечно нам поддержка.
В путь не выходи, не взяв
Материнского благословения.

С чаями с матрёшкой
Матери всегда нас ждут.
Пусть не стынут самовары,
Пусть кипят они всегда.

Мысли, желания матерей -
Вечно нам поддержка.
В путь не выходи, не взяв
Материнского благословения.
```
