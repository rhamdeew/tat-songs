# Оригинал

### Ризида Мөхәммәтҗанова / Элмира Латыйпова - Кыз хыялы

```
Теләгем зур язгы кичтә
Тамчы булып тамарга.
Кереп татлы төшләреңә,
Серләреңне табарга.

Йоклап ятканда йөрәгең
Пәрдәсенә кагарга.
Юллар күрсәтеп төшеңдә,
Маяк булып янарга

Кояш нуры булып көн дә
Җылытырга җаныңны.
Үзгәреш юк, нигә шулай
Кыйныйм икән аңымны?..

Карыйм хыял көчем белән
Зиһенеңне яктыртып.
Шуннан соң да карамасаң,
Йөр(е)мәм башка ялкытып.
```

# Перевод

### Ризида Мухамметжанова / Элмира Латыйпова - Девичья мечта

```
Желанье мое - в весенний вечер
Каплей на землю падать.
В сладкие сны твои проникнуть,
Твои тайны отгадать.

Когда ты спишь, к завесе сердца
Тихонько прикоснуться,
В грезах дорогу показать,
Маяком светиться.

Солнечным лучом каждый день
Согревать твою душу.
Нет перемен - отчего же так
Терзает разум мой?.. 

Взглядом мечты освещаю
Твой светлый ум.
А если и после ты не взглянешь -
Больше я не вспыхну.
```
