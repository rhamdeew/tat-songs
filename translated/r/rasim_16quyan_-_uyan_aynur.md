# Оригинал

### rasim_16quyan_ - Рәсим / 16quyan - Уян, Айнур

```
Уяна бар дөнья,
Син дә йоклап ятма.
Кояш сиңа дәшә:
Уян, Айнур!

Хәерле иртә, Айнур!
Хәерле иртә, Айнур!

Йолдызлар да сүнде,
Ай да инде ятты.
Барсы сиңа дәшә:
Уян, Айнур!
```

# Перевод

### rasim_16quyan_ - Проснись, Айнур

```
Просыпается весь мир,
Ты тоже не спи, не лежи.
Солнце тебе говорит:
Проснись, Айнур!

Доброе утро, Айнур!
Доброе утро, Айнур!

И звёзды уже погасли,
И луна легла на покой.
Все тебе говорят:
Проснись, Айнур!
```
