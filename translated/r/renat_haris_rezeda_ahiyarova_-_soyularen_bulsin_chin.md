# Оригинал

### Ренат Харис / Резеда Ахиярова - Сөюләрең булсын чын

```
Якты йөзең белән яктырттың
Язмышымның моңсу даласын.
Минем өчен мәңге-мәңгегә
Син кояшым булып каласың...

Син сихерчем минем,
Тылсымчым!
Сөюләрең генә
Булсын чын!

Назлы күзең белән назладың
Күзләремнең дымлы камышын.
Әйтәләм хәзер ышанып -
Шатлык белән инде таныш мин.

Син сихерчем минем,
Тылсымчым!
Сөюләрең генә
Булсын чын!

Җылы сүзең белән җылыттың
Күңелемнең бәсле көннәрен.
Чәчәк атты инде мәңгегә
Бәхетемнең алсу гөлләре.

Син сихерчем минем,
Тылсымчым!
Сөюләрең генә
Булсын чын!
```

# Перевод

### Ренат Харис / Резеда Ахиярова - Пусть любовь будет настоящей

```
Светлым лицом своим озарил
Ты грустную равнину моей судьбы.
Для меня на веки вечные
Ты останешься моим солнцем...

Ты моё волшебство,
Мой чародей!
Пусть лишь любовь твоя
Будет настоящей!

Нежным взором своим баловал
Влажные камыши моих глаз.
Теперь могу я сказать с верой -
С радостью я уже знакома.

Ты моё волшебство,
Мой чародей!
Пусть лишь любовь твоя
Будет настоящей!

Тёплым словом своим согрел
Тревожные дни моей души.
Расцвели навсегда
Розовые цветы моего счастья.

Ты моё волшебство,
Мой чародей!
Пусть лишь любовь твоя
Будет настоящей!
```
