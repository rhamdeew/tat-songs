# Оригинал

### Рәфыйк Юныс / Мәсгудә Шәмсетдинова - Айдала

```
Айдалада ай нуры да, Айдалада таң нуры,
Ай нуры да, таң нуры, атым тоттым туп-туры.

Айдалада зəңгəр күккə чөйдем соры карчыга,
Карчыгам, карчыгам очты кояш каршына.

Айдалада йөргəн чакта аттым кара капланны,
Капланны, капланны аттым-ектым - капланды.

Айдалада ак кылганнар айкала да чайкала.
Айдала, Айдала, Айдалада эз кала.
```

# Перевод

### Рафыйк Юныс / Мәсгудә Шәмсетдинова - Айдала

```
В Айдале - свет луны, в Айдале - свет рассвета,
Свет луны и свет рассвета - конь мой мчится прямо в это.

В Айдале, в синеву, взмыл серый мой карчига,
Карчига, мой карчига, к солнцу он взлетел навстречу.

В Айдале, в путь пустившись, бил я черного каплана,
Каплана, каплана - сразил и повалил, пал каплан.

В Айдале белогривые качаются, колышутся.
Айдала, Айдала - в Айдале след ложится.
```
