# Оригинал

### Роберт Шәймәрданов - Хушлаштык

```
Язның матур бер көнендә
Кинәт очраштык.
Айлы кичләргә сокланып
Ныклап дуслаштык.

Хисләр хискә уралынгач
Без бит кавыштык.
Тормыш йөген сөйрәгәндә
Ник соң ялгыштык?

Гомеребез агышында
Юкка орыштык.
Үкенәчәкне уйламый
Тиздән хушлаштык...

Хисләр хискә уралынгач
Без бит кавыштык.
Тормыш йөген сөйрәгәндә
Ник соң ялгыштык?
```

# Перевод

### Роберт Шаймарданов - Расстались

```
В один прекрасный весенний день
Вдруг повстречались.
Лунными вечерами любуясь,
Крепко подружились.

Чувства в чувства сплелись -
Ведь мы сошлись.
Таща жизненный груз,
Отчего же ошиблись?

В теченье наших лет
Зря ссорились.
Не думая о сожаленье,
Скоро простились...

Чувства в чувства сплелись -
Ведь мы сошлись.
Таща жизненный груз,
Отчего же ошиблись?
```
