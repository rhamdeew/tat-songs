# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Агыгыз сулар кирегә (Ага сулар)

```
Ага сулар, ага сулар,
Ага сулар туктамый.
Гомер үтә, йөрәк көтә,
Кавышырга җай чыкмый,

Агыгыз сулар кирегә -
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.

Яшел үләнле аланда
Уеннар матур иде.
Безнең яшьли сөйгән ярлар
Башкалар яры инде.

Агыгыз сулар кирегә -
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.

Бәхетең үзең кулыңда,
Сайланып күп йөремә.
Яшьли сөйгән үз ярыңны
Берүк ятларга бирмә.

Агыгыз сулар кирегә -
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.
```

# Перевод

### Рамил Чурагулов / Роберт Тимербаев - Текут воды назад (Текут воды)

```
Текут воды, текут воды,
Текут воды без остановки.
Жизнь проходит, сердце ждёт,
Не встретиться нам вовремя.

Текут воды назад -
К молодости.
Там меня любившая половинка
Всё ещё ждёт.

На зелёной лужайке
Игры были прекрасными.
Наших юношей любившие девушки
Теперь другие невесты.

Текут воды назад -
К молодости.
Там меня любившая половинка
Всё ещё ждёт.

Счастье в твоих руках,
Много не выбирай.
Юношей любившую свою невесту
Чужим постель не отдавай.

Текут воды назад -
К молодости.
Там меня любившая половинка
Всё ещё ждёт.
```
