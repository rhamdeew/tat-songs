# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Төсле җыр

```
Нигә күк һәрвакыт зәп-зәңгәр, гел зәңгәр?
Сагышлы зәңгәрлек сибелә күзләрдән?

Сөйсәң син әгәр дә, -
Сагыш бар зәңгәрдә.
Зәңгәр күк, зәңгәр күз, зәңгәр моң...
Мәхәббәт сагыштан тора ул.

Нигә таң көн саен алсу гел, гел алсу?
Таңнар да нигә соң һәрвакыт ямансу?

Сөйсәң син әгәр дә, -
Таң - алтын мәгарә.
Алсу кук, алсу йөз, алсу моң...
Мәхәббәт шатлыктан тора ул.

Зәңгәр күк, алсу таң, якты көн, кара төн...
Син мине, ахрысы, мәңгегә караттың.

Төсләрне тараттың...
Мин сине яраттым.
Сөюем - бәхеткә илткән юл...
Сагышлы шатлыктан тора ул.
```

# Перевод

### Разил Вәлиев / Рәшит Абдуллин - Цветная песня

```
Почему же небо вечно сине-синее, всё синее?
Печальная синева из глаз струится, веет?

Если любишь ты -
В синеве живёт печаль.
Синее небо, синие глаза, синяя грусть...
Любовь из тоски соткана у нас.

Почему же рассвет каждый день всё алый, алый?
И почему же зори постоянно грустноваты?

Если любишь ты -
Рассвет - как золотая пещера.
Алый небосклон, алый лик, алый напев...
Любовь из радости соткана у нас.

Синее небо, алый рассвет, светлый день, чёрная ночь...
Ты, видно, навсегда меня окрасил.

Краски расплескал...
Я тебя любила.
Моя любовь - дорога к счастью...
Она из грустной радости соткана.
```
