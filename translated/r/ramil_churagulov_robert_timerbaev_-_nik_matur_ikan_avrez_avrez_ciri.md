# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Ник матур икән Әврез!? (Әврез җыры)

```
Түбәнге Әврез авылы
Дөньяда берәү генә.
Аның бары берәүлеген
Беләм, бар җиһан белә.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм - Әврез матур,
Таһир, Зөһрәләр белән!

Хан кызыннан чибәрерәк
Әврезем гүзәлләре.
Кыйгач кашлы, нечкә билле,
Кирәкми бүтәннәре.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм - Әврез матур,
Таһир, Зөһрәләр белән!

Бер типкәндә тимер өзәр
Егетләр дә бар бездә.
Күз салмагыз сез аларга,
Бик кирәк үзебезгә.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм - Әврез матур,
Таһир, Зөһрәләр белән!
```

# Перевод

### Рамиль Чурагулов / Роберт Тимербаев - Как красива же Аверез! (Песня Авереза)

```
Село Аверез-Тюбень
В мире только одно.
Его единственность эту
Я знаю, весь мир знает.

Что ж, друг, если наше село
Такой красивый, скажешь,
Секрет скажу - Аверез красив
С Таиром, Зухрой!

Краше, чем дочь хана,
Девушки моего Авереза.
С бровями стрелками, тонкой талией,
Другие не нужны.

Что ж, друг, если наше село
Такой красивый, скажешь,
Секрет скажу - Аверез красив
С Таиром, Зухрой!

Мужественнее железа
У нас есть и парни.
Не пяльтесь вы на них,
Очень нужны нам.

Что ж, друг, если наше село
Такой красивый, скажешь,
Секрет скажу - Аверез красив
С Таиром, Зухрой!
```
