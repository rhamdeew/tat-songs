# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Алма сатам, чия сатам...

```
Алма сатам, алма сатам,
Алмасыз калма тагы.
Алма кебек ярлар белән
Каршылыйк әле таңны.

Хөрмә сатам, хөрмә сатам,
Хөрмәсез кала күрмә.
Хөрмә кебек чибәрләрнең
Урыны була түрдә.

Чия сатам, чия сатам,
Чиясез кермә өйгә.
Күңелләрем чияләнмәс
Җырласаң безнең көйгә.

Алма сатам, хөрмә сатам,
Хисләрем кушып сатам.
Минем өчен җанын биргән
Бер чибәрне яратам.
```

# Перевод

### Разил Вәлиев / Луиза Батыр-Болгари - Яблоки продаю, вишни продаю...

```
Яблоки продаю, яблоки продаю,
Не оставайся вновь без яблок ты.
С милыми, что румяны, как яблоко,
Встретим вместе рассвета лучи.

Финики продаю, финики продаю,
Не дай остаться миру без финика.
Красавицам, как финик, пристало
Быть на почетном месте, вгордости.

Вишни продаю, вишни продаю,
Без вишни не входи в мой дом.
Сердца не заалеют вишней,
Если ты запоешь наш напев.

Яблоки продаю, финики продаю,
Чувства свои впридачу продаю.
Я люблю одну красавицу,
Что за меня отдала бы жизнь.
```
