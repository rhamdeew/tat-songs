# Оригинал

### Рәшит Нигъмәти / Заһир Исмәгыйлев - Җиңүчеләргә дан

```
Яу ораны салды Ватан,
Ат атланды батырлар -
Ил дошманнарын (явыз дошманны) кыйратып,
Яудан җиңеп кайтырлар.

Һай, асыл ил уллары,
Яудан җиңеп кайтыгыз.
Һай, алтын ил кызлары,
Зур дан белән балкыгыз.

Яуга озаттык ирләрнең
Асылын, батырларын.
Яуга озаттык кырларның
Алтынын, матурларын.

Һай, асыл ил уллары,
Яудан җиңеп кайтыгыз.
Һай, алтын ил кызлары,
Зур дан белән балкыгыз.

Яз кояшы кебек балкып,
Батырлар кайтты яудан.
Яз ташкыны кебек ургып,
Җырлары килә алдан.

Дан сезгә, ил уллары,
Ватан җырлый сезгә дан!
Дан сезгә, ил кызлары,
Хөрмәтли сезне Ватан!

Доншманны җиңде егетләр
Җиңделәр ил кызлары.
Даннары кояш кебекләр
Алтыннан йолдызлары.

Дан сезгә, ил уллары,
Ватан җырлый сезгә дан!
Дан сезгә, ил кызлары,
Хөрмәтли сезне Ватан!
```

# Перевод

### Рәшит Нигъмәти / Заһир Исмәгыйлев - Слава победителям

```
Родина клич бросила в бой,
Встали кони, встали герои -
Вражьих недругов (злых врагов) сокрушат,
С победой с поля вернутся домой.

Ай, сыны земли родной,
С победой из боя вернитесь.
Ай, дочери, золото края,
Великою славой сияйте.

В бой проводили мужчин -
Лучших, отважных, достойных.
В бой проводили полей
Золото, красоты живые.

Ай, сыны земли родной,
С победой из боя вернитесь.
Ай, дочери, золото края,
Великою славой сияйте.

Как весеннее солнце - светло,
Герои вернулись из боя.
Как весенний разлив - широко,
Их песни приходят до строя.

Слава вам, сыны страны,
Родина поёт вам славу!
Слава вам, дочери страны,
Родина вас чтит по праву!

Врага одолели ребята,
И дочери страны победили.
Славы их - словно солнце,
Звёзды их - из чистого злата.

Слава вам, сыны страны,
Родина поёт вам славу!
Слава вам, дочери страны,
Родина вас чтит по праву!
```
