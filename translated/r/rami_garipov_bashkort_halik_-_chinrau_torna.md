# Оригинал

### Рәми Гарипов / Башкорт халык - Чыңрау торна

```
Чыңрый-чыңрый очып килә
Чыңрау торнакай.
Уралыңнан сәлам диде
Чыңрау торнакай.

Уралкаем, Уралкаем,
Минем туган Уралкаем!
Исән-сау, ди, Урал-тау, ди
Чыңрау торнакай.

Юк хафа, ди, юк афәт, ди
Чыңрау торнакай.
Илдә ир-ат - Салават, ди
Чыңрау торнакай.

Уралкаем, Уралкаем,
Минем туган Уралкаем!
Ил көйләр ди, тел сөйләр ди,
Уйнар курайкай.

Куанычым, юанычым
Булды Уралкай.
Җыр канатлы, нур канатлы
Чыңрау торнамдай.

Уралкаем, Уралкаем
Минем туган Уралкаем!
Буатлардан буатларга
Очып баргандай.
```

# Перевод

### Рами Гарипов / Башкирская народная - Звонкий журавль

```
Курлыча-курлыча летит
Звонкий журавушка.
С Урала привет передал
Звонкий журавушка.

Мой Урал, мой Урал,
Мой родной Урал!
Жив-здоров, мол, Урал-гора, мол,
Звонкий журавушка.

Нет печали, мол, нет беды, мол,
Звонкий журавушка.
В краю герой - Салават, мол,
Звонкий журавушка.

Мой Урал, мой Урал,
Мой родной Урал!
Народ поёт, мол, язык звучит, мол,
Играет курай.

Радость моя, утешенье моё
Стал мой Урал.
С крыльями песни, с крыльями света,
Как звонкий мой журавль.

Мой Урал, мой Урал,
Мой родной Урал!
Из поколенья в поколенье
Словно улетая.
```
