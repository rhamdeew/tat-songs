# Оригинал

### Рамил Гыймрани - Мин ышанам

```
Күк йөзендә талгын гына очкан
Торна тавышларын ишетәм.
Сагыш белән үзем ялгыз калам
Шул торналар белән син китсәң.

Сынаулар бит һәркемгә дә килә,
Түзәргә шул, җаным, көч кирәк.
Мәхәббәтең ачы җилләренә
Түзгәннәре була бик сирәк.

Өмет сүнми кошларың да кайтыр,
Сагыш кына җанны өзгәли.
Мин ышанам берсен-берсе сөйгән
Җаннар генә түзә беләдер.
```

# Перевод

### Рамил Гыймрани - Я верю

```
В небесах, паря тихонько в выси,
Слышу крик печальных журавлей.
С грустью я один останусь в тиши,
Если ты уйдешь с их белых стай.

Испытанья всем даются в мире,
Выдержать их, милый, нужен дар.
В горьких ветрах нашей любви
Выстоят немногие, как жаль.

Не угаснет свет - вернутся птицы,
Только грусть душе моей болит.
Я верю: кто друг друга любит,
Лишь те умеют пережить.
```
