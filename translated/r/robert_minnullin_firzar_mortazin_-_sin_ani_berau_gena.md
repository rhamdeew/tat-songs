# Оригинал

### robert_minnullin_firzar_mortazin_ - Син, әни, берәү генә

```
Безнең хакта гына, әни,
Уйланасыңдыр инде.
Борчыласыңдыр инде син,
Куанасыңдыр инде.

Без диеп шатландың да син,
Без диеп сагышландың.
Бөтен гомереңне, әни,
Син безгә багышладың.

И, көясеңдер инде син,
И, янасыңдыр инде...
Төннәрен дә безне уйлап
Уянасыңдыр инде.

Син - йолдызларның яктысы,
Гөлләрнең хуш ислесе.
Әйбәт әниләр күп лә ул,
Юк шул синең төслесе.

Безгә сиздермичә генә
Елап аласың инде.
Киткән саен безгә бәхет
Юрап каласың инде.

Аз инде, аз, әни, сиңа
Рәхмәтләр әйтү генә.
Син бит, әни, безнең өчен
Бер генә - берәү генә!
```

# Перевод

### Роберт Миңнуллин / Фирзәр Мортазин - Ты, мама, одна-единственная

```
Только о нас, мама,
Думаешь ты, видно, снова.
Беспокоишься, конечно,
Радуешься ты, конечно.

Радовалась слову «мы»,
Слову «мы» грустила ты.
Всю свою судьбу, мама,
Ты нам посвятила прямо.

Ах, томишься ты уже,
Ах, горишь ты в тишине...
Даже ночью, нас подумав,
Пробуждаешься ты снова.

Ты - сиянье всех созвездий,
Аромат у всех цветов.
Много добрых мам на свете,
Но таких, как ты, нет вовсе.

Не давая нам заметить,
Плачешь тихо ты порою.
Уходя, нам на дорогу
Счастье шепчешь и пророчишь.

Мало, мало, мама, просто
Говорить тебе «спасибо».
Ты ведь, мама, для нас в жизни
Одна-единственная!
```
