# Оригинал

### Рәис Риянов / Флүр Әхмәтсафин - Чәчәктә чакта сирень

```
Сиреньнәрнең чәчәкләре
Керфек каккан чагында,
Торасың син иләсләнеп
Татлы сер кочагында.

Сиреньнәрне син бәхетле
Булыйм дип үстерәсең;
Һәр чәчәктә назлы өмет
Чаткыларын күрәсең.

Шундый сылу күренәсең
Сөйгәнгә сине сирень.
Ак сирень чәчәге кебек
Иң гүзәл чагың синең.

Ак сиреньнең ямьнәрендә
Чагыла саф хисләрең.
Сине алиһәгә тиңләп
Ярата дус-ишләрең.
```

# Перевод

### Раис Риянов / Флюр Ахметсафин - Когда цветёт сирень

```
Когда цветки сирени
Раскроют свои лепестки,
Очутишься ты в сладких объятьях
Таинственной мечты.

Сирень ты растешь с надеждой,
Что счастье она принесёт;
В каждом цветке нежную искру надежды
Ты увидеть поспешит.

Ты кажешься столь прекрасной
Для того, кто любит сирень.
Как цветок белой сирени -
Твой прекраснейший момент.

В очаровании белой сирени
Сияют твои чистые чувства.
Приравнивая тебя к богине,
Любят тебя друзья-товарищи.
```
