# Оригинал

### Рәшит Гарипов / Ләбиб Айтуганов - Безнең күңелләр

```
Ничәнче кат инде (тагын) язлар килде
Чәчәкләргә җирне күмделәр.
Һаман шулай бер күрергә зарыгып
Яшәр микән безнең күңелләр?!..

Яннарыңа очып барыр идем -
Безнең куллар канат түгелләр...
Шул якларга кошлар очкан саен
Ник талпына икән күңелләр?
```

# Перевод

### Рашит Гарипов / Ләбиб Айтуганов - Наши души

```
В который раз уже (снова) весны пришли
Землю цветами укрыли.
Всё так же, одной той мечтой устав,
Не жить ли нашим душам?!..

К тебе бы полетела я -
Но руки наши - не крылья...
Когда туда птицы летят
Почему тоскуют души?
```
