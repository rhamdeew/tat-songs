# Оригинал

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Яңгыр (икенче вариант)

```
Кешеләр яңгыр көткән,
Яумаган яңгыр күптән,
Яңгыр кирәк, дигәннәр,
Яусын тизрәк, дигәннәр.

Яңгыр ялындырмаган,
Көттереп тә тормаган,
Яңгыр ява башлаган,
Шактый явып ташлаган.

Ә кешеләр өркешеп,
Этешеп тә төртешеп,
Чатырларын ачканнар,
Чатыр чабып качканнар.

Ә яңгыр үпкәләгән,
Әз генә сипкәләгән,
Бер-ике тамган аннан,
Тиз генә китеп барган.
```

# Перевод

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Дождь (второй вариант)

```
Люди дождь давно ждали,
Дождь не падал много дней,
«Нужен дождь», - они твердили,
«Пусть прольется поскорей».

Дождь не стал томить молчаньем,
Не заставил ждать себя,
Дождь внезапно начался,
Щедро ливнем поливая.

А люди, всполошившись,
Толкаясь и толкнувшись,
Зонты свои раскрыли,
С зонтом бегом укрылись.

А дождь обидой вспыхнул,
Чуть-чуть всего накапывал,
Раз-два упал - и тут же
Он быстро прочь ушел.
```
