# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Бер гомер җитәр микән? (Серләрем)

```
Ярларына килеп җиткәч,
Дулкыннар тынар микән?
Серләремне сөйләр идем
Ул мине тыңлар микән?

Диңгезләрне кемнәр кичә -
Канатлы кошлар микән?
Диңгезләрнең ике яры -
Безнең язмышлар микән?

Диңгезләрдә давыл чыкса
Яшенләп үтәр микән?
Бер мәхәббәт табар өчен
Бер гомер җитәр микән?
```

# Перевод

### Разил Вәлиев / Рәшит Абдуллин - Хватит ли одной жизни? (Мои тайны)

```
Когда к берегам подойду,
Утихнут ли волны тогда?
Я тайны свои бы поведал -
Услышит ли он меня?

Кто море способен пройти -
Лишь птицы крылатые, да?
Два берега моря - не наши ль
Судьбы, разделенные навсегда?

Если в море восстанет буря,
Молнией пронесется она?
Чтоб отыскать одну любовь,
Хватит ли жизни одна?
```
