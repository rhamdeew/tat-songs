# Оригинал

### Рамил Чурагулов - Без солдатлар

```
Гүзәл илем, гүзәл Ватаныма,
Ябырылса кансыз дошманнар.
Акты тамчы, изге канын түгеп
Саклап калган аны бабамнар.

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга - изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Ил-Ватанга биргән изге антны,
Үтеп чыгу, ул бит егетлек.
Көнне төнгә ялгап, без хөзмәттә,
Дөньяларда булсын иминлек!

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга - изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Маңлайларда кызыл йолдыз яна,
Сары каеш безнең билләрдә.
Үсеп җитеп без дә салдат булдык -
Ризык булсын туган илләрдә!

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга - изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Өйдә көтсен безне әткәй, әнкәй,
Тогры дуслар, сөйгән җан-ярлар.
Җырлап, гөрләп яшәү безгә язсын,
Имин-аман булып дөнъялар.

Без солдатлар, безнең биргән антлар -
Ил-ватанга изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат.
```

# Перевод

### Рамиль Чурагулов - Мы солдаты

```
Прекрасный мой край, прекрасная Отчизна,
Если нападут бесчувственные враги.
Капля за каплей, святую кровь проливая,
Защитили её наши деды.

Мы солдаты,
Наши клятвы
Стране-Отчизне - священная доверенность.
Мы настоящие мужчины, мы - орлы,
Мы настоящие соколы, мы - истинные солдаты!

Сдержать священную клятву, данную Стране-Отчизне,
Исполнить её - это и есть мужество.
День связывая с ночью, мы на службе,
Да будет покой во всем мире!

Мы солдаты,
Наши клятвы
Стране-Отчизне - священная доверенность.
Мы настоящие мужчины, мы - орлы,
Мы настоящие соколы, мы - истинные солдаты!

На челе сияет красная звезда,
Желтые ремни на наших поясах.
Вырос и возмужал я тоже стал солдатом -
Да будет благополучие на родных землях!

Мы солдаты,
Наши клятвы
Стране-Отчизне - священная доверенность.
Мы настоящие мужчины, мы - орлы,
Мы настоящие соколы, мы - истинные солдаты!

Пусть дома ждут нас отец, мать,
Верные друзья, любимые сердцем.
Пусть с песнями, с радостью будет написана нам жизнь,
В безопасности и мире пусть живут миры.

Мы солдаты, наши клятвы -
Стране-отчизне священная доверенность.
Мы настоящие мужчины, мы - орлы,
Мы настоящие соколы, мы - истинные солдаты.
```
