# Оригинал

### Рәис Гыймадиев / Виталий Агапов - Көзге туйлар

```
Кошлар сайраудан туктагач,
Алтын көзләр дә җитә.
Якынайган көзге туйлар,
Кыз күңелен җилкетә.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр...

Шаулатып көзге авылны,
Каз өмәләре үтте.
Көмеш кыңгырау чыңлатып,
Кавышыр көннәр җитте.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр...

Ак мунчаның идәненә
Каз мамыгы сибелгән.
Өмәче кыз киләчәге
Яшь егеткә бәйләнгән.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр...
```

# Перевод

### Раис Гимадиев / Виталий Агапов - Осенние свадьбы

```
Когда умолкнут птичьи трели,
Златая осень к нам придёт.
Осенних свадеб ожиданье
Девичье сердце взволнует.

Обещанья уже даны,
Приняты все подарки.
Возле домов, где живут невесты,
Видны сватов фигурки...

Прошумев по осенней деревне,
Проплыли гусиные стаи.
Позванивая серебряным колокольцем,
Дни встреч настали.

Обещанья уже даны,
Приняты все подарки.
Возле домов, где живут невесты,
Видны сватов фигурки...

На половице белой бани
Гусиный пух рассыпан.
Будущее девушки-мастерицы
С молодым джигитом связано.

Обещанья уже даны,
Приняты все подарки.
Возле домов, где живут невесты,
Видны сватов фигурки...
```
