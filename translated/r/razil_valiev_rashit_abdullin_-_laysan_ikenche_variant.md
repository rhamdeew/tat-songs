# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Ләйсән (икенче вариант)

```
Ләйсән яңгырлар яуганда,
Бар кешеләр көлешә.
Җирдә бөтен кешегә дә
Көлү шундый килешә.

Яу, яңгырым, яу, Ләйсән!
Дөнья язга әйләнсен!
Дөнья язга әйләнсен.
Җырлар җырга бәйләнсен!

Ләйсән яңгыр бар дөньяга
Бәхет, шатлык өләшә.
Җиргә, илгә, кешеләргә
Бәхет шундый килешә.

Яу, яңгырым, яу, Ләйсән!
Дөнья язга әйләнсен!
Дөнья язга әйләнсен.
Җырлар җырга бәйләнсен!
```

# Перевод

### Разил Вәлиев / Рәшит Абдуллин - Ләйсән (второй вариант)

```
Когда идет лейсана дождь,
Смеются все вокруг.
На всей земле любому
Так к лицу веселый смех.

Лейся, дождик, лейся, Ләйсән!
Пусть весною станет мир!
Пусть весною станет мир,
Пусть к песне песня льнется!

Дождь Ләйсән по всей планете
Счастье, радость раздает.
Для земли, страны, людей
Счастье так идет к лицу.

Лейся, дождик, лейся, Ләйсән!
Пусть весною станет мир!
Пусть весною станет мир,
Пусть к песне песня льнется!
```
