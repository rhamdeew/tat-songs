# Оригинал

### Ринат Мөслимов / Зөфәр Хәйретдинов - Өйдә буран булмасын

```
Китәм, диеп ник әйтәсең,
Күңелгә сагыш салып?
Китсең алып китәсең бит,
Ярты йөрәгем алып.

Аңлаешып яшиек лә,
Үкенечкә калмасын.
Тышта буран булса булсын,
Өйдә буран булмасын.

Бу дөньяда безнең гомер
Бер генә тапкыр килә.
Аерылгач ни буласын
Тик Ходай гына белә.

Язмышларда ялгышлар күп,
Ашыкма, уйлап кара.
Кабат ялгыш, кабат сагыш,
Гомер бит узып бара.
```

# Перевод

### Ринат Мослимов / Зуфар Хайретдинов - Пусть в доме не будет бурана

```
Зачем ты говоришь: «Уйду»,
Вселяя грусть в душу мне?
Если уйдёшь, то унесёшь
Половину сердца с собой.

Давай жить, понимая друг друга,
Чтобы не было сожалений.
Пусть снаружи буран бушует,
Но пусть в доме бурана не будет.

На этом свете наша жизнь
Приходит лишь единожды.
Что будет после разлуки -
Знает только лишь Бог.

В судьбах так много ошибок,
Не спеши, подумай хорошо.
Опять ошибка, опять печаль,
А жизнь всё проходит мимо.
```
