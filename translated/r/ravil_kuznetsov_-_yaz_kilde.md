# Оригинал

### Равил Кузнецов - Яз килде

```
Яз килде, кояшның балкышы
Күңелгә ювалды.
Ник син кайтмадың, аккошым,
Кайларда югалдың?

Яз килде, карларны эретте,
Юлларны корытты.
Әллә син еракта, аккошым,
Дустыңны оныттың?

Яңадан без туган якларны
Чәчәкләр каплады.
Күңелемдә мин бергә чакларны
Мәңгегә сакладым.

Яз килде, карларны эретте,
Юлларны корытты.
Әллә син еракта, аккошым,
Дустыңны оныттың?
```
