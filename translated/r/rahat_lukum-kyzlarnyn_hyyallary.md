# Оригинал

### Рахат лукум - Кызларнын хыяллары

```
Син дэ уртак белемле лэ,
Мин дэ уртак белемле шул.
Укырга кереп булмагач,
Нишлэтим, нишлэтим шул белемне?

Бас-бас атлаган була,
Урам таптаган була.
Урам таптаган була,
Вакыт житте кызларга
Кияу эзлэп табарга.

Кая барырга, кая гына язарга
Бай татарны табарга?

Энкэй, акча яумас микэн,
Кесэмэ таммас микэн шул?
Маникюр ясатыйм эле,
Коммерсант, коммерсант алмас микэн?

Бас-бас атлаган була,
Урам таптаган була.
Урам таптаган була,
Вакыт житте кызларга
Кияу эзлэп табарга.

Кая барырга, кая гына язарга
Бай татарны табарга?

Алъяпкыч кесэлэре лэ,
Дискотека кичлэре шул.
Без кайчан булырбыз икэн
Бизнесмен, бизнесмен бичэлэре?

Бас-бас атлаган була,
Урам таптаган була.
Урам таптаган була,
Вакыт житте кызларга
Кияу эзлэп табарга.

Кая барырга, кая гына язарга
Бай татарны табарга?
```

# Перевод

### Рахат лукум - Девичьи грёзы

```
И ты со средним образованьем,
И я со средним образованьем тоже.
Не поступила в институт,
Что делать, что делать с этим образованьем?

Топтать будут, топчут,
По улицам будут ходить.
По улицам будут ходить,
Время пришло девушкам
Жениха искать и найти.

Куда идти, куда только податься,
Чтобы богатого татарина сыскать?

Мама, денег не падает ли,
В карман не капает ли мне?
Маникюр сделаю себе,
Коммерсанта, коммерсанта не возьму ли в мужья?

Топтать будут, топчут,
По улицам будут ходить.
По улицам будут ходить,
Время пришло девушкам
Жениха искать и найти.

Куда идти, куда только податься,
Чтобы богатого татарина сыскать?

Маленькие сумочки да,
Дискотечные вечера эти.
Когда же мы станем
Бизнесменок, бизнесменок жёнами?

Топтать будут, топчут,
По улицам будут ходить.
По улицам будут ходить,
Время пришло девушкам
Жениха искать и найти.

Куда идти, куда только податься,
Чтобы богатого татарина сыскать?
```
