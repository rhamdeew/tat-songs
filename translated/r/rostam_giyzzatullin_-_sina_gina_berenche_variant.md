# Оригинал

### Рөстәм Гыйззәтуллин - Сиңа гына (беренче вариант)

```
Юлларымда мин очраттым
Сине, сине генә,
Бәхетем син, пар канатым,
Сиңа, сиңа гына.
Юлларымда мин яраттым
Сине, сине генә;
Мәхәббәтем, күз карашым
Бары сиңа гына.

Сиңа гына бүләк итәм
Күнелемне сөюемне.
Эгәр бик озкка китсәм,
Алып китәрмен сине үзеңне.

Күзләремдә мин үзгәрдем
Синең белән генә,
Авырлыкны күтәрермен
Синең белән генә.
Мәхәббәтем, җылы йөзем
Сиңа, сиңа гына.
Җырларымны бүләк итәм
Сиңа, сиңа гына.

Сиңа гына бүләк итәм
Күнелемне сөюемне.
Эгәр бик озкка китсәм,
Алып китәрмен cине үзеңне.
```

# Перевод

### Рустам Гиззатуллин - Только тебе

```
На своём пути я встретил
Тебя, только тебя,
Ты моё счастье, мои крылья,
Тебе, только тебе.
На своём пути полюбил
Тебя, только тебя;
Моя любовь, мой взгляд -
Всё только для тебя.

Только тебе дарю
Своё сердце и любовь.
Если уйду далеко,
Заберу с собой тебя саму.

В твоих глазах я изменился,
Только с тобой,
Тяготы понесу
Только с тобой.
Моя любовь, моё тепло -
Тебе, только тебе.
Свои песни дарю
Тебе, только тебе.

Только тебе дарю
Своё сердце и любовь.
Если уйду далеко,
Заберу с собой тебя саму.
```
