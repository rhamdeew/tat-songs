# Оригинал

### Ринат Батталов / Марат Гобәйдуллин - Ак калфак (беренче вариант)

```
Ак калфагың чиккән, бөккән,
Ука белән чигелгән.
Ай калфак, ак калфак
Сезгә бигрәк килешә,
Әйдә, җырлыйк янәшә.

Тырыш икән, матур икән
Безнен татар кызлары.
Ай калфак, ак калфак
Дәртләндерми кем җанын,
Мәңге матур бул, җаным.

Безнен татар эшчәннәре
Тирә-якта мактала.
Ай калфак, ак калфак,
Ак төсле ул, кар төсле,
Бер гөнаһсыз жан төсле.

Милләт кигән ак калфакны
Баш очыннан төшермә.
Ай калфак, ак калфак
Һәр ел саен шушылай
Куандыр безне, Ходай!
```

# Перевод

### Ринат Батталов / Марат Гобайдуллин - Белое покрывало (первый вариант)

```
Твое белое покрывало сшито, сложено,
Прошито иголочкой.
Луна покрывала, белое покрывало
Вам особенно идёт,
Давайте петь вместе.

Усердные, красивые
Наши татарские девушки.
Луна покрывала, белое покрывало
Не тревожит ничью душу,
Всегда будь красива, душа моя.

Наши татарские мастерицы
Во всех окрестностях прославлены.
Луна покрывала, белое покрывало,
Оно белого цвета, снежного цвета,
Безгрешного цвета.

Белое покрывало, что народ носит,
Не снимай с головы.
Луна покрывала, белое покрывало
Каждый год так же
Радует нас, Аллах!
```
