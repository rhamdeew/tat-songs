# Оригинал

### Рузилә Гатауллина / Фирзәр Мортазин - Гомер ике килми

```
Гомер ике килми, беребез дә белми
Кайчан өзеләсен гомернең.
Ул бит мизгел генә, яшик гади генә
Бүгенгенең белеп кадерен.

Эреләнеп йөрмә, соңга кала күрмә
Кадерләргә газиз кешеңне.
Бүген ярат, ярат... Сагышларын тарат
Бүген бир син барлык җылыңны.

Уфтанмыйча гына, зарланмыйча гына
Яшик әле мизгел - гомерне.
Үкенмәслек булсын, якты эзең калсын
Нурга күм син һәрбер көнеңне.

Эреләнеп йөрмә, соңга кала күрмә
Кадерләргә газиз кешеңне.
Бүген ярат, ярат... Сагышларын тарат
Бүген бир син барлык җылыңны.

Гомер ике килми, беребез дә белми
Кемгә күпме гомер бирелгән.
Бүген яшә, яшьнә...Иртәгәсе көндә
Ниләр булачагын кем белгән...

Эреләнеп йөрмә, соңга кала күрмә
Кадерләргә газиз кешеңне.
Бүген ярат, ярат... Сагышларын тарат
Бүген бир син барлык җылыңны.
```

# Перевод

### Рузилә Гатауллина / Фирзәр Мортазин - Жизнь не приходит дважды

```
Жизнь дважды не приходит, не знает никто из нас,
Когда оборвется жизни нить.
Это лишь миг короткий - живи проще сейчас,
Цену сегодняшнего дня храни.

Не растворяйся в суете, не опоздай понять,
Как дорожить любимым, родным.
Сегодня люби, люби... Рассеяй его печаль,
Сегодня отдай ему все тепло.

Без вздохов и без жалоб
Давай проживем этот миг - эту жизнь.
Пусть не будет сожалений, светлый след оставь,
Светом наполни каждый свой день.

Не растворяйся в суете, не опоздай понять,
Как дорожить любимым, родным.
Сегодня люби, люби... Рассеяй его печаль,
Сегодня отдай ему все тепло.

Жизнь дважды не приходит, не знает никто из нас,
Кому и сколько жизни дано.
Живи сегодня, вспыхни... А в завтрашний час
Кто знает, что будет суждено...

Не растворяйся в суете, не опоздай понять,
Как дорожить любимым, родным.
Сегодня люби, люби... Рассеяй его печаль,
Сегодня отдай ему все тепло.
```
