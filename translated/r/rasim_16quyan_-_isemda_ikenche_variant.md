# Оригинал

### rasim_16quyan_ - Рәсим / 16quyan - Исемдә (икенче вариант)

```
Исемдә минем һаман,
Ерак ара үтсәм дә.
Хәтерләрдән чыкмый
Елларның ачы януы.

Үтә тормыш су кеби,
Кулдагы яралар юып.
Эзләр калдыра тирән
Әрнеп беткән җаннарда.

Кайтмыячакмын, аңлыйм аны.
Тик оныта алмыйм шуны:
Тәрәзәдән төшкән утларын
Туган йортымның.
```

# Перевод

### rasim_16quyan_ - Исемдә (второй вариант)

```
Все помнится мне поныне,
Хоть дальние прошли пути.
Не выжечь из памяти
Горечь прожитых годов.

Течет жизнь, как быстрая вода,
Смывая раны с ладоней.
Но оставляет глубокий след
В душах, истомленных болью.

Не возвращусь - я это знаю.
Но не забыть мне одного:
Огни, упавшие из окон
Родного дома моего.
```
