# Оригинал

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Кар бәйрәме

```
Кар агыла, кар агыла
Безнең авыл ягына,
Əй ява кар, өр-яңа кар,
Безнең авыл ягына.

Кар бəйрəме, кар бəйрəме,
Куаныч кар булганда,
Ак кызлар да, ак малайлар
Уйный безнең урамда.

Кирəк түгел Кар бабай да
Явып торгач кар болай,
Һəр кыз бүген Кар кызы бит,
Һəрбер малай - Карбабай.
```

# Перевод

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Праздник снега

```
Снег струится, снег струится
К нашему селу с небес,
Эй, идет снег, новый снег
К нашему селу с небес.

Праздник снега, праздник снега,
Радость - если снег идет,
Белые девчонки, белые мальчишки
Играют у нас на улице.

Не нужен нам и Дед Мороз,
Если сыплет так снегопад,
Каждая девочка нынче Снегурочка,
Каждый мальчик - Дед Мороз.
```
