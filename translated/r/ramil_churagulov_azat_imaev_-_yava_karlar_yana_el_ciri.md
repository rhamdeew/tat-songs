# Оригинал

### ramil_churagulov_azat_imaev_ - Рамил Чурагулов / Азат Имаев - Ява карлар (Яңа ел җыры)

```
Күбәләктәй карлар ява,
Яңа елның кар төбендә.
Яңа еллар имин килсен,
Бәндәләрнең мәркеменә.

Бәхет, шатлык, муллык теләп
Күктән ява ап-ак карлар.
Имин килсен Яңа еллар,
Җырлап торсын бу дөньялар.

Тормышларга ашсын иде
Һәркемнең дә хыяллары.
Яратсын, сөйсен һәркемне
Сөйгәне һәм якыннары.

Бәхет, шатлык, муллык теләп
Күктән ява ап-ак карлар.
Имин килсен Яңа еллар,
Җырлап торсын бу дөньялар.

Карлар түгел, бәхет ява,
Шатлык ява һәркем өчен.
Ашыра чынга теләкләрне
Яңа елның сихри киче.

Бәхет, шатлык, муллык теләп
Күктән ява ап-ак карлар.
Имин килсен Яңа еллар,
Җырлап торсын бу дөньялар.
Имин торсын бу дөньялар...
```

# Перевод

### Рамил Чурагулов / Азат Имаев - Идут снега (Новогодняя песня)

```
Снег, как бабочки, кружится
У порога Новых лет.
Пусть они придут с миром
В дом людской, где живет свет.

Счастья, радости, достатка
Небо шлет нам белый снег.
Пусть придут с добром года,
Пусть поет весь этот свет.

Пусть в жизнь каждого взлетят
Все мечты и все желания.
Пусть любимые и близкие
Дарят нежность и признания.

Счастья, радости, достатка
Небо шлет нам белый снег.
Пусть придут с добром года,
Пусть поет весь этот свет.

Не снега - а счастье падает,
Радость льется для людей.
Станут явью наши просьбы
В дивный вечер Новых дней.

Счастья, радости, достатка
Небо шлет нам белый снег.
Пусть придут с добром года,
Пусть поет весь этот свет.
Пусть благословен будет этот свет...
```
