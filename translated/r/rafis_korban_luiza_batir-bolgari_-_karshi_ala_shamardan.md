# Оригинал

### Рафис Корбан / Луиза Батыр-Болгари - Каршы ала Шәмәрдән

```
Ай калка урман артыннан
Шәмәрдәннең күгендә.
Шәмәрдәннең айлы киче
Кайларда да күңелдә.

Шәмәрдәнгә дуслар килсә,
Кунак булып шәһәрдән,
Күгенә аен кабызып
Каршы ала Шәмәрдән.

Шәмәрдән кичләре моңлы,
Сагынгангадыр инде.
Шәмәрдәндә узган гомер
Моңнар тулы җыр инде.

Шәмәрдәнгә дуслар килсә,
Кунак булып шәһәрдән,
Күгенә аен кабызып
Каршы ала Шәмәрдән.

Шәмәрдән туган җиребез,
Гомер кичергән урын.
Кошлар түгел, дуслар бүген
Җырлый Шәмәрдән җырын.

Шәмәрдәнгә дуслар килсә,
Кунак булып шәһәрдән,
Күгенә аен кабызып
Каршы ала Шәмәрдән.

-
* Саба районының Шәмәрдән ит комбинаты директоры Нигъмәтҗанов Илдар Фәтхерахманович заказы буенча язылган.
```

# Перевод

### Рафис Корбан / Луиза Батыр-Болгари - Встречай, Шемардан

```
Луна восходит за лесом
В небе над Шемарданом.
Лунной ночью в Шемардане
В каждом сердце находит дом.

Если в Шемардан приедут друзья,
Гостями из города,
Зажигая на небе луну,
Встречай, Шемардан, их!

Вечера в Шемардане так напевны,
Ты скучаешь по ним уже.
Прожитая в Шемардане жизнь
Теперь песня, полная тоски.

Если в Шемардан приедут друзья,
Гостями из города,
Зажигая на небе луну,
Встречай, Шемардан, их!

Шемардан - наша родная земля,
Место, где мы живём.
Не птицы, а сегодня друзья
Поют песню Шемардана.

Если в Шемардан приедут друзья,
Гостями из города,
Зажигая на небе луну,
Встречай, Шемардан, их!

-
* Написано по заказу директора Шемарданского мясокомбината Сабинского района Ильдара Фатрахмановича Нигматзянова.
```
