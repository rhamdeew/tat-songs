# Оригинал

### Резедә Гобәева / Эльмир Низамов - Улым

```
Мин бәхетнең күзләренә багам,
Кочагыма алам, иркәлим...
Бу дөняның бөтен яме, гаме,
Синдә, балам, синдә икән бит...

Кушымта:
Таянычым син, моңым
Һич тынмас татлы җырым,
Син минем - якты юлым,
Улым, улым...
Өметем син, җан җылым,
Өзелер үзәк кылым...
Ак булсын барыр юлың,
Улым, улым...

Зур күзләрен ачып елмая ул,
Бар да әле аңа өр-яңа,
Үз-үзеңә бүләк ясыйм дисәң,
Бала тудыр икән дөньяга...
```

# Перевод

### Резедә Гобәева / Эльмир Низамов - Сын мой

```
Я гляжу в глаза счастью,
Прижимаю к себе, нежно лелею...
Вся краса и вся боль этого мира -
В тебе, малыш, в тебе же она...

Припев:
Ты опора моя, мой напев,
Мой несмолкаемый сладкий напев,
Ты - мой светлый путь,
Сын мой, сын мой...
Ты - надежда моя, тепло души,
Порвется жила сердца...
Пусть чист будет твой путь,
Сын мой, сын мой...

Откроет большие глаза - и улыбается он,
Все для него еще внове,
Если хочешь себе сделать подарок -
Роди ребенка на свет...
```
