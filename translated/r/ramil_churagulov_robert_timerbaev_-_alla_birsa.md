# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Алла бирсә

```
Кырык эшләр кырам диеп,
Һаваланма син, бәндәм.
Тауларны да күчерерсең
Көч-гайрәт булса тәндә.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.

Алла бирсә генә көнең
Кояшлы бәйрәм булыр.
Алла бирсә, офыгыңнан
Кара болыт таралыр.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.

«Алла бирсә», «бисмилла»сыз
Эшләнгән эшең хәрәм.
Йотам дигән ризыгыңны
Йота алмассың, Адәм.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.
```

# Перевод

### Рамил Чурагулов / Роберт Тимербаев - Если Аллах даст

```
Сорок дел переделаю,
Не хвались ты, человече.
И горы ты сдвинешь,
Если сила есть в плечах.

«Если Аллах даст» - слова эти
Говори с каждым шагом.
Каждому моему деянью
Будь доволен, Боже мой.
«Если Аллах даст» - не забывай
С каждым шагом, душа моя.

Если Аллах даст, твой день
Солнечным праздником станет.
Если Аллах даст, с небосклона
Чёрная туча рассеется.

«Если Аллах даст» - слова эти
Говори с каждым шагом.
Каждому моему деянью
Будь доволен, Боже мой.
«Если Аллах даст» - не забывай
С каждым шагом, душа моя.

«Если Аллах даст», без «бисмиллы»
Сделанное дело - грех.
Свой удел, что хочешь проглотить,
Не проглотишь, о Адам.

«Если Аллах даст» - слова эти
Говори с каждым шагом.
Каждому моему деянью
Будь доволен, Боже мой.
«Если Аллах даст» - не забывай
С каждым шагом, душа моя.
```
