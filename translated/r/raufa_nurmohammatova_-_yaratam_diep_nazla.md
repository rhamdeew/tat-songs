# Оригинал

### Рәүфә Нурмөхәммәтова - Яратам диеп назла

```
Китсәң - озакка китмә (ә)ле,
Синсез өйнең юк яме.
Күңелемә тапмыйм тыңгы,
Чәйләрнең юк бер тәме.

Яратам диеп назла,
Сөюләремә коен.
Синсез башка, бәгырем,
Яшәүләре, әй, кыен.

Тормышымда көтмәгәндә
Өмет нурларын сиптең.
Нурландырып бар дөньяны,
Мине бәхетле иттең.

Яратам диеп назла,
Сөюләремә коен.
Синсез башка, бәгырем,
Яшәүләре, әй, кыен.

Гомер үрләренә, иркәм,
Без җитәкләшеп меник.
Күзләргә күз карашып,
Икәү генә бирешик.

Яратам диеп назла,
Сөюләремә коен.
Синсез башка, бәгырем,
Яшәүләре, әй, кыен.
```

# Перевод

### Рауфа Нурмухаметова - Называю, что люблю

```
Если уйдёшь - не уходи надолго,
Без тебя дом не дом.
Сердцу не найти покоя,
У чаев нет ни вкуса.

Называю, что люблю,
В поцелуи влюблена.
Без тебя, моя душа,
Жить, скажу, тяжело.

В жизни моей неожиданно
Ты рассыпал лучи надежды.
Освещающий весь мир,
Ты сделал меня счастливой.

Называю, что люблю,
В поцелуи влюблена.
Без тебя, моя душа,
Жить, скажу, тяжело.

По тропам жизни, свободно,
Мы идём, держась за руки.
Глядя в глаза друг другу,
Вдвоём только сливаемся.

Называю, что люблю,
В поцелуи влюблена.
Без тебя, моя душа,
Жить, скажу, тяжело.
```
