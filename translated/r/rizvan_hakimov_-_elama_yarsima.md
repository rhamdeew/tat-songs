# Оригинал

### Ризван Хәкимов - Елама, ярсыма

```
Ашмады гамәлгә
Ант итеп бирешкән вәгъдәләр.
Елама, ярсыма -
Син үзең гаепле барсына.

Үтте ул, сүнде ул
Үзебез кабызган мәхәббәт.
Елама, ярсыма -
Син үзең гаепле барсына.

Саф идең, белмәдең,
Син үзең, гаепсез, көнләдең.
Елама, ярсыма -
Син үзең гаепле барсына.

Болытлар таралыр,
Ышаныч кыскартыр араны.
Елама, ярсыма -
Елмаеп кил, иркәм, каршыма.
```

# Перевод

### Ризван Хәкимов - Не плачь, не бушуй

```
Не стали былью
Клятвами скрепленные обещанья.
Не плачь, не бушуй -
Ты сама во всем виновата.

Прошло оно, угасло
Нами же разожженное чувство.
Не плачь, не бушуй -
Ты сама во всем виновата.

Чистой была, не знала,
И без вины ты сама ревновала.
Не плачь, не бушуй -
Ты сама во всем виновата.

Облака разойдутся,
Доверие сократит расстояние.
Не плачь, не бушуй -
С улыбкой иди мне навстречу, родная.
```
