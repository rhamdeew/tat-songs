# Оригинал

### Рушания Габбасова / Рифкать Сәйфетдинов - О, Сөембикә!

```
Гасырлардан яңгырый һаман
Кайтаваздай - Сөембикә!
Казан ханлыгының даны
Аның белән ерак китә.

Сөембикә, Сөембикә,
Татарның гүзәл кызы!
Сөембикә, Сөембикә!
Син - азатлык йолдызы!

Кара болыт - дошман явы -
Татар көрәшкә кузгала.
Иленә, сөйгән халкына
Сөембикә тугры кала.

Сөембикә, синең исемең
Гасырларга күчеп бара.
Син татарның горурлыгы,
Сөембикә, хатын, ана.
```

# Перевод

### Рушания Габбасова / Рифкать Сәйфетдинов - О, Сөембикә!

```
Сквозь века звучит поныне,
Эхом - Сөембикә!
Слава Казанского ханства
С нею вдаль уходит века.

Сөембикә, Сөембикә,
Ты - прекрасная дочь татар!
Сөембикә, Сөембикә!
Ты - звезда свободы и дар!

Чёрной тучей - вражья сила -
Татар к борьбе восстает.
Своей земле, любимому народу
Сөембикә верной живет.

Сөембикә, имя твоё
Сквозь века продолжит путь.
Ты - гордость татарского народа,
Сөембикә, жена и мать.
```
