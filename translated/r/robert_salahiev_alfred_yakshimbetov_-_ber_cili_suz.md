# Оригинал

### Роберт Сәлахиев / Альфред Якшимбетов - Бер җылы сүз

```
Күңлемдә йөртеп гел,
Кичләрен уйлансам,
Җанымны сагышлы моң баса.
Януым, көюем
Басыла бары тик,
Сандугач сайравын тыңласам.

Сызылып таң ата,
Гөлләрне уята,
Җиһанга җылы нур бөркелә.
Ал таңнан юанган,
Син генә дип янган,
Булмастыр минем күк беркем дә.

Җылы язлар килсә,
Болыннар яшәрә,
Күңелем очына - түз генә!
Гүя мин талпынам,
Канатым кагынам,
Әйтсәң син бер җылы сүз генә.
Әйтсәң син бер җылы сүз генә...
```

# Перевод

### Роберт Сәлахиев / Альфред Якшимбетов - Одно тёплое слово

```
В сердце ношу всегда,
Вечером задумаюсь -
Душу покрывает грустный напев.
Горенье, боль моя
Утихнут лишь тогда,
Когда соловья послушаю трель.

Рассвет струится алым,
Цветы вокруг будя,
Тепло и свет по миру разливаются.
Зарёй утешенный,
Горю одной тобой -
Нет никого, кто так же сгорал бы.

Как только весны тёплые придут,
Луга зазеленеют,
Душа взлетает - только вытерпеть!
Будто я трепещу,
Крылом своим машу,
Если бы ты сказала одно лишь тёплое слово.
Если бы ты сказала одно лишь тёплое слово...
```
