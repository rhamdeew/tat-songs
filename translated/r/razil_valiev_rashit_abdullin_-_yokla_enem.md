# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Йокла, энем

```
Йокла инде, энем, йокла инде,
Син йокласын диеп кояш сүнде.
Йокла, энем, йокла инде...
Озын юллар үтеп, йокы килде.

Йокла, энем, йокла, күзләрең йом.
Йокы бит ул шундый кызык уен.
Йолдыз керер төшләреңә,
Күзләрең йом, тик ят, дәшмә генә.

Йокла, энем, таңга кадәр йокла,
Кайгырма син урам өчен юкка.
Төне буе явар ак кар,
Урамыңны Кар бабаең саклар.
```

# Перевод

### Разил Вәлиев / Рәшит Абдуллин - Спи, братишка

```
Спи же, братишка, спи безмятежно,
Чтобы ты спал - погасло солнце нежно.
Спи, братишка, спи...
Долгий путь пройдён - пришёл покой ночной.

Спи, братишка, спи, сомкни глаза.
Сон - игра такая, вот какая - чудеса.
Звёзды в сны твои войдут,
Глазки сомкни, просто лёжа, не шепчи ничуть.

Спи, братишка, до рассвета спи,
Не тревожься за улицу - зря не грусти.
Ночью будет белый снег идти,
Улицу твою Дед Мороз хранит.
```
