# Оригинал

### Ришат Рамазанов / Радик Яруллин - Әйдә, дустым

```
Әйдә, дустым, ал гармунны,
Җырлыйк «Җидегән чишмә»не.
Җырга-моңнарга күмелсен
Авылыбыз кичләре.

Әйдә, әйдә, әйдә, әйдә, әйдә, дустым!
Җырлап, уйнап күрсәт гармуның тавышын.
Бергә-бергә бергәләшеп биик әле,
Дөньяларның рәхәтләрен күрик әле!

Эх, кайда сез, матур чаклар,
Яшел болын буйлары?
Бер сөлге өчен көрәшкән
Җәйге сабан туйлары!

Әйдә, әйдә, әйдә, әйдә, әйдә, дустым!
Җырлап, уйнап күрсәт гармуның тавышын.
Бергә-бергә бергәләшеп биик әле,
Дөньяларның рәхәтләрен күрик әле!

Искә алып яшь гомерне,
Моңлы көйләр сузарбыз.
Хәтер сукмаклары буйлап
Җырлар җырлап узарбыз.

Әйдә, әйдә, әйдә, әйдә, әйдә, дустым!
Җырлап, уййап күрсәт гармуның тавышын.
Бергә-бергә бергәләшеп биик әле,
Дөньяларның рәхәтләрен күрик әле!
```

# Перевод

### Ришат Рамазанов / Радик Яруллин - Айда, друг

```
Айда, друг, возьми гармонь,
Споем «Семь родников».
Пусть утонут в песнях-монох
Вечера наших сел.

Айда, айда, айда, айда, айда, друг!
Пой, играя, покажи гармоньй звонкий̆ звук.
Вместе-вместе, рядом, дружно потанцуем,
Радость мира, все блаженство ощутим мы!

Эх, где вы, годы золотые,
Зеленых лугов края?
Летние сабан-туй,
Где бились за полотенца!

Айда, айда, айда, айда, айда, друг!
Пой, играя, покажи гармоньй звонкий̆ звук.
Вместе-вместе, рядом, дружно потанцуем,
Радость мира, все блаженство ощутим мы!

Вспоминая юность нашу,
Протянем песнй грусть.
По тропинкам памятй
Песней̆-песней̆ пройдем, как путь.

Айда, айда, айда, айда, айда, друг!
Пой, играя, покажи гармоньй звонкий̆ звук.
Вместе-вместе, рядом, дружно потанцуем,
Радость мира, все блаженство ощутим мы!
```
