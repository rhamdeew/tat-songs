# Оригинал

### Римзил Вәлиев / Алик Локманов - Кыңгыраулар (беренче вариант)

```
Сөлге таккан көмеш дуга,
Дугада - кыңгыраулар.
Ак каеннар хәйран калып,
Тып-тын тыңлап торалар.

Кыңгыраулар, кыңгыраулар ...
Юлларга сибелгәннәр. / Сөлгегә чигелгәннәр. /
Юл читендәге үләнгә
Энҗедәй сибелгәннәр ...

Кыңгыраулы атлар белән
Кыңгыраулы таңнарда,
Туйга түгел, яуга ирләр
Киткән хәтәр чакларда!

Киткәннәр дә, кайтмаганнар -
Яу бит ул, уенмы ни! ...
Кыңгыраулы тайлар гына
Болында уйнап йөри ...
```

# Перевод

### Римзил Вәлиев / Алик Локманов - Колокольчики (первый вариант)

```
С полотенцем серебряная дуга,
А на дуге - колокольчики.
Белые березы, зачарованные,
Слушают в тиши, не шелохнутся.

Колокольчики, колокольчики...
По дорогам рассыпаны. / На полотенце вышиты. /
В придорожной траве
Рассыпаны, словно жемчуг...

С конями в колокольчиках
В колокольные рассветы,
Не на свадьбу - в бой мужчины
Уходили в грозный час!

Ушли да не вернулись -
Ведь война, не игра же!...
Лишь жеребята с колокольчиками
В лугах играют да бегают...
```
