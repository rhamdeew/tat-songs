# Оригинал

### Рәми Гарипов / Ренат Фаттахов - Әнкәем (беренче вариант)

```
Әнкәем, бәгърем, алдарак китсәм,
Кайгырма һич минем китгәнгә.
Кайгырсаң, тик кайгыр минем бары
Ак каенны әнкәм иткәндә.

Баш очымда минем утыртсыннар
Кайгысы ак булганы ак каен.
Күз яшеңне коймый, минем белән
Бер көн килеп сөйләш, яз сайын.

Мин әйтермен, сабый яфрак булып,
Бер генә сүз: "Әнкәй! - күз нурым".
Ак каенны сыйпап тыңла бер җыр,
Ак каенда - минем күз нурым.
```

# Перевод

### Рами Гарипов / Ренат Фаттахов - Мама моя (первый вариант)

```
Мама, радость моя, если раньше уйду,
Не печалься о том, что ушёл я вдали.
Если горе придёт - пусть о горе скорбит
Только белая берёза, ставшая матерью мне.

Надо мною пусть посадят
Белую берёзу - белой грусти знак.
Слёзы не роняя, со мной побеседуй
Хоть один раз, каждую весну.

Я скажу, став младенческим листком,
Лишь одно: «Мама - свет моих глаз».
Белой берёзы коснись - и слушай песню,
На белой берёзе - свет моих глаз.
```
