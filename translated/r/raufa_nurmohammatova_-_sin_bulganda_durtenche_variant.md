# Оригинал

### Рәүфә Нурмөхәммәтова - Син булганда (дүртенче вариант)

```
Һәр көнне уянып иртән,
Тыңлыйм сандугач моңын.
Ул да сөя, яратадыр,
Каршылый жырлап таңын.

Күңел тулган синең белән,
Атыла, таша ярыннан.
Яшим куанып, бәгърем,
Син дөньяларда барында.

Мин синең белән канатлы,
Күңелем моңнар дөньясы.
Мин синең белән бәхетле,
Матур шуңа да дөньясы.

Күңел тулган синең белән,
Атыла, таша ярыннан.
Яшим куанып, бәгърем,
Син дөньяларда барында.

Гомеребез үтсен бергә,
Дәртле моңга уралып.
Яшик сагынып-сагындырып,
Бер-беребездән җылы алып.

Күңел тулган синең белән,
Атыла, таша ярыннан.
Яшим куанып, бәгърем,
Син дөньяларда барында.
```

# Перевод

### Рәүфә Нурмөхәммәтова - Когда ты рядом (четвертый вариант)

```
Каждое утро, проснувшись,
Слушаю трель соловья.
Он тоже любит, он тоже нежит,
Песней встречает зарю.

Сердце тобою полно,
Бьется, выходит из берегов.
Живу ликуя, родная,
Пока ты есть на земле.

С тобой я крылатая,
Душа моя - мир напевов.
С тобой я счастливая,
Потому и мир так красив.

Сердце тобою полно,
Бьется, выходит из берегов.
Живу ликуя, родная,
Пока ты есть на земле.

Пусть наша жизнь пройдет вместе,
Вдохновенной мелодией объята.
Жить нам, тоскуя и будя тоску,
Греясь теплом друг друга.

Сердце тобою полно,
Бьется, выходит из берегов.
Живу ликуя, родная,
Пока ты есть на земле.
```
