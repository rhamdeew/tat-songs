# Оригинал

### Рамил Чурагулов / Сабира Идиятуллина-Чурагул - Гәүһәр туй

```
Бер-бер артлы туйлар уза,
Көмеше, алтыннары.
Яшьләр һаман яп-яшь кала,
Ә гомер чаба ары!

Бүген бездә, бүген бездә -
Иң матур туй, затлы туй!
Туйларның иң-иң гүзәле -
Бриллиант туй, гәүһәр туй!
Бокалларга сутлар салып,
Яшьләр өчен эчеп куй!

Озын-озак булсын иде
Мәхәббәт гомерләре.
Һәркемнең алтын-гәүһәр
Туйлары булыр әле!

Бүген бездә, бүген бездә -
Иң матур туй, затлы туй!
Туйларның иң-иң гүзәле -
Бриллиант туй, гәүһәр туй!
Бокалларга сутлар салып,
Яшьләр өчен эчеп куй!

Карт калып була утызда
Һәм яп-яшь - сиксәннәрдә.
Ходай озын гомер бирсен
Сөеп-сөелгәннәргә!

Бүген бездә, бүген бездә -
Иң матур туй, затлы туй!
Туйларның иң-иң гүзәле -
Бриллиант туй, гәүһәр туй!
Бокалларга сутлар салып,
Яшьләр өчен эчеп куй!
```

# Перевод

### Рамил Чурагулов / Сабира Идиятуллина-Чурагул - Бриллиантовая свадьба

```
Свадьбы идут одна за другой,
Серебро, затем золото.
Молодые всё так же молоды,
А жизнь уносится прочь!

Сегодня у нас, сегодня у нас -
Самая красивая свадьба, свадьба знатная!
Самая-прекрасная из всех свадеб -
Бриллиантовая свадьба, свадьба драгоценная!
В бокалы налейте соков,
Выпейте за молодых!

Пусть будут долгими-долгими
Годы любви.
У каждого ещё будут
Золотые и бриллиантовые свадьбы!

Сегодня у нас, сегодня у нас -
Самая красивая свадьба, свадьба знатная!
Самая-прекрасная из всех свадеб -
Бриллиантовая свадьба, свадьба драгоценная!
В бокалы налейте соков,
Выпейте за молодых!

Можно постареть и в тридцать,
И быть юным в восьмидесяти.
Пусть Бог даст долгую жизнь
Тем, кто любит и любим!

Сегодня у нас, сегодня у нас -
Самая красивая свадьба, свадьба знатная!
Самая-прекрасная из всех свадеб -
Бриллиантовая свадьба, свадьба драгоценная!
В бокалы налейте соков,
Выпейте за молодых!
```
