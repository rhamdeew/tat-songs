# Оригинал

### Ризида Мөхәммәтҗанова / Ришат Сәгыйтов - Хатын-кызлар бәйрәме

```
Җан эретер ягымлы да
Хатын-кызлар бәйрәме,
Кемдер сиңа бүләк итәр
Хуш исле гөл бәйләме.

Кушымта:
Кояш көлә, ул да белә -
Соклангыч бәйрәм бүген.
Хатын-кызлар елмаюы
Яулады дөнья күген.

Кояш, сибеп алтын нурын,
Эретә ак карларны,
Бәйрәм белән тәбрикли ул
Гүзәл хатын-кызларны.
```

# Перевод

### Ризида Мөхәммәтҗанова / Ришат Сәгыйтов - Праздник женщин

```
Ласковый, души растаявший -
Праздник женщин в этот день,
Кто-то дарит тебе нежно
Ароматный роз букет.

Припев:
Солнце смеется, оно знает -
Дивный праздник нынче здесь.
Улыбки женщин покорили
Небосвод и мира весь.

Солнце, золотые лучи рассыпав,
Топит белые снега,
С праздником оно поздравит
Женщин светлых и прекрасных.
```
