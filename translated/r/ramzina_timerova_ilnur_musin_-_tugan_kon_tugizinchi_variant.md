# Оригинал

### Рәмзинә Тимерова / Илнур Мусин - Туган көн (тугызынчы вариант)

```
Туган көнең синең, җан дускаем,
Гомереңнең чәчкә чаклары.
Теләкләрнең сиңа иң изгесе -
Йөрәк түрләрендә ятканы.

Чуерташтай чыдам, түземле бул
Тормыш күперләрен үткәндә.
Җан сөйгәнең яннарыңда булсын
Язмыш йомгакларын сүткәндә.

Бәхетләрең телим бер-бер артлы,
Шатлык нуры сирпәп килгәнен.
Күзләреңнең гел балкуын телим,
Елмайганың телим, көлгәнең.

Чуерташтай чыдам, түземле бул
Тормыш күперләрен үткәндә.
Җан сөйгәнең яннарыңда булсын
Язмыш йомгакларын сүткәндә.

Озын гомерле бул, рәхәт яшә,
Кайгы җиле сиңа тимәсен.
Теләр теләкләрең кабул булсын,
Өметләрең һичкем өзмәсен.

Чуерташтай чыдам, түземле бул
Тормыш күперләрен үткәндә.
Җан сөйгәнең яннарыңда булсын
Язмыш йомгакларын сүткәндә.
```

# Перевод

### Рәмзинә Тимерова / Илнур Мусин - День рождения (девятый вариант)

```
День рождения твой, мой друг родимый,
Цветущая пора твоих годов.
Самое святое из желаний -
То, что в сердце бережно живет.

Будь терпелив, как камень, стойким будь,
Когда мосты судьбы проходишь ты.
Пусть рядом будет тот, кого ты любишь,
Когда распутываешь узелки судьбы.

Счастья тебе желаю, снова и снова,
Чтоб светом радость тихо приходила.
Пусть вечно сияют твои глаза,
Пусть улыбаешься и смеешься ты.

Будь терпелив, как камень, стойким будь,
Когда мосты судьбы проходишь ты.
Пусть рядом будет тот, кого ты любишь,
Когда распутываешь узелки судьбы.

Долгих лет тебе, живи в покое,
Пусть ветер горя не коснется тебя.
Пусть сбудутся твои желания,
Пусть надежды никто не оборвет.

Будь терпелив, как камень, стойким будь,
Когда мосты судьбы проходишь ты.
Пусть рядом будет тот, кого ты любишь,
Когда распутываешь узелки судьбы.
```
