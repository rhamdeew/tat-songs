# Оригинал

### Рәмзинә Тимерова / Мөдәрис Газетдинов - Соң инде (өченче вариант)

```
Сагышларга күмеп,
Үзәгемне өзеп,
Моңсуланып миңа карама.
Соң инде, соң инде,
Килмә син яныма.
Тозлар сипмә, зинһар, ярама.
Кабынган учаклар,
Еракта ул чаклар.
Ничек кайтарыплар аласы?!
Сөям сине, димә,
Артларымнан йөрмә,
Кузгатма син күңел ярасын.
Кирәкми сүзләрең,
Ягымлы күзләрең,
Алданмамын кабат, беләмен.
Иң авыры артта:
Саф хисләрне таптап
Узган чакларыңа түзгәнмен.
```

# Перевод

### Рәмзинә Тимерова / Мөдәрис Газетдинов - Поздно уже (третий вариант)

```
В печали утопая,
Сердце разрывая,
Не гляди на меня с тоской.
Поздно уже, поздно,
Не приходи ко мне.
Соль не сыпь, прошу, больной.
Пылавшие костры,
Далёки те поры.
Как их вернуть теперь?!
Не говори «люблю»,
За мною не ходи,
Не тронь мою душевную рану.
Не нужны твои слова,
Твои ласковые глаза,
Я снова не поверю, я знаю.
Самое тяжёлое позади:
Топча чистые чувства,
Я терпела твои прошедшие дни.
```
