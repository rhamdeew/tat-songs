# Оригинал

### Рамил Чурагулов / Сабира Идиятуллина-Чурагул - Сагынам туган якны

```
Челтерәп аккан чишмәләр
Тик туган якта гына.
Туган якны хурламыйбыз,
Син аны макта гына.

Кычытканлы, тигәнәкле,
Чүп үләнле, баткаклы...
Шайтан таягы үссә дә,
Сагынам туган якны.

Кәкре булсыннар каеннар,
Чокыр-чакырлы юллар...
Сине дә чит-ят якларда
Бик күпләр сагыналар.

Кычытканлы, тигәнәкле,
Чүп үләнле, баткаклы...
Шайтан таягы үссә дә,
Сагынам туган якны.

Җырлый белсә, белмәсә дә,
Һәркем үзенчә җырлый.
Җырның ертыгы булмый да
Туган як ямьсез булмый.

Кычытканлы, тигәнәкле,
Чүп үләнле, баткаклы...
Шайтан таягы үссә дә,
Сагынам туган якны.
```

# Перевод

### Рамиль Чурагулов / Сабира Идиятуллина-Чурагул - Скучаю по родному краю

```
Журчащие ручьи бегут
Лишь в родных местах.
Мы не порочим родной край,
Ты только славь его.

Хоть крапива, хоть репейник,
Сорные травы, трясины...
Пусть и шайтанова палка колется,
Я скучаю по родной земле.

Пусть кривыми будут березы,
И дороги в ухабах...
Тебя и в чужих-далеких краях
Многие тоже тоскуют.

Хоть крапива, хоть репейник,
Сорные травы, трясины...
Пусть и шайтанова палка колется,
Я скучаю по родной земле.

Умеет ли, не умеет ли,
Каждый по-своему поет.
У песни не бывает прорех,
Родной край некрасивым не станет.

Хоть крапива, хоть репейник,
Сорные травы, трясины...
Пусть и шайтанова палка колется,
Я скучаю по родной земле.
```
