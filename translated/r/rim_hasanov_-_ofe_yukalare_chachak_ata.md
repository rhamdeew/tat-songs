# Оригинал

### Рим Хәсәнов - Өфе юкәләре чәчәк ата

```
Өфе юкәләре чәчәк ата,
Юкәләрнең матур чаклары.
Юкә белән, гүзәл яшьлек белән,
Бәйләнгән безнең язмышлары.

Өфе юкәләре чәчәк ата,
Яшьлектән җибәрә сәламләр.
Мәңгелекка безне, алар озата,
Коела безнең хәтирәләр,
Җуелмый безнең калган эзләр.

Өфе юкәләре чәчәк ата,
Табигатьнең матур чаклары.
Безнең яшьлек атып узды инде,
Хәтирәләр калды ни бары.

Өфе юкәләре чәчәк ата,
Яшьлектән җибәрә сәламләр.
Мәңгелекка безне, алар озата,
Коела безнең хәтирәләр,
Җуелмый безнең калган эзләр.

Өфе юкәләре чәчәк атар,
Бу дөнъялар үзгәрмәс микән?
Кемдер юкәләрне узган чакта,
Безне дә искә алыр микән?

Өфе юкәләре чәчәк ата,
Яшьлектән җибәрә сәламләр.
Мәңгелекка безне, алар озата,
Коела безнең хәтирәләр,
Җуелмый безнең калган эзләр.
```
