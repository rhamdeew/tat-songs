# Оригинал

### Равил Фәйзуллин / Эльмир Низамов - Бәлки, син

```
Бәлки, син - минем җырланмас
Иң гүзәл җырымдыр?
Бәлки, син - минем чыгалмас
Иң данлы юлымдыр?
Бәлки, син - минем меналмас
Иң биек күгемдер?
Бәлки, син - күтәралмас
Гомерлек йөгемдер?
Бәлки, син - минем беркайчан
кул җитмәс кошымдыр?
Бәлки, син - үзем аңламас
Бәхетем алкышыдыр!
Бәлки, син...
```

# Перевод

### Равил Фәйзуллин / Эльмир Низамов - Возможно, ты

```
Возможно, ты - моя не спетая,
Самая прекрасная песнь?
Возможно, ты - мой недостигнутый,
Самый славный путь?
Возможно, ты - мой недосягаемый,
Самый высокий небосвод?
Возможно, ты - моя непосильная
Пожизненная ноша?
Возможно, ты - моя навек
Неуловимая птица?
Возможно, ты - мой непонятый мною
Аплодисмент счастья!
Возможно, ты...
```
