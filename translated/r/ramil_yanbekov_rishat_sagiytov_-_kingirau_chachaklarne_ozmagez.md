# Оригинал

### Рамил Янбеков / Ришат Сәгыйтов - Кыңгырау чәчәкләрне өзмәгез

```
Кыңгырау чәчәкләрне өзмәгез,
Болыннардан әгәр үтсәгез.
Алар сезгә бәхет теләрләр,
Башларыннан гына сөйсәгез.

Кыңгырау чәчәкләргә багыгыз,
Күзләрегез тулы наз белән.
Алар сезне сагынып көтәрләр,
Чәчәк атып иртә яз белән.

Кыңгырау чәчәкләрнең төсләре,
Зәңгәр күктә, тымык күлләр дә.
Үзәгемне өзә торганы,
Без яртакан зәңгәр күзләр дә.
```

# Перевод

### Рамиль Янбеков / Ришат Сагитов - Не срывайте колокольчики

```
Колокольчики не срывайте,
Если через луга пройдете.
Они счастья вам пожелают,
Лишь головой кивните в ответ.

Колокольчикам улыбнитесь,
Взглядом полным нежности.
Они с тоской вас будут ждать,
Расцветая ранней весной.
Цвета колокольчиков,
В голубом небе, и в тихих озерах.
Рвущие сердце моё,
И нас ранившие голубые глаза тоже.
```
