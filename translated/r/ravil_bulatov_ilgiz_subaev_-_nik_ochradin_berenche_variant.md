# Оригинал

### Равил Булатов / Илгиз Субаев - Ник очрадың? (беренче вариант)

```
Ник очрадың кабат юлларымда
Онытылып барган чагында?
Елмаюың нигә бүләк иттең,
Наз өрмәгәч керсез җаныма?

Инде акыл көтә гел башканы,
Тик күңелдә тагын син генә.
Каршы алам, озатам тик шәүләңне,
Йөз елыймын көнгә, мең көләм.

Горурлыгым йөри гел тирәмдә
Акыл белән көн дә бәхәстә.
Офыкларга карыйм кулым куеп,
Тик күренми анда бернәрсә.
```

# Перевод

### Равил Булатов / Илгиз Субаев - Почему ты встретилась? (первый вариант)

```
Зачем ты вновь на моих дорогах
Встретилась в пору, когда забывалась?
Зачем ты улыбку мне подарила,
Когда в чистую душу мою не вошла ласка?

Ум мой теперь всё другого ждёт,
А в сердце всё равно лишь ты одна.
Встречаю, провожаю только тень твою,
Сто раз на дню рыдаю, тысячу раз смеюсь.

Гордость моя всё бродит рядом,
С разумом каждый день в споре она.
Взираю на горизонты, заслоняясь рукой,
Но там не видно ничего.
```
