# Оригинал

### Разил Вәлиев / Шамил Тимербулатов - Карт өянке

```
Уза гомер, ага сулар,
Яр буенда карт өянке.
Өянкедә сайрый кошлар,
Сайрый өзеп, аһ, йөрәкне.

Чая инеш читкә агар,
Карт өянке ялгыз янар...
Карт өянке күзләреннән,
Инеш булып, яшьләр тамар.

Яр буенда карт өянке.
Ботакларын җил каерган.
Карт өянке төпләрендә,
Сине уйлап, мин кайгырам.

Чая инеш читкә агар,
Карт өянке ялгыз янар...
Карт өянке күзләреннән,
Инеш булып, яшьләр тамар.

Җирдә ялгыз яшәп булмый,
Инеш кибә, тал да корый.
Кайт син миңа, яшьлек ярым,
Инеш кебек урый-урый.

Чая инеш читкә агар,
Карт өянке ялгыз янар...
Карт өянке күзләреннән,
Инеш булып, яшьләр тамар.
```

# Перевод

### Разил Вәлиев / Шамил Тимербулатов - Старый клен

```
Уходит жизнь, бегут воды,
У берега старый клен.
На клене поют птицы,
Поют, разрывая, ах, сердце.

Шустрый ручей в сторону уводит,
Старый клен одиноко горит...
Из глаз старого клена
Ручьем, слезы капают.

У берега старый клен.
Его ветви ветер скрутил.
У корней старого клена
Я грущу, о тебе думая.

Шустрый ручей в сторону уводит,
Старый клен одиноко горит...
Из глаз старого клена
Ручьем, слезы капают.

На земле одному не жить,
Ручей сохнет, и ива вянет.
Вернись ко мне, половина юности,
Кружась, как ручей, кружась.

Шустрый ручей в сторону уводит,
Старый клен одиноко горит...
Из глаз старого клена
Ручьем, слезы капают.
```
