# Оригинал

### Рәсилә Йосыпова / Альберт Сабиров - Булганына шөкер

```
Хәлләремне сорамагыз,
Аңламассыз барыбер.
Һәр кешегә ярап булмый,
Серен сөйләр бу шул без.

Дуслар да бар, дус түгел дә
Бар бит бу дөньяларда.
Якын иткән дус-туганнар
Һәрчак булыр янымда.

Булганына шөкер итәм,
Рәхмәтлемен Ходайга.
Алҗысам да сер бирмимен
Авыр булган чагында.

Дуслар да бар, дус түгел дә
Бар бит бу дөньяларда.
Якын иткән дус-туганнар
Һәрчак булыр янымда.

Яши-яши аңлыйбыз без
Тормышларның асылын.
Башкалардан кер эзләмик,
Бар бит бар үз тормышың.

Дуслар да бар, дус түгел дә
Бар бит бу дөньяларда.
Якын иткән дус-туганнар
Һәрчак булыр янымда.
```

# Перевод

### Рәсилә Йосыпова / Альберт Сабиров - Спасибо за то, что есть

```
О судьбе моей не спрашивайте,
Все равно вам не понять.
Всем угодить невозможно -
Таков наш тайный уклад.

Есть друзья, и есть недруги -
Так устроен этот свет.
Близкие друзья и родные
Будут рядом много лет.

За всё, что есть, я благодарен,
Слава Богу, жизнь дана.
Хоть устану - не выдаю я
Тайн своих в часы темна.

Есть друзья, и есть недруги -
Так устроен этот свет.
Близкие друзья и родные
Будут рядом много лет.

Проживая, мы постигаем
Смысл пути и жизни нить.
В других не ищите пороки -
Ваша жизнь у вас стоит.

Есть друзья, и есть недруги -
Так устроен этот свет.
Близкие друзья и родные
Будут рядом много лет.
```
