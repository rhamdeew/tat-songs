# Оригинал

### Рәфис Мөхәммәтдинов / Фәнир Галимов - Көт, бәгърем (икенче вариант)

```
Саубуллашу җиңел буламы соң -
Тагын калдым юллар чатында.
Кар көртләре күмде юлларыңны,
Озак йөри язган хатың да.

Төшләремдә барам яннарыңа
Күрешергә сузам кулымны.
Сагынасың син дә, сагынасың,
Тоеп яшим синең җылыңны.

Кар-бураннар үтә үзәкләргә,
Юк, кал алмыйм куркып кышлардан.
Көт, бәгърем, барам яннарыңа,
Очар канат алып кошлардан.
```

# Перевод

### Рәфис Мөхәммәтдинов / Фәнир Галимов - Жди, родная (второй вариант)

```
Разве просто нам прощаться?
Вновь стою на перепутье.
Снеговые занесли твои дороги,
И письмо твое все долго не дойдет.

Во снах я к тебе иду,
К встрече руки простираю.
Ты тоскуешь, ты тоскуешь тоже,
Я живу, чувствуя твое тепло.

Вьюги-снега режут сердце,
Нет, не стану, зиму страху не отдамся.
Жди, родная, к тебе иду,
Крылья птиц возьму для полета.
```
