# Оригинал

### Рафаэль Сафин / Хөсәен Әхмәтов - Синдер ул дип уйладым

```
Аланнарны күп буйладым
җырлар җырлап туймадым.
Җырларымның иң яхшысы
синдер ул дип уйладым.

Кырга чыгып, болындагы
гөлгә гашыйк булмадым.
Гөлләрнең дә иң назлысы
синдер ул дип уйладым.

Ерак юлдан сусап килеп,
эчтем (эчәм) чишмә суларын.
Чишмәләр дә баса алмады
йөрәгем ярсуларын.

Чишмәләр дә баса алмагач
йөрәгем ярсуларын,
чишмәләрнең дә чишмәсе
синдер ул дип уйладым.
```

# Перевод

### Рафаэль Сафин / Хөсәен Әхмәтов - Я думал, ты лишь тень

```
Долго блуждал я по полям,
песни пел, но не насытился.
Из всех моих песен лучшей
думал, ты как тень явилась.

На цветок в степи смотрел,
но любовью не влюбился.
Среди всех цветов нежнейший
думал, ты как тень явилась.

С дальней дороги пришёл я,
напился из родника.
Родник не смог утолить
сердца моего боли.

Как родник не смог унять
сердца моего боли,
всё равно думал, ты родник
как тень моя явилась.
```
