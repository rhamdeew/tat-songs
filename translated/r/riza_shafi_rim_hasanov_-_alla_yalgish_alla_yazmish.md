# Оригинал

### Риза Шәфи / Рим Хәсәнов - Әллә ялгыш, әллә язмыш ...

```
Кояш калка да, тулган ай бата,
Ник кавышмыйлар алар офыкта?
Әллә ялгышмы,
Әллә язмышмы, -
Төннәр дә - көндәр арада ята?

Мәктәп еллары калдылар ерак -
Тапмыйча кайттым җир шарын урап.
Әллә ялгышмы,
Әллә язмышмы, -
Яшьлек бит бездә булган дип кунак.

Бала чакларда күршеләр идек,
Сагыну газабын без инде белдек...
Ялгышны җайлап,
Язмышны ялгап
Тагы да бергә булырбыз кебек.
```

# Перевод

### Риза Шәфи / Рим Хәсәнов - То ли ошибка, то ли судьба...

```
Солнце встает, а полная луна садится,
Почему им не сойтись на горизонте?
То ли ошибка,
То ли судьба -
Ночи лежат меж днями на изломе.

Школьные годы остались вдали -
Я обогнул весь шар, да не нашел.
То ли ошибка,
То ли судьба -
Молодость, будто гость, у нас была.

В детстве мы были соседями рядом,
Боль расставаний мы раньше узнали...
Ошибку поправим,
Судьбу свяжем снова -
И, может быть, будем опять вместе, как прежде.
```
