# Оригинал

### Рамил Чурагулов / Эльвира Хаматнурова - Беренче мәхәббәт (өченче вариант)

```
Әби бабай карап кала,
Урамнан уза яшьләр.
Яшьләр йөзендә елмаю,
Картларныкында яшьләр.

Умырзая гомере кебек
Кыска хиснең сәгате.
Өркетмәгез, куркытмагыз
Беренче мәхәббәтне.

Төшләреңә керер әле
Тәүге кабат үбешү.
Янып, көеп, бик сагынып
Көн аралаш күрешү

Умырзая гомере кебек
Кыска хиснең сәгате.
Өркетмәгез, куркытмагыз
Беренче мәхәббәтне.

Тәүге сөю, тәүге хистән
Чын мәхәббәт яралсын
Һәр гашыйк җан бердән берен
Гомерлек ярын тапсын

Умырзая гомере кебек
Кыска хиснең сәгате.
Өркетмәгез, куркытмагыз
Беренче мәхәббәтне.
```

# Перевод

### Рамил Чурагулов / Эльвира Хаматнурова - Первая любовь (третий вариант)

```
Дед с бабушкой смотрят вслед,
Как по улице молодость идёт.
На лицах молодых - улыбки,
А в старых глазах - слёзы.

Как жизнь Умырзаи короток
Час чувства драгоценный.
Не пугайте, не отталкивайте
Первую любовь.

В твоих снах ещё появится
Первый тот поцелуй.
Пылая, сгорая, скучая так сильно,
Встречаясь день ото дня.

Как жизнь Умырзаи короток
Час чувства драгоценный.
Не пугайте, не отталкивайте
Первую любовь.

Из первой любви, из первых чувств
Пусть настоящая любовь родится,
Пусть каждая влюблённая душа
Найдёт свою пару на всю жизнь.

Как жизнь Умырзаи короток
Час чувства драгоценный.
Не пугайте, не отталкивайте
Первую любовь.
```
