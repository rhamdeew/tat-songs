# Оригинал

### Рамил Чурагулов / Рәүф Сәлтәшев - Җырым белән көчлемен мин

```
Хәсрәт килсә йорт илеңә,
Кайгылардан чәчкә сулыр.
Күз яшьләреңне сөртүче,
Юатучым җырым булыр.

Җырым белән көчлемен мин,
Бәхетлемен җырым белән.
Матур җырга тиң, гомерләр җырдан торсын иде,
Гелән җырдан торсын иде, гелән.

Кайгыда да, шатлыкта да,
Кешеләргә җырлар юлдаш.
Кайгы-хәсрәт онытыла,
Күңелгә җыр кереп тулгач.

Җырым белән көчлемен мин,
Бәхетлемен җырым белән.
Матур җырга тиң, гомерләр җырдан торсын иде,
Гелән җырдан торсын иде, җырдан.

Йөрәгемдә җыр йөретәм,
Булдыра алмыйм мин җырламый.
Җырым, моңымнан, дусларымнан,
Аермасын мине Ходай.

Җырым белән көчлемен мин,
Бәхетлемен җырым белән.
Матур җырга тиң, гомерләр җырдан торсын иде,
Гелән җырдан торсын иде, гелән.
```

# Перевод

### Рамиль Чурагулов / Рауф Султанов - Силён я своей песней

```
Если придет горе в твой дом,
От печалей зацветёт.
Тревожный, стирающий слёзы,
Моя песня будет утешать.

Силён я своей песней,
Счастлив я своей песней.
Равняйтесь на прекрасную песню, жизнь из песни должна начаться,
Из песни должна начаться, из песни.

В горе и в радости,
Песни - спутники людям.
Горе-печаль забывается,
Когда песня наполняет душу.

Силён я своей песней,
Счастлив я своей песней.
Равняйтесь на прекрасную песню, жизнь из песни должна начаться,
Из песни должна начаться, из песни.

В сердце песню я ношу,
Не могу я без пения.
Мою песню, мою мелодию, моих друзей,
Пусть Господь не разлучит с ними меня.

Силён я своей песней,
Счастлив я своей песней.
Равняйтесь на прекрасную песню, жизнь из песни должна начаться,
Из песни должна начаться, из песни.
```
