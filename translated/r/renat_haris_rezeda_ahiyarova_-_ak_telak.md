# Оригинал

### Ренат Харис / Резеда Ахиярова - Ак теләк

```
Алтын-алтын яфраклар
Төште, төште учыма.
Алтын-алтын теләкләр
Килделәр тел очыма:

Һәр агачны һәр язда
Яшь яфраклар төрсеннәр.
Кешеләр дә алардан
Гыйбрәт ала күрсеннәр.

Каурый-каурый ак карлар
Кунды, кунды учыма.
Ап-ак, ап-ак теләкләр
Килделәр тел очыма:

Бетмәс-төкәнмәс юллар
Җир тирәли китсеннәр.
Без барасы ак юллар
Ак бәхеткә илтсеннәр.
```

# Перевод

### Ренат Харис / Резеда Ахиярова - Белое пожелание

```
Золотые, золотые листья
Пали, пали мне на ладонь.
Золотые, золотые желания
Пришли на кончик языка:

Чтобы каждое дерево каждой весной
Молодой листвой расцветало.
Чтобы люди, глядя на них,
Урок добра принимали.

Пуховые, пуховые белые снега
Сели, сели мне на ладонь.
Белые, белые желания
Пришли на кончик языка:

Пусть бесконечные дороги
Вокруг земли пролягут.
Наши белые пути
К белому счастью приведут.
```
