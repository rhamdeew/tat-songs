# Оригинал

### Рәфига Усманова / Вәсил Галимов - Сагыну (дүртенче вариант)

```
Җәйнең баллы таңнарында
Сагынам ап-ак карны.
Утлы куздәй йөрәгемә
Шуннан зур дәва бармы?

Сагынам, сагынам ак карны,
Сагынам, сагынам син-ярны.

Давыл эзләп нигә киттем,
Күмелдең ак буранга.
Ни тылсым табып кайтыйм соң
Син калган ак урамга?

Сагынам, сагынам ак карны,
Сагынам, сагынам син-ярны.

Ромашкалар кочагында...
Сагынам ап-ак карны.
Юкларны эзләп адашма,
Югалтма икән барны.

Сагынам, сагынам ак карны,
Сагынам, сагынам син-ярны.
```

# Перевод

### Рафига Усманова / Василь Галимов - Тоска (четвёртый вариант)

```
В летние медовые зори
Тоскую по белому снегу.
По сердцу, пылающему взору
Есть ли от того великое лекарство?

Тоскую, тоскую по белому снегу,
Тоскую, тоскую по тебе, милая.

Почему ушёл в поисках бури,
Исчез в белой метели?
Какое волшебство найдя, вернусь
В белую улицу, где остался ты?

Тоскую, тоскую по белому снегу,
Тоскую, тоскую по тебе, милая.

В середине ромашкового поля...
Тоскую по белому снегу.
Не заблудись в поисках исчезнувших,
Не потеряй, если пойдешь.

Тоскую, тоскую по белому снегу,
Тоскую, тоскую по тебе, милая.
```
