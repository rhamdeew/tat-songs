# Оригинал

### Райлә Хисмәтуллина-Ходайбирдина / Рәзидә Әхмәтова - Моңлы кошчык

```
Моң йөрәктә туа икән,
Җырлана ул йөрәктән.
Мин зар булдым моңнарыңа,
Күзем алмый тирәктән.

Сандугачкай, моңлы кошчык,
Таңда тыңлап уянам.
Бар хәсрәтем юкка чыга,
Сайравыңа куанам.

Моң батшасы сары кошчык,
Алтын да бит зур булмый.
Сандугачсыз җәе булмый,
Тирәгенә кот кунмый.

Сандугачкай, моңлы кошчык,
Таңда тыңлап уянам.
Бар хәсрәтем юкка чыга,
Сайравыңа куанам.
Җырла кошчык, җырла!
```

# Перевод

### Райлә Хисмәтуллина-Ходайбирдина / Рәзидә Әхмәтова - Певчая пташка

```
Моң рождается в сердце,
И поётся он сердцем.
Я истомился по твоим напевам,
Не отвожу глаз от окрест.

Соловейчик, певчая пташка,
На заре, слушая, просыпаюсь.
Вся моя горечь исчезает,
Твоему пению радуюсь.

Царь напевов - рыжая пташка,
И золото ведь не велико.
Без соловья не бывает лета,
И благодать к округе не сойдёт.

Соловейчик, певчая пташка,
На заре, слушая, просыпаюсь.
Вся моя горечь исчезает,
Твоему пению радуюсь.
Пой, пташка, пой!
```
