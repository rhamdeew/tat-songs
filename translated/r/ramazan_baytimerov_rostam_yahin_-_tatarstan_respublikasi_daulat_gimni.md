# Оригинал

### Рамазан Байтимеров / Рөстәм Яхин - Татарстан Республикасы Дәүләт гимны

```
Мәңге яшә, газиз Ватаныбыз,
Халкым тели изге теләкләр!
Гомерлеккә якын туган булып
Яши бездә төрле милләтләр.

Күп гасырлар кичкән чал тарихлы
Данлы илем, үзен бер дастан!
Синдә генә безнең язмышыбыз,
Республикам минем, Татарстан!
```

# Перевод

### Рамазан Байтимеров / Рөстәм Яхин - Государственный гимн Республики Татарстан

```
Вечно живи, Отчизна дорогая,
Народ желает чистых, светлых благ!
Навек родными, близкими мы стали -
У нас живут народы разных стран.

Сквозь вековые годы, в славе старой,
Мой край - как песнь, как вечный эпос дан!
В тебе одна судьба для нас, о дар мой,
Моя республика - Татарстан!
```
