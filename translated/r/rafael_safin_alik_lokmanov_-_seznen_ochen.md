# Оригинал

### Рафаэль Сафин / Алик Локманов - Сезнең өчен

```
Сез дип талпына күңелем,
Дусларым - якыннарым!
Сезнең өчен җырлыймын мин
Җырларның асылларын.

Күкрәгемдә сезнең өчен
Балкыган кояшым бар,
Сезнең өчен иртә - кичен
Сайрап торган кошым бар.

Сез - минем җан күршеләрем,
Үз күргән кешеләрем,
Илем өчен җиргә бәхет
Үстергән кешеләрем.

Сез дип тибә бу йөрәгем,
Дусларым - якыннарым!
Сезнең өчен җырлыймын мин
Җырларның асылларын.
```

# Перевод

### Рафаэль Сафин / Алик Локманов - Для вас

```
К вам тянется сердце мое,
Друзья мои, близкие мне!
Для вас я пою - и живое
Ядро всех напевов во мне.

В груди моей ради вас
Сияет мое солнце светло,
Для вас по утрам и в вечерний час
Поет моя птица тепло.

Вы - соседи моей души,
Люди, кого я узнал,
Для родной земли вы счастье взрастили,
Вы тот, кто ей свет подавал.

Для вас это сердце стучит,
Друзья мои, близкие мне!
Для вас я пою - и звучит
Ядро всех напевов во мне.
```
