# Оригинал

### Ренат Галиәскәр / Илназ Гарипов - Курчаклар

```
Җанны салып калдыр идең
Бер күлмәк төсле.
Тик бу хисләр миннән күпкә
Көчле... үзсүзле...

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар...
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар...

Сикереп төшсәң иде дә
Галәмгә җирдән.
Котылсаң иде мәңгегә
Үтмәс бу чирдән.

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар...
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар...

Язсаң иде сиңа илткән
Өметсез юлдан.
Хисләремә алмаш итеп
Томан... кар... тузан...

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар...
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар...
```

# Перевод

### Ренат Галиәскәр / Илназ Гарипов - Куклы

```
Ты душу бросил, как рубаху,
Небрежно - в сторону.
Но эти чувства, что во мне,
Сильней... упрямей... горьки...

Без тебя вдохновенья нет.
Чужие будут ласкать, обнимать...
Не выдавливай слезы насильно.
Куклы плачут без слез...

Если б прыгнул ты с земли
В бескрайность звёздную,
Если б навсегда ушёл
От этой боли неизлечимой.

Без тебя вдохновенья нет.
Чужие будут ласкать, обнимать...
Не выдавливай слезы насильно.
Куклы плачут без слез...

Если б написал мне
С дороги без надежды,
Подменив мои чувства
Туманом... снегом... пылью...

Без тебя вдохновенья нет.
Чужие будут ласкать, обнимать...
Не выдавливай слезы насильно.
Куклы плачут без слез...
```
