# Оригинал

### Рахмай Хисмәтуллин / Арслан Батыршин - Тот рульне, шофер, тот туры

```
Кыздыра җирне ал кояш нуры,
Эсседән тынган кошларның җыры.
Яңгырата кырны, шаулата кырны
Машиналарның армас моторы.

Әй, рульне, шофер, тот туры,
Көчәйсен, әйдә, шат хезмәт җыры.
Тот рульне, тот туры,
Юлларыңа бәхет яугыры!

Алтындай ашлык чиста һәм коры,
Бу - илебезгә бүләкнең зуры.
Бүләк йөрткәнгә, шатлык илткәнгә
Алда юлларның киңе, такыры.

Әй, рульне, шофер, тот туры,
Көчәйсен, әйдә, шат хезмәт җыры.
Тот рульне, тот туры,
Юлларыңа бәхет яугыры!

Сак үт син, шофер, авыл арасын,
Алышынмасын сизгер карашың.
Машинаңда син колхозчыларның
Илне сөюен алып барасың.

Әй, рульне, шофер, тот туры,
Көчәйсен, әйдә, шат хезмәт җыры.
Тот рульне, тот туры,
Юлларыңа бәхет яугыры!
```

# Перевод

### Рахмай Хисмәтуллин / Арслан Батыршин - Держи руль, шофёр, держи крепче

```
Согревает землю солнца луч,
Песня птиц, что отдыхают от зноя.
Народу поля, шумлют поля,
Моторы машин не устают гудеть.

Эй, держи руль, шофёр, держи крепче,
Пусть звучит громче радостная трудовая песня.
Держи руль, держи крепче,
Пусть счастье сопутствует твоим дорогам!

Золотом хлеб чист и сух,
Это - великий дар нашей стране.
Везя подарок, неся радость,
Впереди дороги широки и гладки.

Эй, держи руль, шофёр, держи крепче,
Пусть звучит громче радостная трудовая песня.
Держи руль, держи крепче,
Пусть счастье сопутствует твоим дорогам!

Езжай осторожно, шофёр, через деревни,
Пусть не отвлекается твой чуткий взгляд.
В своей машине ты колхозников
Любовь к Родине везёшь.

Эй, держи руль, шофёр, держи крепче,
Пусть звучит громче радостная трудовая песня.
Держи руль, держи крепче,
Пусть счастье сопутствует твоим дорогам!
```
