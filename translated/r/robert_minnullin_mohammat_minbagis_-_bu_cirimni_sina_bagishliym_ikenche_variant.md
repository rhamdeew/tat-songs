# Оригинал

### Роберт Миңнуллин / Мөхәммәт Минбагис - Бу җырымны сиңа багышлыйм (икенче вариант)

```
Синең белән узган минутларны
Айга-елларга да алышмим.
Уем белән мин гел синең янда,
Өзгәләнсен әле синең җан да -
Бу җырымны сиңа багышлыйм!

Шатлыкларым күңел тулы иде,
Шатлык идем, инде сагыш мин.
Сиңа ләкин шатлык кына теләп,
Бары сиңа аклык кына теләп,
Бу җырымны сиңа багышлыйм!

Сагынулар, саргаюлар белән,
Югалтулар белән таныш мин,
Син дә мине искә төшерерсең
Сине сагнуымны төшенерсең, -
Бу җырымны сиңа багышлыйм!
```

# Перевод

### Роберт Миңнуллин / Мухаммед Минбагис - Посвящаю тебе эту песню (второй вариант)

```
Минуты, что прошли с тобой,
Я не променяю ни на годы.
Со мной мечта моя всегда с тобой,
Пусть даже душа твоя скорбит -
Посвящаю тебе эту песню!

Радость моя была полна сердца,
Я был счастьем, а теперь тоской.
Но для тебя желаю только счастья,
Только светлых мыслей для тебя -
Посвящаю тебе эту песню!

Знаю я тоску и унынье,
Знаю горечь утраты я.
Если вспомнишь обо мне,
Поймешь тоску мою ты по тебе -
Посвящаю тебе эту песню!
```
