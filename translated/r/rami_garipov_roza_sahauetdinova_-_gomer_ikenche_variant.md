# Оригинал

### Рәми Гарипов / Роза Сәхәүетдинова - Гомер (икенче вариант)

```
Җир җиләккәйләре пеште инде,
Өзелеп тә төшмәс димәгез, -
Ир-егеткәйләрнең яше үтсә,
Бөгелеп тә төшмәс димәгез.

Яфракларны сары алды инде,
Җил шаулатып түкмәс димәгез, -
Үт-яшеңдәй яшьлек гомерләрне
Яшнәп-күкрәп үтмәс димәгез.

Кар сулары шаулап акты инде,
Гел-чәчәкләр атмас димәгез, -
Бишектәге бала, биш төрләнеп,
Ат атланып чапмас димәгез!
```

# Перевод

### Рәми Гарипов / Роза Сәхәүетдинова - Жизнь (второй вариант)

```
Земляника созрела в траве,
Не скажите: не упадет, не сорвется, -
Если годы у парней уходят,
Не скажите: не согнутся, не сломятся.

Листья желтизной охватило давно,
Не скажите: ветер не ссыплет, не сносит, -
Юные годы, как весенний ливень,
Не скажите: не грянут, не пронесутcя.

Талые воды шумом бегут,
Не скажите: цветы не будут всегда, -
Малыш из колыбели, пять раз меняясь,
Не скажите: не помчится, коня оседлав!
```
