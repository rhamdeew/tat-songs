# Оригинал

### Рузәл Минһаҗев / Фирдүс Тямаев - Мәк чәчәге (өченче вариант)

```
Мәк чәчәген өзмәгез,
Узмагыз язмышлардан.
Матур гомер юлыгыз
Тормасын сагышлардан.

Мәк чәчәген өзмәгез,
Тәкыяләр үрмәгез.
Сезгә язган бәхеттән
Ятим кала күрмәгез.
Мәк чәчәген өзмәгез,
Тәкыяләр үрмәгез.
Сезгә язган бәхеттән
Ятим кала күрмәгез.

Гел сокланып карагыз
Чәчәк атканда мәкләр.
Сулар сипсәң аларга,
Кабул була теләкләр.

Мәк чәчәген өзмәгез,
Тәкыяләр үрмәгез.
Сезгә язган бәхеттән
Ятим кала күрмәгез.
Мәк чәчәген өзмәгез,
Тәкыяләр үрмәгез.
Сезгә язган бәхеттән
Ятим кала күрмәгез.
```
