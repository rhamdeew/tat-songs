# Оригинал

### Расих Шамский / Айдар Тимербаев - Миңа инде кабер казылган

```
Берәр шуклык эшләп куйсам әгәр,
Уйламагыз бу, дип, азынган.
Яңа гына яши башласам да (2 тапкыр)
Миңа инде кабер казылган. (2 тапкыр )
Күз алдымда кара кабер тора,
Тактасы да әзер ләхетнең.
Бу дөньядан китеп барам инде, (2 тапкыр)
Татымыйча бәгьзе бәхетне. (2 тапкыр)
Ә яшисем бик-бик килә иде,
Дөнья рәхәт, дөнья матур бит.
Әй, сөеклем, мин кабердә инде, (2 тапкыр)
Өмет итмә, көтмә кайтыр дип. (2 тапкыр)
Өмет итмә, көтмә кайтыр дип...
```

# Перевод

### Рашик Шамский / Айдар Тимербаев - Для меня уже вырыта могила

```
Если бы я совершил какую-то глупость,
Не думайте, что это, прошу вас.
Хоть я и только начал жить (2 раза)
Для меня уже вырыта могила. (2 раза)
Черная могила стоит передо мной,
Даже доска гроба готова.
Я уже ухожу из этого мира, (2 раза)
Не вкусив любовного счастья. (2 раза)
Жизнь шла бы хорошо-хорошо,
Мир спокоен, мир прекрасен.
Да, любимая моя, я уже в могиле, (2 раза)
Не надейся, не жди возвращения. (2 раза)
Не надейся, не жди возвращения...
```
