# Оригинал

### Роберт Миңнуллин / Алик Локманов - Бу җырымны сиңа багышлыйм (беренче вариант)

```
Синең белән узган минутларны
Айлар-елларга да алышмим.
Уем белән мин гел синең янда,
Өзгәләнсен әле синең җан да -
Бу җырымны, бу җырымны,
Бу җырымны сиңа багышлыйм!

Сине уйлап узды гомеремнең,
Сагышларым белән нәкъ эшлим.
Син дә мине сагынырсың әле -
Үткәннәргә сарылырсың әле
Бу җырымны, бу җырымны,
Бу җырымны сиңа багышлыйм!

Шатлыкларым күңел тулы иде,
Шатлык идем, инде сагыш мин.
Сиңа бары шатлык кына теләп,
Сиңа бары аклык кына теләп,
Бу җырымны, бу җырымны,
Бу җырымны сиңа багышлыйм!
```

# Перевод

### Роберт Миннуллин / Алик Локманов - Эту песню тебе посвящаю (первая версия)

```
Минуты рядом с тобой
На месяцы, годы не променяю.
В мыслях я всё время рядом,
Пусть и твоя душа вздрогнет, страдая -
Эту песню, эту песню,
Эту песню тебе посвящаю!

Думая о тебе, прожил я годы,
С тоскою своё ремесло свершая.
Ты ещё будешь скучать обо мне -
За прошлым упрямо держась, вспоминая,
Эту песню, эту песню,
Эту песню тебе посвящаю!

Радости сердцем был переполнен,
Сам был я радостью - теперь я печалью стал.
Только радость тебе желаю, чистоту лишь желаю,
Светлой доли тебе желаю,
Эту песню, эту песню,
Эту песню тебе посвящаю!
```
