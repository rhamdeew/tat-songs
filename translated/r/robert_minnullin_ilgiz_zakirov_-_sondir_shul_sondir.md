# Оригинал

### Роберт Миңнуллин / Илгиз Закиров - Соңдыр шул, соңдыр

```
Тилмереп көткән
Яшьлекләр үткән.
Оныткансыңдыр
Син мине күптән.

Оныткансыңдыр...
Соңдыр шул, соңдыр -
Ялгышлар шулдыр,
Язмышлар шулдыр.

Син инде мине
Сагынмыйсыңдыр.
Ул да сагынмый,
Дип уйлыйсыңдыр.

Сагынмыйсыңдыр...
Соңдыр шул, соңдыр -
Ялгышлар шулдыр,
Язмышлар шулдыр.

Яратмыйсыңдыр...
Яратсаң әгәр,
Килгәләр иде
Синнән бер хәбәр.

Яратмыйсыңдыр...
Соңдыр шул, соңдыр -
Ялгышлар шулдыр,
Язмышлар шулдыр.
```

# Перевод

### Роберт Миңнуллин / Илгиз Закиров - Поздно уж, поздно

```
Томительно ждал я,
Но юность прошла.
Ты, видно, забыла
Меня уж давно.

Ты, видно, забыла...
Поздно уж, поздно -
Ошибки лишь это,
Судьба лишь такова.

Ты больше по мне
Не скучаешь, наверно.
И он не тоскует,
Ты так думаешь верно.

Не скучаешь, наверно...
Поздно уж, поздно -
Ошибки лишь это,
Судьба лишь такова.

Не любишь, наверно...
Если б любила,
От тебя хоть одна
Весть бы пришла.

Не любишь, наверно...
Поздно уж, поздно -
Ошибки лишь это,
Судьба лишь такова.
```
