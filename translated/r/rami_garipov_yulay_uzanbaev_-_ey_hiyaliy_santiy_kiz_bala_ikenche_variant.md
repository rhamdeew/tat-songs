# Оригинал

### Рәми Гарипов / Юлай Үзәнбаев - Эй, хыялый, сантый кыз бала (икенче вариант)

```
Эй, хыялый, сантый кыз бала,
Төн йокыңны синең кем ала?
Белмисең шул ниләр көткәнен,
Аның ниләр өмет иткәнен.

Ул көткәнең килер - кагылыр...
Тик үткәнең синең сагыныр.
Төн йокыңны синең кем ала,
Эй, хыялый, сантый кыз бала?!

Эй, хыялый, сантый кыз бала,
Төн йокыңны синең кем ала?
Белмисең шул ниләр көткәнен,
Аның ниләр өмет иткәнен.
```
