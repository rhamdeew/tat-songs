# Оригинал

### Роберт Миңнуллин / Мәсгудә Шәмсетдинова - Көлке көн

```
Кояш көлə, гел көлə,
Кояш көлгəч көн килə,
Көн килгəчтен гөл көлə,
Гөл көлгəчтен мин көлəм.

Кояш көлə, гел көлə,
Кояш көлгəч көн килə,
Кояш көлə мин көлгəч,
Шуңа да мин гел көлəч.
```

# Перевод

### Роберт Миннуллин / Масгуда Шамсетдинова - Смешной день

```
Солнце смеется - все смеется,
Солнце смеется - день приходит,
День приходит - цветок смеется,
Цветок смеется - и я смеюсь.

Солнце смеется - все смеется,
Солнце смеется - день приходит,
Солнце смеется - я смеюсь,
Потому-то я все смеюсь.
```
