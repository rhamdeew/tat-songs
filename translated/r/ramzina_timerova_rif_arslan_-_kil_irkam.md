# Оригинал

### Рәмзинә Тимерова / Риф Арслан - Кил иркәм

```
Юлларың бик озак,
Бик урау, бормалы.
Әллә соң күрәзәче,
Ялгышмый урады.
Көтәм мин иртәсен,
Күзләрем юлларда.
Син минем янымда,
Хыялым уемда.

Бәхетле булырга,
Кем генә теләми.
Тик һәр кем үз серен,
Сөйләми, сөйләми.
Бер кемгә карама,
Кил иркәм яныма.
Гомерлек юлдаш бул,
Пар бул син җаныма.

Гомернең яртысы,
Тик синсез үтелгән.
Яшьлегем сукмагы,
Үтелгән, сүтелгән.
Яратам дисең син,
Көт мине ташлама.
Минем тормыш юлы,
Тик синнән башлана.

Бәхетле булырга,
Кем генә теләми.
Тик һәр кем үз серен,
Сөйләми, сөйләми.
Бер кемгә карама,
Кил иркәм яныма.
Гомерлек юлдаш бул,
Пар бул син җаныма.
```

# Перевод

### Рамзина Тимерова / Риф Арслан - Приди ко мне

```
Дороги твои далеки,
Очень извилистые, сложные.
Наверное, увидимся поздно,
Не ошибусь в этом.
Жду тебя с утра,
Глаза мои на дорогах.
Ты рядом со мной,
Во снах моих.

Кому не хочется быть
Счастливым.
Но каждый свой секрет
Не говорит, не говорит.
Не смотри ни на кого,
Приди ко мне.
Будь спутником жизни,
Будь парой для моей души.

Половина жизни
Прошла без тебя.
Моя молодость прошла,
Прошла, истаяла.
Если ты любишь,
Не бросай меня.
Моя жизненная дорога
Только с тебя начинается.

Кому не хочется быть
Счастливым.
Но каждый свой секрет
Не говорит, не говорит.
Не смотри ни на кого,
Приди ко мне.
Будь спутником жизни,
Будь парой для моей души.
```
