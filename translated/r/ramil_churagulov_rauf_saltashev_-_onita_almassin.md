# Оригинал

### Рамил Чурагулов / Рәүф Сәлтәшев - Оныта алмассың

```
Хушлашсакта сөю булып,
Калырмын йөрәгендә.
Зәңгәр сагыш булып төнен
Керемен төшләреңә.

Өзелеп-өзелеп теләсәңдә
Булмас мине онытып.
Керфегеннән тугелермен
Татлы күз яшең булып.

Күз алларымнан һич китмәм
Мине онтыта алммассың.
Яныңдагы үз парыңны
Мин дип уйлап назларсың.

Өзелеп-өзелеп теләсәңдә
Булмас мине онытып.
Керфегеннән тугелермен
Татлы күз яшең булып.

Өзелеп-өзелеп теләсәңдә
Булмас мине онытып.
Йөрәгемнән исемемне
Ташлый алмассың сызып.

Өзелеп-өзелеп теләсәңдә
Булмас мине онытып.
Керфегеннән тугелермен
Татлы күз яшең булып.
```

# Перевод

### Рамиль Чурагулов / Рауф Салташев - Не сможешь забыть

```
Когда прощаемся, любовью став,
Останусь в твоём сердце я.
Синей тоской в твою ночь
Войду в твои сны.

Как бы ты ни старалась,
Не сможешь меня забыть.
Из ресниц твоих прольюсь
Сладкой слезой твоей.

Из глаз твоих я не уйду,
Ты не сможешь меня забыть.
Своего спутника рядом
Мною считая, будешь нежна.

Как бы ты ни старалась,
Не сможешь меня забыть.
Из ресниц твоих прольюсь
Сладкой слезой твоей.

Как бы ты ни старалась,
Не сможешь меня забыть.
Из сердца моё имя
Не сможешь стереть ты.

Как бы ты ни старалась,
Не сможешь меня забыть.
Из ресниц твоих прольюсь
Сладкой слезой твоей.
```
