# Оригинал

### Рөстәм Зариф / Фәрит Хатыйпов - Сагыш (алтынчы вариант)

```
Һавадагы ак болыттан
"Син кайдан?" - дип сорадым.
Шул болыттан тамчы тамып
Керфекләрем чыланды.
Кыйбла якка искән җилдән
"Син кайдан?" - дип сорадым.
Шул җил мине, чәчләремнән

Маңгаемнан сыйпады.
Һавадагы аккошларга
Эндәшергә кыймадым.
Тамып калган каурыйларын
Мин тезләнеп җыйнадым.
Сагышларым, хәсрәтләрем
Күкрәгемә сыймады.
```

# Перевод

### Рөстәм Зариф / Фәрит Хатыйпов - Тоска (шестой вариант)

```
У белого облака в небе
«Откуда ты?» - тихо спросил.
С того облака капля упала -
И ресницы мои намочил.
У ветра, что дул к Кыбле,
«Откуда ты?» - вновь спросил.
Тот ветер меня, по волосам,

По лбу моему гладил.
К белым лебедям в небе
Обратиться я не посмел.
Выпавшие их перья
На коленях я собирал.
Тоска моя, горе мое
В груди моей не умещались.
```
