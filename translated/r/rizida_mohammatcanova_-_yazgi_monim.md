# Оригинал

### Ризида Мөхәммәтҗанова - Язгы моңым

```
Бар тирә-як - гел яшеллек,
Нурланып кояш көлә.
Син, кадерлем, колач җәеп
Киләсең икән миңа!

Кушымта:
Бәхетем син, язгы моңым,
Йөрәгемнең яртысы.
Күзләреңдә күрәм синең
Кояшның зур яктысын.

Назлы язга иркәләнеп,
Бөреләр шытым бирә.
Бәхетем син, язгы моңым
Гөрләвек булып йөзә.

Кушымта.

Миңа гына атап бүген
Нурланып кояш көлә. -
Иртән торуга кояштай
Елмаясың бит миңа!

Кушымта.
```

# Перевод

### Ризида Мөхәммәтҗанова - Весенний мой напев

```
Вкруг всё зелёно,
В сиянье солнце смеётся.
Ты, родная, распахнув объятья,
Идёшь ко мне!

Припев:
Ты счастье моё, весенний мой напев,
Половина сердца моего.
В твоих глазах я вижу
Свет большого солнца.

Ласковой весне убаюканны,
Почки дают ростки.
Ты счастье моё, весенний мой напев,
Плывёшь, как журчащий ручей.

Припев.

Лишь мне сегодня, сияя,
Солнце смеётся. -
С утра, как солнце,
Ты улыбаешься мне!

Припев.
```
