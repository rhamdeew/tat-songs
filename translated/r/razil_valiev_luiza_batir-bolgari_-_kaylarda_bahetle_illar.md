# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Кайларда бәхетле илләр?

```
Ник чыга кояш,
Ник ата таңнар,
Нигә сагышлы
Хыялый җаннар?
Ник ява яңгыр,
Ник исә җилләр?
Кайларда икән
Бәхетле илләр,
Кайларда бәхетле илләр?

Ник үсә гөлләр,
Ник шаулый урман,
Минем хыялның тамыры корган.
Ник яши кеше,
Ник дәшми илем?
Мин бу дөньяга
Нигә дип килдем?
Мин бу дөньяга нигә килдем?
```

```
Зачем восходит солнце,
Зачем рассветы наступают,
Почему печальны
Мечтательные души?
Зачем идёт дождь,
Зачем дуют ветра?
Где же находятся
Счастливые края,
Где же счастливые края?

Зачем растут цветы,
Зачем шумит лес,
Корни моей мечты иссохли.
Зачем живёт человек,
Почему не дышит страна?
Зачем я пришёл
В этот мир?
Зачем я пришёл в этот мир?
```
