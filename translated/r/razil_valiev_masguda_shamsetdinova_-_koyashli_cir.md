# Оригинал

### Разил Вәлиев / Мәсгудә Шәмсетдинова - Кояшлы җыр

```
Елмаеп карыйлар, -
Ал, зәңгәр үзләре...
Җирдәге чәчәкләр -
Кояшның күзләре.

Йөргәнмен ишетми
Мин аны элгәре...
Сандугач теледер -
Кояшның телләре.

Манды ул алтынга
Җәйләрне, көзләрне.
Урманда, кырларда -
Кояшның эзләре.

Кояшның үзе күк
Хыяллар югары.
Кояшлы булсын гел
Балачак юллары.
```

# Перевод

### Разиль Валиев / Масгуда Шамсетдинова - Солнечная песня

```
Смеясь, глядят -
Алые глаза...
Цветы на земле -
Солнечные глаза.

Хожу, не слышу
Его голоса...
Соловьиный язык -
Солнечные языки.

Он жаждет золота
Лет, осеней.
В лесу, на полях -
Солнечные следы.

Само солнце в небе
Мечтами высоко.
Пусть будет солнечно
Детская дорога.
```
