# Оригинал

### Рәшит Айчуаков / Юлай Моратов - Сагындырдың иркәм

```
Сагындырдың, иркәм,
Көтәм сине,
Күз нурымны сибеп җан-якка.
Уйланамын, нигә кайтмый, дип,
Тагы калып кайгы-сагышка.

Сагындырдың иркәм!
Шуңа микән
Күңел тулып йөрәк сызлана.
Күзләремә бөртек яшьләр килә,
Сагышлардан йөзем саргая.

Сагындырдың, иркәм, -
Түзәм әле,
Күңелем башкаларга бирелмәс.
Сине көтеп яшьлек мәхәббәтем
Кара җиргә керми сүрелмәс.
```

# Перевод

### Рәшит Айчуаков / Юлай Моратов - Соскучилась по тебе, милый

```
Соскучилась, милый,
Жду тебя,
Разливая свет своих глаз вокруг.
Думаю: почему не возвращаешься,
И вновь остаюсь в тоске и печали.

Соскучилась, милый!
Может, оттого
Полно сердце - и ноет душа.
В глазах моих зернышки слёз появляются,
От тоски моё лицо желтеет.

Соскучилась, милый, -
Я терплю,
Сердце другим не отдам.
Моя юная любовь, тебя ожидая,
В чёрную землю не уйдёт, не погаснет.
```
