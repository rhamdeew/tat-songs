# Оригинал

### Рәис Риянов / Хәмит Булатов - Читекләр

```
Безнең Арча читекләре
Елтыр ука бизәкле.
Читек кигән чибәр кызлар
Өзә бәгырь-үзәкне.
Эх, укалы читекләр!
Читек кигән кызлар белән
Бию рәхәт, егетләр!

Безнең Арча читекләре,
Үкчәләре түгәрәк.
Сабан туйларын шаулатып
Йөрик әле бергәләп.
Эх, каюлы читекләр!
Читек тегүче бер сылу
Каш сикертте, егетләр!

Безнең Арча читекләре,
Яшеле бар, сарысы.
Читекле кыз - гүзәлем дип
Яна-көя барысы.
Эх, чигүле читекләр!
Читек кигән чибәрләрдән
Колак какмыйк, егетләр!
```

# Перевод

### Рәис Риянов / Хәмит Булатов - Читекләр

```
Наши арские читеков
Блеск узорный, шёлк-серебро.
Девы в читеков хороводе
Рвут мне сердце до нутра.
Эх, читекләр с укаю!
С девами в читеков плясать
Сласть, парни, не таю!

Наши арские читекләр,
Каблучки у них круглы.
Сабантуй шумит и светится -
Пойдём вместе, что ж, братцы?
Эх, читекләр каюлы!
Одна мастерица-краса
Бровью взмыла, парни, клюй!

Наши арские читекләр -
Есть зелёные, есть желты.
«Дева в читеков - моя краса»,
Все горят до полноты.
Эх, читекләр с вышивкой!
Не проходим мимо, парни,
Красиц в читеков, с улыбкой!
```
