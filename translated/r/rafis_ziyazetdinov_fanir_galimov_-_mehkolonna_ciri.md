# Оригинал

### Рәфис Зиязетдинов / Фәнир Галимов - Мехколонна җыры

```
Унберенче мехколонна
Һәрбер өйне яктырта.
Алтын куллы ут тартучы
Кызлар күңелен яндырта.

Яшәү яме, тормыш тәме
Электрга бәйләнгән.
Мехколонна егетләре
Асыллардан сайланган.

Баганадан-баганага
Чыбык тартып йөргәндә,
Егетләрнең җырларыннан
Сандугачлар көнләшә.

Яшәү яме, тормыш тәме
Электрга бәйләнгән.
Мехколонна егетләре
Асыллардан сайланган.

Төннәрен көндез итәләр
Аптырамыйлар алар.
Офыкларга йолдыз чәчә
Әйтерсең лә, Аллалар.

Яшәү яме, тормыш тәме
Электрга бәйләнгән.
Мехколонна егетләре
Асыллардан сайланган.
```

# Перевод

### Рафис Зиязетдинов / Фанир Галимов - Песня механизированной колонны

```
Одиннадцатая мехколонна
Каждый дом освещает.
Золоторукий электрик
Сердца девичьи зажигает.

Радость жизни, вкус бытия
С электричеством связаны.
Парни из мехколонны
Из лучших отобраны.

От столба к столбу
Провода протягивая,
Под песни молодцев
Соловьи соревнуются.

Радость жизни, вкус бытия
С электричеством связаны.
Парни из мехколонны
Из лучших отобраны.

Ночи превращают в день они,
Ни минуты не теряют.
Звёзды рассыпают в небеса -
Словно боги, право слово.

Радость жизни, вкус бытия
С электричеством связаны.
Парни из мехколонны
Из лучших отобраны.
```
