# Оригинал

### Резеда Вәлиева / Харис Ибрагимов - Яз җитте

```
Җылы җил сирпеп үтте,
Агачларны селкетте:
- Уяныгыз, уян тизрәк,
- Яз җитте,
- Әй, яз җитте!

Кояш нурларын сипте,
Тәрәз битләрен үпте:
- Ачылыгыз, ачыл тизрәк,
Яз җитте,
Әй, яз җитте!

Сыерчык сайрап утте,
Һавалар зыңлап китте:
- Чыгыгыз, чык, әй, балалар,
Яз җитте,
Әй, яз җитте!

Яңгыр сибәләп утте,
Кырлар яшереп китте.
Нурга тулды туган җирләр,
Яз җитте,
Әй, яз җитте!
```

# Перевод

### Резеда Вәлиева / Харис Ибрагимов - Весна пришла

```
Тёплый ветер пролетел,
Деревья все качая:
- Просыпайтесь, просыпайтесь же скорей,
- Весна пришла,
- Ах, весна пришла!

Солнце лучики рассыпало,
Окна нежно целовало:
- Распахнитесь, распахнитесь же скорей,
Весна пришла,
Ах, весна пришла!

Скворушка запел звонко,
Ветерки звенят вокруг:
- Выходите же, выходите, дети,
Весна пришла,
Ах, весна пришла!

Дождик щедро пролился,
Поля зеленью укрылись.
Светом полнится родная сторона,
Весна пришла,
Ах, весна пришла!
```
