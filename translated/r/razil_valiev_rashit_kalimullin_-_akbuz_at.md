# Оригинал

### Разил Вәлиев / Рәшит Кәлимуллин - Акбүз ат

```
Туй күлмәге кигән таллар,
Бизәнгән урамнары.
Кочагына алды безне
Мәхәббәт бураннары.

Акбүз ат, акбүз ат, акбүз ат, акбүз ат...
Кыш аеның аклыгын ал,
Салкынын язга озат.
Ак канатлы акбүз атны
Гашыйклар көтте озак.

Ак бәхетләр юрый-юрый
Кышлар безгә ашыккан.
Ак юллардан үтә бүген
Җирдә бөтен гашыйклар.

Акбүз ат, акбүз ат, акбүз ат, акбүз ат...
Кыш аеның аклыгын ал,
Салкынын язга озат.
Ак канатлы акбүз атны
Гашыйклар көтте озак.
```

# Перевод

### Разил Вәлиев / Рәшит Кәлимуллин - Акбүз ат

```
Ивы в свадебных платьях,
Улицы в праздничном блеске.
Обняли нас с головой
Вьюги любви, как невесты.

Акбуз-ат, Акбуз-ат, Акбуз-ат, Акбуз-ат...
Возьми белизну зимнего месяца,
Холод в весну проводи.
Белокрылого Акбуз-ата
Долго влюблённые ждали в пути.

Белое счастье пророча,
Зимы к нам спешили сами.
По белым дорогам сегодня
Все влюблённые ходят по земле шагами.

Акбуз-ат, Акбуз-ат, Акбуз-ат, Акбуз-ат...
Возьми белизну зимнего месяца,
Холод в весну проводи.
Белокрылого Акбуз-ата
Долго влюблённые ждали в пути.
```
