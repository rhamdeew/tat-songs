# Оригинал

### Зиннур Сафиуллин - Бэхет тоне

```
Каеннары ай-яй ак икэн,
Сою белэн тулы чак икэн,
Каеннардан якты мон тама.

Бу бит бэхет тоне,
Эзлэ бэхетенне,
Юк, йоклама, егет, йоклама!

Егет яна кызга куз атып,
Сандугачлар йори озатып,
Йолдызлардан энже сибелэ.

Табышалар парлар,
Кавышалар ярлар,
Юк, бэгърем, юк бит син генэ.

Бэхет тоне узып китэрме,
Безгэ бэхет биреп китэрме?
Каеннардан сора, каеннан.

Кунелемнен куркен
Яулый алмас беркем —
Исемнэрен каеп язылган!
```

# Перевод

### Зиннур Сафиуллин - Ночь счастья

```
Берёзы, ай-яй, белы как снег,
Полны любви времена те,
С берёз светлая грусть капает.

Это ведь ночь счастья,
Ищи своё счастье,
Нет, не спи, парень, не спи!

Парень влюблённой девушке подмигивает,
Соловьи песни провожают,
Из звёзд жемчуг рассыпает.

Встречаются пары,
Сходятся милые,
Нет, моя радость, нет ведь только ты.

Ночь счастья пройдёт ли мимо,
Счастье нам подарит ли?
У берёз спроси, у берёз.

Из сердца моего страх
Никто не сможет стереть —
Имена наши на коре написаны!
```
