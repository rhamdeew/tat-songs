# Оригинал

### Айфара - Мин сине сагынам

```
Сэгатлэрне саный, минем кунел оча гына
Иртэгэ инде син кайтасын минем яныма
Инде куп калмады, сизелми дэ утэр бу кон..
Белэсенме иркэм син узен,
Гел юлларда булды бит кузем
Мин бит сине шундый коттем

Мин сине сагынам,мин сине яратам
Тик син элегэ менэ тугел янымда
Курешмэдек куптэн
Инде айлар уткэн
Тик син мина хэзер кадерлерэк тагын да

Ярак безнен ара,
Тик син узен мина якын
Куркыйтмый сынаулар,арабызга кермэс салкын..
Безне сою узган, купме еллар, юллар аша
Иртэгэ сине курэм тагын
Гомергэ кавышыйк без жаным…
Яшэу кыен синнэн башка…

Мин сине сагынам,мин сине яратам
Тик син элегэ менэ тугел янымда
Курешмэдек куптэн
Инде айлар уткэн
Тик син мина хэзер кадерлерэк тагын да
```

# Перевод

### Айфара - Я скучаю по тебе

```
Считаю часы, моё сердце летит,
Завтра ты вернёшься ко мне.
Уже немного осталось, незаметно пройдёт этот день..
Знаешь ли, милый, ты сам,
Всё время мои глаза были на дорогах,
Ведь я так тебя ждала.

Я скучаю по тебе, я тебя люблю,
Только тебя пока нет рядом со мной.
Давно не виделись,
Уже месяцы прошли,
Только ты мне теперь дороже ещё больше.

Далеко между нами,
Только ты сам мне близок.
Не пугают испытания, между нами не войдёт холод..
Нас любовь провела через столько лет, дорог,
Завтра тебя увижу снова,
На всю жизнь соединимся, душа моя…
Жить трудно без тебя…

Я скучаю по тебе, я тебя люблю,
Только тебя пока нет рядом со мной.
Давно не виделись,
Уже месяцы прошли,
Только ты мне теперь дороже ещё больше.
```
