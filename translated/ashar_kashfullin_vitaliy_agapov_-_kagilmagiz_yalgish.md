# Оригинал

### Әсхәр Кәшфуллин / Виталий Агапов - Кагылмагыз ялгыш

```
Җаным тулы сагыш,
Кагылмагыз ялгыш,
Яшем тора атылып чыгарга.
Сорамагыз, нигә
Йөрәк әрни, көя.
Ничек кенә, ничек чыдарга?

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?

Йөрәк тулы сагыш,
Кагылмагыз ялгыш
Күңелдәге нечкә кылларга.
Сабырлык бир, Тәңрем,
Урманнарга керәм,
Бер кычкырып елап, җырларга.

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?

Кан тама күңелдән,
Бу юкка түгел лә.
Яшем тора атылып чыгарга.
Йөрәк дөрләп яна,
Кадерләнә бара
Мәхәббәтсез үткән елларда...

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?
```

# Перевод

### Асхар Кашфуллин / Виталий Агапов - Не трогайте ошибку

```
Душа полна тоски,
Не трогайте ошибку,
Слёзы готовы вырваться наружу.
Не спрашивайте, почему
Сердце мучится, горит.
Как же, как же вытерпеть?

Иль меня судьба
Увела не туда,
Увела по ложным дорогам?
Душа замёрзла, оледенела,
Почему же я страдаю?
Не знаю почему, зачем идти?

Сердце полно тоски,
Не трогайте ошибку,
Не задевайте тонкие струны души.
Дай терпенья, Боже,
В леса уйду,
Чтобы крикнуть, плача, и петь.

Иль меня судьба
Увела не туда,
Увела по ложным дорогам?
Душа замёрзла, оледенела,
Почему же я страдаю?
Не знаю почему, зачем идти?

Кровь из сердца каплет,
И это не напрасно.
Слёзы готовы вырваться наружу.
Сердце трепещет, горит,
Всё ценнее становятся
Годы, прожитые без любви...

Иль меня судьба
Увела не туда,
Увела по ложным дорогам?
Душа замёрзла, оледенела,
Почему же я страдаю?
Не знаю почему, зачем идти?
```
