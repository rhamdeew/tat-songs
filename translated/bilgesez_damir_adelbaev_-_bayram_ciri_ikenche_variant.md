# Оригинал

### Билгесез / Дамир Әделбаев - Бәйрәм җыры (икенче вариант)

```
Алтын туйлы бәйрәм бүген,
Дуслар җыелган бергә.
Уйларым китә әйләнеп
Үткән озын гомергә.
Яулык болгый ераклардан,
Яшьлегем тирәкләре.
Хәтерләтеп үтәннәрне
Җилкетә йөрәкләрне.

Күтәрик без бакалларны,
Бәллүр моңы чыңласын.
Тормыш кадерләрен белик,
Үкенечкә калмасын.

Үткән елларның сагышы
Агып китсен суларда.
Яшьлек җырларын төшерик (искә алыйк)
Бүген алтын туйларда.

Күтәрик без бакалларны,
Бәллүр моңы чыңласын.
Тормыш кадерләрен белик,
Үкенечкә калмасын.
```

# Перевод

### Неизвестен / Дамир Аделбаев - Праздничная песня (второй вариант)

```
Золотая свадьба - праздник сегодня,
Друзья собрались вместе.
Мысли уносятся, кружась,
В прожитые долгие годы.
Платочком машут издалека
Опоры моей юности.
Напоминая о прошлом,
Волнуют сердца.

Поднимем мы бокалы,
Пусть хрустальная грусть звенит.
Ценить научимся жизнь,
Чтоб не осталось сожалений.

Печаль прошедших лет
Пусть утечёт с водами.
Песни юности вспомним
Сегодня на золотой свадьбе.

Поднимем мы бокалы,
Пусть хрустальная грусть звенит.
Ценить научимся жизнь,
Чтоб не осталось сожалений.
```
