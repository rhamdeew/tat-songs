# Оригинал

### Айгуль Туташ - Дуслык жыры

```
Мин бер  кызый, син—егет,
Чыктык иллэр гизэргэ!

Мин бер егет, син—кызый,
Барабыз икэу генэ!

Эйе-ие-ие! Чыктык иллэр гизэргэ!
Эйе-ие-ие! Барабыз икэу генэ!
Эйе-ие-ие! Яшэу рэхэт бит, иркэм!
Эйе! Эйе! Син—шул кеше, мин белэм!

Алтын кояш серлэре
Биеклеккэ илтэчэк,

Ике йолдыз арасы
Бары безнен килэчэк...

Эйе-ие-ие! Биеклеккэ илтэчэк!
Эйе-ие-ие! Бары безнен килэчэк!
Эйе-ие-ие! Яшэу рэхэт бит, иркэм!
Эйе! Эйе! Син—шул кеше, мин белэм!

Без барасы юлларны
Дуслык нуры яктырта,

Янгыр-давыл, буранга
Без елмаябыз гына!

Эйе-ие-ие! Дуслык нуры яктырта!
Эйе-ие-ие! Без елмаябыз гына!
Эйе-ие-ие! Яшэу рэхэт бит, иркэм!
Эйе! Эйе! Син—шул кеше, мин белэм!
```

# Перевод

### Айгуль Туташ - Песня дружбы

```
Я девчонка, ты — парень,
Вышли страны исходить!

Я парень, ты — девчонка,
Идём мы только вдвоём!

Эйе-ие-ие! Вышли страны исходить!
Эйе-ие-ие! Идём мы только вдвоём!
Эйе-ие-ие! Жизнь прекрасна ведь, милый!
Эйе! Эйе! Ты — тот человек, я знаю!

Золотого солнца тайны
К высотам понесут,

Между двух звёзд
Всё наше будущее...

Эйе-ие-ие! К высотам понесут!
Эйе-ие-ие! Всё наше будущее!
Эйе-ие-ие! Жизнь прекрасна ведь, милый!
Эйе! Эйе! Ты — тот человек, я знаю!

Все дороги, что пройдём,
Дружбы свет освещает,

Дождь-буря, буранам
Мы только улыбаемся!

Эйе-ие-ие! Дружбы свет освещает!
Эйе-ие-ие! Мы только улыбаемся!
Эйе-ие-ие! Жизнь прекрасна ведь, милый!
Эйе! Эйе! Ты — тот человек, я знаю!
```
