# Оригинал

### Рашит Вагапов - Кук кугэрчен

```
Кук кугэрчен куктэ уйный,
Кулэгэсе жирдэ уйный.
Минем кунелем сине уйлый,
Кук кугэрчен, сандугачым,
Синен кунелен кемне уйлый?

Кук кугэрчен, кил каршыма,
Хат языйм канатына;
Язган хатым, эманэтем,
Кук кугэрчен, сандугачым,
Тапшырчы жанашыма.

Кук кугэрчен горлидер
Голлэр чэчэк аткайда;
Син жаныемнан хэбэр котэм,
Кук кугэрчен, сандугачым,
Тан сызылып атканда.
```

# Перевод

### Рашит Вагапов - Сизый голубок

```
Сизый голубок в небе играет,
Тень его на земле танцует.
Моя душа о тебе мечтает,
Сизый голубок, соловей мой,
О ком твоя душа грезит?

Сизый голубок, лети навстречу,
Письмо напишу на крыле твоем;
Написанное письмо, завет мой,
Сизый голубок, соловей мой,
Доставь к моей любимой.

Сизый голубок гордо кружится,
Когда цветы распускаются;
Я жду вестей о душе моей,
Сизый голубок, соловей мой,
Когда рассвет занимается.
```
