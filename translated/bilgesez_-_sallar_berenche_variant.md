# Оригинал

### Билгесез - Саллар (беренче вариант)

```
Саллар акмаслар шул инде,
Саллар акмаслар инде.
Агыйделнең ярларында
Саллар какмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Егетләр сал өсләрендә,
Учак якмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Салда барган егетләргә
Кызлар бакмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Салда барган егетләргә
Кызлар бакмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Элеккечә алсу булып,
Таңнар атмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Элеккечә алсу булып,
Таңнар атмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Алар агып кына китә,
Кабат кайтмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Алар агып кына китә,
Кабат кайтмаслар инде.
```

# Перевод

### Бильгесез - Саллар (беренче вариант)

```
Салы не идә бозлы,
Салы не идә бозлы.
Агыйде буе яларда
Салы не утырмалы.

Салы не идә,
Салы не идә бозлы.
Ялкаулар сал өсләрендә
Ут ягарлы.

Салы не идә бозлы,
Салы не идә бозлы.
Салда барган ялкауга
Кызлар карарлы.

Салы не идә,
Салы не идә бозлы.
Салда барган ялкауга
Кызлар карарлы.

Салы не идә бозлы,
Салы не идә бозлы.
Иртәнге алсу булып,
Таңнар атмалы.

Салы не идә,
Салы не идә бозлы.
Иртәнге алсу булып,
Таңнар атмалы.

Салы не идә бозлы,
Салы не идә бозлы.
Алар агып китә дә инде,
Кайтарлы.

Салы не идә,
Салы не идә бозлы.
Алар агып китә дә инде,
Кайтарлы.
```
