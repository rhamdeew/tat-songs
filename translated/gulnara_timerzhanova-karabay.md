# Оригинал

### Гульнара Тимержанова - Карабай

```
Жилфер-Жилфер жил исэ
Житен тамрын жил кисэ
Егет усэ, кыз усэ
Ай, егет кунелен кыз озэ

Ай ли генэем бэлилур генэем
Сары сандугачкаем да, кара карлыгачкаем

Иртэн дэ кил, кич тэ кил
Тэрэзэмне чертэ кил
Чертуеннэн танырмын да
Каршы чыгып алырмын

Ай ли генэем бэлилур генэем
Сары сандугачкаем да, кара карлыгачкаем 

Жилфер-Жилфер жил исэ
Житен тамрын жил кисэ
Жилфер-жилфер жил искэндэй
Яшь гомер утеп китэ

Ай ли генэем бэлилур генэем
Сары сандугачкаем да, кара карлыгачкаем
```

# Перевод

### Гульнара Тимержанова - Карабай

```
Жилфер-жилфер ветер дует,
Ржи корни ветер рвёт,
Парень растёт, девушка растёт,
Ах, парня сердце девушка влечёт.

Ай ли, дорогой мой, милый мой,
Жёлтая иволга моя и чёрная ласточка.

Утром приходи, вечером приходи,
В окошко моё постучи,
По стуку узнаю
И навстречу выйду.

Ай ли, дорогой мой, милый мой,
Жёлтая иволга моя и чёрная ласточка.

Жилфер-жилфер ветер дует,
Ржи корни ветер рвёт,
Как жилфер-жилфер ветер дует,
Молодая жизнь проходит.

Ай ли, дорогой мой, милый мой,
Жёлтая иволга моя и чёрная ласточка.
```
