# Оригинал

### Чәчкә - Туганнар (алтынчы вариант)

```
Синдә - бер кан, миндә - бер кан,
Бер үк кан ага тамырдан.
Син - газиз туганым минем,
Бер нәселдән, бер ырудан.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.

Кайгыларым, шатлыкларым
Синең белән, туганым.
Саклап туганлык хисләрен
Кичик суларын, утларын.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.

Тормыш - матур, дөнья - иркен
Тар булмаганда күңелең.
Синең өчен иртә-кичен
Ачык минем ишегем.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.
```

# Перевод

### Чечке - Родные (шестой вариант)

```
В тебе - одна кровь, во мне - одна кровь,
Одна и та же кровь течёт в жилах.
Ты - дорогой мой родной,
Из одного рода, из одного племени.

Не теряйтесь, родные,
Даже живущие на чужбине.
Не враждуйте, родные,
Даже живущие по соседству.

Мои печали, мои радости
С тобою разделю, родной мой.
Храня родственные чувства,
Прощая обиды и гнев.

Не теряйтесь, родные,
Даже живущие на чужбине.
Не враждуйте, родные,
Даже живущие по соседству.

Жизнь прекрасна, мир свободен,
Когда не тесно на душе.
Для тебя с утра до ночи
Открыта дверь моя всегда.

Не теряйтесь, родные,
Даже живущие на чужбине.
Не враждуйте, родные,
Даже живущие по соседству.
```
