# Оригинал

### Җәмилә Низаметдин / Айдар Гайнуллин - Сине сөйгән кебек, бер кемне дә сөя алмам!

```
Мин беләм, сине сөйгән кебек,
Беркемне дә сөя алмам!
Алдама алтыннар күрсәтеп,
Алдама күз яше белән!

Югалттым, сине, күз нурым!
Хазер син тик хыялым!

Мин беләм, сине сөйгән кебек,
Беркемне дә сөя алмам!
Йөрәгемдәге тойгыларны
Аңлатып булмый сүзләр белән!

Сагынам, сагынам сине!
Синең белән үткән көннәрне!
```

# Перевод

### Джамиля Низаметдин / Айдар Гайнуллин - Так, как любила тебя, никого не смогу полюбить!

```
Я знаю, что так, как любила тебя,
Никого не смогу полюбить!
Не обманешь золотом меня,
Не обманешь слезами!

Потеряла тебя, свет очей!
Теперь ты лишь мечта моя!

Я знаю, что так, как любила тебя,
Никого не смогу полюбить!
Чувства, что живут в моём сердце,
Невозможно выразить словами!

Скучаю, скучаю по тебе!
По тем дням, что были с тобой!
```
