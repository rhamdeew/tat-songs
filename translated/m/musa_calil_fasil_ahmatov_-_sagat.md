# Оригинал

### Муса Җәлил / Фасил Әхмәтов - Сәгать

```
Сәгать суга: «даң, даң!..»
Хәбәр бирә таңнан:
Бакчага барырга
Унбиш минут калган.
Сикереп торды Марат,
Күрә - эшләр харап.
Тагын соңга калган,
Шул йокыга карап.
Сәгать йөри: «келт, келт!..»
- Тиз бакчага кит, кит!..
Марат аңа дәшә:
- Тукта, меня көт, көт!..
Сәгать җырлый: «диң, диң!..»
- Мин бит туктый белмим.
Моннан ары: «зиң, зиң!»
Миңа карап йөр син!..
```

# Перевод

### Муса Джалиль / Фасиль Ахматов - Часы

```
Часы бьют: «дзынь, дзынь!..»
Вестницу приходит от утра:
В сад идти осталось
Пятнадцать минут.
Вскочил Марат,
Видит - дел полно.
Опять опоздал,
Из-за этого самого сна.
Часы идут: «тик-так!..»
- Быстро в сад, иди, иди!..
Марат кричит им:
- Стоп, подожди, подожди меня!..
Часы поют: «динь, динь!..»
- Я ведь не умею останавливаться.
Дальше: «зинь, зинь!»
Ты на меня погляди!..
```
