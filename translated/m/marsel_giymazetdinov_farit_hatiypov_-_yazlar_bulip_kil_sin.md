# Оригинал

### Марсель Гыймазетдинов / Фәрит Хатыйпов - Язлар булып кил син

```
Карлар явып китте,
Кышлар инде җитте -
Кыр казлары очты күлләрдән.
Йөрмә янып-көеп,
Гөлләр сулды диеп,
Язлар яме янсын күзләрдә.

Кышлар үтеп бара,
Кар сулары ага -
Якты нулар җиргә түгелгән.
Гөлләр чәчәк ата,
Киткән кошлар кайта -
Яшеллеккә дөнья күмелгән.

Тургайларга тиң син,
Язлар белән кил син,
Сандугачтай сайра моңланып.
Моңнар диңгезендә
Йөз әле бүген дә -
Җырлар белән күмеп дөньяны.
```

# Перевод

### Марсель Гыймазетдинов / Фәрит Хатыйпов - Приходи весною

```
Снегопад уже стих,
Зимний срок подошёл -
Дикие гуси слетели с озёр.
Не ходи, не горюй,
Что цветы отцвели -
Пусть весны свет горит в твоих глазах.

Зимы мимо плывут,
Талый снег льётся в путь -
Светлые воды разлиты по земле.
Зацветают цветы,
Птицы к дому летят -
Мир утонул в зелёном море.

Ты равна жаворонкам,
С весной ко мне приди,
Пой, как соловей, в ласковой тоске.
В море звуков и грёз
Плыви, как и теперь -
Песнями укрой весь этот мир.
```
