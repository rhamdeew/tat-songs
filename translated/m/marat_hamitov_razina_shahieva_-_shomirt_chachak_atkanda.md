# Оригинал

### Марат Хәмитов / Рәзинә Шаһиева - Шомырт чәчәк атканда

```
Шомырт чәчәк атканда,
Язгы матур чакларда,
Йөрәкләргә дәрт өстәлеп,
Сызылып таңнар атканда.

Шомырт чәчәк атканда,
Хисләр тулып ташканда,
Ике йөрәк бердәй тибәр
Парлап вальс басканда,
Сайрар кошлар кайтканда,
Сөйгән ярың тапканда,
Иң күңелле он(ы)тылмас чак
Шомырт чәчәк атканда.

Тальян гармун тартканда,
Яшьлек дәрте ташканда,
Бәхетле парлар кушыла
Шомырт чәчәк атканда.

Шомырт чәчәк атканда,
Хисләр тулып ташканда.
Ике йөрәк бердәй тибәр
Парлап вальс басканда,
Сайрар кошлар кайтканда,
Сөйгән ярың тапканда,
Иң күңелле он(ы)тылмас чак
Шомырт чәчәк атканда.
```

# Перевод

### Марат Хәмитов / Разина Шафиева - Когда цветёт черёмуха

```
Когда цветёт черёмуха,
В прекрасные весенние дни,
Страстями сердца наполняя,
Рассветы тихо наступают.

Когда цветёт черёмуха,
Чувства переполняют через край,
Два сердца бьются в унисон
Вальс кружит, когда танцуют,
Птицы вернувшиеся поют,
Когда возлюбленную находишь,
Самая радостная, нескончаемая пора
Когда цветёт черёмуха.

Когда гармонь тальянка играет,
Пыл юности разгорается,
Счастливые пары соединяются
Когда цветёт черёмуха.

Когда цветёт черёмуха,
Чувства переполняют через край.
Два сердца бьются в унисон
Вальс кружит, когда танцуют,
Птицы вернувшиеся поют,
Когда возлюбленную находишь,
Самая радостная, нескончаемая пора
Когда цветёт черёмуха.
```
