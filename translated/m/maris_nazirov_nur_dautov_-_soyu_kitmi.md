# Оригинал

### Марис Нәзиров / Нур Даутов - Сөю китми

```
Җилләр кага бәхетемнең ак бишеген,
Хисләремнең чәчәгеннән таҗ өзелә:
Күрдем сине, күрдем сине ятлар белән,
Күрдем сине ятлар белән.

Сурәтеңне ертып кына аттым да ул,
Төшләремә кергән сынны нишләтергә:
Сөю килә язлар белән, язлар белән,
Сөю килә язлар белән...

Еллар койды өметемнең яфракларын,
Яшьлегемнең сукмагына бәс үрелә:
Көн дә үтәм эзләреңнән, эзләреңнән,
Көн дә үтәм эзләреңнән...

Хаталарыңны күптән утка яктым да ул,
Күңелемә сеңгән җырны нишләтергә:
Сөю китми көзләр белән, көзләр белән,
Сөю китми көзләр белән...
```

# Перевод

### Марис Нәзиров / Нур Даутов - Любовь не уходит

```
Ветры качают белую колыбель моего счастья,
С лепестка моих чувств срывается венец:
Я видел тебя, видел тебя с чужими,
Я видел тебя с чужими.

Я разорвал твой образ и бросил его,
Что мне делать с тенью, что входит в мои сны:
Любовь приходит с весной, с весной,
Любовь приходит с весной...

Годы осыпали листья моей надежды,
На тропу моей юности стелется иней:
Каждый день иду по твоим следам, по твоим следам,
Каждый день иду по твоим следам...

Твои ошибки давно предал я огню,
Что мне делать с песней, что в сердце растворилась:
Любовь не уходит с осенью, с осенью,
Любовь не уходит с осенью...
```
