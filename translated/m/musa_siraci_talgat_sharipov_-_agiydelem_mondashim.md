# Оригинал

### Муса Сираҗи / Тәлгать Шәрипов - Агыйделем - моңдашым

```
Болыт кунган чал таулардан
Башлап алып юл башын,
Келтер-келтер тәгәрәтеп
Уралымның вак ташын,
Челтер-челтер агып ята
Агыйделем - юлдашым,
Агыйделем - юлдашым.

Ярларында, талларында
Сайратып мең-мең кошын,
Кайда урман, кайда тургай -
Шунда борып ағышын,
Ага тарих төпкеленнән
Агыйделем - язмышым,
Агыйделем - язмышым.

Ак томаннарга уралып,
Офыкларга олгашып,
Ага халкым йөрәгеннән
Көне төне моң ташып,
Ак дулкынлы, акчарлаклы
Агыйделем - моңдашым ...
Агыйделем - моңдашым ...
```

# Перевод

### Муса Сираҗи / Тәлгать Шәрипов - Агыйделем - мой напевный друг

```
С седых гор, где тучи сели,
Путь берет свой исток,
Перекатывает звонко
Урала мелкий камешек,
Звеня-журча, течет несмолкно
Агыйдель - мой попутчик,
Агыйдель - мой попутчик.

На берегах, среди ивняков
Тысячей голосов поет птица,
Где лес шумит, где жаворонок -
Туда свой поворот стремится,
Из глубин истории течет
Агыйдель - судьба моя,
Агыйдель - судьба моя.

В белых туманах укрываясь,
К горизонтам тянется,
Из сердца моего народа
День и ночь льется песня,
С белой волной, с белой чайкой
Агыйдель - мой напевный друг...
Агыйдель - мой напевный друг...
```
