# Оригинал

### Мостай Кәрим / Салават Низаметдинов - Димәк, мин әле яшимен

```
Юлларым ерак, сөеклем,
Юлларым сиңа бара.
Елларым яфрактай коела,
Тик якынаймый ара.
Юлларым сиңа бара,
Тик якынаймый ара!

Димәк, мин әле сөямен,
Шул юлдан киләм икән,
Димәк, мин әле яшимен,
Әгәр дә сөям икән.

Елларым яфрактай коела,
Тик зарланыр юк зарым!
Бик урау-урау, булсада,
Сиңа бара юлларым.
Зарланырга юк зарым,
Сиңа бара юлларым.

Димәк, мин әле сөям,
Шул юлдан киләм икән,
Димәк, мин әле яшимен,
Әгәр дә сөям икән.

Юлларым ерак, сөеклем,
Юлларым сиңа бара.
Елларым яфрактай коела,
Тик якынаймый ара.
Юлларым сиңа бара,
Тик якынаймый ара!

Димәк, мин әле сөям,
Шул юлдан киләм икән,
Димәк, мин әле яшимен,
Әгәр дә сөям икән.
```
