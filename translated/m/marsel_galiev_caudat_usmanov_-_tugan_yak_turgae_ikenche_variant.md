# Оригинал

### Марсель Галиев / Җәүдәт Усманов - Туган як тургае (икенче вариант)

```
Ак болытлар ага,
Сулар ярга кага,
Тирбәлә җәй туган ягымда.
Тауларына менсәм,
Җылы җилләр исә,
Киң офыклар сыя җаныма.

Ак болыттай үтә,
Кабатланмый китә,
Бу дөньяның якты һәр көне.
Имин еллар төсен,
Биек алып менсен,
Учакларның зәңгәр төтене.

Бәхет тәмен беләм,
Туган җирдә генә
Иген кырларына күмеләм.
Монда гомер юлым,
Монда туган илем, / Монда туган моңым, /
Тургай булып сайрый күңелем.
```

# Перевод

### Марсель Галиев / Джаудат Усманов - Жаворонок родного края (второй вариант)

```
Белые облака плывут,
Воды бьются о берег,
Качается лето в родном краю.
Когда поднимусь на горы,
Веют тёплые ветра,
Широкие горизонты ласкают душу.

Как белое облако проходит,
Не повторяясь, уходит
Каждый светлый день этого мира.
Пусть дым лет несётся,
Высоко поднимаясь ввысь,
Синий дым очагов.

Вкус счастья познаю
Лишь на родной земле,
Припадаю к полям хлебным.
Здесь жизненный путь мой,
Здесь родная земля моя,
Жаворонком поёт душа моя.
```
