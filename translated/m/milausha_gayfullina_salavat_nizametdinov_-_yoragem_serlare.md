# Оригинал

### Миләүшә Гайфуллина / Салават Низаметдинов - Йөрәгем серләре

```
Мәктәпкә килгәндә
Тап булдың юлыма.
Елмаеп карадың,
Кагылдың кулыма.

Дәресне тыңламый,
Үткәрәм вакытны.
Укыймын йөз кабат
Җибәргән хатыңны.

Тактада дус кызым
Нәрсәдер аңлата.
Алдымда рәсемең
Елмаеп тик ята.

Сәгатьнең телләре
Ашыкмый, ашыкмый.
Йөрәгем серләрен
Ачыклый, ачыклый.
```

# Перевод

### Миляуша Гайфуллина / Салават Низаметдинов - Тайны сердца моего

```
Когда пришла в школу,
Случайно встретилась ты мне.
Улыбнувшись, взглянула,
Коснулась моей руки.

На уроке не слушаю,
Провожу время впустую.
Читаю сто раз подряд
Твоё записанное письмо.

На парте подруга моя
Что-то нашёптывает.
Передо мной образ твой
Лишь улыбается лежит.

Стрелки часов
Не спешат, не спешат.
Тайны сердца моего
Раскрывают, раскрывают.
```
