# Оригинал

### motal_ramov_ilshat_yahin_ - Мотал Рәмов / Илшат Яхин - Безнең яклар

```
Безнең яклар - шундый илһамлы як,
Күңелләрдә дала киңлеге.
Ижад уты дөрли йөрәкләрдә,
Көдрәт бирә уйлар берлеге!

Сибай, Сибай - син бит һәр яктан бай,
Алтынга тиң халкың - хәзинә!
Горурланып яши синдә һәр кем,
Читкә киткән сине сагына!

Иле өчен җанын фида кылыр
Күпме уллар, кызлар үстердең.
Ата-баба йолаларын саклап,
Милләт рухын күккә күтәрдең!

Сибай, Сибай - син бит һәр яктан бай,
Алтынга тиң халкың - хәзинә!
Горурланып яши синдә һәр кем,
Читкә киткән сине сагына!

Офыклардан байкап офыкларны
Күкне айкый дала бөркете.
Милләт хәзинәсе булыр көчне
Безгә Ходай үзе беркетте!

Сибай, Сибай - син бит һәр яктан бай,
Алтынга тиң халкың - хәзинә!
Горурланып яши синдә һәр кем,
Читкә киткән сине сагына!
```
