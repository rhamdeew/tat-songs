# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Салагыш болыннары

```
Зәңгәр Кама буйларында
Салагыш болыннары.
Шунда мәхәббәт юлыбыз
Кушылган урыннары.

Салагыш, Салагыш!
Мин китәмен еракларга,
Бәгырем, хуш, бәгырем, хуш!

Күлдә йөзгән аккошларга
Сокланып карап уздык;
Аккошлар мәхәббәтенә
Тиң булсын безнең дуслык!

Салагыш, Салагыш!
Мин китәмен еракларга,
Бәгырем, хуш, бәгырем, хуш!

Йөрәгемдә алып китәм
Сине, Салагыш җире!
Чәчәкле болыннарыңа
Кайтып килермен кире.

Салагыш, Салагыш!
Мин китәмен еракларга,
Бәгырем, хуш, бәгырем, хуш!
```

# Перевод

### Мостафа Ногман / Рөстәм Яхин - Салагышские луга

```
У синей Камы берега -
Луга Салагыша.
Там наш любовный путь
Сошёлся воедино.

Салагыш, Салагыш!
Я ухожу в далекий край,
Бәгырем, прощай, бәгырем, прощай!

На лебедей, что в озере,
С восторгом мы глядели;
Любви лебединой
Пусть равна будет дружба наша!

Салагыш, Салагыш!
Я ухожу в далекий край,
Бәгырем, прощай, бәгырем, прощай!

В сердце уношу с собой
Тебя, земля Салагыша!
К цветущим лугам твоим
Вернусь я вновь обратно.

Салагыш, Салагыш!
Я ухожу в далекий край,
Бәгырем, прощай, бәгырем, прощай!
```
