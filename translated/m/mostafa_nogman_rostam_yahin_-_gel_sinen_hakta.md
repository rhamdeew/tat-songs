# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Гел синең хакта

```
Зәңгәр ефәк күккә бер-бер артлы
Көмеш бөртекләре
Сибелгән чакта,
Уйлыйм, бәгърем, мин синең хакта,
Гел синен хакта.

Күктә йөзеп йөргән түгәрәк ай
Ямансулап һәр көн
Кимегән чакта,
Уйлыйм, бәгърем, мин синең хакта,
Гел синең хакта.

Шат йолдызлар сүнеп, ай югалып,
Дөнья төнге нурга
Төренгән чакта,
Өзелеп уйлыйм мин синең хакта,
Гел синең хакта.
```

# Перевод

### Мостафа Ногман / Рөстәм Яхин - Всегда о тебе

```
Когда лазурное шелковое небо
Серебряными крупинками
Осыпается одна за другой,
Думаю, милый, я о тебе,
Только о тебе.

Когда круглый месяц, плывущий в небе,
Тоскуя, день за днем
Начинает таять,
Думаю, милый, я о тебе,
Лишь о тебе.

Когда веселые звезды гаснут, месяц прячется,
И мир укутан
Ночным сияньем,
С надрывом думаю я о тебе,
Вечно о тебе.
```
