# Оригинал

### Айрат Кәримов / Рафаэль Шәхрәзиев - Яратам (пятый вариант)

```
Очраттым мин сине,
Тормышым юлымда.
Ә (тик) синең язмышың,
Башкалар кулында.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.

Булыр дип куркасың,
Гайбәтме, бәләме.
Сөюүне кешедән,
Яшереп буламы.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.

Тоелган сөюгә
Ни генә янамый.
Яратам дисәң син
Ярамый, ярамый.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.
```

# Перевод

### Айрат Кәримов / Рафаэль Шәхрәзиев - Люблю (пятый вариант)

```
Я встретил тебя
На пути своей жизни.
А (лишь) твоя судьба
В чужих руках зависла.

Пусть слышит весь мир,
Я миру громко крикну.
Я тебя люблю,
Люблю, люблю.

Ты боишься, что будет -
Сплетни ли, беда ли.
Разве можно любовь
От людей укрывать?

Пусть слышит весь мир,
Я миру громко крикну.
Я тебя люблю,
Люблю, люблю.

К той любви, что пришла,
Разве можно не сгорать?
Если скажешь «люблю» -
Нельзя, нельзя.

Пусть слышит весь мир,
Я миру громко крикну.
Я тебя люблю,
Люблю, люблю.
```
