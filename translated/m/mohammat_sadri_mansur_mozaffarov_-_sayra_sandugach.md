# Оригинал

### Мөхәммәт Садри / Мансур Мозаффаров - Сайра, сандугач

```
Сайра, сайра, сандугачым,
Талдан талларга очып,
Күңелең теләгән җиргә очып,
Матур чәчәкләр кочып.

Сандугачым сайрый чут-чут, дип,
Безнең илдән матур ил юк, дип.

Сайра, сайра, сандугачым,
Бездә шат тормыш көлә,
Бездә, матурлап синеңчә,
Бар халык җырлый белә.

Сандугачым сайрый чут-чут, дип,
Безнең илдән матур ил юк, дип.

Сайра, сайра, җырчы кошым,
Җырла бер дә армыйча.
Җир шарын шау-герләтеп без
Җырлыйк бер туктамыйча.

Сандугачым сайрый чут-чут, дип,
Безнең илдән матур ил юк, дип.
```

# Перевод

### Мөхәммәт Садри / Мансур Мозаффаров - Пой, соловей

```
Пой, пой, мой соловей,
С ветки на ветку лети,
Туда, куда зовет душа,
Обними красивые цветы.

Мой соловей поет: чут-чут,
Нет страны прекрасней нашей, мол.

Пой, пой, мой соловей,
У нас смеется радость дней,
У нас, красиво, как ты сам,
Петь умеет весь народ.

Мой соловей поет: чут-чут,
Нет страны прекрасней нашей, мол.

Пой, пой, певчая птица моя,
Пой, не уставая ни на миг.
Пусть весь земной шар загудит,
Поймо без остановки вместе мы.

Мой соловей поет: чут-чут,
Нет страны прекрасней нашей, мол.
```
