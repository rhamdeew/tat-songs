# Оригинал

```
Кузгала, поезд кузгала
Китәмен еракларга
Аерылулар җинел булмый,
Барыбер елап калма
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Еламый түзеп кара.
Яраларың кузгалганда,
Еламый түзеп кара.

Тәгәрмәчләр сулкылдыйлар
Җаннарны өзеп-өзеп.
Җырларымнан туктап калсам,
Мин дә елармын кебек.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырламый түзеп кара.
Яраларың кузгалганда,
Җырламый түзеп кара.

Аерылулар беркайчан да,
Булмыйдыр җинел генә.
Мин үземнең хәлләремне
Беләм тик үзем генә.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырламый түзеп кара.
Яраларың кузгалганда,
Җырламый түзеп кара.

Кузгала, йөгерә, кузгала
Язмышның поездлары
Җаннарымны юатырга
Җырларым калыр бары.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырларым сездә кала.
Яраларың кузгалганда,
Җырларым сездә кала.
```

# Перевод

### Марат Кабиров / Ризван Хакимов - Поезд трогается

```
Тронется, поезд тронется,
Уйду за дальние дали.
Расстаться нелегко, но все же
Не плачь, не слезы пролей.
Тронется, поезд тронется,
В сердце шевелится рана.
Когда она снова всколыхнется -
Попробуй не плакать, упрямо.
Когда она снова всколыхнется -
Попробуй не плакать, упрямо.

Колеса всхлипывают глухо,
Рвут душу раз за разом.
Когда мои песни смолкнут -
И я зарыдаю, пожалуй.
Тронется, поезд тронется,
В сердце шевелится рана.
Когда она снова всколыхнется -
Попробуй молчать, не запевая.
Когда она снова всколыхнется -
Попробуй молчать, не запевая.

Расставанья никогда
Не бывают совсем легкими.
Свою судьбу, свои боли
Знаю лишь сам, потихоньку.
Тронется, поезд тронется,
В сердце шевелится рана.
Когда она снова всколыхнется -
Попробуй молчать, не запевая.
Когда она снова всколыхнется -
Попробуй молчать, не запевая.

Тронутся, ринутся, тронутся
Поезда моей судьбы.
Лишь песни останутся рядом
Душу мою утешить, увы.
Тронется, поезд тронется,
В сердце шевелится рана.
Когда она снова всколыхнется -
Мои песни с вами останутся.
Когда она снова всколыхнется -
Мои песни с вами останутся.
```
