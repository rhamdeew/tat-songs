# Оригинал

### Мөнир Шәрәфетдинов - Сөйгән ярым, миңа ачуланма

```
Сөйгән ярым, миңа ачуланма
Төшләреңә кереп йөдәтсәм.

Шәмдәлләрдә янган утлар булып
Нурлы йөзләреңне күзәтсәм.

Сөйгән ярым, миңа ачуланма
Юлларыңда чәчәк булмасам.

Синең серле зәңгәр күзләреңә
Шаян нурлар сибә алмасам.

Сөйгән ярым, миңа ачуланма
Таң йолдызы булып янмасам.

Синең серле зәңгәр күзләреңә
Шаян нурлар сибә алмасам.
```

# Перевод

### Мунир Шарафетдинов - Милый, рәхим ит миңа

```
Милый, рәхим ит миңа,
Әгәр хыялларыңа кереп йөрсәм.

Свечаларда янып торган ут булып
Нурлы юзүңне күзәтсәм.

Милый, рәхим ит миңа,
Пыяларыңда чәчәк булмасам.

Синең серле күк күзләренә
Якты нурлар сибә алмасам.

Милый, рәхим ит миңа,
Иртәнге йолдыз булып ялмасам.

Синең серле күк күзләренә
Якты нурлар сибә алмасам.
```
