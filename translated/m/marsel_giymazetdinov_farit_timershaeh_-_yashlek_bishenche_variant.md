# Оригинал

### Марсель Гыймазетдинов / Фәрит Тимершаех - Яшьлек (бишенче вариант)

```
ЯШЬЛЕК ТОРА ТАШЫП
Яз кошлары күктә,
Яшьләр язны көтә,
Кояш нуры көләч йөзләрдә.
Яшьлек тора ташып,
Моң агыла шашып,
Сөю хисе кунган күзләргә.

Яшьлек яшьлек инде,-
Гүя алып килде
Сөю тулы назлы язларны.
Су буенда парлар,
Учак ягып алар,
Каршылыйлар киек казларны.

Язлар белән бергә
Кыр казлары килә,
Тик нигә бар ялгыз ак казлар?
...Шаулы язлар үтәр,
Сизми калыр күпләр,
Әйтерсең лә яшьлек шул язлар.
```

# Перевод

### Марсель Гыймазетдинов / Фәрит Тимершаех - Молодость (пятый вариант)

```
МОЛОДОСТЬ ПЕРЕПОЛНЕНА
В небе птицы весны,
Юные ждут весны,
Солнечный свет смеётся в лицах.
Молодость переполнена,
Песнь течёт, не сдержана,
Взглядов, где любовь устроила гнёзда.

Молодость, молодость -
Словно принесла
Нежную весну, полную любви.
У реки стоят пары,
Разведя костры,
Встречают диких гусей.

С весной приходят
Полевые гуси,
Только отчего есть одинокие белые?
...Пройдут шумные вёсны,
Многие не заметят,
Словно молодость - те самые вёсны.
```
