# Оригинал

### Муса Җәлил / Җәүдәт Фәйзи - Кызыма (өченче вариант)

```
Шаян кызым син минем,
Таң йолдызым син минем,
Йөрәгемдә кабынган
Шатлык җырым син минем.

Синең гомрең җыр булып
Яңрар бәхет илендә.
Йолдыз булып янарсың
Син туган ил күгендә.

Таңнан торып, бакчада
Син уйнадың көнозын.
Аргансыңдыр, кил, кызым,
Йокла инде, йолдызым!

Әлли-бәлли, бәлли-бәү!
Әлли-бәлли, бәлли-бәү!
```

# Перевод

### Муса Җәлил / Җәүдәт Фәйзи - Доченьке (третий вариант)

```
Шаловливая ты моя,
Звёздочка зари ты моя,
В сердце вспыхнувшая
Песня радости ты моя.

Пусть твоя жизнь песней
Отзовётся в краю счастья.
Будешь звёздочкой сиять
В небе родины твоей.

С раннего утра в саду
Ты играла целый день.
Устала, поди, ну, доченька,
Спи уже, звёздочка моя!

Баю-бай, баю-баю!
Баю-бай, баю-баю!
```
