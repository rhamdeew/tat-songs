# Оригинал

### Мәхмүт Хөсәен / Рафаил Бакиров - Бәхетле булыйк! (беренче вариант)

```
Йолдызлар бар җирдә,
Җирдә кояш бар.
Эштә киң җәелә
Безнең колачлар.

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.

Мәхәббәт бар җирдә,
Яну җирдә бар.
Зур юлдаш гомергә -
Ихлас сөйгән яр.

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.

Тынычлык бар җирдә,
Дуслык җирдә бар.
Дуслыкка берни дә
Җитми, туганнар!

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.
```

# Перевод

### Мәхмүт Хөсәен / Рафаил Бакиров - Давайте будем счастливы! (первый вариант)

```
Есть звезды на земле,
И солнце есть в пути.
В труде расправим мы
Свои могучие плечи.

Пусть будет чистым небо,
Пусть колосится рожь.
Пусть в трудных буднях люди
Счастливы будут тоже.

Есть на земле любовь,
Есть жар в сердцах живой.
Спутник большой на жизнь -
Любимый, верный, родной.

Пусть будет чистым небо,
Пусть колосится рожь.
Пусть в трудных буднях люди
Счастливы будут тоже.

Есть на земле покой,
И дружба есть везде.
Дружбе ничто не может
Сравниться, братья, нет!

Пусть будет чистым небо,
Пусть колосится рожь.
Пусть в трудных буднях люди
Счастливы будут тоже.
```
