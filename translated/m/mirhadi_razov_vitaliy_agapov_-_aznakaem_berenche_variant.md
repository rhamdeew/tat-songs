# Оригинал

### Мирһади Разов / Виталий Агапов - Азнакаем (беренче вариант)

```
Борма-борма юлларыңның
Читләре ап-ак каен.
Ак каеннар, ак бәхетләр
Җире син, Азнакаем.

Челтер-челтер чишмә ага
Һәрбер тау астың саен.
Йөрәгемнән чыккан моңым,
Җырым син, Азнакаем.

Урманнарың - әкият иле,
Күгеңдә серле аең.
Былбыл итеп, канат биреп
Очырдың, Азнакаем.

Челтер-челтер чишмә ага
Һәрбер тау астың саен.
Йөрәгемнән чыккан моңым,
Җырым син, Азнакаем.

Сагындырып төшкә керә
Җәйләрең һәр таң саен.
Җир җиләкле болыныңның
Гөле мин, Азнакаем.

Челтер-челтер чишмә ага
Һәрбер тау астың саен.
Йөрәгемнән чыккан моңым,
Җырым син, Азнакаем.
```

# Перевод

### Мирһади Разов / Виталий Агапов - Азнакай (первый вариант)

```
По извилистым дорогам твоим
Края белые от седин.
Белые кеды, белое счастье
Ты для меня, Азнакай.

Струится родник
Под каждым перевалом.
Звук, что с сердца моего,
Песня моя, ты, Азнакай.

Леса твои - сказочная страна,
В небе твоем луна тайна.
Соловьем, дав крылья,
Ты летала, Азнакай.

Струится родник
Под каждым перевалом.
Звук, что с сердца моего,
Песня моя, ты, Азнакай.

Из тоски вхожу я
В лето твое с утра.
Цветок земляничного поля
Мой, Азнакай.

Струится родник
Под каждым перевалом.
Звук, что с сердца моего,
Песня моя, ты, Азнакай.
```
