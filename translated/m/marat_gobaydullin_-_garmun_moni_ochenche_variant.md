# Оригинал

### Марат Гобәйдуллин - Гармун моңы (өченче вариант)

```
Гармуныңны уйныйсың да,
Урамнарны буйлыйсың.
Күңелең тулы моң гына,
Ни турында уйлыйсың.

Кушымта:
Уйнат, дустым, гармуныңны
Безнең авыл көенә.
Синең моңлы уйнавыңа
Күңелләрем сөенә.

Яшьлек хатирәсе булып,
Чыңлый гармун тавышы.
Шул гармун моңы тарата
Йөрәгемнең сагышын.

Кушымта:

Кушылып гармун тавышына,
Җыр суза безнең яшьләр.
Гармун моңнары булганда
Авыллар мәңге яшәр.

Кушымта:
```

# Перевод

### Марат Гобәйдуллин - Мелодия гармони (третий вариант)

```
Ты играешь на гармони,
По улицам идешь.
В душе одна лишь песнь,
О чем мечты бережешь?

Припев:
Играй, мой друг, на гармони
Под нашу деревни напев.
Твоей напевной игрой
Сердца мои радуешь ты.

Как память о юности
Звенит голос гармони.
Эта мелодия гонит
Тоску из моего сердца.

Припев:

Вплетаясь в голос гармони,
Тянет песню наша молодежь.
Пока звучат гармонные напевы,
Деревни будут жить вечно.

Припев:
```
