# Оригинал

### Марат Гобәйдуллин - Татарстан - гүзәл республикам

```
Идел ага тулып ярларыннан,
Гасырларны кичеп, ерактан.
Хезмәт сөйгән горур халкы белән
Гөрләп чәчәк ата Татарстан.

Татарстан - гүзәл республикам,
Халкыбызның гомер бишеге.
Яңгырасын ерак гасырларга
Бүгенгедәй мәгърур исемең.

Гасырлардан килгән ирек хисе,
Китмәс бер дә халкым күңленнән.
Ерактагы якты маяк булып,
Әйдәп торыр гомер-гомергә.

Татарстан - гүзәл республикам,
Халкыбызның гомер бишеге.
Яңгырасын ерак гасырларга
Бүгенгедәй мәгърур исемең.

Еллар арты еллар уза торыр,
Алышыныр күпме буыннар.
Тарих битләренә язылырлык
Халкым узган якты эзләр бар.

Татарстан - гүзәл республикам,
Халкыбызның гомер бишеге.
Яңгырасын ерак гасырларга
Бүгенгедәй мәгърур исемең.
```

# Перевод

### Марат Гобәйдуллин - Татарстан - прекрасная республика

```
Идил течет, переполненный, из берегов,
Через века, издалека.
С трудолюбивым, гордым народом
Пышно расцветает Татарстан.

Татарстан - прекрасная республика,
Колыбель жизни нашего народа.
Пусть звучит сквозь далекие века
Твое имя, как ныне, гордое.

Чувство свободы, пришедшее из веков,
Не уйдет из сердца моего народа.
Дальним светлым маяком
Будет вести из века в век.

Татарстан - прекрасная республика,
Колыбель жизни нашего народа.
Пусть звучит сквозь далекие века
Твое имя, как ныне, гордое.

Годы за годами будут проходить,
Поколения будут сменяться.
Есть светлые следы пути народа,
Достойные страниц истории.

Татарстан - прекрасная республика,
Колыбель жизни нашего народа.
Пусть звучит сквозь далекие века
Твое имя, как ныне, гордое.
```
