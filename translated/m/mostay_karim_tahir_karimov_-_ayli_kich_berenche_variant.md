# Оригинал

### Мостай Кәрим / Таһир Кәримов - Айлы кич (беренче вариант)

```
Айның якты нуры төшә
Колхозым кырларына.
Исемнәреңне кушамын
Сагыну җырларыма.

Тәрәзәмне ачып куям,
Син җырлап үткән чакта.
Минутларымны саныймын
Сагынып көткән чакта.

Кыр казлары тезелеп оча
Кыйдаш су буйларыннан.
Әллә гашыйк булдым инде,
Чыкмыйсың уйларымнан.
```

# Перевод

### Мостай Карим / Тагир Каримов - Лунная ночь (первый вариант)

```
Яркий свет луны падает
На поля моего колхоза.
Твои имена добавляю
К своим песням тоски.

Открываю своё окно,
Когда ты поёшь, проходя мимо.
Считаю свои минуты,
Тоскливо ожидая.

Полевые гуси летят строем
От берегов Кыдаша.
Вероятно, стал уже влюблён,
Не выходишь ты из моих мыслей.
```
