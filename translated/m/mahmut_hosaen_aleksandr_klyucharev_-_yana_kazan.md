# Оригинал

### Мәхмүт Хөсәен / Александр Ключарев - Яңа Казан

```
Әкрен генә Идел дулкынында
Ай тирбәлә, кояш чайкала;
Киң җәелеп фәннәр кочагында
Үсә Казан - нурга бай кала.

Бәхетле гомергә
Ил белән гел бергә,
Үсә Казан,
Гүзәл Казан,
«Дәртле Казан, моңлы Казан,
Нурлы Казан», яңа Казан.

Бизи аны яңа матур йортлар,
Гөлбакчалар белән урала.
Ялкынланып хезмәт учагыңда
Үсә Казан - яңа зур кала.

Бәхетле гомергә
Ил белән гел бергә,
Үсә Казан,
Гүзәл Казан,
«Дәртле Казан, моңлы Казан,
Нурлы Казан», яңа Казан.

Сәйдәш моңы, Тукай, Җәлил җыры
Шаулап тора әллә кайларга;
Татарстан халкын шатландырып,
Үсә Казан - җырга бай кала.

Бәхетле гомергә
Ил белән гел бергә,
Үсә Казан,
Гүзәл Казан,
«Дәртле Казан, моңлы Казан,
Нурлы Казан», яңа Казан.
```

# Перевод

### Махмут Хусаен / Александр Ключарев - Новый Казань

```
Тихо на волнах Иделя
Луна колышется, солнце сверкает;
Раскинувшись широко в объятиях наук
Растёт Казань - богатая светом город.

На счастливую жизнь
Всегда вместе со страной,
Растёт Казань,
Прекрасный Казань,
«Страстный Казань, певучий Казань,
Светлый Казань», новый Казань.

Украсили его новые красивые дома,
Садами окружили.
Пылая в очаге труда
Растёт Казань - новый большой город.

На счастливую жизнь
Всегда вместе со страной,
Растёт Казань,
Прекрасный Казань,
«Страстный Казань, певучий Казань,
Светлый Казань», новый Казань.

Мелодия Сайдаша, Тукая, песня Джалиля
Гремят то тут, то там;
Радуя народ Татарстана,
Растёт Казань - богатый песнями город.

На счастливую жизнь
Всегда вместе со страной,
Растёт Казань,
Прекрасный Казань,
«Страстный Казань, певучий Казань,
Светлый Казань», новый Казань.
```
