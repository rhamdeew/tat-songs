# Оригинал

### Мияссәр Басыйров / Данил Хәсәншин - Кулларыңда гармуның

```
Бер сыздырып җибәр әле,
Кулларыңда гармуның.
Күңелгә сеңгән көйләрнең.
Онытыласы юк аның.

Кара атҡа кара дуга,
Кирәк каеш дилбегә.
Егет чаклар җилеп бара,
Тавышың калсын билгегә.

Бер яшлектә, бер картлыкта,
Бездән генә калмасын.
Бармакларың талса - талсың,
Кызлар йоклый алмасын.

Хисләремә ялкыннар сип,
Йөрәк пыскып янмасын!
Безнең хәтер калса - калсын
Кызларныкы калмасын!
```

# Перевод

### Мияссәр Басыйров / Данил Хәсәншин - Гармонь в твоих руках

```
Пролей-ка наигрыш звонкий -
Гармонь в твоих руках.
Напевы, в сердце впитанные,
Не забываются никак.

Вороному - вороную дугу,
Нужны ремни, вожжа тугая.
Юность мчится - пусть как знак
Твой голос в памяти играет.

И в юности, и в старости
Не только нам пусть будет дано.
Устанут пальцы - пусть устанут,
Девчатам не уснуть дано.

В чувства мои огонь осыпь,
Пусть сердце тленьем не болит!
Пусть память наша остаётся,
А девичья - пускай не спит!
```
