# Оригинал

### Мидхәт Миншин / Данил Хәсәншин - Кыңгырау тагалар яхшы атка

```
Кыңгырау тагалар яхшы атка
Яхшы атлар белән юл җиңел.
Аманатка тогры дуслар гына
Авыр чакларыңда кул бирер.

Кыңгырау тагалар яхшы атка
Яхшы атлар юлын югалтмый.
Яхшы аты булган егетләр дә
Сөйгән ярын юкка елатмый! Һай! Һай, һай!

Кыңгырау тагалар яхшы атка
Яхшы атлар илдә билгеле.
Яхшы аты булган егетләрнең
Даны калка үзенә меңнәре.
Һай, һай!
```

# Перевод

### Мидхәт Миншин / Данил Хасаншин - Колокольчики вешают на хорошего коня

```
Колокольчики вешают на хорошего коня
С хорошими лошадьми путь лёгок.
На доверие только верные друзья
В тяжкие времена подадут руку.

Колокольчики вешают на хорошего коня
Хорошие лошади путь не потеряют.
Парни с хорошими лошадьми тоже
Свою любимую не будут пустыми слезами лить! Хай! Хай, хай!

Колокольчики вешают на хорошего коня
Хорошие лошади в стране известны.
У парней с хорошими лошадьми
Слава вырастает как грива их коней.
Хай, хай!
```
