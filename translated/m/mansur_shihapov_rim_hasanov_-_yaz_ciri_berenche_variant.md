# Оригинал

### Мансур Шиһапов / Рим Хәсәнов - Яз җыры (беренче вариант)

```
Җылы нурлар җир өстеннән бөркелә,
Йөрәкләрне җилкендереп яз килә.

Язны котлап җырлаек, дуслар,
Безгә кушылып сайрый кошлар.
Зәңгәр күктә сөенешеп
Яна, яна йолдызлар.

Кояшка таба үрелә саф гөлләр,
Хезмәт җырына күмелә күңелләр.

Язны котлап җырлаек, дуслар,
Безгә кушылып сайрый кошлар.
Зәңгәр күктә сөенешеп
Яна, яна йолдызлар.

Бәхет балкый һәр кешенең йөзендә,
Безне сагынып язлар килә үзе дә!..

Язны котлап җырлаек, дуслар,
Безгә кушылып сайрый кошлар.
Зәңгәр күктә сөенешеп
Яна, яна йолдызлар.
```

# Перевод

### Мансур Шиһапов / Рим Хасанов - Песня весны (первый вариант)

```
Тёплые лучи по земле разливаются,
Сердца встряхнув, к нам весна возвращается.

Встретим весну песней, друзья,
С нами вместе поют птицы, звеня.
В синем небе, ликуя,
Пылают, пылают звёзды.

К солнцу тянутся чистые розы,
Песней труда наполняются грёзы.

Встретим весну песней, друзья,
С нами вместе поют птицы, звеня.
В синем небе, ликуя,
Пылают, пылают звёзды.

Счастье сияет на каждом лице,
Скучая по нам, сама весна идёт!..

Встретим весну песней, друзья,
С нами вместе поют птицы, звеня.
В синем небе, ликуя,
Пылают, пылают звёзды.
```
