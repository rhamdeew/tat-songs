# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Сәгать теле

```
Текелди сәгать телләре,
Текелди тек-тек кенә.
Сәгать суга, гомер уза,
Тек-тек-тек итеп кенә.

Текелди сәгать телләре,
Текелди тек-тек кенә.
Сәгать суга, гомер уза,
Тек-тек-тек итеп кенә.

Без чабабыз, да чабабыз,
Мәшәкать үтмәс кебек.
Һәм аларны бетерергә,
Бер гомер җитмәс кебек.

Текелди сәгать телләре,
Текелди тек-тек кенә.
Сәгать суга, гомер уза,
Тек-тек-тек итеп кенә.

Без чабабыз, да чабабыз,
Вакытта чаба гына.
Ни белән килеп җитәрбез,
Гомернең азагына.

Текелди сәгать телләре,
Текелди тек-тек кенә.
Сәгать суга, гомер уза,
Тек-тек-тек итеп кенә.
```

# Перевод

### Марат Кабиров / Ризван Хакимов - Стрелка часов

```
Тикают стрелки часов,
Тикают тик-так едва.
Часы бьют, жизнь утекает,
Тик-так-тиком едва.

Тикают стрелки часов,
Тикают тик-так едва.
Часы бьют, жизнь утекает,
Тик-так-тиком едва.

Мы бежим, все бежим снова,
Будто заботам нет края.
И чтобы их вычерпать дочиста,
Одной жизни, похоже, не хватит.

Тикают стрелки часов,
Тикают тик-так едва.
Часы бьют, жизнь утекает,
Тик-так-тиком едва.

Мы бежим, все бежим снова,
И время бежит за нами.
С чем дойдём, когда добежим
До последнего срока жизни?

Тикают стрелки часов,
Тикают тик-так едва.
Часы бьют, жизнь утекает,
Тик-так-тиком едва.
```
