# Оригинал

### Мәхмүт Хөсәен / Ренат Еникеев - Ак дулкыннар

```
Ах, дулкыннар,
Ак дулкыннар...
Уйный җилләр чыгуга.
Әй, кара күз, күңелем төште
Син сөйкемле сылуга.

Ах, дулкыннар,
Ак дулкыннар...
Ярга кага канатын.
Әй, кара күз, әллә сине
Чынлап тамы яраттым?!

Ах, дулкыннар,
Ак дулкыннар...
Уйный кояш нурында.
Әй, кара күз, ник моңарчы
Очрамадың юлымда?
```

# Перевод

### Махмут Хусаин / Ренат Еникеев - Белые волны

```
Ах, волны,
Белые волны...
Играют ветра на закате.
Эй, черноглазая, душа моя упала
На милую твою красу.

Ах, волны,
Белые волны...
К берегу прибивают крылья.
Эй, черноглазая, неужели тебя
По-настоящему я полюбил?!

Ах, волны,
Белые волны...
Играют в солнечных лучах.
Эй, черноглазая, почему же грустно,
Что не встретилась на пути моём?
```
