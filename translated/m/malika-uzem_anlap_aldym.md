# Оригинал

### Малика - Узем анлап алдым

```
«Сина атап яздым бу койне» - дип
Мина матур Бер кой уйнадын.
Яратулы назлы караш белэн
Минем йорэгемне яуладын.

«Мин яратам аны, яратам!» - дип
Ярсып типкэн иде йорэгем,
Э соныннан узем анлап алдым,
Эсир иткэн язган койлэрен.

Жыр доньясын гына яратасын,
Аннан башка яши алмыйсын.
Кунел биреп кой уйнаган чагын
Назлы итэ икэн карашын.

Язган койлэренэ соклануым
Йорэк белэн тоеп алгансын.
Мина атап матур кой язуын –
Соклануым очен жавабын.
```

# Перевод

### Малика - Сама поняла

```
«Для тебя писал я эту песню» -
Ты красивую мелодию играла.
Любящим нежным взглядом своим
Ты пленила моё сердце.

«Я люблю её, люблю!» -
Так билось моё сердце,
А потом я сама поняла,
Что пленили меня написанные песни.

Ты любишь только мир песен,
Без него жить не можешь.
Когда душой играешь песни,
Взгляд твой становится нежным.

Моё восхищение твоими песнями
Ты сердцем почувствовал.
Красивую песню для меня писать -
Это ответ на моё восхищение.
```
