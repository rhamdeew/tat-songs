# Оригинал

### Мансур Гыйлаҗев / Азат Хөсәенов - Сагышлы җыр (Авыл җыры)

```
Кара бөдрә чәчләремне
Үрдерәсем килмидер...
Эчем ялкын, тышым салкын,
Белгертәсем килмидер.

Безнең турдан казлар киткән.
Түгәрәк күл кипкәнгә.
Әллә, дускай, үкенәсеңме,
Яшь гомерләр үткәнгә?!

Әкрен барчы, әкрен барчы,
Безнең турдан узганчы.
Карап калыем, дускаем,
Ике күзем туйганчы.
```

# Перевод

### Мансур Гыйлаҗев / Азат Хөсәенов - Печальная песня (Деревенская песня)

```
Чёрные кудри свои
Заплетать не хочу я...
Внутри - пламя, снаружи - холод,
Показывать не хочу я.

От наших мест улетели гуси.
Круглое озеро высохло.
Неужто, дружок, жалеешь ты
О молодых годах, что прошли?!

Иди тише, иди тише,
Пока мимо наших мест пройдёшь.
Дай мне, дружок, наглядеться,
Пока не насытятся мои глаза.
```
