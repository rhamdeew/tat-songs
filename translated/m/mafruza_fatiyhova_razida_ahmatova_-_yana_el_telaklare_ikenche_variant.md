# Оригинал

### Мәфрүзә Фатыйхова / Рәзидә Әхмәтова - Яңа ел теләкләре (икенче вариант)

```
Ап-ак карларга уралып
Килсен безгә Яңа ел!
Карлар кебек ак бәхетле
Узсын безнең яңа юл.

Яңа елда бар теләкләр
Тормышка, чынга ашсын.
Һәркемнең якты иртәсе
Бәхетләр белән барсын.

Яңа ел, яңа өметләр,
Куанып, алып килсен.
Дөньядагы һәрбер кеше
Бәхетле була күрсен.

Яңа елда бар теләкләр
Тормышка, чынга ашсын.
Һәркемнең якты иртәсе
Бәхетләр белән барсын.

Иске елда борчылулар
Калсыннар, иске елда.
Яхшы кешеләр очрасын
Безнең барачак юлда.

Яңа елда бар теләкләр
Тормышка, чынга ашсын.
Һәркемнең якты иртәсе
Бәхетләр белән барсын.
```
