# Оригинал

### Марат Гобәйдуллин - Туган авылым - Каракүл

```
Тирә-ягың чиксез урман,
Иркен синең кырларың.
Бары сиңа багышлыймын
Йөрәгемнең моңнарын.

Әй, гүзәл Каракүлем,
Туган авылым минем.
Күңелемдә мәңге калыр
Сөекле газиз җирем.

Чия сабы буйларында
Ямьле Каракүле бар.
Туган якның матурлыгын
Үзендә саклый һаман.

Әй, гүзәл Каракүлем,
Туган авылым минем.
Күңелемдә мәңге калыр
Сөекле газиз җирем.

Эш сөючән халкың синең,
Җыр-моңга гашыйк алар.
Лаеклы алмаш булырлар
Безнең газиз балалар.

Әй, гүзәл Каракүлем,
Туган авылым минем.
Күңелемдә мәңге калыр
Сөекле газиз җирем.
```

# Перевод

### Марат Гобәйдуллин - Родная деревня - Каракуль

```
Вокруг тебя леса без края,
Просторны твоего поля.
Лишь тебе посвящаю
Песни сердца моего.

Ах, прекрасный Каракуль,
Деревня родная моя.
В душе навеки останется
Любимая, святая земля.

По берегам вишнёвых ручьёв
Есть милый Каракуль.
Красоту родных мест
Он в себе хранит всегда.

Ах, прекрасный Каракуль,
Деревня родная моя.
В душе навеки останется
Любимая, святая земля.

Трудолюбив твой народ,
Песне и мелодии верен.
Достойной сменой станут
Наши дорогие дети.

Ах, прекрасный Каракуль,
Деревня родная моя.
В душе навеки останется
Любимая, святая земля.
```
