# Оригинал

### Мостай Кәрим / Роза Сәхәүетдинова - Мәхәббәт җыры (җиденче вариант)

```
Талпынасың, ашкынасың,
Күзләреңдә нур гына;
Эй, яшь егет, йөрәгеңнән
Нинди шатлык ургыла?

- Янып-шашып, янып-шашып
Булдым гашыйк!..

- Чал чикәле узаман ир,
Нигә авыр канатың,
Нинди хәсрәт төште сиңа,
Нигә моңсу карашың?

- Янып-шашып, янып-шашып
Булдым гашыйк...
```

# Перевод

### Мостай Кәрим / Роза Сәхәүетдинова - Песня любви (седьмой вариант)

```
Рвёшься, мчишься, неуёмный,
Взгляд лучится, свет один;
Эй, юноша, из сердца
Какая радость бьёт ключом?

- В пламени, в безумном пламени
Я влюбился!..

- С седыми висками проходишь,
Почему тяжки твои крылья,
Какая скорбь на тебя пала,
Почему взгляд твой печален?

- В пламени, в безумном пламени
Я влюбился...
```
