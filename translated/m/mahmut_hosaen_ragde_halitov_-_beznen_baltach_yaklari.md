# Оригинал

### Мәхмүт Хөсәен / Рәгъде Халитов - Безнең Балтач яклары

```
Балтач, Балтач! дип балкый
Туган җир йолдызлары.
Ярышларда сынатмый
Безнең Балтач кызлары.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.

Балтач, Балтач! дип дәшә
Зәңгәр урман юллары.
Илгә байлык өләшә
Безнең Балтач уллары.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.

Балтач, Балтач! дип дәшә
Шау чәчәкләр болында.
Безнең күңел гел уйлый
Ил бәхете турында.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.
```

# Перевод

### Мәхмүт Хөсәен / Рәгъде Халитов - Наши Балтачские края

```
Балтач, Балтач! - сияют
Звезды родной земли.
В состязаниях не подводят
Наши Балтачские девы.

Чистый воздух, белый рассвет -
Наши Казанские края.
В труде страны прославлены
Наши Балтачские края.

Балтач, Балтач! - зовут
Дороги синих лесов.
Богатство стране даруют
Наши Балтачские сыновья.

Чистый воздух, белый рассвет -
Наши Казанские края.
В труде страны прославлены
Наши Балтачские края.

Балтач, Балтач! - зовут
Шумные цветы в лугах.
Наше сердце всегда помнит
О счастье родной земли.

Чистый воздух, белый рассвет -
Наши Казанские края.
В труде страны прославлены
Наши Балтачские края.
```
