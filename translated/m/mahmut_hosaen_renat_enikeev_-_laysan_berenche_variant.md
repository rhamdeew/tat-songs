# Оригинал

### Мәхмүт Хөсәен / Ренат Еникеев - Ләйсән (беренче вариант)

```
Ләйсән, диләр,
Ләйсән, диләр,
Ләйсән - язның яңгыры.
Дәртле яздан дәрман алып,
Дөрли йөрәк янгыны!..

Ярым, дисәң,
Янып сөйсәң,
Хәтәр түгел миңа үлем дә!..
Синең исем,
Синең хисең
Минем өчен ләйсән күңелемдә.

Җир яшәрә,
Кыр ямьләнә,
Ләйсән җылы таратса.
Икеләтә дөнья матур,
Өзелеп ярым яратса.

Бел син, бәгырем,
Сөю кадерен,
Әйләнмәсен йөрәк күмергә.
Синең исем,
Синең хисең
Минем өчен ләйсән гомергә.
```

# Перевод

### Мәхмүт Хөсәен / Ренат Еникеев - Ләйсән (первый вариант)

```
Ләйсән, говорят,
Ләйсән, говорят,
Ләйсән - дождь весны.
Смелой весной окрыляясь,
Пламенеет сердца жар!..

Милый, скажешь,
Вспыхнешь, любя,
Не страшна мне и смерть!..
Твоё имя,
Твоё чувство -
Для меня Ләйсән в душе.

Земля зеленеет,
Поле хорошеет,
Если Ләйсән дарит тепло.
Вдвойне прекрасен мир,
Если милый любит безмерно.

Знай ты, родной,
Цену любви,
Пусть не станет сердце углём.
Твоё имя,
Твоё чувство -
Для меня Ләйсән навсегда.
```
