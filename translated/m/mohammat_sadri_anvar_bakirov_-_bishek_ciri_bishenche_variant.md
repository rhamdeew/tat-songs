# Оригинал

### Мөхәммәт Садри / Әнвәр Бакиров - Бишек җыры (бишенче вариант)

```
Әлли-бәлли, ак бәби,
Ак бишектә ят, бәби.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Әлли-бәлли ит, бәби,
Тәмле йоклап кит, бәби.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Әлли-бәлли, бал гына,
Йокы сиңа ял гына.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Йокла, йокла, багалмам,
Үскәч бишеккә салмам.
Әлли-бәлли, бәбием,
Балдан татлы нәнием!
```

# Перевод

### Мухаммат Садри / Анвар Бакиров - Колыбельная песня (пятый вариант)

```
Баю-бай, белый малыш,
Лежи в белой колыбели, малыш.
Баю-бай, моё дитя,
Слаще меда моя бабушка.

Баю-бай, малыш,
Сладко засыпай, малыш.
Баю-бай, моё дитя,
Слаще меда моя бабушка.

Баю-бай, только мёд,
Сон для тебя - только покой.
Баю-бай, моё дитя,
Слаще меда моя бабушка.

Спи, спи, мой дорогой,
Подрастёшь - укачивать не буду.
Баю-бай, моё дитя,
Слаще меда моя бабушка!
```
