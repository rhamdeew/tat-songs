# Оригинал

### Мансур Шиһапов / Людмила Аланлы - Алмалары пешәр әле

```
Алмалары пешәр әле,
Җыеп алырсың әле.
Хәзер сагынмыйм, дисәң дә,
Бер сагынырсың әле.

Алмалары пешәр әле,
Җыеп сакларсың әле.
Алмагачлы юллар буйлап
Бергә атларбыз әле.

Алмалары пешәр әле,
Җыеп алырсың әле.
Алма кебек матур чаклар
Исеңә төшәр әле.

Алмалары пешәр әле,
Җыеп сакларсың әле.
Алмагачлы юллар буйлап
Бергә атларбыз әле.
```

# Перевод

### Мансур Шиһапов / Людмила Аланлы - Яблоки еще поспеют

```
Яблоки еще поспеют,
Соберешь их ты потом.
Хоть сейчас «не скучаю» скажешь -
Все равно соскучишься в нем.

Яблоки еще поспеют,
Сбережешь их ты потом.
По дороге с яблонями
Мы вдвоем еще пройдем.

Яблоки еще поспеют,
Соберешь их ты потом.
Дни, как яблоко, прекрасны,
Вспомнишь вдруг о них потом.

Яблоки еще поспеют,
Сбережешь их ты потом.
По дороге с яблонями
Мы вдвоем еще пройдем.
```
