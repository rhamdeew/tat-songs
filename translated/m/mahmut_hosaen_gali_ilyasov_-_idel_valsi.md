# Оригинал

### Мәхмүт Хөсәен / Гали Ильясов - Идел вальсы

```
Синең буеңда, Идел,
Шаулый кара урман.
Халыкта синең хакта.
Күпме җырлар туган.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Каеннарың моңы
Истән чыкмый бер дә.
Кыя таудай горур мин,
Син барында җирдә.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Синең ярыңда, Идел,
Яши минем ярым.
Каен кайрысыдай саф
Мәхәббәте аның.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Кайда гына йөрсәм дә
Синең җылың тоям,
Син - бабаларым җире.
Минем нурлы кыйблам!
```

# Перевод

### Махмут Хусаен / Гали Ильясов - Вальс Идели

```
На твоих берегах, Идел,
Шумит тёмный лес.
О тебе в народе
Столько песен сложено.

Я люблю тебя, Казань,
Тебя, лазурный Идел.
Чайки машут крыльями,
Волнуется душа.

Мелодия твоих берёз
Никого не оставит равнодушным.
Горжусь я, как утёс,
Что ты есть на этой земле.

Я люблю тебя, Казань,
Тебя, лазурный Идел.
Чайки машут крыльями,
Волнуется душа.

На твоём берегу, Идел,
Живёт моя половинка.
Чист, как берёзовая кора,
Её цветок любви.

Я люблю тебя, Казань,
Тебя, лазурный Идел.
Чайки машут крыльями,
Волнуется душа.

Где бы я ни бродил,
Всегда ощущаю твоё тепло,
Ты - земля моих предков,
Мой светлый куб!
```
