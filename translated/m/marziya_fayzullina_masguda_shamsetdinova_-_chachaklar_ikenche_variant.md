# Оригинал

### Мәрзия Фәйзуллина / Мәсгудә Шәмсетдинова - Чәчәкләр (икенче вариант)

```
Ак чәчәкләр, күк чәчәкләр,
Күк чәчәкләр аланда,
Без исемен белми торган
Чәчәкләр дә бар анда.

Кем буяган, ничек алар
Күп төрле төскә кергән,
Ә аннан соң кем аларга
Ислемай сөртеп йөргән?

Кайсы сары, кайсы кызыл,
Кайсы аның зәп-зәңгәр,
Тик шуңарга искитәрлек
Уйлап карасаң әгәр.

Кем буяган, ничек алар
Күп төрле төскә кергән,
Ә аннан соң кем аларга
Ислемай сөртеп йөргән?
```

# Перевод

### Марзия Файзуллина / Масгуда Шамсетдинова - Цветы (второй вариант)

```
Белые цветы, синие цветы,
В синих луговинах,
Есть и те цветы, что мы
Не знаем по названьям.

Кто раскрасил, как они
В столько красок облачились,
А потом кто по росе
Ароматом их умыл?

Одни жёлтые, другие алые,
Третьи синевой сияют,
Только стоит задуматься об этом -
Чудом всё это назовёшь.

Кто раскрасил, как они
В столько красок облачились,
А потом кто по росе
Ароматом их умыл?
```
