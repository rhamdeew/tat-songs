# Оригинал

### Марат Гобәйдуллин - Ветераннар (икенче вариант)

```
Авыр еллар хәтирәсе булып,
Сыкрый әле иске яралар.
Сезнең батырлыктан үрнәк ала
Җирдә күпме газиз балалар.

Кызганмыйча соңгы каныгызны,
Сакладыгыз изге Ватанны.
Сезнең исем җирдә генә түгел,
Хәтта Галәмгә дә яңрады.

Үткән тормышыгыз китап кебек:
Һәрбер бите чиста, пакь аның.
Сезнең бөек исем, һич тапланмый,
Күп гасырлар буена сакланыр.

Рәхмәт сезгә, безнең ветераннар,
Бу бәхетле, якты көннәр өчен.
Саклар һәркем җирдә тынычлыкны,
Аямыйча үзенең бөтен көчен.
```

# Перевод

### Марат Гобәйдуллин - Ветераны (второй вариант)

```
Память о тяжёлых годах,
Ноют по старому давние раны.
Сколько родных детей на земле
Взяли пример с вашего подвига.

Не жалея последней крови,
Сберегли вы святую Родину.
Ваше имя звучало не только на земле,
Но и echoed в самом космосе.

Прошедшая жизнь ваша словно книга:
Каждая страница её чиста и непорочна.
Ваше великое имя, нет сомнения,
Будет храниться века многие.

Благодарность вам, наши ветераны,
За эти счастливые, светлые дни.
Пусть каждый хранит на земле покой,
Не жалея для этого всех своих сил.
```
