# Оригинал

### Марс Хафизов / Виталий Агапов - Миңа кирәкми бүтән

```
Гомерлек төн булмас, әгәр
Яктылыкны эзләсәң.
Мәңгелек сөюләр булмас,
Әгәр үзең сөймәсәң.

Сөяр чакта сөйик әле,
Өметләр өзелмәгән.
Без кабызган мәхәббәтнең
Сүнәр чагы җитмәгән.

Бәхетләрен эзләп кемнәр
Янмаган да көймәгән.
Янып сөйгән ярларының
Саф сөюен күрмәгән.

Сөяр чакта сөйик әле,
Өметләр өзелмәгән.
Без кабызган мәхәббәтнең
Сүнәр чагы җитмәгән.

Кавышканда юлга сипкән
Чәчәк гомере үткән.
Бер сөйгән яр - мәңгелек яр,
Миңа кирәкми бүтән.

Сөяр чакта сөйик әле,
Өметләр өзелмәгән.
Без кабызган мәхәббәтнең
Сүнәр чагы җитмәгән.
```
