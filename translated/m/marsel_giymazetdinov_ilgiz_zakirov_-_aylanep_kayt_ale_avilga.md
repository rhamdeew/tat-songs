# Оригинал

### Марсель Гыймазетдинов / Илгиз Закиров - Әйләнеп кайт әле авылга

```
Юксынам дисең син, нигә соң?
Гомерең узмаган бит әле.
Туган як туфрагын сагынып,
Әйләнеп авылга кайт әле.

Саф чишмә сулары, һавасы,
Тугае, болыны, урманы
Кочаклап алырлар үзеңне -
Әйләнеп авылга кайт әле.

Күзләрең ямансу, боеклар -
Алар да авылны сагына.
Кайгырма, моңайма, юксынма,
Әйләнеп кайт кына авылга.

Юксынам дисең син, нигә соң?
Гомерең узмаган бит әле.
Туган як туфрагын сагынып,
Әйләнеп авылга кайт әле.
```

# Перевод

### Марсель Гыймазетдинов / Илгиз Закиров - Вернись хоть раз в родную деревню

```
Ты говоришь: скучаю... почему?
Ведь жизнь твоя ещё не вся прожита.
Соскучившись по родной земле-перину,
Вернись хоть раз в деревню, в край открытый.

Кристально-чистых родниковой воды вкус,
И воздух, и луга, и лес-дружина -
Они обнимут, примут без вопроса -
Вернись домой, в деревню возвратись же.

Твои глаза печальны, потемневшие -
Их взгляд тоскует по селе родимом.
Не грусти, не страдай, не томись зря -
Вернись всего на миг в деревню милую.

Ты говоришь: скучаю... почему?
Ведь жизнь твоя ещё не вся прожита.
Соскучившись по родной земле-перину,
Вернись хоть раз в деревню, в край открытый.
```
