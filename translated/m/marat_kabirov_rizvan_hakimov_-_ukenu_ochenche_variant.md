# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Үкенү (өченче вариант)

```
Син нигәдер әйтмәгәнсең,
Мин никтер күрмәгәнмен.
Сөюеңне сизмәгәнмен,
Ятларны эзләгәнмен.

Эх, никтер сиздермәгәнсең
Сиздерсәң сизәр идем.
Ярларымның барсы белән
Араны өзәр идем.

Көннәреңне бизәр идем,
Назыңда йөзәр идем.
Сөендерсәң - сөенер идем,
Әрнетсәң - түзәр идем.

Мин сине күрмәгәнмен шул,
Эй, хәзер күрер идем.
Алларыңа килер идем,
Артыңнан йөрер идем.

Үткән көннәр кире кайтса,
Нишләргә белер идем.
```

# Перевод

### Марат Кәбиров / Ризван Хәкимов - Сожаление (третий вариант)

```
Ты почему-то не сказала,
Я почему-то не увидел.
Твоей любви не почувствовал,
Чужих искал.

Эх, почему-то не давала понять,
Дала б - понял бы.
Со всеми твоими соперниками
Прервал бы отношения.

Твои дни украсил бы,
В ласке купался бы.
Радуешь - радовался бы,
Расстроишь - потерпел бы.

Я тебя не видел тогда,
Эх, сейчас бы увидел.
На встречу пришёл бы,
Вслед за тобой ходил бы.

Прошедшие дни вернутся если,
Знал бы, что делать.
```
