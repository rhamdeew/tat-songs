# Оригинал

### Мидхәт Әбделманов - Әнкәем (дүртенче вариант)

```
Ак бәс дигән чәчең
Коеладыр, әнкәй,
Коеладыр, бик еш тарама.
Балам кайтыр диеп
Матур күлмәк киеп,
Буш юлларга зарыгып карама.

Без дигәнчә түгел
Чит илләрнең күге,
Юллары да чиксез, бик олы.
Кайтам дисәм дә мин
Яннарыңа әнкәем,
Кайта алмасмын, ахры, бу юлы.

Баксам эчләренә
Бүләк итеп сиңа
Нинди гөлләр генә чәчимен?
Ал чәчсә дә балаң,
Гөл чәчсә дә балаң,
Бурчым бетмәс сиңа әнкәем.
```

# Перевод

### Мидхат Абдельманов - Мама (четвёртый вариант)

```
Твои седые волосы
Редеют, мамочка,
Редеют, не расчёсывай их часто.
Надеясь, что дитя вернётся,
Нарядившись в красивое платье,
Не уставай глядеть на пустые дороги.

Небо чужих стран
Не наше, родное,
Дороги их бесконечны, так велики.
Хоть и хочу вернуться,
Рядом с тобой, мамочка,
Не вернусь уже, увы, в этот раз.

Если бы я могла заглянуть в их сердца
В подарок тебе,
Какие бы цветы ни посадила?
Пусть твоё дитя их сажает,
Пусть твоё дитя цветы сажает,
Моя тоска по тебе не иссякнет, мамочка.
```
