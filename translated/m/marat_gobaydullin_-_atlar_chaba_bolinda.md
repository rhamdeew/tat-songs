# Оригинал

### Марат Гобәйдуллин - Атлар чаба болында

```
Томырылып атлар чаба,
Атлар чаба болында.
Кайсы егет җиңү яулар
Быел сабан туенда.

Кушымта:
Яшь йөрәге талпына,
Еракларга ашкына.
Җилдәй җитез егет күңеле -
Иярләнгән ат кына.

Милли көрәш, ат чабышы
Сабантуйның бизәге.
Тугры дусты аты өчен
Өзелә егет үзәге.

Кушымта:

Беренче килгән егетне
Көтә кызлар бүләге.
Кайсы кызны бәхетле итәр
Тынгы белмәс йөрәге.
```

# Перевод

### Марат Гобайдуллин - Лошади мчатся по лугу

```
Стремительно кони мчатся,
Лошади мчатся по лугу.
Кто из парней сорвёт победу
На нынешнем Сабантуе.

Припев:
Юное сердце трепещет,
Рвётся в даль, к горизонту.
Как ветер резв душою юный парень -
Ему бы лишь конь под седлом.

Борьба родная, скачки коней -
Сабантуй ими украшен.
За верного друга-коня
Рвётся парня сердце настежь.

Припев:

Парня, что первым домчится,
Ждёт девичий дар и ласка.
Кого он сделает счастливой
Своим беспокойным сердцем?
```
