# Оригинал

### Муса Җәлил / Исмай Шәмсетдинов - Килсәнә

```
Былбылкаем минем, килсәнә!
Кулларыңны кулга бирсәнә,
Кара-каршы килеп бер гөрләшик,
Гөрләү чараларын күрсәнә!

Күгәрченем монда калсана,
Күзләремә хәят салсана,
Йөрәк дуслыгының чынлыгына
Биргән бүләгемне алсана!
```

# Перевод

### Муса Җәлил / Исмай Шәмсетдинов - Приди же

```
Соловушка моя, приди же!
Руку к руке мне подай же,
Встретимся лицом к лицу - поворкуем,
Праздник воркованья увидишь же!

Голубка моя, останься здесь,
Жизнь в мои глаза всели же,
Во имя правды сердечной дружбы
Прими мой дар, что я дал тебе же!
```
