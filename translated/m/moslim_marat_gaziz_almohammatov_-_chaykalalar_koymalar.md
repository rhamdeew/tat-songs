# Оригинал

### Мөслим Марат / Газиз Әлмөхәммәтов - Чайкалалар көймәләр

```
Бар табигать серләшә, айлы төндә
Йөгереп уйный дулкыннар Дим өстендә.
Акчарлакка охшап, яр читендә
Чайкалалар көймәләр.

Көр тавышым яңгыратты матур җырлар,
Таңга калып тыңлады тын тургайлар,
Башка килде кайнар, дәртле уйлар
Чайкалганда көймәләр.

Җилләр исте су өстен сырландырып,
Иркәм чәчен дулкындай тирбәлдереп,
Сокландылар, ахры, безне күреп, -
Чайкалдылар көймәләр...
```

# Перевод

### Муслим Марат / Газиз Альмухамметов - Качаются лодочки

```
Вся природа шепчется в лунную ночь,
Бегут, играя, волны по Каме.
Словно чайки, у берега
Качаются лодочки.

Голос курая зазвучал красивыми песнями,
До утра застыв, слушали тихие жаворонки,
На ум пришли горячие, страстные думы,
Когда качались лодочки.

Ветры разнесли аромат над водой,
Волосы любимой, как волны, колыхая,
Залюбовались, наконец, нами глядя, -
Качались лодочки...
```
