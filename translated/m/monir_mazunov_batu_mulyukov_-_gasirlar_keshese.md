# Оригинал

### Мөнир Мазунов / Бату Мулюков - Гасырлар кешесе

```
Җир халкы яктыга, җылыга сусады,
Торса да йолдызлар нурында.
Бармы соң җиһанда тиңләрлек бер кояш
Гасырлар кешесе Ленинга!

Азатлык хакына, тынычлык хакына
Халыклар тудырган Гимнда
Туктаусыз яңгырый мәхәббәт хисләре
Гасырлар кешесе Ленинга!

Ураклы, чүкечле Советлар вымпелы
Ай ташын эреткән урынга,
Ай йөзен яктыртып, без һәйкәл салырбыз
Гасырлар кешесе Ленинга!

Ул әйткән акыллы, кадерле киңәшләр
Ирешер буыннан буынга.
Алдагы көрәшләр батыры баш ияр
Гасырлар кешесе Ленинга!
```

# Перевод

### Мөнир Мазунов / Бату Мулюков - Человек веков

```
Народ земли к свету и теплу тянулся,
Хотя и стоял в сиянии звезд.
Есть ли во вселенной равный солнцу
Ленину - человеку веков!

Во имя свободы, во имя покоя
В Гимне народов, рожденном в борьбе,
Непрестанно звучит чувство любви
Ленину - человеку веков!

Вымпел Советов с серпом и молотом
Там, где расплавлен лунный камень,
Осветив лицо Луны, воздвигнем мы памятник
Ленину - человеку веков!

Его мудрые, дорогие советы
Дойдут от поколения к поколению.
Герой грядущих битв склонит чело
Ленину - человеку веков!
```
