# Оригинал

### Муса Җәлил / Мансур Мозаффаров - Хат ташучы җыры

```
Җилбер-җилбер җил исә,
Гөл сабагын җил кисә.
Җилгә сәлам әйтер идем,
Сөйгәнемә җиткерсә.

Җилбер-җилбер җил үтә,
Чәч толымын җил сүтә.
Җил аркылы сәлам әйтсә,
Сөйгәнемнән шул җитә.

Җилбер-җилбер җилләрем,
Матур туган җирләрем.
Туган илдә сагынып көтә
Нечкә билле чибәрем!

Җил искәнен ишетсәм, мин
Җыр җырламый түзәлмим.
Җырлаганда сөйгәнемне
Телгә алмый түзәлмим.
```

# Перевод

### Муса Джалиль / Мансур Мозаффаров - Песня, несущая письмо

```
Веет легкий ветер,
Стебель розы ветер срезает.
Попросил бы я ветер передать привет,
Чтобы доставил моей любимой.

Веет легкий ветер мимо,
Прядь волос ветром развевает.
Если через ветер привет передаст,
От моей любимой дойдет.

Ветры мои легкие, шелестящие,
Мои красивые родные края.
Вдали на родине с тоской ожидает
Моя тонкая станом любимая!

Когда слышу, как дует ветер, я
Без песни не могу сдержаться.
А когда пою, свою любимую
На языке назвать не могу сдержаться.
```
