# Оригинал

### Муса Җәлил / Фәрит Яруллин - Соңгы көрәшкә

```
Талаучылар килде ил чигенә,
Бәхетебезне тартып алырга.
Күтәрел, ил, дошман өеренә,
Соңгы көрәш, соңгы давылга.

Ут ачыйк дошманнар явына
Корыдан, диңгездән, һавадан.
Соңгы кат без чабыйк тамырына,
Торырлык булмасын яңадан!

Мул башаклы алтын кырыбызга
Комсыз дошман кулын сузмасын.
Алтын бодай, татлы бал урнына
Авыз итсен ачы кургашын.

Ут ачыйк дошманнар явына
Корыдан, диңгездән, һавадан.
Соңгы кат без чабыйк тамырына,
Торырлык булмасын яңадан!

Үлем тапсын фашист безнең кулдан,
Соңгы хөкем шушы аңарга:
Без яудырган туплар ялкынында
Төтен булып очсын һавага.

Ут ачыйк дошманнар явына
Корыдан, диңгездән, һавадан.
Соңгы кат без чабыйк тамырына,
Торырлык булмасын яңадан!
```

# Перевод

### Муса Җәлил / Фәрит Яруллин - К последней битве

```
Грабители пришли к рубежам страны,
Счастье наше силой отнять.
Поднимайся, народ, на вражью орду,
На последний бой, на последний шквал.

Откроем огонь по вражьему полчищу
С суши, с моря, с высоты небес.
В последний раз срубим корень их,
Чтоб уже не поднялись!

Наши золотые, хлебные нивы
Пусть не тронет жадная рука.
Вместо сладкой пшеницы, вместо мёда
Пусть проглотят горький свинец.

Откроем огонь по вражьему полчищу
С суши, с моря, с высоты небес.
В последний раз срубим корень их,
Чтоб уже не поднялись!

Пусть фашист от нашей руки найдёт смерть,
Вот ему последний приговор:
В пламени наших грохочущих пушек
Пусть дымом взовьётся в воздух.

Откроем огонь по вражьему полчищу
С суши, с моря, с высоты небес.
В последний раз срубим корень их,
Чтоб уже не поднялись!
```
