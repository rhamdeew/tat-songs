# Оригинал

### Мәнзүмә Сибгатуллина / Ильмира Нәгыймова - Сары роза

```
Юл буенда сары роза
Кызганмыйча тапталган.
Кемнең икән шул кадәрле
Мәхәббәте тапланган?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?

Сары роза - горур чәчкә,
Кемнең назлы бүләге,
Әйтелмичә калган сүзме,
Әллә соңгы теләге?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?

Юл буена ташлаганнар
Затлы чәчәк розаны.
Сөюләрне шул чәчәктәй
Сез ташларга ризамы?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?
```

# Перевод

### Манзума Сибгатуллина / Ильмира Нагиева - Жёлтая роза

```
По дороге жёлтая роза
Без сожаления растоптана.
Чья же это такая,
Чья любовь растоптана?

Жёлтая роза, жёлтая роза,
Кто тебя бросил,
Когда целовал, как цветок,
Почему началось целование?

Жёлтая роза - гордый цветок,
Чей это нежный подарок,
Не высказанное слово,
Или последнее желание?

Жёлтая роза, жёлтая роза,
Кто тебя бросил,
Когда целовал, как цветок,
Почему началось целование?

По дороге бросили
Драгоценную розу.
Разве вы не против,
Чтобы ваши поцелуи были как этот цветок?

Жёлтая роза, жёлтая роза,
Кто тебя бросил,
Когда целовал, как цветок,
Почему началось целование?
```
