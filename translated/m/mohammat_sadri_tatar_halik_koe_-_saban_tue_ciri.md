# Оригинал

### mohammat_sadri_tatar_halik_koe_ - Мөхәммәт Садри / Татар халык көе - Сабан туе җыры

```
Сабан туе - хезмәт туе,
Шатлык һәм бәхет туе, шул;
Уйнап, биеп, җырлап, көлеп,
Бәйрәм итик көн буе.

Шау-гөр килә киң болыннар,
Кырлар һәм су буйлары, шул;
Котлы булсын, колхоздашлар,
Ямьле сабан туйлары!

Бәйрәм итик шатланышып,
Туганнар, бергә-бергә, шул;
Безнең бергә булуыбыз
Китсен бөтен гомергә.
```

# Перевод

### Мөхәммәт Садри / Татар халык көе - Песня Сабантуя

```
Сабантуй - трудовой наш праздник,
Праздник радости, счастья, так;
С песней, пляской, смехом, играми
Будем праздновать целый день.

Шумно гудят просторы лугов,
Поля и берега рек, так;
Счастья вам, колхозные друзья,
Славный праздник Сабантуя!

Празднуем, ликуя вместе,
Родные, вместе-вместе, так;
Пусть единство наше дружное
Останется на весь наш век.
```
