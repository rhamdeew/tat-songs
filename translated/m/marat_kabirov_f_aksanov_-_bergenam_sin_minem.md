# Оригинал

### Марат Кәбиров / Ф. Әкъсәнов - Бергенәм син минем

```
Язларын, көзләрен
Мин сине эзләдем,
Сиңа тиң ярларны тапмадым.
Алыйм да кулыннан
Гомерләр юлыннан
Бергәләп атлыйк без, аппагым.

Бергенәм син минем,
Өзелеп сөюем
Озайтсын бәхетле мәлләрне.
Вәгъдәләр бирешик,
Яшәргә сөешеп
Гомерләр соңына кадәрле.

Вәгъдәләр нык булсын,
Балкытырлык булсын
Күңелләрне сөю нурыннан.
Авырлыкны җиңеп,
Сөеп һәм сөелеп
Бергә атлыйк гомер юлыннан
```

# Перевод

### Марат Кәбиров / Ф. Әкъсәнов - Мой единственный

```
Весной и осенью
Я тебя искал,
Не нашёл тебе равной любви.
Возьму за руку
По дороге жизни -
Пойдём вместе, белая моя.

Ты мой единственный,
Любовь до боли
Пусть продлит счастливые мгновения.
Дадим друг другу слово
Жить, любя друг друга,
До последнего дня пути.

Пусть обещания будут крепки,
Пусть сияют,
Согревая сердца лучом любви.
Побеждая трудности,
Любя и любимыми,
Вместе пройдём дорогу жизни
```
