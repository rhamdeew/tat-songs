# Оригинал

### Мәдинә Миңнегулова - Шәһри Болгар (икенче вариант)

```
Элек бу җирдә
Зур шәһәр булган.
Исеме аның -
Шәһри Болгар.
Бик күп илләрдән
Килгән кәрваннар.
Су юлы буйлап
Йөзгән кораблар.
Шәһри Болгар.

Торган урыны
Идел буенда.
Серле ташлар тик
Калган шул монда.
«Изге җир» - аяк
Басарга куркам.
Әче сагышка
Дучармын, дучар.
Шәһри Болгар.

Җиргә күмелгән
Урамнарында
Чулпы тавышы
Азмы чыңлаган.
Тирән моңга ия
Чын татар көен
Нечкә билле кызлар
Азмы җырлаган.
Шәһри Болгар.

Азан яңгырап торган
Манараларның хәрабәләре
Җиргә иңгәннәр.
Сиңа килгән татар
Балалары бүген
Нигез ташыңа
Башларын игәннәр.
Шәһри Болгар.
```

# Перевод

### Мадия Миннегулова - Город Болгар (второй вариант)

```
Раньше на этой земле
Был большой город.
Имя его -
Шэри Болгар.
Из множества стран
Приходили караваны.
По водному пути
Плывали корабли.
Шэри Болгар.

Стоял на берегу
Волги.
Только загадочные камни
Остались здесь.
«Святая земля» - боюсь
Ступать ногой.
На горькую тоску
Наткнусь, наткнусь.
Шэри Болгар.

На погребённых в землю
Её улицах
Голубиный голос
Ли звенел.
Истинной татарской песне,
Обладающей глубокой мелодией,
Станые девочки с тонкими талиями
Ли пели.
Шэри Болгар.

Руины минаретов,
Где раздавался призыв на молитву,
Упали на землю.
Пришедшие к тебе сегодня
Татарские дети
Поклонились головой
Твоему фундаменту.
Шэри Болгар.
```
