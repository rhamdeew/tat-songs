# Оригинал

### Миңлегөл Хисамова / Венер Абдуллин - Яңа ел төнендә

```
Талпынышып кына карлар ява,
Әйтерсең дә үзем күбәләк.
Рәхәт миңа хыялларым белән
Биюләре очып, чүгәләп.

Учка эреп төшкән йолдыз-кардай
Ах, бу елым күздән югалды.
Көйсезләнеп кенә, үзсүзләнеп
Сабый гына җырым тумалды.

Яңа елның шигри бу төнендә
Йөрәгемне өтә мең юрау.
Мин үзем дә хыял атларында
Чыңлап кына калыр кыңгырау.

Учка эреп төшкән йолдыз-кардай
Ах, бу елым күздән югалды.
Көйсезләнеп кенә, үзсүзләнеп
Сабый гына җырым тумалды.
```

# Перевод

### Миңлегөл Хисамова / Венер Абдуллин - В новогоднюю ночь

```
Снег летит, чуть трепеща крылами,
Словно я - легчайшая бабочка.
Мне так сладко с грёзами моими -
В пляске взмыть и низко приседать.

Как звезда-снежинка на ладони -
Ах, мой год из глаз моих исчез.
Непослушный, упрямый, капризный -
Новорожденный мой тонкий напев.

В эту поэтичную новогоднюю ночь
Тысяча гаданий жжёт мне сердце.
Я и сам на конях моих мечтаний -
Лишь звенящий, дальний колокольчик.

Как звезда-снежинка на ладони -
Ах, мой год из глаз моих исчез.
Непослушный, упрямый, капризный -
Новорожденный мой тонкий напев.
```
