# Оригинал

### Марат Гобәйдуллин - Бәхетле балачак

```
Балачакның шат җырлары
Яңгырасын бөтен җирдә.
Һәрчак тату, дуслар булып
Яшик без туган илдә.

Без - бәхетле, шат балалар,
Безнең уйнар, көләр чак.
Гел шатлыклы җырлар белән
Узсын безнең балачак.

Шатланып каршы алабыз
Һәрбер туар иртәне.
Аяз күктә якты кояш
Безне назлап иркәли.

Без - бәхетле, шат балалар,
Безнең уйнар, көләр чак.
Гел шатлыклы җырлар белән
Узсын безнең балачак.

Безне көтә якты тормыш,
Шатлык һәм бәхет алда.
Лаеклы хезмәт итәрбез
Газиз туган Ватанга!

Без - бәхетле, шат балалар,
Безнең уйнар, көләр чак.
Гел шатлыклы җырлар белән
Узсын безнең балачак.
```

# Перевод

### Марат Гобайдуллин - Счастливое детство

```
Радостные песни детства
Пусть звучат повсюду.
Всегда в мире и дружбе
Живём мы в родном краю.

Мы - счастливые, радостные дети,
Нам играть и смеяться пора.
Пусть лишь с радостными песнями
Пролетит наше детство.

С радостью мы встречаем
Каждое новое утро.
Ясное небо, яркое солнце
Нас ласково балуют.

Мы - счастливые, радостные дети,
Нам играть и смеяться пора.
Пусть лишь с радостными песнями
Пролетит наше детство.

Нас ждёт светлая жизнь,
Радость и счастье впереди.
Достойно трудиться мы будем
Для любимой Родины!

Мы - счастливые, радостные дети,
Нам играть и смеяться пора.
Пусть лишь с радостными песнями
Пролетит наше детство.
```
