# Оригинал

### Марсель Галиев / Ганс Сәйфуллин - Миләүшә (өченче вариант)

```
Каен белән бас янәшә,
Сиңа бик тә килешә.
Болында бар чәчәк сиңа
Елмаюын өләшә.
Авылыбызның гүзәле дип
Бөтен халык сөйләшә, Миләүшә.
Каен белән бас янәшә,
Сиңа бик тә килешә, Миләүшә.
Авылыбызның гүзәле дип
Бөтен халык сөйләшә.

Каен белән бас янәшә,
Сиңа бик тә килешә.
Мең ел гомер теләп сиңа
Кәккүкләр дә эндәшә.
Бер егеткә күз сирпесәң
Җитмеш егет көнләшә, Миләүшә.
Каен белән бас янәшә,
Сиңа бик тә килешә, Миләүшә.
Бер егеткә күз сирпесәң
Җитмеш егет көнләшә.

Каен белән бас янәшә,
Сиңа бик тә килешә.
Кырда, җырда һәм җәйләүдә
Бар төс сиңа тиңләшә.
Сабантуе батырына
Авыл сине ярәшә, Миләүшә.
Каен белән бас янәшә,
Сиңа бик тә килешә, Миләүшә.
Сабантуе батырына
Авыл сине ярәшә.
```
