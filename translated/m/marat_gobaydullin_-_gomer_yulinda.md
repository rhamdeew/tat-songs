# Оригинал

### Марат Гобәйдуллин - Гомер юлында

```
Гомер юлларыннан балачакны эзлим,
Күңелемдә һаман бала мин.
Балачактан бәхетлерәк мәлне
Эзләсәм дә, никтер, табалмыйм.

Гомер юлларыннан яшьлегемне эзлим,
Эзләсәм дә, никтер, табалмыйм.
Тик шулай да ышанычны өзмим,
Өмет белән яшим һаман мин.

Гомер юлларыннан сабыр гына,
Салмак кына атлап барам мин.
Газиз туган илдән кадерлерәк
Бер җирне дә бүтән табалмыйм.
```

# Перевод

### Марат Гобайдуллин - На жизненном пути

```
С жизненных путей детство ищу,
В сердце всё ещё ребёнок я.
Времени счастливее чем в детстве
Ищу, но что-то не нахожу.

С жизненных путей молодость ищу,
Ищу, но что-то не нахожу.
Но всё же не теряю веру,
С надеждой живу всё ещё я.

По жизненным путям терпеливо,
Размеренно шагаю я.
Родной страны дороже
Ни одной другой земли не найду.
```
