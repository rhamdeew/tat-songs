# Оригинал

### Мөнир Мазунов / Бату Мулюков - Җырла, сеңелем

```
Җырла, сеңелем, иң яраткан көйләреңне,
Утыртканда нәркиз гөлләреңне.
Җырларыңнан сандугачлар көнләшсеннәр,
Сине тыңлап, күгәрченнәр гөрләшсеннәр.

Җырла, сеңелем, сафлыгыңны, пакълегеңне,
Ишетсеннәр тулы шатлыгыңны,
Була кайчак уйланулар, моңаюлар,
Җансыз яткан ташлар гына уйланмыйлар.

Җырла, сеңелем, хисләреңне тотма бәйдә,
Яшьлегеңне ныграк чыңлат әйдә!
Канатландыр кешеләрне күркәм эшкә,
Күңел тоя,күңел сизгер, күңел нечкә.
```

# Перевод

### Мөнир Мазунов / Бату Мулюков - Пой, сестренка

```
Пой, сестренка, самые любимые напевы,
Пока сажаешь нарциссы - цветы свои.
Пусть соловьи от песен твоих ревнуют,
Пусть голуби, слушая, воркуют.

Пой, сестренка, о чистоте и светлой правде,
Пусть услышат радость полную твою.
Бывают думы, бывает тоска и тени,
Лишь безжизненные камни не знают раздумий.

Пой, сестренка, не держи чувств на привязи,
Зазвени громче, молодость, смелей!
Окрылюй людей на прекрасные дела,
Сердце чует, сердце чутко, сердце нежно.
```
