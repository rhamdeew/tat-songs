# Оригинал

### Мөхәммәт Садри / Татар халык көе - Каз канаты (беренче вариант)

```
Каз канатларын эзләргә
Төштем су буйларына.
Су буйларында йөргәндә,
Син генә уйларымда.

Каз канатларын санадым,
Тезелеп кагынганда.
Тулган айларга карадым,
Өзелеп сагынганда.

Каз канатын кагынганда,
Мамыкларын җил ала.
Яшь вакыттагы мәхәббәт
Мәңгелек булып кала.
```

# Перевод

### Мухаммат Садри / Татарская народная песня - Гусиное крыло (первый вариант)

```
В поисках гусиных крыльев
К берегам воды сошла.
Вдоль речных брожу раздолий,
Только ты в моих мечтах.

Пересчитала гусиные крылья,
Когда стаей пролетали.
Смотрела я на полную луну,
Когда тоской изнывала.

Когда летят гусиные крылья,
Ветер пух с них уносит.
Любовь из юности далёкой
Вечной остаётся.
```
