# Оригинал

### Марат Кәбиров / Фәрит Хатыйпов - Көтәм әле

```
Һәркем җиргә пары белән туа,
Язмышына язган яры була...
...Син һаман да мине эзлисеңдер,
Өмет өзмисеңдер табуыңа.

Ялгызыңа авырдыр бит, диләр,
Авырлыкны горур үтәм әле.
Син, барыбер, мине табарсың дип,
Мин һаман да сине көтәм әле.

Бер мин генә ялгыз түгелмендер,
Язмышыма язган ярым бардыр.
Язмышыбыз һаман бердер әле,
Юлларыбыз гына аерылгандыр.

Ялгызыңа авырдыр бит, диләр,
Авыр да бит, сабыр итәм әле.
Син, барыбер, мине табарсың дип,
Мин һаман да сине көтәм әле.

Өметеңне генә өзә күрмә,
Синең әле мине табасың бар.
Күзләреңне танып алачакмын
Меңәрләгән күзләр арасыннан.

Ялгызыңа авырдыр бит, диләр,
Авыр да соң, сабыр итәм әле.
Син, барыбер, мине табарсың дип,
Мин һаман да сине көтәм әле.
```

# Перевод

### Марат Кабиров / Фарит Хатыпов - Всё жду

```
Каждый рождается на свете с парой,
Судьбой написанный суженый бывает...
...Ты всё ещё меня ищешь, наверно,
Надежду не теряешь найти меня.

Одному тяжело, говорят,
Тяжесть гордо перенесу пока.
Ты, всё равно, меня найдёшь, думая,
Я всё ещё тебя жду.

Не я один одинок, наверное,
Есть написанный судьбой мой суженый.
Судьбы наши всё ещё едины,
Только пути разошлись.

Одному тяжело, говорят,
Тяжело, но потерплю пока.
Ты, всё равно, меня найдёшь, думая,
Я всё ещё тебя жду.

Не обрывай только надежду,
Ты ещё меня найдёшь.
Твои глаза узнаю
Среди тысяч других глаз.

Одному тяжело, говорят,
Тяжело, но потерплю пока.
Ты, всё равно, меня найдёшь, думая,
Я всё ещё тебя жду.
```
