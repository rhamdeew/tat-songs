# Оригинал

### Айрат Кәримов / Ризван Хәкимов - Китәсе килми

```
Туган якларга кайтыр чакларда
Хәтта бер минут көтәсе килми.
Яшьлек тавышыдай авылым каршылый,
Мине һич тә ят итәсе килми.

Монда дусларым, монда ишләрем,
Дусларсыз юллар үтәсе килми.
Әткәй кабере... Әнкәй кабере ...
Яшьләрем тама, сөртәсе килми.

Сынаусыз-нисез, сагышсыз-моңсыз
Матур көннәрнең иртәсе килми.
Туган төягем, синдә йөрәгем,
Синнән аерылып китәсе килми.
Китәсе килми ...
```

# Перевод

### Айрат Кәримов / Ризван Хәкимов - Уезжать не хочется

```
Когда в родные стороны вернуться срок приходит,
И минуты ждать не хочется совсем.
Деревня встречает, словно голос юности -
Меня она не делает чужим.

Здесь друзья мои, здесь близкие мои,
Без друзей дороги не пройти.
Могила отца... могила матери...
Слёзы катятся - не хочется стереть.

Без испытаний, без печалей и без песен
Не приходит утро светлых дней.
Родной мой край, в тебе моё сердце,
От тебя уйти, расстаться не хочу.
Не хочется уезжать...
```
