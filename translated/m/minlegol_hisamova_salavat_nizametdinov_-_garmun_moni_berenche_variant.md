# Оригинал

### Миңлегөл Хисамова / Салават Низаметдинов - Гармун моңы (беренче вариант)

```
Гармун моңы тавышлары
Нигә тын калдың әле?
Йөрәгемне сурып алып,
Утларга салдың әле.

Гармун моңы тавышлары,
Кемне сагындың әле?
Кемдер өчен югалдың да,
Минем өчен табылдың.

Гармун моңы тавышлары,
Нигә сызландың әле?
Сүнеп барган йөрәгемә
Нигә күз салдың әле?

Гармун моңы тавышлары,
Кемне сагындың әле?
Кемдер өчен югалдың да,
Минем өчен табылдың.

Гармун моңы тавышлары,
Баттым хис диңгезенә.
Ярсый-ярсый эй моңлана,
Уй салыр өнсезенә.

Гармун моңы тавышлары,
Кемне шулай сагындың?
Кемдер өчен югалдың да,
Минем өчен табылдың.
```

# Перевод

### Минлегуль Хисамова / Салават Низаметдинов - Мелодия гармони (первый вариант)

```
Звуки мелодии гармони,
Почему замолчал ты?
Высосав сердце моё,
В огонь его бросил ты.

Звуки мелодии гармони,
Кого тоскуешь ты?
Для кого-то потеряв, ты
Мне нашёлся.

Звуки мелодии гармони,
Почему заболел ты?
На угасающее сердце моё
Почему взглянул ты?

Звуки мелодии гармони,
Кого тоскуешь ты?
Для кого-то потеряв, ты
Мне нашёлся.

Звуки мелодии гармони,
Погрузился я в море чувств.
Больно-больно эй, поёшь,
Думу кладёшь в безмолвие.

Звуки мелодии гармони,
Кого так тоскуешь ты?
Для кого-то потеряв, ты
Мне нашёлся.
```
