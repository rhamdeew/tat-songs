# Оригинал

### Мостай Кәрим / Хисмәтулла Дәүләтов - Мин кабаттан туыр булсам

```
Мин кабаттан туыр булсам,
Чишмә булып туыр идем.
Чылтыр-чылтыр назлап кына,
Аягыңны юыр идем.

Күзләреңә (йөзләреңә) генә түгел,
Әзләреңә табынамын.
Көнгә кырык тапкыр сүнеп,
Кырык тапкыр кабынамын.

Ал мине күк, ай янында,
Йолдыз булып яныймчы мин.
Очкан чакта капыл сүнеп,
Ап-ак көлгә калыймчы мин.
```

# Перевод

### Мостай Кәрим / Хисмәтулла Дәүләтов - Если бы я вновь родился

```
Если б я вновь родился,
Родником бы я стал.
Звонко-звонко, нежно,
Твои ноги умывал.

Не только твоим глазам (лицу),
Я и твоим редким вздохам молюсь.
Сорок раз за день гасну,
Сорок раз заново разжжусь.

Возьми меня к небу, рядом с луной,
Пусть звездой пылаю я.
И в полете вдруг угаснув,
Пусть останусь белым пеплом я.
```
