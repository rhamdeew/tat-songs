# Оригинал

### Марсель Гыймазетдинов / Алсу Салихова - Синең өчен (тугызынчы вариант)

```
Синең өчен алсу таңнар ата,
Синең өчен исә җилләр дә.
Синең өчен сандугачлар сайрый,
Синең өчен чәчәк җирләрдә.

Синең өчен тук башаклар шаулый
Иксез-чиксез басу-кырларда.
Синең өчен яңа көннәр туа.
Синең өчен туа җырлар да.

Синең өчен салмак җилләр исә,
Синең эчен яшен яшьни, дим.
Синең өчен җиргә килүләрем,
Синең өчен генә яшим мин.

Тик белә күр минем кадеремне -
Югалтма син мине, югалтма...
...Мин булганга шулай гүзәл дөнья,
Син булганга шулай таң ата.
Соң булса да кадеремне бел син,
Һич югалтма мине, югалтма...
```

# Перевод

### Марсель Гыймазетдинов / Алсу Салихова - Для тебя (девятый вариант)

```
Для тебя алеют зори ранние,
Для тебя и ветры веют в тишине.
Для тебя поют в садах соловушки,
Для тебя цветут цветы на всей земле.

Для тебя шумят колосья спелые
В бесконечных полях и на равнине.
Для тебя рождаются рассветы новые,
Для тебя рождаются и песни мне.

Для тебя струятся ветры мягкие,
Для тебя, скажу я, молнии внутри.
Для тебя пришел я на эту землю,
Для тебя одной я в мире и живу.

Только знай же цену мне, прошу тебя -
Не теряй меня, не потеряй...
...Мир прекрасен лишь когда я рядом,
И рассвет встает, когда ты рядом.
Поздно ли - но цену мне узнай ты,
Никогда меня не потеряй...
```
