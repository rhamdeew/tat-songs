# Оригинал

### Мәхмүт Һибәт / Сөләймән Абдуллин - Каенгабак

```
Каен гына кабак моңнар сымак,
Сыртларыннан искән җилкәе, җилкәе.
Газиз генә микән, ай үз микән,
Һәр кемгә дә туган җиркәе, җиркәе.

Каен гына кабак, бөдрә куак,
Бөдрәләрең бетә көз җитсә, көз җитсә.
Бер көн яшәүең дә бер гомердер,
Туып үскән җирем үз итсә, үз итсә.

Каен гына кабак буйларында
Карчыгалар канат кагына, кагына.
Читтә чакта илкәй уйларымда,
Җиркәемне күңел сагына, сагына.
```

# Перевод

### Мәхмүт Һибәт / Сөләймән Абдуллин - Берёзовая тыква

```
Берёзовая тыква - как тихий напев,
По спине твоей ветер струится, струится.
Мила ли она, уж своя ли она -
Каждому дорог родной его край, его край.

Берёзовая тыква, кудрявый кусток,
Кудри меркнут, когда осень придёт, придёт.
И один прожитый день - это жизнь,
Если родная земля нас признаёт, признаёт.

По долинам берёзовой тыквы
Коршуны крыльями машут, машут.
Вдалеке - в мыслях отчизна моя,
И сердце по краю тоскует, тоскует.
```
