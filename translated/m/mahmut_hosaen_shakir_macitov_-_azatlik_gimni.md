# Оригинал

### Мәхмүт Хөсәен / Шакир Мәҗитов - Азатлык гимны

```
Туганнар, тыңлагыз,
Кушылып җырлагыз,
Безнең җыр бәйсезлек зурында.
Азатлык байрагы,
Чын хаклык байрагы
Татар халкының үз кулында.

Дошманнар, кызмагыз
Без ирек куймабыз
Татарның бәхетен буарга!
Халыклар мул яш-ар,
Кояш гел нур чәчәр,
Татарстан бәйсез булганда!

Азатлык турында
Буыннан-буынга
Шаулар җыр, таралыр безнең дан.
Түгелмәс кайнар кан,
Сүрелмәс алсу таң,
Яшәсен бәйсез Татарстан!
```

# Перевод

### Мәхмүт Хөсәен / Шакир Мәҗитов - Гимн свободы

```
Братья, слушайте,
Вместе пойте,
Песня наша о великой свободе.
Знамя вольности,
Знамя правды -
В руках народа татарского оно.

Враги, не пылите,
Не дадим вам
Счастье татар задушить!
Народы жить будут в достатке,
Солнце будет щедро сиять,
Пока Татарстан свободен!

О свободе
Из рода в род
Песнь прогремит, разнесется наша слава.
Не прольется жаркая кровь,
Не угаснет алая заря,
Да здравствует свободный Татарстан!
```
