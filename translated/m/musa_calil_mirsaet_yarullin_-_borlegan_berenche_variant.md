# Оригинал

### Муса Җәлил / Мирсәет Яруллин - Бөрлегән (беренче вариант)

```
Җыеп кырдан бөрлегән,
Килгән иркәм, бер генәм.
Ә мин аңа сөюемне
Белдерәлми тилмерәм.

Кунак итә гел генәм
Мине бөрлегән белән.
Белми ләкин - иреннәре
Ин өлгергән бөрлегән.
```

# Перевод

### Муса Җәлил / Мирсәет Яруллин - Бөрлегән (первый вариант)

```
С поля срывая бөрлегән,
Пришла, родная, ты одна.
А я любовь свою не смею
Ей выразить - тоской полна.

Угощает лишь ею
Она меня бөрлегәном.
Не знает, что её губы -
Самый спелый бөрлегән.
```
