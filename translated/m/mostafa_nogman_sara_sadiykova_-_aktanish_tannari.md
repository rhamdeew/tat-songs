# Оригинал

### Мостафа Ногман / Сара Садыкова - Актаныш таңнары

```
Акчарлаклар очып уйный
Актаныш күлләрендә.
Ул гына яши, елмаеп,
Күңелем түрләрендә.

Актаныш таңнары,
Ул минем йөрәк серемне
Аңлармы?

Зәңгәр кичтә Агыйделгә
Йолдызлар коела сыман.
Аны гына сайлап алдым
Бик күпләр арасыннан.

Актаныш таңнары,
Ул минем йөрәк серемне
Аңлармы?

Очып кына китәр идем
Актаныш якларына.
Өзгәләнгән минутым бар
Юксынган чакларымда.

Актаныш таңнары,
Ул минем йөрәк серемне
Аңлармы?
```

# Перевод

### Мостафа Ногман / Сара Садыкова - Рассветы Актаныша

```
Чайки играют, летая
Над озёрами Актаныша.
Только она живёт, улыбаясь,
В образах моей души.

Рассветы Актаныша,
Поймёт ли она
Тайну моего сердца?

В синюю ночь к Агидели
Звёзды сыплются, словно город.
Только её избрал я
Средь множества других.

Рассветы Актаныша,
Поймёт ли она
Тайну моего сердца?

Улетел бы я скорей
На края актанышские.
Есть моя изменившаяся минута
В времена тоски по прошедшему.

Рассветы Актаныша,
Поймёт ли она
Тайну моего сердца?
```
