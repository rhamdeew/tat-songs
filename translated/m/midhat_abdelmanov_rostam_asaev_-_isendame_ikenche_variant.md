# Оригинал

### Мидхәт Әбделманов / Рөстәм Асаев - Исеңдәме? (икенче вариант)

```
Исеңдәме искән җилләр,
Исеңдәме көн дәме,
Таң алдыннан Зөһрә йолдыз
Безне алдан йөрткәне?
Истә, истә айлы кичләр,
Ярымның ирен тәме.
Сөйгән ярсыз, туган илсез
Каламы яшәү яме?

Исеңдәме туган ягым,
Борма-борма юллары?
Узган яшьлек шул юлларда,
Кышын карларын ярып.
Күз алдында туган ягым,
Олы юлның тузаны.
Тик күрмәдем шул юллардан
Яшьлегемнең узганын.

Исеңдәме ак каеннар,
Каен суының тәме.
«Сөям» дигән назлы сүзләр
Төшеңдәме, өндәме?
Истә һаман ак каеннар,
Каен суының тәме.
Таң алдыннан Зөһрә йолдыз.
```

# Перевод

### Мидхәт Әбделманов / Рөстәм Асаев - Помнишь? (второй вариант)

```
Помнишь ветры, что веяли?
Помнишь дыханье дня,
Как предутренняя Зөһрә-звезда
Вела нас вперед, маня?
Помню, помню лунные вечера,
Вкус губ моей половины.
Без любимой, без отчизны родной
Останется ль сладость жизни?

Помнишь мой край родимый,
Те извилистые пути?
Юность прошла по ним,
Снег зимой разрезая в пути.
Перед глазами родимый край,
Пыль большого тракта.
Только я не увидел на тех дорогах,
Как миновала юность когда-то.

Помнишь белые берёзы,
Вкус берёзового сока.
Нежные слова «люблю»
То во сне, то в строках?
Помню всё те белые берёзы,
Вкус берёзового сока.
И предутренняя Зөһрә-звезда.
```
