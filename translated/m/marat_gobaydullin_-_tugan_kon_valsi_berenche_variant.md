# Оригинал

### marat_gobaydullin_ - Марат Гобәйдуллин - Туган көн вальсы (беренче вариант)

```
Салмак кына чайкалабыз
Гомерләр диңгезендә.
Куаныч-шатлык китерсен
Һәрбер туар көн безгә.

Чәчкә-гөлләргә күмелсен
Безнең йөргән юлыбыз.
Йөрәкләр һич картаймасын,
Сүрелмәсен җырыбыз.

Туган көн - изге бәйрәм ул
Һәрберебез өчен дә.
Дус-ишләрнең җылы сүзе
Яшәр өчен көч бирә.

Чәчкә-гөлләргә күмелсен
Безнең йөргән юлыбыз.
Йөрәкләр һич картаймасын,
Сүрелмәсен җырыбыз.

Яратабыз туган көнне
Җыр-биюләре өчен.
Һич сүнмәсен яшьлек дәрте,
Югалтмыйк яшәү көчен.

Чәчкә-гөлләргә күмелсен
Безнең йөргән юлыбыз.
Йөрәкләр һич картаймасын,
Сүрелмәсен җырыбыз.
```

# Перевод

### Марат Гобәйдуллин - Вальс дня рождения (первый вариант)

```
Тихо мы покачиваемся
В море наших лет.
Пусть несет нам радость-счастье
Каждый новый рассвет.

Пусть цветами осыпается
Наш пройденный путь.
Пусть сердца не старятся,
Не угаснет песнь ничуть.

День рожденья - праздник светлый
Для каждого из нас.
Теплое слово друзей
Дает нам силы жить сейчас.

Пусть цветами осыпается
Наш пройденный путь.
Пусть сердца не старятся,
Не угаснет песнь ничуть.

Любим мы день рожденья
За песни и пляски в нем.
Пусть не гаснет жар юности,
Силы жить мы не отдадим.

Пусть цветами осыпается
Наш пройденный путь.
Пусть сердца не старятся,
Не угаснет песнь ничуть.
```
