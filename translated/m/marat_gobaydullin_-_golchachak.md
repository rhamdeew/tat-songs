# Оригинал

### Марат Гобәйдуллин - Гөлчәчәк

```
Гөлчәчәкле болында
Танышкан идек.
Бер-беребездән вәгъдәләр
Алышкан идек.

Аллы-гөлле чәчкәләр -
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.

Гөлчәчәк исемен кушкан
Сиңа әткәең.
Ак бәхетләр теләгән
Газиз әнкәең.

Аллы-гөлле чәчкәләр -
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.

Гөлдәй ямьле йөзләрең,
Назлы сүзләрең.
Мәңгегә әсир итте
Серле күзләрең.

Аллы-гөлле чәчкәләр -
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.
```

# Перевод

### Марат Губайдуллин - Гульчечек

```
На цветущем лугу
Мы с тобой повстречались.
Друг другу обещания
Мы тогда давали.

Яркие цветы -
Тебе мой подарок.
Быть навеки твоим
Моё желание.

Имя Гульчечек дал
Тебе твой отец.
Светлого счастья желала
Дорогая твоя мать.

Яркие цветы -
Тебе мой подарок.
Быть навеки твоим
Моё желание.

Лицо прекрасное, как цветок,
Нежные слова твои.
Навеки пленили меня
Твои таинственные глаза.

Яркие цветы -
Тебе мой подарок.
Быть навеки твоим
Моё желание.
```
