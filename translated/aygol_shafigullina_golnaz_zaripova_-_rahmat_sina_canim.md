# Оригинал

### Айгөл Шәфигуллина / Гөлназ Зарипова - Рәхмәт сиңа, җаным

```
Синең белән икәү җитәкләшеп
Менеп киттек тормыш үренә.
Җил-яңгырлар, давылларын үттек,
Алда якты кояш күренә.

Рәхмәт сиңа, җаным, авыр минутларда
Сабыр-түзем булдың һәрвакыт.
Күңелем түрендә, утлар сүнгән чакта,
Юлларымны бардың яктыртып.

Кыен вакытларда, чыдам булып,
Каршы тордык давыл-җиленә,
Каян алдың шундый сабырлыклар,
Таңнар калдым синең көчеңә.

Рәхмәт сиңа, җаным, авыр минутларда
Сабыр-түзем булдың һәрвакыт.
Күңелем түрендә, утлар сүнгән чакта,
Юлларымны бардың яктыртып.

Тормышның юллары борма-борма,
Ялгыш юлга кереп китмәдек.
Яратканга күрә, икебез дә
Буш сүзләрне бик тиз үткәрдек.

Рәхмәт сиңа, җаным, авыр минутларда
Сабыр-түзем булдың һәрвакыт.
Күңелем түрендә, утлар сүнгән чакта,
Юлларымны бардың яктыртып.
```
