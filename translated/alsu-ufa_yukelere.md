# Оригинал

### Алсу - Уфа юкэлэре

```
Яшьлек урамнары,
Тулы хыяллары,
Без — шатлыкка туймас исэрлэр.
Уфа юкэлэре
Безнен хакта да эле
Яфрак яра-яра сойлэрлэр.

Гомер ике килми,
Утлар ике сунми,
Без — мен янып тик бер сунэрлэр.
Уфа юкэлэре
Онытмаслар эле,
Яфрак коя-коя сойлэрлэр.

Безнен менгэн атлар,
Еллар, аргамаклар,
Комеш тезген, алтын йогэннэр.
Уфа юкэлэре
Безнен хакта да эле
Чэчэк ата-ата сойлэрлэр.
```

# Перевод

### Алсу - Липы Уфы

```
Молодости улицы,
Полные мечтаний,
Мы - радости ненасытные пьяницы.
Липы Уфы
О нас еще
Листок роняя-роняя расскажут.

Жизнь дважды не приходит,
Огни дважды не гаснут,
Мы - тысячу горя лишь раз погаснем.
Липы Уфы
Не забудут еще,
Лист роняя-роняя расскажут.

Наши скачущие кони,
Годы, скакуны,
Серебряные поводья, золотые попоны.
Липы Уфы
О нас еще
Цветок роняя-роняя расскажут.
```
