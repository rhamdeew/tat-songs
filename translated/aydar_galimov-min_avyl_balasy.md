# Оригинал

### Айдар Галимов - Мин авыл баласы

```
Сагындым чишмэлэр суларын,
Иренне ошеткэн салкынын.
Мин, авыл баласы, сагындым
Ислэрен ю(ы)ртак  атынын.

Сагындым учаклар ялкынын,
Тоннэрен очкынлап алганын.
Мин, авыл баласы, сагындым
Кызларнын союдэн янганын.

Сагындым болыннар печэнен,
Хуш ислэр боркетеп кипкэнен,
Мин, авыл баласы, сагындым
Капкадан энием коткэнен.

Сагындым кырларнын ислэрен
Игеннэр олгергэн чагында,
Мин, авыл баласы, кайтамын,
Кунелем авылым ягында.
```

# Перевод

### Айдар Галимов - Я — сельский житель

```
Соскучился по родниковой воде,
По прохладе, что губы целует.
Я, сельский житель, соскучился
По запахам дорогой лошадки.

Соскучился по пламени очагов,
Что ночи искрами озаряет.
Я, сельский житель, соскучился
По девичьей горящей любви.

Соскучился по сену лугов,
Что благоуханьем окутано,
Я, сельский житель, соскучился
По маме, ждущей у ворот.

Соскучился по запахам полей
Во время созревания хлебов,
Я, сельский житель, возвращаюсь,
Душа моя рядом с деревней.
```
