# Оригинал

### Әхмәт Ерикәй / Нәҗип Җиһанов - Тагын хуш ис бөрки сиреньнәр

```
Җир йөзе яшәрә, шатлана май,
Сиреньнәр хуш исләр тарата;
Яз килгәч, яшәрәм гөл сыман мин,
Күнелем яз ямен ярата.

Уйларны тибрәтә-тибрәтә җыр —
Өздереп Иделдә җырлыйлар;
Шомыртлар талларга, без дуслар дип,
Хуш исле кулъяулык болгыйлар,

Алдымда бормалы-бормалы юл,
Мин карыйм еракка-еракка;
Күк күзләр һаман да, һаман да бит
Тынгылык бирмиләр йөрәккә.

Елмая кәккүкле, тургайлы яз,
Күк күзләр дә, бәлки, елмаер:
Ай кебек, бер генә елмайса да,
Язларның гомере озаер…
```

# Перевод

### Ахмат Ерикей / Наджип Джиханов - Снова благоухают сирени

```
Земля оживает, ликует май,
Сирени разносят свой аромат;
Когда приходит весна, цвету, как цветок,
Душа моя любит весеннюю красоту.

Песня волнует, волнует мысли —
Над Волгой разрываясь, поют;
Черемухи ивам, мы друзья, говорят,
Благоуханным платком становятся.

Передо мной извилистый путь,
Смотрю я вдаль, в далекие дали;
Голубые глаза всё так же, всё так же
Покоя не дают сердцу моему.

Улыбается весна с кукушками, жаворонками,
Голубые глаза, быть может, улыбнутся:
Хоть, как луна, лишь раз улыбнутся,
Весенняя жизнь продлится дольше…
```
