# Оригинал

### Альфина Азгамова - Сина багышладым

```
Таннан торып, чишмэлэргэ бардым
Курер очен анын агышларын.
Кунелемдэ туды бер яна жыр,
Мин ул жырны сина багышладым.

Шушы жырда бэхет-шатлыкларым,
Шушы жырда сагыну-сагышларым,
Шушы жырда минем бар тормышым,
Мин уземне сина багышладым.

Бакчаларга чыктым тынлар очен
Сандугачнын монлы тавышларын,
Кунелемдэ туды ор яна жыр,
Мин ул жырны сина багышладым.

Шушы жырда бэхет-шатлыкларым,
Шушы жырда сагыну-сагышларым,
Шушы жырда минем бар тормышым,
Мин уземне сина багышладым.
```

# Перевод

### Альфина Азгамова - Тебе посвятила

```
Утром встав, к родникам пошла
Посмотреть их течения.
В душе моей родилась новая песня,
Я ту песню тебе посвятила.

В этой песне мои счастье-радости,
В этой песне мои тоска-печали,
В этой песне вся моя жизнь,
Себя саму тебе посвятила.

В сады вышла послушать
Соловья печальные голоса,
В душе моей родилась еще новая песня,
Я ту песню тебе посвятила.

В этой песне мои счастье-радости,
В этой песне мои тоска-печали,
В этой песне вся моя жизнь,
Себя саму тебе посвятила.
```
