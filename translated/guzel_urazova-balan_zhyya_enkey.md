# Оригинал

```
Балан жыя энкэй жырлый жырлый
Чэчкэйлэрен жиллэр тарата
Эллэ инде энкэй бу мизгелне
Барысыннанда ныграк ярата

Нигэ генэ узен телэгенчэ
Язып булмый икэн язмышны
Балдай татлыларны язар идем
Алмас идем эрну ялгышны

Гуя энкэй тэлгэшлэрне тугел
Ялгышларын жыя озелэ
Хэр тэлгэшне озгэн саен аны
Йорэккэе гуя озелэ

Нигэ генэ узен телэгенчэ
Язып булмый икэн язмышны
Балдай татлыларны язар идем
Алмас идем эрну ялгышны

Учларымда купме эче балан
Язмышларын кебек эчелэ
Шул баланнан белэн бер сойлэшсэм
Кунеллэрем никтер очыла

Балан жыя энкэй жырлый жырлый
Бите буйлап ага куз яше
Юк энием бу куз яше тугел
Эче татлы балан тэлгэше

Нигэ генэ узен телэгенчэ
Язып булмый икэн язмышны
Балдай татлыларны язар идем
Алмас идем эрну ялгышны
```

# Перевод

### Гузель Уразова - Мать собирает детские осколки

```
Мать собирает детские осколки, напевая
Лепестки разносят ветра
Наверно мать этот миг
Больше всех других любит

Почему же по своему желанию
Нельзя написать судьбу
Сладкие как мёд написала бы
Не взяла бы горькие ошибки

Словно мать собирает не осколки,
А свои ошибки, мучаясь
С каждым осколком, что рвёт,
Её сердце тоже рвётся

Почему же по своему желанию
Нельзя написать судьбу
Сладкие как мёд написала бы
Не взяла бы горькие ошибки

В моих снах сколько горьких детских
Судеб словно горчат
Если б поговорить с тем ребёнком,
Душа моя как-то просветлела бы

Мать собирает детские осколки, напевая
По лицу течёт слеза
Нет, мамочка, это не слеза
Это горький детский осколок
```
