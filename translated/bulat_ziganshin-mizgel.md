# Оригинал

### Булат Зиганшин - Мизгел

```
Тумаган да кебек бу доньяга,
Утелмэгэн дэ кебек зур юллар.
Тик киресен дэлиллэр барын
Эшли-эшли таушаган куллар.

Якты доньяларда яшэулэрдэн
Кем туя?
Ташу булып аккан чишмэлэрне
Кем тыя?

Мизгел генэ кунак бу безнен
Бу берконлек фани доньяда.
Бер карасан, купме яшэлгэн бит
Узен эле тойма да.

Якты доньяларда яшэулэрдэн
Кем туя?
Ташу булып аккан чишмэлэрне
Кем тыя?
```

# Перевод

### Булат Зиганшин - Миг

```
Будто не родился в этот мир,
Будто не пройдены великие пути.
Только видишь — есть доказательства всему:
Руки, что усердно трудились.

От жизней в светлых мирах
Кто устанет?
Бурлящие источники
Кто остановит?

Лишь мгновение гостим мы
В этом однодневном бренном мире.
Посмотришь — сколько прожито,
А всё кажется — не насытился.

От жизней в светлых мирах
Кто устанет?
Бурлящие источники
Кто остановит?
```
