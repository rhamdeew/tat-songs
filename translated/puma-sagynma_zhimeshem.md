# Оригинал

### Пума - Сагынма, жимешем

```
Хыялым турендэ
Мин синен белэн.
Акылынны жинэм,
Янына килэм.

Мин очам хыялга,
Канатлар талса да,
Сагыну давылы
Йорэкне ярса да.
Кунелдэ сурэтенэ
Шаярып куз кысам,
Очрашу мизгелендэ
Мин сине кочам.

Сагынма, жимешем,
Сагынма, килэм мин,
Сагынма, яратканым син.
Син мине кот, син мине кот,
Сагынма, килэм мин.

Юлларын чигендэ
Карашын куренэ,
Менлэгэн нурларын
Балкышы син генэ.

Сагынма, жимешем,
Сагынма, килэм мин,
Сагынма, яратканым син.
Син мине кот, син мине кот,
Сагынма, килэм мин.
```

# Перевод

### Пума - Не скучай, мой плод

```
В моих мечтах всегда
Я рядом с тобой.
Твой разум обретаю,
К тебе приду.

Лечу я в грёзы,
Хоть крылья устают,
Пусть буря разлуки
Сердце разрывает.
В душе твой образ
Бережно храню,
В миг встречи долгожданной
Тебя я обниму.

Не скучай, мой плод,
Не скучай, приду я,
Не скучай, любимый мой.
Ты меня жди, ты меня жди,
Не скучай, приду я.

На перекрёстке дорог
Твой взгляд виднеется,
Из тысячи лучей
Сияешь только ты.

Не скучай, мой плод,
Не скучай, приду я,
Не скучай, любимый мой.
Ты меня жди, ты меня жди,
Не скучай, приду я.
```
