# Оригинал

### Фарида Алсу - Эбием коймаклары

```
Сагынабыз балачакны
Жен-пэрине хэр тоннен,
Урамнарда ялан аяк
Тэпилэгэн жэй конен.

Усеп житкэч хэтирэлэр
Куз алдыннан утэлэр,
Балачакнын хэр мизгеле
Туктап безне котэлэр.

Озелеп-озелеп сагындыра
Эбием коймаклары,
Мичтэ кызган табасынын
Кулында уйнаклавы.

Сагындыра эниемнен
Ачуланган чаклары,
Эшкэ-тэртипкэ ойрэткэн
Этием чыбыклары.

Хэзер рэхмэт кенэ эйтэм
Этием-эниемэ.
Аларны мина тудырган
Кадерле эбилэргэ.

Озелеп-озелеп сагындыра
Эбием коймаклары,
Мичтэ кызган табасынын
Кулында уйнаклавы.

...
```

# Перевод

### Фарида Алсу - Бабушкины оладьи

```
Скучаем по детству
С душой-феей каждой ночью,
По летним дням, когда босиком
Топтали улицы.

Когда взрослеем, воспоминания
Проходят перед глазами,
Каждый миг детства
Останавливается и ждёт нас.

Томительно-томительно напоминают
Бабушкины оладьи,
На горячей сковороде
Игривые в её руках.

Напоминают мамины
Сердитые времена,
Папины прутики,
Учившие работе и порядку.

Теперь только спасибо говорю
Отцу и матери,
Дорогим бабушкам,
Что их мне родили.

Томительно-томительно напоминают
Бабушкины оладьи,
На горячей сковороде
Игривые в её руках.

...
```
