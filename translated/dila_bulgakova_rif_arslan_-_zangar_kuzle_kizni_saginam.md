# Оригинал

### Дилә Булгакова / Риф Арслан - Зәңгәр күзле кызны сагынам

```
Узган яшьлек сыман,
Ярсу хисләр ерак,
Агалар да, еллар, агалар.
Еллар сәйләнендә, гөлләр бәйләмендә,
Шиңми торган яшьлек җыры бар.

Үткәннәргә калган,
Сүнмәс өчен янган,
Хыял утларына табынам.
Зәңгәр күлмәк кигән,
Зәңгәр гөлләр сөйгән,
Зәңгәр күзле кызны сагынам.

Тойгыларым аша,
Җылы хисләр таша.
Әйтелмәгән, әйтер сүзем бар.
Олы мәхәббәтем,
Сафлык белән бөтөн.
Җырларымда яши тәүге яр.

Үткәннәргә калган,
Сүнмәс өчен янган,
Хыял утларына табынам.
Зәңгәр күлмәк кигән,
Зәңгәр гөлләр сөйгән,
Зәңгәр күзле кызны сагынам.
```

# Перевод

### Диля Булгакова / Риф Арслан - Скучаю по девушке с синими глазами

```
Словно прошлая юность,
Горькие чувства вдали,
Текут и годы, текут.
Когда годы бегут, в букете цветов,
Жива нестареющей юности песня.

Оставшаяся в прошлом,
Горевшая, чтоб не угаснуть,
Я молюсь на огни мечты.
В синем платье ходила,
Синие цветы любила,
Скучаю по девушке с синими глазами.

Сквозь мои чувства,
Тёплые эмоции переполняют.
Есть слова, что хочу сказать.
Моя большая любовь,
Со всей чистотой.
В песнях моих живёт первая любовь.

Оставшаяся в прошлом,
Горевшая, чтоб не угаснуть,
Я молюсь на огни мечты.
В синем платье ходила,
Синие цветы любила,
Скучаю по девушке с синими глазами.
```
