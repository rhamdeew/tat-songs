# Оригинал

### Билгесез / Урал Рәшитов - Карлыгач (беренче вариант)

```
Карлыгач оя ясый,
Юеш балчыкны ташып.
Сиңа булган сөю хисем
Ургый, йөрәктән ташып.

Карлыгачлар биек оча,
Бүген дә булыр аяз.
Яннарымда син булганда,
Һәрвакыт кояшлы яз.

Карлыгачның түшендә
Тамгасы бар ак төстә.
Бергә булган минутларның
Һәрбер мизгеле истә.

Карлыгачлар биек оча,
Бүген дә булыр аяз.
Яннарымда син булганда,
Һәрвакыт кояшлы яз.

Карлыгачлар су эчәләр,
Төшмичә генә җиргә.
Яшәсәң иде дәньяда
Сөйгәнең белән бергә.

Карлыгачлар биек оча,
Бүген дә булыр аяз.
Яннарымда син булганда,
Һәрвакыт кояшлы яз.
```

# Перевод

### Неизвестен / Урал Рашитов - Ласточка (первый вариант)

```
Ласточка гнездо вьёт,
Мягкую глину таская.
Моя любовь к тебе
Бьётся, через сердце переливаясь.

Ласточки высоко летают,
Сегодня будет ясно вновь.
Когда ты рядом со мною,
Всегда солнечная весна.

На грудке у ласточки
Есть отметина белого цвета.
Каждый миг из тех минут,
Что были вместе, в памяти.

Ласточки высоко летают,
Сегодня будет ясно вновь.
Когда ты рядом со мною,
Всегда солнечная весна.

Ласточки воду пьют,
Даже не опускаясь на землю.
Вот бы так жить на свете
Вместе с тем, кого любишь.

Ласточки высоко летают,
Сегодня будет ясно вновь.
Когда ты рядом со мною,
Всегда солнечная весна.
```
