# Оригинал

### Малика - Яратмыйм сине жаным

```
Йормэ минем артымнан,
Мин синеке булалмам.
Кочлэп яратып булмый
Мин башканы яратам.

Яратмыйм сине, жаным.
Булалмыйм синен парын,
Эйтергэ жинел тугел,
Башка дип минем ярым.

Уйлама минем хакта,
Карама минем якка,
Коченне сарыф итмэ,
Бир союенне ятка.

Тизрэк онытсан мине,
Жинелрэк булыр иде.
Икебезнен дэ жанга
Рэхэтлек килер иде.
```

# Перевод

### Малика - Не люблю тебя, душа моя

```
Не иди за мной следом,
Я не могу быть твоей.
Нельзя любить по принуждению,
Я люблю другого.

Не люблю тебя, душа моя.
Не могу быть твоей парой,
Нелегко сказать это,
Другой - моя половинка.

Не думай обо мне,
Не смотри в мою сторону,
Не трать силы понапрасну,
Отдай любовь другой.

Если бы ты забыл меня быстрее,
Было бы легче.
В душу нас обоих
Пришло бы успокоение.
```
