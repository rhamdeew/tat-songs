# Оригинал

### Энҗе Замилова / Айнур Сәйфиев - Бәгырькәем, бергенәм

```
Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Төшләремдә сине күрдем,
Каршыма килә идең.
Сөясеңме, сөймисеңме,
Җанкисәгем, син мине?

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Шаян итеп сөйләүләриң
Нур бөркетә җаныма.
Сагынуга чик-чама юк,
Барам әле яныңа.

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Сиңа киткән юлларым да
Гел чәчкәгә күмелгән.
Үзеңә дә чибәрлекне
Ходаем чиксез биргән.

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.
```

# Перевод

### Энже Замилова / Айнур Сайфиев - Прижмись, возьми меня

```
Прижмись, возьми меня,
Со мной разлучайся с болью.
Без тебя жить - не по себе,
К тебе я иду, любимый.

Во снах видела тебя,
Навстречу ты шёл мне.
Любишь ты меня или нет,
Скажи, моя душа?

Прижмись, возьми меня,
Со мной разлучайся с болью.
Без тебя жить - не по себе,
К тебе я иду, любимый.

Твои добрые слова
Светом греют мою душу.
Нет границ тоске по тебе,
Иду я всё рядом с тобой.

Прижмись, возьми меня,
Со мной разлучайся с болью.
Без тебя жить - не по себе,
К тебе я иду, любимый.

Дороги, что ведут к тебе,
Всегда цветами устланы.
Тебе Господь безгранично
Даровал любовь мою.

Прижмись, возьми меня,
Со мной разлучайся с болью.
Без тебя жить - не по себе,
К тебе я иду, любимый.
```
