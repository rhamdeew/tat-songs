# Оригинал

### Асылъяр - Карлыгачлар

```
Офыкка жэя канатын
Алсу таннын балкышы.
Мин сине уйлап уянам
Кунелемнен аккошы.

Карлыгачлар оя ясый,
Шуны курэсем килэ.
Уз оямны синен белэн
Бергэ урэсем килэ.

Тан нуры безнен йозлэрне
Балкытып торсын иде.
Минем сою йорэгенне
Жилкетеп куйсын иде.

Карлыгачым,сэлам бирче
Минем сойгэн ярыма.
Ул да мина жан атадыр
Кунеленэ аккошына
```

# Перевод

### Асылъяр - Ласточки

```
К горизонту расправляя крыло
Розовой зари сияние,
Я просыпаюсь, думая о тебе,
Лебедь души моей.

Ласточки гнездо вьют,
Это хочется увидеть,
Свое гнездо с тобой
Вместе свить хочется.

Рассветный свет наши лица
Пусть озаряет,
Мое любящее сердце
Пусть волнует.

Ласточка моя, привет передай
Моей любимой милой,
Она и мне душу дарит,
Лебедю души своей.
```
