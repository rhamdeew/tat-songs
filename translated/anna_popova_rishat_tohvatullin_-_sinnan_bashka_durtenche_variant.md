# Оригинал

### Анна Попова / Ришат Төхвәтуллин - Синнән башка (дүртенче вариант)

```
Синнән башка ничек яшәгәнмен,
Төшми инде хәтта хәтеремә.
Яратам дип әйткән көннән бирле
Хыял уты яна йөрәгемдә.

Синнән башка бәхет бәхет түгел,
Язларның да яме күренми.
Беләсеңме: йөрәккә синнән башка бу җирдә
Беркем дә, беркем кирәкми.

Синнән башка көннәр якты түгел,
Кояш нурлары җылытмый мине.
Төнге күктә йөрәкне җилкетеп
Йолдызлар да ялтырамый төсле.

Синнән башка бәхет бәхет түгел,
Язларның да яме күренми.
Беләсеңме: йөрәккә синнән башка бу җирдә
Беркем дә, беркем кирәкми.

Синнән башка сандугач җырлары
Күңелемнең түрләренә җитми.
Ярый әле син бар яннарымда,
Ходаема рәхмәт диеп яшим.

Синнән башка бәхет бәхет түгел,
Язларның да яме күренми.
Беләсеңме: йөрәккә синнән башка бу җирдә
Беркем дә, беркем кирәкми.
```

# Перевод

### Анна Попова / Ришат Тухватуллин - Без тебя (четвёртый вариант)

```
Без тебя как я жила,
Даже в памяти не сохранилось.
С того дня, как сказала «люблю»,
Огонь мечты горит в моём сердце.

Без тебя счастье - не счастье,
Даже весны краса не видна.
Знаешь ли: сердцу, кроме тебя, на этой земле
Никто, никто не нужен.

Без тебя дни не светлы,
Солнечные лучи не греют меня.
В ночном небе, волнуя сердце,
Звёзды словно не сияют.

Без тебя счастье - не счастье,
Даже весны краса не видна.
Знаешь ли: сердцу, кроме тебя, на этой земле
Никто, никто не нужен.

Без тебя соловьиные песни
Не достигают глубин моей души.
Хорошо ещё, что ты рядом со мной,
Живу, благодаря Бога.

Без тебя счастье - не счастье,
Даже весны краса не видна.
Знаешь ли: сердцу, кроме тебя, на этой земле
Никто, никто не нужен.
```
