# Оригинал

### Эльвира Сафина / Гөлназ Закирова - Җырларымда син генә

```
Бир кулыңны, бергә үтик,
Озын гомер юлларын.
Бер-беребезгә багышлап
Саф мәхәббәт җырларын.

Кавыштырды җырлы дөнья,
Җырларымда син генә.
Җырлый-җырлый, җырлый-җырлый
Яратам сине генә.

Елмаеп, көлеп яктыртсак,
Болытлы көннәрне дә.
Сөю йолдызы нур сибәр
Караңгы төннәргә дә.

Кавыштырды җырлы дөнья,
Җырларымда син генә.
Җырлый-җырлый, җырлый-җырлый
Яратам сине генә.

Бир кулыңны, бергә үтик,
Матур гомер юлларын.
Назлы җырлар тибрәндерсен
Күңелебез кылларын.

Кавыштырды җырлы дөнья,
Җырларымда син генә.
Җырлый-җырлый, җырлый-җырлый
Яратам сине генә.
```

# Перевод

### Эльвира Сафина / Гульнар Закирова - В моих песнях только ты

```
Дай руку, пройдем вместе,
Длинные жизненные пути.
Подарим друг другу
Песни чистой любви.

Сплела песенная жизнь,
В моих песнях только ты.
Пою-пою, пою-пою
Люблю только тебя.

Улыбаясь, освещаем
Даже пасмурные дни.
Звезда любви светит лучом
Даже в темные ночи.

Сплела песенная жизнь,
В моих песнях только ты.
Пою-пою, пою-пою
Люблю только тебя.

Дай руку, пройдем вместе,
Прекрасные жизненные пути.
Нежные песни пусть зазвучат
На струнах наших сердец.

Сплела песенная жизнь,
В моих песнях только ты.
Пою-пою, пою-пою
Люблю только тебя.
```
