# Оригинал

### Җәүдәт Дәрзаман / Ренат Еникеев - Наза йөри бакчада

```
Кояш күптəн батса да,
Наза йөри бакчада.
Су сибə чилəк-чилəк,
Авыз эченнəн көйлəп:

«Чəчəклəрем ал булсын,
Бал кортына бал булсын.
Үзем эшлим барын да -
Əниемə ял булсын».

Сузылып яр буена,
Чəчəклəр су коена.
Зур тамчы китсə тиеп,
Куялар биеп-биеп.
```

# Перевод

### Джаудат Дарзаман / Ренат Еникеев -Наза ходит в саду

```
Хоть солнце давно зашло,
Наза ходит в саду.
Водой поливает, ведро за ведром,
Напевая тихонько:

«Пусть цветы мои алыми станут,
Пусть для пчёл они мёдом будут.
Всё сама я сделаю -
Чтобы маме отдых был».

Вдоль берега тянутся,
Цветы водицы испить хотят.
Боясь, что крупная капля упадёт,
Зайчики танцуют, танцуют.
```
