# Оригинал

### Әзһәр Зәйни / Зиннур Сафиуллин - Ак каен (беренче вариант)

```
Сандугачка серен сөйли
Өй артында ак каен.
Шул каенга сыенып елыйм
Сине сагынган саен.

Мин елыйм, мин елыйм
Сине сагынган саен.

Җырчысын сагынып, каен
Көтеп ала май аен.
Ник булмадык икән лә без:
Мин - сандугач, син - каен?

Ник булмадык, ник булмадык
Мин - сандугач, син - каен?

Алар бездән бәхетлеләр-
Серләшәләр таң саен.
Ялгыз җанымны яндырма,
Бергә булыйк - тап җаен.

Җанымны яндырма,
Бергә булыйк - тап җаен!..
```

# Перевод

### Азһар Зайни / Зиннур Сафиуллин - Белая берёза (первый вариант)

```
Соловью секреты шепчет
Белая берёза за домом.
Прижавшись к той берёзе, плачу
Всякий раз, скучая по тебе.

Я плачу, я плачу,
Всякий раз, скучая по тебе.

Скучая по своему певцу, берёза
Ожидает месяц май.
Почему же не стали мы с тобой:
Я - соловьём, ты - берёзой?

Почему не стали, почему не стали
Я - соловьём, ты - берёзой?

Они счастливее нас -
Беседуют каждый рассвет.
Не жги мою одинокую душу,
Давай будем вместе - найди решение.

Не жги мою душу,
Давай будем вместе - найди решение!..
```
