# Оригинал

### Әхмәт Ерикәй / Татар халык җыры - Алсу (икенче вариант)

```
Авылыбызда бер Алсу бар.
Бит алмалары алсу;
Алсу, Алсу, йөрәк ярсу.
Син булмасаң ямансу.

Алсу матур, Алсу батыр,
Бар эшкә дә уңган ул;
Кырга чыкса, кыр яңгырата
Аның матур җырлары.

Шомырт чәчәк аткан чакта
Алсу йоклап ятырмы?!
Сөймим, дисәң дә, сөярсең
Алсу кебек матурны.
```

# Перевод

### Ахмат Ерикей / Татарская народная песня - Алсу (второй вариант)

```
В нашей деревне есть Алсу одна,
Щёки румяные, как заря;
Алсу, Алсу, сердце ранишь ты,
Без тебя на душе темно.

Алсу красива, Алсу смела,
Всякое дело ей по плечу;
Выйдет в поле — степь звенит
От её прекрасных песен.

Когда черёмуха цветёт,
Разве Алсу может спать?!
Хоть и скажешь: не люблю,
Но полюбишь ту, что так прекрасна, как Алсу.
```
