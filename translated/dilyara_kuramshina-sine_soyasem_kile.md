# Оригинал

### Дилияра Курамшина - Сине соясем килэ

```
Голчэчэклэр булэк итсэм
Союемне сойлэрлэр инде.
Шомырт кара кузлэренэ
Карап туялмам инде.

Сине соясем килэ
Сине убэсем килэ
Синен ярын, синен парын
Минем буласым килэ.

Очрашмыйча да басылмый
Бетмэстер янып кою
Сине генэ хэрчак уйлыйм
Газапмы эллэ бу сою.

Сине соясем килэ
Сине убэсем килэ
Синен ярын, синен парын
Минем буласым килэ.

Зифа буй-сыннарынны да
Ерактан куреп таныйм
Гол кырларында булсан да
Узенэ карап туймыйм

Сине соясем килэ
Сине убэсем килэ
Синен ярын, синен парын
Минем буласым килэ.
```

# Перевод

### Дилияра Курамшина - Хочется тебя полюбить

```
Если подарю букет цветов,
Расскажут о моей любви.
В твои черные глаза-черемуха
Глядеть не могу насытиться.

Хочется тебя полюбить,
Хочется тебя приласкать,
Твоей подругой, твоей парой
Хочется мне стать.

Даже не встречаясь, не утихает
Неугасимо пылающая любовь,
Только о тебе всегда думаю —
Мука ли эта любовь?

Хочется тебя полюбить,
Хочется тебя приласкать,
Твоей подругой, твоей парой
Хочется мне стать.

Стройную твою фигуру
Издалека узнаю,
Даже в цветочных полях
Глядя на тебя, не наглядеюсь.

Хочется тебя полюбить,
Хочется тебя приласкать,
Твоей подругой, твоей парой
Хочется мне стать.
```
