# Оригинал

### Гульнара Гараева - Азаккы куперне яндырма

```
Хуш диеп кулынны бирэсен,
Вэгъдэнне ашыгып озэсен.
Кузлэрен яшерми соясен
Еламыйча ничек тузэсен.

Азаккы куперне яндырма
Син мине ометсез калдырма,
Кузлэремэ тагын бер кара
Курерсен йорэгем гел яра.

Доньяга килеп булмый кабат
Тормышта без икэу пар канат
Канатсыз оча алмый кошлар
Аерылмасын безнен язмышлар

Азаккы куперне яндырма
Син мине ометсез калдырма,
Кузлэремэ тагын бер кара
Курерсен йорэгем гел яра.
```

# Перевод

### Гульнара Гараева - Не сжигай последний мост

```
Скажешь "прощай" и руку дашь,
Торопясь, обещание нарушишь.
Глаза не скроешь, любишь,
Как вытерпишь, не плача.

Не сжигай последний мост,
Не оставляй меня без надежды,
Взгляни еще раз в мои глаза,
Увидишь — сердце мое все в ранах.

В мир нельзя прийти снова,
В жизни мы двое — пара крыльев,
Без крыльев не могут летать птицы,
Пусть не разлучатся наши судьбы.

Не сжигай последний мост,
Не оставляй меня без надежды,
Взгляни еще раз в мои глаза,
Увидишь — сердце мое все в ранах.
```
