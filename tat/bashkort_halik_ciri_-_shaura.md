# Оригинал

### Башкорт халык җыры - Шәүрә

```
Билге дә генә куйдым, ай, биләнгә,
Шәүрә килен киләм дә дигәнгә.
Кәнафер дә чәйнәп, беректем бадыян
Шәүрә килен дә кигән җиләнгә.

Шәүрә килен дә кигән җиләннең
Итәк очкайлары да мунчаклы.
Айга да гына, көнгә тиңләмим,
Сылулыгың, Шәүрәкәй, шул чаклы.

Һавада гына очкан, ай, асыл кош,
Лачын гына дисәм дә, биле ала.
Эчкенәем тулып, ай, ут яна,
Сүнсен генә дисәм, бүтән ни кала?
```
