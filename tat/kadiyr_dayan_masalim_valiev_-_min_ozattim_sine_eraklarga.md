# Оригинал

### Кадыйр Даян / Мәсәлим Вәлиев - Мин озаттым сине еракларга

```
Мин озаттым сине еракларга,
Агыйделдә бозлар акканда.
Без хушлаштык, шунда, су буенда
Алсу-нурлы таңнар атҡанда.

Мин китәмен дидең, бик озакка,
Онытма син мине, көт дидең.
Көтәм сине, бәгърем, бик сагынам,
Йөрәгемдә бүтән юк минем.

Көтәм сине, бәгърем, мин онытмам,
Мин онытмам сине гомергә.
Мин кайтырмын, туган ил сагына,
Без булырбыз шунда гел бергә.
```
