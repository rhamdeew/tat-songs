# Оригинал

### Әхмәт Фәйзи / Җәүдәт Фәйзи - Зәйтүнәкәй

```
Үпкәләмәде микән,
Ачуланмады микән?
Уйнап әйткән сүзләремне,
Зәйтүнәкәй, акчарлагым,
Чынга алмады микән?

Алмагачның алмасын
Ыргытсана, аппагым!..
Хәтерең калган сүзләр булса,
Зәйтүнәкәй, акчарлагым,
Онытсана, аппагым.

Су читенә иелдем,
Төнбоекка үрелдем;
Бер елмаеп кара миңа,
Зәйтүнәкәй, акчарлагым,
Күтәрелсен күңелем.
```
