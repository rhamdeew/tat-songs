# Оригинал

### Радик Хәкимҗанов / Наил Шәймарданов - Бәхетле булсын һәркем (беренче вариант)

```
Тормыш шатлык, кайгылардан,
Сынаулардан тора гел.
Көннәр төрле булсалар да,
Якын безгә һәр мизгел.

Хәерле тусын иртәләр,
Кояшлы булсын һәр көн.
Кеше бәхет өчен туа,
Бәхетле булсын һәр көн.

Таң җилләре алып килсен,
Хәбәрләрнең яхшысын.
Бу дөнъяда һәркем тапсын,
Серле бәхет ачкычын.

Хәерле тусын иртәләр,
Кояшлы булсын һәркөн.
Кеше бәхет өчен туа,
Бәхетле булсын һәр көн.

Тормыш юллары тайгак ул,
Бормалы, сикәлтәле.
Аклык, сафлык өчен янып,
Гомерләр үтсен әле.

Хәерле тусын иртәләр,
Кояшлы булсын һәркөн.
Кеше бәхет өчен туа,
Бәхетле булсын һәр көн.
```
