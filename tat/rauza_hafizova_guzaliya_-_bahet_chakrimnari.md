# Оригинал

### Рауза Хафизова / Гүзәлия - Бәхет чакрымнары

```
Тәрәзәдән әни карый,
Кочаклап яраннарын.
Шул яраннарга кызыгып,
Әй, китеп барганнарым.

Кайтыр юллар гына көтсен,
Туган йортыма илткән.
Әти-әнигә илткән юл,
Иң бәхетле юл икән!

Сагынып китеп барам ла,
Сагынып кайтам барын!
Кочагыма җыя-җыя,
Бәхет чакырымнарын!

Капкалар кала ябылып,
Капка төпләре чирәм…
Киткәндә ярты күңелне
Шунда калдырып киләм.

Кайтыр юллар гына көтсен,
Туган йортыма илткән.
Гөлләргә күмелеп әни
Тәрәз каршында көткән!

Сагынып китеп барам ла,
Сагынып кайтам барын!
Кочагыма җыя-җыя,
Бәхет чакрымнарын!
```
