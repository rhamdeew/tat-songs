# Оригинал

### Миләүшә Гайфуллина / Салават Низаметдинов - Йөрәгем серләре

```
Мәктәпкә килгәндә
Тап булдың юлыма.
Елмаеп карадың,
Кагылдың кулыма.

Дәресне тыңламый,
Үткәрәм вакытны.
Укыймын йөз кабат
Җибәргән хатыңны.

Тактада дус кызым
Нәрсәдер аңлата.
Алдымда рәсемең
Елмаеп тик ята.

Сәгатьнең телләре
Ашыкмый, ашыкмый.
Йөрәгем серләрен
Ачыклый, ачыклый.
```
