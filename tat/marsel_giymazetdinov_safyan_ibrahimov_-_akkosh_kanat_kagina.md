# Оригинал

### Марсель Гыймазетдинов / Сафьян Ибраһимов - Аккош канат кагына

```
Саргаямын сагышлардан,
Сәгать саен сагынам.
Сиңа сәлам әйтергә, дип,
Аккош канат кагына.

Сандугачлар өзеп сайрый
Минем сагышларымны.
Барсыннар әле яныңа,
Борма аккошларымны.

Аккошлар сүзсез аңлаша,
Эх, син дә аңла, иркәм.
Аккошлар тәрәзә какса,
Ач тәрәзәңне иртән.

Сандугачлар өзеп сайрый
Минем сагышларымны.
Барсыннар әле яныңа,
Борма аккошларымны.

Әйтергә генә кыймыйсың,
Син дә сөясең, беләм.
Сөюеңне язып җибәр
Аккош канаты белән.

Сандугачлар өзеп сайрый
Минем сагышларымны.
Барсыннар әле яныңа,
Борма аккошларымны.
```
