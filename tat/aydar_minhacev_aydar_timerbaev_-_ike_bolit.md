# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Ике болыт

```
Һаваларда ике болыт
Ике якка агалар.
Алар да безнең шикелле
Ерагая баралар.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.

Кавыша (а)лмый ике болыт,
Эх, белергә уйларын.
Юатыйм диеп үреләм,
Җитмәсә дә буйларым.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.

Ике болыт мөлдерәмә,
Туладыр күңелләре.
Бәлки, күрештерми калмас,
Кояш бит сүрелмәде.

Ике болыт арасына
Күпер саласым килә.
Шул күпердән синең белән
Бергә каласым килә.
Шул күпердән синең белән
Бергә каласым килә.
```
