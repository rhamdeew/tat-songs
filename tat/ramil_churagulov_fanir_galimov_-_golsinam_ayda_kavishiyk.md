# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Гөлсинәм (Әйдә кавышыйк!)

```
Кич җиткәнне көтмә, җаным,
Кил син, бәгырем, Гөлсинәм.
Су буенда өзелеп-өзелеп
Сагынып сине мин көтәм.

Күктә ялгыз Зөһрә-йолдыз.
Без ялгыз булмыйк.
Минем Зөһрәм син — Гөлсинәм,
Гел бергә булыйк!
Айлы кичтә су буенда
Әйдә кавышыйк!

Мин — Таһирың, Зөһрәм-йолдыз
Син ул — минем Гөлсинәм.
Кирәк икән зәңгәр күктән
Йолдызлар алып бирәм.

Күктә ялгыз Зөһрә-йолдыз.
Без ялгыз булмыйк.
Минем Зөһрәм син — Гөлсинәм,
Гел бергә булыйк!
Айлы кичтә су буенда
Әйдә кавышыйк!

Мең-мең йөрәк мең-мең кабат
Сөйгән гомер, сөелгән.
Хис-ялкынның иң кайнары
Бары минем күңелдә.

Күктә ялгыз Зөһрә-йолдыз.
Без ялгыз булмыйк.
Минем Зөһрәм син — Гөлсинәм,
Гел бергә булыйк!
Айлы кичтә су буенда
Әйдә кавышыйк!
```
