# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Аллага шөкер! (икенче вариант)

```
Биссмиладан көнне башлап
Рәхмат укыйм ходайга
Рәхмат җирләр күкләргә
нурлы кояшым айга

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк

Әнә безне шатландырып
үсеп килә балалар.
Изге теләкләрне теләп.
җырлансын матур җырлар

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк

Гөнаһлардан бәлаләрдан
Фәрештәләр сакласын
Җитәкләшеп атлый торган
юлларыбыз уң булсын

Аллага шөкер дип яшик
Аллага шөкер дип әйтик
Иншолла диеп куаныйк
Иншолла диеп юаныйк
```
