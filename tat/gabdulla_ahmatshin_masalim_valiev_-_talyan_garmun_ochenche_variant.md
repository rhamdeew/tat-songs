# Оригинал

### Габдулла Әхмәтшин / Мәсәлим Вәлиев - Тальян гармун (өченче вариант)

```
Сакмар-суның буйларында
Яңгырый тальян моңнары;
Ил сакларга яуга китә
Уралтауның уллары.

Сакмар-суның буйлары да
Сулары, болыннары.
Йөрәкләргә ялкын сала
Гармуныңның моңнары.

Сакмар-суның буйларында
Яңгырый тальян моңнары;
Яуда йөреп, җиңеп кайттыр
Уралтауның уллары.
```
