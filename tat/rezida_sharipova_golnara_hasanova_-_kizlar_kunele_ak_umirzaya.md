# Оригинал

### Резида Шәрипова / Гөлнара Хәсәнова - Кызлар күңеле — ак умырзая

```
Ак умырзая кардан баш төрткән,
Елмая аңа шук кояш күктән.
Назланып кына ачыла бөре,
Ак умырзая — кызлар күңеле.

Кызлар күңеле
Гел язлар тели.
Саф сөю көтә,
Сөйгәнен эзли.
Мәхәббәт гөле
Таҗларын җәя,
Кызлар күңеле —
Ак умырзая.

Ак умырзая — сафлык билгесе,
Гашыйклар өчен сөю өлгесе.
Энҗе таҗлардан хисләр түгелә,
Кызлар күңеле — ак умырзая.

Кызлар күңеле
Гел язлар тели.
Саф сөю көтә,
Сөйгәнен эзли.
Мәхәббәт гөле
Таҗларын җәя,
Кызлар күңеле —
Ак умырзая.

Кызлар күңелен рәнҗетә күрмә,
Нәфис гөлләрне өшетә күрмә.
Яшьлек еллары узмасын зая,
Кызлар күңеле — ак умырзая.

Кызлар күңеле
Гел язлар тели.
Саф сөю көтә,
Сөйгәнен эзли.
Мәхәббәт гөле
Таҗларын җәя,
Кызлар күңеле —
Ак умырзая.
```
