# Оригинал

### Разил Вәлиев / Татар халык көе - Һай, дуслар

```
Елга ага, елга ага,
Ярлар озатып кала.
Бәхетен эзләп, елганың
Зәңгәр күзләре тала.

Һай, дуслар!
Чит җирләрдә
Бәхет эзли
Елгалар, җилләр генә,
Елгалар, җилләр генә.

Җилләр исә, җилләр исә,
Җилләр исә шәрекъка.
Бәхетен таба алмыйча,
Җилнең гомере үтә.

Һай, дуслар!
Чит җирләрдә
Бәхет эзли
Елгалар, җилләр генә,
Елгалар, җилләр генә.

Бәхет эзләп талган картлар
Карый иленә таба…
Ир-егетләр үз бәхетен
Туган җирендә таба.

Һай, дуслар!
Чит җирләрдә
Бәхет эзли
Елгалар, җилләр генә,
Елгалар, җилләр генә.
```
