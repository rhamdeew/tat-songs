# Оригинал

### Сәйфи Кудаш / Бәхти Гайсин - Сындырма шомырт агачын

```
Сындырма шомырт агачын,
Сындырсаң корый, корый, диләр.
Хурланган кызлар шикелле,
Хурланган кызлар шикелле,
Сулкылдап елый, диләр.

Искәмә шомырт чәчәген,
Искәмә, искәмә, исерерсең!
Язлар саен юксынырсың,
Язлар саен юксынырсың,
Исеңә төшерерсең …
```
