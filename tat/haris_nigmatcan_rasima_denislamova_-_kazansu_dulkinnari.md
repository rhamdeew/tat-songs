# Оригинал

### Харис Нигъмәтҗан / Рәсимә Денисламова - Казансу дулкыннары

```
Казансуның йөгерек дулкыннары
Хәтерлиләр безне бүген дә
Яр буенда вәгьдәләшкән идек
Аерылмабыз диеп гомергә

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар,
Очрашуга тагын килдем бүген,
Сине көтеп таңнар аттылар.

Казансуның йөгерек дулкыннары
Сагыш өстәп ярга кагалар
Сине генә күрү теләге
Янган утка мине салалар.

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар,
Очрашуга тагын килдем бүген
Сине көтеп таңнар аттылар

Казансуга карап уйга баттым
Сулар белми минем хәлемне
Күпме еллар инде үтеп киткән
Тик нәрсәдер өти йөрәкне.

Вәгьдәләрне җилләр алып китте
Хыялларым суга актылар
Очрашуга тагын килдем менә
Сине көтеп таңнар аттылар.
```
