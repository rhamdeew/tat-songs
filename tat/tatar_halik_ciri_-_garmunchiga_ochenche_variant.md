# Оригинал

### Татар халык җыры - Гармунчыга (өченче вариант)

```
Кулларыңда тальян гармун
Әйдә әле, бер тарт әле.
Уйнап үскән, җырлап киткән
Көйләрне сайрат әле.

Шөгер-Арчаларны уйна,
Шахтаны, Сарманнарны.
Уйна, искә алыйк
Илгә кайта алмый калганнарны.

Уйна, гармун, җаныбызны
Сагыш сарган көннәрдә.
Шушы моңнар алып кайта
Туган үскән җирләргә.

Алып кайта, алып китә,
Качып булмый язмыштан.
Өзгеләнеп, туган-үскән
Җирдән китеп ялгышкан.

Урам көйләре хәзер дә
Колакта чыңлап тора.
Туган якларга кайтабыз,
Гармуннар тыңлап кына.
```
