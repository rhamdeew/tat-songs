# Оригинал

### Шәйхи Маннур / Фәрит Хатыйпов - Яратам (өченче вариант)

```
Яратам мин өемне,
Әти белән әнине,
Абый белен энемне,
Бишектәге сеңлемне.

Яратам мәктәбемне.
Укытучы апамны,
Дусымны, иптәшемне,
Уйнап йөргөн бакчамны.

Яратам мин Иделне,
Кафказны һәм Себерне,
Яшәгән киң җиремне —
Бөек туган илемне!
```
