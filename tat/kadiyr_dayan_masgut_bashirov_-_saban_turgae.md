# Оригинал

### Кадыйр Даян / Мәсгуть Бәширов - Сабан тургае

```
Сайра, сайра, канат кагып,
Күккә үрлә син, тургай.
Синең көмеш тавышыңнан
Чыңлап торсын киң тургай.

Күккә менеп, җиргә карап,
Мактап җырла кырларны,
Мактап сайра аланнарны,
Безнең эшчән кызларны.

Язгы тугай бигерәк матур,
Сокланып сайрый тургай.
Сайра, сайра, җырчы тургай,
Чыңласын безнең тугай.
```
