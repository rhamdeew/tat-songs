# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Алла бирсә

```
Кырык эшләр кырам диеп,
Һаваланма син, бәндәм.
Тауларны да күчерерсең
Көч-гайрәт булса тәндә.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.

Алла бирсә генә көнең
Кояшлы бәйрәм булыр.
Алла бирсә, офыгыңнан
Кара болыт таралыр.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.

«Алла бирсә», «бисмилла»сыз
Эшләнгән эшең хәрәм.
Йотам дигән ризыгыңны
Йота алмассың, Адәм.

«Алла бирсә» дигән сүзне
Әйтә бел адым саен.
Кылган һәрбер гамәлемә
Риза бул, Ходайкаем.
«Алла бирсә»не онытма
Адым саен, җанкаем.
```
