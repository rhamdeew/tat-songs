# Оригинал

```
Кояш шикелле йөзегез,
Шомырт кара күзегез.
Үзегезнең гүзәллекне
Беләсезме үзегез?

Эх, татар сылулары,
Ай, Ай, Айсылулары,
Миң, Миң, Миңсылулары,
Таң, Таң, Таңсылулары.
Ничек соң, кызлар, ничек
Сылу кыз булулары?
Ничек соң, кызлар, ничек
Сылу кыз булулары?

Килешле бар төшләрегез,
Энҗе бер тешләрегез.
Күзләрегез бигрәк моңсу,
Ничек соң эшләрегез?

Эх, татар сылулары,
Ай, Ай, Айсылулары,
Миң, Миң, Миңсылулары,

Сылу кыз булулары?
Ничек соң, кызлар, ничек
Сылу кыз булулары?

Бигрәк серле карашыгыз,
Биеп тора кашыгыз.
Барыгыз да бердәй сылу,
Минеке соң кайсыгыз?

Эх, татар сылулары,
Ай, Ай, Айсылулары,
```
