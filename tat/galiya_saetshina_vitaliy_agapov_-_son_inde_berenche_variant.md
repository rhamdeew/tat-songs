# Оригинал

### Галия Сәетшина / Виталий Агапов - Соң инде (беренче вариант)

```
Кошлар моңын тыңлыйм, сахраларда
Ак чәчәкләр өздем үрелеп.
Сиңа әйтер сүзем күп калды бит,
Чәчәк арасына төрелеп.

Язның матур айлы кичләрендә
Очраштык без икәү бүген дә.
Ялгыштыкмы икән әллә, диеп
Үкенүләр сиздем йөзеңдә.

Карап тордың минем күзләремә,
Күңелләрең тулы моң иде.
Әйтер сүзләремне әйтер өчен,
Соң шул инде хәзер, соң инде.
```
