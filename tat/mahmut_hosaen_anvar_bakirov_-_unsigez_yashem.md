# Оригинал

### Мәхмүт Хөсәен / Әнвәр Бакиров - Унсигез яшем

```
Агыйделкәй ак микән,
Бездән бик ерак микән?
Хатлар саен сөям диеп
Язулары хак микән?

Унсигез яшем,
Унсигез яшем,
Унсигез яшь мәхәббәттән
Баш әйләнгән чак икән.

Агыйделкәй ак йөзле,
Әйтерсең лә ак көзге.
Син кайчан кавыштырырсың,
Агыйделкәем, безне?

Унсигез яшем,
Унсигез яшем,
Унсигез яшь мәхәббәттән
Баш әйләнгән чак икән.
```
