# Оригинал

### Муса Сираҗи / Вәсил Галимов - Җиремә

```
Җирем, җирем, яшел бишегем син,
Гашыйк түгел сиңа кем генә?
Туган көннән башлап син минеке,
Мин дә синеке бит мәңгегә.

Сабый чактан минем күңелемә
Йолдыз, кояшыңны күчердең.
Син өйрәттең Илне яратырга,
Беләгемә бетмәс көч бирдең.

Мул булгандыр, ахыры тормышымда
Өлешемә чыккан көмешем.
Шуңа күрә дә бит яшимен мин
Җырым, хисем белән бүлешеп.

Болытлардан өстә чайкалганда
Син якынсың миңа тагында,
Күңелем белән, алтын чәнгелдәгем,
Урап кайтам һәр чак яныңа.
```
