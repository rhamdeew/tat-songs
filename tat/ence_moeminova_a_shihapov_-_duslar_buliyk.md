# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Дуслар булыйк

```
Сукмаклар төрле булса да,
Бер безнең юллар.
Дуслар булыйк гомергә дип
Бирешик куллар.

Кушымта:
Дуслар булыйк,
Дуслар булыйк,
Булыйк гел бергә.
Дуслар булыйк,
Дуслар булыйк,
Гомер-гомергә.

Бердәм көч таулар күчерә,
Күчерә диңгез.
Тау күчерер дуслар булыйк,
Дуслар булыйк без.

Берни булмас безнең өчен
Давыллар, җилләр.
Без аларга каршы барып
Яуларбыз үрләр.
```
