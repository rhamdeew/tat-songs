# Оригинал

### Гарифҗан Мөхәммәтшин / Әнвәр Шаһиморатов - Тәрәзәңне ач әле

```
Чыксаң таңда иңнәреңә
Салып зәңгәр шәлеңне,
Таң җилләре килеп сөйләр
Сиңа минем хәлемне.

Шунда аңламый калмассың
Минем күңелем ярасын —
Бер чыланса керфекләрең,
Кичерермен барысын.

Сөйләр сиңа сандугачлар,
Сөйләми калмас әле —
Сандугачлы язгы таңда
Тәрәзәңне ач әле.

Шунда аңламый калмассың
Минем күңелем ярасын —
Бер чыланса керфекләрең,
Кичерермен барысын.

Сөйләп үтәр кыр казлары,
Түгә-түгә сагышын,
Мине төшерер исеңә
Язгы сулар агышы.

Шунда аңламый калмассың
Минем күңелем ярасын —
Бер чыланса керфекләрең,
Кичерермен барысын.
```
