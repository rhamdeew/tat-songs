# Оригинал

### Лилия Сәгыйдуллина / Финат Нурлыгаянов - Әрем кадере

```
Авылымнан киткәндә мин
Өзеп алдым әремен.
Чит җирләрдә йөргән чакта
Басар йөрәк әрнүен.

Учларыма әрем салам:
Шундый хуш ислесең син.
Аңлатырга сүзләр җитми…
Ахры, сагыну төслесең.

Таш калада — таш далада
Кадерең артыр, әрем!
Кулларыма әрем учлап
Йөри торган көннәрем…

Учларыма әрем салам:
Шундый хуш ислесең син.
Аңлатырга сүзләр җитми…
Ахры, сагыну төслесең.

Авылымнан киткәндә мин
Өзеп алам әремен…
Еллар аша килде миңа
Аңлау әрем кадерен…

Учларыма әрем салам:
Шундый хуш ислесең син.
Аңлатырга сүзләр җитми…
Ахры, сагыну төслесең.
```
