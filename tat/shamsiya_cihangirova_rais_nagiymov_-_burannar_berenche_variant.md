# Оригинал

### Шәмсия Җиһангирова / Рәис Нәгыймов - Бураннар (беренче вариант)

```
Сиңа кайтыр юлларымда —
Җәяүле буран
Юлларымны йөгрә-йөгрә
Чистартып куйган.
Колач җәеп каршы алды,
Нәкъ синең сыман.
Сиңа кайтыр юлларымда
Җәяүле буран.

Синнән китәр юлларымда —
Өермә буран
Юлларыма өем-өем
Көрт сарып куйган.
Җибәрмәде, кире борды,
Нәкъ синең сыман.
Синнән китәр юлларымда
Өермә буран.

Нигә икән юлларымда
Шул ике буран
Чиратлаша һаман-һаман
Язмышлар сыман?
Сиңа кайтыр юлларымда —
Җәяүле буран,
Синнән китәр юлларымда —
Өермә буран.
```
