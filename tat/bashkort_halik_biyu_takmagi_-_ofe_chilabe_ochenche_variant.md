# Оригинал

### Башкорт халык бию такмагы - Өфе — Чиләбе (өченче вариант)

```
Ардагы дуслык юлын,
Без корычтан салабыз.

Өфе, Чиләбе,
Пешә кура җиләге.

Дәртле җырлар җырлый-җырлый
Без шул юлдан барабыз.

Өфе, Чиләбе,
Пешә кура җиләге.

Бергәләшеп эшләгәндә,
Җиңел юллар табыла.

Өфе, Чиләбе,
Пешә кура җиләге.

Тормыш юлын үзгәртәек,
Яңыртаек тагын да.

Өфе, Чиләбе,
Пешә кура җиләге.
```
