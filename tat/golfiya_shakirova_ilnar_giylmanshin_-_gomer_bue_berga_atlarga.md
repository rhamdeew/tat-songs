# Оригинал

### Гөлфия Шакирова / Илнар Гыйльманшин - Гомер буе бергә атларга

```
Синдә генә бар күңелем,
Синдә генә уйларым.
Сине уйлап су буйларын
Үкенмичә урадым.

Сөюләрең чынлап булса,
Бирмә мине ятларга.
Җитәкләшеп бергә-бергә
Гомер буе атларга.

Күңелдәге сорауларның
Җавап эзлим бар(ы)сына.
Сине, сине бер күрсәм дә
Йөрәк нигә ярсына?

Сөюләрең чынлап булса,
Бирмә мине ятларга.
Җитәкләшеп бергә-бергә
Гомер буе атларга.

Чынлап булган саф сөюне
Югалтмыйк аны, иркәм.
Тигез мәхаббәттә генә
Безнең тормышлар күркәм.

Сөюләрең чынлап булса,
Бирмә мине ятларга.
Җитәкләшеп бергә-бергә
Гомер буе атларга.
```
