# Оригинал

### Башкорт халык җыры - Гөләллә

```
Килегез дә көлегез,
Килегез дә көлегез,
Уйнагыз да көлегез.

Гөләллә, гөләллә,
Гөләиллә-ләем.

Матур баса бит әле,
Матур баскан биюләре
Күңел ача бит әле.

Гөләллә, гөләллә,
Гөләиллә-ләем.

Изәннәре бөгелсен,
Әнә шулай биегәндә
Бар дөньялар гөр килсен.

Гөләллә, гөләллә,
Гөләиллә-ләйем.

Йөзләрегез ал икән,
Сикереп-төшеп биюләре
Биг(е)рәк күңелле икән. (Үзе бер гомер икән)

Гөләллә, гөләллә,
Гөләиллә-ләем.
```
