# Оригинал

### Илшат Гыйниятуллин / Илдар Ногманов - Сөю бураннары

```
Тагын җиргә ап-ак карлар яуган
кабат сине искә төшереп.
Бөтерелеп яуган кар бөртеге.
Кулларымда тордым эретеп.

Күбәләктәй ап-ак карлар ява,
уйный җирдә карлар бураны.
җан сөелеп сине саг(ы)нуымны
бураннарым инде тыңлады

Ап-ак күлмәк кигән кебек кышлар
хәтерләнеп сине бүген дә
тына белми хисләр туңа күңел
буран уйный гашыйк күңелдә

Күбәләктәй ап-ак карлар ява
уйный җирдә карлар бураны.
җан сөелеп сине саг(ы)нуымны
бураннарым инде тыңлады

Ява карлар, ява басылмады
басылмады хисләр бураны
Күңелләрем сине уйлады да
сагынудан йөрәк туймады

Күбәләктәй ап-ак карлар ява
уйный җирдә карлар бураны.
җан сөелеп сине саг(ы)нуымны
бураннарым инде тыңлады
```
