# Оригинал

### Гәрәй Рәхим / Мәсгудә Шәмсетдинова - Игеннәр илендә

```
Нәкъ менә шушында
Тормыш башланып китәдер шикелле
Тоела күңелгә.
Нәкъ менә шушында —
Игеннәр илендә, игеннәр илендә,
Игеннәр илендә.

Нәкъ менә шушында,
Мин яраткан авыл җирендә,
Игенче хезмәте тормышка юл сала,
Тормышның юлдашы булган икмәк ясала.
Игеннәр илендә, игеннәр илендә,
Игеннәр илендә.
```
