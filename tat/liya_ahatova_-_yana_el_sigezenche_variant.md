# Оригинал

### Лия Әхәтова - Яңа ел (сигезенче вариант)

```
Эх, Яңа ел, Яңа ел,
Эх, Яңа ел, Яңа ел.

Ап-ак карлар явып үтте
Безнен юлларыбызга.
Яңа ел да килеп житте,
Ашыгып безнең янга.

Эх, Яңа ел, Яңа ел
Җилкетә күңелләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Җилкетә күңелләрне,
Эх, Яңа ел, Яңа ел.

Яңа елда чынга аша
Бөтен безнең хыяллар.
Күктәге бик күп йолдызлар
Ак юл теләп калалар.

Эх, Яңа ел, Яңа ел
Җилкетә күңелләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Котлыйк әле бәйрәм белән,
Теләп без теләкләрне.
Җилкетә күңелләрне,
Эх, Яңа ел, Яңа ел.
```
