# Оригинал

### Рәшит Минһаҗ / Айгөл Ихсанова - Баланнар балланганда

```
Баланнар балланганда
Чакырсаң аланнарга,
Минут та тормас идем
Шау-шулы калаларда.

Авышты көзгә көннәр —
Кыраулар төшер инде,
Аланда тәлгәш-тәлгәш
Баланнар пешәр инде.

Баланнар пешәдерме —
Төшләргә керә алан,
Аланнар уртасында
Иренең — ике балан.

Авышты көзгә көннәр —
Кыраулар төшәр инде,
Аланда тәлгәш-тәлгәш
Баланнар пешәр инде.
```
