# Оригинал

### Татар халык җыры - Сәгать чылбыры

```
Иртә лә томан, кич тә томан,
Томан кырауларны эретә;

Әй, югалттым сәгать лә чылбырым,
Күрмәдеңме, сары ла былбылым?

Бармас идем, йөрмәс идем,
Ризыкларым шулай йөретә.

Әй, югалттым сәгать лә чылбырым,
Тапмадыңмы, сары ла былбылым?

Иртә лә томан, кич тә томан,
Томан азакларында чык булган;

Әй, югалттым сәгать лә чылбырым,
Күрмәдеңме, сылуым, былбылым?

Син килерсең мине сагынып,
Шул чагында, җаныем, мин булмам.

Әй, югалттым сәгать лә чылбырым.
Тапмадыңмы сылуым, былбылым?
```
