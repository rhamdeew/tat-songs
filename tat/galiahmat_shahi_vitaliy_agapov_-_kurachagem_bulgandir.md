# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Күрәчәгем булгандыр

```
Гөлбакчага кермим инде,
Гөл чәчәге сулгандыр.
Сөйгән ярым ятка калды —
Күрәчәгем булгандыр.

Килде болыт, яуды яңгыр,
Язгы сулар тулгандыр.
Ялгышларым, сагышларым
Күрәчәгем булгандыр.

Мин кунасы талчыбыкка
Башка кошлар кунгандыр.
Язмышымны сайламадым —
Күрәчәгем булгандыр.

Килде болыт, яуды яңгыр,
Язгы сулар тулгандыр.
Ялгышларым, сагышларым
Күрәчәгем булгандыр.

Күз алдында якты йолдыз —
Киләчәгем булгандыр.
Киләчәккә ашкынуым —
Күрәчәгем булгандыр.

Килде болыт, яуды яңгыр,
Язгы сулар тулгандыр.
Ялгышларым, сагышларым
Күрәчәгем булгандыр.
```
