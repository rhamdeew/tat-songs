# Оригинал

### Гөлшат Зәйнәшева / Татар халык көе - Зөһрә йолдыз (беренче вариант)

```
Зөһрә йолдыз — таң йолдызы,
Таң беленгәч тә сүнер.
Кунак булып, кына, дуслар,
Килә икән яшь гомер.

Идел буен иңләп-буйлап,
Икәүләп йөрмәгәнбез.
Кунак кына яшь гомернең
Кадерен белмәгәнбез.

Челтер-челтер агып китәр,
Таш ташлама чишмәгә.
Яшь гомерем кире кайтса,
Белер идем нишләргә.

Яшьлектә үткән юлларга
Борылып карыйм һаман.
Адаштырса да яшьлегем,
Үкенмәс идем сыман.
```
