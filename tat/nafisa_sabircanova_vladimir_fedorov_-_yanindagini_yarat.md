# Оригинал

### Нәфисә Сабирҗанова / Владимир Федоров - Яныңдагыны ярат

```
Күпме еллар синсез үткән,
Очрашканбыз без, без соңга калып.
Гашыйк булырымнан куркып,
Китеп барам мин, мин моңга тарып.

Нишләргә соң? Ике ут арасы…
Нишләргә соң?
Сөйгәнем,
Җаныңны кыйнама кеше сүзләренә карап.
Сөйгәнем,
Мин китәргә диеп килдем, бар кешегә ярап.
Яныңдагыны ярат…

Бәхетле итә алмассың
Берьюлы ике, син ике ярны.
Бүләккә калдырып китәм
Быелгы карлы, буранлы язны.

Нишләргә соң? Ике ут арасы…
Нишләргә соң?
Сөйгәнем,
Җаныңны кыйнама кеше сүзләренә карап.
Сөйгәнем,
Мин китәргә диеп килдем, бар кешегә ярап.
Яныңдагыны ярат…
```
