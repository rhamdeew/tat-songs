# Оригинал

### Кадыйр Даян / Бәхти Гайсин - Гармунчы (икенче вариант)

```
Эх, гармуның, гармуныңда
Гармуныңның тавышы!
Уйна дәртле, шат-көйләрне,
Бетсең йөрәк сагышы.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.

Эх, гармуның, гармуныңда
Гармуныңның телләре!
Син уйнасаң, эшчәннәрнең
Рәхәт үтә көннәре.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.

Гармуныңның телләренә
Дәрт ялкыны кабынсын.
Син уйнаган шат-көйләрне
Яшь йөрәкләр сагынсын.

Эх, гармун тавышы,
Бетсең йөрәк сагышы.
```
