# Оригинал

### Таһир Гарипов / Илгиз Закиров - Төшләремә минем кер әле

```
Сине көткөн таңнар,
Кайда калды алар.
Хыялларга чумам
Чумам тын гына.
Җырларым да калды,
Тик исемең генә,
Ә йөрәктә калды
Сагыш моң гына,
Сагыш моң гына.

Бик сагынган чакта
Ялындырмый гына.
Төшләремә минем
Көлеп кер әле.
Сары сагыш сарган
Моңсу күзләремне
Тик бер генә килеп
Бәгърем, күр әле.

Йөрәк сыкрый-сыкрый
Сине көтә һаман
Керфекләрдән кайнар,
Әче яшь тама.
Минем исемемне,
Шашып сөюемне
Күңелеңдә зинһар
Сызып ташлама.
```
