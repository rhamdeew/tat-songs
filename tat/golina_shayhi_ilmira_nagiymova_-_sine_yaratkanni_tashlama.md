# Оригинал

### Гөлинә Шәйхи / Ильмира Нәгыймова - Сине яратканны ташлама

```
Горурлыкта ялгыз калдырдың да
Китеп бардың миннән читләргә.
Миңа ярдәм кирәк булган чакта
Нигә кирәк иде китәргә?

Йөрәгемне бүлеп син икегә,
Алмаштырдың мине башкага.
Матурлар бик күп булса да алар,
Сине яратканны ташлама.

Күпме гомер көтеп яшәдем мин,
Аңларсың дип ялгыш адымың.
Сабыр төбе алтын түгел икән,
Шунда булган иң зур ялгышың.

Йөрәгемне бүлеп син икегә,
Алмаштырдың мине башкага.
Матурлар бик күп булса да алар,
Сине яратканны ташлама.

Кулларымда бүген газиз бала,
Яннарымда көчле терәгем.
Гафу үтенеп син хатлар язма,
Юк шул инде мина кирәгең.

Йөрәгемне бүлеп син икегә,
Алмаштырдың мине башкага.
Матурлар бик күп булса да алар,
Сине яратканны ташлама.
```
