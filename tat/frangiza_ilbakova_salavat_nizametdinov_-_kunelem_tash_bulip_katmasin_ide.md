# Оригинал

### Франгизә Илбәкова / Салават Низаметдинов - Күңелем таш булып катмасын иде

```
Күңелем таш булып катмасын иде,
Йөрәгем пычракка батмасын ине.
Тик ничек сакларга аларны,
Себереп ташларга чүп-чарны?

Алдыйлар, мактыйлар, җанымны тапыйлар,
Тырныйлар-теләләр, газабын беләләр.
Йөрәгем горур, тик көрәшер көчем юк,
Сөйгәнем сатылган, таяныр кешем юк.
Яман җил-давыллар җәфалый мине. Җәфалый.

Артымда дус-ишләр гайбәт чәйниләр,
Сафлыкка омтылсам, күңелем бәйлиләр.
Алдымда ялагай бишкә бөгелә,
«Хак син» дип куш йөзле хахылдап көлә.

Алдыйлар, мактыйлар, җанымны тапыйлар,
Тырныйлар-теләләр, газабын беләләр.
Йөрәгем горур, тик көрәшер көчем юк,
Сөйгәнем сатылган, таяныр кешем юк.
Яман җил-давыллар җәфалый мине. Җәфалый.
```
