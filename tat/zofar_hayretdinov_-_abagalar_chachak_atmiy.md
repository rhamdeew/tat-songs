# Оригинал

### Зөфәр Хәйретдинов - Абагалар чәчәк атмый

```
Кайда микән бәхет диеп,
Күпме эзләнеп йөрдең.
Абага ярдәм итәр дип
Карурманнарга кердең.

Серле томан чорнап алган
Карурман аланнарын.
Тик беркемнең күргәне юк
Чәчәген абаганың.

Тормыш юлы карурмандай,
Адашырсың берүзең.
Абагалар чәчәк атмый,
Юл күрсәтер сөюең.

Серле томан чорнап алган
Карурман аланнарын.
Тик беркемнең күргәне юк
Чәчәген абаганың.

Сөю белән бәхет бергә
Бәйләнгән бер төенгә
Абагалар чәчәк атмый,
Үссәләр дә өеңдә.

Серле томан чорнап алган
Карурман аланнарын.
Тик беркемнең күргәне юк
Чәчәген абаганың.
```
