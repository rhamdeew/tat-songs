# Оригинал

### Разил Вәлиев / Мәсгудә Шәмсетдинова - Таң (икенче вариант)

```
Зəңгəр яулык япкан иртə
Йокысыннан уянды,
Ул күзлəрен ачу белəн
Алланып офык янды.

Һəм ул күрде шаян җилнең
Агачны уятканын,
Тулган айның тау артында
Югалганын, батканын.

Аннан күрде юынганын
Чык белəн үлəннəрнең,
Һəм ишетте чəчəклəрнең
Чылтырап көлгəннəрен.

Тыңлады ул тыпылдавын
Иң соңгы тамчыларның,
Кояш алтын нуры белəн
Тəрəзə шакыганын.
```
