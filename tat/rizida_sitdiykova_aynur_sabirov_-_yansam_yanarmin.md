# Оригинал

### Ризидә Ситдыйкова / Айнур Сабиров - Янсам янармын

```
Сиңа, җаным, кабат киләмен,
Сөйгәнемне мин бит беләмен.
Аңлашмыйча йөрүебезгә
Син дә, җаным, ничек түзәсең?!

Янсам — янар, көйсәм — көярмен,
Мин барыбер сине сөярмен.
Мәхәббәтнен кайнарлыгында
Кулларымда гына йөртермен!

Аңлашыйк без, әйдә, аңлашыйк,
Саубуллашып хата ясамыйк.
Җанга урын таба алмыйча,
Тормыш буйлап аерым атламыйк!

Янсам — янар, көйсәм — көярмен,
Мин барыбер сине сөярмен.
Мәхәббәтнен кайнарлыгында
Йөрәгемдә генә йөртермен!
Янсам — янар, көйсәм — көярмен,
Мин барыбер сине сөярмен.
Мәхәббәтнен кайнарлыгында
Кулларымда гына йөртермен!
```
