# Оригинал

### Ләйлә Дәүләтова / Оскар Усманов - Ялгыз булу гөнаһ диләр

```
Сандугач балалары күп,
Сайрый иде кичләрен.
Сүрелде дә мени инде
Ялкынланган хисләрең.

Ялгыз булу гөнаһ диләр,
Төшкә генә кермә син.
Сине көткән бәхетеңне
Ятлар эләктермәсен.

Бәхетемдер дигән идем,
Син томан гына бугай.
Хисләремне сындырырсың
Гел сынап йөрсәң болай.

Ялгыз булу гөнаһ диләр,
Төшкә генә кермә син.
Сине көткән бәхетеңне
Ятлар эләктермәсен.

Көткән чагымда килеп кал,
Сабырлыгым чамалы.
Күпләр эзләргә телиләр
Тулып пешкән алманы.

Ялгыз булу гөнаһ диләр,
Төшкә генә кермә син.
Сине көткән бәхетеңне
Ятлар эләктермәсен.
```
