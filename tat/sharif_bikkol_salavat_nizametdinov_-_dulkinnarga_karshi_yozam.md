# Оригинал

### Шәриф Биккол / Салават Низаметдинов - Дулкыннарга каршы йөзәм

```
Йөзеп барам! Миңа каршы
Аҗгырып килә дулкын.
Күтәреп бәрде дә мине,
Давам итте үз юлын.

Диңгез усал. Ярларына
Ыргыта ул чүп-чарны.
Ә мин кеше! Алышамын
Алга сузып кулларны,
Алышамын.
Дулкыннарга каршы йөзәм,
Кояшка карап йөзәм.
Чакыра яр! Бәхетем бар,
Оу! Кыен! Ләкин мин түзәм,
Дулкыннарга каршы йөзәм,
Йөзәм,түзәм, йөзәм, түзәм!

Чак тын калдым, авызын ачып
Икенче дулкын килде,
Жәлләмәде! Чумдырып бер
Тагы артабан җилне.

Диңгез усал. Ярларына
Ыргыта ул чүп-чарны.
Ә мин кеше! Алышамын
Алга сузып кулларны,
Алышамын.
Дулкыннарга каршы йөзәм,
Кояшка карап йөзәм.
Чакыра яр! Бәхетем бар,
Оу! Кыен! Ләкин мин түзәм,
Дулкыннарга каршы йөзәм,
Йөзәм,түзәм, йөзәм, түзәм!

Тормыш — диңгез, дулкыннарда
Мин йөзәм, йөзәм һаман.
Талусыз алга барырга
Өйрәтә мине заман.

Диңгез усал. Ярларына
Ыргыта ул чүп-чарны.
Ә мин кеше! Алышамын
Алга сузып кулларны,
Алышамын.
Дулкыннарга каршы йөзәм,
Кояшка карап йөзәм.
Чакыра яр! Бәхетем бар,
Оу! Кыен! Ләкин мин түзәм,
Дулкыннарга каршы йөзәм,
Йөзәм,түзәм, йөзәм, түзәм!
```
