# Оригинал

### Илдар Юзеев / Шакир Мәҗитов - Шофер

```
Шофер, сии машинаң белән
Ил байлыгын ташып йөртәсең
Караңгы төндә дә,
Давыллы көндә дә
Барлык авырлыкларны
Җиңеп үтәсең.
Шофер, иркен зур юллардан
Кешеләргә бәхет илтәсең.

Шофер, тыңла мотор тавышын,
Карама җилгә, давыл, яңгырга.
Күпер юк, дип торма,
Яңа юллар эзлә,
Ятып калма кырда.
Шофер, ятып калма кырда,
Эчең пошса, матур җыр җырла.

Шофер, очасың җил кебек,
Ак каеннар калалар җилпенеп.
Кырлар тыңлый синең
Дәртле җырларыңны.
Кыскарт, шофер, тизрәк кайтыр юлларыңны.
Шофер, иркәң кочагына
Очып кайт син, кайт син җилкенеп.
```
