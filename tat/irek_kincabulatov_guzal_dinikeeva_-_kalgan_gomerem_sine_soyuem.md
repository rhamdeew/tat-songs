# Оригинал

### Ирек Кинҗәбулатов / Гүзәл Диникеева - Калган гомерем — сине сөюем

```
Эзләгәнмен сине гомер буе,
Сиңа гомер буе килгәнмен.
Әйтерсең лә мине сөюеңне,
Җирдә барлыгыңны белгәнмен,
Җирдә барлыгыңны белгәнмен.

Син бит — сөйгәнемнең йөрәккәем,
Синең өчен яна сөюем.
Калган гомерем — сине табуларым,
Калган гомерем — сине сөюем.

Бөтен эзләгәнем синдә булган,
Синдә булган бөтен үткәнем.
Каян белеп очраштыра язмыш,
Көтәр мәлләрнең җиткәнен,
Көтәр мәлләрнең җиткәнен.

Син бит сөйгәнемнең йөрәккәе,
Синең өчен яна сөюем.
Калган гомерем — сине табуларым,
Калган гомерем — сине сөюем.

Карашларың бүләк булган иде,
Сөю нуры анда чагыла.
Минем күзләр шул күзләргә бага,
Мәхәббәтем сиңа табына,
Мәхәббәтем сиңа табына.

Син бит сөйгәнемнең йөрәккәе,
Синең өчен яна сөюем.
Калган гомерем — сине табуларым,
Калган гомерем — сине сөюем.
Калган гомерем — сине сөюем…
```
