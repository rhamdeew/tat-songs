# Оригинал

### Нәфис Сафиуллин / Фасил Әхмәтов - Декабрь бураннары

```
Килде инде көтеп алган
Декабрь бураннары,
Килеп җитәр-җитмәс безне
Туңдыра торганнары.

Декабрь бураннары —
Туңдыра торганнары,
Елый-елый морҗалардан
Сызгыра торганнары.

Урап-урап бураннары
Томалый урамнарны.
Ап-ак калын юрган белән
Яба карурманнарны.

Декабрь бураннары —
Көрт өя торганнары,
Улый-улый кар туздырып
Тезә кар диварлары.
```
