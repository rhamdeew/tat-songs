# Оригинал

### Әхмәт Ерикәй / Татар  халык җыры - Алсу (икенче вариант)

```
Авылыбызда бер Алсу бар.
Бит алмалары алсу;
Алсу, Алсу, йөрәк ярсу.
Син булмасаң ямансу.

Алсу матур, Алсу батыр,
Бар эшкә дә уңган ул;
Кырга чыкса, кыр яңгырата
Аның матур җырлары.

Шомырт чәчәк аткан чакта
Алсу йоклап ятырмы?!
Сөймим, дисәң дә, сөярсең
Алсу кебек матурны.
```
