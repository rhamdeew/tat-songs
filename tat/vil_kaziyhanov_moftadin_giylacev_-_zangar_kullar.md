# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Зәңгәр күлләр

```
Зәңгәр күлләр, зәңгәр күлләр
Урмандагы аланда.
Күңел сагышларын басам,
Шул күлләргә барам да.

Зәңгәр күлләр кояшларда
Күзләрне чагылдыра.
Иң беренче сине күргән
Чакларны сагындыра.

Зәңгәр күлнең таң җилләре
Дулкыннарын уйната.
Зәңгәр дулкын зәңгәр күзле
Сөйгәнемне уйлата.

Зәңгәр күлләр кояшларда
Күзләрне чагылдыра.
Иң беренче сине күргән
Чакларны сагындыра.

Акчарлаклар очып уйный
Зәңгәр күл өсләрендә.
Әллә күрешергә микән,
Еш күрәм төшләремдә?

Зәңгәр күлләр кояшларда
Күзләрне чагылдыра.
Иң беренче сине күргән
Чакларны сагындыра.
```
