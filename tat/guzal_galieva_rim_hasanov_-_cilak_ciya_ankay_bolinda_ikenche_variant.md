# Оригинал

### Гүзәл Галиева / Рим Хәсәнов - Җиләк җыя әнкәй болында (икенче вариант)

```
Ак яулыгын уңай (каерып) бәйләп,
Пешкәннәрен генә сайлап,
Җиләк җыя әнкәй болында.
Чиләге дә матурырак
Җиләге дә алсуырак
Күренә бит аның кулында.

Гомер җилдәй җилгәнендә,
Һәр бер өзгән җиләгендәй
Татлы булмагандыр һәр көнө.
Килер гомер агышында
Эзерәк бирсен сагышын да,
Нур-кояшлы булсын һәр көнең!

Нинди рәхәт — җирдә җәйләр!
Нинди бәхет — сау әнкәйләр!
Күңелемдә нурлар бәйләме.
Аланнарым (туган җирем) — гөл-чәчәктә.
Күберәк булсын киләчәктә,
Әнкәй, җиләк җыйыр көннәрең (җәйләрең)!
```
