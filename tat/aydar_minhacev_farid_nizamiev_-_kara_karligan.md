# Оригинал

### Айдар Минһаҗев / Фәрид Низамиев - Кара карлыган

```
Дәшми генә түзалмадым,
Ярый җае табылган.
Җырлый-җырлый син берүзең
Җыя идең карлыган.

Кара карлыган, кара карлыган,
Кара булса да бәхеткә булган.
Килгән ул бәхет бергә барудан,
Бәхеткә булган кара карлыган.

Тигез итә белде безне —
Мәхаббәт алдамады.
Карлыган кара булса да
Күңелләр каралмады.
Безнең кебек сөйгәннәрнең
Бар микән булганнары?
Бергә җыярбыз быел да
Кара карлыганнарны.

Кара карлыган, кара карлыган,
Кара булса да бәхеткә булган.
Килгән ул бәхет бергә барудан,
Бәхеткә булган кара карлыган.

Бер дә онытырлык түгел,
Кергәнсең күңелемә.
Ничә меңнәр арасыннан
Сайладым сине генә,
Сайладым мине генә.
Ярый булган карлыган,
Пар иткән, кавыштырган.

Кара карлыган, кара карлыган,
Кара булса да бәхеткә булган.
Килгән ул бәхет бергә барудан,
Бәхеткә булган кара карлыган.
Килгән ул бәхет бергә барудан,
Бәхеткә булган кара карлыган.
```
