# Оригинал

### Данис Фәхретдинов / Гөлназ Закирова - Әниемә (җиденче вариант)

```
Күпме генә җылы күренсә дә
Күк йөзендә кояш нурлары,
Һәр иртәдә назлап уяталар
Әнкәемнең җылы куллары.

Бәхетләргә төренеп яшә, әнкәй,
Тыныч үтсен тормыш юлларың.
Фәрештәдәй сине саклап торыр
Син яраткан улың, кызларың.

Баланнарны бигрәк татлы диләр,
Булсалар да алар бик ачы.
Безнең өчен син — яраткан әни,
Шатлыкларың күбрәк булсачы.

Бәхетләргә төренеп яшә, әнкәй,
Тыныч үтсен тормыш юлларың.
Фәрештәдәй сине саклап торыр
Син яраткан улың, кызларың.

Гел елмаеп кына яшә, әнкәй,
Моңсулану сиңа килешми.
Әткәй белән иңне иңгә куеп
Тормыш сынавына бирешми.

Бәхетләргә төренеп яшә, әнкәй,
Тыныч үтсен тормыш юлларың,
Фәрештәдәй сине саклап торып,
Син яраткан улың, кызларың.
```
