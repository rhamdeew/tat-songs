# Оригинал

### Әнәс Кари / Ильяс Гыйләҗетдин - Ләйсән ява

```
Яз иртәсен котлагандай,
Матур булып
Зәңгәр күкне каплап алды
Аксыл болыт.
Бизәкләнгән төсле булды
Бөтен һава.
Кар өстенә шыбыр-шыбыр
Яңгыр ява.

Ләйсән, ләйсән,
Иң беренче язгы яңгыр.
Ләйсән, ләйсән,
Син куандыр…
Ләйсән, ләйсән,
Иң беренче язгы яңгыр,
Көчлерәк яу,
Күңелләрне син куандыр.

Җир өстендә яткан карлар
Эреп бетсен.
Басуларда эш кайныйсы
Көннәр җитсен.
Без чыгарбыз бакчаларга
Ул көннәрдә.
Яшелчәләр утыртырбыз
Түтәлләргә.
```
