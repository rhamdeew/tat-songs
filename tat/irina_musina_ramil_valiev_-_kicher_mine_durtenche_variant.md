# Оригинал

### Ирина Мусина / Рамил Вәлиев - Кичер мине (дүртенче вариант)

```
Кичер мине, кичер, мәхаббәтем,
Серле карашыңны күрмәдем.
Бүләк иткән тугай чәчәкләрен
Чәчләремә кушып үрмәдем.

Шатлыклардан йөзең балкыганда,
Шаянланып, уйнап көлгәндә,
Миңа булган кайнар хисләреңне
Соң иде шул инде белсәм дә.

Кичер мине, кичер, мәхаббәтем,
Белми суздым сөю гөлләрен.
Ашыкмагыз, дуслар, яратырга,
Аңламыйча сөю серләрен.
```
