# Оригинал

### Якуп Колмый / Рим Хәсәнов - Лилиям (өченче вариант)

```
Җәйге күктә кара болыт
Яңгырлы яшен көйлә.
Кайда сугылырга белми
Иелә, бөгелә Лилиям.

Лилиям, Лилиям!
Иелә, бөгелә Лилиям!

Җилләр исәр, болыт күчәр,
Гөлләр җиргә иелә.
Ниди язмыш көтә сине
Җирнең күрке Лилиям.

Лилиям, Лилиям!
Җирнең күрке Лилиям!

Язлар килер, күктәй булыр,
Зәңгәр кыңгырау мин беләм.
Күңелем чәчәк булып
Чәчәк ат син, Лилиям!

Лилиям, Лилиям!
Чәчәк ат син, Лилиям!
```
