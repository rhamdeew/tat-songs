# Оригинал

### Муса Җәлил / Латыйф Хәмиди - Уңыш бәйрәме

```
Аллы-гөлле гөлчәчәкләр
Бакчабызны ямьләде.
Килде язгы дәртле эшнең
Җанлы хезмәт бәйрәме.

Йөгерә-йөгерә алмагачтан
Без җыябыз алмалар.
Безнең эштән шундый үскән,
Шундый пешкәннәр алар.

Алмасын, алмасын
Өз, агачта калмасын!
Зур уңышның бәйрәме —
Бар да көлсен, уйнасын.
```
