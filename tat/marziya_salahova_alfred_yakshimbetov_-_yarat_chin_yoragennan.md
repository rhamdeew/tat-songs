# Оригинал

### Мәрзия Сәләхова / Альфред Якшимбетов - Ярат чын йөрәгеңнән

```
Шаян кар бөртегем минем,
Хисләрнең буранлысы!
Шашып гашыйк йөрәкләргә
Киртәләр буламы соң?!

Кышкы салкын, кайнар Сөю —
Нинди капма-каршылык!
Җәяүле буран юлларда
Кар себерә ашыгып.

Чәчләремә килеп кунган
Туктамас бураным да.
Ай нуры, ятып йоклаган,
Төннәрен юрганымда.

Җылы җилләр булып кулың
Битемне назлап сыйпый.
Бәс булып назлы сүзләрең
Керфекләремдә йоклый.
```
