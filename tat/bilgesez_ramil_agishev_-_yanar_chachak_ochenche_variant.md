# Оригинал

### Билгесез / Рамил Агишев - Янар чәчәк (өченче вариант)

```
Синең өчен чәчәк аткан идем,
Язларымда нигә килмәдең?
Янар чәчәк булып балкыдым мин,
Белмәдең син аны, белмәдең.

Җәйләремдә чия булып пештем,
Өзеп кенә кабар чак иде.
Меңне әсир иттем, сине көттем,
Ул минем бик сабыр чак иде.

Көзләремдә чаган булып янам,
Чәчәкләрдән чибәр — айларым.
Яна-яна керәм кышларыма,
Язларымның алып кайнарын.
```
