# Оригинал

### Әсхәр Кәшфуллин / Таһир Хәмитов - Гөрләшеп яшикче, дусларым! (беренче вариант)

```
Күңелне тут ала,
Тормышлар туктала,
Гаиләдә татулык булмаса.
Җырларсың сагыштан,
Җырларсың ялгыштан,
Тормышта язмыштан уңмасаң.

Татулык озайта,
Ызгышлар кыскарта
Болай да бик кыска гомерне.
Кергәнче ак кышка,
Чәчәктәй (болай да) бик кыска
Гомернең кадерен белерлек.

Дусларың булышса,
Кояштай җылытса,
Өшетмәс иде ул кышларын.
Мәхәббәт җиләктә,
Дуслыкка ни җитә,
Гөрләшеп яшикче, дусларым!
```
