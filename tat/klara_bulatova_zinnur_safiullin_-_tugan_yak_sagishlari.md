# Оригинал

### Клара Булатова / Зиннур Сафиуллин - Туган як сагышлары

```
Күп югалттым туган яктан
Читтә йөргән көннөрдө.
Ишетмәдем арышларның
Шаулаганын җилләрдә.

Карабодай кырларының
Акны салып, көрәнен,
Анысын салып, каралҗымын
Кигәннәрен күрмәдем.

Ишетмәдем бу җөйдәге
Соңгы кәккүк җырларын.
Яшел питрау көннәрендә
Мәтрүшкәләр җыймадым.

Туган яктан ерак чакта
Күп югалттым, күп таптым.
Күпме җырын җыеп кайттым
Таулы якның — Карпатның.
```
