# Оригинал

### Кәүсәрия Харрасова / Фәридә Сәхәбиева - Ана моңы

```
Җырлый-җырлый әнкәй, гөлләр карый,
Тәрәз саен — кызыл яраннар.
Сагышларын сибеп гөлләренә,
Газизләрен көтә аналар.

Сабыена сөю мөлдерәгән
Изге җаны җырлый ананың.
Бишегеннән сеңә ана моңы
Нәни йөрәгенә баланың.

Якты тәрәзәгә үрли гөле,
Гөлләрдәй гүзәл күңеле дә.
Йөрәк моңы сеңгән җырларыннан
Өй эчләре нурга күмелә.

Сабыена сөю мөлдерәгән
Изге җаны җырлый ананың.
Бишегеннән сеңә ана моңы
Нәни йөрәгенә баланың.

Җырлый-җырлый, гөлләр карый әнкәй,
Моңсуланып ала күңелләр.
Сагынырбыз шушы мизгелләрне,
Әниләр бит мәңге түгелләр.

Сабыена сөю мөлдерәгән
Изге җаны җырлый ананың.
Бишегеннән сеңә ана моңы
Нәни йөрәгенә баланың.
```
