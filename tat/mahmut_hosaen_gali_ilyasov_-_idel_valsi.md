# Оригинал

### Мәхмүт Хөсәен / Гали Ильясов - Идел вальсы

```
Синең буеңда, Идел,
Шаулый кара урман.
Халыкта синең хакта.
Күпме җырлар туган.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Каеннарың моңы
Истән чыкмый бер дә.
Кыя таудай горур мин,
Син барында җирдә.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Синең ярыңда, Идел,
Яши минем ярым.
Каен кайрысыдай саф
Мәхәббәте аның.

Мин яратам сине, Казан,
Сине, зәнгәр Идел.
Акчарлаклар канат кага,
Дулкынлана күңел.

Кайда гына йөрсәм дә
Синең җылың тоям,
Син — бабаларым җире.
Минем нурлы кыйблам!
```
