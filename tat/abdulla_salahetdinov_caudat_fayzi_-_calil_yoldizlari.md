# Оригинал

### Абдулла Сәләхетдинов / Җәүдәт Фәйзи - Җәлил йолдызлары

```
Озак килә озын юллардан
Җиргә ерак йолдыз нурлары…
Ерак йолдыз булып кабынган —
Әсирлектә илен сагынган
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.

Узган чакта озын юллардан
Сүрелмиләр йолдыз нурлары…
Йолдыз нуры кебек саф калган —
Тап төшерми антын саклаган
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.

Озак килә озын юллардан
Мәңге сүнми йолдыз нурлары…
Сүнмәс йолдыз нуры шикелле,
Җир йөзенә балкып сирпелде
Җәлилнең дә үлмәс…
Җәлилнең дә үлмәс җырлары.
```
