# Оригинал

### Башкорт халык җыры - Чулпан йолдыз

```
Сызылып аткан таңнар чакта,
Су өсте дулкынлана.
Сине сагынып яшь йөрәгем шул
Су кебек дулкынлана.

Ал таң ата, чулпан калка,
Чулпан — таңның көзгече.
Чулпан — ал таңның көзгече
Минем сагыну билгесе.

Ал таң ата, чулпан калка,
Чулпан таңны уята.
Чулпан ал таңны уята,
Миңа, сине уйлата.
```
