# Оригинал

### Татар - Башкорт бию такмагы - Әпипә (икенче вариант)

```
Бас, кызым, Әпипә,
Син басмасаң, мин басам.
Синең баскан эзләреңә
Мин дә китереп басам.

Бас, кызым, Әпипә,
Син басмасаң, мин басам.
Әпипәнең туйларында
Биеп калам, ичмасам.

Бас, кызым, Әпипә,
Идәннәре селкенсен;
Идәннәре селкенсен дә,
Кияү күңеле җилкенсен.

Бас, кызым, Әпипә,
Сындыр идән тактасын;
Сындырсаң идән тактасын,
Кияү түләр акчасын.
```
