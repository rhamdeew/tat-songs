# Оригинал

### Татар - Башкорт халык* / Башкорт халык - Хәтирә (икенче вариант)

```
Капка төбендә торасың,
**
Нечкә билең борасың, Хәтирәм,
Нечкә билләреңне борып,
Кемне кызыктырасың, Хәтирәм?

Тәрәзеңне ачып куеп,
Кемгә күлмәк кисәсең, Хәтирәм?
Кашың кара, буең зифа,
Кем бәхетенә үсәсең, Хәтирәм?

Алма бакчасына кереп,
Гөлгә үрелдек мәллә, Хәтирәм?
Матурлык өлешкән чакта,
Алдан йөгердең мәллә, Хәтирәм?

Алма сатучылар килгән,
Үлчәүләре бар микән, Хәтирәм?
Әҗәлгә дару бар диләр,
Мәхәббәткә бар микән, Хәтирәм?
```
