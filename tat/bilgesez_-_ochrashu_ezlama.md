# Оригинал

### Билгесез - Очрашу эзләмә

```
Еллар аркылы
Килдең кичегеп,
Үткән сөюне
Искә төшереп.
Кабат очрашу
Кирәкме икән?
Олы хисләрне,
Уятып, иркәм?!

Булып кал бары
Яшьлек ялгышым,
Күңел түремдә —
Назым, сагышым
Эзләмә мине,
Хәзер без үткән.
Гомерләр үткән,
Соң инде, иркәм…

Әйләнеп кайтмас,
Яшьлек тә — үткән.
Тәүге сөюләр…
Беренче үпкән…
Тормыш йомгагын
Инде без сүткән.
Язмышта безне
Башкалар көткән…

Булып кал бары
Яшьлек ялгышым,
Күңел түремдә —
Назым, сагышым
Эзләмә мине,
Хәзер без үткән.
Гомерләр үткән,
Соң инде, иркәм…
```
