# Оригинал

### Гөлчәчәк Садретдинова / Фирдүс Тямаев - Үз-үзеңне алдама

```
Бергә булган көннәр,
Сихри айлар, төннәр
Яши күңел түремдә, түремдә.
Араларны ялгап,
Яратырга кабат
Телим бүген дә, бүген дә.

Килмә, килмә диеп,
Йөрмә, йөрмә диеп,
Үз-үзеңне алдама, алдама.
Йөрәгемне телеп,
Хисләремнән көлеп,
Тозлар салма ярама.

Соң түгелдер: карашыңнан күрәм,
Үзең дә бит сөясең, сөясең.
Күзләреңә карап, килә сине кабат
Бердән-берем диясем, диясем.

Килмә, килмә диеп,
Йөрмә, йөрмә диеп,
Үз-үзеңне алдама, алдама.
Йөрәгемне телеп,
Хисләремнән көлеп,
Тозлар салма ярама.

Син генә бит җанда —
Аңла шуны, аңла:
Яратам сине шашып, шашып.
Онытырлык түгел сине, тели күңел,
Язмыштан булмый качып.

Килмә, килмә диеп,
Йөрмә, йөрмә диеп,
Үз-үзеңне алдама, алдама.
Йөрәгемне телеп,
Хисләремнән көлеп,
Тозлар салма ярама.
```
