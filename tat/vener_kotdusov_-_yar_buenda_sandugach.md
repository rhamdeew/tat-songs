# Оригинал

### Венер Котдусов - Яр буенда сандугач

```
Урам аша чыккан чакта
Тайпылдым, егылмадым.
Зиһенем тарала, дускаем,
Сине сагынган саен.

Агыйдел ярында
Сайрыйдыр сандугач.
Сандугачка кушылып җырлыйм,
Өзелеп-өзелеп сагынгач.

Алма агач(ы) чәчкә ата,
Майның уртасы була.
Төн йокламый таңнар ата,
Өзелеп сөйгән чак була.

Агыйдел ярында
Сайрыйдыр сандугач.
Сандугачка кушылып җырлыйм,
Өзелеп-өзелеп сагынгач.

Ишек алды яшел чирәм,
Йөрсәм тапталыр инде.
Үпкән-кочкан җилгә очкан,
Мин сине он(ы)та алмам инде.

Агыйдел ярында
Сайрыйдыр сандугач.
Сандугачка кушылып җырлыйм,
Өзелеп-өзелеп сагынгач.
Агыйдел ярында
Сайрыйдыр сандугач.
Сандугачка кушылып җырлыйм,
Өзелеп-өзелеп сагынгач…
```
