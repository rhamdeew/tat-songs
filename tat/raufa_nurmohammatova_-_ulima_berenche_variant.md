# Оригинал

### Рәүфә Нурмөхәммәтова - Улыма (беренче вариант)

```
Яшәүнең шунда яме,
Дөньяң түгәрәк булса.
Шул дөньяларга гамъ өстәп
Көтеп алган улың туса.

Улыкаем, төпчегем,
Минем алтын бөртегем.
Таянычым булырсың,
Миннән үрнәк алырсың.

Төнлә атылды йолдыз,
Юраганым юш килде.
Бәхетле әти бул, диеп,
Ходаем улымны бирде.

Улыкаем, төпчегем,
Минем алтын бөртегем.
Таянычым булырсың,
Миннән үрнәк алырсың.

Нәселемнең дәвамы
Киләчәккә юл ала.
Миннән соң җирдә яңгырап
Якты, матур җыр кала.

Улыкаем, төпчегем,
Минем алтын бөртегем.
Таянычым булырсың,
Миннән үрнәк алырсың.
```
