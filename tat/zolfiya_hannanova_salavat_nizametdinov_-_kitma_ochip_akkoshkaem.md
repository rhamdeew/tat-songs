# Оригинал

### Зөлфия Ханнанова / Салават Низаметдинов - Китмә очып, аккошкаем!

```
Күл өстендә ялгыз аккош —
Ул югалту, әллә табыш.
Аның сыны шундый таныш.
Йөрәгемдә якты сагыш.

Китмә очып, аккошкаем,
Кавышыйкче табып җаен!
Мин бит дога урынына
Бер үк сүзне кабатлыймын:
Син кояшым минем, аем!
Син кояшым минем, аем!
Шаһит таллар, шаһит каен:
Мин бүтәнне яратмыймын!
Яратмыймын!

Күл өстендә ялгыз аккош
Чакырсам да бирми тавыш.
Араларны бикли камыш
Әллә рәнҗү, әллә каргыш…

Китмә очып, аккошкаем,
Кавышыйкче табып җаен!
Мин бит дога урынына
Бер үк сүзне кабатлыймын:
Син кояшым минем, аем!
Син кояшым минем, аем!
Шаһит таллар, шаһит каен:
Мин бүтәнне яратмыймын!
Яратмыймын!

Күл өстендә ялгыз аккош —
Мәхәббәтем газап катыш.
Сине читкә илтә язмыш,
Әллә юрый, әллә ялгыш.

Китмә очып, аккошкаем,
Кавышыйкче табып җаен!
Мин бит дога урынына
Бер үк сүзне кабатлыймын:
Син кояшым минем, аем!
Син кояшым минем, аем!
Шаһит таллар, шаһит каен:
Мин бүтәнне яратмыймын!
Яратмыймын!
```
