# Оригинал

### Билгесез / Таһир Абдршин - Ырымбурга килегез!

```
Сезнең белән танышырга
Килдем мин ераклардан.
Җырлар белән, моңнар белән
Кайнар сәламнәр белән.

Килегез, килегез
Ырымбурны күрегез!
Килегез, килегез
Безгә кунакка!

Урал буйларыннан мин килдем.
Сакмар суларын кичтем.
Шул якларда туып үстем,
Шул якларның суын эчтем.

Килегез, килегез
Ырымбурны күрегез!
Килегез, килегез
Безгә кунакка!

Хушлашырга вакыт җитте,
Дусларым, кардәшләрем.
Бергә булган минутларны
Онытмам мин һәр көн саен.

Килегез, йөрегез
Җәлил йөрегән әзләрдән.
Килегез, дусларым
Безгә кунакка!
```
