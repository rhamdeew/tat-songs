# Оригинал

### Гөлшат Зәйнәшева / Сәйяр Хәбибуллин - Йолдызыма (икенче вариант)

```
Әллә туып-үстеңме син
Йолдызлар арасында?..
Йолдызларның нуры балкый
Күзеңнең карасында.

Әллә үзең бер йолдызмы —
Җиргә вакытлы төшкән?..
Җир йолдызга әйләндеме
Шул без очрашкан кичтән?..

Җир һәм күк тоташ бер нурда,
Бар дөнья йолдыз кебек.
Синсез дөнья булыр сыман,
Мәңгелек бер боз кебек.

Әллә үзең бер йолдызмы —
Җиргә вакытлы төшкән?..
Җир йолдызга әйләндеме
Шул без очрашкан кичтән?..

Калдырма, мине алып кит
Йолдызлар арасына.
Югалыйм, кереп югалыйм
Күзеңнең карасына!..
```
