# Оригинал

### Гөлүсә Әхмәтҗәнова / Гүзәлия - Югалтасым килми

```
Ходай биргән бәхетем син.
“Гомерлеккә” — дип килдең.
Югалтасым килми сине,
Без икебез бербөтен.

Сындырма канатымны,
Югалтма өметемне,
Тартып алма Ходаем
Бердәнбер бәхетемне

Кулларымда синең куллар,
Йөрәгемдә исемең.
Кавыштырды безне язмыш,
Күзләр генә тимәсен.

Сындырма канатымны,
Югалтма өметемне,
Тартып алма Ходаем
Бердәнбер бәхетемне
```
