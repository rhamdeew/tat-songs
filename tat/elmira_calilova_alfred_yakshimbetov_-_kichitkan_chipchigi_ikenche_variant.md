# Оригинал

### Эльмира Җәлилова / Альфред Якшимбетов - Кычыткан чыпчыгы (икенче вариант)

```
Синдәй утны, синдәй шукны
Булмагандыр күргәнем.
Башларымны әйләндердең,
Җаныма ут үрләдең.

Сине күрдем, шунда белдем
Яратуның чып-чынын.
Чырык-чырык көлеп торган
Син кычыткан чыпчыгын.

Йөзләреңдә кояш микән,
Әллә ай нурларымы?
Урыныннан купкан йөрәк,
Аны син урладыңмы?

Сине күрдем, шунда белдем
Яратуның чып-чынын.
Чырык-чырык көлеп торган
Син кычыткан чыпчыгын.

Син минем бәхет кошыммы,
Әрнү-сагышым микән?
Кычыткан чыпчыгын сөеп,
Әллә ялгыштым микән?!

Сине күрдем, шунда белдем
Яратуның чып-чынын.
Чырык-чырык көлеп торган
Син кычыткан чыпчыгын.
```
