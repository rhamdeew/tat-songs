# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Аллаhның барлыгын онытма

```
Кайгысы үтәр дип, бәхете көтәр дип,
Киләчәк көненә ышанып,
Кайчакта көч бетә, тарала өмет тә,
Хәл калмый дөньясын кочарлык.

Үтәр ул, алланыр, балланыр, жайланыр,
Кояшы куренер офыкта.
Авырлык килгәндә кул салыр иңнәргә,
Аллаhның барлыгын онытма.

Көннәре үтмәс дип, сабырлык җитмәс дип,
Куярга теләсәң ноктаны,
Ашкынып ялгышма, юл куйма ялгышка,
Сабыр ит аз гына, тукта син.

Үтәр ул, алланыр, балланыр, жайланыр,
Кояшы куренер офыкта.
Авырлык килгәндә кул салыр иңнәргә,
Аллаhның барлыгын онытма.

Арасын өзәм дип, барына түзәм дип,
Эчеңә бикләнмә, кирәкми.
Барсы да яхшыга, бары да яктыга,
Сынаулар сынауны сирәкли.

Үтәр ул, алланыр, балланыр, жайланыр,
Кояшы куренер офыкта.
Авырлык килгәндә кул салыр иңнәргә,
Аллаhның барлыгын онытма.
Үтәр ул, алланыр, балланыр, жайланыр,
Кояшы куренер офыкта.
Авырлык килгәндә кул салыр иңнәргә,
Аллаhның барлыгын онытма.
Авырлык килгәндә кул салыр иңнәргә,
Аллаhның барлыгын онытма.
Аллаhның барлыгын онытма…
```
