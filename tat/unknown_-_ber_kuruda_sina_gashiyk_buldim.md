# Оригинал

### Эльвира Сафина / Артур Минһаҗев - Бер күрүдә сиңа гашыйк булдым

```
Бер күрүдә сиңа гашыйк булдым,
Гашыйк булып яшимен әле дә.
Шундый назлы карашларың синең.
Наз булганга яшәү ямьле дә.

Яратам мин сине, җаным,
Бар назымны бүләк итәм.
Бул әле минем яннарымда
Син гел, иркәм.

Башкаларны күрми яшим хәзер,
Бергә чакта рәхәт шул җаныма.
Җырларымда минем саф хисләрем
Кадерлемә, сөйгән ярыма.

Яратам мин сине, җаным,
Бар назымны бүләк итәм.
Бул әле минем яннарымда
Син гел, иркәм.

Бер күрүдә сиңа гашыйк булдым,
Гашыйк булып яшимен әле дә.
Өзелепләр яратуым сине
Яшәвемнең мәгънәсе, яме дә.

Яратам мин сине, җаным,
Бар назымны бүләк итәм.
Бул әле минем яннарымда
Син гел, иркәм.
```
