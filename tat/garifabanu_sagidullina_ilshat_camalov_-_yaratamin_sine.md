# Оригинал

### Гарифабану Сагидуллина / Илшат Җамалов - Яратамын сине

```
Яратам сине, диеп,
Кулыма бер гөл бирдең.
Утырт гөлемне, шиңдермә,
Онытма мине, дидең.

Искә алып сагынучы
Берсе син ул, берсе мин.
Сагыштан йөрәк сызласа —
Эретүче син дә мин.

Биргән гөлең — синең төсең,
Үсә тәрәз төбемдә.
Кил җаныем, көттермә,
Күңелдән моң түгелә.

Искә алып сагынучы
Берсе син ул, берсе мин.
Сагыштан йөрәк сызласа —
Эретүче син дә мин.

Икебез бергә булганда
Ятлар керми арага.
Яраланса йөрәгебез —
Сөю бар бит ямарга.

Сагынганда юатучы
Берсе син ул, берсе мин.
Бу дөньяда яшәү рәхәт
Бергә булгач син дә мин.
Искә алып сагынучы
Берсе син ул, берсе мин.
Сагыштан йөрәк сызласа —
Эретүче син дә мин.
```
