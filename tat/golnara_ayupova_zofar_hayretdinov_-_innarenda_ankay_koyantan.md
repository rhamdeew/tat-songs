# Оригинал

### Гөлнара Әюпова / Зөфәр Хәйретдинов - Иңнәреңдә, әнкәй, көянтәң

```
Истәлегең булып миңа калды, әнкәй,
Чишмәләргә барган көянтәң.
Шул көянтәң алып, иңнәремә салып,
Һәр көн суга барам мин иртән.

Син кочкандай буласың, әнкәй,
Иңнәремә салсам көянтәң.
Чайкалышып күз яшь түгәләр
Мөлдерәмә тулы чиләкләр.

Чишмәләргә һәр көн суга барган чакта
Юлларыма гүя чыгасың.
Шул чагымда, әнкәй, сине юксынудан
Күңел тулмый ничек чыдыйсың.

Син кочкандай буласың, әнкәй,
Иңнәремә салсам көянтәң.
Чайкалышып күз яшь түгәләр
Мөлдерәмә тулы чиләкләр.

Чишмә буйларында үскән бөдрә таллар
Җилләр искән саен бөгелә.
Синең хакта, әнкәй, барлык хәтирәләр
Күңелемдә моңга әверелә.

Син кочкандай буласың, әнкәй,
Иңнәремә салсам көянтәң.
Чайкалышып күз яшь түгәләр
Мөлдерәмә тулы чиләкләр.
Син кочкандай буласың, әнкәй,
Иңнәремә салсам көянтәң.
Чайкалышып күз яшь түгәләр
Мөлдерәмә тулы чиләкләр…
```
