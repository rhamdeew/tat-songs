# Оригинал

### Виталий Агапов - Яңа ел белән, дуслар!

```
Искә төшә балачакның
Шау-шулы бәйрәмнәре,
Яшел чыршы әйләнәли
Яңа ел түгәрәге.

Ак карлардай ак бәхетләр
Телим мин яңа елда.
Сөенечләр, куанычлар
Сез дусларга.

Күз алдымнан бер дә китми
Балачак Кыш бабае.
Шундый матур Кар кызларын
Хәзер кайдан табасың?

Ак карлардай ак бәхетләр
Телим мин яңа елда.
Сөенечләр, куанычлар
Сез дусларга.

Оныта алмыйм бүгенгедәй
Кыш бабай бүләкләрен —
Нечкә билле перәннекләр,
«Барбарис» кәнфитләрен.

Ак карлардай ак бәхетләр
Телим мин яңа елда.
Сөенечләр, куанычлар
Сез дусларга.
```
