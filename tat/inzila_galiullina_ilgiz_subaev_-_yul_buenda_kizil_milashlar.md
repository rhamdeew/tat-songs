# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Юл буенда кызыл миләшләр

```
Тәлгәш-тәлгәш элеп жимешләрен,
Юл буенда үсә миләшләр.
Сердәш тә сез сагыш сарган чакта,
Бирәсез күк төпле киңәшләр.

Кушымта:
Ут ялкыны гүя камап алган,
Ялкын уттай кызыл миләшләр.
Йөрәктәге газап әрнүләрдән
Сыгылмаслык бирче киңәшләр.

Язмыш сынаулары әче жимеш,
Миләшләргә тиңдәш тәмнәре.
Тик миләшлэр кышын баллана —
Карлы миләш җиһан ямьнәре.

Кушымта:

Тәлгәш-тәлгәш кызыл миләшлэр
Юлчыларга бирә киңәшләр.
Саклагыз, дип сөю хисләрен,
Түгелмәсен иде күз яшьләр…
```
