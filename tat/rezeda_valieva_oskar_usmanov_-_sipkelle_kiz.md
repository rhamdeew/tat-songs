# Оригинал

### Резеда Вәлиева / Оскар Усманов - Сипкелле кыз

```
Таныштым бер кыз белән,
Яратуымнан үләм.
Шул кадәр яратуымны
Сипкелләреннән күрәм.

Ялкын чәчен тарыймын,
Күзләренә карыймын.
Битләреннән убә-убә,
Сипкелләрен саныймын.

Йолдыз-йолдыз сипкелләр,
Сибелгән йөзләренә.
Толым-толым чәчләре
Төшкәндер тезләренә.

Ялкын чәчен тарыймын,
Күзләренә карыймын.
Битләреннән убә-убә,
Сипкелләрен саныймын.

Чәчләрен, кызыл, диләр,
Әлле шуңа бик чибәр.
Кызыл түгел, алтындыр,
Янып торган ялкындыр.

Ялкын чәчен тарыймын,
Күзләренә карыймын.
Битләреннән убә-убә,
Сипкелләрен саныймын.

Сипкелләре күп төрле,
Балкый йолдыз шикелле.
Чибәрлеге чиктән ашкан,
Өстәвенә — сөйкемле!

Ялкын чәчен тарыймын,
Күзләренә карыймын.
Битләреннән убә-убә,
Сипкелләрен саныймын.
```
