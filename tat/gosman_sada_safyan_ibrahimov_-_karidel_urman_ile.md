# Оригинал

### Госман Садә / Сафьян Ибраһимов - Каридел — урман иле!

```
Карлыганнар җыйыйм дисәң,
Каридел — урман иле.
Карлыгандай кара күзле
Кызларның туган иле.
Кызларның туган иле,
Каридел — урман иле.

Карлыганга барган чакта,
Икәүләп барып кара.
Кайсы юлдан барсагыз да,
Яшьлеккә алып бара.
Яшьлеккә алып бара,
Икәүләп барып кара.

Кабып кара, татып кара
Кап-кара карлыганнар.
Гүя кызлар юри генә
Өзмичә калдырганнар.
Өзмичә калдырганнар,
Кап-кара карлыганнар.

Карлыганнар җыйыйм дисәң,
Каридел — урман иле.
Карлыгандай кара күзле
Кызларның туган иле.
Кызларның туган иле,
Каридел — урман иле!
```
