# Оригинал

### Илфак Фаризов / Наил Шәймарданов - Авылым картлары

```
Исәнмесез, әбиләрем,
Исәнмесез, бабайлар!
Сезнең белән күрешкәнгә
Үтте инде күп айлар.

Урамнарда, табыннарда
Бер сөйләшү — бер гомер.
Миннән сезгә чәй янына
Күчтәнәчкә җыр булыр.

Авылымның олылары
Сәлам бирә ерактан.
Фатихалар, догалары
Саклап килде газаптан.

Урамнарда, табыннарда
Бер сөйләшү — бер гомер.
Миннән сезгә чәй янына
Күчтәнәчкә җыр булыр.

Картаерга ашыкмагыз,
Булыр әле вакытлар.
Мохтаҗ сезнең киңәшләргә
Үсеп килгән оныклар.

Урамнарда, табыннарда
Бер сөйләшү — бер гомер.
Миннән сезгә чәй янына
Күчтәнәчкә җыр булыр.
```
