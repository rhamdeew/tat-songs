# Оригинал

### Мәхмүт Хөсәен / Ринат Гобәйдуллин - Бир кулыңны, гүзәлем

```
Агым суга ташламыйлар
Ак каен каерысын;
Аккошлар да бергә-бергә,
Без нигә аерым соң?

Бир кулыңны, гүзәлем,
Синсез өзелә үзәгем.

Иртән торып тәрәз ачтым
Гөрли күгәрченнәрем;
Бик сагындым, бәгырькәем,
Килә күрәселәрем.

Бир кулыңны, гүзәлем,
Синсез өзелә үзәгем.

Сандугач баласын тоттым
Урман аланнарыннан;
Синең хәлләреңне сорыйм
Былбыл балаларыннан.

Бир кулыңны, гүзәлем,
Синсез өзелә үзәгем.

Биек таудан чәчәк җыйдым,
Җыймадым сарыларын;
Үзең тапсаң, миңа да бир
Сабырлык даруларын.

Бир кулыңны, гүзәлем,
Синсез өзелә үзәгем.
```
