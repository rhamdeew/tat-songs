# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Рәхмәтеңнән, Ходаем, ташлама!

```
Дөнъя авыр. Кайчак танып булмый:
Кайда аклык, кайда каралык.
Бер гөнаһсыз ап-пак чәчкәләр дә
Җил-давылда бетә каралып.

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!

Гөнаһларың булса, һич яшермә
Кил тәүбәгә һәр чак, эй кеше.
Дөнъяга син, өстә тик яктылык,
Бисмиллалы булсың һәр эшең!

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!

Кыл якшылык, чәч син иман нурын,
Бүген чәч, иртәгә соң булыр;
Синең чәчкән шушы нурларыңнан
Бу дөнъя барыбер яктырыр!

Бәхет, тәүфыйк сорыйм Ходаемнан,
Дога белән көнем башлана.
Абынсам да бер көн, егылсам да
Рәхмәтеңнән, Ходаем ташлама!
Рәхмәтеңнән, Ходаем ташлама!
```
