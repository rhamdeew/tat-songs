# Оригинал

### Дилбәр Гәрәева / Оскар Усманов - Мин бәхетле (өченче вариант)

```
Син ул минем шатлыгымны
Ихластан бүлешүчем,
Син ул миңа "Гүзәлем" дип
Иркәләп эндәшүчем!

Мин бәхетле яннарыңда,
Синдәй ярым булганга,
Иңнәремә канат куеп,
Үсендереп торганга.
Күңел җырлап торсын өчен
Яннарымда син кирәк.
Авыр чакта таянырга
Тормышымда син — терәк.

Сынауларны бергә үтеп,
Тормыш буйлап атларбыз.
Еллар аша саф хисләрне
Суытмыйча сакларбыз.

Мин бәхетле яннарыңда,
Синдәй ярым булганга,
Иңнәремә канат куеп,
Үсендереп торганга.
Күңел җырлап торсын өчен
Яннарымда син кирәк.
Авыр чакта таянырга
Тормышымда син — терәк.

Кая барсак, ни күрсәк тә
Гел бергә без тормышта.
Кавыштырган өчен безне
Рәхмәтем зур язмышка.

Мин бәхетле яннарыңда,
Синдәй ярым булганга,
Иңнәремә канат куеп,
Үсендереп торганга.
Күңел җырлап торсын өчен
Яннарымда син кирәк.
Авыр чакта таянырга
Тормышымда син — терәк.
Авыр чакта таянырга
Тормышымда син — терәк…
```
