# Оригинал

### Башкорт халык җыры - Хәтирә (беренче вариант)

```
Капка төбендә торасың,*
Нечкә билең бөгәсең, Хәтирәм,
Нечкә билләреңне бөгеп,
Кем өчен өзеләсең, Хәтирәм? (Кемнең эчен өзәсең, Хәтирәм!)

Без урамнан үткән чакта,
Капкаң булсын кагарга, Хәтирәм,
Бер дә хәйлә табалмасаң,
Чык тәрәзә ябарга, Хәтирәм,

Аллы гөлле чәчәк бирсәм,
Кадарсыңмы түшеңә, Хәтирәм?
Кич ятканда уйлап ятам,
Кермәссеңме төшемә, Хәтирәм?

* — һәр юлы ике кабатлана
```
