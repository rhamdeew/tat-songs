# Оригинал

### Дилә Булгакова / Рамил Гыймрани - Хәбәр ит

```
Бүген генә килгән көннәр сиңа
Бүләк итә кайгы-карасын.

Хәбәр итми түзмә,
Утлар йота күрмә,
Ерак диеп юллар арасын.

Көтмәгәндә шатлык килгән чакта,
Ялгыз булсаң каушап каласың.

Шул чакта син ярым,
Хәбәр итче яме,
Якынайтып юллар арасын.

Ни булса да уртак бүлешербез,
Тик башыңны гына имә син.

Безгә килгән кайгы
Үтеп китәр җаным,
Илгә генә кайгы килмәсен!
```
