# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Төшәрмен исләреңә

```
Керермен төшләреңә,
Төшәрмен исләреңә.
Минем генә исем китмәс
Синең ул хисләреңә.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.

Тарарсың чәчләреңне,
Көтерсең дәшкәнемне.
Минем кулъяулыгым белән
Сөртерсең яшьләреңне.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.

Килерсең яннарыма,
Басарсың алларыма.
Барыбер түзә алмассың
Йөрәгең янганына.

Теләрсең кочакларга,
Янарсың учакларда.
Әмма синең яннарыңда
Мин булмам ул чакларда.
```
