# Оригинал

### Лариса Абдуллина / Динар Ганиев - Сары чәчәкләр

```
Тугайлардан чәчәк җыйдым,
Өзмәдем сарыларын.
Тик син килеп бүләк иттең
Яз умырзаяларын.

Өзмәгез сары чәчәкләр,
Бүләккә дә бирмәгез!
Ташлап китәр булгач алай,
Вакытлыча сөймәгез.

Аңламадым ул чагымда
Сары чәчәк билгесен.
Белмәдем шул синең якта
Мине нәрсә көтәсен.

Өзмәгез сары чәчәкләр,
Бүләккә дә бирмәгез!
Ташлап китәр булгач алай,
Вакытлыча сөймәгез.

Сары чәчәк — сары сагыш,
Гелән минем хәтердә.
Сары чәчәк — сары сагыш,
Күз алдымда хәзер дә.

Өзмәгез сары чәчәкләр,
Бүләккә дә бирмәгез!
Ташлап китәр булгач алай,
Вакытлыча сөймәгез.
```
