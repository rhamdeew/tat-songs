# Оригинал

### Рамил Закиров / Айнур BoooRe Шарапов - Алга!

```
Кемнең күкрәгендә яна — алга,
Урыныңда утырма, атла!
Кемнең күкрәгендә яна — алга,
Урыныңда утырма, атла!

Күкрәгемдә минем яна,
Булган хисләремне аңла.
Ничек син шуны күрмисең,
Әйткәнне көтеп-көтеп тик йөрисең.

Без бит синең белән
Бергә буласын беләм,
Синең яныңа киләм,
Бүген сине мин яратам, диям.

Кызлар арасыннан тапсаң ин матурын —
Эй, батыр егет, торма син боегып,
Мәхәббәт җирендә булмый сайлап,
Бу сүзләрне кат-кат син кабатла.

Бергә керик, керик мәхәббәт җиренә,
Бергә керик, керик әкият иленә,
Бергә керик, керик мәхәббәт җиренә,
Бергә булыйк, бергә.
```
