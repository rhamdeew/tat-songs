# Оригинал

### Земфира Муллагалиева / Салават Низаметдинов - Син кайда, мин кайда?

```
Әнә ул ал кояш,
Әнә ул тулган ай.
Без килеп очрашыр,
Елашыр юл кайда?

Без бит синең белән
Икегә бүленгән.
Бер бөтен, бер дөнъя,
Син кайда, мин кайда?
Мин кайда, син кайда?
Мин кайда?

Кайда ул алмагач?
Кайда ул алсу таң?
Бер назлы каштан
Чәчәкләр күз ачкан.

Без бит синең белән
Икегә бүленгән.
Бер бөтен, бер дөнъя,
Син кайда, мин кайда?
Мин кайда, син кайда?
Мин кайда?

Көз җитә бакчада,
Өлгерә алмалар.
Өлгергән алмалар
Ятларга калалар.

Без бит синең белән
Икегә бүленгән.
Бер бөтен, бер дөнъя,
Син кайда, мин кайда?
Мин кайда, син кайда?
Мин кайда?
```
