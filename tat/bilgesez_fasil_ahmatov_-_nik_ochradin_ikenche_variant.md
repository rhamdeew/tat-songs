# Оригинал

### Билгесез / Фасил Әхмәтов - Ник очрадың? (икенче вариант)

```
Ник очрадың минем юлымда син,
Нигә йөрәгемә ук аттың?
Югалырлык булгач, ник табылдың?
Мин югалттым сине, югалттым.

Матур җырың, татлы сүзең белән
Боек күңелемне юаттың.
Тагын киләм, дидең, ник килмәдең?
Мин югалттым сине, югалттым.
```
