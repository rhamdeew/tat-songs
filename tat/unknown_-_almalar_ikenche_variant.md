# Оригинал

### Гөлнара Рәшитова / Айрат Нәбиев - Алмалар (икенче вариант)

```
Алмалар үсә икән,
Өзелеп төшә икән.
Алма кебек дуслар барда
Күңелләр үсә икән.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.

Алмалар уңган икән,
Бәхеткә булган икән.
Серләремне бүлешердәй
Дуслар табылган икән.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.

Алмаларны өзәрбез,
Өстәлләргә тезәрбез.
Дуслар белән бу дөньяларда
Дәрьяларны гизәрбез.

Алмалары ал гына,
Тәмләп кара, бал гына.
Дуслар белән аралашу
Күңелләрдә ял гына.
```
