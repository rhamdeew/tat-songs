# Оригинал

### Зөлфәт Хәким - Казан-кала

```
Ник болай караңгы бу көн,
Күнелдә гарәсәт бүген?
Син минсез яши алырсың,
Мин синсез яши алмам?!

Шуны белгәнгәдер инде,
Өметләнеп сиңа сеңдем.
Кай тарафтан ярдәм көтик,
И, гүзәл Казан-кала?!

Тетрәтеп татар ил көнен,
Караңгылап якты күген,
Боектырып Кабан күлен,
Чиркәүнең чаңы кага.

Күз яшемне көчкә тоям,
Үз-үземә әле дә сыям,
Син моңа ничек түзәсең,
И, бөек Казан-Кала?!

Җырлый бөек халык
"Узган гомер аккан су" — дип.
Кыйблаларга таба ага
Мәнгелек Идел-елга.

Тоткынлыкта мин калмамын,
Тик синсез яши алмамын!
Агып ла китикме әлле,
И, горур Казан-кала!?
```
