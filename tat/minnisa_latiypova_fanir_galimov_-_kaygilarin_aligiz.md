# Оригинал

### Миңниса Латыйпова / Фәнир Галимов - Кайгыларын алыгыз

```
Бәхет барда уйламыйсың
Кайгы килеп кагасын.
Ачы язмышка саласым,
Күз яшьләре агасын.

Сынганнарның очар канатларын,
Кайгыларын бераз алыгыз.
Сыгылмаска хәсрәт давылында
Калдырмаек аны ялгызын.

Урап үтәр, диеп кайгылар,
Өметләр бит ялгыштыра.
Ә көтмәгәндә килеп ул,
Сарыга сабыштыра.

Сынганнарның очар канатларын,
Кайгыларын бераз алыгыз.
Сыгылмаска хәсрәт давылында
Калдырмаек аны ялгызын.

Халык әйткәч, рас килә ул,
Кайгы йөрми агач башыннан.
Аяз күктә яшен ялтлагандай,
Кайгы килә кеше башына.

Сынганнарның очар канатларын,
Кайгыларын бераз алыгыз.
Сыгылмаска хәсрәт давылында
Калдырмаек аны ялгызын.
```
