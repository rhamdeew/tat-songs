# Оригинал

### Вера Усманова / Илназ Сафиуллин - Киек казлар (икенче вариант)

```
Күктә моңлы җыр калдырды
Киек казлар чылбыры
Яңа язларга кәдәрле
Сагынырмын бу җырны.

Йөрәгем дә, хисләрем дә
Хушлашкан кошлар белән
Алар минем хәлне белми
Мин аларныкын беләм.

Беләм авыр минутлары
Талган канаттай авыр
Киткәннәр булсын түземле
Көткәннәр булсын сабыр.
```
