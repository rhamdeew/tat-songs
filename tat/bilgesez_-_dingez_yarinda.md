# Оригинал

### Билгесез - Диңгез ярында

```
Кояшлы җәйләрдә диңгезгә бардым мин
Нигә бик азга?
Санаулы көннәрем минем нигә соң, нигә
Тиз уза?

Син миңа елмайдың диңгез, диңгез янында,
Мин сине сайладым нигә, нигә бик азга?
Сагынсаң әгәр дә, бик тә, бик тә сагынсаң,
Кил тагын диңгезгә, диңгезгә кил.

Дулкынлы диңгездә “Яратам” дидең син
Ышанып миңа.
Диңгезнең комыннан зур сарай өйдең син,
Тик дулкын юды.

Ник миңа ышандың диңгез, диңгез янында,
Мин сине сайладым нигә, нигә бик азга?
Сагынсаң әгәр дә, бик тә, бик тә сагынсаң,
Кил тагын диңгезгә, диңгезгә кил.
```
