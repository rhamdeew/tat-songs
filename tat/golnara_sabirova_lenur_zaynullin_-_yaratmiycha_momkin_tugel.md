# Оригинал

### Гөлнара Сабирова / Ленур Зәйнуллин - Яратмыйча мөмкин түгел

```
Яратуны берни җиңә алмый,
Яратмаган кеше бәхетсез.
Ярый әле гашыйк итәлгәнсең
Минем белән бергә китәлгәнсең,
Син булганда яшәү тора җырдан.
Син булмасаң, шундый күңелсез.

Яратмыйча сине мөмкин түгел,
Карашыңда гына күпме наз.
Рәхмәт сиңа, янда булган өчен,
Син бирәсең миңа яшәү көчен.
Яратмыйча сине мөмкин түгел,
Аңлатырга сүзләр генә аз.

Яратудан торган гомерләрдә
Аяз булып ата һәрбер таң.
Синең кочагыңда шундый рәхәт,
Яратканың өчен мең рәхмәт.
Мин көн саен сиңа гашыйк булам,
Мин көн саен ныграк яратам.

Яратмыйча сине мөмкин түгел,
Карашыңда гына күпме наз.
Рәхмәт сиңа, янда булган өчен,
Син бирәсең миңа яшәү көчен.
Яратмыйча сине мөмкин түгел,
Аңлатырга сүзләр генә аз.

Яратмыйча сине мөмкин түгел,
Карашларың җанны өтәрлек.
Мин канатсыз идем, канат бирдең
Ялкыннарга салып, ярат дидең.
Җирне җылытырлык минем сөю
Гомер буйларына җитәрлек.

Яратмыйча сине мөмкин түгел,
Карашыңда гына күпме наз.
Рәхмәт сиңа, янда булган өчен,
Син бирәсең миңа яшәү көчен.
Яратмыйча сине мөмкин түгел,
Аңлатырга сүзләр генә аз.
```
