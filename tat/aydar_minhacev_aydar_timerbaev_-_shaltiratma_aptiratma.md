# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Шалтыратма, аптыратма…

```
Көн тыныч үтте дисәм, юк икән бит: булмады,
Татлы йокыларым бүлеп, телефон шалтырады.
Беләм инде: син бу тагын, кайчан инде котылам,
Нигә диеп миңа болай иярдең койрык сыман.

Шалтыратма, аптыратма,
Безнең мәхәббәт үткән.
Синең матур әкиятләр
Кирәкми миңа бүтән.

Ник әйткәнне аңламыйсың, ни җитми дә, ни кирәк?
Синең кебек бәйләнчекләр буладыр җирдә сирәк.
Сөйләмә дә, туйдым инде юклы-барлы сүзеңнән,
Элек ышанганым өчен әле дә гел үкенәм.

Шалтыратма, аптыратма,
Безнең мәхәббәт үткән.
Синең матур әкиятләр
Кирәкми миңа бүтән.

Көн тыныч үтте дисәм, юк икән бит: булмады,
Татлы йокыларым бүлеп, телефон шалтырады.
Онытырга вакыт мине, өметләнәсең юкка,
Ничә тапкыр әйттем инде: бүген соңгысы, нокта.

Шалтыратма, аптыратма,
Безнең мәхәббәт үткән.
Синең матур әкиятләр
Кирәкми миңа бүтән.
Шалтыратма, аптыратма,

Кирәкми миңа бүтән…
```
