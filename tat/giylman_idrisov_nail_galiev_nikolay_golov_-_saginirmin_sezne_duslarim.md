# Оригинал

### Гыйльман Идрисов / Наил Галиев (Николай Голов) - Сагынырмын сезне, дусларым

```
Мине вакыт тагы ашыктыра,
Заман җиккән ярсу атларын.
Язмыш мине, кайда ташласа да,
Сагынырмын сезне, дусларым.

Хушлашырга биргән кулыбызның
Кайнарлыгын саклый учларым.
Еллар үткән саен мин ныгырак
Сагынырмын сезне, дусларым.

Бер-беребезне мәңге онытмаек,
Килеп торсын сәлам хатлары.
Бар чагыздан хәбәр көтә-көтә
Сагынырмын сезне, дусларым.
Сагынырмын, инде сагынырмын,
Искә алып сезне, дусларым.
```
