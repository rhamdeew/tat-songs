# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Бисмилла

```
Иртән торгач мөселманның
Тәүге сузе-бисмилла.
Бисмилланы әйтсә генә
Күңеленә хуш була.
Бисмилланы кабатласаң
Савап була, эш уңа.
Безгә күктән иңдерелгән
Тылсымлы сүз-бисмилла!

Кешем, изге догалардан
Син үзеңә көч юлла.
Дөньясында иң күркәм сүз-
Дога башы-бисмилла!

Бисмилла ул-җан сафлыгы,
Арттыру юк һич монда.
Илаһыга якынайткан
Кодрәтле сүз-бисмилла!
Бөек Затка-Илаһыма
Бөтен җиһан буйсына.
Рәхмәт булып күк катына
Ирешсәңче, бисмилла!

Кешем, изге догалардан
Син үзеңә көч юлла.
Дөньясында иң күркәм сүз-
Дога башы-бисмилла!
```
