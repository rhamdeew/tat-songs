# Оригинал

### Шәриф Биккол / Салават Низаметдинов - Сөмбелем

```
Тугайдагы гөлләр сымак
Табигать ямьнәребез,
Сөмбелем, җан гөлем.

Бер-беребез өчен генә
Яралган җаннарыбыз.
Сөмбелкәем, Сөмбелкәем гөлкәем
Яралган җаннарыбыз.

Әйтешмәдек авыр сүзләр,
Итмәдек хыянатлар,
Сөмбелем, җан гөлем.

Кавышырбыз дип торганда,
Араларга керде ятлар.
Сөмбелкәем, Сөмбелкәем гөлкәем
Араларга керде ятлар.

Әлегечә аерымбыз
Рәнҗемә, кичер мине,
Сөмбелем, җан гөлем.

Кавышыр чаралар булса
Дәръялар кичер идем.
Сөмбелкәем, Сөмбелкәем гөлкәем
Дәръялар кичер идем.
```
