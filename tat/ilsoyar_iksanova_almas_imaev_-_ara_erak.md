# Оригинал

### Илсөяр Иксанова / Алмас Имаев - Ара ерак …

```
Ара ерак …
Очрашулар көтмим,
Берни өмет итмим,
Бары сагынам.
Язларымда, сусап назларыңа,
Каеннарга барып сарылам.

Арак ерак …
Беләм, аңлыйм барын.
Тик оныта алмыйм,
Бары яратам.
Җәйләремдә, үзәгемне өзеп,
Йөрәгемне телеп ата таң.

Ара ерак …
Синсез аткан таңда
Ераклыкны аңлап,
Сыза йөрәгем.
Бәлки, янда булсаң, шушы чаклы (тикле)
Сөймәс идем сине… сөйгәнем.

Сөйгәнем, сөйгәнем,
Сөйгәнем.
```
