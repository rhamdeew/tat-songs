# Оригинал

### Эльза Шагова / Вәгыйзь Газизов - Гомер юлларым (икенче вариант)

```
Юлларымда очрамасын
Үтәлмәслек киртәләр.
Очрасыннар тугры дуслар,
Имин генә булсын юллар,
Балкып атсын иртәләр.

Әй, юлларым, юлларымда
Сагышларым, моңнарым.
Раббым Үзе насыйп итсен:
Сез, дусларым, белән үтсен
Матур гомер юлларым.

Кемнең кем икәнен шулай
Гомер узгач беләсең.
Иң авыр вакытларында
Иңнәренә таянырдай
Дусны сайлап бирәсең.

Дусларымны Ходай бирә
Яхшысын да яңаны.
Язмыш үзенекен итә,
Үзе белән тигезли ул,
Дөрес белән ялгана.

Начар дуслар төшеп кала
Тормышның арбасыннан.
Юллларымны җырлап үтсәң
Ышанычлы дуслар белән,
Юллар борылмасыннар.

Әй, юлларым, юлларымда
Сагышларым, моңнарым.
Раббым Үзе насыйп итсен:
Сез, дусларым, белән үтсен
Матур гомер юлларым.

Начар дуслар төшеп кала
Тормышның арбасыннан.
Юллларымны җырлап үтсәң
Ышанычлы дуслар белән,
Юллар борылмасыннар.

Әй, юлларым, юлларымда
Сагышларым, моңнарым.
Раббым Үзе насыйп итсен:
Сез, дусларым, белән үтсен
Матур гомер юлларым.
Раббым Үзе насыйп итсен:
Сез, дусларым, белән үтсен
Матур гомер юлларым…
Матур гомер юлларым…
```
