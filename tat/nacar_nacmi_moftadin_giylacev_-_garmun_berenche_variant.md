# Оригинал

### Наҗар Нәҗми / Мөфтәдин Гыйләҗев - Гармун (беренче вариант)

```
Кулларыма алсам сине,
Билләрең бөгелмәле.
Бер сыздыру белән, моңың
Күңелгә түгелмәле.

Кечкенә бармак эзләрем
Саклана телләреңдә.
Тәү уйнаган көйләр чыңы
Һаман да йөрәгемдә.

Моңында — туган ягымның
Кояшлы тарафлары,
Агайларымның яшьлеге,
Әнием карашлары.

Юк синдә һич ялган тавыш, —
Кушылып җырлыймын да.
Ярый гармунчы булганмын,
Елыймын, уйныймын да,
Җырлыймын, уйныймын да.
```
