# Оригинал

### Рафаэль Сафин / Хөсәен Әхмәтов - Синдер ул дип уйладым

```
Аланнарны күп буйладым
җырлар җырлап туймадым.
Җырларымның иң яхшысы
синдер ул дип уйладым.

Кырга чыгып, болындагы
гөлгә гашыйк булмадым.
Гөлләрнең дә иң назлысы
синдер ул дип уйладым.

Ерак юлдан сусап килеп,
эчтем (эчәм) чишмә суларын.
Чишмәләр дә баса алмады
йөрәгем ярсуларын.

Чишмәләр дә баса алмагач
йөрәгем ярсуларын,
чишмәләрнең дә чишмәсе
синдер ул дип уйладым.
```
