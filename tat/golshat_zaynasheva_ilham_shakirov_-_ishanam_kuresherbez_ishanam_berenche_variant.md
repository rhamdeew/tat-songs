# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Ышанам! (Күрешербез, ышанам!..) (беренче вариант)

```
Нигә башын түбән идең,
Аерлабыз дисеңме?
Бәлки әле аерылмабыз,
Аерылмабыз шикелле.

Соңгы тапкыр күрәм диеп
Башын салма иңемә.
Бәлки әле очрашырбыз,
Очрашырбыз бүген дә?..

Очрашырбыз, күрешербез,
Иркәм, синең белән без!
Бәлки әле кавышырбыз,
Кавышырбыз әле без!..
```
