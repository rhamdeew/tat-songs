# Оригинал

### Ренат Харис / Эльмир Низамов - Өмет йолдызы

```
Җилләр исә,
сулар ага,
кояшны Ай алыштыра.
Кайдандыр галәм түреннән
йолдыз нуры агып тора.

Болытлар килә,
болытлар китә —
томанлы юллар адаштыра.
Өмет йолдызы
яктылыгыннан
күңелем түремдә балкыш тора.

Туры юллар,
урау юллар
аера һәм кавыштыра.
Өмет дигән йолдыз сүнсә
кайгыларга сабыштыра.

Болытлар килә,
болытлар китә —
томанлы юллар адаштыра.
Өмет йолдызы
яктылыгыннан
күңелем түремдә балкыш тора.
```
