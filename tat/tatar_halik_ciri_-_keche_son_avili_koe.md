# Оригинал

### Татар халык җыры - Кече сөн авылы көе

```
Сөн буендагы талларга
Сандугач кунаклатам шул;
Кунаклатсам — биздермимен,
Сагынсам — сиздермимен.

Сандугачым, кил каршыма,
Кил, кун беләкләремә;
Кил, кун, сайра, булсын файда
Янган йөрәкләремә.

Түгәрәк күл суы салкын,
Куйдым гөлләр үрергә шул;
Ике кулым канат булса,
Килер идем күрергә.
```
