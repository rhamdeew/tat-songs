# Оригинал

### Альбина Мортазина / Рамил Әсхәдулла - Миңа син генә кирәк

```
Сагынып яна йөрәк, очрашабыз ник сирәк?
Яшәр өчен, яшәр өчен миңа бары син кирәк.
Сиңа барыр юлымда үсә куе тал-тирәк,
Адаштырмас өчен юлда миңа бары син кирәк.

Мең төрле газапларны сыйдыра бит бер йөрәк,
Иң бәхетле булыр өчен миңа син генә кирәк.

Сәлам килә, үзең юк, өзелә шуңа үзәк,
Серләремне сөйләр өчен миңа бары син кирәк.
Яратулы караштан гөлләр үсә тизерәк,
Иң матурым дияр өчен миңа бары син кирәк.

Мең төрле газапларны сыйдыра бит бер йөрәк,
Иң бәхетле булыр өчен миңа син генә кирәк.

Йwгереп менәр идем, таулары биегерәк,
Шунда бергә менәр цчен миңа син генә кирәк.
Авыр чаклар килгәндә булырмын сина терәк,
Сынауларны узар өчен миңа бары син кирәк.

Мең төрле газапларны сыйдыра бит бер йөрәк,
Иң бәхетле булыр өчен миңа син генә кирәк.
```
