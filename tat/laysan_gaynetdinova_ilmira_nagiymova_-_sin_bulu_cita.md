# Оригинал

### Ләйсән Гайнетдинова / Ильмира Нәгыймова - Син булу җитә

```
Салкыннар булса җылытырга,
Челәләр булса суытырга.
Бар борчуларны онытырга,
Бәхет кошкаемны тотырга,
Бәхет кошкаемны тотырга.

Син булу җитә яннарымда,
Татлы сөюгә уралырга.
Һәр сулыштан рәхәт алырга,
Һәрбер минутка куанырга.

Хыялларга якынаерга,
Күңел бәхетләргә баерга.
Урын калдырмыйча моңаюга,
Нурлар сибеп гел елмаерга,
Нурлар сибеп гел елмаерга.

Син булу җитә яннарымда,
Татлы сөюгә уралырга.
Һәр сулыштан рәхәт алырга,
Һәрбер минутка куанырга.

Иң биек үрләрне алырга,
Иң якты учакта янарга.
Һәр көн сөенеп уянырга,
Иң зур бәхетне тоярга,
Иң зур бәхетне тоярга.

Син булу җитә яннарымда,
Татлы сөюгә уралырга.
Һәр сулыштан рәхәт алырга,
Һәрбер минутка куанырга.
```
