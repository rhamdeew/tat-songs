# Оригинал

### Баян Котлыев / Роза Сәхәүетдинова - Күл буенда сирень гөлләре

```
Күл буенда сирень гөлләре
Быел назлар икән кенмнәрне?
Искә аласыңмы тугайда
Күбәләкле алсу гөлләрне.

Сагынам бергә үткән,
Сагынам бергә үткән,
Сагынам бергә үткән көннәрне.

Күл буенда сирень гөлләре
Хәтерләтә үткән көннәрне.
Хәзер инде безне юксына.
Күлнең карлыгачлы ярлары.

Төнбоеклы күлнең,
Төнбоеклы күлнең,
Төнбоеклы күлнең  ярлары.

Искә аласыңмы тугайда
Күбәләкле алсу гөлләрне.

Сагынам бергә үткән,
Сагынам бергә үткән,
Сагынам бергә үткән көннәрне.
Күл буенда сирень гөлләре.
```
