# Оригинал

### Ләйлә Дәүләтова / Салават Низаметдинов - Күпме көттек бу мизгелне

```
Көзге салкыннарда адашырлык
Язгы назлар барын сиздеңме?
Ятлар түгел, үзебездә белми
Күпме көттек, көттек бу мизгелне.

Ятлар түгел, үзебездә белми
Күпме көттек, без бу мизгелне.

Давыллардан күпме саклап килдем
Түгелергә торган хисемне,
Сүнеп барган өметләрне ялгап,
Күпме көттек, көттек бу мизгелне.

Сүнеп барган өметләрне ялгап,
Күпме көттек, без бу мизгелне.

Җирдә вакыт туктап калса калсын
Тик калдырсын янда үзеңне.
Яшәү көче беткән чакларда да
Күпме көттек, көттек бу мизгелне.

Яшәү көче беткән чакларда да
Күпме көттек, без бу мизгелне.
```
