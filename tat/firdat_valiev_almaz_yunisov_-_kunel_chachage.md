# Оригинал

### Фирдәт Вәлиев / Алмаз Юнысов - Күңел чәчәге

```
Ага сулар, ага тора
Үтә тора гомерләр.
Яшьлек кенә мәңге яши
Яшьлектә бит күңелләр.

Күңелем чәчәге сулмый,
Яшьлек сөюе уңмый,
Мәхәббәтсез яшәп булмый,
Йөрәк яратып туймый.

Язлар кабатлана ел да,
Сөю үтми шуңамы?
Күпме генә яшәсәң дә
Күңел әллә уңамы?

Күңелем чәчәге сулмый,
Яшьлек сөюе уңмый,
Мәхәббәтсез яшәп булмый,
Йөрәк яратып туймый.
```
