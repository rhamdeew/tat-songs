# Оригинал

### Диләрә Әхмәдуллина / Марат Гобәйдуллин - Ак калфагым (икенче вариант)

```
Кәләпүше, калфак, сөлгеләре —
Халкыбызның күңел җәүһәре.
Сандык төпләрендә сакландылар
Әби-бабамнарның ядкаре.

Калфакларга сеңгән милләтемнең
Милли даны, рухи куәте.
Кайтсын иде килер буыннарга
Үткәннәрнең гореф-гадәте.

Ак калфаклар, матур кәләпүшләр —
Аклык, сафлык, зәвык билгесе.
Энҗе калфак кигән татар кызы —
Милләтемнең якты көзгесе.
```
