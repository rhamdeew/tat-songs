# Оригинал

### Рамил Чурагулов / Илдар Шәрипов - Гомеркәйләр тиз узмасын

```
Гомеркәйләр, гомеркәйләр,
Бик тиз үтә, үтә икән.
Әйләнеп тә караганчы
Гомеркәйләр үтеп киткән.

Гомеркәйләрнең сәгате
Тиз узмасын, тиз узмасын.
Гомерләрнең һәр минуты
Сәгать булып сузылсын.

Мәхәббәтем, мәхәббәтем,
Килдеңме син, килмәдеңме?
Әле дә синең килүеңне
Үзем микән сизмәдемме?

Гомеркәйләрнең сәгате
Тиз узмасын, тиз узмасын.
Гомерләрнең һәр минуты
Сәгать булып сузылсын.

Урам буйлап әби-бабай
Китеп бара, чәче агарган.
Йөзләрендә буразналар,
Йөрәкләре картаймаган.

Гомеркәйләрнең сәгате
Тиз узмасын, тиз узмасын.
Гомерләрнең һәр минуты
Сәгать булып сузылсын.
```
