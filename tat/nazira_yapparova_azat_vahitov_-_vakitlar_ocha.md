# Оригинал

### Нәзирә Яппарова / Азат Вахитов - Вакытлар оча

```
Вакытлар оча үз юлы белән,
Кабатланмый ул гомерләр,беләм.
Эх матур тормыш,якты дөньлар
Үненмик бердә фани дип алар.

Җуела бара яшәү серләре,
Йөрәктә кала иң кадерлесе:
Әнкәй  күзләре,әткәй сүзләре,
Йөгереп үткән сабый эзләре.

Вакыт очышын ялгый гомерләр
Киткән гомерләр-килгән гомерләр.
Эх матур тормыш,якты дөньялар
Үкенеч кенә фани бит алар
```
