# Оригинал

### Мөнир Ишбулатов / Флүрә Үзәнбаева - Гел сагынам

```
Еллар аръягыннан,
Уйлар дәръясыннан.
Кабат кайтам яшьлек таңына.

Йөрәк хисләремдә,
Авыл кичләрендә,
Киләм иркәм синең яныңа.

Мизгел буранында,
Каен урманында.
Яссы яндык утта, ялкында.

Кояш нурларында,
Идел суларында.
Тибрәндек без назлы дулкында.

Былбыл куаклары,
Тургай сукмаклары.
Саклый безнең яшьлек эзләрен.

Дөрләп кабынамын,
Һаман сагынамын.
Җәйге төннең йолдыз күзләрен.
```
