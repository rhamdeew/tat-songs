# Оригинал

### Муса Сираҗи / Салават Низаметдинов - Син генә юк янымда (беренче вариант)

```
Сагышымдай саркып ага,
Черле чишмә алдымда.
Син генә юк,
Син генә юк,
Син генә юк янымда.

Таңны үбеп сайрый кошлар
Талпынышып талында.
Син генә юк,
Син генә юк,
Син генә юк янымда.

Хисләремне җырлап торган
Шаулы урман бар мында…
Син генә юк,
Син генә юк,
Син генә юк янымда.

Хур кызылай сылулар да
Йөри тирә-ягымда…
Син генә юк,
Син генә юк,
Син генә юк янымда.
```
