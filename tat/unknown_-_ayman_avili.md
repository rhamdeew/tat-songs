# Оригинал

### Солтан Шәрипов / Сара Садыйкова - Айман авылы

```
Хуш исләрен бөркеп язлар саен
Шомырт чәчәк ата аланда.
Матурлыгын Айман язларының
Оныта алмыйм нигә һаман да?

Бар нурларын кояш-айларының
Үзенә җыйган минем Айманым.
Иген кырларында туып үстем,
Һөнәремне шунда сайладым.

Алтын куллы Айман кызын сөйдем,
Килегез сез яшьлек туена.
Шатлык җыры яңгырый Айманымда —
Сахраларда, чишмә буенда.

Айман күлләрендә йөзә үрдәк,
Оча казлар канат кагынып.
Читтә торган авылдашлар кайта,
Кайта үз Айманын сагынып.
```
