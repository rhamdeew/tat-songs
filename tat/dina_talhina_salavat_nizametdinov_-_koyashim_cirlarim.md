# Оригинал

### Динә Талхина / Салават Низаметдинов - Кояшым (Җырларым)

```
Кояшым, cин минем кояшым
Җылынам нурларыңа, җылынам нурларыңа.
Нурларыңнан югалам,
Юл алам җырларыма…

Җырларым, син минем җырларым
Төренәм җырларыма.
Мин бәхетле, син очрадың
Син очрадың юлларымда
Мин бәхетле, син очрадың
Син очрадың юлларымда!

Йолдызым, син минем йолдызым
Чорналам нурларыңа, чорналам нурларыңа.
Нурларыңнан югалам,
Юл алам җырларыма…

Җырларым, син минем җырларым
Төренәм җырларыма.
Мин бәхетле, син очрадың
Син очрадың юлларымда
Мин бәхетле, син очрадың
Син очрадың юлларымда!

Ләйсәнем, син минем ләйсәнем
Коенам нурларыңа, коенам нурларыңа.
Нурларыңдан югалам,
Юл алам җырларыма…

Җырларым, син минем җырларым
Төренәм җырларыма.
Мин бәхетле, син очрадың
Син очрадың юлларымда
Мин бәхетле, син очрадың
Син очрадың юлларымда!
```
