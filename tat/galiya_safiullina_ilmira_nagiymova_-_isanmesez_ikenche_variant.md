# Оригинал

### Галия Сафиуллина / Ильмира Нәгыймова - Исәнмесез (икенче вариант)

```
Мин илемнән килдем сезгә,
Күп сәламнәр тапшырып.
Җырларымны бүләк итәм,
Каршылагыз ашкынып.

Исәнмесез, душ-исләрем,
Саумысыз, кардәшләрем!
Сезне күргәч, чәчкә атты
Күңелдәге хисләрем.

Сандугач сайравын отып,
Җырларымны моңла.
Сезнең алкышларыгыз
Илһам өсти җырыма.

Исәнмесез, душ-исләрем,
Саумысыз, кардәшләрем!
Сезне күргәч, чәчкә атты
Күңелдәге хисләрем.

Бергә җырлыйк, җырларыма
Кушылыгыз, дусларым.
Бәхетле булыгыз, дуслар,
Сез — бәхет кошларым.

Исәнмесез, душ-исләрем,
Саумысыз, кардәшләрем!
Сезне күргәч, чәчкә атты
Күңелдәге хисләрем.
```
