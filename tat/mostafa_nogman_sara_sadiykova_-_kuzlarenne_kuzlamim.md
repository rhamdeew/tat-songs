# Оригинал

### Мостафа Ногман / Сара Садыйкова - Күзләреңне күзләмим

```
Күзләреңне, күзләреңне,
Күзләреңне күзләмим.
Мин матурлыгыңны синең
Күзләреңнән эзләмим.

Алларын бирсәң икән,
Гөлләрен бирсәң икән.
Сагынган минутларымда
Ашкынып килсәң икән.

Күзләреңне күзләмим дип,
Алдый гына башладым.
Күз алдымнан китми бер дә
Ягымлы карашларың.

Алларын бирсәң икән,
Гөлләрен бирсәң икән.
Сагынган минутларымда
Ашкынып килсәң икән.

Гөл сулары сибәр идем
Син йөргән эзләреңә.
Бар матурлыкны җыйгансың
Сихерле күзләреңә.

Алларын бирсәң икән,
Гөлләрен бирсәң икән.
Сагынган минутларымда
Ашкынып килсәң икән.
```
