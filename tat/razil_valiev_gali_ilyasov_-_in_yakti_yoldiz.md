# Оригинал

### Разил Вәлиев / Гали Ильясов - Иң якты йолдыз

```
Син еракта, әллә кайда
Биектә идең.
Арабызга болыт керде,
Йолдызым минем.

Йолдыз төсле күзләрең…
Нигә моңсу йөзләрең?
Күктә йолдыз атылганда
Ничек түзәрмен?

Кулларымны күккә сузган
Җир улы идем.
Ә син мәңге минем янга
Төшалмам, дидең.

Йолдыз төсле күзләрең…
Нигә моңсу йөзләрең?
Күктә йолдыз атылганда
Ничек түзәрмен?

Күзен кыса күпме йолдыз…
Мин һаман ялгыз.
Миңа карап яна күктә
Иң якты йолдыз.

Йолдыз төсле күзләрең…
Нигә моңсу йөзләрең?
Күктә йолдыз атылганда
Ничек түзәрмен?
```
