# Оригинал

### Наҗар Нәҗми / Фәрит Хатыйпов - Чәчкәләр арасыңда

```
Сизмәдем, мавыгып әллә
Күзләрең карашына?
Без икәү кердек ал да гөл
Чәчкәләр арасына.

«Матур чәчкә сайла», — дидең,
Матурын таба алмадым,
Чөнки синнән күзләремне
Аерып ала алмадым.

Гашыйк түгелмен ал да гөл
Чәчкәләр карашына.
Син үзең шиңмәс чәчкә бит
Чәчкәләр арасында.

«Матур чәчкә сайла», — дидең,
Матурын таба алмадым,
Чөнки синнән күзләремне
Аерып ала алмадым.

Аллы — гөлле чәчәкләрдән
Тәкыяләр үрәмен.
Бар дөньяның матурлыгын
Күзләреңнән күрәмен.

«Матур чәчкә сайла», — дидең,
Матурын таба алмадым,
Чөнки синнән күзләремне
Аерып ала алмадым.
```
