# Оригинал

### Дилә Булгакова / Артур Туктаголов - Бисмиллалы моңлы Казан

```
Күлмәгең ак, күңелең пакъ,
Түбәтәең чигүле.
Чит итмисең, үз итәсең,
Дус итәсең син һәр кемне.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.

Саклап калдың, аклап калдың,
Гасырларның чал тарихын.
Бармагыңда, көмеш балда,
Хат догалы Кол Шәрифең.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.

Нурлы булсын туган көнең,
Мәчетләрдән килә азан.
Намазыңны калдырмадым
Бисмиллалы моңлы Казан.

Котлы булсын туган көнең,
Мәңге яшә олуг Казан.
Мең гасырлы күкләрендә
Язылгандыр исеме Агъзам.
```
