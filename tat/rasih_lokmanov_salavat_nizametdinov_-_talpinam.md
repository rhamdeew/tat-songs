# Оригинал

### Расих Локманов / Салават Низаметдинов - Талпынам

```
Талпынам мин, талпынам,
Мин елларга артылам,
Куып җитеп та булмыйчы
Яшьлегемнең артыннан.

Талпынам мин, талпынам,
Мәхәббәтем артыннан.
Чәчләремә кар яуганмы?
Тартынам шул тартынам…

Калкынам мин, калкынам,
Җиде диңгез артыннан —
Өнемдә дә, төшемдә дә
Мин очарга талпынам.

Талпынам мин, талпынам —
Киләчәгем артыннан.
Ватанымнан — мин көч алам,
Куәт алам мин — халкымнан!
```
