# Оригинал

### Феликс Мортазин - Кошларым кайта

```
Кайталар кошлар
Сайрарга таңда,
Оя корырга,
Чәчәкле язда!

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!

Кайталар кошлар
Канат кагынып,
Шашып, ашкынып
Язны сагынып,

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!

Кайткан кошкайлар
Кире китмәгез
Кошчыккайларны
Ятим итмәгез!

Мәрхәмәтле бул
Кошларны атма!
Ирексез итмә,
Читлеккә япма!
```
