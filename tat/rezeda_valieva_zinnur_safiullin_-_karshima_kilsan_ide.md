# Оригинал

### Резеда Вәлиева / Зиннур Сафиуллин - Каршыма килсәң иде

```
Кошлар кайткан чак иде,
Сулар аккан чак иде;
Яшьлегебез яз гөледәй
Чәчәк аткан чак иде.

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомеремдә
Югалтмам, дигән идең.
Югалтмам, дигән идең.

Кая соң вәгъдә сүзең,
Юлларда һаман күзем.
Көзен киткән кошлар кебек,
Югалдың нигә үзең?

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомеремдә
Югалтмам, дигән идең.
Югалтмам, дигән идең.

Су өсләре ак күбек,
Мамык тараткан кебек.
Берәүне дә ярата алмам
Сине яраткан кебек.

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомеремдә
Югалтмам, дигән идең.
Югалтмам, дигән идең.
```
