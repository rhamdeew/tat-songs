# Оригинал

### Зинаида Абдуллина (Яковлева) / Виталий Агапов - Кушай авылы җыры

```
Озындыр урамнары,
Каршыда урманнары.
Урманнардан ишетелә
Күке кычкырганнары.

Туган авылым, Кушаем,
Чәчәкле болыннарың.
Киткәннәрең сагынып яши
Туган як җыр-моңнарын.

Матур инеш буйларың,
Үсә бөдрә талларың.
Бөдрә таллар арасында
Уза сабан туйларың.

Туган авылым, Кушаем,
Мәйдан тота ирләрең.
Киткәннәрең сагынып яши
Туган-үскән җирләрен.

Җиләкле аланнарың
Дусия тау буйларында.
Яшәртү шифалары бар
Чишмәңнең суларында.

Туган авылым, Кушаем,
Җиләкле тау буйларың.
Киткәннәрең сагынып яши
Туган як җир-суларын.
```
