# Оригинал

### Ләйлә Дәүләтова / Рамил Агишев - Татлы бүләк

```
Мин сине ялгыш күрдем
Көтмәгән чакта.
Ә хәзер бөтен уйларым
Тик синең хакта.
Әллә мин ялгышып кына
Гашыйк булдыммы?
Әллә соң бу бер мавыгумы?

Син бер елмайсаң,
Сөеп карасаң,
Бөтен дөньялар якты бигрәк.
Синең күзләрең,
Синең йөзләрең,
Синең назларың татлы бүләк.

Бөтен уй-хыялларымны
Чуалттың бит син.
Күңелемнең тынычлыгын
Югалттым бит мин.
Тик шулай да кирәктер ул
Бер шашып кую,
Зур бәхет бит ул гашыйк булу.

Син бер елмайсаң,
Сөеп карасаң,
Бөтен дөньялар якты бигрәк.
Синең күзләрең,
Синең йөзләрең,
Синең назларың татлы бүләк.
```
