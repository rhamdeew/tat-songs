# Оригинал

### Әхмәт Саттар / Наилә Фатехова - Нардуган

```
Лә-лә-лә-лә-лә
Нардуган, Нардуган,
Нардуганда ел туган,
Нардуган, Нардуган,
Нардуганда көн туган.

Кушымта:
Нардуган, әй Нардуган,
Нардуганда ел туган,
Нардуган, әй Нардуган,
Нардуганда көн туган.

Нардуган, Нардуган,
Урамнарда кар, буран.
Нардуган, Нардуган,
Уйный, җырлый Нардуган.

Нардуган, әй Нардуган,
Нардуганда ел туган,
Нардуган, әй Нардуган,
Нардуганда мин туган.
```
