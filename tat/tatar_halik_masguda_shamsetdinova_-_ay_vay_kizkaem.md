# Оригинал

### Татар  халык / Мәсгудә Шәмсетдинова - Ай, вай, кызкаем

```
Ай-вай, кызкаем, печәнчегә бирәем,
Ай-вай, кызкаем, печәнчегә бирәем,
Ай-вай, димәем,
Печән чабу бигрәк авыр,
Ай-вай, сөймәем.

Ай-вай, кызкаем, терлекчегә бирәем,
Ай-вай, кызкаем, терлекчегә бирәем,
Ай-вай, димәем,
Терлекченең эше авыр,
Ай-вай, сөймәем.

Ай-вай, кызкаем, балыкчыга бирәем,
Ай-вай, кызкаем, балыкчыга бирәем,
Ай-вай, димәем,
Балыкчының тәне суык,
Ай-вай, сөймәем.
```
