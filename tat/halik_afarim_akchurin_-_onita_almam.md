# Оригинал

### Халык / Афарим Акчурин - Оныта алмам

```
Агыйделне иңләп, буйлап
Йөзәрсең үрдәкәем.
Ташлар булсаң, ташла бәгърем
Таш булса йөрәккәең.

Җәйгә чыксам, барам әле
Агыйдел буйларына.
Ташласаң да оныта алмам
Гомерем буйларына.

Минем чәчем җиде толым
Җидедән үрмәсәм дә.
Җиде ятка күз дә алмам
Җиде ят күрмәсәмдә.
```
