# Оригинал

### Роберт Әхмәтҗанов / Зиннур Гыйбадуллин - Ишем вальсы

```
Безнең Ишем балкып тора
Ирень суы көзгесендә.
Күбәләкләр, кошлар белән,
Хезмәт сөйгән дуслар белән,
Дуслар белән
Яшәргә язсын синдә!

Әй, безнең Ирень буе —
Тугайларда сабан туе.
Яшьлек язы,
Сөю назы,
Якты заман авазы!

Син минем яшел бишегем,
Һавалар синдә биек!
Тугаеңа кайтсам, Ирень,
Яшьлегемдә кебек йөрим,
Җырлап йөрим,
Җилкенеп, күккә тиеп.

Әй, безнең Ирень буе —
Тугайларда сабан туе.
Яшьлек язы,
Сөю назы,
Якты заман авазы!
```
