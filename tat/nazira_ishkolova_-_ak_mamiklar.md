# Оригинал

### Нәзирә Ишколова - Ак мамыклар …

```
Айлар, еллар үтте, көзләр җитте,
Тагы кошлар китә тезелеп.
Канат җилпеп очкан шул кошларның
Мамыклары оча өзелеп.

Ак мамыклар, ап-ак мамыклар,
Явар микән җиргә кар булып.
Ояларын ташлап очкан кошлар,
Кайтыр микән кире пар (зар) булып?

Тагы мына соңгы көнен барлый,
Алтын сайрай сала көз җиле.
Бу тормышта, бары алышына,
Яшьлек кенә кайтмый шул кире …

Ак мамыклар, ап-ак мамыклар,
Явар микән җиргә кар булып.
Ояларын ташлап очкан кошлар,
Кайтыр микән кире пар (зар) булып?
```
