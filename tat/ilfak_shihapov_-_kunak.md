# Оригинал

### Илфак Шиһапов - Кунак

```
Чыклы юлдан ялан аяклар
Атлый, таптап еллар тузанын.
Санамыйча күпме үткәнен,
Ничә еллар юллар узганын.
Җир һәм күкнең үги баласы,
Кайдан туктап почмак табасы.

Өстә күк, аяк астында җир,
Һаман атлый алга мосафир.

Чакырмыйча килгән кунакны
Ишек ачып алмыйлар каршы.
Йөрәк йозаклары бикләнгән,
Ялгызлыкта дәрвиш карашы.
Юл меңнәрчә елга сузылган,
Шул елларны юлда узылган.

Өстә күк, аяк астында җир,
Һаман атлый алга мосафир.

Без — күктәме, җирдәме — кунак,
Әйтә алмый акыл иясе.
Шуңа да бу җиргә тезләнеп,
Күкләргә башыңны иясе.
Җиргә кунак булып туганнар,
Күктә дә кунакта булганнар.

Өстә күк, аяк астында җир,
Һаман атлый алга мосафир.
```
