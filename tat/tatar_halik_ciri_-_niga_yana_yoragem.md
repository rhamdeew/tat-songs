# Оригинал

### Татар халык җыры - Нигә яна йөрәгем?

```
Сандугачлар ияләшкән
Су эчәргә миләштән.
Сагынуларым бетәр әле
Сине күреп сөйләшсәм.

Уф, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!

Агыйделгә басма салдым
Каенын сайлап кына.
Яннарыңа барыр идем
Кош булып сайрап кына.

Уф, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!

Агыйдел суы тирбәлә
Уртасында сал булса.
Суга салсаң, калкып чыгар
Безгә насыйп яр булса.

Уф, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!

Агыйделкәйне кичкәндә
Җырлыйм ишкәк ишкәндә.
Өзелә үзәк, яна йөрәк
Җаныем искә төшкәндә.

Уф, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!
```
