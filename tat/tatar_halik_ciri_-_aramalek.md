# Оригинал

### Татар  халык җыры - Әрәмәлек

```
Әрәмәләр арасында
Сайрый тургай баласы.
Туган җирем — гөл арасы,
Үзем — авыл баласы.

Әрәмәгә керәм, бәгърем,
Тотмыйсың беләгемнән.
Тамыр җәйгән гөлләр кебек
Китмисең йөрәгемнән.

Әрәмәдә карама шул,
Карамада кондыз бар.
Караңгы дип килми калма,
Ай булмаса йолдыз бар.
```
