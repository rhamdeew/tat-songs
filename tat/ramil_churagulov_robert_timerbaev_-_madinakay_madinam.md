# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Мәдинәкәй, Мәдинәм

```
Күп сиреннәр, алмагачлар,
Гөлгә төрелгән идем.
Мәдинәкәй гөлкәем, дип,
Җырлап йөрегән идем.

Мәдинәкәй, Мәдинәм,
Яшь чаклар, иң шәп чаклар.
Эх, Мәдинәм, Мәдинәм,
Язлар узмаган беләм.

Язлар узып, җәйләр үткәч,
Көзләр килә дөнъяга.
Көзләремдә салкын җилләр,
Кермәс безнең арага.

Мәдинәкәй, Мәдинәм,
Яшь чаклар, иң шәп чаклар.
Эх, Мәдинәм, Мәдинәм,
Язлар узмаган беләм.

Узсын сөеп — сөелепләр,
Һәр кемнең гомерләре.
Безнең олы мәхәббәткә,
Шат бит язның гөлләре.

Мәдинәкәй, Мәдинәм,
Яшь чаклар, иң шәп чаклар.
Эх, Мәдинәм, Мәдинәм,
Язлар узмаган беләм.
```
