# Оригинал

### Билгесез - Ачы балан

```
Өй түрендәге баланны
Кисмә — ярамас, балам.
Язмышыңа кагылмасын
Балан ачысы, балам.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.

Озакламас карлар явар,
Җыя торчы баланын.
Язмыш юлы якты булсын,
Әй, Ходаем, баламның.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.

Балан ачы булса да ул,
Чәчәге матур аның.
Авыр чакта ярдәм сузган,
Онытма, Ходай барын.

Балакаем, балан ачы,
Ашама, берүк, өзеп.
Ачы баланның тәмнәре
Гомергә калыр сеңеп.
```
