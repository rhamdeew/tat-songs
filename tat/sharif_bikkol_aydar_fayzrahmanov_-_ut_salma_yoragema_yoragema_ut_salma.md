# Оригинал

### Шәриф Биккол / Айдар Фәйзрахманов - Ут салма йөрәгемә (Йөрәгемә ут салма)

```
Кулларында тальян гармун
Өздерә үзәгемне.
Әйтмәсәң дә белеп торам
Сиңа тиң түгелемне.

Бакча эчләренә кереп
Өзеп алдым куш алма.
Күзең ялкын, килмә якын,
Йөрәгемә ут салма.

Төн йокымнан уятасың,
Уйнама гармуныңны.
Ничек кенә онытаем
Дөнъяда барлыгыңны.

Бакча эчләренә кереп
Сайлап өзәм куш алма.
Күзең ялкын, килмә якын,
Йөрәгемә ут салма.

Шушы көйне уйныйсың да,
Өзәсең үзәгемне.
Уфтанмаган минутым юк,
Тик түзгәч түзәм инде.

Бакча эчләренә кереп
Сайлап өзәм куш алма.
Күзең ялкын, килмә якын,
Йөрәгемә ут салма.
```
