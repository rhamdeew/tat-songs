# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Ник матур икән Әврез!? (Әврез җыры)

```
Түбәнге Әврез авылы
Дөньяда берәү генә.
Аның бары берәүлеген
Беләм, бар җиһан белә.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм — Әврез матур,
Таһир, Зөһрәләр белән!

Хан кызыннан чибәрерәк
Әврезем гүзәлләре.
Кыйгач кашлы, нечкә билле,
Кирәкми бүтәннәре.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм — Әврез матур,
Таһир, Зөһрәләр белән!

Бер типкәндә тимер өзәр
Егетләр дә бар бездә.
Күз салмагыз сез аларга,
Бик кирәк үзебезгә.

Әгәр, дустым, безнең авыл,
Ник шулай матур дисәң,
Серен әйтәм — Әврез матур,
Таһир, Зөһрәләр белән!
```
