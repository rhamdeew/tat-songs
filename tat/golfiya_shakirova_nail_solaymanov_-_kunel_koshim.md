# Оригинал

### Гөлфия Шакирова / Наил Сөләйманов - Күңел кошым

```
Ятим булыр иде талы
Сандугачлар булмаса.
Нигә кирәк яратулар
Чын күңелдән булмаса.

Сайрамый шул күңел кошы
Кунар гөлен тапмаса.
Хыянәтсез яратып та
Сөйгән ярың ташласа.

Мине эзләп төннәремә
Хәсрәт булып кагылма,
Әрнетергә яраларны
Зинһар өчен сагынма.

Сайрамый шул күңел кошы
Кунар гөлен тапмаса.
Хыянәтсез яратып та
Сөйгән ярың ташласа.

Мөмкин булса, ышандыр син
Чын сөю барлыгына,
Саф хисләрсез, мәхаббәтсез
Дөньялар тарлыгына.

Сайрамый шул күңел кошы
Кунар гөлен тапмаса.
Хыянәтсез яратып та
Сөйгән ярың ташласа.
```
