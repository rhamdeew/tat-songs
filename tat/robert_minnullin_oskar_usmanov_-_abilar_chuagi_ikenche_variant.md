# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Әбиләр чуагы (икенче вариант)

```
Күренеп күзләргә
Табигать үзгәрә.
Калдырып яшьлекне,
Керәбез көзләргә,
Әллә көз килгәнгә,
Яшь килә күзләргә.
Язларны, җәйләрне
Кайлардан эзләргә?

Хисләрем чуалды,
Керфекләр чыланды.
Чылатты шул аны
Әбиләр чуагы…

Киләдер көзләр дә
Үзәкне өзәргә.
Бу моңсу көзләргә
Ничекләр түзәргә?
Әллә көз килгәнгә,
Яшь килә күзләргә.
Язларны, җәйләрне
Кайлардан эзләргә?

Хисләрем чуалды,
Керфекләр чыланды.
Чылатты шул аны
Әбиләр чуагы…
```
