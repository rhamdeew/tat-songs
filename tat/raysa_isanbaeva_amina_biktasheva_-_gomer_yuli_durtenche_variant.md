# Оригинал

### Рәйсә Исәнбаева / Әминә Бикташева - Гомер юлы (дүртенче вариант)

```
Җилеп үтте гомер юлым
Сизмидә калдым үзем,
Әйтерсең дә хәтфә болын,
Ә мин җил булып истем…

Үкенмичә яшәүләргә
Бу дөнъяда ни җитә:
Үкенечле булмый гомер
Изгелекләрдә үтсә.

Гомер үтә, йөгәнсез уй
Яшлеккә кире җилә:
Күңелемне саклый бит ул —
Картлыкны кичектерә.

Үкенмичә яшәүләргә
Бу дөнъяда ни җитә:
Үкенечле булмый гомер
Изгелекләрдә үтсә.

Ак чәчләрне, маңгай сырын
Гомер үзе саладыр…
Изге эшләрне фәрештәләр
Гамәлләргә язадыр.

Үкенмичә яшәүләргә
Бу дөнъяда ни җитә:
Үкенечле булмый гомер
Изгелекләрдә үтсә.
```
