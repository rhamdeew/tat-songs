# Оригинал

### Татар халык җыры - Озату (өченче вариант)

```
Урманнарда йөргәнем дә
Адаштым юлларымны да.
Күңелем йомшап елармын дип
Бирмәдем кулларымны.

Йөгердем җитә алмадым ла,
Сикердем читтәннәрен дә.
Ни булды икән теллләремә
Сау бул дип әйтә алмадым?

Өй артындагы талыңны
Яратмасаң кисәрсен лә.
Бик сагынсаң тәрәзен ач
Мин җил булып исәрмен.
```
