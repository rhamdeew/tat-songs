# Оригинал

### Виталий Агапов - Саубуллашу (беренче вариант)

```
Килдек сезгә, килдек, дуслар,
Җырыбызны бүләк итәргә.
Сез дусларның алкышларын,
Җан җылысын алып китәргә.

Саубуллашыр вакыт та җитте,
Очып үтте минутлар.
Очрашырбыз без, насыйп булса,
Сау булыгыз инде, дуслар!

Килсен бәхет, җырлар белән,
Теләкләрнең сезгә изгесе.
Корычтай нык сәламәтлек,
Зур уңышлар килә телисе.

Саубуллашыр вакыт та җитте,
Очып үтте минутлар.
Очрашырбыз без, насыйп булса,
Сау булыгыз инде, дуслар!
```
