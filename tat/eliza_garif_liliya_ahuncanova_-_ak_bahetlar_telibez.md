# Оригинал

### Элиза Гариф / Лилия Ахунҗанова - Ак бәхетләр телибез

```
Ак бәхетләр телибез без,
Без сезгә туегызда.
Саф мәхаббәт билгесе —
Пар балдаклар кулыгызда.

Котлы булсын, котлы булсын,
Котлы булсын туегыз.
Типсен бергә ике йөрәк,
Бердәм булсын уегыз.

Берсе берсен өзелеп табып,
Кушыла ике йөрәк.
Тигез тормыш, озын гомер —
Бездән сезгә шул теләк.

Котлы булсын, котлы булсын,
Котлы булсын туегыз.
Типсен бергә ике йөрәк,
Бердәм булсын уегыз.

Җыелышкан дусларыгыз,
Бар(ы)сы — туган-тумача.
Җилкендереп күңелләрне
Бәйрәм итик таңгача.

Котлы булсын, котлы булсын,
Котлы булсын туегыз.
Типсен бергә ике йөрәк,
Бердәм булсын уегыз.
Котлы булсын, котлы булсын,
Котлы булсын туегыз.
Типсен бергә ике йөрәк,
Бердәм булсын уегыз…
```
