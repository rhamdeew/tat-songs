# Оригинал

### Фирая Абдуллина / Марат Гобәйдуллин - Күрешербез (икенче вариант)

```
Күңелемнең түрләрендә
Туган иң матур хисләр
Әйтелмәде әле сиңа —
Табалмыйм кирәк сүзләр.

Кушымта:
Килер бер көн — күрешербез,
Очрашыр безнең күзләр,
Мәхәббәтне аңлатырга
Кирәк булырмы сүзләр?

Сүзләрен дә табар идем —
Син миннән ерак әле,
Сине сагынудан күңел
Яңгырдай елак әле.

Хисләремне сүзләр белән
Әйтеп бирә алмасам,
Аңларсың яратуымны
Күзләремә карасаң.
```
