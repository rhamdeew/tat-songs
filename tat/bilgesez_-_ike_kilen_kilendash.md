# Оригинал

### Билгесез - Ике килен килендәш

```
Нинди серебез булса да,
Гаеп итәр димибез.
Апалы-сенелле кебек,
Бер-беребезгә сөйлибез.
Әле дә ярый язган безгә
Килендәшләр булырга,
Житсә иде бу рәхәтләр
Гомеребез буена.

Иң кадерле дускаем да,
Сердәшчем дә син минем.
Сайрашып яшибез шулай
Килендәш ике килен.

Булмады йөргәннәребез
Җиңелне генә эзләп.
Акларын да карасын да
Бүләбез без тигезләп.
Юкка гына түгелдер лә,
Әй рәхәт безнең өйдә.
Үз кызлары сыман күрә
Әти белән әни дә.

Иң кадерле дускаем да,
Сердәшчем дә син минем.
Сайрашып яшибез шулай
Килендәш ике килен.
```
