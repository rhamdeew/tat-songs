# Оригинал

### Дамир Әделбаев - Бу гомер (беренче вариант)

```
Узып кына барган бу гомернең
Кире кайтмаслыгын мин беләм.
Ә шулай да артык пошынмыймын,
Үз юлымнан һаман мин киләм.

Әлми чактан алып хәзергәчә
Фәрештәләр безне җитәкли.
Имин-аман торса бу дөнъялар,
Миңа артык бер ни кирәкми.

Бу дөнъяның тәмен күп татыдым,
Күп түләдем хакын жәлләми.
Җырларымны сезгә җырлап торсам,
Миңа артык бер ни кирәкми.
```
