# Оригинал

### Алсу Шәйдуллина / Марсель Вәгыйзов - Яшисе килә (бишенче вариант)

```
Яшисе килә шулкадәр,
Алда бик зур бурычлар.
Тыныч булсын илем күге,
Гүзәл булсын тормышлар.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.

Яшисе килә шулкадәр,
Насыйп булсын гомерләр,
Тулы, тигез, чиксез бәхет,
Якты кояшлы көннәр.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.

Яшисе килә шулкадәр,
Кеше булып, көч биреп.
Күңелемдәге барлык моңны
Җырларым аша биреп.

Яшисе килә шулкадәр,
Гүзәл итеп дөньяны.
Һәр көн иртән шатлык белән,
Бәхет белән уянып.
```
