# Оригинал

### Фирая Зыятдинова / Фасил Әхмәтов - Җырың буласым килә

```
Ай кызыдай ачык йөзле,
Матур буласым килә.
Җил-давылга бирешмәслек
Батыр буласым килә.

Җилдән дә җитез буласым,
Уңган буласым килә.
Дуслар сөенерлек матур
Тормыш корасым килә.

Синең өчен тулган айның
Нуры буласым килә.
Күңелеңнән мәңге чыкмас
Җырың буласым килә.
```
