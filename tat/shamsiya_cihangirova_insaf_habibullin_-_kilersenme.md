# Оригинал

### Шәмсия Җиһангирова / Инсаф Хәбибуллин - Килерсеңме

```
Килерсеңме икән — килерсеңме?
Күзләреңә генә карар идем.
Тел тибрәтеп бер сүз әйтмәсәң дә,
Аңлар идем сине, аңлар идем.

Килерсеңме икән — килерсеңме?
Ятларга һич күңел салмас идем.
Көлеп торган май кояшы кебек,
Үзеңнән мин бер дә калмас идем.

Килерсеңме икән — килерсеңме?
Һәрбер көнем синсез ятим үтә.
Кыз балага тулы бәхет өчен
Тик бер генә сөю — сөю җитә.
```
