# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Сагындым, Күзәем

```
Күзәемә кайтам көзен,
Бар табигать алтында.
Ак каеннар кулын суза,
Алар мине сагынган.

Әй, Күзәем, матур көзен,
Язын, җәен, кышларын,
Җир читләрендә булсам да
Синдәй авыл күрмәдем.

Ут уйнатып эшлибез без
Без — гармунчы, без — җырчы.
Биюен дә биибез без,
Имән идән сынганчы.

Әй, Күзәем, матур көзен,
Язын, җәен, кышларын,
Җир читләрендә булсам да
Синдәй авыл күрмәдем.

Батыр уллар, гүзәл кызлар,
Арттыра авыл данын.
Күзәйдә тик бер булсаң да
Гомер ашкыныр җаның.

Әй, Күзәем, матур көзен,
Язын, җәен, кышларын,
Җир читләрендә булсам да
Синдәй авыл күрмәдем.
```
