# Оригинал

### Рафига Усманова (Закирова) / Роза Якупова - Җан дусларга

```
Җан дусларым, сезнең белән
Гөрләп үтә гомерләр.
Бергә үткән гомеркәйләр
Үкенерлек түгелләр.

Барчагызны да яратам,
Шатлыктан очыйм микән?!
Кочакка һәммәгез сыяр,
Җир шарын кочыйм микән?!

Үткәнебез ямьле безнең,
Өметләр тәмле безнең.
Тоткан әләм, эш-гамәлләр
Бары да гамьле безнең.

Барчагызны да яратам,
Шатлыктан очыйм микән?!
Кочакка һәммәгез сыяр,
Җир шарын кочыйм микән?!

Очырашып шулай балкып,
Бергә булыйк һәрвакыт.
Яшьнәп яшәделәр, диеп,
Соклансыннар бервакыт.

Барчагызны да яратам,
Шатлыктан очыйм микән?!
Кочакка һәммәгез сыяр,
Җир шарын кочыйм микән?!
```
