# Оригинал

### Энҗе Замилова / Айнур Сәйфиев - Бәгырькәем, бергенәм

```
Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Төшләремдә сине күрдем,
Каршыма килә идең.
Сөясеңме, сөймисеңме,
Җанкисәгем, син мине?

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Шаян итеп сөйләүләрең
Нур бөркетә җаныма.
Сагынуга чик-чама юк,
Барам әле яныңа.

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.

Сиңа киткән юлларым да
Гел чәчкәгә күмелгән.
Үзеңә дә чибәрлекне
Ходаем чиксез биргән.

Бәгырькәем, бергенәм,
Өзелеп сине сөям.
Синсез яшәве күңелсез,
Яннарыңа мин киләм.
```
