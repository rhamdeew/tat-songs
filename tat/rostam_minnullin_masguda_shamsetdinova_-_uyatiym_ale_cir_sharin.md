# Оригинал

### Рөстәм Миңнуллин / Мәсгудә Шәмсетдинова - Уятыйм әле Җир шарын

```
Уятыйм əле Кояшны,
Көннəрем якты булсын,
Уятыйм əле бакчаны,
Көннəрем татлы булсын.

Уятыйм əле Əнине,
Көннəрем җылы булсын,
Уятыйм əле кошларны,
Көннəрем җырлы булсын.

Уятыйм əле дусларны,
Көннəрем якты булсын,
Уятыйм əле Җир шарын,
Көннəрем гөрлəп торсын.
```
