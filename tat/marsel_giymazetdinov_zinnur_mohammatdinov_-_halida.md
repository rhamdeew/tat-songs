# Оригинал

### Марсель Гыймазетдинов / Зиннур Мөхәммәтдинов - Халидә

```
Карлар явып китте,
Кышлар инде җитте —
Кыр казлары очты күкләрдән.
Йөрмә янып-көеп,
Гөлләр сулды диеп,
Язлар яме янсын күзләрдә.

Кышлар үтеп бара,
Кар сулары ага —
Җанга рәхәт бүген, күңелгә.
Гөлләр чәчәк ата,
Киткән кошлар кайта —
Яшеллеккә дөнья күмелгән.

Тургайларга тиң син,
Язлар белән кил син —
Сандугачтай сайра моңланып.
Моңнар диңгезендә
Йөз әле бүген дә —
Җырлар белән күмеп дөньяны.
```
