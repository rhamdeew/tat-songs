# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Кил яныма

```
Елмайганда, чәчәкләр дә
Көнләшә торган чагың.
Күмелгәнсең сагышларга,
Ни булды сиңа тагын?

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.

Күңелеңнең болытларын
Кул белән таратырмын.
Күзләремә каратырмын,
Өзелеп яратырмын.

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.

Су буйлары куе томан,
Чакырам томаннарга.
Алып барам чишмәләргә,
Чылтырап торганнарга.

Кая китте шатлыкларың,
Нигә боек карашың?
Кил яныма, төзәтермен
Йөрәгеңнең ярасын.
```
