# Оригинал

### Эльмира Җәлилова / Фирзәр Мортазин - Өйдә

```
Без җыясы ризык бетмәгәндер,
Без эчәсе сулар бар әле.
Әле дә, шөкер, мең чакрымнар урап
Өйгә кайтыр юллар бар әле.

Өйдә көтә гаилә учаклары —
Хәләлебез, кыз һәм улыбыз.
Әй, Ходаем, кисә күрмә берүк,
Өйгә алып кайтыр юлыбыз.

Төрле җирдә йөртә безне язмыш,
Һәрберебез нидер майтара.
Эштән арып талчыкканда безне
Өй җылысы тартып кайтара.

Өйдә көтә гаилә учаклары —
Хәләлебез, кыз һәм улыбыз.
Әй, Ходаем, кисә күрмә берүк,
Өйгә алып кайтыр юлыбыз.

Шунда кичтек бәхет белән тулы,
Сөю белән тулы елларны.
Әй, Ходаем, шатлыклардан түшә
Өйгә алып кайтыр юлларны.

Өйдә көтә гаилә учаклары —
Хәләлебез, кыз һәм улыбыз.
Әй, Ходаем, кисә күрмә берүк,
Өйгә алып кайтыр юлыбыз.
```
