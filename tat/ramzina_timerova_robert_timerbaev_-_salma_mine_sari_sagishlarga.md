# Оригинал

### Рәмзинә Тимерова / Роберт Тимербаев - Салма мине сары сагышларга

```
Күптән сизәм җылы карашыңны,
Югалтудан куркам үземне.
Серләр тулы, күзләреңнән беләм,
Әйтмәсәмдә әйтер сүземне.

Салма мине сары сагышларга,
Яман уйлар зинһар уйлама.
Соңга калган олы мәхәббәтем,
Якты таңнар булып уяна.

Бу тормышның гөлләре син булсаң,
Чәчәкләре булып тулырмын.
Мәхәббәтең сүрелмәслек булса, / Мәхәббәтем сүнмәс йолдыз булса, /
Утларыңда балкып торырмын.

Салма мине сары сагышларга,
Яман уйлар зинһар уйлама.
Соңга калган олы мәхәббәтем,
Якты таңнар булып уяна.

Юлларымда очрагының өчен,
Үкенмәсәм сине гомергә.
Кайгы-шатлыкларын уртак итеп,
Каршыласак иде гомергә.

Салма мине сары сагышларга,
Яман уйлар зинһар уйлама.
Соңга калган олы мәхәббәтем,
Якты таңнар булып уяна.
```
