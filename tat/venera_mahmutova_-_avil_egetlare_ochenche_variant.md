# Оригинал

### Венера Мәхмүтова - Авыл егетләре (өченче вариант)

```
Еракта булсак та, калада торсак та,
Бар(ы)бер авыл егетләре без.
Күңелләр тартыла, сагынгач авылга
Кайтабыз һәрвакыт үзебез.

Шәһәрдә торсак та,
Без — авыл егетләре.
Егетләрнең шәбе була,
Нәкъ безнең кебекләре.
Без җыелсак — урам тула,
Без — авыл егетләре.
Күңелебез җырга-моңга
Тулып ташкан бер мәле.

Вакыт алга чаба, гомер узып бара,
Күп кирәкми, исән йөрегез.
Гел соңга калма, кайт, егет, авылга,
Авыл баласы бит үзегез.

Шәһәрдә торсак та,
Без — авыл егетләре.
Егетләрнең шәбе була,
Нәкъ безнең кебекләре.
Без җыелсак — урам тула,
Без — авыл егетләре.
Күңелебез җырга-моңга
Тулып ташкан бер мәле.

Сукмагың суытма, беркайчан онытма
Туып-үскән авыл нигезен.
Авыл урамы да, кайту юллары да
Көтеп тора һәрчак егетен.

Шәһәрдә торсак та,
Без — авыл егетләре.
Егетләрнең шәбе була,
Нәкъ безнең кебекләре.
Без җыелсак — урам тула,
Без — авыл егетләре.
Күңелебез җырга-моңга
Тулып ташкан бер мәле.
Күңелебез җырга-моңга
Тулып ташкан бер мәле…
```
