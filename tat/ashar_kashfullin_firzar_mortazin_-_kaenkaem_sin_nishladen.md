# Оригинал

### Әсхәр Кәшфуллин / Фирзәр Мортазин - Каенкаем, син нишләдең

```
Каенкаем, син нишләдең,
Күрәмме әллә ялгыш?
Ник ямь яшел шәлкәеңнән
Коела сары сагыш?

Ник шәлкәең иртә уңган,
Әли бит җәй уртасы?
Әле синең гомерең —
Башланмаган яртысы.

Әле көтә күпме язын,
Көтә күпме җәйләрен.
Яран булса әйт үземә,
Үзем кысып бәйләрмен.

Башланмаган язларыңа
Әле кабат кайтырсың.
Назланып яшел шәлләрдә
Куаныч таратырсың.
```
