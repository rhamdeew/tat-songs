# Оригинал

### Сафуан Әлибай / Фәнир Галимов - Вәгъдә итмә

```
Миңа йолдыз вәгъдә итмә,
Омтылсаң да зәңгәр күккә.
Атылыр да сүнәр йолдыз,
Эзсез калыр мәңгелеккә.

Миңа ташкын вәгъдә итмә,
Дулкыннарың чайкый-чайкый.
Күбекләнеп боз актаргач,
Ул да тыныр, кабат чыкмый.

Миңа таулар вәгъдә итмә,
Янар таулар җирдә сирәк.
Юлдаш булсын тау күтәрер,
Синең генә тугры йөрәк.
```
