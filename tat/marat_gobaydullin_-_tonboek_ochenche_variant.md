# Оригинал

### Марат Гобәйдуллин - Төнбоек (өченче вариант)

```
Төнбоеклы күл буенда
Бергә яккан учаклар.
Күпме генә сагынсаң да,
Кире кайтмый ул чаклар.

Әй. төнбоек, төнбоек,
Күңелем нигә боек.
Әйтче, әйтче. төнбоек,
Күңелем нигә боек.

Гүзәл яшьлек үзәннәре,
Онытылмас гүзәлләре.
Бергә чаклар сагындыра,
Өздерә үзәкләрне.

Әй. төнбоек, төнбоек,
Күңелем нигә боек.
Әйтче, әйтче. төнбоек,
Күңелем нигә боек.

Эх, бер кайтып килсәң иде
Шул гүзәл яшь чакларга.
Яшьлек мәңге онытылмый,
Күңелләрдә саклана.

Әй. төнбоек, төнбоек,
Күңелем нигә боек.
Әйтче, әйтче. төнбоек,
Күңелем нигә боек.
```
