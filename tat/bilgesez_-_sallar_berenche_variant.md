# Оригинал

### Билгесез - Саллар (беренче вариант)

```
Саллар акмаслар шул инде,
Саллар акмаслар инде.
Агыйделнең ярларында
Саллар какмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Егетләр сал өсләрендә,
Учак якмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Салда барган егетләргә
Кызлар бакмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Салда барган егетләргә
Кызлар бакмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Элеккечә алсу булып,
Таңнар атмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Элеккечә алсу булып,
Таңнар атмаслар инде.

Саллар акмаслар шул инде,
Саллар акмаслар инде.
Алар агып кына китә,
Кабат кайтмаслар инде.

Саллар акмаслар,
Саллар акмаслар инде.
Алар агып кына китә,
Кабат кайтмаслар инде.
```
