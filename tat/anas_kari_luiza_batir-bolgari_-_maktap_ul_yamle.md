# Оригинал

### Әнәс Кари / Луиза Батыр-Болгари - Мәктәп — ул ямьле

```
Мәктәп — ул ямьле,
Бик күңелле, ди,
һичкемнең аннан
Кайтасы килми.
Анда әти күк
Олы абый бар,
Аны "Укытучы" дип
Бар(ы)сы атыйлар.

Балаларны ул
Сөеп укыта,
Аларга китап,
Дәфтәр тарата.
Иң кирәклесе
"Әлифба" икән,
Белем ачкычы бит
Шун(ы)сында икән.

Мәктәп — ул ямьле,
Бик күңелле, ди,
һич кемнең аннан
Кайтасы килми.
Белемле итәр
Мине ул мәктәп,
"Әлифба" дигән
Кадерле китап.
```
