# Оригинал

### Фәнис Яруллин / Мәсгуть Имашев - Өзелә үзәгем, өзелә…

```
Өзелә үзәгем, өзелә,
Кайлардан табыйм соң дәвалар?
Сулышым кагылса гөлләргә,
Гөлләр дә ялкынлап яналар.

Өзелә үзәгем, өзелә,
Йолдызлары сүнде күгемнең.
Кайда соң яп-якты көннәрем,
Ник гамьсез бер сабый түгелмен?

Өзелә үзәгем, өзелә,
Хыялның каерылды канаты…
Нигә соң язмышым күз-яшьнең
Ачысын, тозлысын яңартты?
```
