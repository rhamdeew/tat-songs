# Оригинал

### Фәннур Сафин / Риф Гатауллин - Хушлашырга ашыкма (беренче вариант)

```
Хушлаштың да китеп бардың,
Аерылу авыр бит…
Күзләреңнең нурын
Йөрәгемдә саклыйм,
Тукта, дустым, сабыр ит!

Гомер буе ярсу йөрәк
Сине сагынып сулкылдар.
Иделләрдә дулкын, —
Чү, ашыкма, дустым!
Күңелдә ул дулкыннар!..

Сайрар кошлар синсез нишләр,
Ничек агар бу сулар?
Урман ничек шаулар,
Ишелмәсме таулар?
Икебез дә моңсулар…
```
