# Оригинал

### Мокыйм Солтанов / Рамилә Абдуллина - Кар өстенә җырлар яздым

```
Кар өстенә җырлар яздым,
Җырларымда исемең,
Әллә сине яратканга,
Мин җырлымын,хислемен.

Җырлар язма ак карларга,
Язма, дисең, исемне,
Нинди җырлар язсаң да син,
Буран күмәр, дисеңме?

Карлар ява, буран күмә
Икебезнең эзләрне.
Тик бураннар баса алмас
Үзәгем өзелгәнне.

Җырлар язма ак карларга,
Язма, дисең, исемне,
Нинди җырлар язсаң да син,
Буран күмәр, дисеңме?

Яздым тагын ак карларга,
Яздым синең исемне,
Салкын карга, ялкын кебек,
Яздым кайнар хисемне.

Җырлар язма ак карларга,
Язма, дисең, исемне,
Нинди җырлар язсаң да син,
Буран күмәр, дисеңме?
```
