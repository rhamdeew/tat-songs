# Оригинал

### Венера Мәхмүтова / Ирхан (Раиф Мөхәммәтрәхимов) - Пар булсын дөньялары

```
Елгаларга су эчәргә
Төшә парлы аккошлар.
Пар канатларны каермый
Яшәсәк иде дуслар.

Гүзәл бакчада корылган
Аккошлар оялары.
Кителмәсен бәхетләре,
Пар булсын дөньялары.

Тимәсен яман күзләре
Көнче күңелләренең.
Өзелмәсен нык җепләре
Яшьли гомерләренең.

Гүзәл бакчада корылган
Аккошлар оялары.
Кителмәсен бәхетләре,
Пар булсын дөньялары.

Парлап үскән чәчәкләргә
Карап күңел соклана.
Бер тамырдан чыккан гөлдә
Яшәгез сез дөньяда.

Гүзәл бакчада корылган
Аккошлар оялары.
Кителмәсен бәхетләре,
Пар булсын дөньялары.
```
