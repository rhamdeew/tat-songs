# Оригинал

### Нәкыя Йосыпова / Алексей Абрамский - Пар торналар

```
Cанап калдым, парлап-парлап,
Парсызлыкның хәлен аңлап.
Берсе калды, кыйгак-кыйгак,
Йөрәк әрни аны уйлап.

Зәңгәр күктә минем күзем,
Кайтырсыз дип сезне күзлим,
Йөрәгемдә шомнар туа,
Китәрсез дип тагын көзен.

Канат-канат, кабат-кабат
Кайтырсыз сез парлап-парлап.
Хыялымны каршы алам,
Пар торналар сезгә карап.
```
