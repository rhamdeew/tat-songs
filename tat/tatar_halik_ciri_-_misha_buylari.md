# Оригинал

### Татар халык җыры - Мишә буйлары

```
Уйна, дустым, гармуныңны,
Якташларым тыңласын;
Син уйнасаң, мин җырлармын
Мактап Мишә елгасын.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.

Туган авылым Агайбаш ла,
Мишә су буйларында;
Мишә буе бөдрә таллар
Гел минем уйларымда.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.

Мишә буен яңгыратып
Уйна матур көйләргә;
Син уйнаган, мин җырлаган
Калсын сагнып сөйләргә.

Ямьле Мишә буйлары,
Дулкынлана сулары;
Ярларында кошлар сайрый,
Сагындыра шулары.
```
