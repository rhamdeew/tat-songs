# Оригинал

### Гөлшат Зәйнәшева / Марс Макаров - Минем иркәм биюче

```
Иркәң кем, дип сорасагыз,
Минем иркәм — биюче.
Иркәм бик оста биюче,
Мине өзелеп сөюче.

Эх, тыпыр-тыпыр-тыпыр,
Тыпырдап биюләре.
Чынлап микән, юри микән,
Яратам диюләре.

Иркәм әйтә, син дә өйрән —
Биюче булырсың, ди.
Бергә яшәсәк, каршымда
Гел биеп торырсың, ди.

Эх, тыпыр-тыпыр-тыпыр,
Тыпырдап биюләре.
Чынлап микән, юри микән,
Яратам диюләре.

Мин әйтәм, җырлап торырмын
Гел синең яннарыңда,
Күтәреп кенә йөртерсең
Син мине кулларыңда.

Эх, тыпыр-тыпыр-тыпыр,
Тыпырдап биюләре.
Чынлап микән, юри микән,
Яратам диюләре.
```
