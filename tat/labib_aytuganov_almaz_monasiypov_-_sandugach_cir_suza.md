# Оригинал

### Ләбиб Айтуганов / Алмаз Монасыйпов - Сандугач җыр суза

```
Сандугач җыр суза
Тын гына, моң тына;
Өзелеп-өзелеп,
Сандугач җыр суза
Тын гына, моң гына —
Кемне ул юксына?

Ул көтә назлысын,
Килми шул сөйгәне;
Сагына-сагына,
Ул көтә назлысын,
Килми шул сөйгәне —
Бер генәм дигәне.

Сандугач җыр суза
Тын гына, моң гына;
Өзелеп-өзелеп,
Сандугач җыр суза
Тын гына, моң гына —
Иркәсен юксына.
```
