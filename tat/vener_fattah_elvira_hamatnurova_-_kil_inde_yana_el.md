# Оригинал

### Венер Фәттах / Эльвира Хаматнурова - Кил инде, Яңа ел!

```
Кыш бабай, кыш бабай,
Кар ява, кил инде,
Янчыгың чишеп сал,
Бүләкләр бир инде.

Кыш бабай, син безне
Җырлат та биет тә,
Ә йөлдыз чыршыда
Балкысын биектә.

Кар сылу апабыз,
Йөрт әле җитәкләп,
Уйныйк без бергәләп
Чыршыны тирәләп

Кыш бабай, җырларбыз
Гармуның көенә,
Шигырләр сөйләрбез,
Яңа ел, кил генә!
```
