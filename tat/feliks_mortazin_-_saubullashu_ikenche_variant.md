# Оригинал

### Феликс Мортазин - Саубуллашу (икенче вариант)

```
Гөлләр арасында үскәнгәме,
Бигрәкләр дә матур икәнсез!
Әллә йолдызлардан төшкәнгәме
Йөзегездә нурлар исәпсез!

Сезне зурлар өчен дип килдек без
Сезне хөрмәт итеп килдек без
Җырланасы җырлар калды әле
Чакырсагыз тагын килербез!

Җырладык та без, биедек тә без
Бергә-бергә күңел ачтык без
Без китте дип, безне онытмагыз
Хәзер инде сез безгә килегез!

Сезнең белән узган бу миинутлар,
Мәңге калыр күңелебездә,
Аллабирса киләсе көзгә
Әйләнеп кайтырбыз без сезгә!
```
