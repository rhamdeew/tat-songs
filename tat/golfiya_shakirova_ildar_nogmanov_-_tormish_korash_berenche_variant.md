# Оригинал

### Гөлфия Шакирова / Илдар Ногманов - Тормыш — көрәш (беренче вариант)

```
Яшәү кад(е)рен белеп яшик әле,
Тормыш ямен белеп яши белгәнгә.
Бер-береңә ярдәм кулын сузып,
Авырлыклар аеры килгәндә.

Тормыш — хыял, ашырыгыз,
Тормыш — уен, аны уйнагыз.
Тормыш — бәхет, шуны тудырыгыз,
Тормыш — көрәш, аңа чыдагыз.

Тормыш — сер ул, аңа төшенгәндә
Сынауга, дустым, бирешмә.
Моң-зар алар булса — тәвәкәллә,
Гаделсезлек белән килешмә.

Тормыш — хыял, ашырыгыз,
Тормыш — уен, аны уйнагыз.
Тормыш — бәхет, шуны тудырыгыз,
Тормыш — көрәш, аңа чыдагыз.

Мөмкинлекләр булса — файдаланып,
Хозурларын күр, син, рәхәтен.
Мәхаббәттән ләззәт алам, диеп,
Башкаларның алма бәхетен.

Тормыш — хыял, ашырыгыз,
Тормыш — уен, аны уйнагыз.
Тормыш — бәхет, шуны тудырыгыз,

Тормыш — көрәш, аңа чыдагыз…
Тормыш — көрәш, аңа чыдагыз…
```
