# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Мин сине яратам (икенче вариант)

```
Кайчан гына без шушында
Икеү утырган идек.
Күктән яуган йолдызларны
Учка тутырган идек.
Учка тутырган идек.

Еллар кары гына нигә
Күңел хисенә көчсез?
Күпме яуса да, күмә алмый,
Йолдыздай балкый сүз:
"Мин сине яратам,
Яратам, яратам!"

Хәзер ялгыз кырпак карга
Уйларымны тагатам.
Мин килгән идем — син һаман юк,
Тик карлар уйный җилдә.
Тик карлар уйный җилдә.

Еллар кары гына нигә
Күңел хисенә көчсез?
Күпме яуса да, күмә алмый,
Йолдыздай балкый өч сүз:
"Мин сине яратам,
Яратам, яратам!"
```
