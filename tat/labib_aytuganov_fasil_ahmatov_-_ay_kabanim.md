# Оригинал

### Ләбиб Айтуганов / Фасил Әхмәтов - Әй, Кабаным!

```
Әкрен генә шәфәкъ сүнә,
Әй, Кабаным, сиңа ямьле кич килә!
Көмешләнә бөдрә дулкыннарың,
Карап туя алмыйм һич кенә!

Кемнәр генә сине белми,
Әй, Кабаным, сине Тукай җырлаган!
Сиңа килсәм, ишетәм кебек һаман
Аның җырын ерак еллардан.

Яратам мин якты кичең,
Әй, Кабаным, күңелем түзми килмәсәм!
Әйтерсең лә, ярларыңа басып,
Чал тарихың белән сөйләшәм.
```
