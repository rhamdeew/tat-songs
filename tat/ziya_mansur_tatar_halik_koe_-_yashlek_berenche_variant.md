# Оригинал

### Зыя Мансур / Татар халык көе - Яшьлек (беренче вариант)

```
Бакчадагы гөлгә сине
Тиңләр идем, яшьлегем.
Гөлләр көзен саргаялар,
Ә син мәңге яшь минем.

Тиңләр идем шаулап аккан
Язгы ташкын суына.
Сулар кибә, яшьлек шаулый,
Шаулый гомер буена.

Күңел шатлык белән тулы,
Җаным һәрчак яшь минем.
Туган илем — ирек иле,
Мәңге сүнмәс яшьлегем.
```
