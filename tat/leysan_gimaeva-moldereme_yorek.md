### Молдерэмэ йорэк

```
Молдерэмэ, молдерэмэ тулып ташкан хислэр,
Кочагында эри икэн, белмим, хэзер кемнэр.
Актарыла ташкын булып жанда хатирэлэр,
Зинхар очен, утенэм, тотма мине, жибэр.

Жаным, тэнем тели хаман бары сине генэ,
Сина кирэк тугел сою яши йорэгемдэ.
Мин яраткан йорэк тибэ синен кукрэгендэ.

Кузем йомсам, куз алдында бары синен кузлэр,
Назларына сусаган жан, белмим, купме тузэр?
Тошлэремдэ эзэрлекли иреннэрен тэме,
Ничек белим синен уйны, синен кунелдэгене?

Жаным, тэнем тели хаман бары сине генэ,
Сина кирэк тугел сою яши йорэгемдэ.
Мин яраткан йорэк тибэ синен кукрэгендэ.

Тоннэрендэ тошлэрендэ саташтырыр идем,
Союемнен буранында адаштырыр идем.
Мин ялгызым дорлэп янам сою учагында,
Кем бэхетле икэн буген синен кочагында?

Жаным, тэнем тели хаман бары сине генэ,
Сина кирэк тугел сою яши йорэгемдэ.
Мин яраткан йорэк тибэ синен кукрэгендэ.
```
