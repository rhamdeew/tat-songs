# Оригинал

### Мөфтәдин Гыйләҗев - Әгәр үтсәң минем юллардан

```
Әгәр үтсәң минем үткән юлдан
Очратырсың бала чагымны.
Яланнарда йөгереп ялан аяк
Күбәләкләр куган чагымны.

Әгәр үтсәң минем үткән юлдан
Очратырсың яшьлек елларны.
Ишетерсең сагышларга тулы
Яшьләр аша аккан моңнарны.

Тыңлый, тыңлый тормыш җырларымны
Юлларымнан бәлки узарсың.
Юлларымда әгәр туктап калсам,
Миңа килеп кулың сузарсың…
```
