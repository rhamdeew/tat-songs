# Оригинал

### Җәлил Насыйбуллин - Мин сөярмен сине шашынып

```
Кара көзләр үтте, матур кышлар җитте,
Беренче кар ява очынып.
Кара көзләрдә дә, ап-ак кышларда да,
Мин сөярмен сине шашынып.

Ап-ак карлар ява, ап-ак карлар ява,
Тик бәхеткә яусын йомшак кар.
Бары бергә булсын, тик бәхетле булсын
Берсен берсе сөйгән ике яр.

Карлар алып килсен ап-ак бәхетләрне,
Йомшак юрганына төренеп.
Мине сөясең син, мин дә сөям икән,
Яшим бары шуңа сөенеп.

Ап-ак карлар ява, ап-ак карлар ява,
Тик бәхеткә яусын йомшак кар.
Бары бергә булсын, тик бәхетле булсын
Берсен берсе сөйгән ике яр.

Яуган карлар эрер, елга-сулар агар,
Тик мәхаббәт безнең мәңгелек.
Шуңа шатланам мин: янымда син барга,
Газиз ярым минем гомерлек.

Ап-ак карлар ява, ап-ак карлар ява,
Тик бәхеткә яусын йомшак кар.
Бары бергә булсын, тик бәхетле булсын
Берсен берсе сөйгән ике яр.
```
