# Оригинал

### Әсхәр Кәшфуллин / Виталий Агапов - Кагылмагыз ялгыш

```
Җаным тулы сагыш,
Кагылмагыз ялгыш,
Яшем тора атылып чыгарга.
Сорамагыз, нигә
Йөрәк әрни, көя.
Ничек кенә, ничек чыдарга?

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?

Йөрәк тулы сагыш,
Кагылмагыз ялгыш
Күңелдәге нечкә кылларга.
Сабырлык бир, Тәңрем,
Урманнарга керәм,
Бер кычкырып елап, җырларга.

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?

Кан тама күңелдән,
Бу юкка түгел лә.
Яшем тора атылып чыгарга.
Йөрәк дөрләп яна,
Кадерләнә бара
Мәхәббәтсез үткән елларда…

Әллә мине язмыш
Алып китте ялгыш,
Алып китте ялгыш юллардан?
Җан туңып бозланган,
Нигә мин сызланам?
Белмим нигә, нигә юрарга?
```
