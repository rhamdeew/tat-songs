# Оригинал

### Фирая Зыятдинова / Луиза Батыр-Болгари - Бердәнберем син минем

```
Алтыным ла син, диеп,
Ялкыным ла син, диеп
Үстерәм бәбкәемне
Иркәләп, сөеп.
Улыкаем син минем,
Юлыма нур сибүче
Бәхетем минем.
Бәлли-бәү, бәлли-бәү.

Син минем ышанычым,
Бердәнберем дигәнем;
Син — бөркетем, лачыным,
Илгә бүләгем.
Матур булып үсәрсең,
Батыр булып үсәрсең,
Ак өметем син минем,
Бәхетем минем.
Бәлли-бәү, бәлли-бәү,
Бәлли-бәү, бәлли-бәү.
```
