# Оригинал

### Гандәлибә Аҗгихина / Сәүбән Чуганаев - Яңгырлар

```
Күк йөзендә яуган болытлар,
Ява да ява, елап яңгырлар.
Әллә инде табигатьтә дә,
Бар сагынулар, бар югалтулар.

Әй, яңгырлар ява да ява,
Җитүемә инде көзләрне.
Әрнеп өзгәләнгән йөрәгем,
Сагышларга түзәрмен.

Хәтеремдә әле һаман да,
Әйтә алмый калган, назлы сүзләрем.
Сары сагышларга салырга,
Ашыкма әле, минем көзләрем.

Әй, яңгырлар ява да ява,
Җитүемә инде көзләрне.
Әрнеп өзгәләнгән йөрәгем,
Сагышларга түзәрмен.

Тезелешеп очкан кошлардай,
Болыт артыннан, агыла болыт.
Калдың инде, калдың күңелдә,
Мәңге онытылмас, моңлы җыр булып.

Әй, яңгырлар ява да ява,
Җитүемә инде көзләрне.
Әрнеп өзгәләнгән йөрәгем,
Сагышларга түзәрмен.
```
