# Оригинал

### Керәшен халык җыры - Ак куянның ак баласы

```
Ак куянның ак баласын
Җибәр кырга — уйнасын.
Бер уйнасын, ик(е) уйнасын,
Гаеп итеш булмасын.

Ак куяннар атсаң, чүгәләп ат,
Тимәс микән түшенә.
Кич батканда исәпләп җат,
Кермәс микән төшеңә?

Ак куянның ак баласын
Атып аласым килә.
Сезнең кеүек туганнарны
Сатып аласым килә.

Ак куянның ак баласын
Тотсам — үтермәс ием.
Сиксән сигез җәшкә җитсәм,
Әй! Тукта әле…
Туксан тугыз җәшкә җитсәм,
Үлсәм — үкенмәс ием.
```
