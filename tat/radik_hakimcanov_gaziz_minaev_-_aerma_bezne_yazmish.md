# Оригинал

### Радик Хәкимҗанов / Газиз Минаев - Аерма безне язмыш!

```
Көзнең салкын әче җиле
Үтә бит үзәгемә.
Япа-ялгыз торып калдың
Сагышлар үзәгендә.

Көннәрем үтә газаплы
Язларны көтеп кенә.
Куандыр мине, шатландыр
Тик китмә үтеп кенә.

Ялгызлыкны сөйми тормыш,
Яшәү яме — парлылар!
Адашканнар байтак әле
Шуңа микән зарлылар?

Көннәрем үтә газаплы
Язларны көтеп кенә.
Куандыр мине, шатландыр
Тик китмә үтеп кенә.

Кичәге ярсу йөрәкләр —
Моң-зарга калса ялгыш:
Бәхеткәйләр өзелмәсен,
Аерма безне язмыш!

Көннәрем үтә газаплы
Язларны көтеп кенә.
Куандыр мине, шатландыр
Тик китмә үтеп кенә.
```
