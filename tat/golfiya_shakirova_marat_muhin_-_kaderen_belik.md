# Оригинал

### Гөлфия Шакирова / Марат Мухин - Кадерен белик

```
Әй, матур безнең дөньябыз,
Гөрләп бара көнебез,
Серле тылсымнарга тулы
Ай-йолдызлы төнебез.

Кадерен белик мизгелләрнең,
Кадерен белик гомернең,
Ходай язмышыңа язган
Синең бердәнбереңнең.

Олы рәхмәт сүзен әйтик
Дусларга, туганнарга,
Яратып, кадерләп торган
Яныңда булганнарга.

Кадерен белик мизгелләрнең,
Кадерен белик гомернең,
Ходай язмышыңа язган
Синең бердәнбереңнең.

Мәхаббәт дигән олы сүз
Булсын һәрчак күңелдә.
Бар(ы)сы да яхшы булыр дип
Әйтәм менә бүген дә.

Кадерен белик мизгелләрнең,
Кадерен белик гомернең,
Ходай язмышыңа язган
Синең бердәнбереңнең.
```
