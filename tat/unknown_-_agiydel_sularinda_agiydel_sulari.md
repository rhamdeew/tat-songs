# Оригинал

### Мәдинә Хуҗагалиева / Илшат Фатыйхов - Агыйдел суларында (Агыйдел сулары)

```
Агыйделнең суларында
Эх, бер йөзәсе иде.
Яна йөрәк, иркәм ерак —
Ничек түҙәргә инде?

Агыйделнең суларында
Пар аккошлар кагына.
Миндә очып барыр идем
Иркәм синең яныңа.

Агыйделнең суларында
Чумып аласы иде.
Сиңа тиң бүләк булырдай
Мәрҗән табасы иде.

Агыйделнең суларында
Пар аккошлар кагына.
Миндә очып барыр идем
Иркәм синең яныңа.

Агыйделнең суларында
Салда агасы иде.
Бәхетемә, бергенәмә
Йөзеп барасы иде.

Агыйделнең суларында
Пар аккошлар кагына.
Миндә очып барыр идем
Иркәм синең яныңа.
```
