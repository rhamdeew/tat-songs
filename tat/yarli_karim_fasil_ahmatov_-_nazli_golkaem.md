# Оригинал

### Ярлы Кәрим / Фасил Әхмәтов - Назлы гөлкәем

```
Сөям, бел, сине,
Сөям гел сине.
Моң-зарларымнан
Көям шул инде.
Моң-зарларымнан
Көям шул инде;
Гөлкәем, бәгырем,
Нишлим соң инде?!

Иртә мин уйлыйм,
Кич тә мин уйлыйм.
Күрсәм үзеңне,
Һич тә мин туймыйм.
Күрсәм үзеңне,
Һич тә мин туймыйм.
Иркә баладай,
Килче, кил, уйныйм!

Килче, кил, бәгырем!
Кил, кил, серләшик;
Ал чәчәкләрдән
Илһам эзләшик,
Ал чәчәкләрдән
Илһам эзләшик;
Былбыллар кебек
Гөр-гөр гөрләшик…
```
