# Оригинал

### Марс Нуриев / Рамил Гыймрани - Акланнарга кил!

```
Йөгереп кенә кердем акланарга,
Күк чәчәктән таҗлар үрергә.
Чәчәкләрдән гүзәл җанкаемны
Языр, бәлки, шунда күрергә.

Язган икән, миңа язган икән
Бер очрашып сине күрергә.
"Гомерлеккә", — диеп бәхет юрап,
Чәчләреңә таҗлар үрергә.

Очрадың син миңа акланнарда,
Иңнәремә салып зәңгәр шәл.
Рәхмәт язмышыма — очраштырды,
Гомер мизгелләрен көткән мәл.
```
