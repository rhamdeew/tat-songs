# Оригинал

### Гыйз-әл Габид / Марат Галимов - Җәйге җилләр

```
Җәйге җилләр, ирекле сез,
Көньякларга очыгыз,
Иркәмә сәлам әйтегез,
Зифа билен кочыгыз.

Пышылдагыз колагына,
Шомырт чәчләрен сыйпап:
«Дускаең сине сагына…»

Таң атканда өмет өзмә,
Чыгармы кояш диеп, —
Кайтырмын, бәлки, көзгә,
Дөньяларны яулап.

Пышылдагыз колагына,
Шомырт чәчләрен сыйпап:
«Дускаең сине сагына…»
```
