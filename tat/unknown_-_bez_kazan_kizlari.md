# Оригинал

### Гөлшат Зәйнәшева / Сара Садыйкова - Без — Казан кызлары

```
Идел буенда үстек без,
Идел суын эчтек без.
Идел камышлары кебек
Зифа булып үстек без.

Безнең Казан матур кала
Ямьле Идел буенда.
Егетләр безне күзлиләр
Язгы сабан туенда.

Мәскәүдән дә, Өфедән дә
Яучылар килә безгә.
Кияү егетләре йөри
Әйләнә-тирәбездә.

Мәскәү, Өфе егетләре
Күп килеп йөрмәсеннәр.
Яңа бистә егетләре
Көнләшә күрмәсеннәр.

Безнең Казанның кызлары
Нечкә билле һәм чибәр.
Кызлар нечкә билле була
Эшләп үскәнгә диләр.

Безнең Казанның кызлары
Тегәләр кама бүрек.
Егетләр хәйран калалар
Безнең бүрекне күреп.

Без биесәк, сәхнәләрне
Аккош кебек йөзәбез.
Без җырласак моңлы көйләр,
Үзәкләрне өзәбез.

Без Казан кызлары шулай
Яшибез горурланып.
Яшибез без горурланып,
Көннән-көн матурланып.
```
