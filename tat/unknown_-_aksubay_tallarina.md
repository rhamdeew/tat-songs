# Оригинал

### Газиз Кашапов / Арслан Батыршин - Аксубай талларына

```
Инеш буе яшел чирәм,
Яши күз алларымда.
Кайткан саен сәлам бирәм
Аксубай талларына.

Сокланып һич тә туялмыйм
Туган як таңнарына.
Нигә кош булып куналмыйм
Аксубай талларына.

Кайтып киләм туган якка,
Озын юл алларымда.
Баш иям чыгып юллардан
Аксубай талларына.
```
