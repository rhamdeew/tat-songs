# Оригинал

### Рамил Янбеков / Ришат Сәгыйтов - Кыңгырау чәчәкләрне өзмәгез

```
Кыңгырау чәчәкләрне өзмәгез,
Болыннардан әгәр үтсәгез.
Алар сезгә бәхет теләрләр,
Башларыннан гына сөйсәгез.

Кыңгырау чәчәкләргә багыгыз,
Күзләрегез тулы наз белән.
Алар сезне сагынып көтәрләр,
Чәчәк атып иртә яз белән.

Кыңгырау чәчәкләрнең төсләре,
Зәңгәр күктә, тымык күлләр дә.
Үзәгемне өзә торганы,
Без яртакан зәңгәр күзләр дә.
```
