# Оригинал

### Эльмира Җәлилова / Фирзәр Мортазин - Яратып кавышканнар

```
Өзелеп яратканнар күп,
Бар соңлап табышканнар.
Бик бәхетле бу дөньяда
Яратып кавышканнар.

Мәхәббәтле йортта гына
Яшәве рәхәт була.
Яратудан гына җиргә
Бәхетле бала туа.

Һәр гаилә үзенчә яши —
Һәр өй эче тулы сер.
Яратып яшәсәң дөнья
Матурырак барыбер.

Мәхәббәтле йортта гына
Яшәве рәхәт була.
Яратудан гына җиргә
Бәхетле бала туа.

Сакласын Ходай канатны
Каерыр ялгышлардан.
Мәңге бергә яшәсеннәр
Яратып кавышканнар.

Мәхәббәтле йортта гына
Яшәве рәхәт була.
Яратудан гына җиргә
Бәхетле бала туа.
```
