# Оригинал

### Гөлфия Юнысова / Эльвира Хаматнурова - Башкортостан кызлыры

```
Еребеҙ бай йылға-күлгә,
Тәмле шишмә һыуҙарыбыҙ
Шишмә һыуҙар эсеп үҫкән
Сафлыҡтары йәнгә күскән
Башҡортостан ҡыҙҙары беҙ.

Еребеҙ бай урман-тауға,
Йомарт ялан-ҡырҙарыбыҙ.
Урман-тауҙай серле, ғорур,
Ҡүңелендә һөйоу булыр
Башҡортостан ҡыҙҙары беҙ.

Еребеҙ бай хазинаға
Ғамле хыял-уйҙарыбыҙ.
Алтын ҡуллы, заман моңло
Йөрәгендә наҙҙар тулы
Башҡортостан ҡыҙҙары беҙ.

Еребеҙ бай; рухлы халкым,
Дәртле,моңло йырҙарыбыҙ.
Усаҡ яғыр, бала бағыр,
Ил язмышын иңгә һалыр
Башҡортостан ҡыҙҙары беҙ.
```
