# Оригинал

### Гарифҗан Мөхәммәтшин / Азат Хөсәенов - Ак җыр (беренче вариант)

```
Бар дөнья ак кына,
Күңелләр шат кына,
Коела, коела ап-ак кар;
Ак уйнак буранда
Без икәү урамда —
Бу ак төн гомерлек бер ядкарь.

Яңа ел кичендә
Ак карлар эчендә
Адаштык, ахры, без адаштык;
Сихри ай нурында
Мәхәббәт турында
Иң изге вәгъдәләр алмаштык.

Яңа ел кичләре
Керсез саф хисләрне
Гомергә бер җепкә бәйләде;
Коела ап-ак кар,
Бу ак төн — ак ядкарь,
Каеннар шаһите вәгъдәнең.

Яңа ел кичендә
Ак карлар эчендә
Адаштык, ахры, без адаштык;
Сихри ай нурында
Мәхәббәт турында
Иң изге вәгъдәләр алмаштык.
```
