# Оригинал

### Мәүлетбай Ямалетдинов / Газиз Дәүләтбирдин - Күңелемне биләп алдың

```
Безнең ара йолдыз кебек бик ерак,
Синнән хәтта сәламләр дә бик сирәк.

Күңелемне биләп алдың тагы да,
Саф мәхәббәт газаплардан ярала.

Син үзең дә әкияттәге батырым,
Күкнең түгел, җирнең якты йолдызы.

Күңелемне биләп алдың тагы да,
Һәр бер нурда синең төсең чагыла.

Хыялымның күкләрендә балкыйсың,
Сиңа булсын йөрәгемнең ялкыны.

Күңелемне биләп алдың тагы да,
Саф мәхәббәт адашса да, табыла.
```
