# Оригинал

### Галия Арсланова / Ризида Мөхәммәтҗанова - Авылым (икенче вариант)

```
Менә тагын сиңа кайттым, авылым,
Гади генә кызың хыялый.
Шатлыгыннан шашып тибә йөрәк,
Күкрәк читлегенә сыялмый.

Кушымта:
Сагындым сине, авылым,
Ибраһимым, Ибраһим.
Җырларымда җырламасам,
Кичермәстер илаһым.

Көтәләрдер мине күл буйлары
Бакалары туктап җырыннан.
Кыр үрдәкләре дә очмагандыр,
Канат кагып өем кырыннан.

Кушымта.

Менә тагын йортым, тимер капка,
Бөдрә таллар япкан читәнне.
Сирәк кайтасың, дип, гүя япкан
Әткәй тоткан агач киртәне.

Кушымта.

Менә тагын синдә, туган йортым,
Исәнмесез, күрше — күләнем.
Матур җырлар сездә генә туа,
Җырым булсын сезгә бүләгем.

Кушымта.
```
