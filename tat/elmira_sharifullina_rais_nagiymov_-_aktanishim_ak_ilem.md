# Оригинал

### Эльмира Шәрифуллина / Рәис Нәгыймов - Актанышым — ак илем

```
Актанышым — ак илем,
Ак илемдә — ак көнем.
Илдө-көндә, тирә-юньдә
Шөһрәт казанган җирем.

Актанышым — ак илем.
Канат чыгарган җирем.
Еллар буйлап күп йөредем,
Тапмадым сиңа тиңен.

Басуларың, кырларың —
Бәрәкәт иңгән җирем.
Болындагы чәчәкләрең
Күңелдә таба тиңен.

Актанышым — ак илем,
Ак илемдә — ак көнем.
Мәрхәмәтле кочагыңда
Оятлы булмас йөзем.
```
