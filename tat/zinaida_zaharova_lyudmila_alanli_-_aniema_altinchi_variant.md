# Оригинал

### Зинаида Захарова / Людмила Аланлы - Әниемә (алтынчы вариант)

```
Табыныңнан ризык өзелмәсен,
Тәмле булсын эчкән чәйләрең.
Тыныч булсын, әни, күңелләрең,
Җылы гына булсын җәйләрең.

Үтәсе юлларың булсын озын,
Алтын гына булсын гына гомер көзең.
“Гел бәхетле яшә, әни!”, — диеп,
Балаларың теләк тели бүген.

Авырлыклар сине урап узсын,
Яшькә чыланмасын күзләрең.
Беркайчан да онытмабыз, әни,
Безгә әйткән назлы сүзләрең.

Үтәсе юлларың булсын озын,
Алтын гына булсын гына гомер көзең.
“Гел бәхетле яшә, әни!”, — диеп,
Балаларың теләк тели бүген.
```
