# Оригинал

### Гөлнара Рәшитова / Ленар Бикмурзин - Оча казлар

```
Исә әче, җилләр исә,
Кыш киләсе сиздереп.
Каңгылдашып казлар китә
Үзәкләрен өздереп.

Оча казлар, оча казлар,
Очалар еракларга.
Бүген мында башкайлары
Иртәгә кай якларда?

Исә җилләр, җилләр исә,
Исәләр бер файдасыз.
Көзләр килгәч, китәсездә,
Яз сагынып кайтырсыз.

Оча казлар, оча казлар,
Очалар еракларга.
Бүген мында башкайлары
Иртәгә кай якларда?

Кошны түгел, кешене дә,
Аямый язмыш җиле.
Туган яктан аерылгач
Сагышландырмый кемне?

Оча казлар, оча казлар,
Очалар еракларга.
Бүген мында башкайлары
Иртәгә кай якларда?
```
