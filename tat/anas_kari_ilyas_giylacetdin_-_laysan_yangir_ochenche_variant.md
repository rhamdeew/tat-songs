# Оригинал

### Әнәс Кари / Ильяс Гыйләҗетдин - Ләйсән яңгыр (өченче вариант)

```
Яз иртәсен котлагандай,
Матур булып
Зәңгәр күкне каплап алды
Аксыл болыт.
Бизәкләнгән төсле булды
Бөтен һава.
Кар өстенә шыбыр-шыбыр
Яңгыр ява.

Кушымта:
Ләйсән, ләйсән,
Иң беренче яңгыр.
Ләйсән, күңелләрне
Син куандыр…
Ләйсән, ләйсән,
Язгы җылы яңгыр,
Ләйсән, ләйсән,
Ләйсән, ләйсән!

Җир өстендә яткан карлар
Эреп бетсен.
Басуларда эш кайныйсы
Көннәр җитсен.
Без чыгарбыз бакчаларга
Ул көннәрдә.
Яшелчәләр утыртырбыз
Түтәлләргә.
```
