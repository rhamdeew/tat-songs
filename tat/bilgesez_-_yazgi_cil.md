# Оригинал

### Билгесез - Язгы җил

```
Тал бөреләре уяна,
Сандугачын күр әле.
Яшь йөрәкләрдә кузгала
Ярлар сөю теләге.

Көзен япкан тәрәзәмне
Язгы җил ачты инде.
Йөрәкләрдә язгы сагыш,
Йокылар качты инде.

Сыерчык бураны чыга,
Күбәләктәй күбәлә.
Кояш күзен кысып тора,
Җир читенә чүгәләп.

Көзен япкан тәрәзәмне
Язгы җил ачты инде.
Йөрәкләрдә язгы сагыш,
Йокылар качты инде.

Язлар җитсә — су кузгала,
Ташу китсә — ком кала.
Карлар китсә — бозы кала,
Ярлар китсә — моңы кала.

Көзен япкан тәрәзәмне
Язгы җил ачты инде.
Йөрәкләрдә язгы сагыш,
Йокылар качты инде.
```
