# Оригинал

### Сафуан Әлибай / Роберт Газизов - Рәсимә

```
Синең кайнар хисләреңнән,
Таң бизәгән чәчләреңнән
Алтын нурлар чәчелә,
Алтын нурлар чәчелә.

Син елмайсаң, бозлар эрер,
Җанга ярып язлар керер.
Кояш кызы Рәсимә,
Кояш кызы Рәсимә.

Әллә ниләр бирер идем,
Хисләремне үрер идем.
Нур чәчелгән чәчеңә,
Нур чәчелгән чәчеңә.

Таң юлыннан син үтәсең
Җиркәеңне гөл итәсең,
Кояш кызы Рәсимә,
Кояш кызы Рәсимә.

Күңелемә борчу салып,
Йә елмаеп, йә югалып,
Болытларга яшенмә,
Болытларга яшенмә.

Күзләремне камаштырма,
Йөрәгемне адаштырма,
Кояш кызы Рәсимә,
Кояш кызы Рәсимә.
```
