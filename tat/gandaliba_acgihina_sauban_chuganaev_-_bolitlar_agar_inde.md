# Оригинал

### Гандәлибә Аҗгихина / Сәүбән Чуганаев - Болытлар агар инде

```
Кышларымда узып бара,
Тагы бер язым җитә.
Көтүләрем бушка булгач,
Гомерем синсез үтә.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.

Салкын томаннар сарыла
Иртәләрем талларга.
Ходаем, түземлекләр бир
Ялгыз калган ярларга.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.

Сөенәм дә, көенәм дә,
Язларым узып бара.
Булмадың бәхет йолдзым,
Булдың гомерлек яра.

Болытлар агар инде,
Тамчылар тамар инде.
Янарга калган йөрәгем,
Янар да янар инде.
```
