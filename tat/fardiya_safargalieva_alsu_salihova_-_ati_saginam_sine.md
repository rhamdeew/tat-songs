# Оригинал

### Фәрдия Сәфәргалиева / Алсу Салихова - Әти, сагынам сине

```
Төннәргә әйләнгән көннәремдә
Исән чагың уйлап юанам.
Әти, әти, синең елмаюны
Төшләремдә күреп уянам.

Җир дә, күк тә башкачарак иде,
Нурлы иде, син исән чакта.
Сине сагынам, бик юксынам, әти,
Җырларым да гел синең хакта.

Дөнья тынды, төсләр дә югалды,
Су да акмый кебек елгада.
Сине эзлим, сине сагынам, әти,
Авыр синсез яшәү дөньяда.
```
