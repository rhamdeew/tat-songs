# Оригинал

### Әхмәт Саттар / Наилә Фатехова - Рамазан (өченче вариант)

```
Рамазанда, Рамазанда уразага керербез.
Рамазанда, Рамазанда туганнарны күрербез.
Рамазанда, Рамазанда зур өстәлләр корырбыз.
Рамазанда, Рамазанда кунаклар чакырырбыз.

Рамазан, Рамазан — шифалы ай Рамазан.
Шифалы ай Рамазанда гафу сорыйк Ходайдан.

Аллаһу әкбәр, Аллаһу әкбәр,
Ләә иләәһә илләллаһү, Аллаһу әкбәр,
Аллаһу әкбәр вә лилләәһил хәмд.

Рамазанда, Рамазанда акланып пакьланыгыз,
Ходайның изге аенда гөнаһтан сакланыгыз.
Рамазанда, Рамазанда артсын иде бәхетләр.
Рамазанда, Рамазанда дога көтә шәһитләр.
```
