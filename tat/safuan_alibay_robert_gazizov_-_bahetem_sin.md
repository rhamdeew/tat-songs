# Оригинал

### Сафуан Әлибай / Роберт Газизов - Бәхетем — син

```
Иркә нурлар күңелемдә
Уйный бүген, уйный.
Язлар яна йөрәгемдә
Кояш булып елмай.

Бәхетем — син,
Язым — син,
Мәхәббәтем, назым — син.

Язлар җиткәч, җәй дә килер,
Күктә калкыр болыт.
Аяз кояш көйдергәндә,
Яу син ләйсән болыт.

Бәхетем — син,
Җәем — син,
Күкрәгемдә җаным — син.

Карлар бурап, кыш та килер,
Безгә саллар сибәр.
Тик йөрәгем ул чакта да
Сөю булып тибәр.

Бәхетем — син,
Кышым — син,
Ташлап китмәс кошым — син.
Бәхетем — син,
Җырым — син,
Мәхәббтем, нурым — син.
```
