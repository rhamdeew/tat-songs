# Оригинал

### Родион Мортазин - Кайт инде, бала чагым

```
Су буйларын ялгыз урадым
Бала чагым кайт дип юрадым.
Туган якта калган бит эзем,
Син минем төшләремә кердең.

Кайт инде, бала чагым минем,
Мин бит сине бик нык сагындым.
Сагышларга түзә алмичә
Авылыма әйләнеп кайттым.

Су буйлары бөдрә тал икән
Талларымда сагынган икән.
Сәлам биреп башын ияләр
Алар да бит кечкәй иделәр.

Кайт инде, бала чагым минем,
Төшләремдә генә булса да.
Күбәләкләр куып уйнармын
Чәчләремә кырау кунса да.

Су буенда акчарлак чыңы
Балачакка әйдидер тагы.
Аның тавышы йөрәктә ята
Авылма җанымны тарта.

Кайт инде, бала чагым минем,
Мин бит сине бик нык сагындым.
Мин бит сине өзлеп сагындым
Авылыма әйләнеп кайтым.
Сагышларга түзә алмичә
Авылыма әйләнеп кайттым.
```
