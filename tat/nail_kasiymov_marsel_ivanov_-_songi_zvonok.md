# Оригинал

### Наил Касыймов / Марсель Иванов - Соңгы звонок

```
Еллар шавы баса алмый
Соңгы звонок тавышын.
Һәр көн сине күрә алу
Бәхет иде, классташым.

Соңгы дәрес, соңгы звонок,
Ник бераз соңламадың,
Сүзсез генә сөюемне
Күземнән аңламадың.

Көн дә иртән сине көттем
Алдан килеп дәрескә,
Юксына ук башлый идем
Мин сине тәнәфестә.

Соңгы дәрес, соңгы звонок,
Ник бераз соңламадың,
Сүзсез генә сөюемне
Күземнән аңламадың.

Озын тәнәфес шикелле
Аерылган елларың.
Башланылган яңа дәрес
Килерсең күк елмаеп.

Соңгы дәрес, соңгы звонок,
Ник бераз соңламадың,
Сүзсез генә сөюемне
Күземнән аңламадың.
```
