# Оригинал

### Гөлия / 16quyan - Себер тракты

```
Киртәләргә карап
Үтә елларым.
Авыр һава сулап
Китә җаннарым.

Юллардагы утлар
Яктырталар мине.
Елап беткән еллар
Чагылалар инде.

Күктә яна ай.
Янында йолдыз.
Нигә соң болай
Миңа күңелсез?

Көчле бураннар
Томалый юлны.
Башымда шул бер
Борчый торган тынгы.

Буран тик үземә
Юлларда тыныч,
Тик күңелемдә
Бар бер аяныч.

Күктә яна ай.
Янында йолдыз.
Нигә соң болай
Миңа күңелсез?
```
