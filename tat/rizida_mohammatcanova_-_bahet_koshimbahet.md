# Оригинал

### Ризида Мөхәммәтҗанова - Бәхет кошым(Бәхет)

```
Бәхет кошым якын, сизәм,
Тик нигә килми һаман?
Иңемә кун, күп көттермә,
Арыдым көтеп тәмам.

Бәхет, шатлык өмет итеп,
Яши һәр кеше шулай.
Киләчәктән назлар көтеп,
Үзенә бәхет юрый.

Бәхет көтеп,чын бәхетне
Югалтмыйбызмы шулай?..
Кинәт очып килде нидер, —
Бәхетем кошы бугай.
```
