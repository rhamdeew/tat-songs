# Оригинал

### Равил Гозәеров / Галимҗан Тамендаров - Ашкынам мин туган ягыма

```
Ничә айлар, ничә еллар
Акшынам туган илгә.
Белмим, нигә язмышларым
Кайтармый туган җиргә.

Кайтсам иде үз ягыма
Алсу таңнар атканда.
Кошлар моңын тыңлар идем
Тугай буйлап кайтканда.

Ялан-аяк йөрер идем
Чирәмгә чык төшкәндә.
Йөрәкләрем янмас иде
Салкын суын эчкәндә.

Кошлар да бит җылы яктан
Әйләнеп кире кайта.
Кошлар булып, канат кагып
Кайтыйк без туган якка.
```
