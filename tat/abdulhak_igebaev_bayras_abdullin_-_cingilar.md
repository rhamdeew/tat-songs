# Оригинал

### Абдулхак Игебаев / Байрас Абдуллин - Җиңгиләр

```
Сөйкемле дә, гаҗәп шәфкатьле дә,
Безнең яклардагы җиңгиләр.
Юкка мени алар инсафлыгын
Алан гөлләренә тиңлиләр.

Ихласлыгы белән ямьне арбый
Безнең яклардагы җиңгиләр.
Киңәш сорап килсәң һәрчак алар
Яхшылыкка гына тиңлиләр.

Хәстәрле дә алар, җигәрле дә,
Һич кенә дә тыңгы бирмиләр.
Абыйларым дәртле булгангамы?
Яшь йөрәкле безнең җиңгиләр.
```
