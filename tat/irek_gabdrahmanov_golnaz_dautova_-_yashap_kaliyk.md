# Оригинал

### Ирек Габдрахманов / Гөлназ Даутова - Яшәп калыйк!

```
Ходай безне сынаса да,
Язмыш безне кыйнаса да,
Сагышка юл куймабыз!
Без бит җиргә яшәр өчен,
Яшеннәрдәй яшьнәр өчен,
Җырлар өчен туганбыз.

Яшәр чакта яшәп калыйк,
Һәр минуттан илһам алыйк,
Гомерләр бит бер генә!
Узган гомер елларының,
Узган гомер юлларының,
Кадерләрен бел генә!

Язмышларга каршы барыйк,
Кайгыларга башны салмыйк,
Бирешмичә җилләргә!
Тап елмаеп хәл белишик,
Шатлыкны уртак бүлешик,
Иң бәхетле көннәрдә.

Яшәр чакта яшәп калыйк,
Һәр минуттан илһам алыйк,
Гомерләр бит бер генә!
Узган гомер елларының,
Узган гомер юлларының,
Кадерләрен бел генә!

Бер Аллаһка шөкер кылып,
Изге догаларны укып,
Гөрләшип яшик әле!
Үткәннәргә кайтып булсас,
Узган гомер кире кайтмас,
Җырлашып яшик әле!

Яшәр чакта яшәп калыйк,
Һәр минуттан илһам алыйк,
Гомерләр бит бер генә!
Узган гомер елларының,
Узган гомер юлларының,
Кадерләрен бел генә!
```
