# Оригинал

### Роберт Шәймәрданов - Без бәхетле

```
Сөясеңме дип сорамыйм,
Яратканыңны беләм.
Йөрәгеңне дә тыңламыйм
Мин диеп яна — сизәм.

Синең өчен генә яшим,
Кирәк түгел башкасы.
Бары сиңа,бары сиңа
Хисләремнең бакчасы.

Көтәсеңме дип сорамыйм
Юлларда синең күзең.
Читләрдә озак йөрсәм дә,
Көтәсең сабыр, түзем.

Синең өчен генә яшим,
Кирәк түгел башкасы.
Бары сиңа,бары сиңа
Хисләремнең бакчасы.

Артык сүзләр кирәк түгел,
Без бәхетле яшибез.
Бездән дә яхшы яшәгез!
Беркемнән көнләшмибез.

Синең өчен генә яшим,
Кирәк түгел башкасы.
Бары сиңа,бары сиңа
Хисләремнең бакчасы.
```
