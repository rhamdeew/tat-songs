# Оригинал

### Мансур Шиһапов / Мәсгуть Имашев - Кайтыр идем

```
Кайтыр идем туган авылыма,
Күрер идем моңсу ярларын.
Аланында сузылып,
Сузылып, кырын ятып
Тыңлар идем камыш шауларын.

Чылтырап аккан чишмәләрнең җырын
Тыңлар идем басып ташына.
Кайтыр идем, авылым,
Туган авылым, сиңа,
Кайтыр өчен йөрәк ашкына.

Кайтыр идем туган авылыма,
Буйлар идем елга буйларын.
Бик сагындым, авылым,
Туган авылым, сине,
Сиңа кайта һаман уйларым.
```
