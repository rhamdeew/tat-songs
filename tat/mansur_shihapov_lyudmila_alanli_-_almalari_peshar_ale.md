# Оригинал

### Мансур Шиһапов / Людмила Аланлы - Алмалары пешәр әле

```
Алмалары пешәр әле,
Җыеп алырсың әле.
Хәзер сагынмыйм, дисәң дә,
Бер сагынырсың әле.

Алмалары пешәр әле,
Җыеп сакларсың әле.
Алмагачлы юллар буйлап
Бергә атларбыз әле.

Алмалары пешәр әле,
Җыеп алырсың әле.
Алма кебек матур чаклар
Исеңә төшәр әле.

Алмалары пешәр әле,
Җыеп сакларсың әле.
Алмагачлы юллар буйлап
Бергә атларбыз әле.
```
