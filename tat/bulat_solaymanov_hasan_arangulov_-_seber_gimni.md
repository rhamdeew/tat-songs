# Оригинал

### Булат Сөләйманов / Хәсән Арангулов - Себер гимны

```
Син тугансың
Минем йөрәгемдә,
Мин туганчы
Газиз анамнан.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.

Җаным кебек
Бик кадерле миңа
Туган җирнең
Изге туфрагы.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.

Бу дөньяда
Себер агачының
Мин серле
Моңлы яфрагы.

Себер, Себер,
Бер син генә җирдә
Минем мәхәббәтем
Казанган.
```
