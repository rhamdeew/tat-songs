# Оригинал

### Динә Талхина / Салават Низаметдинов - Эргәмдә булсаң икән

```
Кар бизәге булып кына
Учыңа кунар идем.
Күзләреңнән явын алып,
Бер ялтлап алыр идем!
Күз яшеңә әйләнәр дә
Учыңа калыр идем.

Ак яңгырлар булып кына
Чикәңнән сөер идем.
Җәйләреңнең яме булып,
Бер балкып алыр идем!
Күз яшеңә әйләнәр дә
Учыңа калыр идем.

Ак томаннар булып кына
Кочаклап алыр идем.
Бер яндырып, бур туңдырып,
Утларга салыр идем!
Күз яшеңә әйләнәр дә
Учыңа калыр идем.
```
