# Оригинал

### Наҗар Нәҗми / Заһир Исмәгыйлев - Мендем биек Урал тау башына

```
Мендем биек Урал тау башына,
Бөркет тора кыя ташында.
Баш очымда балкып яна кояш,
Болыт ята аяк астында.

Мында җилләр, урман-елгалар да,
Бересен-берсе килеп табыша.
Бу җирләрдә йолдыз белән йолдыз,
Таулар белән таулар кавыша.

Мендем биек Урал тау башына,
Җилләр тауга сөйли серләрен.
Шушы инде җил-давыллар кичкән
Минем туып-үскән җирләрем.
```
