# Оригинал

### Рәис Гыймадиев / Виталий Агапов - Онытты дип уйлама

```
Асыл кошлар кунып сайрый
Яр буе талларына.
Күзем йомсам, бергә чаклар
Килә күз алларыма.
Көн артыннан көннәр үтә,
Сагынып көткән чагым.
Кавышыр көннәр килерме?
Әйт әле, зифа талым.

Тамчы-тамчы яңгыр тама,
Тал кояшын юксына.
Юллар чатында булмасам,
Оныткан, дип уйлама.

Йомшак җилләр сыйпап исте
Тал яфрагын таңнарда.
Гомер юлдашым булырдай
Яр, кайда син, кайларда?
Томырылып яңгыр килде
Син кайтасы яклардан.
Кайтуыңны сизгән икән —
Тал юкка шауламаган.

Тамчы-тамчы яңгыр тама,
Тал кояшын юксына.
Юллар чатында булмасам,
Оныткан, дип уйлама.
```
