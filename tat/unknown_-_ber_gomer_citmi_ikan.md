# Оригинал

### Әсхәр Кәшфуллин / Фәнзил Шәрипов - Бер гомер җитми икән

```
Яшьлектәге гыйшык уты
Күңелдән китми икән.
Сөеп-сөелеп яшәргә
Бер гомер җитми икән.

Онытлыса да кайгыларың,
Ярасы бетми икән.
Бу тормышны агартырга
Бер гомер җитми икән.

Максатына җитсен бу дип,
Минутлар көтми икән.
Яшәешеңне аңларга
Бер гомер җитми икән.

Онытлыса да кайгыларың,
Ярасы бетми икән.
Бу тормышны агартырга
Бер гомер җитми икән.
```
