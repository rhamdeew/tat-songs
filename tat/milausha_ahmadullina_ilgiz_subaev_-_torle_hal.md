# Оригинал

### Миләүшә Әхмәдуллина / Илгиз Субаев - Төрле хәл

```
Төрле хәлләр була… дөнья бит бу,
Агы була… була каралары…
Сөрлегүләр, егылулар аша
Тән дә ала канлы яраларны…

Тешләр кысып… ирен тешләп кенә
Мөлдер күзләр ары төбәлер…
Еллар аша онытып барлыкларны
Күңел җәрәхәттән төзәлер…

Яраларга бәйләп җан җылысын,
Җитәкләшеп ары атларга-
Хәлеңдәме… мәърур очкан чакта
Гомер сукмакларын сакларга.

Якты итеп… үтә нурлы итеп…
Гелән генә сүнмәс хыялларда.
Үрергәме… барыр юлларыбыз-
Гөмбәзләрдән биек кыяларда…

Ирен тешләп… тешләр кысып кына
Канлы җәрәхәтләр терелерме…
Дөнья бит бу… төрле хәлләр була,
Сукмаклар да гөлгә күмелерме…
```
