# Оригинал

### Шәйхи Маннур / Әнвәр Бакиров - Ана җыры (беренче вариант)

```
Туган җирдән еракларда
Калды синен әтиең,
Калды бәхтен яклаганда
Син үскән бөек илнең.
Үс, улым, син, күз нурым син,
Үс минем куанычка.
Сакчысы бул аналарның,
Таммасын яшьләре аларның,

Яшәсеннәр тынычта!
Теләмибез сугышны без!
Без тынычлык телибез. —
Безнен якты тормышыбыз

Мәңге балкысын! — дибез.
Үс, улым, син, күз нурым син,
Тынычлык сакчысы бул.
Синең бөек бу илеңә,
Синең яшьлек гомереңә
Дошманнар сузмасын кул!
Океан арты дошманары
Ишетсен, белсен бары: —
Хезмәт халкы каршы сугышка!
Булдырмабыз без аны!
Үс, улым, син, күз нурым син,
Тынычлык сакчысы бул.
Бөтен дөнья белән бергә
Каршы торсак без үлемгә,
Без куймабыз аңа юл!
```
