# Оригинал

### Алия Солтан / Лилия Вахитова - Бәхетле булыйк! (икенче вариант)

```
Гөлләр назлы чәчәчәкләре булса
хуш ис сибеп гөлгә үрелә.
Хатын-кызлар шунда бәхетле
бәхеттән күз яше түгелсә.

кушымта
Бәхет телик барыбызга
бәхетле булыйк!
Тормышларны шау чәчкәле
бәхеттән корыйк!

Яннарыңда сөйгән ярың булса
иркәләп сөеп назлап торса.
Без бәхетле кайгы сагышлары
Кагылмайча гел урап узса.

кушымта
Бәхет телик барыбызга
бәхетле булыйк!
Тормышларны шау чәчкәле
бәхеттән корыйк!

Сөю белән атса бит ул таңнар
Яннарыңда булса туганнар
Яшик әле шулай гел рәхәттә.
Язмыш язган матур бәхеттә!

кушымта
Бәхет телик барыбызга
бәхетле булыйк!
Тормышларны шау чәчкәле
бәхеттән корыйк!
```
