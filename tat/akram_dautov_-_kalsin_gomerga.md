# Оригинал

### Әкрәм Даутов - Калсын гомергә

```
Калсын гомергә
Безнең күңелдә
Яшлектән килгән
Дуслык эзләре.
Туган җиремнең
Зәңгәр күлләре,
Күрше кызының
Зәңгәр күзләре.

Эх авыл, авыл,
Үтте җил-давыл,
Җилләр исте дә
Гөлне кистеләр.
Кайгы-нуҗалар
Кагылып уздылар,
Шул авылымнан
Алып киттеләр…

Калсын гомергә
Безнең күңелдә
Яшлектән килгән
Дуслык эзләре.
Туган җиремнең
Тып-тын күлләре.
Үсмер кызларның
Ялкын күзләре,
Йомшак сүзләре.
```
