# Оригинал

### Роберт Миңнуллин / Виталий Агапов - Энем өйрәнә

```
Энем йөрергә өйрәнә,
Тотынмыйча йөрергә.
Хәтта йөрергә дә түгел,
Исәбе — йөгерергә.

Матур гына атлап китә,
Бер атлый да абына.
Абына да торып китә,
Тик егыла тагы да.

Торырга кайчан өлгерә?
Егыла кай арада?
Егылып яткан була ул
Кайчан гына карама.

Көн дә шулай өйрәнә ул
Урамда да, өйдә дә.
Йөрергә өйрәнми ул,
Егылырга өйрәнә.
```
