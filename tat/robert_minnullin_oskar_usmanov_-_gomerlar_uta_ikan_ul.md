# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Гомерләр үтә икән ул

```
Гомерләр үтә икән ул,
Үтә дә китә икән.
Яшьлектәге кайнар хисләр
Бик азга җитә икән!

Гомерләр үтә икән ул,
Яна да бетә икән.
Күңелләр генә һаман да
Нәрсәдер көтә икән!

Гомерләр үтә икән ул,
Бәгырьне өтә икән.
Сөюеңне кем көтәр соң?
Түземе җитәр микән?

Гомерләр үтә икән ул,
Яна да бетә икән.
Күңелләр генә һаман да
Нәрсәдер көтә икән!

Гомерләр үтә икән ул,
Ялкыны бетә икән.
Ашыга-ашыга безне
Кайларга илтә икән?

Гомерләр үтә икән ул,
Яна да бетә икән.
Күңелләр генә һаман да
Нәрсәдер көтә икән!
```
