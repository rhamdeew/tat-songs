# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Күзләреңә куркам карарга

```
Сезгә бар серемне ачам,
Гөлгә тулды: өем — бакча,
Каян җиткерәдер акча, —
Инде белмәссең.
Бер егет йөри яратып,
Сөеп,өзелеп,җаннар атып,
Тик миндәге сөю утын
Сизә күрмәсен.

Юк, яратмыйм мин йөрмә дә янымда.
Ходаем, үзең сакла утыннан.
Чәчәкләреңә хәтта карамыйм да.
Тик күзләреңә карарга куркам.
А-А-А, син шаярма,егет
А-А-А, яратам, диеп.
А-А-А, сөюемне белеп,
А-А-А, өремә көлеп.

Сизмәсен дип хисләремне,
Сирәк чыгам кичләрен дә.
Килсә сөю җиңел генә
Булмас гөмерле.
Сыныйм әле уйларыңны,
Син ашыкма туйларыңа,
Ачуланма тыйганыма
Яме, кадерлем.

Юк, яратмыйм мин йөрмә дә янымда.
Ходаем, үзең сакла утыннан.
Чәчәкләреңә хәтта карамыйм да.
Тик күзләреңә карарга куркам.
А-А-А, син шаярма,егет
А-А-А, яратам, диеп.
А-А-А, сөюемне белеп,
А-А-А, өремә көлеп.
```
