# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Туган як (алтынчы вариант)

```
Чәчкәләр җыйганда
Хәтфәдәй тугайда
Тургайларның җырын тыңладым
Күңелем тулганга,
Көй таныш булганга
Аларга кушылып җырладым

Кадерле туган як,
Сөекле туган як,
Уятасың җырлар күңелемдә.
Кадерле туган як,
Сөекле туган як
Һәрвакыт йөрәгем түрендә.

Хуш исле болонда
Әсәреп уенга
Колыннардай уйнап туймадым.
Сокланып киңлеккә,
Рәхәттә, иректә
Җырлаган күңлемне тыймадым.

Кадерле туган як,
Сөекле туган як,
Уятасың җырлар күңелемдә.
Кадерле туган як,
Сөекле туган як
Һәрвакыт йөрәгем түрендә.

Кайтырмын сагынып,
Кошлардай талпынып,
Китсәм еракларга илемнән.
Һичкайчан җырлың,
Таныш көй-моңнарың
Җуелмас сагышлы күңлемнән.

Кадерле туган як,
Сөекле туган як,
Уятасың җырлар күңелемдә.
Кадерле туган як,
Сөекле туган як
Һәрвакыт йөрәгем түрендә.
```
