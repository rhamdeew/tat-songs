# Оригинал

### Фәнәвис Дәүләтбаев / Ирек Габдрахманов - Ана кайгысы

```
Авыр юллар үтеп туган якка
Кошлар кабат кайта язларда.
Шатланалар туып үскән җиргә
Исән-имин кайту язганга.

Улсыз калган газиз анакайның
Баш июле, моңсу карашы.
Өметсезлек, сагыш баскан җанны —
Кошлар кайта, кайтмый баласы.

Юк анага моннан да зур җәза —
Газиз балаң табутта кайту.
Нәгъләт яусын сугыш ачканнарга!
Авыр кайгы — бала югалту.

Юк ич булган югалткан ананың
Канатлары сынык кош сыман.
Сиздермәскә күпме тырышса да
Килә алмый һаман һушына.

Сүнеп бара соңгы өмет-чаткы,
Төзәлмәслек — йөрәк ярасы.
Нигә шулай язмыш мәрхәмәтсез,
Кошлар кайта, кайтмый баласы?!
```
