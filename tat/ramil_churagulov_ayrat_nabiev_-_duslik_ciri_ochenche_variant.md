# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Дуслык җыры (өченче вариант)

```
Таулар очып, елга кичеп,
Җир чигенә барырмын,
Назга сусаган дусларны
Кочагыма алырмын.

Чын дуслыкка киртә түгел
Биек Урал, Каф-таулар.
Дуслыкларны аералмый
Киң һәм тирән елгалар.
Берләштерә, дуслаштыра
Агыйдел, Сөн елгалары.
Җырлап кына узсын гомер,
Җырлап барсын дөньялар.

Дуслар булсын һәр җирдә дә:
Казанда һәм Уфада,
Самарларда, Чиләбедә,
Мәскәүдә, Сарытауда…

Чын дуслыкка киртә түгел
Биек Урал, Каф-таулар.
Дуслыкларны аералмый
Киң һәм тирән елгалар.
Берләштерә, дуслаштыра
Агыйдел, Сөн елгалары.
Җырлап кына узсын гомер,
Җырлап барсын дөньялар.

Чын дуслыкны еллар, юллар,
Матур җырлар ныгыта.
Чын дуслыкның чылбырлары
Йөрәкләр аша үтә.

Чын дуслыкка киртә түгел
Биек Урал, Каф-таулар.
Дуслыкларны аералмый
Киң һәм тирән елгалар.
Берләштерә, дуслаштыра
Агыйдел, Сөн елгалары.
Җырлап кына узсын гомер,
Җырлап барсын дөньялар.

Йөрәкләрдән йөрәкләргә
Күпер салсын моң-җырлар.
Чын дуслыкка киртә түгел
Киң Иделләр, Каф-таулар.

Чын дуслыкка киртә түгел
Биек Урал, Каф-таулар.
Дуслыкларны аералмый
Киң һәм тирән елгалар.
Берләштерә, дуслаштыра
Агыйдел, Сөн елгалары.
Җырлап кына узсын гомер,
Җырлап барсын дөньялар.
```
