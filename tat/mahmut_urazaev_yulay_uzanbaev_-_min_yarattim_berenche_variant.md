# Оригинал

### Мәхмүт Уразаев / Юлай Үзәнбаев - Мин яраттым (беренче вариант)

```
Мин яраттым сине, яраттым,
Иң саф сөю белән яраттым.
Мин җан аттым, син, дип җан аттым,
Карлыгачым, диеп яраттым.

Мәхәббәт — ул шундый изге хис,
Мәхәббәт — ул шундый изге көч.
Шушы хискә генә кабындым,
Шушы хискә генә талпындым.

Мин яраттым сине, яраттым,
Яратуым белән караттым.
Сиңа гына наздай мин бактым,
Бул гомергә, дидем, канатым.

Мәхәббәт — ул шундый изге хис,
Мәхәббәт — ул шундый изге көч.
Шушы хис бит нуры юлымның,
Сүнмәс уты бит син моңымның.

Мин яраттым сине, яраттым,
Иң саф сөю белән яраттым.
Мин җан аттым, син, дип җан аттым,
Карлыгачым, диеп яраттым.

Мәхәббәт — ул шундый изге хис,
Мәхәббәт — ул шундый изге көч.
Шушы изге хистән сафлансын,
Йөрәк белән йөрәк шатлансын.
Мин яраттым сине, яраттым.
```
