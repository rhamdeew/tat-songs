# Оригинал

### Рамил Чурагулов / Ришат Акъегет - Синең  күзләр

```
Күзләрең якты йолдыздай,
Нурлы кояштай якты.
Синең күзләр йөрәгемдә
Мәхәббәт утын якты.

Күзләреңдә гел-гел генә
Очкыннар уйнап торсын.
Шушы күзләр, нурлы күзләр
Минеке генә булсын.

Мең-мең күзләр арасында
Иң якты күзләр синдә.
Алар минем күңелемдә
Һәм йөрәгем түрендә.

Күзләреңдә гел-гел генә
Очкыннар уйнап торсын.
Шушы күзләр, нурлы күзләр
Минеке генә булсын.
```
