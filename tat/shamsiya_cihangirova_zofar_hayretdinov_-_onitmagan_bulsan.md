# Оригинал

### Шәмсия Җиһангирова / Зөфәр Хәйретдинов - Онытмаган булсаң…

```
Син оныткан булсаң әгәр мине,
Назлы җилләр хәбәр китерер.
Чәчләреңнән сыйпап иркәләр дә,
Кулларымны исеңә төшерер…

Онытмаган булсаң әгәр мине,
Килер булсаң бары кичегеп;
Көтәрмен мин, бәгърем, көтәрмен мин,
Ялгышларның барсын кичереп.

Син оныткан булсаң әгәр мине,
Яз гөлләре хәбәр китерер.
Зәңгәр ялкын булып бөркелер дә,
Күзләремне исеңә төшерер…

Онытмаган булсаң әгәр мине,
Килер булсаң бары кичегеп;
Көтәрмен мин, бәгърем, көтәрмен мин,
Ялгышларның барсын кичереп.

Син оныткан булсаң әгәр мине,
Ак бураннар хәбәр китерер.
Чаллар булып чәчеңә сарылыр да,
Сагышымны исеңә төшерер…

Онытмаган булсаң әгәр мине,
Килер булсаң бары кичегеп;
Көтәрмен мин, бәгърем, көтәрмен мин,
Ялгышларның барсын кичереп.
```
