# Оригинал

### Нәҗип Әсәнбаев / Рим Хәсәнов - Зәйтүнгөл (беренче вариант)

```
Бер үк сукмаклардан йөрегәнбез,
Бер чишмәнең суын эчкәнбез.

Ничек яшәр идек бу дөньяда,
Әгәр икәү бергә булмасак?
Бәхет кошы читләп үтәр иде,
Бер-беребез өчен тумасак!

Бәхет тә бит сайрар кошка охшаш,
Аның да бар ике канаты;
Бер канаты аның — мәхәббәте,
Икенчесе — сайрар бакчасы.
```
