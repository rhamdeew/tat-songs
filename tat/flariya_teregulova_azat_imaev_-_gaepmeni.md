# Оригинал

### Флария Терегулова / Азат Имаев - Гаепмени?

```
Гаепмени оныта алмау,
Бары сине сагыну?
Гаепмени яшьлек ярын
Искә алып талпыну.

Ирек бирәм хисләремә,
Хыялымда булса да.
Сагышлардан коткар мине,
Кер төшкә бер булса да!

Гаепмени өмет дигән
Гүзәл сүзгә табыну?..
Гаепмени, онытылып,
Яшьлегеңне сагыну?
Сөюеңне сагыну?..
```
