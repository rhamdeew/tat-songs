# Оригинал

### Башкорт халык җыры - Ямьле Агыйдел буйлары (өченче вариант)

```
Агыйделкәй алкын, суы салкын
Ак үрдәкләр каршыда йөзә алмый.

Ямьле Агыйдел буйлары шул
Бетерәдер хәсрәтле уйларны.

Әчем генә, тышым, әй ут яна,
Бер җырламый җаным да түзә алмый.

Ямьле Агыйдел буйлары шул
Бетерәдер хәсрәтле уйларны.
```
