# Оригинал

### Алсу Гарипова - Йөрәгемдәге хисләрем

```
Якты кояш нурларында
Синең серле күзләрең
Уйларымнан бер дә китмиләр.
Мәхаббәтемне тоярсың,
Әле хыялларым
Бәхет канатлары өстиләр.

Син аңларсың әле
Күңелемдә булган хисләрне.
Белсәң иде, җаным,
Син дип йөрәгемнең типкәнен.

Мин каршыңа чыгам,
Назлы җилләр миңа әйтерләр,
Сине эзләреңнән табармын.
Кулны кулга алып,
Икәү бергә генә без барыйк,
Үтик, җаным, тормыш юлларын.

Син аңларсың әле
Күңелемдә булган хисләрне.
Белсәң иде, җаным,
Син дип йөрәгемнең типкәнен.
```
