# Оригинал

### Венера Мәхмүтова / Гөлия Миңгалиева - Аңларсыңмы

```
Туар таңда сине очраттым,
Мәңгелеккә диеп яраттым.
Аңларсыңмы минем йөрәгемне,
Йөрәгеңә минем кирәгемне.

Янар кояш сыман нурын сиптең,
Җәй җилләре сыман сыйпап үттең.
Аңларсыңмы минем сүзләремне,
Аңларсыңмы, сиңа гына баккан күзләремне.

Ак болытлар сыман килдең, киттең,
Күңел хисләремне таптап үттең.
Аңларсыңмы минем сөйгәнемне,
"Син — бәхетем минем" дигәнемне.

Янар кояш сыман нурын сиптең,
Җәй җилләре сыман сыйпап үттең.
Аңларсыңмы минем сүзләремне,
Аңларсыңмы, сиңа гына баккан күзләремне.

Соң булганын хәзер аңларсыңмы,
Өзелгәнне әллә ялгарсыңмы.
Аңларсыңмы минем уйларымны,
Кулга алсаң әгәр кулларымны.

Янар кояш сыман нурын сиптең,
Җәй җилләре сыман сыйпап үттең.
Аңларсыңмы минем сүзләремне,
Аңларсыңмы, сиңа гына баккан күзләремне.
```
