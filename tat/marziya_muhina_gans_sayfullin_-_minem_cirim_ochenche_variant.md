# Оригинал

### Марзыя Мухина / Ганс Сәйфуллин - Минем җырым (өченче вариант)

```
Әткәм-әнкәмнең җылысы —
Күңел күзем яктысы.
Сезгә генә күренәдер
Ак биләүнең ак төсе.

Барысын да бүлеп салып,
Языладыр язмышлар.
Яратып сынавымы әллә?
Язмышымда ялгыш бар.

Болындагы чәчкәләрнең
Исләрен иснәр идем.
Җырларым бар, моңнарым бар,
Булмаса нишләр идем.

Әткәм-әнкәмнең җимеше,
Татлымы мин, әчеме?
Сыйпаганда өзеп алам,
Әнкәйнең чал чәчеме?

Әнкәем озатып кала,
Кулым әткәй кулында.
Саклый күр мине, Ходаем,
Бу караңгы юлымда.

Моңнар ярсый, дулкын-дулкын
Җилкәннәргә җил тула.
Таң нурына сусап җырлыйм,
Юанычым шул була.
```
