# Оригинал

### Билгесез - Дим буйлары сары чәчәк

```
Дим буйлары сары чәчәк,
Өзәсен дә аласын.
Сары чәчәкләрне өзеп,
Үзең ялгыз каласын.

Дим буйларына төшмәсәм,
Өзмәгән булыр идем.
Сары чәчәкләрне өзеп,
Йөрмәгән булыр идем.

Эх дусларым, сезгә әйтәм,
Сары чәчәк өзмәгез.
Сары чәчәкләрне өзеп,
Ялгыз кала күрмәгез.
```
