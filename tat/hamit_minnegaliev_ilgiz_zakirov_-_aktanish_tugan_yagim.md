# Оригинал

### Хәмит Миңнегалиев / Илгиз Закиров - Актаныш — туган ягым

```
Таң нуры — тургайлар җыры
Киң кырларны уята,
Шул кырларда безнең яшьлек
Ялкыны ут уйната.

Актаныш — туган ягым,
Әй, гүзәл тугайларым.
Синдә туып канат ярды
Иң татлы хыялларым.

Алтын башаклар диңгезе
Чайкала назлы җилдә.
Уй-хисебез бар гомерне
Багышлау туган илгә.

Актаныш— туган җирем,
Әй, көмеш Агыйделем.
Шул Агыйдел буйларында
Сөйгәнем яши минем.

Ерак-еракларга китте
Актанышымның даны.
Яшьлек, хезмәт җыры белән
Тагын да бизик аны.

Актаныш — туган ягым,
Әй, хәтфә яланнарым.
Синдә туып канат ярды
Иң татлы хыялларым.
```
