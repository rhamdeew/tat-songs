# Оригинал

### Шаһинур Мостафин / Фатих Мөхәммәтшин - Килермен

```
Килермен мин тагын бер сезнең якларга,
Арада күп еллар, күп юллар ятса да.

Мәхәббәт — мәңгелек,
Яши ул гомергә;
Йолдыздай балкый ул
Күңелнең түрендә…

Килермен мин тагын бер сезнең якларга,
Эзләрең бит синең һәр җирдә саклана.

Яшьлек чак җырга тиң,
Тынуны һич белми.
Йөрәктә — мәхәббәт,
Сүнми ул, сүрелми…

Килермен мин тагын бер сезнең якларга,
Йөрәктә урын юк, бәгырем, башкага.

Гомерем буена
Гел сине эзләрмен.
Нур сибәр юлымда
Һаман да күзләрең…
```
