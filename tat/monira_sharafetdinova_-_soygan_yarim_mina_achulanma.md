# Оригинал

### Мөнирә Шәрәфетдинова - Сөйгән ярым, миңа ачуланма

```
Сөйгән ярым, миңа ачуланма
Төшләреңә кереп йөдәтсәм.

Шәмдәлләрдә янган утлар булып
Нурлы йөзләреңне күзәтсәм.

Сөйгән ярым, миңа ачуланма
Юлларыңда чәчәк булмасам.

Синең серле зәңгәр күзләреңә
Шаян нурлар сибә алмасам.

Сөйгән ярым, миңа ачуланма
Таң йолдызы булып янмасам.

Синең серле зәңгәр күзләреңә
Шаян нурлар сибә алмасам.
```
