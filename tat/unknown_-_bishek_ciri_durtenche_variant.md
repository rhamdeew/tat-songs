# Оригинал

### Габдулла Тукай / Татар  халык җыры - Бишек җыры (дүртенче вариант)

```
Әлли-бәлли итәр бу,
Мәдрәсәгә китәр бу;
Тырышып сабак укыгач,
Галим булып җитәр бу.

Йокла, улым, йом күзен,
Йом, йом күзең, йолдызым;
Кичтән йокың кала да,
Елап үтә көндезең.

Әлли бәлли көйләрмен
Хикәятләр сөйләрмен
Бәхетле бул диярмен
Синең теләк теләрмен

Гыйззәтем син, кадрем син,
Минем йөрәк бәгърем син;
Куанычым, шатлыгым
Тик син минем, син, син, син!
```
