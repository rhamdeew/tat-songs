# Оригинал

### Зөләйха Минһаҗева / Илфат Миңнегалиев - Таң чыгы

```
Йолдызлар елаган күренә күргәнгә
Күз яше җемелди таңнарда үләндә.
Чулпанмы елаган, Зөһрә яшьләреме,
Әллә ишеткәнме ул минем дәшкәнне.

Хыялый җаннарга
Күренә таң чыгы.
Йолдызлар яшенә
Тиңлим ул тамчыны.

Мин аларны җиргә чакырган идем лә
Я, кем инде күктән гади җиргә төшә.
Күз яшьләре тамган, ялтырый үләндә
Йолдызлар елаган күренә күргәнгә.

Хыялый җаннарга
Күренә таң чыгы.
Йолдызлар яшенә
Тиңлим ул тамчыны.
```
