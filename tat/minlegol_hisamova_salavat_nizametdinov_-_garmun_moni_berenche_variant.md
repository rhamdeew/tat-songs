# Оригинал

### Миңлегөл Хисамова / Салават Низаметдинов - Гармун моңы (беренче вариант)

```
Гармун моңы тавышлары
Нигә тын калдың әле?
Йөрәгемне сурып алып,
Утларга салдың әле.

Гармун моңы тавышлары,
Кемне сагындың әле?
Кемдер өчен югалдың да,
Минем өчен табылдың.

Гармун моңы тавышлары,
Нигә сызландың әле?
Сүнеп барган йөрәгемә
Нигә күз салдың әле?

Гармун моңы тавышлары,
Кемне сагындың әле?
Кемдер өчен югалдың да,
Минем өчен табылдың.

Гармун моңы тавышлары,
Баттым хис диңгезенә.
Ярсый-ярсый эй моңлана,
Уй салыр өнсезенә.

Гармун моңы тавышлары,
Кемне шулай сагындың?
Кемдер өчен югалдың да,
Минем өчен табылдың.
```
