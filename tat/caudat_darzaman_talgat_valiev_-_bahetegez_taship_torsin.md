# Оригинал

### Җәүдәт Дәрзаман / Тәлгать Вәлиев - Бәхетегез ташып торсын

```
Әй, дусларым, сезнең алда
Өздереп җырлыйм әле.
Куллар чабып җырлыйм әле,
Тартынып тормыйм әле.

Эшләрегез үрнәк булсын,
Дусларыгыз артып торсын,
Гомерегез озын булсын,
Бәхетегез ташып торсын.

Әй, дусларым, сезнең янда
Шатлана күңелләрем.
Җырым сезгә, моңым сезгә,
Сезгә, алсу гөлләрем.

Эшләрегез үрнәк булсын,
Дусларыгыз артып торсын,
Гомерегез озын булсын,
Бәхетегез ташып торсын.

Әй, дусларым, сезнең алда
Өздереп җырлыйм әле.
Куллар чабып җырлыйм әле,
Тартынып тормыйм әле.

Эшләрегез үрнәк булсын,
Дусларыгыз артып торсын,
Гомерегез озын булсын,
Бәхетегез ташып торсын.
Гомерегез озын булсын,
Бәхетегез ташып торсын…
Бәхетегез ташып торсын…
```
