# Оригинал

### Башкорт халык җыры - Ашказар

```
Җанкай җанаш китте, ай, сунарга,
Ашказаркай буе ла чәшкегә.*
Чәшкеләргә китеп, вафат булды,
Башкынаем калды ла яшь көйгә.

Аткайларга менеп, тезген тоткач,
Мылтыккаен бирдем лә кулына.
Күрә алмастай булып карап ла китте,
Киткән икән соңгы ла юлына.

Җанкай җанаш китте, ай, сунарга,
Калын киемнәрен лә киенеп;
Суга төшкән җирен саклап торган
Акбүз аткайлары ла тибенеп.

Кайтты ла гына, диеп, ишек ачсам,
Ак борзае тора ла, үзе юк.
Күзкәйләрен текәп, миңа карый,
Сөйләр дә иде, теле-сүзе юк.

* — икенче һәм дүртенче юллар ике тапкыр җырлана
```
