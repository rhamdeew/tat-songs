# Оригинал

### Рәми Гарипов / Хисмәтулла Дәүләтов - Ат эчерәм җылудан

```
Ат эчерәм җылудан,
Күз алмыймын сылудан,
Сызгырып та җибәрәм,
Әллә йөрәк ярсудан.

Их, суга баруларың,
Су сирпеп алуларың,
Тик баса алмый (сүндерми) суларың
Йөрәгем ярсуларын.

Сызылып таңнар атканда,
Сулар алып кайтканда,
Түгелмәде суларың —
Түгелде күз нурларың.

Түгелде күз нурларың,
Түгелде күңелемә.
Су ташымый, ут ташыйсың —
Әллә җиңелме миңа?

Ат эчерәм җылудан,
Күз алмыймын сылудан,
Сызгырып та җибәрәм,
Әллә йөрәк ярсудан.

Ал таңның атулары,
Ак томан ятулары,
Ат эчереп, су күтәреп,
Аһ, бергә кайтулары!..
```
