# Оригинал

### Татар халык җыры - Мөслим (икенче вариант)

```
Мөслимнең киң болыннары,
Матур әрәмәләре шул,
Матур әрәмәләре;
Шунда чыгып күңел ача
Эшчән, батыр яшьләре.

Уралып-уралып ага
Мөслимнең Ык сулары шул,
Мөслимнең Ык сулары;
Шул суларда йөзеп үсә
Алсу йөзле кызлары.

Ал, зәңгәр чәчәкләр җыйдым
Мөслим су буйларыннан шул,
Мөслим су буйларыннан;
Туган илем, үскән җирем
Чыкмыйсың уйларымнан.
```
