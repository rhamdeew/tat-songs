# Оригинал

### Нури Арсланов / Мәсгуть Имашев - Көймәдә

```
Күтәр күңелләремне,
Ки алсу күлмәгеңне.
Кисәң алсу күлмәгеңне,
Алалмыйм күзләремне.

Әйдә утыр көймәгә,
Карама су тирәнгә.
Суларда дулкын тирбәлә,
Дулкынланып сөйгәнгә.

Сулары ла сулары,
Көймәдә барулары;
Көймәләрдә барган чакта
Йөрәкнең янулары!

Үзәкләрем өзелә,
Бәгырем, белмисең генә.
Таңга — Чулпан, Айга — Зөһрә,
Миңа шатлык син генә.
```
