# Оригинал

### Венер Фәттах / Лилия Вахитова - Бәхетле булсын һәркем (икенче вариант)

```
Бәхетле булсын һәркем
Күл артына төшеп кояш байый
Көннең болганышын туктатып
Сәгат кенә кеше гомерләрен
Бер туктамый саный тук та тук

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында

Ә гомерләр арыш башагыдай
Ачы җилләр исә бөгелә
Җилләрдән ачы кайгыдан да
Кайнарланнан яшләр түгелә

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында

Сынауларын биреп язмыш шулай
Туземлеккә ахры өйрәтә
Фәрештәләр колакка пышылдап
Сабыр бул диеп берүк җәлләтмә

Басулардан арыш исе килә
Сыерчыклар сайрый болында
Булсын иде һәркем дә бәхетле.
Сайлап алган тормыш юлында
```
