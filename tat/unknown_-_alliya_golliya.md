# Оригинал

### Татар халык уен җыры - Аллия, Гөллия

```
Кулымдагы йөзегемнең
Исемнәре Мортаза.
Аяк атлауларың матур,
Тагын да чык уртага.

Аллия, Гөллия,
Гөллия белән Бану;
Сиңа да, миңа да авыр
Гыйшык утында яну.
Аллия, Гөллия,
Гөллия тирәккәем;
Яшә гөлләр арасында —
Шул сиңа теләккәем.

Аклы ситса күлмәгеңә
Каралар тигәләгән.
Кояш синең йөзләреңә
Нурларын сибәләгән.

Аллия, Гөллия,
Гөллия белән Бану;
Сиңа да, миңа да авыр
Гыйшык утында яну.
Аллия, Гөллия,
Гөллия тирәккәем;
Яшә гөлләр арасында —
Шул сиңа теләккәем.

Уртада басып торасың,
Әйләнмиме башларың?
Карлыгачка мисал икән
Синең кара кашларың.

Аллия, Гөллия,
Гөллия белән Бану;
Сиңа да, миңа да авыр
Гыйшык утында яну.
Аллия, Гөллия,
Гөллия тирәккәем;
Яшә гөлләр арасында —
Шул сиңа теләккәем.
```
