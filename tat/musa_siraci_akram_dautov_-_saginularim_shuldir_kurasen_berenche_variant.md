# Оригинал

### Муса Сираҗи / Әкрәм Даутов - Сагынуларым шулдыр, күрәсең (беренче вариант)

```
Нинди хәлдер…
Чишмә җырласа да,
Шауласа да Идел камышы,
Ишетелә синең тавышың;
Сулкылдатып йөрәк сагышын.
Ишетелә синең тавышың,
Сулкылдатып,
Сулкылдатып
Йөрәк сагышын!

Нинди хәлдер,
Зәңгәр күккә баксам,
Йолдыз түгел, синең күзләрең,
Тулган айда чагыла йөзләрең,
Әкрен искән җилдә — сүзләрең,
Әкрен искән җилдә, иркәм,
Синең сүзләрең.

Нинди хәлдер,
Күрәм төшләремдә,
Син елмаеп миңа киләсең,
Иркәлисең мине, үбәсең…
Сагынуларым шулдыр, күрәсең,
Төшләремә бик еш керәсең,
Сагынуларым, сагынуларым
Шулдыр, күрәсең.
```
