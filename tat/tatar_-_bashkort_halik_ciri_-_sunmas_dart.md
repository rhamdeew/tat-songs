# Оригинал

### Татар - Башкорт халык җыры - Сүнмәс дәрт

```
Агыйделдә ак кораб ла,
Утырдым алга карап.
Бер генә сүз әйтер идем
Нурлы йөзеңә карап.

Талдан тартма ясар идем —
Тал кабыгы сузылмый.
Синең күңелең белмим ничек,
Минем күңелем суынмый.

Зәңгәр чиләгең бик матур,
Су белән тулса иде.
Күңелләремә охшадың,
Гомерлек булсаң иде.
```
