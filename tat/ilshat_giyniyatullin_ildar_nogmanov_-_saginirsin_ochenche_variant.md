# Оригинал

### Илшат Гыйниятуллин / Илдар Ногманов - Сагынырсың (өченче вариант)

```
Күрешкән юк, хатлар алышкан юк.
Уйларыңда белмим кемнәрдер?
Әлегә мин сагынамын сине,
Сагынырсың әле син дә бер.

Сагынмамын, димә, сагынырсың,
Сөюләргә кабат табынырсың.
Назларыма сусап сагынулардан,
Кочагыма кереп сарылырсың …

Онытылмаска теләп син үзең дә
Уйларыма килеп керәсең.
Дәшмәсәң дә әллә инде бәлки
Күрешүләр өмет итәсең.

Сагынмамын, димә, сагынырсың,
Сөюләргә кабат табынырсың.
Назларыма сусап сагынулардан,
Кочагыма кереп сарылырсың …

Мин дә түзем, син дә сабырлар ит,
Кайнар хисләр кабат кабыныр.
Берүк ерагайтма араларны,
Сагынырсың, ләкин соң булыр.

Сагынмамын, димә, сагынырсың,
Сөюләргә кабат табынырсың.
Назларыма сусап сагынулардан,
Кочагыма кереп сарылырсың …
```
