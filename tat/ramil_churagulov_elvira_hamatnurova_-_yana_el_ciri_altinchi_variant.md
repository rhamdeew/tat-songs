# Оригинал

### Рамил Чурагулов / Эльвира Хаматнурова - Яңа ел җыры (алтынчы вариант)

```
Атлар булып еллар чаба.
Безнең кулда йөгәне.
Яңа елның килсен әйдә,
Бәхет алып килгәне!
Кайгы-хәсрәт, бәлә-каза,
Күмелеп калсын юлда.
Бәхет атының йөгәне,
Булсын гел безнең кулда!

Әй сез, еллар — тулпар атлар,
Чабыгыз алга!
Бәхет-шатлык, куанычлар,
Көтә бит анда!
Бәлә-каза, кайгы-хәсрәт,
Тапалып калсын!
Яңа җырлар, яңа еллар,
Гел кабатлансын!!!

Көн-атналар, айлар чаба,
Елгыр еллар куйнында.
Һәр кемнең дә, уй-хыялы,
Тик яхшылык турында!
Күбәләкләп карлар ява,
Ак карлар җем-җем итә.
Һәрберебез яңа елдан,
Ашкынып нидер көтә!

Әй сез, еллар — тулпар атлар,
Чабыгыз алга!
Бәхет-шатлык, куанычлар,
Көтә бит анда!
Бәлә-каза, кайгы-хәсрәт,
Тапалып калсын!
Яңа җырлар, яңа еллар,
Гел кабатлансын!!!
```
