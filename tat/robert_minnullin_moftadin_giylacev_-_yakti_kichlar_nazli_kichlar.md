# Оригинал

### Роберт Миңнуллин / Мөфтәдин Гыйләҗев - Якты кичләр, назлы кичләр

```
Икәү эчкән чишмә шул ук,
Икәү тапкан тугай шул.
Тик үзебез  менә,
Тик үзебез генә
Үзгәргәнбез бугай шул.

Без үзебез үзгәрсәк тә
Кыр да шул ук, тугай шул.
Әле дә мин сине,
Әле дә син мине
Яратабыз бугай шул.

Һавада икәү сокланган
Бар йолдызлар шул, ай шул.
Андый якты кичләр,
Андый назлы хисләр
Онытылмый бугай шул.
```
