# Оригинал

### Венера Шәрәфисламова / Зарема Әюпова - Ник сайрыйсың, сандугач?

```
Ап-ак чәчәккә күмелгән
Бакчамдагы алмагач.
Таң атканда өзеп-өзеп
Сайрый анда сандугач.

Ник сайрыйсың, сандугачым,
Бәгырькәемне телеп?
Юатырга телисеңме,
Йөрәк янганын белеп?

Ялгызмы соң әллә син дә,
Нигә моңлы сайрыйсың?
Күңелемдә ниләр барын
Әлләме соң аңлыйсың?

Ник сайрыйсың, сандугачым,
Бәгырькәемне телеп?
Юатырга телисеңме,
Йөрәк янганын белеп?

Бәргәләнә, кошчык сыман,
Урын табалмый җаным.
Өзәсең бит үзәкләрне
Сузыплар сайрап тагын.

Ник сайрыйсың, сандугачым,
Бәгырькәемне телеп?
Юатырга телисеңме,
Йөрәк янганын белеп?
Ник сайрыйсың, сандугачым,
Бәгырькәемне телеп?
Юатырга телисеңме,
Йөрәк янганын белеп?
Юатырга телисеңме,
Йөрәк янганын белеп?..
```
