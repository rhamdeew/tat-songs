# Оригинал

### Ринат  Мөслимов / Зөфәр Хәйретдинов - Әткәй тауда печән чаба

```
Яшәүләр җиңел булмаган,
Зарланмаган, беләбез.
Әткәй тауда печән чаба.
Булышырга менәбез.

Салкын чишмә суын эчә
Сусавын басар өчен.
Әткәем печән чабарлык
Һәрвакыт булсын көчең.

Беркемгә дә сиздермәде
Кайгылары булганда.
Анда булган күп гадәтләр
Сеңгәннәр безнең канга.

Салкын чишмә суын эчә
Сусавын басар өчен.
Әткәем печән чабарлык
Һәрвакыт булсын көчең.

Печәннәре кибеп җиткәч,
Матур кибәннәр куя.
Яше җитмешкә җитсә дә,
Үзен егеттәй тоя.

Салкын чишмә суын эчә
Сусавын басар өчен.
Әткәем печән чабарлык
Һәрвакыт булсын көчең.
```
