# Оригинал

### Флера Тарханова / Харис Ибраһимов - Нигә кердең әле уйларыма?

```
Нигә кердең әле уйларыма?
Тыныч иде дөньям — бозылды,
Төндә генә күргән төшләр хәтта
Көннәремә кадәр сузылды.

Тырыша-тырыша кусам да уемнан,
Син әйләнеп кабат киләсең,
Киләсең дә, ай-ваема куймый,
Хисләремә үтеп керәсең.

Йөрмә әле тынычлыгым бозып,
Бәгыремә атма син уклар…
Ә шулай да җанга сихәт өсти
Синең белән йөргән минутлар.
```
