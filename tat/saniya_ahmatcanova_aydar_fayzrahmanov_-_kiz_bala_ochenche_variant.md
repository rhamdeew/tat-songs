# Оригинал

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Кыз бала (өченче вариант)

```
Әти ясаган кабада
Җеп эрлим җырлый-җырлый.
Кулымда әни орчыгы
Әйләнә зырлый-зырлый.

И кыз бала, кыз бала,
Моңаймале, түз, бала!
Юксынып туган йортын
Гомер итә кыз бала.

Төн йокылары күрмәгән
Әнием яшь чагында,
Кул җылыларын тоям күк
Шул җитез орчыгында.

И кыз бала, кыз бала,
Моңаймале, түз, бала!
Юксынып туган йортын
Гомер итә кыз бала.

Чорналган җептәй гомерләр,
Чуалды заманалар…
Уйлармы туган нигез, дип
Без үстергән балалар?

И кыз бала, кыз бала,
Моңаймале, түз, бала!
Юксынып туган йортын
Гомер итә кыз бала.
```
