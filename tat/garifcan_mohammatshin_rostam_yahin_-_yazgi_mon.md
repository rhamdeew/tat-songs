# Оригинал

### Гарифҗан Мөхәммәтшин / Рөстәм Яхин - Язгы моң

```
Чит җирләрдән, илен сагынып,
Тургайлар кайтты инде;
Тургай күк талпына күңелем —
Моңнарым артты инде.

Ярсуыннан ярга сыймый,
Ташкыннар акты инде.
Күңеллемдә язгы ташкын —
Моңнарым артты инде.

Яшьлек хыялларым кебек,
Таңнарым якты инде.
Күңелемдә яз яктысы —
Моңнарым артты инде.

Яшьлегем — гомерем язы,
Язлар бик татлы инде.
Мәхәббәтен көтә күңелем,
Моңнарым артты инде…
```
