# Оригинал

### Билгесез - Яшьлек сөйгән ярым

```
Яшьли сөйгән ярым, еракларда калдың
Еракларда яшьлек тагында
Төшләремә кер син
Өннәремдә кил син
Якты кояшым бул янымда.

Яшьли сөйгән ярым, йөрәгемдә калдың
Аерсада ачы язмышлар
Үткәнгә уфтанмыйк
Язмышка сылтамыйк
Язмышлардан табыйк узмышлар.

Азаплама көндә, кереп төшләремә
Синең яшәү үзе зур газап
Кайтып килик дустым
Яшьлек таңнарына
Җитәкләшеп тагын бер кат.
```
