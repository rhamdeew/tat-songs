# Оригинал

### Фәридә Тимербаева / Роберт Тимербаев - Кантюковка юллары

```
Олы күлгә сикердем,
Рәхәтләнеп су кердем.
Яшь чакларны искә алып
Болыннарда йөгердем.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.

Бабай бакчасында үсә
Баланның иң татлысы.
Бу якларда гына үсә
Егет-кызның затлысы.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.

Юл төшкәндә керегез,
Бу авылны күрегез.
Кантюковка суын эчкәч
Шунда тартыр күңелегез.

Кантюковка юллары,
Уза гомер еллары.
Күңелемдә генә саклыйм
Яшьлектәге җырларны.
```
