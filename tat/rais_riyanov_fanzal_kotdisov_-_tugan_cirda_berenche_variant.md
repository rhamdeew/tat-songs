# Оригинал

### Рәис Риянов / Фәнзәл Котдысов - Туган җирдә (беренче вариант)

```
Туган җирнең газиз малае мин —
Заман шулай ясап калдырган.
Үстерделәр мине кеше итеп
Кара икмәк белән балтырган.

Уралдагы урман малае мин,
Бүредән көч алган кешемен.
Бөркет күз нурларын бирде миңа,
Туган җирдә бөркет ишемен.

Язмышыма шөкер, Кариделгә
Салып юган өчен мине дә.
Мин чәчәкмен алтын төбәгемдә,
Тыным озын Туган илемдә.
```
