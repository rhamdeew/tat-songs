# Оригинал

### Фәүзия Котлыгилдина / Лев Кулушев - Кайда

```
Кайда гына йөрөсәм дә
Сине янымда тоям,
Сөюемнең учагына
Хис-тойголарым коям…
Син мәңгелек мөхәббәтем,
Минең гәзиз яргынам,
Янәшәмнән генә килгән
Иң якыным, бергенәм.

Кайда син, матурым, кайда син,
Мин бит тик сине көтәм,
Сагындым, матурым, сагындым,
Мин көтәм…

Йөрәгемдә тамырланып
Үсә синең исемең,
Кәсәсенән ләззәт эчәм
Олы сөю хисенең.
Иркәләнәм, назланамын
Мөхәббәт кочагында,
Җылынамын сөюемнең
Сүрелмәс учагында.

Кайда син, матурым, кайда син,
Мин бит тик сине көтәм,
Сагындым, матурым, сагындым,
Мин көтәм…
```
