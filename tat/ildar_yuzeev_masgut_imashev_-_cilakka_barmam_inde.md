# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Җиләккә бармам инде

```
Җиләккә дип барган идем,
Юри генә адаштым.
Сине күргәч, сүзсез калдым,
«Адаштым» дип алдаштым.

Синең йөргән юлларыңда
Җиләкләрем түгелде.
Салкын каравыңны белгәч,
Күңелләрем сүрелде.

Җиләкләргә бармам инде.
Бармам инде, адашмам.
«Җиләклектә адаштым», — дип
Инде бүтән алдашмам.
```
