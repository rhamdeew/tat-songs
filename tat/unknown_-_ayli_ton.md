# Оригинал

### Әхмәт Ерикәй / Заһид Хәбибуллин - Айлы төн

```
Җәйге төн, якты төн,
Күктә ай елмая.
Шул айның шәүләсе
Тирбәлә елгада.

Ай күрә чәчәкле,
Бизәкле кырларны;
Көнләшеп тыңлый ул
Җирдәге җырларны,

Сандугач үзенең
Күңелле җырында
Белдерә сөмбелне
Сөюе турында.

Ә сөмбел оялчан,
Җырламый, көйләми,
Хисләре турында
Кешегә сөйләми.

Гармунчы төн буе.
Сыздыра, түзми ул.
Кемнедер юксына,
Нәрсәдер эзли ул.

Юксынма, моңайма,
Ташламас чибәрен.
Уйна син, уйна син,
Кавышу көйләрен!..
```
