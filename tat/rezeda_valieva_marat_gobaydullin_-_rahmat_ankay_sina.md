# Оригинал

### Резеда Вәлиева / Марат Гобәйдуллин - Рәхмәт, әнкәй, сиңа

```
Шомырт чәчәк атса,
Илгә кошлар кайтса,
Җаным тула минем назларга.
Рәхмәт, әнкәй, сиңа,
Якты тормыш миңа
Бүләк иткән өчен язларда.

Йөрәк җылың белән,
Бишек җырың, белән
Күңел кылларымны чирткәнсең.
Рәхмәт, әнкәй, сиңа,
Сабый чакларымнан
Бу дөньяга гашыйк иткәнсең.

Язгы кояш төсле,
Сулмас яшәү көчен
Биргәнсең син миңа гомергә.
Рәхмәт, әнкәй, сиңа,
Кояш булып миңа
Мәңге-мәңге яшә бу җирдә.
```
