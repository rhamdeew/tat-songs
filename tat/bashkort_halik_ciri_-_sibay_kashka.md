# Оригинал

### Башкорт халык җыры - Сыбай кашка

```
Сыбай да гына кашка, басма ташка,
Юлсыз гына җирдән дә үткәндә.
Бу дөнъяда авыр ялгыз башка,
Дус-иш кирәк дөнъя да көткәндә.

Сыбай гына кашка кырач ялын
Таудан гына өргән дә җил тарай.
Ерак җиргә китсәк, сыбай кашкам
Башкынаен борып та бер карый.

Ерактан ук агарып, ай, күренгән
Урал таукаеның да ак ташы.
Кайда гына булмый, нидәр күрми
Ир башкае белән дә ат башы.
```
