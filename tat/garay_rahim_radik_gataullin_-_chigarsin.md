# Оригинал

### Гәрәй Рәхим / Радик Гатауллин - Чыгарсың

```
Бар халык йоклаган,
Урамнар тын калган,
Күктә ай елмайган,
Айның Чулпаны бар анда…

Чыгарсыңла, чыгарсың…
Соң ничек чыдарсың,
Урамда мин ялгыз булганда?

Бу таныш тыкырык…
Гармунын суздырып,
Моңлы көй сыздырып,
Тәрәз төбендә торамын.

Чыгарсыңла, чыгарсың…
Соң ничек чыдарсың,
Урамда мин ялгыз булганда?

Җыр сузып җибәрәм,
Битемә җил бәрә,
Яшь каен тирбәлә,
Яфрак шыгырдый тын гына…

Чыгарсыңла, чыгарсың…
Соң ничек чыдарсың,
Урамда мин ялгыз булганда?
```
