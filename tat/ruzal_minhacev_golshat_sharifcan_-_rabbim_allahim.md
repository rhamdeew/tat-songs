# Оригинал

### Рузәл Минһаҗев / Гөлшат Шәрифҗан - Раббым Аллаһым

```
Тормыш сукмагыннан атлаганда
Юлым ерак алга барасым.
Гомер тауларына менгән чакта
Ярдәм итәр Раббым Аллаһым.
Тез астына суккан чакта язмыш
Белмим ничек аумый каласын.
Сабыр канатларым сындырмыйча
Саклый мине Раббым Аллаһым.

Гамәл дәфтәрләрен башлаганда
Сайламыйк без юлның карасын.
Аклы хыял белән яши алса күңел,
Яратыр ул Раббым Аллаһым.

Тормыш дигән сынау сукмагында
Үкенечле эзләр калмасын.
Вакытында гафу сорый алсак
Ярлыкар ул Раббым Аллаһым.
Безнең янәшәдән атлап барган
Якыннарны иртә алмасын.
Бер-беребезне кадерләп яшәргә
Мөмкинлек бир, Раббым Аллаһым.

"БисмилЛаһ"ны укып көнем башлыйм,
Эчтән укыйм "Кульһу Аллаһы"н.
Туры юлга тугры калыйм, диеп,
Яшәт мине, Раббым Аллаһым.
Туры юлга тугры калыйм, диеп,
Яшәт мине, Раббым Аллаһым…
```
