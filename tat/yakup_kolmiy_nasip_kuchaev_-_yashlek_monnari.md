# Оригинал

### Якуп Колмый / Насип Кучаев - Яшьлек моңнары

```
Үтә көннәр, үтә еллар,
Яшьлек язым артта кала.
Былбылым да алсу таңда
Күңелемә (йөрәгемә) моңнар сала.

Бер минутка, бер сәгатька
Яшьлегемә кайтыр идем,
Җырым булып, моңым булып
Тагын чәчкә атар идем.

Шаян яшьлек ялкын чәчеп
Бер килдең дә, мәңге калдың.
Гомер буе синең утта
Йә җылындым, йә бер яндым.

Бер минутка, бер сәгатька
Яшьлегемә кайтыр идем.
Җырым булып, моңым булып,
Тагын чәчкә атар идем.

Ерак калган яшьлек һаман,
Елмаясың айлы кичтә.
Канат каккан дәртле заман
Һаман истә, һаман истә.

Бер минутка, бер сәгатька
Яшьлегемә кайтыр идем.
Җырым булып, моңым булып,
Тагын чәчкә атар идем.
```
