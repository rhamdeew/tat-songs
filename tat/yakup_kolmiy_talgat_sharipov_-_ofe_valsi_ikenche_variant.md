# Оригинал

### Якуп Колмый / Тәлгать Шәрипов - Өфе вальсы (икенче вариант)

```
Нурлы Өфем минем, җырлы (матур) Өфем,
Син бирәсең кояш җылысын.
Тирбәләсең бакча кочагында,
Мәңге яшәр тормыш җыры син.

Балкыйсың син Өфем
Идел ярында.
Ялкын яктың миңа
Яшьлек язымда.

Якты Өфем минем, алтын Өфем,
Синең нурдан ай да көнләшә.
Мәхәббәт тә синдә бөреләнә —
Йөрәк белән йөрәк серләшә.

Балкый гүзәл Өфем
Идел ярында.
Ялкын яктың миңа
Яшьлек язымда.

Гүзәл Өфем минем, моңлы Өфем,
Синдә минем килер язмышым.
Читкә китсәм, бик тиз кайтып җитәм,
Баса алмый күңел сагышын.

Балкыйсың син Өфем
Идел ярында.
Ялкын яктың миңа
Яшьлек язымда.

1977
```
