# Оригинал

### Нәфисә Сабирҗанова / Илнур Фәйзуллин - Үзем әни булгач аңладым

```
Гөрләп үтте минем бала чагым,
Бер кайгысыз атты таңнарым.
«Бәхет илен» әнкәй корган булган, —
Үзем әни булгач аңладым.
Синең сүзең, киңәш, кисәтүең
Күңелемә нигә салмадым?
Һәрбер сүзе изге дога булган, —
Үзем әни булгач аңладым.
Борчуларын миннән яшергәндә,
Үзем белеп нигә калмадым?..
Әнкәемнең хәлен, кадерләрен
Үзем әни булгач аңладым.
Шатлыктан да күз яшьләре тама, —
Үзем әни булгач аңладым.
Әнкәемнең хәлен, кадерләрен
Үзем әни булгач аңладым.
```
