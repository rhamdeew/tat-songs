# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Тальян гармун (беренче вариант)

```
Урам буйлап егет узды
Тальян гармун сыздырып.
Моңына күмеп авылны,
Кызларны кызыктырып.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.

Нигә соң мин былбыл түгел
Көмеш айлы кичләрдә.
Яшьлектә калган сөйгән яр
Ник кермисең төшләргә.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.

Тыңла, дустым, яңгырап чыга
Тальянның тавышлары.
Шул тальянның тавышында
Гомернең агышлары.

Тальян гармун, тальян гармун,
Каян шундый моңлы син?
Яшьлегемә кабат кайтып,
Килә сине тыңлыйсым.
```
