# Оригинал

### Хәйдәр Әхмәтшин / Илназар - Аккошым дип эндәшәм (Ул — минеке)

```
Кыр казлары кебек ул да
Сулар, гөлләр ярата.
Аккош күлләренә барып,
Күңелләрен юата.

Кояш белән ул яна да,
Кошлар белән серләшә.
Мин дә аны якын күреп,
Аккошым дип эндәшәм.

Әнә шундый серле кызны
Сез дә килеп күрегез.
Гашыйк булу кирәк түгел,
Ул — минеке, белегез!

Нинди бәхет аның белән
Бер күрешеп сойләшү!
Күзләренә туры карап,
Кадерлем дип эндәшү!

Кулларында гөлләр үсә:
Кулы жиңел, күрәсең.
Үзе уңган, үзе чибәр,
Күзләр генә тимәсен!

Гөлләр кебек назлый-назлый,
Аңа гына җыр язам.
Җырым аша әйтәм аңа:
"Булма, — диеп, җан ярам!"

Нинди бәхет аның белән
Бер күрешеп сөйләшү!
Күзләренә туры карап,
Кадерлем дип эндәшү!
```
