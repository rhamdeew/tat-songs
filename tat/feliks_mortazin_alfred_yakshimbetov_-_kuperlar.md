# Оригинал

### Феликс Мортазин / Альфред Якшимбетов - Күперләр

```
Һәркемнең дә үз язмышы
Һәркемнең үз күпере
Сикәлтәле күперләрне
Ай һай кыен үтүе

Кирегә ябык юлларым,
Яктым мин күперләрне
Шул утларның яктысында
Таптым синең эзләрне.

Кемдер калды аргы ярда
Кем егылды күпердән
Тырышканга гына ходай
Юл үтәргә көч биргән

Кирегә ябык юлларым,
Яктым мин күперләрне
Шул утларның яктысында
Таптым синең эзләрне.

Язмыш юлларын узганда
Дус кирәк таянырга
Курыкмагыз югалтудан
Ташламас чын дус булса

Кирегә ябык юлларым,
Яктым мин күперләрне
Шул утларның яктысында
Таптым мин үз үземне.

Күперләрне ялгыз узмыйк
Адашмыйк бураннарда
Гомер көзләрем куркытмый,
Ышанам туганнарга

Кирегә ябык юлларым,
Яктым мин күперләрне
Шул утларның яктысында
Таптым синең эзләрне.

Кирегә ябык юлларым
Яктым мин күперләрне
Шул утларның яктысында
Таптым синең үзеңне
```
