# Оригинал

### Гөлшат Зәйнәшева / Екатерина Шатрова - Бер атнада ничә көн?..

```
Бер атнада ничә көн?
Бер атнада җиде көн.
Җиде төрле исеме —
Җидесен дә яттан бел:

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе —
Онытма мәңге!..

Шимбә белән якшәмбе —
Икесе ял көннәре.
Ял көннәре атнаның —
Чәчәкләре, гөлләре.

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе —
Онытма мәңге!..

Ял көннәре атнаның
Чәчәкләре, гөлләре.
Атнаның яфраклары,
Димәк, калган көннәре.

Дүшәмбе, сишәмбе, чәршәмбе,
Пәнҗешәмбе, җомга,
Шимбә, якшәмбе —
Онытма мәңге!..
```
