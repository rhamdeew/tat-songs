# Оригинал

### Венер Котдусов - Җәйге җыр

```
Агыла да болыт, агыла да болыт,
Җилфер-җилфер җилгә кушылып,
Җәйге аяз күкләрдә, җәйге аяз күкләрдә.
Дус-иш белән яшәү рәхәт туып үскән илләрдә.

Йөгереп-йөгереп каршы бардым
Җаныем-бәгърем су алганда,
Чишмәләрдә-суларда, чишмәләрдә-суларда.
Аерылмагыз, аерылмагыз, куллар булса кулларда.

Тәрәз ачтым, айга карыйм,
Ай каршында йолдыз уйный
Җәйге аяз кичләрдә, җәйге аяз кичләрдә.
Йөрегән юллар киң булса да җитми туган илләргә.

Йөрегән юллар киң булса да җитми туган илләргә,
Йөрегән юллар күп булса да җитми туган илләргә…
```
