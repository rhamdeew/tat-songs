# Оригинал

### Илдар Ногманов / Гүзәл Мингашаева - Йөрәк түре

```
Моңсуланып, уйга таласың да,
Күкрәгемә башың иясең.
"Оныта (а)лмыйм, бәгърем, яратамын,
Бик авыр бит, аңла" — диясең.
Күңелең йомшак, күзләреңдә яшьләр,
Көтәсеңдер миннән бер хәбәр.
Синең күк мин дә сине уйлыйм,
Бик сагынам, иркәм, сагынам.
Очрашырга безгә булыр микән,
Очрашканбыз гомер юлында.
Йөрәкләрдә урын түрдән алыйк,
Аерылмыйк гомер буена.
```
