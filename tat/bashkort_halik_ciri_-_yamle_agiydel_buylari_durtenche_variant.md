# Оригинал

### Башкорт халык җыры - Ямьле Агыйдел буйлары (дүртенче вариант)

```
Агыйделкәй кайда, ай, тугайда,
Агыйделкәй кебек су кайда.
Агыйделкәйләрнең, ай, һавасы
Җанга рәхәт, тәнгә зур файда.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны шул,
Бетерәдер хәсрәтле уйларны.

Идел суын эчсәң, ай, һуш керә,
Талган беләкләргә көч керә.
Безне генә сөйләр хас дошманнар,
Йоклаганда, куркып төш күрә.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны шул,
Бетерәдер хәсрәтле уйларны.

Агыйделкәй сыуы, ай, шифалы,
Тултырып алаекчы шешәне,
Агыйделкәй суын, ай, эчкәнгә,
Кайгылардан эчем бушады.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны шул,
Бетерәдер хәсрәтле уйларны.

Агыйделкәйләрнең, ай, буенда,
Сандугачлар сайрый шомыртта.
Идел суы эчкән ир-егетләр
Яман теләк йөретмәс уйында.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны шул,
Бетерәдер хәсрәтле уйларны.

Агыйделкәй суы, ай, тын икән,
Яр читтәре бигерәк комлы икән.
Агыйделдә үскән, ай, кызларның
Тавышлары бигерәк моңлы икән.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны шул,
Бетерәдер хәсрәтле уйларны.
```
