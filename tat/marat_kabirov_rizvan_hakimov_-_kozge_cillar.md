# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Көзге җилләр

```
Яздан ерак йөрми икән
Гомер көзе дә.
Көзнең сары яфраклары
Җилгә өзелә.
Күреп гомер агачыңның
Саргайган чагын,
Сагышларга бирелмә син,
Бирелмә, җаным.

Көзге җилләр, көзге җилләр,
Ныграк исегез.
Газапларны, сагышларны
Алып китегез.

Көзге җилләр, ачы җилләр
Ерак исегез.
Кайгыларны, сагышларны
Ерак итегез итегез.
Көзләрнең үз матурлыгы,
Үзенең назы.
Сагышларга бирелмә син,
Бирелмә тагы.

Көзге җилләр, көзге җилләр,
Ныграк исегез.
Газапларны, сагышларны
Алып китегез.

Яздан ерак йөрми икән

```
