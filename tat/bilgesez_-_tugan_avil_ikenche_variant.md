# Оригинал

### Билгесез - Туган авыл (икенче вариант)

```
Авылымны ташлап киткәнгә
Инде ничемә еллар узды
Бер мизгельга барым онытоп
Авылыма кайтасы иде

Кайтсаң әгәр туган авылыма
Өзөлеп-өзөлеп елар идем
Шатлыгымнан урам уртасында
Өздреп-өздреп бер җырлар идем

Айрылганда тыуып үскән җирдән
Яшләр килде моңсу күзләргә
Сагынам мин авыл урамнарын
Балачагым керә төшләргә

Туган авылым мине каршы алса
Кайтыр идем әллә кайлардан
Чирәмендә ятып авынар идем
Суын эчар идем коелардан
```
