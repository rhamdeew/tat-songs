# Оригинал

### Ришат Галимов / Фәнис Мортазин - Туган көн (өченче вариант)

```
Күк йөзе чалт аяз бүген,
Кояш та серле елмая…
Җил талган татлы йокыга,
Җир йөзе тулган нурларга…
Ә бүген синең туган көн,
Унсигез яшь тула сиңа!
Йөзләрең шатлыктан балкый,
Күзләрең бәхеткә тула!

Туган көн белән,
Котлыйбыз сине!
Теләкләрдән
Иң саф, изгесе!

Дусларың бергә җыелган,
Котлаулар синең хөрмәткә.
Зур бәхет, шатлык телиләр,
Күмәләр сине бүләккә!

Туган көн белән,
Котлыйбыз сине!
Теләкләрдән
Иң саф, изгесе!

Зур юлга аяк басасың,
Күп булыр алда киртәсе;
Бирешмә язмышларга син,
Заяга гомрең үтмәсен!

Туган көн белән,
Котлыйбыз сине!
Теләкләрдән
Иң саф, изгесе!
```
