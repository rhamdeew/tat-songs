# Оригинал

### Илдар Юзеев / Әнвәр Бакиров - Авылдашка хат

```
Язгы сулар кебек ташып
Сүрелгән яшь чаклардан,
Туфрагында ятып калган
Былтыргы яфраклардан

Сәлам сиңа, авылдашым,
Туган-үскән җирләрдән,
Туган-үскән җирләренә
Кайтып искән җилләрдән.

Бала чактай җемелдәгән
Таңгы нарасый чыктан,
Һәр яз оясына кайтып
Сайраган сыерчыктан

Сәлам сиңа, авылдашым,
Туган-үскән җирләрдән,
Туган-үскән — җирләренә
Кайтып искән җилләрдән.

Авылыңның уйчан төнен
Яктырткан тын аеннан,
Әнкәң туфрагында моңсу
Шаулаган ак каеннан

Сәлам сиңа, авылдашым,
Туган-үскән җирләрдән,
Туган-үскән җирләренә
Кайтып искән җилләрдән…
```
