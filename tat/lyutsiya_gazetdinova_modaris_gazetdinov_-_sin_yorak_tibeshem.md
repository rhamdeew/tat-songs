# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Син йөрәк тибешем

```
Җаннарыбыз безнең
Табышкан, кавышкан.
Мәхаббәт, яратып,
Иңебездэн кочкан…
Икебез — бербөтен!
Бер йөрәк, бер сулыш!
Җиденче күкләргә
Йолдыз булып очкан!

Төшемдәге түгел,
Өндәге "жимешем"!
Җанымның яртысы,
Син йөрәк тибешем!

Сине дә, мине дә
Саф хисләр тилерткән….
Сөюебез безне
Назлардан исерткән!
Сихри төннәрне дә,
Матур көннәрне дә
Сиңа да, миңа да
Сөю бүләк иткән!

Төшемдәге түгел,
Өндәге "жимешем"!
Җанымның яртысы,
Син йөрәк тибешем!

Хисләрем түгелә,
Моң булып үрелә!
Өзелеп яратам!..
Күз нурым!.. Йолдызым!
Син минем кадерлем,
Иң татлы күз яшем!
Син — минем бәхетем,
Хыялым, язмышым!..

Төшемдәге түгел,
Өндәге "жимешем"!
Җанымның яртысы,
Син йөрәк тибешем!
```
