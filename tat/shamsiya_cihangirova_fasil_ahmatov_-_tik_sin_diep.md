# Оригинал

### Шәмсия Җиһангирова / Фасил Әхмәтов - Тик син диеп

```
Тик син диеп,
Янып-көеп,
Үтә лә гомерләрем;
Үткән гомеремне уйлап,
Ярсына күңелләрем.

Тик син диеп
Өзгәләнәм,
Уйланам иртә-кичен;
Ике гомер яшәсәм дә
Янармын синең өчен.

Тик син диеп,
Янып-көеп,
Үтмәс лә гомерләрем —
Очрашырбыз, кавышырбыз,
Ышана күңелләрем.
```
