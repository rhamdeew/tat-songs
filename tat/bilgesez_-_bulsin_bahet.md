# Оригинал

### Билгесез - Булсын бәхет

```
Бәхет булсын янәшәңдә,
Саулык булсын чын дустың.
Елмаеп-көлеп яшә син,
Шатлык булсын юлдашың.

Дөньяның һәр яңа көне
Яңа бәйрәм тудыра.
Еллар чаба юргалардай,
Берсен берсе уздыра.

Бәхет булсын янәшәңдә,
Саулык булсын чын дустың.
Елмаеп-көлеп яшә син,
Шатлык булсын юлдашың.
Теләкләрем йөрәгемдә,
Чәчәкләрем кулымда.
Тәбриклимен, чын күңелдән
Кабул ит син барын да.

Бүләк итәм бу җырымны,
Бары тик сиңа атап.
Чәчәк бәйләмнәре сузам,
Ихластан сине котлап.

Бәхет булсын янәшәңдә,
Саулык булсын чын дустың.
Елмаеп-көлеп яшә син,
Шатлык булсын юлдашың.
Теләкләрем йөрәгемдә,
Чәчәкләрем кулымда.
Тәбриклимен, чын күңелдән
Кабул ит син барын да.

Бәхет булсын янәшәңдә,
Саулык булсын чын дустың.
Елмаеп-көлеп яшә син,
Шатлык булсын юлдашың.
Бәхет булсын…

Күңелеңдәге ялкының
Бер кайчан да сүнмәсен.
Йөрәгеңдә булган хисләр
Гомерлеккә дөрләсен.

Бәхет булсын янәшәңдә,
Саулык булсын чын дустың.
Елмаеп-көлеп яшә син,
Шатлык булсын юлдашың.
Теләкләрем йөрәгемдә,
Чәчәкләрем кулымда.
Тәбриклимен, чын күңелдән
Кабул ит син барын да.
```
