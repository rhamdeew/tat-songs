# Оригинал

### Гөлшат Әхмәткужина / Салават Низаметдинов - Яшәү хакы белүдәдер

```
Кеше илдә кызык бит ул,
Алда кебек кичерере.
Мәхәббәте, күңеленең
Иң аһәңле моңы, сере.
Бәхетнең дә олырагы
Киләчәктә кебек аңа.
Әйтерсең дә, иртәгә ул
Күберәкне аңлый, тоя.

Иң зур бәхет — бүгенгенең
Кадерләрен белүдәдер,
Шат чакларда моңсуланып,
Елар мәлдә көлүдәдер,
Яшәү хакы, яшәү хакы,
Яшәү хакы белүдәдер.

Ә ачылда минутларың
Иң асылы артта калган.
Ашкынулар, талпынулар
Маңгайларга сырлар салган.
Сөюнең дә кабатланмас
Чәчәк атар мәле үткән.
Тәүге сөю каеннары
Яфрак койган инде күптән.

Иң зур бәхет — бүгенгенең
Кадерләрен белүдәдер,
Шат чакларда моңсуланып,
Елар мәлдә көлүдәдер,
Яшәү хакы, яшәү хакы,
Яшәү хакы белүдәдер.
```
