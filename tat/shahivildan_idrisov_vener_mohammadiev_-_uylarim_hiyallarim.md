# Оригинал

### Шаһивилдан Идрисов / Венер Мөхәммәдиев - Уйларым — хыялларым

```
Еллар аша кире кайтам,
Туган туксаннан узам.
Яңа пешкән икмәк исен
Тагы тойгандай булам.

Әнкәемнең елмаюын
Тагы күргәндәй булам.
Тупылдатып яратуын
Аркамда тоям һаман.

Балачакка кайтарсаң да,
Күкләргә күтәрсәң дә.
Туган җирдән аермагыз
Уйларым — хыялларым.
```
