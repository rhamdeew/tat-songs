# Оригинал

### Сания Әхмәтҗанова / Фәрит Әхмәтов - Туган авылым Түнтәр

```
Борма-борма Түнтәр юлы
Еракка алып китә.
Читләргә чыгып киткәннәр
Юксынып гомер итә.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.

Хәтерем сукмакларыннан
Үткәннәремә кайттым.
Сагышларыма дәваны,
Түнтәрем, синдә таптым.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.

Баш урманның аланнарын
Каплаган куе үлән.
Бер җырлап утырыйк әле
Яшьлек дусларым белән.

Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.
Саваплы чишмәләреңнең
Суын эчеп туймадым.
Сандугач моңнарын тыңлап,
Урамнарны уйладым.
```
