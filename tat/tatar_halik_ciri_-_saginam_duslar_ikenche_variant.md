# Оригинал

### Татар  халык җыры - Сагынам, дуслар (икенче вариант)

```
Сагынам, дуслар, сагынам, дуслар,
Сагынам, дуслар, илкәйне, шул,
Сагынам, дуслар, илкәйне.
Илкәй белән бергә сагынам
Яшьли сөйгән иркәмне.

Яшь коям эзләреңә, дә,
Баш куям тезләреңә шул,
Баш коям тезләреңә.
Әллә ниләр бирер идем
"Аһ" дигән сүзләреңә.

Ишек алды яшел чирәм
Былбыл баласы кача, шул,
Былбыл баласы кача.
Бик сагынсам айга карыйм
Айны да болыт баса.
```
