# Оригинал

### Гыйләҗ Минаев / Фәрит Хатыйпов - Болгар — чишмә яныңда

```
Чишмәләр дә шаһит үткәннәргә,
Күпме гасыр узып китсә дә,
Ташларында калган кан таплары,
Күз яшьләре күптән кипсә дө.

Янәшәдә үскән төз каеннар,
Нык тамырлы, горур имәннәр.
Искә алып данлы үткәннәрне
Чал башларын түбән игәннәр.
```
