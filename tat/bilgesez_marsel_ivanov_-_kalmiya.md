# Оригинал

### Билгесез / Марсель Иванов - Калмыя

```
Урман кырына урнашкан
Авылым минем Калмыя.
Туган якларын күрергә
Кемнәр кайтмый калыр, я?

Гадилеге белән матур
Авылым минем Калмыя.
Үзенә гел тарта торган
Сихер көченә ия.

Авылыма кайтам, дуслар,
Вакытым тар булса да.
Карлы юлларыннан атлыйм
Итеккә кар тулса да.

Гадилеге белән матур
Авылым минем Калмыя.
Үзенә гел тарта торган
Сихер көченә ия.

Барлап киләм туганнарны,
Ак сакаллы картларны.
Керфер күлгә сәлам биреп,
Түбән иям башларны.

Гадилеге белән матур
Авылым минем Калмыя.
Үзенә гел тарта торган
Сихер көченә ия.
```
