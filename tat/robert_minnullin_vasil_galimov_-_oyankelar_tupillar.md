# Оригинал

### Роберт Миңнуллин / Вәсил Галимов - Өянкеләр, тупыллар

```
Өй алдында өянкеләр,
Тупыллар исән әле.
Үткәреп үзәкләренә
Җилләр дә исә әле.

Беләм, гомер буйларына
Миңа дога укырлар.
Әнкәй кебек өянкеләр
Әткәй кебек тупыллар.

Өянкеләр өйрәттеләр
Яшь аралаш көләргә.
Туган җирдән китүемне
Яшем белән түләргә.

Беләм, гомер буйларына
Миңа дога укырлар.
Әнкәй кебек өянкеләр
Әткәй кебек тупыллар.

Алар мине сагынадыр,
Миңа сәлам юллыйдыр.
Тупыллар сиздермәсә дә
Өянкеләр елыйдыр.

Беләм, гомер буйларына
Миңа дога укырлар.
Әнкәй кебек өянкеләр
Әткәй кебек тупыллар.
```
