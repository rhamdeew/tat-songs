# Оригинал

### Гөлшат Зәйнәшева / Рәшит Абдуллин - Мандолина

```
Уйна, дустым, мандолина,
Кыллары чыңлап торсын.
Чишмә кебек челтери дип,
Егетләр тыңлап торсын.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары.

Уйна, дустым, мандолина,
Кыллары нечкө-нечкө.
Сөйсәң, сөй ярның матурын,
Калмасын үкенечкә.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары.

Уйна, дустым, мандолина,
Кылларын чиртеп кенә.
Яшик җырлап, бу илләрдә,
Күңелле итеп кенә.

Терлин, терлин, терлин, терлин,
Мандолина кыллары.
Камыш бармак, көмеш тырнак,
Син дустымның куллары!..
```
