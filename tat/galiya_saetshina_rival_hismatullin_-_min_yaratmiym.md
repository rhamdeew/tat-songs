# Оригинал

### Галия Сәетшина / Риваль Хисмәтуллин - Мин яратмыйм

```
Мин яратам җирдә кешеләрнең
Туры сүзле, гадел булганын.
Тормыш сукмагыннан дөрес атлап,
Изгелекләр генә кылганын.

Мин яратмыйм җирдә кешеләрнең
Гайбәт җыеп синнән көлгәнен.
Мин кем диеп, үзен генә сөеп,
Ялган сүзләр сөйләп йөргәнен.

Мин яратам җирдә кешеләрнең
Кояш кебек балкып йөргәнен.
Сорау бирсәң — боргаланмый гына
Туры итеп җавап биргәнен.

Мин яратмыйм җирдә кешеләрнең
Гайбәт җыеп синнән көлгәнен.
Мин кем диеп, үзен генә сөеп,
Ялган сүзләр сөйләп йөргәнен.

Мин яратам җирдә кешеләрнең
Күңелемне аңлый белгәнен.
Газиз туган җирен онытмыйча
Вакыт табып кайтып йөргәнен.

Мин яратмыйм җирдә кешеләрнең
Гайбәт җыеп синнән көлгәнен.
Мин кем диеп, үзен генә сөеп,
Ялган сүзләр сөйләп йөргәнен.
Мин кем диеп, үзен генә сөеп,
Ялган сүзләр сөйләп йөргәнен…
Ялган сүзләр сөйләп йөргәнен…
```
