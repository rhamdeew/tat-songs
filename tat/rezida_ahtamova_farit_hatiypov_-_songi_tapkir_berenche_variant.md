# Оригинал

### Резидә Әхтәмова / Фәрит Хатыйпов - Соңгы тапкыр (беренче вариант)

```
Мин бик озак сине эзлеп тапкан идем,
Юлларыңа шашып йөгереп чапкан идем.
Инде менә очрашабыз соңгы тапкыр,
Кара көздә безне котлап ява яңгыр.

Соңгы тапкыр очрашабыз синең белән,
Без генәме мәхәббәттән качып киткән!
Йөрәктәге яралардан әрнү тама,
Авыр сагышлардан эзләр сыгылып кала.

Соңгы тапкыр очрашабыз синең белән,
Юлыбызга көзләр җәйгән сары келәм.
«Мин яраттым, мин яраттым сине, кичер», —
Җилләр сиңа соңгы тапкыр сәлам илтер.
```
