# Оригинал

### Рамилә Абдуллина - Яшьлеккә кайтсаң иде

```
Яшьлеккә, яшьлеккә
Юллар ачылмый хәзер.
Мәктәп елларын сагынып,
Өзгәләнәдер бәгырь.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Ак алъяпкыч, кара форма
Тагын бер кисәң иде.

Яшьлекнең, яшьлекнең
Хатирәләре якты.
Шуңа күрә сагынабыз
Бу гүзәл, матур чакны.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Шул бәхетле мизгелләрне
Тагын кичерсәң иде.

Яшьлек бит, яшьлек бит —
Иң көчле, җегәр вакыт.
Таулар-ташлар актарырдай,
Зур көчләр тора ташып.

Яшьлеккә, яшьлеккә
Бер кайтып килсәң иде.
Анда калган көч-дәртләрне
Монда күчерсәң иде.
```
