# Оригинал

### Мөнир Мазунов / Илдус Якупов - Энесенә ул апа

```
Җырлый-җырлый кыз бала
Энекәшен йоклата,
Биштә генә булса да,
Энесенә ул апа.

Әлли-бәлли бәй итә,
Энем йокыга китә.
Әнкәй, мәми пешереп,
Әткәйне эштән көтә.
Кошлар кунган тирәккә,
Аларны җил тибрәтә.
Тышта җилләр искәндә,
Бәби йоклый мендәрдә.
Әлли-бәлли бәй итә,
Энем йокыга китә.
Йоклатканга энемне,
Әнкәй мине пәп итә.
Йоклый бала, төш күрә,
Көлә, күккә сикерә,
Имеш, аны әтисе
Үсти-үсти иттерә.
Әлли-бәлли бәй итә,
Энем йокыга китә.
Энем йокыга киткәч,
Әни бераз ял итә.
```
