# Оригинал

### Роберт Шәймәрданов - Хушлаштык

```
Язның матур бер көнендә
Кинәт очраштык.
Айлы кичләргә сокланып
Ныклап дуслаштык.

Хисләр хискә уралынгач
Без бит кавыштык.
Тормыш йөген сөйрәгәндә
Ник соң ялгыштык?

Гомеребез агышында
Юкка орыштык.
Үкенәчәкне уйламый
Тиздән хушлаштык…

Хисләр хискә уралынгач
Без бит кавыштык.
Тормыш йөген сөйрәгәндә
Ник соң ялгыштык?
```
