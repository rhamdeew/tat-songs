# Оригинал

### Әсхәр Кәшфуллин / Зөфәр Хәйретдинов - Әнкәем күзләре

```
Әнкәем күзләре,
Кызартмам йөзләрең,
Онытмам алтындай сүзләрең.
Яшьләнә күрмәсен,
Кайгылар белмәсен
Әнкәемнең нурлы күзләре.

Күрергә тилмерәм,
Төшләрдә тик күрәм
Әнкәемнең нурлы күзләрен.

"Тәүфыйклы бул, балам,
Сабыр бул," — дип кала,
Чыксам әгәр ерак сәфәргә.
Җитсә дә көзләре,
Әнкәем күзләре
Дәрт һәм куәт бирә яшәргә.

Күрергә тилмерәм,
Төшләрдә тик күрәм
Әнкәемнең нурлы күзләрен.

Әнкәем күзләре
Очкынлы үзләре,
Йөрәктә акыллы сүзләре.
Дөньяга нур чәчә,
Әнкәем, мең яшә,
Кояш кебек синең күзләрең.

Күрергә тилмерәм,
Төшләрдә тик күрәм
Әнкәемнең нурлы күзләрен.
```
