# Оригинал

### Айрат Галиев / Марат Шәйбәков - Сөйгәннәргә ачу тотмагыз

```
Мин гәепле, иркәм, синең алда,
Башым иеп гафу үтенәм.
Зинһар өчен кичер мине диеп,
Әйткән сүзләремә үкенәм.

Кошларны да тотып ябып була,
Әммә сүзләр, алар кош түгел.
Кышларыма кабат синсез кердем,
Шуңа көя минем яшь күңел.

Кулларымнан килсә тормыш юлын,
Борып кайтыр идем үткәнгә.
Бер генә дә ялгыш басмас идем,
Шул юлларны кабат үткәндә.

Эх дусларым, сезгә боерып әйтәм,
Ярыгызга яра салмагыз.
Вакытында кичерә белү кирәк,
Сөйгәннәргә ачу тотмагыз!
```
