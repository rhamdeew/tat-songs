# Оригинал

### Илсур Хөснетдинов / Ганс Сәйфуллин - Бәхет дагасы

```
Иярле ат, кыңгыраулы дуга
Безнең яшьлек, яшьлек иртәсе.
Шул атларга утырып, сөйгән ярың белән
Еракларга килә китәсе.
Шул атларга утырып, сөйгән ярың белән
Еракларга килә китәсе.

Кыңгыраулы атлар туйга чаба,
Дугаларга сөлге уралган.
Сөлгеләрен әнкәң, әнкәң чиккән аның,
Сиңа шулай бәхет юраган.
Сөлгеләрен әнкәң, әнкәң чиккән аның
Сиңа шулай бәхет юраган.

Иярле ат, кыңгыраулы дуга,
Дагасына бәхет чорналган.
Шул даганы төштә, төштә тапкан идем,
Таң атканчы дагам югалган.
Шул даганы төштә, төштә тапкан идем,
Таң атканчы дагам югалган.

Төшеп калган бәхет дагасын
Әнкәң белән эзләп табарсың.
Баскан саен изге, изге дагалар
Бәхет-шатлык теләп калалар.
Баскан саен изге, изге дагалар
Бәхет-шатлык теләп калалар.
```
