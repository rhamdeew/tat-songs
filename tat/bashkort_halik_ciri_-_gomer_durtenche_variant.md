# Оригинал

### Башкорт халык җыры - Гомер (дүртенче вариант)

```
Елтыр да гына елтыр ут күренә
Юлаучыннан калган бер күмер,
Юлаучыннан калган бер күмер.
Җырсыз да гына тормышның яме юк,
Җырлап яшәүләре бер гомер,
Ыргый-сикерә килә күк юргам.
Ыргый-сикерә килә күк юргам,
Җитсен сезнең гомер күп елга.

Аягыңа кигән итегеңнең
Тузган микән әллә олтаны,
Тузган микән әллә олтаны?
Утта да гына янмас, суда батмас
Ир-егеткәйләрнең солтаны,
Мәчет манарасы югары.
Мәчет манарасы югары,
Сездән якын дуслар юк әле.
```
