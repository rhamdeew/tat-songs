# Оригинал

### Эльмира Шәрифуллина / Фәрит Хатыйпов - Сине бик сагынам

```
Сагынам, сине бик сагынам,
Хәтерем гөл сиңа кагыла:
Алтынсу көземдә
Таң чыгы күзеңдә
Буй-сының, уй-йөзең чагыла.

Сагынам, сине бик сагынам:
Уйларым гарешкә агыла.
Тургайлы тугайда,
Син шунда бугай ла…
Кавышсак иде без тагын да.

Сагынам, сине бик сагынам,
Күкләргә ашыйм, дип, кагынам.
Чынлыкта үлсәң дә,
Мин шуны белсәм дә
Сагынам, син дип, кабынам.
```
