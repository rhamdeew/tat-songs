# Оригинал

### Резеда Вәлиева / Харис Ибрагимов - Бишек җыры (беренче вариант)

```
Кояш ятты бишегенә,
Күзләрен йомды.
Кошлар да сайрамый инде.
Чишмәләр тынды.
Яфраклар да серләшмиләр —
Йоклады каен.
Без дә йоклыйк,
Бәү-бәү итик,
Улым, улкаем…
Песиең дә арыды әнә,
Боегып калды.
Башын куеп тәпиенә,
Йокыга талды.
Аюыңа борынын төртеп
Йоклый аткаең.
— Без дә йоклыйк.
Бәү-бәү итик,
Улым, улкаем…
Тәрәзәдән ай да безгә
Карый үрелеп.
— Малайларым нишли анда,
Йоклыймы? — диеп,
Әкрен басып Йокы бабай
Йөри өй саен…
Әйдә, йоклыйк,
Бәү-бәү итик,
Улым, улкаем…
```
