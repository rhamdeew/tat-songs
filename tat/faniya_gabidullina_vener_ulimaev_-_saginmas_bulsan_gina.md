# Оригинал

### Фәния Габидуллина / Венер Улимаев - Сагынмас булсаң гына

```
Уңышлы юлга чыккансың
Сине табышлы иттем.
Бәхет эзлеп йөрәгенсеңдер
Ә мин сагышлы иттем.

Күзләремә туры кара
Абынмас булсаң гына.
Үткәнеңә сагышланып
Сагынмас булсаң гына.

Чәчәк өзик, исләреннән
Исермәс булсак кына.
Бергә чакта моңсу хисләр
Кичермәс булсаң гына.

Күзләремә туры кара
Абынмас булсаң гына.
Үткәнеңә сагышланып
Сагынмас булсаң гына.

Күз нурымны күпәр итәм
Бергәләп үтсәк кенә.
Сикәлтәле юлларыңда
Тип — тигез итсәк кенә.

Күзләремә туры кара
Абынмас булсаң гына.
Үткәнеңә сагышланып
Сагынмас булсаң гына.
```
