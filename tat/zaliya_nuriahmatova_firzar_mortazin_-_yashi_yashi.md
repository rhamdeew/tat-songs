# Оригинал

### Залия Нуриәхмәтова / Фирзәр Мортазин - Яши, яши…

```
Дөнья дибез, тормыш дибез,
Язмыштан зарланабыз…
Уңышларны саный-саный…
Ялгыштан акланабыз.

Эх, уза гомерләр…
Айлар, еллар, мизгелләр…
Яши-яши аңлыйбыз шул,
Без мәңгелек түгелләр.

Акча дибез, байлык дибез,
Һаман тормыш куабыз…
Бәхет эзләп, сөю эзләп,
Яшәү тәмен тоябыз.

Эх, уза гомерләр…
Айлар, еллар, мизгелләр…
Яши-яши аңлыйбыз шул,
Без мәңгелек түгелләр.

Дуслар белән, туган белән,
Бик кадерләп вакытны
Яшик әле, дөньяларда,
Бүлеп кайгы-шатлыкны.

Эх, уза гомерләр…
Айлар, еллар, мизгелләр…
Яши-яши аңлыйбыз шул,
Без мәңгелек түгелләр.
```
