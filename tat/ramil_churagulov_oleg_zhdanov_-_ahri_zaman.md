# Оригинал

### Рамил Чурагулов / Олег Жданов - Ахры заман

```
Дөрләп яна дөнъя,
Бакчалар әйләнде, ком сүлгә.
Ахры заман килде,
Каннарсымнарлар җитми кешегә.
Заман ахры килде,
Ата — улны, ана — кызны белми.
Кешегә ни булды?
Талаш — ызгышлар, утлы — сугышлар кабынды.

Бу ахры заман?
Кем яклар соң  толларны?
Үлем чалгы кайрый
Кем саклар йортларны?
Күктән кергән изге көръән күрсәтер юлны
Гөнаһлыны юлга салыр.

Заман ахры килде,
Кешеләр әйләнде мәхлүккә.
Хурладылар динне
Имансын әхлак яткан чүплекта.
Заман ахры килде,
Җәнлек булып кеше кеше ашый.
Югалтык без юлны
Гөнаһлы җаннар куркмый Ходайдан.
Ни булды?

Бу ахры заман?
Кем яклар соң  толларны?
Үлем чалгы кайрый
Кем саклар йортларны?
Күктән кергән изге көръән күрсәтер юлны
Гөнаһлыны юлга салыр.
```
