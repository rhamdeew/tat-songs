# Оригинал

### Наҗар Нәҗми / Заһир Исмәгыйлев - Кичке Агыйдел

```
Җилкенсез көймә шикелле,
Ай йөҙә талгын (ялгыз) гына.
Ай янында җем-җем иткән
Йолдыз да йолдыз гына
Ай йөзә ялгыз гына,
Ай йөзә ялгыз гына,
Ай йөзә.

Кала утлары җемелдәп
Идел өстенә төшә.
Әйтерсең алар сузылып
Агыйдел суын эчә,
Идел өстенә төшә,
Идел өстенә төшә,
Агыйдел.

Серләшә кебек бер җиһан,
Бар агачлар тын гына.
Коча кебек Димне Идел
Әкерен генә, тын гына,
Әкерен генә, тын гына,
Әкерен генә, тын гына,
Тын гына.
```
