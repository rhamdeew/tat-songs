# Оригинал

### Разил Вәлиев / Илгиз Закиров - Бәхет нигә ерак?

```
Уза гомер, уза
Искән җил кебек.
Язлар килгән саен
Куям җилкенеп.

Төнем нигә кара,
Йолдыз ник дәшә?
Бәхет нигә ерак,
Кайгы янәшә?

Уза гомер синсез,
Уза язларым.
Җанны телә һаман
Сөю газабы.

Төнем нигә кара,
Йолдыз ник дәшә?
Бәхет нигә ерак,
Кайгы янәшә?

Уза гомер, уза, —
Җитәр көзләрем.
Гомер буе, ахры,
Сине эзләрмен.

Төнем нигә кара,
Йолдыз ник дәшә?
Бәхет нигә ерак,
Кайгы янәшә?
```
