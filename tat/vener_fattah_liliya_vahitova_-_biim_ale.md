# Оригинал

### Венер Фәттах / Лилия Вахитова - Биим әле!

```
Яшәү рәхәт бии белсәң,
Дәртләнеп алсаң.
Күңелеңне җырлы, моңлы
Шатлыкка мансаң.

Кушымта:
Ә мин биим, биим әле,
Күңелем бии.
Бии белсәң, күңел бер дә
Зарлана белми.
Ә мин биим, биим әле,
Дәртләнә күңел.
Бии белгән дәртле күңел
Моң-зарлы түгел.

Тормыш матур: бии белсәң,
Моңнан җыр үрсәң,
Биеп яшим, мин сәламәт,
Бәхетле дисәң!

Дөнья күркәм: биеп йөрсәң,
Яшәрә белсәң,
Шау, шат булып биеп яшим
Чөнки, дип, исән!
```
