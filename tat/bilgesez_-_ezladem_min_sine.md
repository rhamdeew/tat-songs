# Оригинал

### Билгесез - Эзләдем мин сине

```
Мин өзмәдем болын чәчәкләрен,
Калыр диеп ялгыш эзләрем.
Кемнедер мин көттем еллар буе
Кемнедер мин һаман эзләдем.

Эзләдем, мин сине эзләдем,
Таба алмам дип өмет өзмәдем.
Сине көтеп уззды көннәрем,
Озын-озын яшьлек төннәрем.

Очрата алсам сине юлларымда
Болыннарга алып китәрмен.
Бәхет чәчәкләрен шунда җыеп
Сиңа аларны бүләк итәрмен.

Эзләдем, мин сине эзләдем,
Таба алмам дип өмет өзмәдем.
Сине көтеп уззды көннәрем,
Озын-озын яшьлек төннәрем.

Сине алып чәчәкләре белән,
Ерак-еракларга китәрмен.
Очратмасам сине юлларымда
Гомерем буе сине көтәрмен.

Эзләдем, мин сине эзләдем,
Таба алмам дип өмет өзмәдем.
Сине көтеп уззды көннәрем,
Озын-озын яшьлек төннәрем.
```
