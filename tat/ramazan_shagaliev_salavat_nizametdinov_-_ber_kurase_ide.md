# Оригинал

### Рамазан Шагалиев / Салават Низаметдинов - Бер күрәсе иде

```
Бер күрәсе иде,
Бер үтәсе иде
Синең белән яшьлек эзеннән.
Таң аттырып бергә,
Куркып тормый бер дә,
Олыларның әйткән сүзеннән.
Бер күрәсе иде,
Бер үтәсе иде!

Бер күрәсе иде,
Бер үрәсе иде
Такыялар сиңа икәүләп.
Карап туймас идем,
Ару тоймас идем,
Көне буе сине иркәләп.
Бер күрәсе иде,
Бер үрәсе иде!

Бер күрәсе иде,
Бер үбәсе иде,
Миңнәреңнән назлап, күзеңнән.
Сөю җырын көйләп,
Эч серемне сөйләп,
Төшермичә сине теземнән.
Бер күрәсе иде,
Бер үбәсе иде!
```
