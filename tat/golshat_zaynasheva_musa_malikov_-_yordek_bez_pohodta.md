# Оригинал

### Гөлшат Зәйнәшева / Муса Маликов - Йөрдек без походта

```
Мендек биек тауга, күрмәдек бөркет,
Бөркет качып беткән, кешедән куркып.
Кердек карурманга, күрмәдек төлке,
Төлке эзләп йөрү хәзер бит көлке.

Көне буе шулай
Йөрдек без походта.
Вакыт әрәм итеп
Һәм арып юкка.

Йөрдек болын буйлап, күрмәдек кошлар,
Кайтмаганнар безгә канатлы дуслар.
Эзләп йөрдек икән соң нигә генә,
Калган, ахры, алар мультфильмдә генә.

Көне буе шулай
Йөрдек без походта.
Вакыт әрәм итеп
Һәм арып юкка!
```
