# Оригинал

### Шәмсия Җиһангирова / Фәрид Низамиев - Сөеп калыйк ятлар алганчы

```
Һаваларда очкан кошкайларга
Карап калдым күзем талганчы.
Эх! Сөеп калыйк ятлар алганчы шул!
Сөеп калыйк ятлар алганчы…

Ул булмасмы диеп уңга киттем
Басу капкаларын чыгуга.
Эх! Без сөйгәнгә ятлар кызыга шул!
Без сөйгәнгә ятлар кызыга…

Басып кына торып карап калдым
Гомеркәйләр үтеп барганны.
Эх! Сизеп тордым ятлар алганны шул!
Белеп тордым ятлар алганны…

Чикләвекләр ватып ашап калыйк
Аҗаганнар гына алганчы!
Эх! Безне сөючеләр бар анысы!
Безне сөючеләр бар ансы!

Япь-яшь кенә тайның билкәйләре
Авыр гына йөктән бөгелә.
Эх! Ярым уйлап җаннар өзелә шул!
Ярым уйлап җаннар өзелә…
```
