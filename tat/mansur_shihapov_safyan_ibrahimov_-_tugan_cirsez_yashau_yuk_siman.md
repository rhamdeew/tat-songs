# Оригинал

### Мансур Шиһапов / Сафьян Ибраһимов - Туган җирсез яшәү юк сыман …

```
Ямьле авыл таңын,
Каеннарның шавын,
Май кояшы нурын юксынам.
Мин кырларга гашыйк,
Авылым, сагынам шашып,
Туган җирсез яшәү юк сыман.

Тугайларның нурын,
Тургайларның моңын
Сагынып, йөрәк, гомер тиз ага.
Шәһәр — бәхет нуры,
Ә мин авыл улы —
Җирне сагынып күңел сызлана.

Көрәшләрдә күпме
Данлы юллар үтте
Буразнада үскән батырлар.
Алар җырын җырлап,
Туган җирне зурлап
Шаулашалар моңсу каеннар…
```
