# Оригинал

### Энҗе Мөэминова / Роберт Андреев - Бишек җыры (сигезенче вариант)

```
Мендәреңә, юрганыңа
Ак тыш кидерәм.
Җырлый-җырлый күңелеңә
Аклык иңдерәм.

Кушымта:
Йокла инде, йокла, сабый,
Ак төшләр күреп,
Ак болытлар арасында
Уйнап йөр син, уйнап йөр син,
Ак колын кебек.
Әлли, бәлли, бәү…

Ак бишеккә салам сине,
Ак чаршау корам.
Аклык белән, сафлык белән,
Тулсын бар дөньяң!
```
