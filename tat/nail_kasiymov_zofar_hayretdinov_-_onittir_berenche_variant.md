# Оригинал

### Наил Касыймов / Зөфәр Хәйретдинов - Оныттыр (беренче вариант)

```
Үткәннәр турында сорама,
Кагылма авырткан ярама.
Көнләмә, ул диеп каршымда,
Яратсаң оныттыр барсын да.

Онытырга телимен күптәннән,
Күтәрмә пәрдәсен үткәннең,
Калсын ул булып бер ялгышым,
Була алсаң, бул минем язмышым.

Үткәннәр турында сорама,
Ярамнан әле дө кан тама.
Ялгышлар күтәрмәм кабаттан,
Ышандыр, оныттыр, яратсаң.
```
