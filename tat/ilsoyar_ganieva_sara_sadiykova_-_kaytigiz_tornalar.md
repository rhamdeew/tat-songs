# Оригинал

### Илсөяр Ганиева / Сара Садыйкова - Кайтыгыз, торналар

```
Күл карап озатты
Торналар китүен;
Сагынып көтә ул
Кабат яз җитүен.

Чылбырдай тезелеп,
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?

Йөрмәгез боегып,
Каңгырып чит илдә;
Оялар корырсыз
Яңадан шул күлдә.

Чылбырдай тезелеп,
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?

Туган як күле киң,
Күңелләр күлдәй киң;
Туган җир тормышы
Шул гүзәл күлгә тиң.

Чылбырдай тезелеп.
Кайтыгыз, торналар;
Бу күлдәй камышлы
Гүзәл күл кайда бар?
```
