# Оригинал

### Ибраһим Биектаулы / Раиф Балаһутдинов - Чиксез рәхмәтем

```
Минем күңелгә
Нур булып кердең,
Бөтен гомергә
Киң сулыш бирдең.

Яшим хисләрнең
Олысын тоеп,
Синең йөрәкнең
Җылысын тоеп.

Җирнең тылсымы
Синдәгенәдер,
Таудай зур бәхет
Миндәгенәдер.

Синең елмаю —
Дөнья рәхәте.
Сиңа, бәгырем,
ЧИКСЕЗ РӘХМӘТЕМ!
```
