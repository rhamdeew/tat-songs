# Оригинал

### Гөлшат Зәйнәшева / Ганс Сәйфуллин - Акбүз атым (беренче вариант)

```
Акбүз атым, акбүз атым,
Ак ялларың — ак канатын.
Кая болай ашкынасың,
Сулышларың — ярсу ялкын.

Акбүз атым, ак канатым,
Очыйк бергә далаларга
Хыялдагы далаларга,
Әкияттәге калаларга.

Акбүз атым, акбүз атым,
Ак аягың ник дагалы?
Аңламыйсың, минем йөрәк
Нигә сызлый, ник яралы.

Акбүз атым, ак канатым,
Очыйк бергә далаларга
Хыялдагы далаларга,
Әкияттәге калаларга.

Акбүз атым, акбүз атым.
Ялың сибелә билләреңә.
Мәңге синнән калмас идем,
Алсаң мине иңнәреңә.

Акбүз атым, ак канатым,
Очыйк бергә далаларга
Хыялдагы далаларга,
Әкияттәге калаларга.
```
