# Оригинал

### Фая Агадуллина / Фирүзә Хафизова - Үпкәләмим

```
Кышлар узды, язлар узды,
Җәйләр дә үтеп бара.
Сагынасыңмы дип сорама,
Сагынмый түзеп кара.

Үпкәләмим инде ничә
Җәйләрнең үткәненә.
Тормыш юлын мәхәббәтсез
Һәм синсез үткәнемә.

Гомеремдә бер ялгыштым
Сине ятларга биреп.
Бер генә эндәшсәң иде
"Сөеклем" диеп, килеп.

Үпкәләмим инде ничә
Җәйләрнең үткәненә.
Тормыш юлын мәхәббәтсез
Һәм синсез үткәнемә.

Үпкәләмим гомеремнең
Мин синсез үткәненә.
Уйларымда гына кайтам
Яшьлеккә — үткәнемә.

Үпкәләмим инде ничә
Җәйләрнең үткәненә.
Тормыш юлын мәхәббәтсез
Һәм синсез үткәнемә.
```
