# Оригинал

### Халисә Мөдәрисова / Мөдәрис Газетдинов - Җәй уртасы

```
Гомеремнең җәй уртасы
Бар югалту, табышлары.
Күкрәүләре күкри әле,
Салмаклана агышлары.

Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.
Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.

Гомеремнең җәй уртасы,
Кичләремнең җиләслеге,
Ләкин инде сагындыра
Яшьлегемнең иләслеге.

Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.
Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.

Сагындыра ташкын агым,
Ярсу күкрәү, яшеннәре.
Нәкъ уртасы, соңы түгел
Синең сөю, тәүге бөрең.

Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.
Гомеремнең җәй уртасы,
Чәчкәләре эре әле.
Тик юк инде өмет гөлем,
Ачылмаган бөреләре.
```
