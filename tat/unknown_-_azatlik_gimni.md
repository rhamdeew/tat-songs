# Оригинал

### Мәхмүт Хөсәен / Шакир Мәҗитов - Азатлык гимны

```
Туганнар, тыңлагыз,
Кушылып җырлагыз,
Безнең җыр бәйсезлек зурында.
Азатлык байрагы,
Чын хаклык байрагы
Татар халкының үз кулында.

Дошманнар, кызмагыз
Без ирек куймабыз
Татарның бәхетен буарга!
Халыклар мул яш-ар,
Кояш гел нур чәчәр,
Татарстан бәйсез булганда!

Азатлык турында
Буыннан-буынга
Шаулар җыр, таралыр безнең дан.
Түгелмәс кайнар кан,
Сүрелмәс алсу таң,
Яшәсен бәйсез Татарстан!
```
