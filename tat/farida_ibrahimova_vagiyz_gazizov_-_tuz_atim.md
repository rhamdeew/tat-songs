# Оригинал

### Фәридә Ибраһимова / Вәгыйзь Газизов - Түз, атым

```
Әйдә, атым, алга табан атлыйк,
Чыда инде, азга булса да.
Юл газабы — гүр газабы, диләр,
Без үтәсе озын юлларда.

Тәгәрмәчләр алга тәгәриләр,
Сикертмәгез, җайлы барыгыз.
Фәрештәләр канаты астында,
Уңып торсын барыр юлыбыз.

Ярсып-ярсып атым чабып бара,
Гүя аңлагандай ул мине.
Ярдәмеңне сал Син, газиз Аллам,
Була күр син, атым, түземле.

Дөнья баса, тормыш кыса безне,
Җигеләбез атлы арбалар.
Күпме чакрымнар калган соң икән
Без барасы алдагы юллар.

Адәм башы ниләр генә күрми,
Чыдый, түзә йөрәк барсына.
Сабыр канатларыбыз сынмасын,
Түзик бергә, берүк ярсынма.

Тәгәрмәчләр алга тәгәриләр,
Сикертмәгез, җайлы барыгыз.
Фәрештәләр канаты астында,
Уңып торсын барыр юлыбыз.
Дөнья баса, тормыш кыса безне,
Җигеләбез атлы арбалар.
Күпме чакрымнар калган соң икән
Без барасы алдагы юллар.
Күпме чакрымнар калган соң икән
Без барасы алдагы юллар…
Без барасы алдагы юллар…
```
