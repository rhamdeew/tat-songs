# Оригинал

### Разил Вәлиев / Резеда Ахиярова - Соңгы хат

```
Оныт, зинһар өчен, оныт инде, —
Утка яктым синең хатларны.
Утка яктым ялган, ялган вәгъдәләрне,
Утка яктым бергә чакларны.

Оныт, зинһар өчен, оныт инде…
Җир тарата учак көлләрен.
Җилгә очты бөтен, бөтен хыялларым,
Утта янды гашыйк көннәрем.

Оныт, зинһар өчен, оныт инде…
Салма мине утлы күмергә.
Кайтып булмый сүнгән, сүнгән мәхәббәткә,
Кайтып булмый узган гомергә.
```
