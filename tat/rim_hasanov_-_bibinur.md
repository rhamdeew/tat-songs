# Оригинал

### Рим Хәсәнов - Бибинур

```
Авылда үссәң дә син,
Кала кызы кебексең.
Атлауларың, карашың
Югалтты егет башын.

Бибинур, Бибинур,
Кояш сибә, чәчә нур.
Бибинур чибәр икән,
Бергә булырбыз микән?

Бибинурым һәр атна
Кайта үз авылына
Нигә еш кайта икән,
Кемне сагына икән?

Бибинур, Бибинур,
Кояш сибә, чәчә нур.
Бибинур чибәр икән,
Бергә булырбыз микән?

Кар эреп кышлар бетәр.
Язлары килеп җитәр.
Яз кыюрак булырмын,
Серләремне ачармын.

Бибинур, Бибинур,
Кояш сибә, чәчә нур.
Бибинур чибәр икән,
Бергә булырбыз микән?
```
