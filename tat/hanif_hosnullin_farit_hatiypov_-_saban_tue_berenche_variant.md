# Оригинал

### Хәниф Хөснуллин / Фәрит Хатыйпов - Сабан туе (беренче вариант)

```
Туганнар кайткан сагынып,
Ерактан дуслар килгән.
Сабан туе шулай безне
Чакыра бөтен илдән.

Сабан туе, сабан туе,
Изге бәйрәм илемдә.
Ак сөлгеләр уйнап тора
Көрәшчеләр билендә.

Уйната да, биетә ул,
Җырлата, гөрләштерә.
Сабан туе күңелләрне
Мәңгегә берләштерә.

Сабан туе, сабан туе,
Изге бәйрәм илемдә.
Ак сөлгеләр уйнап тора
Көрәшчеләр билендә.

Сабан тургайлары җыры
Яңгыраган чакларда —
Илне иңләп бәйрәм килә
Безнең туган якларга.

Сабан туе, сабан туе,
Изге бәйрәм илемдә.
Ак сөлгеләр уйнап тора
Көрәшчеләр билендә.
```
