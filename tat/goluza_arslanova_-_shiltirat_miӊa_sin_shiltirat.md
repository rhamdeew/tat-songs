# Оригинал

### Гөлүзә Арсланова - Шылтырат, миӊа, син, шылтырат

```
Шылтырат, миӊа, син, шылтырат,
Минем хәлкәйләремне сорап,
Ярсу күӊелкәемне юат,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.

Шылтырат, миӊа, син, шылтырат,
Тулган күӊелкәеӊне бушат,
Хәлкәйләреӊне генә сөйләп,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.

Озын-озын юлларны урап,
Син килерсеӊ хәлемне сорап,
Ышанам очрашуга юрап,
Шылтырат, миӊа, син, шылтырат.

Алло, алло, мин тыӊлыймын,
Тауышыӊны сагынганмын.
```
