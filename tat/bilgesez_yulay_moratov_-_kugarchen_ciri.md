# Оригинал

### Билгесез / Юлай Моратов - Күгәрчен җыры

```
Гөлдер, гөлдер,
Хәзер язгы көндер.
Бирешмәдек сыуыкка,
Кыш үткәрдек кыекта.

Гөлдер, гөлдер,
Яз, безне сөендер.
Тышта гөрләп йөрербез,
Рәхәт үтәр көнебез.

Гөлдер, гөлдер,
Гөлләр үсәр көндер.
Һәр чак күңелле булсын,
Иртәләр тыныч тусын.
```
