# Оригинал

### Дилә Булгакова / Наил Шәймарданов - Ай гүзәлем, Айгизәм

```
Бөрлегәнгә бергә бардык
Тулмады чиләкләрем.
Сагындырыр, юксындырыр
Бөрлегән җыйган мәлем.

Өлгергән бөрлегәннәр,
Җыяр соң тагын кемнәр,
Ай гүзәлем, Айгизәм.

Ишелеп пешкән бөрлегән
Учларымнан коелды.
Эх, беләсе иде синең
Хыялыңны, уеңны.

Өлгергән бөрлегәннәр,
Җыяр соң тагын кемнәр,
Ай гүзәлем, Айгизәм.

Кемнәр өчен җыйдым икән
Бөрлегәннән гөлләмә?
Бергенәм, дип эзләгәнем,
Бөрлегәнем синме әллә?

Өлгергән бөрлегәннәр,
Җыяр соң тагын кемнәр,
Ай гүзәлем, Айгизәм.

Үпкәләмә Айгизәм,
Айым, кояшым дисәм,
Ай гүзәлем, Ай гүзәлем
Ай гүзәлем, Айгизәм.

Өлгергән бөрлегәннәр,
Җыяр соң тагын кемнәр,
Ай гүзәлем, Айгизәм.
```
