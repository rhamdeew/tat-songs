# Оригинал

### Билгесез - Онытма мине

```
Соң булса булсын,
Таңнар яңарсын,
Син шул таңнарда
Чулпан булып янарсың.
Икәү сүтәрбез гомер карасын,
Тик ярату калсын.
Минсез калсаң —
Төннәр йотар сине.
Онытма мине, онытма мине.

Онытма мине,
Кул жылым яши hәр болытта,
Онытма мине,
Утларда яндыр, тик онытма,
Тик онытма.
Онытма мине,
Бик якын тоям бәхет исен.
Онытма мине,
Юл эзлим сиңа кайтыр өчен,
Кайтыр өчен.

Соң булса булсын,
Язлар калсыннар.
Шул язда сине минем назым каршылар.
Бергә үрербез бәхет толымын,
Тик ярату булсын, булсын, онытма.
Минсез калсаң —
Көзләр йотар сине.
Онытма мине, онытма мине.

Онытма мине,
Кул жылым яши hәр болытта,
Онытма мине,
Утларда яндыр, тик онытма,
Тик онытма.
Онытма мине,
Бик якын тоям бәхет исен.
Онытма мине,
Юл эзлим сиңа кайтыр өчен,
Кайтыр өчен.

Үкенмә син, кайгырма…

Минсез калсаң —
Төннәр йотар сине.
Онытма мине, онытма мине.

Онытма мине,
Кул жылым яши hәр болытта,
Онытма мине,
Утларда яндыр, тик онытма,
Тик онытма.
Онытма мине,
Бик якын тоям бәхет исен.
Онытма мине,
Юл эзлим сиңа кайтыр өчен,
Кайтыр өчен.
```
