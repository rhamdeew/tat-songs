# Оригинал

### Рәмзия Вәлиуллина / Эльвира Хаматнурова - Әтиемнең йолдызы

```
Әтиемнең йолдызы мин
Ул миңа шулай дәшә
Тәмле кәнфит, чикләвек,
Уенчык алып кайта.

Бер матур җыр
Бүләк итәм
Әтием җыр ярата:
Ля ля ля…

Кулымнан тотып әтием
Көндә бакчага илтә
Үзенең эше беткәч
Алырга килеп җитә.

Бер матур җыр
Бүләк итәм
Әтием җыр ярата:
Ля ля ля…

Кичен әкият укыйбыз
Әтием белән бергә
Бик яратам әтиемне
Өстеннән төшмим бер дә

Бер матур җыр
Бүләк итәм
Әтием җыр ярата:
Ля ля ля…
```
