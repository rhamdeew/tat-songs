# Оригинал

### Галия Сафиуллина / Айнур Сәйфиев - Рәхмәтлемен сезгә, газизләрем

```
Бу дөньяның сихри матурлыгын,
Җылы кояш, аймы, йолдызны,
Әткәй-әткәй, яхшы тормыш биреп,
Күрсәттегез миңа бар назны.

Исән булыгыз сез, әткәй-әнкәй, —
Минем сезгә изге теләгем.
Җырлы гомер бүләк иттегез сез,
Бу җырым ул — сезгә бүләгем.

Сезнең белән мин бәхетле җирдә,
Яшәсәгез — дөньям түгәрәк.
Илһам алам нурлы йөзегездән,
Моң чишмәсен эчәм чүгәләп.

Исән булыгыз сез, әткәй-әнкәй, —
Минем сезгә изге теләгем.
Җырлы гомер бүләк иттегез сез,
Бу җырым ул — сезгә бүләгем.

Рәхмәтлемен сезгә, газизләрем,
Үлчәүләргә сыймас бурычым.
Пар күгәрченнердәй яшәсәгез,
Матур булыр минем тормышым.

Исән булыгыз сез, әткәй-әнкәй, —
Минем сезгә изге теләгем.
Җырлы гомер бүләк иттегез сез,
Бу җырым ул — сезгә бүләгем.
```
