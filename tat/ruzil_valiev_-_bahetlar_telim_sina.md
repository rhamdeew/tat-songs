# Оригинал

### Рузил Вәлиев - Бәхетләр телим сиңа

```
Күпме вакыт узды инде
Бергә була (а)лмаганга.
Сиңа булган хисләремне
Язам мин хатларымда,
Язам мин хатларымда.

Ноктасын да куяр идем,
Оныту авыр миңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа.

Күңелемне телгәлисең,
Өздереп йөрәгемне.
Хәзер син башка яр белән,
Оныттың сөйгәнеңне,
Оныттың сөйгәнеңне.

Ноктасын да куяр идем,
Оныту авыр миңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа.

Аерылып карамыйча
Кадерен белмибез шул.
Бер сындырган ипекәйне
Ябыштырып булмый шул,
Ябыштырып булмый шул.

Ноктасын да куяр идем,
Оныту авыр миңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа…
Ноктасын да куяр идем,
Оныту авыр миңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа.
Бөтен йөрәгемнән алып
Бәхетләр телим сиңа…
Бәхетләр телим сиңа…
```
