# Оригинал

### Зөлфия Асылгәрәева / Айнур Сәйфиев - Тик син

```
Күзләреңә карау белән
Җанда туа сөю хисе.
Әйтерсең лә яз(ы)н ата торган
Чәчәкләрнең сихри исе.
Әйтерсең лә яз(ы)н ата торган
Чәчәкләрнең сихри исе.

Күңелләрем сине көтә,
Уйларымда тик син генә.
Чын йөрәктән сине сөйгән
Мин ул, җаным, тик мин генә.
Чын йөрәктән сине сөйгән
Мин ул, җаным, тик мин генә.

Сөюләрем чиксез минем,
Йөрәгемнең син түрендә.
Көйгә салган җырлар сыман
Моңнар булып ул үрелә.
Көйгә салган җырлар сыман
Моңнар булып ул үрелә.

Әйдә, җаным, болыннарга,
Каршы алыйк таң атканын.
Онытылыйк бер мизгелгә,
Күрик әле кояш батканын.
Онытылыйк бер мизгелгә,
Күрик әле кояш батканын.
```
