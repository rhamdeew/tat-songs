# Оригинал

### Венера Мәхмүтова - Өзмә

```
Йөрәгемдә шытты бүген
Мәхәббәт бөреләре.
Яратып суларын сипсәң
Ата хисләр чәчәге.
Йөрәк җылылыгың белән
Ач мәхәббәт таҗларын.
Саф чыклы сөю чәчәген
Синең өчен саклармын.

Өзмә, өзмә, өзмә, өзмә
Мәхәббәт чәчәгемне.
Синнән башка күрә алмыйм
Якты киләчәгемне.
Өзмә, өзмә, өзмә, өзмә,
Өзмә күңелләремне.
Саф чыклы сөю чәчәген,
Өзмә гомерләремне.

Үз чәчәген тапты бүген
Күңелем күбәләге.
Хуш исле сөю исеннән
Башкайларым әйләнде.
Миңа гына бүләк ит син
Йөрәгең җылылыгың.
Адаштырып озак йөрмә,
Миңа илтәр юлларың.

Өзмә, өзмә, өзмә, өзмә
Мәхәббәт чәчәгемне.
Синнән башка күрә алмыйм
Якты киләчәгемне.
Өзмә, өзмә, өзмә, өзмә,
Өзмә күңелләремне.
Саф чыклы сөю чәчәген,
Өзмә гомерләремне.

Икәү кочыйк бәхет күген,
Пар итеп канатларны.
Күзләремнән аңла, иркәм,
Тик сине яратканны.
Бары синең өчен ачам
Йөрәгемнең ишеген.
Син якты киләчәгем,
Син бәхетем бит минем.

Өзмә, өзмә, өзмә, өзмә
Мәхәббәт чәчәгемне.
Синнән башка күрә алмыйм
Якты киләчәгемне.
Өзмә, өзмә, өзмә, өзмә,
Өзмә күңелләремне.
Саф чыклы сөю чәчәген,
Өзмә гомерләремне.
```
