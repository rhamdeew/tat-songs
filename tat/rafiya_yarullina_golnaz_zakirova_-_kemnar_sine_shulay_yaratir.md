# Оригинал

### Рафия Яруллина / Гөлназ Закирова - Кемнәр сине шулай яратыр?

```
Матур җәйге кичтә,
Зәңгәр күл буенда,
Тыңлаганда гармун көйләрен,
Тал-тирәккә кунып
Кошлар сайраганда,
Очраттым мин сине сөйгәнем.

Сөям, сине сөям,
Сөям сине генә,
Беләсеңме, иртә-кичләрен.
Көндез күз алдымда,
Төнлә төшләремдә,
Ашкынадыр сиңа хисләрем.

Синең матурлыктан
Телем югалганны
Сиздеңме икән шушы мизгелдә?
Серле карашларың,
Матур елмаюың,
Мәңгелеккә калды исемдә.

Сөям, сине сөям,
Сөям сине генә,
Беләсеңме, иртә-кичләрен.
Көндез күз алдымда,
Төнлә төшләремдә,
Ашкынадыр сиңа хисләрем.

Очрашуны, иркәм,
Көтеп арып бетәм,
Йөрәккәем шуны аңлатыр.
Мәхәббәт утында
Әгәр янып бетсәм,
Кемнәр сине шулай яратыр?

Сөям, сине сөям,
Сөям сине генә,
Беләсеңме, иртә-кичләрен.
Көндез күз алдымда,
Төнлә төшләремдә,
Ашкынадыр сиңа хисләрем.
Ашкынадыр сиңа хисләрем…
Ашкынадыр сиңа хисләрем…
```
