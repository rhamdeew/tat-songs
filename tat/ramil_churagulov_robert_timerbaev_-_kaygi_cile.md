# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Кайгы җиле

```
Кайгы килсә, сорап тормый,
Килми ишеген кагып.
Бәхет тулы тормышының
Килә тәрәзен ватып.

Мин туган йортның капкасын
Кайгы җиле какмасын.
Әти-әни, туганнарны
Фәрештәләр сакласын!

Әткәй-әнкәй — изге җанның
Түрдә булсын урыны.
Алар безне кеше иткән
Озын булсын гомере.

Мин туган йортның капкасын
Кайгы җиле какмасын.
Әти-әни, туганнарны
Фәрештәләр сакласын!

Кайда барма, кайда булма
Туган йортны ташлама.
Туган нигез ташларыннан
Туган илләр башлана.

Мин туган йортның капкасын
Кайгы җиле какмасын.
Әти-әни, туганнарны
Фәрештәләр сакласын!

Әти-әни, туганнарны
Ходай үзе сакласын!
```
