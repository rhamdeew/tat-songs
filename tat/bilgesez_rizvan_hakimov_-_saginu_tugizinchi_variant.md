# Оригинал

### Билгесез / Ризван Хәкимов - Сагыну (тугызынчы вариант)

```
Мин сагынып киләм сине,
Мин сагынып киләм сине,
Ашкына сиңа җаным.

Сагышлардан сарыларга
Сагышлардан сарыларга
Чагыша язган чагым.

Сагынам синең сүзләрне,
Сагынам синең сүзләрне,
Сагынам карашыңны.

Сизеп торам: колач җәеп,
Сизеп торам: колач җәеп,
Кочаклап аласыңны.

Сагынулар җиңел түгел,
Сагынулар җиңел түгел,
Сагынулар әрнүле.

Син үзең дә сагынасың бит,
Син үзең дә сагынасың бит,
Аңлыйсыңдыр хәлемне.

Сагышланып юлга чыктым,
Сагышланып юлга чыктым,
Йөрәгемне моң баса.

Очраша алмый яшәр идек,
Очраша алмый яшәр идек,
Сагынулар булмаса.

Сагынулар булмаса…
```
