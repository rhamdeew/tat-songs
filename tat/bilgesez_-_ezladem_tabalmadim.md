# Оригинал

### Билгесез - Эзләдем табалмадым

```
Күзләреңне, күзләреңне,
Күзләреңне күзләдең.
Синең зәңгәр күзләреңнән
Мин өметләр эзләдем.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.

Күзләремнән, күзләремне,
Бер минут алмас идем.
Язмыш кая ташласа да
Мин синнән калмас идем.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.

Сөя микән, сөйми микән,
Белсәң икән серләрен.
Белки аның үзенеңдә
Өзеләдер үзәге.

Мин эзләдем даладан,
Син эзләдең каладан.
Кайда син, бәгърем син,
Табалмадым мин.
```
