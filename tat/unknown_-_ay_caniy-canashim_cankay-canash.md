# Оригинал

### Татар халык җыры - Ай җаный-җанашым (Җанкай-җанаш)

```
Атландым ла атның, ай, биленә,
Өч әйләндем Казан каласын.

Елама ла, җаный, ник елыйсың?!
Барыбер бездән аерылып каласың.

Кәккүкләр дә сайрар җир түгел лә,
Былбыллар да моңланыр җир түгел.

Елама ла, җаный, ник елыйсың?!
Йөгереп кенә кайтыр җир түгел.
```
