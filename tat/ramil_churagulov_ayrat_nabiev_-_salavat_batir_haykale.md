# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Салават батыр һәйкәле

```
Салават батыр һәйкәле,
Янында көтәм сине.
Йолдыз уты кебек сүнмәс,
Сиңа аңлатыр хисем.

Таныштыра, кавыштыра,
Баһадирем һәйкәле.
Тапмаган бердән-береңне,
Кич уенга кил әле.
Тик уза гомер заяга,
Кич уенга кил яме.

Салават батыр һәйкәле,
Авылым кебек йокым.
Тик мында гына агыла,
Без сагынган якын моң.

Таныштыра, кавыштыра,
Баһадирем һәйкәле.
Тапмаган бердән-береңне,
Кич уенга кил әле.
Тик уза гомер заяга,
Кич уенга кил яме.

Салават батыр һәйкәле,
Янына кил кич булгач.
Мында сине өзелеп көтә,
Сиңа насыйп кыйгач каш.

Таныштыра, кавыштыра,
Баһадирем һәйкәле.
Тапмаган бердән-береңне,
Кич уенга кил әле.
Узмасын гомер заяга,
Кич уенга кил яме.
```
