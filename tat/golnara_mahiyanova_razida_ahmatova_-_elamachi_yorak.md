# Оригинал

### Гөлнара Махиянова / Рәзидә Әхмәтова - Еламачы йөрәк

```
Ялгыш кына ялгыз булырмын дип,
Бу тормышта уйламадым һич тә.
Күкне болыт каплап алган кебек,
Нурлы көнем әверелде кичкә.

Еламачы, йөрәк, еламачы!
Нигә язмыш балан кебек ачы?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?

Килче, бәхет, килче, ишек шакы!
Юлга чыгыйм мәллә? Үзең чакыр …
Нигә гомер парым белән бергә
Үтми икән язмыш, бергә-бергә?

Еламачы, йөрәк, еламачы!
Нигә язмыш балан кебек ачы?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?

Ал кулымнан, ал, җитәклә мине!
Бәхетемә илтеп тапшыр инде!
Гомер итик яратканнар белән:
Бу тормышта үз парым бар, — беләм!
Ал кулымнан, ал, җитәклә мине!
Бәхетемә илтеп тапшыр инде!
Гомер итик яратканнар белән:
Бу тормышта үз парым бар, — беләм!

Еламачы, йөрәк, еламачы!
Нигә язмыш балан кебек ачы?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?
Нигә юллар парың белән бергә
Үтми икән, язмыш, бергә-бергә?
```
