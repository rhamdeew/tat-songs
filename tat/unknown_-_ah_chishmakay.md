# Оригинал

### Илсөяр Ганиева / Рәис Нәгыймов - Ах, чишмәкәй!

```
Туган телләремдә җырлый-җырлый
Чылтырап аккан чишмә буенда,
Талир тәңкәләрен чылтыратып,
Әнкәм йөргән төсле тоела.

Ах, чишмәкәй, мәңге онытма син,
Әткәм-әнкөм сөйгән телләрне,
Синнән башка нинди ямьнәр калыр,
Кемнәр назлар асыл гөлләрне?

Дәһшәтләрдән соңгы шәмдәй тетрәп,
Исән калган моңлы телебез.
Саф чишмәнең моңнарында туган
Мәңге шиңмәс гүзәл гөле — без!!!
```
