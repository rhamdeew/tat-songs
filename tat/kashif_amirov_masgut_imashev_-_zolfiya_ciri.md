# Оригинал

### Кәшиф Әмиров / Мәсгуть Имашев - Зөлфия җыры

```
Син килдең,
Көтмәгәндә килдең,
Күңелемә ут салдың,
Ут салдың,
Ут салдың.
Айдыр син,
Йолдыздыр син.

Караңгы төнне
Яктыртып,
Яктыртып,
Күңелемә
Нурлар салдың син.
Күңелемә
Моңнар салдың син.

Син килдең,
Көтмәгәндә килдең,
Күңелемә ут салдың,
Ут салдың,
Ут салдың.
Ай булдың,
Йолдыз булдың.

Караңгы
Төнне яктыртып,
Күңелемә аккан нур булдың,
Күңелемнән аккан моң булдың…
```
