# Оригинал

### Татар халык җыры - Өфе (икенче вариант)

```
Өфе кайда? Өфе кайда?
Өфе Урал ягында шул,
Өфе Урал ягында.

Яннарыңа барыр идем,
Бик сагынган чагымда.

Өфенең ак суларында,
Вак балыклар уемда шул,
Вак балыклар уемда.

Алма кебек матур кызлар,
Үсә Өфе буенда.

Өфе дисең дә карыйсың,
Өфе калды бит инде шул,
Өфе калды бит инде.

Яратып сөйгән ярымны,
Җаннар алды бит инде.

Өфеләргә барган чакта,
Яшел урманнар үтәм шул,
Яшел урманнар үтәм.

Сине дә көтәм, сабыр итәм,
Кирәкми миңа бүтән.
```
