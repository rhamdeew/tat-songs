# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Әткәй-әнкәй (беренче вариант)

```
Рәхмәтлемен язмышыма,
Ярый әле жирдә сез бар,
Ярый әле жирдә сез бар.
Кояштай нурлар сибүче,
Айдай нурлар яктыртучы,
Әтием бар, әнием бар.

Үрнәк безгә сезнең тормыш,
Яшисез пар тирәкләрдәй,
Яшисез пар тирәкләрдәй.
Каршылыкларны жиңәргә,
Авырлыкларга тузәргә,
Өйрәттегез, әткәй, әнкәй.

Тырышабыз без яшәргә
Тап төшерми эзегезгә,
Тап төшерми эзегезгә.
Гомерләрнең ин озынын,
Саулыкларның иң корычын,
Әткәй, әнкәй, телим сезгә.

Гомерләрнең ин озынын,
Саулыкларнын ин корычын,
Әткәй, әнкәй телим сезгә.
```
