# Оригинал

### Әнисә Таһирова / Салават Низаметдинов - Могҗизага килә ышанасым

```
Күкне каплап алган кара болыт,
Җирдә салкын томан, җил-явым.
Баш очында кояш балкыр әле,
Могҗизага килә ышанасым.

Күңел күкләрендә сагыш йөзә,
Сүнәр — сүнмәс яна учагым.
Учакларым дөрләп янар әле,
Могҗизага килә ышанасым.

Югалдылар якты хыялларым,
Күпме сүнеп, күпме токанды.
Югалганым бер табылыр әле,
Могҗизага килә ышанасым.
```
