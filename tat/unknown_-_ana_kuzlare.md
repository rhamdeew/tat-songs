# Оригинал

### Заһид Фәйзи / Әнвәр Бакиров - Ана күзләре

```
Балалар бәхете өчен борчылып,
Азмы сызлана ана йөрәге;
Күпме нур сипми ана йөзләре,
Күпме яшь түкми ана күзләре.
Шулай булса да якты, сөйкемле
Ана йөзләре, ана күзләре!

Тирән шатлыклар, кайгы-хәсрәтне
Азмы кичерә ана йөрәге;
Тормыш гөлләре балкып үссенгә
Күпме нур сипми ана йөзләре!…
Ана йөзләре якты, сөйкемле!
Якты, сөйкемле ана күзләре!

Балалар беләме хисчән ананың
Күпме түгелә йөрәк җылысы?!
Газиз баланың ана каршында
Түләп бетмәслек олы бурычы!
Бары яктыга илтә баланы
Изге теләкле ана сүзләре.

Күпме нур сипми, күпме яшь түкми
Ана йөзләре, ана күзләре!…
Саргаймасыннар, нурлы булсыннар
Ана йөзләре, ана күзләре!
```
