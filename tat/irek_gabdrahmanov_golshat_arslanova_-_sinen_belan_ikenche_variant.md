# Оригинал

### Ирек Габдрахманов / Гөлшат Арсланова - Синең белән (икенче вариант)

```
Синең назлы, җылы куеныңда
Көтеп җәйге таңнар атканын,
Бар йөрәгем белән сөяр идем,
Тыңлап кошлар канат какканын.

Синең белән уянасым килә,
Назларыңны килә тоясым.
Сөю тулы, назлы карашыңда
Мәңгелеккә килә каласым!

Таң атканчы бер серләшер идек,
Бер уфтанып, берчә көлешеп.
Искә алып бәхет мизгелләрен,
Үпкәләшеп, аннан килешеп.

Синең белән уянасым килә,
Назларыңны килә тоясым.
Сөю тулы, назлы карашыңда
Мәңгелеккә килә каласым!

Кайнар мәхаббәтне сакларга дип,
Көн дә ярдәм сорыйм Ходайдан.
Безнең тормыш озын бер юл бит ул!
Без бит сөяр өчен яралган!

Синең белән уянасым килә,
Назларыңны килә тоясым.
Сөю тулы, назлы карашыңда
Мәңгелеккә килә каласым!

Синең белән уянасым килә,
Назларыңны килә тоясым.
Сөю тулы, назлы карашыңда
Мәңгелеккә килә каласым!
```
