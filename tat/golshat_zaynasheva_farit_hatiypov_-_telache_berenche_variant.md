# Оригинал

### Гөлшат Зәйнәшева / Фәрит Хатыйпов - Теләче (беренче вариант)

```
Мишә буйлап еракка нур сибә
Теләчедә аткан алсу таң,
Китсәм ерак, җырлыйм мин моңаеп,
Мишә буен өзелеп сагынудан.

Мишә буе, Мишә буе,
Мишә буе тал гына.
Теләченең чишмәләре,
Эчим дисәң, бал гына.
Мишә буйлары тал гына,
Тал яфрагы бал гына,
Баллы яфрагың булырмын,
Яннарымда кал гына.

Ай калыкмаса да айлы кебек,
Теләченең зәңгәр төнендә.
Теләченең үз ае, йолдызы
Балкый микән әллә күгендә?

Мишә буе, Мишә буе,
Мишә буе тал гына.
Теләченең чишмәләре,
Эчим дисәң, бал гына.
Мишә буйлары тал гына,
Тал яфрагы бал гына,
Баллы яфрагың булырмын,
Яннарымда кал гына.

Теләчедә нигә шулай икән? —
Кешеләрнең йөзе яктырак.
Теләченең кырларында үскән
Икмәге дә тәмле, татлырак.

Мишә буе, Мишә буе,
Мишә буе тал гына.
Теләченең чишмәләре,
Эчим дисәң, бал гына.
Мишә буйлары тал гына,
Тал яфрагы бал гына,
Баллы яфрагың булырмын,
Яннарымда кал гына.
```
