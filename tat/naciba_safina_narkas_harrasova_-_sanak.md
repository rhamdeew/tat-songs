# Оригинал

### Нәҗибә Сафина / Нәркәс Харрасова - Санак

```
Замана кыен диләр дә,
Килешәсем килми бит.
Мин үзем дә кичә генә
Шулай уйлый идем бит.
Компьютерга язып эләм
Сөям дигән сүзләрен,
Көтеп торамын санакка
Фотосын ул элгәнен.

Эх, компитр, эй санак,
Утырам минут санап.
Санактагы рәсеменнән
Күземне алмый карап.
Ул да көтә, мин дә көтәм,
Очрашырга җай кирәк.
Бер-беренә бу кадәр дә
Туры килгәннәр сирәк.

Әверелделәр көннәрем
Егетне сагынуга.
Әллә бөтенләй дә тоймый,
Уемда тик ул барын.

Эх, компитр, эй санак,
Утырам минут санап.
Санактагы экраныннан
Күземне алмый карап.
Ул да көтә, мин дә көтәм,
Очрашырга җай кирәк.
Бер-беренә бу кадәр дә
Туры килгәннәр сирәк.

Замана кыен диләр дә,
Килешәсем килми бит.
Мин үземдә кичә генә
Шулай уйлый идем бит.
Егет санакка карый да
Оныткан мине ахры.
Әллә бөтенләй дә тоймый
Янәшәдә мин барын.

Эх, компитр, эй санак,
Утырам минут санап.
Санактагы экраныннан
Күземне алмый карап.
Ул да көтә, мин дә көтәм,
Очрашырга җай кирәк.
Бер-беренә бу кадәр дә
Туры килгәннәр сирәк.
```
