# Оригинал

### Разил Вәлиев / Ренат Ибраһимов - Балачакны сагыну

```
Балачак, әй, шат чак,
Гел сүнмәс учак.
Шушы учак ялкыны җанда
Мәңге, мәңге яначак.

Моңланам мин җыр җырлап,
Юанам сине уйлап.
Балачакка кайтып барам
Гомерем кыры буйлап.

Балачак, әй, шат чак,
Гел сулмас чәчәк.
Чәчәк тотып, төшемә минем
Балачагым керәчәк.

Моңланам мин җыр җырлап,
Юанам сине уйлап.
Балачакка кайтып барам
Гомерем кыры буйлап.
```
