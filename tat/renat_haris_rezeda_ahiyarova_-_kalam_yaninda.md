# Оригинал

### Ренат Харис / Резеда Ахиярова - Калам яныңда

```
Китсәм дә, мин
калам яныңда,
калам яныңда.
Китсәм дә, калам яныңда

бер хатирә булып, сер булып,
ахыры юк моңлы җыр булып,
язны көткән карлы кыр булып,
карлы кыр булып.

күз төбеңә яткан сыр булып —
бер хатирә булып, сер булып,
калам яныңда.

Китсәм дә, мин
калам яныңда,
калам яныңда.
Китсәм дә, калам яныңда

меңнәр арасында үз булып,
күзләреңә баккан күз булып,
иң тылсымлы җылы сүз булып,
җылы сүз булып.

кыш катыш җәй, язлы көз булып,
күз төбеңә яткан эз булып —
меңнәр арасында үз булып,
калам яныңда.

Китсәм дә, мин,
калам яныңда,
калам яныңда.
Китсәм дә, калам яныңда…
```
