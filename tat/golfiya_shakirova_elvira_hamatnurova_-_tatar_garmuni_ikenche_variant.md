# Оригинал

### Гөлфия Шакирова / Эльвира Хаматнурова - Татар гармуны (икенче вариант)

```
Нинди генә гармуннар юк,
Безнең татар халкында.
Тальян,баян,бик күп алар,
Саный китсәң барсында.

КУШЫМТА:
Алтын микән,көмеш микән,
Гармунның төймәләре.
Без белеп бетерми торган,
Әллә бармы серләре.

Җырлата да, биетә дә,
Ял иттерә күңелне.
Йөрәкләрне дәртле иткән,
Татар моңы түгелме.

КУШЫМТА:
Алтын микән,көмеш микән,
Гармунның төймәләре.
Без белеп бетерми торган,
Әллә бармы серләре.

Әйтер сүзе,әйтер сере,
Гармунның телләрендә.
Ул— уйларны канат итә,
Кереп күңелләреңә.

КУШЫМТА:
Алтын микән,көмеш микән,
Гармунның төймәләре.
Без белеп бетерми торган,
Әллә бармы серләре.
```
