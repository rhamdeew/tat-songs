# Оригинал

### Саҗидә Сөләйманова / Сәүбән Чуганаев - Ах, җаныем, Бибисара!

```
Сүс дилбегә, юкә чана,
Алып булмый акчага.
Сырхауларга дәвалар бар,
Сагнуларга юк чара.

Ах, җаныем, Бибисара,
Сагнуларга юк чара.

Яна күмер, уза гомер,
Узган гомер бичара.
Узган гомер берни түгел,
Бәхтең җуйсаң ни чара.

Ах, җаныем, Бибисара,
Бәхтең җуйсаң ни чара.

Карлыгачым муеның ала,
Канат очларың кара.
Без йөрибез кавышалмыйча,
Яшь гомер үтеп бара.

Ах, җаныем, Бибисара,
Яшь гомер үтеп бара.
```
