# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Ник язган язмышлар ярату

```
Моң булып тамганда җаныңа
Сөюнең иң соңгы тамчысы.
Өметнең җепләре өзелә,
Йөрәкнең киселә яртысы.

Ник язган язмышлар ярату,
Исертеп мизгеллек уенда.
Синең дә үз ярың яныңда,
Минем дә үз парым куенымда.

Соң сулыш алганда мәхаббәт,
Гүя җан аерыла тәнемнән.
Без тойган сөюне аңлату,
Килми бит, килми һич хәлемнән.

Ник язган язмышлар ярату,
Исертеп мизгеллек уенда.
Синең дә үз ярың яныңда,
Минем дә үз парым куенымда.

Офыклар артында таралды
Алсуы бәхетле таңнарның.
Әллә син тугансың иртәрәк,
Әллә мин бәхеткә соңардым.

Ник язган язмышлар ярату,
Исертеп мизгеллек уенда.
Синең дә үз ярың яныңда,
Минем дә үз парым куенымда.
Синең дә үз ярың яныңда,
Минем дә үз парым куенымда…
```
