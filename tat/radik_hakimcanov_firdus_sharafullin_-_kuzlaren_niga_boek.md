# Оригинал

### Радик Хәкимҗанов / Фирдүс Шәрәфуллин - Күзләрең нигә боек?

```
Әллә кояшың сүндеме,
Күзләрең нигә боек?
Күңел күгемне биләгән
Сагышмы әллә болыт?

Күзләрең нигә боек,
Сагышмы әллә болыт?
Сагышмы әллә болыт,
Күзләрең нигә боек?

Нигә моңсу икән йөзең,
Күзләрең нигә боек?
Былай да тулган күңелгә
Яңгырмы яуа коеп?

Күзләрең нигә боек,
Яңгырмы яуа коеп?
Яңгырмы яуа коеп,
Күзләрең нигә боек?

Йөрәккәем сызланудан
Күзләрең микән боек?
Үпкәләмә, көт син мине,
Килермен язлар булып.

Килермен язлар булып,
Шатлыгың ташыр тулып.
Шатлыгың ташыр тулып,
Килермен язлар булып.
```
