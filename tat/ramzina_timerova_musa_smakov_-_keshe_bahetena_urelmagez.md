# Оригинал

### Рәмзинә Тимерова / Муса Смаков - Кеше бәхетенә үрелмәгез!

```
Бер бересен өзлеп сөйгәннәрне,
Гайбәт утларына салмагыз!
Кагылмагыз, зинһар, пычратмагыз,
Сөю бит ул сафлык — аңлагыз!

Кеше бәхетә үрелмәгез,
Көнләшмәгез, сабыр булыгыз!
Бүген — мин бәхетле, иртәгә — син,
Балкып торсын барыр юлыгыз!

"Кеше сүзе — кеше үтрә …" — диләр,
Бу сүз бик хак дөрес әйтелгән,
Үлгәннәр бит кире кайталмыйлар,
Әйткән сүзгә соңыннан үкенмә!

Кеше бәхетә үрелмәгез,
Көнләшмәгез, сабыр булыгыз!
Бүген — мин бәхетле, иртәгә — син,
Балкып торсын барыр юлыгыз!
```
