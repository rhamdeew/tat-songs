# Оригинал

### Марат Кәбиров / Ф. Әкъсәнов - Бергенәм син минем

```
Язларын, көзләрен
Мин сине эзләдем,
Сиңа тиң ярларны тапмадым.
Алыйм да кулыннан
Гомерләр юлыннан
Бергәләп атлыйк без, аппагым.

Бергенәм син минем,
Өзелеп сөюем
Озайтсын бәхетле мәлләрне.
Вәгъдәләр бирешик,
Яшәргә сөешеп
Гомерләр соңына кадәрле.

Вәгъдәләр нык булсын,
Балкытырлык булсын
Күңелләрне сөю нурыннан.
Авырлыкны җиңеп,
Сөеп һәм сөелеп
Бергә атлыйк гомер юлыннан
```
