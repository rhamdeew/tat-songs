# Оригинал

### Илшат Гыйниятуллин - Сине уйлап янам (икенче вариант)

```
Беркөн күрде сине күзләр,
Гашыйк итте яшел күзләр.
Син күрмәдең мине ул чакта…

Ә мин әле, ә мин һаман
Сине генә уйлап янам,
Син кабызган сөю учакта…

Яшерен генә сине сөеп,
Сагынуларга нихәл түзеп
Яшәгәнмен инде болай да…

Суынмаслар кайнар хисләр,
Җитсәләр дә салкын көзләр.
Син яшисең минем уйларда!
```
