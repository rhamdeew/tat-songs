# Оригинал

### Фәннур Сафин / Зиннур Гыйбадуллин - Тал басмада

```
Җиргә басар-басмас,
Чиләк асар-асмас,
Суга бара кызлар, киленнәр.
Тал басмада калдың,
Өзелеп сөйгән ярым,
Назлы сүзләр калды күңелдә.

Аралардан күпме
Ачы җилләр үтте,
Тик без көткән көннәр килерләр.
Бик сагындым, иркәм,
Очып кайтыйм микән
Таңда искән йомшак җил белән.

Айлар, еллар үтәр,
Кавышыр чак җитәр,
Чәчәк атар тагын сиреньнәр.
Кадерлерәк юктыр
Чәчәкләрдән матур
Сөйгән ярдан, туган җиремнән.
```
