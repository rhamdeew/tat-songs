# Оригинал

### Гөлфия Сәлихова - Яныңда булырмын

```
Мин гел синең янда булырмын,
Миннән башка яши белмәссең.
Мин булырмын күктә янган йолдыз,
Башка йолдызларны күрмәссең.

Мин булырмын челтерәп аккан сулар,
Кояш булып таңны чыгармын.
Таллар булсам җилләр кабат-кабат
Тотып йолкыса да чыдармын.

Мин булырмын көмөш ай-нурлары,
Сузылып тәрәзәңә керермен.
Танырсыңмы мине син аралап,
Сихерләдем диеп көләрмен.

Таңда чыклар булып тамырмын да,
Тирбәтермен йомшак томан да.
Танырсың син мине һәр төн үтеп
Мин якты көн булып туганда,
Мин яҡты көн булып туганда.
Мин гел синең янда булырмын!
```
