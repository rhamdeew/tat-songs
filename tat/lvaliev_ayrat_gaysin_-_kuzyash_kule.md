# Оригинал

### Л.Вәлиев / Айрат Гайсин - Күзяшь күле

```
Күзяшь күле дулкынлана
Күңелем тулган саен.
Ничек итеп коткарыймын
Дулкынга төшкән айны.

Күз яшьләрем тамды суга
Өстәмә күлгә тамчы.
Тамчылардан җыелганга
Күзяшь күле шифалы.

Туздым айга йөрәгемне
Ул алмады, елмайды.
Елмайды да әкерен генә
Күл төбенә югалды.

Күз яшьләрем тамды суга
Өстәмә күлгә тамчы.
Тамчылардан җыелганга
Күзяшь күле шифалы.

Ай дигәнем сагылыш микән
Сагылышмы әллә хисләрем.
Мәхәббәтсез калгач нишләрмен
Ничек үтәр айсыз кичләрем?

Күз яшьләрем тамды суга
Өстәмә күлгә тамчы.
Тамчылардан җыелганга
Күзяшь күле шифалы.
```
