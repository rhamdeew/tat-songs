# Оригинал

### Марат Кәбиров / Чулпан Зиннәтуллина - Бәйрәм җыры (беренче вариант)

```
Авырлыктан гына
Тормый әле дөнья,
Күрсәтсә дә кайчак моң-зарын.
Күңелләргә безнең
Канат куя торган
Бәйрәмнәре дә бит бар аның.

Бәйрәмнәрдә гел дә.
Гел дә бергә-бергә,
Табындашлар, дуслар, туганнар…
Онытмагыз кайчак
Авыр хәлгә калсак,
Табыннарда бергә булганнар.

Күңел сүрелмәсен,
Өмет өзелмәсен,
Йөрәкләрдә бәйрәм сүнмәсен.
Ярдәмләшеп яшик,
Бәйрәмнәргә дәшеп
Зарлыларның сөртик күз яшен.
```
