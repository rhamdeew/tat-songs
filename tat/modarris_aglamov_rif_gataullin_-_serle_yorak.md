# Оригинал

### Мөдәррис Әгъләмов / Риф Гатауллин - Серле йөрәк

```
Миңа баккан күзләреңнән
Үбәм сине шашып…
Үбәм… миннән башкаларга
Булмасын ул гашыйк.
Күзләрең белән берочтан
Керфегеңне үбәм…
Усал күзләр караганда
Алар төшсен түбән.

Йөрәгеңнән үбәр идем,
Күренми шул йөрәк —
Ах шул йөрәк, серле йөрәк,
Яши тирәндәрәк.

Колак яфракларың синең
Кытыклансын, үбәм.
Беркайчан да усал сүзләр
Ишетмәсен миннән.
Күзләреңнән, битләреңнән,
Маңгаеңнан үбәм —
Җыерчыклар алмасыннар
Минем гаеп белән.

Йөрәгеңнән үбәр идем,
Күренми шул йөрәк —
Ах шул йөрәк, серле йөрәк,
Яши тирәндәрәк.
```
