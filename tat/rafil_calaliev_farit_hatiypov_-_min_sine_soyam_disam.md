# Оригинал

### Рафил Җәләлиев / Фәрит Хатыйпов - Мин сине сөям дисәм?

```
Нидән башлап әйтим икән
Бугендә сине курсәм,
Ничек генә тузәрмен
Әгәр дә син юк дисән?

Чынга алырсыңмы икән
Хисләремне стиздерсәм?
Риза булырсыңмы иркәм
Мин сине сөям дисәм?

Сөюгә урын калдырмый,
Дус булыйк диярсеңме?
Минем хисләремә кереп
Чын чынлап сөярсеңме?

Чынга алырсыңмы икән
Хисләремне стиздерсәм?
Риза булырсыңмы иркәм
Мин сине сөям дисәм?

Хисләребез уртак булса
Бәлки синдә сөярсең,
Куңел колым тибрәдереп
Пар килгәнбез диярсең.

Чынга алырсыңмы икән
Хисләремне стиздерсәм?
Риза булырсыңмы иркәм
Мин сине сөям дисәм?
```
