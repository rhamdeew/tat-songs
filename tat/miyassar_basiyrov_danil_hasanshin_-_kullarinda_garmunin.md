# Оригинал

### Мияссәр Басыйров / Данил Хәсәншин - Кулларыңда гармуның

```
Бер сыздырып җибәр әле,
Кулларыңда гармуның.
Күңелгә сеңгән көйләрнең.
Онытыласы юк аның.

Кара атҡа кара дуга,
Кирәк каеш дилбегә.
Егет чаклар җилеп бара,
Тавышың калсын билгегә.

Бер яшлектә, бер картлыкта,
Бездән генә калмасын.
Бармакларың талса — талсың,
Кызлар йоклый алмасын.

Хисләремә ялкыннар сип,
Йөрәк пыскып янмасын!
Безнең хәтер калса — калсын
Кызларныкы калмасын!
```
