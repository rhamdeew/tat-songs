# Оригинал

### Ринат Мөслимов / Марсель Иванов - Яшим бит, яшим әле

```
Берәү килә, берәү китә,
Җир түгәрәк әйләнә.
Бер кайгыны җиңү белән
Икенчесе бәйләнә.

Мәшәкатьләр күп булса да,
Уф, диеп әйтмим әле.
Авыр булса да түзәмен,
Яшим бит, яшим әле.

Эзли-эзли табып була
Дөньяның җайлы җирен.
Яши-яши ачып була
Тормышның күпме серен.

Мәшәкатьләр күп булса да,
Уф, диеп әйтмим әле.
Авыр булса да түзәмен,
Яшим бит, яшим әле.

Мең тапкырлар үкенәсең
Бер гөнаһ кылган өчен.
Кешеләр, сокланып карап,
Менә бит, яши, дисен.

Мәшәкатьләр күп булса да,
Уф, диеп әйтмим әле.
Авыр булса да түзәмен,
Яшим бит, яшим әле.
```
