# Оригинал

### Нәби Дәүли / Александр Ключарев - Онытма (беренче вариант)

```
Гел — сине уйлыйм мин,
Гел сине сагынам,
Бәхетем син генә дөньяда.
Шәл салып җилкәңә,
Зәңгәрсу кичләрдә
Чык әле җырымны тыңларга.

Бу җырым тик сиңа,
Син тыңла, син тыңла.
Йөрәкнең сагышы бу җырда,
Кайгылы көнеңдә,
Караңгы төнеңдә
Мин телим юлдашың булырга,

Инде хуш димә син,
Юлымны өзмә син,
Соңгы кат сау бул дип озатма.
Чакырсаң килермен,
Кулымны бирермен,
Булсам да еракта, еракта.
```
