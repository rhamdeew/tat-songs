# Оригинал

### Рагидә Янбулатова / Хөсәен Әхмәтов - Гүзәл Өфем — баш калам

```
Тау башында балкый бер кала,
Ул кала Өфе дип атала.
Яктырак яна йолдызлары:
Матурлар егет һәм кызлары.

Алмалары пешә балланып,
Агыйдел ага боргаланып.
Аңа кушыла ямьле Диме,
Ах, Өфем, өзелеп сөям сине!

Хуш ис бөрккән айлы кичләре
Тирбәлдерә йөрәк хисләрен.
Бакчасында иркен тын алам,
Ах, Өфем, минем гүзәл калам!

Синдә үстем, эчтем суларын,
Тыңладым җыр, курай моңнарын.
Кайда барсам, сиңа ашкынам,
Ах, Өфем, Өфем, син баш калам!
Кайда барсам, сиңа ашкынам,
Ах, Өфем, гүзәл Өфем, минем баш калам!
```
