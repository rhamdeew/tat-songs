# Оригинал

### Мөнир Мазунов / Мәсгуть Латыйпов - Дуслык байрагы

```
Чигенде, бәрелеп кыяга,
Дошманның яулары.
Көч бирде ирекле Кубага
Камышлар шаулавы.

Камышлар, шаулагыз,
Кояшка, кояшка карагыз,
Давыллы җилләрдә аумагыз,
Бик якын, бик якын арагыз.

Ишетсен авазын Иделнең
Гавана ярлары.
Китерсен сәламен бу илнең
Камышлар шаулавы.

Камышлар шаулагыз,
Кояшка, кояшка карагыз,
Күтәрдек без дуслык байрагын,
Бик якын, бик якын арабыз.

Бу илнең һавасы бик җылы,
Яз кебек таңнары.
Халкының иң горур бер җыры —
Камышлар шаулавы.

Камышлар, шаулагыз,
Кояшка, кояшка карагыз.
Җилферди чын дуслык байрагы,
Бик якын, бик якын арабыз.
```
