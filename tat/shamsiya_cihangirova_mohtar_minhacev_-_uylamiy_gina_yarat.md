# Оригинал

### Шәмсия Җиһангирова / Мөхтәр Минһаҗев - Уйламый гына ярат

```
Бер карасаң — ят кебек син,
Бер карасаң — бик үз дә…
Кулларың алма кулымнан,
Күзләрең алма күздән.

Бер карама, мең кара син,
Тутырып кара кабат!
Киләчәктә ни буласын
Уйламый гына ярат!

Бер карасаң — син еракта,
Бер карасаң — янымда…
Бүгенемнән дә аерма,
Үткәндә дә калдырма.

Бер карама, мең кара син,
Тутырып кара кабат!
Киләчәктә ни буласын
Уйламый гына ярат!

Бер карасаң — сөям кебек,
Бер карасаң — сөймим дә…
Икеләнеп йөрүемнең
Сәбәпләре бит синдә.

Бер карама, мең кара син,
Тутырып кара кабат!
Киләчәктә ни буласын
Уйламый гына ярат!
```
