# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Яңа ел сәгатьләре

```
Әйдә, искә алыйк әле
Борынгы гадәтләрне…
Бер минутка тынып торыйк,
Онытып сәгатьләрне.

Үткән елда калсын әле
Кайгының сәбәпләре…
Кавыштырсын гашыйкларны
Яңа ел сәгатьләре.

Мәңге ташып торсын әле
Шатлык-сәгадәтләре.
Сәгать суга, сәгать суга —
Яңа ел сәгатьләре.
```
