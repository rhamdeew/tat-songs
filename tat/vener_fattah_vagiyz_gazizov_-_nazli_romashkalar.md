# Оригинал

### Венер Фәттах / Вәгыйзь Газизов - Назлы ромашкалар

```
Чәчәк тотып барам,
Сөюемнән янам,
Кыр чәчәкләре кулларымда.
Син минеке генә,
Иш син күңелемә,
Син бит җанымда, тыннарымда.

Күп тә кирәк түгел,
Синдә минем күңел,
Син уйларымда, юлларымда.
Кояш миңа карый,
Хыялый дип саный,
Син илһамымда, җырларымда.
Җырлый-җырлый барам,
Үзем эчтән янам,
Кулларымда ак чәчәкләрем.
Дөнья шундый матур,
Сөю — гүзәл чак ул,
Ромашка сиңа бүләгем.

Ап-ак ромашкалар,
Назлы карашалар,
Һәр таҗында минем сөюем.
Ап-ак ромашкалар,
Назлар тараталар,
Чәчәк тотып сиңа килүем.

Күп тә кирәк түгел,
Синдә минем күңел,
Син уйларымда, юлларымда.
Кояш миңа карый,
Хыялый дип саный,
Син илһамымда, җырларымда.
Җырлый-җырлый барам,
Үзем эчтән янам,
Кулларымда ак чәчәкләрем.
Дөнья шундый матур,
Сөю — гүзәл чак ул,
Ромашка сиңа бүләгем.
```
