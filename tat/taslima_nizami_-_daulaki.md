# Оригинал

### Тәслимә Низами - Дәүләки

```
Шаулый шомырт урманнары
Бары тик Дәүләкидә.
Урамнарда — азан моңы,
Иман нуры — һәр өйдә.

Дәүләки тугайларында
Калды яшьлек эзләре.
Йөгереп кайттым Сабантуйга
Бик сагындым — түзмәдем.

Йөрәккәем ярсып яна,
Сизеп гомер үткәнне.
Кыз тавына карап багам
Алда ниләр көткәнне.

Бола, Чирмешәннең гашыйк
Дулкыннары Зөядә.
Ике елга кавышкан бит,
Кавышмасак, сөймә дә.

Сабантуйда яшьлек үзе
Кайтыр су буйларына.
Дөрләтик сөю учагын,
Ялкын сал кулларыма.
```
