# Оригинал

### Дания Гайнетдинова / Мансур Хәсәнов - Гомер чишмәсе

```
Илаһи чишмәм, моң тулы синдә,
Сихри җырыңа гомер табынам.
Идел—диңгезләр эзләп барасың,
Күңелем ярында уелып каласың.
Гомер чишмәсе, әй изге чишмәм.

Үзәкләремне нигә өзәсең,
Бәгырьләремне нигә теләсең?
Туган авылымның чишмә чыңнары,
Күңелемдә тулы сагыш моңнарың.
Гомер чишмәсе, әй изге чишмәм.

Назлы җырыңнан, хәтфә ярыңнан
Мәхрүм итмә син, ятим итмә син.
Синдә уянды, синдә сафланды,
Тәүге сөюнең ядкаре калды.
Гомер чишмәсе, әй изге чишмәм.
```
