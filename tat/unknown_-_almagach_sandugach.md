# Оригинал

### Башкорт халык җыры - Алмагач, сандугач

```
Алтын балдагым да бар (да),
Көмеш балдагым да бар.
Алмагач, сандугач,
Син ай булсаң, мин — кояш.

Ни сорасаң, шуны бирәм,
Кайчан алдаганым бар?
Алмагач, сандугач,
Син ай булсаң, мин — кояш.

Безнең арта салкын чишмә
Туңа торгандыр инде.
Алмагач, сандугач,
Син ай булсаң, мин — кояш.

Һәр кемдә дә бер үкенеч
Була торгандыр инде.
Алмагач, сандугач,
Син ай булсаң, мин — кояш.

Куй самовар, чәй эчәек,
Жәлләмәгез күмерне.
Алмагач, сандугач,
Син ай булсаң, мин — кояш.

Ашап-эчеп, уйнап-көлеп
Үткәрәек гомерне.
Алмагач, сандугач,
Син ай булсаң, мин — кояш.
```
