# Оригинал

### Зөлфәт Хәким - Сине югалтсам

```
Төнгә кереп барабызмы
Нигә былай караңгы?
Әллә томан арабызда,
Әллә авыр карадың?

Миннән китте, синдә йөри,
Күңелемнең яртысы.
Сине югалткан көнне ук
Мин үләрмен ахрысы.

Сәгать саен күз алдымда,
Белмим нигә юксынам?
Мындый тирән сагыш өчен
Хәсрәтем дә юк сымак.

Миннән китте, синдә йөри,
Күңелемнең яртысы.
Сине югалткан көнне ук
Мин үләрмен ахрысы.

Мизгел саен яңа дулкын,
Мизгел саен сагынам.
Ни булгандыр зиһенемә,
Бары сиңа табынам?

Миннән китте, синдә йөри,
Күңелемнең яртысы.
Сине югалткан көнне ук
Мин үләрмен ахрысы.
```
