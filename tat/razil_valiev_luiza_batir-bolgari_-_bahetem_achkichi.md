# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Бәхетем ачкычы

```
Ашкына йөрәгем,
Юлларның юк очы,
Кемнәрнең кулында
Бәхетем ачкычы?

Кем мине назлады,
Кем сөйде, кем кочты?
Кемнәрнең кулында
Мәхәббәт ачкычы?

Хисләрем ник ташты,
Йөрәгем кем ачты,
Кемнәрнең кулында
Бәхетем ачкычы?

Кем мине назлады,
Кем сөйде, кем кочты?
Кемнәрнең кулында
Мәхәббәт ачкычы?
```
