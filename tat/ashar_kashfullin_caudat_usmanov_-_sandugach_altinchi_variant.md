# Оригинал

### Әсхәр Кәшфуллин / Җәүдәт Усманов - Сандугач (алтынчы вариант)

```
Сагышларга тулган чагым,
Яшькәйләрем тора тамарга.
Әй өздерә дә соң моңлы бер сандугач
Тау астында, талларда.

Нигә шулай өздерә ул,
Мәхәббәтен әллә тапканмы?
Әллә сөйгән ярын, сөйгәнен югалтып,
Сагышларга ул да батканмы?

Сагышларга тулган чагым
Яшькәйләрем тора тамарга.
"Нигә өзгәләнә соң бу?" — димәгез,
Мөмкиндер бит мине аңларга?!
```
