# Оригинал

### Ләйсән Гайнетдинова / Илнур Мусин - Син бит үзең шулай теләдең

```
Төннәрен дә утны сүндермичә
Өебездә сине мин көтмәм.
Кайнар мәхәббәтнең назлы сүзен
Бүтән инде сиңа мин әйтмәм.

Аяк астында җир убылдымы…
Ачынып елады йөрәгем.
Тик мин киттем сүзсез, эндәшмичә,
Син бит үзең шулай теләдең.

Алсу таңнар тагын сызылып атар,
Тик мин синең белән уянмам.
Гомер учаклары дөрләп янар,
Тик мин синең белән яналмам.
```
