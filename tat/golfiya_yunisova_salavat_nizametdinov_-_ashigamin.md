# Оригинал

### Гөлфия Юнысова / Салават Низаметдинов - Ашыгамын

```
Ашыгамын. Кешеләргә
Әйтә торган сүзләрем бар.
Ашыгамын. Алларымда
Артыласы үрләрем бар.

Аермасын җырларымнан…
Күпме килер кулларымнан…
Ашыгамын, ашыгамын…
Туктатмагыз юлларымнан.

Ишетәсе моңнарымны
Ишетәем тизерәк, дим,
Алда күпме күрәсем бар,
Күрәемче күберәк, дим.

Аермасын җырларымнан…
Күпме килер кулларымнан…
Ашыгамын, ашыгамын…
Туктатмагыз юлларымнан.

Көтсен мине ярсу язым,
Көтсен мине алтын көзем.
Диярләргә дисен иде
Вакытында изгелеген.

Аермасын җырларымнан…
Күпме килер кулларымнан…
Ашыгамын, ашыгамын…
Туктатмагыз юлларымнан.
```
