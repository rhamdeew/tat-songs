# Оригинал

### Мәхмүт Хөсәен / Ренат Еникеев - Ак дулкыннар

```
Ах, дулкыннар,
Ак дулкыннар…
Уйный җилләр чыгуга.
Әй, кара күз, күңелем төште
Син сөйкемле сылуга.

Ах, дулкыннар,
Ак дулкыннар…
Ярга кага канатын.
Әй, кара күз, әллә сине
Чынлап тамы яраттым?!

Ах, дулкыннар,
Ак дулкыннар…
Уйный кояш нурында.
Әй, кара күз, ник моңарчы
Очрамадың юлымда?
```
