# Оригинал

### Татар - Башкорт халык җыры - Беләзегем-йөзегем (икенче вариант)

```
Сары матур, сары матур,
Сары матур сайласаң.
Әллә кайлардан танырмын
Кошлар булып сайрасаң.

Беләзегем-йөзегем
Киярмен әле үзем.
Ятка бирер ярларым юк,
Сөярмен әле үзем.
Ай, урманда, урманда,
Урманда, куш каенда,
Шулай сагына торгандыр
Минем җаныкаем да.

Сандугач җибәрәм сезгә,
Тыңларга сүзегезне.
Бакчагызда сайрап торган
Күрмәгән үзегезне.

Беләзегем-йөзегем
Киярмен әле үзем.
Ятка бирер ярларым юк,
Сөярмен әле үзем.
Ай, урманда, урманда,
Урманда, куш каенда,
Шулай сагына торгандыр
Минем җаныкаем да.
```
