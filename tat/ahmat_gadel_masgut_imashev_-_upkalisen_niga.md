# Оригинал

### Әхмәт Гадел / Мәсгуть Имашев - Үпкәлисең, нигә?

```
Үпкәлисең, нигә?
Сөйләшмисең, нигә?
Күңелеңдә ңинди шөбһә бар?
Елмаюың белән
Барлыгымны билә,
Югалсын яман шик бездә, яр!

Моңланасың, нигә?
Уйланасың, нигә?
Күңелеңне борчыр ниләр бар?
Елмаюың белән
Бар дөньямны билә,
Балкысын шатлыктан йолдызлар!

Кулыңны бир, җаным,
Бул шифалы таңым.
Яр сөеп яшәүгә тиң ни бар?!
Бак минем күзләргә,
Минем шат күзләрдә
Син генә ачарлык серләр бар!
```
