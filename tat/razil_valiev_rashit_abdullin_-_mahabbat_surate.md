# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Мәхәббәт сурәте

```
Язларда эзлим, эзлим гел сине,
Эзлим гөл арасыннан.
Юлларга карыйм, яз җиткәч, юллар
Яныңа илтә сыман.
Язларда мин барам гел урманга, урманга.
Адашам мин яшел җил-буранда анда.
Урманнарда чәчәк юрган,
Күңелемдә язгы буран,
Күкелдә буран,
Күңелдә язгы буран.

Язларда языла мәхәббәт сүрәсе,
Язларга охшаган мәхәббәт сурәте.
Язларга охшаган мәхәббәт сурәте,
Язларда языла мәхәббәт сүрәсе.

Язларда эзлим, эзлим гел сине,
Дөньяда — язлар назы.
Кайларда икән, кайларда икән
Мәхәббәт, сөю яэы?
Язларда мин карыйм гел һавага, һавага,
Һавада гел ак болытлар ага, ага.
Һаваларда — ак болытлар,
Күңелемдә — янар утлар,
Күңелдә — утлар.
Күңелдә — язгы утлар.

Язларда языла мәхәббәт сүрәсе,
Язларга охшаган мәхәббәт сурәте.
Язларга охшаган мәхәббәт сурәте,
Язларда языла мәхәббәт сүрәсе.
```
