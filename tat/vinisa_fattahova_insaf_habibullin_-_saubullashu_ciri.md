# Оригинал

### Виниса Фәттахова / Инсаф Хәбибуллин - Саубуллашу җыры

```
Киләсе очрашуга тиклем,
Соңдыр булып торыгыз.
Безгә бик шат күтерә алса,
Күңелебезнең җырыбыз.

Сезнең күңелгә хуш килсә
Безнең җырлаган җырлар.
Дус булырбыз исән чакта,
Аерылмас мәңге юллар.

Зур шатлыкларга урады
Дуслар безне алкышлар.
Инәнде күңелләребез
Кабат туйгандай кошлар.

Сезнең күңелгә хуш килсә
Безнең җырлаган җырлар.
Дус булырбыз исән чакта,
Аерылмас мәңге юллар.

Тагын очрашырга язсын,
Дөньялар имин булсын.
Барыбызга да тазалык,
Тәләкләр амин булсын.

Сезнең күңелгә хуш килсә
Безнең җырлаган җырлар.
Дус булырбыз исән чакта,
Аерылмас мәңге юллар.
```
