# Оригинал

### Рәми Гарипов / Әминә Бикташева - Белсәң икән

```
Белсәң икән йолдызларның телен,
Мәңгелеккә алар ни сөйли?
Белсәң икән дулкыннарның көен,
Ярларына алар ни сөйли?

Без — мәңгелек, диеп күз кысамы
Бу йолдызлар кәккүк кичендә?
Күпме хата эшли кеше, диме,
Күбәләктәй гомере эчендә?

Ә үләннәр, дәррәү үрелешеп,
Ни сөрәнли икән җилләргә!
Безнең кебек тату булыгыз гел,
Төренегез, диме, гөлләргә?
```
