# Оригинал

### Ләйсән Кәшфи / Гөлнара Хәсәнова - Аймылыш язмышлар

```
Яшьлекнең язында очраштык, таныштык,
Табыштык — аккоштай тиң ярлар.
Син утка, мин суга әйләнеп хушлаштык,
Аерды яшенле яңгырлар.

Аймылыш язмышлар,
Юлларда ялгышлар.
Син мине, мин сине тапмадым.
Ә уйлар гел бергә,
Гамь биреп гомергә,
Яшьлекнең назларын саклады.

Гомерләр юлында очраштык без тагын
Яшьлеккә бер адым, бер адым.
Эйтерсең, үлчәдек шул адым арасын,
Син җиргә, мин күккә карадым.

Аймылыш язмышлар,
Юлларда ялгышлар.
Син мине, мин сине тапмадым.
Ә уйлар гел бергә,
Гамь биреп гомергә,
Яшьлекнең назларын саклады.

Җир белән күк кебек якын без, ерак без
Әйтерсең, кояш бар арада.
Ул күктән елмаеп ләйсәнле нур сибә
Яшеннәр көйдергән ярага.

Аймылыш язмышлар,
Юлларда ялгышлар.
Син мине, мин сине тапмадым.
Ә уйлар гел бергә,
Гамь биреп гомергә,
Яшьлекнең назларын саклады.
```
