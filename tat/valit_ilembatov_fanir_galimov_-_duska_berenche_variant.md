# Оригинал

### Вәлит Илембәтов / Фәнир Галимов - Дуска (беренче вариант)

```
Шатланам дуслар булганга,
Алма бишкә бүләрлек.
Баш катканда киңәш сорап,
Эч серләрен сөйләрлек.

Каршы чыгып ат бәйләшү —
Бик матур йола икән.
Дуслар якын туганнардай
Кадерле була икән.

Туйлар, бәйрәмнәрдә генә
Дус булу — ул ерунда.
Чын дуслар атлыгып килер
Сиңа кыен чагында.

Каршы чыгып ат бәйләшү —
Бик матур йола икән.
Дуслар якын туганнардай
Кадерле була икән.

Шатлык-кайгы уртак булсын,
Дуслар дусны какмасын.
Дошман белән дус булудан
Ходай үзе сакласын!

Каршы чыгып ат бәйләшү —
Бик матур йола икән.
Дуслар якын туганнардай
Кадерле була икән.
```
