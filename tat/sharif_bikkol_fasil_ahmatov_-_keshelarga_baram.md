# Оригинал

### Шәриф Биккол / Фасил Әхмәтов - Кешеләргә барам

```
Куанычтан җан җылынса,
Якты сукмак ярам;
Шатлыгымны өләшергә
Кешеләргә барам!

Күңелемне юатырга
Бетсә соңгы чарам,
Хәсрәтләрдән арынырга
Кешеләргә барам.

Кемлегемне сынар булсам
Кешеләргә барам!
И, кешеләр, сезнең белән
Өзелмәсен арам!
```
