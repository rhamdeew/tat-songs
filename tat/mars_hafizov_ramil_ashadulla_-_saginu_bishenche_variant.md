# Оригинал

### Марс Хафизов / Рамил Әсхәдулла - Сагыну (бишенче вариант)

```
Көзләр җиткән саен казлар юлы
Сагыш тезмәсеннән әйләнә,
Көзләрендә казлар юлы аша
Яшьлегемә килеп әйләнә.

Киек казлар, сезнең тавышлардан
Күңелемә бигрәк ямансу.
Яшьлегемә кире кайталмадым,
Бары калды калды сагыну.

Киек казлар, алып кайтыгызчы
Канатларда яшьлек ягына,
Яшьлегемне әгәр очратсагыз,
Әйтегезче: "Ул бит сагына!"

Киек казлар, сезнең тавышлардан
Күңелемә бигрәк ямансу.
Яшьлегемә кире кайталмадым,
Бары калды калды сагыну.
```
