# Оригинал

### Алия Солтан / Эльвира Хаматнурова - Җилләр (өченче вариант)

```
1. Йомшак жилләр әкрен генә
Тәрәз кагып утсәләр
Алсу таңда уян диеп
Бер хәбәр китерсәләр
Тәрәзәңне ач тизерәк
Җил булып мин исәрмен
Иркәләп кенә чәчеңнән
Мин ул сыйпап утәрмен

Припев:
Җилләр шаярса яныңда
Кагылып битләреңдә
Яраткан бер моңың булып
Төшәрмен исләреңә

2. Исләреңә төшәрмендә
Бер елмаеп куярсың
Шаян җилләр убеп алыр
Ул мин диеп уйларсың
Тәрәзәңне ач тизерәк
Җил булып мин исәрмен
Иркәләп кенә чәчеңнән
Мин ул сыйпап утәрмен
```
