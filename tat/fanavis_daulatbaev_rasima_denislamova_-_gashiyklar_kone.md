# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Гашыйклар көне

```
Җан яраткан кешесенә
Күңел тартыла.
Чын гашыйклар утка керә
Сөю хакына.

Йөрәкләрнең мәхәббәтне
Ачыклар көне.
Араларны якынайтсын
Гашыйклар көне!

Күңелләрне ташламасын
Ярату хисе.
Шат уйларда,тойгыларда
Яшәсен кеше.

Йөрәкләрнең мәхәббәтне
Ачыклар көне.
Араларны якынайтсын
Гашыйклар көне!

Шашып типсен, ярсып типсен
Йөрәкләр бергә.
Чын ихластан яратулар
Булсын гомергә!

Йөрәкләрнең мәхәббәтне
Ачыклар көне.
Араларны якынайтсын
Гашыйклар көне!
```
