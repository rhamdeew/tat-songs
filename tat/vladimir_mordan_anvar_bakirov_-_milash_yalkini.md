# Оригинал

### Владимир Мордань / Әнвәр Бакиров - Миләш ялкыны

```
Данлы Идел дулкыны шаулый,
Идел өстендә акчарлак.

Миләш ялкыны,
Балан ялкыны
Күнлемнән чыкмас
Дусларым, гомергә.

Славута дулкыннары шаулый,
Язмышлар бергә бәйләнгән.

Миләш ялкыны,
Балан ялкыны
Күнлемнән чыкмас
Дусларым, гомергә.

Идел-Днепр — алар бертуганнар,
Нур сибә дуслык йолдызы.

Үргә атлыйбыз
Ил белән бергә.
Дуслык йолдызы
Нур сибәр гомергә.
```
