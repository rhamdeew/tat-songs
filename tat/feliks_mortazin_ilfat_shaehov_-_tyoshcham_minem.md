# Оригинал

### Феликс Мортазин / Илфат Шәехов - Тёщам минем

```
Бар бу дөньяларда
Ике бик шәп кеше
Аның берсе мин, кияү
Берсе минем теща!
Кияү җаныкаем диеп
Гел биеп кенә тора
Минем хатынны бары тик
Ул гына җиңә ала!

Хатыныма әни
Бабаема хатын
Балаларга дәү әни
Минем тещам алтын

Тещам оста куллы минем
Куллары йокмый кулга
Мин кунакка килгән саен
Табалар чыжлап тора
Кырмыскадай эшчән
Минем алтын тещам
Кайчак кырмыскадай
Алгалый ул тешләп!

Хатыныма әни
Бабаема хатын
Балаларга дәү әни
Минем тещам алтын

Өрмәгән җиргә утыртмый
Йөзе һәрвакыт көләч
Зур рәхмәт шундый шәп кызны
Үстереп миңа биргәч!
Бар бу дөньяларда
Ике бик шәп кеше
Аның берсе мин, кияү
Берсе минем теща!

Хатыныма әни
Бабаема хатын
Балаларга дәү әни
Минем тещам алтын
```
