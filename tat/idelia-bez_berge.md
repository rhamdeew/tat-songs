### Без бергэ

```
Иркэлэп тотып синен куллардан
Кунелне чиртеп нечкэ кыллардан.
Эрисем килэ синен куенында
Бары син генэ минем уемда.
Бул янымда.

Без бергэ, без бергэ
Без бэхетле айлы тондэ.
Без бергэ, без бергэ
Синен белэн икэу генэ.

Мина бик жылы синен янында,
Синен сулышны тоям жанымда
Мина пышылдый синен иреннэр
Сою бэхетен алар соймилэр.
Бул янымда.

Без бергэ, без бергэ
Без бэхетле айлы тондэ.
Без бергэ, без бергэ
Синен белэн икэу генэ.

Безнен бэхеткэ берни кирэкми,
Безнен союне беркемдэ белми
Буген без бергэ, шуны онытма
Якынрак кил син яныма.

Без бергэ, без бергэ
Без бэхетле айлы тондэ.
Без бергэ, без бергэ
Синен белэн икэу генэ.
```
