# Оригинал

### Хәния Фәрхи / Рифкать Сәйфетдинов - Әфганстан

```
Күзләреңә туры карый алмыйм,
Сиңа карап күңелем тулышкан.
Бер юату сүзе эзлим сиңа,
Тик мин көчсез, кем гаепле?
Әфганстан, Әфганстан.

Кулларыңны килә бик тоясым,
Мине кочып, сөеп яраткан.
Тик алар юк, бары бушлык анда.
Нәрсә өчен, кем гаепле?
Әфганстан, Әфганстан.

Күпме ана өмет баглап — багып,
Бердәнберен сиңа ул озаткан.
Улы кайткан тимер табут белән.
Нәрсә өчен, кем гаепле?
Әфганстан, Әфганстан.
```
