# Оригинал

### Татар халык җыры - Эх, карап торулары

```
Кашың кара, кашың кара,
Каралткангадыр инде.
Карыйм, карыйм, карап туймыйм,
Яраткангадыр инде.

Эх, карап торулары,
Күз сирпеп алулары!
Синнән генә булды инде
Йөрәгем янулары.

Кашың кара, күзең зәңгәр,
Керфегең каралтасың.
Күз карашың магнитлы,
Гел үзеңә тартасың.

Эх, карап торулары,
Күз сирпеп алулары!
Синнән генә булды инде
Йөрәгем янулары.
```
