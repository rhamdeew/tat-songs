# Оригинал

### Ринат Мөслимов / Зөфәр Хәйретдинов - Эзләрмен, табармын

```
Төннәр дә бик озын
Булмагач син үзең
Йокы да hич керми күзләргә
Елгада ике яр
Бары юк күперләр
Чыгамын мин сине эзләргә.

Эзләрмен, табармын
Ут булып янармын
Күңелем гел сине
Уйлый бит.
Өзелеп сөюдән
Янудан, көюдән
Күкрәккә йөрәгем
Сыймый бит.

Минеме аңлапмы
Болытны таратты
Булыша юлымда җилләр дә
Чәчәкләр шиңәрләр,
Йолдызлар сүнәрләр
Тапмасам мин сине әгәр дә.

Эзләрмен, табармын
Ут булып янармын
Күңелем гел сине
Уйлый бит.
Өзелеп сөюдән
Янудан, көюдән
Күкрәккә йөрәгем
Сыймый бит.

Былбыллар ах диләр
Чишмәләр тап диләр
Кояш та мул сибә нурларын.
Табармын эзләгәч
Шатлыктан түзмәгәч
Кочаклап үбәрмен җир шарын.

Эзләрмен, табармын
Ут булып янармын
Күңелем гел сине
Уйлый бит.
Өзелеп сөюдән
Янудан, көюдән
Күкрәккә йөрәгем
Сыймый бит.
```
