# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Сөям дигән сүзең җитә

```
Сөям дигән сүзең җитә–
Күтәрергә күкләремә.
Шул сүзеңнән дөрләп китәр
Сөю уты күкрәгемдә.

Сөям дигән сүзең җитә-
Мәхәббәттән шашынырга.
Сусаганда дымын көткән
Чәчәк булып ачылырга.

Сөям дигән сүзең җитә–
Талпынырга, очынырга.
Мәңгелеккә колы булып
Мәхәббәткә буйсынырга.

Сөям дигән сүзең җитә–
Тәмам һуштан яздырырга.
Мәхәббәтнең дөрләп торган
Учагында яндырырга.

Сөям дигән сүзең җитә–
Яулар өчен күңелемне.
Иң бәхетле кеше итеп
Тояр өчен мин үземне.
```
