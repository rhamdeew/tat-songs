# Оригинал

### Тимер Йосыпов / Рамил Медъяр-Камал - Әй, туган як

```
И туган як, туган як,
Бабайларым торган як,
Ояларын корган як,
Дөньяларын куган як.
Минем туган ягымда
Торна канат кагына.
Чит җирләргә китсә дө,
Туган ягын сагына.

Әй, туган як, туган як,
Әткәм билен буган як,
Әнкәй буйга җиткән як,
Килен булып төшкән як.

Мин дөньяга килгәнмен,
Елаганмын, көлгәнмен,
Колын сыман йөргәнмен,
Басуларга чыкканмын.
Җилләр тамак кырган,
Әнкәем урак урган,
Алма сыман пешкән ул,
Бәхет өчен үскән ул.

Әй, туган як, туган як,
Ак нурларга тулган як,
Әткәм егет булган як,
Асыл ирләр торган як.
```
