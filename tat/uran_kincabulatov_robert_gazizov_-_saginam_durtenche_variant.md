# Оригинал

### Уран Кинҗәбулатов / Роберт Газизов - Сагынам (дүртенче вариант)

```
Син — мәхәббәт җырым минем,
Тик бер сине җырлыймын.
Син күз сирпеп карыйсың да
Күңелемне нурлыйсың.

Син, сөйеклем, яз кояшым,
Мәхәббәтем, хәзинәм!
Дөнъям минем юк бит бүтән,
Ялгызлыкны очып үтәм,
Очкан саен сагышланам,
Очкан саен сагынам!

Син — күз нурым, сөю җырым,
Сиңа карап туймыймын.
Кайда барсам, сиңа барам,
Син миндәдер, мөгаен.

Син, сөйеклем, яз кояшым,
Мәхәббәтем, хәзинәм!
Дөнъям минем юк бит бүтән,
Ялгызлыкны очып үтәм,
Очкан саен сагышланам,
Очкан саен сагынам!
```
