# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Түбән Кама утлары

```
Уяна Урал ягында
Алсу таң билгеләре,
Нурлана Кама ярында
Безнең шәһәр өйләре.

Түбән Кама, Түбән Кама —
Яшьлек калабыз.
Синдә киләчәк көннәрнең
Нигезен салабыз.

Йолдызларга үрелеп усә
Без салган гүзәл йортлар:
Таңнарга кадәр җемелди
Күгеңдә алтын утлар.

Түбән Кама, Түбән Кама —
Яшьлек калабыз.
Синдә киләчәк көннәрнең
Нигезен салабыз.

Беренче сөю хисләрем
Дөрләп кабынган шәһәр.
Гомремнең гүзәл юллары
Синдә табылган шәһәр.

Түбән Кама, Түбән Кама —
Яшьлек калабыз.
Синдә киләчәк көннәрнең
Нигезен салабыз.
```
