# Оригинал

### Роберт Миңнуллин / Илдус Низамов - Кайнар чәй

```
Чәеңнең кайнарлыгыннан
Чәеңнең хуш исеннән
Әллә нишлим эреп китәм
Әллә нишлим исерәм.

Яндыр мине, яндыр кайнар
Мәхәббәт учагында.
Кайнар синең чәйләрең дә,
Кайнардыр кочагың да.

Карашыңның кайнарлыгы
Өтеп ала җанымны.
Чәең кайнар, Үзең кайнар,
Кыздырасың канымны.

Яндыр мине, яндыр кайнар
Мәхәббәт учагында.
Кайнар синең чәйләрең дә,
Кайнардыр кочагың да.

Чәең кайнар,үзең кайнар
Чәй эчәм өрә-өрә.
Мәхәббәттә ничә кабат,
Авыз пешкәнгә күрә.

Яндыр мине, яндыр кайнар
Мәхәббәт учагында.
Кайнар синең чәйләрең дә,
Кайнардыр кочагың да.
```
