# Оригинал

### Ләйлә Дәүләтова / Зөфәр Хәйретдинов - Яратмасаң — яратырсың

```
Читкә караган буласың,
Янәшәңнән узганда.
Эреп төшәсең бит, җаным,
Икәү генә калганда.

Яратмыйм дигән сүзеңне
Ишетмим, кире борам.
Яратмасаң — яратырсың,
Мин бераз көтеп торам.

Сөймим дигән сүзләрең дә
Сөям дип ишетелә.
Йозак элеп куй, җаныем,
Йөрәгең ишегенә.

Яратмыйм дигән сүзеңне
Ишетмим, кире борам.
Яратмасаң — яратырсың,
Мин бераз көтеп торам.

Миңа гашыйк буласыңны
Әллә син белмисеңме?
Ни кадәр киреләнсәң дә,
Гашыйк итәм мин сине.

Яратмыйм дигән сүзеңне
Ишетмим, кире борам.
Яратмасаң — яратырсың,
Мин бераз көтеп торам.
```
