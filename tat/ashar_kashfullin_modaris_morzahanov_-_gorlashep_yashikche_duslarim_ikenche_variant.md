# Оригинал

### Әсхәр Кәшфуллин / Мөдәрис Морзаханов - Гөрләшеп яшикче, дусларым! (икенче вариант)

```
Күңелне тут ала,
Гомерләр туктала,
Гаиләдә татулык булмаса.
Җырларың сагыштан,
Еларсың ялгыштан,
Тормышта ярыңнан уңмасаң.

Татулык озайта,
Ызгышлар кыскарта
Болай да бик кыска гомерне.
Кергәнче ак кышка,
Чәчәктәй бик кыска
Гомернең кадерен белерлек.

Дусларың булышса,
Кояштай җылытса,
Өшетмәс иде ул кышларын.
Мәхәббәт җиләктә,
Дуслыкка ни җитә,
Гөрләшеп яшикче, дусларым!
```
