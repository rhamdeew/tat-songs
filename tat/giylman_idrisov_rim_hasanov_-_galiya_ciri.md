# Оригинал

### Гыйльман Идрисов / Рим Хәсәнов - Галия җыры

```
Айның якты нуры җиргә атылганда,
Чәчләремнән сыйпап җилләр искәндә.
Хисләр ташый микән
Яшьли сөйгән ярым
Тәүге умырзая бүләк иткәндә?

Сәлам белән бергә, хатлар китерсәләр
Үтеп киткән язның кипкән гөлләрен.
һәр тибеше саен йөрәк көтә микән
Тиздән килеп җитәр кавышу көннәре.
```
