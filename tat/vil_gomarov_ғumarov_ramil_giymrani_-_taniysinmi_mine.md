# Оригинал

### Вил Гомәров (Ғүмәров) / Рамил Гыймрани - Таныйсыңмы мине

```
Язлар кабат килеп,
чәчәкләргә күмеп,
Назлы җилләр исеп —
дөнья үзгәрде.
Ташкын кичеп эзләдем мин —
Синең таныш күзләрне.

Язмыш кушса бәлки,
очрашырбыз, дидем.
Кисешерләр шунда —
яшьлек юллары
Сөю җырын җырлар безгә
Аерылышу еллары.

Урам буйлап килеп
сине шунда күреп,
Иркәм мине таныр
диеп уйладым.
Шатлыгымнан, сиңа арнап,
Шушы җырны җырладым.
```
