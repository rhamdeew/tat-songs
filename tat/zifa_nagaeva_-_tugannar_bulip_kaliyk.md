# Оригинал

### Зифа Нагаева - Туганнар булып калыйк

```
Әтидән алгансың батырлык,
Әнидән җыйгансың сабырлык.
Торырсың, уңгансың һәр яклап булгансың
Сиңа карап хәйран калырлык.

Дөнъяга нур тул тулган,
Дөнъяга җыр тулган,
Туганыбыз безнең син тугач.
Әйт әле рәхәт дип,
Яшәве бәхет дип,
Яннарымда синең без булгач.

Суынмасын безнең аралар,
Гомер күпме генә үтсәдә.
Яшә батыр, яшә матур булып,
Яшим ничәләргә җитсәдә.

Дөнъяга нур тул тулган,
Дөнъяга җыр тулган,
Туганыбыз безнең син тугач.
Әйт әле рәхәт дип,
Яшәве бәхет дип,
Яннарымда синең без булгач.

Бәйрәмеңне бүген котларга
Җыелышып килдек яныңа.
Ни генә булмасын тиеш без һәр вакыт
Без туганнар булып калырга.

Дөнъяга нур тул тулган,
Дөнъяга җыр тулган,
Туганыбыз безнең син тугач.
Әйт әле рәхәт дип,
Яшәве бәхет дип,
Яннарымда синең без булгач.
```
