### Мин синеке тугел

```
Э мин синеке тугел, син минеке тугел...

Монсуланма, кайгырма,
Булыр вакыт синен,
Йорэк яна, ашкына...

Нигэ син елыйсын, мине гаеплисен?
Гаепле мин тугел.
Мине анламыйсын, мине ишетмисен,
Эйтергэ бар сузем.

Мин синеке тугел, син минеке тугел...
Булырбыз без аерым,
Саубул, жаным!

Калды сина кайтырга,
Юллар ябык сина.
Алдан иде уйларга...

Нигэ син елыйсын, мине гаеплисен?
Гаепле мин тугел.
Мине анламыйсын, мине ишетмисен,
Эйтергэ бар сузем.
Тынла мине...

Мин синеке тугел, син минеке тугел...
Булырбыз без аерым,
Саубул, жаным!

Нигэ син елыйсын, мине гаеплисен?
Гаепле мин тугел.
Мине анламыйсын, мине ишетмисен,
Эйтергэ бар сузем.
Тынла мине...

Мин синеке тугел, син минеке тугел...
Булырбыз без аерым,
Саубул, жаным!
```
