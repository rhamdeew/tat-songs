# Оригинал

### Ф. Нагаев / Вәсил Галимов - Күңелдән китмәсен сагыну

```
Өзелеп сагындым, шүңамы
Бар дөнья сарыдан ябынган.
Тузгытып сап-сары яфраклар.
Син киттең күңелем ярыннан.

Мәхәббәт диңгезен ялгызым
Җилкәнсез ничекләр кирчәрмен.
Айрылу, сагыну сагышын
Ничек соң бер үзем эчәрмен.

Бәгырьне телсә дә газаплар,
Юк инде, юк инде ялыну.
Күңелдән китмәсен бары тик,
Бары тик калсын шул сагыну.
```
