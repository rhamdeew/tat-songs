# Оригинал

### Сибгат Хәким / Фирзәр Мортазин - Ага чишмә…

```
Үзем су алам икән,
Үзем юанам икән.
Карыйм, карыйм,
Һич табалмыйм,
Нигә соң янам икән?

Ага чишмә
Чылтыр-чылтыр,
Сөйли, чишмә:
Ничек онытыр,
Ничек онытыр?..

Чишмәләр ага икән,
Юлларын таба икән.
Карыйм, карыйм,
Һич табалмыйм,
Күзләрем тала икән.

Ага чишмә
Чылтыр-чылтыр,
Сөйли, чишмә:
Ничек онытыр,
Ничек онытыр?..

Юлларга чыгам икән,
Уйларга чумам икән.
Карыйм, карыйм,
Һич табалмыйм,
Еракта томан икән.

Ага чишмә
Чылтыр-чылтыр,
Сөйли, чишмә:
Ничек онытыр,
Ничек онытыр?..
```
