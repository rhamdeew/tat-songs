# Оригинал

### Наҗар Нәҗми / Марат Ягафаров - Сөймәсәң, сөйкәлмә

```
Чыкмадың юлларга, бәхетләр юлларга,
Төрле вак тавышлар өзеннән.
Мин түгел — син үзең, мин түгел — син үзең,
Килдең бит син үзең үзеңнән.

Сөймәсәң — сөйкәлмә,
Син димәк бүтәнгә.
Мин димәк бүтәнгә,
Сөймәсәң — сөйкәлмә.

Бар дөнъям син кебек, хыялга тиң кебек,
Хыялга тиң булу җиңелме?
Тик синең өчен мин, тик синең өчен мин,
Дөнъяга туганмын шикелле.

Сөймәсәң — сөйкәлмә,
Син димәк бүтәнгә.
Мин димәк бүтәнгә,
Сөймәсәң — сөйкәлмә.

Шатлыклар көчендә, газаплар эчендә,
Керәсең төннәрен төшләргә.
Күңелдә гел юрау, күңелдә бер сорау,
Нишләргә, нишләргә, нишләргә?

Сөймәсәң — сөйкәлмә,
Син димәк бүтәнгә.
Мин димәк бүтәнгә,
Сөймәсәң — сөйкәлмә.
```
