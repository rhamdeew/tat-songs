# Оригинал

### Зәйтүнә Хөсәенова / Рамил Гыймрани - Кайтырсың бит

```
Таулар белән таулар очрашмыйлар,
Безнең йөрәкләр бит таш түгел.
Язлар белән бәлки, кайтырсың дип,
Өметләнеп көтә яшь күңел.

Эзли-эзли керфекләрем талды,
Айкап чыктым барлык җыһанны.
Суга төшкән кебек юкка чыктың,
Кайларга соң китеп югалдың?

Йолдызлар да сине күрмәгәннәр,
Ай да сиңа нурын сипмәгән.
Үләннәрне чык ташлап киткән,
Тик чәчәкнең яше кипмәгән.

Уем — хисем белән сине эзләп,
Айкап чыктым барлык җыһанны.
Суга төшкән кебек юкка чыктың,
Кайларга соң китеп югалдың?

Сине һаман көтеп яшимен мин
Күпме айлар, еллар узса да.
Бик сагындым, иркәм, ник кайтмыйсың,
Төлшәремә кереп булса да.

Уем — хисем белән сине эзләп,
Айкап чыктым барлык җыһанны.
Суга төшкән кебек юкка чыктың,
Кайларга соң китеп югалдың?
```
