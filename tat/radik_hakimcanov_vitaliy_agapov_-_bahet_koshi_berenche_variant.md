# Оригинал

### Радик Хәкимҗанов / Виталий Агапов - Бәхет кошы (беренче вариант)

```
Бәхет кошы кайда яши икән,
Белсәң иде аның серләрен.
Бик аз гына бәхет сорар идем,
Их, аңларга иде телләрен.

Тормыш серле, кызык, катлаулы бит,
Аңламассың инде дөньяны.
Хыялларда яши бәхет кошы,
Бер тотарбыз кебек без аны!

Бәхет кошы кайда яши икән,
Таулардамы, әллә урманда?!
Аны бары күреп кала, диләр,
Тик иртән чак таңда туганнар.

Тормыш серле, кызык, катлаулы бит,
Аңламассың инде дөньяны.
Хыялларда яши бәхет кошы,
Бер тотарбыз кебек без аны!

Бәхет кошы җирдә яши, диләр,
Дөрес микән, әллә юк микән?!
Гомер буе безне ымсындырган,
Тоттырмаган сихри кош микән?!

Тормыш серле, кызык, катлаулы бит,
Аңламассың инде дөньяны.
Хыялларда яши бәхет кошы,
Бер тотарбыз кебек без аны!
```
