# Оригинал

### Наҗар Нәҗми / Нур Даутов - Кара шомырт чәчәгендәй

```
Кара шомырт чәчәгендәй
Ап-ак синең йөзләрең.
Кара шомырттай кап-кара
Чын карашың күзләре.

Кара шомырттай кап-кара
Чын карашың күзләре.

Шомырт кап-кара булса да
Ап-ак чәчәкләр ата.
Йөрәгемдә шомырт түгел,
Ап-ак чәчәкләр ята.

Йөрәгемдә шомырт түгел,
Ап-ак чәчәкләр ята.

Быелда кап-кара шомырт
Ап-ак чәчәкләр атты.
Сине күрсәм хәтерлимен
Шомырт чәчәкле чакны.

Сине күрсәм хәтерлимен
Шомырт чәчәкле чакны.
```
