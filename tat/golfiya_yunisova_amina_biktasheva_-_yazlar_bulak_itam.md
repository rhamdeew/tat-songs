# Оригинал

### Гөлфия Юнысова / Әминә Бикташева - Язлар бүләк итәм

```
Кулларыма башын куйгач,
Нигә көрсендең икән,
Мин түгелме гомер буе
Сине сагынып көткән?

Язлар бүләк итәм сиңа —
Мин яратам язларны,
Язлар ап-ак чәчәк итәр
Чәчендәге карларны.

Кар яуган чәчеңнән сыйпап,
Карадым күзләреңә.
Шунда гына көрсенүнең
Төшендем серләренә.

Язлар бүләк итәм сиңа —
Мин яратам язларны,
Язлар ап-ак чәчәк итәр
Чәчендәге карларны.

Яурыныңнан йомшак кына
Кагаемчы иркәләп.
Көн элгәре көенми тор —
Картаерга иртәрәк.

Язлар бүләк итәм сиңа —
Мин яратам язларны,
Язлар ап-ак чәчәк итәр
Чәчендәге карларны.
```
