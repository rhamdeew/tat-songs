# Оригинал

### Нәфисә Сабирҗанова / Рәис Нәгыймов - Ошатам сине

```
Ошый миңа синең күзләрең,
Сөеп караганда, яратып.
Торыр идем һәрчак аларны
Гел үземә генә каратып.

Ошый миңа синең тавышың,
Мин хәзер дә аны ишетәм.
Һәрбер әйткән сүзең исемдә,
Кабатлыймын саг(ы)нып кич-иртән.

Ошый миңа синең назларың,
Карашыңнан китәм гел эреп.
Син янымда чакта, дөньяда
Икәү генә калганбыз (калабыз) кебек.

Ошый миңа синең күзләрең,
Ошый миңа синең тавышың,
Ошый миңа синең назларың.
```
