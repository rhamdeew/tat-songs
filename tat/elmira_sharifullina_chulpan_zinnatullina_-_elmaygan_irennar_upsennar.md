# Оригинал

### Эльмира Шәрифуллина / Чулпан Зиннәтуллина - Елмайган иреннәр үпсеннәр

```
Күтәреп күңелем пәрдәсен,
Җен түгел, син кердең, сөюем.
Бу минем мәңгегә булсын дип,
Чын итеп, зур итеп сөюем.

Сөешеп яшәгән парларның
Көннәре кайгысыз үтсеннәр.
Күзләре яктылык сизсеннәр,
Елмайган иреннәр үпсеннәр.

Елмайган иреннәр үпсеннәр.
Юлларга түшәлсен бер бизәк.
Тизрәк кайт минем сөюем,
Калдым мин озатып, ул изә. (?)

Сөешеп яшәгән парларның
Көннәре кайгысыз үтсеннәр.
Күзләре яктылык сизсеннәр,
Елмайган иреннәр үпсеннәр.

Елмайган иреннәр үпсеннәр.
Ихластан әйтелсен ярату.
Иң олы гөнаһ бу — кемнәрнең
Ялганлап йөрәген канату.

Сөешеп яшәгән парларның
Көннәре кайгысыз үтсеннәр.
Күзләре яктылык сизсеннәр,
Елмайган иреннәр үпсеннәр.
```
