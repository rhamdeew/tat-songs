# Оригинал

### Гөлфия Шакирова / Альбина Маклакова - Якты җыр

```
Сөю гөле чәчкә ата,
Назларны тойса гына.
Мәхәббәт сер булып кала,
Җаныңа сыйса гына.

Ихлас хисләрдән моң туа,
Якты җыр булып кала.
Чын мәхәббәт кенә сине
Ялкын-утларга сала.

Йөрәккә сыймаган сөю
Ул сөю буламыни?
Сер булмаса, бәхет булып,
Җаныңа туламыни?

Ихлас хисләрдән моң туа,
Якты җыр булып кала.
Чын мәхәббәт кенә сине
Ялкын-утларга сала.

Язлар саен яңара да
Чәчкәләргә күмелә.
Ә аннары моңнар булып,
Күңелеңнән түгелә.

Ихлас хисләрдән моң туа,
Якты җыр булып кала.
Чын мәхәббәт кенә сине
Ялкын-утларга сала.
```
