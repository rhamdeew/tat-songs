# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Гафу ит син

```
Маңгайларга сала буразналар
Башка төшкән хәсрәт кайгылар.
Кайгы хәсрәтләрне таратучы
Син янымда якын кешем бар.

Гафу ит син мине, гафу ит син
Саксыз сузем белән рәнҗетсәм.
Бер көн уйламаган адым ясап
Кайнар йөрәгеңне өшетсәм.

Көзгеләргә карап уфтанма әлә,
Ямансулап яшьлек узганга.
Картлык безгә тимәй урап узар
Без хисләргә тугры булганда.

Гафу ит син мине, гафу ит син
Саксыз сузем белән рәнҗетсәм.
Бер көн уйламаган адым ясап
Кайнар йөрәгеңне өшетсәм.

Чал чәчләрең миңа хәтерләтер,
Шау чәчәкле ап-ак язларны.
Сөю шарапларын эчеп куйыйк
Тамчыгача безгә язганын.

Гафу ит син мине, гафу ит син
Саксыз сузем белән рәнҗетсәм.
Бер көн уйламаган адым ясап
Кайнар йөрәгеңне өшетсәм.
```
