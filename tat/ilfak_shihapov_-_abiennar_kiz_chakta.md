# Оригинал

### Илфак Шиһапов - Әбиеңнәр кыз чакта

```
Булган шундый заманалар,
Шундый чиста чак булган.
Егетләр батыр булган.
Кызлары тыйнак булган.
Бабаң чакырган чагында,
Әбиең килә алмаган,
"Мин сине хочу, җаным", дип,
SMS җибәрмәгән.

Уты янган учакта
Әбиеңнәр кыз чакта.
Матур итеп сөешкәннәр
Бабаңнар егет чакта.

Шундый заманалар булган:
Секс бер дә блмаган,
Ләкин секс юк дип кенә,
Бала тумый тормаган.
Әбиеңнәр кыз чагында
Сериал карамаган.
Интернетта көне-төне
Дусларын барламаган.

Уты янган учакта
Әбиеңнәр яшь чакта.
Утлар чәчеп сөешкәннәр
Бабаң көчендә чакта.

Әбиеңнәр кыз чагында
Дискотека булмаган.
Әбиеңнәр бабаң белән
Кафеда утырмаган.
Әбиеңнәр кыз чагында,
Егет булган бабаңнар,
Никах белән бирмәсәләр —
Кыз килеш урлаганнар.

Уты янган учакта
Әбиеңнәр кыз чакта.
Матур итеп сөешкәннәр
Бабаңнар егет чакта.
Уты янган учакта,
Әбиеңнәр яшь чакта.
Утлар чәчеп сөешкәннәр
Бабаң көчендә чакта.
```
