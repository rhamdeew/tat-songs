# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Икебезгә бер

```
Әйткәннәр саен әйтеп туялмыйм,
Әйтәселәрем килә яңадан.
Әле дә ярый күңел күзләре
Бер-беребезне күрми калмаган.

Икебезгә бер яна утыбыз,
Икебезгә бер узар юлыбыз.
Мондый бәхетләр бездә генәдер,
Икебезгә бер, икебезгә бер.

Икебезгә бер яна утыбыз,
Кайнары җитәр гомер буена.
Сине күргәнче мин әле бер дә
Яшәмәгән дә сыман тоела.

Икебезгә бер яна утыбыз,
Икебезгә бер узар юлыбыз.
Мондый бәхетләр бездә генәдер,
Икебезгә бер, икебезгә бер.

Күзләрдән күзне килми аласы,
Гел карап кына килә торасы.
Кая барсам да, нурын сибүче
Син генә бит ул күңелем кояшы.

Икебезгә бер яна утыбыз,
Икебезгә бер узар юлыбыз.
Мондый бәхетләр бездә генәдер,
Икебезгә бер, икебезгә бер.
```
