# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Иртән торып эшкә барам

```
Иртән торып эшкә барам,
Ачып керәм йозакны.
Ачып булмый кешеләрнең
Җанындагы тозакны.

Берсен-берсе талый-талый
Ерткыч була кешеләр.
Их кешеләр, их кешеләр…
Без дә сезнең ишеләр.

Иртән торып эшкә барам,
Җырлап «Сарман» көенә.
Минем сөенгәнне күреп,
Ярты Казан көенә.

Берсен-берсе талый-талый
Ерткыч була кешеләр.
Их кешеләр, кешеләр…
Без дә сезнең ишеләр.

Иртән торып эшкә барам…
Әллә инде бармыйсы?
Эшләми дә гомер буе
Яшәдек бит ярыйсы.

Берсен-берсе талый-талый
Ерткыч була кешеләр.
Их кешеләр, кешеләр…
Без дә сезнең ишеләр.
```
