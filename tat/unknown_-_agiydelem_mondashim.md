# Оригинал

### Муса Сираҗи / Тәлгать Шәрипов - Агыйделем — моңдашым

```
Болыт кунган чал таулардан
Башлап алып юл башын,
Келтер-келтер тәгәрәтеп
Уралымның вак ташын,
Челтер-челтер агып ята
Агыйделем — юлдашым,
Агыйделем — юлдашым.

Ярларында, талларында
Сайратып мең-мең кошын,
Кайда урман, кайда тургай —
Шунда борып ағышын,
Ага тарих төпкеленнән
Агыйделем — язмышым,
Агыйделем — язмышым.

Ак томаннарга уралып,
Офыкларга олгашып,
Ага халкым йөрәгеннән
Көне төне моң ташып,
Ак дулкынлы, акчарлаклы
Агыйделем — моңдашым …
Агыйделем — моңдашым …
```
