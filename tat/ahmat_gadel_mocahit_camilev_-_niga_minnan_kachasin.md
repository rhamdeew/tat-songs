# Оригинал

### Әхмәт Гадел / Мөҗәһит Җәмилев - Нигә миннән качасың?

```
Белә алмыйм сәбәп нидә,
Нигә миннән качасың?
Булыр-булмас дусларыңа
Яш(е)рен серең ачасың?

Судан чыктым — утка салдың,
Дөрләп яна йөрәгем.
Уйламаган идем һич тә —
Синнән хәсрәт күрәмен.

Һич беләлмыйм сәбәп нидә,
Нигә болай качасың?
Нигә инде уртак серне
Чит ятларга ачасың?!
```
