# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Дөнья матур син булганга

```
Башка беркем кирәк түгел,
Мин бәхетле синең янда.
Әй сөеклем— асыл кошым,
Дөнья матур син булганга!

Синең нурлы карашыңнан
Тарала күк болытлар да.
Болытлы көн — кояшлыдай,
Дөнья матур син булганга.

Уйлар бер булса тормышта
Җиңел үтелә юллар да.
Газабы да рәхәт сыман,
Дөнья матур син булганга.

Сиңа булган мәхәббәтем
Күңелемә уелган ла.
Юк бер борчу, юк бер кайгы–
Дөнья матур син булганга.

Шат күңелем,син сөеклем,
Елмаеп алда торганда.
Караңгы төн якты көндәй–
Дөнья матур син булганга.
```
