# Оригинал

### Галиәхмәт Шаһи / Ринат Гобәйдуллин - Рәхмәт, тормыш!

```
Рәхмәт, тормыш, газиз башны
Шушы яшькә җиткердең.
Суга салдың, тартып алдың,
Утларыңда киптердең.

Исеңдәме, мине кәкере каенга син —
Терәттең, терәттең.
Авыз тулы кан булганда, төкермәскә
Өйрәттең, өйрәттең.

Рәхмәт, тормыш, газиз башны
Шушы яшькә җиткердең.
Суга салдың, тартып алдың,
Утларыңда киптердең.

Яшәргә син илһам бирдең, күңелемне
Үстердең, үстердең.
Егылганда, кабат торып китәргә, син
Көч бирден, көч бирден.

Рәхмәт, тормыш, газиз башны
Шушы яшькә җиткердең.
Суга салдың, тартып алдың,
Утларыңда киптердең.

Кайгылар белән сынадың, шатлыклар белән
Сынап кара, сынап кара.
Мин көтәм әле, хәзер дә соң түгел,
Уйлап кара, уйлап кара.

Рәхмәт, тормыш, газиз башны
Шушы яшькә җиткердең.
Суга салдың, тартып алдың,
Утларыңда киптердең.
```
