# Оригинал

### Флария Терегулова / Ильяс Аллаяров - Зәңгәр болыт

```
Таңда сөләдем серемне
Күзен ачкан гөлләргә.
Җил булып сибелсен хисем,
Барып җитсен сөйгәнгә.

Зәңгәр болыт сөюемне
Тартканмы тауларга?
Сөю тауларнда йөрим
Сөйгән ярым син кайда?

Яңгырлар җырлап уздылар
Серен сипкән сукмактан.
Алган серне зәңгәр болыт
Чәчәк тулы акланнан.

Зәңгәр болыт сөюемне
Тартканмы тауларга?
Сөю тауларнда йөрим
Сөйгән ярым син кайда?

Җыр булып кирегә кайтты,
Барып җиткәндер тавыш.
Моңнары уралды миңа
Булып ла сөю-сагыш.

Зәңгәр болыт сөюемне
Тартканмы тауларга?
Сөю тауларнда йөрим
Сөйгән ярым син кайда?
```
