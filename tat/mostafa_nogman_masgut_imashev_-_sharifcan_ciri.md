# Оригинал

### Мостафа Ногман / Мәсгуть Имашев - Шәрифҗан җыры

```
Урал тауларының, ай, иркене,
Йөзә ак болытлар төркеме.
Исән генә булып, туган илгә
Әйләнәсе иде беркөнне.

Сандугачлар уйный, ай, ояда,
Урал бөркетләре кыяда.
Җаннарыңа якын дуслар белән
Яши алсаң икән дөньяда.

Урал тауларының, ай, иркене,
Йөзә ак болытлар төркеме.
Исән генә булсак, туган илгә
Әйләнербез әле бер көнне.
```
