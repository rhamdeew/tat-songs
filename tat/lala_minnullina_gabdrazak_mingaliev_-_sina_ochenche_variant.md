# Оригинал

### Ләлә Миңнуллина / Габдразак Миңгалиев - Сиңа (өченче вариант)

```
Без танышкан төндә ятлар идек,
Уйнап-көлеп кенә сөйләштек,
Ятлар безгә әгәр күз салсалар,
Бер-беребездән никтер көнләштек.

Китмә, китмә зинһар яннарымнан,
Бер елмаеп миңа кара әле,
Гашыйк булдым серле күзләреңә,
Минем яннарымда кал әле.

Синең серле күзләреңә карап,
Тулган күңелемне юатам.
Көтмәгәндә гашыйк булдым, ахыры,
Ахыры, сине чынлап яратам.

Китмә, китмә зинһар яннарымнан,
Бер елмаеп миңа кара әле,
Гашыйк булдым серле күзләреңә,
Минем яннарымда кал әле.
```
