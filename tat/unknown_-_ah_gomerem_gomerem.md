# Оригинал

### Шәмсия Җиһангирова / Фәрит Хатыйпов - Аһ, гомерем, гомерем

```
Карлар да бетә инде,
Язлар да җитә инде,
Кар суларына ияреп,
Гомерләр үтә инде.
Каршыма чыксаң иде,
Кулыңны сузсаң иде,
Барган җиреннән гомерне
Туктатып торсаң иде.
Туктатып торсаң иде.

Тукталып торсын иде,
Жәлләгән булсын иде,
Бөтенләй туктамаса да
Әкренләп узсын иде.
Язгы тамчылар тамганны
Бергә санашсын иде,
Узам диган чакта гына
Юлын адашсын иде.
Юлын адашсын иде.

Карлар да бетә инде,
Язлар да җитә инде,
Кар суларына ияреп
Гомерләр үтә инде.
АҺ, гомерем, гомерем,
Тукталып торсын иде,
Бөтенләй туктамаса да,
Әкренләп узсын иде.
Әкренләп узсын иде.
```
