# Оригинал

### Ризида Мөхәммәтҗанова - Сиңа юл алдым

```
Чәчәккә тулы ялан,
Нигәдер ошый балан.
Чәчәгенең ак таҗлары —
Гүя яшьлегем язлары —
Күңелкәем сездә һаман.

Кушымта:
Яшьлегем, син кайда китеп
Әйтче, югалдың?
Яшьлек, сиңа җырым белән
Күрче — юл алдым.

Җимешләргә бай урман,
Нигәдер ошый балан,
Җимешенең ачылыгын,
Яшьлегемнең яктылыгын
Җырларымда язам һаман.

Кушымта.

Карларга тулды ялан,
Җимешле һаман балан.
Бозланган да җимешләре…
Үткән яшьлек гомеркәен
Көйләремә кабат салам.

Кушымта.
```
