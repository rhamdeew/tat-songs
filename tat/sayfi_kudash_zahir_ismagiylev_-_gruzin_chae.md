# Оригинал

### Сәйфи Кудаш / Заһир Исмәгыйлев - Грузин чәе

```
Грузинский чәй чәйнектә,
Өстәлдә башкорт балы,
Яныңда җаның сөйгәне, —
Башка ни кирәк тагы?

Грузин чәе тәмле була,
Эчсәң юкә бал белән,
Кара-каршы утырып кына
Җаның сөйгән яр белән,.

Әй, эчәсен өреп-өреп,
Атылып тирләр чыкканчы,
Тастымалларны сыкканчы,
Самоварны екканчы,

Тирләп-пешеп чәй эчкәчтен
Баллап, каймаклап кына,
Барып тагын биш норманы
Эшләп кайтырга була.

Әй, тульский ак самовар,
Баллы, майлы өстәлең,
Үзе бит бер озын гомер
Тирләп бер чәй эчкәнең.
```
