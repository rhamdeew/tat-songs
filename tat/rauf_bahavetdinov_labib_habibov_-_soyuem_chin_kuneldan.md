# Оригинал

### Рәүф Баһаветдинов / Ләбиб Хәбибов - Сөюем чын күңелдән

```
Сагышлардан үксеп елыйм,
Күз яшемдә тоз тәме.
Синсез яшәү яшәү түгел,
Яшәвемнең юк яме.

Гашыйк җанның гүзәллеге
Шашындырып исерткән.
Йөрәкләрне тоташтырып,
Сине генә үз иткән.

Әлдә сине очыраттым,
Сөюем чын күңелдән.
Сарыларга сабыштырма,
Ташып хисләр түгелгән.

Гашыйк җанның гүзәллеге
Шашындырып исерткән.
Йөрәкләрне тоташтырып,
Сине генә үз иткән.

Сине күргән көннән бирле
Мин бит син дип янам.
Уралсаң да күк томанга,
Барыбер эзләп табам.

Гашыйк җанның гүзәллеге
Шашындырып исерткән.
Йөрәкләрне тоташтырып,
Сине генә үз иткән.
```
