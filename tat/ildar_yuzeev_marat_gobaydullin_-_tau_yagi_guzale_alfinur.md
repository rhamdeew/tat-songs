# Оригинал

### Илдар Юзеев / Марат Гобәйдуллин - Тау ягы гүзәле Әлфинур

```
Китмә син яшьлектән, Әлфинур,
Унҗиде яшеңдә торып кал,
Мәңгегә картая белмәүче
Тау ягы гүзәле булып кал.

Китмә син яшьлектән, Әлфинур,
Үзәнгә сибелсен нурларың,
Әйтсеннәр уфтанып: "Нигә мин
Тау ягы егете булмадым?"

Китмә син яшьлектән, китмә син,
Эзеңдә, йөзеңдә балкый нур,
Унҗиде яшеңдә торып кал,
Җырымда булса да, Әлфинур…
```
