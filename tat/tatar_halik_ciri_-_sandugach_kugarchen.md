# Оригинал

### Татар халык җыры - Сандугач, күгәрчен

```
Сары сандугач тотаем,
Телен ничек отаем?!

Сандугач, күгәрчен,
Гөлләр үскән күрәмсең.

Бергә үстек, бергә йөрдек,
Шуны ничек онытаем?!

Сандугач, күгәрчен,
Гөлләр үскән күрәмсең.

Сары сандугач баласын
Кулларымда сайратам.

Сандугач, күгәрчен,
Гөлләр үскән күрәмсең.

Күрер күзгә чибәр түгел,
Әллә нигә яратам.

Сандугач, күгәрчен,
Гөлләр үскән күрәмсең.
```
