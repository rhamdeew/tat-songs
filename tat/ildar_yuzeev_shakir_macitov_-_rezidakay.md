# Оригинал

### Илдар Юзеев / Шакир Мәҗитов - Резидәкәй

```
Резидәкәй,
Зәңгәр күлмәгеңне
Кайсы елгаларда чайкадың?
Оныттыңмы әллә, бәгырем, мине,
Нигә килми сәлам хатларың?

Резидәкәй,
Алсу йөзләреңне
Шаулы урамнардан эзләдем.
Эзләдем, тик барыбер мин тапмадым,
Өзелеп җыр җырладым, түзмәдем.

Резидәкәй,
Белсә идең шуны:
Синдә генә минем уйларым.
Резидәкәй бәгырем, ник дәшмисең,
Кайда синең зифа ла буйларың?
```
