# Оригинал

### Рәис Риянов / Рафикъ Васфиев - Усолка

```
Усолканың чишмәләре
Яшьлегем төсле икән,
Талган йөрәккә шифасы,
Дәвасы көчле икән.

Бер чыганак әче тозлы,
Дулкыны күбек-күбек;
Икенчесе аның уйный
Шаян терекөмеш кебек.

Һай, мондагы cалкын сулар
Агыйделдән ак икән.
Кояш нурыннан терерәк,
Саф сөюдән пакь икән.

Усолка буенда һәрчак
Ташып елмая сулар.
Күңелләрдә юшкыннар юк,
Сөенә сулар шуңар.
```
