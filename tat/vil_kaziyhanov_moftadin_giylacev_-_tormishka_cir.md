# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Тормышка җыр

```
Тәү башлап чәңгелдәк эченнән
Дөньяга яңгыраган бер тавыш.
Шушы көннән башлап яшәргә
Кабул итеп алган бу тормыш.

Шул чакта әнкәем әйткәндер:
Шушыннан башлана ул тормыш,
Катлаулы яшәүнең юллары,
Тайпылмый атларга син тырыш.

Бу тормыш өйрәтте көләргә,
Әнкәем, телендә сөйләргә,
Халкым һәм туган илемне,
Ярымны өзелеп сөяргә.

Тормыш ул кадерле һәр кемгә,
Хаклыбыз без аны зурларга.
Тормыш үзе яшәү һәм көрәш, —
Шуңа ул мактала җырларда.
```
