# Оригинал

### Гөлнур Хәйруллина / Ильмира Нәгыймова - Гомерем буе

```
Җәйләреңдә көт дип әйтсәң миңа,
Җәйләреңдә сине көтәрмен.
Кызу челлә көнне сусаганда
Чишмәләрдән сулар илтәрмен.

Гомерем буе көт дип әйтсәң миңа,
Сагышларга төренеп көтәрмен.
Җылы җилләр булып исәрмен дә
Арган күңелеңне назлап китәрмен.

Кышларыңда көт дип әйтсәң миңа,
Бураннарга чыгып көтәрмен.
Ап-ак бәсләр сарган агачларны
Чәчәк итеп бүләк итәрмен.

Гомерем буе көт дип әйтсәң миңа,
Сагышларга төренеп көтәрмен.
Җылы җилләр булып исәрмен дә
Арган күңелеңне назлап китәрмен.
Гомерем буе көт дип әйтсәң миңа,
Сагышларга төренеп көтәрмен.
Җылы җилләр булып исәрмен дә
Арган күңелеңне назлап китәрмен…
```
