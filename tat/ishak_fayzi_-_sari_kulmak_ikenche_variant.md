# Оригинал

### Исхак Фәйзи - Сары күлмәк (икенче вариант)

```
Истә, сине тәүге кабат
Су буенда күргәнем.
Сары күлмәгеңә карап,
Ни әйтергә белмәдем.
"Сагышка" дип уйладыммы
Күлмәгең сарылыгын?
Әллә инде, тоймадыммы
Йөрәгең җылылыгын?

Ничә еллар эзләдем мин
Сары төс юлларымда.
Өметемне өзмәдем мин
Каршыма чыгарыңа.
Бер күрүдә гашыйк булып,
Өзелеп сөям, иркәм.
Өстеңдәге сары күлмәк
Бәхеткә булган икән.

Сагышлар — түгел сарыдан,
Хәзер инде аңладым.
Гомерлеккә яннарымнан
Җибәрмәм сине, җаным.
Бер күрүдә гашыйк булып,
Өзелеп сөям, иркәм.
Өстеңдәге сары күлмәк
Бәхеткә булган икән.
```
