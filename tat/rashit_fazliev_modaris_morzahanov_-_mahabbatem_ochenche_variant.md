# Оригинал

### Рәшит Фазлыев / Мөдәрис Морзаханов - Мәхәббәтем (өченче вариант)

```
Үзеңә каратып бер сөй,
Кочаклап ярат инде!
Сине сагынган чакларым,
Сагыштан сакла мине!

Үзеңә каратып бер сөй,
Кочаклап ярат инде!
Үтә кайгылы чакларым,
Кайгымнан сакла инде!

Мәхәббәтем, сиңа чиксез,
Юк җирдә аның тиңе.
Синнән башка яши алмам,
Бәхетсез итмә инде!
```
