# Оригинал

### Нәзирә Ишколова / Риф Сәлишев - Сакла язмыш

```
Шушы тормыш үзе тотыш сынау,
Мәхрүм итмә сөю-назларыңнан.
Өметләрнең көчле канатлары
Аермасын җырлы язларымнан.

Хыянаттан сакла, зинһар, язмыш,
Кайгы-хәсрәтләрдән арала.
Килер көннәремне төссез итмә,
Гомер кыска гына, арада.

Куәт тә бир, авырлыклар бирсәң,
Сабырлык бир йөрәк сызлаганда.
Күңелемдә якты моңнар туса,
Көдрәт өстә ярсып җырлаганда.

Хыянаттан сакла, зинһар, язмыш,
Кайгы-хәсрәтләрдән арала.
Килер көннәремне төссез итмә,
Гомер кыска гына, арада.
```
