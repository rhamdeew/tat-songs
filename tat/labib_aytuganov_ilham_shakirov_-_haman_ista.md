# Оригинал

### Ләбиб Айтуганов / Илһам Шакиров - Һаман истә

```
Йомшак кына җемелдиләр
Йолдызлар җәйге кичтә…
Туган якның зәңгәр күге
Истә, һаман да истә.

Маңгай чәчен тибрәлдереп,
Таңнарда җилләр исә.
Туган якның тын җилләре
Истә, һаман да истә.

Яшел шәл ябына каен,
Сандугачлы яз җитсә…
Шәлен болгап, озатып калган
Сердәш каеным истә.

Яктыра күңел кырларым,
Ал кояш нурын сипсә…
Күңелемнән чыкмый һич тә —
Истә туган як, истә.
```
