# Оригинал

### Абдулхак Игебаев / Данил Хәсәншин - Сөйгән яр (беренче вариант)

```
Сөйгән ярың нинди дисең,
Идел аккошы кебек,
Керфек очкайларын сирпеп,
Бер елмаеп карауы да
Йолдыз балкышы кебек.

Хәтта төшләремдә аның
Назларын тоям кебек,
Ярымның күңелкәе бит
Кояшны, айны сыйдырган
Тылсымлы җиһан кебек!

Ул янымда булса гына
Шат моң агылыр кебек,
Тормыш дулкыннары капыл
Сөйгән ярымнан аерса,
Җаным ярылыр кебек!
```
