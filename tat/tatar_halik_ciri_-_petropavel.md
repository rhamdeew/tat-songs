# Оригинал

### Татар халык җыры - Петропавел

```
Сикереп төштем бакчага
Нәзек чыбык алмага;
Нәзек чыбык, тал чыбык ла,
Җанкәй көтә талчыгып.

Урамыгыз төз түгел,
Төзүчесе без түгел;
Кызларыгыз ялынмагач,
Ялынучы без түгел.

Сездә айның ничәсе,
Бездә кадер кичәсе;
Җанкәй кушкан кич кунарга,
Барасымы, нишләсе?
```
