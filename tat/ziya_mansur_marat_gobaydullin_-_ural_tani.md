# Оригинал

### Зыя Мансур / Марат Гобәйдуллин - Урал таңы

```
Калка чулпан,
Баш ия төн,
Елмая таң, көн килә.
Елмая таң,
Һәр җир өстен
Балкытып нур бөркелә.

Кайный тормыш:
Гөрли цехлар,
Исә җилләр тын гына.
Иркен сулыш
Ала кырлар,
Мәңге гашыйк мин моңа.

Чал Уралда
Уйный яшь кан,
Күкри яшьлек җырлары.
Тауларында ―
Бөркет чыңы,
Күлдә ― аккош моңнары.
```
