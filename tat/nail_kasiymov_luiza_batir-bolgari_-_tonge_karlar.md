# Оригинал

### Наил Касыймов / Луиза Батыр-Болгари - Төнге карлар

```
Төн пәрдәсен ачып,
Ак мамыктай очып,
Ява карлар, ява карлар.
Сызып яңа юллар
Уза яңа карлар,
Хәтерләтә сине алар, сине алар.

Күбәләктәй җиңел,
Нурлар сибеп җем–җем,
Матур булып карлар ява, карлар ява.
Син генә юк янда,
Ялгыз эзләр карда.
Ялгызлыктан йөрәк яна, йөрәк яна.

Чәчеп җиргә аклык,
Төнге утта балкып,
Тып–тын гына карлар ява, карлар ява.
Яңа яуган карда
Яңа өмет бар ла,
Ялгызлыктан юк шул дәва, юк шул дәва…

Күбәләктәй җиңел,
Нурлар сибеп җем–җем,
Матур булып карлар ява, карлар ява.
Син генә юк янда,
Ялгыз эзләр карда.
Ялгызлыктан йөрәк яна, йөрәк яна.
Тып–тын гына карлар ява, карлар ява…
```
