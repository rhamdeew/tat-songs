# Оригинал

### Илсур Хөснетдин / Фәрит Хатыйпов - Туган илем (беренче вариант)

```
Туган илем, кояш балкый синдә,
Нурлар сибеп, таңнар аттыра.
Иман нуры синдә, шәфкать нуры,
Бабам нигезе кабат яктыра.

Борынгыдан килә татар исеме
Яуда җиңдек явыз дошманны.
Кукрәгемдә йөртәм васыять итеп,
Үтәр өчен илем кушканны.

Һәр намазың булсын саф куңелдән,
Рухың белән яшә киңәшеп.
Без килгәнбез якты бу дөньяга,
Җиңәр өчен җирдә көрәшеп.

Горурлан син, халкым, илем белән,
Татар булып мәңге калырсың,
Изгелектә булсын гел ниятең,
Зур бәхеткә юллар ярырсың.
```
