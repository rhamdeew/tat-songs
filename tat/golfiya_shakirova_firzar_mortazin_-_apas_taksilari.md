# Оригинал

### Гөлфия Шакирова / Фирзәр Мортазин - Апас таксилары

```
Эх, Апас яклары,
Барлык уңайлыклары.
Анда-монда йөреп тора
Апасның таксилары.

Әби-бабай рәхмәт әйтә,
Ә кызлар елмаялар.
Озак көтәсе дә юк,
Кирәк җиргә баралар.

Эх, Апас яклары,
Барлык уңайлыклары.
Анда-монда йөреп тора
Апасның таксилары.

Дусларны каршы алырга
Торасы юк аптырап.
Элекке кебек тормыйбыз
Җил-яңгырда калтырап.

Эх, Апас яклары,
Барлык уңайлыклары.
Анда-монда йөреп тора
Апасның таксилары.
```
