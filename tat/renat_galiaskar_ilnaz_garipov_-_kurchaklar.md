# Оригинал

### Ренат Галиәскәр / Илназ Гарипов - Курчаклар

```
Җанны салып калдыр идең
Бер күлмәк төсле.
Тик бу хисләр миннән күпкә
Көчле… үзсүзле…

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар…
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар…

Сикереп төшсәң иде дә
Галәмгә җирдән.
Котылсаң иде мәңгегә
Үтмәс бу чирдән.

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар…
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар…

Язсаң иде сиңа илткән
Өметсез юлдан.
Хисләремә алмаш итеп
Томан… кар… тузан…

Синсез илһам килмәс миңа.
Ятлар сине назлар, кочаклар…
Мәҗбүриләп сыкма яшьләреңне.
Күз яшьләрсез елый курчаклар…
```
