# Оригинал

### Саҗидә Сөләйманова / Рәис Нәгыймов - Диңгезче язы

```
Ак болытлар йөзә,
Кош юллары уза,
Фирүзәдәй күкне күзәтәм,
Сезнең якка, дуслар,
Туган якка, дуслар,
Кәрван-кәрван кошлар озатам.

Таш кыялар ягы,
Олы диңгез яры,
Шомлы диңгез яры давыллы.
Кайтасызмы, кошлар,
Китәсезме, кошлар,
Көтә сезне туган авылым.

Ак томаннар төшә,
Акчарлаклар йөзә,
Дулкыннарга кунмый торналар.
Ерак-ерак ярдан
Оча кәрван-кәрван
Аккаенлы яшел урманга.

Яшел диңгез — кырлар
Тургай җырын тыңлар,
Сандугачлар сайрар талларда.
Исән кайта кошлар!
Кошлар сыман, дуслар,
Без кайтырбыз язгы айларда.
```
