# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Саташуларым мәллә?

```
Йокларга җыенсам да
Юрганымны ябынып,
Юк шул инде, йоклап булмый,
Ятам сине сагынып.

Әй, нишләдем мин?
Саташуларым мәллә?
Икебезгә җиткән юрган
Җитми берүземә дә.

Әй, рәхәт булган икән
Янымда син булганда.
Тәнем түгел, өши күңел,
Җылыта алмый юрган да.

Әй, нишләдем мин?
Саташуларым мәллә?
Икебезгә җиткән юрган
Җитми берүземә дә.

Ничек итеп ябынсам да,
Җылыта алмый юрганым.
Ә бәхетләр була бит ул,
Саклый белсәң булганын.

Әй, нишләдем мин?
Саташуларым мәллә?
Икебезгә җиткән юрган
Җитми берүземә дә.
```
