# Оригинал

### Рәзинә Мөхияр / Юлай Үзәнбаев - Дуслар (беренче вариант)

```
Йөрәгемдә утлар янса,
Әйтер сүзем күп җыелса,

Абынганда,егылганда,
Арыганда, бөгелгәндә.
Таянырга дуслар кирәк,
Таянырга дуслар кирәк.

Яраларың ачылганда,
Әрни-әрни авыртҡанда,

Кайгылардан бөгелгәндә,
Ярсый-ярсый елаганда,
Юатырга дустлар кирәк,
Юатырга дуслар кирәк.

Шатлыкларың ташып чыкса,
Уңышлардан баш әйләнсә,

Шатлыклы көннәр булса да,
Кайгылы төннәр булса да,
Таянырга дуслар кирәк,
Юатырга дустар кирәк.
Таянырга дуслар кирәк,
Ни булса да дуслар кирәк!
```
