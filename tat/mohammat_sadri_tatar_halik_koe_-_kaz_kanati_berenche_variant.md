# Оригинал

### Мөхәммәт Садри / Татар халык көе - Каз канаты (беренче вариант)

```
Каз канатларын эзләргә
Төштем су буйларына.
Су буйларында йөргәндә,
Син генә уйларымда.

Каз канатларын санадым,
Тезелеп кагынганда.
Тулган айларга карадым,
Өзелеп сагынганда.

Каз канатын кагынганда,
Мамыкларын җил ала.
Яшь вакыттагы мәхәббәт
Мәңгелек булып кала.
```
