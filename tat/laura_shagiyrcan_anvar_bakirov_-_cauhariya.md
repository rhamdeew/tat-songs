# Оригинал

### Лаура Шагыйрьҗан / Әнвәр Бакиров - Җәүһәрия

```
Җәүһәрия, бер елмайдың,
Хисләремне кузгаттың.
Кыйгач кашың җәя итеп,
Йөрәгемә ук аттың.

Күз тимәсен, бигрәк сылу
Икән Җәүһәриясе.
Булсаң икән, булсаң икән
Сылу җәүһәр иясе…

Качтың үзең, чиерттең дә
Күңелем кылларына.
Хисләремне кушып үрче
Ефәк толымнарыңа.

Күз тимәсен, бигрәк сылу
Икән Җәүһәриясе.
Булсаң икән, булсаң икән
Шундый җәүһәр иясе…
```
