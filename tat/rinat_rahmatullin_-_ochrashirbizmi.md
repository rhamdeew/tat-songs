# Оригинал

### Ринат Рахматуллин - Очрашырбызмы…

```
Очрашырбызмы диеп,
Кавышырбызмы диеп,
Сорыйм сандугачтан,
Чишмәләргә эндәшәм,
Йолдызларга да карыйм,
Сине генә көтәм мин һаман.

Кирәкми миңа бүтән,
Мин аны гына көтәм,
Кил әле яннарыма.
Тәкыялар мин үрәм,
Төшемдә аны күрәм,
Ул минем җырларымда.

Язмыш миннән көләме,
Ул башканы сөяме,
Әйтегезче, талларым?
Син бит мине күрмисең,
Мин барлыгын белмисең,
Аерыла юлларым…
```
