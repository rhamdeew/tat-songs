# Оригинал

### Муса Җәлил / Заһид Хәбибуллин - Сагыну (икенче вариант)

```
Гөл чәчәкләрен өзәм мин,
Юлларына тезәм мин;
Бик сагындым, бәгырем,
Бик сагындым сине,
Әле дә ничек түзәм мин.

Таң атканда уянырмын.
Җилкәмә шәл салырмын
Таң шикелле балкып.
Иртән кайтыр булсаң,
Каршы чыгып алырмын.

Җилбер-җилбер җил искәндә,
Үләнгә чык төшкәндә
Бик күңелле булыр.
Бик кызыклы булыр,
Кайтсаң, сагынып көткәндә.
```
