# Оригинал

### Рамил Гыймрани - Мин ышанам

```
Күк йөзендә талгын гына очкан
Торна тавышларын ишетәм.
Сагыш белән үзем ялгыз калам
Шул торналар белән син китсәң.

Сынаулар бит һәркемгә дә килә,
Түзәргә шул, җаным, көч кирәк.
Мәхәббәтең ачы җилләренә
Түзгәннәре була бик сирәк.

Өмет сүнми кошларың да кайтыр,
Сагыш кына җанны өзгәли.
Мин ышанам берсен-берсе сөйгән
Җаннар генә түзә беләдер.
```
