# Оригинал

### Венера Асаева / Роберт Тимербаев - Исләремә төшәр кебек

```
Яздан киттем,
Наздан киттем,
Синән киттем
Ятим иттем
Йөрәгемне.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.

Сөйгән күзләр,
Назлы сүзләр,
Җылы куллар,
Кайтыр юллар
Сагындырды.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.

Синсез сулып,
Сагыш тулып
Йөрәккәем
Коштай булып
Талпындырды.

Исләремә төшәр кебек,
Исләремә төшәр кебек,
Төшләр кебек.
```
