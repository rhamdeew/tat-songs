# Оригинал

### Сибгат Хәким / Рөстәм Яхин - Ни сөйли урман

```
Иртән барсак гамьсез, хәсрәтсез ул,
Яфраклары йомшак, саф нурдан,
Бишек җырын көйләп колагыма
Ни сөйли урман, ни сөйли урман.

Көндез керсәң, шаулый горурланып,
Тавышы гөрләп менә тамырдан,
Әкиятләрнең теле белән миңа
Ни сөйли урман, ни сөйли урман,

Кичен килсәң, ул уйларын җыя,
Оеп утыра кебек арудан,
Күптәнге бер истәлеккә бирелеп
Ни сөйли урман, ни сөйли урман.
```
