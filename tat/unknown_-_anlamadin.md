# Оригинал

### Зөһрә Фәйзуллина - Аңламадың

```
Сөюемнең кайнарлыгын
Аңламадың, үкенеч.
Юлларыбыз аерылса да
Оныта алмам инде һич.

Бакчамдагы бер сандугач
Моңланып бер сайрады.
Белсә иде күңелдәге
Син калдырган яраны.

Сәламнәремне җибәрәм
Син яшәгән якларга.
Икебезгә язган хисне
Ник бирдең син ятларга?
Ник тараттың ятларга?

Бакчамдагы бер сандугач
Моңланып бер сайрады.
Белсә иде күңелдәге
Син калдырган яраны.

Бәлки язмыш очраштырыр
Уйламаган бер мәлдә.
Исемеңне җырга куша
Сине сөйгән бер бәндә.

Бакчамдагы бер сандугач
Моңланып бер сайрады.
Белсә иде күңелдәге
Син калдырган яраны.
```
