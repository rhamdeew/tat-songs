# Оригинал

### Татар - Башкорт халык җыры - Шахта (өченче вариант)

```
Китә казлар, китә казлар,
Китә казлар Донбасска шул;
Миндә карап (уйлап) торам әле
Шул казлардан калмаска.

Сусадым, сулар тапмадым
Үзем су уйларымда да.
Гомеремнең буйларында
Булырсың уйларымда.

Аргы яктан, бирге якка
Кемнәр күпер салганнар шул?
Без аерылдык та сайргайдык,
Саргаймасын калганнар.
```
