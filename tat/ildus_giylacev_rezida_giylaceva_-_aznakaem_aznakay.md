# Оригинал

### Илдус Гыйләҗев / Резидә Гыйләҗева - Азнакаем, Азнакай

```
Без тирбәнгән бишек кебек
Уңайлы кечкенәсең,
Чатыр тавым итәгендә
Иркәләп үстерәсең.

Азнакаем, Азнакай,
Яшьлегем каласы син.
Дөнья күреп таныткан
Күземнең карасы син,
Күземнең карасы син.

Осталыкка тапшырабыз
Кырларда имтиханны,
Зур эшләргә кузгалганда
Сорыйбыз фатихаңны.

Азнакаем, Азнакай,
Бәхетем төбәге син.
Нечкә кылы җырымның,
Сандугач тирәге син.

Какшамас өч терәгем бар —
Язмышым сөенече:
Туган илем, туган анам,
Азнакай өченчесе.

Азнакаем, Азнакай,
Киләчәк каласы син,
Соңгы көнемә кадәр
Озата барасы син.
```
