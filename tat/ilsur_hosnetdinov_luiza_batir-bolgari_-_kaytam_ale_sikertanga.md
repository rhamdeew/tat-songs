# Оригинал

### Илсур Хөснетдинов / Луиза Батыр-Болгари - Кайтам әле Сикертәнгә

```
Кайтам әле, кайтам Сикертәнгә,
Анда әнкәм, анда дусларым.
Шунда иде, шунда сөйгән ярым,
Яшьлегемдә йөргән сукмагым.
Шунда иде, шунда сөйгән ярым,
Яшьлегемдә йөргән сукмагым.

Су буенда шомырт исе,
Таң алдыннан кошлар сайрый.

Су буенда шомырт исе,
Таң алдыннан кошлар сайрый.
Кулларыңда моңлы тальян гармун,
Чибәр егет кызлар сайлый.

Гөлбакчалы минем Сикертәнем,

Башым иеп, чишмәләрнең суын
Синдә бит мин, авылым, эчкәнмен.
```
