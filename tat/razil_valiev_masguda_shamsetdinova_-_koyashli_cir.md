# Оригинал

### Разил Вәлиев / Мәсгудә Шәмсетдинова - Кояшлы җыр

```
Елмаеп карыйлар, —
Ал, зәңгәр үзләре…
Җирдәге чәчәкләр —
Кояшның күзләре.

Йөргәнмен ишетми
Мин аны элгәре…
Сандугач теледер —
Кояшның телләре.

Манды ул алтынга
Җәйләрне, көзләрне.
Урманда, кырларда —
Кояшның эзләре.

Кояшның үзе күк
Хыяллар югары.
Кояшлы булсын гел
Балачак юллары.
```
