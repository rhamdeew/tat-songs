# Оригинал

### Нури Арсланов / Александр Ключарев - Миләүшә (беренче вариант)

```
Шуклыгы җитәр микән дә,
Язын көз итәр микән?
Торналар кебек еракка
Бер очып китәр микән?

Көлгәләгем, күбәләгем, Миләүшә!
Үзе бер сөймәгән була,
Бер карасаң, көнләшә.

Еракларга китәр булса,
Канатларын каермам.
Яратып яр итәр булса,
Гомерлеккә аерылмам.

Көлгәләгем, күбәләгем, Миләүшә!
Үзе бер сөймәгән була,
Бер карасаң, көнләшә.

Борчулар үтәр әле лә,
Сагышлар бетәр әле.
Өметсез шайтан дигәннәр —
Мин өмет итәм әле.

Көлгәләгем, күбәләгем, Миләүшә!
Үзе бер сөймәгән була,
Бер карасаң, көнләшә.
```
