# Оригинал

### Гөлнара Сабитова / Оскар Усманов - Өзгәләнмә

```
Язмышлардан язмышларга күчеп
Кемне, бәгърем, бүген эзлисең?
Үткәннәрне хаталарга бәйләп
Юлларыңда кемне күзлисең?

Очрашканда карама син
Өзгәләнеп күзләремә.
Үткәннәргә юллар ябык,
Чирәм үскән эзләренә.

Ташка тиңлисеңдер йөрәгемне,
Әллә миңа җиңел дисеңме?
Йөрәгемне учлап басып торам,
Алалмыйча күктән күземне.

Очрашканда карама син
Өзгәләнеп йөзләремә.
Кирәк түгел инде хәзер
Яз кояшы көзләремә.
```
