# Оригинал

### Нәби Дәүли / Әнвәр Бакиров - Син мине кичер

```
Яз балкышын күргән иртәдә,
Кыр казлары очып үткәндә,
Мине исеңә төшер, исеңә төшер,
Йөрәгеңне кайгы алганда,
Күзләреңнән яшьләр тамганда,

Син мине кичер, мине,
Син мине кичер, мине.
Мин гаепле,
Бары мин үзем…

Чәчләреңне үреп тарасаң,
Көзгеләргә килеп карасаң,
Мине исеңә төшер, исеңә төшер,
Син кайларда диеп сорасаң,
Сагышланып авыр сызлансаң,

Син мине кичер, мине,
Син мине кичер, мине.
Мин гаепле,
Бары мин үзем…

Ялгыз калып авыр уйлансаң,
Җырлар җырлап ялгыз моңлансаң,
Мине исеңә төшер, исеңә төшер.
Нигә киттем, нигә, хуш, дидем,
Мин гаепле, бары мин үзем…

Син мине кичер, мине,
Син мине кичер, мине.
Мин гаепле,
Бары мин үзем…
```
