# Оригинал

### Әхмәт Ерикәй / Сәйяр Хәбибуллин - Сөясеңме, дип, ник сорыйсың?

```
Дулкынланып карыйм басуларга —
Син үстергән алтын бодайга.
Сөясеңме, дип, ник сорыйсың,
Билгеле бит инде болай да.

Күргән саен кыңгырау чәчәкләрен
Сине күрәм төсле тугайда.
Сөясеңме, дип, ник сорыйсың,
Билгеле бит инде болай да.

Күз алдыма сине китерәм мин,
Бер йолдызга карап, бер айга.
Сөясеңме, дип, ник сорыйсың,
Билгеле бит инде болай да.
```
