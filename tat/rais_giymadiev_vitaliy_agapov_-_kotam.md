# Оригинал

### Рәис Гыймадиев / Виталий Агапов - Көтәм!

```
Сине көтә күңелкәем,
Күптәннән юк килгәнең.
Елына бер чәчәк ата
Минем кура җиләгем.

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоек чәчәгендә.

Минем әле һәрнәрсәгә
Куанып яшәр чагым.
Бар куанычым синдә бит,
Картайма инде, җаным!

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоек чәчәгендә.

Бәлки, сөю бакчабызда
Мин гади бер чәчәктер,
Синдәй мәхәббәт кошлары
Бу дөньяда сирәктер.

Яңа аткан таңнар белән
Хисләр туа күңелдә.
Былбылым, кичә сайрадың
Төнбоек чәчәгендә.
```
