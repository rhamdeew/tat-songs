# Оригинал

### Марсель Галиев / Ганс Сәйфуллин - Актүбә чишмәләре

```
Әй, туган җир, җилләреңә
серләрем сөйлим әле.
Яшьлектә туган җырларны
сагынып көйлим әле.

Җырлап ага, уйнап ага
Актүбә чишмәләре.
Суларын эчсәң, сулары
телләрне чишәр әле,
Актүбә чишмәләре
исеңә төшәр әле.

Әй туган җир, тауларыңа
таңнарда меним әле.
Кояшның тауга таянып
чыкканын курим әле.

Җырлап ага, уйнап ага
Актүбә чишмәләре.
Суларын эчсәң, сулары
телләрне чишәр әле,
Актүбә чишмәләре
исеңә төшәр әле.

Әй, туган җир, тугайларда
адашып калыйм әле.
Беркем дә сукмак салмаган
чишмәләр табыйм әле.

Җырлап ага, уйнап ага
Актүбә чишмәләре.
Суларын эчсәң, сулары
телләрне чишәр әле,
Актүбә чишмәләре
исеңә төшәр әле.

Әй туган җир, юлларыңнан
җәяүләп урыйм әле.
Гомерләр бит ике килми,
бәхеткә юрыйм әле.
```
