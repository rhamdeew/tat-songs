# Оригинал

### Алмаз Әхмәтов / Исмай Шәмсетдинов - Туган ягыбыз — Биләр

```
Урманга бай, кырларга бай
Туган ягыбыз — Биләр.
Бездә үскән егетләрне
Арыслан кебек, диләр.

Чирмешән буйлары ямьле,
Ял итәргә килегез.
Эштә уңган егетләрне
Безгә килеп күрегез!

Су буенда, киң болында
Була сабан туйлары
Бер күрүдә гашыйк итәр
Безнең Биләр кызлары.

Чирмешән буйлары ямьле,
Ял итәргә килегез
Эштә уңган яшь кызларны
Безгә килеп күрегез!

Мул игеннәр шаулап үсә
Без эшкәрткән җирләрдә
Тыныч хезмәт, шат мәхәббәт
Чәчәк ата Биләрдә

Чирмешән буйлары ямьле,
Ял итәргә килегез.
Бәхетле матур тормышны
Безгә килеп күрегез!
```
