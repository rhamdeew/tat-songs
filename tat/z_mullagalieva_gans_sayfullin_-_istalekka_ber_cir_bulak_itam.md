# Оригинал

### З. Муллагалиева / Ганс Сәйфуллин - Истәлеккә бер җыр бүләк итәм

```
Үтмәс тесле иде бергә чаклар,
Мәңгелек җәй иде йөрәктә.
Кошлар китеп бара безнең яктан,
Моң калдырып туган төбәктә.

Күңелем тула, көзләр җитә икән,
Сары сагыш тулган дөньяга.
Истәлеккә бер җыр бүләк итәм,
Җырларсыңмы күңелең тулганда?

Җәйнең яме кошлар белән китә.
Истәлеккә кала җырлары.
Үтмәс төсле көннәр үтә икән,
Такыялар кала нибары.

Күңелем тула, көзләр җитә икән,
Сары сагыш тулган дөньяга.
Истәлеккә бер җыр бүләк итәм,
Җырларсыңмы күңелең тулганда?
```
