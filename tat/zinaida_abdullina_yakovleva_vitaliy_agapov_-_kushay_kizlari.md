# Оригинал

### Зинаида Абдуллина (Яковлева) / Виталий Агапов - Кушай кызлары

```
Чәчкәйләрең озын толым,
Алсу йөзең сөйкемле.
Бакча түтәлләрендәге
Мәк чәчәге шикелле.

Их, Кушай кызлары,
Чибәрләрдер үзләре.
Үзәкләремне өзәдер
Нур балкыган йөзләре!

Күзләреңә текәп карыйм,
Ниләр бар күңелеңдә?
Карап миңа бер елмайчы,
Бәхетле ит мине дә.

Их, Кушай кызлары,
Чибәрләрдер үзләре.
Үзәкләремне өзәдер
Нур балкыган йөзләре!

Авылыңа көн дә барам,
Чык инде очрашырга.
Ялындырма шулкадәре —
Вакыттыр танышырга.

Их, Кушай кызлары,
Чибәрләрдер үзләре.
Үзәкләремне өзәдер
Нур балкыган йөзләре!
```
