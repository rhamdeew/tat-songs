# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Аллага шөкер! (беренче вариант)

```
Бер максатсыз, гел заяга
Узмасын гомер.
Исәнмесез, яшибез без,
Аллага шөкер!

Кайгы белән, шатлык белән
Уза бу гомер.
Без бит бергә — шунсы шатлык,
Аллага шөкер!

Алла бирсә, җырларбыз без,
Узганчы гомер.
Исәнмесез, җырлыйбыз без,
Аллага шөкер!
```
