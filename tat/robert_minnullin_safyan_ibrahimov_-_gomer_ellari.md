# Оригинал

### Роберт Миңнуллин / Сафьян Ибраһимов - Гомер еллары

```
Язмыш шулай язган безгә —
Җирдә гомер итәбез.
Дөнья буйлап, гомер буйлап
Сәфәр чыгып китәбез.

Артта — гомер еллары ла,
Алда — гомер юллары.

Әй гомернең ул юллары —
Бигрәкләр дә бормалы.

Көннән-көнгә, айдан-айга,
Елдан-елга күчәбез.
Әкияттәге шикелле
Утлар-сулар кичәбез.

Артта — гомер еллары ла,
Алда — гомер юллары.
Әй гомернең ул юллары —
Бигрәкләр дә бормалы.

Атлыйбыз да йөгерәбез,
Чабабыз — тик тормыйбыз.
Арттан дөнья куа килә,
Әлегә тоттырмыйбыз.

Артта — гомер еллары ла,
Алда — гомер юллары.
Әй гомернең ул юллары —
Бигрәкләр дә бормалы.
```
