# Оригинал

### Рөстәм Зәкуан / Фирзәр Мортазин - Әй бу гомер дигәннәре

```
Әй бу гомер дигәннәре,
Күрештерми кемнәрне,
Ник белмисең сине көтеп,
Бәгырем өзелгәнне?

Әй бу гомер дигәннәре,
Сөюдән көйгәннәре,
Кавыштыра ятлар белән,
Аера сөйгәннәрне.

Әй бу гомер дигәннәре,
Сыный ул түзгәннәрне,
Язмыш кабат очраштыра
Сагынып көткәннәрне.

Әй бу гомер дигәннәре,
Сөюдән көйгәннәре,
Кавыштыра ятлар белән,
Аера сөйгәннәрне.

Әй бу гомер дигәннәре,
Ярата сөйгәннәрне,
Очраштыра-кавыштыра,
Ярата белгәннәрне!

Әй бу гомер дигәннәре,
Сөюдән көйгәннәре,
Кавыштыра ятлар белән,
Аера сөйгәннәрне.
```
