# Оригинал

### Татар халык сүзләре / Салих Сәйдәшев - Тургай (беренче вариант)

```
Сайрый тургай, сайрый тургай,
Баласын җуйган бугай.
Баласын җуйган тургайдай,
Ник моңаям мин болай?

Кыйгак-кыйгак каз кычкыра,
Нинди моңлы каз икән?
Җырласам, елыйсым килә,
Әллә гомерем аз микән?!
Моңлы бала түгел идем,
Миңа ниләр булды икән?!
```
