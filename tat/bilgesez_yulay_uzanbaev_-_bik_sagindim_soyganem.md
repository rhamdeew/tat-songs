# Оригинал

### Билгесез / Юлай Үзәнбаев - Бик сагындым, сөйгәнем

```
Бик сагындым, сөйгәнем,
Сине килә күрәсем,
Алсу гына битләреңнән
Сине килә үбәсем.
Ә-әй, сине килә үбәсем.

Бик сагындым, сөйгәнем,
Төшләремә керәсең,
Төшемдә син, өнемдә син —
Нык сагындым күрәсең.
Ә-әй, нык сагындым күрәсең.

Бик сагындым, сөйгәнем,
Мин яратам, беләсең,
Сөюемне ҡабул итми
Үзәгемне өзәсең.
Ә-әй, үзәгемне өзәсең.
```
