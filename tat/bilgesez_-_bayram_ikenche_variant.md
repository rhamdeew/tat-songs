# Оригинал

### Билгесез - Бәйрәм (икенче вариант)

```
Күңелләр шат бүген, йөрәкләр сөенә,
Йөзләрдә елмаю.
Бәйрәмдә без бүген, без буген.
Нигә син, чибәр кыз,
Бик моңсу торасың?
Оялма кил эле, тиңсез бит син үзең,
Син үзең…

Бәйрәм бүген, бәйрәм,
Тыңлыйк дәртле җырлар,
Балкып яна утлар.
Ә без синең белән йөрибез янәшә,
Карап кала дуслар.
Тик мин сине бүген бирмим берсенә дә.
Керик тә иң түргә,
Алсу таңга кадәр биик әле ярсып,
Биик әле бергә…

Ничек рәхәт бүген,
Янымда син булгач.
Син икән бәйрәмнең кояшы — бар яме, бар яме.
Китәсең син иртән,
Мин сине көтәрмен,
Онытма бу кичне,
Бәйрәмгә кил яме, кил яме…

Бәйрәм бүген, бәйрәм,
Тыңлыйк дәртле җырлар,
Балкып яна утлар.
Ә без синең белән йөрибез янәшә,
Карап кала дуслар.
Тик мин сине бүген бирмим берсенә дә.
Керик тә иң түргә,
Алсу таңга кадәр биик әле ярсып,
Биик әле бергә…

Күзлиләр дә бар да, безне күзәтәләр,
Дусларым да сине үз итеп сөйләшә, сөйләшә,
Бәйрәмдәге кебек.
Гомер юлларын да
Үтик микән әллә,
Гел бергә янәшә, янәшә..

Бәйрәм бүген, бәйрәм,
Тыңлыйк дәртле җырлар,
Балкып яна утлар.
Ә без синең белән йөрибез янәшә,
Карап кала дуслар.
Тик мин сине бүген бирмим берсенә дә.
Керик тә иң түргә,
Алсу таңга кадәр биик әле ярсып,
Биик әле бергә…
```
