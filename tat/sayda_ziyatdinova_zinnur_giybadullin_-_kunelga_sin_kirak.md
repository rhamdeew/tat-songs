# Оригинал

### Сәйдә Зыятдинова / Зиннур Гыйбадуллин - Күңелгә син кирәк

```
Күңелгә син бик кирәк —
Кояшым юк синсез күгемдә.
Сызланам, уйланам, моңланам,
Әгәр килмәсәң бер көнендә.

Куңелгә син бик кирәк —
Күз алдымда синең карашың.
Якты йолдыз сүнә таңнарда,
Тик син генә балкып торасың.

Күңелгә син бик кирәк —
Серемне син генә аңлыйсың.
Керфек какмый төннәр кичеп,
Көтеп алган якты таңым син.

Күңелгә син бик кирәк —
Исемең йөрәгем түрендә.
Көтәм өзелеп сине һаман,
Сагыш җыры белән күңелдә.
```
