# Оригинал

### Татар халык җыры - Авыл көе (бишенче вариант)

```
Абау, Алла, уф Алла,
Торган җирләрем кала.
Торган җирләрем жәл түгел,
Туып үскән җир кала.

Идел суын чыккан чакта
Җуйдым беләзегемне.
Китәрсең дә кайтмассың дип,
Өзмәгез үзәгемне.

Агыйделгә басма салдым
Тар булмаска чыгарга.
Бергә булган вакытларны
Исләреңнән чыгарма.

Өфеләргә барам әле,
Идел суы тәмлегә.
Бир кулыңны, мин дә бирәм,
Аерылмаска мәңгегә.

Аерылмаска, аерылмаска,
Аерылмаска мәңгегә.
```
