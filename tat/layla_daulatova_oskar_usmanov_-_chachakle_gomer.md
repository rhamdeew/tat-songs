# Оригинал

### Ләйлә Дәүләтова / Оскар Усманов - Чәчәкле гомер

```
Күкләрдә болыт күренсә
Җил куып алып китә.
Борчуларым, сагышларым
Суларга агып китә.

Сагындырып килер өчен
Үтеп тә бара җәйләр.
Бүгенгедәй гөлчәчәкле
Булсачы гомеркәйләр.

Бу җәйләрнең матурлыгын.
Күңелебезгә алыйк.
Яктылыкның, җылылыкның
Кадерен белеп калыйк.

Сагындырып килер өчен
Үтеп тә бара җәйләр.
Бүгенгедәй гөлчәчәкле
Булсачы гомеркәйләр.

Шат елмаеп кояш килә
Табигатьтә тантана.
Якты, назлы, чәчәкле җәй
Йөрәктә яшәп кала.

Сагындырып килер өчен
Үтеп тә бара җәйләр.
Бүгенгедәй гөлчәчәкле
Булсачы гомеркәйләр.
```
