# Оригинал

### Татар халык җыры - Ай-һай, гөлкәем

```
Җирән атым, ай, ялларын
Алтын тарак белән лә
Тарыймын, шул,
Ай-һай, гөлкәем.

Сез дусларым килә дигәч,
Кайдан калку — шуннан ла
Карыймын, шул,
Ай-һай, гөлкәем,.

Алмагачның, алмаларын
Был-был комлар кунып ла
Ашасын, шул,
Ай-һай, гөлкәем,

Уйнап җибәр, тартып җибәр —
Был-был кошлар сайравына
Охшасын, шул
Ай-һай, гөлкәем.
```
