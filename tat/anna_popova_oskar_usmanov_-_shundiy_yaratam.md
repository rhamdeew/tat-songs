# Оригинал

### Анна Попова / Оскар Усманов - Шундый яратам

```
Көзгемнән инде бик күптән
Елмаюлар нуры китми.
Йөрәк моңга тулып,
Гашыйк булып,
Тик син диеп яшим.

Мин кайнар кочагыңда
Онытам барын да
Синең яныңа
Коштай очам.
Белсәң иде, җаным,
Шундый сагынам,
Шундый яратам.

Сине сөйми мөмкин түгел,
Карашыңнан эреп төшәм.
Бәхет кәсәсеннән
Әкрен генә
Тамчы-тымчы эчәм.

Мин кайнар кочагыңда
Онытам барын да
Синең яныңа
Коштай очам.
Белсәң иде, җаным,
Шундый сагынам,
Шундый яратам.

Сөюең турында сорап,
Чәчәкләрнең таҗын өзмәм.
Болай да билгеле,
Син минеке
Беркемгә дә бирмәм.

Мин кайнар кочагыңда
Онытам барын да
Синең яныңа
Коштай очам.
Белсәң иде, җаным,
Шундый сагынам,
Шундый яратам.
```
