# Оригинал

### Михаил Пляцковский / Владимир Шаинский - Елмаю (өченче вариант)

```
Елмаюдан көннәр яктыра,
Елмаюдан офыкта кояш кабына…
Бүлешсәк без елмаюларны,
Күзләребездә нурлар артыр тагын да .

Шунда бар эшен ташлап,
Болытлар бии башлар,
Чикерткәләр нинди матур җыр башкара.
Инешләрдән юл ала
Идел кебек елгалар,
Дуслык исә елмаюлардан башлана.

Елмаюдан гөлләр уяныр,
Елавыннан туктар иң моңсу яңгырлар.
Тын торган урманны гөрләтеп,
Яшь яфракларның алкышлары яңгырар.

Шунда бар эшен ташлап,
Болытлар бии башлар,
Чикерткәләр нинди матур җыр башкара.
Инешләрдән юл ала
Идел кебек елгалар,
Дуслык исә елмаюлардан башлана.

Елмаюдан һәркем җылына,
Елмаюлар балкысын башта үзеңдә.
Әйдәгез,елмаюларны да
Йолдыз кебек кабызыйк җир йөзендә.

Шунда бар эшен ташлап,
Болытлар бии башлар,
Чикерткәләр нинди матур җыр башкара.
Инешләрдән юл ала
Идел кебек елгалар,
Дуслык исә елмаюлардан башлана.
```
