# Оригинал

### Флария Терегулова / Мөдәрис Газетдинов - Сиңа (икенче вариант)

```
Гомеремнең буйларына,
Бары сине яраттым.
Яраттым да, караттым да,
Бәхетне синдә таптым.

Иңнәргә терәп иңнәрне,
Гел шатлыклар таратып,
Яшикче, иркәм, бергә,
Бер — беребезне яратып.

Йөрәгемдә башкаларга
Урыным бушатмадым.
Тормыш юлын бергә үтик,
Ялгыша курмә тагын.

Иңнәргә терәп иңнәрне,
Гел шатлыклар таратып,
Яшикче, иркәм, бергә,
Бер — беребезне яратып.

Язмышым син, бәхетем син,
Кирәкми миңа башка.
Сөю китмәсә йөрәктән,
Гомерләр үтмәс бушка.

Иңнәргә терәп иңнәрне,
Гел шатлыклар таратып,
Яшикче, иркәм, бергә,
Бер — беребезне яратып.
```
