# Оригинал

### Әлфирә Бикмәтова / Булат Махиянов - Туймазы

```
Туймазының җир астыннан
Ургыла кара алтын.
Шул алтынга җан бирүче
Егетләр уттай ялкын.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.

Туймазы уңганнарына
Буйсына Урал ташы.
Алар ясаган вазалар,
Әйтерсең, йөзек кашы.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.

Туймазыны данга күмә
Алдынгы маяклары.
Эшләргә дәрт, илһам бирә
Җырларга бай яклары.

Әй, Туймазы, Туймазы
Ямьле Өсән буйлары.
Сине сөеп, җырга салам
Күңелдәге уйларны.
```
