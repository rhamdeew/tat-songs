# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Җиләккә дип барган идем…

```
Җиләккә дип барган идем,
Юри генә адаштым,
Сине күргәч, сүзсез калдым,
Адаштым, дип алдаштым.
Синең йөргән юлларыңда
Җиләкләрем түгелде,
Кырын каравыңны белгәч,
Күңелләрем сүрелде.
Җиләклеккә бармам инде,
Бармам инде, адашмам.
Карурманда адаштым, дип,
Инде бүтән алдашмам…
```
