# Оригинал

### Мөнир Ишбулатов / Муса Смаков - Бердәмлектә безнең көчебез

```
Сәлам сезгә, дуслар — туганнар —
Йөрәкләре моңга тулганнар!
Хозур Урал — безнең илебез,
Чәчәк атсын газиз җиребез.

Күңелләрдә безнең бер теләк —
Дус яшик әйдәгез бергәләп.

Күккә ашкан урман — таулары,
Чыңлап ага көмеш сулары
Җир оҗмахы Башкортстанның —
Сиңа, миңа уртак Ватанның.

Күңелләрдә безнең бер теләк —
Дус яшик әйдәгез бергәләп.

Җил тирбәткән каен яфрагы,
Иген үскән изге туфрагы —
Торганы бер олы бәрәкәт,
Янамасын аңа һәләкәт.

Күңелләрдә безнең бер теләк —
Дус яшик әйдәгез бергәләп.

Тотынышыйк кысып кулларны,
Якын итеп ерак юлларны
Тыгыз сафта бергә атлыйк без,
Ватаныбыз котын саклыйк без.

Күңелләрдә безнең бер теләк —
Дус яшик әйдәгез бергәләп.

Тыныч булсын аяз күгебез,
Нур беркесен көләч йөзебез.
Хуҗалары илнең үзебез,
Бердәмлектә безнең көчебез.

Күңелләрдә безнең бер теләк —
Дус яшик әйдәгез бергәләп.
```
