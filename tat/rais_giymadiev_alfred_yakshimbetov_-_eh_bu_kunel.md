# Оригинал

### Рәис Гыймадиев / Альфред Якшимбетов - Эх, бу күңел

```
Килче иркәм күл буена,
Ялдырма кил инде.
Шул мәхәббәт утларында,
Янып үләм бит инде.

Эх, бу күңел дигәне,
Кырлары үзәннәре.
Була бит сөйгән ярларның,
Үзәкне өзгәннәре.

Серле нурлар сибә торган,
Күзләреңә карармын.
Толымнарым сүтә — сүтәч,
Чәчләремне тарармын.

Эх, бу күңел дигәне,
Кырлары үзәннәре.
Була бит сөйгән ярларның,
Үзәкне өзгәннәре.

Үз үземдә урын тапмый,
Күпме йөрермен тагын.
Кайда басканыңны белми,
Очып туймаган чагым.

Эх, бу күңел дигәне,
Кырлары үзәннәре.
Була бит сөйгән ярларның,
Үзәкне өзгәннәре.

Килче иркәм күл буена,
Ялдырма кил инде.
Шул мәхәббәт утларында,
Янып үләм бит инде.

Эх, бу күңел дигәне,
Кырлары үзәннәре.
Була бит сөйгән ярларның,
Үзәкне өзгәннәре.
```
