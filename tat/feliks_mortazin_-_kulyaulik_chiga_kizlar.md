# Оригинал

### Феликс Мортазин - Кулъяулык чигә кызлар

```
Алтыннандыр җепләре,
Көмештәндер җепләре
Кулъяулык чигә кызлар
Ил сагында егетләре
Исән-имин кайтсыннар дип
Кулъяулык чигә кызлар!

Бераз моңсу күзләре
Әмма көләч йөзләре
Кулъяулык чигә кызлар
Тиздән кайтыр егетләр дип
Җырлый, җырлый кич утырып
Кулъяулык чигә кызлар!

Изгедер теләкләре
Сабырдыр йәрәкләре
Кулъяулык чигә кызлар
Армиядә егетләре
Илләр имин булыр диеп
Кулъяулык чигә кызлар!
```
