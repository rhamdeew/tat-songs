# Оригинал

### Роберт Әхмәтҗанов / Рамил Курамшин - Кичер, болыным!

```
Кичер мине, болыным, сабый чактан
Синең куеныңда көч алдым.
Еракларда сине юксынганда,
Син дә сагына дип уйладым.

Кичер мине, болыным, кайткан саен
Гөлләреңне өзгән өчен дә.
Туган җирнең тоеп гүзәллеген,
Тугайларың гизгән өчен дә.

Кичер мине, болыным, күпсенмә син
Чишмәләрдән эчкән суыңны.
Оныттыңмы яшел тауларыңда
Яланаяк чапкан улыңны?

Кичер, болыным, инде ераклардан
Яннарыңа сирәк кайтканга.
Куеныңда калган балачакның
Тавышы килә таңнар атканда.
```
