# Оригинал

### Резедә Гобәева / Эльмир Низамов - Изге җаннар (Гимн Благотворителей)

```
Күкләр имин булса — җаннар тыныч,
Дөнья якты — хыял яшәсә.
Кеше матур изге эше белән,
Изгелекләр аны яшәтә.

Илаһи бер яктылыктан җаннар,
Мең сөенеп, горурлыктан таша.
Җиребезне сөю хисе бездә,
Татарстан, бары синең аша!

Бердәм булыйк һәрбер изге эштә,
Туган илне гөлгә күмик без!
Өметләрне элик калкан итеп,
Яшәү яшәүмени өметсез!?.

Илаһи бер яктылыктан җаннар,
Мең сөенеп, горурлыктан таша.
Җиребезне сөю хисе бездә,
Татарстан, бары синең аша!

Даһиларың сиңа данлы җырлар,
Шагыйрьләрең язган күпме юллар!
Гөлләр иле булып Татарстан
Яшәле син, яшә миллион еллар!

Илаһи бер яктылыктан җаннар,
Мең сөенеп, горурлыктан таша.
Җиребезне сөю хисе бездә,
Татарстан, бары синең аша!
```
