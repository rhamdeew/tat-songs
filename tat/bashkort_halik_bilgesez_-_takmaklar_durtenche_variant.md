# Оригинал

### Башкорт халык / Билгесез - Такмаклар (дүртенче вариант)

```
Гөлҗимеш гөлен яратсаң
Рәнҗемә энәсенә.
Өйгән сагышына да
Төз үтсәдә үзәгеңә.

Ай-ли, Гөлмилә
Гомер ике килми дә.

Абау Аллам ни кылыймын
Былай булуларыма.
Бар хәсрәтем бетсә иде
Иртән торуларыма.

Ай-ли, Гөлмилә
Гомер ике килми дә.
Ай-ли, Гөлмилә
Ятсам йокым килми дә.

Җилсез көндә дә тирбәлә
Безнең бакча таллары.
Бетсә дә ул, табыла ул
Фани дөнъя маллары.

Ай-ли, Гөлмилә
Гомер ике килми дә.
Ай-ли, Гөлмилә
Ятсам йокым килми дә.
```
