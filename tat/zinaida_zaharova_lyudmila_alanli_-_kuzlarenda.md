# Оригинал

### Зинаида Захарова / Людмила Аланлы - Күзләреңдә

```
Күзләреңдә диңгез тирәнлеге,
Дулкынында килә йөзәсем.
Килә бары синең белән генә
Гомер диңгезләрен кичәсем.

Әйтерсең лә кояшның ал нурлары
Уйный синең якты йөзеңдә.
Күңелеңдә туган хисләреңне
Күрсәм иде зәңгәр күзеңдә.

Йөрәгеңдә сөю ялкыннарын
Кем кабызган, килә беләсем.
Килә синең күңел түрләреңдә
Бер нәфис гөл булып үсәсем.

Әйтерсең лә кояшның ал нурлары
Уйный синең якты йөзеңдә.
Күңелеңдә туган хисләреңне
Күрсәм иде зәңгәр күзеңдә.
```
