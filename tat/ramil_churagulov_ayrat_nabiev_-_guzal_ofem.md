# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Гүзәл Өфем

```
Көмеш сулы Агыйделем
Каридел кочагында.
Каршы ала гүзәл Өфем
Биек тау башларында.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!

Гөл-чәчәкле бакчалары,
Бульварлары, урамнары.
Мең йөрәкләрдә уята
Мең-мең мутур җырларны.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!

Салават батыр тулпар атында
Каршы ала, озата.
Өфедә тик бер булганнар
Кире әйләнеп кайта.

Өфем, Өфем, гүзәл Өфем!
Үткәнем, киләчәгем!
Рәсәйемнең матур гөле,
Уралымның гөл-чәчәге!!!
```
