# Оригинал

### Радик Хәкимҗанов / Рәис Вәгыйзов - Аккошлар нигә моңсулар?

```
Аккошлар нигә моңсулар,
Әллә китәләр микән?
Язлар, җәйләр нигә шулай
Бик тиз үтәләр икән?

Кешеләр дә — аккошлардай,
Китәләр еракларга,
Туган җирләр уелып кала
Мәңгегә йөрәкләрдә!

Аккошлар тынып калганнар,
Ниләр уйлана икән?
Туган күлләр кала диеп,
Эчтән моңая микән?

Кешеләр дә — аккошлардай,
Китәләр еракларга,
Туган җирләр уелып кала
Мәңгегә йөрәкләрдә!

Аккошларга да кыендыр
Аерылу күлләреннән,
Аерма язмыш кошларны
Бәхетле көннәреннән!

Кешеләр дә — аккошлардай,
Китәләр еракларга,
Туган җирләр уелып кала
Мәңгегә йөрәкләрдә!
```
