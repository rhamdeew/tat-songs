# Оригинал

### Рәфис Зиязетдинов / Фәнир Галимов - Стәрлебаш яклары

```
Җирнең асыл ноктасы
Стәрлебаш исемле.
Шул якларга ашкындыра
Әллә соң син тылсымлы?!

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!

Кайтыйм әле, күрим әле
Стәрлебаш якларын.
Күңелемнән җуйыйм әле
Сагыну газапларын.

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!

Туган як ул-әни кебек:
Изге дә, кадерле дә.
Мәхәббәтем шуңа көчле,
Сүнми дә, сүрелми дә.

Урманнарында чияләр
«Килче-килче», дияләр.
Син кабызган гыйшык утын
Әллә инде беләләр?!
```
