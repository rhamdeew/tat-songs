# Оригинал

### Зифа Нагаева - Әти-әнине зурлап

```
Әле генә сабый идек,
Әле генә бәләки.
Ни арада оягыздан
Ни арада оягыздан
Очканбыз әни, әти.
Очырдык та оныт түгел,
Бүген дә бала дисез.
Яшәгәнчә без дип канат,
Яшәгәнчә без дип канат
Кага торгансыз бит сез.

Әти! Әни! теләкләрем
Күкрәкләремдә таша.
Сезгә булган хисләремне,
Балалык рәхмәтләремне
Ишеттерәм шушы җырның
Көе, сүзләре аша.
Бәхет шатлык сәламәтлекләр
телим сезгә җырлап.
Гомеремнең буйларына
Әтиемне, әниемне
Яшәсәм иде зурлап.

Яшьлегебез үтте диеп
Уфтанып утырмагыз.
Сезнең яшьлек көзгесе — без!
Сезнең яшьлек көзгесе — без!
Безгә генә карагыз.
Сабырлык — ул әни синнән,
Батырлык — әти синнән.
Тормыштагы түземсезлекләр
генә үзебездән
Мәрхәмәтлекләр сездән!

Әти! Әни! теләкләрем
Күкрәкләремдә таша.
Сезгә булган хисләремне,
Балалык рәхмәтләремне
Ишеттерәм шушы җырның
Көе, сүзләре аша.
Бәхет шатлык сәламәтлекләр
телим сезгә җырлап.
Гомеремнең буйларына
Әтиемне, әниемне
Яшәсәм иде зурлап.

Сезнең янда без дә хаман
Бала булып калабыз.
Хәлләрнең төрле чагы бар,
Хәлләрнең төрле чагы бар…
Сездән көчләр алабыз.
Вакытлары җитми диеп
Көяләнгәнегездә
Сезнең кадәр булмасак та,
Сезнең кадәр булмасак та,
Сез дип янарбыз без дә!
```
