# Оригинал

### Айгөл Вәлиуллина / Дилүс Шәйхразиев - Миләшләрем (икенче вариант)

```
Миләшләргә кызыл төшкән,
Ялкын уты җанга күчкән.
Әллә нәрсә булган минем күңелемә.
Миләш кебек утта янып,
Күзләреңнән җылы алып,
Гашыйк булып йөрим әле
Мин үзеңә.

Миләшләрем уттай дөрли,
Алар серен беркем белми,
Мин дә сиңа ачар идем йөрәгемне.
Кочагыма сине алып,
Яннарыңда мәңге калып,
Сөйләр идем сине ничек сөйгәнемне.

Миләшләр дә гашыйк бугай,
Гүзәл булмас иде болай,
Тәлгәшләрен үбеп ала җилләр килеп.
Йөрәк ярсый, ничек түзәр,
Каян табыйм икән сүзләр
Җанда булган хисләремне
Җиткерерлек.

Миләшләрем уттай дөрли,
Алар серен беркем белми,
Мин дә сиңа ачар идем йөрәгемне.
Кочагыма сине алып,
Яннарыңда мәңге калып,
Сөйләр идем сине ничек сөйгәнемне.
Миләшләрем уттай дөрли,
Алар серен беркем белми,
Мин дә сиңа ачар идем йөрәгемне.
Кочагыма сине алып,
Яннарыңда мәңге калып,
Сөйләр идем сине ничек сөйгәнемне.
Сөйләр идем сине ничек сөйгәнемне…
```
