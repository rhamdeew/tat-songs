# Оригинал

### Гөлнур Корбанова / Габдразак Миңгалиев - Зарланма

```
Һәр көн саен бер ялгызың
Чишмәләргә киләсең,
Сөйгән ярым ташлады, дип,
Күз яшеңне түгәсең.

Зарланм(а) әле дөньядан,
Зарланмачы, әй, сеңел!
Күрче, дөнья нинди матур,
Зарланыр чагың түгел.

Парлыларны күрсәң әгәр,
Үзәкләрең өзелә.
Һәрнәрсәнең үз чиге бар,
Син хәзергә түз генә.

Зарланм(а) әле дөньядан,
Зарланмачы, әй, сеңел!
Уйнап-көлеп, җырлап-биеп,
Шаулап үтсен яшь гомер.
```
