# Оригинал

### Гөлфия Шакирова / Айнур BoooRe Шарапов - Ай-яй

```
Карап торсаң аңа,
Сулыш жанга каба.
Син дип йөрәк яна, яна, яна.
Бөтен егетләр дә
Йөгерә артыннан,
Килеп төштең каян, каян, каян.

Җитмеш егет арасыннан
Сайланма син алай.
Татар егете ярата белә,
Ай-яй, ай-яй.

Күзеңә карасам —
Башларым әйләнә,
Язмышым син түгел, димә, димә.
Яннан үтеп китсә —
Телләрем бәйләнә,
Йөрәк еш-еш тибә, тибә, тибә

Җитмеш егет арасыннан
Сайланма син алай.
Татар егете ярата белә,
Ай-яй, ай-яй.
```
