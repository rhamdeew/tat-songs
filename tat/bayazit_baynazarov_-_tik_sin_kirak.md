# Оригинал

### Баязит Байназаров - Тик син кирәк

```
Мин бәхетле бу тормышта,
Язганга тик син язмышка,
Гүя моңсу бу дөнья мени?
Әйләндердең син оҗмаһка…

Бәхетле без икәү бергә,
Пар аккошлар кебек без дә.
Янып-көеп яшәр өчен
Тик син кирәк миңа җирдә.

Мин бәхетле гомерем буе,
Була алганга сиңа терәк.
Кочагыма алып җылытырмын,
Өши дисәң әгәр йөрәк.

Бәхетле без икәү бергә,
Пар аккошлар кебек без дә.
Янып-көеп яшәр өчен
Тик син кирәк миңа җирдә.

Мин бәхетле, еллар узар,
Маңгайларга сырлар чыгар,
Алда әле, җаным, алда
Җырланмаган безнең җырлар.

Бәхетле без икәү бергә,
Пар аккошлар кебек без дә.
Янып-көеп яшәр өчен
Тик син кирәк миңа җирдә.
```
