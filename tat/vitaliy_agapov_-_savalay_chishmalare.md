# Оригинал

### Виталий Агапов - Сәвәләй чишмәләре

```
Сәвәләйне сагынам мин,
Сызылып таңнар атканда.
Йөрәккәем җилкенәдер,
Сәвәләйгә кайтканда.

Әй, Сәвәләй чишмәләре,
Таулары, инешләре.
Аерылганда мәңгелеккә,
Сездән башка нишләрмен.

Аккошлар да, язлар җиткәч,
Туган җиргә кайталар.
Бу дөньяга мәңгелеккә
Килмәдек бит без, дуслар.

Әй, Сәвәләй чишмәләре,
Таулары, инешләре.
Аерылганда мәңгелеккә,
Сездән башка нишләрмен.
```
