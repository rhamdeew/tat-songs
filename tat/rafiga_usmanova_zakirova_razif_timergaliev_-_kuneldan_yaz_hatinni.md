# Оригинал

### Рафига Усманова (Закирова) / Рәзиф Тимергалиев - Күңелдән яз хатыңны

```
Хат алабыз, хат салабыз
Якыннарга.
Хатлар чорный ялкынга да,
Салкынга да.

Гел ялкынлы сәламнәр яз
Хатларыңда.
Күңелдән яз вакытың юк
Чакларыңда.

Озак килми торса алар,
Өмет өзмә.
Ышанма "ул язмас" дигән
Гайбәт сүзгә.

Гел ялкынлы сәламнәр яз
Хатларыңда.
Күңелдән яз вакытың юк
Чакларыңда.

Хатның була жибәрелми
Калганнары.
Кәгазьләргә язылганы,
Язмаганы.

Гел ялкынлы сәламнәр яз
Хатларыңда.
Күңелдән яз вакытың юк
Чакларыңда.
```
