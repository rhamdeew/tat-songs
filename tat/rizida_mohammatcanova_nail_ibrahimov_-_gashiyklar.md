# Оригинал

### Ризида Мөхәммәтҗанова / Наил Ибраһимов - Гашыйклар

```
Гашыйкларның бәгырьләрен
Ниләр генә өтмәгән.
Йөрәкләрен яралаган —
Саф сөюе үтмәгән.

Кушымта:
Үлмәс сөю гомергә дә,
Гашыйклар булган җирдә.
Каты тормыш юллары да —
Сез киртә түгел бер дә.

Саф сөюгә кергән чакта
Кемнәр ләззәт тоймаган?
Мәхәббәтен югалтканнар
Кайнар яшен коймаган?

Кушымта.

Кышның салкын төннәрендә
Кемнәр генә туңмаган.
Мәхәббәттән бу тормышта
Бик сирәкләр уңмаган.

Кушымта.

Бер-берсенә гашыйк җаннар
Тормышыбыз бизәге.
Бәхетлеләр диңгезендә
Син дә бергә йөз әле.

Ак җилкәнле пар караплар
Диңгезләрдә күп булсын.
Тормышыбыз якты нурга
Сөюгә назга тулсын.
```
