# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Көзге ярату

```
Яңгыр, яңгыр, җылы яңгыр, чәчләр чылата,
Каеннарны чыклар белән, август елата.
Җәй безнең белән хушлаша,
Күңелем минем адаша,
Көзге яңгыр сагышлар ата.

Әй, көз, килми тор, саргайтма ямьне,
Әй, көз, суытма безнең араны.
Көзнең моңлану, яфрак саргайту,
Нигә бик моңсу көзге ярату?

Соңгы җылы яңгыр тама керфек очыннан,
Көзләр белән сары яфрак төшә каеннан.
Көзләр белән сагышларым,
Минем бик моңсу дусларым,
Күңелләрне тибрәтә һаман.

Әй, көз, килми тор, саргайтма ямьне,
Әй, көз, суытма безнең араны.
Көзнең моңлану, яфрак саргайту,
Нигә бик моңсу көзге ярату?

Соңгы җылы яңгыр ява бүген биредә,
Өметләрне сүндермә син, яңгыр күңелдә.
Язмышлардан узмыш бармы?
Әллә көзге юллар тармы?
Моңсу ямьнәр өчен бу җирдә.

Әй, көз, килми тор, саргайтма ямьне,
Әй, көз, суытма безнең араны.
Көзнең моңлану, яфрак саргайту,
Нигә бик моңсу көзге ярату?
```
