# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Каршы чыгып алырбыз

```
Кырлар тынып йокога талганнар
Кар-буранны көтәләр.
Ак туннарын киеп куйганнар,
Кыш бабайны көтәләр.

Бик зарыгып сагынсалар да
Алар сабыр итәләр
Йөгерешеп безнең бәйрәмгә
Чабып килеп җитәләр

Кыш бабайны каршы алырга
Бар җәнлекләр килгәннәр
Безнең җырны ишеткән алар,
Утлар яна күргәннәр.

Кыш бабабыз,синең килгәнне,
Көтәбез без барыбыз.
Тавыш бир, бабай арысаң,
Каршы чыгып алырбыз.
```
