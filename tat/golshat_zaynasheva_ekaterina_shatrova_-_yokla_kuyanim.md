# Оригинал

### Гөлшат Зәйнәшева / Екатерина Шатрова - Йокла, куяным

```
Иң яраткан уенчыгым,
Ап-ак йомшак куяным.
Көне буе уйнасам да,
Бер дә синнән туймадым.

Ай калыкты, төн җитте,
Йоклыйк инде, куяным.
Үз яныма сиңа да
Урын җәеп куямын.

Уенчыкларым күп минем,
Төлкем дә бар, аю да.
Аю баласы тәгәрәп
Йоклый кояш баюга.

Ай калыкты, төн җитте,
Йоклыйк инде, куяным.
Үз яныма сиңа да
Урын җәеп куямын.

Аю, төлке, бүре, керпе,—
Барчасы йоклап киткән.
Бишек җыры җырлый, өнә,
Йолдызлар зәңгәр күктән.

Ай калыкты, төн җитте,
Йоклыйк инде, куяным.
Үз яныма сиңа да
Урын җәеп куямын.
```
