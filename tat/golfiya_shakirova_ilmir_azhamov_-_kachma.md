# Оригинал

### Гөлфия Шакирова / Илмир Әзһамов - Качма

```
Кабынган сөюнең
Сүндермә хисләрен.
Җавапсыз калдырып,
Туңдырма тезләрең.

Качма, качма,
Бәхетеңнән ятларга качма,
Йөрәгеңнән качма, качма,
Бәхетеңнән ятларга качма,
Йөрәгеңнән.

Кабул ит син аны,
Җилләргә очырма.
Килгән мәхәббәтне,
Зинһар, кирегә борма.

Качма, качма,
Бәхетеңнән ятларга качма,
Йөрәгеңнән качма, качма,
Бәхетеңнән ятларга качма,
Йөрәгеңнән.
Качма, качма
Бәхетеңнән.
Качма, качма
Бәхетеңнән…
Качма, качма,
Бәхетеңнән ятларга качма,
Йөрәгеңнән качма, качма,
Бәхетеңнән ятларга качма
Йөрәгеңнән.
```
