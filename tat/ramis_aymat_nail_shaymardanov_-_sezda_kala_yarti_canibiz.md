# Оригинал

### Рәмис Әймәт / Наил Шәймарданов - Сездә кала ярты җаныбыз

```
Утлар сүнәр тиздән бу залларда,
Алкышлар да тынар, җырлар да.
Җырланачак яңа җырларыбыз
Чакыра безне ерак юлларга.

Без китәбез, моңлы җырлар булып
Атсын сезнең hәрбер таңыгыз
Китсәк тә без ерак еракларга,
Сездә кала ярты җаныбыз.

Утлар сүнәр, ләкин йөрәкләрдә
Мәңге сүнмәс кебек бу хисләр.
Безне җыр канатында тирбәткән
Кабатланмас кебек бу кичләр.

Без китәбез, моңлы җырлар булып
Атсын сезнең hәрбер таңыгыз
Китсәк тә без ерак еракларга,
Сездә кала ярты җаныбыз.

Күрешербез әле тагын, дуслар,
Бирсен генә Ходай гомерләр.
Күрешербез сезнең йөрәкләргә
Моңнар белән үреп күперләр

Без китәбез, моңлы җырлар булып
Атсын сезнең hәрбер таңыгыз
Китсәк тә без ерак еракларга,
Сездә кала ярты җаныбыз.
```
