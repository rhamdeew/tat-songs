# Оригинал

### Фәнәвис Дәүләтбаев / Рәзидә Әхмәтова - Язгы матурлык

```
Чәчәккә күмелде, күмелде
Шомыртлар, сиреньнәр.
Җаннарны сихерли, сихерли
Туган җир, синең ямь!

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.

Тыңлап шул туймаслык, туймаслык
Кошларның сайравын.
Тылсымы язларның — язларның —
Агыла җанда моң.

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.

Үләннәр өстенә, өстенә
Чәчәкләр сибелгән.
Яратып туймаслык, туймаслык
Туган җир, синең ямь!

Кыр, алан, болыннар, болыннар
Яшелдән киенә.
Тәбигать яшәрә, яшәрә,
Күңелләр сөенә.
```
