# Оригинал

### Мәхмүт Һибәт / Абдулла Хәлфетдинов - Соңлап килдең

```
Соңлап килдең урау юллардан,
Киң адашкан аккош шикелле.
Синсез калгач язмыш кителде,
Сагыш йотып еллар үтелде.

Күңелемне һаман еллар буе,
Син биләдең шиңмәс гөлләрдәй.
Сагышларны хәзер исләмимен,
Калсыннар алар узган көннәрдә.

Без яшибез әле бергә без,
Бергә аккан ике елгадай.
Ә шулайда әрнү җуелмый
Очрашмадык нигә соңламый?

Соңлап килдең, соңлап, ник шул алай,
Соңлап тудың нигә тулган ай?
Берсен-берсе сөйгән йөрәкләр
Кавышсыннар иде соңламый.
```
