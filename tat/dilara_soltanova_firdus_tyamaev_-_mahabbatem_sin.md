# Оригинал

### Диләрә Солтанова / Фирдүс Тямаев - Мәхәббәтем син

```
Яшьлек хатасы беләнме
Мин сине сөйдем.
Язмышыма минем тигән
Бәхетем дидем.

Мәхәббәтем, бәхет кошым,
Киләчәгем син.
Сөям сине, бары сине,
Сөйгән ярым син.

Чәчләремнән иркәләдең,
Билемнән кочтың.
Шул сөюдән мин исердем,
Күкләргә очтым.

Мәхәббәтем, бәхет кошым,
Киләчәгем син.
Сөям сине, бары сине,
Сөйгән ярым син.

Чынмы синең бу сөюең,
Әллә ялганмы?
Белсәң иде мәхәббәттән
Ничек янганны.

Мәхәббәтем, бәхет кошым,
Киләчәгем син.
Сөям сине, бары сине,
Сөйгән ярым син.
Сөям сине, бары сине,
Сөйгән ярым син…
Сөйгән ярым син…
```
