# Оригинал

### Билгесез - Бөктермә

```
Исләрем китеп йөримен,
Нишләде бу сөйгәнем?
Кыскартам да бетерәм, —
Ди яңа матур күлмәген.

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр — бөктерсеннәр,
Ә син менә бөктермә.

Нигә соң бозырга, җаным?
Килешә бит күлмәгең.
Миңа ошар дисеңме әллә
Бөктереп йөрүләрең?

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр — бөктерсеннәр,
Ә син менә бөктермә.

Әллә нишләп куя күңел,
Күз алдыма китереп.
Синнән башка да җитәрлек
Йөрүчеләр бөктереп.

Үзеңне дә, мине дә син
Адәм хуры иттермә.
Бөктергәннәр — бөктерсеннәр,
Ә син менә бөктермә.
```
