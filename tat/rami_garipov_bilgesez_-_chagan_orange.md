# Оригинал

### Рәми Гарипов / Билгесез - Чаган (Өрәңге)

```
Күрә күктән көзге кояш:
Боега бер карт чаган.
Бар яфрагын койса да ул,
Оч яфрагын саклаган.

Һаман да ул нурга туймый,
Күз төбәгән кояшка.
Ә кояшы сирәк карый,
Җылытмый инде башка.

Өзгеләй җил оч яфрагын,
Һәм кыраулы көзге төн.
Ә ул һаман яшел әле,
Һаман өзми өметен!…
```
