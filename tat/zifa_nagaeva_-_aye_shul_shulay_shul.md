# Оригинал

### Зифа Нагаева - Әйе шул, шулай шул!

```
Җыелган көнебездә
Елмаю йөзебездә.
Әти-әни утырасыз
Ямь биреп түребездә.
Рәхмәт, әткәй, әнкәй дибез,
Туган җанлы итеп безне
Үстергәннәрегезгә.

Әйе шул, шулай шул
Үстергәннәрегезгә.
Әйе шул, шулай шул
Үстергәннәрегезгә.

Кыенда чыданнарбыз,
Кайгыда сыналганбыз.
Һәрберебез туган өчен
Үләрлек туганнар без.
Апа, сеңел, эне, абый,
Берсеннән берсе шәп жаный,
Кая куйсаң да ярый.

Әйе шул, шулай шул
Кая куйсаң да ярый.
Әйе шул, шулай шул
Кая куйсаң да ярый.

Чит як гаилә балалары
Кушылышып киткәнбез,
Кушылышып киткәнбез дә,
Туганлашып беткәнбез.
Җизни, җиңги, килен, кияү,
Бергә гомер үтә-үтә
Туганлашып беткәнбез.

Әйе шул, шулай шул
Туганлашып беткәнбез.
Әйе шул, шулай шул
Туганлашып беткәнбез.

Түгәрәк бәхетләр булсын
Түгәрәк өстәлләрдәй.
Ким хур булмыйк башкалардан,
Булыйк әллә кемнәрдәй.
Күрше керә, дус-иш күрә,
Үз дошман араларында
Булыйк әллә кемнәрдәй.

Әйе шул, шулай шул
Булыйк әллә кемнәрдәй.
Әйе шул, шулай шул
Булыйк әллә кемнәрдәй.
```
