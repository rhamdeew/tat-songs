# Оригинал

### Марс Хафизов / Виталий Агапов - Бер елмаюың җитә

```
Төннәр үтеп таң атканда,
Зефир җилләре исә.
Күңелемә авыр чакта
Бер елмаюың җитә.
Шатлыклары тормышның аз,
Киләм дигәндә китә.
Күзләремә шул чагында
Бер елмаюың җитә.

Елмай әле, бер елмайсаң,
Тарала кайгыларым.
Кабат-кабат чәчәк атсын
Сөюем, тойгыларым.

Вакыт бездән сорап тормый,
Тормыш сукмагын сүтә.
Сүтә бирсен, миңа, иркәм,
Бер елмаюың җитә.
Бергә чакта сөеп калыйк,
Гомер бигрәк тиз үтә.
Бер бәхетне ике иткәч,
Шул елмаюың җитә.

Елмай әле, бер елмайсаң,
Тарала кайгыларым.
Кабат-кабат чәчәк атсын
Сөюем, тойгыларым.
```
