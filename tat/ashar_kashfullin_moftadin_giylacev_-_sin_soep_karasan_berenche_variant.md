# Оригинал

### Әсхәр Кәшфуллин / Мөфтәдин Гыйләҗев - Син сөеп карасаң (беренче вариант)

```
Сандугач тирәктә
Моң сибә йөрәккә.
Кил, бәгърем, син дә кил тыңларга.
Тыңлыйк без бергәләп,
Дөньялар түгәрәк,
Син барда янымда, син барда.

Сайрый да сандугач,
Кояшлы май тугач,
Сандугач — мәхәббәт җырчысы.
Сандугач сайрамый,
Болытлар таралмый,
Җаныңны сагышлар борчыса.

Йолдызлар янганда,
Күз ачып йомгандай
Ул безнең гомерләр арасы.
Сандугач сайраса,
Син сөеп карасаң,
Төзәлә күңелем ярасы.
```
