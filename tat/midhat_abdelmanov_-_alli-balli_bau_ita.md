# Оригинал

### Мидхәт Әбделманов - Әлли-бәлли бәү итә

```
Кояш баткан, Ай калыккан,
Баламны бишек көтә.
Курчаклар да арыганнар,
Әлли-бәлли бәү итә.

Әлли-бәлли бәү, бәү,
Әйтер сүзем тик берәү.
Сау-сәламәт үс, балам.
Ак бәхет теләп калам.

Әлли-бәлли, күз йомылган,
Әкият тә төш күрә.
Бишек җырым болыннарда
Чәчкәдән таҗлар үрә.

Әлли-бәлли бәү, бәү,
Әйтер сүзем тик берәү.
Сау-сәламәт үс, балам.
Ак бәхет теләп калам.

Әлли-бәлли… илең тыныч,
Ата-анаң кырыңда.
Бәхет, тәүфыйк, иман телим
Әлли-бәлли… җырымда.

Әлли-бәлли бәү, бәү,
Әйтер сүзем тик берәү.
Сау-сәламәт үс, балам.
Ак бәхет теләп калам.
```
