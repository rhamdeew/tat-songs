# Оригинал

### Разил Вәлиев / Илгиз Закиров - Кояштай елмай

```
Айның язмышы
Төнгә бәйләнгән.
Кояш тирәли
Мәңге әйләнгән.
Толымы төннең
Чем-кара икән.
Күлмәген дә төн
Карадан теккән.

Кояш тирәли
Ай гел әйләнгән…
Минем язмышым
Сиңа бәйләнгән…
Йолдызлар биек,
Моңсу һаман Ай…
Таратып төнне,
Кояштай елмай.

Минем язмышым
Сиңа бәйләнгән…
Көннәрем шуңа
Төнгә әйләнгән.
Күзләре төннең
Зәңгәрдән икән,
Такыясын төн
Йолдыздан чиккән.

Кояш тирәли
Ай гел әйләнгән…
Минем язмышым
Сиңа бәйләнгән…
Йолдызлар биек,
Моңсу һаман Ай…
Таратып төнне,
Кояштай елмай.
```
