# Оригинал

### Мансур Шиһапов / Сафьян Ибраһимов - Иркәләнер көннәр алда әле

```
Каен тәлгәшеннән моңнар тамып,
Сагыш басса, яшьләр түкмә әле!
Уртаклашыр серләр, бүленгән хис,
Аңлашасы сүзләр күпме әле.

Дуамал җил көтмә диеп шаулар,
Белмәс кичне, белмәс иртәне.
Ярын җуйган ялгыз аккош сыман,
Ярдан ташланырга иртә әле.

Йөрәгемнең сүнмәс (балкыр) нурларында
Бүләк итәр җылы калды әле.
Миләш кызын үпкән тирәк сыман
Иркәләнер көннәр алда әле.
```
