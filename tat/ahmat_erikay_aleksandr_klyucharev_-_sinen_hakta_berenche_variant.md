# Оригинал

### Әхмәт Ерикәй / Александр Ключарев - Синең хакта (беренче вариант)

```
Күктә йолдыз янып тора,
Көмеш нурлар тамып тора.
Көтәм мин, алтыным,
Сине шундый төндә.
Көтәм мин, юксына
Йөрәгем буген дә.

Күктә йолдыз ярсып тора,
Йөрәгемне чәнчеп тора,
Нигә соң ашкына
Шулай минем йөрәк,
Нигә соң юатмый
Мине яшел тирәк?

Синең хакта сорыйм гөлдән,
Әкрен генә искән җилдән.
Матур бер җыр ишетәм
Иркәм синең хакта,
Ләкин ул җыр әллә
Кайда, бик еракта…
```
