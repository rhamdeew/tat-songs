# Оригинал

### Гөлшат Зәйнәшева / Алмаз Монасыйпов - Нигә шулай?

```
Юлга чыксам, иңнәремә үсә канат,
Әйтерсең лә очып барам алга карап.
Юлым ерак йолдызларга илтә кебек,
Йөрәк ярсый, йөрәк ярсый,
Йөрәк тибә гел җилкенеп.

Нигә шулай, нигә шулай икән, дисәм,
Дуслар әйтә, дуслар әйтә, шулай була.
Шулай була, өзелеп сөйсәң.

Эш башласам, ару белмим, талу белмим,
Авырлыклар каршында мин һич бөгелмим.
Янар таулар актарырлык көч бар кебек,
Йөрәк ярсый, йөрәк ярсый,
Йөрәк тибә гел җилкенеп.

Җыр башласам, күңелдә гел моң тибрәнә,
Тирә-якта былбыл сайрый, гөл тибрәнә.
Дөньяда мин иң бәхетле кеше кебек,
Йөрәк ярсый, йөрәк ярсый,
Йөрәк тибә гел җилкенеп.
```
