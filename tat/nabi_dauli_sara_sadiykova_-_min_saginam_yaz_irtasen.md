# Оригинал

### Нәби Дәүли / Сара Садыйкова - Мин сагынам яз иртәсен

```
Мин сагынам яз иртәсен,
Аккан чакта ташкын сулар,
Йөрәгемдә чаң кагалар
Язгы уйлар, язгы моңнар.

Тагын күпме алда гомер,
Язмыш әле нәрсә язар?
Җирдә күбрәк торган саен
Азрак кала мина язлар.

Йолдызларга сузылмадым —
Нигә йолдыз, җирем булгач,
Миңа җитә бер ярты ай,
Бер талда бер сандугач,

Мин шуңа да риза идем,
Ул көннәрне кем сагынмас?
Миңа бер җыр бүләк итеп,
Очып китте ул сандугач.

Тагын күпме алда гомер,
Язмыш әле нәрсә язар?
Җирдә күбрәк торган саен
Азрак кала миңа язлар.
```
