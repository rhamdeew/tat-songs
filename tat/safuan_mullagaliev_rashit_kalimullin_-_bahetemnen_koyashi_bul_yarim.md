# Оригинал

### Сафуан Муллагалиев / Рәшит Кәлимуллин - Бәхетемнең кояшы бул, ярым

```
Кешелеккә яз китергән таңның
Яктысында синең йөзне күреп,
Актарылып Идел ташыган күк,
Мәхәббәтем алды тулы ирек…

Яз китергән таңда, карлы тауда.
Упкыннарны үтеп, сиңа бардым.
Онытмыйча авыр юллар барын,
Бәхетемнең кояшы бул, ярым.

Каршылыклар тулы тирә-ягым,
Барыбер сине генә сөя җаным.
Ни көтсә дә көтсен киләчәктә
Беркемгә дә сине бирмим, ярым.

Яз китергән таңда, карлы тауда,
Упкыннарны үтеп, сиңа бардым.
Онытмыйча авыр юллар барын,
Бәхетемнең кояшы бул, ярым.
```
