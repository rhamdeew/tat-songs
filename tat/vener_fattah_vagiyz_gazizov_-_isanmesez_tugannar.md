# Оригинал

### Венер Фәттах / Вәгыйзь Газизов - Исәнмесез, туганнар

```
Җырлыйк әле, биик әле,
Бездә бүген тантана.
Барыбыз да исән-саубыз,
Күңел шуңа шатлана.

Исәнмесез, саулармысыз,
Кодачалар, кодалар.
Исәнмесез, безгә килгән
Кияү, килен, балалар.
Табыныбыз балкып тора,
Күңел ача туганнар.
Без биегән чакта, әйдә,
Күтәрелсен тузаннар.
```
