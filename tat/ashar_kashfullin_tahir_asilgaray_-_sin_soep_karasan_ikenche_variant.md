# Оригинал

### Әсхәр Кәшфуллин / Таһир Асылгәрәй - Син сөеп карасаң (икенче вариант)

```
Май тугач сайрый бер сандугач,
Моңлана күңеле тулгач.
Мин көтәм моң арасында,
Син дә кил күңел язына.
Син кил күңел язына,
Көтәм моң арасында.

Сайрый да сандугач,
Кояшлы май тугач,
Сандугач — мәхәббәт җырчысы.
Сандугач сайрамый,
Болытлар таралмый,
Җаныңны сагышлар борчыса.
Сандугач тирәктә
Моң сибә йөрәккә.
Кил, бәгърем, син дә кил тыңларга.
Тыңлыйк без бергәләп,
Дөньялар түгәрәк,
Син барда янымда, син барда.

Яз якын, сайрый бер сандугач
Тавышы күңел эчендә.
Моңнары йолдыздай сирелә
Таңнары сизелми килә.
Син кил күңел язына,
Көтәм моң арасында.

Йолдызлар янганда,
Күз ачып йомгандай
Ул безнең гомерләр арасы.
Сандугач сайраса,
Син сөеп карасаң,
Төзәлә күңелем ярасы.
Сандугач тирәктә
Моң сибә йөрәккә.
Кил, бәгърем, син дә кил тыңларга.
Тыңлыйк без бергәләп,
Дөньялар түгәрәк,
Син барда янымда, син барда.
```
