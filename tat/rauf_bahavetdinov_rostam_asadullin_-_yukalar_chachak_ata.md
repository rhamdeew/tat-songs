# Оригинал

### Рәүф Баһаветдинов / Рөстәм Әсәдуллин - Юкәләр чәчәк ата

```
Болын чәчкәләрен җыйдым,
Үзәннәре киң икән.
Сине көтәм, эх, сагындым,
Кайчан килерсең икән?

Ә юкәләр чәчәк ата,
Күңелемне юата.
Гөрләп узган яшьлегемдә
Йөрәгемә ут салма.

Синсез дөньяларны гизәм,
Гизәм кич, гизәм иртән.
Йөрәгемдә син бит чәчкә,
Килми калма, кил, иркәм.

Ә юкәләр чәчәк ата,
Күңелемне юата.
Гөрләп узган яшьлегемдә
Йөрәгемә ут салма.

Сагышларым хуш-ис чәчкә,
Чишмәләрдән су алам.
Бәгърем, бергә булсак сүз бир,
Синең утларда янам.

Ә юкәләр чәчәк ата,
Күңелемне юата.
Гөрләп узган яшьлегемдә
Йөрәгемә ут салма.
```
