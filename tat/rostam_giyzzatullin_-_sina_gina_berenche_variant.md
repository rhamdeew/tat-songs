# Оригинал

### Рөстәм Гыйззәтуллин - Сиңа гына (беренче вариант)

```
Юлларымда мин очраттым
Сине, сине генә,
Бәхетем син, пар канатым,
Сиңа, сиңа гына.
Юлларымда мин яраттым
Сине, сине генә;
Мәхәббәтем, күз карашым
Бары сиңа гына.

Сиңа гына бүләк итәм
Күнелемне сөюемне.
Эгәр бик озкка китсәм,
Алып китәрмен сине үзеңне.

Күзләремдә мин үзгәрдем
Синең белән генә,
Авырлыкны күтәрермен
Синең белән генә.
Мәхәббәтем, җылы йөзем
Сиңа, сиңа гына.
Җырларымны бүләк итәм
Сиңа, сиңа гына.

Сиңа гына бүләк итәм
Күнелемне сөюемне.
Эгәр бик озкка китсәм,
Алып китәрмен cине үзеңне.
```
