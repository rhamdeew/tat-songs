# Оригинал

### Рамазан Шагалиев / Рафаил Бакиров - Арыдым сине көтеп

```
Арыдым сине көтеп,
Көтер йомгагын сүтеп,
Сүндерми учагыбызны,
Керфекләремне өтеп.

Арыдым мин сине,
Мин сине көтеп арыдым!

Арыдым мин сине көтеп,
Өмет кылларын сиртеп.
Өмет кылларым өзелсә,
Ялгарбыз ничек итеп?

Арыдым мин сине,
Мин сине көтеп арыдым!

Арыдым сине көтеп,
Ялгыз юллардан үтеп.
Буранда адашып калсам,
Табарсың ничек итеп?

Арыдым мин сине,
Мин сине көтеп арыдым!
```
