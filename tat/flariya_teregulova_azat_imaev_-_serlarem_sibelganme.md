# Оригинал

### Флария Терегулова / Азат Имаев - Серләрем сибелгәнме…

```
Таңда бүлештем мин
Сөю серләремне,
Талгын гына искән жил белән.
Җилләр сиңа
Ахры җиткергәндер,
Сагышларым кайтты жыр белән.

Сипкән ахры җилләр
Минем серләремне,
Сукмакларда сары яф(ы)рак.
Хыялымда һаман
Күрешүләр көтәм,
Сагышлар таралсын тиз(е)рәк.

Онтылыплар тыңлыйм
Һәрчак кошлар җырын,
Хайран калып алар тавышына.
Кушылгандыр ахры
Сиңа әйтер сүзләр,
Назлар булып кошлар тавышына.

Салмак кына атлап.
Көзләремә керәм,
Тирә-якта сары яф(ы)рак.
Сабыр гына көтәм,
Сагышка уралып,
Килүеңне синең мин, иркәм.
```
