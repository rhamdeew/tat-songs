# Оригинал

### Нурия Абдуллина - Исән йөрегез, балам!

```
Ыргый-сикерә ага сулар,
Алга җилә замана.
Җилеп барган аргамактай
Ашыгалар балалар.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.

Аңлый ана, алар да бит
Балалар үстерәләр.
Балалары бәхетенә
Алар да йөгерәләр.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.

Сезнең дә кочыкларыгыз
Канат ныгытыр әле.
Утырып уйлар уйлый-уйлый,
Сез дә көтерсез әле.

Бересе кайта, бересе китә,
Туктатырга юк чара.
Ана бәгере өзелеп кала,
Исән йөрегез, балам!.
```
