# Оригинал

### Марат Кәбиров / Илдар Низамов - Утларга салып киттең

```
Сүз дә әйтеп тормадың син,
Күз генә салып үттең.
Күз салмадың, йөрәгемә
Куз гына салып киттең.

Көннәренмең-төннәремнең
Тынлыгын алып киттең.
Утларыңда янам инде,
Утларга салып киттең…

Коткарыгыз, янам бит, дип
Кемнәргә дәшим хәзер.
Утларга салып киттең дә,
Утларда яшим хәзер.

Кемнәр уттан тартып алсын,
Кемнәр соң коткарсыннар?!
Янудан туктаса йөрәк,
Яшәүдән туктар сыман.
```
