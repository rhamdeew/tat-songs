# Оригинал

### Зәлифә Кашапова / Фирдүс Шәрәфуллин - Айсылу

```
Айсылуны очраттым мин
Зәңгәр чишмә янында,
Күрешергә сүз куештык
Аның белән тагын да.

Зәңгәр күзле, бөдрә чәчле,
Тулган ай кебек сылу.
Йөрәгемдә ут кабыздың,
Әй, сылуым, Айсылу.

Айсылуны ай кызына
Тиңләдем уйларымда.
Серләшеп туя алмадык
Без Базы буйларында.

Зәңгәр күзле, бөдрә чәчле,
Тулган ай кебек сылу.
Йөрәгемдә ут кабыздың,
Әй, сылуым, Айсылу.

Җиһанның бар гүзәллеген
Җыйгансың син, сылуым.
«Сөям» дигән сүзең көтәм,
Бергенәм, Айсылуым.

Зәңгәр күзле, бөдрә чәчле,
Тулган ай кебек сылу.
Йөрәгемдә ут кабыздың,
Әй, сылуым, Айсылу.
```
