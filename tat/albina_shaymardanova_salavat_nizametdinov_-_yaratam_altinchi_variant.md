# Оригинал

### Альбина Шәймарданова / Салават Низаметдинов - Яратам (алтынчы вариант)

```
Йөрәгемене тоялмыймын,
Мәхәббәтсез яши алмыймын.
Тик бер синнән көч аламын,
Бар дөнъяга кычкырамын:

Яратам, мин яратам,
Тик бер сиңа рәхмәтлемен.
Яратам, мин яратам,
Синең белән бәхәтлемен.

Гомер юлын куган чакта,
Һәм бик соңлап очрашсакта.
Тик бер сиңа рәхмәтлемен,
Синең белән бәхетлемен.

Яратам, мин яратам,
Тик бер сиңа рәхмәтлемен.
Яратам, мин яратам,
Синең белән бәхәтлемен.

Үз-үземне тыялмыймын,
Синсез бөркет торалмыймын.
Килче тагы бәгърем миңа,
Сөрән салдым бөтен илгә.

Яратам, мин яратам,
Тик бер сиңа рәхмәтлемен.
Яратам, мин яратам,
Синең белән бәхәтлемен.
```
