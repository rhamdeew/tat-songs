# Оригинал

### Наҗар Нәҗми / Нариман Сабитов - Агыйделдә ак томан (икенче вариант)

```
Агыйделдә ак томан,
Параход тавышлары.
Дымлы керфекләр шикелле,
Ярында камышлары.
Ярында камышлары,
Параход тавышлары.
Ярында камышлары,
Параход тавышлары.

Агыйделдә ак томан,
Ат кешни болыннарда.
Ат эзләп йөргән эзләрем
Калган шул урыннарда.
Калган шул урыннарда,
Чыкланган болыннарда.
Калган шул урыннарда,
Чыкланган болыннарда.

Агыйделдә ак томан,
Борылган Идел белән.
Нигә шул кадәр якынсың,
Әй, туган Иделгенәм?!
Әй, туган Иделгенәм!
Йөрәгем синең белән!
Әй, туган Иделгенәм!
Йөрәгем синең белән!
```
