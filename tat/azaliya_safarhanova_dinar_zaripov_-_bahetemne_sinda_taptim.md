# Оригинал

### Азалия Сәфәрханова / Динар Зарипов - Бәхетемне синдә таптым

```
Сөенәмен чын йөрәктән,
Очраштырды дип безне язмыш.
Тигезлектә үтсә иде
Безгә бирелгән бу тормыш.

Бәхет дигән читлекләргә
Икәү бергә без урнашыйк.
Бәхетемне синдә таптым,
Мәхаббәт илендә кайнашыйк.

Кадерләрен белеп яшик
Якты дөньяның һәр минутын.
Мәхаббәт ул олы бәхет,
Сүндермикче без сөю утын.

Бәхет дигән читлекләргә
Икәү бергә без урнашыйк.
Бәхетемне синдә таптым,
Мәхаббәт илендә кайнашыйк.

Бер-беребезне яратудан
Йөрәгебез дөрләп янсын.
"Мин синеке, син минеке" —
Йөрәкләргә язылып калсын.

Бәхет дигән читлекләргә
Икәү бергә без урнашыйк.
Син минеке, мин синеке, диеп,
Яратып гомер буе яшик.
Бәхет дигән читлекләргә
Икәү бергә без урнашыйк.
Бәхетемне синдә таптым,
Мәхаббәт илендә кайнашыйк.
Бәхетемне синдә таптым,
Мәхаббәт илендә кайнашыйк…
```
