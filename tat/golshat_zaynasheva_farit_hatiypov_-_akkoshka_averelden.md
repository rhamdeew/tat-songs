# Оригинал

### Гөлшат Зәйнәшева / Фәрит Хатыйпов - Аккошка әверелдең…

```
Гаҗәп бер төш күрдем бүген,
Хәйран калып уяндым.
Тынгы тапмый йөрәгемә,
Көне буе уйландым.

Икәү җырлап бара, идек
Бергә Идел ярыннан,
Аккошка әверелдең дө,
Очып киттең янымнан.

Айрылу көтәдер безне,
Күңелемә шик килде.
Төшләргә ышанмыйм да бит,
Нәкъ өндәге шикелле.

Икәү җырлап бара, идек
Бергә Идел ярыннан,
Аккошка әверелдең дө,
Очып киттең янымнан.

Син янымда, бу бәхеткә
Ышанмыйм әле һаман.
Төштә түгел, өндә шулай
Китеп барырсың сыман.

Икәү җырлап бара идек,
Бергә Идел ярыннан,
Аккошка әверелдең дө,
Очып киттең янымнан!..
```
