# Оригинал

### Рәүфә Нурмөхәммәтова - Бир түзем

```
Туктаек әле бер мизгелгә,
Барыр юлларыбыз ерак.
Бу дөньяга зарланмыйк,
Мәңгелектә нишләрбез азак?

Таңнар аткач, ачып күзем
Мең шөкерләр әйтәм үзем.
Сүнмәсен күңелләрдә өмет,
Эй, Ходаем, бир түзем!

Җирдә безнең өчен сынаулар,
Булмас ул кире кайтулар.
Калсын бездән изгелек
Йолдызларны җәлеп итәрлек!

Таңнар аткач, ачып күзем
Мең шөкерләр әйтәм үзем.
Сүнмәсен күңелләрдә өмет,
Эй, Ходаем, бир түзем!
Таңнар аткач, ачып күзем
Мең шөкерләр әйтәм үзем.
Сүнмәсен күңелләрдә өмет,
Эй, Ходаем, бир түзем!
```
