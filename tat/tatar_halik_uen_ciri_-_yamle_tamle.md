# Оригинал

### Татар халык уен җыры - Ямьле, тәмле

```
Ал чәчәкле илебез,
Гөл чәчәкле илебез;
Җир шарында кояш булып
Тора безнен илебез.

Ямьле Агыйдел буйлары,
Тәмле Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары.

Урак урган җирләрем,
Печән чапкан җирләрем;
Ак чәчәктәй күкрәп үсә
Минем туган илләрем.

Ямьле Агыйдел буйлары,
Тәмле Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары.

Тәрәзәмнең төпләрендә
Аллы-гөлле гөлчәчәк;
Совет иле — бәхет иле,
Гөлчәчәктәй үсәчәк.

Ямьле Агыйдел буйлары,
Тәмле Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары,
Ямьле, тәмле шул
Агыйдел сулары.
```
