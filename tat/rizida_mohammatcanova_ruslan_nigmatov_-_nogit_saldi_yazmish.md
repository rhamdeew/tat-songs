# Оригинал

### Ризида Мөхәммәтҗанова / Руслан Нигматов - Ногыт салды язмыш

```
Әрнү булмас, хозур дөнья
Яшәр өчен биргән түзем.
Бәхет юрап күзне буып
Ногыт салды язмыш үзе.

Кушымта:
Артык янмый, сүнми генә
Учак яна дөрләп көн дә.
Гәүһәр туйга җитми булмас,
Ашыгу юк әле төнгә.

Рәхмәт язмыш, шушы кеше —
Минем бәхет! Кайдан белдең?..
Аяз күккә карый күзләр,
Җылы учак ягып бирдең.

Кушымта.

Безнең бәхет ләйсән төсле
Ява безгә, күктән көлеп.
Илле бу ел, язмыш үзе
Ак, бәхеткә куйды төреп.

Кушымта
```
