# Оригинал

### Флера Гыйззәтуллина / Таһир Абдршин - Кайтам илкәемә

```
Барлый ил чакрымнар белән
Үлчәп юл араларын,
Кайчандыр читләргә китеп
Адашкан балаларын.

Мин дә сибелгән татарның
Бер газиз баласы ла;
Иделем аша юл сузам
— Чакрымнар арасыннан.

Сагыну-кайту юлының
Шатлыгын-сөенечен,
Акчарлак очыра Идел
Багышлап минем өчен.

Мин дә сибелгән татарның
Бер газиз баласы ла;
Иделем аша юл сузам
Чакрымнар арасыннан.

Идел киң, акчарлак җанның
Уйңаклап очарына.
Кайтам мин, кайтам, илкәем,
Сыйдырсаң кочагыңа.

Мин дә сибелгән татарның
Бер газиз баласы ла;
Иделем аша юл сузам
Чакрымнар арасыннан.
```
