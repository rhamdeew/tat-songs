# Оригинал

### Резеда Вәлиева / Илназ Баһ - Нигә икән, нигә шундый без?

```
Ни җитми соң безгә дибез һаман,
Чын дус җитми безгә кирәктә.
Җаның өшеп күңелең сүрелгәндә,
Бер җылы сүз җитми йөрәккә.

Эх, туганнар, ник соң шулкадәрле
Бер-беребезнең кадерен белмибез?
Күз алдында моңлы җаннар сула
Без аларны гүя күрмибез.

Кеше киткәч мактау яудырабыз,
Алтыннарга аны тиңлибез.
Кемгә кирәк ул соңлаган сүзләр?
Шуны да бит аңлый белмибез.
```
