# Оригинал

### Шәмсия Җиһангирова / Илфат Дәүләтшин - Каерма канатың (Пар канатлар, Ташлама)

```
Пар чәчәкләр гөлдә,
Пар аккошлар күлдә,
Пар сандугач булса — җыр матур.
Ярыңнан аерылып.
Канатың каерылып,
Калсаң сине кемнәр юатыр.

Җаныңны канатып,
Каерма канатың,
Ташлама ярыңны, ташлама.
Синекеннән артык
Түгел кеше яры,
Үз канатың бирмә башкага.

Елгаларның яры
Һаман парлы-парлы,
Көн — кояшы, төннең ае бар.
Парлыларны күреп,
Үзәкләрен өзеп,
Гомер итә аерым-аерылар.

Җаныңны канатып,
Каерма канатың,
Ташлама ярыңны, ташлама.
Синекеннән артык
Түгел кеше яры,
Үз канатың бирмә башкага.

Бул син гел янында
Кайгылы чагында,
Шатлыгын да тигез бүлешеп.
Күзләренә карап,
Бер-береңне аңлап,
Гомерләр ит сөеп-сөешеп.

Җаныңны канатып,
Каерма канатың,
Ташлама ярыңны, ташлама.
Синекеннән артык
Түгел кеше яры,
Үз канатың бирмә башкага.
```
