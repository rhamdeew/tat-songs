# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Кузгала поезд

```
Кузгала, поезд кузгала
Китәмен еракларга
Аерылулар җинел булмый,
Барыбер елап калма
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Еламый түзеп кара.
Яраларың кузгалганда,
Еламый түзеп кара.

Тәгәрмәчләр сулкылдыйлар
Җаннарны өзеп-өзеп.
Җырларымнан туктап калсам,
Мин дә елармын кебек.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырламый түзеп кара.
Яраларың кузгалганда,
Җырламый түзеп кара.

Аерылулар беркайчан да,
Булмыйдыр җинел генә.
Мин үземнең хәлләремне
Беләм тик үзем генә.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырламый түзеп кара.
Яраларың кузгалганда,
Җырламый түзеп кара.

Кузгала, йөгерә, кузгала
Язмышның поездлары
Җаннарымны юатырга
Җырларым калыр бары.
Кузгала, поезд кузгала
Кузгала җанда яра.
Яраларың кузгалганда,
Җырларым сездә кала.
Яраларың кузгалганда,
Җырларым сездә кала.
```
