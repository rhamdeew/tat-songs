# Оригинал

### Дилә Булгакова / Риф Арслан - Еллар сүтеп килдем

```
Еллар сүтеп килдең син нигә?
Аккошым бул дидең ярым.
Күңел түрләренә үтәр идем
Җанкайларым тапса үз парын.

Аккошым дип исем кушасың син
Еракларда ишетелә моң барын.
Адашып ла ялгыз калдың микән
Юлларыңда минем үз парым.

Хыялыма канат каурый куеп
Бәхетемә юллар юрасаң.
Ак дөнъяңда аккошкаең булам
Наз — гөлкәем генә булалмам.

Аккошым дип исем кушасың син
Еракларда ишетелә моң барын.
Адашып ла ялгыз калдың микән
Юлларыңда минем үз парым.
```
