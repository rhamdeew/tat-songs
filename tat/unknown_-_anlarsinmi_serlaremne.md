# Оригинал

### Фәйрүзә Мөслимова / Зиннур Гыйбадуллин - Аңларсыңмы серләремне?

```
Аерылганда икәү бергә
Аланнарны урадык.
Кавышыр көннәребезне
Кәккүкләрдән сорадык.

Аңларсыңмы серләремне,
Сөйләр идем, ачылып?
Таң юллары икебезне
Тора кебек чакырып.

Еллар үтте, без утырткан
Пар каен шаулап үсте.
Аланнарга ялгыз чыгам —
Сине күрермен төсле.

Аңларсыңмы серләремне,
Сөйләр идем, ачылып?
Таң юллары икебезне
Тора кебек чакырып.

Гел елмаеп яшим җирдә —
Син бар кебек янымда.
Ак фәрештәм булып калдың
Канатлы хыялымда.

Аңларсыңмы серләремне,
Сөйләр идем, ачылып?
Таң юллары икебезне
Тора кебек чакырып.
```
