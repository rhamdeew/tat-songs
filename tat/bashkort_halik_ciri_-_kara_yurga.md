# Оригинал

### Башкорт халык җыры - Кара юрга

```
Йомры матур тоягыңны
Дагаларбыз, юргам.
Кырач матур ялларыңны
Укаларбыз, юргам.
Өстеңдәге сылыуыңны
Төшерәле, юргам
Һайт, кара юргам,
Тайт, кара юргам.

Башыңдагы йөгәнеңне
Көмешләрбез, юргам,
Кара-кырпу үңдегеңне
Нагышларбыз, юргам.
Өстеңдәге сылыуыңны
Төшерәле, юргам.
Һайт, кара юргам,
Тайт, кара юргам.
```
