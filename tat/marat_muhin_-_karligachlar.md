# Оригинал

### Марат Мухин - Карлыгачлар

```
Офыкка җәя канатын
Алсу таңның балкышы.
Мин сине уйлап уянам,
Күңелемнең аккошы.

Карлыгачлар оя ясый,
Шуны күрәсем килә.
Үз оямны синең белән
Бергә үрәсем килә.

Таң нуры безнең йөзләрне
Балкытып торсын иде.
Минем сөю йөрәгеңне
Җилкетеп куйсын иде.

Карлыгачлар оя ясый,
Шуны күрәсем килә.
Үз оямны синең белән
Бергә үрәсем килә.

Карлыгачым, сәлам бирче
Минем сөйгән ярыма.
Ул да миңа җан атадыр,
Күңеле аккошына.

Карлыгачлар оя ясый,
Шуны күрәсем килә.
Үз оямны синең белән
Бергә үрәсем килә.
```
