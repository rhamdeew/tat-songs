# Оригинал

### Илдар Юзеев / Фәнир Галимов - Аерылмас җыр

```
Сүзен, көен әйбәт белми торып,
Матур җыр башлавы бик кыен.
Җанга якын җырны бик тиз таптык:
Син сүзләрен белдең, мин — көен.

Бер-беребезне ихлас сөйгәнгәдер,
Күңел җыры бик тиз табылган.
Ул җыр хәзер моңсу сәлам булып,
Ишетелә яшьлек ягыннан.

Бергә булалмадык, ялгыз җырлап,
Гомерләрне күпме яндырдык.
Өзелеп сөйгәннәргә ядкар итеп,
Аерылмас бер җыр калдырдык.
```
