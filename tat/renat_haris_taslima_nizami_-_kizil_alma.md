# Оригинал

### Ренат Харис / Тәслимә Низами - Кызыл алма

```
«Яратмыйм…» дигән сүзләрнең
Ялкынын йота-йота
Елаганым бар, егетләр,
Йөрәкне тота-тота,

Аһ, ул елауның ачысы!
Ук очында уттыр ул.
Аның белән тиңләшерлек
Башка утлар юктыр ул…

«Яратам!..» дигән сүздән дә
Булса да сыктаганым,
Сөттәй пакь кире кагулар,
Эх, килсен иде тагын.

Кадалсын иде йөрәккә,
Ялкыны чыксын йөзгә…
Һәр алманың, бер кызарып
Карыйсы килә көздә…
```
