# Оригинал

### Илдус Гыйләҗев / Илгиз Закиров - Уразай авылы яшьләре җыры

```
Янәшәдә шаулап ага
Стәрле боргаланып.
Чатыр тау да сиңа карый,
Сокланып, горурланып.

Уразаем — туган авылым,
Гөрләп үсәсең.
Дәртләндереп күңелләрне
Җырга күчәсең.

Армый эшләп арттырабыз
Кырларның уңдырышын.
Хыялларга канат итеп
Яшьлегем омтылышын.
Киләчәгем юлы булып
Күренә урамнарың.
Чатыр таудан югарырак
Күтәрелсен даннарың.

Уразаем — туган авылым,
Гөрләп үсәсең.
Дәртләндереп күңелләрне
Җырга күчәсең.
```
