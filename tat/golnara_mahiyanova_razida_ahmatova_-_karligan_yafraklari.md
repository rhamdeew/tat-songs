# Оригинал

### Гөлнара Махиянова / Рәзидә Әхмәтова - Карлыган яфраклары

```
Карлыган яфраклары
Хуш исләр таратканга.
Чәем тәмле, тормыш ямьле —
Син мине яратканга!

Карлыган яфраклары
Хуш исләр таратканга.
Чәем тәмле, тормыш ямьле —
Мин сине яратканга!

Карлыганның яфраклары
Чәйгә тансык — җәй тәмен
Болыннарда бергә җыйдык
Хуш исле ялан ямен.

Карлыган яфраклары
Хуш исләр таратканга.
Чәем тәмле, тормыш ямьле —
Мин (син) сине (мине) яратканга!

Җәйләр үтәр, көзләр җитәр.
Тагын-тагын парлашып,
Алла бирса, гомер булса,
Чәй эчәрбез, җанашым!

Карлыган яфраклары
Хуш исләр таратканга
Чәем тәмле, тормыш ямьле —
Мин сине яратканга!
```
