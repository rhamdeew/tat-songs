# Оригинал

### Башкорт халык җыры - Тал бөгелә (беренче вариант)

```
Бер карадым күзләремә,
Икенче — йөзләреңә.
Бер караудан мәхәббәтле
Күрендең күзләремә.

Тал бөгелә, тал бөгелә,
Әллә тамырсыз микән?
Сандугачлар булып килсәм,
Мине танырсың микән.

Аклы ситса күлмәгемнең
Җиңе тар беләгемә.
Кошлар килеп, сайраса да,
Ял була йөрәгемә.
```
