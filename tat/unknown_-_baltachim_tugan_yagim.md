# Оригинал

### Самат Шакир / Сара Садыйкова - Балтачым — туган ягым

```
Балтач болынында уйнап үстем,
Тәмле чишмә суларын эчтем.
Кая гына барып йөрсәм дә мин,
Туган ягым исемә төште.

Балтачым, Балтачым!
Кадерле туган ягым,
Синдәй ямьле, суы тәмле
Нинди генә җирләр бар тагын?

Өзелеп сөйгән ярым синдә генә,
Синдә минем зәңгәр кичләрем.
Синдә эшләп үстем, канат җәйдем,
Синдә туды тәүге хисләрем.

Балтачым, Балтачым!
Кадерле туган ягым,
Синдәй ямьле, суы тәмле
Нинди генә җирләр бар тагын?

Киләчәктә матур көнгә карап
Балтачлылар килә сафларда.
Хезмәт сөйгән батыр халкы белән
Даннар ала мине якларга.

Балтачым, Балтачым,
Кадерле туган ягым,
Синдәй ямьле, суы тәмле
Нинди генә җирләр бар тагын?
```
