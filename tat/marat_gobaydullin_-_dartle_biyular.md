# Оригинал

### Марат Гобәйдуллин - Дәртле биюләр

```
Дәртле биюгә чакыра
Тальян гармун моңнары.
Егетләре гармун уйный,
Кушылып җырлый кызлары.

Әй, күңелле биюләр
Яшьләр бергә бииләр.
Күпме генә биесәк тә,
Бер дә ардык, димиләр.

Урталарга син чыккансың,
Өздереп биеп җибәр.
Егетләре нинди дәртле,
Кызлары бигрәк чибәр.

Әй, күңелле биюләр
Яшьләр бергә бииләр.
Күпме генә биесәк тә,
Бер дә ардык, димиләр.

Дәртле бию көйләренә,
Күтәрелсен күңелләр.
Шулай бергә тату гына
Үтсен безнең гомерләр.

Әй, күңелле биюләр
Яшьләр бергә бииләр.
Күпме генә биесәк тә,
Бер дә ардык, димиләр.
```
