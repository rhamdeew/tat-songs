# Оригинал

### Әхсән Баян / Фасил Әхмәтов - Тәрәзә юа кызлар

```
Пыялага зын-зын итеп
Яшь нурлар сибеләләр.
Керсен өйгә язгы хәбәр,
Ачылсын тәрәзәләр. Их!
Кайтканда каңгылдап казлар,
Акканда соңгы бозлар,
Тәрәзә юа кызлар әй, кызлар!

Кушылып ага сулар белән
Кайгылар, авыр сүзләр.
Юыла шик, көнләшүләр,
Яктыра, балкый йөзләр. Их!
Җырларга күчкән тал-билләр,
Безне назлаган куллар,
Кузгата гаҗәп уйлар, әй, кызлар!

Кызлар тәрәзә юганда
Сокланмый кемнәр чыдар.
Чибәрләнә бара кызлар,
Яктыра бара еллар.
Кочып алыр безнең куллар —
Сикерегез биектән,
Кавышыр чаклар җиткән, әй, кызлар!
```
