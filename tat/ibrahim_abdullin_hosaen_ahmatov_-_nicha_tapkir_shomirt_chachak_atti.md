# Оригинал

### Ибраһим Абдуллин / Хөсәен Әхмәтов - Ничә тапкыр шомырт чәчәк атты

```
Ничә тапкыр шомырт чәчәк атты,
Ничә тапкыр ташты ташкыннар,
Шомыртлар да инде картайдылар,
Тик картаймый икән ярсулар.

Ярсуларым ярга кайтмас инде,
Яр булмагач алар сыерлык —
Яр булмагач ярсып сөерлек,
Гомер буе янып-көерлек.

Ничә тапкыр шомырт чәчәк атты,
Ничә тапкыр килде торналар.
Торналар да китә озак тормый,
Ә сагышлар һаман торалар,
Ә сагышлар һаман торалар.
```
