# Оригинал

### Диләрә Рәхимова / Илдар Шәрипов - Әтием юкәләре

```
Өй артында шаулап үсә
Әтинең юкәләре.
Әкрен генә тирбәләләр
Табигать күркәмнәре.

Башын аска игән юкә,
Ул юксына, моңая.
Гүя миннән сорый сыман:
Нигә озак, ул кая?
Бакчаларга кергән саен
Юкәләргә карыймын.
Йөрәккәем өзгәләнмә,
Әтиемне уйлыймын.

Кайтам туган нигеземә,
Сыенам юкәләргә.
Тик әти юк яныбызда
Яратып иркәләргә.

Башын аска игән юкә,
Ул юксына, моңая.
Гүя миннән сорый сыман:
Нигә озак, ул кая?
Бакчаларга кергән саен
Юкәләргә карыймын.
Йөрәккәем өзгәләнмә,
Әтиемне уйлыймын.
Йөрәккәем өзгәләнмә,
Әтиемне уйлыймын…
```
