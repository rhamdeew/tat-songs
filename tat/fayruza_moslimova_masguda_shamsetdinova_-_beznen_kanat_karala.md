# Оригинал

### Фәйрүзә Мөслимова / Мәсгудә Шәмсетдинова - Безнең канат карала

```
Каз канаты, каз канаты
Кагына да агара,
Нигə соң без, нигə соң без
Охшамаган казларга?

Болгана агымсулар да,
Лəм утыра ярларга,
Чумсалара да, йөзсəлəр дə,
Кер дə кунмый казларга.

Без коенган сулар көмеш,
Чүп тə кунмый аларга,
Тик алай да безнең каурый,
Безнең канат карала.

Зəңгəр күлдə дулкын уйный,
Агымсулар болгана,
Казлар кагынганга карап
Күңеллəрем моңлана.
```
