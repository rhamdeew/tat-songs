# Оригинал

### Флария Терегулова / Фирзәр Мортазин - Бәхетемне эзлим

```
Гомерем буе көткән
Бәхет гөлен таптым,
Җылы җәйләр үткән, алда көз.
Гомерем көзе җиткән,
Инде нишлим икән,
Киңәш бирче, әйтче берәр сүз!..

Күзләремдә — көзең,
Хисләремдә — үзең,
Йөрәк тулы сагыш, дулкыннар.
Кулларымны сузып,
Сиңа атлар идем,
Ярамый бит, алда упкыннар.

Тоям хуш исләрен
Куе чәчләреңнең,
Йөрәгемә әйтәм, әйдә түз.
Гомерем көзе җиткән.
Инде нишлим икән,
Киңәш бирче, әйтче берәр сүз!..
```
