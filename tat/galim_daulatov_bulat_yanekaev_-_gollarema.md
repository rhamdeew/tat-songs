# Оригинал

### Галим Дәүләтов / Булат Йәнекәев - Гөлләремә

```
Онытсаң да мине, гөлләремә
Су сибәргә берүк онытма.
Күзләреннән нурлар (кояш) тамган (янган) чакта,
Яшь тамызма, иркәм боекма.

Гөлләремә бакчы (бактым), гөл күзеннән
Миңа охшаган моңым чагылыр.
Мин сыйпаган йомшак чәчләреңә
Кулым булып җилләр кагылыр.

Онытсаң да мине, гөлләремә
Су сибәргә берүк онытма.
Күзләреннән нурлар (кояш) тамган (янган) чакта,
Яшь тамызма, иркәм боекма.
```
