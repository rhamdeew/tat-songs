# Оригинал

### Марат Кәримов / Ризван Хәкимов - Китәсе килми

```
Туган якларга кайтыр чакларда
Хәтта бер минут көтәсе килми.
Яшьлек тавышыдай авылым каршылый,
Мине һич тә ят итәсе килми.

Монда дусларым, монда ишләрем,
Дусларсыз юллар үтәсе килми.
Әткәй кабере… Әнкәй кабере …
Яшьләрем тама, сөртәсе килми.

Сынаусыз-нисез, сагышсыз-моңсыз
Матур көннәрнең иртәсе килми.
Туган төягем, синдә йөрәгем,
Синнән аерылып китәсе килми.
Китәсе килми …
```
