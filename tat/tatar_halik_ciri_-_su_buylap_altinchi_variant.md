# Оригинал

### Татар халык җыры - Су буйлап (алтынчы вариант)

```
Идел бит ул, тирән бит ул,
Тирән бит ул, киң бит ул (ла).
Караңгы төн, томанлы көн —
Без аерылган көн бит ул.

Идел ага, Идел ага,
Идел диңгезгә китә (шул).
Аккан сулар, искән җилләр
Йөрәгемне җилкетә.

Идел буйларына барсам,
Әйтер идем Иделгә (шул).
Минем кайгыларымны да
Илтеп сал дип, диңгезгә.

Су буеннан сулар алып,
Сибәмен гөлләремә (шул);
Мәңге шиңмәс гөлләр булып
Калдың күңелләремдә.
```
