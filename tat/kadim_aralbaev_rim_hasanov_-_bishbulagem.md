# Оригинал

### Кәдим Аралбаев / Рим Хәсәнов - Бишбүләгем

```
Бишбүләгем — моң төягем,
Бишегем, газиз җирем,
Синдә минем тамырларым,
Кырыңа тамган тирем.

Бишбүләгем — хис төягем,
Мәхәббәт ҡаеннарым.
Чишмәләрең суын эчтем,
Сөюдән айнымадым.

Бишбүләгем — Дим буйлары,
Моңло сандугач иле.
Тальяныңның көмеш чыңы
Мәңге тынмасын иде.

Бишбүләгем — бердән-берем,
Син дип тибә йөрәгем.
Биш яшәүдәй бер гомерем
Булсын сиңа бүләгем …
```
