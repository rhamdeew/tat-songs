# Оригинал

### Шәүкәт Галиев / Илдус Якупов - Бииләр итек, читекләр

```
Гармун уйныйм, гармун уйныйм,
Һаман гармун уйныйм мин,
Остарып җиткәч, барсын да
Биетергә уйлыйм мин.

Учларыңны шакылдатып,
Чәчләреңне сыпыр да
Тыпыр-тыпыр, тыпыр-тыпыр,
Тыпыр-тыпыр тыпырда.

Булыр әле абыйларны
Биеткән көннәрем дә,
Әлегә бармак биетәм
Мин гармун телләрендә.

Тыпыр-тыпыр тыпырдап,
Тыкыр-тыкыр тыкырдап,
Кинәт идән уртасына
Чыкты апам башмагы,
Абый ботинкасы белән
Кинәт бии башлады.

Бии әти итеге дә,
Әнием галошы да,
Бии әби читеге дә —
Бииләр барысы да!

Читтә тора яңа туфли,
Ник уртага кермидер?
Кибеттән генә кайткан шул,
Әле бии белмидер…

Тыпыр-тыпыр тыпырдап,
Тыкыр-тыкыр тыкырдап,
Әй, бии итек, читекләр!
Әй, уйныйм мин, уйныйм мин,
Минем кебек шәп гармунчы
Юктыр диеп уйлыйм мин!
```
