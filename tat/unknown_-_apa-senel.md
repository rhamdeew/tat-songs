# Оригинал

### Ләйсән Гайнетдинова / Ильмира Нәгыймова - Апа-сеңел

```
Күңелләрем тулган чакта,
Күңел биреп тыңлыйсың.
Бар дөнья аңламаганда,
Мине һәрчак аңлыйсың.

Дустым гына түгел бит син,
Күпме наз сыйган күңел,
Сердәшем һәм якын кешем,
Без туган апа-сеңел.

Бик акыллы киңәшләрең,
Хәттә син биргән шелтә.
Синең сүзләр һәм теләкләр,
Гел яктылыкка илтә.

Дустым гына түгел бит син,
Күпме наз сыйган күңел,
Сердәшем һәм якын кешем,
Без туган апа-сеңел.

Якын дуслар бердәмлеге,
Күчерә ул тауларны.
Син янымда булган чакта,
Сизмәмен соң аларны.

Дустым гына түгел бит син,
Күпме наз сыйган күңел,
Сердәшем һәм якын кешем,
Без туган апа-сеңел.
```
