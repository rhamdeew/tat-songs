# Оригинал

### Гөл Чәчәк / Владимир Федоров - Кадерлеләрем

```
Бәхетлемен диеп уянам,
Һәр яңа көн җиргә туган саен.
Якыннарым булсын янәшәмдә,
Мәхрүм итмә шуннан, Ходаем!

Кызым, улым — бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.

Сагынып кайтамын ераклардан,
Кочагыма алып шатланам.
Бәхетемнең күреп җимешләрен,
Куңелем оча канатланып.

Кызым, улым — бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.

Сезнең белән бергә елмаям,
Сагышлардан табам дәваны.
Килгәнсез шушы якты дөньяга,
Яшьлегемнең булып дәвамы.

Кызым, улым — бердәнберләр,
Минем кадерлеләрем.
Сөю бакчасында үскән
Гөлләрнең бөреләре.
```
