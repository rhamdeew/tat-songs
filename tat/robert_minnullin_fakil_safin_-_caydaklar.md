# Оригинал

### Роберт Миңнуллин / Факил Сафин - Җайдаклар

```
Атларда чаба малайлар,
Атларда чаба малайлар,
Атлары да атлары!
Малайларның әле шулай
Малайларның әле шулай
Чаба торган чаклары.

Чаба алар, оча алар,
Чаба алар, оча алар,
Калмый хәтта эзләре.
Канатлы әллә атлары,
Канатлы әллә атлары,
Әллә инде үзләре.

Кирәк булса оча алар,
Кирәк булса оча алар
Йолдызларга, айларга …
Кемдер атларга соклана,
Кемдер атларга соклана,
Ә кемдер — малайларга!
```
