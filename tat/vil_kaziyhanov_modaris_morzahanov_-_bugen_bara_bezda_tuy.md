# Оригинал

### Вил Казыйханов / Мөдәрис Морзаханов - Бүген бара бездә туй

```
Җыелган туган-тумача,
Өстәлләрдә хөрмәт-сый.
Кавышалар яшь йөрәкләр,
Бүген бара бездә туй.

Җырлыйк күңелле җырларны,
Моң-зарларны читкә куй.
Яңа буын хөрмәтенә
Бүген бара бездә туй.

Килгәннәр якын дусларым,
Бокалларга шәраб кой.
Шатлык-куанычка булсын,
Бүген бара бездә туй.

Җырлыйк күңелле җырларны,
Моң-зарларны читкә куй.
Яңа буын хөрмәтенә
Бүген бара бездә туй.

Туй күлмәге кигәннәргә
Бер теләк бездә, бер уй:
Ак бәхетләр юлдаш булсын, —
Шуны тели сезгә туй.

Җырлыйк күңелле җырларны,
Моң-зарларны читкә куй.
Яңа буын хөрмәтенә
Бүген бара бездә туй.
```
