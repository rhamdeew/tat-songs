# Оригинал

### Резеда Вәлиева / Сәүбән Чуганаев - Кошлар кайткан чак иде

```
Кошлар кайткан чак иде,
Сулар аккан чак иде;
Яшьлегебез яздай гөрләп
Чәчәк аткан чак иде.

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомергә дә
Югалтмам, дигән идең.
Югалтмам, дигән идең.

Күңелдә вәгъдә сүзең,
Юлларда һаман күзем.
Язын каткан кошлар кебек
Кайтырсың микән үзең?

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомергә дә
Югалтмам, дигән идең.
Югалтмам, дигән идең.

Су өсләре ак күбек,
Мамык тараткан кебек.
Бер кемне ярата алмам
Сине яраткан кебек.

Каршыма килгән идең,
Елмаеп көлгән идең.
Инде таптым, гомергә дә
Югалтмам, дигән идең.
Югалтмам, дигән идең.
```
