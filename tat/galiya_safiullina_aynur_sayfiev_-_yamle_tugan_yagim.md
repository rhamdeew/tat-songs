# Оригинал

### Галия Сафиуллина / Айнур Сәйфиев - Ямьле туган ягым

```
Күңел җимешләрем пешеп
Өлгереп тулган чагы.
Туган якка кайтып килдем,
Хисләрем тулды тагын,
Хисләрем тулды тагын.

Туган якта гына дәва алам
Сагынудан сулган җыныма.
Туган якта гына илһам алам
Күңелемә тулган моңыма.

Чишмәбез суларын эчеп
Сафланды күңелләрем.
Туганнар белән күрешеп,
Атты бәхет гөлләрем,
Атты бәхет гөлләрем.

Туган якта гына дәва алам
Сагынудан сулган җыныма.
Туган якта гына илһам алам
Күңелемә тулган моңыма.

Туган якның каеннары
Калды миңа моңаеп.
Бәхет талында сайрады
Минем күңел тургаем,
Минем күңел тургаем.

Туган якта гына дәва алам
Сагынудан сулган җыныма.
Туган якта гына илһам алам
Күңелемә тулган моңыма.
```
