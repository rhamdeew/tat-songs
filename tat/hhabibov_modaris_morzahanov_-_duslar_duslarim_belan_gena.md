# Оригинал

### Х.Хәбибов / Мөдәрис Морзаханов - Дуслар (Дусларым белән генә)

```
Салкында калган баланнар
Туңмый тәмләнә генә.
Дуслар белән бергә булсак
Дөнья ямьләнә генә.

Салкын чишмәнең сулары
Тәмсезләнә җылынса.
Күңелләргә ямь өстәлә
Җан дусларың җыелса.

Кантларын ныгымыйча
Кош аермый баласын.
Дусларымның җылы сүзе
Баса йөрәк ярасын.

Аккошлар парсыз яшәми
Тик пары белән генә.
Кешедә ышаныч туа
Дуслары белән генә.
```
