# Оригинал

### Нәҗип Мадьяров / Илгиз Закиров - Җәйге кичтә

```
Җәйге кичтә чыгам болыннарга —
Икәү генә йөргән җирләргә.
Ул вакытлар калды, микән ни соң
Инде сагынып кына сөйләргә?!

Сагынып, зарыгып
Уйланам, моңланам.
Кавышу көннәрен
Күзаллап юанап.

Тәннәремә шифа була кебек
Чишмәләргә карап моңайсам.
Кавышуга булган өметләрем
Яңаралар сине уйласам.

Уйлыймын мин сине
Көнен дә, төнен дә.
Күңелем түрендә
Син генә, син генә.

Ялгыз гына кичке болыннарда
Йөрмәс идем эчем пошмаса.
Йөрәгемдә янган утларымны
Җырлап басыйм әле ичмасам.

Уйлыймын мин сине
Көнен дә, төнен дә.
Күңелем түрендә
Син генә, син генә.
```
