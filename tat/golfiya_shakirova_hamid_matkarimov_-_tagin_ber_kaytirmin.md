# Оригинал

### Гөлфия Шакирова / Хәмид Мәткәримов - Тагын бер кайтырмын

```
Беләм мин, беләм мин: син мине
Көтмисең кайтыр дип.
Тагын бер очрашу булыр дип
Уйламыйсың, өмет итмисең.

Ләкин мин тагын бер әйләнеп,
Тагын бер мин сиңа кайтырмын.
Әйтелми калган сүземне
Күзеңә бер карап әйтермен.

Бүген соң, соң инде бит безгә
Ниндидер нәтиҗә ясарга.
Үткәнне калдырып үткәндә,
Әйләндек бит хәзер ятларга.

Ләкин мин тагын бер әйләнеп,
Тагын бер мин сиңа кайтырмын.
Әйтелми калган сүземне
Күзеңә бер карап әйтермен.

Әйтелми калган шул бер сүзем
Тыңгылык бирми һич күңелгә.
Чыгамын, чыгамын юлга дип,
Талпынамын мин бүген дә.

Ләкин мин тагын бер әйләнеп,
Тагын бер мин сиңа кайтырмын.
Әйтелми калган сүземне
Күзеңә бер карап әйтермен.
```
