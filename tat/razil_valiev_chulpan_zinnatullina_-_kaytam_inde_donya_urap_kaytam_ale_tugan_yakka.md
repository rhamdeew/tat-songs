# Оригинал

### Разил Вәлиев / Чулпан Зиннәтуллина - Кайтам инде дөнья урап (Кайтам әле туган якка)

```
Кайтам инде дөнья урап,
Туган як болынына.
Болыннарда атлар чаба,
Мин анда колын гына.

Чит илләрдә, чит җирләрдә
Газиз гомерләр үткән.
Диңгез суы нигә тозлы? —
Әллә күз яше микән?

Кайтам инде дөнья урап,
Адашам урманымда.
Минем язмыш туры килде
Юлларның уравына.

Чит илләрдә, чит җирләрдә
Газиз гомерләр үткән.
Диңгез суы нигә тозлы? —
Әллә күз яше микән?

Кайтам инде дөнья урап,
Туган як кырларына.
Йөрәккәем түзәр микән
Тургайлар җырлавына?!

Чит илләрдә, чит җирләрдә
Газиз гомерләр үткән.
Диңгез суы нигә тозлы? —
Әллә күз яше микән?

Кайтам инде дөнья урап,
Шәл япкан талларыма.
Әнкәй үзе, шәл ябынып,
Чыгар күк алларыма.

Чит илләрдә, чит җирләрдә
Газиз гомерләр үткән.
Диңгез суы нигә тозлы? —
Әллә күз яше микән?
```
