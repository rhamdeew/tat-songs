# Оригинал

### Сания Әхмәтҗанова / Оскар Усманов - Ана җыры (икенче вариант)

```
Бәхеткәйләр сагындырып кына
Шакый, диләр, өйнең ишеген.
Белгән догам укып эчтән генә,
Өй түремә элдем бишегең.

Әй балам, багалмам,
Бәхетеңне бирсен бер Аллам.

Кайгы-хәсрәт рөхсәт сорап тормый,
Үтә икән өйнең түренә.
Алларыма алып сине сөйсәм,
Ачы хәсрәтем дә сүрелә.

Әй балам, багалмам,
Бәхетеңне бирсен бер Аллам.

Тигез канат җылысына сыенып,
Насыйп булса иде яшәргә.
Соң сулышта, башкаеңнан сыйпап,
Риза-бәхил, диеп дәшәргә.

Әй балам, багалмам,
Тәүфыйгыңны бирсен бер Аллам.
```
