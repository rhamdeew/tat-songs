# Оригинал

### Әлфирә Бикмәтова / Ризван Хәкимов - Бик еш кайтам

```
Бик еш кайтам туган якларыма,
Кар-буранын хәтта сагынып.
Юл төтмәсә, хыялымда кайтам,
Кайтам болытларга сарылып,
Туган ягым сине сагынып.

Бик еш кайтам туган якларыма,
Әнкәм гомере шунда үткәнгә.
Әтиемнең кабере өсләренә
Яшел хәтфә үлән үскәнгә,
Әнкәм гомере шунда үткәнгә.

Бик еш кайтам туган якларыма,
Салкын чишмәләрен сагынып.
Юл төшмәсә, хыялымда кайтам,
Кайтам болытларга сарылып,
Уңган кешеләрен сагынып,
Туган ягым сине сагынып.
```
