# Оригинал

### Таңсылу Карамышева / Роза Сәхәүетдинова - Күз нурым (икенче вариант)

```
Син ураган юлны мин урыймын,
Эзләреңә түгәм күз нурын.
Син сулаган язны мин сулыймын,
Кайда соң син, үзең, күз нурым?

Син сөялгән талга мин сөяләм,
Бөресенә коям күз нурын.
Син сөелгән җилгә мин сөйлимен,
Кайда соң син, үзең, күз нурым?

Кемнәр генә аңлар хисләремне,
Тоныкланса якты күз нурым?
Җилләр генә сөя чәчләремнән,
Кайда соң син, үзең, күз нурым?
```
