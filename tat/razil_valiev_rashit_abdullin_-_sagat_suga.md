# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Сәгать суга

```
Әйдә, искә алыйк әле
Борынгы гадәтләрне…
Бер минутка тынып торыйк,
Онытып сәгатьләрне.

Без бит җирдә һәммәбез дә
Табигать сәнгатьләре…
Җир-анага саулык бирсен
Яңа ел сәгатьләре.

Иске елда калсын әле
Кайгының сәбәпләре.
Кавыштырсын гашыйкларны
Яңа ел сәгатьләре.

Ташып торсын туган илнең
Шатлык-сәгадәтләре…
Сәгать суга, сәгать суга, —
Яңа ел сәгатьләре.
```
