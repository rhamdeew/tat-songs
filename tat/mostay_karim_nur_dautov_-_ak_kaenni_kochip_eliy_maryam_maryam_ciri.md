# Оригинал

### Мостай Кәрим / Нур Даутов - Ак каенны кочып елый Мәръям (Мәръям җыры)

```
Ак каенны кочып елый Мәръям.
Күз яшьләре тама ак тузга.
Ниләр булды икән, ниләр генә
Ун җидесе тулган бу кызга?

Ак каенны кочып елый Мәръям.
Нинди газап. Нинди әрнүләр?
Әле генә аңа: "Сөям",— диде
Ун сигезе тулган Әнвәр.

Мына шуңа, бары шуңа гына
Күз яшьләре тузга тамдылар,
Кыз үзе дә каен кебек кенә,
Яшьләре дә шундый татлылар.
```
