# Оригинал

### Гөлфия Шакирова / Илнар Гыйльманшин - Матур яшәргә тырыш

```
Яшибез без булсын диеп,
Тырышып көн итәбез.
Туасы уртак таңнарны
Өмет белән көтәбез.

Бергә, бергә, бердәмлектә
Гел дә ямьле бу тормыш.
Бер бирелгән гомердә син
Матур яшәргә тырыш.

Дуслар булсын яныбызда,
Иң якыннар янәшә.
Канатлы булган хыяллар
Алга чак(ы)рып эндәшә.

Бергә, бергә, бердәмлектә
Гел дә ямьле бу тормыш.
Бер бирелгән гомердә син
Матур яшәргә тырыш.

Бергә булыйк, бердәм булыйк,
Тормышларны гөл итик.
Бәхет-шатлыклары белән
Бергә гомерләр көтик.

Бергә, бергә, бердәмлектә
Гел дә ямьле бу тормыш.
Бер бирелгән гомердә син
Матур яшәргә тырыш.
Бер бирелгән гомердә син
Матур яшәргә тырыш…
```
