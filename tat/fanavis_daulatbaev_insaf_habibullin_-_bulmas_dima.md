# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Булмас димә!

```
Авыр булсада чигенмә,
Алга бар син, туктама!
Тырышканга җае чыга,
Һич кенә дә уфтанма!

Булмас димә, юкта димә —
Күңелеңне төшермә!
Ходай үзе ярдәм итә
Күңелле тик кешегә.

Яшибез дип — яшә җирдә,
Теләгең төчен барда.
Яшә җирдә өметләнеп,
Барсы да диеп алда.

Булмас димә, юкта димә —
Күңелеңне төшермә!
Ходай үзе ярдәм итә
Күңелле тик кешегә.

Теләгеңә ирешерсең
Булса гына түземлек.
Мәрхәмәтле кеше итеп
Күрсәт, ирдә, үзеңне!

Булмас димә, юкта димә —
Күңелеңне төшермә!
Ходай үзе ярдәм итә
Күңелле тик кешегә.
```
