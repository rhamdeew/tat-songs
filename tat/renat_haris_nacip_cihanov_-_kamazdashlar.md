# Оригинал

### Ренат Харис / Нәҗип Җиһанов - Камаздашлар

```
Без җыелдык Камага,
Чаллы дигән калага.
Алып килде безне монда
Хыял яктырткан юллар,
Тотындык зур тезелешкә
Яшьләрчә җиңел кулдан,

Рус, татар, башкорт дуслар,
Украиннар, чувашлар —
Менә дигән юлдашлар,
Хәзер без камаздашлар,
Хәзер без камаздашлар.

Без эшлибез Камада,
Чаллы дигән калада.
Яшьлегебез мәйдан тота
Халык, заман каршында.
Ватаныбыз нәрсә кушса,
Без үтәрбез барысын да,

Рус, татар, башкорт дуслар,
Украиннар, чувашлар —
Менә дигән юлдашлар,
Хәзер без камаздашлар,
Хәзер без камаздашлар.

Без калабыз Камада,
Чаллы дигән калада.
Үзебез салган йортларда
Яшьнәп яшисе килә.
Соклансыннар безгә карап:
Нинди тату гаилә!

Рус, татар, башкорт дуслар,
Украиннар, чувашлар —
Менә дигән юлдашлар,
Хәзер без камаздашлар,
Хәзер без камаздашлар.
```
