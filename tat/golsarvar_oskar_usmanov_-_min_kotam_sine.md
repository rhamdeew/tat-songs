# Оригинал

### Гөлсәрвәр / Оскар Усманов - Мин көтәм сине

```
Син мине белмисең әлегә,
Мин инде күптән беләм.
Гашыйк итеп кердең күңелгә
Беренче күргән көннән,
Чыкмыйсың йөрәгемнән.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.

Хыялым сиздермим һич кемгә,
Күңелдә өмет кенә.
Төштә дә,өндә дә син генә
Юл күрсәт бәхетемә,
Хыялым йөрәгемдә.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.

Әгәр дә карасаң күземә,
Танырсың мине,беләм.
Китәрмен җирнең дә читенә,
Китәрмен синең белән,
Чакырсаң үзең белән.

Хыялым аша,
Уйларым аша
Белсәң иде дим,
Син үзең мине.
Җырларым аша
Йөрәгем дәшә
Килсәң иде дим,
Мин көтәм сине.
```
