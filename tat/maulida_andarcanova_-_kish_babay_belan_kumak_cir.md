# Оригинал

### Мәүлидә Әндәрҗанова - Кыш бабай белән күмәк җыр

```
Кыш бабай, кыш бабай
Кунакка килгән.
Килмәс иде, безнең белән
Биисе килгән.

Лә-лә-лә, лә-лә-лә,
Биисе килгән.
Лә-лә-лә, лә-лә-лә,
Биисе килгән.

Кыш бабай белән бергә
Кар кызы килгән.
Елтыр-елтыр, ап-ак бәсле
Күлмәген кигән.

Лә-лә-лә, лә-лә-лә,
Кар кызы килгән.
Лә-лә-лә, лә-лә-лә,
Күлмәген кигән.

Алар белән биергә дип,
Төлке дә чыкты.
Керпе белән тиен дус та
Парлашып басты.

Лә-лә-лә, лә-лә-лә,
Парлашып басты.
Лә-лә-лә, лә-лә-лә,
Парлашып басты.

Кыш бабайның каршысында
Аю тыпырдый.
Себеркесенә атланып
Убырлы зырлый.

Зыр-зыр-зыр, фыр-фыр-фыр,
Убырлы зырлый.
Зыр-зыр-зыр, фыр-фыр-фыр,
Убырлы зырлый.
```
