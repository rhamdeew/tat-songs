# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Син дип кенә

```
Мин бит җирдә син дип кенә яшим,
Синең нурың балкый күңелдә.
Син булганга шулай ләззәтле дә,
Күңелле дә яшәү бу җирдә.

Син дә җирдә мин дип яшисең бит,
Гомеремне минем бизәүчем.
Яратканга шулай көләчсең син,
Яратканга шулай гүзәлсең.

Синең янда шатлыкларым арта
Синең янда кими моң-зарым.
Бәхет өчен җирдә күп кирәкми,
Белү җитә җирдә син барын.

Бер-беребез өчен сөенеп яшик,
Тормыш сукмакларын узганда.
Мин шатланам җирдә син булганга,
Син сөенеп яшә мин барга.
```
