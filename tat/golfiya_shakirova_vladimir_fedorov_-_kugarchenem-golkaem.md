# Оригинал

### Гөлфия Шакирова / Владимир Федоров - Күгәрченем-гөлкәем

```
Атлыйсың белеп кенә,
Матур иттереп кенә.
Ничек шулай килештереп,
Сөйлисең көлеп кенә?

Күгәрченем-гөлкәем,
Гөлкәем, бәгырькәем,
Сине уйлап, көнен-төнен
Өзелә үзәккәем.

Карашларың ялкыннан,
Җаным сиңа талпынган.
Бер күрергә күңелкәем
Сине шундый сагынган.

Күгәрченем-гөлкәем,
Гөлкәем, бәгырькәем,
Сине уйлап, көнен-төнен
Өзелә үзәккәем.

Чибәрем син, гүзәлем,
Тирән дәрья күзләрең.
Балдан, шикәрдән дә татлы
Сөям дигән сүзләрең.

Күгәрченем-гөлкәем,
Гөлкәем, бәгырькәем,
Сине уйлап, көнен-төнен
Өзелә үзәккәем.

Назлы ярым син бит ул,
Бәхетләргә тиң бит ул.
Безгә дигән киләчәк юл,
Кара, ничек киң бит ул!

Күгәрченем-гөлкәем,
Гөлкәем, бәгырькәем,
Сине уйлап, көнен-төнен
Өзелә үзәккәем.
```
