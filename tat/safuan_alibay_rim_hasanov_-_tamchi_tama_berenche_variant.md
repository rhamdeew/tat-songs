# Оригинал

### Сафуан Әлибай / Рим Хәсәнов - Тамчы тама (беренче вариант)

```
Яз турында көй чыгарып,
Тагы тамды тамчылар.
Әллә шатлык яше булып,
Күзләремнән тамдылар.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.

Җырлый — җырлый тамчы тамса,
Җирдә гөлләр уяныр,
Сөймәсләр дә сөяр язда,
Уйланмаслар — уйланыр.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.

Күңелемә тамчы белән
Язым нурлар яудырсын,
Гашыйк итеп, язлар тагы,
Акыллардан яздырсын.

Чутылдашып тамчы тама,
Чиртеп көмеш кылларын,
Язны гына сагындырып
Әйләнәләр елларым.
```
