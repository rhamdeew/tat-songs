# Оригинал

### Рәис Риянов / Фәнзәл Котдысов - Аргамагым сиңа алып килде

```
Атла әле, ашык, аргамагым:
Җитәсем лә килә, күрәсем.
Башын игән, сүрән генә атлый,
Ул нәрсәдер сизә, күрәсең…

Менә тагын сиңа таба барам.
Нигә генә шунда йөрим соң!
Арабызда торган киртә аша
Ничек кенә сиңа керим соң?

Аргамагым сиңа алып килде.
Чыгып аңа, бәлки, дәшәрсең.
Мәхәббәтне тышка чыгармыйча,
Дөрләмичә күпме яшәрсең!
```
