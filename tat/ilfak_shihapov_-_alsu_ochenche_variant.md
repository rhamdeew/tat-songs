# Оригинал

### Илфак Шиһапов - Алсу (өченче вариант)

```
Ят шәһәрнең кичке урамнары,
Урамнарда тузан бураннары.
Шул буранда басып тора моңсу,
Сөембикә нәселеннән — матур Алсу.
Кемне көтә, ниләр өмет итә,
Юлдан матур машиналар үтә.
Онытылган бүген ана-ата,
Чибәр Алсу матур тәнен сата.

Онытылган бәбкә үләненнән
Яланаяк йөгереп йөргәне,
Дәү әнинең җылы кочагында
Матур-матур төшләр күргәне.
Онытылган тәүге саф мәхаббәт,
Күз яшьләре аша сөюләр,
Иреннәрең куркып дерелдәве,
"Мин яратам сине" диюләр.

Ялгыз түгел, туктый машиналар,
Төрле ирләр сөя Алсуны.
Акча кирәк матур яшәр өчен,
Басар өчен тәүге ярсуны.
Ерак авыл гади ата-ана
Эшли, түгеп ачы тирләрен.
Укый Алсу безнең, зур шәһәрдә укый…
Кочагында пычрак ирләрнең:
Өрек саткан сасы базар карты,
Гамьсез түрә, шаян егетләр —
Барсын сөя Алсу, тешен кысып сөя,
Урам тулы Алсу кебекләр.
Сөя алар матур машинада,
Фатирларда, пычрак өйләрдә.
Ирләр көлә, ямьсез итеп көлә,
Кызлар түгел алар, коллар ирләргә.

Укы балам, диеп ата-ана
Шәһәрләргә кызлар озата.
Күпме кызлар, тормыш фәнен укып,
Яшәр өчен матур тәнен сата.
Ят шәһәрне пычрак тузан баскан,
Күпме кызлар бүген урамда.
Авылларын, нәселләрен онытып,
Төренгәннәр соры томанга.

Онытылган бәбкә үләненнән
Яланаяк йөгереп йөргәне,
Дәү әнинең җылы кочагында
Тәмле-тәмле төшләр күргәне…
Өрек саткан сасы базар карты,
Гамьсез түрә, шаян егетләр —
Барсын сөя Алсу, тешен кысып сөя,
Урам тулы Алсу кебекләр.
Сөя алар матур машинада,
Фатирларда, пычрак өйләрдә.
Ирләр көлә, ямьсез итеп көлә,
Кызлар түгел алар, коллар ирләргә.
```
