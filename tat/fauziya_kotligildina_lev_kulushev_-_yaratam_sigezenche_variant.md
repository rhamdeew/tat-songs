# Оригинал

### Фәүзия Котлыгилдина / Лев Кулушев - Яратам (сигезенче вариант)

```
Яратам мин зәңгәр күктә
Мең-мең йолдыз янганын,
Гашыйҡ җаннар ай нурында
Бәхеткә койынганын.

Яратам мин, яратам мин,
Яратам мин тормышны,
Бары тик үземә генә
Насыйп булган язмышны!

Яратам елга буенда
Сандугачлар сайравын,
Язын оя корыр өчен
Үзенә пар сайлавын

Яратам мин, яратам мин,
Яратам мин тормышны,
Бары тик үземә генә
Насыйп булган язмышны!

Яратам мин чишмәләрнең
Челтер-челтер акканын,
Гөлләрдә күбәләкләрнең
Назлы канат какканын.

Яратам мин, яратам мин,
Яратам мин тормышны,
Бары тик үземә генә
Насыйп булган язмышны!
```
