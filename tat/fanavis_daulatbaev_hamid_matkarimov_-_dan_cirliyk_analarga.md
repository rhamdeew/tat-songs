# Оригинал

### Фәнәвис Дәүләтбаев / Хәмид Мәткәримов - Дан җырлыйк аналарга

```
Күңелләре — гөлбакчадай,
Наз тулы карашлары.
Һич күргән юк әле җирдә
Алар күк яратканны.

Бәхетлеләр, кадерлеләр —
Дөньяда алар барда.
Без аларның балалары —
Дан җырлыйк аналарга!

Онытмыйкчы,һәммәбез дә
Бурычлылар аларга.
Мәрхәмәтле,ихтирамлы
Булыйкчы аналарга!

Бәхетлеләр, кадерлеләр —
Дөньяда алар барда.
Без аларның балалары —
Дан җырлыйк аналарга!

Тик бәхетле көннәр генә
Көтсен аларны алда.
Һәрвакытта кадер-хөрмәт
Күрсәтик аналарга!

Бәхетлеләр, кадерлеләр —
Дөньяда алар барда.
Без аларның балалары —
Дан җырлыйк аналарга!
```
