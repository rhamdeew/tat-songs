# Оригинал

### Рауза Хафизова / Айдар Тимербаев - Яратам (уналтынчы вариант)

```
Мәхәббәтең белән урадың син,
Тормышыма керттең яңалык.
Көчле минем сиңа булган хисләр,
Сөю китаплары язарлык.

Назларыңа күмелә жаннарым,
Елмаеплар миңа карасаң.
Кабатлап, кабатлап әйтәм сиңа:
Яратам, яратам, яратам.

Ямьле иттең иртә-кичләремне,
Чәчәк атты сөю гөлләрем.
Исемең кабатлап күңелемнән,
Сине уйлап йөргән көннәрем.

Назларыңа күмелә жаннарым,
Елмаеплар миңа карасаң.
Кабатлап, кабатлап әйтәм сиңа:
Яратам, яратам, яратам.

Синең өчен мин әзер бар(ы)сына,
Син шикләнмә, ышан үземә.
Чын күңелдән, чын йөрәктән әйткән
"Яратамын" дигән сүземә.

Назларыңа күмелә жаннарым,
Елмаеплар миңа карасаң.
Кабатлап, кабатлап әйтәм сиңа:
Яратам, яратам, яратам.
Кабатлап, кабатлап әйтәм сиңа:
Яратам, яратам, яратам…
Яратам, яратам, яратам…
```
