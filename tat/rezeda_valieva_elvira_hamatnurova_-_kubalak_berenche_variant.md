# Оригинал

### Резеда Вәлиева / Эльвира Хаматнурова - Күбәләк (беренче вариант)

```
Болында уйнап йөрдем,
Матур күбәләк күрдем.
Йөгереп янына килдем,
Әйдә, уйныйбыз, дидем.

Күбәләккәй куанды,
Яныма килеп кунды
Канатларын җилпи-җилпи,
Әй уйнарга тотынды.

Кинәттән ул юк булды
Чәчәкләр арасында.
Чәчәкләрне коча-коча
Карадым барысында.

Ул оча, мин йөгерәм,
Мин йөгерәм, ул оча,
Алтын канатларын белән
Гөлләрне коча-коча.

Тапкач, янына килсәм,
Бер сыйпап карыйм дисәм,
Күбәләккәй кызык итте:
Биеккә очты китте.

Ул оча, мин йөгерәм,
Мин йөгерәм, ул оча,
Алтын канатларын белән
Гөлләрне коча-коча.
```
