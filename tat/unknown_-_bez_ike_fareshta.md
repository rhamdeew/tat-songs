# Оригинал

### Билгесез - Без ике фәрештә

```
Хыяллар илендә,
Мәхаббәт җирендә,
Без биербез икәү
Бәхетнең түрендә.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.

Тотышып куллардан,
Утлардан, сулардан,
Бергәләп үтәргә
Язсын бу юллардан.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.

Катырак очарга,
Күкләргә очарга,
Телим синең белән
Дөньядан качарга.

Син генә, мин генә,
Мин генә, син генә.
Бу якты дөньяда
Без ике фәрештә.
```
