# Оригинал

### Халисә Мөдәрисова / Мөдәрис Газетдинов - Араларда ай гына

```
Ярда — егет, көймәдә — кыз,
Су өстендә ай йөзә.
Ике ярны тоташтырып
Дулкыннарда юл тезә.

Яр астында тал гына,
Яратмасаң, кал гына.
Ә яратсаң, йөгереп төш
Араларда ай гына!

Тал толымын су тарата
Хәтерләтеп юл чатын.
Ике якта яр ерак ла —
Арадагы яр якын.

Чәчрәп китте көмеш дулкын,
Таллардан тамчы тама.
Бер көймәдә икәү бергә —
Бербөтен бәхет ага!
```
