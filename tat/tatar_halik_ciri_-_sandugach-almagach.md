# Оригинал

### Татар халык җыры - Сандугач-алмагач

```
Аллар булганда гына ла,
Гөлләр булганда гына,
Күтәрелә күңелләр шул,
Бергә булганда гына.

Сандугач-алмагач,
Син ай булгач, мин — кояш.

Кайгы димә, хәсрәт димә,
Кайгыларга баш имә.
Кайгыларга башың исәң,
Рәхәт яшәрмен димә.

Сандугач-алмагач,
Син ай булгач, мин — кояш.

Сандугачлар өзеп сайрый
Иртән аяз таңнарда.
Авыр чакта бигрәк кирәк
Бер-береңне аңларга.

Сандугач-алмагач,
Син ай булгач, мин — кояш.

Куй самовар, чәй эчәрбез,
Жәлләмичә күмерне.
Уйнап-көлеп, җырлап-биеп,
Үткәрик лә гомерне.

Сандугач-алмагач,
Син ай булгач, мин — кояш.
```
