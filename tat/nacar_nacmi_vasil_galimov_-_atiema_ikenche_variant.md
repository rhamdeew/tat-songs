# Оригинал

### Наҗар Нәҗми / Вәсил Галимов - Әтиемә (икенче вариант)

```
Бу дөньяда син бар чагында да
Кайнагандыр тормыш, кайнаган.
Бер карасаң, синнән мина, әткәй,
Мирас булып берни калмаган.

Әткәй малы ул мәңгелек түгел
Мал тапмасаң, үзең бил бөгеп.
Мирас булып канга сеңеп калган,
Хезмәт итүләрең тир түгеп.

Мирас булып синнән калган, әткәй,
Туган жиргә булган мәхәббәт.
Саф күңелең өчен, рухың өчен
Рәхмәт сина, әткәй, мең рәхмәт.
```
