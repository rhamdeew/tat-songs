# Оригинал

### Мидхәт Әбделманов / Илшат Фатыйхов - Көт, сөйгәнем, мине

```
Көт, сөйгәнем, мине,
Очратырмын сине.
Тормыш көтәр өчен гомергә,
Тормыш көтәр өчен гомергә.
Шатлыгың да аның
Кайгы-хәсрәтен дә
Бергә-бергә уртак бүлергә,
Бергә-бергә уртак бүлергә.

Көт, сөйгәнем, мине,
Очратырмын сине
Яшьлегемнең алсу таңында,
Яшьлегемнең алсу таңында.
Искән еллар сымак
Язмышымны сынап,
Тормыш юлын гизгән чагында,
Тормыш юлын гизгән чагында.

Очратырга сине
Күп калмаган инде,
Бик күп булса таңнар арасы,
Бик күп булса таңнар арасы.
Бәлки,яннарыңа
Өч-дүрт адым гына
Бары калган миңа барасы,
Бары калган миңа барасы.
```
