# Оригинал

### Гөлнара Сабирова / Илназар - Сөюемне кичер

```
Дисбе итеп тезәм сагышымны,
Һәр төймәсе саен мәхаббәт.
Гөнаһларым минем ярлыкагыл,
Йә Ходаем, күрсәт мәрхәмәт.

Син гөнаһлы сөюемне кичер
Һәм сабырлык бирче бер саплам.
Мәхаббәттән башым әйләнмәскә
Үзең терәк булчы, йә Аллам.

Сагыш басты минем күңел күген,
Кояш янмый миңа, юк аең.
Язмышларны ялгышлардан сакла,
Сөюемнән якла, Ходаем.

Син гөнаһлы сөюемне кичер
Һәм сабырлык бирче бер саплам.
Мәхаббәттән башым әйләнмәскә
Үзең терәк булчы, йә Аллам.
```
