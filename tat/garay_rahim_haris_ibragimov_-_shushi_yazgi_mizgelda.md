# Оригинал

### Гәрәй Рәхим / Харис Ибрагимов - Шушы язгы мизгелдә

```
Тып — тып итеп
Тыпылдап,
Чылтыр — чылтыр
Чылтырап,
Биек йортлар түбәсеннән
Боз сөңгеләр коела.

Ялт — йолт итеп
Ялтырап,
Дерелдәп һәм
Калтырап
Урамдагы гөрләвектә
Кояш нуры коена.

Чырык — чырык
Чыркылдап,
Очып йөри
Пырхылдап
Өй турендә чыпчыклар,
Шушы язгы мизгелдә.

Карамыйча бернигә
Өйдән чыгып чабар идем —
Әни кушкан эшем бар.
```
