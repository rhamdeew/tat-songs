# Оригинал

### Феликс Мортазин - Икенче әнием

```
Әле кайчан гына
Сине белми идем,
Ә бүген син минем
Икенче әнием!

Кайнана белән бер ел
Ике елга тиң диләр
Безгә ике йөз ел бергәләп
Яшәргә язган, димәк!

Йөзең кояш синең
Коймакларың тәмле,
Бәби карыйсың бар
Озак яшә яме!

Әнием икәү хәзер
Бәхетем тулы инде
Бу бәхетне мине яратып
Ходаем үзе бирде

Әнием диярмен
Улым диярсеңме?
Тудырган әнием
Үпкәләмәс әле!

Кайнана белән бер ел
Ике елга тиң диләр
Безгә ике йөз ел бергәләп
Яшәргә язган димәк!

Бу бәхетне мине яратып
Ходаем үзе бирде
```
