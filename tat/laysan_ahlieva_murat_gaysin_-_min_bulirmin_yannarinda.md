# Оригинал

### Ләйсән Әһлиева / Мурат Гайсин - Мин булырмын яннарыңда

```
Кайгы-сагыш какса тәрәзәңне,
Авыр булса беркөн яшәргә…
Исләреңә төшер мин барлыкны,
Ирек бирмәм түбән төшәргә.

Көчсезләнеп язмыш сынавыннан,
Хәлең бетсә алга атларга…
Кулым бирәм сиңа таянырга,
Тик ышана күрмә ятларга…

Ялгыш кына язсаң кыйблаңнан,
Сүнә калса өмет чаткысы…
Мин булырмын һәрчак яннарыңда,
Соңгы тамчы яшәү яктысы…
```
