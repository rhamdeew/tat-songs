# Оригинал

### Расих Ханнанов / Җәүдәт Усманов - Азан моңы

```
Туктап калып барыр, барыр юлларымда,
Шундый бер моң бүген тыңладым.
Тыңладым да үзен, үткәннәрен уйлап,
Күңелләрем әрнеп елады.
Күңелләрем әрнеп елады.

Азан моңы, азан моңы,
Ничек сагынабыз без аны?!

Әллә инде җирнең йөрәк төпкелендә,
Әллә күктә инде бер аваз?
Шул еракка таба, Алла(һ) бит кулында,
Җанны эретерлек җылына.

Азан моңы, азан моңы,
Ничек сагынабыз без аны?!

Күңел кошым минем, читекләрен ватып
Канат какты очып күгемдә.
Нинди шатлык бездә, иман нуры кайта,
Бәндәләргә коръән телендә!

Азан моңы, азан моңы,
Ничек сагынабыз без аны?!
```
