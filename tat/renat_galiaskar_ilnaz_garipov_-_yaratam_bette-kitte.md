# Оригинал

### Ренат Галиәскәр / Илназ Гарипов - Яратам, бетте-китте

```
Ни өчен диеп сорама,
Очраттым да яраттым.
Сиңа гашыйк булганымны
Сизмичә дә калганмын.
Ничек кенә тырышсам да
Сине оныта алмадым.
Хәзер инде синсез яши
Алмавымны аңладым.

Син — минем соңгы тукталышым,
Йөрәк сине үз итте.
Ни өчен диеп сорама,
Яратам. Бетте-китте.

Берәр хәбәр килсә әгәр
Син яшәгән тарафтан,
Яхшы кәеф белән тулы
Кызарып ата һәр таң.
Ничек кенә тырышсам да
Сине оныта алмадым.
Хәзер инде синсез яши
Алмавымны аңладым.

Син — минем соңгы тукталышым,
Йөрәк сине үз итте.
Ни өчен диеп сорама,
Яратам. Бетте-китте.
Сорама…
Яратам. Бетте-китте…
```
