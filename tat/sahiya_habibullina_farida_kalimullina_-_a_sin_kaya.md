# Оригинал

### Сәхия Хәбибуллина / Фәридә Кәлимуллина - Ә син кая?

```
Кара төндә күмерләнеп әрнү кала.
Әкрен генә төнне ярып көн юл ала.
Күңел эзли, йөрәк сызлый.
Ә син кая, ә син кая?
Алсу таңның якты нуры балкый бара,
Шикләнү юк, ышанычым арта бара.
Күңел бизми, сине эзли.
Ә син кая, ә син кая?
Акыл хискә ирек бирми — аяк чала,
Буй-сыннарың күз алдында гына кала…
Күңел эзли, йөрәк сызлый.
Ә син кая, ә син кая?
```
