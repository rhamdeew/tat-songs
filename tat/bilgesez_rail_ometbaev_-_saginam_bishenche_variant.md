# Оригинал

### Билгесез / Раил Өметбаев - Сагынам (бишенче вариант)

```
Сорама миннән, хәлең ничек дип сорама, сабыр итәм генә,
Ә күңелем очрашулар көтә синең белән, җаным,
Аңла син минем сөюемне, аңла син минем сагышымны,
Кош булсам очар идем синең яннарыңа, җаным.

Эх, килсәңче бүген минем яннарыма.
Нигә, җаным минем, син тагын еракта.
Сагынам сине, аңла, мин бик нык сагынам, җаным.
Төнен айга барып ачам мин йөрәкне,
Уртаклашам аның белән серләремне,
Белмимен ниләр булды миңа, мин белмимен, җаным.

Аңла син минем сөюемне, аңла син минем сагышымны,
Кош булсам очар идем синең яннарыңа, җаным.
Сорама миннән хәлең ничек дип сорама, сабыр итәм генә,
Ә күңелем очрашулар көтә синең белән, җаным.

Эх, килсәңче бүген минем яннарыма.
Нигә, җаным минем, син тагын еракта.
Сагынам сине, аңла, мин бик нык сагынам, җаным.
Төнен айга барып ачам мин йөрәкне,
Уртаклашам аның белән серләремне,
Белмимен ниләр булды миңа, мин белмимен, җаным.
```
