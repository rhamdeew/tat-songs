# Оригинал

### Халык / Сара Садыйкова - Туган якка кайтыгыз

```
Акланмагыз намус алларында,
Вакытлар аз, тизерәк кайтыгыз.
Кешеләргә гомер ике килми,
Туган якка ешырак кайтыгыз!

Кайтыгыз сез сабан туйларына,
Кайтыгыз сез кышын тагын да.
Сагынганда җан җылысын бирә
Туган якнын салкын кары да.

Читтә йөргәннәрнең күңелендә
Тирбәлдерә торган сагыш бар.
Очар кошлар булып кайтыгыз сез
Тирбәлгәндә күлләр, камышлар.
```
