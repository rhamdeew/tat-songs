# Оригинал

### Асия Минһаҗева / Вәсим Әхмәтшин - Төшләремә керәсең

```
Ниләр уйлап йөрисең син,
Менә бу минутларда.
Күмер булыйм учагыңда,
Янсаң минем утларда.

Һаман да бер төстә булып,
Төшләремә керәсең.
Мәхәббәттә Сак-Сок булу,
Язган инде күрәсең.

Алдан белеп булса икән,
Тагын ниләр күрәсен.
Яздырыр идем язмыштан,
Кавышулар сүрәсен.

Һаман да бер төстә булып,
Төшләремә керәсең.
Мәхәббәттә Сак-Сок булу,
Язган инде күрәсең.

Еллар гафу итмәс безне,
Кичермәс, алыр үчен.
Тугач-тугач, нигә аны
Тумадың минем өчен?!

Һаман да бер төстә булып,
Төшләремә керәсең.
Мәхәббәттә Сак-Сок булу,
Язган инде күрәсең.
```
