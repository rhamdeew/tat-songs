# Оригинал

### Билгесез / Зөфәр Хәйретдинов - Алтын алма

```
Алманың алтын чагы,
Көзләрнең салкын чагы.
Алтын алмам, кил яныма,
Ачык кайнар кочагым.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.

Ачык кайнар кочагыма
Кер әйдә, җылытармын —
Көзнең ачы салкыннарын
Гомергә онытырсың.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.

Аңладым нигә алманың
Алтын ул дигәннәрен,
Алтын алмам, кайнар алмам
Нәкъ миңа тигәнгәдер.

Алтын алмаларда гына
Җәйнең кызу айлары.
Әллә миңа гына тиде
Алманың иң кайнары.
```
