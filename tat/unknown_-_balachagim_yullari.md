# Оригинал

### Ирек Сакаев - Балачагым юллары

```
Балачагым юлыннан
Кайтам туган ягыма,
Еллар аша шул юлның
Тузаннарын сагынам.

Бигрәк урау бормалы,
Балачагым юллары,
Ычкындырмый җибәрми,
Мине яшьлек еллары.

Балачагым юлыннан,
Урап Кодаш буеннан,
Тальян гармун уйнатып,
Кайтам кичке уеннан.

Бигрәк урау бормалы,
Балачагым юллары,
Ычкындырмый җибәрми,
Мине яшьлек еллары.

Балачагым юлыннан,
Яшьлегем сукмагыннан
Килә кебек өметем,
Беренче мәхәббәтем.

Бигрәк урау бормалы,
Балачагым юллары,
Ычкындырмый җибәрми,
Мине яшьлек еллары.
```
