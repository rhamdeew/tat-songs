# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Юк, теләмим!

```
Юк, теләмим синнән башкаларны
Үткәрергә күңел түренә.
Мәхаббәттә хисләр кайнарлыгы
Еллар узгач ныграк беленә.

Юк, теләмим синсез язларымны,
Юк, теләмим синсез көзләрне.
Сиңа гына кабатлыйсым килә
Икәү белгән серле сүзләрне.

Юк, теләмим ятка, җаным, боздай,
Бозлы җанда гөлләр терелмәс.
Кояш нуры, ләйсән яңгырлары
Синсез генә күзгә күренмәс.

Юк, теләмим синсез язларымны,
Юк, теләмим синсез көзләрне.
Сиңа гына кабатлыйсым килә
Икәү белгән серле сүзләрне.

Юк, теләмим ятлар учагына
Кочак-кочак сөю салырга.
Сиңа гына телим табынырга
Синең өчен генә янарга.

Юк, теләмим синсез язларымны,
Юк, теләмим синсез көзләрне.
Сиңа гына кабатлыйсым килә
Икәү белгән серле сүзләрне.
```
