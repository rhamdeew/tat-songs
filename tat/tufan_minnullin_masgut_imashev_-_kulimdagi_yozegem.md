# Оригинал

### Туфан Миңнуллин / Мәсгуть Имашев - Кулымдагы йөзегем

```
Кулымдагы йөзегемнең
Исемнәре эчендә.
Мин бит үзем генә түгел,
Янам синең өчен дә.

Кулымдагы йөзегемнең
Исемнәре ким түгел.
И дускаем, синнән башка
Яшәве мөмкин түгел.

Кулымдагы йөзегемнең
Исемнәре — Гыйльфаным.
Сине уйлап, үзәкләрем
Өзелә инде, җаным.

Кулымдагы йөзегемнең
Исемнәре — Майбәдәр.
Сине, дустым, оныталмам
Кыямәт көнгә кадәр.
```
