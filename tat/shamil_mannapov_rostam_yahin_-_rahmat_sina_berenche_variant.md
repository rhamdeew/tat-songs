# Оригинал

### Шамил Маннапов / Рөстәм Яхин - Рәхмәт сиңа (беренче вариант)

```
Рәхмәт сиңа матур итеп көлгән өчен,
Көтмәгәндә башым әйләндергән өчен,
Әлегәчә гади булган бар нәрсәне
Әллә нинди сихри ямьгә төргән өчен.

Рәхмәт сиңа төшләремә кергән өчен,
Көтеп алган яңгыр булып килгән өчен,
Күңелемдәге сулып барган гөлләремә
Яшәртүле яшәү көче биргән өчен.

Рәхмәт сиңа иң бәхетле көннәр өчен,
Кавыштырган сихри айлы төннәр өчен.
Рәхмәтемнең иң олысы: мәхәббәтне
Гомер буе саклап кала белгән өчен.
```
