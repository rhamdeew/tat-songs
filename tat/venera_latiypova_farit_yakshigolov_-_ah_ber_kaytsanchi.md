# Оригинал

### Венера Латыйпова / Фәрит Якшыголов - Әх, бер кайтсаңчы …

```
Эх, бер генә кайтып әйләнсәңче,
Ана, ата булган чакларга.
Үбеп, сөеп иркәләнер идем
Икесенең дә җылы кочагында.

Әнкәй-әнкәй, минем әнкәем
Өзлеп-өзлеп сине сагындым.
Әткәй-әткәй, акыллы (хәстәрле) әткәем
Өзлеп-өзлеп сезне сагындым.

Эх, бер генә кунып китәр идем,
Туып үскән газиз өемдә.
Мәтрүшкәле тәмле чәйләр эчеп,
Мичтәге җылы икмәккә.

Әнкәй-әнкәй, минем әнкәем
Өзлеп-өзлеп сине сагындым.
Әткәй-әткәй, акыллы (хәстәрле) әткәем
Өзлеп-өзлеп сезне сагындым.

Эх, бер генә җиңел әйтүләре
Гомер үткән, кайтмас вакытка.
Ә еракта калган бала чакны
Оныторга бирми хәтер җибе.

Әнкәй-әнкәй, минем әнкәем
Өзлеп-өзлеп сине сагындым.
Әткәй-әткәй, акыллы (хәстәрле) әткәем
Өзлеп-өзлеп сезне сагындым.
```
