# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Мин ниндиме?

```
Мин ниндиме?
Менә шундый:
Ничек бармын — күз алдыңда.
Иңдәш булсам, булырмын, тик
Көндәш булмам хыялыңда.

Мин әзермен артмакларга
Шатлыгың да, көюең дә.
Җандаш булсам, булырмын, тик
Ярдаш булмам сөюеңдә.

Ялгыз итмәм аймылышта,
Хак хөкем көт ялгышыңда.
Көйдәш булсам, булырмын, тик
Өйдәш булмам язмышыңда.
```
