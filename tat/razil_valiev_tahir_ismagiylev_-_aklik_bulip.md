# Оригинал

### Разил Вәлиев / Таһир Исмәгыйлев - Аклык булып

```
Тагын бер ел иңләп узды —
Язмышым урамнарын.
Кайгыларны адаштыра
Яңа ел бураннары.

Тагын бер ел шаулап узды,
Туздырып ап-ак карны.
Ак кар булып уйнап оча
Яңа ел шатлыклары.

Тагын бер ел узды җирдән,
Тик хәтер булып калды.
Аклык булып җиргә ята
Яңа ел болытлары.

Тагын бер ел гөрләп узды —
Сиздем гомер узганын.
Бәхет кебек балкый күктә
Яңа ел йолдызлары.
```
