# Оригинал

### Шәмсия Җиһангирова / Ганс Сәйфуллин - Карап калам кошлар артыннан

```
Көзләр җиткән чакта язын безгә кайткан
Очар кошлар юлга җыена.
Зарыгып кына көткән бәхетемне алар
Алып китә төсле тоела.
Зарыгып кына көткән бәхетемне алар
Алып китә төсле тоела.

Карашларым уйчан. Тезелешеп очкан
Кошларны мин юлга озаттым.
"Язлар җиткәч кабат очрашырбыз әле…" —
Үз-үземне шулай юаттым.

Тезелешеп тагын кошлар кайтыр язын,
Туган җиргә юллар бер генә.
"Бәхетемне юлда югалтмаслар микән?" —
Шуны уйлап өзәк өзелә.
"Бәхетемне юлда югалтмаслар микән?" —
Шуны уйлап өзәк өзелә…
```
