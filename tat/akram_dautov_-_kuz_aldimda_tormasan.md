# Оригинал

### Әкрәм Даутов - Күз алдымда тормасаң

```
Дөнья киң, юл иркен, көннәр чуак
Тоелмас иде һис бер кайчан,
Миңа көч, сүнмәс дәрт, илһам биреп,
Күз алларымда син тормасаң.

Бер шатлык күңелгә ятмас иде,
Өметем кояшы булмасаң,
Белмимен, ничек мин яшәр идем,
Күз алларымда син тормасаң.

Ниндидер шомлы саз моңын ишетәм,
Көзге таң җилләрен тыңласам,
Моңнарга күмелеп шашыр идем,
Күз алларымда син тормасаң.

Гел яшә нур сибеп, ләззәт чәчеп,
Ал төсең уңмасын һич кайчан.
Дөньяның бар яме сүнәр төсле,
Күз алларымда син тормасаң.
```
