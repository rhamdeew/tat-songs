# Оригинал

### Зөлфәт / Сәлимә Шәмсина - Төнге утлар

```
Төнге утлар, төнге утлар…
Калдылар еракларда.
Калды утлар, син дә калдың,
Кул болгап ерак ярда.

Кагылма, язмыш, каралтма
Син аның карашларын!
Төнге утларга карыйдыр
Минем сөйгән ярым.

Чәчләренә кагылды җил,
Шаулады язгы Идел.
Утсыз калыр барлык төннәр,
Онытсам сине, дидем.

Төнге утлар, төнге утлар…
Кара бер кабынганда:
Ераклардан балкыр алар
Мин сине сагынганга.

Арабызда ерак юллар —
Йолдыз нурлары тама.
Сөю сагышлары булып
Төнге утлар яна…
```
