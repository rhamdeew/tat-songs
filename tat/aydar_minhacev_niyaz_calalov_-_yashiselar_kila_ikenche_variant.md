# Оригинал

### Айдар Минһаҗев / Нияз Җәләлов - Яшиселәр килә (икенче вариант)

```
Башкаемны ияселәр килә
Сине тәүге күргән көнемә.
Саташулыр гына түгел лә бу,
Булдың бит син минем үземә.

Яшиселәр килә, яшиселәр,
Сине генә сөеп өзелеп.
Түгел әле, түгел яшәү генә,
Үлергә дә була син диеп.

Сөюемнең бөртегенә кадәр
Яулап ала белдең син генә.
Әлдә ярый син бар яннарымда,
Мең рәхмәтләр әйтәм үзеңә.

Яшиселәр килә, яшиселәр,
Сине генә сөеп өзелеп.
Түгел әле, түгел яшәү генә,
Үлергә дә була син диеп.

Әлдә ярый син бар яннарымда,
Бәхетемә исем китерлек.
Бирә күрсен Ходай сөюләрне
Гомер буйларына җитәрлек.

Яшиселәр килә, яшиселәр,
Сине генә сөеп өзелеп.
Түгел әле, түгел яшәү генә,
Үлергә дә була син диеп.
```
