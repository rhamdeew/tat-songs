# Оригинал

### Әнгәм Атнабаев / Хисмәтулла Дәүләтов - Җыр кызы

```
Син күктәге йолдыз түгел,
Син җирдәге бер кыз гына.
Җирр егетен чакырма син
Күкләрдәге йолдызына.

Җирдә тудым, җирдә торам,
Күңел шуңа күнгән инде.
Синең йолдыз — тик ҡкабына,
Минең йолдыз — сүнгән инде.

Синең тормыш — киләчәктә,
Минем тормыш — үткәннәрдә.
Киләчәккә барып була,
Кайтып булмый үткәннәргә.

Киләчәктәй ерак та син,
Үткәннәрдәй якын да син.
Җирдә үзең бер йолдыз бул,
Йолдызларга чакырма син.
```
