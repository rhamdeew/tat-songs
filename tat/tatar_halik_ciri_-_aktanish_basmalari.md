# Оригинал

### Татар халык җыры - Актаныш басмалары

```
Агыйделдән күренеп тора
Актаныш басмалары шул;
Кызлар чәчен үреп салса,
Җилферди тасмалары.

Апрель җитми, агып бетми
Агыйделнең бозлары шул;
Мул уңышлар яулап ала
Егетләре, кызлары.

Каян барсаң, шуннан туры
Актанышның юллары шул;
Хезмәтләрдә җиңү яулый
Безнең район уллары.
```
