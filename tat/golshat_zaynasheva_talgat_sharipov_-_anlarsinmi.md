# Оригинал

### Гөлшат Зәйнәшева / Тәлгать Шәрипов - Аңларсыңмы? ..

```
Язгы күкрәү булып,
Идел кебек тулып,
Йөрәгемә килеп кердең син.
Язлар килгән-киткән,
Гомер синсез үткән,
Әлегәчә кайда йөрдең син? ..

Утка каршы барам,
Суга каршы йөзәм,
Тик бер елмай мине чакырып.
Күпме яшәп җирдә,
Тынгы белми бер дә
Сине эзләгәнмен, ах(ы)ры.

Инде син табылдың,
Йөрәктә кабындың,
Күктә яңа йолдыз янгандай.
Бер аңларсың микән,
Сабыр гына көтәм,
Җир язларны көтеп алгандай.
Җир язларны көтеп алгандай.
```
