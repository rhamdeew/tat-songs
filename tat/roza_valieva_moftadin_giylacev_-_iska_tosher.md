# Оригинал

### Роза Вәлиева / Мөфтәдин Гыйләҗев - Искә төшер

```
Хискә тулы йөрәгемне
Тагын бер яндыр әле.
Керфекләрем очларыннан
Тамчылар тамдыр әле.

Бер исемә төшер әле.
Күңелле чакларымны.
Күз алдыма китер әле
Үз туган якларымны.

Искә төшер авылымның
Тыныч айлы төннәрен.
Кышларында чәчәк аткан
Әнкәемнең гөлләрен.
```
