# Оригинал

### Хәсән Туфан / Сара Садыйкова - Беренче тапкыр күрүем

```
Беренче тапкыр күрүем,
Шулай да таныйм кебек.
Күзләрең аша йөземә
Язмышым карый кебек.

Сурәтең хыял күгемдә
Ай булып тулган иде.
Тик минем ялгыз йөрәктән
Еракта сыман иде.

Беренче күрү таңының
Аягы җиңел микән.
Үкенечкә булыр микән,
Бәхетләр килер микән.
```
