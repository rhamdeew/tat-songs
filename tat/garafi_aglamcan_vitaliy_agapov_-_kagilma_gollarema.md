# Оригинал

### Гарәфи Әгъләмҗан / Виталий Агапов - Кагылма гөлләремә

```
Сандугач суга сусаган,
Эчалмый гөлләр чыгын.
Коела көмеш чыклары,
Таңнарда җилләр чыгып.

Узма янымнан җил булып,
Кагылма гөлләремә.
Шатлык бул, янымда калып,
Туасы көннәремә.

Күзләреңдә шатлык күрәм;
Карадың — җан җылынды.
Гөлләремә кушып үрәм
Сиңа дигән җырымны.

Узма янымнан җил булып,
Кагылма гөлләремә.
Шатлык бул, янымда калып,
Туасы көннәремә.

Үрим дә тезим юлыңа,
Тургайлар көй чыгарсын.
Эчендә моң тулы тургай
Җырсыз ничек чыдар соң?

Узма янымнан җил булып,
Кагылма гөлләремә.
Шатлык бул, янымда калып,
Туасы көннәремә.
```
