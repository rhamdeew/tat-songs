# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Ерактагы якын кешемә (икенче вариант)

```
Янып яшим инде күпме еллар,
Син кабызган сөю утында.
Йөрәк бәйдә, хисләр тоткынлыкта,
Ычкындырсаң сиңа омтыла.

Ерактагы якын кешем минем,
Тагын кердең бүген төшемә.
Әйләндең син, мәңге сөеп туймас,
Йөрәктәге якын кешемә!

Синең яктан аккан болытларның
Ни әйтәсен күптән белсәм дә.
Сиңа булган сөю ташкыныннан
Әйләнәм бит инде исәргә.

Ерактагы якын кешем минем,
Тагын кердең бүген төшемә.
Әйләндең син, мәңге сөеп туймас,
Йөрәктәге якын кешемә!
Ерактагы якын кешем минем,
Тагын кердең бүген төшемә.
Әйләндең син, мәңге сөеп туймас,
Йөрәктәге якын кешемә!..
```
