# Оригинал

### Татар  халык җыры - Казансу

```
Әбәү, Алла, уф, Алла,
Торган җирләрем кала.
Торган җирләрем жәл түгел,
Туып үскән ил кала.

Казансуны чыккан чакта,
Җуйдым беләзегемне.
Китәрсең дә кайтмассың дип,
Өзмәгез үзәгемне.

Казансуга басма салдым,
Тар булмаска чыгарга.
Бергә булган вакытларны
Исләреңнән чыгарма.

Кашың кара, кашың кара,
Каралткангадыр инде.
Карыйм, карыйм, карап туймыйм,
Яраткангадыр инде.

Казаннарга барам әле,
Казан суы тәмлегә.
Бир кулыңны, бергә булыйк,
Аерылмаска мәңгегә.

Аерылмаска, аерылмаска,
Аерылмаска мәңгегә.
```
