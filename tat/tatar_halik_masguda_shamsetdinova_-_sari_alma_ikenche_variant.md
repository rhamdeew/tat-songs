# Оригинал

### Татар халык / Мәсгудә Шәмсетдинова - Сары алма (икенче вариант)

```
Сандык кынам тулы сары алма,
Эх, нурым, җанашым, җанашым,
Түбәләмә, тулама.

Ялтыр да гнай ни ялтырый,
Эх, нурым, җанашым, җанашым,
Сөйгәнемнең тәңкәсе.

Алыр да гнай идем шул матурны,
Җанашым, бирми әнкәсе.
А-а…
```
