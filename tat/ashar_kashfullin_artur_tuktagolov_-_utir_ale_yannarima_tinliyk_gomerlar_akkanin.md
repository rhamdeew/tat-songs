# Оригинал

### Әсхәр Кәшфуллин / Артур Туктаголов - Утыр әле яннарыма (Тыңлыйк гомерләр акканын)

```
Утыр әле яннарыма,
Оныт хәсрәтләреңне.
Бушат әле йөрәгемә.
Күтәргән йөкләреңне.
Тыңлыйк без әле чишмәнең
Чылтыр — чылтыр акканын.
Син бит минем яшьлегемдә
Иң тәүге яртаканым.

Утыр әле яннарыма,
Тыңлыйк гомерләр акканын.
Син бит минем мәхәббәтем,
Югалтканым, тапканым.

Гөлләр дә ычык бөртеге,
Көлә кояш какырып.
Гашыйк булган чакларыдай,
Чәчкәйләреңне тарат.
Тыңлыйк без гөлләр үскәнен,
Сикерткәләрнең җырын.
Иңнәремә башын куеп,
Сөю утында җылын.

Утыр әле яннарыма,
Тыңлыйк әле гомер акканын.
Син бит минем мәхәббәтем,
Югалтканым, тапканым.

Табигатькә соклагандай,
Сокланып туялмим мин.
Яратуымны белгертеп,
Әйтергә дә кыймим мин.
Утыр әле яннарыма,
Оныт хәсрәтләреңне.
Бушат әле йөрәгемә.
Күтәргән йөкләреңне.

Утыр әле яннарыма,
Тыңлыйк гомерләр акканын.
Син бит минем мәхәббәтем,
Югалтканым, тапканым.
```
