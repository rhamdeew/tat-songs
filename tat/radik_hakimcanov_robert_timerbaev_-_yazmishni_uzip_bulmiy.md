# Оригинал

### Радик Хәкимҗанов / Роберт Тимербаев - Язмышны узып булмый

```
Кайчан гына ике йөрәк
Гел бергә тибә инде.
Парлар булып бер ямь булып
Якты нур сибә инде.

Язмышны узып булмый,
Арада куе томан.
Шуңа да күңелемә
Сары сагышлар тулган.

Язмышларны узып булмый
Булгандыр күрәчәгем.
Белмәдем сөюемнең
Бик иртә сүнәчәген.

Язмышны узып булмый
Арада куе томан.
Шуңа да күңелемә
Сары сагышлар тулган.

Нигә аңламадың икән
Мәхәббәт хисләремне.
Бер сүз әйтми карап калдың
Ялгызым киткәнемне.

Язмышны узып булмый
Арада куе томан.
Шуңа да күңелемә
Сары сагышлар тулган.
```
