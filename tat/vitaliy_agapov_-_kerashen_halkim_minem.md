# Оригинал

### Виталий Агапов - Керәшен халкым минем

```
Керәшен халкым минем,
Җырларыңда синең,
Чишмәләрдәй чылтырап моң ага.
Шушы моңнар безгә
Гасырлардан килгән,
Ата-бабалардан җыр аша.
Керәшен халкым минем,
Яшә мәңге,
Җырларыңны сакла, моңыңны.
Җырларыбыз аша,
Моңнарыбыз аша,
Күрсен дөнья минем халкымны!
Чит җирләрдә йөрдем,
Күпме илләр күрдем,
Тик күрмәдем сезнең кебекне.
Әйтер өчен сезгә
Халкым, диеп, дуслар,
Җыяр идем сезне бер җиргә.
```
