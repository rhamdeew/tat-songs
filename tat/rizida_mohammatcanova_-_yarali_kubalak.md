# Оригинал

### Ризида Мөхәммәтҗанова - Яралы күбәләк

```
Китәр идем ерак-ерак,
Ал язмыш таңнарына.
Яралы күбәләк булып
Очам гел яннарыңда.

Кушымта:
Яшәвемнең яме микән
Хәсрәт-сагышларым да?
Яралы күбәләк булдым
Синең язмышларыңда.

Энҗе бөртекләре тама
Гөл-чәчкә таҗларына.
Язмышым илтә хисләрне
Җәһәннәм сазларына.

Кушымта.

Иртә таңда кайнар энҗе
Чыкның тамчыларында.
Җәрәхәткә тозлар сибеп,
Җанны камчылый гына.

Кушымта.
```
