# Оригинал

### Ризван Хәкимов - Елама, ярсыма

```
Ашмады гамәлгә
Ант итеп бирешкән вәгъдәләр.
Елама, ярсыма —
Син үзең гаепле барсына.

Үтте ул, сүнде ул
Үзебез кабызган мәхәббәт.
Елама, ярсыма —
Син үзең гаепле барсына.

Саф идең, белмәдең,
Син үзең, гаепсез, көнләдең.
Елама, ярсыма —
Син үзең гаепле барсына.

Болытлар таралыр,
Ышаныч кыскартыр араны.
Елама, ярсыма —
Елмаеп кил, иркәм, каршыма.
```
