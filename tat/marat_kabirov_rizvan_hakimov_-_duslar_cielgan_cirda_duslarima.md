# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Дуслар җыелган җирдә (Дусларыма)

```
Шатлыклар уртаклашырга
Җыелган дусларыбыз.
Дуслар белән бергәләшеп
Күңел ачкан чагыбыз.

Әйдәгез, уйнап-көлешеп
Утырыйк әле бергә.
Бәйрәмнәрнең иң зурысы
Дуслар җыелган җирдә.

Дуслар дус булып калсыннар
Иң авыр чагында да.
һәм ямь өстәп утырсыннар
Шатлыклы табынга да.

Әйдәгез, уйнап-көлешеп
Утырыйк әле бергә.
Бәйрәмнәрнең иң зурысы
Дуслар җыелган җирдә.

Иң-иң якын, иң кадерле
Дуслар җыелган бүген.
Безнең якты хисләребез
Балкытсын күңел күген.

Әйдәгез, җырлап-көлешеп
Утырыйк әле бергә.
Бәйрәмнәрнең иң зурысы
Дуслар җыелган җирдә.
```
