# Оригинал

### Сибгат Хәким / Сара Садыйкова - Туй итә Казан

```
Батыр һәйкәле алдына,
Өзеп алып хисләрдән,
Салалар туй чәчәкләре
Яңа өйләнешкәннәр.

Киләчәк бөек эшләргә
Үзен багышлый яшьлек.
Уйлана ерак чорларга
Узып барышлый яшьлек.

Гөр килеп туй итә яшьлек,
Үзе башкода Казан,
Ленталар белән бизәлеп,
Кая ашкына Казан?

Киләчәк якты елларның
Каршына чаба Казан.
```
