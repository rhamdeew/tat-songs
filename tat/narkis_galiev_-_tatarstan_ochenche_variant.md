# Оригинал

### Нәркис Галиев - Татарстан (өченче вариант)

```
Бер сер чишәм сезгә:
Мин — бәхетле кеше,
Туган җирем минем —
Гүзәл Татарстан.

Агалар урманнар,
Тигез-тигез таулар,
Барысы да миңа багалар.
Агалар урманнар,
Биек-биек таулар,
Барысы да миңа багалар.

Әнием телендә,
Әтием телендә,
Горур сөйләшәм
Татар телемдә.

Сезнең алдыгызда
Сезгә вәгъдә бирәм:
Онытмаска татар телемне.

Туган телем минем —
Минем горурлыгым.
Белсәк иде һәркөн
Аның моңлылыгын.

Сезнең алдыгызда
Сезгә вәгъдә бирәм:
Онытмаска туган телемне.
Сезнең алдыгызда
Сезгә вәгъдә бирәм:
Онытмаска татар телемне.
Онытмаска татар телемне…
```
