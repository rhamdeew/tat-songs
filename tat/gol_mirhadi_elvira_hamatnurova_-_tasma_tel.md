# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Тасма тел

```
Кыз:
Бергә мәктәпкә барырга
Үзең вәгдә итәсең.
Бер үзем бардым мәктәпкә
Тагын ташлап киткәнсең.
Малай:
Үпкәләмә бик хәерле
Үзең генә барганың
Сине төшләремдә күреп
Мин бит йоклап калганмын

Кыз:
Дуслар шундый була мыни:
Көн дә тартасың чәчтән.
Елап беттем. Шуңа кәрә
Башым авыртты кичтән.
Малай:
Күбәләк бантигың белән
Үзәгемне өзәсең
Шундый матур чәчләреңнән
Ничек тартмый түзәсең

Кыз:
Сандугачлар җим ашата
Бер-беренә сайрашып.
Кичә биргән кәнфитеңне
Бүген алдың талашып.
Малай:
Сине уйлап үзәкләрем
Өзелеп кипкән иде.
Синең кулдан кәнфит капкач
Җан кереп киткен иде.

Кыз:
Шушы тасма телең белән
Әйләндерәсең башны
Ичмасам мәктәпкә кәдәр
Сумкаларымны ташы!
```
