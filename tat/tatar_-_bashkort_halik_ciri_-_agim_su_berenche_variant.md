# Оригинал

### Татар - Башкорт халык җыры - Агым су (беренче вариант)

```
Агым суның буйларында
Матур кырлар, урманнар.
Шул кырлар да тырышып эшли
Безнең дуслар, туганнар.

Акчаралаклар суга төшә, / Акчарлаклар йөзеп уйный /
Агым су буйларына. /Агым су өсләрендә /
Иртән — истән, төнен — төштә
Сыкмыйсың уйларымнан.

Ага сулар, ага сулар,
Ага сулар гел шулай.
Аккан сулар кире кайтмый
Яшь гомерләр дә шулай. /Бу гомерләр дә шулай. /
```
