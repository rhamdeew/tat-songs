# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Сине күрсәм (өченче вариант)

```
Сине күрсәм — тик торганнан
Янамын да пешәмен.
Күзеңә дә карый алмыйм,
Җебимен дә төшәмен.

Мин гашыйк булганмын икән,
Шуны да тоймаганмын.
Мин бит сине гомер буе
Яратып туймаганмын.

Сине күрсәм — тик торганнан
Башым әйләнә башлый.
Яратам дип әйтә алмыйм,
Телем бәйләнә башлый.

Мин гашыйк булганмын икән,
Шуны да тоймаганмын.
Мин бит сине гомер буе
Яратып туймаганмын.

Сине күрсәм — тик торганнан
Чуала фикерләрем.
Син мине әллә нишләттең,
Син мине сихерләдең.

Мин гашыйк булганмын икән,
Шуны да тоймаганмын.
Мин бит сине гомер буе
Яратып туймаганмын.
Мин бит сине гомер буе
Яратып туймаганмын…
```
