# Оригинал

### Татар халык сүзләре / Инсаф Хәбибуллин - Яратам (беренче вариант)

```
Синең кара кашларың,
Синең кара күзләрең,
Синең ягымлы сүзләрең…
Яна синең йөзләрең.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.

Сандугачның күрсәң иде
Юлга тезелгәннәрен.
Һичкем белми йөрәгемнең
Нигә өзелгәннәрен.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.

Алма бакчасына керәм,
Үргән чәчемне сүтәм.
Сабыр итәм, сине көтәм,
Кирәкми миңа бүтән.

Мин сине яратам,
Яратам, яратам.
Мин сине, барыбер,
Үземә каратам.
```
