# Оригинал

### Әхмәт Ерикәй / Мансур Мозаффаров - Сипкелле диләр мине…

```
Туткыллы димиләр мине,
Сипкелле диләр мине.
Әзрәк сипкелле булсам да,
Сөйкемле диләр мине.

Буйларың зифа димиләр,
Аз гына кыска диләр.
Ләкин шуңа карамастан.
Эшкә бик оста диләр.

Кашларың кара димиләр,
Күзләрең серле диләр,
Шушы күзләр егетләрнең
Күңелен сихерли диләр.

Чәчләрең бөдрә димиләр,
Билләрең бөглә диләр.
Яраттыра белмәсә дә,
Ярата белә диләр.

Туткыллы димиләр мине,
Сипкелле диләр мине.
Әзрәк сипкелле булсам да,
Сөйкемле диләр мине.
```
