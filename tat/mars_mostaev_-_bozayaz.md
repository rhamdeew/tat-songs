# Оригинал

### Марс Мостаев - Бозаяз

```
Бозаязым туган авылым
Уралган урман белән,
Сулга барсаң Өчимәнең
Уң якта Караталың.

Бозаязым, Бозаяз
Бозаязда көн аяз.
Бозаязым, Бозаяз
Бозаязда көн аяз.

Бозаязым туган авылым
Матурда тугайларың,
Сагындыра сөендерә
Синең аяз таңнарың.

Бозаязым, Бозаяз
Бозаязда көн аяз.
Бозаязым, Бозаяз
Бозаязда көн аяз.

Бозаязым туган авылым
Печәндә кешеләрең,
Үзе шаян, үзе дәртле
Шундый синең яшьләрең.

Бозаязым, Бозаяз
Бозаязда көн аяз.
Бозаязым, Бозаяз
Бозаязда көн аяз.

Бозаязым туган авылым
Бигрәк тәмле суларың,
Кайда гына булсам да мин
Синдә генә уйларым.

Бозаязым, Бозаяз
Бозаязда көн аяз.
Бозаязым, Бозаяз
Бозаязда көн аяз.
```
