# Оригинал

### Әхсән Баян / Фуат Әбүбәкеров - Кил яшьлектән бүгенге көннәргә

```
Синең өчен чәчәк җыям, эзләп син сөйгәннәрен.
Ишетмим җилләрнең сине тапмадык дигәннәрен.

Ышанмыйм, ышанмыйм, ул калды ерак.
Эзләмә, эзләмә дигән сүзләргә.

Сине көтәм, көтәм, кил яшьлектән,
Кил яшьлектән бүгенге көннәргә.

Синең өчен чәчәк җыям, булмасаң да янымда.
Килмәссеңне белеп көтәм, көтсәм дә бераз булыр.

Синең өчен чәчәк җыям, кыскармый калмас ара.
Очрашулар, мәхәббәтем чәчәкләр илтә бара.
```
