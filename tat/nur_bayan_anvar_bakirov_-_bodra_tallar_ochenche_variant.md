# Оригинал

### Нур Баян / Әнвәр Бакиров - Бөдрә таллар (өченче вариант)

```
Хат язсам да бармас төсле,
Ахры ерак киттем илемнән,
Бала чакларымда уйнап йөргән,
Чия чәчәк аткан җиремнән.

Ерак киттем. Бөдрә таллар
Көтәләрдер таныш җырчысын;
Көтәләрдер кырлар, биек таулар
Шат күңелле еллар юлчысын.

Борчылмагыз, минем дуслар,
Шат җырчыгыз сезне онытмас,
Бер кайтыр ул якты кояшсыман,
Дошманнардан җирне корыткач.
```
