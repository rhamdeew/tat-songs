# Оригинал

### Нур Әхмәдиев / Зиннур Сафиуллин - Карлыгачым (беренче вариант)

```
Нәрсә булды, тик син генә,
Син генә күз алдымда.
Карлыгачлар уйный кебек
Каш сикерткән чагында,
Каш сикерткән чагыңда.

Карлыгачым — кыйгач кашым,
Кара, кара канатым.
Сизеп торам: үкенечкә
Сине, сине яраттым.

Карлыгачлар, карлыгачлар
Ник уйнап оча икән?
Йөрәкнеке йөрәкләргә
Әйт, кайчан күчә икен?
Әйт, кайчан күчә икән?

Карлыгачым — кыйгач кашым,
Кара, кара канатым.
Сизеп торам: үкенечкә
Сине, сине яраттым.

Кайрылмасын канатлары
Карлыгачкайларымның.
Бер елмаеп карадың да,
Кайгылардан арындым,
Кайгылардан арындым.

Карлыгачым — кыйгач кашым,
Кара, кара канатым.
Сизеп торам: үкенечкә
Сине, сине яраттым.
```
