# Оригинал

### Халык / Афарим Акчурин - Агыйделне иңләп буйлап

```
Агыйделне иңләп, буйлап,
Йөзәсең үрдәккәем.
Ташлармын дисәң, ташларсың,
Таш булса бәгеркәең.

Җәйгә чыксам барам әле
Агыйдел буйларына.
Ташласаң да оныталмассың
Гомерең буйларына.

Минем чәчем җиде толым,
Җидедән үрмәсәм дә,
Җиде якка күзне салмам,
Җиде ел күрмәсәм дә.
```
