# Оригинал

### Клара Булатова / Искәндәр Биктаһиров - Каеннар юллар буенда

```
Каеннар юллар буенда
Җырлыйлар иде кебек.
Килеүең көттем, юлына
Күз караларын түгеп.

Килмәде, килә алмады,
Килергә теләмәде.
Каеннар тынып калдылар,
Кайгылы, күләгәле …

Юлларга зәңгәр тасмадай
Нурлар сузылган иде.
Сүзләрнең әйтми калганы
Күпме җыелган иде?

Килмәде, килә алмады,
Килергә теләмәде.
Каеннар тынып калдылар,
Кайгылы, күләгәле …

Каеннар күләгәсеннән
Караңгы эзләр калды.
Әйтә алми калган сүзләрем
Йөрәккә яра салды.

Килмәде, килә алмады,
Килергә теләмәде.
Каеннар тынып калдылар,
Кайгылы, күләгәле …
```
