# Оригинал

### Марсель Гыймазетдинов / Вил Усманов - Ташлама тирәкләрне

```
Инеш буенда тирәкләр
Нигәдер уйга талган.
Бергә йөргән сукмакларда
Яшьлек эзләре калган.

Уйный идек, көлә идек
Тирәкле су буенда.
Бергә йөргән, бергә узган
Кичләр минем уемда.

Сукмакларны табалмадым,
Үләннәр баскан икән.
Тирәгебез дә нигәдер
Моңаеп башын игән.

Уйный идек, көлә идек
Тирәкле су буенда.
Бергә йөргән, бергә узган
Кичләр минем уемда.

Аңладым, тик соң аңладым,
Тирәкләрне ташлама.
Гомерең нинди булса да,
Тәүге эздән башлана.

Уйный идек, көлә идек
Тирәкле су буенда.
Бергә йөргән, бергә узган
Кичләр минем уемда.
```
