# Оригинал

### Венер Котдусов - Менә шулай

```
Әллә нигә, әллә нигә, әллә нигә
Ай каршында ялгыз йолдыз бер яна да бер сүнә.
Әллә нигә ярсып-ярсып йөрәк тибә,
Җилкуарлар дөнья куа, эшкуарлар төш күрә.

Ерак-ерак калды туфрак, туган яклар,
Төн үтә дә керфек какмый, туа таңнар.
Яна, яна баш очында тик тулган ай,
Ай, ай, ай, менә шулай.

Менә шулай килеп җиттек, менә шулай,
Ата үрдәк инә казны сүгә-сүгә зар җылый.
Эшләр шулай, эх, дускаем, эшләр шулай,
Дөнья хәлен аңлый алмый урманда бүре улый.

Ерак-ерак калды туфрак, туган яклар,
Төн үтә дә керфек какмый, туа таңнар.
Яна, яна баш очында тик тулган ай,
Ай, ай, ай, менә шулай.

Ерак-ерак калды туфрак, туган яклар,
Йөрәк сызлый…
Төн үтә дә керфек какмый, туа таңнар.
Нидер сизеп йөрәк сызлый…
Яна, яна баш очында тик тулган ай,
Иртә таңдан яңа мәйдан, шайтан Сабантуедай,
Ай, ай, ай, менә шулай…
```
