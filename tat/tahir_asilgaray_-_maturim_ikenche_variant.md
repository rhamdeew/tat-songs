# Оригинал

### Таһир Асылгәрәй - Матурым (икенче вариант)

```
Матурым, иң матурым,
Кара, ачып, миңа туры.
Нишләсәм дә сиңа таба
Бара юлым.
Матурым, иң матурым,
Кара әле син миңа туры.
Нишләсәм дә сиңа таба
Бара юлым.

Мин сине күрми торалмыйм,
Җаныма урын табалмыйм.
Сиңа гына минем күзем төште,
Сайланмыйм.
Сиңа гына минем күзем төште…

Матурым, иң матурым,
Кулларымда синең кулың.
Нишләсәм дә сиңа таба
Бара юлым.
Матурым, иң матурым,
Кулларымда назлы кулың.
Нишләсәм дә сиңа таба
Бара юлым.

Мин сине күрми торалмыйм,
Җаныма урын табалмыйм.
Сиңа гына минем күзем төште,
Сайланмыйм.
Сиңа гына минем күзем төште…

Матурым, иң матурым,
Язмышыбыз бергә булыр.
Нишләсәм дә сиңа таба
Бара юлым.
Матурым, иң матурым,
Тормышыбыз бергә булыр.
Нишләсәм дә сиңа таба
Бара юлым.

Мин сине күрми торалмыйм,
Җаныма урын табалмыйм.
Сиңа гына минем күзем төште,
Сайланмыйм.
Мин сине күрми торалмыйм,
Җаныма урын табалмыйм.
Сиңа гына минем күзем төште,
Сайланмыйм.
Сиңа гына минем күзем төште…
```
