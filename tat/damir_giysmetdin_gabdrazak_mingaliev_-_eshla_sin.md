# Оригинал

### Дамир Гыйсметдин / Габдразак Миңгалиев - Эшлә син!..

```
«Гомер узган», — дип уфтанма,
Узмагандыр әле ул.
«Йөрәккәем», — дип сыкранма,
Тузмагандыр әле ул.

Күпме узган, күпме тузган —
Иблис аны санасын.
Көннәр буе шыңшып йөрсәң —
Бөкерәеп катарсың.

«Тормыш авыр, заман кырыс,
Акча юк», — дип йөрмә син.
«Баҗай яши белә», — диеп,
Кара янып йөрмә син.

Күпме узган, күпме тузган —
Иблис аны санасын.
Көннәр буе шыңшып йөрсәң —
Бөкерәеп катарсың.

Уфтанмыйча, шыңшымыйча,
Көнен-төнен эшлә син,
Бөтен җирең авыртса да,
Айгыр кебек кешнә син!

Күпме узган, күпме тузган —
Иблис аны санасын.
Көннәр буе шыңшып йөрсәң —
Бөкерәеп катарсың.
```
