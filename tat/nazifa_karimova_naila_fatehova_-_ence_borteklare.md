# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Энҗе бөртекләре

```
Әй, балачаклар, энҗе табар чаклар,
Якты дөньяга без дә чыгыйк әле,
Халык күңеленнән барлап табарбыз,
Еллар җуймаган энҗе бөртекләрен!

Халкым җырлары, сандугач моңы
Йөрәкләрне бер җилкетер әле
“Сарман”, Карурман”, “Җидегән чишмәләрдә”
Энҗе бөртекләре, энҗе бөртекләре!

Тукайда кебек, безнең күңелдә дә,
“Туган тел” диеп җырлар туар әле,
Саклыйк кадерләп, ана телебезне
Энҗе бөртекләрен, энҗе бөртекләрен!
```
