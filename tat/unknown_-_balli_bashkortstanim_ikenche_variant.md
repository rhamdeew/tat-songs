# Оригинал

### Дилә Булгакова / Нур Даутов - Баллы Башкортстаным (икенче вариант)

```
Башкортстаным кунакчыл як,
Ул безнең изге кала.
Баллап кына чәйләр эчсәң,
Тәнеңә шифа була.

Яшәртә дә, давалый да
Гәрәбәләй балабыз.
Юкә балы, чәчәк балы
Күчтәнәчкә алыгыз.

Кәрәзле бал — уңган төяк,
Данлы төяк — даныбыз.
Бүләкләргә лаек булды
Башкортстан балыбыз.

Медальларга лаек булды
Башкортстан балыбыз.
Кәрәзле бал — уңган төяк,
Таралган бит даныбыз.
```
