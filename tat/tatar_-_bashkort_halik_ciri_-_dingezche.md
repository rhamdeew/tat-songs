# Оригинал

### Татар - Башкорт халык җыры - Диңгезче

```
Сәгать суга сигезе,
Чыгып киттек диңгезне.
Якты йолдызлар сүнгәнчә
Уйлап ятам мин сезне.

Урман буйларын чабыгыз,
Үткен булса чалгыгыз.
Диңгезләрдә дулкын ничек,
Безнең шундый чагыбыз.

Идел буйларына барсам,
Әйтер идем Иделгә.
Минем кайгыларымны да
Илтеп сал дип, диңгезгә.

Кара диңгез, ак диңгез,
Бүген төнлә китәрбез.
Иҫән булсак, бер кайтырбыз,
Өмет өзми көтегез.
```
