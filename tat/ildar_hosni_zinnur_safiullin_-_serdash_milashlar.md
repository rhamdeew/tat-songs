# Оригинал

### Илдар Хөсни / Зиннур Сафиуллин - Сердәш миләшләр

```
«Керфекләрең яшьле», — димә, дустым,
Юк, елаудан түгел ул яшьләр.
Койды бары яңгыр тамчыларын
Мин кагылып узган миләшләр.

Алар минем сердәш миләшләрем,
Шуңа алар җанга кадерле.
Ничәнче ел менә сыйпап узам,
Тыеп буламыни хәтерне!

Борыла да кайта миләшләргә
Сукмаклары гомер-җилкәннең.
Керфегемә кунган яшь-тамчылар
Сагыну хәбәредер иркәмнең.
```
