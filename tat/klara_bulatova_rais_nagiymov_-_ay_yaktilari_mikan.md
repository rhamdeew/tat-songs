# Оригинал

### Клара Булатова / Рәис Нәгыймов - Ай яктылары микән?

```
Ай яктылары микән,
Кар яктылары микән.
Әллә сызылып кына аткан
Таң яктылары микән.

Йоклап ятасың микән,
Уйлап ятасың микән.
Әллә илне сагынудан
Елап ятасың микән.

Җилләр йөгерә микән,
Хәбәр белдерә микән.
Кайтыр юлың чишмә булып
Төшеңә керә микән.

Йоклап ятасың микән,
Уйлап ятасың микән.
Әллә сагынуларың тартып
Илгә кайтасың микән.
```
