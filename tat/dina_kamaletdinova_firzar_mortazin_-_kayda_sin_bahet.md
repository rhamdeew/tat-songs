# Оригинал

### Динә Камалетдинова / Фирзәр Мортазин - Кайда син бәхет?

```
Кайда соң, син, кайда,
Сөйлим микән айга,
Серемне юксынган чагымда?
Ай мине аңламас,
Сердәшем булалмас,
Аның бит йолдызы янында.

Син миннән качтыңмы,
Күкләргә очтыңмы,
Киек каз юллары буйлата!?
Урамда ап-ак кыш,
Күңелдә моң-сагыш,
Гел синең турыда уйлата, бәхетем.
Гел синең турыда уйлата.

Давыллы буранда
Чыгам да урамга,
Юксынып карыйм ак юлларга.
Бик көтәм бит сине,
Кайтырсың шикелле
Буранга урала, урала, бәхетем,
Буранга урала, урала…
```
