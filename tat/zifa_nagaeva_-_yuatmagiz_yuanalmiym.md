# Оригинал

### Зифа Нагаева - Юатмагыз, юаналмыйм

```
Юатмагыз, югалттым дип,
Елатасыз гына шул.
Әрнү катыш булганда да,
Рәхәтем дә, назларым да,
Бәхетем дә булган ул.

Юатмагыз, юаналмыйм,
Юк шул юанычларым.
Аның белән киткән инде
Онытылмас, кабатланмас,
Кайтмас куанычларым.

Юатмагыз, төннәр чиксез,
Көннәр болытлы инде.
Уа бар чакта төн йолдызлы,
Көн кояшлы, таңнар нурлы,
Тормышым җырлы иде.

Юатмагыз, юаналмыйм,
Юк шул юанычларым.
Аның белән киткән инде
Онытылмас, кабатланмас,
Кайтмас куанычларым.

Юатмагыз, сагышымны
Тирәнәйтәсез генә.
Уа барында көз яз иде,
Син бар идең, мин бар идем,
Хәзер тоташ көз генә.

Юатмагыз, юкка гына
Мин бит сине сагынам.
Мин сине дә түгел, бәлки,
Янып йөргән чакларымны,
Мин үземне сагынам.
```
