# Оригинал

### Рамил Закиров / Айнур BoooRe Шарапов - Без бергә идек

```
Хәзер мин, беләсең, төшеңдә,
Күрәсең, кар кебек очабыз.
Яз җиткәч, качарбыз,
Син мине беләсең бит, кире кайтмам,
Сине бүтән кочакламам,
Син бит миннән үзең киттең,
Ул күбрәк бирер дидең.

Искә алып безнең йөргәнне, син елыйсың,
Сине өйгә озаткан юлга тәрәзә карыйсың,
Хәтеремдә: бергә йөрдек, кочаклаштык һәм баллы үбештек,
Һәм без бергә, бергә, бергә идек.

Башта курчак кызларга син ошамаган идең,
Ләкин тик син генә минем башымны әйләндердең,
Син белә идең мин синсез яши алмавымны,
Башка белән йөрдең, белә торып сиңа каравымны.
Син мине яратмадың, минем белән уйнадың,
Башка кеше сиңа күбрәк бирер дип ни уйладың?
Ә син киткәч, мин, әйтерсең, җәһәннәмгә эләктем,
Шул чак, белсәң иде син, мин күпме көч түктем.

Искә алып безнең йөргәнне, син елыйсың,
Сине өйгә озаткан юлга тәрәзә карыйсың,
Хәтеремдә: бергә йөрдек, кочаклаштык һәм баллы үбештек,
Һәм без бергә, бергә, бергә идек.
```
