# Оригинал

### Гөлшат Зәйнәшева / Ганс Сәйфуллин - Син икәнсең (беренче вариант)

```
Юллар уздым, илләр гиздем,
Юкка гына гизмәдем.
Юллар узып, илләр гизеп,
Мин бит сине эзләдем.

Син икәнсең, син икәнсең
Гомерем буе эзләгән.
Бу дөньяның гүзәллеген
Күрдем синең күзләрдән.

Юллар уздым, илләр гиздем, —
Кайда гына йөрмәдем.
Сине эзләп йолдызларга,
Айга гына менмәдем.

Син икәнсең, син икәнсең
Гомерем буе эзләгән.
Бу дөньяның гүзәллеген
Күрдем синең күзләрдән.

Юллар уздым, илләр гиздем,
Нинди генә илләр юк!
Үз илемдә таптым сине,
Үз илемә тиңнәр юк!

Син икәнсең, син икәнсең
Гомерем буе эзләгән.
Бу дөньяның гүзәллеген
Күрдем синең күзләрдән.
```
