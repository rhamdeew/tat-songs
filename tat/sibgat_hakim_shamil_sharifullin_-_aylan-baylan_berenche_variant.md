# Оригинал

### Сибгат Хәким / Шамил Шәрифуллин - Әйлән-бәйлән (беренче вариант)

```
Әйләнәбез, тулганабыз,
Әйтерсең лә тулган ай.
Кил, тотын кулларга безнең,
Читтән карап торма, әй!
Дус, кил якын безгә,
Дуслашырбыз гомергә,
Бир кулың, булыйк түгәрәк,
Кил, җан дусларга тотын.
Әй, яшьлек, мәңгелек яшьлек,
Зур язмышларга тотын.

Кил, уйна әйлән-бәйлән син,
Кул-беләкләргә тотын.
Әй, яшьлек — якты, бөек ул,
Зур теләкләргә тотын.
Әй, дуслашырбыз гомергә,
Әйләник түгәрәк булып,
Очыйк күбәләк булып.
Калсын җир тыныч, бәхетле,
Мәңге түгәрәк булып.
```
