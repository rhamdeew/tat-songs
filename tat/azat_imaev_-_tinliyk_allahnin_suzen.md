# Оригинал

### Азат Имаев - Тыңлыйк Аллаһның сүзен

```
Бу дөньяда барчабыз да,
Без адәм балалары.
Гасырлар аша күрәбез,
Кан, сугыш, яраларны.

Аллаһның мәрхәмәте зур,
Кайтуыбыз тик аңа.
Гөнаһлардан ярлыка дип,
Тәкъдир әйтик, иншаллаһ.

Эй, адәмнең балалары.
Тыңлыйк Аллаһның сүзен.
Гөнаһлардан сакланыйк без,
Залимнар булмыйк бүген.

Җирдә сугышлар булмасын
Я, Ходай, үзең сакла.
Шайтаннарның тоткасыннан,
Мөселманнарны якла.
```
