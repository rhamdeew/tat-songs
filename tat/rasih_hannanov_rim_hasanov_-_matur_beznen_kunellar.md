# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Матур безнең күңелләр

```
Тормыш, тормыш. Әкият дөнъясы син,
Хыял белән тула күңелләр.
Әллә яшьлек матур, әллә гөлләр матур,
Әллә матур безнең күңелләр.

Тормыш, тормыш. Илһам дәръясы син,
Туа җырлар, уңа (үсә) игеннәр.
Әллә еллар матур, әллә җырлар матур,
Әллә матур безнең күңелләр.

Тормыш, тормыш. Яшьлек чишмәсе син,
Тыныч җирне назлый саф җилләр.
Әллә җирләр матур, әллә җилләр матур,
Әллә матур безнең күңелләр.
```
