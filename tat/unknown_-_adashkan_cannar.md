# Оригинал

### Нурия Абдуллина - Адашкан җаннар

```
Күзләрне күзләргә төбәп,
Йөрәкне йөрәккә түгеп,
Агыйделгә салдым уйларны.
Алсу таңнарны аттырып,
Таңсыкларыбыз канганчы
Сөйләшәсе иде тагы бер.

Сөюдән туймаган яшьлектә
Әйтеп бетәлмәгән сүзләр бар.
Иренне иренгә терәп,
Вәгъдәләр бирелмәгән,
Җан сөюе белән яшәлгән.

Кызларның оялчанлыгын,
Егетләрнең чаялыгын
Җырга гына салып җырларлык.
Һәр беребезнең бар үз яры,
Язмыш бүләк иткән пары,
Ә шулай да йөрәк януы.

Мәшәкатләреннән арынып,
Җир тормышыннан бер аерылып,
Җиде күк катламнарында
Адашкан җаннар табышып,
Мәңгелеккә бергә калырлар.
```
