# Оригинал

### Резедә Гобәева / Эльмир Низамов - Татлы газап

```
Сөюемне җиткерердәй сүзләр булса,
Синең янда тик бер генә таңым туса,
Мин дөньядан ваз кичәргә әзер шуңа,
Билгесезлек газабыннан җаным туңа..

Мәхәббәтем татлы газап,
Шул газаптан итче азат!
Югалыймчы күзләреңдә,
Тик син генә сүзләремдә!

Синең белән әзермен мин барсына да,
Синсезлектән йөрәк җаннан тарсын ала.
Күңелеңә ничек юллар табыйм икән,
Нигә сөю шулай җанга кадый икән!?

Бар гомерем сине көтеп үтәр микән,
Кавышуга тик бер гомер җитәр микән!?
Синең өчен бу дөньяның бар гөлләре,
Синең белән бәйле җанның бар серләре…
```
