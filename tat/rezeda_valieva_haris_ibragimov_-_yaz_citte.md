# Оригинал

### Резеда Вәлиева / Харис Ибрагимов - Яз җитте

```
Җылы җил сирпеп үтте,
Агачларны селкетте:
— Уяныгыз, уян тизрәк,
— Яз җитте,
— Әй, яз җитте!

Кояш нурларын сипте,
Тәрәз битләрен үпте:
— Ачылыгыз, ачыл тизрәк,
Яз җитте,
Әй, яз җитте!

Сыерчык сайрап утте,
Һавалар зыңлап китте:
— Чыгыгыз, чык, әй, балалар,
Яз җитте,
Әй, яз җитте!

Яңгыр сибәләп утте,
Кырлар яшереп китте.
Нурга тулды туган җирләр,
Яз җитте,
Әй, яз җитте!
```
