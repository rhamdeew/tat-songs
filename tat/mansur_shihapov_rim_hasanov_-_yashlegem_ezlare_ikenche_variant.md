# Оригинал

### Мансур Шиһапов / Рим Хәсәнов - Яшьлегем эзләре (икенче вариант)

```
Яр буйлап шаулаган каеннар
Яңарта үткәннең хисләрен.
Бөреләр чәчәккә күмелгәч
Сагынам авылның кичләрен,
Йөрәктә ялкыннар кабынып
Танышкан яшьлекнең эзләрен.

Кич тулып, йолдызлар балкуга,
Югалтҡан бәхетне эзләдем.
Бары тик хәтергә төшердең
Чылтырап агышын чишмәнең.
Таңында шатлыклар тудырып
Кавышкан яшьлекнең эзләрен.

Бирелеп уйларга, хыялга,
Тыңлыймын агышын чишмәнең.
Чишмәләр җырлыйлар җыр итеп,
Җырлыйлар яшьлегем хисләрен.
Мәңгегә сакласын авылым
Давыллы яшьлегем эзләрен.
```
