# Оригинал

### Фәрдия Сәфәргалиева / Ильяс Фәйзрахманов - Син (икенче вариант)

```
Сөйкемле син, кадерле син,
Тапкыр син һәм серле син.
Кайчак көләч, кайчак моңсу,
Кайчак уйчан — төрле син.

Бик җитез син, шаян да син,
Кызыклы син, кыю син,
Сокланырлык бай рухыңа
Гашыйк иттең мине син…

Төрле, төрле, төрле дә син,
Өстәвенә — сабыр син.
Акыллы да син, көчле дә,
Йөрәгеңә бармы тиң?
```
