# Оригинал

### Мөхәммәт Садри / Әнвәр Бакиров - Бишек җыры (бишенче вариант)

```
Әлли-бәлли, ак бәби,
Ак бишектә ят, бәби.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Әлли-бәлли ит, бәби,
Тәмле йоклап кит, бәби.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Әлли-бәлли, бал гына,
Йокы сиңа ял гына.
Әлли-бәлли, бәбием,
Балдан татлы нәнием.

Йокла, йокла, багалмам,
Үскәч бишеккә салмам.
Әлли-бәлли, бәбием,
Балдан татлы нәнием!
```
