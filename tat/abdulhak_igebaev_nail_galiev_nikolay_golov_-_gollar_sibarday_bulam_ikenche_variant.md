# Оригинал

### Абдулхак Игебаев / Наил Галиев (Николай Голов) - Гөлләр сибәрдәй булам (икенче вариант)

```
Бер чак йокыңнан иркәләп
Уятыр төсле булам,
Бер чак сине тыным белән
Тартып алырдай булам.

Бер чак аккошлар шикелле,
Парлап йөзердәй булам,
Берчә йөрегән юлларыңа
Гөлләр сибердәй булам.

Бер чак җәйгор булып кына
Йөгереп барырдай булам,
Берчә сине тыным белән
Тартып алырдай булам.
```
