# Оригинал

### Кави Нәҗми / Рөстәм Яхин - Лирик табын җыры

```
Бөек бәйрәм таңында
Яндыр, күгем, тагын да
Яктырак йолдызыңны!
Чакыр, илем, табынга
Батырлыкта танылган
Улыңны һәм кызыңны,
Улыңны һәм кызыңны!

Ирек җырын ишетсен
Иң газаплы, иң читен
Юлны кичкән туганнар.
Күтәрелсен берьюлы
Мәхаббәт белән тулы,
Ялкынлы тустаганнар,
Ялкынлы тустаганнар!

Көйдереп дошманнарны,
Сөендереп дусларны,
Туган ил даны өчен,
Туып-үскән йорт өчен,
Ирек һәм бәхет өчен
Күтәрәбез, дусларым,
Бөек бәйрәм тостларын!
```
