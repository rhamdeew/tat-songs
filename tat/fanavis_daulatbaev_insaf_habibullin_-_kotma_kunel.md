# Оригинал

### Фәнәвис Дәүләтбаев / Инсаф Хәбибуллин - Көтмә, күңел

```
Әйтегезче: кемнәр кискән
Сандугач сайрар талны?
Әйтегезче: нишлим, дуслар,
Югалткачтын сөйгән ярны.

Юк сандугач сайрамас шул,
Сайрамас шул башка талда.
Юк шул инде әйтә алмам
Сөеклем дип башка ярга.

Күген ташлап бер атылгач
Сүнгән йолдыз кабынмас.
Ник басмасын сагыш җанны
Мәхәббәттә ялгыз калгач.

Юк сандугач сайрамас шул,
Сайрамас шул башка талда.
Юк шул инде әйтә алмам
Сөеклем дип башка ярга.

Көтмә күңел, өмет итмә:
Ятларда шул сөйгән ярың.
Кызыгып гомер үтәр микән
Бәхетенә парлыларның?

Юк сандугач сайрамас шул,
Сайрамас шул башка талда.
Юк шул инде әйтә алмам
Сөеклем дип башка ярга.
```
