# Оригинал

### Фәнәвис Дәүләтбаев / Эдуард Хөснетдинов - Киләм, иркәм

```
Ошатамын сине, иркәм,
Чәчәк бәйләменә.
Синең аша хыялымда
Кайтам жәйләремә.

Киләм сиңа, киләм, иркәм,
Каршы алырсыңмы?
Мәхаббәтнең утларында
Син дә янырсыңмы?

Эрет әле, эрет мине
Кайнар кочагыңда.
Мәхаббәт ул татлы була
Бары шул чагында.

Киләм сиңа, киләм, иркәм,
Каршы алырсыңмы?
Мәхаббәтнең утларында
Син дә янырсыңмы?

Күңелдәге хисләреңне
Белдер авырсынмый.
Киләм сиңа, киләм, иркәм,
Каршы алырсыңмы?

Киләм сиңа, киләм иркәм,
Каршы алырсыңмы?
Мәхаббәтнең утларында
Син дә янырсыңмы?
```
