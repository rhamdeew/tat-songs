# Оригинал

### Вәзыйх Фатыйхов / Расил Әхәтов - Яңа еллар бәхет китерсен

```
Якты уйлар белән бу кичтә
Яңа елга атлап керәбез.
Чынбарлыкка ашсын хыяллар,
Чын күңелдән теләп теләгез.

Яңа еллар бәхет китерсен
Һәр йортка, һәр йөрәк, һәр җанга.
Сабыйлар шатлыгы, мәхәббәт сафлыгы
Иркәләп уятсын һәр таңда.

Яхшылык, татулык, дуслыкта,
Өметләрдә яныйк һәрвакыт.
Яшик бергә якты уй белән,
Яшик тормышларны яратып.

Яңа еллар бәхет китерсен
Һәр йортка, һәр йөрәк, һәр җанга.
Сабыйлар шатлыгы, мәхәббәт сафлыгы
Иркәләп уятсын һәр таңда.

Мизгел генә булган гомернең
Кадерләрен белик һәр көнне.
Үз шатлыгы, үз бәхете булсын
Бу дөньяда туган һәр көннең.

Яңа еллар бәхет китерсен
Һәр йортка, һәр йөрәк, һәр җанга.
Сабыйлар шатлыгы, мәхәббәт сафлыгы
Иркәләп уятсын һәр таңда.
Сабыйлар шатлыгы, мәхәббәт сафлыгы
Иркәләп уятсын һәр таңда…
Яңа еллар бәхет китерсен…
```
