# Оригинал

### Сәкинә Хәйруллина / Фирдүс Тямаев - Ялгыз торна

```
Төшләремә кер әле бер,
Хәлләремне бел әле бер.
Синең җылы кочагыңа мохтаҗ чагым,
Өзгәләнә бәргәләнә ятим җаным.

Кайнар утта янам,
Бозлы суда калам,
Нихәл итим икән,
ничек үтим икән,
тормыш дулкыннарын…
Мин бит ялгыз торна,
Зинһар адаштырма,
Нихәл итим икән,
ничек үтим икән,
тормыш упкыннарын.

Була икән, шундый мәлләр,
Шәмдәлләрдә сүнә шәмнәр.
Ургый кайный хәсрәтемнең, дулкыннары,
Томырылып көтеп торган упкыннары.

Кайнар утта янам,
Бозлы суда калам,
Нихәл итим икән,
ничек үтим икән,
тормыш дулкыннарын…
Мин бит ялгыз торна,
Зинһар адаштырма,
Нихәл итим икән,
ничек үтим икән,
тормыш упкыннарын.

Таулар ташлар ишелер күк,
Барыр юлым киселер күк.
Бәхет диеп алданганмын, бары рәшә,
Коткаручым бул әле дип сиңа дәшәм.

Кайнар утта янам,
Бозлы суда калам,
Нихәл итим икән,
ничек үтим икән,
тормыш дулкыннарын…
Мин бит ялгыз торна,
Зинһар адаштырма,
Нихәл итим икән,
ничек үтим икән,
тормыш упкыннарын.
```
