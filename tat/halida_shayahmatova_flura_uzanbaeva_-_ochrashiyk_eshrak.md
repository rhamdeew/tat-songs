# Оригинал

### Халидә Шәяхмәтова / Флүрә Үзәнбаева - Очрашыйк ешрак

```
Бу гомерләр аккан судай,
Тиз генә үтеп китә.
Күрешүләр, сөйләшүләр
Безне җан дулсар итә.

Очрашик без, очрашик без,
Очрашик без ешырак.
Сагынулар, саргаюлар
Торсыннар бездән ерак.

Су буенда сандугачлар
Серләшә талга качып.
Бер күрешү — бер гомер бит
Шунысы көн кебек ачык.

Очрашик без, очрашик без,
Очрашик без ешырак.
Сагынулар, саргаюлар
Торсыннар бездән ерак.

Кайгы кими уртаклашсаң,
Шатлык арта тагын да.
Күрешик без, гөрләшик без
Исәнлек бар чагында.

Очрашик без, очрашик без,
Очрашик без ешырак.
Сагынулар, саргаюлар
Торсыннар бездән ерак.
```
