# Оригинал

### Гөлшат Зәйнәшева / Мәсгудә Шәмсетдинова - Сөмбелә

```
Сөмбелә дип матур исем
Кушты икән кем генә?
Сөмбел гөлдән өзлеп төшкән
Чәчәк кебек Сөмбелә.

Сөмбелә, Сөмбелә,
Сөмбел гөлдән өзлеп төшкән
Чәчәктер ул, кем белә?

Әткәсенә, әнкәсенә
Ул күз карасы кебек.
Гөл арасында сайраган
Былбыл баласы кебек.

Сөмбелә, Сөмбелә,
Гөл арасында сайраган
Былбылдыр ул, кем белә?

Чәчәк кебек сөйкемле ул,
Былбыл кебек моңлы ул.
Таң аткач та сүнми торган
Йолдыз кебек нурлы ул.

Сөмбелә, Сөмбелә,
Таң аткач та сүнми торган
Йолдыздыр ул, кем белә?
```
