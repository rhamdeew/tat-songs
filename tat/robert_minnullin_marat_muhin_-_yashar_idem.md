# Оригинал

### Роберт Миңнуллин / Марат Мухин - Яшәр идем

```
Яшәр идем мин бу җирдә
Былбыл булып кына,
Сайрар идем көйләремне
Талга кунып кына…

Ә мин һаман яшим әле
Кеше булып кына…
Кош була алырмын, ахры,
Кабат туып кына.

Очар идем мин биектә
Бөркет булып кына,
Күзәтер идем җиремне
Өстән торып кына…

Ә мин һаман яшим әле
Кеше булып кына…
Кош була алырмын, ахры,
Кабат туып кына.

Кайтыр идем туган җиргә
Кырказ булып кына,
Гомер итәр идем илдә
Оя корып кына…

Ә мин һаман яшим әле
Кеше булып кына…
Кош була алырмын, ахры,
Кабат туып кына.
```
