# Оригинал

### Татар халык җыры - Казан көе

```
Казаннарга барсаң, дустым,
Казан суларын буйла;
Җилсез көнне тал селкенсә,
Мин сагына дип уйла.

Без урамнардан узабыз
Җырлый-җырлый сыздырып;
Йокың килсә, йокла, җаный,
Уятырмын сызгырып.

Аккошлар оча һавада
Ерактагы далада;
Җаным сөйгән сандугачым
Көтә булыр калада.
```
