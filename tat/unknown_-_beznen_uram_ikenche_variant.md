# Оригинал

### Татар - Башкорт халык җыры - Безнең урам (икенче вариант)

```
Безнең урам — матур урам,
Тезелеп киткән ак каен.
Шул каеннарга караем
Сезне сагынган саен.

Аргы очтан кайтып килә
Чөгендер утаучылар.
Белсәң иде, кемнәр икән
Араны бутаучылар?!

Биек тауга менгән чакта,
Таянырсың миләшкә.
Таянырсың да еларсың,
Мин булмамын киңәшкә.

Өстәлләрдә сары лимон
Саргаймый дисезме әллә.
Уйнап-көлеп йөрегәч тә
Сагынмый дисезме әллә.
```
