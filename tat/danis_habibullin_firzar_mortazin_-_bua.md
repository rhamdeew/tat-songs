# Оригинал

### Данис Хәбибуллин / Фирзәр Мортазин - Буа

```
Синдә үтте гомеремнең
Матур чаклары.
Шуңа истән чыкмый бер дә
Буа яклары.

Кайларга гына китсәм дә,
Сине сагынам.
Аермасын иде язмыш
Туган ягымнан.

Аз түгел синдәге кебек
Юмарт басулар.
Шундый ук яшел урманнар,
Шундый ук сулар.

Кайларга гына китсәм дә,
Сине сагынам.
Аермасын иде язмыш
Туган ягымнан.

Яшьләрең эштә ярсулар,
Тыйнак картларың.
Онытырга мөмкинме соң
Буа якларын?

Кайларга гына китсәм дә,
Сине сагынам.
Аермасын иде язмыш
Туган ягымнан.
```
