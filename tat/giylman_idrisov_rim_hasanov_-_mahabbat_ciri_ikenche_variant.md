# Оригинал

### Гыйльман Идрисов / Рим Хәсәнов - Мәхәббәт җыры (икенче вариант)

```
Айның көмеш нуры
Җиргә сибелгәндә,
Чәчләреннән сөеп,
Җилләр үпкәндә,
Хисләр таша микән
Янып сөйгән дустың
Тәүге умырзая бүләк иткәндә?

Җандай күргән ярың
Карап күзләреңә:
«Сөям, — дисә, — сине,
Бәгырем, өзелеп».
Талпынамы икән
Күңел акчарлактай,
Мәхәббәтең дулкынына күмелеп?

Сәлам белән бергә
Хатлар китерсәләр,
Үтеп киткән язның
Кипкән гөлләрен,
Һәр тибеше саен
Йөрәк көтә микән
Тиздән килеп җитәр
Кавышу көннәрен?
```
