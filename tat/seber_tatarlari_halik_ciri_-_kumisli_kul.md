# Оригинал

### Себер татарлары халык җыры - Кумыслы күл

```
Ҡумыслы күл пуйларынта
Үтте минең йәшлек йылларым.
Йасмыш антай йаман нимә игән, —
Ташлап киттем күлнең пуйларын.

Ҡумыслы күлнең төбөнтә
Ләмтек түгел игән — ҡайыр ҡом.
Атларыма тышау салған итем,
Пашҡа күрмәсмен тип ҡайғыртым.

Ҡаратыр минең, ай, цацым.
Күпне күрте инте пы пашым.
Уйсыу йертә тәрйә пулып аҡты
Минең түккән ҡайнар күс йәшем.

Ике ләй күгәл ашаттым.
Ашаттым та күлгә пушаттым.
Күгәлкәйем китте күлне пуйлап….
Өмөрләрем үтте сесне уйлап.
```
