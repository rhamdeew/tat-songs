# Оригинал

### Юныс Миңнемөхәммәтов / Шамил Шәрипов - Болгарым, минем Ватаным

```
Болгар буйларында җәйрәп яткан
Киң далада үсә кылганнар.
Дошманнардан саклап газиз илен
Бабаларым һәлак булганнар.

Иделемнең акчарлагы елый,
Болгарымны, беләм, сагына.
Акчарлаклар булып кайткан ахры
Ата-бабам туган ягына.

Офыкларда уяна да кояш,
Шәһри Болгарымда таң ата.
Ак мәчеттә яңгрый азан моңы,
Иман нурын илгә тарата.

Туган ягым, ата-бабам җире,
Җан түрендә бит син,Болгарым.
Гасырларга шаулар бөек даның,
Яшәгәндә кыз һәм улларың!
```
