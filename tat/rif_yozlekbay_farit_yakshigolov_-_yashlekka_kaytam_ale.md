# Оригинал

### Риф Йөзлекбай / Фәрит Якшыголов - Яшьлеккә кайтам әле

```
Кыңгыраулы атлар җигеп
Үттек яшьлек елларын.
Сөелдек тә, сөйдек тә без
Безнең заман кызларын.

Йөгән элеп, атка менеп,
Яшлеккә кайтам әле!
Йөрәгемдә син һаман да —
Сөям яратам әле,
Яшлеккә кайтам әле!

Күкрәк киреп, гармун тартып
Кыңгыраулы атларда.
Кызлар урлап төн уздырды
Таралышкан таңнарда.

Йөгән элеп, атка менеп,
Яшлеккә кайтам әле!
Йөрәгемдә син һаман да —
Сөям яратам әле,
Яшлеккә кайтам әле!

Чанадагы пар атларда,
Ап-ак карга эз салып.
Үткән яшлек елларымдан
Узамын искә алып.

Йөгән элеп, атка менеп,
Яшлеккә кайтам әле!
Йөрәгемдә син һаман да —
Сөям яратам әле,
Яшлеккә кайтам әле!
```
