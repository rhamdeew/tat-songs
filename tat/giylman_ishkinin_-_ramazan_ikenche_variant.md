# Оригинал

### Гыйльман Ишкинин - Рамазан (икенче вариант)

```
Бүген мәчет тупсасыннан
Мин оҗмахҡа керәм.
Әйтерсең дә, язмышымда
Шулай кушкан (язган) Көрьән.

Эй, мөселман дөньясы,
Көмеш тавышлы азан.
Эй, мөселман дөньясы,
Хуш, хәерле Рамазан!

Туган җирем туфрагыннан
Бер уч алып хаҗга.
Чыксам иде — мә иминлек,
Сәфәр бирсен Ходай!

Бирсен Ходай халыкларга
Саулык һәм мәрхәмәт.
Фәкир хәлен уртаклашкан
Пәйгамбәр Мөхәммәт.

Эй, мөселман дөньясы,
Көмеш тавышлы азан.
Эй, мөселман дөньясы,
Хуш, хәерле Рамазан!
```
