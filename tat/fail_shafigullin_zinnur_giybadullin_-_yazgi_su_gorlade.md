# Оригинал

### Фаил Шәфигуллин / Зиннур Гыйбадуллин - Язгы су гөрләде

```
Әйтмәде сүзләрең, әйттеләр күзләрең
Беренче хисләрең дөрләвен.
Сереңне әйтмәдең, әйтмәдең сереңне,
Язгы су гөрләде, гөрләде язгы су.

Сүзсез дә аңладык, аңладык ул чакта,
Яшьлекнең беренче серләрен.
Ничәнче мәртәбә җыр җырлап шул хакта
Язгы су гөрләде, гөрләде язгы су.

Ут ягып бозларга, озаттык без ярдан,
Уйламый киләчәк көннәрне…
Боз акты еракка, ут калды йөрәктә,
Язгы су гөрләде, гөрләде язгы су.
```
