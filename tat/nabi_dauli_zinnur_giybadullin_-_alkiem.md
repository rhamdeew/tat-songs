# Оригинал

### Нәби Дәүли / Зиннур Гыйбадуллин - Әлкием

```
Әлкием, әллием, бәллием,
Син мине таң белән уяттың.
Язмышым бу җирдә башланды,
Син мине дөньяга озаттың.

Әлкием, әллием, бәллием, бәллием,
Син минем балачак бишегем, бишегем.
Күңелдә мәңгелек исле гөл,
Йөрәктә яши лә исемең.

Әнкәем нан биргән кулыма,
Су биргән Чирмешән елгасы.
Яктырак, җылырак шикелле
Әлкинең үзенең кояшы.

Әлкием, әллием, бәллием, бәллием,
Син минем балачак бишегем, бишегем.
Күңелдә мәңгелек исле гөл,
Йөрәктә яши лә исемең.

Китсәм дә мин тагын каядыр,
Әлкине сагынып җырлармын.
Гафу ит, туган як, улыңны,
Яңадан кайтмаса юлларым.

Әлкием, әллием, бәллием, бәллием,
Син минем балачак бишегем, бишегем.
Күңелдә мәңгелек исле гөл,
Йөрәктә яши лә исемең.
```
