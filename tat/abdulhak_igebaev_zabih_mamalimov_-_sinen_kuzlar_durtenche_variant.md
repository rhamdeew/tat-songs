# Оригинал

### Абдулхак Игебаев / Зәбих Мәмәлимов - Синең күзләр (дүртенче вариант)

```
Тынгы белми торган юлчы сымак,
Мин гизсәм дә бик күп җирләрне,
Тормыш юлларымда очратмадым
Синең күздәй серле күзләрне.

Ничә җәйне көзләр алмаштырды,
Кышлар алмаштырды көзләрне,
Тик һаман да оныта алмыймын шул
Синең көләч, шаян күзләрне.

Йөрәгемә ак нур белән яздым
Үзеңә генә әйтер сүзләрне,
Син сизмәссең… Ә мин сагынырмын
Синең мәхәббәтле күзләрне
```
