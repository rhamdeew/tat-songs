# Оригинал

### Сәгыйть Әхмәтҗанов / Ганс Сәйфуллин - Баулым

```
Баулым таулы-таулы,
Тауларында шаулый
Җил-давылдан саклап, әй, урман!
Шунда бабам туган,
Буразналар ерган,
Мең хәсрәтен чәчеп моң урган.

Хисләр ташкынына
Салып башкаеңны
Баулы тавын йөгереп менсәнә!
Йөрәккәең ярсыр,
Җирең үзенә тартыр
Күпме генә биек менсәң дә.

Сагышларым алып,
Җырларыма салып,
Ерак-еракларга юл алсам,
Офыкларда томан,
Адашырмын сыман,
Тавыш бирче, Баулым, югалсам…
```
