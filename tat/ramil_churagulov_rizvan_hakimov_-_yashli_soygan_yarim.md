# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Яшьли сөйгән ярым

```
Яшьли сөйгән ярым,
Еракларда калдың.
Еракларда, яшьлек таңында.

Төшләремә кер син,
Өннәремдә кил син,
Якты кояшым бул янымда.

Яшьли сөйгән ярым,
Йөрәгемдә калдың.
Аерса да ачы язмышлар.

Үткәнгә уфтанмыйк,
Язмышка сылтанмыйк,
Язмышлардан табыйк узмышлар.

Газаплама көн дә,
Кереп төшләремә.
Синсез яшәү үзе зур газап.

Кайтып килик дустым
Яшьлек таңнарына.
Җитәкләшеп тагын бер кабат.
```
