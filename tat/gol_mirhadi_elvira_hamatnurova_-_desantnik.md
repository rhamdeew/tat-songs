# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Десантник

```
Мин үскәч, десантник булам,
Әзерләнәм бүгеннән.
Кул булгармын әле сезгә
Туган илем күгеннән.

Биекләрдән сикерергә
Мин курыкмаска тиеш
Нәрсәсе бар: очто, төштең
Парашют кигән килеш.

Өйрәнергә, чыныгырга
Мин хәзер үк ашыгам.
Мендем әле сикерергә
Шифонерның башынан.

Биекләрдән сикерергә
Мин курыкмаска тиеш
Нәрсәсе бар: очто, төштең
Парашют кигән килеш.

Кирәк укчага басырга
Очоп төшеп иденгә.
Парашютсыз да сикерәм,
Әткәй куйган мендәргә.

Биекләрдән сикерергә
Мин курыкмаска тиеш
Нәрсәсе бар: очто, төштең
Парашют кигән килеш.
```
