# Оригинал

### Лилия Галимова / Гайфулла Абзалов - Гүзәл-Гүзәлия

```
Күзем төште сылу-чибөргө
Менә кайгы — инде нишләргә?!
Айлы кичләрен дә,
Ямьле көннәрен дө
Сине уйлыйм, Гүзөл-Гүзөлем.

Әй! Гүзәл-Гүзәлия,
Аңламыйсың син нигә?
Әй! Гүзәл-Гүзәлия,
Яратмыйсың син нигә?

Сөю утында мин, Гүзәлем,
Сине генә көткән йөрәгем,
Бик ялкынлы икән,
Янып китмәс микән,
Сине уйлыйм, Гүзөл-Гүзөлем.

Әй! Гүзәл-Гүзәлия,
Аңламыйсың син нигә?
Әй! Гүзәл-Гүзәлия,
Яратмыйсың син нигә?
```
