# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Инеш

```
Төштем инеш буена,
Суда таллар коена.
Син дә бүген тал янына
Килерсең күк тоела.

Аһ, инеш, инеш, инеш,
Хәлемне минем белеш.
Син диңгезең кочагында,
Мин талдай ялгыз килеш.

Моңлы инеш агышы,
Талда сөю сагышы.
Таллы инеш буйларында
Күпме ярлар кавышты.

Аһ, инеш, инеш, инеш,
Хәлемне минем белеш.
Син диңгезең кочагында,
Мин талдай ялгыз килеш.

Инеш буе тал гына,
Кочагыңа ал гына…
Таллар кала, инеш ага,
Ага диңгез ягына.

Аһ, инеш, инеш, инеш,
Хәлемне минем белеш.
Син диңгезең кочагында,
Мин талдай ялгыз килеш.
```
