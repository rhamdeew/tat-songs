# Оригинал

### Рәшит Гәрәй / Фәрит Хатыйпов - Сез парлашып йөргән болыннар

```
Болыннарда күпме гөлләр тибри,
Сокланмассың шуңа ничек син?
Болыннарга чыгып сызгыр әле,
Болындагы гөлләр ишетсен.

Болын гөлләредәй бер матурны,
Сайлап ал син аннан үзеңә.
Ул елмайган чакта, якты кояш
Көлеп торсын синең йөзеңдә.

Яшьлек бит ул язлар кебек килә,
Матур итеп сөй һәм сөел син.
Сез парлашып йөргән болыннарда
Сезне күреп гөлләр сөенсен.
```
