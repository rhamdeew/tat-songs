# Оригинал

### Румия Әфләтүнова / Юлай Үзәнбаев - Бию

```
Тыпырдатып бер басмагач,
Бию була мени ул?
Йөрәгемне яндырмагач,
Сөю була мени ул?

Һәтс — тердә — ритәем дә,
Һәтс — тердә — ритәем.

Басаемчы бер өздереп,
Ничек тораем түзеп.
Әчемдәге серләремне
Әйтеп бирәем өзеп.

Һәтс — тердә — ритәем дә,
Һәтс — тердә — ритәем.

Тыпырдатып бер басканда,
Шатлык чәчрәсә әгәр.
Йөрәгемне зымбырдатып
Җитсен зиһенгә кадәр.

Һәтс — тердә — ритәем дә,
Һәтс — тердә — ритәем.

Тыпырдатып бер басмагач,
Бию була мени ул?
Йөрәгемне яндырмагач,
Сөю буламы сөю?

Һәтс — тердә — ритәем дә,
Һәтс — тердә — ритәем.
```
