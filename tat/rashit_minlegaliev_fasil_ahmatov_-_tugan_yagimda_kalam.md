# Оригинал

### Рәшит Миңлегалиев / Фасил Әхмәтов - Туган ягымда калам

```
Монда бар җир чәчәк кенә,
Урманы, болыннары.
Һәрбер талы таныш миңа,
Газиз Норма буйлары.

Еракларга моң сибелә
Тургайларның җырыннан.
Алар да бит читкә китми
Туып үскән кырыннан.

Акыл җыйган картлар сыман
Каршылый наратлары.
Кешни монда балачакның
Кыңгыраулы атлары.

Еракларга моң сибелә
Тургайларның җырыннан.
Алар да бит читкә китми
Туып үскән кырыннан.

Азмы якташ туган җирдән
Читләрдә башын салган?
Алар өчен дә яшим мин,
Туган ягымда калам.

Еракларга моң сибелә
Тургайларның җырыннан.
Алар да бит читкә китми
Туып үскән кырыннан.
```
