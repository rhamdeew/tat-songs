# Оригинал

### Виталий Агапов - Җөри сабантуйлары

```
Питрауларга кайтам әле
Мин Җөри авылына.
Кайтмыйм, барам, тумаган бит
Мин Җөри авылында.

Җөри сабантуйлары,
Шаулый Лыя буйлары.
Җырлап-биеп күңел ачыйк,
Питрау сабантуйлары.

Дуслар җыелышкан бүген
Кала-тау итәгенә.
Әй, кунаклар, керегез сез,
Питрау түгәрәгенә!

Җөри сабантуйлары,
Шаулый Лыя буйлары.
Җырлап-биеп күңел ачыйк,
Питрау сабантуйлары.

Шаулый мәйдан, гөрли Җөри,
Кунак кызлары килгән.
Ә уртада иң чибәре,
Питрау гүзәле икән.

Җөри сабантуйлары,
Шаулый Лыя буйлары.
Җырлап-биеп күңел ачыйк,
Питрау сабантуйлары.
```
