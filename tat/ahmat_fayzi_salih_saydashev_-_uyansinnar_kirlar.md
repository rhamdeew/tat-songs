# Оригинал

### Әхмәт Фәйзи / Салих Сәйдәшев - Уянсыннар кырлар

```
Дәртле җырлар агыла киң күкрәктән,
Ташып чыга дулкын шикелле.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Таш мае иде дәртләр ташкын сыман,
Туган кырлар җанга сөйкемле.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Эшкә бара колхоз батырлары,
Җәелеп ята иркен басулар.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!

Эшкә бара колхоз матурлары,
Дәртлеләр һәм эштә ярсулар.

Уянсыннар кырлар,
Җырласын батырлар,
Хезмәт тавышы җирдә
Яңрасын!
```
