# Оригинал

### Радик Шәрәфиев - Эзләдем сине

```
Уйламадым да сине
Очратырмын дип өндә.
Ничә еллар инде син
Яшисең минем күңелдә.

Эзләдем, эзләдем сине,
Эзләдем сине бу җирдә.
Мин таптым, мин таптым сине,
Яратуым булыр мәңгегә.

Синең бер карашыңнан
Мин шашам хыяллардан.
Көлмә минем хисләремнән,
Мин уйнамыйм синең белән.

Эзләдем, эзләдем сине,
Эзләдем сине бу җирдә.
Мин таптым, мин таптым сине,
Яратуым булыр мәңгегә.

Шатландыр көн дә елмаеп,
Чәчәк ат минем күңелдә.
Бир кулыңны шулай оялып,
Алып барам бәхет иленә.

Эзләдем, эзләдем сине,
Эзләдем сине бу җирдә.
Мин таптым, мин таптым сине,
Яратуым булыр мәңгегә.
```
