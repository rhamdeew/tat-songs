# Оригинал

### Римма Күчемова / Ризван Хәкимов - Кире кайт син миңа

```
Эрер идем синең кочагыңда,
Сагынудан өзелә үзәгем.
Таңнар туа, көннәр синсез үтә,
Әллә юкмы сиңа кирәгем?

Иркәлидер, бәлки, чәчләреңне
Башка назлы куллар җылысы.
Мәхрүм итмә мине сөюеңнән,
Мәхаббәтнең син бит соңгысы.

Йөрәгеңдә әгәр миңа дигән
Урын калган булса бүтәннән,
Мин табырмын сиңа юлларыңны,
Кире кайт син миңа үткәннән.
```
