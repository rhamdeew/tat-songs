# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Өфеләргә барам әле

```
Өфеләргә барам әле
Ул — минем гүзәл калам.
Анда үткән һәр минутым
Мәңге күңелдә кала.

Сагынамын бакчаларын,
Сихри матур кичләрен.
Анда тәүге мәхәббәтем,
Анда тәүге хисләрем.

Барам әле урап аккан
Дим суларын карарга,
Салаватның һәйкәленә
Ал чәчкәләр салырга.

Өфеләргә барам әле,
Анда җан дусларым бар.
Яшәвемә илһам биргән
Әл дә Өфеләрем бар.
```
