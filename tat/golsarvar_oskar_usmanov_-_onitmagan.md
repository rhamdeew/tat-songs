# Оригинал

### Гөлсәрвәр / Оскар Усманов - Онытмаган

```
Башка белән күргән чакта
Үпкәләгән идем.
Гафу ит дип килгәнеңдә
Кичермәгән идем.
Эндәшәсем килгәндә дә
Эндәшмәгән идем.

Онытмаган, онытмаган,
Ул мине, онытмаган.
Җылы хисләрен җуймаган,
Йөрәгендә саклаган.
Онытмаган, онытмаган,
Ул мине онытмаган.

Аны йөрттем күңелемдә,
Ул — бәхетем, дидем.
Үзем аны сагындым да,
Килсә иде, дидем.
Ышандым да өметемә,
Эзләп табар, дидем.

Онытмаган, онытмаган,
Ул мине, онытмаган.
Җылы хисләрен җуймаган,
Йөрәгендә саклаган.
Онытмаган, онытмаган,
Ул мине онытмаган.

Каршы төшмим бәхетемә,
Ул бит миңа килде.
Үпкәләмә үткәннәргә,
Син — бәхетем, диде.
Өметемне сүндермичә
Яратуым белде.

Онытмаган, онытмаган,
Ул мине, онытмаган.
Җылы хисләрен җуймаган,
Йөрәгендә саклаган.
Онытмаган, онытмаган,
Ул мине онытмаган.
```
