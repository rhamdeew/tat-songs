# Оригинал

### Айдар Минһаҗев / Марат Гобәйдуллин - Бертуганнар

```
Бер ояда үскән кошлар кебек
Таралабыз, йөртә язмышлар.
Бертуганнар булгач, уртак бит ул
Югалтулар безнең, сагышлар.

Бертуганнар — гел туганнар,
Бәхет белән алар атласын.
Сез бер гөлнең чәчкәләре диеп,
Ходай үзе безне сакласын.

Әниемнең истә әйткән сүзе,
Аерылмагыз диде гомергә.
Туганнар гел янәшәдә әле,
Бергә әле, әни, без бергә.

Бертуганнар — гел туганнар,
Бәхет белән алар атласын.
Сез бер гөлнең чәчкәләре диеп,
Ходай үзе безне сакласын.

Сикәлтәле гомер юлы буйлап
Иңне иңгә куеп барганда.
Җилләр бездән читтә исәр әле,
Гел елмаер безгә замана.

Бертуганнар — гел туганнар,
Бәхет белән алар атласын.
Сез бер гөлнең чәчкәләре диеп,
Ходай үзе безне сакласын.
```
