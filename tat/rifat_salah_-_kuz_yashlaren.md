# Оригинал

### Рифат Сәлах - Күз яшьләрең

```
Гомеремнең күпме мизгелләре сине уйлап үтте,
Күңелемнең бөтен хисе синең өчен иде.
Сурәтеңне соңгы тапкыр назлап-назлап үптем, үптем…
Киләчәгем үткәнемне кичерерме?

Сине уйлап күпме гомер
Узгандыр ул, узгандыр ул.
Күз яшьләрең, бәлки, мине
Кызгандырыр.
Сине эзләп күпме гомер
Узгандыр ул, узгандыр ул.
Күз яшьләрең, бәлки, мине
Кызгандырыр.

Өр-яңадан кайтып киләм синсез үткән гомеремә,
Күңелемнең дөрләп янган чагы үтте менә.
Без ярата алмадык шул мәхәббәтнең бер көнендә.
Йөрәгемдә учак сүнгән. Ул көл генә.

Сине уйлап күпме гомер
Узгандыр ул, узгандыр ул.
Күз яшьләрең, бәлки, мине
Кызгандырыр.
Сине эзләп күпме гомер
Узгандыр ул, узгандыр ул.
Күз яшьләрең, бәлки, мине
Кызгандырыр.
```
