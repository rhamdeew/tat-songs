# Оригинал

### Виталий Агапов - Ун ел гомер узып киткән

```
Ун ел гомер узып киткән,
Минуты да калмаган.
Секундлары тик-так килеп,
Галәмнәргә таралган.

Ун ел гомер узып киткән,
Ничек куып җитәсе.
Әллә инде килгән якка,
Яшьлегемә китәсе?

Ун ел гомер узып киткән,
Күпме сулар аккандыр.
Дөнья күрдек, илләр гиздек,
Күпме дуслар арткандыр.

Ун ел гомер узып киткән,
Ничек куып җитәсе.
Әллә инде килгән якка,
Яшьлегемә китәсе?

Ун ел гомер узып киткән,
Бәлки шулай кирәктер.
Сез дусларым кебек дуслар
Җир йөзендә сирәктер.

Ун ел гомер узып киткән,
Ничек куып җитәсе.
Әллә инде килгән якка,
Яшьлегемә китәсе?
```
