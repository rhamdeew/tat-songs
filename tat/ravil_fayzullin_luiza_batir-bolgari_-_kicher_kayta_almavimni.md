# Оригинал

### Равил Фәйзуллин / Луиза Батыр-Болгари - Кичер кайта алмавымны…

```
Язмышымны белми
Озак көткәнсеңдер.
Кайтыр, диеп,
Өмет иткәнсеңдер.
Илгә кайтасылар
Килми идеме соң!
Гомер киселүен
Тели идемме соң?

Кичер, әнкәй, мине,
Кичер, әнкәй, кичер!
Яу кырыннан
Кайта алмавымны…
Кичер, әнкәй, кичер
Чит ил туфрагында
Мәңгелеккә
Ятып калганымны…

Сугыш фаҗигасын
Онытмасын еллар.
Язмышлары миндәй —
Миллионнар!
Барсын илгә бирдем, —
Күпме көчем булды,
Һәлак булуым да
Халкым өчен булды.

Кичер, әнкәй, мине,
Кичер, әнкәй, кичер!
Яу кырыннан
Кайта алмавымны…
Кичер, әнкәй, кичер
Чит ил туфрагында
Мәңгелеккә
Ятып калганымны…
```
