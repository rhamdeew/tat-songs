# Оригинал

### Мәхмүт Хөсәен / Бәхти Гайсин - Хатынга булмый ярап

```
Тәмәкене тартмадым, тартмадым,
Эчә идем — ташладым, ташладым. Эх!
Ах, шулай да эш харап:
Хатынга (бичәгә) булмый ярап.

Суын булса — китерәм, китерәм,
Керен булса — киптерәм, киптерәм. Эх!
Ах, шулай да эш харап:
Хатынга (бичәгә) булмый ярап.

Уртанчысын уйнатам, уйнатам,
Иң кечесен йоклатам, йоклатам. Эх!
Ах, шулай да эш харап:
Хатынга (бичәгә) булмый ярап.

Тозлыймын кыяр, каклыймын каз…
Ял итимен дисәм бер аз,
“Син — ир кеше, сиңа, — ди ул, —
Җитмеш төрле һөнәр дә аз!” Эх!
Ах, шулай да эш харап:
Хатынга (бичәгә) булмый ярап.
Эх, егетләр!
```
