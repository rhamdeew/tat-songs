# Оригинал

### Хәсән Туфан / Фәрит Хатыйпов - Синең нурың (Синең елмаюың)

```
Кояшта тудың микән,
Йолдызда тордың микән?
Өйгә ай кергәнме дисәм,
Ул синең нурың икән.

Кигәннәрең ал икән,
Гөләп белән пар икән.
Алны уңмый диләр иде,
Халык сүзе хак икән.

Керфекләрең кагасың,
Моңсу гына багасың.
Ай кояштан нурлар ала,
Ә син каян аласың?
```
