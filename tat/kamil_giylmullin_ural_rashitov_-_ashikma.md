# Оригинал

### Камил Гыйльмуллин / Урал Рәшитов - Ашыкма

```
Ашыкма, алма син кулыңны —
Тәнемә җылысын сеңдерим.
Сөенеч канатын киереп,
Җанымны күкләргә менгерим.

Ашыкма тиз генә алырга
Иреннәр иреннәр алырга.
Ике җан син дә мин бергәләп
Йөзик без зәп-зәңгәр күкләрдә.

Ашыкма, төшәргә күкләрдән,
Гизик без болытлар арасын.
Җирдәге дус-ишләр барысы
Безгә тик сокланып карасын.

Ашыкма алырга күзләрдән
Наз тулы, дәрт тулы карашың.
Тик күзләр бер-берсен аңлыйлар.
Йөрәкне утларга сала шул.
Ашыкма, акшыкма, ашыкма,
дәрт тулы, моң тулы каршың.
Тик күзләр бер-берсен аңлыйлар.
Йөрәкне утларга сала шул.

Ашыкма, кара син елмаеп,
Сирпелсен күңелең җылысы.
Кирәкми алтыннар-мәрҗәннәр —
Бул миңа бүләкнең олысы.

Ашыкма бу җирдән китәргә
Гомер бит бер генә бирелә.
Тау булып таулар да ишелә,
Сулар да акмый бит кирегә.
Ашыкма, ашыкма, ашыкма,
Ашыкма бу җирдән китәргә.
Тау булып таулар да ишелә,
Сулар да акмый бит кирегә.
```
