# Оригинал

### Рәмзинә Тимерова / Риф Арслан - Кил иркәм

```
Юлларың бик озак,
Бик урау, бормалы.
Әллә соң күрәзәче,
Ялгышмый урады.
Көтәм мин иртәсен,
Күзләрем юлларда.
Син минем янымда,
Хыялым уемда.

Бәхетле булырга,
Кем генә теләми.
Тик һәр кем үз серен,
Сөйләми, сөйләми.
Бер кемгә карама,
Кил иркәм яныма.
Гомерлек юлдаш бул,
Пар бул син җаныма.

Гомернең яртысы,
Тик синсез үтелгән.
Яшьлегем сукмагы,
Үтелгән, сүтелгән.
Яратам дисең син,
Көт мине ташлама.
Минем тормыш юлы,
Тик синнән башлана.

Бәхетле булырга,
Кем генә теләми.
Тик һәр кем үз серен,
Сөйләми, сөйләми.
Бер кемгә карама,
Кил иркәм яныма.
Гомерлек юлдаш бул,
Пар бул син җаныма.
```
