# Оригинал

### Әлфия Гаязова / Шамил Шәрипов - Йөрәк җылысы

```
Утларымны яндырырга түгел,
Җылытырга диеп мин яктым.
Күңелкәем сүнеп барганда
Йолдыз булыр кеше мин таптым.

Утыннары итеп салдым күңелемнең
Ялгызлыктан сулган гөлләрен.
Төтен булып очсын, дидем зәңгәр күккә,
Газапларда үткән көннәрем.

Учакларга салмый әле саклап торам
Хыялларның бары зәңгәрен.
Җырларымның барын, моңнарымның барын,
Тормышымның матур мәлләрен.

Учакларның яннарында сине көтә
Җырларымның иң-иң моңлысы.
Учакларның утларыннан көчле,
Сиңа булган йөрәк җылысы.
```
