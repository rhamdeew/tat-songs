# Оригинал

### Зифа Нагаева - Мин бит сиңа пар идем

```
Баш бирмәдең кушканнарга,
Баш идең куштаннарга.
Түзәрбез дидең — түзмәдең,
Суз калды дошманнарга.

Мин бит сиңа пар идем,
Гомерлеккә кал дидем,
Кадерләремне белмәдең,
Сөяп туймас яр идем.

Ике битен алланды,
Әйтер сузең калганмы.
Саубуллашу сагышымнан
Яна жырым яралды.

Мин бит сиңа пар идем,
Гомерлеккә кал дидем,
Кадерләремне белмәдең,
Сөяп туймас яр идем.

Сина яр бутән әле,
Мин бәхет көтәм әле.
Гомерләрең мине сагынып
Саргаеп үтәр әле.

Мин бит сиңа пар идем,
Гомерлеккә кал дидем,
Кадерләремне белмәдең,
Сөяп туймас яр идем.
Кадерләремне белмәдең,
Сөяп туймас яр идем…
Мин бит сиңа пар идем…
```
