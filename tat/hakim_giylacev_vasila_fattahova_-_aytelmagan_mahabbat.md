# Оригинал

### Хәким Гыйләҗев / Вәсилә Фәттахова - Әйтелмәгән мәхәббәт

```
Сине күреп янып йөрдем
Әйтә алмый теләгемне
Әйтелмәгән мәhәббәтем
Яндыра йөрәгемне.

Үзем сөям, үзем көям
Үзем качам күренсәң
Чәчкә булып үсәр идем
Чәчкәләргә үрелсәң.

Йә елмаеп, йә моңаеп
Каршыма килсәң иде
Тормыш юлын берәү булып
Үтәек дисәң иде.

Сине күреп янып йөрдем
Әйтә алмый теләгемне
Әйтелмәгән мәhәббәтем
Яндыра йөрәгемне.
```
