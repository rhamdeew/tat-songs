# Оригинал

### Юлай Кәримов / Айрат Кобагышев - Авылымда ак томан

```
Авылымда ак томан,
Томаннары таралыр.
Туып үскән якка кайтсам
Күңелдә җыр (моң) яралыр.

Тугайларда ак томан,
Томаннары таралыр.
Туып үскән якка кайтсам
Мәхәббәтем яңарыр.

Тын кичләрдә ак томан,
Томаннары таралыр.
Моңлы хис һәм сары сагыш (Моңлы җырдай, сары сагыш)
Мәңге озатып барадыр (Күңелләрдә каладыр).

Авылымда ак томан (Уйсуларда ак томан),
Томаннары таралыр.
Яшьлектәге кайнар хисләр (Тын кичләрдәге моңлы җыр)
Йөрәкләрдә каладыр (Бәгеремне өзәдер).
```
