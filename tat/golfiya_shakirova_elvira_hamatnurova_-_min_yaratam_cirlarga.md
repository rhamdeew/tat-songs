# Оригинал

### Гөлфия Шакирова / Эльвира Хаматнурова - Мин яратам җырларга

```
Баш очымда — калфагым,
Аягымда — башмагым.
Ничек инде җырламыйсың,
Минем бит шундый чагым.

Хыялларым — ак җилкән,
Кузгала һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.

Күлмәгем аллы — гөлле,
Аллы — гөлле чәчәкле.
Чәчәкле генә түгел ул,
Үзенә бер бизәкле.

Хыялларым — ак җилкән,
Кузгала һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.

Шатлана бит күңелем,
Ямьле иртә — бүгенем.
Мин бит аңлыйм: барсы шуның,
Бер дә юкка түгелен.

Хыялларым — ак җилкән,
Кузгала һәркөн иртән.
Моңнарны кушып моңнарга,
Мин яратам җырларга.
```
