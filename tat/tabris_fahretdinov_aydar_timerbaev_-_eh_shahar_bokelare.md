# Оригинал

### Тәбрис Фәхретдинов / Айдар Тимербаев - Эх, шәһәр бөкеләре

```
Машиналар тыз-быз чаба,
Кемнәр кая ашыга.
Ашыкканда озын чират,
Бөке тора каршыда,
Бөке тора каршыда.

Эх, шәһәр бөкеләре,
Озаклап көтүләре.
Үкендерә гомерләрнең
Заяга үтүләре.

Утырабыз бөкеләрдә,
Зарланып уф, ах итеп.
Тик булмый шул бу чиратны
Урап, әйләнеп үтеп,
Урап, әйләнеп үтеп.

Эх, шәһәр бөкеләре,
Озаклап көтүләре.
Үкендерә гомерләрнең
Заяга үтүләре.

Сабыр гына үтә алсаң
Урамдагы мәхшәлне.
Эзләп чыгып китәрмен күк
Бөкеләрсез шәһәрне,
Бөкеләрсез шәһәрне.

Эх, шәһәр бөкеләре,
Озаклап көтүләре.
Үкендерә гомерләрнең
Заяга үтүләре.
```
