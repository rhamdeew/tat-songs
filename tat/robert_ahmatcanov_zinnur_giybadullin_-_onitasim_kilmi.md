# Оригинал

### Роберт Әхмәтҗанов / Зиннур Гыйбадуллин - Онытасым килми

```
Тик бер генә көлеп дәшкән идең,
Истәлеге калды озакка.
Онытканнар мине онытсыннар,
Ә мин риза татлы газапка.

Елмаюың синең балкый һаман
Көзгесендә якты елганың.
Онытасым килми, онытасым, —
Синдә минем якты елларым.

Хәтеремдә әле кабатланмас
Шомыртларның исе, ак төсе.
Онытасым килми, онытасым —
Синдә минем күңел яктысы.

Елмаюың синең балкый һаман
Көзгесендә якты елганың.
Онытасым килми, онытасым, —
Синдә минем якты елларым.

Сагышлы моң тулы көннәремдә
Елмаюың якты таң иде.
«Оныт» димә миңа, «оныт» димә —
Җырым булып кына кал инде.

Елмаюың синең балкый һаман
Көзгесендә якты елганың.
Онытасым килми, онытасым, —
Синдә минем якты елларым.
```
