# Оригинал

### Расих Ханнанов / Салават Низаметдинов - Яшәү — бәхет бит ул, дусларым (Бу дөнъяны яратам)

```
Бу дөнъяны өзелеп яратам мин,
Көтеп алам һәр көн иртәсен.
Килә минем туган илкәемне
Әкиятләрдән күркәм итәсе.

Йөрәкләрдә — көчле омтылышлар,
Яңа хисләр юлдаш уйларга…
Күңел тели җирнең һәр бизәген
Иң матур җыр итеп җырларга.

Бу дөнъяны өзелеп яратам мин,
Көтеп алам язын, кышларын…
Ямь өстенә, ямьнәр өстәп яшәү
Үзе бәхет бит ул, дусларым!
```
