# Оригинал

### Рәис Гыймадиев / Альфред Якшимбетов - Казанымда килә яшисем

```
Туар җирен кеше сайлый алмый,
Кайда тели — шунда яшәсен.
Шәхсән үземнең тик мәркәземдә,
Казанымда килә яшисем.

Казан дисәм, күрәм татар халкын.
Татар йорты димен Казанны.
Болгар каны сиңа дәшә безне
Тыңлагыз дип изге азанны.

Моңлы тальян бары синдә генә,
Герман әргәненнән кадерле.
Горур, күркәм халкым йолалары,
Балдан татлы татар сүзләре.

Балам диеп каршы алырсыңмы,
Җылы кочагыңа сыенсам?
Мин бәхетле улың булыр идем,
Азан тавышына уянсам.
```
