# Оригинал

### Марат Гобәйдуллин - Яшьлекне сагыну

```
Үтте-китте яшьлек,
Аккан сулар кебек,
Сизелмәде аның узганы.
Безнең йөргән эзне
Күптән күмгән инде
Гомерләрнең авыр тузаны.

Инде артта калды
Гомерләрнең язы,
Саф чишмәдәй өмет-хыяллар.
Тик яши бүген дә
Күңелнең түрендә
Йөрәкне җилкеткән ул ярлар.

Гомер үткән саен
Артка борылып карыйм:
Безнең яшьлек зая узмаган.
Сөю бакчасында
Без утырткан гөлләр
Шау чәчәктә әле, сулмаган.

Яшьлек үтә диеп,
Йөрмик янып-көеп,
Гомерләрнең соңы түгел лә.
Гүзәл чәчкә кебек
Мәңге шиңмәс яшьлек
Яши һаман йөрәк түрендә.
```
