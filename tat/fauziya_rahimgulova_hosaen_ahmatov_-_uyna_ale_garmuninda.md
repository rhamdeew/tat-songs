# Оригинал

### Фәүзия Рәхимгулова / Хөсәен Әхмәтов - Уйна әле гармуныңда

```
Уйна әле, уйна гармуныңны,
Яшәп булмый җирдә ялкынсыз.
Еракларда янган бер йолдызым
Күк томаннар аша балкысын.

Уйна әле, уйна гармуныңны,
Күңелләрдә базлау сүнмәсен.
Уйна дускай, дәртләндереп уйна,
Йөрәк сүлпәнлектән сүнмәсен!

Уйна әле, уйна гармуныңны,
Балкып китсен дөнья, гөрләсен.
Сыктанмасын йөрәк, сызланмасын,
Сызланмасын янсын, дөрләсен.
Сыктанмасын йөрәк, әйдә уйна,
Тирбәлдереп гармун телләрен!
```
