# Оригинал

### Галия Сафиуллина / Айнур Сабирҗанов - Ал гөлләрем

```
Бөреләнгән гөлләремә
Сулар сиптең иркәләп,
Хисләремнән көлдең бугай,
Калмадым ла үпкәләп.

Ал чәчкә атты гөлләрең
Көн дә сулар сипкәнгә,
Өзелер әле үзәгең
Мин исеңә төшкәндә.

Гөлләремнең чәчәкләрен
Карыйм һәр көн сокланып,
Бәхетле мизгелләремне
Искә алам шатланып.

Ал чәчкә атты гөлләрең
Көн дә сулар сипкәнгә,
Өзелер әле үзәгең
Мин исеңә төшкәндә.

Чәчкәләрнең таҗларына
Сибелә күз нурларым,
Хисләрем ташканга туды
Күңелемдә җырларым.

Ал чәчкә атты гөлләрең
Көн дә сулар сипкәнгә,
Өзелер әле үзәгең
Мин исеңә төшкәндә.
```
