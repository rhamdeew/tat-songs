# Оригинал

### Гыйлемдар Рамазанов / Нариман Сабитов - Җырлый тургай

```
Иртә таңнан, иртә таңнан
Киләм иркен кырга.
Һәм уятам, һәм уятам
Урман, тугайны.
Минем белән бергә,
Бергә эшкә чыга —
Күккә үрли сабан тургае.

Җырлый тургай, җырлый тургай
Дәртле бер җырны:
Мул-уңышлы булсын, ди кырны.

Яз сабанда, яз сабанда
Әгәр син сайрашсаң,
Көз ындырда, көз ындырда
Сине зурлыйлар.
Шуңа бүген хезмәт,
Хезмәт таңын мактап,
Өзелеп-өзелеп сайрый тургайлар.

Җырлый тургай, җырлый тургай
Дәртле бер җырны:
Мул-уңышлы булсын, ди кырны.
```
