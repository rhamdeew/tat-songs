# Оригинал

### Хәйдәр Әхмәтшин / Айдар Тимербаев - Эзләнмә, йөрәк

```
Эзләнмә, йөрәк, эзләнмә,
Соң бит инде эзләргә.
Узганнар белән килешеп
Яшәү кирәк безгә дә.

Эзләнмә, йөрәк, эзләнмә,
Жәллә инде үзеңне.
Узганнарга кайтып булмый,
Тыңлар идең сүземне.

Эзләнмә, йөрәк, эзләнмә,
Эзләргә дә тезләнмә.
Гомерләр дә бирелә бит
Узар өчен бездән дә.

Эзләнмә, йөрәк, эзләнмә,
Жәллә инде үзеңне.
Узганнарга кайтып булмый,
Тыңлар идең сүземне.

Сызланма, йөрәк, сызланма,
Узганнарны кызганма.
Кемне генә гаеплик соң
Гомер бик тиз узганга.

Эзләнмә, йөрәк, эзләнмә,
Жәллә инде үзеңне.
Узганнарга кайтып булмый,
Тыңлар идең сүземне.
Узганнарга кайтып булмый,
Тыңлар идең сүземне…
```
