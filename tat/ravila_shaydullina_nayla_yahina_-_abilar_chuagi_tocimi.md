# Оригинал

### Равилә Шәйдуллина / Найлә Яхина - Әбиләр чуагы тоҗымы

```
Урала, урала
Битләргә чорнала.
Әбиләр чуагы тоҗымы.
Син бит ялгыз түгел,
Син бит ялгыз түгел,
Ни эзли соң күңел?
Ни эзли соң күңел:
Әллә көз, әллә көз,
Әллә көз килүен тоямы?!

Икәү бергә яшәү —
Парлы булу түгел,
Тоҗымга уралса җаннарың.
Өзелеп көтсәң дә,
Өзелеп көтсәң дә,
Атмыйча тилмертсә,
Атмыйча тилмертсә,
Бәхеткә, бәхеткә,
Бәхеткә илтүче таңнарың…
```
