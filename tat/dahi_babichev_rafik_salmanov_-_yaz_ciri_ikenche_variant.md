# Оригинал

### Даһи Бабичев / Рафик Сәлмәнов - Яз җыры (икенче вариант)

```
Яфрак яра чуклы каен,
Татлы исен сибеп,
Бөреләрен ачты гөлләр,
Язгы тынны сизеп.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.

Чәчәк ата туган ягым,
Яшәрделәр кырлар.
Якты кояш чәчә нурын,
Шаулый (яңгырый) дәртле җырлар.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.

Гөрләп илем каршы ала
Хезмәт белән көнне.
Алга, данлы яшьлек, алга,
Иҗат көтә безне.

Яз, яз килә
Илгә, дуслар.
Мактап язны,
Котлап безне,
Сайрарлар кошлар.
```
