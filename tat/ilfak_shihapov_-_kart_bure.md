# Оригинал

### Илфак Шиһапов - Карт бүре

```
Шомлы кара урман,
Ак карлар тапталган,
Бөтенесе дә нидер көтә.
Кычкырды саескан:
"Карт бүре арыган,
Ул сезне бүген ташлап китә!"

Сусап яңа иткә
Яшь бүреләр көтә,
Вакыт-вакыт куялар улап.
Үз хәлен аңлаган,
Мең тапкыр санаган,
Карт бүре басып тора уйлап.

Белә кара урман,
Нинди сукмаклардан
Карт бүре көтүне йөрткәнен.
"Минеке хак", — диеп,
"Безнең сукмак ", — диеп,
Аю белән дә өзешкәнен.

Яшь бүреләр исә
Гәүдәгә тиз үсә,
Карт бүре тапкан ашап итне.
Тик килә беләсе:
Шулардан берәрсе
Ияртә алырмы көтүне?!

Сакланып атларга,
Салган сукмаклардан
Саклап аучылардан көтүне.
Капканны күрергә,
Тапканны бүлергә
Көтүдә онытмый беркемне…

Дәшми кара урман,
Тынды төнге буран,
Бөтенесен озак уйлады.
Сукмакларны салган,
Эзләрне күп калган
Карт бүре бер читкә атлады.

Дәшми тора көтү,
Җиңел түгел китү,
Бөтенесе никтер куркынган.
Әйтерсең саташкан
Кычкыра саескан:
"Нәрсә көтә сине, әй, урман?"

Шомлы кара урман,
Ак карлар тапталган,
Бөтенесе дә нидер көтә.
Бу көтүне җыйган
Карт бүре арыган,
Ул бүген мәңгелеккә китә…
Ул бүген мәңгелеккә китә.
```
