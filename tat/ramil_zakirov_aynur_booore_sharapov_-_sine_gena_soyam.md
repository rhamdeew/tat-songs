# Оригинал

### Рамил Закиров / Айнур BoooRe Шарапов - Сине генә сөям

```
Күктәге ай ул миңа карый сыман,
Уйларымны укый алса,
Югарыга алып менә калса,
Уттай аны мин тоям, калдырмам,
Бары син кирәк, кирәк бит.

Җанымны, канымны
Бирәм бар(ы)сын да, кабул ит,
Сине генә сөям — шуны тик бел.
Яныңда гына булырмын гел,
Барырмын, табармын.
Яныңа очам, сабыр ит.
Күзләреңне мин күрәм,
Җаныңа юлны беләм.

Төштә булсаң — мин сине бары тоям,
Кулларыңны кулга алам.
Арабызда әгәр киртә булса —
Сине уйлап мин очам, ашыгам,
Бары син кирәк, кирәк бит.

Җанымны, канымны
Бирәм бар(ы)сын да, кабул ит,
Сине генә сөям — шуны тик бел.
Яныңда гына булырмын гел,
Барырмын, табармын.
Яныңа очам, сабыр ит.
Күзләреңне мин күрәм,
Җаныңа юлны беләм.
```
