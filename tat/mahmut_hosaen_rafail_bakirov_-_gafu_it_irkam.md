# Оригинал

### Мәхмүт Хөсәен / Рафаил Бакиров - Гафу ит, иркәм!

```
Тәүге кабат сине
Мин күргән идем.
Сиңа гашыйк булып
Мин йөргән идем.

Без кавышмадык,
Гафу ит, иркәм.
Бергә булырга
Язмаган икән.

Яшьлек артта калды,
Сынмады күңел.
Ачы газаплар да
Булмады түгел.

Без кавышмадык,
Гафу ит, иркәм.
Бергә булырга
Язмаган икән.

Синсез яшәү хәзер
Авыр бигрәк тә.
Бергә булмасак та,
Син гел йөрәктә.

Без кавышмадык,
Гафу ит, иркәм.
Бергә булырга
Язмаган икән.
```
