# Оригинал

### Рәис Гыймадиев / Альфред Якшимбетов - Иделкәй

```
Идел суына карыймын,
Каеннар арасыннан.
Дулкын-дулкын булып ага,
Гомерем ага сыман.

Кайда агасың, Иделкәй?
Кайларга ашыгасың?
Ярсу агашыңа карап
Мин дә бит ашкынамын.
Кайда агасың, Иделкәй?
Кайларга ашкынасың?
Синең агашыңа карап
Мин дә бит ашкынамын.

Булдырганга бирермен ди,
Булдыксыздан алырмын.
Әзләгәннәр таба диләр,
Мин дә сине табармын.

Кайда агасың, Иделкәй?
Кайларга ашыгасың?
Ярсу агашыңа карап
Мин дә бит ашкынамын.
Кайда агасың, Иделкәй?
Кайларга ашкынасың?
Синең агашыңа карап
Мин дә бит ашкынамын.

Күккә мендем, суда йөздем,
Йөрдем киң сәхрәләрдә.
Күрешмәсәк үкенерсең,
Исеңә төшерсәңдә.

Кайда агасың, Иделкәй?
Кайларга ашыгасың?
Ярсу агашыңа карап
Мин дә бит ашкынамын.
Кайда агасың, Иделкәй?
Кайларга ашкынасың?
Синең агашыңа карап
Мин дә бит ашкынамын.
```
