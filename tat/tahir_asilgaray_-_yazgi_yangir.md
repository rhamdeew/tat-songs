# Оригинал

### Таһир Асылгәрәй - Язгы яңгыр

```
Үтәр әле кышлар да,
Килер сагынган язлар.
Кояш нурыннан алда,
Болыннарымны назлар.

Язгы яңгыр, ямь уятыр,
Кышның әзләрен юар.
Тәүге тапкыр, нур уйнатып,
Яшь үләннәре туар.

Көннәр чумар чәчкәгә,
Төсләрен яңгыр буяр.
Төннәр йолдыз чәчсә дә,
Уңыш яңгыры кояр.

Язгы яңгыр, ямь уятыр,
Кышның әзләрен юар.
Тәүге тапкыр, нур уйнатып,
Яшь үләннәре туар.
```
