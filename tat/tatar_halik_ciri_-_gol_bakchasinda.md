# Оригинал

### Татар халык җыры - Гөл бакчасында

```
Гөл бакчасына керәм мин,
Гөлләргә су сибәм мин дә.
Гөлләргә су сибәм мин.
Гөл чәчәген сибә-сибә
Чын күңелемнән, чын күңелемнән,
Чын күңелемнән үбәм мин.

Кызыл-кызыл бөрлегән лә,
Юл буена тезелгән лә,
Юл буена тезелгән.
Минем сымак булмас инде
Синең өчен, синең өчен,
Синең өчен өзелгән.
```
