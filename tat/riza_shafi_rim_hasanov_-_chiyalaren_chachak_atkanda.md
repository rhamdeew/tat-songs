# Оригинал

### Риза Шәфи / Рим Хәсәнов - Чияләрең чәчәк атканда

```
Чияләрең чәчәк атканда,
Чияләргә томан ятканда,
Сиңа юлны табасы иде,
Чияң булып каласы иде.

Чияләрең чәчәк койганда,
Чияләрең шәрбәт җыйганда,
Күңелеңә туласы иде,
Сагыну җырың буласы иде.

Чияләргә карлар яуганда,
Чияләрең гөлдән калганда,
Давылларны куасы иде,
Җәй кояшың буласы иде.

Алсуланып чия пешкәндә,
Чияләргә чыклар төшкәндә,
Нурлы йөзеңә багасы иде,
Нурың булып тамасы иде.
```
