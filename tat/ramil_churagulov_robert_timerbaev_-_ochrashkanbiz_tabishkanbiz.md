# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Очрашканбыз, табышканбыз

```
Җир — өстеннән, әй өстеннән,
Йолдызлардан эзләдем.
Мин яныңда йөреүемне,
Ник син җаным белмәдең.

Очрашканбыз, табышканбыз,
Аерылмабыз гомергә.
Кайгыны да, шатлыкны да,
Безгә бергә бүлергә.

Язлар көтеп, назлар көтеп,
Кояшка үрелмәдем.
Мин бит ап — ак умырзая,
Гөл идем, күрмәдең.

Очрашканбыз, табышканбыз,
Аерылмабыз гомергә.
Кайгыны да, шатлыкны да,
Безгә бергә бүлергә.

Әлдә әле очрашканбыз,
Кавышканбыз йә Ходай.
Күңелебезгә сөю салган,
Ходабызга мең дога.

Очрашканбыз, табышканбыз,
Аерылмабыз гомергә.
Кайгыны да, шатлыкны да,
Безгә бергә бүлергә.
```
