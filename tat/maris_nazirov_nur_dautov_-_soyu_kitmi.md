# Оригинал

### Марис Нәзиров / Нур Даутов - Сөю китми

```
Җилләр кага бәхетемнең ак бишеген,
Хисләремнең чәчәгеннән таҗ өзелә:
Күрдем сине, күрдем сине ятлар белән,
Күрдем сине ятлар белән.

Сурәтеңне ертып кына аттым да ул,
Төшләремә кергән сынны нишләтергә:
Сөю килә язлар белән, язлар белән,
Сөю килә язлар белән…

Еллар койды өметемнең яфракларын,
Яшьлегемнең сукмагына бәс үрелә:
Көн дә үтәм эзләреңнән, эзләреңнән,
Көн дә үтәм эзләреңнән…

Хаталарыңны күптән утка яктым да ул,
Күңелемә сеңгән җырны нишләтергә:
Сөю китми көзләр белән, көзләр белән,
Сөю китми көзләр белән…
```
