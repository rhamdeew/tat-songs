# Оригинал

### Галиәхмәт Шаһи / Салават Шәрәфиев - Исәнмесез!

```
Исәнмесез, дуслар, исәнмесез!
Һәр очрашу шулай башлана,
Һәр күрешү шатлык алып килә,
Куанычтан күзләр яшьләнә.

Исәнмесез, кадерле дуслар!
Исәнмесез, әби — бабайлар,
Исәнмесез, апа — абыйлар.
Исәнмесез, кызлар, малайлар!

Кара каршы торып бер сөйләшик,
Бер серләшик әле җыр аша.
Дөнья ямьле, гөлләр чәчәк атса,
Гөлгә кунып, күңел җырласа.

Исәнмесез, кадерле дуслар!
Исәнмесез, әби — бабайлар,
Исәнмесез, апа — абыйлар.
Исәнмесез, кызлар, малайлар!

Тормышыгыз якты, матур булсын.
Өстәлләрдә ризык мул булсын.
Кайгы — хәсрәт күрми яшәгез сез,
Бездән изге теләк шул булсын.

Исәнмесез, кадерле дуслар!
Исәнмесез, әби — бабайлар,
Исәнмесез, апа — абыйлар.
Исәнмесез, кызлар, малайлар!

Без телибез сезгә сәламәтлек,
Бәхетнең дә зурын телибез.
Гомерлеккә дуслар булып калыйк,
Очрашуга кабат килегез.

Исәнмесез, кадерле дуслар!
Исәнмесез, әби — бабайлар,
Исәнмесез, апа — абыйлар.
Исәнмесез, кызлар, малайлар!
```
