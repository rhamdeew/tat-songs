# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Гашыйклар тавында

```
Ниндидер моңнар бар,
Ниндидер шомнар бар
Каеннар, имәннәр шавында,
Онытыйк шомнарны,
Каршылыйк таңнарны
Бергәләп Гашыйклар тавында.

Без — тауның башында,
Мәңгелек каршында —
Гашыйклар очрашкан урында.
Җанда — Ай нурлары
Хисләр — саф, югары:
Биектә — Киек каз юлында.

Син әле янымда,
Хушлашу таңында
Сүз бирик изге җир алдында:
Аерылган көннәрдә
Булыйк гел үрләрдә,
Булыйк гел Гашыйклар тавында!
```
