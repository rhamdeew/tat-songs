# Оригинал

### Ләйсән Сабитова / Айдар Юнысов - Безнең язлар инде кабатланмас

```
Безнең язлар инде кабатланмас,
Янып китмәс сүнгән учаклар.
Исләремә төшә әле һаман
Шул бәхетле безнең яшь чаклар.

Еллар уза, гомеркәйләр уза,
Хәзер инде парлы икебез.
Төшләремә генә кайчак керә
Без утырткан сөю гөлебез.

Күптән сине, җаным, кичерсәм дә,
Үткәннәргә инде юл ябык.
Мин бәхетле, ник бәхетсез булыйм?
Без яшибез матур пар булып.

Еллар уза, гомеркәйләр уза,
Хәзер инде парлы икебез.
Төшләремә генә кайчак керә
Без утырткан сөю гөлебез.

Сәлам биреп үтмә яннарымнан,
Юлларыңда мине очратсаң.
Җавап бирмә, әгәр тагын бер кат
Күзләреңә синең карасам.

Еллар уза, гомеркәйләр уза,
Хәзер инде парлы икебез.
Төшләремә генә кайчак керә
Без утырткан сөю гөлебез.
```
