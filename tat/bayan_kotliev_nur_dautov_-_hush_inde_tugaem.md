# Оригинал

### Баян Котлыев / Нур Даутов - Хуш инде тугаем

```
Эй, тургаем, киң тугаем,
Ак чәчәкле юлларың.
Ак чәчәкләр кебек калдың,
Безнең яшьлек еллары.

Хуш инде тугаем,
Елда кайт тургаем.
Тагы да, ояга, тирәккә.
Туган як тугае,
Һава да тургае,
Моң-зары сызыла йөрәккә.

Эй, тургаем, тугаемны,
Тошләремдә буйладым.
Моңнарыңны изге уйга,
Кайтыр юлга юрадым.

Хуш инде тугаем,
Елда кайт тургаем.
Тагы да, ояга, тирәккә.
Туган як тугае,
Һава да тургае,
Моң-зары сызыла йөрәккә.

Эй, тургаем, күңелемдә,
Мәңге яшәр тугаем.
Кабат чакыр яшьлегемә,
Сайра-сайра тургаем.

Хуш инде тугаем,
Елда кайт тургаем.
Тагы да, ояга, тирәккә.
Туган як тугае,
Һава да тургае,
Моң-зары сызыла йөрәккә.
```
