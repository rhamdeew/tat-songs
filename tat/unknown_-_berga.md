# Оригинал

### Гөлинә Шәйхи / Илдус Дәүләнов - Бергә

```
Без cүзләрдән арыган,
Без бит үзгә тамырдан,
Көчле рухлы булу каннарга салынган.
Җиһан бүген киң ялан,
Мин, җирем, синең балаң,
Синең өчен ут-су кичеп арылмаган.

Бергә булсак кына көчле,
Беркем җиңмәс төсле,
Бергә, бердәм калыйк җирдә,
Уйлан син, кеше.

Тик бер генә яшибез.
Ник соң әле дәшмибез?
Җиргә чатырдап ябышып
Без яшьнибез.
Җанны ут алган бүген,
Тик артка чикми күңел.
Үзәгемнән шартлатсалар да
Гомер түгел.

Бергә булсак кына көчле,
Беркем җиңмәс төсле,
Бергә, бердәм калыйк җирдә,
Уйлан син, кеше.
Бергә булсак кына көчле,
Беркем җиңмәс төсле,
Бергә, бердәм калыйк җирдә,
Уйлан син, кеше.
Бергә…
Уйлан син, кеше…
```
