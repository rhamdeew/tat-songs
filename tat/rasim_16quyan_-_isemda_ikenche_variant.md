# Оригинал

### Рәсим / 16quyan - Исемдә (икенче вариант)

```
Исемдә минем һаман,
Ерак ара үтсәм дә.
Хәтерләрдән чыкмый
Елларның ачы януы.

Үтә тормыш су кеби,
Кулдагы яралар юып.
Эзләр калдыра тирән
Әрнеп беткән җаннарда.

Кайтмыячакмын, аңлыйм аны.
Тик оныта алмыйм шуны:
Тәрәзәдән төшкән утларын
Туган йортымның.
```
