# Оригинал

### Татар халык җыры - Замана такмаклары (өченче вариант)

```
Минем иркәм армиядә,
Атлар эчерәм, дигән.
Төпсез чиләк, сапсыз көрәк
Минем кулымда, дигән.

Биек тауның башларында
«ДТ» буксовать итә.
Безнең авыл егетләре
Кызлар сортировать итә.

Әнә кило автомобиль,
Төягәннәр семечка.
Яр өстенә яр сөясез
Бу нинди привычка.

Татарча «кая барасың»,
Русча «куда пойдешь».
Ярыңны кеше алса
Пожалуй, с ума сойдешь.
```
