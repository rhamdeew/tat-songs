# Оригинал

### Сафуан Әлибай / Мөфтәдин Гыйләҗев - Яшьлек белән

```
Диңгез булып һәрчак чабып булмый,
Ярлар тыя берчак ташкынны.
Уй-хисләрем дулкынланыр әле,
Яшьлегемә кайтсам ашкынып.

Яшьлек артта калган,
Белеп булмый алдан
Ниләр кичеп, ниләр күрерне.
Нинди яме калыр,
Нинди сәме калыр
Яшьлегемнән язган гомернең.

Яшен булып гомер яшьнәп булмый,
Җаным өшер бәлки салкыннан.
Тик йөрәгем һәрчак янып торыр
Яшьлегемнән калган ялкыннан.

Яшьлек артта калган,
Белеп булмый алдан
Ниләр кичеп, ниләр күрерне.
Нинди яме калыр,
Нинди сәме калыр
Яшьлегемнән язган гомернең.
```
