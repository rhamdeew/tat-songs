# Оригинал

### Татар-Башкорт халык җыры - Шахта (дүртенче вариант)

```
Китәм инде, китәм инде,
Китәм инде, мин китәм, шул.
Аерылганнар бер кавышыр
Дип кенә өмет итәм.

Китә казлар, китә казлар,
Китә казлар Донбасска шул;
Миндә уйлап торам әле
Шул казлардан калмаска.

Шахтер егет йоклар иде
Юрганнарын ябынып шул.
Ятса, йокылары килми,
Туган илен сагынып.
```
