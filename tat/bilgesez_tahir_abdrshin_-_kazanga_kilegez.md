# Оригинал

### Билгесез / Таһир Абдршин - Казанга килегез!

```
Сезнең белән күрешергә
Килдем мин ераклардан.
Җырлар белән, моңнар белән
Кайнар сәламнәр белән.

Килегез, килегез
Казаннарны күрегез!
Килегез, дусларым
Безгә кунакка!

Идел буйларыннан мин килдем.
Кама суларын кичтем.
Шул якларда туып үстем,
Шул якларның суын эчтем.

Килегез, килегез
Казаннарны күрегез!
Килегез, килегез
Безгә кунакка!

Хушлашырга вакыт җитте,
Дусларым, кардәшләрем.
Бергә булган минутларны
Онытмам мин һәркөн саен.

Килегез, йөрегез
Тукай йөрегән әзләрдән!
Килегез, дусларым
Безгә кунакка!
Килегез, дусларым
Безгә Казанга!
```
