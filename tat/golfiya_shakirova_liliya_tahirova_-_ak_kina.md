# Оригинал

### Гөлфия Шакирова / Лилия Таһирова - Ак кына

```
Зәңгәр күктән кояш көлсә
Елмаялар гөлләрем.
Сине уйлап үтә микән
Иртәләрем, кичләрем?

Тәрәз төбем вак кына,
Чәчкәләрем ак кына.
Мәхаббәтнең юлларыннан
Үтик әле чак кына.

Күңел түре гөлбакчамның —
Синдер мәхаббәт гөле.
Хисләремне аңлатырга
Кирәк сандугач теле.

Тәрәз төбем вак кына,
Чәчкәләрем ак кына.
Мәхаббәтнең юлларыннан
Үтик әле чак кына.

Назлар кирәк гөлләргә дә,
Күңелләргә — җылы сүз.
Саф сөюле гомер белән
Яшәсәк иде лә без.

Тәрәз төбем вак кына,
Чәчкәләрем ак кына.
Мәхаббәтнең юлларыннан
Үтик әле чак кына.
```
