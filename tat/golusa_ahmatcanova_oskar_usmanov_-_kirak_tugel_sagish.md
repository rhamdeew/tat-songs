# Оригинал

### Гөлүсә Әхмәтҗәнова / Оскар Усманов - Кирәк түгел сагыш

```
Күз яшьләре түгеп,
Еллар үтте диеп,
Сагышланып утта
Яшәгәнмен көеп.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.

Бәхет килер диеп
Көткән чакларымны,
Искә алам хәзер
Яшьлек елларымны.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.

Бәхет килә икән
Көтә белсәң генә.
Уйлар синең белән,
Бәхетем син генә.

Кирәк түгел сагыш.
Кирәк түгел ялгыш.
Сагышларда, ялгышларда
Адаштырма язмыш.
```
