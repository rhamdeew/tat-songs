# Оригинал

### Сәкинә Хәйруллина / Илдус Садыйков - Кайтам инде авылыма

```
Саг(ы)нуларым чиктән ашты,
Бар җанымны якты сагыш басты,
Көннәремнең яме китте, йокыларым да качты,
Бар хисләрем ярларыннан ташты.

Кайтам инде авылыма,
Кайтам инде газиз туган якларыма.
Авылым төшемә кереп төннәрен саташтыра,
Болын чәчкәседәй матур кызлары сагындыра,
Кайтам инде!

Кайтам инде, юллар үтәм,
Беләм: авыл кайтуымны көтә,
Мин чит булмам беркайчан да туган төбәгемдә,
Сөю уты яши йөрәгемдә.

Кайтам инде авылыма,
Кайтам инде газиз туган якларыма.
Авылым төшемә кереп төннәрен саташтыра,
Болын чәчкәседәй матур кызлары сагындыра,
Кайтам инде!

Кайтам инде авылыма,
Кайтам инде газиз туган якларыма,
Шунда эшләп, шушы җирдә бәхетемне табармын,
Авылымның тарихына яңа юллар язармын,
Кайтам инде!
Кайтам инде…
```
