# Оригинал

### Физәлия Дәүләтгәрәева / Рәзидә Әхмәтова - Кайтыгыз (беренче вариант)

```
Хәбәр сала әнкәебез безгә,
Кайтып төшсә кинәт кайсыбыз.
“Балакайлар, туганыгыз кайтты!
Күрешергә тиз(е)рәк кайтыгыз!”

Туган йортым безне очраштыра.
Әнкәй җыя әле янына.
Әнкәй безне һаман берләштерә.
Әнкәй безне һаман сагына.

Җыя тагын канат асларына
Бишебезне биш як тарафтан.
Бер ояда үскән балаларны
Язмыш юлы ерак тараткан…

Туган йортым безне очраштыра.
Әнкәй җыя әле янына.
Әнкәй безне һаман берләштерә.
Әнкәй безне һаман сагына.

Берәребез кинәт кайтып төшсә,
Хәбәр сала әнкәй: “Кайтыгыз!”
Без җыела торган оя булып,
Мәңге калсын иде йортыбыз.

Туган йортым безне очраштыра.
Әнкәй җыя әле янына.
Әнкәй безне һаман берләштерә.
Әнкәй безне һаман сагына.
```
