# Оригинал

### Фәнир Рәхмәтуллин / Ришат Батыров - Тигез картлык

```
Тигез картлык — нинди тирән мәгнә
Бу сүзләргә дуслар бирелгән
Сөярмен дип яшли биргән вәгдә,
Үткән гөмер бергә үрелгән

Тормыш юлы — тигез басу түгел
Җиңел генә атлап үтергә.
Ялгыз калгач күтәрелми күңел,
Бергә җиңел тормыш көтәргз

Тигез картлык — бәхет балаларда,
Шатланалар сезгә оноклар.
Картайгачта кирәк сез аларга,
Бергә җиңел тормыш көтәргз

Ике йөрәк бердән бергә типсә,
Яшер бер берсенә таянып.
Ходай теләп озон гөмер бирсә,
Яшәгезче бергә шатланып
```
