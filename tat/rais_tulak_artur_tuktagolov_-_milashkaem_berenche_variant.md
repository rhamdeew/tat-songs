# Оригинал

### Рәис Түләк / Артур Туктаголов - Миләшкәем (беренче вариант)

```
Тәрәземне ачтым бакчага,
Аннан миләш миңа баш кага.
Күңелем иләс, серләремне, миләш,
Сөйләмим мин синнән башкага.

Ул өметсез көзнең салкынын
Тоймыймы ни синең ялкының?
Миләшкәем, нишләп эндәшмисең,
Нигә шулай соңлап балкыдың?

Тугайлардан башка кош кунмый,
Түзәрсеңдер ничек пошынмый?
Миләшкәем, сиңа килешәм мин,
Тогрылыкны бары кыш сыный.
```
