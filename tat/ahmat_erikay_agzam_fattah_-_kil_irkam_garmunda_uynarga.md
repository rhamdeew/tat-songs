# Оригинал

### Әхмәт Ерикәй / Әгъзам Фәттах - Кил, иркәм, гармунда уйнарга

```
Кичләрен күңелле Иделдә,
Мин телим көймәдә йөрергә;
Мин телим яңа җыр җырларга,
Кил, иркәм, гармунда уйнарга.

Мин синен чәчеңне сыйпармын,
Иң матур көеңне отармын.
Таралсын көйләрең кырларга,
Кил, иркәм, гармунда уйнарга.

Түшеңә чәчәкләр кадармын,
Күзләрең күленә карармын.
Яратам моңлы көй тыңларга,
Кил, иркәм, гармунда уйнарга.

Күз нурым, акыллым, чибәрем,
Мин сине кадерләп үбәрмен.
Күңелле булсын бар дусларга,
Кил, иркәм, гармунда уйнарга.
```
