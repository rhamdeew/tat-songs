# Оригинал

### Нәзирә Ишколова / Радик Мөхәмәдиев - Тәүге мәхәббәт

```
Үткәннәрне эзләп кайттым тагы
Сагышларга җаным чорналды.
Кичке тальян моңы, яшьлегемнең чыңы
Йөрәгемне сыгып уралды.

Тавыш бирде тәүге сөю хисе
Яшьлегемнең ерак таңыннан.
Хаталарга юмарт, гаделлеккә саран
Язмыш юлын үттем яңыдан.

Ялгыз аккош күлдә кыйгак — кыйгак
Татлы төшләремнән уята.
Таңнар кыелганда, уйлар уелганда
Тәүге сөю күңел юата.

Үрсәләнгән чакта бөтен дөнъям
Уйлар дәръясында югалсам.
Өмет сүрелгәндә, яшьләр ябырылганда
Тәүгә мәхәббәттән көч алам.
```
