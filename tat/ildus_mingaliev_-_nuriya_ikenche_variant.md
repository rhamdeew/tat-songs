# Оригинал

### Илдус Миңгалиев - Нурия (икенче вариант)

```
Йөрәккәем дулкынлана,
Өз(е)леп күрәсем килә.
Бу кадәрлек чибәрлекне
Ходай бирсә дә бирә.

Ак күлмәгеңне киясең,
Гөлдәй матур икәнсең.
Нурия, Нурия!
Күзләр генә тимәсен.

Чәчләрең алтындай балкый,
Битең тулган ай кебек.
Күзләрең шулкадәр матур,
Иреннәрең бал кебек.

Ак күлмәгеңне киясең,
Гөлдәй матур икәнсең.
Нурия, Нурия!
Күзләр генә тимәсен.

Кайда да йөрәк түремдә
Йөртәм синең рәсемне.
Көлгәмдә дә иреннемдә
Булыр синең исемен.

Ак күлмәгеңне киясең,
Гөлдәй матур икәнсең.
Нурия, Нурия!
Күзләр генә тимәсен.
```
