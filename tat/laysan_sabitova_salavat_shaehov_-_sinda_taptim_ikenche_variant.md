# Оригинал

### Ләйсән Сабитова / Салават Шәехов - Синдә таптым (икенче вариант)

```
Ерак юлга чыгу
Һәм көчле ярату
Җаннарыма бирә зур шатлык.
Үткәннәрне уйлап,
Һәм шигырьләр юллап
Үтәр микән минем дә картлык?

Ерак юлга чыгам,
Синең янга барам,
Сагынганга, бәгырем!
Синдә генә,
Синдә таптым яшәү кадерен!
Синең янга барам,
Сагынганга, бәгырем!

Балалар үстереп,
Сөеп һәм сөелеп,
Синең белән килә яшисе!
Гомер буе шулай,
Сөю дулкынында
Килә бит ул минем эшлисе.

Ерак юлга чыгам,
Синең янга барам,
Сагынганга, бәгырем!
Синдә генә,
Синдә таптым яшәү кадерен!
Синең янга барам,
Сагынганга, бәгырем!

Җан өшерлек сүзләр,
Яман начар күзләр
Һич кермәсен безнең арага!
Ант итәм мин сиңа,
Вәгъдә бирәм сиңа,
Бирмәм сине, бирмәм ятларга!

Ерак юлга чыгам,
Синең янга барам,
Сагынганга, бәгырем!
Синдә генә,
Синдә таптым яшәү кадерен!
Синең янга барам,
Сагынганга, бәгырем!
Синдә таптым яшәү кадерен!
Синдә таптым…
Синең янга барам,
Сагынганга, бәгырем!
```
