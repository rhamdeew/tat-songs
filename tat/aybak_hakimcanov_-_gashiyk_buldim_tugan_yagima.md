# Оригинал

### Әйбәк Хәкимҗанов - Гашыйк булдым туган ягыма

```
Бигърәк матур шул Урал — туган ягым,
Синнән дулкынган күңел — уйлата, җырлата.
Юкә аңкый, сыймый күкрәккә,
Бал кортлары оча геү килеп.

Салкын суда уйный багыр,
Идел буйлап.
Сандугачлар сайрый кичен
Җәйне ямьләп.

Курайны тотып сызам, түзми йөрәк,
Суга да ярдәм чумам, балалай куанып.
Кыр үрдәкләр йөзә камышта,
Бөркөтләр чыңкылдый болытта.

Салкын суда уйный багыр,
Идел буйлап.
Сандугачлар сайрый кичен
Җәйне ямьләп.

Тулы чәчәккә алан, җиләк пешкән,
Бөдрә юкәле урман сихырлый йөрәкне.
Онтылмас, калыр гомергә,
Табигатьнең җәйге ямьлесе.

Салкын суда уйный багыр,
Идел буйлап.
Сандугачлар сайрый кичен
Җәйне ямьләп.
```
