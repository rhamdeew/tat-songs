# Оригинал

### Азат Хәнифәтуллин / Тәслимә Низами - Оныттырма

```
Син дә шулай ометләнәсеңме,
Онытылыр әле дисенме?
Югалтудан, сагышлардан түгел,
Онытудан куркам мин сине.

Ә син оныт, әйдә оныта алсан,
Мине онытудан курыкма.
Тик үзеңне, дустым, тик үзеңне
Оныттырма миннән, онытма.

Мина сине хәтерләтә һаман
Һаваларда йөзгән ак болыт.
Яшә әле, яшә йөрәгемдә
Ак болытлардан да ак булып.

Ә син оныт, әйдә оныта алсан,
Мине онытудан курыкма.
Тик үзеңне, дустым, тик үзеңне
Оныттырма миннән, онытма.

Югалтудан, сагышлардан түгел,
Онытудан куркам мин сине.
Ак болыттан тамган тан тамчылар
Йөрәгемә язды исмеңне.

Ә син оныт, әйдә оныта алсан,
Мине онытудан курыкма.
Тик үзеңне, дустым, тик үзеңне
Оныттырма миннән, онытма.
```
