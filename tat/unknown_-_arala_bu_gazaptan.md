# Оригинал

### Хәния Фәрхи - Арала бу газаптан

```
Назлы карашларың беләм,
Зинһар, сөеп карама.
Бара алмыйм яннарыңа —
Упкын бар бит арада.

Нигә элек күрмәдем?
Белмәдем бит белмәдем —
Бу дөньяда син барын
Ягымлы, назлы ярым.

Синең сөю миңа газап,
Үзең дә бит беләсең.
Аерылып китү белән,
Килә сине күрәсем.

Нигә элек күрмәдем?
Белмәдем бит белмәдем —
Бу дөньяда син барын
Ягымлы, назлы ярым.

Кавышырга мөмкин түгел,
Җан әрни ялгызлыктан,
Йә оныттыр, чарасын тап,
Арала бу газаптан.

Нигә элек күрмәдем?
Белмәдем бит белмәдем —
Бу дөньяда син барын
Ягымлы, назлы ярым.
```
