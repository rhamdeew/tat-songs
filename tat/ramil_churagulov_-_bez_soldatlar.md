# Оригинал

### Рамил Чурагулов - Без солдатлар

```
Гүзәл илем, гүзәл Ватаныма,
Ябырылса кансыз дошманнар.
Акты тамчы, изге канын түгеп
Саклап калган аны бабамнар.

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга — изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Ил-Ватанга биргән изге антны,
Үтеп чыгу, ул бит егетлек.
Көнне төнгә ялгап, без хөзмәттә,
Дөньяларда булсын иминлек!

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга — изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Маңлайларда кызыл йолдыз яна,
Сары каеш безнең билләрдә.
Үсеп җитеп без дә салдат булдык —
Ризык булсын туган илләрдә!

Без солдатлар,
Безнең биргән антлар
Ил-Ватанга — изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат!

Өйдә көтсен безне әткәй, әнкәй,
Тогры дуслар, сөйгән җан-ярлар.
Җырлап, гөрләп яшәү безгә язсын,
Имин-аман булып дөнъялар.

Без солдатлар, безнең биргән антлар —
Ил-ватанга изге аманат.
Без чын егет, без бит бөркет,
Без чын лачын, без бит чын солдат.
```
