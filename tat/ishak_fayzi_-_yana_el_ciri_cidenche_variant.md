# Оригинал

### Исхак Фәйзи - Яңа ел җыры (җиденче вариант)

```
Бизәкле чыршылар
Ерактан балкыйлар,
Бәйрәмгә чакырып торалар.
Тотышыйк кулларны,
Туачак елларны
Бергәләп каршылыйк, туганнар.

Биегез, җырлагыз,
Оялып тормагыз,
Беркем дә күңелсез калмасын.
Яңа ел кичендә
Һәр кеше өчен дә
Өр-яңа матур җыр яңгырасын.

Яңа ел килгәндә
Синдә дә, миндә дә
Иң якты хыяллар уяна.
Яңа ел, Яңа ел,
Бәхетләр алып кил,
Шатлыклар алып кил дөньяга.
```
