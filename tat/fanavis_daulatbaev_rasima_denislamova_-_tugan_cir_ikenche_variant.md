# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Туган җир (икенче вариант)

```
Туган җирдән тәпи басып
Атлап киткәнбез.
Шушы җирдә авыр дими
Гомер итәбез.

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!

Җирдә кылган изгелектән
Безне танырлар.
Туган җирдә тармакланган
Безнең тамырлар.

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!

Туган җирем — сызылыплар
Таңнар аткан җир.
Кадерле син гомерлеккә
Бәхет тапкан җир!

Дәрт-көч алып яшәгән җир,
Шатлык тойган җир.
Күңелләргә һәрчак якын,
Газиз туган җир!
```
