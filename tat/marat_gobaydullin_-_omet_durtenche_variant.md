# Оригинал

### Марат Гобәйдуллин - Өмет (дүртенче вариант)

```
Гомерләр катлаулы,
Була төрле чагы.
Кайчак моңсу мәле,
Куаныч, шатлыгы.

Кушымта:
Бик авыр булса да,
Өметең өзмә син.
Яшәү көчең синең
Беркайчан сүнмәсен.

Бәхетле мизгелләр
Гел булып тормый ул.
Язмышта язылган
Һич эзсез узмый ул.

Кушымта:

Көн арты көн туа,
Гомер шулай уза.
Рәхмәтле булыйк без
Бу насыйп язмышка.
```
