# Оригинал

### Рамил Чурагулов / Эльвира Хаматнурова - Яшик сөеп, сөелеп!

```
Синсез булган минутларым
Көннәр булып тоела
Син көнемдә, син төнемдә
Һич чыкмыйсың уемнан.
Бар хәсрәтләр онытыла
Әгәр булсаң янымда
Килә булысым әргәндә
Тагын, тагын, тагында

Матур бер язмышлар булып
Җанга чыкмас булып тулып
Янып, көеп һәм сөелеп
Яшик куш йөрәкләр булып

Яннарымнан һич югалма
Конемдә һәм төнемдә
Бул син, бул син яннарымда
Иртәгә һәм бүгендә
Синсез булган минутларым
Көннәр булып тоела
Син көнемдә, син төнемдә
Һич чыкмыйсың уемнан.

Матур бер язмышлар булып
Җанга чыкмас булып тулып
Янып, көеп һәм сөелеп
Яшик куш йөрәкләр булып
```
