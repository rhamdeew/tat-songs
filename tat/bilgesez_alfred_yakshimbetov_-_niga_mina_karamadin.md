# Оригинал

### Билгесез / Альфред Якшимбетов - Нигә миңа карамадың

```
Яшь чагында икәү бергә
Йөрегән идек без.
Бер-беребезне өзлеп-өзлеп
Сөйгән идек без.

Нигә миңа карамадың?
Калдырдың да киттең бәгърем
Мине ялгызым.

Синнән башка ялгыз башым
Үт түгел мәлен
Синнән башка чәчәк атмый
Минем гөлләрем.

Нигә миңа карамадың?
Калдырдың да киттең бәгърем
Мине ялгызым.

Күңел сине генә уйлый,
Сине юксына.
Синнән башка, мин ялгызга
Тормыш юк сыман.

Нигә миңа карамадың?
Калдырдың да киттең бәгърем
Мине ялгызым.
```
