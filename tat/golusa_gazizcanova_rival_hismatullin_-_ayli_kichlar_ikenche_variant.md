# Оригинал

### Гөлүсә Газизҗанова / Риваль Хисмәтуллин - Айлы кичләр (икенче вариант)

```
Без яраткан айлы кичләр
Һаман да шулай икән.
Йолдыз санаган кичләр дә
Еракта калган икән,
Еракта калган икән.

Без йөргән сукмакларга
Үләннәр үскән инде.
Яшьлектә сөйгән ярларны
Башкалар сөйгән инде.

Без яраткан чәчәкләрне
Кыраулар төшеп алган.
Сөйсәм дә, башкалар белән
Ятларга насыйп булган,
Ятларга насыйп булган.

Без йөргән сукмакларга
Үләннәр үскән инде.
Яшьлектә сөйгән ярларны
Башкалар сөйгән инде.

Кабат-кабат уйлап куям,
Исләргә төшкән саен.
Сөюләр сүрелгән инде
Оныту гына кыен,
Оныту гына кыен.

Без йөргән сукмакларга
Үләннәр үскән инде.
Яшьлектә сөйгән ярларны
Башкалар сөйгән инде.
Яшьлектә сөйгән ярларны
Башкалар сөйгән инде…
```
