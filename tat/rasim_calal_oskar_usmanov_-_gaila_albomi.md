# Оригинал

### Рәсим Җәләл / Оскар Усманов - Гаилә альбомы

```
Кадерләп саклыйсың альбомны,
Сак кына ачасың битләрен.
Саргайган рәсемнән елмая,
Яратып кискән читләрен.

Кадерле альбомга тупланып
Саклана гаиләбез җылысы.
Балачак үткәне беленми,
Бары тик рәсемдә каласы.

Гөрләшеп яшәгән чагымда
Ялгызым калмамын дигәнсең.
Түзгәнсең балалар хакына,
Никадәр авырлык күргәнсең.

Кадерле альбомга тупланып
Саклана гаиләбез җылысы.
Балачак үткәне беленми,
Бары тик рәсемдә каласы.

Рәсемдә гөнаһсыз балалар,
Эчкерсез елмаеп карыйлар.
Сизелми үткәнге балачак,
Үсеп тә җитешкән сабыйлар.

Кадерле альбомга тупланып
Саклана гаиләбез җылысы.
Балачак үткәне беленми,
Бары тик рәсемдә каласы.
Кадерле альбомга тупланып
Саклана гаиләбез җылысы.
Балачак үткәне беленми,
Бары тик рәсемдә каласы…
Балачак үткәне беленми,
Бары тик рәсемдә каласы…
```
