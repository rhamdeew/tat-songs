# Оригинал

### Билгесез / Нурия Абдуллина - Сөелергә килгән идем

```
Сөяркә булырга түгел, сөелергә
Килгән идем җиргә.
Гомер кояшыбыз баеганын
Каршыларга бергә.

Нишлисең бит, сөюлүләр түгел,
Сөйәркәлек насыйп.
Яратабыз, зинһар, ярлыка дип,
Ходайга ялбырып.

Кичтә мине, мине иркәләгән
Кулларыңны үбеп,
Гомер итү ямен рәхәтенә
Тезләремне чүгеп.

Нишлисең бит, сөюлүләр түгел,
Сөйәркәлек насыйп.
Яратабыз, зинһар, ярлыка дип,
Ходайга ялбырып.

Кичтә мине багу бу дөньяга
Синең күзләр белән.
Нахак арасыннан агарырга
Синең сүзләр белән.

Нишлисең бит, сөюлүләр түгел,
Сөйәркәлек насыйп.
Яратабыз, зинһар, ярлыка дип,
Ходайга ялбырып.
```
