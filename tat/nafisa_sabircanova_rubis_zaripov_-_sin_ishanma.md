# Оригинал

### Нәфисә Сабирҗанова / Рубис Зарипов - Син ышанма

```
Син ышанма, кара болыт чыгып,
Зәңгәр күкне каплап киткәнгә.
Син ышанма сылу каеннарның
Яфраклары корып кипкәнгә.
Син ышанма җәйнең үткәненә,
Моңсу көзнең сары төсенә.
Син ышанма, төнлә сискәндереп
Уяткан ул кыска төшеңә.
Син ышанма, әгәр сагышланып
Төбәлсәм мин башка кешегә.
Син ышанма бу дөньяның безгә
Каршы булган барлык эшенә!
```
