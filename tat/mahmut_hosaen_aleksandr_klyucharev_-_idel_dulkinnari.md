# Оригинал

### Мәхмүт Хөсәен / Александр Ключарев - Идел дулкыннары

```
Җем-җем итә чыклар таңда
Бизәп мул кырны.
Иркәли Казан каласын
Идел дулкыны.

Казан! Казан!
Дәрле, матур кала.
Башкалабыз Мәскәүдән ул
Якты нур ала.

Казан порты яннарында
Чайкала диңгез.
Су буйлап безгә кунака,
Дуслар, килегез!

Казан! Казан!
Дәрле, матур кала.
Башкалабыз Мәскәүдән ул
Якты нур ала.
```
