# Оригинал

### Галия Сафиуллина / Айнур Сәйфиев - Буралым — туган авылым

```
Батыр уллар, уңган кызлар —
Буралының көзгесе.
Җырга-моңга гашыйк халкы,
Хезмәт сөя һәрберсе.

Буралым — туган авылым,
Сиңа кайтым барышым.
Наратлыкта туктап тыңлыйм
Сандугачлар тавышын.

Сагынып кайтап авылымны,
Таллы инеш буйларын.
Шифа биреп, челтерәп аккан
Изге чишмә суларын.

Буралым — туган авылым,
Синдә туды моңнарым.
Күңелемә дәва бирә
Иген үскән кырларың.

Сабантуйлар гөрләп уза,
Авылдашлар җыелып.
Рухи байлык ала һәркем,
Табигатькә сыенып.

Туган ягым — җан бишегем,
Сагынып кайттым, Буралым.
Яшьлегем эзләрен эзләп,
Урамнарны урадым.

Ак яулыклы әбиләрем,
Түбәтәйле картларым,
Сез укыган изге дога
Авыл халкын саклады.

Күңелемнән китми һаман
Көтү кайткан мизгелләр,
Ялан тәпи йөгерә-йөгерә,
Яшьлектә узган көннәр.

Буралым — туган авылым,
Сиңа кайтым барышым.
Наратлыкта туктап тыңлыйм
Сандугачлар тавышын.

Буралым — туган авылым,
Синдә туды моңнарым.
Күңелемә дәва бирә
Иген үскән кырларың.
```
