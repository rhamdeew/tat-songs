# Оригинал

### Харис Мостафин / Булат Шәрипов - Каз бәбкәле сабый чак

```
Хәтеремдә әле һич җуелмый
Каз бәбкәсе саклаган чаклар
Су буенда безне көтәдер күк
Көйгән бәрәңгеле учаклар.

Әбәт алып төшкән әбиемнең
Ипи белән сөтен сагынам.
Төшләремдә әле мин еш кайтам
Каз бәбкәле сабый чагыма.

Шул бәбкәләр кебек тезелешеп,
Чыгып киттек гомер юлына.
Хатирәләр калка күңелемдә,
Каз каурые алсам кулыма.

Әбәт алып төшкән әбиемнең
Ипи белән сөтен сагынам.
Төшләремдә әле мин еш кайтам
Каз бәбкәле сабый чагыма.

Урап кайттым чишмә яннарыннан,
Бәбкә саклаучылар эзләдем.
Су буйлары инде тынып калган,
Чирәм баскан учак эзләрен.

Әбәт алып төшкән әбиемнең
Ипи белән сөтен сагынам.
Төшләремдә әле мин еш кайтам
Каз бәбкәле сабый чагыма.
```
