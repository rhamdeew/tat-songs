# Оригинал

### Ленар Шәех / Тәслимә Низами - Мин сине эзләгәнмен

```
Гомер буе бары сиңа,
Мин сиңа табынганмын.
Инде сүндем дигәч
Яңадан кабынганмын.

Өнемдә дә,тәшемдә дә
Мин сине эзләгәнмен.
Син минем язмышым икән,
Үзем дә сизмәгәнмен.

Тормышны яратам диеп,
Мин сине яратканмын.
Сүрелеп барган хисләрне
Кабаттан яңартканмын.

Өнемдә дә,тәшемдә дә
Мин сине эзләгәнмен.
Син минем язмышым икән,
Үзем дә сизмәгәнмен.

Фани дөньяда яшим дип,
Хыялда яшәгәнмен.
Газаплы бәхеттән генә
Берни эндәшмәгәнмен.

Өнемдә дә,тәшемдә дә
Мин сине эзләгәнмен.
Син минем язмышым икән,
Үзем дә сизмәгәнмен.
```
