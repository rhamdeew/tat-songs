# Оригинал

### Галимҗан Гыйльманов / Ганс Сәйфуллин - Сиңа килдем (беренче вариант)

```
Йөрәгемнең чакыруын
Ишетеп килдем менә
Ә син минем күңелемне
Өшетеп көлдең генә.

Килдем әле, килдем әле,
Бар дөньямны онытып.
Йөрим әле, йөрим әле
Синең өчен җан атып.

Каршыларсың дигән идем
Назлы кочагың белән.
Җылытырсын дигән идем
Күңел учагың белән.
Минем мең ут алырлык
Йөрәгеңдә ялкын юк.
Карашыңда сагыш нуры,
Күңелеңдэ салкынлык.

Килдем әле, килдем әле,
Бар дөньямны онытып.
Мәңге шулай йөрермен әле,
Синең өчен җан атып.
```
