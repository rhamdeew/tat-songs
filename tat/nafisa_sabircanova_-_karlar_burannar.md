# Оригинал

### Нәфисә Сабирҗанова - Карлар, бураннар

```
Карлы да, буранлы да
Кышның кыска көннәре.
Кемгәдер бәхет яуган,
Көртләр күмгән кемнәрне.

Кыш җиткән, гомернең дә
Яртысы үтеп киткән.
Без эзләгән ул бәхет
Яшәүнең үзе икән.

Карларда, бураннарда
Калган минем эзләр дә.
Бәхетне танып була
Янып торган күзләрдә.

Кыш җиткән, гомернең дә
Яртысы үтеп киткән.
Без эзләгән ул бәхет
Яшәүнең үзе икән.

Карларда, бураннарда
Йөрим җанны иркәләп.
Үткәнне уйлап яшәргә
Безгә әле иртәрәк.

Кыш җиткән, гомернең дә
Яртысы үтеп киткән.
Без эзләгән ул бәхет
Яшәүнең үзе икән.
```
