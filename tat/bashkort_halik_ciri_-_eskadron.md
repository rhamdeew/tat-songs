# Оригинал

### Башкорт халык җыры - Эскадрон

```
Карасана, дускай, карасана,
Урал тауы тора күгәреп.
Урал тауга карап, бер җырлаек,
Уйларны да ерак җибәреп.

Җырлар яздым Урал кыясына
Истәлектәр булып калсын дип.
Яуга китеп, исән кайталмасам,
Урал тауы искә алсын дип.

Урал гына кошы, ай, сандугач,
Ат менгәндә, моңлы сайрадың.
Кындан алып, алмас кылысымны
Урал ташкаена кайрадым.
```
