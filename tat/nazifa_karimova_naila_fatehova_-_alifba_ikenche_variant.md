# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Әлифба (икенче вариант)

```
Әлиф, бә, тә, сә,
Күңел күзе күрсә,
Җим, ха, хә, дәл,
Маллар булыр хәләл.
Әлиф, бә, тә, сә, җим, ха, хә, дәл.

Зәл, ра, шин, син,
Коръән нуры — Йәсин.
Шин, сад, дад, та —
Изгелекләр кайта.
Зәл, ра, шин, син, шин, сад, дад, та.

За, гайн, гәйн, фә.
Кыл Аллага сәҗдә.
Каф, кәф, ләм, мим,
Сабырлыклар телим.
За, гайн, гәйн, фә, каф, кәф, ләм, мим.

Нун, һә, уау, йә.
Көн узмасын зая.
Йә Раббым, син бар,
Йә Аллаһу әкбәр,
Йә Аллаһу әкбәр.
```
