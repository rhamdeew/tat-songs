# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Бәхтизин

```
Ил өстендә давыл купкач,
Канга таплангач җирләр.
Кыннан суырды кылычын
Бәхтизин кебек ирләр.

Кыйратты яуда дошманны
Уралның бөркетләре.
Онытмый Ватан, онытмый,
Бәхтизин кебекләрне.

Кырып салдылар дошманны,
Кичтеләр утны, суны.
Бәхтизин булды иң алда, —
Гайрәтле башкорт улы.

Кыйратты яуда дошманны
Уралның бөркетләре.
Онытмый Ватан, онытмый,
Бәхтизин кебекләрне.

Яу кырында ятып калды,
Күрмәде җиңү көнен.
Чапай һәм Шайморатовтай,
Ул безнең сафта бүген.

Кыйратты яуда дошманны
Уралның бөркетләре.
Онытмый Ватан, онытмый,
Бәхтизин кебекләрне.
```
