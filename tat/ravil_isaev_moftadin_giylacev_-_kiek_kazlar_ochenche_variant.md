# Оригинал

### Равил Исаев / Мөфтәдин Гыйләҗев - Киек казлар (өченче вариант)

```
Каңгылдашып күктән
Киек казлар үтә.
Казлар күңеле сизә көз аен.
Казлар сагыныр фәкать
Җылы язны гына,
Мин бит сагынам сине көн саен.

Киек казлар күктән
Шуңа күнеп беткән,
Яшәү өчен күчү бер гадәт.
Мин сөюем белән,
Янып яшим гелән,
Януыма бит иркәм син сәбәп.

Яз кояшы балкыр,
Казлар кабат кайтыр,
Туган күлләр өчен талпынып.
Шул казларга карап,
Синсез еллар санап,
Сүнәр микән сөю ялкыным.
```
