# Оригинал

### Динә Талхина / Ришат Сәгыйтов - Җырларым минем кошлардан

```
Аккошкай — сылу кошкай,
Ялгыз җырың өзә үзәкне.
Мин дә шулай ялгыз калсам,
Моңсу җырлар бетә йөзәтеп.

Сандугачкай — сары кошчык —
Сагышыннан моңга күмелә.
Мин дә шулай сагышлардан
Җырлар гына булып түгеләм.

Сабантургай — соры кошчык —
Зәңгәр күктә җырлар йомгагы.
Синең төсле йөрәгемә,
Йөрәгемә җырлар йомганмын
```
