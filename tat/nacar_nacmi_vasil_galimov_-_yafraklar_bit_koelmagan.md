# Оригинал

### Наҗар Нәҗми / Вәсил Галимов - Яфраклар бит коелмаган

```
Көтмәгәндә җир өстенә
Ябырылды ябалак кар.
Бөгелеп төште каеннар,
Шартлап сынды алмагачлар.

Шартлап сынып төште җиргә
Имәннәрнең ботаклары.
Карлар ява, агачларның
Коелмаган яфраклары.

Нинди аклык, нинди сафлык,
Тып-тын, юк бер ыгы-зыгы.
Агачларга нинди авыр
Дөньяның бу матурлыгы.

Ыңгыраша төсле алар —
Кем уйлаган, кем уйлаган.
Карлар, карлар яумагызмы,
Яфраклар бит коелмаган.

Җиргә генә түгел, алар
Миңа да бит ява төсле.
Ак булса да, саф булса да,
Өсләремә ява төсле.

Карлар, карлар яумагызмы,
Күп җырларым җырланмаган.
Яфраклар бит коелмаган,
Саргайса да, коелмаган.
```
