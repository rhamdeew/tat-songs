# Оригинал

### Гөлфия Галимова / Ирек Габдрахманов - Син килдең таңымда

```
Сөю яшь сорамый,
Ул яшькә карамый,
Килә унсигездә, илледә.
Бар да яратылып,
Яратышып яшик,
Каршы килми аңа беркем дә.

Син килдең таңымда
Нур булып җаныма,
Йолдыз булып күктән атылдың,
Якты нурлар сиптең,
Миңа бәхет бөркеп.
Мин йолдызлар булып кабындым.

Минем җанда туган
Якты йолдызчыклар
Матур нурлар сибеп яндылар.
Алларымда бии
Шаян җилләр, дәшеп:
"Бәхетле бу", — диеп калалар.

Син килдең таңымда
Нур булып җаныма,
Йолдыз булып күктән атылдың,
Якты нурлар сиптең,
Миңа бәхет бөркеп.
Мин йолдызлар булып кабындым.

Әл дә бәхетле мин
Бары синең белән,
Бик кадерләп саклыйм бәхетне.
Син мине, и, җаным,
Күкләргә ашырдың,
Матур итеп бизәп тәхетне.

Син килдең таңымда
Нур булып җаныма,
Йолдыз булып күктән атылдың,
Якты нурлар сиптең,
Миңа бәхет бөркеп.
Мин йолдызлар булып кабындым.
```
