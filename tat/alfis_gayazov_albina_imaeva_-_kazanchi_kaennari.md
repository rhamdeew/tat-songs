# Оригинал

### Әлфис Гаязов / Альбина Имаева - Казанчы каеннары

```
Хәтерлисез, онытмыйсыз,
Казанчы каеннары.
Сезнең араларда калган
Беренче адымнарым.

Җырлар белән каршылыйлар
Казанчы каеннары.
Сездә генә серен чишәм
Сез минем туганнарым.

Яшел шәлгә төренгәннәр
Казанчы каеннары.
Ара арасыннан йөри
Яшьлегемнең таңнары.

Җырлар белән каршылыйлар
Казанчы каеннары.
Сездә генә серен чишәм
Сез минем туганнарым.
```
