# Оригинал

### Фирая Зыятдинова / Фәрит Хатыйпов - Йөгереп чыктым урамнарга

```
Йөгереп чыктым урамнарга, —
Таңнар аткан чак иде:
Дөнья әле алсу иде,
Үзәннәргә зәңгәр өфрәк
Томан яткан чак иде.

Йөгереп чыктым урамнарга, —
Кояш батар чак җиткән:
Җиргә моңсу эңгер төшкән,
Бер дә үтмәс төсле иде,
Бу гомер кайчан үткән?!
```
