# Оригинал

### Якуп Колмый / Юлай Үзәнбаев - Син елмайсаң

```
Син елмайсаң көн яктыра,
Яшә шулай көлеп кенә,
Зифа каен кебек кенә,
Тормышка ямь биреп кенә.

Карашыңнаң көн яктыра,
Йөзең кояш кебек миңа.
Шул кояштай көлеп кенә
Яшә җылы биреп кенә.

Канатланып, җырлап яшәп,
Күңел кылын чиртеп кенә.
Ул җыр безне җилкендерсен
Йөрәккә дәрт биреп кенә.

Гөлгә сине тиңләр идем,
Язгы гөлләр иртә шиңә.
Нур өләшеп яшә җирдә
Тормышка ямь биреп кенә.
```
