# Оригинал

### Мәхмүт Хөсәен / Рафаил Бакиров - Бәхетле булыйк! (беренче вариант)

```
Йолдызлар бар җирдә,
Җирдә кояш бар.
Эштә киң җәелә
Безнең колачлар.

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.

Мәхәббәт бар җирдә,
Яну җирдә бар.
Зур юлдаш гомергә —
Ихлас сөйгән яр.

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.

Тынычлык бар җирдә,
Дуслык җирдә бар.
Дуслыкка берни дә
Җитми, туганнар!

Саф торсын ил күге,
Игеннәр уңсын.
Хезмәттә кешеләр
Бәхетле булсын.
```
