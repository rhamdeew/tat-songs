# Оригинал

### Әхсән Баян / Сара Садыйкова - Аңла, дустым

```
Шауласа кичен агачлар,
Йомшак җил кагылудан.
Аңла, дустым, мин җырлармын,
Җырлармын, җырлармын сагынудан.

Тулган ай чыкса һавага,
Агылса саргылт нурлар.
Аңла, дустым, таңга кадәр
Күңелем гел сине, гел сине уйлар.

Елмайса кырлар, авыллар,
Таң нуры сибелүдән.
Аңла, дустым, безгә гел бергә,
Гомергә, гомергә синең белән.
```
