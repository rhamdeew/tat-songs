# Оригинал

### Әхмәт Ерикәй / Шакир Мәҗитов - Синең җырың, Һиндстан

```
Илебезнең гүзәл каласында,
Төрле матур җырлар арасында,
Бер җыр тыңладым мин,
Бер җыр тыңладым мин…
Ул җыр безгә сәлам, булып килгән,
Сездән килгән, Һиндстан!..

Дуслык турындагы уйларыңа,
Чәчәк кебек нәфис җырларыңа
Кушыласы килә,
Кушыласы килә…
Җырың моңлы, күңелләргә җылы,
Йөзең нурлы, Һиндстан!..

Тыңлаганда матур җырларыңны
Күз алдыма китерәм кырларыңны,
Илең җырың сыман,
Җырың илең сыман…
Синең җырың сәлам булып килгән,
Безгә килгән, Һиндстан!..
```
