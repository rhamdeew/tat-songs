# Оригинал

### Азамат Усманов / Салават Низаметдинов - Ак мизгелләр

```
Шомырт күпереп чəчәк атсынмы,
Җирне каплап тəүге кар япсынмы —
Хыял илтə бала чакларыма,
Яшьлегемə, туган якларыма …

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!

Кырпак карда тəүге (яңа) эз калдырып,
Көннəр буе аякны талдырып,
Барлык хəсрəтлəрне оныткандай,
Канатланам яңа җыр откандай.

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!

Тəүге кар, шау чəчəк — ап‑ак мизгел,
Яралы күңелне дәвалый гел.
Актан‑ак, пакътан‑пакъ хыял төсле,
Ак мизгелсез яши алмам төсле …

Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр!
Актан-ак, пакътан-пакъ мизгелләр
Килсеннәр, килсеннәр, килсеннәр!
```
