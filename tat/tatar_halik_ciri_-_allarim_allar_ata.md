# Оригинал

### Татар халык җыры - Алларым аллар ата

```
Алларым аллар ата,
Гөлләрем гөлләр ата.
Сине уйлап ята-ята,
Айлар батып, Чулпан калка.
Сызылып таңнар ата.

Ал чәчәген өздеңме,
Гөл чәчәген өздеңме?
Гөл бирәем дидең мәллә.
Гыйшык утларына салып,
Яндырыйм дидең мәллә.

Ал йөзен аллар ача,
Гөл йөзен гөлләр ача.
Белмим көнне, белмим төнне.
Ниләр генә кылыйм икән.
Хыялый булам ласа.

Алларга ал сан икән,
Гөлләргә гөл сан икән.
Бу мәхәббәт дигән нәрсә
Бер шытып тамыр җибәрсә,
Тәмам исертә икән.
```
