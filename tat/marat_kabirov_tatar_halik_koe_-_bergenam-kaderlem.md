# Оригинал

### Марат Кәбиров / Татар халык көе - Бергенәм-кадерлем

```
Төнге күктә берьялгызы
Сагышланып ай йөзә.
Бергенәм, кадерлем.
Сагышланып ай йөзә, бердәнберем.
Ай да ялгыз, син дә ялгыз,
Мин дә ялгыз мәңгегә.

Бергенәм, кадерлем.
Без дә ялгыз мәңгегә, бердәнберем.

Беребез — анда, беребез — монда,
Язмышларның ярында.
Бергенәм, кадерлем.
Язмышларның ярында, бердәнберем.
Яннарымда булмасаң да
Син һәрвакыт жанымда.

Бергенәм, кадерлем.
Син һәрвакыт янымда, бердәнберем.

Айлар туган, айлар тулган.
Тулган сагышка гына.
Бергенәм, кадерлем.
Тулган сагышка гына, бердәнберем.
Тулган айны охшатамын
Гел үз язмышларыма.

Бергенәм, кадерлем.
Гел үз язмышларыма, бердәнберем.
```
