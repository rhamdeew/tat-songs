# Оригинал

### Роберт Миңнуллин / Рөстәм Зарипов - Чишмә җыры (дүртенче вариант)

```
Челтер-челтер ага чишмә,
Һич тә тыймалы түгел.
Ә сулары шундый йомшак,
Эчеп туймалы түгел,
Эчеп туймалы түгел.

Без киләбез чишмәбезнең
Салкын суын эчмәгә.
Эчкәндә, үзеннән-үзе,
Баш иелә чишмәгә,
Баш иелә чишмәгә.

Чыңлый чишмә, җырлый чишмә
Җилферди зәңгәр толым.
Чишмә җырын тыңлап ята
Зәңгәр толымлы болын,
Зәңгәр толымлы болын.

Чыңла, чишмә, җырла, чишмә,
Тынма, чишмә — чыганак!
Бу болыннар, уйсулыклар —
Сиңа яшел чынаяк,
Сиңа яшел чынаяк.
```
