# Оригинал

### Госман Әхмәтҗанов / Мәсгуть Имашев - Белсәң иде хисләремне…

```
Йөрәктәге хисләремне
Уятма син, уятма син!
Күңелдәге хисләремне
Кузгатма син, кузгатма син!

Белсәң иде күкрәгемдә
Нинди дулкын чайкалганын.
Белсәң иде күкләремдә
Нинди нурлар балкыганын.

Яфрак яра, чәчәк ата
Бәхетемнең алмагачы.
Сайрый анда гүзәл таңда
Мәхәббәтем сандугачы.
```
