# Оригинал

### Зифа Нагаева - Бәхеткә түзеп кара

```
Агы бар, каралары,
Язмышның яралары.
Ах иткән  төннәрнең, дуслар,
Бар икән дәвалары.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.

Җиле бар, томаннары,
Кары бар, бураннары.
Икегә бүлеп җиңәбез
Язмышның сынауларын.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.

Гомер таулары бик текә,
Үрләргә менәсе бар.
Үргә менеп безгә насыйп
Бәхетне күрәсе бар.

Атлыйбыз икәү бергә,
Янымда тиңем бара.
Сагышларга кем дә түзә,
Бәхеткә түзеп кара.
```
