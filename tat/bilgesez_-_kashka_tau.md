# Оригинал

### Билгесез - Кашка тау

```
Туган авылым тау битендә,
Утлары якты яна.
Безне сагынып кайтканны,
Көтеп торалар әнә.

Корсак буе сагындыра,
Сагындыра Кашка тау.
Кайтырбыз сиңа авылым,
Булаек без исән сау.

Авылымнан илһам алам,
Күңелләр канатлана.
Туган җирләр онытылмый,
Йөрәкләрдә саклана.

Корсак буе сагындыра,
Сагындыра Кашка тау.
Кайтырбыз сиңа авылым,
Булаек без исән сау.

Корсак суында коенып,
Чишмәләрдән су эчәм.
Өяңкәләргә, талларга,
Йәшлек серемне чишәм.

Корсак буе сагындыра,
Сагындыра Кашка тау.
Кайтырбыз сиңа авылым,
Булаек без исән сау.

Сандугачлы язгы кичтә
Күктә сердәш ай йөзә.
Сагышлы гармун тавышы,
Әй, үзәкләрне өзә.

Корсак буе сагындыра,
Сагындыра Кашка тау.
Кайтырбыз сиңа авылым,
Булаек без исән сау.
```
