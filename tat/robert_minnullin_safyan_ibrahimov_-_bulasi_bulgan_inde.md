# Оригинал

### Роберт Миңнуллин / Сафьян Ибраһимов - Буласы булган инде

```
Һаман шул булса да йөрәк,
Шул ук булса да күңел.
Мин элекке була алмам —
Син дә элекке түгел.

Икебез генә яраткан
Чәчәкләр сулган инде.
Икебез генә кичергән
Хисләр суынган инде.

Ул чактагы төсле сөю
Башка булмастай инде.
Кабатланмастай инде,
Онытылмастай инде.

Икебез генә яраткан
Чәчәкләр сулган инде.
Икебез генә кичергән
Хисләр суынган инде.

Элекке сөю хисләре,
Күңелдә кабат янмас.
Йөрәкләр кабат янса да,
Мәхәббәт кабатланмас.

Икебез генә яраткан
Чәчәкләр сулган инде.
Икебез генә кичергән
Хисләр суынган инде.
```
