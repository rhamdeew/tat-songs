# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Син миңа барыбер кадерле

```
Нигә соң мине син,
Нигәдер күрмисең?
Шаяртып юри син,
Көлепме йөрисең?

Телгәлә, телгәлә
Син минем бәгырьне.
Син миңа барыбер,
Барыбер кадерле.

Өзелеп сөйгәнне
Үзең дә беләсең.
Синнән дә әйбәт яр
Юктыр ул, күрәсең.

Телгәлә, телгәлә
Син минем бәгырьне.
Син миңа барыбер,
Барыбер кадерле.

Син минем җанымны
Газаптан арындыр.
Син минем беренче
Һәм соңгы ярымдыр.

Телгәлә, телгәлә
Син минем бәгырьне.
Син миңа барыбер,
Барыбер кадерле.
Кадерле…
```
