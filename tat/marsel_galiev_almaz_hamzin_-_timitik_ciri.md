# Оригинал

### Марсель Галиев / Алмаз Хәмзин - Тымытык җыры

```
Яңа бер җыр туды, диеп
Сөенче алыйк әле.
Ык буенда буй үстереп,
Мәңгегә калыйк әле, һәй!

Тым-мы, тым-мы, тым-мы, тымызык ла
Тымытыкның төнбоеклы сулары.
Су буенда кер чайкыйлар
Кызлары ла, кызлары ла, кызлары.

Караталсаң, карат, диләр,
Кызларның асылларын.
Безнең якта бар да сылу,
Сайлыйм соң кайсыларын? һәй!

Тым-мы, тым-мы, тым-мы, тымызык ла
Тымытыкның төнбоеклы сулары.
Су буенда кер чайкыйлар
Кызлары ла, кызлары ла, кызлары.

Болыннарым, колыннарым,
Тургайлы тугайларым.
Күпме генә җырласам да,
Сокланып туялмадым, һәй!

Тым-мы, тым-мы, тым-мы, тымызык ла
Тымытыкның төнбоеклы сулары.
Су буенда кер чайкыйлар
Кызлары ла, кызлары ла, кызлары.

Арыш кыры — дулкын-дулкын,
Күмелә ат дугасы.
Туган җирдә җилләрнең дә
Дәвасы бар, догасы, һәй!

Тым-мы, тым-мы, тым-мы, тымызык ла
Тымытыкның төнбоеклы сулары.
Су буенда кер чайкыйлар
Кызлары ла, кызлары ла, кызлары.

Зәңгәр төтен күтәрелә
Көтүче учагыннан.
Учак утларын сүндермик
Туган җир кочагында, һәй!

Тым-мы, тым-мы, тым-мы, тымызык ла
Тымытыкның төнбоеклы сулары.
Су буенда кер чайкыйлар
Кызлары ла, кызлары ла, кызлары.
```
