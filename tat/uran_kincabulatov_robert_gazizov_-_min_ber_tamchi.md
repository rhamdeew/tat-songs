# Оригинал

### Уран Кинҗәбулатов / Роберт Газизов - Мин — бер тамчы

```
Мин — Җиремдең бер бөртеге,
Бер тамчы саф һавала.
Нур йотканга базлап янам,
Яшәү — яну дөнъяда.

Мин — бер тамчы,
Син — бер тамчы,
Ул — бер тамчы.
Без — диңгез, ярсу дәръя!
Мин — якты нур,
Син — якты нур,
Ул — яҡты нур.
Без — кояш, якты дөнъя!

Ак кар булсам, яңгыр булсам,
Мин җәйгорда идем дә.
Мин — шат тамчы бу дөнъяда,
Мин — икмәктә, игендә!

Мин — бер тамчы,
Син — бер тамчы,
Ул — бер тамчы.
Без — диңгез, ярсу дәръя!
Мин — якты нур,
Син — якты нур,
Ул — яҡты нур.
Без — кояш, якты дөнъя!

Кирәк чакта туган илгә,
Мин булам, ук булам!
Мин — Җирнеке, Җир — минеке,
Бер кайчан да юк булмам!

Мин — бер тамчы,
Син — бер тамчы,
Ул — бер тамчы.
Без — диңгез, ярсу дәръя!
Мин — якты нур,
Син — якты нур,
Ул — яҡты нур.
Без — кояш, якты дөнъя!
```
