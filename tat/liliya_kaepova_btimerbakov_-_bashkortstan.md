# Оригинал

### Лилия Кәепова / Б.Тимербәков - Башкортстан

```
Сүрелмәс йолдыздай балкыйсың,
Рәсәйне җылытыр ялкынcың,
Җанымның кадерле җир генәм,
Күңелемнән дан җыры түгелә.

Уралым тауына сыенган,
Кураем моңына коенган,
Нурлыстан, җырлыстан, гөлестан,
Илгенәм, бергенәм, Башкортстан!

Җигәрле, даланлы халкым бар,
Дөнъяны шаулатыр даның бар,
Галимнәр, шагыйрләр төбәге —
Ил түрен бизәрлек хакың бар.

Уралым тауына сыенган,
Кураем моңына коенган,
Нурлыстан, җырлыстан, гөлестан,
Илгенәм, бергенәм, Башкортстан!

Елгалар, күлләрең, урманың,
Җыр-моңы ургылган кураең,
Эй, горур халкымның уллары,
Дан җырлап мин сезне зурлыймын.

Уралым тауына сыенган,
Кураем моңына коенган,
Нурлыстан, җырлыстан, гөлестан,
Илгенәм, бергенәм, Башкортстан!
```
