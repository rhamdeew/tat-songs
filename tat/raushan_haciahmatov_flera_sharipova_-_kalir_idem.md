# Оригинал

### Раушан Хаҗиәхмәтов / Флера Шәрипова - Калыр идем

```
Кошлар китә, көзләр җиткәч,
Читтә кышлау — алар булмышы.
Кешеләр дә китә җирдән,
Соңгы тапкыр алгач сулышын.

Мөмкин булса калыр идем,
Уйнар өчен ап-ак карларда.
Кышларның бит нуры якты,
Дөнья ямьле шушы нурларда.

Кошлар китә, көзләр җиткәч,
Ишетелә канат кагышы.
Тормыш юлын байкый кеше,
Гомеренең җиткәс баешы.

Мөмкин булса калыр идем,
Уйнар өчен ап-ак карларда.
Кышларның бит нуры якты,
Дөнья ямьле шушы нурларда.
```
