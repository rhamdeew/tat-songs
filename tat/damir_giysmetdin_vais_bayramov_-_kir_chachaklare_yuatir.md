# Оригинал

### Дамир Гыйсметдин / Вәис Бәйрәмов - Кыр чәчәкләре юатыр…

```
Ерак юлга киткән чакта
«Онытма мине!» — дидең.
Дерелдәгән кулың белән
Кыр чәчәкләре бирдең.

Кыр чәчәкләре каршымда,
Сине хәтерләтәләр.
Икәү бергә вәгъдәләшкән
Болыннарга илтәләр.

Безне бәйләгән җепләрне
Тәгәрмәчләр чорныйлар.
Язмышның әче каргышы
Колагымда чыңлыйлар.

Кыр чәчәкләре каршымда,
Сине хәтерләтәләр.
Икәү бергә вәгъдәләшкән
Болыннарга илтәләр.

Аермас безне язмышлар,
Ничек тырышсалар да…
Чәчкәләр юатыр мине,
Кибеп корысалар да.

Кыр чәчәкләре каршымда,
Сине хәтерләтәләр.
Икәү бергә вәгъдәләшкән
Болыннарга илтәләр.
```
