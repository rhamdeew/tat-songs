# Оригинал

### Венера Шәрәфисламова / Зарема Әюпова - Тамчы тама (икенче вариант)

```
Моңсу бер жыр ишетелә
Тып-тып тамган тамчыдан.
Туеп булмый яшәүләрдән,
Гомер бигерәк аз сыман.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.

Шыбыр-шыбыр ява яңгыр,
Түбәдә бии тамчы.
Яшнәп торган яшьлек үтә
Күзләр ачып йомганчы.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.

Тамчы тама, тамчы тама,
Ага яңгыр сулары.
Аккан судай гомер үтә,
Уза яшьлек еллары.

Тамчы тама, тамчы ага
Яңгыр сулары булып.
Шул сулардай үткән гомер
Кайтмый шул болыт булып.
```
