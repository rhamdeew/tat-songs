# Оригинал

### Гөлфия Шакирова / Гүзәлия Вәлиева - Каршы чык

```
Елмаеп бер көл генә,
Күңел гөл генә,
Йөрәккә утлар яккан
Ул бит, җаным, — син генә.
Очрашулар көт кенә,
Кил дип әйт кенә,
Матур тормышның юлын
Минем белән үт кенә.

Каршы, каршы, алдыма
Чык син юлларыма,
Көттем, көттем, диеп кагыл
Минем кулыма, минем кулыма.

Уйларым синең янда,
Йокысыз таңда,
Мәхәббәт һәм син дә мин
Калабыз бу җиһанда.
Каршы ала ай юлы,
Шатлык туп-тулы,
Татлы хыялның бар(ы)сын
Сөйлим сиңа бу юлы.

Каршы, каршы, алдыма
Чык син юлларыма,
Көттем, көттем, диеп кагыл
Минем кулыма, минем кулыма.
Каршы, каршы, алдыма
Чык син юлларыма,
Көттем, көттем, диеп кагыл
Минем кулыма, минем кулыма.
```
