# Оригинал

### Баян Котлыев / Роза Сәхәүетдинова - Кызылгыкүл буйлары

```
Кызылгыкүл сулары,
Яшел тугай буйлары.
Кошлар моңы бөдрә таллар да.

Ташкын сулар аккач,
Балан чәчәк аткач.
Күлгә киләм йөрәк талган да.

Кызылгыкүл язлары,
Табигатьнең назлары.
Күлдә казлар язга куана.

Төшләремдә күрдем,
Син аккошы күлнең.
Сабыр йөрәк көтә, юана.

Кызылгыкүл буйларын,
Күп тирәкле юлларын.
Бергә үтәр идек тагын да.

Бик сагындым иркәм,
Ерак юлга китәм.
Юксынырмын ялгыз чагымда.
```
