# Оригинал

### Татар халык җыры - Авыл көе (Карачтау җыры) (икенче вариант)

```
Агыйделнең парахуты,
Җитә алмый кызыл ярга, шул,
Җитә алмый кызыл ярга.

Җитә алмый, дуслар, җитә алмый
Өзелеп сөйгән ярга.

Агыйдел сулары булсам,
Ургылып агар идем, шул,
Ургылып агар идем.

Яңгыр тамчылары булсам,
Алдыңа тамар идем.

Агыйделнең күпере лә,
Челтер-челтер итүе, шул,
Челтер-челтер итүе.

Сызылып таңнар аткандай,
Гомерләрнең үтүе.
```
