# Оригинал

### Гөлнара Рәшитова / Радик Юлъякшин - Бәхет кошы (икенче вариант)

```
Тылсымлы бәхет кошын,
Үзем бер күрсәм иде.
Төшләремә булсада,
Ул килеп керсә иде.
Ышанамын нигәдер,
Шул кошның барлыгына.
Кайчанда бер мәл килеп,
Ул мине табырына.

Чындырмы ул юктырмы,
Бар диләр бәхет кошы.
Аны күргән һәр кемнең,
Әйләнә икән башы.
Башлары әйләнә дип,
Төрле татлы сәбәптән.
Әкиятти мәхәббәттән,
Көтөлмәгән бәхеттән.

Мөгҗиза белән тулы,
Бу дөнъя без яшәгән.
Шуңада бардыр бик күп,
Серләр бик без белмәгән.
Кабул була хыяллар,
Күңел биреп теләсәң.
Бәхет кошы килердә,
Моңлы җырлар көйләсәм.
```
