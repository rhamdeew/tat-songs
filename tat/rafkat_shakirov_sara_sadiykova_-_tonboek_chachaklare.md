# Оригинал

### Рәфкать Шакиров / Сара Садыйкова - Төнбоек чәчәкләре

```
Су буйларына төштем мин
Төнбоеклар өзәргә.
Яшьлек дустым ерак калды,
Туры килә түзәргә.

Төнбоеклар моңаялар,
Салкын җилләр искәндә.
Мин дә шулай моңаямын,
Син исемә төшкәндә.

Иртән чыклар төшеп күмә
Төнбоек чәчәкләрен.
Аерылган минутларның
Белмәдем сәбәпләрен.
```
