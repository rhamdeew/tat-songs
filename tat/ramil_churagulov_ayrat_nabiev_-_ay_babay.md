# Оригинал

### Рамил Чурагулов / Айрат Нәбиев - Әй, бабай!

```
Алма бакчасына төштем…
Егет булганга күрә.
Күрше абый йоклап ята,
Матур төш күрә-күрә.
Бакчаларга төшкән идем,
Алмалар өзәм диеп.
Пешкән алмалардай матур
Гүзәл кыз торган көтеп.

«Әй, бабай, әй, бабай!
Дус булыйк әле давай!
Кызыңны урлап киттем бит —
Ярамыйга карамый…»

«Кызыңны урлыймын!» — дисәм,
«Ярамый!» — дия идең.
Сүземдә тордым — урладым!
Кояш алды, ай күрде.
Сизмәгәнсең, бакча карап,
Гомереңнең үткәнен.
Алмагачларга ияреп,
Кызың үсеп җиткәнен!

«Әй, бабай, әй, бабай!
Дус булыйк әле давай!
Кызыңны урлап киттем бит —
Ярамыйга карамый…»
```
