# Оригинал

### Гөлшат Зәйнәшева / Илгиз Мәҗитов - Кайтырмын!..

```
Һаваларда болыт,
Ак җилкәннәр булып,
Кая бара, кая агыла?..
Юк, йөрмәмен озак,
Чит жирләрдә чыдап,
Кайтырмын мин туган ягыма!..

Тау башында тургай,
Ялгыз калган бугай,
Очам, диеп, канат кагына.
Юк, йөрмәмен озак,
Чит жирләрдә чыдап,
Кайтырмын мин туган ягыма!..

Дуслар исән микән?..
Хат алалмыйм күптән,
Әллә кемне күңел сагына.
Юк, йөрмәмен озак,
Чит жирләрдә чыдап,
Кайтырмын мин туган ягыма!..

Поездларга карыйм,
Вагоннарын саныйм,
Кайтыйм дисәң, юллар табыла.
Юк, йөрмәмен озак,
Чит жирләрдә чыдап,
Кайтырмын мин туган ягыма!!..
```
