# Оригинал

### Равил Кузнецов - Чын Вәли

```
Авылыма ял итәргә,
Кайтам әле ел саен.
Өй артында мине көтә,
Зифа буйлы ак каен.

Балачагым гөрләп үтте,
Кыр күле буйларында.
Самый чагым туган ягым,
Иң татлы уйларымда.

Хәзер яшим зур шәһәр дә,
Юллар, өйләр таш кына.
Авылыма болыннарга,
Күңелем һаман ашкына.

Балачагым гөрләп үтте,
Кыр күле буйларында.
Самый чагым туган ягым,
Иң татлы уйларымда.

Аерылышу килеп җиткәч,
Туган ягымны бәйли.
Көт син мине мин кайтырмын,
Туган авылым Чын Вәли.

Балачагым гөрләп үтте,
Кыр күле буйларында.
Самый чагым туган ягым,
Иң татлы уйларымда.
```
