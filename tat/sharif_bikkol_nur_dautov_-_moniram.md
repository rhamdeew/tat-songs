# Оригинал

```
Яшьлегемә ямь бирә
Җан бизәгем, Мөнирәм.
Бер көн генә торми торсам
Саргаерга өлгерәм.
Әй, сыулым, Мөнирәм!
Кочак-кочак гөл бирәм.
Күзләремә бер кара син
Назларыңа тилмерәм.

Мөнирәм, Мөнирәм,
Җан бизәгем, Мөнирәм!

Яннарыңа йөгерәм.
Әй, сыулуым, Мөнирәм!

Әй, сыулуым, Мөнирәм!
```
