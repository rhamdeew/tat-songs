# Оригинал

### Лилия Сәлахетдинова / Илдус Дәүләнов - Үзем тезгән бәхет

```
Күрдем дә сөйдем, нәкъ гашыйк идем,
Сөюемнән гел хыялда йөздем.
Берни күрмәдем, "без бит пар", дидем,
Иң зур бәхеткә сине тиңләдем.
Һәр минут саен уема кердең,
Бөтен барлыгым минем биләдең.
Тик син яратмый, уйнап йөргәнсең,
Уенчык диеп мине белгәнсең.

Сөенеч керми ишекне бәреп,
Байлык түгел ул тормышта бәхет.
Саулык, зур гаилә — менә ул бәхет,
Балалы өй ул иң зур рәхәтлек.

Ярый, ачылды шул чак чын йөзең,
Әл дә мин хыянәтеңне белдем.
Бик читен булды, әмма мин түздем,
Кайнар хисләргә бозлы су сиптем.
Бик тиз оныттым — үзең күрәсең,
Әл дә ярый хисләрне күмдең.
Яшибез менә парлы күгәрчен,
Бәхетебез тиң, күзләр тимәсен.

Сөенеч керми ишекне бәреп,
Байлык түгел ул тормышта бәхет.
Саулык, зур гаилә — менә ул бәхет,
Балалы өй ул иң зур рәхәтлек.
Сөенеч керми ишекне бәреп,
Байлык түгел ул тормышта бәхет.
Саулык, зур гаилә — менә ул бәхет,
Балалы өй ул иң зур рәхәтлек…
```
