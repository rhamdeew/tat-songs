# Оригинал

### Башкорт халык / Хисмәтулла Дәүләтов - Җиделе шәм

```
Җиделе лә шәм якты яна
Эчендә мае булса.
Җиде уянып, җиде торып,
Җиде төн уртасында.

Җиделе шәм, мамык милтә,
Ватылды, җиргә төште.
Уф, йөрәгем, янам-көям,
Иркәм исемә төште.

Җиделе шәм, мамык милтә,
Яна да күмерләнә.
Алсу алмам бәгырекәем (Оныта алмаем бәгырькәем)
Онытмам гомергә дә (Оныта алмам гомергә дә).
```
