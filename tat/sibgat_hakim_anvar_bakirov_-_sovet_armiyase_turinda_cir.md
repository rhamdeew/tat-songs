# Оригинал

### Сибгат Хәким / Әнвәр Бакиров - Совет армиясе турында җыр

```
Утларга кердең,
Курку белмәдең.
Ватан йөзенә
Тап төшермәдең.

Кичтең суларны,
Даласын, тавын.
Син ул юк иттең
Дошманнар явын.

Озын һәм авыр
Үтелгән юллар.
Онытмас сине
Килер буыннар.

Ирек байрагын
Югары йөрттең.
Бик күп илләрне
Син азат иттең.

Син узган җирдә
Шаулый игеннәр.
Яшәрде тормыш,
Яшәрде илләр.

Тынычлыкны син
Алып бирүче.
Синең сафларда
Туган ил көче.
```
