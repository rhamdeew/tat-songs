# Оригинал

### Хәмид Мәткәримов / Владимир Федоров - Яшьлек тавышы

```
Безнең әле җитез чаклар,
Яшь тайларны җигәр чаклар.
Шул чакларның кадерен белеп
Яшәп калыйк.
Безнең әле егет чаклар,
Ташка кадак кагар чаклар.
Тимернең дә кызуында
Сугып калыйк.

Яшьлек эзләре артта кала,
Ә мин ашыгам, талпынам алга.
Туктатып булса
Бераз гомер агышын,
Тыңлар идем кабат
Яшьлек тавышын.

Безнең айга менәр чаклар,
Күктән йолдыз өзер чаклар.
Шул йолдызның нурларында
Йөзеп калыйк.
Безнең әле матур чаклар,
Яшь кызларны кочар чаклар,
Шул мизгелләрнең рәхәтен
Татып калыйк.

Яшьлек эзләре артта кала,
Ә мин ашыгам, талпынам алга.
Туктатып булса
Бераз гомер агышын,
Тыңлар идем кабат
Яшьлек тавышын.

Айзирәк тә Байтирәк,
Сезгә бездән ни кирәк?
Үтсә үтсен яшь гомерләр,
Биеп калырга кирәк.
Без бит татар яшьләре,
Юктыр безнең ишләре,
Әйдә, утырма,
Безгә кушылып җырла.

Яшьлек эзләре артта кала,
Ә мин ашыгам, талпынам алга.
Туктатып булса
Бераз гомер агышын,
Тыңлар идем кабат
Яшьлек тавышын.
Яшьлек эзләре артта кала,
Ә мин ашыгам, талпынам алга.
Туктатып булса
Бераз гомер агышын,
Тыңлар идем кабат
Яшьлек тавышын…
Туктатып булса
Бераз гомер агышын,
Тыңлар идем кабат
Яшьлек тавышын…
```
