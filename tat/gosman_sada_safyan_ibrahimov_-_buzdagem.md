# Оригинал

### Госман Садә / Сафьян Ибраһимов - Бүздәгем

```
Язын Бүздөк чияләре
Чәчәк ата күпереп.
Чиядән дә сылу ярым
Чыгар сыман йөгереп.

Туган якның матурлыгын
Йолдызлардан эзләдем.
Йолдызга тиң матурлыклар
Синдә генә, Бүздәгем.

Бүләк иттең яшьлегемә
Сандугачлы таңымны.
Иртәләрен искән җил дә
Үзең кебек ягымлы.

Туган якның матурлыгын
Йолдызлардан эзләдем.
Йолдызга тиң матурлыклар
Синдә генә, Бүздәгем.

Чияләрең алсуланып
Өлгерә җәйгә кергәч.
Бәхет кошы очып килер,
Язмышны бергә үргәч.

Туган якның матурлыгын
Йолдызлардан эзләдем.
Йолдызга тиң матурлыклар
Синдә генә, Бүздәгем.

Күпме чия балкып үсәр
Безнең гомер юлында.
Аппак чәчәк бәйләмнәре
Булыр синең кулыңда.

Туган якның матурлыгын
Йолдызлардан эзләдем.
Йолдызга тиң матурлыклар
Синдә генә, Бүздәгем.
```
