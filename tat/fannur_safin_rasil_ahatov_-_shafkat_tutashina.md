# Оригинал

### Фәннур Сафин / Расил Әхәтов - Шәфкать туташына

```
Кар кызыдай килдең,
Төшләремә кердең,
Коткаручым булып нигәдер.
Елмай әле тагын,
Дөньяның шат чагын
Минем дә күрәсем килә бер.

Елмаерга ләкин
Иртәрәктер бәлки,
Иртәрәктер миңа… кем белә.
Көннәр китсә җирдән,
Төннәр килә икән,
Сердәш булып кала төн генә.

Шәфкать туташыдай,
Кояш сыйпап уза,
Бөтен хәят нурга күмелә.
Елмаерга безгә,
Соң түгелдер бәлки,
Соң түгелдер, туташ… Кем белә?!
```
