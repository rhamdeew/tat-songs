# Оригинал

### Динә Талхина / Айрат Кобагышев - Тагы җәйләр җитте

```
Җылы җәйнең яме —
Җәйгор балкып яна.
Җиргә яңгыр түгел,
Җиргә гәүһәр тама.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Ак чәчәкләр атты аланда.
Чалгы тотып, күз алдында
Син торасың һаман да.

Җылы җәйнең җаны —
Сандугачкай өзә үзәкне.
Җәй тугайларга чигә
Асылларын бизәкнең.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Бер туктаусыз кәккүк чакыра,
Сиңа еллар санаганда
Сакауланган, бахырың.

Җылы җәйнең тәме —
Җир җиләге бал гына.
Сибелеп кенә үскән,
Чиләгеңне ал гына.

Әткәй, әткәй,
Тагы  җәйләр җитте,
Акланкайлар шау гөл генә.
Кайтмаска киттең син,
Җәйләр килер гел генә.
```
