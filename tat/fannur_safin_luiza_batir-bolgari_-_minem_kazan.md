# Оригинал

### Фәннур Сафин / Луиза Батыр-Болгари - Минем Казан

```
Дөнья якты ямьгә күмелгәндәй була,
Нурлар сибә бөтен тирә-якка Казан.
Барча җирләр нур көлтәсе белән тула, —
Казан туды микән әллә гүзәл яздан?

Туган җирнең күркәм батырлыгын җыйган,
Аңа гүя галәм матурлыгы сыйган, —
Яңа биек үрләр яулый Казан,
Һаман шулай гөрли Казан, Казан!

Таңда җилләр сөртеп ала хезмәт тирен,
Тормыш сыман дәртле бер моң көйли Казан.
Уйга талдым: минем туган-үскән җирем —
Наздан, туды микән әллә сүнмәс наздан?

Туган җирнең күркәм батырлыгын җыйган,
Аңа гүя галәм матурлыгы сыйган, —
Яңа биек үрләр яулый Казан,
Һаман шулай гөрли Казан, Казан!
```
