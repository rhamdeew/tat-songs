# Оригинал

### Гыйлемдар Рамазанов / Сара Садыйкова - Башкортстан кызы

```
Йөрәк кебек ярсып, ургып ташып,
Агыйделдә дулкын сикерә.
Бөдрә дулкыннарда уйнап үсте
Талдай зифа буйлы Зөлфирә.

Азмы сылу арасында
Бердәнбер сылу бит ул.
Башкортстан кызы бит ул,
Агыйдел кызы бит ул.

Нәзек таллар сайлап, шашып сайрап,
Сандугачлар җәйне китерә.
Сандугач моңнарын отып алып,
Сөю җырын җырлый Зөлфирә.

Туган илемнең күгендә
Сүнмәс йолдызы бит ул.
Башкортстан кызы бит ул,
Агыйдел кызы бит ул.

Язның ямен күңелләргә салып,
Шомыртлардан хуш ис бөркелә.
Нурлы йөзең күрсәм, Зөлфирәкәй,
Күңелемә назлы җыр килә.

Җырларымның моңы бит ул,
Бәхетемнең нуры бит ул.
Башкортстан кызы бит ул,
Агыйдел кызы бит ул.
```
