# Оригинал

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Мәхәббәт йәйгоры

```
Күзләрем гел синдә, гүзәлем,
Мәхәббәт утында йөрәгем.
Мин көтәм кояшлы яңгырны,
Балкыр дип мәхәббәт йәйгоры.

Зәңгәр су гөмбәзле күңелем,
Мин нигә бәхетле түгелмен.
Яу әйдә кояшлы яңгырым,
Балкытып мәхәббәт йәйгорын.

Ишет син мәхәббәт тавышын,
Карашлар мәңгегә кавышсын.
Яшьлекнең кояшлы яңгыры
Балкытсын мәхәббәт йәйгорын.
```
