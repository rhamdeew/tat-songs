# Оригинал

### Юлия Бәхтиева / Тимур Рамазанов - Туган җирнең тавышы…

```
Уралымның тарихында
Чагыладыр булмышым.
Ата-бабалар сөрәне —
Туган Җирем тавышы.
Гасырларның дарьясыннан
Шушы көчне аламын.
Халкымның асыл улларын,
Ыруымны данлармын!

Зур көч минем канымда,
Халкымда, тарихымда,
Батырларның рухында.
Заманалар булгандыр:
Халкым киңәш коргандыр,
Үз илен данлагандыр.

Чал тарихка илтә йөрәк моңнары,
Мәңге яңгырый халкымның җырлары.
Җиһаннарга илтә йөрәк моңнары,
Мәңге яңгырый халкымның сөрәне!
Йөрәк түрендә йөретәм
Туган җирнең тавышын,
Рухлы тамырларым аша
Тоям Җирнең сулышын.
Үз җиремдә басып торам,
Данлы минем ыруым,
Гасырлардан гасырларга
Илемә сокланырмын!
```
