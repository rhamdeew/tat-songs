# Оригинал

### Муса Сираҗи / Роза Сәхәүетдинова - Йокларга, йокларга!

```
Кояш йөреп еракларда,
Кайтып төште офыкларга
Йокларга, йокларга.

Куяннар да аланнарда
Чаба-чаба арыганнар да
Посканнар, ди, куакларга
Йокларга, йокларга.

Күр, утлар да күзен йомды
Күп йортларда.
Ай да чумды болытларга
Йокларга, йокларга.

Вакыт инде әкияткә
Туктарга, туктарга!
Урын җәям курчакларга
Йокларга, йокларга.
```
