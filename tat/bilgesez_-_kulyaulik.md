# Оригинал

### Билгесез - Кулъяулык

```
Бүгенгенә бүләк алдым,
Җибәргән  чибәркәем,
Мин куандым, бик сөендым,
Өнытмаган гөлкәем.

Ох, җаныем гөлгәнәм,
Җир йөзендә бергенәм
Ох, җаныем гөлгәнәм,
Өнытмаган бергенәм.

Кулъяулыкңын читчитләре
Ефәк белән чигелгән,
Ефәкләр белән чигелгән,
Сөгәнемә бөөргән.

Ох, җаныем гөлгәнәм,
Җир йөзендә бергенәм
Ох, җаныем гөлгәнәм,
Өнытмаган бергенәм.

Синең  истәлегең итеб
Кулъяулыгыңны  үбәм.
Душманнарны барсын җиңеп,
Без кайтырбез гөлгенәм

Ох, җаныем гөлгәнәм,
Җир йөзендә бергенәм
Ох, җаныем гөлгәнәм,
Өнытмаган бергенәм.
```
