# Оригинал

### Мәхмүт Хөсәен / Тәлгать Вәлиев - Ашкыну (икенче вариант)

```
Иделем, гел сиңа ашкынам,
Дулкыннар атылган ярыңа.
Ашкынам, сөенәм… Ә кайчак көенәм,
Тынгылык табалмыйм җаныма.

Иделем, гел сиңа ашкынам.
Суыңда коенган талларга.
Ашкынам, талпынам… Ә үзем тартынам
Күзеңә тутырып карарга.

Иделем, гел сиңа ашкынам.
Җаныма ял бирә таң ату;
Бер рәхәт, бер кыен… Аңарда бар уем —
Шул микән әллә чын ярату?!
```
