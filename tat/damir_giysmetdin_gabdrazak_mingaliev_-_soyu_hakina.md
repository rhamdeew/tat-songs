# Оригинал

### Дамир Гыйсметдин / Габдразак Миңгалиев - Сөю хакына

```
Ничек табыштык без — аңлый алмыйм,
Фәрештәләр ярдәм иткәндер…
Моңа хәтле гөрләвекле язлар
Бездән башка ничек үткәндер?!

Арабызда шактый вакыт ята,
Тик күңелем синең яшьтәшең…
Башкаларга күз салмасын диеп
Йөрим әле көеп-көнләшеп…

Күңелдәге мәхәббәт сазының
Ишетелми иде тавышы…
Син кагылдың аңа, алсу гөлем,
Әллә белеп, әллә ялгышып.

Әй, Җир шары, бераз әкрен әйлән,
Язгы кояш, син дә ашыкма!
Вакытларны тезгенлисе килә
Ошбу сөюләрем хакына…
```
