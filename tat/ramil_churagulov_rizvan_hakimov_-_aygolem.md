# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Айгөлем

```
Синсез, нурсыз көнем,
Ай — йолдызсыз көнем,
Сөю — назлар көтеп үтә гомерем.

Бәхет, язым килер,
Җанга сөю керер,
Килсәң яныма Айгөлем.

Көннәр аяз булсын,
Күңелдә яз булсын,
Мәңге балкып торсын таң йолдызы.

Йөрәгемнән китмә,
Мине ялгыз итмә,
Матур җырла тиң оҗмах кызы.

Айлар, еллар үтәр,
Күпме гөлләр шиңәр,
Син генә сулсмассың сөю гөлем.

Сөю таҗын үрим,
Гомер бергә булыйк,
Килче яныма Айгөлем.
```
