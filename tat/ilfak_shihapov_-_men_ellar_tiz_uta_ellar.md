# Оригинал

### Илфак Шиһапов - Мең еллар (Тиз үтә еллар)

```
Тиз үтә еллар,
Ә күктә без туганда янган йолдызлар,
Һич үзгәрмичә, hаман да шулай ук
Җемелдәп яналар…

Һаман да чылтырый чишмәләр шулай ук таңнарда,
Һаман да ачы әрем исен тарата далада
Җил сыйпап узганда…

Без гел бер уйда,
Әйтерсең бер ялгыз яшибез дөньяда.
Бар могҗизалар була бит бары тик
Без җиргә туганга…

Мең еллар таулардан шулай ук чишмәләр агалар,
Мең еллар күкләрдә шулай ук йолдызлар яналар,
Мәңгелекләр алар…

Ник кыска гомер?
Кичә янган учак буген салкын күмер.
Тик китсәк тә без,
Әрем үсәр, чишмә агар, күк күренер..

Мең елдан чишмәләр шулай ук агарлар таулардан,
Мең елдан язларын әрем исе килер даладан,
Без үткән юллардан…
```
