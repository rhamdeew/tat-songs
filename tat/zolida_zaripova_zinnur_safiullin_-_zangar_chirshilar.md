# Оригинал

### Зөлидә Зарипова / Зиннур Сафиуллин - Зәңгәр чыршылар

```
Зәңгәр күлмәкле чыршылар
Чумган ылыс исенә,
Җанга рәхәт, сихәт бирә
Ап-ак карның төсе дә.

Чыршыкай-матуркай,
Сиңа сузыла кулым.
Туган ягым чыршысына
Илт әле мине юлым.

Чыршыларның зәңгәрлегә
Күзләремә сеңгәннәр.
Авылын ташлап китә икән
Уйсыз-хиссез йөргәннәр.

Чыршыкай-матуркай,
Сиңа сузыла кулым.
Туган ягым чыршысына
Илт әле мине юлым.

Зәңгәр карда зәңгәрме ул?
Ап-ак карлы чыршылар.
Гүзәллеген көйгә салып
Җырлыйлармы җырчылар?

Чыршыкай-матуркай,
Сиңа сузыла кулым.
Туган ягым чыршысына
Илт әле мине юлым.
```
