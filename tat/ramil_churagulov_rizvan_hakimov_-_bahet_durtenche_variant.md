# Оригинал

### Рамил Чурагулов / Ризван Хәкимов - Бәхет (дүртенче вариант)

```
Ник берәүләр елмайганда
Берәүләр көлә икән?
Бәхетле булып яшәүнең
Серләре нидә икән?

Нигә Ходай бәхетләрне
Бертигез бирми икән?
Фәрештәләр өеп-өеп
Өләшеп йөрми икән.

Берәүләргә үзе килә,
Кадерен белмәсә дә.
Ә, берәүләр һич табалмый
Күпмедер эзләсә дә.

Нигә Ходай бәхетләрне
Бертигез бирми икән?
Фәрештәләр өеп-өеп
Өләшеп йөрми икән.

Серләре нидә икән?
Мең яралы күңелләрне
Нурларга күмсәң икән.

Безгә Ходай бәхетләрне
Бертигез бирсә икән?
Фәрештәләр өеп-өеп
Өләшеп йөрсә икән.
```
