# Оригинал

### Мәликә Әминева / Артур Туктаголов - Мине онытырга мөмкин түгел

```
Мине онытырга мөмкин түгел,
Онытырмын диеп уйлама.
Сихерләдем сине, алып сөюемне,
Эчәр суым булыр һавадан.

Китәрмен дә онытырмын диеп
Үз үзеңне генә алдама.
Мине онытырга мөмкин түгел,
Онытырмын диеп уйлама.

Айга баксаң әгәр, анда минем сыным,
Йолдызларда минем карашлар.
Кош очмаса — җилләр алып килер сиңа
Җырларымны минем, тарафтан.

Китәрмен дә онытырмын диеп
Үз үзеңне генә алдама.
Мине онытырга мөмкин түгел,
Онытырмын диеп уйлама.
```
