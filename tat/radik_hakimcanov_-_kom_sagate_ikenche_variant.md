# Оригинал

### Радик Хәкимҗанов - Ком сәгате (икенче вариант)

```
Ком сәгатенең ишеге юк,
Ишетелми тавышлары.
Ком-таш булып вакыт тама,
Сизелми агышлары.

Ком сәгате, ком сәгате
Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.
Ком сәгате, ком сәгате
Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.

Ком сәгате саный һаман
Минутларны, көннәрне.
Нигә ашыктыра икән
Безнең бу гомерләрне?

Ком сәгате, ком сәгате
Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.
Ком сәгате, ком сәгате
Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.

Яшибез без дөньялыкта,
Җил, давыл, карлар ярып.
Үтә дә китә икән ул
Гомер дигәне агып.

Ком сәгате, ком сәгате
Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.

Кайда шулай чабасың?
Күңелләрне сискәндереп,
Сагышларга саласың.
```
