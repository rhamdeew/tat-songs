# Оригинал

### Фәнил Әсәнов / Тәлгать Шәрипов - Исеңдәме? (өченче вариант)

```
Исеңдәме, иркәм,
Таныштык без икәү,
Язын Димдә (җылы яклардан) кошлар кайтканда.
Көннәр аяз иде,
Күңелдә дә яз иде
Яшьлегебез чәчәк атканда.

Гөлләр үскән Димдә
Синең белән бергә
Каршыладык күпме язларны?!
Кошлар җырын тыңлап,
Тугайларны буйлап,
Чәчәкләрдән җыйдык назларны.

Язлар бик тиз үтә,
Димнән (көзен) кошлар китә…
Минутларны куып, Дим ага.
Кошлар белән бергә,
Еллар очып үтә,
Әллә күңел(ем) шуңа моңлана.
```
