# Оригинал

### Филүс Хисаметдинов / Рәдиф Зарипов - Яныңа барам

```
Күңелемдә һәрчак аяз
Янымда син булганда.
Син булмасаң яннарымда —
Күңелем сиңа талпына.

Яннарыңа, яннарыңа,
Яныңа, яныңа үзем барам!
Синең белән күрешүләр,
Бәгърем, көтеп алам!

Сандугачлар тавышы да
Исләремә төшерде
Синең моңлы тавышыңны,
Синең якты йөзеңне.

Яннарыңа, яннарыңа,
Яныңа, яныңа үзем барам!
Синең белән күрешүләр,
Бәгърем, көтеп алам!

Бәгърем, көтеп алам!
```
