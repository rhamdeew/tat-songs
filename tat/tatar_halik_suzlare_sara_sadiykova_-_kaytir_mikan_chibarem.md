# Оригинал

### Татар халык сүзләре / Сара Садыйкова - Кайтыр микән чибәрем

```
Сандугачлар кунып сайрый микән
Су буенда бөдрә талларга.
Сандугачым минем, асыл кошым,
Зифа буйкайларың кайларда.

Сандугачның канатына салып,
Сагыну хатларымны җибәрәм.
Сандугачлар кайткан кебек язын
Кайтыр микән минем чибәрем.

Сандугачкайларның балалары
Басып сайрарга дип тал тапмый.
Газиз җанкайларым нихәл итсен,
Синнән башка, бәгърем, ямь тапмый.
```
