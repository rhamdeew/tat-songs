# Оригинал

### Керәшен халык җыры - Уйлану (туй җыры)

```
Карурманнарны ла, ай, кискәндә,
Таянырга таяклар тапмадым,
Таянырга таяклар тапмадым.

Алдым да лай каләм, яздым сәлам,
Тапшырырга кешеләр тапмадым,
Тапшырырга кешеләр тапмадым.

Тәрәзәләремне ачып куегыз,
Күрдем печәннәрнең дә кипкәнен,
Күрдем печәннәрнең дә кипкәнен.

Төзәтмәсәгез дә, күреп торам
Яшь гомеркәйләрнең дә үткәнен,
Яшь гомеркәйләрнең дә үткәнен.
```
