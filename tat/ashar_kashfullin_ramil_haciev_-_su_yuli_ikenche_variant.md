# Оригинал

### Әсхәр Кәшфуллин / Рәмил Хаҗиев - Су юлы (икенче вариант)

```
Әнә кызлар яшь киленгә
Күрсәтәләр су юлын.
Су чайпала, тамчысын да
Түгә күрмә, сылуым!

Су юллары гөл генә,
Кадерләрен бел генә!

Су юлыңда су анасы
Бүләксез бирми суны.
Алтын тарак, яулык сорый
Су анасы бу юлы.

Су юллары гөл генә,
Кадерләрен бел генә!

Тарагын да, яулыгын да
Бирәбез, әйдә, алсын!
Мөлдерәмә чиләкләргә
Тик бәхет тулып калсын.

Су юллары гөл генә,
Кадерләрен бел генә!

Гармун моңнары агыла,
Су юллары гөл генә!
Сайрашып бергә яшәүнең
Кадерләрен бел генә!

Су юллары гөл генә,
Кадерләрен бел генә!
```
