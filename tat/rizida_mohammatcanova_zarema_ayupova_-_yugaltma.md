# Оригинал

### Ризида Мөхәммәтҗанова / Зарема Әюпова - Югалтма

```
Бәхетеңне балкытырга
Йолдыз булып янармын.
Яңгыр булып явармын да
Юлларыңа тамармын.

Тормыш юлыңда адашсаң,
Юл күрсәтеп барырмын.
Күңелеңә кышлар кунса,
Җылы сүз дә табармын.

Кирәк булса, кояш булып,
Яктыртырмын юлыңны.
Авыр чакта җәйгор булып,
Бизәкләрмен уеңны.

Тормыш юлыңда адашсаң,
Юл күрсәтеп барырмын.
Күңелеңә кышлар кунса,
Җылы сүз дә табармын.

Кирәк булса, синең өчен
Җиһаннан көч алырмын.
Сайлап алдым үзем сине —
Җаныңа моң салырмын.

Бәгърем, чык син сәхрәләргә,
Мин — мәхәббәт акланың.
Язмыш биргән озын юлны
Үтәрбез канатланып.
```
