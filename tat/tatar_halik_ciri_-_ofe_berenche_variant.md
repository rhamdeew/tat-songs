# Оригинал

### Татар халык җыры - Өфе (беренче вариант)

```
Өфенең ак суларында,
Вак балыклар уемда шул,
Вак балыклар уемда.

Алма кебек матур кызлар,
Үсә Өфе буенда.

Өфе кайда, Өфе кайда,
Өфе Урал ягында шул,
Өфе Урал ягында.

Кошлар булып очар идем,
Бик сагынган чагымда.

Өфеләргә барган чакта,
Кара урманнар үтәм шул,
Кара урманнар үтәм.

Сине сагынам, сине (дә) көтәм
Кирәкми миңа бүтән.
```
