# Оригинал

### Клара Булатова / Ифрат Хисамов - Якын дустың барын онытма

```
Тау итәген томан сарган чакта
Җыр яңарса кичке тынлыкта,
Таулар аша юллар узган чакта
Сагынучың барын онытма,
Сагынучың барын онытма.

Иңнәреңә ялгыз күтәрмәстәй
Авырлыклар төшсә тормышта,
Янәшәңә басып, иңен куяр
Якын дустың барын онытма,
Якын дустың барын онытма.

Май кояшы, җылы нурын коеп,
Чәчләреңнән сөйгән минутта,
Ерактан да сине якын тоеп,
Яратучың барын онытма,
Яратучың барын онытма.
```
