# Оригинал

### Сания Гомәрова / Мәсгуть Имашев - Синең кулда язмышы

```
Тормыш гөлем чәчәге
Ярты юлда коелды,
Бәхет җебе урталай
Өзелгәндәй тоелды.
Бәхет җебе урталай
Өзелгәндәй тоелды,
Тормыш гөлем чәчәге
Ярты юлда коелды.

Сине күргәч, бар дөньям
Ямьле нурга төренде,
Тормыш гөлем чәчәге
Сиңа таба үрелде.
Тормыш гөлем чәчәге
Сиңа таба үрелде,
Сине күргәч, бар дөньям
Ямьле нурга төренде.

Үсәрме гөл, шиңәрме? —
Синең кулда язмышы.
Синең белән күрешү
Бәхетме ул, ялгышмы?
Синең белән күрешү
Бәхетме ул, ялгышмы?
Үсәрме гөл, шиңәрме? —
Синең кулда язмышы.
```
