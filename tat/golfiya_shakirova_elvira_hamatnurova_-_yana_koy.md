# Оригинал

### Гөлфия Шакирова / Эльвира Хаматнурова - Яңа көй

```
Алсу  таң  туа,
Яңа  көн  килә.
Туган  җиремә,
Яңа  көй  иңә…

Бу  көйдән,
Сөенеч  тусын.
Бәхет-шатлыклар,
Уртакка  булсын.

Ул  көйне җырлый,
Сайрар кошчыклар.
Биектә  йөзгән,
Зәңгәр  болытлар.

Бу  көйдән,
Сөенеч  тусын.
Бәхет-шатлыклар,
Уртакка  булсын.

Ярсу  елгалар,
Җырлап  кузгала.
Урманнар, кырлар,
Моңга  урала.

Бу  көйдән,
Сөенеч  тусын.
Бәхет-шатлыклар,
Уртакка  булсын.
```
