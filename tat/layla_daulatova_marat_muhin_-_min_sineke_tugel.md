# Оригинал

### Ләйлә Дәүләтова / Марат Мухин - Мин  синеке түгел

```
Син акыллы,
Син бик әйбәт,
Син бик асыл Ир-егет.
Сине күргәч
hич, Бер кыз да
Түзеп торалмас кебек.
Минеке бул дисең миңа
Кулымнан тотасың да.
Тик мин беләм иртәгәчә
Син мине онытасың да.

Тик мин синең
Меңнән — берең
Меңнән — берең булганчы
Берәрсенең бердәнбере
Булуны артык күрәм
Тик мин синең
Меңнән — берең
Меңнән — берең булганчы
Берәрсенең бердәнбере
Булуны артык күрәм.
```
