# Оригинал

### Мөхәммәт Закиров / Лира Фатыйхова - Сирень бураны

```
Адашыпмы килеп чыктык
Яр буе урамына,
Көтмәгәндә торып калдык
Сиреньнәр буранына.

Адашыпмы килеп чектык
Яр буе урамына,
Ялгыз гына карап торам
Сиреньнәр буранына.

Койма аша таша иде
Күк сиреньнар күбеге,
Талпынса да, сиреньнарнең
Юктыр синең кебеге.

Адашыпмы килеп чектык
Яр буе урамына,
Ялгыз гына карап торам
Сиреньнәр буранына.

Һавадагы купшы кошка
Күңелегезне салмагыз.
Гомер буе сызланырлык
Җан ярасы алмагыз.

Адашыпмы килеп чектык
Яр буе урамына,
Ялгыз гына карап торам
Сиреньнәр буранына.
```
