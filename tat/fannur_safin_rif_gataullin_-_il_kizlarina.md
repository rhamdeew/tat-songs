# Оригинал

### Фәннур Сафин / Риф Гатауллин - Ил кызларына

```
Чәчәк эзләп йөрим кырларда,
Белмим сезне ничек зурларга.
Теләп сезгә изге теләкләр,
Йолдыз булып яна йөрәкләр.

Балкый алда еллар,
Без барасы юллар.
Кала якты нурлар
Сезнең күзләрдә.

Кайгыларны сыйпап аласыз,
Хәерле юл теләп каласыз.
Сез бит шундый назлы һәм чыдам,
Сезне уйлап юлга мин чыгам.

Күзләрегез көлеп торсын гел,
Юлыгызга яусын алсу гөл.
Сезнең өчен якты җырларым,
Данлыйм илнең батыр кызларын.
```
