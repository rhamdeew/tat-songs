# Оригинал

### Билгесез / Ринат Гобәйдуллин - Мәхәббәт җыры (сигезенче вариант)

```
Яшерәсең, әллә кош тоттыңмы?
Ачып күрсәт әле учыңны.
Сандугачмы,карлыгачмы икән,
Әллә очып китәр дисеңме?

Ул кош миннән, күңел бакчасыннан,
Кереп кара, әгәр теләсәң.
Синең өчен генә сайратырмын,
Җең белән түләсәң.

Сөенәсең, әллә төш күрдеңме?
Сөйләп күрсәт әле төшеңне.
Сөйлә инде, нидән шикләнәсең,
Әллә юрый белмәс дисеңме?

Сөенәм мин чәчләремне сыйпап,
Күзләремә сөеп карасаң.
Төшләремне сөйләргә дә риза,
Әгәр бәхетемә юрасаң.
```
