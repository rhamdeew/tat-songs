# Оригинал

### Гөлфия Шакирова / Фирзәр Мортазин - Изге теләкләр (беренче вариант)

```
Без бит кирәк
Бер-беребезгә,
Бергә ямьле, күңелле.
Якыннарың янда булу
Сөендерә күңелне.

Бәхет-шатлык дулкынында
Калыйк әле гомергә.
Елмаюлар бүләк итеп
Яшик шулай гел бергә.

Еллар узып торса торсын,
Картаймасын күңелләр.
Сөю-шатлык нурларыннан
Балкып торсын гел йөзләр.

Бәхет-шатлык дулкынында
Калыйк әле гомергә.
Елмаюлар бүләк итеп
Яшик шулай гел бергә.

Акның карасы булган күк
Төннең дә була көне.
Яшәү бәхет дигән уйлар
Ташламасын ла безне.

Бәхет-шатлык дулкынында
Калыйк әле гомергә.
Елмаюлар бүләк итеп
Яшик шулай гел бергә.
Бәхет-шатлык дулкынында
Калыйк әле гомергә.
Елмаюлар бүләк итеп
Яшик шулай гел бергә…
Елмаюлар бүләк итеп
Яшик шулай гел бергә…
```
