# Оригинал

### Ленар Бикмурзин - Гармун уйныйм мин

```
Балачактан гармун уйнап,
Җырлап үстем мин.
Җитмәсәмдә озын буйга
Аптырамим мин.

Өзәкләрегез өзелсен
Сәбәбчесе кем?
Кем булсын инде сизелә
Гармун уйныйм мин.

Уйнамый түзеп буламы?
Моңга салышкач.
Гармуннан бизеп буламы?
Кулга ябышкач.

Өзәкләрегез өзелсен
Сәбәбчесе кем?
Кем булсын инде сизелә
Гармун уйныйм мин.

Кызлар кочкандай күкерәккә
Кочам билләрен.
Яндырсын әле йөрәкне
Көмеш телләре.

Өзәкләрегез өзелсен
Сәбәбчесе кем?
Кем булсын инде сизелә
Гармун уйныйм мин.
```
