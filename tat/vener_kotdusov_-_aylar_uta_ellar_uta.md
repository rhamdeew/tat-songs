# Оригинал

### Венер Котдусов - Айлар үтә, еллар үтә

```
Бер күз ачып йомган ара,
Гомерләр узып бара.
Сагыш-хәсрәт елап кала,
Ә бәхет алдан чаба.

Айлар үтә, еллар үтә,
Гомерләр үтә китә.
Гомерләр үтә китә дә,
Язганы безгә җитә.

Балачагым — назлы чагым —
Елмая еллар аша.
Җан картаймый, гомеркәем,
Узсаң да, узмасаң да.

Айлар үтә, еллар үтә,
Гомерләр үтә китә.
Гомерләр үтә китә дә,
Язганы безгә җитә.

Агыйделдә боз таша да,
Уйсу җирне су баса.
Очрашканда уйна да көл,
Аерылгач сагыш баса.

Айлар үтә, еллар үтә,
Гомерләр үтә китә.
Гомерләр үтә китә дә,
Язганы безгә җитә.
```
