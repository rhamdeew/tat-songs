# Оригинал

### Флария Терегулова / Фирзәр Мортазин - Илтсен сиңа юлларым

```
Чыклы, чәчкәле болында
Таңнар аттырсак икән.
Сандугачлар да кавыша,
Без дә кавышсак икән.

Сагышлы, сагышлы
Икән барыр юлларым.
Барысына да түзәрмен,
Илтсен сиңа елларым.

Язлар белән гөрләвектәй.
Син миңа килсәң икән.
Серле сукмак сине миңа,
Мине сиңа илтсә икән.

Сагышлы, сагышлы
Икән барыр юлларым.
Барысына да түзәрмен,
Илтсен сиңа елларым.

Болын чәчкәләрен жыеп.
Юлыма чыксаң икән.
Кочакларыңа атылып,
Эреп югалсам икән.

Сагышлы, сагышлы
Икән барыр юлларым.
Барысына да түзәрмен.
Илтсен сиңа елларым.
```
