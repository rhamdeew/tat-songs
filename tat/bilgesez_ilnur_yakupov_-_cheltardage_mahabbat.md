# Оригинал

### Билгесез / Илнур Якупов - Челтәрдәге мәхаббәт

```
Сорау — җавап,
Җавап — сорау,
Инде соң, инде иртә,
Урамда көзге кырау,
Урамнар мине көтә.
Көн буе сине уйлыйм
Һәм көтәм очрашуны,
Сүзләрне сиңа сайлыйм,
Хәрефтән тезәм аны.

Челтәрдәге мәхаббәт —
Безнең яшерен элемтә,
Челтәрдәге мәхаббәт —
Галәм көтә.
Челтәрдәге мәхаббәт,
Кавыштыра USB.
Очрашырбыз микән?
Maybe..

Сорау — җавап,
Җавап — сорау,
Мин тагын бу челтәрдә,
Диңгездә ялгыз утрау,
Тәрәзә зур шәһәрдә.
Йолдызлар миңа:
“Син кайт”,
Сүзләрең миңа:
“Син кал”
Һәм хисләр күп мегабайт,
Очрашу минем хыял.

Челтәрдәге мәхаббәт —
Безнең яшерен элемтә,
Челтәрдәге мәхаббәт —
Галәм көтә.
Челтәрдәге мәхаббәт,
Кавыштыра USB.
Очрашырбыз микән?
Maybe..
```
