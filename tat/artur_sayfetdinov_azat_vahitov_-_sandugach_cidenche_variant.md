# Оригинал

### Артур Сәйфетдинов / Азат Вахитов - Сандугач (җиденче вариант)

```
Кунасың да моңланасың,
Сандугачым, талларга.
Кемне шулчак юксынасың
Көн дә алсу таңнарда?

Эй, сандугач, әйтче серең,
Кем өчен өзелүең?
Бәлки, мин дә, җырлый-җырлый,
Юатырмын күңелең.

Сандугачым тапты ярын,
Сабыр булгач йөрәге.
Мин сандугач түгел, бәлки,
Син бит күңелем терәге.

Эй, сандугач, әйтче серең,
Кем өчен өзелүең?
Бәлки, мин дә, җырлый-җырлый,
Юатырмын күңелең.

Сагынуымны басар өчен,
Мин дә киләм талларга.
Таллар елый, ә мин җырлыйм
Сандугачтай таңнарда.

Эй, сандугач, әйтче серең,
Кем өчен өзелүең?
Бәлки, мин дә, җырлый-җырлый,
Юатырмын күңелең.
```
