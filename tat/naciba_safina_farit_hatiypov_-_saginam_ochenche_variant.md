# Оригинал

### Нәҗибә Сафина / Фәрит Хатыйпов - Сагынам (өченче вариант)

```
«Ни хәлләрең бар синең?» — дип,
Соравыңны сагынам.
Күңелемә сеңеп калган
Моңнарыңа табынам.
Сагынам, сагынам,
Моңнарыңа табынам.
Сагынам.

Гомеремнең юлларында,
Минут саен абынам.
Ә шулай да «очамын» дип,
Һаман күккә талпынам.
Сагынам, сагынам,
Һаман күккә талпынам.
Сагынам.

Галәм аша кулларымны
Сиңа сузам — зарыгам.
Чәчләреңә, күзләреңә,
Күңелеңә кагылам…
Сагынам, сагынам,
Күңелеңә кагылам…
Сагынам.
```
