# Оригинал

### Мәхмүт Хөсәен / Ренат Еникеев - Ләйсән (беренче вариант)

```
Ләйсән, диләр,
Ләйсән, диләр,
Ләйсән — язның яңгыры.
Дәртле яздан дәрман алып,
Дөрли йөрәк янгыны!..

Ярым, дисәң,
Янып сөйсәң,
Хәтәр түгел миңа үлем дә!..
Синең исем,
Синең хисең
Минем өчен ләйсән күңелемдә.

Җир яшәрә,
Кыр ямьләнә,
Ләйсән җылы таратса.
Икеләтә дөнья матур,
Өзелеп ярым яратса.

Бел син, бәгырем,
Сөю кадерен,
Әйләнмәсен йөрәк күмергә.
Синең исем,
Синең хисең
Минем өчен ләйсән гомергә.
```
