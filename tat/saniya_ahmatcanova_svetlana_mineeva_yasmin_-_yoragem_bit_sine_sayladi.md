# Оригинал

### Сания Әхмәтҗанова / Светлана Минеева (Ясмин) - Йөрәгем бит сине сайлады

```
Ни сәбәпле мине яратасың, дип
Сорама син, кабат сорама.
Сине гашыйк иткән сәбәпләр эзләп,
Йөзләремә сынап карама.

Яратам мин сине, бары яратам,
Җаннарыма якын күргәнгә.
Яшерен серләремне ачам ышанып,
Синең яратканны белгәнгә.

Серләреңне сорап торасы да юк,
Укып була, дисең, күзеңнән.
Энҗеләре тирән, ә күзләремдә
Чайпалганы гына күңелдән.

Яратам мин сине, бары яратам,
Җаннарыма якын күргәнгә.
Яшерен серләремне ачам ышанып,
Синең яратканны белгәнгә.

Буйсынмыйча миңа, ихлас хисләрем
Җырлар булып чыга күкрәктән.
Йөрәгем бит сине, сине сайлады,
Сәбәпләрен сора йөрәктән.

Яратам мин сине, бары яратам,
Җаннарыма якын күргәнгә.
Яшерен серләремне ачам ышанып,
Синең яратканны белгәнгә.
```
