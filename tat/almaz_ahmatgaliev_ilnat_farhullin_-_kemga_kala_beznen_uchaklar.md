# Оригинал

### Алмаз Әхмәтгалиев / Илнат Фәрхуллин - Кемгә кала безнең учаклар?

```
Чайкала да дөнья чайкала,
Кояш сүнә, йолдызлар яна.
Сөелмәгән күпме бәгырьләр
Тик сагынып үткәндә кала.

Кемгә кала безнең учаклар,
Сылу кызлар, кайнар кочаклар?
Үбеп аның татлы иренен,
Йокламыйча төн чыккан чаклар?
Кемгә кала безнең учаклар?

Ерак кала сине сөйгәннәр,
Ерак кала минем сөйгәннәр.
Шәм кабызып нурсыз бүлмәдә
Бер ялгызы гына көйгәннәр.

Кемгә кала безнең учаклар,
Сылу кызлар, кайнар кочаклар?
Үбеп аның татлы иренен,
Йокламыйча төн чыккан чаклар?
Кемгә кала безнең учаклар?

Чайкала да дөнья чайкала,
Кояш сүнә, йолдызлар яна.
Гашыйк чакның аулак урыны,
Әйтче, язмыш, кемнәргә кала?

Кемгә кала безнең учаклар,
Сылу кызлар, кайнар кочаклар?
Үбеп аның татлы иренен,
Йокламыйча төн чыккан чаклар?
Кемгә кала безнең учаклар?
Кемгә кала безнең учаклар?..
```
