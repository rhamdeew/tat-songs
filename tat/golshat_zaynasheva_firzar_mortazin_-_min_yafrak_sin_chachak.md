# Оригинал

### Гөлшат Зәйнәшева / Фирзәр Мортазин - Мин — яфрак, син — чәчәк

```
Һәркемнең дә үз чәчәге бар,
Табигать шуңа күнеккән.
Без дә бит табигать баласы,
Табигать гөле иркәм.

Пар килгәннәр болар дип,
Соклансыннар башкалар да.
Мин — яфрак, син — чәчәк булып
Тибрәлик бакчаларда.

Һәр таңның да үз нурлары бар,
Иртәләр шуңа күнеккән.
Без дә бит таң нурларыннан
Бу җиргә иңдек, иркәм.

Пар килгәннәр болар дип,
Соклансыннар башкалар да.
Мин — яфрак, син — чәчәк булып
Тибрәлик бакчаларда.

Һәр җырның да үз көе бар,
Җыр белән тормыш күркәм.
Тормышыбыз күркәм булсын,
Парлап җырлыйк без, иркәм.

Пар килгәннәр болар дип,
Соклансыннар башкалар да.
Мин — яфрак, син — чәчәк булып
Тибрәлик бакчаларда.
```
