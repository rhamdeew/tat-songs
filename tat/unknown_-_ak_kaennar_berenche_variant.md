# Оригинал

### Кәүсәрия Шәфикова / Гөлнара Зиязова - Ак каеннар (беренче вариант)

```
Җәй буена каеннарга
Серләремне сөйләдем.
Талгын җилгә кушылдым да,
Сагынуымны сөйләдем.

Каеннарны сагышсыз, дип
Уйлап ялгышканмын ла.
Язмышлары тиң икән бит
Кеше язмышларына.

Ак каеннар тын тордылар,
Сабыр иткәннәр икән.
Үз хәлләрен сөйләр өчен
Көзне көткәннәр икән.

Каеннарны сагышсыз, дип
Уйлап ялгышканмын ла.
Язмышлары тиң икән бит
Кеше язмышларына.
```
