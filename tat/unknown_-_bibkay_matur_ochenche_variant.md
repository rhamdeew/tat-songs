# Оригинал

### Татар  халык җыры - Бибкәй матур (өченче вариант)

```
Су алырга барсаң бар иртәрәк
Соңга калсаң, сулар ла болгана.
Бибкәем, аерылам ла
Күпкә түгел, күпкә түгел,
Күпкә түгел, хәзергә.
Шушы көннән үлгәнчегә
Бибкәй туйган без бергә.
Шушы көннән үлгәнчегә
Бибкәй туйган без бергә.

Эчкенәең пошса, чык сәхрәгә,
Сәхрәләрдә кошлар ла моңлана.
Бибкәем, алдама ла;
Газиз башым, газиз башыма
Янган утларга салма.
Газиз башым, газиз башыма
Янган утларга салма.
```
