# Оригинал

### Татар халык җыры - Питрәч «тала»сы

```
Питрәч кайда, Питрәч кайда, —
Питрәч Казан ягында;
Кошлар булып очар идем
Бик сагынган — чагымда.

Эх, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!
Талмас иде беләккәем,
Талдыручы булмаса;
Янмас иде йөрәккәем,
Яндыручы булмаса.

Без ал идек, без гөл идек,
Кайнаша торган идек;
Җыен дуслар бергә килеп,
Сайраша торган идек.

Эх, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!
Талмас иде беләккәем,
Талдыручы булмаса;
Янмас иде йөрәккәем,
Яндыручы булмаса.

Челтәрле күпер астында
Ак чиләгем күмелде.
Әйдә, дуслар, барыйк әле —
Питрәч ягы күңелле.

Эх, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!
Талмас иде беләккәем,
Талдыручы булмаса;
Янмас иде йөрәккәем,
Яндыручы булмаса.

Питрәчләрдә сигез күпер,
Берсеннән чыгасым юк.
Бәгырькәем, һичбер вакыт
Исемнән чыгасың юк.

Эх, тала, тала, тала,
Тала ике беләгем.
Нигә тала, нигә яна
Икән минем йөрәгем?!
Талмас иде беләккәем,
Талдыручы булмаса;
Янмас иде йөрәккәем,
Яндыручы булмаса.
```
