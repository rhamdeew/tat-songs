# Оригинал

### Татар халык җыры - Фирдәвескәй

```
Кашын кара, буең зифа,
Керфегең бөгеләдер;
Синең өчен җанкисәгем,
Фирдәвескәй, кара күзкәй,
Үзәгем өзеләдер.

Каш карасы күзеңдә лә,
Буй зифасы үзеңдә.
Ай туганда тудың мәллә,
Фирдәвескәй, кара күзкәй,
Ай нуры бар йөзеңдә.

Кашың кара, күзең күк,
Сиңа сөйләр сүзем күп;
Сөйләр идем дә сүземне,
Фирдәвескәй, кара күзкәй,
Сирәк күрәм үзеңне.
```
