# Оригинал

### Ринат Мөслимов / Рим Хәсәнов - Сыерчыклар

```
Өметем өзелгәч,
Кайгыдан түзмәдем,
Ни эшләргә белми,
Киңәшче эзләдем.

Сез очып кундыгыз
Тәрәзәм төбенә,
Китәбез инде, дип
Әйттегез үземә.

Сыерчыклар, сыерчыклар,
Канатлы дусларым,
Бирегез киңәш сез,
Ярамны төзәтми
Ник ташлап китәсез?

Көзләрен саргаеп,
Мин сезне озатам.
Килерсез әле дип,
Күңелне юатам.

Киткәндә еракка
Алыгыз искә сез;
Мин калдым бу якта
Ялгызым, иркәсез.

Сыерчыклар, сыерчыклар,
Канатлы дусларым,
Бирегез кинәш сез,
Ярамны төзәтми
Ник ташлап китәсез?
```
