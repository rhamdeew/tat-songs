# Оригинал

### Рагидә Янбулатова / Сара Садыйкова - Әйт, дускаем, әйт!

```
Дим өстендә ай нурлары
Йөзә моңлы җыр булып,
Төнбоеклар тирбәләләр
Ак чәчәкләрен йомып,

Йөрәгем яна, ашкына күңел,
Бу — бер дә юкка, тикмәгә түгел,
Бу — тәүге сөю, тәүге мәхәббәт.
Миңа нишләргә, әйт, дускаем, әйт!

Тонык Димдә толымнарын
Юа тал иркәләнеп,
Төнбоеклар уянырга
Көтә таң, иртәләрен.

Йөрәгем яна, ашкына күңел,
Бу — бер дә юкка, тикмәгә түгел,
Бу — тәүге сөю, тәүге мәхәббәт.
Миңа нишләргә, әйт, дускаем, әйт!
```
