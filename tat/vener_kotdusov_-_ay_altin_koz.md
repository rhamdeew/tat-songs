# Оригинал

### Венер Котдусов - Әй, алтын көз

```
Урамнарда йөри алтын көз,
Сайрар кошлар, кайларда сайрыйсыз?
Көзге күктә ай, ата йолдызлар,
Күктәге йолдыз, җирдә гашыйклар.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.

Сайрар кошлар тезелеп очтылар,
Юксынма син, язын кайтырлар.
Язын кошлар, сине сагынып,
Тәрәзәңә килеп кагарлар.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.

Урамнарда инде җәй түгел,
Сагындыр, көз, тик һаман яшь күңел.
Урамнарда йолдызлар кебек
Чибәр кызлар йөри нур сибеп.

Әй, алтын көз, янар йолдызларым,
Ярсып ата, юрый хыялларым:
Әй, гашыйклар, әйтегез теләкләр,
Калмасын дип ялгыз йөрәкләр.
```
