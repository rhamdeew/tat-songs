# Оригинал

### Уран Кинҗәбулатов / Мөфтәдин Гыйләҗев - Кайтыр юллар бар

```
Әйди матур тормыш,
Ялгышмаса язмыш,
Безгә дә бер кайтыр юл ятыр…
Безнең яшьлек җыры —
Язгы кояш нуры
Бөреләрне назлап уятыр.

Үтәр айлар, еллар,
Яшел ак каеннар
Саргаерлар безне сагынгач.
Якты сөю хисен
Сайрар иртә, кичен
Безне гашыйк иткән сандугач.

Туып үскән җирдә
Дуслар белән бергә
Гөрләшербез әле бер заман.
Яшьлек тугаена,
Сабан туйларына
Кайта алмасак булыр бик яман.
```
