# Оригинал

### Рәфкать Шәрәфетдинов / Наил Табанаков - Әти, көтмим сине күптән

```
Бер ялгызы бала үстерүе
Җиңел булмаган шул, булмаган.
Ничек кенә түзгән әнием, диеп
Мин әле дә, әле дә уйланам.

Еллар үткән инде, мин дә үскән,
Көтмим инде, әти, сине бүтән.
Тик әнием генә күпме гомер
Сине генә, бары сине көткән.

Ачы җиле булган, бураны да,
Гайбәт сүздән күпме елаган.
Үз баласын ятим калдырганда
Әти кеше ни соң уйлаган?
Ә шулай да бер күрәсе иде,
Ишетергә иде җавабын.
Соң булса да, хәлен аңлар идең
Әти көтеп үскән баланың.

Еллар үткән инде, мин дә үскән,
Көтмим инде, әти, сине бүтән.
Тик әнием генә күпме гомер
Сине генә, бары сине көткән.
```
