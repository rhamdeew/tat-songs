# Оригинал

### Шәриф Биккол / Ризван Хәкимов - Бозма вәгъдәләреңне

```
Сөюләрем көчле,
Һәтәр яшен төсле,
Сак бул, син, ут белән уйнама!
Вәгъдәләрең бозсаң,
Ятларга кул сузсаң,
«Гафү итәр бу!» — дип уйлама.

«Тиң, түгел бит, ул, — дип, —
Миңа гына булл!» — дип,
Хатлар язып сине юл чыкмам,
Төнең тәрәз чиртеп,
«Күрим әле бер!» — дип,
Чакырырсың,
Ә мин, мин чыкмам.
Төнең тәрәз чиртеп,
«Күрим әле бер!» — дип,
Чакырырсың,
Ә мин, мин чыкмам.

Мен газапка түзеп,
Мине дөнья гизеп,
Эзләрсең син,
Ә мин табылмам,
Мин табылмам көтмә,
Синең өчен күктә,
Балкыр йолдыз булып кабынмам.
Мин табылмам көтмә,
Синең өчен күктә,
Балкыр йолдыз булып кабынмам.
```
