# Оригинал

### Ризида Мөхәммәтҗанова - Киләчәк

```
Алтын бодай яланнарда
Гөрләп кенә үсәчәк.
Матур булсын, якты булсын
Һәрчак безнең киләчәк.

Кушымта:
Киләчәккәй, киләчәк шул
Коштай очып киләчәк.
Бодайкайлар үскән кебек,
Җирдә бәхет үсәчәк.

Безнең йөрәк гел моң гына,
Киләчәккә җыр уйлап,
Бөдрә таллар, юл күрсәтеп,
Тезелгәннәр су буйлап.

Кушымта.

Сандугачлар сайраганга
Ага елга җыр җырлап.
Сине сайлап алдым, бәгърем,
Утырам бәхет юрап.

Кушымта:
Киләчәккәй, киләчәк шул,
Коштай очып киләчәк.
Саф чишмәләр кебек
җырлы
Булса иде күрәчәк!
```
