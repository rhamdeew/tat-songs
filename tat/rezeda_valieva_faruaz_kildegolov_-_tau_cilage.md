# Оригинал

### Резеда Вәлиева / Фәрүәз Килдеголов - Тау җиләге

```
Тау буенда йөрдем җәен,
Тау җиләгем эзләдем.
Таптым үзем бер дигәнен,
Таптым, ләкин өзмәдем.
Җиләкләр пешкән икән
Алланып төшкән икән.
Җиләк кебек бу кыз бала
Кайларда үскән икән?

Чибәрнең дә чибәре,
Кулларында чиләге.
Бу ни бу? — дип сораганга,
— Бу, — диде, — тау җиләге.

— Тулган бит соң чиләгең,
Җыйма инде, чибәрем.
Кунак егет ялгыз йөри,
Бармы синең йөрәгең?..
Көлде кызый, түзмәде,
Алсуланды йөзләре.
Каш астыннан кыйгач карап
Ул да мине күзләде…

Чибәрнең дә чибәре,
Кулларында чиләге.
Бу ни бу? — дип сораганга,
— Бу, — диде, — тау җиләге.

Җәйләр үтте, көзләр җитте,
Туйлар иттек көзләрдә.
Инде бергә киләбез без
Тау җиләген эзләргә.
Җиләкләр уңа икән,
Алланып тула икән.
Тау җиләген тапкан егет,
Бәхетле була икән!

Чибәрнең дә чибәре,
Кулларында чиләге.
Бу ни бу? — дип сораганга,
— Бу, — диде, — тау җиләге.
```
