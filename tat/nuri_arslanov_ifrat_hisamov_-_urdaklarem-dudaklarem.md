# Оригинал

### Нури Арсланов / Ифрат Хисамов - Үрдәкләрем-дүдәкләрем

```
Яктырып уянды таң.
Вак-вак атлап сукмактан
Китте чибәр Нәсимә
Үрдәкләр фермасына.

Үти-үти, үрдәкләрем,
Ап-акларым, күмәкләрем,
Нихәлсез, дүдәкләрем.

Ак алъяпкыч, ак яулык,
Үрдәкләре ак мамык;
Шаулашып төшеп суга
Йөзделәр чума-чума.

Үти-үти, үрдәкләрем,
Матурларым, күмәкләрем,
Уйнагыз, дүдәкләрем.

Ә егет торган саклап:
Кызны сарды кочаклап,
Чибәр еш-еш тын алды,
Кояш төсле кызарды…

Үти-үти, үрдәкләрем,
Үрдәкләрем-дүдәкләрем,
Талпына йөрәкләрем…

Барлык үрдәкләр шул чак
Кычкырды күреп: — Бак, бак!
Бактың ни, бакмадың ни,
Ярлар бар дип тә белми.

Үти-үти, үрдәкләрем,
Үрдәкләрем-дүдәкләрем,
Эреде йөрәкләрем.

Күңел үз уен итте,
Яшьлек үз туен итте…
Җиргә нурлар төштеләр —
Чыклар да үбештеләр…

Үти-үти, үрдәкләрем,
Үрдәкләрем-дүдәкләрем,
Үтәлде теләкләрем!
```
