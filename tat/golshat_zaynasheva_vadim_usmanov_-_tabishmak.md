# Оригинал

### Гөлшат Зәйнәшева / Вадим Усманов - Табышмак

```
Бер җылы сүзең, татлы сүзең
Көч бирә, нур өсти күңелемә.
Мәңгелек йолдызлар булып син
Сибелдең бар дөньям күгенә.

Тик үзең син мине белмисең,
Бу җырым сиңа бер табышмак.
Кошлар бит сайрашып табыша.
Их, без дә җырлашып табышсак!

Бер җылы караш, бер елмаю
Мине гел бәхетле яшәтә.
Мин тоям ул җылы карашны,
Синнән бик еракта яшәп тә.

Тик үзең син мине белмисең,
Бу җырым сиңа бер табышмак.
Кошлар бит сайрашып табыша,
Их, без дә җырлашып табышсак!
```
