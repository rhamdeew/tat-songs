# Оригинал

### Гөлсәрвәр / Габдразак Миңгалиев - Бәхет өчен яшик әле

```
Бүгенгесе күңелләргә
Шатлыклар бирсен әле.
Үтеп киткән көннәрдән дә
Яктырак булсын әле.

Бәхет өчен яшик әле
Бүген дә, иртәгә дә.
Ташып тора яшәү дәрте
Бәхетле йөрәкләрдә.

Калдыра бел үткәннәрдә
Кайгылы хисләреңне.
Шатлык аша бәхет килә,
Белә ул көткәнеңне.

Бәхет өчен яшик әле
Бүген дә, иртәгә дә.
Ташып тора яшәү дәрте
Бәхетле йөрәкләрдә.

Моңсу бар йөрәкләрдә
Күренми яшәү көче.
Яши изге теләкләрдә
Ярата белгән кеше.

Бәхет өчен яшик әле
Бүген дә, иртәгә дә.
Ташып тора яшәү дәрте
Бәхетле йөрәкләрдә.
```
