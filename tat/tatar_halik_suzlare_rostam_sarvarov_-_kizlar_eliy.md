# Оригинал

### Татар халык сүзләре / Рөстәм Сәрвәров - Кызлар елый

```
Мендәренә чумып бер кыз елый,
Күз яшьләрен күрә ай гына.
Ерактагы ярын сагына ул —
Аны уйлый, көтә, кайгыра.

Төн тынлыгын кочып бер кыз елый,
Тик ай гына белә хәлләрен.
Шатлыгыннан елый — китергәннәр
Сөйгәненең кайтыр хәбәрен.

Ак каеннар янына бер кыз килгән,
Сорамагыз, нигә килгән ул.
Каеннарга сарылып кызлар елый
Ятлар белән күрсә сөйгәнен.
```
