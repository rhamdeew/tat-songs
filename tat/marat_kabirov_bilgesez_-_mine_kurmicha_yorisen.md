# Оригинал

### Марат Кәбиров / Билгесез - Мине күрмичә йөрисең

```
Мине күрмичә йөрисең…
Мин син дип янам…
Сиңа булган мәхәббәтем
Сүрелми һаман…

Көннәрен дә сине уйлыйм,
Уйлыйм төнен дә…
Китмәс өчен кердең мени
Минем күңелгә…

Мин бит сине, мин бит сине
Шундый яратам…
Хыялымда сине көн дә
Үземә каратам…
Синең хакта уйлап кына
Туа яңа таң…
Мин бит сине, мин бит сине
Шундый яратам…

Тизрәк бергә булыйк инде…
Түзми бит йөрәк…
Беркемгә дә сине бирмәм,
Тагы ни кирәк?!.
Нәрсә көтәсең инде син,
Кемне көтәсең…
Тизәйтәсең килми мени
Бәхет иртәсен?!.

Уңга карап, сулга карап
бәхет эзлисең.
Эзләгәнең, бәлки, миндер.
Ә син күрмисең…
```
