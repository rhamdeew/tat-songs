# Оригинал

### Әхмәт Ерикәй / Заһид Хәбибуллин - Сиңа (беренче вариант)

```
Ниләр сөйли озын керфекләрең,
Ниләр сөйли шомырт төсле күзләрең?
Кемнәр өчен үрелеп чәчәк ата
Яшьлегеңнең назлы гөлләре?

Кемнәр өчен ямьле тау буенда
Тәмле булып кызарып пешә бөрлегән?
Бөрлегәндәй алсу иреннәрең
Кемне сагынып көтә, гөл генәм?..

Кемнәр өчен төнге караңгыда
Айдан якты нурлар агылалар?
Кемнәр өчен ап-ак йөзләреңдә
Сөю шәүләләре чагылалар?..
```
