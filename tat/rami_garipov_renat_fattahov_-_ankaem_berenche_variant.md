# Оригинал

### Рәми Гарипов / Ренат Фаттахов - Әнкәем (беренче вариант)

```
Әнкәем, бәгърем, алдарак китсәм,
Кайгырма һич минем китгәнгә.
Кайгырсаң, тик кайгыр минем бары
Ак каенны әнкәм иткәндә.

Баш очымда минем утыртсыннар
Кайгысы ак булганы ак каен.
Күз яшеңне коймый, минем белән
Бер көн килеп сөйләш, яз сайын.

Мин әйтермен, сабый яфрак булып,
Бер генә сүз: "Әнкәй! — күз нурым".
Ак каенны сыйпап тыңла бер җыр,
Ак каенда — минем күз нурым.
```
