# Оригинал

### Абдулла Сәләхетдинов / Зиннур Гыйбадуллин - Яшь төзүчеләр

```
Күз өстендә каш кебек ул — төзүче,
Йортлар сала таш тезеп ул — төзүче;
Кояш кертә өйләргә ул — төзүче,
Жирне бизи ашкынып ул — төзүче.

Зур гигантлар,
Комбинатлар,
Яна, матур калалар
Төзүчегә һәйкәл булып
Гасырларга калалар.

Өйрәнәбез шундый бөек һөнәргә —
Хезмәт белән туган илне бизәргә.
Таянырбыз эштә дәрткә, белемгә,
Ил таяныр безнең көчле иннәргә.

Зур гигантлар,
Комбинатлар,
Яна, матур калалар
Төзүчегә һәйкәл булып
Гасырларга калалар.

Якты юллар ача белем йортыбыз,
Бәхет — һөнәр серендә, дип укыйбыз.
Хезмәт кочагында дөрләп кабынды
Гомер буе сүнмәс йөрәк утыбыз.

Зур гигантлар,
Комбинатлар,
Яна, матур калалар
Төзүчегә һәйкәл булып
Гасырларга калалар.
```
