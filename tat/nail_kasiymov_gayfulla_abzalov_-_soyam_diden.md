# Оригинал

### Наил Касыймов / Гайфулла Абзалов - Сөям, дидең

```
Җирдә яшәү серләрен мин
Үзгәрүдән эзләдем.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.

Тын күлләр дә гел тын булмый
Ярда дулкын эзләре.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.

Үзгәрер шул күзләрең дә
Үзгәрер шул йөзләрең.
Үзгәрешсез калсын бары
«Сөям» дигән сүзләрең.
```
