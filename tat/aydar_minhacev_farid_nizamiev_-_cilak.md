# Оригинал

### Айдар Минһаҗев / Фәрид Низамиев - Җиләк

```
Нигә микән җиләкләр
Коелырга чамалый?
Җыеп аласылар килә,
Соңга калу ярамый.
Әйдә, бергә җыябыз дип,
Дәште мине чибәрем.
Белми дә калдым шикелле
Ни арада килгәнен.

Их, җиләге, җиләге,
Тула инде чиләге.
Җиләк кебек иренеңнән
Үбәселәр киләдер.

Үзем җиләкләр җыям,
Үзем сиңа күз атам.
Җыясы да килмәс иде
Әгәр дә син булмасаң.
Юк, мин әкрен генә җыям,
Әйдә, вакыт үтмәсен.
Синең яннан гына бер дә
Килми лә шул китәсем.

Их, җиләге, җиләге,
Тула инде чиләге.
Җиләк кебек иренеңнән
Үбәселәр киләдер.

Җыелып бетә җиләкләр,
Күңелсезләнеп калдым.
Иртәгә дә, иртәгә дә
Өлгерсә иде тагын.
Җыелып бетә җиләкләр,
Күңелсезләнеп калдым.
Иртәгә дә, иртәгә дә
Өлгерсә иде тагын.

Их, җиләге, җиләге,
Тула инде чиләге.
Җиләк кебек иренеңнән
Үбәселәр киләдер.
```
