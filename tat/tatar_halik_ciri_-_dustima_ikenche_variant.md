# Оригинал

### Татар халык җыры - Дустыма (икенче вариант)

```
Әйдә, дустым, су буена,
Бөдрә чәчең юарсың ла,
Син юарсың, мин җырлармын,
Күңелең тулса еларсың.

Кызыл-кызыл бөрлегән лә,
Юл буена тезелгән лә,
Булмас инде минең кебек
Синең өчен өзелгән.

Өфе юллары пыяла,
Таш булса таймас иде лә,
Минем йөрәккәем яна, —
Син булсаң янмас иде.
```
