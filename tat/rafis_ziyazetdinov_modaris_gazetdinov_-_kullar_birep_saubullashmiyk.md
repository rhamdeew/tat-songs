# Оригинал

### Рәфис Зиязетдинов / Мөдәрис Газетдинов - Куллар биреп саубуллашмыйк

```
Очрашулар була кыска,
Ә җыр иле мәңгелек.
Җыр яшәсә йөрәкләрдә,
Җинәр җирдә изгелек.

Куллар биреп саубуллашмыйк.
Аерылмаска озакка.
Моңнар араларны бәйләр,
Без китсәк тә еракка.

Күңелләрне җыр ташытыр
Язгы жил исмәсә дә.
Хисләр гөлбакчада йөзә,
Бураннар уйнаса да.

Куллар биреп саубуллашмыйк.
Аерылмаска озакка.
Моңнар араларны бәйләр,
Без китсәк тә еракка.

Йөрәкләр тылсымланыр
Моң агымында калып.
Яшик әле дөньяларда
Сайрашып кына, янып.

Куллар биреп саубуллашмыйк.
Аерылмаска озакка.
Моңнар араларны бәйләр,
Без китсәк тә еракка.
```
