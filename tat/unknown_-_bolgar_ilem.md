# Оригинал

### Мирһади Разов / Виталий Агапов - Болгар илем

```
Чал тарихта, бормаланып-бормаланып,
Гасырлардан килгән юллар бар.
Җимерелгән, янган, диләр…
Дөрес түгел!
Йөрәкләрдә безнең Болгар бар!

Безнең җанда яна Болгар учаклары,
Дөрләп-дөрләп, сүнми бүген дә.
Бабаларның изге рухы булып, әнә —
Алтын ае балкый күгендә.

Балкый ае, киселсә дә манарасы,
Шуңа якты хәтта төнебез.
Болгардан ук килә туган телебез дә,
Болгардан ук килә динебез.

Күп дошманнар теш кайрады аңа, ләкин
Бездә көчле, корыч куллар бар.
Болгар исән, һәрбер татар йөрәгендә
Мәңгелек һәм япь-яшь Болгар бар!
```
