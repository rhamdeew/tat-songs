# Оригинал

### Вәрис Акбашев / Урал Рәшитов - А жена татарочка

```
Сатучылар, без бай диләр
Алар да чыга очка.
Бер байлыгым булмаса да
А, жена татарочка

Бәхет диеп, байлык эзләп
Кереп баттым бурычка.
Алтын балык тотмасам да
А, жена татарочка

Санга сукмый машинамны
Мерседес алга оча.
Мерседесны узмасам да
А, жена татарочка

Айлы төндә, төшләремдә
Алтыннар яуды учка.
Булмаса да алтын учта
А, жена татарочка
```
