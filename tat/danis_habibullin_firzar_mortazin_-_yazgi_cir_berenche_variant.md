# Оригинал

### Данис Хәбибуллин / Фирзәр Мортазин - Язгы җыр (беренче вариант)

```
Язлар килә, язлар,
Кайта киек казлар,
Су буйлары ямьгә күмелә.
Ә син, иркәм, нигә
Кайтмыйсың бирегә,
Язлар китермисең күңелгә?

Майлар җитә, майлар,
Көтеп алган айлар,
Ак чәчәккә күмелә бакчалар.
Тик шат булыр идем
Көзләрем дә минем
Шушы язда чәчәк атсалар.

Еллар уза, еллар,
Гүя аккан сулар,
Аккан сулар алар тиң безгә.
Аерым аккан сулар,
Үтеп озын юллар,
Кушылырлар әле диңгездә.

Диңгездәге судай,
Үтеп озын юллар,
Кушылырбыз әле без бергә,
Кушылырбыз әле гомергә.
```
