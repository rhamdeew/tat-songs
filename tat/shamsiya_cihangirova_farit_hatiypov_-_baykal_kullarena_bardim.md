# Оригинал

### Шәмсия Җиһангирова / Фәрит Хатыйпов - Байкал күлләренә бардым

```
Байкал күлләренә бардым,
Су буйларын буйладым.
Байкал күле «Уф-уф…» дигәч,
Кайгыдан дип уйладым.

Байкал күлләренә бардым,
Каршы ярын күрмәдем.
Уф — уфларың ишетсәм дә,
Эч сереңне белмәдем.

Байкал күлләренә бардым,
Кабат баралмам инде.
Салкын сулар алып эчтем —
Кабат яналмам инде.
```
