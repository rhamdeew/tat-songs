# Оригинал

### Илсөяр Иксанова / Эльвира Хаматнурова - Иртә (беренче вариант)

```
Бер ягымда — кызыл Кояш.
Бер ягымда — ап-ак Ай.
Мине озата баралар,
Адашмасын дип бугай.

Әни минем Кояш инде,
Әти Ай дип ышанам.
Уң якта — Ай,
Сулда — Кояш,
Ә мин кемгә охшаган?

Бер кулымнан Әни тоткан,
Бер кулымнан — Әтием.
Шыпырт кына уйлап барам,
Үзләренә әйтмимен.

Әни минем Кояш инде,
Әти Ай дип ышанам.
Уң якта — Ай,
Сулда — Кояш,
Ә мин кемгә охшаган?
```
