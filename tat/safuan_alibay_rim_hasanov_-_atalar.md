# Оригинал

### Сафуан Әлибай / Рим Хәсәнов - Аталар

```
Аталар, аталар, аталар…
Илемнең, җиремнең терәге;
Без тыныч, без исән, ил исән,
Типкәндә аталар йөрәге.

Аталар, аталар, аталар…
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар…
Көтәбез сагынып кайтканны,
Аталар китерә түгелме
Учында күтәреп ал таңны?!

Аталар, аталар, аталар…
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар…
Күпләре әйләнеп кайтмыйлар…
Кайтканнар яралы көенчә
Сафлардан калмыйлар, атлыйлар.

Аталар, аталар, аталар…
Гел алда, гел алда атлыйлар.

Аталар, аталар, аталар…
Саклыйлар, данлыйлар Ватанны.
Сабыйлар көтсеннәр атаны,
Көтсеннәр тик эштән кайтканны!

Аталар, аталар, аталар…
Гел алда, гел алда атлыйлар.
```
