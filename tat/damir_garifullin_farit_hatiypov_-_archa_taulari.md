# Оригинал

### Дамир Гарифуллин / Фәрит Хатыйпов - Арча таулары

```
Арча, диләр, Арча, диләр,
Арча биек тауларда.
Язмыш адәм балаларын
Йөртә әллә кайларда.

Их… Арча тауларыннан,
Менәлмим алларыннан,
Тормыш җиле аермасын,
Ярларын парларыннан.

Арча тавы матур булып
Күренә алсу таңнарда.
Сөю гөле бер кайчан да,
Сулмый гашыйк җаннарда.
```
