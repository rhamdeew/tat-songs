# Оригинал

### Мөнир Мазунов / Алмаз Монасыйпов - Хәлимә

```
Хәлимәнең чәчләренә
Сулар килеште микән?
Толымнарын дулкын итеп
Җилләр үреште микән?

Хәлимә, их, Хәлимә,
Килсәң иде яныма.
Сөюемне беләсең бит,
Күз салмыйсың хәлемә.

Хәлимәнең җырлаулары
Тургай тавышы микән?
Чишмәләр агышы микән,
Кызлар сагышы микән?

Хәлимә, их, Хәлимә,
Килсәң иде яныма.
Сөюемне беләсең бит,
Күз салмыйсың хәлемә.

Хәлимәнең керфекләре
Гөлләр куесы микәй?
Күзләренең тирәнлеге
Серләр коесы микән?

Хәлимә, их, Хәлимә,
Килсәң иде яныма.
Сөюемне беләсең бит,
Күз салмыйсың хәлемә.
```
