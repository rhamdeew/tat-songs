# Оригинал

### Марат Хәмитов / Рәзинә Шаһиева - Шомырт чәчәк атканда

```
Шомырт чәчәк атканда,
Язгы матур чакларда,
Йөрәкләргә дәрт өстәлеп,
Сызылып таңнар атканда.

Шомырт чәчәк атканда,
Хисләр тулып ташканда,
Ике йөрәк бердәй тибәр
Парлап вальс басканда,
Сайрар кошлар кайтканда,
Сөйгән ярың тапканда,
Иң күңелле он(ы)тылмас чак
Шомырт чәчәк атканда.

Тальян гармун тартканда,
Яшьлек дәрте ташканда,
Бәхетле парлар кушыла
Шомырт чәчәк атканда.

Шомырт чәчәк атканда,
Хисләр тулып ташканда.
Ике йөрәк бердәй тибәр
Парлап вальс басканда,
Сайрар кошлар кайтканда,
Сөйгән ярың тапканда,
Иң күңелле он(ы)тылмас чак
Шомырт чәчәк атканда.
```
