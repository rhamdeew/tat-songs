# Оригинал

### Илзирә Вәлиуллина / Нияз Вәлиев - Яннарымда ул булганда

```
(Яннарымда ул булганда ял гына)
(Яннарымда ул булганда ял гына)
Бәхет үзе коча бүген иңемнән,
Матур сүзләр ишетәмен тиңемнән.
Син гомергә булсаң иде янымда,
Урын түрдән булыр бәгърем, җанымда,
Җанымда.

Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!

Булды "Хәлең ничек?" — диеп сораучы,
Төшләремне гел яхшыга юраучы.
Мин бәхетле гади сүзгә шул гына:
Андый сүзләр әйтә ала ул гына,
Ул гына.

Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Яннарымда ул булганда ял гына.

Ярты сүздән күңелемне аңлый ул,
Йөрәгемә һич бер яра салмый ул.
Әйтерсең лә без кавышкан күптәннән,
Без бәхетле бәхет безне көткәннән,
Көткәннән.

Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!
Яннарымда ул булганда ял гына,
Үзгәрмичә шундый булып кал гына!..
```
