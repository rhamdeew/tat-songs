# Оригинал

### Марат Кәбиров / Гүзәл Җәлалова - Тал бөгелә җилләргә

```
Елгалары тар гына,
Елга яры — тал гына.
Ялгыз калган йөрәгемә
Сары сагыш сарыла.

Хәлем сөйлим кемнәргә —
Тал бөгелә җилләргә.
Парлылар да бөгелгәндә,
Ялгыз ничек түзәргә?!

Карыйм бөдрә талларга,
Карыйм агымсуларга.
Алар мине аңламаслар,
Калыр авыр суларга.

Карыйм бөдрә талларга,
Таллар елый таңнарда.
Әллә минем хәлләремне
Аңлый микән алар да.

Хәлем сөйлим кемнәргә —
Тал бөгелә җилләргә.
Парлылар да бөгелгәндә,
Ялгыз ничек түзәргә?!
```
