# Оригинал

### Чыңгыз Мусин / Гафият Нигъмәтуллин - Тар басма

```
Челтер-челтер сулар ага,
Басмалары асылмалы.
Сикереп төшеп сулар эчтем,
Сусауларым басылмады.

Тар басмага басып кына,
Сулар алдым чиләгемә.
Сулар алдым, су сипмәдем
Ялкынланган йөрәгеңә.

Үтә айлар, үтә еллар,
Агым судай ага гомер.
Тар басмада аерылыштык,
Арабызда хатлар йөрер.

Тар басмага басып кына,
Сулар алдым чиләгемә.
Сулар алдым, су сипмәдем
Ялкынланган йөрәгеңә.

Әкрен генә кояш чыга,
Башын салып болытларга.
Тар басмага чыгып карыйм
Бик сагынган минутларда.

Тар басмага басып кына,
Сулар алдым чиләгемә.
Сулар алдым, су сипмәдем
Ялкынланган йөрәгеңә.
```
