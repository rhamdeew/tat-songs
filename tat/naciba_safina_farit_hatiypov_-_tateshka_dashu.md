# Оригинал

### Нәҗибә Сафина / Фәрит Хатыйпов - Тәтешкә дәшү

```
Тәтеш дигән гүзәл җир бар,
Татар илендә.
Иң кадерле, изге җир ул
Минем күңелдә.

Эй, Тәтеш, Тәтеш, Тәтеш,
Күреп туя алмас төш.
Аерылганда, юл яраганда,
Тәтешемнән алам көч.

Тау башында, ай ташында
Йөзек кашыдай.
Балкый, нурлана Тәтешем,
Күңел кошымдай

Эй, Тәтеш, Тәтеш, Тәтеш
Әйтерсең бер моңлы төш
Аерылганда, юл яраганда,
Тәтешемнән алам көч.

Идел иле, алма иле
Минем Тәтешем.
Сиңа карап тигезләшә
Йөрәк тибешем.

Эй, Тәтеш, Тәтеш, Тәтеш,
Өннәремә кергән төш.
Күңелемнән моң сарганда,
Тәтешемнән алам көч.
```
