# Оригинал

### Гөлфия Юнысова / Наил Галиев (Николай Голов) - Без яшь чакта (беренче вариант)

```
Без яшь чакта күк йолдызлы,
Ә таңнар алсу иде.
Төннәр кыска юллар җиңел,
Язлар гел ярсу иде.

Без яшь чакта уйыннар да
Хәлем күңелле иде:
Гармуннар шәп, җырлар моңлы,
Биюләр җилле иде.

Без яшь чакта тимер йомшак
Бакырлар атын иде.
Нинди генә эшне алма,
Бик тиз эшләнә иде.

Без яшь чакта әбиләр — кыз,
Бабайлар егет иде.
Кояшы да кайнарырак,
Яктырак кебек иде.
```
