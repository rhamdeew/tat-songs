# Оригинал

### Вәлит Илембәтов / Ризван Хәкимов - Бергә булыгыз!

```
Ялгызлыклар газап бит ул,
Күңелне баса сагыш.
Җан сөйгән ярыңнан язсаң,
Бәхетсез була язмыш.

Сөйгән ярларыгыз белән
Бергә булыгыз!
Тик шул чакта бу тормышта
Уңар юлыгыз!

Ялгызлыклар газап бит ул,
Ут йотып көннәр үтә.
Җан сөйгән ярыңнан ярсаң,
Төннәр йокысыз үтә…

Сөйгән ярларыгыз белән
Бергә булыгыз!
Тик шул чакта бу тормышта
Уңар юлыгыз!

Ялгызлыклар газап бит ул,
Йөрәкне ялкын өтә.
Җан сөйгән ярыңнан язсаң,
Әрәмгә гомер үтә.

Сөйгән ярларыгыз белән
Бергә булыгыз!
Тик шул чакта бу тормышта
Уңар юлыгыз!
```
