# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Белсәң иде хәлләремне

```
Уяндылар гөрләвекләр, акты ярсып,
Ашкындылар гүя якты яздан качып.
Алар кебек, без дә язда калалмадык,
Аерылмаска сәбәпләрне табалмадык.

Белсәң иде минем хәлләремне,
Син булмагач, бәхет килсен каян,
Тыңгы белми йөрәк янар һаман-һаман,
Беркемне дә болай яраталмам.

Кичке шәфәкъ нурларында яшь бөреләр
Мөлдерәмә яшьләр булып күренделәр.
Саубуллашып, соңгы сүзләр әйтелделәр,
Мең сораулар сине миннән яшерделәр.

Белсәң иде минем хәлләремне,
Син булмагач, бәхет килсен каян,
Тыңгы белми йөрәк янар һаман-һаман,
Беркемне дә болай яраталмам.

Чыга алмый йөрим кебек томаннардан,
Сагышларым күзләремне томалаган.
Булмады, дип, үпкәләшкән чагыбызда
Ни җитмәде икән безгә языбызда.

Белсәң иде минем хәлләремне,
Син булмагач, бәхет килсен каян,
Тыңгы белми йөрәк янар һаман-һаман,
Беркемне дә болай яраталмам.
Тыңгы белми йөрәк янар һаман-һаман,
Беркемне дә болай яраталмам…
Беркемне дә болай яраталмам…
```
