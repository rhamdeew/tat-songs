# Оригинал

### Ризида Мөхәммәтҗанова - Күңел кошында очып

```
Авылымда каенлыклар
Биек таулар башында.
Ниг(ә) әле, әнием, синең
Һәркөн кулың кашыңда?

Көтәсең, беләм, көтәсең,
Догаларыңны укып,
Кайтабыз, көн дә кайтабыз
Күңел кошында очып.

Сагындым, әнием, сине,
Сагындым авыл кичен,
Һәрвакыт кайтып булмай шул,
Әнием, бәгърем, кичер.

Көтәсең, беләм, көтәсең,
Догаларыңны укып,
Кайтабыз, көн дә кайтабыз
Күңел кошында очып.

Авылым, сагындырды бит
Тауларың, урманнарың,
Әни, бәгърем, кайгырма син,
Юш килер юраганнарың.

Сизәмен, беләм, сизәмен,
Күзең талгандыр көтеп,
Кайтып киләбез, әнием,
Зур авыр юлны үтеп.
```
