# Оригинал

### Тәслимә Кәримова / Гөлназ Гарифҗанова - Кукмара (икенче вариант)

```
Кукмара юллары борма,
Юл чите нарат кына.
Кукмараны кайтып күрсәм —
Мәңгегә яраттыра!

Кукмара таулары текә,
Тау битләре җиләкле.
Кукмарада кызлар җырлый
Өзеп-өзеп узәкне.

Кукмараның итекләре
Чәчкә-чәчкә бизәкле.
Кукмара кызлары чаян
Өзә торган үзәкне.

Кукмара таулары текә,
Тау битләре җиләкле.
Кукмарада кызлар җырлый
Өзеп-өзеп узәкне.

Тегүчеләр, игенчеләр
Данга күмә бу якны.
Кукмаралар колач жәеп
Каршылыйлар кунакны.

Кукмара таулары текә,
Тау битләре җиләкле.
Кукмарада кызлар җырлый
Өзеп-өзеп узәкне.
```
