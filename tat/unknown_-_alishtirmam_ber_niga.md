# Оригинал

### Расих Ханнанов / Тәлгать Шәрипов - Алыштырмам бер нигә

```
Күргәнем бар сәхрәләрне,
Әкият дөньяларын.
Бер нигә дә алмаштырмам
Балачак эзләрен салган
Туган як тугайларын.

Күргәнем бар елгаларның
Чишмәдәй ярсуын.
Бер нигә дә алмаштырмам
Җиремә ямь биреп торган
Агыйделнең ак суын.

Күргәнем бар күп җирләрне,
Гүзәллеккә юк тиңе.
Бер нигә дә алмаштырмам
Бәхеткәем ташып торган —
Башкортстаным сине.
```
