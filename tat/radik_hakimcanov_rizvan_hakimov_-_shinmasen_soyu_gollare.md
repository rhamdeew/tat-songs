# Оригинал

### Радик Хәкимҗанов / Ризван Хәкимов - Шиңмәсен сөю гөлләре

```
Көз җитсәдә яз ямьләрен саклый,
Түземле шул көзге чәчкәләр.
Алар бераз моңсу бусалар да,
Һаман әле нурлар чәчәләр.

Көз җитсәдә яз ямьләрен саклый,
Күңелләрне арбаб тарталар.
Көзге гөлләр бигърәк кадерлеләр,
Үткәннәргә алып кайталар.

Көз җитсәдә яз ямьләрен саклый,
Көзнең соңгы чыдам гөлләре.
Сөйгән ярдай, янып, сагышланып,
Көтә кавышу көннәре.

Синдә әллә яз сулышын тыйдың,
Искән дә дә көзнең җилләре.
Бу дөнъяны бизәп яшисе бар,
Тик, шиңмәсен сөю гөлләре.
```
