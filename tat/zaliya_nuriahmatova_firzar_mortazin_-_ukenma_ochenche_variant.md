# Оригинал

### Залия Нуриәхмәтова / Фирзәр Мортазин - Үкенмә (өченче вариант)

```
Үкенмә үткәннәргә,
Үкенмә киткәннәргә.
Үкен син, дус булып йөреп,
Сереңне сүткәннәргә.
Үкенмә үз-үзеңә..
Кешеләрнең сүзенә..
Һәрвакытта дөресен әйтеп,
Туры кара күзенә.
Үкенүләр, сагышлар..
Югалтулар, табышлар.
Һәркемгә дә аерым-аерым,
Бирелә шул язмышлар.
Зарланма белгәннәргә,
Карама көлгәннәргә.
Ышанма, мәңгегә-диеп,
Вәгъдәләр биргәннәргә.
Яши бел син үкенми,
Начар сүздән сүтелми..
Ялгышларсыз, сагышларсыз,
Гомер юлы үтелми.
```
