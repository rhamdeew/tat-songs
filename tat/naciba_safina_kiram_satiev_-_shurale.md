# Оригинал

### Нәҗибә Сафина / Кирам Сатиев - Шүрәле

```
Җен дә түгел, җил дә түгел
Күренеп йөри торган
Шүрәлене саклап тора
Татарда кара урман.

Шүрәле, Шүрәле,
Син Айгөлне күр әле!
Үзе бии, үзе җырлый —
Куллар чабып тор әле!

Бер мөгез дә, койрык бер.
Кети-кети уйный ул.
КИем-салым кими ул,
Балаларга тими ул.

Шүрәле, Шүрәле,
Син Газине күр әле!
Җырга оста — җырласын,
Һич оялып тормасын.
```
