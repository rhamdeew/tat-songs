# Оригинал

### Ризидә Ситдыйкова / Айнур Сабирҗанов - Нигә бергә түгел без?..

```
Сүз куешкан идек без, бәгърем,
Бер-беребезгә тугры булырга.
Ал чәчәк тә гөл чәчәк булып,
Тормышларда бергә балкырга.

Ал чәчәгем минем булмадың,
Гөл чәчәгең синең булмадым.
Кайсыбыз соң, кем соң гаепле?
Кайсыбыз соң хәзер бәхетле?

Мин бит сине күреп туймадым,
Җылы биреп, сөеп туймадым.
Аерылышу безнең ул чакта
Вакытлыча диеп уйладык.

Ал чәчәгем минем булмадың,
Гөл чәчәгең синең булмадым.
Кайсыбыз соң, кем соң гаепле?
Кайсыбыз соң хәзер бәхетле?

Нигә, нигә бергә түгел без? —
Күрешкәндә шул бит уебыз.
Шундый көчле мәхаббәтебез
Утларында һаман янабыз.

Ал чәчәгем минем булмадың,
Гөл чәчәгең синең булмадым.
Кайсыбыз соң, кем соң гаепле?
Кайсыбыз соң хәзер бәхетле?
```
