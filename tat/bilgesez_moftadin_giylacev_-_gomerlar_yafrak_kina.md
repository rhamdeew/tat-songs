# Оригинал

### Билгесез / Мөфтәдин Гыйләҗев - Гомерләр — яфрак кына…

```
Нинди генә җилләр исми,
Ә тормыш гөрли һаман.
Гомер генә алга бара
Тибрәлгән яфрак сымак.

Газиз гомер яфрак кына,
Балачак — язгы бөре.
Яшел чагы — яшьлек аның,
Сарысы — гомер көзе.

Шуңа аның һәр мизгеле
Үзенчә гүзәл, назлы.
Ә шулай да күңел көтә
Бөреле җылы язны.

Газиз гомер яфрак кына,
Балачак — язгы бөре.
Яшел чагы — яшьлек аның,
Сарысы — гомер көзе.

Һәр яфрак — бер бизәлеш,
Була тик агачында.
Гомер — мизгел, яшәү — бәхет,
Туган җир кочагында.

Газиз гомер яфрак кына,
Балачак — язгы бөре.
Яшел чагы — яшьлек аның,
Сарысы — гомер көзе.
```
