# Оригинал

### Гөлсәрвәр / Ленур Зәйнуллин - Үзсүзле

```
еше сүзен тыңлыйм,
Тик үземчә эшлим,
«Үзсүзле син, — диләр, — үзсүзле!»
Шушы сүзне миңа
Әнкәем дә әйтте.
Тик үзгәртә алмыйм, алмыйм үземне.

Мин башкача була алмыйм,
Үзгәртмәмен үземне.
Үзем ничек — шулай калыйм,
Үзсүзле мин, үзсүзле.

Минем якын кешем,
— Үзгәр әле, — диеп, —
Кабатлыйсың син дә шул сүзне.
Үзгәрешле итеп
Мине күрим дисәң,
Син үзгәртеп кара, кара үзеңне!

Мин башкача була алмыйм,
Үзгәртмәмен үземне.
Үзем ничек — шулай калыйм,
Үзсүзле мин, үзсүзле.

Усал түгел, җаным,
Яхшылыкта — күңел,
Кирәк чакта сабыр, түземле.
Ләкин аңлый алмыйм,
Күпме еллар инде:
«Үзсүзле син, — диләр, диләр, — үзсүзле!»

Мин башкача була алмыйм,
Үзгәртмәмен үземне.
Үзем ничек — шулай калыйм,
Үзсүзле мин, үзсүзле.
```
