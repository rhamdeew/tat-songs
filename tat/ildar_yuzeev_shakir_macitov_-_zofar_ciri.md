# Оригинал

### Илдар Юзеев / Шакир Мәҗитов - Зөфәр җыры

```
Йомшак җилләр булып исәрмен дә
Тәрәзәңне килеп кагармын,
Мин кагармын, син качарсың, —
Йөрәгемә утлар ягармын.

Гөлчәчәкләр булып үсәрмен дә
Тәрәзәңә үрелеп карармын,
Мин карармын, син качарсың, —
Сөю ялкынында янармын.

Йолдыз булып күккә менәрмен дә
Өй турында балкып янармын,
Мин янармын, син качарсың, —
Төсең итеп кемгә багармын?
```
