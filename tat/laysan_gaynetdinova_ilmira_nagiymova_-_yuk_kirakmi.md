# Оригинал

### Ләйсән Гайнетдинова / Ильмира Нәгыймова - Юк, кирәкми

```
Өр-яңадан башлыйк әле дисең,
Башлыйк әле, дисең, ак биттән,
Бу сүзләрне инде синнән, җаным,
Ничәнчеләр тапкыр ишетәм.

Юк, кирәкми, юк кирәкми бүтән,
Башлау мөмкин түгел ак биттән.
Йөрәгемдә булган саф хисләрне
Син эзләдең нигәдер читтән.

Бәхетлесе, күбрәк күз яшьлесе
Синең белән узган минутлар.
Тормыш китабымда барысы да,
Сабырлыкның да бит чиге бар.

Юк, кирәкми, юк кирәкми бүтән,
Башлау мөмкин түгел ак биттән.
Йөрәгемдә булган саф хисләрне
Син эзләдең нигәдер читтән.

Син бит берәү, синдәйләр юк, дисең,
Бәхетемне, дисең, таныдым,
Ә мин инде сине көтә-көтә,
Сабыр итә-итә арыдым.

Юк, кирәкми, юк кирәкми бүтән,
Башлау мөмкин түгел ак биттән.
Йөрәгемдә булган саф хисләрне
Син эзләдең нигәдер читтән.
```
