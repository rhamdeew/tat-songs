# Оригинал

### Билгесез - Син дип тибә йөрәгем

```
Рәхәт, бәгърем, миңа синең белән,
Керфекләрең битне кытыклый.
Күз карашың гүя мине сыйпый,
Елмаюың мине култыклый.

Син, син диеп йөрәгем тибә,
Уртак шатлыкны бүлик без бергә.
Яшибез икән — шашып сөяргә,
Рәхәт дөньяда бергә яшәргә.

Рәхәт, бәгърем, миңа синең белән,
Йомшак җиргә каршы барганда,
Гөрләвекле матур язлар җиткәч,
Кыр казларын каршы алганда.

Син, син диеп йөрәгем тибә,
Уртак шатлыкны бүлик без бергә.
Яшибез икән — шашып сөяргә,
Рәхәт дөньяда бергә яшәргә.

Рәхәт, бәгърем, миңа синең белән,
Керфекләрең битне кытыклый.
Күз карашың гүя мине сыйпый,
Елмаюың мине култыклый.

Син, син диеп йөрәгем тибә,
Уртак шатлыкны бүлик без бергә.
Яшибез икән — шашып сөяргә,
Рәхәт дөньяда бергә яшәргә.
```
