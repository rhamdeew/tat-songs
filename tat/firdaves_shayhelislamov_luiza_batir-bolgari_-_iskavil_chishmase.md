# Оригинал

### Фирдәвес Шәйхелисламов / Луиза Батыр-Болгари - Искавыл чишмәсе

```
Кайттым Мөслим якларына,
Ык суларын буйладым.
Юл тоттым Искавылыма,
Түзалмадым, сагындым.

Әй Искавылым таулары,
Урманнары җиләкле.
Шифалы чишмә сулары
Дәртләндерә йөрәкне.

Һәрбер юлчы юлдан арып
Элек чишмәгә төшәр.
Хәл яңарткач, хәер кылып
Көмеш тәңкә бүләк итәр.

Әй Искавылым таулары,
Урманнары җиләкле.
Шифалы чишмә сулары
Дәртләндерә йөрәкне.

Авыз итми чишмә суын
Өйгә кайтып кералмыйм.
Сагынсам да, тик шифасын
Тоймыйча китә алмыйм.

Әй Искавылым таулары,
Урманнары җиләкле.
Шифалы чишмә сулары
Дәртләндерә йөрәкне.
```
