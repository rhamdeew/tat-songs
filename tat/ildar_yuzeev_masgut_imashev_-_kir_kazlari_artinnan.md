# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Кыр казлары артыннан

```
Кыр казлары артыннан
Биеклеккә тартылам.
Хыялыма китәр идем
Кыр казлары артыннан.

Кыр казлары артыннан
Китәр идем, тартынам.
Үз-үземне бәйләгәнмен
Хыялым канатыннан.

Кыр казлары артыннан
Очарга дип талпынам.
Тик очалмыйм, карап калам
Кыр казлары артыннан.
```
