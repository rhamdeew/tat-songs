# Оригинал

### Сәкинә Хәйруллина / Фирдүс Тямаев - Тормыш — көрәш (икенче вариант)

```
Тормыш — көрәш, без көрәшче,
Бил алышабыз.
Яшәү дигән киң мәйданда
Гел ярышта без.

Гомерләрне куып җилләр
Исә дә исә.
Без җилләргә бирешмәбез,
Аллаһым бирсә.

Һәр яңа көн — яңа ачыш,
Офыклар ерак.
Янып яшә, диеп дәшә
Ялыкмас йөрәк.

Гомерләрне куып җилләр
Исә дә исә.
Без җилләргә бирешмәбез,
Аллаһым бирсә.

Без — бу җирнең сөеп баккан
Газиз баласы.
Язган безгә бәхетне дә
Шуннан табасы.

Гомерләрне куып җилләр
Исә дә исә.
Без җилләргә бирешмәбез,
Аллаһым бирсә.
Без җилләргә бирешмәбез,
Аллаһым бирсә…
```
