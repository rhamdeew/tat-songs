# Оригинал

### Рөстәм Гыйззәтуллин - Әнкәйләрне рәнҗетмәгез

```
Бала чактан карап үстергәнсең,
Сөюеңде миңа биргәнсең.
Еллар үткәч балам кайтамы дип,
Карашыңны юлдан өзмисең.

Әнкәйләрне рәнҗетмәгез,
Авыр сүзләр берүк әйтмәгез.
Дөнъяларда без мәңгелек түгел,
Әнкәйләрнең кадерен белегез.

Кайсы бердә ялгыш әрнетәбез,
Әнкәйләрнең нечкә күңелен.
Һәр бер бала тели әнкәйләргә,
Озын гомер, бәхет, түземлек.

Әнкәйләрне рәнҗетмәгез,
Авыр сүзләр берүк әйтмәгез.
Дөнъяларда без мәңгелек түгел,
Кадерләрен генә белегез.
Әнкәйләрнең кадерен белегез.
```
