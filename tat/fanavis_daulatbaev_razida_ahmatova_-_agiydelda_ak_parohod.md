# Оригинал

### Фәнәвис Дәүләтбаев / Рәзидә Әхмәтова - Агыйделдә — ак пароход

```
Агыйделдә — ак пароход,
Күптән инде үткәне.
Килми, хәлне белми диеп
Белдермәче үпкәңне.

Агыйделдә-ак пароход.
Матур аның йөзүе!
Сагындым дип сөеклемнең
Үзәгемне өзүе.

Агыйделдә-ак пароход,
Киткән иде кычкыртып.
Яратам дип әйткән сүзе
Йөртә күктә очыртып!

Агыйделдә-ак пароход.
Матур аның йөзүе!
Сагындым дип сөеклемнең
Үзәгемне өзүе.

Агыйделдә-ак пароход.
Ул шул яшь чакларымда.
Сөю татыйм, бәгырькәем,
Кайнар кочакларыңда!

Агыйделдә-ак пароход.
Матур аның йөзүе!
Сагындым дип сөеклемнең
Үзәгемне өзүе.

Агыйделдә-ак пароход,
Дулкын юллый ярларга.
Башка ярлар кирәк түгел
Сөеп туймас яр барда.

Агыйделдә-ак пароход.
Матур аның йөзүе!
Сагындым дип сөеклемнең
Үзәгемне өзүе.
```
