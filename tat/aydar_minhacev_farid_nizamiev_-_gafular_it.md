# Оригинал

### Айдар Минһаҗев / Фәрид Низамиев - Гафулар ит

```
Соңгысы бул диеп килгән булам,
«Сау бул» диеп, кулларымны сузам.
Үз-үземне нигә алдыйм икән?
Булмый ла бит, оныта алмыйм, иркәм.

Гафулар ит ярсу йөрәгемне,
Гафулар ит әле дә сөйгәнемне.
Сүрелмәгән өчен яратулар
Ит гафулар, ит гафулар.

Гафулар ит түзә белмәвемне,
Сине генә, сине эзләвемне.
Төшләреңә килеп керә-керә
Дөньяларың бутап йөргәнемне.

Гафулар ит ярсу йөрәгемне,
Гафулар ит әле дә сөйгәнемне.
Сүрелмәгән өчен яратулар
Ит гафулар, ит гафулар.

Гафулар ит сөя белгәнемне,
Соң булса да кабат килгәнемне.
Өметләрне бер дә өзми генә
«Миңа гына бул син» дигәнемне.

Гафулар ит ярсу йөрәгемне,
Гафулар ит әле дә сөйгәнемне.
Сүрелмәгән өчен яратулар
Ит гафулар, ит гафулар.
```
