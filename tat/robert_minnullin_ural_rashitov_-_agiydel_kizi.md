# Оригинал

### Роберт Миңнуллин / Урал Рәшитов - Агыйдел кызы

```
Син Агыйдел кызы инде,
Агыйдел табынганын.
Төштә курдем иңнәреңә.
Дулкыннар кагылганын.

Агыйделдә коенасың
Чыланмасын толымың
Көлүең чыга ярлардан
Ник күзләрең тулы моң.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Агыйдел кызы.
Агыйдел кызы.

Сокланып карап торасың
Дулкыннар уйнаганга.
Төшләремә керәсеңдер
Гел сине уйлаганга.

Агыйделдә коенасың
Чыланмасын толымың.
Көлүен чыга ярлардан
Ник күзләрең тулы моң.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Агыйдел кызы.
Агыйдел кызы.

Сиңа талпына сөюем
Кайдан алыйм туземне.
Агыйдел минем өчен дә
Иркәләсен узеңне.

Агыйделдә коенасың
Чыланмасын толымың.
Колуен чыга ярлардан
Ник күзләрең тулы моң.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Эх, Агыйдел кызы.
Агыйдел кызы.
Агыйдел кызы.

Эх, Агыйдел кызы.

Эх, Агыйдел кызы.

Агыйдел кызы.
```
