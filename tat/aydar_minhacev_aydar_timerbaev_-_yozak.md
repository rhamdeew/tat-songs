# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Йозак

```
Сагындырган идең инде,
Күрешмәдек шул озак.
Килгән идем менә бүген,
Синең ишектә йозак.

Таныш йозак, салкын йозак
Каршымда тора эленеп.
Бикле шул синең йөрәгең дә
Миңа шул йозак кебек.

Ачып керә идем үзем
Шул йозакны мин элек.
Кулымда ачкыч булса да
Хәзер булмый шул кереп.

Таныш йозак, салкын йозак
Каршымда тора эленеп.
Бикле шул синең йөрәгең дә
Миңа шул йозак кебек.

Безнең сөю кайнарлыгын
Йозак күптән оныткан.
Учларыма алам аны
Гүя сине җылытам.

Таныш йозак, салкын йозак
Каршымда тора эленеп.
Бикле шул синең йөрәгең дә
Миңа шул йозак кебек.
```
