# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Аллары алдыр инде

```
Алы кайларда микән,
Гөле кайларда микән…
Уңда ярлар, сулда ярлар
Кайсын сайларга микән.

Аллары алдыр инде
Гөлләре гөлдер инде.
Ал да уңа, гөл дә уңа
Барсы да бердер инде.

Гөлләр — гөлдер. Мин гөлләргә
Тиң түгелмендер инде.
Гөлләрдән күз алалмыйча,
Алга үрелдем инде.

Аллары алдыр инде.
Алларын алдым инде.
Алларына алдандым да
Гөлләрдән калдым инде.
```
