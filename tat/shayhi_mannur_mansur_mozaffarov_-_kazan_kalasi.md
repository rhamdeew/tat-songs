# Оригинал

### Шәйхи Маннур / Мансур Мозаффаров - Казан каласы

```
Казан каласы таш кала,
Безнең сөйгән башкала.
Күңелләрне дәртләндергән
Җырлар шунда башлана.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.

Агыйделдән, Ык, Шөгердән,
Зәй, Зөядән Казанга
Килә яшьләр, канатланып,
Кошлар кебек язларда.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.

Илләр гизеп, сулар кичеп
Йөрсә дә егет һаман,
Кайтып килә ул яңадан —
Сагындыра шул Казан.

И-их, Казан каласы,
Гөлбакчалар арасы!
Яшь йөрәкнең,
Ярсу дәртнең
Синдә табыла дәвасы.
```
