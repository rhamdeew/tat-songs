# Оригинал

### Кадыйр Даян / Таһир Кәримов - Әйдә, җиңгә, биеп кал

```
Төнбоеклы күл буенда
Үсә икән биек тал.
Гомеркәең үтеп бара
Әйдә, җиңгә, биеп кал!

Бие, әйдә, бие, әйдә,
Аякларың талмасын.
Дәртең булгач, уйна да көл,
Үкенечкә калмасын.

Кымыз эчкән кодагыйның
Башы эйләнә икән.
Ике тустак кымыз эчсә
Теле бәйләнә икән.

Кымыз эчкән кодагыйның
Күңелләре җилкенә.
Тапырдатып баскан чакта
Урал тауы селкенә.
```
