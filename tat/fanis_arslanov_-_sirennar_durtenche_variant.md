# Оригинал

### Фәнис Арсланов - Сиреньнәр (дүртенче вариант)

```
Сиреньнәр чәчкәдә,
Өф итә чәчкәдәй.
Йөрибез бакчада,
Сиреньнәр шатлана.

Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә.
Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә.

Гел бергә булырбыз,
Гел бергә юлыбыз.
Сиреньнәре гөлләр сөя,
Гел бәхет сезгә дия.

Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә.
Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә.

Сиреньнәр чәчкәдә,
Өф итә чәчкәдәй.
Йөрибез бакчада,
Сиреньнәр шатлана.

Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә.
Гел бергә, гомер-гомергә,
Сиреньнәр гел бергә-бергә,
Сиреньнәр гел бергә-бергә…
```
