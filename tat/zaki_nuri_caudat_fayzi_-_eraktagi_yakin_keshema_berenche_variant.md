# Оригинал

### Зәки Нури / Җәүдәт Фәйзи - Ерактагы якын кешемә (беренче вариант)

```
Таулар сере — синең ачкан серләр,
Таулар керә минем төшемә.
Күңел күзем һаман сиңа карый —
Ерактагы якын,
Ерактагы якын кешемә.

Мәхәббәтем кайнар сулыш өрсен
Көзен-кышын юлда өшемә.
Сиңа оча минем йөрәк хисем
Ерактагы якын,
Ерактагы якын кешемә.

Сукмакларын үрдән-үргә узсын,
Уңыш телим авыр эшеңдә.
Җыр җибәрәм яңа көйгә салып
Ерактагы якын,
Ерактагы якын кешемә.
```
