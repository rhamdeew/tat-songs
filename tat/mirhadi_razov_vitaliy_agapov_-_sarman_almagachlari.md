# Оригинал

### Мирһади Разов / Виталий Агапов - Сарман алмагачлары

```
Майлар җитсә, туган якка
Тарта сагыш, тарта моң.
Ярсу язларын сагынып,
Сарманыма кайтамын.

Ап-ак туй күлмәген кисә
Сарман алмагачлары,
Сөю җырын җырлап туймый
Дәртле сандугачлары.

Назлап искән җилләрендә
Сагышларны таратам.
Алмагачлар, сердәшләрем,
Шуңа күрә яратам.

Ап-ак туй күлмәген кисә
Сарман алмагачлары,
Сөю җырын җырлап туймый
Дәртле сандугачлары.

Майлар җитсә, туган якка
Тарта сагыш, тарта моң.
Сарманымда мин үзем дә
Гөлләр булып атамын.

Ап-ак туй күлмәген кисә
Сарман алмагачлары,
Сөю җырын җырлап туймый
Дәртле сандугачлары.
```
