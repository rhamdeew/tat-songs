# Оригинал

### Фәридә Шакирова / Сәит Абдуллин - Ташлама (икенче вариант)

```
Ташлама, мине ташлама,
Мәхәббәт түгел уен.
Дөрләп яна йөрәккәем,
Аңлату миңа кыен.

Ташлама, мине ташлама
Ялгызлык упкынына,
Болай да йөрәгем яна
Мәхәббәт ялкынына.

Ташлама, мине ташлама,
Сагышларыңнан көям.
Бәгърем hич оныта алмыйм,
Сиздерми генә сөям.

Ташлама, мине ташлама
Ялгызлык упкынына,
Болай да йөрәгем яна
Мәхәббәт ялкынына.
```
