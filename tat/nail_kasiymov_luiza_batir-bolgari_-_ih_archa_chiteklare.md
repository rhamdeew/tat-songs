# Оригинал

### Наил Касыймов / Луиза Батыр-Болгари - Их, Арча читекләре!

```
Читек кигән, иген иккән
Халкым элек-электән.
Читек кигән халкым күреп
Дошман борылып киткән.

Их, Арча читекләре
Чигүле, искитмәле.
Арча читекләре киеп
Урамнан бер үт әле.

Чия пешкән, алма төшкән
Каюлы да, челтәрле.
Илдә итекләр күп булыр,
Читеккә соң җитәрме?!

Их, Арча читекләре
Чигүле, искитмәле.
Арча читекләре киеп
Урамнан бер үт әле.

Арча кызлары ник чибәр?
Читек кигәнгә чибәр.
Читек киеп уйнап, биеп,
Әйдә, бер җырлап җибәр!

Их, Арча читекләре
Чигүле, искитмәле.
Арча читекләре киеп
Урамнан бер үт әле.
```
