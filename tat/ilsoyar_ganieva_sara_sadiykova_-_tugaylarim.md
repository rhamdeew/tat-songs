# Оригинал

### Илсөяр Ганиева / Сара Садыйкова - Тугайларым

```
Тугайларым, ахры, мине гашыйк итте,
Күңелем калды шунда бәйләнеп.
Агыйделкәй, кайтырмын ла сиңа,
Киек казлар кебек әйләнеп.

Юксынма, вакытлыча гына китәмен,
Җырларыма салып синең гүзәллегең
Якын дусларыма илтәмен.

Нигә сине шулай өзелеп яратуның
Үзем генә беләм серләрен.
Йөрәгемә сүнмәс хисләр салдың,
Иң кадерле туган җирләрем.

Юксынма, вакытлыча гына китәмен,
Җырларыма салып синең гүзәллегең
Якын дусларыма илтәмен.
```
