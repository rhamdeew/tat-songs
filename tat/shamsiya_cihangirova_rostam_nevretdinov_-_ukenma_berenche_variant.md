# Оригинал

### Шәмсия Җиһангирова / Рөстәм Невретдинов - Үкенмә (беренче вариант)

```
Кешеләргә биргән бәхет
Безгә язмаган.
Язмыш безне мәхрүм иткән
Сөю-назлардан.
Юлларыбыз төрле якка
Адашып киткән.
Аерым-аеры гел заяга
Гомерләр үткән.

Зая узган гомерләрне
Уйлап үкенмә.
Шатланыйк без бер-беребезне
Тапкан өчен дә.
Күзләреңнән бары бәхет
Яшьләре тамсын.
Безнең сөю төсе булып,
Шушы җыр калсын.

Көтмәгәндә очрадың син
Гомер көземдә.
Эзләгән булгансың икән
Мине үзең дә.
Сагыш тулы күзләреңә
Барып сыендым.
Сөю-сәгадәт диңгезенә
Чумып коендым.

Зая узган гомерләрне
Уйлап үкенмә.
Шатланыйк без бер-беребезне
Тапкан өчен дә.
Күзләреңнән бары бәхет
Яшьләре тамсын.
Безнең сөю төсе булып,
Шушы җыр калсын.
```
