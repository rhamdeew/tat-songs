# Оригинал

### Саҗидә Сөләйманова / Мәсгуть Имашев - Гүзәлия җыры

```
Таң атканда уйланам,
Су акканда моңланам.
Таң нурында син йөзәсең,
Утларында мин янам.

Биек тауга менимме,
Агым суга керимме?
Дусларыма сөйлим микән,
Үзем генә белимме?

Таулар текә, су тирән,
Тирән булса да керәм.
Ерак икәнен дә беләм,
Нигәдер якын күрәм!
```
