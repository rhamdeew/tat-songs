# Оригинал

### Ильмира Нәгыймова - Сагышлы мәхәббәт (икенче вариант)

```
Бер ялгызым йөрим
Төнге урамнарда,
Каеннарга сөйләп зарымны.
Язган хатларыңны
Кат-кат укып чыгам,
Хисләремне утларга ягам!

Ә безнең мәхаббәт,
Саф иде мәхаббәт.
Сагышлы мәхәббәт,
Хәзер инде җыр булып калсын.

Бергә чакларымны
Исләремә алам,
Онытылмаслык хәтирә булып.
Насыйп ит, Ходаем,
Киләчәк тормышта
Мәхаббәттә газапланмаска.

Насыйп ит, Ходаем,
Үз ярымны табарга.
Бергә җылыны тоеп,
Мәхаббәт утында янарга.

Бер ялгызым йөрим
Төнге урамнарда,
Каеннарга сөйләп зарымны.
Язган хатларыңны
Кат-кат укып чыгам,
Хисләремне утларга ягам!

Ә безнең мәхаббәт,
Саф иде мәхаббәт.
Сагышлы мәхәббәт,
Хәзер инде җыр булып калсын.
```
