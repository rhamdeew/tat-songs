# Оригинал

### Гөлсәрвәр / Ильмира Нәгыймова - Белмәдеңме?

```
Бүген кинәт хәбәр алдым:
Сагынасың яратып.
Киткән идең башка ярга
Йөрәгемне елатып.

Белмәдеңме киткән чакта
Мине сагыначагың?
Яратусыз бу тормышта
Җиңел булмаячагын.

Юк, кагылма син кабаттан
Йөрәгемнең кылына.
Әйләнмәсен ул яңадан
Хисләреңнең колына.

Белмәдеңме киткән чакта
Мине сагыначагың?
Яратусыз бу тормышта
Җиңел булмаячагын.

Хисләремне аңламадың,
Яннарымнан югалдың.
Син бит мине ташламадың,
Үзең өчен югалдың.

Белмәдеңме киткән чакта
Мине сагыначагың?
Яратусыз бу тормышта
Җиңел булмаячагын.
```
