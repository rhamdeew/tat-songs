# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Син булсаң янымда (беренче вариант)

```
Мин — ялгыз дулкын,
Син — коткарыр ярым, —
Киң күкрәгеңә
Ничекләр башны салыйм.

Син булсаң янымда
Бик кайгылы чагымда,
Юатсаң, яратсаң,
Сагышларым тарала.
Син булсаң янымда
Бик бәхетле чагымда
Бәхетем, шатлыгым
Кабатлана тагында.

Мин — адашкан моң,
Син — эзләгән җырым.
Мәңге сайрарга
Йөрәккәеңә куныйм.

Син булсаң янымда
Бик кайгылы чагымда,
Юатсаң, яратсаң,
Сагышларым тарала.
Син булсаң янымда
Бик бәхетле чагымда
Бәхетем, шатлыгым
Кабатлана тагында.

Мин — синсез бушлык.
Син минсез бар микән?
Минсез сиңа да
Бу дөньялар тар микән?

Син булсаң янымда
Бик кайгылы чагымда,
Юатсаң, яратсаң,
Сагышларым тарала.
Син булсаң янымда
Бик бәхетле чагымда
Бәхетем, шатлыгым
Кабатлана тагында.
```
