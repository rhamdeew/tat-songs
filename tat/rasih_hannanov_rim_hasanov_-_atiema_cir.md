# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Әтиемә җыр

```
Минем әткәй көн дә эшкә китә,
Арып кайта аннан кичләрен.
Елмаюы итеп алып кайта
Шәфәкъләрнең алсу төсләрен.

Әткәй, әткәй, синнән башка тагы
Кем бар шундый бала бәгырьле.
Кояш сыман әнкәй берәү булса,
Син дә безгә шулай кадерле.

Сайрап туймас сандугачлармени —
Сүзләребез бетми серләшеп.
Чәнечкеле битләреннән сыйпап
Иркәләнү үзе зур бәхет.

Әткәй, әткәй, синнән башка тагы
Кем бар шундый бала бәгырьле.
Кояш сыман әнкәй берәү булса,
Син дә безгә шулай кадерле.

Минем әткәй көн дә эшкә китә,
Йолдызларны кайта ияртеп.
Йолдызлар да йоклап китә төсле
Әткәемнән әкият сөйләтеп.

Әткәй, әткәй, синнән башка тагы
Кем бар шундый бала бәгырьле.
Кояш сыман әнкәй берәү булса,
Син дә безгә шулай кадерле.
```
