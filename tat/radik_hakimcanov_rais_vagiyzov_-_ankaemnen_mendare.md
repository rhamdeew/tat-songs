# Оригинал

### Радик Хәкимҗанов / Рәис Вәгыйзов - Әнкәемнең мендәре

```
Шундый матур, йомшак бит ул,
Әнкәй биргән мендәре.
Җылылыгы ни тора бит,
Кышкы кичен, төннәрен.

Йомшак мендәр, җылы мендәр —
Әнкәемнең бүләге.
Бәхетле бул, балам, дидең,
Кабул булсын теләгең!

Шундый матур, йомшак бит ул,
Якты уйлар уйлата,
Изге теләк теләп таңда,
Гел иркәләп уята.

Йомшак мендәр, җылы мендәр —
Әнкәемнең бүләге.
Бәхетле бул, балам, дидең,
Кабул булсын теләгең!

Һәр көн иртән күңелемнән,
Әнкәйгә дога кылам.
Рәхмәтемне җиткерергә
Ярдәм итсәңче, Ходам!

Йомшак мендәр, җылы мендәр —
Әнкәемнең бүләге.
Бәхетле бул, балам, дидең,
Кабул булсын теләгең!
```
