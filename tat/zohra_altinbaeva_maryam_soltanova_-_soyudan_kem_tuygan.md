# Оригинал

### Зөһрә Алтынбаева / Мәрьям Солтанова - Сөюдән кем туйган?

```
Сөюгә суаган җанга
Мәхәббәт булып тулдың.
Синнән очратып уңдыммы,
Әллә бөтенләй туңдым?!

Мәхәббәтем терәк миңа
Янып-көенгәнемдә.
Уңсам да, тулсам да риза
Сөеп-сөелгәнемә.

Онытылмас ядкәр булып,
Күңелләргә моң тулган.
Сөеп кара күзләремә
Сөелүдән кем туйган?!
```
