# Оригинал

### Хәсән Туфан / Сара Садыйкова - Туган туфрак (икенче вариант)

```
Кәккүкләр моңая таң саен,
Ялгызның язмышы яман, дип.
Син кайсы йолдызлар астында
Йөрисең, бәхетләр табам, дип?

Адашкан, тилмергән кош булып, кош булып,
Көн саен төшләргә керәсең, керәсең.
Хатларга язмаган хәлеңне, хәлеңне
Җаннарым сизәдер, сизәдер, күрәсең,

Җырларым барсыннар яныңа,
Илеңнән сәламнәр, илтсеннәр.
Иделләр, Камалар иленә
Кайт инде, кайт инде! — дисеннәр,

Туган ил, туган җир сыймый шул, сыймый шул,
Яулыкка төйнәгән туфракка, туфракка.
Киткәннәр киек каз, киек каз шикелле
Әйләнеп кайталар, кайталар бу якка.
```
