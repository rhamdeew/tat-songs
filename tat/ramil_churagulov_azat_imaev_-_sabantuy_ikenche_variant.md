# Оригинал

### Рамил Чурагулов / Азат Имаев - Сабантуй (икенче вариант)

```
Исән-аман кышлап чыкты
Мал-туар йорт-курада.
Ашлык, икмәк күпереп үсәр
Беркөн киң басуларда.

Батырга да бер ял кирәк —
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм — сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!

Мәйданнарда көч сынаша
Сабантуй батырлары.
Чын егет сайлый кәләшкә
Кызларның асылларын.

Батырга да бер ял кирәк —
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм — сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!

Сабантуйларын гөрләтеп
Яңгырасын матур җыр.
Имин торсыннар дөньялар!
Икмәк булса, җыр булыр!

Батырга да бер ял кирәк —
Мең эшеңне читкә куй.
Җырлап-биеп, гөрләп узсын
Матур бәйрәм — сабантуй!
Эх! Сабантуй, сабантуй,
Иң матур туй, иң шәп туй!
```
