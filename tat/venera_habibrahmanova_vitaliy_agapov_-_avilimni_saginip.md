# Оригинал

### Венера Хәбибрахманова / Виталий Агапов - Авылымны сагынып

```
Төнге тынлык җырлый
Кайчак таңга кадәр.
Йокы оча хыял күгендә.
Күңел — тирән чоңгыл,
Гүя шуннан тыңлыйм
Әтәчләрнең моңын бүген дә.
Элек безнең сәгать
Чылтырамый иде,
Тик әтәчләр җырлап уятты.
Моңга чумган авыл,
Бәлки, йоклар иде,
Ал таңнар күз сирпеп оялтты.
Без яшисе көнне
Уятырга теләп
Яңгыраган әтәч моңнары.
Күкнең куеныннан
Таңны уятырлык,
Аңа охшаш серле моң бармы?
```
