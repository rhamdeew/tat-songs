# Оригинал

### Люция Әблиева / Эльвира Нигъмәтуллина - Иртән

```
Уяна иртән иртүк,
Сөрәнли: «Кик-ри-күк!»
Әтәчкә башка эш — чүп,
Тавык-чебешләре күп.

«Кик-ри-күк!», «Кыт-кыт-кыт!»
Ничек тиз уза вакыт?!
«Чип-чип-чип-чип-чип-чип-чип»
Чык суын куйдык эчеп!

«Кыт-кыт» кытлый тавыклар,
Тулымы,— дип,— савытлар?
«Чип-чип» чили чебиләр,
Бернәрсә дә белмиләр.

«Кик-ри-күк!», «Кыт-кыт-кыт!»
Ничек тиз уза вакыт?!
«Чип-чип-чип-чип-чип-чип-чип»
Чык суын куйдык эчеп!
```
