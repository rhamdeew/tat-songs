# Оригинал

### Рамил Чурагулов - Саклык банкы

```
Кем оекта, кем кыекта
Саклый тапкан акчасын,
Беркөн җирдә төзермен дип,
Гүзәл оҗмах бакчасын.
Акчаның милләте булмый.
Акча — акча инде ул.
Хәләл тапкан акчаңның син
Һәр тиененең кадерен бел!

Саклык банкы акча саклый,
Һәр тиен кадерен белеп.
Хуҗасына илтеп бирә,
Тиенне сумга үстереп.
Илтеп бирә хужасына,
Миллионнар иттереп.
Саклык банкы, сиңа халкың,
Дан җырлый, рәхмәт укып!

Хәләл акчалар да яна…
Ут-ялкыннар аяусыз.
Бур-караклар урлап китсә,
Калуың бар акчасыз.
Пыяла банкада тугел,
Чын банкта ятсын акчалар.
Дуслар саткан чакларда да,
Акчалар һич сатмыйлар.

Саклык банкы акча саклый,
Һәр тиен кадерен белеп.
Хуҗасына илтеп бирә,
Тиенне сумга үстереп.
Илтеп бирә хужасына,
Миллионнар иттереп.
Саклык банкы, сиңа халкың,
Дан җырлый, рәхмәт укып!

Дуслар булса, акча булса,
Яшәүләре мең рәхәт.
Саклык банкы акчаларга
Өсти куәт, бәрәкәт.
Ил-йортларның дан-шөһрәтен
Арттыра саклык банкы,
Исәнлек-саулык, байлыкта
Яшәсен диеп халкым.

Саклык банкы акча саклый,
Һәр тиен кадерен белеп.
Хуҗасына илтеп бирә,
Тиенне сумга үстереп.
Илтеп бирә хужасына,
Миллионнар иттереп.
Саклык банкы, сиңа халкың,
Дан җырлый, рәхмәт укып!
```
