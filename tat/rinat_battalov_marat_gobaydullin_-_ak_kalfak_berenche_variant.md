# Оригинал

### Ринат Батталов / Марат Гобәйдуллин - Ак калфак (беренче вариант)

```
Ак калфагың чиккән, бөккән,
Ука белән чигелгән.
Ай калфак, ак калфак
Сезгә бигрәк килешә,
Әйдә, җырлыйк янәшә.

Тырыш икән, матур икән
Безнен татар кызлары.
Ай калфак, ак калфак
Дәртләндерми кем җанын,
Мәңге матур бул, җаным.

Безнен татар эшчәннәре
Тирә-якта мактала.
Ай калфак, ак калфак,
Ак төсле ул, кар төсле,
Бер гөнаһсыз жан төсле.

Милләт кигән ак калфакны
Баш очыннан төшермә.
Ай калфак, ак калфак
Һәр ел саен шушылай
Куандыр безне, Ходай!
```
