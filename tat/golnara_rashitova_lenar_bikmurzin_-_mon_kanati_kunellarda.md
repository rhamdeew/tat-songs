# Оригинал

### Гөлнара Рәшитова / Ленар Бикмурзин - Моң канаты күңелләрдә

```
Сандугачлар сайраганы,
Чишмәләр чылтыраганы.
Күңелләргә, мәңгелеккә,
Моңнар булып уелганмы?
Моңнар булып уелганмы?

Моң канаты күңелләрдә,
Җыр күәте йөрәкләрдә.
Оча җырлар еракларга
Талпыналар биекләргә.

Моңнар белән тулгач күңел
Киртәләрне үтү җиңел.
Бу дөнъяда җырсыз — моңсыз
Яшәүләре мөмкин түгел.
Яшәүләре мөмкин түгел.

Моң канаты күңелләрдә,
Җыр күәте йөрәкләрдә.
Оча җырлар еракларга
Талпыналар биекләргә.

Сезне моңнар белән төреп
Көйне — көйләр белән үреп.
Эр яңадан туа җырлар
Безгә яшәү көчен биреп.

Моң канаты күңелләрдә,
Җыр күәте йөрәкләрдә.
Оча җырлар еракларга
Талпыналар биекләргә.
```
