# Оригинал

### Вәрис Акбашев / Юлай Үзәнбаев - Аем

```
Синең кулда минем күңел аем
Син күрәсең гашыйк җанымны.
Тау чишмәсе йөрәк моңын тыңлап
Ярсып аккан кайнар канымны.

Ишетмәгән чишмә мындый моңны
Тынып ҡалған тирән хулышта.
Йә, Ходаем ничә гасыр агып
Тәү тапкыр мин тындым сагышта.

Синең йөрәк минем күңел кылым
Кайсын чиртсәң шундый моң чыга.
Моңны тыңлап йолдызларым, аем
Әйләндергән икән тамчыга.
```
