# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Кайнар ипи

```
1. Әни салган кайнар икмәк
шул ипинең тәмләре.
Безне тылсым бала чакка
кайтара күңелләрне.

кушымта
Кайнар икмәк, кайнар ипи
аны бит әни салган.
Аның һәрбер валчыгында
күңел җылысы калган

2. Ипинең ярату хисе
аның кайнарлыгында
Гаилә дигән хисләрдә
бәхетнең барлыгында.

кушымта
Кайнар икмәк, кайнар ипи
аны бит әни салган.
Аның һәрбер валчыгында
күңел җылысы калган

3. Зур ускэч тэ шул чакларга
кайтасы килеп тора.
Кайда булсаң да туган өй
Бит үзенә чакыра.
кушымта
Кайнар икмәк, кайнар ипи
аны бит әни салган.
Аның һәрбер валчыгында
күңел җылысы калган
```
