# Оригинал

### Ризида Мөхәммәтҗанова - Әтиемнең тавышы

```
Ераклардан ишетелә
Әтиемнең тавышы.
Җырдай үтте гомеркәе,
Күрәчәге, язмышы.

Әтиме печән чаба?..
Барамын аңа табан.
Яланда эшли сыман —
Күңелдә ул җыр һаман.

Кайтсам, чыгам яланнарга,
Туры килсә ялларым.
Тик нигәдер табып булмый —
Ерагайган яланым.

Әтиме печән чаба?..
Эзлимен, син, дип, кайда?
Чалгысы чыңлый сыман,
Яланда микән һаман?..

Мәтрүшкәле чәй эчәбез —
Төшемдә шат елмая.
Әтиемнән дә изгерәк
Һичкем юк ул дөньяда.

Бала чагыма кайтып,
Көтәм мин ишек ачып.
Ничек кенә уйлама,
Язмыштан булмый качып.
Ничек кенә уйнама —
Язмыштан булмый качып.
```
