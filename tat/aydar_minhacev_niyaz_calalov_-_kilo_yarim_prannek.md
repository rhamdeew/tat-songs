# Оригинал

### Айдар Минһаҗев / Нияз Җәләлов - Кило ярым прәннек

```
Әле генә кайткан идек,
Инде китәргә вакыт.
Тутыр әнкәй бәрәңгеңне
Безгә бер-ике капчык.

Рәхмәт, әнкәй, рәхмәт яусын,
Без дә бит буш килмәдек.
Алып килдек, әнә, сиңа
Кило ярым прәннек.

Бир, кәбестә, бир әле, бир,
Алып китик кабак та.
Тагын нәрсә, дип сорама,
Безгә барысы да бата.

Рәхмәт, әнкәй, рәхмәт яусын,
Без дә бит буш килмәдек.
Алып килдек, әнә, сиңа
Кило ярым прәннек.

Тутыра күр, тутыр гына,
Итләр дә кирәк, әйдә.
Җаең булса, биреп җибәр,
Бер ун дистә күкәй дә.

Рәхмәт, әнкәй, рәхмәт яусын,
Без дә бит буш килмәдек.
Алып килдек, әнә, сиңа
Кило ярым прәннек.
```
