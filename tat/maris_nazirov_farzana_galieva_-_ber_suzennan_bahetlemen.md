# Оригинал

### Марис Нәзиров / Фәрзәнә Галиева - Бер сүзеңнән бәхетлемен

```
Бер сүзеңнән бәхетлемен,
Ярсу язларым, диген.
Караңгы төннәрем, димә,
Алсу таңнарым, диген.

Аяз көннәрем, дип ата,
Көләч кояшым, диген.
Җил-буранда, давылларда
Юлчым-юлдашым, диген.

Бер сүзеңнән бәхетлемен,
Йә, кил, иркәлә, диген,
Ягымлы үткәнем, димә,
Якты иртәгем, диген.
```
