# Оригинал

### Әхмәт Ерикәй / Әнвәр Бакиров - Эзләнү

```
Нурлар алып нур чишмәләреннән
Сукмакларга сибәр идем мин;
Сукмакларны түгел, бөтен илне
Саф нурларга күмәр идем мин.

Тапсам әгәр бәхет хәзинәсен,
Шатландырыр идем һәркемне.
Эзләнүгә бирәм, хезмәтемә
Бар көчемне, барлык дәртемне.

Эзлим, эзлим, табамын дип эзлим,
Нур чишмәсен, бәхет диңгезен,
Дөньядагы барлык матурлыкка
Күмәр идем бөтен җир йөзен.
```
