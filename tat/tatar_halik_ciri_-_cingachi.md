# Оригинал

### Татар халык җыры - Җиңгәчи

```
Сикреп төштем тирәккә лә.
Гөл яфрагы кирәккә.

Ай җиңгәчи,
Сөйгәнемне димләче.

Гөл яфрагы дару диләр
Яна торган йөрәккә.

Ай җиңгәчи,
Сөйгәнемне димләче.

Сандугачлар сайраса да,
Йөрәккә ял буладыр;

Ай җиңгәчи,
Сөйгәнемне димләче.

Биш минутның өч минуты
Сезне уйлап узадыр.

Ай, җиңгәчи,
Сөйгәнемне димләче.
```
