# Оригинал

### Татар халык җыры - Хәмдия (өченче вариант)

```
Агиделкәй аккын, суы салкын,
Барма якын, дулкын какмасын.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.

Бер битең айдай, берсе кояш,
Яман күздән аллам сакласын.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.

Ярлыгынай булып яраралмаган,
Җаннар көйдерер өчен яралган.
```
