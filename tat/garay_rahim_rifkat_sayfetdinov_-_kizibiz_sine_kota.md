# Оригинал

### Гәрәй Рәхим / Рифкать Сәйфетдинов - Кызыбыз сине көтә

```
Ташлап киттең син мине,
Тол хатын диләр мине.
Үпкәләргә хакым юк,
Мине ташлап киттең.
Ә безнең кызыбыз, нәни кызыбыз
Сине өзелеп көтә.

Ул бит иртән дә, кичен дә
Сине кайтыр дип көтә.
Ярата ул сөекле әткәсен,
Әмма сабый чагы ятимлектә үтә.

Мин инде көтмим сине,
Өздем күптән өметемне.
Шулай да күзләремне
Ала алмыйм тәрәзәдән.
Бүген дә кызыбыз-йолдызыбыз
Сине көтте янәдән.

Ул иртән дә, кичен дә
Сине кайтыр дип көтә.
Ярата ул сөекле әткәсен.
Әмма сабый чагы ятимлектә үтә.
```
