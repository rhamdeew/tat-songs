# Оригинал

### Миңҗамал Шәйхинурова / Илгиз Зарипов - Шәйдәүләт чишмәсе

```
Әрәмәле су буенда
Шәйдәүләт чишмәсе бар.
Киленнәрнең иртүк торып
Чишмәгә төшәсе бар.

Тирә ягы яшел чирәм,
Әйләнәсе яр гына.
Челтер-челтер каршы алыр,
Су алырга бар гына.

Чишмә суы белән чәйләр
Хуш исле, тәмле була.
Туганнар белән чәй эчсәң
Яшәүләр ямьле була.

Күңелләр гел шунда тарта,
Йөрсәк тә без еракта.
Тәүге юлын әткәй салган,
Чишмә җыры йөрәктә.

Иртәнге таң нуры төшкән
Матур манарасына.
Шаян яшьлегебез качкан
Каеннар арасына.

Шул чишмәдән су китергән
Яшьлегем кая икән?
Салкын суы йөрәктәге
Сагышка дәва икән.

Күңелләр гел шунда тарта,
Йөрсәк тә без еракта.
Тәүге юлын әткәй салган,
Чишмә җыры йөрәктә.
```
