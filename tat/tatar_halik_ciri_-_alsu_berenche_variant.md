# Оригинал

### Татар халык җыры - Алсу (беренче вариант)

```
Алъяпкычын алсу-алсу,
Алсуга мандыңмы әллә?
Бик ямансу утырасың,
Ярыңнан калдыңмы әллә?

Алъяпкычым алсу-алсу,
Алсуга манмасам да.
Моңаям мин, ямансулыйм,
Ярымнан калмасам да.

Алъяпкычың алмы әллә,
Алдыңда алма бармы әллә?
Әйләнәсең; тулганасың,
Бер йомышың бармы әллә?

Алъяпкычың ал-бишни,
Яңгыр көннәрдә нишли?
Кеше яры килә-китә,
Синеке өйдә нишли?
```
