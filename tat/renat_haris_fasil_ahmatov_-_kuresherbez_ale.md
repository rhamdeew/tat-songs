# Оригинал

### Ренат Харис / Фасил Әхмәтов - Күрешербез әле…

```
Хисләремә ялкын кабынган,
Өметләрем канат кагына.

Күрешербез әле,
Сөйләшербез әле
Кайчан да булса,
Кайда да булса,
Бер күрешү — үзе бер гомер.

Юк иде лә шулай талпынган,
Юк иде лә болай сагынган.

Күрешербез әле,
Сөйләшербез әле
Кайчан да булса,
Кайда да булса,
Бер күрешү — үзе бер гомер.

Эзебезне еллар күмгән,
Буран узган — ялкын сүнмәгән.

Күрешербез әле,
Сөйләшербез әле
Кайчан да булса,
Кайда да булса,
Бер күрешү — үзе бер гомер.
```
