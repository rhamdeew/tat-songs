# Оригинал

### Татар халык җыры - Авыл көе (дүртенче вариант)

```
Биек тауларның башында,
Ах, мин каеннар булсам;
Ах, үзем каеннар булып,
Син былбыл булып кунсаң.

Безнең янда салкын чишмә —
Гәрәбә эри торган;
Бер сандугач булсын икән
Арада йөри торган.

Ал дими күңелләрем,
Гөл дими күңелләрем;
Синнән башка бүтәннәрне
Теләми күңелләрем.
```
