# Оригинал

### Альфира Бикмәтова / Фәнир Галимов - Каз өмәсе (беренче вариант)

```
Каз өмәсе су буена
Озаталар казларны.
Ак йолдызлар бии бии
Иркәлиләр кызларны.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.

Чишмә суы көмеш кебек,
Кызлар казлар юалар.
Мамык кебек карны яра,
Болыт яллы юргалар.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.

Шәмдәй шома ак каеннар
Чишмә юл буен бизи.
Кояш кызы уянган да,
Елмая, кулын изи.

Андый чакта туган якның
Киче дә якты була.
Каз өмәсе олы бәйрәм:
Кунаклар килеп тула.
```
