# Оригинал

### Алмаз Юнысов - Бул минем янымда

```
Мин сине күргәнче бик озак эзләдем,
Гомерем буенда, чәчәкле болында.
Бер карашың булды, хисләр ташып куйды,
Гомерем буе бул янымда!

Бул минем янымда, мәхәббәт таңында!
Чәчәкле язларда наз бирик күңелгә!
Син булсаң янымда, онытып барын да,
Сөелеп яшәрбез без бергә!

Бул иркәм янымда шатлыктан шашкаңда,
Бул минем янымда кайгылы чагымда,
Без ике бер йөрәк, миңа тик син кирәк,
Иркәм бул янымда, янымда!

Бул минем янымда, мәхәббәт таңында!
Чәчәкле язларда наз бирик күңелгә!
Син булсаң янымда, онытып барын да
Сөелеп яшәрбез без бергә!
```
