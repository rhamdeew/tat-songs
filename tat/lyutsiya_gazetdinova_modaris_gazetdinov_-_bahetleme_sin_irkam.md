# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Бәхетлеме син, иркәм?

```
Гомер көндәлеген барлыйм
Бергә узган елларны.
Бәхетле итә алдыммы?
Белмим шул юлларыңны.

Бернигә дә үкенмичә
Гомер йомгагын сүтәм.
Мин үзем чиксез бәхетле.
Бәхетлеме син, иркәм?

Сөенәмен очратканга
Юлымда сине күптән.
Ә син мине сайлаганга
Үкенмисеңме, иркәм?

Бернигә дә үкенмичә
Гомер йомгагын сүтәм.
Мин үзем чиксез бәхетле.
Бәхетлеме син, иркәм?

Дөньялар яктырып китәр,
Шул сүзеңне ишетсәм.
Мин бәхетле синең белән,
Син диеп яшим, дисәң!

Бернигә дә үкенмичә
Гомер йомгагын сүтәм.
Мин үзем чиксез бәхетле.
Бәхетлеме син, иркәм?
Әйтче миңа чын күңелдән,
Барыбер сүзең көтәм.
Мин үзем чиксез бәхетле,
Бәхетлеме син, иркәм?
```
