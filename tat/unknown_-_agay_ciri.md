# Оригинал

### Газинур Морат / Айдар Фәйзрахманов - Агай җыры

```
Үтте гомер күмер чабып,
Чабалмадык атларда.
Газиз башлар читтә йөрде,
Ярлар калды ятларга.

Әй, җил, безнең сәламнәрне
Илтче Туган илләргә.
Сызгырып уз аулыбыздан
Без җырлаган көйләргә.

Үтте гомер…
Узды яшьлек,
Читләрдә ят һава да.
Әй, җил,
Туган җир һавасын
Алып килче дәвага.

Әй, җил, безнең сәламнәрне
Илтче Туган илләргә.
Сызгырып уз аулыбыздан
Без җырлаган көйләргә.

Сыздырып уз…
Сызуларның
Үз сихәте бар аның.
Исләренә төшмәбезме
Ятка калган ярларның…

Әй, җил, безнең сәламнәрне
Илтче Туган илләргә.
Сызгырып уз аулыбыздан
Без җырлаган көйләргә.
```
