# Оригинал

### Әлмира Шәрифуллина / Луиза Батыр-Болгари - Әбиемнең васыятьләре

```
Иңдәреңдә фәрештәләр,
Ди миңа ак әбием.
Фәрештәм син дип сөяләр
Әтием һәм әнием.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
Үзе тарих, тере тарих
Безнең әби-бабабыз.
Зурласак кына аларны
Олы булып калабыз.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
Әбиемнең васыятьләре
Офыгымны киңәйтә.
Иңнәремдә фәрештәләр
Изгелеккә өйрәтә.
Һәр иртәне — якты көнне
Фатиха белән башлыйм.
Һәр эшемне, гамәлемне
Изгелеккә багышлыйм.
```
