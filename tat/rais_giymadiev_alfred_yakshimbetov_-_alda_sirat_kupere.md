# Оригинал

### Рәис Гыймадиев / Альфред Якшимбетов - Алда сират күпере

```
Язмасын ялгыз калырга
Тормыш шаулап торганда.
Әйләнә-тирә ягыңда
Кешеләр кайнаганда.

Беркайчан да онытмагыз:
Алда сират күпере.
Әниләр безне кичерер,
Тик Ходай кичерерме?!

Күрәчәк килми кычкырып,
Көнне-төнне белми ул.
Ялгызны яклар кеше юк,
Кеше белән бергә бул!

Беркайчан да онытмагыз:
Алда сират күпере.
Әниләр безне кичерер,
Тик Ходай кичерерме?!

Санасаң, санап бетмәслек,
Ялгыз әти-әниләр.
Кичә ил тоткасы булган,
Бүген тере ятимнәр.

Беркайчан да онытмагыз:
Алда сират күпере.
Әниләр безне кичерер,
Тик Ходай кичерерме?!
```
