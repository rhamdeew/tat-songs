# Оригинал

### Ренат Галиәскәр / Әбри Хәбриев - Смайлик куеп елмайсам да…

```
Көннәр үтә бер-бер артлы,
Ә мин өлгермәм кебек.
Ачу да чыга кайвакыт
Кояш баюын күреп.
Хыяллардан максатларга —
Бу дөньяның бар яме.
Йөрәгем буш түгел. Менә
Шуңа да мин бәхетле.
Бәхетле, бәхетле…

Тапшырмаган хатларыма
Килә хатлар аласым.
Телим әле эзләремнең
Тирән уелып каласын.
Төн пәрдәсен ертып кайчак
Аттырамын таңнарны.
Рәхәт танырга улымда
Үземнең чалымнарны.

Смайлик куеп елмайсам да кайчак,
Күпләр белмәс ихлас хисләрне.
Күзгә карап ишетүче бар ул
Әйтелмичә калган сүзләрне.

Аңлый беләм: һәр кешенең
Үз дөреслеге барлыгын,
Бер кешедә тупланганын
Акның һәм дә караның.
Яңа иртә кисәтми кайчак
Мәшәкатьләргә күмә.
Кат-кат пышылдыйм уземә:
"Әйдә, туган, сүрелмә!"
Сүрелмә…

Кабатланмас кабат үткәннәрем.
Һәрбер көн башкача башлана.
Ялгышларсыз булмас. Иң беренче
Матурлык бит күзгә ташлана.

Смайлик куеп елмайсам да кайчак,
Күпләр белмәс ихлас хисләрне.
Күзгә карап ишетүче бар ул
Әйтелмичә калган сүзләрне.

Кабатланмас кабат үткәннәрем.
Һәрбер көн башкача башлана.
Ялгышларсыз булмас. Иң беренче
Матурлык бит күзгә ташлана.
Ант итүләр, акланулар артык.
Сүздә тору мөһим, кыскасы…
Дуслык, хөрмәт, ихтирам итүнең,
Яратуның бар үз шкаласы.

Смайлик куеп елмайсам да кайчак,
Күпләр белмәс ихлас хисләрне.
Күзгә карап ишетүче бар ул
Әйтелмичә калган сүзләрне.
Сүзләрне, сүзләрне…
```
