# Оригинал

### Финат Шакирҗанов / Юлай Моратов - Авылымда җиде урам

```
Авылымда җиде урам,
Тирәсе яшел чирәм.
Сандугачлар белән бергә,
Һәр язда кайтып күрәм.

Җиде урам, җиде яктан
Йөрәгемә бәйләнгән.
Таңга тиклем гармун тартып,
Җидесен бер әйләнәм.

Авылымда җиде урам,
Тора гел мине көтеп.
Күңелемнән сагышларны
Таратам җырлар илтеп.

Җиде урам, җиде яктан
Йөрәгемә бәйләнгән.
Таңга тиклем гармун тартып,
Җидесен бер әйләнәм.
```
