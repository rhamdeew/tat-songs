# Оригинал

### Ризида Мөхәммәтҗанова / Гүзәллия Хәкимҗанова - Парлылар җыры (икенче вариант)

```
Җанда — гөлем, сөю гөлем,
Гөлләргә күмсен көн дә.
Кояш күктән елмайганда
Җылы җиткерер җанга.

Кушымта:
Парлылар җаны моңлы,
Парлылар һәркөн җырлы,
Еллары матур үтә,
Өйләре булгач җылы!
Күзнең нурын яулап кына
Кояш алтын нур сибә.
Яктылыкта гомер узсак —
Бәхетле син дә, мин дә.

Кушымта.

Сыенамын бала сыман,
Без — бәхетле, без — парлы.
Хәзинәле безнең гомер,
Парлылар була җырлы.

Кушымта.
```
