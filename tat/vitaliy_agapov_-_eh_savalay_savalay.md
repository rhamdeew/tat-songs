# Оригинал

### Виталий Агапов - Эх, Сәвәләй, Сәвәләй

```
Һәркем сөя туган җирен,
Туган илен ярата.
Мин дә сезгә җырлыйм әле
Сәвәләем турында.

Эх, Сәвәләй, Сәвәләй,
Ерак түгел синнән Зәй,
Көймә белән кайтыр идем,
Инешебез бәләкәй.

Бардыр бездә Бакыр тавы,
Сандугач, Мәктәп тавы.
Күңелләрне җилкендерә
Каеннар, таллар шавы.

Эх, Сәвәләй, Сәвәләй,
Ерак түгел синнән Зәй,
Көймә белән кайтыр идем,
Инешебез бәләкәй.

Халык җырлый, бии бездә
Гармунның моңнарына.
Питрауларда төшеп китә
Ямьле Зәй буйларына.

Эх, Сәвәләй, Сәвәләй,
Ерак түгел синнән Зәй,
Көймә белән кайтыр идем,
Инешебез бәләкәй.
```
