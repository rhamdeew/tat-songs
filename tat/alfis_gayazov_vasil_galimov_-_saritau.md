# Оригинал

### Әлфис Гаязов / Вәсил Галимов - Сарытау

```
Сарытауга барам әле,
Сары май (бал) алам әле.
Башкаларга карар булсаң
Утларга салам әле.

Сарытауга барыйм микән,
Сары бал (май) алыйм микән?
Сарыларым, сагышларым
Суларга салыйм микән?

Сарытауга барам әле,
Иделне кичәр өчен
Коштай очып барыр идем
Серемне чичәр өчен.

Сарытауга барыйм микән,
Сары бал (май) алыйм микән?
Сарыларым, сагышларым
Суларга салыйм микән?

Сарытауга барам әле,
Сары гөл алам әле.
Сагындырып та килмәсәң
Яныңа барам әле.

Сарытауга барыйм микән,
Сары бал (май) алыйм микән?
Сарыларым, сагышларым
Суларга салыйм микән?
```
