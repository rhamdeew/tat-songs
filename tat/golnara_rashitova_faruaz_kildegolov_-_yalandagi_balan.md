# Оригинал

### Гөлнара Рәшитова / Фәрүәз Килдеголов - Яландагы балан

```
Күпме гомер читтә узган
Тик төшкә керә һаман.
Туган якларымда калган
Баланлы ямьле ялан.

Яландагы балан
Ап-ак чәчкәләр аткан.
Сине сагынам һаман
Балан чәчкәле ялан.

Иң бәхетле чагым булган
Балачак синдә узган.
Туган як еракта калган
Күң(е)лемнән чыкмый һаман.

Яландагы балан
Ап-ак чәчкәләр аткан.
Сине сагынам һаман
Балан чәчкәле ялан.

Тәүге яр да синдә калган
Үкенеп йөрәк янган.
Шуңадыр тәлгәшле балан
Ачы яшь тәмен алган.

Яландагы балан
Ап-ак чәчкәләр аткан.
Сине сагынам һаман
Балан чәчкәле ялан.
```
