# Оригинал

### Татар халык җыры - Су буйлап (бишенче вариант)

```
Идел бит ул, тирән бит ул,
Тирән бит ул, киң бит ул.
Караңгы төн, болытлы көн —
Без аерылган көн бит ул.

Су буйлары моңсу була
Җәйнең алсу таңында;
Тормыш дулкыннары безне
Аерды бит тагын да.

Су буйлары тыныч була,
Ямьле була кич-иртән;
Кайтырсың, бәгърем, кайтырсың,
Тик көнен белмим, иркәм.

Су буеннан сулар алып,
Су сибәм гөлләремә;
Мәңге шиңмәс гөлләр булып
Калдың күңелләремдә.

Су буйлары моңсу иде,
Без айрылган чагында;
Тормыш дулкыннары безне
Кушар әле тагын да.
```
