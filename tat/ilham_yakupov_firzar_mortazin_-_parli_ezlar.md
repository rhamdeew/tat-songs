# Оригинал

### Илһам Якупов / Фирзәр Мортазин - Парлы эзләр

```
Барам әле урам буйлап, нидер уйлап.
Эре-эре карлар ява кайнап-кайнап.
Күз камаша аклыклардан-сафлыклардан,
Саф һавасын булмый туеп һич тә сулап.

Яндырмачы, ак(ы)лым, сагыш утларында,
Дөнья матур бары парлап атлаганда.
Янәшәдә һәрчак булсын аклык-сафлык,
Китерсеннәр туган көннәр бары шатлык.

Барам әле урам буйлап, иптәш булып.
Мондый матурлыктан булмый һич тә туеп.
Бәхет бит ул һәрчак безнең янәшәдә,
Яшәвебез ничек рәхәт шуны тоеп.

Яндырмачы, ак(ы)лым, сагыш утларында,
Дөнья матур бары парлап атлаганда.
Янәшәдә һәрчак булсын аклык-сафлык,
Китерсеннәр туган көннәр бары шатлык.

Атлыйк әле урамнарны парлап-парлап,
Кадерен белеп булган чакта янда парлар.
Артыбыздан һәрчак калсын парлы эзләр,
Ташып торсын күңелләрдән матур сүзләр.

Яндырмачы, ак(ы)лым, сагыш утларында,
Дөнья матур бары парлап атлаганда.
Янәшәдә һәрчак булсын аклык-сафлык,
Китерсеннәр туган көннәр бары шатлык.
Янәшәдә һәрчак булсын аклык-сафлык,
Китерсеннәр туган көннәр бары шатлык…
```
