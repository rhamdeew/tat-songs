# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Безнең гомер (икенче вариант)

```
Син көткәндә — мин килмәдем,
Мин килгәндә — син көтмәдең,
Барсы да ялгыш булды*.

Бер-беребезне яратсак та,
Син дип, мин дип, җан атсак та
Әллә нинди язмыш булды.

Безне сөю табыштырды,
Саф мәхәббәт кавыштырды,
Бөтен гомер ялгыш узды. /Гомер тоташ сагыш булды./

Бер-беребезне яратсак та,
Син дип, мин дип, җан атсак та
Әллә нинди язмыш булды.

Ул моң-зарлар бетмәс бер дә,
Мәңге калыр күңелләрдә,
Син көткәндә — мин килмәдем. /Син килмәдең — мин көткәндә./

Бер-беребезне яратсак та,
Син дип, мин дип, җан атсак та
Әллә нинди язмыш булды.

* — Һәр күплет ике кабат җырлана
```
