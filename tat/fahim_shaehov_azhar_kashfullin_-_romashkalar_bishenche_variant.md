# Оригинал

### Фәһим Шәехов / Әзһәр Кәшфуллин - Ромашкалар (бишенче вариант)

```
Яратамы, яратмыймы?
Ярата микән?

Ромашкаларның керфеген
Өзмәгезче берәмләп.
Күңелләрне сөендереп,
Кырларны тора ямьләп.

Ромашкаларның керфеген
Өзгәч булмый шул ялгап.
Яратамы, яратмыймы?
Язмышны булмый алдап.

Гөлләрнең дә җаннары бар,
Тик сөйләшә белмиләр.
Ак керфекләре өзелгәч,
Еламыйлар, көлмиләр.

Ромашкаларның керфеген
Өзгәч булмый шул ялгап.
Яратамы, яратмыймы?
Язмышны булмый алдап.

Яратамы, яратмыймы?
Ярата микән?
Яратамы?
```
