# Оригинал

### Рәми Гарипов / Нәзир Әбдиев - Башкортстаным (дүртенче вариант)

```
Башкортстаным чиген үтү белән,
Туып-үскән җиргә җитү белән,
Нур тулгандай була күңелгә,
Җыр тугандай була күңелдә.

Башкортстаным — безнең Бөркетстаным,
Иркең курчып, күпме бөркет очкан!..
Башкортстаным — йөрәк кагышым,
Җирсенмәсәм, төшсен каргышың!

Яшел бәрхәт төсле җиркәйгенәм,
Кызлар кулы төсле җилкәйгенәм
Иркәләй бит талган күземне,
Иркәләй бит арыган йөземне.

Башкортстаным — безнең Бөркетстаным,
Иркең курчып, күпме бөркет очкан!..
Башкортстаным — йөрәк кагышым,
Җирсенмәсәм, төшсен каргышың!

Йөгереп чыгу белән каеннарым,
Онытылды барлык кайгыларым,
Күрү белән күген Уралның,
Күзләремә яшьләр уралды…

Башкортстаным — безнең Бөркетстаным,
Иркең курчып, күпме бөркет очкан!..
Башкортстаным — йөрәк кагышым,
Җирсенмәсәм, төшсен каргышың!

Юк, кирәкми миңа аһ-ваһларың,
Ары торсын җәннәт-оҗмахларың,
Башкортстаным — туган төйәгем,
Синдә ятсын ятса сөягем!..

Башкортстаным — безнең Бөркетстаным,
Иркең курчып, күпме бөркет очкан!..
Башкортстаным — йөрәк кагышым,
Җирсенмәсәм, төшсен каргышың!
```
