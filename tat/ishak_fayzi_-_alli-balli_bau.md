# Оригинал

### Исхак Фәйзи - Әлли-бәлли бәү

```
Иртән иртүк торабыз,
Кырда урак урабыз.
Көрмәле бәби арбасын
Кыр читенә куябыз.

Куш:
Әлли-бәлли бәү,
Әлли-бәлли бәү,
Уракчылар җырын тыңлап,
Сабыебыз үссен дәү.

Чайкалалар игеннәр,
Урак ура киленнәр.
Көрмәле арба янында
Үсә бара кибәннәр.

Йокла, йокла сабыем,
Эш эшләсен әниең.
Саф һавада йокың туйгач,
Тәмле булыр мәмиең.
```
