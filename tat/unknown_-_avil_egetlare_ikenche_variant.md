# Оригинал

### Хәйдәр Әхмәтшин / Шахзат Размәтов - Авыл егетләре (икенче вариант)

```
Авылларда гына буладыр ул,
Егетләрнең менә дигәне!
Алар гына белә бәхетләрнең
Туган якта гына икәнен!..

Эх, бар әле авылларда
Менә дигән зәһәр егетләр.
Ничек кенә тормыш сынаса да,
Алар баш иярлек түгелләр!

Ут уйнаткан алар кулы белән
Күпме матур йортлар төзелә.
Мәйданнарда ирләр көрәшкәндә
Сөлгеләр дә шартлап өзелә.

Эх, бар әле авылларда
Менә дигән зәһәр егетләр.
Ничек кенә тормыш сынаса да,
Алар баш иярлек түгелләр!

Никадәр дә кырыс булсалар да
Күңелләре матур һәрвакыт.
Яратсалар чынлап яраталар,
Сөйгәннәре өчен җан атып.

Эх, бар әле авылларда
Менә дигән зәһәр егетләр.
Ничек кенә тормыш сынаса да,
Алар баш иярлек түгелләр!
```
