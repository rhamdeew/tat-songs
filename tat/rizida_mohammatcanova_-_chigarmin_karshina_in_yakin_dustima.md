# Оригинал

### Ризида Мөхәммәтҗанова - Чыгармын каршыңа (Иң якын дустыма)

```
Синең йөрәгең дә сызланадыр,
Ара ерак, син — ерак илдә,
Бүләккәең — челтәр шәлне көн дә
Сыйпап-сыйпап салам мин иңгә.

Кушымта:
Кайтырыңны хәбәр ит,
Чыгармын каршыңа,
Куанып кайт, шатлыклар
Очрасын каршыңа.

Кайталмасаң бүген, бер кайтырсың,
Ишек ачык, тик син кайт кына,
Юлларыңа дога кылам көн дә,
Кайтыр юлың булсын ак кына.

Кушымта.

Гаҗәп бизәкле шәл, сокланамын,
Серләр бардыр челтәр һәр күздә.
Кояшлы көн, якшы хәбәр килде:
“Кайтам, бәгърем”, дигән сүз белән.

Кушымта:
Кайтасың бит, кадерлем,
Чыгамын каршыңа,
Шатлыклы көн, шәлеңне
Ябындым башыма.
```
