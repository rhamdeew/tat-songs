# Оригинал

### Миләүшә Әфләтунова / Гаяз Селявсинов - Синең яныңда

```
Өзелеп тирәктә сайрый сандугач,
Күмелә яшел бөрегә һәр агач.
Балкый, яна күктә алсу кояш,
Нуры сибәдер сөю, наз.
Янымда син булган чагында
Минем күңелдә һәрчак яз.

Синең яныңда
Минем җанымда
Уяна алсу җылы таң.
Иркәм, яныңда
Сиңа карыйм да
Барысын да мин онытам.

Өзелеп тирәктә сайрый сандугач,
Кушыла аның моңына карлыгач.
Ата гүзәл гөлләр, исми җил дә,
Сөюнең күге гел аяз.
Күземә карап син елмайсаң
Минем күңелдә кабат яз.

Синең яныңда
Минем җанымда
Уяна алсу җылы таң.
Иркәм, яныңда
Сиңа карыйм да
Барысын да мин онытам.
Синең яныңда
Минем җанымда
Уяна алсу җылы таң.
Нур карашыңда,
Кара кашыңда,
Йөрәкне минем җылыта.
Синең яныңда
Минем җанымда
Уяна алсу җылы таң.
Нур карашыңда,
Кара кашыңда,
Йөрәкне минем җылыта.

Минем җанымда
Уяна алсу җылы таң.
Иркәм, яныңда
Сиңа карыйм да
Барысын да мин онытам.
```
