# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Гүзәл җыры

```
Әнкәйнең назлы кызы мин
Сайрар кошчык — илемдә.
Якты моңнар, матур җырлар
Яши гүзәл күңлемдә

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Сөя җирем, назлый җилем,
Тоям кояш җылысын,
Үсәм шатлык өстәп җиргә,
Көтеп таңның җырлысын.

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Таңда-кошчык, төндә-йолдыз
Булып яшисем килә.
Күңелләргә якты моңнар,
Җыр өләшәсем килә.

Җырлар өчен сезгә, дуслар,
Дөньяга мин туганмын.
Моңнар булып җиргә
Кунганмын.

Моңнар булып кунганмын.
```
