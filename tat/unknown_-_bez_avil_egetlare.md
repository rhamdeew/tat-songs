# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Без — авыл егетләре

```
Авыр чакта сыгылмаска
Авыл безне өйрәтте.
Ай шикелле ялтыраттык,
Күгәртмәдек көрәкне.

Без бит авыл егетләре,
Иркә булып үсмәдек.
Көнгә төнне ялгый-ялгый
Эшләдек тә эшләдек.

Сөйдек ярның гүзәлләрен
Иркәләп, назлап кына.
Безнең яшьлек талларына
Сандугач сайрап куна.

Без бит авыл егетләре,
Уйныйбыз да җырлыйбыз.
Кирәк икән биибез дә,
Ялындырып тормыйбыз.

Дан артыннан да кумыйбыз,
Кызыкмыйбыз байлыкка.
Безнең байлык — җир-су, кояш,
Йолдызлар бар ай юкта.

Без — авыл егетләре, дип
Горурлансак та ярый.
Калада торсак та, күңел
Авыл ягына карый.
```
