# Оригинал

### Әнисә Таһирова / Рафаэль Шәхрәзиев - Килереңне көттем

```
Килереңне (килеүеңне) көттем моңсуланып ,
Һаман килмәдең син, килмәдең.
Йөрәгемдә сүнмәс ялкыннарның кабынганын
Белмәдең, белмәдең, белмәдең.

Сөюемнең (йөрәгемнең) акчарлагы булып
Канат какмадың син, какмадың.
Йөрәгемдә өнсез сагышларым ташкан чакта (мәлдә)
Күрмәдең, күрмәдең, күрмәдең.

Юл үткәндә юлдарымдан язма,
Шулай гел генә син, гел генә.
Йөрәгемдә мәңге (һәр чак) шиңмәс (сүнмәс) шатлык җыры чыңлар,
Кил генә, кил генә, кил генә.
```
