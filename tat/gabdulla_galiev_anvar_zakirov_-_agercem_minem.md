# Оригинал

### Габдулла Галиев / Әнвәр Закиров - Әгерҗем минем

```
Гөрләп үсә Әгерҗебез
Туган ил кочагында.
Былбыл моңы талларында
Таңнар аткан чагында.

Иж буйлары бигрәк ямьле,
Бормалы юллар;
Яшьлегемә канат булып
Тибрәлә җырлар.

Ягыбызны иңләп уза
Илнең корыч юллары.
Киң кырларның Чал Уралга
Кушылган урыннары.

Иж буйлары бигрәк ямьле,
Бормалы юллар;
Яшьлегемә канат булып
Тибрәлә җырлар.

Бөдрә наратларга карап
Елмая зәңгәр Кама.
Ераклардан сәлам биреп,
«КамАЗ» утлары яна.
```
