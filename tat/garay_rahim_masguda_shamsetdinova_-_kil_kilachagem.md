# Оригинал

### Гәрәй Рәхим / Мәсгудә Шәмсетдинова - Кил, киләчәгем

```
Кил, кил, кил, кил,
Минем киләчәгем,
Син, син, син, син —
Минем күрәчәгем,
Яначагым, көячәгем,
Утларга керәчәгем.

Кил, минем киләчәгем,
Син — минем күрәчәгем,
Син —минем ап-ак чәчләргә,
Нурга төренәчәгем.

Кил, кил, кил, кил,
Минем киләчәгем,
Син, син, син, син
Минем күрәчәгем,
Бәхетемне даулый-даулый
Утларга керәчәгем.

Кил, минем киләчәгем,
Син — минем күрәчәгем,
Син —минем ап-ак чәчләргә,
Нурга төренәчәгем.
```
