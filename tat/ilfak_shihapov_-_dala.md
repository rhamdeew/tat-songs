# Оригинал

### Илфак Шиһапов - Дала

```
Кылганлы Бөек дала,
Ярсыган атлар чаба.
Баш очында эленгән
Бер ялгыз Тилгән.

Артта калыптай тузан,
Атлар ерак юл узган.
Горур басып калалар
Дәшмәс балбаллар.
Артта калыптай тузан,
Атлар ерак юл узган.
Горур басып калалар
Дәшмәс балбаллар.

Төнге далада утлар,
Ял итә арган атлар.
Иртәгә тагын юлга
Чакыра дала.

Очкыннар чыга таштан,
Көн белән төн тоташкан.
Алда бетмәс юл көтә
Тоташып күккә.
Очкыннар чыга таштан,
Көн белән төн тоташкан.
Алда бетмәс юл көтә
Тоташып күккә.
```
