# Оригинал

### Мөхәммәтнур Муллануров / Илгиз Кадыйров - Сабан туе — дуслык туе

```
Сабан туе — дуслык туе,
Җәйнең матур чагында, шул;
Гөлләр чәчәккә күмелгән
Табигать кочагында,
Табигать кочагында.
Сабантуйга килгән кунак —
Кадерле кунагыбыз, шул;
Сезнең белән бик күңелле,
Җырлашып куанабыз,
Җырлашып куанабыз.
Сабантуйда төрле милләт —
Мари, чуаш, руслар, шул;
Бәйрәм итәбез гөрләшеп,
Без барыбыз да — дуслар,
Без барыбыз да — дуслар.
```
