# Оригинал

### Башкорт халык җыры - Косилка, молотилка (икенче вариант)

```
Уйнап алаек әле /лә/,
Җырлап алаек әле,
Уйнаек та, җырлаек та,
Биеп алаек әле.

Косилка, молотилка,
Сортировка, веялка.

Уйныйбыз бергә-бергә /лә/,
Биибез бергә-бергә.
Уйнап-көлеп, шау-гөр килеп,
Эшлибез бергә-бергә.

Косилка, молотилка,
Сортировка, веялка.

Алай итербез әле,
Былай итербез әле.
Алай итербез әле /лә/,
Былай итербез әле.

Косилка, молотилка,
Сортировка, веялка.

Уйныйбыз бергә-бергә /лә/,
Биибез бергә-бергә.
Уйнап-көлеп, шау-гөр (гөрләп) килеп,
Яшибез бергә-бергә.

Косилка, молотилка,
Сортировка, веялка.
```
