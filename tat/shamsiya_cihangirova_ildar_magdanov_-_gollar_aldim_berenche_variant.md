# Оригинал

### Шәмсия Җиһангирова / Илдар Мәгъдәнов - Гөлләр алдым (беренче вариант)

```
Гөлләр алдым, суга салдым,
Агып китте гөлләрем.
Аккан сулар кебек яшьлек
Үтәр диеп белмәдем…

Гөлләр алдым, суга салдым,
Тотып адылар микән?
Гөлләр кебек яшьлегемне
Күреп калдылар микән?

Кире барсам, гөлләр алсам,
Суларга салмас идем.
Яшьлегемнең кадерләрен
Белмичә калмас идем.
```
