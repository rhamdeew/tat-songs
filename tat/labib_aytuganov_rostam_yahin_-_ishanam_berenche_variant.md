# Оригинал

### Ләбиб Айтуганов / Рөстәм Яхин - Ышанам (беренче вариант)

```
Ерак хәзер арабыз —
Урман, таулар аера.
Тик күңелгә якын син,
Якын, янымда гына.

Очрашырбыз, ышанам,
Ләкин кайчан — билгесез.
Нигә моңсу күңелгә —
Яшәгәнгәме синсез?

Кем уйлаган, аерылгач,
Ялкын кабыныр, диеп.
Ямансулар бу күңел,
Өзелеп сагыныр, диеп.

Очрашырбыз, ышанам,
Ул көн бик якын кебек.
Аңла, туган ягыңда
Сине көтә бер егет.
```
