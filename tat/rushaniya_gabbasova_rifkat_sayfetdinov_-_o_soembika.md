# Оригинал

### Рушания Габбасова / Рифкать Сәйфетдинов - О, Сөембикә!

```
Гасырлардан яңгырый һаман
Кайтаваздай — Сөембикә!
Казан ханлыгының даны
Аның белән ерак китә.

Сөембикә, Сөембикә,
Татарның гүзәл кызы!
Сөембикә, Сөембикә!
Син — азатлык йолдызы!

Кара болыт — дошман явы —
Татар көрәшкә кузгала.
Иленә, сөйгән халкына
Сөембикә тугры кала.

Сөембикә, синең исемең
Гасырларга күчеп бара.
Син татарның горурлыгы,
Сөембикә, хатын, ана.
```
