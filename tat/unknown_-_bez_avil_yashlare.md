# Оригинал

### Роза Хәмидуллина / Гафият Нигъмәтуллин - Без авыл яшьләре

```
Омтылып яктыга,
Заманча яшәргә,
Дусларча берләшик
Туган дип дәшкәнгә.

Ихластан тырышкач,
Алдынгы эшләре.
Бар яктан сәләтле
Без авыл яшьләре.

Тирәндә тамырлар,
Авылдан без чыккан.
Шул җирдән алган көч
Җирләрдә чыныккан.

Ихластан тырышкач,
Алдынгы эшләре.
Бар яктан сәләтле
Без авыл яшьләре.

Күмәк көч кузгата,
Тауларны кубарып.
Эх, калсын тормышта
Бездән соң нур булып.

Ихластан тырышкач,
Алдынгы эшләре.
Бар яктан сәләтле
Без авыл яшьләре.
```
