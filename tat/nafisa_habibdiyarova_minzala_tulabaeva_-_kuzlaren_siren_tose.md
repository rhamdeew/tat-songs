# Оригинал

### Нәфисә Хәбибдиярова / Миңзәлә Түләбаева - Күзләрең — сирень төсе

```
Сирень сәйләне көела,
Күк йөзенең яшедәй.
Йолдыз-йолдыз күгең елар
Күзләреңне яшермә.

Серләремне сөйләр идем,
Күк сирень — күкләр төсе.
Күккә карыймын, сиңа карыймын,
Күзләрең шундый хисле.

Күзләреңдә — күп сиреннәр,
Күзләреңдә — күк төсе.
Сирень төсе, сирень исе,
Язлардай яшьлек хисе.

Серләремне сөйләр идем,
Күк сирень — күкләр төсе.
Күккә карыймын, сиңа карыймын,
Күзләрең шундый хисле.

Бик сагынсаң күзләреңне,
Зәңгәр күккә карармын.
Күк сиреньдәй кемдә генә
Карашыңны табармын.

Серләремне сөйләр идем,
Күк сирень — күкләр төсе.
Күккә карыймын, сиңа карыймын,
Күзләрең шундый хисле.
```
