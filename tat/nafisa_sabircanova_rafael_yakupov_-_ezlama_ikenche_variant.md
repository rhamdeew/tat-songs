# Оригинал

### Нәфисә Сабирҗанова / Рафаэль Якупов - Эзләмә (икенче вариант)

```
Эзләмә карлы урамда
Юлдан минем эзләрне.
Эзләмә, күрә алмассың
Сөю тулы күзләрне.
Эзләмә, ишетә алмассың
Назлы таныш тавышны.
Эзләмә, күптән югалттык
Ул иң якты сагышны.
Эзләмә мине болыннан —
Кыңгыраулы, җиләкле.
Эзләмә мине бураннан, —
Аңламадың кирәкне.
Эзләмә акланыр сүзләр,
Туплап бар хәтереңне.
Эзләмә, таба алмассың
«Сөеклең», «кадерлең»не.
```
