# Оригинал

### Дилә Булгакова / Артур Туктаголов - Әткәй йорты (беренче вариант)

```
Әткәйле йорт — атлы йорт,
Төзек ипле, маллы йорт.
Әткәйле йорт — кунакчыл йорт,
Бишекле йорт, баллы йорт.
Әткәйле йорт — җылы йорт,
Учагыңда сүнмәс ут.
Арбалы йорт, чалгылы йорт,
Бу йорттан һич китмәс кот.

Гореф-гадәтен саклаган,
Нигезле йорт, нурлы йорт.
Балалы йорт, курайлы йорт,
Тальян моңлы, җырлы йорт.

Изге тупсаң каршы алган
Табынлы йорт, туганчыл йорт.
Фатыйхасын биреп калган,
Бисмиллалы газиз йорт.

Гореф-гадәтен саклаган,
Нигезле йорт, нурлы йорт.
Балалы йорт, курайлы йорт,
Тальян моңлы, җырлы йорт.
Әткәйле йорт, нурлы йорт.
```
