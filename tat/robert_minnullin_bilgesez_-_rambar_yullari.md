# Оригинал

### Роберт Миңнуллин / Билгесез - Рамбар юллары

```
Сиңа барыр юлларыма
Кар ява бурап-бурап.
Рамбар юллары бик ерак,
Көтәсең микән елап?

Кайтасың икән яңадан —
Шул юлдан бер уз гына.
Рамбар юллары боз гына,
Күз яшьләрем — тоз гына.

Ишетәсең микән мине,
Җырымны сиңа сузам.
Рамбар юллары — бик тузан,
Тузан булса ла узам.

Бер яңгызың чыгасыңмы
Юлларга мине юллап?
Рамбар юллары — бик ерак,
Кайтырмын әле урап.
```
