# Оригинал

### Муса Җәлил / Җәүдәт Фәйзи - Карак песи

```
Чоландагы ак майга
Ияләшкән күселәр,
Мин чоланны сакларга
Куштым ала песигә:

— Мияу, мияу, пескәем!
Барчы, зинһар, чоланга,
Майны сакла күседән,
Оясыннан чыгарма!

Песи күргәч күселәр
Койрыкларын кыстылар.
Йөрмәделәр чабышып,
Ярыкларга постылар.

Мияу, мияу, пескәем!
Колак салдың сүземә!
Каравылдан бушагач,
Сөт бирермен үзеңә!

Иртә белән майга дип
Керсәм, күзем акайды:
Явыз песи берүзе
Ялт иттергән ак майны!

— Мияу, мияу, пескәем,
Хыянәтче икәнсең.
Саклый торган маеңны
Үзең ашап киткәнсең!
```
