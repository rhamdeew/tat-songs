# Оригинал

### Әкрәм Даутов - Ана җавабы

```
Ничә тапкыр инде
Якты язлар килде,
Ничә тапкыр тамды тамчылар.
Юлларыңа карап,
Көннәремне санап (барлап),
Хәбәр көтәм һәрбер юлчыдан.

Бәлки, эше күптер,
Бәлки, вакыты юктыр,
Бәлки, күптер якын дуслары?
Тик (һич) күңелемнән генә
Китмисең шул бер дә, / Чыкмыйсың шул бер дә, /
Һич (тик) китми шул яулык очлары.

Мына тагын матур
Якты язлар килер,
Кайтыр әле бер, — дип юанам.
Тәрәзәгә карап,
Чәчләремне тарап,
Чәч толымнарымнан юл салам.
```
