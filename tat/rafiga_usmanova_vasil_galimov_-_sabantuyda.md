# Оригинал

### Рәфига Усманова / Вәсил Галимов - Сабантуйда

```
Ул чакта өр-яңа матур
Күлмәгем кигән идем.
Сабантуйда авылыгызга
Кунакка килгән идем.

Сабантуйга, сабантуйга
Нигә ашкына күңел?
Сабантуйда, сабантуйда
Нигә безнең туй түгел?

Син ул чакта көрәшләрдә
Батырга калган идең.
Чиккән сөлгем алган идең,
Озатып барган идең.

Сабантуйга, сабантуйга
Нигә ашкына күңел?
Сабантуйда, сабантуйда
Нигә безнең туй түгел?

Без башкача очрашмадык
Теләк киртәләр кормый.
Тормыш нигә гел тоташтан
Сабантуйлардан тормый?

Сабантуйга, сабантуйга
Нигә ашкына күңел?
Сабантуйда, сабантуйда
Нигә безнең туй түгел?
```
