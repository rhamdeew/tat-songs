# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Бәхетле онык

```
Олы бәхет бүләк итеп,
Арттырып гаилә ямен.
Сабый бүген аваз салды,
Оныгым туды бүген.

Кушымта:
Дәвамчысы гаиләбезнең,
Бәхетле бул сабыем.
Әти-әни җылысында
Узсын килер гомерең.

Җил-давылдан саклап сине
Күз нурым, дип дәшәбез.
Аллаһ биргән олы бүләк,
Без син диеп яшәрбез.

Кушымта:

Әниеңнең өмете бул,
Әтиеңнең кызчыгы.
Әби-бабайларың өчен
Яраткан оныкчыгы.

Кушымта:

Тәүфыйклы бул, сәламәт бул,
Тигезлектә үс җимеш.
Баламның бар өмете син,
Син — безнең алтын, көмеш.
```
