# Оригинал

### Илдар Ногманов - Авылкаем (өченче вариант)

```
Кайтып киләм авылыма,
Каршы ала күл буйлары.
Балык тотып, уйнап үскән
Инешләре, елгалары.
Кайтып киләм авылыма,
Сукмак төшкән урамнарың.
Урап узам әле тагын
Бер туздырып тузаннарын.

Сагындыра, сагындыра
Кайтмый торсам авылыма.
Кайта сиңа балакаең,
Хәлләр ничек, авылкаем?

Кайтып киләм авылыма,
Көткәнсеңдер син дә мине.
Тәпи киткән туган нигез,
Туган өем тарта мине.
Ни хәлләр сез, туганнарым,
Әти-әни, якыннарым?
Кайтып киләм сезгэ тагын,
Сагындым бит, авылкаем.

Сагындыра, сагындыра
Кайтмый торсам авылыма.
Кайта сиңа балакаең,
Хәлләр ничек, авылкаем?
```
