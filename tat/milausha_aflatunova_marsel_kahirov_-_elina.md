# Оригинал

### Миләүшә Әфләтунова / Марсель Каһиров - Элина

```
Яшерен хисләр, яшерен сөю
Миңа бүләк итте сары көзләр.
Туктады вакыт, мин тынып калдым,
Мине әсир итте сихри күзләр.
Елмаеп син карасаң —
Оялам сабый сыман.
Сөюне хисләремне
Сизмисең минем һаман.
Мәхәббәтемне саклыйм,
Яши йөрәгемдә сер.
Синең моңлы көйләрең
Мәңге тыңларга әзер.

Көннәрен Элина,
Төннәрен Элина,
Йөрәк син дип тибәдер.
Уйларымда
Көннәрен Элина,
Төннәрен Элина
Төшләремә киләдер.
Көннәрен Элина,
Төннәрен Элина,
Йөрәк син дип тибәдер.
Уйларымда
Көннәрен Элина,
Төннәрен Элина
Төшләремә киләдер.

Сагынып көтә җаным күрешү,
Булса да алар бары биш минут.
Син минем янда булган чагында
Өзелеп яна йөрәгемдә ут.
Елмаеп син карасаң —
Оялам сабый сыман.
Сөюне хисләремне
Сизмисең минем һаман.
Мәхәббәтемне саклыйм,
Яши йөрәгемдә сер.
Синең моңлы көйләрең
Мәңге тыңларга әзер.

Көннәрен Элина,
Төннәрен Элина,
Йөрәк син дип тибәдер.
Уйларымда
Көннәрен Элина,
Төннәрен Элина
Төшләремә киләдер.
Көннәрен Элина,
Төннәрен Элина,
Йөрәк син дип тибәдер.
Уйларымда
Көннәрен Элина,
Төннәрен Элина
Төшләремә киләдер.
```
