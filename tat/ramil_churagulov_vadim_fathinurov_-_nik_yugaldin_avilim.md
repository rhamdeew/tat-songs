# Оригинал

### Рамил Чурагулов / Вадим Фәтхинуров - Ник югалдың, авылым!?

```
Бер генә урамлы идең,
Минем өчен зур идең.
Берничә өең калса да,
Син дөньяда юк инде.

Нишләдең, ни булды сиңа?
Ник югалдың, авылым?
Дөньяда авыллар бик күп.
Мөмкин түгел, мөмкин түгел
Тик синдәен табуы.

Безнең балачак эзләрен
Саклый авыл урамы.
Буш урамда ишетелә
Бүген буран улавы.

Нишләдең, ни булды сиңа?
Ник югалдың, авылым?
Дөньяда авыллар бик күп.
Мөмкин түгел, мөмкин түгел
Тик синдәен табуы.

Урам буйлап аунап ята
Нигездәге ак ташлар.
Кәртәләнмәгән зиратта
Дога көтә әрвахлар.

Нишләдең, ни булды сиңа?
Ник югалдың, авылым?
Дөньяда авыллар бик күп.
Мөмкин түгел, мөмкин түгел
Тик синдәен табуы.

Кешеләрдәй авыллар да
Туа һәм үлә икән…
Авыллылар авылсызның
Хәлләрен белә микән?

Нишләдең, ни булды сиңа?
Ник югалдың, авылым?
Дөньяда авыллар бик күп.
Мөмкин түгел, мөмкин түгел
Тик синдәен табуы.
```
