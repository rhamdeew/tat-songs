# Оригинал

### Риза Ишморат / Сара Садыйкова - Әй, ямьле син, Минзәләм ягы

```
Кырларында гөрләп иген уңа,
Урманында балан, миләшләр,
Гөл-чәчәкләр күмгән болыннарын,
Бакчаларың тулы җимешләр.

Әй ямьле син, Минзәләм ягы,
Синдә минем яшьлегем таңы.

Күңелгә дәрт бирә иртәләрең,
Моңландыра айлы кичләрең.
Җанга якын синең бөтен җирең,
Синдә минем гүзәл хисләрем.

Әй ямьле син, Минзәләм ягы,
Синдә минем яшьлегем таңы.

Кешеләрең синең киң күңелле,
Ут-су кичкән көчле батырлар.
Хезмәтләре белән танылганнар,
Җырга оста, сүзгә тапкырлар.

Әй ямьле син, Минзәләм ягы,
Синдә минем яшьлегем таңы.
```
