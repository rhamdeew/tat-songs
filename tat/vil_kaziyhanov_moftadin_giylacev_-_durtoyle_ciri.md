# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Дүртөйле җыры

```
Агыйдел буенда үстек,
Яшибез Дүртөйледә,
Бездә халык эшкә уңган,
Тыйнак та, сөйкемле дә.
Әллә ли, һиллә лә,
Тыйнак та, сөйкемле дә.

Күңелләр көр, кәефләр шәп,
Эшләр бара гел көйле.
Ярышларда кем җиңүче?
Безнең район Дүртөйле.
Әллә ли, һиллә лә,
Безнең район Дүртөйле.

Су юллары, асфальт юллар
Дүртөйлегә илтәләр,
Авыл диеп киләләр дә
Шәһәр диеп китәләр.
Әллә ли, һиллә лә,
Шәһәр диеп китәләр.
```
