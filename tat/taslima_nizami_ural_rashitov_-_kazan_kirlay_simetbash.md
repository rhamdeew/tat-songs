# Оригинал

### Тәслимә Низами / Урал Рәшитов - Казан — Кырлай — Симетбаш

```
Ник бу кадәр сагындыра икән
Газиз авылым Симетбаш?
Туган нигез моңаймасын,
Кайтыйк әле, кайтыйк, авылдаш.

Казан — Кырлай — Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.

Олыгая барган саен тарта,
Тарта икән туган як.
Симетбашым, әл дә син бар,
Минем бик бәхетле, иң бәхетле чак.

Казан — Кырлай — Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.

Нурислам чишмәсе суын эчә
Өзелеп сайрар сандугач.
Жырларымда синең моңың
Мәңге яшәр, мәңге, Симетбаш.

Казан — Кырлай — Симетбаш арасы
Якын җырлап узганга.
Таба алмыйм яшьлегемне,
Күмелгән ул, күмелгән томанга.
```
