# Оригинал

### Татар халык - Ага сулар (дүртенче вариант)

```
Ага сулар, ага сулар
Ага сулар үзәнгә.
Бер киткәч, тиз кайтып булмый
Туып-үскән илләргә.

Ага сулар, ага сулар,
Ага сулар гел шулай.
Аккан сулар кире кайтмый,
Яшь гомерләр дә шулай.

Ага сулар, ага сулар,
Ага сулар, туктамый.
Чәчәк кебек гүзәл яшьлек
Мәңгегә онытылмый.
```
