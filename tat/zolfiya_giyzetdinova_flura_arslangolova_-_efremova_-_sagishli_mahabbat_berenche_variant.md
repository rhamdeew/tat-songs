# Оригинал

### Зөлфия Гыйзетдинова / Флүрә Арсланголова - Ефремова - Сагышлы мәхәббәт (беренче вариант)

```
Кошлар китә җылы якка,
Мин кайда китимен соң?
Сагнуыма чамалар юк,
Әйтегез, нишлимен соң?

Ялгыз калган аккошларның
Хәлләрен белдем инде.
Күрә торып, үтка кердем,
Ник сине сөйдем инде?

Табышканнар без идек лә,
Кавышканнар без түгел.
Сайрамый торыгыз кошлар,
Юатучы сез түгел.

Җилләр булып исәр идем
Өшетер мендәр төсле.
Кайлардан гына алыймын
Синсез яшәтер көчне.
```
