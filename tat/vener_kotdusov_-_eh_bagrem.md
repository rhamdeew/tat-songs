# Оригинал

### Венер Котдусов - Эх, бәгърем

```
Агыйделкәй ага, ага,
Дулкыны ярга кага.
Дулкын-дулкын уйларым да,
Дулкын-дулкын хисләрем дә,
Эх, бәгърем, сиңа барам шул,
Эх, бәгърем, сиңа барам.
Дулкын-дулкын уйларым да,
Дулкын-дулкын хисләрем дә,
Эх, бәгърем, сиңа барам…

Син гашыйк яшел җәйләргә,
Миңа якын язгы буран.
Язгы буран, карлы яңгыр,
Үзе салкын, үзе ялкын,
Түз, бәгърем, булсаң буран шул,
Түз, бәгърем, булсаң буран.
Язгы буран, карлы яңгыр,
Үзе салкын, үзе ялкын,
Түз, бәгърем, булсаң буран…

Агымсу диңгезгә таша
Чакрымнар, еллар аша.
Хисләрем җыр булып таша,
Саг(ы)нулар, сагышлар аша
Эх, бәгърем, сиңа таба шул,
Эх, бәгърем, сиңа таба.
Хисләрем җыр булып таша,
Саг(ы)нулар, сагышлар аша
Эх, бәгърем, сиңа таба…

Дулкын-дулкын уйларым да,
Дулкын-дулкын хисләрем дә,
Эх, бәгърем, сиңа бара шул,
Эх, бәгърем, сиңа бара.
Хисләрем җыр булып таша,
Саг(ы)нулар, сагышлар аша
Эх, бәгърем, сиңа таба шул,
Эх, бәгърем, сиңа таба…
```
