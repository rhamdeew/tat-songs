# Оригинал

### Роберт Миңнуллин / Рөстәм Яхин - Хатыннар күңеле

```
Авыр чакта таяныгыз
Сез хатыннар күңеленә.
Ышанырсыз — хатыннарның
Күңеле таш түгеленә.

Хатыннарның күңелләрен
Тырышыгыз аңларга.
Болай да бит җиңел түгел,
Җиңел түгел аларга.

Кирәк чакта алар өчен
Керсен ирләр ялкыннарга —
Гомер буе сызланырга
Димәгән бит хатыннарга.

Хатыннарның күңелләрен
Тырышыгыз аңларга.
Болай да бит җиңел түгел,
Җиңел түгел аларга.

Без җыр җырлыйк сеңелләргә,
Апаларга, аналарга —
Җылы безнең тормышыбыз
Янәшәдә алар барга.

Хатыннарның күңелләрен
Тырышыгыз аңларга.
Болай да бит җиңел түгел,
Җиңел түгел аларга.
```
