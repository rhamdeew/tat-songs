# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Кил әле яныма

```
«Урамда көз», — дисең
Әллә син сизмисең:
Күңелдә язлар әле.
Сөюле җанымда
Син булган чагымда
Язлардан язмам әле.

Күгем аяз әле
Күңелдә яз әле…
Синең язларың бит ул.
Бирешмичә бер дә
Яшәтүче җирдә
Синең назларың бит ул

Кил әле яныма
Күм әле назыңа
Назга күмәләп яшийк
Бәхет өчен генә
Яшибез без җирдә
Бәхетле булып яшийк

«Урамда көз», — дисең
Әллә син сизмисең:
Күңелдә язлар әле.
Сөюле җанымда
Син булган чагымда
Язлардан язмам әле.
```
