# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Гүзәл ил ( Башкортостан )

```
Бар җирдә бер гүзәл ил —
Минем туган илем.
Ул — яшел урман — кырлыр,
Йомшак назлы җиле.
Кош булып канат җәеп
Оч икән, коч икән!
Әй җирем синдәй гүзәл
Бар микән илләр бүтән,
Бар микән.

Мин сине генә сөям,
Сиңа гына баш иям,
Әй, Башкортостан, туган илем
Газиз җирем минем
(Гүзәл җирем минем)

Иркәли, сөя мине
Агыйдел дулкыны
Дус миңа ак кышларың,
Көзләрең ялкыны.
Ак каен урманнарда
Адашып уйнадым.
Юк, синең гүзәллеккә
Сокланып туялмадым,
Туймадым.

Мин сине генә сөям,
Сиңа гына баш иям,
Әй, Башкортостан, туган илем
Газиз җирем минем
(Гүзәл җирем минем)

Чал Урал саклый минем
Илемнең үткәнен,
Ышанам: тик өметле
Киләчәк көткәнен
Үткер күзен еракка
Төбәгән Салават, —
Батыр ул — кызларына
Васыятен әйтә кебек кабат.

Мин сине генә сөям,
Сиңа гына баш иям,
Әй, Башкортостан, туган илем
Газиз җирем минем
(Гүзәл җирем минем)
```
