# Оригинал

### Марат Гобәйдуллин - Гөлчәчәк

```
Гөлчәчәкле болында
Танышкан идек.
Бер-беребездән вәгъдәләр
Алышкан идек.

Аллы-гөлле чәчкәләр —
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.

Гөлчәчәк исемен кушкан
Сиңа әткәең.
Ак бәхетләр теләгән
Газиз әнкәең.

Аллы-гөлле чәчкәләр —
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.

Гөлдәй ямьле йөзләрең,
Назлы сүзләрең.
Мәңгегә әсир итте
Серле күзләрең.

Аллы-гөлле чәчкәләр —
Сиңа бүләгем.
Гомерлеккә тиң ярың
Булу теләгем.
```
