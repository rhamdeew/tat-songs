# Оригинал

### Билгесез - Кайт, сөйгәнем, кире

```
Ташлап киттең мине,
Моңсулатып өйне,
Бер дә юкка, бәгырем, үпкәләп.
Син булмагач янда,
Күңел юксынганда
Кем юатыр мине иркәләп?

Өзгәләнә үзәк,
Телгәләнә йөрәк,
Бергә булган көнне хәтерләп.
Мин бит сине сөйдем,
Якын дустым, дидем,
Якын күрәм сине хәзер дә.

Җаным сызлый минем,
Күңел сыкрый, тиңем,
Яна йөрәк утсыз, күмерсез.
Кайт, сөеклем, бире,
Кайт, гүзәлем, кире,
Синнән башка тормыш күңелсез.
```
