# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Курай (беренче вариант)

```
Курай, курай, ышанычлы дус та,
Сердәш тә син татар халкына.
Мәңге яшәр халык мәхәббәте
Җырга-моңга, чабыш атына.

Курай, курай, ничә гасыр инде
Халкым сине кулдан төшерми.
Татар бит ул шундый, атына да
Сызгырмыйча су да эчерми.

Курай, курай, бар сагышын сөйли,
Бар моңнарын көйли татарның.
Оча күңел хыялына таба
Иярендә тулпар атларның.

Курай, курай, ничә гасыр инде
Юлдашы син татар җырының.
Синең кебек әнә сайрый тургай
Күкрәгендә иген кырының.
```
