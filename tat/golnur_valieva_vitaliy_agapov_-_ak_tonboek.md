# Оригинал

### Гөлнур Вәлиева / Виталий Агапов - Ак төнбоек

```
Су буйларында йөримен,
Камышлар шавын тыңлап.
Төшләремә ник кермисең, иркәм,
Сөю җырларын җырлап.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.

Су буйларында йөримен,
Сагышларым таратып.
Яннарымда булмасаң да,
Яшим сине яратып.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.

Су буйларында йөримен,
Синең эзләрең буйлап.
Үтәр айлар, үтәр еллар,
Туймамын сине уйлап.

Су буенда ак төнбоек
Йөзәдер салмак кына.
Күңелләрем нигә боек,
Белә тик таллар гына.
```
