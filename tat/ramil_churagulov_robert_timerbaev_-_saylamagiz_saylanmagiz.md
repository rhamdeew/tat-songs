# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Сайламагыз, сайланмагыз!

```
Күпләрнең башын әйләндердем,
Үтте-китте тиле яшьлегем. / Үтеп китте тиле яшьлегем. /
Әкият батырын көткән идем …
Кайда соң батыр ул, кем ул, кем?

Сайламагыз, сайланмагыз!
Яшь гомерләр бик тиз үтә (уза).
Көзгә калган чәчәкләрнең
Ал таҗын җилләр өзә!

Шушы кичләр бигрәк моңсу үтә,
Ялгыз үтә кара төннәрем.
Яшьлек хаталары өчен язмыш
Үч алыр дип бер дә белмәдем.

Сайламагыз, сайланмагыз!
Яшь гомерләр бик тиз үтә (уза).
Көзгә калган чәчәкләрнең
Ал таҗын җилләр өзә!

Зәңгәр күктә аккошлар да парлы,
Каеннар да парлы… Җан ярлы.
Кайда куйыйм икән, кайда илтим
Назларга сусаган бу җанны?

Сайламагыз, сайланмагыз!
Яшь гомерләр бик тиз үтә (уза).
Көзгә калган чәчәкләрнең
Ал таҗын җилләр өзә!
```
