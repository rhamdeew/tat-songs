# Оригинал

### Илдар Ногманов - Бер түзәр әле җаным

```
Китәсең нигә хушлашмый,
Араларны өзәсең?
Сизмисең мени: җанымны
Бер ялгызы итәсең.

Юлларыңда киртә булмам,
Бер түзәр әле җаным,
Йөрәгеңә, йөрәгеңә
Кагылмам инде тагын.

Түзәлмәслек яра булып
Каласың күңелемдә.
Онытырмын ләкин бар(ы)бер,
Керерсең төшләремә.

Юлларыңда киртә булмам,
Бер түзәр әле җаным,
Йөрәгеңә, йөрәгеңә
Кагылмам инде тагын.

Кулларымны кулларыңа
Алмадың нигә диеп,
Нигә мине бер шашынып
Сөймәдең янып-көеп.

Юлларыңда киртә булмам,
Бер түзәр әле җаным,
Йөрәгеңә, йөрәгеңә
Кагылмам инде тагын.
```
