# Оригинал

### Сәләх Шәрипов / Фәридә Кәлимуллина - Үкенечле сөю

```
Көтмәгәндә генә очрадың да
Күзләремә тутырып карадың,
Хисләремне белеп, җырлап, көйләп
Чәч толымнарымны тарадың.

Елмаюлы йөзең, серле күзең,
Кадерлем, дип сүзләр башладың,
Яшьлек хисләремне көйгә салгач,
Нигә инде шулай ташладың?

Хатлар яздым, җавап сәламең юк.
Сагынулардан, ахры, саргаям.
Көн кояшсыз, төннәр айсыз миңа,
Синең өчен сулам, картаям.

Сөю ләззәт, ярсыз калу газап,
Тарымагыз, дуслар, бу хәлгә,
Бик яратсаң — үкенечкә, диләр,
Шуның мисалымы мин әллә?
```
