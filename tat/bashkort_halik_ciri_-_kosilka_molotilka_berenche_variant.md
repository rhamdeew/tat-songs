# Оригинал

### Башкорт халык җыры - Косилка, молотилка (беренче вариант)

```
Уйнап алабыз әле /лә/,
Җырлап алабыз әле,
Уйнасак та, җырласак та,
Биеп алабыз әле.

Косилка, молотилка,
Сортировка, веялка.

Уйныйбыз бергә-бергә /дә/,
Җырлыйбыз бергә-бергә.
Уйнап-көлөп, җырлап-биеп,
Эшлибез бергә-бергә.

Косилка, молотилка,
Сортировка, веялка.

Улай итәргә кирәк /лә/,
Былай итәргә кирәк.
Җитәр инде бу уенны
Тәмам итәргә кирәк.

Косилка, молотилка,
Сортировка, веялка.
```
