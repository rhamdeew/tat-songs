# Оригинал

### Рәфкать Шаhиев / Марат Галимов - Мин

```
Язмышыңа язылганны
Ул аркасы белән төртеп ташлап булса,
Үткән юлым ялгыш диеп,
Кире кайтып хаталарсыз башлап булса…

Ул мин булмам, булмыйм, булмам,
Ул мин булмам, булмыйм, булмам…

Бер туктамый авыз ерып кына,
Җырлап кына гомер юлын узып булса,
Йөрәгеңә якын алмый,
Шулай гына борчулардан китеп булса…

Ул мин булмам, булмыйм, булмам,
Ул мин булмам, булмыйм, булмам…

Авыз тулы кара канны
Күрсәтмичә чит ятларга түзеп булса,
Җаным тулы сагышларга,
Ялгышларга батмаенча йөзеп булса…

Ул мин булам, ул мин булам,
Ул мин булам, ул мин булам,
Мин булам, мин булам…
```
