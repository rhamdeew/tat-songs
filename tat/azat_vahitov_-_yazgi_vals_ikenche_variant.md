# Оригинал

### Азат Вахитов - Язгы вальс (икенче вариант)

```
Каңгылдашып кайта казлар туган ягына…
Яшьлегенә кайта дөнья, язга аклана!

Онытылмаслык нинди көн бу?
Нинди матур чак.
Мәңгелеккә хәтерләргә җуелып калачак.

Гөрләвекләр җырлый-җырлый, юлдан ашыга,
Кубәләкләр вальс әйләнә кояш каршында.

Онытылмаслык нинди көн бу?
Нинди матур чак.
Мәңгелеккә хәтерләргә җуелып калачак.

Арын мәшәкатьләреңнән — тышта яз, дустым!
Илһамланып шул мизгелдә җырлар яз, дустым.

Онытылмаслык нинди көн бу?
Нинди матур чак.
Мәңгелеккә хәтерләргә җуелып калачак.
```
