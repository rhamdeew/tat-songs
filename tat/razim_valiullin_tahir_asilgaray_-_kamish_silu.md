# Оригинал

### Разим Вәлиуллин / Таһир Асылгәрәй - Камыш сылу

```
Һай, камыш сылукай
Җырлале, моң булсын.

Сагышлар ярасын
Басарлык җыр тусын.

Камышым, сылуым
Кубызлар кушылсын.

Яшь кызлар ушыннан
Язарлык җыр тусын.

Кураем, уйнале
Егетләр ут алсын.

Очкыннар чәчрәтеп
Биерлек көй калсын.

Камышым сылуым
Сыз да сыз төрлесен.

Әбиләр — апайлар,
Агайлар — бабайлар,
Малайлар — шалайлар
Барсыда биесен.
```
