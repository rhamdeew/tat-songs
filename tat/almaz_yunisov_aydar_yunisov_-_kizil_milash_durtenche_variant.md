# Оригинал

### Алмаз Юнысов / Айдар Юнысов - Кызыл миләш (дүртенче вариант)

```
Өзеп алдың да тоттырдың
Кулыма кызыл миләш.
Тик нигәдер китеп бардың,
Мин сине сөям дигәч.

Кызыл миләш, кызыл миләш
Тәлгәше учларымда.
Кызыл миләшләрдәй янам
Мәхаббәт утларында.

Кызыл миләш ачы була
Җитлегеп пешкән чакта.
Кызыл миләш күргән саен
Уйларым синең хакта.

Кызыл миләш, кызыл миләш
Тәлгәше учларымда.
Кызыл миләшләрдәй янам
Мәхаббәт утларында.

Учларымда кызыл миләш,
Тик син юк яннарымда.
Хисләреңне аңламадым,
Ут ялкыны жанымда.

Кызыл миләш, кызыл миләш
Тәлгәше учларымда.
Кызыл миләшләрдәй янам
Мәхаббәт утларында.
```
