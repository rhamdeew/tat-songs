# Оригинал

### Саҗидә Сөләйманова / Шамил Кашапов - Туй җыры (өченче вариант)

```
Ай-кояш йөзләре,
Кояшлы көзләре…
Табышкан, кавышкан
Пар былбыл үзләре.

Карлыгачкаем ла,
Сандугачкаем ла;
Сайратыр, ай, матур
Яр булгач янында.

Талчыбык билләре,
Бал кебек телләре;
Гөрләшкән, серләшкән
Пар былбыл үзләре.

Әйдер генәем,
Бәйдер генәем,
Юл булмый, туй булмый
Теләкләр теләми.

Бер генә дигәне —
Юктыр ла тиңнәре…
Уң булсын, гөл булсын
Бәхетле көннәре.

Аллары, гөлләре,
Гөлбакча илләре.
Уң булсын, гөл булсын,
Бер булсын серләре.
```
