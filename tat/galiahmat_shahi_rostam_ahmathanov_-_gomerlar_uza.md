# Оригинал

### Галиәхмәт Шаһи / Рөстәм Әхмәтханов - Гомерләр уза …

```
Агачтан өзелеп
Яфраклар коела.
Яшь чакта гомерләр
Үтмәс күк тоела.

Сизми дә каласың
Гомернең үткәнен.
Күңелдә сайраган
Былбылның киткәнен.

Хәтерләр чиртәләр
Күңелнең кылларын.
Җыр булып очалар
Канатлы елларым.

Вакытның үткәнен
Үзгәртеп булмый шул!
Гомернең узганын
Кайтарып булмый шул!
```
