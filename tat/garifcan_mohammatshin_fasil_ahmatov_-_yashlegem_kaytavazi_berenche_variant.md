# Оригинал

### Гарифҗан Мөхәммәтшин / Фасил Әхмәтов - Яшьлегем кайтавазы (беренче вариант)

```
Ерак язлар аша
Таныш казлар дәшә —
Үткәннәрдән алар кайтаваз.
Әмма синсез генә,
Әмма минсез генә
Болыннарга шаулап кайта яз.

Инде утлар сүнгән,
Моңлы сагыш иңгән
Синең белән минем күзләргә.
Кат-кат чәчәк атып,
Сулган гөлләр ятып,
Без калдырган серле эзләргә.

Бер сандугач кына
Өзелеп сайрый, тына,
Белгән кебек уртак хисләрне.
Пар аккошлар кебек
Очып кына килеп,
Кабатлыйк шул тиңсез кичләрне!
```
