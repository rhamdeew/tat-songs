# Оригинал

### Миләүшә Әфләтунова / Марат Яруллин - Минем серем

```
Килә минем кулларыңнан тотып,
Якты урам буйлап үтәсе.
Безне күргән чит күзләргә карап,
Ул минеке, диеп әйтәсе.

Сизмәсен дип ятлар, яшерәмен күзем,
Бары тик
Син язган хатлар саклый сөю сүзен…
Минем серем — ул назлы сүзләр,
Минем серем — ул янган күзләр,
Минем серем — ул синең күзләр,
Җылы иренеңнән калган эзләр.

Акыл аңлый, ләкин карый читкә,
Булмый тик йөрәккә боерып.
Өзгәләнә, күмелә сагышка
Яшереп, сөюне тыелып.

Сизмәсен дип ятлар, яшерәмен күзем,
Бары тик
Син язган хатлар саклый сөю сүзен…
Минем серем — ул назлы сүзләр,
Минем серем — ул янган күзләр,
Минем серем — ул синең күзләр,
Җылы иренеңнән калган эзләр.
```
