# Оригинал

### Муса Җәлил / Ренат Еникеев - Бәхет (икенче вариант)

```
Үткәндә кичергән —
Кайгылар, газаплар
Барысы да онтыла,
Барысы да җуела.

Төн үтеп, күңелле
Көн тугач, азактан
Берсе дә булмаган
Шикелле тоела.

Аһ, ләкин онтылмый
Гомергә, гомергә
Электә татыган
Шатлыклар, рәхәтләр,

Яндырып йөрәкне,
Саклана күңелдә
Кадерле минутлар,
Бәхетле сәгатьләр.
```
