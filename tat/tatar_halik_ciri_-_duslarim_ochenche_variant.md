# Оригинал

### Татар халык җыры - Дусларым (өченче вариант)

```
Әй дусларым, сезгә ниләр куйыйм,
Казаннардин кайткан маллар юк.
Якын дуслар җыелганнар,
Бер җырламый хәлем юк.
Дусларым!
Сез дусларым хакы өчен
Бер җырламый хәлем юк.
Дусларым!
Сез дусларым хакы өчен
Бер җырламый хәлем юк.

Кистем дә генә агач, китте кыйгач,
Эчләреннән очты сандугач.
Кычкырып җырлап җибәрәм
Сез дусларны сагынгач.
Дусларым!
Сәхраларга чыгып җырлыйм
Сез дусларны сагынгач.
Дусларым!
Сәхраларга чыгып җырлыйм
Сез дусларны сагынгач.
```
