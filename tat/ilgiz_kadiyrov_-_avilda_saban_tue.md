# Оригинал

### Илгиз Кадыйров - Авылда — Сабан туе

```
Сабан туе — яз билгесе,
Җир-анага зур хөрмәт.
Ата-баба истәлеге,
Милләтеңә мәхәббәт.

Бүген бәйрәм, зур тантана,
Гөрли халык, шатлана.
Барысының уртак уе:
"Авылда — Сабан туе,
Авылда — Сабан туе"

Сабан туе — яз бәйрәме,
Егетләргә көч сынау.
Карт-корыга күңел ачу,
Ә кызларга яр сайлау.

Бүген бәйрәм, зур тантана,
Гөрли халык, шатлана.
Барысының уртак уе:
"Авылда — Сабан туе,
Авылда — Сабан туе"

Сабан туе — ул бүләкләр,
Ул — тамаша көн буе.
Мавыктыргыч ат чабышы,
Шуңа ул Сабан туе!

Бүген бәйрәм, зур тантана,
Гөрли халык, шатлана.
Барысының уртак уе:
"Авылда — Сабан туе,
Авылда — Сабан туе"

Сабан туе — кунак сыйлау,
Көтеп алган күрешү.
Бәхетле назлы парларга —
Таң атканчы үбешү.

Бүген бәйрәм, зур тантана,
Гөрли халык, шатлана.
Барысының уртак уе:
"Авылда — Сабан туе,
Авылда — Сабан туе"
```
