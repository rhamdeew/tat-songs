# Оригинал

### Харис Ибраһимов - Чабаталар

```
Чабаталар — чабата
Сезне халык ярата.
Чабатамны кигәндә
Аякларым рәхәттә.

Чабаталар корылган
Матур үреп ясалган.
Табигатьнең исләре,
Очсыз, җайлы, уңайлы.

Җәен һава бизәрли
Көзен сулар җибәрми.
Кышын аяк җылыта
Бигрәк әйбәт чабата.

Кунакларга барганда
Аягымда чабата,
Басмагыз сез аякка!
Кидем яңа чабата.
```
