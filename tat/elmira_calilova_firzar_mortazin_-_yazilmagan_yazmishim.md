# Оригинал

### Эльмира Җәлилова / Фирзәр Мортазин - Язылмаган язмышым

```
Таң туса да, ай тулса да,
Син төшәсең исемә.
Кай арада әйләндең син
Иң кадерле кешемә.

Язылмаган язмышыммы,
Татлы сагышыммы син?!
Иң тәүге сөюемме бу,
Соңгы язмышыммы соң?!

Сине күрсәм, яшьлегемә
Кире кайткандай булам.
Бер елмайсаң, кояш кебек,
Яктырып китә дөньям.

Язылмаган язмышыммы,
Татлы сагышыммы син?!
Иң тәүге сөюемме бу,
Соңгы язмышыммы соң?!

Сине уйлыйм иртәләрен,
Син дип янам кичләрдә.
Судан алып утка салдың,
Миңа хәзер нишләргә?!

Язылмаган язмышыммы,
Татлы сагышыммы син?!
Иң тәүге сөюемме бу,
Соңгы язмышыммы соң?!
```
