# Оригинал

### Хания Фәрхи / Зиннур Сафиуллин - Зиннурым

```
Газапларың үтте инде хәзер,
Бары сөю калды күңелдә.
Елмаеп уяндым әле бүген,
Төшләремә килеп кердең дө.

Зиннурым,
Дип дәшәм, күз нурым,
Сагыну сәламем бу җырым.

Үзәкләрне өздерерлек итеп,
Син уйнадың, ә мин җырладым.
Ике моң бергә кушылган җирдә
Тик төш кенә икән, уяндым.

Зиннурым,
Дип дәшәм, күз нурым,
Сагыну сәламем бу җырым.

Бу ике моң гел, Сак-Сок кебек,
Йөрер инде мәңге адашып.
Исемеңне җырга салдым, тыңла,
Гомер буе яшим җан атып.

Зиннурым,
Дип дәшәм, күз нурым,
Сагыну сәламем бу җырым.
```
