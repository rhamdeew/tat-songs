# Оригинал

### Рөстәм Зәкуан / Альфред Якшимбетов - Туганнар — тумачалар

```
Якын күреп җыелган сезне дуслар.
Дусларсыз яшәп булмый бу дөньяда.
Тормыш матур була ул дуслар булса,
Якын туганнар гел яныңда булса.

Туган-тумачалар,
Эх, кода-кодачалар.
Уйнап-көлеп, җырлап-биеп,
Күңел ачалар.

Матур теләкләр сезгә һәммәбездән,
Кода-кодача, күрше-күләннән.
Баллансын әйдә әче шәрабләр
Кияү-киленне сөеп үткәндә.

Туган-тумачалар,
Эх, кода-кодачалар.
Уйнап-көлеп, җырлап-биеп,
Күңел ачалар.

Ятларны җанга кертми гел үрегез
Гомеркәйләрнең парлы сәйләнен.
Бәхеткә сөенеп үтсен һәр көнегез,
Кайгы-хәсрәтләр үтсен әйләнеп.

Туган-тумачалар,
Эх, кода-кодачалар.
Уйнап-көлеп, җырлап-биеп,
Күңел ачалар.
```
