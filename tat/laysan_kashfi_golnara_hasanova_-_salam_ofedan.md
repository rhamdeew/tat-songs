# Оригинал

### Ләйсән Кәшфи / Гөлнара Хәсәнова - Сәлам Өфедән

```
Без Агыйделдән җыр алып килдек,
Урал таулары сәлам юллады.
Ничә кат инде җир шарын урый
Татар җырының данлы юллары.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек —
Өфе яклары.

Кая барсак та, кайда йөрсәк тә,
Сез, туганнарым, каршы аласыз.
Йөрәк тоташа моңлы җыр аша,
Без туган җирнең газиз баласы.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек —
Өфе яклары.

Сагынган чакта, җирсегән чакта
Безнең моңнардан шифа алыгыз.
Без бит җир шарын җырга чорныйбыз,
Хәерле сәфәр теләп калыгыз.

Агыйдел яры
Моңга уралган,
Сәлам Өфедән,
Сәлам Уралдан.
Йөретә безне
Җыр канатлары.
Моңлы гөлбишек —
Өфе яклары.
```
