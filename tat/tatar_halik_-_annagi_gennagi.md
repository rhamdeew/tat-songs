# Оригинал

### Татар халык - Әннәги — геннәги

```
Әннәги-геннәги,
Аягымда сандали.
Сандали түгел пима,
Яшь күңелемне тыйма.
Әннәги-геннәги,
Аягымда сандали.
Сандали түгел пима,
Яшь күңелемне тыйма.

Аклы ситсы күлмәгемнең
Итәген сырлатканчы
Аклы ситсы күлмәгемнең
Итәген сырлатканчы
Үзем чыгып биим әле,
Кешене биеткәнче
Үзем чыгып җырлыйм әле,
Кешене җырлатканчы

Әннәги-геннәги,
Аягымда сандали.
Сандали түгел пима,
Яшь күңелемне тыйма.
Әннәги-геннәги,
Аягынмда сандали.
Сандали түгел пима,
Яшь күңелеңмне тыйма.

Безнең урам аркылы,
Ага суның салкыны.
Безнең урам аркылы,
Ага суның салкыны.
Су сипсәң дә басылмыйдыр,
Яшь йөрәкнең ялкыны.
Су сипсәң дә басылмыйдыр,
Яшь йөрәкнең ялкыны.

Әннәги-геннәги,

Яшь күңелемне тыйма.
Әннәги-геннәги,
Аягымда сандали.
Сандали түгел пима,
Яшь күңелемне тыйма.
```
