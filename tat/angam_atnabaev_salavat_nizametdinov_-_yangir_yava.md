# Оригинал

### Әнгам Атнабаев / Салават Низаметдинов - Яңгыр ява

```
Шыбыр — шыбыр яңгыр ява,
Шыбыр — шыбыр яңгыр ява,
Чылата керләреңне.

Керфекләрең ник чыланган,
Керфекләрең ник чыланган,
Сагындың кемнәреңне?
Сагындың кемнәреңне?

Елама, дип әйтсәмдә,
Елама, дип әйтсәмдә.
Барыбер еларсыңдыр.

Болытлар бит тулышканнар,
Болытлар бит тулышканнар —
Яңгырны тыярсыңмы?
Яңгырны тыярсыңмы?

Керфекләр дә, керләр кебек,
Керфекләр дә, керләр кебек,
Тиз генә кипсә икән.

Сагнулар да, яңгыр кебек,
Сагнулар да, яңгыр кебек,
Килсә дә китсә икән!
Килсә дә китсә икән!
```
