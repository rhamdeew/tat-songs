# Оригинал

### Илсөяр Ганиева / Мөфтәдин Гыйләҗев - Кайтырмын

```
Бер кайтырмын туган якларыма,
Искән җилләр булып кайтырмын.
Урал буйларыннан гүзәл җирне
Табалмадым диеп әйтермен.

Бер кайтырмын ямьле Агыйделгә,
Ярсу ташкын булып кайтырмын.
Еллар ташкыны да күмә алмас
Сагышларым диеп әйтермен.

Упкын яры киссә юлларымны
Бер моң булып урап кайтырмын.
Туган ягым онытылмас җырым,
Соңгы сәламем дип әйтермен.
```
