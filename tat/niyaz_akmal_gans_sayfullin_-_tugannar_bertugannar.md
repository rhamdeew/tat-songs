# Оригинал

### Нияз Акмал / Ганс Сәйфуллин - Туганнар, бертуганнар

```
Сезне уйлап юанам мин, юанам.
Сезне уйлап куанам мин, куанам.
Төшләремдә сезне күрсәм
Шатлык белән уянам.

Бер анадан, бер атадан
Безгә гомер бирелгән.
Җидегән йолдызлар шикелле
Нәсел җебе үрелгән.
Җидегән йолдызлар шикелле
Нәсел җебе үрелгән.

Бер ояда тулып үстек, тулып үстек.
Бер ана балалары, балалары.
Кадерен белик, туганнар,
Сөекле аналарны.

Җидегән йолдызлар шикелле
Без дә бергә үрелгән.
Туганым, дип эндәшәм мин,
Йөрәгемнең түреннән.
Туганым, дип эндәшәм мин,
Йөрәгемнең түреннән.

Туганнарымны уйласам, уйласам,
Күңелдә шатлык арта, шатлык арта.
Туган йорт үзенә дәшә,
Туган йорт якынайта.

Туганыңны туган итү
Иң изге йола бит ул.
Туганым дип бер әйтсәң дә
Күңелләр була бит ул.

Туганнарым, туганнарым,
Туганым, бертуганым,
Әле ярый җирдә сезгә
Туган булып туганмын.
Әле ярый җирдә сезгә
Туган булып туганмын…
Туган булып туганмын…
```
