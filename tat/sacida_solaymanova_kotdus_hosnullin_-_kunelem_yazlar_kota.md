# Оригинал

### Саҗидә Сөләйманова / Котдус Хөснуллин - Күңелем язлар көтә

```
Кайсы язның ярсу елгалары
Ярсыттылар минем йөрәкне,
Кайсы таңнын сайрар сандугачы
Сагыну җырын миңа өйрәтте?

Кайсы таңда искән йомшак җилләр
Алып килер синнән бер хәбәр?
Йөрәгемә язлар ярсу салды,
Җилкетәләр җәйге иртәләр.

Җәйләр җитсә, күңел кышны сагына,
Кышлар җитсә, күңел яз көтә.
Яз айларын көткән кошлар сыман,
Талпынадыр күңелем кич-иртә.
```
