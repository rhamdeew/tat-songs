# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Өн белән төш арасында

```
Әллә хыял, әллә чынлык булды,
Әллә өн бу, әллә төш булды.
Өннәр белән төшләр арасында
Юлларыбыз бергә кушылды.

Өннәр белән төшләр арасында
Адашканнар хисләр-өметләр.
Мәхәббәтнең кайнар сукмагында
Аз дисәң дә безнең кебекләр.

Ялгышларны үзем аңласам да,
Тыя алмам кебек хисемне.
Адашу да татлы, синең белән
Ялгышу да бәхет шикелле.

Төннәр белән төшләр арасында
Без аз гына бүтән үлчәмдә.
Ялгыш булыр, сине сөймим дисәм,
Ялгышырмын, сөям дисәм дә.
```
