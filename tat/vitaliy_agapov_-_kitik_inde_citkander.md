# Оригинал

### Виталий Агапов - Китик инде, җиткәндер

```
Кунак булып килдек сезгә,
Бүләк итеп җырларны.
Матур икән сезнең якның
Табигате, кызлары.

Китик инде, җиткәндер,
Түземегез беткәндер.
Җырлар тыңлап тамак туймый,
Ашыйсыгыз килгәндер.

Бүләгебез зурдан түгел,
Кабул итеп алыгыз.
Тамашаны ошатсагыз,
Бәлки кул да чабарсыз.

Китик инде, җиткәндер,
Түземегез беткәндер.
Җырлар тыңлап тамак туймый,
Ашыйсыгыз килгәндер.
```
