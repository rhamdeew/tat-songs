# Оригинал

### Сәкинә Хәйруллина / Фирдүс Тямаев - Сәлам (икенче вариант)

```
Шәһәргә китеп барышым,
Авыл тузанын кагып.
Хуш дигән сүзем югалды,
Инеш суыннан агып.

Артта — авыл, алда — шәһәр,
Шау-шулы серле галәм.
Авыл егетеннән сиңа
Ялкынлы кайнар сәлам.

Хыяллар күпкатлы йорттай,
Зәңгәр күкләргә ашкан.
Бәхет чокып чыгар кебек
Юлга төзелгән таштан.

Артта — авыл, алда — шәһәр,
Шау-шулы серле галәм.
Авыл егетеннән сиңа
Ялкынлы кайнар сәлам.

Авылга кайтып барышым,
Тарта урман-сулары.
Төшемә кереп йөдәттең,
И-и,иген басулары.

Артта — авыл, алда — шәһәр,
Шау-шулы серле галәм.
Авыл егетеннән сиңа
Ялкынлы кайнар сәлам.
Артта — шәһәр,алда — авыл,
Иң изге газиз калам.
Синдә тудым,синдә үстем,
Мәңгегә синдә калам.
```
