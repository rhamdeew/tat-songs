# Оригинал

### Татар халык җыры - Зәңгәр чиләк

```
Сандугачлар җыелганнар,
Без алай җыелмыйбыз.
Дуска түгел, дошманга да
Без кара коелмыйбыз.

Зәңгәр чиләк матур була
Су белән тулса гына
Ярны яр дип сөеп була,
Сүзендә торса гына

Сандугачым, син кечкенә,
Зур буласың кагынгач.
Төн караңгы, көн ямансу
Була икән сагынгач.

Кызыл яулык бәйлә, дустым,
Кызыл яулык килешә.
Кызыл яулык бәйләгән кыз
Теләгенә ирешә.

Сандугачым, каралгансың,
Суларга кереп коен.
Чәчәгеннән аерылган
Гөл кебек булдым быел.

Зәңгәр чиләк матур була
Су белән тулса гына
Ярны яр дип сөеп була,
Сүзендә торса гына.
```
