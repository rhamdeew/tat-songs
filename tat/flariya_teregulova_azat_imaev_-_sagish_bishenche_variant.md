# Оригинал

### Флария Терегулова / Азат Имаев - Сагыш (бишенче вариант)

```
Сагындырдың, иркәм, кил инде,
Юлларыма гөлләр сип инде.
Сөюләрем сиңа чамасыз,
Синнән башка тормыш ямансу.

Нурлы йөзеңне күрергә
Зар — интизарлар булдым.
Кайгы — сагышсыз үтәек
Алдагы гомер юлын.

Өлешемә төшкән көмешем,
Язмышыма кергән ялгышым.
Күзләремдә сөю сагышы,
Тизрәк кил, йөрәк багышым.

Нурлы йөзеңне күрергә
Зар — интизарлар булдым.
Кайгы — сагышсыз үтәек
Бергәләп гомер юлын.

Ялгышым булсаң булырсың,
Кирәкми миңа башка.
Сөюем булса янәшә,
Гомерләр үтмәс бушка.

Нурлы йөзеңне күрергә
Зар — интизарлар булдым.
Мәхәббәтә, тик бәхеттә
Уткәрик гомер юлын.
```
