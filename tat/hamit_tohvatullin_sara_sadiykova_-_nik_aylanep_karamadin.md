# Оригинал

### Хәмит Төхвәтуллин / Сара Садыйкова - Ник әйләнеп карамадың?

```
Ник әйләнеп карамадың,
Соңгы тапкыр күрешеп киткәндә,
Нигә, бәгърем, син дәшмәдең,
Соңгы тапкыр кулың үпкәндә?

Кайгыларым онытылыр инде,
Бер генә миңа карасаң.
Күңел күзе белән карар идең,
Әгәр дус дип мине санасаң.

Син китсәң дә, синең белән
Минем мескен арган йөрәгем.
Төшләремдә һаман сине күрәм,
Өнемдә дә күрү — теләгем.

Кайгыларым онытылыр инде,
Бер генә миңа карасаң.
Күңел күзе белән карар идең,
Әгәр дус дип мине санасаң.
```
