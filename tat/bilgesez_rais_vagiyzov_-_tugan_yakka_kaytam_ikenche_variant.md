# Оригинал

### Билгесез / Рәис Вәгыйзов - Туган якка кайтам (икенче вариант)

```
Яз булдымы, Инҗәремә кайтам,
Кошчык булып, нурга сөенеп.
Япь-яшь чагым каршы алыр кебек,
Шомырт чәчәгенә төренеп.

Җәй булдымы, авылыма кайтам,
Сәлам биреп өзер гөлләргә.
Оскын-ялкын алам Инҗәремнән,
Өмет яга килер көннәргә.

Көз булдымы, Инҗәремә кайтам,
Саргайганда урман-кырлары.
Сарыларга сагышырмын микән,
Тыңлаганда әнкәй җырларын?

Кыш булдымы, Инҗәремә ҡайтам,
Ак кар бөртегөндәй сагынып.
Ак кар бөртекләре көрткә чума,
Ә мин торам сине сагынып.
```
