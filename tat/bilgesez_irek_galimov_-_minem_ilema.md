# Оригинал

### Билгесез / Ирек Галимов - Минем илемә

```
Башкортстаным гүзәл җирем,
Синдә үтә минем гомерем,
Яшьлек елларым, язмыш юлларым.

Әткәй, әнкәй, туганнарым,
Миңа якын булганнарым,
Рәхмәтле сиңа, газиз илемә.

Чит илләр дә бәхет тапмам,
Сездән мәңге баш тартмам,
Туган җиркәем, әткәй — әнкәем.

Язмыш безне аермасын,
Ходай ятим калдырмасын,
Яшик гел бергә, бергә гомергә.

Башкортстаным гүзәл җирем,
Синдә үтә минем гомерем,
Яшьлек елларым, язмыш юлларым.

Алда әле бар күп эшләр,
Күп ачышлар, төзелешләр,
Барсы да сиңа, минем илемә.
```
