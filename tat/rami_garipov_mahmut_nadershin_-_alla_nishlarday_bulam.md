# Оригинал

### Рәми Гарипов / Мәхмүт Нәдершин - Әллә нишләрдәй булам

```
Әллә кайларга ашкынам,
Әллә нишләрдәй булам.
Упкын кебек күзләреңә
Очып төшердәй булам…

Әллә кайларга ашкынам,
Әллә нишләрдәй булам.
Чия кебек иренеңне
Өзә тешләрдәй булам…

Әллә кайларга ашкынам,
Әллә нишләрдәй булам.
Толымыңны муеныма
Үреп ишердәй булам…
```
