# Оригинал

### Гөлнара Рәшитова / Мөдәрис Газетдинов - Тормышымның яме

```
Кояш нуры булып,
Чишмәдәй ургылып,
Күңелемә минем
Килдең сөю алып.

Айдай якты йөзең,
Балдай татлы сүзең.
Эри бәгырькәем,
Бер елмайсаң үзең.

Бердән-берем дисәң,
Иң бәхетле сизәм.
Тормышымның яме
Синдә генә икән.

Күземә кара әле,
Әл дә син бар эле.
Синең белән миңа
Шул тикле күңелле.

Булгач күңелдә яз,
Бәхет күгем аяз.
Сөю бәйрәм итә,
Йөрәк тулы гел наз.

Синең белән генә
Гөрләшеп гомергә
Яшәрбез әле без,
Иркәм, бергә-бергә.
```
