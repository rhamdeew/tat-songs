# Оригинал

### Татар халык сүзләре / Татар халык көе - Дусларым — үз ишләрем

```
Идел суы салкын, килә дулкын,
Ак үрдәкләр каршы йөзәлмый.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.

Сез, дусларым, миңа бигрәк якын,
Бер җырламый күңелем түзалмый.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.

Мин җырламыйм, дуслар, җырлый белеп,
Җырлыйм сезне якын күргәнгә.
Һәй, дусларым-үз ишләрем,
Сез дуслардин башка нишләрмен.
```
