# Оригинал

### Илсур Хөснетдинов / Ганс Сәйфуллин - Кайт тизрәк

```
Кошлар таңда сайрарга
Куна нечкә талларга.
Кызыкмагыз, дусларым,

Артык чибәр ярларга.
Су буендагы талым
Сөйлә миңа син барын.
Сөйгән ярым кайтыр микән,
Әйтче син, талым, талым.

Таң алдыннан кошлар сайрый
Тал чыбыгына кунып.
Чибәр ярларга кызыкма

Сөйгән ярларың булып.
Талларымның бөресен
Яз көнендә күрерсең.
Бик сагындым, кайт тизрәк,
Читтә нишләп йөрисең?
```
