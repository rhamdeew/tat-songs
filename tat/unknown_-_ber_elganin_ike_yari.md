# Оригинал

### Гөлфия Шакирова / Фирзәр Мортазин - Бер елганың ике яры

```
Ике яры булса гына,
Була бит сулар елга.
Беләсеңме, назлы ярым,
Нәкъ менә без шул юлда.

Елганың без ике яры,
Сулардай ага тормыш.
Сындырдың да алып кердең
Агымга нигә ялгыш.

Елгадай ага шул тормыш,
Туктамый да һич ялгыш.
Син булсаң да мин бит ялгыз,
Күңелдә бетмәс сагыш.

Елганың без ике яры —
Сулардай ага тормыш.
Сындырдың да алып кердең
Агымга нигә ялгыш.

Елганың без ике яры,
Һич бергә бул(а) алмыйбыз.
Икәү бәхет кесәсенә
Моң булып тул(а) алмыйбыз.

Елганың без ике яры —
Сулардай ага тормыш.
Сындырдың да алып кердең
Агымга нигә ялгыш.
```
