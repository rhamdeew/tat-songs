### Мэхэббэт булганда

```
Мэхэббэт булганда,
Зарыгып котэсен
Кавышыр минутлар житуен.
Кавышкач, тоймыйсын,
Шул мизгел гомернен
Куз ачып йомганчы утуен.

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.

Мэхэббэт булганда,
Кошлардай сайрыйбыз,
Кунелен кыллары монлана.
Сойгэнен булмаса,
Сагынган минутта
Картайган йорэк тэ сызлана.

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.

Мэхэббэт мэнгелек
Кабынган учактай
Хэрвакыт союне остэсэ,
Нигэ сон яшэргэ,
Озелеп соймэскэ,
Нишлэрмен ялкынлап соймэсэм?

Мэхэббэт булганда,
Без гомер итэбез.
Котэргэ, алда дип, озын юл.
Сойгеннэр сойсеннэр,
Ометлер эйтсеннер,
Яраткан ярын белэн бергэ бул.
```
