# Оригинал

### Салават Юлаев / Ишмулла Дилмөхәмәтов - Уралым (беренче вариант)

```
Уралым, Уралым,
Күгәреп яткан Уралым.
Нурга чумган түбәсе,
Күккә ашкан Уралым.

Кызарып кояш калканда,
Сайрап каршылый кошларың.
Тонҗорап кояш батканда,
Сайрап озата кошларың.

Сине мактый җырларым,
Сине данлый җырларым.
Син бит минем төягем,
Сине сөя йөрәгем,
Уралым, Уралым!…

аааааа…
аааааа…
лә-лә, лә-лә
лә-лә, лә-лә
лә-лә, лә-лә-лә-ль, лә-лә, лә-лә
лә-лә, лә-лә, лә-лә-ль, лә-лә
лә, лә-лә, лә, лә-лә, лә-лә.
```
