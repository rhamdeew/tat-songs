# Оригинал

### Альбина Мортазина / Рамил Әсхәдулла - Язмышымның иң зур табышы

```
Очрашулар булмый юкка,
Күктә куйган язылып.
Йолдыз булып атылдың да,
Җанга тулдың наз булып.

Йөрәгемнең назлы, назлы сагышы син,
Күңелемнең моңнар агышы,
Сөюемнең сүнмәс, сүнмәс балкышы да
Язмышымның иң зур табышы.

Бәхет капкаларын ачып
Сине көткәнмен икән.
Көтүләрем юкка түгел,
Хәзер без бергә, иркәм.

Йөрәгемнең назлы, назлы сагышы син,
Күңелемнең моңнар агышы,
Сөюемнең сүнмәс, сүнмәс балкышы да
Язмышымның иң зур табышы.

Язмышыма рәхмәтлемен,
Ялгышмаган язганда.
Бергә гомергә булыйк дип,
Телим йолдыз яуганда.

Йөрәгемнең назлы, назлы сагышы син,
Күңелемнең моңнар агышы,
Сөюемнең сүнмәс, сүнмәс балкышы да
Язмышымның иң зур табышы.
```
