# Оригинал

### Наҗар Нәҗми / Вәсил Галимов - Туган ягыма җыр (беренче вариант)

```
Чыгам иркен яланнарга,
Чәчкәләр кочагына.
Бар юлларым килеп тоташа
Яшьлегем сукмагына.
Чәчкәләр кочагына.

Чыгам яшел болыннарга,
Агыйдел буйларына,
Йөрәк моңнарымны кушам
Туган җир моңнарына,
Агыйдел буйларына.

Чыгам алтын кырларыма —
Игеннәр дөньясына.
Рәхмәт сүзләремне әйтәм
Кырларым хуҗасына —
Игеннәр дөньясына.

Чыгам китәр юлларыма —
Юллар үзенә тарта.
Шул ук юллар туып үскән
Җиремә алып кайта.
Җир-су үзенә тарта.
```
