# Оригинал

### Рамил Чурагулов / Илгиз Закиров - Туган якка кайтам!

```
Чит-ят җирләр тәмам ялыктырды.
Төшкә керә газиз туган як.
Без гомергә монда килмешәкләр —
Шәһәр җире безгә чит һәм ят.

Туган ягым, кайттым гомерлеккә!
Тик син генә мине кабул ит.
Чит-ят җирләр тәмам ялыктырды,
Ташлап киттем бер көн, гафу ит!
Гомерлеккә кайттым, кабул ит!

Һич юк монда газиз иркен кырлар,
Каеннарың сирпеми яктылык,
Һич кирәкми асфальт урамнарың,
Таш йортларың бөркә салкынлык.

Туган ягым, кайттым гомерлеккә!
Тик син генә мине кабул ит.
Чит-ят җирләр тәмам ялыктырды,
Ташлап киттем бер көн, гафу ит!
Гомерлеккә кайттым, кабул ит!

Мин авылга гомерлеккә кайттым,
Чылтыр чишмәм монда, урманым.
Монда минем газиз туган ягым,
Эчәр суым монда, җырларым.

Туган ягым, кайттым гомерлеккә!
Тик син генә мине кабул ит.
Чит-ят җирләр тәмам ялыктырды,
Ташлап киттем бер көн, гафу ит!
Гомерлеккә кайттым, кабул ит!

Туган туфраклардан еракларда
Бәхет табалмассың гомергә.
Шәһәргә син бары кунакка кил,
Бер атнага түгел, бер көнгә.

Туган ягым, кайттым гомерлеккә!
Тик син генә мине кабул ит.
Чит-ят җирләр тәмам ялыктырды,
Ташлап киттем бер көн, гафу ит!
Гомерлеккә кайттым, кабул ит!
```
