# Оригинал

### Сәрия Вахитова / Лилия Вахитова - Кайтсан иде яшь чакларга

```
1. Гомер уртасына житкэч,
Еш кына уйланабыз.
Яшь чакларны искэ алып
Хислэнеп монланабыз.

Кушымта:
Кайтсан иде яшь чакларга,
Тик бэхетле коннэргэ.
Соеп соелгэн чакларга,
Сандугачлы тоннэргэ.

2. Кайда калды яшьлектэге
Синен хэм минем эзлэр.
Тирэ якка нурлар чэчеп,
Бэхет яктырткан йозлэр.

Кушымта:
Кайтсан иде яшь чакларга,
Тик бэхетле коннэргэ.
Соеп соелгэн чакларга,
Сандугачлы тоннэргэ.

3. Шинэ голлэр су сипмэсэн,
Корый чэчэк атмый ул.
Хэр мизгелнен уз вакыты,
Кояш тонлэ калкмый ул.

Кушымта:
Кайтсан иде яшь чакларга,
Тик бэхетле коннэргэ.
Соеп соелгэн чакларга,
Сандугачлы тоннэргэ.
```
