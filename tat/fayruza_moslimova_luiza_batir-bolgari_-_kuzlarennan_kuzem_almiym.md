# Оригинал

### Фәйрүзә Мөслимова / Луиза Батыр-Болгари - Күзләреңнән күзем алмыйм

```
Тамчы булып чәчләреңә
Уйнап тамасым. килә.
Күңелеңдә ләйсән булып
Урын аласым килә.

Күзләреңнән күзем алмый
Карап торасым килә.
Үзем яраткан кадәрле
Яраттырасым килә.

Коштай очып, кочагыңа
Бер атыласым килә.
Иркәләнеп, бер мизгелгә
Онытыласым килә.

Күзләреңнән күзем алмый
Карап торасым килә.
Үзем яраткан кадәрле
Яраттырасым килә.
```
