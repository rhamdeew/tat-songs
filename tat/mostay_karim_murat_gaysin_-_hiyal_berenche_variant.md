# Оригинал

### Мостай Кәрим / Мурат Гайсин - Хыял (беренче вариант)

```
Күтәрелдем биек түбәләргә,
Сәлам бирде миңа шан-шөһрәтләр.
Тауга карап, тау буласым килде.
Эх, хыялым минем — шул таулар.

Диңгез кичтем. Дулкыннар, сез нигә
Йөрәгемә бәреп индегез?
Индегез дә кабат тынмадыгыз.
Эх, хыялым хәзер — шул диңгез.

Гомерлеккә гашыйк булдым сиңа,
Һәр бер сулышыңны тоямын.
Сөйдең дә син, сөйдерә дә белдең
Тик һаман син минем хыялым.
```
