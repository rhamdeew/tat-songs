# Оригинал

### Ләйсән Кәшфи / Людмила Аланлы - Язны алып кил

```
Язларны син алып кил инде,
Сагындым, сагындым, сагындым.
Ничә ел язларсыз үтелде,
Әрнүле көзләргә табындым.

Язларым, көзләрең нурыдай
Офыклар артында адашкан.
Боз булган күңелем дәрьясы
Шашыныр синең бер караштан.

Язларга соңлама ялгыш та,
Боз астын ташкыннар тарсынган.
Син генә таратыр сагышта
Оҗмахлар җылысы бар сыман.
```
