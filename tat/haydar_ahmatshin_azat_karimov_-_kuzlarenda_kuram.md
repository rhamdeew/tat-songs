# Оригинал

### Хәйдәр Әхмәтшин / Азат Кәримов - Күзләреңдә күрәм

```
Күзләреңдә, иркәм,
Яшәү ямен күрәм.
Шатлык нуры синнән бөркелә.
Әле минем кебек,
Йөрәк серләреңнән,
Аңлый алмас иде беркем дә.

Карама син иркәм,
Карама да читкә,
Кара миңа гына, наз белән.
Икәү булсак бергә,
Бәхет килер безгә,
Гомерләр дә узар яз белән.

Күзләреңдә, иркәм,
Ямле язлар күрәм.
Яшәрәсең күзгә күренеп.
Килерсең дә кинәт,
Елмаерсың кебек
Кояш нурларына төренеп.

Карама син иркәм,
Карама да читкә,
Кара миңа гына, наз белән.
Икәү булсак бергә,
Бәхет килер безгә,
Гомерләр дә узар яз белән.
```
