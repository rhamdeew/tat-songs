# Оригинал

### Рәфикъ Юныс / Сара Садыйкова - Син кирәк миңа

```
Син кирәк миңа — кайгы телгәндә,
Син кирәк миңа — шатлык килгәндә.
Көләч кояшка карап көлгәндә,
Төн йолдызларга сагыш килгәндә.

Сагыныр өчен еракларда,
Сокланыр өчен бергә чакларда.
Барча йөзләрдә, синең төсмерне
Сагынып эзләргә — син генә кирәк миңа!

Син кирәк миңа, кирәк хатларда
Матур исмеңне кабатларга.
Хыял дөньясын тынмый гизәргә,
Йөз өмет өзеп мең кат түзәргә!

Сагыныр өчен еракларда,
Сокланыр өчен бергә чакларда.
Барча йөзләрдә, синең төсмерне
Сагынып эзләргә — син генә кирәк миңа!
```
