# Оригинал

### Фәнүзә Зөлкарнаева-Нәбиуллина / Рәзидә Әхмәтова - Үтте яшьлек юләрлеге

```
Үтте яшьлек юләрлеге,
Булмаганда йөгәнлеге.
Назларымнан торган язым,
Моңнарымны калдым язып.

Үтте яшьлек юләрлеге,
Бар дөньяның күңеллеге.
Сандугачлар җырын тыңлап,
Таңнар атты, йөрәк сызлап.

Үтте яшьлек юләрлеге,
Җитте алтын мизгелеге.
Каршы алды шулай көзем,
Ышанмадым, хәтта үзем.
Килеп җиткән икән көзем,
Ышанмадым хәтта үзем.
```
