# Оригинал

### Ленар Шәех / Рүзил Мөхетдинов - Айсылукай

```
Исемеңнең яктысыннан
Сулган гел дә терелә.
"Айсылу!"— дип язып куйдым
Күңелемнең түренә.
Айсылукай!
Ай,сылукай!
Күңелемнең түрендә
Йолдыз алыйм ,
Утта яныйм
Ай караган төнемдә.

Күңелеңнең яктысыннан
Җаным күккә үрелә.
"Айсылу!"— дип уеп куйдым
Йөрәгемнең түренә.
Айсылукай!
Ай,сылукай!
Йөрәгемнең түрендә
Җилкәй булыйм,
Бәхет юрыйм
Сиңа сөю үрендә.

Исемеңнең яктысыннан
Дөньям нурга күмелә.
"Айсылу!"— дип язган Ходай
Язмышым,гомеремә.
Айсылукай!
Ай,сылукай!
Язмышым,гомерем дә.
Син — үткәнем,
Син — көткәнем,
Син — киләчәк көнем дә.
```
