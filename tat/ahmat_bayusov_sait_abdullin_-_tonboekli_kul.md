# Оригинал

### Әхмәт Бәюсов / Сәит Абдуллин - Төнбоеклы күл

```
Төнбоеклы тирән
Күл ярлары чирәм.
Чирәмендә ятып ял итәм.
Күлнең суын буйлыйм,
Үзем сине уйлыйм,
Үзем сине сагынып көтәм.

Төнбоеклы тирән
Күл ярлары тирән.
Тал-тирәктә сайрый сандугач.
Сандугачның җырын
Тыңлаганда елыйм,
Иркәм, сине өзелеп сагынгач.

Төнбоеклы тирән
Зәңгәр күлгә киләм.
Юкка гына түгел бит, иркәм,
Күлнең суы салкын,
Йөрәк тулы ялкын —
Янар булсам суга сикерәм.
```
