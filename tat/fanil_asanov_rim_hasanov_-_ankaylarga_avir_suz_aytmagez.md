# Оригинал

### Фәнил Әсәнов / Рим Хәсәнов - Әнкәйләргә авыр сүз әйтмәгез

```
Әнкәйләргә авыр сүз әйтмәгез,
Караймасын матур йөзләре.
Йөрәкләрдә мәңге иркәләсен
Әнкәйләрнең сөю сүзләре.

Әнкәйләргә авыр сү әйтмәгез,
Хәтерләрдә берүк калмасын.
Назлы, нечкә ана күңелләре,
Кайгы-хәсрәт күреп янмасын.

Әнкәйләргә авыр сүз әйтмәгез,
Күзләреннән яшләр акмасын.
Әнкәй безгә тормыш бүләк итә,
Әнкәй ача бәхет капкасын.
```
