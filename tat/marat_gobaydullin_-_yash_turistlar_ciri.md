# Оригинал

### Марат Гобәйдуллин - Яшь туристлар җыры

```
Туган ил буйлап сәфәргә
Чыктык без бергә.
Бар табигать шундый ямьле,
Якын күңелгә.

Без — яшь туристлар,
Аерылмас дуслар.
Юлдаш безгә якты кояш
Һәм сайрар кошлар.

Бөтен тирә-яктан күпме
Дуслар җыелган.
Җыр-биюләр тынмый бер тә
Кичләр буена.

Без — яшь туристлар,
Аерылмас дуслар.
Юлдаш безгә якты кояш
Һәм сайрар кошлар.

Ярышларда зур җиңүләр
Өмет итәбез.
Тагын яңа очрашуны
Сагынып көтәрбез.

Без — яшь туристлар,
Аерылмас дуслар.
Юлдаш безгә якты кояш
Һәм сайрар кошлар.
```
