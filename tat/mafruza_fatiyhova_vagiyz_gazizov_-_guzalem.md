# Оригинал

### Мәфрүзә Фатыйхова / Вәгыйзь Газизов - Гүзәлем

```
Матурым син, гөлем син,
Минем татлы җимешем,
Нурлы кояш, якты аем,
Челтерәп аккан …

Әй, матурым, чибәрем,
Кара кашым, гүзәлем,
Көн дә төшемдә күрәм,
Өзәсең бит үзәгем.

Матурым син, гөлем син,
Син — минем гомерлек яр.
Мәхаббәтем, сөйгәнем,
Миңа бүләк насыйп пар.

Әй, матурым, чибәрем,
Кара кашым, гүзәлем,
Көн дә төшемдә күрәм,
Өзәсең бит үзәгем.

Матурым син, гөлем син,
Киләчәгем, өметем,
Шатлык, бәхетем, назым,
Минем йөрәк тибешем.

Әй, матурым, чибәрем,
Кара кашым, гүзәлем,
Көн дә төшемдә күрәм,
Өзәсең бит үзәгем.
Әй, матурым, чибәрем,
Кара кашым, гүзәлем,
Көн дә төшемдә күрәм,
Өзәсең бит үзәгем…
Көн дә төшемдә күрәм,
Өзәсең бит үзәгем…
Өзәсең бит үзәгем…
```
