# Оригинал

### Билгесез - Ай, Уралым

```
Туган авылым Каратау буенда,
Юрүзәнем ага дулкын кагып ярга.
Мендем Каратауга, карап тирә юнгә,
Өзеп алдым курай уйнарга.

Ай, Уралым минем, туган газиз җирем,
Ирекле төйәгем, һәм әнкәм телем,
Шатланам җыр моңга, гашыйкмын мин аңа,
Һәм синең куйныңда туганга.

Курай моңнарымны алып китә җилләр,
Ерак сәфәрләргә, чит-ят җиррләргә.
Ишетсәгез иде курай моңнарымны,
Урыны булсын күңелләрегездә.

Ай, Уралым минем, туган газиз җирем,
Ирекле төйәгем, һәм башкорт телем,
Шатланам җыр моңга, гашыйкмын мин аңа,
Һәм синең куйныңда туганга.
```
