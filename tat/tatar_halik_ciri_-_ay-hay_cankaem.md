# Оригинал

### Татар халык җыры - Ай-һай, җанкаем

```
Сызылып киткән кашларың ла,
Карлыгандай күзләрең.
Йөрәгемне җилкендерә
Сөям дигән сүзләрең.
Ай-һай, җанкаем,
Сөям дигән сүзләрең!

Бакчаларга кояш төшә,
Кызарып алма пешә.
Алма кебек сөйгән ярым
Көн дә исемә төшә.
Ай-һай, җанкаем,
Көн дә исемә төшә!

Сандугачтыр өзелеп сайрый
Су буйенда, тирәктә.
Күз карашың бигерәк матур,
Рәхәт бирә йөрәккә.
Ай-һай, җанкаем,
Рәхәт бирә йөрәккә!

Сибелә чәчем, сибелә чәчем,
Иртән искән җилләрдә.
Бергә рәхәт, тормыш корыйк
Туган үскән илләрдә!
Ай-һай, җанкаем,
Туган үскән илләрдә!
Бергә рәхәт, тормыш корыйк
Туган үскән илләрдә!
```
