# Оригинал

### Риф Мифтахов / Фаил Абдраков - Таң алдында гармун моңы

```
Таң алдында гармун моңы…
Хыялый итәр кебек.
Үтеп барган яшьлегемә
Кабаттан илтәр кебек.

Таң алдында гармун моңы…
Йөрәгем эрер кебек.
Миңа язмаган ярымны
Алыр да килер кебек.

Таң алдында гармун моңы…
Йолдызлар тыңлый кебек.
Ә ай, бахыр, мине аңлап,
Сыгылып елый кебек…
```
