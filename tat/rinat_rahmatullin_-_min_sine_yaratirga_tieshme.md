# Оригинал

### Ринат Рахматуллин - Мин сине яратырга тиешме?

```
Сагышлардан-әрнүләрдән килеп кердең син күңелемә,
Сүнеп барган күзләремне караттың син, җаным, үзеңә.
Мин бит сине шушы көнгә кадәр күргәннәрем булмады,
Нишләтим соң йөрәгемне, ул бит бары сине сайлады.

Мин бит сине хәзер өзелеп яратам,
Гомерлеккә минеке, диеп, үземне юатам.
Бәлки инде, мин сине яратырга тиеш түгелдер,
Кинәт кенә синең белән очрашулар безнең — ялгыштыр.

Яшь араларын әгәр безнең уйлап кына карасак,
Без бәхетле, җил-давылларга әгәр каршы тора алсак.
Ходай сине бүләк итеп, очраштырды ике йөрәкне,
Хәзер инде тоеп яшим синең миңа ничек кирәкне.

Мин бит сине хәзер өзелеп яратам,
Гомерлеккә минеке, диеп, үземне юатам.
Бәлки инде, мин сине яратырга тиеш түгелдер,
Кинәт кенә синең белән очрашулар безнең — ялгыштыр.

Көткән чакларыңда килмәүләрем өчен мине гафу ит,
Аңла инде: мин дә ике ут арасында яшим бит.
Нигә иртәрәк соң очрамадың икән син тормышымда,
Бергә булу бәхетләре бары инде Ходай кулында.

Мин бит сине хәзер өзелеп яратам,
Гомерлеккә минеке, диеп, үземне юатам.
Бәлки инде, мин сине яратырга тиеш түгелдер,
Кинәт кенә синең белән очрашулар безнең — ялгыштыр.
```
