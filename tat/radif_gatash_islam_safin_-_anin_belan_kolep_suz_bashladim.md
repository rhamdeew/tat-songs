# Оригинал

### Рәдиф Гаташ / Ислам Сафин - Аның белән көлеп сүз башладым

```
Аның белән көлеп сүз башладым,
Көлеп җавап бирде.
Уйнап кына: «Чыгасыңмы?» — дидем.
Уйнап: «Ярар!» — диде.

Көлеп кенә кичен кочакладым,
Көлеп үптем үзен.
Уйнап кына үпкәләгән булды,
Яшерде дә йөзен…

Көлеп кенә суга төштек икәү,
Утка кердек уйнап.
Аерылышкач кына белдем аны
Сөюемне чынлап!
```
