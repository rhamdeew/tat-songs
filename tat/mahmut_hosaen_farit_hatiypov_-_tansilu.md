# Оригинал

### Мәхмүт Хөсәен / Фәрит Хатыйпов - Таңсылу

```
Куктә — Зөһрәкәй,
Җирдә син чибәр.
Сокланып сиңа:
«Таңсылу!» — диләр.
Син чишмәдәй саф,
Син гөлдәй гүзәл!
Сине күрмичә
Нинди җан түзәр?!

Таңда уянгач
Сагынып сине
Тәрәзәм кага
Мәхәббәт җиле.
Йөрәгем хәлен
Аңла, Таңсылу…
Синсез үткән көн —
Сыкрап җан сылу…
Аңла, Таңсылу!
```
