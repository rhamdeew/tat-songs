# Оригинал

### Сафуан Әлибай / Әмир Әбелханов - Тыйма йөрәгеңне!

```
Үкенмә үткәнгә,
Үткәннәр калганнар.
Каушама, кушылса
Чынлыкка ялганнар.

Тыйма, йөрәгеңне тыйма,
Ашкынган чакта хисләрең!
Хисләрең ташып бер тынса,
Нишләрсең, нишләрсең!

Югалту, сагышка
Әйләнсә төннәрең,
Кадерен белергә
Өйрәтер көннәрнең.

Тыйма, йөрәгеңне тыйма,
Ашкынган чакта хисләрең!
Хисләрең ташып бер тынса,
Нишләрсең, нишләрсең!

Таңнарны каршыла
Шатлана — шатлана.
Өр-яңа гомерең
Таң саен башлана.

Тыйма, йөрәгеңне тыйма,
Ашкынган чакта хисләрең!
Хисләрең ташып бер тынса,
Нишләрсең, нишләрсең!
```
