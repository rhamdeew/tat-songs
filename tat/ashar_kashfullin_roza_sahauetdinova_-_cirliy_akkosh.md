# Оригинал

### Әсхәр Кәшфуллин / Роза Сәхәүетдинова - Җырлый аккош…

```
Җырлый аккош соңгы җырын,
Мәхәббәтен юксынып,
Шауламый урман, исми җил —
Галәмнәр калган тынып.

Җырлый аккош җиргә-күккә
Моңнарын сибә-сибә,
Ярда камышлар яшь түгә
Яфракның яше кибә.

Ярсу елга шарлавыклар
Аның җырын куәтләп,
Никтер кыймыйча торалар
Кушылырга күмәкләп.

Җырлый аккош моңлы җырын,
Җан өзгеч хәтирәләр,
Табигать бит — хыянәтсез, —
Хыянәтсез мәхәббәтне
Күкләргә күтәрәләр.
```
