# Оригинал

### Рәмзинә Тимерова / Роберт Тимербаев - Кайт, сөеклем

```
Бөтерелеп сары яфрак
Эзләреңә генә ява.
Сагынуларны басар өчен
Тап әле миңа дәва.

Синең белән булган көннәр
Нинди рәхәт, нинди ямьле.
Шул бәхетле минутларым
Булалмады, никтер мәңге.

Ташлап киттең, ялгыз иттең,
Үпкәләдең юкка гына.
Сөю тулы күзләремнән
Яшьләр тама сукмагыңа.

Бергә чаклар яңа гына,
Хисләр тора яңарырга;
Өзгәләмә үзәгемне,
Кайт, сөеклем, яннарыма.

Бөтерелеп сары яфрак
Эзләреңә генә ява.
Сагынуларны басар өчен
Тап әле миңа дәва.
```
