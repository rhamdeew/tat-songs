# Оригинал

### Роза Хәбибуллина / Рөстәм Невретдинов - Яктылыгым җырымнан

```
Ялгышыммы син, әллә язмышыммы?
Җан түрендә әллә сайрар кошыммы?
Исемеңне белмәсәм дә күптәнге танышыммы?
Кем булсаң да ялгышырга мин риза,
Җан түрендә сандугачым җыр сыза,
Язмышышкаем сөя белү бәхетен миңа суза.

Күңелем түрендә сине күргәч,
Өздереп сайрады сандугач.
Сандугач тавышы булып чыңлый,
Бар җиһан мин гашыйк булгач.

Язмышыммы син, әллә ялгышыммы,
Шатлыгыммы, иң татлы сагышыммы?
Бәхетемә ашкынып очып кына барышыммы?
Канатлы мин, син барын уйлавымнан.
Агыла җыр урманым кырларымнан,
Сагышлы да, бәхетле дә яктылыгым җырымнан.

Күңелем түрендә сине күргәч,
Өздереп сайрады сандугач.
Сандугач тавышы булып чыңлый,
Бар җиһан мин гашыйк булгач.
```
