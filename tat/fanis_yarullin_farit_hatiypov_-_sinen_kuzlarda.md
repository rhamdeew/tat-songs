# Оригинал

### Фәнис Яруллин / Фәрит Хатыйпов - Синең күзләрдә

```
Уткән язларның,
Уткән назларның,
Матурлыкларын
Чыкмыйм эзләргә.
Язлар шатлыгы,
Назлар сафлыгы,
Синең күзләрдә,
Синең күзләрдә.

Теләгән чакта,
Колачым җәеп,
Сөю диңгезен
Иңләп йөзәргә.
Һич уйлап тормый,
Кереп югалам,
Синең күзләргә,
Синең күзләргә.

Яңа җәйләрнең
Җыеп ямьнәрен,
Баеп хисләргә,
Баеп төсләргә —
Яшьлегем минем
Адашып йөри,
Синең күзләрдә,
Синең күзләрдә.
```
