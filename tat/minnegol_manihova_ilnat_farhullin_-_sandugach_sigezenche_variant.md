# Оригинал

### Миңнегөл Манихова / Илнат Фәрхуллин - Сандугач (сигезенче вариант)

```
Ян бакчамда — чия, балан,
Түр бакчамда — алмагач.
Алмагачка кунып сайрый
Өздереп лә сандугач.
Шул кадәрле моңлы итеп
Ничек лә сайрап була?
Бәгырьләргә үтеп кереп,
Җаннарны айкап чыга.
Эзлимен — күрә алмыймын,
Кайларда син, сандугач?
Илкәеңнән айрылдыңмы,
Сайрыйсыңмы сагынгач?
Сайрыйсыңмы сагынгач?..
Сайрыйсыңмы сагынгач?..
```
