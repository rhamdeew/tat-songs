# Оригинал

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Җәйге ямьле басуларда

```
Җәйге ямьле басуларда
Бик ямансу кичләрен.
Җәйге кичтә яратам мин
Кырда арыш исләрен.

Җәйге ямьле басуларда
Дулкынлана арышлар.
Йөрәктә уянып куя
Онытылган сагышлар.

Җәйге ямьле басуларда
Карыйм сезнең якларга.
Чык каршыма, күрешербез
Эштән кайткан чакларда.
```
