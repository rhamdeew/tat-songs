# Оригинал

### Динә Талхина / Роза Сәхәүетдинова - Сагынырсың әле, сагынырсың

```
Сагынырсың әле мине, сагынырсың
Ярсу язның ташкынында.
Ташканында синең хисләрең,
Мин булмамын яннарыңда.
Нишләрсең һин, нишләрсең?

Сагынырсың әле мине, сагынырсың
Ямьле җәйнең таңнарында.
Таңда кошлар өзәр үзәкне.
Мин булмамын яннарыңда,
Үткәннәрне булмас төзәтеп.

Сагынырсың әле мине, сагынырсың
Күркәм көзнең кырларында.
Көзнең яме җанны кыер.
Мин булмамын яннарыңда,
Бер йөрәккә сагыш ничек сыер?

Сагынырсың әле мине, сагынырсың
Салкын кышның буранында.
Моңсуланыр синең карашың.
Мәңгелеккә безнең сөюебез
Бураннарда йөрер адашып.
```
