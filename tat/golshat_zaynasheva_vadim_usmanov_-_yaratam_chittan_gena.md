# Оригинал

### Гөлшат Зәйнәшева / Вадим Усманов - Яратам читтән генә…

```
Очрашканда, саумы, диеп,
Елмаеп үтәм генә.

Үзе белми, мин бит аны
Яратам читтән генә.

Ул парлы икән, мин белмәдем,
Ишеттем кичә генә.

Үзе белми, мин бит аны
Яратам читтән генә.

Парлы булгач, яратмам дип,
Мин антлар итәм генә.

Үзе белми, мин бит аны
Яратам читтән генә.

Үзе белсә нишләрмен дип,
Кайгырам эчтән генә.

Үзе белми, мин бит аны
Яратам читтән генә!..
```
