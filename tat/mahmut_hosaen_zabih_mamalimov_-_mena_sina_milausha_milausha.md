# Оригинал

### Мәхмүт Хөсәен / Зәбих Мәмәлимов - Менә сиңа Миләүшә (Миләүшә)

```
Күрше йорттагы Миләүшә —
Мин яраткан сылу кыз.
Тик ул озын, мин тәбәнәк,
Ул чая, мин кыюсыз.
Сандугачлар көнләшә.
Җырлап куйса Миләүшә.

Күргәндә тик бер елмайса,
Баш түбәм тия айга!..
Чакырсам да бармый үзе
Минем белән беркая.
Бармаса да, сөйләше
Мәхәббәтем Миләүшә.

Миләүшәгә өйләнәм дип
Әни белән киңәшәм (сөйләшәм).
Ни тамаша: үзе безгә
Килеп керде Миләүшәм…
Туена безне дәшә…
Менә сиңа Миләүшә!..
```
