# Оригинал

### Рамазан Шагалиев / Рафаэль Бакиров - Бии белмәсәң, оялма

```
Башка кызлар биегәндә,
Син нигә биемисең?
Алар белән уйнап-көлеп,
Син нигә сөенмисең?

Кушымта:
Бии белмәсәң, оялма,
Кил, үзем өйрәтермен.
Ялгышкан җирләрең булса,
Әйтермен, төзәтермен.

Зал түгәрәк, әйләнербез,
Безгә дә урын җитәр.
Яшь вакытта биеп калыйк,
Яшьлек үтәр дә китәр.

Кушымта:
Кичә тәмамланып килә,
Син утырасың һаман.
Кызлар, әнә, бии-бии
Арып беттеләр тәмам.
```
