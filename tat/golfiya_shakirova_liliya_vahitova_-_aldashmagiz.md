# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Алдашмагыз!

```
Алдадым диеп шатланмагыз.
ул бит куаныч түгел
кемнең нинди сүз сөйләвен
Барыбер тоя күңел.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.

Дөреслекне юк итмәгез
гөнаһларга кермәгез.
Изгелек эшләр урынга
Ялган, хәйлә үрмәгез.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.

Ходай биргән бер гомергә
бик матур яшәү кирәк.
Яшәү бәхет шушы җирдә.
Диеп типсен безнең йөрәк.

Алдашмагыз, алдашмагыз
ялганнар сөйләмәгез.
Зинһар өчен кешеләргә
начарлык теләмәгез.
```
