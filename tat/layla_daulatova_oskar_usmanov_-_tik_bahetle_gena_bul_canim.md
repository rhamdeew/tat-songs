# Оригинал

### Ләйлә Дәүләтова / Оскар Усманов - Тик бәхетле генә бул, җаным

```
Сине күргән саен, шатланамын
Бәхетемә тигән өлештән.
Бер елмаеп карауларың синең
Кадерлерәк алтын-көмештән.

Син бәхетле булсаң — миңа житә.
Ниләр тели алыйм мин тагын?
Минем беләнме син, түгелме син,
Тик бәхетле генә бул, җаным!

Яратырга мине өйрәттең син,
Онытылып бөтен дөньядан.
Һәр яңа көн җиргә туган саен,
Гашыйк булам сиңа яңадан…

Син бәхетле булсаң — миңа житә.
Ниләр тели алыйм мин тагын?
Минем беләнме син, түгелме син,
Тик бәхетле генә бул, җаным!

Җаныкаем диеп эндәшсәм дә,
Син минеке түгел әлегә.
Синең белән таныштырган язмыш
Кавыштырса иде мәңгегә…

Син бәхетле булсаң — миңа житә.
Ниләр тели алыйм мин тагын?
Минем беләнме син, түгелме син,
Тик бәхетле генә бул, җаным!
```
