# Оригинал

### Радик Хәкимҗанов / Ф.Шәрәфуллина - Туган яклар

```
Туган яклар — изге яклар,
Күңелдә яши шулай.
Безнең якта зәңгәр һава,
Алтын кояш, көмеш ай.

Торган җир якын булса да,
Туган җир кадерлерәк.
Гомер буе сиңа кайтам,
Ашкынып тибә йөрәк.

Туган яклар — изге яклар,
Синдә атты таңнарым.
Сагындырган авылдашлар,
Болыннарым, тауларым.

Торган җир якын булса да,
Туган җир кадерлерәк.
Гомер буе сиңа кайтам,
Ашкынып тибә йөрәк.

Туган яклар — изге яклар,
Сиңа кайтулар шатлык.
Каеннарың кызларыңдай,
Һис тә күреп туймаслык.

Торган җир якын булса да,
Туган җир кадерлерәк.
Гомер буе сиңа кайтам,
Ашкынып тибә йөрәк.

Әрнүләрем басылсын, дип
Кайттым шул ярсып — ярсып.
Кабат китәм, язмыштыр ул,
Сүрелмәс ялкын алып.

Торган җир якын булса да,
Туган җир кадерлерәк.
Гомер буе сиңа кайтам,
Ашкынып тибә йөрәк.
```
