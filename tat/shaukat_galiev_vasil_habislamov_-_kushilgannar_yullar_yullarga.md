# Оригинал

### Шәүкәт Галиев / Вәсил Хәбисламов - Кушылганнар юллар юлларга

```
Кушылганнар юллар юлларга,
Кушылганнар куллар кулларга,
Йөрәкләрнең якын арасы,
Йөрәкләрнең якын арасы.

Офыклардан офыкларны үтеп,
Гел барасы иде, барасы!

Табыштылар безнең йөрәкләр,
Кавыштылар безнең теләкләр,
Шатлансын да дуслар, соклансын,
Шатлансын да дуслар, соклансын!

Мәхәббәтнең шушы кайнарлыгы
Гомер буйларына саклансын!

Аерылмасын күзләр күзләрдән,
Борылмасын йөзләр йөзләрдән,
Үтеләсе юллар үткәнче,
Үтеләсе юллар үткәнче!

Үтеләсе юлларыбыз үтеп,
Безгә дигән гомер беткәнче!
```
