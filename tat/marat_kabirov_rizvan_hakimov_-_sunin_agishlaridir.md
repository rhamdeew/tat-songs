# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Суның агышларыдыр

```
Суның агышларыдыр…
Җилнең кагышларыдыр…
Кайлардан килеп кайларга,
Китеп барышларыдыр.

Суның агышларыдыр…
Җилнең кагышларыдыр…
Аерылдык та кавышалмадык,
Кемнең каргышларыдыр.

Суның агышларыдыр…
Җилнең кагышларыдыр…
Кеше каргышлары түгел,
Яшьлек ялгышларыдыр.

Суның агышларыдыр…
Җилнең кагышларыдыр…
Йөрәкләрдә — кайтавазы
Сак-Сок тавышларының.
```
