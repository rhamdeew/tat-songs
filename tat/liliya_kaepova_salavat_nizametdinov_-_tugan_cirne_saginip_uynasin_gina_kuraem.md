# Оригинал

### Лилия Кәепова / Салават Низаметдинов - Туган җирне сагынып (Уйнасын гына кураем)

```
Урманым бар, булганым бар,
Хисләрем кочагында.
Салкын — ялкын эчкәнем бар
Чишмәләр учагыннан.

Уйнасын гына кураем,
Тынмасын туган телем.
Уйнасын курайкаем,
Тынмасын туган телем.

Хыялымда җирне ташлап
Йолдызга очканымда.
Сагышымнан йолдыз эреп
Яуды җиргә яңгыр-кар.

Уйнасын гына кураем,
Тынмасын туган телем.
Уйнасын курайкаем,
Тынмасын туган телем.

Җир чакыра, моң чакыра,
Димәк мин әле тере.
Күктә канатларым тала
Утрауым башкорт җире.

Уйнасын гына кураем,
Тынмасын туган телем.
Уйнасын курайкаем,
Тынмасын туган телем.

Читтә-читтә йөргән чакта
Үз моңым ишетелә.
Димәк мин әле тирбәләм
Үз халкым бишегендә.

Уйнасын гына кураем,
Тынмасын туган телем.
Уйнасын курайкаем,
Тынмасын туган телем.
```
