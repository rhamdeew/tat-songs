# Оригинал

### Фәния Габидуллина / Илдар Шәрипов - Яратып яшик (икенче вариант)

```
Аерма, язмыш, аерма
Шау чәчәкле юлларны.
Сүндермә ялгыш, сүндермә
Ялкынлы йөрәкләрне.

Яратып яшик дөньяны,
Дусларны, туганнарны,
Тормыштан ләззәт табып,
Дөрләп-ялкынлап янып.

Елмаеп карыйк дөньяга,
Ул да безгә елмаер.
Авырсынган күңелләр дә
Җиңеллек, тыңгы табыр.

Яратып яшик дөньяны,
Дусларны, туганнарны,
Тормыштан ләззәт табып,
Дөрләп-ялкынлап янып.

Яшәү матур, яшәү күркәм,
Кадерең белсәң генә.
Килгән һәрбер мизгелгә дә
Сөенә белсәң генә.

Яратып яшик дөньяны,
Дусларны, туганнарны,
Тормыштан ләззәт табып,
Дөрләп-ялкынлап янып.
```
