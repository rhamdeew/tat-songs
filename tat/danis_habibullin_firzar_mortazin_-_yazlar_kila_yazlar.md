# Оригинал

### Данис Хәбибуллин / Фирзәр Мортазин - Язлар килә, язлар

```
Язлар килә, язлар,
Кайта киек казлар.
Су буйлары ямьгә күмелә.

Э Син,иркәм,нигә
Кайтмыйсың бирегә,
Язлар китермисең күңелгә?

Майлар җитә,майлар,
Көтеп алган айлар,
Ак чәчәккә чума бакчалар.

Бик шат булыр идем
Хисләрем дә минем
Шушы язда чәчәк атсалар.

Еллар уза, еллар
Гуя аккан сулар,
Аккан сулар, алар тин безгэ

Айлап аккан сулар,
утеп озын юллар
Кушылырлар эле дингезгэ
Дингездэге судай
утеп озын юллар
Кушылырбыз эле без бергэ
```
