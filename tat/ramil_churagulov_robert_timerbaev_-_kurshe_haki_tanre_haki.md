# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Күрше хакы — Тәңре хакы

```
Утым бетсә, учагымда,
Күршем, дип мин аңа керәм.
Ут күршемдә җан җылысы
Һич саекмасын беләм.

Күрше белән гел дус булып
Яшик әле гомер бакый.
Онытмыйча изге сүзне:
Күрше хакы — Тәңре хакы.

Башкаема кайгы төшсә —
Иң беренче күршем белә.
Хәсрәтемне үз итүче,
Юатучым күршем була.

Күрше белән гел дус булып
Яшик әле гомер бакый.
Онытмыйча изге сүзне:
Күрше хакы — Тәңре хакы.

Йорт-җиренә шатлык килсә —
Ашка чакырып күршем керә.
Куанычлар бәлешеннән
Миңа да ул өлеш телә.

Күрше белән гел дус булып
Яшик әле гомер бакый.
Онытмыйча изге сүзне:
Күрше хакы — Тәңре хакы.
```
