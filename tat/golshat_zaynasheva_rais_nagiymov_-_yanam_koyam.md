# Оригинал

### Гөлшат Зәйнәшева / Рәис Нәгыймов - Янам, көям

```
Янам, көям, янам, көям,
Ут капты күлмәгемә.
Ут капмады күлмәгемә,
Капты яшь йөрәгемә.

Янмамын дигән идем,
Көймәмен дигән идем.
Сөю — газап, беркемне дә
Сөймәмен дигән идем.

Уф, дип, йолдызларга карыйм,
Карыйм зәңгәр төннәрдә.
Гашыйк булдым буй җитмәслек
Йолдызга тиң берәүгә.

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю — газап, беркемне дә
Сөймәмен дигән идем.

Килә икән ул мәхәббәт
Юрап тормыйча гына.
Тиңме сиңа, тиң түгелме,
Сорап тормыйча гына.

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю — газап, беркемне дә
Сөймәмен дигән идем.

Яшерен генә сөеп йөрсәм,
Мәңге түзәрмен кебек.
Үзе белсә, сөймим дисә,
Егылып үләрмен кебек…

Янмамын дигән идем,
Көймәмен дигән идем;
Сөю — газап, беркемне дә
Сөймәмен дигән идем.
```
