# Оригинал

### Халисә Хәкимуллина / Мәсгудә Шәмсетдинова - Язгы тамчы

```
Тып, тып, тып,
Ты-пы-тып, тып, тып, тып,
Тамчы тама талларга,
Талларымның бөреләре
Куаныша шуңарга,
Куаныша шуңарга.

Тып, тып, тып,
Ты-пы-тып, тып, тып, тып,
Яз тама тамчы булып.
Кояш нурлары биешә
Шул тамчыга куанып,
Шул тамчыга куанып.
```
