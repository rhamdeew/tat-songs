# Оригинал

### Фәрит Суфияров / Мөдәрис Газетдинов - Күңелем сине көтә

```
Яшь гомер узар да китәр,
Сизелми кала еллар.
Очраштыра, курештерә,
Чакырып тора юллар

Сиңа әйтер сүзләрем бар,
Ямьле язларым утә!
Бер курешеп сөйләшергә
Куңелем сине көтә!

Очып кил син, акчарлагым,
Көттермә, соңга калма.
Юксынам да, борчылам да
Сагыш утына салма.

Серем дә син, моңым да син,
Теләкләр чикләнмәсен!
Тормыш кочагында назла
Сөюдән шикләнмә син!
```
