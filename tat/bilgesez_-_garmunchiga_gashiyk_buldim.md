# Оригинал

### Билгесез - Гармунчыга гашыйк булдым

```
Гармунчыга гашыйк булдым
Нишләргә дә белмимен
Дөньяларымны онытып
Янып көеп йөримен.

Әй, гармунчы, гармунчы
Җырчы кызга яр булчы
Яшәр идек, без парлашып
Былбыллардай сайрашып.

Чит егетләр күзен алмый
Тик син генә күрмисең.
Әллә кыелугың җитми,
Әллә сөя белмисең?

Әй, гармунчы, гармунчы
Җырчы кызга яр булчы
Яшәр идек, без парлашып
Былбыллардай сайрашып.

Һаман гармун уйныйсың
Ничек бер дә туймысың
Гармуның сагышын сөйли
Үзең ниләр уйлыйсың?

Әй, гармунчы, гармунчы
Җырчы кызга яр булчы
Яшәр идек, без парлашып
Былбыллардай сайрашып.

Гармунчының мәхәббәтен
Көтәдер көюләрен
Бәлки ачылыр телләрең
Үтә бит гөмерләрең.
```
