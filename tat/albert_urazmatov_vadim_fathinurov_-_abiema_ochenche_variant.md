# Оригинал

### Альберт Уразмәтов / Вадим Фәтхинуров - Абыема (өченче вариант)

```
Тормышымда чиксез бәхетле мин
Бертуганым булса янымда.
Олы ярдәм кулын сузып торган
Мең-мең рәхмәт әйтәм абыйга.

Синең кылган миңа мәрхәмәтең
Мин онытмам гомер-гомергә.
Бертуганнар булып, горурланып,
Яшик әле, абый, гел бергә.

Йомшак күң(е)лең, олы мәрхәмәтең,
Изгелеккә тулган йөрәгең.
Син булганда, абый җаныкаем,
Тынычланып тибә йөрәгем.

Синең кылган миңа мәрхәмәтең
Мин онытмам гомер-гомергә.
Бертуганнар булып, горурланып,
Яшик әле, абый, гел бергә.

Абый җаным, сиңа гына телим
Челтерәп аккан чишмә сафлыгын.
Исән-имин булып, шатлык белән
Тормышымның телим пакълыгын.

Синең кылган миңа мәрхәмәтең
Мин онытмам гомер-гомергә.
Бертуганнар булып, горурланып,
Яшик әле, абый, гел бергә.
Синең кылган миңа мәрхәмәтең
Мин онытмам гомер-гомергә.
Бертуганнар булып, горурланып,
Яшик әле, абый, гел бергә…
Бертуганнар булып, горурланып,
Яшик әле, абый, гел бергә…
Яшик әле, абый, гел бергә…
```
