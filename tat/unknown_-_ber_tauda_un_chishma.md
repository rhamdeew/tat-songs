# Оригинал

### Сибгат Хәким / Әгъзам Фәттах - Бер тауда ун чишмә

```
Бер тауда ун чишмә,
Унысы ун төстә;
Югыйсә бер үк тау,
Югыйсә бер үк җир,
Юлы бер…

Бер тауда ун чишмә,
Тынмыйлар бер кич тә,
Чәчәкле җәй әле,
Атлы һәм җәяүле
Юлыгыр…

Бер тауда ун чишмә,
Унысы ун төстә;
Берсе тиз болгана,
Берсе тиз моңлана,
Ком-ташта…

Бер тауда ун чишмә,
Юлларын син кисмә;
Агышы ун төрле,
Тавышы ун төрле,
Моң башка…

Бер тауда ун чишмә,
Ун яшь кыз — ун төштә,
Сөйләшеп туймыйлар,
Туй, диләр, туй, диләр,
Чибәрләр…

Бер тауда ун чишмә,
Сереңне син чишмә,
Сереңне чибәрләр
Дөньяга сибәрләр,
Сибәрләр…
```
