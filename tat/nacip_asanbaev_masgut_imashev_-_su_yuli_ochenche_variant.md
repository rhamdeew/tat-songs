# Оригинал

### Нәҗип Әсәнбаев / Мәсгуть Имашев - Су юлы (өченче вариант)

```
Килен булып төштем бу авылга,
Таныш түгел әле күршеләр.
Буй-сынымны сынап карар өчен,
Су юлына алып төштеләр.

Су юлы, су юлы,
Сынатмаммы бу юлы?
Сынатмасам бу юлы,
Сынатмам гомер буе!

Тугайлары ап-ак чәчкә генә,
Су буенда бөдрә каеннар.
Чылтыр-чылтыр аккан чишмәсенә
Сандугачлар кушылып сайрыйлар.

Су юлы, су юлы,
Сынатмаммы бу юлы?
Сынатмасам бу юлы,
Сынатмам гомер буе!

Күңелемә якын су юллары,
Монда минем ярым-канатым.
Рәхмәт сезгә, рәхмәт, әй кешеләр,
Бу якларны мин бик яраттым!

Су юлы, су юлы,
Сынатмаммы бу юлы?
Сынатмасам бу юлы,
Сынатмам гомер буе!
Су юлы, су юлы,
Сынатмам гомер буе!
```
