# Оригинал

### Татар халык җыры - Сания апа

```
Сания апа суга бара,
Яшел чиләген асып.
Сандугачлар чут-чут сайрый,
Көянтәсенә басып.

Сания апа суга бара,
Көянтәсе алмалы.
Алмалы көянтәсенә
Кунмаган кош калмады.

Сания апа суга бара,
Алма-алма чиләге.
Чиләгенә караганда,
Үзе кура җиләге.

Сания апа, чиләгеңнән
Суларың түгелмәсен;
Башкалар сүзенә карап,
Күңелең сүрелмәсен.

Кая бардың, Сания апа,
Кайдан кайтып киләсең?
Киләсең дә, гел китәсең,
Бик ямансу итәсең.
```
