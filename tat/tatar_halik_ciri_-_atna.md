# Оригинал

### Татар халык җыры - Әтнә

```
Җырлыйк Әтнә көйләренә,
Җырчыларга таралсын;
Мул уңышлар үстерүдә
Безнең Әтнә дан алсын.

Матур итеп кошлар сайрый
Әтнә су буйларында;
Ятсам — төштә, торсам истә —
Син генә уйларымда.

Кызарып кояш чыкмыйча,
Уйсу җирдә су кипми;
Көлемсерәп карауларың
Күз алларымнан китми.

Шушы көйне матур итеп
Җырлыйлар Әтнәләрдә;
Үзәкләремне өзәләр,
Исмеңне әйтәләр дә.
```
