# Оригинал

### Эльмира Шәрифуллина / Сәйяр Хәбибуллин - Нурга мана

```
Күз карашың таң нурыдай
Мәрхәмәтле, сөйкемле.
Бер кеше дә елмаялмый
Миңа синең шикелле.

Кушымта:
Көләч йөзең бар нәрсәдән
Ягымлырак күренә.
Шул килүдән кереп калдың
Йөрәгемнең түренә.

Җырларың белән зурладың,
Моңнарыңа исердем.
Кемдер татый да алмаган
Сихри тойгы кичердем.

Күз карашың уйга салды,
Наз биләде җанымны.
Бер елмаеп карауларың
Нурга мана таңымны.
```
