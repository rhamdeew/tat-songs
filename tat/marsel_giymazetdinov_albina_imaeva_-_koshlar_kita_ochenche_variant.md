# Оригинал

### Марсель Гыймазетдинов / Альбина Имаева - Кошлар китә (өченче вариант)

```
Берсен-берсе күзләп,
Үскән җирен эзләп,
Кошлар кайта туган ягына.
Ара якын тугел,
Сине көтә күнел,
Кайтчы, кайтчы иркәм, яныма.

Берсен-берсе эзлләп,
Ерак юлны күзләп,
Кошлар китә җылы якларга.

Тагын килде язлар,
Мине кем сон назлар,
Моңсуланып көткән чагымда?
Йөрәк ничек түзәр, —
Ул бит өмет өзәр
Син кайтмасан, иркәм, тагын да…

Берсен-берсе эзләп,
Ерак юлны күзләп,
Кошлар китә җылы якларга.
```
