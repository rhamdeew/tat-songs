# Оригинал

### Татар халык җыры - Әгерҗе

```
Әгерҗе көйләрен уйна,
Бар да кушылып җырласын;
Бәхетлеләрнең шат җырын
Ватаныбыз тыңласын.

Җырлыйбыз илебез
Гөлбакча булганга;
Дуслык үзәннәрен
Яктыртып торганга.

Үстерәбез мул уңышлар
Иркен кырлар өстендә;
Тиздән балкыр Алтын йолдыз
Әгерҗеләр өстеңдә.

Җырлыйбыз илебез
Гөлбакча булганга;
Дуслык үзәннәрен
Яктыртып торганга.

Гөлбакчабыз туган илне
Күз нурыдай сакларга;
Әгерҗедә батыр уллар
Әзер аны сакларга.

Җырлыйбыз илебез
Гөлбакча булганга;
Дуслык үзәннәрен
Яктыртып торганга.
```
