# Оригинал

### Клара Булатова / Рәис Нәгыймов - Кырга чыктым, юлга чыктым

```
Кырга чыктым, юлга чыктым,
Зәңгәр күккә карадым.
Зәңгәр күкләргә карадым,
Тугыз йолдыз санадым.

Зәңгәр күкләргә карадым,
Тугыз йолдыз санадым.
Зәңгәр күктә синнән якты
Йолдыз таба алмадым.

Зәңгәр күктә йолдызлар күп,
Иң яктысы бер генә.
Тирә-юньдә дусларым күп,
Иң якыны син генә.

Күңелемдә дә, күгемдә дә
Үз йолдызым бер генә.
Шул йолдызым син икәнен
Үзең белмисең генә.
```
