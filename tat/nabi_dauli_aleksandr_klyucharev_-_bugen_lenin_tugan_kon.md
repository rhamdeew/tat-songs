# Оригинал

### Нәби Дәүли / Александр Ключарев - Бүген Ленин туган көн

```
Кояш карый җиргә сокланып,
Кар сулары гөрләп агалар.
Сайрый кошлар, сайрый шатланып,
Тәрәзәдән җилләр кагалар.

Бүген Ленин туган көн,
Илдә шатлык булган көн;
Җылы яңгыр яуган көн,
Агач яфрак ярган көн.

Ялкын төсле кызыл тасмалар,
Алсулата урам йөзләрен;
Салкын якта гомер итсәң дә,
Яз җылысын сизәр йөрәгең.

Бүген Ленин туган көн,
Илдә шатлык булган көн;
Җылы яңгыр яуган көн,
Агач яфрак ярган көн.

Бусагада балкый алтын көн,
Безне көтә бәхег юллары.
Бу иртәдә газиз аналар
Уяталар назлы улларын,

Бүген Ленин туган көн,
Илдә шатлык булган көн;
Җылы яңгыр яуган көн,
Агач яфрак ярган көн.
```
