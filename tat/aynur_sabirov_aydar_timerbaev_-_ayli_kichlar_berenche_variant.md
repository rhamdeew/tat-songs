# Оригинал

### Айнур Сабиров / Айдар Тимербаев - Айлы кичләр (беренче вариант)

```
Җәйге матур аяз төннәрдә,
Ай шәүләсе күлгә төшкәндә,
Хәтерендәме, икәү бергә
Йөргән идек айлы кичләрдә.

Кайда калды җәйге аяз төннәр,
Синең белән йөргән,
Синең белән йөргән,
Синең белән йөргән айлы кичләр?
Су өстендә үскән
Серле төнбөеклар
Икебезне искә алырлар.

Җәйге яңгырлар һаман да ява,
Салкын җилләр һаман исәләр.
Синең белән йөргән айлы кичләр
Бик еш исләремә төшәләр.

Кайда калды җәйге аяз төннәр,
Синең белән йөргән,
Синең белән йөргән,
Синең белән йөргән айлы кичләр?
Су өстендә үскән
Серле төнбөеклар
Икебезне искә алырлар.
```
