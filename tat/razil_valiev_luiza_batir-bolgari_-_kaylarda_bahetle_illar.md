# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Кайларда бәхетле илләр?

```
Ник чыга кояш,
Ник ата таңнар,
Нигә сагышлы
Хыялый җаннар?
Ник ява яңгыр,
Ник исә җилләр?
Кайларда икән
Бәхетле илләр,
Кайларда бәхетле илләр?

Ник үсә гөлләр,
Ник шаулый урман,
Минем хыялның тамыры корган.
Ник яши кеше,
Ник дәшми илем?
Мин бу дөньяга
Нигә дип килдем?
Мин бу дөньяга нигә килдем?
```
