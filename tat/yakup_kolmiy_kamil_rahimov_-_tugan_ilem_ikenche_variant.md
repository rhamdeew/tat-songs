# Оригинал

### Якуп Колмый / Камил Рәхимов - Туган илем (икенче вариант)

```
Туган илем — шиңмәс гөл бакчасы,
Нинди җылы синең язларың.
Чәчәкләр уяна,
Күңелләр куана,
Җирне ямьли Урал тауларың,
Мәңге балкысын таңнарың.

Туган илем — сүнмәс кояшым син,
Кояш сымак — син дә бер генә.
Елмайган әнкәмдәй,
Иркәли-иркәли,
Үстерәсең безне, илгенәм.
Газиз әнкәмдәй, илгенәм.

Туган илем — бетмәс хәзинам син,
Тик син генә минем телемдә.
Уңгандыр улларың,
Җигәрле кызларың,
Шат яшиләр туган илемдә,
Күкрәп чәчәк аткан җиремдә,
Юмарт җиремдә, илемдә.
```
