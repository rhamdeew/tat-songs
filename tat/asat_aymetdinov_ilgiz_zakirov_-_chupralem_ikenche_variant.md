# Оригинал

### Асат Айметдинов / Илгиз Закиров - Чүпрәлем (икенче вариант)

```
Туган як дип сиңа әйтсәм дә азсынам,
Гүзәл як дисәм дә аз сыман.
Балкыйсың, Чүпрәлем, күңелем түрендә
Ташкыннар китергән яз сыман.

Искәндә саф җилләр туган як кырында,
Җилкетә игеннәр рәшәсен.
Саф моңнар, ак бәхет,
Ал кояш, якты нур
Чүпрәлем күгендә яшәсен.

Бирдең син хыялга очарга канатлар,
Үрләргә омтылган чагымда.
Урадың чәчәккә мәхәббәт гөлемне
Гөлләрем назга зар чагында.

Искәндә саф җилләр туган як кырында,
Җилкетә игеннәр рәшәсен.
Саф моңнар, ак бәхет,
Ал кояш, якты нур
Чүпрәлем күгендә яшәсен.

Җыр аша мин сиңа рәхмәтем яңартам,
Маяк син язмышым күгендә.
Туган як — торган як, син бөтен гомергә
Җыр булып калырсың күңелдә.

Искәндә саф җилләр туган як кырында,
Җилкетә игеннәр рәшәсен.
Саф моңнар, ак бәхет,
Ал кояш, якты нур
Чүпрәлем күгендә яшәсен.
```
