# Оригинал

### Хәбир Ибраһим / Мәсгуть Имашев - Авыл күле

```
Авыл күле тирән икән,
Чумганнарым булмады.
Бу хәсрәтләр килер диеп
Уйлаганым булмады.

Ялгыз үрдәк йөзә күлдә,
Шаулый бөдрә тал.
Газиз ярым, назлы җырым
Булып мәңге кал.

Белмәгәнмен кадереңне,
Соңлап кына аңладым.
Бер кайтырмын, дип киттең дә
Кайта гына алмадың.

Ялгыз үрдәк йөзә күлдә,
Шаулый бөдрә тал.
Газиз ярым, назлы җырым
Булып мәңге кал.

Иртән торып сулар сибәм
Бакчамдагы гөлемә.
И, Ходаем, сабырлык бир
Алда килер көнемә.

Ялгыз үрдәк йөзә күлдә,
Шаулый бөдрә тал.
Газиз ярым, назлы җырым
Булып мәңге кал.
```
