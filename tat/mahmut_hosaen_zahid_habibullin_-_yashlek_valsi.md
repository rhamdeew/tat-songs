# Оригинал

### Мәхмүт Хөсәен / Заһид Хәбибуллин - Яшьлек вальсы

```
Яңгыр тамчысы төшмичә,
Чәчәк ачылмас;
Җырламыйча, йөрәгемдә
Ялкын басылмас.

Яшәү — матур,
Дуслар — тату,
Йөрәкләрдә — чын ярату;
Өзелеп сөйсәң,
Янсаң, көйсәң,
Авыр аңлату!..

Алмалары биек үскән,
Ничек өзәргә?
Иркәм белән биемичә,
Ничек түзәргә?

Яшәү — матур,
Дуслар — тату,
Йөрәкләрдә — чын ярату;
Өзелеп сөйсәң,
Янсаң, көйсәң,
Авыр аңлату!..

Алмагачкаем янында
Сирень күрәм мин;
Якты залда биим, иркәм,
Синең белән мин.

Яшәү — матур,
Дуслар — тату,
Йөрәкләрдә — чын ярату;
Өзелеп сөйсәң,
Янсаң, көйсәң,
Авыр аңлату!..
```
