# Оригинал

### Фирүзә Абдуллина / Идел Нургалин - Бәхет кошы (өченче вариант)

```
Колагымда чыңлый әле булса,
Күңелемдә калды сакланып: —
Мин барыбер сиңа бәхет бирә
Алмас идем, — дидең, акланып.

Бәхет кошы безне урап үтте,
Нигә аны тота алмадык?
Без бәхетле ярлар булыр идек,
Нигә шуны соңлап аңладык?

Бәхет бирә алмас идең микән,
Әллә яхшыракны эзләдең?
Ялыныр дип әллә уйладыңмы,
Алларыңа синең тезләнеп? —

Бәхет кошы безне урап үтте,
Нигә аны тота алмадык?
Без бәхетле ярлар булыр идек,
Нигә шуны соңлап аңладык?

Мин бит горур идем, — дигән булдың…
Мин дә горур! Кемне алданык?
Бер бәхетле бөтен булыр идек —
Соңлап, үтә соңлап аңладык.

Бәхет кошы безне урап үтте,
Нигә аны тота алмадык?
Без бәхетле ярлар булыр идек,
Нигә шуны соңлап аңладык?
```
