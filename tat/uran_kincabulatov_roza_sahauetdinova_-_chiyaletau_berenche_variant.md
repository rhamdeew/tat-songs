# Оригинал

### Уран Кинҗәбулатов / Роза Сәхәүетдинова - Чиялетау (беренче вариант)

```
Чиялетау — хыялый тау,
Юл чатыннан ята читтә.
Чияле тау аша үткән
Яшьлек юлын бәхет көтә.

Чиялетау очраштырды
Матур яшьлек иртәсендә.
Гөл бәйләме ятып калды
Чиялетау итәгендә.

Кушкаенлы чиялетау,
Баш очында йолдыз яна.
Якты йолдыз, калса ялгыз,
Ишен эзләп җиргә тама.

Чиялетау очраштырды
Матур яшьлек иртәсендә.
Гөл бәйләме ятып калды
Чиялетау итәгендә.

Чиялетау — иң серле тау,
Җыярга кил җимеш — җиләк.
Калды безнең арбызда
Чиялетау — янар йөрәк!

Чиялетау очраштырды
Матур яшьлек иртәсендә.
Гөл бәйләме ятып калды
Чиялетау итәгендә.
```
