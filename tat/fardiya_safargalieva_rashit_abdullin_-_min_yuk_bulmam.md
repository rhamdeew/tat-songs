# Оригинал

### Фәрдия Сәфәргалиева / Рәшит Абдуллин - Мин юк булмам

```
Мин юк булмам, туган авылымның
Болынында чәчәк булырмын.
Йә булмаса, сихри кичләрендә
Йолдыз булып балкып торырмын.

Мин юк булмам, язлар килгән саен
Әйләнермен җылы җилләргә.
Тауларда үсәрмен чирәм булып,
Тургайдай сайрармын күкләрдә.

Шуннан мине юксынганнар эзләр,
Сагынганнар аңлар сүземне.
Туган ягымның бер тамчысыдай
Яшим мин хис итеп үземне.
```
