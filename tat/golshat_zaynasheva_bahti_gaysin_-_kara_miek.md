# Оригинал

### Гөлшат Зәйнәшева / Бәхти Гайсин - Кара мыек

```
Сызылып киткән кара мыек, —
Әллә каралта микән?
Мыек астыннан елмая,
Әллә ярата микән?

Сызылып киткән кара мыек
Йөрәгемә ут салды.
Бер көн озата килде дә
Әллә кая югалды.

Сызылып киткән кара мыек
Кайларда йөри микән?
Кайнар мәхәббәт сүзләрен
Кемнәргә сөйли икән?

Сызылып киткән кара мыек —
Буяган мыек икән;
Буялган мыеклыларга
Ышану кыен икән!..
```
