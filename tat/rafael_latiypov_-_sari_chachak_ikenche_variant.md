# Оригинал

### Рафаэль Латыйпов - Сары чәчәк (икенче вариант)

```
Болыннарда үскән чәчәкләрнең
Җыямын мин матур таҗлысын.
Мин аларның арасыннан эзлим
Тик матурын, тик-тик сарысын.

Сары төсне сагыш төсе диләр,
Сагышларның төсен кем күргән?
Сары чәчәкләрне кемнәр жыеп,
Жыйганына кем соң үкенгән?

Тимәгез сез сары чәчәкләргә,
Күрсәң алар төсе югала.
Гомеремнең соңгы көннәрендә
Булсын иде куллар кулларда.
```
