# Оригинал

### Нәҗип Мадьяров / Виталий Агапов - Зурлардан өйрәнәбез

```
Без әле зур түгел, әмма
Зурларга иярәбез.
Зурлар җырлаган шикелле
Җырларга өйрәнәбез.

Шулай ук яңадан-яңа
Көйләр дә өйрәнәбез.
Шигырьләрне дә алар күк
Сөйләргә өйрәнәбез.

Биеп, җырлап, биючегә,
Җырчыга әйләнәбез.
Зурлар кебек матур итеп
Яшәргә өйрәнәбез.
```
