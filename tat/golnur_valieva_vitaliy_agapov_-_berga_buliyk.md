# Оригинал

### Гөлнур Вәлиева / Виталий Агапов - Бергә булыйк

```
Чишмәләрдән кайта идең,
Аккоштай йөзеп кенә.
Чибәрлегеңә сокланып
Күзәттем читтән генә.

Кил әле чишмә буена —
Серләремне сөйләрмен.
Зәңгәр күзләреңә карап,
Бергә булыйк, диярмен.

Югалттым тынычлыгымны,
Сине күргәннән бирле.
Дулкын чәчләр, зәңгәр күзләр
Сихерләделәр мине.

Кил әле чишмә буена —
Серләремне сөйләрмен.
Зәңгәр күзләреңә карап,
Бергә булыйк, диярмен.

Чишмә юлларына чыгам,
Алсу таң ату белән.
Бер күрүдә гашыйк булу
Бигрәк тә читен икән.

Кил әле чишмә буена —
Серләремне сөйләрмен.
Зәңгәр күзләреңә карап,
Бергә булыйк, диярмен.
```
