# Оригинал

### Алсу Васикова / Алия Ганиева - Өмет итәм бу язга

```
Өй артында үскән шомырт
Ак чәчәккә күмелгән.
Әйтерсең ул, кәләш кебек,
Ак туй күлмәге кигән.

Яшь күңелем сөю көтә,
Өмет итәм бу язга.
Ә син, вакыт, ашыкмый тор,
Тукта инде беразга.

Бакчадагы ялгыз каен
Никтер башкаен игән.
Сөйгән ярын көтә-көтә
Сагыш томалын кигән.

Яшь күңелем сөю көтә,
Өмет итәм бу язга.
Ә син, вакыт, ашыкмый тор,
Тукта инде беразга.

Каенны юатырга дип
Күктән кояш елмая.
Кар астыннан баш калкытып
Чыга яшь умырзая.

Яшь күңелем сөю көтә,
Өмет итәм бу язга.
Ә син, вакыт, ашыкмый тор,
Тукта инде беразга.
```
