# Оригинал

### Дамир Гыйсметдин / Илгиз Мәҗитов - Гомер уза (беренче вариант)

```
Гомер уза аккан сулар кебек,
Йөрәкләрдә кала ярасы.
Кемнәр әйтер, кемнәр санап бирер
Тагын күпме калды барасын?

Сөенәм мин кайчак иртәләрен,
Шөкер ана кылам язмышка.
Төн-кичләрен, янә утлар сүнгәч,
Төшәмен, шул, кайгы-сагышка.

Яшәү-үлем — бары тик бер сукмак,
Нигә башка сукмак юк икән?
Адаштым, дип кире кайтыр идем,
Яшьлегемә кайсы юл илтә?

Гомер уза искән җилләр кебек,
Йөрәкләрдә кала үкенеч.
Әҗәлләрдән дару булмас инде,
Ходай яшәр өчен бирсен көч.
```
