# Оригинал

### Гөлнур Якупова / Салават Низаметдинов - Шәраб

```
Эңгер саркуында,
Шәмдәл балкуында
Ялтыр күлмәк киям,
Ялтыр сүзләр сөям.
Ишегемне ачам,
Сөймәгәнне кочам…
Йөрәгемдә — сагыш,
Һәм ул шундый таныш.

Шәраб күбекләнә,
Телләр җибәкләнә.
Алдыйсың син мине,
Алдыймын мин сине…

Нечкә аулар үрәм,
Үтәләйгә күрәм:
Шәраб коясыңмы,
Ауны тояһыңмы?
Йөрәгемдә — сагыш,
Һәм ул шундый таныш.
Их, исерткеч күбек —
Безнең уен кебек…

Шәраб күбекләнә,
Телләр җибәкләнә.
Алдыйсың син мине,
Алдыймын мин сине…
```
