# Оригинал

### Зөфәр Билалов - Әнкәй-Әткәй

```
Көлеп үткәрдегез, көлеп күтәрдегез
Шатлыгын, тормыш авырлыгын да
Әнкәй-әткәй, әткэй-әнкәй
Сезнең киңәш, безнең һәрчак гомер барышында

Әнкәй-әткәй, әткэй-әнкәй
Ике йолдыз, ике моң, ике жыр
Кузләргә җылы наз,
Кунелләрдә моңлы аваз
Сез булганда гына куңелдә яз

Рәхмәт сезгә җан җимешләр, олы рәхмәт!
Сез пар йолдыз, пар былбыл, пар аккош
Доньяда иң назлы, иң татлы ике суз
Әнкәй, әнкәйдән аерма язмыш!
```
