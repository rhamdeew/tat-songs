# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Кырык яшь миңа (сиңа)

```
Кырык яшь миңа (сиңа), кырык яшь, кырык яшь
Өй алдында үсә миләш.
Миләшем дә миңа тиңдәш
Димәк аңа да шул кырык яшь.

Чакырма яшьлек чакырма,
Йөрәгемне арбап торма,
Үткәннәрне сагындырма,
Назларымны кабындырма.

Кырык яшь миңа (сиңа) инде тулган
Давыл булып яшьлек узган.
Аны мин сагынам кайчак
Ләкин яшьлек кайтмаячак.

Чакырма яшьлек чакырма,
Йөрәгемне арбап торма,
Үткәннәрне сагындырма,
Назларымны кабындырма.

Өй алдында кипте миләш
Нәрсә булды сиңа яшьтәш?
Кем сындырды әйтче сине,
Калдырып ялгызга мине?

Чакырма яшьлек чакырма,
Йөрәгемне арбап торма,
Үткәннәрне сагындырма,
Назларымны кабындырма.
```
