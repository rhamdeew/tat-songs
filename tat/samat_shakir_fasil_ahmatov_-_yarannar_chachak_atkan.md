# Оригинал

### Самат Шакир / Фасил Әхмәтов - Яраннар чәчәк аткан

```
Тәрәзәгә баккан да,
Ялгыз ана моңлана,
Җыерчыклы кулына
Энҗе яшьләре тама.
Авыр сугыш еллары
Маңгаен сырлап үткән.
Өй түрендә диварда
Дүрт рәсем тора күптән.
Батырлар кая киткән?
Бик авыр еллар үткән…
Матчага кыстырылган
Дүрт кәгазь кибеп беткән.
Яраннар чәчәк аткан,
Тик ана уйга баткан.
Өч малаен, ирен ул
Изге яуда югалткан…
```
