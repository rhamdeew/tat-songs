# Оригинал

### Гөлфия Шакирова / Гүзәлия - Былбыл (икенче вариант)

```
Шул кадәрле сайрый былбыл,
Нигә өздерә икән?
Уйлар сагышка әйләнеп,
Үзен сиздерә икән.

Сайра, сайра, былбыл кошкай,
Сайрачы өзеп-өзеп.
Бик күрәсем килгән чагым,
Булырмы икән түзеп?

Кайткан ла, кайткан былбыл кош,
Үзенең талларына.
Син килгәндә йөгерепләр,
Чыгарырмын алларыңа.

Сайра, сайра, былбыл кошкай,
Сайрачы өзеп-өзеп.
Бик күрәсем килгән чагым,
Булырмы икән түзеп?

Әй, былбыл кош, минем кебек
Бигрәкләр моңлысың.
Үзәкләремне өзәсең,
Яратуың чынмы соң?

Сайра, сайра, былбыл кошкай,
Сайрачы өзеп-өзеп.
Бик күрәсем килгән чагым,
Булырмы икән түзеп?
```
