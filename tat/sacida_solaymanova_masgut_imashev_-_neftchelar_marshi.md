# Оригинал

### Саҗидә Сөләйманова / Мәсгуть Имашев - Нефтьчеләр маршы

```
Җырлап чыгабыз иртән кырларга,
Кырлар, урманнар уяна.
Көч бар беләктә,
Кодрәт йөрәктә —
Кеше бәхеткә юл яра.

Суда йөзәргә,
Күктә очарга,
Галәм серләрен
Ачарга!

Корыч манара аумас җилләрдә,
Күңел омтыла тирәнгә.
Таулар селкенә,
Йөрәк җилкенә,
Максат бер генә — җиңәргә!

Суда йөзәргә,
Күктә очарга,
Галәм серләрен
Ачарга!

Җырлап чыгабыз иртән кырларга,
Кырлар, урманнар уяна.
Утлар кабына,
Нурлар агыла,
Җыр кала туар елларга.
```
