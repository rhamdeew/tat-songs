# Оригинал

### Нурия Измайлова / Илһам Хисмәтуллин - Адаштым (беренче вариант)

```
Ашыгулар ялгышуга була,
Ашыгырга никтер яраттым.
Ашыгулар — урман,
Ялгышларым — буран,
Йөгерә-йөгерә кереп адаштым.

Ашкынулар үкенечкә була,
Юк шул минем сабыр канатым.
Үкенечләр — урман,
Урманнарда — буран,
Көлә-көлә кереп адаштым.

Ашыгулар һәлакәткә илтә,
Ник мин сине шашып яраттым?!
Карашларың — урман,
Мәхәббәтем — буран,
Җырлый-җырлый кереп адаштым.

Карашларың — урман,
Мәхәббәтем — буран,
Елый-елый кереп адаштым.
```
