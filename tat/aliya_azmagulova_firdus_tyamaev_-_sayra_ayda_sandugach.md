# Оригинал

### Алия Әзмәгулова / Фирдүс Тямаев - Сайра, әйдә, сандугач

```
Бакча артындагы талда
Өздереп сайрый бер кош,
Сайрый ул җәйләрдә генә,
Ишетелми көзен-кыш.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Сайрый минем сандугачым,
Күңелдә эз калдырып,
Сайра, иркәм, син дә шулай,
Йөрәгемне яндырып.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Иркәм килми, хәбәр бирми,
Сандугач та сайрамый,
Йөрәккәем синнәң башка
Башкаларны сайламый.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.

Бакча артындагы талда
Өздереп сайрый бер кош,
Сайрый ул җәйләрдә генә,
Ишетелми көзен-кыш.

Әйдә, әйдә, сандугач,
Үтә син теләгемне,
Син генә аңлый аласың
Минем яшь йөрәгемне.
```
