# Оригинал

### Сания Әхмәтҗанова / Зөфәр Хәйретдинов - Кура җиләк

```
Кура җиләк пешкәндер дип,
Чакырдың, урманнарга;
Икәү бергә күмелдек без
Хуш исле кураларга.

Коела инде куралар —
Җил үтте аралардан.
Ялгызым йөрим адашып,
Куралы аланнарда.

Җем-җем яна кура җиләк —
Җыймыйча булмый түзеп.
Уч тутырып кура җиләк
Ашаттың миңа өзеп.

Коела инде куралар —
Җил үтте аралардан.
Ялгызым йөрим адашып,
Куралы аланнарда.

Иреннәрдә җиләк тәме —
Моңаймый булмый түзеп.
Кура җиләк кабат пешәр,
Тик кем ашатыр өзеп?

Коела инде куралар —
Җил үтте аралардан.
Ялгызым йөрим адашып,
Куралы аланнарда.
```
