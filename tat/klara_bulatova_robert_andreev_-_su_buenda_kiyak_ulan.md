# Оригинал

### Клара Булатова / Роберт Андреев - Су буенда кыяк үлән

```
Су буенда кыяк үлән
Язын булмый торгандыр.
Бәлки ул да таңнар белән
Мине уйлый торгандыр,
Мине уйлый торгандыр.

Су буенда кыяк үлән
Ул да җыя торгандыр.
Бәлки ул да сагыш белән
Уф, дип куя торгандыр,
Уф, дип куя торгандыр.

Су буенда кыяк үлән
Билдән күмә торгандыр.
Бәлки ул да чын күңелдән
Янып сөя торгандыр,
Янып сөя торгандыр.

Су буенда кыяк үлән
Чабып алсаң ни була?
Уйлап йөргәч, сөям, дигән
Хәбәр салса ни була,
Хәбәр салса ни була?
```
