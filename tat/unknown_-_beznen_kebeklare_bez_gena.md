# Оригинал

### Зифа Нагаева - Безнең кебекләре без генә

```
Ялгыз булу барыбер бәхет түгел,
Ялгыз гомер әрәмгә үтә.
Үз тиңеңне табу җиңел түгел,
Парлы булуларга ни җитә!

Парлашканнар бик күп бу дөньяда,
Безнең кебекләре без генә.
Минем кебекләре мин генә дә
Синең кебекләре син генә.

Парлы бәхет кенә тулы бәхет,
Язмышларны бәхеткә илтә.
Бер-береңә бәхет бүләк итеп
Яши алуларга ни җитә!

Парлашканнар бик күп бу дөньяда,
Безнең кебекләре без генә.
Минем кебекләре мин генә дә
Синең кебекләре син генә.

Син дә миңа, мин дә сиңа берәү,
Өлешләргә көмеш тигәне.
Ахыр чиккә кадәр бул янәшә,
Гомерлеккә генә бул, яме.

Парлашканнар бик күп бу дөньяда,
Безнең кебекләре без генә.
Минем кебекләре мин генә дә
Синең кебекләре син генә.
```
