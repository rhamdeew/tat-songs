# Оригинал

### Кәүсәрия Нуруллина / Шамил Шәрипов - Ялгыз аем

```
Тәрәзләрдә ай яктысы,
Иңемдә мамык шәлем.
Караңгы төн, без икебез
Ялгызлык һәм мин үзем.

И аем, аем, аем,
Синең дә юк бит парың.
Зәңгәр киңлектә йөзәсең,
Ялгыз ничек түзәсең.

Өем эче караңгылык,
Өстәлдә ялгыз шәмем.
Моңлы шәлем ябынсам да,
Җылы табалмый тәнем.

И аем, аем, аем,
Синең дә юк бит парың.
Зәңгәр киңлектә йөзәсең,
Ялгыз ничек түзәсең.

Күктә йөзә ялгыз аем,
Иңемдә мамык шәлем.
Үзе татымаган белми
Ялгыз калганнар хәлен.

И аем, аем, аем,
Синең дә юк бит парың.
Зәңгәр киңлектә йөзәсең,
Ялгыз ничек түзәсең.
```
