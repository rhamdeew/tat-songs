# Оригинал

### Әхмәт Исхак / Әнвәр Бакиров - Партия турында җыр

```
Яши идек без,
Белми илдә ирек,
Сулган гөлләр кебек,
Сусап суларга.
Син, Партиябез,
Күрсәттең юл безгә
Дуслар белән бергә
Ирек яуларга.

Аерылмый дуслардан,
Син күрсәткән юлдан
Бар халыклар бергә
Бардык без алга.
Ирек таптык бергә,
Чәчәк аттык бергә,
Килеп җиттек илгә
Бәхет нуры балкыган
Якты таңнарга.

Туган елгабыз —
Идел ярларында
Көннән-көн нурлырак
Тормыш яктыра.
Рәхмәт әйтәбез
Сиңа чын күңелдән,
Бәхет китергәнгә
Татар халкына!

Аерылмый дуслардан,
Син күрсәткән юлдан
Яна җиңүләргә
Барабыз алга.
Партия, синнән без
Мәңге аерылмабыз,
Җиңеп көрәшләрдә,
Туган илне бизәрбез
Сүнмәс данга! Данга!
```
