# Оригинал

### Халык / Рәис Ханнанов - Катерина (Катаржан юлы)

```
Әнкәй безне тугачтын да
Ак биләүгә биләгән,
Ак биләүләргә биләгән
Бәхетең булсын димәгән.
Әби патша юл салдырган,
Безгә ирек бирмәгән.

Катаржан юлы, катаржан юлы,
Татарлар юлы, эх, такыр микән,
Күңелем сизә, шушы юллардан
Газиз башкаем, эх, кайтыр микән.
Катерина, Катерина,
Юлларың такыр,
Катерина, Катерина
Әйләнеп кайтыр.

Кояш чыга әйләнеп,
Ак болытка бәйләнеп.
Насыйп булса, без кайтырбыз,
Без кайтырбыз әйләнеп,
Әби патша юлларында
Богауларга бәйләнеп.

Катаржан юлы, катаржан юлы,
Татарлар юлы, эх, такыр микән,
Күңелем сизә, шушы юллардан
Газиз башкаем, эх, кайтыр микән.
Катерина, Катерина,
Юлларың такыр,
Катерина, Катерина
Әйләнеп кайтыр.

Кай якларга очасың дип,
Кыр казыннан сорадым.
Ул кычкырып җавап бирде,
Мин утырып еладым,
Зәңгәр күктә кыр казларын
Кайтуыма юрадым.

Катаржан юлы, катаржан юлы,
Татарлар юлы, эх, такыр микән,
Күңелем сизә, шушы юллардан
Газиз башкаем, эх, кайтыр микән.
Катерина, Катерина,
Юлларың такыр,
Катерина, Катерина
Әйләнеп кайтыр.
```
