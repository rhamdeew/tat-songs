# Оригинал

### Әминә Яхина / Роза Сәхәүетдинова - Ак чәчәк

```
Без болында икәү идек
Син бер ак чәчәк өздең.
Сөя — сөйми дип иркәләп,
Син аны өзгәләдең.

Күп сайлама чәчәкләрне,
Син минем сөйгәнне бел.
Өзгәләмә үзәгемне,
Ул сиңа чәчәк түгел.

Аланнарда яз җиленә
Ак чәчәк иркәләнә.
Шул чәчәкнең исләренә
Күңелем иркәләнә.

Күп сайлама чәчәкләрне,
Син минем сөйгәнне бел.
Өзгәләмә үзәгемне,
Ул сиңа чәчәк түгел.
```
