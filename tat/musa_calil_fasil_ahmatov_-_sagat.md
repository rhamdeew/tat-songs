# Оригинал

### Муса Җәлил / Фасил Әхмәтов - Сәгать

```
Сәгать суга: «даң, даң!..»
Хәбәр бирә таңнан:
Бакчага барырга
Унбиш минут калган.
Сикереп торды Марат,
Күрә — эшләр харап.
Тагын соңга калган,
Шул йокыга карап.
Сәгать йөри: «келт, келт!..»
— Тиз бакчага кит, кит!..
Марат аңа дәшә:
— Тукта, мине көт, көт!..
Сәгать җырлый: «диң, диң!..»
— Мин бит туктый белмим.
Моннан ары: «зиң, зиң!»
Миңа карап йөр син!..
```
