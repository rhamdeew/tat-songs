# Оригинал

### Асия Юнысова / Тәслимә Низами - Айлы юл

```
Айны юлдаш иттем,
Юлга чыгып киттем,
Яшь наратлар йөгерә каршыма.
Көмеш нурлар тама,
Айдан бар дөньяма,
Юлларыма йолдыз коела.

Ай нурланыр узып,
Ике кулың сузып,
Үзең чыгып мине каршыла.
Көткәнеңне беләм,
Шуңа очып киләм.
Әкият булып җиһан тоела.

Бик ашкындым быел,
Ай толымы бу юл,
Сиңа илтер мине,бергенәм.
Хыял белән өнем
Бу кушылган төнем.
Көтеп алыр кешем син генә.
```
