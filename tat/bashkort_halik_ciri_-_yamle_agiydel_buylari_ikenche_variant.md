# Оригинал

### Башкорт халык җыры - Ямьле Агыйдел буйлары (икенче вариант)

```
Агыйделкәйләрнең, ай, кичкәндә
Бер җырладым ишкәкләр ишкәндә.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.

Бер җырладым, бәгърем, бер еладым
Туган җирләр искә лә төшкәндә.

Ямьле Агыйдел буйлары ла
Бетерәдер хәсрәтле уйларны.
```
