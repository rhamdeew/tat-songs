# Оригинал

### Асия Басырова / Билгесез - Сөю уты сүнми (беренче вариант)

```
Карашларым белән эзлимен,
Назлы күзләреңне күзлимен.
Хыялымда тик син,
Уйларымда тик син —
Бары үзең генә белмисең.

Сөю уты белән шаярма,
Онытыр бу диеп уйлама.
Бәхетемдә тик син,
Өметемдә тик син —
Башкаларга берүк күз салма.

Алсу томаннардан юл ярдым,
Шатлыкларга аны юрадым.
Мәхәббәтем моңсу
Йөрәгемә сагыш булып
Нигә генә уралдың?

Сөю уты белән шаярма,
Онытыр бу диеп уйлама.
Бәхетемдә тик син,
Өметемдә тик син —
Башкаларга берүк күз салма.

Алсу томаннардан юл ярдым,
Шатлыкларга аны юрадым.
Мәхәббәтем, нигә
Моңсу йөрәгемә
Утлы сагыш булып уралдың?

Сөю уты белән шаярма,
Онытыр бу диеп уйлама.
Бәхетемдә тик син,
Өметемдә тик син —
Башкаларга берүк күз салма.
```
