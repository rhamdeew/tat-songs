# Оригинал

### Билгесез - Зәңгәр күзләр (икенче вариант)

```
Икәү бергә үстегез сез,
Аерылмас дус идегез.
Яшьнәп торган яшьлек белән
Авырлыклар җиңдегез сез.

Зәңгәр күзләр, зәңгәр күзләр,
Нигә шулай иттегез сез,
Ике дусны ике якка
Аерып алып киттегез?

Яшь чакларда яшь йөрәкләр
Икеләтә ешрак тибә.
Арага керде мәхаббәт,
Дошман итте бер көндә.

Зәңгәр күзләр, зәңгәр күзләр,
Нигә шулай иттегез сез,
Ике дусны ике якка
Аерып алып киттегез?

Сөйгәннәр бәхетле диләр,
Кемнәр чишәр бәхәсне?
Чын дусларга яра салып,
Пычак кисте бәхетне.

Зәңгәр күзләр, зәңгәр күзләр,
Нигә шулай иттегез сез,
Ике дусны ике якка
Аерып алып киттегез?
```
