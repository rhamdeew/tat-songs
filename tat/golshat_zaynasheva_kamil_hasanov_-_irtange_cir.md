# Оригинал

### Гөлшат Зәйнәшева / Камил Хәсәнов - Иртәнге җыр

```
Таңны мактап талларда
Кошлар канат кактылар.
Тамчы гөлләр тибрәнеп,
Алкаларын тактылар.

Күбәләкләр уянып,
Канатларын җилпеде.
Кондызлар җемелдәвек
Ефәк күлмәген киде.

Чәчәкләр чык суында
Коенып юындылар.
Бал кортлары безелдәп,
Чәчәкләргә кундылар.

Син дә, матур сеңелем,
Татлы йокыңнан уян.
Җәйге таңның яменә,
Гүзәллегенә куан.

Уян, сеңелем, күзең ач,
Җәйге иртә нурында.
Кояш чыкты алланып
Синең тәрәз турыңда.
```
