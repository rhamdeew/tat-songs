# Оригинал

### Гөлшат Зәйнәшева / Татар халык көе - Кердем карурманнарга

```
Кердем карурманнарга,
Калдым караңгыларга.
Газиз башлар сау булганда,
Нигә ямансуларга?..

Кердем карурманнарга,
Бер кычкырып җырларга.
Бәгърем, бергә булсак сүз бир,
Ни күрсәк тә чыдарга.

Кердем карурманнарга,
Кермәдем аланнарга.
Көтегез, кайтырбыз, дидек,
Илләрдә калганнарга!..
```
