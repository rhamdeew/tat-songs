# Оригинал

### Газиз Моталлапов / Сания Әхмәтова - Безнең Азнакаебыз

```
Күпләрне гашыйк итәрлек
Азнакай матурлары,
Һаман күбрәк муллык яулый
Безнең ил батырлары.

Балкый Азнакай таңы,
Таңнардай якты даны,
Ничек онытасың аны?

Дан казанды илебездә
Кара алтын — җир мае.
Азнакайдан үсеп чыкты
Бөек хезмәт герое.

Балкый Азнакай таңы,
Таңнардай якты даны,
Ничек онытасың аны?

Киң урамнар, биек йортлар,
Тигез асфальт юлыбыз.
Көн дә үсә, матурлана
Безнең Азнакаебыз.

Балкый Азнакай таңы,
Таңнардай якты даны,
Ничек онытасың аны?
```
