# Оригинал

### Фәнүзә Зөлкарнаева-Нәбиуллина / Рәзидә Әхмәтова - Моң дөньясы

```
Җырлары тын чишмә сыңнарыннан,
Сандугач моңнар бүләк итте.
Чәчәкләр дә назлы елмаешып,
Моң тулы карашларын сирпте.

Минем күңел моңнар дөньясы ул,
Көе хистән туҡылган дөньям.
Нурлы, чагу, матур шул дөньяда
Мин үземде ирекле тоям.

Әнкәй — бергенәмнең бишек җыры —
Моң дөньясының тәхетендә.
Күңелемнең җыр-моң дөньясыннан —
Ходай, аерма бәхетемнән!
```
