# Оригинал

### Кадыйр Даян / Мәсәлим Вәлиев - Өфе каласы

```
Агыйделнең буе ямьле,
Гөл чәчәкле даласы.
Шул Иделнең ярларында
Матур Өфе каласы.

Эх, Өфе каласы —
Ямьле Идел буенда.
Кайда гына йөресәм дә
Син булырсың уемда.

Өфе урамнары асфальт,
Биек яңа йортлары.
Кара төнне көндез итә
Балкып янган утлары.

Эх, Өфе каласы —
Ямьле Идел буенда.
Кайда гына йөресәм дә
Син булырсың уемда.

Өфе бакчалары матур,
Гөл чәчәккә күмелгән.
Җәйге көндә бер күрсәң дә
Мәңге китми күңелдән.

Эх, Өфе каласы —
Ямьле Идел буенда.
Кайда гына йөресәм дә
Син булырсың уемда.
```
