# Оригинал

### Мөфтәдин Гыйләҗев - Әнкәемнең чулпы чыңнары

```
Тормышыңда әнкәй күпте күрде,
Чәчләренә кунды чаңнары.
Шул чәчләрдә һаман чыңлый кебек
Көмеш чулпыларның чыңнары.

Ходай биргән гомер чиксез түгел,
Тыналар шул чулпы чыңнары.
Бергә чакның каҙерләрен бел син,
Сагышлы ул чулпы чыңнары.

Ул серле моң һаман яши миндә,
Сагындырып үткән елларны.
Йөрәк әрни, бер тыңлыйсы иде
Әнкәемнең чулпы чыңнарын.
```
