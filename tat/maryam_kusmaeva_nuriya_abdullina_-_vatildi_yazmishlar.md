# Оригинал

### Мәръям Күсмәева / Нурия Абдуллина - Ватылды язмышлар

```
Ватылды язмышлар,
Ватылды — ялганмый.
Төзәтеп булыр, дип
Күңелне алдама.

Мин беләм: һич кайчан
Ватылган ялганмый.
Тик эри күңелем
Бу татлы ялганга.

Кителде күңелем
Хәтта бер сүздән,
Яшьләрем күземнән
Тукталмый түгелә.

Мин беләм: һич кайчан
Ватылган ялганмый.
Тик эри күңелем
Бу татлы ялганга.

Какшамастай иде
Без корган бу нигез.
Шатлыклар — аерым,
Кайгылар — бер тигез.

Мин беләм: һич кайчан
Ватылган ялганмый.
Тик эри күңелем
Бу татлы ялганга.
```
