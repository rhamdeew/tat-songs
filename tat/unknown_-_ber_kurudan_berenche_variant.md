# Оригинал

### Ризида Мөхәммәтҗанова / Гүзәллия Хәкимҗанова - Бер күрүдән (беренче вариант)

```
Бер күрүдән, матур елмаюдан
Күзләремнән шатлык коелды.
Кар бөртеге кебек нурын чәчкән
Шатлык җыры сәер тоелды.

Кушымта:
Матур безнең уртак тормышыбыз,
Рәхмәт, Ходай, теләк үтәлә.
Элеккечә насыйп итсәң иде,
Гомер юлын тату үтәргә.

Яңа ай да йөзә, яктысыннан
Күзләреңдә — йолдыз ялкыны.
Без бергәбез, арабызда гүя
Энҗе карның аклык, сафлыгы.

Кушымта.

Бер мизгелдә карлар елагандай,
Эреде дә куйды учымда.
Кыш очраштык. Шушы яктылыклар
Көтсен иде мизгел очында.

Кушымта.
```
