# Оригинал

### Лилия Гыйбадуллина / Илнур Ганиев - Син очар канатым

```
1. Хыялым каршына,
Мин сиңа ашкынам.
Юлларга ак томан сарылган.
Таңнарым атса да,
Кояшым батса да,
Мин сине, бер сине сагынам.

2. Син — минем сагышым,
Гомерлек табышым.
Нигә соң соңарып табылдың?
Язмышлар яралы,
Һай текә (текәдер) ярлары.
Мин сине өзелеп сагындым.

3. Гомерләр узганчы,
Йөрәкләр тузганчы,
Мәңгегә, гомергә яраттым.
Хыяллар күгендә
Без бергә, без бергә.
Син — минем очар пар канатым.
```
