# Оригинал

### Марсель Гыймазетдинов / Илгиз Закиров - Әйләнеп кайт әле авылга

```
Юксынам дисең син, нигә соң?
Гомерең узмаган бит әле.
Туган як туфрагын сагынып,
Әйләнеп авылга кайт әле.

Саф чишмә сулары, һавасы,
Тугае, болыны, урманы
Кочаклап алырлар үзеңне —
Әйләнеп авылга кайт әле.

Күзләрең ямансу, боеклар —
Алар да авылны сагына.
Кайгырма, моңайма, юксынма,
Әйләнеп кайт кына авылга.

Юксынам дисең син, нигә соң?
Гомерең узмаган бит әле.
Туган як туфрагын сагынып,
Әйләнеп авылга кайт әле.
```
