# Оригинал

### Башкорт халык җыры - Уел (өченче вариант)

```
Ике лә генә егет печән чаба
Уел буйкаенда ла чабында.
Кайда ла гына йөреми, ниләр күрми
Ир-егеткәй исән дә чагында.

Менгән генә атым, ай, кир булыр,
Тибенгесе аның да тир булыр.
Урал буйкаеның асыл егете
Данкайлары чыгыр да ир булыр.

Уелып кына аккан Уел суы
Уйландыра уйчан да ирләрне,
Уйландырып калмый, моңландыра,
Сагындыра туган да җирләрне.
```
