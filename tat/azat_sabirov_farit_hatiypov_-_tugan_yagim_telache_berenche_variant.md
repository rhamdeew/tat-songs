# Оригинал

### Азат Сабиров / Фәрит Хатыйпов - Туган ягым Теләче (беренче вариант)

```
Кемгәдер кирәк яшел җәй,
Кемгәдер ап-ак буран.
Миңа кирәк туган ягым,
Теләчене сагынам.

Эх, Теләчем, Теләчем
Китмисең уйларымнан.
Аермасын мине язмыш,
Туып үскән ягымнан.

Су буең, урманың ямьле
Туя алмыйм сокланып.
Кошларда бит кире кайта,
Туган ягын сагынып.

Эх, Теләчем, Теләчем
Китмисең уйларымнан.
Аермасын мине язмыш,
Туып үскән ягымнан.

Таң атканда Мишә буе
Ак томанга төренә.
Талларында кош сайраса,
Шифа була күңелгә.

Эх, Теләчем, Теләчем
Китмисең уйларымнан.
Аермасын мине язмыш,
Туып үскән ягымнан.
```
