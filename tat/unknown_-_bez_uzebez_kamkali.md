# Оригинал

### Гөлшат Зәйнәшева / Фәрит Хатыйпов - Без үзебез Камкалы

```
Пиана дигән елганың
Гөлчәчәкле ярлары.
Пиана буенда устек,
Без — Нижгар татарлары

Без үзебез Камкалы…
Алтын — көмеш капкалы.
Дусты килсә, мишәр — татар
Киереп ача капканы!..

Безнең авыл бик борынгы,
Исме нигә Камкадыр?..
Күккә, йолдызларга ашкан
Мәчетләре калкадыр…

Без үзебез Камкалы…
Алтын — көмеш капкалы.
Дусты килсә, мишәр — татар
Киереп ача капканы!..

Дан тота Нижгар татары
Уйсу җирләре белән.
Рәшит Баязит шикелле
Асыл ирләре белән.

Без үзебез Камкалы…
Алтын — көмеш капкалы.
Дусты килсә, мишәр — татар
Киереп ача капканы!..

Эшли Нижгар Мөскәүдә дә,
Самарда, Себердә дә.
Авыр эштән мишәр — татар
Курыкмас гомергә дә!

Без үзебез Камкалы…
Алтын — көмеш капкалы.
Дусты килсә, мишәр — татар
Киереп ача капканы!..
```
