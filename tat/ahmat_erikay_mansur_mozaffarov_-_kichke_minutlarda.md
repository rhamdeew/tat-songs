# Оригинал

### Әхмәт Ерикәй / Мансур Мозаффаров - Кичке минутларда

```
Йолдызлы кичне каршылый
Дус-ишләр туган илдә.
Мин телим керсез йөрәктән
Хәерле кичләр сезгә.
Хәерле кичләр, хәерле кичләр,
Хәерле кичләр сезгә!

Җыр да моңлы, юл да нурлы
Туган ил күкрәгендә.
Кичке татлы минутларда
Яңа җыр туа миндә.
Яңа җыр, яңа җыр туа,
Яңа җыр туа миндә.

Ташка баскан лачын кебек,
Таудагы бөркет кебек,
Талпына күңел үргә.
Җырларым белән телим мин
Хәерле кичләр илгә.
Хәерле кичләр, хәерле кичләр,
Хәерле кичләр илгә.

Кем яратмый җиребезнең
Иркенлеген, киңлеген!
Галәмгә чаклы яңратып
Җырлыйсы килә бүген!
Хәерле кичләр, хәерле кичләр,
Телисе килә бүген!..
```
