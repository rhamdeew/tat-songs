# Оригинал

### Әнисә Мортазина / Илназ Гарипов - Сагынам сине (икенче вариант)

```
Син бар чакта күңелемне иркәли яз иртәсе,
Мәхәббәт буласы булса, син аның фәрештәсе.
Үбәр идем иренеңнән, юк, җаным, янымда син,
Бергә булганчыга кадәр йөрәгем яртыга ким.
Җилләр тузгый йөзләремдә, фәрештәм көтә мине,
Сагынышып көткәннәргә мәхәббәт — бәхет иле.

Көндезен сагынып уйланам, төннәрем сагынып уянам,
Сагынам ялгызым калганда, сагынам, чөнки бик яратам.

Син минем якты кояшым, аем караңгы төндә,
Һәрвакыт бергә булырбыз, бирмәм сине беркемгә.

Көндезен сагынып уйланам, төннәрем сагынып уянам,
Сагынам ялгызым калганда, сагынам, чөнки бик яратам.
Көндезен сагынып уйланам, төннәрем сагынып уянам,
Сагынам ялгызым калганда, сагынам, чөнки бик яратам.
```
