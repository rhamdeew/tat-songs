# Оригинал

### Филүс Хисаметдинов / Илнат Фәрхуллин - Гомер кыска

```
Гомер судай ага, бар да вакытлыча,
Ходай бирә шулай сынауны, сынауны.
Әйтче, кәккүк миңа, бәхет кошы: нигә
Гомеркәйләр кыска, санаулы, санаулы?

Ходай биргән гомер бигрәк кыска икән,
Сәгать саный һәрбер минутын.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.

Санаулы шул гомер, вакыт җиткәч туктый
Ярсып беткән адәм йөрәге, йөрәге.
Ашыктырма, зинһар, еллар агышларын,
Яшим әле язмыш биргәнен, биргәнен.

Ходай биргән гомер бигрәк кыска икән,
Сәгать саный һәрбер минутын.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.

Ходай биргән гомер бигрәк кыска икән,
Сәгать саный һәрбер минутын.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен.
Әй, кәккүгем минем, син дә санамачы,
Кыскартмачы йөрәк тибешен…
```
