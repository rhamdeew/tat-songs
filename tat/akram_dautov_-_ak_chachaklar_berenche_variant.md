# Оригинал

### Әкрәм Даутов - Ак чәчәкләр (беренче вариант)

```
Таллыкта алан,
Аланны япкан
Ап-ак чәчәкләр.
Без бергә идек,
Калфаклар үрдек
Шул ак чәчкәдән.

Алсу йөзеңә
Бигрәк килешә
Чәчкә калфагың!
Синнән дә сылу
Чәчәкләр булу
Мөмкинме тагын?!

Дөрләсен хисләр!
Тәмле, хуш исләр
Җилдә уйнасын!
Безнен күңелләр
Сулган чәчәкләр
Кебек булмасын!
```
