# Оригинал

### Нурия Гыйләҗева / Ризида Мөхәммәтҗанова - Эх, гармун, гитара!

```
Гармун алдым кулларыма
Телләренә басарга.
Гитара алдым кулыма,
Якын дусым ясарга.

Кушымта:
Эх, гармун, гитара,
Эх, гармун, гитара.
Икесе дә кызыктыра,
Син анда уйнап кара.

Кил әле син, кил әле син,
Гармун уйный бер егет.
Бер елмаеп көл әле син,
Җырлап, үзеңнеке ит!

Кушымта.

Гармунчының иң шәбе син,
Гармуныңны бир әле.
Телләрендә тылсым тулы,
Мин дә уйный(ы)м әле.

Кушымта.
```
