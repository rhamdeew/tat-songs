# Оригинал

### Флюсә Хәмитова / Оскар Усманов - Ялгыз бакчам

```
Ялгыз бакчам түрләрендә
Үсә миләш һәм балан.
Сагынганда,саргайганда
Күзем алалмыйм аңнан.

Миләшемнең,баланымның
Җимешләре бик ачы.
Түзәм ачы язмышларга,
Мине килеп син алчы.

Миләшем дә,баланым да
Ахры,мине аңлыйлар.
Сынаганда ачы язмыш,
Алар миннән калмыйлар.

Миләшемнең,баланымның
Җимешләре бик ачы.
Түзәм ачы язмышларга,
Мине килеп син алчы.

Ялгыз бакчамны бизиләр
Миләшем белән балан.
Сагышларым,саг(ы)нуларым,
Мин сезнең белән калам.

Миләшемнең,баланымның
Җимешләре бик ачы.
Түзәм ачы язмышларга,
Мине килеп син алчы.
```
