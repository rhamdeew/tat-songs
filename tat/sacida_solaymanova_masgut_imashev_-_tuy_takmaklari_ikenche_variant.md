# Оригинал

### Саҗидә Сөләйманова / Мәсгуть Имашев - Туй такмаклары (икенче вариант)

```
Аккошлар килерләр безгә,
Эзләрен күлдән эзлә.
Аккошлар юлы шикелле
Ак бәхет килсен сезгә!

Аллар үссен, гөлләр үссен
Гомер юлларыгызда.
Тагын бер биесәк иде
Алтын туйларыгызда.

Бергә яшик, әй туганнар,
Тыныч торсын дөньялар.
Килен-кияү киләчәккә
Матур тормыш коралар.

Аерылмагыз, каерылмагыз,
Пар булсын канатыгыз.
Айга түгел, елга түгел,
Гомергә яратыгыз.

Аккошлар килерләр безгә,
Эзләрен күлдән эзлә.
Аккошлар юлы шикелле
Ак бәхет килсен сезгә!
```
