# Оригинал

### Әнвәр Зарипов / Резеда Ахиярова - Кирәк түгел, дисең

```
Килер идем мин каршыңа синең,
Ачар идем күңел серләремне.
Бирер идең кабат өмет уты,
Кабат сынап ярсу хисләремне.

Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…
Яратсам да сине
Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…

Бүләк итәр идем сиңа гөлләр,
Үбәр идем, җаным, йөзләреңнән.
Сөеклем, дип атар идем сине
Күзләремне алмый күзләреңнән.

Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…
Яратсам да сине
Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…

Санар идең кичен йолдызларны,
Оча-оча хыял канатында.
Ал нурларга чумар иде дөнья
Йөрәкләрдә ялкын кабынганда.

Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…
Яратсам да сине
Кирәк түгел, дисең,
Ләкин, ләкин, ләкин…
```
