# Оригинал

### Билгесез - Нигә икән, нигә?

```
Нигә икән,нигә, җәйге аяз төндә
Сары сагыш баса җанымны?!
Кереп төшләремә, төшеп исләремә
Өзгәләрсең тагын-тагын мы?!

Мәңгегә дип сөйдем, ышанмаган идем
Аерылырбыз диеп бер заман.
Исемең телемдә,кайнар яшь күземдә,
Уйларымда бары син һаман.

Яныма син бер көн барыбер килерсең
Җаныңа табалмый тыңгылык.
Гомернең калганын җанга кушып җанны
Без яшәрбез икәү бер булып.
```
