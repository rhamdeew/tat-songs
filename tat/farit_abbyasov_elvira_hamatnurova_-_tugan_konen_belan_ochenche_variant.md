# Оригинал

### Фәрит Аббясов / Эльвира Хаматнурова - Туган көнең белән! (өченче вариант)

```
Туган көнең белән сине тәбрик итәм.
Сизә күрмә,төннең салкынын!
Гомерең буе югалтма син.
Мәхәббәтнең кайнар ялкынын.

Мәхәббәтле кеше бик тиз чәчәк ата.
Бирә аңа саулык һәм шатлык.
Яратулар тәнгә шифа бирә.
Парлы яшәү үзе зур байлык.

Яннарыңда булса сөйгән ярың.
Фәрештәләр саклар күк йөзен.
Ике йөрәк бергә тибәр әле.
Гөлләр шиңмәс өчен һәр көзен.

Кояш нурын,сөеп сипсен сиңа!
Йөзеңнән һич нурлар сүнмәсен.
Матур тормышыңда,татлы сулыш.
Бер кайчанда синнән китмәсен.
```
