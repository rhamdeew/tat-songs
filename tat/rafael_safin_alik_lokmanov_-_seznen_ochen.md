# Оригинал

### Рафаэль Сафин / Алик Локманов - Сезнең өчен

```
Сез дип талпына күңелем,
Дусларым — якыннарым!
Сезнең өчен җырлыймын мин
Җырларның асылларын.

Күкрәгемдә сезнең өчен
Балкыган кояшым бар,
Сезнең өчен иртә — кичен
Сайрап торган кошым бар.

Сез — минем җан күршеләрем,
Үз күргән кешеләрем,
Илем өчен җиргә бәхет
Үстергән кешеләрем.

Сез дип тибә бу йөрәгем,
Дусларым — якыннарым!
Сезнең өчен җырлыймын мин
Җырларның асылларын.
```
