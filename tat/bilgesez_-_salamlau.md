# Оригинал

### Билгесез - Сәламләү

```
Борынгыдан килгән хәбәр —
Исәнлекне сорашу.
Гомерлектә истә кала
Бер күрешеп очрашу.

Исәнмесез, якын дуслар,
Исәнмесез, туганым.
Ни хәлләрдә яшисез,
Кадерле тамашачым?
Исәнмесез, вәгаләйкум,
Вәгаләйкум ас-сәлам.
Рәхмәт, җыелып килгәнсез,
Барчагызга күп сәлам.

Барчагызга сәлам юллыйм,
Ничек килеп җиттегез?
Күңелгә рәхәт бирә
Сезнең нурлы йөзегез.

Исәнмесез, якын дуслар,
Исәнмесез, туганым.
Ни хәлләрдә яшисез,
Кадерле тамашачым?
Исәнмесез, вәгаләйкум,
Вәгаләйкум ас-сәлам.
Рәхмәт, җыелып килгәнсез,
Барчагызга күп сәлам.

Елмаеп сәлам бирәм
Мондагы балаларга.
Хөрмәт белән баш иям
Әби һәм бабайларга.

Исәнмесез, якын дуслар,
Исәнмесез, туганым.
Ни хәлләрдә яшисез,
Кадерле тамашачым?
Исәнмесез, вәгаләйкум,
Вәгаләйкум ас-сәлам.
Рәхмәт, җыелып килгәнсез,
Барчагызга күп сәлам.
```
