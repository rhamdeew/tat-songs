# Оригинал

### Мөнир Мазунов / Әнвәр Бакиров - Тагын язны көтеп

```
— Исәнме йөрдегез, кечкенә кошкайлар?
Кайларда гиздегез, канатлы дускайлар?

— Без очтык бергәләп табигать иркендә,
Барчабыз да исән-сау, тимәде беркем дә.

Тик китәбез инде, сезнең якта көз бит.
Синең янга килдек саубуллашырга дип.

Утыр да бакчада, тыңла җырыбызны.
Бик яратып тордык без сезнең җирегезне.

— Сайрагыз, кошкайлар, шундый моңлы итеп.
Бөреләнсен күңел, тагын язны көтеп.
```
