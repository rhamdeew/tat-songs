# Оригинал

### Айрат Суфиянов / Флүрә Талипова - Йөрәкләргә шакыйм

```
Туган телнең киләчәген уйлап,
Көлдән тудым кабат…
Кемгә кирәк йөрәгеңдә йөргән
Мәхаббәтле газап?..

Ялан тәпи салкын бозга басып,
Түнә-түнә, балкыйм…
"Ачыгыз!", дип, кешеләрнең бикле
Йөрәкләрен шакыйм, йөрәкләрен шакыйм…

Беркемгә дә беркем кирәк түгел,
Үз көчеңне җуйсаң…
Яшәү мөмкин җирдә тик елмаеп,
Әгәр ерткыч булсаң…

Ялан тәпи салкын бозга басып,
Түнә-түнә, балкыйм…
"Ачыгыз!", дип, кешеләрнең бикле
Йөрәкләрен шакыйм, йөрәкләрен шакыйм…

Чыкмаган җан, анда шайтан түгел,
Һаман өмет яши…
Шул өметне көрәш җыры итеп,
Горур кеше яши…

Ялан тәпи салкын бозга басып,
Түнә-түнә, балкыйм…
"Ачыгыз!", дип, кешеләрнең бикле
Йөрәкләрен шакыйм, йөрәкләрен шакыйм…
```
