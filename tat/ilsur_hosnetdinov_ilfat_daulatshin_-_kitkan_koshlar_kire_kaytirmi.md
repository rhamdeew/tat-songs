# Оригинал

### Илсур Хөснетдинов / Илфат Дәүләтшин - Киткән кошлар кире кайтырмы?

```
Көзге җилләр өзә үзәкләрне —
Киткән кошлар кире кайтырмы?
Бакчадагы гөлләр күптән шиңгән —
Язлар җиткәч чәчәк атырмы?

Язлар җиткәч бәлкем, бәлкем, гөлләремә
Икәү бергә сулар сибәрбез.
Җылы яктан кайткан, кайткан кошларыма
Серебезне бәлки сөйләрбез.

Моңсуланып калам кошлар киткәч —
Белеп булмый кайсы кайтасын.
Тамырлары тирән гөлләремнең —
Әйтә алмыйм чәчәк атасын.

Язлар җиткәч бәлкем, бәлкем, гөлләремә
Икәү бергә сулар сибәрбез.
Җылы яктан кайткан, кайткан кошларыма
Серебезне бәлки сөйләрбез.

Язларымны көтәм, ышанамын,
Җылы җилләр алып килсен дим.
Гөлләр чәчәк атсын, кошлар кайтсын,
Шуны теләп, янып йөрим мин.

Язлар җиткәч бәлкем, бәлкем, гөлләремә
Икәү бергә сулар сибәрбез.
Җылы яктан кайткан, кайткан кошларыма
Серебезне бәлки сөйләрбез.
```
