# Оригинал

### Гәрәй Рәхим / Рифкать Сәйфетдинов - Бәйрәм (беренче вариант)

```
Татар җыелган җирдә
Бәйрәмсез булмый бер дә,
Бәйрәм, бәйрәм.
Ике татар — бер базар,
Базарда телләр язар,
Бәйрәм, бәйрәм, бәйрәм.

Җыелышса бергә —
Татар бергә гомергә.

Җәйның матур көнендә,
Илнең бөтен җирендә
Бәйрәм, бәйрәм.
Җыелышып татарлар
Сабан туе ясарлар.
Бәйрәм, бәйрәм, бәйрәм.

Җыелышса бергә —
Татар бергә гомергә.
```
