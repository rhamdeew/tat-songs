# Оригинал

### Анна Попова / Нияз Вәлиев - Тәкыялар

```
Җәйге яшьлек аланында мин чәчкә өзгән идем,
Чәчәкләр өзгән идем дә бәхетләр көткән идем.
Җәйге яшьлек аланында мин чәчкә өзгән идем,
Сөю билгесе дип сиңа тәкыя үргән идем.

Аллы-гөлле, аллы-гөлле
Тәкыялар үрәм хисләремнән.
Беркемгә дә, беркемгә дә инде
Сине бирмәм, сине бирмәм.

Ак чәчәкләр — ак бәхеткә, аллары — саф сөюгә,
Мин сине беркемгә бирмәм, син дә мине җибәрмә.
Ак чәчәкләр — ак бәхеткә, аллары — саф сөюгә,
Чәчәкләрдәй язмышыбыз үрелде тәкыяга.

Аллы-гөлле, аллы-гөлле
Тәкыялар үрәм хисләремнән.
Беркемгә дә, беркемгә дә инде
Сине бирмәм, сине бирмәм.

Чәчәкләрнең көне кыска, безнең сөю — гомерлек,
Тәкыя бәйләме сулса, сулмас хисләр гөлләре.
Чәчәкләрнең көне кыска, безнең сөю — гомерлек,
Беркайчан онытма шушы вәгъдә биргән мизгелне.

Аллы-гөлле, аллы-гөлле
Тәкыялар үрәм хисләремнән.
Беркемгә дә, беркемгә дә инде
Сине бирмәм, сине бирмәм.
```
