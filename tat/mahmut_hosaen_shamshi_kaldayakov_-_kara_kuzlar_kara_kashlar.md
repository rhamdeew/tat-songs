# Оригинал

### Мәхмүт Хөсәен / Шәмши Калдаяков - Кара күзләр, кара кашлар

```
Әйтче, иркәм, сөйгән гөлем:
Нигә икән сине күрдем?
Нигә икән өзелеп сөйдем?

Кара күзләр,
Кара кашлар.

Чын сөюнең сере тирән…
Газап икән сөю дигән.
Нигә сине сөйдем, иркәм?!

Кара күзләр,
Кара кашлар.

Алар ялкын-ут чәчәләр,
Сүзсез генә сөйләшәләр,
Миңа бәхет өләшәләр —

Кара күзләр,
Кара кашлар.
```
