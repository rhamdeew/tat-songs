# Оригинал

### Разил Вәлиев / Мирсәет Яруллин - Кызлар җырлый

```
Кызлар җырлый, кызлар җырлый,
Кызлар җыры — сагышлы;
Кызлар җырына җыелган
Язмышларның ялгышы.

Кызлар җырын тыңлаганда
Җан җылылары алам.
Җир үзенең сөенечен,
Бәхетен һәм көенечен
Кызлар җырына салган.

Кызлар җырлый, кызлар җырлый,
Кызлар җыры — канатлы.
Кызлар җыры күңелдәге
Сагышларны таратты.

Кызлар җырын тыңлаганда
Җан җылылары алам.
Җир үзенең сөенечен,
Бәхетен һәм көенечен
Кызлар җырына салган.

Кызлар җырлый, кызлар җырлый,
Кызлар җыры — кояшлы.
Кызлар җырында — шатлыгым,
Сагышларымның башы.

Кызлар җырын тыңлаганда
Җан җылылары алам.
Җир үзенең сөенечен,
Бәхетен һәм көенечен
Кызлар җырына салган.
```
