# Оригинал

### Гөлфия Юнысова / Нурия Абдуллина - Дим буйлары

```
Туган ягым Дим буйлары,
Үзәнеңдә әрәмә.
Әрәмәгә барсам, әнкәй,
Озакласам, әрләмә.

Әрәмәдә пешкән диләр
Шомырт, балан, бөрлегән.
Яшь кояшлы булгангадыр,
Җимеш иртә өлгергән.

Чакыра төсле үзенә
Куак артыннан куак.
Әйтерсең дә, бала чагым
Яшенеп тора сымак.
```
