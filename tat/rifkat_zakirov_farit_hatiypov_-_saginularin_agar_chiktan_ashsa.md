# Оригинал

### Рифкать Закиров / Фәрит Хатыйпов - Сагынуларың әгәр чиктән ашса

```
Аңламадың микән мине чынлап,
Сер бирмәдең микән аңлап та…
Син киткәннән бирле шуны уйлап,
Ямансулап йөрим аулакта.

Иртәсен дә киләм, кич тә киләм:
Яр буенда тере шәүлә мин.
Үзәгем өзелеп сагынудан —
Сикереп суга төшәр хәлдә мин!..

Сагынуларың иркәм, чиктән ашса,
Суга түгел, утка керерсең…
Хәлләремне… Күктән суга төшкән
Ай йөзенә карап белерсең…
```
