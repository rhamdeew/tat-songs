# Оригинал

### Рәис Гыймадиев / Фирзәр Мортазин - Моңлы язмышым

```
Киләчәккә уйларым бар, якты уйларым.
Сезнең белән үтсә иде гомер юлларым.

Сезнең белән бәйләдем моңлы язмышны,
Яннарымда сез булганга сөям тормышны.

Күңелемдә сагышым бар, моңлы сагышым.
Мизгеленә кәдәр тоям вакыт агышын.

Сезнең белән бәйләдем моңлы язмышны,
Күңелемдә җыр булганга сөям тормышны.

Йөрәгемдә җырларым бар, туган җырларым.
Насыйп булса, сезгә тагын, тагын җырлармын.

Сезнең белән бәйләдем моңлы язмышны,
Йорәгемдә җыр булганга сөям тормышны.
```
