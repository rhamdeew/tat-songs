# Оригинал

### Алмаз Хәмзин / Фирзәр Мортазин - Казанда торсаң иде

```
Таулы-таулы тирәләрең
Күкләргә тияр сыман.
Таллы-таллы тирәкләрең,
Таллы-таллы тирәкләрең
Күлләргә кереп чумган.

Сине миңа бәйли торган
Бауларың булса иде.
Дус-ишләрем, сезнең белән,
Җан дусларым, сезнең белән
Казанда торсаң иде.

Ярсу-ярсу йөрәкләрнең
Күбесе синдә туган.
Моңсу-моңсу күңелләрнең,
Моңсу-моңсу күңелләрнең
Хәсрәтен шатлык куган.

Сине миңа бәйли торган
Бауларың булса иде.
Дус-ишләрем, сезнең белән,
Җан дусларым, сезнең белән
Казанда торсаң иде.

Янып-көеп яшик әле,
Эх, гузәл туган якта.
Уйнап-биеп җырлыйк әле,
Уйнап-биеп җырлыйк әле,
Казаным, синең хакта.

Сине миңа бәйли торган
Бауларың булса иде.
Дус-ишләрем, сезнең белән,
Җан дусларым, сезнең белән
Казанда торсаң иде.
```
