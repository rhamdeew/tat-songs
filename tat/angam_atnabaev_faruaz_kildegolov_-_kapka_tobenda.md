# Оригинал

### Әнгәм Атнабаев / Фәрүәз Килдеголов - Капка төбендә

```
Авыл өстен тынлык каплагач,
Карт-корылар ятып йоклагач,
Йөрүчеләр беткәч күз атып,
Егет килде кызын озатып.

Кабатлана шулай көн саен
Яшьлек дигән нәрсә, мөгаен.
Йокысыз һәм айлы төн белән
Башланадыр капка төбеннән.
Онытмам, онытмам
Яшьлегем  урамнарын.

Туктадылар капка төбенә,
Кыз ашыккан булды өенә.
Егет аны кыстап туктатты,
Җылы сүзләр әйтеп юатты.

Кабатлана шулай көн саен
Яшьлек дигән нәрсә, мөгаен.
Йокысыз һәм айлы төн белән
Башланадыр капка төбеннән.
Онытмам, онытмам
Яшьлегем  урамнарын.

Сүзләр бетми йөрәк түрендә,
Алар һаман капка төбендә.
Алар сүзне ҡабат яңыртты,
Ул арада инде таң атты.

Кабатлана шулай көн саен
Яшьлек дигән нәрсә, мөгаен.
Йокысыз һәм айлы төн белән
Башланадыр капка төбеннән.
Онытмам, онытмам
Яшьлегем  урамнарын.
```
