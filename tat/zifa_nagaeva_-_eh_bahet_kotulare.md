# Оригинал

### Зифа Нагаева - Эх, бәхет көтүләре

```
Кояш чыгу беләнмени
Җанда болытлар булгач.
Ярың булу беләнмени
Күңелең тулы зар булгач.

Болыт өсләрендә болыт
Яуды ни, яумады ни.
Үкенерлек ярлар түгел,
Калды ни, калмады ни.

Болыт өсләрендә болыт,
Явармы, яумас микән.
Үзебезгә насыйп ярлар
Язармы, язмас микән.

Болыт өсләрендә болыт,
Яуса ни, яумаса ни.
Яр булып та, пар булмаса
Язса ни, язмаса ни.

Болытлар китәр әле дә,
Явымнар үтәр әле.
Кешеләргә килгән бәхет
Безгә дә җитәр әле.

Болытның китүләре,
Эх, бәхет көтүләре.
Көтүләре бер хәл юк, жәл,
Гомернең үтүләре.
```
