# Оригинал

### Ризида Мөхәммәтҗанова / Владислав Шәрифуллин - Табам!

```
Яңгыр түгел, чишмә түгел,
Давыл җырын сайладым
Йөрәгем нык, беләктә көч
Артканнан арткан саен.

Кушымта:
Еллар үтте, көчем бетте,
Давыл екты аяктан.
Чишмә җыры челтер-челтер
Ишетелә аръяктан.

Көчем туплап, калкынамын,
Барам чишмәгә таба.
Үземә кирәк җырларны
Саф судан эзләп табам.

Кушымта.

Яңгыр яусын, чишмә аксын,
Уйдан куам давылны.
Киләчәккә өметем зур
Якын дуслар табылды.

Кушымта.
```
