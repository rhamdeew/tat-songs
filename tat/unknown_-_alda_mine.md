# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Алда мине

```
Күңел өчен генә булсын, әйдә,
Алда мине, юри, сөям, диген,
Җаннарымда давыл басылсын да
Кояш нуры күмсен күңел күген.

Алда мине, әйдә, алда мине,
Алда мине, килә сөясем.
Алда мине, килә алданасым,
Син кабызган утта көясем.

Әйдә, бетсен, янсын дөрләп йөрәк,
Мәхәббәтсез аңа нәрсә кирәк…
Сөюләрдә адашасым килә,
Гел янасым килә, тик шул теләк.

Алда мине, әйдә, алда мине,
Алда мине, килә сөясем.
Алда мине, килә алданасым,
Син кабызган утта көясем.
```
