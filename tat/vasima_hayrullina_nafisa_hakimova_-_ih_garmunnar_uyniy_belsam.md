# Оригинал

### Вәсимә Хәйруллина / Нәфисә Хәкимова - Их, гармуннар уйный белсәм

```
Их, гармуннар уйный белсәм,
Моңлы җыр сузар идем.
Моңсызларга моңнар биреп,
Бер җырлап узар идем.

Их, гармуннар уйный белсәм,
Телләрен басар идем.
Яшь чакларны искә алып,
Кич уен ясар идем.

Их, гармуннар уйный белсәм,
Уйнап сыздырыр идем.
Биючеләрнең табанын
Уттай кыздырыр идем.

Их, гармуннар уйный белсәм,
Сезне җырлатыр идем.
Гамьсезләргә гамьнәр биреп,
Их, бер елатыр идем.
```
