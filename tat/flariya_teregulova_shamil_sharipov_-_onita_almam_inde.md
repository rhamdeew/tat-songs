# Оригинал

### Флария Терегулова / Шамил Шәрипов - Оныта алмам инде

```
Оныта дигән идем,
Уйламыйм дигән идем,
Күзләреңнең карашларын
Хәтердэн җуйган идем

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.

Шомыртлар ап-ак чәчәктә,
Сагышларым яңара.
Йөрәгемнең яралары
Язлар җиткәч яңара.

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.

Тимим, калсын сагышым,
Язылган ул язмышым.
Еллар үтә, арта моңнар,
Гомер үтеп барышы.

Үзгәртә алмам инде,
Оныта алмам инде.
Мөлдерәмә моңнарымнан
Арына алмам инде.
```
