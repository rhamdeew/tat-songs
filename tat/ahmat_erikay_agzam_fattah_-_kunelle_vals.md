# Оригинал

### Әхмәт Ерикәй / Әгъзам Фәттах - Күңелле вальс

```
Тәрәзәдәге утлар
Ерактан балкып тора,
Музыкалы, якты клуб
Күңелне тартып тора.

Залда вальс өермәсе,
Егетләр бии, кызлар; —
Таралып тирә-якка,
Тирбәлә талгын моңнар.

Вальс моңы, уен, көлке —
Клубның киң залларында;
Күңелләр күтәрелә,
Яшьлек канатларында,

Биегез, әйләнегез
Музыка уйнаганда;
Ямьләнә бәйрәмебез
Шат дуслар җыелганда.

Җырлыйбыз хозурланып,
Эшлибез горурланып;
Яшь буын алга бара,
Рухланып, матурланып.
```
