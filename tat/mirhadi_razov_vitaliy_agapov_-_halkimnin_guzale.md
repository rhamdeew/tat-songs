# Оригинал

### Мирһади Разов / Виталий Агапов - Халкымның гүзәле

```
Чигүле алъяпкычларың
Бик килешә үзеңә.
Мәрҗән муенсаларыңнан
Нур сибелә йөзеңә.

Әй, гүзәлләрнең гүзәле,
Үзәкләрне өзмәле!
Килдем әле сиңа, иркәм,
Килдем әле, түзмәдем.

Кашларыңны кыйгач итеп
Карлыгачлар сызгандыр.
Ирен өстеңдә нәни миң —
Таңнар үбеп узгандыр.

Әй, гүзәлләрнең гүзәле,
Үзәкләрне өзмәле!
Килдем әле сиңа, иркәм,
Килдем әле, түзмәдем.

Елмаюлы карашларың
Әллә мине күзләде.
Эштә уңган, төскә булган,
Әй, халкымның гүзәле.

Әй, гүзәлләрнең гүзәле,
Үзәкләрне өзмәле!
Килдем әле сиңа, иркәм,
Килдем әле, түзмәдем.
```
