# Оригинал

### Рәис Гыймадиев / Виталий Агапов - Көзге туйлар

```
Кошлар сайраудан туктагач,
Алтын көзләр дә җитә.
Якынайган көзге туйлар,
Кыз күңелен җилкетә.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр…

Шаулатып көзге авылны,
Каз өмәләре үтте.
Көмеш кыңгырау чыңлатып,
Кавышыр көннәр җитте.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр…

Ак мунчаның идәненә
Каз мамыгы сибелгән.
Өмәче кыз киләчәге
Яшь егеткә бәйләнгән.

Вәгъдәләр инде бирешкән,
Алынылган бирнәләр.
Кызлы өйләр тирәсендә
Яучылар күренәләр…
```
