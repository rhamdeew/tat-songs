# Оригинал

### Равил Фәйзуллин / Шамил Шәрифуллин - Якты моң (беренче вариант)

```
Күчә җилдә ак комнар,
Ярга кага дулкыннар.
Акчарлаклар Иделдә,
Туган илем күңелдә.

Чык ялтырый гөлләрдә,
Аяз таңлы көннәрдә.
Алтын кояш күгемдә,
Газиз әнкәм күңелдә.

Җилкәннәргә җил бәрә,
Ярда рәшә тирбәлә.
Рәхәт туган илемдә,
Сөйгән ярым күңелдә.
```
