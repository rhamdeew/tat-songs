# Оригинал

### Зарема Әюпова / Ринат Гыйззәтуллин - Әнкәй гөле (икенче вариант)

```
Иркәләп һәм сөеп кенә
Үстерә әнкәй гөлләр.
Әнкәй догасыннан тора
Алдагы безнең көннәр.

Туган йорттан киткәч ерак,
Булсын дип истәлеге,
Бер сабагын алдым өзеп,
Әнкәй синең гөлеңнең.

Араларны якынайтып
Гөлең гел чәчәк ата.
Күзләремә килә яшьләр
Уйласам синең хакта.

Туган йорттан киткәч ерак,
Булсын дип истәлеге,
Бер сабагын алдым өзеп,
Әнкәй синең гөлеңнең.

Әнкәй гөлләрдәй саф,
Баладай киң күңелең.
Беләсең син балакаем
Бик якын да түгелем.

Әнкәемнең үсә гөле
Ямь биреп ерак җирдә.
Сагынганда көч аламын,
Карыйм да синең гөлгә.
```
