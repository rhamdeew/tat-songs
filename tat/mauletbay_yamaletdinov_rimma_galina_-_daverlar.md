# Оригинал

### Мәүлетбай Ямалетдинов / Римма Галина - Дәверләр

```
Яшьлек бит ул — йөгәнсез ат,
Башын бирми дә куя.
Шул дәвер була бер кабат,
Шул дәвер була бер кабат,
Үтсә, килми дә куя.

Картлык — карт ат, туйда җилми,
Җилсә, үзмый да куя.
Картлык синнән сорап килми,
Картлык синнән (миннән) сорап килми,
Килсә, китми дә куя.
```
