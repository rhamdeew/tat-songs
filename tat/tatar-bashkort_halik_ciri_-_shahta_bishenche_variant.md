# Оригинал

### Татар-Башкорт халык җыры - Шахта (бишенче вариант)

```
Китә казлар, китә казлар,
Китә казлар Донбасска шул;
Миндә уйлап торам әле
Шул казлардан калмаска.

Сәхрәләрдә бер сызгырдым,
Печәннәрем кипсен дип, шул.
Безгә кемнәр теләде икән
Гомерләре аерым үтсен, дип?

Урамнардан урамнарга
Кемнәр күпер салганнар шул?
Без аерылдык та сайргайдык
Саргаймасын калганнар.
```
