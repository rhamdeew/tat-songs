# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Инәки чишмәсе

```
Кендек каны бәйли икән
Иң якын туганнарны…
«Туганым!» — дип өзелеп торган,
Җан атып торганнарны.

Туган якта — изге якта
«Инәки» чишмәсе бар!
Изге чишмә суын эчеп
Шәп яшик без, туганнар!

Чишмәләр бергә кушылса,
Диңгезләр, күлләр була.
Туган «Туган!» дип яшәсә,
Һәр көнең гөлләр була.

Туган якта — изге якта
«Инәки» чишмәсе бар!
Изге чишмә суын эчеп
Шәп яшик без, туганнар!

Туган кадерен туган белсен —
Гомерләр кунак кына!
Яшәү чишмә булып җырлар —
Гел бергә булсак кына.

Туган якта — изге якта
«Инәки» чишмәсе бар!
Изге чишмә суын эчеп
Шәп яшик без, туганнар!
```
