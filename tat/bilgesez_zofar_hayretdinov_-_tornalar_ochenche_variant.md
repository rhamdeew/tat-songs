# Оригинал

### Билгесез / Зөфәр Хәйретдинов - Торналар (өченче вариант)

```
Торналар оча көньякка,
Тасма булып төзелеп.
Күк йөзеннән сибелә кебек,
Сагышлы моң төзелеп.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.

Сокланам да, көнләшәм дә
Сезгә карап торналар.
Бәхетлеләр сезнең кебек
Һәрчак бергәдер алар.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.

Торналар биек очалар,
Кайталмаслар тиз генә.
Түзәр алар күрешмичә
Түзә алмам шул мин генә.

Сагышланмагыз, кошларым,
Сез бит бергә, бер төркем.
Ятим җанны, ялгыз җанны
Юата алмас беркем.
```
