# Оригинал

### Роберт Миңнуллин / Фәнир Галимов - Әбиләр чуагы (өченче вариант)

```
Күренеп күзләргә
Табигать үзгәрә.
Калдырып яшьлекне,
Керәбез көзләргә.

Хисләрем чуалды,
Керфекләр чыланды.
Чылатты шул аны
Әбиләр чуагы…

Әллә көз килгәнгә,
Яшь килә күзләргә.
Язларны, җәйләрне
Кайлардан эзләргә?

Хисләрем чуалды,
Керфекләр чыланды.
Чылатты шул аны
Әбиләр чуагы…

Киләдер көзләр дә
Үзәкне өзәргә.
Язларны, җәйләрне
Кайлардан эзләргә?

Хисләрем чуалды,
Керфекләр чыланды.
Чылатты шул аны
Әбиләр чуагы…
```
