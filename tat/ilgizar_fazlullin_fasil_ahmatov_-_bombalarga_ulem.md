# Оригинал

### Илгизәр Фазлуллин / Фасил Әхмәтов - Бомбаларга — үлем!

```
Җир шарына афәт яный
Без яшәгән шомлы көннәрдә.
Кояш үзе атом бомбасыннан
Кача кебек кара төннәргә.

Кушымта:
Җирдә, суда, ерак йолдызларда
Юк сиңа юл, каһәрләнгән атом!
Туган илнең, бөтен җир шарының
Тынчлыгын тели минем халкым.

Бишектәге нәниләрнең
Яннарында булсын әнкәсе.
Белеп булсын елмаюларыннан
Алар безгә нәрсә әйтәсен.

Онытмыйбыз кырык берне,
Тизләтә ул йөрәк тибешен.
Үлем алып килсен бомбаларга
Һәр адымың, хезмәт җимешең!
```
