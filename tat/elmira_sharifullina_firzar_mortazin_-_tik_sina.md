# Оригинал

### Эльмира Шәрифуллина / Фирзәр Мортазин - Тик сиңа

```
Давылларны узып,
Йолаларны бозып,
Мин сиңа, мин сиңа киләмен.
Җанымның бар назың,
Кайгымның бик азын
Тик сиңа, тик сиңа бирәмен.

Кояштан да көчле,
Моңнардан да хисле
Йөрәгем утына төрәмен.
Җанымның бар назын,
Кайгымның бик азын
Тик сиңа, тик сиңа бирәмен.

Төшеңә керәмен,
Мин сиңа киләмен,
Бәхетне бер синдә күрәмен.
Кайгымның бик азын,
Җанымның бар назын
Тик сиңа, тик сиңа бирәмен.
```
