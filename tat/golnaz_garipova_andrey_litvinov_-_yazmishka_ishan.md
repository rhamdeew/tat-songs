# Оригинал

### Гөлназ Гарипова / Андрей Литвинов - Язмышка ышан

```
Күкләрдән әзли hәр кызый
Җирдән көтәсен.
Яшьлек шундый җүләр инде
Нихәл итәсен.
Күкләрдән күзли hәр кызый
Җирдән табасын, тик табалсын!
Очып кына үтә күңел
Җир-күк арасын!

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара —
Юллар тоташып бара…
Юллар тоташып бара…

Беркайчан да ашыктырма
Килер язларны.
Таш яуса да сиңа булыр
Сиңа язганы.
Язларыңны ашыктырма,
Язмышка ышан, ышан гына!
Көтеп алган бәхет кенә
Кадерле була.

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара —
Юллар тоташып бара…
Юллар тоташып бара…

Күпме йөкләр күтәрсәң дә
Арымас иңең.
Мин — җиңеллек китерүче
Бәхетең синең.
Күпме юллар үтелсә дә
Язмышка ышан, ышан гына!
Гомер дигән тирән савыт
Тамчылап тула.

Алып бар мине хыялыма,
Күтәр дә көчле кулларыңа.
Сорама берни, миңа кара —
Юллар тоташып бара…
Юллар тоташып бара…
```
