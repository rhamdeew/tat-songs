# Оригинал

### Рөстәм Зәкуан / Альфред Якшимбетов - Китмә әле

```
Китәм дисең, китмә әле,
Син сагышлы итмә әле.
Безнең бит алда ярсу язлар,
Сөю, назлар көтә әле.

Китмә әле, китмә әле,
Җанны ятим итмә әле.
Янып торган учакларны
Сүндерергә иртә әле.

Бергә булыйк, сизәрсең син
Минем моңсу күзләремне.
Китмә әле, мин бит яратам
Синең серле сүзләреңне.

Китмә әле, китмә әле,
Җанны ятим итмә әле.
Янып торган учакларны
Сүндерергә иртә әле.

Синдә яшәвем шатлыгы,
Якты нуры көннәремнең.
Китмә әле, син китә калсаң —
Мин дә сиңа иярермен.

Китмә әле, китмә әле,
Җанны ятим итмә әле.
Янып торган учакларны
Сүндерергә иртә әле.
```
