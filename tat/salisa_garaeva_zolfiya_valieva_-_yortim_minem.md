# Оригинал

### Салисә Гәрәева / Зөлфия Вәлиева - Йортым минем

```
Алтын куллы авылдашлар ашыгалар,
Инде әзер, бизәп куйган кашагалар.
Баскычлары иркен, текә, биек,
Кызым көлә: «Монда ничек меним диеп?»

Бүген иртүк таңнан торып
Йорт салырга керештем мин өмә корып.
Йортым минем иркен булсын,
Авылга ямь өстәп, балкып торсын.

Йомычкалар уем-уем, бөдрә-бөдрә,
Сары сагыз кунган нарат хуш ис бирә.
Йомычкалар җыеп, җирдә уйный балам,
Җырлый-җырлый аның өчен мин йорт салам.

Бүген иртүк таңнан торып
Йорт салырга керештем мин өмә корып.
Йортым минем иркен булсын,
Авылга ямь өстәп, балкып торсын.

Кояш тулсын яңа йортка, бәхет иңсен,
Бүрәнәләр аңкытсыннар нарат исен!
Бала-чагам уйнап йөрсен, өем тулып,
Гөрләп яшик, илләр-еллар имин булып!

Бүген иртүк таңнан торып
Йорт салырга керештем мин өмә корып.
Йортым минем иркен булсын,
Авылга ямь өстәп, балкып торсын.
```
