# Оригинал

### Гөлүсә Шаһбан / Алия Исрафилова - Хатлар яздың

```
Хатлар яздың миңа…
Сыздың, боздың…
Гомереңнең бик кечкенә, гомернең бер өлешен….
Син барыбер, син барыбер минем белән уздың…

Сөю бүләк иттең миңа,
Әллә чын, әллә ялган…
Гомереңнең якты бер мизгеле
Минем янда калган…

Җырлар суздың миңа
Иртә-кичен…
Син сизмәстән гомереңнең, гомернең бер кисәген
Багышладың, багышладың бары минем өчен…

Сөю бүләк иттең миңа,
Әллә чын, әллә ялган…
Гомереңнең якты бер мизгеле
Минем янда калган…

Хатлар яздың миңа…
Кабат яздың…
Гомереңнең бик кечкенә, гомернең бер өлешен
Син барыбер минем белән, минем белән уздың…

Сөю бүләк иттең миңа,
Әллә чын, әллә ялган…
Гомереңнең якты бер мизгеле
Минем янда калган…
```
