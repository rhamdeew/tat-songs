# Оригинал

### Айгөл Вәлиуллина / Хәмид Мәткәримов - Бәхет гөле

```
Өебезнең түрләренә
Утырттым бәхет гөле
Сөенечемнән куанып,
Ашкынып йөргән көнен.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.

Бәхетлеләрнең бәхете
Гөлгә дә күчә икән.
Чәчкәләргә бәхет өстәп
Яшәргә язсын, иркәм.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.

Гөлебезнең хуш исеннән
Айнымый йөргән чагым.
Ялкынына сөю кушып,
Саклыйк гаилә учагын.

Бәхет гөле бүләк иттем
Сөекле сөйгәнемә.
Бәхет гөле чәчәк ата
Бәхетле өйдә генә.
```
