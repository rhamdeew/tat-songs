# Оригинал

### Илдар Юзеев / Фирзәр Мортазин - Хуш инде

```
Хуш инде, мәңгегә
Хуш инде.
Китә яз, китә наз —
Кыш инде.
Тагын бер текәлеп
Карыймчы:
Рәсемеңне күңелемә
Алыймчы.

Ул китә, мәңгегә
Дип китә.
Соң килде мәхәббәт, —
Ник китә?
Чаналар кузгалып
Киттеләр,
Йөрәкне өздереп
Үттеләр.

Эзләрен кар-буран
Сылады.
Юлларга җил ятып
Елады.
Хуш инде, мәңгегә
Хуш инде.
Үтте яз, буранлы
Кыш инде.
```
