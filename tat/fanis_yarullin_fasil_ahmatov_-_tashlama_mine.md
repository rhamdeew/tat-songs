# Оригинал

### Фәнис Яруллин / Фасил Әхмәтов - Ташлама мине

```
Ташлама мине, ташлама,
Рәхәт син барны тою,
Актык көннәремә чаклы
Ташлама мине, Сөю!

Ташлама мине, ташлама,
Гел минем җанда уян,
Соңгы сулышыма кадәр
Ташлама мине, Хыял!

Ташлама мине, ташлама,
Гел тор сафлыгың чәчеп,
Йөрәгем соң кат типкәнче
Ташлама мине, Яшьлек!
```
