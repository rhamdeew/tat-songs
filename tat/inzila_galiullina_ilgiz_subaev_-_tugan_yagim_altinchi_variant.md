# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Туган ягым (алтынчы вариант)

```
Туган ягым, җанга якын
Тузанлы урамнарың.
Колач җитмәс киңлекләргә
Жәелгән урманнарың.

Кушымта:
Киң болыннар, шау чәчәкле
Җиләкле аланнарың.
Сандугач моңнары тулган
Сызлып аткан таннарың.

Туган ягым — мәгърур таулар,
Челтери чишмәләре.
Хисләрдән ташкан күңелем
Җырларын көйли әле.

Кушымта:

Балачагым сукмаклары,
Яшьлегем иртәләре.
Сиңа булган хисләр якты —
Йөрәктә йөртәм әле.

Кушымта:
```
