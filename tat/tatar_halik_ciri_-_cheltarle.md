# Оригинал

### Татар халык җыры - Челтәрле

```
Ак алъяпкыч челтәрле,
Илдә матурлар бетәрме?
Илдә матурлар бетәрме,
Җан сөйгәнгә җитәрме?

Челтәр элдем читәнгә,
Җилфер-җилфер итәргә;
Без килмәдек буш китәргә,
Килдек алып китәргә.
Алын алырсыз микән,
Гөлен алырсыз микән?
Урталарга чыгып, сайлап,
Кемне алырсыз икән?
Алларын да алырбыз,
Гөлләрен дә алырбыз;
Күңелебезгә кем ошаса
Шуны сайлап алырбыз.

Урамнан узып бара
Кашыннан күзе кара.
Керфегеннән гөлләр тама
Күрсәм йөрәгем яна.

Челтәр элдем читәнгә,
Җилфер-җилфер итәргә;
Без килмәдек буш китәргә,
Килдек алып китәргә.
Алын алырсыз микән,
Гөлен алырсыз микән?
Урталарга чыгып, сайлап,
Кемне алырсыз икән?
Алларын да алырбыз,
Гөлләрен дә алырбыз;
Күңелебезгә кем ошаса
Шуны сайлап алырбыз.

Аллы ситса күлмәк кияр,
Итәге җиргә тияр.
Үзе сөйләр, үзе көлер —
Ул ярлар кемгә тияр?

Челтәр элдем читәнгә,
Җилфер-җилфер итәргә;
Без килмәдек буш китәргә,
Килдек алып китәргә.
Алын алырсыз микән,
Гөлен алырсыз микән?
Урталарга чыгып, сайлап,
Кемне алырсыз икән?
Алларын да алырбыз,
Гөлләрен дә алырбыз;
Күңелебезгә кем ошаса
Шуны сайлап алырбыз.
```
