# Оригинал

### Алсу Васикова / Алия Ганиева - Ак бәхетләр телим сезгә

```
Бүген сезнең туй көнегез,
Кавыштыгыз гомергә.
Ялкынлы мәхаббәтегез
Нур сибә тирә-юньгә.

Ак бәхетләр телим сезгә,
Котлы булсын туегыз.
Олы-олы мәхаббәттә үтсен
Бергә барган юлыгыз.

Балдай татлы мәхаббәтне
Ачы сүздән саклагыз.
Бер җан булып олы юлдан
Һәрчак бергә атлагыз.

Ак бәхетләр телим сезгә,
Котлы булсын туегыз.
Олы-олы мәхаббәттә үтсен
Бергә барган юлыгыз.

Бергә корган оягызга
Салкын җилләр кермәсен.
Каракоштай сез икегез,
Күзләр тия күрмәсен.

Ак бәхетләр телим сезгә,
Котлы булсын туегыз.
Олы-олы мәхаббәттә үтсен
Бергә барган юлыгыз.
```
