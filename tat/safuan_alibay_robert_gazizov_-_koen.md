# Оригинал

### Сафуан Әлибай / Роберт Газизов - Коен

```
Ярсып-ярсып, өерелеп,
Коен уйный, коен.
Калсаң коен уртасында,
Кыен икән, кыен.

Юлларыма гөлләр үскән,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Моңаямын шулай?

Шомнар салып, кемдер ярсып
Үтте сымак җырлап.
Уйларымны өерелтеп
Алып китте ерак.

Юлларыма гөлләр шыткан,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Ямь тапмыймын шулай?

Күңелемдә өерелеп
Коен уйный, коен.
Тыныр иде, син янда юк / Тындырырга, син янда юк, /
Кыен миңа, кыен.

Юлларыма гөлләр үскән,
Кошлар моңы чыңлый.
Тик мин генә нигә әле
Ямь тапмыймын шулай?
```
