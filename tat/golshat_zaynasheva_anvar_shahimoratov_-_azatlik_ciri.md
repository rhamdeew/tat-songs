# Оригинал

### Гөлшат Зәйнәшева / Әнвәр Шаһиморатов - Азатлык җыры

```
Ирек, ирек диеп аккан Идел
Халкым өчен ирек теләгән.
Дулкыннары мәңге, мәңге бер көй:
«Ирек, азатлык!» — дип көйләгән.

Газиз халкым ирек яулый бүген,
Иркен сулыш алсын Идел дә.
Яңа тормыш, иркен, азат тормыш
Төзик бүген туган җиремдә!

Тарих шаһит: без бу җиргә хуҗа,
Бик тирәннән кило тамырлар.
Хезмәт, эш яраткан, дуслык сөйгән
Болгар бабаларның даны бар.

Газиз халкым ирек яулый бүген,
Иркен сулыш алсын Идел дә.
Яңа тормыш, иркен, азат тормыш
Төзик бүген туган җиремдә!

Татарстан жире — байлык иле.
Түкми-чәчми яши белгәндә.
Күңелләр киң, хуҗа булып түгел,
Кунак булып дуслар килгәндә.

Газиз халкым ирек яулый бүген,
Иркен сулыш алсын Идел дә.
Яңа тормыш, иркен, азат тормыш
Төзик бүген туган җиремдә!
```
