# Оригинал

### Фәнәвис Дәүләтбаев / Рәзидә Әхмәтова - Синсез нишләрмен?

```
Яратам дип, иркәм, әйтмәсәм дә
Аңларсыңдыр, беләм уеңны.
Җан тилмереп көткән тансык сүзне
Әйтү әллә дисең кыенмы.

Йөрәк хисен туры белдерергә
Син ашыкма, әле иртәрәк.
Яшәгән чак сөю дөньясында
Уем белән сине иркәләп.

Хыялымда тыңлап кинәнергә
Очар кошым син бит биектә.
Белче иркәм, һич тә туймамындыр
Гомер буе сине сөеп тә.
```
