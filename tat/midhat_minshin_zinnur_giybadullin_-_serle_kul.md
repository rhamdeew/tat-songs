# Оригинал

### Мидхәт Миншин / Зиннур Гыйбадуллин - Серле күл

```
Бәбкәләр күк тезелешеп,
Барабыз күлгә,
Уйнап-көлеп су керәбез,
Кояшлы көндә.

Күлгә иелгән таллар, камышлар,
Тарта күңелне серле тавышлар.
Судан чыккач апа безгә
Әкият сөйли.

Нигә икән Су анасы
Үзе дә килми?..
Әй, бик матурдыр алтын тарагы,
Күрсәң иде бер чәчен таравын.
```
