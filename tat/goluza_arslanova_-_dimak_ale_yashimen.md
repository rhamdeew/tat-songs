# Оригинал

### Гөлүзә Арсланова - Димәк әле яшимен

```
Алсу таңның нурларына,
Көлеп уянам икән.
Кояш мине каршы ала,
Әле яшимен икән.

Кояшны болыт капласа,
Яшәү авыр димимен.
Әгәр мин җырлыймын икән,
Димәк әле яшимен.

Сагышларым күңелемдә,
Моң булып чыга икән.
Күз яшләренә әйләнсә,
Әле яшимен икән.

Кайгыларда, елаганда,
Яшәү авыр димимен.
Әгәр мин җырлыймын икән,
Димәк әле яшимен.

Якыннарым иң якында,
Еракларым — еракта.
Ерактагы — якыннарым,
Җырларым сенең хакта.

Туганнар ерак булсада,
Яшәү авыр димимен.
Әгәр мин җырлыймын икән,
Димәк әле яшимен.
```
