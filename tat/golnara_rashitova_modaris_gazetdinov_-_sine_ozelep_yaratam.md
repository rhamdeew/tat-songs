# Оригинал

### Гөлнара Рәшитова / Мөдәрис Газетдинов - Сине өзелеп яратам

```
Өмет өзми, көтеп алган
Син бит олы табышым!
Синең белән, бәгырькәем,
Тулыланды тормышым.

Дөньяларны яңгыратам,
Шатлыгымны таратам.
Яшермимен беркемнән дә:
Сине өзелеп яратам!

Үткәннәр үткәндә калды,
Ә син — бүгенге көнем!
Мин көткәннән дә яхшырак
Син — янымдагы өнем!

Үзеңә тиңнәрне сайлау
Түгел шул ул җиңелдән.
Бәхет икән «яратам» дип,
Әйтү ихлас күңелдән.
```
