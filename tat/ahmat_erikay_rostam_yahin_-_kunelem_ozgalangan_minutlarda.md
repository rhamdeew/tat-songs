# Оригинал

### Әхмәт Ерикәй / Рөстәм Яхин - Күңелем өзгәләнгән минутларда

```
Күңелем өзгәләнгән минутларда,
Эштән бушаганда кичләрен,
Күзләремә тагын очрамасаң,
Син булмасаң, бәгърем, нишләрмен?

Истән чыкмый синең һәрбер сүзең,
Күз алдымнан китми төсләрең.
Салкын җилләр керсә арабызга,
Син ташласаң мине нишләрмен?

Сүрелерлек түгел мәхәббәтем,
Суынырлык түгел хисләрем,
Гомерем буйларына газаплансам,
Оныталмасам сине нишләрмен?
```
