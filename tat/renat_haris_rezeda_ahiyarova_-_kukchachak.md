# Оригинал

### Ренат Харис / Резеда Ахиярова - Күкчәчәк

```
Зәңгәр-зәңгәр ким иде,
Зәңгәр-зәңгәр кар иде;
Зәңгәрлеккә уралган
Әйтер сүзем бар иде.

Зәңгәр кичне төн алды,
Зәңгәр төнне көн алды;
Зәңгәр карлар эчендә
Әйтәсе сүзем калды.

Зәңгәр язлар җитәчәк,
Зәңгәр карлар китәчәк;
Сүзем калган урында
Үсеп чыгар күкчәчәк.

Зәңгәр кичне төн алды,
Зәңгәр төнне көн алды;
Зәңгәр карлар эчендә
Әйтәсе сүзем калды.

Зәңгәр-зәңгәр кич иде,
Зәңгәр-зәңгәр кар иде;
Зәңгәрлеккә уралган
Әйтер сүзем бар иде.

Зәңгәр кичне төн алды,
Зәңгәр төнне көн алды;
Зәңгәр карлар эчендә
Әйтәсе сүзем калды.
```
