# Оригинал

### Рамазан Шагалиев / Салават Низаметдинов - Арыдым!

```
Арыдым сине көтеп,
Көтер йомгагын сүтеп,
Сүндерми учагыбызны,
Керфекләремде өтеп.

Арыдым!
Арыдым мин сине көтеп,
Арыдым, арыдым, арыдым!

Арыдым мин сине көтеп,
Өмет кылларын чиртеп.
Өмет кылларым өҙөлһә,
Ялгарбыз ничек итеп?

Арыдым!
Арыдым мин сине көтеп,
Арыдым, арыдым, арыдым!

Арыдым сине көтеп,
Ялгыз юллардан үтеп.
Буранда адашып калсам,
Табарсың ничек итеп?

Арыдым!
Арыдым мин сине көтеп,
Арыдым, арыдым, арыдым!
```
