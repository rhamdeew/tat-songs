# Оригинал

### Зифа Хәкимова / Радик Юлъякшин - Кызыма (бишенче вариант)

```
Яңа гына әле сабый идең,
Җиткән кыз да булдың инде, балам.
Тик күңелең hаман шул сабыйдай
Саф hәм якты булып кала алган.

Алгансың үзеңә язларның назларын,
Алтын көз иш иткән хисләрнең байлыгын.
Бөгелмә җилләргә, кыш сипсә карларын
Яныңда hәрвакыт мин саклап барырмын.
Кызыкайым, кызгынам,
Акыллы, сылусың (сулышың),
Озын, тигез hәм ак булсын.

Үтәчәк гомер юлың,
Бәхет-шатлыкларга тулсын
Үтәчәк гомер юлың.
Иң бәхетле әни иттең мине,
Ак нурларга төрдең дөньямны.
Теләгемнең изгеләре сиңа,
Кабул итсен Ходай догамны.

Алгансың үзеңә язларның назларын,
Алтын көз иш иткән хисләрнең байлыгын.
Бөгелмә җилләргә, кыш сипсә карларын
Яныңда hәрвакыт мин саклап барырмын.
Кызыкайым, кызгынам,
Акыллы, сылусың,
Мәхаббәтле, парлы бул син.
```
