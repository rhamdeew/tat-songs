# Оригинал

### Зилә Лира / Билгесез - Туйга теләк

```
Чәчегез чәчкә бәйләнде —
Бүген сезнең туегыз!
Бергә булсын, якты булсын,
Уртак булсын уегыз!
Тик бәхеттән генә торсын
Сезнең барыр юлыгыз!
Кайгы — хәсрәт, көенечне
Читкә алып куегыз!

Яныңда ярың — кулларда куллар!
Әткәй — әнкәй, дус — туганнар
Табынга җыелганнар!
Телибез сезгә иң изге теләк:
Бирсен Ходай күрешергә
Безгә алтын туйларда!

Берегез ай, берегез кояш,
Матур, зифа буйыгыз!
Аккошлардай пар килгәнсез,
Тик бәхетле булыгыз!
Татулыкта, иминлектә
Үтсен һәр бер көнегез!
Пар канатлы мәхәббәттең
Кадерләрен белегез!

Яныңда ярың — кулларда куллар!
Әткәй — әнкәй, дус — туганнар
Табынга җыелганнар!
Телибез сезгә иң изге теләк:
Бирсен Ходай күрешергә
Безгә алтын туйларда!

Табышып кавышкансыз,
Сүрелмәсен тойгыгыз!
Тормыш юлын җитәкләшеп,
Янәш кенә узыгыз!
Җил — буранда, караңгыда
Бер — үк, сез адашмагыз!
Ярты юлда өзелмәсен
Мәхәббәтле җырыгыз!

Яныңда ярың — кулларда куллар!
Әткәй — әнкәй, дус — туганнар
Табынга җыелганнар!
Телибез сезгә иң изге теләк:
Бирсен Ходай күрешергә
Безгә алтын туйларда!
```
