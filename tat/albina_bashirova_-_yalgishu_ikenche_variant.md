# Оригинал

### Альбина Бәширова - Ялгышу (икенче вариант)

```
Кичер мине, зинһар, мин ялгыштым,
Инде бәх(е)тем шул дип уйладым.
Шаяруың гына булган икән,
Ә мин, юләр, чынга юрадым.

Инде онытылган хисләремә
Ялкын өстәп кабат кабыздың.
Назың белән кинәт исерттең дә
Ташлап киттең мине ялгызым.

Күз яшьләрем күптән инде кипте,
Ник сулкылдый, елый бу йөрәк?
Еллар үтә, бәгырьдәге яра
Әрнеп сызлый һаман көчлерәк.
```
