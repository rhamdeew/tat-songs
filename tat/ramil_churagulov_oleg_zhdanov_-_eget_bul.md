# Оригинал

### Рамил Чурагулов / Олег Жданов - Егет бул

```
Сынаулардан, көрәшләрдән
Тора безнең гомерләр.
Салдат кына тимер корыч
Тыңмыйлар асыл ирләр.

Еллар калсын арта,
Һәр чакта алга барыйк.
Гомер юлларында
Ирләр булып та калыйк.
Тулпарларны куып
Һәр чакта алга барыйк.
Тормыш куйган сынау
Егет булып та калыйк.

Ирләр булып, егет булып
Үтик без тормыш юлын.
Һәр кем сиңа дустым диеп
Бирерлек булсын кулың.

Еллар калсын арта,
Һәр чакта алга барыйк.
Гомер юлларында
Ирләр булып та калыйк.
Тулпарларны куып
Һәр чакта алга барыйк.
Тормыш куйган сынау
Егет булып та калыйк.

Үткәннәрдә торып калсын
Сагышлар, янып көю.
Туар көннәр китерсеннәр
Өмет, ышаныч, сөю.

Еллар калсын арта,
Һәр чакта алга барыйк.
Гомер юлларында
Ирләр булып та калыйк.
Тулпарларны куып
Һәр чакта алга барыйк.
Тормыш куйган сынау
Егет булып та калыйк.
```
