# Оригинал

### Әхмәт Ерикәй / Мансур Мозаффаров - Беренче мәхәббәт (дүртенче вариант)

```
Бигрәк якты иде ай яктысы,
Безнең каплый иде сиреньнәр.
Син беренче тапкыр "сөям" дидең,
Кушылдылар безнең иреннәр.

Дөнья һаман матур. Кошлар сайрый.
Мин кайгырмам төсле үлсәм дә.
Эх! Беренче кичнең тулылыгы
Кабатланмас гомер сөйсәң дә.
```
