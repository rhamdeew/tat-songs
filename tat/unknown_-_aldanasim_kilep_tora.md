# Оригинал

### Әнгәм Атнабаев / Венер Улимаев - Алданасым килеп тора

```
Күккүк, кәккүк, кәккүк саный
Гомеремнең калганлыгын.
Озак саный. Белеп торам
Мин аның наданлыгын.
Исәп-хисап дәресеннән
“Икеле” алганлыгын,
Ырымнарның юраулары,
Гомүмән ялганлыгын.

Алда кәккүк, алда мине,
Гомерне озак сана!
Гомер буе алдадылар,
Ышанам — алдасаң да!
Алда кәккүк, алда мине,
Гомерне озак сана!
Алданасым килеп тора,
Ышанам — алдасаң да!

Белеп торам — алдый кәккүк,
Шулай да рәхәт була.
Әйтерсең дә күңелемә
Шифалы ләззәт тула.
Китәселәр килеп китә
Сабыйлык тарафына,
Атланасым килә менеп
Болытлар канатына.

Алда кәккүк, алда мине,
Гомерне озак сана!
Гомер буе алдадылар,
Ышанам — алдасаң да!
Алда кәккүк, алда мине,
Гомерне озак сана!
Алданасым килеп тора,
Ышанам — алдасаң да!

Эй, очасым килә ерак,
Гонаһлы җирдән туйдым!
Җир йөзеннән — күк йөзенә
Күчеп китәргә булдым.
Анда —киңлек, анда — тиңлек,
Инде тик сафлык кына!
Күңелләрең нәрсә тели —
Барысын да табып була.

Алда кәккүк, алда мине,
Гомерне озак сана!
Гомер буе алдадылар,
Ышанам — алдасаң да!
Алда кәккүк, алда мине,
Гомерне озак сана!
Алданасым килеп тора,
Ышанам — алдасаң да!
```
