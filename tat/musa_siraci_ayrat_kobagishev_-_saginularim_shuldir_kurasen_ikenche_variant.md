# Оригинал

### Муса Сираҗи / Айрат Кобагышев - Сагынуларым шулдыр, күрәсең (икенче вариант)

```
Нинди хәл бу?
Чишмә җырласа да,
Шаулап ага тигез камышым.
Ишетелә кебек тавышың
Сулкылдатып йөрәк,
Сулкылдатып йөрәк сагышын.

Нинди хәл бу?
Күккә карасаң да,
Йолдыз булып карый күзләрем,
Тулган айдай балкый йөзләрең,
Җилләр белән килә,
Җилләр белән килә сүзләрең.

Нинди хәл бу?
Күрәм төшләремдә,
Син елмаеп каршы киләсең,
Иркәлисең мине, сөясең,
Сагынуларым шулдыр,
Сагынуларым шулдыр, күрәсең.
```
