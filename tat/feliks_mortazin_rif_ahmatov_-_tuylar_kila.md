# Оригинал

### Феликс Мортазин / Риф Әхмәтов - Туйлар килә

```
Туйлар килә урамнарны иңләп
Туйлар килә, ярсый пар атлар
Чылтыр-чылтыр кыңгыраулар килә
Җирән кашка тиз аргамаклар

Алда әле канатланыр чаклар.
Алда әле чәчкә атар чаклар,
Алда әле күкрәп, күкрәп яшнәр чаклар,
Алда әле гөрләп, гөрләп яшәр чаклар!

Яшлек килә, яшнәп яшлек килә
Туй атлары тузан туздырып,
Җил давыллар белән туйлар килә
Яшьүсмерләр кала кызыгып.

Алда әле канатланыр чаклар.
Алда әле чәчкә атар чаклар,
Алда әле күкрәп, күкрәп яшнәр чаклар,
Алда әле гөрләп, гөрләп яшәр чаклар!

Канатланып, очып, ярсуланып
Бөтерелә сөю ташкында
Ак күлмәкләр, сөю балдаклары
Ярсый йөрәк, йөрәк ашкына

Алда әле канатланыр чаклар.
Алда әле чәчкә атар чаклар,
Алда әле күкрәп, күкрәп яшнәр чаклар,
Алда әле гөрләп, гөрләп яшәр чаклар!
```
