# Оригинал

### Зөфәр Вәлитов / Флера Шәрипова - Елама, җаным!

```
Яфрак койган агачларның
Йоклаганын да көтми.
Җиргә каныккан болытка,
Уй җитсә дә, буй җитми.

Язын — ләйсән, көзен — коен,
Нигәдер моңсу быел.
Бер кыенның бер рәхәте…
Елама, җаным, тыел!

Тик ул ишетә дисеңме,
Аның бит үз хәсләре…
Язын сөю нуры яуды,
Көзләрдә — күз яшьләре.

Язын — ләйсән, көзен — коен,
Нигәдер моңсу быел.
Бер кыенның бер рәхәте…
Елама, җаным, тыел!

Көзләрнең дә, язлар кебек
Буладыр үз хәстәре;
Салкын акыллы булганга
Салкындыр күз яшьләре.

Язын — ләйсән, көзен — коен,
Нигәдер моңсу быел.
Бер кыенның бер рәхәте…
Елама, җаным, тыел!
```
