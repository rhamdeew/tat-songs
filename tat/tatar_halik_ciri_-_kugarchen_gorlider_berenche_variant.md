# Оригинал

### Татар халык җыры - Күгәрчен гөрлидер (беренче вариант)

```
Син дә күгәрчен идең лә,
Мин дә күгәрчен идем;
Һаваларда очып йөрсәм,
Күреп сөйләшер идем.

Күгәрчен гөрлидер,
Язын, көзен белмидер;
Безне дошманнар сөйлидер,
Ни күрәсен белмидер.

Күк күгәрчен күлләрдә лә,
Иркәм, күңелең кемнәрдә?
Иркәм, күңелең миндә, булса,
Күрешик җомга көннәрдә.

Күгәрчен ояда,
Оялары пыяла;
Күгәрченнәр кебек гөрләп,
Торсаң иде дөньяда.
```
