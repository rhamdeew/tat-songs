# Оригинал

### Фирая Зыятдинова / Мәсгудә Шәмсетдинова - Кар вальсы

```
Яфрак-яфрак кына
Ап-ак, ап-ак кына
Карлар ява зəңгəр күклəрдəн.
Тияр-тимəс кенə,
Кыяр-кыймас кына
Кар бөртеге үбə битлəрдəн.

Əй ашкына күңел —
Бəйрəм алды бүген,
Бөтен дөнья актан киенгəн.
Күздə — йолдыз нуры,
Йөздə шатлык уйный;
Һəммəсе дə шундый сөенгəн.

Зəңгəр бəрхет һава,
Ап-ак карлар ява,
Ул карларда пакьлык-сафлык бар.
Юлларым да ап-ак,
Ул юлларда мине
Көтə кебек бары шатлыклар.
```
