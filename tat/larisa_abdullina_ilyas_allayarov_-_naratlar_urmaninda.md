# Оригинал

### Лариса Абдуллина / Ильяс Аллаяров - Наратлар урманында

```
Исеңдәме яңгыр яуды
Наратлар урманында.
Юк, яумады ул, җырлады ул,
Без икеү турасында.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.

Күңелләргә сеңеп калды
Моңнары шул мизгелнең.
Әллә ярсу чагы иде
Иңнәремә түгелдең.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.

Шул урманда яктыртың син,
Сүрелгән хисләремне.
Синең белән үтсәм иде
Калган гомерләремне.

Җырлады яңгыр, җырлады,
Күңелләрне нурлады.
Еламсырап җыр тыңлады
Яшь наратлар урманы.
```
