# Оригинал

### Рөстәм Зәкуан / Айдар Тимербаев - Авыл көтә һәрбер баласын

```
Шәһәр тузаннарын кага-кага,
Кайтып киләм, авылым, сагынып.
Дөнья зары синдә онытыла,
Күңелем кала синдә сакланып.

Юл тузаны, әйдә, тузса тузсын,
Җаныбызга сагыш булмасын.
Авыл күңеле, әнкәй күңеле кебек,
Каршы ала һәрбер баласын.

Урманнарың сөеп каршы ала,
Һәр агачың туган шикелле.
Имәннәрең миңа ныклык бирде,
Каеннарың назлар үстерде.

Юл тузаны, әйдә, тузса тузсын,
Җаныбызга сагыш булмасын.
Авыл күңеле, әнкәй күңеле кебек,
Каршы ала һәрбер баласын.

Дога кылсам туган нигеземдә
Җаным китә гүя чистарып.
Син яшә, газиз авылкаем,
Син булганда, күңел саф калыр.

Юл тузаны, әйдә, тузса тузсын,
Җаныбызга сагыш булмасын.
Авыл күң(е)ле, әнкәй күң(е)ле кебек,
Каршы ала һәрбер баласын.
Юл тузаны… Һә-ә-әй!
Юл тузаны, әйдә, тузса тузсын,
Җаныбызга сагыш булмасын.
Авыл күңеле, әнкәй күңеле кебек,
Каршы ала һәрбер баласын.
Каршы ала һәрбер баласын…
```
