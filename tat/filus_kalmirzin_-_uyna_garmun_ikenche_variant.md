# Оригинал

### Филүс Калмырзин - Уйна, гармун (икенче вариант)

```
Сызгырт әле моңлы гармунны,
Тарат әле йөрәк сагышын.
Дәртләндереп күңел кылларына
Чиртсен телләреңнең тавышы.

Уйна, дускай, уйна гармуныңны,
Күңелләргә кереп уралсын.
Хис-тойгылар аша чыккан моңнар
Халык җыры булып таралсын.

Гармун тавышлары дәртләндереп
Алып кайта яшьлек илемә.
Авылымның тугайларын буйлап
Йөрер идем тагын бер генә.

Уйна, дускай, уйна гармуныңны,
Күңелләргә кереп уралсын.
Хис-тойгылар аша чыккан моңнар
Халык җыры булып таралсын.

Гармуннарның серле сагыш моңы
Ургылып чыга гармун теленнән.
Ничә еллар гармун тавышлары
Моңлы җырлар булып түгелгән.

Уйна, дускай, уйна гармуныңны,
Күңелләргә кереп уралсын.
Хис-тойгылар аша чыккан моңнар
Халык җыры булып таралсын.
```
