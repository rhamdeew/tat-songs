# Оригинал

### Гөлсәрвәр / Радик Халиков - Якынымның туган көне бүген

```
Төрле ел фасылы безнең якта —
Көзе, кышы, язы, җәйләре.
Һәркайсында һәркөн бәйрәм була,
Кемнәрнеңдер туган көннәре.

Якынымның туган көне бүген,
Ул бәхетле, рәхәт яшәсен.
Кайгырмасын гомер үтә, диеп,
Кадерләрен белеп яшәсен.

Җыелышып килдек синең янга,
"Бәйрәм булсын", — әле әйдәле.
Бик күңелле бүген безнең җанга,
Бергәләшеп биеп әйләник.

Якынымның туган көне бүген,
Ул бәхетле, рәхәт яшәсен.
Кайгырмасын гомер үтә, диеп,
Кадерләрен белеп яшәсен.

Озын булсын иде безнең гомер,
Тынычлыкта үтсен көннәре.
Бәйрәм итеп кенә Һәрбер елда
Каршылыйк без туган көннәрне.
```
