# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Кил, Яңа ел!

```
Ак карларны биетеп
Яңа ел килә җиремә.
Көтеп алабыз ашкынып,
Күңел ак нурга төренә.

Кил, яңа ел,
Кил, яңа ел,
Күңелнең үз түренә.
Өр яңа юл
Яручы бул
Минем киләчәгемә

Яңа елның сәгәтләре
Суксын бәхет теләп кенә
Җирдә барча кешеләрең
Яшәсеннәр көлеп кенә.

Кил, яңа ел,
Кил, яңа ел,
Күңелнең үз түренә.
Өр яңа юл
Яручы бул
Минем киләчәгемә

Еллар очрашып, могҗиза
Өмет иткән мизгелләрдә
Өлгер барчасын юрарга
Якты, матур, изгеләргә.
```
