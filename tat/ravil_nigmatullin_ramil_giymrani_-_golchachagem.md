# Оригинал

### Равил Нигъмәтуллин / Рамил Гыймрани - Гөлчәчәгем

```
Таралсын чәчең нурланып,
Таралсын үрелмәсен,
Миннән башка чәчләреңә
Бер кем дә үрелмәсен!

Гөлем, гөлем, Гөлчәчәгем —
Син минем киләчәгем.

Килешә сиңа калай да,
Җилгә чәчеңне тарат,
Башкаларга күз салма син,
Үземне генә ярат.

Гөлем, гөлем, Гөлчәчәгем —
Син минем киләчәгем.

Таралсын чәчең нурланып,
Таралсын үрелмәсен,
Мәхәббәткә кер кунмасын,
Сөюең сүрелмәсен?

Гөлем, гөлем, Гөлчәчәгем —
Син минем киләчәгем.
```
