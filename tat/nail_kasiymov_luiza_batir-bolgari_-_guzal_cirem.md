# Оригинал

### Наил Касыймов / Луиза Батыр-Болгари - Гүзәл җирем

```
Нурлы җирем минем, моңлы җирем,
Яшә мәңге тыныч балкышларда!
Чәчәк исе килсен җилләреңнән
Аклык кына булсын ак кышларда!

Гүзәл җирем минем,
Зәңгәр күзле җирем.
Күзләремә карыйсың моң белән.
Суларыңның тәмен, чәчәкләрнең ямен
Саклар кеше мәңге-мәңгегә!

Кояш назлый көндез, төнлә йолдыз,
Галәм белми синдәй гүзәл җирне.
Гөлләр үсә бары синдә генә,
Синдә генә сандугачлар җыры.

Гүзәл җирем минем,
Зәңгәр күзле җирем,
Күзләремә карыйсың моң белән.
Суларыңның тәмен, чәчәкләрнең ямен
Саклар кеше мәңге-мәңгегә!
```
