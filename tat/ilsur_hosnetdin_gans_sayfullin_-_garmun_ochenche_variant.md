# Оригинал

### Илсур Хөснетдин / Ганс Сәйфуллин - Гармун (өченче вариант)

```
Авыл буйлап гармун уйнап
Җырлыйсы иде тагы.
Искә төшә, сагындыра
Туган авылым ягы.

Сандугачларны уздыра
Кичке тальян моңнары.
Сыздырып уйнап җибәрә —
Их, моңлы гармуннары.

Гармун моңнары тарала
Авыл урамнарына.
Кайтам әле авылыма —
Дуслар, туганнарыма.

Сандугачларны уздыра
Кичке тальян моңнары.
Сыздырып уйнап җибәрә —
Их, моңлы гармуннары.

Авылымда таллыгымда
Була кичке уеннар.
Уйна әле гармуныңны —
Сиңа дигән җырым бар.

Сандугачларны уздыра
Кичке тальян моңнары.
Сыздырып уйнап җибәрә —
Их, моңлы гармуннары.
```
