# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Зилә-Зилә-Зиләйлүк

```
Сиңа көн дә гашыйк булып,
Сиңа көн дә кызыгып,
Син, мөгаен, Казанның
Иң матур бер кызыдыр.

Зил-зилә, әй, кил яныма,
Җилә-җилә кил-әйдүк.
Казанның иң матур кызы
Зилә-Зилә-Зиләйлүк.

Казанда матур кызлар күп,
Синең төслесе сирәк.
Минем үземә нәкъ менә
Синең төслесе кирәк.

Зил-зилә, әй, кил яныма,
Җилә-җилә кил-әйдүк.
Казанның иң матур кызы
Зилә-Зилә-Зиләйлүк.

Син алмадай өлгергән кыз,
Син чиядәй пешкән кыз.
Нәкъ шундый булырга тиеш
Минем күзем төшкән кыз.

Зил-зилә, әй, кил яныма,
Җилә-җилә кил-әйдүк.
Казанның иң матур кызы
Зилә-Зилә-Зиләйлүк.
Казанның иң матур кызы
Зилә-Зилә-Зиләйлүк…
Зилә-Зилә-Зиләйлүк…
```
