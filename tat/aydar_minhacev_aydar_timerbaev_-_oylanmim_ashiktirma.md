# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Өйләнмим (Ашыктырма)

```
Югалсам да азга гына,
Син үзең килеп яныма
Үпкәләгән буласың, буласың.
Әле алай гына түгел,
Хәзер инде менә син гел
Туй дип ашыктырасың, ашыктырасың.

Юк, димә әле, мин өйләнмим,
Нигә киергә камыт.
Бер генә килгән яшьлекнең
Кадерен бел, ди халык.

Менә үзең санап кара:
Күп тә түгел танышканга,
Нигә һаман туй дисең, туй дисең?
Мин яныңда бит көн саен ,
Ниләр җитми инде тагын,
Нигә түзә белмисең, түзә белмисең?

Юк, димә әле, мин өйләнмим,
Нигә киергә камыт.
Бер генә килгән яшьлекнең
Кадерен бел, ди халык.

Әйтсәм — гел ачуың килә,
Туй итәргә инде әллә…
Өлгермәбез дисеңме, дисеңме?
Әгәр яратасың икән,
Ашыктырма мине, иркәм,
Тыя бел нәфесеңне, нәфесеңне.

Юк, димә әле, мин өйләнмим,
Нигә киергә камыт.
Бер генә килгән яшьлекнең
Кадерен бел, ди халык.
```
