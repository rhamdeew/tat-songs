# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Үзең килче…

```
Чакырма, килмим синең яныңа, мәхәббәтем,
Кайгы да салмам тыныч җаныңа, диеп әйттем.
Кыю бул, иркәм, дисең.
Йосыф булган кыю да… —
Ташлаганнар коега.

Минем янга, иркәм, син үзең килче.
Йә, булмаса, җибәр син димче.
Мин көтәрмен, еллар буе түзәрмен,
Юлла сөю сүзләрең.

Мине кемдер яратырга тиеш, кемдер тиеш,
Үлеп яратырга шушы килеш, шушы килеш.
Таһир Зөһрәне тапкан,
Ә соңыннан үзе дә
Сандыкта суда аккан.

Минем янга, иркәм, син үзең килче.
Йә, булмаса, җибәр син димче.
Мин көтәрмен, еллар буе түзәрмен,
Юлла сөю сүзләрең.

Кирәк түгел сөю кайгылары, кайгылары.
Килсен кояшлысы, айлылары, айлылары.
Мәҗнүн сөйгән, яраткан,
Кара кайгыга баткан,
Аннан башын югалткан.

Минем янга, иркәм, син үзең килче.
Йә, булмаса, җибәр син димче.
Мин көтәрмен, еллар буе түзәрмен,
Юлла сөю сүзләрең.
```
