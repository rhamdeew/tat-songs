# Оригинал

### Гөлшат Зәйнәшева / Сафьян Ибраһимов - Гимназиядә дуслаштык

```
Гимназия ишекләре
Ачылды безгә бүген.
Яулап алыйк мәгърифәтнең
Якты йолдызлы күген.

Гимназиядә дуслаштык,
Безнең дуслык гомерлек.
Беребез өчен беребез
Йолдызларга менәрлек.

Иделнең изге суында
Битебезне юдык без.
Идел-йортта игелекләр
Эшләр өчен тудык без.

Гимназиядә дуслаштык,
Безнең дуслык гомерлек.
Беребез өчен беребез
Йолдызларга менәрлек.

Туган телебез көмештер,
Чыңлап тора сүзләре.
Туган телебез аркылы
Ачылыр фән серләре.

Гимназиядә дуслаштык,
Безнең дуслык гомерлек.
Беребез өчен беребез
Йолдызларга менәрлек!..
```
