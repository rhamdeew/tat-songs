# Оригинал

### Мостай Кәрим / Таһир Кәримов - Шофер җыры (Такмак)

```
Ал итә, итә, итә,
Гөл итә, итә, итә,
Яннарыңа барыр идем
Ял итә, итә, итә.

Ал булмый, булмый, булмый,
Гел булмый, булмый, булмый.
Бер болганмый су да тынмый,
Болганмый булмый, булмый.

Ал итә, итә, итә,
Гөл итә, итә, итә,
Яннарыңа барыр идем
Ял итә, итә, итә.
```
