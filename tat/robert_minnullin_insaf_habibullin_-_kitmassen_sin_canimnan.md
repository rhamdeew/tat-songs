# Оригинал

### Роберт Миңнуллин / Инсаф Хәбибуллин - Китмәссең син җанымнан

```
Төнге утлар көрсенә,
Бер яна да, бер сүнә.
Китәсеңме ияреп,
Шул утларның берсенә.

Китәсең дип сарылмам,
Китмә, дип тә ялынмам.
Китәсең дә син янымнан,
Китәлмассың җанымнан.

Ут яктысы — йөзеңдә,
Төн карасы — күзеңдә.
Юкка гына китәсең,
Аңлыйсыңмы үзең дә?

Китәсең дип сарылмам,
Китмә, дип тә ялынмам.
Китәсең дә син янымнан,
Китәлмассың җанымнан.

Үпкәм кара төннәргә,
Арабызга кергәнгә.
Мине кара төннәрдә
Калдырасың кемнәргә.

Китәсең дип сарылмам,
Китмә, дип тә ялынмам.
Китәсең дә син янымнан,
Китәлмассың җанымнан.
```
