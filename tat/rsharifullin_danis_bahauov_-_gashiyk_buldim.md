# Оригинал

### Р.Шәрифуллин / Данис Баһауов - Гашыйк булдым

```
Бер кызны күрдем дә яраттым,
Елмаеп үземә караттым.
Аңа унбиш яшь, миңа унҗиде,
Күзләре миңа «яратам», диде.

Миңа нишләргә, әйтегез сез, дуслар!
Егетләр хәлен аңлармы кызлар.
Хисләрем минем сүрелде хәзер,
Килерме миңа яңадан назлар.

Аның әйткән hәрбер сүзләре
Сөю назы бирә күңелгә.
Сөю назларында гел шулай
Булсак иде гомер-гомергә.

Миңа нишләргә, әйтегез сез, дуслар!
Егетләр хәлен аңлармы кызлар.
Хисләрем минем сүрелде хәзер,
Килерме миңа яңадан назлар.

Мин дә бит нык яратам аны,
Ничек әйтергә соң үзенә.
Ояламын хәттә карарга
Аның серле зәңгәр күзенә.

Миңа нишләргә, әйтегез сез, дуслар!
Егетләр хәлен аңлармы кызлар.
Хисләрем минем сүрелде хәзер,
Килерме миңа яңадан назлар.
```
