# Оригинал

### Рәүф Сәлтәшев - Яшьләр туе

```
Яз исенә тулган авыл.
Яшь йөрәкләр ярәшә.
Кәләшнең ак күлмәгеннән
Умырзая көнләшә.

Кыңгырау чыңлавы белән
Авыл урамы тулы.
Яшь йөрәкләр ярәшәләр —
Авылда яшьләр туе.

Тулпар атлар җилеп уза
Авылым урамыннан.
Яшьләр туе — табигать туе
Зәңгәр яз моңнарыннан.

Кыңгырау чыңлавы белән
Авыл урамы тулы.
Яшь йөрәкләр ярәшәләр —
Бәхетләр булсын тулы.

Яшьләр туе. Барсы бии,
Кияве, кәләшләре…
Гомер булсын, булсын туйлар!
Алтыны, көмешләре.
```
