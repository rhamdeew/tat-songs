# Оригинал

### Миңнетдин Шәфыйков / Раил Өметбаев - Яңа ел теләкләре (беренче вариант)

```
Тагын бер ел үтеп китте,
Яңа еллар килеп җитте.
Зур өметләр белән аны
Җир йөзендә һәркем көтте.

Яңа еллар белән котлап,
Мин изгелек теләем сезгә.
Уй-хыяллар кабул булсын,
Матур итеп яшәек бергә.

Яңа еллар килә-китә,
Еллар аша гомер үтә.
Безнең һәрбер үткән көннәр
Гомер йомгакларын сүтә.

Яңа еллар белән котлап,
Мин изгелек теләем сезгә.
Уй-хыяллар кабул булсын,
Матур итеп яшәек бергә.

Әй, Яңа ел, син алып кил
Тынычлык көннәре безгә,
Бәхет-шатлык, куанычлар
Бәрәкәтле яшәешкә.

Яңа еллар белән котлап,
Мин изгелек теләем сезгә.
Уй-хыяллар кабул булсын,
Матур итеп яшәек бергә.
```
