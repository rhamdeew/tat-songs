# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Сагыну җыры (өченче вариант)

```
Җылы язлар җиткәч,
Бозлар агып киткәч.
Зәңгәр күккә аккош омтыла
Җәйнең ал таңында,
Юксынган чагымда
Очып кына килсәң ни була?

Сахраларга чыксам,
Хәйран гына булам.
Сокланамын кошлар тавышына
Син гүзәлем белән.
Сөйгән ярым белән
Насыйп булсын иде кавышырга

Җылы язлар җиткәч,
Бозлар агып киткәч,
Зәңгәр күккә аккош агыла
Әллә аккошларның
Канаткаен алып,
Үзем барыйм микән яныңа?
```
