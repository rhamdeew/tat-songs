# Оригинал

### Башкорт халык җыры - Таулар биек (икенче вариант)

```
Таулар биек, таулар биек,
Таулар биек менәргә.
Бачкычлар юк, бачкычлар юк,
Шул тауларга менәргә.

Болындагы чәчәкләрне
Тапамам дигән идем.
Туып үскән авылымны
Ташламам дигән идем.

Болындагы чәчәкләрне
Тапарга туры килде.
Туып үскән авылымны
Ташларга туры килде.
```
