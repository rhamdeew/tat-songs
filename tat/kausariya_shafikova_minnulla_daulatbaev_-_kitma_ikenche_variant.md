# Оригинал

### Кәүсәрия Шәфикова / Миңнулла Дәүләтбаев - Китмә (икенче вариант)

```
Канатлар да каерылмасын,
Йөрәкләр дә аерылмасын,
Кара кашлар китмә, китмә.
Ул бит синең яшьлек таның,
Өзелеп сөгән тәүге ярың,
Боектырма, ялгыз итмә.

Еракларга китеп кенә,
Онытырсың ничек кенә
Тәңгәл хисләр, уртак хисләр.
Язгы сулар аккан чакта,
Бөреләр чәчәк аткан чакта,
Йөрәккәем инде нишләр?

Бергә чаклар нинди ямьле,
Яннарымда бул син мәңге
Сайрар гына кошкаемның:
Үзәккәем өзелмәсен,
Күз яшьләре түгелмәсен,
Җанга якын дускаемның.
```
