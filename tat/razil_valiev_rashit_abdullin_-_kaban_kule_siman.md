# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Кабан күле сыман

```
Кабан күле сыман
Моңсу кайчак күңел,
Сүзләр белән генә,
Юатырлык түгел.

Салды безне еллар
Язмыш учагына…
Кабан күле мәңге
Казан кочагында.

Кабан күле сыман
Ярсый кайчак җаным.
Күмелгәндер тарих
Төпләрендә аның.

Салды безне еллар
Язмыш учагына…
Кабан күле мәңге
Казан кочагында.

Кабан күле якты
Халкым йөзе сыман.
Кабан ярларына
Безнең язмыш сыйган.

Салды безне еллар
Язмыш учагына…
Кабан күле мәңге
Казан кочагында.
```
