# Оригинал

### Ризида Мөхәммәтҗанова / Радик Юлъякшин - Шатлык хисләре

```
Кайтып кердем авылга
Язгы кичләрдә.
Күптән күрмәгән идем
Дус-иптәшләрне.

Шашып таша күңелемдә
Шатлык хисләре.
Шомыртлар чәчәк атканнар —
Татлы исләре!

Эх, дусларым, әйдәгез
Тауның башына!
Анда учак ягарга
Күңел ашкына.

Шашып таша күңелемдә
Шатлык хисләре.
Шомыртлар чәчәк атканнар —
Татлы исләре!

Кабатланды шул кичтә
Яшьлек җырлары,
Шашып күңел хисеннән
Йөрәк җырлады.

Авыл кичен дә яратам,
Дуслар, сезне дә.
Җыелсак иде шулай без
Язын гел бергә.
```
