# Оригинал

### Рәүфә Нурмөхәммәтова - Соңлама (беренче вариант)

```
Сандугач моңлы сайрый,
Кайдан ала ул моңны?
Синең белән шул сандугач,
Матурлата доньяны.

Соңлама, иркәм, соңлама,
Айда калыкы күптән,
Көттермә мине, зарактырма,
Ышанмам сиңарга бүтән.

Вакыт мы әкрен үтә?
Әллә йөрәк түземсез?
Йолдызларда җимелдәми,
Бу айлы көндә синсез.

Соңлама, иркәм, соңлама,
Айда калыкы күптән,
Көттермә мине, зарактырма,
Ышанмам сиңарга бүтән.

Яндырма син, көттермә,
Ай сокмаганнан кил че,
Сагынган йөрәгемә,
Назлы сөюләр бир че.

Соңлама, иркәм, соңлама,
Айда калыкы күптән,
Көттермә мине, зарактырма,
Ышанмам сиңарга бүтән.
```
