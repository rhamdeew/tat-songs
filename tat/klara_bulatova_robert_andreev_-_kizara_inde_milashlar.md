# Оригинал

### Клара Булатова / Роберт Андреев - Кызара инде миләшләр

```
Кызара инде миләшләр,
Ни көтә тиле күңел?..
Әйтерсең, әле гел яшьләр,
Әйтерсең, илле түгел.

Әйтерсең, әле күзем дә
Күрә гел көннең алын.
Әйтерсең, әле синең дө
Артымнан йөргән чагың.

Әйтерсең, әле телең дө
Гел матур сүзләр сибә.
Үзгәрдең юкса, үзгәрдең,
Үзгәрдең… Көзләр килә.

Сүзгә дә саран димәсләр,
Сайламый гөлен генә.
Кызара инде миләшләр,
Көз дими күңел генә.
```
