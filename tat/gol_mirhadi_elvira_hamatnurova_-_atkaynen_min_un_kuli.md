# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Әткәйнең мин уң кулы

```
Әткәемнең мин улы,
Һәр эштә дә уң кулы,
Һәр эштә дә уң кулы.
Өшеп — туңып кайттык әле.
Өшеп — туңып кайттык әле
Хоккей карап бу юлы!

Барын да булдырам
Әткәй янда булганда мин.
Әткәй мактап көлеп тора:
«Улым уңган, — булдыра!»

Әткәемнең мин улы,
Һәр эштә дә уң кулы,
Һәр эштә дә уң кулы.
Утын ярды пешеп-янып.
Карап тордым арып-талып,
Бик булыштым бу юлы!

Барын да булдырам
Әткәй янда булганда мин.
Әткәй мактап көлеп тора:
«Улым уңган, — булдыра!»

Әткәемнең мин улы,
Ни кырырбыз бу юлы?
Ни кырырбыз бу юлы?
Нинди эш тә пүчтәк кенә:
Тиз ташларбыз эшләп менә.
Чөнки мин бар — уң кулы!

Барын да булдырам
Әткәй янда булганда мин.
Әткәй мактап көлеп тора:
«Улым уңган, — булдыра!»
```
