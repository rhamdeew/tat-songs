# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Бәхетле мизгелләр

```
Өз(е)леп сөйгән ярлар гына
Мәңгелеккә кавыша,
Язмышны кушып язмышка
Пар балдаклар алыша!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!

Юлыгызга, бүгенгедәй,
Сибелсеннәр ак гөлләр!
Йөрәкләрдә гел саклансын
Бу бәхетле мизгелләр!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!

Югалмасын, тапланмасын,
Суынмасын саф хисләр!
Арагыздан үтмәсеннәр
Хәтта юхалы җилләр!

Бәхет иле гомерлеккә
Ачсын җылы кочагын.
Яшәгез сез сүндермичә
Гайләгезнең учагын!
```
