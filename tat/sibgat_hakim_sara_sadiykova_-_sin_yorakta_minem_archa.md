# Оригинал

### Сибгат Хәким / Сара Садыйкова - Син йөрәктә минем, Арча

```
Арча, Арча,
Бөтен дөнья сокланыр як,
Батырлары эштә ярышалар.

Арча — ямьле урман, кырлар ягы,
Арча — данлы Тукай-Кырлай ягы.
Шуңа күрә горур, гөрләп үтә гомер,
Арчалылар, без бит Арчалар.

Арча, Арча,
Бөтен дөнья сокланыр як,
Күлләр зәңгәр.

Син йөрәктә, минем җырым төсле,
Халкым эшчән, кыю, батыр, көчле.
Шуңа күрә горур, гөрләп үтә гомер,
Арчалылар, без бит Арчалар.

Арча, Арча,
Бөтен дөнья сокланыр як,
Мәңге изге җирең.

Арча оста читекчеләр ягы,
Читекләргә чигелә Арча даны.
Шуңа күрә горур, гөрләп үтә гомер,
Арчалылар, без бит Арчалар.
```
