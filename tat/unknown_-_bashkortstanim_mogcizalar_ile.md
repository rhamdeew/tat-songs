# Оригинал

### Билгесез / Риф Арслан - Башкортстаным — могҗизалар иле

```
Туган илне салап алмыйлар да,
Язмышка ул язып куела.
Күңелеңә хис һәм тойгылардан,
Нечкә нагыш белән тоела.

Башкортстаным — могҗизалар иле,
Һәр тарафың синең хозурлык.
Матурлыгың кучә күңелләргә,
Синең өчен булып горурлык.

Туган илне сатып алмыйлар да,
Хаклары юк аны түләрлек.
Йөрәгемдә яшә изге сөю,
Аның өчен яуда үләрлек.

Башкортстаным — могҗизалар иле,
Һәр тарафың синең хозурлык.
Матурлыгың кучә күңелләргә,
Синең өчен булып горурлык.
```
