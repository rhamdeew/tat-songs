# Оригинал

### Флария Терегулова / Азат Имаев - Их, янасым килә…

```
Онытылып, бер янасым килә
Син кабызган сөю учагында.
Дөньяларны онытып
Югаласым килә,
Югаласым синең кочагында.

Сарыласым килә кайнар куеныңа,
Югаласым синең карашыңда.
Ишетәсем килә
Назлы сүзләр,
Ишетәсем килә колагымда..

Их! Каласым килә,
Иң кадерлең булып,
Иң кирәклең булып йөрәгеңдә.
Их! Янасым килә тик бер кабат,
Син кабызган көзге учагыңда…

Учагыңны як та чакыр…
```
