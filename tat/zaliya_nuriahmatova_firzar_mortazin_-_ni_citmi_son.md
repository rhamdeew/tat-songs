# Оригинал

### Залия Нуриәхмәтова / Фирзәр Мортазин - Ни җитми соң?

```
Ник керәсең төшләремә,
Нәрсә өмет итәсең?
Юлларымда очрасаң —
Узасың да китәсең.

Ни җитми соң, ниләр эшлим,
Газаптан каралыйммы?..
Дөрләп янып, көлгә калып,
Җилләрдә таралыйммы?

Сөю тулы күзләреңне
Нигә соң яшерәсен?
Хыялымда күккә очам…
Син җиргә төшерәсең.

Ни җитми соң, ниләр эшлим,
Газаптан каралыйммы?..
Дөрләп янып, көлгә калып,
Җилләрдә таралыйммы?

Әй, җаныем, мендәремә
Хисләремне түгәмен.
Йөрәгем тулы мәхаббәт,
Ничек итеп түзәрмен?

Ни җитми соң, ниләр эшлим,
Газаптан каралыйммы?..
Дөрләп янып, көлгә калып,
Җилләрдә таралыйммы?
```
