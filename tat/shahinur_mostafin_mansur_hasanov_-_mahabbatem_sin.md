# Оригинал

### Шаһинур Мостафин / Мансур Хәсәнов - Мәхәббәтем син…

```
Мәхәббәтем син, мәхәббәтем син…
Сагынып көткән язым —
Бәхетнең үзеннән килеп,
Бәхетле иткән ярым.

Мәхәббәтем син, мәхәббәтем син…
Яратып туймас җырым —
Өметем минем, хыялым
Таңнарым, барыр юлым.

Мәхәббәтем син, мәхәббәтем син…
Мәңге сүнмәс кояшым —
Тормышымның юлларында
Нур сибүче юлдашым!
```
