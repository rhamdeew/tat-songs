# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Сау бул мәхәббәтем!

```
Сау бул инде, сау бул мәхәббәтем,
Мин сөеп тә мине сөймәгән.
Ялкын сибеп, балкып эчтән-тыштан
Миңа гына яну димәгән.

Сау бул инде, сау бул мәхәббәтем,
Хушлашырга вакыт җиткәндер.
Элеккечә хыялый җан булып
Могҗизалар синнән көтмәмдер.

Сау бул инде, сау бул мәхәббәтем,
Күрәсеңдер, инде мин кышта.
Соң аңладым шуны— минем өчен
Хыялда бар, юк син тормышта.

Сау бул инде, сау бул мәхәббәтем,
Хушлашырга вакыт җиткәндер.
Элеккечә хыялый җан булып
Могҗизалар синнән көтмәмдер.

Сау бул инде, сау бул мәхәббәтем,
Җибәрәмен сине иреккә.
Сизеп торам, минем йөрәк кысан–
Сиңа урын күктә— биектә.

Сау бул инде, сау бул мәхәббәтем,
Хушлашырга вакыт җиткәндер.
Элеккечә хыялый җан булып
Могҗизалар синнән көтмәмдер.

Сау бул инде, сау бул мәхәббәтем,
Гаҗәп түгел берни дәшмәвең.
Үзем кебек мине яратмагач
Хәерлерәк синсез яшәвем.

Сау бул инде, сау бул мәхәббәтем,
Хушлашырга вакыт җиткәндер.
Элеккечә хыялый җан булып
Могҗизалар синнән көтмәмдер.
```
