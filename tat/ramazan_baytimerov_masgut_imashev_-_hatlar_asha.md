# Оригинал

### Рамазан Байтимеров / Мәсгуть Имашев - Хатлар аша

```
Төштә күргән кебек булдым,
Төшкә кергән кебек булдың.
Ни күреп калдым икән?
Син гади танышым микән,
Бәхетем, шатлыгым микән,
Сагышым, кайгым микән?

Исемнәреңне ятлармын,
Килеп, торсалар хатларың,
Сиздереп йөрәк җылың.
Хисләрне хат ялкынлата,
Хатлар гына якынайта
Аралар ераклыгын.

Хатларым озын юл үтәр,
Тиздән сиңа барып җитәр
Сәламем тапшырырга.
Хатлар безне таныштырды,
Хатлар безне табыштырды,
Бәлки, кавыштырыр да…
```
