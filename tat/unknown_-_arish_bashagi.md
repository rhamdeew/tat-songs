# Оригинал

### Сафуан Әлибай / Рим Хәсәнов - Арыш башагы

```
Чайкала дулкыннар, чайкала,
Офыктан офыкка ашкынып.
Җиремдә чал диңгез күңел дә,
Калкынды башаклар ташкыны.

Арыш башагы,
Арыш башагы,
Синнән дә изгерәк
Ни генә бар тагы?!
Син — кояш нурыдыр,
Син — көрәш җырыдыр,
Арыш башагы!

Хәтерли газиз җир, хәтерли:
Яулар да кем генә аумаган,
Туфракка тир түгел, кан тамып,
Һәр бөртек көрәштә яуланган.

Арыш башагы,
Арыш башагы,
Синнән дә изгерәк
Ни генә бар тагы?!
Син — кояш нурыдыр,
Син — көрәш җырыдыр,
Арыш башагы!

Тирбәлә җилләргә, тирбәлә,
Кырлар да ашлыклар, арышлар.
Безнең бу ярсыган күңелләр (гомерләр) —
Үзләре мәңгелек алышлар.

Арыш башагы,
Арыш башагы,
Синнән дә изгерәк
Ни генә бар тагы?!
Син — кояш нурыдыр,
Син — көрәш җырыдыр,
Арыш башагы!
```
