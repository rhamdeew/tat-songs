# Оригинал

### Гыйлемдар Рамазанов / Рәүф Мортазин - Газиз илем — Туган җирем

```
Бала чагым узган җирем,
Газиз илем — Туган җирем,
Туган җирем,
Син тормышым, җырым минем,
Мәхәббәтем, моңым минем,
моңым минем,
Сөям сине, нурлы җирем.

Синнән башка миңа ямь юк,
Ят суларда мындый тәм юк,
Мындый тәм юк.
Сине мактап туеп булмый,
Синнән читтә торып булмый,
торып булмый,
Синнән башка җырым тумый.
```
