# Оригинал

### Фәрит Суфияров / Роберт Тимербаев - Бирешмә (бирешмик) кайгыларга

```
Аклыгын-сафлыгын саклый,
Карыймын каеннарга.
Җил давыллар кагылса да,
Бирешми(к) кайгыларга.

Ялгыз башым уйга төштең,
Кайгы күлләре тирән шул,
Тормыш юлларын үтәргә
Түзем булырга өйрән.

Аклыгын-сафлыгын саклый
Көмеш чишмә сулары.
Яшьлектә йөргән сукмаклар
Күңелгә моң чорнады.

Ялгыз башым уйга төштең,
Кайгы күлләре тирән шул,
Тормыш юлларын үтәргә
Түзем булырга өйрән.

Аклыгын-пәкълеген саклый,
Балкып ата(р) таңнары.
Сандугачлар сайраганда
Башын ия талларым.

Ялгыз башым уйга төштең,
Кайгы күлләре тирән шул,
Тормыш юлларын үтәргә
Түзем булырга өйрән.
```
