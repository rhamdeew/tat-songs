# Оригинал

### Әхсән Халилов / Рамил Гыймрани - Рәнҗемә, җаным!

```
Кинәт югалдың, иркәм,
Салдың сагыш зарына.
Кавышу язын юрыймын
Кәккүк тавышларыннан.

Рәнҗемә, җаным, берүк
Гаебем булса кичер.
Өзмә өметләреңне,
Йөрәгеңдә сакла сер.

Җавап хатларым адашты,
Аралар хәтсез ерак.
Язын кайткан кошлар кебек
Очып кайтырсың сыман.

Рәнҗемә, җаным, берүк
Гаебем булса кичер.
Өзмә өметләреңне,
Йөрәгеңдә сакла сер.

Кинәт югалдың иркәм
Салдың сагыш зарыма.
Кавышу язын юрыймын
Кәккүк тавышларыннан.

Рәнҗемә, җаным, берүк
Гаебем булса кичер.
Өзмә өметләреңне,
Йөрәгеңдә сакла сер.
```
