# Оригинал

### Айдар Минһаҗев / Айдар Тимербаев - Җайланырмы хәлләр?

```
Син дә гаепледер, мин дә гаепледер,
Җитми лә шул безгә сабырлык.
Тынды сөйләшүләр, инде ничә көннәр
Үпкәләүдән булмый арынып.

Җайланырмы хәлләр, эх, бу эндәшмәүләр
Әле тагын күпме сакланыр?
Я, кайсыбыз башлап тәүге адым ясар,
Гафулар ит, диеп, әйтә (а)лыр?

Йөрәк тибә ярсып, узмый бер дә вакыт,
Көннәр бигрәк озын тоела.
Горурлыгым җиңеп, йөгерепләр килеп
Сыенасы иде куй(ы)ныңа.

Җайланырмы хәлләр, эх, бу эндәшмәүләр
Әле тагын күпме сакланыр?
Я, кайсыбыз башлап тәүге адым ясар,
Гафулар ит, диеп, әйтә (а)лыр?

Араларны өзеп, саг(ы)нуларга түзеп,
Интегепләр нигә йөрергә?
Ни җитми соң безгә, күр(е)шеп икебезгә,
Гаепләрне таный белергә.

Җайланырмы хәлләр, эх, бу эндәшмәүләр
Әле тагын күпме сакланыр?
Я, кайсыбыз башлап тәүге адым ясар,
Гафулар ит, диеп, әйтә (а)лыр?
Я, кайсыбыз башлап тәүге адым ясар,
Гафулар ит, диеп, әйтә (а)лыр?..
Гафулар ит, диеп, әйтә (а)лыр?..
```
