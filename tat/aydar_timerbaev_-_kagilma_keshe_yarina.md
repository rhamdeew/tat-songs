# Оригинал

### Айдар Тимербаев - Кагылма кеше ярына

```
Ничек кенә теләсәк тә
Гаептән сакланырга,
Булгалый шул күңелләрнең
Адашкан чаклары да.
Кайчак чит бакча гөлләре
Матурырак күренә,
Лэкин тыя бел үзеңне,
Үрелмә син, үрелмә!

Кагылма кеше ярына,
Уйлап кара тагын бер.
Синең ярыңа да шулай
Кагылучы табылыр.

Әллә ярыңа беркем дә
Кызыкмый дип беләсең?
Син аны күрә белгәндәй
Ник бүтәннәр күрмәсен?
Читтән җылы эзләүчеләр
Адым саен табыла.
Кирәк бары мизгел генә,
Чыксын бары җай гына.

Кагылма кеше ярына,
Уйлап кара тагын бер.
Синең ярыңа да шулай
Кагылучы табылыр.

Әллә күз салмас дисеңме
Насыйп ярың бүтәнгә?
Син читтә йөргәнне белсә,
Аңа нәрсә көтәргә?

Кагылма кеше ярына,
Уйлап кара тагын бер.
Синең ярыңа да шулай
Кагылучы табылыр.
```
