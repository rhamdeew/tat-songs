# Оригинал

### Шәмсия Җиһангирова / Марат Мухин - Әллә нәрсә булды миңа быел язда

```
Кыйгач-кыйгач кошлар парлап очып үтте,
Кайтулары туган яккадыр.
Бәлки, ярсуларым юкка гына булган,
Яну-көюләрем юккадыр.

Әллә нәрсә булды миңа быел язда,
Чәчәк чагым, алма чакларым.
Аклык кунды бугай, кунды минем җанга
Аклыгыннан алмагачларның.

Борылышта безне бәхет көтеп тора,
Мин сулышын аның сизенәм.
Шундый изге хисләр татыр идем мени,
Син булмасаң әгәр, бергенәм.

Әллә нәрсә булды миңа быел язда,
Чәчәк чагым, алма чакларым.
Аклык кунды бугай, кунды минем җанга
Аклыгыннан алмагачларның.

Кулларымны, әйдә, ал кулларыңа,
Бәхетләргә каршы йөгерик.
Икәү генә бары, икәү генә, бәгърем,
Чын сөюнең тәмен без белик.

Әллә нәрсә булды миңа быел язда,
Чәчәк чагым, алма чакларым.
Аклык кунды бугай, кунды минем җанга
Аклыгыннан алмагачларның.
```
