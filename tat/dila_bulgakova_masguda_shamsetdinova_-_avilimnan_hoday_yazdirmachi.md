# Оригинал

### Дилә Булгакова / Мәсгудә Шәмсетдинова - Авылымнан, Ходай, яздырмачы

```
Кайталмыйча торсам туган якка
Сагышларга кереп уралам,
Авылымнан, Ходай, яздырмачы,
Башка Синнән берни сорамам.
Шомырт исе килә урманнардан,
Сирень исе аңкый урамнан,
Авылымнан, Ходай, яздырмачы,
Башка синнән берни сорамам.

Әгәр килсәң минем туган якка,
Каршы алыр чиксез кырларым,
Күкләр кочып, урман моңын тыңлап
Отарсың син сөю җырларын.
Шомырт исе килә урманнардан,
Сирень исе аңкый урамнан,
Авылымнан, Ходай, яздырмачы,
Башка синнән берни сорамам.

Кунакчыл як — минем туган ягым,
Каршы алыр дөнья баласын,
Авылым бишегендә тибрәләнеп
Балачакта торып каласың.
Шомырт исе килә урманнардан,
Сирень исе аңкый урамнан,
Авылымнан, Ходай, яздырмачы,
Башка синнән берни сорамам.
```
