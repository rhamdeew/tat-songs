# Оригинал

### Суфиян Поварисов / Рәшит Җиһанов - Авылымның айлы кичләре

```
Язгы моңнар чишмә кебек агыла,
Шаулап тора басу эшләре;
Нурлар бөркеп кырдан каршы ала
Авылымның айлы кичләре.

Урамнарда тәмле булып аңкый
Яңа пешкән икмәк исләре.
Иркәләнә гармун моңнарында
Авылымның айлы кичләре.

Синдә күңелем, уем, тойгыларым,
Йөрәгемнең кайнар хисләре.
Чатнап торган салкында да матур
Авылымның айлы кичләре.
```
