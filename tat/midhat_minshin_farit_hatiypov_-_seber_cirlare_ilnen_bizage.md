# Оригинал

### Мидхәт Миншин / Фәрит Хатыйпов - Себер җирләре — илнең бизәге

```
Себер юллары
Булган моң — зарлы,
Әле дә онытылмый
Богау чыңнары.

Инде ул көннәр
Калды еракта
Өр — яңа җырлар
Туа бу якта:

Себер җирләре —
Илнең бизәге.
Монда килмичә
Йөрәк түзәме?

Килә Бөгелмә,
Килә Минзәлә
Изге ашкыну
Балкый йөзләрдә

Дан алган ирләр
Әлмәт, Шөгердә
Нефть чыгара
Бүген Себерде

Себер җирләре —
Илнең бизәге
Монда килмичә
Йөрәк түзәме?

Эшкә — ярышка
Нинди киң мәйдан
Себер үзе дә
Туа яңадан.

Ерак елларга
Бүләк булыр ул
Килер көннәрне
Зурлап торыр ул

Себер җирләре —
Илнең бизәге
Монда килмичә
Йөрәк түзәме?
```
