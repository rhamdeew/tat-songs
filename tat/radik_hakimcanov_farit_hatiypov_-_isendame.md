# Оригинал

### Радик Хәкимҗанов / Фәрит Хатыйпов - Исеңдәме?!

```
Исеңдәме һаман, ул чәчәкле ялан,
Такыяләр үрдек бергәләп,
Шатлыктан без очтык, бар дөньяны кочып,
Син идең бер — сихри күбәләк.

Күпме еллар үтте, кабат җәйләр җитте,
Килдем тагы шушы болынга,
Сине эзли күңел, онытырлык түгел,
Нигә әле моңсу бик монда?!

Соңдыр инде димә, күңел һаман синдә,
Килче, иркәм, килче яныма,
Алып кайтчы мине, еллар аша кире,
Яшьлегемнең алсу таңына!!!
```
