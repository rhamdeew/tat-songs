# Оригинал

### Фәрхәт Кунафин / Ризван Хәкимов - Тагын җырлап үтәм

```
Кайчан гына әле ярсып-шашып,
Йокыларын бүлеп кызларның,
Көмеш телле гармун моңнарына
Куша идек сөю җырларын.

Өй, туган як, туган як,
Гел җырлардан торган як.

Элеп алыр иде сандугачлар.
Гашыйк булып безнең җырларга.
Нинди хисләр дәртләндерде икән
Моңландырып гармун уйнарга?

Әй, туган як, туган як,
Моңландыра торган як.

Шул мәлләргә тагын ярсып кайттым
Тик кайда соң гармун-сандугач?!
Җүләрләнгән диеп уйламагыз,
Тагын җырлап.үтәм сагынгач.

Әй, туган як, туган як,
Сагындыра торган як.
```
