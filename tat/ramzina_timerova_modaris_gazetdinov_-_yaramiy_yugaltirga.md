# Оригинал

### Рәмзинә Тимерова / Мөдәрис Газетдинов - Ярамый югалтырга

```
Гомерем агышларымнан,
Таптым да, югалттым да.
Өзелеп сөйгән ярларны,
Ярамый югалтырга.

Ялгыз аккошлар шикелле,
Булмасын язмышкаем.
Үкенеп уйларга язмасын,
Мәхәббәт ялгышларын.

Кичер алмас, киң утлар күп,
Кирәккә бу итәргә.
Нигә соң йөрәк яраңны,
Калдырып та китәргә.

Ялгыз аккошлар шикелле,
Булмасын язмышкаем.
Үкенеп уйларга язмасын,
Мәхәббәт ялгышларын.

Тынычландыр күңелеңне,
Ашыкма, сабырлар ит.
Күптәр гомерен үздыра,
Сөйгәнен табырга дип.

Ялгыз аккошлар шикелле,
Булмасын язмышкаем.
Үкенеп уйларга язмасын,
Мәхәббәт ялгышларын.
```
