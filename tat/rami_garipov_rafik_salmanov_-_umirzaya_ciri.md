# Оригинал

### Рәми Гарипов / Рафик Сәлмәнов - Умырзая җыры

```
Дивана мин, дивана,
Дөнья белән куанам (юанам).
Дөнья минем — үз өем,
Дөнья моңы — үз көем.

Үзем ничек телимен,
Җирдә шулай яшимен.
Бер кемгә юк зыяным, —
Мин тик умырзаямын.

Һәр терегә сөенәм,
Тик үлемгә көенәм;
Ә хәзергә — куанам,
Диванам мин, дивана!..
```
