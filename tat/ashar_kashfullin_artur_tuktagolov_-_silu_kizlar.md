# Оригинал

### Әсхәр Кәшфуллин / Артур Туктаголов - Сылу кызлар

```
Эх, сез кызлар, нурлы күзләр,
Ут чәчрәтеп үтәсез.
Мәхәббәтле караш белән,
Йөрәкләрне өтәсез.

Эх, сез, сылулар,
Карагыз безгә.
Йөрәкләргә ялкын өстәп,
Үтәсез  гөлдә.
Эх, сезгә сылулар,
Сөю уты кабынган.
Бу утларның ялкынында,
Безнең сөю салынган.

Күңелләрне җилкендереп,
Сөю назын уятып.
Сез үткәндәй ялкын чәчеп,
Калам мин хисләргә батып.

Эх, сез, сылулар,
Карагыз безгә.
Йөрәкләргә ялкын өстәп,
Үтәсез  гөлдә.
Эх, сезгә сылулар,
Сөю уты кабынган.
Бу утларның ялкынында,
Безнең сөю салынган.
```
