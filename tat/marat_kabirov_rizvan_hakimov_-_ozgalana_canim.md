# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Өзгәләнә җаным

```
Өзгәләнә җаным,
Өзгәләнә тагын.
Телгәләнә йөрәк, теленә.

Өзелергә торган
Сары яфрак сыман
Кереп барам гомер көзенә.

Көзге яфрак сыман
Сагыш белән тулган
Язмышларга ничек түзәсе?!.

Берсен-берсе өзелеп,
Өзелеп сөйгәннәрнең
Бәхетләре булмый, күрәсең.

Өзгәләнә җаным,
Өзгәләнә тагын.
Нәрсә җитми икән бу җанга?!.

Әллә сине өзелеп,
Өзелеп яратканга,
Әллә гомер синсез узганга.
```
