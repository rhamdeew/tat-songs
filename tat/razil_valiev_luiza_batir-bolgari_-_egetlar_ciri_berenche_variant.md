# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Егетләр җыры (беренче вариант)

```
Әйдә барыйк, кызлар карыйк, —
Орчык эрләгәннәрен;
Аяз көнне яшен суксын
Безне сөймәгәннәрен.

Әйдә барыйк, кызлар карыйк,
Керик куеннарына.
Асылынсак, асылыныйк
Кызлар муеннарына.

Әйдә барыйк, кызлар карыйк
Казан ярминкәсендә.
Сынатмасын егет кеше
Мәхәббәт өлкәсендә.
```
