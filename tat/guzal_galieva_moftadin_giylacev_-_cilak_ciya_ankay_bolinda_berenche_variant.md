# Оригинал

### Гүзәл Галиева / Мөфтәдин Гыйләҗев - Җиләк җыя әнкәй болында (беренче вариант)

```
Ак яулыгын артка бәйләп,
Пешкәннәрен генә сайлап,
Җиләк җыя әнкәй болында.
Чиләге дә матурырак
Җиләге дә алсуырак
Күренә бит аның кулында.

Безне назлап үстердең син,
Яшәр өчен көч бирдең син
Кояш кебек балкыткан саен.
Гомеремнең һәр бер язы,
Ак чәчәкле җиләк таҗы,
Ак яулыклы җиләк әнкәем.

Нинди рәхәт — җирдә җәйләр!
Нинди бәхет — сау әнкәйләр!
Күңелемдә нурлар бәйләме.
Туган җирем — гөл-чәчәктә.
Күберәк булсын киләчәктә,
Әнкәй, җиләк җыйыр җәйләрең!
```
