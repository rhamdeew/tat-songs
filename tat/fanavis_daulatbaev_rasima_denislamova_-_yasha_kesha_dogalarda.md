# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Яшә кешә, догаларда!

```
Тузан түгел, селкеп кенә
Гөнаһларны коеп булмас.
Ялгышларны төзәтергә
Өр-яңадан туып булмас.
Бу буталчык дөньясында
Ялгышмый да яшәп булмас.
Җирдә безгә-бәндәләргә
Иман-тәүфыйк булсын юлдаш!

Ничек яшәвебез җирдә —
Үзебездән тора бар да.
Киңәшем шул — азма, тузма,
Яшә кешем, догаларда!

Аллаһыга ялварганда
Кичерелер ялгышыбыз.
Чын кешеләр булып яшик,
Тапланмасын намусыбыз!
Иртә-соңмы кешеләргә
Бакыйлыкка китеп барыш.
Кемнәр генә булсак та без
Игелекләр кылу фарыз.

Ничек яшәвебез җирдә —
Үзебездән тора бар да.
Киңәшем шул — азма, тузма,
Яшә кешем, догаларда!

Игелекләр кылу фарыз —
Аллаһының хак таләбе.
Иман белән яшәгәнне
Оҗмах көтә ул адәмне.
```
