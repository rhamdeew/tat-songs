# Оригинал

### Гөлсинә Сабирова / Флүрә Талипова - Җырларда язмышлар

```
Җырларыбыз — безнең язмышыбыз,
Матурлыкны җиргә тарату.
Шәфәк уты урап алган сыман
Өзөлөп-өзөлөп яну, ярату.

Язмышларда, бары язмышларда
Язылган ул, диләр, ялгышлар.
Ялгышларны, язмышларны үзгәрт,
Үз кулыңда синең көч-көдрәт.

Күрә белсәң, күз яшеңдә нурлар,
Син бәхетле, бәхет кулыңда.
Аера алсаң акны һәм караны,
Якын дуслар очрар юлыңда.

Язмышларда, бары язмышларда
Язылган ул, диләр, ялгышлар.
Ялгышларны, язмышларны үзгәрт,
Үз кулыңда синең көч-көдрәт.

Һаваларны очып үтәр өчен
Пар канатлар кирәк кошларга.
Ышыкланыр өчен ышанырдай
Оя кирәк сайрар кошларга.

Язмышларда, бары язмышларда
Язылган ул, диләр, ялгышлар.
Ялгышларны, язмышларны үзгәрт,
Үз кулыңда синең көч-көдрәт.
```
