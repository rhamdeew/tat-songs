# Оригинал

### Раушан Садыйков / Таһир Фәйзрахманов - Әтнә гүзәлләре

```
Мактала безнең Әтнөбез
Эш сөйгән халкы белән.
Саф күңеле, нурлы йөзе,
Ягымлы сүзе белән.

Ашыт суы буйларын,
Тукай йөргән юлларын,
Әтнә ягы гүзәлләрен
Күреп күзем туймады.

Әтнә ягы җыр-биюле,
Мәңге якын күңелгә.
Шуңа да аның шөһрәте
Таралды бөтен илгә.

Ашыт суы буйларын,
Тукай йөргән юлларын,
Әтнә ягы гүзәлләрен
Күреп күзем туймады.

Йөрәгеңә тиң булырлык
Мәхәббәт — синең телең.
Игенчеләр, терлекчеләр
Яшәгән туган җирем.

Ашыт суы буйларын,
Тукай йөргән юлларын,
Әтнә ягы гүзәлләрен
Күреп күзем туймады.
```
