# Оригинал

### Әхмәт Әһлиуллин / Ф.Әхмәдуллин - Сирень чәчәкләре (Ак сиреннәр)

```
Мине сагынып килер юлларыма
Сирень чәчәкләрен сибәрсең.
Мин яратам, бәгърем, сирень чәчәкләрен,
Берсен миңа өзеп бирерсең.

Язлар җитте, ямьле җәйләр килде,
Сирень чәчәкләре сибелде.
Мин яратам, бәгърем, сирень чәчәкләрен,
Берсен миңа өзеп бирерсең.
```
