# Оригинал

### Равил Фәйзуллин / Риф Гатауллин - Нигә шулай…

```
Нигә шулай багам һаваларга,
Канатларым юк ла иңемдә?
Туган якка кайткан кошлар юлы
Үтә мәллә минем күңелдән?

Нигә шулай тыңлыйм искән җилне,
Әллә берәр хәбәр көтәмме?
Хәбәр түгел, көтәм кошларымның
Талга басып сайрар иртәне.

Нигә шулай тансык кошлар өне,
Кош сайраудан сагыш кимерме?
Кошлар булып, ахры, җаным җырлый
Искә алып үткән гомерне.
```
