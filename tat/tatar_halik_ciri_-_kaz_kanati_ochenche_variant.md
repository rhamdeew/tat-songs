# Оригинал

### Татар халык җыры - Каз канаты (өченче вариант)

```
Каз канаты кат-кат була,
Ир канаты ат була;
Чит җирләрдә бик күп йөрсәң,
Туган илең ят була.

Каз канатлары кавырсын
Иркәм авыр — сабырсың.
Бик авырсаң, бик сабырсың
Ахры ятка калырсың.

Каз канатларын санадым
Тезелеп кагынганда.
Инде кемнәргә карармын
Өзелеп сагынганда?
```
