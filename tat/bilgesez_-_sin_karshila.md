# Оригинал

### Билгесез - Син каршыла

```
Минем сөю башлангандыр, бәлки,
Тәрәзәңә кар аткан көннәр.
Миннән башка тагын кем соң бар әле
Сине болай ярата белгән?

Урамнарны аралап, мин
Томаннарны аралап, мин
Яңгырларны аралап, мин,
Ашыгамын сиңа, сиңа.

Кояшлы елмаюларың белән
Син каршыла, син каршыла.

Синең сөю башлангандыр, бәлки,
Яз гөлләрен тараткан көннәр.
Синнән башка тагын кем соң бар әле
Мине болай ярата белгән?
```
