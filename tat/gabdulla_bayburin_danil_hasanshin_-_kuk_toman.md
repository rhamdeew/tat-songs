# Оригинал

### Габдулла Байбурин / Данил Хәсәншин - Күк томан

```
Киләм Идел буйларына
Идел буе күк томан.

Иделдә күк томан,
Күңелдә күк томан.
Мин сине юксынам.

Таңнар атыр,
Таң җилләре
Күк томанны таратыр.

Тик шушы иртәдә,
Кем сине иркәләр,
Кем сине яратыр?

Ятлар өчен яр сөймәдем,
Сер бирмәдем гомергә.

Иделдә күк томан,
Күңелдә юк томан.
Без бергә, гел бергә.
```
