# Оригинал

### Галимулла Нәбиев / Илдус Низамов - Яшьлек кыңгыраулары

```
Кыңгыраулы юргалар кебек
Чыңлап узды яшьлек елларым.
Ят таңнарга омтылды күңел,
Үз бәхетем читтән юлладым.

Нигә яшьлекнең юллары урау?
Ерак-еракта чыңлый кыңгырау…

Басып торам юллар чатында,
Уйлата шул синең хакыңда.
Кыңгыраулар, җиз кыңгыраулар
Моң уйната һаман якында.

Синнән хәбәрме моңлы яңгырау?
Сәламнәр әйтеп чыңлый кыңгырау.

Кыңгыраулар чыңлый якында,
Борма-борма юллар чатында.
Чәчәк булып кая атылма,
Тик онытма сөю хакында.

Күңелдә һаман чыңлый яңгырау,
Кавыштырырмы җырлы кыңгырау?..
```
