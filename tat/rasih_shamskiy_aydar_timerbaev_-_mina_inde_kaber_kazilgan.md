# Оригинал

### Расих Шамский / Айдар Тимербаев - Миңа инде кабер казылган

```
Берәр шуклык эшләп куйсам әгәр,
Уйламагыз бу, дип, азынган.
Яңа гына яши башласам да (2 тапкыр)
Миңа инде кабер казылган. (2 тапкыр )
Күз алдымда кара кабер тора,
Тактасы да әзер ләхетнең.
Бу дөньядан китеп барам инде, (2 тапкыр)
Татымыйча бәгьзе бәхетне. (2 тапкыр)
Ә яшисем бик-бик килә иде,
Дөнья рәхәт, дөнья матур бит.
Әй, сөеклем, мин кабердә инде, (2 тапкыр)
Өмет итмә, көтмә кайтыр дип. (2 тапкыр)
Өмет итмә, көтмә кайтыр дип…
```
