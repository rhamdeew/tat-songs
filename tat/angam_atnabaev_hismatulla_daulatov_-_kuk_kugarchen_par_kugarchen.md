# Оригинал

### Әнгәм Атнабаев / Хисмәтулла Дәүләтов - Күк күгәрчен, пар күгәрчен

```
Гөлдер-гөлдер, гөлдер-гөлдер
Гөрелдәй күк күгәрчен.
Күк күгәрчендәй гөрелдәп,
Төшләремә керәсең,
Төшләремә керәсең дә,
Куларыңны бирәсең.
Чәчләр-чәчәк, чәчләр-чәчәк
Бәйләнгәндер күрәсең.

Гөлдер-гөлдер, гөлдер-гөлдер
Парлаша күк күгәрчен.
Пар күгәрчендәй бездә бер
Парлашырбыз күрәсең.
Парлашырбыз гомерлеккә
Җандай тарткан ярлар без,
Өзелешеп, сер бүлешеп,
Гөр килешеп яшәрбәз.
```
