# Оригинал

### Фәнәвис Дәүләтбаев / Вәгыйзь Газизов - Җанда мәхаббәт барда

```
Матур итеп үткәрик
Тик бер генә гомерне.
Бер-береңне яратып
Яшәүләре күңелле!

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!

Күңелдә сөю барда
Уртак теләк-уйлар да.
Иңгә иң куеп, иркәм,
Атлыйк әле без алга!

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!

Булсын куллар кулларда,
Чыгыйк изге юлларга.
Бәхет ул җирдә яшәү
Җанда мәхәббәт барда.

Бик бәхетле чакларым,
Син булганга шат җаным.
Син генә яратканым,
Эзләгәнем, тапканым!
```
