# Оригинал

### Наил Габдүшев / Хәмис Азмуханов - Йөрәгемне яндырма

```
Аяз көндә яшен сукты —
Гашыйк булым мин сиңа.
Тигез атлап китер кебек
Икәү гомер буена.

Шаярма син моңсу булып,
Күзләремә карама.
Сөям диеп башкаларны
Йөрәгемне яндырма.

Бүген тагын айлы кичтә
Озатып кайттым сине.
Карадым мин күзләреңә,
Белә алмадым сөреңне.

Шаярма син моңсу булып,
Күзләремә карама.
Сөям диеп башкаларны
Йөрәгемне яндырма.

Бүләк итәм ал чәчәкләр,
Бәхет телим гомергә.
Җәйге кичләр истә калсын,
Күңелеңдә мәңгегә.

Шаярма син моңсу булып,
Күзләремә карама.
Сөям диеп башкаларны
Йөрәгемне яндырма.
```
