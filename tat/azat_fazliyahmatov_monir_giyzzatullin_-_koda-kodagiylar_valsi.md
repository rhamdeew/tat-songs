# Оригинал

### Азат Фазлыйәхмәтов / Мөнир Гыйззәтуллин - Кода-кодагыйлар вальсы

```
Кан кардәшләр булмасак та
Без туганнар чын-чынлап.
Туганлаштык сезнең белән
Балаларга туй туйлап.

Бәйрәм бүген, без җыелдык
Зур мәҗлес-табынга.
Бәхет-шатлыклары белән
Яшик әле тагын да.

Парлы булсын гомеребез,
Парлы үтсен юлыбыз.
Тик бәхетле була күрсен
Улыбыз hәм кызыбыз.

Бәйрәм бүген, без җыелдык
Зур мәҗлес-табынга.
Бәхет-шатлыклары белән
Яшик әле тагын да.

Уртак булыр сынау килсә,
Уртак безнен шатлыклар.
Жаннар тыныч, имин тормыш,
Көтсен тигез картлыклар.

Бәйрәм бүген, без җыелдык
Зур мәҗлес-табынга.
Бәхет-шатлыклары белән
Яшик әле тагын да.

Уңыш hәрчак юлдаш булсын,
Бәхет килсен өелеп.
Оныкларны сөя-сөя,
Яшик hәрчак сөенеп.

Бәйрәм бүген, без җыелдык
Зур мәҗлес-табынга.
Бәхет-шатлыклары белән
Яшик әле тагын да.
```
