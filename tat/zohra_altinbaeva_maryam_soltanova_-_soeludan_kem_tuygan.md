# Оригинал

### Зөһрә Алтынбаева / Мәрьям Солтанова - Сөелүдән кем туйган?!

```
Каршымда сүрәтең тора,
Искә төшермәк булдым.
Сине очратып уңдыммы мин,
Әллә бөтенләй туңдым?!

Мәхәббәтем терәк миңа
Янып-көйөнгәнемдә.
Уңсам да туңсам да риза
Сөеп-сөелгәнемә.

Онытылмас ядкәр булып,
Яшьлек елларым узган.
Күңел генәм әле дә яшь,
Сөелүдән кем туйган?!
```
