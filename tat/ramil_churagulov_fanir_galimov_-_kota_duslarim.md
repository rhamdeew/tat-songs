# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Көтә дусларым

```
Бисмиллалар әйтеп, юл догасын укып,
Чыгып китәм озын юлларга.
Озын юллар хәтәр, бәлә килсә әгәр,
Таянам мин якын дусларга.

Җиде юллар чатында
Җан дуслар көтеп торсын.
Кайнар чәйләр белән сыйлап,
Җырлап, озатып куйсын.

Тормышыбыз кайный, юл йөреми булмый,
Мең эшләр туа көн саен.
Сагынып көтә өйдә, сагынып көтә көн дә,
Балаларым, газиз җанкаем.

Җиде юллар чатында
Җан дуслар көтеп торсын.
Кайнар чәйләр белән сыйлап,
Җырлап озатсын.

Такыр булсын юллар, юлдаш булсын җырлар.
Имин туып килсен һәр еллар.
Гомерлек дус булсын, юлларда очрасын,
Үзеңнән дә шәберәк җан дуслар.

Җиде юллар чатында
Җан дуслар көтеп торсын.
Кайнар чәйләр белән сыйлап,
Җырлап, озатып куйсын.
```
