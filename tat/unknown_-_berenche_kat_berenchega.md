# Оригинал

### Рамил Чурагулов / Эльвира Хаматнурова - Беренче кат беренчегә

```
Үзем хәтлем портфель тотып
Мәктәпкә китеп барам
Күп саннарны,хәрефләрне
Өйрәтер матур апам

Кушымта:
Беренче кат беренчегә
Китеп барам,дусларым
Җиңел булмас яулап алу
Биек белем тауларын
Беренче кат мэктәбемә
Китеп барам ,дусларым
Сау-саләмәт,җырлап-гөрләп
Узсын мәктәп еллары!

Барда яңа,барда серле
Сихри дөнҗя мәктәбем
Гел бишлегә генә тулсын
Көн саен көндәлегем
```
