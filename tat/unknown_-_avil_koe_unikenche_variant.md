# Оригинал

### Татар халык җыры - Авыл көе (уникенче вариант)

```
Карар инде,карар инде
Кара күзләре белән
Үзәгемне өзәр инде
Әйткән сүзләре белән

Гөлбачасына керәм дә
Эзлимен сөйгәнемне шул
Эзлимен сөйгәнемне
Эзләмә,табалмассың дип
Өзәләр үзәгемне

Тауга менәм,ташка басам
Су сибәм гөлләремә шул
Су сибәм гөләремә.
Мәңге онытылмаслык булып
Калдың күңелләремдә.
```
