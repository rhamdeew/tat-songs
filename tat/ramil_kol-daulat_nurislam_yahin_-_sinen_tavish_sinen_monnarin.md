# Оригинал

### Рамил Кол-Дәүләт / Нурислам Яхин - Синең тавыш, синең моңнарың

```
Шул тиклем дә якын сихри моңны
Аерамын мең-мең тавыштан,
Шуңа да мин төнлә юллар табып,
Сиңа киләм алыс алыстан.

Күңелемдә әгәр яши икән
Синең тавыш, синең моңнарың,
Бу дөньяда, юк-юк, адашмамын,
Сиңа гына илтер юлларым.

Шул тиклем дә якын сихри моңны
Аерамын мең-мең тавыштан,
Шуңа да мин төнлә юллар табып,
Сиңа киләм алыс алыстан.
```
