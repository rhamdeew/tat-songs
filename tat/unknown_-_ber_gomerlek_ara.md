# Оригинал

### Венер Котдусов - Бер гомерлек ара

```
Әллә язмыш микән, әллә ялгыш микән
Очрашулар, танышу-кав(ы)шулар.
Еллар судай ага, бер гомерлек ара,
Дөнья белән узышу-ярышулар.

Ерак-ерак офыкларда хыял уты, учак яна
Хәтерләтеп үткән гомерне.
Йөрәкләрдә сөю һаман, әллә дәва, әллә яра,
Әллә сүнми калган күмерме.

Уңы уң булмады, сулы сул булмады,
Бәхет көлеп баштан сыйпады.
Җан тарткан ярләр дә, күз катып ятларга,
Каршы ярдан сәлам юллады.

Ерак-ерак офыкларда хыял уты, учак яна
Хәтерләтеп үткән гомерне.
Йөрәкләрдә сөю һаман, әллә дәва, әллә яра,
Әллә сүнми калган күмерме.
```
