# Оригинал

### Филүс Хисаметдинов / Рәдиф Зарипов - Бәгърем

```
Җаннарымда тибә наз чишмәсе,
Әллә инде ихлас сөюдән.
Туңган йөрәгемнең бозы эри
Яратамын дигән сузеңнән.

Яраланган йөрәгемне төрдең назга,
Коендырдың татлы назларга.
Гомерем буе парын көткән җаным
Тынычланды ярын тапканга.

Калган гомеремне синсез, бәгърем,
Уздыралмам инде бер көн дә.
Язмышым син, ышанычым булдың,
Тик янымда булчы гомергә.

Яраланган йөрәгемне төрдең назга,
Коендырдың татлы назларга.
Гомерем буе парын көткән җаным
Тынычланды ярын тапканга.

Икәү бергә татып алсак икән
Мәхаббәтнең бәхет алмасын.
Язмышымны, Ходай, парлы иттең,
Ялгызлыкка берүк салмасын.

Яраланган йөрәгемне төрдең назга,
Коендырдың татлы назларга.
Гомерем буе парын көткән җаным
Тынычланды ярын тапканга.
```
