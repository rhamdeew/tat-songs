# Оригинал

### Рамил Чурагулов / Алмаз Гафаров - Ана һәм бала

```
Ана:

Җиргә бетмәс ямнәр өстәп
Дөнъяга килдең кызым.
Куанычым, таянычым,
Минем якты йолдызым.

Бала:

Син үзен дә, әнкәем,
Мәнге яшь, мәңге нурлы.
Синен белән дөнъя якты.
Һәм нурлы йортның түре.

Ана:

Бәхетле бул, тәүфыйклы бул,
Сәламәт бул, әй, балам,
Сәламәт күңел һәм тәннән,
Балкылыплар тора иман.

Бала:

Чынга ашсын теләкләрең,
Ихластан догаларын,
Ярдәм итсен адым саен
Синең фатыйхаларын.

Ана:

Куанычым, таянычым,
Минем якты йолдызым,
Минем якты йолдызым,
Кызым, кызчыгым!
```
