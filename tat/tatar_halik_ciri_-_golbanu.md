# Оригинал

### Татар халык җыры - Гөлбану

```
Ак бүрегем бар дисең,
Нигә аны кимисең?
Сөйгән ярым бар дисең,
Нигә аны сөймисең?

Ай-яй-яй, Гөлбану,
Көн дә яныңда булам!
Көндә-көндә, айлы төндә
Яннарыңда мин булам!

Әгәр килсәм иртәсен
Тагын мине кертәсең.
Мондый алсу йөзеңне
Үпми ничек китәсең?

Ай-яй-яй, Гөлбану,
Көн дә яныңда булам!
Көндә-көндә, айлы төндә
Яннарыңда мин булам!

Ишек алды тал-тирәк.
Талга сандугач кирәк.
Озын буйлы егетләргә
Кыска буйлы кыз кирәк.

Ай-яй-яй, Гөлбану,
Көн дә яныңда булам!
Көндә-көндә, айлы төндә
Яннарыңда мин булам!

Ишек алды түгәрәк.
Очып йөри күбәләк.
Берсе — озын, берсе — кыска
Үбешәләр чүгәләп.

Ай-яй-яй, Гөлбану,
Көн дә яныңда булам!
Көндә-көндә, айлы төндә
Яннарыңда мин булам!
```
