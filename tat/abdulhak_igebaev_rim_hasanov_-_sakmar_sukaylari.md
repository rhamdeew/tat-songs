# Оригинал

### Абдулхак Игебаев / Рим Хәсәнов - Сакмар сукайлары

```
Сакмар сукайлары җырлап ага,
Йөрәктә тибрәтә тавышын,
Чыңгырап килгән моңны ишеттемме —
Тарала да китә сагышым.

Сакмар сукайлары талгын ага,
Ярсу дулкыннарын уйнатып,
Дулкыннардай ярсу яшьлегемнең
Гүзәл хәтирәсен кузгатып!

Сакмар сукайлары коеп ага,
Җил иркәли сылу талларын,
Оныта алмам инде Сакмарымның
Былбыл сайрап торган таңнарын.
```
