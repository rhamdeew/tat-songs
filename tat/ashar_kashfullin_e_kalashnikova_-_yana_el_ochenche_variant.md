# Оригинал

### Әсхәр Кәшфуллин / Э. Калашникова - Яңа ел (өченче вариант)

```
Яңа елда җанны пакьлек бизи,
Ак карлардан күзләр чагыла.
Көтелмәгән ак бәхетләр килеп,
Тартып алыр кебек янына.
Йолдызлардан энҗе бер коелып,
Ак чәчәкләр иңгә кагыла.
Яңа елда татлы хис уянып,
Күңелләргә назлар агыла.

Бәйрәм хисе үтеп керә күңелләргә,
Дуслар булган чакта яныңда.
Көтелмәгән бәхет, бәхет килер кебек
Яңа елның якты таңында.

Яңа елда гашыйк җаннар өчен
Сөю уты дөрләп кабына.
Еллар узган саен сагышлы җан
Яшьлек мәхәббәтен сагына.
Йолдызлардан энҗе бер коелып,
Ак чәчәкләр иңгә кагыла.
Яңа елда якты хис уянып,
Күңелләргә назлар агыла.

Бәйрәм хисе үтеп керә күңелләргә,
Дуслар булган чакта яныңда.
Көтелмәгән бәхет, бәхет килер кебек
Яңа елның якты таңында.
```
