# Оригинал

### Людмила Аланлы / Хәмид Мәткәримов - Баллы сөю

```
Бал кортлары бал җыя,
Кәрәзләргә коела.
Чын күңелдән әйткән сүзең
Балдан татлы тоела.

Татлыдан татлы балым син,
Йөрәккәем, назым син.
Күреп тә, сөеп тә туймас
Җанга якын ярым син.

Бал кортлары уңганлыгын
Алдың мәллә үзеңә?
Татлы сөю кәрәзенә
Эләктең бит, түз генә!

Татлыдан татлы балым син,
Йөрәккәем, назым син.
Күреп тә, сөеп тә туймас
Җанга якын ярым син.

Бердәнберем — син, диеп әйт,
Яшермичә сүзеңне.
Тиңем итеп сайлаганга
Яратам ич үзеңне!

Татлыдан татлы балым син,
Йөрәккәем, назым син.
Күреп тә, сөеп тә туймас
Җанга якын ярым син.
```
