# Оригинал

### Әхмәт Гадел / Ренат Шәрибҗанов - Гомерем буе эзләдем

```
Болыннарда йөрдем,
Урманнарга кердем.
Чәчәкле аланнарда
Сине эзләп йөрдем.

Кайда икән эзләрең,
Җылы-нурлы күзләрең?
Синең назлы карашыңны
Гомерем буе эзләдем.

Көн йомгагын сүтте,
Аһ, елларым үтте…
Гомер канатларыннан
Каурый очты-китте.

Чәчләрем агарды,
Башкаем чаларды.
Янгыныма һичбер чара
Таба алмый арыдым.

Кайда икән эзләрең,
Җылы-нурлы күзләрең?
Синең назлы карашыңны
Гомерем буе эзләдем.
```
