# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Эх, туйлары!

```
Тирә-юнь көлеп тора,
Яшь парлар гаилә кора.
Гөрләп торып тирә якны,
Алар бит туй уздыра.

Эх,туйлары, туйлары!
Сезгә котлау җырлары.
Гомерлеккә бергә дигән,
Гомерлеккә бергә дигән,
Кабул булсын уйлары.

Кулны алмый куллардан,
Барып якты юллардан
Җитәкләшеп атлагыз сез,
Калышмый хыяллардан.

Эх,туйлары, туйлары!
Сезгә котлау җырлары.
Гомерлеккә бергә дигән,
Гомерлеккә бергә дигән,
Кабул булсын уйлары.

Килдек сезне котларга,
Котлап җырлар җырларга.
Изге матур теләкләрне
Кушам шушы моңнарга.

Эх,туйлары, туйлары!
Сезгә котлау җырлары.
Гомерлеккә бергә дигән,
Гомерлеккә бергә дигән,
Кабул булсын уйлары.
```
