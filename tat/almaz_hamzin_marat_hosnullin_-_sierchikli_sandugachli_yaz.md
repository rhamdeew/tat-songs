# Оригинал

### Алмаз Хәмзин / Марат Хөснуллин - Сыерчыклы, сандугачлы яз

```
Ал кояшта безнең белән
Уянып көтәм.
Гел елмая, гел үзенчә,
Гел-гел үзенчә.
Уйный-көлә, назлый-сыйпый
Иркәли генә.
Андый кояш башка җирдә
Юк ул минемчә.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.

Һәр кемнең үз ягы җырлы,
Моңлы, гүзәлле.
Кызлар ул сылулар,
Чиксез чибәрләр.
Язлары да, көзләре дә
Бигрәк күңелле.
Ай йолдызы кояшына,
Күңел үрелде.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.

Сезнең якта ай сагышлы,
Нигә сагышлы?
Ул бит ялгыз, бер ялгызы,
Япа-ялгызы.
Сары сагыш, ачы язмыш
Безгә язмасын.
Мин елмаеп айга карыйм,
Ул да елмайсын.

Сыерчыклы, сандугачлы,
Тургайлы һәм гармунлы.
Сагынам-сагынам ал кояшлы
Туган үскән ягымны.
```
