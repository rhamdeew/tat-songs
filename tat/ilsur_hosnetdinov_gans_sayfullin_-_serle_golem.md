# Оригинал

### Илсур Хөснетдинов / Ганс Сәйфуллин - Серле гөлем

```
Тәрәзәләргә карыйм,
Яран гөлләрен саныйм.
Шул гөлләрнең нурында
Алсу йөзеңне таныйм.

Серле гөлем, сөйлә серең,
Тәрәзәдә күрермен.
Әгәр елмаеп карасаң,
Килеп сулар сибәрмен.

Язның кояшлы нурлары
Үтеп керер тәрәзәдән.
Үсәрсең син нурлар белән,
Әйт, сагыну нәрсәдән?..

Серле гөлем, сөйлә серең,
Тәрәзәдә күрермен.
Әгәр елмаеп карасаң,
Килеп сулар сибәрмен.

Су сибәрмен сиңа гөлем,
Моңлы минем күңелем.
Үс син, гөлем, сөйлә серең,
Яннарыңа килермен.

Серле гөлем, сөйлә серең,
Тәрәзәдә күрермен.
Әгәр елмаеп карасаң,
Килеп сулар сибәрмен.
```
