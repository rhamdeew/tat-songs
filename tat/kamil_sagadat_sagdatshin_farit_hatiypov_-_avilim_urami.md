# Оригинал

### Камил Сәгадәт (Сәгъдәтшин) / Фәрит Хатыйпов - Авылым урамы

```
Мин ачылып килгән чәчәк,
Син да бөредә генә.
Кулларыңа кагылу кая,
Киләм күрергә генә.

Авылым урамы кыска,
Бар булганы җиде өй,
Һаман яңгырый күңелдә,
Нинды моң бу, нинди көй?!

Сине башкалар үпкәнгә,
Рәнҗүем күптән үткән,
Яшьлекнең җыры кыска дип,
Әйтмәгез генә бүтән.

Авылым урамы кыска,
Бар булганы җиде өй,
Һаман яңгырый күңелдә,
Нинды моң бу, нинди көй?!

Югалтмас өчен ярыңны,
Кыю булу да кирәк.
Безнең яшьлекне сагына
Бергә утырткан тирәк.

Авылым урамы кыска,
Бар булганы җиде өй,
Һаман яңгырый күңелдә,
Нинды моң бу, нинди көй?!
```
