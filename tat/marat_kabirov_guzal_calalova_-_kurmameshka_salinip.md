# Оригинал

### Марат Кәбиров / Гүзәл Җәлалова - Күрмәмешкә салынып

```
Күрмәмешкә салынып син
Узып киттең янымнан.
Күңелемдә әрнеп калды
Син кузгаткан давыллар.

Авыр миңа, авыр миңа
Ялгызыма давылда.
Авырлыкларга түзәрлек
Сабырлык бар җанымда.

Гаепләмим дә сине мин,
Үпкәләмим дә инде.
Син — йөрәк түремдә түгел,
Үткәннәремдә инде.

Синсез калган йөрәгемдә
Давыллар дулар әле.
Шашынып дулар-дулар да…
Дулар да… тынар әле.

Авыр миңа, авыр миңа
Ялгызыма давылда.
Авырлыкларга түзәрлек
Сабырлык бар җанымда.
```
