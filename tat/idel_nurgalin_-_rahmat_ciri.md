# Оригинал

### Идел Нургалин - Рәхмәт җыры

```
Бу дөньялар ике килми безгә,
Кадерен белеп калыйк без.
Ягымлы да әткәй-әнкәйләрнең,
Бөеклеген инде аңлыйк без.

Рәхмәтемне кабул итегез,
Сезгә җырлар җырым барында,
Бәхет — шатлык сезгә тәлимен,
Рәхмәт сезгә әткәй-әнкәем.

Теләкләрем фатыйхалар тулы,
Саф йөрәклесең син әнкәем.
Беләм инде әткәй гомерең буе,
Сез балларым диеп яшисең.

Рәхмәтемне кабул итегез,
Сезгә җырлар җырым барында,
Бәхет — шатлык сезгә тәлимен,
Рәхмәт сезгә әткәй-әнкәем.
```
