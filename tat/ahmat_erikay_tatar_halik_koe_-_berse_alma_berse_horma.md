# Оригинал

### Әхмәт Ерикәй / Татар халык көе - Берсе алма, берсе хөрмә

```
Матур кызлар суга бара,
Челтери чиләкләре;
Берсе сирень, берсе нәркиз.
Берсе гөлнең зәңгәре.

Матур кызларны күргәндә
Ярсып тибә йөрәгем;
Берсе алма, берсе хөрмә,
Берсе кура җиләге.

Өчесе дә бигрәк матур,
Өчесе дә бигрәк шәп;
Берсе җәүһәр, берсе гәүһәр,
Берсе алтын зөбәрҗәт.

Күз алмыйча карап торам
Кызлар барган юлларга;
Берсе уңган, берсе елдам,
Берсе алдынгылардан.

Өчесе дә бер дигәннәр,
Ефәк күлмәк кигәннәр;
Кайсын сайларга да белмим…
Өчесе дә чибәрләр!
```
