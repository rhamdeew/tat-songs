# Оригинал

### Ишмулла Дилмөхәмәтов - Курайчы егеткә

```
Төш, сылуым, курай тартам,
Тыпырдатып бас;
Яшермәче йөзеңне, сылуым,
Француз яулыгыңны ач.

Эх, сылуым кашың кара,
Буең зифа тал.
Курай тартҡан егеткә, сылуым,
Бер генә күзеңне сал.

Төш, сылуым, курай тартам,
Тыпырдатып бас;
Яшермәче йөзеңне, сылуым,
Бер генә йөзеңне ач.
```
