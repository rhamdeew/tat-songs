# Оригинал

### Илфак Шиһапов - Булмый инде булмагач

```
Тумыштан акча яратам, акча мине яратмый.
Күпме генә тырышсам да бер тиенгә дә артмый.
Әллә ниләр эшләп карыйм — кесәгә акча тулмый.
Төн буе хыяллансам да нигәдер баеп булмый.

Булмый инде булмагач,
Сабаларда тумагач,
Ичмасам, берәр туганың
Казанда утырмагач.
Булмый инде булмагач,
Актанышта тумагач,
Ичмасам, берәр абыең
Зур начальник булмагач.

Миллионнар булса — мин дә калганын табыр идем,
Күршеләрне утка салып, "Мерседес" алыр идем.
Зур итеп йорт салыр идем, диңгезгә барыр идем.
Аннан кайткач хәергә дә 100 тәңкә салыр идем.

Булмый инде булмагач,
Арчаларда тумагач.
Ичмасам, берәр туганың
Сөт җыючы булмагач.
Булмый инде булмагач,
Әлмәтләрдә тумагач,
Ичмасам берәр абыең
Зур нефтяник булмагач.

Эшләп баеп булса — ат та булыр иде акчалы.
Һаман җигелеп тартса да кияргә юк чалбары.
Акча диеп чабышабыз егылганчы картаеп.
Тормасалар арттан этеп — бу илдә булмый баеп.

Булмый инде булмагач,
Сабаларда тумагач,
Ичмасам, берәр туганың
Казанда утырмагач.
Булмый инде булмагач,
Актанышта тумагач,
Ичмасам, берәр абыең
Зур начальник булмагач.
Булмый инде булмагач,
Дөрес җирдә тумагач.
Җырлап кына баеп булмый,
Зур абзыең булмагач.
```
