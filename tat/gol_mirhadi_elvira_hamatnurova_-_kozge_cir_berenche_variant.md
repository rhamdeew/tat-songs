# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Көзге җыр (беренче вариант)

```
Мин көзге сукмактан
Мәктәпкә барам
Иелеп кулыма
Яфраклар алам.
Кайчандыр ямь — яшел
Үскән яфраклар
Кабызды учыма
Кызыл учаклар

Моңсу көз кояшы
Нурларын суза.
Тезелеп көньякка
Кошларым оча
Җырларын сагынып
Сайрар кошларның
Узарбыз сукмагын
Салкын кышларның

Көздән соң, кыштан соң
Килер яз ае.
Мин үсәм мизгелләр
Алышкан саен.
Ашыгып мәктәпкә
Атлыйм юлымнан
Яфраклар бәйләмен
Тотоп кулыма.
```
