# Оригинал

### Мәхмүт Хөсәен / Сара Садыйкова - Вальс турында вальс

```
Вальс! Вальс! —
Яңгырый киң залда.
Вальс! Вальс! —
Канат ул һәр җанга
Вальс! Вальс!
Кушылыйк без аңа.
Вальс залда, киң залда, вальс һәркайда!
Янымда — иркәм…
Безгә нур сипкән,
Гашыйклар иткән,
Бәхетле иткән —
Вальс! Вальс! —
Син сихри өермә!
Вальс! Вальс!
Яз шаулый күңелдә.
Вальс! Вальс!
Безнең сөю мәңге —
Мәхәббәт сүрелми гомергә!
Мәхәббәт сүрелми гомергә!..
```
