# Оригинал

### Сәрия Вәлиева / Урал Рәшитов - Башкортстаным — туган җирем

```
Башкортстаным — туган җирем,
Җырлы каеннар иле.
Күккә ашкан горур таулар,
Кылганлы дала иле.
Кылганлы дала иле —
Башкортстаным — туган җирем.

Сандугачлар өзлеп сайрар
Елгалы күлләр иле,
Гөл-чәчәкле, күбәләкле
Яшел тугайлар иле,
Яшел тугайлар иле.
Башкортстаным — туган җирем.

Бөек шагыйрь Мостай иле,
Данлыклы Рәми җире.
Сая бөркет Салаваттай,
Сүнмәс башкортым теле.
Сүнмәс халкымның теле —
Башкортстаным — туган җирем.

Сандугачлар өзлеп сайрар
Елгалы күлләр иле,
Гөл-чәчәкле, күбәләкле
Яшел тугайлар иле,
Яшел тугайлар иле.
Башкортстаным — туган җирем.
```
