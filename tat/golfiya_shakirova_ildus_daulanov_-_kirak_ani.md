# Оригинал

### Гөлфия Шакирова / Илдус Дәүләнов - Кирәк, әни

```
Инде үсеп, буйга җитсәбез дә,
Балалар дип сулыш аласың.
Туган йортка кайтып киткән чакта
Хәерле юл теләп каласың.

Кирәк, әни, безгә әйтер сүзең,
Кирәк, әни, якты карашың.
Булсын иде һәрчак өстәлеңдә
Тәмле чәең, тәмле ашларың.

Без кайтырбыз бергә җыелышып
Туган йортка, туган нигезгә.
Балачакка кайтыр өчен дә бит
Син ул кирәк, син кирәк безгә.

Кирәк, әни, безгә әйтер сүзең,
Кирәк, әни, якты карашың.
Булсын иде һәрчак өстәлеңдә
Тәмле чәең, тәмле ашларың.
Кирәк, әни, безгә әйтер сүзең,
Кирәк, әни, якты карашың.
Булсын иде һәрчак өстәлеңдә
Тәмле чәең, тәмле ашларың…
```
