# Оригинал

### Равил Булатов / Илгиз Субаев - Кал янымда (беренче вариант)

```
Күзләреңә карап уйга талам,
Карашыңда наз һәм моң гына.
Сине уйлап таңны каршы алам,
Елмаюың шифа җаныма.

Кичләремдә тагын сине күзлим,
Телләреңдә татлы бал гына.
Төннәремдә сине уйлап янам,
Яннарымда мәңге кал гына.

Күзләреңә карап уйга талам,
Карашыңда наз һәм моң гына.
Сине уйлап таңны каршы алам,
Елмаюың шифа җаныма.
```
