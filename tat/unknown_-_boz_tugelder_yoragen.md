# Оригинал

### Феликс Мортазин - Боз түгелдер йөрәгең

```
Бозлар эри, сулар ага
Юып кышкы эзләрне
Әллә чынлап әйттең микән
Сөям дигән сүзләрне

Сабыр гына көтәм әле
Уртак язның килгәнең,
Язлар җиткәч эрер әле
Боз түгелдер йөрәгең!

Энҗе бөртекләре булып
Тамчы тама түбәдән
Рөхсәт бирче нурың булыйм
Булыйм синең күләгәң.

Сабыр гына көтәм әле
Уртак язның килгәнең,
Язлар җиткәч эрер әле
Боз түгелдер йөрәгең!

Вәгъдә итмим алтын таулар
Назым булсын бүләгем
Алларыңа тезләним дә
Бүләк итим йөрәгем

Сабыр гына көтәм әле
Уртак язның килгәнең,
Язлар җиткәч эрер әле
Боз түгелдер йөрәгең!
```
