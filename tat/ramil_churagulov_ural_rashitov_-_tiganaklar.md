# Оригинал

### Рамил Чурагулов / Урал Рәшитов - Тигәнәкләр

```
Туган яктан киткән чакта
Елап калды ак чәчәкләр —
Авылымнан ябышып килгән
Илгәзәкләр-тигәнәкләр.

Тигәнәкләр-илгәзәкләр,
Сез бит минем сердәшләрем,
Туган яктан еракларда
Сез сөртәсез күз яшьләрем!

Тигәнәкнең энәләре
Чәнчеп тора җаннарымны.
Кайтып күр син сагынганда
Газиз туган якларыңны.

Тигәнәкләр-илгәзәкләр,
Сез бит минем сердәшләрем,
Туган яктан еракларда
Сез сөртәсез күз яшьләрем!

Тигәнәкләр-илгәзәкләр,
Карамагыз ачу катыш.
Кармагыз, хурламагыз,
Читтә йөрү — безнең язмыш!

Тигәнәкләр-илгәзәкләр,
Сез бит минем сердәшләрем,
Туган яктан еракларда
Сез сөртәсез күз яшьләрем!
```
