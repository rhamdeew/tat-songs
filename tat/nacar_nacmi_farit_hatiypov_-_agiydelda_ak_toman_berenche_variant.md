# Оригинал

### Наҗар Нәҗми / Фәрит Хатыйпов - Агыйделдә ак томан (беренче вариант)

```
Агыйделдә ак томан,
Параход тавышлары.
Дымлы керфекләр шикелле,
Ярында камышлары
Параход тавышлары.
Ярында камышлары
Дымлы керфекләр шикелле,
Ярында камышлары.

Агыйделдә ак томан,
Ат кешни болыннарда.
Ат эзләп йөргән эзләрем
Калган шул урыннарда
Чыкланган болыннарда,
Калган шул урыннарда.
Ат эзләп йөргән эзләрем
Калган шул урыннарда.

Агыйделдә ак томан,
Борылган Идел белән.
Нигә шул кадәр якынсың,
Әй, туган Иделгенәм?
Йөрәгем синең белән
Әй, туган Иделгенәм.
Нигә шул кадәр якынсың,
Әй, туган Иделгенәм?
```
