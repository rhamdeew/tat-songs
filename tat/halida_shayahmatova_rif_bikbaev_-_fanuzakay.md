# Оригинал

### Халидә Шәяхмәтова / Риф Бикбаев - Фәнүзәкәй

```
Ходай үзе белеп сынауларын
Бирә, диләр, бары көчлегә.
Синең сызамлыгың, сабырлыгың
Үрнәк булсын бөтен кешегә.

Фәнүзәкәй, сөйгән ярым!
Фәнүзәкәй, сөйгән ярым!

Чәчләрең бит җилбер-җилбер итеп
Тибрәләләр талгын җилләргә.
Синең кебек көчле рухлы җаннар
Бармы икән башка җирләрдә?

Фәнүзәкәй, сөйгән ярым!
Фәнүзәкәй, бердәнберем!
Фәнүзәкәй, сөйгән ярым!
Фәнүзәкәй, сөйгән ярым!

Бу тормышның кадерләрен белеп,
Яшәп киләсең бит, Фәнүзә.
Бар кызларның яхшы сыйфатлары
Җыелганнар төсле үзеңдә.

Фәнүзәкәй, сөйгән ярым!
Фәнүзәкәй, бердәнберем!
Фәнүзәкәй, сөйгән ярым!
Фәнүзәкәй, сөйгән ярым!
```
