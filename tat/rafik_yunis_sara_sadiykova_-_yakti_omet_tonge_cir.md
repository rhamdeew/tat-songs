# Оригинал

### Рәфикъ Юныс / Сара Садыйкова - Якты өмет (Төнге җыр)

```
Тамчы-тамчы ак йолдызлар
Күңелемә тамалар.
Ак уйларым ераклардан
Сине эзләп табарлар.

Якты-якты өмет күрәм
Айның уйчан йөзендә.
Син дә хәзер шушы зәңгәр
Төн чатыры эчендә!

Татлы-татлы хыяллардан
Мөмкин түгел арыну.
Сагынасым килеп тора
Арткан саен сагыну!
```
