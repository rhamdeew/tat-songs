# Оригинал

### Эльвира Әхмәтова / Илдус Дәүләнов - Биш минут

```
Биш минут та әле үтмәгән,
Өзелепләр килә күрәсем.
Мизгелгә дә китмә янымнан,
Яратам бит сине, беләсең!

Ошаганбыз бар яктан да,
Килешкәнбез ничек шулай.
Әллә инде икебезгә
Бер йөрәкне бүлгән Ходай.
Аңлаталмам сүзләр белән,
Күңел тулган, хисләр — ташкын.
Парлы булып, гомер буйлап
Икәүләшеп атлау язсын.
```
