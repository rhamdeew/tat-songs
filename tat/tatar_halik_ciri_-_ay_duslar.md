# Оригинал

### Татар халык җыры - Ай, дуслар

```
Алга дибез, алга атлыйбыз,
Алга таба барабыз.
Алга барырга кушкан бит
Безгә Ленин бабабыз,
Алга барырга кушкан бит
Безгә Ленин бабабыз.

Ай, дуслар,
Нурлар алыйк,
Юллар салыйк,
Гөлләр кадыйк маякка,
Гөлләр кадыйк маякка!

Кошлар кунган имәнгә,
Яфракларын игәнгә.
Рәхмәт Ленин бабабызга
Рәхәт тормыш биргәнгә.
Рәхмәт Ленин бабабызга
Рәхәт тормыш, биргәнгә.

Ай, дуслар,
Нурлар алыйк,
Юллар салыйк,
Гөлләр кадыйк маякка,
Гөлләр кадыйк маякка!
```
