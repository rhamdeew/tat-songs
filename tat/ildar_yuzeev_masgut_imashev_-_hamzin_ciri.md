# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Хәмзин җыры

```
Сизми калдым талларыма
Нинди кош кунганлыгын.
Очып киткәч кенә белдем
Сандугач булганлыгын.

Ак самовар борыннары,
Күңелле борулары.
Сиңа ничек, миңа авыр
Аерым торулары…

Чәчең чылбыр, чәчең чылбыр,
Тартып өзәрлек түгел.
Чәчең чылбыр, үзең былбыл,
Күрми түзәрлек түгел…
```
