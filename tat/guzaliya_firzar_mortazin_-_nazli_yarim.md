# Оригинал

### Гүзәлия / Фирзәр Мортазин - Назлы ярым

```
Диңгез буйларында мин уйлап йөредем,
Дулкыннарда синең төсеңне күрдем.

Елама, юксынма беркайчан, җаным,
Күңелем түрендәге назлы ярым,
Кара кашым син минем, җәүһәр ташым.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан.

Батты синең күзләреңдә күзләрем,
Йөрәгеңә җавап бирде йөрәгем.

Елама, юксынма беркайчан, җаным,
Күңелем түрендәге назлы ярым,
Кара кашым син минем, җәүһәр ташым.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан.

Аерылсабыз, чит тавышы белән
Тәрәзгә шакый көтмәгәндә кинәт.

Ташлап китмә, минем мәхәббәтем син,
Тәрәзәләргә шакыды бары төн.
Киләчәгем син минем, бердәнберем.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан.
Кулымны сиңа сузам, барыйкчы без бер юлдан,
Бәхет-шатлык тулы тормыш юлыннан…
```
