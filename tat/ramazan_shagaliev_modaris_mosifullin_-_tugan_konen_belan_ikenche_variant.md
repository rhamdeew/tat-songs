# Оригинал

### Рамазан Шагалиев / Мөдәрис Мөсифуллин - Туган көнең белән! (икенче вариант)

```
Туган көнең белән, дускай,
Котларга килдем сине.
Бүләк алып килә алмамын,
Гафу ит инде мине.

Буш кул белән килгәнсең дип
Үпкәли күрмә миңа.
Туган көнең белән котлап
Җыр алып, җыр алып,
Җыр алып килдем сиңа!

Чәчәк алып килер идем,
Мин аны сулыр дидем.
Сулыган чәчәккә карап,
Күңелең тулыр, дидем.

Буш кул белән килгәнсең дип
Үпкәли күрмә миңа.
Туган көнең белән котлап
Җыр алып, җыр алып,
Җыр алып килдем сиңа!

Җырым сердәш, юлдаш булсын,
Йөрегән якларыңда.
Яңа көч һәм дәрт өстәсен
Арыган чакларыңда.

Буш кул белән килгәнсең дип
Үпкәли күрмә миңа.
Туган көнең белән котлап
Җыр алып, җыр алып,
Җыр алып килдем сиңа!
```
