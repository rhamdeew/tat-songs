# Оригинал

### Гүзәллия Хәкимҗанова - Сөйгәнемә

```
Бормалы булды, бормалы
Тормышымның юллары.
Үкенерлек эшләрем юк,
Җыр белән кордым оямны.

Юлым булды сикәлтәле,
Шатлыклар да күп булды.
Тормыш җилләре искәндә
Бер Ходаем саклады.

Гомерләрем горур үтә,
Ә күңел нидер көтә.
Мәхаббәт сихри иленә
Керәсем килә, иркәм.

Килә синең белән бергә
Болыннардан үтәсемм.
Чәчәкләрне җыя-җыя,
Уйларыңны беләсе.

Таң алдыннан узар идем
Чәчләреңне мин сыйпап.
Йолдызларга, айга карап,
Сандугач белән җырлап.

Янып сөймәгәч тормышның
Яме бар микән инде?
Гомерләр чишмәләр кебек
Җыр белән үтәр инде.

Сөю мәңгелек, бәгырем,
Күңелем кылларын чирт.
Былбыл кебек канат җәеп,
Сөйгәнеңә килеп җит.

Җырларымны сиңа гына
Багышлармын, сөйгәнем.
Тизрәк кил син, аерылмыйк,
Көтә сине гүзәлең.
Җырларымны сиңа гына
Багышлармын, сөйгәнем.
Тизрәк кил син, аерылмыйк,
Көтә сине гүзәлең.
Тизрәк кил син, аерылмыйк,
Көтә сине гүзәлең…
Көтә сине гүзәлең…
```
