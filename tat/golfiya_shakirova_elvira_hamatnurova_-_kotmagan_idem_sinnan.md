# Оригинал

### Гөлфия Шакирова / Эльвира Хаматнурова - Көтмәгән идем синнән

```
Көтмәгән идем мин синнән,
Шундый ялгыш адымны.
Инде күптән бер хәбәрсез,
Газаплыйсың җанымны.

Башкадан түгел, мин синнән,
Көтмәгән идем моны.
Нигә-нигә, нәкъ менә син,
Аңламыйм бүген шуны.

Салып булса, салыр идем,
Агымсуга уйларны.
Син ни кичерәсең икән,
Беләсе иде шуларны.

Башкадан түгел, мин синнән,
Көтмәгән идем моны.
Нигә-нигә, нәкъ менә син,
Аңламыйм бүген шуны.

Гафу итә алмыйм сине,
Мин аңа әзер түгел.
Мең төрле газап эчендә,
Чишмәләрдәй бу күңел.

Башкадан түгел, мин синнән,
Көтмәгән идем моны.
Нигә-нигә, нәкъ менә син,
Аңламыйм бүген шуны.
```
