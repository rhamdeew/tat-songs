# Оригинал

### Рәсим Җәләл / Оскар Усманов - Мин синең йолдызың

```
Кайларда йөрисең, кемнәрне күзлисең,
Мин барлыкны белмисең микән.
Бер генә күрсәң дә, ни генә дисәң дә,
Син минеке булырсың, иркәм…

Уйнап кына ярат, күзләремә карап,
Әйт әле бер "яратам" диеп…
Язның нурларыннан, җәйнең гөлләредәй
Назлар көтәм яратып-сөеп.

Читләрдән эзләмә, ятларны күзләмә,
Мин бит ул эзләгән тиң ярың.
Әллә каян көткән, Ходай насыйп иткән,
Янәшәңдә йөри бит парың…

Уйнап кына ярат, күзләремә карап,
Әйт әле бер "яратам" диеп…
Язның нурларыннан, җәйнең гөлләредәй
Назлар көтәм яратып-сөеп.

Бир әле кулыңны, яшермә сереңне,
Онытылсын хыяллар йолдызың…
Җиденче кат күктән, өмет белән көткән
Мин бит ул йолдызың, жир кызың…

Уйнап кына ярат, күзләремә карап,
Әйт әле бер "яратам" диеп…
Язның нурларыннан, җәйнең гөлләредәй
Назлар көтәм яратып-сөеп.
Язның нурларыннан, җәйнең гөлләредәй
Назлар көтәм яратып-сөеп…
```
