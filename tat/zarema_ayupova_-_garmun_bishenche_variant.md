# Оригинал

### Зарема Әюпова - Гармун (бишенче вариант)

```
Ниләр булды икән, таңга кадәр иркәм
Сагыш тулы бер моңны уйнадым.
Гармун белән бергә күңел өзгәләнә,
Аерылу көе дип уйладым.

Сагышларым салсам талгын үткән суга,
Алып китәр микән яшьлеккә?
Бары тик бер кичкә кайтып килер идем,
Кайтып булса әгәр үткәнгә.

Очрашулар көтеп, ялгыз юллар узып
Гомер уза, суы сагышлы.
Без барабыз икәү иркен ярдан,
Кичеп булмый әмма язмышны.
```
