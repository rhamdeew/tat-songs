# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Агыгыз сулар кирегә (Ага сулар)

```
Ага сулар, ага сулар,
Ага сулар туктамый.
Гомер үтә, йөрәк көтә,
Кавышырга җай чыкмый,

Агыгыз сулар кирегә —
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.

Яшел үләнле аланда
Уеннар матур иде.
Безнең яшьли сөйгән ярлар
Башкалар яры инде.

Агыгыз сулар кирегә —
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.

Бәхетең үзең кулыңда,
Сайланып күп йөремә.
Яшьли сөйгән үз ярыңны
Берүк ятларга бирмә.

Агыгыз сулар кирегә —
Яшьлеккә табан.
Анда мине сөйгән ярым
Көтәдер һаман.
```
