# Оригинал

### Уран Кинҗәбулатов / Абрар Габдрахманов - Көнбагыш

```
Чәчәк тамыры
Туфрактан сут ала.
Минем күзләрем
Тик сиңа уктала.

Басылмый
Йөрәктә моң-сагыш.
Син — кояш,
Ә мин — бер көнбагыш.

Ялбыр болытлар
Капласа кояшны,
Һәр чак эзлимен
Мин синең карашны.

Басылмый
Йөрәктә моң-сагыш.
Син — кояш,
Ә мин — бер көнбагыш.

Назлап үстердем
Мәхәббәт гөлләрен.
Мәңге исемдә
Чәчәкле көннәрем.

Басылмый
Йөрәктә моң-сагыш.
Син — кояш,
Ә мин — бер көнбагыш.
```
