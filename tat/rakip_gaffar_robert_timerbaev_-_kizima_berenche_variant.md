# Оригинал

### Ракип Гаффар / Роберт Тимербаев - Кызыма (беренче вариант)

```
Нәрсә керә төшләреңә
Кызым минем, кызым минем?
Чәчәк булып үсәсеңме
Күбәләкләр сөендереп?
Сандугач булып моңланып,
Талдан талга күчәсеңме?
Нәрсә керә төшләреңә
Кызым минем, кызым минем?

Нәрсә керә төшләреңә
Кызым минем, кызым минем?
Алма булып пешәсеңме,
Таң атканда алсуланып?
Парларыңны эзләп күлгә
Аккош булып төшәсеңме?
Нәрсә керә төшләреңә
Кызым минем, кызым минем?

Йокла-йокла назлы гөлем,
Тәмле, матур төшләр күреп.
Бик тиз үсеп җитәрсең син,
Кызым минем, кызым минем.
Уңган егетнең өенә
Килен булып төшәрсең син.
Нәрсә керә төшләреңә
Кызым минем, кызым минем?
```
