# Оригинал

### Татар халык җыры - Хәмдия (икенче вариант)

```
Ач тәрәзәң — җиләк бирәм,
Җиләктән кайтып киләм лә
Кайтып киләм.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.

Шул дустымның нурлы йөзен
Инде мин кайчан күрәм лә,
Кайчан күрәм?

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.

Ай-һай, дустым, керфегең,
Бирче миңа бөртеген лә,
Бирче бөртеген.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.

Чит илләргә мин китәмен,
Төсең итеп йөртермен лә,
Ай, йөртермен.

Хәмдиянең ак калфагы
Яңа каралган чагы;
Хәмдиягә сүз әйтмәгез,
Зиһне таралган чагы.
```
