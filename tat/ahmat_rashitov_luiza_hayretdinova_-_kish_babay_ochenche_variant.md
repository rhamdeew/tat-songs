# Оригинал

### Әхмәт Рәшитов / Луиза Хәйретдинова - Кыш бабай (өченче вариант)

```
Кыш бабай, Кыш бабай,
Безгә бик таныш бабай;
Сакал-мыек ап-актан,
Килгән безгә ерактан.

Бүләкләр алып килгән,
Бүреген кыңгыр кигән.
Әйлән-бәйлән уйната,
Биетә һәм җырлата.

Безгә конфетлар бирә,
Үзе елмая-көлә.
Кыш бабай безгә тагын
Яңа ел алып килә.
```
