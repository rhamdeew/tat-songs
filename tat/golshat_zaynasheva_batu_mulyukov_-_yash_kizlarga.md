# Оригинал

### Гөлшат Зәйнәшева / Бату Мулюков - Яшь кызларга

```
Чәчәккә күмелсен,
Кояшка үрелсен, —
Үстер син яшьлеген гөлләрен
Гомергә бер килә,
Кадерен бел генә —
Унсигез яшь тулган көннәрең.

Җырла син өздереп,
Уйна син сыздырып,
Жәлләмә гармуның телләрен.
Гомергә бер килә,
Кадерен бел генә —
Унсигез яшь тулган көннәрең.

Хезмәттә үсәрсең,
Алмадай пешәрсең,
Эшләсәң сызганып җиңнәрең.
Гомергә бер килә,
Кадерен бел генә —
Унсигез яшь тулган көннәрең.
```
