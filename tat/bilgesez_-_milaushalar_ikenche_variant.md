# Оригинал

### Билгесез - Миләүшәләр (икенче вариант)

```
Мөлдерәмә тулган күңелем,
Чайкала, гел түгелә.
Үсеп чыккан миләүшәләр,
Текәлгәннәр күземә.
Үсеп чыккан миләүшәләр,
Текәлгәннәр күземә.

Кочагыма алып кыстым
Миләүшә чәчәкләрен.
Түзми йөрәк, гел кабатлый
Син сөенеп киләчәген.
Түзми йөрәк, гел кабатлый
Син сөенеп киләчәген.

Сиңа гына багышладым
Бар җырларым, Миләүшә.
Җырларымда, хыялымда
Синең белән серләшәм.
Җырларымда, хыялымда
Синең белән серләшәм.
Синең белән серләшәм…
Серләшәм…
```
