# Оригинал

### Зөлфия Минһаҗева / Мөхтәр Минһаҗев - Бөек Җиңү бәйрәме

```
Медальләр ялтырый кояшта
Ветераннар түшендә.
Күзләре яшьле булса да,
Елмаю йөзләрендә.

Бүген Җиңү бәйрәме,
Ачы сугыш беткән көн.
Яу кырында калганнарны
Искә ала торган көн.

Парадка бара бабайлар,
Баралар очрашуга.
Бер-берсен барлап чыгарга,
Бераз елап алырга.

Бүген Җиңү бәйрәме,
Ачы сугыш беткән көн.
Яу кырында калганнарны
Искә ала торган көн.

Кайгы китерә сугышлар,
Туктасын иде алар.
Тынычлыкта яшәсеннәр
Җирдә барлык балалар.

Бүген Җиңү бәйрәме,
Ачы сугыш беткән көн.
Яу кырында калганнарны
Искә ала торган көн.
```
