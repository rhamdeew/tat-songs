# Оригинал

### Мәхмүт Хөсәен / Ренат Еникеев - Казан — гомер бишегем

```
Ил хакын даулаган утлы давыл
Күптән тынды…
Хәсрәт, зур кайгылар
Онытылды…
Кырыс еллар кичкән каеннар
Яфрак яра!
Күңелләрдә тибри дәртле көйләр —
Исән Җәлилләр!

Әй, гомер бишегем, әй, Казаным!
Мәңге күрмә сугыш газабын,
Бул гел кояшлы,
Бул киң колачлы —
Артсын, калам, даның!

Сез, тирән яралар, күптән инде
Савыктыгыз…
Көлдән, әй, калалар,
Калыктыгыз…
Тузан-туфрак күрми — алмалар
Ямь тарата!
Кулларда балкый изге утлар —
Исән Кутуйлар,

Әй, гомер бишегем, әй, Казаным!
Мәңге күрмә сугыш газабын,
Бул гел кояшлы,
Бул киң колачлы —
Артсын, калам, даның!
```
