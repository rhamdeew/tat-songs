# Оригинал

### Рәми Гарипов / Роза Сәхәүетдинова - Гомер (икенче вариант)

```
Җир җиләккәйләре пеште инде,
Өзелеп тә төшмәс димәгез, —
Ир-егеткәйләрнең яше үтсә,
Бөгелеп тә төшмәс димәгез.

Яфракларны сары алды инде,
Җил шаулатып түкмәс димәгез, —
Үт-яшеңдәй яшьлек гомерләрне
Яшнәп-күкрәп үтмәс димәгез.

Кар сулары шаулап акты инде,
Гел-чәчәкләр атмас димәгез, —
Бишектәге бала, биш төрләнеп,
Ат атланып чапмас димәгез!
```
