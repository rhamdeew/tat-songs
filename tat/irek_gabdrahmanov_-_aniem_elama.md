# Оригинал

### Ирек Габдрахманов - Әнием, елама!

```
Күкләр елый, җиһан сыкрый,
Монда килдек көзләрдә.
Кайнар яшьләр тама, ага,
Сагыш баскан күзләрдә.

Әнием, елама,
Сагыш җыйма җаныңа.
Кошлардай канатланып,
Кайтырмын мин яныңа.

Озатканда елап калдың,
Җибәрми кулларымны.
Еракларга киттем, әни,
Киек каз юлларыннан.

Әнием, елама,
Сагыш җыйма җаныңа.
Кошлардай канатланып,
Кайтырмын мин яныңа.

Догаларың саклар мине
Сугыш ялкыннарыннан.
Мин кайтырмын, син көт, әни,
Күз алма юлларымнан.

Әнием, елама,
Сагыш җыйма җаныңа.
Кошлардай канатланып,
Кайтырмын мин яныңа.

Кайтырмын мин, әнкәй. Син елама.
Син мине көт, яме. Мин кайтырмын.
Елама, елама, елама…
```
