# Оригинал

### Сибгат Хәким / Җәүдәт Фәйзи - Торыгыз, Мусалар

```
Кайвакыт күп төрле
Уйлардан йөдисең…
Килә бер җир буйлап
Кычкырып йөрисем:
Басуда — арышлар,
Бакчада әнисләр
Өлгерде, торыгыз,
Торыгыз, Алишлар…

Кайвакыт бүлмәдән
Качасың юри син…
Килә бер җир буйлап
Кычкырып йөрисем:
Дошманнар тынмаслар,
Коялар яңадан
Кургашлар, торыгыз,
Торыгыз, Курмашлар…

Кайвакыт исним дә
Чәчәкле җил исен,
Килә бер җир буйлап
Кычкырып йөрисем:
Чәчәкләр үсәләр,
Көн саен тормышта
Сез кирәк, торыгыз,
Торыгыз, Мусалар…
```
