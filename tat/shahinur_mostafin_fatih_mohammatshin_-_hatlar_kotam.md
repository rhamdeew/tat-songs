# Оригинал

### Шаһинур Мостафин / Фатих Мөхәммәтшин - Хатлар көтәм

```
Ләйсән яңгыр көткән җиргә охшап, иркәм,
Түземсезлек белән синнән хатлар көтәм.

Хатлар белән бергә
Бәхет киләсыман,
Күңел киңлегендә
Кояш көләсыман…

Тургай җырын көтеп, татлы уйга талган
Иген кыры сыман, хатлар көтәм һаман.

Хатлар килеп торса,
Җырлап яши күңел.
Гомер матур уза,
Тормыш итү җиңел.

Чәчәк атар өчен яз килүен көткән
Бөре кебек, синнән хатлар көтәм күптән.

Очкын — күзләремдә,
Ялкын — йөрәгемдә.
Янып китәрмен күк,
Кинәт дөрләрмен дә…
```
