# Оригинал

### Сәрия Вәлиева / Урал Рәшитов - Мин кайтырмын язлар җиткәч

```
Мин кайтырмын сиңа еллар үткәч
Салкын кышның ыҗгыр буранында.
Карга бата-чума бер йөрербез
Туып үскән авыл урамында.

Мин кайтырмын сиңа ярсу язда,
Уралыпмы йөрәк хисләремә.
Хуштан язып исерербез икәү
Ап-ак чәчәк шомырт исләренә.
Мин кайтырмын салкын бәйрәмендә,
Сабантуйның суык җәйләрендә.

Елмаюлар җыеп нур өстәрбез
Бәхет гөлләмәсе бәйләменә.
Мин кайтырмын сиңа көзләреңдә,
Сагыш басыр алыр йөзләремне.

Сары яфракларның арасында
Табалмас мы ахры эзләремне.
Сабул диер тагы иреннәрем,
Тик билгеле сиңа кайта чагым.
Хыялымда кабат юлга чыгам,
Еракларда калган бала чагым.
```
