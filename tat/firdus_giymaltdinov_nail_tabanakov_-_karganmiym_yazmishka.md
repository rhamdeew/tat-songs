# Оригинал

### Фирдүс Гыймалтдинов / Наил Табанаков - Карганмыйм язмышка

```
Ишекне син яптың,
Башканы син таптың.
Мин көтәрмен диеп уйладым.
Ялгызлык бик яман,
Тик мин ялгыз калмам,
Тормышымнан мин әле туймадым.

Карганып язмышка,
Буылып сагышка,
Мәңге яшәрмен диеп, ялгышма.
Мин синнән көчлерәк,
Табармын бер терәк,
Хисләремә бүтән ябышма.

Китүләр бик авыр,
Җаннарны өшетә,
Сулкылдый күңелдә үпкәләп.
Артыңа борылма,
Яныма ыргылма,
Хисләр ул дөрли дә, үтәләр.

Карганып язмышка,
Буылып сагышка,
Мәңге яшәрмен диеп, ялгышма.
Мин синнән көчлерәк,
Табармын бер терәк,
Хисләремә бүтән ябышма.

Ник кабат кайтасың,
Җанга ут саласың?
Йөрәгем ачкычы бүтәндә.
Ир булып кала бел,
Мәхәббәт була бер,
Үткәннең тузанын күтәрмә.

Карганып язмышка,
Буылып сагышка,
Мәңге яшәрмен диеп, ялгышма.
Мин синнән көчлерәк,
Табармын бер терәк,
Хисләремә бүтән ябышма.
```
