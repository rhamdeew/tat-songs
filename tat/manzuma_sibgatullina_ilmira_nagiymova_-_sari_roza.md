# Оригинал

### Мәнзүмә Сибгатуллина / Ильмира Нәгыймова - Сары роза

```
Юл буенда сары роза
Кызганмыйча тапталган.
Кемнең икән шул кадәрле
Мәхәббәте тапланган?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?

Сары роза — горур чәчкә,
Кемнең назлы бүләге,
Әйтелмичә калган сүзме,
Әллә соңгы теләге?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?

Юл буена ташлаганнар
Затлы чәчәк розаны.
Сөюләрне шул чәчәктәй
Сез ташларга ризамы?

Сары роза,сары роза,
Сине кемнәр ташлаган,
Сөю гөлен ташлар булгач,
Нигә сөю башланган?
```
