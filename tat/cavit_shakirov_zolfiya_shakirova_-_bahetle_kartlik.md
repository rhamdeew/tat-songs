# Оригинал

### Җәвит Шакиров / Зөлфия Шакирова - Бәхетле картлык

```
Яшь чагында, көч барында
Йөзләрдә шатлык.
Яшьлек үтәр, алда көтсен
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

Көч барында гармуннарны
Уйныйсың тартып.
Җырлы-моңлы булсын иде
Без көткән картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

Матур тормыш сукмагыннан
Бергәләп атлыйк,
Алда көтсен һәммәбезне
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.

"Байлык — бәхет" дигәннәргә
Ышанма артык,
Бу дөньяда иң зур байлык —
Бәхетле картлык.

Картаям дип уфтанмыйм да,
Кайгырмыйм артык.
Безне урап узмас әле
Бәхетле картлык.
```
