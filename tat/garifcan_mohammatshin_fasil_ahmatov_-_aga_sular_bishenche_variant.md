# Оригинал

### Гарифҗан Мөхәммәтшин / Фасил Әхмәтов - Ага сулар (бишенче вариант)

```
Ага сулар, ага сулар,
Бер болгана, бер тына.
Агымсулар кебек күңелем
Кая болай талпына?

Ашкыналар хыялларым
Агымсулар артыннан.
Мәхәббәтем чыгар кебек
Барыр юлым чатыннан.

Ага сулар, ага сулар,
Дулкынлана, болгана.
Соңлаганнар ялгыз кала,
Ашык, күңелем, соңлама.
```
