# Оригинал

### Гыйльман Идрисов / Феликс Әбдрәшитов - Каршыма кил минем сөйгәнем

```
Өфе урамнарын буйлап,
Синең хакта уйлап.
Ялгыз үтә минем кичләрем.
Айлар, еллар үтә,
Күңел өмет итә,
Сүрелтмичә сөю хисләрен.

Өфе урамнарын буйлап
Җилләр исә уйнап.
Урый-урый карлар явалар.
Ява ап-ак карлар,
Ерак әле язлар,
Якынаймый безнең аралар.

Өфе урамнарын буйлап,
Тирә якны нурлап.
Каршыма кил минем сөйгәнем.
Шатлыгымда син бит,
Сагышымда син бит,
Каршыма кил минем сөйгәнем.
```
