# Оригинал

### Марс Гайфетдинов - Яндырасың-көйдерәсең

```
Нигә мине аңламыйсың,
Нигә мине газаплыйсың,
Урамнарда очрашкач та —
Син миңа карамыйсың.

Яндырасың, көйдерәсең,
Салкын сулар бирмисең,
Яшь йөрәкнең яратуын
Нигә аңлый белмисең?

Яшь йөрәгем түзалмады..
Янды, көйде, каралды,
Куңелемдәге барлык серне
Җырлар аша чыгарды.

Яндырасың, көйдерәсең,
Салкын сулар бирмисең,
Яшь йөрәкнең яратуын
Нигә аңлый белмисең?

Аңла инде күңелемне,
Өзелеп сөюемне,
Мин бит сиңа бүләк итәм
Шушы хисле җырымны.

Яндырасың, көйдерәсең
Салкын сулар бирмисең,
Яшь йөрәкнең яратуын
Нигә аңлый белмисең?
```
