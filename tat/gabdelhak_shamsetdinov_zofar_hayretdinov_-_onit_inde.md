# Оригинал

### Габделхак Шәмсетдинов / Зөфәр Хәйретдинов - Оныт инде…

```
Оныт инде мине мәңгелеккә,
Йөрәгеңнән алып ташла да…
Гафу итмә мине, гафу итмә —
Алыштырдым сине башкага.

Кичер, димим, очрашулар эзләп
Юлларыңа чыкмыйм хәзер мин…
Мәңге төзәнмәслек яра булып
Йөрәгемдә калдың, кадерлем.

Ялгышларым сагышларга салды.
Мин һаман да сине юксынам…
Бу дөньяда синнән кадерлерәк,
Синнән якын һичкем юк сыман.

Оныт инде мине мәңгелеккә,
Йөрәгеңнән алып ташла да…
Гафу итмә мине, гафу итмә —
Алыштырдым сине башкага.
```
