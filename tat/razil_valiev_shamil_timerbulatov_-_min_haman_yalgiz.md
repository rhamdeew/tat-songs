# Оригинал

### Разил Вәлиев / Шамил Тимербулатов - Мин һаман ялгыз

```
Нигә сөйдем мин,
Нигә көйдем мин?
Көлмәче, кызый,
Йөрәгем сызлый,
Сызлый-сызлана.
Күкләргә очты
Хыялым кошы,
Очты хыялым кошы.
Ә син соң кайда, —
Мендеңме айга?
Ә син соң кайда?

Булмый онытып,
Йөрим ут йотып.
Йөрәктә яра,
Килче, кил, кара.
Йөрәк яралы, —
Төшләрдә күрәм,
Куйныңа керәм,
Керәм утларга, керәм.
Ялгызым иртән
Уянып китәм, —
Мин һаман ялгыз икән…
```
