# Оригинал

### Фирдәүсә Кәримова / Флера Шәрипова - Ялгыз кала күрмәгез

```
Урам буйлап җайдак килә,
Дагалы микән аты?
Парлы чакта ялгыз хәлен
Белми шул адәм заты.

Озын кара чәчегездән
Парсыз толым үрмәгез,
Дөнья йөге таштай авыр,
Ялгыз кала күрмәгез.

Кара булып болыт килә,
Яңгыр микән, кар микән?
Ялгызлыктан авыр хәлләр
Дөньялыкта бар микән?

Озын кара чәчегездән
Парсыз толым үрмәгез,
Дөнья йөге таштай авыр,
Ялгыз кала күрмәгез.

Сусаганда сулар эзләп,
Чүлгә чыга күрмәгез.
Күктән иңгән сөю хисен
Өч өлешкә бүлмәгез.

Озын кара чәчегездән
Парсыз толым үрмәгез,
Дөнья йөге таштай авыр,
Ялгыз кала күрмәгез.
Дөнья йөге таштай авыр,
Ялгыз кала күрмәгез…
Ялгыз кала күрмәгез…
```
