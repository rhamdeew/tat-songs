# Оригинал

### Гөлшат Зәйнәшева / Мөдәрис Газетдинов - Котлы булсын туйларыгыз

```
Карлыгач канатлы егет,
Сандугач тавышлы кыз
Кайсы йолдыз асларында
Очрашып кавыштыгыз?

Котлы булсын туйларыгыз!
Бәхетле көннәрегез.
Гел чәчәк атсын, сулмасын
Мәхәббәт гөлләрегез.

Чәчегезне үреп-үреп
Чәчләргә бәйләдегез.
Гомергә бер генә булсын
Туй дигән бәйрәмегез!

Котлы булсын туйларыгыз!
Бәхетле көннәрегез.
Гел чәчәк атсын, сулмасын
Мәхәббәт гөлләрегез.

Мәхәббәтегез юлыннан
Бердәм атлап китегез.
Ике язмышны бер итеп
Матур гомер итегез!

Котлы булсын туйларыгыз!
Бәхетле көннәрегез.
Гел чәчәк атсын, сулмасын
Мәхәббәт гөлләрегез.
```
