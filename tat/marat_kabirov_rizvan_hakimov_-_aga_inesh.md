# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Ага инеш

```
Тилмерешеп яшәр өчен
Килек мени бу дөньга?
Язгы таңда икебезне
Табыштырган инеш ага.

Сөю җырын көйли-көйли
Ага инеш, ага инеш.
Синең дә бит бер кемең юк,
Мин дә һаман ялгыз килеш.

Моңсу гына ага инеш,
Үпкәләгән төсле безгә.
Бер-беребез өчен генә
җиргә туган төслебез дә…

Сөю җырын көйли-көйли
Ага инеш, ага инеш.
Синең дә бит бер кемең юк,
Мин дә һаман ялгыз килеш.
Аймылышу ялгыштыр ул,
Безнең язмыш хак түгелдер.
Мәхәббәткә хаклы булгач,
Бәхеткә дә хаклыбыздыр.

Сөю җырын көйли-көйли
Ага инеш, ага инеш.
Синең дә бит бер кемең юк,
Мин дә һаман ялгыз килеш.
```
