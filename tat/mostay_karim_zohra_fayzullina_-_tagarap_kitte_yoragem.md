# Оригинал

### Мостай Кәрим / Зөһрә Фәйзуллина - Тәгәрәп китте йөрәгем

```
Эзләренә төшмәк булып
Илдәге бер чибәрнең,
Йомгак түгел, йөрәгемне
Тәгәрәтеп җибәрдем.

Тәгәрәп китте йөрәгем,
Китте дә югалды бит,
Эзен таптым, үзен тапмый
Инде зар елатты бит.

Кайда барып сугылаем,
Кем капкасын кагаем:
— Тәгәрәгән йөрәгемне
Күрмәдеңме, агаем?

Юл чатында мосафир карт
Шундый акыл өйрәтте:
— Юләрлек ул — юллап йөрү
Бер ычкынган йөрәкне.

Әй, әкият, мин һаман да
Беркатлы исәр балаң.
Тәгәрәгән йөрәгемнең
Артыннан куып барам.

Тәгәрәп китте йөрәгем,
Китте дә олакты бит,
Эзен таптым, үзен тапмый
Инде зар елатты бит.
Эзен таптым, үзен тапмый
Инде зар елатты бит…
```
