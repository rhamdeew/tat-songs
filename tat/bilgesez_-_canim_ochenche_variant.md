# Оригинал

### Билгесез - Җаным (өченче вариант)

```
Җаным, син булмасаң мин дә булалмамын,
Күзләреңә карап, кулларыңнан алып,
Падишаһың була алмамын.

Җаным, мин булмасам син дә булалмассың,
Былбыл кошлар булып, иңнәремә кунып,
Моңлы җырлар суза алмассың.

Мәхаббәтсез генә яшәп булмый,
Борма-борма аның юллары,
Борма-борма юллар киртә түгел,
Көчле мәхаббәтнең кыллары.

Тук-тук итеп тибә йөрәк,
Син, син диеп таң ата.
Сине генә сөям, сине генә көтәм,
Сине генә, җаным, яратам.

Җаным, мин булмасам син дә булалмассың,
Былбыл кошлар булып, иңнәремә кунып,
Моңлы җырлар суза алмассың.

Җаным, син булмасаң мин дә булалмамын,
Күзләреңә карап, кулларыңнан алып,
Падишаһың була алмамын.

Сөюләрсез генә яшәп булмый,
Сөюләрдә бакча гөлләре.
Утларына салып, ялкынында
Яндырмыйк бит сөю көннәрен.

Тук-тук итеп тибә йөрәк,
Син, син диеп таң ата.
Сине генә сөям, сине генә көтәм,
Сине генә, җаным, яратам.

Мәхаббәтсез генә яшәп булмый,
Борма-борма аның юллары,
Борма-борма юллар киртә түгел,
Көчле мәхаббәтнең кыллары.
```
