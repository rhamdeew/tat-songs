# Оригинал

### Вилдан Вәлиев / Венер Мөхәммәдиев - Ачытау (икенче вариант)

```
Ачытаудан көтеп алдым
Сызылып таңнар атканын.
Күңел җылый, өзелә үзәк,
Талмасын канатлары.
Салкын чишмә чылтырыйдыр
Ачытау асларында.
Бер җылы сүз җитәр иде
Сагынган чакларымда.
Йөгерәм дә менеп китәм
Ачытау башларына.
Исән чакта язасым бар
Шул тауның ташларына,
Истәлеккә язасым бар
Ул тауның ташларына.
Ачытау башларына…
```
