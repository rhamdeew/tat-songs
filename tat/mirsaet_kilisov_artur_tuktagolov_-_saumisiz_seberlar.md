# Оригинал

### Мирсәет Кылысов / Артур Туктаголов - Саумысыз, Себерләр

```
Саумысыз, дус-ишләрем,
Ерак Себернекеләр!
Себер, Себер, дисәләр дә,
Бары да безнекеләр.

Ак бураннар, ак бураннар,
Адаштыр сагышларны.
Сагышларны адаштыр да,
Кауыштыр язмышларны.

Олы саләм алып килдек
Туып үскән җирләрдән —
Яшьлек дигән илебездән,
Тәүге үпкән ярлардан.

Ак бураннар, ак бураннар,
Адаштыр сагышларны.
Сагышларны адаштыр да,
Кауыштыр язмышларны.

Салкын көзләр кошлар юлын
Чит илгә суза икән.
Кошлар ҡайта, тик гомерләр
Кайтмаска уза икән.

Ак бураннар, ак бураннар,
Адаштыр сагышларны.
Сагышларны адаштыр да,
Кауыштыр язмышларны.
```
