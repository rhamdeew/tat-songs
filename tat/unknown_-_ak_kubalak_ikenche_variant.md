# Оригинал

### Зөһрә Ганиева / Мөфтәдин Гыйләҗев - Ак күбәләк (икенче вариант)

```
Күп йөредем гөлләр арасында,
Чәчкәләрне үзең өзмәдең.
Ак күбәләк булып тугайларда
Кемне шулай сагынып эзләдең.

Сине генә көттем, сагынып көттем.
Каршыларга таңны килмәдең.
Ник соң үзең минем тормышыма
Ак күбәләк булып кермәдең?

Йөрәккәем ялкын белән тулы,
Хыялларым синдә чагында.
Искә ал син мине, иркәләнеп
Ак күбәләк очса яныңда.
```
