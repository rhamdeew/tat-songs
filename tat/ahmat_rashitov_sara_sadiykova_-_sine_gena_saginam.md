# Оригинал

### Әхмәт Рәшитов / Сара Садыйкова - Сине генә сагынам

```
Йөрәгемә сагыш уты өстәп,
Алмагачлар ак шәл ябынган.
Ә мин сине, солдат сөйгәнемне,
Бары сине генә сагынам.

Матур сүзләр сөйләп, бар егетләр
Каш сикертеп уза янымнан.
Башкаларга күзем төшәр димә,
Бары сине генә сагынам.

Тәүге үбүләрем вәгъдә сиңа,
Гыйшкым уты синнән кабынган.
Көннәр буе, айлар, еллар буе
Бары сине генә сагынам.

Сөйгәнемә җырым сәлам илтсен
Туып үскән Идел ягыннан.
Онытма син, бәгърем, онытма син,
Бары сине генә сагынам.
```
