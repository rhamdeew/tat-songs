# Оригинал

### Татар халык җыры - Сагыну (беренче вариант)

```
Чәчәк идек илләрдә лә,
Чәчелдек чит җирләргә.
Тагы да чәчәк булыр идек,
Кайтсак туган илләргә.

Кайсы якка очасың дип,
Кыр казыннан сорадым.
Ул кычкырып җавап бирде,
Мин утырып еладым.

Урамнардан урамнарга
Кемнәр күпер салганнар.
Без аерылдык та саргайдык,
Саргаймасын калганнар.

Салкын чишмә буйларында
Атлар эчерсәң иде.
Туган-үскән җирләреңдә
Гомер кичерсәң иде.
```
