# Оригинал

### Билгесез - Ах, Җырларым

```
Ах, җырларым бөтөн җиргә
Таралып китсен иде.
Җыр сөючеләрнең күңлен
Тибрәтеб үтсен иде.
Җайлапкына йөрәкләрнең
Кылларын чиртсен иде,
Кылларын чиртсен иде.

Үтсен иде юлларында
Киртәләр чырнатса да.
Кеше күңлен тапсын иде,
Ничек кенә тапса да.
Кайгыдан югалтсын иде
Шатлыктан елатса да,
Шатлыктан елатса да.

Ах, җырларым ярты юлда
Туктап тормасын иде.
Күкләрдә күкрәсен иде,
Җирдә тонмасын иде.
Сафирәкле газиз халкым
Җырлап туймасын иде,
Җырлап туймасын иде.
```
