# Оригинал

### Лена Шагыйрьҗан / Фәрит Хатыйпов - Яңа Әтнә җыры

```
Хисләремне ямьгә төргән,
Язмышымның ядъкаре.
Җыр булып йөрәккә кергән,
Безнең Әтнә яклары.

Сагындым су буйларын,
Сагындым урамнарын,
Сагындым сабан туйларын,
Каенсар урманнарын.

Әкиятле, шигъриятле,
Тукайлы ягымдыр ул.
Кайларда гына йөрсәм дә,
Тарта шул ягымлы нур

Сагындыра әткәбез,
Сагындыра әнкәбез.
Китсәгез дә кайтырсыз дип
Көтә безне Әтнәбез.

И, туган ягым күрдек без,
Бахетле таңнарыңны.
Бергә-бергә күтәрербез.
Элекке даннарыңны.

И, моңлы ягым минем,
И, нурлы ягым минем.
Кайтам сиңа бөтенләйгә,
Бәхетле чагым минем.
```
