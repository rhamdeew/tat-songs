# Оригинал

### Рәшит Фазлыев / Сара Садыйкова - Ешрак яз, әнием, хатларыңны

```
Еракларда минем туган ягым,
Еракларда сөйгән авылым.
Ешрак яз, әнием, хатларыңны,
Бигрәкләр дә сине сагындым.

Кошлар очса әгәр сезнең якка,
Сәлам әйтеп калам артыннан.
Мин үзем дә очар кошлар кебек,
Күңелем белән сиңа ашкынам.

Сагынадыр диеп мине уйлап,
Көтәсеңдер, әнкәй, моңаеп.
Сезнең якта көн дә кояш чыга,
Алсу йөзең кебек елмаеп.
```
