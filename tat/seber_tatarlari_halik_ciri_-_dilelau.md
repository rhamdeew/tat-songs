# Оригинал

### Себер татарлары халык җыры - Дилеләү

```
Алма сатучылар килгән
Үлчәүләре бар микән?
Әҗәлгә дару бар, диләр,
Мәхәббәткә бар микән?
Алма кисәм, алма кисәм,
Алманың түгәрәген.
Хәзер мин ни уйлыйм микән
Мәхәббәт түгәрәгем?

Һай, дилеләү, дилеләү, дилеләү,
Һай, дилеләү, дилеләүләү.

Алма пешкән, өзелеп төшкән,
Күгәрчен башын тишкән.
Матур егет сайлый — сайлый,
Чәчләреңә ак төшкән.
Алма алырсың әле,
Алмый калырсың әле.
Күп егетләр сайлап торсаң,
Ялгыз калырсың әле.

Һай, дилеләү, дилеләү, дилеләү,
Һай, дилеләү, дилеләүләү.

Алма бөреле, алма бөреле,
Алма чәчәге төрле.
Башкаларга күз дә салмыйм,
Сине күргәннән бирле.
Агыйделнең аръягында,
Атлар эчерсәң иде.
Киләчәккә бергә — бергә,
Гомер кичерсәк иде.

Һай, дилеләү, дилеләү, дилеләү,
Һай, дилеләү, дилеләүләү.
```
