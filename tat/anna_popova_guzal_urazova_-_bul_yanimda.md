# Оригинал

### Анна Попова / Гүзәл Уразова - Бул янымда

```
Күпме юллар язган икән узасы,
Сине яңадан үземә табарга.
Ике йөрәк — ерак дарья арасын
Мәңгегә ялгарга.
Әллә инде, әллә күкләрдә
Кавышкан җаннар да бу жирдә.
Бер-берсен табалмый туктамый?
Әллә инде, әллә күкләрдә
Кавышкан җаннарга бу җирдә
Сөюсез яшәп булмый, булмый?

Яңа таңнар аткан чагында
Уйлар һаман сиңа ашкына,
Бул янымда,
Янымда бул гына!
Син — терәгем минем дөньяда,
Югалма тик, зинһар, югалма,
Бул янымда,
Янымда бул гына!

Төшем булсаң — кабатлан һаман, һаман,
Өн булсаң — югалма, янымда кал.
Син минем күкләрдән сорап алган
Гомерлек яр.
Әллә инде, әллә күкләрдә
Кавышкан җаннар да бу жирдә
Бер-берсен табалмый туктамый?
Әллә инде, әллә күкләрдә
Кавышкан җаннарга бу җирдә
Сөюсез яшәп булмый, булмый?

Яңа таңнар аткан чагында
Уйлар һаман сиңа ашкына,
Бул янымда,
Янымда бул гына!
Син — терәгем минем дөньяда,
Югалма тик, зинһар, югалма.
Бул янымда,
Янымда бул гына!

Булды, булды, булды,
Яра тулы йөрәгем,
Син килдең дә уяндым,
Бәхет өчен күпне сорамам,
Тик онытма:
Сине яратам.
```
