# Оригинал

### Ренат Харис / Эльмир Низамов - Туган ил (бишенче вариант)

```
Туган ил ул — мәңгелек ил! —
Безнең йөрәктә исән.
Бары туган илдә генә
Бәхет гөлләре үсә!

Туган ил ул — мәңгелек ил —
Дөнья беткәнче исән!..
Туган илдән аерылма,
Бәхетле булыйм дисәң!
Бәхетле булыйм дисәң!

Туган ил ул — мәңгелек ил! —
Туган ил мәңге исән.
Коры туфракка җан керә
Изге күз яшем тисә.

Туган ил ул — мәңгелек ил! —
Безнең өметтә исән.
Күңелләрем күтәрелә
"Әй, туган илем!" — дисәм!
"Әй, туган илем!" — дисәм!

Туган ил ул — мәңгелек ил —
Кан хәтерендә исән.
Сагынуга түзеп булмый,
Туган яктан җил иссә!
Туган илдән аерылма
Бәхетле булыйм дисәң!
Бәхетле булыйм дисәң!..
```
