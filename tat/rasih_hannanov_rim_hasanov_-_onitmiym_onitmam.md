# Оригинал

### Расих Ханнанов / Рим Хәсәнов - Онытмыйм (Онытмам)

```
Сандугачымны очырдым,
Былбыл тоталмам инде.
Син онытсаң-онытырсың,
Мин оныта алмам инде,
Мин оныта алмам инде.

Сандугачлар капкада,
Канатлары тактада.
Бәгырькәем, мин онытмыйм,
Син оныткан чакта да.

Сандугачның балалары
Сайрый алмый читлектә.
Кара кашым, карлыгачым,
Күрешә алмыйбыз ничек тә.
Күрешә алмыйбыз ничек тә.

Сандугачлар капкада,
Канатлары тактада.
Бәгырькәем, мин онытмыйм,
Син оныткан чакта да.

Сандугач баласы булсам,
Күктә уйнар (очыр) идем мин.
Синең сөймәсеңне белсәм.
Күптән үләр идем мин.
Күптән үләр идем мин.

Сандугачлар капкада,
Канатлары тактада.
Бәгырькәем, мин онытмыйм,
Син оныткан чакта да.
```
