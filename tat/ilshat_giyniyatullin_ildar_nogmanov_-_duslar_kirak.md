# Оригинал

### Илшат Гыйниятуллин / Илдар Ногманов - Дуслар кирәк

```
Гомеркәйләр үткән саен
Аңлыйм дуслар кирәген.
Сез бит минем менә дигән
Язмышымның миңа биргән
Бер кадерле бүләге.

Утка салса авырлыклар,
Дуслар кирәк киңәшкә.
Авыр чакта ярдәм итеп,
Рәхәт чакта бәйрәм итеп.
Ни җитә соң яшәүгә ?

Аңлыйм, беләм : сездәй, дуслар,
Күп түгелен, сирәген.
Бик еш сезне күрмәсәм дә,
Тоеп яшим янәшәмдә
Чын дусларның терәген.

Утка салса авырлыклар,
Дуслар кирәк киңәшкә.
Авыр чакта ярдәм итеп,
Рәхәт чакта бәйрәм итеп.
Ни җитә соң яшәүгә ?

Вакытлар белән сынала
Кемнәрнең кем икәне.
Шул чакта генә беленә
Чын дус булып кала белә
Дуслар һәм дус дигәне.

Утка салса авырлыклар,
Дуслар кирәк киңәшкә.
Авыр чакта ярдәм итеп,
Рәхәт чакта бәйрәм итеп.
Ни җитә соң яшәүгә ?
```
