# Оригинал

### Гөлшат Зәйнәшева / Асия Зәйнашева - Әбиемне яратам

```
Мин үзем шәһәрдә яшим,
Авылга кайтам җәен.
Чөнки ул авылда яши
Яраткан әбекәем.

Әбиемне яратам, яратам,
Әбием янына кайтам.

Әбием анда ялгыз яши,
Миңе сагынып көтә.
Мин кайткач, шатланып йөри,
Мине бик кунак итә.

Әбиемне яратам, яратам,
Әбием янына кайтам.

Мин үскәч, әбием белән
Авылда торачакмын.
Әбием картаеп киткәч,
Ярдәмче булачакмын.

Әбиемне яратам, яратам,
Әбием янына кайтам!..
```
