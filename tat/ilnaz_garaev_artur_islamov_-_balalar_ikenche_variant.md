# Оригинал

### Илназ Гәрәев / Артур Исламов - Балалар (икенче вариант)

```
Куш санагыңны, чык тизрәк Инетка,
Әйдә, сөйләшәбез электрон контентта,
Без бит киләчәкнең манарасыз патшалары,
Әни, китер акчаңны! Без clubstyle балалары.

Смартфон ята кесәдә — акыл нигә кирәк,
Җибәр берәр таблетка — булыр рәхәтерәк.
Төнге клуб безне дәшә, кара күпме кызлар,
Dj яңа Бетховен, ә без — чибәр йолдызлар.

Изге рокны алыштырып
Черек гламур канунына,
Исерткечләр салыштырып
Яшь тәнебез карынына…

Безнең кызлар ирләрнең серләрен беләләр,
Стильны нечкә сизәләр, ундүрткә җитәләр,
Купләребез кызларны бөтенләй яратмыйлар —
Әнә зәңгәр экранда матур итеп җырлыйлар.

Поп-йолдызлар өйрәткән надан фәнне укып
Белдек тормышның нигезен — рәхмәт, теледулкын!
Күгәргән күңел өстеннән ялт(ы)рыйк күлмәк киеп,
Әткәй биргән машинада борыннарны чөеп.
Йөрибез төнне яңг(ы)ратып, эзләгәндәй мәгънә,
Билгеле максат юк бездә, ә төп фикер менә:
Изгеләр дә сатыла, ришвәт — ныклы терәк,
Әй, әби, синең динең хәзер кемгә кирәк!

Изге рокны алыштырып
Черек гламур канунына,
Исерткечләр салыштырып
Яшь тәнебез карынына…
```
