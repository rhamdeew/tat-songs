# Оригинал

### Сибгат Хәким / Сара Садыйкова - Әй, язмыш, язмыш

```
Әй, язмыш, язмыш,
Туган җиремнең
Тугач та кергән
Миңа үз төсе.
Кая барсам да
Мин шат канатлы,
Ирекле, азат,
Һаман үз кеше.

Әй, язмыш, язмыш,
Теләмим башка,
Башка бер җәйне —
Мин үз җәемнән.
Аерма мине
Туган илемнән,
Кара икмәгем,
Сөтле чәемнән.

Әй, язмыш, язмыш,
Ташлама читкә,
Читтә сулырмын
Кайгыдан, моңнан.
Аерма мине
Өй түремдәге
Миләшләремнән,
Каеннарымнан.
```
