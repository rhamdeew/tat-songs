# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Дуслар булса кайгырма!

```
Матур җырдан гына тормый,
Гомерләрнең һәр көне.
Кечкәй генә шатлыктан да,
Эшлә, син бәйрәм үзең!

Кайгы хәсрәтләр дә узар,
Дуслар булса кайгырма!
Җыр-моңнардан,
Җан-дуслардан,
Ходай, безне аерма!

Болыт килер, яңгыр кояр,
Кояш чыгар азактан.
Кайгыларда, шатлыкларда,
Җан-дусларга җан атам.

Кайгы хәсрәтләр дә узар,
Дуслар булса кайгырма!
Җыр-моңнардан,
Җан-дуслардан,
Ходай, безне аерма!

Күктә кояш, һәрчак көләч,
Бәхет нурын өләшә.
Бәхетсез көннәр булмыйлар,
Ямь таба, белеп яшә!

Кайгы хәсрәтләр дә узар,
Дуслар булса кайгырма!
Җыр-моңнардан,
Җан-дуслардан,
Ходай, безне аерма!
```
