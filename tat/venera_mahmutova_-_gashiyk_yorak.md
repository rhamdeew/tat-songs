# Оригинал

### Венера Мәхмүтова - Гашыйк йөрәк

```
Кайчак гашыйк буласың,
Кайчак гашыйк итәсең.
Шул берәүнең йөрәгенең
Түрләренә үтәсең.

Ярату дип уйлама да,
Гашыйк булып йөремә.
Гашыйк булу бер мизгелле,
Яратуы гомерле.

Син дә башта гашыйк буласың,
Кинә яратыласың.
Шул берәүнең йөрәгендә
Гомерлеккә каласың.

Ярату дип уйлама да,
Гашыйк булып йөремә.
Гашыйк булу бер мизгелле,
Яратуы гомерле.
Ярату дип уйлама,
Гашыйк булып йөремә.
Гашыйк булу бер мизгелле,
Яратуы гомерле.
Гашыйк булу бер мизгелле,
Яратуы гомерле…
```
