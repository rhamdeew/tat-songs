# Оригинал

### Татар халык җыры - Баламишкин

```
Ват, җимер коймасын,
Урамга атын куймасын;
Кызлы кеше кызын тыйсын,
Егетләрне тыймасын.

Әнкәй миңа хат язган,
Утын бетте, кайт, диеп;
Хатка каршы хатлар яздым,
Әй почмагын як, диеп.

Зимагур булды башларым,
Бер нужа да күрмәдем;
Караңгы төн, ява яңгыр,
Өстемә бер күлмәгем.
```
