# Оригинал

### Рамил Чурагулов / Галинур Юламанов - Ассы җыры

```
Җаның сусаса җыр-моңга,
Ассыларга килеп чык.
Исәнлек-саулык утравы —
«Ассы» дигән патшалык.

Ассы, Ассы, җанда калчы
Матур, гүзәл җыр булып.
Инҗәр буен, таулар илен
Бар дөнья тора белеп.
Җирдәге оҗмахлар бит ул —
Ассы дигән гүзәллек.
Җанга, тәнгә дәва алып,
Тагын йөз ел яшәек!

Изге як ул Ассы ягы —
Җирдәге оҗмах-җәннәт.
Изге җанлы табиплары
Тәннәргә өсти куәт.

Ассы, Ассы, җанда калчы
Матур, гүзәл җыр булып.
Инҗәр буен, таулар илен
Бар дөнья тора белеп.
Җирдәге оҗмахлар бит ул —
Ассы дигән гүзәллек.
Җанга, тәнгә дәва алып,
Тагын йөз ел яшәек!

Батыр була, матур була
Ассыларда булганнар.
Тәннәр һәм җаннар яшәрә
Саф һава, саф сулардан.

Ассы, Ассы, җанда калчы
Матур, гүзәл җыр булып.
Инҗәр буен, таулар илен
Бар дөнья тора белеп.
Җирдәге оҗмахлар бит ул —
Ассы дигән гүзәллек.
Җанга, тәнгә дәва алып,
Тагын йөз ел яшәек!

Башкортстан, Россиямның
Син, Ассы, горурлыгы.
Уралымның гүзәл гөле,
Үзеңә чакыр тагын!

Ассы, Ассы, җанда калчы
Матур, гүзәл җыр булып.
Инҗәр буен, таулар илен
Бар дөнья тора белеп.
Җирдәге оҗмахлар бит ул —
Ассы дигән гүзәллек.
Җанга, тәнгә дәва алып,
Тагын йөз ел яшәек!
```
