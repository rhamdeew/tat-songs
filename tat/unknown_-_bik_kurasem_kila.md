# Оригинал

### Әхмәт Ерикәй / Рөстәм Яхин - Бик күрәсем килә

```
Торсам иртән, уйларымда йөртәм,
Ятсам — төшләремә керәсең.
Бик сагындым, иркәм, нишлим икән,
Бик күрәсем килә, күрәсем.

Әллә ят кешеләр, әллә үзең
Кавышу, өметләрен җимрәсең.
Көтә-көтә талды ике күзем,
Бик күрәсем килә, күрәсем.

Алмаштырмый хатка язган сүзләр,
Алмаштырмый сине бу рәсем.
Белмим инде, йөрәк ничек түзәр,
Бик күрәсем килә, күрәсем…
```
