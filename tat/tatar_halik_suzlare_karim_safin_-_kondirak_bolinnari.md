# Оригинал

### Татар халык сүзләре / Кәрим Сафин - Кондырак болыннары

```
Кондыракта чәчәк (җиләк) җыйдым
Печәнлек арасыннан;
Йөрәкләргә рәхәт була
Хуш исле һавасыннан.

Бигерәк күңелле
Кондырак болыннары;
Җәйнең матур кичләрендә
Яңгырый тальян моңнары.

Җәй көнендә шул Кондырак
Чәчкәләргә күмелә;
Туган якның болыннары
Мәңге якын күңелгә.

Бигерәк күңелле
Кондырак болыннары;
Җәйнең матур кичләрендә
Яңгырый тальян моңнары.

Алсуланып аткан чакта
Җәйнең матур таңнары;
Таң нурларына төренә
Кондырак болыннары.

Бигерәк күңелле
Кондырак болыннары;
Җәйнең матур кичләрендә
Яңгырый тальян моңнары.
```
