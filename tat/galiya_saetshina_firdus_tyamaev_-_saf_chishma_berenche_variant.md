# Оригинал

### Галия Сәетшина / Фирдүс Тямаев - Саф чишмә (беренче вариант)

```
Шундый матур кичне
Аңлаттык без хисне.
Ай да кача кереп болытка.
Безне таныштырган,
Безне кавыштырган
Ямьле, якты кичне онытма.

Саф чишмәгә төшеп,
Көмеш суын эчеп,
Серен чишеп җилгә таратма.
Чәчәк өзеп биргән,
Өзелеп сөям дигән
Сүзләреңне һич тә онытма.

Әйтелгән ул сүзләр,
Бәхетле, иң гүзәл
Мизгелләреңне дә онытма.
Кайларда йөрсәң дә,
Ни генә күрсәң дә
Мин барын син гел онытма.

Саф чишмәгә төшеп,
Көмеш суын эчеп,
Серен чишеп җилгә таратма.
Чәчәк өзеп биргән,
Өзелеп сөям дигән
Сүзләреңне һич тә онытма.

Чәчәк өзеп биргән,
Өзелеп сөям дигән
Сүзләреңне һич тә онытма.
Сүзләреңне һич тә онытма…
Сүзләреңне һич тә онытма…
```
