# Оригинал

### Илсур Тайсин / Альфред Якшимбетов - Язмышым, рәхмәт сиңа

```
Язмыш каратса карага,
Ак агырак күренер.
Аяз көннең кадерләрен
Давыл көннәр күрсәтер.

Әй, язмышым, рәхмәт сиңа
Шушы бәхетем өчен.
Ачы-төчесен татытып
Татлысын биргән өчен.

Без югалтмый аңламыйбыз,
Тазалык — байлык икән,
Мал-мөлкәттән дә кыйммәтле,
Иң затлы бүләк икән.

Әй, язмышым, рәхмәт сиңа
Шушы бәхетем өчен.
Ачы-төчесен татытып
Татлысын биргән өчен.

Сөюсез яшәтеп Ходай
Сөюгә юл ярадыр.
Килгән кайгы, бәлки, безгә
Бәхеткә юл ачадыр.

Әй, язмышым, рәхмәт сиңа
Шушы бәхетем өчен.
Ачы-төчесен татытып
Татлысын биргән өчен.
```
