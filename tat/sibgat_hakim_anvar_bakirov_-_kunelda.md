# Оригинал

### Сибгат Хәким / Әнвәр Бакиров - Күңелдә

```
Без йөргән аланнар еракта
Күмелә, күмелә, күмелә…
Сөйләмә, кирәкми, барсы да
Күңелдә, күңелдә, күңелдә.

Камышы шаулавын ишетәм
Күлнең дә, күлнең дә, күлнең дә;
Сөйләмә, кирәкми, барсы да
Күңелдә, күңелдә, күңелдә.

Яшьлек ул гомерендә бер генә
Кемнең дә, кемнең дә, кемнең дә;
Сөйләмә, кирәкми, барсы да
Күңелемдә, күңелемдә, күңелемдә.
```
