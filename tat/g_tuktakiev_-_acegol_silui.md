# Оригинал

### Г. Туктакиев - Әҗегол сылуы

```
Әҗегол үстерә
Кызларның сылуын.
Тиңнәре бар микән
Сезгә чит кызларның?

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!

Матурлый йөзеңне
Әҗегол таңнары.
Күзләреңдә балкый
Йолдызлар нурлары.

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!

Җидегән чишмәдәй
Өфенең сулары.
Биг(е)рәк сөйкемле
Әҗегол кызлары.

Әҗегол сылуы,
Әҗегол сылуы,
Өзелеп сагынам,
Юксынам йөзеңне!
```
