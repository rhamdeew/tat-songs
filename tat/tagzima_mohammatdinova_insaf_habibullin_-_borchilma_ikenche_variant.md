# Оригинал

### Тәгъзимә Мөхәммәтдинова / Инсаф Хәбибуллин - Борчылма (икенче вариант)

```
Бер ялгызың каршыласаң
Алсу таннар атканын,
Каршыларга туры килсә
Көтеп кояш батканын.

Борчылма син, нык түзем бул,
Төрле шиккә юл куйма.
Түземлек ул бу тормышта
Алтыннан койган койма.

Әгәр кайтып җитә алмасаң
Вәгъдәләшкән сәгатьтә,
Күңел шәмен сүндермичә,
Көт, бәгырем, әлбәттә.

Борчылма син, нык түзем бул,
Төрле шиккә юл куйма.
Түземлек ул бу тормышта
Алтыннан койган койма.

Икебезнең ышанычын
Күп сынар әле язмыш,
Ялгызың гына чакта да
Көчле булырга тырыш.

Борчылма син, нык түзем бул,
Төрле шиккә юл куйма.
Түземлек ул бу тормышта
Алтыннан койган койма.
```
