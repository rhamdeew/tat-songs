# Оригинал

### Ринат Мөслимов / Марат Мухин - Әнкәй (җиденче вариант)

```
Дөнья булгач, әнкәй, бар(ы)сы була.
Бер аз гына читтә торабыз.
Сиңа кайкач кына иркәләнеп
Бишектәге сабый булабыз.

Бүген дә бит, әнкәй, без йөрибез
Канат асларына сыенып.
Чәең генә бер дә суынмасын,
Без кайтырбыз бергә җыелып.

Морҗәләрдән тагын төтен чыга,
Пич эчендә тәмле бәлештер.
Зур-зур итеп безгә кисеп бирдең,
Кечкенәсе — синең өлештер.

Бүген дә бит, әнкәй, без йөрибез
Канат асларына сыенып.
Чәең генә бер дә суынмасын,
Без кайтырбыз бергә җыелып.

Туган яктан җылы җилләр исә,
Әллә әнкәй мунча якканмы?
Көне-төне безне көтәсеңдер,
Йокламыйча ачып капканы.

Бүген дә бит, әнкәй, без йөрибез
Канат асларына сыенып.
Чәең генә бер дә суынмасын,
Без кайтырбыз бергә җыелып.
```
