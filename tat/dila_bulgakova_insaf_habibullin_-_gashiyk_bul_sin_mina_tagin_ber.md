# Оригинал

### Дилә Булгакова / Инсаф Хәбибуллин - Гашыйк бул син миңа тагын бер

```
Офыкларны нурлап балкып киткән
Салават күпере күк бу гомер —
Көннәрне серле итәм дисәң,
Гашыйк бул син миңа тагын бер.

Ак көннәрне төннәр алыштыра,
Ай яктысы уртак, таңы бер —
Елларыңны җырлы итәм дисәң,
Гашыйк бул син миңа тагын бер.

Син сулаган һава — минем һава,
Без яшәгән гомер, бары бер —
Үткәннәрең үкенмәслек булсын дисәң,
Гашыйк бул син миңа тагын бер.
```
