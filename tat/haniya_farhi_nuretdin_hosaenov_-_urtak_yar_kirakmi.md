# Оригинал

### Хәния Фәрхи / Нуретдин Хөсәенов - Уртак яр кирәкми

```
Язлар белән бергә,
Керсез мәхәббәтем,
Ялындырмый гына кил инде.
Хисләр ташый ярдан,
Мөлдерәмә күңел,
Йөрәк көтә сине, бел инде.

Мөлдерәмә күңел,
Яз килгәнгә түгел, —
Мәхәббәттән ташый ярыннан.
Җырым белән бергә, һаман сиңа таба
Ашкынамын яшьлек таңыннан.

Уртак яр кирәкми,
Мәхәббәтем минем, —
Вакланмасын, керсез саф булсын;
Йөрәкләр дөрләсен,
Ятка сер бирмәсен,
Яз үтсә дә, сөю үтмәсен.
```
