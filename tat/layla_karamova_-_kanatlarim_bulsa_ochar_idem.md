# Оригинал

### Ләйлә Кәрамова - Канатларым булса очар идем

```
Кемгә рәхәт — әйтегезче миңа,
Тормыш сукмаклары катлаулы.
Кем уйлаган ерак төбәкләрдә
Салкын окопларда ятканны.

Канатларым булса очар идем,
Якыннарым, сезне күрергә.
Чәчләреннән сыйпап әнкәемнең,
Тиздән кайтам, көтче, дияргә.

Туган илебезне саклар өчен
Җилләренә каршы барабыз.
Гаиләм, балаларым, сезнең өчен
Еракларда утта янабыз.

Канатларым булса очар идем,
Якыннарым, сезне күрергә.
Чәчләреннән сыйпап әнкәемнең,
Тиздән кайтам, көтче, дияргә.

Имин көннәр җитсен иде диеп,
Шул теләкләр белән калабыз.
Рәсәй дигән гүзәл илебезне
Һәрвакытта саклый алабыз.

Канатларым булса очар идем,
Якыннарым, сезне күрергә.
Чәчләреннән сыйпап әнкәемнең,
Тиздән кайтам, көтче, дияргә.
```
