# Оригинал

### Дамир Гыйсметдин / Зөфәр Хәйретдинов - Ярты

```
Авылларда киң таралган
Исеме аның «Ярты».
Ансыз яши алмый бүген,
Ни яше дә, ни карты.

Печәнен дә әзерләтә,
Онын кайтарта тартып.
Утынын кистереп куя —
Эшне тиз тота ярты.

Ифрат зур ярдәме тия
Әби һәм бабайларга.
Яхшысын да, яманын да
Ярты чыга майларга.

Күпме көрәшсә дә влач,
Җиңәлмәде яртыны.
Инде хәзер хөрмәт итә
Ихтыяҗын халкының.

Ярты-йорты дип көлмәгез,
Әйләнеп бара бергә,
Имән кебек егетләрне
Күгәреп сылый җиргә (Җиңеп аудара җиргә).

Ярдәме аның бер көнлек,
Зыянын кем санаган.
Соңгы юлга шуның үзен
Озатыйк безнең арадан.
```
