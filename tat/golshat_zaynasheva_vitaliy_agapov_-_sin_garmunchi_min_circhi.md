# Оригинал

### Гөлшат Зәйнәшева / Виталий Агапов - Син — гармунчы, мин — җырчы

```
Сөйгән ярың кем, дисәгез,
Гади генә бер кеше.
Гармун уйнау, баян уйнау —
Аның бар белгән эше.

Син гармунчы, мин җырчы,
Мин җырчы, син гармунчы.
Гармуныңда матур итеп
Гел уйнап кына торчы.

Башка кызларга уйнасаң,
Көнләшүдән үләрмен.
Шул гармуның челпәрәмә
Ватылсын, дип теләрмен.

Син гармунчы, мин җырчы,
Мин җырчы, син гармунчы.
Гармуныңда матур итеп
Гел уйнап кына торчы.

Башка кызлар җырлаганда,
Гармуныңда уйнама.
Миннән башка бер генә дә
Кыз турында уйлама.

Син гармунчы, мин җырчы,
Мин җырчы, син гармунчы.
Гармуныңда матур итеп
Гел уйнап кына торчы.
```
