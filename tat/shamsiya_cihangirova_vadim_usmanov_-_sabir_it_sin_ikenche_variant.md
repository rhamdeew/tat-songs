# Оригинал

### Шәмсия Җиһангирова / Вадим Усманов - Сабыр ит син (икенче вариант)

```
Сабыр ит син димәгез,
Сабырларым сынды инде,
Сабыр итә-итә, итә
Сабырсыз булдым инде.

Сабыр ит син димәгәз
Сабырлыктан ни чара?
Сабыр итмәгән кешегә
Үтмәслек түгел ара.

Кушымта : икенче тапкыр

Сабырлыклар алтын түгел,
Сондыкларга сагырла
Йөзләренә сара кунып,
Йөрәкә йөмсагында!

Килми ярың, яныма,
Сабыр итә торгандыр.
Сабыр итмәс, килер диеп,
Мине көтеп торгандыр.

Сабыр ит син димәгәгез
Сабырлыклар итмим бүтән.
Сабыр канатлары сынды,
Оча алмим, җәяү китәм.

Кушымта : дүртенче тапкыр

Сабырлыклар алтын түгел,
Сондыкларга сагырла
Йөзләренә сара кунып,
Йөрәкә йөмсагында!ң
```
