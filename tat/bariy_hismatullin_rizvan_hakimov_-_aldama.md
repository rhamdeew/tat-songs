# Оригинал

### Барый Хисмәтуллин / Ризван Хәкимов - Алдама

```
Алдама мине, алдама,
Алдап юуаныч тапма.
Йөрәгемә утлар яккач,
Башкаларны яратма.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт — уенчык тугел,
Аның белән шаярма!

Күңелләрнең хисләрен
Сөйлимен мин кичләрен.
Эзләсәң дә таба алмассың
Минем кебек ишләрне.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт — уенчык тугел,
Аның белән шаярма!

Өзелгән җеп ялганса да,
Анын төене кала.
Уеннан уймак чыгарсан,
Күзләрдән яшьләр тама.

Алдама ла, әй, алдама,
Йөрәгемдә ут яна!
Мәхәббәт — уенчык тугел,
Аның белән шаярма!
```
