# Оригинал

### Тәслимә Низами - Өзгәләнмә, үзәгем

```
Ерак юлга бәхет теләп,
Назга төреп озатам.
Вакытлыча гына диеп,
Күңелемне юатам.

Өзгәләнмә, өзгәләнмә, үзәгем,
Мәңгелеккә генә китмә,
Сагышларга түзәрмен.

Ерактагы кояшым, тик
Яктыртасың көннәрен.
Ай нурларында тибрәлеп,
Синсез үткән төннәрем.

Өзгәләнмә, өзгәләнмә, үзәгем,
Мәңгелеккә генә китмә,
Сагышларга түзәрмен.

Язмыш уйный, вакыт сыный,
Ләкин без һаман бергә.
Аерылгач вакыт узмый,
Кавышкач, сизелми дә.

Өзгәләнмә, өзгәләнмә, үзәгем,
Мәңгелеккә генә китмә,
Сагышларга түзәрмен.
…Мәңгелеккә генә китмә,
Сагышларга түзәрмен…
```
