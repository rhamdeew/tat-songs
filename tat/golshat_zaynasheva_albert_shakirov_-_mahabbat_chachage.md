# Оригинал

### Гөлшат Зәйнәшева / Альберт Шакиров - Мәхәббәт чәчәге

```
Яран гөлем чәчәк атты
Язлар җиткәнне тоеп.
Үсә ике кызыл чәчәк
Бер-берсенә баш куеп.

Ике чәчәк әкрен генә
Киңәшәләр шикелле.
Мәнге бергә булыйк диеп
Серләшәләр шикелле.

Парлы чәчәк атмас иде
Яран гөл серледер ул.
Парлы яшәргә куша ул
Мәхаббәт гөледер ул.

Ике чәчәк әкрен генә
Киңәшәләр шикелле.
Мәнге бергә булыйк диеп
Серләшәләр шикелле

Бездә шулай парлы чәчәк
Булсак иде бер гел дә.
Серләшер иде йөрәкләр
Гөлләр серләшкән телдә.

Ике чәчәк әкрен генә
Киңәшәләр шикелле.
Мәнге бергә булыйк диеп
Серләшәләр шикелле.
```
