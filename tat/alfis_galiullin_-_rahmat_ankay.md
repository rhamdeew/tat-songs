# Оригинал

### Әлфис Галиуллин - Рәхмәт, әнкәй

```
Рәхмәт сиңа, рәхмәт, әнкәй,
Картлыгың тыныч узсын.
Кайгы-хәсрәтләр бер вакыт
Йорт ишеген какмасын.

Гомерем буе бурычлы мин
Синең алдыңда калам.
Гомерем буе сиңа рәхмәт
Әйтеп догалар кылам.

Әти белән тигез гомер
Итсәгез иде (Яшәгез әле) шулай.
Балаларның оныкларын
Күрергә язсын Ходай.

Әти-әни гел көчле сез
Булсагыз иде һәрчак.
Безне шулай, колач җәеп,
Көтсәгез иде һәрчак.
Безне шулай, колач җәеп,
Көтсәгез иде һәрчак…
```
