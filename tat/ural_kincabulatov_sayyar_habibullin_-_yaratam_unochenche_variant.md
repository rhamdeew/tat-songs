# Оригинал

### Урал Кинҗәбулатов / Сәйяр Хәбибуллин - Яратам (унөченче вариант)

```
Кыш уртасын сайлап алган
Ялган язым булып калма.
Бул юлымда, язмышымның
Бәхет язы булсаң гына.

Йөрәгемдә син — алсу таң,
Гөлләр булып мин нур йотам,
Бәхет язы кебек янам.
Сине күрсәм, хискә батам,
Сине күрсәм, чәчкә атам.
Мин яратам, яратам!

Ак каеннар кебек, язын
Тамыр җәям, яфрак ярам.
Син булмасаң, мин боегам,
Мин моңаям, мин саргаям.
```
