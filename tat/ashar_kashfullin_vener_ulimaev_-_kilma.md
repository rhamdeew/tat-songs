# Оригинал

### Әсхәр Кәшфуллин / Венер Улимаев - Килмә

```
Язгы назлы нурдай,
Килеп кердең җырдай,
Җәйгор булып җанны бизәдең.
Килдең соң син кайдан,
Төшкәнсеңдер айдан,
Өзелердәй чакта үзәгем.

Юк, кирәкми, килмә,
Язмышымнан көлмә,
Килмә миңа, килмә, гүзәлем.
Яшерәлмим, бәгърем,
Син минем йөрәгем,
Җаннарымны минем бизәдең.

Ләйсән булып яудың,
Тынычлыгым алдың,
Инде сине күрми түзәлмим.
Язгы назлы нурдай,
Килеп кердең җырдай,
Җәйгор булып җанны бизәдең.
```
