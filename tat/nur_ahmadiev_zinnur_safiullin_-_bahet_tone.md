# Оригинал

### Нур Әхмәдиев / Зиннур Сафиуллин - Бәхет төне

```
Каеннары ай-яй ак икән,
Сөю белән тулы чак икән,
Каеннардан якты моң тама.

Бу бит бәхет төне,
Эзлә бәхетеңне,
Юк, йоклама, егет, йоклама!

Егет яна кызга күз атып,
Сандугачлар йөри озатып,
Йолдызлардан энҗе сибелә.

Табышалар парлар,
Кавышалар ярлар,
Юк, бәгърем, юк бит син генә.

Бәхет төне узып китәрме,
Безгә бәхет биреп китәрме?
Каеннардан сора, каеннан.

Күңелемнең күркен
Яулый алмас беркем —
Исемнәрең каеп язылган!
```
