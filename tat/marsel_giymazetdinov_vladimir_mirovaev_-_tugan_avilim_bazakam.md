# Оригинал

### Марсель Гыймазетдинов / Владимир Мироваев - Туган авылым — Бәзәкәм

```
Басуларда — башак җыры,
Тургай моңы кырыңда.
Зифа буйлы сылуларың
Уйный йөрәк кылында.

Кушымта:
Әй Бәзәкәм, Бәзәкәм,
Языңны көтеп алам.
Тирәк, камыш, каен моңын
Йөрәк җылыма салам.

Комбайннар, тракторлар,
Егетләр җырлый кырда.
Бер симфония язарсың,
Иренми тыңлап тор да.

Кушымта:
Әй Бәзәкәм, Бәзэкәм,
Языңны көтеп алам.
Тирәк, камыш, каен моңын
Йөрәк җылыма салам.

Иген игә егет-кызың,
“Кара алтын” таба илгә.
Шатланам, Бәзәкәм, мин бит
Синдә дөньяга килгән,

К у ш ымта:
Әй Бәзәкәм, Бәзәкәм,
Языңны көтеп алам.
Тирәк, камыш, каен моңын
Йөрәк җылыма салам.
```
