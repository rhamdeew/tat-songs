# Оригинал

### Дамир Гыйсметдин / Илгиз Мәҗитов - Нәүрүз чибәре

```
Гасырлардан гасырларга,
офыклардан офыкларга
Оча-оча Нәүрүз кайта
тарихтан оныкларга

Әй, син, Нәүрүз чибәре
Язлар белән кил әле
Дөрләп янган йөрәгемә
Тамчыларны сип әле

Соң булса да, халкыбызга
Кайта Нәүрүз, кайтса яз
уянырга өнди безне
ерактагы кайтаваз

Әй, син, Нәүрүз чибәре
Язлар белән кил әле
Дөрләп янган йөрәгемә
Тамчыларны сип әле

Яңа елда умырзая
булсын Нәүрүз бүләге
иман иңсен кешеләргә
янсын яшьләр йөрәге

Әй, син, Нәүрүз чибәре
Язлар белән кил әле
Дөрләп янган йөрәгемә
Тамчыларны сип әле
```
