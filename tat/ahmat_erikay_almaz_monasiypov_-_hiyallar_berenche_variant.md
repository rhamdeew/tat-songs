# Оригинал

### Әхмәт Ерикәй / Алмаз Монасыйпов - Хыяллар (беренче вариант)

```
Ай нурыннан рәсемең ясар идем,
Ай нурларын кулыма тоталсам;
Таң җырларын сиңа, сиңа җырлар идем,
Таң җырларын әгәр оталсам.

Былбылларга карап торыр идем,
Синең төсле матур булсалар;
Мин гөлләрдән һәйкәл, һәйкәл корыр идем,
Әгәр гөлләр шиңми торсалар.

Күккә менеп йолдыз санар идем,
Яшь йөрәгең шуны теләсә;
Үзем йолдыз булып янар, янар идем,
Күңлең мине үзенә тиңләсә.
```
