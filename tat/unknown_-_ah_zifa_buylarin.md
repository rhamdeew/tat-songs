# Оригинал

### Гөлшат Зәйнәшева / Зиннур Гыйбадуллин - Аһ, зифа буйларың

```
Аһ, зифа буйларың,
Мин күреп туймамын,
Гел сиңа үрелә
Ак челтәр уйларым.

Көн туа алланып,
Чык тама балланып,
Дөньяда син барга
Яшим мин куанып.

Йөрегән юлларың
Тын гына буйлармын.
Гел сиңа үрелә уйларым.

Яз килә Иделгә,
Моң салып күңелгә,
Дөньяда мин барын
Син белмисеңдер дә! /Син белмисең генә!/
```
