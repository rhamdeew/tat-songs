# Оригинал

### Фәнил Әсәнов / Рим Хәсәнов - Исеңдәме? (беренче вариант)

```
Исеңдәме, иркәм,
Таныштык без икәү,
Җылы яктан кошлар кайтканда.
Көннәр аяз иде,
Күңелдә яз иде
Яшьлегебез чәчәк атканда,
Яшьлегебез чәчәк атканда.

Гөлләр үскән Димдә
Синең белән бергә
Каршыладык күпме язларны?!
Кошлар җырын тыңлап,
Тугайларны буйлап,
Чәчәкләрдән җыйдык назларны,
Чәчәкләрдән җыйдык назларны.

Язлар бик тиз үтә,
Көзен кошлар китә…
Минутларны куып, Дим ага.
Кошлар белән бергә,
Еллар очып үтә,
Әллә күңел шуңа моңлана,
Әллә күңел шуңа моңлана.
```
