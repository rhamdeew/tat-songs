# Оригинал

### Илдар Юзеев / Мирсәет Яруллин - Яшьлек дустыма

```
Ядкарь язам мин сиңа, дустым,
Китәм ерак якларга.
Яшьлегеңне искә төшерерсең
Бик ямансу чакларда

Онытма син: гел бергә йөрдек,
Аерылмас дус идек.
Каршыладык күпме таңнарны,
Бергә сөйдек ярларны

Ядкарь булсын ялкынлы яшьлек,
Эзе калсын елларда
Онытмаска сүзләр куешып, без
Кузгалабыз юлларга

Җырла син дә бу моңлы җырны,
Көннәрең булсын нурлы
Мәңге сакла дуслык сүзләрен,
Җирдә калсын әзләрең
```
