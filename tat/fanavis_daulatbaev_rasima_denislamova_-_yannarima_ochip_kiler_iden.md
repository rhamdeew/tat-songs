# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Яннарыма очып килер идең

```
Әллә инде сизгерлегең җитми,
Әллә инде инде үзем оялчан (кыюсыз).
Яннарыма очып килер идең
Хисләремне әгәр тоялсаң.

Мәхәббәткә киртә булып кара —
Мин юләрне тый соң,тыялсаң!
Яннарыма очып килер идең
Хисләремне әгәр тоялсаң.

Яннарыма очып килер идең,
Мәхәббәткә өстәп ялкының.
Сихерләнгән минем күңелемә
Булганы юк синдәй якынның.

Мәхәббәткә киртә булып кара —
Мин юләрне тый соң,тыялсаң!
Яннарыма очып килер идең
Хисләремне әгәр тоялсаң.

Ихлас микән моның сөюе дип
Тормас идең миннән шикләнеп.
Сине чорнап алган"ярамый"ның
Бәреп чыгар идең чикләрен.

Мәхәббәткә киртә булып кара —
Мин юләрне тый соң,тыялсаң!
Яннарыма очып килер идең
Хисләремне әгәр тоялсаң.
```
