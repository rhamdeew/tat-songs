# Оригинал

### Айрат Җамалетдинов - Мин, син, без, сез

```
Рәхмәт, дуслар, килгән өчен,
Рәхмәт сезгә, туганнар.
Бердәм булыйк, терәк булыйк,
Яңгырасын шушы юллар.

Мин, син, без, сез,
Бергә бәхетле без.
Мин, син, без, алар,
Бергә бәхетле торалар.
Мин, син, без, сез,
Барыбыз да бер тигез.
Мин, син, без алар,
Бергә бәхетле торалар.

Күрешербез сагынышып
Сезнең белән кабаттан.
Алкышларыгызны сагынып
Кайтырбыз без яңадан.

Мин, син, без, сез,
Бергә бәхетле без.
Мин, син, без, алар,
Бергә бәхетле торалар.
Мин, син, без, сез,
Барыбыз да бер тигез.
Мин, син, без алар,
Бергә бәхетле торалар.

Безнең йөрәк бергә типсен,
Моңаерлык булмасын.
Тормыш юлга чәчкә сипсен,
Күрешү ешрак булсын.

Мин, син, без, сез,
Бергә бәхетле без.
Мин, син, без, алар,
Бергә бәхетле торалар.
Мин, син, без, сез,
Барыбыз да бер тигез.
Мин, син, без алар,
Кулны бергә чабалар.

Мин, син, без, сез,
Бергә бәхетле без.
Мин, син, без, алар,
Бергә бәхетле торалар.
Мин, син, без, сез,
Барыбыз да бер тигез.
Мин, син, без алар,
Бергә бәхетле торалар.
```
