# Оригинал

### Билгесез - Синең тәрәзәңдә ут яна

```
Көзге көнне ярып алга атлыйм,
Салкын җилләр өрә дөньяга.
Туңган күңелләргә җылы биреп
Йөрәгемне алсу гөлгә төреп,
Синең тәрәзәңдә ут яна.

Нурлар балкый синең тәрәзәңдә,
Күңелләрдә шатлык уяна
Якынайтып бергә булыр көнне,
Синең тәрәзәңдә ут яна.

Көзге көннең вак яңгыры кабат
Тәрәзәңне юа иркәләп.
Яңгыр белән туктап сөйләшәм дип,
Тамчысыннан аның көнләшәм дип,
Күргәнгә ул сине иртәрәк.

Нурлар балкый синең тәрәзәңдә,
Күңелләрдә шатлык уяна.
Якынайтып бергә булыр көнне,
Синең тәрәзәңдә ут яна.
```
