# Оригинал

### Эльмира Җәлилова / Фирзәр Мортазин - Кадерле мизгел

```
Дөнья куып гомер үтә,
Онытыла дус-туган.
Яшәүнең һәрбер мизгеле
Мәшәкать белән тулган.

Яшик әле, яшен булып, ут булып,
Бу дөньяда гомердән дә
Кадерлерәк берни юк.

Туктап бер тын алыйк әле,
Онытыйк мәшәкатьне.
Дус-туган белән күрешик,
Яңартыйк мәхәббәтне.

Яшик әле, яшен булып, ут булып,
Бу дөньяда гомердән дә
Кадерлерәк берни юк.

Гомер ташкындай ургый да,
Үтә ул сизелми дә.
Фанилектә безнең яшәү
Бер татлы мизгел генә.

Яшик әле, яшен булып, ут булып,
Бу дөньяда гомердән дә
Кадерлерәк берни юк.
Бу дөньяда гомердән дә
Кадерлерәк берни юк…
```
