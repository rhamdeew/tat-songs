# Оригинал

### Татар халык җыры - Кашың кара, буең зифа (Чибәр кызга)

```
Карлыгандай кара, ай, күзләрең,
Бөрлегәндәй йөзләрең.
Шикәрләрдән тәмле, ай, балдан татлы
Сөям дигән сүзләрең.

Алма бакчасының юлы
Тәрәзәләргә туры.
Әллә сиңа бирелдеме
Бөтен дөньяның нуры?!
Кашың кара, буең зифа,
Күзең дә кара синең.
Кара күзеңә карасам,
Йөрәгем яна минем.

Бишмәтеңдә синең, ай, биш төймә,
Нигә төймәләмисең?
Аралар да безнең, ай, ерак түгел,
Нигә килгәләмисең?

Алма бакчасының юлы
Тәрәзәләргә туры.
Әллә сиңа бирелдеме
Бөтен дөньяның нуры?!
Кашың кара, буең зифа,
Күзең дә кара синең.
Кара күзеңә карасам,
Йөрәгем яна минем.
```
