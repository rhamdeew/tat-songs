# Оригинал

### Радик Шәрәфиев - Түгел

```
Кавыштырды безне, иркәм, ямьле язлар,
Безнең сөюдән көнләшеп атылды йолдызлар.
Ай нурында ачтым сиңа серләремне,
Минем йөрәк синдә яши күптән, сизәсеңме?
Хыялларда адашып мәхәббәттән көйдем,
Җавапсыз хисләрең, син: "Яратмыйм" — дисең.
Ялгандыр сүзләрең, килешә алмый күңел.
Тик бүгеннән инде мин синеке түгел,
Ә син минеке түгел.

Түгел иң якын, синең өчен,
Түгел иң җылы, синең өчен,
Түгел яратылган сөю өчен,
Түгел, түгел, түгел, түгел.

Күрәм салкын синең күзләреңдә,
Димәк, хисләр сүнде, димәк, сөю үлә.
Димәк, синең елмаюың мине бүтән җылытмас
Һәм иртәгә яз безгә бүләк итмәс наз.
Син югалырсың офыкта, күмелеп томаннарга,
Яңгыр юар эзләреңне, яңгыр аңлый бар(ы)сын да.
Ул арыды минем кебек синең күз яшьләреңнән,
Китсәң — кит, тик соңгысы булсын, кермә төшләремә.

Түгел иң якын, синең өчен,
Түгел иң җылы, синең өчен,

Түгел, түгел, түгел, түгел.
```
