# Оригинал

### Мидхәт Әбделманов - Әнкәем (дүртенче вариант)

```
Ак бәс дигән чәчең
Коеладыр, әнкәй,
Коеладыр, бик еш тарама.
Балам кайтыр диеп
Матур күлмәк киеп,
Буш юлларга зарыгып карама.

Без дигәнчә түгел
Чит илләрнең күге,
Юллары да чиксез, бик олы.
Кайтам дисәм дә мин
Яннарыңа әнкәем,
Кайта алмасмын, ахры, бу юлы.

Баксам эчләренә
Бүләк итеп сиңа
Нинди гөлләр генә чәчимен?
Ал чәчсә дә балаң,
Гөл чәчсә дә балаң,
Бурчым бетмәс сиңа әнкәем.
```
