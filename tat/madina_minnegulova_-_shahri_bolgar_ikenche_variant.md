# Оригинал

### Мәдинә Миңнегулова - Шәһри Болгар (икенче вариант)

```
Элек бу җирдә
Зур шәһәр булган.
Исеме аның —
Шәһри Болгар.
Бик күп илләрдән
Килгән кәрваннар.
Су юлы буйлап
Йөзгән кораблар.
Шәһри Болгар.

Торган урыны
Идел буенда.
Серле ташлар тик
Калган шул монда.
«Изге җир» — аяк
Басарга куркам.
Әче сагышка
Дучармын, дучар.
Шәһри Болгар.

Җиргә күмелгән
Урамнарында
Чулпы тавышы
Азмы чыңлаган.
Тирән моңга ия
Чын татар көен
Нечкә билле кызлар
Азмы җырлаган.
Шәһри Болгар.

Азан яңгырап торган
Манараларның хәрабәләре
Җиргә иңгәннәр.
Сиңа килгән татар
Балалары бүген
Нигез ташыңа
Башларын игәннәр.
Шәһри Болгар.
```
