# Оригинал

### Ризида Мөхәммәтҗанова - Юанычым

```
Иң зур байлык — оныгым,
Шатлыгым, куанычым,
Кайгыларны куа торган
Бердәнбер юанычым.

Кушымта:
Тәпи басты — тәп-тәп-тәп.
Мин җырлыймын, ул бии.
Әле сөйләшә белми,
Кушылып нидер көйли.

Килсә безгә кунакка,
Яктырып китә өем,
Күңелемнән моңнар ага —
Ул бит иң матур көем.

Кушымта.

Эшләр бетте бүгенгә,
Уйныйбыз икәү бергә,
Мин арыдым, син соң нигә
Арый белмисең бер дә?

Кушымта:
Әллә аңлады, көлә.
Мин җырлыймын, ул бии.
Әле сөйләшә белми,
Кушылып нидер көйли.
```
