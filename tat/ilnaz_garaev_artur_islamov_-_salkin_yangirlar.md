# Оригинал

### Илназ Гәрәев / Артур Исламов - Салкын яңгырлар

```
Мин түзә алмыйм әгәр дә
Син булмасаң янымда.
Шаулы һәм салкын шәһәрдән
Очам синең яныңа.

Яусын салкын яңгырлар,
Бүлеп ерак юлымны.
Синең ефәктәй куллар
Эретерләр җанымны,
җанымны…

Тик синең сихри күзләрең
Мине коткара ала.
Тик синең сихри сүзләрең
Миндә мәңгегә кала.

Яусын салкын яңгырлар,
Бүлеп ерак юлымны.
Синең ефәктәй куллар
Эретерләр җанымны,
җанымны…
Яусын салкын яңгырлар,
Бүлеп ерак юлымны.
Синең ефәктәй куллар
Эретерләр җанымны.
Яусын салкын яңгырлар,
Бүлеп ерак юлымны.
Синең ефәктәй куллар
Эретерләр җанымны…
```
