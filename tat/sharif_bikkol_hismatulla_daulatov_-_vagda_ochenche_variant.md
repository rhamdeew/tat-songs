# Оригинал

### Шәриф Биккол / Хисмәтулла Дәүләтов - Вәгъдә (өченче вариант)

```
Сөюләрем көчле, хәтәр яшен төсле,
Тик сак бул: ут белән уйнама, уйнама.
Вәгъдәмне мин бозсам, ятларга кул сузсам,
Гафу итәр бу, дип уйлама.

Пар түгел бит ул, тип, миңа генә бул, дип,
Хатлар язып сине йончытмам, йончытмам.
Төнен тәрәзә чиртеп, күрәемче бер, дип,
Чакырырсың, ә мин, мин чыкмам.

Мең газапҡа түзеп, мине дөнья гизеп
Эзләрсең син, ә мин табылмам, табылмам.
Мин табылмам — көтмә, синең өчен күктә
Балкыр йолдыз булып кабынмам.
```
