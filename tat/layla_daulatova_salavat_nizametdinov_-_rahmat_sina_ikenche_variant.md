# Оригинал

### Ләйлә Дәүләтова / Салават Низаметдинов - Рәхмәт сиңа (икенче вариант)

```
Инде нигә көннәр буе,
Инде нигә көннәр буе
Мәхәббәттән шашып йөрим.
Ярдан чыккан диңгез кебек,
Ярдан чыккан диңгез кебек
Күңелемнән ташып йөрим,
Күңелемнән ташып йөрим.

Мәхәббәттән исерткәнең өчен,
Уйларымны тилерткәнең өчен
Рәхмәт сиңа җаным,
Рәхмәт сиңа.
Синең белән кабат күрешүләр,
Иркәләүләр, назлап үбешүләр
Насыйп булсын тагын,
Рәхмәт сиңа.

Юллар күпме очраштырды,
Юллар күпме очраштырды,
Ә мин сине күрми идем.
Көннәр-төннәр буе болай,
Көннәр-төннәр буе болай
Сине уйлап йөрми идем,
Сине уйлап йөрми идем.

Мәхәббәттән исерткәнең өчен,
Уйларымны тилерткәнең өчен
Рәхмәт сиңа җаным,
Рәхмәт сиңа.
Синең белән кабат күрешүләр,
Иркәләүләр, назлап үбешүләр
Насыйп булсын тагын,
Рәхмәт сиңа.

Син иң затлы шәраб кебек,
Син иң затлы шәраб кебек
Бу мәхәббәт көчле икән.
Яратудан бөтен йөрәк,
Яратудан бөтен йөрәк
Янып бетәр төсле икән,
Янып бетәр төсле икән.

Мәхәббәттән исерткәнең өчен,
Уйларымны тилерткәнең өчен
Рәхмәт сиңа җаным,
Рәхмәт сиңа.
Синең белән кабат күрешүләр,
Иркәләүләр, назлап үбешүләр
Насыйп булсын тагын,
Рәхмәт сиңа.
```
