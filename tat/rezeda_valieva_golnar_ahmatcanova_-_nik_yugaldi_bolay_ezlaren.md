# Оригинал

### Резеда Вәлиева / Гөлнар Әхмәтҗанова - Ник югалды болай эзләрең?

```
Кайтырмын дип киткән идең,
Бәхет вәгъдә иткән идең,
Оныттыңмы биргән сүзләрең?
Кар-бураннар куды микән,
Язгы сулар юды микән,
Ник югалды болай эзләрең?

Күңелемнән ничек куйыйм,
Җаннарымнан ничек җуйыйм,
Күңелем һаман сине көтә бит.
Вәгъдәсез яр нигә кирәк,
Оныт аны, оныт йөрәк!
Яшь гомерем зая үтә бит.

Кошлар кайта, кошлар китә,
Гомеркәйләр һаман үтә
Аккан сулар кебек тын гына.
Язлар бәхет алып килми,
Якты таңнар шатлык бирми,
Күңелемдә сагыш-моң гына.
```
