# Оригинал

### Вәзыйх Фатыйхов / Гүзәлия - Яныңа агылам

```
Кәефем юк чакта
Мин сиңа киләмен,
Дәвасы синдә бит —
Бик яхшы беләмен.

Җил булып, нур булып
Яныңа агылам.
Минем соң кемем бар
Югалтмый сагынган?

Шатлыгым ташканда
Мин очам яныңа.
Мизгелләр эчендә
Мең кабат кабынам.

Җил булып, нур булып
Яныңа агылам.
Минем соң кемем бар
Югалтмый сагынган?

Аз гына югалтсам,
Мин сине сагынам.
Саф сөю алдында
Баш иям, табынам.

Җил булып, нур булып
Яныңа агылам.
Минем соң кемем бар
Югалтмый сагынган?
```
