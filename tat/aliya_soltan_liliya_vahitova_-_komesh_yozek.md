# Оригинал

### Алия Солтан / Лилия Вахитова - Көмеш йөзек

```
Аккан суга салдым йөзегемне,
Көмеш йөзек йөзеп китәр дип.
Әллә йөзегемне, әллә соң үземне,
Аккан судан нидер көтәм дип.

Бармагыма кигән көмеш йөзек,
Сөям, дип, әйтмәдең үзе,
Мине уйга салды, йокыларым алды,
Хәбәр көтеп арыдым түзеп.

Сөеп көюләрне уйламыйча,
Көмеш йөзек бүләк итмәгез,
Йөрәккә зар ягып, сагышларны тагып,
Әрнүләргә күмеп китмәгез.

Көмеш йөзегемне суга салдым,
Үткәннәр аксын дип суларда.
Өметләр багламыйм, очрашу юрамыйм,
Тик бер күрер идем шулай да.
Тик бер күрер идем шулай да…
```
