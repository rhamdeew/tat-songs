# Оригинал

### Билгесез / Сәйдәш Хәсәнҗанов - Яраттым шул, яраттым (икенче вариант)

```
Кыр гөлләре җыйдым сиңа,
Чәчеңә када әле, чәчеңә када әле.
Саг(ы)нуымны аңлар өчен
Күземә кара әле, күземә кара әле.

Сөенә кил, сайрар кошым,
Иңемә кунасыңмы, иңемә кунасыңмы?
Йөрәгемне бүләк итсәм,
Үземә буласыңмы, үземә буласыңмы?

Искән йомшак җилләр булып
Чәчеңә бер кагылыйм, чәчеңә бер кагылыйм.
Калмас өчен артларыңнан
Шәүләң булып тагылыйм, шәүләң булып тагылыйм.

Берни белми, берни күрми
Яраттым шул, яраттым, яраттым шул, яраттым.
Берне елмаюың белән
Бит үзеңә караттың, бит үзеңә караттың.

Бу сөюне мин бит әзер
Гомер буе сакларга, гомер буе сакларга.
Син булганда күңелемдә
Урын юк бит ятларга, урын юк бит ятларга.
Урын юк бит ятларга…
Яраттым шул, яраттым.
```
