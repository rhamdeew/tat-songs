# Оригинал

### Роберт Миңнуллин / Илгиз Закиров - Ак каеннар безне какмаслар

```
Тузганаклар кебек туздык инде,
Ак каеннар безне тапмаслар.
Киттеләр дә югалдылар диеп,
Тиргәмәгез безне, якташлар.

Ак каеннар безне көтә-көтә,
Күз йоммаслар, керфек какмаслар.
Кайтырлар дип, кайтмый калмаслар дип,
Көтәсезме безне, якташлар?

Картайсак та, каеннарга кайтып
Иелерләр безнең ак башлар.
Без сагынган кебек үзегез дә,
Сагынасыз микән, якташлар?

Ак каеннар безне аңлар әле,
Ак каеннар безне какмаслар.
Туган ягыбызга бер әйләнеп,
Без кайтырбыз әле, якташлар.
```
