# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Ышанам!

```
Нигә башын түбән идең,
Аерлабыз дисеңме?
Бәлки әле аерылмабыз,
Аерылмабыз ышанам!

Ышанам, ышанам,
Аерылмабыз ышанам!

Соңгы тапкыр күрәм диеп
Башын салма иңемә.
Бәлки әле кавышырбыз,
Кавышырбыз ышанам!

Ышанам, ышанам,
Кавышырбыз ышанам!

Без барасы озын юллар
Бер булырга охшаган.
Бәлки әле очрашырбыз,
Очрашырбыз ышанам!

Ышанам, ышанам,
Очрашырбыз ышанам!
```
