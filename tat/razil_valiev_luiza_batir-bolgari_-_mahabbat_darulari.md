# Оригинал

### Разил Вәлиев / Луиза Батыр-Болгари - Мәхәббәт дарулары

```
Янам инде, көям инде
Мәхәббәт учагында.
Нигә мине юатмыйсың
Тылсымлы кочагыңда?

Ничек кенә басыйм икән
Йөрәгем януларын?
Кайдан гына табыйм икән
Мәхәббәт даруларын?

Сагышланам таңнар белән,
Моңаям кичләр белән.
Нигә мине терелтмисең
Тылсымлы хисләр белән?

Ничек кенә басыйм икән
Йөрәгем януларын?
Кайдан гына табыйм икән
Мәхәббәт даруларын?

Ала алмыйм күзләремне
Кояшлы күзләреңнән.
Нигә мине назламыйсың,
Тылсымлы сүзләр белән?

Ничек кенә басыйм икән
Йөрәгем януларын?
Кайдан гына табыйм икән
Мәхәббәт даруларын?
```
