# Оригинал

### Самат Габидуллин / Газиз Дәүләтбирдин - Сагындым шулай

```
Еракта, офыкта
Яшен утлары, яшен утлары.
Еракта, шул якта
Минем дусларым, минем дусларым.

Еракта, офыкта
Кошлар тавышы, кошлар тавышы.
Күңелдә шул чакта
Сагыну (дуслар) сагышы, сагыну (дуслар) сагышы.

Кошларны сагынып,
Шомыртлар шаулый, шомыртлар шаулый.
Мин дә бит дусларны
Сагындым шулай, сагындым шулай.
```
