# Оригинал

### Инзилә Галиуллина / Илгиз Субаев - Ак болытлар (бишенче вариант)

```
Зәңгәр күктә ак болытта
Күрдем кебек үзеңне.
Төшем түгел, өнем икән,
Яшь томалый күземне.

Китеп барасың еракка,
Биштәрең иңнәреңдә.
Кемнәр юатыр, кадерлем,
Ялгыз мин төннәремдә.

Киткән юлдан кире кайт та,
Ал әле кочагыңа.
Бары төш кенә булган дип,
Калчы гел яннарымда.

Күктә йөзгән ак болытта
Күрдем синең үзеңне.
Сагынудан дәвалар юк,
Яшь томалый күземне.
```
