# Оригинал

### Иҗат Сабитов / Сәйяр Хәбибуллин - Сиңа (дүртенче вариант)

```
Сине күрсәм, кайдан килеп чыга
Татар булганымны хисләшү?
Күңелемдә була сымак чынга
Үз халыгым белән серләшү.

Ягымлысүз, моңлы күз карашың
Канда бар нәселне уята.
Гүя, иркәм, миндә синең аша
Үткән тормышларга таң ата.

Туя алмый карыйм күзләреңә,
Сокланамын ефәк йөзеңә.
Башым иеп төшәм тезләремә,
Сыялмагач хисләр сүземә.

Күп буыннар безгә саклаганнар
Мәхәббәтнең якты нурларын.
Алар сүнмәс өчен калдырганнар
Үзләренең йөрәк моңнарын.

Меңләп булган гашыйк хатыннарны
Күңелемнән сиңа охшатам.
Безгә тормыш биргән буыннарны
Барсын сөеп, сине яратам!
```
