# Оригинал

### Марат Гобәйдуллин - Кадерле кешемә

```
Икәү бергә үткән сукмаклардан
Йөрим әле һаман моңланып.
Бергә чаклар инде артта калган,
Сизелмәде гомер узганы.

Төшләремә кереп юатсаң да,
Рәхмәт әйтер идем үзеңә.
Язмыш җилләреннән ышыкларга
Йөрәк җылың җитми күңелгә.

Чәчләреңә япкан ак яулыгың
Җылысын саклый гүя бүген дә.
Аклык, сафлык сөйгән изге күңелең
Мәңге яши йөрәк түрендә.

Төшләремә кереп юатсаң да,
Рәхмәт әйтер идем үзеңә.
Язмыш җилләреннән ышыкларга
Йөрәк җылың җитми күңелгә.

Сабый чактагыдый сөеп-назлап,
Чәчләремнән кабат сыйпа әле.
Йөрәк һаман сине сагынып көтә,
Күңелемнең моңсу бер мәле.

Төшләремә кереп юатсаң да,
Рәхмәт әйтер идем үзеңә.
Язмыш җилләреннән ышыкларга
Йөрәк җылың җитми күңелгә.
```
