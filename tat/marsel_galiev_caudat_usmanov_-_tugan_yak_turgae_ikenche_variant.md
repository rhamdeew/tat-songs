# Оригинал

### Марсель Галиев / Җәүдәт Усманов - Туган як тургае (икенче вариант)

```
Ак болытлар ага,
Сулар ярга кага,
Тирбәлә җәй туган ягымда.
Тауларына менсәм,
Җылы җилләр исә,
Киң офыклар сыя җаныма.

Ак болыттай үтә,
Кабатланмый китә,
Бу дөньяның якты һәр көне.
Имин еллар төсен,
Биек алып менсен,
Учакларның зәңгәр төтене.

Бәхет тәмен беләм,
Туган җирдә генә
Иген кырларына күмеләм.
Монда гомер юлым,
Монда туган илем, / Монда туган моңым, /
Тургай булып сайрый күңелем.
```
