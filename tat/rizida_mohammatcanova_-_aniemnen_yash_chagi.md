# Оригинал

### Ризида Мөхәммәтҗанова - Әниемнең яшь чагы

```
Әниемнең яшь чагы да,
Ә минем — бала чагым.
Уйларым шул матур мәлгә
Әйләнеп кайта тагын.

Кушымта:
Үткәннәрне искә алып,
Моңлана әнием күңеле.
Ерактагы яшьлеге дә
Нурга гына тора күмелеп.

Икәүләшеп җиләк җыйдык
Чиләкләрне тутырып.
Сандугачның моңын тыңлап
Яланнарда утырдык	Кушымта:
Үткәннәрнең яланында
Моңлана әнием күңеле…
Җиләклекләр арасында
Яшьлеге дә тора күренеп…

Әни җырлый кунак барда,
Тальян гармунда уйнап.
Күзләрендә энҗе бөртек —
Моңлана соң ни уйлап?

Кушымта:
Үскән йортын искә алып
Моңлана әнием күңеле.
Ерактагы яшьлеге дә
```
