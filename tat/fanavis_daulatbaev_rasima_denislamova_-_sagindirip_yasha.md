# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Сагындырып яшә!

```
Мәхәббәтем-тансык бүләк җанга.
Мәхәббәтем-язмыш сынавы.
Көтмәгәндә көзен чәчәк атты,
Гаҗәп түгел аның соңлавы.

Мин тапмадым сине яшьлегемдә,
Бәхетемә соңга калгансың.
Әсир итте тәмам йөрәгемне
Нур сирпүче серле карашың.

Мин ышандым синең барлыгыңа,
Сине табарыма ышандым.
Белче шуны — җирдә юк синнән дә
Кадерлерәк кешем юк, җаным.

Мин тапмадым сине яшьлегемдә,
Бәхетемә соңга калгансың.
Әсир итте тәмам йөрәгемне
Нур сирпүче серле карашың.

Җанда булган ихлас сөю белән
Бәхет атлый икән янәшә.
Табындырып, һәрчак сагындырып
Бәгыркәем, мәңге син яшә!

Мин тапмадым сине яшьлегемдә,
Бәхетемә соңга калгансың.
Әсир итте тәмам йөрәгемне
Нур сирпүче серле карашың.
```
