# Оригинал

### Гыйльман Идрисов / Феликс Әбдрәшитов - Җырларым сезгә кала!

```
Безнең гомер мәңгелектә
Бер күз ачып йомган ара,
Шул ара да күпме вакыт ята?
Көннәр аша, айлар аша еллар ага,
Елларымнан җырлар кала, кала,
Җырларым сезгә, дуслар, кала!

Безнең гомер мәңгелектә
Бер күз ачып йомган ара,
Шул ара да шатлыкларым булган,
Сагышлардан бу дөньяда калган яра.
Сагышлардан җырлар, моңнар кала,
Җырларым сезгә, дуслар, кала!

Безнең гомер мәңгелектә
Бер күз ачып йомган ара,
Шул ара да миңә сөю килгән.
Мәхәббәтсез бу дөньяда яшәп кара,
Мәхәббәттән җырлар, моңнар кала,
Җырларым сезгә, дуслар, кала!
```
