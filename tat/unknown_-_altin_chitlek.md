# Оригинал

### Рамил Чурагулов / Илдар Шәрипов - Алтын читлек

```
Читлекләргә кошлар япмагыз сез,
Кош богавы — тимер читлек бит.
Газиз кошкайларга зәңгәр күксез,
Зәңгәр күксез яшәү читен бик.

Күккә очам, очам галәмнәргә…
Күккә ашу — кошлар язмышы.
Зәңгәр күктә якты җыр җырлаучы
Туган җирнең мин дә бер кошы.

Алтын читлекләрне якын күреп,
Байлык диеп үлә бик күпләр.
Азат кошкайларны кошлар итә,
Җырчы итә бары тик күкләр.

Күккә очам, очам галәмнәргә…
Күккә ашу — кошлар язмышы.
Зәңгәр күктә якты җыр җырлаучы
Туган җирнең мин дә бер кошы.

Алтын читлек һич тә болын түгел.
Күпме кошлар анда ялвара.
Бу дөньяны, моңга күмәм диеп,
Канат кагам мин дә — бичара!

Күккә очам, очам галәмнәргә…
Күккә ашу — кошлар язмышы.
Зәңгәр күктә якты җыр җырлаучы
Туган җирнең мин дә бер кошы.
```
