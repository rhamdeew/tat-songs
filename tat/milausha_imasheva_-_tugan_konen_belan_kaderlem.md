# Оригинал

### Миләүшә Имашева - Туган көнең белән, кадерлем

```
Кояш та сиңа елмая,
Тәрәзәңнән кара (ә)ле.
Туган көнең белән котлап,
Сиңа җыр яздым әле.

Сүзләрен алдым былбылдан,
Көем сандугачлардан.
Чишмәләрнең чылтыравы
Шушы җырга кушылган.

Назлы җилләр иркәләсен
Бүген сине, кадерлем.
Саклап кына йөретсеннәр
Сине фәрештәләрең.

Ап-ак бәхет шәлкемнәрен
Бүләк итсен Ходаем.
Туган көнең белән котлап,
Сиңа җырлар җырлыймын.

Туган көнең белән котлап,
Гөлләмәләр бирсеннәр.
Теләкләрнең иң изгесен
Гел сиңа теләсеннәр.

Күкләрдән йолдыз сорадым
Сибергә юлларыңа.
Кояш нурын, ай яктысын
Куштым мин җырларыма.

Сүзләрен алдым былбылдан,
Көем сандугачлардан.
Чишмәләрнең чылтыравы
Шушы җырга кушылган.
Чишмәләрнең чылтыравы
Шушы җырга кушылган…
```
