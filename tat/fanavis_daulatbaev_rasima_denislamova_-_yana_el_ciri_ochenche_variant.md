# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Яңа ел җыры (өченче вариант)

```
Хушлашыйк, Иске ел
Китәргә ашыга.
Елмаеп каршыла,
Яңа ел-каршыла!

Килде ул яңа ел,
Килде ул тагын да.
Җырлыйбыз, биибез
Без чыршы янында.

Хуш, сау бул Иске ел,
Яңа ел син яшә!
Кыш бабай ул безгә
Күчтәнәч өлешә.

Килде ул яңа ел,
Килде ул тагын да.
Җырлыйбыз, биибез
Без чыршы янында.

Кыш бабай — дус бабай
Түгел ул ялгызы.
Гүя таң йолдызы
Янында кар кызы.

Килде ул яңа ел,
Килде ул тагын да.
Җырлыйбыз, биибез
Без чыршы янында.

Җырлыйбыз, биибез
Чыршыны әйләнеп.
Күр, нинди күңелле —
Яңа ел бәйрәме!

Килде ул яңа ел,
Килде ул тагын да.
Җырлыйбыз, биибез
Без чыршы янында.
```
