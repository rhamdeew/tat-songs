# Оригинал

### Кәүсәрия Шәфикова / Мәсгуть Имашев - Яратып откан җырым син…

```
Яратып откан җырым син,
Җырламый ничек түзим?
Өзелми бит өметләрем,
Өзелми… ничек өзим?

Яратып откан җырым син,
Оныту мөмкин түгел.
Көзләрдә дә яздагыдай
Наз көтә минем күңел.

Яратып откан җырым син,
Яратып җырлыйм һаман.
Сүн, димә, сулар да сипмә,
Янасым килеп янам.
```
