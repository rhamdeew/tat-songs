# Оригинал

### Илфат Мусин / Җәүдәт Гыйльманов - Такыя

```
Болыннарда йөрдем,
Күп чәчәк күрдем.
Матурларын өзеп,
Такыя үрдем.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.

Нинди генә төсләр
Юк такыямда:
Ак та,кызыл,зәңгәр
Барсы бар анда.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.

Бәләкәй мин үзем
Җәй кызы кебек.
Куйдым такыяны
Башыма киеп.

Минем такыяма
Кояш сыенган.
Шушы түгәрәккә
Ямьле җәй сыйган.
```
