# Оригинал

### Гөлшат Зәйнәшева / Луиза Батыр-Болгари - Әниләр бәйрәме (беренче вариант)

```
Беренче тамчылар тама,
Яз килүгә шатланып.
Язлар җиткәч, таңнар ата
Зәңгәрләнеп, сафланып.

Алып кил безгә, яз,
Чәчәкләр бәйләме.
Килеп җитте матур бәйрәм
Өниләр бәйрәме.

Әниләр мәңгелек шатлык,
Язгы шатлык, язгы нур.
Әниләргә багышлана
Дөньяда иң назлы җыр.

Алып кил безгә, яз,
Чәчәкләр бәйләме.
Килеп җитте матур бәйрәм
Өниләр бәйрәме.

Үз әнкәсе, үз газизе
Һәркемнең күңелендә.
Әни-әни дип сайрыйдыр
Кошлар да үз телендә.

Алып кил безгә, яз,
Чәчәкләр бәйләме.
Килеп җитте матур бәйрәм
Әниләр бәйрәме!..
```
