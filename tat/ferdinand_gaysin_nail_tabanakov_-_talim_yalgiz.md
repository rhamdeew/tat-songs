# Оригинал

### Фердинанд Гайсин / Наил Табанаков - Талым ялгыз

```
Яшьлегемә кайтып киләм,
Хисләрдән җырлар уреп.
Су буенда — ялгыз талым,
Шатланыр инде куреп.

Су буенда талым ялгыз,
Аерды безне язмыш,
Бәхет эзләп еракларга,
Дуслар, киткәнмен ялгыш.

Күрмәгәндә шактый еллар,
Талым картаеп киткән,
Хәлләреңне белмәсәм дә,
Күнелдә сине йөретәм.

Су буенда талым ялгыз,
Аерды безне язмыш,
Бәхет эзләп еракларга,
Дуслар, киткәнмен ялгыш.

Ялгыз җаным, ялгыз талга
Кайткан саен сыенам.
Яннарыңда сүнмәс яшьлек
Көтәр кебек тоела.

Су буенда талым ялгыз,
Аерды безне язмыш,
Бәхет эзләп еракларга,
Дуслар, киткәнмен ялгыш.
```
