# Оригинал

### Рамилә Абдуллина - Мәхәббәт (бишенче вариант)

```
Мәхәббәт, мәхәббәт-
Зур хисләр ташкыны,
Күкләргә мендергән
Хыяллар баскычы.
Бик күпләр, бик күпләр
Тик сиңа табына,
Күктәге йолдызлар
Шикелле кабына.

Ялкыныңда мин янам,
Салкыныңда мин туңам.
Мәхәббәт, мәхәббәт,
Синең колың мин булам.

Мәхәббәт, мәхәббәт —
Зур хисләр дәрьясы.
Синсез һич торалмый
Кешелек дөньясы.
Һәркемне бәхетле
Итәсең, мәхәббәт,
Тик үзең ярдәм ит:
Кемгә кемнәр кирәк!

Ялкыныңда мин янам,
Салкыныңда мин туңам.
Мәхәббәт, мәхәббәт,
Синең колың мин булам.

Мәхәббәт, мәхәббәт —
Гашыйклар дөньясы.
Сөйгәнем янында
Һәрвакыт булсамчы.
Мәхәббәт, мәхәббәт,
Адашсам — юл күрсәт,
Тайпылмый барырга
Миңа син гел кирәк.

Ялкыныңда мин янам,
Салкыныңда мин туңам.
Мәхәббәт, мәхәббәт,
Синең колың мин булам.
```
