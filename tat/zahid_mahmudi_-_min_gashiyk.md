# Оригинал

### Заһид Мәхмүди - Мин гашыйк

```
Мин кочаклыйм сине шашып,
Сине, гүзәлем.
Алсу йөзеңнән үпмичә
Ничек түзәлим?

Назлы сүзләр көтмә миннән,
Үбәргә ашык.
Мәхәббәт сүзләр яратмый,
Мин — тиле гашыйк.

Мин үбәрмен сине шашып,
Син дә бул уяу.
Хисләремнән айнымасам,
Көтмә син аяу.

Тын куыргыч үбүләрем
Чәчәк күреп ал.
Бергә чаклар бик тиз үтәр,
Кадерен белеп кал.

Мин көләрмен тиле сыман,
Ә син еларсың.
Югалырмын хыял кебек,
Яшең юарсың.

Назлы сүзләр көтмә миннән,
Үбәргә ашык.
Мәхәббәт сүзләр яратмый,
Мин — тиле гашыйк.
```
