# Оригинал

### Разил Вәлиев / Фуат Әбүбәкеров - Чаптарым

```
Нинди язмыш безне көтә икән?
Кая чаба икән чаптарым?
Аргамагым эзләп чаба төсле
Бер кайгысыз колын чакларын.
Аргамагым чаба, һаман чаба, —
Тирә-юньдә бары ут кына…
Берәүләргә алтын, алтын яңгыр ява,
Алтын ирләр яуда, яуда ятып кала…
Юллар илтә безне упкынга.

Аргамагым, күпме чапсаң да син,
Әйләнмәссең кабат колынга.
Икебезнең язмыш бер үк төсле —
Икебез дә җирдә кол гына.
Безнең өйгә бәхет килер микән?
Булыр микән бәйсез чакларым?
Кан коюлар безгә, безгә кирәк түгел,
Ирек даулап ярсый, ярсый безнең күңел, —
Ярсып чаба минем чаптарым.

Аргамагым илләр буйлап чаба,
Ирек даулап керә яуларга.
Кара көннәр, янып, артта кала,
Таңнар ата, офык аллана.
Ирләр белән бергә илләр чаба,
Эзләп чаба бәйсез чакларын…
Батыр ирләр, илләр яуда бәхет таба…
Бәхет эзләп һаман, һаман алга чаба
Ялкын яллы кызыл чаптарым!
```
