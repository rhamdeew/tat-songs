# Оригинал

### Зәйтүнә Галиева / Раил Едиханов - Язны сагыну (икенче вариант)

```
Дим сулары шашып,
Ярларыннан ташып,
Шаулап ага язгы ташкында.
Күкрәгемдә ярсып,
Йөрәк тибә шашып,
Туган якларыма талпына.

Кайталмадым тагын,
Сагынсам да барын:
Дим буеның бөдрә талларын,
Бала чакның язын
Күрәлмәдем тагын

Авылымның язгы таңнарын.
Күпме гомер үткән,
Тагын язлар җиткән.
Дим өстендә бозлар киткәндер.
Су буенда таллар,
Моңсуланып ярда,
Өметләнеп мине көткәндер.
```
