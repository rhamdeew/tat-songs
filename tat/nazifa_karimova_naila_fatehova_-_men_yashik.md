# Оригинал

### Нәзифә Кәримова / Наилә Фатехова - Мең яшик

```
Кайда татар яши — шунда татар иле.
Мәскәүләр дә күптән безнең туган ил.
Кайда гына бакма — татар урамнары
Моңлы Тукай килде, данлы Муса Җәлил.

Кушымта:
Бер булыйк,
Бергәләп мең яшик.

Мәскәүләрдә яшәү — үзе бер зур сынау.
Күз яшьләренә һич Мәскәү ышанмый.
Ничә гасыр инде алышына буын.
Татар турында төрле-төрле сүз тынмый.

Без бит шундый милләт: без бетмибез әле,
Мең үлеп түгелме мең тапкыр терелгән.
Чит, диеп, читләтмәгез — Татарстан чиге
Сызып үтә татарның йөрәк түреннән.
```
