# Оригинал

### Мансур Гыйлаҗев / Азат Хөсәенов - Сылукай

```
Дулкын-дулкын булып кала
Агымсудан су алсаң.
Үкенерсең, Сылукаем,
Күз алдыңнан югалсам.

Айкала суларыгыз,
Чайкала суларыгыз.
Кайсы йолдызлар астында
Туган урыннарыгыз?

Айның унбише булган,
Түгәрәкләнеп тулган.
Сагынырсың, саргаерсың —
Шул чагында мин булмам.

Айкала суларыгыз,
Чайкала суларыгыз.
Кайсы йолдызлар астында
Туган урыннарыгыз?

Кашыңның каралары,
Тар икән аралары.
Синең кебек булса булыр
Сандугач балалары.

Айкала суларыгыз,
Чайкала суларыгыз.
Кайсы йолдызлар астында
Туган урыннарыгыз?
```
