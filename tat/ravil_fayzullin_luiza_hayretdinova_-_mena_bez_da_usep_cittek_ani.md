# Оригинал

### Равил Фәйзуллин / Луиза Хәйретдинова - Менә без дә үсеп җиттек, әни…

```
Без үстек,
Үсеп җиттек, әни
Менә без дә үсеп җиттек.

Кыр казлары
Түбәбездән үтә,
Элеккечә үтә,
Үтә икән…
Кыр казлары элеккечә
Түбәбездән үтә икән.
Күк һаман шул, шул,
Казлар гына бүтән икән.

Без үстек,
Үсеп җиттек, әни.
Менә без дә үсеп җиттек.

Сабан туйлар
Тау башында үтә.
Элеккечә үтә,
Үтә икән.
Сабан туйлар элеккечә
Тау башында үтә икән.
Тау һаман шул, шул,
Кызлар гына бүтән икән…

Без үстек,
Үсеп җиттек, әни
Менә без дә үсеп җиттек

Синен күзләр
Без кайтуны көтә,
Элеккечә көтә,
Көтә икән
Синең күзләр элеккечә
Без кайтуны көтә икән.
Күзләр гел шул, шул,
Яшьләр генә бүтән икән!.
```
