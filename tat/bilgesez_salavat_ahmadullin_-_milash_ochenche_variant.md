# Оригинал

### Билгесез / Салават Әхмәдуллин - Миләш (өченче вариант)

```
Өй түрендә миләш
Иелә җилләр тигәч,
Тәрәзәгә карап кызара.
Өзгәләнмә миләш —
Булчы якын сердәш,
Мәхәббәтен көткән кызларга.

Язын-көзен димә,
Башкаеңны имә
Йөрәкләрен тыңла көн дә кич.
Әйтмә таң җиленә
Тик бер сүзең генә
Ятлар берүк сизә күрмәсен.

Өй түрендә миләш
Авыл берни белми
Ник кызарган болай димәсләр.
Яшерен яна-яна
Кулын болгап кала
Әллә кызлар, әллә миләшләр.
```
