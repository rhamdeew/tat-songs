# Оригинал

### Фәнәвис Дәүләтбаев / Рәсимә Денисламова - Күбәләк кар

```
Эзләсәм дә таба алмам
Күңелемә дәва.
Ак чәчәкләр булып җиргә
Күбәләк кар ява.

Күбәләк кар, бул карлыгач,
Талпынып оч җилдә!
Сагышлардан арыныймын
Сине күреп мин дә.

Күбәләк кар, күбәләк кар —
Йолдызларга тартым.
Сагыш тулы күңелемне
Чорнап алган ялкын.

Күбәләк кар, бул карлыгач,
Талпынып оч җилдә!
Сагышлардан арыныймын
Сине күреп мин дә.

Гаҗәп түгел, моңсуланып
Буран көтсә җаным.
Ялгыз гына мәхәббәтем
Үттем мин җилләрен.

Күбәләк кар, бул карлыгач,
Талпынып оч җилдә!
Сагышлардан арыныймын
Сине күреп мин дә.
```
