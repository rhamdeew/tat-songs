# Оригинал

```
Җиргә килә кеше үзе теләп түгел
Түгел үзе теләп алган вакыт илдә
Язмышның кулында бары
Бер уенчык кына, —
Бөекме син, кеше, уйла,җирдә?!

Язмышның кулында бары
Бер уенчык кына, —

Үзгәртергә аны бармы хакың, Кеше?
Әллә түзеп гомер буе
Көрәшмичә генә,
Яшәргәме җирдә кысып тешне?!

Язмыштан узмышлар булмас дисәләрдә,
Кеше күңелендә яши хыял кошы.
Аңа талпынырга ирек бирсәң
Буладыр ул
Үз кулың белән язып язмышны.
```
