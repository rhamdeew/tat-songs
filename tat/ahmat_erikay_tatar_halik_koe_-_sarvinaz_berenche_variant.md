# Оригинал

### Әхмәт Ерикәй / Татар халык көе - Сәрвиназ (беренче вариант)

```
Көн аяз, килә яз,
Чәчәккә күмелеп.
Син дә кил, Сәрвиназ,
Шатлыклы көн кебек.

Кил, иркәм, нур сыман
Балкып кил яныма.
Ягымлы җыр сыман
Рәхәт бир җаныма.

Кичтем мин ут-сулар
Ил тыныч торсын дип.
Күрешүләр, кавышулар
Күңелле булсын дип.

Син килсәң, Сәрвиназ,
Күңелемдә җылы яз.
Шатлыгым, бәхетем зур,
Хәсрәтем, кайгым аз.

Кил, бәгърем, бергәләп
Кырларга чыгарбыз.
Илнең саф һавасын
Иркенләп суларбыз.
```
