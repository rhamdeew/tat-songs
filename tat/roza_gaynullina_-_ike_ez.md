# Оригинал

### Роза Гайнуллина - Ике эз

```
Ак кар өстендә
Ике эз калган.
Ике сукмакны
Кемнәрдер салган.

Ике эз бара
Һаман янәшә.
Очрашыр чакта
Кинәт читләшә.

Бер-берсен эзләп,
Озак барганнар.
Юл киселешен
Күрми калганнар.

Тәкъдир аларны
Кавыштырмаган.
Утка салмаган,
Саташтырмаган.

Ике эз калган
Ак кар өстендә.
Ике ялгыз моң —
Сагыш эчендә.

Эзләрне күмеп,
Ап-ак кар яуган.
Ике ялгыз моң —
Очрашмый калган.
```
