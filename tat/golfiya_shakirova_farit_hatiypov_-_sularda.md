# Оригинал

### Гөлфия Шакирова / Фәрит Хатыйпов - Суларда

```
Суларда сал йөзәдер лә,
Елгаларны гизәдер.
Өзелеп лә сагынуыңны
Күңелкәем сизәдер.

Суларда сал тирбәлә лә
Яннарыма килгәлә.
Сирәк кенә булса да соң
Хәлләремне белгәлә.

Салларда йөзәр идем дә,
Сагынуга түзәр идем.
Кирәк дисәң, синең өчен
Зур сулар гизәр идем.

Сузылганнар тагында ла,
Саллар елга ярында.
Нигә икән без аерым
Җанда сөю барында?
```
