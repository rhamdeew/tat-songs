# Оригинал

### Гөлкәй Вәлиәхмәт / Раил Өметбаев - Сиңа гына (өченче вариант)

```
Бүлек итимме йолдызлар,
Син әйт кенә кайсысын.
Бары синең өчен генә
Сайлыйм иң-иң яктысын.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.

Сиңа гына чәчәк ата
Бакчадагы гөлләрем.
Бәхет-шатлыкка күмелә
Синең янда көннәрем.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.

Канатланып, канатланып
Яныңа кайтып киләм.
Минем өчен бик кадерле
Икенлегеңне беләм.

Сиңа гына, сиңа гына
Ай-кояшның нурлары.
Сиңа гына багышлана
Җырларның матурлары.
```
