# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Күзләрем тик сине эзлиләр

```
Бакчага ямь керә,
Хуш исләр бөркелә,
Сиреньнәр чәчәккә бөреләнсә.
Син сөйгән сиреньнәр,
Син күргән сиреньнәр —
Күңелемә иң якын бер нәрсә.

Җырларың тыңласам,
Син барны уйласам,
Күңелемне сүрелмәс моң сара.
Канатлар кайрылган,
Юлыбыз айрылган —
Без бергә булырга юк чара.

Мин җырлап юанам,
Гөлләргә күз салам.
Гөлләр дә: юксынма, түз, диләр.
Сызылып таң атканда,
Ал шәфәкъ батканда
Күзләрем тик сине эзлиләр.
```
