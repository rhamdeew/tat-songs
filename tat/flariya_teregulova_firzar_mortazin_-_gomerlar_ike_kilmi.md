# Оригинал

### Флария Терегулова / Фирзәр Мортазин - Гомерләр ике килми

```
Җилләр иркәләр чәчеңнән, —
Син икән, — диеп уйла.
Аларга серемне чиштем,
Сибелсен синең уйга.

Уйга калырсың әле син,
Сагышка күмелерсең.
Гомерләр ике килми бит,
Соң булыр, үкенерсең.

Болындагы чәчкәләрне,
Өзмә, мин диеп уйла.
Хуш исләре исертерләр,
Хәтта калырсың уйга.

Уйга калырсың әле син,
Сагышка күмелерсең.
Гомерләр ике килми бит,
Соң булыр, үкенерсең.

Бәхетең буласым килде,
Үтеп киттең янымнан.
Мөлдерәмә мәхәббәтем
Яңгыр булып явылган.

Уйга калырсың әле син,
Сагышка күмелерсең.
Гомерләр ике килми бит,
Соң булыр, үкенерсең.

Хыялларым гына булып
Калырсың микән инде?
Сине өзлеп сөюемне,
Күрмәссең микән инде?

Уйга калырсың әле син,
Сагышка күмелерсең.
Гомерләр ике килми бит,
Соң булыр, үкенерсең.
```
