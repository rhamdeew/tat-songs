# Оригинал

### Роберт Сәлахиев / Альфред Якшимбетов - Сагынганда (икенче вариант)

```
Сагынганда яңгыр тамчылары
Тәрәзәмә чиртеп китәләр.

Яннарыңа бүген очар идем,
Ул бит сине көтә дисәләр.

Сагынуларны, бәлки, басар иде
Сәламеңне алып килсәләр.

Кошлар булып бүген очар идем,
Ул сагына, көтә, дисәләр.

Сагынганда назлы таң җилләре
Син яшәгән якка иссәләр.

Алар белән бергә очар идем,
Ул күптәннән көтә, дисәләр.
```
