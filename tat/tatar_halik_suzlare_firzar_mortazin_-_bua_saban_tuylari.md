# Оригинал

### Татар халык сүзләре / Фирзәр Мортазин - Буа сабан туйлары

```
Зөя өсләрендә акчарлаклар,
Буа ягын сагынып кайталар.
Зөя буе сабан туйларын
Буалылар көтеп алалар.

Их, Зөя урманнары,
Буа сабан туйлары!
Иделгә барып кушыла
Хезмәт, бәйрәм җырлары.

Зөя дулкыны, иген кырлары
Чайкала Буа җырында.
«Сабан туе»на бииләр
Буа сабан туенда.

Их, Зөя урманнары,
Буа сабан туйлары!
Иделгә барып кушыла
Хезмәт, бәйрәм җырлары.

Килегез Буа якларына,
Котлы сабан туйларына.
Сокланып карап китәрсез
Сулары, кошларына.

Их, Зөя урманнары,
Буа сабан туйлары!
Иделгә барып кушыла
Хезмәт, бәйрәм җырлары.
```
