# Оригинал

### Илдар Ногманов - Китмә диеп ялынам

```
Ялындырган булып йөрисең,
Яратсаң да мине син үзең.
Килмә-йөрмә, дисең артыңнан,
Куасың син мине яныңнан.

Китмә-китмә, диеп ялынам,
Кочагыңа кереп сарылам,
Сагынамы, диеп уйлыйсың,
Ышан, җаным, сине сагынам.

Үпкәлисең юк-бар сүземә,
Карамыйсың хәтта күземә,
Елмаймый янымнан үтәсең,
Син бит миннән шулай көләсең..

Китмә-китмә, диеп ялынам,
Кочагыңа кереп сарылам,
Сагынамы, диеп уйлыйсың,
Ышан, җаным, сине сагынам.

Ә узең бит, үзең, көнлисең,
Читтән генә мине күзлисең,
Күзләр салсам әгәр читләргә,
Эчтән янып-көеп йөрисең.

Китмә-китмә, диеп ялынам,
Кочагыңа кереп сарылам,
Сагынамы, диеп уйлыйсың,
Ышан, җаным, сине сагынам.
```
