# Оригинал

### Миләүшә Әфләтунова / Илдус Дәүләнов - Кызыл иннек

```
Син кирәк җаныма,
Алсу таң ашыга
Җылысын бирергә.
Ә син хисләремне
Телисеңдер белергә.
Бары тик ай сизә:
Ике гашыйк идек,
Мәхәббәт төсендә
Синең кызыл иннек.

Сөюдән йөрәкләр яна,
Сөюдән йолдызлар кабына.
Белә тик аны ай гына:
Син кирәк җаныма,
Кирәк минем җаныма.

Карашымнан аңла
Сөюнең сүзләрен.
Тик синең яныңда
Елмая күзләрем.
Хәзер ул ай белә:
Салды сөю битлек,
Синең иреннәрдә
Минем кызыл иннек.

Сөюдән йөрәкләр яна,
Сөюдән йолдызлар кабына.
Белә тик аны ай гына:
Син кирәк җаныма,
Кирәк минем җаныма.
Син кирәк җаныма…
Кирәк минем җаныма…
Син кирәк җаныма…
```
