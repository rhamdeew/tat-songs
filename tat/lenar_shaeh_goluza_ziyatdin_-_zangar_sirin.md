# Оригинал

### Ленар Шәех / Гөлүзә Зиятдин - Зәңгәр сирин

```
Зәңгәр сирин, дымлы һава.
Тышта җылы яңгыр ява.
Ә мин сирин ышыгында —
Шундый рәхәт, ышык монда.

Җил-яңгыр да миңа тимәс,
Күңелкәең иләс-миләс.
Янып-көеп йөргән чагың,
Чәчәк аткан сөю багың.

Зәңгәр сирин арасында
Рәхәтләнеп каламын да,
Чәчләремә сирин кадап,
Елмаямын күккә карап.

Кулларымда зәңгәр чәчәк.
Йөрәгемдә яшәячәк
Мәхәббәтнең иң зурысы,
Сөенечнең иң олысы.

Иренемдә бәхет дымы,
Янәшәмдә сирин сыны.
Офыкларны нурга манчып,
Шаян кояш тора качып.

Кулларымда зәңгәр чәчәк.
Йөрәгемдә яшәячәк
Мәхәббәтнең иң зурысы,
Сөенечнең иң олысы.
```
