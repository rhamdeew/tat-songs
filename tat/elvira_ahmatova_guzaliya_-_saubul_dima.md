# Оригинал

### Эльвира Әхмәтова / Гүзәлия - Саубул димә

```
Саубуллашмыйм әле синең белән,
Әйтер сүзләр безнең бетмәгән.
Үтте сөю, диеп, уйласаң да,
Йөрәкләрдә учак сүнмәгән.

Сау бул, димә, сау бул, димә миңа,
Хушлашырга бер дә теләмим!
Онытырга син дә әзер түгел,
Карашыңнан күреп тоям мин.

Саубуллашмыйм әле синең белән,
Синсез, гүя таңсыз иртәләр.
Үз — үземнән качыйм дигәндә дә,
Юллар бары сиңа илтәләр.

Сау бул, димә, сау бул, димә миңа,
Хушлашырга бер дә теләмим!
Онытырга син дә әзер түгел,
Карашыңнан күреп тоям мин.

Саубуллашмыйм әле синең белән,
Югалтырга сине теләмим!
Әйтер сүзләр әле әйтелмәгән,
Син дип, сулыйм, син дип, яшим мин.

Сау бул, димә, сау бул, димә миңа,
Хушлашырга бер дә теләмим!
Онытырга син дә әзер түгел,
Карашыңнан күреп тоям мин.
```
