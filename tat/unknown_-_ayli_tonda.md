# Оригинал

### Фердинанд Гайсин / Наил Табанаков - Айлы төндә

```
Айлы төндә йоклый алмый ятам,
Утырам уйга торып.
Ник карадың күзләремә,
Йөрәгемне яндырып?

Акылымнан шашаммы әллә,
Көннәрне төнгә бутап.
Шушы нурлы айлы төндә
Ятасым килми йоклап.

Җанда хисләр өермәсе,
Ярсып тибә йөрәгем.
Синең белән бергә булу —
Минем бөтен теләгем.

Акылымнан шашаммы әллә,
Көннәрне төнгә бутап.
Шушы нурлы айлы төндә
Ятасым килми йоклап.

Үлсәм дә үкенмәс идем,
Сыенам кочагыңа.
Ай нурларын сибәр идем
Мәхаббәт учагына.

Акылымнан шашаммы әллә,
Көннәрне төнгә бутап.
Шушы нурлы айлы төндә
Ятасым килми йоклап.
```
