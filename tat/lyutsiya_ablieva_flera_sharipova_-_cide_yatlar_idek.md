# Оригинал

### Люция Әблиева / Флера Шәрипова - Җиде ятлар идек

```
Җиде ятлар идек кичә
Җиде юл чатларында.
Очраштык та, гүя очтык
Җиде күк катларына.

Көтеп торган безне сөю
Җиде юл чатларында.
Насыйп булган бит янарга
Мәхәббәт утларында.

Эндәшүләр артык иде
Күзләр генә сөйләште.
Сөю тулы карашларны
Бер-берсенә өләште.

Көтеп торган безне сөю
Җиде юл чатларында.
Насыйп булган бит янарга
Мәхәббәт утларында.
```
