# Оригинал

### Разил Вәлиев / Фасил Әхмәтов - Җитен чәчәк (Җитеннәр)

```
Гел кояштан торган юллар безне
Офыкларга таба илттеләр.
Гел хыялдан,
Гел хыялдан торган сүзләремне
Ишеттеләр,
Ишеттеләр бары җитеннәр.
Ишеттеләр,
Ишеттеләр бары җитеннәр, җитеннәр.

Син югалдың еллар томанында,
Гел сагышлы көннәр җиттеләр.
Хәзер инде,
Хәзер инде тәүге мәхәббәттән —
Күңелемдә,
Күңелемдә бары җитеннәр.
Күңелемдә,
Күңелемдә бары аксыл җитеннәр, җитеннәр.

Эзләп йөрдем җитен чәчәкләрен,
Эзләп йөрдем җирнең читеннән.
Көтмәгәндә,
Көтмәгәндә үзе очрады ул,
Чәчләренә,
Чәчләренә кадәр җитеннәр.
Чәчләренә,
Чәчләренә кадәр аксыл җитеннәр, җитеннәр.
```
