# Оригинал

### Гөлфия Шакирова / Альберт Галимуллин - Туган авылым — Акхуҗа

```
Туган ягы һәр кешегә
Җәннәт булып тоела.
Хәтирәләр булып бар(ы)сы
Күңелеңә уела…

Бигрәк якын күңелемә
Туган авылым — Акхуҗа.
Сагынып ла авылымны
Гомер читләрдә уза…

Туган якка кайтуымны
Сәламли урамнарым.
Бар(ы)сы да бит җаннарыма
Бик якын булганнары.

Бигрәк якын күңелемә
Туган авылым — Акхуҗа.
Сагынып ла авылымны
Гомер читләрдә уза…

Зөя, Имәле елгасы,
Зәңгәр матур күлләрең.
Кайтсам туган якларыма,
Шатлана күңелләрем….

Бигрәк якын күңелемә
Туган авылым — Акхуҗа.
Сагынып ла авылымны
Гомер читләрдә уза…
```
