# Оригинал

### Рәүфә Нурмөхәммәтова - Апама (беренче вариант)

```
Сөйләргә сүзләр күбәйде,
Апама барам әле.
Бу дөньяның мәшәкатеннән
Башымнан ашкан әле.

Йөгереп каршы аласың,
Сорашып хәлләремне,
Әйтеп бетерерлек түгел,
Апа, ярдәмнәреңне.

Каен җиләген киптерсәң,
Чәйгә тәмле кышлары.
Апамның куллары алтын,
Бигрәк тәмле ашлары.

Йөгереп каршы аласың,
Сорашып хәлләремне,
Әйтеп бетерерлек түгел,
Апа, ярдәмнәреңне.

Саулык бәхетләр телимен,
Картайма, апа, бер дә.
Яшьлек сине һич җибәрми,
Матур бул шулай гел дә!

Йөгереп каршы аласың,
Сорашып хәлләремне,
Әйтеп бетерерлек түгел,
Апа, ярдәмнәреңне.
```
