# Оригинал

### Данис Хәбибуллин / Фирзәр Мортазин - Синдә генә

```
Сыен әле кайнар күкрәгемә,
Тыңла әле йөрәк тибешен.
Син генә бит минем бу дөньяда
Өлешемә тигән көмешем.

Исемеңне ятлап йөрәк тибә,
Ярсулары аның юкка димә.
Иреннәрең тәме,
Тормышымның яме
Синдә генә бары, синдә генә.

Бары сине уйлап һәм юксынып,
Үтә минем көнем-төннөрем,
Син генә бит минем бу дөньяда
Ямь бирүче назлы гөлләрем.

Исемеңне ятлап йөрәк тибә,
Ярсулары аның юкка димә.
Иреннәрең тәме,
Тормышымның яме
Синдә генә бары, синдә генә.

Шатлыкларың ташсын ярларыннан,
Тик кайгыдан күңелең тулмасын.
Син генә бит минем бу дөньяда
Ышанычлы тормыш юлдашым.

Исемеңне ятлап йөрәк тибә,
Ярсулары аның юкка димә.
Иреннәрең тәме,
Тормышымның яме
Синдә генә бары, синдә генә.
```
