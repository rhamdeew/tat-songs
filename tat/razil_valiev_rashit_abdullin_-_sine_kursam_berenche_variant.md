# Оригинал

### Разил Вәлиев / Рәшит Абдуллин - Сине күрсәм (беренче вариант)

```
Сине күрсәм, сине күрсәм,
Сагышларым уяна.
Сине күрсәм, сине күрсәм,
Сагышларым уяна да,
Йөрәгем утта яна.

Сине күрсәм, сине күрсәм,
Күңелне моңнар сара.
Сине күрсәм, сине күрсәм,
Моңнар сара, әрни башлый
Йөрәгемдәге яра.

Сине күрсәм, сине күрсәм,
Җир әйләнә кирегә.
Сине күрсәм, сине күрсәм,
Җирләр әйләнә кирегә,
Син киләсең бирегә.
```
