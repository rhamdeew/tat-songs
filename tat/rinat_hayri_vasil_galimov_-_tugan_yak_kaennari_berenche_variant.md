# Оригинал

### Ринат Хәйри / Вәсил Галимов - Туган як каеннары (беренче вариант)

```
Ерак якларда адашсам,
Булса юл аермалы,
Йөгереп чыгып каршы ала
Туган як каеннары.

Туган як, туган як,
Туган як каеннары,
Йөгереп чыгып каршы ала
Туган як каеннары.

Җил — давыллар хыялымның
Канатын каермады.
Керфек какмый сакладылар
Туган як каеннары.

Туган як, туган як,
Туган як каеннары,
Йөгереп чыгып каршы ала
Туган як каеннары.

Юллар мине туган яктан
Озакка аермады.
Бигерәк тиз сагынып кайттым
Туган як каеннарын.

Туган як, туган як,
Туган як каеннары,
Йөгереп чыгып каршы ала
Туган як каеннары.
```
