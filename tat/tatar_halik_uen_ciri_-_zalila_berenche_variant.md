# Оригинал

### Татар халык уен җыры - Зәлилә (беренче вариант)

```
Зәңгәр чиләк матур була
Су белән тулса гына.
Ярны сөюе күңелле
Сүзендә торса гына.

Әлләләлем Зәлилә,
Гөлләләлем Зәлилә;
Зәңгәр чиләкләрен алып
Суга бара Җәмилә.
Чиләге тула микән,
Суы түгелә микән;
Синең өчен бер матурның
Үзәге өзелә микән.

Биек тауга менгән чакта
Таянам тезләремә.
Суга тилмергән кошлар күк
Тилмерәм сүзләреңә.

Әлләләлем Зәлилә,
Гөлләләлем Зәлилә;
Зәлилә күк бәхетлеләр
Совет илендә генә.
Әлләләлем Зәлилә,
Гөлләләлем Зәлилә;
Яратамын дусларыңны,
Шулай ук үзеңне дә.
```
