# Оригинал

### Наҗар Нәҗми / Рим Хәсәнов - Телефоннар шалтырыйлар

```
Телефоннар шалтырыйлар,
Сине көтәләр,
Синең бөтен уйларыңны
Миңа илтәләр.

Әйтче, иркәм, әйтче миңа,
Сөясеңме син.
Синең зәңгәр күзләреңә
Гашыйк булдым мин.

Телефоннар шалтырыйлар,
Ә син һаман юк.
Көннәр үтә, айлар үтә,
Күңелдә мин юк.

Әйтче, иркәм, әйтче миңа,
Сөясеңме син.
Синең зәңгәр күзләреңә
Гашыйк булдым мин.

Телефоннар, телефоннар
Һаман шалтырый.
Йөрәк яна, син һаман юк,
Күңел һич тынмый.

Әйтче, иркәм, әйтче миңа,
Сөясеңме син.
Синең зәңгәр күзләреңә
Гашыйк булдым мин.
```
