# Оригинал

### Тимер Йосыпов / Рамил Медъяр-Камал - Биеп кал (икенче вариант)

```
Аланнарга чыгып кына
Бер сызгыраек әле.
Тын кичләрдә яшьлек җырын
Бер сыздыраек әле.

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.

Бозмаек без араларны,
Форсат юк телләшергә.
Берәү сиңа туйга килә,
Берәү бәхилләшергә…

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.

Сылап-сыйпап бетерерсең
Битеңдәге сырыңны.
Туй кызганда, сузма әле
Бәхилләшеү җырыңны.

Эй туйга бер ямь өстәп
Җир селкетеп биеп кал.
Яшьлек үтсә, куа төш тә
Бер кирәген биреп кал.
```
