# Оригинал

### Гөлфия Исхакова / Руслан Абдулов - Җыр булып үзем барам

```
Сагышларым җырга салып
Аңлатыйм дип уйладым.
Тик шулай да бергә үткән
Юлларымны урадым.

Уралдым мин, уралдым мин,
Уралдым уй-хисләргә.
Адаштым сөю утында,
Белмимен ни эшләргә.

Шул яр чишмәсенә киләм
Бормалы юллар буйлап.
Ак каенга сарылам да
Моңланам сине уйлап.

Уралдым мин, уралдым мин,
Уралдым уй-хисләргә.
Адаштым сөю утында,
Белмимен ни эшләргә.

Күз карашың адым саен
Чагылып тора сыман.
Сер бирмәскә тырышам да
Өзелеп сөям һаман.

Уралдым мин, уралдым мин,
Уралдым уй-хисләргә.
Адаштым сөю утында,
Белмимен ни эшләргә.

Яна, әрни, тик төзәлми,
Кан тулы йөрәк ярам.
Мәхаббәтем канатымда,
Җыр булып сиңа барам.

Уралдым мин, уралдым мин,
Уралдым уй-хисләргә.
Адаштым сөю утында,
Белмимен ни эшләргә.
```
