# Оригинал

### Фәннур Сафин / Риф Гатауллин - Балтачым (икенче вариант)

```
Чыксам иркен болыннарга,
Мин сокланып туялмыйм.
Якташларым, сезнең җаннан
Матурлыклар җыям мин.

Балтачым минем, Балтачым,
Күңелем кырларында.
Хыялларым — тауларыңда,
Син минем җырларымда.

Кайчан гына килсәм дә мин
Безнең Шушма буена,
Әйтерсең лә гөлләр җыеп
Барам сабан туена.

Балтачым минем, Балтачым,
Күңелем кырларында.
Хыялларым — тауларыңда,
Син минем җырларымда.

Дәртле җырым канат булсын
Борынгы каеннарга.
Сезнең кебек дуслар барда
Урын юк кайгыларга.

Балтачым минем, Балтачым,
Күңелем кырларында.
Хыялларым — тауларыңда,
Син минем җырларымда.

Читтә дә мин, туган җирем,
Сезнең яктан күз алмыйм.
Шуңа күрә сиңа кайткач,
Җырламыйча түзалмыйм.

Балтачым минем, Балтачым,
Күңелем кырларында.
Хыялларым — тауларыңда,
Син минем җырларымда.
```
