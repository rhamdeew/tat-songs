# Оригинал

### Зөлфия Нигманова / Виталий Агапов - Ялгыз ана (беренче вариант)

```
Ялгыз ана. Картлар йортында…
Өмет чаткылары сүнмәгән.
Җан учагын кабызыр утның
Ерак киткәнлеген сизмәгән.

Әйтә алмады соңгы сүзләрен,
Бик еракта улы, кызлары.
Ялгыз ана картлар йортында…
Язмыш мәрхәмәтен кызганды.

Балаларга биреп барлыгын,
Кеше булсыннар, дип тырышкан.
Бар байлыгы — көмеш чәчләре…
Бәхет тойды микән тормыштан?

Әйтә алмады соңгы сүзләрен,
Бик еракта улы, кызлары.
Ялгыз ана картлар йортында…
Язмыш мәрхәмәтен кызганды.

Яклаучым булыр, дип үстерде,
Тормыш дәрьяларын кичкәндә,
Газиз улы салды башкаен
Хәтәр уен — ерак Чәчәндә.

Әйтә алмады соңгы сүзләрен,
Бик еракта улы, кызлары.
Ялгыз ана картлар йортында…
Язмыш мәрхәмәтен кызганды.

Ике кызын — ике йолдызын
Очырды шул ерак якларга.
Изге хыял утын, өметен
Насыйп иткән Ходай ятларга.

Әйтә алмады соңгы сүзләрен,
Бик еракта улы, кызлары.
Ялгыз ана картлар йортында…
Язмыш мәрхәмәтен кызганды.
```
