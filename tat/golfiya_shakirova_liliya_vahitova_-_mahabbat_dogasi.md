# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Мәхәббәт догасы

```
Язмышка языла
Йөрәккә кагыла
Илаһи моң булып
артыңнан тагыла.
Ходайның биргәне
Ал бәндәм дигәне
Бер килгән гомердән
Өлешкә тигәне

кушымта
Мәхәббәт догасы
Язылган күкләргә
Кояш нуры булып
иңәдер җирләргә
күчәдер күзләргә,
күчәдер хисләргә
Чагыла йөзләрдә
мәхәббәт догасы

Риза мин сөйләргә
җыр итеп көйләргә
тормышны матурлап
керсен ул өйләргә
сөйләргә кирәкми
белә аны йөрәк
мин яратам диеп
әйтик бары күбрәк

кушымта
Мәхәббәт догасы
Язылган күкләргә
Кояш нуры булып иңәдер җирләргә
күчәдер күзләргә, күчәдер хисләргә
Чагыла йөзләрдә мәхәббәт догасы
```
