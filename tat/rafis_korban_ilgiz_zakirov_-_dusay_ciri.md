# Оригинал

### Рафис Корбан / Илгиз Закиров - Дусай җыры

```
Кәккүк моңы ишетелә
Дусай урманнарында.
Ул кайтаваз булып кайта
Дусай урамнарына.

Кәккүк моңы, кошлар җыры —
Синең җырың, Дусаем.
Дусаем дип бер җырладым,
Мин синең бер кошкаең.

Сандугач җырлап уята
Дусайда җәйге таңда.
Сандугач тынса да бер көн,
Моңнары кала җанда.

Былбыл моңы, кошлар җыры —
Синең җырың, Дусаем.
Дусаем дип бер җырладым,
Мин синең бер кошкаең.

Дусай җырын дуслар җырлый,
Син дә кушыл, дускаем.
Тәрәзләрен ачып тыңлый
Дуслар җырын Дусаем.

Кәккүк моңы, кошлар җыры —
Синең җырың, Дусаем.
Дусаем дип бер җырладым,
Мин синең бер кошкаең.
```
