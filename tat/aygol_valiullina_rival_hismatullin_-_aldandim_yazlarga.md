# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Алдандым язларга

```
Язларда табылдың,
Иңемә сарылдың,
Яктыртып язмышым иртәсен.
"Үтәр" дип уйладым,
Ялгышты уйларым,
Гомергә син гашыйк иткәнсең.

Ышандым язларга,
Алдандым назларга,
Күңелем төсләре алышты!
Ышандым назларга,
Алдандым язларга,
Йөрәгем аң белән ярышты.

Кояшка тиңләдем,
Башканы күрмәдем,
Йөрәгем зарыгып наз көтте.
Сөюнең нурлысы,
Сөюнең җылысы
Эремәс җанымны эретте.

Ышандым язларга,
Алдандым назларга,
Күңелем төсләре алышты!
Ышандым назларга,
Алдандым язларга,
Йөрәгем аң белән ярышты.
```
