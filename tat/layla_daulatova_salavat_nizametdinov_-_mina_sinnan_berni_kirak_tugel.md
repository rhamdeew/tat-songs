# Оригинал

### Ләйлә Дәүләтова / Салават Низаметдинов - Миңа синнән берни кирәк түгел

```
Миңа синнән берни кирәк түгел,
Яратмасаң да ярый.
Бары сагышланган йөзең генә
Йөрәгемне яралый.

Бары сагышланган йөзең генә
Йөрәгемне яралый, яралый.

Миңа синнән берни кирәк түгел.
Син бәхетле, шат чакта.
Ләкин күңелеңә авыр булса,
Онытма, онытма минем хакта.

Ләкин күңелеңә авыр булса,
Онытма, онытма минем хакта.

Миңа синнән берни кирәк түгел,
Яшә, бары шул җитә.
Башкага булсаң булырсың.
Иң мөһиме бул гына!

Башкага булсаң булырсың инде.
Иң мөһиме бул гына, бул гына!
```
