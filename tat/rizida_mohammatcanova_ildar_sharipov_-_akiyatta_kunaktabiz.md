# Оригинал

### Ризида Мөхәммәтҗанова / Илдар Шәрипов - Әкияттә кунактабыз

```
Туган чакта кемнәр язган икән
Тормыш юлын бергә үтәргә.
Мәхәббәттән бәхет, шатлык ташып
Изгелекләр язсын көтәргә.

Кушымта:
Кадерлем, дип назла бер генә,
Җаным эрер, ләззәт кичерер.
Гел янда бул, бергә булган көннәр
Матур тойгы гына китерер.

Чын мәхәббәт булса, юлыбызда,
Күңелдә дә гөлләр үсәрләр.
Салкынлык һич булмас арабызда,
Талгын җылы җилләр исәрләр.

Кушымта.

Туган чакта Ходай язган микән,
Тылсымлы дип булсын гомерләр.
Әкияттә кунактабыз бүген —
Якты нурга тулган күңелләр.

Кушымта.
```
