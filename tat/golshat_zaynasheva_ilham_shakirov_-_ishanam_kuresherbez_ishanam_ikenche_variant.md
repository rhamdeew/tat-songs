# Оригинал

### Гөлшат Зәйнәшева / Илһам Шакиров - Ышанам! (Күрешербез, ышанам!..) (икенче вариант)

```
Ник моңаеп башын идең,
Аерлабыз дисеңме?
Бәлки әле аерылмабыз,
Аерылмабыз шикелле.

Соңгы тапкыр күрәм диеп
Башын салма иңемә.
Бәлки әле күрешербез,
Күрешербез иртәгә?..

Без барасы озын юллар
Бер булырга охшаган.
Озакламый очрашырбыз,
Очрашырбыз ышанам!

Ышанам, ышанам,
Очрашырбыз ышанам!
```
