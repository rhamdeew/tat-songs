# Оригинал

### Галиәхмәт Шаһи / Виталий Агапов - Җавап сорама

```
Зинһар өчен, күзләремә карап,
Бәхетлеме син, дип сорама.
Бәхетле, дә диеп әйтә алмам,
Бәхетсез, дә диеп юрама.

Еллар үтә синсез, язмыш булмый көзсез,
Гомер үтсен безнең өчен үкенечсез.

Мин бәхетле идем синең янда,
Башка бәхет өмет итмәдем.
Сине ташлап китү — язмыш эше,
Мин бит бәхет эзләп китмәдем.

Еллар үтә синсез, язмыш булмый көзсез,
Гомер үтсен безнең өчен үкенечсез.

Китми калсам… Нигә кирәк юрау,
Тормышта бит юллар һәрчак урау.
Китми калсам, безнең алдыбызга
Язмыш куяр иде башка сынау.

Еллар үтә синсез, язмыш булмый көзсез,
Гомер үтсен безнең өчен үкенечсез.

Гомер үкенечсез үтсен өчен,
Язмышларны узу кирәктер.
Дөнья тулган үкенүче белән,
Узучылар, ахры, сирәктер.

Еллар үтә синсез, язмыш булмый көзсез,
Гомер үтсен безнең өчен үкенечсез.
```
