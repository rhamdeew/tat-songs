# Оригинал

### Илзирә Вәлиуллина / Марат Галимов - Тукран

```
Тук, тук,
Тук, тук, тук итә
Иртә таңнан тукран.
Тукранга иптәшкә дип
Бер кош килеп утырган.
Тук, тук, тук итә
Көне буе тукран.
Тук, тук иткәнне тыңларга
Кил әле дип чакырам!

Тук, тук, тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.
Тук, тук, тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.

Тук, тук,
Тук, тук тукран
Бигрәк матур якыннан!
Гашыйк булгач, ни хәл итим
Калмый йөрим артыңнан.
Чык, чык, чакырам,
Үпкәләрмен, чыкмасаң.
Күзгә-күз карашып кына
Сөйләшик бер ичмасам!

Тук, тук тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.
Тук, тук, тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.

Тук, тук, тукран
Бик биеккә утырган,
Агачтагы бөҗәкләрне ашап корсак тутырган.
Тук, тук, тукран,
Сиңа карап утырам,
Мин сиңа гашыйк булганмын,
Син тукран һәм мин тукран.

Тук, тук, тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.
Тук, тук, тукран
Бик биеккә утырган.
Тукран тавышын тыңлап
Турыгызда утырам.
Тук, тук, тук, тук, тукран,
Син тукран һәм мин тукран…
```
