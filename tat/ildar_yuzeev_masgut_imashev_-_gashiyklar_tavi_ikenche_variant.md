# Оригинал

### Илдар Юзеев / Мәсгуть Имашев - Гашыйклар тавы (икенче вариант)

```
Ниндидер моңнар бар,
Ниндидер шомнар бар
Каеннар, имәннәр шавында.
Онытыйк шомнарны,
Каршылыйк таңнарны
Бергәләп Гашыйклар тавында.

Без тауның башында,
Мәңгелек каршында —
Гашыйклар очрашкан урында.
Җанда ай нурлары,
Хисләр саф, югары —
Биектә, Киек Каз Юлында.

Син әле янымда,
Хушлашу таңында
Сүз бирик изге җир алдында:
Айрылган көннәрдә
Булыйк гел үрләрдә,
Булыйк гел Гашыйклар тавында.
```
