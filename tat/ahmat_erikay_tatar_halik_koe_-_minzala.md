# Оригинал

### Әхмәт Ерикәй / Татар халык көе - Минзәлә

```
Барасызмы Минзәләгә,
Мин барам Минзәләгә;
Минзәләне күрер өчен
Йөрәгем өзгәләнә.

Минзәләгә барганым юк,
Юллары такыр, диләр;
Такыр юлда йөреп үскән
Кызлары матур, диләр.

Минем иркәм Минзәләдә
Туып үскән бер матур;
Минзәләгә баралмасам.
Аны кемнәр юатыр?!

Минем иркәм Миизәләнең
Урамнарын балкыта.
Үзенең уңганлыгы белән
Тирә-якка дан тота.
```
