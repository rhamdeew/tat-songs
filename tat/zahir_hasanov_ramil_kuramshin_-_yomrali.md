# Оригинал

### Заһир Хәсәнов / Рамил Курамшин - Йомралы

```
Топольләр, башларын иеп,
Тирбәләләр җилләрдә.
Яшьлек гөлем чәчәк атты
Шушы гүзәл илләрдә.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.

Аккошлар килеп төшәләр
Акчарлак күлләренә.
Туган ягын сагынып кайта
Синең гүзәлләрең дә.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.

Шатлыкка, нурга төренеп
Синдә таңнар уяна.
Таңнар шулай мәңге балкыр,
Тыныч булсын бу дөнья.

Йомралы, Йомралы,
Урамнары бормалы.
Бормалы урамнарында
Йөреп күңелем туймады.
```
