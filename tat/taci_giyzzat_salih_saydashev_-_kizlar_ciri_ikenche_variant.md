# Оригинал

### Таҗи Гыйззәт / Салих Сәйдәшев - Кызлар җыры (икенче вариант)

```
Җырлый-җырлый, гөрли-гөрли
Өмәләрдә күңел ачабыз.

Ай-һай, дусларым,
Дусларым, дус-ишләрем.
Аңламыйсыз йөрәгемнең
Нигә ярсып типкәнен.
Аңламыйсыз күңелем тәрәзәсен
Кемнең килеп чирткәнен.

Ак тулалар тиз керешә
Аркау буен тигез эрләгәч.
```
