# Оригинал

### Клара Булатова / Мәсгуть Имашев - Сәлим җыры

```
Таң җырым, уят син,
Иркәләп юат син
Мин эзләп тилмергән
Сагышлы ярымны.
Тирбәлеп таңнарда,
Җиткер син аңарга
Күңелдә төйнәлгән
Бөтен аһ-зарымны.
Таң җырым, уят син,
Иркәләп юат син
Мин эзләп тилмергән
Сагышлы ярымны.

Белсен ул мин барын,
Эзләсен үз ярын…
Сакласын, әйт аңа,
Тик күңел сафлыгын!
Ышансын, белсен ул
Мин килер көн барын,
Чыксын ул каршыма,
Чыксын ул атлыгып.
Белсен ул мин барын,
Эзләсен үз ярын…
Сакласын, әйт аңа,
Тик күңел сафлыгын!
```
