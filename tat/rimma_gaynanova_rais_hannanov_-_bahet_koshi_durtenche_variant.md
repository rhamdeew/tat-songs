# Оригинал

### Римма Гайнанова / Рәис Ханнанов - Бәхет кошы (дүртенче вариант)

```
Бәхет кошы кунган иде
Салынган иңнәремә,
Кулларыма алмадым шул.
Кем белгән китәрен лә.

Җылы учларыма алып,
Кочасы булган икән,
Сөеп-назлап күкрәгемә
Кысасы(м) булган икән.

Күңелемне иш итмәде,
Югалды бәхет кошым.
Булды, ахры, өнгә ашмас
Әкияти матур төшем.

Юксынудан, сагынудан
Йөрәгем тынмас инде.
Бәхет кошы иңнәремә
Яңадан кунмас инде.

Бәхет кошы кунган иде
Сагынган иңнәремә,
Кулларыма алмадым шул.
Кем белгән китәрен лә.

Җылы учларыма алып,
Кочасы булган икән,
Сөеп-назлап күкрәгемә
Кысасы(м) булган икән.

Юксынудан, сагынудан
Йөрәгем тынар әле.
Бәхет кошым иңнәремә
Яңадан кунар әле.
```
