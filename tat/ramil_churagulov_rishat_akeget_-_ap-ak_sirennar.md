# Оригинал

### Рамил Чурагулов / Ришат Акъегет - Ап-ак сиреннәр

```
Бер-берсенә хисләр көтеп
Үреләләр иреннәр.
Ә бакчаларда хистән исереп
Үбешәләр иреннәр.

Чыланмыйча калмасыннар,
Чыланмыйча калмасыннар,
Үбү көткән иреннәр.
Ак бәхет, ак сөю тели,
Ак бәхет, ак сөю тели
Ап-аклардан ап-ак сиреннәр.

Гөр-гөр килә кичке уеннар,
Һәр кем эзли үз парын.
Бәхетле егет кызын озата
Кысып нечкә биленнән.

Чыланмыйча калмасыннар,
Чыланмыйча калмасыннар,
Үбү көткән иреннәр.
Ак бәхет, ак сөю тели,
Ак бәхет, ак сөю тели
Ап-аклардан ап-ак сиреннәр.
```
