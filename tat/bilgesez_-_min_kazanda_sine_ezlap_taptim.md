# Оригинал

### Билгесез - Мин Казанда сине эзләп таптым

```
Мин сагынам назлы язларыңны…
Язларда мин хыял корамын.
Язларда мин мәхәббәтем эзләп,
Мең урадым Казан урамын.

Мин Казанда сине эзләп таптым,
Мин Казанда бәхет юрадым.
Бергә узыйк язмыш урамнарын,
Бергә узыйк Казан урамын!

Мин сагынам озын кышларыңны…
Күңелемә күчә бураның.
Бураннарда сине эзли-эзли
Мең урадым Казан урамын.

Мин Казанда сине эзләп таптым,
Мин Казанда бәхет юрадым.
Бергә узыйк язмыш урамнарын,
Бергә узыйк Казан урамын!

Мин яратам җылы җәйләреңне…
Мин җәйләрдән яңгыр сорадым.
Сагышларым яңгыр юмасмы, дип,
Мең урадым Казан урамын.

Мин Казанда сине эзләп таптым,
Мин Казанда бәхет юрадым.
Бергә узыйк язмыш урамнарын,
Бергә узыйк Казан урамын!

Мин яратам моңлы көзләреңне…
Көзләрдән мин бәхет сорадым.
Икебезгә уртак бәхет эзләп,
Мең урадым Казан урамын.

Мин Казанда сине эзләп таптым,
Мин Казанда бәхет юрадым.
Бергә узыйк язмыш урамнарын,
Бергә узыйк Казан урамын!
```
