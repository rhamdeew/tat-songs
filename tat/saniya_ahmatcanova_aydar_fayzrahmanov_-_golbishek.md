# Оригинал

### Сания Әхмәтҗанова / Айдар Фәйзрахманов - Гөлбишек

```
Сайлап алып матур төсләр
Күңелем түрләреннән,
Үрим әле гүзәл бишек
Сөюем гөлләреннән.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.

Бәйләм-бәйләм гөлләр китер,
Үрелсен хисләребез.
Аллы-гөлле хыялларга
Күмелсен кичләребез.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.

Берәүгә дә бирмим сине,
Тапмадым югалтырга.
Ирек куймам икәү үргән
Хисләрне чуалтырга.

Кушымта:
Тирбәлербез икәү бергә
Мәхәббәт бишегендә.
Мине сөю, бары сөю
Яшәтә яшьлегемдә.
```
