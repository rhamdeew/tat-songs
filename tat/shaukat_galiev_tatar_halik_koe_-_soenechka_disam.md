# Оригинал

### Шәүкәт Галиев / Татар халык көе - Сөенечкә дисәм

```
Кар сулары бетте.
Ярсулары үтте,
Ташлар тәгәрәми ташкында:
Минем генә йөрәк,
Сагынудан йөдәп,
Бәйгедәге аттай ашкына

Аерым торып еллар,
Аймыл булып юллар,
Нигә синнән читтә узган соң?
Бар шул инде парың,
Янып сөйгән ярың,
Кулларыңны аңа сузгансын

Арта уйлар һаман.
Тарта сиңа табан.
Кыен мина ялгыз чакларда,
Сөенечкә дисәм,
Көенечкә икән
Күрешүләр сезнен якларда
```
