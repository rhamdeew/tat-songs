# Оригинал

### Сәрия Вилданова / Рәис Нәгыймов - Бәхетсез димәгез толларны

```
Карларда — йолдызлар,
Күңелдә — ак нурлар,
Җаннарда алланып таң ата.
Бәхетсез димәгез, димәгез толларны,
Алар да берәүне ярата.

Карларда — салкынлык,
Йөзләрдә — яктылык,
Төн килә эңгерле кич аша.
Кыланчык димәгез, димәгез толларны,
Шикле уй хисләрне каралта.

Сөюдә — серлелек,
Язмышта — төрлелек,
Адымнар шыгырдап аңлаша.
Хиссез ул димәгез, димәгез толларны,
Аларда сагышлы моң ята.
```
