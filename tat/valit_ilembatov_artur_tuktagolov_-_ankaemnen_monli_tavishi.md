# Оригинал

### Вәлит Илембәтов / Артур Туктаголов - Әнкәемнең моңлы тавышы

```
Сабый чактан күңелләргә кергән
Әнкәемнең моңлы тавышы.
Безгә багышланган бар гомере,
Безгә арналгандыр язмышы.

Моң чишмәсе әнкәем син,
Без шул моңнан гөрләп үскән.
Сабыр холкың, уңганлыгың,
Әнкәй синнән безгә күчкән.

Синең киңәшеңә мөхтаҗбыз без
Гомер үрләренә менсәктә.
Синең өчен без һаман да бала
Инде күпме яшькә җитсәктә.

Моң чишмәсе әнкәем син,
Без шул моңнан гөрләп үскән.
Сабыр холкың, уңганлыгың,
Әнкәй синнән безгә күчкән.
```
