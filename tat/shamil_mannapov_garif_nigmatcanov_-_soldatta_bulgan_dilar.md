# Оригинал

### Шамил Маннапов / Гариф Нигъмәтҗанов - Солдатта булган, диләр…

```
Билен кысып буган, диләр,
Бигрәк уңган, диләр;
Әллә каян күренеп тора —
Солдатта булган, диләр.

Кечеләрне кече итәр,
Олыны зурлар, диләр.
Әллә каян күренеп тора —
Солдатта булган, диләр.

Комбинезон алган, диләр,
Шинелен салган, диләр.
Шинелен салса да, солдат
Гадәте калган, диләр.
```
