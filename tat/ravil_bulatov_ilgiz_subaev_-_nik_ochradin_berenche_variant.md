# Оригинал

### Равил Булатов / Илгиз Субаев - Ник очрадың? (беренче вариант)

```
Ник очрадың кабат юлларымда
Онытылып барган чагында?
Елмаюың нигә бүләк иттең,
Наз өрмәгәч керсез җаныма?

Инде акыл көтә гел башканы,
Тик күңелдә тагын син генә.
Каршы алам, озатам тик шәүләңне,
Йөз елыймын көнгә, мең көләм.

Горурлыгым йөри гел тирәмдә
Акыл белән көн дә бәхәстә.
Офыкларга карыйм кулым куеп,
Тик күренми анда бернәрсә.
```
