# Оригинал

### Татар  халык җыры - Борынгы көй (беренче вариант)

```
Безнең туган илкәй бигрәк ямьле,
Болыннары, кыры хуш исле.
Сулары да тәмле бал шикелле,
Урманнары җиләк-җимешле

Очасың ла, кошкай, кай илләргә,
Очмыйсыңмы безнең җирләргә?
Оча калсаң, кошкай, шул якларга
Сәлам әйт син туган күлләргә.
```
