# Оригинал

### Билгесез - Уйлану (өченче вариант)

```
Гомер агышларын куып,
Сал өстендә йөзәбез.
Төрле яктан дулкын бәрә,
Түзәбез шул, түзәбез.

Яшен яшьнәп утлар сүнде,
Эзлим кояш яктысын.
Агымсулар алып китте
Гомеремнең яртысын.

Салмы сайда утырмасын,
Давылларда тирбәлеп.
Маңгайга сырлар өстәлә
Сагышлардан тилмереп.

Яртысы микән үткәндер,
Гомеркәем бер килеп.
Озак калдымы яшәргә,
Кемнәр әйтер соң белеп?
```
