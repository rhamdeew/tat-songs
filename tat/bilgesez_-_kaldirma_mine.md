# Оригинал

### Билгесез - Калдырма мине

```
Күңелемдә бүген ачы буран уйный,
Сары сагыш баскан күзләрне томалый.
Каргыш булдымы соң? Язмышлар буталды.
Аңлашмыйча гына киттең дә югалдың.

Яндырма мине, көйдермә мине,
Кайгы утларында, тормыш салкынында
Калдырма мине, калдырма мине…

Кешегә бу җирдә мөмкинлек бирелә,
Син дә бир мөмкинлек! Зинһар, каршы килмә!
Барыр юлларымда уйламадым бер дә
Эләгермен диеп сөю ятмәсенә.

Яндырма мине, көйдермә мине,
Кайгы утларында, тормыш салкынында
Калдырма мине, калдырма мине…

Бер ялгышлык өчен үкенәм мәңгегә,
Син калдырма мине, көч бир, тезләндермә.
Аңламыйча гына кылына ялгышлар,
Сөя белгәннәрне сыный шул язмышлар.

Яндырма мине, көйдермә мине,
Кайгы утларында, тормыш салкынында
Калдырма мине, калдырма мине…
```
