# Оригинал

### Флюр Абдракипов / Ризида Мөхәммәтҗанова - Матур яшик

```
Дуслар белән бергә яшик
Бергә яшәү күңелле.
Бу дөньяда безнең гомер
Мизгел генә түгелме?

Чын дуслыкның кадерен белик,
Чит — ятлар дус түгелләр.
Дус — туганны хөрмәт итик,
Нурлы булсын күңелләр.

Җырлап яшик, гөрләп яшик,
Эшләребез уң булсын.
Кунакларга ишек ачык
Өстәлебез мул булсын.

Тормыш матур, дөньялар киң,
Ачык күңелләребез.
Чын барлыкка әйләнсеннәр
Якты өметләребез.
```
