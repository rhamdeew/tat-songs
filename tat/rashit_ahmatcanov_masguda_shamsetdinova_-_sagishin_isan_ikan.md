# Оригинал

### Рәшит Әхмәтҗанов / Мәсгудә Шәмсетдинова - Сагышың исән икән

```
Ай яктысын шәлләр итеп
Ябармын иңнәреңә,
Тагын бер кат сөенерсең
Дөньяга килгәнемә.

Сәфәрдә чакта гөлләрне
Мин эзләп үбеп китәм,
Кайгыларың таралса да,
Сагышың исән икән.

Һавадан да килә тавыш,
Һәм җирдән дә ишетәм,
Ачылыр да күк капусы —
Әнкәй керер ишектән.

Ай яктысын шәлләр итеп
Ябармын иңнәреңә,
Тагын бер кат сөенерсең
Дөньяга килгәнемә.
```
