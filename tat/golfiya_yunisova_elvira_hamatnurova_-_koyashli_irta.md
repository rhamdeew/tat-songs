# Оригинал

### Гөлфия Юнысова / Эльвира Хаматнурова - Кояшлы иртә

```
Кояш уянган да инде,
Назлап, елмаеп карый.
Күл-көзгесенә иелеп,
Алтын чәчләрен тарый.

Яктылык сирпеп бу нурлар,
Тәрәзәлзргә чиртә:
— Торогыз,балалар,сезне,
Көтә кояшлы иртә!

Тарагы аның — таң җиле,
Ә күлмәге-кызылдан.
Балкып торган нур-чәчләре
Җиргә чаклы сузылган.

Яктылык сирпеп бу нурлар,
Тәрәзәлзргә чиртә:
— Торогыз,балалар,сезне,
Көтә кояшлы иртә!
```
