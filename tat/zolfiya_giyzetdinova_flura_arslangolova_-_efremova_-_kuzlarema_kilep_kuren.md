# Оригинал

### Зөлфия Гыйзетдинова / Флүрә Арсланголова - Ефремова - Күзләремә килеп күрен

```
Синсез үткән һәрбер көнем
Өзгеләнгән күңел гөлем.
Соңлый күрмә. гомерләр кыска,
Күзләремә килеп күрен.

Җанымның юк хис бер тере
Саф хисләрем әле тере.
Сүнмәс боры яшәү нурын
Күзләремә килеп күрен.

Сыер сүзең, сыер серең,
Синсез буш бит йөрәк түрем.
Өметләрем өзелгәнче
Күзләремә килеп күрен.
```
