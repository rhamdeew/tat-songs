# Оригинал

### Венер Фәттах / Вәгыйзь Газизов - Тестостерон шаяра

```
Җилкәдә еллар күп, дими,
Күңел һаман дәрт эзли.
Язлар саен сөю көтә,
Өметен бер дә өзми.

Юк, юк, юк, мин түгел, күз дә түгел,
Тестостерон шаяра.
Гомеремнең шәп чаклары
Үтмәсен, ди, заяга.

Күкерт савытлары тулы,
Күкерте дә коп-коры.
Сөю дигән дәртле утка
Карап тора туп-туры.

Юк, юк, юк, мин түгел, күз дә түгел,
Тестостерон шаяра.
Гомеремнең шәп чаклары
Үтмәсен, ди, заяга.

Җиләктән авыз итәсе,
Төшләр ватасы килә.
Язлар саен сөю көтеп
Йөрәк дәртлерәк тибә.

Э-эй,
Юк, юк, юк, мин түгел, күз дә түгел,
Тестостерон шаяра.
Гомеремнең шәп чаклары
Үтмәсен, ди, заяга.
Гомеремнең шәп чаклары
Үтмәсен, ди, заяга…
```
