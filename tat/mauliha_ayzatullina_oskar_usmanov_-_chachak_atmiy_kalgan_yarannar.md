# Оригинал

### Мәүлиха Айзатуллина / Оскар Усманов - Чәчәк атмый калган яраннар

```
Кеше җиргә бәхет өчен туа,
Бөтен көчен куеп тырыша.
Тик язмышлар шатлык-бәхетләрне
Сабырларга күбрәк өләшә.

Сабыр канатларын сындырмады
Тәүге сөю салган яралар.
Чәчәк атмый калды бөреләнгән
Яшьлектәге кызыл яраннар.

Сабырлыкның ни икәнен белдек,
Балачактан зарлы булмадык.
Егылсак та торып чаба идек,
Күз яшьләрен сөртеп тормадык.

Сабыр канатларын сындырмады
Тәүге сөю салган яралар.
Чәчәк атмый калды бөреләнгән
Яшьлектәге кызыл яраннар.

Безне гашыйк иткән егетләрнең
Мәхаббәтен аңлый алмадык.
Вакыт юар, диеп, яраларны,
Сабыр гына китеп таралдык.

Сабыр канатларын сындырмады
Тәүге сөю салган яралар.
Чәчәк атмый калды бөреләнгән
Яшьлектәге кызыл яраннар.

Сабыр төбе сары алтын икән,
Түзем белән күргән яралар.
Тик һаман да күз алдыннан китми
Чәчәк атмый калган яраннар.
```
