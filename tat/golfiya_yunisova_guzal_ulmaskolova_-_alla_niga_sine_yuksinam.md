# Оригинал

### Гөлфия Юнысова / Гүзәл Үлмәсколова - Әллә нигә сине юксынам?

```
Нигә өметләнеп көтә микән
Чыгарсың дип үзең каршыма.
Ник эзли соң сине күңелкәем,
Борчу уйлар салып башыма.

Әллә нигә сине юксынам,
Әллә нигә сиңа тартылам,
Күрергә кош кебек талпынам,
Очрасаң — кояштай яктырам.

Табыштырды гомер юлларыбыз,
Мин ышанам бары (һәр чак) язмышка.
Бүләк итәм ярсу йөрәгемне,
Күрмәмешкә генә салышма.

Әллә нигә сине юксынам,
Әллә нигә сиңа тартылам,
Күрергә кош кебек талпынам,
Очрасаң — кояштай яктырам.
```
