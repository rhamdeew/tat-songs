# Оригинал

### Раил Гатауллин / Лилия Тарханова - Татар укытучысы авазы

```
Мәктәптә ялгызым,
Мәктәптә берүзем,
Эшлим мин көнозын
Үтәлми тик сүзем.
Эшләгән эшләрем
Таптала, югала.
Мин биргән белемнәр
Кайда соң сез, кайда?
Мин сезне татарча
Уйларга өйрәтәм,
Укытам, яздырам,
Бай мирас күрсәтәм.
Әмма мин мәктәптә
Берүзем, ялгызым.
Бар да рус, мин генә —
Бер татар явызы.

Татар телен укытам
Башкалам Казанда.
Өйрәтәм татарча
Укырга, язарга
Әтиләр-әниләр,
Ник каршы сез миңа?
Тәрбия биргәнгә
Руслашкан кызыңа,
Тәрбия биргәнгә
Руслашкан улыңа?

Татарда бар иман,
Ихтирам, изгелек.
Миллилек сакланган
Яшәп тә изелеп.
Татарда бар Тукай,
Кол Гали һәм Җәлил.
Аларның сүзләре
Бик туры һәм гадел.
Бар бик күп галиме,
Шагыйре, җырчысы,
Актеры, табибы,
Эшкуар, спортчысы.
Татар булып туган
Сезнең дә балалар,
Алар да тарихта
Үз ур(ы)нын табарлар.

Татар телен укытам
Башкалам Казанда.
Өйрәтәм татарча
Укырга, язарга.
Тормыйкчы без каршы
Барасы юлларда:
Балагыз өйрәнсен
Татарча җырларга,
Татарча сөйләргә,
Татарча уйларга!
```
