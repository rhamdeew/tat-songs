# Оригинал

### Нәби Дәүли / Сара Садыйкова - Сине озатканда (Ак пароход китеп бара)

```
Ак пароход китеп бара,
Китеп бара еракка.
Әллә билгесез тарафка,
Әллә син торган якка.

Акчарлаклар канат болгый,
«Хуш» диләр микән алар.
Минем генә сагышымны
Белмиләр икән алар.

Син дә шулай киткән идең,
Озатып калган идем.
Сине сагынып үтә еллар,
Тын гына ага Идел.

Акчарлаклар канат болгый,
«Хуш» диләр микән алар.
Минем генә сагышымны
Белмиләр икән алар.

Офыкларда зәңгәр томан:
Шатлыгыммы, кайгыммы?
Ак пароход китеп бара,
Сине алып кайтырмы?..

Акчарлаклар канат болгый,
«Хуш» диләр микән алар.
Минем генә сагышымны
Белмиләр икән алар.
```
