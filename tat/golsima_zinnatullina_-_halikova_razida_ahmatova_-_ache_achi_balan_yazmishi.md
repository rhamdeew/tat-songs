# Оригинал

### Гөлсимә Зиннәтуллина - Халикова / Рәзидә Әхмәтова - Әче (ачы) балан язмышы

```
Бурап-бурап карлар ява,
Ап-ак булды аланнар.
Тик кып-кызыл яулыкларын
Салмадылар баланнар.

Кышкы җилләр исәр инде,
Баланны өзәм инде.
Баландай әче (ачы) язмышка
Нишлим соң, түзәм инде…

Күзем төшеп бер үрелгәч,
Калалмадым тыелып.
Баланнар әче (ачы) икән дип
Яшәр өчен сыена.

Кышкы җилләр исәр инде,
Баланны өзәм инде.
Баландай әче (ачы) язмышка
Нишлим соң, түзәм инде…

Кызыл җиләк, дип ымсынып
Үрелдем баланнарга.
Әче (ачы) балан язмышлары
Язмасын калганнарга.

Кышкы җилләр исәр инде,
Баланны өзәм инде.
Баландай әче (ачы) язмышка
Нишлим соң, түзәм инде…
```
