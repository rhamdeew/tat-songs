# Оригинал

### Динә Талхина / Салават Низаметдинов - Төшләреңә синең керәм

```
Төшләреңә синең керәм,
Тетрәндерәм күңелне.
Әнә, керфек очына
Тагы да чык эленде,
Төшләреңә синең керәм.

Төшләреңә синең керәм,
Үкендерәм түгелме?
Моңсуланды күзләрем (күзкәйләрең),
Сагышларга күмелде,
Төшләреңә синең керәм.

Төшләреңә синең керәм,
Саташтырып таңнарда.
Башкаларга җан (күз) атма
Бу дөньяда мин барда!
Төшләреңә синең керәм.
```
