# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Әнкәемнең фатыйхасы

```
Ялтыр боз сымак бу дөнья,
Саклар таяуы кирəк.
Егылып китеп, имгəндə,
Əнкəй аяуы кирəк.

Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.
Үз чамамны белми, тормышымнан
Артык күпне өмет иткәнгә.
Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.

Дер селкенеп тора дөнья,
Ныклы терәге кирәк.
Чайкалганда, тотынырга
Әнкәй беләге кирәк.

Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.
Үз чамамны белми, тормышымнан
Артык күпне өмет иткәнгә.
Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.

Дөнья буйлап атлаганда
Əнкəй телəге кирəк.
Кыен чакта сыенырга
Әнкәй йөрәге кирәк.

Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.
Үз чамамны белми, тормышымнан
Артык күпне өмет иткәнгә.
Юлым уңмас инде, әнкәемнең
Фатыйхасын алмый киткәнгә.
```
