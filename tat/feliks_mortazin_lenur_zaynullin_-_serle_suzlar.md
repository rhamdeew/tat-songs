# Оригинал

### Феликс Мортазин / Ленур Зәйнуллин - Серле сүзләр

```
Сөю сукмагыннан
Икәү атлап киткән
Алар гади, серле,
Сүзләр белә икән:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
«Сөям сине, бәгърем»
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Нинди көчле сүзләр!
«Сөям сине, бәгърем»
Нинди хисле сүзләр!

Галәм киңлегеннән
Сөю нуры иңгән
Серле сүзләр белән
Сөю үзе килгән:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
Сөям сине, бәгърем”
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Мәңге яшәр сүзләр!
«Сөям сине, бәгърем»
Янып яшнәр сүзләр,

Серле сүзләр белсәң
Сөю мәңге яшәр
Гади, серле сүзләр
Назлап сиңа дәшәр:

«Сөям сине, бәгърем»
Нинди гади сүзләр,
«Сөям сине, бәгърем»
Нинди серле сүзләр!
«Сөям сине, бәгърем»
Назлы күркәм сүзләр,
«Сөям сине, бәгърем»
Утка керткән сүзләр,

«Сөям сине, бәгърем»
Бигрәк алтын сүзләр,
«Сөям сине, бәгърем»
Җанга якын сүзләр!
```
