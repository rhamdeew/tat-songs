# Оригинал

### Тәбрис Фәхретдинов / Марат Гобәйдуллин - Каракүл

```
Каракүлгә мин гел кайтам
Ерак булса да ара.
Мондый матур, гүзәл җиргә
Кайтмыйча түзеп кара.

Кушымта:
Кайткан чакта Каракүлгә
Сәлам бирә танышлар.
Җырым аша һәммәгезгә
Телим, дуслар, уңышлар.

Кайтсам, төшәм күл буена,
Куркытмый җил дә давыл.
Каракүл син минем өчен
Кадерле туган авыл.

Яр буйлары куе камыш,
Каракүл бик тирән ул.
Өзелеп, сагынып кайтыр өчен
Әти-әни кирәк ул.
```
