# Оригинал

### Нурия Абдуллина - Гомер үтә (икенче вариант)

```
Мин бу җирдә җыр-моң булып
Калыр идем,
Йөрәгеңнең мең кылларын
Чиртәр идем.

Салкын җилләр исү белән
Көзләр җитә,
Бәгырьләрне өтә-өтә
Гомер үтә.

Язлар саен умырзая
Булыр идем,
Күзләреңнең явын (нурын) алып
Балкыр идем.

Салкын җилләр исү белән
Көзләр җитә,
Бәгырьләрне өтә-өтә
Гомер үтә.

Синең белән бер сукмактан
Узар идем,
Чәчләреңә гөл-чәчәкләр
Үрер идем.

Салкын җилләр исү белән
Көзләр җитә,
Бәгырьләрне өтә-өтә
Гомер үтә.
```
