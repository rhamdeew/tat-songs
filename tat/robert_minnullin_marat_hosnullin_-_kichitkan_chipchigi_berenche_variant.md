# Оригинал

### Роберт Миңнуллин / Марат Хөснуллин - Кычыткан чыпчыгы (беренче вариант)

```
Әй, кычыткан чыпчыгы,
Бигрәк матур кыз идең.
Ут идең, йолдыз идең,
Бигрәк тынгысыз идең.

Үсмер чакта булган икән
Яратуның чып-чыны.
Син әле дә шундый микән,
Әй, кычыткан чыпчыгы?

Әй, кычыткан чыпчыгы,
Бер дә тоттырмый идең.
Тоттырсаң да кочакта
Бер дә күп тормый идең.

Үсмер чакта булган икән
Яратуның чып-чыны.
Син әле дә шундый микән,
Әй, кычыткан чыпчыгы?

Әй, кычыткан чыпчыгы,
Шук идең, чая идең.
Шаярып чаба идең
Нәкъ бала-чага идең.

Үсмер чакта булган икән
Яратуның чып-чыны.
Син әле дә шундый микән,
Әй, кычыткан чыпчыгы?

Әй, кычыткан чыпчыгы,
Көләч идең, шат идең.
Күңелгә ут салучы
Тиктормас бер зат идең.

Үсмер чакта булган икән
Яратуның чып-чыны.
Син әле дә шундый микән,
Әй, кычыткан чыпчыгы?
```
