# Оригинал

### Хәния Фәрхи / Наил Шәймарданов - Рәхмәтлемен гомеремә

```
Беләм гомер йомгагымның
Яртысы сүтелгәне,
Җырларымның җырланганы,
Юлларның үтелгәне.

Рәхмәтлемен гомеремә,
Аның һәр бер көненә.
Һәрбер гомер берәү генә,
Кадерле дә, серле дә.

Ятлар җырын җырламадым,
Җырладым үз җырымны.
Белеп яшим бу дөньяда,
Мин — үземнең урыны.

Рәхмәтлемен гомеремә,
Аның һәр бер көненә.
Һәрбер гомер берәү генә,
Кадерле дә, серле дә.

Мин салган юлдан үтәләр,
Җырлар үтәрләр әле.
Егылганымны көткәннәр,
Озак көтәрләр әле.

Рәхмәтлемен гомеремә,
Аның һәр бер көненә.
Һәрбер гомер берәү генә,
Кадерле дә, серле дә.
```
