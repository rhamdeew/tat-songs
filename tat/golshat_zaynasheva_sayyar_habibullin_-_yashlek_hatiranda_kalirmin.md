# Оригинал

### Гөлшат Зәйнәшева / Сәйяр Хәбибуллин - Яшьлек хатирәңдә калырмын

```
Йөрсәң Идел буйлап,
Бергә чакны уйлап,
Сагынып үткәннәрнең барсын да.
Мин, әверелеп ярсу дулкынга,
Килеп баш иярмен каршыңда.

Чыксаң язгы таңда
Кырга, болыннарга
Һәм уйласаң минем турында,
Мин, әверелеп тургай моңына,
Түгелермен зәңгәр болынга.

Кояш баеганда,
Йолдыз кабынганда,
Син мине оныткан минутта,
Мин, әверелеп шәфәкъ нурына,
Кереп югалырмын болытка.

Оныттым диярсең,
Югалттым диярсең,
Синнән шундый хәбәр алырмын.
Мин, әверелеп яшерен сагышка,
Яшьлек хатирәңдә калырмын.
```
