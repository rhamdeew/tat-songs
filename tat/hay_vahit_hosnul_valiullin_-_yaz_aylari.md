# Оригинал

### Хәй Вахит / Хөснул Вәлиуллин - Яз айлары

```
Яз айлары, яз айлары
Яшелләтә кырларны;
Яз айлары хәтерләтә
Онытылган җырларны.

Яз сулары, яз сулары
Актара яр, комнарны;
Яз айлары кабындыра
Сүрелгән ялкыннарны.

Яз нурында, яз нурында
Чәчәк ата үрмә-гөл.
Яз-айлары сагынуымны.
Тирәнәйтә бермә-бер.

Яз аенда кошлар оча
Тирәктән тирәкләргә,
Хезмәт җыры тулып ташый
Йөрәктән йөрәкләргә.
```
