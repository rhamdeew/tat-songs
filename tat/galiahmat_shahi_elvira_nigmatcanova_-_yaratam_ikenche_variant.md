# Оригинал

### Галиәхмәт Шаһи / Эльвира Нигъмәтҗанова - Яратам (икенче вариант)

```
Мин фәрештә, Ходай мине җирдә
Сине саклар өчен яраткан.
Минем күзләр синең күзләреңдә,
Саф куңелең өчен яратам.

Мин сандугач, Ходай мине сиңа
Сайрап торсын өчен яраткан.
Иртәсен дә сайрыйм, кичләрен дә,
Мин яратам, сине яратам.

Мин бер тамчы, Ходай мине синең
Алдыңа тамыргы яраткан.
Минем тамганымны күрмәссең дә,
Ишетерсең бары яратам.

Мин бер чәчәк, Ходай мине сиңа
Өзеп алыр өчен яраткан.
Сулганда да кабатлармын гына
Яратам дип, сине яратам.
```
