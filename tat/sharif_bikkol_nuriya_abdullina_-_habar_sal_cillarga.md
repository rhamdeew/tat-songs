# Оригинал

### Шәриф Биккол / Нурия Абдуллина - Хәбәр сал җилләргә

```
Сагынсаң, саргайсаң, / Кагылсаң, сугылсаң, /
Хәбәр сал җилләргә.
Мин килеп җитәрмен
Син көткән җирләргә.

Сагынсаң, буталсаң, / Сагынсаң, саргайсаң, /
Хәбәр сал җилләргә.
Ерак дип тормамын,
Китәрмен чүлләргә.

Көенсәң, сөенсәң,
Хәбәр сал җилләргә.
Мин алып кайтырмын
Зар иткән илләргә.

Чит-яттан сөелсәң,
Зинһар, дим, белгертмә.
Мин калган тарафка
Җилләрне йөгертмә.
```
