# Оригинал

### Себер татарлары халык җыры - Йөрәккәйем сағышларға түсмәйте (Иртешкәйем)

```
Иртешкәйем тирән сыулары сыуыҡ
Ҡас-үртәкләр оторо йөсмәйте,
Сайрайты палбыл, ҡыцҡыраты кәүек
Йөрәккәйем сағышларға түсмәйте.

Йетәтеген йергә йеталматым —
Нейләр пулған песнең тур атҡа,
Сабырлыҡлар пелән йәшәгән пулсам
Ирешер лә итем лә моратҡа.
```
