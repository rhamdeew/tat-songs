# Оригинал

### Люция Газетдинова / Мөдәрис Газетдинов - Мәхаббәтне саклагыз

```
Табышкансыз, кавышкансыз,
Бүген сезнең туегыз.
Тормыш туйдай ямьле булыр
Бердәм булса уегыз.

Язсын сезгә бергә-бергә
Аерылмыйча гомергә,
Тормыш серен чишәргә.
Икегез дә бер җан булып,
Җирдә бәхетле пар булып,
Яшәргә дә яшәргә.

Карап сезгә туялмаслык
Бигрәкләр пар килгәнсез!
Мәңге бергә булырга дип,
Алтын балдак кигәнсез.

Язсын сезгә бергә-бергә
Аерылмыйча гомергә,
Тормыш серен чишәргә.
Икегез дә бер җан булып,
Җирдә бәхетле пар булып,
Яшәргә дә яшәргә.

Сөю гөлегез сулмасын,
Чит-ятлардан яклагыз.
Язмыш сезгә бүләк иткән
Мәхаббәтне саклагыз!

Язсын сезгә бергә-бергә
Аерылмыйча гомергә,
Тормыш серен чишәргә.
Икегез дә бер җан булып,
Җирдә бәхетле пар булып,
Яшәргә дә яшәргә.
```
