# Оригинал

### Светлана Шәрипова / Лилия Вахитова - Яши белә әнкәй сабыр итеп

```
Бу дөньяда ярый син бар әле,
Йөрәгемне ачар сердәшем.
Син барында тормыш нурга тула,
Әнкәем — кадерле кешем.

Кайтып киләм, тагы бер көн табып
Бу дөньяның бетмәс эшләреннән.
Әнкәй сөеп чәчләремнән сыйпый
Әллә өндә, әллә төшләремдә.

Уңганлыгың эзе — яраткан куллар
Чәчләремнән сыйпый, сабый итеп.
Сыгылмыйча язмыш җилләреннән,
Яши белә әнкәй сабыр итеп.

Кайтып киләм, тагы бер көн табып
Бу дөньяның бетмәс эшләреннән.
Әнкәй сөеп чәчләремнән сыйпый
Әллә өндә, әллә төшләремдә.

Ап-ак кырау әллә төшкән инде
Ак яулыктан чыккан чәчләреңә.
Риза булчы, әнкәй, зинһар, бездән,
Без үскәндә тамган яшьләреңә.

Кайтып киләм, тагы бер көн табып
Бу дөньяның бетмәс эшләреннән.
Әнкәй сөеп чәчләремнән сыйпый
Әллә өндә, әллә төшләремдә.
```
