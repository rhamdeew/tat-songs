# Оригинал

### Рәмис Әймәт / Фәрид Низамиев - Нигә сүзләр

```
Нигә сүзләр? Инде бар да ачык —
Язмышлардан булмый барыбер качып.
Кергән идек икәү бу дөньяның
Сөю дигән капкаларын ачып…

Кергән идек…
Әле ул чагында
Тылсымланган төшләр кочагында
Без тирбәлгән идек. Сүрәтебез
Янган иде балкып таң чыгында.

Таң чыклары аккан инде эреп.
Ялгыш кына сүрәтеңне күреп
Сискәнермен кебек. Хәзер инде
Ул чакларны йөрәгемә төреп,
Саклап йөртәм бер истәлек итеп,
Әле кабат кавышулар көтеп.
Ярсый-ярсый типкән йөрәгемне
Әллә нинди уйлар ала өтеп.

Әллә нинди хисләр кабына да…
Үткәннәрне йөрәк сагына ла.
Ялгыз шәмдәй "Сүнәм!" дигәндә дә,
Җанны хәтирәләр йолып кала.

Күмелсә дә инде күптән эзләр,
Күңел һаман ул чакларны эзләр…
Качып булмый барыбер язмышлардан —
Инде бар да ачык. Нигә сүзләр?

Таң чыклары аккан инде эреп.
Ялгыш кына сүрәтеңне күреп
Сискәнермен кебек. Хәзер инде
Ул чакларны йөрәгемә төреп,
Саклап йөртәм бер истәлек итеп,
Әле кабат кавышулар көтеп.
Ярсый-ярсый типкән йөрәгемне
Әллә нинди уйлар ала өтеп.
```
