# Оригинал

### Нәби Дәүли / Мансур Мозаффаров - Москва

```
Мин уянам алсу таң атканда,
Җырлаганда гүзәл Москва.
Туган илнең бөек башкаласы,
Йөрәгемдә сүзем бар сиңа.

Дөньяда син бер генәсең, Москва, Москва,
Гомергә син күңелдәсең, Москва, Москва!

Чал Кремль мәгърур басып тора,
Гасырларны алга үткәреп.
Ул чакыра безне киләчәккә,
Ялкын төсле байрак күтәреп.

Дөньяда син бер генәсең, Москва, Москва,
Гомергә син күңелдәсең, Москва, Москва!

Таш кыялы диңгез артларыннан
Кеше уе сиңа ашкына.
Халыкларга иркен тормыш бирдең,
Туган ана, яңа Москва!

Дөньяда син бер генәсең, Москва, Москва,
Гомергә син күңелдәсең, Москва, Москва!
```
