# Оригинал

### Мостафа Ногман / Рөстәм Яхин - Ак җилкән

```
Ак җилкәннәр белән кичеп үтәм
Хыялымның күксел диңгезен.
Син яшәгән ярлар ерак микән, —
Шуны гына уйлыйм мин үзем.

Таң җилләре чәчләремдә уйный,
Акчарлаклар озата көймәмне.
Беләсеңме икән, назлы тынам,
Мин ашкынып сиңа килгәнне?

Хыялымда язлар чәчәк ата,
Тибрәнгәли кебек бөдрә тал.
Ак җилкәндәй ак күңелем белән
Яннарыңа киләм, каршы ал.
```
