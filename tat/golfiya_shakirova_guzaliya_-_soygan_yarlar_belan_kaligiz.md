# Оригинал

### Гөлфия Шакирова / Гүзәлия - Сөйгән ярлар белән калыгыз

```
Сөйгән ярлар булсын гомерлек,
Хисләреңә тәңгәл килерлек.
Көткән бәхет, сөю-мәхаббәт,
Көннәреңә шатлык бирерлек.

Сөйгән ярлар белән калыгыз,
Гомер буе парлап барыгыз.
Ачык булып, якты булып торган,
Киләчәккә юллар салыгыз.

Кадерледән булып кадерле,
Яшик әле шушы тормышта.
Бер уй белән бергә яшәсәк,
Килер шатлык, килер уңыш та.

Сөйгән ярлар белән калыгыз,
Гомер буе парлап барыгыз.
Ачык булып, якты булып торган,
Киләчәккә юллар салыгыз.

Гомер — мизгел, аккан су кебек,
Сынау арты килә сынаулар.
Бушка гомер узмасын ла, дигән,
Кабул булсын безнең юраулар.

Сөйгән ярлар белән калыгыз,
Гомер буе парлап барыгыз.
Ачык булып, якты булып торган,
Киләчәккә юллар салыгыз.
```
