# Оригинал

### Татар халык җыры - Хуш киләсез, туганнар

```
Чәчәкләр белән каршылый
Безне иркен урманнар.
Ерак юлны якын итеп,
Хуш киләсез, туганнар!

Хуш киләсез, хуш киләсез,
Хуш киләсез туганнар!

Эшләр әйбәт бара безнең,
Якларыбыз — мул ярар.
Түребез дә, күңелебез дә
Ачык сезгә, туганнар!

Хуш киләсез, хуш киләсез,
Хуш киләсез туганнар!

Татар халкы иҗат иткән
Җырлап туймас җырлар бар.
Шул җырларны бергәләшеп
Җырлыйк, әйдә, туганнар!

Хуш киләсез, хуш киләсез,
Хуш киләсез туганнар!
```
