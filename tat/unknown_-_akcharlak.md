# Оригинал

### Буранбай Искужин / Юлай Үзәнбаев - Акчарлак

```
Җил-давыллап очкан кошлар
Акчарлак түгел микән?
Син киткәнгә мин түзәрмен,
Йөрәгем түзәр микән? ..

Акчарлаклар уйнар күлдә
Табып алдым ак канат.
Сөюеңнең билгесе дип
Төшемә куйдым казап.

Каурый гына торып калды
Канатлы сөюеңнән
Кайтауазлар гына калды
«Яратам», — диюеңнән …

Җил-давыллап очкан кошлар
Акчарлак түгел микән?
Син киткәнгә мин түзәрмен,
Йөрәгем түзәр микән? ..
```
