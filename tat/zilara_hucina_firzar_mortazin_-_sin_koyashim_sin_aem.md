# Оригинал

### Зиләрә Хуҗина / Фирзәр Мортазин - Син — кояшым, син — аем

```
Син — кояшым, син — аем,
Син — минем йөрәк маем.
Яшәр өчен илһам алам
Мин синнән, җаныкаем.
Син — кояшым, син — аем,
Син — минем йөрәк маем.
Яшәр өчен илһам алам
Мин синнән, җаныкаем.

Син — яңгырым, син — карым,
Син — бердәнберем, парым.
Син булмасаң, белмәс идем
Сөю-мәхәббәт барын.

Син — җәйләрем, син — көзем,
Син — сөйкемле һәм түзем.
Синең өчен таңнар булы
Атар идем мин үзем.
Син — җәйләрем, син — көзем,
Син сөйкемле һәм түзем.
Синең өчен таңнар булып
Атар идем мин үзем.

Син — яңгырым, син — карым,
Син — бердәнберем, парым.
Син булмасаң, белмәс идем
Сөю-мәхәббәт барын.

Син — кышларым, син — язым,
Син минем моңлы сазым.
Синең өчен, синең өчен —
Сөю-мәхәббәт, назым.
Син кышларым, син — язым,
Син минем моңлы сазым.
Синең өчен, синең өчен —
Сөю-мәхәббәт,назым.

Син — яңгырым, син — карым,
Син — бердәнберем, парым.
Син булмасаң, белмәс идем
Сөю-мәхәббәт барын.
```
