# Оригинал

### Билгесез - Эх, Ләйсән, Ләйсән

```
Чимшә буена суга барасың,
Чиләкләреңне суга саласың.
Чишмә суыдай саф ул мәхәббәт,
Йөрисең, Ләйсән, миңа үпкәләп.

Эх, Ләйсән, Ләйсән, сагынам сине,
Судагы шәүләң нәкъ үзең төсле.
Эх, Ләйсән, Ләйсән, яратам сине,
Тагын бер тапкыр елмайсаң иде.

Кышкы кичләрдә без йөргән чаклар,
Мамыктан ялган эрере карлар.
Ник бу вакытлар тиз ътеп китте
Мәхббәтемне ник алып китте?

Эх, Ләйсән, Ләйсән, сагынам сине,
Судагы шәүләң нәкъ үзең төсле.
Эх, Ләйсән, Ләйсән, яратам сине,
Тагын бер тапкыр елмайсаң иде.
```
