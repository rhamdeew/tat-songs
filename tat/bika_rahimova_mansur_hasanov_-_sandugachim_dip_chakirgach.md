# Оригинал

### Бикә Рәхимова / Мансур Хәсәнов - Сандугачым, дип чакыргач…

```
Сандугачым гына дип чакыргач,
Килеп китсәң генә ни була?
Тәрәз чиртеп сәламнәрем килсә,
Искән җилдә мин бар дип уйла.

Сандугачым гына дип чакыргач,
Кунган талың әйтсәң ни була?
Сине күзләп күктә — йолдыз яна,
Ул йолдызны мин диеп уйла.

Сандугачым гына дип чакыргач,
Килеп кунсаң гына ни була?
Сайрар кошым, сине сагынып көткән
Якын дустың бар диеп уйла.
```
