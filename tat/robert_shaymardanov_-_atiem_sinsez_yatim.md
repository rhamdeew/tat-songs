# Оригинал

### Роберт Шәймәрданов - Әтием, синсез ятим

```
Салкын кышларны мин көтәмен,
Көзләр кинәттән көндәшем.
Салкыннарда, бәлкем, бозга
Әйләнер күк күз яшем.

Котырып уңган миләшләре,
Әти, җилдә җиргә коела.
Тәнең суынган агач төбендә
Һаман җаным тоела.

Дәва эзләп бозлы тәрәзәгә
Ясар идем, әти, бер рәсем.
Сүрәтеңне телим тылсымлап
Исән килеш күрәсем…

Гомерләре ничек булса да
Синсез,әти,мин ятим.
Күзләр аккан яшьләремне …
Әткәй,бәгърем,нишләтим…
```
