# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Безгә хәзер егерме!

```
Тулпар итеп йөгәнләдек,
Җилдә елгыр елларны.
Юлдаш итеп юлга алдык
Иң шәп матур җырларны.

Безгә хәзер егерме,
Без шәп әле җигәрле,
Гомерләр алда әле.
Җырның шәптән-шәпләрен,
Җырлыйсылар бар әле,
Яшиселәр бар әле.

Егерме яшь, чын егетнең,
Батыр чагы, шәп чагы.
Бездә япь-яшь, без шәптән-шәп,
Җанда җыр, моң усагы.

Безгә хәзер егерме,
Без шәп әле җигәрле,
Гомерләр алда әле.
Җырның шәптән-шәпләрен,
Җырлыйсылар бар әле,
Яшиселәр бар әле.

Безнең җырлар, сезнең янда,
Бәйрәмнәр дә, табын да.
Очрашырга,язып торсын,
Тагын, тагын, тагында.

Безгә хәзер егерме,
Без шәп әле җигәрле,
Гомерләр алда әле.
Җырның шәптән-шәпләрен,
Җырлыйсылар бар әле,
Яшиселәр бар әле.
```
