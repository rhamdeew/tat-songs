# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Үпкәләттемме әллә?

```
Мин яраткан көздә, нур балкыган йөздә
Энҗе кебек яшьләр ялтырый.
Кояш каралганмы, сагыш таралганмы,
Бәхет чаткылары балкымый.

Үпкәләттемме әллә, ник син шундый хәлдә,
Гаепләрем бармы алдыңда?
Үпкәләмә җаным, аңлашыйк без барын,
Салма мине, салма ялкынга.

Ниләр булды инде, нинди җилләр тиде,
Ашыгыпмы каты кагылдым?
Бүген туган үпкәм мәшәкатьvе икән,
Әллә инде бераз тарыды.

Үпкәләттемме әллә, ник син шундый хәлдә,
Гаепләрем бармы алдыңда?
Үпкәләмә җаным, аңлашыйк без барын,
Салма мине, салма ялкынга.

Синең зарларыңа, йөрәк моңнарыңа
Мин бүгеннән киртә куярмын.
Гомер буе сөеп, утларыңда көеп
Яшәсәм дә сөеп туймамын.

Үпкәләттемме әллә, ник син шундый хәлдә,
Гаепләрем бармы алдыңда?
Үпкәләмә җаным, аңлашыйк без барын,
Салма мине, салма ялкынга.
```
