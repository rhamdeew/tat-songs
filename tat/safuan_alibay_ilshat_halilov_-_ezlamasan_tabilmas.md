# Оригинал

### Сафуан Әлибай / Илшат Хәлилов - Эзләмәсәң — табылмас

```
Ашкынам айлар, еллар,
Сине эзлимен мин,
Кайда гына югалдың?
Һаман белмимен мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.

Эзләрмен җирдә, күктә,
Янармын утта мин.
Мең урап гәләмнәрне,
Барыбер табармын мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.

Бергә әйләнәбез
Иксез-чиксез дөнъяда.
Җиргә бит кешеләр,
Серле генә күзләп,
Сагышлардан йөзәп,
Бер-бересен эзләп
Табар өчен ярала.

Ашкынам айлар, еллар,
Сине эзлимен мин,
Кайда гына югалдың?
Һаман белмимен мин,
Ә мәхәббәт
Эзләмичә мәңгегә табылмый,
Йөрәгеңдә ялкыннар кабынмый.
```
