# Оригинал

### Нәҗип Әсәнбаев / Юлай Үзәнбаев - Чиялетау (икенче вариант)

```
Чиялетау буйларында
Калды яшьлек эзләрем.
Диңгез кичеп, чит җирләрдән
Мин бәхетләр эзләдем!

Зәңгәр офыклар артында
Чиялетауым калды.
Ак тирмәле зәңгәр күктә
Ата-бабалар калды.

Кояш чыкса офыклардан,
Чиялетауны күрәм.
Туган илем, сине уйлап,
Күз яшьләремне түгәм!…
```
