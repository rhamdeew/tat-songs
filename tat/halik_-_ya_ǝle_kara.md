# Оригинал

### Халык - Йә, ǝле кара

```
Әй, дускаем буе зифа
Тал кебек бөгеләсең.
Таллар кебек бөгеләсең
Кем өчен өзеләсең?

Йә, әле кара-кара,
Кара күземә генә.
Карасаң күземә генә
Булырсың үзеңә генә.

Әй, дускаем алып барам
Саф чишмә суларына.
Чишмә су шифа диләр
Йөрәк ярсуларына.

Йә, әле кара-кара,
Кара күземә генә.
Карасаң күземә генә
Булырсың үзеңә генә.

Әй, дускаем сиңа әйтәм
Сер сөйләмә дустыңа.
Дустымның да дусты булып
Ул да сөйләр дустыңа.

Йә, әле кара-кара,
Кара күземә генә.
Карасаң күземә генә
Булырсың үзеңә генә.
```
