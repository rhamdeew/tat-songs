# Оригинал

### Гәүһәр Йосыпова - Өмет (өченче вариант)

```
Таралдылыр томан кебек,
Яшьлекнең хыяллары.
Үкенечсез узмый бит ул
Гомерләрнең юллары.

Дөнья булгач — бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!

Гомер юлларында күпме
Хата, ялгышлары да,
Өмет җитәклидер мине
Адашыр чакларымда.

Дөнья булгач — бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!

Айкалдыра, чайкалдыра
Тормышның киртәләре,
Бик өмет белән нурлансын
Һәркемнең иртәләре.

Дөнья булгач — бар да була:
Булса авыр чаклары.
Алга кара: анда яна
Өметнең маяклары!
```
