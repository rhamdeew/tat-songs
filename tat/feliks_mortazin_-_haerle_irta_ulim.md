# Оригинал

### Феликс Мортазин - Хәерле иртә, улым

```
Елмаеп кояш чыкканга
Җиргә якты көн килә
Минем улым уяна да
Дөньяга аваз бирә

Улым минем куанычым син
Киләчәктә таянычым син
Улым минем йөрәк җимешем
Мәхәббәтем, алтын, көмешем
Хәерле иртә — кояшым
Хәерле иртә — улым

Нәни кулларын суза ул
Сөю гөлебез чәчәге
Ике күзе, ике төймә
Уң булсын киләчәге

Улым минем куанычым син
Киләчәктә таянычым син
Улым минем йөрәк җимешем
Мәхәббәтем, алтын, көмешем
Хәерле ирт — кояшым
Хәерле иртә — улым

Дөнья белән таныша ул
Әти-әни бәхете
Кадерле ул, акыллы ул
Дөньяның шәп егете

Улым минем куанычым син
Киләчәктә таянычым син
Улым минем йөрәк җимешем
Мәхәббәтем, алтын, көмешем
Хәерле иртә — кояшым
Хәерле иртә — улым
```
