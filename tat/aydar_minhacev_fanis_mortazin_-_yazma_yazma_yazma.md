# Оригинал

### Айдар Минһаҗев / Фәнис Мортазин - Язма, язма, язма

```
Нигә миңа хатлар язып,
Вакытыңны бүләсең?
Алар бит кала җавапсыз,
Үзең шуны беләсең.
Хатларыңны ташлый барам.
Килми сине күрәсем.

Язма, язма, язма, язма.
Син миңа кирәк түгел.
Элек кирәк идең, хәзер
Башканы сөя күңел.

Узып барган чакларда син,
Ник карыйсың күземә?
Оныттым бит инде күптән.
Ышан инде сүземә.
Мине гаепле санама,
Гаеп синең үзеңдә.

Оныт мине, оныт мине.
Миңа рәнҗеп карама.
Кабат мине уйлап йөреп,
Тоз саласың яраңа.

Очрашулар эзләмә син
Йөрмә инде юлымда.
Миндә түгел бит бәхетең,
Бәхетең үз кулыңда.
Шул бәхетне өзәп бару,
Юкмы дисең куемда.
```
