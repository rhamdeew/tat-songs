# Оригинал

### Гөлшат Зәйнәшева / Гөлнара Беляева - Без — табигать дуслары

```
Җәйге ямьле иртәдә
Чыгабыз болыннарга.
Әйлән-бәйлән уйныйбыз
Иң матур урыннарда.

Без — табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.

Керәбез урманнарга,
Йөрибез аланнарда.
Куркытмыйбыз кошларны,
Тимибез боланнарга.

Без — табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.

Җәйге ямьле иртәдә,
Көн уянган сәгатьтә,
Күңелле дуслар белән
Ял итү табигатьтә.

Без — табигать баласы,
Без табигать дуслары.
Безне сәламләп сайрый
Туган якның кошлары.
```
