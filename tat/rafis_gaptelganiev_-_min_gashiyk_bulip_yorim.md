# Оригинал

### Рафис Гаптелганиев - Мин гашыйк булып йөрим

```
Бакчаларда мин йөрим,
Сиңа чәчкәләр эзлим.
Ак чәчәк аткан шомырт,
Шау чәчәктә ак сирень.

Ак шомырт та ак сирень,
Юк, аларны мин өзмим.
Ак шомырттай ак сиреньгә
Мин гашыйк булып йөрим.

Әкрен генә таң ата,
Кошлар сайрап уята.
Шомырт җиләк ярата,
Сирень хуш ис тарата.

Ак шомырт та ак сирень,
Юк, аларны мин өзмим.
Ак шомырттай ак сиреньгә
Мин гашыйк булып йөрим.

Бакчаларда мин йөрим,
Вакыт үткәнен сизмим.
Яратканыңны белмим,
Бер минутка да түзмим.

Ак шомырт та ак сирень,
Юк, аларны мин өзмим.
Ак шомырттай ак сиреньгә
Мин гашыйк булып йөрим.
```
