# Оригинал

### Рауза Хафизова / Гүзәлия - Каенкаем

```
Каенкаем, зифа буең,
Җилләрдән сыгыласың.
Иркәләнеп имәннәргә
Киләдер сыенасың.
Имәннәр каен түгел шул,
Каеннар имән булмас.
Имәннәр баш имәс димә,
Язмыш ни генә кылмас.

Бер карыйм каеннарыма,
Бер карыйм имәннәргә.
Ышанмагыз язмышлардан
Узмыш юк дигәннәргә.

Бер каенга — бер каендыр,
Имәнгә имән тигән.
Язмыштан узмыш түгелме
Бер каенга — бер имән.
Пар итә каеннарын да
Баш игән имәннәре.
Язмышларга карамый шул
Бер-берсен сөйгәннәре.

Бер карыйм каеннарыма,
Бер карыйм имәннәргә.
Ышанмагыз язмышлардан
Узмыш юк дигәннәргә.
Бер карыйм каеннарыма,
Бер карыйм имәннәргә.
Ышанмагыз язмышлардан
Узмыш юк дигәннәргә.
Ышанмагыз язмышлардан
Узмыш юк дигәннәргә…
```
