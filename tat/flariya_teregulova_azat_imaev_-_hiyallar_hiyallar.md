# Оригинал

### Флария Терегулова / Азат Имаев - Хыяллар, хыяллар…

```
Үкенүләрдән тора үткән еллар,
Ә шулай да килә кайтасы.
Чая яшьлек калган аланнарга,
Яланаяк тагын басасы.

Төшләремдә яшьлегемә кайтам,
Тоям кочагыңның җылысын.
Иреннәрең үпкән кайнарлыктан
Уянамын кысылып сулышым.

Бер кайтасы килә, очрашасы,
Вәгъдәләшкән таллар куенында.
Көздә язлар көтеп яшим һаман.
Син кайда соң, кемнәр уеңда?
Син кайда соң, кемнәр куенында?..
```
