# Оригинал

### Сиринә Сәлахиева / Җәүдәт Гыйльманов - Гөлнарам

```
Мәхәббәт ул шундый серле нәрсә:
Елмайта,кайчак елата.
Авырлыклар килгән вакытта да,
Синең караш мине елмайта.

Гөлнарам,Гөлнарам,
Син диеп көям мин,янам.
Сине генә яратам диеп,
Кабатлармын мин һаман.

Миңа карап назлы бер елмайсаң,
Чабып түгел, мин очып барам.
Син булганда, мин бәхетле җирдә,
Мин яратам сине,Гөлнарам.

Гөлнарам,Гөлнарам,
Син диеп көям мин,янам.
Сине генә яратам диеп,
Кабатлармын мин һаман.

Чәчкәйләрең,серле күз карашың
Гашыйк итте мине үзеңә.
Сине генә сөям мин,Гөлнарам,
Карап туялмыйм күзеңә.

Гөлнарам,Гөлнарам,
Син диеп көям мин,янам.
Сине генә яратам диеп,
Кабатлармын мин һаман.
```
