# Оригинал

### Клара Булатова / Рәис Нәгыймов - Тамчы җыры

```
Тып-тып тама тамчы колак салчы,
Дөп-дөп тибә йөрәк, тыңлачы!
Тып-тып төшә — карны, бозны тишә,
Үзенекен итә бу тамчы.

Тып-тып төшә тамчы — бозны тишә,
Дөп-дөп тибә йөрәк, ни кирәк?
Бу тамчылар үзенекен итә,
Синең сүзне көтә бу йөрәк.

Күз алдыңа юри китер әле:
Шул тамчыдай, минем серләр, дип,
Юлсыз гына, тавыш-тынсыз гына
Йөрәгеңә синең үрләр, дип…

Тып-тып төшө тамчы — бозны тишә,
Дөп-дөп тибә йөрәк, ни кирәк?
Бу тамчылар үзенекен итә,
Синең сүзне көтә бу йөрәк.
```
