# Оригинал

### Роберт Рәкыйпов / Зиннур Сафиуллин - Сүнмәс йолдызым

```
Мин еракта, синнән бик еракта,
Мин кайтасы юллар бик озын.
Шул юлларда маяк нурым минем
Син, кадерлем, сүнмәс йолдызым.

Мин еракта, синнән бик еракта,
Бигрәк авыр миңа ялгызым.
Кайнар сулышыңны тоям төсле
Ялгыз чакта, янар йолдызым.

Мин еракта, синнән бик еракта,
Мин үтәсе көннәр бик озын.
Бу төннәрдә, юк-юк, мин адашмам,
Син кабынсаң, өмет йолдызым.
```
