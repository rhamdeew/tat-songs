# Оригинал

### Фәрит Бариев - Югарыдан хат

```
Гомерлеккә яшәрмен дип,
Аваз салдым якты дөньяга.
Исән-имин бәбкәм яшәсен дип,
Исем кушкан әнкәй балага.

Кайгырмагыз, якын туганнарым,
Янәшәдә гел мин булырмын.
Алсу таңның аткан нурларында,
Әни, җаным, ул мин булырмын.

Күбәләктәй күккә күтәрелдем,
Мин калдырдым газиз барысын.
Нишләтәсең, Ходай шулай кушкач,
Туганнарга калды сагышым.

Кайгырмагыз, якын туганнарым,
Янәшәдә гел мин булырмын.
Алсу таңның аткан нурларында,
Әни, җаным, ул мин булырмын.

Кадерләрен белеп яшибезме
Ходай биргән бер көн дөньяның?
Фатихасын ала алабызмы
Гомер биргән газиз әнкәйнең?

Кайгырмагыз, якын туганнарым,
Янәшәдә гел мин булырмын.
Алсу таңның аткан нурларында,
Әни, җаным, ул мин булырмын.

Беркемгә дә артык бирми икән
Без теләгән Ходай гомерне.
Һәммәбезнең якын туганнары
Булсын иде һәрчак кадерле.

Кайгырмагыз, якын туганнарым,
Янәшәдә гел мин булырмын.
Алсу таңның аткан нурларында,
Әни, җаным, ул мин булырмын.
Алсу таңның аткан нурларында,
Әни, җаным, ул мин булырмын…
Әни, җаным, ул мин булырмын…
```
