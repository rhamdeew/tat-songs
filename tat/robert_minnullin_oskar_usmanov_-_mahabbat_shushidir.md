# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Мәхәббәт шушыдыр

```
Мин сиңа карыйм да,
Елларны саныйм да,
Гомернең узганын аңламыйм.
Тик яшьлек узмаган,
Узмаган ул заман,
Диеп мин үземне алдыймын.

Күңелләр табышты,
Күңелләр кавышты,
Күңелләр мәңгегә кушылыр!
Син минем кадерлем,
Мин синең бәгырең —
Мәхәббәт шушыдыр, шушыдыр.

Без яшьне тоймадык,
Яратып туймадык,
Хисләрне тыймадык без бер дә.
Сөюдән исердек,
Барсын да кичердек
Без бергә, без бергә, без бергә.

Күңелләр табышты,
Күңелләр кавышты,
Күңелләр мәңгегә кушылыр!
Син минем кадерлем,
Мин синең бәгырең —
Мәхәббәт шушыдыр, шушыдыр.

Бер яндым, бер тындым,
Мин сиңа омтылдым,
Тик сиңа тартылам бүген дә.
Нәкъ кояш кебек син,
Елмаеп, көлеп син
Балкыйсың гомерем күгендә.

Күңелләр табышты,
Күңелләр кавышты,
Күңелләр мәңгегә кушылыр!
Син минем кадерлем,
Мин синең бәгырең —
Мәхәббәт шушыдыр, шушыдыр.
```
