# Оригинал

### Факил Әмәк / Әнвәр Бакиров - Кеше юлга чыкса…

```
Кеше юлга чыкса, икмәк теләп
Калдыра, ди,
Ишек өстендә.
Аннан инде ничек кайтмый калсын,
Аны өйдә ризыгы көтсен дә…

Еракларда,
Туган якта
Ризыгы көтсен дә…

Күңеле саф булса, ниятләре
Игелекле,
Юлы изгедән,
Юлчы үзе белән бер уч туфрак
Алып китәр бердәнбер нигездән.

Үзе уйлар,
Һаман җырлар,
Кайтырмын, дип, тиздән.

Халкым шул ук һаман, йолаларны
Һаман бозмый,
Һаман озата.
Шуңадырмы, ирләр кайталмаса,
Илләр гизгән моң булып җыр кайта.

Аналарны,
Сөйгән ярны
Кайтыр дип юата…
```
