# Оригинал

### Татар халык җыры - Замана такмаклары (беренче вариант)

```
Ага сулар, ага сулар,
Ага сулар арыктан.
Кызын озатырга баргач,
Куркып качкан сарыктан.

Синең иркәң армиядә
Танктан атам, дигән.
Алучы булса—бара тор,
Өйләнеп кайтам, дигән.

Безнең авыл егетләре
Аклы күлмәк кимиләр.
Кино билетыннан куркып
Кызлар белән йөрмиләр.

Хәзер егетләр кызыга
Байга да, матурга да.
Ахыр чиктә риза була
Балалы хатынга да.

Әнә килә автомобиль
Син генә утыр инде.
Сөйгән ярың кырык тугыз
Иллегә тутыр инде.

Егетләрнең кайберләре
Чәчләрен тарый майлап.
Чәче агара, егет үзе
Картая, кызлар сайлап.
```
