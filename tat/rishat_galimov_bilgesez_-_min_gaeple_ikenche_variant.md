# Оригинал

### Ришат Галимов / Билгесез - Мин гаепле (икенче вариант)

```
Соң булса да, сиңа мин хат язам,
Ничә еллар күрми яшәдем.
Исемең дә инде онытылган,
"Чибәр кыз" дип кенә дәшәмен.

Гаебем зур минем синең алда,
Яшьлегемдә сине алдадым.
Соң булса да, гафу сорыйм менә,
Инде түзәр чамам калмады.

Мин юләргә ник ышандың икән?!
Кочагыма кердең, дәшмәдең.
Ул мизгелдә, бәлки, сөйгәнсеңдер,
Тик соңыннан акты яшьләрең.

Гаебем зур минем синең алда,
Яшьлегемдә сине алдадым.
```
