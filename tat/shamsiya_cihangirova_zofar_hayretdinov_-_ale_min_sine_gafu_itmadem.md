# Оригинал

### Шәмсия Җиһангирова / Зөфәр Хәйретдинов - Әле мин сине гафу итмәдем

```
Әле мин сине гафу итмәдем,
Иңемә килеп сарыла күрмә.
Күз яшьләреңне күрсәтмә миңа,
Ялына күрмә, ялына күрмә.

Әле мин сине гафу итмәдем,
Өметләр баглап чәчәк китермә.
Йомшак ла аның күңеле диеп,
Гафу үтенмә, гафу үтенмә.

Әле мин сине гафу итмәдем,
Каршыма килеп башыңны имә.
Яратуымны белеп торасың,
Кире кайт, димә, кире кайт, димә.
```
