# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Сүнмәс өметләр

```
Миңа җан дуслар
Күңелем юатып,
Ярата сине, көтә ул диләр.
Өмет сүнмәсә,
Гөлләр шиңмәсә,
Еллар үтә ул, сизелми диләр.

Тан, җилләре дә,
Кош телләре дә
Синнән сәламнәр әйтер кебекләр.
Сагыну чиге
Кавышу бит ул,
Тик сүрелмәсен безнең өметләр.

Тормыш юлында
Бик күп киртәләр,
Аларның барысын җиңеп үтәрмен.
Йөрәгемдә саф
Сөю йөреткәч,
Кавышуга мин өмет итәмен.
```
