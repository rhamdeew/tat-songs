# Оригинал

### Айрат Шәйхетдинов / Айдар Тимербаев - Үткәннәр

```
Кем белән булгансың,
Кем белән кунгансың,
Белмимен, теләмим белергә.
Ул чакта үпкәннәр
Инде бит үткәннәр.
Ник кирәк тикшереп йөрергә?

Хәзер син янымда,
Мин синең назыңда,
Бездән дә бәхетле кеше юк.
Электән ни булган,
Яныңда кем торган,
Анысында минем эшем юк.

Карамыйм үткәнгә,
Алда ни көткәнгә,
Сөямен сөюдә мин бүген!
Кояш бит күңелдә,
Нурлары түгелә
Балкытып, яшәүнең зур күген.

Хәзер син янымда,
Мин синең назыңда,
Бездән дә бәхетле кеше юк.
Электән ни булган,
Яныңда кем торган,
Анысында минем эшем юк.

Үткәннәр үткәннәр,
Үләннәр үскәннәр,
Җир өсте капланган яшелгә.
Йолкымый үләнен
Шул яшел келәмнең,
Кирәктер сөелеп яшәргә.

Хәзер син янымда,
Мин синең назыңда,
Бездән дә бәхетле кеше юк.
Электән ни булган,
Яныңда кем торган,
Анысында минем эшем юк.
```
