# Оригинал

### Илшат Харисов / Оскар Усманов - Синең урам

```
Күпме урам арасында
Бар миңа кадерлесе.
Синең белән шул урамнан
Килә парлап йөрисе.

Синең урам, синең урам
Тартып тора күңелне.
Шатлык белән үтик, җаным,
Бергә һәрбер мизгелне.
Ике йөрәк дулкынлана
Күрешүне көткәндә.
Без бәхетле бу дөньяда
Парлап, бергә үткәндә.

Тормыш дигән олы юлга
Шул урамнан китәрбез.
Үзем сине, ә син мине
Бәхетлеләр итәрбез.

Синең урам, синең урам
Тартып тора күңелне.
Шатлык белән үтик, җаным,
Бергә һәрбер мизгелне.
Ике йөрәк дулкынлана
Күрешүне көткәндә.
Без бәхетле бу дөньяда
Парлап, бергә үткәндә.
Синең урам, синең урам
Тартып тора күңелне.
Шатлык белән үтик, җаным,
Бергә һәрбер мизгелне.
Ике йөрәк дулкынлана
Күрешүне көткәндә.
Без бәхетле бу дөньяда
Парлап, бергә үткәндә.
Без бәхетле бу дөньяда
Парлап, бергә үткәндә…
```
