# Оригинал

### Ренат Харис / Рөстәм Яхин - Бер генә сүз

```
Бер генә сүз,
Бер генә сүз әйтә алмый калдым,
Таба алмый калдым вакытын.
Очып киткән кошлар төркеменә
Иярмәс тә иде,
Иярмәс тә иде минем бәхетем…

Мин ул сүзне үземә дә әйтмәм,
Мин ул сүзне җырыма да кертмәм,
Мин ул сүзне йөрәгемдә йөртәм
Сиңа гына диеп,
Сиңа гына диеп
Йөртәм, иркәм.

Бер генә сүз,
Бер генә сүз әйтә алмый калдым,
Табалмадым җаен-вакытын…
Кем уйлаган — шулай кечкенәдән
Тора диеп,
Тора диеп кеше бәхете…
```
