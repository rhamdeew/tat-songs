# Оригинал

### Билгесез - Елама (беренче вариант)

```
Авыр сүз ишетсәң,
Бик авыр булса да,
Күзләрең күз яшькә,
Күз яшькә тулса да.
Күз яшькә тулса да…

Елама, елама!
Күз яшең килешми,
Гомерләр бик кыска,
Яшик без бирешми.
Елама, елама!

Бердәнбер көннәрең
Үз җирен ташласа,
Өзелеп сөюдән
Өметләр өзсәң дә.
Өметләр өзсәң дә…

Елама, елама!
Күз яшең килешми,
Гомерләр бик кыска,
Яшик без бирешми.
Елама, елама!

Кояшлы көннәрен
Үз җирен ташласа,
Иң якын дусларың
Хыянәт итсә дә.
Хыянәт итсә дә…

Елама, елама!
Күз яшең килешми,
Гомерләр бик кыска,
Яшик без бирешми.
Елама, елама!
```
