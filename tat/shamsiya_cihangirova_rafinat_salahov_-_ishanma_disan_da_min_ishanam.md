# Оригинал

### Шәмсия Җиһангирова / Рафинат Сәлахов - Ышанма дисәң дә мин ышанам

```
Туры гына диеп барган идем,
Юлкайларым китте урауга.
Газиз әнкәй вакыт тапмагандыр
Баласына бәхет юрарга.

Баланнарны гына капкан идем,
Карлар явып киткәч балдыр дип.
Безләргә лә кемнәр юрады икән
Башкайларын ялгыз калдыр дип.

Утырдым ла биек яр читенә,
Суның агышларын күрергә.
Әгәр язган булса ирешермен
Чәчкәйләрне чәчкә үрергә.

Баланнарны гына капкан идем,
Карлар явып киткәч балдыр дип.
Безләргә лә кемнәр юрады икән
Башкайларын ялгыз калдыр дип.

Караланып килә кара болыт,
Кай якларда явар кар булып.
Ышанма дисәң дә мин ышанам
Гомер итәремә пар булып.

Баланнарны гына капкан идем,
Карлар явып киткәч балдыр дип.
Безләргә лә кемнәр юрады икән
Башкайларын ялгыз калдыр дип.
```
