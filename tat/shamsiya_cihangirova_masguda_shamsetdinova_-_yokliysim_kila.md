# Оригинал

### Шәмсия Җиһангирова / Мәсгудә Шәмсетдинова - Йоклыйсым килә

```
Кешелəр күптəн
Йоклаган инде,
Машиналар да
Туктаган инде.

Уенчыкларым
Йоклыйлар күптəн,
Тəти сеңлем дə
Бəү-бəүгə киткəн.

Күзлəрем əйтə:
“Йокым килə”, ди,
Телем дə əйтə:
“Əзрəк сөйлə”, ди.

Киемнəрем дə
Ял итə чөйдə,
Минем дə бик-бик
Йоклыйсым килə.
```
