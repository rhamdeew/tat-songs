# Оригинал

### Гөлнара Рәшитова / Ирек Галимов - Мәктәбем (өченче вариант)

```
Дәрескә чакыра кыңгырау чыңнары,
Он(ы)тылмас мәңгегә мәктәбем моңнары.
Ашык, ди, тизерәк, дәрескә соңлама,
Тормышка зур адым мәктәптән башлана…
Мәктәбем, йөз еллык үтә бай тарихлы,
Нигезләре аның какшамаслык ныклы.
Бүләк иттең миңа белем ачкычларын,
Көч бирдең үтәргә тормыш баскычларын…

Мәктәбем, мәктәбем, бәхетле балачак,
Ул еллар гомергә йөрәктә калачак.
Мәктәбем, мәктәбем, рәхмәтлемен сиңа,
Тормышка ишекләр ачып бирдең миңа.

Биеккә очарга канатлар куйдың син,
Мәңгегә рәхмәтле шуның өчен дә мин.
Мәктәбем, онытмам, еракка очсам да,
Кайтырмын сагынып, кайларда булсам да.

Мәктәбем, мәктәбем, бәхетле балачак,
Ул еллар гомергә йөрәктә калачак.
Мәктәбем, мәктәбем, рәхмәтлемен сиңа,
Тормышка ишекләр ачып бирдең миңа.
Мәктәбем, мәктәбем…
Мәктәбем, мәктәбем, бәхетле балачак,
Ул еллар гомергә йөрәктә калачак.
Мәктәбем, мәктәбем, рәхмәтлемен сиңа,
Тормышка ишекләр ачып бирдең миңа.
Рәхмәтле мин сиңа…
```
