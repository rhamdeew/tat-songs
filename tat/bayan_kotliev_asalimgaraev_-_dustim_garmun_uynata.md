# Оригинал

### Баян Котлыев / А.Салимгәрәев - Дустым гармун уйната

```
Дустым гармун уйната,
Был — был кебек сайрата.
Җырларга да ярата.
Көлеп күңел юата.

Гомер бушка үтмәсен,
Яшәү дәрте сүнмәсен.
Шатланыгыз, көлегез,
Җырлагыз сез, биегез.

Дустым гармун уйната,
Сихры моңнар тарата.
Моңсыз, гәмсез, йөрәкләр дә,
Изге хисләр уята.

Гомер бушка үтмәсен,
Яшәү дәрте сүнмәсен.
Шатланыгыз, көлегез,
Җырлагыз сез, биегез.

Тамашачы күп булса да,
Гармунчылар санаулы,
Өздереп гармун уйныйлар,
Көйләре дә давалы.

Гомер бушка үтмәсен,
Яшәү дәрте сүнмәсен.
Шатланыгыз, көлегез,
Җырлагыз сез, биегез.
```
