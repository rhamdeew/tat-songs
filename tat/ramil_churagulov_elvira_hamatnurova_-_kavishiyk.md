# Оригинал

### Рамил Чурагулов / Эльвира Хаматнурова - Кавышыйк

```
Сагындым, сагындым, сагындым
Син кайда, син кайда, син кайда
Бер көнне очрашып
Бер көнне табышып
Мәңгегә кавышыйк без әйдә

Җил бураннар
Кар яңгырлар
Һич кәртә була алмас
Кавышу минутын көтеп
Очрашуга барып бас

Кавышыйк, кавышыйк, кавышыйк
Син кайда, син кайда, син кайда
Һич сүнмәс, сүрелмәс
Һич сүнмәс саф хисләр
Ак хисләр кабызыйк без җанда

Җил бураннар
Кар яңгырлар
Һич кәртә була алмас
Кавышу минутын көтеп
Очрашуга барып бас

Табышыйк, очрашыйк, кавышыйк
Тын кичтә, ак төндә, ал таңда
Түз инде бәхетем
Түз инде ашыкма
Утларга син мине һич салма!

Җил бураннар
Кар яңгырлар
Һич кәртә була алмас
Кавышу минутын көтеп
Очрашуга барып бас
```
