# Оригинал

### Билгесез / Юлай Моратов - Яңа ел (икенче вариант)

```
Бар җиһанга бәхет теләп
Җирнең шары әйләнә.
Әткәй — әнкәй туганнарың,
Җан дусларың эргәдә.

Яңа ел, яңа җыр,
Барсыда шат булыр.
Тәннәргә, җаннарга,
Яңа ел тулыр.

Яңа елның, яңа җыры,
Мәңге истә калачак.
Сихри моңнарга күмелгән
Безнең алтын балачак.

Яңа ел, яңа җыр,
Хыяллар чында булыр
Бәхеткә, шатлыкка,
Яңа ел булыр.

Киләчәккә өметләр зур
Сүнмәс яшьлек дәртебез.
Томыш юлын дуслар белән
Бергә — бергә үтәрбез.

Яңа ел, яңа җыр,
Хыяллар чында булыр
Бәхеткә, шатлыкка,
Яңа ел булыр.
```
