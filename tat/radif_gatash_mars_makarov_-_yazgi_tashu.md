# Оригинал

### Рәдиф Гаташ / Марс Макаров - Язгы ташу

```
Канат какты гомер,
Җилкендерә җанны
Язгы җырларында.
Сокландык без, карап,
Язның бозларына,
Язның суларына.

Язгы ташу,
Су тавышы —
Моң агышы:
Әллә кавышу,
Әллә аерылышу?!

Гөрләгәндә сулар,
Безнең ярсу җырлар
Нигә һаман тынмый?
Язлар саен сулар
Шаулаганда, шулар
Нигә онытылмый?

Язгы ташу,
Су тавышы —
Моң агышы:
Әллә кавышу,
Әллә аерылышу?!

Язлар шулай килер,
Еллар моңы эрер
Ташкын суларында.
Кабатланыр мәңге
Безнең күңел яме
Яшьлек җырларында.

Язгы ташу,
Су тавышы —
Моң агышы:
Әллә кавышу,
Әллә аерылышу?!
```
