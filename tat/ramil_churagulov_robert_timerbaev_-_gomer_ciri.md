# Оригинал

### Рамил Чурагулов / Роберт Тимербаев - Гомер җыры

```
Талгын гына яуган карлар
Күмә көзнең гөлләрен,
Чәчәкләрне җылытам дип,
Үрелә иреннәрем.

Сөю хисе татымаган
Кызлар кебек ул гөлләр,
Кар астында күмелепме
Калган төсле гомерләр.

Язларда һәм җәен-көзен
Урман-аланда гөлләр,
Һәр кемгә дә төрле язмыш
Биргән бит ул гомерләр.

Сөю-хисләр кисерсеннәр
Шушы җиргә килгәннәр,
Сөелә күктә ай-кояш,
Җирдә иң матур гөлләр.

Умырзая кылларының
Хисләренән кайный кар,
Кар астында язса гомер
Узгара көн-атналар.

Сөю хисе татымаган
Кызлар кебек ул гөлләр,
Кар астында күмелепме
Калган төсле гомерләр.
```
