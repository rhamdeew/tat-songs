# Оригинал

### Илсөяр Шиһапова / Эльмир Низамов - Тормыш тузаны

```
Борма-сырма чаба юллар.
Чабам һаман үзем дә.
Кая барсам, илтә сыман
Юллар туган җиремә.

Яр буенда үргән чәчем
Таратмасын җил генә.
Тормыш тузаны кагылды
Кер белмәс күңелемә.

Еллар үтә, бер туктамый,
Ни син, гомер, ни яшәү?!
Яшәгәндә кирәктер ул
Ялкынланып бер күкрәү.

Яр буенда үргән чәчем
Таратмасын җил генә.
Тормыш тузаны кагылды
Кер белмәс күңелемә.

Чапсын юллар, үтсен гомер,
Булсын газап-әрнүләр.
Сагындырып килгән чакта
Татлы була бәхетләр.

Яр буенда үргән чәчем
Таратмасын җил генә.
Тормыш тузаны кагылды
Кер белмәс күңелемә.
Кер белмәс күңелемә…
```
