# Оригинал

### Гарифҗан Мөхәммәтшин / Фасил Әхмәтов - Нигә соң?

```
Яз дими, җәй дими, көз дими,
Күңелем гел якты моң тулы.
Ник белми бу йөрәк,
Ник белми бу йөрәк,
Ник белми бу йөрәк онытуны.

Азмыни бу җирдә чишмәләр —
Януың басарлык саф сулы,
Нигә соң бу йөрәк,
Нигә соң бу йөрәк,
Нигә соң бу йөрәк ярсулы?

Күңелем яктыга омтыла,
Әйтерсең гүя бер алмагач.
Кирәкме бу йөрәк,
Кирәкме бу йөрәк,
Кирәкме бу йөрәк янмагач?
```
