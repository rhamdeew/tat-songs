# Оригинал

### Билгесез - Күз яшьләре

```
Яшь аралаш елмайдым мин,
Син — бит минем бәгырем.
Сине күреп еладым мин,
Ихтыярсыз күз яшем.

Шатлыктан бу күз яшьләре,
Син минем янәшәмдә.
Шатлыктан бу күз яшьләрен
Яшералмыйм йөрәктә.

Көтмәдем мин кавышуны,
Югалттым дип уйладым.
Чиксез иде сагынуым,
Күз яшьләрем тыймадым.

Шатлыктан бу күз яшьләре,
Син минем янәшәмдә.
Шатлыктан бу күз яшьләрен
Яшералмыйм йөрәктә.

Аңладым, дип әйттең миңа,
Яратуның кадерен.
Авыр сүзләр әйтмәм сиңа,
Түкмә, җаным, күз яшен.

Шатлыктан бу күз яшьләре,
Син минем янәшәмдә.
Шатлыктан бу күз яшьләрен
Яшералмыйм йөрәктә.
```
