# Оригинал

### Вәлит Илембәтов / Гүзәл Үлмәсколова - Мәңгелеккә кил, иркәм!

```
Йолдыз балкый нурланып
Аяз күкнең йөзендә.
Син балкыйсың күңелдә,
Гашыйк булдым үзеңә.

Газаплама җанымны,
Кил иркәм!
Син йөрәктә шиңми торган —
Гөл икән.
Син бит миңа,
Мин бит сиңа —
Тиң икән!
Мәңгелеккә кил, иркәм!..

Йолдыз сүнеп кабына,
Күңел сине сагына.
Әле сиңа багышлап
Җыр чыгарам тагы да.

Газаплама җанымны,
Кил иркәм!
Син йөрәктә шиңми торган —
Гөл икән.
Син бит миңа,
Мин бит сиңа —
Тиң икән!
Мәңгелеккә кил, иркәм!..

Йолдыз сүнсә ае бар,
Кояш чыга ай батса.
Кеше көчле, бәхетле,
Үзенә тиң яр тапса.

Газаплама җанымны,
Кил иркәм!
Син йөрәктә шиңми торган —
Гөл икән.
Син бит миңа,
Мин бит сиңа —
Тиң икән!
Мәңгелеккә кил, иркәм!..
```
