# Оригинал

### Флера Шәрипова / Зарема Әюпова - Җылы сүз (беренче вариант)

```
Тормыш диңгезләрен кичкән чакта
Җил-давылдан мине сакладың.
Җылы сүзләр җаным тулы булды
Тик әйтергә чара тапмадым.

Нинди генә кырыс чаклар булсын
Авыр миңа иркәм, димәдең.
Күпме еллар, юллар үтелсә дә
Сөюемә кырау тимәде.

Син шатлыкта урый белдең мине
Сагыш кайчак иңгә кунса да.
Күзләреңә карап бүген сиңа
Рәхмәт әйтәм соңлап булса да.
```
