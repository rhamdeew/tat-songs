# Оригинал

### Рәсилә Йосыпова / Альберт Сабиров - Чишмә җыры (бишенче вариант)

```
Урап-урап тау буйларын
Чылтырап ага чишмә.
Сине күрер өмет белән
Суга төштем мин кичтән.

Чылтыр-чылтыр чишмә җырын
Җырлый сагынып, сагыштай.
Аерылгач исемә төште:
Әйтер сүзләрем калган.

Чишмә суы борма-борма,
Юлын таба елгалар.
Күңелем, бел, сине күрсәм,
Сагынудан йөрәк яна.

Чылтыр-чылтыр чишмә җырын
Җырлый сагынып, сагыштай.
Аерылгач исемә төште:
Әйтер сүзләрем калган.

Сиңа булган хисләремне
Йөрим мин сиздермичә.
Сине күргәч, ничек түзим
Яратуым әйтмичә.

Чылтыр-чылтыр чишмә җырын
Җырлый сагынып, сагыштай.
Аерылгач исемә төште:
Әйтер сүзләрем калган.
Аерылгач исемә төште:
Әйтер сүзләрем калган…
Әйтер сүзләрем калган…
```
