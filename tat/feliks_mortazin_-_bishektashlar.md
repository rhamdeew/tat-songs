# Оригинал

### Феликс Мортазин - Бишектәшләр

```
Бер ананың газиз балалары
Тәгәрәшеп, уйнап үстегез
Шатлыгын да, хәсрәтен дә күреп
Бергәләшеп буйга җиттегез

Хәзер инде үсеп җитсәгез дә,
Туган йорттан чыгып китсәгез дә,
Бишектәшем диеп дәшегез!!
Кан кардәшләр булып яшәгез!

Кайгыларны җиңеп була икән
Гайбәтләрне булмый кичереп
Авырлыклар җирдә булса булсын
Булмасын тик җирдә көнчелек.

Хәзер инде үсеп җитсәгез дә,
Туган йорттан чыгып китсәгез дә,
Бишектәшем диеп дәшегез!!
Кан кардәшләр булып яшәгез

Бер кайчан да әйтми авыр сүзләр
Үрнәк булып җирдә яшәрсез
Чын йөрәктән генә яратырсыз
«Бишектәшем» диеп дәшәрсез!

Хәзер инде үсеп җитсәгез дә,
Туган йорттан чыгып китсәгез дә,
Бишектәшем диеп дәшегез!!
```
