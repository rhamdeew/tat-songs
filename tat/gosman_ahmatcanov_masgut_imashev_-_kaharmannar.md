# Оригинал

### Госман Әхмәтҗанов / Мәсгуть Имашев - Каһарманнар

```
Туган илдә алсуланып
Матур таңнар ата микән?
Көзен киткән сандугачлар
Язын кабат кайта микән?
Бакчаларда — алмагачлар,
Кырда иген үсә микән?
Җәйге таңда тугайларга
Энҗе чыклар төшә микән?

Энҗе чыклар төшә таңда,
Язын кабат кошлар кайта.
Кайтмый бары каһарманнар,
Ил сагында калган алар.

Идел талгын ага микән?
Ярларына кага микән?
Нәниләрнең күзләрендә
Шатлык нуры яна микән?
Ярлар сөйгән ярсу җаннар
Кавышып бәхет таба микән?
Туган илдә алсуланып
Матур таңнар ата микән?

Идел кага ярларына,
Гөлләр калка, бәхет арта.
Язлар булып, таңнар булып
Каһарманнар илгә кайта.
```
