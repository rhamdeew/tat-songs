# Оригинал

### Мирһади Разов / Эльмира Галимова - Кышкы балан

```
Йөрәгемне бүләк иттем сиңа,
Ул синеке, сакла, дидем мин.
Уңны-сулны белми, берни күрми,
Нигә шулай өзелеп сөйдем мин.

Кышкы бакчаларда туңган балан,
Минем йөрәк гүя эленеп калган.
Телләремдә аның әче тәме,
Синсез юк бит яшәүнең дә яме.

Өметләрем сүнә бер-бер артлы,
Ник ышандым татлы ялганга…
Буем җитми, өзеп алыйм дисәм,
Үрелсәм дә туңган баланга.

Кышкы бакчаларда туңган балан,
Минем йөрәк гүя эленеп калган.
Телләремдә аның әче тәме,
Синсез юк бит яшәүнең дә яме.

Язны көтәм, сине көтмим инде,
Үпкәләмим салкын кышларга.
Чүпләгәннәр кызыл баланнарны,
Рәхмәт әйтим мәллә кошларга.

Кышкы бакчаларда туңган балан,
Минем йөрәк гүя эленеп калган.
Телләремдә аның әче тәме,
Синсез юк бит яшәүнең дә яме.
```
