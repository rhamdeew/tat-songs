# Оригинал

### Самат Хафизов / Илдар Хәсәнов - Чәчәкләрем

```
Сандугачлар сайрый су буенда,
Хисләремне минем телгәләп.
Күңелемнән китми гел уемда,
Чәчәк җыйган чаклар бергәләп.

Чәчәкләрем алсу гөлләрем
Чишмәләрем зәңгәр күлләрем,
Әтегезче кайда гүзәлем
Сагындым шул килә күрәсем.

Сандугачлар сайрый су буенда,
Алар моңын тыңлау әй рәхәт.
Елмагайнын күрәм төшләремдә,
Кавыштырчы безне мәхәббәт.

Чәчәкләрем алсу гөлләрем
Чишмәләрем зәңгәр күлләрем.
Әтегезче кайда гүзәлем
Сагындым шул килә күрәсем.

Сандугачлар сайрый су буенда,
Ямьләндереп җәйге таңымны.
Рәхмәт сиңа язмыш, яшлегемдә
Очраштырган өчен ярымны.

Чәчәкләрем алсу гөлләрем
Чишмәләрем зәңгәр күлләрем.
Әтегезче кайда гүзәлем
Сагындым шул килә күрәсем.
```
