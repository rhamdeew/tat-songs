# Оригинал

### Шәриф Биккол / Нурия Абдуллина - Кичер мине (беренче вариант)

```
Кичерә алсаң, әгәр, кичер мине,
Гаепләрем минем тау тиклем.
Мин гашыйкмын сиңа әвәлгечә,
Мин яратам сине, сөеклем.

Кичерә алсаң, әгәр, кичер мине,
Аткан угың тигән киекне.
Яралган җаным сыкрый һаман,
Яратам шул сине, сөеклем.

Кичерә алсаң, әгәр, кичерә күр,
Сөюемә килеп сөрлектем.
Гомермнең соңгы чигенәчә
Яратырмын сине, сөеклем.
```
