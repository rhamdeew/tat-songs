# Оригинал

### Гөлсинә Йосыпова / Виталий Агапов - Сүнгән өметләр

```
Өметләрем тагын сүнде инде,
Ничәнче кат сыный язмышым.
Нигә язмыш миңа шулай кырыс,
Кайда булды икән ялгышым?

Бәхетемне таптым инде, дисәм,
Әмма юкка, тагын югалтам.
Сагыш белән үтә һәрбер кичем,
Сагыш белән ата һәрбер таң.

Татлы хисләремне җимермичә,
Саклап күңелемнең түрендә,
Уйларым һәм сагыш белән бергә
Ялгыз калам томан эчендә.

Адашырмын күксел томаннарда,
Мәхәббәтем, тизрәк тартып ал.
Кайда соң син, иркәм, назлы гөлем,
Кайда соң син, чынлап сөйгән яр?
```
