# Оригинал

### Мирһади Разов / Ринат Гобәйдуллин - Чаллы сиреньнәре

```
Якты урамнары,
Чәчәк бураннары,
Тальян моңы китә еракка …
Чаллы сиреньнәре,
Айлы ак төннәре
Сөю бүләк итте йөрәккә.

Ләйсән яңгырлары,
Дәртле таң җырлары,
Ашкындыра җанны иртәләр.
Чаллы сиреньнәре,
Язның саф җилләре
Кабат-кабат безне иркәләр.

Чулман буйларында,
Яшьлек туйларында
Ялкынланып яна учаклар.
Чаллы сиреньнәре,
Кайнар иреннәрең —
Сагындыра дәртле шул чаклар.
```
