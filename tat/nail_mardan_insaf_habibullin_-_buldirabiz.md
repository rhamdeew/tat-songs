# Оригинал

### Наил Мәрдан / Инсаф Хәбибуллин - Булдырабыз!

```
Бездә җырлар бетә торган түгел,
Җилкәбезгә кошлар кундырабыз.
Күңелебез тулы моңнар белән
Тотынабыз икән, булдырабыз!

Бездә икмәк бетә торган түгел,
Шушы куллар белән уңдырабыз.
Җиң сызганып “биссмилла”ны әйтеп,
Тотынабыз икән, булдырабыз!

Бездә иман бетә торган түгел,
Җомгаларны тыныч уздырабыз.
Изгелеккә дога белән генә
Тотынабыз икән, булдырабыз!

Бездә егәр бетә торган түгел,
Могҗизалар җирдә тудырабыз.
Бөтен дөнья татарлары бергә
Тотынабыз икән, булдырабыз!
```
