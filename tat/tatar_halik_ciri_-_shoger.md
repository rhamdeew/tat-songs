# Оригинал

### Татар халык җыры - Шөгер

```
Безнең Шөгер әйләнәсе
Сигез ярым чакрым;
Синнән башка сөйгәнем юк,
Ышанамсың, матурым?

Тальян гармун уйнаталар
Безнең Шөгер буенда;
Ятсам-торсам, керсәм-чыксам
Тик син генә уемда.

Шөгерләргә барам әле
Шөгер суы тәмлегә;
Бир кулыңны, мин дә бирәм
Аерылмаска мәңгегә.
```
