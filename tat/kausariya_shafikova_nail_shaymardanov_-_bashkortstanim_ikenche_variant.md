# Оригинал

### Кәүсәрия Шәфикова / Наил Шәймарданов - Башкортстаным (икенче вариант)

```
Бар бөркетле каялары
Каенлы урманнары.
Шифалы яуган яңгыры,
Шифалы яуган яңгыры,
Бар карлы бураннары.

Бу минем Башкортстаным,
Бу минем горурлыгым.
Шөкер бүген дә җырладым,
Туган җир матурлыгын.

Җиләкле  аланнары бар,
Бар икмәкле кырлары.
Балы бар, кара алтыны,
Балы бар, кара алтыны,
Балыклы елгалары.

Бу минем Башкортстаным,
Бу минем горурлыгым.
Шөкер бүген дә җырладым,
Туган җир матурлыгын.

Егетләре курай уйный
Җиде кыз тора биеп.
Әй бииләр чылытыр-чылтыр,
Әй бииләр чылытыр-чылтыр,
Тәңкәле түшлек киеп.

Бу минем Башкортстаным,
Бу минем горурлыгым.
Шөкер бүген дә җырладым,
Туган җир матурлыгын.

Сузылып ерак-еракка
Бормалы юллар ята.
Аккошлы якты күлләре,
Аккошлы якты күлләре,
Гөлләре чәчәк ата.

Бу минем Башкортстаным,
Бу минем горурлыгым.
Шөкер үзем дә җырладым,
Туган җир матурлыгын.
```
