# Оригинал

### Гөлара Шәрипова / Илгиз Субаев - Яшәү яме (икенче вариант)

```
Яшем арткан саен күңелемә,
Бөтенләй дә үзгә гамь керде.
Ыгы-зыгы, бәхәсләрдән качам,
Күңел тынычлыгым кадерле!

Саулык китә кайчак, әрнү-сыкрау
Урап ала тәнне, бәгырьне.
Дөнья малы берни түгел бит ул,
Сәламәтлек ай-һай кадерле!

Яшем белән зирәклегем арта,
Гомерләрнең арта кадере.
Чын күңелдән әйтәм, бүген миңа
Имин аткан таңым кадерле!
```
