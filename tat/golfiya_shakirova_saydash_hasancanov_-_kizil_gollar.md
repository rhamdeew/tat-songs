# Оригинал

### Гөлфия Шакирова / Сәйдәш Хәсәнҗанов - Кызыл гөлләр

```
Кызыл гөлләр, кызыл гөлләр,
Бигрәк якын төсләре.
Ялкын булса бәхетләре,
Юк икән шул исләре.

Сиңа бүләгем булыр дип,
Кызыл гөлләр эзләдем.
Авыр сүзләргә дә түзәм,
Аерылуга түзмәдем.

Кызыл гөлләр, кызыл гөлләр,
Күңелемдә калсыннар.
Кайнар хисләр гомер буе
Сине озата барсыннар.

Сиңа бүләгем булыр дип,
Кызыл гөлләр эзләдем.
Авыр сүзләргә дә түзәм,
Аерылуга түзмәдем.

Кызыкта син, кызганычта
Мине уйлап янасың.
Бер якын буласың кебек,
Бер еракта каласың.

Сиңа бүләгем булыр дип,
Кызыл гөлләр эзләдем.
Авыр сүзләргә дә түзәм,
Аерылуга түзмәдем.
```
