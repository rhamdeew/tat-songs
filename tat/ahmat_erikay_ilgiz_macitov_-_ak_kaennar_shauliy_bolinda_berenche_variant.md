# Оригинал

### Әхмәт Ерикәй / Илгиз Мәҗитов - Ак каеннар шаулый болында (беренче вариант)

```
Ак каеннар шаулый болында,
Әкрен генә җилләр искәндә;
Нурлы йөзең искә төшкәндә
Сагыну хисе таша җырымда.

Тау буенда, чишмә буенда
Ал чәчәкләр йөгергән төсле.
Синнән хәбәр китергән төсле,
Ак каеннар шаулый болында.

Ямьле күлнең зәңгәр суында
Ялгыз аккош моңаеп йөзә.
Үзәкләремне өзә-өзә,
Ак каеннар шаулый болында.
```
