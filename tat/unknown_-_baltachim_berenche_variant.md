# Оригинал

### Мидхәт Миншин / Фәрит Хатыйпов - Балтачым (беренче вариант)

```
Таш өстендә гөлләр үстерердәй
Кешеләрең синең, Балтачым.
Кем туса да газиз туфрагыңда,
Гел бәхеткә таба атласын.

Балтачым, Балтачым…
Гел бәхеткә таба атласын.

Шушма буйларыннан, дәртле җырың
Еракларга китә, Балтачым.
Мәйданнарны тотсын, батырларың,
Хезмәтләрең илдә макталсын.

Балтачым, Балтачым…
Хезмәтләрең илдә макталсын.

Бөреләнеп үскән талларыңа,
Сандугачлар кайта, Балтачым.
Кая сузылса да язмыш юлы,
Син үзеңә безне тартасың.

Балтачым, Балтачым…
Син үзеңә безне тартасың.
```
