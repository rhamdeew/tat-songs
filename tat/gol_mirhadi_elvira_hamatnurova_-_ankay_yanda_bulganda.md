# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Әнкәй янда булганда

```
Яратам иртәләрне
Уятса әнкәй сөеп,
Чакырса чәй эчергә
Тәмле коймаклар оеп.

Нинди рәхәт,
Нинди рәхәт
Әнкәй янда булганда,
Кочагына алып сөеп,
Назлап кына торганда.

Әнкәй карап торганда
Хәрефләр язам тезеп,
Мисалларны да эшлим
Азакка тиклем, түзеп.

Нинди рәхәт,
Нинди рәхәт
Әнкәй янда булганда,
Кочагына алып сөеп,
Назлап кына торганда.

Әнкәй теләп калганда
Бишлеләр дә эләгә,
Арымыйм да, талмыйм да,
Тарымыйм да бәлагә.

Нинди рәхәт,
Нинди рәхәт
Әнкәй янда булганда,
Кочагына алып сөеп,
Назлап кына торганда.

Кичен әкият сөйләп
Куенымда йоклата.
Еласам яшем сөртеп,
Үскәнем дип юата.

Нинди рәхәт,
Нинди рәхәт
Әнкәй янда булганда,
Кочагына алып сөеп,
Назлап кына торганда.
```
