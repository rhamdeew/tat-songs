# Оригинал

### Марсель Галиев / Ганс Сәйфуллин - Чатыр тау

```
Тургай булып,
тургай булып оча күңел,
менсәң биек тауларга.
Йолдызлар да,
йолдызлар да күзен кыса
без яраткан ярларга.

Синдәдер хыялы күпләрнең,
туган як тәхете — Чатыр тау, тау, тау!
Син безне биеккә күтәрдең,
туган як бәхете — Чатыр тау.

Монда җилләр,
монда җилләр алып кайта
бер гөнаһсыз чагыма, чагыма.
Каен кызы,
каен кызы, йөг(е)реп менеп,
кер әле кочагыма.

Синдәдер хыялы күпләрнең,
туган як тәхете — Чатыр тау, тау, тау!
Син безне биеккә күтәрдең,
туган як бәхете — Чатыр тау.

Тауларның да,
тауларның да маңгаена
языладыр, языладыр язмышлар.
Без ялгышсак,
без ялгышсак, тауларга да
кагыладыр ялгышлар.

Синдәдер хыялы күпләрнең,
туган як тәхете — Чатыр тау, тау, тау!
Син безне биеккә күтәрдең,
туган як бәхете — Чатыр тау.

Аргамаклар,
аргамаклар таудан чаба,
тугайларда, тугайларда — томаннар.
Бу дөньяны, бу дөньяны без шушылай
яратырга туганнар.

Синдәдер хыялы күпләрнең,
туган як тәхете — Чатыр тау, тау, тау!
Син безне биеккә күтәрдең,
туган як бәхете — Чатыр тау.
```
