# Оригинал

### Чыңгыз Мусин / Әнвәр Закиров - Ак хыялым

```
Ак хыялым ал таңнарда
Тирбәлә рәшә кебек.
Ул мине чиксез-кырыйсыз
Җирләргә дәшә кебек.

Дулкында кораб тирбәлә,
Ак хыялдай ак җилкән.
Җилкәнне җил ега, диләр,
Хыялны ега микән?

Ак хыялым — ак канатым,
Давылларга тиң көче.
Ак хыялын юрга итеп,
Дөнья гизә ир кеше.
```
