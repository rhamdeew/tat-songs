# Оригинал

### Марис Нәзиров / Фаил Абдраков - Чакыру

```
Без язмыш — ике ялыгыш,
Юлларыбыз аймылыш.
“Бәхетлесең” — дип уйлама,
Күзләремдә — моң-сагыш.

Онытырмын дигән идем,
Онытмадым — син кирәк.
Мин, күрәсең, хисләремне
Яшергәнмен төпкәрәк.

Яшли сөю — яшел сөю,
Упкын аша сикерү.
Көл булганчы янып-көю,
Йә зарыгып (төңелеп) сүрелү.

Үкенмәслек кебек идек,
Үкендерә — син кирәк.
Арадагы күперләрне
Яндырганбыз иртәрәк.

Ике (без) ялгыш — ике сагыш,
Ярлар һаман (сала) аймылыш.
Кошлар да ялгыз яши алмый,
Ник түгелбез без аккош?

Эндәшмәс тә кебек идем,
Чакырамын — син кирәк.
Күперләрне тергезик тә,
Дөнья булыр түгәрәк.
```
