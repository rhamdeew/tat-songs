# Оригинал

### Татар халык җыры - Түкбикә

```
Туксан урау урадым
Түкбикәнең турыннан.
Тәрәз ачып бер караса,
Күзләр чаг(ы)ла нурыннан.

Суга бара Түкбикә
Гел бакчасы буеннан.
Түкбикә узып киткәчтен
Хуш ис килә юлыннан.

Түкбикәнең кигәне
Аклы күлмәк бөрмәле.
Бу тирәләрдә бар микән
Түкбикәдәй чибәре?

Иртән торып таңнарда
Эчтем чишмә суларын.
Кая барсам, кайда йөрсәм
Түкбикәдә уйларым.
```
