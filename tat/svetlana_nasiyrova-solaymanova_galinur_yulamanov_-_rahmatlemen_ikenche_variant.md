# Оригинал

### Светлана Насыйрова-Сөләйманова / Галинур Юламанов - Рәхмәтлемен (икенче вариант)

```
Рәхмәтлемен синең әнкәеңә
Сиңа гомер бүләк иткәнгә.
Сиңа яшәү биреп, мине хисле итеп,
Тормышымны нурга төргәнгә.

Минем өчен туган матурым син,
Язмышым да минем, бәхетем дә.
Гомерем буе сине яшәтәсе иде
Сөеп кенә күңел тәхетендә.

Рәхмәтлемен мәңге Ходаема
Сине миңа насыйп иткәнгә.
Еллар дәвамында меңнәр арасында
Сине миңа саклап йөрткәнгә.

Рәхмәтлемен сиңа, жан сөйгәнем,
Язмышымны гөлгә күмгәнгә.
Күреп күзем туймас, сөеп күңел тынмас
Мәхаббәтем булып килгәнгә.
```
