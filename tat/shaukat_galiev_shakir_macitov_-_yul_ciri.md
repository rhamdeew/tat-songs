# Оригинал

### Шәүкәт Галиев / Шакир Мәҗитов - Юл җыры

```
Паровоз чакыра тагын
Ерак җирләргә,
Өйдә яту эшмени ул
Бездәй ирләргә!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Алда без шәһәр салачак
Далалар көтә.

Сөйгән ярлар да көтсен бер,
Сагынсын әле.
Тын яткан далада утлар
Кабынсын әле!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Алда без юллар сузачак
Урманнар көтә.

Җилкендереп күңелләрне
Ерак җир тарта,
Йөргән саен, күргән саен
Барыр җир арта!

Тәгәрмәчләр,
Тәгәрмәчләр
Йөгереп үтә,
Безне якты гомер юлы,
Зур тормыш көтә!
```
