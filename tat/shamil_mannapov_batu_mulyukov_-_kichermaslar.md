# Оригинал

### Шамил Маннапов / Бату Мулюков - Кичермәсләр

```
Синең белән аерылганнан бирле
Телим гөлгә үрелмәскә,—
Безне таныштырган
Алсу гөлчәчәкләр
Кичермәсләр төсле, кичермәсләр

Синең белән аерылганнан бирле
Йоклый алмыйм, ай, кичләрдә,—
Безне кавыштырган
Айлы матур кичләр
Кичермәсләр төсле, кичермәсләр

Синең белән аерылганнан бирле
Телим искә төшермәскә,—
Рәнҗетелгән яшьлек,
Уртак сөю-хисләр
Кичермәсләр безне, кичермәсләр.
```
