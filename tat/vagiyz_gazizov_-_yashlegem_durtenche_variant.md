# Оригинал

### Вәгыйзь Газизов - Яшьлегем (дүртенче вариант)

```
Чүршәм тугай буе,
Тукмак суы тәмле.
Дим елгасы ага,
Тукмагым да ямьле.
Кайтмый яшьлек, кайтмый,
Ике яшьлек булмый.
Кем сине сагынмый,
Кем генә юксынмый?

Яшьлегем, яшьлегем,
Ник шулай тиз үттең?
Яшьлегем, яшьлегем,
Ник үттең, тиз үттең…

Әти баскан эзләр,
Яшел хәтфә үлән,
Бергәләп йөгереп
Без авылда үскән.
Иң кадерлеләрем:
Әткәй, инәкәем,
Туганым сеңелем,
Сез, авылдашларым.

Яшьлегем, яшьлегем,
Ник шулай тиз үттең?
Яшьлегем, яшьлегем,
Ник үттең, тиз үттең…
Яшьлегем, яшьлегем,
Ник шулай тиз үттең?
Яшьлегем, яшьлегем,
Ник үттең, тиз үттең…
Ник үттең, тиз үттең…
```
