# Оригинал

### З. Насыров / Вәсил Галимов - Сине көтәм (беренче вариант)

```
Авыл өсләрендә айлы кичтә
Безнең дә бар иде пар йолдыз.
Аерылмабыз кебек иде һич тә
Бүген килеп нигә мин ялгыз?

Язмышлардан бер дә узып булмый
Ник соң аерды тормыш безне,
Аккан судай үтте гөмеребез,
Бик соңлап ачты безнең күзне.

Кичер мине, яшьлек мәхәббәтем
Синсез үтте бит гомер җәем.
Көз сагышы үтә үзәкләргә,
Сине көтәм мин колач җәеп.
```
