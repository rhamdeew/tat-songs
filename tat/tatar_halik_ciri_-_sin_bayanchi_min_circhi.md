# Оригинал

### Татар халык җыры - Син баянчы, мин җырчы

```
«Сөйгән ярың кем?» — дисәгез,
Гади генә бер кеше.
Гармун уйнау, баян уйнау
Аның бар белгән эше.

Син баянчы, мин җырчы ла,
Мин җырчы, син баянчы.
Баяныңны матур итеп
Гел уйнап кына торчы!

Башка кызларга уйнасаң,
Көнләшүдән үләрмен,
Шул баяның челпәрәмә
Ватылсын, дип теләрмен.

Син баянчы, мин җырчы ла,
Мин җырчы, син баянчы.
Баяныңны матур итеп
Гел уйнап кына торчы!

Башка кызлар җырлаганда
Баяныңда уйнама,
Миннән башка бер генә дә
Кыз турында уйлама!

Син баянчы, мин җырчы ла,
Мин җырчы, син баянчы.
Баяныңны матур итеп
Гел уйнап кына торчы!
```
