# Оригинал

### Сәет Шәкүр / Әнвәр Бакиров - Таң (беренче вариант)

```
Ерак тау артыннан күтәрелеп
Таң билгесе сызыла офыктан.
Куе караңгылык сыегая,
Йомшак җилләр исә котыптан.

Уяна да татлы йокысыннан
Таңны мактап сайрый сандугач.
Сәхрәләрдә, яшел болыннарда
Эш иртәсе туа, таң тугач.

Таң көч бирә эшчән беләкләргә,
Таң күтәрә рухны, күңелне.
Таң, җыр суза, биек зәңгәр күккә
Таң бизәкли безнең гомерне.
```
