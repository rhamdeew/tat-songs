# Оригинал

### Булат Ибраһимов / Илнур Ганиев - Исем эзлим песиемә

```
Әти алып кайтты безгә
Песи баласы.
Шул песигә нинди генә
Исем табасы.
Морза диеп кушаргамы,
Булсынмы Шаян?
Исем эзлим песиемә,
Табыйм соң каян?

Шаян песи баласы,
Нинди исем табасы?
Йомры йомшак, нәни йомгак,
Матур песи баласы.

Мыраубатыр диеп кушсам
Булырмы батыр?
Әллә инде тычканнардан
Ул куркып ятыр?
Кәтүк диеп кушыйм микән,
Йөри тәгәрәп.
Сезнең ярдәм кирәк, дуслар
Уйлыйк бергәләп.

Шаян песи баласы,
Нинди исем табасы?
Йомры йомшак, нәни йомгак,
Матур песи баласы.
```
