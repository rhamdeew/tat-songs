# Оригинал

### Татар халык / Мәсгудә Шәмсетдинова - Нәниләргә такмаклар

```
Барып кердем Хәйдәргә,
Башын тыккан әйрәнгә,
Башын алгач әйрәннән,
Куркып киттеп Хәйдәрдән.

Барган идем Якупка,
Башын тыккан катыкка,
Башын алгач катыктан,
Куркып киттеп Якуптан.

Барып кердем Ногманга,
Башын тыккан комганга,
Башын алгач комганнан,
Куркып киттем Ногманнан.

Барып кердем Гатага,
Башын тыккан катага,
Башын алгач катадан,
Куркып киттем Гатадан.

Барып кердем Хаматка,
Башын тыккан кабакка,
Башын алгач кабактан,
Куркып киттем Хаматтан.

Барып кердем карчыкка,
Башын тыккан капчыкка,
Башын алгач капчыктан,
Куркып качтым карчыктан.
```
