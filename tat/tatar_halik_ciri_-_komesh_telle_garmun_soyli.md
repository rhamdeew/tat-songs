# Оригинал

### Татар халык җыры - Көмеш телле гармун сөйли

```
Гармун алдым мин кулыма,
Чыктым урман юлына.
Кызлар белән таныштыр, дип,
Куштым мин гармуныма

Яңгыратып бистәләрне,
Гармун уйный кичләрен.
Көмеш телле гармун сөйли
Яшь егетнең хисләрен.

Кызлар күңеле гармунчыда,
Гармунчыга яр булчы.
Гармунчы белән гөрләшеп,
Гомрең рәхәт узсачы.
```
