# Оригинал

### Шәриф Биккол / Бәхти Гайсин - Гел генә

```
Ак пароход йөзеп килә
Агыйдел буйлап кына.
Гомеркәем үтәр инде
Гел сине уйлап кына.

Ак җилкәндәй ак каурый
Агыйдел өсләрендә.
Аппагым дип әйтүләрең
Гел генә исләремдә.

Көтәм сине җәйге кичтә
Агыйдел ярларында.
Гомер буе булсаң иде
Гел генә яннарымда.
```
