# Оригинал

### Булат Ибраһимов / Илнур Ганиев - Дуслык җыры (җиденче вариант)

```
Кояш балкып яна күкләрдә,
Кояш яшәү бирә күпләргә.
Без тынычлык телик һәр илгә,
Дуслык җыры булсын һәр җирдә.

Дус булмасак — дуслашыйк,
Дуслык ишеген ачыйк.
Тотынышып кулга кул,
Җирдән гөлбакча ясыйк.

Безгә диеп җирдә җил исә,
Безгә диеп җирдә гөл үсә.
Шатланабыз һәр яңа көнгә,
Бәхет юлдаш булсын гомергә.

Дус булмасак — дуслашыйк,
Дуслык ишеген ачыйк.
Тотынышып кулга кул,
Җирдән гөлбакча ясыйк.

Безнең җырдан күпер үрелсен,
Безнең җырдан җылы бөркелсен.
Дусларыбыз күп булсын җирдә,
Кайда яшәсәк тә без бергә!

Дус булмасак — дуслашыйк,
Дуслык ишеген ачыйк.
Тотынышып кулга кул,
Җирдән гөлбакча ясыйк.
```
