# Оригинал

### Мөнир Мазунов / Рөстәм Яхин - Язгы хисләр (беренче вариант)

```
Яратам язларның
Шау-шулы ярсуын.
Иртәнге таңнарның
Балкуын, алсуын.
Яратам суларның
Ярлардан ашуын,
Көз киткән кошларның
Янадан кайтуын,
Яңадан кайтуын.

Яратам язларның
Ягымлы җилләрен,
Без туган якларның
Чәчәкле гөлләрен.
Яратам язларның
Шул матур айларың,
Җаннарны җылыткан
Кояшлы майларын,
Кояшлы майларын.

Яратам йөрәккә
Дәрт биргән чакларны,
Уйларны уяткан
Шул шаулы язларны.
Яратам, яратам
Хисләрнең кайнарын,
Илемне яңарткан,
Эшләрнең кайнавын,
Эшләрнең кайнавын.
```
