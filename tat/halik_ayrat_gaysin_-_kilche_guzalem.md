# Оригинал

### Халык / Айрат Гайсин - Килче, гүзәлем!

```
Күл буенда камышлар
Шашып шаулыйлар.
Мәхәббәтсез кешеләр
Яши алмыйлар.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.

Күл буенда бөдрә таллар
Сер чишә дулкыннарга.
Чорнап алды мәхәббәтең
Әйләнеп ялкыннарга.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.

Серләрем күп, сердәшем юк,
Сөйләргә ме талларга?
Гөлләр булып үсәр идем
Син киләсе юлларга.

Килче, килче, гүзәлем,
Килче яныма!
Өзелә, өзелә, үзәгем
Бик сагынган чагымда.
```
