# Оригинал

### Нәфисә Хәбибдиярова / Миңзәлә Түләбаева - Хушлашулар авыр шул

```
Аһ, өзелә бит үзәкләр,
Казлар оча каңгылдап.
Җилләр кисә канатларын,
Оча каурый зың — зыңлап.

Тезелешеп оча казлар,
Каурыйлары кавырсын.
Газиз җирне ташлап китү,
Калдыруы авыр шул.

Казлар оча диңгез кичеп,
Таладыр канатлары.
Чит җирләрдә кемнәр генә,
Бәгъерен канатмады.

Тезелешеп оча казлар,
Каурыйлары кавырсын.
Миңа илтәсе юлларны,
Кош юлыннан табарсың.

Киткән кошлар бер кайтырлар,
Ояларын табарлар.
Вакытлар кирегә акмый,
Сулар акса, агарлар.

Тезелешеп оча казлар,
Каурыйлары кавырсын.
Кавышулар мөмкин түгел,
Хушлашулар авыр шул.
```
