# Оригинал

### Татар  халык җыры - Су буйлап (икенче вариант)

```
Идел бит ул, тирән бит ул,
Тирән бит ул, киң бит ул (ла).
Караңгы төн, ямансу көн —
Без аерылган көн бит ул.

Идел буйларында йөрдем
Тугайларын буйладым (ла).
Аерылгада кул бирмәдем
Күрешербез дип уйладым.

Идел буйларына барсам,
Сөйләр идем Иделгә (лә).
Минем сагышларымны да
Илтеп сал дип, диңгезгә.
```
