# Оригинал

### Гөлшат Зәйнәшева / Илгиз Мәҗитов - Урман юлында

```
Аерылдык урман юлында,
Җир җиләкле аланда.
Бик ямансу булды миңа
Бер ялгызым калганда.

Борылдым урман юлыннан,
Атладым синең эздән.
Җир җиләгенә чык түгел,
Яшь тамды минем күздән.

Җир җиләкләре коелыр,
Урман юлы тапталыр.
Көтәрмен сине гомергә,
Мәхәббәтем саф калыр.
```
