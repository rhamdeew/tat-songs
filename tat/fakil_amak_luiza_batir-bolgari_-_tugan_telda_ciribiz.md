# Оригинал

### Факил Әмәк / Луиза Батыр-Болгари - Туган телдә җырыбыз!

```
Кояш кебек кадерле,
Туган җир ул бер генә.
Туган илдә безнең җыр
Яңгырасын гел генә!

Туган телдә җырыбыз,
Матур да соң көебез.
Әйдә, дуслар, бергәләшеп
Биеп җырлыйбыз!

Кояшка гөл үрелә,
Дөнья нурга күмелә.
Туган илем, Туган тел
Якын, газиз күңелгә.

Туган телдә җырыбыз,
Матур да соң көебез.
Әйдә, дуслар, бергәләшеп
Биеп җырлыйбыз!

Аяз булсын күгебез,
Якты кояш көлсен гел.
Туган илем, Туган җир
Тыныч — имин булсын гел!

Туган телдә җырыбыз,
Матур да соң көебез.
Әйдә, дуслар, бергәләшеп
Биеп җырлыйбыз!
```
