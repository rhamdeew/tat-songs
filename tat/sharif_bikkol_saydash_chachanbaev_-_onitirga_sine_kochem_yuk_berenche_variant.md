# Оригинал

### Шәриф Биккол / Сәйдәш Чәчәнбаев - Онытырга сине көчем юк … (беренче вариант)

```
Кайдан килгән сиңа,
Кайдан килгән
Бу чаклы да яман саранлык?
Җанкаемны фида кылганда да,
Йөрәгеңне яулар чарам юк.

Күңелемә кырау була алмас,
Карашларың өргән салкынлык.
Син сөймәскә мине иреклесең,
Сөйдермәскә исә хакым юк.

Яздырсаңда соңғы өметемнән,
Ачуым юк сиңа, үсем юк.
Гомер буе көтер түземем бар,
Онытырга сине көчем юк …
```
