# Оригинал

### Динә Талхина / Рәзидә Әхмәтова - Алтын гомер (икенче вариант)

```
Алтын гына алтын балдак,
Алтын гына еллар,уза.
Алтын ашалуы сизелми,
Безнең алтын гомер туза.

Алтын гына алтын балдак,
Вакытны булмый шул алдап.
Алтын балдак биргән язларны
Көзләргә булмый шул ялгап.

Тузыр да гына гомерләрнең
Кадерен белми тотыла.
Шатланып яшәр урынга,
Үкенечтән ут йотыла.

Алтын гына алтын балдак,
Вакытны булмый шул алдап.
Алтын балдак биргән язларны
Көзләргә булмый шул ялгап.
```
