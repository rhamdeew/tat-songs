# Оригинал

### Рәүф Баһаветдинов / Халык - Белсәң иде уйларымны

```
Айлы төндә, аяз күктә,
Йолдызлар кабынганда.
Миңа рәхәт синең белән
Гел без бергә булганда.

Белсәң иде уйларымны,
Йокылардан да язганымны.
Әллә чынлап та, әллә чынлап та,
Сөю килдеме?

Яшәү ямем синдә генә,
Мин назлап сөяр идем.
Аз(ы)рак тагын күрми торсам
Зарыгып бетәр идем.

Белсәң иде уйларымны,
Йокылардан да язганымны.
Әллә чынлап та, әллә чынлап та,
Сөю килдеме?

Бәхет өстенә мин бәхет
Гел өстәп сөяр идем.
Нинди матур парлар, — диеп,
Бик күпләр сөйләр иде ..

Белсәң иде уйларымны,
Йокылардан да язганымны.
Әллә чынлап та, әллә чынлап та,
Сөю килдеме?
```
