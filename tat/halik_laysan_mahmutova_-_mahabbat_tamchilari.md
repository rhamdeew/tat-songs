# Оригинал

### Халык / Ләйсән Мәхмүтова - Мәхәббәт тамчылары

```
Аклымы килгәнегез,
Күклеме килгәнегез?
И скажите, пожалуйста:
Кем сезнең сөйгәнегез?
Ал алыймы, гөл алыймы
Тәрәзәңә пәрдәңә?
Әрәм буларсың дип куркам
Әллә нинди бәндәгә.

Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду,
Дигәнеңне яратам.
Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду,
Дигәнеңне яратам.
Яратам, яратам,
Яратам, яратам.

Ак пирчәткеләр кигәнсең,
Кулың ничек тирләми?
Бу кадәрле чибәр булып
Кайда үстең чирләми.
Ак буяуларга буя
Тәрәзә ырамнарын.
Ничек итеп оештыру
Кавышу пыланнарын?

Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду,
Дигәнеңне яратам.
Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду,
Дигәнеңне яратам.
Яратам, яратам,
Яратам, яратам.

Кулымдагы йөзегемнең
Исемнәре Әүхади.
Если любишь — приходи,
А не любишь — уходи.
Мине сөеп җилкәм атым
Туры белән саураска.
Син согласна, мин согласна,
Ник йөрибез напрасна?

Люблю тебя,
Тебя люблю.
Люблю тебя,
Тебя люблю.
Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду
Дигәнеңне яратам.
Там, там, тамчы там,
Тамчыларга сокланам.
Люблю тебя, не забуду,
Дигәнеңне яратам.
Яратам, яратам,
Яратам, яратам.
Яратам…
```
