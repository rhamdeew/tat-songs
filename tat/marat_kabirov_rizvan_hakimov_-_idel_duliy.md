# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Идел дулый

```
Идел дулый, Идел дулый —
Идел киң дә, тирән дә.
Ялгызлыгымны онытам,
Иделләргә киләм дә.

Сөю микән, сагыш микән —
Нидер дулый Иделдә.
Иделләр кебек дулаган
Чакларым бар минем дә.

Аккан сулар дулый инде,
Дулый да тынып кала.
Дөнья үзгәрми, шулай да
Җиңелрәк булып кала.

Җиңел була күңелемә,
Иделләргә киләм дә…
Шатлыкларым — иреннәрдә,
Сагышларым — тирәндә.
```
