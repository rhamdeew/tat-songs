# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Сиңа гына (икенче вариант)

```
Сиңа бүген, сиңа гына
Багышлыймын йөрәк җырын.
Бергә узган гомеркәйнең
Маңгаеңа яткан сырын.

Шул сырларның кадерләрен
Беләм үзем, үзем генә.
Дөнья матур яшәгәндә
Без икебез түзем генә.

Сиңа бүген, сиңа гына
Багышлыймын җылы сүзләр.
Миңә гына,миңа гына
Карый назлап синең күзләр.

Шул карашның кадерләрен
Беләм үзем, үзем генә
Дөнья матур яшәгәндә
Без икебез түзем генә.

Без икебез, икәү генә
Сабыр үттек гомер юлын.
Сагынамын, искә алам —
Мең кадерле һәрбер елын.

Шул елларның кадерләрен
Беләм үзем, үзем генә.
Дөнья матур яшәгәндә
Без икебез түзем генә.
```
