# Оригинал

### Зөлфәт Хәким - Апрельдә кар ява

```
Нинди сагыш соң бу, килә көн дә,
Булса иде шуңа бер дәва.
Арабызны тагын да салкынайтып
Апрель азагында кар ява.

Ява карлар, ява карлар эзләреңә синең,
Бураннарга керәм ялгызым.
Нишләргә соң, әйтче миңа, синсез килсә әгәр
Шулай салкын булып һәр язым?..

Табигатьнең булмый хаталары,
Тик тормышта туа хаталар.
Һаман ява карлар, яз булса да,
Эзләреңне күмеп ява кар.

Ява карлар, ява карлар эзләреңә синең,
Бураннарга керәм ялгызым.
Нишләргә соң, әйтче миңа, синсез килсә әгәр
Шулай салкын булып һәр язым?..
```
