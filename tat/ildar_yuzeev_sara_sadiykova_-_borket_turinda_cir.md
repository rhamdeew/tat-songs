# Оригинал

### Илдар Юзеев / Сара Садыйкова - Бөркет турында җыр

```
Хәбәр булып,
Килде болыт,
Кояш нурын каплады.
Давыл узды,
Оя тузды —
Бөркет дустын тапмады.
Ә баласы —
Күз карасы —
Сизмәде дә яшенне.
Ике канат,
Канап-канап,
Зур давылдан яшерде.

Очты бөркет,
Төште бөркет,
Йөрәгендә ярасы.
Яшь аралаш
Елмая ул:
Югарыда — баласы.
Ә баласы —
Күз карасы —
Сизмәде дә яшенне.
Ике канат,
Канап-канап,
Зур давылдан яшерде.
```
