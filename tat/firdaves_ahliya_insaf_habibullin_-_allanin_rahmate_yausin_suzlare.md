# Оригинал

### Фирдәвес Әһлия / Инсаф Хәбибуллин - Алланың рәхмәте яусын сүзләре

```
Алланың рәхмәте яусын
Тудырган әнкәеңә
Алланың рәхмәте яусын
Үстергән әбкәеңә.

Алланың рәхмәте яусын
Олысы, кечесенә.
Сиңа ярдәм иткән һәрбер
Ходайның кешесенә.

Алланың рәхмәте яусын
Сине яратканнарга.
Исемеңне догаларда
Һаман яңартканнарга.

Алланың рәхмәте яусын
Сиңа су биргәннәргә.
Өшкереп изге-шифалы
Сулышын өргәннәргә.
```
