# Оригинал

### Мәүлидә Әндәрҗанова - Теләк (бишенче вариант)

```
Сайрар кошлар тезелешеп оча
Туган ягыбызның күгендә.
Аяз күк! Кояш! Әти һәм Әни!
Барлык сабыйларның телендә.

Туган җирем миңа көч-дәрт бирә,
Авырлыкны җиңеп үтәргә.
Ана, кояш, җир җылысын тоеп,
Игелекле булып үсәргә!

Зәңгәр күктә, кояш нурларында
Һәрчак килә минем коенасым.
Җир-анама озын гомер теләп,
Килә минем җырлар җырласым.

Тынычлык дип, күгәрченнәр гөрли,
Киң елгалар ага шаулашып.
Болыннарда гөп-чәчәкләр үсә,
Иртән туган чыкка коенып.

Таңнарыбыз һәрчак аяз булсын,
Якты булсын туар көнебез.
Ятимлекләр күрми, гөрләп яшик
Тыныч булсын туган илебез!
```
