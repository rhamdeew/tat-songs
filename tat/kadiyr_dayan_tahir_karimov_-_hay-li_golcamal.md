# Оригинал

### Кадыйр Даян / Таһир Кәримов - Һай-ли, Гөлҗамал

```
Һай-ли, Гөлҗамал,
Көтүләре бик яман,
Бичәлеккә алам диеп,
Көтәм дә көтәм һаман.

Һай-ли, Гөлҗамал,
Нигә болай мин янам?
Унике ел көтә-көтә,
Әллә инде алданам.

Һай-ли, Гөлҗамал,
Синнән башка кемне алам?
Инде хәзер, алам диеп,
Әйтергә дә оялам.

Һай-ли, Гөлҗамал,
Көтәм әле мин һаман,
Сиңа болай күп, ялынмам.
Үкенерсең бер заман.

Һай-ли, Гөлҗамал,
Көтүләре бик яман,
Быел көзгә дә килмәсәң,
Инде бүтәнне алам.
```
