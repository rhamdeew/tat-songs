# Оригинал

### Мансур Шиһапов / Сара Садыйкова - Комсомол җыры

```
Безнең яшьлек нидән башланды? —
Ирләр утка кергән дәвердән,
Ятимнәргә назлы җылысын биргән
Әнкәемнең мамык шәленнән.

Бәлкем башлангандыр әткәй булып
Колхоз кырын сөргән таңнардан.
Унҗидедә чишмә буйларында
Сөю бүләк иткән язлардан.

Бәлкем башлангандыр чик буенда
Илгә биргән антны үтәүдән.
Безнең яшьлек КамАЗ нигезен салып,
БАМнардагы юлны күтәргән.

Безнең яшьлек әле үтелмәгән,
Үтеләчәк Данлы юлга тиң.
Безнең яшьлек әле язылмаган,
Җырланачак моңлы җырга тиң.
```
