# Оригинал

### Әхмәт Ерикәй / Сара Садыйкова - Сагыну вальсе

```
Иртән торсам, кичен ятсам,
Сине уйлыйсым килә.
Йөрәгемнең юксынуын
Җырда җырлыйсым килә.

Уйларым тирән диңгездәй,
Уйласам, уйга батам.
Йөрәгемнең ашкынуын
Көчкә генә юатам.

Сагынып, сагынып җырлар җырлыйм,
Ишетә микән иркәм.
Ул да шулай минем кебек
Өзелеп сагына микән.
```
