# Оригинал

### Гөлүзә Арсланова - Алтын тау эзләмәгез

```
Алтын тау эзләмәгез,
Көмеш тау эзләмәгез
Мөмкин түгел аның булуы.
Алтыннан кыймәтлерәк,
Көмештән кадерлерәк
Туганнарның бергә булуы.

Җаннарыма тимәгез,
Мине ялгыз димәгез
Якын туганнар бар чагында.
Яшьлекне искә алып,
Матур җырлар җырлашып
Утыраек әле тагын да.

Туган ягыңнан читтә,
Бәхет кошыңны тапсак
Чит җирләр дә яҡын тоела.
Кайгыларын үз иткән,
Шатлыгыңа сөенгән
Сиңа якын дуслар табыла.

Җаннарыма тимәгез,
Мине ялгыз димәгез
Яҡын туганнар бар чагында.
Яшьлекне искә алып,
Матур җырлар җырлашып
Утыраек әле тагын да.

Алтын тау эзләмәгез,

Алтыннан кыймәтлерәк,

```
