# Оригинал

### Флюр Абдракипов / Эльвира Хаматнурова - Яраттым (икенче вариант)

```
Мин сине яраттым,
Карагач күзләргә.
Мәхәббәткә чумдым,
Чумгандай күлләргә.

Кушымта:
Бар дөнья әйләнә
Тик сине уратып.
Хисләрем бәйләнә
Тик сине яратып.

Мәхәббәт очкыны
Читләргә тимәсен.
Ярату хисләрем
Сүнмәсен, сүнмәсен.

Йөрәгеңә синең
Ут-ялкын өрермен.
Сөюемнең назын
Тик сиңа бирермен.
```
