# Оригинал

### Татар халык сүзләре / Чулпан Зиннәтуллина - Агыйдел буйлары (беренче вариант)

```
Агыйделнең пароходы,
Кычкырта сузып-сузып.
Ургылып аккан сулардай,
Гомерләр бара узып.

Агыйделнең ар ягында,
Каенлыкта элмәлек.
Гомернең узасын белдем,
Тиз узасын белмәдем.

Агыйделнең ар ягында,
Казлар канат төзәтә.
Булма матур, бул бәхетле,
Бәхет матур күрсәтә.

Агыйделдә ялгыз үрдәк,
Югалткандыр баласын.
Баш имәгез кайгыларга,
Табыгыз бер чарасын.
```
