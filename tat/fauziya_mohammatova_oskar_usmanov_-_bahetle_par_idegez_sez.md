# Оригинал

### Фәүзия Мөхәммәтова / Оскар Усманов - Бәхетле пар идегез сез

```
Бәхетле пар идегез сез,
Дусларга көнләшерлек.
Бәхетегез мизгелләрен
Безгә дә өлешерлек.

Янәшә дә без булсак та,
Яши әнкәй моң-зарлап.
Язмаган шул тәкъдиренә
Гомер итәргә парлап.

Балалары ятим калды,
Тол калды әниебез.
Яшьли китте шул дөньядан
Сөекле әтиебез.

Янәшә дә без булсак та,
Яши әнкәй моң-зарлап.
Язмаган шул тәкъдиренә
Гомер итәргә парлап.

Ялгыз гомер кичә әни,
Әтием хакын хаклап,
Тол хатыннар дәрәҗәсең
Үз язмышында аклап.

Янәшә дә без булсак та,
Яши әнкәй моң-зарлап.
Язмаган шул тәкъдиренә
Гомер итәргә парлап.
```
