# Оригинал

### Сәкинә Хәйруллина / Нияз Вәлиев - Соңга калдым (икенче вариант)

```
Озын юл кая илтә?
Анда сине ни көтә?
Уйламадың, ашкындың.
Туктатырга теләдем,
Ишетмәде йөрәгең
Минем хисләр ташкынын.

Ник тыңламадың?
Ник аңламадың?
Аңлата алмадым,
Соңга калдым.

Юлларың ерак ятты,
Хатларым кире кайтты,
Гөлләр таралып көйде.
Бер сорау җанны телде,
Ник язмыш бездән көлде,
Бәхет янәшә иде.

Ник тыңламадың?
Ник аңламадың?
Аңлата алмадым,
Соңга калдым.

Еллар үтте, соң инде,
Сөю кояшы иңде,
Кырыс вакыт хөкеме.
Бары йөрәкне тоеп,
Ник аңлашмадык көеп?
Яши җан да укенеп…

Ник тыңламадың?
Ник аңламадың?
Аңлата алмадым,
Соңга калдым.
```
