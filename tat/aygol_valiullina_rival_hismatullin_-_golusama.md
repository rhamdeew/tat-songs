# Оригинал

### Айгөл Вәлиуллина / Риваль Хисмәтуллин - Гөлүсәмә

```
Яшәгәнмен икән күпме еллар
Барлыгыңны җирдә белмичә,
Кошлар җырын гүя ишетмичә,
Тормыш гүзәллеген күрмичә.

Синең белән шундый рәхәт миңа,
Синең белән шундый күңелле.
Яшьлегемнең якты язларына
Алып очар идем үзеңне.

Мәхәббәтнең кайнар бураннары
Йөрәгемне төрде сөюгә.
Күңелемнең һәрбер кылы саен
Синең җаның җылы мәл генә.

Синең белән шундый рәхәт миңа,
Синең белән шундый күңелле.
Яшьлегемнең якты язларына
Алып очар идем үзеңне.

Синең тында тойдым яратуны,
Җаннарыма бәхет урадың.
Сиңа кадәр яшәмәгәнмен бит,
Мин үзем дә чит-ят булганмын.

Синең белән шундый рәхәт миңа,
Синең белән шундый күңелле.
Яшьлегемнең якты язларына
Алып очар идем үзеңне.
Яшьлегемнең якты язларына
Алып очар идем үзеңне…
Алып очар идем үзеңне…
```
