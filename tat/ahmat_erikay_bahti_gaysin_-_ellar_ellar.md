# Оригинал

### Әхмәт Ерикәй / Бәхти Гайсин - Еллар, еллар

```
Еллар, еллар, искән җилләр сыман,
Аккан сулар сыман узасыз.
Түрегездә йөрдем, үрегезгә мендем,
Инде миңа нәрсә кушасыз?

Җырларым да, соңгы сулышым да
Бүләк сезгә, рәхим итегез.
Тылда түгел — юлда, тормыш дулкынында,
Һөҗүмнәрдә алда йөртегез.
```
