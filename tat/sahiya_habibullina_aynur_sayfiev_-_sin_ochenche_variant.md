# Оригинал

### Сәхия Хәбибуллина / Айнур Сәйфиев - Син (өченче вариант)

```
Син көемдә минем, моңымда,
Шатлыгымда, якты таңымда.
Күңелдәге борчуларны юдың,
Саташканда булдың аңымда,
Саташканда булдың аңымда.

Хыялыма ике канат бирдең,
Ялкын бүләк иттең җырыма.
Син, елмаеп, язмыш җилен өрдең,
Җитәкләдең иҗат юлында.

Тик син генә, назлы йөрәгең,
Якты нурың йөргән юлымда.
Тормышымда тик син куанычым,
Мин бәхетле синең яныңда,
Мин бәхетле синең яныңда.

Хыялыма ике канат бирдең,
Ялкын бүләк иттең җырыма.
Син, елмаеп, язмыш җилен өрдең,
Җитәкләдең иҗат юлында.

Тик син генә сөю ялкынын
Сүндермичә саклый беләсең.
Җылытасың тормыш салкынында,
Дөрләп яшәү дәрте бирәсең,
Дөрләп яшәү дәрте бирәсең.

Хыялыма ике канат бирдең,
Ялкын бүләк иттең җырыма.
Син, елмаеп, язмыш җилен өрдең,
Җитәкләдең иҗат юлында.
```
