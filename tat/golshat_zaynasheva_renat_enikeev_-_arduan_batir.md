# Оригинал

### Гөлшат Зәйнәшева / Ренат Еникеев - Ардуан батыр

```
Батырларның гомере мәңге сүнмәс,
Ил күгендә йолдыз булып балкыр.
Шундый якты йолдызларның берсе —
Ардуан батыр, Ардуан батыр.

Урал итәгендә завод төзеп,
Ил шаулаткан даны һаман артыр.
Нигезләргә бәхет, ташын тезгән
Ардуан батыр, Ардуан батыр.

Халык бәхете өчен көрәшкәннең
Узган юлларында эзе матур.
Гади эшче исемен бөек йөрткән
Ардуан батыр, Ардуан батыр.

Лачыннарга тиңдәш үз улларын
Онытмый халкым, югалтмый хәтер.
Безнен белән ил төзешә бергә
Ардуан батыр, Ардуан батыр!
```
