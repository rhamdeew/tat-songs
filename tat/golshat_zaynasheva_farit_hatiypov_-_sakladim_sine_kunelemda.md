# Оригинал

### Гөлшат Зәйнәшева / Фәрит Хатыйпов - Сакладым сине күңелемдә

```
Сакладым сине күңелемдә,
Сагынуларыма төреп.
Гомергә бергә булсак та,
Туялмам кебек күреп.

Әйтче, иркәм, әллә юкмы
Сине сөяргә хакым?..
Син бит миңа күз керфегем,
Күз алмам кебек якын.

Саклармын сине күңелемдә,
Сатуларыма төреп.
Гомергә бергә булсак та,
Туялмам сине күреп.
```
