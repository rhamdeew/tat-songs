# Оригинал

### Марат Кәбиров / Рәис Ханнанов - Эх, яшьлек еллары

```
Яшьлектә сөйгәннәр,
Сез кайсы илләрдә?!
"Бергенәм минем!.." — дип
Дәшәсез кемнәргә?

Эх, яшьлек җилбәзәк,
Эх, яшьлек тиле шул…
Без бергә калмадык…
Каласы иде шул…

Вәгъдәләр биргән чак
Сөяргә гомергә,
Истәдер әле ул,
Истәдер сезнен дә?!.

Эх, яшьлек еллары.
Бер генә мизгелгә
Кайтыгыз әле сез,
Кайтыгыз бер безгә.
```
