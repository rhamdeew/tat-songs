# Оригинал

### Мотал Рәмов / Вилюр Мәүлитов - Онытмыйк

```
Артта калсалар да күпме көннәр
Онытылмый шаңлы ул еллар.
Батырларны данлап,
Җиңү җырын җырлап,
Алмашынды ничә буыннар…

Юктыр инде безнең араларда
Яу юлынан үткән ул-кызлар.
Ерак-ераклардан
Кайтмый калганнардан
Истәлеккә балкый йолдызлар.

Саргаешып шиңде әниләр дә
Көтеп ала алмый улларын.
Агылсыннар моңнар,
Онытмыйк, дусларым,
Ватан сугышының корбанын.
```
