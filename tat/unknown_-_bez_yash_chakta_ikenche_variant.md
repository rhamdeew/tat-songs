# Оригинал

### Гөлфия Юнысова / Фәнзил Шәрипов - Без яшь чакта (икенче вариант)

```
Без яшь чакта күк йолдызлы,
Ә таңнар алсу иде.
Төннәр кыска юллар җиңел,
Язлар гел ярсу иде.
Без яшь чакта уйыннар да
Кызык, күңелле иде:
Гармуннар шәп, җырлар моңлы,
Биюләр җилле иде.

Без яшь чакта уйыннар да
Кызык, күңелле иде:
Гармуннар шәп, җырлар моңлы,
Биюләр җилле иде.

Без яшь чакта тимер йомшак
Бакырлар атын иде.
Кайда гына барам димә,
Аралар якын иде.
Без яшь чакта әбиләр — кыз,
Бабайлар егет иде.
Кояшы да кайнарырак,
Яктырак кебек иде.

Без яшь чакта уйыннар да
Кызык, күңелле иде:
Гармуннар шәп, җырлар моңлы,
Биюләр җилле иде.
```
