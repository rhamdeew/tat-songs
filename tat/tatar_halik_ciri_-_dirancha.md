# Оригинал

### Татар  халык җыры - Дыранча

```
Көянтәсе үзәкле лә…
Көянтәсе үзәкле лә,
Чиләкләре бизәкле.
Үтеп киткәч борылып карый,
Шунысы өзә үзәкне.

А-а-а-а-й…
Өй түбәсе дыранча ла,
Давыл чыкса җил ача.

Өстәлләрдә матур көзге…
Өстәлләрдә матур көзге
Күзләрне чагылдыра.
Киткән чакта "сау бул" дигән
Сүзләре сагындыра.

А-а-а-а-й…
Өй түбәсе дыранча ла,
Давыл чыкса җил ача.
```
