# Оригинал

### Гөлшат Зәйнәшева / Сәйяр Хәбибуллин - Яңгырдан соң

```
Яңгырдан соң сызылып киткән
Салават күпере кебек
Сызылдың син күңелемдә,
Төрле төсләргә кереп.

Сызылдың ал, яшел булып,
Дөньям яктырып китте.
Күзләреңнән күзләремә
Җылы нур явып үтте.

Миңа гына ява микән
Синең күздән җылы нур?..
Мөмкин булса, гомер буе
Миңа шулай җылы бул!..
```
