# Оригинал

### Гөлшат Зәйнәшева / Зөфәр Хәйретдинов - Тәрәзәңә ай килер

```
Хатлар көтмә, хат язалмам,
Ятлар кулына керер.
Минем хат булып кичләрен
Тәрәзәңә ай килер.

Күрсәң айның яңа туган,
Сызылып киткән .чакларын.
Ул булыр минем өзелеп,
Сине көткән чакларым.

Тәрәзәңә ай караса,
Уйла — миннән хәбәр ул.
Мин булып иркәләр, назлар
Сине таңга кадәр ул.

Күрсәң айның яңа туган,
Сызылып киткән чакларын.
Ул булыр минем өзелеп,
Сине көткән чакларым.

Хатлар көтмә, хат язалмам,
Ятлар кулына керер.
Минем хат булып кичләрен
Тәрәзәңә ай килер.

Күрсәң айның яңа туган,
Сызылып киткән чакларын.
Ул булыр минем өзелеп,
Сине көткән чакларым.
```
