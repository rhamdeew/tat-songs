# Оригинал

### Әлфирә Гайнетдинова / Оскар Усманов - Ялгышларны ничек төзәтим?

```
Яшьлегемнең бөреләнгән гөле
Чәчәк атмый калды ялгышып.
Салкыннарга түзми бөреләре
Ап-ак карга калды ябышып.

Ялгышларны ничек төзәтим?
Язмышларны ничек үзгәртим?
Онытырга барын мөмкин түгел,
Уйларымны әйтче,нишләтим?
Ялгышларны ничек төзәтим,
Язмышларны ничек үзгәртим?
Онытырга сине мөмкин түгел,
Уйларымны әйтче,нишләтим?

Чәчәк бөреләрен җылытырга
Бер назлы сүз җитте ул чакта.
Син әйтмәдең,мин дә кыенсындым,
Сыенырга синең кочакка.

Ялгышларны ничек төзәтим?
Язмышларны ничек үзгәртим?
Онытырга барын мөмкин түгел,
Уйларымны әйтче,нишләтим?
Ялгышларны ничек төзәтим,
Язмышларны ничек үзгәртим?
Онытырга сине мөмкин түгел,
Уйларымны әйтче,нишләтим?

Язмыш мине синнән аерды да,
Яралары калды йөрәктә.
Кар өстендә калган яшьлек гөлем
Төшләремә кереп йөдәтә.

Ялгышларны ничек төзәтим?
Язмышларны ничек үзгәртим?
Онытырга барын мөмкин түгел,
Уйларымны әйтче,нишләтим?
Ялгышларны ничек төзәтим,
Язмышларны ничек үзгәртим?
Онытырга сине мөмкин түгел,
Уйларымны әйтче,нишләтим?
```
