# Оригинал

### Мәхмүт Хөсәен / Рәгъде Халитов - Безнең Балтач яклары

```
Балтач, Балтач! дип балкый
Туган җир йолдызлары.
Ярышларда сынатмый
Безнең Балтач кызлары.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.

Балтач, Балтач! дип дәшә
Зәңгәр урман юллары.
Илгә байлык өләшә
Безнең Балтач уллары.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.

Балтач, Балтач! дип дәшә
Шау чәчәкләр болында.
Безнең күңел гел уйлый
Ил бәхете турында.

Саф һавалы, ак таңлы
Безнең Казан артлары.
Ил эшендә мактаулы
Безнең Балтач яклары.
```
