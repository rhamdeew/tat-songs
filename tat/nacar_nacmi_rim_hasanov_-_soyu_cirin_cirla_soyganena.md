# Оригинал

### Наҗар Нәҗми / Рим Хәсәнов - Сөю җырын җырла сөйгәнеңә

```
Сөю җырын җырла сөйгәнеңә,
Күңелләрең ихлас, саф икән.
Ул сөймәсә, әйдә, сөймәсен дә,
Сөюләрең әгәр хак икән.
Сөю җырын җырла сөйгәнеңә.

Сөю җырын җырла сөйгәнеңә,
Сөймәсә дә, җырдан җибәрмә.
Алып китсен җырың күңелләрне
Гүзәл җирдән гүзәл җирләргә,
Сөю җырын җырла сөйгәнеңә.

Сөю җырын җырла сөйгәнеңә,
Мин бәхетсез, димә, бер уйлап.
Башка берәү сине тапмый калмас,
Бәхет килми калмас җыр буйлап,
Сөю җырын җырла сөйгәнеңә.
```
