# Оригинал

### Марат Кәримов / Рафаэль Шәхрәзиев - Яратам (бишенче вариант)

```
Очраттым мин сине,
Тормышым юлымда.
Ә (тик) синең язмышың,
Башкалар кулында.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.

Булыр дип куркасың,
Гайбәтме, бәләме.
Сөюүне кешедән,
Яшереп буламы.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.

Тоелган сөюгә
Ни генә янамый.
Яратам дисәң син
Ярамый, ярамый.

Ишетсен бар дөнья,
Дөньяны яңгратам.
Мин сине яратам,
Яратам, яратам.
```
