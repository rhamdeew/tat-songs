# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Мин бит дөньяда яшәргә өйрәнәм

```
Өйрәнергә дус булып,
Яшәргә кеше белән
Кирәк миңа. Тик бүген
Сугыштым энем белән…
Нәниләрне якларга
Кирәклеген дә беләм,
Кайчак кагыйдәләрне
Онотам берәм-берәм…

Гафу итегез мине
Кыерсытсам энемне, —
Мин бит дөньяда
Яшәргә әйрәнәм.
Эшләсәм дә начарлык,
Бик тиз таныйм хатамны,
Мин бит дөньяда
Яшәргә өйрәнәм.

Өйрәнергә тиешмен
Хөрмәтлергә өлкәнне,
Миннән алда булганны,
Атам-бабам күргәнне.
Булышырга тиешмен
Әбиемә су илтеп,
Тик бүген онотканмын
Көн буе футбол тибеп…

Гафу итегез мине
Кыерсытсам энемне, —
Мин бит дөньяда
Яшәргә әйрәнәм.
Эшләсәм дә начарлык,
Бик тиз таныйм хатамны,
Мин бит дөньяда
Яшәргә өйрәнәм.

Өйрәнергә бар эшкә
Тиешмен бәләкәйдән,
Балта, пычкы тотарга
Әткәйдән, зүрәткәйдән.
Тиеш алып кайтырга
Мин каз-үрдәкне судан.
Коенып су буада
Онотканмын бит, туган…

Гафу итегез мине
Кыерсытсам энемне, —
Мин бит дөньяда
Яшәргә әйрәнәм.
Эшләсәм дә начарлык,
Бик тиз таныйм хатамны,
Мин бит дөньяда
Яшәргә өйрәнәм.
```
