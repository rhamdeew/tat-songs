# Оригинал

### Кәбир Акбашев / Роза Сәхәүетдинова - Югалма юлларымнан, югалма

```
Кояш белән ай язмышлы булдык,
Очрашмабыз, диеп ут салма.
Язмышлардан син бит көчлерәксең —
Югалма юлларымнан, югалма.

Язмышымда чулпан йолдыз бул син,
Тик сагышлар булып кагылма.
Синсез адашырмын кебекмен мин, —
Югалма юлларымнан, югалма.

Эңерләрдә чишмәләргә төшеп,
Җырсыз гына килеп су алма.
Мин ай булып сиңа түгелермен, —
Югалма юлларымнан, югалма.

Сиңа карап бәхетемә барам,
Синең белән дөнъям ягымлы.
Син яшәгәч — яшиселәр килә, —
Югалма юлларымнан, югалма.
```
