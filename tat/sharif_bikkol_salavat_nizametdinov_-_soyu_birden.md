# Оригинал

### Шәриф Биккол / Салават Низаметдинов - Сөю бирдең

```
Сөю бирдең һәм мин шашып,
Кояшлы җырлар яздым.
Күкләргә шатлык өләшеп,
Гел җырлап торды сазым.

Шатлык — бәхет кенә түгел,
Сагыш та биреп киттең.
Сөеп ярты җырчы итсәң,
Калдырып бер бөтен иттең.

Җилгә бөгелгән имәндәй,
Ыңгыраша бу кыллар.

Шатлык — бәхет кенә түгел,
Сагыш та биреп киттең.
Сөеп ярты җырчы итсәң,
Калдырып бер бөтен иттең.
```
