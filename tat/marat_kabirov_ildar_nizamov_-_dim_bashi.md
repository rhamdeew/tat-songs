# Оригинал

### Марат Кәбиров / Илдар Низамов - Дим башы

```
Ярларына сыймас елга түгел,
Чишмә генә икән Дим башы.
Дим суының якты чишмәләре —
Бер-берсенең тугъры юлдашы.

Дим башының якты чишмәләре
Безгә шуны әйтә шикелле:
“Гомерләрең якты узсын дисәң,
Ялгыз итмә якын кешеңне”

Ә бит без дә аккан сулар кебек,
Гомер буйларына юлдабыз.
Аерылсак — нәни инеш кенә,
Бергә чакта гына елга без.

Аккан сулар кебек узган чакта
Гомер дигән авыр юллардан.
Шул чишмәләр кебек бергә булсак,
Югалмабыз әле, туганнар.
```
