# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Күңелле балачак җырлары

```
Уйныйм яшел болонда,
Ак чәчәкләр кулымда,
Әти-әни янымда, —
Җырым алар турында.

Моңнарын басуларның,
Чишмәләрнең суларын,
Чәчәкле болоннарның
Чабышкан колыннарын
Куштым шушы җырыма, —
Калсын йөрәк түрендә
Күңелле балачак җырлары.

Назлы куен — балачак,
Шаян ил син — балачак!
Бәхет көең, балачак,
Мәңге истә калачак!

Моңнарын басуларның,
Чишмәләрнең суларын,
Чәчәкле болоннарның
Чабышкан колыннарын
Куштым шушы җырыма, —
Калсын йөрәк түрендә
Күңелле балачак җырлары.

Алда көтә хыялым,
Чынга ашар уйларым!
Янымда бар дусларым,
Күңелдә бар җырларым!

Моңнарын басуларның,
Чишмәләрнең суларын,
Чәчәкле болоннарның
Чабышкан колыннарын
Куштым шушы җырыма, —
Калсын йөрәк түрендә
Күңелле балачак җырлары.
```
