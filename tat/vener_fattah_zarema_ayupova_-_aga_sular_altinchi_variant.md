# Оригинал

### Венер Фәттах / Зарема Әюпова - Ага сулар (алтынчы вариант)

```
Ага сулар, ага сулар,
Яры микән бик уңай.
Аккан сулар кире кайтмас,
Яшь гомерләр дә шулай.
Еллар, еллар, әллә туктап,
Бераз ял итәсезме?
Әллә безне тыңламыйча
Яшьлекне сүтәсезме?
Еллар судай тиз акса да,
Йөрәгебез яшь әле.
Яшьлек, яшьлек, үткәннәргә
Җыр булып эндәш әле.
Җыр булып эндәш әле…
```
