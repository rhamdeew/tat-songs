# Оригинал

### Разил Вәлиев / Резеда Ахиярова - Чаба поезд, чаба…

```
Чаба поезд, чаба
Туган якка таба, —
Каршы ала горур имәннәр.
Бозып яшьлек антын,
Туган якка кайта
"Хуш мәңгегә, сау бул!" дигәннәр.

Чаба поезд, чаба
Туган якка таба, —
Түземнәрем узды чамадан.
Авылыма кайтсам,
Дусларымны тапсам,
Яши башлармын күк яңадан.

Чаба поезд, чаба
Туган якка таба, —
Йөрәк ярсый, күңел ашкына.
Әйтерсең лә бүген,
Ак каеннар түгел,
Балачагым чыккан каршыма.
```
