# Оригинал

### Абдулхак Игебаев / Хөсәен Әхмәтов - Әллә бөтенләй оныттың

```
Әллә бөтенләй оныттың,
Туктады хатларың да,
Диңгезгә карап җырлыймын
Сагынганчакларымда.

Җырларымны ишетә дә
Акчарлак йөзеп килә.
Йөрәккәем (үҙәккәем) өзелгәнен
Бары ул гына белә.

Синнән хатлар көтә-көтә
Зарыгып үтә көнем,
Акчарлаклардай (әгәр акчарлач) булмаса,
Ялгызым нишләр идем?!
```
