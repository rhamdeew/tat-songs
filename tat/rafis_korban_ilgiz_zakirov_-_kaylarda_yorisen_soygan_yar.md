# Оригинал

### Рафис Корбан / Илгиз Закиров - Кайларда йөрисең, сөйгән яр

```
Кәккүкләр моңая,
Кәккүкләр еракта,
Кешеләр шикелле алар да.
Гомерләр юрасын,
Бәхетләр сорасын
Кәккүкләр без сөйгән ярларга.

Кайларда йөрисең,
Син хәзер кем белән,
Яшьлектә өзелеп сөйгән яр?
Кәккүкләр юраган
Бу озын юллардан
Гел бергә барырбыз дигән яр.

Кәккүкләр ник моңлы?
Кәккүкләр исәпли
Аерым үтелгән елларны.
Онытмас микәнни
Кәккүкләр без бергә
Откан ул җырларны, моңнарны?!
```
