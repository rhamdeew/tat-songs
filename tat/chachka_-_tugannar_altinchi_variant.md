# Оригинал

### Чәчкә - Туганнар (алтынчы вариант)

```
Синдә — бер кан, миндә — бер кан,
Бер үк кан ага тамырдан.
Син — газиз туганым минем,
Бер нәселдән, бер ырудан.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.

Кайгыларым, шатлыкларым
Синең белән, туганым.
Саклап туганлык хисләрен
Кичик суларын, утларын.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.

Тормыш — матур, дөнья — иркен
Тар булмаганда күңелең.
Синең өчен иртә-кичен
Ачык минем ишегем.

Югалышмагыз, туганнар,
Хәтта читләрдә торганнар.
Дошманлашмагыз, туганнар,
Хәтта күршедә торганнар.
```
