# Оригинал

### Флария Терегулова / Азат Имаев - Ак бураннар (икенче вариант)

```
Ак бураннар буйлап киләм
Ак бураннар эченнән.
Гомерем дә әнә шулай
Күпме бураннар кичергән.

Бик риза мин язмышыма
Бураннар язганына.
Ак бураннар белән барам
Мин якты язларыма.

Бураннарга буйсынам мин,
Буйсындыра да алам.
Торган саен якыная
Бураннар белән арам.

Бик риза мин язмышыма
Бураннар язганына.
Ак бураннар белән барам
Мин якты язларыма.

Кышның утлы бураннары
Ярсып уйный канымда.
Кызу канлы бураннарны
Кичәр идем тагын да.

Бик риза мин язмышыма
Бураннар язганына.
Ак бураннар белән барам
Мин якты язларыма.
```
