# Оригинал

### Әхмәт Ерикәй / Сара Садыйкова - Айлы кич (икенче вариант)

```
Айлы кичне яратам мин,
Кошлар сайрап торганда.
Ай тулганда, җил тынганда,
Син янымда булганда.
Җир йөзе күмелә нурга,
Җырлыйсы килә шунда.

Айлы кичне җанга рәхәт,
Табигать кочагында.
Ике йөрәк бер йөрәктәй
Бергә типкән чагында.
Җир йөзе күмелә нурга,
Җырлыйсы килә шунда.
```
