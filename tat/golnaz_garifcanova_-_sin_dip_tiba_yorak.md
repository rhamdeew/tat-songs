# Оригинал

### Гөлназ Гарифҗанова - Син дип тибә йөрәк

```
Күз карашың бигрәк матур,
Тартып тора күңелне.
Синең белән булулары
Үзе бәхет түгелме?

Син дип тибә минем йөрәк,
Әллә шуны сизәсең.
Серле күзең кысып йөри,
Шаярасың, көләсең.

Мәхаббәтем тылсымлы кош,
Очышта канатлана.
Ничек бар — шулай яратам,
Чарасыз гашыйк сиңа.

Син дип тибә минем йөрәк,
Әллә шуны сизәсең.
Серле күзең кысып йөри,
Шаярасың, көләсең.

Син дип тибә минем йөрәк,
Әллә шуны сизәсең.
Серле күзең кысып йөри,
Шаярасың, көләсең…
```
