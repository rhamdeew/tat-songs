# Оригинал

### Рамил Чурагулов / Фәнир Галимов - Җан ярыңны сөйләп барма

```
Җан сереңне сөйләп барма
Якынга да, читкә дә.
Сереңне белеп алганнар
Бик тиз менә(р) җилкәңә.

Ачык сандыклар булмасын
Йөрәгең парәләре.
Күпчак иң якыннар сала
Иң тирән яраларны,
Йөрәкнең яраларын.

Кулда бармаклар да төрле,
Тигез күрмә һәркемне.
Бар дөньяны кочып булмый,
Ярата бел үзеңне.

Ачык сандыклар булмасын
Йөрәгең парәләре.
Күпчак иң якыннар сала
Иң тирән яраларны,
Йөрәкнең яраларын.

Дус дигән дә, сөйгән яр да
Бетмәсен серең белеп.
Кирәгең калмас һичкемгә…
Укылган китап кебек.

Ачык сандыклар булмасын
Йөрәгең парәләре.
Күпчак иң якыннар сала
Иң тирән яраларны,
Йөрәкнең яраларын.
```
