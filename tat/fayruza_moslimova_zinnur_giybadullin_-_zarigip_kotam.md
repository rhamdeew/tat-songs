# Оригинал

### Фәйрүзә Мөслимова / Зиннур Гыйбадуллин - Зарыгып көтәм

```
Хатларың укыйм да яңадан моңланам,
Бергә чак шатлыгы турында уйланам.
Арада юл озын, очрашу мөмкинме?
Күрешү көннәрен зарыгып көтимме?

Очрашыр көн ерак, син шуны бел әле.
Шулкадәр сагындым, төшемә кер әле!
Шатлыктан яктырып күзләрем ачылсын,
Беразга булса да сагыну басылсын.

Сагышка шатлыкны мәрҗәндәй ялгармын,
Янымда һәрвакыт син бар дип санармын.
Җыр булып түгелер өзелеп сөюем,
Шул җырны ишетеп кайтырсың син үзең.
```
