# Оригинал

### Роберт Миңнуллин / Илгиз Закиров - Соңдыр шул, соңдыр

```
Тилмереп көткән
Яшьлекләр үткән.
Оныткансыңдыр
Син мине күптән.

Оныткансыңдыр…
Соңдыр шул, соңдыр —
Ялгышлар шулдыр,
Язмышлар шулдыр.

Син инде мине
Сагынмыйсыңдыр.
Ул да сагынмый,
Дип уйлыйсыңдыр.

Сагынмыйсыңдыр…
Соңдыр шул, соңдыр —
Ялгышлар шулдыр,
Язмышлар шулдыр.

Яратмыйсыңдыр…
Яратсаң әгәр,
Килгәләр иде
Синнән бер хәбәр.

Яратмыйсыңдыр…
Соңдыр шул, соңдыр —
Ялгышлар шулдыр,
Язмышлар шулдыр.
```
