# Оригинал

### Рәмзия Шәрәфетдинова / Лилия Вахитова - Аклык — сафлык

```
Ап — ак карлар ява нинди рәхат.
Әйтерсең лә җаннар аклана
Шул мизгелләр хәтирәләр булып.
Күңелләрдә һәрчак саклана.

Аклык — сафлык баса җирнең йөзен
аны күреп күзләр камаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.

Йөрәкләргә күчә назлы моңнар
искә алып шушы чакларны.
Хисләремне җырларыма кертәм.
Мактар өчен туган якларны.

Аклык — сафлык баса җирнең йөзен
аны күреп күзләр камаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.

Нинди матур гүзәл тамаша.
Ап — ак карлар ява сихри дөнья
тик рәсемдә күрәм аларны.
Җырлар туа уйламаган җирдән.
Сагынганда ап— ак карларны

Аклык — сафлык баса җирнең йөзен
аны күреп күзләр камаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.
Күбәләкләп ява ап — ак карлар.
Нинди матур гүзәл тамаша.
```
