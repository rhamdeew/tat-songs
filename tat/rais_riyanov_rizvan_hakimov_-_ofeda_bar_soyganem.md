# Оригинал

### Рәис Риянов / Ризван Хәкимов - Өфедә бар сөйгәнем

```
Өфе юллары карурман,
Өфе юллары озын.
Шунда сөйдем ил гүзәлен —
Урал таулары кызын.

Төшеп Йөрүзән буена,
Җиләген кабалмадым.
Өфе юлына чыкмыйча,
Һич тынгы табалмадым.

Зәңгәрле күлмәк кигәнем —
Өфедә бар сөйгәнем.
«Уф!..» дигәннәрне бетерә
Бары Өфе дигәнем.

Өфе кичләрен сихерли
Айның балкып чыгышы…
Җаныңны назлап эретә
Саф мәхәббәт сулышы.
```
