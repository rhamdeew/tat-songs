# Оригинал

### Марат Гобәйдуллин - Хуш, мәктәбем

```
Исемдә әле тәүге кат
Мәктәпкә барган көнем.
Беренче укытучымның
Елмайган көләч йөзе.

Хуш, мәктәбем, инде җитте
Саубуллашу сәгате.
Яңгырый шушы җырымда
Сиңа олы рәхмәтем.

Рәхмәт сезгә, укытучылар,
Безне үз күргән өчен.
Әткәй-әнкәй кебек назлап,
Сөеп үстергән өчен.

Хуш, мәктәбем, инде җитте
Саубуллашу сәгате.
Яңгырый шушы җырымда
Сиңа олы рәхмәтем.

Сабыйлыгым, балачагым
Үтте татлы төш кебек.
Инде без дә таралабыз
Кош балалары кебек.

Хуш, мәктәбем, инде җитте
Саубуллашу сәгате.
Яңгырый шушы җырымда
Сиңа олы рәхмәтем.

Моңсу тавышлы кыңгырау
Яңгырый соңгы кабат.
Олы юлга кузгалабыз
Иңнәрдә тоеп канат.

Хуш, мәктәбем, инде җитте
Саубуллашу сәгате.
Яңгырый шушы җырымда
Сиңа олы рәхмәтем.
```
