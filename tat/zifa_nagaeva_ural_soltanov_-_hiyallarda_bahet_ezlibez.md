# Оригинал

### Зифа Нагаева / Урал Солтанов - Хыялларда бәхет эзлибез

```
Бала чагыбызда,
Хыял атыбызда
Без оча идек
Болыттан алда.
Ак болытлар арты,
Тылсым илләреннән
Могҗизалар көттек
Без хыялларда.

Гомерләребез буе
Тормышларга ашмас
Хыялларда бәхет эзлибез.
Ә бит бәхетебез
Җирдә гади генә
Яшәү икән, шуны белмибез!..

Зәңгәр күкләр иле —
Яшьлек тиле иде.
Ак хыялларда нидер көтәбез,
Йолдыз күзләгәнбез,
Ярлар эзләгәнбез.
Янәшәдә булган,
Тик күрмәгәнбез.

Гомерләребез буе
Тормышларга ашмас
Хыялларда бәхет эзлибез.
Ә бит бәхетебез
Җирдә гади генә
Яшәү икән, шуны белмибез!..

Еллар үткән, алдап,
Алга өмет баглап
Шул хыялдагы яр эзләгәнбез.
Уйлап иртәгәсен,
Ктләчәк киләсен,
Бүгенгесен күрми калып үтәбез…

Гомерләребез буе
Тормышларга ашмас
Хыялларда бәхет эзлибез.
Ә бит бәхетебез
Җирдә гади генә
Яшәү икән, шуны белмибез!..
Ә бит бәхетебез
Җирдә гади генә
Яшәү икән, шуны белмибез!..
```
