# Оригинал

### Тамара Ганиева / Афарим Акчурин - Ашкадар

```
Нинди уйдан күңелең шулай тирән,
Сабыр гына аккан Ашкадар?
Тугаеңның уткүз миләшләрен
Мәрҗән итә микән башкалар?

Таңнарда мин уянам
Сандугач җырларыннан.
Эзләп төштем яшьлегемне
Ашкадар буйларыннан.

Юлдар төшеп, якты хыялларым
Болыннарга гына ашсалар,
Йөзек кашы итеп бер ташыңны
Тагып йөрер идем, Ашкадар.

Таңнарда мин уянам
Сандугач җырларыннан.
Эзләп төштем яшьлегемне
Ашкадар буйларыннан.

Сагынуларың бу кәдәр дә булыр,
Яшьлегемнең иле — Ашкадар.
Тугаеңнан хәзер моңлы итеп
Җырлап үтә микән башкалар?

Таңнарда мин уянам
Сандугач җырларыннан.
Эзләп төштем яшьлегемне
Ашкадар буйларыннан.
```
