# Оригинал

### Рифкать Закиров / Фәрит Хатыйпов - Җырым синең хакта (беренче вариант)

```
Арада ярлар… уртада агым —
Син аргы якта, мин бирге якта.
Шулай булса да минем җырларым
Гел синең хакта, гел синең хакта.

Аерды еллар, аерылды юллар,
Мин юл чатларын ялгыз таптадым;
Ары атладым, бире атладым,
Әммә тапмадым, сине, аппагым.

«Калдың инде, дип, син җиде ятка» —
Шәүләң еракта тирбәлгән чакта.
Юксынуларым, моңсу җырларым
Тик синең хакта, тик синең хакта.
```
