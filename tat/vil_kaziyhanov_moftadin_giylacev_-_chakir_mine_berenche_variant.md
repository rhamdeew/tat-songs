# Оригинал

### Вил Казыйханов / Мөфтәдин Гыйләҗев - Чакыр мине (беренче вариант)

```
Урманнарга барсаң, миңа да әйт,
Юлларыңа гөлләр сибәрмен.
Сиңа булган кайнар хисләремне
Кош телләре белән сөйләрмен.

Болыннарга барсаң, миңа да әйт,
Чәчәк җыеп бүләк итәрмен.
Сөюләрең әгәр ихлас булса,
Гомерем буйларына көтәрмен.

Су буена төшсәң, чакыр мине,
Сулар тирән, йөзмә бер үзең.
Тормыш юлын бергә үтәр өчен
Бары җитә синең бер сүзең.
```
