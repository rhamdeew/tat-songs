# Оригинал

### Марис Нәзиров / Ринат Гобәйдуллин - Синсез калган учак

```
Яшьлек сукмагында икәү яккан
Усак эзе хәтта күмелгән.
Сарылганда кайнар кочагыңа
Язмыш аерыр, тип кем белгән?

Ә мин һаман учак ялкыннарын
Саклык белән генә актарам.
Синең төсең итеп кадерләрлек,
Хәтерләрлек эз дә калмаган.

Актарырга күптән күмелгәнне
Ниләр өнди икән, ни көтәм?
… Аерылганда моңсу гына баккан
Күз карашың китми күңелдән.
```
