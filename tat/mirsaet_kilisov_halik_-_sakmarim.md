# Оригинал

### Мирсәет Кылысов / Халык - Сакмарым

```
Саумы, Сакмарым буйлары,
Каенлы урманнарым, шул.
Саумы, тәүге сөю белән
Атлаган урамнарым.

Бу дөнъяда һәр кем кемнең дә
Үз юлы, үз язмышы, шул.
Нинди көдрәтле икән
Туган якның сагышы.

Туган яклар искә төшсә,
Тулышып китә күңел, шул.
Аны ташлап китү җиңел,
Тик кайтулары авыр.

Саумы, минем туган ягым,
Бәхетле булган чагым, шул
Гомер булса, бер әйләнеп
Кайтырмын сиңа тагын.
```
