# Оригинал

### Рифгать Закиров / Камил Хәсәнов - Чын сөю булмаса…

```
Чалт аяз булса да бүген көн,
Еракта урманнар күгелҗем;
Күгелҗем урманнар артына
Кердең дә күмелдең…

Сизелер-сизелмәс җил бәрә,
Кыр буйлап, күр, рәшә тирбәлә,
Ә йөрәк талпына: син аны
Эзләмә… эзләмә…

Эзләмәс идем дә — уй баса,
Сагынмас идем дә — моң баса;
Берсен дә эшләмәс идем лә
Чын сөю булмаса!..
```
