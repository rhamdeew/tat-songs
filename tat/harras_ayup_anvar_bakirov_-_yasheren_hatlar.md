# Оригинал

### Харрас Әюп / Әнвәр Бакиров - Яшерен хатлар

```
Сагынганда хатлар яздык сезгә —
Вәгъдәләргә төшмәде таплар.
Яшьлек тургайлары булып килер
Гомерегезгә

без язган хатлар,
без язган яшерен хатлар.

Еракларга китеп барырбыз да,
Җиталмаслар аргамак атлар.
Бәхет тамчылары булып тамар
Җаныгызга

без язган хатлар,
без язган яшерен хатлар.

Еллар узар, бәгърегезне телеп,
Искә төшәр безнең бу чаклар.
Югалмыйча, юатырлар һаман
Юксынганда

без язган хатлар,
без язган яшерен хатлар.

Сезнең өчен янган сөю утын
Сүрелдерми гел шулай саклар —
Моңлы җырлар булып калыр сездә
Гомерлеккә

без язган хатлар,
без язган яшерен хатлар…
```
