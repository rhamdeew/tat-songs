# Оригинал

### Әминә Бикчәнтәева / Ренат Салихов - Җилләрең, бураннарың…

```
Шатланып көтә күңелләр
Яңа ел туганнарын,
Ябынсын кырлар, болыннар
Ак мамык юрганнарын.

Кадерле безгә, туган ил,
Җилләрең, бураннарың.

Күңелле карап торырга
Зур йортлар төзелгәнен.
Онытмыйк авыр елларда
Үзәкләр өзелгәнен.

Кадерле безгә, туган ил,
Җилләрең, бураннарың.

Тынычлык — ул кадерле сүз,
Әй, газиз туганнарым,
Хәтерләп үтик батырлар
Дошманны куганнарын.

Күңелгә рәхәт, туган җир,
Җилләрең, бураннарың.
```
