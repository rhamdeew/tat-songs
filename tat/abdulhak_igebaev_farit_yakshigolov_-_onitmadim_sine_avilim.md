# Оригинал

### Абдулхак Игебаев / Фәрит Якшыголов - Онытмадым сине, авылым

```
Күңелемә дәва кебек була
Көндә сине искә алуым,
Нишләр идем әгәр бу дөнъяда
Син булмасаң, туган авылым?!

Хушлашканда, сизмәгәнмен икән
Йөрәгемнең синдә калуын,
Гомер үткән сайын, мин ныгырак
Сагынамын сине, авылым!

Язмышымның күрдем күкрәүен дә,
Күрдем җан тетрәткеч давылын,
Юлым кайда гына юллыкса да,
Онытмадым сине, авылым!

Хушлашканда, сизмәгәнмен икән
Йөрәгемнең синдә калуын,
Гомер үткән сайын, мин ныгырак
Сагынамын сине, авылым!

Юккамы ни сиңа ашкынуым,
Хис кузында, бадрап януым,
Әнкәм элгән бишекләрем синдә,
Әткәм йорты синдә, авылым!

Хушлашканда, сизмәгәнмен икән
Йөрәгемнең синдә калуын,
Гомер үткән сайын, мин ныгырак
Сагынамын сине, авылым!
```
