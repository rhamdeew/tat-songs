# Оригинал

### Гөлфия Шакирова / Раил Өметбаев - Якыннарга

```
Янда булган якыннарың —
Яшәүләрнең асыл мәгънәсе.
Һәрвакытта якыннарның
Шат йөзләрен килә күрәсе.
Тормышларның аяз күген
Бәхетләргә килә лә төрәсе.

Сөйгән ярың, балаларың,
Кирәк сиңа дуслар-туганнарың.
Һәрвакытта якыннарның
Шат йөзләрен килә күрәсе.
Тормышларның аяз күген
Бәхетләргә килә лә төрәсе.

Бергә-бергә көннәр ямьле,
Бер табында ашлар бигрәк тәмле.
Һәрвакытта якыннарның
Шат йөзләрен килә күрәсе.
Тормышларның аяз күген
Бәхетләргә килә лә төрәсе.
```
