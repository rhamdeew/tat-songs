# Оригинал

### Гөлшат Әхмәткужина / Мөфтәдин Гыйләҗев - Сөю килә

```
Ашкына поезд, ашкына,
Каршы килә яшел кыр.
Әй сөюем, олы юлдай,
Мине үзеңә чакыр.

Бәхетем син әле килгән
Гөлчәчәкле кыр сыман.
Йөгерче, йөгер каршыма,
Сабыр итүе озак.

Көтүе ай-һай авыр бит,
Поезд килә дә килә.
Тилмертеп җаным, тилертеп,
Каршыма сөю килә.
```
