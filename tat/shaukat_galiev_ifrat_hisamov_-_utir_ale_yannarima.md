# Оригинал

### Шәүкәт Галиев / Ифрат Хисамов - Утыр әле яннарыма

```
Оныт мәшәкатьләреңне,
Оныт инде беразга,
Сыйпап-сыйпап чәчләремне,
Бер иркәлә, бер назла.

Утыр әле яннарыма,
Ял булсын җаннарыма.

Йөзләреңә төбәп карап,
Килә бер сөенәсем,
Килә бер иркәләнәсем,
Килә бер сөеләсем.

Утыр әле яннарыма,
Ял булсын җаннарыма.

Күзләремә ашыгып кына
Салкын карап үтмә син;
Өстәп тор сөю ялкынын,
Гел өсти тор, бетмәсен.

Утыр әле яннарыма,
Ял булсын җаннарыма.
```
