# Оригинал

### Марс Хафизов / Фәһим Шәехов - Кайтчы, яшьлек

```
Үткән гомер очар кошлар кебек,
Канатларым көчен сизмәдем.
Очар кошлар арасына яшьлек
Синең кайтуларың беләдер.

Кайтчы, яшьлек, очар кошлар булып,
Кайтчы, яшьлек, яшел язларым.
Күкрәк тут(ы)рып кабат җырлар идем
Синдә калган тәүге җырларым.

Тормыш мәшәкате тапкан саен,
Яшьлек үтүләрен сизмәдем.
Көзге яфракларның арасыннан
Яшелләрен һаман эзләдем.

Кайтчы, яшьлек, очар кошлар булып,
Кайтчы, яшьлек, яшел язларым.
Күкрәк тут(ы)рып кабат җырлар идем
Синдә калган тәүге җырларым.

Яшел яфрак очар кошлар сезгә
Яшьлегемне һаман охшатам.
Үткән юллар якты хәтирәсен
Хәзер уйлап, үзем юатам.

Кайтчы, яшьлек, очар кошлар булып,
Кайтчы, яшьлек, яшел язларым.
Күкрәк тут(ы)рып кабат җырлар идем
Синдә калган тәүге җырларым.

Күкрәк тут(ы)рып кабат җырлар идем
Синдә калган тәүге җырларым…
Кайтчы, яшьлек…
Кайтчы, яшьлек…
Яшьлек язлары…
```
