# Оригинал

### Гөлгенә Кәримова / Эльмира Шәймәрданова-Гыйльфанова - Туганнар (бишенче вариант)

```
Бик зур мөлкәтем бар бәхетемә,
Байлык түгел — туплап җыйганнар.
Сездән башка ничек яшәр идем,
Ярый әле сез бар, туганнар.

Чын туганнар бергә сөенгәннәр,
Кайгыларны бергә куганнар.
Чын туганнар булып туган өчен
Рәхмәт сезгә, рәхмәт, туганнар.

Ыруыннан читкә киткәннәрне
Борынгылар барлап җыйганнар.
Әйткән вәсыятен бозмый торыйк,
Аерылмый яшик, туганнар.

Чын туганнар бергә сөенгәннәр,
Кайгыларны бергә куганнар.
Чын туганнар булып туган өчен
Рәхмәт сезгә, рәхмәт, туганнар.

Җан да тарта, кан да тарта безне,
Без түгел ул кадер җуйганнар.
Күмәк чакта гына көч беленә,
Аерылмый яшик, туганнар.

Чын туганнар бергә сөенгәннәр,
Кайгыларны бергә куганнар.
Чын туганнар булып туган өчен
Рәхмәт сезгә, рәхмәт, туганнар.
```
