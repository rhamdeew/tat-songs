# Оригинал

### Марат Кәбиров / Ризван Хәкимов - Ялгыз балан

```
Урман эче… Нәни алан…(Урман уртасында аклан,)
Аландагы ялгыз балан (Акландагы ялгыз балан)
Әнкәемне искә төшерде
Аның да бит язмышлары —
Газаплары, сагышлары
Ачы шушы балан шикелле.

Әнкәемне күргән сыман
Баланнарга карап торам…
Кичерегез безне (мине), баланнар!.
Күреп кайгы, мең җәзалар (газаплар)
Үстергән дә безне алар,
Ә үзләре ятим калганнар (Хәзер инде ялгыз калганнар).

Безне сагынып көтә-көтә.
Күз яшьләрен сөртә-сөртә,
Яшәүләре рәхәт (җиңел) дисеңме?!
Гомер буе ялгыз (көткән) әнкәм,
Кичер, әгәр кичерәлсәң…
Мин үзем дә балан шикелле.

Утлы-ачы (ачы-татлы) күз яшьләре.
Гүя, балан тәлгәшләре,
Керфегемдә туңып калганнар.
Әнкәемне күргән сыман,
Баланнарга карап торам…
Аңлый микән мине баланнар?..
```
