# Оригинал

### Рөстәм Зәкуан / Риваль Хисмәтуллин - Килмә, дисең

```
Бер күрүдә гашыйк булдым,
Нишләтим йөрәгемне?
Ә син мине аңламыйча
Өзәсең үзәгемне.

Тимә дисең, өзмә дисең,
Кагылма йөрәгемә.
Мин башканы сөям дисең,
Ул гына йөрәгемдә.

Әмер бирә алмыйм, бәгърем,
Йөрәккәем тыңламый.
Синең ниләр уйлаганын
Ул һич кенә аңламый.

Тимә дисең, өзмә дисең,
Кагылма йөрәгемә.
Мин башканы сөям дисең,
Ул гына йөрәгемдә.

Аптырата калдың инде
Ике ут арасында.
Килмә-йөрмә дигән саен
Сөюем арта гына.

Тимә дисең, өзмә дисең,
Кагылма йөрәгемә.
Мин башканы сөям дисең,
Ул гына йөрәгемдә.

Бер күрүдә гашыйк булдым,

Өзәсең үзәгемне.

Тимә дисең, өзмә дисең,
Кагылма йөрәгемә.
Мин башканы сөям дисең,
Ул гына йөрәгемдә.
```
