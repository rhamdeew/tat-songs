# Оригинал

### Альбина Мортазина / Рамил Әсхәдулла - Дөрли әле

```
Йөрешүләр,
Күрешүләр.
Айлы кичтә
Үбешүләр
Ерак калган инде димә.
Онытылмый,
Синсез булмый.
Җаным әрни,
Уйлый, уйлый.
Ерак калган инде димә.

Сүнми әле,
Дөрли әле.
Яшәтә шул
Сөю тәме.
Ерак калган инде димә.

Көлешүләр,
Серләшүләр,
Юкка гына
Көнләшүләр
Ерак калган инде димә.
Көннәр үтә,
Күңел көтә,
Хәтер җебен
Сүтә-сүтә.
Онытырмын мине димә!

Сүнми әле,
Дөрли әле.
Яшәтә шул
Сөю тәме.
Ерак калган инде димә.
```
