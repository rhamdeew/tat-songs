# Оригинал

### Мансур Шиһапов / Сафьян Ибраһимов - Җан ярсуы

```
Чишмәлектә язын бик тә моңсу,
Тирәклектә сайрый таң кошы.
Сагынып еларсыңмы су алганда,
Чагылып китсә гәүдәм балкышы?..
Чагылып китсә гәүдәм балкышы…

Ташкын булып аккан хисләреңә
Сагыш алып килсә кар сулар,
Ирнем тәмен искә алырсыңмы,
Җаның сусаганда ярсудан,
Җаның сусаганда ярсудан…

Төшләреңә туй көннәре керсә,
Йөрәгеңдә калсын сер булып,
Язгы таңда сулар сибәрсеңме,
Шытып чыкса җаным гөл булып,
Шытып чыкса җаным гөл булып…

Таллар шавы көз үзәкне өзә,
Ялгыз йөрсәң күлләр буенда…
Кабул итәрсеңме бу җырымны
Җыр җырласам синең турында.
… Бергә чакны искә алырсыңмы,
Җыр җырласам синең турында…
```
