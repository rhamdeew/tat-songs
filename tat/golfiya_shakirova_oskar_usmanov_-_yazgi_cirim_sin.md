# Оригинал

### Гөлфия Шакирова / Оскар Усманов - Язгы җырым син

```
Сиңа әйтер
Матур сүзләремне
Энҗеләрдән тездем, кара әле.
Күңел назын
Салып шул сүзләргә,
Сөйлиселәр минем бар әле….

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң….

Ашыгыплар
Миңа килгән чакта,
Адымнарың килә тыңлыйсым.
Синең эзне
Меңнәр арасыннан
Килә иркәм минем таныйсым…

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң….

Сагыш күлен
Бигрәк тирән диләр,
Төшә күрмә мине калдырып.
Карап синең
Килер юлларыңа,
Көтәм, бәгърем, күзем талдырып…

Сандугачның
Җыры булган кебек,
Минем моңым, язгы җырым син.
Җаным диеп,
Ихлас эндәшүең
Бәхетләргә икән үзе тиң….
```
