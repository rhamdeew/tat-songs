# Оригинал

### Разил Вәлиев / Язнур Гарипов - Нигә шаян күзләрең? (беренче вариант)

```
Нигә син көләсең,
Бәгырьне теләсең?
Югыйсә син генә
Серемне беләсең.
Нигә син көләсең,
Көнләшеп үләрсең —
Кемнәрнең шатлыгын
Урталай бүләсең?

Нигә, нигә, нигә, нигә,
Нигә шаян күзләрең?
Шаян күзләреңне күргәч,
Мин дә көлдем, түзмәдем.
Нигә, нигә, нигә, нигә,
Йолдыз төсле күзләрең?
Син бит минем күгем —
Бәхетемне бүген
Йолдызлардан эзләрмен.

Янәшә йөрисең,
Сагышым күрмисең,
Көнләшү утына
Саласың юри син.
Төшләргә керәсең,
Кояштай көләсең,
Минем дә кушылып
Көләсен беләсең.

Нигә, нигә, нигә, нигә,
Нигә шаян күзләрең?
Шаян күзләреңне күргәч,
Мин дә көлдем, түзмәдем.
Нигә, нигә, нигә, нигә,
Йолдыз төсле күзләрең?
Син бит минем күгем —
Бәхетемне бүген
Йолдызлардан эзләрмен.
```
