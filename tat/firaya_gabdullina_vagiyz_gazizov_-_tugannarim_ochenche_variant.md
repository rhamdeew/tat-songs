# Оригинал

### Фирая Габдуллина / Вәгыйзь Газизов - Туганнарым (өченче вариант)

```
Туганнарым, минем бәгырьләрем,
Хәлләрегез ничек, белешик.
Барчабызның кадерләрен белик,
Туганлашыйк, әйдә, күрешик.

Туганым син, диеп, эндәшик без,
Бердәм булыйк, бергә серләшик.
Яратышып яшик, ачуланмый,
Күгәрченнәр кебек гөрләшик.

Туганнарым, мәрхәмәтле булыйк,
Намус белән яшик гөрләшеп.
Туганым, дип, типсен йөрәгебез,
Ярдәм кулы сузыйк берләшеп.

Туганым син, диеп, эндәшик без,
Бердәм булыйк, бергә серләшик.
Яратышып яшик, ачуланмый,
Күгәрченнәр кебек гөрләшик.

Онытмыйча, туган булып калыйк,
Учагыбыз мәңге сүнмәсен.
Туган җепләр өзеп-өзелмәсен,
Арабызга дошман кермәсен.

Туганым син, диеп, эндәшик без,
Бердәм булыйк, бергә серләшик.
Яратышып яшик, ачуланмый,
Күгәрченнәр кебек гөрләшик.
Туганым син, диеп, эндәшик без,
Бердәм булыйк, бергә серләшик.
Яратышып яшик, ачуланмый,
Күгәрченнәр кебек гөрләшик…
Яратышып яшик, ачуланмый,
Күгәрченнәр кебек гөрләшик…
Күгәрченнәр кебек гөрләшик…
```
