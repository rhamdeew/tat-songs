# Оригинал

### Рөстәм Гыйззәтуллин - Гафү ит мине

```
Кем уйлаган безнең сөю
Шулай итеп үрелер диеп.
Кем уйлаган икебезне
Ике ярда калыр диеп.

Гафү ит мине суык кулларыма
Гафү ит ит мине китә алганга.
Гафү ит мине моңсу җырларыма
Гафү ит мине ялгыз булганга.

Гаепләргә ашыкмагыз
Хәлләремне белмичә.
Кеше хәлен кеше белми
Үз башкаена төшмичә.

Гафү ит мине суык кулларыма
Гафү ит ит мине китә алганга.
Гафү ит мине моңсу җырларыма
Гафү ит мине ялгыз булганга.
```
