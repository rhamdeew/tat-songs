# Оригинал

### Әкрәм Даутов - Мәхрүм итмә сәлам хатыңнан

```
Ерак юллар мине ташласа ла
Шаулы диңгезләрнең артына.
Айгөлем, ай нурым!
Мәхрүм итмә сәлам, хатыңнан.
Яшьлек мәхәббәтем хакына,
Мәхрүм итмә сәлам хатыннан.

Ак күбекле диңгез дулкыннары
Туган ярларыма җитәрме?
Айгөлем, ай нурым!
Туган ярларыма җитәрме?
Акчарлаклар канат җилперме?
Сагыну хатларымны илтерме?

Диңгезченең колачлары — җилкән,
Вәгъдә бирсә, корыч вәгъдәсе.
Айгөлем, ай нурым!
Тәүге мәхәббәтнең вәгъдәсен
Капламасын еллар пәрдәсе.
Яшьлек гөлен яшен бәрмәсен.
```
