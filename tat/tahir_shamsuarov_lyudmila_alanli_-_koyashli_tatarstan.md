# Оригинал

### Таһир Шәмсуаров / Людмила Аланлы - Кояшлы Татарстан

```
Күбәләк кар явар төбәк,
Бураннар бурар урын;
Кышлары елап китсә дә,
Аклыгы калыр урын.

Күкләре гел зәңгәр аның,
Узганы — моңлы дастан.
Гүзәл як минем туган җир —
Кояшлы Татарстан!

Гөрләвекләрне җырлатып
Кояшым мартта көлә.
Яз үзе безнең якларны
Сагынып, эзләп килә.

Күкләре гел зәңгәр аның,
Узганы — моңлы дастан.
Гүзәл як минем туган җир —
Кояшлы Татарстан!
```
