# Оригинал

### Шәүкәт Галиев / Хөснул Вәлиуллин - Шомырт чәчәкләре ак кына

```
Бакчаларга тулган шомырт исе
Тәрәзәдән кергән өйләргә.
Исереп бер шомырт исләреннән,
Исереп бер яшьлек хисләреннән,
Җырлыйк әле шушы көйләргә…

Шомырт чәчәкләре ак кына,
Сөюләрең булса хак кына,.
Сөенешеп йөрер чак кына…

Чагыйк әле кырга икәүләшеп,
Дөнья җитә миңа син булгач;
Ник карыйм ди ятлар күзенә мин,
Ник карыйм ди кеше сүзенә мин,
Икебезнең йөрәк тиң булгач!

Шомырт чәчәкләре ак кына,
Сөюләрең булгач хак кына,
Сөенешеп йөрер чак кына.
```
