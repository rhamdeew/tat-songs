# Оригинал

### Татар халык җыры - Шомыртым

```
Өй артында шомыртым бар,
Яннарында бөрлегән шул,
Яннарында бөрлегән.
Сине сөям, сине көтәм,
Син бит минем бер генәм.

Өй артында шомыртым бар,
Чәчәкләре хуш исле шул,
Чәчәкләре хуш исле.
Мин шул чәчәкләргә карыйм,
Чөнки ул синең төсле.

Өй артында шомыртым бар,
Чәчәккә күмеләчәк шул,
Чәчәккә күмеләчәк.
Чәчәк аткан шомырт кебек
Булсын безнең киләчәк!
```
