# Оригинал

### Сабит Моканов / Латыйф Хәмиди - Казах вальсы

```
Йомшак нурлар сибә-сибә килә май,
Яшел киемнәрен кия киң тугай.
"Сагынып кайттым туып үскән илне" — дип,
Киң тугайда канат кага бер тургай.

Ай-һай, чибәр тургай,
Сайрап җибәр, тургай!
Безнең туган илдә
Шат яшәргә уңай.
Сайра, сайра киң тугайда,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!

Тургай сөя туып үскән якларын,
Туган илнең кырларын, куакларын.
Тургай сайрый мактый-мактый җырында
Игенченең сабан сөргән чакларын

Ай-һай, чибәр тургай,
Сайрап җибәр, тургай!
Безнең туган илдә
Шат яшәргә уңай.
Сайра, сайра киң тугайда,
Тургай, тургай!..
Тургай!
А-а-а-а! А-а-а-а!
```
