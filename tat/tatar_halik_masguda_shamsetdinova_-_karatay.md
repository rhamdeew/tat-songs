# Оригинал

### Татар халык / Мәсгудә Шәмсетдинова - Каратай

```
Кара тайга, кара тайга,
Кара тайга чебен кунган,
Чебен түгел кигәвен,
Кара тайга чебен кунган,
Чебен түгел кигәвен.

Карама, кыз, карама, карама,
Буй кыз, карама, буй кыз, карама,
Карама, карама, карама, карама,
Булмам синең, булмам сөйгәнең.
Карама, карама, карама, карама,
Булмам, булмам синең сөйгәнең.

Кара тайга, кара тайга,
Кара тайга чебен кунган,
Чебен түгел кигәвен,
Кара тайга чебен кунган,
Чебен түгел кигәвен.
```
