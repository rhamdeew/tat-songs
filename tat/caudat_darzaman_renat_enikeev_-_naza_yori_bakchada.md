# Оригинал

### Җәүдәт Дәрзаман / Ренат Еникеев - Наза йөри бакчада

```
Кояш күптəн батса да,
Наза йөри бакчада.
Су сибə чилəк-чилəк,
Авыз эченнəн көйлəп:

«Чəчəклəрем ал булсын,
Бал кортына бал булсын.
Үзем эшлим барын да —
Əниемə ял булсын».

Сузылып яр буена,
Чəчəклəр су коена.
Зур тамчы китсə тиеп,
Куялар биеп-биеп.
```
