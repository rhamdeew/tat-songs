# Оригинал

### Румия Әфләтүнова / Юлай Үзәнбаев - Биеп кал (беренче вариант)

```
Тыпыр-тыпыр бер басмагач,
Бию була мени ул.
Йөрәгемне яндырмагач,
Сөю була мени ул.

Ай — тердә — ритәем дә,
Һай — тердә — ритәем.

Бие әйдә, бие әйдә,
Аякларың талмасын.
Талса талсын аякларың,
Үкенечкә калмасын.

Ай — тердә — ритәем дә,
Һай — тердә — ритәем.

Әйдә әле, әйдә әле,
Әйдә басаек әле.
Уйнап-көлеп, җырлап-биеп,
Күңел ачаек әле.

Ай — тердә — ритәем дә,
Һай — тердә — ритәем.

Алмаларны кәрзиннәргә,
Матурлап тутыралар.
Аяк киемнәрен жәлләп,
Биеми утыралар.

Ай — тердә — ритәем дә,
Һай — тердә — ритәем.
```
