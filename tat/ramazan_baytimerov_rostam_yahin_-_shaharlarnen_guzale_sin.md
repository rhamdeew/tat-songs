# Оригинал

### Рамазан Байтимеров / Рөстәм Яхин - Шәһәрләрнең гүзәле син

```
Синдә чәчәк атты, синдә үтте
Якты язлы яшьлек эзләрем.
Сине сөюемне сөйләр өчен
Җитмәс кебек җылы сүзләрем.

Кушымта:
Казаным, әй Казаным!
Һәрбер эштәзур дан казандын.
Шәһәрләрнең гүзәле син,
Җирнең гуя үзәге син,
Горурлыгы газиз ватаннын.

Туган илдә матур шәһәрләр күп,
Күңелемә алар ят түгел.
Тик берсе дә аның Казан түгел,
Беркайсы да туган як түгел.

Кушымта.

Күпме даһи акыл икләре
Бәйләгәннәр сиңа язмышын.
Йөрәк моңым минем, бәхетем син,
Мәхәббәтем, гомер агышым.

Кушымта.
```
