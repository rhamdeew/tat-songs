# Оригинал

### Уран Кинҗәбулатов / Рәшит Җиһанов - Мин гашыйк булдым

```
Мин гашыйк булдым,
Әнкәй, тиргәмә.
Гашыйк күңелдә
Дөнья тирбәлә.

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге —
Мәхәббәт икән!

Безнең тормышны
Мәхәббәт нурлый.
Гашыйк күңелдә
Сандугач сайрый!

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге —
Мәхәббәт икән!

Яшьлекнең яме —
Мәхәббәт икән.
Безнең бәхеткә
Мәхәббәт — җилкән.

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге —
Мәхәббәт икән!

Атылган йолдыз
Бер балкып сүнә,
Сүнмәскә балкый
Мәхәббәт кенә!

Мәхәббәт көтеп,
Йөрәгем типкән.
Бәхет бишеге —
Мәхәббәт икән!
```
