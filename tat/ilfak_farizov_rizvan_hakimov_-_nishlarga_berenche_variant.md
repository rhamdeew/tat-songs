# Оригинал

### Илфак Фаризов / Ризван Хәкимов - Нишләргә? (беренче вариант)

```
Күзем төште береүгә
Тиң булмаган чибәргә.
Әйтегезче дускайларым
Миңа хәзер нишләргә?

Янып-көеп килдем әле
Сер чишәргә чишмәгә.
Әйтегезче чишмәләрем
Миңа хәзер нишләргә?

Арып-талып тауга менәм,
Сорау бирәм җилләргә.
Әйтегезче җилкәйләрем
Миңа хәзер нишләргә?
```
