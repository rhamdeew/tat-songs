# Оригинал

### Шәриф Биккол / Наил Галиев (Николай Голов) - Озата кил

```
Сайра, сайра, сандугачым,
чиртмә-чиртмә талларында;
хыялыйлар иткән идең
мине яшьлек таңнарымда.

Әле булса,
тыңлый-тыңлый,
моңнарыңа хәйран калам:
чут-чут сайрар минутыңны
мин зарыгып көтеп алам.

Чиртә-чиртә күңелемнең
иң-иң нечкә кылларына,
озата кил
гомеремнең
яшәлмәгән елларына!
```
