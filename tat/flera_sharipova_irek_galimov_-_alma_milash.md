# Оригинал

### Флера Шәрипова / Ирек Галимов - Алма — миләш

```
Янәш үсә алма белән миләш,
Кызыл булсалар да, бер түгел.
Татлы диеп татып, сөюләрнең
Ачысына әрнеде күңел.

Нигә икән сөю язда түгел,
Ачы көздә килеп ачылган?
Мин баллардан баллы алма идем,
Син, миләштәй, булдың ачыдан…

Ак чәчкәгә күмелсә дә алан,
Актан түгел икән күңелләр…
Кызыл булып пешсәләр дә, нигә
Миләшләр соң татлы түгелләр?

Нигә икән сөю язда түгел,
Ачы көздә килеп ачылган?
Мин баллардан баллы алма идем,
Син, миләштәй, булдың ачыдан…

Алма белән миләш бер бакчада,
Ачы-татлы икәү янәшә.
Ходай җанга нигә бәхетләрне
Кайгы белән алмаш өләшә?

Нигә икән сөю язда түгел,
Ачы көздә килеп ачылган?
Мин баллардан баллы алма идем,
Син, миләштәй, булдың ачыдан…
```
