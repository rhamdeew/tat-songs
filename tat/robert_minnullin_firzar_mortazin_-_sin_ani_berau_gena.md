# Оригинал

### Роберт Миңнуллин / Фирзәр Мортазин - Син, әни, берәү генә

```
Безнең хакта гына, әни,
Уйланасыңдыр инде.
Борчыласыңдыр инде син,
Куанасыңдыр инде.

Без диеп шатландың да син,
Без диеп сагышландың.
Бөтен гомереңне, әни,
Син безгә багышладың.

И, көясеңдер инде син,
И, янасыңдыр инде…
Төннәрен дә безне уйлап
Уянасыңдыр инде.

Син — йолдызларның яктысы,
Гөлләрнең хуш ислесе.
Әйбәт әниләр күп лә ул,
Юк шул синең төслесе.

Безгә сиздермичә генә
Елап аласың инде.
Киткән саен безгә бәхет
Юрап каласың инде.

Аз инде, аз, әни, сиңа
Рәхмәтләр әйтү генә.
Син бит, әни, безнең өчен
Бер генә — берәү генә!
```
