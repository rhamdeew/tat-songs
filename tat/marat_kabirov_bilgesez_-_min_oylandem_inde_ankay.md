# Оригинал

### Марат Кәбиров / Билгесез - Мин өйләндем инде, әнкәй

```
“Өйлән, өйлән” — дидең, әнкәй,
“Килен” дә “Килен” дидең.
Картлык көнеңдә киленсез
Һич мөмкин түгел, дидең.

Күрдем дә ал алманың
Алланып балланганын,
Күзләрем алалмадым, әй,
Өйләнми калалмадым.

Өйләндем инде, әнкәй,
Киленең — килен инде.
Сизеп торам, ни кылса да
Үз кызың түгел инде.

Тиргәмә инде, әнкәй.
Сүзләмә инде, әнкәй.
Киленеңнең кире ягын
Эзләмә инде, әнкәй.

Кире якларын эзләмә,
Әйбәт якларын эзлә.
Хәзер бит бөтен гомергә
Бергә яшәргә безгә.

Килен белән килешүе
Җиңел түгелдер, беләм.
Аңа рәхәт дисең мени
Килешү синең белән.
```
