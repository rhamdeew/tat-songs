# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Мин яшимен сөю илләрендә

```
Мин яшимен сөю илләрендә,
Сөю дигән тылсым илендә.
Сөю ялкыннары күңелемдә,
Яратасым килә минем дә.

Мин яшимен сөю илләрендә,
Кирәк булсам — үзең табарсың.
Табарсың да минем белән бергә
Сөю ялкынында янарсың.

Мин яшимен сөю илләрендә,
Сцю дулкыннары җанымда.
Мин үземә урын таба алмыйм,
Дулкынланып ташкан чагымда.

Мин яшимен сөю илләрендә,
Кирәк булсам — үзең табарсың.
Табарсың да минем белән бергә
Сөю ялкынында янарсың.

Мин яшимен сөю илләрендә,
Күзләремдә сөю яктысы.
Алда әле, алда шатлыкларым,
Сөю сәгадәтнең татлысы.

Мин яшимен сөю илләрендә,
Кирәк булсам — үзең табарсың.
Табарсың да минем белән бергә
Сөю ялкынында янарсың.
```
