# Оригинал

### Илсөяр Иксанова / Марсель Иванов - Онытмадым

```
Сарыларга салган каеннарны
Ничә тапкыр кочып үпмәдем.
Онытыл, дип ничә кабат әйттем
Ә син, ә син җаннан һаман китмәдең.

Кара төннең салкын куенында
Ничә еллар сине көткән мин.
Онытмадым, бәгърем, онытмадым,
Мәңге кайтмаска дип киткәнең.

Һәр әйләмгә исемең кушып үргән
Толымнарым күпме сүтмәдем.
Оныттым, дип ничә кабат әйттем,
Ә син йөрәктән һаман китмәдең.

Кара төннең салкын куенында
Ничә еллар сине көткән мин.
Онытмадым, бәгърем, онытмадым,
Мәңге кайтмаска дип киткәнең.
```
