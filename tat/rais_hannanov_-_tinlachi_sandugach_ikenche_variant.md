# Оригинал

### Рәис Ханнанов - Тыңлачы, сандугач (икенче вариант)

```
Күл буе сандугач,
Сайрыйдыр талларда.
Шуңа да яшь күңлем —
Йокламый таңнарда.

Тыңлачы, сандугач,
Үзәкне өзә ич.
Ул минем яшьлекне —
Моңнарга күмә ич.

Ишет син, сөйгән яр,
Күзең ач, йоклама.
Шундый да эчкерсез —
Татлы көй буламы?

Тыңлачы, сандугач,
Үзәкне өзә ич.
Ул минем яшьлекне —
Моңнарга күмә ич.

Моңнарым язларда
Үрелеп назларга,
Былбылга әйләнеп —
Ачылды таңнарда.

Тыңлачы, сандугач,
Үзәкне өзә ич.
Ул минем яшьлекне —
Моңнарга күмә ич.

Күл буе сандугач,
Син бит яшьлегем.
Күңелемә моң салып —
Булдың киләчәгем.

Тыңлачы, сандугач,
Үзәкне өзә ич.
Ул минем яшьлекне —
Моңнарга күмә ич.
```
