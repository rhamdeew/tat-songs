# Оригинал

### Гөлшат Зәйнәшева / Чулпан Зиннәтуллина - Аерылмагыз (беренче вариант)

```
Су буенда ялгыз аккош,
Каерылган канатлары.
Канатларны каера ул
Аерылу газаплары.

Аерылмагыз, аерылмагыз,
Булса да сәбәпләре.
Канатларны каера ул
Аерылу сәгатьләре.

Ят кошны пар итмәс аккош,
Ялгыз канат какса да.
Без дә бергә булалмабыз,
Сулар үргә акса да.

Ялгыз аккош күргән саен
Өзелә бәгырьләрем.
Нигә без бергә чакларның
Белмибез кадерләрен?!

Су буенда ялгыз аккош,
Каерылган канатлары.
Канатларны каера ул
Аерылу газаплары.

Аерылмагыз, аерылмагыз,
Булса да сәбәпләре.
Гомерләрне кыскарта ул
Аерылу сәгатьләре.
```
