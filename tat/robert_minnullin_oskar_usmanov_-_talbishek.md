# Оригинал

### Роберт Миңнуллин / Оскар Усманов - Талбишек

```
Бездә бишек әллә ничек
Талдан-җырдан үрелә.
Җырлы бишек, җылы бишек
Эленә өй түренә.

Ничек җырламасын бишек,
Сөн талларыннан үр дә?
Талбишектә үстем мин дә,
Сизелмимени бер дә?

Җырлый бишек, елый бишек,
Җырлый, елый, моңлана.
Әллә бишек, әллә инде
Бишектә яткан бала.

Ничек җырламасын бишек,
Сөн талларыннан үр дә?
Талбишектә үстем мин дә,
Сизелмимени бер дә?

Сеңгәндер ул талбишеккә
Әнкәмнең моңнары да.
Шулайдыр шул, талбишекнең
Юк шуңа тынганы да.

Ничек җырламасын бишек,
Сөн талларыннан үр дә?
Талбишектә үстем мин дә,
Сизелмимени бер дә?
```
