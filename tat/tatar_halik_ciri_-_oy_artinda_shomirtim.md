# Оригинал

### Татар халык җыры - Өй артында шомыртым

```
Өй артында шомыртым бар,
Яннарында бөрлегән шул,
Яннарында бөрлегән;
Бөрлегәндәй матур булып
Чәчәк ата ил генәм

Өй артында шомыртым бар,
Чәчәкләр — яннарында шул,
Чәчәкләр — яннарында;
Без туганбыз илебезнең
Иң матур таңнарында.

Өй артында шомыртым бар,
Яннарында чияләр шул,
Яннарында чияләр;
Чияләр дә безгә карап:
«Сез бәхетле!» — дияләр.
```
