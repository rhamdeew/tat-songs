# Оригинал

### Алинә Хисамиева / Альфред Якшимбетов - Мин барын онытма

```
Бәгърең дә җаның туңса,
Көннәреңнең төсе уңса,
Галәмдә мин барын онытма.
Яныңа дип йөгеремен,
Хисләреңә түгелермен,
Сагышларга гына юлыкма.

Җылытырмын, юатырмын,
Ихлас янып яратырмын,
Сөеп туймас кешем син генә.
Күңелеңә кояш элеп,
Язмышлардан сорап көтеп,
Мәңге яратучы мин генә.

Йолдызларың сүрелсә дә,
Кара болыт күренсә дә,
Ай яктысын бергә эзләрбез.
Шәмнәреңә ут кабызып,
Өметеңә нур тамызып,
Бер учакта янып көярбез.

Җылытырмын, юатырмын,
Ихлас янып яратырмын,
Сөеп туймас кешем син генә.
Күңелеңә кояш элеп,
Язмышлардан сорап көтеп,
Мәңге яратучы мин генә.

Тетрәндерсә яшәүләрең,
Сызландырса күкрәүләрең,
Мин бар җирдә, җаным — онытма.
Мин чыгармын синең янга,
Төшләремне ялгап таңга,
Яшәүләрдән генә сулыкма.

Җылытырмын, юатырмын,
Ихлас янып яратырмын,
Сөеп туймас кешем син генә.
Күңелеңә кояш элеп,
Язмышлардан сорап көтеп,
Мәңге яратучы мин генә.
Мәңге яратучы мин генә…
```
