# Оригинал

### Миләүшә Имашева - Язлар булып килсәм әгәр

```
Язлар булып килсәм сиңа,
Кояш булып җылытсам сине,
Назлы җилләр булып иркәләсәм —
Яратырсыңмы мине, яратырсыңмы мине?

Күңелемдә көзләр минем,
Назлар көтәм ялгызым.
Зәңгәр күкләрдә парлашып
Оча торган син кошым.

Сулар булып аксам әгәр —
Дулкыным булырсыңмы?
Яраларым сызлаганда
Дәвалап алырсыңмы, дәвалап алырсыңмы?

Мин дулкының булсам әгәр —
Яратырмын ярларыңны.
Сөюгә сусаган чакта
Эчәрмен назларыңны.

Сагыш урманына кереп
Адаштым ялгызлыкта.
Җылынырга учак яктым,
Карап атла шул утка, карап атла шул утка.

Парлылар җырын җырларбыз,
Ишетсен безне галәм.
Безнең тормышны язырга
Шагыйрьләр алсын каләм.
Шагыйрьләр алсын каләм…
```
