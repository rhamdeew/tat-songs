# Оригинал

### Хәния Фәрхи / Рифкать Сәйфетдинов - Җиләк җыйганда (икенче вариант)

```
Каршы алды мине тагын
Җиләкле аланым.
Тик мин берьялгызым,
Син юк шул янымда.
Ялгызлык җыры җанымда.

Тик мин берьялгызым,
Син юк шул янымда.
Ялгызлык җыры җанымда.

Иреннәрдә җиләк суты,
Йөрәкләрдә сөю уты.
Күзләр күздә иде,
Вәгъдәләр бирелде.
Иреннәр иренгә үрелде.

Күзләр күздә иде,
Вәгъдәләр бирелде.
Иреннәр иренгә үрелде.

Мин бит гаепле шул синең алда,
Кадеркәйләреңне белмәдем.
Яшьлек уза бара,
Бар да онытыла,
Тик син генә сүнмәс теләгем.

Мәңгелек бер әрнү булып
Калдың күңелдә.
Мәңгелек бер әрнү булып
Калдың күңелдә.
Үзем җиләк җыям,
Күземнән яшь коям,
Үкенермен инде гөмергә.

Үзем җиләк җыям,
Күземнән яшь коям,
Үкенермен инде гөмергә.
```
