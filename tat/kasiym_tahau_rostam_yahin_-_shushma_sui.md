# Оригинал

### Касыйм Тәхау / Рөстәм Яхин - Шушма суы

```
Шушма суы—бормалы су,
Ярларында — тал-тирәк.
Бер кыз җырлый әрәмәдә —
Ничек түзсен яшь йөрәк.

Былбыл сайрый, тал тирбәлә
Шушма суы буенда.
Алсу йөзем, зәңгәр күзем,
Тик син генә уемда.

Шушма суы дулкынлана
Таң җилләре искәндә.
Бәгыремне өзә җанкай
Судан кайтып килгәндә.

Шушма суы тыныч була
Кояш чыкканда, иртән.
Шушма буенда үскәнгә
Сылу булгансың, иркәм!
```
