# Оригинал

### Рәми Гарипов / Нур Даутов - Бар гомерем

```
Бәхет — һәр чак соңлап килер кунак,
Килеп инә табын таралгач!..
Тик көтәсең, көтеп арыгач та
Бәхет өчен генә яралгач.

"Көтә белгәннәргә соң түгел" дә,
Яшьлек кенә көтеп тормый шул.
Юк шул аның туктар тукталышы
Йә үткәнгә юлын бормый ул.

Кала миңа бары еллар буе
Җырлар белән күңел юату.
Бар гомерем минем — сине эзләү,
Бар гомерем сине югалту…
```
