# Оригинал

### Чулпан Фиргали / Владимир Федоров - Әйдә, елмай

```
Елмайсаң, галәмнәр яктырыр,
Көн болытлы булганда.
Йөрәктә шат хисләр уяныр,
Бары бер елмай гына.
Синең шаян күзләрең
Вулкан сыман ут чәчә.
Ах, харап булдым кебек,
Гыйшыктан йөрәк чәнчә.

Әйдә, әйдә, бер елмай әле,
Сихри нурлар чәчеп бар җиһанга
Сиңа карап, мин елмаермын,
Кинәт килеп чыгып алларыңа.

Тик сиңа карап елмаялар
Кояш, ай һәм йолдызлар.
Күрәсең, тик сиңа сокланып.
Синнән үрнәк алалар.
Мизгелгә елмайсаң да,
Йөрәгем дулкынлана,
Уйларым мине ташлап,
Бары сиңа ашкына.

Әйдә, әйдә, бер елмай әле,
Сихри нурлар чәчеп бар җиһанга
Сиңа карап, мин елмаермын,
Кинәт килеп чыгып алларыңа.
```
