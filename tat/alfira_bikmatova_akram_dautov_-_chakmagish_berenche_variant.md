# Оригинал

### Әлфирә Бикмәтова / Әкрәм Даутов - Чакмагыш (беренче вариант)

```
Гашыйк булдым хәтфә болыныңа,
Диңгез кебек бодай кырыңа.
Кайтам әле, шомырт кара күзле
Җилдәй җитез кызыкай кырына.

Чакмагыш, Чакмагыш,
Уңган кызларың таныш.
Күрми торсам шул кызларны,
Күңелгә тула сагыш.

Киң күңелле кешеләрен сөям,
Сагынамын кара күземне.
Ерак юллар үтеп сиңа кайтам,
Сөю тарта минем күңелне.

Чакмагыш, Чакмагыш,
Тасмадай юлың таныш.
Күңелемдә гел син генә,
Йөрәгемдә гел сагыш.
Шул сагышны таратырга
Кайтам әле, Чакмагыш.
```
