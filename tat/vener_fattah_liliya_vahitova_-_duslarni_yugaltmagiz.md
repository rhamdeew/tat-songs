# Оригинал

### Венер Фәттах / Лилия Вахитова - Дусларны югалтмагыз

```
Югалтмагыз иң якын дусларны,
Дуслар алар якты нурга тиң.
Дуслык үзе йөрәкләрне кушкан,
Җаннарны бер иткән җырга тиң.

Жәлләмәгез иң якын дусларга
Җылы сүзләр, ихлас елмаю.
Сәламнәрне җибәреп торыгыз,
Читтә йөрсен сагыш, моңаю.

Аңлашыгыз, кичерә белегез,
Хаталарга күзне йомыгыз,
Ялгышларны бергә төзәтегез,
Шулчак уңып торыр юлыгыз.

Жәлләмәгез иң якын дусларга
Җылы сүзләр, ихлас елмаю.
Сәламнәрне җибәреп торыгыз,
Читтә йөрсен сагыш, моңаю.

Югалтмагыз иң якын дусларны,
Һәр дусның кадерен белегез.
Кимчелекне күрмәмеш кылыгыз,
Суфый түгел ласа беребез..

Жәлләмәгез иң якын дусларга
Җылы сүзләр, ихлас елмаю.
Сәламнәрне җибәреп торыгыз,
Читтә йөрсен сагыш, моңаю.
```
