# Оригинал

### Ризида Мөхәммәтҗанова - Җиде язмыш

```
Өстәл тулды ризык белән —
Барысы да кайтты бүген.
Җиде бала җыя табын:
—Әнкәй, утыр, үзем, үзем…

Кушымта:
Җиде язмыш — серле сагыш.
Нурлы кояш, туп-тулы ай.
Елмаялар, боегалар
Җыелсалар бергә шулай.

Нурга тулды өйнең эче,
Шатлык бөркә әнкәй күзе.
Җидесен дә назлап сөя.
Ничек булдың безгә түзем?..

Кушымта.

Җиде җимеш татлы, ачы.
Без дә үстек, булдык чыдам.
Берчә яңгыр, берчә буран,
Давылдан соң кояш чыга.

Кушымта.

Җәйгор* чыга, гомер төсле,
Җәйгорның да җиде төсе.
Күкләр якты, нурлы балкыш —
Әнкәй, синең күңел төсле.

Кушымта.

*Җәйгор — салават күпере.
```
