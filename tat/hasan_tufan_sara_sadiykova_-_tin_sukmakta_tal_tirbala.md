# Оригинал

### Хәсән Туфан / Сара Садыйкова - Тын сукмакта тал тирбәлә

```
Тын сукмакта тал тирбәлә:
Кулың талга тиядер.
Йөрәк әйтә: сагынгандыр,
Су буена киләдер…

Таллар тагын талпындылар:
Иеләләр, күрәсең;
Чәчләреңне сыйпап синең
Сөенәләр, күрәсең…

Таллар шулай тирбәлүне
Киләсеңдер син дисәм,
Син түгел лә, син түгел шул,
Син түгел ул, җил икән,
Җил икән…
```
