# Оригинал

### Зөһрә Котлыгилдина / Салават Низаметдинов - Миңа сине онытырга кирәк (икенче вариант)

```
Мине сине онытырга кирәк,
Үткәннәрне кат-кат исләмәскә.
Тик ничек соң, кайдан көч табарга,
Очрашулар кабат эзләмәскә?

Кояшым, йолдызым,
Нигә соң сүнәсең?
Нигә соң сөйгәнем китәсең?
Мәхәббәт нигә соң үтәсең?

Мине сине онытырга кирәк,
Бердән-берем диеп табынмаска.
Тик ничек соң, кайдан көч табарга,
Каршыңны синең сагынмаска?

Кояшым, йолдызым,
Нигә соң сүнәсең?
Нигә соң сөйгәнем китәсең?
Мәхәббәт нигә соң үтәсең?

Миңа сине онытырга кирәк,
Буш хыяллар белән юанмаска.
Тик ничек соң, кайдан көч табарга,
Сине күргән чакта куанмаска?

Кояшым, йолдызым,
Нигә соң сүнәсең?
Нигә соң сөйгәнем китәсең?
Мәхәббәт нигә соң үтәсең?
```
