# Оригинал

### Кадыйр Даян / Хәбибулла Ибраһимов - Зәйтүнә

```
Зәйтүнәкәй, бик матурсың,
Йөзең алсу таң кебек.
Ай-ли, Зәйтүнә,
Йөзең алсу таң кебек.
Кашың карлугач канаты,
Буең зифа тал кебек,
Ай-ли, Зәйтүнә,
Буең зифа тал кебек.

Зәйтүнәкәй, серне әйтәм:
Гашыйк булдым мин сиңа.
Ай-ли, Зәйтүнә,
Гашыйк булдым мин сиңа
Син бәхетем, мәхәббәтем,
Якты йолдыз син миңа,
Ай-ли, Зәйтүнә,
Якты йолдыз син миңа.
```
