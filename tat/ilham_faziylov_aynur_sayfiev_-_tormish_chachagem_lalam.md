# Оригинал

### Илһам Фазыйлов / Айнур Сәйфиев - Тормыш чәчәгем — Ләләм

```
Бакчаларда иң матуры —
Ләлә дигән бер чәчәк.
Сөйгән яр да — Ләләм минем,
Якты булсын киләчәк.

Безнең тормыш ал да гөл,
Гаиләмне бизи чәчәк.
Ләләм белән бергә яшәп,
Яшим әле мин яшьнәп.

Мәхаббәтем бакчасында
Бик күп сайрар гөлләре.
Бер яраткан чәчкәм белән
Үтсә иде көннәрем.

Безнең тормыш ал да гөл,
Гаиләмне бизи чәчәк.
Ләләм белән бергә яшәп,
Яшим әле мин яшьнәп.

Безнең гөлне өздем, чәчәк
Ләлә дип аталганын.
Сайлап алдым шул гөлне мин
Сөйгәнгә күрә аны.

Безнең тормыш ал да гөл,
Гаиләмне бизи чәчәк.
Ләләм белән бергә яшәп,
Яшим әле мин яшьнәп.
```
