# Оригинал

### Кадир Әлибаев / Юлай Моратов - Көнләшәм

```
Яннарыңда кызлар булса,
Кызлардан, кызлардан көнләшәм,
Иренеңә карлар кунса,
Карлардан, карлардан көнләшәм.

Кулларыңда гөлләр булса,
Гөлләрдән, гөлләрдән көнләшәм,
Куеныңа җилләр кунса,
Җилләрдән, җилләрдән көнләшәм.
Көнләшәм.

Су буена төшеп китсәм,
Җирекләрдән көнләшәм.
Егетләргә сүз әндәшсәм,
Егетләрдән көнләшәм.

Кулларыңда гөлләр булса,
Гөлләрдән, гөлләрдән көнләшәм,
Куеныңа җилләр кунса,
Җилләрдән, җилләрдән көнләшәм.
Көнләшәм.

Керфегеңә тамчы тамса,
Тамчылардан көнләшәм,
Җиткән кызны соратсалар
Яучылардан көнләшәм.

Кулларыңда гөлләр булса,
Гөлләрдән, гөлләрдән көнләшәм,
Куеныңа җилләр кунса,
Җилләрдән, җилләрдән көнләшәм.
Көнләшәм.
```
