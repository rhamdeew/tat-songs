# Оригинал

### Энҗе Мөэминова / Ә. Шиһапов - Сүнмәс утка салдың

```
Чагылып кына киттең, күз алдымнан,
Керфегеңне сирпеп кенә алдың.
Тирә-ягым сихри нурга тулды,
Аккан сыман булды бәхет таңым, —
Керфегеңне сирпеп кенә алдың!

Чагылып кына киттең күз алдымнан,
Тоеп кына калдым карашыңны.
Бәхет бүләк иткән мизгелләрең
Тоташ газап белән алмашынды, —
Тоеп кына калдым карашыңны.

Чагылып кына киттең күз алдымнан,
Өздереп бер карап кына алдың.
Мизгелләрдә туган изге хисләр
Еллар буе сүнмәс утка салды, —
Өздереп бер карап кына алдың!
```
