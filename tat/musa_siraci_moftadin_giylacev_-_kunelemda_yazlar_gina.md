# Оригинал

### Муса Сираҗи / Мөфтәдин Гыйләҗев - Күңелемдә язлар гына

```
Синең яктан кояш чыга,
Синең яктан килә нурлар,
Нурлар белән бергә туа
Йөрәгемдә моңлы җырлар.

Синең яктан исә җилләр,
Йомшак җилләр, хушбуй җилләр.
Алар исеп үткән чакта
Күзен ача чәчәк-гөлләр.

Күңелләрдә кышлар булмый,
Күңелләрдә язлар гына.
Син җибәргән җилләрдә мин
Тоям бары назлар гына.
```
