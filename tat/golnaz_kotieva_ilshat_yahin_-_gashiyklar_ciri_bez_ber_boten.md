# Оригинал

### Гөлназ Котыева / Илшат Яхин - Гашыйклар җыры (Без бер бөтен)

```
Син — күктә яшәр йолдыз,
Айдан үзеңне көнләшәм.
Сөюемнән моңнар үреп,
Айдай тулып, мин эндәшәм.

Күктә җаннар талпынды,
Изге хистән яктык учак.
Җаным, синсез яшәү юк.
Җаным, синсез яшнәү юк.
Без бер бөтен, җанкаем!

Син — тиңе Кояш, Айның,
Якты бер йолдыз юлымда.
Бәхет кошым, күз нурым.
Гөлбәзирем, Салаватым (Сөйгән ярым, бердән-берем).

Күктә җаннар талпынды,
Изге хистән яктык учак.
Җаным, синсез яшәү юк.
Җаным, синсез яшнәү юк.
Без бер бөтен, җанкаем!
```
