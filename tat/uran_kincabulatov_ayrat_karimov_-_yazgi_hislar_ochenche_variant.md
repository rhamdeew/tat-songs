# Оригинал

### Уран Кинҗәбулатов / Айрат Кәримов - Язгы хисләр (өченче вариант)

```
Сагыш сары димә,
Гомер буе янып көймә.
Кабат уйный нурлар,
Ага (кайный) ташкын сулар.

Үтте салкын кышлар,
Үтте давыл бураннары,
Безгә гашыйк язлар,
Ярга сыймый дулкыннары.
Агым сулар кире акмый,
Узган гомер кире кайтмый,
Мәхәббәт яңыдан туа,
Күңелгә нур тула.

Гомер гәҗәп елга,
Сөю язын көтеп ага,
Сүнгән учакларда
Яңа утлар яна.

Үтте салкын кышлар,
Үтте давыл бураннары,
Безгә гашыйк язлар,
Ярга сыймый дулкыннары.
Агым сулар кире акмый,
Узган гомер кире кайтмый,
Мәхәббәт яңыдан туа,
Күңелгә нур тула.

Гөлләр сибеп юлга,
Яңа бәхет әйди алга.
Сөю шундый дәръя
Яңа (серле) матур дөнъя.

Үтте салкын кышлар,
Үтте давыл бураннары,
Безгә гашыйк язлар,
Ярга сыймый дулкыннары.
Агым сулар кире акмый,
Узган гомер кире кайтмый,
Мәхәббәт яңыдан туа,
Күңелгә нур тула.
```
