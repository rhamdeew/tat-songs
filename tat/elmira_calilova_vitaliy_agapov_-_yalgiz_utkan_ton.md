# Оригинал

### Эльмира Җәлилова / Виталий Агапов - Ялгыз үткән төн

```
Бүген очрашмадык. Үкенәсеңме,
Ялгыз үткәненә бу төннең?
Мин үкенәм. Син бит якты нуры
Туачак һәр таңның, һәр көннең.

Бүген очрашмадык. Сагындыңмы,
Искә алып күпме юандың?
Мин сагындым. Әмма төштә күреп,
Сабый бала кебек куандым.

Бүген очрашмадык. Тик барыбер
Килдеңме син капка төбенә?
Ай нурлары миңа үткән төннең
Хатирәсе булып күренә.

Бүген очрашмадык. Кайгырма син,
Сагышланма, бары түз генә.
Чыгармын мин синең яннарыңа,
Әзерләп тор җылы сүз генә.
```
