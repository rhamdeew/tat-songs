# Оригинал

### Залия Нуриәхмәтова / Булат Шәрипов - Хыяллар (икенче вариант)

```
Хыялларым Акбүз ат сыман,
Ашкыналар һаман еракка..
Ни эзлиләр алар язмыштан?
Нидер җитми минем йөрәккә..
Хыяллар, хыяллар..
Корасыз оялар,
Сагышлы йөрәгем түренә,
Өметләр уяна,
Күңелләр моңлана,
Йолдызлар сибелә күгемә.
Йолдызлы күкләрдә ай балкый,
Хыяллар яңарта хисләрне.
Каядыр югала сагышлар..
Искәртеп шул үткән кичләрне.
Хыяллар, хыяллар,
Корасыз оялар
Сагышлы йөрәгем түренә.
Өметләр уяна,
Күңелләр нурлана.
Көзләрем яз булып күренә.
Яз җитсә урманнар уяна,
Былбыллар оялый гөлләргә.
Хыяллар дәвалый йөрәкне,
Ашкынып яз килгән көннәрдә.
Хыяллар, хыяллар..
ЗАЛИЯ НУРИӘХМӘТОВА
```
