# Оригинал

### Ренат Харис / Резеда Ахиярова - Казан утлары (беренче вариант)

```
Очраштык Казан-су буйларында,
Талгын су талларны агыза.
Тын кичнең яменә тылсым өстәп,
Казаным утларын кабыза.

Казан утлары,
Казан утлары,
Сез шаһит —
Йөрәкләргә
Кагылды мәхәббәт уклары.

Без генәмени ул Казан-суга
Иң татлы яшьләрен тамызган?!
Һәрвакыт бәхеткә юл күрсәтеп,
Казаным утларын кабызган.

Казан утлары,
Казан утлары,
Сез шаһит —
Йөрәкләргә
Кагылды мәхәббәт уклары.

Без тагын Казан-су буйларында,
Талгын су елларны агыза.
Кемнәрдер киченә тылсым өстәп,
Казаным утларын кабыза.

Казан утлары,
Казан утлары,
Сез шаһит —
Йөрәкләргә
Кагылды мәхәббәт уклары.
```
