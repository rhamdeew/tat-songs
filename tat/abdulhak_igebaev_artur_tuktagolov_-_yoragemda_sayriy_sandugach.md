# Оригинал

### Абдулхак Игебаев / Артур Туктаголов - Йөрәгемдә сайрый сандугач

```
Купшылланып гөрләп чәчәк ата
Син утырткан зифа алмагач.
Алмагачны кочаклардай булып
Йөрәгемдә сайрый сандугач.

Аерылышу сагышларын йотып,
Сине һуштан язып сагынгач,
Уфтанма, дим назлап юаткандай
Йөрәгемдә сайрый сандугач.

Хозурланып таңгы офыкларда
Син яраткан йолдыз кабынгач,
Йолдыз нурларында талпынгандай,
Йөрәгемдә сайрый сандугач.

Аерылышу сагышларын йотып,
Сине һуштан язып сагынгач,
Уфтанма, дим назлап юаткандай
Йөрәгемдә сайрый сандугач.
```
