# Оригинал

### Ренат Галиәскәр / Илдус Дәүләнов - Карандашым-китабым

```
Бер дәрестәй узып китте
Унбер ел вакыт.
Коридорны тәнәфестә
Үткәндәй чабып.
Ашыкканбыз булып чыкты
Без бер дә юкка.
Төбәлеп карап торабыз
Чиксез офыкка

Карандашым-китабым,
Узды-китте елларым.
Дәфтәрем һәм пеналым,
Сау булыгыз, дусларым.

Зур тормышларга старт бирә
Соңгы кыңгырау.
Төшкә генә керер инде
Дәресләр тозлау.
Тормыш формулаларын да
Сөендердек әйбәт.
Канат куйган укытучым,
Сезгә мең рәхмәт!

Карандашым-китабым,
Узды-китте елларым.
Дәфтәрем һәм пеналарым,
Сау булыгыз, дусларым.

Олы тормыш, зур калалар,
Шаулый тулай торак.
Хәрби хезмәт, бич-пакетлар,
Утырмабыз елап.
Зачеткалар, тәүге сөю,
Йокысыз таң аттыру.
Нинди һөнәр сайласак та,
Максатчан кеше булу.

Карандашым-китабым,
Узды-китте елларым.
Дәфтәрем һәм пеналым,
Сау булыгыз, дусларым.
```
