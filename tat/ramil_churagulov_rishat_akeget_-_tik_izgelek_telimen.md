# Оригинал

### Рамил Чурагулов / Ришат Акъегет - Тик изгелек телимен

```
Күңелемдә сер йөретеп,
Үчләр саклый белмимен.
Бар кешеләр миңа әйбәт,
Начарларын күрмимен.

Яратам мин бу тормышның
Җәйләрен дә, кышын да.
Күңелемдә һичбер кер юк,
Эчемдәге — тышымда!

Тормыш тора ак-карадан,
Пакълыктан, пычырактан,
Яхшылыктан, ямалыктан,
Саваплардан, гөнаһтан.

Яратам мин бу тормышның
Җәйләрен дә, кышын да.
Күңелемдә һичбер кер юк,
Эчемдәге — тышымда!

Йөрәгемдә кер йөретеп,
Үчләр саклый белмимен.
Дусларга һәм дошманнарга
Тик изгелек телимен.

Яратам мин бу тормышның
Җәйләрен дә, кышын да.
Күңелемдә һичбер кер юк,
Эчемдәге — тышымда!
```
