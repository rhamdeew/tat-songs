# Оригинал

### Флария Терегулова / Азат Имаев - Сабырлык (икенче вариант)

```
Ходай Тәгаләдән сорыйм
Иң авыр чакларымда.
«Бирче, Раббым, Сабырлык», дим
Балалар хакларына,

Бисмилләсез юлга чыкмыйм,
Күңелдә гел изгелек.
Дөньялар тыныч торсын, дип,
Барчабыз теләк телик.

Догалар көйләп атлыймын
Тормыш сукмакларыннан.
И, Ходаем үзең сакла
Юньсез шайтаннарыңнан.

Бисмилләсез юлга чыкмыйм,
Күңелдә гел изгелек.
Дөньялар тыныч торсын, дип,
Барчабыз теләк телик.

Тынгы табармын күңелгә
Мәрхәмәтлек эшләсәм.
Эреләнмә, гафу үтен
Бер начарлык эшләсәң.

Раббым мине гафу ит, дим,
Ялгышкан чакларымда.
Бирче Ходай сабырлык дим,
Сөюем хакларына,
Балалар хакларына…

Бисмилләсез юлга чыкмыйм,
Күңелдә гел изгелек.
Дөньялар тыныч торсын, дип,
Барчабыз теләк телик.
```
