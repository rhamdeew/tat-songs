# Оригинал

### Илсөяр Иксанова / Салават Низаметдинов - Сагын мине

```
Буран булып сырыласым килә,
Яңгыр булып чылатасым килә,
Моңландырып елатасым килә,
Сагынасым килә, сагынасым килә.

Сагын мине һәм сагындыр,
Сине өзлеп сагынасым килә.

Яфрак шауы, яфрак шауы булып,
Йөрегеңә кагыласым килә.
Сөю моңы булып күкрәгемдә
Сагынасым килә, сагынасым килә.

Сагын мине һәм сагындыр,
Сине өзлеп сагынасым килә.

Юлларымда йөгерәсем килә,
Язларыңа өлгерәсем килә,
Күзләремдә сагыласым килә,
Сагынасым килә, сагынасым килә.

Сагын мине һәм сагындыр,
Сине өзлеп сагынасым килә.
```
