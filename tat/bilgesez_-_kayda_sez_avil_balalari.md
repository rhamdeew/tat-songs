# Оригинал

### Билгесез - Кайда сез, авыл балалары?

```
Элек монда олы авыл иде,
Балкып яна иде утлары.
Калган бүген истәлекләр булып
Нигезләре генә нибары.

Кайларда сез, авыл балалары?
Таралдыгыз нигә узышып?
Бәхетләрне табалдыгыз микән,
Туган нигезләрне онытып?

Күрешмисез, бер дә күрешмисез,
Әрәмәләр басты юлларны.
Сездән калган нигезләрне жәлләп,
Җилләр кылып йөри доганы.

Кайларда сез, авыл балалары?
Таралдыгыз нигә узышып?
Бәхетләрне табалдыгыз микән,
Туган нигезләрне онытып?

Күрешмисез, бер дә күрешмисез,
Түгел дә бит ерак аралар.
Үкенергә туры килмәс микән,
Рәнҗемәсме ата-бабалар?

Кайларда сез, авыл балалары?
Таралдыгыз нигә узышып?
Бәхетләрне табалдыгыз микән,
Туган нигезләрне онытып?
```
