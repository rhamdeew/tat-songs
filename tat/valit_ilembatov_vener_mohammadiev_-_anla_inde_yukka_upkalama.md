# Оригинал

### Вәлит Илембәтов / Венер Мөхәммәдиев - Аңла инде (Юкка үпкәләмә)

```
Күңелкәем канатланды
Сине тәү күргән чакта;
Хәзер яна йөрәккәем
Уйлаcам синең хакта.

Юкка үпкәләгән булып,
Ятларга китәсең бит;
Аны, мине, үзеңне дә
Бәхетсез итәсең бит.

Оныт, дисең, араларны
Ничек өзәем икән;
Аерылу газабына
Ничек түзәем икән?

Юкка үпкәләгән булып,
Ятларга китәсең бит;
Аны, мине, үзеңне дә
Бәхетсез итәсең бит.

Хәтерләсәң яшьлегеңне,
Күзләреңә яшь тулыр;
Эх, диеп бер үкенерсең —
Гомерләр үткән булыр…

Юкка үпкәләгән булып,
Ятларга китәсең бит;
Аны, мине, үзеңне дә
Бәхетсез итәсең бит.
```
