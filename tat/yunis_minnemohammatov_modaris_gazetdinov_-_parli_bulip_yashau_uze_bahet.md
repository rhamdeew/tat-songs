# Оригинал

### Юныс Миңнемөхәммәтов / Мөдәрис Газетдинов - Парлы булып яшәү үзе бәхет

```
Яратасым килә шашып, ярсып,
Мәхәббәтне көтә күңелләр.
Зәңгәр күлдә йөзгән аккошлар да
Минем кебек ялгыз түгелләр.

Парлы булып яшәү үзе бәхет,
Шул бәхетне килә тоясым.
Кошлар да бит үз ишләрен таба,
Корыр өчен бергә оясын.

Сөйгән ярың белән гаилә корып,
Сабыйларга биреп гомерләр,
Алар уйнаганны карап торып,
Шатлансыннар иде күңелләр.

Парлы булып яшәү үзе бәхет,
Шул бәхетне килә тоясым.
Кошлар да бит үз ишләрен таба,
Корыр өчен бергә оясын.
Парлы булып яшәү…
Парлы булып яшәү үзе бәхет,
Шул бәхетне килә тоясым.
Кошлар да бит үз ишләрен таба,
Корыр өчен бергә оясын.
Корыр өчен бергә оясын…
Парлы булып яшәү үзе бәхет…
```
