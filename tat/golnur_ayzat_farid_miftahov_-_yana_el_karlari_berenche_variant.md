# Оригинал

### Гөлнур Айзәт / Фәрид Мифтахов - Яңа ел карлары (беренче вариант)

```
Бер көнгә булса да кайтыйк
Балачак өйләренә.
Яңа ел суык булса да
Күп еллар иңнәрендә.

Яңа ел карлары кебек
Килеп тора очасы,
Яшьлектә калган ярларны,
Эх, бер генә кочасы.

Хыяллар сыя (а)лмый күңелгә,
Дәртләнеп тибә йөрәк.
Кабул була Яңа елда
Соңлап теләгән теләк.

Тик яңа яуган карлардан
Икәү узасы иде.
Яшьлектә калган сөюгә
Күпер сузасы иде.

Йөрәгемнең тибешләрен
Саный сәгать телләре.
Яңа ел кавыштыра,
Табыштыра кемнәрне.

Тик яңа яуган карлардан
Икәү узасы иде.
Яшьлектә калган сөюгә
Күпер сузасы иде.
```
