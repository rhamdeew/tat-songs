# Оригинал

### Элина Балагутдинова / Шахзат Размәтов - Бәхет сере

```
Бәхет ачкычларын табу —
Тормышның зур мәгънәсе.
Ачкыч кемдә, шулдыр инде —
Мәхаббәтнең иясе.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары…

Ике ягыңда фәрештә —
Әти-әниең булса.
Дус-туганнар янда булып,
Ярдәм кулларын сузса.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары…

Бәхеттер ул хәтта, дустым,
Бу дөньяда яшәүләр.
"Мин яратам", "Мин бәхетле"
Диеп көн дә дәшүләр.

Өзелеп яратулары,
Сөеп җан атулары,
Шулдыр ул бу дөньяларда
Бәхетле булулары…
```
