# Оригинал

### Равил Галиәкбәров / Венер Мөхәммәдиев - Таң җыры (икенче вариант)

```
Күтәрелә томан
Бишек җыры сыман,
Агыла моң таңнар атканда.
Тыңлыйм таңның җырын,
Туган якның моңын,
Чыңлап чишмәләре акканда.

Балачагым иле,
Ямьле инеш күле,
Җиләкле аланым хуш исе.
Тугайларың гизеп,
Суларыңда йөзеп,
Эх, синдә генә яшисе!

Ак дулкыннар уйный,
Күңел сөенеп туймый,
Тәүге нурлар күлгә сибелә.
Талпындырдың ничек,
Түзим, яшел бишек,
Тагын синдә, синдә тибрәләм.

Тугайларың гизеп,
Суларыңда йөзеп,
Эх, синдә генә яшисе,
Эх, синдә генә яшисе,
Эх, синдә генә яшисе!
```
