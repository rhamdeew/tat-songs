# Оригинал

### Башкорт халык бию такмагы - Өфе — Чиләбе (икенче вариант)

```
Бас, бас эзенә,
Күз тимәсен үзенә.

Өфе, Чиләбе,
Пешкән кура җиләге.

Егетлек билгәсе чыккан
Аның матур йөзенә.

Өфе, Чиләбе,
Янмый микән йөрәге?

Бас, бас, бас әле,
Тагы да басырбыз әле.

Өфе, Чиләбе,
Пешкән кура җиләге.

Табын корып, халык җыеп,
Туйлар ясарбыз әле!

Өфе, Чиләбе,
Кабул булсын теләге!

Әйттер әле, малай,
Сиптер әле, малай.

Өфе, Чиләбе,
Пешкән кура җиләге.

Уйна гармун, тарт кураен,
Биеп алаек, әле!

Өфе, Чиләбе,
Янмый микән йөрәге?

Син дә баса беләсең,
Мин дә басам, күрәсең.

Өфе, Чиләбе,
Пешкән кура җиләге.

Тыпырдатып басып килеп
Үзәгемне өзәсең.

Өфе, Чиләбе,
Янмый микән йөрәге?

Бие, бие, гүзәлем,
Күрми ничек түзәрем.

Өфе, Чиләбе,
Пешкән кура җиләге.

Тыпыр-тыпыр биеп йөрегән
Ул бит минем чибәрем.

Өфе, Чиләбе,
Ничек икән теләге.
```
