# Оригинал

### Гөлшат Зәйнәшева / Ганс Сәйфуллин - Баулы (икенче вариант)

```
Ык буенда кошлар сайрап,
Уята алсу таңны,
Алсу таң белән уяна,
Шуңа ямьледер Баулы.

Ык сулары бормалы,
Баулы юллары таулы.
Тау астында бакча кебек,
Гөлбакча кебек Баулы.

Нефть безнең ил байлыгы,
Нефть ул кара алтын.
Кара алтын бар дөньяга
Танытты Баулы халкын.

Ык сулары бормалы,
Баулы юллары таулы.
Тау астында бакча кебек,
Гөлбакча кебек Баулы.

Ык суы туган җир буйлап
Ерактан агып килә.
Баулыда туып-үскәннәр
Дуслыкның кадрен белә.

Ык сулары бормалы,
Баулы юллары таулы.
Тау астында бакча кебек,
Гөлбакча кебек Баулы.
```
