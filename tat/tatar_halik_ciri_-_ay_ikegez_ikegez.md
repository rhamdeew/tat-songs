# Оригинал

### Татар халык җыры - Әй, икегез, икегез

```
Әй, икегез, икегез,
Икегез дә бертигез;
Икегез дә пар килгәнсез,
Тигез гомер итегез.

Самавырны тиз кайната
Кызыл миләш күмере;
Килен белән яшь кияүнең
Озын булсын гомере.

Сокланып карап туймассың
Зифа буегызга да.
Рәхмәт, безне чакыргансыз
Матур туегызга да.

Агыйделгә сал килгән,
Сал килгән җир тар килгән.
Сайлап алган алмалар күк,
Икегез дә пар килгән.

Сандугачлар ояда,
Оялары пыяла;
Сандугач кебек сайрашып,
Шат яшәгез дөньяда!..
```
