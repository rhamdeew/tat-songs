# Оригинал

### Рәми Гарипов / Башкорт халык - Чыңрау торна

```
Чыңрый-чыңрый очып килә
Чыңрау торнакай.
Уралыңнан сәлам диде
Чыңрау торнакай.

Уралкаем, Уралкаем,
Минем туган Уралкаем!
Исән-сау, ди, Урал-тау, ди
Чыңрау торнакай.

Юк хафа, ди, юк афәт, ди
Чыңрау торнакай.
Илдә ир-ат — Салават, ди
Чыңрау торнакай.

Уралкаем, Уралкаем,
Минем туган Уралкаем!
Ил көйләр ди, тел сөйләр ди,
Уйнар курайкай.

Куанычым, юанычым
Булды Уралкай.
Җыр канатлы, нур канатлы
Чыңрау торнамдай.

Уралкаем, Уралкаем
Минем туган Уралкаем!
Буатлардан буатларга
Очып баргандай.
```
