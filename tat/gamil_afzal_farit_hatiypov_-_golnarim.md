# Оригинал

### Гамил Афзал / Фәрит Хатыйпов - Гөлнарым

```
Сандугачтай сайрап уйнадың да
Кына гөлләренә кунмадың.
Гөлнарым,
Үкенечкә ярым булмадың.

Яулыгыңны болгап җырладың да
Йөрәкләрне телде моңнарың.
Гөлнарым,
Борылмалы киткән юлларың.

Сине сагынып уйлар уйладым да
Чишмә буйларыннан урадым.
Гөлнарым,
Күз алдымда зифа буйларың
```
