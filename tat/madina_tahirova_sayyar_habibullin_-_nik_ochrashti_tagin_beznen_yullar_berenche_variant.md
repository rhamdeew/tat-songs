# Оригинал

### Мәдинә Таһирова / Сайяр Хәбибуллин - Ник очрашты тагын безнең юллар (беренче вариант)

```
Нәрсә булды безгә, әй дускаем,
Ни аерды безне, аңламыйм?
Айрылышты уйлар, ераклашты юллар,
Ну сорауга җавап табалмыйм.

Ник очрашты тагын безнең юллар?
Инде күңел түзми — ашкына.
Сүнеп барган хисләр кинәт кабындылар
Әверелеп язгы ташкынга.

Тәрәзәмә килеп сайрый кошлар,
Моңсулатып кичем-иртәмне.
Дулкынланды йөрәк, сине күргәч бигрәк,
Белгән кебек өзелеп көткәнне.
```
