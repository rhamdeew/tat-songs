# Оригинал

### Айдар Минһаҗев / Айдар Әгъләмов - Нишләдек без, туганнар?

```
Үткән саен гомер караңгырак,
Усалрак һаман бураннар,
Җан да тартмый, кан да тартмый бүген,
Ерагая гына туганнар.
Аларның тик файдалысын гына
Үз күрәбез, якын итәбез.
Ярлыларын җиде ятлар кебек
Сәламен дә алмый үтәбез.

Ә син мине гаеп итмәссеңме,
Туганым дип дәшсәм әгәр дә?!
Нишләтте бу каһәр суккан заман,
Нигә төштек мондый хәлләргә?!

Күрим дисәң ерак туганыңны,
Мал бүләргә чыксын җай гына.
"Мин дә сезнең нәсел әле" — диеп,
Эт тә бет тә килә яныңа.
Бер-беребез өчен яну-көю,
Сөенүләр күптән он(ы)тылган,
Көнчелекнең суык төтеннәре
Бәгырьләргә таштай утырган.

Ә син мине гаеп итмәссеңме,
Туганым дип дәшсәм әгәр дә?!
Нишләтте бу каһәр суккан заман,
Нигә төштек мондый хәлләргә?!

Бөтенесен күреп үсәләр бит,
Рәнҗемәсме безгә балалар?
Сүнеп бетсә нәсел учаклары,
Алар каян җылы табарлар?!

Ә син мине гаеп итмәссеңме,
Туганым дип дәшсәм әгәр дә?!
Нишләтте бу каһәр суккан заман,
Нигә төштек мондый хәлләргә?!
```
