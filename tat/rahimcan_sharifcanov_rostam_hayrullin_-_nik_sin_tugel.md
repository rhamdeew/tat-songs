# Оригинал

### Рәхимҗан Шәрифҗанов / Рөстәм Хәйруллин - Ник син түгел?

```
Җәйге таңның ал нурында
Гөлчәчәкләр балкып уяна.
Ник син түгел, юлларыма
Башка берәү чыгып елмая.

Сагышланам, сабырланам,
Сагынуларга ник соң юк чара?!
Яраланган аккош кебек
Өзгәләнә йөрөк, бичара.

Өмет өзми, сине көтөм,
Сине уйлап күңел юана.
Ник син түгел, ник син түгел —
Башка берәү миңа елмая.
```
