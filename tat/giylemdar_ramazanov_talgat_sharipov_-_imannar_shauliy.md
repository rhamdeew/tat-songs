# Оригинал

### Гыйлемдар Рамазанов / Тәлгать Шәрипов - Имәннәр шаулый

```
Тормыш кочагында,
Икәү яшь чагында,
Бер-беребезне өзелеп сөйгәндә.

Алтын көз көнендә,
Басу киңлегендә
Утырттык без япь-яшь имәннәр.

Безнең гомер белән
Ярышып үсә имән,
Ямь-яшеллек каплый җирләрне.

Без утырткан өчен
Биреп яшьлек көчен
Җиңәр алар хәтәр җилләрне.

Ярсу еллар үтәр,
Якты язлар җитәр,
Шул яланга икәү килербез.

Гомер матур үтә,
Урман шаулап үсә
Яшьлек дәвам итә диярбез.
```
