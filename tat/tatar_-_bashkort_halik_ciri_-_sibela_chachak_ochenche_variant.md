# Оригинал

### Татар - Башкорт халык җыры - Сибелә чәчәк (өченче вариант)

```
Гөлбакчага керәсең,
Гөлбакчага керәсең лә,
Гөл чәчәген өзәсең.
Өзелгән гөл чәчәгедәй
Аерылуны сизәсең.

Сибелә чәчәк, җилләр лә искәндә
Өзелә үзәк, исләремә төшкәндә.

Гөлбакчага керәм мин,
Гөлбакчага керәм мин лә,
Гөл чәчәген өзәм мин;
Өзелгән гөл чәчәгедәй
Аерылуны сизәм мин.

Сибелә чәчәк, җилләр лә искәндә
Өзелә үзәк, исләремә төшкәндә.
```
