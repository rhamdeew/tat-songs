# Оригинал

### Фирдүс Вахитов - Кил әле (беренче вариант)

```
Кояш нурларыдай балкып,
Миңа карап елмайдың.
Айның нурларыдай калкып,
Татлы йокыга салдың,
Миңа карап елмайдың.

Кил әле син, кил әле,
Яннарыма кил әле!
Зинһар, ярсытма йөрәкне,
Битләремнән үп әле!

Әкияттәй дөньяларда
Яшәве җиңелерәк.
Мине күргәч карап үт син,
Син бит минем мәхаббәт,
Син бит минем мәхаббәт.

Кил әле син, кил әле,
Яннарыма кил әле!
Зинһар, ярсытма йөрәкне,
Битләремнән үп әле!

Син яратсаң — баш әйләнә,
Очып йөримен җирдә.
Син яратсаң — мин бәхетле!
Сөймәм башка бер кемне!
Яратсаң — мин бәхетле!

Кил әле син, кил әле,
Яннарыма кил әле!
Зинһар, ярсытма йөрәкне,
Битләремнән үп әле!
```
