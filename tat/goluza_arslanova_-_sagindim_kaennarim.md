# Оригинал

### Гөлүзә Арсланова - Сагындым каеннарым

```
Мине сагындыгызмы,
Мине таныдыгызмы,
Шаулап куандыгызмы, каеннар?

Көткәнегезне беләм,
Ашкынып кайтып киләм,
Сезне өзелеп сагынганга каеннар!

Балачагым эзләре,
Әткәй-әнкәй төйәге.
Һәр чак үзенә тарта торган як!

Мине көткәнне беләм,
Ашкынып кайтып киләм,
Сине өзелеп сагынганга туган як!

Хәсрәтем таратырлык,
Яралар уңалтырлык,
Әлдә изгенән-изге төйәк бар!

Көткәнегезне беләм,
Ашкынып кайтып киләм,
Сезне өзелеп сагынганга туганнар!
```
