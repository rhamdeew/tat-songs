# Оригинал

### Илдар Шәрипов - Кулга кул тотынып

```
Исеңдәме, җаным, синең белән
Тәүге кабат очрашулар.
Шул мәлдән соң безнең гашыйк җаннар
Гомерлеккә бергә кавыштылар.

Кулга кул тотынып, гомер юлын
Үтәбез без гомер гел бергә.
Минем гашыйк күзләр — синең күздә,
Синең күзләр — минем күзләрдә.

Еллар уза, ә без синең белән
Шул ук күркәм, шул ук гашыйклар.
Тугры булган безнең мәхаббәткә
Барча галәм һәрчак шаһитлар.

Кулга кул тотынып, гомер юлын
Үтәбез без гомер гел бергә.
Минем гашыйк күзләр — синең күздә,
Синең күзләр — минем күзләрдә.

Мәхаббәтнең җимешләре булып
Имин үсә газиз балалар.
Тырышканда барысын Ходай бирә,
Имин генә торсын дөньялар.

Кулга кул тотынып, гомер юлын
Үтәбез без гомер гел бергә.
Минем гашыйк күзләр — синең күздә,
Синең күзләр — минем күзләрдә.
```
