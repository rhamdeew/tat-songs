# Оригинал

### Муса Җәлил / Латыйф Хәмиди - Син дулкыннар тавышын тыңладыңмы?

```
Син дулкыннар тавышын тыңладыңмы
Өермәле шаулы диңгездә?
Без дулкыннар, менә, күбекләнеп,
Ашкынабыз кич тә, көндез дә.

Безнең көчләр океан, безнең көчкә
Тиң килердәй көчләр табылмас!
Безнең дәртләр кире сүрлә белми,
Яңкынланып бер кат кабынгач.
```
