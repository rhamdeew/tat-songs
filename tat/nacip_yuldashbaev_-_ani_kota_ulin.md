# Оригинал

### Нәҗип Юлдашбаев - Әни көтә улын

```
Тагы тора кояшлы май,
Күктә кошлар кайталар.
Тирә-якка асыл гөлләр
Гөрләп чәчәк аталар.

Әни көтә улын, әле кайтыр, диеп,
Әни көтә улын, исән, диеп.

Көтә иде әни улын,
Нигә улы кайтмады?
Чәчәк атты ун тугыз ел,
Нигә башка атмады?

Әнкәй елый, улым кайда калды? — диеп,
Әнкәй елый, җирдә бәхет юк, диеп.

Әни тора, башын игән,
Сары чәчәк — кулында.
Бу әнине тагын ниләр
Көтә тормыш юлында?

Әнкәй әйтми башка, улым, саумы, диеп,
Әнкәй әйтми башка, улым, саумы, диеп.
Әни көтә улын, әле кайтыр, диеп,
Әни көтә улын, исән, диеп.
```
