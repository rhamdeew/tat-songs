# Оригинал

### Заһид Фәйзи / Рөстәм Яхин - Ник соң аерылдык?

```
Ятлар сөйләвенә карап, юкка
Үпкәләшеп, хәтер калдырып,
Керсез мәхаббәткә хыянәт итеп,
Тәкәбберлек белән аерылдык.

Уза гомер инеш суы сыман,
Яш(е)рен сагыш белән тын гына,
Сүнми икән тәүге сөю уты,
Күңелем һаман сине юксына.

Җитәкләшеп йөргән сукмакларга
Сагышланып карыйм, тукталып;
Бергә булган чакның хәтирәсен
Искә алырга калды уфтанып.

Яшерен сагыш белән уза гомер,
Инеш суы сыман тын гына;
Калды, ахры, чынлап йөрәкләрдә
Сызландыргыч уртак моң гына…
```
