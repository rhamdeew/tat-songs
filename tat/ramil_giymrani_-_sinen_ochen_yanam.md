# Оригинал

### Рамил Гыймрани - Синең өчен янам

```
Синең өчен — туганмын җиргә,
Янәш барырга дип гомергә.
Пар аккоштай очаек җаным,
Киң офыкларга гел бергә.

Сулар һавам, йөрәк дәвам,
Синең өчен генә янам.

Җил давыллар, авыр сынаулар,
Бергә чакта гына, сизелми.
Терәк булсаң минем тормышымда,
Кыелыклар үтәр, сизелми.

Сулар һавам, йөрәк дәвам,
Синең өчен генә янам.

Карашымны синең маяк итеп,
Күзләреңә генә табынам.
Син булганда минем тормышымда,
Сикәлтәләр булса абынма.

Сулар һавам, йөрәк дәвам,
Синең өчен генә янам.
```
