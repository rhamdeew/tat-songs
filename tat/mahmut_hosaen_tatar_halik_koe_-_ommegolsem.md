# Оригинал

### Мәхмүт Хөсәен / Татар халык көе - Өммегөлсем

```
Чулпан калка, ал таң ата.
Таң Гөлсемне уята
Гөлсем таңнан кырга барып,
Ай-урагын уйната.

Бас әйдә, Өммегөлсем,
Идәннәре бөгелсен.
Басуыңда зур бәхетнең
Чаткылары күренсен!

Гөлсем шук та, Гөлсем ут та, —
Егет күңелен яндыра.
Бик еш шулай сөйгәнен дә
Эштә хәйран калдыра.

Бас әйдә, Өммегөлсем,
Идәннәре бөгелсен.
Басуыңда зур бәхетнең
Чаткылары күренсен!

Көн дә дими, төн дә дими,
Ару-талуны белми.
Ярышта да безнең Гөлсем
Алдынгылыкны бирми.

Бас әйдә, Өммегөлсем,
Идәннәре бөгелсен.
Басуыңда зур бәхетнең
Чаткылары күренсен!
```
