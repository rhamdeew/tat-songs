# Оригинал

### Якуп Колмый / Рафик Сәлмәнов - Мин сине эзлимен

```
Мин сине эзлимен,
Тынгылык (тынычлык) белмимен,
Эзлимен, түзмимен…
Эндәш, кайда син?
Хәтердә күзләрең,
Елмайган йөзләрең,
Ягымлы сүзләрең…
Эндәш, кайда син?

Уйлаган уемдай,
Җырлаган җырымдай,
Балкыган йолдыздай
Бигерәк матур син!
Сөйкемле каендай,
Саф аяз таңымдай,
Чәчкәле язымдай
Миңа якын син!

Үтсәдә елларым,
Аксада суларым,
Йөрегән юлларым
Урау булса да,
Мин көтәм, акыллым,
Эзлимен — табармын,
Мин беләм син барын
Якты дөньяда!
```
