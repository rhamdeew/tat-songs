# Оригинал

### Фәнәвис Дәүләтбаев / Лилия Вахитова - Гомернең үтелгән еллары

```
Дисбегә тезелгән шикелле
Гомернең үтелгән еллары.
Күңелгә һәр елы кадерле,
Кимеми, зурая ул бары.

кушымта
Шатлыклы, газаплы шул еллар
Айлардан, көннәрдән җыелган.
Күпләре хәтердән җуелган,
Азлары күңелгә уелган.

Гомернең юллары кичелгән-
Үрләргә менелгән, төшелгән.
Канәгать булганнан, тапканнан.
Әлегә шөкер дим эчемнән.
Хурланырлык түгел язмышым.
Күп анда югалту-табышым.

кушымта
Шатлыклы, газаплы шул еллар
Айлардан, көннәрдән җыелган.
Күпләре хәтердән җуелган,
Азлары күңелгә уелган.

Күкләрдә язылган тәкъдирне
Үзгәртеп буламы аны соң?!
Туар таң шикелле бәхетне
Файдасыз үткәннән юллавым.
Аралар өзелгән, аръякта —
Гомернең үтелгән еллары.
кушымта
Шатлыклы, газаплы шул еллар
Айлардан, көннәрдән җыелган.
Күпләре хәтердән җуелган,
Азлары күңелгә уелган.
```
