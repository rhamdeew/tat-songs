# Оригинал

### Резеда Вәлиева / Таңсылу Габидуллина - Йөрәгемә куштым

```
Йөрәгемә куштым башка сине яратмаска,
«Синең зәңгәр күзләрең!» — дип җанатмаска.
Юк, кирәкми, зәңгәр күзләр, ялган күзләр,
Сине күрмичә дә түзәр җаным түзәр, түзәр.

Юк, кирәкми, зәңгәр күзләр, ялган күзләр,
Сине күрмичә дә түзәр җаным түзәр, түзәр.

Акылым белән аңга барын аңлатырмын,
«Хыянәтче күзләр алар», — дип әйтермен.
Җаным минем сизгер бит ул, ул тиз сизәр,
Ул яратмас башка сине, сүнәр, бизәр.

Җаным минем сизгер бит ул, ул тиз сизәр
Ул яратмас башка сине, сүнәр, бизәр.

Син диеп инде башка һич ут йотмамын.
Хыянәтең мәңге, мәңге онытмамын.
Зәңгәр күзләр хыянәтче була икән,
Юк, кирәкми, сөймим сине, сөймим бүтән.

Зәңгәр күзләр хыянәтче була икән,
Юк, кирәкми, сөймим сине, сөймим бүтән.
```
