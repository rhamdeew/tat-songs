# Оригинал

### Татар  халык җыры - Исле гөл (өченче вариант)

```
Исле гөлем, исле була
Яран гөл төсле була ла.
Сагынганда айга карыйм
Ай синең төсле була.

Исле гөлем, исле гөлем,
Исле гөлләрем сиңа ла.
Исле гөлем саф күңелем
Бүләк итәм мин сиңа.

Исле гөлем, исле була
Чәчәкләр аткан чакта ла.
Шул чәчәкләргә карыйм мин
Сезне сагынган чакта.
```
