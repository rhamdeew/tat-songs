# Оригинал

### Флүс Латыйфи / Рөстәм Сәрвәров - Ташлыярым

```
Ташлыярым, синдә — ярым,
Ташлыярым, синдә — якын дусларым.
Су буйларында йөргәндә
Җырларыма кушармын.

Кичләрен су төшеп алгач
Яр буена Мәлләгә,
«Сөям», — дигәч,
«Мин дә», — дигән
Сүзләреңне жәлләмә.

Ташлыярым, синдә — ярым,
Ташлыярым, синдә — якын дусларым.
Табигатем кочагында
Җырларыма кушармын.

Чагыл тауларын менәргә
Бик күп көч кирәкми лә,
Вәгъдәдә тору өчен
Көч кирәк йөрәкләргә.
```
