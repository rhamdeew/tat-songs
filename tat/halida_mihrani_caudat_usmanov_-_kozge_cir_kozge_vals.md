# Оригинал

### Халидә Миһрани / Җәүдәт Усманов - Көзге җыр (Көзге вальс)

```
Яфракларга сары төште,
Йөрәкләрне сагыш өтте,
Хушлашабыз инде җәй белән.
Әйтерсең дә сак белән сок,
Иртәгәсе кем бар кем юк,
Дөнья калыр синнән һәм миннән.

Сарыларны җуеп булмый,
Матурлыктан туеп булмый,
Озагырак торсын иде җәй.
Дөньялыкта без бит кунак,
Гомер (тормыш) үзе бик зур бүләк,
Тик тормошлар (гомерләр) булсын (узсын) иде җай.

Өзелеп бер яфраҡ төште,
Бөрелеп бар миләш пеште,
Кышлар салкын булыр ахыры.
Җәйләребез ямьле булды (үтте),
Көзләребез нурга тулды,
Матур (шулай) үтсен гомер юллары!
Шулай үтсен иде гомерләрнең агымы!
```
