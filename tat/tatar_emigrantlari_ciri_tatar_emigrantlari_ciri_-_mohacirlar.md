# Оригинал

### Татар Эмигрантлары Җыры / Татар эмигрантлары җыры - Мөһаҗирләр

```
Әйа кош, сән очарму сән,
Һаваларда йөрермү сән?
Бән аерылдым туган илдән,
Алар хәлен белермү сән?

Агымсу тигрәсе таллы,
Гаҗәеп яфрагы баллы.
Үзенең кодрәте белән
Бу гамьне кальбемә салды.

Әйа җил, сән искел бик тиз,
Бәнем хәлем илтеп җиткез:
Бән аерылдым туган илдән,
Къәрарым калмады һәргиз.
```
