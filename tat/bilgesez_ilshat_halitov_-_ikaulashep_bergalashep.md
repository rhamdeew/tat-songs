# Оригинал

### Билгесез / Илшат Хәлитов - Икәүләшеп бергәләшеп

```
Язын килгән хисләр, синең нурлы күзләр,
Шушы якты кичтә мине уйлата.
Сине төштә күрсәм, чәчәк ата галәм,
Мәхәббәттә дәва бармы дөнъяда.

Икәүләшеп җитәкләшеп йолдызлага тизлек алды,
Җыһанда сатылган йолдыз кире кайтмады.
Икәүләшеп бергәләшеп, томаннарда бер адашып,
Мизгелләр үтәп табыштык аймылышка.

Сиңа гашыйк булдым, тагы чәчәк аттым,
Миңа серен ачтың тәүге караштан.
Синнән кап-каралган шатлыклардан очам
Күңелем белән хәтта җыһанны кочам.

Икәүләшеп җитәкләшеп йолдызлага тизлек алды,
Җыһанда сатылган йолдыз кире кайтмады.
Икәүләшеп бергәләшеп, томаннарда бер адашып,
Мизгелләр үтәп табыштык аймылышка.
```
