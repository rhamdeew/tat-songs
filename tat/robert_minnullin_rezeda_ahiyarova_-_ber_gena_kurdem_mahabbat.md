# Оригинал

### Роберт Миңнуллин / Резеда Ахиярова - Бер генә күрдем (Мәхәббәт)

```
Мин бит сине гомеремдә бер генә күрдем.
Бер генә күрдем — шунда ук күңелгә кердең.

Сиңа гашыйк булуымны
Калдым сизми дә.
Башларымны газапларга
Салдың син нигә?

Мин бит сина ялгышкына күз аткан идем.
Уйнап кына, уйнап кына сүз әйткән идем.

Сиңа гашыйк булуымны
Калдым сизми дә.
Башларымны газапларга
Салдың син нигә?

Мин бит шаяртырга гына теләгән идем.
Мин бит болай булыр диеп белмәгән идем.

Сиңа гашыйк булуымны
Калдым сизми дә.
Башларымны газапларга
Салдың син нигә?

Үзәкләремне өздең син, бәгырьне телдең.
Югыйса, сине бер генә бер генә күрдем.

Сиңа гашыйк булуымны
Калдым сизми дә.
Башларымны газапларга
Салдың син нигә?
```
