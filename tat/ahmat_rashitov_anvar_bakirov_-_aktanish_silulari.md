# Оригинал

### Әхмәт Рәшитов / Әнвәр Бакиров - Актаныш сылулары

```
Актаныш кызлары белән,
Ах, таныш булулары!
Күпме егетне саргайта
Актаныш сылулары.

Ярлар эзләп илләр гиздем,
Ятларга күз атмадым.
Актаныш кызлары кебек
Гүзәлләрне тапмадым.

Агыйделдә үскән мәллә,
Суларын эчкән мәллә.
Таң алсуы йөзләренә
Нур булып күчкән мәллә?

Актаныш кызлары белән,
Ах, таныш булулары!
Күпме егетне саргайта
Актаныш сылулары.
```
