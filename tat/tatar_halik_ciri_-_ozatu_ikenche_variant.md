# Оригинал

### Татар халык җыры - Озату (икенче вариант)

```
Урманнарда йөргән чакта
Адаштым юлларымны.
Күңелем йомшап елармын дип
Бирмәдем кулларымны.

Сикердем читтәннәрдән лә,
Йөгердем җитә алмадым.
Ни булды икән телләремә
Сау бул дип әйтә алмадым?

Өй артындагы талларны
Яратмасаң кисәрсен шул.
Мин сагынып тәрәз ачсам
Син җил булып исәрсен.
```
