# Оригинал

### Билгесез - Бәхет сулышын тыңла

```
Тәүге бәхет килә аны көткәннәргә,
Саф йөрәкләр тик көтеп алыр.
Минем бәхет синдә, булыйк мәңге бергә,
Синсез, җаным, күңелем сулыр.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.

Күңел күзе күргән, ярын аңлый белгән,
Бәхеткәен шул саклап кала.
Тыйма хисләреңне, тыңла йөрәгемне,
Бәхетебез сулышын тыңла.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.

Бәхет үзебездә саклый белик кенә,
Оҗмах була икән җирдә дә.
Зәңгәр күккә очып, бар җиһанны кочып,
"Мин — бәхетле!" диясем килә.

Сиңа уйлар илтәләр, сиңа,
Миңа син бит кадерле, миңа.
Тула күңел хисләрдән, тула,
Тыңла бәхет сулышын, тыңла.
```
