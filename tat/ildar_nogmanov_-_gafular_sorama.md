# Оригинал

### Илдар Ногманов - Гафулар сорама

```
Сөюләр көткән син йөрәгемә
Кагылма, кит дидең, кит дидең,
Өметләр биргән күңел түрендә
Син идең, син идең.

Ялгышлар диеп, язмышлар диеп,
Алдама, алдама, алдама.
Яныма килеп гафулар миннән
Хәзер син сорама, сорама.

Икәү кабызган сөю утында
Мин янам, мин янам, мин янам.
Бергә бәхетле булган чакларны
Уйланам, уйланам, уйланам.

Ялгышлар диеп, язмышлар диеп,
Алдама, алдама, алдама.
Яныма килеп гафулар миннән
Хәзер син сорама, сорама.

Исеңдә микән, янәшәмдә мин
Бәхетле, бәхетле мин дидең.
Нигә соң икән, көтмәгәндә син
Сүнде бит, сөюләр юк дидең

Ялгышлар диеп, язмышлар диеп,
Алдама, алдама, алдама.
Яныма килеп гафулар миннән
Хәзер син сорама, сорама.
```
