# Оригинал

### Татар халык җыры - Одес дигән зур кала

```
Одес дигән зур калада
Урамнан солдат бара;
Безнең хәлне сорасагыз,
Аркабыздан тир ага.

Иртән торып тышка чыксам,
Былбыл сайрый өн белән;
Марш сугып без йөрибез
Урам буе көн берлән.

Аягымда күн итегем,
Походта тузар микән?
Казарма эчендә ятып
Яшь гомер узар микән?

Өстемдәге шинелемне
Ничек итеп туздырыйм?
Җыен ятлар арасында
Ничек гомер уздырыйм?

—
* Солтан Габәши архивыннан, фольклор
белгече Геннадий Макаров транскрипциясендә.
```
