# Оригинал

### Нәҗип Әсәнбаев / Морат Әхмәтов - Бик сагындым сине, Туган илем

```
Бик сагындым сине, Туган илем,
Кайтыр юллар әле төшмәгән.
Газиз әнкәемнең учагында
Мин ашыйсы ашлар пешмәгән.

Бик сагындым сине, Туган илем,
Кайтыр юллар әле төшмәгән.
Мин тартасы моңлы курайлар да (Газиз Уралымның итәгендә)
Уралымда әле үсмәгән (Мин тартасы курай үсмәгән).

Бик сагындым сине, Туган илем,
Кайтыр юллар әле төшмәгән.
Кошлар очып җитмәс бу якларда
Ризыкларым әле бетмәгән.
```
