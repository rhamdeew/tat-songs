# Оригинал

### Рәми Гарипов / Абрар Габдрахманов - Иң бөек сүз әле әйтелмәгән

```
Иң бөек сүз әйтелмәгән,
Иң бөек көн әле ул алда.
Килер ул көн, килер иң зур бәйрәм,
Туй ясарбыз шаулап урамда.

Ул туйга мин, бәлки, барып җитмәм,
Ярсу атым янып егылыр.
Тик минсез дә тулы булыр ул туй,
Мин белмәгән дуслар җыелыр.

Сөйләшсәләр әгәр туган телдә,
Телгә алыр алар мине дә.
Чөнки мин киләсәк белән бергә
Саклап чаптым, телем, сине дә.
```
