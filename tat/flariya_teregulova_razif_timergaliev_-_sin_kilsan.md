# Оригинал

### Флария Терегулова / Рәзиф Тимергалиев - Син килсәң…

```
Киттең дә югалдың,
Кайларга юл алдың,
Юксынам шул сине, югалтып.
Өметемне өзмим,
Синнән хәбәр көтәм,
Сагынып кайтырсың дип, бервакыт.

Берни әйтә алмыйм,
Өзгәләнә күңел,
Моң-зарларга чумып көн саен.
Күзләремнең нуры
Син диеп түгелә,
Сагышларга салма, кайт, җаным.

Еллар арасында.
Моңлы карашымда
Хыял кошы булып йөрисең.
Көтә сине йөрәк,
Аңа дәва кирәк.
Язлар килде, ник син килмисен?

Килсәң яннарыма,
Кил син җәйләремә,
Назлар тулы җәем үтмәгән.
Сиңа гына булган,
Иң кайнар хисләрем
Еллар буена да бетмәгән.
Сине көтә, көтә, сүнмәгән…
```
