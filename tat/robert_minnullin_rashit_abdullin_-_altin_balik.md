# Оригинал

### Роберт Миңнуллин / Рәшит Абдуллин - Алтын балык

```
Уйный суда, уйный суда алтын балык,
Уйный, чума алтынланып, алтынланып.
Әллә инде минем белән, минем белән
Уйный шулай алтын балык, алтын балык.

Алтын балык, алтын балык,
Сөйгәнемә барчы алып,
Алтын балык!

Уйный суда, уйный суда алтын балык,
Нигә шулай уйный икән, уйный икән?
Әллә инде минем белән, минем белән
Сөйләшергә уйлый микән, уйлый микән?

Алтын балык, алтын балык,
Сөйгәнемә барчы алып,
Алтын балык!

Уйный суда, уйный суда алтын балык,
Миңа карап әйтә нидер, әйтә нидер:
Сөйгәнеңә әйдә, дидер, әйдә, дидер,
Тик безнеңчә әйтелмидер, әйтелмидер.

Алтын балык, алтын балык,
Сөйгәнемә барчы алып,
Алтын балык!
```
