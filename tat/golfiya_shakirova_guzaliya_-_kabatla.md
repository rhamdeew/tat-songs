# Оригинал

### Гөлфия Шакирова / Гүзәлия - Кабатла

```
Елмаясың яратам дип,
Сөйлисең, сөйлисең дә,
Шул чакта бәхет чаткысы
Кабына бит йөзеңдә,
Кабына бит йөзеңдә,
Мәгънә бар һәр сүзеңдә,
Күң(е)лем белән шул сүзеңә
Кушылам бит үзем дә,
Кушылам бит үзем дә.

Кабатла шул сүзеңне,
Кабатла тагын, тагын,
Мин дә шулай өзелепләр
Яратам сине, җаным.

Кабатла (ә)ле шул сүзеңне,
Кабатлаудан туктама.
Бәхетемнән сөенеп,
Шатлык яшьләре тама.
Яратуыңны белсәм дә
Һаман тыңлыйсым килә.
Шул сүзләрне көйгә салып
Минем җырлыйсым килә.

Кабатла шул сүзеңне,
Кабатла тагын, тагын,
Мин дә шулай өзелепләр
Яратам сине, җаным…
Җаным!..
```
