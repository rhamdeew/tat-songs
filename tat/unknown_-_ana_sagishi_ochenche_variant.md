# Оригинал

### Нәкыйп Галимов / Фәнир Галимов - Ана сагышы (өченче вариант)

```
Киткән чакта кайтыр юлларыңны
Көпчәкләрга урый киткәнсең,
Күкрәгемә кысып, чәчләреңнән
Бер сыйпаудан мәхрүм иткәнсең.

Әйтегезче, зинһар, аңлатыгыз,
Кемгә кирәк сугыш кем өчен?!
Бер гаепсез балам гомер белән
Алдыгыз сез миннән кем өчен.

«Яз кайтмасам, әнкәй, хафаланма.
Көт, кайтырмын, — дидең. — Көзләрен».
Яз да үтте, инде көз дә җитте,
Саргайды бит, балам, йөзләрем.

Әйтегезче, зинһар, аңлатыгыз,
Кемгә кирәк сугыш кем өчен?!
Бер гаепсез балам гомер белән
Алдыгыз сез миннән кем өчен.

Өзгәләнә үзәк, әнкәйләрдән
Ник иртәрәк китә балалар?
Шуны күрер өчен үстердекме?!
Йөрәккәйләр әрнеп яналар.

Әйтегезче, зинһар, аңлатыгыз,
Кемгә кирәк сугыш кем өчен?!
Бер гаепсез балам гомер белән
Алдыгыз сез миннән кем өчен.
```
