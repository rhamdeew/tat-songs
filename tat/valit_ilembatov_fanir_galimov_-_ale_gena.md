# Оригинал

### Вәлит Илембәтов / Фәнир Галимов - Әле генә…

```
Уйланам да хәйран калам,
Гомернең агышына.
Төшенеп тә өлгермәдем
Дөньяның аешына.

Әле генә мәхәббәттән,
Могҗиза көтә идем.
«Кайчан үсеп җитәрмен?» дип
Зарыгып бетә идем.

Бәхет нурына коенып,
Шатлана-көлә идек.
Гомер буе гел дә шулай
Булыр дип белә идек.

Әле генә мәхәббәттән,
Могҗиза көтә идем.
«Кайчан үсеп җитәрмен?» дип
Зарыгып бетә идем.

Яшьлек язым әллә кайда,
Еракта калган икән.
Язлар алда дип юану
Матур бер ялган икән.

Әле генә мәхәббәттән,
Могҗиза көтә идем.
«Кайчан үсеп җитәрмен?» дип
Зарыгып бетә идем.
```
