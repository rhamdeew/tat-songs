# Оригинал

### Наҗар Нәҗми / Мөдәрис Морзаханов - Кандаулы тау

```
Кандаулы тау текә-текә,
Менгәч, текәсе бетә.
Текә күңелле сөйгән яр
Тауга менгәнне көтә,
Менгәч, текәсе бетә.

Кандаулы тау башы төтен —
Учаклар сүнмәгән чак,
Учаклар сүнгән чакта да
Күңел сүрелмәгән чак,
Учаклар сүнмәгән чак.

Кандаулы тау асты болын,
Агыйдел алкын ага.
Зәңгәр киңлекләр үзенә
Җаныңа тартып ала,
Агыйдел алкын ага.

Кандаулы тау башында җил,
Бүген җил тынган икән.
Туған җирнең (якның) гүзәллеген
Кем күреп туйган икән?
Кем җырлап туйган икән?
```
