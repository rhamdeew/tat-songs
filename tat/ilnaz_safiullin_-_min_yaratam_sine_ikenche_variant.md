# Оригинал

### Илназ Сафиуллин - Мин яратам сине (икенче вариант)

```
Күпме дөньяда кайгылар,
Күпме дөньяда хәсрәтләр!
Берни дә тормый,
Беркем кирәкми,
Сакла безне ялгызлыктан…
Мин яратам сине,
Мин юксынам сине…
Сакла безне ялгызлыкта.

Мин ышанам,
Кайтыр ул бергә чаклар;
Мин ышанам,
Килер ул узган вакытлар,
Мин яратам сине,
Мин юксынам сине…
Сакла безне ялгызлыктан…

Күпме дөньяда бәхетләр,
Күпме дөньяда шатлыклар!..
Беркем кирәкми,
Берни дә тормый
Аерым үткәрелгән чаклар…
Мин яратам сине,
Мин юксынам сине…
Яши алмам синнән башка.

Мин ышанам,
Кайтыр ул бергә чаклар;
Мин ышанам,
Килер ул узган вакытлар,
Мин яратам сине,
Мин юксынам сине…
Яши алмам синнән башка.
```
