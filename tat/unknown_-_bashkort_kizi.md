# Оригинал

### Гөлфия Юнысова / Эльвира Хаматнурова - Башкорт кызы

```
Башҡорт ҡыҙы — һинең йөрәгеңдә
Заман моңо, заман ялҡыны.
Милләтемдең борондарҙан килгән
Яҡты рухлы зирәк аҡылы.

Минең йөрәктә лә балҡый
Илһөйәрлек тойғоһо.
Мин дә йәшәү усағына
Ҡуҙ өҫтәр башҡорт ҡыҙы.

Кәрәк саҡта тыуған ерен ҡурсып
Яуға сапҡан башҡорт ҡыҙҙары.
Баш эйәләр ихлас тоғролоҡҡа
Ысыҡтай саф хыял-уйҙары.

Башҡорт ҡыҙы һаман ғәмле бөгөн,
Ил яҙмышы уның ҡанында.
Киләсәкте ҡурсып, һаҡлап тора
Халҡым сәңгелдәге янында.
```
