# Оригинал

### Эдуард Хәмидуллин - Карлыган

```
Бер күрүдә гашыйк булдым,
Карлыгандай күзләрең.
Кызыл чия иреннәрең,
Балдай татлы сүзләрең.

Җәйнең айлы төннәрендә
Пешә кара карлыган.
Ярата сине диярсең,
Яннарыңда мин булмам.

Гомер бит ул үтми тормый,
Көзләр дә килеп җитте.
Сиңа дигән карлыганым
Күптәннән пешеп кипте.

Җәйнең айлы төннәрендә
Пешә кара карлыган.
Ярата сине диярсең,
Яннарыңда мин булмам.

Сандугачлар кунып сайрый
Карлыганга тагын да.
Кавышырбыз микән тагын
Карлыганнар янында.

Җәйнең айлы төннәрендә
Пешә кара карлыган.
Ярата сине диярсең,
Яннарыңда мин булмам.
```
