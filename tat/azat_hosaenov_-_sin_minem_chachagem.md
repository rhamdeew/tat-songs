# Оригинал

### Азат Хөсәенов - Син — минем чәчәгем

```
Син килдең каршыма
Чәчәкләрне кочып.
Син үзең, сылукай,
Гөлләрдән бер чәчәк.

Мин сине яратам,
Яратып дәшәмен:
"Гомеремдә син минем
Иң сылу чәчәгем".

Кайларда йөрсәм дә
Кемнәрне күрсәм дә —
Чәчәктәй буй-сының
Гел минем уемда.

Мин сине яратам,
Яратып дәшәмен:
"Гомеремдә син минем
Иң сылу чәчәгем".

Ай үтәр, ел үтәр,
Көтелгән көн җитәр —
Булырбыз без бергә
Мәңгегә, гомергә!

Мин сине яратам,
Яратып дәшәмен:
"Гомеремдә син минем
Иң сылу чәчәгем".
```
