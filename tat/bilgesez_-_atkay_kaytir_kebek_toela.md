# Оригинал

### Билгесез - Әткәй кайтыр кебек тоела

```
Әтиемне төшләремдә күрәм
Битләремә куя уң кулын
Ул елмаеп күзләремә карый
Сагындынмы, улым, күз нурым?

Актарылып аккач язгы ташу
Елгалар да кайта юлына
Яңа гына чыккан чирәм өсләп
Әткәй кайтыр кебек тоела.

Мине сиңа бүләк иткән язлар
Миннән сиңа алды мәңгегә
Ак төсләрдән генә тормый донья
Бурчуларын булса рәнҗемә.

Мәнгелеккә диеп китмәден син
Баш очында сирин куагы
Синен җанын язлар җиткән саен
Ак чәчкәләр булып туадыр.
```
