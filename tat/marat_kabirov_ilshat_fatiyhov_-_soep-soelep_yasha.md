# Оригинал

### Марат Кәбиров / Илшат Фатыйхов - Сөеп-сөелеп яшә

```
Аллардан ал чагың әле
Гөлдән гөл чагың әле.
Балланган алмалар сыман
Тәмләнгән чагың әле.

Бәхет нурына коенып,
Назга күмелеп яшә.
Тиңең белән тигез булып,
Сөеп — сөелеп яшә.

Иң ямьле чагың әле,
Иң тәмле чагың әле.
Тәмләнәсең, ямьләнәсең,
Бардыр ла тагын әле.

Бәхет нурына коенып,
Назга күмелеп яшә.
Тиңең белән тигез булып,
Сөеп — сөелеп яшә.
```
