# Оригинал

### Гөлфия Шакирова / Лилия Вахитова - Исән-сау булсыннар киткәннәр!

```
Әй, ходаем, ишет әле мине,
Ишет әле, һәрбер әйткән сүзне.
Кичерергә шушы мәхшәрне,
Һәр кешегә бирсен түземне.

Кан, яшьләр түгелмәсен җирдә,
Сугыш, үлем кирәк түгел бер дә.
Ишетегез, галәм,җирләр, күкләр,
Исән-сау булсыннар киткәннәр!
Исән-сау булсыннар киткәннәр!

Тизрәк бетсен иде сугышы,
Ир-егетләр кайтсын өйләргә.
Улларын сугышка озаткан,
Сабырлык бирсен әниләргә.

Кан, яшьләр түгелмәсен җирдә,
Сугыш, үлем кирәк түгел бер дә.
Ишетегез, галәм,җирләр, күкләр,
Исән-сау булсыннар киткәннәр!
Исән-сау булсыннар киткәннәр!

Сабырлык бирсен балаларга,
Тизрәк кайсын иде әтисе.
Әти диеп, тотып кулыннан,
Сөенече бардыр әйтәсе.

Кан, яшьләр түгелмәсен җирдә,
Сугыш, үлем кирәк түгел бер дә.
Ишетегез, галәм,җирләр, күкләр,
Исән-сау булсыннар киткәннәр!
Исән-сау булсыннар киткәннәр!
```
