# Оригинал

### Сибгат Хәким / Хөснул Вәлиуллин - Иртәнге сихерле юллар

```
Иртәнге сихерле юллар,
Тик юллар юанычым,
Барам, табармын шикелле
Сузылган юллар очын.

Әйтсеннәр күргән-белгәннәр;
Хыялый, дивана дип,
Гомрен юлларда үткәрә,
Юлларда юана дип.

Алдымнан минем юлларга
Сыялмый үтте юллар,
Хыяллар якты булганга,
Хыялый итте еллар.

Иртәнге сихерле юллар,
Тик юллар юанычым,
Барам, табармын шикелле
Узылган юллар очын.
```
