# Оригинал

### Татар - Башкорт халык җыры - Кара карлыгач микән (икенче вариант)

```
Карлыгачым, әйт үземә,
Кайсы талга кунасың?

Кара карлыгач микән,
Сары сандугач микән
Аны үстергән әнкәсе?

Күзем йомсам, күз алдымда
Торган кебек буласың.

Кара карлыгач микән,
Сары сандугач микән
Аны үстергән әнкәсе?
```
