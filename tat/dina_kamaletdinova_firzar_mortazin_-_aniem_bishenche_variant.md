# Оригинал

### Динә Камалетдинова / Фирзәр Мортазин - Әнием (бишенче вариант)

```
Көзләр җитә, диеп моңсуланма,
Өмет өзмә, килер язлар да.
Көзләрен дә гөлләр яшәрерлек,
Әни, синең җылы назларда.
Айдан микән, әллә кояштанмы?
Күзләреңә күчкән яктылык.
Балам, диеп, сөеп бер карасаң,
Бөтен дөньям китә яктырып.
Еракларда синнән яшәсәм дә,
Йөрәгемдә һәр көн син минем.
Бу җырымны сиңа багышладым,
Мәрхәмәтле, газиз әнием.
```
