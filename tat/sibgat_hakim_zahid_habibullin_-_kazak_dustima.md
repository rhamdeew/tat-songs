# Оригинал

### Сибгат Хәким / Заһид Хәбибуллин - Казакъ дустыма

```
Казан дигән каладан, Юлдашлар,
Идел белән Камадан, Юлдашлар,
Тукай йөргән төшләрдән,
Меңнәрчә дус-ишләрдән

Сезгә күп сәлам,
Сезгә күп сәлам!

Җамбулны онытмаган, Юлдашлар,
Картлар һәм оныклардан, Юлдашлар,
Атаклы Ала-тауга,
Күпләгән бала тауга,

Бездән зур сәлам,
Бездән зур сәлам!

Безнең хәлгә килгәндә, Юлдашлар,
Кызлар сездә кияүдә, Юлдашлар,
Әйбәт нигез корсыннар,
Әйбәт, тигез торсыннар.

Әйтегез сәлам,
Әйтегез сәлам!

Яшел чирәм далада, Юлдашлар,
Хәзер кая карама, Юлдашлар,
Эштә безнең якташлар,
Хезмәттә сынатмаслар.

Әйтегез сәлам,
Әйтегез сәлам!
```
