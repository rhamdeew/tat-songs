# Оригинал

### Гөлшат Зәйнәшева / Азат Хөсәенов - Куаныч күле

```
Куаныч күле кайда? —
Идел буе тугайда.
Идел буенда йөрдек,
Куаныч күлен күрдек.

И Куаныч күлләре,
Тирән булган элгәре.
Суы беткән, саеккан,
Шуңа күрә боеккан.

И Куаныч күлләре,
Күл читендә гөлләре.
Күл буйлары ямансу,
Килмәгән кыр үрдәге!..

Җыйналыйк җөй көннәрне,
Чистартыйк без күлләрне.
Тирән булса Куаныч,
Безгә булыр юаныч!
```
