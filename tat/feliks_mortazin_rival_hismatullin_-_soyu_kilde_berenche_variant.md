# Оригинал

### Феликс Мортазин / Риваль Хисмәтуллин - Сөю килде (беренче вариант)

```
Җәйләр икебезгә,
Уртак хисләр бирде
Сандугачлар безгә
Сөю алып килде!

Сандугачларның җырларын
Икәү генә тыңларбыз,
Гомер юлларын икебез!
Җитәкләшеп узарбыз!

Кочагыма алып
Назлап сөям сине,
Сандугачлы таңнар
Безгә сөю бирде!

Сандугачларның җырларын
Икәү генә тыңларбыз!
Гомер юлларын икебез!
Җитәкләшеп узарбыз!

Дөнья нинди матур
Тормыш нинди ямле,
Безнең хисләребез
Яшәр әле мәңге!

Сандугачларның җырларын
Икәү генә тыңларбыз,
Гомер юлларын икебез!
Җитәкләшеп узарбыз!
```
