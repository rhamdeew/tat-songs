# Оригинал

### Мөнир Мазунов / Алмаз Монасыйпов - Сине уйлап

```
Сине уйлап йөргәндә мин
Утсыз көям, утсыз янам.
Ә үзеңне күргәндә мин
Кинәт әйтер сүзсез калам.

Кушымта:
Җан теләгән җавап көтеп
Ай узармы, ел үтәрме,
Әллә йөрәк янып бетеп,
Бар өметем дә сүнәрме?

Бер тутырып карысың да,
Айный алмыйм тәэсиреңнән.
Керфекләрең арасына
Нинди серләр яшеренгән?

Ике күзең коесына
Ие йолдыз нуры сыйган.
Чыга алмаслык булып шунда
Мәхәббәтем кереп чумган.
```
