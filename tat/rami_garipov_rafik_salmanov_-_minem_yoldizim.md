# Оригинал

### Рәми Гарипов / Рафик Сәлмәнов - Минем йолдызым

```
Гомерем буе мин бит ялгызым,
Эзлимен сине, якты йолдызым.
Югалдың кайларда?
Юл алдың кайдарга?
Синсез, йолдызым, юлым бик озын…

Нигәдер күпне көтә күңелем,
Көткәнгә микән күпне киң илем!
Дауыллы елларда,
Бормалы юлларда
Мин җиңел йөкле булмам, сөеклем!

Чакыра тагы ерак юлларым,
Юлдашым минем — йөрәк моңнарым,
Тынгысыз көннәрдә,
Йолдызсыз төннәрдә
Күрче мин барын, яктырт юлларым!
```
