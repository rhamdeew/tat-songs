# Оригинал

### Хәйдәр Хәлимуллин / Сания Әхмәтова - Азнакай нефтьчеләре җыры

```
Нефть биргән данлы бу җирләр,
Геройларга юмарт бу төшләр.
Азнакайда дистәләгән ирләр —
Легендага кергән кешеләр.

Милләте төрле булса да,
Әйтерсең, бер туганнар.
Авырлыклар каршысында
Каушап калмый торганнар.

Җир өстендә бодай дулкынлана,
Җир куйнында ята кара алтын.
Бөтен илебезгә дан итәләр
Әнә шулар Азнакай халкын.

Районыбыз хөрмәтенә
Армый-талмый эшлибез.
Азнакай безгә дан бирә,
Без аңа дан өстибез.

Тырыш хезмәт белән кайнап тора
Һәр көнебез, атна, аебыз.
Бишьеллыклар аша алга бара
Сөеклебез — Азнакаебыз.

Тормышыбыз яңарганга
Яшик әле шатланып.
Хезмәт уңышлары белән
Күңелләр канатланып.
```
