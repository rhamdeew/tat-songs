# Оригинал

### Эльверт Хәмәтшин / Мөфтәдин Гыйләҗев - Каеннарым мине көтәләр

```
Үстем туган якта,
Тыңлап балачакта
Каеннарның моңлы җырларын.
Онытмамын һич тә,
Булыр мәңге истә
Ак каенлы туган җирләрем.

Колагымда чыңлый,
Бер дә истән чыкмый
Таң атканда кошлар сайравы.
Сагышларым арта,
Үзләренә тарта
Каеннарның моңлы шаулавы.

Сагынып туган якны,
Гүзәл балачакны.
Сагышларда көннәр үтәләр.
Ә бит туган якта,
Гүзәл балачакта,
Каеннарым мине көтәләр.
```
