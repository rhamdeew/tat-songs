# Оригинал

### Гөл Мирһади / Эльвира Хаматнурова - Укытучыма (өченче вариант)

```
Киләчәге зур илемнең
Синең кулыңда, —
Укытучым! Бүген җырлыйм
Синең турыда.

Иң ягымлы,сабыр, гадел
Укытучым — сез!
Күңелемне, уйларымны
Яктыртучы — сез!

Олы йөрәгеңне тиңлим
Чиксез далага, —
Акыллы сүзләр табасың
Һәрбер балага.

Иң ягымлы,сабыр, гадел
Укытучым — сез!
Күңелемне, уйларымны
Яктыртучы — сез!

Җитәкләп алып менәсең
Белем үренә,
Шуңа урының да синең
Йөрәк түрендә.

Иң ягымлы,сабыр, гадел
Укытучым — сез!
Күңелемне, уйларымны
Яктыртучы — сез!
```
