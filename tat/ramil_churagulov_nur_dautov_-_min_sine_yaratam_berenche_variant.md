# Оригинал

### Рамил Чурагулов / Нур Даутов - Мин сине яратам (беренче вариант)

```
Вальс уйный. Таныш көй. Без икәү.
Бәхеттән бар дөнья әйләнә.
Тәннәрне, җаннарны — һәммәсен
Исерткеч мәхәббәт бәйләгән.
Бар дөнья бәхеттән әйләнә.

Ник кыска шушы вальс, ник кыска?
Музыка яңадан уйнасын.
Яныңда бер минут булсам да
Тагы да бик килә буласым.
Музыка тагын да уйнасын.

Без икәү булганга бәхетле —
Йөрәкләр мәңгегә бәйләнгән.
Мәхәббәт исерткән дөньяны,
Бар җиһан, йолдызлар әйләнә.
Йөрәкләр мәңгегә бәйләнгән.

Вальс уйный. Сихри көй. Без бергә.
Иң изге серемне аңлатам.
Бар җиһан, йолдызлар ишетсен:
Мин сине яратам, яратам.
Яратам, яратам, яратам!!!
```
